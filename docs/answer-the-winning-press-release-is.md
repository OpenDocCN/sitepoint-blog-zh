# 答:获奖的新闻稿是…..

> 原文：<https://www.sitepoint.com/answer-the-winning-press-release-is/>

https://www.sitepoint.com/guess-which-press-release-generates-more-press-and-business/

上面的博客问的是哪篇新闻稿(一篇通过讲述一个企业家使用我的服务重建他的企业的故事，背对卡特里娜飓风；一个是关于产品的普通香草:http://www.prweb.com/releases/2005/9/prweb286170.htm)。

你的帖子很有见地；谢谢！就个人而言，正如你从博客中看到的，我希望卡特里娜飓风的故事会被提起。的确如此，路透社的一篇报道(已发表)引起了一位《华尔街日报》记者的兴趣。不幸的是，媒体关注的是企业家和他的奋斗，而不是我的服务！没有一家当地报纸/媒体在佛罗里达州萨拉索塔报道此事。

与此同时，这项服务的普通版本给我带来了一些真正的客户。它在萨拉索塔佛罗里达州的当地出版物上被采用(较小市场的当地媒体喜欢宣布新的业务和服务)，潜在客户的反应很好——特别是考虑到 0 美元的投资。

这里有一些经验教训:

1.首先，你永远不知道新闻稿什么时候会被拿走，或者被谁拿走。这个想法就是不断尝试，用一系列有新闻价值的文章和技巧。

2.与目标媒体建立关系。这是让你的发布引起注意的最好方法。在这种情况下，vanilla 产品的发布是由一家媒体渠道获得的，我认识这家媒体渠道的出版商，我以前也曾获得过发布。小心地建立关系，不要乞求。联系编辑和记者，在 30 秒内告诉他们为什么他们的读者想读你的故事。

3.目标市场是最好的。刊登我这篇文章的出版物关注技术、新公司和我所在的萨拉索塔市场。我的“香草”释放击中所有这三个。你的新闻稿越有针对性，你就越能专注于选定的媒体，在撰写读者感兴趣的内容的同时，与编辑和记者建立关系。

4.与第二点和第三点相反，电子邮件和传真的效果远不如私人联系。编辑被这种垃圾塞满了，使用这些服务要花很多钱。

这次不要问问题了，对吗？
