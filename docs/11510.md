# SitePoint 综述#1:瑞普丹舒尔茨，谷歌关闭辩论，以及更多！

> 原文：<https://www.sitepoint.com/sitepoint-roundup-1-rip-dan-schulz-google-closure-debate-and-the-podcast-turns-one/>

我们的社区是 SitePoint 的核心和灵魂。这里有一个在过去一段时间里我们社区发生的事情的快速综述。

## 纪念丹·舒尔茨

我们非常悲痛和震惊地获悉，深受爱戴的 SitePoint 论坛成员丹·舒尔茨上周突然去世。Dan 在很长一段时间里都是我们社区的成员，他的帖子帮助了各种各样的人。他赢得了几个社区奖项，包括 2007 年我们的年度成员奖，并且是设计论坛的顾问。

在这份特别声明中，你可以读到更多关于所发生的事情，并看到对这一消息的一些反应。大量的回复证明了我们对他的爱和思念。

## 这个月的*社区宣传员*

*Community Crier* 是 SitePoint 的每月时事通讯，报道我们社区周围发生的事情，第 164 期于周三送达我们订户的收件箱。这也是我从琳达·简金森手中接过《T2 报》编辑的第一期杂志。琳达，你做得很好，感谢你过去的努力工作！

不幸的是，我已经需要发布一个**更正**。在介绍中，我提到了我们无畏的论坛管理员莎拉，就像这样:

> …你可能见过我带着闪亮的新红色管理员徽章在那里跑来跑去，还有莎拉·泰勒。

这应该读莎拉·拉奇——莎拉·霍克·泰勒实际上是我们同样无畏的支持忍者。我为这个错误感到非常抱歉，并向所有读者道歉——特别是两个萨拉！

订阅《呐喊者》和我们的其他时事通讯非常简单，只需几分钟。我们不会分享您的电子邮件地址，您可以随时取消订阅。来看看吧！

## 脸书的粉丝们建造了漂亮的东西

我们的[脸书页面](http://www.facebook.com/sitepoint)拥有超过 1800 名粉丝——设计师、开发者、博客作者和企业家，他们都在各自的项目上努力工作。我们给我们的脸书朋友发了一个[大喊](http://www.facebook.com/sitepoint#/sitepoint?v=feed&story_fbid=189112241584&)，看看他们在做什么。如果你还不是粉丝，来看看我们吧！
