# 一年后，1 亿次下载，火狐

> 原文:[https://www . site point . com/one-year-and-1 亿-downloads-later-firefox/](https://www.sitepoint.com/one-year-and-100-million-downloads-later-firefox/)

[SpreadFirefox.com](http://www.spreadfirefox.com/)有一张[图库截图](http://www.spreadfirefox.com/image/tid/84)上来庆祝[火狐的第一个生日](http://www.spreadfirefox.com/node/19854)。

在正式发布的第一年，它已经获得了超过 1 亿次的下载量(查看[的庆祝图片画廊](http://www.spreadfirefox.com/?q=image/tid/82)),现在预计已经达到 10%的市场份额(参见 [Asa Dotzler 的精彩博客](http://weblogs.mozillazine.org/asa/archives/2005/11/what_a_year_its.html))。

在发布 1.0 一年后，1.5 版本指日可待，现在可以下载第二个候选版本。我已经跟踪[补丁](http://forums.mozillazine.org/viewforum.php?f=23)和[提交 bug](https://bugzilla.mozilla.org/)一年了，我很自豪我报告的一些 bug*已经在即将到来的版本中修复了。向 Mozilla 团队提交 bug 报告是一次令人印象深刻的积极和有益的经历，这要归功于它的支持社区以及开发人员和修补人员团队。*

对我来说，火狐 1.5 最令人兴奋的更新之一是它的增量更新系统。更新可以在后台自动下载，并在下次浏览器重新启动时安装。此外，这些更新是增量式的——不再需要每次发布较小的更新就下载整个 6 MB。

如果你决定下载并安装 Firefox 的候选发布版本，你可能需要手动升级部分或大部分扩展，你可能会发现有些扩展没有更新版本。我们鼓励扩展作者在 1.5 版本发布之前，基于新的候选版本升级他们的扩展。

SitePoint 的书 [Firefox Secrets](https://www.sitepoint.com/books/firefox1/) ，这是一个关于使用、个性化和调整 Firefox 的完整指南，也包含了 Firefox 1.5(该书出版时称为 Firefox 1.1)的许多新功能的信息，例如新的简化选项面板。[这本书的精选章节可以在网上免费获得](https://www.sitepoint.com/books/firefox1/)。

我的下一步？也许有一天我会试着写一个 Firefox 扩展。也许我可以更深入 XUL，试着提交一个补丁。
