# 谷歌发布 Chrome 4.0

> 原文:[https://www.sitepoint.com/google-chrome-40-released/](https://www.sitepoint.com/google-chrome-40-released/)

对于浏览器厂商来说，这是忙碌的一周。微软一直忙着[给 IE](https://www.sitepoint.com/microsoft-emergency-ie-update/) 打补丁， [Mozilla 发布 Firefox 3.6](https://www.sitepoint.com/firefox-36-released/) ，现在谷歌又出了 Chrome 4.0。该浏览器可从 [Chrome 网站](http://www.google.com/chrome)获得，或者您可以通过点击工具图标，选择**关于谷歌 Chrome** ，并按照更新说明进行更新。

自 2008 年 10 月推出以来，Chrome 已经走过了漫长的道路。这款浏览器已经占据了超过 5%的全球市场份额，并且已经是第四版了。尽管谷歌以将产品保持在长期测试阶段而闻名，但 Chrome 版本号平均每五个月增加一次。我心中的愤世嫉俗者认为这是超越火狐和追赶 Safari 的营销策略。但是有人关心浏览器版本号吗？

从外观上看，Chrome 3 和 4 之间几乎没有什么变化。这款浏览器保留了其简洁的界面和速度的声誉。有两个主要的新功能:

**1。书签同步**
如果你有一个谷歌账户(谁没有？)，Chrome 会把你的书签上传到网上。你可以通过[谷歌文档](http://docs.google.com/)文件夹访问它们，或者将它们与浏览器的其他安装同步。据我所知，Chrome 是第一个提供这种功能而没有插件的浏览器。

**2。版本 3 支持扩展**
扩展，但是你需要在 Chrome 快捷方式中添加一个命令行参数。第 4 版允许每个人扩展，从 https://chrome.google.com/extensions 有近 2000 个可用。

Chrome 没有火狐提供的[插件的数量和质量，但我怀疑它可能会吸引一些用户远离 Mozilla 的浏览器。然而，对于开发人员来说，扩展相对较少；可能是因为 Chrome 没有提供火狐插件程序员所享受的集成水平。](https://addons.mozilla.org/firefox/)

其他值得注意的东西:

*   改进的 HTML5 支持。
*   完全 100/100 [酸 3](http://acid3.acidtests.org/) 通过。Chrome 现在可以匹配 Safari 和 Opera。火狐 3.6 得分 94/100，IE 8.0……20/100。
*   所有领域的性能改进。

我喜欢 Chrome。如果您正在网上冲浪或使用在线应用程序，并且需要一个快速、稳定的浏览器，这是一个很好的选择。在我看来，Firefox 仍然为高级用户和开发者提供了更好的全方位体验，但 Chrome 正在快速追赶。
