# 10 个 jQuery 浏览器优化工具

> 原文：<https://www.sitepoint.com/10-jquery-browser-tweakers/>

对 jQuery 来说，调整浏览器并不是一项不同的任务。事实上，jQuery 有很多插件可以真正调整你的浏览器。虽然有几种浏览器调整，但我们将重点关注 jQuery 浏览器调整，因为 jQuery 是我们这里的主要话题。所以，我又一次浏览了不同的网站，找到了这些对你有用的 jQuery 浏览器调整。

他们在这里:

## 1.jQuery 可访问

将提高网站可访问性的片段集合。它包括一组编码为测试的“建议”,以改进它，甚至超越 javascript，然后这些建议可以在一个页面上进行尝试，并获得结果摘要。
*来源:*
【http://flesler.blogspot.com/2007/12/jqueryaccessible.html】T4

## 2.内容可编辑

此属性将允许您使文档的某些部分可编辑。支持火狐 3，IE6/7/8，Safari，Chrome 和 Opera。
*来源:*
【http://valums.com/edit-in-place/】T4

## 3.Ajax 上传

通过使用 XHR，这个插件可以在火狐 3、Safari 4、Chrome 上上传多个带有漂亮进度条的文件。
*来源:*
【http://valums.com/ajax-upload/】T4

## 4.不良浏览器

这将为使用 Internet Explorer 浏览器的用户显示一个通知。您必须提供将要显示的 HTML。
*来源:*
【http://code.webs.ly/jquery-badbrowser/】T4

## 5.将对象底部对齐

这将使用间隔将对象与底部对齐。它兼容 iPhone 用户。
*来源:*
【http://www.edwardhotchkiss.com/】T4

## 6.黑白色

Internet Explorer 用户将看到您的网站带有灰度颜色。这可能会帮助你的 IE 访问者继续使用他们的旧浏览器。

## 7.Canvasswap

这个 jQuery 插件将在悬停时进行图像交换。Canvasswap 不仅可以在 hover 上进行惊人的图像交换，它还支持 Internet Explorer 6 上的透明 PNG。
*来源:*
【http://www.trevorsheridan.com/canvasswap/】T4

## 8.复选框模式 2

这将增强 pages 中的实际复选框。这将取决于操作系统和主题。实际上，用户有不同的复选框外观和感觉。Windows XP 有一个好看的复选框，但如果与 Windows 7 相比，它甚至更漂亮。
*来源:*
【http://nordz.sauleil.com/jquery-plugins/】T4

这个简单的 jQuery 插件将允许你以跨浏览器的方式复制文本。
