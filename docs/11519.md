# 如何不无聊

> 原文:[https://www.sitepoint.com/how-not-to-be-boring/](https://www.sitepoint.com/how-not-to-be-boring/)

我们中的许多人都依赖于常规。我们每天都有自己特定的管理方式，我们倾向于习惯于这样做。这并不是说我们不会经常偏离轨道，但是我们大多数人都有一个偏好，那就是如何度过每一天。

这种类型的例行公事，不管是不是有意的，都可以帮助我们变得更有生产力和效率。但是，这也会让我们更容易陷入常规。当我们陷入这种困境时，我们可能会变得有些…嗯，无聊。

这里有一些方法来增加你的有趣因素，避免无聊的耻辱。

## 追求不可预知的事物

自发性是让自己变得更有趣的一个好方法，尤其是如果你通常是时间表的奴隶。尝试通过做一些意想不到的事情，说一些与性格不符的话或凭冲动行事来挣脱束缚。

## 专注于对话

采用对话的语气而不是陈述的语气肯定会让你不那么无聊。这意味着问更多的问题，仔细倾听，并专注于创造一个持续的对话。

## 保持原创

不要随大流——不管是通过你的营销活动、日常计划过程，还是网络方法——找到你自己的路。跳出框框思考新的想法和新的方法来做平常的事情，让自己变得更有趣。

## 表明立场

如果你想进行一些建设性的辩论，就一个流行的问题表明立场，并让大家知道。你肯定能够通过宣布你对一个争论已久的主流问题的观点来避免变得无聊。请确保您准备好备份它。

## 展示你愚蠢的一面

每个人都喜欢开怀大笑。做一点傻事来缓解一下这一天的喜剧气氛。这不仅会显示你有多有趣，还会让你更可爱，更平易近人。

## 教点东西

你脑子里有很多诀窍，也许是时候分享一下了。提供一些鲜为人知的小技巧、建议或工具，帮助别人学到一些有价值的东西。如果你经常这样做，你一定会避免这个无聊的标题。

## 进行人身攻击

你上一次与你的客户、同事或朋友分享个人信息是什么时候？讲一个故事，描述你的一次经历，或者分享一些别人不太了解你的事情。这可以帮助你加强你的人际关系，并显示你真的很有趣。

你怎样保持有趣，避免无聊？
