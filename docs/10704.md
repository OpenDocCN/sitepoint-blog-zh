# IE9 测试版–2010 年 9 月 15 日

> 原文:[https://www.sitepoint.com/ie9-beta-release/](https://www.sitepoint.com/ie9-beta-release/)

微软将于 2010 年 9 月 15 日发布 Internet Explorer 9.0 的首个公测版。这将是一个完全可操作的浏览器，没有开发者预览版的限制(它将有地址栏，导航，没有受限的功能)。

IE9 是微软赶超竞争对手的尝试。该浏览器将采用硬件加速的 HTML5、CSS3、SVG、canvas、新的 JavaScript 引擎和大幅改进的标准支持。

不幸的是，IE9 测试版只能在 Windows Vista 或 7 上运行，并将覆盖您当前版本的 Internet Explorer。如果您只有一台 Windows PC 用于 IE8 测试，请小心。你有一个月的时间考虑选择:

1.  再买一台 PC 做 IE9 测试。这当然是大公司的一个选择。
2.  不想升级的 XP 用户将需要在虚拟机中安装 Windows Vista 或 7 来测试 IE9。
3.  Windows Vista 或 7 用户可以安装 IE9 beta 并在虚拟机中运行 IE8。Windows 7 专业版及更高版本包含 [XP 模式](https://www.sitepoint.com/article/ie6-ie7-ie8-win7-xp-mode)，因此无需购买更多 Windows 许可证。
4.  或者，如果你不想在最终发布之前影响你的 PC 上的软件和设置，微软将提供另一个版本的[平台预览版](http://ie.microsoft.com/testdrive/)。同样，这将只能安装在 Windows 7 或 SP2。

当然，如果能在同一台电脑上同时安装多个版本的 Internet Explorer，微软会让这变得容易得多。为 Windows XP 提供 IE9 也是一个额外的好处。但是浏览器看起来很有前途，所以也许我们应该庆幸 IE 终于赶上了竞争对手。

你会被诱惑去尝试 IE9 的测试版吗？
