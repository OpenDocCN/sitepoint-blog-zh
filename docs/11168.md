# 在 Photoshop 中创建工作室风格的背景

> 原文:[https://www . site point . com/create-a-studio-style-background-in-Photoshop/](https://www.sitepoint.com/create-a-studio-style-backdrop-in-photoshop/)

如果你在你的网站或平面广告中展示一个产品，你可以通过给它一个像样的背景和添加一些特殊效果来使它看起来更专业。

对于我的“产品”，我使用的是我从 Stock Xchng 免费下载的一张非描述性媒体播放器的照片。感谢 [Pzado](http://www.sxc.hu/profile/pzado) 。
[![media_player](../Images/01eda4c904e9ec49a3c1ecb7877760e3.png "media_player")](https://www.sitepoint.com/wp-content/uploads/2010/03/media_player.jpg)

1.创建新的 Photoshop 文档。对于本例，将其设置为 800 x 600 像素，分辨率为 72ppi。

2.通过按 D 键将前景色设置为黑色，然后按 Alt + Backspace (windows)或 Option + Delete (Mac)，用黑色填充整个文档。

3.从工具栏中选择椭圆选框工具，画出一个圆形选框。按住 Shift 键以在拖动时将其约束为一个正圆。用白色填满圆圈。按 Ctrl + D (windows)或 Cmd + D (Mac)取消选择该圆。

[![Black Background White Circle](../Images/d131c6bc2cd01297510acfe189aa6f6e.png "Black Background White Circle")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/BlackBackgroundWhiteCircle.jpg)

4.选择“滤镜”>“模糊”>“高斯模糊”,将半径设置为 90-100 像素，这样圆形的硬边就被完全去除了，你会得到这样的效果。

[![Black Background White Blur](../Images/2506469562fd6a88579c886fbe8bcf60.png "Black Background White Blur")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/BlackBackgroundWhiteBlur.jpg)

5.选择矩形选框工具并拖出一个选区，该选区大约覆盖文档底部的三分之二。

[![Two Thirds Selection](../Images/3515c4c896b9891273771c6bd77a533d.png "Two Thirds Selection")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/TwoThirdsSelection.jpg)

6.按 Ctrl + J (windows)或 Cmd + J (Mac)复制选区，并将其放在自己的图层上。

[![Layers](../Images/94c6fc8aee4514be8880633294efdbca.png "Layers")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/Layers.jpg)

7.使用移动工具，将新图层向下拖动到文档底部的三分之一处。这将是反映我们产品的表面。

[![SelectionMoved](../Images/2c1427e4828f0d39bb094aaa66d1cc20.png "SelectionMoved")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/SelectionMoved.jpg)

8.我们需要这两层之间有一些对比。确保选择了顶层，然后选择“图像”>“调整”>“色阶”。向右拖动中间的滑块，使该层变暗。单击确定。按 Ctrl + E (windows)或 Cmd + E (Mac)来合并这两个文件。

[![Levels](../Images/1801fb033863f200993f21947bea26cd.png "Levels")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/Levels.jpg)

9.现在添加一点颜色。这种背景效果在很多颜色中都很好看，但是深蓝和深绿往往看起来很有型。选取“图像”>“调整”>“变化”。“变化”对话框允许您查看图像在各种色调下的外观。点击更多青色预览。这将为所有预览图像添加更多青色。

[![Variations](../Images/4e945449cd1f252bb72ebaf663782b9b.png "Variations")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/Variations.jpg)

在屏幕的顶部，你可以看到你原来和现在的选择。再次按下“更多青色”,使你当前的选择看起来像这样。然后单击确定。

[![variations-cyan](../Images/66535562358b66cbbd20def86155a5c3.png "variations-cyan")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/variationscyan.jpg)

10.好了，现在我们准备放置我们的产品。打开从 Stock Xchng 下载的媒体播放器的图像。这张图片比我们创建的背景大得多，所以我们需要在接下来的几个步骤中考虑到这一点。我使用钢笔工具在播放器周围做了一个选择，但是你可以使用任何你想要的选择工具，只要确保你的选择中不包括任何白色背景。如果你想使用你自己的产品，在白色背景上拍摄。

[![media_player](../Images/25bc9d3bacf15400977fef4e84c4b7e5.png "media_player")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/media_player1.jpg)

11.做出选择后，按 Ctrl + C / Cmd + C 复制选择。转到您的背景图像，然后通过按 Ctrl + V / Cmd + V 粘贴到扬声器中。正如已经提到的，媒体播放器与背景相比会显得很大。按 Ctrl + T / Cmd + T 执行自由变换。如果您看不到自由变换手柄，只需按 Ctrl + 0 / Cmd + 0，使
图像和手柄适合屏幕。

[![PasteSpeaker](../Images/a91f8d77046045b3b2d3268bec6708b8.png "PasteSpeaker")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/PasteSpeaker.jpg)

现在按住 Shift 键的同时向内拖动右下角的手柄。如果要旋转媒体播放器，请将光标移到变换角手柄的边缘，并以圆周运动方式拖动。

[![mediaplayer-background](../Images/82480b8151c24e517da356f552c07361.png "mediaplayer-background")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/mediaplayerbackground.jpg)

12.要进行反射，复制媒体播放器层。按 Ctrl + T / Cmd + T 再次调出自由变换手柄，然后右键单击或按住 Control 键在边界框内单击。从出现的下拉菜单中，选择垂直翻转。向下拖动翻转的图像，直到播放器的边缘接触到反射的边缘，这样你就会得到这样的效果。

[![reflection](../Images/bd87c5a62d4925a3e752485b5ffb9ae0.png "reflection")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/reflection.jpg)

13.为了改善反射，降低这一层的不透明度到 10%到 20%之间。请记住，这是一个相当程式化的作品，所以它应该看起来半真实，但它不会看起来 100%真实。

[![reflection-complete](../Images/0b9793d8dbab6f8743f5571519b42821.png "reflection-complete")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/reflectioncomplete.jpg)

14.将反射层拖到媒体播放器层下面。

[![Layers With Reflection](../Images/90b3093043ea9219850b94bb6b155b00.png "Layers With Reflection")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/LayersWithReflection.jpg)

15.现在，为了让倒影更真实地消失，我们将添加一个图层蒙版。确保你的反射层在图层面板中被选中，然后点击图层面板底部的添加图层蒙版图标。一个白色缩略图出现在图层上。单击缩略图将其选中。现在选择渐变工具，选择前景到背景渐变。

[![Gradient](../Images/65142207283fce10ea2e8e0c3a50d847.png "Gradient")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/Gradient.jpg)

点按反射图像的顶部，然后向下拖到您想要图像淡出的位置。

16.现在我们将整个图像变暗一点。它减小了光晕的大小，但整体上看起来更具戏剧性。选取“图像”>“调整”>“色阶”,并将中间的滑块拖到右边。

[![LevelsOnBackground](../Images/a24277c00f4b07346ba48d641461171e.png "LevelsOnBackground")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/LevelsOnBackground.jpg)

您的图像现在应该看起来像这样:

[![Darkenedfinal](../Images/3f66833ede299875728c19932dd160b6.png "Darkenedfinal")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/Darkenedfinal.jpg)

最后，我添加了一些文字。此处使用的字体是 Century Gothic，标题为粗体，其余部分为常规字体。

[![FinalProductImage](../Images/b0d8e6d070d5141cde79c5045b634840.png "FinalProductImage")T2】](https://www.sitepoint.com/wp-content/uploads/2010/03/FinalProductImage.jpg)

现在你知道了。这是一个你可以在许多产品中重复使用的模板，并且很容易通过变化或者色调和饱和度调整来改变背景颜色。
