# 谷歌发布 Chrome 9

> 原文：<https://www.sitepoint.com/google-chrome-9/>

你可能没有注意到，你的 Chrome 浏览器安装已经神奇地更新到了版本 9。如果没有，点击工具图标，然后点击“关于谷歌浏览器”

最令人兴奋的功能是 web GL——一种在 HTML5 `canvas`元素上提供原生浏览器硬件加速 3D 图形的新技术。在 [Chrome 实验](http://www.chromeexperiments.com/webgl)网站上有一些令人惊叹的演示:

Chrome 是第一个支持 WebGL 的主流浏览器，但在 Firefox 4.0 和 IE9 中应该也有。绝对值得一看的技术。

下一个新功能是 Google Instant。你可能在谷歌搜索上见过这种情况:当你键入一个搜索短语时，结果会出现并改变。我并不特别喜欢这个工具，但是当你使用 omni box(Chrome 地址栏的愚蠢名字)时，现在可以看到谷歌即时结果。默认情况下不启用它——在选项对话框的基本选项卡上有一个复选框。

最后，美国的 Chrome 用户可以在“新标签”页面上链接到 Chrome 网上商店。该商店提供了一系列免费和商业的 Chrome 应用程序、扩展和主题。如果你不在美国，你仍然可以在 https://chrome.google.com/webstore[的](https://chrome.google.com/webstore)用你的谷歌账号登录商店。

尽管还不到两年半，Chrome 的版本现在已经领先于 Internet Explorer，并将在今年年底超过 Opera。当然，这一切都有点傻——甚至谷歌自己的博客公告也没有提到这个数字。

然而，看看谷歌是否必须解决两位数的用户代理问题将会很有趣，Opera 在从版本 9 升级到版本 10 时遇到了这个问题。虽然[浏览器嗅探是一种可怕的做法](https://www.sitepoint.com/why-browser-sniffing-stinks/)，许多网站检查浏览器版本的第一个数字。因此 Opera 10 被认定为 Opera 1，网站被屏蔽或降级。这个问题不太可能对谷歌产生同样程度的影响，但我们会在几个月后 Chrome 10 发布时知道更多。
