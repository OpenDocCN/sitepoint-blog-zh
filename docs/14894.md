# 回顾网站可用性和导航:初学者指南

> 原文:[https://www.sitepoint.com/navigation-beginners-guide/](https://www.sitepoint.com/navigation-beginners-guide/)

对许多人来说,“初学者”这个词有负面的含义。但事实是，即使你认为自己很有经验，也有无数的事情你不知道，所以每天继续学习是很重要的。

尽管这本书的标题中有“初学者”这个词——即使你已经设计和运行网站很多年了——从《网站可用性和导航:初学者指南》中还是可以学到很多东西。

记住，这不是网页设计的“初学者”指南。这是一本关于网站可用性和导航的“初学者”指南。尽管网页设计和导航设计有时是紧密相连的，但它们是两个不同的主题，都值得你关注。很多网页设计师都没有(或者就是拿不到！)足够的网站可用性经验——这本书可以帮助你拓展你的经验。

这本书有将近 550 页(包括附录)，不是一个快速阅读的书，但它是一个信息丰富的书——非常值得花时间来消化这本书提供的有价值的信息。网站可用性和导航:初学者指南可能不会涉及超高级的概念，但它确实涵盖了很多内容，是开始深入研究这个主题的一个极好的地方。

##### 第一部分:从用户的角度看问题

本书的开篇部分向你展示了如何以用户的身份体验 Web，即抛开你所有的 Web 开发知识，按照你的用户意愿使用你自己的站点。这一部分还帮助你创建网站使用场景和用户档案，这将帮助你分析你的用户群，并确定他们是谁，以及他们如何使用你的网站。

考虑用户的技术限制也很重要，这样你就可以避免开发你的网站需要使用技术，或者提供可能实际上阻碍访问者使用你的网站的“功能”。第一部分是一些关于网络的一般信息，它的局限性和用途。

##### 第二部分:为你的用户设计网站

这本书的两个主要部分中的第二个也是最大的一个，第二部分详细介绍了如何设计你的网站，使它们对最终用户尽可能有效。这个讨论首先解释以用户为中心的设计过程，以及如何通过这个过程的不同阶段实现以用户为中心的设计。

第二部分继续介绍如何选择正确的域名，并确保您拥有正确的内容。还讨论了基本的信息架构策略，随后探讨了导航设计。

这本书接着讨论了一些常见的可用性缺陷(包括浏览器问题、闪屏和插件)。它还包括一个关于屏幕可用性设计的详细部分，该部分讲述了当你设计一个能在不同尺寸和类型的显示器上成功显示的网站时，你必须忍受的复杂性。

为网络准备用户友好的内容是下一个主题，接下来是对几个网站的案例研究。这些研究将帮助你理解如何分析其他网站的网站可用性问题，并从他们的错误中学习。

我之前提到过可用性测试，尽管在书中经常提到，但在第二部分的结尾有一个完整的模块专门讨论可用性测试。这个模块向你展示了如何计划和进行可用性测试，以及如何解释测试结果。

本书的这一部分以对网络未来的讨论结束，以及作为一个网页设计师，你应该在网络可用性的新兴世界中寻找什么。

##### 精通检查

这本书分为三部分:上面提到的两部分，加上附录。

前两部分进一步细分为涵盖特定主题的“模块”。一旦你完成了一个模块，你会发现一个方便的掌握检查，这是一个测试，以确保你理解和理解你刚刚阅读的内容。

你可以参加或忽略这些测试，但是，不管怎样，它们以后会成为一个方便的参考。每个模块末尾所提问题的答案(以及问题本身)也可以在本书的附录部分找到，让您可以快速轻松地阅读问题并找到答案。

附录还包括一些网站可用性清单的样本，你可以在你自己的网站上使用，或者作为开发你自己的清单的基础。
