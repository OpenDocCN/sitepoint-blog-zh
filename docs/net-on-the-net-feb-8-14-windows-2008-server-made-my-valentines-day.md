# 。2 月 8 日至 14 日:Windows 2008 服务器让我过了情人节

> 原文：<https://www.sitepoint.com/net-on-the-net-feb-8-14-windows-2008-server-made-my-valentines-day/>

嘿女士们和小伙子们。对于. NET 来说，这是非常非常重要的一周。首先， [Windows Server 2008 正式发布](http://blogs.technet.com/windowsserver/archive/2008/02/04/windows-server-2008-rtm.aspx)，目前 MSDN 和 TechNet 用户均可使用。对我们这些 ASP.NET 类型的人来说，最重要的一点是，这打开了生产 IIS7 托管的大门。这通常解释为“不再需要调用系统管理员来调整一些 IIS 设置，因为我可以在我的 web.config 中设置它们”或“天哪，蝙蝠侠，这个集成管道太棒了！”此外，对于我们这些疯狂到要在 Vista 上开发的人来说， [Service Pack 1 也于本周向生产部门](http://windowsvistablog.com/blogs/windowsvista/archive/2008/02/04/announcing-the-rtm-of-windows-vista-sp1.aspx)发布，应该会在 3 月 18 日之前广泛发布。

此外，微软还发布了一个 Visual Studio 修补程序,旨在显著提高性能并消除 web 设计工具中的一些错误。一个非常重要的分支是 [JQuery 现在可以与 Visual Studio 2008 intellisense](http://blogs.msdn.com/webdevtools/archive/2008/02/08/jscript-intellisense-working-with-jquery.aspx)一起工作。最后，Scott Guthrie 更新了 ASP.NET MVC 路线图，同时，为了那些困在 SharePoint 中的可怜人，Visual Studio 团队发布了一个重要的更新。

在可以说是本周最酷的发布会上，蒂姆·斯内思宣布发布视频。显示 1.0 。对于外行人来说，它实际上是一个 Silverlight 驱动的 YouTube-in-a-box 解决方案，看起来非常非常漂亮。欲了解更多信息，请参见[乔恩·盖洛威关于发布](http://weblogs.asp.net/jgalloway/archive/2008/02/12/video-show-1-0-a-ready-to-run-solution-for-hosting-video-on-the-web.aspx)的帖子。我们可能很快就会在非微软所有的地产上看到 Silverlight。

说到乔恩(一个好人，也是 SitePoint 的《ASP.NET 选集》的合著者)，他也发表了一篇经典的关于使用 JavaScript 和 ASP.NET 在 AJAX 之外交谈的文章。同样在客户端脚本方面，Rick Strahl 告诉你如何以干净的方式在 JavaScript 中嵌入 ASP.NET 服务器变量。不要让这个伟大的标题欺骗了你，但是冯#的“[放下 XmlNode，远离 Stringbuilder](http://www.vonsharp.net/PutDownTheXmlNodeAndStepAwayFromTheStringBuilder.aspx) ”是一个很好的 XmlSerialization 入门指南。

对于那些对多媒体感兴趣的人来说，这里有一些亮点。首先，Joe Stagner [给了我们一个 4 集的视频系列](http://weblogs.asp.net/joestagner/archive/2008/02/12/new-4-part-series-of-videos-on-downloading-with-asp-net.aspx)关于每个人最喜欢的难题:在 ASP.NET 上传。其次，他还在 ASP.NET 3.0 中最酷但最不为人所知的部分:Windows Workflow Foundation 上[翻出了一个 7 集的视频系列](http://weblogs.asp.net/joestagner/archive/2008/02/05/windows-workflow-foundation-tutorial-series.aspx)。最后，对于那些做播客的人来说。NET Rocks [采访了 Mono 项目](http://www.dotnetrocks.com/default.aspx?showNum=313)的米盖尔·德·伊卡萨和杰夫·诺顿。

最后但并非最不重要的是，有两个很棒的帖子是关于我最关心的一些最佳实践的:让你的东西安全运行。在红色角落，Gustavo Duarte 告诉我们如何[锁定 Sql Server 2005](http://duartes.org/gustavo/articles/Lock-Down-SQL-Server-2005.aspx) 。在蓝色角落，我们请到了来自 daptivate.com 的 Kyle，他将为我们讲述生产 ASP.NET 应用的[10 大最佳实践](http://daptivate.com/archive/2008/02/12/top-10-best-practices-for-production-asp-net-applications.aspx)。

尽情享受吧！
