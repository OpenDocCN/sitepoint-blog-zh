# 谷歌 Chrome 如何变得更受欢迎？

> 原文：<https://www.sitepoint.com/google-chrome-more-popular/>

![Google Chrome](img/5eaba2d6beb92f247a899872d26cd973.png "Google Chrome")谷歌最近在纽约时代广场进行的[调查揭示了一些有趣的事实:](https://www.sitepoint.com/what-is-a-web-browser/)

1.  只有不到 8%的人知道浏览器是什么。许多人将其与搜索引擎或公司相混淆。
2.  大多数用户不知道也不关心他们使用的是哪种浏览器。这只是他们点击进入网络的一个图标。
3.  几乎没人听说过 Chrome。

谷歌品牌的网络浏览器 Chrome 已于 2008 年 9 月上市。它以干净、简约的界面、快速的渲染速度、匿名浏览功能和高稳定性而闻名。每个选项卡都由一个单独的进程处理，所以一个页面崩溃不会导致整个应用程序崩溃。

毫无疑问，这是一款很棒的浏览器，在第一年内就获得了 2-5%的市场份额。然而，很难确定 Chrome 吸引了多少非技术用户。网络开发人员当然会使用它，但谷歌的调查表明，它还没有达到主流意识。

谷歌——如果你在听的话——以下是我对提高 Chrome 使用率的建议。

**1。改名字**
*【Chrome】*是一个聪明又极客的名字，但对大多数人来说绝对没有任何意义。我怀疑 IT 行业中很少有人理解其重要性。谷歌有以应用程序的功能(文档、地图、电子邮件等)命名应用程序的历史。)，他们的浏览器应该也不例外，例如

*   谷歌互联网
*   谷歌网络
*   互联网
*   或者仅仅是“Google ”,因为这是大多数用户打开的第一个页面。

**2。让它出现在用户桌面上**
绝大多数用户只有在已经安装了 Chrome 的情况下才会使用它。因此，谷歌应该:

1.  与个人电脑制造商和软件生产商做生意。
2.  提供 Chrome 作为相关软件的可选附件，如谷歌桌面。
3.  在谷歌所有的网站上推广该浏览器。Chrome 的下载链接已经显示给了 Google.com 的 IE6/7 用户。)，所以这个程序可以扩展到像 YouTube 这样的网站。谷歌需要小心强势广告，但一次退出点击可能适用于所有网站。

**3。保持浏览器简单**
让我们停止装腔作势。谷歌有一个主要目标:击败微软和 ie 浏览器。许多用户没有理由从 IE 切换，所以谷歌应该集中精力说服他们，Chrome 易于安装，更简单，使用起来更快。安全性和稳定性是额外的好处，但是，由于很少有 IE 用户故意遇到问题，这些因素不应该成为主要营销信息的一部分。

我不反对谷歌用插件等功能来改进浏览器，但大多数用户永远不会关心或使用这些设施。如果一个选项导致额外的复杂性(甚至是一个额外的菜单项)，Google 应该认真考虑它的实现。

开发人员和高级用户已经有了 Firefox 和 Opera，因此匹配这些浏览器中的功能集可能会转化一部分懂 IT 的用户。然而，如果谷歌将 Chrome 定位于 92%不知道什么是浏览器的人，或许会更好。

你有其他想法吗？谷歌能说服人们放弃 IE 吗？Chrome 会不会只是抢走其他浏览器的市场份额？
