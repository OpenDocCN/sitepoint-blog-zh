# HTML 5 有 Logo！

> 原文：<https://www.sitepoint.com/html-5-has-a-logo/>

在过去的一年中，HTML 5 已经成为了一个大辩论、大热情和小困惑的主题。在某些角落，HTML 5 被誉为网页设计的救星，也许真的如此。谁知道呢？作为一个设计迷，我感兴趣的新闻是万维网联盟(W3C)最近公布了 HTML5 的新标志。该标志还不是 ***的官方标志*** ，但已经推出，以建立社区兴趣和精神。如果像 W3C 所希望的那样，社区接受这个标志，W3C 将在 2011 年第一季度采用它作为自己的 HTML5 官方标志。

该标志是一个简单的盾牌，有两种色调的橙色，用来表达盾牌的弯曲和白色的大型风格化数字 5。它确实有点超级英雄的味道。设计事务所为什么选择橙色？“橙色意味着兴奋，但也暗含着谨慎。橙色抓住了你的注意力，但并不为此尖叫……性感带着安全网，有组织，有力量，持久。HTML5，这个 logo，和 orange 很搭”。

[![HTML5_Logo_512](img/3ec01f20f7ee3c5b2fd199b4be7d409e.png)](https://www.sitepoint.com/wp-content/uploads/2011/02/HTML5_Logo_5121.png)

该标志由设计机构 [Ocupop](http://ocupop.com/) 创作。设计师 Michael Nieling 在 T2 的一份声明中说:“HTML5 这个词已经有了自己的生命；当这个术语不仅仅指规范本身时，对于 HTML5 到底是什么，开发人员社区和公众都有很大的困惑和争论……这个标准需要一个标准。也就是说，HTML5 需要一个一致的、标准化的视觉词汇来作为对话、演示和解释的框架。

在 [logo 的新网站](https://www.w3.org/html/logo/index.html)(想象一个 logo 有一个完整的网站专用于它)上，你可以读到 W3C 如何设想这个 logo 将被使用，以及这个 logo 在不同网站上的例子

> 它像你所写的标记一样强大、真实、有弹性、通用。它就像你这样有远见、有奉献精神的 web 开发人员一样明亮、大胆。这是标准的标准，进步的旗帜。而且它肯定不会用表格来布局。

[![HTML5_Badge_256](img/18afd487f453b0ea5f7c8ccb33a4166e.png)](https://www.sitepoint.com/wp-content/uploads/2011/02/HTML5_Badge_256.png)

在你的网站或博客上使用这个标志并不意味着有效性或一致性，而是你对这个概念的支持。HTML5 徽标是在知识共享署名 3.0 下许可的。网站上的所有标志都可以免费使用和“重新想象”，这是一个很好的机会让他们弄脏手，玩它。如果你真的喜欢这个标志，也有免费的贴纸和 t 恤。
