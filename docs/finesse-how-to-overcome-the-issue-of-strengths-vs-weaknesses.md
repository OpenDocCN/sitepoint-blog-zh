# 策略:如何克服优势和劣势的问题

> 原文：<https://www.sitepoint.com/finesse-how-to-overcome-the-issue-of-strengths-vs-weaknesses/>

上一篇博客让你思考你最大的弱点，或者为什么客户不雇佣你。然后它问你如何处理与客户的关系。

以下是我的回答:

每个优点都是缺点，每个缺点都是优点。那些知道如何将弱点转化为优势(将竞争对手的优势转化为弱点)的人会比那些不知道的人获得更多的客户。

把你的弱点变成优势的方法是玩“这正是你应该雇用我的原因”的游戏。它是这样工作的:

1.客户告诉你一个问题或察觉到的弱点。
2。你说，“这正是你应该雇用我的原因。”
3。巧妙的解释原因。

以下是一些例子:

客户:你太年轻了，连高中都没毕业。你:这正是你应该雇用我的原因。我对网页设计充满热情，而一些老对手可能有点厌倦了，死记硬背。此外，我知道帮助你创建一个强大的网站的最新创新，而那些比我更久的人可能会有点生疏。另外，我知道有些客户可能会怀疑。这就是为什么我会尽我所能变得灵活、反应灵敏，并尽我所能让你开心。最后，开销对我来说不是一个大问题，所以你会为高质量的工作得到一个公平的价格。

客户:你没有像我一样的以前客户的证明。你:这正是你应该雇用我的原因。我是这个行业的新手，正在建立客户群。虽然有大量推荐人的网页设计师可能会认为你是理所当然的，但我会尽一切可能赢得你的信任，把你变成推荐人。

客户:我不喜欢你。你:这正是你应该雇用我的原因。很多人对我有时粗暴的个性反应消极。但这意味着我不用担心受欢迎。我可以保持客观，专注于帮助你实现商业目标——即使这意味着给你一些艰难的建议。而且，我脸皮厚，不介意你告诉我你的想法。

客户:你似乎对你的事业不够投入。毕竟，你只是在做兼职。你:这正是你应该雇用我的原因。我是兼职的，所以为了管理我的时间，我必须仔细选择我的客户。如果我们一起工作，你将是少数几个精选客户之一。这样，我就能全神贯注地关注你了。此外，我不介意周末加班，通宵为你完成工作，所以你会发现我比那些朝九晚五的全职员工更容易接近，反应也更快。

好了，现在够了。在销售和营销中，几乎没有什么是绝对的。这一切都是为了创造感知。这需要技巧。
