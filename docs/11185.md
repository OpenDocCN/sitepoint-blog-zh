# 设计颜色:黑色

> 原文:[https://www.sitepoint.com/color-in-design-black/](https://www.sitepoint.com/color-in-design-black/)

我们即将结束这个系列。上周我发布了关于灰色的文章，并附有一些它在网页设计中的应用的例子。本周，我们将采用更深的颜色，重点是黑色。像灰色一样，黑色提供了一种接近中性的背景，这使得设计中的其他元素更加突出。许多摄影网站使用黑色背景，让照片成为焦点。

黑色是一种强有力的颜色。它可以是严肃的和复杂的，它使其他(较浅)的颜色流行。使用黑色时有一种重量感，这种感觉在其他颜色中是不存在的。偶尔，你会在印刷中看到颠倒的文字，即黑底白字，这通常出现在名片上，但在网上我们经常看到，看起来非常醒目。

**好黑/坏黑**

一些与黑色有关的联想当然是哀悼、死亡和邪恶，但从好的方面来看，黑色可以散发出优雅、紧张、自信、神秘、力量和力量。

我们还可以思考一下日常语言中对黑色的一些积极和消极的看法:一些花哨的东西可以被描述为黑色领带，或者一个专家(不仅仅是武术方面的)可以被称为黑带。从负面来看，一个人可能是一个无赖、黑心、害群之马或者被列入黑名单。

**黑色设计**

黑色几乎可以和任何颜色搭配。它使其他颜色显得更明亮。即使是非常深的颜色也可以和黑色搭配得很好，只要纹理能把它们区分开来。与白色搭配使用，对比度更高。黑色和红色也非常抢眼和引人注目，而与橙色相结合，它仍然很有吸引力，但有万圣节的内涵。黄色真的是从黑色背景中蹦出来的，而浅蓝色给人一种保守的感觉。
