# 证明的重要性

> 原文:[https://www.sitepoint.com/the-importance-of-testimonials/](https://www.sitepoint.com/the-importance-of-testimonials/)

[https://www.sitepoint.com/blog-post-view.php?id=170248](https://www.sitepoint.com/blog/)

上面的博客有很多关于证明的重要性的讨论。你们中的一些人表示，在你们的经历中，证明实际上会降低可信度。

读到这两个回答时，我感到非常惊讶，因为一次又一次的测试表明，正确执行的推荐会增加各种专业人士的反应，包括 web 设计人员/开发人员。

因此，本博客旨在提出几点来阐明推荐的作用:

1.“社会证明”的概念在市场营销中非常重要。社会证明，简单地说，意味着人们倾向于相信其他人相信的东西，尤其是他们尊重的人。因此，如果你能召集一群人，尤其是那些对你赞不绝口的意见领袖，你就建立了可信度。不幸的是，这就是我们人类的工作方式。那么，为什么不在你的营销工作中利用社会证明呢？

2.有很多方法可以利用社会证明来建立可信度:证明、案例研究、转载你的文章或当地报纸对你的采访，以及推荐系统。在一个不同的行业，信不信由你，“在电视上看到的”贴花已被证明可以大大增加某些产品的零售额。

3.推荐不是一个非此即彼的命题。你还想展示你的作品集。然而，许多潜在客户(包括我自己)无法区分好的投资组合和坏的投资组合。这就是为什么我们需要网页设计师，对不对？投资组合往往更多的是关于你而不是客户——特别是如果它们不包括客户试图实现的一些细节，你为他们获得的具体结果，以及引用客户自己的话证明你有多棒的话。那么，为什么不利用所有这些营销功能来优化你的信息呢？

4.如果做得好，在你的主页顶部放一篇推荐，会立即吸引读者，尤其是在以下情况下:

–他们是公认的意见领袖；你附上他们的缩略图；
–你给他们的报价加一个标题，抓住他们所说内容的本质；
——你要写上他们的全名、头衔和公司；
–你将引言分开，这样人们就不会错过(例如，漂亮的边框)；和
——你链接到你为他们设计的网站。

事实上，我的测试表明，散布在一个专业网站上的推荐要比一个写着“人们对我们的评价”并在那里隔离推荐的页面效果更好。

最近的测试表明，包括对你赞不绝口的人的音频甚至视频反馈(在不破坏你的页面下载的情况下)会增加更多的回应！

5.我认识一些非常成功的营销人员，他们设计的网站上有成百上千的推荐(有一些创造性的方法可以快速获得大量的推荐)。这种策略对他们来说有奇效。

6.即使你没有在你的网站上发布推荐(无论什么原因)，你也应该收集它们。当你与潜在客户见面时，他们是展示潜在客户的绝佳营销宣传材料。

7.说了以上所有，感谢提出这个关键问题的个人。最后，你必须测试在你自己的网站上什么可行，什么不可行。所以继续挑战，继续测试！！！！！
