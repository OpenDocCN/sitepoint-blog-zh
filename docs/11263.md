# 设计颜色:紫色

> 原文:[https://www.sitepoint.com/color-in-design-purple/](https://www.sitepoint.com/color-in-design-purple/)

[![image](../Images/3a56e2c454b72add61c53afbf463bf83.png "image")](https://www.sitepoint.com/wp-content/uploads/2010/02/image72.png) 上周我看了喜气洋洋的[色黄色](https://www.sitepoint.com/color-in-design-yellow/)，本周轮到黄色的互补色——紫色了。如果用错了手，它可能看起来有些夸张，但是用得好的话，紫色是充满活力和时尚的。观察色轮，我们知道紫色是暖红色和冷蓝色的交汇点，由中间的许多色调组成。因此，紫色在洋红色和栗色端被认为是暖色，在紫色端被认为是冷色。

紫色是一种浓郁的颜色，与繁荣和财富、皇室和高贵联系在一起。它还具有复活节和紫色长袍的宗教内涵。有趣的是，如果你看看一些基督教和教会的网站，没有多少使用紫色的阴影。这种颜色也通过花卉和植物与自然紧密联系在一起，如丁香、薰衣草、紫罗兰和兰花，许多紫色色调的名称取自自然。2008 年，Pantone 选择模糊虹膜(PANTONE 18-3943)作为其年度颜色。潘通色彩研究所的执行主任莉特丽斯·艾斯曼形容蓝色虹膜带来了

> 蓝色是可靠的一面，紫色是强烈的、深刻的一面。在情感上，它是锚定和冥想，带有一丝魔力。它巧妙地结合了更深的李子、红棕色、黄绿色、葡萄色和灰色。

紫色显然是吉米·亨德里克斯最喜欢的颜色，我们知道王子也喜欢它。

**使用紫色**

如前所述，紫色跨越了温暖和凉爽，可以根据搭配的颜色创造不同的心情。带有一点红色的深紫色会产生一种暖色调。浅紫色是精致的，通常被认为是浪漫的。与粉色相结合，打造出女性化的配色方案。用深紫色搭配黑色和深灰色可以传达出更有男子气概的感觉。朴实自然的配色方案将深紫色与米色或浅紫色与绿色相结合。黄色和紫色是互补色，形成强烈的醒目对比。
