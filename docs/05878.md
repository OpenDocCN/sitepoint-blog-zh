表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏表示比例变化，即上个月有 6.4%的 IE8 用户更换了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

也许最引人注目的观察是使用数字是多么不起眼！Chrome、Firefox、Safari 和 Opera 有小幅增长，而 IE 的跌幅则相当小。总的来说，图表保持相对稳定。在某些方面，这是一种耻辱；很高兴看到应用程序对市场产生了颠覆性的影响。不幸的是，供应商越来越难以应对前五名。新浏览器只能梦想歌剧般的用户数量。

年度统计数据更有趣一些:

*   四分之一的 IE 用户已经迁移到其他地方。
*   火狐已经失去了十分之一的用户群。
*   Chrome 的用户增加了近 20%。

这并不十分准确，因为网络用户的数量逐年上升。然而，这确实表明人们越来越意识到这些选项，并且不那么害怕切换浏览器。

Chrome 可能不会像以前那样每月在桌面上有所增长，但移动市场正在升温…

## 移动浏览器的使用

二月份，手机使用量增长了近 1%，达到所有网络活动的 24.68%。

热门移动浏览应用:

| 移动浏览器 | 一月 | 二月 | 变化 | 亲戚 |
| 机器人 | 27.00% | 26.44% | -0.56% | -2.10% |
| 苹果手机 | 21.38% | 20.88% | -0.50% | -2.30% |
| Opera Mini/Mobile | 14.61% | 13.70% | -0.91% | -6.20% |
| 铬 | 8.59% | 11.95% | +3.36% | +39.10% |
| UC 浏览器 | 11.57% | 11.30% | -0.27% | -2.30% |
| 诺基亚浏览器 | 6.11% | 5.64% | -0.47% | -7.70% |
| 黑莓 | 2.58% | 2.22% | -0.36% | -14.00% |
| 其他人 | 8.16% | 7.87% | -0.29% | -3.60% |

移动版 Chrome 以 3.36%的惊人增长超过了 UC 浏览器。Nexus 等设备现在预装了 Chrome，但我不认为这是增长的唯一原因。除非这是一个统计异常，否则用户似乎已经在他们的手机上发现了 Chrome 而且他们喜欢它。
