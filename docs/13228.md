# 你有经营自己企业的合适性格吗？

> 原文:[https://www . site point . com/do-have-the-right-personality-for-running-your-own-business/](https://www.sitepoint.com/do-have-the-right-personality-for-running-your-own-business/)

我总是遇到一些人，他们投身于自由职业或小企业生活，却没有真正了解自己的技能矩阵。尽管开创自己的事业很有回报，但也有一些负面影响。你是 PHP 专家，或者能像他们中最好的人一样设计，这很好，但是你真的想经营自己的事业吗？

这里有一些经营自己企业的神话，以及我从中发现的现实；

我可以整天做我想做的事情，拒绝其他事情。
当然——如果你的梦想是花大量的时间处理你的会计工作，与客户无休止地开会，规划你的现金流，和债务人打交道，那么我想这个神话是真的。然而，我们其余的人会发现，我们几乎没有时间去做自己想做的事情。

经营自己的事业意味着我可以随心所欲地工作。
在情理之中，这是真的，然而你也会发现，尤其是在早期，当你的客户希望你工作的时候，你就会结束工作，而从来没有真正参加过每周一次的高尔夫比赛。当你意识到计费时间等于收入时，你会比现在工作更多，而不是更少。

我会得到比现在工作更多的报酬。一旦你考虑了税收、退休金、设备、办公室租赁等成本，你会在短期内过得更好吗？不太可能。

一旦我有能力雇用员工，我就可以休息了。
完全不真实。一旦你开始有其他的嘴要养活，你不仅要加倍努力，你还有额外的压力，知道你是通过自己的愿景来支持他们的家庭。如果你拐错了弯，你可能最终不得不解雇某人。实际管理他人，并确保你能支付他们的额外压力，让许多人不愿雇用员工。

如果你和我一样，你也认为你的方式是最好的方式，所以放手让别人来承担工作量是一件很难的事情。

当然，这并不都是厄运和黑暗(我不后悔我选择的道路)——你会很快发现，掌控自己的命运是一件令人兴奋的事情，值得你去吸取教训，战胜挑战。重要的是，不仅要在你所做的事情上有天赋，还要有动力去忽略消极因素，迎接挑战，创造自己的未来。

不要盲目乐观地经营自己的生意，尤其是在短期内，不要不了解其中的陷阱。在迈出这一步之前，尽可能多的学习，找一些在你之前的人做你的导师，最重要的是，对你自己的技能有一个现实的期望。

没有规定说你不能从小做起，先做兼职，然后当你觉得你已经准备好做全职的时候。保持你的目标在望，别忘了享受乐趣！
