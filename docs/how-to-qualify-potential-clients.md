# 摆脱犹豫不决的人:如何确定潜在客户

> 原文：<https://www.sitepoint.com/how-to-qualify-potential-clients/>

建立成功企业的一个关键部分是确定什么样的客户适合你，并能够根据这个基线来衡量潜在客户。虽然你的理想客户标准可能对你来说是独一无二的，但在决定一个潜在客户是否会成为一个好客户时，有一些重要的因素是通用的。

如果你不花必要的时间分析这些因素，你可能会发现自己被拖着走，没有任何承诺，花时间跟进，几乎没有潜力，或者只是在一个既不适合你也不能做出必要的运营和预算决定来完成工作的客户身上浪费时间。

**测量客户的 RID**

RID 代表准备程度、兴趣和决策能力，这些是确定潜在客户的核心因素。在分析潜在客户的这些要素时，你应该注意以下几点:

*   准备:很多时候，客户在过程的早期就开始寻找承包商。虽然这通常是一件好事，因为这表明他们对工作很认真，并找到了合适的人来完成这项工作，但这也可能意味着项目本身还有很长的路要走。这取决于你想花多少时间在计划上，以及你是否觉得这是一个可行的机会。但是感觉一下他们准备好了没有坏处，所以一定要确定项目的时间表，以及他们预计什么时候开始，等等。然后你就可以做出明智的决定。
*   兴趣:客户是在讨价还价，还是他们真的对你提供的东西感兴趣？这通常很容易衡量——也就是说，如果客户主要关心的是你的价格，而你的服务、可用性和体验排在第二位。
*   决策权:与你联系的人是决定将工作授予谁的人吗？如果不是，他们可能只是一个信息收集者，所以你可能要限制你投入的时间。虽然你肯定希望获得信息，以便进入下一阶段的选择过程，但为了你的最大利益，最好安排与决策者进行一对一的会谈，以询问你需要得到答案的问题，从而确定这是否是一个好的选择。

如果潜在客户没有展示所有这些因素，他们可能不会很快成为付费客户，所以现在可能是时候避开了。

**其他标准**

如果潜在客户通过了 RID 测试，那么就开始探究有助于判断潜在客户是否合格的次要因素:

*   讨论他们的需求:确保你了解他们需要做什么，并且你是合适的人选。
*   谈钱:不要等到合同阶段才讨论你的价格。如果你一开始就列出你的价格，而他们马上拒绝了，也许是时候继续前进了。
*   检查一下:对客户和他们的公司做一些自己的调查，以确保没有不可告人的秘密，这不会有什么坏处。谷歌他们，检查更好的商业局，并留意来自其他承包商的反馈。

使用这些因素作为工具来衡量客户的潜力，但在一天结束时，你能做的最好的事情就是跟着感觉走。通常你的第一印象很能说明问题，只要你不忽视任何明显的危险信号，你的直觉可能会把你引向正确的方向。
