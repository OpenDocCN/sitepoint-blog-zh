# 软件盗版破坏了 Windows 7 的发布

> 原文:[https://www.sitepoint.com/windows-7-pirated/](https://www.sitepoint.com/windows-7-pirated/)

本周，Windows 7 的新版本将登陆人们的邮箱，并到达大街上的商店。对微软来说不幸的是，他们最新操作系统的盗版已经在中国上市几个星期了。售价仅为 3 美元，也比标准零售价低得多。

中国是世界第二大个人电脑市场，但估计 80%的软件是盗版的。盗版背后的主要驱动力是价格:即使是 Windows 7 家庭基本版也要花费一个中国学生一个月的收入。微软在一定程度上解决了软件价格问题，并在去年将 Office 2007 降价 70%以上。然而，与盗版软件的成本和广泛的可获得性相比，它仍然是不利的。

有迹象表明执法正在改善——IT 研究公司 Gartner 估计，到 2012 年，盗版率将下降到 50%。这将使中国与大多数亚洲市场处于同一水平，但专家们仍然认识到，整个地区的盗版是一个长期问题。

对于网络传播者来说，有一个显而易见的解决方案。网络应用和云计算可以根除软件盗版。使用微支付或广告支持收入的分销模式在打击盗版方面可能会更成功。不幸的是，绝大多数中国居民，尤其是农村地区的居民无法使用互联网。在高速宽带连接普及之前，软件盗版可能仍是微软和许多其他公司的问题。
