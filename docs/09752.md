# 被锁定:Windows Phone 7 Live 磁贴

> 原文:[https://www . site point . com/get-pinned-windows-phone-7-live-tiles/](https://www.sitepoint.com/get-pinned-windows-phone-7-live-tiles/)

我经常被要求对“为什么选择 Windows Phone 7”发表评论，我花了很长时间才整理出一个令我满意的答案。如果你看看主要的智能手机平台，从硬件和核心操作系统的角度来看，它们都提供非常相似的功能。它们都有集成摄像头、3G 连接、电容屏、WiFi、应用市场等。虽然用于在平台之间构建应用程序的技术之间存在显著差异，但它们都提供非常相似的服务。例如，它们都提供基本的 UI 控件，如按钮、标签、选择控件以及导航系统，并且它们都提供与一些设备功能连接的能力，如相机和联系人。

我将把这个讨论分成两半，因为用户将根据他们的个人经验来决定使用哪个平台。有些用户已经是智能手机用户，并且有使用 iOS 或 Android 的经验。还有一些用户来自我称之为功能手机的领域。这些用户可能拥有也可能不拥有智能手机，但只使用核心平台功能。他们不下载或使用第三方应用程序，传统上只使用设备进行基本的电话功能:短信、电话、联系人，甚至可能还有电子邮件。

### 人们升级到智能手机

先说第二组用户。想象一下，一位顾客走进一家手机店，查看各种可用的手机。他们知道他们想要一部智能手机，能够下载、安装和使用应用程序，并且能够浏览网页。Windows Phone 7 在多家制造商的多种不同硬件设备上提供一致的用户体验。有带键盘的，有不带的；有一些带 5MB 摄像头的；其他配备 4.3 英寸屏幕。解锁设备后，用户会看到开始体验，它显示用户可能感兴趣的当前信息，如即将到来的约会、未读电子邮件的数量。这种体验可以被定制成包括来自用户感兴趣的应用的磁贴。这些标题可以动态更新，以显示与用户相关的信息。整体体验易于上手，保持一致，同时让用户能够选择适合他们工作方式的设备，并对其进行定制以突出显示他们的信息。

### 现有智能手机用户

有经验的智能手机用户更难被说服。他们已经习惯了现有设备提供的功能。他们可能习惯了 iPhone 的简单性；或者他们可能能够处理 Android 的复杂性。无论哪种方式，他们都希望手机平台具备基本的智能手机功能。我认为 Windows Phone 7 区别于其他智能手机平台的一点是开始屏幕上的实时磁贴。

### Windows Phone 7 实时互动程序

Live Tiles 有点用词不当，因为没有第三方代码在后台运行。仅仅因为用户把你的应用程序钉在开始屏幕上，并不意味着你的应用程序在后台运行。但是，您的应用程序的图块可以通过两种不同的方式进行更新。首先，您的应用程序可以安排定期更新图块。例如，您可能希望每小时更新一次图块，以显示反映应用程序状态的不同图像。第二种方法是通过推送通知。可以向该设备发送推送通知，这将导致该区块被更新。

此外，live tile 有三个方面可以更新:背景图像、计数徽章和标题文本。您通常会使用计数徽章来查看应用程序中未读或新项目的数量。大多数应用程序将利用背景图像来反映应用程序的状态。因为这个图像可以从服务器上检索，所以可以动态地创建这个图像来包含图像和文本内容，这样您就可以完全自由地向用户显示您想要显示的任何信息。

### 被钉住

在 2011 年 4 月在拉斯维加斯举行的 MIX11 上，微软宣布下一版本的 Windows Phone 将允许第三方应用程序将多个磁贴钉在主屏幕上。这是 Windows Phone 7 的一个独特功能，凸显了移动设备上工作效率的新潜力。以展示的 Qantas 应用程序为例。Qantas 应用程序列出了用户已经预订的航班。新的锁定功能意味着用户可以选择一个或多个航班，并将它们锁定在开始屏幕上。当这些航班的状态改变时，或者当航班临近时，可以在标题上显示关于该航班的不同信息。这可能包括航班信息、航班登记是否开放以及用户何时可能错过航班。经常乘坐飞机的人发现这些信息对于安排他们繁忙的生活是非常宝贵的。

正如你所看到的，有一些令人信服的理由让新老智能手机用户选择 Windows Phone 7。应用程序市场正在迅速扩大，与办公室和个人生活的整合带来了简单、高效的体验。
