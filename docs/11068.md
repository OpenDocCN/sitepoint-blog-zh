区别在于 Chrome 实际上有自动升级功能。当一个新版本出来时，它不会询问用户，而是升级并应用它，而用户对此一无所知。所以第二天就会出现一个新版本。现在我肯定有些人不太喜欢这样，但这就是 Chrome 的做法。

IE 在光谱的另一端，他们真的不唠叨用户太多关于升级和它弹出作为一个关键的更新，但如果你不这样做，它只会坐在那里，并不断要求你这样做，每次你运行 Windows 更新。

火狐采取了不同的方法。现在他们实际上强制自动进行小的更新。例如，从 3.6.1 到 3.6.2 会自动发生。你不会知道它发生了，它只会升级。然而，对于主要版本，他们不会把这些强加给用户。所以用户必须接受升级，并允许 Firefox 来完成升级。

但是真的很有意思。我正在看这些图表，很明显，由于 Chrome 强制升级，你可以看到在大约三个月内，Chrome 3.0 的市场份额为 0%，而 4.0 的市场份额达到了 7%或 8%，而 IE 的蓝色 IE 6 系列仍然下降到 10%左右，IE 8 的市场份额上升了 25%左右。所以看到每个浏览器采用不同的方法肯定是很有趣的。
