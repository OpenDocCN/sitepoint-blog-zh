# 与 Spreadable 共享的新方式

> 原文:[https://www.sitepoint.com/spreadable-sharing-promotion/](https://www.sitepoint.com/spreadable-sharing-promotion/)

[Spreadable](http://spreadable.com) 是一个新的(测试版)工具，您可以定制并集成到您的网站中，使您的访问者更容易分享您的内容并向您发送推荐。Spreadable 的有用功能包括易于设置的小工具、定制和实时分析。

## 简单设置

每个 Spreadable widget 都有两个主要组件:一个覆盖图(单击 Spreadable 按钮时显示)和一个独立的推荐页面。你可以自己建立网页，并把它放在你自己的域名上，或者他们会把它放在 spreadable.com 的一个子域上。

## ![](../Images/946b1381d049aa3f28e5596ff111e555.png "spreadable")

## 用户化

该服务具有广泛的定制功能。你可以定义你的配色方案，添加你的标志，并使用自定义信息。您甚至可以为每个社交频道定制消息，并选择显示哪些社交网站。

![](../Images/2982e06eeb90e2ae8c3026463628eeec.png "spreadable-example")

## 分析学

该服务包括分析，所以你可以跟踪结果，并根据生成的统计数据改进您的网站。您可以根据地理位置跟踪和分析您的推荐，它们来自您网站的哪个位置(哪个页面)，最常用的共享渠道和社交网站，以及您的主要个人推荐人。

所有的跟踪都是实时进行的，所以你总是能得到关于你的推荐项目执行情况的可测量的结果。

![](../Images/ee304c55a76113bac08cdc1bb35ec89b.png "spreadable-analytics-1")

这项服务目前处于测试阶段，但它有潜力提供许多网站所有者可以轻松集成的东西，以了解他们网站访问者的更多信息，推广他们的服务并改进他们的网站。
