## 4.垂直巨型菜单

这个插件从标准的 HTML 嵌套无序列表中创建垂直的巨型菜单。
大菜单可定制多种功能:
>选择显示/隐藏、淡入或滑出动画效果
>设置动画速度
>设置每行子菜单数量
>根据菜单位置在左侧或右侧显示弹出大菜单

 [![Vertical Mega Menu](../Images/97be3b7ecf7e1a65ed2950a4db5ca786.png)](http://www.designchemical.com/lab/jquery-vertical-mega-menu-plugin/getting-started/) 
[来源](http://www.designchemical.com/lab/jquery-vertical-mega-menu-plugin/getting-started/)
[演示](http://www.designchemical.com/lab/jquery-vertical-mega-menu-plugin/examples/)

## 6.埃利克森剧院

是一个可扩展的类似旋转木马/幻灯片的 JQuery 插件，支持多种动画效果。处理图像或任何其他类型的丰富格式的 HTML 内容。
支持的特效:
> 3d
> 3d 轨道
> css3:圆形
> css3:幻灯片
> css3:堆栈
>水平
>垂直
>淡入淡出
>显示
>幻灯片

 [![Elixon Theatre](../Images/f3ce5e0623a61f4ef723b55ea28d6b5c.png)](http://www.webdevelopers.eu/shop/5/doc/index) 
[来源](http://www.webdevelopers.eu/shop/5/doc/index)
[演示](http://www.webdevelopers.eu/shop/5/demo)
