# Dollar ReDe$ign 项目

> 原文:[https://www.sitepoint.com/dollar-redeign-project/](https://www.sitepoint.com/dollar-redeign-project/)

[Dollar ReDe$ign 项目](http://richardsmith.posterous.com/)是一个由纽约设计师理查德·史密斯组织的公开提交项目，旨在重塑美元品牌。该项目背后的理念是“快速经济复苏的‘唯一’现实途径是通过一个彻底、深入的重塑品牌计划——从重新设计标志性的美元开始。”来自世界各地的设计师响应了这一号召，就像许多设计比赛一样，质量从荒谬到卓越不等。

通过查看一些条目，这是一个再见绿背，你好紫背，黄背和红背的例子。以下是 2010 年提交的部分作品。

法比亚诺·皮内尔的《音乐人》。“每一美元钞票代表一种真正的美国音乐流派，如布鲁斯、爵士乐、灵魂乐、乡村音乐、说唱音乐和摇滚乐。”
