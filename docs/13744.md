# 指导的重要性

> 原文:[https://www.sitepoint.com/the-importance-of-mentoring/](https://www.sitepoint.com/the-importance-of-mentoring/)

指导他人并找到导师是明智的商业策略。

通过指导他人，随着你的“学员”发展并承担更多责任，你获得了潜在客户的来源。因为学员通常更年轻(尽管不总是像你马上会读到的那样)，你让自己跟上新趋势、新问题，有时甚至是新兴技术。你还可以保持思维敏捷，因为一个好的学员可以用很棒的问题和创造力来挑战你。

通过得到一个导师，你让自己接触到智慧和新的机会。我在经济上的成功很大程度上归功于导师，我在每一种商业关系中都寻找导师。从任何人身上都能学到很多东西。与此同时，有些人非常友好地把我置于他们的羽翼之下。一位是投资银行家，他教会了我很多真正的企业家是如何思考的。最近，一位年龄只有我一半(差不多)的 Sitepoint 读者同意教我一个他成功运用的商业模式。在前一种情况下，投资银行家得到了我的想法和“汗水资本”——以及一些无偿劳动——作为他承诺的交换。在后一种情况下，我们达成了一项交易，如果我成功了，它将帮助每个人(我希望)赚很多钱。

当我独自做事，试图重新发明轮子时，我往往会跌倒。导师帮助我找到最有效的成功之路。

你有导师吗？你在接受学员吗？你如何找到你的导师/学员？你发现了哪些好处？
