# 欧盟对老大哥英国采取法律行动

> 原文:[https://www.sitepoint.com/big-brother-britain/](https://www.sitepoint.com/big-brother-britain/)

由于英国政府未能保护英国互联网用户的隐私，欧盟委员会开始对其采取法律行动。该案是在 Phorm 审判后引发的；一种分析个人互联网流量和内容的系统，用于定向广告。英国电信巨头英国电信公司承认在 2006 年和 2007 年期间，未经用户同意，秘密测试 Phorm。

正如我们在 3 月报道的，蒂姆·伯纳斯·李爵士向英国政府成员讲述了他对 Phorm 和其他深度数据包检测系统引入后的互联网隐私的担忧。欧盟信息社会和媒体专员维维亚娜·雷丁宣布了行动的第一阶段:

> 互联网行为广告等技术可能对企业和消费者有用，但它们的使用方式必须符合欧盟规则。
> 
> 一个人的信息只能在他们事先同意的情况下使用。我们不能放弃这一基本原则，让我们所有的交流都受到监控、调查和存储，以换取“更相关”广告的承诺。当一个欧盟国家没有履行这一义务时，我不会回避采取行动。

欧盟委员会已经收到了来自英国互联网用户的数百份关于 Phorm 试验的投诉。在与英国政府讨论后，委员会担心个人数据没有得到充分保护。此外，英国执行欧盟电子通信保密规则的方式存在问题:

*   虽然联合王国法律将非法拦截通信定为犯罪，但这一犯罪的范围仅限于"故意"拦截。
*   根据同一法律，当截取者有“合理的理由相信”已同意截取时，截取被视为合法。

欧盟委员会给了英国政府两个月的时间来回应他们的侵权诉讼。最终，该案件可能会因涉嫌违反欧盟指令而被提交至欧洲法院。

此外，亚马逊和维基百科最近都宣布退出 Phorm 有争议的定向广告技术。
