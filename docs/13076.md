# 。网上网 3 月 2 日至 9 日:混合宿醉版

> 原文:[https://www . sitepoint . com/net-on-the-net-March-2-9-mix-hangover-edition/](https://www.sitepoint.com/net-on-the-net-march-2-9-mix-hangover-edition/)

万一你错过了，mix 08 T1 来了又走了。我贴了一些感想[这里](https://www.sitepoint.com/mixing-it-up-day-one/)和[这里](https://www.sitepoint.com/mixing-it-up-day-two/)。如果有机会，一定要看看[网络广播](http://sessions.visitmix.com/) [NB:先安装 Silverlight 2]。展会上有很多很棒的东西发布到网络上——Matthew Podwyoski 列出了一个很好的清单，上面有 ASP.NET 特有的东西。围绕 IE8 Beta 1 的发布，相关团队向我们介绍了 [IE8 和 IP 许可](http://blogs.msdn.com/ie/archive/2008/03/06/ie8-and-ip-licensing.aspx)、[活动和 Web 切片](http://blogs.msdn.com/ie/archive/2008/03/06/activities-and-webslices-in-internet-explorer-8.aspx)、 [IE8 和 JScript](http://blogs.msdn.com/jscript/archive/2008/03/05/jscript-in-internet-explorer-8-beta-1-for-developers.aspx) 、 [CSS 2.1 测试](http://blogs.msdn.com/ie/archive/2008/03/06/ie8-and-css-2-1-testing.aspx)，以及 [IE8 开发者工具](http://blogs.msdn.com/ie/archive/2008/03/07/improved-productivity-through-internet-explorer-8-developer-tools.aspx)。稍微平凡一点，但在我的生活中更重要的是来自 IIS 团队的发布洪流。亮点包括[用于 XP、2003 和 Vista](http://blogs.iis.net/rlucero/archive/2008/03/04/internet-information-services-iis-7-0-manager-is-now-available.aspx) 的 IIS7 管理工具、[配置扩展性](http://blogs.iis.net/bdela/archive/2008/03/05/iis-7-configuration-basic-configuration-extensibility.aspx) [x2](http://blogs.iis.net/bdela/archive/2008/03/10/iis-7-configuration-extending-existing-configuration.aspx) 以及一篇关于[面向开发人员的 IIS7 诊断工具](http://blogs.iis.net/mailant/archive/2008/03/07/iis7-diagnostics-for-developers-talk-at-mix08.aspx)的非常强大的文章。

鉴于 ASP.NET 世界的相当大一部分人正忙着在维加斯闲谈和饮酒，网络圈里没有太多活动。最近,《LINQ》和其他花哨、俗气的东西风靡一时，但《Reamped.NET》让我们想起了我们的老朋友:IClonable 和 IComparable。如果你不知道如何复制和排序，你就不能在编程方面走得太远。Matt Berseth 告诉我们如何建立[一个非常光滑的文件夹浏览器](http://mattberseth.com/blog/2008/03/building_a_vista_style_folder.html)。对于版本来说，[的人 AJAX 框架](http://www.codeplex.com/folks)达到了 0.10 的里程碑，而 JetBrains 的人正在发布 [Resharper 夜间版本](http://www.jetbrains.net/confluence/display/ReSharper/Download)。

关于 DC 地区的事件，如果我能在合理的时间逃离办公室，我计划明天晚上参加 [RockNUG 会议](http://rocknug.org/)。[乔纳森·科格利](http://weblogs.asp.net/jcogley/archive/2008/03/07/refactoring-in-c-at-rocknug-this-week.aspx)通常是一个好节目，和当地的人聊聊总是很好的。还有，[新星密码营南](http://novacodecamp.org/)29 号就要到了。

最后，我将把这个每周的帖子移到本周早些时候。我发誓这个决定和锦标赛篮球的出现没有任何关系。
