# 从美国公司吸取的教训

> 原文：<https://www.sitepoint.com/lessons-learned-from-corporate-america/>

在屈服于稳定且难以抗拒的创业诱惑之前，我在“现实世界”中从事一份典型的朝九晚五的工作，实际上是几份不同的朝九晚五的工作。这些工作我都不喜欢，但毫无疑问，每一份都给我上了宝贵的一课，这一课我至今仍在我的工作中应用。

在我职业生涯的最初几年里，我很感激学到了一些经验。

## 一致性是有回报的

在公司环境中，每个人都想出人头地，争夺老板的注意，所以他们都跑到了下一次晋升、加薪或奖金的最前面。如果你在一家大公司工作，很难被人注意到，也很难展示你的价值。有时候，你必须自己创造机会，要求新的工作，提出改进建议，主动做比要求你做的更多的事情……然后每次都做得很出色。有时候，取得成功仅仅是展示你坚定的奉献精神，即使你感到无聊和没有挑战。

以我的经验来看，你的工作质量越稳定，你在你的经理面前就越可靠，你的责任也就越大。这也适用于商业。为了成为客户依赖的有价值的资源，你需要产生一致的结果，是可靠的，并且有一个你遵循的常规沟通过程。

## 尊重是双向的

对于那些在公司中晋升的人来说，一个常见的挑战是赢得同事的尊重。当你刚开始一份工作时，尤其是刚从学校毕业时，你是一个底层的人，有时你需要走很长的路才能获得尊重并感觉自己是团队中有价值的一员。

我在企业界的经验告诉我，赢得尊重的最有效方式是在适当的时候给予尊重。如果你不尊重他人，不认真倾听，不尊重他人的贡献，你需要做很多工作来赢得尊重。

这反映在各种商业关系中。如果你对你的客户、同事，尤其是你的竞争对手缺乏尊重，你就不可能成功。表现出尊重不仅能帮你赢得尊重，还能打开新联盟和合作的大门。

## 团队合作中没有“我”

我想在我面试的每一份工作中，面试官都会问我如何融入团队环境。即使想要的职位并不是团队的一部分，这个职位总是涉及到以某种身份与他人合作，所以每个潜在的雇主都想知道我与他人合作得如何是有道理的。

在自主创业的世界里，你也需要作为团队的一员有效地运作，即使你是一个人的商店。无论你是与客户合作，与其他服务提供商合作，还是行业组织的一部分，你都需要能够在团队环境中有效地与他人合作。

## 专业很重要

如此多的组织已经走上了休闲工作环境的道路，从宽松的着装，到不那么结构化的工作时间，再到非正式的交流方式。在我过去的生活中，我看到了许多这种随意完全错误的情况。当然，我完全赞成穿牛仔裤，通过即时消息聊天，没有严格的工作时间，但应该有一个压倒一切的专业标准来衡量你自己。轻松的工作环境不应该解释为缺乏专业精神。例如，在海滩呆了一段时间后去上班与管理层开会，穿着你在日光浴时穿的衣服可能不是你能为你的职业生涯做的最好的事情(真实故事)。

当你是个体经营者时，专业精神同样重要。如果你一个人在家工作，很容易陷入完全放松的陷阱，并且在表现自己是专业人士时变得懒散。我已经认识到保持我的业务专业性是多么重要，从电子邮件通信，到业务管理流程，到我的客户政策，即使我不“必须”这样做。

你学到了什么？你是从工作中学到宝贵经验的前雇员吗？
