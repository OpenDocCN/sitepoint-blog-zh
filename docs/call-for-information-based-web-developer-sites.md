我想开始收集尽可能多的网页设计师/网页开发者的例子，他们在自己的网站上包含以下元素:

–在主页的前三分之一位置展示证明材料(可立即建立可信度)；

–免费文章、白皮书、成功和失败网站的案例研究，以及其他教育和告知受众的资源(这会引起他们的兴趣，并将您定位为专家)；

–标题和副标题能吸引访问者的注意力，并保持他们的兴趣(例如，它们能引起好奇心，或显示真正的好处)；

–竞争对手无法比拟的可靠保证(将潜在客户的风险转移开)；

–谈论潜在客户的内容是谈论网页设计师/开发人员的两倍(吸引潜在客户)；

–吸引访问者留下联系信息的方法，以便网站开发人员可以随时跟进，继续建立信誉；和

–专注于特定的目标市场，利用吸引该市场独特问题和语言的强大营销信息。

你不需要具备上述所有要素，但我确实希望在 Sitepoint 上收集一些网络商务网站，这些网站不仅仅是通常所说的“我们是谁，我们的产品组合，我们提供的服务等等”

所以，如果你的公司或任何你认识并尊重的网站开发/设计公司符合上述任何标准，请在这个博客上发布他们的链接，以便其他人可以从中学习。在可能的情况下，包含你通过包含这些元素而得到的结果。

这样，这个博客的读者就可以开始把他们的网站与通常的普通内容区分开来。
