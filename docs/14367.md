# 来自真实设备的界面灵感

> 原文：<https://www.sitepoint.com/interface-inspiration-from-real-world-devices/>

我倾向于为 Fireworks 和 Flash 扩展设计许多界面，有些可以很简单，有些更复杂，有许多用户能够控制的不同参数。我为 SitePoint 博客编写的大多数 Flash 命令面板都有简单的控件(如刻度盘和滑块),用户可以联系起来，因为他们已经牢牢地扎根于现实世界中，并且是根据我日常使用的产品经验中的控制器(旋钮、刻度盘和滑块)设计的

当设计一个用户将与之交互的界面时，给用户基本的输入框来输入他们的数据是很容易的，但是我发现当使用“敏感的”控件时，体验会得到增强。毕竟，从打开电视到使用烤面包机，我们在家里控制着大量的设备，所以如果你正在寻找灵感，那么通常没有必要走出前门，它可能就在你的面前！

以下面的例子为例，一个免费的 Fireworks MX 命令的接口，它是我在近两年前编写的 Fireworks MX / MX 2004 附带的现有版本的大修；所有控制都以简单整洁的方式布局，主滑块在顶部，效果调节器在底部(转盘)。一切都唾手可得，并且易于使用，根据用户输入在预览区域给出反馈。简单但有效
