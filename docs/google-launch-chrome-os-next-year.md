# 谷歌推出 Chrome 操作系统(明年)

> 原文：<https://www.sitepoint.com/google-launch-chrome-os-next-year/>

你一定会爱上谷歌的营销机器。尽管关于 Chrome OS 没什么新消息，但媒体已经迫不及待地报道了本周的*【发布】*。如果你没有读到一位兴奋的记者写的东西，你就无法打开报纸，这位记者显然对操作系统和浏览器之间的区别感到困惑。如果你像我一样，希望下载一个完全可启动的 ISO，你会失望的。此次发布包括:

1.  名称从“Chrome OS”更改为“Chrome OS”。
2.  一个新的[Chromium.org](https://www.chromium.org/chromium-os)网站，为用户界面设计者和开发者提供信息。
3.  源代码的早期版本。

谷歌表示，这款操作系统距离发布还有整整 12 个月的时间。源代码不是测试版，甚至不是预发布版。有可能[创建一个版本](https://www.chromium.org/chromium-os/building-chromium-os),但是很少有人会去费心:你需要 Linux、各种开发人员包和很大的耐心。截屏和视频是可用的，但它们大多是概念性的线条画，而不是真实世界使用的实际镜头。这个视频是信息更丰富的视频之一:

那么我们有没有发现在之前[我们不知道的事情呢？](https://www.sitepoint.com/google-chrome-operating-system/)

*   这是一个纯粹的基于浏览器的操作系统。它本质上是带有额外弹出管理的谷歌 Chrome 浏览器。即使是简单的应用程序，如计算器，似乎也是基于网络的，而不是可执行的操作系统。
*   最左边的标签显示了 Chrome 菜单，链接到 GMail 或 panels 等标准应用程序。
*   可以打开覆盖主窗口的小面板，或者可以停靠在屏幕的一侧。

操作系统基本上符合我的预期，但我真的希望今天启动一个虚拟机。谷歌已经发布了代码，并要求开发者做出贡献，这很好，但激起媒体的狂热兴奋并不是最好的举动。我不禁感到有些索然无味。你构建或尝试过 Chromium OS 吗？
