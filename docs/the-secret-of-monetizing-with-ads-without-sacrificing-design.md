# 用广告赚钱而不牺牲设计的秘密

> 原文：<https://www.sitepoint.com/the-secret-of-monetizing-with-ads-without-sacrificing-design/>

我在工作中面临的一个主要问题是，找到一种不削弱设计或整体信息的方法，用广告赚钱，我怀疑这是许多设计师都在处理的问题。我应该先说，我对广告有一种不科学但坚定的厌恶。当我看到一个(我认为的)广告过多的网站时，我会有一种轻微的恶心感，而且我很可能不会停留很长时间。

当然我理解很多网站的目的是创收。我知道许多网站必须产生收入来支持自己。事实上，我正在为自己的企业开发这样一个网站。但这并没有让我更容易下咽。顺便说一下，在我看来，博客的货币化是一种非常不同的动物，所以我主要谈论的是传统网站。

在努力帮助客户找到一种不像广告牌一样赚钱的方法时，我发现秘诀就是提前考虑，从一开始就考虑大局。这里有一些指导你这样做的原则。

广告必须是相关的。

如果我在网站上看到广告，我希望它们与目标受众相关。如果你想发展忠实的粉丝，那么为了做广告而做广告对我来说毫无意义。即使你有一个基于印象的活动，你的目标应该是为你的访问者提供附加值，这意味着给他们更多他们想要的东西。最终，你希望你的观众点击你的广告。如果你在宣传他们不关心的东西，那就不会发生。

非文字广告应该设计得很好。

我们都见过一个漂亮的网站，上面有难看的图形广告。我知道很多时候这是我们无法控制的，但如果保持一个漂亮的设计是最重要的，你或你的客户可能需要监控网站上显示的广告，并阻止那些不符合标准的广告。或者，如果你的广告被批准并在内部投放，主动帮助广告商制作广告，或者作为他们赞助的一部分免费，或者收费。
 **让你的广告成为设计的一部分。**

如果你在开始设计之前就知道网站的盈利类型和水平，你可以通过在实际的广告设计中加入位置来创建一个支持广告的设计。这很明显是有预谋的如何将网站货币化，可以使整体流量更有效。

**广告应该前后一致。** 
像网站导航一样，广告需要出现在每一页的同一个地方。这看起来很简单，但是让广告在整个网站上跳来跳去是一个很大的缺点。另外，广告的格式和尺寸应该一致。

广告应该是次要的。

即使在网站所有者的心目中，广告是网站最重要的元素，但在其他人看来，广告的重要性应该是次要的。精心编写和有效的内容需要是网站的第一和主要部分，因为这是人们访问的目的，而不是广告。

这个公式对我来说很有效，但是我很想听听你的意见，以及你如何设计来支持货币化。
