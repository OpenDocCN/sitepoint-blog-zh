# 专用托管比较:10 个专用托管计划述评

> 原文:[https://www . site point . com/dedicated-hosting-comparison-10-plans-reviewed/](https://www.sitepoint.com/dedicated-hosting-comparison-10-plans-reviewed/)

对于我们大多数人来说，共享虚拟主机解决方案可能是我们在线创业之初最合适的选择。为什么？仅仅是因为它是 VPS 或专用托管解决方案的廉价替代品。

然而，这一规则也可能有例外，例如当一个项目预计将经历指数级增长，并且一开始就选择了专用的托管服务。即使你最初选择了共享主机服务，你也可能会意识到共享主机已经不能满足你网站的高要求了。如果你的项目现在有稳定的收入流，迁移到一个专用的托管平台的成本可能会比开始时显得不那么吓人。

我对十家提供入门级专用托管计划的大公司进行了研究。其中包括 [1 & 1](http://order.1and1.com/xml/order/ServerPremium?ac=OM.US.USa09K18193T7073a) 、 [HostDime](http://www.hostdime.com) 、 [HostGator](http://www.hostgator.com) 、 [Hostway](http://www.hostway.com.au) 、 [InMotion](http://www.inmotionhosting.com) 、 [LiquidWeb](http://www.liquidweb.com) 、 [LunarPages](http://www.lunarpages.com) 、 [NetHosting](http://www.nethosting.com) 、 [ResellersPanel](http://www.resellerspanel.com) 和 [ServePath](http://www.servepath.com) 。我们根据专用主机的所有最重要的标准对这些公司进行了深入的比较，包括处理器速度、硬盘容量、RAID 选项、RAM、每月带宽、IP 地址和许多其他因素。除此之外，我还考虑了在不升级的情况下扩展计划的可能性，这使它更加灵活，因为它使您不必迁移到更高级别的计划。

所有经过审查的计划都包括完全根访问和硬件更换保证，以及 99%或更高的正常运行时间保证。此外，SSH 访问对所有计划都是通用的，大多数计划的上行速度是 100Mbps，ResellersPanel、InMotion 和 HostDime 除外。

在平等的基础上比较所有这些公司的主要障碍是，在几乎所有情况下，一个或多个功能没有在计划描述页面上指定。因此，我通过各种可用的联系方式联系了这些公司，以便找出缺失的功能。我与他们中的大多数人进行了实时聊天，对于那些不提供实时聊天的人，我使用电子邮件或电话。

最后，经过许多天的研究，我能够收集所有的数据，我已经在一个全面的专用主机对照表中展示了这些数据。我不得不询问托管公司的未指定功能在表中以斜体显示。我还对这个表格的内容进行了分析，指出了每个专用托管计划的优缺点。

现在，让我们来详细看看每个托管计划。

## 1 & 1

在硬盘容量和提供的 FTP 备份方面，双核 XL 计划比其他专用主机计划更有优势。1 & 1 提供两个 400GB 硬盘和 200GB FTP 备份，这比本次比较中的任何其他 web 主机都多。它还提供了附带的 RAID 选项，使频繁备份变得轻而易举。

就包含的 RAM 数量而言，1&1 提供了 4GB，这与 HostGator 和 ServePath 一起，是所有受审查主机中最多的。1&1 还提供了附带的软件防火墙和 SSL 证书，这对电子商务网站来说可能真的很有价值。在计划描述页面上没有提到正常运行时间保证。然而，我们与主机的实时聊天显示，t1&1 提供了 99.99%的正常运行时间保证，在所有审查的主机中排名第二。

至于每月带宽，1&1 提供了慷慨的 3000GB，并成为 LiquidWeb 的亚军，liquid web 提供了最高的 6000GB 带宽。然而，在许多特性上，1&1 似乎处于劣势，例如处理器规格、硬盘升级选项、RAM 升级选项以及提供的 IP 地址数量。

该计划每月费用为 199.99 美元，与 ServePath 持平，略高于我们比较中的最低月价格(LiquidWeb 为 189 美元)。1&1 计划的年度注册总折扣为 100 美元。

## HostDime

HostDime 在其 [QC1](http://www.hostdime.com/services/dedicated/quadcore_qc1.php) 计划中提供了比 1&1 更好的处理器规格，并且还提供了升级处理器、硬盘和 RAM 的选项。另一方面，附带的 160GB 硬盘是我们比较中价格较低的产品之一。但是，在此计划中，RAID 配置有多个选项可用。

虽然附带的 1GB RAM 是比较的计划中最低的，但它可以升级到最大 4GB。HostDime 每月提供 2000GB 的带宽配额，在所有受评主机中排名第三，唯一较好的选择是 1&1 和 LiquidWeb，分别提供 3000GB 和 6000GB。该计划仅包括四个 IP 地址，但是可以以每月 2 美元的价格获得额外的 IP 地址，可以购买的 IP 地址数量没有上限。

该计划提供了多种操作系统，包括 Fedora Core 和 Windows 2003 server。HostDime 允许您选择 CPanel 或 Plesk 控制面板，每月费用在 25 美元到 50 美元之间。该计划还提供可选的托管服务以及 FTP 备份解决方案。该软件包捆绑了一个免费的 CSF 软件防火墙，而每月 100 美元可以获得一个 Juniper NetScreen 5GT 硬件防火墙。SSL 证书也是可选的，可以以 50 美元的年费购买。

服务器由 HostDime 监控，该计划没有安装费。该计划的月价格为 220 美元，是 LiquidWeb、1&1、ServePath 和 LunarPages 之后的第五低价格。HostDime 为第一台服务器提供每月 20 美元的折扣，相当于每年 240 美元的总折扣。HostDime 不提供退款保证。

## 宿主

HostGator 在其[标准](http://www.hostgator.com/dedicated.shtml)计划中提供了英特尔双核至强处理器，这可能是服务器平台的最佳处理器。也就是说，该计划没有处理器升级选项。HostGator 提供的双 250GB 硬盘使其在所有受调查主机中的存储容量排名第三。它配有 RAID 配置，还允许您将硬盘升级到更高的容量。

像 1&1 一样，HostGator 包含 4GB 的 RAM。RAM 也可以升级，尽管没有提到可以购买的最大内存量。一个月内总共允许 1500GB 的数据传输，这是所提供的最低配额。该计划有 10 个 IP 地址，是 LiquidWeb 计划的亚军，liquid web 计划提供 16 个 IP 地址。可以从 HostGator 购买额外的 IP 地址，尽管没有提到每个地址的价格。

该计划提供三种操作系统，包括 Red Hat Linux 4。仅有的另外两个提供 Red Hat Linux 的 web 主机是 Hostway 和 ServePath，所以如果这两个是您选择的操作系统之一，它可能是选择 HostGator 的一个好理由。CPanel 是唯一可用的控制面板，然而好消息是它没有额外的成本。FTP 备份可以通过 WHM 执行。该计划还包括软件防火墙，但 SSL 证书是可选的。没有安装费，也没有退款保证。

该计划的价格为每月 219 美元，这使得它正好处于审查计划的中间。HostGator 目前提供 20%的折扣，可以使用优惠券代码“雪人”申请，这使得该计划成为所有计划中最具成本效益的。如[表](http://sitepoint-examples.s3.amazonaws.com/dedicatedhosting/Dedicated_Hosting_Comparison.html)所示，该计划每年节省的资金总额相当于 525 美元。

## Hostway

Hostway 在其 [TFS 800](http://www.hostwayenterprise.com/dedicated-servers/all-dedicated-servers.html) 计划中提供了两个双核英特尔至强处理器。与其他被审查的主机相比，没有更好的处理器规格可用。因此，这个计划对于希望部署耗电应用程序的用户来说似乎是理想的。就硬盘容量而言，Hostway 的计划提供两个 160GB 的硬盘，从而与 InMotion 共享第三大存储容量。Hostway 还提供灵活的硬盘升级选项，最大容量为 2TB，每月 70 美元。RAID 1 也包含在该计划中。

提供的 RAM 总量为 2GB，尽管有一个选项可以将 RAM 升级到最大 16GB，每月费用为 140 美元。至于包含的 RAM，Hostway 的计划是 1&1、HostGator 和 ServePath 的亚军，它们的计划中包含 4GB 的 RAM。Hostway 在每月数据传输比较中排名第三，提供 2000GB。就包含的 IP 地址总数而言，Hostway 以其提供的 5 个地址再次排名第三，仅次于 LiquidWeb 和 HostGator，它们分别提供 16 个和 10 个 IP 地址。每月 2 美元可以获得额外的 IP 地址，但是你需要提供一个技术上的理由来获得它们。

Hostway 还提供多种操作系统，包括 Windows 和 Red Hat Linux。它提供了一个被称为“站点控制”的自定义控制面板，尽管有一个额外费用的十域许可使用 Plesk 的选项。计划中包括最大 50GB 的 FTP 备份，以及软件防火墙。然而，SSL 证书是可选的，可以以每年 49.95 美元的价格获得。

这项计划不收取启动费。Hostway 提供每月 219 美元的 TFS 800 计划，从而与 HostDime、HostGator、InMotion 和 ResellersPanel 并列第五。它还提供 15%的折扣或一年预付两个月的免费托管，这相当于为这台主机节省了 438 美元——这是本评论中最高的折扣之一。Hostway 也因提供 30 天退款保证而脱颖而出。
