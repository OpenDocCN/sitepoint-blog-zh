# 建立点击网站的 10 个步骤

> 原文：<https://www.sitepoint.com/building-click-ready-website/>

没有理由不建立一个点击就绪的网站！这里有一个简单的 10 步检查表，确保你不会忽略任何东西。

**1)制定一个与你的整体营销流程相结合的网站开发计划**

内容要和线下素材一致。图形/图像不必与传统的
媒体完全相同，但应该与您的整体品牌、风格指南、颜色使用等保持一致。

**2)雇佣一家了解你的市场定位的网站设计公司，一家不会让“极客疯狂”的公司**

这意味着他们太喜欢自己的设计能力了，你的网站陷入了图形、插件、GIF 垃圾、
等等的泥潭。但是，反过来，当你与你的设计公司合作时，检查一下你的自我——我见过很多好的网站设计被客户毁掉，他们不能或不愿意听我们告诉他们的！

**3)注意“加载次数”**

用 56 KBPS 的调制解调器加载你的网站需要多长时间？如果超过 25 秒，你可能会经历“死亡之击”

–网站加载速度不快，冲浪者也不见了。当然，如果你的目标是通过 ISDN 或 DSL 访问你的网站的宽带客户，那么你可以建立一个包含多媒体内容的网站，这些内容可能包括流式音频或视频，或者 Shockwave 或 Flash 功能——继续前进，让那些数字极客被尖端内容冲昏头脑吧！

我敢说，“保持简单”

让你的网站易于移动，建立一个符合行业标准的菜单结构，本地菜单(对于一个
页面或部分)在左边，全局菜单(整个网站导航)在每个页面的顶部和/或底部，尽可能多的保留
信息在“折叠线之上”(在显示器底部的截止点之上)，除非绝对必要，不要让人们使用水平滚动条
。

将“数字速度”灌输到你的整体网站设计中

您的客户/顾客应该能够在一两次鼠标点击之内到达他们想要的网站区域；如果他们不得不点击多个菜单来找到他们想要的信息，他们会很快感到沮丧。

**6)开发网络内容**

人们不会像阅读线下媒体那样阅读网站内容。保持你的段落简短，不超过两到三句话，在空白处写下你的内容，在你的网页中包含链接——不要试图在你的网站上讲述你的整个营销故事

–让人们给你打电话(你好，电话还能用！)，发送电子邮件或填写个人资料表格(见下文)。

**7)让你的网站做好基于许可的营销准备**

我喜欢塞思·戈丁的《http://www.permission.com/许可营销》一书，我们向我们所有的客户推荐这本书——他主张通过请求客户许可继续向他们营销，并将价值/信息融入所有的营销/沟通过程，从而与客户建立长期关系。

确保你的网站为搜索引擎优化

确定 8-12 个人们会用来找到你的网站的关键词，然后将这些关键词整合到你的网站内容中(以提高与 s/engine 蜘蛛/机器人的相关性),然后手动将你的网站提交给前十大搜索引擎。我们一般不推荐大多数免费或 19.99 美元的特价商品。

被列在 350 页的第 75 页并不会真正给你的网站带来合格的流量。你需要第 1-3 页列出的前十大引擎，才能真正带动合格的流量。这通常需要整合一些非常专业的流程和技术。

客户非常有效。](http://www.wolfblast.com)

**9)钻研你的日志服务器文件**

发现通过你的网站制作的“数字轨迹”——你的日志文件是原始文件，显示人们如何从哪里访问你的网站，他们去了你的网站的哪里，他们停留了多长时间，等等。Web Trends 是事实上的行业标准，[http://www.webtrends.com/default.htm/](http://www.webtrends.com/default.htm/)但是我们使用并向我们的客户[http://www.fantomaster.com/](http://www.fantomaster.com/)推荐一家在欧洲有市场经验的公司，Fantomaster 有限公司
——他们有一套搜索引擎产品，可以免费下载或
购买。

**10)在你的整体网站设计中考虑全球化**

最大的互联网增长发生在北美以外的地区，所以建立一个世界各地的人都能方便访问的网站是非常重要的。

你需要考虑哪些问题？(再次)装载时间非常重要。开发避免口语化的内容，那些口语化的内容可能不会被说同一种语言的人理解。您可能想让您的网站内容以多种语言提供。有许多新兴的应用程序将促进这一进程，并确保您的电子商务能力可以被所有人利用。

就是这样…遵循这些方便的提示，并确保您的网站是点击就绪！
