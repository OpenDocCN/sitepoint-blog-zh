# 从失败中学习

> 原文：<https://www.sitepoint.com/learning-from-failure/>

这一周对我来说是学习的一周。我一直相信，当你诚实时，你会收获最大的回报，本周我发现这是多么的真实。我祈祷好运，屏住呼吸，出版了最新一期的《社区报》。我决定用它作为一个媒介来交流我在社区 t 恤项目中失败的经历，我把我的经历编辑成了社论。结果，我得到了很多很好的反馈，听起来，你们中的一些人已经能够从我的错误中吸取教训了。这一切都是值得的。

作为我为这篇文章所做研究的一部分，我在论坛上向[征求了你对 t 恤项目的反馈。我收到了大量的回复，这引发了我感兴趣的另一个问题——即](https://www.sitepoint.com/forums/showthread.php?t=706414)[论坛](https://www.sitepoint.com/forums/showthread.php?t=706611)的总体文化。感谢那些给了我你的想法和意见的人——即使它们不是我想听的——谢谢，它给了我很多思考。

![SitePoint Forums](img/104d83889068ade58ba6a544b89cfd06.png "sitepoint-forums")

我的注意力现在转移到了社区图书项目上。第一轮提交已经到来，我对写作的质量留下了非常深刻的印象。我真的很兴奋，我们已经找到了一种方法来利用论坛中就在我们眼皮底下的一些知识。如果你不知道这个项目，并认为你有一些贡献，[让我们知道](https://www.sitepoint.com/forums/showthread.php?t=680804)。如果我们还没有涵盖的主题，而你能够立即得到写作，那么我们可能能够满足你。

本周，在我们的[脸书页面](http://www.facebook.com/sitepoint)上，我为你带来了一个真正的惊喜。我们的联合创始人 Mark Harbottle 分享了他的背景和一些关于创业品牌的宝贵建议。如果你不太了解马克，[这是你了解他的机会](http://www.facebook.com/sitepoint?v=app_11007063052)。别告诉他是我说的，但除了鼓舞人心之外，他其实是个很好的人。下周，我将向你们介绍克里斯汀·霍尔登，我们最新工具包《搜索引擎优化商业指南》的合著者。如果你错过了我们上周的独家网络研讨会，你现在可以从我们的脸书页面下载录音。
