# 微软放弃 Windows E 并展示浏览器投票屏幕

> 原文:[https://www.sitepoint.com/microsoft-windows-browser-ballot/](https://www.sitepoint.com/microsoft-windows-browser-ballot/)

![browser ballot](../Images/cb64177bb0893727eba1a565ff899a2e.png)微软副总裁兼副总法律顾问 Dave Heiner 透露了有关 Windows 7 的更多信息，以及该公司将如何遵守有关 Internet Explorer 的欧洲竞争法。

Windows 7 E 最初打算不带 Internet Explorer 或任何其他浏览器。这个想法并没有打动欧洲委员会。微软的合作伙伴和电脑制造商也表示担心这会导致产品混乱。因此，包含 IE 的单一版本 Windows 7 将在全球范围内发售。

*注意:尚未宣布 Windows 7 E 预购事宜。E 版是完整的 Windows 安装盘，而不是 XP/Vista 升级版，后者可以在 7 月份以 50%的折扣预订。*

浏览器投票屏幕是一个网页，将向任何将 Internet Explorer 设置为默认浏览器的欧洲 Windows 用户显示。将会出现:

*   在第一次自动更新期间新安装了 Windows 7 之后
*   在将来自动更新 Vista 和 XP 时，以及
*   每当用户选择返回网页时。

![browser ballot screen](../Images/b2553ab474f549a09748bdb9fd5ae0ed.png)微软浏览器的投票屏幕

我希望那不是最终的设计！

正如怀疑的那样，IE、Firefox、Safari、Chrome 和 Opera 按照市场份额递减的顺序提供。因此 IE 占据了最左边的突出位置，尽管我认为 Firefox 和 Opera 更突出。我相信很多人会点击谷歌，因为这是一个众所周知的名字。

IE 的市场份额几乎肯定会减少，因为只有当人们将 IE 设置为默认浏览器时，屏幕才会显示。如果电脑默认安装了与之竞争的浏览器，则不会出现此信息。微软承认，谷歌、苹果和其他竞争对手可能与个人电脑制造商达成浏览器分销协议，但他们无力阻止。

为了遵守欧盟的规定，投票屏幕显示人们可以选择网页浏览软件，并可以随意改变主意。微软希望欧盟委员会接受这一提议，并在正式批准后实施该技术。
