# 雅虎！老板:现在实际上对谷歌有利

> 原文:[https://www . sitepoint . com/Yahoo-boss-now-actually-good-for-Google/](https://www.sitepoint.com/yahoo-boss-now-actually-good-for-google/)

雅虎！今天[推出了一个重大更新](http://ysearchblog.com/2009/02/11/boss-update-open-monetization-pricing-structured-data-and-more/)到它的建立你自己的搜索服务(BOSS)。虽然媒体主要关注每天产生超过 10，000 个 API 查询的搜索引擎的新定价，但公告中最重要的部分是 BOSS API 中语义标记数据的新可用性和开放的货币化模型。

首先，雅虎！宣布将通过 BOSS API 向开发者提供对其 [SearchMonkey](http://developer.yahoo.com/searchmonkey) 服务收集的结构化数据的访问。这意味着开发者现在可以使用 BOSS 来构建他们自己的语义搜索引擎。例如，搜索运动员不仅可以返回关于该运动员的网站，还可以返回统计数据、传记信息、新闻、记录等。可以构建在所有结构化数据之上的搜索混搭类型的可能性是无限的，也是相当令人兴奋的。

雅虎！还宣布了针对大批量 BOSS 用户的新[定价模式](http://developer.yahoo.com/search/boss/fees.html)。这并不令人惊讶，也不会影响他们在 11 月宣布的[网站搜索项目](https://www.sitepoint.com/yahoo-wants-to-power-your-site-search/)的许多用户。(如果你的网站搜索每天产生超过 10，000 个查询，你可能付得起一点钱。)

真正令人惊讶的是，雅虎！宣布了一项开放的货币化政策，允许开发者在搜索引擎上投放第三方广告。BOSS 团队成员 Ashim Chhabra 在 Yahoo！搜索博客。“由于显而易见的原因，货币化对 BOSS 生态系统至关重要，因此为了提供尽可能多的机会，我们决定调整我们的条款，为开发者提供更多的灵活性。”

7 月份，我们实际上推测 BOSS 可能最终会为 Google 赚钱，即使它设法削弱了他们的市场份额。当时，雅虎！我还没有透露他们计划如何让开发者将他们的 BOSS 作品货币化，我们推测，“如果雅虎！允许开发者随心所欲地将搜索结果货币化，那么无论如何，更多的垂直市场最终都会成为谷歌的赢家。”

到了 12 月，这种想法似乎已经被抛到了一边。随着 BOSS 每天产生超过 1000 万个查询，Yahoo！说货币化将通过雅虎来实现！受控频道。然而，似乎雅虎！已经改变了主意，Google AdSense for Search 确实有赚钱的可能。这意味着 BOSS 可能是谷歌的胜利，也可能是雅虎的胜利。

雅虎！BOSS 对公司来说是一个聪明的策略。他们还无法用自己的搜索排名和显示算法与谷歌竞争，所以开放他们的索引并让其他人尝试一下是个好主意。正如我们在 7 月份所写的，“如果有人触及任何革命性的东西，以至于用户开始选择它而不是谷歌，雅虎！将处于收购的最佳位置，因为它已经使用了他们的索引和后端技术。”

与此同时，BOSS 每天为雅虎产生数千万个新的搜索查询。这可以通过两种方式为他们赚钱——通过选择雅虎的开发者。将搜索结果货币化的技术，以及需要付费使用的高容量开发者。
