# Photoshop 的柔光

> 原文:[https://www.sitepoint.com/photoshops-soft-light/](https://www.sitepoint.com/photoshops-soft-light/)

Photoshop 这样描述柔光:

> 根据混合颜色，使颜色变暗或变亮。该效果类似于将漫射聚光灯照射在图像上。

> 如果混合色(光源)比 50%的灰色亮，图像会变亮，就像被减淡了一样。如果混合色比 50%的灰色暗，则图像会变暗，就像是加深了一样。用纯黑或纯白绘画会产生明显较暗或较亮的区域，但不会产生纯黑或纯白。

这是它的样子。请注意，50%的灰色带根本不会改变图像，而较亮和较暗的值分别会使图像变亮或变暗:

![](../Images/33acca6245f2956945c3f421a92493c8.png)

事实上，柔光的工作原理与叠加类似，但更加微妙:

![](../Images/a666c047851fb1fa894b41803125d15a.png)

([下载样本。psd 文件](https://examples.pixelmill.net/sitepointblog/040901/040901.psd)

柔光可以用来“平衡”亮部或暗部反差极大的图像。例如，这里有一张我和一些朋友的照片。在明亮的天空下，我们的脸变得非常暗:

![](../Images/47784bf69dbc8ec8b0e9ea14a1824ce2.png)

使用“色阶”对话框，我可以显著地使我们的脸变亮。但是如果我想要稍微多一点的平衡呢？

首先，我将复制图层，将其反转(Ctrl-I 或 Command-I)，然后取消其饱和度(Ctrl-U 或 Command-U 访问色调和饱和度对话框，然后将饱和度滑块一直拖到 0)。

![](../Images/e051d6c3ff6ff37933cd44a6aab35c1f.png)

通过将这个反转图层设置为柔光，图片中较亮的区域变暗(注意天空中有更多的蓝色)，图片中较暗的区域变亮。我们的脸可以使用其他调整和混合模式多一点工作，但图片作为一个整体肯定有所改善！

![](../Images/6c493cecc657cfa1d37fc0a4b78e8f2a.png)

([下载样本。psd 文件](https://examples.pixelmill.net/sitepointblog/040901/040901a.psd)

您还可以使用柔光稍微加强“褪色”照片中的颜色，并在有人穿着浅色衣服的照片中创建柔和的“发光”效果。在下面的例子中，“A”是原始的、稍微褪色的照片。“B”有一个设置为柔光的复制层，你可以看到颜色在燕尾服，花束和木材背景中得到了加强，但皮肤颜色看起来仍然很自然(如果你只是提高了整个图像的饱和度，情况就不会是这样了)。“C”为柔光复制层添加了高斯模糊，使白色婚纱看起来更加明亮，几乎发光！

![](../Images/add6a86ab34388c59347e6d4e966c25f.png)
