# 介绍:RubySource

> 原文：<https://www.sitepoint.com/introducing-rubysource/>

在 SitePoint，我们很兴奋地为这个家族推出了另一个新成员: [RubySource](https://www.sitepoint.com/) 。

RubySource 拥有一个才华横溢、见多识广、充满激情的博客团队，比如 Dan Cheail 和 Xavier Shay。这些 Ruby 从业者希望分享他们的知识，以便将 Ruby 带给 web 开发社区中的每一个人。

您可以找到关于 Ruby 和 Rails 生态系统的建议、教程和评论，包括:

*   测试、开发和部署基于 Ruby 的应用程序的最佳实践
*   对转换到 Ruby 感兴趣的有经验的开发人员指南
*   洞察 Ruby 的语法、约定和特质
*   更广泛的 Ruby 社区中的版本、事件和其他事件的更新

请[访问新网站](https://www.sitepoint.com/)，加入我们的[脸书页面](http://www.facebook.com/rubysource)，或者[在 Twitter 上关注我们](https://twitter.com/rubysource)，了解所有激动人心的 Ruby 新闻。我们希望听到您所有的反馈，所以请让我们知道您的想法。请确保您也密切关注一个特别的 Twitter away，它将很快为我们的关注者提供一些很酷的赠品！

如果你想通过写文章或教程成为 RubySource 的贡献者，请联系 editor@designfestival.com 的 T2。

一如既往地感谢您对 SitePoint 和[设计节](http://designfestival.com/)的持续支持。我们很感激，并且相信你会和我们一样喜欢 [RubySource](https://www.sitepoint.com/) 。
