# 评估关键词相关性

> 原文：<https://www.sitepoint.com/assessing-keyword-relevance/>

如果你一直在关注我(偶尔被打断)关于关键词策略的想法，你将有望回忆起几个核心概念:相关性和加权流行度。你可以在这里阅读我关于[关键词指标的完整文章。](https://www.sitepoint.com/blog/)

加权流行度的基本思想是，你并不是真的想知道一个给定的搜索词被使用了多少次，你想知道它被真正寻找你所提供的东西的人使用了多少次。用一个百分比来表示相关性，我们可以做一点数学计算来估计有多少目标搜索可用。

例如，如果我们认为使用给定搜索项的 50%的搜索者正在寻找我们所提供的内容，并且每天有 1000 次使用该搜索项的搜索，我们可以将 1000 乘以 50%以达到 500 的加权流行度，或者每天 500 次目标搜索。

我已经讨论了不同的方法来估计一个搜索词的受欢迎程度，我相信我们很快会再次触及这个主题。我今天要讨论的是如何估计一个搜索词的相关性。最简单的方法就是尽你最大的努力去猜测，但是就像很多简单的事情一样，这种方法并不一定有什么好处。

相反，我推荐一个稍微更有纪律的过程，我将在这里举例说明。

Wordtracker(和关键字发现)提供的最好的工具之一是一个全面的搜索工具，它允许您深入特定的搜索词，并找到包含它的所有搜索词。例如，如果我们看看“虚拟主机”，Wordtracker 使用其多重搜索工具显示了以下计数:

 `free web hosting 4231
web site hosting 1181
cheap web hosting 952
web hosting services 581
web hosting service 533
free web site hosting 359
business web hosting 328
adsl web hosting 258
web page hosting 240
affordable web hosting 192
professional web hosting 191
web design and hosting 184`

Wordtracker 实际上列出了数百个以上的搜索词，但这是博客，不是科学！总的来说，这些搜索词代表了搜索者提炼“虚拟主机”搜索的不同方式这让我们更好地了解使用通用搜索词的人实际上在寻找什么。

将这些计数加在一起，我们总共有 9，230 次搜索。判断这些较长的搜索词是否真正相关要容易得多。让我们假设的虚拟主机公司审查这些条款，并让他们决定“是，不是，或者可能”每个条款是否与他们的业务相关。

对于“是”的回答，我们会说该术语 100%相关，对于“可能”，50%，对于“否”，0%。然后，我们将每个搜索词的计数乘以这些百分比，得到一个加权的流行度。下面的列表显示了搜索词，分配的百分比，以及加权受欢迎程度。

 `free web hosting, 0%, 0
web site hosting, 100%, 1181
cheap web hosting, 0%, 0
web hosting services, 100%, 581
web hosting service, 100%, 533
free web site hosting, 0%, 0
business web hosting, 100%, 328
adsl web hosting, 0%, 0
web page hosting, 50%, 120
affordable web hosting, 50%, 96
professional web hosting, 100%, 191
web design and hosting, 0%, 0`

将这些加权的流行度分数加在一起，我们得到 3，030 个目标搜索。用 3，030 个目标搜索除以 9，230 个总搜索，我们得出的相关值为 33%。基于这个练习，我们可以(乐观地)估计，33%的“虚拟主机”搜索实际上与我们假设的主机公司相关，对于这样一个通用搜索词来说，这实际上是一个非常积极的结果。

在你投入大量资源来定位非常普通的搜索词之前，仅仅因为它们看起来非常流行，就有必要花几分钟来评估它们的相关性和针对性。
