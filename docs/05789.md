## 2.可重复使用的 D3 图表和组件

reD3 是用 D3.js 框架构建的可重用图表和组件的集合。它包括折线图、饼图、条形图、面积图、堆积面积图、气泡图、树形图和热图，可以轻松定制、扩展和重复使用。

[![reD3.jpg](../Images/bd346ec846e5a1c344ba34e60984a5b6.png)](http://bugzu.github.io/reD3/#/) 
[源+演示](http://bugzu.github.io/reD3/#/)

## 9.在 JavaScript 中使用媒体查询

如果你练习响应式设计，那么你会使用大量的媒体询问。媒体查询是 CSS 的一个特性，它赋予设计者添加类似 if 语句的能力。我的意思是，只有当当前页面满足某些条件时，它才允许你应用规则。在本文中，我将向您展示如何在 JavaScript 中使用媒体查询。
