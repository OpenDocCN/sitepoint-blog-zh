# 谁需要 Flash？CSS3 动画蜘蛛侠卡通

> 原文:[https://www.sitepoint.com/spider-man-css3-animation/](https://www.sitepoint.com/spider-man-css3-animation/)

这令人印象深刻。我们都见过纯 CSS3 动画，但这个例子是迄今为止最雄心勃勃的例子之一。它复制了经典的 1967 年蜘蛛侠漫画标题序列(在 Chrome 或 Safari 等 Webkit 浏览器中打开链接)。

啊，快乐的回忆！这个序列是由美国网络开发机构 Optimum7 的 Anthony Calzadilla 构建的。他在沃克创作了《T2 星球大战》动画，[安德鲁·泰特劳在二月](https://www.sitepoint.com/star-wars-makes-css3-animations-and-transformations-make-sense/)对其进行了分析。

[![Spider-Man CSS3](../Images/48b9e830469608ff5187440abe2554dc.png)T2】](http://www.optimum7.com/css3-man)

蜘蛛侠序列主要使用 HTML5 和 CSS3 以及-webkit-animation 声明。使用了一点 jQuery 支持的 JavaScript 来切换场景，但是所有的动画效果都是由 CSS 处理的。最后，一个 HTML5 `audio`标签被用于主题曲。

Anthony 表示，他最大的问题是动画和场景的定时，尽管他认为用 CSS3 而不是 JavaScript 来处理场景切换是可能的。

很明显，演示中投入了大量的精力，安东尼提供了一些详细的技术解释。它展示了什么是可能的，尽管我不相信许多开发人员会有时间或耐心手工编码。然而，如果 Adobe 等公司考虑在其应用程序中提供动画 CSS3 导出功能，我不会感到惊讶(特别是因为 [Flash 在*某些*平台](https://www.sitepoint.com/steve-jobs-criticizes-adobe-and-flash-again/)被禁止)。

有没有发现什么很棒的 CSS3 动画例子？
