随着原型开发工具的数量和种类不断增加，找出哪些工具满足什么需求，以及它们适合谁变得越来越困难。自从我们在 2015 年第一次写这篇文章以来，无数的设计应用程序已经主导(并改变了)了原型制作领域。

利益相关者的反馈和用户测试现在在 UI 设计中扮演着越来越重要的角色，这种新一代的工具旨在连接设计过程中这两个以前独立的阶段。客户*希望*参与进来，而电子邮件已经不再起作用了。一些应用程序如 UXPin 也负责线框阶段，而其他应用程序如 InVision App 通过提供设计移交工具来弥合设计师和开发人员之间的差距。

此外，现在具有云共享的桌面工具(Adobe XD、Axure、Balsamiq、Sketch+InVision)和协作在线工具(Figma、UXPin、Fluid UI、Proto.io)之间有了明确的界限。

这些工具中的许多似乎都集中在一个特性集上，这个特性集定义了现代用户体验设计师的角色。*TL；DR——这里有一个原型工具的快速比较。*

*本文于 2019 年 3 月更新，以反映原型开发工具市场的现状。*

## [Adobe XD](http://www.adobe.com/products/experience-design.html)

[//video.tv.adobe.com/v/16395t1?autoplay=false&quality=6](//video.tv.adobe.com/v/16395t1?autoplay=false&quality=6)

在 Sketch 兴起的过程中，Adobe 可能会打盹，但他们很快用 Adobe XD 弥补了这一点。它于 2016 年 3 月推出，是 Adobe Creative Cloud Suite 的最新成员。虽然你可以在 Craft 插件的帮助下在 Sketch 中构建交互原型，但 Adobe XD 令人印象深刻地提供了这些开箱即用的工具，因此设计师们已经像长期竞争对手一样将 Adobe XD 与 Sketch 进行比较。

如果你对一个工具感兴趣，它绝对值得一看，这个工具在一个应用程序中涵盖了你的所有基础(低保真原型，*高保真*原型，用户流，分享和反馈)。

**优势**

*   适用于 macOS 和 Windows
*   一个应用中包含您需要的一切

**弱点**

*   插件工作流不存在，你被锁定在 Adobe 生态系统中

## [视觉应用](https://www.invisionapp.com/craft)

[https://player.vimeo.com/video/198067805?title=0&byline=0&portrait=0](https://player.vimeo.com/video/198067805?title=0&byline=0&portrait=0)

[https://player.vimeo.com/video/153858834?title=0&byline=0&portrait=0](https://player.vimeo.com/video/153858834?title=0&byline=0&portrait=0)

InVision App 是目前市场上最大、最成功的设计协作工具，是严肃设计师和企业团队的首选工具。凭借白板协作、动态原型制作、评论、设备预览、视频通话、用户测试、版本控制和设计移交等工具，InVision 已经成为原型制作领域的一股巨大力量，但当你考虑到它的草图和 Photoshop 集成时，它就变成了一个一体化的设计套件，特别是当你加入 Craft 时，它可以将许多功能直接引入你选择的设计应用程序。

自本文第一版以来，该公司还发布了 InVision Studio，这是一款免费但功能强大的 UX 设计工具，受到了好评。这允许用户放弃草图和 Photoshop，将他们的整个设计过程保留在 InVision 生态系统中。它现在可以在 Mac 上使用，Windows 版本正在开发中。

**优势**

*   强大、成熟的平台
*   与草图完全集成，实现高保真设计
*   不断更新新功能

**弱点**

*   一开始，特性集可能有点让人不知所措
*   InVision Studio 目前仅适用于 macOS，但 Windows 支持即将推出。Sketch 只对 macOS 用户可用(但是你可以在 Windows 上把 InVision 和 Photoshop 配对，尽管 Photoshop 严格来说不是一个 UI 设计工具)

## [漫威 App](https://marvelapp.com/)

[https://player.vimeo.com/video/153059286](https://player.vimeo.com/video/153059286)

对于那些寻找更简单、更友好的 InVision 应用程序替代品的人来说，这是一个非常受欢迎的选择。漫威 App 擅长创建原型工具，适用于高级 UX 设计师和那些只想交流高保真和低保真度概念的人。此外，虽然他们支持使用 Sketch，但他们也提供组件库，以便在漫威实现完整的在线工作流程。漫威应用程序最近还集成了粉丝最喜欢的 [POP](https://marvelapp.com/pop/) ，允许设计师将他们的笔/纸想法转化为 iPhone 和 Android 应用程序。

**优势**

*   极大地支持额外的真实性过渡
*   对非设计师更友好，尤其是在给出反馈时

**弱点**

*   仅基于网络
*   没有离线设计

## [UXPin](https://www.uxpin.com/)

[https://www.youtube.com/embed/SQFGbPCjMN8](https://www.youtube.com/embed/SQFGbPCjMN8)

UXPin 是 UX 设计师最完整的在线解决方案。虽然您可以从 Sketch 和 Photoshop 导入，但您也可以使用 UXPin 的内置库设计复杂且响应迅速的界面，使 UXPin 也成为一种线框工具。凭借其[设计系统](https://www.uxpin.com/products/uxpin-systems)功能，UXPin 成为自动化文档、设计师/开发人员交接和协作功能方面最复杂的工具之一。计划从每月 20 美元开始。

**优势**

*   带断点的响应式设计
*   强大的动画(*不仅仅是链接屏幕*)
*   完整的设计协作和移交解决方案

**弱点**

*   附加特征增加了使用的复杂性

## [网络流量](https://webflow.com/)

Webflow 是一个用于设计响应式网站的可视化工具，它还可以输出干净的代码——它消除了从*设计*到*发布在网络上*的头痛。Webflow 与 WordPress 的竞争不亚于与 Sketch App 的竞争，它允许您设计包含后端(API)数据的全功能响应网站，并且只需单击一个按钮，就可以自动部署到完全可扩展的无忧托管。对于关心干净代码和移动友好的网页设计的现代设计师来说，它基本上是 Adobe Dreamweaver。

[https://www.youtube-nocookie.com/embed/Ymn1sPuNMt8?rel=0&showinfo=0](https://www.youtube-nocookie.com/embed/Ymn1sPuNMt8?rel=0&showinfo=0)

**优势**

*   可以包含真实数据(来自 API/JSON/etc)
*   创建高质量、可重用的代码
*   响应式网站可以轻松设计和部署

**弱点**

*   对设计原生移动应用没有帮助
*   需要一些 HTML/CSS 的知识才能最有效地发挥作用

## [图](https://www.figma.com/)

[https://www.youtube.com/embed/Mq3nJ_HqfaM](https://www.youtube.com/embed/Mq3nJ_HqfaM)

Figma 是原型开发领域的一个新成员，它拥有任何原型开发工具中最令人着迷的实时设计协作功能，同时在 Sketch 和 Adobe XD 的许多直观设计工具(如符号和设备预览)上建模其功能集，以及一系列通常为在线人群保留的工具(如版本控制和设计移交)。2017 年 7 月推出的 2.0 版本包括具有热点和开发人员移交的原型模式，以进一步简化设计工作流程。它可以在 macOS 的浏览器中运行，在 WIndows 的*和*中运行，尽管有时会很慢。

**优势**

*   实时协作设计功能是首屈一指的
*   功能齐全，从头到尾都是设计师的理想选择

**弱点**

*   Figma 有时会滞后，特别是在实时协作中

## 9 个值得考虑的原型工具

### [流体界面](https://www.fluidui.com/)

[https://player.vimeo.com/video/179337002?color=46A8FB&title=0&byline=0&portrait=0](https://player.vimeo.com/video/179337002?color=46A8FB&title=0&byline=0&portrait=0)

Fluid UI 专注于简单性和通信，包括内置的高保真和低保真度组件库、实时团队协作、设备预览和视频演示，使其成为设计师、产品经理和创始人等的一流解决方案。

### [协议 io](https://proto.io/)

[https://player.vimeo.com/video/172378883?title=0&byline=0&portrait=0](https://player.vimeo.com/video/172378883?title=0&byline=0&portrait=0)

Proto.io 成熟且功能丰富，最适合希望在浏览器中创建高保真和高度动画原型的设计师使用。

### [Axure RP 8](https://www.axure.com/)

[https://www.youtube-nocookie.com/embed/FaSKeCPC_Uo?rel=0&showinfo=0](https://www.youtube-nocookie.com/embed/FaSKeCPC_Uo?rel=0&showinfo=0)

成立于 2003 年的 Axure 是 UX 设计师的绝佳选择，他们需要为设计和动画制作提供最详细的说明。Axure 支持条件流交互、动态内容和自适应/响应设计，以及高保真和低保真度原型设计。Axure 是严肃设计师的严肃工具。

### [成帧器 X](https://framer.com/)

[https://www.youtube-nocookie.com/embed/ElJOsCzm05U?rel=0&showinfo=0](https://www.youtube-nocookie.com/embed/ElJOsCzm05U?rel=0&showinfo=0)

Framer X 跨越了设计和代码之间的界限，允许设计师用最少的代码量创建令人惊叹的细节动画(它使用简化版的 JavaScript coffee script ),并与 Sketch 和 Figma 等其他工具很好地集成。你必须编写动画和交互的代码，但是可能性基本上是无限的。

### [Mockplus](https://www.mockplus.com/)

[https://www.youtube-nocookie.com/embed/hLYQOljR4eI?rel=0&showinfo=0](https://www.youtube-nocookie.com/embed/hLYQOljR4eI?rel=0&showinfo=0)

Mockplus 是一个线框和低保真原型工具，具有相对简单的学习曲线，并支持 Windows，macOS，iOS *和* Android。Mockplus 拥有 200 多个内置组件和 3，000 个图标来帮助您快速构建模型，是一个相当可靠的快速原型制作解决方案，适用于使用各种设备和操作系统的团队。

### [Justinmind](https://www.justinmind.com/)

[https://player.vimeo.com/video/221875381](https://player.vimeo.com/video/221875381)

Justinmind 是一个功能丰富的桌面应用程序，对于一个 8 年来功能不断增加的工具来说，它令人惊讶地保持了出色的可用性。虽然不像 InVision 和 Adobe XD 那样主流，但它相当容易使用，设计师可以毫不费力地创建可点击的线框和功能性的高保真原型。绝对是个靠谱的选择。

### [弗林特](https://flinto.com)

[https://player.vimeo.com/video/137078262](https://player.vimeo.com/video/137078262)

Flinto 是一个 macOS 桌面应用程序，有一个更轻便的基于网络的替代应用程序，名为 Flinto Lite，两者都非常注重高保真移动应用程序原型，带有可定制的动画。Flinto 有一个有趣的定价结构，这使它与众不同——用于 macOS 的 Flinto 一次性为 99 美元，Flinto Lite 为 20 美元/月，但也带有用于 macOS 的 Flinto。如果你是一名独立工作的设计师，Flinto for macOS 是更便宜、更智能的解决方案。

## 动画原型工具

### [折纸工作室](https://origami.design/)

折纸是脸书的设计应用程序。草图中的设计可以复制到折纸中，强大的动画功能将让您探索不同的动画，并发挥无穷的创造力。Origami 附带了 iOS 和 Android 的预览应用程序，但核心应用程序仅适用于 macOS 用户。原型只限于动画。

### [Mac 的原理](http://principleformac.com/)

[https://www.youtube-nocookie.com/embed/vFrivX021-A?rel=0&showinfo=0](https://www.youtube-nocookie.com/embed/vFrivX021-A?rel=0&showinfo=0)

Principle 是另一个基于 macOS 的工具，专注于动画，尽管它的成本高达 129 美元/用户，但它有一个非常快乐的客户群。Principle 允许你在他们的 iOS 应用程序上导入草图和预览设计，但遗憾的是，没有用于测试的 Android 应用程序。

## 线框工具≠原型制作工具

原型不一定=线框。你可以创建高(更)保真度模型的线框(设计师通常更喜欢事后在线框上绘制低保真度模型)，你也可以以线框*风格*制作低保真度模型的原型，这就是为什么一些原型制作应用程序也包括线框工具(如 Axure 和 UXPin，我在上面提到过)。如果您认为有任何其他原型工具值得特别提及，请在评论中告诉我们！
