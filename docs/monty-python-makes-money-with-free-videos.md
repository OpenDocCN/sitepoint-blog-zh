# 蒙蒂·派森用免费视频赚钱

> 原文：<https://www.sitepoint.com/monty-python-makes-money-with-free-videos/>

“三年来，你们这些 YouTube 上的人一直在敲我们的竹杠，拿走了成千上万的我们的视频，并把它们放在 YouTube 上。现在形势逆转了。现在是我们自己解决问题的时候了。”

那是 YouTube 上官方[巨蟒频道](https://www.youtube.com/montypython)描述中的开篇段落。它是几个月前推出的，目的是提供在 YouTube 上流传多年的更高质量的剪辑版本，并为喜剧团网站[python online](https://pythonline.com/)提供流量。

Python 帮坦率地说出了他们对观众的期望。他们写道:“(我们不想要)你那些胡言乱语、无脑的评论。”。“相反，我们希望你点击链接，购买我们的电影和电视节目，减轻我们这些年来对被欺骗的痛苦和厌恶。”到目前为止，这种方法奏效了。

[据 YouTube](https://www.youtube.com/blog?entry=n-q9Enl2O2Y) 报道，自从推出他们的频道并参与该网站的点击购买计划以来，Monty Python DVDs 在亚马逊的销量已经增长了 23000%。这是销售额的巨大增长。即使他们以前每周只卖出几张 DVD，23，000%的增长也可能意味着数百万美元(嗯，如果可以长期持续的话)。

然而，人们很容易误解 Monty Python 的经验对该行业的启示。这里的教训不应该是放弃你的内容会自动让你变得富有——因为它不会。像赠送音乐的[九寸钉乐队](https://www.sitepoint.com/the-future-of-music-dont-forget-about-nin/)和赠送电视的[乔斯·韦登](https://www.sitepoint.com/dr-horrible-the-future-of-television/)这样的 Python 已经有了很高的知名度。这一品牌声望是通过多年的传统方法建立起来的，并得到了主流媒体出版商的大力支持。

然而，对于娱乐圈来说，这无疑是一个教训，一个需要做出的选择。Monty Python 在 YouTube 的经历为越来越多的证据提供了佐证，即在网上免费分发实际上可以增加销售额。该行业将需要选择哪条道路是有意义的:一条是让他们扮演执法者的角色，限制免费访问他们的内容，这似乎没有阻止盗版的浪潮；另一条是让他们将内容放在那里，让用户在他们想要的地方消费，这是一种成功的推广手段。

然而，他们不能两者兼得。正如 Mashable 的 Stan Schroeder 写的那样，“你不能在 ISP 层面上控制、监视和欺负用户，同时从内容的免费再分发中获益。娱乐圈迟早要做出选择。”

现在有大量可论证的商业模式证明，使用免费的、基于网络的分发，专业内容是可以盈利和成功的。希望业界注意到这一点，并通过像九寸钉乐队和巨蟒小组那样的方法控制内容的流动，而不是通过法庭来打击盗版。
