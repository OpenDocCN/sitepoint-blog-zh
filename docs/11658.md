# 在 Photoshop 中的形状内创建文字

> 原文:[https://www . site point . com/create-type-in-a-shape-in-Photoshop/](https://www.sitepoint.com/create-type-within-a-shape-in-photoshop/)

Adobe Illustrator 和 InDesign 都以其出色的文字处理能力而闻名。Photoshop 在处理大量文本时并不在同一个水平上，但你可以通过使用形状作为文本框来创建一些漂亮而不常见的字体设计。

您可以通过多种方式创建形状。

**使用钢笔工具**
1 .从工具栏中选择钢笔工具(P)，设置选项为“创建形状层”它是左边的第一个图标。您也可以选择第二个图标“创建工作路径”

![PenToolOptions](../Images/4c1cf20b0290f72380e070365bfa5461.png "PenToolOptions")

2.用钢笔画出任何形状。前景设置成什么颜色并不重要，因为我们稍后会隐藏它。

![PenShape](../Images/68a7a51854cf12f9abe0ca0e28c15ec0.png "PenShape")

3.选择文字工具(T)并将光标放在您刚刚绘制的形状上。光标变为类型区域光标。
![TypeArea](../Images/dc91e608e80887649802f82dba68baeb.png "TypeArea")

4.点击一次，开始输入。当您键入时，文本将填充您所绘制的形状。
![TypeFilled](../Images/9abe9e5305025c71f38df94a208efdce.png "TypeFilled")
5。要移除您绘制的形状的背景颜色，只需在图层调板中隐藏形状图层。
![TypeNoBackground](../Images/83cd2e031494e321ad201567ac57b210.png "TypeNoBackground")

**使用形状工具**对于更复杂的形状，或者用钢笔绘制有点困难的形状，您可以使用自定义形状工具(U)。Photoshop 安装了很多形状，你可以下载更多。

1.  Select the Custom Shape Tool (U) and in the tool options bar, click on the Shapes drop down menu.
     **![CustomShapes](../Images/bc76581a5f8699eaa7ea801cf5421d3c.png "CustomShapes")**

    在这个例子中，我将选择心形。您可以通过点击右上角的弹出菜单并选择加载形状来找到更多的形状。

2.  拖出形状。按住 Shift 键以约束比例。
    ![heartshape](../Images/ecfb731cddb21a02030f562ce0f6f536.png "heartshape")
3.  选择文字工具(T)，单击一次并开始键入。您的文本将保留在您绘制的形状内。
4.  当您添加足够的文本来填充形状时，您可以通过对段落对齐方式进行一些更改来获得更好的效果。打开段落调板，选择“居中对齐”。
    ![HeartShapeParagraph](../Images/b15e2ffede4254bab78eaaaaf0127427.png "HeartShapeParagraph")
5.  最后，点击任何其他层隐藏形状的边缘。你最终的图像应该是这样的:
    ![FinalHeart](../Images/bab292fb65f24173623bcbda42de4d95.png "FinalHeart")
