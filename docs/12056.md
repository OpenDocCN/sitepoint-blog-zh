# FullCodePress:获胜者是…

> 原文：<https://www.sitepoint.com/fullcodepress-and-the-winner-is/>

在今天下午的 WebForward 大会上，宣布了第二届 FullCodePress 活动的获奖者。

经过 24 小时紧张的需求收集、设计、内容编写、编码、测试和压力测试，来自新西兰的**code blacks**大获全胜，保住了卫冕冠军的头衔。获胜的网站是为彩虹青年建立的，彩虹青年是新西兰同性恋青年的支持团体。

这场胜利对澳大利亚队来说是一个打击，他们在 2007 年也被击败后，一直在寻求复仇。对澳大利亚人来说，一个有利的方面是，澳大利亚队的客户 DDLC 新南威尔士州能够带走一个网站，他们很高兴也很兴奋地使用这个网站来实现他们帮助处于不公平劣势的残疾人的目标。

CodeBlacks 的项目经理海顿·汤姆森在他的获奖感言中评论说，他对仅 24 小时后就交付的网站感到“震惊”。

来自 SitePoint 的组织法官 Raena Jackson Armitage 评论说，得分非常接近，但新西兰人在标准方面击败了澳大利亚人，部分原因是几个草率的验证错误。

最终的网站可以在网上查看，交接将在两个团队都睡了之后进行！

*   [查看新西兰网站](http://fcp-nz.com)
*   [查看澳大利亚网站](http://test.fcp-aussie.com)

两队都做得很好！
