# 系统管理员宣言

> 原文:[https://www.sitepoint.com/sysadmin-manifesto/](https://www.sitepoint.com/sysadmin-manifesto/)

在 Linux.com，布莱恩·琼斯从系统管理员的角度列出了他的“完美世界”愿望清单。虽然我们不会在所有问题上都达成一致，但他确实提出了一些我们都在处理或思考的核心问题。

就像任何其他宽泛的问题一样，乌托邦永远不会实现，然而，作为系统和网站管理员，我们找到的共同点越多，我们的共同声音就越大。通过将我们的集体立场投入市场，我们确实有助于推动一些创新，解决这些问题。

我们现在看到了尝试解决 Jones 观点的创新，包括:

*   共存于网络上的操作系统，降低了复杂配置的级别
*   基于标准的组织发展势头良好，包括非营利组织和私人组织(W3C、邮件滥用工作组、ConsortiumInfo.org 等)
*   PDA 和智能手机制造商将移动功能和基于网络的服务推向了新的高度

各行各业的技术专家需要找到共同点，并不知疲倦地为他们游说，而不是因为分歧而崩溃。这的确很重要。就我个人而言，我认为我们应该将这种观点传达给世界各地的政治家和他们的工作人员，并确保政府机构在技术问题上受到适当的教育，而不是在慷慨激昂的网站上花费精力，抱怨我们没有的东西。
