# 苹果超越微软成为最大的科技公司

> 原文：<https://www.sitepoint.com/apple-overtakes-microsoft-biggest-it-company/>

除非你是商业报刊的狂热读者，否则你可能没有注意到苹果最近超越微软成为世界上最大的 it 公司。投资者对苹果的估值为 2220 亿美元，而微软略低，为 2190 亿美元。当你开始讨论这种规模的估值时，几十亿美元无所谓。此外，微软仍享有 146 亿美元的年利润，高于苹果的 57 亿美元。然而，估值在商业世界中仍然很重要，苹果将会陶醉于其第一的新地位。

苹果上一次比微软大是在 21 年前的 1989 年。那次成功被一次引人注目的失败所取代，该公司在 20 世纪 90 年代中期几乎倒闭。

苹果的复兴归功于史蒂夫·乔布斯的重新掌权。多彩的 iMac 个人电脑让公司重回正轨。紧随其后的是 iPod 音乐播放器、iTunes 音乐商店、iPhone、新款 iPad 和苹果应用商店的成功。

虽然产品很棒，但我怀疑苹果的成功在很大程度上取决于人们对该公司的看法。在微软被指控从事邪恶的反竞争商业活动的时候，他们被视为处于劣势的创新者。

现在苹果是无可争议的冠军，看看这些看法是否会改变将会很有趣。他们的产品竞争更加激烈，它们不再是独家产品了(似乎每个人都有一部 iPhone！苹果已经开始从事自己的欺诈交易行为。

相反，微软因 Windows 7 以及新版 Office 和 Visual Studio 获得了普遍好评。在 Vista、IE6 和各种欧盟诉讼的公关灾难之后，该公司似乎更加谦逊了(尽管他们仍然倾向于制作一些愚蠢的东西，如荒谬的 IE9 HTML5 兼容性图表)。
