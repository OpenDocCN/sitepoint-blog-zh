# 出售你的摄影作品的地方

> 原文:[https://www.sitepoint.com/15-places-to-sell-your-photos/](https://www.sitepoint.com/15-places-to-sell-your-photos/)

我相信你听说过这样一句话“一张图片胜过 1000 句话”不管你是一个经验丰富的专业摄影师，还是一个有抱负的业余摄影师，这都是事实。许多人没有意识到的是，你的照片可能值 1000 美元。这可能有点夸张，但出售你的摄影作品赚钱并不像你想象的那样遥不可及。许多股票摄影网站非常愿意给你在他们的网站上出售你的作品的利润分成。

这是它的工作原理。你收集了很多很棒的照片，并想开始出售它们。你和一家库存摄影公司签约，作为回报，如果有人购买你的照片用于他们的工作，那么你就可以从每笔销售中获得一定的百分比。如果你的作品足够好，并且你的照片与某个特定的主题相关，它们可能会很值钱。

由于自由平面设计师和初创设计公司的增加，股票摄影网站近年来如雨后春笋般涌现。经济让每个人都在关注自己的预算，为一个项目聘请一名专业摄影师可能会很昂贵。建立他们想要的场景也可能需要大量的工作、时间和金钱。这就是一个好的股票摄影网站的用武之地。你去他们的网站，注册成为会员，你就可以立即开始通过主题搜索图片。一些较大的图片公司的数据库中有 100 多万张图片。你找到一张符合你创意的图片，支付合理的价格(5-50 美元)，下载图片，你就可以在几分钟内创作出你的设计作品。一些库存照片网站提供每月会员费，您可以下载任意数量的图像，或者您可以每月下载预定数量的图像。这通常取决于你的会员级别。这将专业水平的摄影快速、轻松地交到设计师手中，设计师节省了大量时间。

## 哪里卖你的照片

iStockPhoto–加入 istock photo 可能有点困难。你必须加入他们的网站，阅读他们的手册，并参加一个测验。然后你必须提交你最好的作品样本，如果被接受，你就可以开始出售你的照片。如果你向多个网站销售，你可以通过 iStockPhoto 获得 15%的收入，但如果你决定成为他们网站的独家，那么你可以从你的照片中获得 22-45%的版税。如果你满足某些要求，你可以把你的图片卖给整个 Getty Images 家族。

ShutterStock——免费投稿。你所要做的就是注册，你可以从每张图片下载 0.25 美元到 28 美元不等。如果你的图片很普通或者主题很一般，同时又很专业，这将会非常有利可图。

Fotolia 有一个巨大的图片库供专业人士使用。你可以注册一个免费账户。然后，你只需上传你的图片，并给它们相关的标签。Fotolia 将你的赚钱能力分为两类。第一种是单个图像下载，你可以从每个图像中获得 20%- 63%的收入，但这个比率取决于你是否是 Fotolia 的独家，以及你的排名。第二种是当你的图片通过订阅下载出售的时候。可以赚 0.25 学分到 0.40 学分。

Shutterfly–Shutterfly 做的事情略有不同。你只需支付年费，而不是直接在他们的网站上出售你的照片，他们允许你建立一个画廊来营销和出售你的照片。这种方式的好处在于，你可以支付合理的年费来建立自己的画廊。99 美元可以为你的图片提供 5GB 的托管空间，每年 199 美元可以为你提供无限的托管空间，所以如果你有非常多的摄影收藏想要出售，这个包就是为你准备的。另一个很棒的方面是，你为你的照片设定自己的价格，这意味着你从你的作品中制作你想制作的东西。人们可以订购你的作品，你可以得到报酬，或者他们可以通过数字下载购买你的作品。

[shutter point](http://www.shutterpoint.com/Sell-Photos.cfm)–shutter point 让您轻松加入并立即开始出售您的照片。你所要做的就是注册。他们还接受所有类型的照片，所以在你开始销售之前没有测试或接受过程。他们还提到了一个批量上传工具，所以你可以快速上传许多照片。他们对完全免版税图像的支付率为 85%，对标准免版税图像的支付率为 70%。

这个网站允许你注册，但是你必须完成一个教程才能开始上传你的图片。在你开始在 BigStockPhoto 上出售你的图片之前，他们的审查小组必须批准它们。当有人下载一张图片时，预付费购买将向您支付 30%的美元价格。这相当于 29.70 美元。当有人使用 BigStock 信用购买您的图像时，您将从每笔消费中获得 0.50 美元。

123RF 是另一个卖照片的好网站。只需注册，上传你的照片，你就可以根据他们下载你照片的方式获得报酬。如果有人用信用支付你的照片，那么你将得到售价的 50%。如果他们在订阅计划中下载你的照片，那么你每次下载将获得 0.36 美元。

dream stime–dream stime 允许任何人成为会员，但要出售你的照片，你的作品必须经过审查，你必须被接受。你得到的报酬取决于你的照片卖出时的价格。每个图像从 0 级开始，图像被下载的次数越多，它的级别就越高。根据某些标准，每个级别都有一个百分比。整个过程相当复杂，要理解，最好在他们的网站上查看图表。有 5 个级别，根据规模大小有不同的工资等级。

Corbis 是一个专业的股票摄影网站，也允许提交摄影作品。这里的区别是，他们对作为贡献者的细节非常模糊。你必须填写一份申请表，但网站上没有任何地方说明你的报酬会是多少，费率，如何运作，或者任何细节。Corbis 在薪酬方面不像这里列出的其他网站那样用户友好和直接。

CanStockPhoto–CanStockPhoto 让你轻松在网上出售照片。注册一个帐户，填写一份申请，一旦你被接受，你就可以开始快速上传你的照片。你也可以因为推荐其他摄影师而获得报酬。独家版权照片赚取 50%的佣金，不同大小的图像和资产赚取不同数额的佣金。[点击此处](http://www.canstockphoto.com/payout_schedule.php)查看详细列表。

[deposit photos](http://depositphotos.com/seller-price.html)–deposit photos 使向您销售图像变得快速而简单。注册一个账户，通过一个快速考试(他们向你保证这很容易)，然后你就可以开始上传图片了。唯一的障碍是，每张图片在正式展示在网站上出售之前都必须得到批准。他们的工资等级类似于 Dreamstime，当他们卖出一定数量时，你的图像等级就会增加。级别越高，每次销售赚的钱就越多。他们的网站上有一张图表，上面有一些例子。他们的一个突出特点是，你也可以出售矢量图像，但你必须以指定的格式上传它们。
