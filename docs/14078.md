# Photoshop CS2 中的消失点

> 原文:[https://www.sitepoint.com/vanishing-point-in-photoshop-cs2/](https://www.sitepoint.com/vanishing-point-in-photoshop-cs2/)

继续我上一篇文章的“Adobe Creative Suite 2”主题，我想我会带你了解 Photoshop 的一个最新、最棒的功能。

消失点是 Photoshop 中的一个新的迷你应用程序，允许你在任何 2D 图像上添加 3D 框架。有了这个网格，您的所有交互(剪切、粘贴、绘制、绘画、克隆、选择等)就像在 3D 环境中一样。这里有一个快速演示。

1)从“过滤器”菜单(或 Alt+Ctrl+V)激活消失点会将您的图像启动到一个消失点工作区，它有自己的一组控件和工具。

默认情况下,“制作平面工具”被选中并等待运行。如果你选择接受它，你的任务是在你的基础图像上找到并标出一个“参考矩形”，你可以合理地期望它有直角。这可能是一扇窗户、一栋建筑、一面墙或一扇门，但显然越大误差越大(我的例子不太方正)。

![](../Images/58ab28242a10964d6a1754be32cd24c2.png)

2)一旦你的矩形就位，你可以通过拖动它的边缘来随意缩放它。然而，按住[CRTL]同时拖动同一条边将创建一个垂直于当前平面的新平面。这相对容易掌握，几分钟后你就可以毫不费力地绕过拐角，爬上建筑物。

![](../Images/f65bb370cd815d9c3f21575206f9330b.png)

3)框架完成后，你就可以开始阅读仿 3D 作品了。例如，从剪贴板粘贴任何项目都会立即使其扭曲到光标所接触的平面的角度。同样，当你画的线倾斜远离你时，它们会扭曲和变细，即使是方形选择工具也会符合它所在的平面。很好。

![](../Images/de478a934817dcc659f6fe2cbe18bee2.png)

我错过的唯一功能是从一个平面画到另一个平面，而不必停止和重新开始你的笔画。也许是下一个版本的内容。

最后，虽然在你发现自己在使用它之前，很难说一个特性到底有多有用，但我希望将来能从这个特性中获得一些价值。
