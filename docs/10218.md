# 成为一名优秀的在线社区成员

> 原文:[https://www . site point . com/become-a-great-online-community-member/](https://www.sitepoint.com/become-a-great-online-community-member/)

今天对我来说是令人兴奋的一天。这是第二个年度[社区经理感谢日](http://www.web-strategist.com/blog/2010/12/20/second-annual-community-manager-appreciation-day-jan-24th-2011-cmad/)。我承认这可能不像其他任何一天那么重要，但作为一名社区经理，我会接受的。我一直在滥用所有的社交媒体来增强我的自尊心。毕竟这样的工作如果不好好利用又有什么意义呢？

所以现在我会做一些社区管理来赢得你的赞赏。

本周博客文章的主题并不新鲜，但它很重要，而且作为社区经理感谢日，感觉这是再次提出它的绝佳机会。我收到很多支持邮件，询问一些问题，作为一个经验丰富的论坛用户，我认为这是理所当然的。不久前，我写了一篇文章，概述了成为优秀社区成员的基本要素，包括论坛使用的基本要素和我们在 SitePoint 论坛上实施的一些指导原则。无论你是新手还是对某件事有点不确定，读一读这些关于成为优秀在线社区成员的提示。

这篇文章涵盖了搜索、寻求帮助的基础，以及最重要的一点——让我们喜欢你。成为社区的一员不仅仅是问问题。它是关于从你的同行那里获得想法，寻找新的想法，找出如何使你当前的网站变得更好——一大堆事情。论坛的工作人员不仅仅是确保你遵守规则，我们还想确保你和我们一起度过愉快的时光，并尽你所能从中获益。因此，请阅读这篇文章，如果需要，请随时寻求帮助。我得想办法挣钱！

接下来我想告诉你的是我们这周的新比赛。，从 2 月 1 日开始。这个会很棒的！在论坛上，我们正在举办纯 HTML & CSS 竞赛。有一些很棒的奖品等着你去争夺，所以请确保你的眼睛一直盯着开球。

![SitePoint Community](../Images/9bb2be66119f2936eef6bd9dee6364c7.png "SitePoint Community")

我们的脸书页面本周充斥着博客帖子。我上周报道说，我打算尝试一下 twitterfeed，到目前为止一切顺利。如果你有任何反馈，一定要在我们的墙上张贴。我很想听听。

本周最后一个——非常感谢所有参加我们的[募捐义卖来帮助昆士兰洪灾](https://www.sitepoint.com/forums/showthread.php?t=726358)受害者的人。我们非常自豪能够尽自己的绵薄之力提供帮助——当然，没有你们我们是不可能做到的。所以谢谢——你真棒。
