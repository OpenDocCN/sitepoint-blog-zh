# 现在你可以和我一样酷了——去拿 CF 8 测试版吧！

> 原文：<https://www.sitepoint.com/now-you-can-be-as-cool-as-me-go-get-cf-8-beta/>

是的，伙计们，现在你们可以像我一样酷了！在过去的几周/几个月里，我一直在参与 ColdFusion 8(又名天蝎座测试版)的工作。今天你也可以[玩 ColdFusion 的下一个版本](http://labs.adobe.com/technologies/coldfusion8/)。Adobe 今天发布了 ColdFusion 8 的第一个候选版本或公开测试版！！我给你一秒钟时间跳来跳去，从地板上爬起来，或者从头再看一遍。

那么为什么这个版本会这么棒呢？你期待探索什么？以下是我的亮点:

*   内置服务器监控–是的，现在您可以监控服务器的调整、瓶颈和性能问题
*   内置调试器——如果你是 Eclipse 的粉丝，这将是你梦寐以求的东西。现在你可以了！
*   Spry 集成–是的，伙计们，使用 ColdFusion 和 Spry 集成来构建 AJAX 应用程序并不容易。
*   。网络支持——虽然我自己没有玩过，但是现在你们中有需要的人可以使用它了。

我最感兴趣的东西/标签:

*   CFFeed 标签——轻松读取 RSS / ATOM 并创建 RSS 2.0 / Atom 1.0 订阅源！
*   新的 CFPDF / CFPDFForm 标签可以用 LiveCycle 和 PDF 做一些很酷的事情
*   CFExchange 集成–我真的还需要说什么吗？
*   cf image——标准化的图像处理，它会达到预期吗，可能它内置了验证码:)
*   CFZip——最终，我们可以压缩和解压缩我们的内容，而不需要其他自定义代码的 CFX
*   服务器监控！
*   cf thread–哦，这将会非常有趣！
*   根据应用程序设置！基于每个应用程序设置映射和自定义标记路径。
*   CF 管理员不再需要单点登录
*   OnMissingTemplate in application . CFC——我不知道为什么他们在早期版本中没有这样做，但谢天谢地现在有了！
*   新的文件操作功能——通过文件操作了解真实情况
*   CFML 表达式中的 Javascript 操作符——你想念代码中的++吗？现在你可以把它带回老学校。
*   CFDbInfo 标签——将各种数据库元数据显示在屏幕上。

所以你有它的乡亲。现在，由于这是一个测试版，我们不知道什么将被视为标准功能，什么将被视为企业。所以，当你在玩测试版的时候，大声说出来，让 Adobe 知道什么应该去哪里！

去拿吧！
