# 为什么阻止广告拦截器会失败

> 原文:[https://www.sitepoint.com/blocking-ad-blockers-will-fail/](https://www.sitepoint.com/blocking-ad-blockers-will-fail/)

![Blocking blockers will fail](../Images/9e8feb375d4a276ebeab2848eee0dc1c.png "Blocking blockers will fail")我的上一篇文章，[如何阻止广告拦截器](https://www.sitepoint.com/block-ad-blockers)提供了一种利用 URL 正则表达式匹配的技术。这种方法对使用广告屏蔽技术的用户隐藏了你的优质内容。

这个解决方案可能会取悦一些网站所有者和出版商。不幸的是，有几个原因可以解释为什么屏蔽屏蔽是徒劳的，会弄巧成拙，并可能导致*更少的*广告收入。

**1。你不会阻止每一个广告拦截器**
提供的代码有时会对一些广告拦截器起作用，但它肯定不会阻止每一个。没有一种技术适合所有情况。实施多种解决方案所需的成本和精力能够收回吗？

**2。可访问性问题**
任何禁用了 JavaScript 的人都无法访问你的页面。对于使用屏幕阅读器等替代设备的用户来说，它也可能会失败。这对你来说可能是可以接受的——那些用户也不会看你的广告。然而，这也可能会影响你的一部分常客。做好投诉和额外支持费用的准备。

**3。击败**
很容易，任何技术一般的人都可以阻止你的阻挡。Adblock 的大多数用户都懂技术:他们都是安装了浏览器插件的 Firefox 用户，许多人会理解拦截器的工作原理，有些人会编辑自己的正则表达式。

您的内容可以通过更改 Adblock 规则、查看源代码、禁用 CSS、使用替代样式表或使用 Firebug 或 Web Developer 工具栏修改代码来显示。

**4。您的收入可能不会受到影响……**
当访问者点击广告时，大部分广告收入都会增加。AdBlock 用户点击广告的可能性要小得多，即使他们是被迫观看广告的。屏蔽屏蔽者真的会增加你的广告收入吗？

**5。…或者它可能下降**
屏蔽你的优质内容只是让一部分用户看不到它。这会让他们担心吗？还是他们会去别的地方？这些访问者是你的客户:为难他们会损害你的生意。

我们已经确定，使用广告拦截器的访问者很可能精通技术。这些人可能有博客，使用在线书签，或者参与社交网络。如果你让内容更难浏览，他们会链接到你的内容吗？这将如何影响你的链接建设和病毒式营销的努力？

网络不同于印刷，你不能强迫用户按照你想要的方式浏览网页。任何人都可以屏蔽广告、图片、CSS、JavaScript、Java、Flash、ActiveX，或者根据个人喜好修改页面的任何方面。

如果你写文章是为了赚钱，并且确信广告拦截器影响了你的收入，有一个简单的解决办法:开始对内容收费。为什么在免费/赞助的幌子下发布内容，然后抱怨一部分用户没有点击广告？真正的问题是什么——广告拦截器还是商业模式？

最终，我怀疑广告拦截器对大多数网站的金钱影响可以忽略不计。如果收入下降，也许有更明显的问题需要首先解决？
