# 通过收缩来扩展你的简历

> 原文：<https://www.sitepoint.com/expand-your-resume-by-contracting/>

当大多数人想到从事自由职业时，他们会想到通过自己的销售和营销来获得客户。这一切都很好，但还有其他方法吗？

是的，有。

还有一个被忽视的潜在工作来源:其他人的客户。我这么说是什么意思？客户盗窃？不，不，不，那不公平！

我的意思是:**合同工作**。

现在有很多网页设计公司、网页开发公司和广告公司有他们自己的专业网页部门。他们都需要熟练的网络专业人员。有些人可能已经雇佣了他们需要的所有员工，但是以我的经验来看，大多数人都很乐意听到自由职业网络专家的意见。

他们为什么想收到你的来信？对他们有什么好处？

拥有一些可靠的外部承包商对公司来说是一个巨大的优势，因为这给了他们额外的能力和承担通常超出他们技能范围的工作的可能性。还有其他好处——他们不必向承包商支付工人福利，也没有义务在工程完工后继续支付工人福利。实际上，承包商是一个“现成的”员工，只在需要时才被雇佣。

对他们来说，不利的一面是什么？

*   合同工的时薪通常比正式员工高。
*   承包商给项目增加了额外的风险因素。他们会按时完成吗？他们的工作会有足够高的水准吗？这种风险通常在某种程度上被他们签订的合同所抵消。

那么这对你有什么好处呢？

*   工作、金钱和参与项目的机会，以及与你通常不屑一顾的公司合作的机会
*   更好的简历和更少的管理工作
*   增加曝光率和向潜在的未来客户展示你的技能的机会

有哪些不好的方面？

*   你通常会得到比理想时薪更低的报酬。
*   你可能要等更长时间才能拿到工资(取决于公司和合同协议)。
*   你可能有很紧的截止日期，这将需要你长时间的工作和周末。
*   客户不是你的客户。
*   公司会要求并期望你有更高的工作标准，可能还会比正式员工更快的周转。
*   作为一名“现成的”员工，你还会在你工作的每家公司都有一个新老板——这可能是你开始从事自由职业所要避免的。

所以，承包还是不承包，这是一个问题。归根结底，这是个人的决定。我强烈建议你考虑签约，不仅是为了获得经验和潜在客户，也是为了在简历中加入一些知名客户。

在我的下一篇文章中，我会给你一些如何接近这些公司的建议。
