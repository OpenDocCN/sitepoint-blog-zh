# 在 Photoshop 中更改多个文本层

> 原文：<https://www.sitepoint.com/change-multiple-text-layers-in-photoshop/>

来自 Akatombo Media 的 Bob McDonald 很友好地分享了这个关于 Photoshop 文字图层的技巧。

如果您想要对许多文本层进行相同的更改(例如，更改字体、颜色-选项栏中的任何内容)，您不必到每个文本层进行更改。(这要花很多时间。)

相反，链接所有要更改的文本层，然后选择其中一个，并在按住 Shift 键的同时使用选项栏进行更改。
