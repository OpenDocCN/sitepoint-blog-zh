# 2012 年 2 月浏览器趋势:Chrome 16 抹杀 IE8

> 原文:[https://www.sitepoint.com/browser-trends-february-2012/](https://www.sitepoint.com/browser-trends-february-2012/)

我预计[上个月的浏览器统计数据](https://www.sitepoint.com/browser-trends-january-2012/)会有点不寻常。随着西方世界很大一部分人在度假，家用与商用的比例上升。通常情况下，IE 的使用率会下降，而其他浏览器会上升。你会期望工业工程的波动会随着一月份商业的恢复而停止。因此，让我们看看最新的[全球 StatCounter 统计数据](https://gs.statcounter.com/)，看看是否发生了这种情况:

| 浏览器 | 十二月 | 一月 | 变化 | 亲戚 |
| IE 9.0 以上 | 10.75% | 11.45% | +0.70% | +6.50% |
| IE 8.0 | 22.12% | 20.82% | -1.30% | -5.90% |
| IE 7.0 | 4.00% | 3.63% | -0.37% | -9.30% |
| IE 6.0 | 1.78% | 1.56% | -0.22% | -12.40% |
| 火狐 4.0 以上版本 | 19.81% | 20.01% | +0.20% | +1.00% |
| 火狐 3.7- | 5.46% | 4.77% | -0.69% | -12.60% |
| 铬 | 27.33% | 28.45% | +1.12% | +4.10% |
| 旅行队 | 6.09% | 6.61% | +0.52% | +8.50% |
| 歌剧 | 1.99% | 1.96% | -0.03% | -1.50% |
| 其他人 | 0.67% | 0.74% | +0.07% | +10.40% |
| IE(全部) | 38.65% | 37.46% | -1.19% | -3.10% |
| Firefox(全部) | 25.27% | 24.78% | -0.49% | -1.90% |

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 12.4%的 IE6 用户放弃了浏览器。派对时间！有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

我之前报道过，尽管有广泛的报道， [Chrome 15 作为世界上使用最多的浏览器版本](https://www.sitepoint.com/browser-trends-january-2012/)的胜利是短暂的。2011 年 12 月 13 日发布的 Chrome 16 分裂了用户群，因此 IE8 很快保持了领先地位。

然而，我们现在已经到了 Chrome 16 的生命尽头。事实上，Chrome 17 已经过期一周左右，所以绝大多数 Chrome 用户——25.79%——都在使用 16 版*(我很想称它为‘旧的’，但它在七周大的时候几乎没有过它的全盛时期！下一个版本将再次分裂 Chrome 的用户群，但 IE8 失去阵地的速度太快，无法跟上。继上个月下跌 1.88%之后，1 月份又下跌了 1.3%。未来几周，这一比例可能会降至 20%以下。*

虽然 IE9 这个月表现不错，但总体来说，IE 下跌了 1.19%。火狐也下跌了 0.5%。虽然这些用户中的大多数都转向了 Chrome，但 Safari 也获得了意外的提升。假期收到新的 Mac 或 iPad 了吗？如果是这样的话，也许你要为这 0.5%的涨幅负部分责任。

Chrome 看起来不可战胜。它每月 1%的增长保持稳定，很可能在年中超过工业工程。

## 移动浏览器的使用

一月份，移动市场依然繁忙，使用量占所有网络活动的 8.49%。

主要的移动浏览应用程序有:

1.  opera Mini/Mobile——23.34%(下降 0.88%)
2.  安卓系统——21.39%(增长 1.17%)
3.  iPhone——19.51%(增长 1.10%)
4.  诺基亚浏览器——11.82%(下降 1.10%)
5.  黑莓——6.68%(下降 0.85%)

对这些数字做过多的解读是没有意义的；有太多的手机妓女换手机比换内裤还频繁！唯一明显的趋势是黑莓持续的不幸；但是你不需要浏览器统计数据就能看出这一点。
