# 奥普拉读书俱乐部？我不这么认为…

> 原文:[https://www.sitepoint.com/oprahs-book-club-i-dont-think-so/](https://www.sitepoint.com/oprahs-book-club-i-dont-think-so/)

你们中的一些人已经知道[交互设计协会](http://www.ixda.org/) (IxDA) [邮件列表](http://www.ixda.org/discuss.php)。对于那些不知道它的人来说，网站和邮件列表对于任何对交互设计、用户体验和设计感兴趣的人来说都是很好的资源和讨论来源。

> IxDA 是一个会员支持的组织，致力于满足国际交互设计社区的需求。在全球 1500 多名会员的帮助下，我们提供了一个讨论交互设计问题的论坛。

正如迈克尔·杰克逊曾经说过的:“我是一个潜伏者，不是一个作家”..好吧，也许是我。但尽管如此，在过去的几天里，我一直饶有兴趣地关注着一场非常好的讨论。List 成员已经发布了他们推荐给工程管理人员(或者其他任何人，我敢打赌)的“一本书”。

话不多说，下面是一些关于用户体验、可用性和交互设计的优秀读物的总结:

*   囚犯们正在管理艾兰·库伯的精神病院
*   [从屏幕的两面进行设计:设计师和工程师如何合作开发合作技术](http://books.google.com.au/books?id=errIIx0p0EQC&dq=designing+from+both+sides+of+the+screen+sample&pg=PP1&ots=I3qUSX47Dg&sig=WZZrIDV9VvfmWslbIEsYdDr5BhM&hl=en&prev=http://www.google.com.au/search?hl=en&client=firefox-a&rls=org.mozilla:en-GB:official&hs=WB3&q=%22%22designing+from+both+sides+of+the+screen%22+sample&btnG=Search&sa=X&oi=print&ct=title&cad=one-book-with-thumbnail)，艾伦·艾萨克斯&艾伦·瓦伦多夫斯基
*   [传达设计](http://www.communicatingdesign.com/)，丹·布朗
*   [日常用品的设计](http://www.amazon.com/Design-Everyday-Things-Donald-Norman/dp/0385267746)，唐纳德·诺曼
*   [易变:创造伟大的产品&为不确定的世界服务](http://www.amazon.com/o/ASIN/0596516835/adaptivepat07-20/ref=nosim/)，Peter Merholz，Todd Wilkens，Brandon Schauer，David Verba
*   [关于交互设计的思考](http://www.thoughtsoninteraction.com/)，乔恩·科尔科
*   [不要让我思考](http://www.sensible.com/chapter.html)(样章)，史蒂夫·克鲁格
*   [为人民而设计](http://books.google.com/books?hl=en&id=3ss6VTFH4oEC&dq=designing+for+people+henry+dreyfuss&printsec=frontcover&source=web&ots=02T_hDDeqt&sig=qgyUwiG35BhN5S8xBmA_ZgXsAWA#PPA5,M1)，亨利·德雷夫斯
*   [一个设计师的艺术](http://www.amazon.com/Paul-Rand-Designer%60s-Art/dp/0300082827)，保罗·兰德
*   [展望信息](http://www.edwardtufte.com/tufte/books_ei)，爱德华·塔夫特
*   设计的普遍原则，威廉·利德威尔，克里斯蒂娜·霍尔登，吉尔·巴特勒，T2
*   [交互设计:超越人机交互](http://www.id-book.com/)，夏普，罗杰斯&普里斯
*   设计显而易见的东西。

现在*很有可能*这些都不会成为 2008 年奥普拉的读书俱乐部，但是不要让*这*阻止你！

那么，你会向用户体验新手推荐哪一本书呢？你最喜欢哪一个？
