# 使用 Flash Builder 进行数据驱动的开发，第一部分

> 原文：<https://www.sitepoint.com/data-driven-development-with-flash-builder-part-i/>

如果您一直在关注我们最近的 Flash Catalyst 和 Flash Builder 教程，您将会熟悉 Flex 大师 Toby Tremayne 清晰简洁的“操作方法”风格。在今天的文章中，他向我们展示了如何在 Flash Builder 中从 Flash Catalyst 模型过渡到全功能的数据驱动开发项目。

Toby 以简要概述第一部分开始，向我们展示了如何将模型作品导入 Flash Catalyst，然后将元素转换为工作组件。假设对 Eclipse 平台有一些基本的了解，Toby 然后带我们通过几个简单的步骤，这将允许我们在 Flash Catalyst 和 Flash Builder 之间无缝地移动。
![](img/16be4ea3ba44fbf82db8d80aa96b8eff.png "Flash Builder 4")
本教程结束时，你将学会如何在 MySQL 中创建一个数据库来存储联系人数据，以及如何配置 Flash Builder 通过 PHP 接口读取这些数据。下周，我们将结束该系列，向我们的联系人列表应用程序添加一些功能，并完善我们对 Flash Catalyst 和 Flash Builder 的介绍。
