# 史蒂芬·柯林斯:如何在商业中拥抱 Web 2.0

史蒂芬·柯林斯是公认的参与式文化的主要支持者之一，为企业和政府提供 Web 2.0、Enterprise 2.0 和社交网络方面的建议。他在为公共和私营部门的不同客户群提供咨询方面拥有丰富的经验。

Stephen 从繁忙的日程中抽出时间来和我们谈论 Web 2.0 和社交媒体。

你好，斯蒂芬。你最近在澳大利亚共同主持了一个 Web 2.0 大学研讨会。是什么让 Web 2.0 如此特别，以至于获得了它一直以来的关注？

有些人，尤其是那些有着老派思维的人，认为围绕 Web 2.0 和 Enterprise 2.0 的整个革命——这是一场革命——是关于我们可以使用的所有伟大工具。我的观点是，工具本身是整套工具中最不重要的部分。2.0 的改变是关于人和文化的，这是十年前 [*《列车宣言》*](http://www.amazon.com/Cluetrain-Manifesto-End-Business-Usual/dp/0738204315/) 传达的信息。

如果你对希望接受 web 2.0 理念的 Web 行业之外的人有一条建议，那会是什么？

开放并公开。赋予人们权力。做人吧。不要害怕犯错误。要尊重。

那么，作为一名网络自由职业者或网络公司，我们能做些什么来开始在我们自己的企业中接受 Web 2.0 的理念呢？

从阅读或重读《线索列车宣言》开始，开始实践它所宣扬的东西。那么，就拥抱 2.0 的经营方式吧。这样做生意。它可以而且确实有效。甚至可能签署并使用类似于[公司-客户契约](http://ccpact.pbwiki.com/)的东西。

还有很多其他值得一读的好书，任何希望实现“go 2.0”(天哪，这是个老掉牙的说法)的企业都应该把它们放在每个员工的办公桌上。排名不分先后(只看我的书柜):

*   *[海星和蜘蛛](http://www.amazon.com/Starfish-Spider-Unstoppable-Leaderless-Organizations/dp/1591841437/)* 奥里·布拉夫曼和罗德·贝克斯特罗姆
*   *[风起云涌](http://www.amazon.com/Groundswell-Winning-Transformed-Social-Technologies/dp/1422125009/)* 作者:李夏琳和乔希·贝诺夫

随着 Web 2.0 的发展，社交媒体备受关注。这对公司来说是一把双刃剑，不是吗？前一分钟，一家公司可能是“社交领域”的代表，下一分钟，它们可能就在外面了。他们能做些什么来避免成为错误的一方？

我认为社交媒体作为一种风险的概念是错误的。如果你做得不够熟，那才是冒险。你不会不加考虑就做出其他商业决策吧？选择合适的人成为你的品牌在线的传播者和导师。授权他们参与对话，并使之成为他们日常工作的一部分，而不是额外的任务。逐步给予企业中每个想参与的人他们需要的技能，然后让他们飞起来！

在这方面做得好的品牌在使用社交媒体方面取得了巨大成功。例如，你很难从社区中找到关于 Zappos 的坏话。康卡斯特(Comcast)和戴尔(Dell)等品牌的情绪正朝着非常积极的方向发展，因为它们已经实施了良好的、规划良好的社交媒体方法。计划周密并不意味着缓慢或合作；而是选择合适的渠道和合适的人，让他们自己去做。

我帮助许多客户制定社交媒体策略。这不应该是轻而易举的事，它确实需要一些思考。但是你不能慢慢来——你的竞争对手可能已经和我谈过了，或者和我工作相似的其他聪明人。

网络确实在变化。你认为我们这些建立网站的人需要调整我们的服务吗，或者在未来的五年或十年里仍然会有客户寻找标准的网站吗？

宣传册网站可能会在五年后仍然存在，但可能不是十年。至少在发达国家没有。越来越多的客户在寻找全方位的服务方式:品牌战略、市场营销、社交媒体、沟通等等。大机构已经这样做了，但我认为他们的产品并不总是像小的精品公司提供的那样好。

我认为，我们这些在网络行业经营小企业的人——无论是设计、开发还是战略——需要开始以非正式的方式合作，与大机构竞争。如果这些机构认识到一些精品和专业公司应该在他们的专家咨询名单上，那就更好了。

即使在经济不景气的时候，每个人都有足够多的工作要做，但我们应该更多地一起玩耍，而不是试图将对方拒之门外。很有 2.0 的我，不是吗？

谢谢你的时间，斯蒂芬。
