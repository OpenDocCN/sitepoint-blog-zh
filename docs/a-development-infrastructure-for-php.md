# PHP 的开发基础设施

> 原文：<https://www.sitepoint.com/a-development-infrastructure-for-php/>

刚刚遇到了一个 PHP 开发基础设施，它讨论了一个通用策略，T2·托尼·T3 在用 PHP 构建应用程序时使用了这个策略。他在 [FAQ](http://www.tonymarston.net/php-mysql/infrastructure-faq.html) 中进一步阐明了一些设计决策。

我认为 Tony 做得非常好——这是一个端到端的视图，展示了他为构建 web 应用程序而发展的方法，或多或少也是第一次——我还没有见过任何人敢如此详细地描述一个完整的 PHP 架构。这里面有很多有价值的见解，并得到了“理智”的总体印象(即考虑 PHP 固有的优点和缺点)。我认为这里有足够的弹药给马可。

Tony 也写了他对许多其他 PHP 相关主题的想法，加上一些关于 T2 XML 和 XSL 的优秀信息。正在阅读他关于[工作流程](http://www.tonymarston.net/php-mysql/workflow.html)的想法。
