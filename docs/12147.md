# 如何提高您的费率

> 原文：<https://www.sitepoint.com/how-to-raise-your-rates/>

工作越多，经验越多，就越优秀，就越应该充电。此外，随着做生意的成本增加，你需要定期提高利率，以维持你的业务。即使在经济衰退时期，提高利率也是很重要的，尽管在预算削减时期可能会有较小的增幅。这里有一些事情你可以做，使利率上升过程顺利进行。

**发出通知**

当你必须让你的客户知道你在提高利率时，总会有点担心。为了让你的客户更容易处理这个过程，尽可能多的通知他们。我尽量至少提前三个月通知客户价格将会上涨，这样我的客户就有足够的时间进行必要的预算调整。

**设定日程**

加息的时机不应该是随意的。制定一个时间表(例如每年一次)，并确保你的客户事先知道这一点。这是一个非常重要的条款，可以在你的合同中体现出来，所以从一开始就有明确的预期。

**让它变得公平**

你最不希望的事情就是失去一个客户，因为他们无法承受你的价格大幅上涨。如果你刚刚起步，并意识到你的利率远低于他们可以，你可能会失去客户的风险，如果你一次翻倍你的利率。你可以按照一个更频繁的时间表逐步增加，这样你的客户会更容易消化。你也可以开始以更高的价格对所有新作品收费。虽然这可能需要你在簿记实践方面进行更多的管理，但它可以让你慢慢地让你的现有客户达到你的新利率，而不会失去从新工作中获得更多利润的潜力。

**以书面形式发送**

书面通知你的客户你的价格上涨。在涨价前的几个月里，我总是在他们的发票底部注明，并通过电子邮件发给我所有的客户。必要时，你也可以考虑在合同中添加一个附录，详细说明新的条款。

**平衡一下**

即使你的利率涨幅很小，你的客户可能也不会兴奋地迎接它。提供一个小奖励——一个特殊的，一次性的折扣，赠品——来改善这种情况。一定要感谢你的客户，表达你的感激之情。这有助于让加薪更容易被接受。
