# 免费 PHP 网络研讨会:如何通过缓存提高性能

> 原文:[https://www.sitepoint.com/free-php-webinar/](https://www.sitepoint.com/free-php-webinar/)

Zend 将于 4 月 22 日星期三在旧金山举办一场免费的网络研讨会(太平洋标准时间上午 9:00，世界协调时下午 4:00/GMT)。该活动由沙哈尔·埃夫隆主持，主题为*“如何使用缓存提高应用性能”*。沙哈尔将进行一小时的现场演示，展示使用 Zend Server 优化和提升 PHP 应用程序性能所需的步骤。

该公司还发布了 Zend Server Community Edition，这是一款易于安装和使用的免费 web 应用服务器。它推荐给在生产环境中运行 PHP 的开发人员，并且可以在 Windows、Mac 和 Linux 上下载。该套件包括:

*   Apache 2.2 web 服务器
*   服务器端编程语言（Professional Hypertext Preprocessor 的缩写）
*   Zend Optimizer+(操作码缓存和优化)
*   Zend 缓存(数据和输出缓存)
*   Zend Guard Loader(编码 PHP)
*   Zend 框架(PHP 库)
*   Zend 调试器
*   Java 服务器(Java PHP 扩展)
*   Java 桥(PHP 中 Java 库的集成)
*   Oracle 即时数据库客户端
*   IBM DB2 RTCL 数据库客户端
*   MySQL 数据库
*   phpMyAdmin (MySQL 管理工具)
