# 向朋友致敬

> 原文:[https://www.sitepoint.com/tribute-to-a-friend/](https://www.sitepoint.com/tribute-to-a-friend/)

上周，得知奈杰尔·麦克法拉尼不幸去世的消息，我深感悲痛，他是我们所有人在 SitePoint 的好朋友和同事。

Nigel 是一位广受尊敬的开源专家、倡导者和评论员，他与我们即将发行的《Firefox Secrets》一书的出版团队密切合作。他惊人的技术知识、简洁而优雅的措辞以及典型的极客式幽默感使他在 SitePoint 办公室一炮而红。

我们都为未来的图书项目和未来的啤酒感到遗憾，我们永远没有机会与他分享。

非常想念的朋友。
