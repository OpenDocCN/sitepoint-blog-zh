# 把瘸子带出来

> 原文：<https://www.sitepoint.com/bring-out-the-gimpshop/>

Gimp 很强大。Gimp 很快。Gimp 是免费的。

从任何客观的角度来看，这都是一个非常方便的组合，所以问题仍然存在。为什么我们不都用它呢？虽然目前可能有三个主要原因，但其中一个可能会在黑客攻击中消失——至少对 Mac 用户来说是这样。

**原因 1——不可抗拒的习惯力量:**无论是我们的早餐麦片、操作系统还是牙医，坚持你所知道的总是最容易的，而不是为了未知的回报去冒险。

**原因二——行业标准:**面对现实吧。在 Monster.com 上快速搜索会产生一个[三个需要 Gimp 知识的职位列表](http://jobsearch.monster.com/jobsearch.asp?q=gimp&cn=&sort=rv&vw=b&cy=US&re=14&brd=1%2C1862%2C1863)。这三个人都把它和 Photoshop 技能列为一项要求。类似的搜索“ [Photoshop](http://jobsearch.monster.com/jobsearch.asp?q=photoshop&re=0&sort=rv&vw=b&cy=US&brd=1%2C1862%2C1863&rad=&zip=&x=0&y=0) ”的结果页面显示..1000 多英尺。简而言之，*此时*你不会选择 Gimp 作为一个纯粹的“职业行动”。

理由 3——界面:当你开始意识到你无法理解界面时，没有什么比打开一个漂亮的新应用程序的兴奋感慢慢被愚蠢的感觉所取代更好的了。我个人记得 LightWave 和 3D Studio Max 是这方面的犯罪者。我最终艰难地通过了测试，并最终爱上了 LightWave，但我从未和 Max 有过太多接触(然而，不管怎样，我上周安装了 GMax)。

在这三个 Gimp 问题中，前两个是紧密相连的，并且只能随着时间的推移而改变。当很多设计师用 Gimp 展示他们的技能时，雇主自然会找他们。当然，大多数小企业可能会欣赏开源许可模式的简化效果。

正如 Firefox 所展示的，当动机足够强烈时，根深蒂固的习惯是可以被打破的。Mozilla 团队也足够聪明，意识到尽可能简单地向他们的软件过渡的重要性。无论在什么地方，他们都保持按钮配置、菜单和快捷键与当时的标准——IE6 完全相同。

斯科特·莫斯切尔对 Gimp 采取了非常相似的方法。Gimpshop 是 4000 万名黑客，旨在改造 Gimp，使其外观和行为更像行业标准——Adobe Photoshop。
