# 在 Mac 上查看您的网站

> 原文：<https://www.sitepoint.com/see-your-web-site-on-a-mac/>

我在德国的好朋友费利克斯经营着一个很棒的网站，为那些想在苹果电脑上使用 Safari、IE 5 或 Mozilla(苹果截屏服务)浏览自己网站的开发者提供了一项很棒的服务。

我自己不需要这项服务——在 OS X 平台上工作——但是我从他的工作中获得了一些非常有用的 CSS、JavaScript 和 PHP 技术。

Fundisom.com 包括:

*   CSS 技巧——布局和悬停技巧
*   PHP 脚本——大量有用的函数和现成的脚本，涵盖了广泛的任务。我特别喜欢 MySQL 的[超越](http://fundisom.com/phparadise/php/databases/mySQL_to_excel)和[裁剪图像](http://fundisom.com/phparadise/php/image_editing/crop_image)的提示。Felix 还包括一些在目录中递归工作的优秀方法。
*   这是一个 PHP 镜像函数，允许在没有 FTP 服务的情况下进行服务器到服务器的复制。我自己也用了这个，因为我最近需要这种东西，并成功地用他的代码移动了一个站点。

除了 Mac 屏幕截图服务，人们还可以使用谷歌位置跟踪器，浏览漂亮的在线 RSS 新闻阅读器，并使用一个非常酷的颜色选择器来玩颜色，该颜色选择器可以显示您使用十六进制代码输入的每种颜色的所有可能的色调和阴影。

最后但同样重要的是，有一些简洁的 JavaScript 脚本，包括一个很酷的 CSS 和 HTML 代码的实时预览，用于测试您的想法(在源代码编辑器中键入时预览)。
