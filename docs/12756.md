# Twitter 开始重视垃圾邮件

> 原文:[https://www.sitepoint.com/twitter-gets-serious-about-spam/](https://www.sitepoint.com/twitter-gets-serious-about-spam/)

Twitter 上的垃圾邮件显然已经成为一个相当大的问题。我从来没有真正注意到它，但我也只有几百个[粉丝](http://twitter.com/catone)，所以也许我还不够显眼，不足以成为一个目标。在其创始人对 Twitter 的宕机灾难感到失望之前，非官方的 [Twitter 黑名单](http://twitterblacklist.com/)网站确定了 561 个已知的 Twitter 垃圾邮件发送者——其中大部分已经被删除。

今天，Twitter 联合创始人比兹·斯通宣布，他们将开始对垃圾邮件发送者采取严厉措施。Twitter [在其博客](http://blog.twitter.com/2008/08/turning-up-heat-on-spam.html)上宣布了三项新的反垃圾邮件措施:

*   **暂停账户**–Twitter 开发了一种新的内部管理工具，可以更容易地快速暂停滥用账户。5 月，Twitter 表示他们将开始[移除被标记为潜在垃圾邮件发送者的违规账户](http://staynalive.com/articles/2008/05/06/twitter-now-removing-blacklisted-accounts/)，并联系账户所有者，允许他们为自己的案件辩护。
*   **社区提醒**–Stone 还宣布他们的管理工具现在将自动跟踪被其他用户屏蔽的账户。越多的人阻止一个帐户，越多的危险信号就会出现，因为该帐户属于一个潜在的垃圾邮件发送者或其他 TOS 滥用者。
*   专注的员工——这一变化可能显示了他们对打击垃圾邮件的最大承诺，那就是 Twitter 宣布将雇用一名全职、专注的员工来处理垃圾邮件。“我们的第一个‘垃圾邮件元帅’将于下周在 Twitter 上启动，”斯通写道。

社交媒体工具中的垃圾邮件总是一个大问题。例如，MySpace 开发团队[今天宣布了处理滥用应用程序的协议](http://developer.myspace.com/Community/blogs/devteam/archive/2008/08/21/when-good-apps-go-bad.aspx)。脸书经常改变其应用程序必须遵守的规则，以减少其新闻提要中的垃圾应用程序。

垃圾邮件已经超过了最流行的网络交流工具:电子邮件，达到所有用户的 90-95%。垃圾邮件制造者继续发送垃圾邮件的原因是，这是一种非常有效的赚钱方式。安全公司 Marshal 的一份报告发现 [29%的用户曾通过垃圾邮件购买过东西](http://www.readwriteweb.com/archives/29_of_internet_users_have_boug.php)——29%！

如果 95%的信息来自垃圾邮件发送者，Twitter、脸书、MySpace 和其他社交网络肯定会失去吸引力。很高兴看到流行的社交工具的所有者如此认真地对待垃圾邮件，并试图控制它，使它们不会遭遇与电子邮件相同的命运。
