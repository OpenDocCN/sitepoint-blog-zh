# 让你的用户参与进来

> 原文:[https://www.sitepoint.com/involve-your-users/](https://www.sitepoint.com/involve-your-users/)

我们中的许多人都参与了大型网站和电子商务。上述许多网站不擅长的一个元素是用户生成的内容。我喜欢用户生成的内容，许多人也是如此。调查结果显示，在公司宣传之前，人们会信任他人，因此，如果你在网上销售产品，请加强你的评论，并寻找可以通过增强社区权能来提供的替代内容。

这里有一个很棒的故事。我喜欢认为自己是一个精明的网络用户。我可以通过那些自动发送的电子邮件和 nag 屏幕在许多网站上注册成为会员。我也是一个经常出差的人，无论是出差还是旅游。我最近去了趟国外，回来后，我在我所属的网站猫途鹰上写了几篇评论。

这是他们变得真正聪明的地方。在过去的几年里，我已经写了十几本了，但是我忽略了回顾我呆过或吃过的每一个地方(见鬼，那会填满我的周末！).大约在我上一篇评论发表一周后，我收到了一封电子邮件，标题是“人们喜欢你的评论！”打开它，我得到了统计数据，有多少人阅读了我的评论，有多少人给了他们一个有用的评级。

这封邮件以一个很棒的钩子结尾:“你离下一枚徽章只有一个评论了。继续前进！这个周末你做了什么？”

我被迷住了！那天晚上，我又写了两三篇评论，只是为了获得下一枚徽章。我需要下一个徽章吗，或者它能为我提供任何有形的价值吗？没有。总的来说，人类喜欢工作得到回报吗？是啊！

我在这里，放弃了我宝贵的时间来写评论，张贴在另一个网站上，这使得他们的钱来自所有这些用户生成的内容。干得好！干得好，我还赢得了额外的荣誉徽章。

你如何在你的网站中融入这些想法呢？也许一个“评论这个产品”的促销活动，或者一个“写一篇客座博文”可能会起作用——开动脑筋，你可能会找到一个赢家。

如果进展顺利，我可能也会在上面写东西——只要我为我的下一个徽章写完下一打评论！
