# 七个 jQuery 插件，让你用图像做很酷的事情

> 原文:[https://www.sitepoint.com/seven-jquery-plugins-images/](https://www.sitepoint.com/seven-jquery-plugins-images/)

在最近的一篇文章中，我写了关于 jQuery 插件的文章，这些插件可以让你轻松地将[时尚的图库和幻灯片](https://www.sitepoint.com/five-jquery-galleries-and-slideshows/)添加到你的站点中。今天，我们有六个插件，可以让你给图片添加一些很酷的功能和有趣的效果。作为一名设计师，我发现像这样的插件非常有用，因为我自己永远无法编写或开发这样的功能。因此，感谢慷慨的开发者社区分享所有这些伟大的工作。现在来看看插件…

**[jParallax](http://webdev.stephband.info/parallax.html)**
jParallax 将选中的元素变成一个“窗口”通过这个窗口，你可以看到许多绝对定位的层。这些层会随着鼠标的移动而移动，从而产生视差效果。去年在流行的[银背应用](http://silverbackapp.com/holding/)网站上利用了视差效果。

要下载插件并查看工作演示，请访问[插件](http://webdev.stephband.info/parallax.html)网站。

![parallax](../Images/f728cd5ce6b0f851632ec5d29b6639ee.png "parallax")

**jQuery [360 虚拟游](http://www.openstudio.fr/jQuery-virtual-tour.html?lang=en)T3】**

这是一个非常好的插件，它允许你在你的网站上添加一个全景图像，并具有额外的交互性。用户可以平移图像，也可以点击热点进行近距离查看。

[下载插件](http://www.openstudio.fr/jQuery-virtual-tour.html?lang=en)。

[观看演示](http://openstudio.fr/jquery-virtual-tour/)。

**![panorama](../Images/d3129f92f8225fb8aa945d6eb515c71e.png "panorama")T2】**

**[发光字母](http://www.openstudio.fr/Des-enluminures-avec-jQuery.html?lang=en)** 也是由开放工作室制作的。他们创造了 360 虚拟旅游插件。

这个插件允许你给你的文本添加一个装饰性的首字下沉。首字下沉是插件提供的 zip 文件中的图像。我想，如果您想使用不同的样式，用您自己的字母图像替换提供的图像是相当简单的。
**T2】**

**![illuminatedLetters](../Images/2c648af95ac9c42c1ae17004b472a718.png "illuminatedLetters")T2】**

[在网站上下载并查看工作实例](http://www.openstudio.fr/Des-enluminures-avec-jQuery.html?lang=en)。顺便提一下，这个页面上的大部分信息都是法语的。

jCrop 让用户选择一幅大图像的一部分，并将其裁剪成第二幅较小的图像。这将是理想的情况下，如创建一个头像或缩略图图像。

[此处演示。](http://deepliquid.com/content/Jcrop_Examples.html)

[下载这里](http://deepliquid.com/content/Jcrop.html)。

![jcrop](../Images/7e5cfe1d8525bcb953115506488524d1.png "jcrop")**T2】**

**[图像注释](http://code.google.com/p/jquery-image-annotate/)** 让您选择图像上的热点，您可以在其中添加工具提示或注释，它们会在鼠标滑过时显示。如果你使用 Flickr，你可能已经无数次看到过这种效果。

[在这里演示。](http://www.flipbit.co.uk/jquery-image-annotation.html)

[在此下载。](http://blog.flipbit.co.uk/2009/03/jquery-image-annotation-plugin.html)

**![imageannotation](../Images/ab9a312891c3cdd3b7cd3ed5d1693e8a.png "imageannotation")T2】**

**[jCaption](http://www.newmediacampaigns.com/page/jcaption-a-jquery-plugin-for-simple-image-captions)** 插件让你轻松地给图片添加标题。就这么简单。
[在这里演示。](http://www.newmediacampaigns.com/files/posts/jcaption/index.html)

[在此下载。](http://www.newmediacampaigns.com/page/jcaption-a-jquery-plugin-for-simple-image-captions#download)
