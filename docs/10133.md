# 支撑客户:你的面包和黄油

> 原文：<https://www.sitepoint.com/underpinning-clients-your-bread-and-butter/>

如果你不得不从零开始为你的企业赢得每一个新项目，而你没有任何持续的工作，你会发现你自己工作在追逐你的尾巴模型。这意味着将你所有的注意力集中在交付客户的项目上，然后突然意识到一旦你完成了，你需要再卖一份工作来维持业务。总是盛宴或饥荒。怎么才能避开这个陷阱呢？通过拥有一系列的常客，重复*来支撑客户*。

当我接手一家中等规模的网站设计业务时，我回顾了客户群，发现一个客户就占了该业务收入的 33%。虽然这个客户很大，并且已经建立了很好的关系，它为我们赢得了很多媒体报道，但我认为这个客户是我们最大的风险。如果他们离开我们，生意就真的有麻烦了。

当然，作为一个大组织，它总是被广告公司和其他想分一杯羹的网络公司接洽。此外，因为它是一个大客户，所以存在难以控制的额外风险。

### 客户越多，风险越小

虽然我不想减少我们从那个客户那里获得的业务，但我确实想扩大我们的基础客户群。在接下来的几个月里，我们继续很好地服务这个客户，但我们开始建立其他的基础客户，以提供更大的稳定性和更强的基础，使我们的新网络业务摆脱这种危险的境地。

基础客户是一个稳定的客户群的一部分，他们为你提供定期、持续的工作，让你支付租金，控制工资。这也给了你花更多时间赢得工作的机会。你花在销售基本工作上的时间更少了，你可以把时间投入到执行企业的战略任务上。简而言之，基础客户为未来的稳健增长提供了基础。

### 建立稳固的关系

当你为任何客户完成第一个项目时，你都应该评估他们的需求，并寻找与他们合作的其他方式。这些客户中的一部分——但不是全部——将继续与你合作，他们很可能成为你业务的重要基础客户。对于那些这样做的人来说，你有了解客户和他们的业务的优势，你有信心你们可以很好地合作，互相依赖。

毫无疑问，你还会从这些客户那里收到很好的推荐信——这一点至关重要。当另一个潜在客户想和你现有的客户谈论你的工作时，你可以把他们介绍给你的基础客户，一定会得到一个极好的评价。渐渐地，你会扩大你的基础客户群，为你的企业未来的发展打下基础。

那么，哪些客户最有可能成为你的基础客户呢？他们通常有一种将工作外包给专家的组织心态，以及持续、复杂的需求。你知道哪些公司符合这个要求吗？也许你应该联系一下。
