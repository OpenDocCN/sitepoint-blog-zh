# 你的在线社区在哪里？

> 原文：<https://www.sitepoint.com/wheres-online-community/>

“建立一个社区，你就建立了一个企业。”这是网络空间中经过验证的新商业模式。

Hotmail.com，Geocities.com，ICQ.com。这个清单还在继续。这些公司(以及数以千计的类似公司)基于这种模式建立了繁荣的在线业务。

这真的很简单。如果足够多的“网民”聚集在你的网站上，找到一种方法来创造收入是很容易的！

你可以向他们出售你的产品或服务。你可以邀请其他人向他们推销，并收取一定的百分比。你可以收取广告费。你甚至可以发明自己的收入模式。整个想法是这样的:当你创建一个在线社区时，一个坚实的客户基础就会随之而来。

现在，让我们把事情缩小一点。我意识到，和我一样，许多小企业主没有创建自己的 Hotmail 或 Geocities 商业模式所需的资金。但这并不是说你不能利用社区建设模式！

这里有一个主要的例子…

访问 http://bizweb2000.com/wwwboard，你会看到这些“缩小”的社区之一。那个网址是我的“网络营销信息板”的地址。这只是一个自我推进的社区，任何人都可以去学习互联网营销。有问题吗？在那里问它。有解决别人困境的方法吗？把你的答案贴在那里。

也许一个公告板会很好地配合你的网站？毕竟，一个“问答论坛”可以很好地适应任何主题，而且它会成为一个很好的社区！如果你在寻找网站的回头客，你不会找到更好的方法来获得他们。

不过，在你一头扎进去之前，在你的网站上建立一个公告栏之前，这里有几个建议需要考虑。

**1。等到你的网站有了相当大的流量后，你才开始营业。一块非常慢的板子可能比没有板子更糟糕。如果访问者停留在你的留言板上，并且在很长一段时间内只有少量的帖子，他们可能不会再回来。**

**2。选择一个简单的公告板界面**，就像我在我的公告板上使用的一样。它不仅易于导航，而且根据需要调整和删除帖子也很简单。(下一页将详细介绍如何选择接口。)

**3。不要试图独自经营你的董事会。**像我在我的董事会上所做的那样，雇佣一个合格的小型版主团队。知道网络价值的专家会很乐意接受一个主持人的职位。我自己的版主每周花一个小时左右的时间进行交易，以获得知名度和带来的新业务。

好了，你准备好搭建你自己的板子了吗？以下是实现这一目标的一些细节…

网上有许多免费的“公告板 cgi 脚本”。我的公告栏 cgi 脚本来自马特的脚本存档:[http://www.worldwidemart.com/scripts](http://www.worldwidemart.com/scripts/)

你可以去那个网址下载马特的 WWWboard，或者，如果你喜欢不同的风格，更多的公告板脚本可以在:[Cgi-Resources.com](http://www.cgi-resources.com/)

一旦你确定了你想要的脚本，你需要把它安装在你的服务器上。这需要修改用脚本下载的文件，并将它们上传到 web 服务器的正确目录中。由于大多数脚本各不相同，所以最好阅读脚本附带的“readme.txt”文件。它将详细说明安装的所有细节。

一旦你的冲浪板竖起来了，就要每天维护它。

不要允许在你的董事会上做明显的广告或自我推销。这使得人们懒得再去访问一个董事会。你可能想发布一个“禁止广告”的声明，并通过尽快删除不受欢迎的帖子来监督你的董事会。

至于哪些帖子被删除，哪些留在你的留言板上，你有最终决定权。毕竟你是版主。这是你的社区。明智地对待它，它会让你得到数倍的回报！
