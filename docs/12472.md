# 歌剧 10 阿尔法到达

> 原文：<https://www.sitepoint.com/opera-10-alpha-arrives/>

![](img/dfe06dd2c3fe573b9dd87ba3cb6d9584.png "opera-logo") [根据网络应用](http://marketshare.hitslink.com/browser-market-share.aspx?qprid=0)的数据，只有大约 0.71%的网民使用 Opera 浏览器。但是根据谷歌分析，使用 Opera 的 SitePoint 用户数量是这个数字的三倍(显然是一群比一般网络大众更开明的人)。今天 Opera [宣布发布适用于 Mac、Windows 和 UNIX 的 10.0 Alpha 1 版本](http://my.opera.com/desktopteam/blog/2008/12/03/peregrine-takes-flight-opera-10-0-alpha-is-here)，距离他们上一个稳定版本 9.62 发布仅一个月左右。

新版本包括他们的核心 Presto 渲染引擎的更新版本。Presto 2.2 获得了改进的 CSS 性能，并在 [Acid3](http://acid3.acidtests.org/) web 标准测试中获得了 100/100 的分数。新的 Presto 引擎还配备了一个改进的正则表达式引擎，Opera 表示这将从整体上加快浏览器体验。

Opera 10 也有一些其他浏览器已经有一段时间的追赶性改进，包括内嵌拼写检查(如 Firefox)，在没有用户通知的情况下自动更新的能力(如 Google Chrome)，以及对 Opera Mail 的改进，允许用户发送富文本电子邮件，并允许他们在指定的天数后从 POP 服务器删除电子邮件。

Opera 10 Alpha 1 可以[在这里](http://my.opera.com/desktopteam/blog/2008/12/03/peregrine-takes-flight-opera-10-0-alpha-is-here)下载。
