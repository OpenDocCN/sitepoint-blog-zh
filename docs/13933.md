# 为什么免费咨询不起作用

> 原文：<https://www.sitepoint.com/why-free-consultations-dont-work/>

许多专业人士为他们的客户提供免费咨询或免费试用。

这些通常不作为初始报价。为什么？因为人们持怀疑态度。他们一开始就不认识你，所以:

1.我想你会做一次推销(你确实是这样做的)。

2.不想让你介入他们的生意，因为他们不了解或不信任你。让某人进入他们的私人和个人事务的负面风险太高了。

用对他们来说风险较小、不那么亲密的东西来展示你的价值要好得多——比如一份解决他们更迫切的问题(也是你解决的问题)的免费报告。

作为一个有点相关的题外话，我昨天去雷蒙德·詹姆斯体育场看坦帕湾海盗队和海豚队的比赛。弗雷斯卡公司向粉丝们提供免费的汽水样品。现在，天气很热，而且离比赛还有一段时间，所以你会认为每个人都会拿一罐免费的汽水。不是真的。我数了一下，大概有 25-33%的人喝了免费的苏打水。其余的人走过去了——忙得没有时间回应，对这样的好意表示怀疑，或者就是不感兴趣。

不管怎样，弗雷斯卡人得到了很好的曝光率，并让一群人尝试他们的苏打水。

如果提供免费试用，专业人士的回复率会低得多。

这是另一个例子:

当我刚开始从事咨询行业时，我所在的当地商会举办了一场贸易展，以突出当地企业。他们需要免费的门奖。我提议给一位幸运的获胜者提供 3 小时的高管培训。商会主席拒绝了我，原因与上面提到的相同。她告诉我，没有人想要免费的专业服务，因为他们需要了解提供服务的人——尤其是当我们处于一个没有认证的领域。

所以你走吧。
