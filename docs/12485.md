# 微软的坏消息:Windows，IE 网络使用率下降

> 原文：<https://www.sitepoint.com/bad-news-for-microsoft-windows-ie-net-usage-dip-lower/>

就在电视节目《辛普森一家》*的《T2》[滑稽地嘲笑苹果公司](https://www.engadget.com/2008/12/01/the-simpsons-mocks-m-apple/)的一天后，这家位于加州库比蒂诺的公司笑到了最后，因为其操作系统的互联网份额达到了历史最高水平，而 Windows 的份额则跌至历史最低点。根据苹果 2.0 博客，Mac 的网络点击率几乎达到了 9%，创下了 11 月份的记录。*

与此同时，Windows 操作系统的互联网使用率首次跌至 90%以下。在网络应用追踪调查中，ie 浏览器也首次跌破 70%对 69.78%的心理关口。尽管 Windows Vista 取得了收益，但这不足以弥补 Windows XP 操作系统在互联网份额上的损失。

与此同时，IE 的下滑可以主要归因于 Firefox 浏览器的增长，该浏览器本月首次超过 20%。据 Net Applications 称，Firefox 在 11 月份出现如此巨大的增长，是因为该浏览器越来越多地被居民用户使用，而且这个月挤满了商业用户在家上网的日子:美国大选、感恩节假期和额外的周末。

虽然 Net Application 的净使用数据并不能真正衡量传统意义上的“市场份额”，但这些数据应该会给微软敲响警钟。即使苹果的收益可以归因于苹果用户比 Windows 用户更多地使用网络，而不是市场份额的实际转移，考虑到未来的操作系统可能是基于网络的，这对微软来说仍然是个坏消息。
