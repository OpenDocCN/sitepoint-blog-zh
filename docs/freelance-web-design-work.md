# 寻找自由网页设计工作？

> 原文：<https://www.sitepoint.com/freelance-web-design-work/>

如果你是一名独立的专业人士，你就会知道找到下一个项目有多难。

有几种方法可以找到合同工作，包括推荐和重复客户。但是当你停滞不前，没有工作等着你，或者只是想扩大你的全球影响力时，可能是时候转向网络了。

在这篇文章中，我回顾了三个非常不同的寻找独立合同项目的网站。首先，我考察了 Guru.com 寻找短期异地或现场工作的人，ComputerJobs.com 寻找与大公司长期合同的人，elance.com 寻找从小型到大型项目的个人。

##### [Guru.com](http://www.Guru.com)

Guru.com 是最受欢迎的求职网站之一，拥有超过 40 万注册用户。Guru 提供了一个有组织的易于使用的界面，有两种浏览工作的方式，搜索功能和一个类似雅虎的目录。使用高级搜索，你可以根据工作类型、地点和时薪找到工作。如果你不确定你要找什么，或者如果你想扩大搜索范围，请使用目录。

Guru.com 像许多其他求职网站一样，当与您的搜索标准相关的新列表被添加到网站时，会向您发送电子邮件，这一功能可能很有用，但您应该至少每隔一天监控该网站。

当你在 guru 注册时，你需要创建一个个人资料。个人资料包括你的技能清单，概述，教育，经验和参考，以及对你理想工作的简要描述。除了概述，你的个人资料中的所有内容都是可选的，但如果你想在其他 40 万名大师中脱颖而出，你需要尽可能包括所有信息。

完成个人资料后，你可能需要搜索你在标题和描述中列出的关键词，看看雇主会如何看待你的个人资料。如果你正在寻找各种大大小小的项目，Guru.com 值得你的关注。

##### [Computerjobs.com](http://www.ComputerJobs.com)

如果你不介意在大公司现场工作，ComputerJobs.com 是你最好的资源。新的工作岗位不断增加，正如我提到的，大部分是大公司和招聘人员，他们希望找到长期合同和永久工作。

如果你决定在这个网站上申请工作，准备好经历一个漫长而严格的过程，原因？招聘人员通常会对他们的候选人进行更彻底的筛选和面试。

ComputerJobs.com 网站很容易使用，通过高级搜索功能，找到你想要的东西又快又容易。建立你的简历很简单，但是简历系统存在一些问题，比如它限制你只能列出八项技能。幸运的是，每份工作清单上都提供了电子邮件地址，这样你就可以直接向你的潜在雇主发送一份更详细的简历。

如果你提供顾问服务，ComputerJobs.com 允许你在他们的顾问角免费列出你的服务，但有超过 6000 名顾问列出，你应该尝试给你的服务添加一个独特的旋转或找到一个利基。

总的来说，ComputerJobs.com 是寻找企业项目的资深专业人士的绝佳资源。

##### [eLance.com](http://www.eLance.com)

eLance.com 是一个独特的网站，允许你对个别项目进行投标。你可以从几个类别中选择，包括页面设计和图形设计。在所有类别中，竞争都非常激烈，每分钟都有投标。投标通常会持续几个星期，所以你需要耐心，经常检查你的出价状态。

所有的出价在 eLance.com 都是公开的，这意味着不仅你的出价可以被所有人看到，你发给买家的信息也可以被看到。如果有电子邮件地址，你最好通过电子邮件私下交谈。

如果你有时间和耐心，整个过程可能会非常令人兴奋和有教育意义，但如果你决定竞标一个项目，要非常小心，因为有时项目细节非常粗略。不知道完成这个项目需要什么，你可能会在不知不觉中深陷其中。再次电子邮件买家，如果你有任何问题，然后把你的教育出价。

当你开始独立承包人的职业生涯时，记住这些在线网站不应该取代你的线下工作。没有一个网站可以取代快乐回头客或向他人推荐你的客户。也就是说，在线网站可以扩大你与全球客户的联系，同时让你接触到新的长期客户关系。
