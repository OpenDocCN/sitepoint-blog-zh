# 信息孤岛对网络不利

> 原文:[https://www . site point . com/information-silos-is-bad-for-the-web/](https://www.sitepoint.com/information-silos-are-bad-for-the-web/)

上周我们写道，如果计算的未来存在于像脸书和 iPhone 这样的封闭网络平台上，这对用户和整个网络来说都是不利的。同样的事情当然也适用于网站——封闭的信息孤岛对网络不利。

今天，Tim O'Reilly 想知道自我链接是否正在成为网络的标准。奥赖利指出，像 [TechCrunch](http://www.techcrunch.com/) 和 [New York Times](http://www.nytimes.com/) 这样的媒体网站越来越少链接到他们自己的地产线之外。例如，TechCrunch 链接到它拥有的一个名为 [CrunchBase](http://www.crunchbase.com/) 的网站，以获取它所覆盖的公司的更多信息。

据我所知，gadget blog [Engadget](http://www.engadget.com/) 开创了许多博客现在使用的内部链接策略，通过这种策略，链接几乎总是指向网站上的其他内容，而不是将用户发送到网站的其他部分。在[这篇文章](http://www.engadget.com/2008/08/18/google-takes-whitespace-to-the-people-with-free-the-airwaves/)中，例如，关于谷歌的“免费电波”倡议，Engadget 链接到他们网站上“空白”的标签搜索和三篇以前的文章(两篇链接自组织名称，而不是直接链接到那些组织)。谷歌关于这个主题的博客文章的唯一链接——可以说是最重要的链接——出现在他们文章的最底部，有一个小小的“已读”链接。

内部链接在大型媒体公司也很常见，甚至在搜索引擎上也是如此。例如，谷歌最近被发现在英国给予其[商家搜索](http://www.google.co.uk/MerchantSearchBeta/search)房屋贷款比较网站[特殊待遇](http://blogoscoped.com/archive/2008-08-12-n81.html)。几年前，他们在美国尝试了一种类似的叫做“谷歌提示”的东西，这种东西首先会推送指向谷歌内容的链接。(Tips】在推出后几周就撤下了。)

这种内部链接的原因很容易理解:你越能让人们留在你的网站上，你就越能通过向他们提供广告来赚钱。但它对 web 来说也是危险的，因为它会导致信息孤岛的产生，这些信息孤岛通过将用户暴露于一个观点和数据源而降低了 web 对用户的效用。

“当这种趋势蔓延时(我说的是‘当’，而不是‘如果’)，这将是对网络效用的征税，必须通过中间页面的效用来平衡，”O'Reilly 说。“如果它们真的很好，有很多你在别处不容易找到的有用的、经过筛选的数据，这可能是一种可以接受的税收。事实上，它们甚至可能是有益的，并且是增加网站对读者价值的真正方式。如果它们纯粹是为了获得额外的点击，它们将是网络基本货币的退化，就像黑帽搜索引擎页面从搜索引擎结果中构建链接农场一样。”

奥赖利列出了两条规则，供任何人在链接到自己网站上以前的内容或特别制作的内容之前考虑:

> 1.  Make sure that no more than 50% of the links on any page are for yourself. (Even this number may be too high. )
> 2.  Make sure that the pages you create in these destinations are indeed more valuable to your readers than any other external links you may provide.

我自己对 SitePoint 上的博客文章的链接政策(改编自我在那里的时候读写网的政策)是这样的:

1.  总是尽可能多的链接资料，为读者提供更多的阅读材料，并加强你的论点。
2.  如果文章提供了你要找的上下文，只做内部链接(也就是说，不要仅仅为了给你的内容一个链接而链接到你自己——只有在有意义的时候才这样做)。
3.  永远不要链接到关于一家公司的文章，而不是公司本身——读者不想点击一家公司的名字并被带到以前的报道，他们点击那个链接是为了进入该公司的网站，所以让他们去那里。(对于更大的公司，比如谷歌，或者只是顺便提及的公司，我可能会完全放弃链接。)

你做了多少内部链接对于任何一个网站来说都是很重要的。信息孤岛对网络不利，网站应该尽可能地互相链接，而不仅仅是链接到自己。
