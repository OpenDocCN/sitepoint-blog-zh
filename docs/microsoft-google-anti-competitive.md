# 微软指控谷歌反竞争行为

> 原文：<https://www.sitepoint.com/microsoft-google-anti-competitive/>

是的，你没看错标题。微软副总裁兼副总法律顾问戴夫·海纳(Dave Heiner)在谷歌暗示欧盟反垄断调查是由与微软有业务联系的公司发起的之后，在发表了一篇[官方博客文章。](http://microsoftontheissues.com/cs/blogs/mscorp/archive/2010/02/26/competition-authorities-and-search.aspx)

> 谷歌对这种日益增长的监管担忧的公开回应指向了其他地方——微软。谷歌告诉记者，对搜索的反垄断担忧是不真实的，因为一些投诉来自其仅存的搜索竞争对手之一。
> 
> 值得一问的是，谷歌的回应是否真正解决了人们提出的担忧。竞争法案件中的投诉*通常*来自竞争对手。

这是绝对正确的。如果欧盟认为有理由进行调查，谷歌应该严肃对待这些指控，并进行彻底的内部调查。编造阴谋论不能回答问题，也不会打动监管者。仅仅因为你很大，很赚钱，提供免费产品，并不意味着你不是*【作恶】*。

海纳的文章继续批评谷歌，并建议受影响的公司联系竞争主管机构:

> 近年来，随着谷歌实力的增长，我们越来越多地听到各种公司——大大小小——对谷歌各种商业行为的抱怨。一些投诉只是反映了谷歌咄咄逼人的商业姿态。有些反映了谷歌在许多领域运作的隐秘性。有些似乎提出了严重的反垄断问题。如你所料，许多相关公司已经找到我们，询问我们的反应，甚至是建议。当他们的反垄断问题看起来很严重时，我们建议公司与竞争法机构对话。

微软公开诽谤是不寻常的。这不符合公司文化——当你身居高位时，你不需要削弱竞争对手:

*   微软仍然是最具优势和盈利能力的 IT 公司之一，主要是因为 Windows 和 Office 的持续收入。
*   他们通过生产更好的产品、降低成本或收购公司击败了危险的竞争对手。
*   当每个人都是你的客户时，你不能批评别人。我确信谷歌会购买微软的软件——即使只是为了评估功能。

然而，来自谷歌的威胁是不同的；微软正在努力竞争，还没有从在线投资中获利。尽管谷歌尚未侵占他们的领地，但 Chrome OS 和 Docs 有可能扰乱微软的核心收入基础。

因此，如果微软正在考虑卑鄙的活动，这是可以理解的。但是反竞争行为的指控呢？我猜他们一定是那个领域的专家！

谷歌已经成为垄断和反竞争了吗？微软有义务公布这些投诉吗？还是两家公司都应该成长起来，闭上嘴？！
