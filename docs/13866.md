# 从一个词中获利

> 原文:[https://www.sitepoint.com/profit-off-a-single-word/](https://www.sitepoint.com/profit-off-a-single-word/)

许多人在考虑如何赚钱的时候会停止搜索横幅广告。他们可能在顶部有一个横幅，在右边有一个摩天大楼，在底部有另一个横幅，晚上睡觉时认为他们正在尽一切努力从他们的网站上获利。事实上，要做的事情还很多。

现在，你可以在网站上使用的横幅数量既受到良好品味的限制，也受到可用屏幕空间的限制。如果你希望你的网站适合 800×600 的分辨率，并且你的内容需要 700 像素的水平空间，那么除了一座瘦瘦的摩天大楼，你没有太多的选择。

因此，要增加你的每次页面浏览利润，你需要跳出框框思考:不是横幅，而是文字。我之前讨论过[出售文本链接](https://www.sitepoint.com/selling-text-links/)的业务，这是一种可行的方法，可以在占用很少屏幕空间的情况下将你的网站货币化。你也可以使用文字链接从从属节目，这是一个完全不同的话题，不是我今天想讨论的。今天我想讨论我最喜欢的两种把单词变成钱的方法。

我要提到的两种方法都是通过将内容中的一个单词(或短语)转化为指向网站上另一个页面的链接来实现的。在第一种方法中，该页面是来自像 [SearchFeed](http://www.searchfeed.com) 或 [RevenuePilot](http://www.revenuepilot.com) 这样的公司的搜索列表结果集。这些程序的工作方式类似于[谷歌广告链接](https://www.google.com/adsense/adformats#adlinks)和[谷歌广告搜索](https://www.google.com/adsense/ws-overview)的组合。基本上这些公司有他们自己的点击付费列表数据库，他们允许你的访问者在你的网站上搜索它们。它们还允许你直接链接到你选择的关键词的搜索结果集，这是谷歌不允许的。因此，虽然这些程序没有你期望从谷歌得到的回报，但这一点点津贴让它们变得非常强大。

原因很简单，你可以从你的内容中提取一个单词或短语，比如虚拟主机，然后把它变成一个链接，指向虚拟主机上的 PPC 列表。当你的访问者点击这些列表时，你就可以分享收益。这是我所知道的从网站上赚钱的最简单的方法之一，不需要牺牲任何屏幕空间。这不会是一大笔钱，但在足够大的范围内，这种做法可以为你的网站提供可观的额外收入。

我喜欢使用的另一种方法是做大致相同的事情，但不是链接到 RevenuePilot 或 SearchFeed 的页面，而是链接到 Ebay 列表的页面。Ebay 允许你使用为他们的联盟计划创建的工具 [Editor Kit](http://affiliates.ebay.com/tools/editor-kit/) 基于搜索词制作他们的列表页面，乍一看，当注册 [Ebay 的联盟计划](http://affiliates.ebay.com/program-details/)时，它看起来不像你能赚很多，但这可能是骗人的，而且即使不是很多，也不像你使用大量像素来推广它。这种方法当然最适合购物相关的术语，或者特别是易趣上经常出售的东西，所以它并不适合所有的网站。

我提到的这两种方法都是增加收入的好方法，而不需要在屏幕上塞进更多的横幅广告。我真的建议你尝试在你的网站上实现一个或两个功能。
