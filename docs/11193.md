# Opera 10.5 是有史以来最好的浏览器吗？

> 原文:[https://www.sitepoint.com/opera-best-browser-ever/](https://www.sitepoint.com/opera-best-browser-ever/)

Opera 拥有最快的浏览器发布时间表，10.5 版本现已推出——正好赶上微软的欧盟浏览器选择界面。你需要从 Opera.com 下载，尽管之前版本的自动更新很快就会出现。

Opera 10.5 是一个重大的更新——其他浏览器厂商会提高主要版本号。以下是您可以预期的结果:

*   出色的 W3C 标准支持
*   私人浏览模式
*   本地小部件——独立的应用程序，在浏览器不运行时仍能继续工作
*   一个新的界面
*   一个新的 JavaScript 和图形引擎——Opera 声称它是世界上最快的浏览器。

这最后两个方面是最大的变化…

## 惊人的冲浪

Opera 一直是更好看的浏览器之一，但乔恩·希克斯让 10.5 看起来很棒。最明显的变化是类似 Chrome 的标题栏和菜单的移除。

![Opera screen](../Images/f89e70ee505e48bca75ef9199164a9a4.png)

如果需要，可以点击红色的“O”图标来访问最常用的选项并显示菜单栏。

![Opera screen](../Images/1d90e477986334404c99ab77a34083b8.png)

整个界面可以根据您的喜好进行配置。可以添加和删除图标，可以重新排列面板，甚至可以改变窗口颜色。

它也是第一个提供完全集成的 Windows 7/Vista 体验的非微软浏览器，支持 Aero、跳转列表和任务栏标签。这是一个受欢迎的附加功能，许多用户将会欣赏它的生产力优势。

最后，Opera 去掉了恼人的模态对话框，代之以页面覆盖或面板。“在页面中查找”是我见过的最好的——它淡化了页面内容，突出了单词的所有实例。我预计竞争对手的浏览器将很快复制这一功能。

![Opera screen](../Images/957bdf0adff5f5d401abaa86d779691a.png)

## 卓越的速度

尽管谷歌 Chrome 以速度著称，但在过去一年中，它似乎超过了 Opera。

Opera 对他们的新 Carakan JavaScript 引擎做出了大胆的声明。我用 SunSpider 基准测试软件对它进行了测试，Opera 名列前茅。它只比 Chrome 快 1%，但比 Firefox 3.6 快两倍多。JavaScript 基准测试并不能说明全部情况，您的体验可能会有所不同，但引擎无疑是最快的。

每个供应商都声称他们的浏览器是最快的——甚至是微软——并且可以通过利用已知优化的测试来提供*“证明”。然而，在[汤姆的硬件](http://www.tomshardware.com/reviews/firefox-chrome-opera,2558.html)，可以获得对前五名浏览器的最佳独立评论之一。总而言之:*

 **   Opera 和 Chrome 名列榜首，但 Chrome 以更多的第二名遥遥领先。
*   Opera 在处理 Flash、Silverlight、Java 等插件时特别快。YouTube 页面加载速度几乎是 Firefox 的两倍，是 Chrome 的三倍。
*   Opera 的单个标签页的内存使用量更高，尽管随着更多标签页的打开，这种差异会减少。

不过，别管那些图表了。大多数测试中的浏览器边距都是以毫秒为单位来衡量的，差异是无法察觉的。重要的是在你的机器上浏览器的速度有多快。根据我的经验，Opera 不仅仅是 Chrome 的对手，它还提供了更多的功能。

Opera *是不是“地球上最快的浏览器”*？我找不到理由来反驳他们的要求。

## 直接切换？

好吧，歌剧很快。看起来棒极了。它比任何竞争对手都有更多现成的功能。它提供了优秀的开发工具。标准支持非常好，通常是第一个通过 ACID 测试的。几乎浏览器的每个方面都可以定制或配置。

然而，它不是我的默认浏览器。Opera 保持着 2%的市场份额，在过去的十年中几乎没有变化。继续阅读 [*歌剧的问题……*](https://www.sitepoint.com/the-problem-with-opera/)
