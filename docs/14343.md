# 使用 Illustrator 或 Freehand 创建 XP 风格的图标

> 原文:[https://www.sitepoint.com/create-xp-style-icons/](https://www.sitepoint.com/create-xp-style-icons/)

在我们矢量图形系列的第三篇也是最后一篇文章中，我们将一步一步地介绍一个现实生活中的实际应用，我想你会发现它非常有用。

您将学习如何制作自己的 Windows XP 风格的作品，最终可能会用于网站图形、软件应用程序图标或其他项目。

*注意:我将使用 Adobe Illustrator 8 制作我的图表和命令，但是那些使用 Freehand 的人应该也能很容易地理解。*

让我们从解构一个熟悉的 XP 图标开始——此处显示的平板显示器:

![1418_1](../Images/298ffed34d70582dd2e38affddf79bb7.png)

首先，使用钢笔工具为屏幕创建一个矩形。如果你对如何用透视画法有一点了解的话，这是很有帮助的(如果你想了解更多，请参见本文末尾的链接)。

![1418_2](../Images/fa157a8955e074c69dc81d073cb1685d.png)

用白色到浅绿色的放射状渐变填充形状。

![1418_3](../Images/671d0d851523e6a359f3f6de183c3cbb.png)

使用渐变工具，在形状的中上边缘附近单击并按住，然后向下拖动，在到达形状底部之前释放鼠标按钮。

![1418_4](../Images/dbebf356fc934178b4ecccf29c6ad81f.png)

现在，我们将在屏幕周围创建一个圆边形状，作为显示器的框架。将形状的填充设置为“无”以帮助您看到您正在做的事情，然后使用钢笔工具在屏幕周围用圆角绘制一条边。

![1418_5](../Images/6eb664d8fbfd63e0b76fb63e0eb16fc5.png)

如果形状看起来像你喜欢的样子，用灰色到紫色的渐变填充它。

![1418_6](../Images/73899169eeca7690f4f44f7d13c8c56d.png)

右键单击该形状，然后选择排列>置于底层。这将允许蓝屏形状位于顶部。

![1418_7](../Images/903879040fe6947ad0ae988c34c85651.png)

现在，使用钢笔工具沿着屏幕的顶部和左侧边缘创建一个圆形的“L”形状，并用紫色到白色的渐变填充它。使用渐变工具填充形状，从左上角的白色，使它看起来像光来自那个方向。

![1418_8](../Images/3841ca77ba2353cbcb4c04a614ae0273.png)

使用钢笔工具沿着屏幕底部创建一个高亮显示的边缘。不要害怕放大和使用白色箭头工具来调整边缘。我们也用同样的白色到紫色渐变填充它。

![1418_9](../Images/d337fa14310703fa0749620694eed058.png)

屏幕部分完成了！现在，对于立场…

支架基本上由两种形状组成，即前表面和侧表面。首先，创建前表面的形状。我用灰色到紫色的渐变填充它。

![1418_10](../Images/c251459474ce8d0417ea7bbf05867553.png)

接下来，创建侧边形状，并用紫色到白色的渐变填充它。从上到下，渐变从暗到亮，这使得光线看起来好像是从那一侧照射到显示器上，但顶部被屏幕稍微遮挡了一下。

![1418_11](../Images/952857c60e2803239d7397b969a63357.png)

最后，为形状创建一个“边界”。这个新形状在支架的左侧和底部稍微突出，但与支架的右边界对齐。

![1418_12b](../Images/b7cd27548bb9f7fe3d96b51c43ffba54.png)

当您填充此形状(用深紫色或紫色到灰色的渐变)时，它将隐藏组成支架的一对形状。

![1418_12](../Images/a6e55726871fb49db9ae814862112446.png)

同样，右键单击形状，然后选择“排列”>“置于底层”,将“边框”置于展台形状的后面。

![1418_13](../Images/752b5b94652ec4ae8d96ea76def91ab5.png)

最后，使用椭圆工具，创建基地的立场，填补了浅白色到淡紫色填充。

![1418_14](../Images/fb037a5f704608b764953007e905ef1b.png)

再次“放在后面”将底座放在所有其他对象的后面。

![1418_15](../Images/f71dda96e03b726a5be7a0a690271c8b.png)

选择缩放工具。

![1418_16b](../Images/d361e7754b9178de1544a3fc5acb251f.png)

在基础仍处于选中状态的情况下，单击并按住鼠标按钮将其稍微缩小。按住 Alt 键(Mac 的选项),同时稍微向内拖动并放开鼠标按钮。这是缩小形状的副本。现在你应该有两个椭圆，一个在另一个里面。

![1418_16](../Images/c745087b5b7fe2f4802112a140adb543.png)

选择外部形状，并用较暗的灰色到紫色渐变颜色填充它。

![1418_17](../Images/c3ef017a67dd227018e777b497b3f1cd.png)

现在你有了它:一个平面显示器的矢量版本，看起来令人不安地像 Windows XP 显示器图形，使用了九个单独的形状和大量的渐变。

![1418_19](../Images/47468d72184ef6dc7d199d4d3c1c5704.png)

现在，让我们把这个形状放入 Photoshop，并添加一个阴影，使它看起来“正式”的 XP 风格。选择整组对象并复制。

![1418_18](../Images/0554635deb80ac280049471a50f2b8b1.png)

将形状粘贴到 Photoshop 文档中(“像素”)，根据需要调整大小，然后前往“图层”>“图层样式”>“投影”。

![1418_20](../Images/74b899d77c14a78d39e556d1bc4c9e08.png)

将投影的角度设置为 135 度。将距离和大小设置为 2 或 3 个像素，并根据图形的大小进行调整。

![1418_21](../Images/abf842d59359d7435f68af5dbf7bdad1.png)

这就是你拥有的:一台 XP 风格的电脑显示器！

![1418_22](../Images/324aad93cd2afcf29c1b580df1c3f779.png)

现在，创建和制作你自己的 Windows XP 风格的图形需要更多的艺术技巧。特别是，你必须熟悉透视画法，而且你必须能够把你想画的对象“简化”成小尺寸或大尺寸的图标或图形。

第一步是熟悉[微软创建 Windows XP 图标的风格和设计指南](http://msdn.microsoft.com/library/default.asp?url=/library/en-us/dnwxp/html/winxpicons.asp)。这为你提供了一些关于“外观”的指导(“乐趣、颜色和能量”)，提供了一个通用的调色板(复制如下)，并给出了物体应该如何角度和分组的例子。

![1418_gif](../Images/e66e8b9c763c8b1f674665ec87281709.png)

Windows XP 图标中使用的调色板。

你喜欢阅读风格指南吗？然后，让我们从最后一步一步的例子开始，在这里我将向你展示我如何在 Windows XP 风格下制作一个狗屋图形。

##### 在狗窝里！

首先，我根据微软提供的角度指南，画出一个我想要的图标的草图。

![1418_23](../Images/74cdd63f3d048de8ae4ad13b099a2c46.png)

然后，我从微软风格指南中创建一个与透视网格相匹配的线条网格，并使用钢笔工具创建狗窝的正面。(您可以[下载一个具有相同网格](https://www.sitepoint.com/examples/xpicons/grid.zip)的 Illustrator 文件。)

![1418_24](../Images/277881d2338f6f964784cd55fb34ef05.png)

一旦我有了狗屋的正面，我隐藏网格(视图>隐藏参考线)并使用钢笔工具绘制狗屋的侧面:

![1418_25](../Images/1ecf23d78ba96cd0a11984fe940fad1b.png)

我用红色到暗红色的渐变填充侧面和前面:

![1418_26](../Images/fd3fcbe29fa7625d2cb9585453b057bc.png)

然后我为屋顶又画了两个形状，用更暗的红棕色渐变填充。第二个形状(右图)是屋顶的背面；如果你能“看穿”房子，它就是这个形状。它在图中看起来有点奇怪，因为它位于狗屋所有其他部分的顶部。

![1418_27](../Images/cfc6706440de0619efd85a9a0f730273.png)

因此，我右键单击第二个形状(“后”屋顶)，然后选择“排列”>“置于底层”。

![1418_28](../Images/a46b5239a2ced3b28ef513a6d9ed96a1.png)

这给了我一个漂亮的狗屋形状。

![1418_29](../Images/a4964c7ceb6790c7424b7c50dc773113.png)

我再次使用钢笔工具创建一个门的形状，并用较暗的渐变颜色填充它。

![1418_30](../Images/e640e0cedf93608a6c3baed1620546ca.png)

然后，我使用钢笔工具——设置为“无填充”——在整个形状周围画一个轮廓。我用更暗的红棕色渐变填充它，以获得形状的“边界”。

![1418_31](../Images/7e2cb4e5db94cb9451ba56bc8fcdac65.png)

由于这个新形状位于其他所有形状之上(并隐藏了图像的其他部分)，我像以前一样将它放在后面。

![1418_32](../Images/36bc57959d9bd0e50c43f42629c803b2.png)

我完成的矢量图形:

![1418_33](../Images/b9c346b1a4732be601eaf348f25e5109.png)

在我把它复制到 Photoshop 中并添加阴影后，图形就完成了！

![1418_34](../Images/1b9834d42373b1bc7b02ba28bb3c70db.png)

这并不太难，是吗？
