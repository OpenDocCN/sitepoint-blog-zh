# 你的工具箱里有什么？

> 原文:[https://www.sitepoint.com/whats-in-your-toolkit/](https://www.sitepoint.com/whats-in-your-toolkit/)

在经历了太多“你钱包里有什么”的广告之后，我决定分享一些我的工具包的关键部分。不，它们不包括精灵、公主、龙或奇怪的半人马动物。总之，我最喜欢的工具是:

*   [Fiddler](http://www.fiddlertool.com/fiddler/) :当处理具有大量远程请求的现代应用程序(也称为 AJAX)时，拥有一个能够枚举和暴露这些请求的工具常常是告诉开发人员为什么某个组件失败的关键。我的一些同事认为这是魔术，但我只是叫它提琴手。这是我最近打开 Internet Explorer 的三个原因之一(另外两个是 SharePoint 和我们的时间表)。
*   QA 是交付可靠、稳定的应用程序的一个经常被忽视但又至关重要的组成部分。m0n0wall 是一个基于 BSD 的防火墙/路由器/边缘设备，允许创建具有良好前端安全性的相对复杂的网络。外部利益相关者可以实时查看应用程序，而外部开发人员可以使用 PPPTP VPNs 获得对机器的安全访问。
*   [虚拟服务器](http://www.microsoft.com/windowsserversystem/virtualserver/):围绕生产中的虚拟化，有很多宣传和悬而未决的问题。但是就开发而言，虚拟化是天赐之物。只需通过简单的复制粘贴操作备份和恢复机器的整个状态，而不是使用 Ghost 或手动调整，就可以节省您职业生涯中的许多时间。
*   [SecretServer:](http://www.thycotic.com/products_secretserver_overview.html) 所以你有一堆乱七八糟的开发网络，你有一堆乱七八糟的虚拟服务器，留给你一堆乱七八糟的密码。现在，您可以拥有一个包含这些信息的相当不安全的电子表格。或者你可以使用这款出色的产品，将所有这些东西存放在一个安全、集中且易于访问的商店中。
*   [NtBackup:](http://www.microsoft.com/resources/documentation/windows/xp/all/proddocs/en-us/ntbackup_command.mspx?mfr=true) 拥有任何种类的数据而没有备份，与没有数据差不多。虽然我不会说内置的 Windows 备份实用程序是备份解决方案的全部，但它们是相当有效和可用的。现在，实际的备份并不重要，除非它以某种方式运送到异地，这将导致我们。。。
*   [FTP.exe](http://www.microsoft.com/resources/documentation/windows/xp/all/proddocs/en-us/ftp.mspx?mfr=true):当与–s 开关一起使用来加载命令文件时，这个工具可以让你上传任何东西到远程服务器来完成备份操作。

那么，你的工具包里有什么应该在我的工具包里？
