# CSS 主题周…这就是所有的乡亲！

> 原文:[https://www . site point . com/CSS-theme-week-and-that-the-all-folks/](https://www.sitepoint.com/css-theme-week-and-thats-all-folks/)

我们以一篇整理了最重要的 CSS 定位属性的经典文章来结束我们的第一个 CSS 主题周。

Nigel Peck 的文章 [CSS 定位属性一览指南](https://www.sitepoint.com/properties-glance-guide/)于 2003 年首次出版，但五年后仍然是 CSS 定位基础的简明摘要(尽管我们在重新出版前更新了一些要点)。事实上，这篇文章的概念形成了 [SitePoint CSS 参考](https://reference.sitepoint.com/css)的基础，然后扩展到包括 [HTML 参考](https://reference.sitepoint.com/html)，我们有一个 JavaScript 参考正在开发中，所以从我们的角度来看，这篇文章是相当开创性的。

如果你错过了一篇文章，这里是我们本周发表的文章的完整列表:

*   **周一:** [使用 CSS 的漂亮导航技巧](https://www.sitepoint.com/html-css/)(瑞秋·安德鲁斯)——这篇文章仍然很受初学者的欢迎，他们希望创建视觉上有趣的基于语义标记的站点导航元素
*   **星期二:** [打破 CSS 布局的框框](https://www.sitepoint.com/breaking-out-of-the-box/)(纪娜·波尔顿)——这是一个很好的例子，说明如何破除所有 CSS 布局都是框框的神话
*   **周三:** [警告:这个秘密的 CSS 技术会让你大吃一惊！](https://www.sitepoint.com/css-animation-technique/)(亚历克斯·沃克)——就在你以为已经看到了 CSS 能做的一切的时候，亚历克斯想出了这个疯狂的效果
*   **星期四:** [使用 CSS 的奇特表单设计](https://www.sitepoint.com/article/fancy-form-design-css)(Cameron Adams)——一直是 SitePoint 上最受欢迎的文章，月复一月
*   **星期五:** [CSS 定位属性一览指南](https://www.sitepoint.com/properties-glance-guide/)(Nigel Peck)——引用之前[引用](https://reference.sitepoint.com/css/)

我希望你喜欢我们的 CSS 周——希望你学到了一些新东西，并从档案中重新发现了一些黄金。下周我们将把你带回我们每周两篇文章的常规日程。

同时…祝你周末愉快，祝你 CSS 编码愉快！
