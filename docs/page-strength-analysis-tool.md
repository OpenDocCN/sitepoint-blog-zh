# 页面强度分析工具

> 原文：<https://www.sitepoint.com/page-strength-analysis-tool/>

SEOMoz.org 开发了一个神奇的工具，可以计算他们称之为“页面强度”的东西，可以用来衡量任何网站或网页的重要性和可见性。

我认为这是任何初露头角的企业家或营销人员都应该拥有的一个伟大工具，因为页面强度分数可能比 Alexa Rank 或 Google PageRank 更准确地反映了一个网站的重要性。

页面强度基于许多因素:

*   指向 URL 和域的链接
*   标题标签的前四个单词在 Google 中的位置
*   域的年龄
*   链接来自。edu 和。gov 域
*   域名的年龄
*   来自维基百科、del.icio.us、DMOZ.org 的链接
*   Google PageRank，Alexa Rank 等等

在你自己的网站上试试 Page Strength，或者下次你想通过 [SitePoint MarketPlace](https://www.sitepoint.com/marketplace) 获得它。

试一试。
