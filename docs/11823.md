# 微软放弃无浏览器的 Windows 7 E

> 原文:[https://www . sitepoint . com/Microsoft-browser-windows-7-Europe/](https://www.sitepoint.com/microsoft-browser-windows-7-europe/)

当 2009 年 10 月新操作系统发布时，微软将提供一个与欧洲版 Windows 7 竞争的网络浏览器选择。该公司希望这一行动将遵守欧盟立法，并抵御进一步的反垄断罚款。尽管 Windows 7 在 T2 卖得很好，微软的收入却下降了近五分之一，而且昂贵的诉讼也无济于事。

微软最初的提议是[将 ie 浏览器从 Windows 7 E](https://www.sitepoint.com/ie-windows-7-europe/) (欧版)中移除。目前，在欧洲预订 Windows 7 的用户会收到一条警告，称该操作系统将不提供浏览器。例如，亚马逊英国发布了[详细的浏览器下载说明](http://www.amazon.co.uk/gp/feature.html?docId=1000321063) *(尽管任何需要这些的人可能不应该尝试安装操作系统！硬件供应商可以自由安装他们选择的网络浏览器，但是 IE 很可能仍然是最受欢迎的选择。*

欧洲委员会不认为 Windows 7 无浏览器是一个可行的解决方案。这与没有媒体播放器的失败版本的 Windows 太相似了——他们更喜欢投票屏幕来恢复浏览器竞争。尽管完整的细节尚未最终确定，但微软已经发布了以下提议:

1.  微软——而不是硬件制造商——将控制浏览器投票屏幕。
2.  Windows 7 E 将提供 Internet Explorer。
3.  欧洲新安装的 Windows 7 将显示一个网页，提供五种流行的网络浏览器供选择。
4.  欧洲 Windows XP 和 Vista 用户将在未来的自动更新中看到相同的投票屏幕(如果 IE 被设置为他们的默认浏览器)。
5.  备选浏览器列表将基于 6 个月的使用统计数据，并将每年审查两次。

这对微软来说是正确的决定吗？我怀疑是这样的——欧盟本可以对该公司处以罚款，并下令对操作系统进行更严厉的修改。至少微软已经完全控制了投票屏幕，并且可以安装 IE 而不用担心立法报复。
