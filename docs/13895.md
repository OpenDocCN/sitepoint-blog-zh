# 从顾问到大师

> 原文:[https://www.sitepoint.com/from-consultant-to-guru/](https://www.sitepoint.com/from-consultant-to-guru/)

咨询和专业服务模式有一个重大缺陷:作为一名顾问，你别无选择，只能用你的时间换取金钱。所以，你限制了你能赚到的收入。即使你知道如何“评估价格”(根据分配给客户的任务的价值来定价，而不是根据你的时薪来定价)，你仍然会限制自己。

我正以最快的速度离开咨询模式(在过去的两年里，我真的一直在远离它)，尤其是在我接近 40 岁生日的时候(喘息！)下周。否则:

–我不会拥有一家可以出售给其他人的公司。

–我的收入总是有限的。

–我的退休生活不会像以前那样快乐。

另一种选择？创建一个不依赖于你的公司。然后你就创造了真正的企业价值，并在远离客户的时候赚钱。

有许多方法可以做到这一点，但它们都涉及到杠杆:

–通过开发可重复的流程来利用您的时间，以便他人可以为您赚钱。

–通过销售产品来利用您的时间，这样您就可以一次生产，然后反复销售。

–通过销售为您带来持续收入流的订阅式计划来增加您的收入。

–利用信息产品(如书籍、报告、演讲、互动软件)，这些产品可以轻松地从一种媒体转换到另一种媒体，因此可以重复使用。

–通过成为“专家中的专家”来影响他人

–利用你的声誉，因为随着你的产品和可重复流程越来越多，越来越多的人将你视为所在领域的专家。

–通过创建网站并从中获利来利用您的专业知识，ala Chris Beasley 广受欢迎的博客和建议。

上述方法都是“古鲁”战略的一部分，也是著名古鲁所做的。他们不会用时间换美元，但会利用巨大的杠杆。

我希望你正在努力成为一名古鲁。最坏的情况？你有一堆伟大的、信息丰富的产品和过程，它们建立了你的信誉，让你在你的市场上可见。换句话说，如果你必须坚持咨询模式，那么“大师”方法只会建立你在他人眼中的可信度和价值。

我今天和一个迷人的家伙共进午餐。他以在研讨会、网站和营销计划中设立“大师”为业。他还训练他们公开演讲，这样他们就能得到观众的好评。换句话说，古鲁支持正在成为一个独立的行业(同样，我最近的书也是关于这个主题的)。

在今天的世界里，只有少数人挣大钱，并爬到金字塔的顶端。这些是古鲁。你可以用正确的策略成为一个人，或者你可以在金字塔的底部挣扎。

各地的网页设计者和开发者都有机会成为大师，而不是继续依赖小时工资和直接为客户服务。这需要心态上的明确转变，而且每个想成为大师的人都必须面对一些障碍。但是当你克服了这些障碍，生活立刻变得更有意义。

我希望你正在考虑成为大师的方法，而不是传统的顾问。
