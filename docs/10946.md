# Microsoft Office Online:初步观察

> 原文:[https://www.sitepoint.com/microsoft-office-online-review/](https://www.sitepoint.com/microsoft-office-online-review/)

Docs.com 提供微软的 Word、Excel 和 PowerPoint 在线版本。任何拥有脸书账户的人都可以使用该系统:虽然这是一个有争议的决定，但它将允许你与朋友和同事分享文件和协作。

应用程序是测试产品。不用说，不要把它们用于任何重要的工作，或者，如果你用了，下载你创建的任何文档的副本。

令人惊讶的是，这些应用程序在我测试的大多数浏览器中都可以工作——I E8、Firefox 3.6、Opera 10.5 和 Chrome 4。然而，每个浏览器都有一些独特的不一致和问题。例如:

1.  如果启用了脚本调试，IE8 会不断弹出 JavaScript 错误警告。这将影响每一个优秀的 web 开发人员，所以我不知道为什么它没有被修复？
2.  Firefox 不断地重新加载脸书登录屏幕，并且不允许你重命名文档(它显示一个带有一大块 XML 的警告框？)
3.  Opera 经常死机，Excel 加载失败。

总的来说，Chrome 似乎提供了最好的体验。我怀疑这会取悦微软！

## 陶醉在丝带中

Office 功能区出现在所有三个应用程序中，Office 2007/2010 用户将立即有宾至如归的感觉。

![MS Office ribbon](../Images/e177126d4162e3b5ec8708a6e983fb78.png)

我不太喜欢丝带。然而，这主要是因为我从第一个版本开始使用 Office，并熟悉了笨重的旧菜单系统。也就是说，功能区对于新的 Office 用户来说更容易使用。我已经对它很熟悉了，但仍然发现自己不时地寻找选择。

我不相信网络版需要丝带。Word 提供了最多的选项，但是有几个部分的内容很少。我相信随着应用程序的发展和获得更多的功能，ribbon 会变得更加有用。

## 单词

Word 是最复杂的应用程序，但它仍然不如 Windows 写字板强大。标准格式和拼写选项是可用的，尽管它不会在您键入时进行拼写检查。

![MS Office Word Web App](../Images/e5f3c52e53d7792640f9aec614323219.png)

可以添加表格、图像、MS 剪贴画和超链接，但是不要期望有一系列选项，也就是说，不可能给表格添加边框。

有一个标准的编辑视图和一个阅读视图。奇怪的是，Word 在阅读视图中随机启动，因此您不能立即编辑文档。切换到编辑视图只需点击一下，但这会降低用户体验。

我发现最奇怪的是“保存”图标。网络应用程序通常会保存输入的数据，但是如果你不点击保存，你会看到一个令人恼火的*“你确定要离开这个页面吗”*对话框。也许对于现有的 Word 用户来说不那么困惑，但是 Excel 或 PowerPoint 中没有出现保存图标？

## 擅长

Excel 是基本的，虽然我发现它是最快的应用程序。您可以输入数据和设置数据格式，但是没有图表等高级功能。您也很难控制数据类型，例如，货币和日期总是使用美国格式。

![MS Office Excel Web App](../Images/ed7f5818bf16322fc07b66efa7ceb1f1.png)

公式很难输入。该应用程序支持广泛的公式，当您键入“=”和一个字母时，会出现一个长长的下拉列表。但是接下来你就只能靠自己了——Excel 不会告诉你公式是做什么的，也不会告诉你它需要什么参数。

## 文稿演示软件

PowerPoint 是该套件中开发最不完善的应用程序。可以添加、删除或编辑幻灯片，但设施有限。您不能更改背景颜色、添加内容块或重新排列框。

![MS Office PowerPoint Web App](../Images/22175f2146f83e68cea14b3c7c509b77.png)

图形可以添加“智能艺术”。实际上，这将文本项的分层列表转换为组织结构图、表格或其他图表。这很聪明，但我怀疑这会让很多人困惑。

## 摘要

Microsoft Office Online 是一个测试版，尽管感觉更像是预测试版。无论你使用哪种浏览器，你肯定会遇到不一致、错误和问题。没有任何帮助，速度很慢，而且频繁得令人担忧。

该套件无疑会有所改进，但在现阶段，我不建议将其用于日常使用。如果你想尝试在线办公软件，那就用谷歌文档或 Zoho 吧:它们要好得多，还支持 MS Office 兼容文档。
