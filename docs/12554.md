# 谷歌推出论坛搜索

> 原文:[https://www.sitepoint.com/google-launches-forum-search/](https://www.sitepoint.com/google-launches-forum-search/)

![](../Images/c07bddd282e5f82737dc3fc6f8a7d04e.png "google-groups") [ Alex Chitu 报道](http://googlesystem.blogspot.com/2008/11/google-forum-search.html)谷歌在其[谷歌集团](http://groups.google.com/)产品中推出了一个论坛搜索引擎，作为扩展搜索服务的一部分。该网站上的搜索现在不再局限于谷歌托管的群组和新闻组讨论，而是包括整个网络的论坛主题。

不过，这项技术当然并不完美，因为我们注意到一些来自博客和其他非论坛来源的页面混杂在搜索结果中。目前还不清楚谷歌如何确定一个网站是否是一个论坛，但显然他们的方法有一些问题需要解决。

Chitu 指出，谷歌正在使用上个月添加到常规搜索结果中的相同的增强搜索片段进行团体论坛搜索。对于任何被确定为论坛主题的页面，它包括帖子的数量、不同作者的数量以及最后一篇帖子的日期。这有助于搜索者确定信息有多新，以及发布到论坛的问题是否有可能得到搜索到的响应。

![](../Images/27a9d742ff9793d3fa9fc22a556575f1.png "google-forum-search")

在其他与谷歌论坛相关的新闻中，该公司正在将自己的帮助论坛从群组迁移到一个类似雅虎问答的问答服务，该服务显然代号为“困惑”。 [Google Talk](http://www.google.com/support/forum/p/Talk) 、 [Chrome](http://www.google.com/support/forum/p/Chrome) 和 [AdSense](http://www.google.com/support/forum/p/AdSense) 帮助论坛已经做出了改变。Chitu 说，一旦所有的帮助论坛都迁移到新系统上，谷歌有可能会发布该平台的代码。
