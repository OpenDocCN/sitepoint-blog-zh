# 三分之一的网络用户害怕购物

> 原文:[https://www.sitepoint.com/web-users-scared-to-shop/](https://www.sitepoint.com/web-users-scared-to-shop/)

几乎三分之一的网站访问者认为网上购物不安全。英国公平交易办公室最近发布的一份报告发现，30%的互联网用户不会交出他们的信用卡信息。该报告得出结论，消费者信心正在增长，但速度缓慢。在线交易可能会停滞很多年，尤其是当英国的在线销售额是欧洲平均水平的两倍时。

当媒体报道大规模的安全问题时，信任问题就变得更加棘手了。在过去的几天里，艾伯特·冈萨雷斯(Albert Gonzalez)和两名未透露姓名的俄罗斯助理被指控闯入由 [Heartland](http://www.heartlandpaymentsystems.com/) 运营的系统，Heartland 是几家大型零售商(包括 7-Eleven 连锁店)的在线支付提供商。检察官指控黑客窃取了多达 1.3 亿张信用卡的详细信息，并意图出售这些数据。如果罪名成立，冈萨雷斯可能会被判 25 年监禁。

令相关商店尴尬的是，信用卡信息被一次 SQL 注入攻击获取了。尽管司法部声明这是一种 [*【复杂的黑客技术】，*](http://www.usdoj.gov/opa/pr/2009/August/09-crm-810.html) 开发者多年前就已经知道这些攻击。虽然没有一个系统是 100%安全的，但是 SQL 注入通常可以通过基本的数据清理和安全格式的 SQL 命令来阻止。

黑客的“成功”故事会产生立竿见影的效果。7-Eleven 的网上销售当然受到了影响，但该案件将在整个网络上产生多米诺骨牌效应。

一般来说，网上购物比把你的信用卡交给另一个人更安全。然而，当网络安全问题真的发生时，后果要严重得多，它们会影响更多的人，并且盗窃会得到大量的媒体报道。

事实仍然是很大一部分用户不信任网络。除非我们有效地解决这个问题，否则网上购物永远不会发挥其全部潜力。

你是网上信用卡或身份盗窃的受害者吗？你信任网店吗？支付提供商应该为基本的安全漏洞承担更多责任吗？
