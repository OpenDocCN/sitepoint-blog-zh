# 向您的客户展示您的关心

> 原文:[https://www.sitepoint.com/show-your-customers-you-care/](https://www.sitepoint.com/show-your-customers-you-care/)

商务旅行有时很棘手…飞机晚点，酒店预订丢失。事情发生了。通常情况下，决定基调和创造(或摧毁)品牌大使的不是发生了什么，而是如何处理这个问题。

几天前，我前往全国各地参加客户会议。这次飞行平安无事，一切都很顺利。会议本身是非常富有成效的，我期待着回家看到我的妻子和孩子，睡在自己的床上。回程的时候，我被预订了一个中转航班。虽然这通常不是一个问题，但在这种情况下，第二个航班开始登机，因为第一个航班应该降落…没有多少出错的余地！

当然，不可避免的事情发生了——我们的第一次航班延误了几分钟，我们在第二次航班应该关闭登机门的时候着陆。没问题，我想，我会搭下一班飞机去亚特兰大。我去顾客柜台，却发现*是*飞往亚特兰大的最后一班飞机！更糟糕的是，坐在桌子后面的那个家伙似乎根本不在乎他的航空公司刚刚毁了我的一天，因为他们连十分钟都等不了。他在电脑上点了几分钟，递给我一张酒店券，漫不经心地指着几分钟后会有班车来接我的地方。

这时候我非常生气，不是因为我错过了航班，甚至不是因为我不得不在一个不太好的机场酒店住了一夜，甚至不是因为我又要多一天见不到我的孩子了。不，让我恼火的是那些和我打交道的航空公司员工的态度——他们似乎根本不在乎。

我走到酒店班车接送区，等了大约 25 分钟。班车到了，我走过去正要往后面装行李，司机通知我货车已经满了。它显然没有满，所以我说“什么？”他告诉我他在等一些乘务员，而我必须等下一辆货车。是的，同一家航空公司的空乘人员第一个到达酒店，而滞留在那里的顾客不得不等待。

这还没有结束。当我终于到达酒店时，航空公司的两名飞行员插队到队伍的前面办理登机手续，经过了几位因错过航班而流离失所的顾客(包括我)。一天之内，我三次被告知，这家航空公司的员工只关心他们自己，顾客排在最后。

### 不关心比搞砸更糟糕

唯一比犯错更糟糕的事情是没有向你的客户表明你关心他们并感到抱歉。错误时有发生，这是生活的现实。当你搞砸了，承认并说“对不起”然后，尽你所能来纠正这种情况。

就我而言，如果我遇到的第一个航空公司员工对我的处境表现出一些同情，我可能根本不会注意到其他事情，或者我可能真的不会在意。基调已定——他们就是不在乎。

### 它始于富有爱心的员工

几周前，我谈到了销售漏斗的另一面[，并讲述了丽思卡尔顿的员工如何被授权在一位顾客身上花费高达 2000 美元，而无需获得管理层的任何批准。他们大胆地向员工声明，他们有能力照顾他们的顾客。](https://www.sitepoint.com/the-other-side-of-the-sales-funnel/)

另一方面，航空公司没有尽力让我的情况好转。只是在我在推特上抱怨我有多不开心之后，他们才提出愿意为此做点什么，即便如此，他们也只是提供了 50 美元的旅游券。在 500 美元的航班上，我损失了一天时间，他们给了我 50 美元的代金券。

给你的员工权力去帮助你的顾客，你将向他们展示关心顾客是他们的*工作*。“客户服务”不只是嘴上说说，你公司的每个人(包括你)都应该负起责任，为客户做力所能及的事情。

### 两种情况都有可能

你可以创造品牌大使，告诉每个人你有多棒，或者你可以让顾客在脸书和推特上向他们的朋友抱怨你。它真的可以向任何方向发展。如果航空公司的人花一点时间了解更多关于我的情况，事情可能会完全不同。

下次你和客户或顾客有困难的时候，设身处地地为他们着想，真正试着理解他们的立场。同情他们。即使你不能解决他们的问题，简单地表现出你的关心也会有很大的不同。
