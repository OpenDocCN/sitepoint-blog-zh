# 铺上红地毯…获胜者在房子里

> 原文：<https://www.sitepoint.com/roll-out-the-red-carpet-the-winners-are-in-the-house/>

本周在 SitePoint 论坛上，我们都在谈论黄金，宝贝！**具体来说，金帖奖项目**。第一周已经接近尾声[获胜者已经宣布](https://www.sitepoint.com/forums/showthread.php?p=4589120#post4589120)。我想祝贺 *DCrux，死亡阴影 60，编年史大师 1，amabaie，qvalpro* 和 *iMeow* 他们出色的获奖帖子——他们每个人都将收到他们选择的 SitePoint book。比赛还有三周时间，所以赶快去论坛发帖吧。你可以[在这里](https://www.sitepoint.com/forums/showthread.php?t=674281)阅读如何获得资格的更多细节。

除了致力于帖子的质量，我最近还花了一些时间专注于整理和整合众多的论坛和分论坛。当你像我一样每天都呆在 SitePoint 论坛上时，你往往会忽略它们对一个新成员来说是多么令人生畏。这个地方已经变得有点像兔子窝，一些潜力巨大的论坛(想到了摄影)被忽视了，因为它们埋藏得太深了。所以我的下一个项目是做一些重组。我将剔除一些未使用的论坛，移动一些有价值但未被充分利用的论坛，并添加那个早该添加的 WordPress 论坛。因此，请密切关注这个社区博客，以获得进一步的更新。
