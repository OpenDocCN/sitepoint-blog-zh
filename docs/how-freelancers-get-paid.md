# 自由职业者:如何获得报酬？

> 原文：<https://www.sitepoint.com/how-freelancers-get-paid/>

自由职业者和其他企业主的一个普遍目标是快速全额获得工作报酬。你可以向客户提供几种常见的支付方式，每种方式都有自己的优缺点。此外，你还可以做一些事情，让客户更容易向你付款，并按时付款。

**支付选项**

以下是你可能获得报酬的几种选择:

支票——接受商业或个人支票是获得工作报酬的一种非常常见的方式。最大的挑战是必须等待支票寄到你的邮箱，然后在你的账户里结算。

贝宝——许多自由职业者接受通过贝宝和其他类似服务支付。这可以提供即时支付，只需要客户完成一个简短的在线过程来发送付款。最大的败笔是从你的付款中扣除的第三方交易费。

信用卡——如果你有一个商业账户，并设置了信用卡支付，这是一个很好的选择。它允许客户通过信用卡支付，如果你与客户有长期协议，你甚至可以自己处理付款。同样，即时支付也会产生费用。

电汇——电汇不是一种很受欢迎的支付方式，但它比支票支付要快得多。当然，你将不得不分享你的银行信息，包括你的账号，这并不吸引很多人。当然，还有电汇费。

汇票——通过汇票支付需要客户去第三方，如银行或邮局购买，然后才能邮寄给你。这也不是一个非常受欢迎的方法，但对于那些没有能力用支票支付的人来说，这是一个有效的选择。

**鼓励支付的方式**

不管你向客户提供什么样的支付方式，当你期望得到报酬时，有几种方法可以让你自己得到报酬。

第一步是创建标准和一致的计费实践。如果你在每个月的同一时间以同样的方式付款，你和客户都更容易提前计划。你还应该在合同中概述你的计费方式，并在必要时与客户一起审核。

如果你发现延迟付款是你经常面临的问题，你可以考虑对错过或延迟付款收取罚款。同样，这应该在你的合同中列出，并与你的客户讨论，这样就不会产生混淆。

根据你所做项目的类型，存款是确保你获得报酬的另一种方式。你可以要求你的客户预先支付总费用的一部分，然后设定额外付款的里程碑。如果你特别担心收到付款，你也可以等到收到第一笔定金后再开始工作。

最后，你也可以考虑提供一个付款计划选项。然而，这可能不是一个可行的选择，除非你已经与客户建立了关系。如果一个长期客户经历了意想不到的经济困难，付款计划可以减轻他们的经济负担，同时有助于确保你得到报酬，即使是在一段特定的时间内以较小的金额支付。
