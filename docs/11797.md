# 我们需要你:加入 SitePoint 社区的 bbPress 主题项目！

> 原文:[https://www . site point . com/we-want-you-join-the-site point-communities s-bb press-theme-project/](https://www.sitepoint.com/we-want-you-join-the-sitepoint-communitys-bbpress-theme-project/)

在 SitePoint 论坛上，我们的社区设计了一个有趣的新项目:[我们正在为 bbPress 创建一个新主题。](https://www.sitepoint.com/forums/showthread.php?t=630061)

[bbPress](http://bbpress.org/) 是 WordPress 的创作者们推出的论坛包。它是免费的，轻量级的，看似简单——隐藏在它朴素的外表下是一些非常酷的论坛功能的强大基础。不幸的是，很少有很好的主题可供选择，计划如下:

*   首先是规划用户界面和创建一些线框— [看看这里发生了什么。](https://www.sitepoint.com/forums/showthread.php?p=4335792)
*   接下来是 HTML 和 CSS 编码。
*   第三，有一个计划是用 PHP 在主题中加入一些很酷的特性。
*   我们在 99designs 的朋友们将举办一场比赛，帮助决定主题的最终外观。
*   当然，所有这些艰苦的工作都需要插入到一些 bbPress 主题模板中。
*   没有朗朗上口名字的酷主题是什么？
*   最后，在主题正式发布之前，需要对主题进行彻底的 beta 测试。

正如你所看到的，各种各样的人都有需求——无论你是修补 PHP、编写 JavaScript、创建可爱的设计，还是编造有趣的名字，这个项目都有适合每个人的东西。听起来很有趣？然后跳上船！如果你还不是 SitePoint 论坛的会员，[只需几分钟就能建立一个免费账户。](https://www.sitepoint.com/forums/register.php)
