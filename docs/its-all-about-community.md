# 这都是关于社区的

> 原文：<https://www.sitepoint.com/its-all-about-community/>

我整个周末都沉浸在被称为 [FullCodePress](http://www.fullcodepress.com/) 的极客狂潮中。SitePoint 是这次活动的官方赞助商，所以我是评委团的[成员。这是一个美妙的周末，被迫制作的网站质量惊人，更不用说鼓舞人心了。对于我不再参与开发游戏的事实，我确实感到有点难过。但只是一点点。只有几秒钟。毕竟，没有什么比得上社区…](http://www.fullcodepress.com/the-fullcodepress-judges/)

关于社区的话题，我们的官方 SitePoint T 恤大赛还有一天就要投票了，所以如果你在乎你穿什么，请确保你现在就过来发表你的意见。我觉得结果有些令人惊讶。目前的领先者有点像黑马。我不得不承认，我最喜欢的看起来没有获胜的希望，但是有很多人说他们不认为它穿在女人身上会很好看，这可能是最好的。

因为我对社区很感兴趣，我很高兴看到我们的社区图书项目被证明是一个受欢迎的想法。如果你喜欢写作，并且对网页开发的任何方面都了如指掌，那么一定要注册。还有几周的时间来报名，然后你有三个月的时间来应征。

![SitePoint forums](img/1ca598dd6b7e55f9d0f60654f8b532af.png "sitepoint-forums")

我们为我们的[脸书粉丝](http://www.facebook.com/sitepoint)准备了一个特别的新项目，所以如果你想在我们的粉丝页面上留下你的印记，一定要留意更多的细节。但这就是我现在要说的。

我想就这些了——这是我今天的简短发言，因为我仍在从周末的恢复中。

**本周热门话题**
随着我们最新的《网络摄影》一书即将发行，我想我应该通过让你注意到[摄影](https://www.sitepoint.com/forums/forumdisplay.php?f=302)论坛上的一个帖子来庆祝一下。六月是所有关于[并列](https://www.sitepoint.com/forums/showthread.php?t=680340)。

在 [Graphics](https://www.sitepoint.com/forums/forumdisplay.php?f=8) 论坛中有一个关于如何让[访问 GUI 可访问](https://www.sitepoint.com/forums/showthread.php?t=684176)的有趣帖子。

引起我注意的一个帖子是关于老板们拿多少薪水的。你可以在[职业与教育论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=178)找到它。

在[普通聊天](https://www.sitepoint.com/forums/forumdisplay.php?f=9)中，我很有兴趣听到[如果你可以做除了你现在做的以外的任何事情，你会选择什么工作](https://www.sitepoint.com/forums/showthread.php?t=681960)！
