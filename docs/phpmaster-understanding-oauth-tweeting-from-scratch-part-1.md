# PHPMaster:理解 OAuth——从零开始，第 1 部分

> 原文：<https://www.sitepoint.com/phpmaster-understanding-oauth-tweeting-from-scratch-part-1/>

对 OAuth 的一个常见抱怨是它很难理解，但这种混淆可能是因为人们期望由第三方库提供的抽象将消除理解 OAuth 事务步骤的需要——事实并非如此。这篇由两部分组成的文章演示了 OAuth v1 是如何工作的，它解释了仅使用几个内置函数将一个 PHP 应用程序连接到 Twitter API 以向用户的 Twitter 流发送消息的过程。本文中提供的示例代码仅用于教育目的。它缺乏很多实际用途和错误处理，任何真正的开发都应该利用现有的 OAuth 库，但是在阅读本文之后，您将对 OAuth 如何工作有更好的理解，并且能够更好地解决可能出现的任何问题。
