# 面向设计师的 19 款 Firefox 插件

> 原文:[https://www.sitepoint.com/firefox-add-ons-for-designers/](https://www.sitepoint.com/firefox-add-ons-for-designers/)

你是一个网页设计师，希望你能在一个方便的地方做更多的事情，而不是打开多个程序吗？感谢庞大的 Firefox 插件开发社区，你的网络浏览器几乎可以为你做任何事情。

我们已经介绍了面向开发者的 Firefox 插件，但是我们不能忽略那些坐在那里实际设计开发者需要构建的东西的人。这 19 个附件可以让你选择颜色、测量尺寸、保存截图等等。请注意，你应该有选择地在你的浏览器中添加什么插件，因为事实上你添加的越多，它运行得越慢，但是你真的能抵制这些插件吗？

## 颜色工具

ColorSuckr : ColorSuckr 会从图像中挑选出 12 种最常见的颜色，并允许你用它们来构建调色板。

![](../Images/c0a86698f5c87f151982e226af60ca66.png)

**[ColorZilla](https://addons.mozilla.org/en-US/firefox/addon/271)**:ColorZilla 允许你从浏览器的任何一点(不仅仅是页面)获取颜色读数，你可以调整颜色并粘贴到其他程序中。除了识别颜色，这个插件还允许你测量任意两点之间的距离，从预定义的集合中选择颜色，获取 DOM 元素的信息等等。

![](../Images/de5fbc32c84c0bf35fe8aacda6eb5478.png)

**[fire palette](https://addons.mozilla.org/en-US/firefox/addon/12377)**:fire palette 在 Firebug 的 CSS 面板中增加了一个颜色选择工具，这样你就可以轻松挑选 CSS 兼容的颜色。

[![](../Images/0c46bb1677573b2cbc8dd2476cd93eb0.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/ffdesigners/firepalette.png)

**[调色板抓取器](https://addons.mozilla.org/en-US/firefox/addon/2290)** :调色板抓取器允许你从你访问的任何页面制作一个调色板，然后将其导出到 Photoshop、Flash、PaintShop Pro、GIMP、Adobe Fireworks 和 Mac OS X 拾色器调色板格式。

![](../Images/1cf95bc7f46d471a6fd4cd7a4f938e73.png)

## 测量工具

**[GridFox](https://addons.mozilla.org/en-US/firefox/addon/4904)** :可以在任意页面顶部绘制网格，测量列宽等项目。

![](../Images/62a36dc9aa7c538a305b1f40d8730fdd.png)

**[MeasureIt](https://addons.mozilla.org/en-US/firefox/addon/539)** :在页面元素周围画一个标尺，看看它有多大像素的快速简单方法。

![](../Images/223cb6418c4f7bd12cf4d21b834033cd.png)

**[Pixel Perfect](https://addons.mozilla.org/en-US/firefox/addon/7943)**:Pixel Perfect 允许设计师将他们的构图叠加在开发者的进度之上，以查看元素有多远，需要进行哪些调整。

![](../Images/54806351ce83e5714aa011a439630bf8.png)

## 杂项工具

**[Firebug](https://addons.mozilla.org/en-US/firefox/addon/1843)** : Firebug 一般和开发者联系在一起，但是布局和 CSS 检查器可以相当得心应手。

[![](../Images/49606c78c288bcc72c82e588e7dc8856.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/ffdesigners/firebug.jpg)

:这个插件可以让你高亮显示页面中的任何元素，并查看它的 CSS 信息。如果你想识别一种字体，这是非常方便的。

![](../Images/680d611a5070e4f1d1258c0a5eda856c.png)

**[查看依赖关系](https://addons.mozilla.org/en-US/firefox/addon/2214)** :查看依赖关系在“页面信息”中添加了一个标签，这样您就可以查看与您刚刚加载的页面相关的所有文件及其大小。这是一个很方便的工具，可以用来确定是什么减缓了加载时间。

[![](../Images/6808ac986ad2a0b150a79c26a110fa35.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/ffdesigners/ViewDependencies.png)

**[Web Developer](https://addons.mozilla.org/en-US/firefox/addon/60)** :尽管名字如此，Web Developer 对设计师来说有很多有用的功能，比如检查、编辑和禁用 CSS，以及图形。你也可以用它来测量元素，调整浏览器大小，按媒体类型显示 CSS 等等。

![](../Images/b166996b8dfa799196c49f782191262a.png)

**[窗口缩放器](https://addons.mozilla.org/en-US/firefox/addon/1985)** :允许您在不同的屏幕分辨率下快速查看页面。只需点击一下，您就可以在 640×480、800×600、1024×768、1280×800、1280×1024 和 1600×1200 之间切换。

![](../Images/d563061cf2408b02be0dd0773ef560d5.png)

## 截图工具

**[诱拐！](https://addons.mozilla.org/en-US/firefox/addon/3408)** :诱拐！允许您保存页面的一部分或整个页面，以便向其他人展示潜在的错误。您还可以打印或复制您保存的任何图像。

[![](../Images/7149d17d6796a2a18c3ecca1ecb20b0a.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/ffdesigners/abduction.png)

:FireShot 为您提供了许多选择，让您可以拍摄部分或完整页面的图像，然后您可以通过电子邮件发送、保存或上传到免费主机。

[![](../Images/e34d32f0cb4e33ea47074399d03213d9.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/ffdesigners/fireshot.png)

**[图像缩放](https://addons.mozilla.org/en-US/firefox/addon/139)** :选择您想要重新调整图像大小的尺寸，或者重置自动滚动，以便您可以使用鼠标上的滚轮来放大和缩小图像以检查质量。

![](../Images/62d9f82ec231b02bc356f4d0d5b8a00f.png)

**[Picnik](https://addons.mozilla.org/en-US/firefox/addon/4889)** :这个插件可以让你保存你所在页面的图像，并直接发送到在线图像编辑器 Picnik。

![](../Images/69b62522ba75b44ccea4b49cab04482d.png)

**[Pixlr](https://addons.mozilla.org/en-US/firefox/addon/9924)** :您可以直接在 Pixlr 在线图像编辑器中编辑并打开您拍摄的图像或截图。

![](../Images/7d5685dd21066dd250414f3f384ff4a0.png)

**[Screengrab](https://addons.mozilla.org/en-US/firefox/addon/1146)** :你可以抓取整个可滚动的文档，只抓取可见的部分，或者一个可拖动的选择。Screengrab 甚至可以只保存单个帧的内容，也可以拍摄 Flash 和 JavaScript 元素的图像。

![](../Images/9d8d4072aa894fc4a0105572b183ea25.png)

**[【Talon】](https://addons.mozilla.org/en-US/firefox/addon/11587)**:Talon 是广受欢迎的 Aviary.com 推出的一款附加软件，可以让你截图保存到你的系统中，直接在在线凤凰图片编辑器中打开，托管在他们的网站上或者复制到剪贴板上。

![](../Images/c317c0b66da1080472250df42adb4647.png)
