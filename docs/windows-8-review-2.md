# Windows 8 第一个月回顾:软件、安全性和稳定性

> 原文：<https://www.sitepoint.com/windows-8-review-2/>

在本次 Windows 8 评测的第一部分中，我们讨论了定价、安装和新界面。在这篇文章中，我将研究新操作系统提供的软件和应用程序。

## Windows 8 软件

令人惊讶的是，全新的 Windows 8 安装中几乎没有应用程序。当然，这并不能阻止 OEM 安装几百兆的垃圾软件，但即使这样也可以通过打开 charms 菜单(Win+C)>设置>更改 PC 设置>通用>刷新 PC 来解决，而不会影响您的文件。

Windows 8 提供了 Metro 应用程序([见我对它们的看法](https://www.sitepoint.com/windows-8-review-1/))，如邮件、消息、人物、日历、新闻、金融、体育、地图、音乐、照片、视频、旅游、天气、SkyDrive 和 Bing 搜索。Microsoft 实施了帐户同步选项来简化您的在线生活。例如，People 应用程序可以从您的微软、Skype、Hotmail、Outlook、谷歌、脸书、Twitter 和 LinkedIn 帐户中导入联系信息。

然而，这些应用程序中的大多数都非常简单。例如，Mail 远没有 Outlook Express 或 Windows Mail 复杂。新手和平板用户可能更喜欢它。我没有，但这并不是说你被迫使用它们…

如你所料，与现有桌面软件的兼容性非常好。微软努力支持应用程序，可以肯定的是，在 Windows 7 中工作的东西可能会在 Windows 8 中运行。也就是说，我确实遇到了一些奇怪的问题，主要是由于加强了安全措施。在大多数情况下，临时以管理员身份运行就解决了问题。

为了降低成本，Windows 8 Pro 没有媒体中心或许多播放 DVD 的基本编解码器。幸运的是，在 2013 年 1 月 31 日之前，您可以免费获得 [Windows 8 媒体中心包](https://www.sitepoint.com/free-windows-8-media-center/)。尽你所能抓住它。

Windows 8 是第一个提供[应用商店](https://www.sitepoint.com/microsoft-windows-store/)的操作系统版本——我怀疑这是升级如此合理的另一个原因。诚然，它有点笨重，与苹果和谷歌的产品相比没有太多的选择，但这可能会改变。微软提供了一系列免费应用程序，类似于以前版本的 Windows 中安装的程序。它包括扫雷，但是，说真的，这款游戏是如何下载到 106MB 的？！

Windows 8 还提供了 Internet Explorer 10，它在 Windows 7 上仍是测试版，永远不会在 XP 或 Vista 上推出。我的[最近的评价](https://www.sitepoint.com/internet-explorer-10-review-ie10/)总体上是正面的；新浏览器提供强大的标准支持、良好的操作系统集成和自动更新。它不再有速度优势，提供的功能也比竞争对手少，但它是一个很好的浏览器，没有理由嘲笑那些喜欢它的人。欧洲用户将看到[浏览器投票屏幕](https://www.sitepoint.com/microsoft-windows-browser-ballot/)，但是，如果 IE10 是你的默认设置，你可以在地铁或桌面模式下使用它。

最后，卸载桌面应用和往常一样:控制面板>程序和功能。然而，Metro 应用程序是不同的-你需要右键单击它们的磁贴并选择卸载。造成普遍混乱的另一个因素。

## Windows 8 安全性

从历史上看，微软比其他任何厂商都更容易引发安全恐慌。该公司已经扭转了局势，尽管 Windows 仍然是黑客的首要目标，但很少听到利用。关键点:

*   Microsoft Security Essentials 防病毒应用程序是默认安装的，并与 Windows Defender 结合使用。它是轻量级和有效的——尽管反病毒公司会试图说服你这还不够。
*   该操作系统具有安全引导架构，可在 Windows 加载时抵御恶意软件。
*   Metro 应用程序运行在沙盒环境中，不能相互交互。

Windows 8 不会阻止用户安装垃圾软件，但病毒和恶意软件开发者面临着更艰巨的任务。

## Windows 8 稳定性

除非你运气不好，或者安装了你遇到的所有不可靠的应用程序，否则 Windows 7 坚如磐石。我不记得三年内出现过重大的操作系统崩溃或蓝屏死机。Windows 8 建立在这一成功的基础上。

到目前为止，我只遇到过一个不稳定的问题:我的键盘出了问题，重启时有一个进程无法关闭。虽然我怀疑这是一个古怪的驱动程序问题，但 Windows 仍然稳定，我没有丢失工作。

综上所述，Windows 8 的软件故障很少。但是对于大多数其他版本的操作系统来说也是如此。经过十年的安全更新，即使是 XP 也是稳固和相当安全的。没什么好兴奋的，但同样，也没有什么大的烦恼。
