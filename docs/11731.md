# 扼杀费和六个其他条款考虑你的合同

> 原文：<https://www.sitepoint.com/clauses-to-include-in-freelance-contract/>

我不是律师，但我已经阅读了太多的协议，无论是为我自己的公司还是代表我的客户。有些条款至关重要，尤其是当你提供合同服务时。以下是一些大问题的分类，以及你为什么要考虑把它们写进你的协议中。

**击杀费** 
击杀费就是取消费用。这对设计师和开发人员来说尤其重要，因为它规定即使客户取消了协议，你仍然可以从迄今为止完成的所有工作中获得报酬。大多数终止费条款的措辞是为了保护承包商，确保他们得到报酬，不管客户取消的原因是什么。
 **独立承包人**

这一条款应该清楚地说明你是一个独立的承包商，而不是你的客户组织的雇员。这是一个重要的条款，因为它确保你不会进入一个非排他性的安排，让你控制如何和何时工作，并声明你不会从客户那里得到好处。基本上，这个条款保护了你和你的独立地位。想了解更多关于雇员和自雇者的困惑，请阅读:[你是谁？员工还是个体户？](https://www.sitepoint.com/employee-vs-freelancer/)

**非排他性**

很多时候，这一条款被纳入独立承包人条款，但它是如此重要，以至于给它单独的一段是有价值的。它应该重申，你不是专门向客户提供服务，你有权在合同期限内为其他客户提供服务。

**赔偿**

赔偿条款被认为是“保持无害”条款，因为它声明你不是公司的代理人。从本质上说，它保护你作为承包商免受客户未来遭受的损失或损害。

**保密性**

该条款旨在为客户及其知识资本提供保护。它规定，作为一名独立承包商，你不会泄露你在工作关系中遇到的机密或专有信息。

**保修**

保证书声明你将以特定的方式做特定的事情，并且客户可以对你的工作关系的结果有特定的期望。这适用于声明你将提供符合行业高质量标准的服务。它还可以指定您将作为协议的一部分提供的任何支持服务、故障排除和错误修复。如果你包括一份保证书，重要的是要限制期限，并非常具体地说明包括什么。

**所有权**

所有权条款对于创意和技术领域的人来说尤其重要，因为它规定了谁拥有最终产品以及在工作过程中使用的工作文件。这个条款也可以概述你作为承包商有什么权利使用你的作品组合中的作品和相关图像。它还可以允许在最终产品上包括一个信用额度，说明谁做了这项工作。

你的协议中应该包含其他重要条款，但这些是最重要的条款之一。请注意，这不应该被认为是法律建议，最好是咨询律师，以确保你的合同是防弹的。
