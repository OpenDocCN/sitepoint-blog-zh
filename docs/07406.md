# 1 天内开始构建 Windows Phone 应用程序:免费 Windows Phone Camp

> 原文:[https://www.sitepoint.com/windows-phone-camp/](https://www.sitepoint.com/windows-phone-camp/)

一个新的移动开发平台的成功运行通常需要大量的努力和长时间的投资。在注意到移动开发者面临的高要求和陡峭的学习曲线后，微软做出了一些高尚(和高效)的努力，以确保快速、轻松和有趣地进入 Windows Phone 8。他们的最新产品——免费的 Windows Phone Camps——是这种以开发者为中心的哲学的典型例子。正如你所料，参加这些活动中的一个可以让你在一天内达到速度。

微软的下一次 Windows Phone 夏令营是在太平洋时间 12 月 7 日星期五上午 9:00(GMT-8)。任何人都可以在线或亲自参加，不像许多类似的高强度培训，它是免费的。本周五的活动将在微软总部举行，同时在 SitePoint.com 的[虚拟举行。以下是周五的活动信息和时间表:](https://www.sitepoint.com/windows-phone-8-camp-streamed-live/)

**日期和时间:**太平洋时间 2007 年 12 月 7 日星期五上午 9:00(GMT—8)

**地点:**美国加州山景城 1065 La Avenida 1 号楼，邮编 94043

**日程:**

*   上午 9:00–面向开发人员的欢迎/ Windows Phone 8 概述
*   上午 9:30–休息
*   上午 9:45–在 Windows Phone 和 Windows 8 之间共享代码、NFC、锁屏、动态磁贴
*   上午 11:45–午餐
*   中午 12:45–语音命令，应用程序购买，诺基亚地图，应用程序到应用程序的通信
*   下午 2:00–结束

如果你没有看过微软的其他 Windows Phone 8 开发者资源，它们也令人印象深刻。他们的 Windows Phone 开发中心为有抱负的 Windows Phone 开发者提供了大量的工具。如果你有超过一天的时间，并且想充分利用这个机会，考虑以下准备:

*   **[下载 Windows Phone SDK](http://www.microsoft.com/click/services/Redirect2.ashx?CR_CC=200117038)**——没有它，你将无法复制手机阵营中展示的技术。
*   **[在 Windows Phone 开发中心](https://dev.windowsphone.com/en-us)** 注册——这将使你加入 [Windows Phone 开发者社区](http://dev.windowsphone.com/en-us/community)，并允许你发布最终完成的 Windows Phone 应用。
*   **[完成 Windows Phone SDK 培训课程](http://msdn.microsoft.com/en-us/WindowsPhoneTrainingCourse)**——开发者可以按照自己的进度完成[初级教程](http://msdn.microsoft.com/en-us/windowsphonetrainingcourse_basic_unit)和[高级课程](http://msdn.microsoft.com/en-us/windowsphonetrainingcourse_advanced_unit)。这些课程是一个完美的起点，也是 Windows Phone 8 手机训练营的理想后续课程。
*   **探索[微软为期 30 天的 Windows 8 开发指导探索](http://www.microsoft.com/click/services/Redirect2.ashx?CR_CC=200117042)**——微软的培训是为许多目的量身定制的，包括 Windows 游戏开发，以及利用你的网络技能来构建 Windows 8 软件。
*   **阅读 BuildMobile 的早期帖子:[立即开始构建 Windows 8 应用的 5 个理由](https://www.sitepoint.com/5-reasons-to-start-building-windows-8-apps-right-now/ "5 Reasons To Start Building Windows 8 Apps Right Now")**——这篇概述强调了微软为吸引像你这样有才华的移动开发者而提供的许多开发者便利。

我们希望周五能见到你！
