# 管理团队的基础知识，第一部分

> 原文：<https://www.sitepoint.com/managing-a-team/>

如果你是团队领导，有自己的员工，或者经常转包工作，你可以做一些事情来营造一个健康、高效和成功的团队环境。如果你是管理他人的新手，这可能会让人不知所措。

这是管理团队的基础知识的第一部分，让新的领导者有一个良好的开端，并为即使是最有经验的经理提供一个检查。

**感谢出色的工作**

当团队成员在某项任务或项目中表现出色时，让他们知道。这是一个团队最容易的士气建设者之一。这也是建立团队团结和忠诚度的有效方法。

**要诚实**

当你不得不告诉某人他们忽略了目标或者他们没有履行他们的职责时，这是很难的。但不幸的是，作为管理者，你最终还是要这么做。诚实、直接和尊重可以让不舒服的情况变得更容易处理。

**有效沟通**

你的团队期待你的指导和反馈。如果你的团队一再失败，你应该审视一下你自己的沟通过程。可能你没有给出完整清晰的信息，也没有以有效的方式传达指令。

不要把它们扔到公共汽车下面

每个人都会犯错，但作为团队领导，你最终是要负责任的。作为一名经理，你能做的最糟糕的事情之一就是因为一个错误而把你的一个员工晾在一边。团队中的每个人都应该负责任，但最终，你需要为你的团队成员尽力，每次都支持他们。

**评估绩效**

定期检查有助于保持团队像一台运转良好的机器一样工作。让你的团队成员知道他们在哪里努力，哪里需要更多的工作，并给他们一个机会让你知道他们对这种情况的看法。

**形成关系**

管理团队的一部分是了解和你一起工作的人。不是所有的关系都必须带有个人性质，但是建立真正的合作关系是关键。

**在信用到期时给予信用**

虽然领导团队的一部分是在出错时承担责任，但你也可以分享团队成员的成功。但要确保庆祝活动渗透到在战壕中工作的每个人。分享荣誉，承认他们在成功中的个人作用。

**雇佣合适的人**

无论是雇佣员工还是分包商，你都要确保你带来了合适的人选。你不仅要确保他们有合适的技能，还要有合适的性格来融入团队。

**确定并分享目标**

你已经知道目标在企业中的重要性，但是不要忽视与你的团队分享这些目标的价值。让每个人朝着相同的目标努力，并使用相同的方法来衡量他们的成功，会有很多收获。

**只是委托**

新任经理面临的一大障碍是对委派工作缺乏安全感。尽管一开始可能很困难，但这是建立团队的第一步。对你的团队成员有信心，给他们所有必要的信息，并相信他们会完成工作。

**保持团队更新**

项目很少完全按照计划进行。客户需求变化，规格变化，新的途径出现。确保一个项目的所有新信息都传递给你的团队成员。

**听取反馈**

你的每个团队成员都有自己过去的经验，以及他们一路走来学到的教训。询问并倾听他们的意见，你可能会对他们带来的价值感到惊讶。

**监控工作**

你已经挑选了一个很棒的团队，你正在委派工作，事情似乎进行得很好。但是不要在开车的时候睡着了。在过程中保持活跃，并定期检查您的团队正在完成的工作，以保持消息灵通并了解过程中的每一步。
