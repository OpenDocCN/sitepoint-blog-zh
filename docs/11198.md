# 别烦人了！如何不在脸书推广你的业务

> 原文：<https://www.sitepoint.com/how-to-promote-your-business-on-facebook/>

使用社交网络的“最佳”方式总是主观的，因为对我有用的不一定对其他人有用。所以我要先说，这篇帖子完全是基于我自己的喜好和经历，尽管我怀疑还有其他人有同样的感受……或者也许你认为我在脸书做的一些事情非常令人讨厌。加入下面的评论，也许我们可以创建一个使用脸书进行商业推广的最佳实践大纲。

以下是你在脸书不应该做的事情(据我所知)来推广你的业务。

## 虐待组消息

我是脸书为数不多的几个团体的成员，其中有几个会向团体成员不断发送信息。这本身并不坏，但是当这些信息完全是自我推销的，与群体的目的无关，或者是每天发送的，这是一个很大的障碍。

与粉丝页面不同，发送给群组的消息被视为普通邮件，因此它们会发送到成员的收件箱中，如果成员在其首选项中设置了该选项，则作为电子邮件通知发送。这意味着在不离开群组的情况下，几乎不可能忽略你不感兴趣的群组信息，我过去就这样做过来结束这种烦恼。

## 一直使用相同的(真正长的)状态

脸书的一个最大特点是能够轻松地与你的朋友分享简短的更新。微博的这一功能不仅让我们有机会更好地了解对方，一次一点点，而且它还提供了一个机会，以一种不引人注目的方式将朋友引向一个网站、新闻、视频或公告。

然而，当状态更新被用于发布迷你博客帖子、销售信函或其他与状态更新的目的不相符的材料时，这变得几乎令人不快。更糟糕的是，一直使用相同的状态更新。这是从我的新闻源中被阻止的快捷方式。

## 假设每个人都在推特上

我们很多人同时使用脸书和推特，所以我们的朋友和关注者之间会有很多重叠。我们中的许多人也使用单独的应用程序或新的内置功能，让我们可以同时在两个地方发布状态更新。这在很多情况下都很棒，因为它让分享信息变得更加容易。

然而，因为脸书和 Twitter 是非常不同的动物，有时这种双重发布可能是无效的。例如，如果你将自己的偏好设置为将你在 Twitter 上发布的所有内容发布到脸书，并且如果你经常发推、转发和/或使用标签，那么这些价值可能会在 Twitter 到脸书的翻译中丢失。

## 盲目的朋友

社交网络，尤其是用于商业推广的社交网络，是一个数字游戏。你联系的人越多，你的知名度就越高，而且(希望)你在市场营销方面的吸引力就越大。但是，对我来说，这并不意味着你应该盲目地和所有人交朋友。

仅仅因为脸书暗示你可能认识一些和你有相同朋友的人，并不意味着你必须和他们交朋友。如果你和你不认识的人成为了朋友，在你的好友邀请上加一条说明是非常好的。不要让我猜测你是谁，为什么我会想和你联系。如果你的私人留言真的提到了我的名字，我会得到额外的加分。

这是我最喜欢的四种不利用脸书做生意的方法。现在轮到你了。关于如何在不影响你的人脉的情况下利用脸书做生意，你有什么建议？
