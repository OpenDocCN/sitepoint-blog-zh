(这些表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏表示比例变化，即上个月有 3.5%的 IE9 用户更换了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)和[StatCounter vs net Market Share](https://www.sitepoint.com/browser-trends-may-2015-can-microsoft-regain-edge/)。)

如此接近，却又如此遥远。Chrome 下滑，过不了 50%的关口。到目前为止，它仍然是使用最多的浏览器，但争夺第二名的竞争变得越来越有趣。

2015 年大部分时间，ie 浏览器和 Firefox 之间的差距一直徘徊在 1-2%左右。IE 有一个合理的月份，大部分时间保持稳定，但离 2001 年 95%的市场份额还有很长的路要走。甚至在五年前，它的使用率超过了 50%。浏览器正处于生命的最后阶段，不久将被微软 Edge 取代。然而，许多人仍然依赖 IE。这是 Windows 的默认浏览器，公司内部应用程序通常依赖它。

Firefox 是解除 IE 束缚的浏览器。早在 2010 年，它的用户数达到了三分之一的峰值，但此后一直在缓慢下降。Firefox 作为一款面向开发者和高级用户的应用享有盛誉。虽然 Chrome 可以被任何人使用，但它提供了更简单的体验，Mozilla 永远不会拥有谷歌的商业影响力。对 Firefox 来说，5 月是艰难的一个月，它进一步落后于 IE——尽管我怀疑长期预测会更好。

这两种浏览器对行业都很重要:

1.  Firefox 是唯一主流的开源浏览器。没有公司股东，也不受商业压力的影响。HTML5 特性是在有用时添加的，而不是因为某个特定的设备或应用程序需要它。
2.  在技术上，IE 落后于竞争对手，但微软为企业应用程序提供了一个稳定的平台。很少有公司会坚持向后兼容。

最重要的是，Firefox 和 IE 是 Chrome 的竞争对手，HTML5 已经蓬勃发展。它可以让发展更具挑战性，但另一种选择是停滞、自满和一家独大的谷歌。

5 月份涨幅最大的是 Safari，奇怪的是，涨幅接近 1%。令人印象深刻，但这可能是几个月平淡无奇后的一次小调整。

最后，有趣的是，IE9 (2.18%)和 IE10 (1.87%)的使用率都落后于 IE8 (3.16%)。

## 2015 年 4 月至 5 月全球移动浏览器统计

移动使用增长了近 1.5%，达到所有网络活动的 34.8%。

热门移动浏览应用:

| 移动浏览器 | 四月 | 五月 | 变化 | 亲戚 |
| 铬 | 32.30% | 33.17% | +0.87% | +2.70% |
| 苹果手机 | 19.49% | 18.87% | -0.62% | -3.20% |
| 机器人 | 17.96% | 17.17% | -0.79% | -4.40% |
| UC 浏览器 | 13.69% | 14.53% | +0.84% | +6.10% |
| Opera Mini/Mobile | 9.09% | 9.92% | +0.83% | +9.10% |
| 不变的 | 2.20% | 2.15% | -0.05% | -2.30% |
| 诺基亚浏览器 | 2.12% | 1.21% | -0.91% | -42.90% |
| 其他人 | 3.15% | 2.98% | -0.17% | -5.40% |

Chrome 的增长开始反映其桌面版的发展，其领先优势看起来不容置疑。

UC 浏览器和 Opera Mini/Mobile 取得了类似的收益。部分原因可能是在亚洲，手机和平板电脑的使用已经与台式电脑持平。

如果你担心 5%的用户仍在使用旧版本的 IE，或者 30%的移动用户采用更简单的浏览器，也许是时候拥抱[渐进式增强](https://www.sitepoint.com/javascript-dependency-backlash-myth-busting-progressive-enhancement/)了，看着你的支持麻烦消失吧！
