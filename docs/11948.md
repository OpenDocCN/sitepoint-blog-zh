# 电子签名服务如何为您节省时间和金钱

> 原文：<https://www.sitepoint.com/electronic-signature-services/>

在管理与客户的合同时，你是如何收集签名的？你会发传真、电子邮件，甚至是普通邮件吗？如果您仍在扫描、传真或邮寄文件以获取签名，使用电子签名可以节省大量时间。

对于那些不使用电子签名的人来说，一个常见的担忧是，电子签名可能不如硬拷贝好。好消息是，自 2000 年美国签署《全球和国家商务中的联邦电子签名(E-SIGN)法案》以来，电子签名已经与物理签名一样具有法律约束力。类似的法律已经在各州通过，在欧盟也是如此。

**好处**

除了通过消除签名、扫描、传真和邮寄来节省时间之外，使用电子签名系统还有许多好处。以下是一些例子:

*   减少等待时间——如果你通常等到签署协议后才开始工作，你可能要等上几天甚至几周才能开始工作。对于您的客户来说，只需点击几下鼠标就可以更快、更容易地签署协议，并让您开始行动。
*   省钱——节省纸张、墨水和邮费。另外，这对环境更好！
*   为所有各方提供副本–一旦协议签署，所有各方都会收到一份数字副本，便于他们日后参考并将其存档在自己的计算机上。
*   创建存档–许多电子签名服务都包括存档功能，因此您可以在一个位置轻松访问所有活动文档。
*   提高安全性——电子签名意味着数字加密，在电子签名时，伪造您的身份要困难得多。

**一次警告…**

因为电子签名执行起来非常快速和容易，所以签署方可能很容易不完整阅读协议。作为服务提供商，你的部分责任(也是避免未来合同纠纷的一种方法)是确保你的客户不仅签署你的协议，而且理解其中的条款。在签字之前，给他们时间回顾并询问他们可能有的任何问题。它更快更容易，但这并不意味着内容不重要，不应该受到合同审查过程。
