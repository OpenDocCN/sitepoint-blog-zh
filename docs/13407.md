# 选择合适的利基市场

> 原文：<https://www.sitepoint.com/pick-the-right-niche/>

并非所有的利基(主题)都是平等的。我们在建立网站网络时发现的一件事是一些市场的竞争有多激烈。

问题是:假设你建立了一个游戏网站。你很快就会变得受欢迎，比如说每天有 2000 个独立访问者，每天有 20000 个页面浏览量。不错吧？开了一个好头。所以你想进入你最喜欢的广告网络，结果被拒绝了。他们的最低要求是 2000 个独立访问者，这有什么问题呢？问题是有太多的游戏网站在竞争这些广告，以至于没有足够的广告。博彩业花了很多钱在网上推广，但是网站的数量远远超过广告商的数量。游戏网站的最低访问量突然变成了每天 10，000 个独立访问者！

这并不是说你不能经营一个成功的游戏网站，你只需要准备好与你的竞争对手进行一场严肃的战斗。你需要有一个更好的网站，更好的内容和更多的访问者。但如果你能击败竞争对手，这是一个报酬丰厚的利基市场。如果你对一个话题有热情，并且全身心地投入，你就会成功。但如果你像我们经常做的那样，纯粹从商业角度来看，那么还有很多竞争不那么激烈的利基市场。

例如，现在对女性有吸引力的话题需求量很大。现在广告商比网站多，所以有大量的广告收入。这些信息不会在任何地方发布，我们必须通过与我们的广告商和广告网络合作来了解这些信息，他们会不断地回来问我们“你们还有其他针对女性的网站吗？如果你做“各种各样的事情”,我们可以提供更多的广告清单。

当你为你的网站选择主题时，你不仅仅要看在搜索引擎中超过他们有多难，还要看这个主题的广告市场有多饱和。你可能已经使用 Adwords 来查看一个关键词的价格，并在 Google index 上查看你面对的网站数量，但是现在你也需要考虑一下广告商。
