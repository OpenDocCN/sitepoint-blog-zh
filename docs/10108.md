# 是时候做出改变了

> 原文:[https://www.sitepoint.com/its-time-to-make-a-change/](https://www.sitepoint.com/its-time-to-make-a-change/)

改变是好事。它可能很可怕，但它伴随着可能性、机遇和潜力。无论你是在考虑创业、转行、改变你的服务项目，还是结束与一个长期客户的关系，都很难放下你非常熟悉的东西，接受未知。

一旦你做到了，那么*改变*就能*永远改变*你的生活。

但是你怎么知道是时候拥抱变化了呢？这里有一些方法可以告诉你是时候做出改变了。

## 你失去了激情

人们做事情有很多不同的原因，但最有力的原因之一是因为激情。生活中最美好的事情是那些我们仅仅因为喜欢而做的事情。但是，有时候，激情会消退。

例行公事可能会扼杀激情。知道你每天都在做什么有时可能是一件好事，但它也可能开始削弱你曾经对做某事的热情，让你质疑你是否应该继续走同一条路。

无论你如何到达那里，最终结果都是一样的——激情的丧失会导致改变。

## 你发现了新的激情

就像你会失去激情一样，你也可以培养新的激情。你还记得当你第一次意识到你有多爱你的工作时是什么样的吗？

也许你的激情驱使你去创业，改变你的商业模式或者与同事合作。也许你的热情帮助你走出了舒适区，或者意识到什么是你生活中真正令人满意的。激情是一种神奇的东西；它可以激励你，让你夜不能寐，思考各种可能性，帮助你排除杂念。

我们的激情变成了我们的孩子，我们无时无刻不在想的事情，并且愿意为此做任何事情。当你有了新的激情，其他一切都退居其次。这是新的和令人兴奋的，这可能会导致你做出改变。

## 你的目标变了

生活变了，重点变了，目标也变了。不管是渐进的还是突然的，你可能正处在一条改变的道路上，这需要你在生活的许多不同方面做出艰难的决定，以便让改变发生。

如果你的日常活动不再支持你的长期目标，那可能是你即将改变的信号。

## *个人笔记*

这是我在 SitePoint 上作为主要商业博主的最后一篇文章。我要感谢 SitePoint 的每一个人欢迎我加入这个团队，让我成为他们出色工作的一部分。感谢那些在过去的两年里评论、互动、提问、逗我笑、挑战我去换个角度思考的读者们。是你让我在这里写作成为如此独特的经历，所以谢谢你。

如果你像我一样，是一个习惯性的生物，那么认识到什么时候该做出改变可能是一个挑战。我喜欢套路；我喜欢时间表；我喜欢结构。对于那些读过我文章的人来说，这并不奇怪。结合我通过帖子与 SitePoint 社区建立的关系，这让我做出离开 SitePoint 的决定非常困难。

我可能会不时回来，但你不会经常在这里看到我。我已经接受了改变，并将尽我所能为我的新方向努力。说到底，如果你不愿意接受改变，你就无法成长，而我已经双脚跳进了火里。

我祝你们所有人成功，再次感谢你们的阅读。没有你，这种变化是不可能的。
