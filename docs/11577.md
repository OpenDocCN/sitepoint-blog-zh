# 你不想犯的 38 个营销错误

> 原文:[https://www.sitepoint.com/common-marketing-mistakes/](https://www.sitepoint.com/common-marketing-mistakes/)

营销可能是商业中具有挑战性的一部分，很多时候，需要大量的尝试和错误来学习营销你自己和你的企业的最佳方式。再加上每个企业、行业和目标受众的独特性，在你找到一套成功的营销活动之前，你会经历大量的反复试验。

为了帮助新的营销人员在他们自己犯错误之前从常见错误中吸取教训，我整理了 38 个很多人都会犯但完全可以避免的营销错误……这将帮助你更成功地营销你的业务。

## 常见的营销错误

1.没有准备营销计划、 [USP](https://www.sitepoint.com/how-to-create-unique-selling-proposition/) 和预算。

2.瞄准所有人，而不是专注于特定的目标受众。

3.歪曲你的经历，提出没有根据的主张。

4.与你的营销信息不一致。

5.忽视你现有的付费客户，只对陌生人营销。

6.没有听取对你营销活动的反馈。

7.跟随竞争，而不是在市场上创造自己的位置。

8.没有迅速跟进线索。

9.上市前没有编辑/校对你的副本。

10.低估客户服务和口碑的力量。

11.不要一直带着名片。

## 在线营销的错误

12.忘记在不同的平台和浏览器上测试你的站点。

13.永远不要更新你的网站设计或内容。

14.假设 SEO 会照顾好自己。

15.在撰写网站内容时，忽略目标受众的需求。

16.不包括容易找到的联系方式。

17.用广告覆盖你的商业网站，分散你的核心信息。

18.从竞争对手的网站复制内容。

19.没有提供一个信息丰富、引人入胜的“关于”页面。

## 电子邮件营销错误

20.向没有选择加入的人发送消息。

21.出售你的订户的联系信息。

22.通过发送不相关和不想要的消息来公然发送垃圾邮件。

23.没有在许多类型的电子邮件客户端中测试 HTML 电子邮件。

24.使得订阅或退订变得困难。

25.不要在每封邮件中包含具体的行动号召。

26.不测试电子邮件个性化以避免<firstname>标签。</firstname>

27.在你的电子邮件中塞满相互竞争的信息。

28.低估你的主题。

29.留下文案作为事后的想法。

30.忽略你的统计数据，没有相应地修改你的信息。

## 社交媒体营销失误

31.忽视你的网络声誉。

32.忘记了这是一场马拉松，而不是短跑，并且塞得太多、太快。

33.没有定期更新您的个人资料并保持一致。

34.专注于销售而不是建立真正的关系。

35.忽视了对话的重要性。

36.在社交媒体上没有可识别的形象。

37.没有定期更新你的博客。

38.试图时时刻刻无处不在。
