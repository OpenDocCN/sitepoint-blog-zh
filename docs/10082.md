# 浏览器市场份额和趋势，2011 年 3 月:IE6 使用率降至 5%以下

> 原文:[https://www . site point . com/ie6-usage-below-5-percent-browser-trends/](https://www.sitepoint.com/ie6-usage-below-5-percent-browser-trends/)

我最近一次关注[市场趋势是在 2010 年 12 月初](https://www.sitepoint.com/chrome-7-overtakes-ie7-third-placed-browser/)。当时，Chrome 刚刚超过 IE7 成为第三大用户浏览器。今天对开发者来说还有一些好消息: **IE6 的全球使用率已经低于 5%** 。仍然有很多人在使用这款浏览器，但它现在落后于 Safari，如果幸运的话，到 2011 年底可能会变得微不足道。

我们正处于浏览器世界几个有趣发展的风口浪尖——特别是 IE9 和 Firefox 4.0 的发布——所以现在是分析市场现状的好时机。让我们来看看最新的 [StatCounter 统计数据](https://gs.statcounter.com/) …

| 浏览器 | 十一月 | 二月 | 变化 | 亲戚 |
| IE 9.0 测试版 | 0.32% | 0.48% | +0.16% | +50.00% |
| IE 8.0 | 29.49% | 30.30% | +0.81% | +2.70% |

此表显示的是市场份额，而不是绝对的使用数据。互联网的使用正在增长，尤其是在非洲和亚洲，所以浏览器有可能在失去市场份额的同时赢得更多用户。

“变化”栏显示市场份额的绝对增加或减少。“相对”栏表示比例变化，即在过去两个月中，15.2%的 IE7 用户迁移到其他浏览器。

和往常一样，最大的整体赢家是谷歌 Chrome。尽管使用模式很复杂，但 Chrome 有效地赢得了用户，其代价是 Internet Explorer 的用户减少，Mozilla Firefox 的用户减少。尽管 IE8 的使用有所增加，IE 的总体市场份额现在保持在 45%，不到其全盛时期用户比例的一半。

苹果 Safari 显示了另一个适度的增长。我怀疑苹果电脑、iPhones 和 iPads 的持续流行是主要原因… *特别是当许多用户被禁止使用其他浏览器的时候*。

Opera 的市场份额基本保持在 2%左右，因此它跟上了互联网的整体增长。Opera 11 在速度和 HTML5 功能上与 Chrome 不相上下，所以很遗憾它没能突破这一障碍。诺基亚采用 Windows Phone 操作系统可能也无济于事。

不出所料，随着发布日期的临近，IE9 和 Firefox 4 测试版的份额有所增加。以下是我对未来几个月的预测:

1.  Firefox 4.0 和 IE9 将在 2011 年 Q2 奥运会开始的几天内相继发布。
2.  微软将试图超越 Mozilla，反之亦然。这会变得有点傻，很少有用户会在意。
3.  火狐 4.0 会很快被采用。Mozilla 的自动更新和浏览器更具技术性的用户基础将超过 IE9 的早期收益。
4.  当微软将其作为 Windows 7 和 Vista 用户的自动更新推出时，IE9 的市场份额将赶上 Firefox 4.0，甚至可能超过它。
5.  Chrome 的增长将开始趋于平稳，因为它目前相对于竞争对手的速度优势正在减弱。
6.  到今年年底，IE9 和最新的 Firefox 都将拥有大约 20%的市场份额。然而:
    *   Mozilla 表示，他们将实施类似 Chrome 的发布时间表，Firefox 7 可能会在 2012 年之前发布。除非这些更新完全自动化，否则使用模式会变得有点复杂。
    *   IE9 仅在 Windows 7/Vista 上可用。50%的 Windows 用户仍然使用 XP，因此 IE9 的采用率只会随着操作系统的升级而增加。因此，IE8 将继续受欢迎，尽管随着 XP 用户更愿意迁移到其他地方，IE 的整体市场份额将会减少。
