# Linkalicious Friday: HTML5 时代精神、新 PHP 和 AWS 好东西

> 原文:[https://www . site point . com/link alicious-Friday-html 5-zeitgeist-new-PHP-and-AWS-goodies/](https://www.sitepoint.com/linkalicious-friday-html5-zeitgeist-new-php-and-aws-goodies/)

我们 12 月份的销售已经进入第二周，而且势头强劲，所以 SitePoint 一直很忙。它在其他网站上也很忙:

*   谷歌刚刚[发布了](http://googleblog.blogspot.com/2010/12/zeitgeist-2010-how-world-searched.html)其[年终搜索精神](http://www.google.com/zeitgeist2010):2010 年搜索次数最多的词汇，以及最新和增长最快的词汇的汇总。今年，谷歌增加了一些流行搜索词的漂亮的 HTML5 可视化，所以仅仅为了令人惊叹的因素就值得一试。
*   PHP 5.3.4 今天发布[。这是一个专注于 bug 修复的维护版本，所以在 changelog 中没有什么特别令人兴奋的东西(除非你一直在努力解决其中的一个 bug！).尽管如此，这对各地的 PHP 开发人员来说都是好消息。](http://www.php.net/index.php#id2010-12-10-1)
*   本周来自亚马逊网络服务的消息数不胜数:他们发布了 Android 和 iOS 的 SDK，为所有 EC2 实例添加了免费的基本 Cloudwatch 监控，T4 推出了一项新服务:Route 53，一个可编程的域名服务。所以对于那些用 AWS 开发的人来说，你有相当多的新玩具可以玩！

这星期就这些了。祝大家周末愉快！
