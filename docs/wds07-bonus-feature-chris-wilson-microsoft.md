# WDS07 附加功能:克里斯·威尔逊(微软)

> 原文：<https://www.sitepoint.com/wds07-bonus-feature-chris-wilson-microsoft/>

![Chris Wilson presenting at Web Directions South 2007](img/077b88fa18036ce61a7a657c2bf773c0.png)

本周在悉尼举行的 2007 年南方网络方向大会上，我采访了微软的 ie 浏览器平台设计师克里斯·威尔逊。自从*有了*网络浏览器以来，Chris 就一直在开发网络浏览器，很高兴在会议的最后一天和他坐在一起。

在他的演讲“推动网络前进”中，Chris 向观众展示了开发世界上最流行的网络浏览器的现实。有超过 500，000，000 个用户需要回答，“不要破坏网络”已经成为该公司在开发下一个版本的 Internet Explorer(目前称为 IE)的工作中压倒一切的口号。下一个)。

我借此机会问了 Chris 几个问题，这些问题是我在听完他的演讲后想到的:

> 凯文·杨克
> 
> “你的团队正在为下一版本的 IE 构建一个全新的渲染引擎。”
> 
> 克里斯·威尔逊
> 
> “我们现在真正关注的是布局引擎部分。确定如何布局文本内容、边框等的部分。并且能够管理一般的文本流。[……]我们意识到，你知道，‘哇——我们*是*微软。这里有很多有设计专长的人。有很多人真的知道如何做好布局。[……]我们需要解放这些人，真正允许他们做一些真正不可思议的事情。
