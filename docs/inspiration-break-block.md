# 灵感——打破僵局！

> 原文：<https://www.sitepoint.com/inspiration-break-block/>

***“还不如找份正经工作。”
“我在开什么玩笑，以为人们会付钱让我靠这个谋生？”
“就是这样！我放弃！这是不可能的！”***

如果在你的职业、爱好或者作为一名网页设计师/网站管理员/自由职业者的生活中，你曾经咕哝过这些短语中的一个，那么你并不孤单。事实上，我就在那里和你在一起，在过去已经很多很多次了。如果你可以诚实地说你从未有过，甚至接近于有过以上的想法，那么现在就预订这篇文章，以后再来。我向你保证你会回来的:我们都会在某个时候被困在这里。

作家将这种感觉称为“作家阻塞”或干旱期。这是一个时期，你似乎不能让事情流动，没有点击，或者你只是没有动力。在这个行业，干旱期通常是一个比买不起 Adobe Photoshop 更大的问题。为什么会是这样的问题？对于初学者来说，它可以让你彻底放弃，放弃，永远不再回到网站管理员的世界。

别害怕，因为琼斯在这里！我已经收集了一些想法，我用它们来让果汁再次流动，让我回到正轨。

***1。修改过去的作品***

看看你过去的工作，看看是否有什么可以改进的地方。在一些项目中，简单的改变可以带来巨大的不同，这是一个很好的思想热身。

***2。承担一个小而有趣的项目***

为朋友、同事或熟人制作一个简单的网站。有时这些项目非常简单，你不会在上面花费超过一两天的时间。但是它们是无价的——它们带你回到简单的编码时代，并给你一个积极的工作前景。没有什么比一句“祝贺你”更能激发动力了

***3。学习别人的***

看看别人的辛勤劳动。不要去想你有多想成为他们那样而压抑自己，而是关注你能从研究他们的技术中学到什么。去我们自己的论坛，评论别人的网站，或者访问像威比奖这样的网站。这些网站很棒，因为它们真的突破了极限，有更多的内容和更多的视觉享受。有时你可以得到一个按钮、横幅或者更好的布局的想法！

***4。超越网页***

滚出去！有时候我们想得太数字化，太网络化了。走出家门，开车出去，去看电影或逛书店。我们周围有无数的形状、形式和风格。有一次，我看到一个小孩在公园里玩耍，就有了制作横幅的想法。让你的整个世界成为你的灵感。

***5。嘟嘟***

没错——拿起一支铅笔和一张纸，只是画画，或者涂鸦，或者只是让你在纸上手舞足蹈。有时候，在使用记事本或 DreamWeaver 之前，最好先在纸上构思你的网站。

***最后……***

如果这些对你都不起作用，那就休息一下。离开电脑，做一些你喜欢的事情。灵感会回到你身边，如果你和我一样，它会发生在你开车的时候，或者和朋友在一起的时候，或者在你离电脑很远的时候…
