# 书签同步的 10 个 Xmarks 替代方案

> 原文:[https://www.sitepoint.com/xmarks-replacement/](https://www.sitepoint.com/xmarks-replacement/)

本周早些时候，克雷格分享了一个令人震惊的消息，即 [Xmarks 将停止跨浏览器书签同步](https://www.sitepoint.com/xmarks-browser-bookmark-synchronization-closure/)。真的没有另一个相同的服务，所以这对于依赖 Xmarks 自动访问书签的 200 万用户来说是一个令人沮丧的消息，不管他们使用的是什么浏览器或电脑。

很明显，找到一个合适的 Xmarks 替代品将是一个挑战，因此许多用户希望 Xmarks 改变步伐，被一个大公司收购，或者甚至可能推动服务开源，以便它可以继续下去。但是，如果这些可能性都没有实现，我们需要一个 Xmarks 的替代品，这样我们就可以继续同步书签。

这里有一些可能性，虽然它们可能不会以同样的方式单方面取代 Xmarks 所做的一切，但其中一个可能是适合你的 Xmarks 替代品。

注意:我仍在探索供我自己使用的选项，但我很想听听你找到了什么替代品。有更好的解决方案吗？在下面的评论里分享吧！

## [庄家](http://www.sheepsystems.com/products/bkmx/)

Mac 用户的书签同步工具，允许用户将书签组织到收藏中，然后跨浏览器和在多台 Mac 上自动同步。有一个演示，然后需要一次性授权费。

## [Chrome 同步](http://www.youtube.com/watch?v=l2cR30Q7WVk)

一个免费的内置书签同步功能，允许 Chrome 用户在任何计算机上的 Chrome 浏览器之间同步他们的书签。

## [好吃](http://www.delicious.com/help/tools)

一种社会化书签服务，允许你在线保存链接并从任何电脑或浏览器访问它们。也有针对特定浏览器书签的书签小程序和插件。

## [Evernote](http://evernote.com)

一个笔记应用程序，通过跨平台同步功能提供了一种非传统的方式来保存和同步书签。通过在各种设备上运行 Evernote，并将链接保存到新笔记的 URL 字段中(或使用 Web Clipper 工具)，您可以绕过特定于浏览器的书签工具。

## [火狐同步](https://addons.mozilla.org/en-US/firefox/addon/10868/)

一个免费的书签同步插件，允许你在不同的电脑上同步和访问你的书签。

## [手机短信](http://www.apple.com/mobileme/)

一项付费服务，允许您在 Mac、PC、iPhone 和 iPad 之间同步数据，包括您的书签文件夹。

## [以后再看](http://readitlaterlist.com/)

这款工具可以让您在所有设备上访问阅读列表，还可以用来为网站添加书签，并从多种浏览器、多台计算机和移动设备上访问它们。

## [Sync2It](http://www.sync2it.com/)

一个免费的书签同步器，可以在不同的计算机、不同的平台和不同的浏览器上自动同步所有的书签。所有书签都是保密的，也可以通过移动设备访问。

## [TidyFavorites](http://www.tidyfavorites.com/favorites/synchronize%20bookmarks.html)

一个可以在 IE、Firefox、Opera 和 Google Chrome 上同步书签的 Windows 应用程序。免费版允许你手动同步，升级版包括自动同步。

## [Windows Live Sync](http://sync.live.com/)

一个免费的同步程序，是 Windows Live 软件包的一部分，允许您在多台 Windows 电脑之间同步文件，包括您的书签文件夹。
