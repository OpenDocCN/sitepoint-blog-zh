# 自由职业者的非虚拟世界:面对面会议

> 原文：<https://www.sitepoint.com/successful-in-person-meetings/>

对我们许多人来说，很多业务都是通过电子邮件和电话在网上进行的。但是你有没有考虑过接触当地的企业来寻找新的机会呢？这可能是一个拓展业务和与当地社区建立有价值关系的好方法。

非虚拟业务的一个重要方面是愿意并能够成功地与客户面对面交流。如果你以前只专注于网上做生意，这可能是一个挑战。为了进行一次成功的面对面会谈，需要考虑很多事情。

**安全第一**

你应该始终确保在安全的公共场所会面。是的，就像相亲一样。由于你的许多潜在客户也可能经营家庭业务，可能没有固定的办公场所来举行会议。以下是一些需要牢记的安全提示:

*   在中心公共场所见面，如餐馆或咖啡店
*   如果可能的话，带上一个商业伙伴或同事
*   与信任的朋友分享你的会议细节
*   把你的手机放在手边

你当然不希望在与潜在客户会面时做最坏的打算，但是做好准备并意识到潜在的问题有助于保证你的安全。而且这不仅仅适用于女性！

**留下良好的第一印象**

不用说，你会希望准时参加会议。尽量不要迟到，但是如果你发现你落后了，打电话给客户让他们知道。

即使你的会面是在一个轻松的地方，比如咖啡厅，展现出一个专业和有经验的形象仍然很重要。穿着得体(即不是三件套)，但仍要努力让自己看起来很专业。

要自信，让你的肢体语言支撑你的自信。交流时，握手要有力，微笑并进行眼神交流。记得放松，做你自己。
 **为你的会议做准备**

开会前，明确议程，以便做好相应的准备，并准备好回答任何预期的问题。

花时间尽可能多地了解客户和他们的需求。通过了解客户的需求，你就能更好地展示你如何满足他们的需求。

带上营销材料，比如名片、小册子、CD 文件夹以及客户可以带回家的任何其他材料。如果你在一个可以上网的地方，带一台笔记本电脑，这样你就可以展示你工作的实例。确保你带了一个记事本，这样你就可以在会议期间做笔记。

如果您计划使用 PowerPoint 演示文稿，请确保您已经练习过并准备好进行演示。万一出现技术困难，有一个备用计划也是一个好主意。
