# 拖延指针

> 原文：<https://www.sitepoint.com/procrastination-pointers/>

拖延是我们个人生产力的最大敌人之一。想做某事并计划去做是好的，但是如果我们不能向前迈进呢？

拖延一天中不重要的事情是一种有用的天赋。然而，对许多人来说，问题是我们在拖延我们一天中重要和关键的事情，降低了我们的个人生产力，增加了我们的压力水平。

这里有五点可以帮助你更好地克服拖延症。(您可以现在实施，也可以明天实施，或者更好，下周实施！)

**1。前一天晚上的日常计划。**

“人们没有计划失败，但他们有时没有计划”。如果在你上班之前没有一个行动计划，你很容易被“东西”困住。电话响了，有人来了，你把时间花在了回应那些要求你注意的最大声的声音上，而不是你盘子里最重要的事情上。前一天晚上准备的行动计划就像是第二天的路线图。你知道你的下一步应该是什么，让你采取有成效的行动，远离拖延。

**2。用干净的桌子工作。**

“眼不见，心不烦。”反之亦然。当它在眼前时，它就在脑海中，我们大多数人都忍不住会分心，然后我们的时间被导向不太重要和容易的任务，导致我们推迟更重要的任务。在干净的桌子或干净的工作环境下工作，可以让我们只处理面前最重要的任务，这样我们就可以把所有的注意力都集中在那个任务上，而不受其他视觉干扰。

**3。将大型项目缩减为小块。**

你如何吃掉一头大象？一次一口。明天你计划做一个三小时的项目。问题是，我们中的许多人没有三个小时来做任何一件事情。我们不得不应付干扰、会议等。(我不知道我甚至有持续三小时的注意力持续时间！)而且我们经常因为“没有足够的时间来完成这个任务”而拖延工作。所以，不要把整个三小时的项目安排在明天，而是安排一点点，一两步，可能需要 20 或 30 分钟。然后把下一步放在第二天的清单上，下一步放在第二天的清单上，等等。这可能需要几天时间，但你会把那头大象一口一口吃掉。

**4。针对干扰做好计划。**

中断往往以可识别的模式出现。我在一天的早些时候被打断，而不是晚些时候。我大部分的干扰都是在一周的早些时候，而不是晚些时候。所以，如果我周一早上第一件事就是计划一个大项目，我就是在制造压力，因为我刚开始，干扰就来了，重新集中了我的注意力，导致我拖延了我真正想做的事情。顺流而下比逆潮流而动要容易得多。因此，我把那些较大的项目安排在一天的晚些时候和一周的晚些时候，因为那时我会受到较少的干扰。

**5。指定截止日期。**

你曾经无法实现新年计划吗？如果是这样，那可能是因为你没有设定最后期限。截止日期促使我们采取行动。如果没有截止日期，事情就会堆积在我们的“尽快”堆里，一个“永不落地”的地方，在那里，事情会在“某一天”，“当我有时间的时候”得到处理。设定一个最后期限，你就会开始行动。
