# 德国考虑禁止谷歌分析

> 原文：<https://www.sitepoint.com/germany-considers-google-analytics-ban/>

全球最受欢迎的网站统计分析系统 Google Analytics 在德国可能是非法的。负责国家数据保护的政府官员正试图禁止分析，理由是它违反了隐私法。

整理详细的 web 统计数据曾经是一项耗时的任务。网站所有者要么求助于基本的页面计数器，要么求助于网络服务器日志分析器，比如 AWstats 或 T2 的 WebTrends。虽然这些提供了有价值的数据，但这些信息通常更适合系统管理员，而不是互联网营销人员。

谷歌于 2005 年 4 月从顽童软件公司收购了 Analytics。虽然它最初是一个商业软件包，谷歌重新标记了这个系统，并从 2006 年开始允许无限制的免费使用。Analytics 不是分析日志文件，而是使用一个 JavaScript 代码片段，放在网站的每个页面上。该代码收集访问者数据，并将其发送到谷歌的服务器进行处理。

分析的真正力量在报告中显而易见。数据通常在几个小时的整理时间内得到处理，该系统提供了大量的分析工具。仪表盘和吸引人的图表是营销人员和统计人员的梦想。今天，很难找到一个不使用谷歌分析的主要网站，包括大约 13%的德国出版商。

谷歌分析的成功引起了德国隐私保护官员的担忧。可以想象，谷歌将能够跟踪个人在整个网络上的活动，并整理来自银行和保险公司等持有个人数据的网站的信息。理论上，谷歌可以创建包括个人兴趣、职业、生活方式、财富、健康、政治和性偏好等信息的个人资料。如果这个人有一个谷歌账户，该公司可以将一个档案与一个已知用户进行匹配。

德国当局还担心数据被转移到国外，储存在美国服务器上。美国的隐私保护法不同，谷歌可能会将信息转移到没有此类法律的国家。

没有证据表明谷歌以这种方式对用户进行描述，但该公司对数据是如何整理、处理和使用的保密。谷歌发言人辩称，用户可以通过禁用 JavaScript 或拒绝 cookies 来退出数据收集。

不幸的是，德国律师已经对反隐私案件垂涎三尺。其中一人声称，使用谷歌分析的德国网站可能被罚款高达 75，000 美元。

你在你的网站上使用分析吗？您担心潜在的隐私泄露吗？还是系统有用到隐私不再重要？
