# 16 个最常见的 WordPress 错误

> 原文:[https://www . site point . com/16-of-the-most-common-WordPress-errors/](https://www.sitepoint.com/16-of-the-most-common-wordpress-mistakes/)

就像太阳从东方升起，在西方落下一样，生活中有一个事实是不容置疑的——我们都会犯错。虽然我们不能做很多事情来防止它们，但我们可以从错误中吸取教训。更好的是，我们可以从别人的错误中吸取教训！

在这里，我列出了 WordPress 用户和开发者最常犯的 16 个错误。重温一下别人做错了什么，这样你就可以在自己的网站上避免同样的错误。

## 1.错过了最好的插件

你需要一些时间来了解 WordPress 的所有插件。通过反复试验，你会发现什么最适合你。然而，我们会让这个小小的研究项目变得无比简单。你*必须*有这些插件:

*   Google XML 网站地图
*   联系表格 7
*   阿基斯米特
*   所有在一个搜索引擎优化包

## 2.忘记备份你的网站

没有什么是完美的，尤其是在科技领域。崩溃会发生。如果你没有准备好，一个非常痛苦的情况可能会向你走来——丢失数据。您可以手动备份您的站点。转到工具，然后导出-或者让生活更容易，使用自动备份插件，如 WP-DB-Backup。

## 3.使用不友好的永久链接结构

你的 WordPress 默认设置不支持搜索。为了让你的 SEO 获得最大收益，确保你在永久链接结构中使用关键词。你应该这样做:

*   转到选项
*   在管理面板中选择永久链接
*   在自定义文本框中，输入“/% category %/% postname %/% post _ id”

## 4.仅使用一台服务器

如果您是 WPWebHost 用户，可以跳过这一步。如果您不是 WPWebHost 用户，请继续阅读。

不要低估内容交付网络(CDN)的价值——它真的可以改善你的读者体验。你应该这样做:

*   研究 CDN，如 MaxCDN 或 Cloudflare
*   将您的内容复制到世界各地的几个镜像服务器上

这将使访客更容易查看您的页面，无论他们从哪里访问。它减少了加载时间，因为不是所有的请求都被发送到主 web 服务器。

## 5.不是咨询分析程序

你会在晚上开着没有前灯的车吗？没有。那你为什么不咨询分析程序就开博客呢？谷歌分析监控网站流量，并记录引导你的网站的关键词。试图在没有分析的情况下给目标受众写信就像把你的钱扔出窗外。

## 6.没有集成网站管理员工具

如果没有一点帮助，搜索引擎不会爬得很深。如果你想让你所有的网页都被索引，提交一个 XML 站点地图到 Google 网站管理员工具。利用网站管理员工具也意味着你将获得大量有用的统计数据。

## 7.忽略谷歌著作权标记

谷歌原创标记的目的是增加可信度和最大化曝光率。谁不想这样呢？这个过程很简单，结果是在搜索结果的元描述旁边有一张照片。

## 8.没有面孔

你知道 Gravatar 是什么吧？如果没有，Gravatar，或全球公认的头像，是出现在博客评论或论坛旁边的图像。它是代表你在整个互联网上的形象。就像 Google Authorship 一样，Gravatars 提供可信度并吸引注意力。如果你没有使用 Gravatar，尽快买一个。注册是一个简单的一次性过程，它将对你的博客产生重大影响。

## 9.使用默认的收藏夹图标

favicon 是与您的网站相关联的图标。它将显示在浏览器的地址栏中，标签页标题的旁边，书签列表中页面名称的旁边。大多数 WordPress 新手使用主题附带的 favicon。但是，如果你想看起来更专业，你真的需要有一个自定义图标。你应该这样做:

*   在网上搜索免费的图标生成器
*   创建你的文件(通常是 16 像素乘 16 像素)
*   转到主题的图像部分
*   用您的自定义设计替换默认的 favicon

## 10.使移动用户的生活变得困难

如今，每个人都有智能手机。基本上，如果你希望收到任何流量，你最好确保你的网站是移动用户友好的。你有两个选择。首先，你可以使用类似 WPtouch 的插件为移动设备定制你的网站。你的另一个选择是使用一个 WordPress 主题，它可以响应智能手机、iPod touch 和平板电脑。

## 11.使用 cPanel 文件管理器进行 FTP 传输

选购 FTP 客户端需要时间。然而，FTP 客户端提高了许多博客所有者的生活质量。虽然 cPanel 中的文件管理器界面较慢，并且经常意外崩溃(或关闭)，但 anFTP 客户端可以轻松上传、下载或更改服务器上的文件权限。

## 12.信息超载

你可能认为你在设计网站时已经考虑到了读者。然而，也有可能你在设计你的网站时已经考虑到了你的钱包。读者喜欢简单。不要用超级杂乱的边栏淹没它们；太多的广告、小工具和链接会分散注意力。将你的边栏限制在最重要的信息上。

## 13.传递博客创作工具

为什么，哦为什么，你要让自己受制于 WordPress 的仪表盘？使用博客创作工具！像 Windows Live Writer 这样的工具是为离线写博客而设计的。你可以写下一篇文章，然后用简单易用的格式工具来增强它。

## 14.忽略 WordPress 更新

WordPress 不断发布新的更新。如果博客所有者忽略这些更新，网站将面临安全问题。有一个简单的解决办法——不要忽视 WordPress 的更新！这些新版本的创建是有原因的！

## 15.利用一个有问题的托管公司

如果你想省钱，选择超低价格的托管公司似乎很有吸引力。然而，这些公司中的许多毕竟不是这么便宜。它们的低价是由于功能较少和支持不令人满意的结果。不要和你发现的最便宜的公司签约，而是咨询专家。获得一些关于哪些托管选项是最好的建议。

## 16.选择有缺陷的主题

在选择主题时，有两件事你应该考虑——初学者的易用性和主题公司的声誉。在你找到了一家值得尊敬的公司和一个易于操作的主题后，你就可以担心价格了。仅仅基于价格的选择可能意味着使用与你的网站(或你)不兼容的主题。
