# “重新设计”不是一个肮脏的词

> 原文:[https://www.sitepoint.com/redesign-not-dirty-word/](https://www.sitepoint.com/redesign-not-dirty-word/)

 ****##### 在重新设计你的网站之前，你需要知道什么？** 

唯一不变的是变化。至少就你的网站而言。

一些网站——就像海洋中的波浪——每小时会变化很多次。

一些变化——比如潮汐——一天一两次。

许多仍然像停滞不前的死水——持续的，不变的，荒凉的，被忽视的，很大程度上没有人来过！

网络冲浪者是一群没有耐心的人。他们在兴奋、即时满足、不断变化和改变中茁壮成长。没有什么比一个日复一日、周复一周保持不变的网站更让他们厌烦的了。门户和社区站点了解这一事实并加以利用。设计变更来得又多又快。对他们来说，网站重新设计是日常生活的一部分，或者至少是每月一次。

因此，问题不是你的网站是否需要重新设计，而是什么时候和多长时间一次？在这篇文章中，我将讨论许多需要思考的问题，许多需要回答的问题，以及重新设计网站需要采取的不同步骤。

谁需要这些信息呢？

你知道！

如果你是一名网站开发人员，设计网站或为客户的网站项目提供咨询，你无疑会被要求修改和改进网站或维护一个需要频繁更新的网站。如果你拥有一个网站或一个在线业务(现在哪些没有呢！)，当您计划改进您的站点时，这些信息是非常宝贵的。

所以让我们开始吧。你只需要问自己三个问题:

1.  你的网站现在有什么问题？
2.  有哪些可以改进的地方？
3.  你怎样才能做得最好？

##### 你的网站有什么问题？

如果没坏，就别修。但是后来几乎都破产了。或者将要。所以，在你被迫改变之前，要明智地改变。测试错误和缺点有很多方面，也有很多方法。您可以:

*   向网站的用户和访问者征求反馈
*   进行正式的可用性测试
*   聘请顾问

您选择使用哪一种取决于您的需求、预算和业务的复杂性。Amazon.com 也许有理由以每小时 5000 美元的价格雇佣一名顾问。你也可以为你的利基网站做一份给网站访问者的反馈问卷。

这些方法的一些基本原理是相同的。

*   在开始之前，明确你的学习目标。如果你想了解你的网站向顾客销售产品的情况，就要关注用户如何快速方便地找到他们想要的产品，如何轻松地完成订购过程，如何有效地将访问者转化为付费顾客，有多少潜在顾客流失(什么都没买就离开你的网站)，以及他们再次回来买东西的可能性有多大。
*   确定您的用户配置文件。然后用一个尽可能接近最终用户的群体来评估你的网站。如果你的网站向富有的美国退休人员推销健康度假村，不要在欧洲的青少年中进行测试。
*   安排项目和学习。确定一个方便的时间和地点。确保你的网站在那个时候正常运行。
*   准备一组问题或定义一组志愿者必须完成的任务。例如，你可以让他们从你的网站上订购两双特定品牌的绿色尼龙袜，看看这有多容易(或多难)。
*   用批判的眼光评估研究产生的数据。你如何改进每一步，让用户更容易操作？
*   实施变革。根据你收到的信息采取行动。
*   再次重复该过程。微调、调整、修改——直到你接近完美！

##### 在重新设计中需要改进什么

几乎一个网站的每个方面都可以用这样或那样的方式来改进。

重新设计背后的理念应该是:

*   尊重你网站的访问者
*   接近最大可达性
*   干净、吸引人、清晰可辨的外观
*   更好的可扩展性和未来维护的便利性

在计划重新设计之前，可能有必要认真考虑以下方面:

*   设计、布局、观感、品牌
*   可用性和导航
*   受众范围
*   未来可扩展性
*   易于推广
*   整合新内容、功能和技术
*   创收模式

##### 设计你的外观和感觉

在缺乏与目标受众面对面接触的情况下，你将自己的形象印在用户脑海中的唯一方式就是通过你网站的形象。不管你的网站的目的是什么，一个干净、整洁而有吸引力的布局一定会给访问者留下深刻印象。

重新设计网站的一个最常见的原因是改变旧网站的外观和感觉。这是如何实现的太个人化了，不能一概而论，完全取决于你的网站的性质。

一些一般原则适用于:

*   你的主页可以只是装饰性的，也可以是信息性的。
*   考虑最小化用户获取内容所需的点击次数。
*   在内容块周围使用清晰的空间，以获得干净整洁的外观。
*   优雅地使用图形来增强网站的用户体验。
*   不要为了技术而使用技术。但也要记住，技术正在迅速发展。越来越多的用户拥有宽带互联网接入，并渴望更具视觉吸引力、身临其境和互动的网络体验。
*   迷恋品牌。颜色的一致使用，在你网站的每一页上显示你的标志和标签，保持一个专业的形象有助于用户认同你的品牌。

##### 可用性和导航

随着网站内容的增加，它变得越来越难以传播。游客更容易在迷宫中迷路，变得越来越沮丧。网站重新设计可以为这样的网站创造奇迹。通过改进导航工具和增强用户体验，重新设计可以极大地放大网站的目的。这个网站上有许多关于设计网站导航的优秀资源。

需要记住的一些概念包括:

*   保持导航工具简单直观。决定按钮的选择和它们在页面上的位置对于达到这个目的是很重要的。
*   为了最小化长导航菜单占用的空间，可以考虑使用 Javascript 或 Java 来创建弹出或下拉导航菜单。
*   必须对层次结构进行排序，以最大限度地减少用户在找到有用内容之前需要点击的次数。
*   在大量的网站上，一个高效的搜索引擎可以让用户愉快地找到内容

##### 受众范围

网络是国际性的，一直都是。但是它的增长率在世界各地是不同的。每天都有新的受众上网，他们有自己独特的偏好、特点和兴趣。随着你越来越受欢迎，越来越多的人听说你的网站并访问它。你的企业和网站可能需要吸引这些不断变化的在线人群。重新设计可能是答案。

重新设计如何帮助你吸引新的观众？

*   您的新目标受众可能有兴趣和需求，您可以通过添加或修改网站内容来满足这些兴趣和需求。
*   你已经添加到网站的内容可能会吸引新的访问者，他们是你其他服务和产品的潜在客户。
*   语言是一个很容易克服的障碍。通过将您的内容翻译成其他语言，您可以被新客户看到。
*   网络冲浪者可能会遭遇阻碍，无法体验你的内容。简单的重新设计元素可能会使你的网站易于被他们访问。例如，通过在网站上智能地混合文本和背景的颜色，你也许能够接触到色盲的浏览者。通过使您的网站具有声音或避免使用固定大小的字体，您可以让有视觉障碍的人访问您的内容。
*   从技术角度来说，浏览器兼容性的问题对于最大化你的网站的影响力是很重要的。如果你的网站是使用带宽极其密集的最新尖端技术设计的，你将失去大多数使用老式电脑通过慢速拨号连接上网的网上冲浪者。
*   在重新设计过程中要记住的另一点是保持新网站与旧的现有内容向后兼容。

##### 未来可扩展性

当你第一次创建一个网站的时候，也许你没有想到它会像现在这样受欢迎。你一头扎进去做了。现在它产生了如此多的页面浏览量，你想让它变得更好。(这个并不罕见的场景促使我对自己的网站进行了第一次重新设计。)更何况你现在知道白手起家有多难了。如果你的网站变得更加出名的话，你会希望把事情做好，这样就很容易扩大你的网站。重新设计是您规划未来扩张和增长的最佳机会。

有两个因素需要记住:

1.  整个网站的组织
2.  设计、编码和编程

一个经过深思熟虑的网站内容层次安排应该可以让你很容易地成长和扩展。集体讨论各种可能性。考虑网站的哪些部分在接下来的几年里会变得更加重要。然后想办法将这种增长因素融入到你的网站设计和布局中。我用我的经历来举例。在我第一次改版时，我的电子杂志只有 250 名订户。我仍然认为这是一个潜在的增长领域。因此，我计划了一个系统来存档过去的问题，邀请新订户加入，并出售广告和赞助。随着时间的推移，随着我的电子杂志读者数量的增长，这个系统几乎不需要额外的努力就可以实现增长。下一个问题——旨在让负责维护网站的人随着时间的推移生活得更轻松——是规划一个系统，通过该系统可以快速、省力地进行网站范围的更改。

有几个选项:

*   一些 HTML 编辑器允许通过修改单个文件或向网站的所有页面插入代码来跨网页进行更改。然而，被改变的页面必须一次上传一个。
*   通过在网站设计中为标题或导航栏使用框架，只需改变框架的内容，就可以对整个网站进行更改。
*   然而，最好的方法是使用层叠样式表(CSS)和服务器端包含(SSI)。

CSS 是一组应用于网页元素的规则。CSS 有三种类型——嵌入的、内联的和链接的(或外部的)。通过在网站的每个页面上使用外部样式表，只需更改包含规则的 CSS 文件，就可以更改网站的外观。例如，如果控制文本或链接元素的 CSS 规则集发生变化，网站的所有文本内容都可以以不同的字体、大小甚至颜色显示，而链接可以被格式化为粗体、下划线或以不同的颜色显示——所有这一切都只需在一个 CSS 文件中进行一次更改。创造性地使用服务器端包含可以让网站布局更加多样化。SSI 允许文件(可以是整个。htm 页面)插入到另一网页的指定部分。通过对单个 SSI 进行更改，更改可以反映到整个站点。例如，如果一个 SSI 文件被用于网站的导航栏，通过改变这个单独的文件，网站的整个导航系统都可以被修改。

##### 易于推广

对于许多网站来说，搜索引擎是一个重要的流量来源。因此，在这些引擎中的高排名是非常理想的。但是搜索引擎经常改变他们的定位算法。为了在网站上保持突出的位置，网站也需要经常重新设计。在重新设计的时候，必须决定如何为不同的搜索引擎优化网页。添加元标签，插入大量关键词，重写内容以获得更高的关键词密度，修改设计和布局(例如将导航栏放在页面的右侧，以便它在文本浏览器中显示在内容之后)都是重新设计中使用的有效工具。

另一个与增加网站知名度相关的问题是网站最好部分的可访问性。通过仔细分析网站访问日志并跟踪访问者浏览网站的路径，就有可能绘制出最佳和最有用的页面。在重新设计的过程中，必须尽一切努力让这些页面在到达网站后不久就呈现在读者面前。这些网页也必须是第一批得到优化，以确保在搜索引擎和目录的突出位置。

##### 集成–内容、功能、技术

网站通过增加内容，包括新功能和实施技术来发展。很快就会出现混乱。重新设计提供了一个极好的机会来巩固和恢复秩序。

##### 内容再造

对于您网站上的现有内容，您可以:

*   重组。重新思考你的策略可以让你对网站的布局、设计和内容安排模式进行重大的结构性改变。
*   重新设计内容。存档文章可以成为电子杂志的特色。论坛线索可以成为主题评论的核心。艺术作品的照片可以被收集到一个虚拟画廊中。
*   从逻辑上重新安排内容。为了说明这一点，新闻网站可能会重新设计其档案，将与某一事件相关的所有故事分组，而不是简单地按日期排序。
*   分散在网站各处的参考内容。如果将分布在站点各处的关于特定主题或问题的单独内容组合在一起，可能会对访问者更有用。

##### 添加功能

随着增长而来的是不同的需求。当一个内容站点达到临界质量时，向它添加社区是有意义的。网上商家将受益于建立一个邮件列表，她/他可以宣布特别优惠和交易。创办电子杂志可以让科学研究网站告知会员最近的发展。建立讨论组可以让虚拟社区保持联系。

从另一个角度来说，您可能需要为特定目的添加功能。最近，为了吸引“大企业”广告商，在你的网站上展示隐私政策成为强制性的。随着您的业务或服务性质的发展，法律免责声明变得必不可少。随着时间的推移，可能需要添加或修改版权声明。

##### 采用新技术

网络上的技术变化非常快。一些发展令人兴奋，具有革命性。其他人的消失就像他们的到来一样突然。重新设计你的网站可以让你尝试新技术。

如果您的网站提供股票报价，移动无线互联网让您有机会随时随地与客户取得最新进展。如果你提供视频电影下载，DSL 用户数量的增长对你来说是个好消息——值得重新设计。

##### 创收

跟随收入流总是一个好主意！

虽然有许许多多的方法可以从网站上赚钱，但可悲的事实是大多数网站都没有。原因是多种多样的，但本质上并不是每种创收方法都适合某个网站。所以，当你为你的网站确定了一个赚钱策略，你最好充分利用它。这意味着你需要重新设计你的网站来充分利用这个优势。

如果你注意到横幅广告对你很有效，你可以考虑修改设计，允许横幅广告放置在最有可能吸引广告商的位置，并产生用户的最佳反应。然而，如果你的网站的内容是它的主要吸引力，你的重点应该是改变设计，以避免广告等不相关的元素掩盖内容。

在购物网站上，重新设计的过程会完全不同。这里的最终目标是从访问者那里获得更多的销售额。简化搜索和订购流程，并引导临时用户找到他们最感兴趣的产品是这里的首要任务。

##### 如何最好地重新设计你的网站？

选择是显而易见的——自己做或者外包给专业人士。首先，参考之前讨论过的问题，加上常识、灵活性和开放的心态，会给你的新网站带来奇迹。如果你决定外包这项工作，还有几件事要记住。

1.  Consider what you’d do differently. Make a list of the areas you’d like to redesign, and suggest the modifications you wish to see implemented.
2.  Communicate these ideas effectively to the designer/developer.
3.  Have a written contract signed by both sides to prevent litigation.
4.  协议应规定:
    *   工作描述(谁将做什么)
    *   工作将如何完成，以及所涉及的时间框架
    *   成本
    *   谁拥有创作的材料，尤其是知识产权。

##### 成功重新设计的技巧和诀窍

*   无论你如何决定进行你的重新设计，确保从你现有的网站中分离出来。开发服务器的使用以一种真实的方式模拟了真实的环境。另一个选择是在你自己的网站或受限的内部网中使用受密码保护的部分。
*   永远保留旧网站的备份。那么在新设计不幸遭遇灾难的情况下，你至少还有旧版本可以依靠！
*   在“上线”之前，从一小部分用户那里获得反馈。这给了你一个机会来修复任何错误，并对新网站做一些最后的调整。
*   为现有客户和成员创建一个常见问题部分，描述您的重新设计并解释新系统工作方式的任何变化。
*   查看你的竞争对手的网站，寻找思路。
*   考虑改变你的虚拟主机服务或者你的网站的操作平台，如果你的改变需要的话。
*   继续向用户征求反馈…下次重新设计时你会用到它！
