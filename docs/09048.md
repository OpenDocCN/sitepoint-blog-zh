# 10 个 jQuery 语法突出显示

> 原文:[https://www.sitepoint.com/10-jquery-syntax-highlighters/](https://www.sitepoint.com/10-jquery-syntax-highlighters/)

作为一名开发者，你可能想在你的网站或博客上分享你的代码(就像我们一样！).根据您选择的编程语言，有许多选项可供您选择。无论是 HTML 页面，还是在 PHP、Ruby、Python、ASP 上运行，总有一款适合你的语法荧光笔。

主要是我想显示 JavaScript，所以我用了名为 SyntaxHighlighter 的 WordPress 插件。这个插件可以让你在我们显示的代码上看到所有漂亮的颜色。你可能想看看[如何在你的博客](http://www.jquery4u.com/plugins/show-jquery-source-code-wordpress-blog/)上设置语法荧光笔来开始。此外，我可选地在右上角的每个代码框中添加了一些文本，如“双击编辑代码”，参见[如何在代码片段中添加自定义文本](http://www.jquery4u.com/snippets/adding-custom-messages-code-snippets-displayed-web-page/)。在不同的语法荧光笔之间切换总是一个挑战，因为它们都有不同的设置，而且**大多数都需要不同的代码标识符标签，所以请确保你选择明智的**，因为你可能会坚持使用它。

在本帖中，我们收集了 **10 个语法亮点来美化你的代码呈现**。它们被分成了不同的语言:jQuery、JavaScript、Mootools、PHP、Ruby & Python。尽情享受吧！

## 用 PHP 写的

### 8.GeSHi–通用语法高亮显示

GeSHi 最初的想法是为 phpBB 论坛系统创建一个通用语法高亮器，但现在已经推广到这个项目。GeSHi 旨在成为一个简单而强大的高亮类，其目标如下:

>支持多种流行语言
>轻松添加新语言用于高亮显示
>高度可定制的输出格式

 [![GeSHi - Generic Syntax Highlighter](../Images/3bc8749922724b822a06c9866c2771d0.png)](http://qbnz.com/highlighter/index.php) 
[源+演示](http://qbnz.com/highlighter/index.php)
