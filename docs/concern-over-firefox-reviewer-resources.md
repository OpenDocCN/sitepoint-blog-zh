# 对火狐浏览器资源的担忧

> 原文：<https://www.sitepoint.com/concern-over-firefox-reviewer-resources/>

最近的一篇博文反映了对火狐代码/补丁审查可用资源数量的担忧。

这篇文章，由一位参与浏览器发展的资深黑客所写，也许显示了火狐爆炸式增长所遇到的挫折和困难。

正如大多数成功的开源项目一样，大量的工作是由志愿者完成的。然而，在许多情况下，公共和私人公司也向高级开发人员承诺项目费用(通过允许开发人员在工作时间承诺贡献时间)。

当然，从企业的角度来看，这并不是出于纯粹的利他主义——公司经常看到对应用程序或技术的真正需求，并希望从其成功中受益。

当志愿者耗尽时间和精力时，后一个元素也经常可以挽救落后的开源项目。也许有一个额外的天使在为火狐做准备。

Mozilla Foundation 没有回应关于火狐审查过程的明确背景以及在不久的将来是否会有付费开发者参与该项目的请求。

即使一个人不是高级黑客，不会为一个大项目跳代码——我们中任何一个大量利用开源的人都应该想办法为我们认为有价值的项目做贡献。这可以通过多种方式实现——协助营销、文档、测试和质量保证、宣传以及代码层面的工作。
