## 采访记录

2010 年 8 月 13 日。我们采访了两家最受欢迎的 WordPress 主题公司的杰出人物。这是 SitePoint 播客#74:内森·赖斯和科里·米勒的 WordPress 主题。

**内森:**是的，基本上现在有两种类型的主题，即使是在免费和商业市场上，现在有我所说的传统主题，还有主题框架，Genesis 是一种基本上提供更多选项的主题框架，基本上当我们将自己与经典主题或传统主题进行比较时，Genesis 有布局选项，所以你可以选择左边的边栏或右边的边栏，或者两个边栏或没有边栏，等等。标记结构做的真的很好；在发布《创世纪》之前，我们提前几个月就想好了。然后我们也包括了 SEO 选项，很多人都不知道，但是你的主题中的标记结构确实会影响你的 SEO。很多人说，我可以使用常规主题，我会使用一体化 SEO 包，这很好，我们也集成了一体化 SEO，我们喜欢 Michael 和他做的插件，但很多人不知道标记结构对 SEO 也很重要。至于市场上的其他主题框架，我们会自动升级，所以你只需在后台点击一个按钮，就可以立即升级。我们有一个来自 Mark Jaquith 的安全审计，如果你不知道他是 WordPress 项目的主要开发者之一，我们还有 18 个子视图，它们的设计来自不同的设计师和其他人。

**Nathan:** 1.2 大多是 WordPress 3.0 的兼容发布；自定义导航将作为兼容性包含在内。我们还加快了一些速度，只是在后端增加了一些额外的选项，诸如此类。所以这是一个次要版本，但它有一个主要的发布周期号，但它主要是为了与 WordPress 3.0 兼容。

布莱德:很高兴听到这个消息。

内森:是的。这对我们来说是一件大事，我们在 StudioPress 一直致力于做的事情，我们可能会一直这样做，就是真正地与 WordPress 的功能紧密集成——发布缩略图、线程评论、3.0 导航，等等。我们真的希望就在那里，而不是重新发明轮子，做与 WordPress 计划不同的事情；我们真的希望在那里做一些对 WordPress 来说不陌生的主题，并使用 WordPress 所有内置的核心功能。所以我们真的想探索自定义帖子类型和分类法的想法，看看我们能做些什么来构建特定的主题；我们已经有了，对于那些不知道的人，如果你是房地产经纪人，我们有 [AgentPress](http://www.agentpress.com/ "AgentPress Real Estate Theme by StudioPress") ，我认为我们将重新检查带有自定义帖子类型的 AgentPress，因为这是有意义的，列表可以是自定义帖子类型。

Nathan: 所有 StudioPress 主题都将被转换到 Genesis。我们这样做的原因是因为我们相信创世纪。我的意思是，就像我说的，我们进行了安全审计，所以我们非常有信心，在你的主题中不会有任何明显的漏洞可以让你暴露于黑客攻击，让你的网站被黑客攻击。所以我们真的尝试了，每个主题都有相同的标记结构，所以你可以改变你的设计；如果你想的话，一周一次，但是你的标记结构永远不会改变，因为从一个主题到另一个主题都是一样的，因为是创世纪在运行这一切。你永远不需要重新输入设置，你永远不需要为一个特定的主题返回并改变布局和类似的东西，顺便说一下，布局设置适用于特定的帖子和页面，所以你可以进入并拥有你的网站，主页有你的内容和侧边栏，但在特定的页面上，如果你不想要侧边栏，你可以关闭它。所以所有这些东西都适用于每一个儿童主题，对我们来说，这是《创世纪》的一个主要好处，所以对我们来说，把每一个主题都转换到《创世纪》中当然很有意义。现在我认为我们可能已经完成了四分之三。
