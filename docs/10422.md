# 8 个最佳 jQuery 媒体插件

> 原文:[https://www.sitepoint.com/8-jquery-media-plugins/](https://www.sitepoint.com/8-jquery-media-plugins/)

jQuery Media plugins are very useful tools if you want your website to be more dynamic and entertaining. It provides video player for Youtube videos, putting some google maps within your website and more. This might be useful too if you’re off to make a website that is mainly focused on designs, but still want that interactivity. **What we have here is a list of jQuery Media plugins**. Enjoy.

相关帖子:

*   [**10 jQuery 音频插件**](http://www.jquery4u.com/media/10-jquery-audio/)

## 1.jquery tubeplayer

这以一种更好的方式实现了 Youtube 播放器 API。

## 2.菲力普花式照片集

以奇特的方式查看 Flickr Photostream！

[![](../Images/14ff56d04cc481c961bb70c7b01b0901.png)T2】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Media-Fancy-Photoset.jpg)

## 3.自动剪切

这将自动从 HTML、CSS 和 Javascript 的源代码中生成代码片段。

[![](../Images/14180020f623e95656a9eb495eff023a.png)T2】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Media-AutoSnippet.jpg)

来源:http://spacebug.com/projects/autosnippet/

## 4.jQuery 实时相关推文栏

从你的 twitter 时间线或任何人那里添加一个与你的帖子相关的实时推文栏，并通过谷歌地理编码坐标对其进行限制。

[![](../Images/70259ca988e0ab443ac44ca550cb8da4.png)T2】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Media-Realtime-Tweetbar.jpg)

## 5.jp player

播放和控制网页中的音频文件，这个插件支持更多使用 HTML5 的设备。

[![](../Images/dc33e2a6416cd8bee2804cabed82a11e.png)T2】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Media-jPlayer.jpg)

## 6.goMaps

这个插件将使用谷歌地图 v3 在你的网站中嵌入谷歌地图。

[![](../Images/c82d7e5710352447442c257496781e6c.png)T2】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Media-goMaps.jpg)

## 7.jQuery 嵌入链接

这取代了媒体中的链接。但是媒体应该得到 oembed.com 的支持。这包括 Youtube、Vmeo 和 Flickr。

## 8.FullScreenr

这是一个很棒的插件，你可以在你的网站上全屏显示背景，即使调整了大小也能保持它的长宽比。
