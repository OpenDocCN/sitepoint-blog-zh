# 使用 Smush It 在线优化您的网页图像

> 原文:[https://www . site point . com/optimize-your-web-images-online-with-smush-it/](https://www.sitepoint.com/optimize-your-web-images-online-with-smush-it/)

昨天，我写了如何使用 Photoshop 优化网页图片。也有一些免费的应用程序可以帮助你压缩图像文件的大小，其中一个特别好的是雅虎的 [Smush.it](http://www.smushit.com/) 。

正如上一篇文章中提到的，尽管宽带在世界上的一些地方几乎无处不在，但仍然有很多人在使用慢速连接。为了确保每个人都能快速加载你的网页，并希望在这个过程中让你的访问者停留更长的时间，你所有的图片都应该被优化。

Photoshop 在图像优化方面做得很好，Illustrator 和 Fireworks 也是如此，但 Smush .它确实可以压缩图像的真实光线，产生非常小的文件大小。它可以处理 JPG，GIF 和 PNG 图像的大小高达一兆字节。来自 Smush.it 网站；

> 它使用特定于图像格式的优化技术来删除图像文件中不必要的字节。这是一个“无损”工具，这意味着它优化了图像，而不改变它们的外观或视觉质量。

**小型案例研究**

在昨天的帖子中，我上传了一张花的照片。图像文件大小为 159KB。在 Smush.it 主页上，我输入了那张图片的 URL，结果如下所示。

![smush-result](../Images/9a6f97135d728f4a660656f649e85b55.png "smush-result")

Smush .它将同一张图片压缩到 112KB，节省了 47KB。生成的图像文件作为链接列出，可以下载。这项服务又快又免费。

**WordPress 插件**

Smush.it 也可以作为一个 [WordPress 插件](http://wordpress.org/extend/plugins/wp-smushit/)在这里获得

在你的博客上安装 Smush.it WordPress 插件后，你添加到文章和页面的任何新图片都会在后台自动优化。你什么都不需要做！对于博客上的现有图像，您可以从媒体库中选择一幅图像，然后按 Smush.it Now 按钮。

还有一个 Firefox 插件，名为 [Yslow](https://addons.mozilla.org/en-US/firefox/addon/5369) 可用，其中包括 Smush.it，我安装了该插件，在编写时，它对我来说无法正常工作。不过，这可能和 Firefox 的最新更新有关。
