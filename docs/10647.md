# 这都是关于你的:设计一个关于页面的想法

> 原文:[https://www . site point . com/its-all-about-you-ideas-for-design-a-about-page/](https://www.sitepoint.com/its-all-about-you-ideas-for-designing-an-about-page/)

在设计网站时,“关于”页面往往被视为网站主页的近亲，然而“关于”页面往往是网站上访问量最大的页面之一，尤其是在你提供服务的时候。人们想了解和他们做生意的人或公司。如果你没有博客，你的“关于”页面是潜在客户看到(尽可能接近)真实的你的地方——这个行业敏感的一端。在本帖中，我们将看看那些让他们的“关于”页面引人注目、有趣且独特的网站。

“关于”页面应该回答几个问题

1.  你是谁？
2.  你是做什么的？
3.  为什么我会选择和你一起工作？
4.  怎么联系你？

**1。提供详细的组织信息**

[Jacob Nielson](http://www.useit.com/alertbox/about-us-pages.html) 建议在组织的“关于”页面上显示以下信息:

1.  主页上的标语:用几个词或一个简短的句子概括该组织的工作。
2.  **摘要:**主*关于我们*页面顶部的 1-2 个段落，提供了关于组织目标和主要成就的更多细节。
3.  概况介绍:摘要之后的一个部分，详细阐述其要点和关于该组织的其他基本事实。
4.  **详细信息:**为希望了解公司更多信息的人提供更深入的辅助页面。

显然，如果你是一家小公司或个体交易者，你可能没有足够的信息来提供，但是你仍然可以在你自己的“关于”页面上应用标语和简短的总结。如果你有很多信息要在你的“关于”页面上传达，把它们分成有组织的块会使它们更容易理解。

**2。使用照片**

这张照片让我们看到你是多么的华丽、专业、古怪或有趣。最重要的是，它把一个名字的脸，可以帮助你与你的目标受众联系起来。许多人一想到在自己的网站上放上自己的照片就感到害怕，但这增加了一种个人风格，可以打破障碍。
