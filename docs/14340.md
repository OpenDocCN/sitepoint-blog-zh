# 你应该在网站上公布价格吗？

> 原文：<https://www.sitepoint.com/should-you-post-prices-on-your-website/>

最近 Sitepoint 论坛提出了一个很好的问题:你应该在你的网站上张贴价格吗？

答案取决于你的策略:

1.如果你主要是一家以产品为导向的公司，而不是一家服务公司，尽一切办法列出你的价格。例如，如果你提供模板或快速标志设计，提供定价。

2.如果你提供一些低成本的产品，然后列出这些价格。例如，在我的网站上，我提供书籍和文案。两者都是我工作的一小部分，我列出价格是为了让人们测试我的知识/服务。

3.有趣的是:

3a。如果你认为自己是一个值得信赖的顾问，并根据价值定价，那么我会避免在网站上公布你的价格。这是因为你想根据你对客户的价值来评估客户的情况和价格，以及这个具体项目的挑战性。此外，通过公布你的时薪，你把自己定位成一个供应商，而不是一个关系驱动的顾问。

3b。说到这里，你可能会看到一位非常成功的顾问，他每小时的工资是 550 美元:[http://www.treyryder.com](http://www.treyryder.com)

他向律师推销，他的哲学是专业人士应该公开他们的收费。

但我尝试了他的方法，得到了 0 个客户，所以我回到 3a。
