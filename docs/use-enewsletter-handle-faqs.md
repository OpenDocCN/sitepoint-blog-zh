# 别再问我了！使用您的电子快讯处理常见问题

> 原文：<https://www.sitepoint.com/use-enewsletter-handle-faqs/>

如果客户打电话来问“我该如何…？”或者“为什么我不能…？”这是你第一千二百万次拔掉自己的头发，是时候把你的时事通讯变成一个技术支持工具了。怎么会？通过使用这 3 种文章类型和 3 个简单的技巧，你可以减少回答所有这些常见问题的次数。

##### 3 篇可以节省你时间的文章！

***文章类型 1-常见问题栏***

使用你的时事通讯解决常见问题。这是显而易见的，但是如果执行得好的话，FAQ 部分真的很有效。不要以推销的方式提问或回答问题，而是让你的回答真正有帮助。

如果你接受读者的提问，确保你回答的是读者真正提出的问题，而不是你想回答的问题(这通常是有区别的)。此外，尽量提供指向你的网站(或以前的常见问题)的链接，读者可以在那里获得关于该主题的更多信息。

***文章类型 2-团队成员简介***

在每期简讯中，介绍你公司的某个人(或你公司的合作伙伴)。让客户看看这个团队成员做什么，以及他或她是如何做的。也包括一些个人信息，这样读者会觉得他们正在了解你的员工。通过展示你公司的人性化的一面，客户会觉得和你在一起更舒服，而且他们也会开始知道在遇到具体问题时应该和谁联系。

档案还可以包括该团队成员对常见问题的回答。即使这个问题已经在你的时事通讯中回答过了，这个团队成员的回答也可能是读者“点击”的那个——所以不要害怕在你的时事通讯中不止一次地回答同一个问题。

***文章类型 3-客户简介***

让你的客户互相了解。通过采访当前的客户，询问他们在与你共事期间可能遇到的问题，你将能够以一种可能被阅读的方式为其他读者回答这些问题。此外，正如你的团队成员可能会提供某些读者感兴趣的解释一样，你的一些读者可能会发现，当一个刚刚开始理解某个概念的人解释这个概念时，理解这个概念会更容易。

读者喜欢阅读关于人的文章；他们很少迷恋技术。因此，让技术个性化，他们更有可能学习。

##### 撰写文章的 3 大技巧

现在，我们已经介绍了有助于您减少重复回答客户问题的 3 种文章类型，让我们来看看如何展示这些文章以提高理解和学习。

***1。考虑将客户分成不同的类别***

根据你有多少客户，以及你为他们提供了多少服务，你可能想要提供 2 个或更多的列表，因为向所有客户发送相同的信息可能会有危险。

举例来说，如果你的客户中有一半雇佣你专门设计网站，另一半雇佣你做后台工作，把他们分成两个列表会让你避免收到询问“为什么我的网站不这样做？”它还可以在某种程度上帮助你区分读者的经验水平——这样，更有经验的读者就不会被基本问题弄得沮丧，而经验较少的读者也不会被更高级的问题弄得不知所措。

***2。写对了***

标点符号至关重要。当你试图解释你的客户可能难以理解的东西时，拼写、句子结构和语法都是关键。你的写作应该尽可能的简单易懂。不要使用缩写或首字母缩略词，并确保定义每个可能不熟悉的单词。

如果你的时事通讯是 HTML 格式的，使用格式使阅读更加容易(例如，不熟悉的单词第一次出现时加粗)。通过使你的文章尽可能清晰明了，优化布局以帮助理解，你将为读者提供他们真正理解你所解释的内容所需的所有工具。

***3。求反馈***

起初，你可能很难向读者解释你非常了解的信息。行话和首字母缩略词可能只是因为你太习惯使用它们了。这就是为什么找一个和你的读者有相似经历的人会很有帮助。

这个“测试读者”可能是一个工作人员或朋友或同事(也许你可以与另一个领域的同事交换简讯)，他们会指出文章中令人困惑的地方，或者提供额外的要点让你涵盖。在最初的几期之后，你应该会发现自己的写作风格更适合你的读者。

所以，不要揪你的头发，开始使用你的时事通讯作为技术支持工具。你的客户——和你的头皮——会感谢你的！
