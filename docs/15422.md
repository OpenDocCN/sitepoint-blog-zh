# 通过提高网站的链接流行度来增加点击率

> 原文：<https://www.sitepoint.com/link-popularity-site/>

就搜索引擎排名而言，链接流行度，即链接到你网站的站点数量，是一个越来越重要的因素。其他事情保持不变，你的网站链接越多，它的排名就越高。

重要的不仅是链接到你的网站的数量，还有链接到你的网站的类型。一个与你的网站相关的链接比一个无关网站的链接更有价值。

在这篇文章中，我探索了不同的方法，通过这些方法你可以提高你的站点的链接流行度。我从你不应该麻烦使用的方法开始，然后继续到适度有效的方法，然后以你可以用来提高你的站点的链接流行度的最有效的方法结束。

##### 1)将您的站点提交到免费的(FFA)页面

一个常见的误解是，虽然 FFA 页面可能不会给你的网站带来流量，但它将有助于提高你网站的链接受欢迎程度，从而通过搜索引擎间接带来流量。

没有什么比这更偏离事实了。大多数 FFA 页面一次只包含一定数量的链接。这意味着当你提交你的网站到一个 FFA 页面时，你的网站会被放在页面的顶部。然而，随着越来越多的人将他们的站点提交到 FFA 页面，你的站点将被向下推，最后，当它到达页面底部时，它将被移除。

现在，因为你可以打赌，很多其他人也提交他们的网站到 FFA 页面，你的网站将在这些页面上停留很短的时间。因此，为了确保搜索引擎在抓取 FFA 页面时看到您的网站，您需要确保定期向这些 FFA 页面提交您的网站——至少每周一次。

即使你使用了自动提交程序来做这件事，你能想象一个更糟糕的方式来花费你的时间和/或金钱吗？此外，大多数搜索引擎将只包含其他网站链接的页面视为 FFA 页面，可能会完全忽略它们。虽然我还没有看到任何证据表明提交到 FFA 页面会真正惩罚你的网站，但这种情况很有可能在未来发生。

因此，谈到 FFA 页面，我的建议很简单:想都别想。

##### 2)加入互惠链接服务

有人推荐，为了增加你网站的链接人气，可以加入一些互惠链接服务。这些服务背后的基本思想是，向您的站点添加一些页面，这些页面包含指向该服务成员的其他站点的链接。作为交换，这些成员还将在他们的网站上添加包含您网站链接的页面。从理论上讲，这项服务的会员越多，你的链接就越受欢迎。

然而，我对使用这种方法来提高你网站的链接流行度有很多保留意见:

大多数这些服务要求你在主页上添加一个可见的图形或文本链接到包含这些链接的页面。如果他们需要一个图形链接，它会完全破坏你的网站的整体外观。即使他们需要一个文本链接，如果你的网站的访问者点击这样的链接，发现你的一个竞争对手(他也是这项服务的成员)就在页面的顶部，你会有什么感觉？此外，虽然大多数这些服务不允许面向成人的网站，但我见过一些成人网站能够成为会员的情况。如果一个成人网站出现在这些网页的顶端，你会有什么感觉？

大多数此类服务都提供相同的页面，其中包含每个成员的链接，即您需要上传到您网站的页面与该服务所有其他成员需要上传到其服务器的页面完全相同。甚至页面的文件名对于所有成员来说都是相同的。大多数搜索引擎现在能够检测不同域中的重复页面，可能会忽略这些页面，甚至可能会因为
发送垃圾邮件而惩罚所有这些域。

这些服务不是仅仅将相关的站点相互链接，而是将所有的成员相互链接。这意味着许多不相关的网站会链接到你的网站。正如我之前提到的，不相关网站的链接不如相关网站的链接有价值。此外，您还需要链接到无关的网站。随着搜索引擎变得以主题为导向(例如，他们考虑网站的整体主题，而不仅仅是单个页面中关键词的位置)，链接到与你的网站完全无关的网站会降低你在搜索引擎中的排名。

我找到的唯一能解决这些问题的互惠链接服务是 SearchEngineWorld 的好友链接程序。不过，这个网站的站长布雷特·塔布克(Brett Tabke)现在已经停止了这个程序。甚至这项服务也有一些问题(它将所有成员的网站相互链接，而不仅仅是将相关的网站相互链接)。

因此，我不建议你加入任何互惠链接计划。

##### 3)与其他站长交换链接

另一种提高网站链接受欢迎程度的方法是与其他网站管理员交换链接，这些网站与你的网站相关，但不存在直接竞争。你可以这样做:

首先，打开一个像 Microsoft Access 这样的数据库程序，创建一个新表，包含诸如名字、姓氏、电子邮件地址、URL 等字段。然后，列出属于你的竞争对手的网站。然后，转到 AltaVista，在搜索框中键入以下
:

链接:somesite.com-网址:somesite . com

其中 somesite.com 是你的一个竞争对手的域名。这会给你一个列表，列出所有链接到该竞争对手的网站。然后，找出在什么样的背景下一个特定的网站链接到你的竞争对手。如果这个网站是你的竞争对手的联盟，那么你从这个网站获得链接的机会是有限的，除非你提供一个更好的联盟计划。但是，如果你发现这个网站有一个链接页面，包含了其他网站的链接，其中一个链接指向你的竞争对手，那么这是一个交换链接的绝佳机会。找出网站管理员的姓名和电子邮件地址，并将它们添加到您的数据库中。这样，浏览所有链接到你的竞争对手的网站，找到那些你认为可能想和你交换链接的网站，并建立你的数据库。

完成后，在您的站点中创建一个链接页面，并将这些站点的 URL 添加到链接页面。然后，给这些网站管理员发一封电子邮件，介绍你自己和你的网站。祝贺他们建立了一个优秀的网站，告诉他们你已经从你的网站添加了一个链接到他们的网站，然后问他们是否愿意添加一个链接到你的网站。在你的电子邮件中，强调这样交换链接对双方都有利，因为这将有助于你们双方增加网站的流量。
等一个月左右看反应。有些站长会同意链接到你。其他人根本不会回应。一个月后，删除那些对交换链接不感兴趣的网站的链接，并使用上述方法，尝试找到更多的网站来交换链接。

当你发送电子邮件给网站管理员，确保你个性化的每封电子邮件。不要每封邮件都以“你好，网站管理员”开头，要以“你好，迈克”开头。如果您愿意，您可以使用电子邮件合并程序来自动个性化每封电子邮件。你可以通过去[www.download.com](http://www.download.com)搜索“电子邮件合并”(不带引号)来查看一些电子邮件合并程序。

这种提高网站链接受欢迎程度的方法的主要问题是，它需要很多时间。你可能会发现，你得到的链接数量与你花在上面的时间不相称。因此，我的建议是尝试一次，看看效果如何。如果你设法得到很多链接，然后重复这个过程。如果没有，就不要再为此烦恼了。然而，即使你没有积极地寻找其他网站来交换链接，也要时刻注意寻找链接的机会。如果在网上冲浪时，你遇到一个与你的网站极其相关但不是竞争对手的网站，写信给网站管理员并提出交换。

你可以做的另一件事是在你的链接页面中提到你愿意与其他网站交换链接。这使得其他网站管理员来到你的网站，提出一个链接交换。

##### 4)启动奖励计划

一个适度有效的方法来提高你的网站的链接流行度是开始一个奖励计划。您可以让与您的网站相关的网站从您的网站申请奖项。赢得奖项的网站有机会展示您的奖项的标志。该徽标链接到您的网站，最好是链接到包含该奖项更多信息的页面。

如果你出版一份时事通讯，考虑在你的时事通讯中宣布获胜者。您也可以在您的简讯中对获奖者的网站进行评论。这给你的时事通讯增加了有用的内容，也给了更多的站长申请你的奖励的动力，因为你可能会在你的时事通讯中查看他们的站点。这也让他们有动力订阅你的时事通讯，看看他们是否会获奖。

确保你只给那些应该获奖的网站颁奖。如果你把你的奖项给了不值得的网站，你的奖项将没有什么可信度，这反过来会损害你公司的可信度。此外，确保你为该奖项设计的标志看起来专业。如果没有，没有多少网站管理员会想在他们的网站上显示它。

##### 5)出具证明

这听起来可能有点不寻常，但给你觉得有用的产品或服务做推荐是另一种提高网站链接受欢迎程度的有效方法。如果你真的喜欢某个产品，只要给公司写信，告诉他们你为什么这么喜欢这个产品，以及它对你有什么帮助。很有可能，公司会给你回信，感谢你的评论，并请求你允许在他们的网站上显示你的评论。告诉公司，如果他们发表你的评论，你没有问题，但是要求他们在推荐的同时添加一个链接到你的网站。公司很有可能会同意，因为公布你网站的网址会让证明更可信。

当然，请不要只是为了提高你的链接受欢迎程度而给你能找到的每一家公司做推荐:-)

##### 6)张贴到留言板和讨论列表

另一个增加网站链接受欢迎程度的有效方法是在网上留言板上发帖。在你发布的每条信息的结尾，你可以通过提及你的名字和你的网址来结束。如果留言板允许的话，你甚至可以在帖子的末尾加入一个简短的关于你的网站的宣传广告。但是，请确保发布到该留言板的单个消息是在静态 HTML 页面中存档的(即，单个消息的 URL 不应包含“？”).否则，搜索引擎可能不会抓取这些页面，因此，将无法找到您的链接。

基于电子邮件的讨论列表以静态 HTML 页面的形式存档在网上，也可以用类似的方式来提高网站的链接受欢迎程度。在这种情况下，您在电子邮件程序中使用的签名文件应该包含您网站的 URL。

##### 7)建立新的网站

还有一个比较有效的方法来提高你网站的链接流行度，那就是建立新的网站，让这些新网站链接到你的主网站。如果你是杰夫·贝索斯(亚马逊的首席执行官)，你有能力购买新的域名，并让这些新的域名链接到你的主网站。如果你不是杰夫·贝索斯，你可以简单地用一些免费的虚拟主机建立新的网站。(当然，您的主网站应该始终位于一个独立的域中)。只需在每个新网站上添加几页独特的内容，并让新网站中的每一页都链接到主网站。

如果使用这种技术，请确保不要在多个网站上重复内容。如果你这样做，搜索引擎可能会惩罚你的网站发送垃圾邮件。您需要确保为这些新网站创建的所有页面都包含独特的内容。

这种方法的缺点是，一些搜索引擎(如 AltaVista)拒绝抓取免费网络主机中的站点。尽管如此，仍有许多搜索引擎会抓取你的站点，因此，你的主站点的链接受欢迎程度会在这些搜索引擎中得到提高。

##### 8)开始链接竞赛

一个提高网站链接受欢迎程度的好方法是，如果其他站长链接到你，就给他们奖励。你发放的奖品最好是其他站长觉得有价值的东西，让他们愿意链接到你的网站上，但是不要花你太多钱。例如，如果你出版了一份时事通讯，并且有未售出的广告库存，你可以在你的时事通讯中给获胜者一些免费广告。如果你卖软件(或者电子书)，你可以给获胜者一份免费的软件或者电子书。

如果你连续不断地举办比赛，并且经常宣布新的获胜者，那么链接比赛的效果最好。如果你举办了几个月的比赛，然后停止了，所有链接到你的网站管理员都会删除他们的链接。然而，如果你持续地运行它，并且每个月左右宣布新的获胜者，网站管理员将会有动力保持他们到你的站点的链接。

此外，确保您要求所有参与者在他们的主页或链接到他们主页的网站内部页面上有一个到您网站的链接。还要确保包含该链接的页面距离其主页的深度不超过两层(即，从主页到包含该链接的页面不应超过两次点击)。如果他们不这样做，搜索引擎蜘蛛可能不会索引包含你的站点链接的页面，因此可能找不到你的链接。

##### 10)开始你自己的加盟计划

这是另一个很好的方法，你可以通过它来提高网站的链接受欢迎程度。当你有了自己的联盟计划，你就给了其他站长链接你的动力。在这种情况下，由于这些网站大多与你所在的行业相关，这些链接比不相关网站的链接更有价值。

现在，当你开始你的联盟计划时，你需要决定你是想自己运行这个计划，还是想把它外包给第三方。虽然外包代销商计划有很多好处，但这样做不会帮助你提高网站的链接受欢迎程度，因为代销商会链接到第三方的网站。

因此，如果启动代销商计划的目标之一是提高网站的链接受欢迎程度，你需要自己购买一些代销商计划脚本来运行代销商计划。
