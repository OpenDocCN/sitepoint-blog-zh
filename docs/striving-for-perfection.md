# 追求完美让我想变得平庸

> 原文：<https://www.sitepoint.com/striving-for-perfection/>

我承认，我有 A 型人格。我的一些根深蒂固的特点是不耐烦、好胜、时间紧迫和雄心勃勃。哦，是的，我几乎不能放松。

除了不可能一直完美这一事实之外，追求完美还会带来一些非常严峻的挑战。以下是我面临的一些最大的挑战，以及我如何努力控制自己超常的个性。

**压力**

不能放松会使限制压力变得困难。当然，这有很多健康方面的影响，谢天谢地，到目前为止我已经避免了。但是我知道每天高度的压力可能是不健康的。

为了战胜这一点，我强迫自己休息一下，以避免筋疲力尽，分清轻重缓急，让 T2 和我的客户划清界限。我还注重在我[管理工作环境的方式上保持一致。](https://www.sitepoint.com/working-from-home-challenges/)

**让我的家人抓狂**

好吧，也许不是疯子，但我知道我的完美主义，尤其是当它在周六早上以冗长的待办事项清单出现时，肯定会影响我的家庭。我做的事情之一是让其他家庭成员对周末计划有发言权(除了我的待办事项清单)。我也试着安排空闲的家庭时间，这样我们都可以从在一起的时间中受益。

**克制自己**

有时候，我想把所有事情都做好，并且把所有事情都做得完美，这让我很难抓住新的机会。有时，我发现自己在想，如果我不能尽我 100%的能力完美地完成某件事，那我就不想做。虽然这通常不是一件坏事，但它有时会剥夺我新的体验。

我试图从三个方面来衡量每一个机会:1)我的参与如何对他人有价值，2)我将如何受益，3)如果我不能完美地完成，最坏的情况是什么。通常，这给了我足够的现实检查，让我意识到什么时候有一个有价值的机会出现并抓住它。

尽管 A 型血有一些负面的含义，但它对我的职业生涯非常有益。当我对公司的工作不满意时，我已经能够把事情掌握在自己的手中，并放手去做。我有动力，当我下定决心要完成某件事时，我很少放弃。我能够为我的客户提供优质的服务，因为我是可靠的，注重细节的，并且能够征服最具挑战性的项目。

在一天结束的时候，我们就是我们自己，我们都有积极和消极的特质，不管我们有什么样的性格。我们能做的最好的事情是拥抱我们积极的个性特征，并继续努力克服那些我们发现具有挑战性的特征。关键是要认识到我们都有进步的空间，不完美也没关系。用伟大的劳里·伯克纳的话说:“我不完美，不，我不完美。我并不完美，但我拥有我所拥有的。”

你有没有觉得很有挑战性的性格特征？你打完美之战吗？
