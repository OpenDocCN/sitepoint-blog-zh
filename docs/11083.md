# Palm 会挥手告别吗？

> 原文:[https://www.sitepoint.com/will-palm-wave-goodbye/](https://www.sitepoint.com/will-palm-wave-goodbye/)

据报道，智能手机和 PDA 制造商 Palm 公司正在出售自己，并可能在几天内寻求其他公司的投标。该公司目前在美国智能手机市场排名第六，但已连续 11 个季度亏损。

本来应该好得多的。Palm 在 20 世纪 90 年代是 PDA 领先的制造商，但却迟迟没有意识到手机市场的潜力。然而，该公司看起来肯定会通过 Palm Pre 和 Pixi 设备重振雄风。这些手机和基于 Linux 的 WebOS 操作系统一样受到了好评。web 开发人员对 WebOS 特别感兴趣，因为它允许使用 HTML、CSS 和 JavaScript 开发本地移动应用程序。

与 iPhone 的比较是不可避免的，但与苹果不同，Palm 提供:

*   多任务操作系统(即将出现在 iPhone 上，但 Pre 从第一天起就有了)
*   Flash 播放器(即将推出)
*   下拉式 qwerty 键盘和触摸屏
*   面向 Windows、Mac 和 Linux 开发者的免费 SDK 和虚拟手机
*   一个限制较少的应用程序目录，允许模拟器运行——例如允许用户运行数以千计的传统 Palm OS 应用程序
*   没有审批流程的“自制”应用程序市场。

这是一款很棒的手机，但销量令人失望。Pre 于 2009 年年中推出 iPhone 已经面市两年了，并且已经在市场上站稳了脚跟。早期的硬件故障和全球缓慢的发布时间表使这个问题变得更加复杂。

那么谁会买 Palm 呢？

微软和苹果不太可能感兴趣。据报道，HTC 和联想已经出价。有人会来拯救 Palm，但是让我们希望他们继续 WebOS 的开发工作。
