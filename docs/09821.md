# 普通自由职业者的“反营销”营销

> 原文：<https://www.sitepoint.com/anti-marketing-marketing-for-the-ordinary-freelancer/>

如今，每个人都是营销人员。有些人真的喜欢推销自己。我受过市场营销的训练，但我不属于那一类。我只是一个普通的自由职业者，更倾向于做好工作而不是推广它。

此外，作为一名代笔作家，我经常做一些不能公开邀功的工作。即使我喜欢把自己“放在那里”(不管那个在哪里)，我展示我的大部分作品的选择非常有限。

多年来，我学会了发展一些不太明显、微妙但有效的自我推销方法。因为你和我一样，可能是一个不喜欢“推销”自己的普通自由职业者，所以我想看看你对这些的看法。

## 1.讲述故事

这项技术是我最喜欢的技术之一。基本上，它包括通过社交媒体(对我来说是 Twitter)讲述一个项目的发展故事。

当然，出于保密的原因，我不能包括那些项目的某些事实。但这些遗漏留下的空白为创造性表达腾出了空间——毕竟，这是我独特贡献的一部分(可能对你也是如此)。

最近，我完成了一个大型的代笔项目。我今天试图把相关的推文拉出来，但是(你可能已经注意到了)，Twitter 有一些服务器问题。简而言之，我发了关于以下的推文:

*   一天写几千字
*   研究拉丁美洲史前史
*   埋头完成一本书的草稿(之后需要一杯烈酒)

这可能看起来很神秘，但这些推文做了很多:

*   向我的短拷贝客户指出我提供长拷贝服务
*   表达我对工作的热情
*   暗示我的多才多艺和适应性——我能做的不仅仅是简单地把几个句子组合在一起

## 2.问这个问题

在使用我技能的行业，我有很多朋友和人脉。每当我看到他们，我都会问他们这个问题:

“你在忙什么？”

我问这个问题是因为我真的对我的朋友们感兴趣的东西感兴趣，他们都对自己的工作充满热情。但这个问题为我创造了一个机会，让我了解他们正在做什么的细节，什么让他们兴奋，以及他们的领域正在发生什么。

这个问题的另一个作用是促使礼貌的回答，“那你呢？”这提供了另一个机会——一个我可以谈论我正在做的项目的机会。

由于我的朋友或联系人刚刚向我讲述了他们工作生活中的亮点，我可以调整我的回答，包括我认为他们会感兴趣的相关项目，和/或需要技能来满足他们自身需求的项目。

## 3.大声说出来

就像我说的，我做了很多不能公开宣称是我自己的工作。但是我知道项目和能力证据的价值——我可以提供的证据。所以我找到了其他方法来展示证据。

我在一些与我的目标读者相关的不同网站上写博客。我喜欢做个人项目，我会积极地吸引别人的注意力。我也喜欢偶尔做一些超出我通常关注范围的工作(比如写一篇杂志文章或举办一个研讨会),并链接或谈论这些工作。

我喜欢与相关领域的同行谈论行业发展，我不怕对我不知道或不了解的事情提出问题，或阐述我对我研究和思考的相关问题的看法。

## 为什么“反营销”有效

这些技巧非常有效地推广了我的服务。首先，它们简单又有趣，所以它们不会让我觉得我在努力“推销”我的“生意”。

其次，它们很微妙——在某些情况下几乎是潜意识的。因此，我的联系人，大多数都不想被推销给他们(鉴于我和他们的友谊，更明显的营销方式会让他们感到不舒服)，不会感到紧张，也不会觉得他们应该提高警惕。

最后，它们是自然的方法，反映了我的个性和工作哲学，以及市场营销类型对我的“目标群体”的需求和期望不要冒显得做作或勉强的风险，因为它们不是。

还有一个很大的好处:由于这些促销的间接性、对话性和非限制性，它们打开了工作的可能性(和新的联系),我相信如果我被网站副本、小册子或固定的收费表所束缚，会更难找到这些工作。

你如何“推销”你的服务？你用“反营销”吗？我很想在评论中听到你的想法。
