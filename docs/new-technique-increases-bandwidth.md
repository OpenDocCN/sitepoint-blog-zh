# 新技术将使互联网带宽翻倍

> 原文：<https://www.sitepoint.com/new-technique-increases-bandwidth/>

今天公布的一项新发现为那些使用拨号调制解调器的人带来了双倍速度互联网的令人兴奋的前景。

一个名为**电缆重组行动计划**的欧洲组织呼吁所有电信公司和互联网服务提供商尽早采纳他们的计划。该团体的一份新闻稿称:

> 我们的解决方案是革命性的，但比现有的网络基础设施更简单、更便宜、更环保。全球推广从今天开始。

互联网数据电缆通常根据地形限制铺设，这可能会导致布线混乱。该组织建议将互联网节点之间的电缆拉直:

1.  更直的电缆意味着电信号在更短的时间内传输更短的距离。
2.  大多数信号包含二进制数据。虽然 0 很少出现问题，但 1 的尖锐边缘可能会被电缆扭结缠住，导致不必要的数据包丢失。

该组织估计，对于大多数用户来说，网速将会翻倍，但对于那些电缆特别弯曲的用户来说，网速可能会更快。他们建议每个人立即利用该系统，并在办公桌下拉直网络或调制解调器电线。

然而，并不是每个人都对这个计划感到满意。一家领先的以太网电缆制造商的发言人声称:

> 这是一个完全荒谬的想法。更短的电缆对我们来说意味着更少的钱。

你把电缆拉直了吗？你的下载速度有显著提高吗？
