# 在线关注 JavaOne 大会 2006

> 原文:[https://www.sitepoint.com/follow-javaone-2006-online/](https://www.sitepoint.com/follow-javaone-2006-online/)

如果你像我一样喜欢在 Java 中工作，那么很可能你现在最想去的地方就是 JavaOne 2006 大会。今年 JavaOne 大会上特别热门的 web 开发话题包括新发布的 [JavaEE 5](http://java.sun.com/javaee/) 平台，用于更轻松地构建企业 Java 应用程序，以及即将到来的 [NetBeans 5.5](http://www.netbeans.org/) 的相关更新。

但是，不管是由于时间、金钱还是地理位置的原因，并不是每个人都能在旧金山享受一周的爪哇之恋。

幸运的是，无论你身在何处，你都可以关注大局。所有的全体会议(即主题演讲人和演示)都可以通过[直播和点播网络广播](http://java.sun.com/javaone/sf/sessions/general/)观看，几乎就像亲临现场一样(此外，您还可以跳过无聊的部分！).

至于更具针对性的技术会议，大多数都发布了包含幻灯片的可下载 pdf，可从[会议目录](http://www.cplan.com/javaone2006/contentcatalog)中免费下载。你也可以等到尘埃落定，这些幻灯片将免费提供，并带有来自 [JavaOne Online](http://developers.sun.com/learning/javaoneonline/) 的同步音频。

如果你喜欢播客，PodTech.net 将主持 JavaOne 的播客系列,包括所有大会的完整音频。不幸的是，JavaOne 程序没有专门的 feeed，所以你可能会发现自己从同时进行的其他技术会议上下载了一些程序。
