# 用于网页定制的 10 个 jQuery 插件

> 原文:[https://www . site point . com/10-jquery-plugins-网页-定制/](https://www.sitepoint.com/10-jquery-plugins-webpage-customization/)

有时候，好东西就在等待被发现，而我们只是懒得去寻找。在这篇文章中，还有另外 10 个 jQuery 插件，你可能会发现它们在创建或定制网页时非常有用。享受吧。

相关帖子:

*   [**10 个优秀的网页开发教程**](http://www.jquery4u.com/plugins/10-excellent-web-development-tutorials/)

## 1.自身抗体

这个插件自动为容器元素(div，body 等等)中的某些元素(例如 H2)创建定位导航。它会自动生成锚点，甚至可以对它们进行编号。当然，整个事情完全可以通过 CSS 进行样式化。
![AutoAnchors.jpg](../Images/b459165ce5d9dac0fc3ce8a2c953b35a.png)
[来源&演示](http://fredibach.ch/jquery-plugins/autoanchors.php)

## 2.背景插件

这个插件允许你用图像填充页面的背景——同时保持宽高比并优雅地调整窗口大小。
![Backgrounder-Plugin.jpg](../Images/2b0f263898c907a861bb9e814149a2af.png)
[来源](http://github.com/bigfolio/jQuery-Backgrounder)
[演示](http://content.casadirocco.nl/projects/jquery-backgrounder/example.html)

## 3.背面清除器

拉伸图像以覆盖页面或 DIV 的背景。
![backstretcher.jpg](../Images/64382f2f78f26062a48a7dc02f9657cf.png)
[来源&演示](http://aquaron.com/jquery/backstretcher)

## 4.蓝莓，一个用于响应式网页设计的 jQuery 图像滑块插件

是一个实验性的开源 jQuery image slider 插件，它是专门为流畅/响应的网页布局而编写的。
![Blueberry-a-jQuery-image-slider-plugin-for-responsive-web-design.jpg](../Images/525e06c54bbddefbd26b483d1f8f2e57.png)
[来源&演示](http://marktyrrell.com/labs/blueberry/)

## 5.BogoTabs

除了传统的标签，它还提供了创建“隐藏标签”的能力。也就是说，选项卡没有关联的激活按钮，但可以通过编程激活它们来显示。这允许用户为选项卡创建他们自己的定制按钮，例如在选项卡面板的侧面或底部。
![BogoTabs.jpg](../Images/bcc6050dbb7276dc0ebc87ca33b9f9ce.png)

## 6.CDZ 图像预加载器

一个用 JavaScript/jQuery 编写的图片预加载器。这将预加载目标元素上的所有图像，包括背景。抱歉，这似乎不是演示。
![CDZ-Image-Preloader.jpg](../Images/c21562e86cba992b90db5888db2a6b1e.png)
[来源](http://codezyne.com/projects/jquery-cdz-imagepreloader/)

## 7.弗卢维

这个小插件接受一个 div(或任何其他元素),并为指定的元素类调整所有元素的大小，使它们适合容器。同样，这个插件没有演示，但是如果你需要在容器中压缩元素，这个插件值得一试。
![fluwi.jpg](../Images/f23134bad4fb94c727ee56dbef819447.png)
[来源](http://sites.google.com/site/fluwijqueryplugin/)

## 8.GridNic 简单 jQuery 网格

是一个简单的 jQuery 网格插件，适用于需要可视化列表的人。
![GridNic-Simple-jQuery-Grid.jpg](../Images/98e125ff2d9cb75dfa1b3a6b90e12ef2.png)

## 9.用于 jQuery 的 haml dombuilder

jquery-haml 是一组 jquery 插件，使得在浏览器中构建用户界面变得容易。这是通过流行的 haml 模板语言的 js 变体实现的。
![haml-dombuilder-for-jQuery.jpg](../Images/860323050c88fff3ac49f00a52108ad7.png)
[来源](http://github.com/creationix/jquery-haml)
[演示](https://vimeo.com/6940290)

## 10.hyjack 选择

Hyjack Select 旨在取代传统的 HTML 选择输入控件(也称为组合框或下拉列表)。Hyjack Select 通过注入一个“search like suggest”文本框来查询页面加载的选择选项项，从而增强了选择。
![hyjack-select.jpg](../Images/4f2b861897157887f2c50e414ddbd764.png)
[来源&演示](http://www.frontensemble.com/hyjack/)
