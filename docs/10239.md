# 你很容易被遗忘吗？以下是如何变得令人难忘的方法

> 原文:[https://www.sitepoint.com/how-to-become-memorable/](https://www.sitepoint.com/how-to-become-memorable/)

你只有一次机会留下好的第一印象。这可能是陈词滥调，但这是真的。无论你是通过社交媒体进行互动，向潜在客户推销你的产品和服务，还是在现场社交活动中进行面对面交流，在每个商业环境中，别人如何看待你都很重要。

要给人留下好印象，没有什么通用的指导方针，但目标是一样的。你希望人们记住你，记住你的名字，记住你独特的品质，让他们更好地了解你。当人们记住你的时候，就更容易跟进、征求意见、开始对话、让人们点击进入、推销合资企业、请求帮助、销售某样东西、销售其他东西……你明白了。

这里有一些方法可以让你在任何情况下都令人难忘。

## 自信一点

自信的人有磁性；我们都想靠近他们，并倾向于听他们说些什么。只要你的自信是有保证的，而不是过分的，自信和控制谈话会让人想记住你的名字。

## 犯错

正如自信可以让你令人难忘一样，犯错也可以。错误展现了你人性的一面。如果你犯了一个错误，拥抱它，用它来让自己踏实。承认你的错误，而不是试图掩盖它，不仅可以表现出坦诚，这将得到他人的赞赏，而且可以让你变得更加平易近人和“人性化”

## 展示你的激情

你对一个热门话题有经过充分研究的观点，并且喜欢辩论吗？你对你每天所做的事情充满热情，并且喜欢谈论它吗？你有一个包含你想法的新项目吗？分享一下。激情是有感染力的，让其他人也像你一样充满激情是一种显而易见的让人难忘的方式。

## 扔掉饼干切割器

你可能已经准备好了一份完美的电梯间推介、一份扎实的介绍和一个无可挑剔的形象。但其他人也一样。做好准备是好的，但不要忽视让自己与众不同的力量。仅仅是一点点的不同就能让你变得令人难忘。

## 倾听…并真正关心

我们都喜欢听自己说话，我们都喜欢感觉人们完全投入到我们所说的话中。这是人之常情。你可以通过积极倾听和参与谈话来让自己令人难忘。尝试让他人发言，然后饶有兴趣地倾听，提出相关问题，并吸收他人的观点。

## 做你自己

你已经拥有了令人难忘的一切，因为你是唯一存在的你。做完美的自己比试图改变自己要容易得多，所以忽略这里分享的所有建议，让自己展现出来。
