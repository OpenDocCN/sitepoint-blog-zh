# Photoshop 中的线性减淡

> 原文：<https://www.sitepoint.com/linear-dodge-in-photoshop/>

快速回顾一下我关于[线性加深](https://www.sitepoint.com/blog/)的条目，提醒我们线性加深类似于乘法模式——只是“更暗”所以，令人惊讶的是，线性减淡类似于屏幕模式——只是“更亮”

让我们从我的玫瑰图开始:

![](img/17cff6c514a73b57e076f2b470151c7c.png)

添加掌纹层:

![](img/b8b4dab9f8c648fda758fa21372db20b.png)

掌纹层上的区域越亮，玫瑰层就变得越亮:

![](img/5aa9e34d91834f31eff63f7a562a5eec.png)

([下载样本。psd 文件](https://examples.pixelmill.net/sitepointblog/040826.psd)

正如线性加深会产生较暗的图像(除了纯白混合区域)，线性减淡通常会产生较亮的图像。这意味着我们可以在不断增长的技术列表中增加另一种快速增亮图像的方法！让我们从我在[屏幕模式](https://www.sitepoint.com/blog/)中使用的婴儿手图像开始，例如:

![](img/ff891fc014af1bf986f156b54f07c8f4.png)

一个复制层，设置为线性减淡，产生一个更亮的图像:

![](img/b7138e241fdd3af612f5b026a22144de.png)

你可以随时调整图层的不透明度来降低亮度。

([下载样本。psd 文件](https://examples.pixelmill.net/sitepointblog/040826/040826b.psd)
