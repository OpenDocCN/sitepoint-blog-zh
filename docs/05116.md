Chrome 是最大的赢家，已经恢复到每月 1%的涨幅。也就是说，浏览器在过去一年仅增长了 3.5%，因此增长已经停滞。

Firefox 和 Safari 几乎没有什么值得报道的，尽管 Opera 有一个小的飞跃。版本 23 最近发布，虽然它仍然缺乏 Presto 版本中的功能，但它是一个比 Chrome 更快的浏览器。

不出所料，Chrome 的增长意味着 ie 浏览器的又一次损失。它设法在 20%以上徘徊了一段时间，但几乎肯定会在 2014 年底前跌破这一门槛。IE11 可能永远达不到 2013 年 10 月 IE10 获得的最大 13%的市场份额。尽管 IE11 是一款优秀的全能浏览器，但它能在更少的平台上使用，并不比其他应用程序更好。它还有一个形象问题；[微软甚至考虑改名](http://www.theregister.co.uk/2014/08/18/ie_name_change_debate/)，以区别浏览器和它不光彩的过去。*(“网络浏览器”会更准确些！)*

*问题:*你还想在下个月的报告中看到 IE6 和 IE7 的单独数字吗？有一个使用的硬核，但浏览器在一年内几乎没有移动，可以总结为一个单一的“IE8-”数字。

## 2014 年 7 月至 8 月全球移动浏览器统计

2014 年 8 月，手机使用量增长了 1.2%，在所有网络活动中的占比[达到了 30.64%。北半球的夏季或许可以解释这种增长，但无法否认](https://gs.statcounter.com/#desktop+mobile-comparison-ww-monthly-201408-201408-bar)[移动增长趋势](https://gs.statcounter.com/#desktop+mobile-comparison-ww-monthly-200812-201408)。

手机使用率在两年内增长了两倍有几个原因:

1.  手机成本。高端市场仍然贵得惊人，但智能手机现在可以使用基本支付计划。
2.  设备能力。更大的屏幕和更好的处理器使我们能够使用手机进行一些活动，而这些活动在几年前还需要电脑来完成。
3.  发展中国家采用。亚洲、非洲和南美洲的移动增长呈爆炸式增长——甚至在 PC 时代未被注意到的地方。许多国家在移动商务和微支付方面领先于西方国家。

正如我上个月预测的那样，Chrome 已经超越 iPhone 和安卓应用，成为移动设备上的头号浏览器。鉴于 Chrome 去年同期仅占移动市场的 4%,而安卓浏览器同期仅下跌 6%,这是一个相当大的成就。

Chrome 现在是最顶级的移动和桌面浏览器。谷歌理应获得这样的地位，但我们应该担心它的统治地位吗？希望他们还记得上一家垄断浏览器市场的公司是怎么回事。
