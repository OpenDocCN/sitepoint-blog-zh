# 拯救地球，节省现金

> 原文:[https://www.sitepoint.com/save-the-planet-and-save-cash/](https://www.sitepoint.com/save-the-planet-and-save-cash/)

现在有很多更“绿色”的好理由，拯救地球，道德义务，等等。但是，让我们变得自私一点——这对我和我的企业有什么好处？我的意思是，做“正确的”事情当然有感觉良好的因素，但是存一些现金呢？

事实证明(并不令人惊讶)，在购买硬件时有点“环保意识”可以为您节省大量的电力和金钱。因此，在您购买下一台电脑之前，这里有一些方便的资源可以帮助您做出决定:

*   [电子产品环境资产工具](http://epeat.net/)可以帮助您发现最环保的台式机、笔记本电脑和显示器。
*   [能源之星评级](http://www.energystar.gov/index.cfm?fuseaction=find_a_product.showProductGroup&pgw_code=CO)。
*   计算在金钱、二氧化碳、树木和汽车方面的节约(Excel 电子表格)。
*   调查在 [80 加上](http://www.80plus.org/index.htm)时电源的效率。
*   使用[交换](http://www.sun.com/servers/coolthreads/swap/#calc)(空间、功率和性能)指标计算您的服务器效率。

您还可以使用电源管理工具检查当前电脑的运行情况，或者使用 [SusiClimate](http://www.susigames.com/populator/) 在 OSX 上监控您的二氧化碳排放量。

下一次购买计算产品时采用智能节能技术将为您节省资金。继续吧。节省一些现金，帮助拯救地球。TTFN。
