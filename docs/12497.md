# 谷歌 Chrome 统计，三个月后

> 原文:[https://www.sitepoint.com/google-chrome-three-months-out/](https://www.sitepoint.com/google-chrome-three-months-out/)

在谷歌基于 Webkit 的新浏览器 Chrome T2 上市后不久，我们调查了一下有多少人在使用它。我们对[早期统计](https://www.sitepoint.com/early-stats-google-chrome-hurts-firefox-not-ie/)的第一次观察发现，令人印象深刻的是，4-5%的用户在 Chrome 发布几天后就开始使用了。也许并不奇怪。鉴于 Chrome 最初针对的是更具技术性的早期用户群体，所以一开始受到打击的是 Firefox 而不是 Internet Explorer，尽管大多数专家都认为谷歌的浏览器计划是针对微软，而不是 Mozilla。

在新浏览器产品最初的兴奋消退后，我们在本月早些时候注意到，在 SitePoint，Chrome 的使用量已经稳定在略高于 3%的水平，仍然高于大约 0 . 25%至 1%的全球平均水平。

现在判断 Chrome 对浏览器竞赛的长期影响还为时过早，但三个月比一周更好，所以再次看看这些数字并看看它们是什么样的并不是一个坏主意。

最新一轮的浏览器份额比较是由 CNET 的斯蒂芬·尚克兰发起的，他本人也因为 Chrome 优越的 JavaScript 速度而转投 Chrome。Shankland 发现，从 9 月到 10 月，访问 CNET 的游客使用 Chrome 的人数比预期的多。尽管 10 月份只有 3.6%的访问者使用 Chrome，但这仍然是一个巨大的飞跃，比前一个月的 1%有所增加。Firefox 也跳了出来，IE 和 Safari 受到了冲击(也许这是 Windows Safari 用户，因为 Mac 版 Chrome 尚未正式存在——或者这可能是 Firefox 的作品)。

在 SitePoint，我们实际上看到了相反的趋势。在 9 月到 11 月之间(这个月还没有结束，但我们认为它有足够的数据可以包含在图表中，记住这一点)，Internet Explorer 实际上在我们的网站上看到了巨大的使用量，而其他浏览器的使用量则逐月大幅下降。原因可能是从 10 月下旬开始，我们有来自 Digg、StumbleUpon 和 Delicious 的大量流量，这是因为一些博客帖子在这些社交媒体网站上表现良好。因此，过去 6 周的流量对 SitePoint 来说是非典型的，因此可能会扭曲我们的数字。

![](../Images/472fb87e53f354758f0bbfc2416aaf36.png "3month-browserstats")

然而，这仍然给了我们一些数据，从中我们可以得出一些结论。如果当我们的流量开始来自更主流的来源时，Chrome 的使用——实际上还有 Firefox、Safari 和 Opera 的使用——下降了，这表明主流仍然非常依赖 IE。Firefox 和 Mac 上的 Safari 可能削弱了 IE 对浏览器市场的垄断，但 Chrome 甚至还没有触及主流。

来自网络应用的最新数据证实了这一理论。根据他们的浏览器份额报告，Chrome 仅被 0.75%的网络用户使用，和 Opera 差不多。Firefox 徘徊在 20%左右，而 Internet Explorer 以超过 70%的使用率保持绝对领先。Chrome 在短短 3 个月内成功抢占的浏览器份额并不小，然而，距离跨越鸿沟还有很长的路要走。
