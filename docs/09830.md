# 一切都要适度——即使是主持人

> 原文：<https://www.sitepoint.com/everything-in-moderation-even-moderators/>

上周，我被邀请做一个关于社区管理的采访，这让我开始思考。其中一个问题是关于员工结构和管理的，这与我的博客帖子相吻合，该帖子谈到了论坛员工的态度如何影响人们对整个社区的看法。

在我为 SitePoint 论坛工作期间，有两次我们不得不非常努力地去动摇一个声誉。第一次是在很多年前，当时我刚成为一名主持人。我们以专横和不受欢迎而闻名。作为一个社区经理，这两个描述让我畏缩。他们离任何一个社区管理者所期望的员工都差得太远了。我们非常努力地想要改变这个名声，尽管这花了几年时间，但我们做到了。SitePoint 论坛成为了一个让新人感到受欢迎和舒适的地方，员工被视为解决问题的人——就像一些城市街角的大使一样——而不是一有人发出声音就马上带上手铐的警察。

最近，工作人员和我一直在讨论如何处理我们社区中被认为是捣乱分子的人。任何做过志愿者的人都知道，你做志愿者是出于热爱。除了我之外，SitePoint 论坛的所有成员都没有工资。他们都做好自己的工作，因为他们重视我们所做的工作，并希望回报社会。他们是品牌大使，他们为 SitePoint 所代表的意义感到自豪。随之而来的往往是强烈的忠诚。当你真的努力工作去建立你引以为豪的东西时，你不喜欢看到它被虐待或误用，当它被虐待或误用时，你会出来抗争。

听着，事情是这样的。网络社区不是民主。我们希望人们能够公开表达他们的观点(只要不是他们对宗教或政治的观点！)但归根结底，我们有责任确保这不会损害其他成员的利益。新成员不想听到老成员仅仅因为他们以前听过这个问题就大发雷霆。我们需要放他们一马。如果他们被吓得不敢回来，他们什么也学不到！另一方面，已经存在一段时间的人不希望一个新人进来并开始诽谤。没人喜欢巨魔。

再加上一点员工忠诚度，事情会变得更加复杂。没有人喜欢看到自己关心的人被侮辱。这完全是一种“我可以说我喜欢我哥哥的什么，但如果你说他的任何坏话，我会杀了你”的心态。如果一个机构群体成员对某个工作人员有异议，那么这个人的团队就很难袖手旁观。我们也是人。我们把事情看得很私人。当然，在一个理想的世界里，我们会说“嘿，我在这里很难做到客观，所以如果有人能代替就太好了”，但是告诉别人你真实的想法会更令人满意。

事情偶尔会出错，这是必然的，但你的反应将决定你的社区在未来的行为和对你的反应。尊重他们，你会得到回报的。挥动你的鞭子，他们会用脚投票。

我们已经意识到，最近我们可能忽略了这一点。在过去的几个月里，我们有几个成员不同意我们的工作方式，我们并不总是做出理想的反应。我倾向于信口开河，我必须非常努力地保持这种状态。车队的其他人帮助我做到了这一点，他们在很大程度上保持了良好的平衡。

但请记住，我们的决定是经过深思熟虑的，我们总是以我们认为最有利于社区的方式行事。如果你有让你心烦意乱的经历，请确保让我们知道，但是想想你是怎么做的。当然，在论坛上公开表达你的不满是一种方式——人们似乎认为这会让我们对自己的行为负责——但这并不总是最有效的处理方式。我们都支持透明度，但为了保护其他成员的隐私，我们不会总是公布细节，所以你不太可能得到你想要的答案。如果您对员工有任何疑问，请发私信或电子邮件给我，我会尽我所能确保我们达成一个让所有人都满意的结果。

这就是健康社区的运作方式。
