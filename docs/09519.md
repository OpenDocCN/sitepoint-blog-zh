# 任务交换:自由职业者生产力的关键？

> 原文：<https://www.sitepoint.com/task-swapping-a-key-to-freelance-productivity/>

如果你读这篇博客是因为你想避免做你不喜欢的事情，我有一个建议给你。

停下来。

如果你不喜欢你正在做的任务，那就做点别的。

## 停下来？！但是我永远也不会完成任何事情！

员工的心态告诉我们，我们必须做丑陋的工作，面对我们无法面对的事情。不要拖延，不要拖延，只管去做。

不过，自由职业者有一个选择。没人在监视我们。我们可以随心所欲地安排我们的一天、一周和工作生活。

所以如果你不喜欢你正在做的任务，你可以推迟它。

是的，你过会儿得回来做它。确实如此。但是通常，是我们的心情让一个没有吸引力的任务——众所周知的小山丘——看起来像一座山。

最近，当我坐在办公桌前苦干一项毫无吸引力的任务时，我意识到我缺乏动力导致这项工作比需要的时间长了十倍，压力大了五十倍。

“伙计，我讨厌这样，”我一直在想。“我希望它结束，这样我就可以做其他的事情。”

这种紧张感越来越强烈，直到我的大脑爆炸前的一瞬间，我决定停下手头的工作，去做其他的事情。

给自己这种自由实现了两件事:

1.  我全身心地投入到另一项任务中，并且乐在其中。
2.  后来回到第一个任务的时候，我是完全不同的心态。原本难以逾越、令人麻木的任务已经变得轻而易举。我几分钟就做完了。

## 为什么有效

通过允许自己放弃(甚至暂时放弃)自己不喜欢的任务，你可以重新获得对工作的掌控感——毕竟，这也是许多人当初选择自由职业的原因。

当你重新获得那种控制感时，你可能会更有动力去做你喜欢的任务，这意味着你更有可能集中注意力，并在更短的时间内高标准地完成它们。

而且，在有趣的、鼓舞人心的任务上花了一些时间后，你可能会对处理无聊的事情感觉更积极。像我一样，你可能会发现当你回到他们身边时，他们似乎根本不需要时间。

如果你能做到这一点而不走极端——错过最后期限，无法交付——那么你也许可以把这个技巧运用到你的生产力库中。

你曾经在工作中把无聊的任务换成有趣的任务吗？这对你提高工作效率有帮助吗？
