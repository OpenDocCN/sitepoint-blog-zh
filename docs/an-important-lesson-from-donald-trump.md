# 唐纳德·特朗普的重要一课

> 原文：<https://www.sitepoint.com/an-important-lesson-from-donald-trump/>

我个人是受不了唐纳德·特朗普的。他的价值观与我的许多价值观相反。他在人际关系中不忠，享受名声，似乎是一个持续渴望自我满足的可悲角色。他本人应该是个很有魅力的人，但我从未见过他，所以不知道。

本周我在《人物》杂志的封面上看到他和他的新生婴儿(不是坐在他旁边的年轻女人，而是她怀里的婴儿！).

无论如何，他在经济上是成功的，我们可以从他的一些策略中学到一些东西。我认为最适用于 Sitepoint 读者(通常是 20 多岁的年轻人，就像他刚开始时一样)的一个策略是他刚开始时做的:

他加入了一些富人和有影响力的人经常出没的高级俱乐部。也许他父亲动用了一些关系让他加入，但他很有可能是用自己的说服力加入的。

然后他继续前进，继续会见重要的联系人。

在你的地区有各种各样的俱乐部。商会就是这样一个俱乐部，尽管要见到真正的行动者和震撼者，你需要加入一个董事会或重要的委员会，并真正承担起领导角色。加入你当地的校友会或大学俱乐部。参观投资者研讨会。加入一个重要的非盈利组织的董事会。

有人成功尝试过吗？
