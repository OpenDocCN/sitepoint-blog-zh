# 信息的独特性

> 原文:[https://www.sitepoint.com/a-singularity-of-information/](https://www.sitepoint.com/a-singularity-of-information/)

昨晚我和一个朋友边喝咖啡边聊天。他刚刚读了一篇文章,作者在文章中假设我们正在接近一个“奇点”——技术改变人类生活本质的时间点。

这不是一个新想法。几年来，已经有关于这个东西的[会议](http://www.accelerationwatch.com/)和[论文](http://www.aleph.se/Trans/Global/Singularity/)和[书籍](http://singularity.com/)。但是，与利用集体智慧的一些 Web 2.0 概念的结合使许多 Web 开发人员想知道 Web 是否将是这种奇点首次亮相的地方。

每天，人们都在发明新的方法在网上完成工作。对于我的工作来说，我必须解决的最大问题是保持在不断上升的新闻、产品和想法浪潮的顶端，这些都在 Web 开发的世界中打转。如果技术奇点出现，所有的信息都会以某种方式被过滤并直接流入我的意识。

我个人认为，阻碍技术奇点到来的不是技术创新的速度，而是人们愿意采用该技术的速度。许多未来学家预测，在未来五十年内，人类将在生物层面上与计算机融合。但是你认为今天出生的孩子中有多少会愿意采用改变他们大脑工作方式的技术呢？

撇开控制论的植入不谈，语义网承诺的毫不费力地链接信息的乌托邦式的未来，并开始由 Web 2.0 传递，这无疑是一种奇特的现象。但与其说是技术奇点，我所说的是*信息*的奇点——确保任何人都能在任何时间、任何地点找到他们感兴趣的信息。

现在*这是*一个我可以加入的奇点。作为网络开发者，我们有责任实现它。
