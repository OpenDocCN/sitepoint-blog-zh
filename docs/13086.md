# 混合:第一天

> 原文:[https://www.sitepoint.com/mixing-it-up-day-one/](https://www.sitepoint.com/mixing-it-up-day-one/)

MIX 08 今天早上以一个多头主题演讲拉开了序幕，其中有很多“哇”的因素。毫无疑问，最大的收获是:微软非常重视互联系统和开放标准。旧的拥抱-延伸-熄灭模式基本上已经死亡。雷·奥茨踢了那东西，但他的流行语密度让我想吐。如果你看了视频，请随意快进，直到你看到斯科特·格思里。除非你喜欢看高管们喋喋不休。一旦我们过了这一关，我们就进入了好东西:IE8 和 Silverlight 2.0。

IE8 的亮点包括:

*   IE8 的主要焦点是改善 web 开发人员的生活。不像 IE7 那样关注用户体验和安全性。
*   全面支持 CSS 2.1。此外，微软将发布一个 BSD 许可的 CSS 2.1 测试包，包含 700 多个测试。
*   javascript 堆栈的显著性能改进。
*   HTML 5 支持的内核。演示的改进是改进了对 AJAX 请求的后退按钮的处理，并利用 HTML 5 的连接事件模型让浏览器更优雅地处理失去服务器的情况。
*   它包括一个与 firebug 不相上下的调试工具，如果你把 net monitor 特性从桌面上拿掉，并且记得那些已经被 Fiddler2 覆盖了。
*   活动—想想 IE 中的 office 智能标记。它们基于新的 OpenServiceSpecification，该规范拥有 Creative Commons 许可证。
*   网页快讯——基本上是 favoritelets，允许更直接地传递内容。规范也在某种形式的知识共享许可下。
*   自己看看吧— [Beta 1 已经发布](http://www.microsoft.com/windows/products/winfamily/ie/ie8/default.mspx)。

银光 2.0

*   总的来说，非常非常酷。对学校来说太酷了。
*   从开发人员的角度来看，主要的改进是新的 WPF 兼容控件，具有源代码可用性，以及改进的网络堆栈，包括跨域请求的能力。
*   自适应流非常流畅。基本上，Silverlight 客户端可以检测网络速度问题并动态切换流速度。除此之外，很明显，它还与运行在 IIS7 上的 Windows Media Services 进行了大量集成。
*   他们演示了一些令人印象深刻的应用程序来展示其功能。全国广播公司的 2008 年奥运会网站将是惊人的，超过 2200 小时的在线直播内容，以连接的方式表现。硬石大事记网站也不赖。在这里，我真的不能公正地对待他们——查看[网络广播](http://wm.istreamplanet.com/customers/ms/750_microsoft_mix_080305.asx)[感谢直接链接 ionix5891]。
*   Beta 1 已经发布，伴随而来的还有 [Visual Studio 模板](http://www.microsoft.com/downloads/details.aspx?FamilyID=e0bae58e-9c0b-4090-a1db-f134d9f095fd&DisplayLang=en)等等。

Scott Guthrie 还提到 ASP.NET AJAX 更新正在进行中，并暗示动态数据控件的更新版本即将推出。最后，更新的 ASP.NET MVC 框架已经发布。

我也借此机会去看看空地。非常非常酷的概念，和几个不知名的“软人们”谈论我不能在这里重复的事情是很棒的。我确实有机会见到了 SitePoint 的博客作者伊恩·穆尔和我的合著者菲尔·哈克。说到社交，派对在陶开始，所以我得去酒吧了。
