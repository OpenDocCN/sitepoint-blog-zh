# Photoshop 中简单的 3D 效果环绕导航菜单

> 原文:[https://www . site point . com/simple-3d-effect-wrap-around-navigation-menu-in-Photoshop/](https://www.sitepoint.com/simple-3d-effect-wrap-around-navigation-menu-in-photoshop/)

在本教程中，我们将看看如何创建一个简单的 3D 效果，可用于网页导航侧边栏。

1.用 Photoshop 500 高 x 300 宽像素创建一个新文档。用#333333(深灰)填充背景。

2.从工具栏中选择矩形工具。在顶部的工具选项栏上，选择形状层，然后单击样式下拉箭头查看可用的样式。我们将使用渐变风格。要将它添加到样式框中，点击样式下拉框右上角的小三角形，从出现的菜单中选择 Web 样式。

![1-RectangleToolOptions](../Images/a87a7652b5ff469e0a51209d90d1d6ba.png)

将询问您是否要替换或追加当前样式。选择追加。向下滚动已经添加的新样式，选择“黑色到透明 2”现在画一个矩形，填充大约一半或三分之二的文档。可以画矩形，这样它就超出了文档的边界。最后，在图层面板中出现的新矩形图层上，添加一个白色的 2 像素厚的描边图层样式。您应该会得到这样的结果:

![3-Rectangle](../Images/475c9b3dd4a7a794d623bef84e3756a8.png)

![4-LayersPalette](../Images/45225de33c209157df69b22f2294efb0.png)

好了，这就是背景设置。左边的区域是我们的侧边栏，我们将在这里添加简单的 3D 菜单。

3.在工具箱中选择圆角矩形工具。它将被隐藏在矩形工具下。在屏幕顶部的工具选项栏中，确保选择了形状图层，并将颜色设置为#990000(深红色)。拖出一个类似这样的矩形:

![5-DrawButton](../Images/5b5a6765c091b792b7c3b8dc3605821a.png)

将形状层重命名为“按钮背景”

5.现在，我们将使用钢笔工具绘制一个形状，使它看起来像按钮被折叠在侧边栏下。选择钢笔工具，设置图层形状，设置颜色为白色。向右放大，以便您可以清楚地看到您正在绘制的内容。画一个类似这样的形状:

![6-PaperFold](../Images/155d184ac1573c8419e5ea2108c2ebff.png)

现在重命名新的形状层为“折叠纸”

6.拖动按钮背景层下的纸张折叠层。

7.下一步是改变按钮背景(这是一个矢量形状)，使右下角看起来像是向下折叠，匹配右上角。从工具箱中选择直接选择工具(白色箭头),然后单击红色按钮的边缘，这样就可以看到锚点。

点击并向下拖动最右边的锚点，这样你就开始得到一个向下的曲线。使用贝塞尔手柄调整曲线的方向和形状。您还需要向左移动锚点，并用该点的手柄调整曲线。试着得到一个类似这样的形状:

![8-AnchorPoints](../Images/cb098c15442667515d25db9b2843bd3e.png)

8.所以我们的按钮现在已经改变了一点点形状，它覆盖了我们的纸折叠层。选择纸折叠层，用移动工具，向下拖动它，使它看起来像角落满足和折叠正确。

![9-Papermeeting](../Images/6d5ffc4297bfc5725c82ef24a4156f1c.png)

9.选择按钮背景层，并添加一个图层样式的中风大小 3，在白色，外面的位置。

![10-Stroke](../Images/1814fbb7c1dbb1e68a5a94c0e7ae0030.png)

10.我们需要删除笔画的右端。要做到这一点，我们可以把中风的影响到一个单独的层，并掩盖了右端。选择背景图层按钮，进入图层>图层样式>创建图层。你会注意到按钮形状层下面多了一层。选择它并添加一个快速蒙版。用黑色画笔涂掉按钮右侧的白色线条。也轻轻地掩盖了顶部曲线的一部分。

![11-MaskStroke](../Images/840a69b684c32cb926c5509e86dbfb9f.png)

11.在白色描边图层仍然选中的情况下，添加一个阴影图层样式。这给了一个非常好的效果，并真正解除了页面上的按钮。摆弄滑块，看看你最喜欢哪种效果。这些是我使用的值。

![11-DropShadow](../Images/895b735287e4cb934b6aa5224c6ff06f.png)

单击“确定”添加投影。

12.选择背景按钮层，并添加一个内部阴影，其值如下:

![12-InnerShadow](../Images/f2689eb6221c142a1187b2a1693f602c.png)

单击“确定”应用内部阴影。

13.快到了！最后一步是向按钮添加一些文本。瞧啊。

![13-TextAdded](../Images/561b54e51b3ee648043728200a621c51.png)

你可能需要不止一个按钮在你的导航条上，所以我们不需要重复整个过程，只需要复制我们已经做的就可以了。

14.在图层面板上，点击“创建一个组”按钮。将群组重新命名为“按钮”现在选择所有你已经创建的层来组成你的按钮，并把它们拖到组中。确保不要拖进你做的第一个背景。

15.要复制该组，通过图层面板向下拖动，放在“创建新图层”图标上。

![14-DragGroup](../Images/aab6b1e3bc93165f35b0cb95ee86b1d0.png)

15.一旦你创建了一个副本，选择工具箱中的移动工具(但不要点击屏幕上的任何地方)，只需使用上下箭头键上下移动整个组。重复这个过程，直到你有所有你需要的按钮，然后改变每个按钮中的文本。这是我完成的导航栏。

![15-FinalButtons](../Images/30ed528dabfc813782cc0d36f8751e02.png)
