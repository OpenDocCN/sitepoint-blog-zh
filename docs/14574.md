# 调查客户第二部分

> 原文：<https://www.sitepoint.com/surveying-clients-part-ii/>

[https://www.sitepoint.com/blog-post-view.php?id=172099](https://www.sitepoint.com/blog/)

在上周的博客中，一位读者(0 型)问，为什么要用同事来调查客户，是否会有报酬。

关于赔偿:是的，赔偿是意料之中的。事实上，许多咨询公司习惯于采访专业服务公司的客户，并反馈结果和建议。

关于是否应该自己动手:

简而言之，这要视情况而定。一个客观的同事通常可以让你的客户分享更多的信息，而且比你自己做的事情更加直率。此外，许多 IT 专业人员不擅长专业地接受反馈。他们倾向于自我防御，因为他们的工作与个人利益息息相关。这可能会伤害你们的关系。所以客观的同事会让事情更专业。

但是，如果你知道如何获取反馈(例如，问开放式问题，超脱自我，引导客户进入你感兴趣的领域，从不争辩或为自己辩护，从不责备客户，心怀感激，将每次采访作为数据点，而不是统计上有效的调查，等等……..)那你自己做吧。事实上，如果你这样做了，而且做得很专业，你最终会比用一个局外人更稳固地改善关系。

因此，鼓励客户给出开放和诚实的反馈，同时像专家一样接受艰难的建议，这取决于你和你自己的能力。
