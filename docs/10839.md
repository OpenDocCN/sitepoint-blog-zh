# 社交媒体噪音的一体化解决方案

> 原文:[https://www.sitepoint.com/solution-to-social-media-noise/](https://www.sitepoint.com/solution-to-social-media-noise/)

我最近偶然发现了 [NutshellMail](http://nutshellmail.com/) 。这是一项由 Constant Contact 提供的免费服务，它收集你所有社交网络的更新，并通过电子邮件摘要发送给你。这项服务旨在帮助你只获取对你来说重要的信息，并节省时间。

## 涵盖的社交网络和数据

以下是您可以从该服务支持的每个社交网络收集的更新的分类:

*   脸书:状态更新、照片、新闻、生日和好友请求
*   **Twitter:** 选定的 Twitter 搜索关键词、新关注者和未关注者、发送的推文、直接消息、转发和@回复
*   MySpace: 更新、照片、视频和好友请求
*   LinkedIn: 人脉、个人资料更新、工作列表、群组更新、推荐和问答更新&
*   **宁:**全宁网活动

## ![](../Images/bbe248b703a0584311198ac600fd0d3b.png "nutshellmail-115x115") NutshellMail 功能

作为一名刚刚注册的新用户，该服务提供的信息深度给我留下了深刻的印象，尤其是当我担心我经常错过我想看但不能同时出现在 5 个地方的信息时，或者因为我听不到社交媒体的噪音时。

此外，我喜欢将所有内容都记录在电子邮件中，并能够在最适合我的时候查看，而不是在我指定的每日检查期间从脸书到 Twitter 再到 LinkedIn。可能会很烦。

这些邮件是 HTML 格式的，所以很容易浏览。甚至还有允许你发起评论的按钮，比如回复、转发等等。就在你的邮件里。

其中一个最吸引人的功能是，你可以设置一个电子邮件简讯，一旦你的脸书粉丝选择加入，它就会自动将最新的消息从你的粉丝页面发送到他们的电子邮件收件箱。我用了不到一分钟的时间在我的一个页面上设置了这个，它看起来确实像它说的那样。这是一个很好的方法，可以让粉丝跟上你的页面并保持参与。

其他一些有用的功能包括:

*   自定义交付计划(日期、时间、频率)
*   能够从同一个社交网络添加多个帐户
*   仔细挑选你想看的和不想看的

## 你的份额？

还有其他的服务会给你发送更新摘要，但是我还没有找到一个可以覆盖多个网络(以及这些网络上的多个账户)的这种级别的数据。我希望它能帮助我跟上我的社交网络，并让我做起来更轻松。

但是，你怎么看？你认为你能从这样的摘要服务中获益吗，或者你更喜欢全部手动完成(或者根本不做)？
