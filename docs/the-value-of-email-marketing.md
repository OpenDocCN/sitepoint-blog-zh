# 电子邮件营销的价值

> 原文：<https://www.sitepoint.com/the-value-of-email-marketing/>

根据直销协会的数据，2006 年，平均来说，电子邮件营销每花费 1 美元就能产生 51.45 美元的平均回报。

如果仅仅是这个统计数字还不能让你相信电子邮件营销能给你的企业带来的价值，那就没什么能让你相信了！当然，电子邮件营销如此受欢迎有很多原因——它非常划算和及时，它有助于与特定的细分市场、子细分市场甚至受众中的个人建立持久的关系，它与其他营销策略(包括病毒营销等)结合得很好——但这个统计数据真的应该让任何在线运营者刮目相看。

你从你的电子邮件营销努力中赚了那么多钱吗？

这是我在最近发布的[电子邮件营销套件](https://www.sitepoint.com/premium/library)的开篇章节中读到 DMA 统计数据时想到的第一个问题。我的意思是，我们已经开展电子邮件活动六年多了——如果你考虑到我们的电子通讯，时间会更长——我认为我们已经把它归结为一种艺术。这并不是说我们什么都知道，每次都能在第一时间写出一流的电子邮件，但我们确实意识到了一些策略的价值，如试发电子邮件，设置活动盈亏平衡点和现实的响应目标，跟踪和分析活动结果，以及尝试不同的电子邮件类型，等等。

问题是，我们花在电子邮件营销上的每一美元都赚不到 51.45 美元。

现在，这个数字是平均值，它不是你认为你的活动成功所需要的投资回报率。但出于同样的原因，这并不是 DMA 报告中被调查公司产生的最大投资回报率。只是一般。

显然，我们在 SitePoint 的工作还有改进的空间。虽然我可以看到一些可以改进的地方(包括雇佣专门的在线营销人员)，但电子邮件营销套件的第一章提供了一些提示，无疑给了我一些思考。

首先是关于建立关系的那一点。该工具包的作者 Jeanne Jennings 指出，通过电子邮件比通过更传统的渠道更容易建立关系。这绝对是我们在电子邮件策略中尝试做的事情。但我认为我们可以做得更好，特别是在我们的购书者俱乐部方面，这通常是在针对特定书籍的电子邮件宣传活动之后的待办事项清单上的第二个账单。我们也没有尽可能好地跟踪来自该列表的回复，尽管购书者俱乐部通常是我们客户关系的起点——在购买之前，与感兴趣的潜在客户建立关系。

Jeanne 关于电子邮件生产速度快这一事实的讨论强调了我认为我们可以关注的另一个领域。当然，当你将它与平面广告或 TVC 相比时，直接电子邮件可以快速开发、测试并到达客户手中。但是，如果我把它与我们的博客或 SitePoint 论坛中的口碑力量进行比较，我不禁会想，我们是否真的尽可能快速有效地开展了活动。当然，电子邮件营销使受众细分变得如此容易，这意味着我们将每一次新的营销活动都视为一次绝佳的机会，来测试新事物和磨练我们从以前的营销活动中学到的想法，而发展这些新概念可能需要时间。但尽管如此，我认为还有改进的空间。

最后，该工具包的第一章指出，作为一种营销工具，电子邮件与其他策略配合得很好——例如，使用电子邮件来提醒客户你已经向他们发送了一封实体邮件，或者补充你已经发布的印刷广告，都很容易。实际上，目前我们所有的图书营销都是在网上进行的，所以这个建议真的引起了我的思考。如果我们的活动与线下促销相结合，会更有效吗？我们最近在澳大利亚当地报纸上的提及无疑提高了我们在当地的知名度，所以我正在认真考虑在美国和英国的可能性，那里是我们最大的受众群。

任何能产生像电子邮件营销那样的投资回报率的营销技术都值得我们考虑。但是你我都知道，要取得这样的结果需要技巧和知识。目前你在做什么来改进你的电子邮件营销活动？读完这篇文章后，你认为你可能忽略了其他方面吗？
