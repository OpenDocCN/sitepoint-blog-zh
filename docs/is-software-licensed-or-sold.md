# 软件是许可的还是出售的？

> 原文：<https://www.sitepoint.com/is-software-licensed-or-sold/>

销售软件是一个法律雷区。当你购买数字产品时，你买的不是 CD/DVD 和包装，而是软件的使用许可。许可证是一份具有法律约束力的协议，决定了您如何使用该产品。如果软件被“出售”，你将是合法的所有者，可以做你想做的事情，例如在任何地方安装、拆卸、赠送或做任何不违反版权法的事情。

然而，当你不再需要那个产品的时候会发生什么呢？是否可以像出售二手书籍、音乐 CD 或电影 DVD 一样，将您的软件许可证转售给另一方？Vernor 诉 Autodesk，Inc .一案凸显了这一法律灰色地带。

Dan Schulz 在 SitePoint 论坛上发表的一篇文章详细描述了全部的法律论据。总之，Vernor 试图在易贝出售 AutoCAD 的合法副本，但在 Autodesk Inc .的干预下，他的帐户被暂停。Vernor 在联邦法院起诉 Autodesk:

*   Vernor 声称他从软件的合法拥有者那里购买了许可证。因此，所有权在购买时转移给他。
*   欧特克公司声称所有权从未转让，Vernor 的销售侵犯了版权，称其许可证是一份“计算机软件协议”，而该行业通常限制所有权转让。

法院不同意欧特克的观点，并认为欧特克提交给它的“软件行业实践的证据不足”。因此，该裁决允许被许可方转让软件所有权，版权所有者对此没有发言权。软件公司销售的是最终用户拥有的产品；如果他们预先得到全部价值，一旦软件进入商业流程，他们就无法控制它。

Autodesk 可能会上诉，但目前 eBay.com 上至少有 12 个 AutoCAD 副本。

这项裁决对网络开发者来说是好消息吗？最终用户很少购买 web 应用程序或“拥有”软件——他们订购服务。许多软件公司将重新考虑他们的软件发行选择。
