# SlideRocket 推出定价、市场

> 原文:[https://www . site point . com/slide rocket-launches-announces-pricing-market place/](https://www.sitepoint.com/sliderocket-launches-announces-pricing-marketplace/)

基于 Flex 的演示文稿创建 web 应用程序 [SlideRocket](http://www.sliderocket.com/) 今天将结束测试，并宣布其高级服务的定价计划。该公司还宣布与几家供应商合作，作为新推出的市场的一部分，该市场将允许用户购买产品和服务，直接在 SlideRocket 中使用他们的演示。

SlideRocket 无疑是网络上最完整的演示应用，也是最有可能与微软的 PowerPoint 竞争的应用。SlideRocket 基本上做了你期望演示应用做的所有事情，但真正让自己与众不同的是许多 web 应用做的事情:协作。SlideRocket 上的每个演示都可以共享到幻灯片级别，并且可以跨幻灯片动态更新更改。例如，假设您的销售部门创建了一个销售宣传资料，并使用了一些由营销部门创建并与他们共享的幻灯片。然后发生了一些变化，营销人员需要更新幻灯片，当他们更新时，这些更新会自动推送到销售演示中使用的幻灯片。

创始人米奇·格拉索(Mitch Grasso)告诉我，如今大约有 3 万人在使用 SlideRocket。我问格拉索，他是否预计到了用户会对一项已经免费了 8 个月的服务增加定价计划。他告诉我，令人惊讶的是，他们中的许多人一直在要求一个能让他们为服务付费的定价方案。许多企业客户在付费使用 web 应用程序时会感觉更舒服。

## 定价

SlideRocket 将于今天向公众推出三级价格计划:免费、每月 15 美元和每月 30 美元(尽管这两种付费计划在推出时都有折扣)。免费和付费之间的最大区别是每个帐户允许的存储容量——免费用户 250mb，付费用户 1GB。价格最高的付费账户也允许多个用户，并开放了 SlideRocket 令人印象深刻的协作功能、版本控制和网络会议应用程序，允许幻灯片与远程观众共享。

付费计划包括电子邮件支持，而电话支持面向所有用户，每次 49 美元。将向企业客户提供电话不限次数的高级支持，收取 20%的订购费(最低 1500 美元)。

## 市场

SlideRocket 今天宣布的最令人印象深刻的部分是市场。当前的 SlideRocket 用户应该已经熟悉了 marketplace 的概念，因为这个应用已经内置了搜索网络上免费内容的功能——比如 Flickr 上 CC 许可的照片。

SlideRocket 与两个合作伙伴一起推出，股票图片网站 [Fotolia](http://www.fotolia.com/) 和打印机[油印](http://www.mimeo.com/)。演示模板提供商 [PresentationPro](http://www.presentationpro.com/) 即将推出。在 SlideRocket 中，可以使用统一的内置信用系统购买来自每个市场提供商的内容。

加入市场是 SlideRocket 的明智之举，因为它允许他们将免费计划货币化。Grasso 告诉我，他还与其他五家公司中的四家签订了合同，最终，SlideRocket 计划将市场扩展为一个面向各种规模供应商的全面服务门户。

希望市场最终成为“iPhone 应用程序演示商店”，出售各种各样的内容和服务，如视频、库存照片、图形/剪贴画、字体、数据馈送(用于动态更新的图形等)。)、平面设计服务、文案写作或公共演讲课。

SlideRocket 的客户将能够从 Fotolia 搜索和购买股票照片，就像他们目前搜索 Flickr 图片一样。购买的图像将直接添加到他们帐户的资产区域。
