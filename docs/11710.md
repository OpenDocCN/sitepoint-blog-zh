# 16 个客户服务/服务台应用程序，让客户支持变得简单

> 原文:[https://www.sitepoint.com/customer-service-help-desk-apps/](https://www.sitepoint.com/customer-service-help-desk-apps/)

我在过去的帖子里讲了很多关于客户服务的重要性。无论你做得多好，如果你不回应客户的反馈，不愿意采取措施改善不满意，你将很难管理好你的客户关系。

管理、跟踪和处理客户查询和支持请求的一种方法是使用帮助台/客户服务应用程序。这里列出了 16 种客户支持工具，您可以使用它们来提高可访问性和响应能力。

**[Artologik 服务台](http://www.artologik.com/en/HelpDesk.aspx) :** 一个基于网络的支持系统，提供内部和外部支持，包括知识库、可搜索的常见问题解答、票证创建和管理、电子邮件提醒和报告。帮助台软件有三种许可选项。

![Artologik HelpDesk](../Images/3fcbfd5de9abe203999e2f986d588fa4.png)

**[bigWebApps help desk](http://helpdesk.bigwebapps.com/):**一个可扩展的、基于 web 的应用程序，包括电子邮件集成、远程协助、时间跟踪、报告和公告功能。价格可应要求提供。

![bigWebApps HelpDesk](../Images/c5528d98a039f802cf24795039f7ae19.png)

**[数据跟踪系统](http://www.magnoware.com/) :** 基于网络的服务管理解决方案，允许您跟踪、分类和优先处理工作请求；生成详细的统计数据和报告；自定义您的界面；为你的客户提供自助区。根据你需要的技术人员的数量，他们提供了几种方案。

![DataTrack System](../Images/fa203fdfdcc3aa00b16c51f4545f6156.png)

 **[h2desk](http://www.h2desk.com/) :** 一个服务台系统，包括基于电子邮件的票证管理、知识库和故障排除程序、自助服务客户端门户、移动访问、定制选项以及与其他第三方应用程序的集成。h2desk 可作为托管或自行安装的解决方案。

![h2desk](../Images/d2afe8f2f6bef0911b48cb51388cd93e.png)

**[服务台试点](http://www.helpdeskpilot.com/) :** 服务台软件，提供电子邮件管理、录音回复、票据管理、自定义字段、跟踪和报告。服务台试点可用于一次性许可费用。

![Helpdesk Pilot](../Images/7e48d58f0508b0b79560bd09092fbb3a.png)

**[帮助点](http://www.userscape.com/products/helpspot) :** 基于网络的帮助台软件允许您过滤、跟踪和组织支持请求；自定义 web 表单、知识库和论坛。并从您的移动设备访问/管理。HelpSpot 可以在你自己的服务器上运行，也可以托管，这两种方式都需要支付许可费。

![HelpSpot](../Images/c4869873ffb799b51ec611086931862a.png)

**[IssueTrak](http://www.issuetrak.com/) :** 客户服务软件，提供问题、日历和提醒、电子邮件通知、工作流和任务管理、基于网络的访问和定制选项的自动分配。IssueTrak 作为托管或租赁解决方案提供。

![IssueTrak](../Images/cb243fd27cce087e2b83bb0210e182ff.png)

 **[伽椰子](http://www.kayako.com/) :** 服务台软件，包括票务支持和电子邮件管理；聊天、访客参与和访客监控；能够管理多个知识库；和任务管理功能。通过托管账户或租赁选项，有三个模块可用。

![Kayako](../Images/ac4eeb9bf48e34aa69953724cc47d787.png)

**[努马拉赛道！](http://www.numarasoftware.com/Track-It/help_desk_software.aspx) :** 面向 IT 组织的帮助台软件，提供基于电子邮件的自助服务支持、可定制的门户网站、警报和通知以及报告选项。价格可应要求提供。

![Numara Track-It!](../Images/fc383e8a1a83eae8c0d6d39ec5f88c36.png)

**[准备](http://www.parature.com/) :** 按需客户服务软件，在线跟踪工作订单的状态，提供聊天功能和在线讨论板，包括票务系统和移动接入。价格可应要求提供。

![Parature](../Images/df757acc32a1a703d352a3201d2b8f26.png)

**[ServiceDesk Plus](http://www.manageengine.com/products/service-desk/index.html) :** 基于网络的帮助台工具，提供故障单、资产管理、采购订单管理、自助服务门户和丰富的知识库。ServiceDesk Plus 根据技术人员的数量收取许可费。

![ServiceDesk Plus](../Images/80d1b3f3f1f81ed23ea586560eb93e77.png)

 **[智能回答](http://www.smartanswer.com/) :** 基于 Web 的客户服务软件，包括可定制的联系表单、票据跟踪、公告板式设置和电子邮件通知。智能答案是每个运营商每月收费，这是麻烦票快递的付费版本。

![Smart Answer](../Images/1f362b694c43fec1709bceb8a155524a.png)

 **[Trouble Ticket Express](http://www.troubleticketexpress.com/):**一个基于 CGI 的开源故障单系统，包括信件跟踪、电子邮件提醒、可定制的布局、故障单监控和用于报告的数据导出。

![Trouble Ticket Express](../Images/bf412dd290d7db1790ebbfa92f6a8fa3.png)

 **[versaSRS 帮助台](http://www.versasrs.com/) :** 基于网络的帮助台解决方案，包括电子邮件支持、定制、可搜索知识库、报告、联系人管理和电子邮件模板。它有几个版本，有不同的许可选项。

![versaSRS](../Images/5d12715a7ff29121c2c90d7ea958a68f.png)

**[WonderDesk](http://www.wonderdesk.com/) :** 基于 Web 的客户支持，允许您跟踪和管理客户电子邮件，支持多种语言，并提供存档和历史跟踪。有两个包选项，都有一次性许可费。

![WonderDesk](../Images/effe84c1c672f9042fb00658e4976351.png)

**[Zendesk](http://www.zendesk.com/) :** 基于网络的客户支持服务台系统，提供电子邮件管理、可定制的客户门户以及与其他客户管理系统的集成。Zendesk 有三个计划，每个代理每月收费。

![Zendesk](../Images/9da89a2e7eae4bfda70d56d0d7950fa6.png)
