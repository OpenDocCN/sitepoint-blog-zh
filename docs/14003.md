# 马特·卡茨:用户界面乐趣:更好的片段

> 原文：<https://www.sitepoint.com/matt-cutts-ui-fun-better-snippets/>

谷歌员工马特·卡茨在
[UI 趣味:更好的片段](http://www.mattcutts.com/blog/ui-fun-better-snippets/ "Permanent link to UI fun: Better snippets")中描述了谷歌搜索结果的**新增强。卡茨先生是搜索引擎领域最受关注的人之一，很高兴看到他写博客。**

几天前，当我在谷歌上搜索的时候，我注意到了这一点，我只是说“嗯，这很酷！”然后今天当我从 SitePoint 收到一封关于我们新的基于 WordPress 的博客系统的邮件时，我知道我已经有了一个关于新系统的第一篇文章的主题。

感谢 Firefox 的 [Just Blog It](http://blog.warmbrain.com/justblogit/) 扩展，我可以立即写任何东西，而不用登录到 SitePoint 论坛，修改 BBCode，看到我的帖子看起来不对，回到论坛编辑，并试错一些可读的东西。

另一件我注意到的事情是，当我在谷歌上寻找博客的网址时……谷歌又打开了一些追踪链接——那个网站的链接(在谷歌上搜索“博客壁虎”之后)指向`http://www.google.com/url?sa=t&ct=res&cd=1&url=http%3A//blog.warmbrain.com/justblogit/&ei=eGwGQ82WNIW2Yba0yI4K`

这不是搜索引擎历史上最令人兴奋的消息，但现在我可以随时随地写博客了，你会发现质量会下降一点……但感谢 SitePoint 团队让 WordPress 开始运行。

很快，欢迎评论，啦啦啦。
