# 5 款超赞的 HTML5 游戏

> 原文:[https://www.sitepoint.com/5-awesome-html5-games/](https://www.sitepoint.com/5-awesome-html5-games/)

这周结束了，距离我上次看基于浏览器的游戏已经有一段时间了。因此，我为你的*研究*目的挑选了一个新的选择……在短短的几个月内，技术进步如此之快，真是令人惊讶。

下面的例子都是用 HTML5 技术实现的，比如`canvas`和 JavaScript。它们可以在大多数现代浏览器中工作，但大多数不能在 Internet Explorer 8.0 和更低版本中工作。Chrome 或 Opera 一般能提供最好的体验。

## [生物实验室灾难![Biolab Disaster](../Images/ec3a732c51f477690dc3434338c368fc.png)](http://playbiolab.com/)

虽然我太年轻了，不记得 80 年代的 8 位游戏，但 Biolab 是对 20-30 年前那些更简单但令人上瘾的游戏的敬意。这款游戏是一款以太空为主题的平台游戏，拥有丰富多彩的精灵、出色的动画和音效。开发者 Dominic Szablewski 也在 ImpactJS.com 发布了基于画布的库。

Biolab 是一款不错的游戏。事实上，它比许多 80 年代发行的更好… *所以我听说过*。

## [水晶星系![Crystal Galaxy](../Images/92cf9243c3113d616fff0a818250370b.png)](http://www.effectgames.com/effect/games/crystalgalaxy/)

水晶银河是一款让人想起 R-Type 的横向滚动太空射击游戏。然而，这个游戏的独特之处在于鼠标控制船的方向和位置。

不同寻常的是，Crystal Galaxy 没有使用 HTML5 canvas 元素。每个精灵都是在屏幕上移动的单独的 HTML `div`。它可以在所有浏览器中快速运行，甚至可以在 IE6 中运行！EffectGames.com T2 的作者也提供免费的工具和库来帮助你写你自己的书。

水晶银河很好玩，再来一次你就忍不住了。

## [帆布骑士![Canvas Rider](../Images/600baceb314db0dd448df4fd9d8e1af5.png)](http://canvasrider.com/)

正如您所料，Canvas Rider 是使用 HTML5 canvas 元素实现的 Line Rider 的一个版本。这个游戏很简单:你骑着自行车越过一系列障碍到达一个目标——但是现实的重力和物理使它成为一个严峻的挑战。

让 Canvas Rider 更特别的是在线赛道编辑器和游戏粉丝提交的[赛道](http://canvasrider.com/tracks/680331) [赛道](http://canvasrider.com/tracks/557438)的[千条](http://canvasrider.com/tracks/561613)。

## [经纪人 008 球![Agent 008 Ball](../Images/69338e83177f3b90af20c7f5a7a8be96.png)](http://agent8ball.com/)

代理 008 球是一个光荣的看起来像一个人的游戏。目标是在计时器结束前尽可能多的投球。

同样，代理 008 球使用 HTML5 画布元素，但它的照片逼真的图形和现实世界的物理使游戏脱颖而出。

## [海盗爱雏菊![Pirates Love Daisies](../Images/ae00596a07895e7f01b6f6daf7eca938.png)](http://www.pirateslovedaisies.com/)

《海盗之恋雏菊》是一款类似塔防的游戏，它将 HTML5 和 JavaScript 的可能性推向了新的极限。它是由 Flash 开发者 Grant Skinner 编写的，他受微软委托开发了一个展示 IE9 新技术的游戏。

这款游戏在大多数浏览器中运行良好，但正如你所料，它最好是在一台运行速度较快的 PC 上，使用带有本地硬件加速图形的浏览器。目前，IE9 测试版提供了最好的体验。

虽然浏览器永远不可能是最好的游戏平台，但这些游戏表明，一点点独创性就能产生惊人的效果。你最喜欢什么，有没有发现什么牛逼的 HTML5 游戏？
