# iPhoney:测试你的网站与 iPhone 的兼容性

> 原文:[https://www . site point . com/iphoney-test-your-site-for-iphone-compatibility/](https://www.sitepoint.com/iphoney-test-your-site-for-iphone-compatibility/)

买不起 iPhone(或者不住在美国)，但渴望查看您的网站在当前最热门的设备上的外观？

![Screenshot of sitepoint.com as viewed in the iPhoney application](../Images/c136fed1d2f580d04715632c3fd6d8ad.png)进入 [iPhoney](http://www.marketcircle.com/iphoney/) :一个小的(1.3MB)桌面应用程序，在新发布的设备上模拟网络浏览。

> iPhoney**不是一个 iPhone 模拟器**，而是为那些想要创建 320 乘 480(或 480 乘 320)的网站以供 iPhone 使用的网站开发者而设计的。它给了你一块画布，你可以在上面测试你的设计的视觉质量。

这个名字很聪明的应用程序对于开发者来说是一个很好的工具，他们想知道他们的网站是否能被前六天注册的一百万 iPhone 用户使用。有了这样的销量，可以想象，几年之内，在 iPhone 上浏览的访问者会占你网站访问者的很大比例。

当然，开发人员总是能够调整他们的窗口大小，以测试一个网站如何在较小的屏幕上呈现，但这个应用程序的美妙之处在于:

*   您可以指示它发送 iPhone 用户代理字符串，以防您正在检查的站点基于该字符串执行任何内容协商。
*   你可以指示它禁用 Flash 和 JavaScript 等插件，iPhone 尚不支持这些插件。
*   您可以快速旋转视口，就像在真实物体上一样。
*   调整浏览器大小并没有在屏幕上加载一个类似 iPhone 的东西有趣。

iPhoney 目前只能在 Mac 上使用，尽管考虑到最近发布的 Safari for Windows ，以及[代码已经开源的事实](http://www.marketcircle.com/blog/?p=37)，我认为 Windows 移植已经不远了。
