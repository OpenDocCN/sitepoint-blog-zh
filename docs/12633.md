# 五分之三的主要搜索引擎使用语义技术

> 原文：<https://www.sitepoint.com/3-out-of-5-major-search-engines-are-using-semantic-tech-what-is-your-search-engine-using/>

Ask.com 最近[加入了](https://blog.ask.com/2008/10/its-here-the-ne.html)(或者说重新加入——他们从自然语言搜索开始)日益壮大的主要搜索引擎社区，这些搜索引擎正在使用语义技术为他们的搜索结果增加额外的功能。

他们专注于让搜索更具情境意识。如果你正在搜索一个音乐组合(在这个例子中是 U2 ),向用户展示相关的东西，图像，维基百科条目等等。

![ask.com search results](img/667f6d052f781cd8bef818b5878a9e04.png)

这有点像雅虎。但是我还没有看到一个公共的 API，看起来他们也不允许人们发布他们自己的结构化数据(微格式、 [RDFa](https://www.w3.org/TR/xhtml-rdfa-primer/) 、 [eRDF](https://getsemantic.com/wiki/ERDF) 或 DataRSS)。当然，这是一个很大的缺口，但我确信这个缺口会很快被填补。

**有趣的趋势**

雅虎今年早些时候推出了 BOSS 和 SearchMonkey。微软不久前收购了 Powerset，这给了他们一个相当大的门槛。现在 Ask 回到了它的根源。

这一切都可以被视为在谷歌的游戏中击败谷歌的最后一搏，但我不这么认为。我不认为这是最后一搏。这是一个有意义的自然进程。

**物联网**

网络正在从“文件网”向“物联网”转变。谷歌统治着文档网，他们的 PageRank 和其他隐藏的秘密喜欢无结构的东西。但是一个由物体和连接构成的结构化网络是一个新游戏。

**不可阻挡的前进**

随着越来越多的网站开始发布嵌入式数据，越来越多的智能搜索引擎开始在线消费和重新组合这些数据，我们将看到搜索创新的复苏。更重要的是，在寻找！

哦，我已经有点厌倦谷歌了…你呢？
