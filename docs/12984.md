# 网络会议的新方向

> 原文:[https://www . site point . com/new-directions-in-web-conferences/](https://www.sitepoint.com/new-directions-in-web-conferences/)

接下来几周将举行几场澳大利亚会议(SitePoint 是其中的媒体合作伙伴)。Web Directions 的工作人员采用了他们传统的网络会议模式(在悉尼和温哥华取得了成功),并将其应用于更多的利基受众。

## Web 方向用户体验

[![](../Images/cd78e15f0603ab0e14ddb6be5b541cb1.png)T2】](http://ux08.webdirections.org/)

*   **内容:**一整天的研讨会，然后是一天的会议，重点是用户的网络体验
*   **地点:**澳大利亚墨尔本
*   **时间:**2008 年 5 月 15-16 日。
*   **谁:**一群大牌演讲者将出席，包括[安迪·巴德](http://ux08.webdirections.org/program/#post-11)以及 SitePoint 的定期撰稿人[卡梅隆·亚当斯](https://www.sitepoint.com/blog/)、[丽莎·赫罗德](https://www.sitepoint.com/blog/)和[马修·帕特森](https://www.sitepoint.com/blog/)。
*   **费用:**会议——595 美元；工作坊——550 美元。SitePoint 的读者有资格享受折扣；购买机票时，只需报出代码 **WDSP** ，即可享受 50 美元优惠。

[查看 UX 网络方向的完整计划](http://ux08.webdirections.org/program/)。

## 免费墨尔本活动

对于那些因各种原因无法参加 UX Web Directions 的墨尔本人来说，仍然可以在研讨会结束后的晚上参加由 Web 标准小组和 Web Directions 主办的活动，感受一下会议的气氛。

5 月 15 日，W3C 代表 Richard Ishida 和 José Manuel Alonso 将于晚上 7 点在墨尔本中央商务区的一家酒吧“墨尔本秩序”( Order of Melbourne)做一场免费演讲。

SitePoint 自豪地赞助这次活动，为所有与会者提供食物，还将有奖品。在 Web 标准组网站上回复。

*   **内容:**一整天的研讨会，然后是一天的会议，重点是用户的网络体验
*   **地点:**澳大利亚墨尔本
*   **时间:**2008 年 5 月 15 日@晚上 7 点。
*   谁: Richard Ishida (W3C 国际化负责人)和 José Manuel Alonso (W3C 电子政务负责人)

SitePoint 代表将出席上述所有活动，请务必前来打招呼！
