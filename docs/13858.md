# 来自 OSDC 的 Mozilla XForms 演示

以下内容转载自《科技时报》第 128 期。

上周，我很高兴在[开源开发者大会](http://www.osdc.com.au/)上谈论 Mozilla XForms。《T2 科技时报》和本博客的长期读者会记得我对最初规范的报道，我对 XForms 采用如此缓慢的原因的分析，以及最近我对 T4 如何编写跨浏览器 XForms 的看法。这次谈话是我探索 XForms 的下一个阶段，推动了 Mozilla 扩展的最新预览版的极限。

如果您有兴趣了解更多关于 XForms 的知识，您可以[听音频](https://www.sitepoint.com/examples/xforms-mozilla/xforms-mozilla.mp3) (MP3，8MB)和[看我的幻灯片](https://www.sitepoint.com/examples/xforms-mozilla/)和完整的笔记(包括几张我没有时间看的附加幻灯片！).

会议本身很有启发性，特别是考虑到它对 Perl 和 Python(我们在 SitePoint 这里不太关注的语言)的明显偏好。SitePointer 的同事 Matt Magain 在开源博客中发布了一篇精彩的会议总结。
