# 如果一个汉堡能让自己与众不同，那么你也能…

> 原文：<https://www.sitepoint.com/if-a-hamburger-can-set-itself-apart-so-can-you/>

这个周末，我的妻子、孩子和妹夫驱车 3 个半小时穿越佛罗里达州，只为了吃一个汉堡。

从技术上讲，我们也在参观一些景点，但这次旅行的催化剂是一个汉堡包。

根据《GQ》杂志和奥普拉·温弗瑞的报道，我们很幸运生活在一个拥有全国最好的汉堡的州。如果你感兴趣的话，那是在佛罗里达好莱坞一个叫 Le Tub 的地方。服务应该是糟糕的，但是汉堡应该是难以置信的。

你为什么在乎？这和网页设计开发有什么关系？

很简单:如果一个汉堡店能让人们(我的家人并不孤单)花 3 个半小时去买它的产品，那么网站设计者和开发者就能想出办法让他们的服务与众不同，同样具有诱惑力。

怎么会？你必须变得不只是一件商品。你必须找到，然后沟通，让你的服务与众不同的方法。你必须到达金字塔的顶端。你必须成为“专家”,如果你愿意，也可以成为大师。

奇怪的是，这样做并没有绝对的科学依据。你和我一样清楚，Le Tub 可能不提供世界上最好的汉堡。但是奥普拉和 GQ 杂志有巨大的可信度。正如幽默报纸《洋葱报》最近指出的，数百万人每天都急切地等待着听到奥普拉的最新指示和命令。(顺便说一句，如果你还没有访问过洋葱网，把这个网站标记为书签:http://www.theonion.com 或者订阅一份)。

你可以通过让你所在市场的意见领袖对你赞不绝口来建立类似的信誉(尽管规模可能没那么大)。你可以在受人尊敬的出版物上写文章。可以收集推荐书。你可以通过展示你专业知识的方式让别人看到你。

一旦你到达金字塔的顶端，神奇的事情就会发生。好莱坞的演员、体育明星和《财富》500 强的首席执行官都拿到了超高的薪酬，因为他们已经到达了金字塔的顶端。他们并不比我们更努力；有些人可能工作得更少。但是他们已经达成了“共识”,他们是特别的。在某些情况下，如在体育运动中，这种协议纯粹基于表现。但在其他情况下，尤其是在营销/商业领域，达成协议的方式更加模糊和主观。

在职业中，这种“协议”部分基于物质，部分基于沟通和营销。

所以你现在就可以通过适当的营销来建立你的信誉。

你也可以成为世界上最好的汉堡。

几天后见，我的肚子又胖了几磅！
