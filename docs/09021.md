# WordPress 的 10 个最佳 jQuery 插件

 **## 1.翻书

是一个完全集成的图片库插件，具有翻页效果。允许您通过专门设计的界面直接从管理员面板创建具有翻页效果的 flash books。

## 2.jQuery-页面栏

这个插件将 jQuery 滑动条显示为一个现代的页面栏，而不是 next 和 prev 链接。因此很容易浏览许多页面。

 [![jQuery-Pagebar](../Images/b56cf69d433a768a733b1bf70246ea34.png)](http://wordpress.org/extend/plugins/jquery-pagebar/) 
[源+演示](http://wordpress.org/extend/plugins/jquery-pagebar/)

[源+演示](http://wordpress.org/extend/plugins/advanced-spoiler/screenshots/)
