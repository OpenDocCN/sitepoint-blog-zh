# 免费做志愿者——这是有回报的。

> 原文：<https://www.sitepoint.com/volunteer-for-free-it-pays/>

在过去的 7 年里，我曾无数次思考，如果我能在 SitePoint 论坛上获得报酬，生活会有多美好。因此，在过去的几周里，我看到一些人在帖子或推特上说他们多么希望为 SitePoint 工作，这并不令我感到惊讶。一个人甚至称之为他的梦想。那很酷。因此，我想分享一下我自己的故事。

几年前，当我作为一名大型公司的. NET 开发人员工作时，我在谷歌搜索一段代码后偶然发现了这个论坛。一个小时后，我发现了 SitePoint 错误。在论坛上闲逛不仅仅是学习。这是关于沉浸在一个充满有伟大想法的人的环境中！像我一样热爱代码的人！知道成为极客很酷的人！在打着学习的幌子在论坛上浪费了一年左右的公司时间后，论坛的工作人员邀请我加入他们的行列，成为他们的导师。在由于时间限制而被迫辞职(以双胞胎的形式)之前，我作为一名志愿者主持人工作了几年，并逐步晋升到团队领导的角色。

![](img/c7269503e296fb715ecc0fd2c14016e8.png "sitepoint-forums")

离开这个地方很难，但公平地说，我有很多东西让我分心。6 个月后，当我收到 SPHQ 的一封邮件，表示愿意付钱让我回来时，我非常惊讶！我现在有了一份全职妈妈的理想工作。一周 20 小时不在家做我喜欢的事情。我终于可以在论坛上混了。

所以现在我打赌你在想你是怎么进入这个导师的工作的。嗯，其实很简单。你在论坛上闲逛，发表非常棒的帖子。你甚至不需要做很多——你只需要在点击新的发布按钮之前想一想。请放心，我们会注意你的。不过，这并不全是好事，也有不好的一面——我是一个暴君式的老板。

我今天在这个话题上写博客有一个不可告人的动机——这个月我会尽我所能让你去做。如果为 SitePoint 工作的想法不适合你，也许一些免费的 SitePoint 书籍会适合你。

为了彻底改变话题，我上周去了办公室，我们的最新项目——[JavaScript Live 正在酝酿着一大堆激动人心的事情！如果学习 JavaScript 在你的愿望清单上，并且你的口袋里有 9.95 美元，那么你不能错过这个机会。如果你够快的话，你还有时间参加我们最新的 Twit-away，](https://www.sitepoint.com/courses/javascript-live)[为你自己赢得一张免费门票](https://www.sitepoint.com/forums/showthread.php?t=672069)。
