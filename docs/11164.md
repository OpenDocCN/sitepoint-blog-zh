# 收件箱零:不要相信炒作

> 原文:[https://www.sitepoint.com/inbox-zero-myth/](https://www.sitepoint.com/inbox-zero-myth/)

今天是星期五，当我们结束一周的工作，并试图让自己相信我们真的控制了我们的电子邮件，而不是相反，收件箱零索赔到处都在涌现。什么是“零收件箱”，为什么它是如此令人垂涎的地位，它在今天以电子邮件为中心的世界里现实吗？

## 什么是零收件箱

零收件箱是一个用来描述你的空收件箱的术语。这意味着你在电子邮件游戏的顶端；你负责接收信息；你真棒。生产力网站 [43 Folders](http://www.43folders.com/) 的梅林·曼是这一现象背后的智囊。他讲述了如何实现和维护一个空收件箱，并提供了大量关于[管理电子邮件的信息。](http://www.43folders.com/43-folders-series-inbox-zero)

像大多数人一样，我想要一个空的收件箱。这比你打开电子邮件时收到数百条恳求你注意的信息压力要小；在电子邮件领域拥有完全的控制权是值得的，也是授权的。但我也知道零收件箱是不可持续或不现实的，除非你想成为收件箱的奴隶。

## 为什么它不起作用

我是一个非常(好吧，非常)系统和有条理的人，但是我很少做到零收件箱。电子邮件的本质——由我们通常无法预料的其他人发起的即时通信——意味着它永远不会停止。

你可以制定一百条规则和过滤器，在邮件到达收件箱时对其进行分类、移动和删除，但你永远无法捕捉到所有信息。除非您将所有收到的邮件都发送到收件箱以外的文件夹，否则您将永远无法维护一个空的收件箱。然后你只是移动，分开和分散混乱，给自己一种虚假的安全感，一种对电子邮件的模拟控制。

收件箱零是一个神话；专注于实现它是浪费时间。好了，我说了。当然，你可能会在周五下午清空收件箱，但在周一早上会是什么样子，在下一次攻击前的几分钟内，你会花费多少时间和精力来归零？你可以说我疯了，但这不是在浪费我的时间。

## 我们从中学到了什么

尽管我批评收件箱零哲学，但我确实认为从中可以学到一些有价值的经验。当你抛开字面上的应用，inbox zero 提供了一些高质量的邮件管理技巧。像开发一个持续的过程来管理输入信息的重要性；对不同信息的价值进行适当的优先排序和权衡；成为自由删除者的需要。

如果你通读他的帖子，你会发现梅林明白了。是大众陷入了一个陷阱，认为一个空的收件箱等于超级生产力和防弹组织。我希望是这样，但事情没那么简单。

## 摆脱电子邮件的魔咒

那么这给我们留下了什么？我们仍然需要一些方法来有效地管理我们的电子邮件，并且花费更少的时间。也许没有一个普遍的答案，但有一些可能性。继续读下去，探索它们。
