# 呵呵，又一个谷歌专利…

> 原文：<https://www.sitepoint.com/ho-hum-another-google-patent/>

不，今天没有新的了，抱歉。其实这个专利，对于[基于历史数据](http://appft1.uspto.gov/netacgi/nph-Parser?Sect1=PTO2&Sect2=HITOFF&p=1&u=/netahtml/PTO/search-bool.html&r=1&f=G&l=50&co1=AND&d=PG01&s1=20050071741&OS=20050071741&RS=20050071741)的信息检索，早就讨论死了。一个很好的例子可以在 [SEOmoz](http://www.socengine.com/seo/guide/google-historical-data-patent.html) 找到。

那么，为什么我自己没有把它写死呢？因为它没有告诉我们谷歌今天在做什么，也很少告诉我们他们未来可能会做什么。阅读 SEOmoz 的讨论和专利，您会发现专利几乎涵盖了收集和使用历史数据的所有可能方式。

如果谷歌决定奖励那些很快获得大量反向链接的新网站，因为伟大的新资源往往会很快获得大量链接，这将受到这项专利的保护。如果谷歌决定惩罚那些匆忙获取大量反向链接的新网站，因为这也是垃圾链接发送者所做的，那么这将会受到这项专利的保护。

因此，他们可能查看的任何历史数据都是如此……问题是从“坏人”案例中辨别出“好人”案例。尽管谷歌的决策者可能不知道他们想做什么，但他们还是申请了专利，因为当你付钱给数百人帮你发明东西时，这就是你要做的。

总有一天，如果我真的想让一些老朋友成为敌人，我会接受一些关于“谷歌沙盒”的完全胡说八道。

显而易见，新网站不再像过去那样受到谷歌的信任，临时链接(链接租赁、RSS、新闻稿)不再像过去那样有影响力，在新链接中过多使用相同的锚文本(通常的“文本广告”模式，也就是链接租赁)会对页面排名产生负面影响。

到目前为止，我只能看到这些。一些网站会因为过度使用垃圾链接而受到长期处罚吗？他们当然是——但这不是新闻，这是谷歌从第一天开始的运作方式。

与几年前相比，现在有更多的网站遇到这种情况吗？当然，因为越来越多的网站所有者被链接租赁的炒作和诱惑所迷惑，而谷歌的回应是在过滤方面变得更加激进。越来越多的人抱怨处罚，因为越来越多的人发送垃圾邮件，越来越多无辜的人被旨在捕捉垃圾邮件发送者的过滤器所捕获。

当我听说一个网站在 MSN 和雅虎上排名第一，过去在谷歌上排名第一(但现在已经找不到了)，我不需要眯着眼睛看专利就知道他们的手被拉里·佩奇的饼干罐抓住了。到目前为止，有很多关于如何从这样的事情中恢复的“理论”，但我仍然怀疑它们是否有效。谷歌没有理由让一个垃圾链接发送者重获青睐，即使他们想这么做，也有很多技术障碍。

所以，说了这么多，还是会有人问我“我应该怎么做才能避免被沙盒化？”如何自然地建立链接，通过不断地从链接到你的网页的其他网页获取链接？换句话说，继续努力从相关网站获得高质量的链接，达到你的目标受众——这是一项有价值的活动，即使明天主要的搜索引擎都转向 100%的点击付费，它也会为你带来回报。

就一分钟，忘记谷歌，忘记专利，忘记逆向工程。寻找链接到像你这样的网站的好网站。给他们一个链接到你网站的理由。重复一遍。重复一遍。重复一遍。
