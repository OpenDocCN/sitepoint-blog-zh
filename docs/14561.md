# 向东走，年轻人！

> 原文:[https://www.sitepoint.com/head-east-young-man/](https://www.sitepoint.com/head-east-young-man/)

所以我在佐治亚州亚特兰大的一家设计公司找到了一份工作。这是一件好事，因为它将为我提供更多的时间来玩 ColdFusion，并在这里传递我的知识。

但这也是一件坏事，因为这意味着我必须在 22 号之前从亚利桑那州搬到乔治亚州。我目前正在打包我的家人、我们的房子和三只猫，准备进行一次越野旅行，这次旅行肯定会比得上任何一部《归途》电影。

所以在接下来的一周左右，我的帖子会很少，因为我要搬家、过渡、安顿下来，当然还有重建互联网接入。

与此同时，请带着您的问题、想法和更多的想法继续访问论坛。他们最近真的很忙，我很高兴看到每个人(包括我自己)的成长以及分享的意愿。

例如，我们自己的 David Medlock，他让这个 CF 博客运行起来，最近发布了一个非常[快速和肮脏的目录浏览器脚本](https://www.sitepoint.com/forums/showthread.php?t=173328)。就我个人而言，我喜欢它，当我处于困境时，我到处都在使用它。

正是这种分享让 PHP 社区变得如此强大，这也是我一直觉得 CF 社区缺乏连续性的原因之一。看起来我们正在改变这一点，从这里开始！

**P.s.** 如果我不在的时候，你有什么想分享的，只要用联系表格发给我就行了。如果我喜欢我所看到的，并且觉得社区的其他人也会喜欢它，我会把它作为一个客座贡献者发布到这个博客上！
