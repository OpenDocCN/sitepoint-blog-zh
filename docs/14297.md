# 如果我有一把锤子，我会在早上锤…

> 原文:[https://www . site point . com/if-I-had-a-hammer-id-hammer-in-the-morning/](https://www.sitepoint.com/if-i-had-a-hammer-id-hammer-in-the-morning/)

有了这个博客作为我好奇心的催化剂，**我进入了 Java 开发的许多新领域，最近**——并不都是基于网络的。虽然同时有这么多球在空中肯定会有压力，但也很有趣。

首先，我有一部支持 MIDP 2.0 的新手机(那是一部手机)，所以**我花了一些时间研究在移动 Java 平台上有什么可能。**该版本对 API 的细微改进使得开发一些常见的程序类型——如游戏——变得更加容易！我已经在找借口给手机写程序了。

与此同时，我发现了 [YourKit](http://www.yourkit.com/) ，这是一个 Java 剖析工具，能够**跟踪和分析任何 Java 程序的 CPU 和内存使用情况，包括在浏览器中运行的 applet。**这个工具让我有可能大幅提高 SitePoint 的内容管理系统富文本编辑器 [Editize](http://www.editize.com/) 的性能(预计很快会有新的版本)。我之前使用的工具 Optimizeit Profiler ，在网络浏览器中根本无法工作。

他们说如果你只有一把锤子，那么所有的东西看起来都像钉子。尽管意识到这个陷阱当然很重要，但拥有一个像锤子——呃，像 Java 这样多功能的工具也是非常值得的。Java 打开了 Web 开发之外的许多大门。
