# 可怜的理查德的网站

> 原文：<https://www.sitepoint.com/poor-richards-web-site/>

**![Poor Richards Web site](img/0fbf247cef2cfc2c3fb9cfd43a24eba3.png)**

有数百本书涵盖了 HTML、JavaScript、Java 和市场营销，但很少有人把它们放在一起。这就是可怜的理查德的网站的由来。这本书提供了“关于建立一个低成本网站的没有极客的常识性建议”。畅销书作家彼得·肯特带你了解在互联网上建立一个网站并推广它的整个过程。

这本书的第一部分包括准备。这是我读过的唯一一本真正提出“你真的需要一个网站吗？”并通过提供一个全面的列表来回答这个问题，这个列表列出了你为什么想把一个网站放到网上的原因，以及在哪些情况下你可能不需要一个网站。第二章讲述了在开始创建网站之前你需要做的事情，为互联网/计算机新手提供了一个良好的起点。接下来的几章涵盖了虚拟主机和域名问题，包括选择虚拟主机公司的广泛指南。在本书的第一部分，你还将学习如何建立一个时事通讯，接受在线支付，以及通过 CGI 脚本在你的网站上添加互动。接受支付的广泛覆盖将证明对那些想开始网上销售的人非常有用。

本书的第二部分涵盖了网站的设计，包括 HTML 的快速概述、设置自动回复、选择创作工具和创建有效的网站。你将学习如何带领你的访问者浏览你的网站，如何让他们方便地四处走动，以及如何避免压倒你网站的访问者等等。我特别喜欢“认为有用而不酷”的建议，因为这是互联网上许多人犯的错误，也是大多数网站设计书籍传播的错误。

第三部分提供了一个网站推广的广泛概述。您将了解如何在互联网上调查对您的网站感兴趣的潜在客户的位置，了解搜索引擎和目录、新闻发布、真实世界公关、跟踪流量等等。这本书最后有两个清单，一个涉及虚拟主机，第二个是关于推广。

尽管这本书 99%的信息都非常有价值，但有几件事我不同意，从长远来看，这可能会给你带来麻烦。例如，彼得·肯特建议你将图形中的颜色数量限制在 16 种，这是完全荒谬的。他还指出，你不需要学习 HTML 来创建一个网站，这是可能的，尽管随着时间的推移会给你带来问题。搜索引擎的一些建议也有点过时了，尤其是关于让你的网站在雅虎上市的信息。

但我在这里非常挑剔。总的来说，这本书对所有网站管理员来说都是非常有价值的信息和想法来源。无论是新手和专家互联网用户和网站管理员一样，会发现这是一个信息丰富，令人愉快的书。无论你经营一个网站是出于爱好还是为了盈利，这本书都是每个人必读的。

这本书的第二版已经发行，其中包括更新的链接和信息。仍然是必读的。

**评级:5/5**
