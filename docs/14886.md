# 广告技术把用户放在第二位

> 原文:[https://www.sitepoint.com/put-users-second/](https://www.sitepoint.com/put-users-second/)

俄罗斯是一个拥有大量未开发的互联网技术创新人才的国家。本文将聚焦于当今俄罗斯互联网领域最强劲的趋势之一——在线广告。

##### 广告商的挑战

正如对俄罗斯网络市场的众多统计研究所表明的，在线广告的影响力正在持续下降。这迫使大多数互联网代理公司寻找新的方法来影响他们的目标受众。

对于今天的开发人员来说，宣布一项真正的技术创新似乎很难，因为技术的标准化与各种各样的障碍和不可预测的障碍结合在一起，例如选择和研究可用的软件等。对于 Web 开发组织来说，使一项发明成为一个普遍的标准还需要他们克服一些与浏览器制造商的忠诚度、市场需求等相关的严重问题。因此，将任何现有的、被广泛接受的技术作为起点，添加一些收尾功能，并宣称它是一种全新的技术要容易得多。

这确实是两家俄罗斯互联网公司所做的。两家都是专门从事网络广告互动解决方案开发的公司: [AdVerso 公司](http://www.adverso.ru)和 [IMHO 视频国际广告集团](http://www.imho.ru)。第一个推广的是一种叫做 AdVerso 的技术，而第二个展示的是 ScreenGlide 商标下的技术解决方案系列。

##### 广告技术

我们要看的第一项新广告技术是 [AdVerso](http://www.adverso.ru) 。Adverso 的主要特点是有机会在网站的基本内容上展示在线广告。

使用 AdVerso 的网站的访问者可能会看到以下任何内容(参见他们网站上[示例页面的示例):](http://www.adverso.ru/examples/)

*   **Short introduction splashes (“Teasers”)**
    The ad is demonstrated over the Website’s contents.
*   **Interactive application launching (Macromedia Flash-based)**
    Some user manipulations can be achieved without closing the main browser window (mail list subscriptions, service or product orders, etc.)
*   **广告关闭/最小化(“提醒”)**
    按下出现在活动广告区域内的“关闭”按钮，使得交互式应用窗口关闭，或者将广告最小化到浏览器窗口的角落。

AdVerso 对其技术的优势解释如下:

1.  The basic Website contents load first. After a page has completely loaded, the interactive animated banner (or application) starts. Moreover, the advertisement loading process is not perceptible to the user as they move around the Website.
2.  If the user is not interested in the ad, they can close it immediately.
3.  Adverso automatically checks the user’s computer configuration, so that it can provide the version of an ad application that’s appropriate to that particular system (including browser settings, operating system, plugins installed, etc.).
4.  The advertiser has total control of the advertisement window’s activity, and can activate an automatic switch to “waiting mode” until the window is made active by the user.
5.  Adverso offers all possible program-based ad combinations, incorporating the “Teaser”, “Reminder”, and interactive application functionality, as well as the option for advertisers to use separate modules independently.
6.  Adverso 提供了很棒的互动设施。用户可以填写问卷、购买产品、订阅讨论列表、进行搜索查询等等，而无需离开显示基于广告的网站。

##### 屏幕滑动技术

那么，竞争技术 ScreenGlide 怎么样呢？ [ScreenGlide](http://www.imho.ru/eng/tech/screenglide) 技术为广告发布者提供了展示拥有强大互动能力的宽幅多媒体广告的绝佳机会。

ScreenGlide 和 AdVerso 一样，也是基于 Macromedia Flash 技术，分为以下几个模块(您可以在 [ScreenGlide 广告廊](http://www.imho.ru/eng/tech/rmsamples)试看广告格式):

*   **ScreenGlide**
    When they move their cursor over a ScreenGlide-based ad banner, the user is presented with an ad window that displays the advertiser’s interactive presentation.
*   **BackScreen**
    The advertisement is loaded in a blank window over the publishers’ Website.
*   **FullScreen**
    A full screen ad is opened over the Website’s contents.
*   **VideoScreen**
    A new window opens, providing high quality graphics and streaming audio. The advertisement doesn’t start loading until the basic Website’s contents are fully loaded. As an additional feature, the ad presentation can be programmed to start after the user clicks to move to another page.
*   **飞行屏幕**
    许多广告块被显示给用户，按照广告商的指定在网站内容上移动。

这些据称是这些技术最吸引人的特征。但也许我们应该忘记营销炒作。让我们从不同的角度来看这些解决方案:从每个产品的开发者和用户的角度。

##### 从开发商的角度看

毫不奇怪，上述两种广告技术的开发者将他们的解决方案描述为网络广告的“革命性在线营销工具”。AdVerso 的开发者声称，“AdVerso 是万维网俄罗斯部分的一项独特技术，它提供了无限的创作可能性，并允许广告商创建用户不能错过的高效广告。”不幸的是，该公司没有适当分析用户对此类广告的可能反应。

ScreenGlide 技术的开发者在用户体验研究方面采取了一些措施，并试图预测人们对通过他们的 ScreenGlide 解决方案系列制作的广告的反应。在他们看来，背屏广告解决方案引起的刺激效果最小，而用户在观看全屏广告时会产生极大的不满。飞行屏幕也可能会激怒用户。

当然，精确的用户体验预测是不可能的——你永远也猜不到当用户接触到像这样的富媒体广告后，他们会想些什么。然而，这家公司对可能的负面用户反应的理解和承认值得称赞。看起来就像是开发者警告潜在的广告客户不同的用户反馈:它可能是好的，但这并不能真正得到保证。

##### 通过用户的眼睛

现在，我们了解了这些技术，我们知道 AdVerso 和 ScreenGlide 的开发者都称赞这些解决方案提供的“好处”。但是每天面对这两种技术的网络用户会说些什么呢？

事实上，在实践中，俄罗斯互联网基础设施中提到的技术集成经常与最初的概念不同。这样做的主要原因是通过网络冲浪的用户被这些大的、侵扰性的广告激怒了。一些最令人恼火的因素包括:

*   **Speed of loading**
    It’s very important for today’s Web users that Websites load quickly. The AdVerso and ScreenGlide technologies are developed so that several application modules built into the advertisement are preloaded while the page opens. These include various client-side analyzers (such as AdVerso’s ClientLookup module, which checks for browser settings, that the necessary plugins are installed, etc.), statistics counting utilities (such as AdVerso’s StatAnalyzer module) and more. All that processing could have a significant, negative impact on the loading speed of the ads and the host site itself.
*   **Unwanted diversion**
    Most Web users are task-oriented. They visit Websites seeking certain data or services; they want to read and compose email messages, chat with pals, take part in Webconferences and forums, download freeware, scrutinize news headlines, etc. How astonished and indignant the user would be to instead be faced by a large presentation that diverted his attention from the task… just to heed advertising slogans!
*   如今，大多数互动广告都是这样设计的,“关闭”按钮(或者只是一个类似 MS Windows 的十字图标)在广告本身的背景下是完全看不见的。结果，用户发现关闭广告几乎是不可能的。很明显，广告商想要尽可能长时间地吸引用户的注意力，但这种动机有时会让用户紧张和沮丧，因为他们试图努力关闭令人厌烦的窗口，但却徒劳无功。

##### 摘要

这些全新技术的发展预示着竞争激烈的网络广告业的进步。然而，像 AdVerso 和 ScreenGlide 这样的新广告解决方案的创建不应该忽视那些创新真正针对的人的需求和偏好，即网络用户。
