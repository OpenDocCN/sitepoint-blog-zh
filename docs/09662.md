# 从自由职业者开始

> 原文:[https://www.sitepoint.com/make-that-move-from-freelancing/](https://www.sitepoint.com/make-that-move-from-freelancing/)

我最近和一个决定离开自由职业的人聊了聊，她在一家不提供服务的公司找了一份全职工作。

正如你所想象的，她的客户被她粘住了，然而她开始发现很难同时扮演这两个角色。我给了她一些建议，她照着做了，对结果很满意，并想与你分享。

在决定就业之前，我建议你去找当地那些你认识并信任的人。如果你不认识他们，那也没关系；但重要的是，你要找到几个你认识的做得很好的人。

一旦你列出了三到五个名字，联系他们，安排一次咖啡约会。然后，在这个会议上，解释你的情况，你想确保你的客户在未来得到照顾。

一旦你面试了这些人，对他们服务现有客户的能力有了良好的感觉，不要羞于要求回报。你带来了你现有的客户和工作——这一定很有价值，对吧？

最简单的解决办法是商定一个百分比，在一段时间内，他们每从你的客户那里得到一份新工作，你将从中获得一定比例的佣金(我的建议？六个月)。

理想的情况是，做一些这样的安排，然后和你的客户沟通。让他们知道你喜欢和他们一起工作，但是你已经决定退出这个行业，这里有几个你可以推荐的人的名字。

也不要只通过电子邮件来做——除了一些时间，打电话花费很少，并且确保你的客户理解你的立场，并且你仍然有一个很好的关系走向未来。

最终的结果是一群快乐的客户，你知道他们会被照顾到未来，也是接下来几个月的被动收入流，因为他们继续你开始的伟大工作。

祝你在新的岗位上好运，并祝贺你成功退出自由职业生涯！
