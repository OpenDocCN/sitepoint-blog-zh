# 纽约公共图书馆标志:字里行间

> 原文:[https://www . site point . com/the-new-York-public-library-logo-between-the-line/](https://www.sitepoint.com/the-new-york-public-library-logo-between-the-lines/)

纽约公共图书馆新标识的发布在网上引起了不小的轰动。

![Old Logo](../Images/268912823845596a111d811fcb746e1e.png) ![New York Public Library Logo](../Images/afefd8b3f5887e903108c55603e18eaf.png)
*旧公共图书馆标志(左)和新。*

重新设计始于 2008 年，设计师们使用了图书馆台阶上狮子雕像的照片和草图。楼外的狮子以“刚毅”和“忍耐”著称。据报道，影响包括日本木刻，彩色玻璃和新艺术。矢量风格的作品被设计成足够多才多艺，可用于多种格式和颜色，以及在小尺寸下复制良好。

新的标志来了，就像通常标志重新设计的情况一样，给图书馆带来了新的使命。他们想传达一种更加数字友好和现代的感觉。

![LogoWith Text](../Images/993beb4718b7f31a367ce09d979fad30.png)

作为一个狮子座的人，同时也是一家公司的老板，我不得不承认我特别喜欢这个。虽然我还在和自己辩论是否喜欢这个 logo。我喜欢这种简单和清晰的线条，但是对于如此重要的知识守护者来说，这是不是有点太卡通了？虽然我喜欢无衬线字体，但我对文本的位置不太感兴趣。这种字体叫做 [Kievit。](http://typophile.com/node/16448)

有些人评论说它看起来像米高梅的标志。我认为他们指的是一个非常短暂的米高梅标志，而不是众所周知的经典。蓝色和金色的米高梅标志在 1968 年被用于少数电影和该公司的录音部门，但后来被我们所知道和喜爱的咆哮的狮子所取代。

![MGM_logo_1968](../Images/5cc2407fcc0fc4d5c1636e7ac9c757d1.png) ![mgm-logo](../Images/475e81ce1b72aec0c19285d76b03e6be.png)

这里有一段来自纽约公共图书馆的有趣视频，概述了这个标志的由来。

在 [Brand New](http://www.underconsideration.com/brandnew/archives/an_iconic_lion_for_an_iconic_institution.php) 和 [New York Times](http://cityroom.blogs.nytimes.com/2009/11/09/a-new-look-for-the-public-librarys-lion-logo/) 网站上查看关于这个标志的其他文章，它们都展示了这个标志的最初草图和想法。

你觉得新标志怎么样？这是一种改进还是对经典的更多干预？
