# 我最喜欢的标签

> 原文：<https://www.sitepoint.com/my-favorite-tags/>

卡梅隆·柴尔德里斯最近[表白了他对 ColdFusion 列表的爱](http://www.sumoc.com/blog/index.cfm?mode=entry&entry=06024C8F-5004-2066-B735F6AE1CBD7811)，这让我开始思考。
我开始想知道我最喜欢的标签是什么，哪些标签是我认为最有趣或最酷的？因此，这里是我的十大最酷、最有趣或者只是普通的伟大 ColdFusion 标签:
[list = 1]
[*][CFSaveContent](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-a29.htm#wp1844410)–我喜欢在一个地方创建一些输出，将其很好地包装在 CFSaveContent 标签中，然后在其他地方使用它。这个标签让我不必多次进行多重查询。
[*][cf output query = " my query " group = " my group "](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-b12.htm#wp1101659)–是的，我知道这几乎是必然的，因为如果没有这个标签，ColdFusion 会是什么样子呢？我喜欢将它与查询和组属性结合起来，这使得生成一个包含组的表比嵌套循环容易得多。
[*][cf loop](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-p76.htm#wp3301338)–任何不认为这是执行循环的最佳方式的人在我看来都是疯了……再也不用在 for 循环、while、do while 等之间做出决定了。
[*][CFLocation](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-p70.htm#wp1100753)–我喜欢它，不属于这里，或者需要去别的地方，然后 cf location！
[*][CFDump](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-p21.htm#wp1734655)——正如我办公室的一个家伙所说的“当有疑问时，就把它扔掉”，老实说，我真不知道没有 cf dump 人们是怎么活下来的。
[*][cf chart](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-a11.htm#wp2619630)–在观看一集《辛普森一家》的时间内构建一个 3D 钻取图表？当然我会花一美元买那个！我们在办公室里疯狂地切换。忘记巨大的 IF/Else 语句，切换它宝贝！我喜欢制作邮件脚本，有了这个标签就简单多了！
[*][cf module](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-pb4.htm#wp1810047)/[cf object](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-pb5.htm#wp1810139)–我不知道为什么，但我就是喜欢这样调用我的 CFC 和自定义标签。
[*][cf try](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-c18.htm#wp1104557)/[cf catch](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-a10.htm#wp3395038)——我在 java 类中学习了 try/catch，对我来说，这是防止和捕获错误的最酷的方法。
[/list]正如我的荣誉提名是:
[list = 1]
[*][cf Include](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-p63.htm#wp1100248)–我是一个 include 怪胎，我对代码的小块和 monster 1500 行文件很感兴趣。
[*][cf Ldap](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-p69.htm#wp1100581)–如果你曾经与 LDAP 服务器打过交道，你就会知道这种痛苦，如果你有 CFLdap 标签，你就会知道如何减轻这种痛苦。
[*][cf Xml](http://livedocs.macromedia.com/coldfusion/6.1/htmldocs/tags-b21.htm#wp2497244)——只是一个用于创建优秀 XML 文档和对象的优秀标签。那么你有什么选择呢？
