# Xara 网页设计师 6:评论

> 原文:[https://www.sitepoint.com/xara-web-designer-6-the-review/](https://www.sitepoint.com/xara-web-designer-6-the-review/)

Xara Web Designer 6 是一个所见即所得的网站设计器。要绝对明确的是，这不是一篇付费评论，我或 SitePoint 与 Xara、他们的合作伙伴、员工、亲戚等也没有任何关系。

请注意，Web Designer 6 不是 Dreamweaver 或 Visual Studio 的替代品。这是一个纯粹的视觉设计应用程序——在导出之前，你不能添加自己的代码或检查 HTML 稍后会详细介绍。

Web Designer 6 适用于 Windows XP、Vista 和 7。如果你运行的是旧的硬件，你会很高兴听到要求很低:只需要 256MB 的内存和 150MB 的磁盘空间。但是，许多模板都包含在单独的下载中，所以如果使用它们，需要一点额外的磁盘空间。与一些打破钱包的竞争相比，这个价格也是合理的: [Web Designer 6 的价格为 49/39/39 欧元](http://bit.ly/cqMnnW)。

本质上，网页设计师 6 包含了 Xara Photo Graphic Designer T1 的许多功能，之前名为 Xara Xtreme(参见 SitePoint 上的[我的开发者评论](https://www.sitepoint.com/xara-xtreme-5-developer-review/)和 [Jen 的设计师评论](https://www.sitepoint.com/xara-xtreme-pro-for-designers/))。界面和功能集几乎相同，但更强调网络发布。Web Designer 6 可能更适合你的需求，而且它的成本是同类产品的一半。

安装是没有痛苦的，幸运的是，Web Designer 不会将自己注册为每个 Web 相关文件的默认应用程序。你很快就会看到新的深色风格界面:

**Xara 网页设计师 6 界面—点击放大**

微软已经为 Expression 和 Visual Studio 采用了深色界面主题。它们无处不在，尽管 Xara Web Designer 很大程度上是装饰性的，但看起来要好得多。使用过另一个 Xara 产品的人会很快熟悉它的布局和工具。与 Adobe 或微软产品相比，它过度依赖弹出窗口，界面感觉有点笨拙，但没有什么根本性的错误，许多 Xara 用户似乎喜欢它。

作为一个图形设计包，Xara Web Designer 几乎没有缺点，你很少需要另一个应用程序:

*   主要的矢量编辑工具非常好，速度极快。即使是新手用户也可以轻松操作贝塞尔曲线。
*   位图可以缩放、裁剪、增强或消除红眼。
*   它提供了简单而有效的 Flash 制作，包括关键帧之间的移动、缩放、旋转、倾斜和重新着色等变换。
*   移动对象时，文本会在对象周围重新流动。
*   可以通过切换到其他层来添加翻转效果。
*   生成的页面或图像可以导出为 PDF、PSD 和许多图形格式。

不幸的是，web 开发方面似乎被附加到了主 <acronym title="DeskTop Publishing">DTP</acronym> 应用程序上。创建页面很容易，但你只能开发固定宽度的设计，并使用基于 IE 的内部网络浏览器进行预览。

有一个有用的设计库，允许你导入主题或小部件，如 Wufoo 表单、谷歌地图、PayPal 按钮、YouTube 视频、Flickr 照片、Twitter 更新和 RSS 订阅。但是，如果您需要的小部件不可用，您就无法自己插入。该应用程序允许您在设计视图中调整小部件的大小，但这可能会导致一些问题，直到您预览或导出站点时这些问题才会显现出来。

您完成的网站可以导出为 HTML 格式。结果令人印象深刻:页面看起来几乎与原始设计相同，并且可以在大多数浏览器中工作。甚至 IE6 看起来也很棒，支持透明 png。文本仍然是 HTML，图形格式使用得当，验证错误很少发生。

然而，HTML 转换是通过将每个图形和文本行(而不是段落)转换成绝对定位的元素来实现的。这是 HTML 的意大利面——你不会想要手动编辑它——缺乏语义标记可能是搜索引擎优化和可访问性的一个问题。最后，Web Designer 6 努力重新导入它导出的 HTML 代码——记得将原始项目文件保存在安全的地方。

公平地说，Xara 并没有把 Web Designer 6 作为最终的 Web 解决方案来宣传。从本质上讲，Web Designer 6 是一个很好的带有 HTML 导出功能的一体化 DTP 包。

**优点:**

*   最低系统要求
*   非常快速的启动和编辑
*   易于使用的图形工具
*   不需要编码技能
*   Flash 和动画效果
*   巨大的帮助、视频和资源
*   大量可供选择的模板
*   易于插入地图、视频、社交网络等小工具。
*   导出的 HTML 页面与您的设计完全相同
*   良好的跨浏览器支持— IE6 工作良好！
*   输出到其他格式，如 PSD，PDF，TIFF，JPG，PNG 和 GIF
*   自动化 FTP 发布
*   合理的价格

**缺点:**

*   太多笨重难看的弹出窗口
*   只能设计固定宽度的网站
*   无法添加或编辑 HTML，CSS，JavaScript 和服务器端代码
*   只有一个预览浏览器(IE)
*   导出的 HTML 代码不能手动编辑
*   如果您已经购买了 Xara Photo & Graphic Designer，网络发布功能的价值可能有限
*   web 开发设施感觉有点被附加了标签
*   “Designer”这个名字可能会让一些人感到困惑:它是一个纯粹的视觉设计工具，而不是前端编码包。

我喜欢 Xara 网页设计师 6。如果你从未打算学习 HTML 或者想要快速制作原型，这是理想的选择。甚至程序员也可能喜欢它简单的图形和 Flash 制作工具，但是在开发最终的网站代码时，您很快就会回到您的 IDE。

如果你不确定它是否适合你， [**试试 Xara Web Designer 6 30 天免费试用**](http://bit.ly/cqMnnW) 。
