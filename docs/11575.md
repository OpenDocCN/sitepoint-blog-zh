# 设计元素:形状

> 原文:[https://www.sitepoint.com/design-elements-shape/](https://www.sitepoint.com/design-elements-shape/)

上周，我们看了不起眼但用途广泛的[系列，它是设计中的一个元素](https://www.sitepoint.com/elements-of-design-the-line/)。今天我们来看看形状。形状被定义为用来决定或给出形式的任何元素。我们每天都能在商标、旗帜、书籍、衣服上看到形状。形状可以单独存在，也可以与文字或线条一起使用。看下面的形状。没有颜色，但许多人会理解这些形状意味着什么或代表什么。![Shapes1](../Images/3c9de673b3431a879e4fff7a2b1d3c93.png "Shapes1")作为一个设计元素，一个形状可以用多种方式形成。它可以是一块颜色，一幅插图或一张照片。设计师[索尔·巴斯](https://designmuseum.org/design/saul-bass)是处理形状的大师。这是金胳膊男人的海报。弯曲手臂的照片和插图是形状的一部分，文本在形状外面或被形状包围。观看《解剖谋杀案》的片头字幕，看看贝斯是如何将文字与简单而大胆的造型结合起来，再加上爵士音乐(我想是艾灵顿公爵)。

照片增加了作品的趣味性，但也增加了造型。看看下面的海报，是电影《惊魂记》的海报。它包含三张剪成不规则形状的照片/插图。可能是用刀砍出来的那种形状？![PsychoPoster](../Images/66f90c03cf26d979d39dd658220c994f.png "PsychoPoster") **用形状创造兴趣**在网页设计中，有一些形状已经成为标准，例如“三个水桶”三个文本块通常设置不同的背景颜色，如雅芳的这个网站。我们的眼睛会被这些积木吸引，因为它们的形状和颜色既吸引人又有趣。形状当然是最强的设计元素之一。![3Buckets](../Images/45cd0f5b03705e7a06a31e6a72bd2b48.png "3Buckets") [诱虫灯](https://abductionlamp.com/)用一些多边形文本框去掉页面的方形外观。通过使用形状将文本分割成更小的部分，我们给读者一个休息的机会，同时也将文本组织成块。就像线条一样，形状可以组织、分离和连接页面上的信息。形状并不总是需要背景色或描边。它们可以单独使用文本来形成。有时候，仅仅是文字的形状就足以引起人们的兴趣，并打破页面。由于没有添加线条或背景色，下面的示例更加清晰。如果你不需要某个设计元素，就不要用它。![TypographyServed1](../Images/f6f4ff387569c23cff164a72da540f44.png "TypographyServed1") ![TypographyServed2](../Images/716a836f51307547ee36c35b60237d47.png "TypographyServed2")图片来自[排版部](https://www.typographyserved.com/Gallery/The-Readies/145022) ![explorations-1](../Images/a6e2bbdb344227b7f7d9736e0a49b09a.png "explorations-1")图片来自排版部。总的来说，当我们看任何布局时，无论是网页上的还是印刷的，我们的眼睛会更快地被一些元素吸引。我们首先被人类的照片吸引，然后是其他照片或插图，然后是大字体如标题，最后是正文中的小字体。使用形状可以让我们随心所欲地控制或引导浏览者浏览布局。**形状使用指南** 1。想一想你使用一个形状最合适的方式是什么。是照片、插图、色块还是文字块？记住，少即是多，不要用太多的形状把你的观众的眼睛拉得到处都是。2 .如果你有一大块正文，如果你用形状把它分开，设计会受益吗？对于读者来说，阅读贯穿整个网页或印刷页面的长句是很累的。如果合适的话，使用分栏。3 .你使用的形状是否能把观众的视线引向正确的地方？4.不要纯粹为了装饰而添加形状。它们是传达信息的有力工具。下周，我会看看质地。**相关阅读:**
