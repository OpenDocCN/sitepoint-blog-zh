# 网络是为四岁的孩子准备的

> 原文:[https://www.sitepoint.com/the-web-is-for-4-year-olds/](https://www.sitepoint.com/the-web-is-for-4-year-olds/)

如果你正在阅读这篇博客，你可能至少对网络有一定的了解，并且使用或知道像 Twitter 和 Flickr 这样的服务。你可能知道什么是 RSS，如何使用它，为什么要使用它。有时候我们会忘记，并不是每个人都像我们一样酷，一样有知识。有大量的人在互联网上完全不知所措，甚至认为 Google *就是*互联网。这些人通过谷歌搜索网址(甚至是他们自己的网站)，而不是在地址栏中输入。

然而，我们有一个共同点——我们都倾向于像四岁时那样上网。

如果它不能快速加载，立即吸引我们并保持我们的注意力，那么我们就会感到无聊并继续前进——标签式浏览加剧了这一特点。

由于网络的庞大和我们能找到的所有酷的、古怪的和有趣的东西，我们不倾向于花时间去发现一个网站能给我们提供什么。如果这还不明显，那么**下一个**，我们走吧——不太可能再回来。我们只是喜欢新东西。最新最棒的是什么？

Digg ，这个曾经的网络新锐之王，现在显然变成了[的旧帽子](http://mashable.com/2008/05/13/yahoo-buzz-is-bigger-than-digg-according-to-comscore/)。它已经被更新更闪亮的[雅虎超越了。嗡嗡声](http://buzz.yahoo.com)。

但是，Digg 会怎么样呢？雅虎会发生什么？当失去光泽的时候嗡嗡叫？老实说，我们在乎吗？我们只想要新的东西——巧克力蛋糕和果冻；Wiggles 视频；妈妈，我能给猫刮毛吗？哦，继续。请请。

为了在网络上生存，我们也需要像四岁小孩一样思考。你四岁的孩子一直想要和喜欢的是什么？

最受欢迎的可爱玩具——你知道的:独眼的，流口水的，被人爱得死去活来的，必须带着它们到处走。最好的网站和应用程序必须像那个可爱的玩具一样——一个亲爱的朋友，理解我们，支持我们。

但是，也许没有流口水。TTFN。
