# 浏览器趋势，2011 年 5 月:火狐 4.0 和 IE9 爆发式增长，但 Chrome 正在逼近

> 原文：<https://www.sitepoint.com/browser-trends-may-2011-firefox-4-ie9-growth/>

我之前的[浏览器趋势报告](https://www.sitepoint.com/firefox-4-beats-ie9-and-ie7-drops-below-10-percent/)提供了 IE9 和 Firefox 4.0 升级的早期迹象。这两种浏览器都已经推出一个多月了，所以让我们来分析一下最新的[统计数据](https://gs.statcounter.com/) …

| 浏览器 | 三月 | 四月 | 变化 | 亲戚 |
| ie9.0 | 0.75% | 2.32% | +1.57% | +209.30% |

此表显示了市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月 20%的 IE7 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

最令人印象深刻的数字是 IE9 的 2.32%和 Firefox 的 7.59%。两个浏览器在一个月内都有超过 200%的增长。然而，当你的起点接近于零时，你的市场更容易翻倍。

火狐 4.0 的采用增长几乎与火狐 3.x 的下降持平。用户没有理由不升级，所以我预计旧版本将在 6 个月内消失。如果 Mozilla 的快速更新系统按计划进行，这可能会更快。

IE 使用模式稍微复杂一点。IE6/7 升级到 IE8 的次数似乎与 IE8 升级到 IE9 的次数相似。总的来说，它对 IE8 没有显著的影响。我不相信是这样，但至少 IE7 的使用率正在迅速下降。好消息。

Opera 和 Safari 的市场份额几乎没有变化，尽管“其他”浏览器经历了显著增长？基础数据中没有线索，所以这可能是一个统计异常；也许是每月一次，或者因为 StatCounter 最近更新了他们的浏览器/版本检测列表。

对微软和 Mozilla 来说，坏消息是 Chrome 继续蚕食他们的市场份额。谷歌浏览器每月持续增长近 1%，其中三分之一的用户从 Firefox 迁移，三分之二的用户从 IE 迁移。没有迹象表明增长已经停滞，Chrome 用户仍然忠实于这款应用。

浏览器排行榜顶部出现三方平局的可能性似乎越来越大。
