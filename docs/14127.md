# 镇静

> 原文：<https://www.sitepoint.com/composure/>

客户和项目可能极具挑战性。

我的一位导师将专业服务比作试图同时击向你的五个棒球——一个曲球、一个快球、一个指关节球、一个豆球和一个滑球。你真的必须保持警觉，敏捷，并保持冷静。

然而，太多的专业人士在销售过程和内容交付的某个环节失去了冷静。我还记得网络时代，当时我在一家大型系统集成公司从事一个依靠签约团队和自由职业者的大型项目。自由职业者经常会冲出办公室，尖叫着抱怨一些挫折。

我自己有时也遭受这种痛苦。就在这个周末，我和一个客户进行了为期两天的战略务虚会，一直持续到深夜，试图就战略方向达成共识。我厌倦了他们无法集中注意力，非常想对这些人大喊大叫，让他们行动起来，做出决定。

我没有，但我肯定观众们感觉到了我的沮丧和不耐烦，即使是通过声音语调和面部表情这样的小信号。

保持专业是很重要的，不管客户会变得多么沮丧或痛苦。每一个好的项目都会在某个时候陷入僵局，你必须处理它。

你是怎么做到的？事实证明，组织研究是有帮助的。

以下是一些想法:

1.在任何时候，您都在努力平衡与您想要实现的业务成果和结果之间的关系。我们中的一些人过于注重结果，给人以强迫的印象。其他人太专注于人际关系，所以逃避，永远得不到结果。你必须知道你的前景/客户情况的正确平衡。问问你自己，你是否达到了适当的平衡，或者向一个方向或另一个方向倾斜了太多。

2.当事情走下坡路时，顶级专业人士会退出，或者像哈佛谈判项目的人所说的那样“去阳台”。换句话说，他们呼吁休息一下，冷静一下，并打算很快再次召开会议。在脱离时，他们会思考如何改变策略，以实现更有成效的互动。

3.关键是要把你作为一个人的身份和你作为一个专业人士的角色区分开来。这样，你就不会针对个人情况。你是一个演员，表现得像一个完美的专业人士。

4.正如柯维所写的，不管你的感受如何，知道如何选择你的行动是关键。
