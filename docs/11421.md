# 在 Photoshop 中创建多张照片拼贴的效果

> 原文:[https://www . site point . com/create-the-effect-of-a-multiple-photo-collage-in-Photoshop/](https://www.sitepoint.com/create-the-effect-of-a-multiple-photo-collage-in-photoshop/)

使用形状层和一个巨大的图案，我们将制作一个看起来像是由几张照片合成的图像。这很快也很容易做到，所以让我们开始吧。

1.打开要创建效果的图像。

[![Morgan](../Images/0e0c6da70f45939d02030fb7ca21e606.png "Morgan")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/Morgan.jpg)

2.通过选择**编辑>定义图案并给图案命名，使整个图像成为图案。**

**[![PatternName](../Images/5c997dde5621946792c5d9e8d8411f8f.png "PatternName")](https://www.sitepoint.com/wp-content/uploads/2009/12/PatternName.jpg)T4】**

3.创建一个新层，然后按 Shift + Backspace (PC)或 Shift + Delete (Mac)。这将打开填充对话框。从“使用”菜单中选择“黑色”，然后单击“确定”用黑色填充新图层。

[![Fill](../Images/c9ba0f981840c2cdc979697994680158.png "Fill")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/Fill.jpg)

4.在工具栏上，选择矩形工具。然后在屏幕顶部的选项栏上，确保选择了形状图层图标。

[![ShapeLayers](../Images/75cc9cec384d0519b657e9efba87cc13.png "ShapeLayers")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/ShapeLayers.jpg)

5.在黑色图层上绘制一个矩形。这个矩形是什么颜色并不重要，因为我们将在它上面放置一个图案。

[![RedRectangle](../Images/419c322144555098471bea68c413f3cc.png "RedRectangle")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/RedRectangle.jpg)

6.点击图层面板底部的 FX 按钮，选择描边，为矩形图层添加图层样式。我设置大小为 10 像素，位置为内侧，填充类型颜色为白色。先不要单击确定。

[![Stroke](../Images/cfdba7900e030d12190ccf10ef68b4ef.png "Stroke")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/Stroke.jpg)

7.点击图层效果对话框左侧的文字图案叠加。选择您在步骤 2 中制作的图案。按“捕捉到原点”按钮，取消选中“与图层链接”

[![PatternOverlay](../Images/df79092ba24bb7ef1f6e18b8983acf8c.png "PatternOverlay")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/PatternOverlay.jpg)

单击“确定”,图像应该如下所示:

[![FirstRectangle](../Images/f6aead7df4d9b20cc7687279bcc28fe3.png "FirstRectangle")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/FirstRectangle.jpg)

8.如果你在图像周围移动矩形，你会看到图案是如何保持在原位的，这是因为我们在上一步中按了“对齐原点”。现在你可以复制第一个矩形，并将它们散布在整个图像中。要复制矩形，确保矩形层被选中并点击 **Ctrl + J (PC)或 Cmd + J (Mac)。**

9。将新矩形拖到图像的不同部分。通过选择 Ctrl + T 或 Cmd + T 来旋转矩形，以自由变换对象。旋转后，点击图层面板上的 FX 按钮，选择投影，添加一个投影效果。

[![DropShadow](../Images/f8c15063c54ddaeeede00deb97526df4.png "DropShadow")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/DropShadow.jpg)

图像应该看起来有点像这样:

[![TwoRectangles](../Images/08a2d567c86770b04c6fe50619c1e07c.png "TwoRectangles")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/TwoRectangles.jpg)

10.继续复制和旋转矩形。添加任意数量的矩形。你不需要再添加阴影效果了。它将与图层一起复制。

[![Final](../Images/73aae4c99fd17b5063ac550af1408299.png "Final")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/Final.jpg)

为了完成图像，你可以改变矩形的堆叠顺序，这将改变一张“照片”放置在另一张上面的方式。

现在你知道了。
