# 微软出价 420 亿美元收购雅虎！

> 原文：<https://www.sitepoint.com/microsoft-offers-42-billion-for-yahoo/>

本周的大新闻——微软已经开始以 420 亿美元收购雅虎！。

> 在周五的电话会议上，微软首席执行官史蒂夫·鲍尔默表示，在雅虎一年前拒绝收购提议后，他不会接受否定的回答。鲍尔默说:“这是我们和我经过深思熟虑后做出的决定我们相信这是微软和雅虎的正确道路。"

不能保证雅虎！会接受这个提议，或者即使他们接受了，这个交易也会获得美国和欧洲反垄断监管机构的批准。但是考虑到每股 31 美元的出价比目前的市场价值高出 62%。美国的股价一直在下跌，如果他们不接受，我会很惊讶。

如果继续下去，看看联合起来的力量能取得什么样的成就，那将是一件有趣的事情。我还想知道雅虎多元化的投资组合。(我想的是 Flickr、del.icio.us、Upcoming 和类似的网站)，其中许多都是由开源技术驱动的，将在微软的保护伞下保留相同的品牌和战略。更不用说融合两种截然不同的文化所面临的困难了。

有趣的时光。小心谷歌！
