# 定义好的设计

> 原文:[https://www.sitepoint.com/defining-good-design/](https://www.sitepoint.com/defining-good-design/)

大多数人从两个主要的角度来判断一个网站设计是“好”还是“坏”有一个严格的可用性角度，它关注功能、信息的有效呈现和效率。然后是纯粹的美学观点，这是关于设计的艺术价值和视觉吸引力。一些人沉迷于美学和图形，忘记了用户，而一些可用性专家迷失在他们的用户测试中，忘记了视觉吸引力。为了吸引人们并保持他们的兴趣，最大化这两者是很重要的。

要记住的最重要的一点是，设计是关于沟通的。如果你创建了一个可以很好地工作和展示信息的网站，但是看起来很丑或者不符合客户的品牌，没有人会想使用它。同样的，如果你做了一个很好看的网站，但是很难用或者无法访问，人们就会离开。事实上，一个完成的网站设计的元素和功能应该作为一个单一的内聚单元工作，这样**用户会对设计感到满意，但会被内容吸引**。

可用性专家最大的担忧之一是用户浏览页面寻找他们想要的信息所花费的时间，无论是一段内容，到另一个页面的链接，还是一个表单域。设计不应该成为障碍；它应该充当用户和信息之间的管道。

尼克·拉的[网页设计师墙](http://www.webdesignerwall.com/)(见下文)是一个既美观又实用的设计典范。Nick 丰富多彩的插图围绕着页面的结构流动，页面上点缀着导航和网站元素，看起来就像是从素描本上剪下来的。大量手工制作的有机元素形成对比，有助于将你的注意力吸引到博客文章的内容上，而不会影响页面的可读性或组织方式。

[![](../Images/312f64c2e40d9bb7479cfec25ef7a72a.png)T2】](http://www.webdesignerwall.com/)

**用户可以通过直观的导航方便地四处移动**:我们稍后会详细讨论导航的位置，但是主导航块本身应该在页面上清晰可见，并且每个链接都应该有一个描述性的标题。一个导航结构，当光标悬停在其上时不仅改变外观，而且还指示活动页面或部分(如下例所示，来自 nclud.com 的[的导航菜单)，帮助用户识别他们在哪里，以及如何到达他们想去的地方。](http://nclud.com)

[![](../Images/1e7aecde00276f1b40fa5e618f7ed381.png)T2】](http://nclud.com)

二级导航、搜索栏和外部链接不应该成为页面的主要特征。如果我们让这些条目容易找到，并在视觉上将它们从内容中分离出来，我们就能让用户专注于这些信息——但当他们准备转到其他内容时，他们会知道去哪里看。

用户认为每个页面都属于这个网站:即使主页的布局和网站的其他部分有很大的不同，所有的网站页面都应该有一个连贯的主题或风格来帮助设计。

看看来自[摩尔商学院网站](http://mooreschool.sc.edu/)的三张截图。尽管这些页面上的内容块划分不同，但有几个视觉指示器可以让用户知道这些是来自同一个站点的页面。这种统一性很大程度上是由于身份和导航块的重复。持续使用非常有限的调色板(黑色、灰色、黄色和红色)也有助于页面的统一。
