不太熟悉 [CodePen](http://codepen.io/about) ？简而言之，CodePen 是一个在线内部开发环境(IDE)，它允许您编写 HTML、CSS 和 Javascript，同时实时预览您的代码。

## 为什么我想要一个专业帐户

我想要一个专业账户已经有一段时间了，主要是因为我真的很喜欢使用这个产品，以及随之而来的支持水平。Alex、Tim 和 Chris 对用户的反馈非常敏感，甚至在我为[练习应用](http://tevko.github.io/practice/random.html)添加一个功能时，他们还为我编写了代码。

其次，[价格直线上升荒谬](https://codepen.io/signup/plans/)。一个专业账户每月只需 9 美元，大约是我每周咖啡花费的 1/5。

当然，想要一个专业帐户的一个更明显的原因是它附带的额外功能。

### 无限量私人钢笔

需要为工作或中情局测试一些超级绝密代码吗？没问题！一个专业帐户，你可以使超级秘密笔只对你可见，或者你可以使超级秘密笔只对那些有超级秘密链接的人可见！

### 实时视图

编写一些需要在移动屏幕上工作的代码？CodePen Pro 为您提供实时视图！一旦保存了该笔，该笔的所有其他实例将立即得到更新！

### 资产托管

专业帐户为您提供 1GB 的空间来存储您需要的任何类型的资产。这个特性的 UI 特别有用，尤其是当您在创建钢笔的过程中需要拖动资源时。

### 协作模式

有与可能在 1000 英里之外的人一起做一些老式结对编程的冲动吗？Collab 模式将让你和你的结对编程朋友实时编辑同一支笔！

### 教授模式

教授模式允许一群人在实时教室里观看你写代码。您键入的每个字符、选择的行或更改的设置都会立即显示在观众(学生
)的屏幕上，就好像你们一起看着同一个屏幕一样。

### 100 条发送到手机的消息

这是一个绝妙的想法，证明了 CodePen 团队对他们的产品投入了大量的思考。此功能允许您向自己或您喜欢的任何人发送包含笔的 URL 的 SMS。输入号码就行了！

### 主题

使用专业帐户，CodePen 允许您调整笔嵌入的外观和感觉以及您的 CodePen 个人资料！

## 迄今为止的经验

现在我有机会作为专业会员使用 CodePen，我不得不说它没有辜负它的宣传。我还没有机会使用所有的专业功能，但我对我用过的功能非常满意。私人笔、资产托管和实时视图功能极大地提高了我的工作效率，尤其是当我需要为工作中的项目创建快速演示时。这也很高兴有一个更好的机会在我的钢笔被精选(这完全发生了，非常令人兴奋)。

我一直是 CodePen 的免费和专业用户，我永远不会后悔成为专业用户。专业帐户的特性、设计和可用性使它成为一种工具，成为我作为一名 web 开发人员工作流程中必不可少的一部分。
