# 微软通过慈善捐赠鼓励 IE6 升级

> 原文：<https://www.sitepoint.com/microsoft-ie6-charity-donations/>

微软收到了网络开发社区对 IE6 的一些批评。过去漏洞百出的浏览器是我们生活的祸根，微软将支持 IE6 到 2014 年的消息遭到了鄙视。

不过，先说清楚一点:**微软对延长 IE6** 的寿命没有兴趣。它的寿命可能因为他们的不作为而被无意中延长了，但微软希望用户升级到 IE7，或者最好是 IE8。

为此，微软将向美国饥饿救济慈善机构“喂养美国”捐赠 8 顿饭，以奖励从他们的“T2 浏览器让生活更美好”网站下载的每一份 IE8。如果你是从 IE6 升级，微软会加倍捐赠。

“喂养美国”向美国 2500 多万面临饥饿的低收入人群提供粮食援助，其中包括 900 多万儿童和近 300 万老年人。微软将捐赠高达 100 万美元，每一次 IE8 下载将为慈善机构贡献至少 1.15 美元。

如果你一直在拖延升级，也许是时候屈服于 IE8 的喜悦了。这是一个很好的浏览器。说实话。你会为慈善事业尽一份力。
