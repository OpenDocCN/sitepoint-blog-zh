# 让你的非苹果电脑预装 Mac OS X

> 原文：<https://www.sitepoint.com/psystar-sells-pcs-with-os-x-leopard-pre-installed/>

我一直认为，如果你是一名负责任的网络专业人士，苹果 Mac 电脑是最好的选择。

我知道这是一个挑衅性的声明，但请听我说完。我是 Mac 用户，但我不是粉丝，这也不是 DHH 式的盲目咆哮。这些年来，我在[也经历过](https://www.sitepoint.com/apples-os-x-utter-crap-according-to-linux-founder/)[打嗝](https://www.sitepoint.com/adobe-cs2-breaks-intel-apple-macs/)的。我提倡网页设计师使用 Mac 的原因不是你通常所说的“我们对他们”的口水仗。病毒，视觉糖果，稳定性，安全性…无论什么。

我认为每个网页设计师都应该使用 Mac 的一个原因是:

你需要在 Mac 版 Safari 上测试你的网站，但你不能在 PC 上运行 OS X。

就是这样。如果你创建网页，那么你需要在 Mac 的 Safari 上测试这些网页。Windows 版 Safari 与 Mac 版 Safari 的近似度还不错，但又不完全相同。

当然，有多种方法可以执行跨平台浏览器测试。使用像 [BrowserCam](http://www.browsercam.com/) 这样的服务，让第二台 Mac 计算机可供您执行测试——这些都是有效的方法。但是如果你打算买一台机器，我总是发现使用 [Parallels Desktop](http://www.parallels.com/) 或 [VMWare Fusion](http://www.vmware.com/products/fusion/) 在虚拟机中运行 Windows 的方法，你可以用它在 IE 中执行测试，从财务和工作流程的角度来看，这是最有意义的方法。你可以在一台机器上做任何你需要做的事情。

直到最近，也就是说。

自从苹果改用英特尔芯片以来， [OSx86 项目](http://www.osx86project.org/)，一个在*任何基于英特尔的个人电脑上运行的逆向工程 OSx 的开源尝试，已经真正成熟。然而，OSx86 项目存在一些问题:*

1.  在非苹果硬件上安装 OS X 是非法的。OS X EULA 清楚地表明了这一点，虽然我不是律师，但我看不出有任何漏洞表明这样做是合法的。
2.  **鉴于上述情况，支持不存在**。
3.  安装是一件痛苦的事。如果你不是一个准备修补的极客，你很可能会把自己的头发都拔掉。
4.  **兼容性各不相同。**你需要做好准备，放弃操作系统偶尔会出现的功能，这些功能可能有用，也可能没用。

一家公司决定忽略第一点，代表他们的客户解决第二点和第三点。Psystar 正在销售一种他们称之为[的开放式电脑](http://www.psystar.com/index.php?&page=shop.product_details&flypage=flypage_images.tpl&product_id=1&category_id=3&option=com_virtuemart&Itemid=72)——一种非苹果电脑，预装了 Mac OS X Leopard。标准个人电脑的零售价为 399 美元，其规格与 Mac Mini 相当。 [OpenPro 电脑](http://www.psystar.com/index.php?&page=shop.product_details&flypage=flypage_images.tpl&product_id=19&category_id=3&option=com_virtuemart&Itemid=72)是一台增强的 2.6 GHz 机器，价格是苹果可比 MacPro 的一小部分。

现在，Psystar 随时都有可能收到苹果法律部门的敲门声，这台打开的电脑将被拆除，再也见不到阳光了。但也有一种可能性是，苹果公司会在远处静静地观察，看看这是一次多么成功的冒险。这可能会影响苹果将客户锁定在他们的硬件上的方向，也可能不会。

当然，有一种观点认为，苹果强迫其客户使用苹果硬件的决定是反竞争的。如果 Psystar 引用微软视窗作为先例成立，那么我们将在桌面操作系统领域有一个全新的游戏。
