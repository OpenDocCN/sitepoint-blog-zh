# 第三部分:股权与现金

> 原文:[https://www.sitepoint.com/part-three-equity-vs-cash/](https://www.sitepoint.com/part-three-equity-vs-cash/)

在上一篇博客(第二部分:股权与现金)中，一位读者写道:

我现在的情况是，客户以正常价格的 1/3 让我参与一个创业项目。还有其他因素促使我接受一半的费用，但这些因素已经开始减少。我最近告诉他，我的费率将会增加，他回复了一条关于激情和冒险的信息，该项目很快就非常有利可图——并承诺分享利润。有没有什么可接受的方法来要求在虚拟利润中获得合同份额，或者以某种方式确保如果/当他的项目变得有利可图时，我将获得一份份额？一旦美元滚滚而来，我可以预见自己将被卷入其中并被解雇。

以下是我的回应:

首先，越来越明显的是，Sitepoint 的许多读者是非常慷慨的人。你们中有太多人似乎不介意成为创业企业的擦鞋垫。谢谢你让资本主义继续发展！你的激情和承担风险的意愿(同样的风险，资本家已经转移给你，所以他不再有那么多)有助于为许多风险投资提供资金。

然而，你确定你没有把激情和愿意冒险与自卑和不会谈判混为一谈吗？

如果你想从这些初创企业中赚钱，你必须变得聪明和自信。

在上面的案例中，如果虚拟利润出现，读者有三种选择来“要求”虚拟利润中的合同份额:

–坚持增加费用和预付现金。告诉资本家，他如何找到现金支付供应商是他的问题。作为一名网页设计师，你不是企业的负责人，而是承包商。你的优势很低，所以你的劣势也应该很低。

–向公司提供贷款。这是公司欠你的一张票据，是一份法律文件，作为负债出现在他们的资产负债表上。它应该具体说明他们付你多少钱，什么时候付(包括利息)，不考虑利润。还应该注意的是，股东和委托人在票据支付之前不会得到任何补偿。如果你不付钱，他们为什么要付钱？毕竟，他们对企业的价值有长期的兴趣，而你没有。

–创建一份合同，规定您从企业中获得任何收入的 x%,在设定的日期后有明确的最低绝对金额，而不管公司的盈利能力如何(因此这种安排是部分贷款部分收入参与)。这样你就限制了自己的劣势，也有了一些优势。一定要明确定义收入为收入减去费用加上任何本金补偿。否则他们会通过零收入同时拿很多收入(这是好莱坞制片人在给演员打分时倾向于做的)来烧伤你。就我个人而言，我反对这种安排，因为它很复杂，而且需要你查阅公司的账簿。此外，您还可以指定，如果公司被出售或发行任何股息，您有资格获得任何收益的 y%。

在我看来，最好的办法——很多伟大的网站设计公司都这么做——是让自己变得有价值，并提前获得报酬。提高你的自尊以及谈判和主张的意愿/能力。并愿意对不符合你收入预期的交易说不。(正如在上一篇博客中提到的，如果其他因素对你有影响，接受一份低薪工作；但长期来看，它不能养活你的家人。
