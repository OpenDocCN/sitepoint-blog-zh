# 一个创造性的营销策略来提高你公司的知名度

> 原文：<https://www.sitepoint.com/a-creative-marketing-tactic-to-generate-awareness-of-your-firm/>

上周末，我在纽约参加了一个大型贸易展览，一家小公司使用了非常有创意的营销策略，值得分享。它适用于任何社交活动，而不仅仅是商展。

他们向遇到的任何人分发上面有数字的别针/纽扣。这些别针包括公司的标志和口号，以及一个与另一个别针完全匹配的 4 位数字。如果别针的主人在交易会上发现另一个人有完全相同的号码，两个人都会得到一份奖品(在这种情况下是 100 美元，但你可以随意编造任何奖品)。

结果是令人着迷的，因为一个小的亚社区形成了，人们互相寻找对方的别针，互相交谈，并且肯定记得赞助公司的名字——以前是没有名字的。很快人们就开始询问这家公司的展位号，以便他们也能参与进来。在一个有 20，000 人参加的贸易展会上，这家公司以相对较小的投资产生了巨大的知名度，当然比那些发传单或等待人们驻足的公司做得更好。

在商会或协会的会议上，你可以发起一场类似的游戏，尽管由于会议规模较小，奖金较少。

即使你不喜欢这个特别的想法，它也显示了你如何变得相当有创造性，以产生更多对你公司的认识。
