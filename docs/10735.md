# 六个新的 Twitter 应用程序

> 原文:[https://www.sitepoint.com/new-twitter-apps/](https://www.sitepoint.com/new-twitter-apps/)

Twitter 应用程序不断涌现。如果你是一个 Twitter 用户，你可能喜欢探索新的工具，看看它们如何帮助你将 Twitter 用于商业和个人目的。

为了帮助你解决一天的 Twitter 问题，也许可以找到一个新的工具来定期使用，这里有六个新的 Twitter 应用程序可以测试。

## [Fflick](http://fflick.com)

[![](../Images/cf61ba92b7a7ace0ff582c16e7361da5.png "fflick")T2】](http://fflick.com)

一个 Twitter 应用程序，用于了解人们对目前正在上映的电影的看法。用户对评论进行评分，每部电影都有一个分数来帮助你决定去看什么。该网站显示当前电影、关于电影的描述和其他信息、关于电影的推文以及投票的地方。

## [LikeMyTweets.com](http://likemytweets.com)

[![](../Images/cee002f572e59db77d25c4187693b503.png "likemytweets")T2】](http://likemytweets.com)

如果你“喜欢”脸书上的很多东西，你可能会喜欢这个工具。你只需安装“喜欢”按钮，从你的 Twitter 客户端发布一条推文，它将包含一个链接，允许追随者“喜欢”它。您可以更改设置，让所有推文或仅特定推文包含“喜欢”按钮，选择当有人喜欢您的推文时是否希望接收直接信息，并更改“喜欢”链接前的措辞。

## [多重](http://multwiple.com)

[![](../Images/a95356ffa7eb217ca088f6e4ef2d753d.png "multwiple")T2】](http://multwiple.com)

搜索新的 Twitter 客户端？Multwiple 是一个基于网络的 twitter 客户端，它像其他几个选项一样工作，允许你在一个地方使用你所有的 twitter 账户。其他功能包括内置的网址缩写、搜索、统计和所有常见的 Twitter 客户端功能。

## [Tweet-U-Later.com](http://tweet-u-later.com/)

[![](../Images/655d6f07fef1e999c9c93afd19556385.png "tweetulater")T2】](http://tweet-u-later.com/)

这是一个非常简单直接的方法来安排以后的推文。您可以通过网站或电子邮件使用 Tweet-U-Later Twitter 应用程序。

## [TwiMMS](http://TwiMMS.com)

[![](../Images/32dc64b34bd59c702dc3ebc134de7c1f.png "twimms")T2】](http://TwiMMS.com)

TwiMMS 是 Twitter 的一个多媒体消息服务，主要使用 Twitter 来交换富媒体消息。您可以通过 tweet 或直接消息向关注者发送 HTML 格式的消息或上传附件。

## twitFlink

[![](../Images/34a68deac5487fa3147087c02c887d85.png "twitflink")T2】](http://twitflink.com)

如果你试图找到一个你在 Twitter 上看到或发布的链接，这个工具可能会有所帮助。这款应用可以让你找到任何人在 Twitter 上发布的链接。你首先输入一个用户名，它就会调出这个人所有包含链接的推文。然后你可以输入一个关键字来过滤结果。
