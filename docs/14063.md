# 谷歌支付？

> 原文：<https://www.sitepoint.com/google-payments/>

《华尔街日报》上周晚些时候报道了谷歌即将推出类似 PayPal 的支付服务的传言，这似乎是网络世界的新主人。

昨晚晚些时候，谷歌首席执行官 Eric Schmidt 的评论澄清了一些猜测，他声称这个在线搜索的神童不会冒险进入易贝的地盘。然而，众所周知，其他创新者利用“摇头”来消除早期的竞争担忧，同时把握市场部门的脉搏，并为其他参与者提供可能的空间。

谷歌确实挥舞着大棒，其收入和股价使其能够进入许多领域，然而，易贝是一个同样根深蒂固的全球品牌，其成本可能太大，无法正面交锋。现在很多猜测都指向一项可能的电子支付服务，这项服务与谷歌现有的 Froogle 购物搜索资产相关联。

自从 PayPal 成立以来，我就一直在个人和专业领域使用它，并且是它 7200 万注册会员中的一员。我发现它作为消费者和开发者都很有价值，因为它易于使用，入门成本低，并且对互联网商务的变化很敏感。我自己对谷歌进入电子支付领域的担忧，来自于看到他们从搜索先锋和专家转变为一家多元化的科技公司，涉足无限多的领域——或许还会分散自己的力量。

他们目前在搜索领域绝对领先——然而，如果他们成为 IBM 或惠普那样的臃肿公司，放缓步伐——易贝对商业的专注肯定会让他们永远处于第二的位置。

在这短短几天里，两人将在科技领域展开正面交锋的热潮蔓延开来，我非正式地询问了一些商业同行，寻求他们对这种可能性的反馈，其中包括一名欧洲的网站所有者、一名涉足美国西部电子商务的搜索引擎优化专家，以及一名精通科技的美国中西部营销人员。他们的评论多种多样且有趣，我想我会分享他们，并寻找 SitePoint 读者的评论。

我住在巴伐利亚阿尔卑斯山脚下的好朋友(一名网络开发人员和多媒体设计师)表示，基于他的 Google AdSense 收入，这将使他受益，允许他将资金存储在 Google 中，或许可以将这些钱用于广告或其他目的。他确实提到，他使用贝宝来接受捐款或通过他的各种在线财产进行销售。

“竞争孕育创新，因此总是受欢迎的。搜索引擎优化专家克里斯·克雷默补充道:“这一特定案例的危险在于，它可能会让谷歌变得无所不能，从而变成一家被人唾骂的野蛮公司——这与其‘不伤害’的信条相抵触。”。

他确实相信，如果谷歌选择进入这个领域，它应该让可用性因素变得如此简单，以至于他的祖母可以设置这项服务。也许这是贝宝的一个小缺点，虽然我自己觉得贝宝很容易使用。

也许最能说明问题的是一位非互联网营销专业人士的评论，尽管他精通技术，对尖端产品很在行。

“竞争是有益的，迄今为止，Paypal 占据了这一领域。在一个细分市场中只有一个玩家通常对消费者是不利的(我希望谷歌能成为 Ebay 的竞争对手。谷歌知道 Ebay 不知道的事情——简单是关键)。”

他补充说，Paypal 在大多数方面都做得很好(安全性是第一位的)，但在营销和部署方面很脆弱。

“他们使用的信息对于一个安全稳定的形象来说是愚蠢和过分的。我认为，在消费者心目中，他们想要占据的位置总是最前沿的——在安全性、开创性的新产品、产品可用性等方面。让秃子坐在办公桌前是错误的做法。”

这就是谷歌的机会所在。正如我前面提到的，易贝和谷歌拥有稳固的品牌，尽管赖斯和特劳特在《定位:为你的思想而战》一书中认为线延伸(谷歌扩大其产品组合)是一条危险的道路——后者引用了我那位博学的营销朋友的话。

作为最后一个想法——虽然网站管理员和开发人员可能会探索谷歌服务——说服普通人接受像 PayPal 这样的支付服务。我的非正式研究小组一致认为，除了零欺诈责任之外，普通消费者只会在在线服务上赌这么多。因此，如果 PayPal 为用户服务，为什么还要尝试其他服务呢？也许这将是最大的障碍，即使谷歌确实避开了贝宝和易贝·沃特斯。
