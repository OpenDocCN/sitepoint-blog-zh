# Ask.com:我们已经得到了所有的答案

> 原文：<https://www.sitepoint.com/askcom-were-going-to-be-the-best/>

在我最近关于 Ask.com 最近裁员和战略方向变化的帖子之后，我和 Ask.com T2 发言人 Nicholas Graham 进行了一次有趣的聊天。在大量猜测 Ask.com 将完全退出搜索功能之际，Graham 澄清了一些细节。

“美联社发表了一篇报道，暗示我们正在淘汰我们的搜索技术——这篇报道并不真实，事实上他们已经收回了这篇报道。”

不管是真是假，这肯定助长了谣言，加上最近的裁员，给 Ask.com 的新战略是否是一个好战略打上了一个大大的问号——这种困境自以来一直让格雷厄姆[灭火。](http://www.marketingpilgrim.com/2008/03/askcom-will-continue-to-be-a-great-search-engine.html)

“事实上，我们拥有非常忠实的用户群——事实上，我们的研究显示，我们的核心用户(碰巧主要是女性)使用 Ask.com 搜索三种类型的信息:参考、健康和娱乐。有趣的是，这些用户实际上使用 Ask.com**的次数是任何其他搜索引擎**的三倍。

“我们肯定还在创新。我们仍然计划成为一个伟大的搜索引擎——事实上，对寻找答案的用户来说是最好的。事实上，我们正在招聘一些关键职位来帮助支持这一新战略。”

事实上，如果你试图满足所有人的所有要求，你肯定会失败，我相信 Ask.com 决定关注其核心用户而不是竞争对手。但是，在搜索引擎中输入查询的用户中，有百分之多少是专门搜索答案的呢？(比如说，与图片、产品、服务、电话号码相比……)你到底如何定义一个“答案”？期望用户用一个搜索引擎寻找“答案”，而用另一个搜索引擎寻找其他东西，这合理吗？(我个人只是什么都用谷歌)。

两年前，当搜索引擎的关注点扩大，Ask.com 宣布他们将在 T1 与谷歌正面交锋时，他们似乎不这么认为。看在老天的份上，我希望他们这次找到了答案。
