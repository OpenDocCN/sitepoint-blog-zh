# 谷歌通过电子邮件向 AdSense 出版商保证

> 原文：<https://www.sitepoint.com/google-reassures-adsense-publishers-via-email/>

今天下午，谷歌向 AdSense 出版商发出了一封信，向他们保证在“经济动荡”时期，谷歌正在努力提高出版商的收入。

谷歌说他们正在做的事情包括:

1.  更好的搜索广告
2.  将双击技术集成到 AdSense 中
3.  提高 AdWords 广告客户锁定内容网络的能力

这封信没有具体说明，所以我们有两个问题要问读者:

1.你有没有注意到自从上个月世界金融市场开始下跌以来，AdSense 的收入下降了？
和
2。谷歌的信让你放心吗？

这封信的全文如下:

> 亲爱的出版商:
> 
> 我们知道最近的经济动荡给 AdSense 出版商的生活带来了很多不确定性。在这段困难时期，我们将继续投资创新，提高内容网络中出版商的盈利能力和广告商的价值。
> 
> 我们正专注于进一步开发我们的产品和提高出版商的广告业绩。我们最近宣布了 AdSense 在搜索和实验方面的进步，以使广告更有效。我们将 DoubleClick 技术引入 AdSense 出版商，并将继续推出新产品和新功能。我们也在继续改进我们为 AdWords 广告商提供的服务，让他们更容易瞄准谷歌内容网络。面向广告商的功能，如新的显示广告生成器，旨在提高 AdSense publisher 网站上的广告性能。
> 
> 我们将继续推动技术进步，但我们最好的资产永远是我们的出版商合作伙伴。AdSense 的优势在于你给用户带来的内容价值和你给广告主带来的站点质量。我们的成功与你们的联系在一起。我们期待与您长期合作，并致力于帮助您取得成功。
> 
> 真诚地
> 
> 金斯科特
> 主管站长在线销售&运营
