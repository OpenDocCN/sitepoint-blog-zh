# 搜索词工具-测量受欢迎程度

> 原文：<https://www.sitepoint.com/search-term-tools-measuring-popularity/>

我们想要搜索的第一个也是最明显的数据类型之一就是它在搜索者中的受欢迎程度。这对所有的搜索引擎营销人员都很重要，无论他们是从事 SEO 还是 PPC 活动。

不幸的是，当谈到关键词流行度时，可用的数据可能很难信任。今天，我将浏览目前可用的主要搜索术语数据库，并讨论每种数据库的优缺点。

所有的搜索词数据库都会受到一定程度的偏斜的影响。主要问题是像 WebPosition 这样的排名检查应用程序，它们向多个搜索引擎提交自动查询，以获得大量的搜索术语。每当有人检查他们的排名(即使是通过手动搜索)，它会增加该搜索引擎上该搜索词的数量…有时在其他搜索引擎上也是如此。

大多数人使用的第一个工具是 **Overture 的搜索词建议工具**(谷歌一下，我会链接，但他们已经不止一次地改变了网址)。它是免费的，所以很受欢迎。不幸的是，如果你打算把它作为搜索引擎优化的关键词研究工具，“免费”是你能说的最好的东西。为什么对 SEO 这么难信任？

Overture 工具中的 Skew:几乎所有的排名检查应用程序都会击中主要的搜索引擎，因为 Overture 的广告是在其中两个(MSN 和雅虎)上进行的，所以每次排名检查都算作 Overture 的 2 次搜索。其他检查竞价排名的应用程序也可能在 Overture.com 上运行查询，进一步夸大一些数字。Overture 也有像 Claria 这样的合作伙伴，在弹出和弹出广告上提供搜索结果，在过期的域名上显示搜索结果等。这些都算作 Overture 的“搜索”，尽管它们并不真正代表一个人在进行网络搜索。

Overture 工具中的搜索词:因为它们纯粹是一种按点击付费的服务，Overture 对搜索词做了一些事情，如果你使用该工具进行 SEO，这些事情并没有真正的帮助。一个搜索词的单数和复数版本将合并成一个，这很重要，因为很多时候只有一个(单数或复数)与一个网站真正相关。一些长搜索词实际上可能是组合在一起的——例如“红色本田汽车”和“红色本田汽车”Overture 工具上显示的搜索词实际上可能代表许多搜索词的组合。这对 PPC 广告来说无关紧要，但对 SEO 来说却是一个障碍。

如今，大多数专业的搜索引擎营销人员都使用关键字数据库。这个数据库是根据之前 60 天在 Dogpile 和 Metacrawler 元搜索引擎上的搜索汇编而成的，这减少了许多由排名检查引起的偏差，因为大多数排名检查器并不关心元搜索引擎。(元搜索引擎结合了几个搜索引擎的结果，所以它们的排名并不能真正代表搜索引擎营销人员可以采取的任何行动)。

因为 Wordtracker 的数据与其他相比相对“干净”,我们在很多方面使用它，从帮助我们识别搜索词，到评估搜索词的相关性。Wordtracker 还提供许多搜索引擎的竞争数据，包括 Overture 的点击付费竞价。

相对较新的是 [**关键词发现**](http://www.keyworddiscovery.com) 数据库，由优先提交付费收录服务的母公司 Trellian 运营。Trellian 的数据库是独一无二的，因为他们有超过一年的数据。这使得营销人员可以搜索本质上是季节性的关键词——如果你需要知道情人节礼物的热门搜索词是什么，但现在是 11 月，你不会在其他任何地方获得这些数据。

Keyword Discovery 的数据库非常大，包含超过 110 亿次搜索，来自……嗯，“来自哪里”是我们不知道的，他们也不能透露。因此，我们对这个数据库提供的搜索计数，甚至是季节性趋势的数据，并不太信任，因为我们不知道某个月数据库中有多少搜索词。因为我们不知道特定月份的数据库大小，所以很难预测搜索引擎营销活动的结果——根本没有办法从它们的计数中估计下个月会有多少次搜索。

除了有季节性搜索词(即使有不可靠的统计，也比没有强！)，关键字发现相对于 Wordtracker 的主要优势是他们开发的帮助营销人员进行研究的工具。其中包括一个优秀的“相关术语”工具，以及一个 URL 分析器，它可以从网页中提取搜索术语，并显示数据库中的实际计数列表。

MSN 的 bCentral 也有一个关键字搜索工具，随他们的月订阅服务一起提供。不幸的是，用户必须跳过这么多的关卡才能得到它，而且所提供的数据是如此浅薄，以至于在这一点上我不能推荐它。不过，我会继续关注它，如果它变得更好，我会让你知道。
