# 你的激情在哪里？

> 原文：<https://www.sitepoint.com/where-is-your-passion/>

这个世界上有两种人:对网络充满热情的人和不热衷的人。

第一类人坚信网络不仅仅是一个分享照片和支付账单的工具，但是他们为什么对此如此热衷却各不相同…

#### 浪漫的

有些人有一种浪漫的想法，认为网络是交流的终极媒介。浪漫主义者非常珍视 W3C 灌输的价值观——网络上的信息是[[来源](https://www.w3.org/Consortium/about-w3c.html)]:

> 适用于所有人，无论他们的硬件、软件、网络基础设施、母语、文化、地理位置或身体或精神能力如何。

同意这种说法是一回事，虔诚地相信是另一回事。我很幸运，在我的朋友中有几个人属于这个群体。这些人散发出一种不可估量的能量:在用户组会议和大会上，人们围着他们听他们说些什么。这类浪漫主义者具有影响力、驱动力和激励力。

#### 企业家

企业家喜欢网络，因为它让竞争变得公平。一个年轻的两个人的公司可以创造一个比成熟的跨国公司更令人印象深刻、看起来更专业的网络形象。

企业家们不断发明新的网络赚钱方式。在线企业的潜在受众是全球性的，网络企业家有能力认识并利用这一事实。只有在网络上，一个企业才能在短短几年内从几乎一文不值变成以数十亿美元的价格出售，而企业家是那些不惜一切代价来实现这一目标的人。

#### 技术专家

从本质上来说，网络鼓励合作和创新。像 Apache、Firefox 和 Linux 这样的开源成功案例都证明了一个事实，即社区驱动的努力可以产生惊人的结果。

我喜欢那些热衷于 Web 开发的技术人员，不管他们使用什么语言，都不断致力于提高自己。关于 PHP 的博客比关于园艺、航海和育儿的加起来还多(来源: [Technorati](http://technorati.com/) )。人们分享他们学到的新技巧；他们基于他人的想法；因此，该行业一直在向前发展。

#### 多面手

多面手参与了几个馅饼。他们既有创造性*又有技术性*。多面手可以写服务器端代码、客户端代码，设计出漂亮的东西；他们学东西很快，适应能力也非常强。

多面手需要工作中的多样性来保持动力——一个他们可以在左脑和右脑任务之间交替的环境是给他们提供工作满足感的东西。

我认为自己属于这一类——我的软件开发背景意味着我总是对技术性的东西感兴趣，但是如果我不做一些有创造性的事情，内心的艺术家就会感到局促。网络是我可以同时做这两件事的媒介；我现在的工作让我可以同时通过我们出版的书籍、文章和博客教育他人。
