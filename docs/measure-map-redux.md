# 测图 Redux？

> 原文：<https://www.sitepoint.com/measure-map-redux/>

大约一周前，我还在怀疑谷歌是否已经放弃了测量地图(T2)。今天早上，出乎意料的，我收到了一封电子邮件，开头有趣地写着“还记得测量地图吗？”。看来谷歌要推出新版本了！这封邮件是这么说的:

> 还记得测量地图吗？几年前，我们给你介绍了我们博客分析软件的早期 alpha 测试。从那以后，发生了很多事情。我们被谷歌收购了，我们重新设计了他们的分析应用程序，然后
> 我们从零开始重建了 Measure Map。

为了移动到新的测量图，电子邮件指示帐户持有人登录到新的服务器，并填写简单的注册表格，以便开始转换。谷歌分析帐户是必需的，因为是在你的博客上安装分析跟踪代码。或许这意味着新的测量图将成为分析数据的替代前端？

总之，我填好了表格，现在等待下一步:

> 太好了！你都准备好了。我们这边有几件事要做。当我们准备好时，我们将向您发送电子邮件(很快！)并说明如何登录。

我会随时通知你的！
