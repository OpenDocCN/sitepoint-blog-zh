# 网站推广速成班

> 原文:[https://www.sitepoint.com/promotion-crash-course/](https://www.sitepoint.com/promotion-crash-course/)

**如果在我的[上一篇文章](http://www.ecommercebase.com/article/621)之后，你已经花大价钱买了一个公司网站，不管是不是专业的设计公司，下一步就是要确保人们知道它在哪里，以及为什么他们应该访问。**

许多中小型公司在网站建成后很少或根本不去实际使用和维护它们。事实上，正是因为这个原因，我在工作中遇到的大多数小企业网站弊大于利。这很可悲，但却是事实。不过，只要稍加努力，这张“网上名片”就能成为你一直以来都知道的宝贵资产。

这篇文章旨在为网站所有者和网站设计者提供一份清单，帮助他们顺利地将网站整合到任何“实体”小企业中。它不包括搜索引擎。关于这个主题的文章有几百篇，这不是其中之一。

好的，我们开始吧…

##### 启动你的网站

希望你对你的新网站感到非常自豪，这可能意味着你想告诉人们它。有几种显而易见的方法，任何人都可以做到这一点:

*   电子邮件
*   电话
*   口碑
*   蜗牛邮件

是的，我知道——这几乎不是火箭科学，但从基础开始总是有帮助的，告诉你所有的客户和朋友关于你的新网站是一个好的开始。

让我们更详细地看一下这几点:

***邮箱***

给你通讯录里的每个人发邮件，如果可能和合适的话，确保你公司的每个人都做同样的事情。但是要小心！这种邮件很容易被认为是垃圾邮件。请记住这两点:

1.  保持简洁明了。4-5 行包括网址就足够了。

3.  **发送纯文本。不要试图发送 html 格式的电子邮件，因为许多人会反对，有些人甚至不会打开它。此外，大多数电子邮件客户端会将 URL 呈现为链接。**

***电话***

在这里，我们将重点关注你如何接近你的客户(你的朋友可能会很高兴收到你的来信，对你的新网站感到兴奋，所以打电话告诉他们不会有什么大不了的)。

以我的愚见(这意味着它一点也不卑微！)，有两个规则可以让你在不引起烦恼的情况下通过电话推销你的网站:

1.  打电话告诉你的客户你的新网站给他们带来的真正好处。

3.  当你下一次在业务过程中与他们交谈时，告诉他们这一点——不要仅仅为了告诉他们你的网站而打电话给他们。

例如，您可能希望告诉您的所有客户他们现在可以在哪里找到重要信息，或者可能利用特殊的“仅网站”优惠。稍后将详细介绍。记得告诉公司里接电话的人，在每次电话中也要提到新的网站。

***口碑***

这实际上不需要太多解释，除了指出当你告诉别人你的新网站时，如果你能给他们一张显示网址的名片(或者至少写下来)，你的推荐会产生更大的影响。

如果与你交谈的人在访问你的网站时感觉到一些好处，比如特别优惠和有关产品和服务的信息，你在这里的努力将再次取得更大的成果。

***蜗牛邮件***

这里的规则与电子邮件基本相同。最好的方法是写一封简短的邮件，详细说明你的网站的具体优势，或者在你给客户的常规信件中包含一个小的注释/段落。

我在 qexo.com 的一个朋友也建议使用明信片。你不会错过网址，因为信息没有藏在信封里，只有小信息的空间——让每个人都开心！

##### 推广你的新网站

好了，现在你已经把你的新网站告诉了所有人，你能做些什么来强化你的信息并进一步提升你的在线形象呢？

这里详细介绍的过程也属于一些相当简单的类别，虽然它们中没有一个会单独产生巨大的影响，但一起使用它们会对你的网站的有效性产生相当大的影响。

我们将关注的基本领域包括:

*   电子邮件
*   打印
*   电话

***邮箱:真实黑仔 App***

你已经给公司集体通讯录中的每个人都发了邮件，你还能做什么？实际上是很多东西。电子邮件是原始的“杀手级应用”,它提供了一种既能推广你的网站，又能为你的客户服务增值的理想方式。以下是一些让你开始的想法:

*1。员工电子邮件账户*

确保每个人都有一个`@yourcompany.com`电子邮件地址，更重要的是，确保他们使用它。在你的网站上提供这些电子邮件地址，并鼓励你的客户通过这种方式进行业务查询。

不过，有一条黄金法则你应该遵守:你必须确保定期检查电子邮件地址——对大多数企业来说，至少一天三次。不回复他们的邮件会让你的客户感到不被爱，并且不可避免地会导致失去生意。

*2。创建标准公司签名*

确保从你公司地址发出的每封邮件都有正确的签名。这应该不超过四行，包含有用的信息，如姓名，职位，电话和传真号码的发送者，当然还有公司的网址。

这里有一些关于签名的好的指导方针，以及如何使用 Outlook 创建签名的解释(抱歉，如果你用的是别的东西，但和我一样，你是少数:-)

*3。创建通用电子邮件地址*

大多数公司都需要信息、销售、帮助或 support@yourcompany.com 等地址，让他们可以通过你的网站方便地访问，并确保他们定期检查和适当的回应。

还有一点，如果你在你的网站上使用 mailto 链接，确保链接文本是实际地址而不是人名。不是每个人都想用电子邮件客户端来联系你的公司，尤其是如果他们碰巧用的是别人的电脑。如果您提供了电子邮件地址，他们将能够记下并在以后给您发送电子邮件。

*4。使用自动应答器*

请咨询您的 ISP，看看他们是否提供自动回复服务——很多都提供，而且通常设置起来并不难。如果你的主机不提供这个功能，那么有一些像这个一样的免费服务，你可能想试试。

有趣的是，从你发邮件的公司收到自动信息，即使你知道它是由机器发送的，它仍然有“我正在被照顾”的性质。如果你不能立即回答客户的问题，简单的“我们已经收到你的信息，很快会回复”也能为你争取一点时间。

***在印刷品上推广你的网站***

这是一个广泛的类别，涵盖了从名片到产品包装、店面和广告的所有内容。这里有一些想法，应该可以帮助大多数小企业。

*文具等。*

包括您的 URL 以及相关的电子邮件地址:

*   信头
*   商业名片
*   收入
*   担保
*   包装
*   钢笔、t 恤衫、马克杯和保险杠贴纸等促销品

…事实上，几乎是一切！

*广告*

如果你的公司在印刷媒体上做广告，那么你必须把这个网址整合到你的作品中。请记住:

*   美好的老黄页
*   传单和邮件
*   海报和广告牌
*   报纸和杂志

你明白了！

你的网站可以作为你现有广告努力的完美延伸，并为你的业务打开一个直接的窗口。如果有人对你的广告感兴趣，有可能一个拥有大量高质量内容的专业网站会打破“你的竞争对手和儿子私人有限公司”的平衡，后者只列出他们的实际地址。

*店面*

如果你有一个，它不需要打破银行让路人知道你现在可以在网上找到。你当地的印刷店可能会为你制作一些海报，如果你附近有一个招牌作者，可以申请一个报价，把你的网址加到商店招牌上。

***通过电话推广你的网站***

如果你能让所有员工都唱同一首赞美诗，用电话来推广你的网站会非常有效。如果你一天要打或接几个电话，这里有一些事情需要考虑。

将有用的网址放在手边

为你的“前台”员工保留一个有用的 URL 列表。当客户询问“我能获得 x、y 或 z 方面的信息”时，请将他们引向相应的页面。更好的是，把链接用电子邮件发给他们。

*收到一条保留信息？*

定制您的“等待”信息以推广您的网站。来吧，做一个魔鬼:扔掉“绿袖子”，为你的网站制作一个漂亮的广告！

*写一个 F.A.Q.*

撰写和维护常见问题。如上所述，提供一个电子邮件链接给询问出现在你的 FAQ 列表中的问题的人。查看来自 HTML 作家协会的[how to](http://www.hwg.org/resources/faqs/howtoFAQ.html)。

*开展纯网站促销活动*

创建只能在你的网站上找到的每周/每月促销活动。这给了你一个很好的理由告诉来电者网站的网址，当然，如果他们访问的目的是查看报价，他们也会接触到你所有其他的销售计划。

请注意，并非所有来电者都在线，其中一些人可能会反对被重定向。他们抽出时间给你打电话，所以礼貌地用同样的方式回复他们，给他们发一个链接，链接到你更详细的在线信息。好了，现在每个人都开心了！

##### 继续推广你的网站

是的，所以你已经花费了大量的时间、精力和金钱将新网站整合到你的小企业中。你已经在所有移动的东西上贴满了 URL，用大棒打败了任何不使用他们公司电子邮件地址的人，你对结果感到非常满意。那现在怎么办？坐下来放松一下？不可能。

真正的工作从这里开始。建立了一个网站，吸引了一些流量，你现在的目标是获得重复访问和推荐。

正如我在开始时所说的，我不会试图在这里介绍搜索引擎(如果我这样做了，我将永远无法完成这篇文章)。如果你想开始阅读这方面的内容，请从这里开始

你还能用什么？我们将关注的领域包括:

*   重复流量
*   病毒式营销
*   多走一步

***重复流量***

很有可能，你已经花了很大力气来鼓励访问者访问你的网站，所以不要让它成为一次性交易。这里有一些想法，任何人都可以用来最大化他们网站的点击率。

*提供常规、新鲜的内容*

对你的工作感兴趣的人会回来，如果他们相信他们会在后续的访问中找到更多的信息。如果他们回来几次，没有发现新的东西，他们可能永远不会回来。

在一定程度上，你甚至可以通过发布新内容的频率来控制访问者的回访频率。例如，我每天访问 SitePoint.com，因为几乎每天都有新的文章，但是我每隔几周访问 Webmonkey.com 的 T2，因为他们不经常发表文章。

*创建“仅网站”优惠*

当然，这也是新鲜的内容，如果你经常这样做，你会增加重复访问(假设你的特别优惠是真正有益的)。试着把这些促销活动的时间安排得比你的平均重复销售频率稍微高一点:这样你就可以缩小第一次和第二次销售之间的差距。不过，注意不要过度。

*创建邮件列表*

为了告诉人们你什么时候发布了新的内容，尤其是新的内容，向你的订阅者发送一份时事通讯。邮件列表很容易建立，如果做得好，可以给你的生意带来巨大的回报。雅虎！在[雅虎集团](http://www.groups.yahoo.com)提供一项你可能想了解的免费而简单的服务。

*提供论坛*

促进网站用户之间对你的产品或服务的讨论。确保你的员工中有知识的成员留意张贴在那里的信息，并在适当的时候回答它们。

这可能是一件非常棘手的事情，但有几个免费的解决方案，其中一个最好的是 [ezboard](http://www.ezboard.com ) ，几乎任何人都可以设置它。如果你雇佣了一个专业的网页设计师来建立你的网站，他们应该不会有太多的麻烦来为你建立一个更全面的网站。

*将技术文档放到网上*

如果你出售或提供任何需要说明或手册的东西，在网上提供它们的副本。人们很容易丢失手册，告诉他们可以在您的网站上找到当前的说明/文档是一件简单的事情——这当然会让他们同时接触到您所有的其他资料。

***病毒式营销手法***

访问者可能很难找到，所以给你的用户一切机会告诉别人你的网站。病毒营销，尽管它的名字，并不是肮脏或不光彩的事情。病毒式营销最著名的例子是 Hotmail 和布莱尔女巫项目。

这是一个巨大的话题，详细的讨论超出了本文的范围，但是这里有一些你可以立即使用的病毒技术来提高你的命中率。

*提供免费电子邮件地址*

给你的朋友和家人电子邮件地址来宣传你的公司。如果你有一个很酷的域名，他们和 ivana@yourcompany.com 在一起会比和 ivana@123ivana_s@somebigprovider.com 在一起开心得多——而且这个简单的动作会在他们每次发邮件的时候分发你的域名。

如果你有无限的电子邮件地址，为什么不把它们提供给每个访问你网站的人呢？有几家公司会帮你做到这一点，包括 Everyone.net

*电子邮件本页*

在感兴趣的页面上使用一个简单的“用电子邮件发送本页”功能，例如文章和特价商品，这样可以方便地共享您的信息。你的信息不仅被你的访问者所传播，还被他们所认可！

*适合打印的页面*

出于同样的原因，为你的重要页面建立一个“打印机友好”版本的链接。这在办公环境中也很有用，因为重要的页面可以很容易地复制并分发给感兴趣的人，可能在会议期间作为参考。

*为免费网站撰稿*

找到与你的主要关注点相关的网站，写一篇文章或提供一个定期的“专栏”来换取一个链接到你的网站。他们受益于内容，你受益于点击率。当然，尽量选择交通便利的地点。

免费赠品！

钢笔、T 恤衫、马克杯、火柴等。这些都是获取 URL 的好方法。你甚至可以尝试免费明信片，就像我们在丹麦做的那样(叫做“Go-Cards”)。在柜台上放一堆印刷精美的卡片，最好是滑稽或有趣的，上面有你的地址，让人们把它们送给他们的朋友。

你可以在 [Wilsonweb](http://www.wilsonweb.com/webmarket/viral.htm ) 找到一些关于病毒式营销的好资源，别忘了 SitePoint 的[游击营销](http://www.promotionbase.com/subcats/39)信息。

***晋级:多做一点***

你还能做什么？如果你已经阅读了整篇文章，我敢肯定你现在意识到，你可以做什么来推广你的公司网站。发挥你的想象力。

这里有一些进一步启发的想法。

*讨论组和论坛*

如果由于你的业务性质，你对某个特定的主题特别了解，试着找一些与你的主题相关的电子邮件讨论和在线论坛。确保你的帖子底部有一个 4 行签名，包括你的网址，尽可能多地回答问题，提供建议。你很快就会看到一大群新的访问者来到你的网站。

不过需要注意的是:没有人喜欢垃圾邮件发送者。免费提供你的建议，不要在你的邮件/信息中添加你网站的链接，除非你链接的页面实际上回答了发帖者的问题。

*创建一个互惠链接程序*

在你的网站上制作一个充满外部链接的网页，这些链接可以链接到更多的免费信息的主题来源。当你这样做了，浏览这些网站，并发送一个简短的，礼貌的请求，他们把你添加到他们的链接列表中。你可以在这里阅读[更多的互惠链接。](http://www.bannertips.com/howto.shtml)

如果他们不想链接到你，不要担心，当然也不要把他们从你的列表中删除——这是给你的网站增加价值的好方法，也展示了某个主题的一定程度的权威性。

*有限垃圾法*

一位名叫 Steelsun 的私家侦探以潜伏在 [SitePoint 论坛](http://www.sitepointforums.com)的黑暗区域而闻名，他喜欢在拜访办公大楼的客户时在电梯里留下几张名片。他还建议把它们放在那些“赢得免费午餐”的鱼缸里。此外，他发誓通过这种方式获得了不止一对付费客户！

别出心裁的 Steelsun 还表示，将名片放在公共图书馆相关主题的书籍中效果很好。这种新颖的方法只是为了证明，重要的不是有多少人访问你的网站，而是什么样的人。不管你是否赞成这些方法，你都必须把它交给那个人——那就是创造力！
