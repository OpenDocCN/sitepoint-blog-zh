# 快速学习谷歌地图 API

> 原文：<https://www.sitepoint.com/learn-the-google-maps-api/>

(Via [Flash Insider](http://www.flashinsider.com/2005/08/30/flash-based-screencast-full-tutorial-for-google-maps-api/) 、[Google Maps Mania](http://googlemapsmania.blogspot.com/2005/08/google-maps-on-video.html))[SFSurvey.com](http://www.sfsurvey.com/)，一个餐馆评论网站，已经制作了一个样本脚本和附带的[视频教程](http://sfsurvey.com/gmaps/)，让你使用 [Google Maps API](http://www.google.com/apis/maps/) 将 [Google Maps](http://maps.google.com/) 集成到你的网站中。示例脚本让您构建一个位置数据库，您可以单击它在您的站点上显示动态 Google 地图。

好消息是:这个视频教程非常适合那些“有意学习谷歌地图”但没有时间研究这个主题的详细文章的人。

坏消息是:教程中展示的脚本是用 ASP 编写的，使用 Access 访问数据库，而不是像 PHP/MySQL 或 ASP.NET/MSDE.这样更流行的选择。从最佳实践的角度来看，用 Dreamweaver 展示的代码也非常可怕……但是我想如果你使用 ASP，这不是你最关心的问题。

不过，这是一个鸟瞰 Google Maps API 的好方法(如果你不介意的话)。

访问[教程页面](http://sfsurvey.com/gmaps/)开始学习。请记住，教程的两个步骤都有一个视频——我在第一个视频的结尾有点困惑，这听起来像是整个教程的结尾。
