# 亚马逊增加新服务:公共数据集

> 原文:[https://www . site point . com/Amazon-adds-new-service-public-data-sets/](https://www.sitepoint.com/amazon-adds-new-service-public-data-sets/)

亚马逊[今天宣布发布](http://aws.typepad.com/aws/2008/12/paging-researchers-analysts-and-developers.html)一项新的网络服务，旨在促进更容易地访问开放的公共数据集。亚马逊网络服务上的公共数据集将试图让大范围的公共数据可供任何人免费使用。用户可以通过 Amazon EC2 机器映像与数据集进行交互，并且只为他们的计算时间付费——他们不必担心存储、下载或清理实际数据。

亚马逊业务开发经理迪帕克·辛格(Deepak Singh)表示，新计划“大大降低了研究人员和数据分析师访问和使用他们社区中一些最常用数据集的障碍。”

以前，利用亚马逊计划托管的用于研究目的的大型数据集是一件乏味的、多步骤的事情。研究人员需要找到数据，下载数据，然后经常转换、清理或定制成他们需要的可用格式。有时仅仅下载数据对研究人员来说就是一个巨大的障碍。例如，亚马逊上的一个数据集是生命科学项目 [Ensembl](http://ensembl.org/) 的 MySQL 数据库，它维护着“对大量真核生物基因组的自动注释”。他们的数据集重达 650 千兆字节，包含 31，000 个文件。对于许多资源有限的研究人员来说，争论这么大的数据库的技术逻辑将是一个不可逾越的障碍。

现在，这些数据将可以在亚马逊网络服务的整个生态系统中使用，研究人员几乎不需要做任何工作就可以启动和运行。亚马逊希望开发人员能够创建公共工具来分析数据，并将其与其他来源融合在一起，通过让更多人更容易获得数据，该项目将有助于促进创新。

亚马逊现在有广泛的公共数据集，并计划在未来增加更多。

在发布时或发布后不久，亚马逊的服务提供来自恩森布尔和国家生物技术信息中心的人类基因组和 DNA 测序数据；印第安纳大学的化学数据；以及来自美国人口普查局、劳工统计局、运输服务局和经济分析局的经济数据。
