# 召开国际电话会议的技巧

> 原文：<https://www.sitepoint.com/conducting-international-teleconferences/>

在网上工作的一大好处是你不会受到地理界限的限制。你可以和世界上任何地方的任何人一起工作。这为任何在网上工作的人，从自由职业者到小企业主，带来了一系列全新的机会。

由于成本低，即时访问和国际可用性，电话会议是一种非常有吸引力的方式与世界各地的其他人合作。当然，不受地理位置的限制并不意味着你在电话会议中不会面临国际商务关系中常见的其他挑战。不同的语言、交流偏好和文化习俗都可能在与来自全球各地的成员的电话会议的成功中发挥作用。

这里有一些应对这些挑战的技巧，这样你就可以在国际电话会议上有效地交流。

*   ![](img/b0b6f30f52e22201ad5fcc32788ee2b4.png "world-hand")使用时区工具，如[世界时钟会议规划器](http://www.timeanddate.com/worldclock/meeting.html)来协调不同时区的会议时间。
*   发送电话确认和提醒时，请提供所有相关时区的适当时间。
*   请确保您的会议线路支持国际电话。
*   为参与者提供另一种进入会议的方式(例如，如果免费号码不起作用，知道本地拨入号码)。
*   选择一位最熟悉所有参与者的主持人。
*   知道向学员讲话的正确方式。
*   建议参与者尽可能使用座机，以减少静电和其他干扰，这些干扰与口音结合在一起会使彼此难以理解。
*   注意会议期间可能出现的任何其他全球习俗、商务或电话礼仪。
*   预见并适应任何语言障碍。
*   提前了解参与者，包括沟通方式。
*   录制电话，并转录或向所有参与者分发录音，以便每个人对所讨论的内容有相同的理解。

这里列出的建议是面向国际元素的，但不要忘记，成功电话会议的所有其他[建议](https://www.sitepoint.com/tips-for-successful-teleconference/)，从会议计划到会议结束后该做什么，仍然适用。
