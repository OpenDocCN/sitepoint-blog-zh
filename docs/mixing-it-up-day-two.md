# 混合:第二天

> 原文：<https://www.sitepoint.com/mixing-it-up-day-two/>

抱歉，这篇文章来晚了——我有点忙，嗯，混音，直到我上了飞机才开始拼凑。不管怎样，我熬过了陶和沙漠的极度干燥——在拉斯维加斯，我似乎睡不到凌晨 5 点半——熬到了第二天。这是多么美好的一天。

培根之后的第一站是一个启发性的小组，名为“开放问题”，它变成了迈克·斯科洛普夫(Mozilla)、安迪·古特曼斯(Andi Gutmans)、米盖尔·德·伊卡萨(Miguel de Icaza)、诺维尔(Novell/Mono/Moonlight)、罗布·科内里(微软/亚音速)之间关于世界如何走向开放模式以及知识产权、开发者文化和经济如何在其中发挥作用的非常开放和滚动的讨论。我真的不能用散文来公正地描述它，如果你对这类问题感兴趣，你真的应该看看网上直播。

接下来是 Guy Kawazaki 和 Steve Ballmer 之间非常非常有趣的主题演讲。不，舞台上并没有拳脚相加，但史蒂夫确实差点砸了盖伊的 MacBook Air。而且——在你问之前——史蒂夫确实演过《猴子男孩》第二部。但是，与 Ozzie 在第一天充满流行词的预言相比，Steve 非常非常坦率，令人耳目一新，同时也很有趣。从讨论中可以得出的主要结论是，首先，微软非常认真地想要进入搜索和广告领域。基本原理是它们一起有效地扩展——更多的搜索、更多的关键词、更相关的广告服务——这就是未来十年资金的去向。我也期待在未来几个月里看到微软对 SilverLight 的大量投入。最后，甚至盖伊也承认“这是一个不同的今天的微软。”

到目前为止，我参加的最好的会议是 Scott Hanselman 对 ASP.NET MVC 框架的演示。首先，斯科特是一个了不起的演讲者——如果你有机会，抓住他。在观众面前用记事本打字的开场很有趣，也是让座无虚席的观众安静下来的非常有效的方式。我建议查看一下[网上直播](http://sessions.visitmix.com/?selectedSearch=T22)，但是如果你不是网上直播的类型，在这个作者看来有几个主要的收获。首先，从技术的角度来看，新的 ASP.NET MVC 已经将路由和抽象从 MVC 中分离出来，放入单独的程序集——这意味着人们可以在普通的旧 WebForms 项目中利用它们。我必须说，从我看到的有限的比特中，路由比特引起了我对许多目的的兴趣。可能更重要的是，他们将以一种非常实验性的方式来运行这个项目。首先，发布将会是迭代的，并且有一个在 CodePlex 上提供源代码的计划正在进行中。特别是在早期，源代码将是使你的项目免受干扰的无价工具。第二，也是更重要的一点是他们正在征求的反馈数量。这个团队意识到，他们需要现实世界的场景，他们甚至没有梦想过，以使这个新框架实现其承诺，他们要求我们所有人帮助使它变得更好。

但毫无疑问，这一天的亮点是与世界上一些最聪明的人的饮料/晚餐和滚动讨论。网络世界。我终于有机会与合著者乔恩·加洛韦(Jon Galloway)见面，并与罗布·科内瑞(Rob Conery)和史蒂夫·哈曼(Steve Harman)进行了长时间的交谈。我也有机会拜访了 Novell 公司的乔·希尔和米格尔·德·伊卡萨。这些人正在用单核细胞增多症做一些真正令人惊奇的事情，而且都是好人。

最后，所有的会议都可以在 MIX sessions 网站上查看。一定要花时间细读你感兴趣的东西——我只能看到我想看的三分之一，而且我在那里。我要去过一个愉快、轻松的周末，但我正在整理一些总结性的想法。敬请关注。
