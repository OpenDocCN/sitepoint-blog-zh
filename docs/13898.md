# MSN 广告中心:预览

> 原文:[https://www.sitepoint.com/msn-adcenter-a-preview/](https://www.sitepoint.com/msn-adcenter-a-preview/)

众所周知，MSN 正准备挑战谷歌 AdWords &雅虎！点击付费广告领域的搜索营销(前身为 Overture)。

[Searchenginelowdown.com](http://www.searchenginelowdown.com/2005/10/msn-adcenter-beta-preview.html)有一个评论&新的 AdCenter 界面截图和一些 MSN 即将推出的很酷的功能。

亮点包括:

*   按国家、具体城市进行地理定位:非常适合当地企业
*   年龄和性别定位(哇！):只针对年龄足够大、有信用卡的人？女性退休人员？男性青少年？现在，您将能够使用这一新功能。你甚至可以根据性别或年龄组，为每次点击出价不同的金额。
*   时间目标。这很有用，例如，如果你的大量订单是通过电话下的。通过按时间和日期定位，您可以将广告限制在电话代表接听电话的时间内

从表面上看，这个界面比 Yahoo！搜索营销——这可能意味着微软在焦点小组和可用性研究上投入了相当多的时间和资源，以确保他们“做对了”。

如果你是美国的 PPC 广告客户，你可以通过[本页](http://www.webmasterworld.com/forum97/663.htm)上的链接注册 MSN AdCenter 试点计划。祝你好运！
