## 地理定位 API 详细信息

GeoPlugin 提供以下功能:

*   地理位置服务
*   位置统计
*   货币转换器
*   位置代码、符号和缩写
*   地点或邮政编码的纬度/经度

地理位置服务有以下格式和编程语言:

*   java 描述语言
*   服务器端编程语言（Professional Hypertext Preprocessor 的缩写）
*   JSON
*   可扩展标记语言
*   动态服务器页面

完整的细节可以在这里找到:[http://www.geoplugin.com/webservices/javascript](http://www.geoplugin.com/webservices/javascript)
