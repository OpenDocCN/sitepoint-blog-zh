# Linkalicious Friday:新的 Rails、Spinners 和另一个 HTML5 演示

> 原文：<https://www.sitepoint.com/linkalicious-friday-new-rails-spinners-and-yet-another-html5-demo/>

咻！又一个重要的一周(如果你错过了，我们推出了有史以来最畅销的书的第二版[)结束了，所以让我们来看看网络世界发生了什么:](https://www.sitepoint.com/books/design2/)

*   [Rails 3.0.3 本周发布](http://weblog.rubyonrails.org/2010/11/15/rails-3-0-3-faster-active-record-plus-plenty-of-fixes)。尽管它在新特性方面增加很少，但在性能方面有显著的调整，所以 Rails-heads 绝对应该尝试一下。
*   YUI 的博客上有一篇非常有趣的帖子，是关于微调器如何影响我们对性能的感知。事实证明，改变状态(例如，它出现，旋转一点，消失，然后再次出现)的微调器给用户的印象是比一直停留在那里的微调器加载更快。在一个演示中，两个微调器中的一个在稍微延迟后出现(而不是在 Ajax 调用启动后立即出现)，确实感觉更快了。
*   这已经发布了几个星期了，但是如果你错过了，【2010 年南方网络方向的所有幻灯片和音频现在都可以使用了。我强烈推荐看看 Steve Souders 关于前端性能的文章，以及 Dan Rubin 关于 HTML5 和 CSS3 的文章——这两篇文章都很棒。
*   本周早些时候我在博客上写了这个，但是如果你错过了， [Jeremy Keith 在 Fronteers 2010](https://vimeo.com/15755349) 的开幕主题演讲非常值得一看。
*   谷歌 Chrome 团队推出了一本名为 [*的互动在线书籍，讲述了我从浏览器和网络*](http://www.20thingsilearned.com/) 中学到的 20 件事。这是一种对所有互联网事物的介绍，是为门外汉准备的，用 HTML5、CSS3 和 JavaScript 做得非常巧妙。所以如果你能忍受*另一个关于 HTML5 和 CSS3 能做什么的*演示，去看看吧。

祝大家周末愉快！
