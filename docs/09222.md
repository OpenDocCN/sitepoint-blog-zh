# 系统、时间、追加销售和价值——网络商业的字母表

> 原文:[https://www.sitepoint.com/systems-time-upselling-value/](https://www.sitepoint.com/systems-time-upselling-value/)

我们几乎处于网络商务系列字母表的末尾。我希望你喜欢这些智慧的片段和思想的火花。这里还有四个来进一步鼓励你…

**S 代表系统**

我说的系统不是指操作系统。我说的是工作流程和过程。您多久检查一次现有的工作流程，并尝试寻找新的方法来改进您的流程？

我想当然地认为你把这些系统都记录在案了——如果你不这样做，你就是疯了。那么，你和你的团队只是一遍又一遍地重复同样的任务而没有一个新的面貌吗？我发现，对我来说，查看工作流程改进的最佳方式之一是要求每个团队成员确定他们真正想要改变的流程。你问了十个员工，你开始发现一些模式。

自己承担起倾听和行动的责任——你会对改进感到高兴，你的团队也会珍惜这个机会。

**T 代表时间**

不管我们做什么，我们每天仍然只有 24 小时。大自然会照顾好这一点，尽管我们总是试图突破极限，但时间确实是非常有限的。

既然知道了这些，应该怎么花呢？我们都在低价值或低回报的活动上花费了我们生命中的大部分时间——为什么我们一直这样做？花点时间想想什么对你和你的公司真正重要，重新审视一下你是如何度过这一周的——我相信一些小的改变会给你带来大的回报。

**U 代表追加销售**

我们都经历过全球汉堡连锁店，在那里他们会问你是否想要薯条或饮料。对他们来说，接受他们报价的 15%的顾客相当于每天数百万的收入。

现在，我并不是建议在网站上销售牛排刀，但是如果你能提供附加服务来满足顾客的需求，那就提供吧！最坏的情况是他们不接受你。要真诚，不要为了推销相关产品或服务而推销，要确保你的客户首先从中受益。

**V 代表数值**

我早年在生意场上学到的一点是，你应该谈论价值，而不是价格。你看，我知道非常便宜的产品只能使用三个月，我也知道价格是四倍的产品可以使用五年，哪种产品的价值更高？

努力教育你的客户:价值和价格不是同一个意思。少谈成本，多谈你的好处:利用价值销售远比在价格上击败竞争对手更有效。
