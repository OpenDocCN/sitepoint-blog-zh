因此，让我们来看看第二部分中的一些快速花絮，它集中在 CSS 工具和工作流上:

*   74%的受访者在大多数新项目中使用 CSS 预处理器
*   73%的被调查者认为像 Sass 这样的语言是 CSS 最初应该被创建的方式

*   24%的受访者手动书写他们的供应商前缀
*   超过 80%的受访者从未对浏览器或框架提出过 CSS 相关的错误
