# 使用表格的 7 个步骤

> 原文：<https://www.sitepoint.com/steps-useable-forms/>

作为一个尽职尽责的丈夫，我过去常常和怀孕的妻子一起坐在沙发上，看电视史上最轻松的节目:“绘画的快乐”，由已故的鲍勃·罗斯在 PBS 电视台主持。他可以在 20 分钟内轻松地完成一幅风景画，这使得它成为不急着去任何地方的人周六早上的完美食物。如果我没记错的话，每幅画中都至少有一朵快乐的小云。每个星期鲍勃都会说，“我感觉到一朵快乐的小云彩要来了，你呢？”我们确实做到了。

现在，我们来看看互联网的残酷现实。把你的网站想象成鲍勃的一幅风景画。前景中有一个很棒的导航方案，背景中有大量有价值的、写得很好的内容。漂亮。但是你网站的表单呢？它们是“快乐的小表格”，还是因为令人困惑、引人注目、冗长且反应迟钝而赶走潜在客户的障碍？

遵循以下 7 个步骤，让您的表单和用户都满意:

##### 1)抓住要点

你可能要求太多的信息。用户不愿意主动提供信息，尤其是当你要求的内容看起来不相关的时候。例如，如果您有一个时事通讯注册表单，用户通常会乐意提供他们的电子邮件地址，但可能不会提供他们的街道地址、年龄和性别。如果你要求额外的信息，清楚地表明你为什么要求它，你打算用它做什么。

不要强迫用户向下滚动到第一个输入字段。我见过一些表单，它们的输入字段隐藏在一两个介绍性文本的下面。如果你真的需要一部短篇小说来帮助你的访问者理解表单，可以考虑把它放在所有输入框的下面。表单顶部的锚链接可以根据需要将访问者带到文本。

##### 2)提供现场焦点

这个简单的“好东西”在显示表单时将光标直接放在第一个输入字段(`first_name`)中。使用`BODY onLoad`事件，您可以节省您的访问者点击鼠标:

```
 <BODY onLoad="document.forms[0].first_name.focus();">
```

##### 3)添加帮助信息

当然，标有“名字”和“姓氏”的字段不需要太多的介绍，但是不太明显和更模糊的字段可能需要。此外，在存在格式问题的情况下，提供一些指导也不会有什么坏处。例如，对于日期字段，您可以提示用户所需的日期格式:

```
 <INPUT ...  

   onFocus="window.status='Enter date in MM/DD/YY format'"  

   onBlur="window.status=''">
```

你的`<INPUT>`标签中的`onFocus`和`onBlur`属性将分别显示和移除浏览器状态栏中的有用提示。

提供字段级帮助信息的另一种可能更容易看到的方法是“预填充”元素`INPUT`或`TEXTAREA`的值:

```
 <INPUT ... value="- Help info goes here -"  

   onClick="document.forms[0].first_name.value='';">
```

当访问者点击输入框时，您的默认值消失，访问者准备输入。

##### 4)使用表格进行校准

提高表单可用性的另一种方法是在两列无边框的表格中对齐表单元素——第一列的标签带有输入框、选择和复选框等。在第二列。

##### 5)控制输入字段大小

通过改变输入字段的宽度，根据用户的预期反应提供视觉线索。例如，对于“中间名首字母”字段，您可以通过添加 SIZE 属性将默认的宽输入框替换为一个更小的输入框:

```
 <INPUT TYPE=TEXT NAME=middleinitial SIZE=1 MAXLENGTH=1>
```

如果您希望对可以输入到文本框中的文本数量进行一些控制，可以添加`MAXLENGTH`属性。

对于文本区域(较大的多行输入框)，使用`ROWS`和`COLS`属性来控制显示的大小:

```
 <TEXTAREA NAME=comments ROWS=3 COLS=40></TEXTAREA>
```

请务必在各种浏览器中测试这些设置。IE 和 Netscape 对它们的渲染是不同的。

##### 6)提供明确的确认信息

提交表单后，访问者会看到什么？至少你应该清楚地确认表格已经收到——将它们发送到网页上显示的回执信息。你还应该告诉他们接下来会发生什么——“你的反馈将在两个工作日内得到答复”，或者“你的订单将在 72 小时内得到处理”。当你这样做的时候，提供清晰和突出的导航，引导他们到其他合理的目的地。

根据表单的用途，发送电子邮件确认对提交者来说也是一项真正的服务。在您的电子邮件中，重申您的确认信息以及访问者提交的数据。例如，对于订单，在电子邮件中包含客户提交的产品、价格和运输信息。

##### 7)超越快乐的小形式

这是对你在步骤 1-6 中优化表单所做工作的奖励。厌倦了`INPUT`和`TEXTAREA`的老一套造型？通过对表单元素应用样式表，添加一些颜色，更改边框，变得疯狂。

```
 <style> 

   input, textarea, select { 

   font-family: Arial, sans serif; 

   font-size: 20pt; 

   color: #ffffff; 

   background-color: #000000; 

   border-width: medium; 

   border-style: ridge; 

   border-color: #00FF00; 

   } 

   </style>
```

作为一名网络专业人士，我一般不推荐这种愚蠢的行为，但是如果你的网站充满了色彩和古怪，那就去做吧。只是不要告诉自封的可用性警长雅各布尼尔森，你从我这里得到这个。我永远也听不完。

…愿快乐的小表格点缀你网站的天空！
