# 除了社区什么都没有

> 原文:[https://www.sitepoint.com/nothing-but-community/](https://www.sitepoint.com/nothing-but-community/)

我很想说这一年的事情已经接近尾声，但不知何故感觉不是那样。事实上，恰恰相反。我觉得我比一整年都要忙。在让你了解我们最新的圣诞销售，组织社区奖和假日摄影比赛之间，事情有点疯狂。不过这都很好——我喜欢一年中这个时候的活力。

上周，我在博客上写了一篇关于我的新 Mac 电脑的文章，引起了相当多的评论。感谢那些给我建议的人——我现在开始对这件事感到舒服了。

所以…社区。上述奖项的投票已经结束，我们正在统计结果。在接下来的一周左右，请关注[新闻和公告论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=38)的结果。一旦我们最终确定了结果并创建了任何双徽章，我就会颁发徽章(给那些赢得两个奖项的人)。感谢每一个花时间投票的人。

![SitePoint Community](../Images/9bb2be66119f2936eef6bd9dee6364c7.png "SitePoint Community")

摄影比赛仍然进行得很激烈。如果你想参加，你有几个星期的时间来拍一些假期照片。我们刚刚发布了第二套主题，还有一套将于一月中旬发布。你可以在这里查看规则和科目。

我们的[脸书页面](http://www.facebook.com/sitepoint)现在看起来很漂亮，有我们销售的所有礼物，还有总部的圣诞派对照片和庆祝活动。我计划在新的一年里熟悉 iMovie，所以请留意一些有趣的办公室滑稽镜头。同时，如果你想在我们的页面上看到任何东西，请告诉我——我随时欢迎反馈。

我想今天就到这里吧。无论你是否在你生活的地方庆祝圣诞节，我都希望你下周六过得愉快。我计划比一个人吃得更多，喝得更多。
