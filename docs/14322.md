# Koders.com——开源开发者的搜索引擎

> 原文:[https://www . site point . com/koderscom-open-source-developers-search-engine/](https://www.sitepoint.com/koderscom-open-source-developers-search-engine/)

随着各种免费或付费的专业搜索引擎的出现，搜索的垂直化已经有一段时间了。当然，这样做的目的是将小麦从谷壳中分离出来，并获得紧密集中的结果，而不是看到“你的搜索词有 2，398，789 个结果”(尽管可以为最终用户提供更高级的搜索方法培训提出论据，但这是一个完全不同的讨论)。

与此同时，Koders.com 推出了针对开源开发者的测试版搜索引擎。搜索引擎索引了跨越多种许可和语言的开放源代码。一些有趣的功能:

*   查看结果时，您可以选择项目所属的项目，并使用浏览器中提供的计算器来确定构建成本(即构建与购买)。您可以输入您将使用的人工月数、您需要的功能百分比以及您每月的人工成本来进行评估。
*   Koders 即将推出一个企业工具，它将允许大型企业实现一个服务器设备，其索引和搜索逻辑类似于公共搜索工具。服务器可以清点公司的所有源代码，而不管语言如何(在防火墙后面)，并更好地实现代码管理、重用利用和许可合规性/监督。

对 Koder 创始人的采访可以在 Newsforge 阅读。记者 Joe Barr 在采访后发表了一个有趣的评论 Koders 是否应该或将要就搜索过程中发现的代码的使用提供一些许可指导和澄清？
