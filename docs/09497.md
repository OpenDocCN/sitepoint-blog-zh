# 如何利用云计算创造新的商机

> 原文:[https://www . sitepoint . com/how-to-use-cloud-computing-to-create-new-business-opportunities/](https://www.sitepoint.com/how-to-use-cloud-computing-to-create-new-business-opportunities/)

到目前为止，世界上大多数国家已经定义了云计算，以匹配他们自己对新服务和解决方案的使用。组织中每个受影响的利益相关者都有自己对“云”的定义，尽管他们可能不知道它对 it 行业利益相关者的一般意义。

对开发人员来说，为云编码意味着构建新的、但并非完全不寻常的接口来访问和管理数据；也许学习一种新的协议或编程语言。首席信息官将云视为另一种需要管理的供应商关系，他们越来越担心其他人如何管理他们的数据，使其安全，并想知道他们如何与其他关键应用程序共享该数据。面向客户的领导者将云视为快速自动化关键业务流程的解决方案，而没有首席信息官和公司标准强加的开销。纵观所有这些观点，有一点是共同的。利益相关者，不管他们的办公室在哪里，都希望云始终可用。无处不在。

我个人的定义包含了所有利益相关者的观点，我认为普遍性在这个定义中至关重要。但我更感兴趣的是云计算对我的业务以及与我共事的合作伙伴和客户意味着什么。我用一个词来概括——机遇。

# 机会

阅读这篇博客的许多人都是开发者和解决方案提供商，他们将新的服务和解决方案推向市场，并将其整合到自己的业务中。*恭喜你！你在利用这个前所未有的机会！*您正在使用云服务开发使用 Force.com、微软 Azure、谷歌应用引擎甚至 Heroku 的解决方案。您在 Rackspace 托管服务，因此您可以根据客户需求进行扩展。从利用 BaseCamp 上的项目管理工具到使用 Quicken.com 管理内部会计服务和工资，您正在使用云服务来运行您的日常运营。没有云，你将无法运行。

就我自己而言，我的商业成功归功于云带来的机遇。我依靠云来支持我自己的商业网站。我使用云来托管和支持我为我的客户开发的几个网站，这些网站使用支持 WordPress、Joomla！和商业催化剂。我还依赖云来订阅管理服务，以处理工资单、税收和 401K 服务。甚至我的商务旅行也使用云服务进行管理。

关键是云服务让我在更多方面获得了成功，这是我做梦都想不到的。我敢打赌，像我一样，云服务让您专注于自己的竞争优势，从而进一步推动您的成功。

# 商业机会

企业领导人，尤其是在当今的经济形势下，必须迅速行动起来，提高收入和利润，或者降低成本。对于他们来说，这是一件好事，因为许多阅读本文的人都是带着这些目标推出云解决方案的。

大型传统 IT 组织在过去十年或更长时间里一直在集成大型业务系统和数据，以便向其业务合作伙伴提供客户的整体视图。虽然这对于我合作过的许多客户来说是成功的，但这一成就的平衡是一个流程繁重的 IT 组织无法快速添加新功能。数据模型必须与所有用户一起修改和协调，并在计划的、不频繁的发布中进行部署。这限制了业务合作伙伴对不断变化的业务条件做出快速反应或向市场快速部署新功能的能力。

云解决方案对企业领导者很有吸引力，因为它们提供了快速部署新功能的机会，而无需遵循传统 IT 组织强加的严格要求。作为一名首席信息官，我可能不喜欢我的业务合作伙伴将我排除在这种新关系之外，但当我自己的组织无法做出足够快的响应时，我又怎么能责怪他们利用机会增加收入呢？

您是如何使用云服务发展业务的？我想从你的成功中学习，并在未来的帖子中与读者分享。

# 全球机遇

互联网技术在全世界发展中国家的广泛传播创造了更多的机会。来自低成本发展中国家服务提供商的竞争将不可避免地加剧。如果你坐在那里，对欠发达国家可以与你竞争的想法不屑一顾，请记住你通过互联网建立主机和技术基础设施的速度有多快。是什么阻止他们做同样的事情？

假设你是一个中东国家的年轻开发人员，想要学习最新的 Ruby 技术。你可以上网学习 learnable.com 的 Ruby 课程。您使用 Heroku 构建了一个基于 Ruby 的新服务。你需要帮助建立你的一些接口，所以你与埃及 eLance.com 的另一个开发者签约。一旦你的服务准备推出，它是如此成功，你雇用更多的员工来满足需求。听起来很牵强？然后让我把你介绍给我在叙利亚的朋友，他们正在开发高质量的 Ruby 应用程序。

我的观点是，云服务将在发达国家和发展中国家产生许多成功的商业机会。对于云解决方案来说，进入市场的成本比我们在大型机甚至客户端-服务器世界时要低得多。不要求全世界的小型开发商店必须购买昂贵的硬件或软件。所有这些都可以通过互联网以公平、等价的价格获得。但是，低收入地区的商店难道连“公平价格”都达不到吗？也许吧，但是你听说过小额融资吗？

另一方面，想想加纳的小企业，以及他们通过云访问低成本解决方案而获得的竞争优势。通过简单地使用低成本的 CRM 解决方案，他们可以提高客户服务和满意度，并比竞争对手更快地发展业务。发展中国家的企业可以接入互联网，并拥有适度的手段，甚至可以将云作为他们的竞争优势。

您是否是一家云解决方案提供商，有许多外国客户订阅您的服务？如果是这样，我希望收到您的来信，并与其他读者分享您的成功。不要担心泄露秘密。我们将更多地关注你的方法，以此鼓励那些尚未达到你成功水平的人。

自从我进入 IT 行业以来，我们的目标一直是随时随地提供对关键数据和流程的访问。我们现在已经达到了一个技术门槛，可以有效地实现这一点，并且价格在几乎每个组织的预算之内。鉴于现在触手可及的可能性，我很高兴能在这个时候在 IT 行业工作。我希望你也一样，通过评论这些帖子来分享你的知识和热情。期待看到你的评论！

CloudSpring 将关注大大小小的企业如何利用云计算取得成功。我将关注进入市场的新云服务以及企业使用云服务的创新方式。我还将重点介绍小型企业如何构建云解决方案，以进入预算较大的大型企业组织——这是任何云解决方案提供商的关键目标。
