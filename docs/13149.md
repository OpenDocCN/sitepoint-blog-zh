# 如何通过三个简单的步骤获得合同工作(第 2 部分，共 3 部分)

> 原文:[https://www . site point . com/how-to-get-contract-work-in-three-easy-steps-part-2-of-3/](https://www.sitepoint.com/how-to-get-contract-work-in-three-easy-steps-part-2-of-3/)

继我的[上一篇文章](https://www.sitepoint.com/how-to-get-contract-work-in-three-easy-steps-part-1-of-3/)之后，我将谈谈那个至关重要的第一封邮件联系。我要看看邮件中应该和不应该写什么，以及如何组织才能达到最佳效果。

**最初的书面联系**
这第一封邮件是“挑逗”邮件。“迈出你的第一步”介绍。仅此而已。这不是你的简历或一篇关于你上网方法的文章。这是一个“你想知道更多吗？”引言。

请记住，这封电子邮件是一个推销——它是你的文字表述。它让收件人对你是谁和你的能力有了第一印象。所以，它需要看起来和读起来都很专业，但也应该显示出一点你的个性。我知道这是一项艰巨的任务，但稍加努力就能完成。
 **把读者放在心上**
在创建你的电子邮件时，你需要考虑阅读邮件的人。他们在寻找什么，他们会问自己什么:

谁在给我发电子邮件，为什么？
基本上，这是我们都会自动进行的垃圾邮件扫描——是谁发来的，我认识他们吗？它是关于什么的？
因此，你的主题行和“发件人”地址最好既可信又有趣，足以让对方打开邮件。类似于“托比·萨默维尔:可靠的 PHP 承包商”。我建议你在邮件的主题行写上你的名字；这是为了帮助读者将来再次找到你的邮件。

他们拥有哪些我们现在或将来可能会发现有用的技能？你的技能必须是他们想要的，否则他们根本不会感兴趣。你的研究应该已经淘汰了那些不太可能需要你技能的公司。但是，到了最后，他们可能就是不感兴趣。

他们有什么相关的经验？
人们希望看到你有经验。承包商应该是他们所选择领域的专家。公司不想花钱让你学习交易。

他们有多好？让我们看一些工作例子。
你的工作风格和/或质量，是他们所期待的吗？

他们期望得到多少报酬？
这是我**从未与**接触过的东西。这是你以后要讨论的事情。

他们是本地人吗？
作为一般规律；公司会希望在某个时候与你面谈。

这不是一个可能问题的详尽列表，但是你得到了这个想法。牢记读者；它可以帮助你预测可能出现的问题，从而在邮件中回答它们。你希望阅读电子邮件的人能够立即获得尽可能多的信息。但是，你也需要给他们大量的联系方式和链接，以了解更多信息。

**邮件格式**
我推荐纯文本邮件，而不是 HTML。原因很简单，它更有可能以你想要的样子出现。一个不错的方法是在邮件中加入一个 HTML 格式的链接。这是一个进一步展示你才能的好地方，你可以追踪谁访问了，他们什么时候访问的，他们点击了什么，点击了多少次。

**电子邮件**

*   保持简短和甜蜜
*   确保没有错别字或错误，链接工作
*   让其他人校对，并给你一个诚实的评价，他们认为内容和语气
*   如果可能的话，对邮件进行个性化处理
*   不要咄咄逼人或傲慢自大。保持礼貌和专业——记住只需点击一下鼠标就能删除
*   直截了当地说，不要含糊其辞
*   首先解释:你是谁，你想要什么
*   链接到你的作品的例子
*   链接到在线简历
*   询问他们是否可以在几天后打电话给他们进行讨论(将在本系列文章的第三部分讨论)
*   你的联系方式——他们可能有足够的兴趣直接联系你。所以，让联系你变得容易。此外，确保任何联系方式听起来都很专业——fluffy_bunny_rabbit2008@hotmail.com 可能不会给他们留下你想要的印象。

**测试一下。测试一下。测试一下。**
准备邮件并保存副本。给自己发一个，保证好用好看。仔细检查一下，让别人检查一下。别忘了你只有一次机会留下第一印象。所以，任何错别字和断链都不会让你好看。
