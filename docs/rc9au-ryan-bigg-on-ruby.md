# RC9au–Ryan Bigg 在 Ruby 上

> 原文：<https://www.sitepoint.com/rc9au-ryan-bigg-on-ruby/>

我第一次在 RailsCamp 看到 Ryan Bigg 时，他穿着超人装，因为他刚刚被宣布为红宝石英雄。我第二次见到他时，他穿得很正式，这让我很困惑，因为我们是在 RailsCamp。我第三次见到他时，他穿着蝙蝠侠的服装。所以很自然，这将是一次非常有趣的采访。

请介绍一下你自己。

大家好，我叫 Ryan Bigg，目前我正在写一本书，名为 *Rails 3 in Action* ，我正在与 Yehuda Katz 合著这本书。我也在悉尼一家名为 RubyX 的开发公司工作，我们做客户咨询、培训、开发、代码审查等各种有趣的事情。除此之外，我没有太多的生活！我做 Rails，而且只做 Rails。

你从事 Ruby 和 Ruby on Rails 项目有多久了？

我是在 2005 年左右开始接触 Ruby 的，当时只是稍微摆弄了一下，然后我看到了 David Heinemeier Hansson 的 15 分钟教程，这是来自 PHP 的令人震惊的东西。所以我决定在这方面做一个论坛项目，并在我的作品中保持这个主题。我目前正在为 Rails 3.1 开发一个名为 Forem 的论坛引擎。我做 Rails 大约有六年了。

你还学过其他什么语言？

我曾经在 Visual Basic for Applications 中工作过，那很有趣，比如烧 Excel 宏之类的东西。然后我转到了 HTML/CSS、PHP，没有真正接触 JavaScript 方面的东西，所以我只知道一点点 JavaScript，我不是这方面的专业人士。然后我看到了 Rails，说“这是适合我的语言”，在过去的六年里，我一直以它为生，所以它肯定是做了一些好事。

我们之前确实谈过这个问题，但是你现在在哪里工作？

这个地方叫做 RubyX，由一个叫 Mikel Lindsaar 的人管理，他以编写 ActionMailer 和 mail gem 而闻名，他是如何开始编写 Mail gem 的，他被任命为 TMail 的维护者，他不断地向 TMail 提交补丁，维护者说你在这里为我做工作，只要有库就行了。所以他接手了这个项目，并意识到这是一种废话，并决定改为写邮件。现在已经有 125 万次下载了。

我们做客户咨询，培训是我真正喜欢的。当你在教别人的时候，他们的眼睛亮了起来，他们意识到并且理解了一些东西，这对我来说是世界上最好的感觉。

Ruby on Rails 有什么特别的地方让你知道你为什么这么做吗？

这种语言是我迄今为止遇到的最好的语言。我用 Visual Basic 做应用程序，那太可怕了，PHP 是，嗯，我曾经看过一篇文章说“如果编程语言是船会怎么样”，C 是核潜艇，Ruby 是这艘小快艇，PHP 是用绳子绑在一起的小竹筏，我认为这非常贴切。我看着自己写的代码，不敢相信是我写的。用 Ruby 编写代码的范例和方式要好得多。

但真正让我留在这里的是我们这个令人敬畏的社区。我们现在在 RailsCamp，大厅里有 150 个人，他们都喜欢 Ruby 和 Rails，或者想了解更多。有很多人也在做 JavaScript，这很好。但是我要去铁路公司，我要去红色。在新加坡的 Ruby 大会上，我见到了所有这些有趣的人，他们都在做 Ruby on Rails，他们都很友好。我不知道 Ruby 社区到底是什么让每个人都如此友好和热情。任何人都可以接近任何人，说“嘿，看，我有这个问题，帮帮我！”

现在是百万美元问题，你对任何对 Ruby 感兴趣或想换成 Ruby 的人最大的建议是什么？你希望有人告诉你的任何事，可以帮你解决六个月的问题。

基本上，要达到一个极端的、高级的水平需要很多努力，任何技能都是如此。我以前练过武术，这个家伙已经练了 40 年，三个月前才拿到黑带。他 65 岁，一个老白人，做后空翻，前空翻和踢腿。不过话说回来，耶胡达·卡茨早在 2006 年、2007 年就加入了 Rails 社区，比我晚了几年，他完成了 Bundler，目前正在做 jQuery，他正在做 Sprout Call，一开始就使用了一些 Ruby 的东西。我想说的是，坚持学习，不要害怕犯错，因为你总有犯错的可能，人们会提醒你，不要害怕自我挫伤，因为你一定会从中学到东西。
