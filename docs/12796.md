# 研究:孩子们不再热衷于社交网络

> 原文：<https://www.sitepoint.com/study-kids-no-longer-into-social-networking/>

Logicalis 对英国 13 至 17 岁青少年的年度调查发现，社交网络的使用正在下降。这项研究发现，78%的英国年轻人在使用社交网络时不再透露个人信息，并且越来越了解自己的数字指纹。就其本身而言，这是一个好消息，因为这意味着将有更少的年轻人成为网络掠夺者的受害者。

但这对社交网络来说可能是个坏消息。研究发现，许多年轻人已经完全停止使用社交网站，大多数人表示他们更喜欢与同龄人面对面的交流，而不是通过脸书这样的网站交流，这是英国最受欢迎的社交网络。46%的人表示，社交网络对他们来说已经变得不那么重要，他们使用在线社交网络的频率也越来越低。

在使用脸书的 1130 万英国人中，13-17 岁的青少年仅占 87.3 万人。然而，18-25 岁的人构成了近 500 万英国脸书用户，这表明在英国的中学后学校的使用很高，正如在我上大学的美国一样(拥有脸书账户是一种成人礼)。最年轻的用户一旦到了大学年龄，对社交网络的兴趣就会下降，看看这种趋势是否会持续下去，这将是一件有趣的事情。

年轻的网络用户总是变化无常的。从 Geocities 和 Tripod，到 Friendster，到 MySpace，再到脸书，年轻的社交网络用户经常在他们跳到新的时尚事物时留下一串网站。社交网络本身是否会被年轻用户抛弃？Logicalis 认为是这样的，在他们更喜欢面对面的交流而不是数字聊天之后，称这些年轻用户为所谓的“实时一代”。

注意:去年冬天有一些关于英国脸书交通量下降的说法，但事实证明这主要是季节性的。
