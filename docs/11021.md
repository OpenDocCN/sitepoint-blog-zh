# Palm 移交给惠普

> 原文：<https://www.sitepoint.com/palm-hands-over-to-hp/>

智能手机和 PDA 制造商 Palm，Inc .在本月早些时候强制出售后得救了。在联想和 HTC 退出后，形势看起来很暗淡。Pre 和 Pixi 用户现在可以放心了——惠普已经签署了最终协议，将收购 Palm。

根据[惠普新闻稿](http://www.hp.com/hpinfo/newsroom/press/2010/100428xa.html):

> 惠普和 Palm，Inc .今天宣布，双方已达成最终协议，惠普将以每股 Palm 普通股 5.70 美元的现金或约 12 亿美元的企业价值收购 Palm，Palm 是一家采用 Palm webOS 移动操作系统的智能手机提供商。该交易已经获得惠普和 Palm 董事会的批准。
> 
> 惠普的全球规模和财务实力与 Palm 无与伦比的 webOS 平台相结合，将增强惠普更积极地参与快速增长、高利润的智能手机和互联移动设备市场的能力。Palm 独特的 webOS 将使惠普能够利用真正的多任务处理和跨应用程序共享最新信息等功能。

Palm 首席执行官乔恩·鲁宾斯坦预计将留在公司。他评论道:

> 我们很高兴惠普对 Palm 的技术领先地位投下了信任票，推出了 Palm webOS 和 Palm Pre 等标志性产品。惠普长期的创新文化、规模和全球运营资源使其成为快速加速 webOS 发展的完美合作伙伴。

这笔交易给两家公司都带来了巨大的好处:

*   棕榈将会幸存。他们努力在苹果 iPhone 主导的市场上销售广受好评的 Pre。
*   虽然惠普与微软在提供基于 Windows 的智能手机方面有着长期的合作关系，但 Palm 交易给了他们自己的操作系统。

惠普宣布他们将对 Palm 的 webOS 平台进行大量投资。我们可能会看到它出现在一系列移动设备中，包括上网本和类似 iPad 的平板电脑。

这对 web 开发人员来说是个好消息，因为 webOS 应用程序可以用 HTML、CSS 和 JavaScript 编写。有一个适用于 Windows、Mac 和 Linux 的免费 SDK，并且没有你在其他设备上发现的专有的“照我们说的做”批准废话！
