# 这是一个做专业网站应该做的事情的网站！

> 原文：<https://www.sitepoint.com/here-is-a-web-site-that-does-what-a-professionals-web-site-is-supposed-to-do/>

[http://www.qmacshelp.com](http://www.qmacshelp.com)

看看上面的链接，你会发现一个网站做了专业网站应该做的事情。

马克本身并不是网页设计师。他是一名专业人员，为使用名为 QMACS 的软件程序的医疗保健组织生成报告。

但是他的网站是每个网页设计师都可以效仿的:

1.非常集中的目标市场。

2.伟大的网址。

3.伟大的标题吸引他的观众，让他们阅读更多。

4.极棒的免费优惠，让潜在客户在购买前测试他。

5.令人印象深刻的证明，解释了为什么马克是独一无二的。

6.提供一份免费报告，表明他教育人们了解他所做的事情，并确立他作为专家的地位。

7.文案关注的是用户，而不是马克有多棒(尽管这一点很明显)。

我知道你们中的一些网页设计师会问:获奖的设计在哪里？花式飞溅页面在哪里？flash 动画在哪里？

好吧，我说:谁在乎？是的，作为一名网络专业人员，你需要融入一些不同的东西。当然了。但是太多的你的网站(你自己的和你的客户的网站)没有利用这些简单的策略。结果，你无法吸引潜在客户的注意力，看起来就像其他网站设计师一样。

马克的简单网站得到了结果。从中吸取教训…
