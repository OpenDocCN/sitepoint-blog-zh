# 让你的 jQuery 代码看起来很漂亮

> 原文:[https://www.sitepoint.com/jquery-code-pretty/](https://www.sitepoint.com/jquery-code-pretty/)

好吧，我在网上寻找一个像样的工具，可以整理我的 jQuery 脚本，使它“看起来很漂亮”，因为我编码很快，讨厌混乱的代码！我通常使用旧的[记事本++](http://notepad-plus-plus.org/) 并进入 ***文本特效>文本特效编辑>重新输入 C++代码*** 这通常工作得很好！

![textnotepad](../Images/81a527709781502799d0f3cda81519b1.png "textnotepad")

不过，我发现了一个很棒的在线工具，叫做 [Javascript unpacker 和 beautiier](http://jsbeautifier.org/)。这个工具实际上让你的 JS 源代码看起来很漂亮，值得一提。
