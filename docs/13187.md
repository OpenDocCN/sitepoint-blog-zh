# 面向 Mac OS X 的 Java 6 又回来了

> 原文:[https://www . site point . com/Java-6-for-MAC-OS-x-back-on-the-radar/](https://www.sitepoint.com/java-6-for-mac-os-x-back-on-the-radar/)

在 Mac OS X 10.5 (Leopard)发布以及随之而来的 Java 开发者对苹果悄无声息地移除 Mac 版 Java 6 预览版的喧嚣近两个月之后，苹果[也悄悄地发布了 Java SE 6 开发者预览版 8](http://arstechnica.com/journals/apple.ars/2007/12/19/apple-finally-releases-java-6-developer-preview-for-leopard) ，这是第一个运行在 Leopard 上的 Java 6 开发者预览版。

许多开发者担心，删除之前的预览版标志着苹果有意忽略 Java 6，而继续对 Java 5 进行小的调整。现在看来，乐观主义者是对的:苹果只是需要更多的时间来完善 Java 6，所以它后退一步，在继续开发 Java 6 之前，为 Leopard 版本对 Java 5 做了一些改进。

事实仍然是，如果苹果只是以一种有意义的方式与它最亲密的盟友——软件开发商——交流，我们就不会被留下来玩这些关于该公司意图的猜谜游戏。
