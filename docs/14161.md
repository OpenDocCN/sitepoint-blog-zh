# 手机 Java:下一代

> 原文:[https://www . site point . com/Java-for-mobile-phones-the-next-generation/](https://www.sitepoint.com/java-for-mobile-phones-the-next-generation/)

[Java 社区进程](http://www.jcp.org/) (JCP)已经批准了开发[移动信息设备规范 3.0](http://www.jcp.org/en/jsr/detail?id=271) (MIDP3)的初步计划，这是规范手机上 Java 应用和游戏的新版本标准。

尽管该标准仍处于开发的最早期阶段，但已获批准的 Java 规范请求提供了一瞥即将进入该平台的新特性:

*   同时运行多个 MIDlets(移动应用程序)
*   在后台运行 MIDlets
*   设备启动时自动启动 MIDlets
*   多屏幕和大显示屏支持
*   更富表现力和可定制的用户界面
*   更丰富、更高性能的游戏

最终规范的公开评审期定于 2006 年 1 月，最终规范将于 2006 年 5 月获得批准。我预计 MIDP3 手机将于 2007 年左右在 Q1 大规模上市。

同时，MIDP2 也是一个不错的平台。我自己在业余时间也在用它做一些工作，[将一个为诺基亚 S60 v1 手机(7650/3650)编写的开源游戏](http://j2mebomber.sf.net/)移植到所有主要制造商支持的 MIDP2 标准。
