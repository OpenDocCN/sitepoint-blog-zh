# 域名:让它们令人难忘

> 原文:[https://www.sitepoint.com/domain-names-make-them-memorable/](https://www.sitepoint.com/domain-names-make-them-memorable/)

当我为新产品或企业名称制定客户策略时，我的首要任务之一就是检查域名是否可用。
因为我在澳大利亚，所以我首先查看. com.au，但我也建议收购。com 地址和其他顶级域名。

事实上，当我们无法保护域名时，我们已经调整了产品的名称；它们对你的品牌和网上可搜索性真的很重要。因此，你可以想象，当我在欣赏总部位于纽约的品牌建设机构 Collins 的一些精彩作品时，却发现他们的域名是 collins1.com，我是多么困惑。

现在，我不想在这里挑他们的毛病，因为我相信他们做了一些伟大的工作，但是为什么你会选择在你的名字前加一个数字来保护域名呢？

这激起了我的兴趣——有哪些可用的域名对他们有用？我花了十分钟，发现了以下可用的域名:

*   collinsagency.com
*   wearecollins.com
*   collinsny.com
*   collinsnewyork.com

没有办法知道这些是否都是他们刚建立网站时拍的，但这似乎不太可能。要不是我为了写这篇文章写下了他们的域名，我早就忘了。难道他们没有更好的选择吗？

说到域名，纽约设计工作室 Open 的域名一直让我印象深刻。显然 open.com 被选中了，所以他们选择了 notclosed.com。这是一个令人难忘的域名！
