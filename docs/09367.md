# 浏览器趋势 2011 年 9 月:有人能阻止 Chrome 吗？

> 原文:[https://www.sitepoint.com/browser-trends-september-2011/](https://www.sitepoint.com/browser-trends-september-2011/)

尽管正在发布 Firefox 版本，但浏览器市场在夏季的几个月里仍然保持平静。

这是最新的统计数据。我已经改变了表格，所以火狐 4，5 和 6 合并成一个；分析单独的数字没有什么意义，因为大多数用户会随着新版本的出现而更新浏览器:

| 浏览器 | 七月 | 八月 | 变化 | 亲戚 |
| ie9.0 | 7.27% | 8.05% | +0.78% | +10.70% |
| IE 8.0 | 26.30% | 25.68% | -0.62% | -2.40% |
| IE 7.0 | 5.45% | 5.07% | -0.38% | -7.00% |
| IE 6.0 | 3.42% | 3.09% | -0.33% | -9.60% |
| 火狐 4.0 以上版本 | 17.66% | 18.10% | +0.44% | +2.50% |
| 火狐 3.6- | 10.30% | 9.39% | -0.91% | -8.80% |
| 铬 | 22.17% | 23.17% | +1.00% | +4.50% |
| 旅行队 | 5.15% | 5.18% | +0.03% | +0.60% |
| 歌剧 | 1.66% | 1.67% | +0.01% | +0.60% |
| 其他人 | 0.62% | 0.60% | -0.02% | -3.20% |

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示了市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 9.6%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

IE9 又过了一个好月。它的进展仍然相对平稳，但如果微软想要大规模采用，有两个解决方案:

1.  向每个人提供运行 Windows 7 的硬件、安装、迁移和培训服务。免费的。
2.  或者，发布一个与 XP 兼容的 IE9 版本。其他供应商支持 XP，并且仍然设法提供诸如硬件加速之类的奇特功能。和 CSS3 文本阴影。

IE 的整体下滑速度本月有所放缓，但我怀疑这只是统计上的暂时现象，因为商业用户正在享受暑假。

火狐 4/5/6 正在崛起，但没有火狐 3/2/1 衰落的速度。虽然快速发布的版本大多很好，但用户对 Mac OS 上的插件兼容性故障和[内存使用问题感到沮丧。Mozilla 正在解决这些问题，但他们正在失去可能永远不会回来的用户。](https://www.sitepoint.com/ultimate-browser-battle/)

歌剧和游猎没什么可报道的。两种浏览器都取得了适度的收益，但都没有点燃市场。

那就只剩下 Chrome 了。同样的故事:使用率继续以每月 1%的速度增长，有时甚至更高。如果目前的趋势继续下去，Chrome 将在 2011 年 12 月超过 Firefox。这种情况已经在英国发生，Chrome 以 23.41%领先于 Firefox 的 21.75%。

个人喜欢 Chrome，定期推荐或安装该浏览器；它快速，简单，稳定，更新没有大惊小怪。然而，我主要使用 Firefox(在 Windows 7 上),因为它有一系列用于冲浪和开发的基本插件。我以为其他人会有同样的想法，但是，在 Google+ 上问了这个问题之后，似乎没有。开发者正成群结队地转向 Chrome。Mozilla 正在失去曾经推广 Firefox 的技术传播者。

## 移动浏览器的使用

[桌面浏览器占 web 活跃度的 92.88%](https://gs.statcounter.com/#mobile_vs_desktop-ww-monthly-201108-201108-bar)。剩下的 7.12%是移动接入，很明显更多的人使用手机进行一般的网络浏览。他们主要使用的应用程序有:

1.  opera Mini/Mobile——21.61%(下降 0.46%)
2.  安卓系统——19.72%(增长 1.55%)
3.  诺基亚浏览器——16.99%(下降 0.11%)
4.  iPhone——14.91%(下降 0.19%)
5.  黑莓——11.64%(下降 0.66%)

请注意，地区差异很大:

*   在美国和加拿大，安卓以 34.2%的份额位居榜首，iPhone 以 26.1%的份额紧随其后。歌剧占比不到 4%。
*   iPhone 在欧洲最受欢迎，占 33.7%，Android 次之，占 23.7%。
*   对于大洋洲来说，iPhone 几乎垄断性地领先 56.7%。安卓以 19.4%的份额远远落后。
*   在亚洲、非洲和南美洲，歌剧和更便宜的诺基亚设备占据了主导地位。

请记住，这些数字是从互联网访问中整理出来的，而不是销售趋势。与使用最新 3G 手机的用户相比，使用老款手机的用户不太可能使用网络。也就是说，在发展中国家，用户可能无法使用 PC，因此移动设备是唯一的选择。
