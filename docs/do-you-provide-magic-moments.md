# 你提供神奇的时刻吗？

> 原文：<https://www.sitepoint.com/do-you-provide-magic-moments/>

嗯，我从华特·迪士尼巡游回来了，在感冒和流感季节接触了 500 个孩子后，我刚刚生病了。

尽管如此，假期还是很棒，迪士尼的人可以给世界各地的专业人士上课。他们热衷于细节，热衷于尽力取悦客人。离开船，走进大陆上的一家商店是令人沮丧的，因为这些商店的帮助比游轮上的服务人员要少得多，友好得多，知识丰富得多。迪士尼的工作人员表现得好像服务是一种荣誉。大多数其他服务人员看起来很无聊，很烦恼，好像他们不能等待另一个沉闷的一天结束。

我将这一点与许多 IT 专业人士进行了比较，他们有时表现得不太喜欢与客户面对面工作。事实上，在离开游轮的前一天，我不幸与我的一个 IT 客户和他们的一个客户通了电话。当客户提出合理的问题时，这位 IT 专业人士打断了她，在整个通话过程中表现得不耐烦，举止浮夸傲慢，最后还让双方的关系倒退了不少。

如果你没有读过，迪士尼系统的支柱之一是神奇的时刻。迪士尼人会尽力找到与顾客互动的方式，这样他们就可以用优质的服务让顾客刮目相看。如果你提到这是你的周年纪念日，他们会给你一个特别的蛋糕。如果你的孩子似乎喜欢某个特定的迪斯尼人物，他们会想办法让你的孩子看到那个人物。真是太神奇了。

这里有一整套系统，包括筛选人员、激励人员和培训人员。但是这个博客的大多数读者应该已经被筛选和激励了，因为他们是自己公司的所有者。

那么，你有系统的方法给你的客户提供神奇的时刻吗？
