# 为什么你应该提高你的写作技巧…以及如何开始

> 原文：<https://www.sitepoint.com/how-to-improve-writing-skills/>

![typewriter](img/4a221b9a93abede49ac3ebae2a44eac7.png "typewriter")是的，写作技巧。如果你想成功，你需要能够写好文章。不管你做什么类型的工作，写作都是其中的一部分。你可以写电子邮件、提案和项目计划，也可以写博客、销售文案和其他营销材料。无效和糟糕的写作会对你的工作、声誉和服务客户的能力产生负面影响。

以下是你需要写好文章的更多理由:

*   有效的写作是提升自己的最好方法之一。
*   文笔好能让人更快信任你。
*   你写作的方式很能说明你是一个怎样的人。
*   当你写下一些东西时，它比你头脑中的想法更具可操作性。
*   你如何写作告诉别人你的智力和专业知识。
*   你写得越好，你的营销文案就越有效。
*   有效的写作可以是有说服力的，并吸引行动。
*   你写的方式可以帮助别人理解更多，能够形成自己的问题。

提高你的写作技巧可能需要一些时间，但它将使你受益终生。你不仅会呈现出自己更专业、更知性的一面，还能赢得更多业务。你会花更少的时间解释和澄清，更多的时间工作和赚钱。

以下是一些开始的方法:

*   拼写检查和语法检查一切
*   保持简短
*   根据格式适当写作
*   多读一些书、杂志、文章和博客
*   从一个大纲开始，然后从那里展开
*   上一堂有效的写作课
*   停止使用缩写
*   与风格一致
*   校对或让别人帮你做
*   使用同义词库
*   避免太正式或太随意
*   了解你的受众
*   走开一会儿，然后再复习
*   练习不同格式的写作
*   大声朗读你写的东西

当你写作的时候，你会想到其中的几个项目？你的写作有效吗？
