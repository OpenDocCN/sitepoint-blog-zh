# 没有比自由控制更好的了！

> 原文:[https://www.sitepoint.com/nothing-like-free-controls/](https://www.sitepoint.com/nothing-like-free-controls/)

需要菜单吗？你网站的论坛？联系人页面的富文本框？

这里有一个 ASP.NET 免费控件和源代码包的快速列表，我不能没有它们:

**UI: [TimothyHumphrey。web controls](http://timothy.humphrey.name/thwc/)T3】**

编码简单，部署简单，Timothy Humphrey 开发了一组控件，提供跨浏览器菜单(甚至可以从 CSS 构建)、框架和反垃圾邮件链接。代码输出简洁整洁，对于菜单控件来说是相当难得的！

**论坛软件:[ASP.NET 论坛](http://www.asp.net/Forums/Download/)T3】**

由微软编写，并在 ASP.NET 官方网站上使用，这个论坛软件安装快速，功能丰富，可以与任何 PHP 板一起使用。

**输入:[自由文本框](http://www.freetextbox.com/)T3】**

这是我最喜欢的控制。光滑，非常强大，现在可以在 Mozilla 和 Firefox 中工作，FreeTextBox 是为您的用户提供的所见即所得编辑器。你甚至可以从控制打印，保存到 RTF，打开一个 RTF，检查它是一个编辑器在你的网页上只有一行代码。

**写博客:[das blog](http://www.dasblog.net)T3】**

DasBlog 是 newtelligence 提供的免费博客解决方案。有了你所期望的所有特性，比如引用通告、pingbacks、Blogger API 实现和一个 zippy 缓存引擎，这就是把你的信息传递给人们的工具！
