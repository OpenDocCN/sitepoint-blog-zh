# FullCodePress 更新:澳大利亚队的挑战

> 原文:[https://www . site point . com/full code press-challenges-team-Australia/](https://www.sitepoint.com/fullcodepress-challenges-team-australia/)

![Team Australia's mascot](../Images/5399aa6a6ed6c839360de177179359bf.png "Team Australia's mascot")
*Raena 在由 SitePoint 赞助的[全代码出版社网站一日赛](http://fullcodepress.com/)上报道。*

我们现在几乎已经完成了全程 24 小时马拉松的一半。澳大利亚团队正在为新南威尔士州的[残疾歧视法律中心、](http://www.ddlcnsw.org.au/)使用曾经流行的 WordPress 建立一个闪亮的新网站。[评委们已经可以看到新西兰黑人守则的网站正在成形，](http://www.fullcodepress.com/2009/05/12/evolution-of-a-website-8pm/)但是澳大利亚队发生了什么呢？早些时候，我采访了 Elle Meredith，她是 Aussie 团队中的一名设计师和 Rails 开发人员。

为什么这么安静？“我们仍然没有一个明确的设计，”Elle 说。“我们有一个想法，亚历克斯仍在努力，但当我们需要的一些东西还没有到位时，继续下去有点困难。”

澳大利亚人面临的另一个挑战是，他们的客户彼得是个聋子。一整天都有一名奥斯兰翻译在场协助，但现在其他人都回家了，团队与他沟通变得有点困难。“这不是最简单的交流方式，”艾尔说。“但是，至少他还醒着！”

当然，现在就考虑放弃 Oz 队是愚蠢的——这些有才华的小伙子和姑娘们有很多斗志，我们不能忽视澳大利亚人的回归。两个队在比赛的这个阶段仍然非常有信心。敬请关注——将会有更多更新！
