# 为什么你需要写一份使命陈述

> 原文：<https://www.sitepoint.com/personal-mission-statement/>

今天，我敦促你写一份使命宣言。事实上，我的观点是你需要两者，一个使命宣言和一个愿景宣言。如果你是一个小企业主，一个自由职业者，一个兼职兼职兼职者，有一份“固定”工作，或者即使你失业了——好吧，所有人——你可以从花些时间来起草这两份声明中受益。让我解释一下。

传统上，使命陈述是一种工具，企业或组织将其作为业务规划过程的一部分，以概述其存在的目的。愿景声明也是如此，但重点是确定组织所持有的推动行动的价值观。

作为一个个体，你已经有了目标，做你所做的事情的目的和引导你度过一生的价值观，但是将这些项目形式化为有形的白纸黑字的陈述可以完成许多事情。以下是使命陈述和愿景陈述可以润色你的演讲，帮助你在个人和职业上取得更多成就的几种方式。

## 合成你是谁

使命和愿景陈述包括关于你的最重要的信息，你做什么，以及在你的生活和工作中指导你的价值观。精心制作的陈述概述了这些细节，不仅可以作为您业务或职业规划的一部分，还可以作为电梯推销的基础，是您营销活动中反复出现的主题，是您决策的指南。

## 给你专注

虽然使命和愿景陈述并不完全是目标驱动的，但两者都可以成为你目标规划过程中使用的工具。由于这些陈述获取了关于你的重要信息，并将其归结为几句话，你可以随着时间的推移回到这些陈述，重申你的目的，并在偏离轨道时重新关注你要实现的目标。此外，明确你为什么要做你正在做的事情可以给你一个继续前进的目标和动力。

## 简化决策过程

在某些情况下，明确的使命和愿景可以帮助你做出正确的决定。这并不总是那么容易，但即使是更困难的决定，你的陈述也可以提醒你什么对你来说是重要的，至少，指引你在做出选择之前需要问自己的问题。

## 让你负责

正如你的使命和愿景陈述可以帮助你做出符合你的目标和目的的决定，当你走错方向时，它们也可以让你承担责任。当你回顾你的陈述并意识到你偏离了轨道时，你可能会决定花些时间找出哪里出了问题，或者你会意识到是时候更新你的陈述以反映你生活和工作的新方向了。

重要的是要记住，使命宣言和愿景宣言都是工具，应该随着你的工作、生活和目标的变化而发展和变化。一旦你写好了，就要定期检查它们的准确性和意义。如果你有一个标准的过程来设定和检查你的目标，那么同时分析你的使命和愿景来保持你所有的计划和行动的联系和同步是有意义的。
