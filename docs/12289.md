# 通过 Skype 交易，诺基亚再次行动

> 原文:[https://www.sitepoint.com/nokia-skype/](https://www.sitepoint.com/nokia-skype/)

![nokiaskype](../Images/a5b4ecf9b5591535a63702fb71ef3c2e.png)诺基亚昨天刚刚在[新闻中](https://www.sitepoint.com/more-app-stores/)宣布了他们即将推出的应用商店。今天，[公司宣布](http://uk.reuters.com/article/technologyNewsMolt/idUKTRE51G1FE20090217)将 Skype 整合到其最新产品 N-97 中。当然，Skype 是许多人首选的网络语音交流平台。然而，自 2005 年易贝收购该公司以来，Skype 一直走在通往生存的艰难道路上。现在，诺基亚和 Skype 正试图利用超过 4 亿的 Skype 用户，这一举措有望成为移动网络的标杆。

用应用赚钱正在成为一种趋势，这是理所应当的。我们已经看到所谓的 Web 2.0 创业公司来来去去，在这种特别不稳定的经济环境下，正如俗话所说，许多人很快将不得不“要么投入，要么闭嘴”。这一消息进一步表明，一些人正在“努力”利用大量的潜在客户(在这种情况下，Skype 的 4 亿用户)，通过在优秀的电子平台上集成应用程序。

##### 自由谈话？

由于担心收入损失，移动运营商一直对与 Skype 合作持谨慎态度。然而，Skype 通过电话运营商 3 与哈奇森·惠姆波公司(Hutchison Whampoa)的成功合作，使 Skype 成为一个赚钱的公司。与 Skype 有关联的 3 家公司收入增长 20%的建议，显然对诺基亚有影响。Hutchinson 的方法是通过让用户使用 Skype 的品牌和价值来利用他们的支持服务。Skype 的明显吸引力在于其“免费性”，在这种情况下，这是企业/消费者关系两端的一个关键妥协。

##### 应用程序作为商品

对于如何向数百万用户推销产品，诺基亚、苹果和微软显然有一套更完善的模式。这种明显的“优势”部分源于他们不仅作为优秀设备生产商的潜力，还源于这些设备支持无数应用的内在能力。

然而，其他网络实体很大程度上基于不太有形的价值，“硬”电子产品提供了显著的优势。移动公司能否从“通话时间”收入转变为基于应用的收入模式仍有待观察，但之前运营商 3 的 Skype 成功显示了巨大的潜力。关于最近的这些举措是否只是短期策略，以及它们是否会造成更深远的负面影响，还存在争议。从诺基亚和 Skype 的角度来看，似乎不会有什么损失，反而会有很大收获。
