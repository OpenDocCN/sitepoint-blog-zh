## 1.用 CSS 和 JQuery 实现图像分割效果

在本教程中，我们将制作一个图像分割效果。那是什么？这类似于滑动门效果，其中图像滑动到左侧或右侧，并显示其后面的文本，但与此不同的是，效果看起来像图像被分成两半，一个向左，另一个向右。

## 8.CrossSlide jQuery 插件

一个 jQuery 插件，用 JavaScript 实现一些常见的幻灯片动画，传统上只有 web 开发人员可以通过 Adobe Flash 或其他专有插件使用。CrossSlide 构建于 jQuery 动画工具之上，因此它像 jQuery 本身一样可以跨浏览器移植。
![CrossSlide-jQuery-Plugin.jpg](../Images/12640bdf036d8f147f650a88790b4a26.png)

## 9.Crop jQuery 插件

裁剪插件获取一个 IMG 元素并将它们裁剪到给定的尺寸。结果是一个 DIV，其背景图像具有高度、宽度和偏移量。
![Crop-jQuery-Plugin.jpg](../Images/35517c1617592ba56dca8d8279e69c75.png)
[来源](http://remysharp.com/2007/03/19/a-few-more-jquery-plugins-crop-labelover-and-pluck/#crop)

## 10.imgAreaSelect jQuery 插件

一个 jQuery 插件，用于选择图像的矩形区域。它允许 web 开发人员轻松实现图像裁剪功能，以及其他用户界面功能，如照片笔记(就像 Flickr 上的那些)。
