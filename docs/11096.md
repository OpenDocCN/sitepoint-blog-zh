# 春天的羽化…在秋天。

> 原文：<https://www.sitepoint.com/spring-ecleaning-in-autumn/>

在新西兰，随着秋天的到来，白天变得越来越短，越来越冷。这似乎是一个奇怪的春季大扫除时间，但有时成堆的灰尘会变得太大，以至于无法扫到地毯下。

一年中的这个时候总是让我有一种打扫的心情。可能是早晨阳光透过我的落地窗照射进来的角度以一种不可忽视的方式显示了灰尘。不管是什么原因，我正在利用这种能量，并将它应用到家庭和工作中。对我来说，这并不意味着打扫办公室，而是清理 SitePoint 论坛。

除了清理一些我每周都去的帖子，我还剔除了一些不再使用的旧论坛。因此，如果你注意到事情看起来比以前更精简了，不要担心…你不会发疯的。

说到清理，我不得不说，论坛工作人员在清理垃圾邮件和“无用”帖子方面所做的努力应该得到一枚奖章。我们已经实施了在论坛上发帖的质量标准，不符合该标准的帖子将被删除。在我逛论坛的过程中，我注意到在大多数情况下，报告的帖子在不到两分钟的时间内就被清理掉了。在我看来，这相当令人印象深刻。

除了打扫之外，我还想了一些事情。我可以同时做这两件事可能会让你们中的一些人感到惊讶，我承认这需要一些练习。这种想法与我们鼓励你在创意发布时跳出框框思考的方式密切相关。一个论坛可以为问题提供答案，但我希望 SitePoint 不仅仅是好的。我希望它成为人们选择去闲逛的地方，因为这里有很多令人兴奋的对话和有趣的人来交流想法。因此，如果那是你希望 [SitePoint 论坛](https://www.sitepoint.com/forums)成为的那种地方，那就开动脑筋，想出几个有趣的话题来讨论。如果你喜欢免费的书，那么你会想在五月逗留…但这就是我现在要说的。；)
