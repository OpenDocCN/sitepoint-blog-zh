太棒了——移除广告比 AdBlock 有趣多了！它的工作原理是在页面上覆盖一个画布，然后使用少量的 JavaScript 魔法来检测和分解 HTML。它甚至可以记录你的分数。

## [JS1k](http://js1k.com/home)

你还记得那些挑战你用 10kb 构建一个全功能 web 应用程序的 JavaScript 竞赛吗？不够硬核？看看 [1Kb 比赛](http://js1k.com/home)。JavaScript 库被禁止，可以容纳 140 个字符的条目被奖励积分。

比赛已经结束，但获胜者提交了一些令人惊讶的作品，[弹跳眼魔的传说](http://js1k.com/demo/635)(一款类似马里奥的游戏)，以及[小象棋](http://js1k.com/demo/750)(令人尴尬的是，它几次击败了我)， [WOLF1K 和彩虹人物](http://js1k.com/demo/762)(一款 Wolfenstein 3D 游戏)，[疯狂多人双面乒乓](http://js1k.com/demo/41)和必玩的[俄罗斯方块](http://js1k.com/demo/730)——其中甚至包括音乐。

你的单片网页运行到几兆字节？你应该为自己感到羞耻！

## 别管子弹了

《别介意子弹》是一部令人惊叹的互动漫画，使用 HTML5、CSS3、SVGs 和 JavaScript 实现了各种效果。你永远也猜不到是谁创造了它…微软。开发它是为了展示 IE9 的新特性，以便它能在那个浏览器中运行得最好。

## [市中心的荒野](http://www.thewildernessdowntown.com/)

这个网站已经在 [SitePoint 播客](https://www.sitepoint.com/podcast/)中提到过，但是对于那些错过它的人来说，值得重复一遍。这是一个 Chrome 实验，展示了为 Arcade Fire 的“我们曾经等待”定制的音乐视频。这是开创性的:HTML5 一代的波西米亚狂想曲。
