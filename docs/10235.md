# 成功发现过程的 9 个技巧

> 原文：<https://www.sitepoint.com/9-tips-for-a-successful-discovery-process/>

如果你刚刚赢得了一个艰难的竞标或者在一家新公司找到了一份工作，那么恭喜你！你要庆祝一下。不过，把真正的派对留到项目完成后，因为你已经落后了。

打造一个有用且相关的网站需要你有很多头脑。你必须了解为什么员工早上起床，高管每天都在做什么，为什么他们的公司有客户，以及这些客户对网站的期望。如果你不熟悉这个行业和竞争对手，那么你需要尽快熟悉他们。

感到不知所措了吗？以下是一些处理复杂发现过程而不丢失它的方法:

**开球前**

*   社交——在社交网络上追踪利益相关者和员工。感受他们的专业知识、个性和互联网知识。
*   侦探——找到他们的网站，阅读他们的博客文章，看会议幻灯片，阅读评论。
*   节俭——如果合同还没有签署，避免在一个项目上花费太多时间，但是即使两三个小时的研究也能让你领先一步。

**在发现期间**

*   学习行话——看起来有些行业有缩略词配额，这样别人就看不懂了。通过立即开始一个行业术语表，当你试图记住 ASN 和 ANS 之间的区别时，你将有助于抑制你在通话中尴尬的沉默。当你在项目中工作时，将术语表放在手边，在你的内部讨论中使用这些术语，并尽可能地吸收。
*   了解全貌——只采访利益相关者或高管可能很诱人，但他们对客户和网站使用的看法可能与现实不符。同样，仅仅根据客户的反馈来改变也是短视的，可能会导致你无法解决高管团队可能会带领公司走向的新方向。在范围和时间允许的情况下，尽可能多地与客户、员工、业务合作伙伴和利益相关者交谈。
*   拥抱无知——利用你缺乏知识的优势，不要害怕问“愚蠢”的问题。在发现过程中很容易掩盖细节，这些细节稍后会显现出来，所以把重点放在你发现自己在做假设的地方。

**整个项目**

*   与时俱进—您客户的业务和行业在不断发展。通过电子快讯、RSS 订阅源和社交网络了解最新的行业新闻。你应该一开始就注册这些，所以现在你只是偶尔查看一下。
*   观察竞争对手——当竞争对手更新他们的网站或改变产品供应时，与你的客户谈论此事。您可以建议相应的编辑，以反映他们在策略上的变化。你的客户会欣赏你的专注和商业头脑。
*   做一个善于交际的人——记住利益相关者是真实的人，你最好的工作将来自于对客户端动态的敏感。发现不仅仅是数据，最成功的产品也会考虑人的因素。

所有这些都会帮助你在短时间内达到速度。但是不管你有三个月、三周还是三天的时间去发现，只有一件事会真正激励你去做伟大的工作:真正的兴趣。如果你想做有趣且有挑战性的伟大工作，那就成为你客户的代言人，其他一切都会水到渠成。
