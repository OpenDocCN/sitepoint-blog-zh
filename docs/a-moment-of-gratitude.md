# 感激的时刻

> 原文：<https://www.sitepoint.com/a-moment-of-gratitude/>

如今，世界上大大小小的悲剧似乎层出不穷，既有人为的，也有自然的。

对于销售网络服务来说，这似乎是一个无关紧要的问题，但确实是关键:你过着自己喜欢的生活吗？你在做你喜欢的工作吗？

我不禁对自己创业、在家工作、自己支配时间、做自己喜欢的事情表示无限的感谢。我认为，我一半的成功来自于热爱我所做的事情。客户和潜在客户感受到了我的热情，并希望与我合作。

生命随时可能结束。今天，明天，在你最意想不到的时候。你不想有遗憾。

如果你正在读这篇文章，并且正在做一份你不喜欢的工作，那就制定一个快速离开的计划。这不是你热爱生活的唯一方面，但肯定是重要的一方面。

你很幸运拥有一系列技能，这些技能可以带你走得更远。

时间是宝贵的。当你的时候不要后悔。

问问你自己:如果你现在工作的大楼发生了可怕的事情，你死了，你会对你工作的地方没有遗憾吗——或者你会在最后一刻说，“我真不敢相信我没有辞掉这份肮脏的工作，去做我真正想做的事情？”

我希望这段话不要太陈腐，至少对阅读它的人有所影响。
