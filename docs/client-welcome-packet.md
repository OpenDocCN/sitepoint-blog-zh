# 使用客户欢迎信息包取得良好的开端

> 原文：<https://www.sitepoint.com/client-welcome-packet/>

你刚找到一个新客户。你手里拿着已经签好的合同，工作即将开始。当你开始的时候，你给客户发送一个客户欢迎包，你的新关系就为长期的成功做好了准备。

等等……你没有客户欢迎包？您可能想要创建一个！在这篇文章中，我将解释什么是客户欢迎包，为什么你可能想要一个，以及你应该在里面包含什么。

## 什么是客户欢迎包？

客户欢迎包就是这样一套文件，欢迎客户来到您的公司，并提供他或她与您有效合作所需的所有信息。它就像一张路线图，引导客户了解您的服务、政策和工作流程。

## 何必呢？

每一段新的感情都有一个学习曲线。当挑战出现时，你当然可以从容应对，但客户欢迎包可以作为客户解决问题的资源，有助于铺平道路。它不会消除问题或挑战，也不会试图消除与客户的直接接触，但它会在一个地方向客户提供有关您公司的重要信息。

您和您的客户都可以从欢迎礼包中获益的其他方式包括:

*   你展现了一个专业的形象。
*   你的文件重申了你的政策和程序。
*   这向客户表明你关心满足他/她的需求。
*   它把你所有的客户放在同一个起点上。
*   其中包含的信息可以提醒你应该在什么时候做什么。

## 客户欢迎包里有什么？

您的客户欢迎包可以包含任何对您的客户有价值的公司或服务的具体信息。这里有一些想法可以让你思考一下你的欢迎礼包应该包括什么:

*   ![](img/bf36eeca6951bc280b23da4c9ad99359.png "folder")感谢/欢迎信
*   合同的签字副本
*   常见问题
*   支付政策
*   信用卡授权表
*   填好的 W-9 表格
*   公司政策
*   你休假的日历
*   联系信息表(您的)
*   联系信息表(供客户填写)
*   你最近的公司时事通讯
*   项目问卷
*   后续步骤

你不想给客户提供过多的信息，所以要有选择性，只包括你认为最重要的信息。

您的欢迎包可以作为包含所有相关 pdf 的 ZIP 文件共享，或者在客户专用区域在线提供。您还可以发送一份客户欢迎包的硬拷贝，并附上额外的印刷材料，如名片和小册子。
