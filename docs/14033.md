# 为你的客户设计高质量的网站时，要问他们一些关键问题

> 原文：<https://www.sitepoint.com/some-key-questions-to-ask-your-clients-when-designing-quality-sites-for-them/>

根据我的经验，许多网页设计师都有很好的机会从网页设计师转变为客户的营销顾问。

事实上，大多数网页设计师都应该是营销顾问，负责帮助客户吸引更多的客户到他们的网站，并获得更多的业务。编码语言、设计原则和其他技能只是实现这一结果的工具。

我知道这是真的，原因有二:

1.你的客户认为你是营销解决方案的一部分，而不是网页设计解决方案的一部分。如果你不同意，问问他们希望他们的网站为他们做什么。

2.我甚至不会编写一行 HTML 或任何其他语言的代码，却可以通过帮助客户改进他们的网站而获得大量报酬。事实上，我比大多数网页设计师赚得多，主要是因为我专注于营销策略。

所以，请确保你在被雇佣的任何时候都向你的客户提出这些问题:

1.谁是你的客户？

2.他们如何做出购买决定？

3.他们购买你的产品/服务的主要标准是什么？

4.他们是怎么听说你的？

5.与贵公司相比，竞争产品/服务如何定位自己？你的定位对你的客户来说是独特和有意义的吗？

6.你的产品/服务有什么独特和优越之处？

7.你希望客户在与你互动时，自始至终的“体验”是什么？

8.你为你的顾客解决什么问题？这个问题让您的客户付出了什么代价？

9.你的解决方案是什么？全面吗？

10.您的解决方案有什么好处？有什么优势？

11.你有什么证据证明你的解决方案是独一无二的、更好的？

在上面的每一个问题下面都有很多子问题要问，但是如果你从这个层次开始，你的客户会觉得你更老练，对他们更有用。
