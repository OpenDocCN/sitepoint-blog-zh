# 2012 年 5 月浏览器趋势:IE9 反击

> 原文：<https://www.sitepoint.com/browser-trends-may-2012/>

自从我们[上次关注浏览器市场](https://www.sitepoint.com/browser-trends-april-2012/)以来，已经过去了一个月。从那以后， [Firefox 12 和 Chrome 18](https://www.sitepoint.com/firefox-12-chrome-18-whats-new/) 发布了，微软开始了一场针对 IE9 的积极的电视、电影和在线营销活动。这是否影响了[全球 StatCounter 统计](https://gs.statcounter.com/#browser_version-ww-monthly-201204-201204-bar)中的使用模式？…

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 12.7%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

自 2011 年 10 月以来，Chrome 的月增长率首次跌破 1%。其他厂商梦想市场份额增长 0.37%，但对 Chrome 来说，这个数字低得异乎寻常。我不相信采用率已经开始稳定，这可能是一个统计信号。也就是说，Chrome 相对于竞争对手的优势比以往任何时候都更加微不足道。

本月最大的赢家是 IE9，自 3 月初以来上涨了 3.6%。三个因素促成了它的成功:

1.  自动化浏览器更新。
2.  微软光鲜亮丽的营销活动。
3.  迁移到 Windows Vista/7 的企业。

然而，微软不能自满。IE8 的下降速度快于 IE9，IE 的总使用量下降了 0.74%。Chrome 仍有望在 2012 年夏季赶超。

Safari 又度过了一个不错的月份，浏览器正在慢慢向两位数的使用数字攀升。iPad 在 7.14%的总用户中占 2.12%，这意味着它现在的用户数量超过了 IE6。

Firefox 和 Opera 的市场份额几乎没有变化。也许这并不奇怪；五大浏览器之间没有太大的差异，我们可能会进入一个稳定期，浏览器市场不会有太大的变化。

## 移动浏览器的使用

4 月份的手机使用量增加到了所有网络活动的 9.58%。这部分是由于北半球天气更好——尽管英国有 100 年来最潮湿的 4 月。

主要的移动浏览应用程序有:

1.  opera Mini/Mobile——21.52%(下降 1.34%)
2.  安卓系统——21.31%(增长 0.15%)
3.  iPhone——20.04%(下降 0.06%)
4.  诺基亚浏览器——11.42%(下降 0.44%)
5.  UC 浏览器——7.77%(增长 1.42%)

前五名中几乎没有变化，尽管 UC 浏览器已经在黑莓中占据了一席之地。

虽然很高兴看到移动市场的良性竞争，但在一系列流行设备上测试网站和应用程序仍然非常困难。你烦吗？
