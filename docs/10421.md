# 5 个 jQuery 用户界面

> 原文:[https://www.sitepoint.com/5-jquery-user-interfaces/](https://www.sitepoint.com/5-jquery-user-interfaces/)

设计用户界面是如此艰难的任务；你可能已经面临这样的情况，你意识到你的设计技巧已经过时了。如果这困扰着你，jQuery 用户界面就是答案。在 jQuery 增强功能的帮助下，设计网页用户界面的每一部分不再是一件大事。有很多 jQuery 用户界面可以从不同的网站下载；你只需要选择适合你的网站。

我这里有一些 jQuery 用户界面列表，可能会对您有所帮助。
