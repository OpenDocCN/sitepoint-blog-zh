# 我宁愿把熏肉钉在脸上也不愿打电话

> 原文:[https://www . site point . com/id-宁要熏肉不要打冷电话/](https://www.sitepoint.com/id-rather-staple-bacon-to-my-face-than-make-a-cold-call/)

2008 年，《销售观察》刊登了一篇题为“为什么决策者讨厌电话推销”的文章如果你想确信电话推销不起作用，这是对时间的巨大浪费，而且是寻找潜在客户的最“无效和昂贵”的方式，那么就跳过我要说的，直接看那篇文章。(一定要阅读那些强烈反对作者的人的众多评论。)

另一方面，如果你想探索电话推销如何成为寻找新客户的好方法，那就留下来，因为这正是我要做的。

让我分享一些我第一次打电话的辉煌经历。当我和我的合作伙伴在 2001 年初开始我们的 web 开发业务时，电话拜访是我们早期尝试的事情之一。没有太多的脚本和绝对没有计划，我们打开电话簿，开始打电话。

尽管如此，我们还是找到了两个客户！我们对自己非常满意，以为自己已经走上了成功之路。但是当现实来临时，我们意识到它们是非常小的项目。其中一个项目进行了几周后，客户(他还没有给我们提供书面副本)开始抱怨我们花的时间太长了。在提供了副本之后，她继续要求我们立即建立她的网站。我们退还了她的押金。

经过深思熟虑(实际上，没有那么多)，我们决定不要打电话。

今天，我是一个电话销售部门的销售经理，所以我对电话推销略知一二。但十年前，我们的无知和随后的失败让我们得出结论，我们在“做错事”，而事实上，我们在“做错事”。

我如何知道，在适当的情况下，打电话拜访是“正确的事情”呢？因为我的部门得到的结果。以下是三个电话销售人员在一个特别好的月份里所做的一些示例:

1618 次拨号
203 次交谈
86 次预约
27 次成交

让我们来看看其中的一些数字。在 1，618 个电话中，我们大约每八个电话就与一个决策者交谈，我们为每 2.4 个与我们交谈的决策者安排了一次约会。那 27 次销售呢？它们代表了五位数的美元收入。这种“低效且昂贵”的寻找潜在客户的方式怎么样？

你知道，对于像我们这样的公司来说，电话推销和游说是最好的潜在客户选择。我们的销售周期很短，所以销售代表没有时间进行网络和建立关系之类的事情。即使对于有能力尝试其他方法的企业来说，在你的营销组合中加入电话推销仍然是一件好事。

你可能想知道“熏肉”、“主食”和“面子”这几个词怎么会出现在同一个句子里——更不用说这和电话推销有什么关系了。你只需要在接下来的几篇文章中找到答案。在第 2 部分中，我将讨论如何确定电话推销是否适合 web 行业(提示:是的)，以及如何让它为您服务。
