# 微软从苹果那里学到了错误的教训

> 原文:[https://www . site point . com/Microsoft-learn-the-error-lesson-from-apple/](https://www.sitepoint.com/microsoft-learns-the-wrong-lesson-from-apple/)

![](../Images/438f92f1d03af76e0ada596a6264aeab.png "windows-logo")在第一个月，苹果的 App Store，销售 iPhone 和 iPod Touch 设备的应用程序，[收入 3000 万美元](http://www.washingtonpost.com/wp-dyn/content/article/2008/08/11/AR2008081100440.html)，并有望在年底实现超过 3 亿美元的销售额。投资公司 Piper Jaffray 认为，2009 年，应用商店的收入可能会超过 12 亿美元(T4)，这将成为苹果公司的主要收入来源。

从微软的角度来看，这一切看起来一定非常诱人。他们更成熟的 Windows Mobile 平台有大约 18000 个应用程序，而苹果的只有 2000 个。然而，与苹果不同的是，微软并不控制其平台上应用程序的分发——事实上任何人都可以建立一个市场并销售 Windows Mobile 应用程序，而且许多人已经这样做了。因此，从理论上讲，微软没有成为其移动平台上应用程序的中央结算中心，因此错过了大量潜在收入。

暂时如此。

昨天有关于 Skymarket 的[消息，这显然是微软计划为其即将推出的 Windows Mobile 7 平台上的移动应用提供的应用市场的名称。根据微软的一份招聘启事，该公司的目标是让 Skymarket 成为“希望分发 Windows Mobile 应用程序并从中获利的开发者的首选之地”。谷歌](http://www.istartedsomething.com/20080831/microsoft-launch-skymarket-applications-marketplace-windows-mobile-7/)[上周宣布了他们手机操作系统安卓的应用市场](http://news.cnet.com/8301-1035_3-10028108-94.html)。

从表面上看，这似乎是微软向苹果学习的一个很好的教训:控制应用程序的分发意味着大量的潜在收入。但是对消费者来说是不是不好呢？

这真的取决于微软有多像苹果。正如我们上个月讨论的那样，[封闭的平台对网络不利](https://www.sitepoint.com/will-closed-platforms-ruin-the-web/)。它们为用户提供了额外的安全性，为平台提供商提供了额外的收入来源，但它们也加剧了“应用程序开发者和平台所有者之间的紧张关系，并[潜在地]让用户感觉他们实际上并不拥有他们正在使用的应用程序。”

《卫报》的杰克·施菲尔德认为[微软不太可能像苹果那样封闭自己的应用平台，因为一旦他们这样做，现有的应用销售者就会叫嚣犯规(当人们开始称他们为垄断时，微软就有麻烦了)。这可能是真的，微软似乎也可能将 Windows Mobile 7 捆绑到其](http://blogs.guardian.co.uk/technology/2008/09/01/microsoft_skymarket_coming_for_windows_mobile_7_apps.html) [Live Mesh 平台](http://www.mesh.com/) ( [我们的报道](https://www.sitepoint.com/microsoft-opens-live-mesh-to-all/))，这在大多数人看来证明了[微软知道如何开放](http://blog.broadbandmechanics.com/2008/08/will-microsoft-connect-their-live-mesh-to-our-open-mesh)。

另一方面，暗示 Skymarket 仅仅是苹果应用商店的克隆是不真诚的。微软在受控的、封闭的平台应用市场方面有丰富的经验。Xbox Live Marketplace 已经推出近 3 年了(从 2004 年开始成为 Live Arcade)。这个市场是一个微软控制的应用商店(在这种情况下销售游戏)，类似于苹果的应用商店。

对他们来说，谷歌已经表明他们的 Android 市场将会是完全开放的。“我们选择‘市场’而不是‘商店’，因为我们觉得开发者应该有一个开放和无障碍的环境来提供他们的内容，”谷歌朱立伦上周对 CNET 说。朱棣文将把应用程序上传到 Android market 的过程比作把视频上传到 YouTube，这表明谷歌对市场的任何控制都将是被动的、事后式的节制。
