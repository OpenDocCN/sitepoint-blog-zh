# 咖啡休息时间:谷歌地图 FlightSim

> 原文：<https://www.sitepoint.com/a-coffee-break-moment-google-maps-flightsim/>

对我来说，API 可能是过去几年 Web 开发中最有趣的领域，它允许资源不多(或者天赋不多)的开发人员开发比他们希望自己创建的系统更大、更丰富的系统。

看看像 FD 的 Flickr Toys 这样的网站，很容易得出 Flickr API 已经被最广泛使用的结论，但这可能是出售来自 Del.icio.us、亚马逊、易贝，当然还有 Google short 的 API。其结果是产生了数百种新的有趣的产品、服务和各种各样的混搭，否则将永远不会出现。

有些非常实用——我想到了 X 时刻和 T2 的住宅地图——而另一些则专注于变得聪明和有趣。像这个:护目镜::谷歌地图飞行模拟。

所以，下次你拿起一杯咖啡，装满它，选择一个城市，带上一副飞行护目镜，飞到世界上任何一个大城市的上空——从伦敦到纽约到苏黎世到悉尼。如果你有耐心，你甚至可以自己选择。基本的应用程序很简单，但会让人上瘾，它使用 Flash 在平滑滚动的地图上叠加一系列简化的 3D 飞机渲染图。除了基本的方向控制，护目镜甚至给你机枪，让你扫射你最不喜欢的城市。那里的所有家庭都很开心。

事实上，我发现使用地图是一种令人惊讶的娱乐方式。不断通过的地图游行让事情变得有趣，尽管当你发现一些很酷的东西时，“冻结”按钮可能会很好。然而，这是一个执行得很好的好主意。

那么，它实际上使用 API 吗？

嗯，严格来说不是，…但是“嘘”..当在悉尼港的豪宅里嗡嗡作响如此有趣时，为什么要挑剔呢？
