# 增加你的薪水:获得认证！

> 原文:[https://www.sitepoint.com/paycheck-get-certified/](https://www.sitepoint.com/paycheck-get-certified/)

关于教育的价值，在计算机领域一直存在着巨大的争论。这种争论在这个领域可能比其他领域更普遍，因为有很多人是自学的，或者只是简单地知道他们在做什么。这是一个奇怪的领域，因为你可以让一些 17 岁的高中生作为顾问一年赚 8 万美元，因为他擅长汇编语言代码。然而这个有学士学位的家伙只挣 5 万英镑。去想想。

##### 变化的速度

然而，在我们计算机领域的职业生涯中，我们大多数人都曾面临着抉择。我们继续深造吗？如果是这样，最好的方法是什么？一个困难是，这个领域的大多数有能力的人都非常忙，他们甚至没有时间好好睡觉，更不用说回学校了。事实上，我们中的许多人最初接触计算机是因为我们讨厌学校。从事计算机程序或设计网站是对那个讨厌的叫做大学的地狱的逃避(至少对我来说是这样)。

嗯，如果你把情绪放在一边，理性思考，这个问题真的不难回答。我知道我仍然把我的大部分问题归咎于我的高中和大学教授，但那真的无关紧要，不是吗？

重要的是计算机行业发展如此之快，以至于每两三年技术就会完全改变。当你想到会计几千年来在概念上没有改变时，那是令人震惊的！与英特尔新处理器的标准发布时间表相比，医药行业进展缓慢。尽管 Windows 2000 中的 4000 万行代码似乎永远也完成不了，但与每一种药物的发布都需要几十年的研究和测试相比，这还是微不足道的。

正是这种速度使得继续教育不可选择。是的，你可能是当今世界上最不可思议的 Fortran 程序员，但一年后，Fortran 可能会被其他东西完全取代。我一次又一次地看到这种事情发生。收入高得令人难以置信的优秀人士发现自己很难找到工作，因为他们的技能不再被需要……或者已经过时，没有人再做那种工作了。

但是等一下…大学需要很多很多年。我的意思是人们花五年、六年甚至七年的时间来完成大学学业！而且是全职！一个每周要工作 50 个小时甚至更多来谋生的专业人士怎么能有时间去上大学呢？？？

##### 从雇主的角度来看

这其中还有一个因素，那就是雇主。想想吧。假设你是一名经理，你刚刚发布了一个招聘高级系统经理的广告，需要有 Windows NT 经验等等。在你拿回大约一百份简历后，你如何确定谁是最合格的呢？你可以尝试参考，但如果你已经这样做了，你就会知道从以前的雇主那里得到任何信息有多难，因为有诉讼和其他事情。你可以看看这个人做了什么，但你可以肯定他选择了他最好的作品，而不是他的正常品质。无论如何，那对编程和 HTML 编码工作有一点帮助，但是那个系统经理的职位呢？如何看待他之前的作品？

你可以问很多关于技术方面的问题，你可以让你的技术人员采访这个人。但是，在面试中很难提出正确的问题，而且面试只有一个小时。

你在看什么？一个经理如何知道？

啊，有答案了，那叫认证！如果你是系统经理，你可以获得 MCSE，这基本上表明你知道如何管理一套微软产品，并且你已经通过了一些测试。你甚至可以更进一步，得到你的 MCSE+I，意思是“你是精英中的一员”。像这样的有效微软认证意味着你在技术上是最新的，因为你必须年复一年地测试新产品以保持你的认证。

如果你再加上一些更广泛的认证，如 Hycurve、CIW、A+和 Network+，你就获得了全面的计算机或互联网教育。

但是那些说“任何一个有 500 美元的混蛋都可以不费吹灰之力获得 CNE 奖”的人呢？唯一对这些测试有问题的人是经理和老 Fortran 程序员。”你能说什么呢？就我个人而言，我甚至不想和这些家伙争论，因为他们已经下定决心了。也许他们太聪明了，不需要教育？

但是对于我们其他人来说，认证很难。诚实地通过 MCSE 或 MCSE+1 考试是非常具有挑战性的。我是一名雇主，我会非常尊重任何通过 MCSE+I 的人。这是一系列非常艰难的考试！

##### 学什么

就个人而言，作为雇员和雇主，我衷心建议每个技术人员回顾自己的职业生涯，并决定哪些证书可能有用。让我们以我前面提到的系统管理器为例。对于一个 Windows NT 系统管理员来说，什么是最完美的认证组合？

这是我的建议。从 A+和 Network+开始。这会让你对计算机的基础有更广泛的了解。通过参加以下考试获得 MCSE(这些是 4.0 考试，Windows 2000 有不同的组合)。

*   Windows NT 工作站或 Windows 95/98
*   Windows NT 服务器
*   Windows NT 企业版
*   网络要素
*   Microsoft Exchange
*   传输控制协议

好的，现在确保你的 MCSE 是最新的，在他们发布的时候重新测试。看看 MCSE+I，决定是否适合你。如果是这样，那就尽一切办法去做吧。

接下来，看看一些更广泛的认证，如 CIW(注册互联网网站管理员)。这些考试远没有 MCSE 考试难，它们有助于拓宽你的视野，让你不仅仅是对微软有偏见。能够理解所有技术是如何结合在一起的很重要…有时有比微软更有意义的其他解决方案。你至少需要明白你为什么选择微软而不是另一家公司，这样你才能为自己辩护。

##### 现在怎么办？

不，如果你想完整，你还没有完成。您还可以获得各种产品的认证，如 Adobe Acrobat 和 Photoshop、IBM 服务器、HP 打印机等等。

但是坚持住！等一下！所有这些要花多少钱？你猜怎么着？这不便宜…这需要时间和金钱。有价值的事情从来都不容易。

做所有这些的好处绝对是惊人的。首先，假设你诚实地完成了所有这些，并且真正地学习了这些材料，你将会成为一个非常抢手的商品！在同样的职位上，拥有 MCSE 的人很容易比没有的人多挣 20%。这只是这个行业的一个事实。第二，你的工作会变得更容易。相信我，比如说，向一位外汇专家询问一个外汇问题，得到一个直截了当的答案，而不是一些废话或“我必须看看它”，这是多么令人耳目一新。知道需要做什么是一种很棒的感觉。

第三，如果你不得不换工作，你会发现它更容易，你会得到一个更高质量的职位。事实上，你可能会发现自己面临着面试和工作机会的选择。

你让你未来的雇主很轻松。他不需要猜…他只要看一下你的证书就能对你的能力有一个很好的了解。

呃，但是那些只是去那些“新兵训练营”或者别的什么地方获得快速和肮脏的认证的人呢？一个好的面试官通常可以确定一个人是否了解他的东西，或者只是一个“书面证明”。如果这个人真的通过了面试，那么，一个真正优秀的经理会在第一个 90 天的磨合期内发现这一点，并利用这一点。

但是，你说你是顾问。你不需要一份全职工作，那么你为什么需要认证呢？就我个人而言，我甚至不会和一个背后没有可靠认证的顾问交谈。当我雇用 Visual Basic 编程顾问时，我希望证明他们知道一些事情。我想看看他们的工作样本，我想看看他们的认证。如果我需要一个专家来修理我的 Exchange 服务器，我想看看他是否足够好，至少能通过 Exchange 考试。毕竟，我会付给这些人每小时 75 到 225 美元，所以他们最好多做一点。

哦，但是，你说，你找工作从来没有问题，计算机对你来说太容易了，而且，证书是给娘娘腔的。我唯一能告诉你的是，这个行业肯定会要求他们雇佣的计算机领域的人进行认证。你喜不喜欢真的不重要。当然，如果你有一个“熟人”或“熟人”，这就不适用了……但这永远是真的吗？

##### 你说，我为什么知道？

我只是重申我的建议。我是这方面的老手了。我现在入行 21 年了，都做到了。从十几台机器学过 Basic，Fortran，Fortran Plus，Basic Plus，Forth，C，C++，Visual Basic，Java，HTML，Teco，assembler，Bliss 甚至还涉猎过固件代码。我用过老式的 PDP-11(每次都必须将启动代码手动输入电脑，而且操作系统是用纸带加载的)、VAXes、Macintosh 和 PC。我写过设备驱动程序、实时水泵系统、各种应用程序、磁盘碎片整理程序、文件恢复程序和汇编语言调试器。你能想到的我都设计写了。我创建了内部网和互联网网站，写了超过 45 本 100 页或更长的手册。我管理过多达 15 人的团队，我用手和膝盖爬行铺设电缆和更换烧坏的电路板。

我的建议是花时间，花钱并获得认证。真实诚实地去做，学习你的手艺。你永远不会后悔。
