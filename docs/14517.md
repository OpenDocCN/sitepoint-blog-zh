# 如何避免廉价客户，第二部分

> 原文：<https://www.sitepoint.com/how-to-avoid-cheap-clients-part-ii/>

关于擎天柱关于避免廉价客户的问题，有很棒的帖子。

如果你错过了评论，请看:[https://www.sitepoint.com/blog-post-view.php?id = 179638 #评论](https://www.sitepoint.com/blog/)

以下是一些额外的想法，基于擎天柱的回应和对情况的自我诊断(例如，他可能被认为是一个 23 岁的神童，而不是一个严肃的专业人士):

首先，你得到了你所吸引的东西，这是基于你如何说话、穿着、表达自己、表现自己以及你所做的工作。换句话说，市场永远是对的，现在市场正给你送来“便宜的”、价格驱动的客户。为了改变你所吸引的东西，你需要彻底改变你的生活方式。你需要重塑你自己和你的生意。

以下是实现这一目标的主要方法:

1.开发一个全面的强大解决方案，解决您的潜在客户/客户的主要业务问题。开始像一个不仅仅是技术神童，也是商业顾问的人那样说话。

2.改变你的营销信息。从你解决的业务问题、为什么你是独一无二的、你提供的好处和证据(从你合作过的顶级客户和你为他们提供的顶级解决方案的角度)来谈论你的业务；如果有必要的话，稍微处理一下)。

3.以不同的方式请求推荐。告诉你的客户你在寻找什么，他们想做什么类型的工作。就您认为他们可能知道的“顶级”潜在客户提出具体问题。

4.向现有客户寻求建议。告诉他们你担心自己被认为是一个 23 岁的神童，并希望被人以不同的方式看待。如果他们处在你的位置，请他们建议他们会怎么做。就你的方法、工作、第一印象等提出具体的问题。

5.当你得到推荐的时候，直接切断价格问题。JMorrow 给你的回复是一种方法。我的脚本不同:“潜在客户先生，在我们开始之前，我有一个问题。许多人看着我，看到一个 23 岁的孩子，认为他们会得到一个荒谬的价格。我必须让你预先知道，如果你正在寻找最低的价格，那么我是错误的人。我提供高质量、全面的解决方案，并做顶级工作。如果你只关心价格而不关心质量，现在就让我知道，这样我们就不会浪费彼此的时间。”

6.给自己录像——从着装到眼神交流、语调、手势等等。—在模拟客户会议中。自己看自己刻画的形象。以获得顶级客户的人为榜样，改变你的一些行为和你可能传达的无意信息。

7.获得推荐来源之外的可见性。找到一种方式来谈论、写作和研究你认为是顶级的客户。确保你有一个强有力的信息给他们，传达你需要传达什么来吸引他们。

8.最后，确定一个便宜的潜在客户真的很便宜。我和很多人合作过，一开始我觉得他们很廉价。一旦我证明了我的价值，他们突然愿意花更多的钱。有时候你需要先建立信任和“沙滩头”。很难知道谁是真正的廉价，谁是持怀疑态度的。

另一个想法是——每一项咨询实践都会随着时间的推移而发展。许多人不得不从“廉价”客户开始，逐渐向食物链的高端移动。此时，顶级顾问优雅地“解雇”老客户，并开始取悦新客户。这是一个自然的过程，需要耐心(尽管上面的步骤会加速这个过程)。

希望这有所帮助…..
