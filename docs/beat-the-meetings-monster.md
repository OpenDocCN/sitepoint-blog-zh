# 击败会议怪物

> 原文：<https://www.sitepoint.com/beat-the-meetings-monster/>

有些星期，我似乎有一半时间都在开会。

事实上，如果你问我以什么为生，我会说“阅读邮件和参加会议”。很疯狂，不是吗？

如果你对会议这个怪物感到不知所措，这里有五个建议给你。

### 只参加你必须参加的会议。

回到组织者那里，问问你是否真的需要去那里。当你组织一次会议时，仔细考虑每一个与会者，看他们是否真的被需要。

### 有一个议程，并坚持下去。

我们都知道没有焦点的会议会走向何方。通过写下议程并坚持下去来保持会议简短明了。避免闲聊，如果需要的话，留到会后再聊。

### 主持站立会议。

与其舒服地坐在椅子上，不如站着开会；你会发现，如果你需要站在周围讨论，人们会渴望回到自己的办公桌前，这意味着所有人都可以更快地开会。

### 快速做出决定，然后继续前进。

一旦做出了决定，就继续下一步。如果你继续解释你的全部推理，这对每个人都是不公平的，尤其是如果不需要的话。

### 留出“空闲会议”时间。

在你的日历上划掉大块的内容。它们可以是每周一整天，也可以是整个上午或下午；任何能让你集中精力把会议减到最少的事情。

祝你好运驯服会议怪物！
