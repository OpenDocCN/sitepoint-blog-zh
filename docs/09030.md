# 让 WordPress 为你工作

> 原文:[https://www.sitepoint.com/make-wordpress-work-for-you/](https://www.sitepoint.com/make-wordpress-work-for-you/)

自从 WordPress 在 2003 年从 b2/cafelog 演变而来，它[超越了所有的竞争对手](http://www.google.com/trends?q=blogger%2C+drupal%2C+sharepoint%2C+wordpress&ctab=0&geo=all&date=all)成为世界内容管理系统的首选。

![wordpress stats](../Images/cab53650b56ced5b3239c45d065ef648.png)

*   **全球 65，810，415 个 WordPress 网站(在撰写本文时)**
*   每天都有超过 100，000 个新的 WordPress 站点被创建！
*   **为 Alexa 互联网“前 100 万”网站中的 14.7%提供支持**
*   **在 22%的新网站中使用** ([http://en.wordpress.com/stats/](http://en.wordpress.com/stats/))

WordPress 成功的核心是它的易用性、简洁的风格和开源精神，正如 WordPress.org 的早期版本 T2 所描述的。
WordPress 的美妙之处在于它适用于企业(驱动着像《连线》、CNN 和《纽约时报》这样的网站)，也适用于小型博客和企业。

你们中的大多数人会使用免费主题建立一个标准的 WordPress 站点，但是有多少人做到了:

*   **完全理解 WordPress 的底层代码**
*   **操纵其服务器端能力**
*   **针对各种项目进行定制**
*   利用其强大的 API、插件和其他扩展？

Mick Olinik 和 Raena Jackson Armitage 的 WordPress 选集 是来自 SitePoint 的新书。它的 336 页为网络开发者提供了广泛的 WordPress 解决方案，让你掌控这个令人印象深刻的 CMS。塞满了成熟的技巧和技术，你会发现如何有效地使用 WordPress 来更好地满足你和你的客户的需求。
