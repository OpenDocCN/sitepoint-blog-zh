# 适用于 OS X 的 PHP 5 及更多版本

> 原文：<https://www.sitepoint.com/php-5-and-more-for-os-x/>

Marc Li yange 继续向麦金塔开发者提供优秀的材料，在他们的系统上利用开源。4 月——PHP 4 . 3 . 11 和 5.0.4 的 OS X 安装包发布。

遵循来自 Liyanage 的一些基本提示(他在每个软件包中都提供了这些提示),人们可以在同一个系统上运行两个版本——只是不能同时运行。

该网站还有一个[活跃论坛，用于讨论 Marc 构建和分发的特定 Mac OS X 软件包](http://www.entropy.ch/phpbb2/)。

对于那些刚接触 Entropy 的人，Marc 主要使用它来交付 OS X 软件包和跑步指南(与捷豹和黑豹兼容):

*   MySQL 数据库服务器
*   PostgreSQL 数据库服务器
*   vim 6.1 文本编辑器
*   nedit 5.2 X11 文本编辑器
*   t1utils PostScript Type 1 字体实用程序
*   TCP 流量记录器
*   ImageMagick 图像处理工具

Marc 还编写了几个自己的应用程序或插件，包括:

*   石英极限检查，一个应用程序，检查石英极限加速是否活跃。
*   BBEdit XSLT 语言词汇表，一个支持 BBEdit XSLT 作者的词汇表。
*   BBEdit Ant 语言词汇表，该词汇表支持 BBEdit 的 Apache Ant Java 构建描述文件的作者。
*   BBEdit JSP 语言词汇表，一个支持 BBEdit JSP 作者的词汇表。
*   BBEdit Perl 语言词汇表，一个支持 BBEdit Perl 程序员的词汇表。
*   BBEdit Java 语言词汇表，一个支持 BBEdit Java 程序员的词汇表。
*   BBEdit 伯恩 Shell 词汇表，该词汇表支持 BBEdit 伯恩 Shell(例如 bash)脚本的作者。
*   TestXSLT，学习和应用 XSLT 转换的应用程序
*   Checksum 是 openssl 工具的 Cocoa 包装器，用于计算文件的 MD5 和其他校验和(消息摘要)
*   LeanCalc 是一款可可计算器程序(仅限 Panther)。
*   打开终端工具栏脚本
*   用于 BBEdit 和单机使用的 Perl 字符集转换脚本
