# 。NET on the NET 3 月 18-25 日:新的 IIS7 优势和 MVC 开启

Wyatt 最近几周有点忙，所以我将成为你的微软新闻和更新的新来源。这是一个新闻缓慢的一周。所以至少我可以慢慢开始。

本周，大多数人谈论的大事件仍然是[ASP.NET MVC 项目](http://www.codeplex.com/MVCContrib)。虽然 Wyatt 已经提到过几次了，但是在过去的一周,[的完整源代码已经发布在 CodePlex](http://www.hanselman.com/blog/TheWeeklySourceCode21ASPNETMVCPreview2SourceCode.aspx) 上。随着人们开始使用 MVC，也有很多关于它的讨论。你们中的一些 Rails 和 Django 家伙可能认为这是旧闻，但是 MVC 给 ASP.NET 带来了一些好东西，比如运行命令行测试和对标记有 100%的控制权。

微软阵营的另一个大新闻是对 IIS7 的一系列改进。ASP.NET 调试博客对这些变化有一个概述，但是最重要的两个是新的 FTP 发布服务和管理包的 CTP。FTP 发布服务提供了对 IPv6 的支持、一些用于运行共享环境的新功能，以及用新的基于 XML 的配置替换 IIS6 风格的元数据库。

除了新的开发之外，在过去的一周中还发布了一些好的代码示例和教程。Chris Pels 发布了一个关于向服务器控件添加 JavaScript 事件处理程序的视频。对于 Silverlight 开发人员，Shawn Burke 创建了一个关于[构建模板化 SilverLight 控件](http://blogs.msdn.com/sburke/archive/2008/03/22/tutorial-writing-a-templated-silverlight-2-control.aspx)的教程，斯科特·莫里森发布了一个关于 SilverLight DataGrid 控件的[演示。](http://blogs.msdn.com/brada/archive/2008/03/21/silverlight-2-datagrid-walk-through-posted.aspx)

最后，如果你还没有查看的话，硬石咖啡馆的纪念品系统已经开始运行了。这可能是最好的 SilverLight 例子，也是新的深度变焦系统的一个很好的技术演示。
