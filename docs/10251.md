# 精简、广播和筹款

> 原文：<https://www.sitepoint.com/streamlining-broadcasting-and-fundraising/>

随着假期的结束和一年工作的开始，本周我又回到了工作中。这一点，以及我不再每天在假日午餐时间喝啤酒的事实，意味着这篇文章应该比上周的文章更有吸引力。至少，我希望如此。

我想你们中的大多数人也已经回到工作中了(如果你休息过的话)，但是我希望你在圣诞节期间有时间拍一些假期照片。如果你这样做了，你还有几个小时的时间让他们参加在[论坛](https://www.sitepoint.com/forums/index.php)举行的[盛大假日摄影比赛](https://www.sitepoint.com/forums/showthread.php?t=715968)。提交将于今晚午夜(太平洋时间)结束，获胜者将于本月底宣布。有一些很棒的奖项可供争夺，但也许最大的惊喜是我们的书 *[网络摄影](https://www.sitepoint.com/books/photography1)* 的作者 Paul Duncanson 是嘉宾评委，他将为参赛作品提供提示和反馈。所以，赶快去论坛，在时间用完之前提交你的图片吧！
![SitePoint Community](img/9bb2be66119f2936eef6bd9dee6364c7.png "SitePoint Community")

关于竞赛的话题，论坛的工作人员正在努力进行一场激动人心的新竞赛。这个是针对你们当中的网页设计师的。如果纯粹的 HTML 和 CSS 是你的东西，那么你要留意这一点。细节将于 2 月 1 日发布。

本周论坛的最后一篇文章是为那些喜欢在旅途中浏览论坛的人准备的。我们已经安装了 Tapatalk 插件。Tapatalk 是一个论坛应用程序，允许您在移动设备上浏览论坛。请注意，Tapatalk 是一个商业应用程序，但与 SitePoint 没有任何关系，因此如果您想使用它，将需要付费。

本周我一直在关注的另一件事是 twitterfeed。我正在试用 twitterfeed 来发布我们的新博客和文章。到目前为止没有帖子，我不能评论它的有效性，但我听说过好的东西，UI 很容易使用。与此同时，我正在尝试将它张贴到我们的[脸书](http://www.facebook.com/sitepoint)墙上，因为我上周征求了反馈意见，表明在我们的墙上张贴博客帖子的通知将是有用的。我会随时向你汇报最新情况。

关于脸书的同样的反馈请求出现了一些有趣的事情，实际上我已经在幕后做了一些工作。如果你去年关注了我的博客，你就会知道，去年 12 月，我骄傲地拥有了一台新的 Macbook Air。转向 Mac 的部分动机是，我看到了一些由一位同事用 iMovie 制作的非常有趣的视频剪辑。好吧——我承认这是一个相当薄弱的要求 SitePoint 为我购买一些昂贵的机器的基础，但它起作用了！为了证明我不只是说说而已，我正在拍摄一些办公室生活的片段。在接下来的几个月里，请关注我们的[脸书](http://www.facebook.com/sitepoint)页面。

这周我听到的最后一句话远没有这么积极。你们中的许多人可能听说过目前肆虐澳大利亚昆士兰州的灾难性洪水。虽然我们的 SitePoint 办公室位于另一个州，因此不受影响，但我们真的想做些什么来帮助解决问题。我们收到了很多你们的请求，希望能像几年前我们为维多利亚森林大火募捐一样，进行一些募捐活动，我们很喜欢这个想法。我们正在制定这样一个募捐活动的细节，但是首先必须满足一些法律要求，所以我们正在尽最大努力组织这个活动。如果一切按计划进行，我们希望在本周晚些时候宣布一些事情。
