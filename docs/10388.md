# 标志设计趋势:像素

> 原文：<https://www.sitepoint.com/logo-design-trends-pixels/>

上周，我开始了一个系列，着眼于过去一年中标志设计的一些明显趋势。虽然关注趋势很有趣，而且这个系列的目的是既有教育意义又有启发性，但重要的是要记住，趋势来来去去，而优秀的经典设计永远看起来很好。因此，向别人学习，但不要忘记像他们说的那样“犁自己的犁沟”。我们上周看了“转变”，今天让我们看看基于像素的标志设计。

网络上的图标让我们领略了一些著名的图像和符号缩小成小方块后的样子。像素(图片元素的缩写)是数字图像的最小部分。像素本质上是位于图像特定部分的一小块颜色。作为数字图像的重要组成部分，设计师在他们的标志设计中使用看起来像巨型像素的东西可能并不奇怪。

就其本质而言，使用像素创建的符号往往看起来非常基本，而且往往非常抽象。这些符号是彩色的，但通常围绕一种单一的颜色有一系列的色调。它们传达了元素聚集在一起创造一个可识别的图标的想法。像素风格的设计需要技巧来有效地完成，否则它们会给人一种(在电脑上)画得很糟糕的印象。

现在，为了给你带来视觉享受和灵感，我们收集了一些以像素为主要设计元素的徽标。(其中一些企业的名称中有 Pixel 或 Pix，这可能不是巧合。)
