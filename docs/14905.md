# 快速闪存的 12 个步骤！

> 原文:[https://www.sitepoint.com/steps-faster-flash/](https://www.sitepoint.com/steps-faster-flash/)

所以你希望你的 Flash 网站加载得更快？你当然知道！

本文将分享许多有用的技术，可以用来减小 Flash 电影的文件大小，从而加快它们的加载速度。

这不是一篇“如何做”的文章，而是提供如何减小 Flash 电影大小的信息。一旦你意识到可用的技术，你就可以自己去实现它们。

***1。对于在电影中多次出现的每个元素，使用动画或其他形式的符号。*T3】**

元件是电影库中可重复使用的元素。当您将一个元件放在舞台上或另一个元件内部时，您正在处理该元件的一个实例。与使用单个图形不同，您可以使用给定元件的许多实例，几乎不会增加电影的文件大小。

将所有作品转换成符号最终会节省你的带宽。因为 Flash 引用单个符号来生成多个实例，所以用户只需下载一次图形，即使您在站点中使用它一百次。

此外，在更改电影时，符号可以节省您的时间和麻烦。这是因为您对元件所做的更改会反映在整个电影中该元件的每个实例中。

***2。使用极少的光栅图形。*T3】**

虽然 Flash 可以让您在电影中添加光栅图形，但它确实不是首选图形。如果使用矢量图形，Flash 会开心很多。

对于那些不熟悉的人来说，矢量图形和光栅图形的简单定义是:矢量图形是用于构建图形的形状和线条的简单数学定义。因为您的图形是由数字和等式创建的，所以您保持了完全可缩放的图像。另一方面，BMP、JPEG 和 GIF 等光栅文件格式不是由数学定义的，而是由一系列按行排列的像素定义的，每个像素(或点)都应用了颜色值。

因为矢量图形本质上只是数学等式，当公式化时，产生图形，它们占用很少的存储空间，并且几乎不增加电影的整体文件大小。

虽然这对一些人来说可能是旧闻，但希望这些信息可以帮助 Flash 新用户避免犯下严重的错误，即利用他们磨练的 Photoshop 技能，试图将所有光栅图形放入 Flash 中。简而言之，偶尔使用一个漂亮的光栅图形可能是好的，只是不要试图从光栅图形构建您的整个 Flash 站点。

***3。更改 jpg 图像的质量。*T3】**

所以你完全忽略了上面的陈述，并继续添加。jpg 到你的 Flash 电影？好吧，至少让我们把它修剪一下。

按 CTRL + L 打开你的库，找到你导入的. jpg 文件，右击它，进入属性。取消“使用导入的 JPEG 数据”，你会发现你可以调整图像质量。你将不得不玩质量设置，直到你找到一个愉快的外观与文件大小的媒介。我总是调整这个设置——它本身就能对你的文件大小产生巨大的影响。

***4。动态加载图像。*T3】**

MX 的一个很棒的新特性是它给了你加载的能力。jpg 动态图形。这与您加载外部。swf —使用`loadMovie()`动作。向舞台添加一个 MovieClip，并为其指定“imageholder”实例名称。然后将以下 actionscript 添加到关键帧中:

```
loadMovie("name_of_jpg.jpg","imageholder"); 
```

警告！使用 loadMovie 动作动态加载时，渐进式 JPEG 图像不会出现。

***5。优化图像的其他有用技巧。*T3】**

使用菜单栏上的“修改—平滑、拉直和优化”命令来优化形状和线条也有助于减小 swf 的整体大小。您可以在形状上重复使用这些命令，直到您对优化程度或图形外观感到满意为止。

“修改”菜单中还有“描摹位图”命令。这个命令会将位图转换为矢量图像，并且通常做得很好——虽然，通常，转换的质量取决于图像的复杂性。通常情况下。gif 比. jpg 转换得更好。您可能想试试这个命令，看看它如何影响不同的图形。

***6。限制特殊线条类型的数量，如虚线、点线、锯齿线等。*T3】**

实线更小。用铅笔工具创建的线条比用画笔描边需要更少的内存。

***7。使用层来分隔在动画过程中变化的元素和不变化的元素。*T3】**

Flash 的一个很好的特性是，无论你使用多少层，这不会对你的 Flash 电影的大小有任何影响。因此，最好将非动画对象放在它们自己的层上，将它们与动画对象分开。

***8。限制字体和字体样式的数量。*T3】**

使用的字体越多，Flash 保存在中的字体就越多。swf——以及保存在。swf，文件就越大。很简单，对吧？

不过，这里有一个有用的提示。假设你刚刚得到了这个非常漂亮的字体，你想在几个按钮上使用它。为了让没有安装该字体的人看到该字体，您必须嵌入它。这非常简单——只需转到该字体的字符选项，您就可以选择嵌入整个字体、大写字符、小写字符、数字和标点符号。

如果你想要的只是一个按钮，上面写着“home”或其他简单的东西，那么嵌入整个字母表是没有用的，只需进入输入框，写着“and these characters”并输入“home ”,这样你在你的。swf 文件。

***9。将文本导入到 Flash 电影中。*T3】**

您可能想决定将文字与 Flash 电影分开。这使您可以轻松地进行修改，而不必打开 Flash，并且可以让不了解 Flash 或 Flash 知识的人轻松地进行更改。他们可以简单地编辑一个. txt 文件。另外，你的。swf 文件将变得更小，因为 Flash 不再负责存储您的文本。

如何创建外部文本文件并将其导入 Flash？

1.  Open up notepad and follow this pattern:

    ```
    variable_name=content&variable_name2=content& 
    ```

    内容附加在每个变量上，很快就会加载到 Flash 中。

    “`&`”是分隔符。保存您的。名为“content.txt”的 txt 文件。

2.  Next, add this script to a keyframe in Flash:

    ```
    loadVariablesNum("content.txt", "0") 
    ```

    这会将 content.txt 文件加载到电影的`level0`中。您也可以使用:

    ```
    loadVariables("content.txt","MovieClip") 
    ```

    它会用给定的实例名将内容加载到 MovieClip 中。

3.  最后，将动态文本字段添加到场景中，并为它们指定 content.txt 中使用的变量名。

因为您的文本不再驻留在您的 Flash 电影中，所以您减小了文件大小，并且您的主 Flash 电影现在将加载得更快:您不必预加载 150K 的 swf，您可能只需要预加载 100K 的 swf。

***10。优化 Flash 中的声音。*T3】**

尽可能使用 MP3 压缩，这是最小的声音形式。

导入 MP3 文件后，用 CTRL+L 打开媒体库，右键单击显示属性。您可能想要取消选中“使用导入的 MP3 质量”——取消选中此选项后，您可以更改比特率和质量。您可能会发现，较低的比特率和质量会减小文件大小，同时产生的声音仍然适合使用。

***11。使用“实例属性”对话框中的“颜色效果”菜单，可以为单个符号创建许多不同颜色的实例。*T3】**

符号的每个实例都有可以修改的特定属性。这些属性仅适用于特定实例，而不是原始元件。

例如，您可以使用所选对象的颜色属性中的“色调”选项来更改其颜色，而不是创建另一个符号。其他属性包括亮度和 alpha。通过多次使用一个对象的单个实例，您再次调整了整个文件的大小。

***12。模块化你的电影。*T3】**

模块化是将一部大型 Flash 电影分解成多个较小的过程。swf 文件，以减少加载时间，并使您的电影更容易管理。你可以通过阅读 SitePoint 题为[模块化 Flash](http://www.webmasterbase.com/article/944 ) 的文章来了解更多关于模块化的知识。

记住斯坦·李的话:“权力越大，责任越大。”负责任的 Flash 用户会采取重大措施确保他们的 Flash 站点尽可能快地加载，即使这意味着牺牲一点视觉效果。
