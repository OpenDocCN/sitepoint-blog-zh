# 离开你的 RSS！

> 原文:[https://www.sitepoint.com/get-off-your-rss/](https://www.sitepoint.com/get-off-your-rss/)

你曾经看过一篇写得很好的博客的每日阅读列表吗？有些列表长得惊人，包含的站点往往比大多数人一个月阅读的还要多！疯狂的事？当你阅读这些博客时，你会看到它们的作者与许多其他博客和网站的反应和争论。人们是如何记录这一切的？一个人如何能在如此频繁地查看这么多网站的同时保住一份工作？

他们聚集在一起。

大多数博客(包括 [SitePoint 博客](https://www.sitepoint.com/blogs))通过 [RSS](https://www.sitepoint.com/advertise/) 以及 HTML 发布内容。与定义文档布局的 HTML 不同，RSS 只是用 XML 记录原始文档。然后，软件可以下载 RSS 提要并根据需要对其进行格式化。因此，RSS 提供了:

1.  将内容整合到其他网站的简单方法

3.  一种独立于布局的内容服务方式

像 SitePoint 这样的网站不仅仅在博客上使用它。甚至[主文都是辛迪加](https://www.sitepoint.com/add-semantic-richness-to-your-markup-with-rdf-ease/)。传统出版物中不断涌现出新的有趣的 RSS 用法，如[BBC](http://holovaty.com/blog/archive/2002/09/03/1206)、 [Wired Magazine](http://www.wired.com/news/rss) 和 [Yahoo！新闻](http://jeremy.zawodny.com/blog/archives/001001.html)，到其他信息源，像加利福尼亚州的[紧急数字信息服务](http://edis.gigo.com/)。

RSS 聚合器将从您指定的网站下载最新的 RSS 源，显示最新的文章、帖子和新闻项目。这是最酷的部分:通过在一个地方列出所有网站的所有文章，聚合器让你可以快速阅读或拒绝文章。通常情况下，你可能会加载一个网站，却发现没有新的帖子。有了聚合器，你可以在新帖子出现时看到它们。

许多聚合器也会列出主要新闻条目和博客文章的评论。当你开始用 RSS 浏览世界时，你使用网络的整个方式都变了。取代了网页和搜索引擎，网络变成了一个相互联系的思想世界，一个以真实、有意义的方式联系在一起的讨论领域。

这里有一个流行的 RSS 聚合软件的简要介绍。

##### 桌面聚合器

如果你想一次跟踪许多博客，而不必打开一百万个浏览器窗口，那么你需要这些工具中的一个。

***视窗***

SharpReader 是一个很棒的免费 RSS 工具，它会以各种布局显示你喜欢的新闻和博客，包括一个漂亮的讨论线索视图。如果你自己开博客，你甚至可以安装一个所见即所得的博客编辑器插件。该编辑器使用 blogger 和 metaWeblog 兼容软件。

在 Outlook 中度过一生？News Gator 可能正是你想要的东西，但它会让你花费几美元。该工具将 RSS 源集成到 Outlook 中。

***OS X***

 *如果你是 Mac 爱好者，你有几个选择。MulleNewz 将让您轻松访问来自 Dock 的新闻提要。其他人对 2003 年 OS X 年度创新者奖得主 [NetNewsWire](http://ranchero.com/netnewswire/) ，一个商业 RSS/Weblog 工具，或其免费软件小弟弟 [NetNewsWire Lite](http://ranchero.com/netnewswire/featureschart.php) 信誓旦旦。

如果你想要一个关于 OSX 的开源解决方案，请阅读下面关于基于浏览器的阅读器的部分。

***GNU/Linux***

 *众多的 RSS 聚合器运行在 GNU/Linux 上，但是 [Straw](http://www.nongnu.org/straw/) 轻松胜出。通过其简单、强大、类似电子邮件的界面，跟踪最近的帖子非常容易。

***手机***

如果你太沉迷于在旅途中阅读 RSS 文章，检查一下你的手机是否能运行[mobilers](http://www.mobilerss.net/)。没有最新的手机？嗯，也许这是件好事…

##### 博客工具

参与博客世界比阅读有趣两倍。这些工具将网站编辑功能与 RSS 新闻阅读相结合:

Radio FM 为编辑博客和阅读新闻提供了一个很棒的 Windows 应用程序。简单、快捷、方便，物有所值。

如果你正在寻找一个开源的等价物，试试 [Python 桌面服务器](http://pyds.muensterland.org/stories/8.html )，一个带有 Web 界面的 Radio Userland 克隆。

Tinderbox ，一个伟大的 OS X 超文本/博客工具，包括一个 RSS 聚合器。在 Tinderbox 中，您可以将 RSS 文档与您的笔记、待办事项、流程图和博客帖子连接起来。

***浏览器基础***

如果您不想安装新的应用程序，或者不想让更多的窗口塞满您的桌面，请尝试以下浏览器插件之一:

新闻怪物是 Mozilla 的一个极好的 RSS 工具。这个跨平台的工具能做你所期望的一切，甚至更多！这个怪物与 Mozilla 紧密集成，它可以跟踪帖子和讨论线索。News Monster 甚至会通过找出你所阅读的作者阅读了哪些博客来帮助你找到新的博客。

Mozilla 的轻量级 [Firefox](http://www.mozilla.org/products/firefox/) 浏览器也有 RSS 阅读器面板。这个漂亮的插件在书签工具栏中工作，而不是实现一个新的对话框，为新文章创建一组自动更新的书签。

***网基***

如果你从一台电脑跳到另一台电脑，或者你想在家里和办公室同步你的新闻阅读，安装一个基于网络的 RSS 阅读器是最好的方法。

是一个易于安装的 PHP 脚本，应该可以在大多数 Web 服务器上运行。很简单，但是很好。

Amphetadesk 是一个更加复杂的跨平台 Perl 解决方案，它包括一个内置的 Web 服务器，供没有 Web 主机的用户使用。
