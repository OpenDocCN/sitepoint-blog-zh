# TagDragon Ultimate jQuery 自动建议

> 原文：<https://www.sitepoint.com/tagdragon-ultimate-autosuggest-jquery/>

我打赌你知道什么是自动暗示。如果没有，这是一个自动的建议过程，无论你在搜索文本字段中键入什么；就像谷歌的搜索一样，如果你输入一个字符，就会自动出现几条建议，这取决于用户是否选择它。

现在，一个 jQuery 自动建议插件 TagDragon 在上述功能方面提供了强大的功能。TagDragon 是膨胀、无黑客、优雅、灵活、健壮和快速的，可以应用于任何可能的环境。

## 关键特征

*   单值、多值和自由值
*   适用于输入框和文本区域
*   一行 JavaScript 代码
*   轻量级(< 5kb)
*   易于造型
*   跨浏览器支持

## 其他功能

*   完全 Unicode 支持
*   键盘导航
*   易于集成到任何后端
*   不与其他插件冲突
*   优雅地处理禁用的 Javascript
*   苗条的标记，没有诡计
*   允许每页有多个控件
*   可配置的值分隔符
*   可配置缓存
*   可配置延迟
*   可配置的结果限制
*   可配置的最小字符数
*   可配置后期数据
*   丰富的回调 API 提供更多控制
*   交替行
*   便宜，同时也支持一个慈善机构
*   友好的许可模式
