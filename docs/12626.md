# 寻找下一份网络开发工作的 20 个地方

> 原文:[https://www.sitepoint.com/20-places-to-find-your-next-job/](https://www.sitepoint.com/20-places-to-find-your-next-job/)

随着世界经济一天比一天不稳定，大公司已经开始削减劳动力，开始找新工作可能不是一个坏主意。像 Craigslist、Monster 和 HotJobs 这样的网站有很多列表，但它们没有得到很好的过滤。在这些网站上寻找工作可能是一件很糟糕的事情，许多最好的工作机会并没有在这些大型网站上发布。

如今，求职的热门趋势是更小的利基求职平台。我们收集了 20 个我们最喜欢的工作，这些工作适合程序员、网络开发人员、设计师和社交媒体专家。人们现在还在招聘，你只需要知道去哪里找。我们的列表在下面没有特别的顺序。

1.  site point Marketplace–site point 的 market place 可能是最著名的买卖网站的地方，但它也有一个很棒的“[招聘](https://marketplace.sitepoint.com/categories/looking-to-hire)”部分，在那里你可以找到大量的网站开发、设计和写作工作。
2.  37signals 求职板——37 signals 求职板的巨大成功可以说是引发求职板热潮的原因。它主要专注于编程和设计工作，但也吸引了一些商业/行政工作和文案职位。37signals 工作板被整合到 design magazine A List Apart 和 Ruby on Rails 官方网站，并有一个配套的 [gig board](http://gigs.37signals.com/) 用于较小的一次性工作。
3.  TechCrunch 是第一批添加工作板块的大型博客之一，并且取得了巨大的成功。CrunchBoard 往往会吸引很多商业/执行和编程工作。
4.  读写网工作委员会–每个博客都有稍微不同的核心受众，会吸引不同的雇主。可能会有一些重叠，但是你一定要试着确定哪个工作委员会的雇主与你想得到的职位最匹配。读写网的招聘栏倾向于吸引大部分网络开发职位。
5.  Ajaxian Job Board–Ajaxian 是一个面向 web 开发人员的工作平台，尽管它并不像你想象的那样以 JavaScript 为中心。
6.  Krop 是一个巨大的网页设计工作平台(虽然有时你会发现一些文案和项目管理的工作混在一起)，它被整合到一长串伟大的设计网站上，比如 morgueFile、COLOURlovers 和 Think Vitamin。辛迪加网站的列表保证雇主将继续使用 Krop，它将始终有一个很好的工作选择供您选择。
7.  [真实工作](http://authenticjobs.com/)–Cameron Moll 的真实工作是另一个优秀的平面设计工作板块，它被整合到顶级设计网站。上市合作伙伴包括 456 伯里亚街，IconBuffet，和减法。
8.  Ars Technica 工作板块–Ars Technica 是互联网上阅读量最大的技术网站之一。好消息是，这意味着他们的工作平台上有很多好的职位，坏消息是，这也意味着竞争非常激烈。Ars Technica 的工作板非常注重编程和 IT。
9.  Zend Job Board–Zend 运营着一个专门面向 PHP 编程工作的工作板，因此是寻找 PHP 相关工作的最佳地点之一。
10.  一年前，CenterNetworks 是一个 web 2.0 博客，主要为那些熟悉 web 2.0 的人所知。现在，所有者艾伦·斯特恩已经把它建成了互联网上顶级的 web 2.0 博客之一。因此，求职板已经成为在 web 2.0 公司找工作的好地方，尤其是在纽约地区。
11.  如果你是一个寻找设计和开发工作的自由职业者，自由职业者的 Switch 工作板可能是一个很好的起点。
12.  在过去的两年里，Smashing Magazine 已经成为网络上最受欢迎的网络开发博客之一。他们的求职公告板很受雇主欢迎，就像他们的网站很受读者欢迎一样，那里张贴着源源不断的编程和设计工作。
13.  [box and Arrows 求职板](http://jobs.boxesandarrows.com/jobs)–每月一期的网络开发杂志 box and Arrows 经营着一个受欢迎的求职板，吸引了许多 UX 和设计列表。
14.  设计师作品集网站 Coroflot 为设计、艺术和广告工作提供了一个很受欢迎的工作平台。每天大约有 15 个新的工作岗位被添加到网站上，所以请每天查看。
15.  [MetaFilter 工作板](http://jobs.metafilter.com/)——新闻网站 MetaFilter 也有一个工作板。这是一个相当折衷的列表，但通常吸引的主要是网络开发招聘广告。
16.  新鲜的网络工作——一个用于网络开发工作的工作板，该网站为求职者提供了一个书签，允许他们将其他网站和工作板中的工作保存到自己的新鲜网络工作帐户中。
17.  社交媒体工作-这可能是在社交媒体和在线社区管理新领域寻找工作的最佳地点。
18.  Slashdot 的工作公告板–Slashdot 的工作公告板是作为一个事后的想法被放在网站上的，所以它没有你想象的那么多的列表。然而，由于 Slashdot 的街头信誉，它确实吸引了一些高质量的 it 和编程工作。
19.  SearchWebJobs.com——一大堆 CSS gallery 网站联合了在 SearchWebJobs.com 列出的工作，这使得它成为寻找网页设计工作的好地方。
20.  ProBlogger 求职板–没有比 ProBlogger 求职板更好的地方来寻找付费博客工作了。

我们错过了什么好的工作机会吗？请在评论中告诉我们。
