# 8+ jQuery 360 度图像显示插件

> 原文:[https://www . site point . com/jquery-360 度-图像-显示-插件/](https://www.sitepoint.com/jquery-360-degrees-image-display-plugins/)

## 1.卷轴 1.1.3

这是一个 jQuery 插件，它采用一个图像标签，并使其成为预先构建的动画帧序列的实时“投影”。它的目的是提供对某物或某地的 360 度视角。广泛使用的 Flash 和 Java 技术的绝佳替代品。

## 4.图片-360 度旋转

 [![picture-360-rotation](../Images/fd177068a18a5458a4185d3d2dbb5817.png)](http://code.google.com/p/picture-360-rotation/)
