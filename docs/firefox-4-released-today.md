# 火狐 4.0 今天发布

> 原文：<https://www.sitepoint.com/firefox-4-released-today/>

经过一年的开发和几次进度延误，Mozilla 将于今天晚些时候在 GetFirefox.com 发布 Firefox 4.0 的最终版本。如果你等不及，我要告诉你一个小秘密……[火狐 4.0 已经可以从 Mozilla 的 FTP 服务器](ftp://releases.mozilla.org/pub/mozilla.org/firefox/releases/4.0/)下载。*不要告诉任何人！*

我们用火狐 3.x 已经很久了。3.0 版本于 2008 年 6 月发布，3.5 版本于一年后发布，3.6 版本于 2010 年 1 月发布。随着 Opera 的新界面 Chrome 的崛起，以及上周 IE9 的发布，这款浏览器看起来已经有点过时了。Firefox 可能仍然是开发的顶级浏览器，但其速度受到了批评，许多用户已经转向 Chrome。

火狐 4.0 可以改变这种情况。它包括:

*   一个新的，简化的用户界面。
*   新的 gerMonkey JavaScript 引擎和图形硬件加速提高了性能。
*   应用程序选项卡和选项卡分组。
*   内置书签，打开标签，历史，偏好和密码同步。
*   改进的隐私控制和安全性。
*   改进的 HTML5 功能，如新的表单元素、高清硬件加速的 WebM 视频和 3D WebGL。
*   CSS3 过渡和转换。
*   改进后的插件管理器。
*   某些触摸屏设备支持多点触控。
*   一个新的网络控制台。
*   碰撞保护。

此外，它有 70 多种语言版本，可以安装在大多数操作系统上。包括 Windows XP。

显然，另一种浏览器意味着 web 开发人员要付出更多的努力。然而，你可以同时安装 3.6 版和 4.0 版，你不太可能体验到根本的渲染差异，幸运的话，Mozilla 将在几周内推出 Firefox 4.0 作为自动更新。

你会安装火狐 4.0 吗？会帮助浏览器增加市场份额吗？还是太少，太晚了？请关注 SitePoint 上的完整评论…
