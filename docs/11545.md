# GazoPa 允许你勾画你的搜索

> 原文:[https://www . site point . com/gazo pa-lets-you-sketch-your-search/](https://www.sitepoint.com/gazopa-lets-you-sketch-your-search/)

我们都喜欢图片搜索，我知道我几乎没有一天不使用它。

图像搜索/处理技术也是一个在过去几年中经历了一些发展的领域。

Cooliris 给我们带来了一个漂亮的 3D 空间图像搜索界面。[字体](http://www.myfonts.com/WhatTheFont/)给字体识别带来了什么智能图像处理 [IDEE](http://ideeinc.com) 用 [Tineye](http://www.tineye.com/) 和 [Piximilar](http://ideeinc.com/products/piximilar/) 给了我们一些令人惊叹的视觉搜索工具。

![Gazopa Draw search](../Images/031ec954ef6417c51b78d63ecf2138c9.png)

上周，随着 GazoPa 公测版的推出，我们看到一名新玩家加入了图片搜索游戏。

与 Tineye 非常相似， [GazoPa](http://www.gazopa.com/) 是一种分析每张图像内容的服务——组成图像的形状、颜色、色调和线条——而不是像大多数传统图像搜索那样，仅仅根据文件名和周围的内容来推断图像的内容。

然而，GazoPa 也为它的弓增加了一些有趣的新弦。

首先，GazoPa 允许你直接在搜索面板中勾画出一个粗略的图像，然后它会尝试进行匹配。

当然，这在理论上是可以的，但实际上行得通吗？

在我的第一次快速测试中，是的，它似乎做了一个合理的工作。我开始画了一幅相当详细和独特的画——我的 iphone 的速写——结果当然很好。

然后我给了它第二个测试案例一个更高级的挑战——一个舞者的简单线条画——结果更加复杂。

![Results from a line sketch ](../Images/0071af8ffdcc48b661195c1902fc417f.png)

目前的算法似乎强调一般的组成，而不是特定的形状和线条配置。因此，大多数结果反映了我的(不重要的)偏心布局，而不是更重要的图形定位。

也就是说，显示在结果页面上的第五张图片(如上)实际上是一个非常好的匹配。

我倾向于认为我的第二个测试更有可能是真实世界的用例，但这是测试版，所以我们必须等待，看看随着他们积累更多的数据，事情会如何发展。

GazoPa 还发布了一款免费的 iPhone 应用程序来补充他们的标准网络服务。该应用程序允许您使用通过相机或手指涂鸦创建的匹配图像。

如果这个工作得很好，对他们来说，这就是一个现成的、冷酷的杀手级应用。你能想象在手机上快速拍照就能鉴别古董灯并给它定价吗？或者鉴定狗的品种？或者是建筑背后的建筑师。人们对各种可能性感到困惑。

再加上 Firefox 扩展、标准的 URL 和基于关键字的搜索，Gazopa 肯定已经准备好了。它还是生的，但看起来很有希望。
