# 14 个免费工具来验证你的 HTML，CSS 和 RSS 源

> 原文:[https://www . site point . com/14-free-tools-to-validate-your-html-CSS-RSS-feeds/](https://www.sitepoint.com/14-free-tools-to-validate-your-html-css-rss-feeds/)

没有什么比创建和设计一个网页更令人烦恼的了，认为你已经做得很完美了，然后在测试的时候发现了一些未知的显示错误，而你就是想不出来。修复此类错误的技巧之一是先**验证你的标记和样式表**。幸运的是，有很多免费的工具可以检查你的 HTML、CSS 甚至 RSS 提要是否有效。有了这些方便的助手，你可以确保你的访问者完全按照你的意愿来体验你的内容。

我们收集了 14 种工具——包括在线工具和一些 Firefox 扩展——所以你可以找到最适合你需求的。

## Firefox 插件

**CSS Validator** :向 W3C CSS validator 添加一个易于使用的链接，您可以通过右键单击上下文菜单或从工具菜单下访问该链接。

[![cssvalidatorcontexts](../Images/cd04a18e543b3532b96e496df1bf3f30.png "cssvalidatorcontexts")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/cssvalidatorcontext.png)

**[Firebug](http://getfirebug.com/)** : Firebug 是一个全功能的调试器和编辑器，允许你处理页面的 HTML、JavaScript、CSS、DOM 等等。您还可以使用该扩展来实时监控 JavaScript、CSS 和 XML，查找其中可能出现的错误，并了解需要做些什么来修复它们。Firebug 是几乎每个设计师的工具库中的一个基本工具，它已经变得如此常见，甚至开始获得自己的扩展(例如 SitePoint 自己的 [CodeBurner](http://getcodeburner.com) 参考工具)。

[![firebugs](../Images/59ec5aff5adaa52d572ced3a974b9962.png "firebugs")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/firebug.png)

HTML Validator :基于 Tidy 和 OpenSP，HTML Validator 给你一个简单的图标通知你正在访问的任何页面的有效性。您可以从该工具请求更多信息，当您查看页面源代码时，会突出显示导致页面无效的错误。更妙的是，如果你自己也搞不清楚哪里出了问题，扩展会为你提出建议。

[![html-validators](../Images/345c2a9d79920068bd52c58b12ddd3b2.png "html-validators")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/html-validator.png)

**页面验证器**:无论是通过打开右键上下文菜单，还是进入工具并选择“验证页面”，页面验证器都会打开一个单独的选项卡，并使用 W3C 的在线验证工具向您显示结果。

[![validatorcontextmenus](../Images/334d3e108505d852a3b22144c9ea94f8.png "validatorcontextmenus")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/validatorcontextmenu.png)

**RSS Validator**:RSS Validator 允许您通过使用右键上下文菜单或从工具菜单中进行选择来轻松检查 RSS 提要的有效性。然后，您将被转到一个单独的选项卡，以查看结果和任何潜在的错误。

[![rsscontextmenus](../Images/ab14db95bc2d0603e81b5c4a3e57d973.png "rsscontextmenus")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/rsscontextmenu.png)

**[Total Validator](http://www.totalvalidator.com/tool/extension.html)**:Total Validator 在一个方便的扩展中给了你大量的工具。浏览到你想要的网页，点击“电视”图标，并根据多个版本的 HTML 进行验证，拼写检查，截图等等。

[![total-validators](../Images/8d43b59024fb50e848aa60af4f7c01c2.png "total-validators")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/total-validator.png)

 **[![validatys](../Images/521c38f5815516bab3dc731821d09535.png "validatys")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/validaty.png)

## 基于网络的验证器

[](http://validator.w3.org/)**:W3C(万维网联盟)是开发所有网络技术标准的组织，所以他们有一个验证器来检查你的 HTML 是否正确是有意义的。要检查的文件可以是在线的或上传的，validator 可以以多种格式显示它的报告——包括建议、大纲、建议等等。如果你要检查你的标记，最好是让那些规定什么有效的人来检查。**

 **[![w3cs](../Images/cee1e9415a3f2eea329451c555ae48be.png "w3cs")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/w3c.png)

[](http://jigsaw.w3.org/css-validator/)**:W3C 还提供了一个 CSS 验证工具，它还会检查你的标记是否有潜在的错误和警告。您还可以选择设置不同的 CSS 配置文件，指定创建样式表的媒介，以及控制报告应该显示多少信息。**

 **[![w3ccsss](../Images/bf4b95917801b8e54edf783a767e6b94.png "w3ccsss")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/w3ccss.png)

:feed validator 首先会确保你传递给它的 feed 是有效的，然后会仔细检查你最近的条目，给你一个问题的总结，突出问题存在的实际行。非常方便地确保您的提要在各种阅读器中正确显示。

[![feedvalidators](../Images/7551880de9b5add597fa5d8dea6ae833.png "feedvalidators")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/feedvalidator.png)

**[Relaxed . VSE . cz](http://relaxed.vse.cz)**:Relaxed 不使用官方的 W3C 规范进行验证，而是使用自己的一些来检验你的文档。您可以选择它使用哪个版本的 HTML，是否显示源代码，使用“脏”解析器，还有一些其他选项。它也可以作为一个[火狐插件](http://relaxed.sourceforge.net/firefox-plugin.html)使用，直接将服务添加到你的浏览器中。

[![relaxeds](../Images/dd2c256c6df846d328be2cd73c336d19.png "relaxeds")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/relaxed.png)

[](http://www.validome.org/)**:Validome 就像你的验证器一站式商店。这个网站提供了检查 HTML、XML、DTD 模式、RSS 和 Google 网站地图的工具。每个工具都提供了许多选项，因此您可以根据自己的需要进行设置。**

 **[![validomes](../Images/ff12545d8f3e64a68199c0acf352ffc5.png "validomes")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/validome.png)

:virtual promote 为开发人员提供了众多用于不同工作的工具，但对于前端开发人员来说，三个重要的工具是 HTML、CSS 和 XML 验证器。

[![virtualpromotes](../Images/9dcc4d08c5cc617772d83951d3861ca6.png "virtualpromotes")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/virtualpromote.png)

**[【xhtml-css.com】](http://xhtml-css.com/)**:xhtml-css.com 给任何网站一个快速浏览，同时验证 HTML 和 CSS。该服务将列出您可能需要注意的任何错误和可能的警告，并提供有关问题性质的详细信息。您还可以使用更多高级选项为 HTML 设置不同的字符编码设置，对于 CSS，您可以设置样式表将用于的配置文件和介质。该网站还提供了一个名为 [BeValid](http://xhtml-css.com/extension) 的 Firefox 扩展，可以让你验证当前正在访问的 URL，从而进一步加速这个过程。

[![xhtml-csss](../Images/f798006f15f2462044df5f4ba5bf1173.png "xhtml-csss")T2】](https://www.sitepoint.com/wp-content/uploads/2009/05/xhtml-css.png)
