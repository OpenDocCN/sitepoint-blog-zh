# 一点谈判策略

> 原文：<https://www.sitepoint.com/a-little-negotiation-strategy/>

商业关系中的几乎所有事情都可以归结为两种紧张关系:建立关系和获得结果。如果你太专注于建立关系，你会倾向于避免冲突，无法实现目标。如果你太专注于获得结果，关系会受到影响，你不会获得长期的结果(因为人们会对你小心翼翼)。

当然，你必须知道什么时候更关注关系，什么时候更关注结果。

例如，我现在正在为我的一个网站(http://www.momsonedge.com)与一家产品制造商合作。制造商在产品交付前两天坚持涨价。他声称，尽管有我几个月的评论和一份合同，产品比他想象的要难做。

与此同时，他也交付了一些好的产品，并且很好合作。

因此，我现在在“结果”方面施加了一点压力，告诉他他的要求是不合适的，以及为什么，并威胁说，如果他继续坚持更高的价格，就去找另一家价格更有竞争力的供应商。

这种结果和关系的不断平衡出现在销售、项目管理、谈判中——实际上是任何一种商业关系。

希望你正确管理余额！

根据我的经验，很多技术人员倾向于避免冲突，甚至逃避冲突。所以他们有不真实的关系，没有足够的努力去得到他们想要的结果。
