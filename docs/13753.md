# moz Backup——火狐和雷鸟档案的备份工具

> 原文：<https://www.sitepoint.com/mozbackup-backup-tool-for-firefox-thunderbird-profiles/>

虽然这不是严格意义上的 web 开发，但我认为我们中的许多人可能(如果你还没有)在某个时候遇到过这个问题——尤其是在使用 Firefox 时——所以这里值得一提。

这个星期，我发现设置我闪亮的新双液晶电脑的忙碌被单调乏味的任务冲淡了，那就是让我喜欢的所有软件和设置再次设置正确。当然，这伴随着设置一台开箱即用的计算机的领域，但是事情通常比它们应该有的要困难得多。

雷鸟是一个很好的例子。虽然它运行得很漂亮，但我对跳圈的[水平感到非常惊讶，它预计会做一些简单的事情，如将邮件和设置从一个安装移动到另一个安装。显然，我并不孤单，因为 Matt (Magain)说他必须将邮件导入 Eudora，然后导出到 Thunderbird 才能获得满意的结果。](http://forums.mozillazine.org/viewtopic.php?t=25720)

无论如何，好消息是我找到了一个第三方解决方案，它不仅能可靠地传输雷鸟的邮件、过滤设置、账户和扩展，还能备份你的 Firefox 书签、扩展(不是 100%可靠)、保存的表单细节、保存的密码、cookies、证书、下载文件列表和历史记录。

[MozBackup](http://mozbackup.jasnapaka.com/) 是 Pavel Cvrcek 的一款客户端应用。在一个小时毫无结果的文件夹切换之后，我用 Mozbackup 在客户端之间通过 4 到 5 次点击传输了超过 1g 的邮件。它简单、有效，最重要的是，它是免费的。
