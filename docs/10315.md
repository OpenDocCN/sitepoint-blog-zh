# 再见 2010-你好 2011！

> 原文：<https://www.sitepoint.com/goodbye-2010-hello-2011/>

在 SitePoint 总部度过了多么美好的一年。

2010 年，我们推出了一系列互动在线视频课程，主题包括 HTML、CSS、博客以及最近的脸书广告。这些课程非常受欢迎，并提供了一种新的互动学习方式。我们对 2011 年即将推出的课程感到非常兴奋，所以请继续关注并查看当前提供的课程。

今年还与 Amazon Web Services 合作发布了期待已久的标题[“在云中托管您的网站”](https://www.sitepoint.com/books/cloud1/)，这已被证明是一个非常受欢迎的资源。

我们非常兴奋地宣布，我们的电子书在 3 月份就可以使用 EPUB 和 MOBI 格式(不包括四本书)，这意味着您可以随时随地在所有您喜欢的设备上阅读我们的书籍！

最后，我们今年有两次疯狂有趣的促销活动，第一次是在 7 月，为庆祝世界杯和现在著名的“SitePoint 圣诞倒计时”而举办的

我们的圣诞销售非常有趣，我们收到了很多积极的反馈。每个参与者都收到了一个价值 397 美元的优惠券包，折扣高达 75%，适用于 Wufoo、Harvest、Visual Website Optimizer、User Testing、Crazy Egg 等公司。

我们要感谢你们所有人的持续支持，并祝愿你们在 2011 年一切顺利。

回头见！
