# 官方:Windows 源码泄露

> 原文:[https://www . site point . com/official-windows-source-is-leaved/](https://www.sitepoint.com/official-windows-source-is-leaked/)

昨天只是谣言，但是今天微软已经证实 Windows 2000 和 Windows NT 4 的部分源代码已经泄露到互联网上。

这很严重。不仅仅是为了微软，也是为了他们产品的数百万消费者，包括我们。网络编码员。根据你在微软划分中的优先顺序，一个可能的好处是，第三方现在可以潜在地查看操作系统的内部工作，因此有希望产生可以在集成方面与微软竞争的应用程序(这对我们有利，对微软不利)。但对我们大多数人来说，这意味着坏消息。黑客寻找新漏洞和释放新病毒的工作变得很容易，因为现在支持 Windows 2000、NT 4、部分 XP 和 Server 2003 的一些源代码是可用的。这对每个人都很严重，尤其是那些运行 Windows 2000 服务器的人。正如一个人在 Slashdot 上说的那样，“Windows ME 之后的任何东西都会受到伤害”。

但是不要走极端，完整的源代码还没有被泄露(完整的源代码大概有 40Gb 大小……这次泄露的大约有 600Mb 大小),但是关键的源代码在病毒创造者的手里绝对不是什么好事。但是他们必须非常熟练才能理解代码，一般的程序员做不了多少。

在接下来的几天里，预计会通过大众媒体听到更多关于此事的消息，预计 MSFT 的股票会下跌。
