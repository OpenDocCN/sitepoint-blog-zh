# IcePHP——PHP 的一些重要的 RPC？

> 原文：<https://www.sitepoint.com/icephp-some-serious-rpc-for-php/>

这里有一些有趣的东西 [Ice for PHP](http://www.zeroc.com/icephp.html) 。

什么是[冰](http://www.zeroc.com/ice.html)？

> 互联网通信引擎 Ice 是面向实用程序员的中间件。作为一个高性能的互联网通信平台，Ice 包含了丰富的分层服务和插件。冰意味着简单、速度和力量。

它似乎很大程度上受到了 CORBA 的启发，但却是出于一种非常特殊的需要而开发的——给大型多人在线游戏《T2 的愿望》带来了一些严肃的合作。对 PHP(仅作为客户端)的支持从一开始就是一个目标，允许通过网络管理游戏，并以 PHP5 扩展的形式出现(必需)。

无论如何——还没有完全开发出来——有很多可以阅读，但是到目前为止，当你需要比基于 XML 的 web 服务更强大/性能更好的东西时，它看起来很有希望成为一种让 PHP 与远程系统对话的机制。第一印象是[零零](http://www.zeroc.com/)知道他们在做什么。
