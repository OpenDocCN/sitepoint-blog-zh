# 设计颜色:棕色

> 原文:[https://www.sitepoint.com/color-in-design-brown/](https://www.sitepoint.com/color-in-design-brown/)

[![image](../Images/bb06fa5a103323a84adfe422cb4c94fb.png "image")](https://www.sitepoint.com/wp-content/uploads/2010/02/image112.png) 上周我们看了一下[紫色](https://www.sitepoint.com/color-in-design-purple/)，一种充满活力的浓郁颜色。今天，我们谈论布朗。在某些方面，棕色几乎是紫色的对立面。棕色被认为是朴实、自然、简单的。它传达了可靠、健康和保守的感觉。从不好的方面来说，它也可能被认为有点沉闷，但这可以通过与明亮的颜色相结合来缓解。

棕色与木材关系密切，在较小程度上与石头有关。几年前，有太多木制背景的网站。这种趋势现在似乎已经结束了，但是如果做得好的话，“木制”网站会引人注目，给人一种很好的质感。当然，棕色也与古老的照片和材料联系在一起，可以用来创造一种温暖和怀旧的感觉。

**使用棕色**

棕色有许多深浅和颜色，它们的许多名字来自大自然。浅棕色包括沙色、奶油色和米色，中棕色包括巧克力色、可可色，深棕色包括咖啡色和桃红色。

使用浅色和深棕色可以产生丰富的组合，如果你想到秋天的颜色，你会看到大自然用红色、橙色和黄色展示棕色——一个传达温暖的现成调色板。

棕色配绿色很土，棕色配薄荷绿很时尚。黄色和橙色使深棕色变亮，同时看起来仍然“自然”如果你想避免使用黑色，深棕色可以是一个很好的替代品，不会显得太硬。
