# 垃圾邮件与法律

> 原文：<https://www.sitepoint.com/spam-and-the-law/>

根据法律有垃圾邮件，然后根据你和我有垃圾邮件。虽然 2003 年的 CAN-SPAM 法案的确为起诉一些群发邮件者制定了可执行的法律，但是你我认为的大多数“垃圾邮件”实际上是合法的邮件。这就是为什么许可在电子邮件营销中如此重要。我们都不喜欢收到垃圾邮件，垃圾邮件的定义会根据你的谈话对象而变化。

## 为什么许可很重要

如果你合法地向某人发送电子邮件，而他认为这是垃圾邮件，并在电子邮件客户端将其标记为垃圾邮件，会发生什么？您可以:

*   被互联网服务提供商(ISP)列入黑名单
*   关闭您的帐户和/或终止托管帐户
*   如果你违反了 CAN-SPAM 等法律，就要承担法律责任

## 什么构成许可

权限必须是明确的，不能转移。仅仅因为客户表示他们希望收到订单信息，这并不意味着你可以向他们发送促销材料。你必须得到许可。那么什么是许可呢？

*   书面许可——此人填写一张卡片，订阅您的列表。
*   电子许可—客户填写在线表格，例如由电子邮件营销服务提供的表格。
*   先前存在的关系——如果你在过去两年中向该公司销售过产品或有财务关系，法律允许你发送电子邮件。我告诫不要简单地将客户添加到列表中；相反，发送一封独特的电子邮件，要求他们订阅新列表。这是一个修剪清单的好方法。

确保你在电子邮件营销的法律和道德方面，并总是寻求许可。如需更多信息和进一步阅读，请查看以下资源:

*   [CAN-SPAM 法案商业合规指南(网络版)](http://business.ftc.gov/documents/bus61-can-spam-act-Compliance-Guide-for-Business)
*   [商业反垃圾邮件法案合规指南(PDF)](http://www.internetformulaofsuccess.net/Docs/CAN-SPAM%20Act.pdf)
*   [澳大利亚垃圾邮件法案 2003 商业指南(PDF)](http://www.acma.gov.au/webwr/consumer_info/frequently_asked_questions/spam_business_practical_guide.pdf)
*   [直销协会反垃圾信息资源](http://www.the-dma.org/antispam/)
*   [各国反垃圾邮件法](http://spamlinks.net/legal-laws.htm)

## 内置权限

使用第三方电子邮件营销解决方案的最大好处之一是它强制(尽可能)遵守 CAN-SPAM 法案。它还要求您验证您的列表是根据其严格的权限指南获得的。
