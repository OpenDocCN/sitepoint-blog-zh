# 顾客永远是对的？网飞抛弃了 Qwikster

客户并不总是对公司在产品供应、营销策略或总体方向上的改变感到满意。

这一点在最近网飞变革引发强烈反对后变得很明显。首先，该公司制定了一个重大的价格上涨，客户在互联网上抱怨，许多人表示他们将完全取消他们的账户或降级到两个计划中较小的一个。

然后，网飞决定将它的两项服务(邮寄 DVD 和在线即时观看电影)拆分成两家完全独立的公司。这一举动不仅引起了客户的抱怨，也引起了投资者和分析师们的争论，他们认为这一举动是明智的还是最糟糕的决定。

这是否是一个明智的商业决策，我们永远不会知道。由于饱受批评，网飞最近宣布他们已经放弃了拆分公司的计划。网飞屈服于严厉的批评并放弃他们的计划，这可能会花费他们数百万美元并逆转数月的计划，这是正确的吗？

我认为你应该总是倾听你的顾客，但是顾客并不总是对的。

然而，如果你的大量客户抱怨最近的政策或产品变化，你一定要注意。单个客户可能不正确，但当大多数人说话时，请倾听他们！

你如何看待网飞放弃 Qwikster 品牌的决定？他们做出改变是正确的，还是这个决定太小、太晚了？
