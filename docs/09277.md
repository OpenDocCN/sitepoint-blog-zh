# 使用 jQuery 对浏览器进行了 10 次调整

> 原文:[https://www.sitepoint.com/ie-browser-fixes-jquery/](https://www.sitepoint.com/ie-browser-fixes-jquery/)

我们已经分享了大量的浏览器调整插件，我们知道你还没有得到足够的，因为你只是喜欢修复浏览器错误！；)本帖，我们再分享一套。看看 jQuery 的这 10 个浏览器调整，你可能会觉得有用。

## 1.wresize–IE 调整事件修复插件

jQuery 插件，用于修复 IE 窗口调整大小的错误。在 IE 中，每次实际调整窗口大小时，窗口大小调整事件都会被触发多次:这是 IE6 和 IE7 的一个众所周知的问题，但是它们的行为模式不同。其实好像 IE6 比 IE7 差。
![Wresize.jpg](../Images/4086ed0511748ba0d5382e40f0afed19.png)
[来源](http://noteslog.com/post/how-to-fix-the-resize-event-in-ie/)

## 2.jQuery ifixpng

这个插件旨在通过对用户指定的元素应用适当的过滤器来解决这个问题，同时保持所有元素标签的完整性。
![jQuery-ifixpng.jpg](../Images/c004e76b94564f6bdbe30cd5606e50b8.png)
[来源](http://jquery.khurshid.com/ifixpng.php)

## 3.jQuery pngFix

此插件将修复 Windows Internet Explorer 5.5 中缺失的 PNG-Transparency。
![jQuery-pngFix.jpg](../Images/07d99c4756b6b0dd83656a8c11f4b043.png)
[来源](http://jquery.andreaseberhard.de/pngFix/)

## 4.背景 Iframe

bgiframe 插件是可链接的，它应用 iframe hack 来解决 IE6 中的 zIndex 问题。它将只在 IE6 中应用自己，并向 iframe 添加一个名为“bgiframe”的类。iframe 被附加为 tabIndex 和 zIndex 为-1 的匹配元素的第一个子元素。
![Background-Iframe.jpg](../Images/439f10f9f63f528cc9c8b89440c8da09.png)
[来源](http://brandonaaron.net/code/bgiframe/docs)

## 5.聂琪-为了在 IE 中正确显示 Q 标签

Q 标签(在 HTML 4.0 和 W3C DOM Level 1 中定义)的目的是在语义上、可访问性上和排版上更正确地表示引文。像许多东西一样，它们没有流行起来，因为 IE 不支持它们。
![QinIE.jpg](../Images/a41946b5331ca14d9765680fb612d128.png)
[来源](http://plugins.jquery.com/project/QinIE)

## 6.jQuery 可访问性插件

要使用 jQuery 可访问性插件，首先你必须有 jQuery 和 jQuery Cookie 插件。这个插件已经在 Ubuntu 7.04 上的 FireFox2 上的 1.1.3 版本中测试过了。
![jQuery-Accessibility-Plugin.jpg](../Images/94cb2ecb4c7107018d4eefc2f66671ab.png)
[来源](http://webrocket.ulmb.com/ability/)

## 7.jQuery 鼠标滚轮插件

有两种方法可以使用 mousewheel 事件。您可以使用 mousewheel 和 unmousewheel 辅助工具，或者只使用典型的 bind 和 unbind 语法。
![jQuery-MouseWheel-Plugin.jpg](../Images/01fadad1acf46d41c204ebba2c66b71e.png)
[来源](http://brandonaaron.net/code/mousewheel/demos)

从链接中删除虚线焦点。
