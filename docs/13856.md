# 所有 PPC 营销人员请注意

> 原文：<https://www.sitepoint.com/type-in-domains-seo-traffic/>

《商业 2.0》杂志有一篇关于域名投资者的深入文章，这些人持有数千，有时是数十万域名，然后将他们的流量转售给谷歌和雅虎，最终是你——点击付费广告客户。

为什么这很重要？

简而言之，在谷歌和雅虎上的数十万广告客户中，很少有人知道他们正在为大量域名组合的流量支付高价，这些域名组合由个人和公司持有，如 [NameAdmin](http://nameadmininc.com/about.html) 和 [Marchex](http://www.marchex.com/) ，他们从其既定的利基市场中获利丰厚:

> 施瓦茨拥有大约 5000 个名字，其中不到三分之一属于“成人”类别。他是这个行业最大的推动者，向任何愿意倾听的人宣扬域名的力量，将域名投资者与谷歌(Google)和雅虎(Yahoo)等公司的投资者和高管聚集在一起。他的左手腕上戴着一块价值 65000 美元的劳力士手表，右手上戴着一个价值 32000 美元的钻石手镯，令他震惊的是，他这个社区大学的辍学生竟然在博卡拉顿的一栋海滨别墅里过着国王般的生活。
