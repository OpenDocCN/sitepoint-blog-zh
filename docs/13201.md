# SitePoint 图书被评为 2007 年最佳图书

> 原文:[https://www . site point . com/site point-books-rated-best-of-2007/](https://www.sitepoint.com/sitepoint-books-rated-best-of-2007/)

SitePoint 的两本书被加拿大书评网站 [YYZtech.ca](http://www.yyztech.ca/) 评为 2007 年最佳网页设计书籍。

[![](../Images/542d0717e6e2f3690f1c3d77f9e2c8ed.png)](https://www.sitepoint.com/books/cssant2/) 雷切尔·安德鲁的[《CSS 选集:101 个基本技巧、诀窍& Hacks(第二版)](https://www.sitepoint.com/books/cssant2/)榜上有名。以下是评论者不得不说的话:

> 又一本关于 CSS 的书？是的，这本书(第二版)是最新版本的 IE 浏览器，设计得很好，很容易阅读……有吸引力的全彩色包装。此外，整本书还有许多提供额外信息的分组框..有幽默感:例如，警告部分用从自行车喇叭(寻找)到手榴弹(小心)的各种项目来说明。

[![](../Images/0a08916343e1a0a1f290c5b5616b927b.png)](https://www.sitepoint.com/books/checklists1/) 同样上榜的还有雪莉·凯瑟的[交付一流网站:101 个必备清单](https://www.sitepoint.com/books/checklists1/)。从[开始评审](http://www.yyztech.ca/articles/view/deliver_first_class_web_sites/):

> 当你阅读这几章的时候，你会发现很多经验都暴露出来了——看看几乎每一页的脚注，它们都是对其他资源的引用，如果读者希望了解更多关于某一点的信息，他们可能会去调查。

一组辛勤工作的员工(委托、编辑、营销等)参与了一本书的创作，因此不仅仅是作者对最终结果感到自豪。我们喜欢看到包含 SitePoint 书籍的评论和列表，所以如果你知道任何其他的，一定要让我们知道！
