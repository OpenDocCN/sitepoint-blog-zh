### [dannyvankooten/vat . PHP](https://github.com/dannyvankooten/vat.php)[162★]

根据他们的文件:

> vat.php 是一个简单的 PHP 库，帮助你处理欧洲增值税规则。它帮助你…
> 
> *   获取任何欧洲成员国的最新增值税税率
> *   验证增值税编号(按格式或存在)
> *   使用 ISO 3166-1 alpha-2 国家代码，确定它们是否属于欧盟。
> *   地理定位 IP 地址
> *   图书馆使用 jsonvat.com 来获取增值税率数据。

这个库只不过是 jsonvat.com 的一个 API 代理，增加了一些实用函数。然而，这些实用功能和这个库的易用性使它成为任何电子商务网站的供应商目录的一个有价值的补充。它甚至支持检查一般的国家数据，如下所示:

```
$countries = new DvK\Vat\Countries();
$countries->all(); // array of country codes + names
$countries->name('NL') // Netherlands
$countries->europe(); // array of EU country codes + names
$countries->inEurope('NL'); // true
$countries->ip('8.8.8.8'); // US 
```

### 微软/tolerant-php-parser [210 ★]

微软？他们真的需要提升/帮助吗？好吧，考虑到这个方案的本质和它隐含的效用，我投赞成票。

容忍 PHP 解析器旨在与 ide 一起使用，通过生成 AST(抽象语法树)来检查代码。两个主要的品质(或者说计划的品质，因为它还很早)是这样一个事实:它是容错的，所以当检测到不完整的代码时不会返回一个完全损坏的 AST(默认情况下，IDE 中的大多数代码都是不完整的，因为它正在编写过程中)，以及它是用 PHP 编写的，但构建时考虑了 IDE。后者特别方便，因为它让 PHP 社区自己贡献一个解析器来验证他们自己的代码，**和**在一堆 IDE 上开箱即用，而不是等待特定 IDE 的开发团队来应用补丁。

该工具正在积极开发中，该团队正在寻求帮助，以使其成为最终形式——这不是一个一次性的实验包，一旦作者获得新的工作，它就会被放弃。这是一个具有巨大潜力的专用工具，值得我们关注。看看吧！

### PHP 入队/入队-dev【103】

Enqueue 是 PHP 的一个“企业队列系统”。乍一看，很明显作者对这个包是认真的——开发库是所有开发发生的地方。生产版本以只读方式单独发布，包含在开发报告中开发的内容的子集。

我们在 SitePoint 这里已经对队列[进行了相当多的处理，但是总是针对特定的目的使用特定的实现。Enqueue 提供了一个实用的抽象排队包，支持开箱即用的多种协议，适用于任何框架、应用程序或 CMS。对于 Symfony 的粉丝来说，一个捆绑包是开箱即用的，但它绝不是 Symfony 独有的东西。](https://www.sitepoint.com/drupal-8-queue-api-powerful-manual-and-cron-queueing/)

如果你不熟悉队列，很难用几个简短的段落来解释，所以我建议你阅读快速浏览和写得很好的文档[这里](https://github.com/php-enqueue/enqueue-dev/blob/master/docs/index.md)。

回购有一些(乍一看)相对容易解决的问题，所以如果你觉得可以去提高你的贡献分数，如果你想尝试一下，别忘了联系我们——我们很乐意为你提供指导，并通过这种方式帮助图书馆增加曝光率。

### [vimeo/诗篇](https://github.com/vimeo/psalm)【243】

另一个著名的产品，诗篇是一个 PHP 的静态分析器，在 Phan 和其他人的精神下，它会在你写的时候(或根据需要)分析你的 PHP 代码，并输出它可能发现的一些差异和错误。

除了说它非常容易使用，并且是一个正在进行的开发项目，欢迎您的帮助之外，没有更多要说的了。去看看！

### [wax im/代金券](https://github.com/waxim/vouchers) [43 ★]

凭证是一个用于创建和验证凭证的库。

需要为在线产品生成折扣代码？访问共享文件的代码？推荐跟踪的促销代码？代金券可以有所帮助。

该库没有对存储做任何假设，这意味着他们不关心你如何做——作者没有强迫你接受任何特定的存储机制，这使得整个事情更加强大和普遍适用。

在撰写本文时，没有问题需要帮助，但他们可以使用文档和一两个教程的帮助，所以如果你觉得可以，我们有足够的预算来支付你！

### [https://github.com/themsaid/laravel-mail-preview](https://github.com/themsaid/laravel-mail-preview)【374★】

简明扼要——这个包可以让你把电子邮件从你的 Laravel 应用程序“发送”到你的本地硬盘。无论是出于静态文件备份的目的，还是为了易于版本控制和转发给 QA 的 HTML 预览，都没有关系。

### https://github.com/JordanRL/Fermat【12★】

> 为任意大小的数字提供数学和统计操作的库

听起来熟悉吗？也许是因为[这个](https://www.sitepoint.com/arbitrary-precision-big-numbers-php/)？是的，有一些其他的图书馆在处理这些事情，但这并不意味着没有创新的空间——每个新玩家都会带来一些见解，事实上，这个漂亮的现代图书馆相当成熟和稳定。

我将很快在一个项目中演示这个库，但同时，在开始之前，请记住以下警告:

> 这个库将某些常数(圆周率、圆周率、欧拉数和黄金分割率)硬编码为 100 位数。因为它执行的许多功能(如对数和三角函数)都依赖于这些常数，所以这个库实际上并不是真正的任意精度。相反，您可以处理精确到小数点后 100 位的数字，并且可以计算精确到小数点后 99 位的三角函数。
