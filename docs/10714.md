# 你会把工作托付给别人吗？

> 原文:[https://www . site point . com/would-you-trust-someone-other-with you-job/](https://www.sitepoint.com/would-you-trust-someone-else-with-your-job/)

当我外出度假时，我总是在努力使我的工作尽可能容易被负责的人照顾和确保我不会让它太容易以至于他们想知道我在这里做什么之间左右为难。我想我上周离开的时候是对的，因为一切似乎都井然有序，而且我还有工作。我失望地注意到，我为上周一精心准备并通过 WordPress 预定的社区博客帖子没有上线。我不确定我做错了什么。当我不在的时候，我们的 WordPress book 发布了，这也许是及时的！没关系。那个假期已经是遥远的记忆了。

今天对我来说是令人兴奋的一天。今天是我的生日。传统上，我会庆祝整整一周——被称为 Hawkstravaganza。谁知道会发生什么样的傻事。我可以确定的一件事是，我将不会得到一个和我们的脸书粉丝蒂姆得到的蛋糕一样酷的蛋糕。这家伙有一个非常有才华的妻子。查看她创作的[照片。](http://www.facebook.com/sitepoint?v=photos#!/photo.php?pid=6173627&op=1&o=global&view=global&subj=119345668711&id=566086822)

当我们谈到脸书时，我想借此机会提醒大家，我的新“了解我们”活动现在已经进入第二周。我不会像上个星期那样详细讲述[，但是如果你对在 SitePoint 工作的人感兴趣，以及为什么我们做我们所做的事情，请前往查看我们的](https://www.sitepoint.com/the-people-behind-sitepoint/)[脸书粉丝页面](http://www.facebook.com/sitepoint?v=app_11007063052)。上周我们采访了马克·西波拉，本周我们采访了砧板上的创意总监马特·马金。

![SitePoint Forums](../Images/ee81ae784c770ab3cd06ad322a9f7ec1.png "sitepoint-forums")

我不在的时候，我把 SitePoint 论坛的管理工作交给了我的志愿者团队。一如既往，他们挺身而出，实际上运行了整个[编程竞赛](https://www.sitepoint.com/forums/showthread.php?t=693031)。参赛者需要编写一个私人消息实用程序。我们现在已经进入了评审阶段，一旦有了获胜者，我们将会立即宣布。

对于那些对编程不感兴趣的人来说，我们推出了摄影教程系列中的第三个。在摄影原理系列的第二部分中，论坛导师 Shaun(of dead)谈到了曝光和快门速度。本周他继续第三部分，这是所有关于[焦距和视角](https://www.sitepoint.com/forums/showthread.php?t=694616)。

本周我要提到的最后一件事是，我们目前正在 SitePoint 招聘员工。我们正在寻找一个有才华的 PHP 开发人员加入我们的团队。如果你住在墨尔本(或者准备搬迁)，并且认为你符合要求，[看看职位描述](http://www.facebook.com/sitepoint?v=app_6009294086)并联系一下。
