# Opera 10.60 发布

> 原文:[https://www.sitepoint.com/opera-10-60-released/](https://www.sitepoint.com/opera-10-60-released/)

又是一天，又是一次浏览器更新！Opera 的发布时间表可能与 Chrome 相当，尽管很明显，他们不像 Google 那样经常更新主要版本号。

尽管是一个小版本更新，还是有几个新特性会引起开发者和 Opera 用户的兴趣…

Opera 声称已经对 JavaScript 引擎做了进一步的优化，使得速度比 10.50 版本提高了 50%。我不怀疑他们的基准，尽管在现实世界中是否明显是另一回事。也就是说，*感觉*要快一点。

**WebM 视频支持**
Opera 是第一个(非测试版)支持新的免版税 WebM 视频标准的浏览器。WebM 有可能在未来几年成为最主要的网络视频格式之一。

进一步的 HTML5 特性
所有的厂商都试图在 HTML5 的改进上超越彼此。Opera 10.60 现在支持:

*   地理位置—浏览器会询问您是否要与位置感知网站共享您的全球位置。
*   web Workers——类似线程的后台 JavaScript 处理。
*   应用缓存—离线 web 应用支持。

**提高安全性**
AVG 的反欺诈系统和网络钓鱼防护增强了浏览器的安全性。当用户试图访问包含恶意内容的网站时，应该会看到一个警告页面。

**各种更新**
Opera 团队致力于提高所有平台的稳定性和性能。很少有人会注意到主要的不同，但是歌剧迷会声称几乎不需要改变。

我注意到的唯一一个小的界面改动是当菜单栏被隐藏时，在 Opera 标志旁边增加了“菜单”。这不在 10.50 中，所以我猜有些人在定位选项时有困难。

如果你已经是 Opera 用户，你肯定会想升级。选择“帮助”>“检查更新”,它会神奇地为您更新。

新用户应该前往 opera.com。试试这个浏览器，你可能会喜欢它。

当然，所有访问 SitePoint 的 web 开发人员都安装了 Opera，并将它作为测试机制的一部分。或者他们有吗？
