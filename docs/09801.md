# 示例颜色 JSON 文件

> 原文:[https://www.sitepoint.com/colors-json-example/](https://www.sitepoint.com/colors-json-example/)

*这个[文章系列](https://www.sitepoint.com/10-example-json-files/)在 2017 年年中用最新的信息和新鲜的例子重写。*

在这个 JSON 示例中，我们将看看如何使用 JSON 格式在文件中存储简单的值。

使用键-值对符号，我们可以存储任何类型的值，包括字符串、数组和文字。当然，我们不能保存 blob 数据(例如视频、音频或压缩数据),因为 JSON 文件基本上是一个文本文件，我们可以使用任何文本编辑器进行编辑。

让我们快速看一下下面的例子:

```
{
  "colors": [
    {
      "color": "black",
      "category": "hue",
      "type": "primary",
      "code": {
        "rgba": [255,255,255,1],
        "hex": "#000"
      }
    },
    {
      "color": "white",
      "category": "value",
      "code": {
        "rgba": [0,0,0,1],
        "hex": "#FFF"
      }
    },
    {
      "color": "red",
      "category": "hue",
      "type": "primary",
      "code": {
        "rgba": [255,0,0,1],
        "hex": "#FF0"
      }
    },
    {
      "color": "blue",
      "category": "hue",
      "type": "primary",
      "code": {
        "rgba": [0,0,255,1],
        "hex": "#00F"
      }
    },
    {
      "color": "yellow",
      "category": "hue",
      "type": "primary",
      "code": {
        "rgba": [255,255,0,1],
        "hex": "#FF0"
      }
    },
    {
      "color": "green",
      "category": "hue",
      "type": "secondary",
      "code": {
        "rgba": [0,255,0,1],
        "hex": "#0F0"
      }
    },
  ]
} 
```

在上面的例子中，你可以看到我们可以提供多少关于特定颜色的数据。请注意所使用的结构和嵌套级别。你也可以使用一个基本的结构来存储你的数据。看看下面的例子:

```
{
  "aliceblue": "#f0f8ff",
  "antiquewhite": "#faebd7",
  "aqua": "#00ffff",
  "aquamarine": "#7fffd4",
  "azure": "#f0ffff",
  "beige": "#f5f5dc",
  "bisque": "#ffe4c4",
  "black": "#000000",
  "blanchedalmond": "#ffebcd",
  "blue": "#0000ff",
  "blueviolet": "#8a2be2",
  "brown": "#a52a2a",
} 
```

*样本取自[Bahamas 10/CSS-color-names](https://github.com/bahamas10/css-color-names/blob/master/css-color-names.json)T3】*
