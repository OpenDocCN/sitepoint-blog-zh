# Photoshop 中的颜色混合模式

> 原文：<https://www.sitepoint.com/color-blend-mode-in-photoshop/>

Photoshop 中的颜色混合模式采用基色的“灰度”并用混合色给它们着色:

> 颜色混合模式:使用基色的亮度以及混合色的色调和饱和度创建结果颜色。这将保留图像中的灰度级，对于单色图像着色和彩色图像着色非常有用。

例如，这里有一个一半是红色，一半是蓝色的图层:

![](img/f47164dc06c3b5e3bd5b3f9d5b7ed73d.png)

设置为颜色模式，并放置在另一个图像上，您会看到:

![](img/7ed5dbcf9fb95233af2ba958dc02cbe0.png)

颜色模式与色调模式非常接近，所以让我们再来看看色调模式，以消除任何混淆。色调模式保持基色的亮度和饱和度，但应用混合层的色调。颜色模式保持基色的亮度，并应用混合色的色调和饱和度。

下面是当蓝色/红色图层设置为色调模式时，场景的变化:

![](img/3edb631bc41d4843138c25417a5d17bf.png)

正如你所看到的，这张图片中有更多的“灰色”;在原始图像饱和度不高的区域，蓝色和红色会被去饱和。
