# 利用电子商务建立内容网站利润

> 原文:[https://www . site point . com/build-content-site-profits-with-ecommerce/](https://www.sitepoint.com/build-content-site-profits-with-ecommerce/)

人们通常认为电子商务是一个很大的步骤，一个人不能轻易进入，如果你正在制作你的第一个电子商务网站，这几乎是真的。然而，人们经常不考虑的一件事是简单地在他们现有的内容网站上销售一些产品，以此来提高内容网站的利润。

SitePoint 是近年来这方面的一个典范。SitePoint 以前广告比较多，现在大多是推广自己的书和软件。虽然他们没有与我分享收入细节，但我认为可以肯定的是，他们不会推广这些产品，除非他们从中获得的收入超过广告收入。

事情是这样的，电子商务网站通常从他们的流量中赚很多钱，对于一个内容网站来说，它需要吸引大约 100 美元的 CPM(每千次展示 100 美元)或更多。电子商务网站比内容网站的每位访问者的利润要高得多，因此你通常可以通过在内容网站中添加一些电子商务项目来大幅增加每位访问者的利润。

在我的一个[网站](http://www.wilderness-survival.net)上，我有一个相关产品的页脚广告，即救生包/刀。这是一个页脚广告，所以它没有占据我的主要广告空间，然而我设法在一个月内从刀的销售中获得几乎和广告销售一样多的收入。事实上，这对我来说非常成功，我正在建立一个全新的销售猎刀和野营刀的电子商务网站。

在另一个网站上，我只是出售我网站内容的光盘。我一个月能从这些 CD 上赚几百美元，我可以用我的 CD 刻录机来制造它们。相比之下，该网站的广告收入并不算多，但对于我菜单中的一个文本链接来说，这是一个不错的收入来源。

当然，正如我上面提到的刀，理想情况下，更好的是当你有一个内容网站和一个完整的电子商务网站，你可以交叉推广。一名 SPF 成员成功做到了这一点。他的内容网站是[Wannabebig.com](http://www.wannabebig.com)一个关于健身的网站。几年前，他发起了[自由营养](http://www.atlargenutrition.com)，他做交叉推广。当你能做到这一点的时候，有很多原因是很棒的。首先，你在发布时就有了固定的客户群，这对于一个电子商务网站来说是非常好的。此外，获得电子商务网站的链接比获得免费内容网站的链接要困难得多，所以通过交叉推广，你可以为你的新电子商务网站获得自己的链接和 PageRank。

当然，除了通过建立一个相关的电子商务网站或者将电子商务整合到你现有的网站来赚钱之外，你还可以获得稳定的收入。如果广告市场不景气，你仍然可以获得电子商务收入。如果更多的广告拦截器开始使用，你仍然有你的电子商务收入。电子商务收入比广告收入稳定得多。

所以开始考虑在你的内容网站上卖些什么，这是增加收入的好方法。你可以简单地使用 [Paypal](http://www.paypal.com) 或类似 [2Checkout](http://www.2Checkout.com) 的各种伪商家账户进行支付，你不需要一个完整的购物车来购买一些产品。对于运输，根据数量，每周去一次邮局或 UPS 也可以，您也可以处理您的货件，并在大多数主要承运商在线请求收件。这并不一定很难或者很费时间。
