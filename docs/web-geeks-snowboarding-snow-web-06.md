# 网络极客+滑雪板=雪网 06

> 原文：<https://www.sitepoint.com/web-geeks-snowboarding-snow-web-06/>

约翰·奥尔索普，今年[网络方向会议](https://www.sitepoint.com/web-directions-sydney-2006-win-an-xbox-360-from-sitepoint/)的组织者之一，和几个朋友一起，正在组织今年 7 月在雪山的滑雪/滑雪板之旅，名为[雪网 06](http://snoweb06.com) 。

> 从 7 月 28 日到 7 月 30 日，周五到周日，雪网将在澳大利亚的雪地里度过一个长周末。白天滑雪或滑板——当你的腿筋疲力尽的时候，在《来自白雪河的男人》见面，谈论 webapps、web2.0、Ajax、CSS 布局、商业模式、RSS、mental airs，任何你的 web 开发人员/设计师喜欢的东西。
