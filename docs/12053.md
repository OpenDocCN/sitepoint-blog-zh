# 如何让自己与众不同

> 原文：<https://www.sitepoint.com/how-to-differentiate-yourself/>

所有自由职业者和企业主都在争夺新业务。正在考虑你的潜在客户可能不知道你提供的服务类型，你对待客户的方式，以及与你的关系将如何发展。客户不了解你。大多数时候，客户在雇佣你的时候会有一个信念的飞跃，相信你的投资组合、经验和推荐信会证明你是这份工作的最佳人选。

当你与潜在客户处于初步探索阶段时，你的工作是回答问题，提供信息，并从本质上说服客户你是这份工作的合适人选(当然，同时确定他是否是你的合适客户)。这里有一些方法可以帮助你超越他人。

**要真诚**

人际关系是生意的一大部分。在面试中坦诚相待对建立新的伙伴关系大有帮助。你以前和一个新客户处于求爱阶段，所以你可能知道客户希望听到的大多数问题的答案。避免千篇一律的答案，通过提供对您和您的企业以及客户需求来说独一无二的信息，使您的回答个性化。

**了解潜在客户**

新客户正在调查你，看你的作品，甚至可能与你的其他客户交谈，以决定他们是否应该雇用你。但是不要忽略了你对他评价的重要性。他是你的理想客户吗？和他一起工作会支持你的商业目标吗？他是否表现出成为[问题客户](https://www.sitepoint.com/caution-six-warning-signs-of-a-bad-client/)的迹象？

**乐于提供信息**

潜在客户需要某些信息来衡量你的经历，以满足她的需求。视情况而定，因为这并不适用于所有情况，请考虑在您的提案中提供一整套初步信息。你可能想附上一些东西，包括文件夹、参考资料和案例研究。

**展示你的激情**

让你与众不同的最好方法之一就是对你所做的事情表现出真实而明确的热爱。你显然对你的工作有热情，否则你不会做它，但我们并不总是愿意在日常工作中表现出来。让你的热情表现出来，客户会欣赏你的兴奋。

**听(和听)**

与新客户合作的一部分是倾听他的需求，但可能更重要的是倾听他没有说出来的话。你需要分析从客户那里得到的信息，同时直觉地知道该问什么问题以获得尽可能完整的信息。这些信息中的大部分可以从你的标准[项目申请表](https://www.sitepoint.com/vital-information-for-a-web-design-project-request-form/)中获得，但是在每种情况下需要时，愿意调整和修改它。

**让你的名声说话**

坚持不懈地为客户做好工作，专注于提供优质的客户服务，并始终保持专业水准，这些对潜在客户来说都很重要。她会在对你的探索中揭开这些细节。建立一个稳固的正面声誉需要时间，但最终它会自己为你工作。
