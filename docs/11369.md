# 10 个用于增强客户端支持的远程桌面访问应用

> 原文:[https://www . site point . com/remote-desktop-access-client-support/](https://www.sitepoint.com/remote-desktop-access-client-support/)

本周，我写了几篇关于为客户提供虚拟培训和支持的文章，以及一些你可以使用的工具。

昨天，我分享了一份清单，上面有 30 种屏幕截图工具 ，它们可以让培训和支持成为您和您的客户更快、更高效的活动。今天，我将重点介绍 10 种允许您远程访问客户计算机的应用程序。

[**任意地点控制**](http://www.anyplace-control.com/)

Anyplace Control 通过局域网、广域网或互联网连接提供安全的远程访问。功能包括文件传输、远程电源管理、双重密码保护和远程访问警报。根据所需的连接数，有三种许可选项。
![anyplace control](../Images/bd53590ccd867a74a284cfaf74276c5d.png)

[**苹果远程桌面 3**T3】](http://www.apple.com/remotedesktop/)

Apple Remote Desktop for Mac OS X 提供对其他 Mac 电脑的远程访问。功能包括跨多个系统的搜索、40 多个 Automator 操作、一个 Dashboard widget 以及用户历史和报告。对于远程支持功能，您可以使用远程拖放复制功能在本地和远程系统之间进行复制和粘贴。该软件包括对 10 个受管系统的访问。
![apple remote desktop](../Images/37cb40e5e9b3f6027d6e83597e9794a0.png)

[](http://beanywhere.com/)

 **BeAnywhere 提供了查看和控制远程计算机以及文件传输功能，而无需在客户端计算机上预先安装软件。BeAnywhere 按支持技术人员定价。
![be anywhere](../Images/d47ee3d442abea690ed43d6679af33f9.png)

[**GoToMyPC**T3】](http://www.gotomypc.com/)

GoToMyPC 允许您从任何网络浏览器实时远程访问 Mac 或 PC。功能包括自动设置、128 位 AES 加密、远程打印、文件同步和传输以及声音。有几个月计划可用。
![gotomypcl](../Images/6a5c89df6e6b6913dc2eb9cda14e7403.png)

[**LogMeIn**](http://logmein.com)

LogMeIn 允许您从任何联网的 web 浏览器连接到远程计算机。功能包括文件传输、文件共享、远程到本地打印、远程到本地音频、诊断工具包、警报和监控，以及高级报告和分析。基于计算机连接数量的帐户费用。
![log me in](../Images/40f3c995284ea737c499ca520ebb5eae.png)

[**RealVNC**T3】](http://www.realvnc.com/)

RealVNC 提供远程控制软件，其功能包括 Mac OS X 访问(企业版)、服务器认证和 128 位 AES 会话加密、打印功能、文件传输、集成地址簿和聊天。RealVNC 提供免费和付费许可选项。
![realvnc](../Images/1af137d954f98f7217717d02dc4a73c9.png)

[**ShowMyPC**T3】](http://showmypc.com/)

ShowMyPC 提供桌面共享和远程 PC 访问。功能包括与您的网站集成、聊天和白板、桌面和音频记录、文件和应用程序共享，以及集成的 Mac 和 Linux 支持。有几种订阅计划可供选择。
![showmypc](../Images/fa518b21ac1bfc07838a785952344e2d.png)

[**SkyFex**](http://skyfex.com/)

SkyFex 是一个远程支持解决方案，允许桌面共享和远程桌面控制。特性包括 128 位加密、无需安装软件以及防火墙/代理透明。提供免费和付费计划。
![skyfex](../Images/29ad93eb0825f1857a748b3e322cae12.png)

[**TeamViewer**T3】](http://www.teamviewer.com/)

TeamViewer 允许您通过 AES (256 位)编码的会话与 Internet 上的任何计算机建立即时连接，并包括集成文件传输、基于浏览器的访问和优化显示质量等功能。这项服务对非商业用户是免费的，并且有几个商业账户的许可选项。
![teamviewer](../Images/364254466b4c76d1eee35827f69b173c.png)

[**UltraVNC**T3】](http://www.uvnc.com/)

UltraVNC 是一种从远程位置访问和控制另一台 Windows 计算机的软件。功能包括数据加密、聊天功能和文件传输。该软件在 GNU 通用公共许可证下是免费的。
![ultra vnc](../Images/74b3d13d1694fbd0de77cee706d2ac8b.png)**
