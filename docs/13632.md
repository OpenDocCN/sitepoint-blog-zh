# 销售漏斗

> 原文:[https://www.sitepoint.com/the-sales-funnel/](https://www.sitepoint.com/the-sales-funnel/)

今天的帖子涵盖了销售中的一个基本原则，但这个原则总是值得重温。

“销售漏斗”是传达整体营销和销售过程的一个很好的形象。想象一个漏斗，顶部宽，底部窄。

你的总体目标市场代表了漏斗的顶端。

然后，随着你有了嫌疑人、潜在客户(在雇佣你的不同阶段)、客户和回头客，漏斗逐渐变窄。

一个有效的销售漏斗的诀窍是在所有阶段尽可能地扩大它。

例如，由于我的各种网站，我收到了各种各样的询问。然而，鉴于互联网的非个人性质，并非所有这些调查都是认真的。例如，就在上周，一家财富 500 强化妆品公司的培训总监联系我，为他们的一些顶级明星提供指导。我以前与这家公司没有任何关系，不太了解化妆品销售(好吧，我对化妆品销售一无所知)，我是该公司正在考虑的几十个人中的一个。所以我被录用的几率相对较低。因此，我需要有很多这样的查询来我的方式来转换一些销售。(与此同时，有了一个集中的目标市场，我会收到更多在我的最佳位置的询问，这些询问很有可能转化为销售。)

对我来说，销售漏斗的主要教训是，你要在目标市场上尽可能引人注目，这样尽可能多的潜在客户才能进入你的漏斗。

这样，“销售”就变成了评估适合度的对话，而不是某种推动我收入的神奇的有说服力的对话。人们来找我是因为针对特定目标市场的有效营销，包括推荐、演讲、写作、有效的网络展示、宣传，以及成为你所在市场的专家。

所以，看看你的销售漏斗。如果你没有获得足够的销售额，为什么？这可能是因为你没有将足够多的潜在客户转化为客户，但我的猜测是，你没有通过有效的营销吸引足够多的人进入你的漏斗。
