##### “其他”选项-不需要商家帐户

如果您手头现金不多，或者销售的产品数量有限，您可以尝试其他加工公司(称为第三方加工公司)。你只需支付每件产品售价的 10-20%,而不是支付安装费、交易费、声明费、折扣费等等。我们不建议长期使用这些公司。如果你刚刚起步，并且你不需要马上从销售中获得现金(例如，CCNow 扣留销售收入的一部分长达三个月，以防止退款)，它们是非常好的，但从长远来看，它们最终会让你付出更多的代价。

一旦你有了足够的资金来为你的公司建立一个真正的商业账户，就去申请一个吧。如果你是一个非美国的企业，最好的方法是使用第三方处理器，至少在真正的商家账户费用下降之前。以下许多公司同时支持美国和非美国业务:

##### 退款和欺诈

*退款*——互联网商家害怕的词。拒付是指交易被撤销。换句话说，不是把钱加到你的账户上，而是被扣除。拒付发生的原因多种多样，例如重复收费、信用卡过期、银行错误和客户纠纷。如果你的商家帐户招致太多退款，你有可能会失去你的帐户。一旦发生这种情况，你将被列入 Visa/MasterCard 匹配名单好几年。所有的商家帐户提供商都可以访问这个列表，如果他们在列表中找到你的名字，他们通常会拒绝向你发放一个商家帐户。但是，如果你是那些失去客户的商人之一，还是有希望的。[银行卡法](http://www.bankcardlaw.com)专门帮助因过度拒付而失去商户账户的公司。

以下是一些有助于消除退款的技巧:

1.  在您的订单确认页面上，记录客户的姓名(电话号码、电子邮件地址等)。)会出现在他们每月的信用卡账单上。
2.  在您的客户下订单时收到的电子邮件确认函中，也要注明这些细节(他们确实收到了，不是吗？如果没有，他们应该…有利于记录)。
3.  使用地址验证。用偷来的卡号订购产品的人永远不会使用真正持卡人的帐单地址，所以这是你阻止订单的机会，否则就太晚了！
4.  仔细审查来自外国的订单。很大一部分欺诈性互联网购物来自印度尼西亚、俄罗斯和其他东方集团和发展中国家。
5.  如果订单可疑，给客户打电话或发电子邮件，尽量核实他们的任何信息。注意陌生订单然后跟进。
6.  最后，如果您运送产品，请在客户发票上附上备注。

永远保留交易的副本，这样，如果你被质疑，你就有了某种防御手段。

不幸的是，网上信用卡诈骗在互联网上非常普遍。作为商人，你能做的降低欺诈成本的最好的事情就是被告知。我强烈建议您花几个小时浏览以下网站，并在您的电子商务中实施尽可能多的建议策略:

*   [反欺诈](http://www.antifraud.com)，包括三千多个用于免费电子邮件的域名列表，这些域名可以隐藏真实客户的身份。
*   [互联网欺诈观察](http://www.fraud.org/ifw.htm)

几乎在所有情况下，当欺诈性信用卡在你的网站上被使用时，你将被追究责任，并且你将被收取销售费用。唉，除非涉及金额巨大，否则当局不太可能介入欺诈案件，尽管你应该毫不犹豫地报告发生的每一起欺诈案件。

##### 安全 SSL 证书

当然你需要一个公司来托管你的网站，但是如果你决定处理信用卡订单，你需要一个可以提供安全服务器的主机。在许多情况下，托管公司会免费为你提供一个安全的服务器，但在其他情况下，除了正常的安全服务托管费用之外，你可能还需要每月支付 10 美元。

如果你想使用自己的域名作为安全服务器，而不是你的托管公司的域名和证书，你需要从 [Thawte](http://www.thawte.com/getinfo/products/secure.html) 或 [Verisign](http://www.verisign.com/products/site/secure/index.html) 购买自己的证书，一年的费用在 125 美元到 350 美元之间。 [EquiFax](http://www.equifaxsecure.com/digitalcertificates/dc_webservcert.html) 是另一个安全证书发行商——他们可以为你的网站发行证书，只需 99 美元。

那么，安全证书是如何工作的呢？安全服务器是一个 Web 服务器应用程序，它确保 Web 服务器和 Web 浏览器之间的安全信息传输。加密是通过一种称为 SSL(安全套接字层)的协议完成的。当您使用浏览器连接到安全服务器时，服务器和浏览器使用安全套接字层相互告知如何加密信息，然后它们可以安全地交换数据。例如，加密将允许您将信用卡信息发送到安全的服务器，而没有人能够识别它是哪种信息。

因此，安全服务器完成的第一项任务是加密您的信息。然而，它还做了更多的事情。安全服务器执行的最重要的任务是向浏览器标识自己，这是通过向请求安全连接的浏览器提供数字证书来实现的。该证书在安全服务器上的工作方式与驾照在你身上的工作方式相同，只是它更难伪造。

对于浏览网站的人来说，上述所有事情都是绝对透明的。他们的浏览器只是告诉他们安全连接已经建立，然后显示页面。从网上商家的角度来看，最重要的事情是，这给了你的客户更多的信心，并使他们相信他们的订单信息将安全可靠地传送给你。

##### 购物车软件

一旦安全的服务器启动并运行，您将需要一个购物车，它将显示订单、存储订单并计算成本。除了确保您的购物车将与您的实时网关一起工作(如果您决定实时处理信用卡订单)，您还应该寻找其他几个主要功能:

*   易于安装和快速执行
*   可定制性，使购物车生成的页面看起来就像你的网站
*   能够计算运输成本和销售税
*   拥有您在可预见的未来需要的所有功能。否则在你意识到之前你就需要换平台了！
*   PGP(相当好的隐私)支持，如果你决定通过你自己的软件进行延期处理
