# 进一步证明了目标市场是正确的选择

> 原文:[https://www . site point . com/further-proof-a-target-market-is-the-way-to-go/](https://www.sitepoint.com/further-proof-that-a-target-market-is-the-way-to-go/)

这是令人欣慰的一天，因为两位购买了我的书的人联系我，确认关于选择目标市场的建议是有效的。

一个人得到了这个教训。他花了一年时间与一位企业主建立业务关系，试图向他出售开发服务。他刚刚发现，尽管老板完全了解这个人的能力，但他还是聘请了另一家公司进行 web 开发，这家公司为他的特定行业提供了完整的解决方案。

第二家来自澳大利亚，正竭尽全力主导一个利基市场。(顺便说一句，我爱澳大利亚的企业家。在哈佛商学院，澳大利亚学生成绩最好，最能喝酒，也最有幽默感。他们已经加入了目标行业的主要协会，并且正在想办法如何在协会中引人注目。他们计划写文章、演讲、发布新闻稿、做研究、定位他们的网站，以及其他一切与垂直战略相关的事情。今天，行业协会的一名董事会成员表示有兴趣支持该公司，因为他们是唯一一家专门从事该特定行业的 IT 公司！

你还在骑墙观望，希望和幻想无限的前景等于大量的客户吗？事实上，一个狭窄的市场和有限的潜在客户意味着大量的客户——而且接触客户的成本要低得多。

但这是你的事…
