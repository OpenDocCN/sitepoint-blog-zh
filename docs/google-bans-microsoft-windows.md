# 谷歌封杀微软 Windows 了吗？

> 原文：<https://www.sitepoint.com/google-bans-microsoft-windows/>

根据主流媒体的一些报道，由于安全问题，谷歌正在逐步淘汰微软在公司个人电脑上的 Windows 操作系统。报告称，这是为了弥补今年早些时候被中国黑客利用的漏洞。

《金融时报》的一篇文章引用了未透露姓名的谷歌员工的话。消息人士透露，谷歌的 10，000 名员工将被鼓励选择 Linux 或 OS X。那些想在 PC 上保留 Windows 的人需要获得高层的批准。

谷歌和微软都没有对这则新闻发表官方评论。

对我来说，整个故事听起来很可疑。有几个原因:

**1。选择的自由**
众所周知，谷歌员工拥有 IT 界最好的福利。他们以 20%的项目而闻名，在这些项目中，工人可以要求每周一天来调查或开发他们选择的技术。那么谷歌为什么会突然限制员工自由呢？许多人已经使用 Windows 多年，当用户迁移到其他平台时，公司的生产力会下降。

**2。安全调查**
如果微软的技术漏洞是安全问题的原因，那么将该公司从 Windows 和 Internet Explorer 中移除与谷歌用户运行的软件无关。

如果员工无法使用导致问题的软件，如何调查漏洞？

**3。浏览器测试**
不管你喜不喜欢，Internet Explorer 是世界上使用最广泛的浏览器。使用可能正在减少，但如果谷歌不能在顶级主流浏览器中进行测试，他们如何构建基于网络的解决方案？

在 Linux 上运行 IE 是可能的，但它不是 Windows 版本的精确复制品。如果你想测试 IE，你需要 Windows。

**4。抄袭竞争**
没有一个软件产品是完全孤立开发的。软件供应商模仿或公然抄袭竞争系统的想法和功能。谷歌也不例外，他们的许多产品都受到了来自微软或其他公司的软件的影响。

如果他们拒绝安装他们的软件，他们怎么能跟上竞争呢？

**5。便捷的操作系统宣传**
尽管两家公司都没有发表评论，但这个故事已经有了自己的生命。它是否由谷歌发起并不重要:他们肯定会受益。

谷歌将在几个月内发布 Chrome 操作系统。如果 Windows 安全性在事件发生前受到质疑，这只是一个巧合。

就我个人而言，我一个字都不信。然而，它显示了大型组织是如何操纵媒体为自己谋利的。
