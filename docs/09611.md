# 构建移动集合

> 原文:[https://www.sitepoint.com/the-build-mobile-muster-20110701/](https://www.sitepoint.com/the-build-mobile-muster-20110701/)

惠普的 TouchPad 将于周五上市，也就是今天。它看起来很酷，Jason Snell 在 Macworld 上有一个乐观的评论，有很多值得推荐的地方。WebOS 为 Web 标准开发者打开了大门，这可能会加深应用程序的财富。惠普开发了“HPplay ”,这是一款 Mac 应用程序，可以解决将音乐和电影下载到设备上的问题。嗯，对于 Mac 用户来说。

在最近的媒体报道中，控制软件和硬件正成为交付高质量小屏幕设备的最佳实践。Austin Carr 在 Fast Company 讨论了这一点，并对惠普与微软和潜在的 Windows 8 的关系以及与 Android 明显缺乏联系做了一些说明。

理查德·蔡秋凤是惠普全球开发者关系副总裁。吉姆·达尔林普尔在 The Loop 采访理查德·蔡秋凤，故事的导语是苹果不是 TouchPad 的目标。此外，惠普的战略是一项长期战略，可能会侧重于企业领域。另一个新出现的主题是，赢家可以不止一个。

Jonathan S Gellar 在 BGR 发表了一封致 RIM 高级管理层的公开信。前段时间[迈克尔·梅斯](http://mobileopportunity.blogspot.com/2010/10/whats-really-wrong-with-blackberry-and.html)和最近[霍勒斯·德迪乌](http://www.asymco.com/?s=rim&submit=Go)提供了大量的分析和弹药。RIM 给了一个守口如瓶的回应，然后同意考虑至少一个主要的批评，拆分董事长职位。

尼尔森网讯(Nielsen Wire)从市场份额之争的输赢评论中稍作休息，退后一步，简单地指出智能手机正以令人难以置信的速度增长。基本上在上个季度，超过一半的购买是智能手机而不是功能手机，比上一年增长了 34%。

这篇文章确实提到了安卓变平，iPhone 再次增长(可能是由于威瑞森), RIM 正在萎缩。然而， [Strategy Analytics 的有趣研究得出结论，黑莓仍以 72%、iPhone 49%和 Android 42%的比例主导美国公司的手机选择。](http://www.mobiletechnews.com/info/2011/06/28/120731.html)

Skype 为 Android 手机带来了视频通话功能但在推出的第一阶段，只有四款手机受到支持，分别是 HTC Desire S、索尼爱立信 Xperia neo、索尼爱立信 Xperia pro 和谷歌 Nexus S。

我的喷气背包在哪里？

令人兴奋的消息是，SitePoint 的一本新书上市了。[为智能设备构建移动网站和应用](https://www.sitepoint.com/books/mobile1/)由 Earle Castledine、Myles Eftos 和 Max Wheeler 撰写，现已上市。将它添加到你的书架上 [HTML5 & CSS3](https://www.sitepoint.com/books/htmlcss1/) 旁边，你就万事俱备了。未来几周，我们将继续在 Build Mobile 上发布《为智能设备构建移动网站和应用》第四章的剩余内容。
