# Opera 10 的新特性(第 1 部分)

> 原文:[https://www.sitepoint.com/opera-10-new-features/](https://www.sitepoint.com/opera-10-new-features/)

对于浏览器厂商来说，这是忙碌的一年。微软发布了 IE8，谷歌 Chrome 在 2 版，火狐 3.5 和 Safari 4 都迫在眉睫。然而，最新的新浏览器是 Opera 10，它现在是测试版下载。

Opera 以领先的创新和网络标准而闻名，同时保持轻量级、快速和功能丰富。安装程序的大小只是大多数浏览器的一小部分，但浏览器包括标签、快速拨号、电子邮件客户端、RSS 阅读器、新闻组支持、IRC、bit torrent 客户端、笔记、鼠标手势、桌面小部件、开发人员工具、小屏幕渲染等等。然而，尽管它在北欧和俄罗斯很受欢迎，但在世界大部分地区却落后于 IE、Firefox、Safari 和 Chrome。

和许多开发者一样，我非常尊重 Opera。我用它来测试，但它从未取代 Firefox 成为我的默认浏览器。Opera 10 能否说服我转行，抢占更大的 PC 浏览器市场份额？

## 新界面

乍一看，Opera 10 与 Safari 有很多相似之处。界面更干净、更时尚、更整洁。所有的功能和界面选项仍然可用，但它们不像以前的版本那样令人生畏。

## 新功能

Opera 可能有几个新特性，尽管很多已经在 Firefox 和其他一些浏览器中使用了一段时间:

*   选项卡预览
*   快速拨号定制
*   自动更新
*   内嵌拼写检查
*   webmail 集成
*   崩溃恢复和报告
*   可调整大小的搜索字段(很多人真的关心这个吗？)

选项卡栏也可以扩展为显示带有页面缩略图的“可视选项卡”。虽然它占用了相当多的屏幕空间，但对于 tab 新手或拥有较大显示器的人来说，这是一个很好的功能。

![Opera visual tabs](../Images/819207f961858d16326e5db9e4ea5732.png "Opera visual tabs")

电子邮件客户端现在支持 HTML 和富文本消息，尽管这导致了一些在 [SitePoint 论坛](https://www.sitepoint.com/forums/showthread?t=619715)上的抱怨。我也不太喜欢图形邮件，但我们似乎是少数。HTML 支持是一个合理的商业决策，应该会带来更多的 Opera 电子邮件用户。我当然会推荐它给[上网本](https://www.sitepoint.com/6-reasons-to-buy-a-netbook/)或者内存有限的老设备。

## 标准和开发者支持

Opera 的渲染引擎 Presto 2.2 对标准的支持一如既往的强大。与大多数竞争对手不同，Opera 以 100/100 的分数通过了测试。

其他新功能包括:

*   支持 CSS @font-face 声明的 web 字体。
*   CSS 颜色可以使用 **HSL(色调、饱和度和亮度)**以及 RGB 来定义。一些 JavaScript 效果更容易编码，例如增加元素的亮度。
*   支持 RGB 和 HSL 声明中的**不透明度**，例如`background-color: rgba(255, 0, 0, 0.5)`将创建 50%透明的红色背景。然而，与单个`opacity`属性不同，它只会改变元素背景的透明度，而不会改变内容或子元素。
*   **类似 CSS 的 DOM 元素选择**带`document.querySelectorAll()`和`document.querySelector().`
*   **改进了 SVG 支持**，为动画设置了网页字体和每秒帧数。
*   初级 **HTML5 支持**。

Opera 的开发和调试控制台 DragonFly 已经更新，包含了更多受 Firebug 启发的特性，如 DOM 编辑和 HTTP 分析。
