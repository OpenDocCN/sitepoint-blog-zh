# 用 Photoshop 文章设计一个网站

> 原文:[https://www.sitepoint.com/design-website-photoshop-2/](https://www.sitepoint.com/design-website-photoshop-2/)

接下来，将您的图像导入 ImageReady，并从菜单中选择“切片”选项。在下拉菜单中，单击“从参考线创建切片”。你的图像应该是这样的:

![881_screen2](../Images/483355d76af69d5d0d75c1561bebd2a3.png)

请注意，所有切片都是根据指导原则创建的，并且都进行了编号。这些数字只是 ImageReady 命名切片图像的默认方法，我们稍后会更改它们。

好了，现在你已经创建了你的切片，在“窗口”对话框中选择“显示切片”。它将显示如下内容:

![881_slices_interface](../Images/6249f98495b4ed7cbf0675a2e30760f5.png)

接下来，我们将组合一些切片并命名。所以选择“切片选择工具”。两条垂直方向的指引线将页眉分割成了三个图像，我们需要将这些片段组合起来，将页眉分割成一个图像。

一旦你选择了切片选择工具，点击左上角的切片 01。然后，按住 shift 键，单击切片 02 和 03。接下来，右键单击中心切片，并从出现的对话框中选择“合并切片”。Mac 用户应该选择切片菜单项，并从对话框中选择“合并切片”。

现在你的图像应该是这样的:

![881_screen4](../Images/d3a2c5b2faf914d244ccf0566282eb9f.png)

这三个切片现在变成了一个切片，用黄色边框突出显示。要命名切片，请转到“显示切片”对话框。在名称字段中键入“页眉”，并对页脚区域重复此过程，将其命名为“页脚”。

但是等等！我们还有一些切片需要命名。就在左侧标题下方，您应该会看到现在的切片 02——一个深蓝色的小方块。因为该切片与切片 04、08 和 10 具有相同的维度，所以我们只需要保存其中一个。所以用切片选择工具点击切片 02，命名为“blue_square”。接下来，点击切片 05，命名为“gray_bar”。同样，您会注意到它与切片 07 相同，因此您只需要保存其中一个。
