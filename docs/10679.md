# 谷歌在 GMail 中加入免费电话语音通话

> 原文:[https://www . site point . com/Google-Gmail-free-telephone-voice-calls/](https://www.sitepoint.com/google-gmail-free-telephone-voice-calls/)

你有一个 [GMail](https://www.gmail.com/) 账户吗(谁没有？)从今天起，你可能会幸运地注意到你的聊天框中有一个新的*【呼叫电话】*选项…

![Google Voice](../Images/82f98d7203063b145e3053e12923d9e2.png)

与现有的 GMail 语音和视频聊天系统不同，Call Phone 允许您拨打标准的座机电话——接收方不需要登录，甚至不需要拥有 PC。此外，拨打美国和加拿大的电话完全免费，[国际通话费率极具竞争力](https://www.google.com/voice/rates)(英国、法国、德国、中国和日本每分钟不到 0.02 美元)。

该电话系统使用与语音和视频聊天相同的浏览器插件，可用于 Windows XP+，Mac OS X 10.4+和 Linux。

谷歌宣布，他们将在未来几天内向美国的 GMail 用户推出这项功能。国际用户和那些在学校或公司使用谷歌应用程序的用户还不会收到它。不过，我要告诉你一个小秘密——我在英国，这个功能已经出现在我的账户里了。这可能是因为我将“英语(美国)”设置为默认语言，虽然我要为英国本地号码付费，但我打给美国的电话是免费的！我怀疑这是一个技术漏洞，谷歌将填补，但我很想知道是否有任何其他非美国居民收到了该功能。

通话电话看起来不错，而且，假设通话质量和低成本可以保持， [Skype](https://www.skype.com/) 有理由担心。

你试过用你的 GMail 账户打电话吗？你的体验如何？
