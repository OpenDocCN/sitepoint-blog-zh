# 如何战胜商业无聊

> 原文:[https://www.sitepoint.com/how-to-battle-business-boredom/](https://www.sitepoint.com/how-to-battle-business-boredom/)

我们有时都会感到无聊，需要重新点燃激情，让我们努力工作，推动我们达到目标，并帮助我们享受我们正在做的事情。因此，为了纪念 7 月成为反无聊月(不，我没有瞎编)，我看了一下无聊，什么导致了无聊，以及如何在日常生活中避免无聊。

什么是无聊？

根据维基百科，无聊是指在缺乏活动或个人对呈现给他们的活动不感兴趣的时期经历的一种情绪状态。如果你在生活中曾经无聊过，你就知道，无聊并不总是意味着无事可做；这意味着你没有什么想做的事。

无聊的常见原因

无聊可能由许多不同的因素引起。无聊的一些常见原因包括:

*   缺乏参与
*   不了解你在做什么/学什么
*   对自己正在做的事情/正在学的东西知道得太多
*   缺乏变化和刺激
*   无法满足你的好奇心
*   感到不知所措和过度刺激
*   一贯的例行任务
*   缺乏发挥创造力的机会

**如何避免无聊**

避免无聊的主要方法是确保你把全部注意力放在你正在做的事情上。通过完全投入和投入，你可以避免陷入无聊的陷阱。当然，这并不总是可能的，所以我们倾向于发展一些小技巧来赶走无聊，帮助我们保持高效。

当你感到无聊时，这里有一些快速简单的方法可以让你重新开始:

*   到外面去工作
*   向客户提供新的服务
*   每天给一个客户打电话签到
*   参加一个学习新技能的课程
*   参加会议或社交活动
*   开始阅读行业出版物
*   向同事伸出援手
*   花时间检查你的工作
*   重温你的目标，创造新的目标
*   考虑[多任务](https://www.sitepoint.com/multitasking-kills-focus-productivity/)
*   请一天假，远离工作
*   改变你正常的工作时间表
*   只做你喜欢的事
*   接受一个项目，把你拓展到一个新的领域
*   志愿者
*   在论坛或讨论板上发表意见
*   听你最喜欢的音乐
