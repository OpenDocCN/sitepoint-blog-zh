# Mozilla 雷鸟 3.0 发布

> 原文：<https://www.sitepoint.com/mozilla-thunderbird-3-released/>

虽然许多人避开电子邮件应用程序来使用网络邮件解决方案，但我们中仍有一些人喜欢配置、安全性和性能都不错的电子邮件客户端。微软凭借 Outlook、Outlook Express 和 Windows Mail 统治了 Windows 世界，但如果你对这些产品不感兴趣，Mozilla 的雷鸟可能适合你。

当 Mozilla 套件被 Firefox 取代时，雷鸟的工作就开始了。虽然它从未达到他们的浏览器的受欢迎程度，但雷鸟是一个值得更多关注的伟大的电子邮件应用程序。

雷鸟 3.0 终于发布了。该客户端有 49 种语言版本，并提供以下新功能:

*   简化的用户界面
*   标签邮件——是的，标签已经从火狐迁移到雷鸟
*   更简单的帐户设置
*   更快更好的搜索工具
*   邮件存档
*   对收件箱视图进行分组的智能文件夹
*   IMAP 文件夹同步，这样您可以在脱机时访问您的邮件

也许最有用的新功能是附件提醒。有多少次你发送或收到的电子邮件承诺的文件从来没有被附加？雷鸟提醒会扫描您的电子邮件文本，查找诸如“附件”之类的词语，然后在邮件发送之前提示您。

雷鸟是免费的，可用于 Windows，Mac OS X 和 Linux。从[](http://www.getthunderbird.com/)**下载。**
