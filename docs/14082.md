# GoogleGuy 转储，Google 泄密

> 原文：<https://www.sitepoint.com/googleguy-dumps-google-leaks/>

被称为“波旁更新”的当前/最近的谷歌更新引起了一点轰动，因为更新的长度和一些搜索结果中发生的重大变化。谷歌的网站管理员代表“GoogleGuy”在[网站管理员世界](http://www.webmasterworld.com/forum30/29720.htm)上发表了一些冗长的评论，包括有用的说明，即在 URL 中使用“id”作为变量可以阻止 Googlebot 爬取网站。显然 GG 最近有很多心事，决定一次全倒出来。

把这个归档在有趣但不是你真正能做的事情下面:[Henk van Ess 的 Search Bistro 向世界展示了谷歌为他们的质量控制团队提供的“秘密”网络界面](https://www.sitepoint.com/forums/showthread.php?t=268610)。[谷歌证实这是真的。](http://www.searchbistro.com/exit.php?url_id=651&entry_id=30)

有趣并且可能值得一些人采取行动:在有限的时间内，你可以下载微软 Word 格式的谷歌垃圾邮件识别指南:[http://www.searchbistro.com/spamguide.doc](http://www.searchbistro.com/spamguide.doc)。在 Search Bistro 博客上已经可以找到大量的评论，实际上并没有什么惊喜，但是读起来很有趣。范·埃斯先生还拿到了另一份文件，一份关于评级搜索的[指南。](http://www.searchbistro.com/guide.pdf)

当我们爬遍 Henk 的博客时，他报告说 META 标签仍然被用来生成显示在 SERPs 上的描述。这算不上什么新闻，但在他的博客上看到一个帖子说**不需要用斜线隔开，这很有趣。我肯定这一周对他来说很不容易…**
