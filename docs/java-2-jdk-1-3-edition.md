# 开始 Java 2-JDK 1.3 版

> 原文：<https://www.sitepoint.com/java-2-jdk-1-3-edition/>

不要与大多数 Web 浏览器支持的轻量级脚本语言 JavaScript 相混淆，Java 是一种成熟的编程语言，它起源于 Web 设计领域，正在缓慢但肯定地扩展到一般编程领域，即使它在 Web 上扩展了它的角色。Java 作为一种整洁、组织良好且通常易于使用的语言而享有盛誉；然而，作为一种成熟的语言，它的规模和功能每天都在增长，从头开始学习 Java 可能是一项令人生畏的任务，就像 WROX Press 出版的 Ivor Horton 的书《Java 2 JDK 1.3 版入门》一样令人生畏。

近 1200 页，这部作品的规模掩盖了其诱人的友好的标题；事实上，这个标题可能会吓跑潜在的读者，他们在阅读这本巨著时可能会问自己，“我必须读完所有这些才能开始学习 Java 吗？”好吧，与其直接回答这个问题，不如让我描述一下这本书涵盖的所有内容，然后你就可以自己决定这本书是仅仅让你起步，还是会带你一路到达你的目标。

这本书的前三分之一左右通常被称为语言之旅。从该语言所基于的基本概念开始(对于 Java 来说，这些是目前所有大学计算机科学课程中教授的经典面向对象编程(OOP)原则)，讨论然后进行到对该语言的全面但令人惊讶地不过分冗长的描述，从基本的程序机制开始，引导读者通过十章和近五百页的课程，了解诸如数据类型、对象和类、继承、异常处理、I/O 和线程等主题。如果这些术语中有许多是你不熟悉的，也不要担心——这本书很好地从实用的角度介绍了每一个概念，并辅以有用的例子，以免陷入理论的泥沼。

这是许多其他“Java 初学者”书籍结束的地方，并建议您进行更高级的阅读，令人惊喜的是，这本书的大部分仍然存在，因为这是 Java 真正开始变得有趣的时候！

这本书从介绍图形用户界面(窗口、按钮、菜单和图形)开始，试图教会你编写完整的图形化 Java 应用程序所需的一切，而不仅仅是基本的命令行程序。我对这本书掩盖某些棘手问题的方式有点失望；然而，作者确实引导读者在很大程度上避免了这些问题，即使在这个过程中留下了一些挥之不去的问号。

然而，在特定问题上的任何轻微的深度不足都可以通过对 Java 用户界面方法的令人惊讶的完整讨论来弥补。我一次又一次地翻动书页，希望作者跳过一些东西，结果却发现了丰富的信息，这些信息我只会从一本专门介绍 Java 用户界面的书中得到。第 14 章和第 15 章分别介绍了 Java 2 中的新 Java2D 特性和实现高级用户界面特性，这两章无疑是本书节奏最快、最有趣的部分，因为它们真正展示了 Java 语言的强大功能和便利性。

这本书有关于将数据保存到文件和为 Java 应用程序添加打印支持(这两个主题在 Java 书籍中很少涉及)、图像、动画和声音的高级章节，再次超越了我对“初学者指南”的期望。最后，最后两章提供了 Java 数据库交互特性的快速浏览。我必须承认，这些似乎有些不合时宜，如果您打算用 Java 进行任何严肃的数据库开发，我肯定可以想到关于这个主题的更好的书籍，但是如果您认为这些章节是一个额外的奖励，可以让您体验一下 Java 还能做什么(它能做很多)，那么真的没有任何理由抱怨。

这个“Java 2 入门”的更新版本是一本以最好的教程参考传统编写的书，它以一种突出这种语言的巨大优势的方式，甚至引导完全的新手轻松地进入 Java 的一些最强大和先进的特性，同时为需要比在线参考更具可读性的有经验的 Java 开发人员提供了一个完美的桌面参考。虽然没有哪本书能够涵盖 Java 的所有功能，但如果某个高级主题与手头的任务有密切关系，这本书从来不会回避这个主题。
