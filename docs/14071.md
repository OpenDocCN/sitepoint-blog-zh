# OpenSolaris 面向企业开放

> 原文:[https://www.sitepoint.com/opensolaris-open-for-business/](https://www.sitepoint.com/opensolaris-open-for-business/)

虽然这已经酝酿并公开了一段时间，但一个坚实的源代码基础和 Sun 公司的免费操作系统的第一次发布今天打开了大门。

该公司一直表示，这是其业务建设战略的一部分，希望看到硬件和服务销售的增长(翻译过来就是 IBM 商业模式，这不是一个不成功的模式)。也许更不言而喻的是，它希望与已经建立的 Linux 发行商如 RedHat 竞争。

Sun 选择了一个相当开放的方法，从一组来自世界各地的独立开发人员开始，他们在过去一年的大部分时间里与 Sun 一起默默工作，直到发布这个版本。该公司希望开源开发者和 Solaris 爱好者将立即开始通过测试、错误报告和其他质量保证贡献。监督 OpenSolaris 的大多数董事会成员也不是 Sun 公司的，包括 Apache 的原始作者(Roy Fielding)。

我看过 Solaris 在金融服务领域的运作，许多人都知道它有着强大而权威的声誉。我计划构建新的操作系统，并开始探索它将如何匹配我目前使用的 Linux 发行版。更不用说，我需要清理一些 Solaris 蜘蛛网，并回到一些容易忘记的细微差异的速度上(有人必须“探测”IP 地址吗？)

对于门外汉来说，当然所有 web 专业人员最喜欢的工具都运行在 Solaris 上——Apache、Tomcat、MySQL、Perl 等等。
