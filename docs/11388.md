# 如何使用三分法裁剪图像

> 原文:[https://www . site point . com/how-to-crop-a-image-using-the-rule of-third/](https://www.sitepoint.com/how-to-crop-an-image-using-the-rule-of-thirds/)

“三分法则”是一个指导方针，当你是摄影师时，它可以帮助你创作出有趣的照片，或者当你在设计中使用别人的照片时，它可以帮助你创作出更引人注目的作品。这条规则是在摄影、艺术和设计领域创作吸引人的作品的最基本的规则之一。

三分法背后的主要概念是想象两条水平线和两条垂直线将一幅图像分成三份，这样你就有 9 个部分，如下所示。一般来说(记住所有的规则都是被打破的，但*大多数时候*)，如果你将焦点或主题放在水平线和垂直线相交的“三分之一”处，你就可以获得更有吸引力的构图。

##### [![RuleOfThirds](../Images/fc6e7c3bc5e21edb1b985497c160a97f.png "RuleOfThirds")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/RuleOfThirds.png)

如果你自己拍照，当你通过取景器看时，你需要想象水平线和垂直线。你会发现，当你记住这条规则时，你的许多照片会变得更加引人注目。

如果照片已经拍摄过，您可以在发送照片进行打印或将其添加到您的设计之前，使用三分法裁剪图像。我将向你展示如何在 Photoshop 中使用网格线的小技巧，但这个概念对于任何允许你使用网格线或指引线的图像编辑软件都是一样的。

1.打开要应用三分法的图像。为了改善我们的原始图像，我们将对其进行裁剪。很明显，裁剪会使图像变小，这是一个小小的缺点。下面你可以看到一些玩具士兵的照片。我在一家玩具店的橱窗里拍了这张照片，我想去掉一些杂乱的背景，把更多的注意力放在左手边的士兵上。

[![Soldiers](../Images/c2aca147d71574bfa498094616f97e68.png "Soldiers")](https://www.sitepoint.com/wp-content/uploads/2009/12/Soldiers.png) 
图片来源:詹妮弗·法利

2.我的原始照片尺寸为 30 x 20 厘米，分辨率为 240 ppi。如果你不确定如何找到这些信息，只需点击**图像>图像尺寸**，图像尺寸对话框打开，显示你的图像信息。创建一个比原始照片稍小的新文档，但要确保使用相同的颜色模式和分辨率。我把我的新文档尺寸设为 24 厘米 x16 厘米。

新文档激活后，选择**编辑>首选项** (PC)或**首选项** (Mac)，然后选择**参考线、网格&切片**。将打开一个对话框，为您提供设置参考线和网格线外观的选项。注意:下面的屏幕截图来自 CS4，与之前版本的 Photoshop 略有不同，但我们感兴趣的选项在所有 CS 版本中都是一样的。

将“网格线间隔”选项设置为 33.3%，将“细分”设置为 1。如果需要，您可以通过单击右侧的方形色样来更改线条的颜色。单击确定。

[![Preferences](../Images/fccaa7496b187392ee437373e23b2872.png "Preferences")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/Preferences.png)

4.为了查看您刚刚设置的首选项，选择**视图>显示>网格**。一旦你这样做，非打印网格将出现在你的文件。这是您将用来作为裁剪的视觉辅助。

[![Grid](../Images/986bd214a91c6c693404364a58980b63.png "Grid")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/Grid.png)

5.回到你的原始文档。按下 **Ctrl + A / Cmd + A** 选择整幅图像。按下 **Ctrl + C / Cmd + C** 复制选择，然后返回到新的空白文档。按下 **Ctrl + V / Cmd + V** 将图像粘贴进去。

选择移动工具( **V** )并拖动图像，使焦点落在其中一个交点上。在这种情况下，我拖动图像，使焦点在左上方交叉点的扳机上的小士兵的手指周围。

[![SmallerImage](../Images/7ddf27e5f9808eebeb6d362cf0ddd00e.png "SmallerImage")T2】](https://www.sitepoint.com/wp-content/uploads/2009/12/SmallerImage.png)

6.一旦你对构图满意了，你就可以裁剪了。按 C 键选择裁剪工具，然后单击并拖动整个图像。按**回车** / **回车**完成裁剪。要隐藏网格线，选择**视图>显示>网格**或按 **Ctrl + '** / **Cmd + '**

由于三分法，你现在应该有一个裁剪得很好，视觉上更吸引人的图像。
