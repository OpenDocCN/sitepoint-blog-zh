# 在 Photoshop 中按住 Ctrl 键单击图层

> 原文:[https://www . site point . com/ctrl-clicking-layers-in-Photoshop/](https://www.sitepoint.com/ctrl-clicking-layers-in-photoshop/)

Kevin 询问了更多关于 Photoshop 中“Ctrl-clicking”图层的细节，这里有一个简单的例子。

按住 Ctrl 键点按(对于 Mac 用户，按住 Command 键点按)层可让您基于该层中的像素创建选区。下面是一个多层 Photoshop 文件的示例:

![](../Images/866442b3a77b7e723b163e83a8ed798f.png)

当您按住 ctrl 键单击图层面板中的图层 3(箭头图层)时，您会根据图层包含的内容创建一个选区(由虚线显示):

![](../Images/c4097a9e16ad67491f2399ead93abe49.png)

同样，按住 ctrl 键并单击图层面板中的图层 2 会创建一个心形选区:

![](../Images/209967422df70feee9221f610dddca55.png)

与许多其他“选择”功能类似，如果您按住 Shift+Ctrl+单击图层面板中的多个图层，选择将被添加到一起。按住 Shift 键和 Ctrl 键单击箭头层可创建心形和箭头形状的选择:

![](../Images/beb2ff7234d09708c8ca6ee87783c476.png)

以这种方式创建选区可以保持透明度。在这个例子中，我有一个图层，它的形状是用喷枪工具创建的，外部边缘是透明的。我按住 ctrl 键单击该层以创建一个选择:

![](../Images/4a70ebd2dcfc29992c58ef5c865b0543.png)

我创建了一个新层(为了演示的目的关闭了原来的层),选择仍然可用:

![](../Images/36114456a20adfc49eb04187c9bad512.png)

我选择油漆桶工具和不同的颜色，并在选区中单击。边缘的透明度被保留！

![](../Images/11f784685bc6842373c698e04ee86ed6.png)
