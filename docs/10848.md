# 如何改变你的营销视角

> 原文：<https://www.sitepoint.com/how-to-revamp-your-marketing-perspective/>

我之前的帖子谈到了曾经有效的营销变坏的一些因素。营销是成功商业的必要持续因素之一，因为市场、潜在客户甚至行业都是变化无常的，不断变化。

此外，你所依赖的营销工具(广告、社交媒体、网站等)。)，也能很快改变。所以在你的营销活动中失去动力是很常见的。而且是可以修复的。

振兴你的营销活动和提高你的效果的最好方法之一是后退一步，看看你的目标受众到底想要什么。

## 透过你潜在客户的眼睛

[了解客户的想法并不容易。你需要超越你自己、你的业务和你的优先事项，看穿他们的眼睛。你可以通过清理你的大脑，试着以你从未见过的方式来看待你的品牌和营销。但这很困难，而且通常还不够。](../2009/11/04/understanding-how-your-clients-think/)

这里有一些关于如何进入你的潜在客户头脑的其他想法，这样你就可以更有效地向他们推销。

### 组织焦点小组

你可以组织一个焦点小组，就你现有的营销流程或正在进行的新活动获得一些反馈。焦点小组基本上是一种让一群人经历一些事情，然后接受采访的方式。例如，你可以向潜在客户介绍你的电子邮件营销活动，然后问他们一些关于他们体验的问题。

挑战在于找到合适的人，并让他们投入必要的时间加入你的团队。

### 与现有客户交谈

你有可能从已经在你名单上的客户那里得到一些关于你营销努力的反馈。因为他们是客户，他们可能喜欢你，也很乐意帮忙。

问题是，他们的输入是否足够公正有用，他们了解你和你的企业这一事实如何扭曲了数据？

### 使用离职调查

如果你想在你的网站上得到反馈，你可以使用一个调查，或者在访问者离开时弹出，或者在他们访问期间周期性地询问他们对用户体验的意见。不过这可能有点棘手。不仅弹出窗口令人不悦，有时还令人讨厌，而且如果有人离开你的网站，很可能他们不想在没有任何激励的情况下留下来回答几个问题。

## 如何处理你学到的东西

当你完成研究后，问题就变成了你如何处理收集到的数据。你可以先在几个月内做一些小的改变，看看你的目标市场会有什么反应。你可以保持现有的营销努力不变，尝试一些新的想法。或者你可以决定放弃现有的一切，重新开始。

最重要的是采取一个持续的计划来分析你的活动，重新审视你的潜在客户脑子里在想什么，并愿意适应和改变这一过程中的每一步。

你有没有退后一步，从目标客户的角度审视你的营销材料/活动？你如何决定什么时候该做出改变？
