# Mozilla 与谷歌的关系“复杂”

> 原文:[https://www . site point . com/mozillas-relationship-with-Google-complicated/](https://www.sitepoint.com/mozillas-relationship-with-google-complicated/)

自谷歌 Chrome 浏览器发布以来，谷歌和 Mozilla 之间的关系陷入了困境，这种说法是错误的，但毫无疑问，事情变得更加紧张了。当谷歌[在 9 月份宣布 Chrome](https://www.sitepoint.com/google-working-on-browser-what-about-mozilla/) 时，就在他们与 Mozilla 重新签订 2011 年合同的一周之后，我们想知道为什么谷歌会“从零开始开发自己的产品，而不仅仅是花更多的精力支持 Mozilla。”

他们的关键当然是控制。正如我们在最近的帖子中所讨论的，Chrome 是他们三管齐下的网络操作系统战略的重要组成部分。与 Gears(离线数据存储)和 Native Client(网络应用的本地 CPU 资源)一起，Chrome 为谷歌提供了一个引人注目的网络应用交付平台。从事网络应用销售业务的谷歌想要控制整个体验，这是有道理的。

然而，这可能会以牺牲他们与 Mozilla 的关系为代价。我们猜测，一旦 Chrome 上的扩展到来，Firefox 在用户采用方面可能会遭受至少一次轻微的打击，因为许多早期采用者会倾向于转换。就在一周前，谷歌[放弃了谷歌包中的火狐](https://www.sitepoint.com/google-turns-its-back-on-firefox/)，转而支持 Chrome。

Mozilla 首席执行官 John Lilly [在一次采访中告诉*计算机世界*](http://www.computerworld.com/action/article.do?command=viewArticleBasic&articleId=9124141)自从 Chrome 发布以来，他的公司与谷歌的关系至少有点紧张。

“我们的关系很好，也很合理，”莉莉说。“但如果我说事情没有过去复杂，那我就是在撒谎。”

Mozilla 收入的绝大部分——88%，约 6000 万美元/年——来自与谷歌的搜索广告交易。谷歌不太可能就此放弃，但既然谷歌已经成为竞争对手，Mozilla 也不指望这种收入流会永远存在。

“我们的目标是在 50 年甚至 100 年内成为网络的倡导者，你不能依赖任何一个组织，”Lilly 告诉《计算机世界》。“我们(目前的)三年协议是我们有史以来最长的一次。这是一个长期的视野，所以我们不需要马上做任何事情。但在未来三年，我们可以继续打造(新)产品，开发(新)收入来源。”

当然，那也是聪明的生意。不过，很明显，自从谷歌决定成为世界第二大浏览器制造商的直接竞争对手以来，谷歌和 Mozilla 的关系并不密切。Lilly 告诉《计算机世界》( computer world )( T1 ), Mozilla 计划与谷歌合作，但他们也计划竞争。最后，只要 Mozilla 继续运行，这对用户来说是一件非常好的事情。
