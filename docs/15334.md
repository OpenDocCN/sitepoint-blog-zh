# 让你的社区付出代价

> 原文:[https://www.sitepoint.com/community-pay/](https://www.sitepoint.com/community-pay/)

内容网站是出版物，这给了它们一种商业模式。商业网站是商店，这给了他们一个商业模式。但是什么是社区网站呢？

它们可以是虚拟的市政厅，可以是辩论社团，可以是沙龙，也可以是饮水机，但这些类比都无法提供商业模式的线索。

20 世纪 80 年代，社区网站在 CompuServe 等系统上出现，从它们产生的在线时间中抽取一部分收入。1993 年，AOL 决定转向“吃到饱”的定价模式，从而终结了这种商业模式。

互联网的繁荣使广告一度成为有效的商业模式。当横幅空间可以以每千个 20 美元的价格出售时，上面的任何内容都是多余的。但是游戏已经结束了。今天，许多社区——无论是基于网站还是电子邮件——向他们的用户讨钱(这从来没有用)或者干脆认输。

原因是许多建立网络社区的人没有理解他们自己的角色。他们正在建立俱乐部，他们想成为其中的成员，而不是他们要去经营盈利的企业。这些社区是爱心的结晶，但现在所有者正遭受捐助疲劳。

然而，有些网络社区模式将内容和商业元素结合在一起。

像内容网站一样，社区网站应该围绕一个市场或生活方式来组织；他们应该注册用户，向广告商提供经过审计的发行量数据；他们应该创造“会员”利益(访问档案，特别活动，电子邮件访问其他成员)，鼓励付费。

Classmates.com 是一个经典的成功故事。他们购买廉价广告，根据免费注册来衡量这些广告的成功，然后根据注册客人可以在电子邮件中看到的高中同学名单，向用户追加销售每年 30 美元的会员资格。

与电子商务网站一样，社区网站的用户可以分享对产品或服务的某些需求。网站所有者可以作为其他网站的会员赚钱；他们可以组织购买池(并从顶部拿走一点)；他们还可以建立“资源页面”——几乎不加掩饰的网上商城，为会员提供折扣。

但是在任何情况下，成功都是从了解你的观众开始的。你需要了解你的目标，就像出版商了解他们的目标一样。你需要像商店经理一样了解你的目标客户需要的产品和服务。你需要通过名字、地址、电子邮件和兴趣来识别你的成员。如果你想置身于自己创造的市场的十字路口，并从中分一杯羹，你需要这些信息。

知道你的目标是谁会告诉你你需要成为谁。如果你的社区是基于一种生活方式，你可能会像美国退休人员协会一样，寻求折扣来产生会员费。如果你的社区是建立在市场基础上的，你可能更像是在虚拟交易会 [Supercomm](http://www.supercomm2001.com) 的人们。

你可能从一个虚拟的俱乐部开始，但现在你已经结束了运行一个协会。你的成员代表了强大的经济力量，但是需要专业的帮助来驾驭这种力量。如果你不想做这一行，找一个想做的人——付给他们他们应得的报酬。或者，如果你真的想进入这个行业，那么:欢迎，但是你的学习曲线很陡。是时候认真对待了。
