# 网络应用程序图库

> 原文:[https://www.sitepoint.com/a-gallery-of-web-apps/](https://www.sitepoint.com/a-gallery-of-web-apps/)

微软的测试版 WebMatrix 通过利用网络应用程序库提供的应用程序，开辟了在 ASP.NET 建设网站的新途径。为了进行测试，Ricky 在 WebMatrix 计划于下个月全面发布之前，带着它兜了一圈，探索建立一个正常运行的电子商务网站是容易还是困难。在这个过程中，他踢了踢轮胎，关上了 nopCommerce 的大门，nop commerce 是一个开源电子商务包，微软已经选择它作为网络应用画廊的合作伙伴之一。在电子商务软件中，整合支付方式通常是一项困难的任务，因此 Ricky 尝试建立一个 PayPal 支付选项。这是研究微软网络应用程序库及其对开发者的意义的系列文章的第一篇。现在就阅读[文章](https://www.sitepoint.com/article/microsoft-web-app-gallery-part-1)，或者参加[测验](https://www.sitepoint.com/quiz/microsoft/web-app-gallery-part-1/)。
