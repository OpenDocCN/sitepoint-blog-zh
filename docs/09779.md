# jQuery 1.6.1 发布

> 原文：<https://www.sitepoint.com/jquery-1-6-1-released/>

嗨，伙计们，快速提醒一下，jQuery 的新版本已经正式发布了。

有许多变化，在您的站点上升级版本之前，您应该仔细遵循这些说明。

[官方发布说明](http://blog.jquery.com/2011/05/12/jquery-1-6-1-released/)

## 快速了解最新动态

1.  **。attr()现在是。prop()**——更新到属性模块——主要是。attr()函数。您的旧代码应该仍然工作，只是注意到一些测试集中在这个更新。
2.  **使用。attr()获取/设置以下内容:**自动聚焦、自动播放、异步、选中、控制、延迟、禁用、隐藏、循环、多重、打开、只读、必需、限定范围、选定。
3.  **使用。prop()获取/设置以下内容:**默认值，节点名，节点类型，selectedIndex，标记名

**从 1.5.2 升级到 1.6.1**

随着新的。prop()方法以及对。attr()方法，jQuery 1.6 引发了一场关于属性和特性之间的区别以及它们如何相互关联的讨论。它还带来了一些向后兼容的问题，这些问题已经在 1.6.1 中得到解决。从 1.5.2 更新到 1.6.1 时，您不必更改任何属性代码。

下面描述了 jQuery 1.6 和 1.6.1 中对属性模块的修改，以及。attr()方法和。prop()方法。然而，如前所述，jQuery 1.6.1 将允许您使用。attr()，就像以前在所有情况下使用它一样。

请注意，1.6 发行说明中描述的有关。data()方法已经得到解决，现在可以在 1.5.2 和 1.6.1 之间无缝地工作。

更多信息请参见官方 jQuery 1.6.1 发行说明。
