# 如何自信而不自大

> 原文：<https://www.sitepoint.com/how-to-be-confident-without-being-arrogant/>

自信是一种强大的特质。[自信能让你成功](https://www.sitepoint.com/confidence-for-success/)。自信的人就像磁铁一样……他们容易吸引周围人的注意、信任和尊重。他们相信他们能做到，所以他们承担风险并获得回报。和一个对自己的工作和完成工作的能力有信心的人一起工作让人放心。自信是会传染的——我们都想和自信的人在一起，让我们觉得我们也能做到。

但是自信和自大之间只有一线之隔。没有人喜欢自以为无所不知的人，或者不听的人，因为他们认为自己没什么可学的。如果你表现出傲慢，你可能会让人们失去兴趣，失去潜在的机会，并损害你的声誉。更不用说这可能是一种孤独的生活方式。因此，这里有一些方法可以最大限度地增强你的自信，同时又不至于自大。

## 平易近人

如果你因此变得冷漠或不友好，拥有自信对你没有任何好处。你希望能够增进和发展关系，而不是让人们对你刮目相看。你可以通过眼神交流和使用自信的身体语言让自己变得更加平易近人。

在社交场合，介绍你自己，用力握手(但不要太用力)，聊天时记住并使用别人的名字。在社交场合，参与谈话——无论是个人的还是与业务相关的——来展示你的个性，并给人们一种与你交流的方式。

## 要真诚

傲慢的人要么很少赞美别人，要么以一种强迫和做作的方式赞美别人，这几乎会破坏他人的成功。经常真诚地感谢你周围的人的努力工作和成功，甚至是那些你认为是竞争对手的人。

当进行日常对话时，认真倾听并专注于发展关系。避免说你认为对方想听的话，而是等到你感到感动的时候，给出一个真实的个人评论。

## 谦虚一点

在对抗傲慢的战斗中，谦虚和自信配合得非常好。要想被认为自信而不傲慢，当你的工作表现得到认可时，要表现得谦逊，并对自己的错误负责。

如果你是一个成功团队中的一员，花点时间分享赞美，给予赞扬，并指出他人的贡献。没有别人的支持，我们很难取得成功，承认别人在我们的成就中发挥的作用是展示自信而不傲慢的一个好方法。

## 错了

傲慢几乎总是伴随着你永远相信自己是对的，而自信则平衡了你倾听和向周围人学习的能力。虽然承认不确定性似乎会削弱你的自信，但事实恰恰相反。清楚自己知道什么和不知道什么是一种解放，这通常会得到他人的赞赏。

## 乐于助人

你自信是因为你擅长自己的工作。不要坐在架子上，把你的知识装在口袋里，而是分享它。在会议上发言，写博客，和你的同事交流，分享一些你获得的智慧。

拥有合作的态度可以防止你被贴上傲慢的标签，并且可以打开新关系和机会的大门。

你认识自信但近乎傲慢的人吗？你会建议他们做些什么来摆脱过度自信？
