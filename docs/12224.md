# YouTube 将屏蔽英国音乐视频用户

> 原文：<https://www.sitepoint.com/youtube-block-uk-music-viewer/>

广受欢迎的谷歌旗下视频分享网站 YouTube 宣布，将阻止英国居民观看在线音乐视频。该声明是在该公司未能与表演权协会(PRS)就新的许可条款达成一致后发布的。

PRS 代表 50，000 名作曲家和艺术家收取版税。该组织正在与 YouTube 谈判一项新协议，据传该协议比之前达成的协议条款高出许多倍，该协议最近到期。

双方都指责对方造成了破裂。

YouTube 主管帕特里克·沃克(Patrick Walker)表示，此举令人遗憾:

> 我们觉得我们之间的分歧太大了，以至于我们不得不在继续与 PRS 谈判的同时删除内容。我们将此消息公之于众，因为网站上的用户会注意到它。

该公司的博客还称:

> 成本对我们来说简直高得令人望而却步——根据 PRS 提出的条款，每次播放我们都会损失大量的钱。

PRS 的音乐首席执行官史蒂夫·波特(Steve Porter)对 YouTube 在持续谈判期间公布的事件版本提出质疑:

> 今天下午晚些时候，我们接到一个电话，通知我们谷歌的激烈行动，我们感到震惊和失望。
> 
> 谷歌告诉我们，他们正在采取这一步骤，因为他们希望向他们服务所依赖的音乐作者支付比目前低得多的费用，尽管 YouTube 的浏览量大幅增加。

无论结果如何，目前唯一明显的输家是英国公众。

这是“旧”经济为保住权力所做的最后一搏吗？或者，我们可以预期传统发行渠道对新媒体公司的诉讼会增加吗？网络是一个世界性的网络；在技术上有可能绕过这些限制的情况下，组织能够现实地规定允许个人浏览哪些区域吗？
