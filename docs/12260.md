# 微软将在 Windows 中提供竞争浏览器？

> 原文:[https://www.sitepoint.com/windows-alternative-browsers/](https://www.sitepoint.com/windows-alternative-browsers/)

欧盟委员会的最新裁决可能会迫使微软在 Windows 系统中提供替代的网络浏览器，以避免这家计算机巨头违反欧盟的竞争规则。据委员会称，微软正在利用其主导地位来限制网络浏览器市场。

如果初步结论得到证实，微软将有义务允许 Windows 用户选择一个与之竞争的网络浏览器，而不是或除了 Internet Explorer 之外。一个可能的解决方案是在 Windows 安装过程中出现“浏览器选择”屏幕，或者允许计算机制造商(如戴尔)提供一组预安装的替代选项。

欧盟委员会希望避免在 Windows Media Player 裁决中犯下的错误。在那种情况下，微软被迫生产 Windows XP N，一个不包含 WMP 的操作系统版本。没有价格折扣，不出所料，对该产品的需求很少。

然而，替代浏览器的规则可能并不是没有自己的问题:

1.  欧盟委员会如何确保浏览器的名单是公平的？提供 IE、Firefox、Opera、Safari 和 Chrome 将是一个合理的选择，但还有许多其他不太受欢迎的替代品。
2.  很少有新手用户理解什么是网络浏览器。他们如何做出明智的决定？如果有必要，他们是否能够重新考虑他们的选择？
3.  苹果 OSX 或 Ubuntu Linux 等其他操作系统会被迫提供类似的浏览器选择吗？

微软必须在 2009 年 3 月中旬之前向欧盟委员会做出正式回应，在最终裁决宣布之前，可能会举行进一步的听证会。

你有什么想法？微软能证明一个纯 IE 版本的 Windows 是正确的吗？这项裁决会有助于还是阻碍替代浏览器的推广？
