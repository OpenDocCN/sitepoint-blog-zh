# 链接会拯救报业吗？

> 原文：<https://www.sitepoint.com/will-links-save-the-newspaper-industry/>

很长一段时间以来，Publishing 2.0 的斯科特·卡普(Scott Karp)一直在推广链接新闻的理念:你越是将访问者引向其他人网站上的好内容，他们将来就越有可能回到你的网站。这与报纸行业多年来盛行的态度相反，即你必须尽可能长时间地锁定访问者，才能从他们身上赚钱。但也是同样的原则让谷歌赚了数十亿美元——谷歌越快送你走，你就越有可能回去。

卡普本月早些时候公布的统计数据显示，Drudge Report，一个几乎 100%链接到外部资源的网站，在所有新闻网站中拥有最高的读者参与度。主流新闻网站开始注意到这一点。

几个月来,《纽约时报》的[科技版](http://www.nytimes.com/technology/)一直在使用他们的算法 Techmeme-like“blog runner”软件链接到科技博客。华尔街日报的 AllThingsD 网站有【T2 之声】部分，链接到精选的科技博客文章。

现在，随着政治浏览器的推出，《华盛顿邮报》也加入了新闻链接的潮流。政治浏览器是一个人工编辑的聚合网站，不仅链接到华盛顿邮报的博客和内容，还链接到第三方网站上精选的内容。

正如《washingtonpost.com》的政治编辑 Eric Pianin[告诉 Scott Karp](http://publishing2.com/2008/09/28/washingtonpostcoms-political-browser-uses-the-news-judgment-of-journalists-to-filter-the-political-web/) 的那样，政治浏览器在选定的报道上盖上《华盛顿邮报》的“批准章”。本质上，它是一个来源可靠的过滤器。皮亚宁告诉卡普,《华盛顿邮报》被《赫芬顿邮报》或《德拉吉报告》等聚合网站的成功“迷住了”,可以利用它们强大的品牌和在读者中的信誉来分一杯羹。

《华盛顿邮报》的页面比《华尔街日报》或《纽约时报》的技术页面走得更远，链接到了可靠的来源，并将他们的品牌与他们选择的链接内容联系起来。这可能会让《华盛顿邮报》和其他效仿他们模式的报纸成为赢家。卡普是对的，过滤行业存在巨大的机会。创建和发布内容的门槛很低，现在每天都有比以往更多的内容和信息向我们涌来。结果是信息过载和低信噪比。

我们已经讨论过[可信度过滤](https://www.sitepoint.com/the-future-of-news-filtering-by-credibility/)作为内容聚合的未来，但是指出了当前群体智慧模型的一些内在缺陷。尽管报纸已经成为读者信任的信息来源，但它可以成为内容聚合者，并通过可信度进行过滤，从而真正引起读者的共鸣。
