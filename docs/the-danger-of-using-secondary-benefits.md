# 使用次级福利的危险

> 原文：<https://www.sitepoint.com/the-danger-of-using-secondary-benefits/>

我的一周 100%是关于文案写作的。

在这一周里，我的一位同事给我上了重要的一课，这一课适用于你写的任何关于你企业的东西:

了解主要利益和次要利益之间的区别，并努力关注主要利益。

一个主要的好处是有人雇佣你的根本原因。次要利益是一种较软的辅助利益，通常支持你的主要利益。

那么，就你的企业而言，客户雇佣你的主要原因是什么？如果你的客户是企业，可能只有一个原因:

–您帮助他们在更短的时间内赚更多的钱

此外还有其他好处:

–你通过改善他们的形象让他们看起来更好

–你为他们提供了接触新潜在客户/客户的新途径

–你很快就能完成任务

–你让他们对其业务传达的形象感觉更好

–不要留下一个糟糕的网站或者根本没有网站

我并不是说以上是唯一的主次好处。但你必须专注于人们从你这里购买的根本原因。你的营销信息由此而来。如果你过于关注次要利益(你可能关心但你的客户不太关心的事情)，你将不会产生尽可能多的收入。

我给你举个例子。我问一位网页设计师，为什么我应该雇佣他。他的回答是:“我可以帮助你的网站设计使用更多的曲线和其他看起来更好的元素。”现在没有办法，我要为曲线和其他元素每小时支付 50 美元或更多。你会吗？这位设计师和许多像他一样的人需要想出一个让人们购买的主要理由。

根据您的目标市场和提供的服务，有许多主要优势。试着把你的贴在这里，以供评论和讨论。

哦，对那些庆祝圣诞节的人来说，祝你有一个和平快乐的圣诞节！
