# 如何设定与客户的界限

> 原文：<https://www.sitepoint.com/how-to-set-boundaries/>

上周，我写了一篇关于[应对在家工作的挑战](https://www.sitepoint.com/working-from-home-challenges)的文章，其中最常见的挑战之一是划分工作和家庭生活的界限。根据我的经验，这是在家成功工作的一个重要部分。

边界设置的一个主要因素是持续管理你的工作关系。这里有一些方法可以做到这一点。

**制定政策**

通过制定管理客户关系的政策，并与它们保持一致，你的界限将成为第二天性。您可能希望创建业务范围策略的一些领域包括:

*   费率、定价和折扣
*   可用性和工作时间
*   计费实践
*   一般周转时间
*   会议请求

你甚至可能想把这些领域的一些内容纳入你的合同。

**确保你的客户理解你的政策**

仅仅制定政策是不够的。你需要能够将这些政策传达给你的客户，并确保他们理解这些条款。如果你不选择在合同中包括你的政策，你可以起草一份文件概述你的政策和程序。将此文档作为欢迎包的一部分发给新客户。

如果需要，你也可以在更非正式的基础上口头传达你的政策。这种方法传达起来可能更温和，但要跟踪谁知道哪些政策可能更具挑战性。

**极限异常**

总是有例外，即使你有非常清楚和明确的界限。例如，对于一个长期客户，当情况允许时，你可以决定对你的可用性和周转时间稍微宽容一些。

为了避免让您的例外超出您的能力，您可能需要考虑将例外构建到您的总体规划中。这些信息不会公开，但它可以帮助你根据具体情况做出决定。

**沟通清楚**

正如有效的沟通在你业务的其他领域是至关重要的一样，在讨论和告知客户你的界限时也是至关重要的。这取决于你创造的界限，然后确保你不仅让你的客户知道，但愿意回应的关注，可能会发展出你的政策。回答问题并提供一个你为什么有这个政策的大概想法(不要证明它)可以让你的客户更愿意接受你制定的规则。

**尊重你的客户和他们的界限**

就像你教你的客户你希望如何被对待一样，你也应该提供你要求他们给予你的同样的尊重。就您设定限制的相同领域，询问您的客户的流程。如果你的客户有某些对他们至关重要的要求，愿意考虑在你的日常关系中采纳他们的政策。

请记住，所有的关系都是建立在给予和接受的基础上的，所以通过清晰的沟通和妥协，你正在与你的客户建立健康互惠的关系。
