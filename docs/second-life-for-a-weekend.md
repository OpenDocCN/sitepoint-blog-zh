# 周末的第二人生

> 原文：<https://www.sitepoint.com/second-life-for-a-weekend/>

度过了一个空闲的周末，而不是做一些明智的事情，比如享受美好的天气，决定看看来自[林登实验室](http://lindenlab.com/)的[第二人生](http://secondlife.com)，这已经在过去几个月里[在](http://secondlife.blogs.com/nwn/2006/01/lawrence_lessig.html)[更多](http://crystaltips.typepad.com/wonderland/2006/05/radio_1s_one_bi.html)和[更多](http://scobleizer.wordpress.com/2006/03/01/introducing-scobleizer-inc-in-second-life/)雷达上出现，并且刚刚开始到达[主流媒体](http://secondlife.com/community/connections.php#news)。

它实际上已经运行了三年多，但是关于它与现实生活，尤其是经济学的关系的严肃讨论似乎引发了更广泛的兴趣。

在阅读它的时候，真正引起我注意的是在游戏相关页面上看到像`secondlife://Shamrock/17/163/99`这样的 URLs 它标识了游戏中的位置。让我想知道他们是否找到了将今天的 2D，基于文本的互联网与 3D 空间连接起来的神奇配方。进一步阅读导致 [LSL](http://en.wikipedia.org/wiki/Linden_Scripting_Language) ，一个迷你语言，供玩家扩展他们的环境，最近还获得了一个 HTTP 客户端，使第二人生混搭。我也想知道这是不是 3D 互联网的原型。

## 什么是第二人生？

在我看来，这是一款在线 [MMORPG](http://en.wikipedia.org/wiki/MMORPG) 游戏，它有一些独特的方面，有别于其他流行的 MMORPG，比如《魔兽世界》。

首先，没有目标:对于“有什么意义？”没有单一的答案。。称之为游戏会让人产生错误的期望。我猜社交网络更接近目标。

更有趣的是游戏中几乎所有的“内容”;树木、房子、衣服，你能想到的都是由玩家创造的，这就引出了一个合理的问题“这是一个[操作系统](https://scobleizer.wordpress.com/2006/03/27/second-life-is-an-os/)吗？”。这个游戏发展了一种非常丰富的文化，以服装、建筑和你能想到的几乎任何东西的形式，都是玩家创造的。

像往常一样，如果你想知道更多，最好的介绍性文章是维基百科。基于一次短暂但激烈的经历，我想我应该抛出一些印象和观点。实际上，我现在已经删除了我的账户——稍后会详细介绍——对于认真的《第二人生》玩家，你可以愉快地指出我的无知——我会证明这一点，理由是第一印象很快就会被遗忘，但有时最有价值。

## 入门指南

第二人生有不同的注册计划，林登实验室最近开始提供免费访问(引起了一些 T2 的抗议),这就是我使用的。如果我没看错小字的话，免费帐号给了你无限的游戏时间，但是限制了游戏“不动产”的所有权(你不能拥有任何不动产)，这可能最终会限制你的体验。但是要了解第二人生是什么，一个免费账户就足够了。

注册后，你在 Nullsoft 的安装程序的帮助下为 Windows 安装一个客户端(Mac 也支持，但较弱，Linux alpha 也在某处)——我提到这一点是因为它让我感觉到第二人生的开发者来自哪里。安装完成后，登录会将您带到初学者集结地，在那里您开始四处走动并理解 UI，在标志/说明的帮助下为您指出正确的方向。

客户端用户界面本身是可以接受的——让出行/做事变得相当高效/友好，但还有一些地方需要改进。例如，从输入 IM 聊天文本切换到四处移动需要鼠标和键盘或者失去文本光标控制，这取决于你在哪个框中输入。此外,“库存”窗口——一个树形控件——变得令人沮丧，因为它开始填满你收集的东西。一个类似于 Outlook +无尽邮件夹的问题:我认为需要一个类似 GMail 的“搜索”功能。除了缺点，它或多或少是直观的——我几乎不需要在在线帮助中查找……这很好，因为在线帮助不多。从这个角度来看，我想知道第二人生是否真的准备好接受它开始受到的关注。特别是，缺乏关于建筑材料的详细信息，至少是文本形式的信息；游戏中有教程，玩家愿意提供帮助，这可能就足够了——我没有深入探索这条道路。

## 建立工作关系网

与其他玩家聊天是“第二人生”的一个重要组成部分——对于初学者来说，这实际上是必不可少的，所以要花时间并且有礼貌。在这方面，它的工作原理与其他聊天应用程序非常相似——即时消息、朋友跟踪等。他们还计划以某种方式支持 Jabber，大概是为了与游戏之外的其他聊天网络挂钩。与此同时，Skype 很受喜欢聊天而不是打字的人的欢迎。

玩家组织的单位是“群”(免费账号不能开始新的群但是你可以属于其他人群)，通过它你也可以广播消息。

## 表演

…总的来说相当不错。当你在 Second Life 中移动时，客户端正在缓存大量数据，而就移动响应而言，滞后并不是一个真正的问题，这是来自欧洲的跨大西洋延迟。值得注意的是，当你进入一个新的位置(传送)时，需要一段时间来加载组成地形的所有矢量对象，再加上更多的时间来加载任何位图，这似乎是“按需”的。光栅图像通常由玩家上传，作为矢量风景的纹理，以及用于游戏制作。

偶尔我也会观察到奇怪的行为，比如进入一个新的地方，发现我的衣服和头发还在路上；有点令人不安。

数据库问题并非不为人知，周六晚上遭到了雷击——有趣的阅读是 [Web 2.0 和数据库第一部分:第二人生](http://radar.oreilly.com/archives/2006/04/web_20_and_databases_part_1_se.html)顺便说一句

转到更主观的话题上…

## 这都是表面现象

在弄清楚如何移动后，第一件要做的事就是改变你的“替身”的外表。游戏在这一点上绝对令人印象深刻——即使是初学者也可以做很多事情来改变他们的风格——我想我第一次尝试就燃烧了将近一个小时。与此同时，在高层次上，它得到了[照片的真实感](http://secondlife.blogs.com/nwn/2006/02/the_skin_youre_.html)。

事后看来，让这成为与游戏的第一次重要互动并非巧合。虽然第二人生没有客观和无限的可能性来创造事物，但某些先决条件定义了游戏中的规范。大多数玩家似乎全神贯注于追求完美的外表(我也不例外)——成为一个超级宝贝/家伙。这也是鉴别新手的一个标准，与专业人士相比，新手看起来是灰色的。我认为这阻碍了基本的第二人生设计决策——一个不同的设计可以把重点放在其他地方。

所以这一切都很有趣，就像玩偶之家一样，但从长远来看，这是一种限制——你要么追求完美的外表，要么试图将自己与完美主义者区分开来——两极分化。

## 仅限成人

说到外表，还应该指出的是，你将会面对带有“成人”标签的事物。类似于[这个](http://www.flickr.com/photos/56083391@N00/188143644)的东西，大概是在第二人生“道德尺度”的“保守”端。我来自西欧，在这一点上我是自由放任主义者，但如果你更敏感的话，请注意。检查:如果你得到了[这个视频](http://video.google.com/videoplay?docid=-6705121006644488439)(非冒犯性)中的幽默，你就没事了。

他们确实有一个针对未成年人的游戏版本，这可能有助于缓解外部观察者对这个问题的关注(斯考伯因此被禁赛)，但我发现自己想起了隔壁的 T2 女孩(T3)——孩子们知道的比成年人多。

实际上，林登实验室没有办法控制所有用户在第二人生中创造的内容。即使他们真的试图打击，毫无疑问他们会失败，只会让玩家心烦意乱。同样，在欧洲人的思维模式下，“遵循这些一般规则，不要愚蠢或失去它”的方法很有意义，似乎在很大程度上得到了遵守——我希望这能继续为他们工作。版权规则似乎也得到遵守——虽然有“混音”,但我没有发现任何明显的抄袭行为。

## 创建内容

…是我没有任何第一手经验的一个领域。除此之外还有很多事情要做，但是如果你认真对待第二人生(有些人非常认真)，这里有一些陡峭的学习曲线。我玩了一点创作工具，但发现游戏中的工具很难管理(第一印象)，几乎没有在线帮助。这是一个边做边学的过程，也许可以和大多数人学习 HTML 的方式相提并论。但这需要时间。

至于人们在创造什么，我想了想…

*   物体/人工制品:从衣服(甚至皮肤)到汽车和建筑。以及任何你可能“拥有”的东西。这在很大程度上涉及到使用游戏中的矢量图形工具，也许还会附加一些 LSL 脚本来允许对象做一些事情。
*   化身动画:除了你的化身可以执行的基本动作，由客户端用户界面控制，你需要 [BVH](http://en.wikipedia.org/wiki/Biovision_Hierarchy) 或 LSL 脚本让他们做更高级的动作。《第二人生》中一个受欢迎的活动是[跳舞](http://video.google.com/videoplay?docid=6988580943302153877)，但是你的虚拟角色只能通过动画来做到这一点，你要么去找/买，要么去游戏中的“俱乐部”找，那里可以提供动画(通常是彩色的球)。
*   媒体:创作/混音音乐，上传艺术品和几乎所有你可以数字化的东西。看看周围，似乎有艺术家专门为第二人生创作内容，甚至一些知名的“品牌”也在客串。

我能评论的是*观察*其他人创造的东西。很多时候《T2》简直是令人目瞪口呆的《T3》——玩家创造了一层又一层的画面，往往细节惊人——你真的会迷失其中。

## 对象交互和用户界面

对我来说，这是一个大问题。给定一个无止境的用户创建的对象，你如何与它们交互？经常是猜测工作点击这里那里和任何地方。通过右键单击，默认动作“Sit”似乎经常是几乎所有没有显式覆盖它的对象的默认动作。与此同时，有些东西，比如门，会对单击鼠标左键作出反应，而你的物品清单中的动画要么贴在你的身上，要么像媒体文件一样被激活。对我来说，这感觉混乱而令人困惑——一个已经泄露的抽象概念。这不是不可用的，我及时发现了这一点，但这与亚马逊的“一次点击”相去甚远。

这是林登实验室在最初的设计中无法完全预见的问题，现在已经被玩家以一种特别的方式解决了，如果不中断人们的工作将很难修复。我觉得缺少的是某种突破，就像 URIs 和链接对万维网和超文本的意义一样。

## LSL 剧本

我尝试过的一点点内容创作，也包括了对第二人生脚本语言的修改。这…怎么说呢…有意思。强类型(有益于机器而不是作者)和围绕对应于对象“事件”如被触摸的回调函数组织，你也有一个可用于与游戏交互的 API 以及一个 XML-RPC 客户端，最近还有 HTTPRequest。

这个*听起来*很有希望，但是我发现 [Dav Yaginuma](http://www.akuaku.org/) [的经历揭示了](http://www.akuaku.org/archives/2005/03/a_day_in_the_se.shtml)(先于 http request API)——我只是没有那么大的动力。

Second Life 确实需要支持像 Javascript 或 Python 这样的语言来获得采用。有很多关于即将在 [Mono](http://en.wikipedia.org/wiki/Mono_development_platform) 上运行的新脚本引擎的参考，所以也许这是可行的。

## 所有权

林登实验室的政策之一是玩家拥有他们创造的内容；最有趣的是，它允许他们出售它(也许最终能赚到真钱)。关于这一点的全部含义有很多讨论，这些讨论似乎还没有得到充分的检验(事实上，让我们希望它保持这种状态——不需要诉讼)。从玩家的角度来看，对于你创造的任何东西，你都有“锁定”的选项，这样你可以出售它，但买家不能复制它或把它给其他任何人。但是你也可以选择免费使用——让任何人做他们想做的事情。

这是我认为基础设计塑造了游戏文化的另一个领域(我认为这里是开源)。虽然编码人员免费提供他们的一些知识和例子，媒体/设计人群似乎有一个[不同的观点](http://www.slboutique.com/index.php?p=buy&cat=27&phrase=Footwear%20Female) 0 你必须付钱给朋友！

## 有缺陷的乌托邦

更一般地说，游戏币是能够拥有有趣体验的一个关键因素，这意味着林登实验室不会因为创造了一个乌托邦而获得任何奖项。努夫说:“你的薪水很少(取决于你的账户类型)，但一旦用完了，你可以用真金白银买路，或者想办法让其他玩家给你钱，比如卖东西或服务。”。对于免费注册的新手来说，这是一个问题，但同时也给了你一些关注的东西。同时，你不需要任何东西来“存在”于“第二人生”——没有食物/睡眠等等。这只是你*想要*多少的问题。

如果你在游戏中经营商店之类的生意，那么这一切都是为了吸引其他玩家的注意力。有一个内置的搜索系统，其中排名似乎通常是基于访问者的数量(这是一个不错的指标，但排除了新的)。如果你在搜索中排名不高，你必须想办法传播这个消息。随着消息的传播，问题来了，你碰巧认识谁——毫无疑问，认识一线明星是有帮助的。

所以在这个社会系统中有很多东西可以提醒我们日常生活。我想我的意思是，作为一个创造内容的空间,《第二人生》将自己限制在一个特定的模式中，这个模式可能会限制长期，但适合短期，给玩家一个容易联系和潜在利用的模式。我想我在找科幻小说。

## 第二人生作为一个平台

退一步说，我的印象是第二人生真的是 3D 互联网的原型。通过 Alexa 查看顶级网站，你不得不得出结论，今天最大比例的用户仍然是书呆子，否则我们不会有像 [PHP 这样的网站排名前 200](http://www.alexa.com/data/details/traffic_details?q=www.php.net&url=php.net) 。虽然非书呆子知道互联网的存在，但它还没有成为他们生活方式的一部分。3D 互联网可能会改变这一点——虽然我们不必将其视为下一个重大购物体验，但这可能就是为什么[亚马逊的创始人是投资者](http://news.com.com/Second+Life+scores+11+million+in+funding/2100-1043_3-6054598.html)？

在网址方面，他们的游戏地点还没有超过 URIs，这是一个遗憾。如果你开始认真考虑将第二人生作为一个做生意的地方，你希望能够给人们提供一个链接，链接到你刚刚创造的(或者甚至计划创造的)东西——考虑独特的*资源*，在[休息](http://en.wikipedia.org/wiki/Representational_State_Transfer)的意义上，让一切，包括头像，都是可识别的..

另一个方向的集成，将互联网带入第二人生，正在(显然)与林登实验室正在开发的 [UBrowser](http://ubrowser.com/) 一起进行——一个使用 Mozilla 代码的游戏内网络浏览器，玩家可以将其附加到对象上。

根据维基百科的文章，已经有玩家对在第二生命网络中添加服务器来托管“土地”感兴趣，但目前林登实验室已经拒绝了。短期内这是可以理解的——引入了太多的新问题。从长远来看，如果它不对所有人开放来扩展基础设施，我不认为它能作为一个平台工作，就像你可以一巴掌打倒一个灯栈，安装 WordPress 然后离开一样。

毫无疑问，第二人生正在破土动工，而林登实验室在解决这一领域的问题方面获得了独特的经验。“第二人生”是否会变得比游戏和新奇事物更有意义，还有待观察。就我个人而言，我持怀疑态度。

## 时间就是金钱

那么，尽管我已经享受了第二人生周末，为什么我还是取消了我的账户呢？

很简单——我没有时间；工作、家庭等。先来。漂浮在那里的上瘾的钩子让我很紧张。有些人[不明白](http://chris.pirillo.com/2006/06/09/second-life-i-dont-get-it/)(时尚！)但我发现自己开始被细节的强度、无限的可能性、社会方面等等吸引住了。足以让我想远离它！

我有时间(也愿意花钱)做的事情是偶尔看一部电影——两个小时的娱乐节目，没有任何事情会在你试图入睡时占据你的大脑。我不知道你是如何在一个网络游戏中做到这一点的——也许就像一个控制账户限制我可以花的时间一样简单？但是，当其他人都在拼命工作时，我怎么才能“出人头地”呢？如果我正在谈话呢？自律飞出窗外…

如果林登实验室(或其他)能够把我当作一个客户，而不是在我醒着(和睡觉)的每一个小时里寻找，也许我会回来。谁知道呢？无论如何——“第二人生”绝对值得一试。盯着时钟就行了。

是时候呼吸点新鲜空气了。
