# 磨砺你的专业工具包

> 原文：<https://www.sitepoint.com/sharpen-your-professional-toolkit/>

我是我们家的厨师，我喜欢在厨房里度过时光。去年，我决定是时候买一些真正高质量的刀了。当他们到达时，他们像刀片一样锋利。尽管我每天使用它们一段时间后都在磨砺它们，但它们明显变钝了。

前几天，我坐下来，开始磨我所有的刀的乏味过程。有些被忽视了，它们被切割了，而且很钝，几乎不能切割。我花了几个小时把刀片恢复到原来的状态。

专业刀是厨师工作需要的工具。他们必须照顾它们，包括定期维护，例如抽出时间将它们打磨回原始状态，以便它们发挥应有的性能。

作为网站开发者和企业主，我们的工具箱里也有工具。它们可能不是刀，但我们的行业工具需要保养和维护，就像一套专业刀一样。尽管我们已经在这个行业里呆了很多年，但是如果没有不断地磨砺我们的工具和知识，我们就会变得迟钝，就像一套被频繁使用的厨房刀具一样。

## 跟上技术的变化

似乎总会有新技术或更新出现，改变我们对一个古老问题的思考方式。Jeffrey Zeldman 刚刚写了一篇关于 Scott Kellum 发现的新 CSS 图像替换方法的文章。我们多年来一直在使用-9999px 黑客，但新方法更有效。

有很多方法可以跟上正在发生的变化，我个人最喜欢的仍然是订阅 RSS 提要。我订阅了 SitePoint、Zeldman 和其他几个 feeds，大约每周阅读一次。你也可以订阅电子邮件时事通讯，关注 Twitter feeds，或者任何最适合你的东西。

## 定期磨练你的技能

仅仅关注博客和网站是不够的——你需要运用这些新方法和新技术。脱离实践是很容易的，尤其是如果你是管理层或者老板的话。但是，如果你长时间不练习，甚至很难有效地评估一个项目。

我已经有一段时间没有对页面布局进行切片和编码了，最近我们人手不足，所以我做了几个登录页面。我很惊讶，短短几年时间，我竟然忘记了这么多。我生疏了，过了几天才真正回到自己觉得舒服的地步。

展望未来，我会试着在这里或那里建立一个网页或网站，或者只是帮助一个项目，以避免变得如此生疏。

## 与其他专业人士建立联系

除了坚持阅读和磨练技能，与其他网络专业人士交流讨论当前话题和建立关系也是很好的。Meetup.com 有网页设计、编程、社交媒体以及你能想到的任何其他主题的小组。

除了 meetups，还有 Google+房间，或者你可以在当地的咖啡店遇到一些网络专业人士，讨论当前的问题。他们还提供了一个很好的网络，当你太忙而无法接手一个项目时，你可以推荐潜在的客户……这是双向的。
