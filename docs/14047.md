# 开源会计

> 原文:[https://www.sitepoint.com/open-source-accounting/](https://www.sitepoint.com/open-source-accounting/)

我正在经历自己业务的转变，希望超越 Quicken 作为我的会计工具。众所周知——Quicken 对于家庭和家庭办公室会计来说是一个无价的软件包——然而，如果一个小企业成长，Intuit 更倾向于自然进化，向 Quick Books 迁移。

最初，我开始使用手动发票系统，并逐渐发展成 Quicken，我需要进入下一个阶段。然而，我不想使用基于网络的应用程序，因为我可能会将详细信息同步到我的 iBook，并在路上、飞机上和机场工作时记录详细信息(是的，我知道 WiFi！).此外，正如许多人通过阅读 Open Sourcery 已经知道的那样，我会在运行 OS X 的 MAC 电脑上加载该系统。

作为尽职调查的一部分，我开始探索对于一个成长中的小企业来说，还有什么可以利用的，这个企业有定制需求，但在很大程度上仍然是虚拟的。我很高兴最终选择了 GnuCash。从 1.8 版本开始，也许更早，因为我已经很久没有重新访问这个程序了，现在支持 OS X(通过 [Fink](http://fink.sourceforge.net/index.php?phpLang=en) )以及 PostgreSQL 后端。

GnuCash 支持我试图实现的发票和帐户管理功能，并且显然使我能够通过其他途径继续支持开源。

也许这也是一个我可以贡献的项目(实践我所宣扬的)，写文档或一些其他功能(因为我不写 C！).
