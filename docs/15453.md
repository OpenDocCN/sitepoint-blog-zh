# Macromedia Fireworks

> 原文:[https://www.sitepoint.com/macromedia-fireworks/](https://www.sitepoint.com/macromedia-fireworks/)

Macromedia Fireworks 是一个图形程序，专为创建基于 web 的图形而开发。

我过去试验过许多图形程序，发现 Fireworks 允许最大程度的控制，使用方便，功能强大。我发现一些绝对必要的功能很好地适应了程序，包括图层、特效、使用 Photoshop 插件的能力等等。

Fireworks 的一些最独特和强大的功能在于图形的导出，传统上必须由第三方软件来处理，以实现可行的压缩。Fireworks 允许创建和编辑动画 GIF 文件、JPG 文件和新的 Internet 标准文件 PNG(便携式网络图形)。

Fireworks 程序的另一个强大功能是图像的所有部分都可以随时编辑。这在图形程序中是前所未有的。基本上这意味着，如果你完成了一幅图像，并决定你不喜欢某个区域，这没问题，任何事情都可以在任何时候改变，不用担心。例如，如果你在一张图片上的一些文字中犯了拼写错误，你可以点击它并改变它。这可以比作自动为图像中的每一个事物创建一个层，以便可以随时编辑它而不影响图形的其余部分。

除了可以随时食用之外，Fireworks 还具有实时效果。这意味着，如果你想试验一个斜面，你可以试一试，然后立刻看到这个变化会如何影响整个图像。(其他一些效果有投影、浮雕、发光等)。

您还可以创建 JavaScript 翻转、动画并优化您在互联网上使用的图像，创建图像映射，批处理您的图形等等。Fireworks 还有一个很棒的导出预览窗口，在您保存图形之前，它会向您显示图形在不同格式和不同设置下的外观。我可以继续下去，Fireworks 的功能数量是任何图形程序都无法比拟的。

烟花唯一的主要挫折是 300 美元的价格标签。虽然对于中档买家来说这是一个很好的价格(Adobe Photoshop 目前售价约为 1000 美元)，但对于预算较低的人来说，Paint Shop Pro 可能是一个更好的选择。

这个程序的一个小缺点是持续的可食用性和实时效果会占用内存。当您的系统上只有运行 windows 的 32 MB 内存时，Fireworks 有时仍然只能使用大约 5mb 的内存。基本上，这意味着我不建议运行任何其他占用大量内存的程序…例如微软的任何东西。

**评级:5**
