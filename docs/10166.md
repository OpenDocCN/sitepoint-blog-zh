# 及时:当时间比时间更重要的时候

> 原文：<https://www.sitepoint.com/timely-when-more-than-time-is-of-the-essence/>

如今，忽视 Twitter 作为商业交流工具的重要性是不可能的，但是使用无数可用的工具找出如何最好地管理它本身几乎是一项全职工作！

我用 Twitter 做生意有两种方式。一个是通知我们的追随者我们最新的博客文章，另一个是在更私人的层面上与我们的社区交流。我通过一项服务实现了第一个目标，这项服务从我们的 RSS 提要中提供博客内容。第二个是我一直手动完成的。我说的是关于论坛比赛的微博，或者问人们他们现在在做什么。一般来说，是社区通知，偶尔还会发布推文来激起狂热。这就是社区管理有趣的一面。

有很多服务可以让你提前安排推文，我过去也用过很多。但我没有像爱上[适时](http://timely.flowtown.com/?utm_source=ft-homepage&utm_medium=hellobar&utm_campaign=hellobar#/)那样爱上她们中的任何一个。公平地说，我一直是一个容易坠入爱河的女孩，但我相当肯定《适时》对你们很多人来说会很有吸引力。引起我注意的是它的不同点。及时分析你过去的 200 多条推文，并计算出发布你预定推文的最佳时间段，以获得尽可能好的报道。对于一个像我这样的新西兰女孩来说，当我们的大多数观众都睡着了的时候，她往往会发微博，这是一场天作之合！

如果你需要证据，数字说明了一切。有一个性能标签，告诉你每条推文有多少点击和转发，以及它的影响范围。@sitepointdotcom 有 68，440 名粉丝，由于及时，我们有超过 82，000 次浏览的定期推文。没什么可抱怨的…

撇开统计数据不谈，这项服务很容易使用，而且是免费的。我给它竖起大拇指。
