# 红宝石高尔夫

> 原文:[https://www.sitepoint.com/ruby-golf-2/](https://www.sitepoint.com/ruby-golf-2/)

Ruby golf 是一门使用尽可能少的字符编写代码的艺术。这个想法起源于 Perl 世界(毫不奇怪，它被称为 Perl Golf)。作为一种语言，Perl 非常适合这一点，因为它有着超乎寻常的结构和语法。Ruby 还包含一些大块的语法糖，这使得它适合打高尔夫。
