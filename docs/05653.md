## 2013 年 4 月至 2014 年 4 月全球台式机和平板电脑浏览器统计

这些表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏表示比例变化，即上个月 24%的 IE7 用户更换了浏览器。有几个警告，所以我建议你阅读[浏览器市场份额是如何计算的](/how-browser-market-share-is-calculated)。

在长期稳定之后，Chrome 单月上涨了 1.6%。即便如此，与它的移动版相比，这也不是特别令人印象深刻。也许这只是一个统计上的暂时现象，但 Chrome 看起来将在 2014 年突破 50%的大关。

Chrome 的一些进步可以用 Windows XP 的消亡来解释。微软在 4 月 8 日停止了支持，所以看到 IE8 下降近 1%并不奇怪。IE11 吸收了一些损失，但 Internet Explorer 看起来很脆弱，如果目前的趋势继续下去，几个月后将低于 Firefox。

火狐本身下跌了一点，但 Mozilla 肯定希望第 29 版会成功。意见有分歧，但大多数用户似乎对三年来的首次重大界面更新感到满意。能否从 Chrome 上诱惑用户，则是另一回事。

尽管 Opera 在过去的 12 个月里有了相当不错的增长，但 Safari 和 Opera 基本没有变化。也就是说，很大一部分人拒绝从 WebKit/Blink 之前的版本(12.x)迁移。

## 全球移动浏览器统计，2014 年 3 月至 4 月

经过几个月 1%的增长后，手机使用量略有下滑，占所有网络活动的 25.02%。如果你的老板/客户正在犹豫是否应该建立一个响应性网站，那仍然是四分之一的网络用户！

热门移动浏览应用:

| 移动浏览器 | 三月 | 四月 | 变化 | 亲戚 |
| 机器人 | 26.44% | 25.52% | -0.92% | -3.50% |
| 苹果手机 | 20.88% | 21.14% | +0.26% | +1.20% |
| 铬 | 11.95% | 15.62% | +3.67% | +30.70% |
| Opera Mini/Mobile | 13.70% | 12.26% | -1.44% | -10.50% |
| UC 浏览器 | 11.30% | 11.09% | -0.21% | -1.90% |
| 诺基亚浏览器 | 5.64% | 4.82% | -0.82% | -14.50% |
| 不变的 | 2.20% | 2.38% | +0.18% | +8.20% |
| 黑莓 | 2.22% | 1.81% | -0.41% | -18.50% |
| 其他人 | 5.67% | 5.36% | -0.31% | -5.50% |

移动版 Chrome 在一个月内又上涨了 3.7%。虽然较新的 Android 手机可以解释这种增长，但现有的移动用户也在转向谷歌的浏览器。不出所料，Android 浏览器的股价下跌了，但 Chrome 看起来很快就会在 iOS 上超过 Safari。

黑莓现在已经落后于相对不知名的 NetFront 移动浏览器，其市场份额为 1.94%。虽然仍有许多设备存在，但我怀疑是时候将黑莓从图表中移除了。从 web 开发的角度来看，这已经变得无关紧要了。
