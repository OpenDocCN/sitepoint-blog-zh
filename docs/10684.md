# 21 条市场调查建议

> 原文：<https://www.sitepoint.com/market-research-tips/>

在我的上一篇文章中，我概述了一些小企业主和自由职业者可以自己做市场调查的方法。虽然你完全有可能进行自己的研究并收集有意义的数据，但为了成功，你需要一个计划。

这篇文章提供了一些建议来确保你在市场调查上投入的时间、精力和金钱是值得的。

1.  在开始市场调查之前，要明确你的目标。
2.  在开始之前，确定你的目标受众、你需要多少受访者以及你希望收集哪些数据。
3.  确保你的目标群体与你的需求相关，并代表你的目标市场。
4.  留意并采纳你在社交网络或评论网站上收到的主动反馈。
5.  对每一个为你的研究做出贡献的人说声谢谢。
6.  为您使用的任何调查提供一个开放式评论的地方。
7.  让每次互动都向客户征求建议成为惯例。
8.  理解[定性和定量研究](http://www.socialresearchmethods.net/kb/qualdeb.php)之间的区别，并相应地安排您的问题。
9.  检查你的调查和问卷，确保你没有任何冒犯性的措辞。
10.  根据收集的数据制作图表或图形，以便更容易直观地分析结果。
11.  尽可能记录访谈或焦点小组，以供审查和分析。
12.  不要因为不想听批评就不去理会；这可能是你收到的最有价值的反馈。
13.  确保您的调查是中立的，不涉及诱导性问题。
14.  让完成您的调查或问卷变得简单快捷。
15.  以足够大的样本群为目标，给你有意义的数据。
16.  创建您的调查和采访，以便关注可测量的数据。
17.  尽可能通过电话或其他个人联系方式完成调查，以完善所提供的答案。
18.  在写调查问题时，要站在潜在客户的角度考虑问题。
19.  确保所有参与者都将保密，以鼓励参与。
20.  在整个过程中不偏不倚，或使用一个外人来协助。
21.  在进行研究之前，对你没有考虑到的新机会和新需求保持开放的心态。
