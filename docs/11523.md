> 随着 JavaScript 框架的激增和广泛采用，聪明的开发人员想知道是否有类似的方法可以解决 CSS 的问题。于是，像 Blueprint、YUI 库 CSS 工具、Boilerplate 等许多 CSS 框架诞生了。在本次会议中，我们将调查 CSS 框架的前景，并考虑每个框架如何处理创建通用的、可重用的 CSS 样式的独特挑战。

> 有许多不同的方法，其中一些比另一些更好。选择正确的框架，你可以省去很多工作。选择错误的，你会发现你的项目被限制性的假设和大量你不理解的代码所拖累。当谈到 CSS 框架时，做出正确的选择就是一切。在这个环节结束的时候，你可能会认为适合你的框架根本就不是框架。
