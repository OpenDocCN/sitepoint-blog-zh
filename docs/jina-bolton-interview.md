# 纪娜·波顿谈创建性感的样式表

> 原文：<https://www.sitepoint.com/jina-bolton-interview/>

纪娜·波顿是一名设计师和艺术家，在硅谷工作和居住。她已经为 SitePoint 撰写了几篇[文章](https://www.sitepoint.com/article/breaking-out-of-the-box)，是《CSS 的艺术&科学 *[的合著者，并将于本月晚些时候在悉尼的 Web Directions South conference 上发表。](https://www.sitepoint.com/books/cssdesign1/)*

SitePoint 的澳大利亚读者通过输入代码 WDS-SP 可以享受票价折扣。你可以在[这篇博文](https://www.sitepoint.com/learn-javascript-from-the-master-at-web-directions-south/)中读到更多细节。

在过去的几年里，你已经成为网络会议的常客，你将在本月晚些时候前往澳大利亚参加南方网络方向会议。是什么让你一直回来参加会议？

实际上，有很多事情。

我住在孟菲斯的时候就开始了。我一只手就能数出孟菲斯地区有多少人和我一样(关心 web 标准，真正拥抱 web 行业的本来面目)。我在 2006 年 11 月参加了我的第一次会议(Refresh 2006 Orlando)，并立即沉浸在一群我觉得“和我一样”的人中。这令人耳目一新(没有双关语)，鼓舞人心，激励人心。

虽然我很有能力自己阅读、研究和学习东西，但没有什么比得上有人给你解释——尤其是如果他们做得好的话。这就是我喜欢学校的原因。会议也可以类似(如果你选对的话)。

在几次会议之后，我开始被邀请去演讲。我不完全确定为什么(我仍然觉得自己像个 noob)，但是只要我能，我就会抓住这些机会！我不仅可以学到更多东西，和我尊敬的、不断激励我的人在一起，还可以和别人分享我学到的东西。

告诉我们更多关于卡拉 ok 痴迷的事情。你是那种有着惊人嗓音的人吗？一旦你开始唱歌，每个人都会竖起鼻子想“谁带她来的？”或者你只是一个普通的发烧友，喜欢成为聚光灯下的五分钟，不管你有多破坏一首歌？

我觉得是后者。我可能会非常害怕在别人面前演讲(尤其是在会议上)，但是当我表演的时候，就不一样了。我年轻的时候经常唱歌和表演。我不像以前那么好了(因为我身材走样，唱功不行)，但我确实喜欢假装成别人(这就是表演和卡拉 ok 的用武之地)。

我刚上大学时，离学生宿舍几个街区远的地方有一家酒吧，每周三晚上都开卡拉 ok。这么说吧，我没有注册周四上午的课程，这样我就可以每周都去了。是的，我每周都去。我甚至有了一个艺名，主要是为了好玩，但也是为了以后当他们接近我时，我不必告诉他们我的真名。

现在，我真的没有机会去卡拉 ok，所以当机会来临时，我会抓住它。尤其是如果基斯·罗宾逊、杰夫·克罗夫特和丹·鲁宾也参与进来的话！

你的个人网站叫做[寿司&机器人](http://sushiandrobots.com/)。对这些的兴趣从何而来？当你不在电脑前时，你还喜欢怎样打发时间？

总的来说，我喜欢亚洲食物，但寿司是我一直觉得非常有趣、健康(取决于你吃什么)和社交的食物；这个演示很有艺术性，也很吸引人。实际上，我在大学时只喜欢寿司，因为我的家人从来没有真正吃过寿司。我非常喜欢它，甚至在工作间隙的时候，我还在一家寿司店工作。(尽管我很喜欢寿司，但我很快发现自己不适合餐饮业。向我不小心洒了一整盘饮料的女士真诚道歉)。

我想我一直对机器人很着迷。我看过的电影，读过的故事……我甚至在大学上顺序叙事插画课时画了一幅以机器人为主角的漫画(类似于漫画小说)。然而最近的执念却是出于一次意外。在键盘上，r 紧挨着 T，我的几个朋友发现很容易不小心输入了 Boltron 而不是 Bolton。我和一个朋友开玩笑说，把我的网站重定向到 jinaboltron.com 会是一个有趣的愚人节玩笑，会有一些可笑的机器人(看起来像我)摧毁一座城市的动画。不知何故玻尔兹龙卡住了。有了新绰号，就有了新的困扰。

另外，谁不喜欢机器人呢？即使是邪恶的机器人也很酷。

你的寿司&机器人网站最近进行了一点翻新。你觉得这个过程怎么样？你是你工作过的最难伺候的客户吗？哦，我喜欢头版的插图——那是谁的作品？

我想我为新网站设计了大约 20 种不同的设计(不是一次完成，而是随着时间的推移，当我对以前的版本感到厌烦时)。在这些设计中，有 10 个实际上是内置在 HTML/CSS 模板中的(同样，随着时间的推移，从一次又一次开始)。其中 5 个实际上被接入了内容管理系统。现在，这些数字并不准确，但它们非常接近。问问我的朋友斯蒂芬·雷尼(他是 [Clear Function](http://www.clearfunction.com/) 的创始人，他构建了 [Reflect](http://reflectyoursite.com) ，我现在正在使用的 CMS)。他看到了一些迭代。

实际上，我从 2006 年开始。直到刚才我才意识到。所以我花了两年时间才到达现在的位置。老实说，我并不完全满意。但我需要现场直播。所以，是的，我是我工作过的最难的客户。我总觉得还可以更好。

这幅插图是天才插画家吉姆·马赫福迪的精美作品。我在 2001 年 9 月 11 日(是的，那个悲伤的日子)遇见了他，在那一天，他在我就读的艺术学院做了一场客座艺术家的讲座。不幸的是，他被困在孟菲斯一周，因为那天早上发生的悲剧导致机场关闭。但对我们来说幸运的是，这意味着我们可以和他一起出去玩。他是个很棒的人，我强烈推荐他。

你的网络方向南方会议叫做[创造性感的样式表](http://south08.webdirections.org/?page_id=7#post-59)。是什么让样式表变得性感？

我觉得有些人曲解了我说性感样式表的意思；我指的不是一个设计性感的网站，而是一个设计和开发得智能、高效(考虑到当前和未来的发展)和可维护的样式表。我发现这本身就是一种艺术形式。当我做这些演讲或写这些文章时，我也是在为自己做。我不认为我的样式表总是“性感的”(特别是当健忘或临近截止日期时)，但我希望它们是。这些是我为之奋斗的东西，所以我与他人分享，以防他们也需要为之奋斗的东西。

作为一名成功地将标准世界和美丽、原创的界面设计世界联系起来的设计师，你已经有点名气了。能告诉我你是怎么找到灵感的吗？你在头脑风暴的时候会使用像 [Flickr](http://flickr.com/) 这样的在线工具或者听音乐吗？

我在很多地方都能找到我的灵感。我真的很喜欢现代简约的设计，我喜欢中性色。但是，我也喜欢带有泥土色彩的纹饰和图案。如果我能使用所有这些找到合适的混合，我很高兴。我在自己的公寓里也这样做。

我总是在工作的时候听音乐。我把独立音乐、trip hop/downtompo、地下 hip hop、老派爵士乐和电子乐混合在一起。我不怎么用 Flickr 来头脑风暴。不过，我看过画廊的网站。我最近最喜欢的是 minimalsites.com。

你认为谁对你有影响？你目前在看什么博客？你最近读过哪些给你留下深刻印象的书？

说到网页视觉和交互设计，我最崇拜的两位设计师是丹·塞德霍尔姆和瑞安·西姆斯。他们太棒了。我现在正在阅读的一些博客有 [NOTCOT](http://www.notcot.org/) 、[公寓疗法](http://www.apartmenttherapy.com/)、 [The Big Noob](http://thebignoob.com/) 、[The ampander](http://ampersand.gosedesign.net/)、[我喜欢排版](http://ilovetypography.com/)、 [Monoscope](http://www.monoscope.com/) 、[Rands In resting](http://www.randsinrepose.com/)、 [Treehugger](http://www.treehugger.com/) 等等。我最近读的一本书给我留下了最深刻的印象:马克·z·丹尼尔夫斯基的《叶之屋》;排版和讲故事真的启发了我。

当你第一次充实一个设计时，有没有一个严格的流程可以遵循？你还在用笔和纸，画缩略图，还是直接跳到 Photoshop/Illustrator？

一切取决于项目。如果我有时间真正探索，那么我会画出我的设计草图。然而，如果我在一个快速周转的截止日期，那么我必须马上投入。我觉得我的流程不死板；我的过程随着我了解的更多或者项目的要求而发展。

在过去设计网站时，我个人一直在努力的一个方面是创建一个漂亮、新颖但又熟悉的界面。你对用户界面惯例有什么看法？

我认为惯例是有原因的。而且关注他们，学习他们，使用他们也是很好的。然而，这并不意味着它必须完全阻止我们创新，只要创新不会引起交互问题。

在去年的 Web Directions South 上，[安迪·克拉克推出了 CSS Eleven](https://www.sitepoint.com/wds07-think-like-a-mountain/) ，这是一个致力于在 W3C 失败的领域塑造 CSS 未来的设计师团队。你能告诉我们更多关于这方面的进展和计划吗？

真的一年了吗？哇哦。我们已经和 Fantasai 联系过了，他会带着一些问题来找我们，我们会通过电子邮件来回答。我们还在开发一个新的网站来取代已经存在很长时间的有争议的页面。然而，这比我们预期的时间要长一点，因为安迪招募了有才华的人，但这些人有真正的工作，让他们很忙。因为等待，我们实际上有一个最近的改组，但我们仍然要去。希望安迪看到这一点，并以自己的想法回应…

最后一个问题:你将和两个人被困在一个岛上 30 天。你带谁去，为什么？

30 天没那么长！我可能会带上我爸和我哥。我爸爸真的需要休假，他曾经在部队服役，所以他也许能教我们一些生存技巧——也许吧。

我会带上我哥哥，因为，为什么不带上他？这可能不是你所希望的，是吗？

哈哈，没关系！谢谢你的时间，纪娜。我期待着很快在悉尼赶上。

谢谢，你也是！
