# 为什么你不一定得到你所付出的，以及如何把这变成你的利益

> 原文：<https://www.sitepoint.com/why-you-dont-necessarily-get-what-you-pay-for-and-how-to-turn-this-to-your-benefit/>

你们中的很多人在这个博客上发帖说，“你得到了你所付出的。”这些帖子是对我与你分享的寻找低价可靠的网页设计/开发服务的考验和磨难的回应。

这篇文章解释了为什么你不一定得到你所付出的，以及作为一个网络专业人士，你如何把它变成你的优势。

想象一个图表，质量横跨 X 轴，价格横跨 Y 轴。让质量意味着它对你的客户意味着什么:结果，速度，创造力，更多的业务，更好的形象…

现在想象一下，你用每个 web 设计/开发专业人员的价格/质量数据点填充这个图表。

如果你在这个行业中得到你所付出的，那么形成的线应该是狭窄的。

但根据经验，我知道事实并非如此。“线”是一个非常宽的散点图。有些人相对于他们的质量要价太高(并得到他们的价格)。有些人按质量收费。有些人收费很低。

当我购买网页设计和/或开发服务时，我的目标是找到不按质量收费的专业人士。

你们中有很多人把自己放在了那个类别中。我称你们为“低定价自尊人群”当我找到这些人时，我很喜欢，因为我以很低的价格获得了很好的服务和专业精神。

这对你有什么影响？好吧，如果你属于“低定价自尊人群”，你需要更好地为你的服务定价，传达你的价值，并说服你的潜在客户，他们会疯了才会付给别人更少的钱。

是的，一般来说，定价和质量是有关系的。但这不是一段完美的关系。

你卖的是看不见的东西，而不是有一套功能的汽车。难道你不想在价格/质量线上或之上吗？
