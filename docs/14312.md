# 为移动设备开发(开放和专有)

> 原文：<https://www.sitepoint.com/developing-for-mobile-devices-open-and-proprietary/>

使用精简的页面为移动设备开发看起来很简单。然而，随着手持设备的复杂性和用户群的增长，对更复杂的 web 内容甚至 web 应用程序的需求正落在 web 开发者的肩上。

当我自己探索这个新领域时，我是从理解最基本的网络内容如何在这些不同的设备上执行和出现的基础开始的。当然，我也有自己的 PDA，不过，众所周知，随着智能手机的出现，有一个广阔的设备宇宙——而且，平台范围从 Palm 到 Windows Mobile 到 Linux 和其他专有系统(想想黑莓)。

随着这些嵌入式操作系统的成熟，新的一年也将带来新的复杂性。例如，Blackberry Enterprise server (4.0)现在将与 Pocket PC 设备交互，并支持 XML 表单来收集和发送数据到手持设备以及后端系统。此外，Windows Mobile 2003 为智能手机以及运行操作系统的传统 PDA 带来了新的和改进的 VPN 支持。因此，内联网和外联网应用程序可能会变得可用，可以通过这些设备进行交互的用户的需求可能会增加。

当移动用户离开笔记本电脑而使用手持设备时，界面需求会发生巨大变化，然而，web 应用程序和数据库仍然需要获取数据，业务也会继续进行。

首先，我们不要讨论代码优化、替代协议或语言(这当然很重要，我们将在后面的文章和专栏中讨论)——让我们确定所有的移动设备仿真器和开发人员资源，以便设置您的工作站，在这个不断增长的前沿领域中进行构建。

不分先后，按平台:

黑莓

黑莓平台是专有的，但却是企业界最受欢迎的平台之一。这种流行的核心是 Lotus Domino 和 Microsoft Exchange 服务器的连接器。对开发者来说更重要的是，可以开发离线表单，允许黑莓用户提交数据，这些数据最终可以进入核心 web 应用程序和支持商业网站的 sql 数据库。

该平台有多种开发工具，包括:

*   桌面模拟器
*   Java 开发环境
*   API 访问文档

这些都可以在网上找到。

此外，还有许多第三方开发工具和丰富的图形工具集。

开源移动设备

运行 Linux 的移动设备开源社区正在蓬勃发展。移动业务中一些最大的公司支持这一运动。

一个主要的开源手持平台是 Symbian，这是一家由 Psion、诺基亚、索尼爱立信和摩托罗拉等公司共同拥有的公司，拥有许多开发者资源。这里可以找到一些 Symbian 设备。

在 Linux 设备上运行 Linux 的手持设备和智能手机[还有一个相当广泛的列表。基于 Linux 设备的现有应用程序的一个很好的资源可以在这里找到](http://www.linuxdevices.com/articles/AT9423084269.html)。

掌上设备

可能是最著名的手持设备，在最近的这个季度，它的销量第一次被 PocketPC 超越。然而，它仍然是皮带和公文包中的核心和主要配件。

[掌上模拟器](http://www.palmos.com/dev/tools/emulator/)和优秀的[开发者资源](http://www.palmsource.com/developers/)都可以找到。

Pocket PC 和 Windows Mobile

在任何软件类别中，微软都是 800 磅的大猩猩，它拥有丰富的移动平台开发者网络。这包括 Windows Mobile [PDA 和智能手机模拟器](http://msdn.microsoft.com/mobility/windowsmobile/default.aspx)，以及扩展的[工具包、SDK 和文档](http://www.microsoft.com/windowsmobile/developers/default.mspx)。

开源也在很多方面找到了通往封闭平台的道路。除了上面提到的基于 Linux 的开源解决方案之外，还有几个针对专有硬件和操作系统的开源软件的销售渠道——这只是其中的两个:

*   [Palm 开源](http://www.palmopensource.com/)
*   [Pocket PC 开源](http://ppc.palmopensource.com/)
*   或者在我最喜欢的资源之一上搜索“移动”或“PDA”——[Sourceforge](http://www.sourceforge.net)

虽然许多 web 开发人员并不急于为正在探索移动应用的客户开发传统软件，但 web 应用程序确实要求更多。随着业务越来越多地跨越国界，用户/客户/合作伙伴需要访问安全的 web 解决方案，为移动领域开发无疑是另一个创收渠道。
