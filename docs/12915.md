# 2013 年 ie 浏览器会灭绝吗？

> 原文:[https://www . site point . com/internet-explorer-extended-by-2013/](https://www.sitepoint.com/internet-explorer-extinct-by-2013/)

微软应该将此视为一个预警——如果过去三年的趋势继续下去，到 2013 年,没有一个访问 sitepoint.com 的人会使用 Internet Explorer。

随着关于新的火狐浏览器和 T2 操作系统的讨论越来越多，我想我应该重新认识一下 SitePoint 的访问者所选择的浏览器。

所以我回到三年前，也就是 2005 年，继续向前，比较我们的读者这些年来使用的浏览器。结果令人吃惊，所以我想我应该分享一下(点击下面的图片看大图):

[![A trend line showing IE decreasing rapidly, and other browsers increasing equally as rapidly](../Images/4db609b65bd27646d154bc5156d4f916.png)T2】](https://i2.sitepoint.com/images/blogs/mattymcg/ie-vs-everyone-else_large.png)

上图显示了 IE 用户相对于其他浏览器的百分比。使用一个简单的线性方程，我们可以推断出每天在 SitePoint 上使用 IE 的人减少了 0.2%。

如果这种趋势持续下去，那么展望 1797 天(或 4.9 年)，IE 使用率将降至零！

现在，我们谈论的是三年的历史，数以百万计的访问者——他们的生活围绕着网络浏览器。是的，微软最近重新将精力集中在开发他们的网络浏览器上。然而，这种趋势表明，这种方法不起作用。

网络行业的专业人士以这种速度流失，如果我是微软，我会害怕…非常害怕。
