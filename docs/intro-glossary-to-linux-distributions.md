# Linux 发行版术语介绍

> 原文：<https://www.sitepoint.com/intro-glossary-to-linux-distributions/>

马丁·法拉帝白天是股票经纪人，晚上是网络迷，他推出了一本不错的入门指南，介绍主要和一些次要的 Linux 发行版。更不用说他的网站还广泛地作为 web 专业人员可用的许多其他工具的资源。

这是我自己作为一名研究人员经常渴望的事情，它鼓励专业人员在开源上收集更多的数据，用于技术规划和支出。法拉帝提供了各种操作系统风格的入门词汇表。

合乎逻辑的下一步是更深入地研究每个发行版的优缺点，以及哪些发行版可能针对特定的技术应用领域(web 服务、桌面、文件服务器等等)。

举个例子，我最初只使用 RedHat，真的没有探索 Linux 的其他风格(1997-1999)。然而，在构建了各种发行版之后，我发现 RedHat Enterprise 是文件服务、邮件服务和存储的理想选择，Mandriva (Mandrake)是一个很好的桌面模型，Fedora 和 SUSE 是作为服务器构建的——仅适用于 web 服务，具有简单高效的修补模型。

不要忽视忠诚者的最爱——我确实喜欢 Debian 和 Slackware，因为这是我最初接触 Linux 的方式。然而，由于赔偿和支持合同的原因，在企业中与 Linux 一起工作会默认地将一个人推向 RedHat 和现在的 Novell。
