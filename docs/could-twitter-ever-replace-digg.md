# Twitter 有可能取代 Digg 吗？

> 原文：<https://www.sitepoint.com/could-twitter-ever-replace-digg/>

Hitwise 研究机构 Heather Dougherty [昨天](http://weblogs.hitwise.com/heather-dougherty/2009/01/twitter_catches_up_to_digg.html)报道称， [Twitter](http://www.twitter.com/) 有史以来第一次超过了 [Digg](http://www.digg.com/) 。这份报告相当可疑，受到了一些顶级博客作者的质疑。

根据其他衡量标准——比如 Compete 和 Quantcast——Twitter 的流量远远不及 Digg。是的，Twitter 最近受到了很多主流的关注，因为它出现在了美国 CNN 的一个下午电视节目中，并且成为了上周纽约市飞机坠毁的照片首次登陆的地方(可以接受这个糟糕的双关语，因为没有人受到严重伤害！).然而，Hitwise 的数据涵盖了 MacWorld/CES 和周二美国新总统巴拉克奥巴马(Barack Obama)就职典礼之间的一周，这导致 Twitter 上的活跃度飙升 500% ，但不是就职典礼本身，也不包括 Twitter API 上的流量——该 API 的活跃度高于 Twitter.com。

一周没有重大(计划的)事件会导致 Twitter.com 的流量超过 Digg 的流量，这似乎令人难以置信，正如我的前同事 Marshall Kirkpatrick 在读写网所说，“这是一个 Hitwise 问题，而不是在线格局的改变。”然而，这确实提出了一个有趣的问题:在链接发现方面，Twitter 有可能取代 Digg 吗？

我们 SitePoint 非常重视 Twitter。我们的 [Twitter 账户](http://twitter.com/sitepointdotcom)拥有超过 20，000 名粉丝(我们希望你能成为其中一员！)，而且因为我们在推特上发布每一篇新文章，推特已经成为我们一个相当大的流量来源。事实上，你很可能是通过 Twitter 上的链接看到这篇文章的。它绝对是比 Digg 更稳定的流量来源，但即使我们的粉丝数量远远超过平均水平，Twitter 也从未在为一篇文章发送流量方面接近 Digg——这可能是对 Hitwise 报告的又一次打击。

然而，即使 Twitter 可能对链接推广很有帮助，只要你有相当多的追随者，它在链接发现方面真的做得很好吗？

之前识别 Twitter 上热门链接的尝试大多失败了。例如，像 [Tweetmeme](http://www.tweetmeme.com/) 这样的网站往往无处不在，从来没有很好地给我指出我经常感兴趣的链接的方向。

但是我认为 Twitter 实际上*可能*是一种有用的链接发现方法。关键是要认识到 Twitter 的力量不在于它能指出人们喜欢什么(正如 Tweetmeme 及其同类试图做的那样)，而在于它能指出你的*朋友*喜欢什么。

如果有一个 Twitter 工具可以查看我的朋友图表，然后找出我朋友在 Twitter 上发布的所有链接，并按照受欢迎程度(他们发布了多少次，对我朋友的权重)和转发次数(我朋友的朋友中有多少人认为这是一个值得的链接)进行排列，它会显示一种类似 Digg 的图片，显示我可能已经觉得有趣的人。做这些的工具已经存在了，但是据我所知，没有一个工具可以通过把它链接到你的朋友图来把它们联系在一起。

这种服务可能还会根据你的朋友发推和转发的链接类型以及你回复的链接来建议新的人关注(也许它可以有一个“转发”按钮，自动转发你最喜欢的链接)。

对于我最初的问题:Twitter 会取代 Digg 吗？广义来说，不，很可能不是。但是，它是否有可能在每个用户的基础上展示更多有用和有趣的链接呢？是的，那绝对是一种可能性。希望有人会像我上面描述的那样建造一些东西，这样我们就能找到答案。
