# SXSWi:伟大的 Spec 工作辩论

> 原文：<https://www.sitepoint.com/sxswi-the-great-spec-work-debate/>

在过去的五六年里，任何一个在 SitePoint 工作的人可能都很熟悉关于平面设计领域中特殊工作的优点的争论——平面设计师所做的工作没有报酬保证。由于衍生网站 [99designs](http://99designs.com/) ，SitePoint 正处于规范工作辩论的中心，这是网络上最大的平面设计竞赛市场，也是直言不讳的“无规范”社区的经常目标。今天早上在德克萨斯州奥斯汀的西南偏南互动节上，在一个名为“特殊工作是邪恶的吗?”的小组会议上，争论达到了顶点。网络创意社区发言。”

不幸的是，我错过了这个小组——相反，我参加了一个电影节小组(电影是一个行业，其中最受欢迎的进入途径之一是通过规范工作)。我通过推特(Twitter)和丽贝卡·卡罗(Rebecca Caroe)的博客(T3)跟踪了这次会议，在推特上，辩论非常激烈。虽然我没有参加小组讨论，但我想在这里分享一下我对 spec 工作的一些看法。

在我开始之前，有一些免责声明。首先，我不再受雇于 SitePoint，也没有因为这个职位获得报酬。第二，我曾经与 99designs 共同拥有一个平面设计竞赛网站(实际上，这是在 99designs 从 SitePoint 的市场中分离出来之前)，但我两年前卖掉了那个网站。第三，我从 2003 年开始就在 SitePoint 工作，首先是作为论坛的成员和版主，然后是一名员工，因此我参与了图形设计竞赛概念的推动，并亲眼目睹了它的发展。

反对规范工作的论点通常是这样的:规范工作不专业，导致低质量的结果，糟糕的(或不存在的)客户/设计师关系，并贬低整个过程。在[no-spec.com](http://www.no-spec.com/articles/ten-reasons/)有一个很好的总体观点。从理论上讲，这些观点很有道理，但实际上，我并不认为它会发挥作用。

Spec 工作可能以业余设计师为主(虽然我不太确定)，但是天赋水平很高。快速浏览一下目前在 99designs 进行的竞赛，可以发现参赛作品从差到好到好。当然，由于设计竞赛是开放的，任何人都可以参加，所以有时信噪比很高，但肯定有高质量、非常专业的工作在进行。

此外，我知道许多有才华的设计师，他们中没有一个人在积极推广他们的可用性，这是对工作的伤害。事实上，我认识的许多设计师工作量都超出了他们的承受能力，他们不得不拒绝一些项目——即使是在当前的经济形势下。我认识的人也没有人需要降低他们的费率来与 99designs 这样的网站竞争——我的一些朋友在过去的一年里随着工作量的增加，实际上已经提高了他们的费率。因此，如果设计师参与基于规范的设计竞赛或做其他类型的规范工作没有伤害那些没有伤害的人，牛肉在哪里？市场显然足够大，可以支持这两种方式。

也就是说，我知道很多参加竞赛网站的设计师向我报告了以下内容:

*   规格竞赛为新的或业余的设计师提供了一个获得为真实客户工作经验的途径。按照客户的规格进行设计不是你在学校能学到的，从头开始写一份简历也很难。竞赛网站为设计师提供了一个在真实项目中工作的地方，并接收建设性的客户反馈，使他们能够做得更好。在这方面，特殊工作和没有报酬的实习有那么大的不同吗？不规范倡导者也建议不要接受无薪实习吗？我很想知道答案。
*   我认识的许多设计师都与通过设计竞赛找到他们的付费客户建立了长期关系。一些客户将竞赛视为一次试演——当他们找到一个他们喜欢的设计作品，并能与之合作愉快时，他们会通过该设计师支付竞赛领域之外的工作。我甚至认识一些设计师，他们的客户在看到他们喜欢的比赛中有一个参赛作品*落选*后会去找他们。
*   虽然事实上我认识的设计师中没有一个是工作紧张的，但有时也会有不景气的时候。我认识的一些人利用比赛来填补罕见的低迷时期的空白，以此来保持敏锐，结识潜在客户，并可能赚些钱。
*   我交谈过的其他设计师将设计竞赛作为一种通过做一些新的和不同的事情来获得灵感的方式。也就是说，对于在公司环境中工作的设计师来说，他们可能会在公司设计项目上花费太多时间，规范工作是做一些不同的事情并获得灵感的一种方式；一种中断他们日常工作的方式。

就像几乎所有的争论一样，这个问题不是非黑即白的。规格工作并不适合所有的设计师和所有的情况——我想没有人会否认这一点。然而，市场是存在的，并且显然为消费者和创造者服务。

许多使用设计竞赛网站或其他专业工作市场的消费者要么是设计过程的新手，因此不具备仅凭其作品来判断设计师的必要技能，要么他们没有传统设计商店或高价知名设计师的预算。如果是前者，客户可能会在经历了相当严格和耗时的评估过程，并留下数十或数百个条目的反馈后，发展这些技能。下一次，客户可能会选择直接雇佣设计师(考虑到一个人的时间成本，对一些人来说，这可能是一个更便宜的选择)。如果是后者，那就真的不是流失客户了。预算限制是绝对的，所以客户不会选择像 99designs 这样的网站，他或她会选择 99designs，因为这是唯一可行的选择。

设计竞赛网站不是平面设计的未来。永远不会有一天所有的设计工作都是通过竞赛来完成的——事实上，我敢打赌绝大多数的设计工作都将通过传统的方式来完成。在很多情况下，设计竞赛并不是正确的方式。也许你心中已经有了完美的设计师，也许你没有时间去举办比赛，也许你需要在设计过程中更加谨慎等等。然而，设计竞赛和其他形式的规范工作肯定是设计未来的一部分，在我看来，这真的不是一件坏事。
