# 情境意识如何帮助你避免灾难

> 原文：<https://www.sitepoint.com/situational-awareness/>

情境意识是你对周围环境的感知和理解。虽然态势感知在军事行动、医疗紧急情况和日常危险环境等高风险情况下至关重要，但在做出正确决策时也很重要，即使您需要做出的决策没有生命危险。

如果你不能完全理解影响你决策的环境因素，并把这些知识转化为对后果的理解，你只能看到事情的一半。让我们用一个简单的例子来说明这一点:

您收到了一封来自托管公司的电子邮件，要求您重新提交您的信用卡信息。你以前在这家公司做过这种事，所以你很快点击并输入你的信用卡信息。

因为你很匆忙，因为你对这家公司感觉很舒服，所以你没有注意到你实际上没有通过安全连接提交你的信息。事实上，你甚至没有把你的信用卡号码提交给实际的托管公司。点击会把你带到一个看起来像托管公司的诈骗网站，但它不是。您已经正式成为网络钓鱼骗局的受害者。

不太好，对吧？情境意识可以在几个方面帮助你避免这种错误。

## 你被迫放弃假设，拒绝自满。

生活教会我们做出假设，以节省时间，减少达到目的所需的产出。但是当这种情况发生时，我们变得自满，容易忽略明显的事实并犯错误。有情境意识会让你从头开始每一个过程，了解环境的本来面目，而不是你所期望的样子。

## 这不仅仅是“看”，而是观察。

情境意识意味着你不仅能看到你周围的事物，还能思考它们，并试图理解它们与你需要做出的决定以及将会带来的后果之间的关系。你需要观察和处理，以便弄清楚你所处的环境，以及你的决定如何适应这个环境。

## 没有捷径可走。

对我们中的许多人来说，对情况的了解可以归结为抛弃我们为了节省时间而创造的变通方法。想想上面的例子。在这种情况下，如果你多花一两秒钟检查邮件的标题，看看邮件是从哪里发出的，核实一下点击链接的网址，或者手动输入托管公司的域名，而不是点击，你就可以完全避免这个问题。

## 但是……情境意识也可能是危险的。

这里的危险是，情境意识会让你变得高度警惕，以至于你花太多时间来确保你在做正确的事情，然后对你做出的每个决定进行事后批评。如果与现实和优先级不一致，情境意识可能会完全出错。你需要能够选择最高优先级——最高“风险”的情况——然后有节制地应用你的情景意识过程。

## 轮到你了。

你对情境意识有什么看法？你认为更多地意识到你的环境会帮助你避免错误吗，或者你认为你采取这种观点是不现实的吗？
