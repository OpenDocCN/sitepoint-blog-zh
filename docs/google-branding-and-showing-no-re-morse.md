# 谷歌:品牌和显示没有再莫尔斯

> 原文：<https://www.sitepoint.com/google-branding-and-showing-no-re-morse/>

有趣的是，谷歌一如既往地标榜营销的黄金法则之一:不要摆弄你的品牌。

为了纪念塞缪尔·莫尔斯的生日(他以莫尔斯电码闻名)，昨天“谷歌”把他们无处不在的彩虹标志换成了莫尔斯电码。

虽然他们多年来一直在玩弄自己的核心品牌——谁能数清所有的熊、雪、运动员和不断变化的字体？——这是我第一次想起他们最核心的互联网身份上没有可读的标识。

我得承认我对此有点犹豫不决。

我的一部分认为很棒，他们打破规则，玩得很开心，做一些疯狂的事情。给他们更多的力量！”。各地的谷歌粉丝都会咧着嘴笑，说‘哎呀！“成为一个小小的‘粉丝’。

但是另一个极端的人呢。当你的奶奶的浏览器打开时，她的主页读起来不再正确，她会怎么想？或者你六岁的侄女？或者你 45 岁的机械师，他多年来一直避免使用电脑，直到现在？他们可能打碎了什么东西吗？他们应该重启吗？他们应该给他们的儿子/朋友/常驻 IT 极客朋友打电话吗？

好的，当然，他们可能会解决的。毕竟，按钮上写着“谷歌搜索”。但是如果他们经历了哪怕 5-10 秒的轻微困惑，这是一个好的交换吗？把莫尔斯字母直接复制到它们的字母表下面会不会效果差很多？我不太确定。

当然，谷歌是休息室里的巨型大猩猩。最终他们可以做他们喜欢的事情。

然而，作为一家以严格的用户测试而闻名的公司——最近道格·鲍曼(Doug Bowman)和他们的 41 种蓝色调测试对此进行了充分的记录——了解他们是否会尝试测试用户对他们的徽标恶作剧的反应将会很有趣。

实际上，早在 2006 年，他们似乎已经对盲文做了类似的事情
