# 10 个有用的 jQuery 扩展

> 原文：<https://www.sitepoint.com/10-jquery-extensions/>

我一直在搜索 jQuery 扩展，我偶然发现不同的网站都有很棒的 jQuery 扩展。我收集了一些反馈和评论都很好的，请欣赏！

## 1.装雾机

这是一个基于 jQuery UI 主题的 Ajax 加载器对话框。它允许您从 jQuery UI 库中扩展现有的对话框，用作模式 ajax 加载器，甚至是进度条。你也可以使用几种风格或选项来传达加载信息。
*来源:*
【http://plugins.jquery.com/project/fogLoader】T4

## 2.正则表达式选择器

在这个 jQuery 扩展的帮助下。现在，您可以选择任何一个或多个标签的内容，而不管标签的文本节点中的文本或文本框的值是否与正则表达式中的模式匹配。
*来源:*
【http://examplet.buss.hk/jquery/selector.regex.php】T4

## 3.JMC _ 雷西兹尔

以一种很好的方式调整图像大小，使其适合另一个元素。
*来源:*
【http://github.com/jufemaiz/jmc_resizr】T4

## 4.AJAX-缩放

这是一个用 jQuery 和 PHP 开发的强大的图片缩放和平移插件。它有 300 个选项，可以很容易地与任何网站或 CMS 集成，甚至像 Magento 这样的网店。

它使用图像块快速生成图像的一部分，并在图像缩放后用水印系统保护它们。
*来源:*
【http://www.ajax-zoom.com/】T4

## 5.受亚马逊启发的图像和文本滚动器

像亚马逊一样设计的图片和内容 jQuery scroller。
*来源:*
[http://www . html drive . net/items/show/244/Amazon-style-image-and-title-scroller-with-jquery . html](http://www.htmldrive.net/items/show/244/Amazon-style-image-and-title-scroller-with-jQuery.html)

## 6.全屏背景/幻灯片

调整图像大小以占据整个浏览器，同时保持主纵横比。它还可以通过带有过渡和预加载器的幻灯片来循环播放图像或背景。它也有导航控制，允许你暂停或播放，前进或后退。
*来源:*
[http://plugins . jquery . com/project/全屏-背景-幻灯片](http://plugins.jquery.com/project/Full-Screen-Background-Slideshow)

## 7.$.回应者

这个插件用于日常处理 AJAX 响应的任何类型的内容。
*来源:*
【http://github.com/brianjlandau/jquery-respond_to】T4

## 8.AJAXFILTER

这个小插件为过滤器的不同功能提供了一个注册表来清理 ajax 响应。
*来源:*
【http://plugins.jquery.com/project/AjaxFilter】T4

哇滑块是一个图像滑块，提供了惊人的视觉效果和花式模板。
