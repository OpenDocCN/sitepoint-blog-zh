# 真正的商业成功:给予的不仅仅是感谢

> 原文：<https://www.sitepoint.com/true-business-success-giving-more-than-just-thanks/>

本周，我们的许多美国读者与他们的家人和朋友一起庆祝感恩节。感恩节是一个欣赏和感谢我们所拥有的一切的时候，当然也是一个尽情享用必不可少的火鸡和填料晚餐的时候。作为网络专业人士，这些日子我们当然有很多要感谢的。互联网繁荣的过山车已经远离我们，我们的行业在全球范围内继续稳步增长，定义我们业务的技术和标准继续快速发展。许多 web 开发人员发现他们可以在家工作，自己设定时间，赚到比他们想象的更多的钱。事情从来都不容易，但生活是美好的！

对我来说，我最感激的事情是我的公司在 2006 年每个月都能向慈善机构进行大量的捐赠。赚取健康的收入当然是值得感激的事情，但是赚到足够的钱来捐钱(和服务)给那些需要的人就更好了！捐款数额并不大，但有总比没有好，我打算随着公司的发展增加捐款数额。

在 SitePoint 上，我们花了很多时间讨论如何创造销售、收入和利润。我们都希望我们的企业发展壮大并取得成功，而赚大钱是这个目标的一部分。然而，重要的是不要忘记真正的目标，因为它们关系到我们的生活，而不仅仅是我们的钱包。像你们所有人一样，我努力工作来建立自己的事业，并为此感到自豪。但是金钱本身并不等同于成功——因为金钱本身并不能保证给我们带来真正的成功，所以有很多富人感到不满足和痛苦。

我希望你们每个人都花时间考虑一下，除了钱，你们真正想要的，最终想要的是什么。比如增加与家人在一起的时间，在工作中发挥创造力，为他人提供工作或培训的机会，旅行，提高行业标准，当然还有将部分利润捐给他人。我们成功的企业可以帮助我们的孩子、父母和社区。我们可以帮助推动我们当地的经济，并以积极的方式参与全球经济。

当你在做生意时，捐赠的可能性会大大增加。企业通常是慈善机构的最大赞助者，如国际关怀组织和 T2 红十字会。T4 比尔和梅林达盖茨基金会在世界范围内分发价值数十亿美元的服务。仅今年一年，美国航空公司就向联合国儿童基金会捐赠了 6000 万美元。

你能想象自己在商业上如此成功，以至于可以舒舒服服地给慈善机构开一张 5 万美元的支票吗？那一定是一种怎样的感觉！作为小企业和专业人士，我们可以给世界带来不可思议的改变。因此，我希望我们都可以从每天的赚钱努力中休息一下，考虑一下当我们成功时我们真正会做什么。

在接下来的一篇文章中，我将提供一些想法，关于我们可以向慈善机构捐赠金钱、时间或服务的各种方式，同时保持健康的增长和利润。与此同时，我很想听听你们都感激的一些事情，以及你们想用你们的财富做的一些好事~~如果~~当你们成功了！
