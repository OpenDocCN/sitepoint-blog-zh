# 如何像专家一样走路、说话和行动

> 原文:[https://www.sitepoint.com/how-to-become-an-expert/](https://www.sitepoint.com/how-to-become-an-expert/)

每个行业都有被贴上专家标签的专业人士。他们经验丰富，通常很受尊重，有能力理解他们所做工作的各个方面，并经常伸出援手帮助他人。我们向他们寻求指导，从他们那里获得新的业务发展想法，并帮助我们在解决问题时超越表面现象。

您可以通过以下关键特征来识别专家:

*   他们经验丰富，对自己的工作有全面的了解。
*   他们相信自己有能力取得成功。
*   他们有权威，被同龄人听。
*   他们受到同事、客户甚至业外人士的信任和尊重。

专家很受欢迎，这导致更多的工作和收入的增加。所以可以理解为什么成为专家是许多专业人士努力的目标。

虽然不是每个人都是专家，但是你可以采取一些行动来让自己成为专家。你准备好让自己加入精英名单了吗？以下是你可以开始的方法。

## 完善你的知识

你有多年的经验，利用这些经验来建立专家声誉的最好方法之一就是继续学习和发展你的技能。与其表面上提供你的经验，不如超越基础，真正了解你所在行业的来龙去脉。试着去上课，实施新技术，向业内其他先驱学习。

## 分享你的经验

专家不会囤积他们的知识或者仅仅为了自己的利益而使用它们。为了被认为是专家，你必须愿意与业内其他人分享你所知道的。你可以通过博客来传授你的洞察力，指导一个后起之秀，并提供其他信息来帮助别人做得更好。

## 制定你自己的规则

专家不会走前人走过的路；他们愿意为行业的发展促成变革。要被认为是专家，就要放下先入之见，探索新的做事方法。一旦你找到了更好的选择，就把它教给其他人，这样他们会变得更有效率和成功。

## 关注人际关系

除非你的同行认为你是专家，否则你不可能成为专家，所以人际关系很重要。当专家平易近人、讨人喜欢、受人尊敬时，他们最成功。不要成为一个“专家岛”，而是建立一个可以从你的经历中受益并促进共同成功的社区。你的人际关系和你赢得的尊重会让你飞黄腾达。

被贴上行业专家的标签是一种荣誉，不应该被认为是理所当然的。尽你最大的努力保持谦逊的态度，你将在未来的岁月里受益。
