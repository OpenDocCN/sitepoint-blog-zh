# 为什么苹果的道德审查注定要失败

> 原文：<https://www.sitepoint.com/why-apples-moral-censorship-is-doomed/>

言论自由运动者抱怨苹果在审查内容。该公司的立场导致讽刺作品和漫画版詹姆斯·乔伊斯的《尤利西斯》被禁，该书以手绘裸体为特色。(顺便提一下，原著在美国和英国被认为是淫秽的，直到 20 世纪 30 年代法院最终接受了这部作品。)

毫无疑问，苹果正在创造辉煌的产品。他们是一家成功的企业，正在保护他们的商业利益。如果你想在 iPhone 上安装应用程序或音乐，你必须从苹果公司购买。这是一个很棒的模式，很少有人能在商界复制。不喜欢就不*需要*买 iPhone。

不幸的是，这种务实的商业理念与苹果拥抱树木、嬉皮士友好的形象相冲突。史蒂夫·乔布斯不能说他是在*保护商业利益，因为他不希望苹果被视为另一个乏味的 IT 供应商，就像 IBM 或微软一样。相反，苹果已经开始了一场道德运动。他们是*“保护客户”*免受不良应用程序、Flash、攻击性内容和色情内容的侵害。*

 *(当然，如果他们真的想保护用户，苹果应该移除 Safari，或者让所有的互联网访问通过过滤代理服务器，阻止攻击性内容。也许他们会？)

苹果已经把自己指定为你的道德守护者，所以它必须审查内容。这是一个风险:

1.  任何一家商业公司如何才能公正地决定什么符合你的利益，什么不符合你的利益？
2.  艺术、色情、良性内容和攻击性文字之间的界限很模糊，主观性很强。然而，苹果必须每天做出一致的非黑即白的决定。
3.  被禁止的应用程序和内容比被接受的应用程序和内容受到更多的关注。这可能会给苹果带来负面影响，开发者甚至可能开始利用这一点为自己谋利。例如，为什么不在你现有的网络/桌面应用中添加一个你知道会被禁止的 iPhone 版本呢？这个故事将会以苹果公司为代价提升你公司的形象。
4.  然而，苹果接受的应用程序和内容可能会导致最大的问题。被认可的产品被认为是高质量的，不含可疑的内容。这是一项艰巨的任务，如果我无意中发现了冒犯我的东西怎么办？这是否意味着苹果失败了？我能要求退款吗？我可以起诉公司吗？

在移动市场相对较小的情况下，苹果很容易管理几十万个应用程序。但它增长很快——当它增长两倍或三倍时，苹果将如何应对？

然而，最终终结苹果“高尚”意图的是商业成本。App Store 中的绝大多数应用程序从未盈利，但苹果必须雇人彻底检查每一个提交的应用程序。如果申请的接受或拒绝引发法律纠纷，那么这一成本将会无限增长。

审查对任何公司来说都是一项巨大的成本，微软应该注意到这一点(他们正在为 Windows 7 手机应用程序制定类似的道德规则)。这可能需要一些法庭案件，但我怀疑它在苹果公司有可行的长期商业前景。*
