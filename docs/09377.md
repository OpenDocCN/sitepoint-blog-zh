# 打造激情品牌大使的关键

> 原文:[https://www . site point . com/the-key to-creating-passive-brand-ambassadors/](https://www.sitepoint.com/the-key-to-creating-passionate-brand-ambassadors/)

随着史蒂夫·乔布斯从苹果公司离职，我想起了苹果的顾客对品牌的重视，以及苹果是如何利用这些品牌大使将他们的公司发展成为世界上最大的公司的。你不是每天都能看到有人试图说服朋友为什么他们选择的品牌更好。想想看——苹果粉丝会走极端把 Windows 用户“转换”到自己这边。

我们都听说过留住一个客户比找到一个新客户更便宜，但是仅仅留住客户是不够的。满意的顾客不会把你的公司告诉他们所有的朋友。快乐的顾客不会特意把你介绍给同事。那么如何把“开心”的顾客变成品牌大使呢？

### 全心全意为客户服务

大多数顾客都会在某个时候需要帮助。可能是在出售后不久，也可能是几年后。如果你看看那些拥有热情的品牌大使的公司，你通常会发现一种客户服务文化。

看看 Zappos，一家以卓越的客户服务而闻名的公司。创始人谢家华喜欢讲的一个故事发生在他会见一些供应商的时候。已经过了酒店客房服务的截止时间，所以谢建议他们给捷步达康打电话要一些披萨。不，Zappos 不卖披萨，但是电话那头的客服代表找到了一家当地的披萨店，这家店还在营业，可以给他们送货。在 Twitter 上搜索@zappos，你会看到成百上千的顾客在讲述类似的故事。

Rackspace 是另一家以支持著称的公司。他们称之为“狂热支持”在一个公司做得越来越少的行业中，Rackspace 通过管理您的 web 服务器的各个方面而走得更远。他们甚至经常尝试帮助完成技术上超出其服务级别的任务，例如安装第三方软件。如果问谁是托管专用主机的领导者，你会发现许多 Rackspace 客户都在大声疾呼。

Lands End 是另一个很好的例子——他们的担保是传奇式的。“保证。句号。”他们将在任何时候收回任何产品，进行换货或全额退款。不问任何问题。一位顾客曾经花 19000 美元买了一辆原版伦敦出租车，这辆出租车出现在 1984 年假日目录的封面上。2005 年，该客户联系 Lands End，询问他们是否可以退还汽车全额退款。当然啦！

### 让服务为您服务

您可能无法在任何时候收回任何产品。你可能不能为你的顾客订购比萨饼。但是你有机会找到让你的客户生活更轻松、提供价值和提供优质服务的方法。

银行可能无法免除服务费，但想象一下，当你的可用余额低于某个阈值时，收到一条短信……“你的余额低于 20 美元，存款或转账，这样你就不会被收费了！”

想办法让你的顾客体验更好。这可能就像一个月去拜访他们一两次，问他们事情进展如何一样简单。你可能会惊讶地发现，几乎每次你来访，他们都开始需要你完成工作！

你也可以问他们喜欢如何收到发票。随着项目的完成，我们的一个大客户收到了发票，每个月都会产生几张发票。他们提到了由此导致的会计问题，所以我们合并了他们的发票，现在每月给他们发送一份所有工作的发票。

### 将技术用于你的优势

就像上面的短信例子一样，想办法利用技术让客户的生活更轻松。我们经常想办法改进我们的流程，但是我们又有多少次想办法让流程对我们的客户更好呢？

你可以在发票上贴上二维码，让顾客在旅途中轻松付款，或者发送短信通知确认预约时间。你可以在 YouTube 或 Vimeo 上创建回答常见技术支持问题的截屏。或者，你可以简单地开始使用像 Basecamp 这样的工具来促进更好的交流。有不在本地的客户？考虑与他们进行视频会议，既节省时间又省钱。

有无数种方法可以利用技术让客户体验更好。花点时间想想你能做些什么来让你的顾客体验更好，你就离把他们塑造成品牌大使更近了一步。
