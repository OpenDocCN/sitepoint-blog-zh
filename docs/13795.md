# 美国在线比谷歌转换得更好

> 原文:[https://www.sitepoint.com/aol-converts-better-than-google/](https://www.sitepoint.com/aol-converts-better-than-google/)

WebSideStory 公布了四大搜索引擎的转化率数据:AOL，MSN，Yahoo！还有谷歌。

> 根据 WebSideStory Index 的数据，今年 1 月，AOL 搜索在企业对消费者电子商务网站上的转化率最高(6.17%)，其次是 MSN(6.03%)、雅虎(4.07%)和谷歌(3.83%)。WebSideStory Index 是一个新的统计指标，从使用该公司屡获殊荣的网络分析技术 HBX 分析(WebSideStory Analytics)访问网站的数百万用户中筛选出技术图形和电子商务趋势。这项研究包括有机和付费关键词的流量。

对这种差异的一种解释是，像 AOL、MSN 和 Yahoo 这样的“门户”网站往往吸引更多的积极购物者。“随着门户网站内容和服务的丰富，AOL、MSN 和 Yahoo 可能会倾向于吸引更多的买家。与此同时，谷歌可能会吸引更多的浏览器——那些购买意愿较低的人，”WebSideStory 的高级数字营销顾问阿里·贝南(Ali Behnam)说。

也就是说，所有四个主要的搜索引擎仍然大大优于“其他搜索引擎”类别，后者在 2006 年 1 月产生了微不足道的 1.97%的转化率，在 2005 年第四季度，最繁忙的网上购物季节，只有 2.3%。
