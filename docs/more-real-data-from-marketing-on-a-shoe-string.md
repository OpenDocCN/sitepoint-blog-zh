# 更多来自鞋带营销的真实数据

> 原文：<https://www.sitepoint.com/more-real-data-from-marketing-on-a-shoe-string/>

我上周写的非营利事业([http://www.actioncorps.org](http://www.actioncorps.org)如果你感兴趣的话)继续向前发展，可以为你提供一些关于什么可行什么不可行的新数据。事实证明，一个报价比其他报价高出 50-100 倍！

你可能还记得，我尝试过通过向当地和全国媒体、我自己的朋友/同事网络以及这个 Sitepoint 博客发布新闻稿来引起关注。

到目前为止，上述唯一真正实现的是向当地一家默默无闻的在线出版物发布的消息，这让我获得了明天在扶轮社演讲的请求，这将很容易让我获得 50-60 个签名(当你演讲时，总是有一份表格，你给所有参与者，并提供给他们，让他们报名)。

但到目前为止，最好的提议/策略是我自己的咨询邮件列表加上一个特别的免费提议。我发出了一条广播消息，邀请人们加入这个项目(只需要提供电子邮件和姓名)，以换取价值 99 美元的免费领导力电子书。

反应如此热烈，以至于我投资了一个自动回复器来收集名字。

这里有什么教训？

1.总是从你所在的地方开始。一个认识你并信任你的小名单几乎总是比一大堆不认识你的人做得更好。这个教训也适用于选择目标市场，因为在一个专注的市场上，你会用更少的工作获得更多的客户，即使这个市场比 1400 万小企业的虚幻吸引力还要小。

2.相关报价将显著增加响应。在这种情况下，非盈利性企业的任务是帮助那些有需要的人。因此，一本关于领导力的书是签约的一大回报。关于你的专业领域的免费书籍或指南可以产生潜在客户的名字，并向他们表明你不仅仅是一个供应商。

非营利企业和营利企业有明显的区别，但是这些关于经营的教训对两者都适用。

(BTW，如果你想要那本免费的书，只要报名就可以得到；我现在将它发送给任何加入的人，因为反应太强烈了，不能不全面执行)。
