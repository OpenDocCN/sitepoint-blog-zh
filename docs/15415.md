# 不要因为侮辱外国游客而失去他们！

> 原文:[https://www.sitepoint.com/visitors-insulting-them/](https://www.sitepoint.com/visitors-insulting-them/)

对于一些网络营销者来说，在他们的网站上提供多语言导航和宣传材料，希望扩大他们的客户群，这已经成为一种前沿策略。虽然母语不是英语的国际用户开始大量上网，但用他们自己的语言格式来迎合他们本身就是一种艺术，并不适合目前在网上兜售的廉价而简单的“不用动脑筋”的伪解决方案。如果你给他们其中的一个，很可能你会永远避开他们。尽管你认为这是一个善意的错误，但事实是，如果你给人的印象是你不在乎提供一流的服务，这些潜在客户可能会相当无情。不要忘记，很多人实际上热爱他们的母语，不喜欢看到它被屠杀。

语言学和翻译本身就是科学，需要得到应有的尊重，或者至少需要专业的处理。非专家应该摆脱的一件事——越快越好！–这是一个荒诞的说法，即熟悉母语意味着你了解语言及其社会影响。这也不是缺乏对一门外语的掌握——更多的时候，是基本概念有缺陷，比如认为逐字翻译，尽管不可否认不是很优雅，但至少会给你对原文内容的“大致想法”。虽然在高度专业化的技术领域的非常有限的背景下，这在某种程度上实际上可能是正确的(关于化学的学术论文充满公式和程序描述就是一个例子)，但古老的法学院格言“一知半解是一件危险的事情”甚至在这里也适用。不用说，依赖不完善的自动化会使事情变得更糟。

翻译机器人往往反映了这种错误的推理，它们对大众观点的支持——通常在这些问题上没有受过教育——也没有多大帮助。

不要试图争辩这毕竟是“聊胜于无”——真正令人讨厌的是所呈现的语言看似熟悉，喜欢幻想“至少得到了它的要点”。因为它可以(而且肯定会！)以如此多的微妙方式将你引入歧途，最终你可能会比一开始一个字都没听懂还要糟糕。简单地问问你自己，你是否会签署这种语言“质量”的百万美元合同…

随着目前美国在互联网上的主导地位明显减弱(正如所有主要的研究和分析将显示的那样)，与国际接轨将变得更加重要。在世界范围内，冲浪者并不满足于坚持使用他们自己语言的本地或本地化搜索引擎:相反，英语正在各地迅速普及，甚至在前共产主义国家，更不用说以前由法语或西班牙语主导的地区了。这些人中有越来越多的人受过良好的教育，掌握双语，相当精通英语，他们越来越多地使用美国本土的搜索引擎。这只是一个时间问题，直到所有美国引擎将不得不适应这种情况，即使只是为了适应他们的国际广告商。因此，显而易见的是，只有基于真实世界网络人口统计的链接流行度(而不是仅仅一厢情愿的想法和既定的政治和文化偏见)才能满足广告商和用户的需求。

所以要么做对，要么不做！要么雇用一个真正的专业翻译服务，要么找人修改你的在线副本，以适应所有那些国际客户，他们的英语掌握得相当好，但不太符合你的美国或英国有偏见的行业行话，呃，说法。

当然，对于非英语网站来说，情况也是如此，反之亦然:不要指望某个翻译机器人能做得很好，让你做成一单生意！虽然你在学校的英语老师可能一遍又一遍地称赞你的热情，但不要自欺欺人地认为这会让你成为一个以英语为母语的人。

如果你对英国、美国、澳大利亚或新西兰的客户感兴趣，不要拒绝承认你的英语水平可能不够完美，从而给他们留下外行无能的印象。毕竟，这没什么好羞愧的——然而，试图以很低的成本解决这个问题才应该感到羞愧！如果在营销中有一件事你想避免的话，那就是在法庭外被嘲笑。
