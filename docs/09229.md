## 1.超级鱼

是一个增强的 Suckerfish 风格的菜单 jQuery 插件，它采用一个现有的纯 CSS 下拉菜单(所以它降级得很好)。
![SuperFish.jpg](../Images/fa1506a723dc23ea59013f0a8678bd30.png)

## 2.YoxView

是一个弹出图像浏览器，使用 jQuery 1.4 构建。它使用简单，但功能强大。

主要特性:
–调整图像大小以适合浏览器窗口
–内容可预缓存
–可使用同一页面中的多个实例
–完全可配置
![YoxView.jpg](../Images/c67d8191ba8e60e3482fdaf31e3c3827.png)
[源](http://www.yoxigen.com/yoxview/)

## 3.同ＳＰＲIＧＨＴＬＹ

是一个由 Artlogic 创建的 jQuery 插件，用于在纯 HTML 和 JavaScript 中创建动态角色和背景动画。这是一个简单的轻量级插件，提供了一些创建动画精灵的简单方法。
![Spritely.jpg](../Images/5c864e206e0edf87304fdd611ab15c27.png)
[来源](http://spritely.net/)

## 4.Piecemaker XML 库

是一个开源的 3D Flash 图像旋转库。
![Piecemaker-XML-Gallery.jpg](../Images/2c7edfc7ae5a8b6f54be7689bf3e2377.png)
[来源](http://activetuts.s3.amazonaws.com/freebies/003_pieceMaker/piecemaker/index.html)

## 5.硬币滑块

是一个具有独特效果的 jQuery 图像滑块。
![Coin-Slider.jpg](../Images/423e30e421bee1e79dfb62d12357a46c.png)
[来源](http://workshop.rs/projects/coin-slider/)

## 6.AviaSLider

特性:
–8 种独特的过渡效果
–许多易于设置的选项来创建您自己的效果
–包含图像预加载器
–停止用户交互的自动播放
–有效的 HTML5 和 CSS 3 标记
–打包版本仅重 8kb
–支持链接的图像
–已经准备好使用 pretty photo light box
–使用 jQuery 1.32 和更高版本
![AviaSLider.jpg](../Images/f45e50db873ce90e0821804b35564604.png)
[来源](http://aviathemes.com/aviaslider/)

## 7.控制。标签 jQuery

它从链接/锚的无序列表中创建一个选项卡式界面，这些链接/锚指向页面上任何具有 id 属性的元素。因为它将这些行为附加到已经有语义含义的 HTML 上，所以对于不支持 JavaScript 并且对搜索引擎友好的浏览器来说，它会优雅地降级。
![ControlTabs.jpg](../Images/f55dc7db33d2324c0d5fcebdf8f8cae0.png)
[来源](http://livepipe.net/control/tabs#image_2)

## 8.用 jQuery 实现圆周运动效果

这个想法是有一个主菜单的圆形内容区。当鼠标悬停在其中一个菜单项上时，会出现一个子菜单元素，围绕内容区域循环。
![Circular-Motion-Effect.jpg](../Images/a9b301d6e3c33de671e8e125e88715de.png)
[来源](http://tympanus.net/Tutorials/CircularPortfolio/)

## 9.Popeye——嵌入式灯箱的替代品

这是一个基于 JavaScript 库 jQuery 的高级图片库脚本。在显示图像集合时使用它来节省空间，并为您的用户提供一种漂亮而优雅的方式来显示您的图像的大版本，而无需离开页面流。
![Popeye.jpg](../Images/8046d2d483bb0ba86d461543af2f7519.png)
[来源](http://dev.herr-schuessler.de/jquery/popeye/demo.html)

是一个基于 jQuery 的 lightbox 克隆。它不仅支持图像，还增加了对视频、flash、YouTube、iFrames 的支持。这是一个成熟的媒体灯箱。
![prettyPhoto.jpg](../Images/23635feed6a0141bc591a635f7dd5d53.png)
