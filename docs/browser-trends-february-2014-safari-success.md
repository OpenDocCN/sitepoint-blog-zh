表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月 19.7%的 IE10 用户更换了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

奇怪。IE、Chrome、Firefox 和 Opera 的市场份额都有小幅、相当不明显的下降。Safari 是大赢家，涨幅高达 0.6%。然而，并非一切都像看上去的那样。对 StatCounter CSV 详细数据的分析显示:

*   4.67%是 Mac 版 Safari(可能还有少数 Windows 用户)
*   5.04%是 iPad 版 Safari。

二十分之一的网络访问来自拥有 iPad 的用户。鉴于 Kindle 仅占 0.25%，这令人惊讶。

然而，对苹果来说，这不全是好消息。桌面版 Safari 的增长充其量是停滞不前；浏览器正逐渐被 Chrome 取代。如果苹果取消对竞争浏览器的禁令，iPad 版本会遭受同样的损失吗？鉴于苹果正在享受平板电脑垄断，这种情况会发生吗？这是一个容易吸引律师和监管者注意的职位。

我对苹果的建议:*向其他浏览器厂商开放 iOS*。如果 Safari 远远落后于其他应用程序，开发者和用户将会放弃他们的 iPads。但是我知道什么？苹果在他们的围墙花园里做得非常好。

其他新闻:IE11 的涨幅最大——主要是以 IE10 为代价。看到 IE6 的市场份额和去年这个时候一模一样，也很让人气愤。

> 我们认为，从 2006 年到 2014 年，支持一款浏览器的成本要高于帮助客户升级其传统硬件的成本。

这是一个聪明的营销策略，因为没有人会“困”在 IE7 上——即使是 XP 用户也可以升级到 IE8 或使用不同的浏览器。希望它能说服一些人放弃他们老化的浏览器。

## 移动浏览器的使用

继上个月的大幅增长之后，手机使用量在一月份又有所增加，达到所有网络活动的 23.77%。

热门移动浏览应用:

| 移动浏览器 | 十二月 | 一月 | 变化 | 亲戚 |
| 机器人 | 26.63% | 27.00% | +0.37% | +1.40% |
| 苹果手机 | 19.65% | 21.38% | +1.73% | +8.80% |
| Opera Mini/Mobile | 16.34% | 14.61% | -1.73% | -10.60% |
| UC 浏览器 | 11.78% | 11.57% | -0.21% | -1.80% |
| 铬 | 7.36% | 8.59% | +1.23% | +16.70% |
| 诺基亚浏览器 | 6.76% | 6.11% | -0.65% | -9.60% |
| 黑莓 | 3.03% | 2.58% | -0.45% | -14.90% |
| 其他人 | 8.45% | 8.16% | -0.29% | -3.40% |

12 月份大家都收到 iPhone 或者 iPad 了吗？！看起来是这样。谷歌也有一个不错的月份，Chrome 在手机排行榜上的上升速度和在桌面上一样快。

最大的输家是黑莓、Opera 和诺基亚。他们正在努力与 iPhone 和 Android 的统治地位竞争，我不相信他们能扭转局面。
