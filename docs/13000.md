# 更大更好:手机屏幕尺寸的趋势

> 原文:[https://www . site point . com/bigger-and-better-trends-in-mobile-screen-sizes/](https://www.sitepoint.com/bigger-and-better-trends-in-mobile-screen-sizes/)

Morten Hjerde 在过去的一周发布了一些有趣的帖子，其中他收集了挪威人的使用数据(彩屏手机，支持 Java 和网络浏览器)。在他的帖子中，Hjerde 说:

*   [绘制手机屏幕尺寸趋势图](http://sender11.typepad.com/sender11/2008/04/mobile-screen-s.html)并宣称**小手机屏幕(小于 240 x 320)已经过时，可以忽略**，
*   建议将 240 x 320 作为开发的目标尺寸，以及
*   鼓励用户界面设计师[开始考虑如何设计宽屏移动显示器](http://sender11.typepad.com/sender11/2008/04/more-on-mobile.html)，因为这是移动设备的发展方向。

他建议:

> 很明显，240 x 320(也称为 QVGA)势如破竹。到目前为止，这是最常见的，而且增长迅速。如果你开发，这应该是你的目标屏幕尺寸。

![Trends in the Norwegian market of mobile screen sizes](../Images/3f7b378b7208fd0b0253fb310af3dfec.png)

展望未来，iPhone 可能不是市场上唯一的手机，但当它发布时，每个人都同意它提高了移动网络体验的标准。

竞争刚刚开始赶上，这意味着开发者有越来越多的像素可以利用。Hjerde 预览了一些最新型号的屏幕尺寸，包括一些将于今年发布的型号:

> 高端手机向宽屏和横向发展的趋势是显而易见的。QVGA 和更大的手机正在获得自动屏幕旋转。风景通常是媒体消费的首选，而手持电影消费在相当长的一段时间内“前景看好”。

![Screen dimensions of future mobile devices](../Images/8af04f84db29c1cc84d59264f3492158.png)

以下是要点:

*   240 x 320(又名 QVGA)是功能手机和智能手机的新基线屏幕尺寸。
*   未来的移动设备最终将能够显示比 CRT 显示器的 640 x 480 像素更多的像素，VGA 显示器——即将推出的索尼爱立信 Xperia 将具有 800 x 480 的屏幕尺寸！

通常的免责声明“测试你自己的用户数据”是适用的，因为挪威的模式不能直接适用于世界其他地方。但仍然是有趣的东西！
