# 10 个在线 JSON 工具

> 原文:[https://www.sitepoint.com/10-online-json-tools/](https://www.sitepoint.com/10-online-json-tools/)

Here is a collection of online JSON tools which I have used while developing with jQuery & JSON data. They each have features which can come in handy to do the following: JSON validation online, JSON parsing online, JSON online editing, JSON online viewing, JSON to XML conversion online, XML to JSON conversion online and more cool JSON stuff. 

## 1.JSONLint

几乎是最流行的在线 JSON 验证器。只需粘贴 JSON 代码，然后单击 validate。

![JSONLint.jpg](../Images/efd232a179f6b4af1f9e89818b309aa1.png)
[来源](http://jsonlint.com/)

## 2.在线 JSON 树查看器

在线 JSON 树层次结构，并查找对深度嵌套(大型)JSON 文件有用的变量/值路径。

![Online-JSON-Tree-Viewer.jpg](../Images/a28d256eb1af354c1ea6419965914721.png)
[来源](http://www.jquery4u.com/tutorials/online-json-tree-viewer/)

## 3.Shell JSON 工具

伟大的在线 JSON 和 JS 工具解析 xml，编码 base 64，解码 base 64，加密密码，编码 md5，评估 regex 等等。

![Shell-JSON-Tools.jpg](../Images/ee6556d795c21dffaf57ddb01d3116fa.png)
[来源](http://www.shell-tools.net/index.php?op=json_format)

## 4.在线 JSON 查看器

非常有用的工具来清理你的 JSON，去除空白，格式化 JSON 等。

![Online-JSON-Viewer.jpg](../Images/32a044c7b1fd8fbf414bacc95b2fa980.png)
[来源](http://jsonviewer.stack.hu/)

## 5.简单的 JSON 查看器

将 JSON 字符串转换成友好可读的格式。

![Simple-JSON-Viewer.jpg](../Images/115a707ab466937153e8ea76761b8415.png)

## 6.在线 JSON 解析器

简单的字符串解析和 JS 在线评估。

![JSON-Parser-Online.jpg](../Images/33911817e63d7239300bdf9303e86bcc.png)
[来源](http://json.parser.online.fr/)

## 7.JSON 编辑器——XML 到 JSON

该编辑器允许轻松编辑 JSON 字符串，并将 XML 快速转换为 JSON。

![JSON-Editor-XML-to-JSON.jpg](../Images/2b58dccac6869724c52f5c9bac402a6b.png)
[来源](http://braincast.nl/samples/jsoneditor/)

## 8.JSON 格式化程序和验证程序

这个工具希望通过格式化和验证 JSON 数据来解决这个问题，以便于人类阅读和调试。

![JSON-Formatter-Validator.jpg](../Images/13da93397ec64046ab9eab962fc6c4fe.png)
[来源](http://jsonformatter.curiousconcept.com/)

## 9.XML 到 JSON 和 JSON 到 XML

在这个页面中，您可以将 xml 转换成 JSON，反之亦然。

![XML-to-JSON-and-JSON-to-XML.jpg](../Images/fa32344d8856b5eba515573972a6736c.png)
[来源](http://jsontoxml.utilities-online.info/)

## 10.JSONedit

JSONedit 是一个简单的 JSON 编辑器，有文本和树形视图。完整的申请。

![JSONedit.jpg](../Images/5d17ada23ba71ea749dac3a6dac3c7d8.png)
[来源](http://tomeko.net/software/JSONedit/index.php?lang=en)
