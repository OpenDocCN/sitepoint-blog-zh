# 好莱坞演员想要在线视频内容每天 759 美元的剩余收入

> 原文:[https://www.sitepoint.com/pay-hollywood/](https://www.sitepoint.com/pay-hollywood/)

随着美国演员工会(T2)和好莱坞电影公司试图敲定一项协议，网络娱乐的未来仍然取决于金钱。在周二的[，在长达八个月的关于艺人网络收入分成的僵局之后，谈判重新开始。核心问题是 SAG 要求网络发行和电视电影一样的钱。电影公司认为，网络收入不会支持同样的模式。与此同时，我们在网上看到什么，以及价格如何，仍然是核心问题。](http://news.yahoo.com/s/nm/20090219/en_nm/us_hollywood_web;_ylt=AhyQnT5BmEzkKIIT8nVo288jtBAF)

SAG 的基本费率是每天 759 美元，适用于通过在线视频发行的每部作品中的每个演员，这与网络消费者最初的想法背道而驰。从音乐到软件，免费是所有东西的在线信条，事实证明，改变这一理念在大多数方面都是徒劳的。据专家称，就专业制作的内容而言，网络娱乐仍然是业余爱好者的市场。那么，这给我们带来了怎样的视觉享受呢？在这种情况下，也许是第一次，好莱坞电影公司是正确的。

##### 网络电视的黑白

YouTube 占有大约 40%的在线视频市场，而它的竞争对手用十几种甚至更多的方式来瓜分“观看馅饼”。YouTube 声称数以百计的业余视频海报赚取了数千美元的广告收入，但即使是数十亿次的观看也无法支付 SAG 所建议的费用。即使是 Hulu 的电视节目重播也只代表了网络观众总数的 1.7%，所以如果 YouTube 在财务上不可行，那么 Hulu 又能如何呢？

视频和在线电视是 Web 2.0 和作为娱乐平台的互联网的绝佳名片。Joost ， [Veoh TV](http://www.veoh.com/) ，以及一大批“可能的”人还没有“拿出钱来”，至少在 SAG 要求的独轮车的全部比例上。

##### 光明的未来——不是现在

根据加州大学洛杉矶分校教授、电影专家和作家霍华德·胡伯的说法，“到目前为止，没有人有想象力来找出你可以在互联网上提供什么样的新东西，而你在任何其他媒体上都无法获得。”我对这个问题的想法是，如果好莱坞不知道如何去做，它可能无法完成。

来自 [Studio 3 Networks](http://www.engadgethd.com/2009/01/28/studio-3-networks-epix-is-the-new-home-of-lionsgate-mgm-and-para/) 的一个[聪明的想法](http://www.readwriteweb.com/archives/hollywood_studios_taking_on_on.php)是基于表格订阅的模式(Epix)，可能是部分解决方案。底线很明显，网络用户迟早要付费。网络制作人的问题将是一如既往地与电视竞争，除了方便和价格，网络视频目前没有优势。

##### 结论

作为分销和货币化媒介的网络仍处于起步阶段。好莱坞电影公司和几乎所有其他商业实体似乎仍然困惑于如何最好地利用其收入潜力。关于 SAG 要求的最新消息反映了一个更广泛的困境，但在可预见的未来，网络消费者的娱乐业可能会包括与 YouTube 傻瓜视频的版权之争的重演。
