# 另一个数据库加入开源

> 原文:[https://www . site point . com/another-database-joins-open-source/](https://www.sitepoint.com/another-database-joins-open-source/)

正如你们中的许多人今天可能已经看到的，CA (Computer Associates)宣布计划在开放源码许可下发布其 Ingres 数据库核心代码，从而加入了开放源码的行列。

有趣的是，CA 没有使用 MySQL 使用的双重许可结构——即你在你的应用程序中发布 MySQL，而且是免费的——那么 MySQL 就是免费的。如果你构建一个专有的应用程序并发布 MySQL，你必须付费。

CA only 将要求开发人员直接共享他们对 Ingres 所做的更改，而不关心在 db 平台上构建的任何应用程序。

根据 eWeek 对 Computer Associates International 数据库产品营销总监 Maurice Donegan 的采访，该公司正在审查整个代码库，以确保在发布中没有借用或非 CA 代码。这将限制下载和使用 Ingres 的用户在未来可能的诉讼中面临的风险。

我发现有意思的是，大规模的公司正在走向开源领域，即使有辅助应用程序。这强化了 Linux 和开源软件正在传统企业市场获得牵引力的观念。在企业中被大量利用的公司不会无缘无故地跳进开源空间。

当然，这些进入者并不一定带着慈善的意图而来。他们想通过与 Linux 做生意赚钱。
