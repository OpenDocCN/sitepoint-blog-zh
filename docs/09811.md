# 阅读附属细则

> 原文:[https://www.sitepoint.com/read-the-fine-print/](https://www.sitepoint.com/read-the-fine-print/)

我们一直在签署法律协议和合同。

有时它们是显而易见的——公寓租约、汽车贷款、客户合同。

其他时候，他们有点暧昧。

你用 iTunes 吗？您是否已经阅读了您仅通过*使用*该程序而同意的全部条款和条件？

写合同和法律协议的唯一目的是将法律权利分配给起草协议的个人或公司(或对另一方的责任)。写合同的个人或公司并没有把你的最大利益放在心上，相反，他们是在为 T2 和他们的 T3 着想。

### 从头到尾阅读协议

我最近被邀请在一个著名的会议上演讲，在同意了演讲细节和报酬后，他们发给我一份演讲人协议，让我签字并寄回。我把协议从头到尾看了一遍，意识到它实质上授予了该公司以任何方式使用演示录音的无限权利。此外，他们有权在任何宣传材料中使用我的姓名、肖像(照片)、声音、公司标志和信息以及传记，以推广对演示文稿的任何使用，无论是现在还是将来。

通过签署合同，我将基本上授予他们从录制的演示中创造任何产品或服务的权利，并使用我和我的公司来销售它。永远。我可能会成为某个产品或服务的“代言人”，而这可能会损害我的个人品牌或公司品牌。

他们的律师可能已经包括这一部分来保护公司，但它在这个过程中剥夺了我的权利，这是我不能接受的。如果我没有通读和重读协议，我可能会简单地签字并做演示。

签署协议时，你可能犯的最大错误是没有彻底阅读和理解文件。这似乎是常识，但许多人(如果不是大多数)并不完全理解书面合同中的语言。如果你对某个部分或单词的意思有任何疑问，那就由你自己去弄清楚。

在搜索引擎中查找这个短语，或者在字典中查找你不理解的单词。不要简单地依赖公司或他们的律师来解释这一部分。

### 聘请律师

一个好的律师可能很贵，但是他们会让你赚到(并且节省)比他们花费更多的钱。他们会一次又一次地为自己付出代价。把律师想象成一份有价值的保险单。

如果你不懂合同中的语言，你不能依靠他们的律师来帮助你。他们的律师只代表他们。只有你的律师在法律上有义务为你的最佳利益提供建议。

我让我的律师仔细检查了我签署的每一份主要合同，几乎在每一个案件中，我们都发现了放弃我的权利或不必要地将责任分配给我的条款或整个条款。在我读过的许多协议中，问题不像上面的例子那么明显。

几年前，我们的律师在一份仓库和办公楼租赁合同中发现了几个需要解决的条款，其中一个条款要求我们负责*所有*建筑维修，而不仅仅是由于我们的疏忽而需要的维修。这是一个简单的疏忽，但如果需要大修的话，可能会让我们付出代价。

### 请求更改

如果你发现合同的某一部分有问题，要求修改。就像你的商业关系一样，几乎在任何情况下合同都是可以协商的。如果问题很小，比如一个简单的遗漏或错误，我通常会给他们发电子邮件，告诉他们我的律师建议在我们签字前改正。

对于更大的问题，比如需要修改或删除的整个条款，我建议让你的律师直接与他们的律师联系。他们可以迅速起草一份所有相关方都同意的新条款或整个合同。

### 不要陷入一份糟糕的合同

如果其他方法都失败了，那就离开。如果你们不能就合同条款达成一致，就不要签字。失去一个潜在的客户或工作，或者失去一份新的办公室租约，都比让自己承担责任要好。签署一份合同可能会让你承担更多的责任，这些责任可能会让你付出远远超出你想象的代价。
