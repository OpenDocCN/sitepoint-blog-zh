# 在线客户服务–您的选择

> 原文：<https://www.sitepoint.com/online-customer-care-options/>

在网络上，消息传播得很快——一份好的客户证明价值连城！如果客户觉得他们受到了企业的不公平对待，那么他们会比有良好经历的人更快地被传出去。今天，公司竭尽全力来吸引他们的客户，但是好的推荐背后的驱动力是用户所享受的客户关怀。

客户服务包括技术支持、账单查询、运输状态以及传统的反馈方式。网络用户，尤其是那些为在线服务付费的用户，通常需要技术支持，而且他们希望尽快得到。订购产品的客户需要能够在送货过程中跟踪他们的货物，有些人只是想向他们订购的公司或服务提供反馈。客户服务可以成就或毁掉你的在线业务。

仔细看看互联网，了解那些把客户服务作为主要卖点的企业。提供售后支持不仅能帮助客户，还能与客户建立良好且往往有利可图的关系。

你的网站必须提供的客户服务类型完全取决于你销售的产品或服务。它可以简单到所有产品的“无问题退款保证”，或提供快速有效的电子邮件技术支持，或礼貌的 1-800 代表回答客户问题；无论你提供什么样的客户服务，都要在你的网站上特别注明你的公司是如何成功建立售后服务关系的。正如你可能在电子邮件讨论列表中经历的那样，你的名声会跟随你到其他潜在的顾客和客户那里。

##### 客户服务的类型

我们已经了解了为每一位访客提供客户关怀的重要性，现在让我们来看看你到底能提供什么。客户关怀的世界是无穷无尽的，创造性的想象力是伟大的，但一些基本的客户关怀方法应该始终提供。

***网站说明***

客户关怀不仅仅是为了售后服务关系。就如何使用你的网站给你的潜在客户具体而直接的指示。

向他们展示如何向购物车中添加商品，如何下订单，以及如何浏览您的产品。然后向他们展示如何去收银台，以及购买所涉及的任务。例如，你接受什么信用卡？运送方式？船运公司？还要向顾客解释他们如何在发货后跟踪他们的包裹。

如果你的网站没有提供实际的产品，向你的客户展示如何使用你的服务。如果您提供服务演示，告诉客户如何使用它们，以及在哪里可以找到它们。给他们一个地方来报告脚本错误和其他网站或支持问题。给他们一切机会，以及他们联系你所需的所有指示。

***常见问题***

FAQ 页面既易于设计，又能非常有效地快速回答客户的问题。对于新网站，网站开发者应该想出一系列他或她希望用户会有的问题。一段时间后，当通过电子邮件或电话提出其他问题时，它们应该被添加到 FAQ 页面。如果问题数量变大，考虑为它们创建一个搜索工具。

***讨论板***

讨论板很棒——它们可以节省技术支持的大量工作！因为这些公告板是公开的，所以您公司的任何客户都可以回答其他客户可能提出的问题，这使得您和您的员工可以从事其他工作。许多虚拟主机公司已经实现了这样一个系统，并且发现它可以用于其他目的，例如:发布关于他们服务的公告。

***邮件支持***

电子邮件支持很便宜，占用的资源很少，而且非常有效。但是，在利用电子邮件支持时，需要特别注意周转时间，即回复客户电子邮件所需的时间。如果您有一个跨多个领域的支持技术人员团队，请为您的用户提供不同的电子邮件地址，并引导他们通过最快的路线找到相应的支持代表。

***电话支持***

如果你的企业足够大，并产生足够的收入，你也可以考虑通过 1-800 号码的电话支持。一提到 1-800 号码就能在你的业务中产生一定程度的可信度，让客户知道你是免费的。

***产品跟踪***

如果你销售实物产品，为客户提供一种跟踪他们订单的方法应该是理所当然的。包裹发运后，向客户提供一个跟踪号和一个到运输公司跟踪系统的链接。

<q>*Borders 允许对书籍、电影和音乐进行库存搜索，以及真正精确的说明。*</q>

这里提到的客户服务类型只是互联网上可能发生的事情的一个缩影。记住你为什么要建立网站——推广你公司的产品和服务，并提供简单和经济的客户互动。

顾客在寻找完全相同的东西，尤其是涉及金钱的时候。如果客户对可靠的售后支持不满意，他们就不会拿出塑料。

##### 最基本的必需品

电子邮件支持是最受欢迎的，也可能是最有效的客户服务方式。只需花一点时间搜索脚本存储库，您就可以在实现组织和管理所有支持请求的系统中找到很多帮助。除了自己编写支持票系统之外，您还可以从脚本数据库下载并实现预先制作的脚本。一些是:

*   [我的描述](http://www.mycgiscripts.com )
*   [热脚本](http://www.hotscripts.com  )
*   [资源索引](http://www.resourceindex.com)

下载一些，测试一下，决定哪种脚本最适合你。当然，不需要脚本，因为电子邮件地址也可以，但是一些脚本有助于在高容量环境中组织和管理支持请求。

##### 底线

客户服务实际上可以归结为一个底线:收入。在销售之前和之后满足和帮助客户将会吸引新的客户到你的网站，并让他们再次光临。给顾客他们想要的，他们会回报你的！
