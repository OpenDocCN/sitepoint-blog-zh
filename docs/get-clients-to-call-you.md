# 让客户打电话给你的三个简单步骤

> 原文：<https://www.sitepoint.com/get-clients-to-call-you/>

如果客户给你打电话，而不是你去追他们，做生意会容易得多。吸引客户比你想象的要容易，只要你知道该怎么做。

在这篇文章中，我将概述我吸引客户的简单公式。

## 定义你的定位

作为一个自雇的专业人士或小企业主，定义你的利基使获取客户更容易。你可以给自己定位来吸引客户，并对如何锁定他们有更清晰的想法。这使得确定你的营销将产生最大效果的地方变得更加容易。澄清这一点也有助于你向你的联系人和同事解释你希望从他们那里得到什么样的推荐。

不可能让每个人都满意，试图这样做注定会失败。有了明确的定位，你就能更好地理解你能为客户提供什么样的利益。你将能够直接针对他们的独特需求，解释你如何解决他们的问题，让他们的生活更轻松。通过了解自己的定位，你可以将自己定位为潜在客户的理想解决方案。

一旦你确定了自己的定位，你就不必再把你的营销精力浪费在不太理想的客户身上，这些客户一开始就不太可能与你合作。你将缩小你的营销对象的范围，这将转化为你在营销上投入的时间和金钱的更大回报。

最重要的是，当你知道自己的定位后，你就可以展示自己是你所在领域的专家。客户通过与你合作而受益，并且大多数愿意为你的服务支付更多。这样的专业知识让你有机会赚取更多的收入。

## 了解功能和优势之间的区别

一旦你的定位被确定，是时候向外看了，看向你的潜在客户。目标是弄清楚你的客户通过与你合作将会取得什么样的*结果。换句话说，你必须为客户定义与你合作的好处。*

 *当与我的客户一起工作时，我发现他们中的许多人不清楚好处和功能之间的区别。如果你想吸引客户，谈论你的业务特点可能会浪费他们宝贵的时间。在大多数情况下，你的特征与客户的需求和挣扎无关。另一方面，如果你能发出一个清晰的信息，说明你的业务的好处，潜在客户更有可能说，“让我们一起合作吧！”客户支付的是利益，而不是功能。

当我谈论特征时，我指的是你公司的属性。这些包括关于你的产品或服务的一般陈述，或者诸如你如何工作、你的公司的规模等因素。

相比之下，利益是你的客户从与你合作或从你那里购买中获得的东西。任何能回答最重要问题的东西:“我从中能得到什么？”对你的客户有利。

例如，关于我自己的生意，这里有一些特点和好处。

Feature

我为客户提供一个免费的发现会议，让他们了解我的业务，以及它如何帮助他们实现目标。

Benefit

客户在雇佣我之前有机会和我联系，所以他们提前知道他们在做正确的选择。

Feature

我给我的客户我们每次谈话的录音。

Benefit

我的客户可以在以后重温这些信息，或者如果他们对某一点不确定的话；没有立即完全掌握一切的压力。

Feature

我有一个 14 步公式(过程),我用在我所有的客户身上。

Benefit

我的客户获得了一个已被证明有效的公式，这个公式为许多其他企业的成功做出了贡献，因此他们可以看到我是为他们产生正确结果的最佳人选。*  *## 了解你的目标市场和理想客户

通过定义你的利基和与你合作的好处，确定你的目标市场变得简单。你的目标市场包括那些需要、想要和能够支付你提供的产品或服务的人。他们是最有可能和你做生意的人。你的理想客户是你目标市场的一部分。

我的一些新客户对营销有一个误解:他们错误地认为他们应该向全世界营销。在某些情况下，他们非常需要业务，以至于竭尽全力，却忘记了营销的目标。不幸的是，他们最终浪费了大量的时间和金钱向所谓的“潜在客户”营销，这些客户从一开始就不会对他们的产品感兴趣。

当你确定了你的目标市场，就更容易把你的精力集中在那些可能与你做生意的人身上。这样，你花在“[挑刺者](http://www.urbandictionary.com/define.php?term=tyre%20kicker)”身上的时间就少了——也就是说，那些最终没有兴趣从你这里买东西的人。

就像钓鱼一样。任何垂钓者都会告诉你，你得知道你想钓什么样的鱼。没有什么万能网。你必须专门化。知道你要接哪种类型的球，并使用最适合这种类型的战术。如果你不确定你的目标是谁，你的营销将是无效的。

另一方面，如果你确切地知道你想瞄准谁，定制你的营销信息就变得容易多了。这有助于你创造一个足够清晰的信息来吸引你的理想客户。它解决了他们独特的问题，让他们知道你能给他们想要和需要的东西。

了解你的目标市场还有几个额外的好处:

*   这会带来你喜欢与之做生意的客户。

*   这使得满足您市场的特定需求变得更加容易。

*   它阐明了谁是你的市场，让你充满信心地前进。

*   一旦你知道谁会使用你的产品，你就可以用所有正确的方式来定位你的产品。

*   您可以根据您的市场和细分市场定制您的信息。

*   你将节省大量的时间和金钱，这些时间和金钱可以用于其他用途。

*   因为你将向那些预先确定了自己资格的客户进行营销，从而使销售变得更加容易。

在我获得客户、建立业务和赚取更多收入的公式中，关键的一步是知道你想从你的业务中得到什么。了解你的目标市场是其中的一部分。当你对你的市场有一个好的感觉时，它会让你更容易知道你想要什么，这让你的业务增长和收入变得更容易。

当你确定了你的目标市场，你就有可能做出与你的业务相关的预测。你可以考虑当前的事件和市场趋势，并在必要时改变方向。例如，在教练工作中，我会查看经济新闻，以确定我的理想客户可能是什么心情。

所有这些都让你与你的目标市场建立更紧密的关系，这让你有能力增加你的知名度、喜欢度和信任度。当你意识到谁是你的客户时，你可以和他们建立良好的关系，在一个相互尊重和信任的地方与他们见面。想想看:如果你对你的目标市场一无所知，你怎么可能满足他们的需求？

最后，通过了解你的目标市场，你将能够弄清楚它在经济上是否可行。迎合这个市场能让你的生意兴隆吗？如果答案是否定的，你可以重新评估你的利基，调整你的业务。

了解你的潜在客户。你将与他们建立关系，并深入了解他们的需求。这是你事业成功的重要一步。*
