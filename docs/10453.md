# jQuery 的优势

> 原文:[https://www.sitepoint.com/benefits-jquery/](https://www.sitepoint.com/benefits-jquery/)

## **什么是 jQuery？**

你可能听说过这个，也可能用过。jQuery 是最优秀的跨浏览器 javascript 库，它是为简化 html 的客户端脚本而编译的。简而言之，它是一个“随时可用”的 javascript 库，具有许多可视化功能，比如流行的渐进渐出效果。因为 jQuery 有很多令人印象深刻的效果，所以它经常被网页设计师用来使他们的设计更加可爱和优雅。

## **为什么要使用 jQuery？**

网页设计师可能在他们的网页设计理念上遇到了困难。他们有时可能会寻找一个即时插件，使他们的设计看起来很好，甚至是最好的。在各种 jQuery 插件(包括图像滑块、内容滑块、弹出框)的帮助下，网页设计师的工作可能会最小化，因为他们所要做的就是调整或定制 jQuery 插件的小部分或大部分，以匹配他们正在进行的设计。

**jQuery 的一些好处:**

**1)** **让新手变成高手。**

*   易于使用，易于理解。它不需要高级编程技能。
*   即使是新手设计师也能在几分钟内上手。

**2)** **加快速度。**

*   5 行代码对 30 行 javascript 代码。
*   简单地将它包含在你的 html 代码的 head 标签中。

3) **让网站看起来更专业。**

*   正确使用效果或过渡，可以让你的网站看起来更专业。

**4)** **jQuery 在 SEO 方面超过 Flash。**

*   有些 jQuery 效果和 Flash 几乎一模一样，但好处是可以在 SEO 方面进行优化。
*   jQuery 可以使用的大小小于一个闪存。
*   jQuery 的内容是搜索引擎机器人可读的。

**5)** **大量插件可用。**

*   你不会很难找到 jQuery 插件，因为它们在互联网上随处可见。
*   一些免费的 jQuery 插件提供了高质量的视觉外观。

这些只是 jQuery 的一些好处。正如你所看到的，即使初学者没有太多的编程技能，他/她也可以让 jQuery 工作，所以它很容易学习和使用，他们可以使用他们想要的任何插件，加上一些 jQuery 开发人员正在尽可能缩短他们的脚本或代码，以便用户可以更好地定制。默认情况下，jQuery 基本语法脚本提供了广泛的动画效果选择；这只是一个选择最适合你的网站的问题。
