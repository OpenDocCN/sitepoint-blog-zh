# BBC 向大众推广开源

> 原文：<https://www.sitepoint.com/bbc-promotes-open-source-to-the-masses/>

对一个组织(BBC)如何找到将开源与大众消费主义甚至公共服务相结合的方法的引人入胜的见解。(感谢斯托扬！)

我们谈论了这么多关于开源和将它应用到我们的技术和商业追求中——这绝对没有错——但这是一个关于一个组织如何以一种微妙的方式将它带给大众的极好视角。

BBC 没有直接向公众推广开源软件，而是想方设法将其融入到更大的整体使命中，在大众媒体猖獗的商业“付费播放”服务中实现转型。

这非常令人耳目一新，因为它提醒我，我可以探索和确定更多微妙的方式，使开源可以使我的客户和他们的客户受益，而不需要考虑开源解决方案处于最前沿的事实。也就是说，也许通过使向街上的消费者交付声明或其他内容变得更经济，对他们来说比降低软件许可成本更有价值。因此，从不同的角度看待一个解决方案可以让整体前景变得光明。

我还发现非常有趣的是，在这么多对流入数字管道的更多内容收费的企业举措中，BBC 正在寻求提供更多免费内容，或许可以通过许可而不是对频道或内容收费来找到新的利润中心。
