# 你如何使用 Twitter 列表？

> 原文：<https://www.sitepoint.com/how-do-you-use-twitter-lists/>

去年 11 月，Twitter 发布了一个大功能:列表。这是一个备受期待的版本，因为它为用户——尤其是那些不使用 Twitter 客户端的用户——提供了一种对人们进行分类和分组的方式。作为提醒，这里有一些关于 Twitter 列表的细节:

*   每份名单目前限 500 人。
*   每个用户最多可以创建 20 个列表。
*   您可以创建公共分发名单或个人分发名单。
*   列表名称不能以数字字符开头。
*   您可以将联系人添加到列表中，而无需关注他们。
*   您可以向多个列表添加关注者。
*   您可以通过阻止列表的所有者将自己从列表中删除。
*   你可以像对任何 Twitter 用户一样提及任何 Twitter 列表，只需在列表所有者用户名前添加一个正斜杠“/”，然后添加列表名称:@ <username>/ <listname>。</listname></username>

## 如何使用 Twitter 列表

Twitter 列表起步有点慢，主要是因为它们使用起来不太直观。最大的困惑来自于关注列表的实际含义(你关注的是列表的整体，而不是列表中的个人)。但它的势头确实越来越大，尤其是像 HootSuite 和 Tweetdeck 这样的 Twitter 客户端将列表集成到了它们的设置中。现在列表已经有几个月的历史了，它们已经大规模起飞，就像 Twitter 本身一样，人们正在寻找自己创造性的方式来使用列表。以下是 Twitter 用户使用列表的一些最受欢迎的方式:

*   推荐其他用户跟随
*   筛选您考虑关注的人
*   留意你不想关注的人的推文
*   观察你的竞争对手在做什么
*   当你关注很多人时，让你的 Twitter 流更容易管理
*   被公认为某一特定领域的专家
*   看看别人是如何给你分类的
*   要了解您最喜爱的博客订阅源
*   跟踪组织、公司或行业中的人员
*   让自己成为目标
*   去发现人们从你的推文中发现的更有用的东西
*   实时跟踪会议和其他事件
*   观察客户和潜在客户的活动
*   提升搜索引擎优化
*   来衡量你的“推特受欢迎程度”
*   向您的网站、博客等提供特定主题的新闻/推文。
*   进行市场调查
*   创建和管理目录
*   寻找、观察和探索利基市场
*   跟踪行业新闻
*   在推特上认识当地人
*   从同龄人和同事那里获得想法
*   与家人交流

正如你从这个列表中所看到的，在你的社交网络活动中有很多使用列表的方式。但是少了什么呢？你如何使用 Twitter 列表？
