# 如何生成精彩的内容

> 原文：<https://www.sitepoint.com/generate-awesome-content/>

作为内容开发者，我们现在都知道是高质量的内容推动了互联网的发展。它是人们消费、参与和与他人分享的东西。因为这是人们想要的，这也是搜索引擎在其结果页面中排名靠前的。

我们也知道制作高质量的内容并不容易。即使是最基本的内容类型之一，[博客帖子](http://www.affilorama.com/blogging/how-to-write-a-blog-post)，也需要大量的努力才能变成人们会觉得有趣和有用的东西，从而成为高度共享和浏览的资源。

*   我们必须想出引人注目的标题。
*   我们必须用正确来源和引用的准确信息来支持我们的声明。
*   我们必须展示正确的例子。
*   我们不得不考虑使用富媒体来补充帖子。
*   我们必须考虑所有内容的长度，从整篇文章到各个部分，到段落，再到句子。
*   我们必须让它可行。
*   我们必须让它有趣、鼓舞人心、扣人心弦、发人深省等等。

我们必须考虑到所有这些因素，完美地平衡每一个因素，才能写出一篇精彩的文章。然而，在文章达到能接触到多少人并迫使其采取行动的上限之前，我们只能对这些元素做这么多。

让一篇文章或任何内容突破这种限制的一个因素是它背后的想法。真正能引起观众共鸣的是源于伟大思想的内容。这是促使他们[笑、哭、思考和行动的原因。](http://communicationsfocus.com/apple-advertising-appeals-to-emotion/)

创意是伟大创意的关键因素。第一次体验全新事物的感觉是无可替代的。

当我们遇到他们从未感受过的事情时，我们会变得兴奋或焦虑或充满希望或恐惧，我们会受到刺激而做出反应。我们经常通过告诉别人我们的经历来做到这一点，这样就有了一种公共纽带，让我们能够分享我们的感受，这在社交媒体上一再得到证明。高质量的内容正好可以做到这一点。

但是不要弄错了，因为想法本身不一定完全是原创的。人类历史的大部分是建立在极少数人重复创新的基础上的，我们都因此变得更好。创意可以通过其形式和方法的新颖性来实现。这就是我们如何解决一个旧的想法，使它变得新鲜。

## 产生内容创意

这里有一些提示，你可以用来帮助你的内容构思过程。

### 消费其他高质量的相关内容

你能想到在你的行业中从未有人做过的事情，这种情况真的很少。你*能*做的是从已经做过的事情中吸收尽可能多的信息，然后利用你从这些内容中学到的东西来获得新的东西。

无论是寻找内容空间的缺口，分析内容的趋势，还是结合不同内容的想法，你都会通过研究已经到位的内容来发现创新。

### 开始头脑风暴会议

有时候，阻碍我们创作高质量内容的障碍是我们的自我。我们想成为想出好主意的人，这样我们就可以独自承担它的创造。我们可能只是太骄傲而不愿寻求帮助。

这是一种弄巧成拙的态度，只会给我们自己带来更多的压力。当我们“失败”时，它会在未来对我们造成更大的伤害。

与你的同事联系，你认为他们具有创造性和/或分析性思维，能够为这一过程做出贡献。你可能会困在一个狭隘的观点中思考，所以通过允许多种观点进入画面来解放自己。

对他们的想法持开放态度，但也要记住让每个人都带着批判性的思维参加会议，过滤那些没有结果的想法，完善那些有方向的想法。

### 采访行业影响者和同事

即使你的同事们带着不同的头脑参与讨论，你们所有人在原创性方面所能提出的东西也可能是有限的。这可能是因为仍然拥有指导业务的相同使命和愿景。这就是为什么走出自己的公司会有所帮助。

行业影响者已经拥有感动每个人的知识和智慧，因此挖掘他们的洞察力来激发你的内容构思是一条显而易见的道路。你也可以安排与你的同行进行面谈，他们可能对你所在的行业有同样重要的事情要说。

### 与你的听众交谈

一旦你与你的同事、同事和思想领袖展开了讨论，与他人沟通的逻辑延伸就是与你想要接触的人交谈。

尽管你认为自己来自一个更开明的位置——成为创造消费者消费的过程的一部分——但你会惊讶于你能从与普通消费者的对话中获得什么。

并不是所有的人都能清晰地表达他们想要什么或者他们有什么问题，但是他们的意见和要求都是来自一个真实的地方。由你来[用正确的问题引导对话](http://social.swiftpage.com/2013/04/22/marketing-moment-solving-customer-service-problems-by-asking-the-right-questions)，这样你就能发现他们真正想要消费和参与的是什么。

### 矛盾

需要明确的是，这并不是为了争论而争论的建议。这是对你所在行业的流行观念采取一种更具批判性的方法，看看它们是否真的经得起推敲。目标是挑战人们的思维方式，并在这样做的过程中开辟新的、可能更好的做事方式。

如果这些概念和实践在您的提问和测试中没有崩溃，那么您可以将它们分解成一篇原创的内容，准确地展示这些原则和实践是如何正确的，以及为什么它们仍然应该被应用。

重申一下，不仅仅是想法本身必须是原创的。如果你能以一种独特的方式提出一个已经被报道过的想法，你就把这个伟大的想法带给了一个全新的观众，他们会觉得这个想法很有趣，很吸引人。

### 做案例研究

我之前提到过，剖析可靠的原则和实践可以成为新想法的灵感来源。您也可以通过案例研究将这种思路用于制作新内容。通过密切关注应用特定原则和实践的公司或网站，你可以通过展示它们对公司或网站的影响来强化它们的有用性或暴露它们的不足。

### 策划高质量的内容

互联网上有很多内容需要解析，尽管搜索引擎很复杂，但它们在结果页面上显示内容的方式仍然有限。由于这些情况，总有人希望以一种可访问的格式收集和组织最好的东西。

将你认为必看的内容编辑成一篇或一系列的文章，并附上一小段文字，告诉读者为什么他们需要查看你列出的内容。公开为什么你策划了这个特别的内容，你自己创造了新的内容。

### 改变内容用途

当你的网站上已经有了很好的内容，你仍然可以通过将信息转换成不同的格式来利用它的价值。一个例子是浓缩一篇长文章的重点，并添加连接各部分的视觉效果来制作信息图。另一种方法是收集一系列相关的博客文章，然后填补其中的空白，写一本关于整个主题的电子书。

重新利用你自己的内容可以让你接触到更多的人，因为不是每个人都只喜欢一种类型的内容。你迎合深度读者、内容浏览者、观察者、听众等。它还允许那些错过了你的内容的原始出版的人发现它，增加它的寿命。

## 结论

内容构思是一个需要花费大量时间和精力的过程，但是生成人们在网上寻找的那种内容是绝对必要的。为了让这个过程更有成效，我们需要沉浸在我们渴望创造的内容中，与理解内容的专业人士进行交流，并与消费内容的人进行互动。以此为基本理念，你就可以制作出你想要的高质量内容。
