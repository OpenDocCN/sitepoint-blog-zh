# 43 个自己动手的营销点子，从今天开始

> 原文：<https://www.sitepoint.com/43-do-it-yourself-marketing-ideas-to-start-today/>

需要一些灵感让你开始吗？这里有 43 个自己动手的营销点子，你可以从今天开始做:

1.  开始电子邮件营销活动
2.  列入在线行业目录
3.  要求并使用客户证明
4.  赞助慈善活动或当地运动队
5.  为你的目标市场设计和安排一个免费的电话会议或网络研讨会
6.  创造[独特的销售主张](http://en.wikipedia.org/wiki/Unique_selling_point)
7.  评论博客帖子
8.  做可穿戴营销(衬衫、帽子、包)赠品
9.  到你的社区去推销你自己
10.  自吹自擂(自我推销)
11.  考虑直接邮寄广告
12.  做一些公益工作
13.  包装你的产品和服务
14.  监控你的在线声誉
15.  加入专业组织
16.  提拔别人
17.  开始回顾你的网站分析
18.  修改你的名片
19.  通过社交媒体发展富有成效的关系
20.  创建一个利基专业
21.  恢复
22.  强调一点，向你的客户表达你的感激之情
23.  开始在线撰写和提交文章
24.  获得一个虚荣心强(且容易记住)的商务电话号码
25.  提供推荐奖励
26.  与你的客户发展个人关系
27.  规划网站重新设计
28.  花一些时间在搜索引擎优化方面提高你的网站
29.  创建电梯间推介
30.  清理您的电子邮件签名
31.  开始为你的客户考虑节日礼物
32.  走红
33.  为你的事业开一个博客
34.  开展点击付费活动
35.  更新您的投资组合
36.  在适当的时候追加销售
37.  研究一下你可以参加的会议和社交活动
38.  报名参加[帮记者出](http://www.helpareporter.com/) (HARO)邮箱列表
39.  举办一场竞赛，并提供免费的“样品”服务
40.  与客户或同事建立战略伙伴关系
41.  撰写并提交一份新闻稿
42.  探索联盟营销
43.  关注口碑营销

在编辑这个列表时，我实际上已经为接下来的几个月制定了自己的迷你营销计划，并将其中的一些付诸行动。你会给这个列表添加什么营销想法？你计划做些什么来加强你的营销活动？
