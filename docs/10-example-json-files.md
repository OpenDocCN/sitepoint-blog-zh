# 在您的项目中使用的 10 个 JSON 示例

> 原文：<https://www.sitepoint.com/10-example-json-files/>

*本系列文章于 2017 年年中重写，提供了最新信息和新鲜示例。*

本文涵盖了您可以在项目中使用的十个 JSON 示例。与曾经流行的 XML 不同，JSON 为不同软件组件和系统之间的数据交换提供了一种更简单、更易读的语法。

与解析和编写 XML 文件的复杂过程不同，处理 JSON 数据既快又容易。大多数现代编程语言目前都原生支持 JSON。

如果你是 JSON 的新手，或者你想提醒自己这是怎么回事，我建议你看看这篇文章:[回到基础:JSON 语法和技巧](https://www.sitepoint.com/basics-json-syntax-tips/)。

## 下载源代码示例

为了跟随示例，您需要设置以下项目，以便访问示例中使用的源代码:

```
git clone git@github.com:sitepoint-editors/json-examples.git
cd json-examples
npm install 
```
