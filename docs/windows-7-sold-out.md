# Windows 7 预购在第一天就销售一空

> 原文：<https://www.sitepoint.com/windows-7-sold-out/>

微软领先操作系统的最新化身 Windows 7 的预购在第一天就销售一空。

该公司允许客户在 10 月 22 日发布之前预订该产品。大多数国家都有很大的折扣。例如，在 7 月 15 日至 8 月 9 日之间订购的英国客户将获得 50%折扣的完整欧洲版*(注意，它需要全新安装，不会从 XP 或 Vista 升级，并且不带 Internet Explorer)* 。

窗口*‘库存’*有限，一直飞下架。大多数零售商在交易的前 8 小时内售罄。亚马逊 Windows 7 的销售记录已经超过了 Windows Vista 整个 17 周的预购量。

Windows 7 测试版和候选版本已经得到了积极的评价。该操作系统稳定、快速，并修复了 Vista 中引入的许多问题。谢天谢地，微软也简化了版本:

*   **Home Premium** 是大多数人会选择的版本。它有标准组件、实用程序和媒体设施。Web 开发人员还应该注意，它包括 IIS web 服务器。
*   **Professional** 包括家庭版的所有功能，加上 Windows XP 模式、更好的企业网络和备份设施。
*   **Ultimate** 包括专业版中的一切，加上数据加密和即时语言切换。

XP 模式是专业版和终极版用户的可选下载。这是一个完全许可的 Windows XP SP3 运行在微软的虚拟 PC 模拟器。然而，新的集成设施允许 XP 应用程序像 Windows 7 系统的一部分一样运行。对网络开发者来说，最大的好处是你可以同时加载 IE6、IE7 和 IE8 的真实副本。[0 日](http://synbay.com/category/0DAY/ "0DAY")

是的，连我都订购了 Windows 7。这是我第一次被迫购买 Windows 的收缩包装版本(而不是通过购买电脑间接购买)。XP 模式是做出这一决定的一个因素。预购价格也有所帮助。但也许我的主要动机只是简单地更换 Vista:我已经受够了缓慢的启动和日常崩溃！

然而，对微软来说，这不全是好消息。在最近对 1，100 名 IT 管理员的调查中，60%的人表示他们的公司没有立即转向 Windows 7 的计划。

你会买 Windows 7 吗？微软的操作系统会成功吗？
