# 自由作家前景看好:报告

> 原文：<https://www.sitepoint.com/outlook-good-for-freelance-writers-report/>

随着世界经济面临潜在的长期低迷，大多数行业都感受到了压力。TechCrunch 裁员追踪系统已经确认，仅在过去两个月里，科技行业就裁员 35000 多人。但是对于自由作家来说，不一定非要这样。

“幸运的是，自由写作是一个在经济好的时候和坏的时候都繁荣的职业，”《InkwellEditorial.com》的出版商 Yuwanda Black 说。“为什么？简单来说，我们生活在信息时代。无论世界上发生什么，信息都需要传播。”

布莱克说，如果一家公司裁掉了内部文案人员，这些文字还是需要传播出去。找到自由写作工作的诀窍是识别热门趋势——磨练你的知识和能力，写人们感兴趣的东西。布莱克认为，识别热门趋势的诀窍是调整你的思维，更多地关注这个世界正在谈论什么。

“我认为有助于自由撰稿人获得更多写作工作的一点是，将自己视为‘信息顾问’，而不是自由撰稿人，”她说。例如，目前自由撰稿人最热门的领域之一是房地产。尽管一些西方国家的房地产和抵押贷款市场出现内爆，但推出有关这些话题的优质内容的机会正在增加。每当有人对缺乏高质量信息的令人困惑的主题感兴趣时，自由撰稿人就有机会利用这个利基市场。

布莱克整理了一份 2008/2009 年自由写作工作报告，确定了自由写作的热门市场，从金融到科技，从健康到色情。这是一份经过充分研究的报告，可能对正在寻找新工作的自由作家有所帮助。

我们不禁会想，布莱克的想法可能也适用于其他领域的自由职业者。公司可能会削减内部员工以节省资金，但他们仍然不得不转向其他地方来完成工作。短期合同工是填补被解雇员工留下的空缺的合适人选。
