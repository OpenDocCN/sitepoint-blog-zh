# 森波选举——一个新的开始？

> 原文:[https://www.sitepoint.com/sempo-elections-a-fresh-start/](https://www.sitepoint.com/sempo-elections-a-fresh-start/)

SEMPO(搜索引擎营销专业组织)成功举行了他们期待已久的选举。在经历了一年多的艰难时期后，最近新一届董事会和官员的选举为 SEMPO 带来了巨大的变革机遇。

SEMPO 过去的困难在其他地方有很好的描述，所以我只会说这对创始人来说是一次真正的学习经历，他们没有运营非营利专业组织的真正经验。坦率地说，这种缺乏经验表现在该集团的许多决定和行动中。

今天森波有了一个重新开始的机会。日常活动现在由一家管理公司负责，这也为最近的选举提供了便利。

新的董事会成员和官员是:

*   Did-it.com 李凯文-主席
*   网站实验室总裁 Dana Todd
*   约翰·桑切斯，尊奇通信-副总裁
*   Jeffrey Pruitt，iCrossing 财务秘书
*   罗恩·贝朗格，克拉互动公司
*   克里斯·丘吉尔，Fathom 在线
*   Webmama.com 芭芭拉·科尔
*   深泽幸一郎，Wasabi 通信公司
*   恩奎罗·戈登·霍奇基斯
*   Mauro Lupi，做水疗
*   杰西·斯特里奇奥拉，炼金术士媒体
*   Advertising.com·朱丽恩·汤姆森
*   大卫·威廉姆斯，360i

我采访了新当选的 SEMPO 董事会成员兼副总裁、总部位于达拉斯的 Zunch Communications 的 John Sanchez(他就住在这条路上，所以更容易得到报价)，他说:

“SEMPO 的新董事会将致力于巩固我们作为全球首屈一指的搜索引擎营销机构的地位。绝大多数 SEMPO 成员密切关注 SEMPO 董事会成员最近的选举。他们在选举过程中的积极参与，以及他们对候选人提出的问题和平台的关注，表明 SEMPO 成员对解决我们组织和我们行业面临的更大问题感到兴奋并准备好了。”

超过 70%的成员公司参与选举，新的董事会有明确的行动授权，而这是最初的(自我任命的)董事会所缺乏的。然而，许多最初的董事会成员回来了，这并不奇怪。
