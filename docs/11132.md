# CSS3 生成器为您编写所有讨厌的代码

> 原文:[https://www . site point . com/css3-generators-write-all-that-pesky-code-for-you/](https://www.sitepoint.com/css3-generators-write-all-that-pesky-code-for-you/)

玩 CSS3 很有趣，但是记住所有这些带有厂商前缀的实验性属性，以及哪个浏览器支持哪个属性，实在令人失望。但是不要担心，CSS3 代码生成器已经解决了这个问题。最近我发现了三个有用的。

[CSS 边框半径](http://border-radius.com/)完全按照标签上说的做；对于基于 Mozilla 和基于 WebKit 的浏览器来说，这是获取`border-radius`属性代码以及标准 CSS3 语法的快捷方式。填入半径值，就可以开始了。同样， [CSS3 渐变生成器](https://www.sitepoint.com/10-best-css3-code-generators/)也可以生成渐变代码。它有一个方便的表单小部件来选择颜色。

[CSS3，求你了！](http://css3please.com/)有好玩的界面；它为您提供了一个样本样式表，其中所有的值都是可编辑的。它涵盖了六个属性，这些属性都有一些现代浏览器的支持，但它也显示了使用专有`filter`属性的合适的 IE 替代品，以及 IE8 的`-ms-filter`版本。

[罗布·艾利森](http://www.rob-allison.co.uk)给我发了一个 [CSS3 生成器](http://css3generator.com/)站点的链接。CSS3 生成器支持九种属性，并为每一种属性提供了针对每种属性的需求定制的个性化表单。它还显示支持属性的浏览器的浏览器图标，以及开始支持属性的版本号。不像 CSS3，拜托！不过，只有在支持 CSS3 属性的情况下，它才会显示 IE 支持。

Style Master 的制作者 Westciv 在他们的 CSS 沙盒中有一系列 CSS3 代码生成工具，包括一个用于[线性渐变](http://westciv.com/tools/gradients/index.html)、[径向渐变](http://westciv.com/tools/radialgradients/index.html)、[文本阴影](http://westciv.com/tools/shadows/index.html)、[框阴影](http://westciv.com/tools/boxshadows/index.html)、[文本笔划](http://westciv.com/tools/textStroke/index.html)和[变换](http://westciv.com/tools/transforms/index.html)的工具。每个生成器都有一个方便的表单界面、一个预览和浏览器支持信息。

所以，去好好玩吧！
