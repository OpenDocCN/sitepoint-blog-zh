# 脸书搞砸了他们的重新设计吗？

> 原文：<https://www.sitepoint.com/did-the-facebook-flub-their-redesign/>

虽然脸书的新设计只向该网站 8000 万用户中的 5%推出，但任何点击[www.new.facebook.com](http://www.new.facebook.com/)的人都可以看到。但是一个潜在的缺陷可能会让一些用户在升级之前三思。但真的是 bug 吗？

网络开发人员兼企业家凯尔·布雷迪(Kyle Brady)周末给我打了个电话，让我了解他在脸书的新设计中发现的一个可能的 bug。Brady 在升级他的帐户后注意到，他的应用程序页面突然挤满了他从未安装过的应用程序，或者以前安装过但后来卸载了的应用程序。据布雷迪称，他现在也很难移除这些应用程序。

出于好奇，我登录了我的脸书应用程序，看看我的列表中是否也有一些奇怪的应用程序。令人惊讶的是，有。脸书列出了至少三个我以前移除过的应用程序，以及几个我不记得曾经安装过的应用程序(尽管一年前该平台刚刚起步时，我收到了 20 封甚至更多的电子邮件，要求我每周审查一个新的脸书应用程序，所以在最初的几个月里，我确实安装和移除了大量应用程序)。不过，我卸下它们没有任何问题。

Brady 回忆说，长期以来有传言说脸书实际上从不删除任何用户数据，它本质上只是注释“删除”的东西，所以渲染引擎不会显示它。原因大概是用户善变，可能以后会改变主意，想要改变决定。

新设计的应用程序页面上打印的一些东西似乎支持了这一理论:“**这些都是你过去添加或登录过的应用程序。**“当然，这也表明这不是一个错误，而是一个功能——新的应用程序页面是你过去添加的任何应用程序的记录，包括你已经删除的应用程序。它不能解释你从未安装的应用程序(尽管我愿意承认，我可能安装了一些短暂“出现”在我页面上的应用程序，只是忘记了)，这可能是一种奇怪的功能——我删除这些应用程序是有原因的。但这是神秘出现的应用程序的一个潜在解释。

升级脸书帐户后，您是否注意到没有添加或删除任何应用程序？
