# 你一直想知道的关于 HTML5 的一切…

> 原文:[https://www . site point . com/everything-you-always-want-to-know-about-html-5/](https://www.sitepoint.com/everything-you-always-wanted-to-know-about-html-5/)

所以 CSS3 让你有圆角，HTML5 给你`<header>`和`<footer>`元素。了不起。有时候，人们很难对加入核心网络技术的东西感到兴奋，但欧内斯特·德尔加多将改变这种情况。

他制作了一个关于 HTML5 中所有新特性的幻灯片(并且，他反复无常地使用术语，将该术语包括 CSS 3 和新的 JavaScript API 特性)。天哪，如果这还不能让你对网络的未来感到兴奋，那就没什么能了。幻灯片本身是一个网页，包含了它所讨论的所有技术，允许例子嵌入到幻灯片中。它是在考虑 Chrome 的情况下开发的，所以许多演示功能仍然不能在 Firefox 中工作，但有一些可以。正如 Ernest 在幻灯片的[免责声明](http://apirocks.com/html5/disclaimer.html)中解释的那样，即使有了 Chrome，一些例子也不会起作用，除非你启用一些实验性的功能。

有客户端存储、地理定位、新的表单输入类型，以及像`@font-face`和`canvas`这样的粉丝最爱。但这还不到一半。Ernest 做了一项了不起的工作，用示例代码和工作实例清楚地解释了所有这些很酷的小花絮。您应该亲自检查一下，体验一下所有的特性，并开始梦想有一天它们都得到很好的支持。为了更容易修改，幻灯片被托管为谷歌代码上的[代码库](http://code.google.com/p/html5-slides/)，所以你可以直接进入并开始播放。
