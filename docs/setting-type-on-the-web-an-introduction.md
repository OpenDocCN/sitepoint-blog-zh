# 网上排版:简介

> 原文：<https://www.sitepoint.com/setting-type-on-the-web-an-introduction/>

我在本系列中的最后一篇文章详尽地介绍了如何将字体发布到 Web 上，涵盖了@font-face 技术的来龙去脉。现在我们可以得到我们想要在网页上呈现的字体，让我们继续漂亮地设置字体。与之前的文章不同，接下来关于更好的美学的文章将被分解成更小、更容易理解的文章。

## 定义和目的

起初，印刷术似乎是一个固有的枯燥领域——处理字体和大小，粗体和斜体，这可能会给人以相当随意或困难的印象:从哪里开始？

定义可以避免混淆，而目的可以让我们看到隧道尽头的亮光。很简单，*印刷术是创造和设置字体的艺术，目的是尊重它设置的文本*。

你已经做了一个“印刷”的决定，为你想做的任何项目选择一个合适的字体。下一步是处理标题、副标题、段落、列表、表格、摘录、引用和它们的许多兄弟(微观)，通过选择它们在页面上的组织方式(宏观:网格、列和边距)。然后，我们可以强调它们之间的关系(排版层次)。

印刷术存在于任何有固定字体(即[活字](https://secure.wikimedia.org/wikipedia/en/wiki/Movable_type "Wikipedia: Movable Type"))的地方:它与金融报告、当地报纸、麦片纸盒、我们床头柜上的小说、宗教经文，以及网络上的“活字”有关。

## 为什么？收获回报

我们每个人可能都很好地理解了什么是好的排版。看看当地的报纸，或者麦片纸盒，或者书架上的书，你可能会列出一系列使它“好”的属性

我们的排版选择的优点是显而易见的，其中最重要的两个是文本的易读性和可读性。

### 易辨认

易读性是指单个字形的可识别性(单个标记*表示*语义字符)。一系列因素影响字形的易读性:笔画、宽度、角度(笔画)、样式(例如，罗马体、全大写字母)、倾斜(整个样式)、颜色(实际颜色和印刷颜色，即对比度)、背景颜色等等。

### 可读性

易读性不同于可读性。可读性指的是整个单词、句子、段落、表格或文本*整体*构成的任何内容的可识别性。一系列宏观因素会影响可读性:度量(行长度)、行距(行高或间距)、对齐或对齐、字体样式、字距调整和字距调整、字体大小等等。

当然，两者是相互关联的；对磅值的改变会影响单个字形和它们组成的单词的易读性。

### 突出关系并提供导航

排版可以给信息带来美学秩序，帮助用户阅读和导航。

就像 WCAG 的( [WCAG 2.0，2.4](https://www.w3.org/TR/WCAG20/#navigation-mechanisms "WCAG 2.0 § 2.4 Navigation Mechanism") )可访问性指南确保超链接可以脱离上下文理解一样，排印层次赋予文本一定的可浏览性。良好的排版层次允许文本作为一个整体轻松导航，这样读者可以再次找到他们的位置，或者跳过多余的介绍性信息，直接进入他们浏览的内容。

一个好的排版层次将为读者提供视觉提示:标题的大小和样式使它们能立即被识别为标题；段落样式统一，首尾分明；引用是可识别的，独立于它们所包围的段落，等等。

好的排版层次结构也将说明页面上元素之间的内在关系:标签被放置在图表、图表和图形附近，通过它们的接近，可以被识别为描述性的符号。

在另一个例子中，想象一个更复杂的双色设计。保存季度利润数据的表格可以采用浅蓝色背景，而旁边详细说明季度亏损的表格可以采用柔和的红色背景，两者的合计在折线图中叠加在一起，利润线采用深蓝色，亏损线采用深红色。图表符号应该提供一个图例，但是颜色也可以用来表示关系。

颜色、字体风格、间距、位置、缩进和一系列其他风格的变化可以用来表示页面上或跨页面的元素之间的关系，以及整个网站的整体关系。

### 获得或未获得的品质

如前所述，我们有意识地(和下意识地)判断文本的排版。回头看看报纸，或者从书架上抽出一本书，描述一下文字的样子。报纸可能很醒目，看起来几乎很廉价，而一本关于社会心理学的书可能整洁有序，有完整的脚注和容易查找的参考资料。

印刷术给文本注入了品质或某种兴趣。一篇文章可能看起来吸引人、优雅、宁静、卡通、狡猾、不平衡和/或乏味，重要的是，这些品质可以是挣来的，也可以是不劳而获的。一个好的印刷工人会谦逊地尊重他们正在排版的文本，尊重文本的意义。糟糕的排版或缺乏细心会让读者产生厌倦感、难以接近感或廉价感。

### 引起兴趣的状态

最后，好的排版能引起潜在读者的兴趣。简单地说，给文本以应有的尊重；如果文本是好的，并且希望被阅读和理解，给它应有的排版尊重。这样做，你会引起一种浏览读者的兴趣。

## 最后

罗伯特·布林赫斯特曾经说过:“精心挑选的词语应该得到精心挑选的字母。”好的网页排版应该通过揭示文本的主旨和意义来吸引读者，并通过统一的层次结构来突出关系，以方便导航。

换句话说，具有好的字体设计的网络体验更容易阅读和使用，也更愉快。
