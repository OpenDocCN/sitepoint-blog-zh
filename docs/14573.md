# WordPress 获奖朋友

> 原文:[https://www.sitepoint.com/wordpress-winning-friends/](https://www.sitepoint.com/wordpress-winning-friends/)

用 PHP 编写的、以 MySQL 为后台的博客应用似乎赢得了越来越多的朋友。

马特·克罗伊登称 WordPress 为[令人印象深刻](http://postneo.com/2003/11/19.html#a3097)；

> 真的很快。真的很滑头。

这引起了 Russell Beattie 的进一步反应: [PHP Web 项目继续给我留下深刻印象](http://www.russellbeattie.com/notebook/1005956.html)。

最近[马克·皮尔格林](http://www.diveintomark.org) [描述了他从](http://diveintomark.org/archives/2004/05/14/freedom-0) [MovableType](http://www.movabletype.org/) 转向 WordPress 的理由，他似乎已经做到了天衣无缝。

从那时起，更多的采用者包括著名的 Java 博客作者；

[Vinny Carpenter](http://www.artima.com/buzz/community.jsp?forum=121&blogger=vscarpente") 描述了一个 [WordPress 装置——第一印象](http://www.j2eegeek.com/blog/archives/2004_06_01_j2eegeek_archive.html#108612756144490493"):

> 我用了大约 4 分钟就完成了整个程序，这是我第一次安装 WordPress。

Java 和甲骨文咨询公司的 Andrej Koelewijn[现在正在使用 WordPress](http://andrej.racchvs.com/archives/2004/05/23/now-using-wordpress/) 。

当然[杰夫](http://www.procata.com/blog/)也在使用 WordPress…

这里有一个有用的博客软件对比；与其他众所周知的应用程序相比，WordPress 更具优势。

WordPress 让我感兴趣的是它的设计决策，比如使用 PHP 本身作为模板语言，支持 HTML 渲染功能。当处理诸如 XML-RPC 和 i18n 之类的“架构领域”问题时，类就发挥了作用。这是一个很好的案例，说明了如何用“普通 PHP”(没有框架)构建一个成功的应用程序。
