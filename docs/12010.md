# 假装直到你成功:自信如何让你成功

> 原文:[https://www.sitepoint.com/confidence-for-success/](https://www.sitepoint.com/confidence-for-success/)

自信是大多数成功人士的共同特征。我们被那些行动自信、流露出自信的人所吸引。我们信任他们；我们想要喜欢他们。

如果你缺乏自尊，并且总是怀疑自己，这对你周围的人来说是非常明显的。事实上，这会让人对你的能力感到不舒服，不确定你能做到你所说的。

即使你在特定的情况下没有完全的自信(总会有让你感到不太安全的情况)，努力展示一幅相反的画面也是有价值的。虽然永远真诚和诚实是至关重要的，但假装比实际上更自信一点也有好处。只要你没有歪曲你自己或你的技能和经验，你就能从展示一个对自己的能力毫不怀疑的人的照片中获益匪浅。以下是您可以从中获益的方式:

客户和同事会信任你。

没有人愿意把重要的责任托付给一个看起来自己不能胜任的人。展示自己完全自信和有能力会让客户和同事更容易信任你，相信你真的有能力。

你会吸引志同道合的人。

自信让你更有可能找到其他自信和成功的人。这不仅会带来新的机会，而且你会发现你自己的目标和计划会变得更大，你会努力做得更多，实现更多。

你将承担更多可预测的风险。

当你自信时，你会发现更多的机会向你走来，你会更愿意冒险。你将学习如何衡量新的机会，并为承担风险制定计划，以增加成功和成就感。

你会开始相信它。

假装的好处是最终你不再需要假装。你将训练自己积极思考，并培养一种积极进取的态度，这种态度将慢慢成为你性格的一部分。那时，你将真正看到自信的好处，无论是在你的事业中，还是在你的个人生活和整体态度中。

但是等等；过度自信也有潜在的问题。一个危险是走得太远会被视为傲慢。你最不希望的事情就是表现得不可接近和冷漠，所以一定要保持克制。

你也想避免超人综合症，在这种情况下，你开始对自己的能力不切实际。这会导致你在描述自己的能力时不够诚实，可能会让你承担超出自己能力的事情。胸怀大志，相信自己可以完成任何下定决心的事情，这是完全可以的，但是你需要控制现实，以避免任何潜在的灾难。

自信让你成功了吗？怎么会？
