# 如何不打销售电话

> 原文:[https://www.sitepoint.com/how-not-to-do-a-sales-call/](https://www.sitepoint.com/how-not-to-do-a-sales-call/)

我今天接到一个电话，是一个销售人员打来的，他自称是一家上市搜索引擎和优化公司。我接受了，因为我需要这个博客的素材。他答应了。

像往常一样，他没有说出自己的名字就开始了通话，只是说了句“你好吗？”

所以我说，“好吧，你是谁？”我已经知道这是一个烦人的推销电话，对方毫无头绪。同样，我的博客需要内容。他继续帮忙。

他告诉我他的公司是做什么的，并开始阅读一个关于他的服务的长篇段落。他告诉我，他是从我的网站上得到我的名字的，而我的网站现在已经离线了。再次，我让他继续，内容。

我打断他。“看，”我说。"关于你们的服务，请随时给我发电子邮件。"

他说，“如果你能让我多告诉你一点……”

我插嘴说我没有时间，也不感兴趣。

他说，“那么，你不希望你的网站有更多的流量吗？”

我说，因为我很烦，“不，现在不行。”

他回答说，那你为什么在线？"

我骂他是个笨蛋——然后挂了电话。所以他也错过了给我发信息的机会。

我想公司雇佣销售人员打这样的电话是因为它有效。但我从未见过这种冷打电话的方法奏效，我和许多尝试过这种方法的公司合作过。

如今，销售电话可以发挥作用，但前提是:

–你从一封清楚传达你的价值的信开始；

–打电话跟进，以简短描述如何帮助他们取得成果开始；

–关注一小部分目标潜在客户，以便在致电时对他们有所了解，并能以有价值的见解定制您的开场白；

–用有价值的信息跟进一段时间以建立关系，并保持耐心。

不要像上面转录的那个可怜的灵魂一样，做愚蠢的，一次性的推销电话。
