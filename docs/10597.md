# 脸书解决四年来最严重的关闭

> 原文：<https://www.sitepoint.com/facebook-shutdown-failure/>

脸书在 9 月 23 日星期四遭遇了灾难性的失败，工程师们称这是自 2006 年以来最糟糕的一次。该公司就这个问题向其 5 亿用户道歉，这个问题在美国造成了将近 3 个小时的停机时间，在欧洲则是中午时分。

虽然脸书还没有透露完整的技术细节，但据了解，这是由一个错误处理错误造成的，造成的损失比它修复的还要多。在估计有 1.35 亿用户试图登录的时候，脸书别无选择，只能关闭服务。大多数人都面临着“DNS 失败”的消息，这个消息很快通过 Twitter 和其他服务传播开来。

对于社交网络来说，这是糟糕的一周。在脸书宕机之前，Twitter 蠕虫利用了一个跨站脚本(XSS)漏洞。然而，这些问题并不相关，网络总体上是可靠的。

但是没有脸书，人们该怎么办呢？停电意味着许多用户不得不与他们真正的朋友面对面。让我们希望没有太多的戳。
