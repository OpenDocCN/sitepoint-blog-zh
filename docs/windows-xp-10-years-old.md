# Windows XP 十岁生日快乐

> 原文：<https://www.sitepoint.com/windows-xp-10-years-old/>

Windows XP 已经 10 岁了。该操作系统于 2001 年 10 月 25 日发布，在 IE6——它的默认浏览器发布后不久。尽管年代久远，XP 仍然是世界上最常用的操作系统之一。它在 2006 年达到 75%的峰值，即使在今天，仍有超过 38%的联网电脑安装了该软件。花了两年多的时间， [Windows 7](https://www.sitepoint.com/windows-7-review-1/) 才超过 XP，而且仅仅领先几个百分点。

“XP”是“Experience”的缩写。这个操作系统是一个雄心勃勃的项目，它将微软的家庭系统(Window 95、98、Millennium)和公司系统(NT、Windows 2000)合并成一个代码库。本质上，Windows XP 将具有稳定性和运行游戏的能力。

很容易忘记最初令人沮丧的几个月。XP 发布时，我在雷德蒙，许多微软员工正在经历安装的麻烦。然而，一旦驱动程序可用，制造商开始支持操作系统，问题就减少了。

很少有 PC 操作系统能达到 10 年的里程碑，那么 XP 坚持下来的主要原因是什么呢？多种因素的结合导致了它的成功:

**1。XP 坚如磐石**
十年的更新和经验已经解决了大部分烦心事。很少有制造商推出没有 XP 驱动程序的新硬件，而且有很多对旧设备的支持。

**2。Longhorn 和 Vista 的溃败**
微软希望 Longhorn 成为 XP 的继任者，如果他们实现了这个目标，它将成为一个革命性的操作系统。但这是 Longhorn 的败笔:它太具有革命性了。

随后的取消和转变导致 XP 和 Vista 之间的 5 年延迟——一个在技术和主流媒体中普遍受到抨击的操作系统。就我个人而言，我不认为 Vista 应该受到所有的指责；假设你有硬件支持，Vista 是一个有能力的操作系统。但是 mud sticks 和 Vista 仍然是微软最大的失望之一。

**3。上网本的崛起**
华硕在 2007 年发布了第一台 Eee PC:这是许多经济型轻便上网本中的第一台，非常适合移动办公的员工。

第一个安装了 Linux，许多人预测它会引领主流接受这个操作系统。微软不能让这种情况发生。Vista 苛刻的硬件要求并不适合这些设备，所以他们别无选择，只能给 XP 重新注入活力。

**4。经济**
我希望你没有注意到，但过去三年世界一直处于衰退之中。企业正在削减成本和人员:当 XP 在普通硬件上运行时，很难证明升级个人电脑和操作系统是合理的。它还将获得微软的支持，直到 2014 年。

**5。像 XP**
这样的用户已经使用 XP 十年了。你可能会觉得可怕，但许多非技术用户放心使用他们知道的东西。

很少有人会认为 XP 比 Windows 7 更好，但新的操作系统并不是一个根本性的改进。Windows 7 看起来很棒，但它不会显著提高你的工作效率。

## Web 开发人员的困境

我怀疑你们中的一些人在质疑我为什么在 SitePoint 上讨论 XP。操作系统通常不会影响网络开发者:如果用户有最新的浏览器，我们很少关心他们的操作系统。

IE8.0 是 Internet Explorer 的最后一个 XP 兼容版本。我们[可能不喜欢它](https://www.sitepoint.com/3-web-wishes-for-2011/)，但微软几乎没有技术或商业理由在他们的旧操作系统上发布 IE9 和未来版本。这些用户可能会转向 Firefox、Chrome、Safari 或 Opera，但许多私人和商业用户在未来几年仍会使用 XP 和 IE8。

在撰写本文时，IE8 是世界上使用最多的浏览器，占有 24%的市场份额。这一数字正在下降，但浏览器不会很快消失——它也不直接支持我们今天采用的许多技术。如果你认为 IE6 是痛苦的，那么想想 2014 年 IE8 HTML5 的发展会有多糟糕。

Windows XP 为我们提供了很好的服务，理应有尊严地死去。然而，它肯定会比微软预期的时间更长。这对我们任何人都没有好处。
