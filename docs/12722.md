# SitePoint 技术编辑可能是你梦想的工作

> 原文：<https://www.sitepoint.com/sitepoint-technical-editor-might-just-be-your-dream-job/>

SitePoint 仍在寻找一名技术编辑加入我们的出版团队。技术编辑的职位总是很难填补；所需的技术和沟通技能是一个独特的组合。但我也认为，填补这种角色的一个大问题是，在网络行业没有多少人会专门考虑进入这种角色。我知道我没有。事实上，在我得到这份工作之前，我根本不知道这种工作的存在。

部分责任可能在于职位名称:“技术编辑”无法与包含“巫师”、“天才”、“大师”或“忍者”等词的网络行业职位相提并论。技术编辑的头衔让这个角色听起来像是给常驻的纳粹分子的。消除那种想法；我们已经有一个单词书呆子了。我们需要的是一个真正的网虫。

如果你是一个人，或者认识一个人，喜欢在网上摆弄东西，只是为了好玩，然后分享你的知识，这可能是一份完美的工作，因为你了解最新的网络技术，并与一群分享热情的铁杆网络极客一起工作。
