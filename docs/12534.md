# 打击垃圾邮件的重大胜利:主要垃圾邮件主机被砍掉

> 原文:[https://www . site point . com/big-victory-in-fight-from-spam-major-spam-host-axed/](https://www.sitepoint.com/big-victory-in-fight-against-spam-major-spam-host-axed/)

![](../Images/0de3a8ef7726dc490aea213b35e68af8.png "broken-picture")[华盛顿邮报报道](http://voices.washingtonpost.com/securityfix/2008/11/major_source_of_online_scams_a.html)一名据称对全球每天发出的高达 75%的垃圾邮件负有责任的美国主机被下线。“在过去的四个月里，Security Fix 一直在从安全行业收集有关 McColo Corp .的数据，这是一家总部位于加利福尼亚州圣何塞的网站托管服务公司，专家称其客户名单中包括一些当今商界最臭名昭著的网络犯罪团伙，”《华盛顿邮报》的布莱恩·克雷布斯写道。

据 MyColo 的主要骨干提供商之一 Hurricane Electric 的营销总监 Benny Ng 称，在 Krebs 撰写的网络欺诈博客 Security Fix 与该公司联系后，该公司决定关闭这个流氓主机提供商。“我们调查了一下，看到了(安全修复)报告的问题的规模和范围，然后说，‘天哪！’一个小时内，我们就终止了与他们的所有联系，”Ng 说。

除了控制产生垃圾邮件的僵尸网络，MyColo 的不良做法还扩展到了其他非法活动，如托管儿童色情网站、管理垃圾邮件和儿童色情网站的支付处理，以及显然窃取了数十万人的银行和信用卡信息的特洛伊木马操作。可以说，他们不会被错过。

然而，不要指望你收件箱里的垃圾邮件数量会减少——或者不会减少很多，也不会持续很长时间。正如我们本周早些时候报道的那样，垃圾邮件仍然是一个非常有利可图的行业，垃圾邮件发送者非常狡猾。他们不会因为一个最大的垃圾邮件中心消失了而简单地消失，他们只会找到一个新的地方来发送垃圾邮件。

网络安全公司 Jose nazar io a[Arbor Networks](http://asert.arbornetworks.com/2008/11/third-bad-isp-dissolves-mccolo-gone/)写道:“随着 McColo 的停播，我不怀疑在未来的几周、几个月和几年里我会无事可做，他们主持的坏事将会消失。”。

一个主要的问题是为什么迈克洛花了这么长时间才被阻止。据安全专家称，他们已经是一个众所周知的罪犯有一段时间了，但当局一直行动迟缓。

“有确凿的证据表明，这种活动已经持续了太长时间，安全社区的许多人已经尽力提高对这个网络的认识，但似乎没有人在乎，”计算机安全公司趋势科技的威胁研究员保罗·弗格森告诉华盛顿邮报。“这是对试图对这些人进行法律起诉的效率低下的一个声明，要做任何事情都需要这么长时间。执法部门说他们已经尽力了，但这还不够。如果执法部门不能及时解决这类问题，那么网络世界中执法的整个概念都需要重新定位，因为它目前几乎没有取得任何进展。”
