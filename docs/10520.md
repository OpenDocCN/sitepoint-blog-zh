# 测量脸书页面影响

> 原文:[https://www.sitepoint.com/measuring-facebook-page-impact/](https://www.sitepoint.com/measuring-facebook-page-impact/)

上周末，我和帕特里克·奥基夫、杰里米·赖特和杰伊·贝尔在博客世界的一个小组讨论题为“社交媒体营销的伦理”。在我们的小组讨论中，出现了社交媒体营销指标的话题。前提是，许多使用不道德做法来增加追随者或“喜欢”数量的营销人员只是试图推动这些数字，因为管理层需要指标，而这是他们所知道的所有跟踪方法。正如我在座谈会上所说，关注者和粉丝数量不是唯一需要追踪的指标，也是最不重要的。*吸引粉丝/追随者*比简单地让一大群人追随你要重要得多。那么，你如何在你的脸书网页上跟踪参与度和互动呢？

### **脸书的见解**

[脸书洞察](http://www.facebook.com/insights/)是一款新的分析工具，可供所有页面管理员使用。它提供了许多因素的详细指标，如跟踪用户增长、活跃用户和用户参与度。让我们看看您应该解决的几个关键领域。Insights 仪表盘中的*月度活跃用户*指标显示了上个月与您的页面互动的用户数量(之前为“粉丝”)以及该指标与之前相比的增长情况。这是一个很好的方法来衡量用户在你的页面上的整体参与度。**用户***用户*部分显示了用户随时间的增长——“喜欢”与“不喜欢”的对比它还会显示一段时间内用户总数的图表。这是衡量一般范围的好方法。 *Activity* 子部分显示页面浏览量和媒体消费，这在查看某些类型的帖子是否更受欢迎时会很有用(例如，照片与视频)。**交互***交互*部分展示了用户如何与你的页面进行交互。*页面活动*子部分显示发布到您页面的提及、墙贴、讨论贴、评论和视频。这证明你向你的用户传达了一个信息——你的页面正在被人们所接受。这是我认为我们需要寻找的地方，以确定我们的页面是否成功。不幸的是，这个工具的新版本不允许您查看历史数据。如果您想要更详细地访问这些信息，您可以从 Insights 工具中导出自定义日期范围的数据，并在 Excel 中查看数据，或者编写自己的程序来解析数据。我们只能希望脸书很快允许通过[图形 API](http://developers.facebook.com/docs/reference/api/) 访问这些信息。

### **旧页见解**

Insights 仪表板最近已被替换，但旧版本仍然可用。当前的仪表板忽略了历史数据，而之前的仪表板显示了一段较长时间内用户或交互的图表。它可以让你跟踪所有的互动，或者只是在一个图表上单独的张贴、评论或“喜欢”。它还能让你看到帖子的质量，帖子的数量，以及每个帖子的互动，这些都是非常有价值的。
