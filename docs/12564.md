# 当搜索引擎成为出版商

> 原文：<https://www.sitepoint.com/when-search-engines-become-publishers/>

上周，我们写了一篇来自 [Compete 的有趣研究。事实上，当你看付费推荐占总推荐的百分比时，](http://www.compete.com/) [在付费搜索方面胜过谷歌](https://www.sitepoint.com/yahoo-actually-leads-in-paid-search/)。然而，正如我们所指出的，这些数据实际上表明了最终用户在 Yahoo！比在谷歌上更糟糕。

原因是有两种类型的搜索。一种是您希望浏览结果并查找某个主题的详细信息，另一种是您希望得到即时、具体的答案。在第一类搜索中，有些人可能会认为三大搜索引擎——谷歌、雅虎。、和 Microsoft Live —性能大致相当。但对于后一种搜索，谷歌似乎是明显的领导者。

搜索引擎通过成为信息发布者而不仅仅是索引者和组织者来提供即时答案。例如，正如我们在上周的帖子中提到的，在谷歌上搜索一杯中的[[盎司]会从内置的单位转换计算器返回一个结果:“一杯= 1.5 美国液体盎司”](http://www.google.com/search?q=ounces+in+a+shot)

这种类型的搜索几乎不可能赚钱，因为有即时答案的用户几乎肯定不会点击广告。但是，显而易见的是，搜索引擎为这些“即时信息”类型的搜索提供的答案越好，用户就越有可能返回该搜索引擎进行更容易赚钱的广泛信息搜索。

谷歌有大量的内置搜索工具。用户可以从谷歌获得天气、股票报价、体育比分、货币兑换、地图、航班状态、字典定义等即时答案。都是雅虎！和微软正在迎头赶上，但它们现在都提供了自己的即时信息搜索工具，令人印象深刻。

雅虎！事实上，可以说，谷歌的榜单在很多方面都比谷歌的更令人印象深刻。他们的“搜索快捷方式”可以做谷歌工具能做的几乎所有事情，以及提供额外的信息，如油价和交通状况。雅虎上称之为快捷方式。，通常在 SERPs 中不如在谷歌上的即时信息结果突出——这可能有助于公众认为谷歌在提供即时信息方面做得更好。

微软要赶上其他两家还有很长的路要走——这一点从他们在搜索市场的第三名，一位数的份额就可以看出来。在本周的一篇博客文章中，Live Search 产品经理 Theo Vachovsky 概述了微软现在已经启用的三种即时信息搜索工具:交通信息、百科查询和星座。它们实际上工作得很好——比雅虎的类似快捷方式要好。(谷歌没有这些领域的搜索工具)。例如，在 Live 上搜索[ [春分](http://search.live.com/results.aspx?q=When+is+the+vernal+equinox)是什么时候]会通过他们的百科全书小工具提供一个明确的答案(下一个是 2009 年 3 月 20 日星期五)。谷歌和雅虎！出现空(虽然雅虎！通过他们的字典快捷方式定义它，谷歌指向顶部有答案的维基百科页面。

当然，正如我们所说，这些搜索引擎提供这些答案的方式是通过*发布*内容。他们正慢慢成为出版商。例如，微软的 Live Search encyclopedia 结果是通过 Encarta encyclopedia(他们拥有的)提供的，而雅虎！美国的词典结果来自美国遗产，谷歌的体育比分来自 STATS，Inc .这些合作伙伴关系会自动将搜索结果的最高排名给予一家公司(或一家内部酒店)。

这比谷歌的知识单元要低调得多，它是一个类似维基百科的用户生成内容的网站，我们在 7 月写了这篇文章，但是搜索引擎与其索引的网站竞争的结果差不多。如果雅虎！可以自动在其搜索结果的顶部显示汽油价格，例如，像 GasBuddy.com 这样依靠搜索流量来吸引游客的网站会发生什么？

七月，杰夫·贾维斯给了谷歌一些关于知识单元的建议，“趁早收手。从一个无与伦比的优势位置与你服务的人竞争不仅仅是一件坏事。是邪恶的。”

即时信息搜索工具也是如此。你有什么看法？
