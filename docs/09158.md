# 对倒卖者的警告

> 原文:[https://www.sitepoint.com/a-warning-to-those-who-resell/](https://www.sitepoint.com/a-warning-to-those-who-resell/)

转售另一种产品或服务是赚取少量剩余收入的好方法。销售足够多的这些副产品，你很快就会看到巨大的利润。

不过，这也有不好的一面:你的声誉岌岌可危。

我以前写过为什么你需要小心你的搭档。选择错误的供应商会给你和你的企业带来不好的影响，即使你可能与这种服务或产品的交付没有什么关系。

上个月，我第一次去了美国。在我住的地方，单程飞行时间超过 20 小时，两周内至少飞行 8 次，这是一个壮举。

其中七次飞行很棒，一次很糟糕。事实上，糟糕的不是实际的飞行，而是灾难发生时我所在航空公司的服务。你看，我因为糟糕的天气被耽搁了，错过了我的转接航班。在已经在空中呆了半天之后，我在午夜到达了一个陌生城市的机场，我真的不想错过那趟航班。

然而，大自然确实碍事，尽管这不是两家航空公司的错，我还是询问了接下来会发生什么。我花了不少于两个小时，在两家航空公司之间进行沟通和谈判，才解决了这个问题。最让我恼火的是，我实际预订并付钱的那家航空公司卖给了我另一家航空公司的服务，这种服务通常被称为“代码共享”，是航空旅行中的一种常见做法。

那很好。然而，当关键时刻到来时，他们不感兴趣了。他们卖给我的航空公司被留下来收拾残局，为我安排另一趟航班。他们很棒——不是我原来的航空公司。

如果你出售托管或域名或任何其他第三方产品或服务，并将其作为自己的，你有一个真正的义务。你已经做出了承诺和建议，如果你的客户带着问题或投诉回来，我希望你拿出最好的客户服务并解决它。

我对大多数小企业的经验正是如此——即使你可能只获得很小一部分利润，他们是你的客户，因此你有责任，而不是把一切都归咎于第三方。

遗憾的是，十亿美元航空公司与你我的感受不同。下次你的客户对你转售的服务有问题时，要敏锐地意识到你是如何处理的——你的反应真的可以成就或破坏一段关系。

我吗？嗯，我上周订了 16 个人的返程机票。猜猜我没用哪家航空公司？
