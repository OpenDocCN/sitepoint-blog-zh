# 网络创始人对网络隐私的担忧

> 原文：<https://www.sitepoint.com/web-founders-fears-for-internet-privacy/>

万维网的发明者蒂姆·伯纳斯·李爵士上周在威斯敏斯特向英国政府成员讲述了他对网络隐私未来的担忧。他特别抨击了一些广告系统越来越多地使用[深度包检测](http://en.wikipedia.org/wiki/Deep_packet_inspection)。

蒂姆爵士对英国政府没有对深度包裹检查采取行动表示担忧。相比之下，美国国会调查了定向广告初创公司 NebuAd，导致其首席执行官辞职。

> 作为一名英国公民和一名美国居民，我感到尴尬的是，美国坚决反对深度包裹检查，而这个国家却没有。

蒂姆爵士补充道:

> 很重要的一点是，你可以在使用互联网时不去想，当我们点击时，第三方会知道我们点击了什么，这可能会影响我们的保险费如何变化，我们是否可以获得人寿保险或另一份工作。
> 
> 允许某人窥探你的互联网流量就像允许一家公司在你的房间里放一个电视摄像头，除了它会告诉他们更多的东西。
> 
> 我觉得使用互联网的行为是我们必须被允许做的事情，而不受任何干涉或窥探。

几家英国互联网服务提供商已经考虑并测试了一个名为 Phorm 的系统，该系统可以分析互联网用户的浏览习惯，从而提供相关的广告。据菲尔·巴特勒上周报道，谷歌也宣布了他们自己的“行为”广告系统。

Phorm 坚称其系统没有侵入性，匿名收集数据，并允许用户选择退出跟踪。其首席执行官肯特·埃图格鲁尔否认了这些攻击，并声称该系统的批评者不了解这项技术是如何工作的。
