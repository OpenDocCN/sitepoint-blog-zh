# 巧妙的人际关系网

> 原文:[https://www.sitepoint.com/an-ingenious-networker/](https://www.sitepoint.com/an-ingenious-networker/)

我刚刚把一些退休基金转给了一位基金经理，他是一位聪明的人际关系网专家。不论职业，我们都可以向他学习。

第一，他以前是网球教练，业务培训很少。现在，30 岁左右的他正迅速成为一名非常成功的资金管理者。这种转变似乎意义重大，因为如果一名网球教练能够转向资金管理，并建立起一个庞大的客户群，那么它会给我们所有人带来希望。

他是这样做的:

1.他加入了当地的网球、壁球和高尔夫俱乐部，并在这些俱乐部中非常活跃。

2.他的网球/高尔夫球友喜欢他，当他在的时候，会对他的工作给予评价。我就是这样了解他的。在周五下午的双打联赛中，他代替了一名正式队员。我的双打搭档谈到了他的公司正在获得的回报，我要了他的名片。我打电话了解更多，他发了一些好的信息。他发送的信息有营销倾向，但也提供了有价值的数据和图表。

3.他非常喜欢社交，喜欢打网球。尽管他是一个比我优秀的网球运动员，但他和我打过几次球，我们已经建立了关系。

4.在适当的时候，他询问了我的家庭，这导致我告诉他，我的父亲一直在寻找一些管理金钱的好主意，所以他在我的允许下给我的父亲打了电话(在我打电话介绍他之后)。

5.当他和我父亲通话时，我姐姐和他在一起，她和他通了电话。

6.他对社区里的一群人重复了这个过程。

他做的任何事情都不会叫嚣着明目张胆的营销。他是一个讨人喜欢的人，会跟进关于投资的好主意和信息，找到建立自己关系网的方法，并迅速建立自己的资产管理。

现在说他是否会获得比整体股市更好的投资结果还为时过早。但他很容易建立关系，并选择体育作为一种引人注目和结识他人的方式。

这里的教训是:

1.人们从他们喜欢的人那里买东西。

2.人们喜欢志趣相投的人。

3.在你可以展示你的技能和能力的地方引人注目，并建立关系。

4.用信息和教育证明你的价值。(这个家伙发出关于不同投资选择的漂亮报告，有大量图表和良好的数据)。

5.跟进。(这家伙说打就打，经常沟通，这是我真正需要的)。

真的不是很难。
