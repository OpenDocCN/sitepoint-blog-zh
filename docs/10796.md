# 屏幕后面有一个人

> 原文：<https://www.sitepoint.com/there-is-a-person-behind-the-screen/>

我有一个朋友是天气预报员。我不知道这是一个国际术语还是一种新西兰主义，但她在电视上看天气预报是为了工作。她一直有一个坚定的规则，从不利用她每天在屏幕上的时间来提及她的私人生活，即使这是传递信息的绝佳机会。我认为这是令人钦佩的。这无疑是一个让你的信息迅速传达给 400 万人的简单方法。

此时，如果您想知道这与 SitePoint 社区有什么关系，那是情有可原的。我确实有道理。我写个人博客已经很多年了，写博客的唯一目的就是把我的想法告诉那些愿意阅读的人。当我开始为 SitePoint 写博客时，我不得不偶尔控制自己，提醒自己这不是我个人的演讲台。但是偶尔会发生一些事情，模糊了这些线，足以让我把它当作一条线来使用。今天就是其中之一。

在过去的一周里，社区里发生了很多事情。随着足球比赛的结束，我们看到了本次杯赛宣传活动的结束——最终我们分发了《jQuery:忍者新手》的 PDF 文件。能够把东西送人总是很有趣的。你得到的善意通常是值得的。但在过去的一周里，我感觉自己花在与愤怒的人打交道上的时间，和花在与感激的人打交道上的时间一样多。

![SitePoint forums](img/5ea7e09e8de0ae2772afe9422f285f6c.png "sitepoint-forums")

这(最终)让我明白了我的观点。在 [SitePoint 论坛](https://www.sitepoint.com/forums/)上，我们就如何相互交流达成了一致。它包括公平和诚实，但尊重。我最喜欢的论坛签名之一包含了一个非常有力的信息。它说“成堆的赞美代替不了一次侮辱。谨慎选择你的用词。”

同样的规则应该适用于任何形式的社交媒体。如果我们做了什么让你烦恼的事，我们希望你能让我们知道。但是互联网提供给我们的匿名水平不应该被用作粗鲁的借口。请记住，仅仅因为你在键盘上输入你的话，而不是对一个人说，并不意味着某人不受这些话的影响。屏幕后面有一个人，人们有时似乎会忽略这个事实。

无论如何，我的咆哮到此结束。我正从讲台上跳下来。
