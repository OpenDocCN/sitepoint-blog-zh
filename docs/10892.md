# 会计头疼吗？也许是时候雇佣一个簿记员了

> 原文：<https://www.sitepoint.com/hiring-a-bookkeeper/>

对于一些自由职业者和小企业主来说，记账是每月都要面对的可怕任务之一，不容忽视。因为你想得到报酬，你不能吹掉它，把它放在待办事项列表中遗忘。如果你想继续做生意，你必须支付你的账单。所有这些都需要时间和一些专业知识。

这就是簿记员的用武之地。雇佣一个簿记员可以弥补一些损失的时间…如果你能找到合适的人，组织起来，把工作委派出去。当你探索外包给簿记员时，有一些事情需要考虑。

## 如何判断记账员是否能帮到你

簿记员并不适合每个人；一些迹象表明，你可能会从簿记员的工作中受益，包括:

![](img/76890b39ac3a7a61f55736eac381289e.png "calculator")

*   您每个月都有大量的发票要生成和发送。
*   你没有时间跟进未付的账单。
*   每个月花在管理书籍上的时间让你不堪重负。
*   您的业务正在增长，但您仍在使用基于纸张的会计系统。
*   你没有完全理解你使用的会计软件。
*   你不能使你的账目一致。
*   你需要帮助收集年底结账所需的信息。

## 簿记员能做的事情

除了接管数字，这样你的眼睛可以不斜视，你可以专注于你的工作，簿记员可以:

*   准备并发送您的发票
*   准备财务报表，这样你就可以看到你每个月的财务状况
*   确保您符合在当地开展业务的所有必要财务要求
*   启动供应商付款
*   正确记录你的交易
*   核对你的账目
*   追踪未付的账单
*   在财政年度结束时，为你的会计师准备好你的文件

虽然雇佣一个合格的、有经验的、值得信赖的簿记员不会让你卸下所有的责任——你仍然需要跟踪开支，为你的簿记员收集和整理信息——但这可以减少时间和压力，因为时间太少，没有兴趣管理你的书籍。让一个合格的、高效的簿记员监督这些数字可以让你安心，这是非常值得的。
