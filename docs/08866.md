### 11.JavaScript MVS 库

JavaScriptMVC (JMVC)是一个 MIT 许可的客户端 JavaScript 工具集，它可以帮助你在最短的时间内构建一个可维护的、无错误的、轻量级的应用程序。它将同类最佳的库和工具打包在一起，保证(并经过测试)能够协同工作。它支持 jQuery 支持的所有浏览器。
