# 宣布 jQuery 标准小组成立

> 原文：<https://www.sitepoint.com/announcing-jquery-standards-team/>

今天，我们很高兴地宣布成立一个新的 jQuery 子团队，名为 **jQuery 标准团队，让 web 开发人员在标准过程中有发言权**。我们都知道网络标准很重要。它们有助于确保我们编写的代码能够跨不同的技术、为不同能力的人工作，最重要的是能够跨所有浏览器工作。

也就是说，我们有多少次感觉到自己的声音、建议和想法被那些负责定义这些标准的团体听到了？事实是，虽然我们中的许多人希望看到变化，但由于时间限制和冗长的正式过程，我们无法参与标准讨论，参与编写规范，并为关于未来功能的会议做出贡献。这让 web 开发者很难有发言权。

![yehunda katz](img/899c2b8af390959a28a68bd36088f7d0.png "yehunda katz")

![paul-irish](img/041fabdb35ff1d5b1e6023850cc61a95.png "paul-irish")

另一个问题是，对于那些参与该过程的人来说，通常感觉参与标准邮件列表中的特定主题影响有限，因为网络社区是如此分散。浏览器供应商在这些列表中非常活跃，几乎所有线程中都包含了大量的机构知识。这些列表上的实现者有他们自己的场所来讨论共同关心的领域，但是希望参与的 web 开发人员没有，除了在会议上偶然的聚会。

jQuery 项目希望能帮助改变这种情况——我们希望你能对如何塑造网络的未来发表意见。
