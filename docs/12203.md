# Microsoft SuperPreview:测试网站的新方法

> 原文:[https://www . site point . com/Microsoft-super preview-website-tester/](https://www.sitepoint.com/microsoft-superpreview-website-tester/)

浏览器测试比 T2 要难得多，但是微软今天在 T4 举行的 MIX09 网络开发者大会上发布的一款新工具可以解决我们的许多问题。

Microsoft Expression Web super preview 允许开发人员同时比较一个页面在不同浏览器中的呈现方式。该工具的概念与 Adobe meer meer meer(由 Josh Catone 于 2008 年 12 月审查)类似，但 SuperPreview 可以作为独立的应用程序使用，并且有免费版本。

SuperPreview 团队访问了客户网站并采访了 web 开发人员，以确定功能集:

1.  Different browsers can be viewed and compared in vertical and horizontal layouts. Scrolling moves all the pages at the same time.

    ![SuperPreview layout](../Images/4792fbff18ba21b96784c4b357ba585f.png)

2.  页面可以以不同的浏览器大小呈现。
3.  像素标尺和整页缩略图有助于页面导航。
4.  PhotoShop-like window guides can be positioned to check element alignment.

    ![SuperPreview ruler and guides](../Images/da0288bd874c8fb0edf3a777c80cd66f.png)

5.  悬停或单击突出显示单个 DOM 元素，并报告 ID、类和维度。

然而，杀手锏是覆盖布局:它允许你在设计模型或任何其他浏览器上叠加一个部分透明的浏览器。渲染差异可以在瞬间发现。

![SuperPreview overlay](../Images/515595dac8bdc3f9b6f9d7d5b4c4f120.png)

SuperPreview 的完整版本将出现在微软 Expression Web 的下一个版本中，并将支持 IE、Firefox 和 Safari。该团队渴望与其他供应商合作，因此随着产品的发展，可能会出现更多的浏览器。

标题为*“用于 Windows Internet Explorer 的 Microsoft Expression Web super preview”*[**现在可以从 Microsoft.com**](http://download.microsoft.com/download/5/6/8/568F0D28-0434-4794-B7FC-FB293BCC98FB/SuperPreview_Trial_en.exe)(250 MB，需要。网)。这是免费产品的预发布版，允许您测试 IE6 和 IE7(或者 IE8，如果您安装了它的话)。几个功能丢失，通常的测试软件警告适用。

SuperPreview 直到现在还没有经过可用性测试。团队会感谢错误报告和建议，所以点击错误图标发送您的反馈。
