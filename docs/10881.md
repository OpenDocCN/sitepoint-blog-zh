# 美国监管机构调查苹果的主导地位

> 原文:[https://www.sitepoint.com/us-regulators-investigate-apple/](https://www.sitepoint.com/us-regulators-investigate-apple/)

美国联邦贸易委员会计划调查苹果的商业行为，以确定该公司是否损害了不断增长的移动应用市场。随着人们对公司主导地位的担忧加剧，该公司正面临诸多挑战:

*   苹果对应用程序开发者的规定禁止谷歌和其他公司在 iPhone 和 iPad 应用程序中销售广告。
*   美国司法部正在调查苹果 iTune 在数字音乐市场的地位。

它最终不得不发生。从某些方面来看，苹果已经超过微软成为世界上最大的 IT 公司，所以不可避免的会有更多的焦点放在它的活动上*(那些可怜的律师必须以某种方式赚钱！)*

与微软在 OS 市场的地位不同，苹果并没有明显的垄断地位。还没有。iPhone 占据了智能手机市场约 30%的份额，仅次于黑莓。然而，iTunes 保留了大约 75%的数字音乐市场。可以说，苹果也完全控制了 iPhone/iPad 移动应用市场，并积极阻止竞争对手的进入。

苹果是否从事反竞争行为将取决于市场规模、苹果控制的比例以及法官对法律的解释。正如我们在以前的 IT 案例中看到的，调查可能需要几个月甚至几年的时间。即使苹果公司*“有罪”，也不一定会有可执行惩罚。*
