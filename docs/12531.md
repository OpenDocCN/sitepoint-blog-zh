# 是时候来点商业头脑风暴了！

> 原文：<https://www.sitepoint.com/its-time-for-some-business-brainstorming/>

以下内容转载自论坛报#421 网站。

前段时间在 [SitePoint 论坛](https://www.sitepoint.com/forums/)， [johnyboy](https://www.sitepoint.com/forums/member.php?u=18056) 提出了一个[【疯狂的支付方案创意】](https://www.sitepoint.com/forums/showthread.php?t=579819)供论坛讨论。它吸引了一些兴趣和一些极好的反应。

johnyboy 的想法是免费为客户建立网站，报酬是销售额的一个百分比。我真的很喜欢他如何运用另类思维为未来企业构建一个不同的商业模式。

但这里有一些来自经历过的人的建议:这个概念可能听起来不错，但实际上这可能是他做过的最糟糕的事情。

几年前，我通过接受销售额的百分比从一个客户那里获得了大幅减少的报酬；这包括网站的开发和营销。第一个月很好，我拿到了工资。第二个月，销售额直线上升，所以客户欠我一大笔钱。方便的是，客户现在看不到我为欠款提供了什么，并拒绝付款。

这件事的结局是，愚蠢的我在一年后同意与另一个客户进行类似的交易。不用说，我又做了一遍！

我从未见过这种安排有效地发挥作用。web 开发者控制不了的因素太多；虽然测量是一回事，实际支付是另一回事！

话虽如此，我仍然鼓励任何人重新审视他们的商业模式或计费和支付方式；事实上，任何可能改变你底线的事情都值得研究。

像 johnyboy 所做的那样，集思广益的想法可以带来创新的实践，帮助您的企业现金流并取得长期的商业成功。
