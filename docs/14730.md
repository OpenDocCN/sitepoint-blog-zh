# ColdFusion 开发者联合起来！

> 原文:[https://www.sitepoint.com/coldfusion-developers-unite/](https://www.sitepoint.com/coldfusion-developers-unite/)

*…发帖人* [大卫·梅德洛克](https://www.sitepoint.com/forums/member.php?u=24385) *:*

非常感谢 Sitepoint 论坛成员 creole 让我知道今晚在纳什维尔 CFUG (ColdFusion 用户组)的会议。我知道纳什维尔有一个 CFUG，但我还没有参加，由于一个非常繁忙的时间表。虽然我今晚不会去，但我计划参加下一次会议，并在这里报告会议的情况。

ColdFusion 用户组一直让我感兴趣，尽管到目前为止我还没有得到他们的支持，因为我一直想知道在我的领域有什么样的 ColdFusion 工作。我注意到今晚会议将涉及的一些话题特别包括就业机会，以及如何面试一个发展职位的人。

作为 ColdFusion 开发人员，采访他人是我们都应该至少了解的事情。我现在已经为两家公司工作过，在那里不仅仅是管理层参与招聘决策。对开发团队的所有成员来说，熟悉一个职位的严肃的申请人是很重要的。你会发现自己每天都在和这些人一起工作，所以我觉得管理层应该允许开发团队一定程度的投入。显然，这不仅仅是针对 ColdFusion 开发人员的。真的适用于任何领域。

无论如何，我真的希望我能听到整个演讲，但是，唉，我晚上已经有安排了。下次也许我会提前知道更多。

如果你想找一个你所在地区的 ColdFusion 用户组，试试 CFUG 的 EasyCFM 列表。是的，巴基斯坦和沙特阿拉伯各有一个，供我所有的中东读者使用。:)
