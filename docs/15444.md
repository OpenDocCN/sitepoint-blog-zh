# 专业 PHP 编程(Wrox 出版社)

> 原文:[https://www.sitepoint.com/php-programming-wrox-press/](https://www.sitepoint.com/php-programming-wrox-press/)

**![Professional PHP Programming](../Images/eadab9f6721b5142d262ad311652fd8c.png) PHP(简称“PHP:超文本预处理器”)是一种类似于 ASP 的服务器端脚本语言。也就是说，它允许您编写嵌入在 HTML 页面中的脚本，这些脚本由 Web 服务器处理，并在发送到浏览器之前转换成普通的 HTML。“专业 PHP 编程”旨在提供 PHP 语言的完整描述，并展示一些实用的技术和示例，以便使用它来构建交互式动态网站。**

Wrox 出版社每本书封面上的口号都是“程序员对程序员”。阅读这本 900 页的厚书时，我很清楚，写这本书的五个人都是职业程序员。这反映在深度报道，在现实世界的例子，以及在写作风格。

在描述 PHP 语言本身的介绍性章节之后，读者将逐步了解在基于 Unix 和 Windows 的平台上安装和配置 PHP 的过程。虽然涉及了许多潜在的设置场景，但作者主要关注在 Unix 下用 Apache 服务器安装它，以及在 Windows 下用微软的 IIS 或 PWS 安装它。也涉及到最流行的可选附加模块的安装。简而言之，该讨论提供了我所见过的最完整的 PHP 安装指南，涵盖了新手用户面临的许多常见陷阱。

随着预备工作的进行，这本书开始对核心语言特性进行极其完整和结构良好的描述。跨越约 200 页，每一个功能都在它自己的一章中介绍，这本书的这一部分在你熟悉这门语言时是一个很好的参考。与此同时，大多数章节建立在相同的实际例子(基于网络的工作申请表)上，更高级的章节开发完整的组件(例如会话对象)。这使得第一次阅读的人可以自然地浏览类似教程的演示，并在最后展示一些真正有用的东西。

##### 高级主题

随着 PHP 的核心在读者的腰带下，这本书然后提出了一系列的 11 章，每一章处理一个 PHP 的高级功能或应用。

一些最有用的包括“PHP 和 SQL 数据库”(提供 PHP 数据库功能的实际操作，主要关注 MySQL API)，“图像生成和操作”(探索 PHP 的特性，允许您编写输出动态生成的 GIF 文件而不仅仅是 HTML 的脚本)，以及“发送和接收电子邮件”(提供 PHP 中演示电子邮件特性的概述和大量示例代码)。这一部分可以被认为是本书的“精华”，因为它提供了大量的实用信息，对于已经掌握了一些 PHP 知识的读者来说非常有用。然而，在我看来，这可能是作者们分道扬镳的地方，他们各自覆盖了自己的专业领域。这些章节，虽然提供了优秀的例子和很大的覆盖深度，在写作风格方面有很大的不同。我发现其中的一两个包含了令人困惑的，有时是不自然的语言。相同的章节经常使用程序员的速记，而完整的解释会更清楚。

本书以四个真实世界的“案例研究”结束，带领读者浏览四个完整的 Web 应用程序的代码:一个购物车应用程序、“Phorum”讨论组软件、一个基于 Web 的电子邮件应用程序和一个数据库浏览器。虽然最好的方法是牢固掌握语言基础，但这几章是让你充分利用 PHP 全部能力的最好方法。

很少有书既能很好地做好“指南”，又能很好地做好“参考”。“专业 PHP 编程”是为数不多的。介绍性的章节，除了 HTML 的基础知识之外什么都不假设，对于第一次接触 PHP(甚至 Web 编程)的人来说是完美的，并且在以后的参考中仍然是有用的。更高级的章节是完善你的基础知识的好工具。附录(尽管有人可能会质疑 60 页的 HTML 参考资料与 PHP 书籍的相关性)是一个方便的快速参考。

**评级:4.5/5**
