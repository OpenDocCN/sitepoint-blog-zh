# Twitter 和生产力:建立联系的 5 种方式

> 原文:[https://www.sitepoint.com/twitter-and-productivity/](https://www.sitepoint.com/twitter-and-productivity/)

有很多关于 Twitter 和一般社交媒体的讨论，以及维持存在所需的时间。不可否认，有效地使用 Twitter 和其他社交网络是很费时间的。

但是，如果你能找到 Twitter 的其他用途，不仅帮助你使用这项服务，还能让你更有效率，那会怎么样呢？这里有几个方法可以做到这一点。

## [![](../Images/10fd245ea1c69303fd00205736d2b215.png "rtm")](http://www.rememberthemilk.com/services/twitter/) 追踪任务跟记奶

如果你有一个[Remember Milk](http://www.rememberthemilk.com/services/twitter/)账号和一个 Twitter 账号，你可以使用直接消息来整合这两个服务。你所需要做的就是在你的任务名或命令前加上“d rtm ”,表明这是一个直接消息，并标记为“记住牛奶”。您可以添加任务、发送任务、检索任务以及以直接消息的形式接收任务提醒。

## [![](../Images/c1b36f8d56b8ad9ddb6248b2f9a6fcca.png "eatbytweet")](http://www.eatbytweet.com/) 用 eatbytweet 获取餐厅特色菜

eatbytweet 将餐厅信息与实时媒体融合在一起，并为用户提供了一种快速找到当地餐厅交易的方式。要搜索餐馆，你只需输入你的位置，你就会看到最新的餐馆推文。如果你找到了感兴趣的列表，你也可以从 tweet 搜索结果中预订一张桌子。

## [![](../Images/4eb48c1a25ee3f82c6ec498d9423ef48.png "twitspend")](http://twitspend.com/) 用 Twispend 管理你的预算

Twitspend 是一款简单的移动现金追踪软件，可以帮助你全面了解你的个人财务状况。你只需通过直接信息将你的收入和支出发送到 Twitspend，确定你是否收到或支出了钱，金额以及你选择的分类标签。您可以查看活动的资产负债表，以帮助您保持在预算范围内。

## [![](../Images/3a17d58074860e1dfe1ec3006ac2dd70.png "drippler")](http://drippler.com) 与您的小工具同步

Drippler 是一项服务，在产品的整个生命周期中，它可以让你快速获得最新的谣言、新闻和更新。通过关注 Twitter 上的一个小工具账户，你可以自动接收评论、价格下降、新发布的产品等等。您还可以使用该服务通过脸书和 RSS 获取更新。
