# 2004 年——开源年回顾

> 原文：<https://www.sitepoint.com/open-source-year-review/>

新的一年的开始，当然是我们回顾和思考即将过去的一年，并为即将到来的一年做准备的时候。在技术领域，这正在以疯狂的速度发生，因为专家们试图提炼过去 12 个月的内容，并可能通过一两个牛眼预测在未来 12 年中获得巨大成功。

我唯一的预测更像是一个自私的愿望——苹果 iPod 将支持蓝牙，包括 802.11b 或 g 无线电功能，并将驱动器容量增加到 120 GB。OS X 可以安装在上面，这个设备将成为一台移动电脑，可以连接蓝牙显示器、键盘和鼠标:一个新的杀手级笔记本！

对于开源社区来说，2004 年的开始有些可怕，因为 SCO 加大了在当时看来可能给 Linux 用户带来灾难的严重的法律诉讼。我们已经看到，关于他们提起的诉讼资金的阴谋论，并没有敲响丧钟；相反，开源世界经历了一个进化时期。也许最令人惊讶的是，我们看到微软试探性地加入开源世界，发布了几个小版本。

这里有一些我认为是 Linux 和开源世界中的重要运动，它们将有助于推进互操作性和市场，并赢得向前发展的新“采纳者”。

##### 实时 Linux

这一年带来了潜在用户数量的巨大增长，因为 Linux 的新用户能够在不经历整个系统构建例程的情况下探索该平台。现场 CD 出现了，各种各样的人——从完全的技术新手到经验丰富的老手——从 CD 启动 Linux，看看他们是否也能加入开源革命。

这些 CD 引导映像对于桌面用户和服务器端管理员都很有价值。然而，Linux 桌面可能是今年最大的话题之一。我个人认为，我们在 OS X 有一个可行的“Linux”桌面，只有当苹果发布低于 500 美元的桌面盒子时，它才会变得真正有价值。

Linux 桌面已经成为如此热门的讨论话题，以至于开源技术小组的主编罗宾·米勒发布了 [Point 和 Click Linux](http://www.phptr.com/title/0131488724) ，该小组包括德高望重的 Slashdot 以及 Newsforge.com、Linux.com 和 Sourceforge.net。这本书的目标读者是任何寻求发现 Linux 和开放源码的用户，包括 SimplyMEPIS live CD 和可安装的发行版。

虽然桌面讨论主要针对公司最终用户，但不熟悉 Linux 并考虑将它添加到他们的产品组合中的设计师和开发人员会发现这本书和任何现场 CD 对于评估都是非常宝贵的。

##### 小说的重生

在我看来，Novell 作为小型到企业级软件市场的重要参与者的重新出现是今年最大的事件。对许多人来说，Novell 品牌让人想起了服务器可以在奔腾之前的系统上运行的尘封已久的旧时光。然而，商界继续将 Novell 的名字与稳定性、可靠性等联系在一起。任何花时间管理 Netware 4.11 的人都会对正常运行时间赞不绝口(以月甚至季度为单位)。

Novell 通过收购 SUSE (Linux 发行版)和 Ximian (Linux 桌面和 MS Exchange server 连接器工具)加入了 Linux 的竞争，这一事实让所有人都知道他们是认真的。这也为 Web 业务中的许多人重新启动或开始向大客户推销基于 Linux 的解决方案打开了大门。

该公司通过率先发布使用新的 2.6 内核的企业 Linux 工具进一步使自己合法化——这一举动显示了从旧的 Netware 世界向 Linux 世界过渡的开发人员的力量。如果这个团队能够将公司自己的操作系统的强大和简单带到 Linux 上，我们应该会看到可用性的巨大收益。这也应该打开所有层次的大门，从小型组织到大型组织，无论企业在哪里寻找开源解决方案。

##### 安全问题

操作系统安全对我们所有人都至关重要。然而，每个系统管理员所需的不同安全级别是不同的。

对于那些寻求增强、加强 Linux 系统安全控制的人来说， [SELinux](http://www.nsa.gov/selinux/) 可能是答案。SELinux 代表安全性增强的 Linux，是美国 NSA(国家安全局)研究项目的成果。它侧重于强制访问控制，提供对用户和设备以及应用程序和服务的强大控制。

SELinux 是作为一组内核补丁发布的，这些补丁包含在现有的 Linux 安装中。美国国家安全局表示，它只在红帽上成功测试了该软件包。

##### 数据库成熟了

数据库当然在 2004 年成熟，不是从开发或生产的角度，而是从商业和开源供应商之间的界限模糊的角度。数据库解决方案现在正以对其专有或非专有性质的漠不关心作为基准。相反，重点是性能、稳定性和可扩展性。像 Firebird、MySQL 和 PostgreSQL 这样的开源播放器继续在高压力的生产环境中变得合法化。

集成了开源方面以前缺少的特性(包括存储过程、子选择等)。)通过成熟的路线图，数据库代表了商品化对开发者有益的应用领域的一个部分。现在，如果商业供应商能够更接近纯粹的 ANSI SQL 遵从性就好了。

##### 版权和知识共享

随着互联网不再被视为一种新奇的技术，而仅仅成为社会的另一部分，版权可能一直是一个令人头疼的问题。关于任何原创内容的合理使用的问题——物理的或智力的——已经成为严重而紧迫的问题。

当消费者、公司和法院争出新的案例法和先例开始解决纠纷时，开发商经常袖手旁观壳震惊，不知道在设计过程中做出什么决定。

Creative Commons 希望通过倡导一种强大的、新的原创作品出版机制来改变这种状况。溢出效益是为那些寻求设计和其他网络项目内容(音频、图像、文本和视频)的人提供的一种新的搜索资源，通过这种资源，人们可以合法地以各种形式和方式整合另一个创作者的原创作品。

这些许可证是为现代内容量身定做的，如下所示，摘自早先一篇关于版权的专栏文章:

选择知识共享许可并不会丧失您的版权；相反，它通过预先批准内容的使用简化了流程。该组织提供四种类型的许可证:

1.  **署名**:其他人被授权复制、分发、展示和表演你的版权作品，以及基于它的衍生作品，但前提是他们提供了你的作者署名。
2.  **非商业性**:其他人被授权复制、分发、展示和执行您受版权保护的作品，以及基于该作品的衍生作品，但只能用于非商业性目的。
3.  **无衍生作品**:其他人仅被授权复制、分发、展示和表演您作品的精确副本，并且没有基于原作者身份的衍生作品。
4.  **相似分享**:您授权他人发布任何衍生作品，但必须获得与您创作的作品相同的许可(类似于开源软件许可)。

##### 新 GNU 2005

由开源倡导者理查德·斯托尔曼领导的自由软件基金会计划在 2005 年修改 GNU 通用公共许可证，这将是 13 年来的第一次修改。随着数字世界将过时的法律和法律哲学推到了临界点，许可证作者也被迫重新评估他们的前提和文本的稳定性和适用性。

FSF 试图保留 GPL 的框架和意图，但正在解决版权和专利实践的全球变化。

经过一段时间的公众意见征询后，该组织将最终确定并发布新的许可证。这不仅对现有的许可证用户是一个福音，而且也可能为犹豫不决的开源用户打开大门，让他们站出来开始探索它的用途。

##### 新领域

在开源的不远的将来，有几个步骤要采取——这些步骤可能会产生许多影响。随着 Linux 在商业上变得更加可行，它的发展道路上将会出现岔路，这可能会改变开源世界至少某些部分的性质。

如果数十亿美元开始涌入金库，竞争变得更加激烈，开源环境会保持其固有的协作性吗？

Linux 和整个开源领域无疑已经在各个层面上加入了日常的技术世界，并且永远地改变了我们工作的环境。我的希望是，互操作性和协作将通过全球标准的方式，在开放和封闭平台之间开始更大规模的传播。
