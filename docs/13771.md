# 为什么你现在应该停止编码和设计

> 原文：<https://www.sitepoint.com/why-you-should-stop-coding-and-designing-right-now/>

许多 Sitepoint 的读者问他们如何才能建立一个有价值的公司，有员工，持续时间比他们参与的时间更长——简而言之，一个他们可以出售的公司。

下面是如何做到这一点:

停止编码。停止设计。停止做一名雇员，开始做一名企业建设者。

就是这样。想象接下来会发生什么:

–你必须找到低成本、高质量的资源为客户服务。你会成为发现优秀人才的专家。

–您必须开发一致的方法来为客户获得结果(例如，克隆您自己和您过去做的事情)。

–你花时间营销公司，建立流程和系统，让公司在没有你的情况下运转。

–在需要时，你可以亲自参与，提供专家指导，设定标准，建立企业文化。

获得客户，获得人才，支持人才，让他们取悦客户:这成为你的新工作。

这并不容易，首先是因为你在建立自己的公司时可能需要一些资金储备。当你不在这个行业工作时，你就放弃了收入。此外，你需要聪明地对待自己的成长——一步一步来，而不是在你需要之前愚蠢地投资全职帮助。

所以，今天解雇你自己。作为一名编码人员、开发人员和/或设计人员参加罢工。开始你的新工作，成为一名建筑商。
