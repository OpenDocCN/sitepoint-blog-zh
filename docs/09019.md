# 伙伴关系:在有人失去一只眼睛之前，一切都是乐趣和游戏

> 原文：<https://www.sitepoint.com/partnerships-its-all-fun-and-games-until-someone-loses-an-eye/>

在我之前的文章中，我写了关于合作关系的利弊。当我们组建自己的团队时，有许多事情我们事先没有计划，但我很幸运，我们友好地分手了，这是因为优先事项和目标的变化，而不是敌意。回顾过去，我可以看到在不同的情况下，事情可能会有糟糕的结局。所以，这里有一些在建立伙伴关系时要考虑的安全建议——在你射瞎你的眼睛之前。

# 表明身份！

适用于合伙企业的法人实体有三种类型:普通合伙企业、有限责任合伙企业和有限责任公司。让我们来看看每一个。

## 普通合伙公司

和独资企业一样，普通合伙企业对个人和企业没有区别。你、你的合作伙伴和你的企业在税收和法律责任上都是一体的。无限的个人责任是不成立公司的缺点之一。每个合伙人不仅要对自己的行为负责，还要对他的合伙人的行为负责。这意味着如果你被起诉，你和你的伴侣的个人资产都处于危险之中。例如，如果一名合伙人在执行公司业务时因车祸造成他人受伤，所有合伙人将承担连带(即个人)责任。

在税收方面，所有的公司收入都作为个人收入传递给每个合伙人，每个合伙人都要纳税。相反，如果企业亏损，合伙人可以根据每个人在合伙企业中的所有权比例，在其个人纳税申报单上申报一部分亏损。

## 有限责任合伙企业或有限责任公司

为了消除普通合伙企业的无限个人责任，两个或两个以上的人可以组成有限责任合伙企业(LLP)或有限责任公司(LLC)。在 LLP 或有限责任公司中，一个合伙人不对另一个合伙人的不当行为或疏忽负责。然而，转嫁税方面与普通合伙企业相同。LLP 和有限责任公司之间有一些细微的区别，所以在决定哪一个法律实体最适合你之前，你需要咨询律师。

# 那不在我的工作范围内

一定要清楚地定义每个合伙人的职责，包括每个人每周要为公司投入多少时间。我知道的一个伴侣关系涉及两个室友。其中一个人每天都在电脑前忙碌，开发他们打算出售的网络应用，而另一个人则在泳池周围闲逛或坐着看电视，而不是寻找客户。当他的其他合伙人就他不够努力的问题找他时，很明显，尽管他是一名经验丰富的销售人员，但他不知道如何寻找新业务，他们也没有明确表示他们希望他这样做。像这样管理不当的期望可以很快成就或破坏一段关系。即使你的伴侣是你的双胞胎兄弟或是你三年级时最好的朋友，也要把它当成一次商业冒险。

# 计划你的逃跑

在建立合作关系时，有一个退出条款是很重要的。正如史蒂文·柯维建议的那样，开始时要考虑到目的。这是建立伙伴关系的一个很好的理念。

退出或买断条款通常适用于两个或两个以上所有者拥有同等股份、同等投资和参与的公司。但即使是普通合伙，起草一份也是个好主意，尤其是在资产分割可能是个问题的时候。如果你和你的合伙人分道扬镳，但仍打算作为独资经营者经营，谁会得到你的客户？如果一个伙伴死了会怎么样？在没有协议的情况下，你可能会发现自己在和一个你从未想过要成为合伙人的配偶做生意。

并不是所有的伴侣关系都以失败告终(我的就不是)，但是最好事先就一方想要退出的情况达成一致。再次，寻求法律顾问。

# 自己赚钱

有时候，看似不利的事情，结果却是有利的。作为独资经营者，你对自己的业务和决策有完全的自主权。不，合伙企业也是如此。这意味着当客户迫使你降低价格时——你很想这样做来赢得投标——拥有一个合作伙伴就成了你的安全阀。不仅仅是你；现在你是一家公司，一家有固定利率的公司。

所以你不再根据你花了多长时间来定价。此外，你参与的时间并不是真正的问题。人们[购买是因为真实或感知的价值](https://www.sitepoint.com/so-what-exactly-is-value-and-how-do-i-use-it-to-sell/)，而不是厂商生产一件东西需要多长时间(如果*是这样的话*，耐克鞋的价格将低于 20 美元)。简单地说，这个网站对你的客户有什么价值？接下来的一年他能从中赚多少？你的成本应该基于这一点，而不是你所花费的时间。但是我跑题了…

我的观点是，当你是合伙人时，你应该有一个标准的收费标准。一个企业既支付它自己(利润)也支付它的员工(工资)，所以作为合伙人，把你自己和企业作为独立的实体对待，即使你们在法律上不是这样。通过计算建立一个基本网站所需的时间，然后乘以你每小时想要赚多少钱，来确定你的盈亏平衡点。这将成为公司支付给你的每个项目的最低金额。一旦你知道了这一点，就把它当作一项硬成本，然后适当地提高它的价格，这样公司就能获利，并把一些钱存入银行。一旦你做到了这一点，当潜在客户开始“谈判”时，你就更容易坚持自己的立场

这些只是建立伙伴关系时要考虑的几件事，所以要明智地计划。分手的痛苦可能类似于离婚。一个好的经验法则是做好最好的打算，做好最坏的准备。
