# 在 Photoshop 中使用剪贴蒙版更改纹理

> 原文:[https://www.sitepoint.com/clipping-mask-in-photoshop/](https://www.sitepoint.com/clipping-mask-in-photoshop/)

从昨天开始，我们学习了如何使用[快速蒙版进行更好的选择](https://www.sitepoint.com/better-selections-in-photoshop-with-quick-masks/)，我将使用我的选择来制作一个剪贴蒙版，并为我们的模特应用一件全新的纹理服装。

我使用和昨天一样的图片，从我们精确选择裙子的地方开始。

![clip_image001](../Images/e1dd21e721fd053eeb5509c51c41f1d7.png)

1.在你精确选择了裙子之后，通过复制选择**图层>新建>图层(或者按 **Ctrl+ J /Cmd + J** )。你的层应该看起来像你下面看到的。**

![clip_image002](../Images/9e7006ce930813558f85aadc538f5a2f.png)
T2】

2.我从(一个很棒的免费纹理图片网站)后的[图片中选取了一些纹理，并将这张——【http://www.imageafter.com/image.php?image=b19metals050.jpg】](http://www.imageafter.com/)复制并粘贴到我的 Photoshop 文件中，这样红色的车库门就覆盖了所有的裙子。

![clip_image006](../Images/aea10da1864e5a6f01d1cb84e01ed060.png)

3.移动车库门层的内容，使其完全覆盖裙子。

![clip_image008](../Images/65ede7103e5d64c2610a6fbff55d9498.png)

5.我们现在想使用服装层来裁剪车库门层。要创建剪贴蒙版，确保选中红色车库门图层，按下 **Ctrl + Alt + G** (windows)或 **Cmd + Option + G** (Mac)。这件衣服现在剪得整整齐齐。

![clip_image010](../Images/621b41be39766de5dc1cb894048880ae.png)

6.现在，为了让它看起来更真实一点，我们需要在图层面板的顶部使用一些混合模式。红色车库门层仍然被选中，点击下拉框查看混合模式可用。(默认情况下，混合模式设置为“正常”)

![clip_image011](../Images/e6797b8f662554f5b0571b4d13dd02fc.png)

我喜欢这个例子中线性燃烧看起来最好的样子，但这是一个好主意，因为根据使用的图像，它们可以给出非常不同的结果。现在，您可以看到服装面料中的褶皱和阴影。

![clip_image012](../Images/daf82f99ff6558b40286344f766dd018.png)

这就完成了图像。我可以看到我已经偏离了裙子边缘的地方，所以我可以回到我的裙子层并擦除这些区域。上面被裁剪的图层会瞬间改变。

我希望你觉得这很有用。这是一种将不同纹理应用于图像或文本的好方法，常用于图形设计和插图。
