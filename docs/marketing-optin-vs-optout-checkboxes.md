# 营销错误:选择加入还是选择退出复选框

> 原文：<https://www.sitepoint.com/marketing-optin-vs-optout-checkboxes/>

你可能已经注意到最近的 BBC 新闻[“欧盟禁止预先勾选的网站框来帮助消费者”](http://www.bbc.co.uk/news/world-europe-15260748)，这是在[网站播客](https://www.sitepoint.com/podcast-134-410-gone/)上特别报道的。新的立法旨在消除隐性收费和成本。欧洲网站将不再能够*有益的*包括额外服务，如保险，客户不得不主动拒绝。这些规定还将涵盖电子邮件促销等服务的注册。

令人遗憾的是，要求立法取缔这种做法。选择加入还是选择退出的争论已经持续了很多年——比网络出现的时间还要长。然而:

**预先勾选的选择框是哑的。**

营销人员想忽悠谁？他们让他们的系统对用户来说更容易吗？最好的情况是，他们希望你——他们的客户——对某个产品或服务如此着迷，以至于你会积极地想要选择加入，一个预先选中的框可以让你省去一次点击。但是，如果你真的被打动了，难道你不乐意花几毫秒点击一个框吗？

不幸的是，在大多数情况下，预先选中的框会把用户视为低能儿。该公司希望你没有注意到小字，这样你会支付更多的钱或收到促销材料。

然后，我们看到了试图用糟糕的隐喻来欺骗用户的可疑做法:

*勾选此处选择退出*

或者更糟，那些使用难以理解的语言:

*如果您不想退出我们的简讯，请取消勾选此框*

最后，当你在初次提交时出错时，许多表单会重新检查选择框。

如果你沉迷于这种做法，你只是在给顾客提供糟糕的服务。他们要么支付了比预期更多的费用，要么现在收到了来自您公司的垃圾邮件。这对任何客户关系来说都不是一个好的开始。也许这将导致最小的短期收益，但重复购买和推荐的可能性要小得多。除非你的商业模式与众不同，否则这会让你的长期声誉和前景面临风险。

公司经常忘记人们已经变得聪明了。许多客户预计会遭遇隐性收费或可疑的营销技巧。如果你不这样做，你已经领先你的竞争对手一步。

顾客是人，每个人都欣赏开放和诚实:

1.  使用清晰简洁的语言。
2.  给他们你最好的价格。
3.  告诉他们一些选择，但不要假设他们是必需的。
4.  为选择加入的人提供退出信息。

例如:

*每月给我发简讯(你可以随时退订)*

不难。然而，我怀疑在未来的许多年里，我们仍然会讨论欺骗性的营销活动。也许是时候说出我们发现的最糟糕的例子并让它们蒙羞了？
