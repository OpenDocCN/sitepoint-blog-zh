# 让游戏来找你

> 原文：<https://www.sitepoint.com/let-the-game-come-to-you/>

我记得当迈克尔·乔丹和拉里·伯德处于巅峰时，人们常说“比赛是他们自己找上门来的”

他们做得更少，却获得了巨大的成果。不知何故，他们能读懂法庭并对形势做出反应，同时也创造新的形势。

我希望你的营销也是如此。市场营销中的“游戏来找你”意味着:

–客户联系你，你就不用追了。

–你被认为是你所在市场的专家。

–当你接触潜在客户时，他们会看到你的价值，并乐于与你交谈。

这与大多数营销策略相反，在营销策略中，你会觉得自己努力得太多，但收效甚微。

你如何让游戏来找你？

–通过教育和信息材料(演讲、文章、研究)引人注目。

–发展和培养积极主动的转介网络和系统。

–用让你与众不同的方式描述你的业务，让你拥有特定的优势(并让夸夸其谈的客户代表你做同样的事情)。

我喜欢为 Sitepoint 写作，因为读者体验的范围非常广泛。你们有些人理智上理解以上，但没有经历过。你们中的一些人已经是“走向”的专业人士，知道游戏向你走来是什么感觉。而你们中的其他人则介于两者之间，有时感觉你在挣扎，有时感觉一切都向你走来。

另一种说法是推和拉的区别。使用推送营销，你会觉得你在推一堵墙。很快就累了。通过拉动式营销，你可以吸引客户。即使当你使用“推动”技巧时，比如给潜在客户打电话，你也是以这样的方式做的，即潜在客户想接你的电话，和你说话，看看是否合适。
