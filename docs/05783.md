表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏表示比例变化，即上个月有 6.5%的 IE8 用户更换了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

同样，我们看到了一个非常稳定的月份。Safari 和 Opera 略有上升，Chrome 基本保持不变，IE 和 Firefox 略有下降。也许这并不奇怪；所有的现代浏览器都是很好的应用程序，并且在很大程度上可以互换。你不太可能找到别处没有的杀手级功能。我们的用户选择采用 Chrome、IE11、Firefox、Safari 还是 Opera 有关系吗？

这个月我把 Safari 分成了桌面版和 iPad 版。平板电脑非常受欢迎；他们负责二十分之一的网络访问量和超过一半的 Safari 市场份额。也就是说，iPad 用户几乎没有选择；Safari 是唯一合适的选择 AppStore 中的大多数其他浏览器都是覆盖 Safari 框架的[皮肤](http://allthingsd.com/20120628/googles-chrome-for-ios-is-more-like-a-chrome-plated-apple/)。

## 2014 年 2 月至 2014 年 3 月全球移动浏览器统计

移动使用率几乎每月增长 1%，已经达到所有网络活动的 25.46%。

热门移动浏览应用:

| 移动浏览器 | 二月 | 三月 | 变化 | 亲戚 |
| 机器人 | 26.44% | 25.76% | -0.68% | -2.60% |
| 苹果手机 | 20.88% | 21.49% | +0.61% | +2.90% |
| 铬 | 11.95% | 13.56% | +1.61% | +13.50% |
| Opera Mini/Mobile | 13.70% | 13.05% | -0.65% | -4.70% |
| UC 浏览器 | 11.30% | 11.17% | -0.13% | -1.20% |
| 诺基亚浏览器 | 5.64% | 5.22% | -0.42% | -7.40% |
| 不变的 | 2.20% | 2.19% | -0.01% | -0.50% |
| 黑莓 | 2.22% | 2.12% | -0.10% | -4.50% |
| 其他人 | 5.67% | 5.44% | -0.23% | -4.10% |

继上个月超越 UC 浏览器之后，Chrome 以 1.6%的涨幅超过 Opera，成为第三大最常用的移动浏览器。虽然这对传统的 Android 浏览器产生了一些影响，但它们的综合统计数据仍然令人印象深刻。

图表下方，黑莓已经跌破 IEMobile 一个并不以其广泛成功而闻名的平台。下降幅度令人震惊——几年前，几乎三分之一的智能手机用户拥有黑莓手机。它说明了公司在移动市场上不能自满。
