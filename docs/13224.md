# 网络方向北:早鸟定价即将结束！

> 原文:[https://www . site point . com/we B- directions-north-early-bird-pricing-ending-soon/](https://www.sitepoint.com/web-directions-north-early-bird-pricing-ending-soon/)

[![Web Directions North](../Images/36b652aebabf611e6c062a8314cadc55.png)](http://north08.webdirections.org/ "Web Directions North") 明年 2 月在温哥华举行的[【Web Directions North】](http://north08.webdirections.org/)买票的[特价](https://secure.webdirections.org/wdn08/)，还有一周多就结束了。

虽然我总是对每年在悉尼举行的最初的 Web Directions South conference 情有独钟，但不可否认的是，它的姐妹会议——现在是第二年——有着最令人羡慕的项目之一。凭借庞大的[令人难以置信的演讲者阵容](http://north08.webdirections.org/speakers/)和[为期两天的优秀研讨会](http://north08.webdirections.org/workshops/)，WDN 无疑是网络专业人士要参加的*会议之一。*

但最重要的是，这次会议打出了王牌，为演讲者和其他与会者提供了为期两天的滑雪或单板滑雪。

SitePoint 将作为会议的赞助商出席，许多 SitePoint 的最新产品将面向与会者出售。此外，Cameron Adams(几本 [SitePoint](https://www.sitepoint.com/books/cssdesign1/) [书籍](https://www.sitepoint.com/books/javascript1/)的作者)将举办名为[JavaScript 前沿](http://north08.webdirections.org/workshops/#adams)的研讨会。如果你已经从这些书中学习了你的 JavaScript 技能，你将会享受这一整天从蓝衣人那里学到的东西。

温哥华可能有很棒的滑雪和滑板，但是澳大利亚有美丽的海滩。哦，该死，温哥华也有——听起来像是举行会议的完美地点…
