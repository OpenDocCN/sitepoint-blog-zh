刷牙。捡起你的袜子。整理床铺。洗盘子。生活中充满了许多做起来并不特别有趣的琐碎工作，但是如果你不做，事情就会变得很糟糕。维护一个网站也是一样。

这里有一组 WordPress 维护清单给你。定期这样做，你的网站将会快速安全。忽视它们，坏事可能会发生:

*   你的网站可能会加载得像糖浆一样慢。
*   重要的功能可能会分解成上百个部分。
*   您的访问者的计算机可能感染了恶意软件。
*   你的网站可能会被搜索引擎除名。

所以保持你的保养。经常做，用不了多长时间。

每个人的站点都不一样，你的维护重点也不一样。我给了你每日、每周、每月、每季度/每半年和每年的清单。请根据您的需要随意调整它们。比起一口气列出季度清单，你可能更喜欢每个月勾选两三个项目。

最后，如果你没有动力去做你的 WordPress 日常工作，尝试一个插件，比如[维护清单](https://wordpress.org/plugins/maintenance-checklist/)来简化事情。

说的够多了。现在开始工作！

## 您的日常维护清单

1.  正常运行时间。访问您的网站，确保其正常运行。如果正常运行时间至关重要，你可以启用来自监控服务的短信通知，如[正常运行时间机器人](https://wordpress.org/plugins/uptime-robot-monitor/)、 [WordPress 监控插件](https://wordpress.org/plugins/sitewarranty-verified-seal-trust-seal/)或[超级监控](https://wordpress.org/plugins/website-monitoring/)。
2.  **备份。**对 WordPress 文件和数据库进行每日异地备份。使用类似 [BackUpWordPress](https://en-au.wordpress.org/plugins/backupwordpress/) 、 [Backup to Dropbox](https://en-au.wordpress.org/plugins/wordpress-backup-to-dropbox/) 或 [BackupBuddy](https://ithemes.com/purchase/backupbuddy/) 这样的插件来自动完成这项工作。一些主机提供商如 [SiteGround](https://www.siteground.com/go/article-sp) 会对你的网站进行每日备份，所以你不必担心手动备份。
3.  **每日安全报告。使用网站安全监控服务(如 [Sucuri](https://wordpress.org/plugins/sucuri-scanner/) )通过电子邮件向你发送每日状态报告，并在可疑情况发生时发送短信。每天检查报告，必要时立即行动。**

## 您的每周维护清单

1.  **评论适度。**批准您审核队列中的任何评论，并检查您的垃圾邮件列表中是否有误报。如果你得到很多评论，你应该每天都这样做。
2.  **更新。如果有新版本，更新 WordPress 以及你的插件和主题。 [SiteGround](https://www.siteground.com/go/article-sp) 自动为所有客户端执行更新，以确保它们拥有最新版本。**
3.  **恶意软件扫描。**扫描任何受感染的文件或恶意软件。同样，如果你的主机像 SiteGround 一样执行自动的每日恶意软件扫描，你的主机会有额外的帮助。如果他们没有，你可以使用类似于 [Sucuri Security](https://wordpress.org/plugins/sucuri-scanner/) 、[防弹安全](https://wordpress.org/plugins/bulletproof-security/)或 [Wordfence](https://wordpress.org/plugins/wordfence/) 的插件。

## 您的每月维护清单

1.  **目视检查。**目视检查您的网站，检查布局或格式是否有问题。使用不同的网络浏览器，包括 Internet Explorer、Firefox、Safari 和 Chrome。从移动设备上查看您的网站，以确保其外观良好且响应迅速。
2.  **验证备份。**确保您的备份文件在指定的位置，并执行恢复。您不想在最需要的时候发现您的备份策略被破坏吧！
3.  **分析。**登录[谷歌分析](https://analytics.google.com)查看趋势、推荐和页面流量。考虑如何使用这些信息来增加你网站的流量。

## 您的季度/半年维护清单

1.  **更改密码。**更改您的密码，并让其他用户也更改他们的密码。不要让黑客轻易得手——使用强密码。
2.  最小化管理员用户。删除或降级不必要的管理员用户(并删除默认管理员用户名)。被黑的管理员帐号会对你的网站造成最大的损害，所以尽量减少数量。
3.  删除不用的插件和主题。停用并删除所有不必要的插件和主题，尽管永远不要删除最新的默认主题。不必要的插件会增加网站的开销和漏洞的风险。
4.  **删除不必要的文件**网站杂乱可以隐藏漏洞。检查您的 *wp-content* 文件夹中不必要的文件，浏览您的媒体库，删除任何您不再需要的图像、视频和音频文件。您可以使用类似[删除未使用的图像](https://en-au.wordpress.org/plugins/dnui-delete-not-used-image-wordpress/)或[媒体清理器](https://wordpress.org/plugins/media-cleaner/)的插件来自动删除未引用的图像。在删除任何东西之前进行备份！
5.  **修复死链接。**您不希望访问者在点击链接时收到“页面未找到”的错误信息。使用链接检查工具，如[断链检查器](https://wordpress.org/plugins/broken-link-checker/)或[链接检查器](http://validator.w3.org/checklink)找到任何死链接。
6.  核实你的联系方式。用你网站的联系方式给自己发一条信息，并确保你能收到。
7.  优化你的数据库。删除任何你永远写不完的草稿，清空你的垃圾评论。然后通过运行类似于[WP-优化](https://wordpress.org/plugins/wp-optimize/)、[删除修订版后优化数据库](https://wordpress.org/plugins/rvg-optimize-database/)或[WP-数据库管理器](https://wordpress.org/plugins/wp-dbmanager/)的插件来优化你的数据库。
8.  **速度审核。**使用 [Pingdom](http://pingdom.com) 或 [Google PageSpeed Insights](https://developers.google.com/speed/pagespeed/insights/) 查看你的网站加载速度。如果耗时超过五秒，考虑实施缓存和其他措施来提高站点速度。慢速网站会失去访问者和搜索排名。

## 您的年度维护清单

1.  查看您的“关于”页面。查看您的“关于”页面，查看是否有任何需要更新的内容:您的联系方式、个人资料图片、使命陈述、评价。
2.  查看你的主题和插件。检查性能优于您当前使用的插件的新插件。寻找吸引你并适合你需求的新主题。
3.  查看你的网站。与公正的朋友坐下来，让他们探索你的网站。一双新的眼睛可能会发现你忽略的问题。更新网站页脚的版权日期。
4.  **站长工具。**登录[谷歌网站管理员工具](http://www.google.com/webmasters/tools)并解决任何错误信息。
