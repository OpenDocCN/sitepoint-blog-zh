# 网站所有者的工作永无止境

> 原文：<https://www.sitepoint.com/site-owners-work-never-ends/>

当我完成我的第一个网站时，我以为我已经完成了“最困难的部分”。在我看来，剩下要做的就是将网站提交给几个搜索引擎，然后等待流量的涌入。几天后，当“洪水”变成涓涓细流——或者更像是零零星星的几滴水——我开始意识到困难的部分还没有结束；事实上，这才刚刚开始！

完成一个网站可能看起来很难，但与努力推广和改善网站相比，这根本不算什么…而且这些事情是一个持续的过程。我们将首先讨论一些与促销相关的领域。

这将是很好的，如果你有一个一步一步的指示，当谈到网站推广。完成步骤 A、B 和 C，然后很快！成千上万的游客！你永远不必做另一件事来推广你的网站！很好，没错，但事情不是这样的。

大多数网站管理员在完成他们的网站后的第一步是提交给搜索引擎和目录。然而，很多时候提交是不够的。你必须不断检查你的排名，以确保你不会下滑。如果你发现你的网站在排名中慢慢下滑，你需要采取行动来纠正这种情况。还有其他搜索引擎优化方法和工具，如网关页面，需要几乎不间断的监控才能成功。为了制定一个搜索引擎游戏计划，你还必须留意哪些关键词被用来找到你的网站，哪个搜索引擎给你带来最多的访问者，等等。

横幅广告应被视为不仅仅是一次性的促销手段。您必须监控哪些横幅广告在点击率和转换率方面最成功，并不断完善您的横幅广告，以帮助提高这些数字。您还必须不断寻找新的和/或更好的网站来放置您的横幅。

开发一个互惠链接程序(或只是一个链接程序)也是非常耗时的。你必须搜索出网站和联系业主要求链接。这是在你的网站生命周期中很有可能要做的事情。将会有大量的时间用于“检查”你的链接伙伴的网站，以确保你的链接仍然是活跃的(这对于像 [WebTrends](http://www.webtrends.com/redirect/14ASJP.htm) 这样的良好跟踪程序或像 HitBox.com 这样的免费服务来说并不难做到)。

关于链接，我没有看到太多讨论的一个方面是，会有一些链接到你的网站，而你不想链接到你。例如，我发现一些成人网站在他们的网站上放置了我的网站的链接。它所需要的只是一封要求网站删除链接的电子邮件，它已经被处理了，但是它仍然需要时间来研究和要求删除。

我们已经讨论了网站推广正在进行的方面，现在是时候转换一下话题，进入一些随着时间的推移改进网站设计的事情了。

网站设计的某些方面，你只有在有很多访问者的情况下才能完善。当然，有些公司在打开网上门户(可以这么说)之前有能力进行广泛的研究和测试，但是对于一般的网上企业家来说，研究和测试信息是在网站上线后以网站访问者的电子邮件的形式出现的。

基于那些访问过我们网站的人的建议、抱怨和观察，我们知道保留什么，调整什么，失去什么。电子邮件对网站管理员来说是有价值的。不要忽视许多访问者表达的共同问题。你可以从访问者的反馈中学到很多，并且比你自己去评估优点和缺点更能改进你的网站。不用说，这也是一个持续的过程。

我希望我已经在这篇文章中向你展示了一些网站设计和推广是一个持续过程的方法。不要对摆在你面前的所有工作的前景感到气馁。开发和维护一个伟大的网站是很辛苦的工作，但是回报绝对是值得的！祝你好运！
