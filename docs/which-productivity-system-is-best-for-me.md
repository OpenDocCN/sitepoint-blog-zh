# 哪种生产力系统最适合我？

> 原文：<https://www.sitepoint.com/which-productivity-system-is-best-for-me/>

![A Mac and a pair of glasses](img/7b7a59b8a1a75d6a35e6b31a546ad90e.png)

作为企业家、企业主或自由职业者，有数百个领域需要你的关注。要测试的产品，要写的文档，要开发票的客户，要建造的原型。这本身不会是一个大问题，但是你已经说服自己**所有**这些任务都很重要，它们**所有*都必须完成，现在*完成**。

追踪这份永无止境的精神“待办事项”清单令人疲惫不堪。再加上每天都要开灯的压力，投入精力去管理你所有的“必须完成这件事”的行动是很困难的。

我是来告诉你，有更好的办法。你需要正确的生产力系统。

现在,“生产力”这个词并没有让人们充满惊奇和兴奋，但是它对于建立、管理和发展一个企业是必不可少的。这个目标会让你充满惊奇和兴奋。一个好的生产力系统是一个使能器，你可以用它来理解什么需要被完成，从混乱中带来秩序，让你免于太多的不眠之夜。

事实上，一个好的效率系统可以让你把原本担心任务的精力用在完成任务上。

## 你所说的生产力系统是什么意思？

生产力系统实际上就是一套指导方针，它们共同帮助你完成工作。他们通常会使用几种技巧，这样你就能明白你需要做什么，什么时候需要做，以及它有多重要。不要把生产力应用或软件工具误认为生产力系统——生产力系统通常是“软件不可知的”

## 但是不知道用哪个生产力 App？

有[loads](https://www.sitepoint.com/5-time-tracking-tools-web-addicted-freelancers/)[great](https://docs.google.com/spreadsheets/d/1yP6ZOXMWK-KsMZ-_Bod8tFhyEGdPndCu4HUBO_3rUww/edit#gid=0)[pieces advice](http://lifehacker.com/5924093/five-best-to-do-list-managers)[和](http://newinternetorder.com/best-to-do-list-manager-compared/) [reviews](https://en.wikipedia.org/wiki/Comparison_of_notetaking_software) 来对比生产力应用和工具。但是事情是这样的——工具本身经常强迫你按照它们的方式做事，所以你最终可能会使用一个对你来说并不完美的生产力系统。最好先确定一个系统，然后再找一个支持它的软件工具或者 app。

这里有各种风格的软件和生产力应用——适合极简主义者、项目经理、视觉思考者、细节规划者等等。一旦你决定了一个好的生产力系统，尝试一些应用程序。但是首先，决定适合你的**系统**，而不是软件。

## 一个好的生产力系统的原则是什么？

在我们开始研究具体的系统之前，让我们先探讨一下好系统的一些原则。简而言之，一个体面的生产力系统应该:

*   **无摩擦** —应该很容易把任务放进去，移动任务，组织它们，然后看看你下一步需要做什么。
*   **适应你想要的工作方式** —生产力系统的主要阻力是它们不适应特定的工作方式。一个好的系统应该支持你想要的工作方式，而不是相反。
*   **易于学习** —你应该能够在学习一个好系统后的一个小时内掌握并开始使用它。
*   **让与他人合作变得容易**——随着世界变得更加互联，以及[团队变得虚拟](https://www.sitepoint.com/how-to-get-the-most-out-of-your-freelancers/)，生产力系统需要适应这种情况。
*   **可以与其他系统一起使用** —生产力系统可以一起工作，许多人使用系统的组合来创建适合他们的工作流程。

## 有我需要避免的陷阱吗？

是的。这些年来，在使用了一些不同的生产力系统后，肯定有一些地方你需要小心:

*   在你选定一个特定的系统之前，尝试几个系统，看看什么对你有用。建立一个系统需要时间和精力，所以在承诺之前要确保它是值得的。
*   一旦你致力于一个特定的系统，不要陷入认为另一个“可能更好”的陷阱——在系统之间移动东西需要做很多工作。
*   请记住，略有不同的系统可能有利于略有不同的事情。不要试图让你的生产力系统变得完美。思维导图是一种不同于待办事项清单的做事方式，但两者都同样有效。
*   不要试图创造“完美”的系统。够好就是，嗯，够好。
*   要明白充分利用生产力系统需要时间、纪律、努力和承诺。只有你坚持使用它们，它们才会起作用。

## 有哪些主要的生产力系统？

有几个。我将逐一介绍最受欢迎的选择，给你一个他们如何工作的快速概述，建议他们适合的人的类型，并让你知道利弊。最后，我将链接到一些资源，您可以在那里找到更多信息。

同样需要注意的是，这些系统都不是“独立的”它们都可以与其他生产力系统、工具和应用程序相结合。

### 把事情做好——一个严肃的生产力系统

把事情做完，或者更普遍的说法是“GTD”，是真正启动了整个“生产力运动”的系统它是由管理顾问大卫艾伦创建的，作为一种帮助高管管理日常业务的方法。从那以后，这个系统被逐渐完善，它可能是当今世界上使用最多的生产力系统，仅次于不起眼的“待办事项”列表。

**基本概念:**

1.  把你生活中每个方面需要做的事情都收集起来。
2.  将所有这些“东西”放到一个“可信系统”中，这样你就可以跟踪和管理它们。
3.  将较大的项目分解成较小的、可行的步骤。
4.  为你需要做的每件事创建任务，问这样的问题“下一步我能做什么来推进这件事？”
5.  任何在特定时间到期的事情，放在日历上。
6.  其他的都变成了“下一个动作”。
7.  为你的每一项任务创造情境，比如“在家”、“办公室”、“购物”等等。
8.  每周回顾你所有的任务、项目和下一步行动。

正如你所看到的，GTD 可能有点让人不知所措，但是对于那些使用它的人来说，它是一个坚如磐石的系统。它可能比大多数其他系统需要更多的训练，但是它非常强大。

GTD 适合:不介意花时间学习一个好的系统，并且有规律地使用它的人。

**GTD 不太适合:**那些不能坚持其指导方针的人，或者那些没有充分利用系统的纪律的人。

### 看板——一个可视化的生产力系统

看板是由敏捷项目开发框架推广的可视化生产力系统。这是一个很棒的团队项目管理工具，因为它让团队成员可以轻松地协作并管理他们之间的工作流。它还可以用于个人任务管理(Trello 是一个为看板设计的应用程序的好例子)。

**基本概念:**

*   看板是基于板子、清单和卡片的想法。
*   每个板块代表一个整体区域或项目(例如，创建一个网站，改造房子，等等。).一块板上通常有几个列表。
*   每个列表代表项目生命周期中不同的部分(例如，待办事项、进行中、测试中、已完成)。一张单子上通常有几张卡片。
*   每张卡片代表一个特定的行动(例如，为网站选择一个技术平台，获得粉刷房屋外部的报价。)
*   当您或其他团队成员处理任务时，他们将任务从一列移动到另一列，因此每个人都知道每个任务在任何时刻的位置。

看板适合:在团队中一起工作的人，视觉思考者，以及想要一目了然地了解所有事情进展情况的经理。

看板不太适合:喜欢严格计划和控制的人。

### 不要打破链条——一个强迫症式的生产力系统

不要打破链条是一个非常简单的生产力系统，据说是杰瑞·宋飞使用的。它的美在于它的简单和它能给人的动力。

**基本概念:**

*   确定一件你想逐渐改进的事情(例如，每天写博客，一个你想养成的习惯)。
*   买一个大的挂历，或者一个在线日历应用。
*   每天，为培养你的习惯或长期目标做些事情。
*   如果你在那一天做了一些事情朝着你的目标前进，在日历框里打个 X。
*   尽可能长时间地保持 X 的链。

就是这样——简单，但有效。

**不打破链条有利于:**想要建立长期习惯或者想要逐步达到一个目标的人。

**不要打破链条不太适合:**日常任务管理、项目规划或传统的待办事项技巧。

### 番茄工作法——保持超级专注的生产力系统

番茄工作法几乎可以和任何其他的生产系统完美结合。这种技术的想法是将工作集中在非常有成效的时期，每个时期之间有短暂的休息。这是完成任务的一种极其简单而有用的方式。

**基本概念:**

1.  买一个计时器(这项技术的名字来源于西红柿形状的厨房计时器——番茄)
2.  决定一项你想做的任务
3.  将你的时间设定为 20-25 分钟
4.  在那段时间里，专心工作，不要分心，检查手机，回复信息，浏览社交网络
5.  在这段时间结束时，休息五分钟
6.  那是番茄
7.  从第 3 步重新开始
8.  当你连续吃了四个番茄，休息 20 分钟

Pomodoro 适合:那些容易分心、想要一种方法来真正完成任务的人。

**Pomodoro 不太适合:**已经纪律如炼钢的人！

### “待办事项”列表——一个基本的生产力系统

待办事项清单其实就是最简单的生产力。它是所有其他系统的基础。

**基本概念:**

1.  把你需要做的每件事都写在一个清单上。
2.  将你的清单按优先顺序排序。
3.  先做最重要的事情。
4.  随着其他事情的到来，将它们添加到列表中。
5.  当你完成任务时，把它们划掉。

待办事项清单适合:那些想要一个快速、简单的系统来记下事情并完成任务的人。

**待办事项列表不适合:**多步骤项目，更复杂的任务，一次组织大量不同的任务。

### “必做”清单——一个基于简单任务清单的生产力系统

“必须做”系统也是一个非常简单的生产力系统。这有助于帮助人们区分他们真正需要做的事情的优先顺序。

**基本概念:**

1.  从你的待办事项清单或你需要完成的任务集合开始。
2.  专注于你绝对**必须*完成的事情*今天**。
3.  把它简化成一个关键任务——你下一步要做的是什么？
4.  写下来。
5.  完成那个任务。
6.  如果你在一天结束前完成了任务，选择下一个最重要的任务。

**必做清单适用于:**难以分清主次的人。

**必做清单不太适合:**那些已经很清楚自己任务的紧迫性和重要性的人。
