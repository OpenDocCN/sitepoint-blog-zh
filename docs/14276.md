# 如何启动一个有试用期的在线网站

> 原文:[https://www . site point . com/how-to-kick-start-a-online-site-with-a-试用期/](https://www.sitepoint.com/how-to-kick-start-an-online-site-with-a-trial-period/)

这个博客谈论付费网站，以及如何让他们开始。这是基于我自己的经验，我把我的营销和销售材料授权给一家专门从事这类工作的在线出版公司。我的简历中列出了这个地点——[http://www.attractnewclients.com](http://www.attractnewclients.com)。

我们刚刚推出了这个网站，尽管还处于早期阶段。开发该网站的公司的首席执行官在推出付费订阅内容网站方面有丰富的经验，我从他那里学到了很多关于如何启动新网站的知识。以下是一些初步经验:

1.完美是你的敌人。当网站“足够好”时启动它然后让尽可能多的人访问网站，获取他们的经验，并改进网站。我们已经计划在本月内进行一系列重大变更，这些变更将允许我们对该计划收取更多费用。

2.建立一种从用户那里获得无情的、无情的反馈的方式。朋友和家人往往会说，“干得好！看起来棒极了！我们为你感到骄傲！”你不认识的人会说，“这样不行。你需要马上这样做……”这很痛苦，但有反馈循环并从那里改进是很重要的。

3.永远不要停止测试。该公司正在测试网站的许多细节，尤其是定价、不同的外观和感觉、不同的功能和不同的主要优势。他们正在以一种系统的、近乎科学的方式优化客户体验。

4.不要提供免费的 beta 测试，而是给付费的早期用户一个折扣试用期。我们发现免费的测试版不起作用。是的，消费软件公司向软件开发者和超级用户提供免费的 beta 测试。但那不是我们的观众。我们需要从小企业主那里获得反馈，他们很可能是真正的客户——那些在网站中看到足够价值来购买和使用它的人。我们发现，通过免费测试，我们得到了许多学生(不是我们的市场)，没有信用卡的人(肯定不是我们的市场)，以及“轮胎踢球者”，他们实际上永远不会为服务付费，但喜欢探索免费的东西(同样，肯定不是我们的市场)。但是通过付费测试(顺便提一下，像 IBM 和 CSC 这样的公司在测试新技术时使用)，我们可以让客户参与进来。

因此，该公司以较低的入门价格(最初研究表明目标市场愿意支付的 1/3)设立了一个试用期。此外，我同意免费赠送我的营销手册(价值 195 美元)，并提供一对一的支持作为激励(也是从人们那里获得坦诚反馈的一种方式)。最后，这些“早期采用者”将会看到网站是如何随着时间的推移而发展的，这对一些人而不是所有人(那些对 web 开发感兴趣的人)都有好处。

我会随时通知你结果。到目前为止，在仅仅一周的时间里，我们已经收到了来自人们的出色回应——尽管我们已经一遍又一遍地测试了该网站，并试图优化用户体验。我们通过我的名单、公司的名单，并通过要求我们的供应商向他们的客户名单发送报价来联系他们。

如果你是一个销售在线产品的网络开发人员，上面的指导方针是启动你的在线网站的好方法。它们提供了一种从朋友和家人之外的真实客户那里获得快速反馈的方式，并建立了一个用户群。
