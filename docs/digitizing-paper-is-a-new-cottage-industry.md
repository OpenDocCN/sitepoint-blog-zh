# 数字化纸张是一个新的家庭手工业

> 原文：<https://www.sitepoint.com/digitizing-paper-is-a-new-cottage-industry/>

如果你像我一样，你有一个装满纸张的文件柜。健康保险单、公用事业账单、汽车维修记录、税单等。它堆积得很快，跟踪它是一件痛苦的事情。对于员工分散在全球各地的虚拟团队来说，成堆的纸张带来的痛苦可能会更明显。在过去的一两年里，出现了一些初创公司，帮助你将纸质文件数字化，并在网上进行分类和存储。

## 邮件

2006 年推出的 [Earth Class Mail](http://www.earthclassmail.com/) 提供所谓的“远程邮件服务”，他们已经为此筹集了 1330 万美元。Earth Class 为个人和商业用户提供了一个“数字邮件室”，由一个真正的邮政信箱组成。当邮件到达时，它被分类(根据收件人)，扫描，在线存储并通过电子邮件发送给适当的收件人。

网络初创公司 [Gliffy](http://www.gliffy.com/) 有一个虚拟团队，成员分布在 4 个不同的大洲。公司首席执行官克里斯·科尔哈特说，通过[使用地球级邮件](http://www.gliffy.com/blog/2008/08/04/handling-snail-mail-in-a-virtual-office/)，他们已经能够简化邮件处理。铺天盖地而来的邮件，现在变得容易管理多了。

## 收入

当许多人想到收据时，他们会想到《宋飞正传》中的那一集，乔治·康斯坦萨的钱包因商店收据而变得肥胖，最终导致他背痛。Shoeboxed 由杜克大学的一名大四学生于 2007 年 1 月创建，并于当年 7 月下旬推出，它为公司和个人扫描收据，这样他们就再也不用面对一个鼓鼓的钱包了。最终，Shoeboxed 的目标是一个完全没有收据的世界。

“例如，在未来，如果你有一张鞋盒卡，你可以在商店刷卡，他们就可以通过电子邮件发送到你的鞋盒账户，这不是很好吗？”公司副总裁丹·英格兰德在六月告诉华盛顿邮报。目前，该公司正专注于将纸质收据数字化，并充当网上购物电子邮件收据的存储点。

## 其他单据

最近推出的 Pixily 被《波士顿环球报》称为“邮箱革命”,它可以扫描文件并将其存储在亚马逊 S3 服务器上的一个私人安全账户中。用户也可以上传 PDF 和 Microsoft Office 文档。

如果你想公开你的文档，并且不介意旁边有广告，Scribd 提供了一种免费扫描文档的“ [Paper to iPaper](http://www.scribd.com/paper) ”。

## 好处和挑战

这些服务的主要好处非常明显:更少的混乱和方便。不用开车去邮局取你的蜗牛邮件，你可以通过电子邮件得到它。你可以在网上搜索带标签的数字化版本，而不是在你的文件柜里整理去年 10 月的商务晚餐收据。

如果这些公司中的任何一家成功地说服世界印刷纸必须被淘汰，也有潜在的长期环境效益。

然而，也有一些主要的障碍，两个主要的障碍是隐私和可靠性。例如，我为什么要信任一家公司，让它保管我的个人医疗保健或税务文件呢？我如何知道我的私人文档不会被读取或共享，并且当它们存储在云中时是真正安全的？此外，如果没有我的纸质备份，如果公司倒闭了，我所有的记录会怎么样？如果我必须保留纸质备份，数字化的一个主要好处(节省空间)就没了。

尽管如此，这些服务还是很有趣，而且似乎抓住了一个独特的市场机会。对于许多没有在全数字时代长大的人来说，将他们的论文数字化的好处是显而易见的，但实际上他们并不愿意自己去做。对于虚拟企业来说，保持一切虚拟化有真正的好处，这将使这些服务值得投资。
