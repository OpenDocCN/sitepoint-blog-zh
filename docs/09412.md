# JavaScript 开发人员的 10 个有用工具

> 原文:[https://www.sitepoint.com/10-tools-javascript-developers/](https://www.sitepoint.com/10-tools-javascript-developers/)

JavaScript has proven that it has to be taken seriously because it can really enhance the web! People are starting to see it’s value is not to provide cheap effects but real web solutions. In this article, I’ve compiled 10 useful tools for all JavaScript developers.

## 1\. FitText.js

FitText 是一个非常有趣的工具，它允许根据父元素的大小自动调整文本的大小。只要浏览一下网站，调整一下你的浏览器:文本会适合你的。现代网站和应用程序的另一个非常有趣的工具！

![FitTextjs.jpg](../Images/4c916ad819fec9c6e4a91e3ca40973c5.png)
[来源](http://fittextjs.com/)

## 2.Ligature.js

不出所料，这是一个可以给任何文本添加漂亮连字的脚本。这是所有排版爱好者的必备之物！
![Ligaturejs.jpg](../Images/dd722bb374efd63e6f7f7cc51ef161be.png)
[来源](http://code.google.com/p/ligature-js/)

## 3.夜明珠

一个超快的异步条件资源加载器，允许你只加载你的用户需要的脚本。
![YepNope.jpg](../Images/758bcc13934eff576e4850b82fe52337.png)
[来源](http://yepnopejs.com/)

## 4.现代化 zr

是一个脚本，它可以帮助老版本的浏览器像最新版本的浏览器一样工作，因此你可以构建在 IE6 和 ie7 上工作的现代应用程序。你的客户会喜欢的，我保证。
![Modernizr1.jpg](../Images/be056c7142b9b198beb031947da2831f.png)
[来源](http://www.modernizr.com/)

## 5.Respond.js

用于最小/最大宽度 CSS3 媒体查询的快速&轻量级 polyfill(适用于 IE 6-8 及更高版本)。
![Respondjs1.jpg](../Images/1d17d7249203328126a371c19dfc832a.png)
[来源](https://github.com/scottjehl/Respond/blob/master/respond.min.js)

## 6.heatmap.js

是一个 JavaScript 库，可用于基于您的数据使用 html5canvas 元素生成 web 热图。热图是完全可定制的——欢迎您选择自己的颜色渐变，改变其不透明度，数据点半径等等。
![heatmapjs1.jpg](../Images/e30c37904a62e8f26ba46ec63d72e177.png)
[来源](http://www.patrick-wied.at/static/heatmapjs/)

## 7.jQAPI

是一个以更加用户友好的方式提供 jQuery 文档的网站，所以现在当我需要任何 jQuery 帮助时，它是我的参考网站。
![jQAPI2.jpg](../Images/8536a65cc36fab6e12c31fb9c26b19f4.png)
[来源](http://jqapi.com/)

## 8.书签生成器

正如你所猜测的，这个工具是一个 bookmarklet 生成器:只需粘贴你的常规 JavaScript 代码，按下按钮，你就会得到一个 bookmarklet——准备安装在你的浏览器栏上。
![Bookmarklet-Generator2.jpg](../Images/419c4322fd1d166a44ace37483ca508b.png)
[来源](http://benalman.com/code/test/jquery-run-code-bookmarklet/)

## 9.Diva.js

是一个用于查看文档的 JavaScript 前端，旨在与数字图书馆合作，将多页文档作为单个连续的项目呈现。它被设计成与 IIPImage 服务器一起工作，对于那些在图书馆或书店网站上工作的人来说，它将是一个非常棒的工具。
![Divajs2.jpg](../Images/bb235dafb2519676a6ec7b4955a34387.png)
[来源](http://ddmal.music.mcgill.ca/diva/)

## 10.MicroJS

是一个全新的网站，旨在为大多数常见任务提供 JavaScript 代码片段:Ajax、JSON、DOM、面向对象的 JavaScript 等等。一个真正的金矿在你的书签里！
![MicroJS1.jpg](../Images/a1c654dbc64cd3d2fd84b6312c36b6c0.png)
[来源](http://microjs.com/)
