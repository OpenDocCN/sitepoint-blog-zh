# 如果你的电脑被淹没(或其他同样毁灭性的灾难)，如何保持你的理智

> 原文：<https://www.sitepoint.com/how-to-backup-computer/>

本周早些时候，在一起涉及一个 8 个月大的孩子、一个打开的水瓶以及注意力和声音判断严重失误的事件中，水溅到了我的 MacBook Pro 的键盘上。

幸运的是，我很快从心脏病中恢复过来，足以关机，将电脑倒过来并寻求帮助(感谢我所有的 Twitter 朋友，他们分享了他们的专业知识和同样痛苦的被淹没的电脑故事)！

这是一次严肃的现实检查，它迫使我确认所有的制衡措施，以便在灾难发生时保持我的业务和理智。幸运的是，我通过了测试。此外，电脑昨晚启动没有停顿，我活着告诉它。

当然，有几个小问题和许多重新学习的课程。所以我想我应该与你分享一些，作为一个回顾，如果你曾经面临被淹没或以其他方式被毁坏的计算机的潜在灾难，希望会有所帮助。

## 确保您的数据已备份，并了解备份计划。

我认为现在每个人都备份他们的数据。但是你多久检查一次你备份的文件以确保数据是正确的呢？您多久检查一次文件同步前的时间间隔？

作为一个超常的人，我有一些不同的备份，本地的和在线的，我经常使用它们，所以我知道数据是完整的和可用的。然而，在拔掉插头之前，我没有时间去验证那天我所做的一切是否都被同步了。比较文件并找出丢失的内容只是一个小小的时间消耗。

## 拥有第二台可以工作的电脑(并拥有您需要的东西)。

如果你的主机出了问题，你打算如何继续工作？您是否有一台装有类似软件的备份计算机，可以访问您备份的文件？

我每天同时在一台 PC 和一台 Mac 上工作，所以我可以直接转移到我的 PC 上，继续工作，只有一个小的停顿。但是，如果我还没有经常使用我的电脑，转移到一个新的机器将需要一些时间。如果你有一段时间没有启动备份计算机，并且在启动之前必须运行更新和其他维护，那么这将花费更多的时间。

## 有办法访问你的“现在”数据。

有趣的是，当有什么事情让我们分心时，我们很快就忘记了自己在做什么。如果你倾向于运行多个应用程序，打开窗口，大脑进程突突前进，你几乎不可能在不同的时间在不同的机器上找到你离开的地方。

这是我在设置所有备份和应急计划时忽略的一个方面。我没有一个能够在“事件”发生的确切时间访问我正在处理的确切信息的计划

例如，我的收件箱充当我的跑步提醒列表。虽然我可以访问我所有的电子邮件，但我不知道我在我的电子邮件客户端中标记、分类和标记了哪些邮件。另一个轻微的头疼问题在我的 Mac 恢复正常后很快就解决了，但如果它死机了，情况可能会严重得多。
