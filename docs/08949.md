## 采访记录

路易斯:大家好，欢迎来到 SitePoint 播客的另一集。我本周节目的嘉宾是一位来自澳洲的网络开发人员，他是一个非常酷的小型网络初创公司 Pygg 的技术负责人，这个公司叫 p-y-g-g，位于安迪·怀特。嗨，欢迎来到这个节目，安迪。

安迪:嗨，路易斯，谢谢你邀请我。

路易:你好吗？

安迪:是的，很好，和以前一样忙。我目前正与另一个本地团体 Ninefold 一起将 Pygg 服务迁移到一个新的云托管环境，所以这让我一直熬夜。

路易斯:是的，你以前是 EC2 的吗？

Andy: 我工作的公司 Pygg 是一家名为 Pollenizer 的初创企业孵化器的一部分，我们已经在许多项目中使用了 AWS 和 Rackspace，Rackspace 是云，我们正试图合理化大约九倍，因为我们的目标受众通常是澳大利亚的初创企业，降低延迟是一个巨大的胜利。

路易:对。因此，让我们稍微回顾一下，讨论一下 Pygg 是什么。那么，您是否想简单介绍一下这款产品的背景和用途。

安迪:是的，当然。因此，我的意思是 Pygg 的电梯推销是，我们正试图扰乱社会支付，我们正试图改变人们相互传递资金的方式，长期目标是扰乱人们与他们的钱和他们的银行互动的方式。

路易斯:酷。让我们把它从电梯间搬到一楼。

安迪:是的。

路易斯:这将是如何使用它，我用它做什么。

安迪:现在的实现是，如果你有一个 twitter 或电子邮件账户，你可以注册 Pygg，你可以向任何有 Twitter 或电子邮件账户的人汇款。目前它只在澳大利亚，但我们确实有走向全球的长期计划。

**路易:**对。你说只有在澳大利亚，因为你把钱取出来的方式是到澳大利亚的银行账户，对吗？

安迪:没错。处理国际货币兑换是一个相当复杂的过程，当你谈到如果我作为一个用户付给你一美元，当你收到它时意味着什么，以及我们实际上如何在界面和消息方面将它呈现给用户。我们尝试了一个早期的实验，它很快变得非常复杂，所以我们的策略是先在澳大利亚市场工作，同时证明我们的想法，一旦我们真正有了我们的腿，然后我们会跳出来，看看潜在的美国市场或其他地方。

**路易:**对。这里的想法是，我发一条推文给 Pygg，还是我公开发一条推文给我理论上的收款人？

安迪:所以，我们运行一个 Pygg 账户@pygg，p-y-g-g，有一段代码在那里监听这个账户上的 Twitter，你向这个账户发送一条 tweet，说你想把钱付给谁，你想付多少钱，然后这个小机器人会给你一个回复，要么说它成功了，要么邀请你注册或者做任何需要做的事情， 因为推特的工作方式而收到钱的人也会收到通知，因为他们在推特上被提到了。

**路易:**对。如果他们没有 Pygg 帐户，那么他们的下一步是不是会收到 Pygg 的另一个回复，告诉他们，嘿，你的帐户里有一些钱，你可以登录，创建一个帐户，然后取出来？

安迪:当然。对于我们发现的一些用户来说，这可能有点令人不安，如果他们没有被简要介绍 Pygg 是什么，他们倾向于认为这是某种垃圾邮件设置。

路易斯:对，没错。

安迪:但是，是的，我们为没有账户的人保留资金，一旦他们注册，资金就会立即出现在他们的账户上。

**路易:**对。为了寄钱，我在 Pygg 创建了一个账户，用 PayPal 或信用卡预付，这样我的账户里就有钱了，然后我就可以用 Twitter 把钱寄出去。

**安迪:**正确。

**路易:**对。当每个人都只有信用卡而没有现金时，我绝对可以看到一个在分割晚餐账单中的用例。

Andy: 这就是我们内部经常使用它的原因。咖啡之旅是另一个大事件；我们已经开发了一个新的工作流程，人们基本上只是走出办公室，说，“嘿，伙计们，我要去喝杯咖啡，”甚至不会停下来接受订单，当他们来到当地的咖啡店时，订单已经出现在他们的 Twitter 时间表中，他们已经为他们准备好了钱，所以他们只是根据他们的 Twitter 时间表当场订购咖啡，然后带回来。

路易:对，太好了。

安迪:再也没有骗人的便利贴了。

路易斯:(笑)是的。不要再为弄错顺序找借口了。

安迪:没错。

路易斯:所以，也许可以稍微深入一点技术细节，因为我很好奇，我想我们的很多听众会对这个项目的极客一面感到好奇，但也许在那之前，你在这方面工作了多长时间，它对公众开放了多长时间？

安迪:我想我从八月份开始就一直在做这件事，所以我们已经接近六个月了，不到六个月了；我是一名程序员，我的数学显然不是很好。

路易斯:(笑)迭代一个变量，直到你得到正确的量，然后—

安迪:对，没错，就是这样，快失败。它可能已经对公众开放了，但我认为其中的七周，前七周我们只有非常有限的公开测试版，我们很快就上线了。

路易:听起来很快。

Andy: 是的，Pollenizer 模型是指我们至少有一些东西可以在两周内测试，通常我们只有 12 周的时间来启动和运行业务，并证明它是可行的，这是一个非常紧迫的时间框架，我相信你可以想象。所以早期的迭代在 PHP 上构建得非常快，我们一直使用 Zend，这可能不是我的选择框架。

路易:听起来很恐怖。我不想冒犯观众中的任何人，但请继续。

Andy: Zend 在这个世界上肯定有它的位置，在我看来，一个快速转变的创业公司可能不适合像 Zend 这样的公司，但从另一方面来说，如果这是你所知道的，并且你愿意与之合作，那么这可能是你最安全的选择。

Louis: 嗯，是的，特别是如果你有两周的时间来启动和运行某个东西，如果你从一个框架开始，你不知道要花两周的时间来学习它。

安迪:没错。所以，是的，目前我们正在经历，我们开始分离应用程序，我们正在用面向服务的架构重建它，所以我接下来几周的主要任务之一是将大部分业务逻辑从 web 应用程序中分离出来，放入一个 API 层，它将介于数据库和 web 服务器之间，长期目标是最终使人们能够编写自己的应用程序，与 Pygg 进行交互。

**路易:**对。所以你确实简单地提到，你有 12 周的时间来证明这是一项可行的业务，那么这里的商业模式是什么，你对每笔交易收费吗？

安迪:是的。因此，人们之间的实际交易是完全免费的，可能我们希望永远都是这样，当你把钱存入系统时，我们收取象征性的费用，我认为目前是固定的 2.50 美元。

**路易:**对。

Andy: 其中一部分只是用来支付我们处理交易的成本，我们目前的目标甚至不是盈利，而是围绕盈利能力和业务运作方式制定长期计划，但第一步只是确保我们有人们会使用和乐于使用的东西，商业模式可以从这一点继续下去。

路易:对。用户的情况如何，你看到了相当多的兴趣吗？

安迪:是的，事实上，相当大的兴趣。用户往往会分成一两个阵营，他们会看一眼，要么是因为我们没有正确解释它，要么是因为他们不想使用它，或者在某些情况下因为我们正在使用 PayPal，他们会很快离开，或者他们开始使用它并开始广泛使用它，在悉尼这里有一些非常有趣的用户群，其中一两个人已经参与进来并爱上了它，对它充满热情，然后带着他们所有的朋友一起去。因此，在某些情况下，我们确实看到了网络效应，这非常令人振奋。

路易斯:是的，那太好了，如果有人展示了它的用处，并用它来支付朋友，这种事情就会突然蔓延开来。我的意思是，你看到像 FourSquare 这样的东西发生了这种情况，它们甚至不是特别有用，所以人们会希望如果某个东西实际上服务于一个目的，人们就会到处都是它。

安迪:是的，没错。我想这里的困难是，它并不总是有用的，直到你真的有其他人在使用它，所以我们接下来几周的任务，或者至少是我们客户开发团队的任务，是尽量降低进入门槛。从长远来看，我们希望 iPhone 应用程序可以使这一点变得更容易，但我们将看到我们如何去做。

**路易:**对。你谈了一些你一直在使用的技术，所以你用 PHP 和 Zend 构建了它，如果你在处理金钱，你在处理人们从安全和隐私角度来看都很重要的潜在问题，你必须与银行和 PayPal 互动，有什么特别的挑战吗？你们面临的挑战或你必须处理的最重要的事情是什么？

Andy: 是的，你知道我们很早就必须决定是否自己处理信用卡交易，你知道我们认为这太冒险了；任何时候你需要获取信用卡的详细信息，或者甚至与商家网关进行交互，你最终都要做一些事情，这样做是非常复杂的。因此，我们已经通过将责任推给其他人和其他组织来减轻这一点，我们可以尝试保持事情良好、精简和敏捷，所以我们实际上不存储任何信息，我的意思是我们存储电子邮件地址，但我们不存储任何比你的银行账号更敏感的信息，在某些方面，这可能有点-我的意思是人们对将他们的银行账号放在那里感到有点可笑，但同时很难滥用它。

路易斯:对，没错。我觉得你用别人的银行账号唯一能做的就是给他们钱。

安迪:对。当然，好的，很好，给我更多的钱；你可以随意滥用它。因此，我不会说这方面存在很多技术挑战，但这绝对是一个考虑周全、尽量减少风险并尽可能保持敏捷的过程。但是，是的，就跟踪交易的一致性而言，确保如果我向你支付 5.00 美元，这是数据库中反映的内容，这是我工作的很大一部分，你知道像这样的产品有很多测试，因为就像你说的，你在处理金钱，其次，我们得到一个错误的交易或金钱从某人的帐户中消失，而没有向他们解释，这显然对我们的用户非常不利，我们绝对不希望这样。

路易:对。在出现问题的情况下，你除了依赖你的支付处理器之外，还依赖 Twitter，支付处理器非常好而且稳定；你是如何发现在重复推文的情况下，这会导致任何问题或类似的事情吗？

Andy: 的确如此。早期，我们的一个开发人员假设一条推文总是 a)按顺序发送，b)从不重复，但我们发现这两点都不正确。Twitter 实际上在他们的 API 文档中说，推文不是按照它们被发送的顺序发送的，而是按照 Twitter 服务器处理它们的顺序发送的，所以你必须小心编写你的代码，不要假设它们是任何顺序的。大多数时候，我们正在处理的任何推文之间都有足够的间隙，这不是问题，但也有一些边缘情况，事情可能会变得有点棘手。谢天谢地，我们已经解决了大部分问题，但还是有一些问题。

路易:是啊，我能想象。从一开始就使用 Twitter 的想法是什么，而不是建立自己的界面或允许人们只使用移动应用程序，例如，转账；为什么要借助 Twitter？

安迪:是的，Twitter 的优势在于你可以立即看到整个人际网络，它有一个很好的 API，使用起来相对简单，这是一个 restful API，它只使用 JSON，我认为你也可以从中获得 XML，但我宁愿不要。

路易:(笑)

安迪:我真的不想。你知道，在早期获得这种可见性对我们来说非常重要；如果我们建立了自己的界面，那么我们可能会使用电子邮件，我们从事这项业务的一个假设是，我们希望交易公开，我们希望人们大声谈论他们的钱，我们希望挑战金钱和支付必须是私人事情的观念，我认为到目前为止这一点做得很好。

**路易:**对。所以你一开始用 Twitter，后来又增加了电子邮件交易。

安迪:没错。因此，我认为大约三四周前，我们对我们的生产环境进行了另一次更新，现在您可以通过电子邮件向人们付款，这更私密、更安静，但我认为迄今为止，绝大多数人仍在使用 Twitter 进行汇款。
**路易:**对。我们之前讨论过一些明显的用例，例如，分摊餐馆账单或支付咖啡费用，你知道，如果你从某人那里借了 5 美元，你想还他们，你见过的其他用例有哪些违背了你对人们将使用该系统的预期？

安迪:墨尔本杯对我们来说是非常有趣的一天。

路易斯:好的，这一次我们要暂停一下，可能会给非澳大利亚听众一些解释。

安迪:当然是关于墨尔本杯日。

路易斯:是的，到达澳大利亚对我来说是一种文化冲击。所以这是一个围绕赛马的全国性节日。

**安迪:**对，没错。

路易斯:(笑)所以每个人都放假，他们都去赌马，这是理所当然的。

安迪:啊，不是所有人。

路易斯:不是每个人都放假，对吧。

安迪:是的，只有澳大利亚的一些州仍然把它作为公共假日，但这在文化上是一个非常重要的日子，这句格言我会解释一下，它的意思是人们度过了一个非常懒散的下午，最后看了赛马，经常喝大量的酒，这是澳大利亚的传统。

Louis: 好了，现在我们已经了解了背景，请告诉我们 Pygg 是如何融入其中的。

Andy: 我们看到很多人在 Twitter 上打赌并互相挑战，所以有几个团体在做——例如，在办公室里，我们在做一个比赛，看谁是最佳着装者，我想获胜者最终会赢得一小笔钱，另一笔钱会捐给慈善机构。所以其他团体也参与进来，实际上用一美元投票。

**路易:**对。

安迪:我们见过一些人做这样的事情，他们只是扔给某人一美元，然后说，嘿；我真的很欣赏你写的那篇博客文章，你知道，这是买咖啡或啤酒的钱，所以人们互相感谢。

路易斯:对，这是一种在线帮助的小费经济。

安迪:是的，没错。我们实际上希望看到更多这样的东西，因为我认为这真的很酷。

路易斯:好的，你有没有见过有人把这种东西插入他们的网站，比如说如果你喜欢这篇文章，这里有一个 Twitter API 集成，可以付费，一美元或五十美分？

安迪:我们实际上提供了。我必须找到网址，目前我还没有找到，但是我们的产品经理 Michael 组装了一个 Ruby 支持的 web 应用程序，它可以生成一个 HTML 片段，你可以把它放到你的网页上，当有人点击它时，它会自动发送一条 tweet，使用 Twitter 的意图来分享内容，所以它会弹出一个漂亮的小 JavaScript 东西，带有预先填充的消息@pygg pay，博客的 Twitter 句柄的所有者，这篇精彩的文章价值 4.00 美元。

**路易:**对。

安迪:我们还没有尽可能地宣传它，但是有一些人在使用它。

**路易:**牛逼。你们遇到过其他人做类似的事情吗？

Andy: 在澳大利亚，尤其是硅谷，有很多公司开始在社交支付领域进行类似银行 2.0 的活动。我们见过 Square，我真的不知道如何描述他们的商业模式，但他们最近发布了一个基于位置的支付应用程序，所以你在手机上打开应用程序，选择你所在的商店，店主会得到一份目前在他们商店登记的人的名单，他们可以通过你的照片认出你，并使用 Square 的 API 向你收费。这是我们见过的特别酷的实现。是的，有很多人在做类似的事情；我还没见过其他人以和我们完全一样的方式使用 Twitter，但看到一些竞争很快涌现出来，我不会感到惊讶。

路易:对。你也提到你确实有最终离开澳大利亚的计划。

安迪:是的。我不确定我们会先去哪里，但是还有一些工作要做，应用程序还没有达到我们想要一次向数亿用户公开的水平；我们还有很长的路要走，我们会到达那里的，别担心。

路易:(笑)

安迪:一旦解决了这个问题，做出了商业决定，我们肯定会着眼于国际市场，但这超出了我的能力范围，这是创始人需要决定的事情。

**路易:**对。你提到的事情之一是你通过银行账号将钱存入银行账户，有没有讨论过可能使用贝宝，允许人们将钱存入他们的贝宝账户？

Andy: 在早期，根据记忆，这与 PayPal 的 API 以及我们必须与他们整合的方式有关，他们有一些非常严格的业务规则来将支付返回到 PayPal。目前，我们的贝宝实现只是一个贝宝立即购买按钮。

路易斯:对，没错。

Andy: 它非常简单，易于实现，生产和运行速度非常快，实际上以一种更结构化的方式与他们的 API 集成，至少在早期，比我们想的要多。因此，这是我们在接下来的四个月里要考虑的事情。我们将扩大让你的资金进出这项服务的选择。

路易:好的，酷。你之前提到过你工作的公司叫做 Pollenizer，Pygg 是少数几个随时都在进行的项目之一。

**安迪:**正确。Pollenizer 目前我猜创业孵化器是描述它的最佳方式，简单地说，它比那稍微复杂一点，但那是它的本质；我们建立最小可行的企业。

**路易:**对。Pygg 是第一批项目中的一个吗？或者这是，你知道，或者过去有一些来自 Pollenizer 的项目吗？

安迪:波利尼泽已经出现了；2 月 29 日我们就要过四岁生日了。Pollenizer 已经建立了许多不同的创业公司，显然不是所有的都成功，在这个行业中，可以肯定的是，你会尝试一系列事情，你会尝试可行的事情，并尝试尽快失败，而不可行的事情，但一些过去著名的名字包括 Dealized 和 SpreetsSpreets 以一笔我可能不允许提及的金额被卖给了雅虎。

路易:(笑)
