# 索尼将在新电脑上提供谷歌浏览器

> 原文:[https://www.sitepoint.com/sony-offer-google-chrome/](https://www.sitepoint.com/sony-offer-google-chrome/)

谷歌 Chrome 浏览器的市场采用受到了索尼的大力推动。在两家科技巨头达成协议后，索尼将在部分笔记本电脑上安装 Chrome 作为默认的 Windows 网络浏览器。这种机器可能会首先在美国上市，随后很快在其他地区上市。

Chrome 的市场份额已经增长到 6%左右，并且有可能超过 Safari 和 Opera 成为网络上第三大最受欢迎的浏览器*(来源:[w3schools.com](http://www.w3schools.com/browsers/browsers_stats.asp))*。谷歌一直在 YouTube 等网站上大力[推广这款浏览器](https://www.sitepoint.com/youtube-drop-ie6/)，我们[怀疑他们会与个人电脑制造商](https://www.sitepoint.com/google-chrome-more-popular/)达成许可协议。索尼是一个受欢迎的品牌，尽管他们是较小的个人电脑制造商之一，谷歌肯定会寻求与戴尔和惠普等公司的交易。

与其他浏览器厂商相比，谷歌处于一个独特的位置:

*   尽管苹果在 Windows PCs 上推广 Safari，但他们最终还是希望人们购买 MAC 电脑。浏览器永远不会成为苹果的首要关注点。
*   Mozilla 是一个开源组织。他们没有相同的商业和业务压力。
*   Opera 没有同样的营销影响力或预算。

浏览器的统计数据不能全信，但 Chrome 似乎正在蚕食 ie 浏览器的份额。Chrome 的成功对谷歌来说有很好的商业意义。相反，微软正在遭受痛苦:该公司发布 IE 更新的速度一直很慢，并被迫在欧洲提供一个浏览器投票屏幕。

Chrome 的未来看起来很有希望。我们会看到微软、Mozilla 和谷歌为争夺令人垂涎的顶级浏览器而展开三方大战吗？
