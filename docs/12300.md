# 写博客能让你赚钱，即使你的博客不能

> 原文：<https://www.sitepoint.com/blogging-can-make-you-money-even-if-your-blog-doesnt/>

![](img/f8ffd6a5151b8d1097668fa714d82415.png "blogging-keys")的确大多数*博客*不直接赚钱。然而，大量的*博客作者*通过他们的博客赚了很多钱，这也是事实。这就是为什么《新闻周刊》的丹·里昂断言博客是“又一个高科技童话”让我觉得特别荒谬。

莱昂斯的博客“史蒂夫·乔布斯的秘密日记”在过去的一两年里广受欢迎，他认为写博客不是赚钱的好方法。莱昂斯的博客经历让他得出结论，“虽然博客可以做许多美妙的事情，但产生大量的金钱不是其中之一。”

莱昂斯承认，少数幸运的博客，以及那些属于大型媒体资产的博客，为他们的作者赚了很多钱——在过去的几年里，为这些类型的博客写作是我作为博客作者如何设法过上相当舒适的生活的。但莱昂斯说，这些成功的故事少之又少。“博客研究员 Technorati 估计，经营广告的博主平均年收入为 5060 美元，”他写道。“先不要给法拉利经销商打电话。”

莱昂斯没有提到的是，博客通过打开大门可以为人们赚多少钱。一些最成功的博客，就其为作者创造的机会而言，是那些甚至没有任何广告的博客。博客先驱 Dave Winer [今天写道](http://www.scripting.com/stories/2009/02/11/howIMadeOver2MillionWithTh.html)在过去的 12 年里，博客通过让他宣传自己的生意和土地咨询工作，让他赚了 200 多万美元。

另一个受欢迎的博主，罗伯特·斯科博，最近才开始在他的博客上投放广告。我敢打赌他们不会让他赚很多钱。但他的博客无疑为他带来了大量付费的咨询和演讲工作，并可能在他如何获得 FastCompany 的专栏作家这一当前工作中发挥了作用。

写博客可以帮助你建立个人品牌，帮助你成为你所在领域的专家。你的博客可以间接为你赚钱，为你赢得关注，吸引你所在领域的付费演出，不管是什么。这实际上是自从 SitePoint 博客论坛开放以来我一直在说的话。“如果你不是顶级博客作者，靠广告赚钱会很难。但是用你的博客来推广你的品牌[……]是一个好主意。我无法告诉你我有多少朋友用他们的博客来获得咨询客户，或者找到工作等等。“13 个月前，我在[的一个论坛帖子中写道。这是](https://www.sitepoint.com/forums/showthread.php?p=3677957#post3677957)[几个月前我在同一个论坛上写的](https://www.sitepoint.com/forums/showthread.php?p=4059790#post4059790):

> 有些个人博客有收入，但大多数没有。或者他们有，但这是如此微不足道的一笔钱，不足以维持生活(甚至可能不足以支付托管费)。
> 
> 然而，个人博客**可以**以其他方式支付红利。写博客是让你成为所在领域专家的好方法。虽然你可能不会直接从广告中获得收入，一个受欢迎的博客可以帮你找到工作，咨询演出，演讲，可以帮你销售书籍，服务，或者你做的任何事情，等等。
> 
> 对许多人来说，他们的个人博客是建立个人品牌的绝佳营销工具。它不能直接为他们赚钱，但确实能为他们赚钱。

那么里昂呢？事实证明，他的博客带给他的远不止他所说的每月从 AdSense 获得的 1000 美元。正如 TechDirt 的凯文·多诺万指出的那样，莱昂斯已经将他博客的成功转化为一些有偿写作和演讲。多诺万写道:“当然，莱昂斯的文章中缺少的是他作为博客的直接结果而获得的巨大成功。他不仅获得了一大笔出书交易，使用了他为博客创造的假史蒂夫·乔布斯的角色，莱昂斯还被邀请就这一主题进行广泛的演讲和写作。此外，令人怀疑的是，他高调地从《福布斯》转到《新闻周刊》,除了博客上的成功之外，还有什么别的好处。"

一句话:你的博客本身可能不赚钱，但博客本身肯定能带来丰厚的回报。
