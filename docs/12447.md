# 谷歌背弃了火狐浏览器

> 原文:[https://www.sitepoint.com/google-turns-its-back-on-firefox/](https://www.sitepoint.com/google-turns-its-back-on-firefox/)

大约一个月前，谷歌[将 StarOffice](https://www.sitepoint.com/google-drops-staroffice-from-pack/) 从[谷歌包](http://pack.google.com/)中删除，这是一个为 Windows 用户提供的免费互联网和生产力软件下载包。StarOffice 是一个桌面办公套件，基于开源的程序，由 Sun 发布，我们推测谷歌正在为未来推动用户放弃桌面软件，转而使用谷歌自己的基于网络的应用程序奠定基础。

“如果把人们推向他们自己的工具套件是谷歌正在玩的游戏，我们预计我们有可能最终会看到 Firefox 从谷歌包中退出，并被 Chrome 取代(至少一旦该浏览器退出测试)，”我们预测。

这一预测几乎成为现实。谷歌[上周从他们的](https://www.sitepoint.com/google-chrome-leaves-beta/) [Chrome 网络浏览器](http://www.google.com/chrome/)中移除了测试标签，现在，他们已经降级了 Firefox。他们还没有彻底移除网络排名第二的浏览器——这是有道理的，因为集成的谷歌工具栏和他们作为火狐默认搜索引擎的地位每年价值数千万美元——但他们已经将火狐推到一边，转而支持 Chrome。

![](../Images/108953dec3568ee29cb9cfbb0cf80f07.png "google-pack")
*上图:作为 Google Pack 下载一部分的默认软件不再包括 Firefox。*

Firefox 曾经是 Google Pack 的默认推荐浏览器，但在周末，关注 Google 的博客作者 Garett Rogers 注意到 Google 用 Chrome 取代了 Firefox 作为默认下载。从技术上来说，Firefox 仍然是软件包的一部分，但是默认情况下它是未选中的，并且它的描述已经被推到了信息页面的下方。

谷歌将人们推向 Chrome 是有意义的，因为它是专门为运行网络应用而构建的。使用 Chrome 的人越多，运行 Gmail、Reader、Docs 和 Spreadsheets 等谷歌品牌网络应用的用户体验就越好。

正如我们所提到的，Chrome 是谷歌三管齐下的网络操作系统战略的重要组成部分。与 Gears(离线数据存储)和 Native Client(网络应用的本地 CPU 资源)一起，Chrome 为谷歌提供了一个引人注目的网络应用交付平台。谷歌需要推动更多的人使用 Chrome，以便合法地赢得他们与微软、Adobe、苹果以及脸书等公司之间正在酝酿的网络操作系统战争。
