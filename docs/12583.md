# 雅虎！实际上在付费搜索中领先

> 原文：<https://www.sitepoint.com/yahoo-actually-leads-in-paid-search/>

来自 Compete 的的一篇[博客文章显示，当你稍微欺骗一下统计数据时，它就是雅虎！在发送付费点击方面做得更好。Compete 的 Alex Patriquin 发现，当你查看付费推荐占总搜索查询的比例时，谷歌占据了付费点击的主导地位。在这种情况下，谷歌上个季度能够将 3.4%的搜索转化为付费推荐。雅虎！另一方面，只有 2.6%的搜索这样做。](http://blog.compete.com/2008/10/24/google-yahoo-live-paid-search/)

然而，Patriquin 说，许多搜索不会导致任何形式的点击。以谷歌搜索一下[ [盎司](http://www.google.com/search?q=ounces+in+a+shot)为例。Google 通过 SERPs 上的一个方便的转换计算器返回答案。有许多类似的查询例子，其中 SERP 本身提供所需的信息，例如查找电话号码、位置、单词拼写、定义等。Patriquin 写道。"有时，所需的信息也可以在每个结果下面的文本片段中找到."

因此，有一定比例的搜索不会导致任何类型的推荐——付费或自然推荐。因此，Patriquin 认为，付费搜索成功的更合适的衡量标准是付费搜索推荐占总推荐数的百分比，而不是搜索数。

按照这个标准，雅虎！是明确的领导者。2008 年第三季度，雅虎！美国的付费推荐率为 5.1%，而谷歌为 4.8%。雅虎！在过去三个季度中一直领先。为什么？因为雅虎！显然，在带领用户徒劳无功方面做得更好。同样的一次搜索[ [盎司](http://search.yahoo.com/search?p=ounces+in+a+shot)]会从 Calibex 和 BizRate 返回一个付费结果，而不是一个对话计算器。

Patriquin 说:“对于用户来说，在 SERP 上直接获得想要的信息非常好，而不必点击进入另一个页面。"对于搜索引擎来说，这是一个错过的赚钱机会."

然而，这项研究实际上是了解雅虎的一个很好的窗口。出错了。尽管如此，正如 Patriquin 所说，“在每笔付费推荐的数量和平均收入相等的情况下，雅虎！会在付费搜索中赚更多的钱”，事实是他们并不平等，原因归结为用户体验。对于上面提到的搜索，Google 和 Yahoo！在自然结果中首先提供相同的链接，但是在谷歌上的体验要好得多。

雅虎！以百分比为基础提供更多付费推荐，但他们的份额比谷歌小得多，所以他们赚不到那么多钱。

“这些发现仍然会给雅虎带来鼓舞。Patriquin 认为，“它在传统智慧上痛击了媒体和市场，认为它在搜索货币化方面不如谷歌。正如我们所看到的，当我们剔除那些不符合货币化条件的搜索时，旧的紫色和黄色实际上领先于该领域。”

但他错了。他的报告实际上进一步证明了消费者很久以前的决定:在谷歌的体验更好。雅虎！的付费搜索转化率更高是因为他们的搜索结果对消费者来说用处不大。只要用户体验差距仍然存在，雅虎！永远无法在搜索领域赶上谷歌。
