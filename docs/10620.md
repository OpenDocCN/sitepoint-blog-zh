# 使用 5S 方法保持组织有序

> 原文：<https://www.sitepoint.com/5s-methodology-for-organization/>

5S 方法指的是一种日本的实现和维持秩序的系统，尤其是在工作场所。这种方法背后的思想是，组织的标准流程和标准措施可以帮助工人建立一个有序和高效的工作场所。

5S 方法之所以如此命名，是因为日语和英语中描述这五个领域的单词都以字母 S 开头:

*   Sort (Seiri)
*   拉直(Seiton)
*   闪耀(Seiso)
*   标准化
*   持续(Shitsuke)

虽然 5S 方法最初指的是员工之间的标准化，以及每个团队成员如何通过完成五个领域中的每一个领域的活动来支持秩序感和结构感，但这一思想也可以应用于个人工作的情况，以实现更大的个人组织。以下是每个领域如何适用于独自工作的人的例子。

## ![](img/6a697e4ee7bfa3a67e0f881784692bcf.png "organized") Sort (Seiri)

想象一下忙碌的一天结束时你的办公桌。你可能有纸、笔、便利贴和其他材料散落一地。现在想一想，忙碌了一周之后，你的办公桌是什么样子。用不了多久，你就会完全失去秩序感。

有一种方法可以让你在快速积累的混乱中保持领先，并使查找信息的过程更加高效，那就是花时间找出你最重要的材料，以适合你工作流程的方式对它们进行分类，并去掉其他的。这可能意味着在你的桌子上放一个迷你归档系统，在每天结束时组织和归档你电脑桌面上的文件，和/或准备扔掉或回收任何不一定需要保存的东西——马上。

## 按顺序排列

一旦你有了什么应该保存，应该保存在哪里，什么可以扔掉的想法，你应该在每天结束时花时间把东西放回原处。不要让它积累；让这个过程成为一个积极的习惯。

## 闪耀(Seiso)

不，当你晚上关掉电脑时，你不必拿出家具上光剂来擦你的桌子。这一步实际上就是采取一种心态，让你能够始终如一地思考，在一个干净整洁的地方工作。这也有助于你避免因凌乱(或肮脏)的工作环境而分心。

## 标准化

维持组织的过程不应该是一周一次的活动，或者更糟糕的是，只有当混乱和杂乱变得如此糟糕以至于你几乎无法工作时，你才会去做。这看起来似乎过于简单，但是每天结束的时候花五分钟整理、整理和擦亮你的工作空间，你将会为自己节省更多的时间和压力。

你也想避免许多不同的组织方法，因为信不信由你，太多的组织策略可能是导致混乱的最大原因之一。找到一两个最适合你的系统，然后全面应用。

## 持续(Shitsuke)

这可能是最困难的步骤，因为它需要持续的努力。让开发结构的其余步骤有效的唯一方法是每天致力于执行和维护它们。

我们中许多认为自己有条理的人可能已经使用了一个类似的系统，可能会根据我们个人的偏好进行修改。我在这个过程中加入的一个变化是后退(是的，另一个 S！).偶尔后退一步让我有机会审视我的组织系统，看看哪里可以调整和改进。

你是否发现自己也遵循着类似的系统来保持条理？你如何使过程个性化？
