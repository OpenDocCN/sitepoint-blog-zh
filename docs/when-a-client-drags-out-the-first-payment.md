# 当客户支付第一笔款项时

> 原文：<https://www.sitepoint.com/when-a-client-drags-out-the-first-payment/>

我曾经是一个好人，当客户告诉我他们将继续工作时，我就开始工作——即使第一笔付款还没有收到。

不会了。

现在，我只有在客户预付了我 50%(或我们约定的任何金额)后才开始工作。

例如，我正与一家上市公司的首席执行官一起工作，销售一种漂亮的软件解决方案。他原则上同意签订一份长期合同，首期付款应提前支付。他给我发了成吨的材料让我开始，并敦促我开始，因为他面临一个关键的截止日期。但是他还没有给我一份签好的合同或者首付款。

在我这边，我想开始了。我知道他的急迫感很快就会变成我的，我担心如果因为我已经开始了这个项目而错过了最后期限，我会受到责备。此外，我喜欢向人们展示我超越自我的能力和积极进取的态度。那么，为什么不现在就开始期待付款呢？

三个原因:

一:我刚好有个同事也是他的同事。这位同事告诉我，这家公司的首席执行官因打击承包商而出名——延迟付款、重新谈判等。他给我举了两个这样的例子，他和他的一个同事。

二:如果我现在开始，我就失去了我的筹码，因为我表明我愿意在他没有表现的时候工作。金钱是杠杆，对双方都是如此。如果他真的有紧急情况，马上电汇资金应该不成问题。

三:作为专业人士和高管，我想和他平起平坐。如果我现在开始工作，而他还没有履行他的承诺，我会失去那个立足点，并可能失去他的尊重。我成了另一个他可以击败的供应商。

这位首席执行官与大多数精明的商界人士没有太大不同。他很强硬，尽他所能从他的承包商那里得到最多。这取决于我坚持和他进行一些艰难的对话。

所以我给他发了一封电子邮件，告诉他在他给我一份签了字的合同并支付第一笔款项之前，我不能开始他的项目。同时，我已经让他知道，如果他不立即行动，我可能不得不推迟他的最后期限。这封邮件每次都管用。

坚守阵地！
