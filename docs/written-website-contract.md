# 需要一份书面网站合同

> 原文：<https://www.sitepoint.com/written-website-contract/>

不知道为什么，一些网页设计师和网站所有者似乎不是从繁荣的角度，而是从匮乏的角度来看待他们之间的关系。有些人似乎选择了失败，而他们本可以双赢。

显然，仍然有许多独立的承包商网站设计者和所有者在没有充分的、法律上可执行的书面合同的情况下，从事设计(并雇佣那些设计者)网站的业务。在网页设计和版权所有的世界里，书面合同不是你生意的附加物——书面合同*是*你的生意。

这种不使用书面合同的做法对网站所有者和设计者来说都是不明智的。在这场口头协议的战争中，任何一方都不可能获胜。但在自由市场资本主义的世界里，没有法律规定智慧。智慧往往来自于许多艰难的，甚至是不必要的教训。

因此，探讨双方在合同关系中的权利是值得的。

##### 书面合同给设计师带来的好处

根据《统一商法典》的规定，似乎为了让设计者在大多数情况下保护自己，必须有一份由场地所有人签署的书面合同，充分说明要做的工作。下面的法典部分指的是统一商业法典的加州版本，但是你应该咨询当地的律师，看看这些或类似的条款是否适用于你所在的州或国家。

《加州商法典》第 2201 节部分规定:

<q>*(1)除非本条另有规定，价值为五百美元(＄500)或以上的货物买卖合同不能通过诉讼或抗辩的方式强制执行，除非有足够的书面文件表明买卖合同已经在当事人之间订立，并由被寻求强制执行的一方或其授权的代理人或经纪人签署。*</q>

《商业法典》接着定义了“商品”,这个定义似乎足够宽泛，可以涵盖一个网站。在这方面，第 2105 条部分规定:

<q>*(1)“货物”是指除支付价款的货币以外，在订立买卖合同时可以移动的一切物品(包括专门制造的货物)……*</q>

<q>*(2)在货物中的任何权益可以转移之前，货物必须既存在又被识别。不是既存在又被识别的商品是“未来”商品。未来货物或其中的任何权益的声称的现在销售作为销售合同起作用。*</q>

上述部分似乎将网站定义为可移动的“货物”(以区别于不动产)。以电子方式在网络上传输网站，似乎是一个人能做到的最可移动的方式。因此，网站的创建似乎受该法条款的管辖。

下面的第 2106 条定义了“销售合同”,这似乎也足以涵盖网站设计协议。

<q>*(1)在本分部中，除上下文另有要求外，“合同”和“协议”仅限于那些与现在或将来的货物销售有关的合同。“销售合同”包括现在的货物销售和将来销售货物的合同。“销售”包括将所有权从卖方转移到买方，以取得价款(第 2401 条)。“现售”指通过订立合同而完成的销售。*</q>

所有这些法律术语的总和是，如果准则适用，并且如果网站设计协议的价格超过 500 美元，那么设计师就有可能无法强制网站所有者做出口头承诺。无论设计师在网站上投入了多少工作，如果合同价格超过 500.00 美元，如果网站所有者没有签署足够的书面文件，网站所有者可能没有义务支付口头商定的价格。这在法律术语中被称为“反欺诈法”，这是一个很久以前就出现的概念，目的是明确在某些非常重要的交易中，当事人必须有书面协议(这一规则有一些潜在的例外，但它们超出了本条的范围，并取决于特定情况的事实)。

因此，有一份由网站所有者签署的有效的、有约束力的书面协议，对设计者来说显然是有利的。该协议的内容当然是另一个问题。

##### 书面合同给网站所有者带来的好处

出于有些不同的原因，拥有一份由设计师签署的书面合同也有利于网站所有者。网站的设计和创建在很大程度上涉及到设计者创建受版权保护的元素，并将这些权利的一部分或全部转让给网站所有者(出于本文的考虑，我假设设计者不是网站所有者的雇员。作为独立承包商，任何转让，无论是作为雇佣作品还是其他作品，都需要有效的书面签名)。

1976 年美国版权法规定，版权中所有权利的所有者是这些权利的创造者或作者。这意味着，从设计者创作出可受版权保护的作品并将其固定在某种有形的表达媒介中的那一刻起，该作品的版权就完全属于设计者。网站所有者对该作品没有任何权利。

《美国版权法》第 201 节部分规定如下:

<q>*秒。201.*版权归属</q>

<q>*(a)初始所有权——受本标题保护的作品的版权最初属于作品的作者。*</q>

<q>*同一著作权法第 204 条继续部分陈述:*</q>

<q>*秒。204.版权所有权转让的执行*</q>

<q>*(a)除非转让文书或转让通知或备忘录是书面的，并由被转让权利的所有人或该所有人的正式授权代理人签字，否则版权所有权的转让无效，法律规定的转让除外。*</q>

其中一些可能看起来很熟悉。又是我们老朋友“反欺诈法”的亲戚。综上所述，美国版权法的这些条款意味着，如果网站所有者没有获得设计师签署的书面合同，网站所有者可能最终除了作为网站的一些非专有权利之外，一无所有。

##### 书面合同对双方的好处

因此，有一份由双方签署的书面合同，明确规定设计关系中各方的权利和义务，显然对场地所有者和设计者都有好处。如果契约足够彻底，就可以消除不确定性。双方可能会稍微松一口气。每一方都知道对方应该做什么，以及各自将获得或放弃什么权利。它简洁、专业、高效，而且如你所见，是必要的。每一方都从这种“双赢”的局面中受益，而如果没有这样的合同，双方都可能发现自己处于“双输”的局面。

##### 结论

尽管这似乎是一个不言而喻的事情——书面合同对双方都有利——但似乎有许多网站所有者和设计者不使用书面协议。

这是因为设计师害怕如果他或她出示书面合同，网站所有者会被“吓跑”吗？是因为网站所有者没有意识到没有书面合同所涉及的版权问题吗？还是那句来自两边的老锯:“我买不起”？在这种情况下，最直接的回答是“你能承受工作得不到报酬吗？”或者“你能承受不拥有你已经支付的东西吗？”

我不知道这些问题的答案。但是我从事法律工作已经很长时间了，我知道有你的书面同意比没有好得多。也许自由市场资本主义应该要求智慧。
