# 管理开源安装和更新

> 原文：<https://www.sitepoint.com/managing-open-source-installation-and-updates/>

在混合商业和自由软件的同一主题中——LAMP 和 Java 开发人员和管理员可能会发现从 Open Logic 中学习 blueglue 非常方便。

blueglue 作为一种商业解决方案出售，以应对管理开源软件堆栈的挑战——几个应用程序的组合使我们在 Linux 或 Windows 服务器上的工作成为可能。例如，可以从一个中央界面维护运行 Apache、MySQL、PHP 和 CVS 的环境的安装、配置和更新。

该产品支持 100 多个著名的开源应用程序，包括 Apache、Perl、PostgreSQL、Tomcat、JBoss 等等。

我发现有趣的是包含了十几个示例项目，这些项目利用那些带有示例配置和代码的开源应用程序来启动探索和测试。

这对于初学者来说是有价值的，或者更重要的是，对于那些认真探索开源但尚未实现飞跃的开发人员或客户来说。

Open Logic 以每年 400 美元的订阅价格出售，但是，考虑到通过自动化安装、配置、测试和修补界面节省的资金和时间，这一成本非常低。

绝对值得一看——对于那些支持 Linux 和 Windows 的开源应用程序来说，它具有跨平台友好性——对于混合环境非常有用。
