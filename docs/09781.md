# 10 个 jQuery 谷歌地图插件

> 原文:[https://www.sitepoint.com/10-jquery-google-map-plugins/](https://www.sitepoint.com/10-jquery-google-map-plugins/)

Here are 10 very different **jQuery Google Maps Plugins** to make life as easy as a street map. You might only want to display a simple map that displays directions to your clients address, or even an interactive map with more options as necessary. The plugins below offer not only an easier method to install a map, they also offer the option to add extra functionality, and should you choose to need them. Happy mapping!

## [1 .gmap〔t1〕](http://gmap.nurtext.de/)

## [2。jQuery GPS](http://www.birdwingfx.com/jQuery_gps/index.html)

一个 jQuery 插件，面向那些想在自己的网站上添加谷歌地图，但又不希望自己的网站因为不需要的功能而停滞不前的人。一个轻量级和简单的解决方案，满足您所有的谷歌地图需求。只需添加默认或自定义 ID，即可添加地址输入。您可以使用任何元素来触发事件！所以它不需要嵌入到表单中。
![jQuery-GPS.jpg](../Images/74f76dbdd5c7ba4c123baca16359b7c6.png)
T3】来源

## [3。谷歌地图](http://www.dyve.net/jquery/?googlemaps)

这个插件使得在你的页面上放置谷歌地图变得很容易。你需要一个来自 http://www.google.com/apis/maps/.的有效的谷歌地图 API 密钥，如果你想知道如何使用这个插件，请看这个页面的源代码。标记可以由 jQuery 对象(包含 geo 微格式的元素)提供，也可以由为弹出窗口提供 lat、lng 和 txt(txt 是可选的)的对象数组提供。
![googlemaps.jpg](../Images/1a93af0e19b3435cec360999df806c62.png)
