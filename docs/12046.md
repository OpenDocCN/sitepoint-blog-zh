# 20 个华丽的主题

> 原文:[https://www . site point . com/20-华丽-wordpress-gallery-themes/](https://www.sitepoint.com/20-gorgeous-wordpress-gallery-themes/)

人们很容易忘记，有时候 WordPress 不仅仅是一个博客平台。由于其开放的性质，并感谢许多主题建设者，它可以很容易地变成一个画廊来展示你的最新作品。为什么你要从有偿工作中抽出时间来经营自己的画廊呢？WordPress 有许多主题可以帮你完成这项工作，而且许多主题提供了简单的定制，所以你的图库不是现成的解决方案。

我们收集了一系列免费和付费的解决方案，因此无论您的预算是多少，您都可以立即使用该图库！
