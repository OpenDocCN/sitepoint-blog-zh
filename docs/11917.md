# SEO:什么是热门，什么不是

> 原文:[https://www.sitepoint.com/hot-seo-metrics/](https://www.sitepoint.com/hot-seo-metrics/)

许多旧的搜索引擎优化策略已经过时，例如，没有人搜索的关键词的排名——你知道，那些“自我提升者”会在谷歌的第一页显示你的网站。将你的网站提交到成千上万的网络目录中以获得链接，将你的网站提交到搜索引擎中以获得索引，这是两种没有严肃的 SEO 会考虑的技术。站长们仍然相信交换链接是获得更高排名的神奇答案(他们在 Google PageRanks 中仍然扮演着一个次要的角色)，许多人仍然纠结于重复内容的惩罚(我们在过去的 [SitePoint 文章](https://www.sitepoint.com/google-seo-myths/)中讨论过)。所有这些技术现在都被没有经验的 SEO 和“老派”DIY 使用，他们不理解 Web 2.0 的动态。

##### SEO 2.0 的新趋势

Web 2.0 是一个社交实体，显然 Web 2.0 的 SEO 需要社交。

在脸书、LinkedIn、Bebo、YouTube、FriendFeed、Twitter 等网站上建立社交网络只是一个方面。你建了他们，追随者不一定会来。你必须**给访问者一个成为你的社区成员的理由**，更重要的是，如果你想充分受益于今天许多网络营销者谈论的“网络效应”，你必须给他们一个点击你提交给他们的链接的理由。Twitter 是这种“网络效应”如何有益的完美例子。例如，戴尔利用 Twitter 宣布特价并与消费者沟通，成功获得了 300 万美元的收入[。](http://bits.blogs.nytimes.com/2009/06/12/dell-has-earned-3-million-from-twitter/)

另一种现代搜索引擎优化技术，也有社交方面:通过发布人们真正感兴趣的内容，你基本上是在鼓励他们“点出”你的网站。好的内容会在几个小时内迅速传播开来。读者会“推特”你的链接，传递它(这很像“口口相传”)，在他们的网站上提到你的内容，等等。这就是如何建立自然链接，这是唯一有意义的方式来开始一个链接运动。

另一个重要的 SEO 方面被定义为“长尾”这个词最初是由克里斯·安德森在 2004 年[创造的，用来描述一些销售大量独特商品的企业的利基战略，每种商品的数量相对较少。如你所见，这个术语的原意与 SEO 毫无关系。然而，SEOs 喜欢这个术语，并应用它来定义处理长关键字短语(通常每个短语包含多达 5 个单词)的 SEO 策略，这通常带来较少的流量但较高的转化率。](http://en.wikipedia.org/wiki/The_Long_Tail)

##### 衡量 SEO 2.0

痴迷于页面浏览量和 Alexa 排名显然不再是衡量 SEO 成功的方式。如果你知道如何阅读，你的分析程序提供了许多或多或少相关的指标。

例如，如果表现最好的关键词是提及你的品牌的关键词，那么你的 SEO 就没那么好。好的搜索引擎优化为非品牌关键词带来的流量要多于品牌关键词。当然，让搜索者寻找你的品牌很重要，但你不想依赖它获得自然流量。

另一个重要的指标，例如，如果你使用谷歌网站管理员工具，是搜索引擎抓取和索引的独特页面的数量。你在搜索引擎中索引的独特页面越多，你增加流量和销售的机会就越大。

为你的网站带来流量的页面数量也很重要:保持这个高数量对你最有利，这样你就能充分受益于上面提到的“长尾”优势。不增加流量的页面实际上是死页面:它们只会让你的网站变得杂乱，而不会带来 SEO 优势。
