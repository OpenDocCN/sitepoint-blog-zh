# 不要揣摩你的潜在客户的想法

> 原文:[https://www.sitepoint.com/dont-read-your-prospects-mind/](https://www.sitepoint.com/dont-read-your-prospects-mind/)

这里有一个有趣的观察:

很多时候，潜在客户打电话给我，问我的项目是 X 还是 Y(例如，在我的拳击健身网站中，我提供继续教育吗？或者，我的认证在英国的健身房是否被接受)。通常答案是“不”，所以当我得到这些问题时，我畏缩了。我担心我不会得到销售。

但我只是诚实地回答，没有试图推销，有趣的事情发生了:

很多人以任何方式报名！

当我销售更大的项目时，也会发生同样的事情。人们对我可能不提供的服务提出尖锐的问题，或者问我是否可以降低价格，正确的答案是“不”。所以我说，“对不起，但是不。”人们经常以任何方式购买。

所以不要因为有人得到了否定的答案就认为他们不会买。不要假设他们需要那个特性。坚持你的立场，诚实地回答。人们尊重这一点，通常还是会购买。有时候他们只是想知道。

还有其他一些解读潜在客户想法的例子会让你陷入困境。如果你假设一个潜在客户想要某个特定的功能，并将其纳入你的提案，你可能会因为添加他们不想要的东西而失去销售。例如，我最近被一所大学叫去做一些组织发展的工作。我想尝试一种新的文化评估工具，并认为客户会喜欢它。但是这个工具让他们非常不舒服，我几乎失去了销售——直到我和他们开诚布公地谈了这件事，发现了他们的顾虑，并把它从范围内拿了出来。

不要假设你知道你的客户想要什么或在想什么。与他们开诚布公地交谈，并在交谈时保持平等。你的销售额会上升，因为你会了解他们想要什么，不想要什么，以及他们真正关心的是什么。
