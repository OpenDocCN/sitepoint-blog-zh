# 如何成功地将社交媒体整合到你的网站中

> 原文:[https://www.sitepoint.com/social-media-best-practices/](https://www.sitepoint.com/social-media-best-practices/)

在旧金山参加 Web 2.0 博览会时，我有机会听到来自 Digg 的 Bob Buch 介绍将社交媒体营销整合到您或您的客户网站的技巧和案例研究。

**Digg 流量**

“登上 Digg 首页的每篇报道都会获得至少 20，000 次点击，反馈给出版商。最好的故事将获得 200，000–250，000 次点击。Buch 说:“Digg 每月向第三方网站发送 8000 万次点击。

其他网站如何受益？

*   TheOnion 每月从 Digg 用户那里获得 60 万的页面浏览量
*   Wired.com 每月从 Digg 用户那里获得一百万的页面浏览量
*   Telegraph.co.uk 是 Digg 最成功的整合伙伴，也是英国最受欢迎的大报，每月从 Digg 用户那里获得 550 万的页面浏览量。

**使用智能 Digg 小工具**

智能 Digg 小工具显示了迄今为止的投票数，比没有计数器的常规“Digg This”图标要成功得多。

**集成最流行的 Widget**

![Digg Widget on Telegraph.co.uk](../Images/2206d7cd77b4ea63882f12e9484df71b.png "Digg Widget on Telegraph.co.uk")

Telegraph.co.uk 上的 Digg Widget

许多内容网站上最受欢迎的元素之一是“最受欢迎的内容”框，它根据最近的页面浏览量突出显示 5-10 篇受欢迎的文章。Time.com 发现“ [Digg Widget](http://digg.com/add-digg) ”集成到他们的网站中，比他们自己的“最受欢迎的文章”框给 5X 带来了更多的互动，并由于大量的额外挖掘活动而大幅增加了他们自己的流量。

**脸书连线**

受欢迎的科技新闻出版商 TechCrunch 的注册量增长了 100%，同时通过整合[脸书连接](http://developers.facebook.com/connect.php)获得了用户的额外人口统计数据。

虽然异常成功，但 Bob 指出，由于干净的脸书连接集成，大多数网站的注册量增加了 30-80%，评论和用户生成的内容增加了 15-80%。您甚至可以将用户照片和其他人口统计数据导入到您的网站中，例如位置、性别和其他信息，如果整合到您自己的注册流程中，这些信息可能会影响您的注册。

脸书连接最大的成功之一是能够通过出现在“新闻”墙上，让在你的网站上发帖、投票、上传或以其他方式与你的网站互动的用户，像病毒一样传播你的信息。

每个脸书用户平均有 120 个脸书朋友，这是一种非常快速的方式，可以将你的品牌展示给很多人，非常快速，同时提供“社交证明”。
