# 思维网:社区的声音

> 原文:[https://www . site point . com/thinking-web-voices-of-the-community/](https://www.sitepoint.com/thinking-web-voices-of-the-community/)

2010 年初，一位论坛工作人员向我建议，我们应该做点什么来发掘 SitePoint 社区中沉睡的大量知识财富。答案很明显——一本合作的书。我还没来得及停下来思考，一个委员会就成立了，项目也启动了。一年后，经过大量的工作，我很自豪能够推出我们的劳动成果——这是 SitePoint 社区的合作书籍，我们将其命名为 ***思维网:社区之声*** 。

**前言**–莎拉·霍克(社区经理)

拉尔夫·梅森(Ralph Mason)
的第一章——网站剖析拉尔夫·梅森的这一章着眼于网站的组成部分，从托管到图标，以及它们之间的一切。如果你是网络世界的新手，那么这篇文章就是为你准备的。

Alex Dawson
撰写的第二章——黑暗中的设计 Alex 探讨了网页可访问性的概念，并向您介绍了识别您的受众并确保您的网站满足他们的需求和期望的过程。

第三章——一切都必须走亚历克斯·道森
亚历克斯回来了，这次他更详细地讨论了网页可访问性的概念。如果你想得到一些实用的建议来确保你的网站为每个人服务，那么这是一本必读的书。

约翰·博尔达
你可能是世界上最好的网络开发人员，但是如果没有人知道这一点，那么这对你来说就没什么用了。约翰着眼于自由职业者为了在残酷的 web 开发世界中取得成功可以做的一些事情。

保罗·奥布莱恩
在这一章中，CSS 专家保罗讲述了引用和执行 PSD 到 HTML 转换工作的复杂性。他指出了一些陷阱，并解释了如何避免它们。

第 6 章——写电子邮件标记，不要在收件箱里爆炸,,作者:Coyote Holmberg
如果你想写电子通讯，那么 Coyote 的这篇文章将是无价之宝。在这篇文章中，她讨论了从设计阶段到发送的 HTML 电子邮件的注意事项。

乌苏拉·科莫
第七章——让你的网站脱颖而出社交媒体已经变得如此普遍，如果你想让你的企业成功，就不能忽视它，乌苏拉解释了社交媒体和博客对你的网站和在线营销策略的重要性。

第八章——信息组织和网络雪莉·库里
你的网站上可能有这个星球上最有价值的信息，但如果很难浏览，那么人们在到达之前就放弃的可能性很大。Sherry 谈到了组织和标记信息的好策略，这样信息就容易获取和查找。

Clive Wickham
在本教程中，Clive 概述了如何使用交互式矢量图形轻松构建一个简单的 Noughts &十字架游戏，并探索了通过 SVG 和 HTML5 Canvas 呈现它的两种方式。

克里斯蒂安·斯诺德格拉斯
撰写的第十章——高效的动作脚本克里斯蒂安研究了一些在大中型项目中使用的更有用的动作脚本技巧、诀窍和技术。

第 11 章——数据库:基本概念 Nuria Zuazo
数据库对于任何动态网站来说都是必不可少的，一些基本知识对于充分利用数据库是必要的。在本文中，Nuria 着眼于现代数据库的一些复杂性，以及如何在自己的站点中利用它的力量。

**第 12 章——TCP/IP 的冰山**作者 Robert Wellock
