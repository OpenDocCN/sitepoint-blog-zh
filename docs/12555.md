# 微软用免费工具向创业公司推销

> 原文:[https://www . site point . com/Microsoft-makes-pitch-to-startups-with-free-tools/](https://www.sitepoint.com/microsoft-makes-pitch-to-startups-with-free-tools/)

在旧金山 Web 2.0 峰会的第一天，微软宣布推出 [BizSpark](http://www.microsoftstartupzone.com/BizSpark/Pages/At_a_Glance.aspx) 。该计划旨在通过向初创公司免费提供一系列微软开发工具和服务来帮助初创公司，同时确保在微软的 web 开发堆栈上开发一种新的 web 应用。

任何收入低于 100 万美元(在某些地理区域更低)且营业时间少于三年的公司都有资格参加 BizSpark 计划。该计划将为加入者提供免费使用各种微软工具的机会，包括 MSDN 会员资格、Visual Studio、Office、Windows、SQL Server 和其他许可证，以及获得微软合作伙伴指导的机会。该计划还承诺将参与者与投资者联系起来，并通过微软网站上的个人资料让他们受到关注。

参与计划的初创公司可获得的软件和服务的完整列表可在[计划指南(PDF)](http://download.microsoft.com/download/4/d/4/4d41081a-d8d9-407a-9bae-5127e6e931ca/BizSpark%20Startup%20Program%20Guide.pdf) 中找到。计划完成时— 3 年或因上市、被收购等原因丧失资格。— BizSpark 参与者必须支付 100 美元的计划费用。他们可以继续使用他们得到的软件，但必须支付许可费，并就托管合同的条款进行谈判。

这并不完全是我们在 8 月份为微软提出的 Y Combinator 式的投资策略，但这是朝着那个方向迈出的一步。微软没有直接投资初创公司，但他们为初创公司使用微软工具提供了丰厚的激励——免费工具、指导和会见投资者的机会。这是一项间接投资，将来会给公司带来回报。

微软知道计算正在慢慢向云转移，他们也知道大部分开发和部署都是使用他们无法控制的工具完成的——PHP、Ruby on Rails、Linux、Apache、Amazon 的 Web 服务等等。—通过 BizSpark，微软希望许多未来的网络初创公司可以使用他们的工具来创建，新一代的开发人员也将因此接触到这些工具。

据 Mary Jo Foley 称，微软并没有要求注册 BizSpark 的开发者使用其最近发布的云基础设施和服务，如 Azure ( [我们的覆盖范围](https://www.sitepoint.com/microsoft-answers-amazon-with-windows-azure/))和 Live Mesh ( [我们的覆盖范围](https://www.sitepoint.com/microsoft-opens-live-mesh-to-all/))，但它鼓励他们使用。这是微软的明智之举，因为更少的限制和更多的自由应该有助于吸引更多的开发人员加入这个项目。

从长远来看，即使没有一家使用 BizSpark 的创业公司成功，结果对微软来说也是积极的，因为微软已经培养了一大批开发人员。
