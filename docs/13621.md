# OSCON 2006:蒂姆·奥莱利主题演讲

> 原文:[https://www.sitepoint.com/oscon-2006-tim-oreilly-keynote/](https://www.sitepoint.com/oscon-2006-tim-oreilly-keynote/)

本周，凯文·杨克从俄勒冈州波特兰市的 [OSCON 2006](http://conferences.oreillynet.com/os2006/) 发回报道。

今天，在 OSCON 开始了第一天的公开会议，Tim O'Reilly 发表了他对开源现状的看法，以及一些目前正在掀起波澜的伟大想法:

**软件开发之外的参与架构(Web 2.0)**

作为 Web 2.0 迷因(以及相关的[会议](http://www.web2con.com/web2006/))的冠军，奥赖利在这个问题上有很多话要说，我在之前已经[报道过这个问题。他认为，Web 2.0 概念的最大特点是将控制权交给用户，这样当你的业务开始突破时，你的用户会觉得是他们实现了这一点，并成为你最大的支持者。](https://www.sitepoint.com/web-20-is-about/)

**开源许可已经过时** 
尽管 Apache 等实体做出了巨大努力，但开发人员仍在继续开发不符合现有许可模式的应用程序。奥赖利举的主要例子是[宁](http://www.ning.com/)，在这个网站上，普通人可以克隆自己喜欢的网络应用程序，做一些修改，然后自己点击几下就可以启动。你用哪个执照做那个？

**不对称竞争**

小公司(如 Craigslist)正在与大公司(如 Yahoo！).人们担心网络的商业成功会将小企业拒之门外，但这种担心并没有成为现实，对于有好主意的个人来说，仍然有机会大赚一笔。

**运营为优势**

许多大的成功案例，如谷歌和 Craigslist，正在解决数据中心的电力成本等问题。但是，小型企业如何才能获得同样的优势呢？Tim 和其他人预见到了小玩家的网络应用托管在大玩家的数据中心的未来。

**打开数据**

这是我在之前[报道过的另一个。简而言之，像](https://www.sitepoint.com/web-20-connectedness/)[微格式](http://microformats.org/)这样的创新使得以前被锁在服务器端数据库中并且只以人类可读的文档格式公开的数据，现在变得可供应用程序和自动爬虫访问。这些同样的创新让用户可以轻松地将数据从一个应用程序中取出，放入另一个应用程序。

奥赖利雷达上的其他亮点包括作为平台的 Firefox、IP 语音(VoIP)和 Ubuntu Linux。
