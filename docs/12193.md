# 为什么您需要强大的电梯间推介

> 原文：<https://www.sitepoint.com/why-you-need-a-powerful-elevator-pitch/>

传统上，电梯间推介是对你要传达给潜在投资者的商业想法或项目计划的快速总结，以争取他们的支持。这被称为电梯推销，因为你应该能够在乘电梯从顶层到底层的时间内完成你的整个演讲。

在实践中，如果你把电梯间推销看作是你人际关系工具箱中的一个工具，它可以完成更多的事情。如果你成功地发展了自己的人际关系，并战略性地使用它，它可以让你在创造新的线索、发展关系和推进职业发展方面更加成功。

为什么要为电梯间推介费心呢？

我相信你听过这样一句话，“你只有一次机会留下好的第一印象。”没错，同样没错的是，我们通常不会考虑第一印象，直到为时已晚。电梯间推介可以帮助你充分利用每一个第一印象，同时使人际交往变得更容易、更有成效。

当你告诉别人你是做什么的时候，你有没有感到有点张口结舌，或者只是不舒服地谈论自己？电梯间推介给了你一个现成的介绍，可以减轻你在人际交往中的压力。此外，当你有一个训练有素的电梯推销时，你可能会表现得更加自信，让别人更愿意信任你，认为你是一个值得结识的人。

创建电梯间推介的过程是一个专注和自我分析的练习。即使你很少使用它，开发它可以让你用最简单的术语思考你是谁，你做什么，你想让别人了解你什么。这可能是一个回顾你的成就和展望你的目标的机会，让你专注于你最初试图通过社交网络实现的目标。

我在电梯推销中应该说些什么？

创建电梯间推介的目标是起草一份陈述，向没有任何行业经验的人解释您做什么、如何做以及为谁做。应该直截了当，容易理解和消化。

这里有一个非常简单的公式，可以作为有力的电梯推销的基础:

*   第一部分:问对方一个问题，找出一个共同的问题。这吸引了他们，并为你进入第二部分提供了线索。
*   第二部分:简要介绍你所做的工作，以及它如何解决你在第一部分中发现的问题。
*   第三部分:提供行动号召，明确告诉对方你希望他们现在做什么。

电梯推销听起来可能是什么样的示例:

> 你有没有试过在一家与你有业务往来的公司里寻找某人的联系信息，但却因为他们的网站混乱无序而找不到？嗯，我做的是为需要更强大、更连贯的在线形象的小企业重新设计网站。通过尽可能多地了解这家公司，我制定了一个战略计划，旨在改造现有网站，使其功能更强大、用户界面更友好。你知道任何可能需要网站改造的小企业吗？这是我的名片……

创建电梯间推介的方法有很多(参见本文末尾列出的参考资料链接)。无论你使用何种形式，电梯营销最重要的部分是确保你非常了解它。你练习得越多，它就越自然，越能有效地创造新的机会……让你在人际交往中更加成功。
