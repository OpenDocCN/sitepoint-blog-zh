# 欢迎来到网络商务博客

> 原文：<https://www.sitepoint.com/welcome-to-the-web-pro-business-blog/>

大家好，欢迎来到 Sitepoint 的“正事”博客的新篇章！我要感谢 Simon 和 SitePoint 的全体员工给我机会参与到这样一个有用且被广泛阅读的博客中。我很高兴能在一个新的层面上与 SitePoint 社区互动，并将尽我所能使这个博客尽可能地丰富和有趣。

我还要感谢 Andrew Neitlich 在过去两年中作为 SitePoint 商业博客作者所做出的杰出贡献。像你们中的许多人一样，我应用了从安德鲁那里学到的销售和营销经验，我的企业因此大幅增加了收入。我肯定会关注安德鲁接下来会做什么！

关于我:我是一个终生的计算机爱好者，我出生在互联网(更准确地说是阿帕网)诞生的同时。1985 年，我成为一名专业技术人员，在一家计算机和零售商店为 PC 和 MAC 提供支持和维修服务。后来我进入了软件世界，为企业、电影制片厂、初创企业和政府开发大型应用程序。我继续担任财富 50 强和政府客户的过程和方法顾问，但是我的主要焦点是 web 开发和外包领域。这些年来，我也做过护理人员、潜水教练和音乐家，并在印度、新西兰、荷兰和香港生活过，但目前我在路易斯安那州、加利福尼亚州和泰国生活。

我是 Sagewing Corporation 的创始人兼总经理，这是一家外包和咨询公司，致力于通过外包服务帮助 web 开发人员成长。由于 Sagewing 对网络专业人员的关注，我有了一个独特的机会与数百个自由职业者和精品开发商店合作，并很好地了解了如何在嘈杂的 WWW 市场中“成功”。

展望未来，我希望将这篇博客的主要焦点从销售和营销转移到 web 开发业务的一些新方面。我将讨论 web 开发人员面临的一些日常管理问题，如提案写作、税收、簿记和法律事务。此外，我希望提供一些有趣和有用的观点，深入到更抽象(但同样重要)的商业元素中，如谈判策略、道德、时间管理、关系建设，以及最重要的让你的企业为你的生活带来更多金钱的艺术。

现在我想听到更多关于你，读者，以及你希望从这个博客中学到什么。我邀请你们所有人留下评论或者给我发一个 PM(T1)，让我知道你希望这个博客朝什么方向发展！一如既往，我最感兴趣的是听听你的最终业务目标是什么，以及你最想从你的业务中长期得到什么。我会尽力满足，希望我们也能玩得开心。
