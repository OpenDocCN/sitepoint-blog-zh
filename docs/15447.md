# Linux 工具包

**[AlgART HTML Packer](http://siams.com/root/products/ahp/packer.htm)–创建压缩的、自解压的 HTML 代码，您可以在大型网页上使用它来减少用户下载时间。**

[ASP 2 PHP](http://home.i1.net/~naken/asp2php/)–ASP 2 PHP 将微软的动态服务器页面转换成 PHP。在 Windows 和 Unix 上运行。

CoffeeCup HTML 编辑器–coffee cup HTML Editor ++是一个基于 GTK 的 Linux HTML 编辑器。它非常强大，包括许多功能，包括:样本 javascript 代码、动画 gif、背景、设计向导、图标、php 代码等等。

html-update-img 是一个用高度和宽度标签更新 html 文件中的 img 标签的程序。

HTML Tidy 是一个免费的工具，可以修复编辑 HTML 时出现的错误。它还将整理松散的 HTML 代码，使其更具可读性。

HTML-Tree 是一个程序，它可以查看你网站的所有目录，并在网络服务器上创建一个基于网页的 HTML 页面的图形地图。

[imaptool](http://www.sspitzer.org/imaptool/)–imaptool 是一个用于创建客户端图像地图的 X/Motif 实用程序。它支持所有三种形状(矩形、圆形和多边形)，并加载 JPEG 和 GIF 图像。

javascript 调试器–一个用于网景通信器的 Javascript 调试器。对于需要创建和调试自己的 JavaScript 代码的高级管理员非常有用。

JHTML 是一个基于 Java 的 HTML 编辑器，开发该编辑器的目的是为了最终支持一些特性，比如智能感知类型的自动完成功能，包括标记及其可能的参数、语法高亮显示、右击标记编辑以及许多其他有用的特性。

Linbot 是网站管理员的专业网站管理工具。Linbot 允许网站管理员查看网站的结构，跟踪断开的链接，找到可能过期的网页，列出指向外部网站的链接，查看内嵌图像的组合，获得按作者分类的问题列表，并在没有用户干预的情况下定期完成所有这些工作。

LinkCheck 是一个软件包，它可以检查一个网站是否有断开的链接。

nodoze 是一个 Perl 脚本，用于修复用 MS Windows 构建的网站，以便它们可以在 UNIX 服务器上正常工作。它将所有文件和目录名，以及 html 中的所有相对 URL 都改为小写，并且它将所有。. html 的 htm 扩展。

NUKE 是 Linux 预览新闻网站上使用的源代码。这就像 Slashdot 和其他一些类似的网站。它管理新闻、旧新闻、每篇文章评论、调查以及新闻站点需要的其他功能。

一个 java/javascript 网站管理程序，它与 java 高度集成，允许将 Java 小程序轻松整合到您的网站中。

SCREEM 是一个设计用来快速创建和维护网站的程序。它使用带有单个 HTML 标签选项的扩展列表，以及更高级的制作表格或指定样式的工具。功能包括网站范围的模板、查找和替换以及预览。

[SlashCode](http://www.slashcode.com/)–用于运行 Linux 新闻网站 Slashdot(http://www . Slashdot . org/)的代码(基于 mod_perl)。具有新闻页面，可配置的“SlashBoxes ”,可用于显示其他网站的标题，以及一个非常复杂的评论系统，具有审核功能。

[Visual DHTML](http://developer.netscape.com/docs/examples/dynhtml/visual/index.html)–Visual DHTML 是一款免费的基于 web 的创作工具，允许您使用各种 DHTML 技术创建交互式 Web 内容。它是用 DHTML 编写的，从浏览器运行。

webgrep 是一个小型工具包，为网页设计者提供了 7 个不同的程序。它包含了一个叫做 blnkcheck 的非常快速的断开链接检查器。
