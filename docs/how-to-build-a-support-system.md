# 如何建立一个不可动摇的支持系统

> 原文：<https://www.sitepoint.com/how-to-build-a-support-system/>

我们都可以从一个稳固的支持系统中受益，尤其是我们这些自雇人士。因为我们通常没有在许多公司和教育环境中有机的办公室友情和同事支持，所以建立我们自己的家庭、朋友和同事的个人支持系统是至关重要的。

无论你是准备开始为自己工作，还是想从现有的支持网络中获得更多鼓励，这里有一些方法可以扩大和增强你的核心支持团队。

**消除消极**

在我们的生活中，总有人喜欢唱反调。虽然这有时非常有益，但这些人通常不会成为你支持团队的最佳成员。重视他们现实的观点，并向他们寻求解决问题的思路和问题预测，但你可能想把他们排除在你的核心啦啦队之外。

遵循给予和接受的原则

消除消极情绪之后，在你开始寻求支持之前，要明白这不可能是单行道。对于你的支持团体的每一个请求和你得到的每一点鼓励，你都应该回报。为了发展一个互惠和可持续的团队，成为你团队中每个成员的支持者，并帮助他们实现自己的成功。
 **分享你的故事**

当你想获得他人的帮助时，公开透明是重要的一步。单独接近每个人，解释你的情况，并概述你对他们角色的设想。在建立一个持久的团队时，让每个人都保持一致可能是决定性因素。
 **求助专家**

你的支持团队中可能会有一些人在他们的工作中表现出色。挑出他们，并在必要时以导师或顾问的形式询问他们的专业知识。你不仅能从他们的经验中受益，而且他们也会对你的成功感兴趣。

**超越显而易见的事物思考**

是的，你的直系亲属、密友和行业同事可以成为你支持团队的重要成员。但是不要限制自己。扩大你的视野，考虑你在非传统渠道认识的人——比如社交媒体。你在网上建立的关系可以帮助你发展一个共享支持的社区，提供你可能无法找到的观点、经验和知识。

如果你被积极和鼓励所包围，在事业中接受和克服挑战会变得容易得多。事实上，如果你有几个关键人物的支持，你会发现冒险并不那么可怕。如果你愿意冒更大的风险，心理和经济上的好处会令人震惊。

你是如何建立你的支持系统的？它是如何帮助你创造成功的？
