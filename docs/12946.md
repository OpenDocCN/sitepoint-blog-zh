# 待售:网络应用-未使用

> 原文：<https://www.sitepoint.com/for-sale-web-application-unused/>

在我看来，网络应用市场正在走向它自己的网络泡沫破灭。基本要素都在那里，风险投资者向初创企业投入巨额资金。一种普遍的氛围:如果你建造了它，他们就会来。然后(当然)谷歌，雅虎！或者微软会收购。在所有炒作的背后，似乎有越来越多的[初创企业走向绝境](http://www.techcrunch.com/tag/deadpool/)。为什么？一些原因(在我看来)是:

*   它们非常小众
*   他们是轻浮的，几乎没有实际用途
*   他们的收入模式存在根本性缺陷

新的网络应用正在推出，因为风投正在向新的创业公司砸钱，希望它们能成为下一个脸书或 YouTube 的第一批投资者，从而成为下一个 10 亿美元销售额的一部分。很多这样的创业公司似乎都忽略了最基本的商业原则——赚钱！我完全支持创业，但如果你要进入商界，就应该是打造一种能够自立的产品，而不会被三大巨头之一收购。

当前的许多网络应用只是浪费时间——它们没有做任何有用的事情(依我看)。并且市场非常有限(有人会说是利基市场)。年轻、时髦和自恋的人可以在 Twitter 上更新他们醒着的每一个时刻，但是，更广阔的世界会蜂拥而至吗？我表示怀疑。你能想象你奶奶使用推特吗？(<q>在世界野生动物基金会决赛前泡一杯可爱的茶</q>)。

这些可笑的名字是怎么回事？老实说，他们忘了域名的黄金法则了吗:

1.  让它令人难忘
2.  让它变得可以拼写
3.  保持简短

大多数人在第三题上得了满分。但是，否则就是 *i* -this，或者遗漏字母 that，或者元音过多 other。然后是完全拼错的或编造的单词。
我承认，很难得到一个涵盖这三个规则的域名，尤其是所有三个字母、四个字母和字典单词。com 域名已经注册。因此，我们需要变得聪明一点，但是整个 web 2.0 命名的事情已经偏离了轨道。仅仅因为你的名字像 Flickr 一样拼写错误，并不意味着你的服务和 Flickr 一样好或有价值。del.icio.us 的流行是因为域名黑客的巧妙运用——不是，域名黑客是个好主意吗？这是有争议的。

当然，并不是所有的网络应用都有奇怪的名字，也没有明显的收入来源。有一些很棒的网络应用有着很棒的名字——举两个例子来说，[RememberTheMilk.com](http://www.rememberthemilk.com)和[背包](http://www.backpackit.com/)。

互联网的一大优点是，它的准入门槛低得令人难以置信。如果你有时间，技能和资源，你可以创建和营销一个网络应用程序。这种进入的便捷也有另一面——退出的速度。网上竞争是残酷的，竞争会迅速适应并挑战你的市场。它不像汽车行业，在那里需要数十亿美元的研究和开发，以及数年的工作来生产一种新的模式。新的网络应用程序可以在几天内完成，有时甚至需要几个小时。永远不要忘记互联网用户是一群无情的人——如果他们不喜欢它或者看不到它的意义，那么点击一下鼠标就有数十亿的其他页面可供探索。如果你不能用一句话解释清楚，你的 web 应用实际上是做什么的——那么很可能没有人会花足够长的时间去寻找答案。

如果你正在寻找创造下一个大东西——确保它满足需求和/或解决问题，并有一个比谷歌花 10 亿美元收购它更好的赚钱计划。TTFN。
