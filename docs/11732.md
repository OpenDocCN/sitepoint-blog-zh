# Xara Xtreme Pro 5:设计师的视角

> 原文:[https://www.sitepoint.com/xara-xtreme-pro-for-designers/](https://www.sitepoint.com/xara-xtreme-pro-for-designers/)

![Xara Xtreme Pro](../Images/1abe721b8c732c31966a8c9a4f2c3f37.png)

昨天，Craig 从一个 web 开发者的角度写了一篇关于 Xara Xtreme Pro 的文章。我也从一个设计师的角度回顾了这个项目。只是要绝对明确，这不是一个付费的审查，也没有我与 Xara 的任何从属关系。

这个版本的程序旨在将大量插图工具、桌面出版工具和网页设计工具打包在一起。如果你用过 CorelDraw 或 Inkscape，你可能会觉得界面很熟悉。像大多数图形程序一样，有一个工具栏，一个用于创建和编辑图像、网页和打印页面的文档窗口，以及一组用于每个工具的上下文相关控件。

你马上注意到的一件事是缺少面板或调色板。如果你需要他们，他们就在那里，但是他们会很好地隐藏起来，让你在工作区域有足够的喘息空间。

![interface](../Images/f6c0bd6f048a69d9eb53cba53873c447.png "interface")

**矢量图**

说到矢量插图工具，Xara 阿哈非常强大的血统，多年来一直在生产高质量的矢量绘图程序。如你所知，矢量图形是无限可扩展的，不会损失清晰度或质量。它们为设计者提供了一定程度的灵活性，这是基于像素的图形所不具备的。然而，有一种看法(不真实)认为矢量绘图非常困难，这往往会吓跑使用矢量程序的人。

那么 Xara 对于矢量图来说怎么样呢？实际上，这很棒。

所有矢量绘图的常见嫌疑人都在这里的形式，文字工具，形状工具，手绘工具和贝塞尔绘图工具。“形状编辑器”工具允许您在路径上的任何位置抓取一条线段，并拖动以更改曲线。这是一种更直观的处理曲线的方式，选择一个锚点，然后拖动贝塞尔手柄。要在 Illustrator 中做同样的事情，您需要安装一个插件，如 Xtreampath。

![shape-editor](../Images/de220396d2c3c6423c8c4e54692b8b9e.png "shape-editor")

Xara 最引人注目的一点是它的渲染速度。下面你可以看到一些用轮廓工具，斜面和 3D 工具操作的物体。虽然下面的示例是简单的形状，但对于更复杂的形状和类型，3D 挤出速度非常快。
![shapes](../Images/f5ce32621f4d1857b96dc745da58ff4c.png "shapes")

绘图、网页设计和桌面出版都有强大的色彩工具。乍一看，似乎只有 30 种颜色可供选择，但是颜色选择器有数百万种颜色可供选择。一个很好的特性是能够在你的设计中链接相关的颜色，然后应用一个全局的改变。透明工具对于制作网络背景和时尚淡入淡出很有用。

![colour-picker](../Images/b7c1ee3d0f15c646790b0255efaf46a9.png "colour-picker")
T2】

但这不全是关于矢量的。

Photo Designer 7 是 Xtreme Pro 套装的一部分。这是一个基于像素的图像编辑器，涵盖了许多你知道和喜欢的照片操作，以及一些不错的效果和过滤器。

![PhotoDesigner7](../Images/fef9d42d049b603a603de73b29d9a263.png "PhotoDesigner7")

很自然地，每个图像编辑器都被比作 Photoshop。这不是 Photoshop。但是，编辑器提供了一些工具，允许您准备和改进图像，以便在 web 上和桌面出版工具上使用。网页设计者会喜欢图像压缩，因为它在保持质量的同时很好地压缩了图像的文件大小。

查看 Craig 的评论，了解关于 Xara Xtreme Pro 中的 [web 开发和 flash 工具](https://www.sitepoint.com/xara-xtreme-5-developer-review/)的更多信息。

**优点**

*   价格非常合理
*   优秀的矢量工具
*   良好的位图支持

**缺点**

*   可能太多面手了
*   Xara Xtreme 没有提供 Adobe 套件所能提供的一切，但它确实以非常合理的价格提供了大量好东西。

该产品有两个版本:标准版(89/69/79 美元)和专业版。你可以[在这里](http://www.xara.com/us/products/xtreme/advanced/)[对比两个版本，在这里](http://www.xara.com/)下载 30 天试用版。
