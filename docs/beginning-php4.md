# 开始 PHP4

> 原文：<https://www.sitepoint.com/beginning-php4/>

得益于程序员友好的语法和大量的社区支持，PHP 变得越来越受欢迎，可以与它在服务器端脚本领域的竞争对手匹敌，包括 Perl、ASP、Cold Fusion 和 JSP。虽然 PHP 被设计成一种向有 C 和 Perl 等语言经验的开发人员展示熟悉面孔的语言，但对于初学者来说，从头开始学习 PHP 也相对容易。尽管如此，很少或没有编程经验的人将从语言入门指南中受益匪浅，这正是 WROX 出版社的“PHP4 入门”所提供的。

在讲述了 PHP 开发所需的组件(PHP 本身和配置为使用 PHP 的 Web 服务器)的安装之后，这本书通过 Web 上服务器端脚本的基本概念指导读者，并继续讨论 PHP 语言的基本特性:语句、变量和数据类型。本书中唯一假定的知识是用 HTML 创建基本的网页，并且每个概念都有一个实际的例子来帮助避免任何混淆。

随后的章节解释了 PHP 的基本特性，包括:处理 HTML 表单提交，使用一些更高级的语言结构，保持 PHP 代码整洁以避免不必要的输入，以及所有重要的 PHP 程序调试。在介绍了语言的基本要素之后，本书继续介绍更高级的语言特性(从 cookies 到会话，从面向对象编程到文件处理)，始终采用面向任务的方法，并辅以实践示例，让读者掌握基本的编程技术。因此，这本书教读者如何使用 PHP 获得结果，即使这意味着错过了语言的一些微妙的细节。重点放在“实际”信息的覆盖面，而不是技术细节的深入研究。这特别适合那些不太懂技术，主要想知道如何完成工作的读者。

“PHP4 入门”由这本书最重要的三章组成，它对数据库连接的处理也是如此。通过向读者介绍一些概念，然后深入到一个实际的例子中，当必须介绍一些新的概念时再次浮出水面，作者设法给出了用 PHP 创建数据库驱动的 Web 应用程序所涉及的技术问题和编程技术的一个相当完整的概述。本书的这一部分主要是针对 MySQL 的，MySQL 是他们的例子中使用的数据库软件。MySQL 是目前最流行的 PHP 数据库解决方案，可以免费下载和使用，本书的这一部分提供了安装指南。

接下来的三章涵盖了 PHP 的高级但非常实用的特性:XML 解析、发送电子邮件和动态生成图形。像这些介绍性书籍结尾的高级章节有一种看起来仓促和不完整的趋势，但是我很高兴地发现这些章节构成了这本书写得最好的章节之一。最后一章介绍了一个完整的基于 Web 的 URL 目录管理器的开发。这个案例研究为读者提供了一个难得的机会，让他们看到整本书中涉及的许多主题在实际情况中的结合使用，对于那些即将开发第一个大规模基于 Web 的应用程序的读者来说，这将是非常有价值的。如果没有别的，这一章揭示了像雅虎这样的大型网络目录的内部工作原理。，提供有价值的洞察力，否则只能通过真实世界的经验获得。

在我看来，这本书的主要缺点是缺乏参考价值。我觉得，这主要是因为这本书的结构(或者说缺乏结构)。作为 PHP 的教程指南，这本书大放异彩；然而，根据我的经验，读者更有可能阅读一本书的前三分之一左右，然后跳来跳去学习他们感兴趣的具体概念，然后将它放在附近的书架上作为方便的参考。不幸的是，本书中的许多例子都依赖于前面章节的代码片段。在这种情况下，适当的参考文献(如“参见第 148 页的代码”)很少，这使得读者很难在文中的任意点跳转。有些主题很难找到，并且经常出现在本书看似不相关的章节中(例如，正则表达式的讨论隐藏在错误处理一章中)。这本书的结构并不仅仅是作为一个参考，试图使用它往往会导致挫折。

总的来说，这本书为初次接触 PHP 语言但几乎没有编程经验的开发人员提供了一个很好的指南。然而，作为一名经验丰富的 PHP 开发人员，我发现这本书的结构有些平淡无奇，并注意到为了方便讨论而牺牲了几处技术细节(例如，第 4 章中关于‘switch’语句的部分包含几处技术错误和误导性的示例代码)。如果你正在寻找一个容易阅读的、实践性的 PHP 介绍，“PHP4 入门”将会很好。然而，作为语言参考，PHP 的免费在线手册提供了更多。
