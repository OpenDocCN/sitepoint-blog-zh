# 苹果批准 Firefox Home 用于 iPhone 应用商店

> 原文:[https://www . site point . com/apple-approve-Firefox-home-iphone-app-store/](https://www.sitepoint.com/apple-approve-firefox-home-iphone-app-store/)

花了几周时间，苹果终于批准了 Mozilla 的 Firefox Home 应用商店。尽管我们最近的民意调查显示，17%愤世嫉俗的投票者认为苹果会拒绝它！

与 Opera 不同，Firefox Home 不是一个替代浏览器，但是在桌面电脑上使用 Firefox 的 iPhone 用户会对这个应用感兴趣。实际上，Firefox Home 是 iPhone 原生 Safari 浏览器的免费前端同步小工具。这是 Firefox Sync(以前称为 Weave Sync)的单向版本，可以将书签、历史记录和打开的标签转移到手机上。

要使用 Firefox 主页:

**1。在你的台式电脑上安装 Firefox Sync**
在你的浏览器上安装 Firefox Sync 插件。重启后，你会在选项对话框中有一个新的**同步**标签(你也可以点击附加组件窗口中的火狐同步“选项”按钮)。

按照向导创建新的同步帐户或指定现有帐户。注意，你必须(目前)选择 Mozilla 的服务器，而不是你自己的。您还可以选择书签、历史记录、密码、偏好设置和标签是否同步。

**2。在你的苹果设备上安装 Firefox Sync**
Firefox Home 适用于装有 iOS 3.1 或更高版本的 iPhone、iPod Touch 或 iPad。[直接从苹果应用商店](https://itunes.apple.com/us/app/firefox-home/id380366933?mt=8)下载。

当应用程序启动时，您可以输入 Firefox Sync 帐户的详细信息，您选择的设置将上传到设备。
