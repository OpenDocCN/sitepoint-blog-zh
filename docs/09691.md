# RC9au 采访 Dougal

> 原文:[https://www.sitepoint.com/rc9au-interview-with-dougal/](https://www.sitepoint.com/rc9au-interview-with-dougal/)

这是众多开发者访谈中的第一个。Dougal 帮助我度过了仅仅安装 Ruby 和 Rails 的阶段，并让我开始了一个真正的项目。

欢迎来到 RubySource。你能告诉我们一些关于你自己的情况吗？

我是 Dougal，我是一个网站的家伙，一个开发人员，我想是一个网站开发人员。主要是 Ruby on Rails，但不是唯一的。我喜欢骑自行车，越野，我不擅长的下坡，我走 B 线，更容易的路线，而不是艰难的道路。

你从事 Ruby 项目有多久了？

从 2007 年开始，那时 Rails 还只是第一个版本。

你还用过其他什么语言？

ActionScript，那是肮脏的过去。我以前和飞侠约会过，但现在她是个神经病前女友。我也做过 PHP，但是是以一种蹩脚的方式，主要是写脚本之类的东西。当然，我已经做了，并且还在做 JavaScript。

你对 Flash 不可避免的死亡有什么看法？

太棒了。什么时候发生？我可以带一个朋友去派对吗？

你现在在哪里工作？

我在墨尔本和 jTribe 一起工作了很长时间。通常是自由职业者，但我已经和他们在一起很长时间了。他们在移动领域，这是一个非常有趣和令人兴奋的领域。

你觉得 RoR 有什么特别吸引人或有趣的地方吗？

在我使用它的所有时间里，它一点也没有停滞不前。很多项目在成熟时可能会慢下来，但 Rails 会继续前进。它在不断发展，出现了许多新的最佳实践，推动了这类事情的发展。

我们能看看你做的东西吗？

我最近完成的作品之一，[luke.com.au](http://luke.com.au)

有什么至理名言吗？

对你妈妈好吗？如果你想了解更多关于 rails 的知识…[http://railscasts.com](http://railscasts.com)
