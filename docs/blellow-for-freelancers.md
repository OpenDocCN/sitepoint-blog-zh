# Blellow:自由职业者的新网站

> 原文：<https://www.sitepoint.com/blellow-for-freelancers/>

寻找一种新的方式来找到自由职业者的工作，建立关系网，并帮助你的自由职业者同伴？如果你是 Twitter 用户，Blellow 可能正符合你的胃口。Blellow 是一个面向自由职业者和创意专业人士的社交网络社区，他们会回答“你在做什么？”

它的工作方式很像推特。您关注并被其他 Blellow 用户关注，您输入状态更新(不超过 300 个字符)。最根本的区别在于它是基于工作的:你描述你正在做的事情，你遇到的问题，或者你在某个项目或问题上需要的帮助。然后，其他 Blellow 成员将会回复您的帖子，提供建议、解决方案或评论。

[![blellow](img/af99238797002c35c435bee58b164f61.png "blellow")](https://www.sitepoint.com/wp-content/uploads/2009/04/blellow.jpg)

以下是关于 Blellow 如何工作的更多信息。

**荣誉**

Blellow 有一个与他人分享知识和协作的驱动目的。经常帮助他人解决问题并找到解决方案的用户可以从某个问题或问题的发帖者那里获得“荣誉”。你可以给每个线程尽可能多的荣誉。这个想法是奖励给你帮助最大的人。

**组**

群组允许对主题进行组织和分类，例如 Photoshop、网页设计师、企业家等。通过加入一个群组，你可以立即与其他成员互动，而不必跟随任何人。

**项目**

项目允许您发布一个项目的帮助请求。项目可以是有偿的，也可以是无偿的。公益项目是无偿项目，旨在增加你的投资组合，但根据项目和受欢迎程度，可能会导致进一步的有偿工作。

**工作岗位**

工作区是一个工作板，允许企业和个人发布全职、兼职和合同职位的工作，类别包括设计、开发、写作和其他。

**聚会**

Blellow 提供了一个现场聚会的机会，可以进行网络交流、社交，或者只是认识你在 Blellow 上交往的人。

总的来说，这是一个有趣的新服务，一旦它起飞，可能会成为自由职业者的潜在动力，尤其是对那些习惯于 Twitter 的人来说。同事之间信息共享的目标本身就很好，但工作板和项目帮助的附加功能也可能使 Blellow 成为寻找自由职业者工作的一种新方式。
