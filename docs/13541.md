# IE 7 不兼容现象

> 原文：<https://www.sitepoint.com/ie-7-incompatibility-sightings/>

我的女朋友即将从教学专业毕业，正在这个地区找工作。在访问今天发布这些工作的政府官方网站时，她注意到了这条消息:

> **微软互联网浏览器 7**
> 
> 微软现在已经可以下载 IE7 版本了。网上招聘基于 PeopleSoft 平台，该平台尚未获得使用新版 Internet Explorer 的认证。已下载 IE7 的招聘在线用户无法成功登录招聘在线。目前没有可以改变的设置来解决这个问题。卸载 IE7 的说明可以从[微软网站](http://msdn.microsoft.com/ie/releasenotes)(【http://msdn.microsoft.com/ie/releasenotes】T2)获得

不错吧。如果这看起来像是试图把责任推给 PeopleSoft，那你就对了。根据[相关讨论组](http://tech.groups.yahoo.com/group/peoplesoft-fans/message/45161)和[其他消息来源](http://www.und.nodak.edu/cnd/)的说法，最近被甲骨文收购的 PeopleSoft 在 IE 7 上没有重大问题，甲骨文预计将很快认证 PeopleSoft 的当前版本以用于 IE 7。

还有人见过野外 IE 7 不备的情况吗？
