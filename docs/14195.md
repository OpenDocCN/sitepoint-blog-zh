# 如何写一篇有效的新闻稿

> 原文:[https://www . site point . com/how-to-write-an-effective-press-release/](https://www.sitepoint.com/how-to-write-an-effective-press-release/)

应前一篇博客读者的请求，这里有一些关于新闻稿的想法。第一，标题“如何写一篇有效的新闻稿”有点误导。这个想法是创造一个有效的媒体宣传活动。一份新闻稿很少能让你到达你想去的地方。

你需要把宣传想象成景观上的一股水流。从短期来看，水的作用不大。从长远来看，它可以雕刻出一个美丽的峡谷。这是因为媒体需要时间来认可你的作品，公众也需要更多的时间。

话虽如此，以下是一些建议:

1.看看竞争对手和其他公司在做什么，开始阅读新闻稿，看看哪些是有效的，哪些是无效的。去[http://www.businesswire.com](http://www.businesswire.com)看新闻发布会。

2.为你的宣传活动设定一个目标。我假设这里的大多数读者都不是上市公司的所有者，他们希望支撑/提高股价。因此，你的首要目标是在你的目标市场中产生知名度。

3.为您的新闻稿获取媒体和媒体联系人列表。如果你读过我写的任何东西，你就会知道我是一个专注的、基于行业的目标市场的大力倡导者。如果你遵循了这个建议，那么你会更容易找到一个目标出版物的小列表。

4.列出一份关于你公司的有新闻价值的事件。其中包括:新能力、新客户、新员工、社区服务、对协会的贡献(例如即将到来的演讲、会议)。也要考虑给你的听众一些实质性的提示，关于那些对他们重要的事情，以及你所知道的事情。出版物不想要广告，所以问问读者是否会觉得你的文章有趣，有新闻价值。

5.写一个好标题。例如:XYZ 网页设计公司为网站提供了五个吸引更多合法客户的技巧；ABC Web Development 宣布为食品连锁店发布全面的后台办公工具套件；DEF 系统集成商宣布与 IBM 公司签约。

6.保持简单和甜蜜的形式。从顶部开始:立即发布。然后放上你的标题。然后以发行日期和地点开始第一段(例如马萨诸塞州波士顿)。用一段关于你的公司和媒体联系人的电话号码和电子邮件来结束这篇文章。

7.你的新闻稿的结构应该遵循金字塔原则，也就是说，你从最重要的段落开始，然后往下写。例如，如果您宣布与客户签订合同:

7a。第一段开始:ABC 网页设计，一家领先的明尼阿波利斯网页设计公司，今天宣布…..

7b。第二段涉及更多细节。

7c。第三段应该引用你或 CEO 的话。

7d。第四段应引自任何其他相关方(如客户或合作伙伴的首席执行官)。确保该方批准发布，否则您可能需要发布撤销声明。

7e。添加任何其他信息，说明为何该公告对受众意义重大。

7f。提供一个关于你的公司和客户的“关于”段落。

8.你发行的数量取决于你有多少有新闻价值的内容以及你在目标市场上的发行频率。我为一些上市公司处理新闻发布，他们试图每周发布两个项目，一个关于新的基础设施，一个关于新的客户合同。

9.通过传真或电子邮件向您的联系人发送新闻稿。如果是电子邮件，把新闻稿放在邮件的正文中，因为由于担心病毒，附件通常会被删除。或者，如果你想覆盖全国，去 businesswire.com 看看，那里有很好的覆盖面(尽管比 prweb 贵)。

10.小心你与媒体打交道的方式。他们没有义务印刷你的新闻稿。不要纠缠或打扰他们。如果他们感兴趣，他们会打电话给你(当他们打电话时，立即回复，因为他们经常在截止日期前)。而且，如果你的发布充满了读者感兴趣的好内容，你最终会看到结果。

宣传游戏还有更多的内容，但对于那些不经常使用这种策略的人来说，这是一个很好的介绍。但是请记住——关键是要有长远的、耐心的眼光。
