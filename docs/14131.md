# JBuilder 模拟 WebSphere

> 原文：<https://www.sitepoint.com/jbuilder-to-emulate-websphere/>

Borland JBuilder 将加入 IBM WebSphere，成为一个基于开源 Eclipse IDE 专有扩展的 Java 开发环境。

正如我之前报道的那样， **Borland 将会把它的 JBuilder IDE 的一些独特的核心特性移植到 Eclipse 上**，这将是 JBuilder 的一个免费开源版本。直到今天，JBuilder 作为一个商业产品发生了什么问题还悬而未决。

据 [The Register](http://www.theregister.co.uk/2005/04/22/jbuilder_eclipse/) , **Borland 将继续销售 JBuilder 的扩展企业版作为商业产品**，但它也将为开源 JBuilder 组件提供商业支持服务，就像 IBM 销售其基于 Eclipse 的 WebSphere 开发环境一样。

确切地说**哪些特性将是开源的，哪些特性将保持商业化还有待观察**，但我预计免费的 [JBuilder 2005 Foundation](http://www.borland.com/jbuilder/foundation/) 版本中当前可用的所有内容都将迁移到 Eclipse，而开发人员和企业版将保持商业化。

在这种情况下，最大的变化将是当前的 JBuilder 用户，他们必须适应 Eclipse 环境。免费版的功能限制会和现在一样。
