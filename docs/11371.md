## Windows 工具

[T1】！快速截屏](http://www.etrusoft.com/) :一个免费的截屏和图像编辑工具，允许你以 10 多种方式从屏幕的任何部分截屏，并以 BMP、JPG 和 GIF 格式保存图像。

[](http://www.analogx.com/contents/download/system/capture.htm)**:一个免费的工具，可以让你捕捉整个屏幕，在任何颜色深度和分辨率，并保存到一个 BMP 文件。**

 **[**CaptureExpress**](http://www.captureexpress.com/) :一款付费工具，可以让你拉伸和移动你的拍摄区域，也可以拍摄各种形状的物品，然后打印、保存或通过电子邮件发送你拍摄的图像。

[**【capture ez Pro】**](http://www.screencapture.com/ezepro.htm):一款付费应用，可以捕捉多张图像，并保存为十几种文件格式。

**Cropper** :一个免费的屏幕捕捉工具，可以捕捉整个窗口或一个定义的区域，然后保存到剪贴板，直接发送到打印机，或者保存为 BMP、PNG 或 JPG 文件。

[](http://www.faststone.org/FSCaptureDetail.htm)**FastStone Capture:一个共享软件工具，用于捕捉和注释屏幕上的任何内容，然后发送到文件、剪贴板、打印机、电子邮件、Word/PowerPoint 文档或上传到网站。
![FastStone](../Images/45fa01b51150341dd0e072835eda2db0.png)**

 **[**FireShot**](https://addons.mozilla.org/en-US/firefox/addon/5648):Firefox 的一个插件，可以创建整个页面的截图或者只是一个可见的部分，并允许用户插入文本注释和图形注释。

[**Greenshot**](http://greenshot.sourceforge.net/) :一个开源工具，可以让你将屏幕截图或者屏幕的一部分保存到文件中，并对截图应用文本和形状。

[**HandySnap**](http://wisepixel.com/) :一个付费的截屏实用程序，用来拍摄计算机屏幕的任何部分的快照，并在捕获的图像中插入注释和各种图形。

[**Jet 截图**](http://www.jetscreenshot.com/) :一个免费的实用程序，可以在几秒钟内截图、编辑并分享到互联网上。
![Jet Screenshot](../Images/d77035bd811eaa2eb24c084942da9804.png)

[**MWSnap**](http://www.mirekw.com/winfreeware/mwsnap.html) :一款免费且免安装的工具，可以让你抓取整个桌面，高亮显示的窗口，活动菜单，控件，或者屏幕固定或自由的矩形部分。

[**截屏**](http://download.cnet.com/ScreenGrab/3000-2192_4-13469.html) :一个免费的截屏工具，使用热键组合，将文件保存为 BMP、TIFF、JPG 或 PNG。

[**截图捕捉器**](http://www.donationcoder.com/Software/Mouser/screenshotcaptor/) :一个免费的截图程序，运行在 Windows 和 u 盘上，包括可定制的设置。
![Screenshot Captor](../Images/c6783918a64cd0ac3de4f7b8c72c19fc.png)

[**ScreenStamp！**](http://www.screenstamp.co.uk/) :一个免费的工具，可以抓取你的屏幕，给截图命名，保存到一个预先定义的位置，并在图片上打上当前的日期和时间。

[**Snagit**](http://www.techsmith.com/snagit/) :一款付费工具，可以让你捕捉屏幕的任何部分，以 AVI 格式捕捉视频和记录屏幕动作，并对图像进行绘制、调整大小、重新着色和注释。
![Snagit](../Images/0563fa67abd81ce9dd7fcb28f2216208.png)

[**截图工具**](http://windows.microsoft.com/en-US/windows-vista/Use-Snipping-Tool-to-capture-screen-shots):Windows Vista 中的一个内置工具，可以让你抓取截图，然后注释、保存、分享图片。

[**Snippy:**](http://www.bhelpuri.net/Snippy/) 一个免费的工具，可以让你在屏幕上划出你想要复制到剪贴板的区域，粘贴到另一个应用程序中。

**[web shot](http://www.websitescreenshots.com/)**——一个免费的 Internet Explorer 扩展，允许你抓取网页截图，并以 JPG、GIF、PNG 或 BMP 格式保存为全尺寸图像或缩略图。

[**窗口剪辑**](http://www.windowclippings.com/) :一款付费应用，可以抓取截图，自定义设置，保存后在其他应用中使用。
![Windows Clippings](../Images/ae144c2730f88e324b1faf5c57b724fe.png)

ZScreen :一个免费的开源截屏程序，可以让你上传截屏、图片、文本和文件，只需轻轻一击就可以将网址放入剪贴板。

## Mac 工具

[](http://www.chimoosoft.com/products/captureme/)**:一个用于 Mac OS X 的免费屏幕捕捉和录制工具，具有浮动捕捉窗口、调整大小能力，并允许您保存为 MPEG-4 QuickTime 电影、PNG、JPG、TIFF 或 GIF。**

 **[**【哥白尼**](http://danicsoft.com/software/copernicus/) :一款免费的截屏工具，从你的屏幕上抓取图像和视频(没有音频)。

[**瞬拍！**](http://projects.digitalwaters.net/index.php?q=instantshot) :一个免费的截图工具，用于捕获整个屏幕，它的一部分或一个窗口，以及制作定时截图或随时间延迟捕获的选项。

[**狗仔！**](http://derailer.org/paparazzi/):Mac OS X 的一个免费工具，可以抓取不适合一个屏幕的网页截图，显示预览，并允许你将截图保存为 PNG、JPG、TIFF 或 PDF 文件。

[**Skitch**](http://skitch.com/) :一个免费的截图工具，包括用文本、图形等进行注释的能力。

**[snap drag](http://www.yellowmug.com/snapndrag/)**:一个免费的工具，让你只需点击一个按钮，拖动窗口就可以截图，并保存为 PNG、TIFF 或 JPG。

**[WebKit 2 png](http://www.paulhammond.org/webkit2png/)**——一个用于 Mac 的命令行工具，可以抓取 PNG 截图。

## 跨平台工具

[**【Jing】**](http://www.jingproject.com/):一款免费工具，可以让你捕捉图像，并为你在屏幕上看到的内容创建视频，然后添加评论和注释，并保存、通过电子邮件发送或在线共享。
![Jing](../Images/69aa3a40f5c3ab5317341996b0832d51.png)

**[screen grab](https://addons.mozilla.org/en-US/firefox/addon/1146):**Firefox 的一个附件，它可以捕捉你在窗口中看到的东西，整个页面，只是一个选择，或者一个特定的框架，并允许你保存到一个文件，或者剪贴板。

[**WebKut**](http://toki-woki.net/p/WebKut/) :一款免费的 Adobe AIR 应用，可以让你抓取整个页面，当前视图，或者只是一个选区。
