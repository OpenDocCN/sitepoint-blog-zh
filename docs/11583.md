# 11 种虚拟机解决方案，简化您的跨平台检查

> 原文:[https://www.sitepoint.com/virtual-machine-solutions/](https://www.sitepoint.com/virtual-machine-solutions/)

几乎每个设计人员和开发人员都曾希望拥有一个以上的操作系统，但由于资金或物理空间的限制，并不是所有人都能负担得起多台计算机。

这就是虚拟机派上用场的地方，因为它们允许您在当前的桌面上运行多个操作系统，成本很低，有时甚至是免费的，而不必购买单独的机器。我们已经收集了 11 种解决方案，应该可以满足您的所有需求。

## 运行多个操作系统的虚拟机

**[Bochs](http://bochs.sourceforge.net/)** : Bochs 是一个方便的虚拟机，可以在许多系统上运行，甚至可以在基于 ARM 的 IPAQs 上运行。客户操作系统不胜枚举，但它们包括了所有常见的可疑操作系统，包括一些像 OS/2 和 QNX 这样的另类操作系统。

[![](../Images/937d3be858e241de7bb067c3dc02b0d3.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/bochs.png)

**[Parallels](http://www.parallels.com/products/desktop/)**:Parallels 可能是最知名的虚拟机，因为它是最容易使用的虚拟机之一，并且因为它是第一个与英特尔芯片合作将 Windows 引入苹果系统的 Mac 主机而受到最多关注。虽然 Parallels 以其运行 Windows 的能力而闻名，但它也能支持 Linux、FreeBSD、OS/2 等。

[![](../Images/a0d8aa09bfd0b0241affa6420c09d788.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/parallels.png)

**[Parallels Desktop](http://www.parallels.com/products/desktop/pd4wl/)** :虽然 Parallels 最常与 MAC 联系在一起，但也有针对 Windows 和 Linux 系统的版本。使用 Parallels Desktop，您可以在系统上运行 Windows、Linux、FreeBSD、OS/2、eComStation、MS-DOS 和 Solaris。

[![](../Images/70cba9f6c23ff9369b3ecb5b95b8519c.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/parallelsdesktop.png)

**[VirtualBox](http://www.virtualbox.org/)**:VirtualBox 是 Sun Microsystems 的 x86 虚拟化解决方案，免费开源。它可以在 Windows、Linux、Macintosh 和 OpenSolaris 主机上运行。客户操作系统包括 DOS，几乎所有的 Windows、Linux、Solaris、OpenSolaris、OpenBSD 等等。

[![](../Images/d95a8fe60ece5ddf954b9f95a302edb9.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/virtualbox.png)

**[虚拟机管理器](http://virt-manager.org/)** : Virt Manager，更普遍的说法是，它是一个虚拟机管理器，允许您在 Red Hat Linux 安装上运行具有多个操作系统的本地和远程虚拟机。

[![](../Images/510b8f9deabdae7ed737670df01935cf.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/virt-manager.png)

**[VMWare](http://www.vmware.com/)** : VMWare 作为一家公司被认为是系统虚拟化市场的行业领导者，因此该公司为操作系统虚拟机提供了如此多不同的解决方案，几乎不可能一一列出。可以这么说，如果你有一个操作系统，他们可能有一个解决方案让你在你的系统上运行另一个操作系统。

[![](../Images/3452f08bbec9926f07c00d323c605aac.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/vmware.png)

**[Xen](http://www.xen.org/)** : Xen 运行在 NetBSD、Linux 和 Solaris 系统上，模拟 FreeBSD、NetBSD、Linux、Solaris、Windows XP、Windows 2003 Server 等。该程序是在 GPL 许可下发布的。

[![](../Images/97b67838dfaf10d6a0a89734699a10c3.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/xen.png)

## 运行微软操作系统的虚拟机

**[Boot Camp](http://www.apple.com/macosx/compatibility/)**:Boot Camp 是 Mac OS X 过去几个版本中包含的一个实用程序，允许基于英特尔的 Mac 在 Macintosh 环境中运行 Windows XP、Vista 或 Windows 7，或者将系统重新启动到。

[![](../Images/288264f61a96e66f6358059be1ff50af.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/bootcamp.png)

**[DOSBox](http://www.dosbox.com/)** : DOSBox 是围绕运行 DOS 游戏而构建的，但有些人已经成功地让 Windows 3.1 在其中运行。由于该计划的架构，DOSBox 已被移植到许多不同的操作系统，包括 Windows，BeOS，Linux，Mac OS X 等。

[![](../Images/a7423ddc72c2cb6e54e99fe2d3c213c1.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/dosbox.png)

**[Internet Explorer 应用程序兼容性 VPC 图片](http://www.microsoft.com/downloads/details.aspx?FamilyId=21EABB90-958F-4B64-B5F1-73D0A413C8EF&displaylang=en)** :有没有想过你的新设计在 SP3 Windows XP 上运行的 Internet Explorer 6 中会是什么样子？没有安装吗？没问题！微软为您提供了大量图片，让您了解在 Windows XP 和 Vista 上运行的各种版本的 Internet Explorer 的外观。Windows 7 全速向我们走来，应该很方便。

[![](../Images/1a94c04290b865a59059155d6cfb6e73.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/ie-vpc-images.png)

**[微软虚拟个人电脑](http://www.microsoft.com/windows/virtual-pc/default.aspx)** :微软虚拟个人电脑原名为虚拟个人电脑，是 Windows 7 中的一项新功能，它将允许您在一个名为“Windows XP 模式”的新功能中运行 Windows XP SP3 的授权版本各种系统配置上的旧版本程序能够运行许多版本的 Windows，在某些情况下，你甚至可以说服它运行 Linux。

[![](../Images/ba5f89f23bf11e8f4f4b15c8f7d2d78a.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/vms/virtualpc.png)
