# 如何卸载 IE9

> 原文：<https://www.sitepoint.com/how-to-uninstall-ie9/>

微软报告说，在头两天，IE9 测试版下载量超过了 200 万次。虽然[浏览器不错](https://www.sitepoint.com/ie9-beta-review)，但也不是没有问题。例如，没有额外的真实或虚拟 PC，可靠的 IE8 测试变得不可能。IE9 还将 XML 呈现为不带错误检查的纯文本。在启动对我的工作至关重要的 IE 专用系统时，我也经历了频繁的崩溃。它仍然是一个测试版，所以我们可以原谅微软浏览器的缺点，如果需要的话可以卸载。或者我们可以吗？IE9 没有出现在控制面板的“程序和功能”中！

幸运的是，微软并不残忍——你可以移除浏览器而不需要重新安装操作系统。请遵循以下步骤:

*   关闭所有应用程序。
*   打开控制面板，进入**程序和功能**。
*   点击左侧任务窗格中的**查看已安装的更新**。
*   滚动到底部，你会找到 **Windows Internet Explorer 9** 。双击它，在*处点击**是**“您确定要卸载此更新吗？”*提示。
*   按照说明操作，并在出现提示时重新启动。

整个过程将需要 5-10 分钟，取决于你的电脑速度。IE7/8 将重现辉煌。我打赌你没错过！
