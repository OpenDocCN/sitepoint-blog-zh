# 纯 CSS 的力量

> 原文:[https://www.sitepoint.com/the-power-of-pure-css/](https://www.sitepoint.com/the-power-of-pure-css/)

随着所有这些很酷的 JavaScript 和 DOM 的出现，有时很难记住有可能将一些纯 CSS 和 HTML 扭曲成一个没有任何脚本的交互式网站。我怀疑这里的大多数读者都看过 Eric Meyer 的 css/edge，在那里他展示了 css 的一些简洁的功能，但是你还可以做得更多。

进入斯图·尼科尔斯和他的 [CSS 游乐场](http://www.stunicholls.myby.co.uk/index.html)。第一个例子会让你觉得“那只是 CSS？?"就是[那个神奇的谜题](http://www.stunicholls.myby.co.uk/menu/amazing.html)。试试看。然后说“哦，真聪明”。

通过纯 CSS 实现某种形式的交互的主要方法是使用:hover 指令，而阻碍它的主要原因是 IE 不支持:hover 在链接之外的任何东西上。尽管如此，一些令人印象深刻的事情是可能的:例如，见[斯图的照片库](http://www.stunicholls.myby.co.uk/menu/gallery.html)。这并不是说这些技术是革命性的，更多的是说它们确实存在，并且可以被利用来向您的站点添加一些简洁的东西，而根本不需要任何脚本。
