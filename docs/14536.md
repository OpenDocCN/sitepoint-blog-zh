# 很高兴见到你！

> 原文:[https://www.sitepoint.com/nice-to-meet-you/](https://www.sitepoint.com/nice-to-meet-you/)

我很高兴成为 SitePoint 社区的一员，并有机会定期在博客上发表我喜欢做的事情！我有一种感觉，当我分享我所知道的设计理论和实践时，我会在研究和解决你们提出的问题时学到更多…所以我期待你们所有的评论和问题！

因为我刚刚开始，这里有一点关于我的:

*   我喜欢项目清单。
*   非计算机兴趣包括阅读、烹饪、吃饭、演奏音乐，以及为我的第一次铁人三项训练。
*   一年夏天，我姐姐“雇佣”我帮助她设计技术手册，从此我的平面设计生涯开始了。我要在 Illustrator 中对齐矢量点。哇哦。(那是在 PageMaker 还在 Aldus 手下的时候。)
*   从那以后，在网页开发和设计的实践领域，我学到了更多关于 Illustrator、Photoshop 之类的东西。请在接下来的几个月中寻找更具体的操作指南。(我从一个关于在 Photoshop 中使用图层蒙版[的基础到中级教程开始。)](https://www.sitepoint.com/article/photoshop-fade-layer-mask)
*   在遇到 ImageReady 的切片服务 HTML 生成器之前，我已经学习了 HTML，我属于 web 开发领域中“遵循规范，正确做事”的那一类，所以你会发现关于设计理论的评论和文章倾向于可用性，并与可访问性重叠。
*   最后，我喜欢帮助人们学习新事物。我喜欢学习新的东西。所以这个博客是一段美好关系的开始！

现在是我无耻地征求反馈、询问和评论的时候了。什么样的文章、博客、研究等。，你在这一带找吗？你希望更多地了解哪些设计问题或 Photoshop 技巧/窍门？你在别的地方看到过什么很酷的东西，你想学着怎么做？贴个评论！
