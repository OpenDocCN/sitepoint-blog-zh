# 凯西·格里芬商业博客指南

> 原文：<https://www.sitepoint.com/the-kathy-griffin-guide-to-business-blogging/>

如果你乐于接受，总会在你最意想不到的地方学到一些东西。昨晚我自己也经历过这种情况，当时我熬夜写了一篇博文，内容是一系列商业博客技巧，背景是电视上的凯西·格里芬。这种意想不到的组合导致我头顶上的灯泡熄灭。两个啮合；刚刚成功了。

如果你不认识她，[《我的生活》中的](http://www.bravotv.com/kathy-griffin-my-life-on-the-d-list)[凯西·格里芬](http://www.kathygriffin.net)是一位有趣且极具争议的单口相声演员兼女演员。她最出名的是关于名人的半开玩笑的笑话，但她不知道的是，如果她将使她成为受欢迎的单口相声演员的公式应用到商业博客上，她会成为一名伟大的商业博客作者。让我解释一下。

## 她喜欢笑

凯西是喜剧演员；她的工作是让人们发笑。但是幽默不应该严格地留给专业人士，信不信由你，它在你的商业博客上有一席之地。

商业博客通常严肃、直截了当，而且我敢说很无聊。但是分享商业相关的信息不一定要这么枯燥。

虽然我不建议讲任何黄色笑话或取笑你所在行业的名人，但在你的博客上采取一种更轻松的方式可以让它变得更有趣。一些想法:写一篇描述两个截然相反的事物之间相似之处的滑稽帖子(就像这篇)，讲述一个相关个人经历的有趣轶事，或者使用幽默的照片来配合你的所有商业帖子。

## 她大声说出来

凯西·格里芬并不羞于表达自己的观点。众所周知，她与某些人和组织有矛盾，她是女同性恋、男同性恋、双性恋和变性者(LGBT)权利的公开支持者。由于直言不讳地表达了她对各种问题的观点，她被禁止出现在无数的场合和电视节目中，并受到无数媒体的指责。然而，她的观点是她性格的一部分。她以直言不讳而闻名(至少对她来说是这样)。

这种表明立场、将其公之于众并坚持到底的想法可能是给商业博客注入活力的一种强有力的方式。每个行业都有热门话题(例如，设计行业的众包和规范工作)，很可能，你对这些问题有自己的观点。

在有争议的问题上，不要骑墙观望，要表明立场，让大家知道。当然，不是每个人都会同意你的观点，但是你的商业博客的一个目标应该是鼓励讨论。表达你的观点是激发人们评论并让对话继续的好方法。

## 她和她的观众有联系

凯西分享了关于名人的内幕秘密和她在好莱坞的经历，幕后讲故事让她看起来脚踏实地，平易近人。如果你出现在观众席上或观看她的节目，你会情不自禁地觉得自己是圈内人。她把自己定位为群众的朋友，并渲染“我们反对他们”的感觉。

在你的博客上，分享你的智慧和经验是很好的，但要让它适用于你的读者。给他们他们想要的，并显示出你在做的时候和他们有共鸣。

无论如何，提供一般的提示，但有时给出可操作的例子来说明如何应用这些提示会更进一步。花点时间通过听他们的评论，跟踪你的网站统计数据，看看人们在读什么，或者简单地问一下，找出你的读者想读什么。然后将这些信息应用到您的博客中，以增强参与度和互动性。

## 她听她妈妈的话

凯西和她母亲的关系非常密切，经常在她的节目中提到她。她的妈妈几乎就像她的良心一样，质疑她说的一些最有争议的事情，提醒她成长过程中受到的教训，并让她承担责任。

虽然你可能不会每次在博客上发表文章时都打电话给父母寻求建议，但你可以通过记住童年时学到的一些最基本的经验来让你的博客变得更好。想到的几个:分享是好的；注意你的语言；如果你没有好话要说，那就什么也别说。

无论你是否是凯西·格里芬的粉丝，这里的要点是，如果你敞开心扉，你可以从生活中的每一次互动中学到一些东西。最近有哪些看似不相关的经历帮助你在生活中或业务上做得更好？
