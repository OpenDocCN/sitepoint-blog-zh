# 创建有效电子邮件签名的 20 个技巧

> 原文：<https://www.sitepoint.com/creating-an-effective-email-signature/>

这篇文章写于 2009 年，至今仍是我们最受欢迎的帖子之一。如果你热衷于学习更多关于这个主题的知识，你可能会对最近这篇关于创建电子邮件签名的文章很感兴趣。

昨天，我确定了糟糕的电子邮件签名的四种常见性格类型。虽然帖子的大部分内容是轻松的，但有效的电子邮件签名是必不可少的。对于签名很长或过于格式化的邮件的收件人来说，找到您的联系信息可能是一项挑战。更不用说糟糕的签名会损害你的职业声誉，增加不必要的累赘，分散你邮件内容的注意力。

有许多方法可以设置电子邮件签名的格式，虽然一种方法可能并不适合所有情况，但有一些通用的指导原则可以帮助您创建专业、易于理解和信息丰富的签名。

## 电子邮件签名提示

1.在提供所有你认为最重要的信息时，尽可能保持简短(四行是公认的标准)。

2.通过使用竖线(|)或冒号(::)分隔文本，将信息压缩到更少的行中。

3.记住简单的纯文本是最好的；跳过颜色、特殊字体和图形。

4.使用公认的签名分隔符(-)来帮助您的签名被电子邮件客户端识别。

5.小心 HTML 格式，因为它可能不会像你希望的那样出现在每个人面前。

6.用尽可能多的电子邮件客户端测试你的签名，尤其是当你使用 HTML 的时候。

7.优化您的徽标或其他图形，将文件上传到您的服务器，并使用绝对 URL。

8.如果您有特定的格式需求或想要包含图形和其他设计元素，请考虑使用电子邮件签名服务。这里有几个例子可供参考:

*   [杠杆作用](http://www.wisestamp.com/)
*   [文具中央](http://www.stationerycentral.com/)
*   [我的现场签名](http://www.mylivesignature.com/)

9.在你的电子邮件中提供书面的 URL，而不是使用超链接，以确保链接会在发送的邮件中通过。

10.避免包含多个电话号码和电子邮件地址。选择你的联系方式，去掉其他的。

11.如果您希望看到您消息的任何人都能以这种方式与您联系，请只提供即时消息的详细信息和 Skype 帐户信息。

12.跳过你的邮寄地址——不是每个收件人都希望或者应该能够获得这些信息。

13.包括你最重要的社交媒体资料的链接，只要它们是合适的。

14.包括你的电子邮件地址。你不能依赖各种电子邮件客户端在回复和转发中包含邮件头信息。

15.创建长度和内容不同的版本，并知道何时使用每个版本(即在回复中使用较短的版本)。

16.如果经常使用移动设备发送消息，请确保更新移动设备上的签名。以下是一些流行设备的快速操作方法:

*   [iPhone](http://www.tech-recipes.com/rx/2493/iphone_change_your_email_signature/)
*   黑莓
*   [手掌预](http://www.technipages.com/palm-pre-add-email-signature.html)

17.不要将电子名片附加到邮件中。不是每个人都使用它们，即使是使用它们的人也不需要重复接收它们。

18.重新考虑是否包含报价；你永远不知道什么时候你可能会冒犯某人或给人错误的印象。

19.除非需要，否则避免包含法律免责声明。

20.避免在每封邮件的底部都附上一封病毒检查过的邮件。
