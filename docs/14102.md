# 产品:你采取行动了吗？

> 原文:[https://www.sitepoint.com/products-are-you-taking-action/](https://www.sitepoint.com/products-are-you-taking-action/)

这篇博客引用了上一篇 Sitepoint 论坛报的文章，特别是 Brendan Sinclair 写的一篇关于采取行动的文章。在那篇文章中，他描述了他是如何通过在互联网上销售产品获得超过 100 万美元的收入的。钥匙？采取行动并进行测试。

虽然他的文章仍然留下了想要的例子和更多细节(例如，什么产品？什么营销战术)，关于采取行动的点是关键。

我的观点是:今天的技术在很大程度上是一种商品。你几乎可以在任何地方以极其低廉的价格获得编码服务。开源只是加速了这一趋势。

如果这一论断是正确的，那么在产品方面，有三条成功之路:

1.测试。知道如何测试产品和服务，以最小化你的负面风险(例如，测试成本低，并扩大什么工作，同时消除什么不工作)。

2.保持敏捷。动作要快，不断测试，直到有东西起飞。

3.营销。学会如何营销，让人们愿意从你这里购买。最成功的专业人士会通过营销使自己与众不同。这适用于服务和产品公司。

以上适用于产品。你怎么想呢?

第二部分:在服务方面，#3(营销)仍然适用。然而，在提供服务时，不成为商品的关键包括:

1.与客户发展长期关系。
2。用你的市场理解的术语传达你的价值和优势。
3。深入特定的利基市场。
4。采取行动在你的市场上引人注目。

那么，在一个一切都变成商品的世界里，服务和产品销售/营销有什么共同点呢？我的回答是:

通过扎实的营销采取行动。

对于那些认为你的技能很棒，人们会自动来找你的人来说，上述观点并不合适。你必须走出去，主动地、创造性地传播信息。抱歉。
