# 测量地图是废弃软件吗？

> 原文：<https://www.sitepoint.com/is-measure-map-abandonware/>

我在个人博客上使用[测量地图](http://measuremap.com/)已经有一段时间了。[杰弗瑞·维恩](http://www.veen.com/jeff/index.html)亲自回复了我发的一封电子邮件，让我知道他会给我一个阿尔法账户，只是因为我礼貌地提出了请求。Measure Map 是一个伟大的博客统计工具；非常简单，但能够传达大量信息。我相信它是如此之好，以至于让 Veen 先生在 Google 得到了一份工作(当他们作为交易的一部分购买 Measure Map 时)，在那里他将 Measure Map 方法应用于 Google Analytics。

显然，在那个时候，每个人都想知道现在谷歌有了一个漂亮的新分析产品，Measure Map 会变成什么样。嗯，我们还在等…

虽然 Measure Map 继续发挥作用，但它就像一些无脑的自动机一样，无休止地生产 Soylent Green 块，而所有人类都已经死亡(并被送入料斗)。堆积在装配线末端的 Soylent Green 正在慢慢变成底部的污泥，溃烂的蟑螂群在那里繁殖……好吧，你明白了。

测量图从未离开 alpha 状态。试着看看你的统计数据，你会得到“对不起，我有问题。请稍后再试或联系 measuremap@google.com。”大约 30%的情况下。如果你能够检查你的统计数据，你会在浏览器部分注意到，Measure Map 从未听说过 Firefox 2，我的统计数据显示的都是 1.0 和 1.5 版本；Safari 3 也是如此。我认为火狐 2 的统计数据将会进入 1.0 的篮子，因为很明显，61%的访问者会继续使用这个浏览器。人们可能由此得出结论，自 Firefox 1.5 以来，还没有对 Measure Map 做过任何工作。

访问[论坛](http://groups.google.com/group/measuremap-forums)，你会发现自 2007 年末以来，没有人说过一句话；上一个官方的测量地图主题来自于 2006 年 4 月，其中提到了一个‘无限’版本的测量地图的发布。它似乎已经关闭了几天[，而谷歌的任何人都没有注意到](http://groups.google.com/group/measuremap-forums-discussion/browse_thread/thread/06768ac590826c96#)。

我真的很喜欢 Measure Map，如果它被关闭，我会很失望，但我宁愿知道一些东西。我正准备从我的博客中删除测量地图脚本。我也认为这真的很奇怪(而且有点粗鲁),它就这么被放弃了——也许他们在等待所有的阿尔法账户持有者放弃并离开？

维恩先生，谷歌怎么样？Measure Map 是怎么回事？
