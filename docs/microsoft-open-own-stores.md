# 微软将开设自己的商店

> 原文：<https://www.sitepoint.com/microsoft-open-own-stores/>

对微软来说，这是重要的一周。Windows 7 将于周四发布，新操作系统被广泛认为会取得成功。微软会争辩说，Vista 很难说是一个失败，但是个人和组织一直在缓慢地从功能强大的 Windows XP 升级。我要做一个可能会让一些人震惊的全面声明: *Vista 没那么糟糕*。它有一些恼人的习惯，需要像样的硬件，但大多数问题已经解决。然而，Vista 给了 Windows 一个坏名声，许多媒体的诽谤也挥之不去。

微软从 Windows 7 中学到了宝贵的经验。它更精简、更快，而且操作系统已经公开测试了好几个月。挺好的——[连我都买了一本](https://www.sitepoint.com/windows-7-sold-out/)。(虽然我可能会重新考虑，如果我先看到微软令人尴尬的可怕的[“举办你的[Windows 7]派对”广告](http://www.youtube.com/watch?v=1cX4t5-YpHQ)！)

易激动的微软首席执行官史蒂夫·鲍尔默将在周四的新闻发布会上推出 Windows 7。行业造谣者也期待他提供更多关于微软独立商店开业的细节，第一家预计在美国亚利桑那州的斯科茨代尔。

这对微软来说是一次新的冒险。除了在云中应对谷歌，他们还希望在大街上击败苹果。但这行得通吗？

微软生产许多产品，所以我毫不怀疑他们可以用 XBox、键盘、鼠标、配件和软件来填满一个商店。然而，这些产品几乎可以从每一个在线和离线电子零售商那里买到。

苹果商店也是如此，但他们有一个更强大、更有触觉的品牌。苹果的客户群对产品有一种不健康的痴迷；粉丝们会坚持和公司在一起，他们对价格不太敏感，并且会很快原谅公司的任何违规行为。

也就是说，微软是世界上最成功的 IT 企业，他们显然可以看到一些潜力。或许他们自己的商店将提升公司的形象，超越无聊的商业软件？但是微软的产品能达到苹果的受欢迎程度吗？

你会去微软商店吗？你会觉得刺激吗？你承认吗？
