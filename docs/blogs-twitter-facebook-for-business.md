# 使用博客、推特和脸书寻找新客户的技巧

> 原文：<https://www.sitepoint.com/blogs-twitter-facebook-for-business/>

我们大多数人都加入了社交媒体的行列，并在日常生活中使用它的不同方面。社交媒体的一大好处是它对每个人都有不同的用途。你可以把它作为工作之余的休息时间，与老朋友联系，结识新朋友。如果你在为自己做生意，社交网络和寻找新业务总是社交媒体的外围应用，即使你并不完全专注于此。

博客、推特和脸书是社交媒体的三个要素，在这里你可以定制你的方法来寻找新客户和其他商业机会。

**商业博客**

博客可以成为你生意的一个有价值的营销工具。如果你有客户、同事和潜在客户感兴趣的好信息分享到你的博客上，那么这是显而易见的。

以下是一些让你的博客成为激发人们对你的生意产生新兴趣的成功方式的建议:

*   提供对他人有价值的高质量免费信息
*   坚持发帖，坚持既定的时间表
*   愿意分享一点你个人的一面
*   有自己的观点，不要害怕分享
*   耐心点，给它时间
*   记住关键词，但不要过分强调
*   在发布之前，花点时间检查和编辑你的文章

因为写博客是一项长期受益的持续活动，所以没有一条建议能单独为你赢得一些新客户。但是如果你对你的博客做出承诺，并坚持到底，你可能会对它的发展感到惊讶。
 **推特**

如果你专注于保持一个专业的个人资料并坚持使用你的账户，Twitter 会是一个寻找客户的有效途径。你甚至可以通过被动寻找新的商机。没错，完全有可能会有新客户来找你。

要在 Twitter 上寻找潜在的商业机会，请尝试:

*   使用 [Twellow](http://www.twellow.com/) 等服务寻找相关人员
*   通过[搜索功能](http://search.twitter.com/)搜索目标关键词
*   在适当的时候，乐于投入并帮助他人
*   参与商业和非商业对话

为了最大限度地将 Twitter 作为一种营销工具，你可以做的最重要的事情之一就是诚实、真诚和尊重每一条推文。

**脸书**

脸书可以是在你的行业建立信誉和专业知识的另一种方式，并逐渐给你带来新的业务。通过创建您的个人资料，并使您的业务成为焦点，您可以发展一个以下，提高您的前景。

要在业务中有效使用脸书，请考虑:

*   在您的个人资料中发布您的商业博客文章的链接
*   接受熟人的好友请求，主动联系那些你不太熟悉的人
*   创建一个商业粉丝页面来增加推广
*   探索脸书的广告
*   保持你的个人资料的专业性，但加入一些个性

脸书只是给你另一种方式来达到你的前景。将社交媒体的这些元素与你的网站和其他在线资源结合起来，你可以很容易地建立势头，最终带来新的业务。
