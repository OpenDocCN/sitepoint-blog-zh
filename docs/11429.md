# Windows 7:第一个月回顾，第 3 部分

> 原文：<https://www.sitepoint.com/windows-7-review-3/>

这是我一个月后对微软 Windows 7 体验的三部分回顾的最后一篇文章。在第一部分中，我们讨论了[安装和接口](https://www.sitepoint.com/windows-7-review-1)。第 2 部分涵盖了[生产力、软件和安全性](https://www.sitepoint.com/windows-7-review-2)。在今天的最后一部分，我们将探讨性能和 Windows 的未来。

## 表演

如果你期望 Windows 7 比 Vista 快很多，你会失望的。

事实上，这并不完全正确——这取决于你的系统。如果你有一台相当现代的电脑，确保 Vista 是最新的，并保持你的系统干净，Windows 7 的速度提升将是微不足道的。抱歉，微软——我可能没有基准测试数据来支持这一点，但 Windows 7 对我来说肯定不会感觉比 T1 快。

然而，我怀疑你会注意到速度的提高，如果你在旧的硬件上运行 Vista，或者有一个堵塞的系统，这将受益于清理。

根据我的经验，启动时间是相似的，尽管只是因为 Vista 在过去几个月里有所改进。我的笔记本电脑大约需要 40-50 秒启动，然后再花 20-30 秒登录。然而，Vista 几乎有 5 分钟无法使用，因为它预先缓存了它认为我可以使用的每个应用程序。Windows 7 更好，虽然冷启动后马上还是有点迟缓。

内存管理得到了改进，磁盘活动大大减少。此外，睡眠，休眠和恢复要快得多(菜单也更符合逻辑现在微软已经删除了 Vista 提供的 57 个愚蠢的选项！)

笔记本电脑用户应该体验更长的电池寿命，您的风扇噪音将明显比以前更安静！

## 未来

微软尚未透露他们对 Windows 8 及更高版本的计划。一些“泄漏”表明他们正在计划 128 位架构，但这肯定取决于处理器和硬件制造商。而且对 32/64 位的困惑也没什么帮助。

然而，谷歌的 Chrome OS 将于 2010 年发布。它有可能改变游戏规则，但我不相信许多企业已经准备好考虑 Windows 的替代品。我怀疑微软会继续这样做，Windows 8 将是渐进的，而不是革命性的。他们试图对 Longhorn(Vista 的最初设想)采取激进的态度，但却以令人沮丧的失败告终。

## 摘要

对我来说，Windows 7 最显著的改进就是你会忘记它的存在。它很稳定，运行良好，不会让您感到厌烦，并且提供了良好的性能。这就是操作系统应该有的样子:启动程序和管理文件的方式。这似乎是显而易见的，但我认为微软迷失了方向，因为他们开始认为操作系统比它运行的应用程序更重要。)

该操作系统可能不是比 Vista 更大的进步，但它解决了问题，解决了批评。如果你喜欢 Vista，而且它运行良好，新的操作系统不会提供太多新的东西。然而，一旦你尝试了 Windows 7，你就不会回头了。

在我看来，Windows 7 最大的优点是:

*   用户界面的改进
*   库和跳转列表
*   较少侵扰的 UAC
*   [XP 模式](https://www.sitepoint.com/article/ie6-ie7-ie8-win7-xp-mode)
*   没有安装不必要的软件
*   良好的性能和电池寿命

最糟糕的是:

*   冷启动仍然相当缓慢
*   缺乏原生 64 位软件及其带来的困惑
*   不一致的程序界面和风格
*   崩溃的小工具
*   很少有 Vista 用户会注意到显著的改进

Windows 7 是我购买的第一个版本(除了收到 PC 硬件的 OEM 版本)，我不后悔这个决定。Vista 的改进可能是微不足道的，但该操作系统受到了负面的影响，从此一蹶不振。微软希望通过 Windows 7 改变人们的看法，这似乎正在奏效。
