# 懒惰的开发人员从文本工具切换的方式

> 原文:[https://www . site point . com/the-lazy-developers-way-to-switch-from-the-text-tool/](https://www.sitepoint.com/the-lazy-developers-way-to-switch-from-the-text-tool/)

杰里米在对上一篇博客的评论中问道:“你如何只用键盘快捷键从文本工具转换到不同的工具？”为了解释这个问题，在许多图形程序中(我最近一直在关注 Illustrator ),你可以使用键盘来键入一个字母，并快速访问一个工具，而不必使用鼠标和点击工具调色板。例如，您可以轻松键入“P”来访问“钢笔”工具，键入“V”来访问 Photoshop 中的“移动”工具或 Illustrator 中的选择箭头，键入“T”来切换到文字工具。然而，一旦你开始打字，杰里米想知道，你如何快速切换到另一个工具？

我不知道你可以按下一组简单的键——我在谷歌上搜索这个问题时运气不太好(有人知道吗？)但这就是我所做的:

在 Illustrator 中:当您仍然在文本框或区域中时，按 Ctrl (PC)或 Command (Mac)键，这应该会将工具临时切换到上次使用的“箭头”工具，然后移动鼠标并在文本框外单击以取消选择它。您仍将处于文字工具中，但只是不在任何特定的文本区域中…这意味着您可以使用键盘访问键切换到钢笔工具、椭圆工具或任何其他您想要的工具…(这仍然需要移动鼠标，但对于像我这样的懒人来说，将鼠标随机移出文本框肯定比试图在调色板中找到工具花费更少的精力！)

在 Photoshop 中:我切换文字工具的捷径是保存(Ctrl-S 或 Command-S)文件。这似乎为我“关闭”了文本区域，然后我可以应用任何我想要的键盘快捷键。

有没有其他人有更“官方”的方法来达到同样的效果…或者任何其他自我发现的捷径？
