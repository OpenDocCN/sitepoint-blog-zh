# 事业营销，人人赢

> 原文:[https://www.sitepoint.com/cause-marketing/](https://www.sitepoint.com/cause-marketing/)

事业营销，或与事业相关的营销，是指涉及企业和非营利组织之间的合作关系的营销。通常情况下，为了双方的利益，企业会将其营销策略与慈善事业的筹款需求联系起来。公益营销不仅仅是一次性的慈善捐赠，还包括企业的积极参与。

## 事业营销背景

虽然在 1983 年之前就有企业和慈善组织合作的营销活动，但这一年美国运通创造了“公益营销”这个术语。该公司为自由女神像修复项目开展了一场营销活动。在这场非常成功的活动中，持卡人每消费一笔，美国运通就为修复自由女神像捐赠一便士。在四个月的时间里，这个项目筹集了 200 多万美元。

## 公益营销的好处

公益营销的目的是增加营利性企业和非营利性组织的曝光率和认知度。当公益营销有效时，你可以期待的一些好处包括:

*   品牌忠诚度的提高
*   员工士气的提升
*   销售额的增长
*   正面的新闻报道和公司评论
*   与竞争对手的区别

最大的潜在好处之一是更有利的公众声誉。事实上，研究表明，大多数消费者表示，如果另一个品牌与良好的事业相关，他们会从一个品牌转向另一个品牌。

当然，非营利组织也会从公益营销中受益。他们有机会扩大对该事业的认识，增加筹款和公众支持。

## 潜在的缺点

虽然公益营销围绕着慈善事业，但它也不是没有缺点。如果公众对这种关系的看法变坏，就会出现一个主要的不利因素。例如，如果你的目标受众认为你与非营利组织结盟是一种营销策略，你的声誉就会受到损害。如果你支持的非营利组织做了一些不道德或有争议的事情，你也会面临问题。那么你也冒着被负面看待的风险。

最后，事业营销不是没有投资的。当你开展一项事业营销活动时，你就在你的常规营销活动中增加了财务和/或时间投入。取决于你如何支持这项事业，这可能是一笔巨大的开支。

## 事业营销在你的企业

由于公益营销包括与慈善组织合作，做一些有营销价值的事情，你需要考虑你的企业做什么或卖什么，以及你如何将其与非营利组织的目标联系起来。

小企业在事业营销中能做的和大公司能做的有很大不同，但概念是一样的。无论你选择哪种与慈善事业相关的活动，关键是将它融入到你的营销活动中，以提高知名度。这里有一些想法可以让你思考如何为你的事业做好事。

*   加入像 [1%为地球](http://www.onepercentfortheplanet.org/)这样的结构化捐赠活动，帮助你将固定比例的销售额捐赠给慈善事业
*   针对即将到来的活动，与当地非营利组织开展联合品牌营销活动
*   选择一个慈善机构，在你的结账过程中添加一个“捐赠”选项
*   采用当地的原因，并提供无偿的设计工作，以巩固他们自己的营销努力
