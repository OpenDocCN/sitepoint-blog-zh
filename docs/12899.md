# 出去在顾客服务舞池里跳舞吧！

> 原文：<https://www.sitepoint.com/get-out-and-boogie-on-the-customer-service-dance-floor/>

我在 SitePoint 的职责之一是管理我们的客户支持团队。我经常把这个团队称为与我们的访客和客户一起“在舞池中”，而作为经理，我们只是“坐在阳台上”。

我用这个比喻是因为，任何去过夜总会的人都知道，在那个舞池里，你可以非常精确地看到一个小区域。在阳台上，你的视野更开阔，但细节少了很多。

在与你的团队合作制定客户服务策略时，欣赏这种不同的视角是很重要的。请记住，你的客户支持团队成员将带来一幅非常精确的画面，展示他们在舞池的*位置*发生了什么，所以**要听听他们怎么说**。如果你是一名经理，你的工作就是从更广阔的角度看待形势，做出有利于整个团队的决策和改变。

不要质疑你的支持团队告诉你的话——倾听、研究并采取行动！

我所见过的确保阳台上的人们不要忘记前线真实情况的最佳举措之一就是 SitePoint 正在做的事情。

现在，我不能完全归功于这个计划，因为它不是我自己的主意(向我们的总经理 [Luke](https://www.sitepoint.com/blog/) 致敬)，但它是如何运作的:我们有一个政策**SitePoint 的每个员工必须每月大约有一天**做一整天的客户支持。这包括我们的联合创始人，我们的总编辑，程序员，设计师，甚至我。

让整个团队定期参与客户支持，可以确保公司的每个人都了解错误或有缺陷的流程会对我们的客户产生什么样的影响，无论他们是否直接参与我们业务中面向客户的部分。

给员工或经理看一份列有 10 个没有按时收到书的客户的报告是一回事，要求那个人*亲自回复每一个客户*则完全是另一回事。我敢打赌，下一次我们的客户支持团队提到“我们可能有一个运输问题”，他们会有一个发声的盟友来帮助维护他们的事业。

实际上，在任何公司的任何职位上，你的客户都是你的生命线。即使你从未接近与他们交谈，重要的是永远不要忘记**你所做的*会对*产生影响。**虽然您可能不会直接从客户那里听到，但如果您的支持团队告诉您有问题，那么肯定会有问题！

所以，如果你发现自己忘记了你的顾客，不管你的公司是卖产品还是卖服务，考虑一下回到舞池，亲自检查一下。
