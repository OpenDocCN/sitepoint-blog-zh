# 谁会发 HTML 邮件呢？

> 原文:[https://www.sitepoint.com/who-sends-html-email-anyway/](https://www.sitepoint.com/who-sends-html-email-anyway/)

所以，让我们澄清一下:HTML 电子邮件既不酷也不时髦。

挂在 SXSW 或 Web Directions 的大厅里，热门话题将是 HTML5 或 iPhone 开发或 CSS3。提到把 HTML 放进电子邮件，你可能会遇到从不感兴趣到厌恶到公开敌意的任何事情。

设计师会告诉你它很挑剔，很原始。公平的决定。技术人员经常哀叹这是不自然和不必要的，尽管在我看来这有点像 90 年代后期的观点。

然而，我相信对 HTML 电子邮件最常见的误解是*人们**认为**是为了*。

向大多数网民提问，他们会告诉你 HTML 电子邮件的用途:

*   垃圾邮件
*   产品或促销技巧
*   时事通讯

不过，仔细看看你的收件箱，你可能会感到惊讶。你会发现大多数 HTML 实际上被用来支持基本的业务流程，而不是销售和市场营销。

让我们来看看 2010 年你会在一个典型的收件箱里发现什么。到底是谁在发送这些 HTML 格式的电子邮件？

#### 1.脸谱网

在脸书的世界里，“好友请求”是基本的构建模块。理论上，在你开始建立双方都同意的联系之前，没有什么真正有价值的事情发生。

大部分的好友请求是如何发出的？我猜想你已经知道这是通过 HTML 电子邮件的魔力，当然。

![Facebook](../Images/19114273ed863d0157d451b16957bd8e.png)

想想看:将世界上最无处不在的社交网络哥斯拉粘合在一起的粘合剂是 HTML 电子邮件。

不管你是不是脸书的粉丝，它肯定会告诉你一些脸书的想法。尽管 HTML 电子邮件存在潜在的交付和格式问题，但他们仍然认为纯文本不适合这项关键任务。

我觉得这很能说明问题。

#### 2.推特

每当有人开始“关注”你时，你的收件箱里就会收到一封格式整洁的蓝边电子邮件，里面有新关注者的头像和计数细节。人们很容易忘记，这只是过去 18 个月的情况。

![Twitter Follow ](../Images/9090cc8b231f858fff171b592dbe083f.png)

如果你记得最初的纯文本版本，这是一个巨大的进步；现在你不用离开你的电子邮件程序就可以评估一个新的追随者。

在 Twitter 方面，HTML 电子邮件是你了解新粉丝的唯一途径，所以我们只能猜测 Twitter 每天必须发送数百万封电子邮件。

#### 3.闪烁（光）

Flickr 是一种视觉体验，所以它的电子邮件也有视觉倾向也就不足为奇了。

![](../Images/2f9676bff2edbfa7ebbdbe93b8aa7598.png)

它目前在两个领域使用 HTML 电子邮件——通知你有新的联系人，以及当你在另一个用户的照片中被标记时。和这个网站一样，Flickr 的邮件保持简单、宽敞和优雅。

#### 4.油管（国外视频网站）

是的，网络视频巨头也在 HTML 邮件中做了一个美味的系列。目前，如果你订阅了任何 YouTube 用户订阅源或频道，你会收到一封总结你订阅源的 HTML 电子邮件。

![Youtube](../Images/4aface311c369e4d351db3dbbcd73b6c.png)

巧妙之处在于，在许多邮件应用程序(当然是 Gmail)中，你可以在邮件中实时播放视频。

#### 5.通过易趣网购买

现在我们真的到了老学校了。易贝是这一领域的先驱，自 21 世纪初以来，他们将很大一部分销售流程委托给了 HTML 邮件。

这在当时是一个大胆的决定，因为就其本质而言，HTML 格式的电子邮件总是比普通文本更容易被屏蔽或过滤。而当人们的*血汗钱*被牵扯进来的时候，事情很快就会变得很难看。

尽管如此，易贝肯定很久以前就得出结论，HTML 的品牌自信和表现优势仍然值得冒险。

所以你有它。网络上五个最大、最时髦的品牌依赖于 HTML 邮件。

在每种情况下，电子邮件的风格都很简单，但绝不是事后才想到的。相反，HTML 电子邮件让这些网站能够通过邮件客户端主动联系用户。

在推文、墙和状态更新的时代，人们很容易忽视这一点的强大。
