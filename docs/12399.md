# 火狐在俄罗斯放弃谷歌

> 原文：<https://www.sitepoint.com/firefox-drops-google-in-russia/>

据报道，在 3.1 版本的官方俄语版本中，火狐将放弃谷歌搜索(T2)作为默认搜索引擎，转而支持基于俄语的 Yandex(T4)。据 Mozilla 总法律顾问 Harvey Anderson 称，这一变化背后的原因是 Mozilla 对俄罗斯用户行为的研究得出结论，俄罗斯用户比谷歌更希望直接访问 Yandex。

这并不奇怪，因为在俄罗斯，Yandex 控制了搜索市场的 43-45%。正如我们[在 9 月](https://www.sitepoint.com/5-countries-stand-betweent-google-and-world-domination/)报道的那样，全球只有五个国家谷歌目前没有占据主导地位——俄国是其中之一。有趣的是，在这五个国家中，谷歌实际上是在俄罗斯竞争最激烈的，那里有大约 35%的搜索量，并且在过去的一年里实际上增加了大约 6 个百分点。

失去 Firefox 的默认状态对谷歌在俄罗斯的竞争来说是一个相当严重的打击。根据 LiveInternet.ru 的统计数据，超过 21%的俄罗斯互联网用户使用 Firefox 浏览网页，大约 17%使用 Firefox 3——这意味着 Firefox 3 自推出以来在该国的总体使用量增加了约 5-6%。去年 4 月，只有大约 15%的人使用 Firefox。

我们上个月写道，自从谷歌发布 Chrome 浏览器以来，Mozilla 与谷歌的关系变得越来越复杂。目前还不清楚通过将 Yandex 作为默认搜索引擎，现金是否已经易手，但因为谷歌占 Mozilla 收入的 88%(通过搜索广告交易)，Mozilla 开始寻求替代收入来源是有意义的。

我们已经预测过，一旦 Chrome 成熟，Firefox 可能会受到影响，所以寻找新的收入来源对 Mozilla 来说是个好主意。如果谷歌的支持不能被如此依赖，Mozilla 将需要开始在其他地方建立合作伙伴关系。

尽管安德森表示，改变的原因是一项用户研究，但我们怀疑，转向 Yandex 的至少部分动机是 Mozilla 对 Chrome 的针锋相对的反应。然而，没有真正的迹象表明这一点，所以这是我们的一种直觉。俄罗斯是 Mozilla 开始寻找替代收入合作伙伴的好地方，因为火狐很受欢迎*和*谷歌不是占主导地位的搜索引擎——这不是一个常见的双重事件。
