# 我的日常工作快把我累死了！自由职业的第一步

> 原文：<https://www.sitepoint.com/first-steps-freelancing/>

所以你在考虑成为一名自由职业者？你需要的第一件事是投资组合——你的投资组合越好，就越容易找到好工作。然后，一旦你准备好了，是时候登陆自由职业者的工作网站，销售你的服务了！

所以，让我们来看看让你的自由职业者出现在网上的步骤:

1.  建立你的投资组合

3.  创建您的个人资料

5.  在线访问自由职业者市场

##### 建立你的投资组合

***第一步。定义你想要的工作类型***

当我大学毕业时，我有一整个作品集，里面都是布莱斯的三维图像。信不信由你，我正试图用这些图片找一份平面设计的工作。问题是，这些不是“图形设计”图像，它们是“计算机图形”图像。差别很大。因此，在开发投资组合之前，明确定义你想申请什么类型的工作非常重要。

***第二步。决定你将包括的工作类型***

不管你申请的是什么工作——不管你是设计师、作家还是程序员——在你的文件夹中只放那些与你申请的工作相符的样本。例如，如果您决定申请一个平面设计职位，请在我们的对开本样本中包含您申请的工作中可能需要创建(或有创建技能)的小册子、时事通讯、邮件、目录和其他项目。

你不会想在你的文件夹里塞满插图、3D 效果图、草图和其他与工作无关的东西。举几个例子可能没问题——但是要集中展示与你想要的工作最相关的技能。

***第三步。作品来源示例***

如果你是一个有经验的专业人士，找到和展示你最合适的作品的高质量例子应该不会太难。但是那些刚开始工作的人可能没有任何过去的工作可以展示——更不用说适合这份工作的工作了。经验不足的人可以通过几种方式建立自己的投资组合:

*   创建虚拟样本 —创建你工作的虚拟样本:模拟小册子、时事通讯、标志和其他展示你技能的东西。尽管这看起来像是浪费时间，但是您可以将这些文档用作将来实际项目的模板。
*   **志愿服务** —接近当地非营利组织和社区组织，志愿提供服务。确保你告诉这些客户，作为“付款”，你坚持在成品上有你的企业名称和电话号码(如果你有网站，你的网址也应该印在成品上)。希望你的第一份出版物能接触到一系列潜在客户。不管怎样，你都会有一些好的、专业制作的作品放在你的对开本里。
*   **和朋友聊聊** —即使你还不知道，你可能也有经营小生意的亲戚或朋友。自愿提供服务，帮助他们获得免费的第一份工作(但要确保他们不期待持续的免费！).再次强调，一定要把你的公司名称、电话号码和网址印在成品的某个地方，并确保把你的作品包括在你的文件夹里！
*   如果你是一名网页设计师，在网上搜索那些你觉得可以使用你的服务的网站，并为他们提供免费的重新设计。或者，如果你是一名平面设计师，你可能会找到可下载的 pdf 格式的宣传册和其他宣传材料——查看这些，并免费提供重新设计公司的宣传册和传单。

***第四步。编译你的作品集***

一旦你完成了每一个步骤，你应该会发现自己在你想要的工作领域有一些很好的工作实例。将这些仔细地编入你的文件夹。如果是在网上，包括高质量的截屏，考虑包括每个工作的简短说明，如果工作出现在网上，链接到它。

如果你的是实物投资组合，确保你的样品是最高质量的。如果你制作了硬拷贝，确保你的样品是真正的成品，而不是模型或复制品。如果您想在脱机状态下显示在线内容，请确保您的打印输出是高质量、全彩和耐用的。

##### 创建您的个人资料

如果你真的想成为一名自由职业者，那么创建一份用于在线自由职业市场的个人资料是至关重要的。每个网站希望你创建个人资料的过程会略有不同，但他们需要的大部分信息都是相似的。

当你建立个人资料时，有很多事情你应该知道——记住这些要点。

***1。使用关键词***

要创建一个有效的档案，使用一点逆向思维。假装你是客户。你会如何在你的职业中寻找一个自由职业者？你会用什么关键词来搜索自己？

访问几个自由职业者网站，搜索“平面设计师”、“内容作家”、“漫画家”或你所在行业的其他关键词。现在，看看前五名的简介。这些人在个人资料中使用了哪些关键词？不要照搬他们的描述，但一定要尝试将这些专业人士在他们的个人资料中使用的技术应用到你的个人资料中。

接下来:头脑风暴，头脑风暴，然后再头脑风暴一些！想出所有与你的技能相匹配的关键词，并将它们包含在你的个人资料中。关键词越多越好，但是你的简介也需要有意义，所以不要把你所有的关键词乱七八糟地塞进去。确保你用尽可能多的关键词清楚地解释你在做什么，你能做什么。

***2。**注意细节*

自由职业者网站上列出了数百名你所在领域的专家——所以你需要详细解释你的才能。如果你是一个 Flash 设计师，你不想设计 HTML 页面，在你的描述和标题中清楚地陈述这一点。否则，你会收到与你的技能或兴趣领域不匹配的线索。但是要小心！如果你需要工作，或者在你特定领域的工作消失了，你可能不想太狭隘地定义你的技能。请务必根据需要更新您的个人资料。但是永远，永远，要具体。

试着把你的简介想象成一封标准的随简历一起寄出的求职信。你不会想都没想就发求职信——你应该用同样的眼光来看待你的个人资料。检查拼写、语法和所有输入的细节，以确保它们是正确的。不要在你填写的任何个人资料表格中留下任何空白——相信我，这会让你失去很多潜在的客户。当你留下空白时，你会让客户怀疑你在隐藏什么。同样，不要忘记在每一个机会使用你的关键字！

***3。知道自己能提供什么***

如果你是自由职业的新手，你可能不确定你能为潜在客户提供什么。仔细考虑你能为客户提供什么。你如何解决他们的问题？他们为什么要选择你而不是你所在领域的其他专家呢？写出来让人印象深刻！

再次，看看其他人的个人资料。您喜欢您看到的个人资料的哪些方面？其他专业人士是如何描述他们的才能的？将这些技巧融入你的写作风格。并且要非常具体——提供你完成的工作的例子，你为之工作的客户，以及解释你与这些组织合作的方式。你是如何解决客户的问题的？你最引以为豪的项目是什么？在您的个人资料中包含所有相关的详细信息。

如果你是这个领域的新手，并且你正在寻找你的第一份自由职业，你可能需要在你的简历中更多地关注你的教育和技能。但是不要忽视你可能从其他工作中获得的经验——你可能有很好的客户和客户服务技能，对质量的专注，或者对最佳实践的承诺。请考虑在您的个人资料中包含这些信息。

***4。不排除标准***

请记住，您选择在个人资料中包含的标准有助于在客户搜索时确定合适的匹配。这些标准包括你可以控制的变量，比如每小时的工资、时间、每周工作的小时数，以及你是在现场工作还是在异地工作，或者两者兼而有之。

一旦你设定了标准，不要排除为了得到一份工作而改变任何标准。例如，如果你喜欢在异地工作，但如果不得不在现场工作，考虑同时选择“现场”和“异地”来获得更多的搜索匹配。此外，做一些研究，看看在你的职业中，与你的经验水平相同的其他人每小时收费是多少。如果你的利率太高，你可能会失去客户。

***5。不要在你的表格中留下空白***

不要在简介表格上留下任何空白。将问题留空会引起潜在客户的关注。他们可能想知道你在试图隐藏什么，或者给人留下你不注意细节的印象，毕竟，他们可能真的想知道你遗漏的信息。

不管是哪种情况，他们可能会转向下一个潜在的候选人。空白会给人留下不好的印象。

***6。使用描述性标题***

当你可以成为鹤立鸡群的人时，为什么要融入其他人呢？“拥有 5 年以上经验的专业 Flash 设计师/开发人员”难道不是在“Flash 设计师”头衔的人群中脱颖而出吗？大多数自由职业者网站都允许你有很长的头衔:利用这一点吧！

客户在进行搜索后会筛选头衔，所以你的头衔可能意味着引起雇主注意和与数百名潜在自由职业者一起被抛弃的区别。如果你有一个吸引人的标题，他们会想知道更多——而且他们肯定会看你的简介！

***7。包含好推荐人***

花些时间收集好的、可靠的参考资料。这些代表了潜在雇主的“普遍观点”，所以如果没有推荐信，你的简历可能会被忽略。雇佣自由职业者是有风险的事情。如果你没有人可以证明你是负责任的、有能力的、专业的，那么潜在的客户怎么能确定你能胜任这份工作——并能与之共事呢？

尽量确保每个推荐人都能证明你能力的不同方面。你可以使用来自不同行业、领域内不同行业或公司不同职位的推荐人来全面展示你的才能。如果有必要，你甚至可以让老同事或大学时的熟人做推荐人。

***8。描述你的工作经历***

收集你过去所有工作的信息，确保把你工作中最令人印象深刻的例子放在最前面。当你描述每份工作时，一定要详细解释你努力的结果。你解决了哪些问题？客户对你的工作满意吗？不要把任何相关的经历从这份清单中去掉——但是把你印象最差的例子放在最后。

##### 保持最新

保持您的个人资料和在线投资组合最新是非常重要的。每当你完成另一份工作，确保你把它包括在你的工作经历中，并把所有的工作样本放在你的文件夹里。随着行业状况和你个人兴趣的变化，你应该调整你的投资组合和个人资料，这样你才能吸引最适合你的雇主和工作！
