# 网络邮件安全漏洞不断

> 原文:[https://www.sitepoint.com/webmail-security-breaches/](https://www.sitepoint.com/webmail-security-breaches/)

对于大型网络邮件提供商来说，这是艰难的一周。多达 30，000 个来自 GMail、Hotmail、Yahoo Mail、AOL、Comcast 和 Earthlink 账户的电子邮件地址和密码被发布在包括[Pastebin.com](http://www.pastebin.com/)在内的许多网站上。

尽管名单上的一些账户看起来很旧、没用过或不正确，但许多账户已被证实是真实的。谷歌、微软和雅虎继续建议用户更改密码，如果他们认为他们的帐户已经被泄露。

应该指出的是，没有一个网络邮件系统被黑客攻击或利用。大多数网络邮件系统是安全的；它们已经成功运行了多年，没有出现重大的安全漏洞。大多数安全专家怀疑账户信息是通过网络钓鱼攻击整理出来的。然而，这个问题的规模让一些人认为键盘记录恶意软件可能是罪魁祸首。

不幸的是，这个问题已经成为自我传播，帐户违规可能会成倍增加。在线安全公司 Websence 发现从 GMail、Hotmail 和 Yahoo 帐户发送的垃圾邮件数量急剧增加。黑客和网络罪犯正在使用网络邮件系统向真实的地址簿联系人发送更多的网络钓鱼电子邮件。

尽管受骗的用户往往是 IT 新手，但网络邮件系统的兴起不可避免地加剧了这一问题。很多人用几个账号；向 GMail 用户发送一封电子邮件，说明他们的 Hotmail 帐户中有一封重要邮件，并提供一个*有用的*链接到登录页面，这很容易。

有解决办法吗？用户教育显然会有所帮助，但是对于没有经验的客户，也许你可以考虑在他们的电脑上安装一个标准的 POP3/IMAP 电子邮件客户端？他们没有远程访问的好处，但他们永远不需要知道自己的电子邮件密码。
