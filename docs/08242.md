# 被录用

> 原文:[https://www.sitepoint.com/get-hired/](https://www.sitepoint.com/get-hired/)

在我关于[制止虐待客户](http://www.johntabita.com/abusive-clients/ "Putting a Stop to Abusive Clients")的系列文章中，我从卖家的角度谈了很多。但是在我给这个系列做最后的包装之前，我要稍微绕一下路，从买方的角度来讨论一些事情。下面说说怎么被录用吧。

在过去的几天里，我不得不发布两个不同的工作机会；一个是全职销售代表，另一个是一些合同网络工作。退一步说，我得到的回复很有趣。

我故意做的一件事是包括电子邮件和我的直接电话号码作为联系方式。你的选择很可能意味着被雇佣和被忽视的区别。

在发布招聘合同职位的广告几分钟后，我接到了一个电话。不久之后，我接到了另一家公司的第二个电话。在谈论他们的公司之前，他们都花时间了解我的目标和我想要达到的目标。

其余的人发了一封电子邮件，类似于“查看我的网站”和“给我回电话”

你认为我在考虑谁？

## 隐藏在 Web 2.0 背后

最近，我儿子的童子军领队给我留了一条语音信息，是关于即将为他队里的一个男孩举行的鹰仪式，他的出席是“强制性的”。唯一的问题是，他给我们的只是举行仪式的教堂的名字…没有地址，没有城市，什么都没有。

在谷歌搜索了几个城市之后，只找到了一个叫这个名字的教堂，我让我儿子确认一下这是不是那个教堂。所以在典礼那天，他打了电话，留了语音信息，试图找到答案。没有人回应，包括童子军领队、他的巡逻队长和他队伍中的其他男孩。最后，出于无奈，他在脸书上发布了这个问题。

15 分钟内，三个男生回复。我担心这对我们年轻人的未来不是一个好兆头。

太多时候，我们使用 Web 2.0 工具来避免直接交互。例如，很容易隐藏在营销背后来避免销售。不要落入那个陷阱。当我给你选择的时候，拿起电话打给我。

我为什么要这么说？两个原因。一:我真的真的很忙。当你打电话给我时，你就获得了先发优势，因为你已经从我太忙的盘子里拿走了一项任务——跟进我收到的所有电子邮件回复。我真的没有时间“查看你的网站”，然后“给你打电话。”此外，你的网站可能和其他人的网站一样，包含一些普通的琐碎内容。不如你来证明为什么你与众不同，为什么我应该雇用你？你可以拿起电话和我进行一次真正的对话。

第二个原因是:它向我展示了你的主动性和人际交往能力。你看，如果我真的有需要，我会想和你谈谈，以确定你是不是能满足我需要的人。如果我是那个躲在 web 2.0 工具后面，让你通过电子邮件或脸书与我交流的人，我可能根本不是一个真正的潜在客户。我可能只是“感兴趣”，或者在购买周期中还没有走到足够远，不能浪费宝贵的时间。但是如果我提供了我的电话号码，那就意味着我想和一个活生生的人说话。那个人可能就是你。

这里有另一个提示:当我要求你跟进，如果你在某个日期还没有收到我的回复，这可能是一个测试。给我回电话表明你想要这份工作。

我没有提到的是*还有*第三家公司打来电话；但是他得到了最后一根稻草——我的语音信箱——再也没有回电话。如果你有真正的线索，继续追查，即使他们不回你电话。如果你看到我的桌子，你就知道我为什么不回你电话了。
