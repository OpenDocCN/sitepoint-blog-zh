好吧，我想发布一些关于公司、组织和人们使用 ColdFusion 做很酷的事情的信息。以下是几个例子:

cfdev.com
我已经下载了一些 CFDev 的人创建的免费标签和应用程序，并打算彻底检查一下。但是，即使他们出售的软件也是值得注意的。想想你能用它做些什么:

–email verifier:使用语法、DNS MX 查找、顶级域验证和邮件服务器验证来检查电子邮件地址。我希望看到有人输入一个带有这种验证的假地址…
–翻译:使用 babelfish 翻译引擎将一种语言翻译成另一种语言。
–CFX _ Soap:一个用 Java 为 ColdFusion 构建的 Soap 客户端。

cfdev.com 上有更多很酷的东西。

另一个非常酷的项目是 FusionScript。这个网站曾经位于 www.fusionscript.com，尽管看起来最近被一个域名抢注者抢走了。这里有一个 SourceForge 项目。FusionScript 在 2001 年或 2002 年初开始开发。我实际上是在[数码脚本](http://www.digiscript.com)和 Nate Nielsen 一起工作，他是 FusionScript 的开发者。(我甚至开始帮助将 ColdFusion 加密算法转换成 Java，但我不认为我曾经完成过它，并且怀疑它是否能成为最终版本……)

FusionScript 允许 ColdFusion 开发人员利用 JavaScript 和 DHTML 功能，而无需了解 JavaScript 或 DHTML。这是实现复杂功能和 UI 元素的一种简化方式，而不需要自己学习开发它们。您可以使用这组自定义标记以您习惯的语法做任何事情。它还拥有一个 ActiveServer 引擎，允许您通过 Java applet 与服务器通信，而无需刷新页面。随着 Flash remoting 的出现，这一点可能并不需要，但在当时这是很酷的东西。

无论如何，我绝对推荐查看这些内容，并分享您感兴趣的项目或您觉得在某种程度上对 ColdFusion 社区有真正贡献的项目。
