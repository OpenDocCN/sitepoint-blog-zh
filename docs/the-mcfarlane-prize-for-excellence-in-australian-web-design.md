# 澳大利亚网页设计麦克法拉尼优秀奖

> 原文：<https://www.sitepoint.com/the-mcfarlane-prize-for-excellence-in-australian-web-design/>

一个新的年度网页设计奖[麦克法拉尼奖](http://www.mcfarlaneprize.com/)已经设立，以表彰澳大利亚网页设计的杰出表现。

> 首届麦克法拉尼奖以著名的澳大利亚网络先驱奈杰尔·麦克法拉尼的名字命名，旨在认可和鼓励澳大利亚开发者在网页设计方面的卓越表现。
> 
> 该奖项还旨在展示澳大利亚最好的网页设计，并激励其他澳大利亚开发者的努力。

一个由设计、可用性、可访问性和编码方面的专家组成的评审团将负责选出获胜者，并于今年九月在悉尼举行的[网络方向大会](http://sitepoint.wd06.com/)上宣布。

奈杰尔·麦克法拉尼是 SitePoint 的 Firefox book 的技术编辑，也是 SitePoint 工作人员所熟知和喜欢的[。就我个人而言，我认为奈杰尔是](https://www.sitepoint.com/tribute-to-a-friend/)[让我对 web 技术](http://www.opinios.com/archives/000076.php)感到兴奋的原因，几年前我在 Web 标准组织的一次会议上见过他，我想不出还有什么更合适的方式来纪念他。

因此，如果你是一个澳大利亚人，并且你为自己在 2005 年 8 月 1 日至今年 7 月期间参与的一个最新(重新)设计感到自豪，[提交你的网站](http://mcfarlaneprize.com/nominationform.html)，你可能会为你的工作赢得一些很酷的奖项和认可！
