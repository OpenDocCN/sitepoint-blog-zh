# 对于开发者来说，WordPress 3.0 有什么新功能？

> 原文:[https://www . site point . com/whats-new-in-WordPress-3-0-for-developers/](https://www.sitepoint.com/whats-new-in-wordpress-3-0-for-developers/)

*这篇文章已经过时了，[点击](https://www.sitepoint.com/whats-new-in-wordpress-4-6/)查看 WordPress 4.6 的新功能。*

WordPress 版于上周末发布。这比预定的 5 月 1 日发布日期晚了一点，但 WordPress 是这个星球上最受欢迎的博客和内容管理系统之一，所以迟到总比给成千上万的网站带来问题要好。

此次更新是 218 名专业贡献者六个月工作的结果，最终修复了 1，217 个错误并增强了功能。那么你对 WordPress 3.0 有什么期待呢？

## 新安装程序

WordPress 一直以来都很容易安装，但现在它变得更加简单了。很少有管理员需要摆弄`wp-config.php`配置文件:现在所有的 MySQL 设置都可以在安装面板中指定。

新的安装程序还允许您指定管理员 ID 和密码。我怀疑在以前的版本中很少有人会费心去改变默认的“admin ”,所以创建你自己的 ID 的功能将有助于安全。

## 新界面

WordPress 3.0 管理面板已经得到了润色。这和版本 2 相比几乎没有什么大的变化，但是它更轻，感觉更光滑。

![WordPress administration screen](../Images/38d33114685e585174fb25e78d2f6dbd.png)

在到达外观部分之前，界面没有什么明显的变化。

## 新默认主题

瑞普·库布里克:你为我们提供了很好的服务，许多网站至今还在使用你。Kubrick 已经被“Twenty Ten”取代，这是一个新的主题，内置了对子主题、背景变更、标题定制和下拉菜单的支持。

![WordPress Twenty Ten theme](../Images/09b067ea125421973126475438e39a6f.png)

该主题的外观和部件可以在管理面板中定制，所以我希望许多人永远不会冒险超过 2010 主题。对于那些这样做的人，有一个新的“安装主题”标签，允许你按颜色、类型和功能搜索模板。

## 多用户

WordPress MU 是一个分支，它允许数百个博客从一个安装上运行；它现在已经与主版本 3.0 应用程序合并。默认情况下它是禁用的，但是可以通过在您的`wp-config.php`文件中添加下面一行来打开它:

```
 define('WP_ALLOW_MULTISITE', true); 
```

这对网络开发者来说可能是最重要的功能:你可以用一个 WordPress 的安装创建多个网站。更新更容易，托管空间大大减少。

## 自定义帖子类型

页面和帖子在 WordPress 的以前版本中是可用的:

*   页面通常用于静态内容，如“关于我们”或“联系我们”页面。
*   帖子通常用于带有日期戳的新闻、文章或博客帖子。

WordPress 3.0 支持自定义文章类型。例如，您可以有一个产品类型，专门用于网站上销售的商品。产品页面可以单独处理；例如，拥有自己的菜单或搜索框。

自定义文章类型是使用 PHP 配置的，而不是在管理面板中。请关注 SitePoint 上的完整教程。

## 你应该现在升级吗？

我毫不犹豫地推荐 WordPress 3.0 进行新的安装。但是如果你从 WordPress 2.x 升级呢？

我很少遇到 WordPress 自动升级的问题；这是一个快速的一键式流程，非常有效。然而，我的第一次尝试彻底失败了，留给我的是一个破碎的网站。你应该注意到这是一个 2.8.6 的测试版本，有很多危险的代码和插件，但是你需要小心:一些主题和插件肯定会被破坏。

我建议在升级服务器之前，先升级你网站的本地测试版本。记得备份你的文件和 MySQL 数据库，不会错的。

如果根据过去的经验，WordPress 团队几乎肯定会在几周内发布 3.0.1。如果你特别紧张，建议你再等一会儿…
