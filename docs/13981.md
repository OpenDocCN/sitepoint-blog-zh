# 谷歌删除片段？

> 原文:[https://www.sitepoint.com/google-dropping-snippets/](https://www.sitepoint.com/google-dropping-snippets/)

我开始看到越来越多的元描述和开放目录(www.dmoz.org)描述出现在谷歌搜索结果中，而在过去，你可以从列出的页面中看到文本内容的片段。

在某些情况下，我肯定这是一件好事，但它也可能导致一些真正可怕的结果。查看谷歌搜索结果中的“ [SEO 咨询](http://www.google.com/search?sourceid=navclient-ff&ie=UTF-8&q=seo%20consulting)”

如果你看到的是和我一样的 SERP(有可能…)，你会在#7 找到我的网站(www.seoresearchlabs.com)，ODP 描述显示在我们的链接下面。不幸的是，由于 ODP 编辑不能拼写得很好，谷歌没有通过展示那个描述给我任何帮助…事实上，描述是误导的，因为我们不提供网站设计。(作为参考，ODP 列出“提供搜索引擎营销、关键词研究报告、网站设计、营销培训和咨询。”)

就在上面的第六页，我看到了另一个网站(www.increaseranking.com ),它的元描述显示在一个片段的位置。这对他们来说是个好消息，因为这基本上是广告文案。

增加排名不在公开目录中列出。我的网站是最近才添加的，直到几分钟前才有一个元描述。

过几天我会跟进，让大家知道元描述是否优先于 ODP 描述。如果 ODP 描述优先于站点内容，我将不再能够推荐提交站点到开放目录。

正如 SEO Fast Start 和 SitePoint SEM Kit 的读者可能知道的那样，出于同样的原因，我也质疑过雅虎目录列表的价值。至少就雅虎而言，你可以找人来纠正拼写错误。
