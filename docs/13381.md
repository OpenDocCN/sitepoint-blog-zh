# Feedburner 被谷歌收购

> 原文:[https://www.sitepoint.com/feedburner-acquired-by-google/](https://www.sitepoint.com/feedburner-acquired-by-google/)

据 Techcrunch 报道，谷歌正在收购 Feedburner 的传言现在已经得到官方证实。直到 Google 或者 Feedburner 贴出来了，对我来说还是谣言！但考虑到消息来源，这笔交易似乎将在未来 2-3 周内完成。我已经看到有人说，有人想离开，只是因为谷歌正在收购他们，尽管我不知道为什么。谷歌收购一家公司什么时候变成这么糟糕的事情了？

我认为 Feedburner 服务不会有太大的改变。谷歌试图启动自己的 feed 广告计划，但很快就失败了。这很可能是谷歌试图重返该市场的一次尝试。Feedburner 还提供基于 CPM 的广告，这是谷歌的另一个很好的选择，谷歌最近一直在这一领域拓展广告。希望这并不意味着他们会用 Adsense 上下文广告取代 CPM 广告，这似乎对许多出版商来说越来越糟糕。

Feedburner 提供的其他功能，如发布你的 RSS 源和提供使用统计数据，也非常适合谷歌。他们可以访问所有这些 feedss，我相信这些 feed 将会被用于他们的博客搜索以及其他事情。谷歌甚至可以通过他们的分析程序提供更好的报告。
