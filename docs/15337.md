挑战在于成为成功抓住这两个关键点的传播者。但是一旦你的信息被了解了会发生什么呢？

##### 遗忘

一条不被记住的信息就是花了一美元却没有回报。听众的记忆率会随着重复而增加，特别是当重复带来了令人满意的经历的相关记忆时。因此，如果你能在你的听众的头脑中挖掘出一种愉快的感觉，你将有更大的机会让你的信息被记住。

但是在学习发生后，遗忘的速度是最快的。幸运的是，我们可以利用学习理论来帮助我们的促销活动更令人难忘。

##### 学习理论的基础

学习理论，以及广告商用来激励他们的观众的技术，基于四个假设:

*   **Drive** – an internal state of tension that requires action
    e.g. The user thinks ‘this computer’s too slow!’
*   **Cue** – an environmental stimulus
    e.g. The user sees your ad for new computers.
*   **Response** – the reaction
    e.g. The user thinks ‘hey, that looks like good value!
*   **强化**–对回答者的回应给予的奖励
    例如，你提供了优质的服务，让用户认为“这是一个多么好的选择”，并鼓励他们再次光顾。

##### 学习理论的变化

学习的主要理论分为两组:

*   不同形式的行为主义和
*   最新的认知理论。

行为主义假设我们通过外部影响学习，并受其制约。认知理论更强调个人做出的内部选择。

***行为主义***

 *行为主义的底线是，我们只会在我们被限制做出反应的时候做出反应，并且更多的限制会产生更大的反应。

这些假设意味着哪个广告商最有钱——并能为他们的传播购买最多数量的媒体广告——将会胜出。对于预算有限的广告商来说，这确实是非常有限和令人沮丧的。

***认知理论***

认知学习的最新理论认为个体具有更大的复杂性。这些理论认为，学习过程更加复杂:人们不仅受到简单的条件限制，还在学习时运用解决问题和富有洞察力的思维。考虑到媒体发展的速度，这两个过程对于广告商信息的有效性至关重要。

最近关于记忆的认知理论表明，学习涉及两个因素:有意识和无意识。虽然我们已经习惯了有意识记忆的想法，但是在大脑中也有自动的、无意识的过程来评估一件物品有多熟悉。这一理论表明，那些利用个人现有熟悉度的广告将比那些仅仅试图让观众记住全新事物的广告更有效。

在你的信息中包含不一致的元素也可以减少衰退。如果一个名字对观众来说有意义，那么它就更有可能被记住。 **(3)**
