# 英国国防部“如何阻止泄密”文件被泄露！

> 原文:[https://www.sitepoint.com/uk-mod-leaks-document-leaked/](https://www.sitepoint.com/uk-mod-leaks-document-leaked/)

作为一名英国居民，我不知道该笑还是该哭。由于持续的安全问题，英国国防部决定撰写一份报告，为国防部、武装部队和情报人员提供有关信息安全和数据窃取的建议。不幸的是，这份长达 2400 页的机密文件已经泄露给了维基解密，一个专门发布政府和其他组织秘密信息的网站。

这份名为联合服务协议 440 (JSP 440)的文件于 2001 年首次发布。虽然相对较旧，但当国防部人员需要证明对网站或其他在线系统的监控是正当的时，它仍然被引用。该报告指出，调查记者与更典型的消息来源一样，对国家安全构成威胁:

> 这种类型的主要威胁来自调查记者、压力集团、调查机构、犯罪分子、心怀不满的员工、不诚实的员工和电脑黑客。
> 
> 调查记者利用了个人税务信息；他们还将商业和金融信息作为目标，寻求经济利益的犯罪分子也是如此。

该报告继续指出:

> 泄露通常以公共媒体报道的形式出现，似乎涉及未经授权披露官方信息(无论是否有受保护标记),从而对英国政府或相关部门造成政治伤害或尴尬。
> 
> 当官方信息泄露破坏政府政策或给政府造成尴尬时，其后果被认为是严重的。

很明显，这个故事会让英国政府，尤其是国防部更加尴尬。这也引发了一些尴尬的问题:

1.  国防部真的需要告诉它的员工信息泄露是对英国安全的威胁吗？
2.  解释什么是泄露真的需要 2400 页吗？这份报告一定花了一大笔钱。
3.  实际上，能指望工作人员消化并保留所有这些信息吗？

我相信这不会是最后一次安全漏洞，但肯定是最荒谬的一次。
