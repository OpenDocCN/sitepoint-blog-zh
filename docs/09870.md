# 红色、黄色和蓝色，或 CMYK

> 原文:[https://www.sitepoint.com/rgb-or-cmyk/](https://www.sitepoint.com/rgb-or-cmyk/)

在本系列关于色彩的上一篇文章 *[色彩理论 101](https://www.sitepoint.com/color-theory-101/ "DesignFestival.com: Color Theory 101")* 之后，我们现在将更好地了解 RGB 和 CMYK 色轮。

我一直惊讶于对红、黄、蓝三原色的不尊重。我听人说这是无效的，过时的，是幼儿园的工具。的确，红色、黄色和蓝色色轮不是光感知的科学精确模型。许多人希望从艺术课程中消除红色、黄色和蓝色色轮，并建立 <acronym title="Cyan, Magenta, Yellow, and Black">CMYK</acronym> (青色、品红色、黄色和黑色)色轮，如图 1 所示，“CMYK 色轮”，作为通用颜色模型。请注意，CMYK 色轮中的第二种颜色是红色、绿色和蓝色，这意味着我们可以使用 CMYK 来说明加色(使用光)和减色(在纸上)。

![Fig. 2, The CMYK color wheel](../Images/9eca8328154fc74760c7add0ba1e4846.png)
*图 1，《CMYK 色轮》*

为了说明转向 CMYK 背后的原因，我使用了水粉颜料，这基本上是装在管子里的水彩。当与水混合时，它们相当半透明，并产生你期望在现代 CMYK 色轮上看到的颜色，如图 2，“玩 CMY 水粉颜料”所示。洋红色和黄色混合产生漂亮的橘红色，而青色和黄色混合产生绿色和薄荷色调。这就是 CMYK 打印的工作原理。墨水是半透明的，它们之间的重叠(以及黑色的使用——不要忘记 good ol’K)给了我们在加色发光显示器或电视上可以看到的大多数颜色。正如著名的电视绘画指导鲍勃·罗斯可能会说，“这是一个快乐的小彩色模型。”

[![Fig. 1, Playing with CMY gouache paints](../Images/a8ef1d07a788b6794d1e10bcfbbf6f80.png)](https://www.sitepoint.com/wp-content/uploads/2011/04/color-paint-cmy.jpg) 
*图 2，“玩 CMY 水粉颜料”* —点击查看高清扫描。

等等！那个紫色斑点是什么？是的，等量的青色和洋红色形成紫色，而不是 CMYK 色轮建议的纯蓝色。当我们混合不透明的颜料时，会出现许多这样的异常现象。基本上，如果你的颜料太厚，以至于你看不到你所画的白纸或画布，CMYK 色轮的概念就开始失效了。在这方面，歌德、Itten 和其他人在过去四个世纪左右开发的传统红、黄、蓝色轮是一个更好的模型。

但是我们用的是像素，不是颜料！许多数字艺术家仍然将红、黄、蓝色轮放在手边的原因是，传统色彩理论的配色方案和概念都是基于这种模式。我们很快就会看到，颜色之间的关系很大程度上取决于它们在色轮上的相对位置。但是这些位置根据车轮的不同而不同；例如，在传统的色轮上，红色和绿色是相反的，而在 CMYK 色轮上，青色是相反的红色。我们不能简单地把色轮周围的红蓝调一下就收工了。因此，我将在接下来的几篇文章中讨论的配色方案是基于红、黄、蓝颜色模型的。

两种色轮模型都有缺陷，互补色就是一个最好的例子。但是真正让你兴奋的是，当我告诉你没有色轮可以完全描述我们从光感知颜色的方式的复杂性。尽管我主要是为网页设计——网页是一种以 RGB 显示的媒介——我仍然使用红色、黄色和蓝色作为我选择颜色的基础。我相信使用红色、黄色和蓝色色轮创造的色彩组合更具美感，好的设计是关于美感的。因此，我将展示我在大学二年级设计基础课上学到的色彩理论:从传统的红色、黄色和蓝色色轮。
