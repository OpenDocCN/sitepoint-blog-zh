# 关注思维导图

> 原文：<https://www.sitepoint.com/a-focus-on-mind-mapping/>

有没有觉得自己有一大堆想法，但是却很难把它们写下来？你并不孤单！许多人试图回忆他们不久前的一个想法，却发现很难把它以一种有用的形式记下来，把它从鞋盒里的一张破纸条转化过来。如果你还没有尝试，我鼓励你创建一个思维导图。如果你以前看过思维导图，但对它周围的热门词汇感到不知所措——语义网络、图形建模等等——我同情你；然而，思维导图的创建和执行远没有听起来那么复杂。这个维基百科页面会让你深入了解什么是思维导图。如果你是一个有创造力的人，并且喜欢记笔记，你可能已经创建了一个简单的思维导图，甚至没有意识到它是什么！我发现思维导图是头脑风暴和想法整理过程中非常宝贵的一部分。没有古旧的结构可以遵循——你只要开始写就行了。这也是一个多才多艺的过程。当然，有软件和工具，但是你可以用一张纸和一支笔来达到同样的效果。没有具体的规则，所以如果你回到你以前创建的规则，你很有可能马上就明白了。我是记事本和笔的粉丝，所以当我在飞机上或在客户等候室时，我有很多潦草的思维导图。然而，软件的种类确实让思维导图更容易编辑，也更容易与他人分享。这里有一些我用过的思维导图工具，我想推荐一下:

*   [FreeMind](http://freemind.sourceforge.net/wiki/index.php/Main_Page) —这是我的最爱，而且是免费的！
*   Mindmeister —一个基于网络的工具(带有 iPhone 应用程序)，我以前在与客户分享思维导图时用过。
*   XMind —一个开源的、基于 Java 的应用程序，它是跨平台的，相当复杂。

那么为什么不试试思维导图呢？这可能只是帮助你将杂乱的想法整理成一个更有意义、更可行的想法。
