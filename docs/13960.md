这么说，你开始创建自己的电子商务网站了？对你有好处！但是你有很多决定要做。你是自己写购物车还是用现有的？谁应该是你的商家账户提供商？谁会给你送货？面对如此多的决策，你几乎没有时间整理所有你需要知道的信息，以便为你的企业做出最佳选择。

##### 什么是网关？

简而言之，一个网关连接你的电子商务网站和你的商家账户。

该网关通过将您的安全订单与您在处理银行的特定商户账户连接起来，方便了在线支付。网关接收提交的表单数据，并将其提交给处理银行。当它收到银行的响应时，它会将返回的数据提交给原始站点进行适当的处理。

网关本身不提供电子商务功能，如购物车、虚拟主机或商家账户，尽管，正如您将看到的，许多大型网关提供商确实提供类似的附加服务。

##### 比较标准

在我们比较网关之前，我们必须决定我们将使用的标准。为了保持这一评估的相关性和简洁，我将把我的比较限制在三个因素上:

1.  可用功能

3.  费用

5.  可靠性

电子商务交易各方最关心的问题之一是安全性。不仅您的敏感信息被存档，而且您的客户的个人信息每天都通过您的网关。这些信息对黑客来说具有相当大的价值，因此必须加以保护。因此，当您选择网关提供商时，您必须确信安全性是提供商的第一要务。

也就是说，这篇评论不会评估我们所考虑的各种网关提供商所提供的安全性。我们将考虑的所有网关提供商都将其运营放在最先进的数据中心，并使用最新的安全方法来保护数据安全。

我们将考察的网关完全符合主要信用卡提供商提出的安全计划，包括 Visa 持卡人信息安全计划(CISP)、MasterCard 网站数据保护(SDP)和 Discover 信息安全与合规(DISC)。如果您决定使用一个不太知名的网关提供商，您应该验证该服务是否符合所有这些计划。如果不是，你可能最终支付更高的费用，你的帐户被关闭，并可能有你的组织添加到匹配文件，信用卡处理黑名单。

##### 特征

大多数在线商家没有意识到他们的网关可能提供比通过在线表格接受信用卡更多的功能。事实是，大多数网关提供商提供增值功能，使商家的生活更轻松，或有助于增加您的收入流。以下是一些比较常见的功能。

***虚拟终端***

假设您接到一个客户的电话，他对您网站上的产品有疑问。你充分回答了她的问题，她决定购买该产品。你会告诉她回到你的网站下订单吗？你可以的。但是，当你可以通过电话直接接受顾客的订单时，为什么要冒着顾客永远无法完成购买的风险呢？您可以使用虚拟终端当场处理客户的信用卡付款！

虚拟终端是商家可以访问的网络表单，允许你手动输入信用卡支付。它还可以用于发放退货和作废以前的交易。

***防诈骗***

欺诈防范工具可能是主要网关提供的最重要但未被充分利用的功能。商店上线超过六个月的商家将被介绍给[退款](https://www.sitepoint.com/articles/chargeback-challenge)，并且可能会对其负面影响有第一手的经验。在线交易退款的主要原因之一是欺诈。利用网关提供商提供的防欺诈工具，您可以减少处理退款的时间，将更多时间用于销售和推广业务。

这里讨论的网关支持基本的欺诈防范工具。其中包括地址验证(AVS)和 CVV2，前者将客户的送货地址与发卡银行记录的地址进行比较，后者是出现在 VISA、MasterCard 和 Discover Card 信用卡背面的三位数安全号码(美国运通卡正面的四位数)。

 **定期向客户收费的商家会发现许多网关提供的重复计费功能可以节省大量时间。使用重复计费，您可以简单地向网关提供计费信息，并告诉它多久向客户收费一次——网关会处理其余的事情。再也不用每个月把信息输入软件或信用卡终端了！这个功能对于订阅和会员制企业来说非常有用。

***eCheck 处理***

虽然信用卡是目前最流行的网上支付方式，但其他支付方式也不容忽视。毕竟，你的目标是让顾客尽可能容易地掏钱。电子支票(eChecks)是一种越来越受欢迎的支付方式，它允许客户只需输入他们的银行信息，而不是通过邮件发送支票。

***整合***

尽管集成可能不是真正的“特性”，但在选择网关时，它仍然是值得考虑的。你应该让你的结账过程对顾客完全透明吗？你是一个编程“新手”想要一个简单的设置，你知道会工作吗？或者，当你有时间和经验时，你是否在等待切换到一个更先进的设置？在不产生任何额外成本的情况下选择集成方法可以帮助您在设置过程中避免麻烦，并在以后的几个月中管理您的业务增长。

##### 费用

与任何商户帐户服务一样，费用与使用网关相关联，这些费用与您的商户帐户提供商收取的费用不同。幸运的是，网关提供商的费用是相当标准的，这使得比较容易。这些是最重要的:

*   **安装费:**这些费用是在您第一次建立您的网关时收取的(在它建立之前)，并且通常是不可退还的。这些费用可能从数百美元到零。
*   **月费:**与大多数商家账户服务一样，网关提供商对他们的服务收取月费。这项费用通常与每月的处理量无关。
*   **交易费用:**大多数提供商对他们处理的每笔交易都收取费用，不管交易是被批准还是被拒绝。
*   **附加功能:**通常，网关会对您可能决定使用的任何附加功能收费。费用可能包括安装费、月租费和上述标准费用之外的交易费。

##### 可靠性

在网络上，似乎一秒钟就等于一个永恒。每个网上商家都知道，如果他们不能在客户准备购买时处理交易，这笔交易就会失败。当客户可以立即去竞争对手的网站购物时，他们为什么要等待问题得到解决呢？你最不希望的就是在你最需要的时候第三方服务失败。您的网关必须全天候运行。

***Authorize.net***

 *成立于 1996 年的 Authorize.net 已经成为“门户”的代名词。

*特性*

**欺诈防范**

Authorize.net 欺诈检测套件(FDS)旨在在您的产品发货之前识别欺诈性交易(您会遇到退款)。该套件的功能包括:

*   为“可接受”销售额设置上限值和下限值的金额过滤器
*   速度过滤器，用于限制每小时从给定客户接收的交易数量
*   一个地址不匹配过滤器，用于检测帐单和送货地址不同的情况
*   从一个 IP 地址隔离可疑活动的 IP 过滤器
*   基于 Authorize.net 自身标准的可疑交易过滤器
*   IP 阻止功能，允许您阻止已知欺诈的 IP 地址

**周期性计费**

这很简单:提供客户的信用卡详细信息、账单金额、付款间隔、订阅时间，然后创建一个循环账单账户。不需要进一步的交互。

**e 检查处理**

Authorize.net eCheck 系统功能强大，功能齐全，具有多种功能。任何考虑处理电子支票的商家都会发现 Authorize.net 的解决方案是一个极好的选择。

**API 的选择**

这是 Authorize.net 最强的特性之一:商家能够以相同的价格在两种不同的集成方法之间进行选择。您可以随时切换方法，无论选择哪种方法，所有功能都可用。

可用的集成方法有:

*   **简单集成方法(SIM):** 毫不奇怪，这是最容易设置的方法，几乎不需要编程，也不需要 SSL 证书。您的安全订单页面直接托管在 Authorize.net 的安全服务器上，但可以根据您网站的外观进行定制。
*   **高级集成方法(AIM):** 这种方法让程序员可以让结账过程完全无缝、透明。客户永远不会离开你的网站，因为交易是在幕后服务器对服务器进行的。这种方法让你完全控制前端，以及你的客户对你的网站的体验。

*成本*

Authorize.net 没有在其网站上列出服务价格，因为它不直接向商家销售。感兴趣的商家需要联系目录中列出的[授权经销商之一。](http://www.authorize.net/solutions/merchantsolutions/stateresellerdirectory/))

定价因经销商而异，但以下是一些粗略的指导原则:

*   **开办费:**
    开办费从 25 美元到 300 美元不等(平均为 100 美元)。该费用通常不包括重复计费或其他增值服务。如果您同意支付更高的月费，一些经销商允许您降低安装费。
*   **月费:**
    月费从每月 10 美元到 30 美元不等，平均 15 美元。该费用包括处理报表和提供商户支持。
*   **交易费:**
    除了您的商户账户提供商收取的交易费之外，Authorize.net 将收取自己的额外交易费。这项费用从 5 美分至 15 美分不等，平均为 10 美分。一些经销商有能力提供免费交易，因为 Authorize.net 通过允许他们的商家每月免费处理他们的前 250 笔交易来奖励活跃的经销商。如果您希望每月交易少于 250 笔，如果您使用提供此折扣的经销商，您可以基本上避免交易费用。
*   **Recurring Billing:**
    Authorize.net’s recurring billing feature is not a standard feature: it incurs additional cost in the form of a set-up fee and a monthly recurring fee. Setup fees run from $40 to $100, with $50 being average. The monthly fee will range from $20 to $40, with $30 being average.
*   **欺诈检测套件:**
    Authorize.net 的欺诈检测套件也会产生额外的设置和经常性费用。安装费从 25 美元到 50 美元不等，平均 40 美元。每月费用因数量而异。如果你的业务每月不超过 100 笔交易，你的成本将在 8 美元到 20 美元之间，平均为 13 美元。如果您的企业进行 101 到 1000 笔交易，FDS 将花费 15 到 50 美元，平均为 30 美元。如果你的业务每月进行超过 1000 笔交易，你将看到每月 50 美元到 150 美元的范围，平均 100 美元。
*   通过 Authorize.net 的电子检查服务与其他服务的处理方式不同。价格不由经销商设定。然而，定价仍然会因商家而异，因为 Authorize.net 根据具体情况设定费率，费用由与账户相关的风险决定。然而，大多数商家通常支付 49 美元的申请费，如果获得批准，还要支付 100 美元的安装费。每张支票的处理费用是支票面值的 1.75%，外加每笔交易 30 英镑。没有月费，但每月至少 10 美元。

*优势*

尽管 Authorize.net 经销商几乎可以收取他们想要的任何价格，但该提供商的网关和服务的定价仍然相对较低。这是由于两个因素:

1.  低购买率:Authorize.net 有很高的购买率，经销商可以通过提高他们的服务价格来获得竞争力和利润。

3.  竞争:如此多的公司转售 Authorize.net 的服务，以至于定价需要保持在低水平:经销商竞争非常激烈！

Authorize.net 提供的功能(重复计费、欺诈检测和电子检查)满足了几乎所有商家的需求。这种全面的产品消除了实施第三方解决方案的需要，并使管理服务变得容易。

Authorize.net 并不认为一种整合形式优于另一种，所以它为客户提供两种形式。新的网站管理员会发现这一功能特别有吸引力，因为他们可能还没有编程技能来实现 Authorize.net 网关通过其可用的 API，但希望有这样的选择，在未来，一旦他们获得这些技能。

Authorize.net 的流行对它的许多用户都有好处。如果你看看目前大多数可用的购物车所提供的功能，几乎每一个都提供了开箱即用的内置 Authorize.net 支持。事实上，许多购物车和电子商务解决方案都在软件特性中列出了对 Authorize.net 的支持。在这些情况下，除非你正在为你的站点编写一个定制的购物车，否则你不必担心集成你的 Authorize.net 网关:它已经为你做好了。如果你选择编写自己的定制购物车，你会发现开发人员已经发布了访问 Authorize.net 的 AIM API 的源代码，这使得应用程序开发变得更快更容易。

*弱点*

尽管 Authorize.net 的服务很受欢迎，但它们并不完美。成为最大的支付网关提供商也有不利的一面。

第一个值得注意的问题是，不同经销商的定价会有很大差异。如果你的目标是获得尽可能低的价格，你需要多逛逛。不幸的是，你也可能会被商家账户的报价淹没，因为绝大多数经销商对获得利润丰厚的商家账户非常感兴趣。有些甚至会拒绝为你建立一个网关，除非你使用他们的商业服务。

作为最大的支付网关提供商，Authorize.net 经常成为黑客攻击的目标。2004 年 9 月，Authorize.net 成为分布式拒绝服务攻击(DDoS)的目标，该攻击持续了几乎整整一周，使一些商家无法处理销售，其他商家无法注册新服务。这不是 Authorize.net 经历的第一次袭击，也不可能是最后一次。诚然，任何企业都可能成为目标(事实上，2checkout.com 也在同一时间遭到攻击)，但公平地说，成为支付网关行业的第一名也意味着成为黑客攻击名单上的第一名。

***威瑞信 Payflow Pro***

Verisign 的名字广为人知，并以安全和信任而闻名。从 SSL 证书颁发到网关服务，它的安全性一直享有盛誉。

*特性*

**周期性计费**

Payflow Pro 循环计费功能非常标准，但确实提供了一个非常有用的功能:一个 API(应用程序编程接口),从中可以创建循环计费配置文件。因此，例如，您可以让您的网站在客户在线注册期间自动为您创建一个重复计费配置文件。对于新用户通过你的网站注册的虚拟主机或其他订阅类型的服务，这可以节省很多时间。

Payflow Pro 网关还能够发送自动定制的电子邮件。您可以向客户发送电子邮件收据以记录他们的记录，或者向他们发送通知，告知他们的卡被拒绝，他们需要为失败的交易做出新的支付安排。报告也可用于计费交易和个人客户档案。

**欺诈防范**

Verisign 将其防欺诈服务分为两个不同的包；每个都有增强保护级别的附加功能。

基本包包括七个基本过滤器，有助于降低商家的欺诈风险。包括的过滤器有:

*   高美元值过滤器
*   高项目数过滤器
*   运输/帐单不匹配过滤器
*   AVS 故障过滤器
*   CSC 故障过滤器
*   高风险邮政编码过滤器
*   货运代理过滤器

您还可以将销售限制在特定的 IP 地址。本质上，您创建了一个允许购买的有效 IP 地址的白名单。

高级包包括基本包的所有过滤器，另外还有 15 个增强保护的过滤器:

*   产品观察列表过滤器
*   高风险箱式过滤器
*   帐号速度过滤器
*   美国邮政服务地址验证过滤器
*   IP 地址风险列表匹配
*   电子邮件服务提供商风险列表匹配
*   地理定位失败
*   黑名单过滤器(信用卡和电子邮件地址)
*   高风险国家过滤器
*   国际送货/帐单地址过滤器
*   国际 IP 地址过滤器
*   国际 AVS 过滤器
*   良好的电子邮件列表过滤器
*   良好信用卡列表过滤器
*   总采购价格下限筛选器

Verisign 在账户监控方面提供了一个有趣的功能。实际上，威瑞信的员工监控你的销售，寻找可疑的交易。如果他们注意到一个，他们会进行调查，以确定销售是否真正合法或欺诈。如果销售看起来是欺诈性的，他们会阻止该销售被结算，并与您联系以进一步讨论交易。要使用此功能，您必须已经注册了基本或高级防欺诈软件包。

最后，Verisign 为希望使用 Visa Verified by Visa 和/或 master card secure code 技术的商家提供了一个买家认证程序。要使用此功能，您必须已经注册了基本或高级防欺诈软件包。

*成本*

*   **设置费用:** Verisign 在账户设置前一次性收取 249 美元的费用。
*   每月费用: Verisign 对这项服务每月收取高达 59.95 美元的费用。

***连接点 API***

[LinkPoint International](http://www.linkpoint.com) 是商业服务行业巨头 First Data Corporation 的全资子公司。LinkPoint 品牌是首个数据商业服务组合的门户分支。他们的连接点网关有两种风格:连接点 API 和连接点连接。LinkPoint API 是通过 LinkPoint 处理销售的“高级”方式。

*特性*

**周期性计费**

LinkPoint API 提供定期计费；计费周期可以设置为每天、每周、每月和自定义间隔。

**欺诈保护**

LinkShield 是 LinkPoint 的欺诈保护服务，它使用一种称为“风险评级”的功能来确定销售是否具有潜在的欺诈性。实现 LinkShield 有三种选择。选项 1 使用行为方法:它从交易中学习，并根据一段时间内学到的东西进行预测。选项 2 使用统计数据:它基本上将销售与以前成功和欺诈销售的数据库进行比较，并试图确定它与哪种交易最相似。选项 3 使用两种方法的组合。

注意:在撰写本文时，这项服务还处于 Beta 测试阶段。

**计税**

LinkPoint API 的一个有趣的特性是它的税收计算器，它根据州和自治市计算销售税。它允许商家决定如何或何时对顾客订购的产品和服务征收销售税。

**运费计算**

运送有形产品的商家的一个常见需求是计算每个订单的运输成本。此功能将根据区域或使用的运输类型计算运输成本。它可以提供统一价格，或根据销售的总价值计算运输成本。

**e 检查**

LinkPoint 的 eChecks 版本名为 VirtualCheck，功能丰富，应该能满足绝大多数商家。资金将在五到七个工作日内直接存入商家的商业支票账户。商家会在两到四个工作日内收到退回支票的通知，这些支票会自动重新提交。还包括电子邮件通知和实时报告。

*成本*

LinkPoint 不直接向商家出售其服务。希望使用 LinkPoint API 支付网关的商家需要联系其授权经销商之一。因此，定价会因经销商而异。

由于无法给出明确的价格，我将在此提供您可以预期的大致价格范围。

*   **安装费用:**建立一个 LinkPoint API 网关的费用从 75 美元到 300 美元不等；295 美元是平均水平。通过支付更高的月费可以获得更低的安装费，但这一优惠因经销商而异。
*   **月费:**link point API 的月费从 15 美元到 25 美元不等，平均为 20 美元。
*   **交易费:**连接点 API 不收取交易费。
*   **重复计费:**与其他主要网关提供商不同，重复计费功能包含在 LinkPoint API 中，无需额外付费。没有额外的安装费，月租费，或交易费。
*   **防欺诈:**当 LinkShield 升级为 gold 时，每月的费用大约为 25 美元。
*   **eCheck:** VirtualCheck 没有安装费，月费 10.00 美元，折扣率 2.00%，交易费率 25 美分。

*优势*

显然，LinkPoint API 最大的优势在于它的定价。虽然有些经销商收取相当高的安装费，但如果你从不支付交易费，这笔费用可以收回。每笔交易的额外 5 或 10 美分可以很快增加，这种产品的交易费用的缺乏可以很容易地补偿高昂的安装费用。如果商家计划使用重复计费功能，仅每月费用节省的成本就可以为注重价格的商家提供帮助。

除了最近迁移到新的数据中心之外，近年来，LinkPoint API 几乎没有经历过停机。最终，LinkPoint 计划建立一个备份数据中心以实现冗余。

*弱点*

LinkPoint API 最大的缺点是它只属于一个处理平台:First Data。解释这一点的最好方法是将其与计算机操作系统进行比较。为 Windows 编写的软件不能在 Mac 或 Linux 上运行。它们使用不同的架构，因此与每个操作系统的通信以不同的方式处理。同样的概念也适用于 Visa 和 MasterCard 处理网络。现有的有十几个，每个都有不同的 API 与网关(和信用卡终端)通信。

由 First Data 拥有的 LinkPoint 只与 First Data 的平台进行通信，因此您必须与能够访问 First Data 平台的商家提供商签约。许多人有；许多人不知道。有限的选择意味着您找到最适合您业务的计划的机会有限。如果您正在使用 LinkPoint API，并试图离开您当前的提供商，这可能会变得非常糟糕。如果您选择一个无法访问第一个数据平台的新提供商，您将需要为您的新网关学习一个新的 API，然后将其重新集成到您的购物车中(更不用说您将不得不熟悉您的新网关)。与任何专有软件一样，这种方法的目的是“锁定”商家。

LinkPoint 的另一大缺点是缺乏强大的反欺诈产品。随着欺诈成为每个在线商家的一个大问题，你会认为防欺诈工具将是任何网关的一个基本功能。虽然反欺诈解决方案正在开发中，但现在还没有，而且当它发布时，与其他网关提供商已经存在的反欺诈工具相比，可能会受到限制。

***连接点连接***

LinkPoint Connect gateway 本质上是一个精简版的 LinkPoint API。就与你的网站整合的方式而言，它类似于 Authorize.net 的 SIM 卡和 Verisign 的 Payflow Link。

*特性*

LinkPoint Connect 不提供任何特殊功能，如重复计费或防欺诈。

*成本*

*   **安装费用:**建立一个 LinkPoint Connect 网关的费用在 75 美元到 300 美元之间，平均为 295 美元。
*   **月费:** LinkPoint Connect 的月费从 15 美元到 25 美元不等；20 美元是平均值。
*   **交易费:** LinkPoint Connect 不收取交易费。

*优势*

就像 LinkPoint API 一样，LinkPoint Connect 不收取交易费用。

缺少 LinkPoint Connect 的功能虽然在下面被列为缺点，但也可以被视为优点。许多商家不想要花哨的功能，或者除了通过安全的订单在线接受信用卡之外的功能。LinkPoint Connect 的简单性消除了附加功能可能给某些商家带来的干扰。

*弱点*

LinkPoint Connect 是第一数据平台的专利。有关更多信息，请参见 LinkPoint API 的缺点。

与 Verisign 的 Payflow Link 一样，LinkPoint Connect 商家无法让结账过程对客户透明:他们必须将客户送到安全订单所在的 LinkPoint 网站。有关信息，请参见 Payflow 链接。

LinkPoint Connect 不提供内置反欺诈工具。有关更多信息，请参见 LinkPoint API 的缺点。

虽然 LinkPoint Connect 的功能有限，但它的价格却不受限制。大多数经销商收取的 LinkPoint Connect 价格与他们收取的 LinkPoint API 价格相同。虽然 LinkPoint Connect 的客户可以随时免费切换到 LinkPoint API，但如果他们不打算利用其附加功能，您基本上是在为您永远不会使用的网关服务付费。

虽然非常适合没有增长或扩张计划的小型企业，但 LinkPoint Connect 缺乏功能限制了它的吸引力。任何希望从 LinkPoint Connect 的基本卡处理功能开始，然后发展到包括其他服务的企业，都需要首先切换到 LinkPoint API，然后才能利用这些服务。如果网站管理员的编程技能有限，他们可能会发现这是不可能的，或者只能用很大的困难或费用来完成。
