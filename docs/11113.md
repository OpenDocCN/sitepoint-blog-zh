# 周末工作？7 种完成任务并脱身的方法

> 原文:[https://www.sitepoint.com/how-to-get-weekend-work-done/](https://www.sitepoint.com/how-to-get-weekend-work-done/)

我承认，我经常在周末工作。虽然这更像是一个有意识的决定，而不是不可避免的需要，但我仍然希望能够尽快完成工作，离开电脑。这里有一些建议，可以帮助你在这个周末加快工作进度，出去享受一些休息时间。

*   **设定一个期限，坚持下去。**虽然你的工作很重要，但我敢打赌你还有其他更想做的事情。因此，与其制定一个导致马拉松式工作的开放式计划，不如设定一个具体的停止时间。有时候，争取家人的帮助会有所帮助。

*   关注你的候选名单。在做其他事情之前，给自己 5 分钟时间办理入住手续，看看情况如何。看看你的待办事项，找出 3 个(或者 2 个、4 个、10 个——任何对你有用的)你可以在设定的时间内切实完成的高优先级事项。

*   除非绝对必要，否则不要查看电子邮件。电子邮件非常耗费时间，尤其是当你时间有限的时候。太容易偏离轨道，被诱惑去处理新的问题，所以最好远离它。

*   远离网络。与电子邮件一样，打开浏览器是危险的，除非你在工作中特别需要它。这包括你的 RSS 源和所有的社交媒体网站。

*   早点完成。我讨厌当我试图享受休假时，工作问题悬在我头上，所以对我来说，最好是早点完成工作，甚至在其他人醒来之前。这让我以后真正从工作中解脱出来。

*   进入安静区。分心会让原本应该快速完成的任务花费两倍的时间。试着给自己一个支持所需的短期强度的环境，这样你就不会与损害你生产力的走走停停做斗争。

*   把它留到星期一。当其他方法都失败时，减少你的损失，并发誓在周一完成。事实是，总会有工作要做，无论是回复电子邮件、推进项目还是处理幕后业务管理任务。有时候你只需要让事情过去一两天，然后专注于现在。

周末在我们的时间表中绝对是少数，所以保持专注，即使你平时不这样做，也要采取所有你可以接受的捷径，这样你就不会失去周末去工作。

你还在等什么？停止阅读这篇文章，这样你就可以完成你的工作，远离电脑，享受你的周末！
