# 数字教材对学生有用吗？

> 原文：<https://www.sitepoint.com/will-digital-textbooks-work-for-students/>

几天前，*西雅图邮报的*亚马逊博客[的 Andrea James 证实了传言](http://blog.seattlepi.nwsource.com/amazon/archives/146874.asp)称亚马逊将推出下一版本的 Kindle 电子书阅读器，目标是大学生。进入大学市场对亚马逊来说意义重大——大学教科书市场估计有[55 亿美元](http://www.nacs.org/public/research/higher_ed_retail.asp)——对学生也有意义。

根据全国大学商店协会的数据，学生每年平均花费 702 美元在必修课程材料上。你可以打赌，其中大部分可能花在课本上，平均每本 53 美元(新的，每本 48 美元)。大学委员会建议学生根据他们就读的学校，在书籍和用品上的预算高达 1229 美元。而且情况越来越糟:美国教科书价格上涨速度[是通货膨胀率](http://www.gao.gov/docsearch/abstract.php?rptno=GAO-05-806)的两倍。

如果电子书的价格能够保持在较低水平——特别是如果这些成本中的任何一项能够得到大学的补贴，那么一台可以在整个大学四年(或更长时间)使用的设备的 359 美元的一次性成本将解决成本问题。许多教科书出版商已经推出了他们书籍的电子版。例如，除了印刷版本，麦格劳-希尔出版了将近 95%的数字化教科书。因此，出版商已经有尝试数字发行的意愿。

但还是有问题。

根据少数学生公共利益研究小组的一项新研究，数字教科书现在正走向学生的歧途。最令人吃惊的是，该研究发现，目前数字教科书实际上比纸质教科书成本更高，这完全抹杀了电子教科书分发的最大潜在好处(除了明显的环境影响)。学生 PIRGs 发现，电子教科书“比在网上购买和销售的相同标题的二手硬拷贝平均贵 39%。”

这项研究还发现，数字教科书很难印刷。打印数字教材似乎有悖直觉，但对一些学生来说是必要的。此外，在数字教科书上记笔记很困难。Kindle 支持键入的文本笔记，但它们远不如荧光笔、便利贴或在印刷书籍的空白处涂鸦那样直观易用。因为每个人的学习习惯不同，在 Kindle 上敲笔记不会对每个学生都有效。

最后，报告发现，许多学生对只能在电脑屏幕上观看的数字教科书持谨慎态度。Kindle 实际上解决了这个问题，但研究发现，75%的受调查教科书在 180 天后过期，学生 PIRGs 说这是不可接受的。一旦购买，学生应该能够无限期地保存和阅读数字教科书。

该报告列出了数字教科书的三个要求，以使它们为学生服务:更低的价格，更好的印刷选择，以及可访问性。亚马逊和教科书出版商完全有可能满足这些标准，并成功创造出学生会发现比纸质书更有用的数字教科书。如果亚马逊真的想从这个 550 万美元的市场中分一杯羹，他们最好留意一下学生们在说什么。
