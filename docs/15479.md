# 管理你的时间

> 原文:[https://www.sitepoint.com/managing-time/](https://www.sitepoint.com/managing-time/)

你最宝贵的资产是你的时间。正确的管理意味着网上成功或失败的区别。

因为在互联网上有太多的事情要做，所以很容易分心，并且忘记你最初打算完成的任务。就在我写这篇教程的时候，我收到了四条 ICQ 消息和五封电子邮件，都来自不同的人。互联网是一种极好的资源，也是一种有效的交流方式，但它也可能成为一种巨大的干扰，浪费你的时间。

遵循以下建议，充分利用每一分钟:

##### 1.)处理电子邮件，快

为了充分利用你的时间，学会管理你的电子邮件是你必须学会做的第一件事。通过使用模板、过滤器和垃圾邮件杀手，你可以大大减少花在浏览邮件上的时间。获得像 Eudora Pro 这样的高级电子邮件程序，并学习使用它的所有功能。虽然最初可能需要一些时间，但从长远来看，回报是巨大的。

通过阅读《管理你的电子邮件》,了解你需要知道的一切。

##### 2.)保持专注

不要偏离你的目标。给自己写一张便利贴，上面写着“这对我的网站有帮助吗？”贴在你的显示器上。每当你浏览网站、回复邮件或在电脑上做其他事情时，问问自己这个问题。如果手头的任务不是增加流量，改善你的网站，或者以另一种方式增加你的业务/收入，那么它可能不值得做。

当然，你也应该留出一些时间做一些有趣的事情。

##### 3.)制定一个时间表

给自己买个笔记本(非电子型——还记得纸质的吗？)，用它列出你需要完成的所有任务，以及一个大概的截止日期。按照重要性给任务编号，然后开始处理它们。

在我的笔记本上，除了列出需要做的事情之外，我还列出了我想回顾的事情，以及我可以做些什么来改进网站的一般想法。

注意:每隔一小时左右给自己一个短暂的休息时间，这很重要。研究表明，在短暂休息后的前半个小时里，你最清醒，效率最高。因此，如果你在工作时间内安排短暂的休息，你会比仅仅继续工作做得更多。

##### 4.)自动化

电脑的伟大之处在于，它们可以整日整夜地工作，完成无聊的、重复的任务。在互联网上，自动化是通过 CGI 脚本实现的。剧本的一个很好的来源是 CGI-Resources.com。我建议您偶尔浏览一下该网站上的索引，寻找能让您的生活更轻松、节省时间的脚本。您甚至可以考虑自己学习用 PERL 编程，这样您就可以编写自己的省时脚本。CGI101.com 大学为非程序员提供了一个很好的教程。如果你经常发现自己在做同样的工作，问问自己是否有办法让它自动化。
