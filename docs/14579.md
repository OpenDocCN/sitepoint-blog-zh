# 很酷的冷聚变东西…

> 原文:[https://www.sitepoint.com/cool-coldfusion-things/](https://www.sitepoint.com/cool-coldfusion-things/)

任何编码语言最酷的事情之一就是看着人们把它从一个领域(网络)带到另一个领域。例如，php 开发人员将[挂钩](http://gtk.php.net/)到 GTK 工具包中，这使他们能够为桌面等构建 GUI 应用程序。

最近有两个很酷的 ColdFusion 产品从网上飘到了我的浏览器上。

第一个是 CF-TALK 邮件列表上的一个家伙，他用各种软件包制作了一个从 CD 上运行的 CF 应用程序。你可以在这里看到[初始线程，在这里](http://www.houseoffusion.com/lists.cfm/link=m:4:32783:164406)看到他的[在线文档。第二个网址可能并不总是有效，因为我认为开发者把它放在家里。我确信你能看到这将带来的营销优势。想想迷你 CD 和促销活动。](http://67.124.145.42:9090/BlueDragon_webapp_61/CFAnywhereDocs/CFAnywhere.html)

第二个项目是一个软件程序，它将允许您创建和运行桌面 ColdFusion 应用程序。CORAL 是一个不错的小应用程序，可以让你从网络中解脱出来。它允许您以包括桌面在内的多种方式开发和部署 ColdFusion 应用程序。这很好，因为每次你想在桌面上部署你的应用程序时，你都不必为 CF 许可证付费。

希望你会发现这些小创意很有趣，也许会激发你的想象力。
