# 像素化的世界:像素艺术网站

> 原文:[https://www . site point . com/a-pixelated-world-pixel-art-websites/](https://www.sitepoint.com/a-pixelated-world-pixel-art-websites/)

像素艺术是与旧的电脑和视频游戏机游戏。一般分为两个子类:等距和非等距。等距型经常出现在计算机游戏中，使用三维视图而没有真正的三维处理。非等轴像素艺术是具有俯视图、侧视图、前视图、仰视图或透视图的像素艺术。随着计算机图形的改进，像素艺术开始看起来有点过时和笨重，但近年来，似乎有一种对所有复古和这种类型的数字艺术的重新欣赏。

今天，我们经常在小型计算机图标中看到这种类型的数字艺术，但像素艺术也已被用于说明杂志文章、电视广告和网站。下面你会找到一些使用像素艺术创造独特外观和感觉的网站，以及一些分享和学习如何创造你自己的像素艺术的资源。

毫无疑问，Eboy 是最著名的像素艺术供应商之一。他们创造可重复使用的像素对象，并利用它们来构建复杂和可扩展的艺术品和玩具。他们的工作已经在《连线》、《大众科学》和《财富》500 强杂志上发表。

[![Eboy](../Images/db9a9e4d757394d6a9e56c441a727aad.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/Eboy.png)

另一个著名的像素艺术网站是艺术家克雷格·罗宾逊的[Flip Flop flying’](http://www.flipflopflyin.com/)，他有一个让像素看起来可爱的真正诀窍。

[![FlipFlop](../Images/f5e6560034c2dbb55233e373edbe52bf.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/FlipFlop.png)

[巨魔军](http://www.armyoftrolls.co.uk/website/html/homepage.html)是数字艺术家加里·卢肯的作品集网站，展示了一些极其细致的像素艺术，这些艺术深受他对视频游戏的热爱的影响。

[![ArmyOfTrolls](../Images/0f180d38e6a9c679849e5001d7ff2b45.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/ArmyOfTrolls.png)

Goon Tower 是 SomethingAwful.com 论坛用户的一个合作项目。这座塔的每一层都由不同的用户建造/绘制。

[![GoonTower](../Images/820a605a99cbbdf2d3e3a2e159102dee.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/GoonTower.png)

[太空入侵者](http://www.space-invaders.com/som.html)是一个由数字艺术家 Invader 创建的网站。这个想法是用受第一代街机游戏启发的角色“入侵”世界各地的城市，尤其是现在经典的太空入侵者。

[![SpaceInvaders](../Images/a660e82731057f8a0fba94c48f58e7dd.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/SpaceInvaders.png)

在多重滚动条和像素艺术之间，中国网站 [Love Pixel](http://www.lovepixel.idv.tw/page/main.htm) 看起来绝对是电脑复古。

[![LovePixel](../Images/3a28eced337c478401a50d5d5e2233c9.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/LovePixel.png)

Polyfonken 的视觉障碍只是一个以音乐和艺术为特色的大型网站的一部分

[![Polyfonken](../Images/7dd802b18bb91c258ad16fe1519cb8ba.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/Polyfonken.png)

今年早些时候，Edushi 制作的香港像素艺术地图在网上引起了相当大的关注。在谷歌地图上，这基本上是三维的香港，但所有的建筑和结构都是像素艺术。超级棒的东西。

[![HongKong](../Images/c88edd60926df0f16cbfe9fd6d81aac1.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/10/HongKong.png)

如果你是像素艺术的忠实粉丝，想要展示你自己的作品或从在线教程中学习，那么你应该去 [Pixel 联合社区](http://www.pixeljoint.com/pixels/new_icons.asp)看看，在那里你会找到很多喜欢他们的像素的志同道合的人。
