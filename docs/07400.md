# WordPress 3.5 有什么新功能

> 原文:[https://www.sitepoint.com/wordpress-35-whats-new/](https://www.sitepoint.com/wordpress-35-whats-new/)

*这篇文章已经过时了，[点击](https://www.sitepoint.com/whats-new-in-wordpress-4-6/)查看 WordPress 4.6 的新功能。*

根据最新数据，WordPress 为五分之一的网站提供支持。一半以上运行内容管理系统的网站也在使用 WordPress。是的，有些人会认为 WordPress 不是 CMS，但他们只是吹毛求疵。而且不对)。最后，30 个月大的 3.x 版本被 95%以上的 WordPress 用户使用。

如果一切顺利，WordPress 3.5 将于今天发布。以下是您一键升级后可以期待的新功能…

## 界面更新

保留了现有的布局和菜单，但重新设计和简化了欢迎屏幕。你们中有视网膜显示器的人会喜欢新的图标集。

## 新媒体上传程序和库

上传、预览和插入图像的系统已经彻底检修。拖放和排列画廊应该容易得多。

## 新的 2012 主题

2012 是第一个实现响应式网页设计技术的原生主题。虽然有大量的第三方响应模板可供选择，但许多网站坚持使用默认模板——从现在开始，它们将是移动友好的。

该主题还包括一个支持小部件的主页和来自谷歌目录的清晰可见的 Open Sans 字体。最后，它实现了一组扩展的文章类型:

*   标准博客帖子
*   **旁白**——简洁的类似推特的帖子，没有标题
*   **图像** —单张图像
*   **链接** —链接
*   **报价** —报价
*   **状态** —状态更新

## 最喜欢的插件

任何插件都可以被标记为“收藏”,这样它就可以在你所有的 WordPress 安装中快速安装和配置。

## 没有更多链接

链接设施已经被驱逐。我从未用过，也不知道有谁用过。如果这对你来说是一个交易破坏者，下载新的[链接管理器插件](http://wordpress.org/extend/plugins/link-manager/)来恢复功能。

## 杂项改进

不够？关于:

*   [oEmbed](http://oembed.com/) 更轻松地从 YouTube 和 Slideshare 等网站插入媒体
*   更好的键盘导航
*   一个新的颜色选择器
*   改进的可访问性
*   移动应用程序可以与 XML-RPC(远程过程调用)集成，这是默认启用的
*   更快的发布 API
*   一个搜索评论的新 API
*   从右向左语言支持
*   默认情况下使用 UTF-8 编码。

## 我应该升级吗？

几乎可以肯定。但是我建议在对你的网站、模板和插件进行全面测试后再这样做。问题很少出现，但是 WordPress 开发团队不可能测试每个配置或者主题和插件的组合。

那些更容易紧张的人可能更喜欢等到 3.5.1 版本，这无疑会修复任何主要的兼容性问题。如果根据以往的经验，它将在几周内发布。
