# 社交媒体中的四个危险假设

> 原文：<https://www.sitepoint.com/assumptions-in-social-media/>

关于社交媒体中的“最佳实践”以及使用社交网站的最佳方式有很多讨论。然而，当你远离正确的使用建议，进入日常使用时，问题不在于这些网站的信息或使用，问题可能是你的信息在另一边被感知的方式，而你甚至不知道。

即使你非常清楚自己的目标和社交网站的用途，并相应地迎合你的信息，你也永远不知道观察者会如何看待你在社交网站上发布的内容。

我们都在假设谁在阅读，人们会怎么想，他们会怎么反应，这就形成了我们用来决定发布什么和不发布什么的界限。但并不总是如我们所愿。以下是我见过的一些有可能适得其反的假设。

## ![](img/39b008787bcb0f27feef6725e0035a56.png "embarrass") 1。我在社交媒体上不活跃的客户看不到我的消息。

我见过很多很多自由职业者和企业主对没有社交媒体账户的客户发表轻蔑的评论，通常是出于对工作或沟通的一时沮丧。我看到他们中的一些人不得不解释、收回这些评论并为此道歉。

在许多情况下，一个人不需要在社交媒体网站上有一个账户来阅读更新，一个你可能认为不太懂技术的客户仍然可以偶然发现你的账户并阅读你的帖子。或者更糟的是，员工、同事或家庭成员可以在他们的评论中填写。

## 2.我有一个私人账户，所以什么都可以。

如今，社交媒体中存在诸多隐私问题和困惑，我认为可以公平地说，你在社交媒体上发布的所有内容都有可能公之于众。即使你已经采取了一切可能的措施来确保你的内容是隐私的，你也不知道你联系的人什么时候会分享你公开发布的内容，从而否定你所有的隐私措施。

## 3.那个有争议的评论只是一个笑话，每个人都知道。

坐在社交媒体的泡泡里，很容易感到安全，因为你知道每个人都在关注你的更新。毕竟，当你有新的关注者或好友请求时，你会得到通知。不过，请记住，可能有人会在不告诉你的情况下跟踪你的更新和帖子，你永远不知道一个黄色笑话会被你甚至不知道正在观看的人察觉到。

## 4.我所有的关注者/朋友都知道我是[填空]。

你知道你的生活中发生了什么，你在社交媒体网站上交流的好朋友和家人也知道你发生了什么。但是当你发布一些没有背景故事却可能有非常不同的含义的东西时，你就有被误解的风险。这并不意味着你需要审查自己，只是说快速过滤你的评论可能是明智的，这样你至少可以意识到潜在的误解。
