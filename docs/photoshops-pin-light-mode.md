# Photoshop 的大头针光照模式

> 原文：<https://www.sitepoint.com/photoshops-pin-light-mode/>

下面是针光模式的官方描述:

> 根据混合颜色替换颜色。如果混合色(光源)比 50%灰色亮，则比混合色暗的像素会被替换，比混合色亮的像素不会改变。如果混合色比 50%灰色暗，则比混合色亮的像素会被替换，比混合色暗的像素不会改变。这对于给图像添加特殊效果很有用。

如果你发现很难接受这样的描述，你不是唯一一个。很长一段时间以来，我试图用一个例子来说明这个概念，要么是我弄错了，要么是描述不太正确。我跟大家分享一下我的尝试，也许你们中一些比较聪明的人可以告诉我我哪里做错了！

我从两个相同的层开始，一个翻转了 90 度。两层之间都有黑色、白色和灰色阴影。中间灰色为 50% (128-128-128)。

![](img/89dcbc107fdc24e3982566765bcd99bb.png)

![](img/da12c5715abb0b7bc9713edddd134632.png)

水平图层在上面，设置为点光源，(两个图层都裁剪了一点，以查看未混合的原始位)下面是结果:

![](img/4400bb65b862a1854488fe7299fde3b9.png)

嗯。让我们再看一遍。我将上面的两个部分标记为“混合层变亮”(小于 50%)，下面的两个部分标记为“混合层变暗”

![](img/9d5988cdd7429cd28af9812c66ed7ea2.png)

重点放在顶部:“如果混合色(光源)比 50%的灰色亮”，这对于顶部的两个部分是正确的，“比混合色暗的像素被替换，比混合色亮的像素不变。”

所以顶部的纯白条纹是有意义的——垂直层中较暗的像素全部替换为白色。

但是第二条，浅灰色条纹…？垂直的白色部分不会改变，因为它比灰色部分亮。但是根据定义，似乎垂直的黑色和深灰色部分应该被浅灰色混合色所取代。嗯嗯。同样的矛盾似乎也存在于较暗的灰色区域。

有人想解释一下吗？(无意双关。)[下载示例。psd 文件](https://examples.pixelmill.net/sitepointblog/040913/040913.psd)如果对你有帮助的话。

嗯，我不懂 Pin Light 模式不代表你我不会用！先来一张极其可爱的狗狗的图片:

![](img/cceca92e5f6914472babcf94956d50d7.png)

将狗放在白色背景上，并将狗图层设置为灯光模式。

![](img/63b13b43dc71fd7615ac569041543f89.png)

哇哦。所有那些不太白的灰色都被淘汰了，取而代之的是一种非常前卫、过度曝光、在杂志和 CD 封面上常见的白色。

由于针光用基色代替了所有的浅灰色，我甚至可以给袜子(狗)一个奇怪的复古圆点效果。好玩！

![](img/ee6ce5cf03f94cbe18d80dea7b2e47a3.png)
