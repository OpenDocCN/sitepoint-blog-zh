# 搜索词、竞争和关键词指标

> 原文：<https://www.sitepoint.com/search-terms-competition-and-keyword-metrics/>

搜索引擎营销可以是一个非常复杂的业务，乡亲。你可以随心所欲地简化它……或者你可以把它变得比需要的复杂得多。无论哪种方式，都始于搜索词策略。当我想到搜索词策略时，我看到了许多起作用的因素，但让我们从这些因素开始:

**相关性:**当我谈到一个搜索词的相关性时，我实际上是在问一个简单的问题:使用这个搜索词的人当中，有百分之多少的人对你的网站提供的东西真正感兴趣*？当我说感兴趣时，我指的是*准备好、愿意并且能够采取行动或购买的人。*相关性会随季节变化——人们可能会在夏天寻找滑雪小屋，但他们不太可能实际预订。*

 *有可能提高网站的相关性，并随之提高搜索词的相关性。这可以通过多种方式实现，比如添加内容，或者改变搜索者到达的登录页面。网站所有者经常高估极其普通的搜索词的相关性，但是有一些简单的方法可以让你得到更好的估计。

**业务相关性:**另一种看待搜索词的方式是考虑它与你的相关性。在网站上有一套 Flash 教程的网站设计师可能不太关心推广这些教程，因为寻求 Flash 教程的人不太可能成为客户。

为了简化问题，我们将从金钱的角度来定义商业相关性——为了让一个感兴趣的访问者访问我们的网站或登陆页面，我们要付多少钱？这不同于我们愿意为“任何”访问者支付的价格。我们想考虑一下，你愿意为一个*准备好了、愿意并且能够采取行动或购买的人支付多少钱。*

有可能提高搜索项的商业相关性。提高网站的转化率是增加商业相关性最简单的方法。或者，我们假设的设计师可能会在教程页面上放置 Adsense 广告来创收，或者为其他设计师提供 Flash 培训。

一些企业认为他们的客户有更大的“终身价值”，这也增加了业务相关性。在我的生意中，我们的大部分客户会反复从我们这里购买，所以我愿意支付更多的钱来赢得第一笔销售。

对于我们的目的来说，一个搜索词的流行度被定义为它在搜索中被使用的次数。这将因不同的搜索引擎而异，逐月等等。

为了简单起见，我们将流行度定义为一个月内所有搜索引擎的搜索次数。确定一个搜索词的真正受欢迎程度是不可能的，但是有一些工具可以让我们做出合理的估计。

虽然人们有时会尝试，但你无法提高一个搜索词的受欢迎程度。“模因词营销”的概念是基于创造一个术语并推广其使用的想法。祝你好运！

**竞争:**对搜索者注意力的竞争也可能难以估计。这是我的公司非常关注的领域，因为关键词是我们的业务，我们已经开发了许多不同的指标来帮助我们了解竞争格局。

了解竞争有助于定义在搜索者面前需要付出的努力程度。在某种程度上，努力的程度转化为竞争的成本。竞争指标也有助于我们理解所涉及的风险程度，或者努力不成功的可能性。

衡量竞争是一件非常复杂的事情，下周我将开始讨论这个问题。

我们能用这些做什么呢？

现在，我们已经定义了一些指标，让我们抛出一些概念和公式来解释一下:

**加权流行度:**搜索词的“加权流行度”是相关度乘以流行度。因此，如果我们认为 10%的搜索者对我们提供的内容感兴趣(相关性)，并且每月有 10，000 次搜索(流行度)，加权流行度公式表示每月有 1000 次(10，000 x 10%) **目标搜索**。

**潜在价值:**搜索词的“潜在价值”是加权的流行度乘以业务相关性。因此，如果我们相信每月有 1000 个目标搜索，并且我们愿意支付 1.00 美元来获得对我们网站感兴趣的访问者，如果我们能够以某种方式让所有这些搜索者访问我们的网站，那么每月的潜在价值是 1000 美元。

这个问题真的很大！

从这两个公式中得出的一个大问题是，这 1000 美元的潜在价值中，我们能为我们的企业带来多少，以及这样做的成本是多少。我想告诉你，这是一个简单的问题，但事实并非如此。我想告诉你，当你使用点击付费广告时，它就不那么复杂了，但事实并非如此。

下周，我将讨论如何使用我们开发的关键词指标和方法来回答这个大问题，以及如何将所有这些整合到一个可靠的关键词策略中。我们将在周一评估有机(SEO)活动的竞争和努力程度，周三整理点击付费，然后在周五查看一些有用的工具。*
