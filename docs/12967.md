# UX 网站方向:让你的用户感到特别

> 原文:[https://www . site point . com/we B- directions-UX-making-your-users-feel-special/](https://www.sitepoint.com/web-directions-ux-making-your-users-feel-special/)

安迪·巴德今天早上开始了[网络方向用户体验](https://ux08.webdirections.org/)会议，他就如何让一个真正*有效的*网站让用户如此难忘发表了一些精彩的见解。

安迪演讲的核心信息是，其他行业早就明白积极的用户体验的重要性，网络可以从中学到很多。

他的演讲给出了许多技巧，并附有大量丰富的例子:

1.  第一印象很重要:酒店经营者明白这一点，因此好的酒店会在大堂提供微笑的迎宾员、巧克力和放在枕头上的便条。
2.  **关注细节引起关注**:苹果是一家意识到关注细节可以带来不同的公司——想想他们在产品包装上所付出的努力，这导致客户实际上拍摄了他们的 iPod 开箱。从新宝马的关门声到迪斯尼主题公园的主题垃圾箱，人们都会注意到这些东西。
3.  个性化和定制的重要性:星巴克允许顾客定制他们的咖啡，任天堂 Wii 和其他游戏允许用户创造他们自己的角色。“第二人生”将此提升到一个新的高度，以满足人们彰显个性的需求。对于 web 开发人员来说，一些简单的事情，比如用人们的名字来欢迎他们，可以对他们在您的网站上的体验产生巨大的积极影响。
4.  安迪提到电脑游戏是他灵感的巨大来源，并展示了一个战斗游戏的视频，教玩家如何逐步完成任务。通过逐步发布关于如何在可消化的块中执行任务的信息，您可以避免淹没它们。例如，Basecamp 显示新功能的可选视频，Yahoo！site 在去年重新设计时利用灯箱来解释新功能。
5.  注意日常事务:安迪在阐述这一点时大量引用了酒店行业——门卫为你叫出租车，服务员在你没有注意到的时候给你斟满酒，酒吧女招待记得你的饮料，这些都是在一瞬间创造积极用户体验的很好的例子，可以让用户在整个访问过程中获得积极的体验。网络类比包括通过预先填充智能默认值来为用户处理不必要的任务。
6.  **让它变得有趣**:安迪展示了一个饮料包装的例子，在容器的底部有文字“不要看我的屁股”。像这样的小接触显然需要正确的上下文和一点思考才能正确地完成，但当它们起作用时，它们可以让你的用户在一天的剩余时间里保持微笑。另一个例子是来自 MOO 的个性化电子邮件，上面写着“你好，我是小 MOO，将管理你订单的软件……”一个银行网站可能不会如此可爱，但绝对有可能既有趣又不专业。
7.  **反馈很重要**:扑克机非常重视反馈——它们会发出“叮当”的声音，当硬币击中金属托盘时，巧妙制作的“叮”声会引起其他玩家的注意，坐下来试着自己使用一台机器。同样，电梯里的一个按钮总是亮着——如果它不亮，你可能会怀疑电梯是否坏了！状态栏和其他指示器让用户知道他们目前在过程中的位置，有助于管理期望和保持用户参与。
8.  推荐是强有力的 : last.fm 是一个很好的例子，说明向用户推荐音乐是一项非常棒的服务，能够真正增加价值。如果你能向你的用户提供上下文信息，利用整个用户群创造的知识，那么你的用户就会注意到。
9.  **用户喜欢竞争**:通过提供竞争元素——无论是通过分数和排行榜，还是仅仅通过让人们“收集”东西——徽章、博客帖子等——你的用户群将有额外的理由回到你的网站并参与你的数据。当然，排行榜也可能产生负面影响——当那些排名靠前的用户为了保持他们在排行榜上的位置而开始发布质量而不是数量时，digg 排行榜就不再是一个有用的指标了。

这些体验可以被绘制成一条[用户体验曲线](https://www.andybudd.com/archives/2007/10/designing_the_u/)——一个真实的图表，虽然主观，但显示了用户随着时间推移的积极和消极体验。安迪提到了马斯洛的*需求层次理论*在网络上的应用。层次结构中的项目有:

1.  功能的
2.  可靠的
3.  可用的
4.  方便的
5.  令人愉快的
6.  有意义的

我同意 Andy 的评论，在网络上的用户体验很少能满足层次结构顶端的体验水平(愉快的、有意义的体验)。他鼓励与会者从其他行业和线下世界的其他领域寻找灵感，为网络带来积极的用户体验。
