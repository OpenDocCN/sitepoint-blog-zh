# 运营移动自由职业者业务所需的 10 款必备应用

> 原文:[https://www . site point . com/the-10-essential-apps-you-needle-to-run-a-mobile-freelance-business/](https://www.sitepoint.com/the-10-essential-apps-you-need-to-run-a-mobile-freelance-business/)

有很多文章声称，为那些希望 100%数字化经营业务的自由职业者展示了 50 个或更多“基本”网络和桌面应用。

现实情况是，任何声称有 50 甚至 30 个“基本”应用程序供自由职业者使用的人可能都没有客户，也没有足够的空闲时间来摆弄软件。我一直在考虑运营我的业务所需的绝对最少的基本软件，这些应用程序需要跨平台(理想情况下是 web 应用程序，因此平台甚至不需要成为考虑因素)，并且具有移动应用程序或可以在移动浏览器上轻松访问。

这样，我就可以随心所欲地四处移动，而不会产生不良影响——或者呆在办公室里，而不会觉得自己为了拥有一套更具移动性的应用程序而牺牲了一些功能。这些是我的结果。

### 1.佩莫

作为一名自由职业者，跟踪你的时间、给你的客户开发票和管理项目是占据你大量时间的管理任务。当你处于紧张状态时，像跟踪你的时间以获得准确的报告和发票这样的事情甚至会增加开销，忽略管理琐事而专注于产出。

Paymo 是自由职业者的瑞士军刀。它拥有适用于各种平台的直观、不引人注目的时间跟踪工具、快速简便的发票开具和强大的项目管理。如果你按小时计费，你会发现用 Paymo 你会得到更多的报酬，因为你可以抓住你的时间，比用其他方式更容易计费。

### 2.less 会计

随着你所有的时间被追踪和发票被寄出，你需要在钱进来和出去的时候保持你的账本。LessAccounting 提供的服务有点像用你的簿记员代替你的每月访问，最好与 Paymo 这样的服务结合使用。你可以在许多银行设置它，每晚输入你的费用和存款。

LessAccounting 的一个优点是，你可以在你的计划中添加一个真正的簿记员的帮助——除了每月 30 美元的费用外，每个季度 6 小时的簿记帮助每月只需 70 美元，或者每个月 7 小时的帮助每月 270 美元。

### 3.要点

自由职业者应该实施的最重要的实践之一是良好的联系管理。即使当你忙于项目时，也要保持你的关系网，和那些你见过的人或者对你的生意表现出兴趣的人保持联系。

虽然你的普通地址簿应用程序可以工作，但一个强大的联系人管理应用程序是 [Gist](http://gist.com/) 。这是最早的社交联系管理器之一，在重新联系之前，很容易就能了解一个人在个人和职业方面都做了些什么。最重要的是，Gist 是免费的，许多这种级别的 web 应用程序都非常昂贵。

### 4.HootSuite

社交媒体营销风靡一时，但对于自由职业者来说，口碑要重要得多。然而，这并不意味着社交媒体不重要。有了 HootSuite，你可以监控包括 Twitter、脸书和 LinkedIn 在内的一系列网络，向每个网络发布更新，并与其他用户交流。

有一个地方可以查看、发布和回复是一个真正节省时间的地方:你没有很多时间在社交网络上玩，但你需要投入时间来加强你建立的联系。HootSuite 可能是在不降低效率的情况下减少你在竞技场上花费的时间的最佳解决方案。

### 5.谷歌文档

谷歌文档是一个显而易见的选择，但也是不可或缺的。每个自由职业者都需要一个写文档的地方，一个与其他人合作的地方，以及一个与目标受害者分享这些文档的简单方法。

对于从事数字化工作的作家来说，有一个地方来创作可以与客户分享并以最方便的格式导出的文章(从 HTML 到 Word)显然是必不可少的。谷歌文档提供了优秀的组织和搜索工具，这对于高产作家来说很重要。对于其他自由职业者来说，建议和报告仍然是生活的一部分，Docs 是除了一些臃肿且通常昂贵的桌面办公套件之外的最佳选择。

### 6.Evernote

有些人可能想知道为什么值得使用 Docs *和* [Evernote](http://www.evernote.com/) 。我最大的原因，也是公认的最肤浅的原因，是因为我喜欢将我正在处理的文档，比如一份提案或一篇文章，保存在 Docs 中，而将 web 剪报、笔记和头脑风暴放在另一个应用程序中——这正是 Evernote 擅长的。我们大多数人每天都需要做很多笔记，剪辑很多信息，这就淹没了谷歌文档中真正的工作。

更重要的是，无论你是在桌面还是移动设备上，都可以更容易地将笔记和照片放入 Evernote。谷歌文档对于快速捕获任务来说太慢了。

### 7.谷歌邮箱

显然，最好在你自己的域名上使用更专业的电子邮件地址，而不是@gmail.com 地址，所以你可以用 [Google Apps for Business](http://www.google.com/apps/intl/en-au/group/index.html) 做到这一点是件好事。但是网络上没有一家提供商能打败 T2 的 Gmail。免费的存储容量是巨大的，网络邮件软件是无与伦比的。

如果你使用的是 ISP 或主机提供商的邮件服务器，那就切换到 Google Apps，关闭那些劣质垃圾邮件。

### 8.网络电话

Skype 取代了许多东西，几乎可以在任何设备上运行良好，从桌面到 iOS 和 Android，再到为 Skype 设计的普通手机。

有了 Skype，您可以设置一个可以随身携带的商业号码，在全球范围内拨打便宜的电话，并免费与其他 Skype 用户通话。大多数自由职业者通过即时消息交流的时间比通过语音交流的时间要长得多，而 Skype 为其他即时消息服务提供了很好的替代方案，可以很好地存档以供日后参考。但是您可能已经使用 Skype 多年，不需要太多的介绍——继续使用它就行了！

### 9 .奇迹

神童是一个出色的任务管理者。我传统上更喜欢 OmniFocus 这样的应用程序，它们提供了很多管理超能力的任务，但有时你只需要列出一个清单并完成它。Wunderlist 可以让你毫不费力地做到这一点，并且可以在大多数桌面和移动平台上运行。

它还拥有生产力应用程序中最简单和最好的任务共享，这对于管理分包商和合作者来说很方便。

### 10.鸟舍

设计师和盗版会有 Photoshop。Mac 用户可以选择奇妙的 [Pixelmator](http://www.pixelmator.com/) 。但是，为了确保您能够快速轻松地编辑图像——我们所有从事数字职业的人至少需要定期调整图像大小和裁剪图像——[鸟舍](http://www.aviary.com/)是一个不错的选择。它可以让你免费编辑图片，截图网站全长等等。

这是击败所有基本应用列表的 10 个基本应用。有很多专业的应用程序没有包括在内，比如面向设计师和摄影师的 Photoshop 和面向 3D 动画师的 Maya，但这些都是在任何地方、任何计算机上运行自由职业者业务的通用工具，而且没有任何麻烦。
