# 最终游戏:除此之外还有什么？

> 原文:[https://www.sitepoint.com/end-game-whats-beyond-this-gig/](https://www.sitepoint.com/end-game-whats-beyond-this-gig/)

有一种观点认为，自由职业是一个不断做和寻找的过程:一旦你完成了一份工作，你就在寻找下一份工作。当然，最理想的是不断寻找新的工作，不管你多么想专注于你现在正在努力完成的工作。

但是不管你对外包工作的方法是什么，在每个项目中都有一个时刻，你会抬起头，意识到终点就在眼前。你很快就会完成这项工作。但是然后呢？

无论是我已经准备好下一个项目，还是我还在谈判下一个演出，我总是在项目结束时尝试完成一些任务，为即将发生的事情做好准备。

## 项目评审和机会搜索

在项目接近尾声时，我开始思考我从中学到了什么。也许我有机会在一个不同的行业工作，做一些我以前没有经历过的事情，或者学习一些团队合作或生产力的新策略。

除了好的方面，我还努力把注意力集中在工作中我会更开心的方面。如果我在项目结束前进行这种回顾，我偶尔会发现仍然有方法来改进我认为不那么热门的领域。我还发现这个项目在我的脑海中仍然很新鲜，所以回忆亮点和不足并不难，并评估我如何改变未来的方法以避免类似的事情发生。

## 日程检查

我喜欢在项目快结束的时候认真看看我的时间表，这样我就知道下个月左右会发生什么。显然，我想为我当前的项目安排任何后续任务，以及即将到来的工作，所以我知道我必须管理什么。

我还发现，项目完成后的那几天可以被我在项目期间推迟的各种零碎事情填满，我喜欢把它们安排在日程中，这样我就不会忘记任何事情。

最后，项目接近尾声的时候通常会提供一个很好的机会来回顾我的周计划，并做出任何一直困扰我的改变。也许下一个项目的时间限制会要求我改变或重新安排一些其他的任务，或者我只是想稍微打乱一下我的日程。现在是时候安排这些计划了。

## 简历或页码更新

不管我是否还在努力寻找我的下一个项目，我都试图找出我当前项目中的哪些材料我想放在我的对开本中。我从痛苦的经历中了解到，事后试图搜集我工作成果的副本可能会非常困难，所以我尝试在进展顺利时收集样本。

我还试着在网上和网下更新我的简历，详细介绍我即将完成的工作，这样我的联系人就能知道我最近在做什么，也能看到我的经历和当前的关注点对他们有什么好处。

## 新项目准备

如果我计划了下一个项目，我会努力为它的开始做准备(至少在心理上)——也许我需要记得去买办公用品，安排与下一个客户的会面，或者做一些背景调查。

当你正处于当前项目的尖端时，试图将注意力集中到下一个项目上可能会很困难和分散，但我发现花一些时间思考我在这一点上需要什么可以让我在下一个项目开始时保持冷静、冷静和镇定，而不是疯狂和准备不足。

## 重调焦距

我认为项目的结束总是让我重新专注于我所做的事情的好时机。所有的项目回顾和简历更新都会让我思考更大的图景，我想做更多的项目，我想和什么类型的客户一起工作。

反思我现在所处的位置，重新关注我想去的地方，反馈到我已经在这里讨论过的许多其他任务中。如果你觉得你的自由职业只是为了得到一些东西——任何东西——让狼远离家门，它还会产生一种方向感和进步感，这可能很难实现。

这些是我喜欢在项目接近尾声时承担的任务。你对项目总结的标准方法是什么？
