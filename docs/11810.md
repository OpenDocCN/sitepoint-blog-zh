# 搜索引擎优化新手的顶级搜索引擎优化书籍

> 原文:[https://www.sitepoint.com/top-seo-books/](https://www.sitepoint.com/top-seo-books/)

两周前，一位评论者问我是否有一些适合新手的 SEO 书籍推荐。我当然知道，我想从一本书开始这个列表，你可以在这里找到，在 SitePoint:

**Brandon Eley 著& Shayne Tilley** 这本书将带领你了解有效的在线营销策略的所有要素，包括搜索引擎优化和社交媒体优化。这本书涉及了公关和新媒体的一些重要方面，它教你如何制定自己的营销计划，以从搜索引擎和其他在线渠道获得最大收益。如果你有一个网站，想推广它，但不知道从哪里开始，这本书是给你的。29.95 美元的价格还包括一份免费的达伦·罗斯所写的【T4 31 天到更好的博客。

另一本优秀的书是亚马逊上的彼得·肯特的《傻瓜搜索引擎优化》 。这本书将教你如何规划搜索引擎优化策略，如何选择正确的关键词，如何使用社交网络来提高你的搜索排名，如何跟踪和衡量你的结果等等。

Jennifer Grappone 和 Gradiva Couzin 的《搜索引擎优化:一天一小时》是另一本很受欢迎的书，解释了为什么 SEO 很重要，并教授如何创建有吸引力的营销网站。此外，作者试图教导如何设计人类可用的网站，这是网站优化的一个非常重要的方面。本书中的许多 SEO 技巧是永恒的:不管算法如何变化，它们都会起作用。

[![](../Images/94457917ea4c0cb821ee09295548e7af.png "Sexy Web Design.")T2】](https://www.sitepoint.com/books/sexy1/)

说到 SEO，你可能应该考虑到，如果网站对用户没有吸引力，为搜索引擎优化是毫无意义的。视觉优化站点和优化文本内容一样重要，这有助于你在 SERPS 中排名。为了理解销售网站设计的基础，我建议你购买由埃利奥特·杰伊·斯托克所著的 SitePoint**[性感网站设计](https://www.sitepoint.com/books/sexy1/)，它将指导你通过应用以用户为中心的设计的永恒原则来创建一个华丽、可用的网站的整个过程。**

谈到页面优化，另一个很好的资源是 **[登陆页面优化:Tim Ash](http://www.amazon.com/gp/product/0470174625)** 的《测试和调整转换的权威指南。这本书教你如何“了解”你的访客，以及如何制定一个能让所有关键人物都接受的行动计划。基本上，这本书回答了一个没有多少网站所有者敢问的问题:你因为糟糕的登陆页面设计损失了多少钱？对于设计糟糕的网站来说，SEO 毫无意义:排名会在那里，但是你的网站不会“卖出去”

与列表中以用户为中心的设计书籍相关，**史蒂文·克鲁格的[不要让我思考:网站可用性的常识方法](http://www.amazon.com/Dont-Make-Me-Think-Usability/dp/0321344758/)** 是任何网站所有者和任何想在未来建立网站的人必须拥有的。事实上，可用性和 SEO 永远不应该分开。

另一本你想要的 SEO 书籍是 aaron Walter 写的 **[构建可找到的网站:网络标准 SEO 和超越](http://www.amazon.com/Building-Findable-Websites-Standards-Beyond/dp/0321526287/)。这本书涵盖的不仅仅是一些 SEO 技巧和诀窍，它还展示了如何建立尊重网络标准的网站，并提供更有吸引力的内容。这本书的主要思想是，你不必牺牲用户体验来创建搜索引擎友好的、可查找的网站。**

最后但同样重要的是，我们都知道好的 SEO 的秘密是好的内容，我想推荐 Mignon Fogarty 的 **[Grammar Girl 的快速和肮脏的技巧，以更好地写作](http://www.amazon.com/Grammar-Girls-Quick-Better-Writing/dp/0805088318/)。这是《纽约时报》的畅销书，也是奥普拉、华尔街日报和其他著名出版物的特色。**

2.  Bing 的 SEO 指南
3.  丹·提斯的《搜索引擎优化快速入门书》
4.  来自 iBusinessPromoter 的免费 SEO 书籍
5.  SEO 精英免费 SEO 书籍

所以，作为感兴趣的人的起点，当然，任何有网站的人都应该这样做，我希望这些信息是有帮助的。在试图回答有时很复杂的问题时，用正确的语气和知识水平来接触每一个读者通常是很困难的。然而，对这些出版物做一些表面的或初步的研究应该对新网站所有者和那些对搜索引擎优化感兴趣的人有所帮助。一如既往地让我知道你的想法，如果你知道一本适合初学者的好的 SEO 书，我没有在这里列出，请在你的评论中提到它。
