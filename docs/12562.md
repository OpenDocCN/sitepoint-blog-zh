# 在线广告价格将大幅下跌？

> 原文:[https://www.sitepoint.com/online-ad-prices-set-to-tumble/](https://www.sitepoint.com/online-ad-prices-set-to-tumble/)

如果你唯一的收入来源是广告，你可能想尽快实现多元化。根据一些关键指标、出版商和分析师的说法，在线广告市场开始走下坡路，而且在好转之前可能会变得更糟。

以下是过去一个月发生的一些事情，这些事情导致在线广告行业越来越不安:

*   搜索引擎上的点击付费广告随着美国股市下跌。YieldBuild.com 首席执行官 Paul Edmonson 告诉 Wired T1，几周前，当股票市场像岩石一样下跌时，主要搜索引擎的广告价格在一天内下降了 9%。自那以后，价格大多有所回升，但仍没有回到以前的高位。埃德蒙森说:“出现大幅下降是非常不寻常的。
*   根据 PubMatic 最新的[广告价格指数](http://pubmatic.com/adpriceindex/index.html)，所有类别的展示广告价格都下降或持平。自 2007 年第四季度以来，价格**下降了近 50%** ，今年每个季度都在下降。
*   [根据 Rubicon Project](http://www.alleyinsider.com/2008/10/ad-rates-dropped-11-in-the-third-quarter) 的数据，从 Q2 到今年第三季度，CPM**的 1，300 家出版商客户平均下降了 11%**。
*   摩根大通分析师伊姆兰·汗在过去的两个月里第二次下调了 T2 的广告增长预期。该分析师预计，2009 年展示广告的增长率为 6%，低于此前预计的 16%，搜索广告明年的增长率将为 17.3%，低于此前预计的 25.5%。
*   博客大亨尼克·丹顿，这位大获成功的 Gawker Media 的幕后人物，看到了一个更加悲观的前景。“任何不准备让广告下降 40%的人都是疯了，”他[上周](http://mediamemo.allthingsd.com/20081103/how-low-will-online-ads-go-lower-says-jp-morgan-very-very-low-says-gawkers-nick-denton/)说。正如《硅谷内幕》的彼得·卡夫卡所说，丹顿总是认为厄运就在眼前。
*   上周，谷歌觉得有必要通过电子邮件向出版商保证，他们正在努力确保广告费率保持不变。许多人认为这是谷歌试图阻止出版商在他们的 AdSense 网络上跳槽。
