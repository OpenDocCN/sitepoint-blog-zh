# 需要设计师，需要作品集

> 原文：<https://www.sitepoint.com/designer-needed-portfolio-required/>

![laptop](img/6167a32c4ff95fb81615efd0b7892cdd.png)在我的文章中，[警告:你将要雇用一个糟糕的设计师或开发人员的 6 个迹象](https://www.sitepoint.com/warning-6-signs-you-are-about-to-hire-a-bad-designer-or-developer)，我的警告之一是设计师或开发人员没有过去工作的例子。虽然我理解后端开发人员可能很难可视化地展示他们的工作(尽管可以提供一些你能做什么的例子)，但网页设计师不应该有这个问题。

我总是很惊讶地发现一个网页设计师没有某种投资组合。当一个匿名的潜在客户试图将你的作品与其他人进行比较以决定雇佣谁时，没有作品集是一个很大的劣势。

你为什么没有作品集？

首先，设计者可能只设计过公司网站或参与过私人项目，可能没有权限展示他们设计的网站或网站的一部分。

第二，网页设计师可能刚刚开始自己的事业。也许他们在学校，没有任何真实世界的经验可以在网上展示。

最后，我想一个不积极找工作的设计师会认为没有必要有一个作品集。虽然在这种情况下，我当然可以提出一个论点，为什么它会节省你的时间，并在以后的工作中维持一个投资组合，因为你永远不知道你的情况何时会改变。

如果你属于这些类别之一，你可能没有投资组合是有道理的。然而，寻找设计师的潜在客户不会考虑你为什么没有作品集。他们只知道你不知道。如果你正在找工作，也许值得你花时间和精力来充实你的投资组合，比如做一些慈善工作，定制你的博客设计，或者创造一些其他工具来展示你的能力。

我能看看你的作品样本吗？

很多时候，你的作品集(和你的网站)是你给潜在客户留下深刻印象的第一次机会。客户在寻找一些非常特别的东西。以下是你的投资组合应该包括的三个必需品:

1.  **作品样本:**这可能是显而易见的，但请记住，你不必将自己局限于完整和活跃的网站。样本可以采取多种形式——链接、截图、互动游览，甚至视频。
2.  概述:除了对你所做的事情给出一个直观的想法，还要提供一个概述来解释你在这个项目中的角色。
3.  联系方式:确保你有一种方式，让潜在客户可以就问题或更多信息与你联系。

另一种给客户更多关于你能做什么的信息的方法是创建一个案例研究，这只是对项目更深入的观察。如果你有一两个你特别引以为豪的项目，并认为能为潜在客户提供有用的信息，这是个好主意。

如果你非常有经验，请记住，你的投资组合不必包括你所做的一切。这意味着是一个概述，突出你最好的，最有创意和独特的项目。查看这些文章，获得一些很棒的投资组合开发技巧:
