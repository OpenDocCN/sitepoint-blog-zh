# 。5 月 1 日至 7 日:Popfly 获得一些新工具

> 原文：<https://www.sitepoint.com/net-on-the-net-may-1-7-popfly-gets-some-new-tools/>

微软一直在忙着想办法解决雅虎的情况，所以几乎没有来自雷德蒙的消息。所以，我借此机会指出微软最近的一些发展，也许能帮到你们。

似乎我认识的每个开发人员，包括我自己，都对下一个伟大的 web 应用程序有一些想法。幸运的是，微软开发领域的一些新事物可能会有所帮助。Popfly 增加了一个游戏创建者，这可能会帮助你的游戏开发者有所突破。[约翰·蒙哥马利](http://blogs.msdn.com/johnmont/)有一些关于新游戏创作者的信息。

如果你真的不确定你想做什么，那么[凯拉](http://cayra.net/)也许可以帮忙。这是一个思维导图应用程序。一个工具，有助于促进头脑风暴，概念图和其他关键任务，为开发您的辉煌的软件想法。虽然 Cayra 不是新的，但它最近引起了轰动，是一个很好的例子。NET 3.0 在行动。

最后，微软对他们的启动中心做了一些很好的补充。该网站为撰写商业计划和规划创业的各个方面提供了一些很好的资源。

在你构建下一个伟大的 web 应用之前，你需要一些技能，所以这里是本周的代码示例和教程。如果你正在寻找一种简单的方法将 Flickr 图片放入你的应用程序，一个新版本的 [LINQ。Flickr](http://www.codeplex.com/LINQFlickr/) 库在 codeplex 上发布。Brad Abrams 写了一个关于利用 SilverLight 的几个[ASP.NET 提供商的坚实教程。最后，来自 MIX 2008](http://blogs.msdn.com/brada/archive/2008/05/03/accessing-the-asp-net-authentication-profile-and-role-service-in-silverlight.aspx) 的 13 个视频在 ASP.NET 发布。

目前就这些；下周请继续关注。
