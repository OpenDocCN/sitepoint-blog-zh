# 微软发布免费防病毒包

> 原文：<https://www.sitepoint.com/microsoft-security-essentials-free-anti-virus/>

杀毒软件是所有 Windows 用户的必需品。如果你相信可怕的广告宣传，那么当你打开电脑的第二秒钟，各种各样的坏事就会发生。那么，为什么大多数反病毒软件设计糟糕、臃肿、过于复杂、占用内存？这些产品是否会让你的电脑运行缓慢，以至于病毒在造成任何损害之前就已经无聊而死了？我心中的愤世嫉俗者认为，商业运营商可能会将他们的软件塞满糟粕，以证明其可笑的昂贵价格。然而，即使许多免费产品也好不到哪里去。

我现在不咆哮了，说重点吧。微软终于发布了他们自己的名为 Security Essentials 的安全软件包。它可以抵御病毒、恶意软件、特洛伊木马、rootkits 和其他恶意软件。

该产品是免费的，任何使用正版 Windows 的人都可以从安全基础网站下载。版本适用于 Windows XP、Vista 和 7。不需要注册，该软件可以安装在任何数量的电脑上，没有任何限制。

Security Essentials 取代了 Windows Live OneCare，后者是一款商业产品，已于 6 月份淘汰。尽管 Security Essentials 是免费的，但它不会随 Windows 一起安装，也不会作为自动更新提供。这可能是明智之举:自动安装可能会与现有的反病毒软件发生冲突。这还会引起反垄断监管机构不必要的关注，尤其是在欧盟。
