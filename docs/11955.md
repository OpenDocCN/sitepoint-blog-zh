# 如何获得和使用证明

> 原文:[https://www.sitepoint.com/how-to-get-use-testimonials/](https://www.sitepoint.com/how-to-get-use-testimonials/)

![testimonials](../Images/72d5e62aeb967a771fa2f9ac3f83e903.png "testimonials")在最近的一篇文章[中，我谈到了客户推荐的价值以及一些从客户那里收集推荐的方法。不过，撰写推荐书只是整个过程的一半。一旦你有了它们，你需要使用它们，并且有效地使用它们。](https://www.sitepoint.com/successful-self-promotion)

以下是一些获得和使用客户推荐的方法。

**获得评价**

获得证明的关键是要求他们。根据我的经验，对你提供的服务感到满意的客户通常更愿意为你提供证明，但你需要主动提出请求。

请求证明的一些格式包括:

*   推荐信——这是一个非常正式的过程，通常在申请加入某个专业组织时使用，作为一个会议的发言人，或另一个结构化的活动。
*   电子邮件推荐——你收到过多少次客户的电子邮件，告诉你他们对你的工作有多满意？这可能是最容易收集的证明格式之一。先征得客户同意使用他们的赞美。
*   音频或视频推荐——潜在客户喜欢看到和听到真人用自己的话讲述他们的经历。如果你的客户愿意记录他们的评价，这是一种非常有效的方式。
*   基于表格的证明——向客户提供标准的证明表格是一种方便证明、请求特定信息并使客户更容易提供反馈的方式。
*   LinkedIn 推荐——通过 LinkedIn 等第三方服务寻求推荐是一种简单的方式，可以让客户有机会在公共论坛上提供反馈(在你审阅后)。

当你展示他们的推荐时，询问你的客户是否可以使用他们的联系信息，或者至少是他们的名字和网址，这也很重要。一份信用明确的推荐信比一份不属于任何人的推荐信更有影响力。

**使用证明**

一旦你从客户那里收集了客户评价，有很多方法可以使用它们。这里有一些想法:

*   网站——在你的网站上创建一个推荐页面，或者有策略地把它们放在你的网站上。如果你有音频和视频的推荐，你也可以使用一个服务来帮助你整合你网站上的媒体文件。为此提供的两项服务是 [AudioAcrobat](http://www.audioacrobat.com) 和[泡泡鉴定](http://www.bubbletestimonial.com)。
*   电子邮件签名/营销–考虑在您的电子邮件签名底部附上一份简短的证明，或者在您发送的任何电子邮件营销邮件中使用。
*   建议书–制作一份推荐书，放在建议书的末尾。
*   营销材料——如果你经常发送宣传册或其他印刷材料，在你的营销材料中加入推荐材料，以鼓励潜在客户选择你。
*   个人动力——当我们需要动力时，我们都有充满挑战的日子。在你的办公室里建立一面激励墙，包括奖状，提醒你明天又是新的一天。
