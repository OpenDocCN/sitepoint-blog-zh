# 2012 年 7 月浏览器趋势:IE9 反击

> 原文:[https://www.sitepoint.com/browser-trends-july-2012/](https://www.sitepoint.com/browser-trends-july-2012/)

2012 年 5 月发生了一件大事，Chrome 取代 IE 成为世界上使用最多的浏览器。上个月的[全球 StatCounter 统计数据](https://gs.statcounter.com/#browser_version-ww-monthly-201206-201206-bar)能与那个历史时刻相匹配吗？不要！…

| 浏览器 | 五月 | 六月 | 变化 | 亲戚 |
| IE 9.0 以上 | 15.59% | 16.55% | +0.96% | +6.20% |
| IE 8.0 | 14.35% | 13.78% | -0.57% | -4.00% |
| IE 7.0 | 1.53% | 1.40% | -0.13% | -8.50% |
| IE 6.0 | 0.66% | 0.57% | -0.09% | -13.60% |

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 13.6%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

上个月，浏览器排行榜几乎没有什么变化。市场经常在年中放缓，但我不得不再次检查这些数字。最大的赢家是 Internet Explorer 9.0，增长了 1%。然而，IE6、7 和 8 的同时下降给微软带来的收益微不足道。

最大的输家是 Firefox。4.0+版自 2011 年 3 月发布以来首次下降。不可否认，它并没有减少太多，但在版本 1、2 和 3 的基础上，Mozilla 浏览器总体损失了 1%。[新标签安全问题](https://www.sitepoint.com/new-tab-thumbnails-expose-private-data/)可能已经产生了一些影响，尽管这个月发生得太晚了，还没有产生重大影响。

在成为无可争议的浏览器冠军之后，Chrome 努力实现 0.3%的增长。虽然许多厂商会对这个结果欣喜若狂，但谷歌一直享受着每月 1%的增长。这是 Chrome 增长平台期的开始吗？有可能——尽管我之前在这一点上一直是错的。

我们现在有一个竞争异常激烈的市场。没有占主导地位的供应商，所以如果 Chrome 保持三分之一的用户，那就太好了。虽然 IE 获得了 95%的市场份额，但微软几乎没有提供基本的网站和服务。如果 Chrome 达到这一点，谷歌将拥有网络。

## 移动浏览器的使用

六月的手机使用率保持稳定，占所有网络活动的 10.4%。许多人预测，移动网络访问最终将超过桌面。我不太相信。它肯定会变得越来越重要，但个人电脑仍然是商业用户最实用的工具。

主要的移动浏览应用程序有:

1.  安卓系统——21.97%(增长 0.91%)
2.  opera Mini/Mobile——21.42%(下降 0.48%)
3.  iPhone——20.99%(增长 1.49%)
4.  诺基亚浏览器——10.99%(下降 0.54%)
5.  UC 浏览器——8.51%(增长 0.04%)

虽然 Android 已经超越 Opera 再次成为第一，但移动浏览器市场几乎没有变化。Opera 或 iPhone 可能会在下个月坐上头把交椅。
