# 2006 年 10 月 11 日新闻专线

> 原文:[https://www.sitepoint.com/oct-11-2006-news-wire/](https://www.sitepoint.com/oct-11-2006-news-wire/)

*   [CSS3 模块:分页媒体](https://www.w3.org/TR/2006/WD-css3-page-20061010/)W3C 发布了 CSS3 的最后一个调用工作草案:分页媒体，增加了分页、边距、大小和方向、页眉和页脚、寡妇和孤儿、图像方向和页码。11 月 3 日前欢迎评论。(标签: [css](http://del.icio.us/sitepointlinks/css) [标准](http://del.icio.us/sitepointlinks/standards) )
*   [CSS in the large](http://szafranek.net/blog/archive/2006/10/10/css_in_the_large/)Krzysztof Szafranek 描绘了他用来管理资产库的方法，包括 CSS 样式表，这些资产在大量站点上共享。通过智能文件夹结构和一些服务器智能，他能够限制重复代码。(标签: [css](http://del.icio.us/sitepointlinks/css) )
*   [MySQL Tools for Visual Studio 1 . 0 . 1](http://devintelligence.com/cs/blogs/netadventures/archive/2006/10/06/1222.aspx)Visual Studio 2005 的免费插件(不支持速成版)，用于管理 Visual Studio 内的 MySQL 数据库结构。尚不支持包括数据集设计器在内的数据驱动应用程序。(标签:[MySQL](http://del.icio.us/sitepointlinks/mysql)T9】。net )
*   [Vista 上的 Java:是的，它工作了](http://weblogs.java.net/blog/chet/archive/2006/10/java_on_vista_y.html)这是对 Java 与 Windows Vista 相关状态的一个很好的总结。简而言之，除了在 Vista 上运行 IE7 时对签名 Java 小程序的一些新限制外，一切都正常。(标签:[Java](http://del.icio.us/sitepointlinks/java)[windows](http://del.icio.us/sitepointlinks/windows))
*   [OpenAjax 更新——解决采用 Ajax 的关键挑战](http://www.coachwei.com/blog/_archives/2006/10/8/2396757.html)OpenAjax 是一项为 Ajax 解决方案开发标准的计划，包括该领域内有影响力的参与者，如 IBM、Dojo、Zimbra 和 Zend。这篇文章报告了小组内部正在进行的工作，以确定需要解决的问题。(标签: [ajax](http://del.icio.us/sitepointlinks/ajax) )
*   这是一个有用的演示，演示了如何使用 Prototype 库的 onDOMReady 扩展向页面添加额外的非语义标记，然后可以使用 CSS 样式化这些标记，以产生圆角等效果。(标签:[CSS](http://del.icio.us/sitepointlinks/css)T9】JavaScript[html](http://del.icio.us/sitepointlinks/html))
*   [Helmi Technologies 发布开源 Java Ajax 平台](http://ajaxian.com/archives/helmi-technologies-releases-open-source-java-ajax-platform)从网站上提供的模糊描述来看，Helmi 开源 RIA 平台是一个类似于 Google Web Toolkit(用 Java 编写 Ajax 应用程序)的解决方案，只是它是开源的。有一个候选版本可供下载，但是细节很少。(标签:[Ajax](http://del.icio.us/sitepointlinks/ajax)[Java](http://del.icio.us/sitepointlinks/java)[open source](http://del.icio.us/sitepointlinks/opensource))
*   IceWeasel预计在 Linux 的下一个 Debian 发行版本中发布，IceWeasel 将是一个没有非自由元素的 Firefox 版本，如 Firefox 徽标和某些默认插件。(标签:[火狐](http://del.icio.us/sitepointlinks/firefox) [开源](http://del.icio.us/sitepointlinks/opensource) )
*   [Subversion 1.4 Gotchas](http://codefeed.com/blog/?p=108)升级到 Subversion 新版本之前值得一读。简而言之，一旦您使用 Subversion 1.4 客户端更新工作副本，使用 1.3 客户端库构建的软件(例如您最喜欢的 IDE)将无法访问该工作副本。(标签:[编程](http://del.icio.us/sitepointlinks/programming) [软件](http://del.icio.us/sitepointlinks/software) [开源](http://del.icio.us/sitepointlinks/opensource) )
*   [谷歌“文档&电子表格”推出](http://www.techcrunch.com/2006/10/10/google-docs-spreadsheets-launches/)谷歌合并并重新推出 Writely 和谷歌电子表格。很遗憾谷歌的名字有这么大的分量，因为我相信竞争对手 ThinkFree Office(http://www.thinkfree.com/)很久以前就有更好的产品了。(标签:[谷歌](http://del.icio.us/sitepointlinks/google) [ajax](http://del.icio.us/sitepointlinks/ajax) )
*   [FileZilla 3.0.0-beta1 已经发布](http://filezilla.sourceforge.net/forum/viewtopic.php?t=2862)领先的免费安全 FTP 客户端正在为其 3.0 版本进行彻底重写，目前已推出 beta 版。用户界面仍然需要大量的工作，但软件现在完全跨平台(Windows，Linux，Mac OS X 和 BSD)。(标签:[软件](http://del.icio.us/sitepointlinks/software) [开源](http://del.icio.us/sitepointlinks/opensource) )
*   [Spring 2.0 最终发布](http://www.springframework.org/go-2.0)首日下载量过万，Spring 2.0 一炮而红。这个新版本包含了许多简化开发的特性(大量关于配置的约定)，以及许多面向 power 开发者的新特性。(标签:[Java](http://del.icio.us/sitepointlinks/java)[opensource](http://del.icio.us/sitepointlinks/opensource)[软件](http://del.icio.us/sitepointlinks/software) )
*   [IE7 这个月就要来了…你准备好了吗？](http://blogs.msdn.com/ie/archive/2006/10/06/IE7-Is-Coming-This-Month_2E002E002E00_Are-you-Ready_3F00_.aspx)就是它了，男孩女孩们。如果你一直在等待一个确定的发布日期来开始更新你的 IE7 网站，你已经得到了。Internet Explorer 7.0 将在 10 月底之前发布(Windows 自动更新将在几周后发布)。(标签:[即](http://del.icio.us/sitepointlinks/ie) )
*   [十问与 YAHOO.ext 作者杰克·斯洛科姆](http://yuiblog.com/blog/2006/10/10/ten-questions-slocum/)雅虎！显示了如何插入雅虎！用户界面(YUI)库团队通过在 YUI 官方博客上采访 Jack Slocum，他是许多广受好评的 YUI 扩展(包括最近在这里介绍的 phpBB 的增强 UI)的作者。(标签: [javascript](http://del.icio.us/sitepointlinks/javascript) [雅虎！](http://del.icio.us/sitepointlinks/yahoo!) )
