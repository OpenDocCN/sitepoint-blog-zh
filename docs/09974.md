# IP v 4.5–互联网地址限制的新解决方案

> 原文:[https://www.sitepoint.com/ip45-ip4-address-limit-solution/](https://www.sitepoint.com/ip45-ip4-address-limit-solution/)

剩下的最后一个 IPv4 地址将于 2011 年 4 月 1 日发放。互联网现在已经爆满，无法满足更多用户的需求。

互联网协议版本 4 是广泛采用的基于标准的寻址系统，网络依赖于此。每台服务器、PC、平板电脑和联网手机都分配有一个唯一的 4 字节地址，允许两台设备相互通信。您可以在[whatsmyip.net](http://whatsmyip.net/)查看自己的 IP 地址。

互联网名称与技术分配机构(ICANT)是负责管理和分配 IPv4 地址的组织。虽然 IPv4 允许超过 40 亿个地址，但地球上有 70 亿人，我们中的许多人拥有不止一个网络设备。IPv6 将最终解决这个问题，因为它提供 3.4 x 10 <sup>38</sup> 个唯一地址。但是部署一直很缓慢——大多数用户仍然依赖 IPv4，并且企业不愿意在经济衰退期间投资新技术。

幸运的是，迈克尔·拉普教授设计了一个新颖的解决方案。IPv4.5 今天发布，将分时分配地址。拉普进一步解释道:

> IPv4.5 依赖于地球自转和网络使用模式。每个 IP 地址将被分配给地球上两极相对的两个用户。如果你在当地时间下午两点上网，你将比世界另一端凌晨两点上网的人有优先权。
> 
> 因此，一个美国用户很可能会与中国的某个人共享他们唯一的知识产权。实际上，这将使 IPv4 地址空间翻倍，而不需要昂贵的新硬件或系统升级。

SitePoint 很幸运地获得了采访 Mike 的机会。

站点点:IPv4.5 将如何工作？

![IPv4.5 diagram](../Images/e5fe78f22093c5da53ab21d094de5aac.png)

MR:系统的核心是一个地址优先服务器，它将数据包发送到正确的设备。如果两个设备都在使用中，服务器可能会向两个设备发送数据，接收应用程序或 web 浏览器将确定是否需要该数据。错误被返回，因此我们的系统可以适应不断变化的流量模式，并随着时间的推移变得更加准确。

IPv4.5 会限制带宽吗？

有一点，但是只有当你晚上上网的时候才会感觉到。影响将被最小化，因为这些用户通常在玩游戏，阅读脸书，在推特上谈论失眠或沉迷于其他无聊的活动。

SP:这种解决方案会带来安全或隐私风险吗？

MR:数据包将不可避免地被分发到错误的设备。然而，有人以他们能够理解的语言接收数据的风险是可以忽略的。

英国和澳大利亚说英语的人呢？

MR:少量的地理异常现象肯定会出现。幸运的是，文化差异将克服语言相似性:双方都不太可能理解对方。

或者，这个问题可能有政治解决办法。例如，英国居民可以采用法语作为他们的母语。或者澳大利亚可以完全由英国管理，这样安全问题就无关紧要了。这是一个双赢的局面——每个人都会很高兴。

世界上的某些地方会有地理优势吗？

马:是的。几个国家的两极都是海洋，因此他们不会体验到 IPv4.5 的分时效应。因此，许多基于网络的公司将员工转移到北非和中东。

IPv4.5 会取代 IPv6 吗？

MR:不是——这是 IPv6 被广泛采用之前的一个临时解决方案。也就是说，我们今天开始实施，我们都知道临时 IT 解决方案的持续时间通常远远超过预期…
