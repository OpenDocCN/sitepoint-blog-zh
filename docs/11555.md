# 竞争对手是否在 AdWords 中劫持你的商标？

> 原文：<https://www.sitepoint.com/competitors-hijack-trademark-keywords/>

Google AdWords、Yahoo Search Marketing 和 Microsoft adCenter 为使用和运营这些服务的公司带来了利润。尽管每个人都希望自己的网站出现在自然搜索引擎结果的显著位置，但是 <acronym title="Search Engine Optimization">SEO</acronym> 并不是一门精确的科学。为赞助列表中的广告付费会更便宜、更容易、更快捷。

大多数 AdWords 类系统都基于相同的原理运行:

1.  你选择一组关键词和短语。
2.  你创建一个或多个广告，通常是纯文本的。
3.  当用户点击你的广告并进入你的网站时，你选择支付一定的费用。

你付的钱越多，你在赞助商列表中的排名就越高。

除了最低收费外，你可以选择的关键字没有任何限制。如果你经营一家名为 Foozle Software 的公司，当输入关键字“Foozle”时，你的任何竞争对手都可以看到他们的广告。

奢侈时装设计师路易·威登揭示了这种情况。该公司声称，根据欧洲法律，谷歌出售商品名称的做法是非法的，还允许假冒交易商出售假货。2006 年，巴黎一家法院裁定谷歌因商标假冒、不公平竞争和误导性广告支付 25 万美元。谷歌已经向欧洲法院提起上诉，并可能在一名顾问最近的声明中获胜:

> 谷歌允许广告客户选择与商标相对应的关键词，并没有构成商标侵权。

商标劫持是一个主要问题吗？在 meta 标签中使用竞争对手的名字也引发了类似的问题。事实仍然是，除了诽谤法，没有什么可以阻止任何人创建一个提到你的品牌名称的网站。搜索引擎应该保持公正；如果该内容与搜索短语相关，则必须允许他们链接到该内容。然而，也许 AdWords 是不公正的，因为它偏袒最大的消费者？

不幸的是，搜索引擎竞争变得如此重要，我们可能会看到许多大企业诉诸诉讼。我怀疑许多律师都在暗地里希望路易威登赢得官司。
