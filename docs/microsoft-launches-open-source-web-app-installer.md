# 微软推出开源 Web 应用安装程序

> 原文：<https://www.sitepoint.com/microsoft-launches-open-source-web-app-installer/>

几周前，微软发布了他们的[网络平台安装程序](http://www.microsoft.com/web/channel/products/WebPlatformInstaller.aspx) (Web PI)，这是一个免费下载的应用程序，可以安装和配置他们的整个网络平台堆栈。该应用程序安装最新版本的。NET 框架 3.5 SP1，IIS7.0 和扩展，IIS FastCGI，Visual Web Developer 2008，SQL Server 2008 Express，SQL Server Management Studio，ASP.NET MVC 和 Silverlight 工具。这个想法是，Web PI 使开发人员更容易安装整个 Microsoft web 堆栈，然后保持最新版本。

但是没有应用程序的网络平台有什么用呢？因此，本周微软推出了 [Web 应用安装程序](http://www.microsoft.com/web/channel/products/WebApplicationInstaller.aspx)，这是一个配套工具，可以自动安装和配置大量广泛使用的开源和社区 Web 应用。该应用程序最初将安装 Graffiti、DotNetNuke、WordPress、Drupal、OSCommerce 和 phpBB。它直接从供应商那里下载应用程序的最新版本。

据 Windows server 团队的 David Lowe 称，未来将会有更多的应用加入到 Web AI 应用中。

在某些方面，Web AI 让我想起了 cPanel 安装程序 [Fantastico](http://netenberg.com/fantastico.php) 。微软已经把安装和配置应用程序的繁重工作从他们的 web 平台栈中去掉，并且使得像安装 phpBB 或设置 WordPress 这样的事情只需一次点击。

正如 Matt Asay 指出的那样，Web AI 并没有让微软成为这些开源应用的经销商，因为它们是从公共源代码库中下载的，但这已经很接近了。这背离了微软过去对开源软件的立场。
