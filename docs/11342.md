# 成为健康工作狂的实用指南

> 原文:[https://www.sitepoint.com/healthy-workaholic-or-driven/](https://www.sitepoint.com/healthy-workaholic-or-driven/)

“工作狂”这个词有负面含义，因为它意味着强迫行为和对工作的沉迷，但我认为这个词可以有两种说法。你可能是一个工作狂，失去了所有的观点，只关注工作，而损害了你生活的所有其他方面。或者，你可以成为一个工作狂，将你的动力和奉献转化为对事业的健康投入。

不管你多么热爱你的工作，可以肯定地说，你可能更喜欢属于第二类。不幸的是，这并不总是那么容易。诀窍在于找出这两个极端之间的区别，并采取具体行动来保持平衡的工作生活，成为一个更忠诚的专业人士，而不是一个痴迷的员工。

## 你是工作狂吗？

首先，让我们通过考虑这些问题来弄清楚你是否可以被认为是工作狂:

*   你是否经常长时间工作，包括周末和假期？
*   你是否为了事业牺牲了个人生活的某些方面？
*   当你发现自己被黑莓束缚住时，你会犹豫要不要去度长假吗？
*   当你不在电脑前时，你会考虑工作吗？
*   你是不是经常因为工作而错过个人大事(然后因为工作而有负罪感)？

如果你对以上问题中的任何一个回答是肯定的，你可能至少是一个边缘工作狂。但是，即使你有时会陷入视野狭窄的工作狂类别，好消息是你可以改变这种情况。所需要的只是视角的改变和优先次序的调整。

## 努力工作与被驱动

“工作狂”这个词的负面含义经常被不准确地用来描述那些仅仅热爱他们所做的工作并且喜欢工作胜过其他活动的人。全心投入工作并乐在其中并没有错。我们中的一些人只是有一种内在的驱动力，使我们经常工作，努力工作，并愿意为实现目标做出牺牲。这可以是好的；这通常会带来成功和成就。

然而，当你变得如此以工作为导向而失去远见，以至于让你的动力支配了你的整个生活，难以进行必要的休息，你的奉献开始变成过度关注时，问题就出现了。

## 如何实现工作狂的平衡

可以说真正的工作生活平衡是无法实现的，尤其是对于工作狂来说。但是，如果你把你对工作的专注和奉献应用到你对生活各方面平衡的渴望中，你可能会惊讶地发现，发展一种以你和你周围的人的幸福为中心的全面的生活方式是多么容易。

以下是开始的方法:

*   写下你的动机——你为什么这么努力？你的工作在个人层面上对你有什么帮助？一天下来，你的工作让你感觉如何？成功对你意味着什么？
*   **诚实地分析你的动机**——你的工作是否弥补了你生活中其他方面的不足？你对自己的动机满意吗？你的动机支持你的目标吗？
*   **描述你的完美平衡**——典型的一天会是什么样子？你目前的工作时间表会有什么变化？你的首要任务是什么？
*   **列出你现在能做的事情**——你能在生活中立即做出哪些改变，让你更接近完美的平衡？为了忠实于你的优先事项，你需要改变什么？
*   **规划未来**——你可以在生活中努力实现哪些长期的改变？你将如何每天提醒自己什么是你的优先事项？

就像生活中的其他事情一样，在工作中适度是至关重要的。但是没有理由为什么你不能成为一个快乐、健康、正常工作的“工作狂”
