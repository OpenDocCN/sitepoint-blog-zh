# 你的网络表单很烂吗？

> 原文：<https://www.sitepoint.com/do-your-web-forms-suck/>

坦率地说，大多数网页形式都是如此，不管是因为可用性差，还是缺乏美学价值，或者两者兼而有之。幸运的是，你们大多数人都意识到了这一点并寻求帮助——使用 CSS 的奇特表单设计的流行就证明了这一点，这是我们有史以来最成功的教程之一。

然而，本教程只能涵盖这么多，所以大约六个月前，我们联系了一些顶尖的互联网重量级人物——德里克·费瑟斯通、纪娜·波顿和蒂姆·康奈尔——与我们分享他们的见解。

如果你在过去的一年里参加过任何网络会议，你可能已经听过这些人在诸如 Web Directions、dConstruct、Event Apart 和 Webmaster Jam 这样的活动中谈论网页设计——这三个人知道什么是花式！

我们现在已经将他们所有的知识和最佳实践建议汇编成一本实用的全彩书籍，从头到尾教你如何创建视觉上令人惊叹的可用 web 表单。

用 *[花式形态设计](https://www.sitepoint.com/books/forms1/)* ，你会:

*   学习网格设计和排版的基础知识
*   了解如何有效地使用颜色和图像
*   了解如何构建表单以获得最大的可用性
*   获得编写有用的说明、错误和建议文本的技巧，以消除用户的挫折感并避免放弃
*   掌握如何使用 CSS 有效地设计 web 表单的样式
*   掌握使用 JavaScript 更新表单的技巧——比如创建带有条件元素、密码强度指示器和自动完成功能的“多路径”表单，以及捕获日期和月份信息
