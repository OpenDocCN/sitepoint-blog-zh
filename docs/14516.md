# 你的客户的安全，红帽和更多

> 原文：<https://www.sitepoint.com/your-clients-security-red-hat-and-more/>

个人安全
Corrado Cau 的一篇很棒的文章出现在[开源开发网络](http://www.itmanagersjournal.com/management/04/06/22/1842232.shtml)上，这篇文章提供了一个简单易懂的入门读物，web 开发人员可以将它分发给他们的客户。

Cau 提出的一个重要观点是，虽然在线安全在保护电子商务方面得到了充分证明，但用户仔细检查他们寻求交易的网站的隐私政策，以确保有关他们的机密数据不会以除保护信用卡和相关信息之外的其他方式共享，这仍然是至关重要的。

Red Hat Certified Architect
Red Hat[扩展了其培训课程中提供的 Linux 认证](http://www.redhat.com/training/architect/courses/)的级别，以包括大型企业组织的架构。这个新的认证，RHCA，是对红帽认证技术人员和红帽认证工程师的补充。

这可以通过扩大一个人可以吸引的客户的规模来扩展 web developer 的能力，除了 web 设计和开发工作之外，还包括需要复杂托管环境的大型公司。

Mac 之道
我在葡萄牙的一位同事是欧洲一家 GSM 运营商的无线分析师，同时也经营着我最喜欢的技术博客之一，[Mac 之道](http://the.taoofmac.com/space/)。芮的报道当然有很强的苹果主题，但也包括了设计者和开发者感兴趣的开源和相关技术的重要报道。查看他的 [Linux 部分](http://the.taoofmac.com/space/Linux)。
