# 撰写销售提案

> 原文:[https://www.sitepoint.com/write-proposals-sell/](https://www.sitepoint.com/write-proposals-sell/)

在今天的市场上，如果你写不出好的提案，你就卖不出去。让我们面对现实吧:就自由职业服务而言，我们处于买方市场。对于你竞标的每一份工作，你都在和五个、十个甚至更多的自由职业者竞争。我最近发布了一份网页设计合同的招标书，收到了 65 位不同设计师的询盘！

当你被困在这样的人群中时，你必须找到一种方法让自己与众不同，或者面对 20:1(甚至更长！赢得那份合同的可能性。正如我在[过去的专栏](https://www.sitepoint.com/blog/)中所描述的，让自己脱颖而出的最好方法是在推销你的业务之前与潜在客户建立个人关系，但写一份好的建议书紧随其后。即使你们之前有关系，一个好的提议也有助于增强你的交付能力。如果你是牲口圈的一员，一个好的提议会把你推到最前面。

如果写作的想法让你惶恐不安(还会想起那篇关于莫比·迪克的该死的论文，你永远也完成不了)，这里有五个简单的步骤，你可以按照它们来写销售建议书。

1.  偷一个可行的提议
2.  理解(并奉承)你的目标
3.  首先是目的，然后是手段
4.  使用简单的单词
5.  价格合适(这是一个买方市场)

现在，让我们来看看这些方法，看看您如何应用它们来让您的下一个提案成为赢家！

##### 1.偷一个可行的提议

写一份好提案的最好方法是从一份已经写好的提案开始！与你的学生时代不同，商业世界是建立在剽窃之上的。下次您从某人那里收到标准文档时，请使用 Word 的“属性”命令来检查在标题、作者和公司字段下列出的姓名。很有可能，一个、两个或所有三个都与给您发送文档的人不同。检查我在其中输入本专栏的文档发现，我从去年发表的一篇文章中窃取了这个模板。

如果你找到一个有用的文档，根据你自己的目的修改它。我的提议借用了几份文件；在我的上一个提案中，我从几个来源中提取了最好的想法，并应用它们来创建这个结构:

1.  背景
2.  情况
3.  项目规格
4.  项目定价和投资回报(ROI)
5.  行动计划
6.  提案接受

当我从零开始写提案文档时，我从我妻子的市场调查提案中借鉴了第 1、3、4 和 6 部分的风格，第 2 和 5 部分来自我作为自由职业者时写的一些提案。虽然你不应该用千篇一律的方式扼杀提案，但你可以也应该尽可能地重复使用成功的成分。

##### 2.理解(并奉承)你的目标

在介绍您的提案时，您可以做的最重要的事情是展示您了解潜在客户的需求和愿望。这用不了多长时间；通常一个快速的谷歌搜索就足以提供你客户的背景信息，你可以把这些信息包含在你的介绍中。

你可能会问，“为什么要告诉客户他们已经知道的东西？”请这样想:如果你不能展示出你了解客户业务的基础，他们肯定不会把一个关键任务项目交给你。

另一件要记住的事情是，奉承很少会造成伤害。如果你的潜在客户自称为该领域的“领导者”，在你的提案中反驳这种说法可能不是一个好主意。相反，承认这种说法，但在与客户打交道时，要记住自己的独立观点。

下面是我如何在我的成功提案中应用这一原则:

*“客户 X 为全球 2000 强品牌营销商及其代理开发了一个战略推广平台。目前，这个平台提供三个推广引擎:锦标赛、优先拍卖和收藏品。客户 X 使用这个平台及其“机制设计”的概念来帮助客户建立消费者互动，将消费者动机和品牌目标联系在一起。*

客户 X 需要开发一套强大的参考账户和一个易于“转移”的客户价值主张，以允许相对较小的公司(22 名员工)使用杠杆销售模式。"

描述我的客户的大部分语言都是从它自己的网站上转述的，但我确保自己清楚地理解销售副总裁想要什么，以及为什么。

##### 3.首先是目的，然后是手段

在你深入研究如何执行项目的细节之前，确保你清楚地陈述了具体的最终目标。这使得客户更容易将您的建议与其他供应商的建议进行比较，并防止以后出现误解。如果我正在阅读大量的提案，我最不想做的事情就是在等待供应商开门见山的时候费力地阅读几页实现细节。

在我上一次成功的提案中，我决定提前陈述目标:

*“客户 X 希望 YZ 咨询公司帮助客户 X 建立推荐账户并发展代理关系。YZ 咨询公司将从客户 x 提供的目标客户名单中产生合格的潜在客户。*

客户 X 已经锁定了 100 家代理商作为渠道合作伙伴。客户 X 希望 YZ Consulting 设计并实施一个销售线索挖掘和鉴定计划，该计划将在三个月内挖掘出 15-25 个合格的销售线索。合格的销售线索是目标机构的客户总监或同等职位，同意与客户 X 会面，并表明该机构有一个客户有兴趣进行客户 X 平台的试用。

*考虑到客户 X 的人员限制，YZ 咨询公司将确定、培训和管理进行潜在客户开发和资格认证所需的资源。YZ 咨询公司会将合格的销售线索交给客户 X，以实际完成销售。我们的提案假设所有目标客户都在美国。"*

如您所见，我清楚地指定并量化了项目的可交付成果和时间框架。一个关键因素是定义“合格的领导”这一术语，这样我们就可以在明确的指标基础上构建我们的绩效薪酬提案。不要忘记指定你参与过程的终点——否则，当你的客户要求你在不调整合同条款的情况下做越来越多的工作时，你可能会经历“功能蠕变”的恐怖。

##### 4.使用简单的单词

虽然提案不是高雅的文学作品，但这并不意味着它们不受优秀作品法则的约束。如果你努力做到尽可能简洁明了，采取主动而不是被动的语气，使用简单的语言，你会让你潜在客户的工作变得更容易，不那么令人恼火——这两个因素都有助于将你的提议推到最前面。不管你从事何种自由职业，这条规则都适用；即使对于平面设计师来说，一份写得好的提案(有强大的图形示例支持)也比一份写得差的提案更有可能卖出去。

例如，考虑如何描述客户 X 的情况:

*“客户 X 的销售部门按以下方式组织。这个部门的负责人是销售副总裁约翰·多伊。在他下面的是一名销售顾问，还有营销总监兼首席执行官简·多伊。到目前为止，客户 X 采用的销售流程是利用管理团队的个人联系以及投资者的联系来确定可能的销售机会。销售副总裁有责任抓住这些机会。在这个时间点上，客户 X 已经实现了一位数的销售额。"*

现在看看如果我们用简单明了的语言描述会发生什么:

*“客户 X 有一名全职销售人员——销售副总裁 John Doe——一名销售顾问，以及另外两名贡献者(营销总监兼首席执行官 Jane Doe)。迄今为止，客户 X 已经使用个人和投资者联系的组合来产生销售线索，并且已经销售了个位数的账户。”*

虽然两个例子说的完全一样，但哪个更有可能吸引客户？

##### 5.价格合适(这是一个买方市场)

最后，尽管这似乎是显而易见的，但一定要现实地为你的提议定价。这是一个服务的买方市场，以一种既能反映服务价值又能反映你的竞争格局的方式为你的服务定价是至关重要的。

这就是和你的潜在客户建立良好关系的好处。询问客户他们愿意支付的范围；如果太低，你就知道不要浪费时间写提案；如果价格太高(这是我们都想遇到的问题)，这是你尽快提出合理建议的提示(在客户与你的竞争对手谈之前！).

我与之交谈的一位供应商与我讨论了她通常的价格，当她意识到我的预算是她通常价格的 1/3 时，她修改了她的建议。如果她只是提出一个标准的建议，我可能甚至不会回复她，认为价格差距太大，无法克服。

就我的客户而言，该公司需要在不增加员工的情况下，为其销售团队创造高水平、高质量的销售线索。我们在预聘加激励的基础上为项目定价，允许我们支付固定成本，但需要成功才能盈利。促成这笔交易的是我们成功奖金的性质:

*“XY Consulting 每产生一个合格的销售线索，将获得 ABC 美元的成功奖金。客户 X 判断其认为销售线索是否合格。”*

通过做出重大让步——允许客户判断我们何时有资格获得成功奖金——我们显著降低了客户的风险，并使他们很容易签署建议书。在实践中，我们从未遇到过客户试图欺骗我们的问题；有结果就很开心了。大多数买家并没有试图欺骗你；他们只是害怕你会让他们失望。放弃一些筹码可能是达成交易的最佳方式。

写一份出色的提案并不能保证销售。你仍然需要跟进，讨好你的客户，完成交易。但在这个市场上，提交一份写得很差的建议书肯定会让你丢掉生意。

##### 你下一个提议的要点

如果你正在拖延你的下一个提议，这五点应该可以帮助你开始。

记住:抄袭是可以的！从其他文档中汲取灵感，创建一个符合潜在客户期望的合理结构。你如何知道这些期望？通过研究——你需要这个来表达你的同情心，奉承你的客户！

预先陈述项目的目标会让你的客户更容易评估你的提案，并诱使他们一直读到最后。注意使用清晰、简洁的语言，并确保你的价格适合潜在客户的预算。

遵循这些指示，你将离赢得工作更近五步！祝你好运。
