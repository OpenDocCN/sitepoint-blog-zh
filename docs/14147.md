# 奥赖利推出 CodeZoo

> 原文:[https://www.sitepoint.com/oreilly-launches-codezoo/](https://www.sitepoint.com/oreilly-launches-codezoo/)

O'Reilly 刚刚发布了一个新的网站 [CodeZoo](http://www.codezoo.net/) ，这个网站对自由软件组件进行分类、分发、评级和提供建议。该网站已经推出了一个免费 Java 组件目录，但 O'Reilly 正在寻求反馈，以了解应该推出哪些其他语言/平台。

该网站的访问者可以提交任何列出的组件的提示和评论，或者建议由 O'Reilly 工作人员审查的组件纳入目录。该网站为其目录中的每个组件提供了 [Atom](http://www.atomenabled.org/) 提要，这样您就可以跟上社区发布的技巧和新版本。

我很有兴趣看看这个网站如何发展。目前，它有点乏味，但如果 O'Reilly 通过设置高编辑标准和提供更多方法来保持最新的自由软件组件，成功吸引了一个体面的社区，它可能是一个赢家。
