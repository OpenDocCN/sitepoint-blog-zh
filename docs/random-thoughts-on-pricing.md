# 关于定价的随想

> 原文：<https://www.sitepoint.com/random-thoughts-on-pricing/>

本周的主题是定价。不知道为什么。以下是一些随机观察:

1.涨价会给你带来奇迹。我刚刚在我的时事通讯列表上宣布了一个在 AttractNewClients 网站上的大涨价(100 美元)(基于测试)，并邀请他们在价格上涨之前加入。反响非常好，比我通常的时事通讯提供的要多 3 倍。有期限的提价会导致持观望态度的人采取行动。

1a。我的一个同事最近也做了同样的事情。他表示，从所有新客户开始，他将把费率从每小时 50 美元提高到每小时 75 美元。直到 11 月，老客户仍然可以享受 50 美元的优惠。他变得很忙，很快。

2.要价高可以为你创造奇迹。我的一个同事分享说，一家目录公司请他投标开发一个网站。他以前和这家公司合作过，知道他们很难共事。所以他出价很高。真的很高。高到足以让他觉得这份工作物有所值。他得到了这份工作！

3.不要成为低自尊定价的受害者。作为一名专业人士，你不想显得很穷。相反，传达你的价值，这样定价就不会成为一个(主要)问题。当然，定价总是在后台。但是如果你展示了你的价值，并且你想要这份工作——但是不需要——你可以要求更高的价格。

4.这是一个新的例子:我给一个客户开了一个低价，比如说 500 美元，让他为他们写一份快速战略计划。我这样做是为了向他们展示我能做什么，并为即将到来的更大的项目定位自己。公司非常喜欢我的工作，坚持要付给我 1000 美元。这在以前从未发生过。但这多少与上面的第三点有关:一旦你确立了自己作为一名有价值的顾问的地位，价格自然会上涨。

对这些评论表示感谢！
