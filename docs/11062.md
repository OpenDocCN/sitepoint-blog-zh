# 什么时候可以使用 Technology X？

> 原文：<https://www.sitepoint.com/when-can-i-use-technology-x/>

SitePoint 提供了一个很好的参考工具。它提供了大量的 [HTML](https://reference.sitepoint.com/html) 、 [CSS](https://reference.sitepoint.com/css) 和 [JavaScript](https://reference.sitepoint.com/javascript) 解释以及相关的浏览器支持级别。

然而，我最近发现[我什么时候可以使用……](http://a.deveria.com/caniuse/)——一个提供不同的浏览器兼容性的工具。它列出了一系列现代网络技术，并报告了它们在 IE、Firefox、Safari、Chrome 和 Opera 中是否受支持。您可以配置需要支持哪些浏览器以及实现的历史水平。

正是这最后一个特点使得这款工具独一无二。最好用一个简单的例子来说明:假设您正在考虑为您的下一个网站或应用程序使用 PNG alpha 透明度。该工具将高兴地报告 png 可以毫无问题地用于过去、现在和未来的所有浏览器。然而，当你选择“很久以前”时，你会被警告 IE6 不提供完全透明，你需要等到浏览器过期。在这种特殊情况下，该工具还提供了一个有用的链接，指向一篇介绍 IE6 PNG 变通方法的文章。

除了 PNG，该工具还提供了包括 HTML5、CSS2、CSS3、DOM、SVG 和 Canvas 在内的类别。您选择的所有功能的摘要可在报告末尾找到。

[什么时候可以用…](http://a.deveria.com/caniuse/) 由 [Alexis Deveria](http://a.deveria.com/) 开发:

> 我找不到一种易于阅读、全面和互动的方式来查看哪个浏览器支持哪个即将推出的功能…所以我做了一个。

Alexis 的彻底性令人印象深刻，他不断地用最新的浏览器实现细节更新该工具。关于最近发布的 [Opera 10.5](https://www.sitepoint.com/opera-best-browser-ever/) 甚至 [IE9 预览版](https://www.sitepoint.com/microsoft-reveal-ie9/)的信息已经有了。

很棒的东西——当你的老板要求知道为什么文本阴影在 IE8、Firefox 3.0 和 Safari 3.1 中不起作用时，非常有用！
