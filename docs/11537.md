# 穿另一只鞋:了解你的客户是怎么想的

> 原文：<https://www.sitepoint.com/understanding-how-your-clients-think/>

商业成功不可或缺的一部分是能够将自己置于客户的位置。从市场营销的角度来看，这种理解不仅是必要的，而且有助于你预测客户需求，进行持续的自我评估，从而改善和发展你的业务。

通过了解和解决客户的需求，您将能够:

*   加强与客户的关系
*   确保客户满意
*   成为您客户有价值的(不可替代的)合作伙伴
*   创造更令人满意的工作环境
*   让你自己与现有客户一起寻找新的机会，增加推荐的可能性

## 了解你的客户

即使你认为你很了解你的客户来自哪里，也总有改进的空间和更好地了解他们的方法。首先，考虑以下问题。

*   他们为什么雇佣你？
*   您的服务解决了什么问题、挑战或需求？
*   他们的商业目标是什么，你扮演什么角色？
*   他们的客户是谁？
*   他们想从和你的关系中得到什么？

## 如何加强与客户的关系

就像所有的关系一样，当基本需求得到满足时，商业关系就会蓬勃发展。通过在五个主要方面保持一致，你可以确保满足客户最基本的需求。

**做伟大的工作**

在建立人际关系时，你的工作质量应该堪称典范，这是显而易见的。如果你不能让你的客户开心，那么建立长久的关系几乎是不可能的。

**定期沟通**

在所有关系中，持续的沟通都很重要。如果合适的话，你可以通过回复电话和电子邮件、安排定期检查、分享公司新闻以及通过社交媒体与客户互动来做到这一点。

**寻求反馈**

在单个项目或达到里程碑后，询问你的客户对事情进展的看法。你可以采取正式的方法，进行客户满意度调查，或者在交谈中非正式地询问他们。然而，获得客户反馈最重要的一步是制定一个计划来解决任何问题或批评，并致力于改善情况。

**依靠你的专业知识**

很多时候，你的客户会欢迎并欣赏关于如何做得更好或更有效的建议。利用你过去的经验和对工作的深入了解，帮助你的客户开发超越他们最初期望的解决方案。

**培养伙伴关系**

通过将每个客户关系视为一种持续的伙伴关系，您可以将这种关系转变为一种合作、互利的关系。这种专注可以让你更成功地建立一种可持续的关系，而不是简单地完成工作然后继续前进。
