# 成为一个专业的演员！

> 原文：<https://www.sitepoint.com/become-professional-act-one/>

在著名的《纽约客》漫画中，一幅狗使用电脑的图片下面的说明写道:“在互联网上，没有人知道你是一只狗”。这幅漫画完美地捕捉到了互联网的可能性——成为你想成为的任何人。如果你想在网上销售任何东西，不管是你自己的产品、服务还是广告空间，看起来专业是必须的。毕竟，如果你在网上的表现让你看起来像是在经营一家二流公司，却不知道自己在做什么，那么没人会给你一分钱。遵循以下三个步骤来大大改善你的网络形象。

##### 1.)获得一个域名

没有什么比一个 URL 读起来像这样更糟糕的了:http://www.geotrash.com/carpal/tunnelSyndrome/2348/或 http://members.aol.com/~username234.当然，如果你只是在尝试，或者如果你从没想过从你的网站赚一分钱，免费托管是很棒的。但是，如果你真的想在网上赚钱，一个好域名是必须的。这意味着避免缩写。准备好花一些时间寻找合适的东西，尽管……甚至像 bbbbbbbbbb.com 这样的域名也被采用了。根据过去的经验，我还可以告诉你，注册域名的不同变体和常见拼写错误也是一项不错的投资。

**一个真实的故事:**我曾经在拥有 850，000 订户的 WINDOWS 杂志上写过一篇关于 WebmasterBase(以前称为网站管理员资源)的一页文章。报道很棒，他们甚至有网站的截图。唯一的问题是，他们公布的网址是 WebmasterResources.com，而不是 Webmaster-Resources.com。另一个域名是成人网站管理员资源网站的主页。我本可以在创建网站的时候注册这个域名，但是我太吝啬了，没有再花 70 美元。几个月后，我的错误追上了我，由于这个愚蠢的错误，我最终损失了价值几千美元的免费广告。

##### 2.)不要使用免费的电子邮件地址

当然，来自雅虎和 Hotmail 的电子邮件地址是一个很好的方法，可以从任何一台有互联网连接的电脑上获取你的电子邮件。但是他们也在网上给了所有者一个坏的形象。如果您必须在这些电子邮件服务中有一个电子邮件地址，请仅将其用于与您的朋友、家人等交流。不要用它给你想发展关系的公司/其他网站的所有者发送电子邮件，也不要在你的网站上使用那个电子邮件地址。

还有一件事，如果你使用美国在线，使用一个替代的电子邮件地址，如 you@yourdomain.com。许多更有经验的互联网用户倾向于给 AOL 用户贴上一无所知的新手标签，他们根本不应该上互联网。这并不总是一个公平的判断，但它确实有一定的正确性。庞大的 AOL 用户群主要由互联网和计算机新手组成。他们中的许多人不懂网络礼仪(即。用大写字母打字就像大喊大叫)，因此他们经常犯错误，这让他们在互联网社区中看起来缺乏经验。

##### 3.)看起来你很大

在互联网上，花一点时间和经验，你可以让你的网站看起来比你最大的竞争对手更好。如果你没有很多设计技能，花 500 到 1000 美元让一个有经验的设计师为你创建一个可以用于整个网站的模板是一个很好的主意。在这个网站的设计文章部分，有大量的信息、想法和指导方针可以帮助你击败竞争对手。
