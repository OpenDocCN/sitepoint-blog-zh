# 案例研究:网页设计公司能带来的改变

> 原文：<https://www.sitepoint.com/case-study-the-difference-a-web-design-firm-can-make/>

上周，我们在怀俄明州/爱达荷州度假，寻找一处夏季木屋。当地的房地产市场提供了一个很好的例子，说明了网页设计公司可以带来的不同。

在我们旅行之前，我通过谷歌和网络联系了一些房地产经纪人，询问有关房产的情况。最终，一位房地产经纪人非常积极地回应了。她给我发了很好的电子邮件，列出了一长串符合我们标准的房产，还建议我去哪里找。她看起来很有资格，也很专业。她的网站很专业，在搜索中即使不是第一，也很早就出现了，所以我猜想她是一个市场领导者。

当我们到达怀俄明州和她见面时，她仍然很有资格和专业，她帮助我们找到了一处房产。但事实是:她在市场上相对默默无闻。事实上，两家房地产公司控制了我们寻找房产的那个县。他们有所有的清单，而且已经放了几十年了。与此同时，“我们的”房地产经纪人刚刚在该地区开设了一个办事处。

但是我们找到她要感谢一个优秀的网页设计师/搜索引擎优化专家，他把她的网址放在搜索引擎的顶部，并帮助她创建了一个满足我需求的网站。

技术并不是故事的全部，因为她非常敏感、专业等。

如果我是她公司的网页设计师，我会到处宣传这个案例研究:尽管竞争由来已久，房地产经纪人如何打入新市场。在类似的情况下，我会关注全国范围内的中小型房地产公司。

这是你应该为你的客户得到的结果，也是你应该在营销信息中包含的解决方案。

正如这个博客试图强调的，你的工作是帮助你的客户成功；网页设计和开发仅仅是一个工具。
