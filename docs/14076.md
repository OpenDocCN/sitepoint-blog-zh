# 你有多容易接近？

> 原文:[https://www.sitepoint.com/how-accessible-are-you/](https://www.sitepoint.com/how-accessible-are-you/)

在销售服务时，根据人们对您的价值提供不同程度的接触是很重要的。

给你最好的客户最大的机会，包括在 20 分钟内回复电子邮件和语音邮件，甚至可能是一条专用电话线。他们是你当前收入、未来工作和推荐的最佳来源。优待他们，用难以置信的服务取悦他们。我和一群网络专业人士一起工作，一般来说，我不太倾向于和那些我知道会给我回复并完成工作的人谈判。让客户感到特别是很重要的。

给那些有迫切需求、有很大机会雇佣你的潜在客户一个绝佳的机会。他们现在正在评判你，知道你在销售过程中如何回应他们是一个很好的指标，表明一旦你雇佣了他们，你会如何回应。我现在和一个团队一起工作，这个团队在销售过程中反应非常迅速，现在仍然如此。很荣幸。

也要让高层接触到你最好的推荐来源，那 20%会带来你 80%的推荐。

至于其他的，你必须做出一些艰难的选择。我发现对于“糟糕”的客户，我越难接近，他们就越看重我。我变得稀缺，也让他们知道我不会忍受他们的垃圾。这让我处于平等的地位，也限制了我对他们的危机做出反应的时间。

对于希望得到免费建议的潜在客户来说，情况也是如此。我很有礼貌，但也说，“我必须优先考虑我的时间支付客户。如果你想继续，我可以为你腾出更多时间。但我不确定你是否有预算或意愿雇佣我。”

写下来，上面听起来很明显。但我观察我的营销咨询客户，以及他们如何打发时间。比你想象的更少的人擅长选择和优先考虑他们如何度过时间，以及他们有多容易接近。
