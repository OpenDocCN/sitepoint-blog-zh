# 5 个优雅的 jQuery 表

> 原文:[https://www.sitepoint.com/5-elegant-jquery-tables/](https://www.sitepoint.com/5-elegant-jquery-tables/)

厌倦了陈旧乏味的 html 表格默认外观？jQuery Table 为普通的 html 表格提供了不同风格的外观，并增加了更多有用的功能，如列排序、显示计数等等。它还提供了普通 html 表格无法提供的独特设计和结构。jQuery 将枯燥的 html 基本表格转换成优雅有用的表格
这是 5 个好看的 jQuery 表格的列表。

## [1。热色](http://www.jnathanson.com/blog/client/jquery/heatcolor/index.cfm)

这是一个插件，允许你根据从元素中得到的值给元素分配颜色。您可以将表行、div 或列表成员等元素的集合绑定到 heatcolor，并让它完成工作。它可以找到所需元素的最小值和最大值，或者您可以手动传递它们。

[![](../Images/371c80d04905d682b85c59853ffc93ab.png)T2】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Table-Heatcolor.jpg)

## [2。英格丽](http://reconstrukt.com/ingrid/)

这个 jQuery 表是一个组件，它向表中添加了 datagrid 行为，包括列大小调整、分页、排序、行和列样式化等。
[![](../Images/9af49432a50a809e4d850f36b87ab94b.png)T3】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Table-Ingrid.jpg)

## [3。fleigrid](https://flexigrid.org/)的缩写

一个轻量级但丰富的数据网格 jQuery 插件，具有可调整大小的列和匹配标题的滚动数据，以及使用 Ajax 加载内容连接到基于 xml 的数据源的能力。
[![](../Images/b815bdcad896303de028ae3e6b7ebb92.png)T3】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Table-Flexigrid.jpg)

## 4.表格分类器

一个 jQuery 插件，用于将带有 HEAD 和 TBODY 标签的标准 HTML 表格转换成可排序的表格，而无需刷新页面。tablesorter 可以成功地解析和排序许多类型的数据，包括单元格中的链接数据。
[![](../Images/caa9ba7039e6c5c10c8c560c003d0f66.png)T3】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Table-Sorter.jpg)

## [5。可滚动表格](http://www.webtoolkit.info/scrollable-html-table-plugin-for-jquery.html)

这是一个插件，可以用来将普通 HTML 中的表格转换成可滚动的表格。不需要额外的编码。
[![](../Images/fe687e57e79a24284df9f9960f672084.png)T3】](https://www.sitepoint.com/wp-content/uploads/jquery4u/2010/11/jQuery-Table-Scrollable.jpg)
