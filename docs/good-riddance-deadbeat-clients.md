# 对赖账客户的好解脱

> 原文：<https://www.sitepoint.com/good-riddance-deadbeat-clients/>

每个人都有这样的客户:慢付费客户和免费客户。如果你已经做了足够长的时间，最终你会得到出色的项目，还有未付的发票。更糟糕的是，有时你建立的网站是实时的和可操作的，而你仍然在等待和希望得到报酬。你甚至可能会拒绝帮助你建立网站的自由职业者或其他代理贡献者的收集工作，直到你得到客户的付款后才能付钱给他们。

##### 防患于未然

防止客户在未经你允许的情况下成为无偿服务者的第一步是使用一份保护你利益的服务协议，然后在整个项目中坚持这份协议。对于您的服务协议，请确保您、您的律师或任何撰写该协议的人包含以下保护您的条款:

2.  前期分期付款 :
    在项目开始时，总是要求提前付款。这是整个项目估算的 25%，还是 50%，或者其他数字，取决于你。但是，除非是你认识多年或者以前合作过的客户，没有钱就开始工作会让你从一开始就落后于现金流曲线。你可能永远也赶不上了。首期付款也确保了该项目是客户的优先事项。如果他们已经直接投资于 it，他们更有可能给你你需要的输入和反馈来使项目建立和交付。

5.  **分期付款** :
    在你开始工作之前的第一次分期付款之后，建立分期付款。让每一个交付物触发一笔足够支付你到下一个到期日的费用。一旦你完成了交付，在项目上做更多的工作之前，等待进度付款。这将产生两个必要的影响。首先，如果客户对你到目前为止所做的有疑问，或者认为它不令人满意，你会在做任何事情之前发现。第二，它会阻止你向客户提供超过一定额度的信贷。

8.  **给沉默开发票** :
    沉默的客户是一种常见的生物。在几个星期左右的时间里，反馈可能是很好的，但是当项目进入第二个月或第三个月时，选择仍然是必需的，依赖于客户的内容缺乏，项目停滞不前。这可能会阻止你获得下一笔分期付款，甚至可能阻止你最终交付项目。在您的协议中加入一条规定，说明如果未能对重复的内容和批准请求做出回应，将导致按规定的小时费率为到那时为止所做的工作开具发票。

12.  在你付钱之前，它都是我的 :
    你的标准客户协议可能已经说明了哪些知识产权属于客户，以及在项目结束时哪些知识产权只是在许可之下。如果没有，把它放在那里。许多协议没有规定你公司创造的代码、文本、布局、设计特征和其他方面的所有版权归你所有，在你得到全额付款之前，客户没有使用它们的许可。这样的条款可以给你一个强大的工具来对付赖账者。在美国，你可以给赖账客户的 ISP 一个数字千年版权法案(DMCA)通知，ISP 应该通过关闭网站来兑现。如果他们还没有支付合同价格，他们复制和展示你的作品就是侵犯了创作者的权利。当然，DMCA 不适用于欧盟或澳大利亚，但在这些地区存在其他版权保护，可以帮助提供一些支付杠杆。

15.  **我会主持它，没问题** :
    在开发期间主持网站，直到所有的付款被支付是另一种保持对交付和支付过程的控制的方式。如果在你坚持付款的情况下，客户实际上无法启动网站，或者无法将网站交给竞争对手完成，他们就更难避免支付欠款。

18.  如果你不得不追着赖账者不放，他们应该付更多的钱 :
    特别是在那些无论谁赢，双方都要支付自己法律费用的国家，在你的合同中写明赖账者客户负责向你讨债所需的任何法律费用和成本是很重要的。还要考虑这样一个规定，即如果在一定时间内未支付发票金额，将开始计息。如果客户发现自己处于不得不决定偿还哪些债务和让哪些债务延期偿还的不幸境地，这两点可以帮助你的发票上升到首位。

##### 如果你在一个洞里，停止挖掘

“我知道我们需要支付您的最后一张发票，但我们的会计部门现在已经拿到了，我们真的需要完成网站才能让它上线。”如果这听起来很熟悉，加入俱乐部。一个缓慢付款的客户通常会报告内部付款的进展，以及一个令人信服的故事，说明如果网站在周末未完成，他们为什么会完全陷入困境。当心。这位客户真正要求的是，尽管有不良的付款记录，你还是要延长信用期。他们要求你继续履行甚至优先履行对他们的义务，而他们却没有履行对你的付款义务。

如果你在一个洞里，停止挖掘。为了加快这个过程，如果他们的办公室离你的办公室很近，你可以主动提出亲自去取，或者让他们知道他们可以用信用卡付款。你仍然可以表现出乐于助人，关心客户的困境，而不会把他们的问题变成你的现金流问题。一旦你收到客户的付款日期，你就可以同意在收到付款后的几天内清空你的日程表来处理他们的项目。

##### 不是你，是我

对你的客户严格要求会很难。大多数客户关系都是合作，开始于热情和信任。如果你担心你的新客户会对一份提到律师费、收集费用、全额支付前所有知识产权的所有权以及其他维护你权利的条款的协议侧目，告诉他们，“不是你的问题，是我的问题。”解释说你以前受过伤，尽管你知道他们不会成为问题客户，但你的商业伙伴(或配偶，或律师，或你脑海中的声音)坚持要你使用反映这些惨痛教训的合同。如果客户抱怨或拒绝，问问你自己，为什么他们同意这些条款会是一个大问题。

这些合同的保护性条款只有在客户未能坚持他们的交易时才有意义。只要他们计划按照约定向你付款，交易中的保护、权利和付款部分就永远不会成为问题。如果你追求的一些保护措施开始看起来像是交易破坏者，那么可能是时候重新考虑你是否想做一份让你孤立无援的工作了。商务人士经常觉得他们不能放弃一个潜在的客户，但是要小心。这些人也不是免费工作的。
