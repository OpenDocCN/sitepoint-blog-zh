# 会议双标题

> 原文：<https://www.sitepoint.com/conference-double-header/>

在过去的两年里，我参加了微软的 MIX 大会和 SXSWi。今年没有什么不同，除了他们是背靠背。我发现会议既可以是一个学习和建立关系网的好机会，也可以是一个收益甚微的巨大开销。为了帮助那些第一次参加这些活动的人，我想就如何充分利用这些活动给出一些建议是个好主意。

#### 混合

在这两个事件中，MIX 更像是一个典型的会议。主要焦点是会议和主题演讲，一切都以微软为中心。关于 MIX 要记住的一件事是，大多数内容都发布在网上，所以我的建议是参加有有趣演讲者的主题。许多使用 Silverlight 的构建应用程序以及在您的网站中添加[insert MS Live Service]都很不错，但可以很容易地在线跟进。许多混合 UX 发言者，如楼卡波恩，是非常鼓舞人心的听，可能会有更多的影响现场比通过网页。

将课程发布到网上也可以让你跳过一些课程。这很好，因为微软提供了一些放松和与其他与会者交谈的好地方。结识新朋友和寻找新的联系是任何会议最有益的部分之一，所以花点时间看看沙盒和其他讨论区。

#### SXSWi

西南偏南互动与 MIX 完全相反。在 SXSW，围绕会议进行的活动与会议和内容一样有趣(也一样重要)。很难推荐任何特定的会话，因为太多了，所以我将更多地关注 SXSWi 的其他方面。

#### 在 SXSW 演讲

虽然会议已经安排好了，但是在 SXSW 上有很多机会让你的声音被听到。和往常一样，BarCampAustin 与 SXSW 正好在同一条街上，这是一个建立关系网的好地方，对任何人都开放。在 SXSW，有两次发言机会。核心对话是由一些小组演讲者领导的讨论组，OpenFormat 是 SXSW 内部的一个非会议；两者都将为你提供一个分享你的想法和创新的途径。
