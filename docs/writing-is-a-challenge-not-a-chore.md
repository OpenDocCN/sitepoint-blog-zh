# 写作是一种挑战，而不是苦差事

> 原文：<https://www.sitepoint.com/writing-is-a-challenge-not-a-chore/>

我喜欢把我在论坛报的写作归功于出色的编辑，他们不知疲倦地在幕后工作，让我的文字流变得有意义；然而，有许多方法可以提高你的写作水平。这是其中的八个。我要求你浏览你的网站，根据这些规则来看你的内容。解决任何问题可能会对你的转换率产生很大的影响。

### 冲动袭来时写作

如果你和专业作家交谈，他们会告诉你，他们每天会坐下来写一整天的文章。我们其他人就没那么幸运了。我发现我只有在有心情的时候才能真正写好。当然，即使我不想写作，我也可以写作；不过，我敢肯定，逼的时候就表现出来了。知道什么时候你在写作状态，什么时候不在。当你想写作的时候，选择写作会大大提高大多数人的写作水平。

### 简单是关键

就像一个新设计师在一个简单的图形上使用十种字体一样，许多新作家认为他们需要使用复杂的单词来看起来聪明。这样做只会让你的作品难以阅读。使用词典来改变你的用词是可以的，但是不要为了一个更难的词而放弃一个好的、描述性的词。

### 避免冗长

如果你能在不改变句子意思的情况下去掉一个单词，那就去做。以这个句子为例:你能去掉哪些不必要的单词？现在让我们再试一次。以这个句子为例:你能去掉哪些词？注意到区别了吗？简洁，尤其是在商务交流中，就像一股清新的空气。读者会因此而感激你。

### 写简洁的句子

写完文章后，大声朗读。你应该能够不喘着气读一个句子。在商务写作中，人们喜欢较短的句子。因此，除了删除不必要的单词，还要力求简洁。

### 保持段落紧凑

人们倾向于浏览网站，而不是像阅读报纸那样从上到下阅读。如果你是为网络写作，限制你的段落长度，你的读者会不断回来看更多。

### 使用主动语态

这不是锻炼你的声带。主动造句比被动造句使用更直接的语言，用更强的动词为动作注入能量。对比以下:网站是我建的。我建了网站。在第一句话中，主语是网站，网站正在被操作；这是被动语态。在第二句话中，我是执行动作的主体。有时是需要强调什么的问题，但更多时候，主动结构的直接性对于清晰的英语来说是更可取的。这个句子现在不仅更短，而且更有力，读起来更好。

### 拼写检查，拼写检查，然后拼写检查

没有什么比在商务交流中看到拼写错误更糟糕的了。我已经记不清看过多少次幻灯片中有错误的演示，或者对一家企业失去信心，因为他们的网站上到处都是拼写错误。当然，使用你的拼写检查软件，但是要小心[同音](http://en.wikipedia.org/wiki/Homophone)，当一个拼写正确的单词被放在错误的地方；比如两个，太，到。慢慢地再读一遍，如果有帮助的话，甚至可以请一位同事读一遍。

### 继续写

你写得越多，进步就越大。我发现定期写作比阅读编辑之类的书籍对我更有帮助。先写一篇短文热身，然后着手更大的工作。在你意识到之前，你会发现自己写得更好，没有痛苦。祝你好运采纳这些规则，把写作当成挑战而不是苦差事。只要做一点准备和深谋远虑，你马上就能写出更好的内容和信件。
