# Pro::PHP 网络直播与微软相遇

> 原文:[https://www.sitepoint.com/prophp-webcast-meets-microsoft/](https://www.sitepoint.com/prophp-webcast-meets-microsoft/)

上周五，马库斯·惠特尼与微软—[布莱恩·戈德法布](http://blogs.msdn.com/bgold/)和[乔·斯塔格纳](http://www.joeon.net/)非常冷静地处理了另一个播客，涵盖了他们的交叉训练策略——[http://www.learn2asp.net/](http://www.learn2asp.net/)——以及他们如何在 Windows 上用 PHP 平滑粗糙的边缘，并创造了一个新术语:“合作竞争”(但让我们只希望它不会进入 Web 2.0 词典)

Brian(如果我没记错的话)有一个特别令人惊讶和深刻的评论，大意是 PHP 开发人员对技术上的“如果”更感兴趣在他看来，问题和完成任务比花时间在语言政治上更重要。这句话所暗示的思考和研究表明他对 PHP 有很好的理解。退一步说，有趣的是，从两年前 PHP 基本上是草根，微软是“敌人”到某种程度的相互理解，事情已经发生了很大的变化。在某种程度上，这与马尔科·塔比尼(Marco Tabini)在回来的时候(T3)说了一句 T2 话有关。
