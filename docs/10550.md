# 为 Android 和 Maemo 发布 Firefox 4.0 移动版

> 原文：<https://www.sitepoint.com/mozilla-firefox-4-mobile-android-maemo/>

Mozilla Firefox 4.0 Mobile 的首个测试版已经发布。虽然[1.0 版本于 2010 年 2 月发布](https://www.sitepoint.com/mozilla-firefox-mobile-1/)，但它只在诺基亚的 Maemo 平台上可用，所以相对来说很少有人能尝试。新版本支持 Android 平板电脑和手机，因此将拥有更广泛的潜在受众。

新的浏览器与 Firefox 4.0 桌面浏览器基于相同的核心，两个版本的最终版本预计将于 2010 年 11 月发布。为了避免混淆，Mozilla 手机版从 1.0 版直接跳到了 4.0 版。

Firefox 是最先进的移动浏览器之一，包括更好的性能、标签、一键式书签、自动同步和减少输入的“令人敬畏的屏幕”*(我仍然讨厌这个名字！然而，[的移动附加产品](https://addons.mozilla.org/en-US/mobile/)最能激起人们的兴趣。目前只有几十个可用的，但你已经可以找到新的主题，AdBlock，NoScript，tab 工具，feed 阅读器，社交网络小工具和俄罗斯方块的克隆。*

Firefox 4.0 Mobile 可以安装在诺基亚 N900 或 Android 2.0 设备上，内置 32MB 存储空间和 12MB SD 存储空间。已知它与以下手机兼容:

*   诺基亚 N900
*   HTC 欲望
*   HTC EVO 4G
*   HTC 谷歌 Nexus One
*   摩托罗拉 Droid 2

Mozilla wiki 上有一份完整的设备列表。
