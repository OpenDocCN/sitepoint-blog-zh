# 设计颜色:粉色

> 原文:[https://www.sitepoint.com/color-in-design-pink/](https://www.sitepoint.com/color-in-design-pink/)

上周，我写了关于设计中的土褐色的文章。本周我们来看看更艳丽的粉色。红色与激情联系在一起，而粉色则与爱情和浪漫联系在一起。这是一种时尚的颜色，虽然通常被认为是女性化的，但它不仅仅适合女孩！你够男人穿粉红色吗？粉红色是一种时尚和有趣的颜色，在网页设计中出奇的受欢迎。

想想粉红色在日常语言中的用法，你就会明白它是如何被认为是一种积极的颜色。一个健康的人有时被称为“精神矍铄”，满足和快乐的人被描述为“开心极了”。

**在设计中使用粉色:**

当然，粉色有许多不同的色调。从浅粉色到橙粉色，再到深粉色，这种颜色有各种名称，如紫红色、玫瑰色、鲑鱼色、肉色、珊瑚色和热粉色。每种场合都有一款粉色。

你可以用淡粉色和淡紫色来创造柔和精致的调色板。粉色与黑色或深灰色的搭配看起来很精致。粉红色和蓝色是俏皮和有趣的，而暗粉红色与酒红色相结合可以创建一个更严肃的基调。橙红色和苔绿色形成了一个自然、朴实的组合。

杂志封面是一个很好的地方看到粉红色调色板的行动。在这张《时尚芭莎》封面中，格温妮丝·帕特洛(Gwyneth Paltrow)穿着一件深粉色连衣裙。字体都是一种字体，但是亮粉色和她的口红很配。鲑鱼粉色和白色的文字添加到调色板。
