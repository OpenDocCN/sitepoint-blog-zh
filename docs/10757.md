# 别浪费我的时间了！选择真正有潜力的促销活动

> 原文：<https://www.sitepoint.com/stop-wasting-my-time-choose-promotions-with-real-potential/>

在传统的营销圈里，媒体浪费指的是到达错误受众手中或未被目标受众看到的促销支出比例。

举例来说，你可能会在给定的出版物中购买广告空间，但是在浏览你的广告的人群中，有些人不会是你的目标受众。另一方面，一些购买了出版物的目标读者不会查看该页面。在这两种情况下，你的广告支出都有一部分被浪费了。如果你花时间准备那个广告，你一定比例的时间也被浪费了。

然而，对于自由职业者来说，媒体浪费的概念超出了付费促销。毕竟，你的时间就是金钱。因此，把媒体浪费看作是任何宣传支出——时间或金钱——未能到达你的受众手中的比例是值得的。这个概念提供了一个快速评估你遇到的每一个促销机会的简洁方法。

## 浪费是如何运作的

当你开始考虑浪费的成本包括你的时间和促销的实际成本时，你会发现自由职业者的促销活动发生了变化。

例如，想象你今晚面临三个机会:你有机会为一个擅长向你推荐工作的客户完成一份小的有报酬的工作，有机会写一篇文章在你的目标受众中广泛阅读的出版物上发表，以及邀请你参加一个社交活动，在那里你预计会遇到你的目标受众中的一系列人。

你有三次机会，一个晚上。你不能都做，那么你会选择哪一个？

评估你的选择的一个方法是关注每项任务能让你接触到多少潜在客户。在这种情况下，您可能会决定撰写文章，因为访问该出版物的人会比其他任何一种方法都多。

另一方面，如果你只考虑浪费，你可能最终会选择低薪工作。毕竟，你的客户有向你推荐客户的良好记录，从获得时间回报的角度来看，这是晚上唯一的付费选择。

但是，如果你觉得你最好的线索是你认识的人，你可能会在参加社交活动，知道你至少会遇到一些不需要你的产品的人，或者做客户工作，希望一个好的项目结果会为你赢得一两个新的客户推荐之间犹豫不决。

## 权衡目标

这种决定从来不是凭空做出的。如果你要在这三个选项中做出选择，你的决定可能会受到你的情绪、你的社交感受、你的现金流状况、你对每个机会的兴奋程度、你最近使用的其他促销方式等因素的影响。

但是也许在做决定时最重要的考虑应该是你的商业目标。如果你的目标是瞄准当地受众并建立面对面的关系，那么你会做出与你的主要目标是作为你所在领域的专家建立行业声誉不同的选择。如果你的主要目标是每月增加一定比例的收入，你的决定可能又会不同。

虽然我们从来不会仅仅根据浪费来评估一个机会，但这是一个有价值的考虑因素，可以帮助你在机会之间做出选择，并就如何花费时间和促销预算做出明智的决定。
