# 自由职业者需要用合同吗？

> 原文:[https://www.sitepoint.com/does-freelancer-need-contracts/](https://www.sitepoint.com/does-freelancer-need-contracts/)

我坚信书面合同，几乎所有项目都需要书面合同。虽然他们肯定不会保护你的一切，也不能保证你会按时得到报酬，但合同是生意中不可忽视的重要部分。

**同意书面合同**

除了为你和你的客户提供法律保护之外，使用合同还有很多好处，有些好处不太明显。书面合同:

*   阐明项目条款
*   指定付款金额、方式和到期日
*   概述担保、责任和其他法律责任
*   有助于避免对将要做什么、何时做以及由谁做的误解
*   向你展示一个专业的、有经验的自由职业者
*   防止项目范围蔓延
*   为收集必要的客户和项目信息提供标准格式

如果没有某种书面文件，很难确保双方对情况有充分的了解，更好的是，有相同的了解。一份书面合同真的可以大大减少沟通失误，解决分歧，并有助于推进项目。

一旦你创建了一个标准合同，就很容易根据新工作的需要修改它。尽管如此，你应该考虑聘请一名律师来审查和/或起草你的初始合同，这样它才是一份有效和清晰的文件。

当然，并不是所有的书面合同都必须是正式的，充满法律术语。合同只需要概述你已经同意的条款，并有双方的签名就具有法律约束力。事实上，在某些情况下，一连串的电子邮件(假设你保留了整个对话的线索)可能会被用作有约束力的合同。

**不需要合同**

我承认，在某些情况下，自由职业者可能会选择放弃书面合同。一些这样的情况可能包括:

*   风险很小的非常小或非常快的项目
*   您目前与长期客户有单独协议的新项目
*   一些公益工作的案例
*   你为家人和朋友做的非正式项目

我的观点是，在每一种情况下，都可以提出一个论点，说明为什么合同仍然是必要的。如果你曾经和家人或朋友一起工作过，你可能知道我的意思！但是只要你在权衡每种情况的收益和风险，做出慎重的决定，就要相信自己的直觉。

你使用合同吗？每一个项目？
