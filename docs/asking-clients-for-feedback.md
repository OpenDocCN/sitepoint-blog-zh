# 你达标了吗？如何向客户寻求反馈

> 原文：<https://www.sitepoint.com/asking-clients-for-feedback/>

如果你专注于取悦你的客户并创造回头客(如果你想要长期的成功，你就应该这样)，你应该专注于询问并吸收客户的反馈。很多时候，客户对你所做的工作是否感到兴奋是显而易见的，但是更进一步问一系列有针对性的问题会有很多收获。

衡量一个项目的成功不仅仅是询问客户是否满意。你应该获得足够的信息来准确了解你做得好的地方和你可以做得更好的地方(总有你可以做得更好的地方)，并能够制定改进计划。向客户发送调查问卷的最佳时机是在项目即将结束时，此时与您合作的经历仍历历在目。

**问什么问题**

在客户反馈调查中，您可能会问各种各样的问题，但一些基本问题包括:

*   您为什么选择我们作为您的服务提供商？
*   我们为你提供了什么服务？
*   你如何评价你接受的服务？(提供评定等级)
*   我们在哪些方面达到或超过了您的期望？
*   我们可以在哪些方面改进？
*   你会把我们推荐给其他人吗？

客户反馈调查最重要的一点是问足够多的(正确的)问题，以产生有价值的反馈，而不要问太多的问题，以免冒根本得不到回应的风险。你也可以考虑为完成调查提供奖励，尤其是当你发现回复率不是很高的时候。

**如何提问**

除了雇佣外部调查公司代表你调查你的客户，这对于大多数自由职业者和小企业主来说是不切实际的，还有一些方法可以让你的客户提供反馈:

非正式地询问——你的客户反馈调查可以简单到向你的客户发送一封带有问题的电子邮件。这种方法既快又容易，但也有一些缺点。也就是说，你冒着信息被搁置一边并最终在混乱中丢失的风险。您还被限制使用开放式问题，这妨碍了所有调查的一致性。最后，您必须手动对收集到的数据执行一些其他操作，以便以任何可用的形式保存数据。

创建您自己的调查–您可以将电子邮件格式更进一步，创建一个更加标准化的调查。您可以在文字处理应用程序中创建受保护的表单、PDF 表单或可以通过您的网站提交的表单。这种格式显然需要更多的时间来创建，但它允许你有一个标准的调查，你发送给所有客户。

使用调查网站——管理客户调查的另一种方式是使用第三方调查网站，如 [SurveyMonkey](http://www.surveymonkey.com) 、 [Constant Contact](http://www.constantcontact.com) 和 [PollDaddy](http://www.polldaddy.com) 。这可能是一种相当快速的方式来创建定制的、广泛的和安全的调查，并可以选择运行报告来分析响应，尽管大多数服务都有与服务相关的月费。

您使用的形式取决于您与客户的关系、在给定时间内您想要调查多少客户、您有多少时间以及您计划如何处理调查结果。
