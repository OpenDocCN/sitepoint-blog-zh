# Visual Studio 2008 RTM 来了！

> 原文:[https://www.sitepoint.com/visual-studio-2008-rtm-is-here/](https://www.sitepoint.com/visual-studio-2008-rtm-is-here/)

如果你一直在岩石下，同居或在监狱或其他不舒服，Visual Studio 2008 RTM 在这里！详情请见斯科特·格思里的官方声明。我个人认为最引人注目的新特性是:

*   Javascript 智能感知和调试。坦率地说，我不擅长 Javascript——即使从头到尾读完了非常棒的[Simply Javascript](https://www.sitepoint.com/books/javascript1/)——主要是因为它需要记住比我目前准备记住的更多的细节。此外，缺乏好的调试器确实会损害事业。警告值并不能解决问题，尤其是现代 javascript 的复杂程度。
*   哈希表。最后。唯一能让我更高兴的是，他们制作了带有添加、更新和删除事件通知的原生集合。

注意，我并没有真的提到 LINQ，这是故意的。LINQ 本身真的非常非常棒。但是 LINQ2SQL 有潜力成为 2008 一代的数据集。就像它的前身一样，它被设计成一个 RAD 工具，让任何人都可以将数据库对象拖放到表单上，并制作简洁、实用的应用程序。它将为简单的事情工作。但是一旦人们开始涉足复杂的真实世界的数据场景，并开始考虑长期的可维护性，它就开始变得平淡无奇了。另一个巨大的缺陷是完全有状态的 DataContext 对象。如果您使用 LINQ2SQL 作为您的对象层，您最终会被绑定到数据库，因为没有办法在真正断开的模式下处理事情。一旦我有机会再踢踢轮胎，以后会有更多的介绍。
