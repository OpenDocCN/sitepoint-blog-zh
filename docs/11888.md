# 12 种检查网站可访问性的工具

> 原文:[https://www.sitepoint.com/site-accessibility-tools/](https://www.sitepoint.com/site-accessibility-tools/)

当你设计任何一个网站时，你的首要目标之一是确保你能获得尽可能多的访问者，但是你检查过你的网站的真正可访问性吗？色盲能看懂吗？你所有的剧本都合作了吗？你的颜色搭配起来可能看起来不错，但是对比是否足够不同以至于都清晰可辨？好吧，这些问题正是这 12 个工具旨在帮助你回答的，并且运行多个工具来确保它们都告诉你相同的事情，这肯定不是一个坏主意。

对这个列表的一个警告是，虽然这些工具是有帮助的，会帮助你发现一些问题，但永远不要相信它们是最终的权威，而更多的是你建立最平稳运行的网站的起点。

**[ACTF 设计者](http://www.eclipse.org/actf/downloads/tools/aDesigner/index.php)** :开源 Eclipse 开发平台的扩展，专门为测试视障人士的可访问性而构建。遗憾的是，它只适用于运行 Windows XP 及更高版本的用户。

[![eclipse valid](../Images/3a96b9c446cbc6a5eaedf941de04fffc.png "eclipse valid")T2】](https://www.sitepoint.com/wp-content/uploads/2009/06/eclipsevalid.gif)

**[Adobe Dreamweaver CS4 可访问性](http://www.adobe.com/accessibility/products/dreamweaver/validation.html)** :对于使用 Adobe Dreamweaver CS4 的设计人员，您会发现一个内置的验证工具，它允许您随时选择要测试的内容，并获得关于它可能发现的任何错误的完整报告。

![dreamweaver validation](../Images/263c42a3e99c4f413ff46b1fcaeeae2e.png "dreamweaver validation")

对比分析器 :不仅很难选择你想在一个网站上使用的颜色，而且你还必须确保它们在可读性方面协调一致。Colour Contrast Analyser 将让您知道两种颜色之间的差异，它还可以帮助您确定有视觉障碍(如色盲)的人是否能够阅读它。

[![colour contrast analyser](../Images/fd4707cfdfd08ff320c3dc05a3637b03.png "colour contrast analyser")T2】](https://www.sitepoint.com/wp-content/uploads/2009/06/colourcontrastanalyser.gif)

[**辛西娅·赛义德**](http://www.contentquality.com/) :辛西娅·赛义德是他的软件中的一个产品，它允许你在 sight 中输入你的网站地址，并获得一份关于它如何符合 508 条款和 WCAG-优先级 1、2 和 3 的报告。

[![cynthia says](../Images/0c1ff1091457e573f3b7810b76b7c6ec.png "cynthia says")T2】](https://www.sitepoint.com/wp-content/uploads/2009/07/cynthiasays.png)

:由伊利诺伊州信息技术和网页可访问性中心(iCITA)创建，这个 Firefox 工具栏包括一个大型工具套件，用于测试您的网站的许多方面，以方便残疾人访问。包括测试文本、脚本、样式等等。

[![firefox accessibility](../Images/b781bb10fe6e8fd40c89b8f35ffc60e3.png "firefox accessibility")T2】](https://www.sitepoint.com/wp-content/uploads/2009/06/firefoxaccessibility.gif)

:功能性易访问性评估器给你一份关于你的网站的许多方面的报告，然后给你一个彩色编码的结果页面，里面有你需要知道的一切的很好的概述。如果你想了解更多信息，你只需点击该类别就可以看到详细的评论。

[![faes](../Images/e6f8040fd00026620ae507446aa4bd62.png "faes")T2】](https://www.sitepoint.com/wp-content/uploads/2009/06/fae.png)

**[富士通网页可访问性检查器](http://www.fujitsu.com/global/accessibility/assistance/wi/)** :富士通网页可访问性检查器主要检查您的页面的构建，因为它可能会出现在老年人和视力障碍者面前。该软件必须下载到你的系统，最大 OS X 或 Windows，然后你可以指向一个本地文件或网站，然后它会生成一个报告，几乎太长，注意到你的文件的每个方面。当我们尝试一个完整的截图时，它的大小超过了 924 MB，所以有点长。

[![fujitsuwebs](../Images/a0dc65b5017dd42101601848c44b17d9.png "fujitsuwebs")T2】](https://www.sitepoint.com/wp-content/uploads/2009/06/fujitsuweb.png)

IBM 的 Rational Policy Tester Accessibility Edition:一个仅用于 Windows 的解决方案，用于测试您的站点对残疾人的可访问性。这是一个付费的解决方案，没有简单的成本链接，但它可能不便宜，因为它附有 IBM 品牌名称。

[![ibm rational](../Images/aa85e52a351124c9c5b449a90fb887c7.png "ibm rational")T2】](https://www.sitepoint.com/wp-content/uploads/2009/06/imbrational.gif)

[**tru wex Online 2.0**](http://checkwebsite.erigami.com/accessibility.html):tru wex Online 2.0 工具检查一系列可访问性选项，如 508 条款、WCAG 级别、隐私、断开的链接等。只需进入该页面，输入您的 URL，选中您想要检查的项目的复选框，它会在此后不久生成报告。

[![truwex](../Images/50f157f8402c7cbe35de1fa75f503d76.png "truwex")T2】](https://www.sitepoint.com/wp-content/uploads/2009/07/truwex.png)

澳大利亚视觉协会 :澳大利亚视觉协会是一个由人们组成的联盟，致力于确保患有各种视力问题的人能够像其他人一样平等地享受生活。他们为 Internet Explorer 和 Opera 提供了两个工具栏，以测试您的网站如何为视力受损的人服务。

[![vision australia](../Images/959b9ecc112839b6769f990a1de3b852.png "vision australia")T2】](https://www.sitepoint.com/wp-content/uploads/2009/07/visionaustralia.png)

**[WAVE](http://wave.webaim.org/)** : WAVE 由 WebAIM 制作，致力于确保网站可以用尽可能多的语言访问。简单地输入网站的地址，上传一个文件或输入一段代码，就可以看到所有的元素。此外，您会收到关于每个元素的位置和标识的反馈。还有一个使用 Firefox 工具栏或在浏览器和网页中安装 WAVE 的选项。

[![wave](../Images/f14bb3aee8342c3f999261e264cb2efc.png "wave")T2】](https://www.sitepoint.com/wp-content/uploads/2009/06/wave.png)

**[Opera](http://www.paciellogroup.com/resources/wat-about.html)**的网页可访问性工具栏:Opera 用户快速访问 Paciello 集团一些最常用的网页可访问性工具的方法。当你离线时，并不是所有的项目都可以运行，因为它们需要访问位于 WAT-C 服务器上的脚本和工具，但是如果你可以访问互联网，你就可以得到所有的东西。
