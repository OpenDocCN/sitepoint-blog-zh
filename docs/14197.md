# 埃阿斯

> 原文：<https://www.sitepoint.com/ajax/>

杰西·詹姆斯·加勒特和 Adaptive Path 的团队创造了一个简写术语“Ajax ”,意思是所有人都在谈论的这些很酷的 DOM、CSS 和 XMLHTTPRequest 的组合。 [Garrett 的文章“Ajax:Web 应用的新方法”](http://www.adaptivepath.com/publications/essays/archives/000385.php)，对 Ajax 的定义如下:

*   使用 XHTML 和 CSS 的基于标准的表示；
*   使用文档对象模型的动态显示和交互；
*   使用 XML 和 XSLT 进行数据交换和操作；
*   使用 XMLHttpRequest 进行异步数据检索；
*   和 JavaScript 将一切绑定在一起。

虽然我怀疑这些概念对于时髦的脚本读者来说并不陌生，但是用一个好听的简称来引用它们肯定是很方便的。这篇文章详细介绍了这些概念是如何联系在一起的，并且列举了支持 Ajax 的应用程序的例子，比如 Google 的大部分最新创新。DOM 的[年是 Ajax 年。](https://www.sitepoint.com/blog/)
