# 10 个基于图片的 WordPress 插件

> 原文:[https://www.sitepoint.com/10-image-based-wordpress-plugins/](https://www.sitepoint.com/10-image-based-wordpress-plugins/)

Last week, we shared tutorials about jQuery-based and Slider-based WordPress plugins. Today, we will learn about Image-based WordPress plugins. These cool and ass kickin’ image plugins were sought right just for your web page! Have fun!

## 2.盛大 Flash 相册库

插件可以让你用幻灯片显示无限数量的照片。

![Grand-Flash-Album-Gallery1.jpg](../Images/e94668855ad094aa210082a77999b106.png)

## 3.卡洛加

Kalooga WordPress 插件可以让你用照片库丰富你的博客。Kalooga 正在网上搜索收藏照片的画廊。这些收藏的预览保存在 Kalooga 视觉画廊索引中。

![Kalooga2.jpg](../Images/7953357e14e15ce9d33fd1f348721c76.png)
[来源](http://integration.kalooga.com/integrations/wordpress/)

## 4.翻页图像库

一个完全集成的带有翻页效果的图片库插件。允许通过专门设计的界面直接从管理员面板创建具有翻页效果的 flash books。

![Page-Flip-Image-Gallery2.jpg](../Images/655b05f3226dd05f10602d0337b82aed.png)
[来源](http://pageflipgallery.com/)

## 5.Cincopa 丰富的多媒体平台

将视频、幻灯片、播客和音乐添加到您的网站再简单不过了。只要遵循我们简单的步骤，你很快就会拥有一个专业的媒体驱动的网站。

![Cincopa-Rich-Multimedia-Platform2.jpg](../Images/cab0b153a6fdfa057838112497592127.png)
