# 什么是网络浏览器？没人知道！

> 原文:[https://www.sitepoint.com/what-is-a-web-browser/](https://www.sitepoint.com/what-is-a-web-browser/)

这真令人沮丧。谷歌最近在纽约时代广场随机采访了一些人，问了一个简单的问题:*“什么是网页浏览器？”*

不到 8%的人知道。大多数人把它和搜索引擎混淆了，大多数人似乎不知道他们使用的是哪种浏览器。更让谷歌担心的是，没人听说过 Chrome…

这有点吓人。也许谷歌的调查是在一个糟糕的日子进行的？不幸的是，我有一种可怕的感觉，无论你在哪里问这个问题，这些数字都将是相似的(除了芬兰，那里似乎有不成比例的高比例的懂技术的人！).

这给 web 开发人员留下了一个问题。虽然我们可能对自己选择的浏览器充满热情，但现实世界中的大多数非极客都会使用他们电脑上的任何浏览器。大多数人只有在技术朋友为他们做的情况下才会改变，我打赌他们不会注意到任何显著的不同。

如果没有人知道我们在谈论什么，我们如何鼓励采用更新更好的技术？我怀疑许多“杀死 IE6”运动的效果可以忽略不计，无知必须为浏览器缓慢而持久的死亡负部分责任。

你如何向 92%不知道或不关心的人解释“网络浏览器”是什么？
