# 使用 PicturePaste 将图像复制并粘贴到 Google、Blogger 和 WordPress 中

> 原文:[https://www . site point . com/copy-and-paste-images-into-Google-blogger-and-WordPress-with-picture paste/](https://www.sitepoint.com/copy-and-paste-images-into-google-blogger-and-wordpress-with-picturepaste/)

PicturePaste 是一个小的 windows 工具，允许您在基于 web 的富文本编辑器中使用复制和粘贴功能。Web 浏览器目前不支持此功能。当然，你可以从浏览器上复制一张图片，然后粘贴到桌面上，但是当你试图将一张图片粘贴到谷歌文档、雅虎邮箱或 WordPress 上时，这是令人沮丧的，因为它就是不能工作。

PicturePaste 通过让你在任何桌面应用程序或网络上点击一张图片，转换它，然后把它拖到几乎任何基于浏览器的编辑器中来解决这个问题。我在 WordPress 和 Gmail 中使用了它，它工作起来就像做梦一样。当您复制图像时，它会进入剪贴板，并且图像会出现在“图片粘贴”窗口中。

[![PicturePaste](../Images/c1cdfe618afe9af4fc2a3cc40014c056.png "PicturePaste")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/PicturePaste.jpg)

您可以选择 PNG 或 JPG，按下转换按钮，然后将转换后的图像拖到您的 web 应用程序中。在 WordPress 中，你需要使用可视化编辑器，而不是 HTML 视图。

[![PicturePasteDrag](../Images/5bccaed057bacb69a2778b22ac5c5366.png "PicturePasteDrag")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/PicturePasteDrag.jpg)

转换后的图像将保存到云中。就这么简单。

您还可以使用 PicturePaste 进行屏幕截图，使用 Print Screen 键捕捉整个屏幕，或使用 Alt + Print Screen 捕捉单个窗口。

该程序可以与以下软件配合使用:谷歌邮箱(Gmail)
、谷歌文档文字处理器
、ZOHO 文字处理器
、Windows Live Hotmail、雅虎邮箱
、Blogger 和 WordPress
…以及几乎所有基于网络的富文本编辑器

您可以[下载免费版的 PicturePaste](http://picturepaste.com/download.htm) ，它允许您使用复制和粘贴 50 幅图像。商业版有无限的照片转换:美国 29.95 美元(+地方税)或英国 18.00(+增值税)21.15 inc .增值税。它需要 Windows XP、Vista 或 Windows 7，可以与除 Opera 之外的大多数现代网络浏览器兼容。在[图片粘贴网站](http://picturepaste.com/index.htm)了解更多信息。
