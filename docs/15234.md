# 下一波——小型网站设计师接管

> 原文:[https://www.sitepoint.com/web-designers-take-over/](https://www.sitepoint.com/web-designers-take-over/)

无论你是网页设计师，还是与网页设计供应商合作过的人，这一行业在过去几年里都发生了巨大的变化。虽然这个市场已经发生了巨大的变化，但有几个主流正在发展，其中大多数代表了独立网络顾问的积极趋势。

##### 在内部，它得到了格恩西岛

更大的财富 500 强公司现在已经把大部分网络工作的责任交给了他们内部的信息技术(IT)部门。虽然市场营销仍然有发言权，但它将做出大多数供应商决策。这是一个转变，对于传统上与营销部门打交道并保持这种关系的网站设计商店或个人顾问来说，这可能是一个问题。

在大多数公司中，营销部门和 IT 部门向来不和。许多网页设计公司可能不具备成为 IT 顾问的技术广度和深度；因此，他们从未真正与它建立起牢固的联系。因为他们没有建立这些联系，网页设计师从公司内部 it 部门获得业务可能会变得更加困难。然而，许多进入网络领域的独立顾问确实有 IT 背景，因此他们可以很容易地过渡到 IT 部门的咨询，而不是营销。

随着这些客户意识到网络不仅仅意味着小册子软件，他们寻求为用户提供更多的功能，网站的发展变得更加依赖于企业的遗留系统。随着他们与业务系统的联系越来越紧密，业务系统(IT)的标准承担者必然会参与其中。随着 IT 变得越来越复杂，他们可能会转向过去使用的承包商。通常这些都是独立的 IT 顾问或更大的咨询公司，他们可能已经用来做其他 IT 咨询工作。

##### 媒介还是应用？

网络正从媒介走向应用。这样一来，用户体验就变成了应用程序的一部分，而以前它是应用程序的一部分。例如，当网络年轻时，它更多的是一种媒介:类似于电视和广播，它不是那么互动，虽然有一些交互性，但这种功能通常不会与核心业务实践相互联系。外观、感觉和界面很重要；目标是创造最佳的用户体验。在少数情况下，有一个用户可以遵循的定义的任务流，但是在早期，用户被期望去探索，而不是被引导。

现在，随着 Web 变得越来越像一个应用程序，外观和感觉已经变得不那么重要，成功地帮助用户完成他们的任务，并且在在线环境中，这是一种需要比熟练的图形设计更多的技能(尽管它确实有所帮助，但不是全部)。

特设接口标准现已出现。不再需要为每个站点设计新的界面和任务流:标准的 web 范例已经被开发出来，可以而且应该在新的设计中被重用。例如:一个产品公司网站可能会使用这些标准的导航项目:产品、支持、客户、关于我们、联系我们。

##### 开发商多如牛毛！

随着预算紧缩，客户不再认为大型网站设计商店，如科学和 Razorfish，2-5 人的公司，甚至是在家工作的独立顾问之间有很大的区别。只要开发商以合理的价格及时生产出高质量的产品，他们就会被考虑。那些能够以更低的成本提供类似服务的公司将赢得这份工作。

大的网络公司已经解散成一些小的独立商店，拥有同样的员工，这些商店已经能够接受和完成原来的公司不能盈利的业务。在某些情况下，独立顾问可以以低得多的成本做同样的工作，方法是召集一个特别的开发团队，逐个项目地工作。

我们看到的是网页设计市场的商品化。面临预算削减的大公司不再有兴趣只与知名品牌公司打交道:像思科这样的财富 500 强公司很乐意与 Brand X Design 打交道，就像他们与 Razorfish 打交道一样，因为当他们并排考虑最终设计时，他们看不到足够的差异来证明大型设计公司的额外成本是合理的。虽然通常在质量和可用性上有一些小的不同，但是对于外行人来说，这种不同并不能提供足够的附加值来补偿增加的成本。

##### 交给你了…

小公司和独立的网络咨询公司正在接管大公司曾经占据的空间——而且他们正在盈利。这个故事的寓意是，小经营者不应该害怕争取大客户:在这个市场上，即使是大公司也在寻找小公司和其他“自由代理人”。只要你制作一个专业的设计，并且有合适的技能:你就可以和大男孩们竞争。你可以(在目前的经济形势下)赢得价格，同时还能为你的客户和投资组合做好工作。
