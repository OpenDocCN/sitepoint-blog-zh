# 如何从头开始构建内容策略

> 原文：<https://www.sitepoint.com/build-content-strategy-ground/>

![Content strategy](img/21e5b23392f3fea553b286344b2c6c32.png)

内容是网络的货币，这是有充分理由的。

然而，很多人希望在发表第一篇文章后就能看到立竿见影的效果。内容营销确实是一个长期的游戏。但是，如果做得好，它可以帮助你在未来几年的流量。

你发布的每条内容的累积影响才是真正让你的企业获得你想要的结果。

从头开始建立一个内容策略需要一些工作，但这比按下发布键和交叉手指要好得多。下面你将学习如何从头开始创建一个优秀的内容策略。

## 为什么内容营销很重要

面对现实吧，很多广告都是侵犯性的。今天的消费者比以往任何时候都更频繁地受到广告的轰炸。

大多数其他形式的营销和销售线索的产生都是着眼于短期的。当然，你可以通过其他方式看到更多立竿见影的效果，但是当涉及到长期商业目标时，你的内容会帮助你实现。

此外，您创建的内容可以帮助您改进任何其他现有的营销策略。例如，内容可以帮助提高你在搜索引擎中的地位，让你在社交媒体上分享一些东西，并巩固你现有的品牌。

## 内容营销的好处

除了增强现有的战略，内容营销本身也有一定的好处。请记住，这是一个长期的策略，所以你不会马上体验到这些好处，但如果你坚持下去，它们就会开始显现出来。

*   更大的搜索引擎存在。你拥有的高质量内容越多，你在谷歌上的索引页面就越多，从而给你一个更大的在线呈现。
*   更有针对性的流量。你发布的每一条内容都有可能为你带来流量。无论是来自你自己的帖子排名，还是来自其他分享和链接到你的内容的推荐流量。
*   提高品牌知名度。当人们阅读、参与和分享你的内容时，你不仅增加了你的在线存在，同时也培养了与这些读者的积极关系。
*   改善客户关系。与客户的关系就是一切。通过制作解决他们问题的高质量内容，你加深了你们的关系，并在他们决定购买之前表明你是可以信任的。
*   随着时间的推移降低成本。内容的伟大之处在于，你只需支付一次费用，它就能在未来很长一段时间内持续为你的企业带来流量和收入。

## 如何建立你的内容策略

每一个伟大的内容策略都是从计划开始的。下面强调的计划可能看起来工作量很大。但是，这项工作你只需要做一次，就可以为你的整个内容策略打下基础。

## 1.组织和建立骨干

一个好的内容策略从了解你的市场开始。你需要知道你的受众是谁，他们的兴趣是什么，他们想要回答什么样的问题，他们喜欢如何消费内容。

你的一些最好的想法将来自已经成功的事情。很难从头开始创作保证会大受欢迎的内容。然而，你可以从在你的领域中已经成功的玩家那里获取想法和策略，并在其中加入你自己的想法。

这在两个方面有所帮助。首先，它让你确信你创造的东西将会成功，不需要重新发明轮子。第二，它立即将你定位为权威，因为你有一篇精心制作的内容，使你处于领先地位。

要建立你的创意银行，问自己以下问题:

*   有没有我的观众喜欢的特定形式的内容？
*   我能在我创作的每件作品上花多长时间？
*   我是否有想要利用的现有数据或内容？

### 1.竞争对手和市场理解

研究你的竞争对手是一种确定的方法，可以看出在你的领域里什么是有效的，什么是无效的。这将有助于你与你的圈子里已经存在的东西区分开来，这样你就可以避免一次又一次地走上同一条路。

将你的内容定位为独一无二的最好方法是找到你的市场中已经存在的漏洞并填补它们。这将有助于你的内容立即变得有用，而不是助长已经存在于网上的噪音。

这里有一些有用的方法可以帮助你找到这些差距:

1.  导航到 Amazon.com，找到你所在领域的畅销书。挖掘负面评论，看看什么问题会反复出现。现有的内容无法满足这些需求，您可以创建一个或一系列内容来解决这个问题。
2.  使用 [Buzzsumo](http://buzzsumo.com/) 研究最成功的内容。输入你的目标关键词，或者你所在领域的热门网站。有了免费版本，你会得到前十名左右的结果。在这些内容中，你看到了什么趋势吗？或者任何能把它炸出水面的东西？

你花越多的时间去熟悉现有的内容格局，你就能更好地定位自己，创造有助于实现你的目标的内容(我们将在下面谈到)。

### 2.选择正确的关键词

关键词非常重要。仅在谷歌上，每天就有超过 35 亿次搜索。通过搜索找到你的网站的访问者也非常有价值。

某些关键词数量的存在也有助于表明市场的规模。如果没有特定关键词短语的搜索量，那么人们就不会关心它。你能做的最糟糕的事情就是花费时间和精力创造一个没人关心的内容。

通过分析现有内容和了解你的市场，你可能有一个与你的业务相关的种子关键词列表。

你可以使用一个免费的工具，比如谷歌的关键字规划器来检查你的列表中每个关键字的总体竞争力和总量。有各种各样的付费工具可以给你更好的数据，但这是一个很好的起点。

如果你想让你的关键词研究更上一层楼，那么看看 Backlinko 的这篇文章吧。

### 3.创造“正确”的内容

你应该把每一部分内容都看作是有助于你实现业务目标的资产。您可以创建多种不同类型的内容:

*   长型极限导柱
*   信息图表
*   幻灯片共享演示和 YouTube 视频
*   内容升级和其他数字下载
*   电子书
*   甚至社交媒体帖子

你的总体内容策略远远超出了你的博客文章。你需要考虑每一部分内容是如何与你所创造的其他东西联系起来的。以及如何将它们结合起来提供价值并推进您的业务目标。

如果你还在寻找更多关于实际创作的想法，那么试试以下资源:

*   浏览受欢迎的子编辑，找到非常受欢迎的内容，看看你是否能创作出类似的内容。
*   使用 [Google Trends](http://trends.google.com) 输入关键词想法，看看是否有关键词越来越受欢迎。跟随即将到来的趋势是值得的。
*   查看 Quora ，看看你的空间中经常出现哪些常见问题，然后创建一个帖子深入回答这些问题。

### 4.规划您的内容

既然你已经知道你要写什么了，是时候制定一个计划来组织和发布这些内容了。这可以像你希望的那样复杂或简单。这取决于制作每一段内容需要什么资源。

有很多工具可以帮助你保持内容生产的正常进行。如果你正在使用 WordPress，有一个很棒的工具叫做[编辑日历](https://wordpress.org/plugins/editorial-calendar/)。如果你正在寻找一些更强大的，也有助于社交媒体日程安排的东西，那么看看[协同日程](https://wordpress.org/plugins/coschedule-by-todaymade/)。这是一个付费工具，但可以节省你很多时间。

另一个好的选择是使用特雷罗。使用 Trello，您可以创建特定的板，在那里您可以保存内容想法，跟踪生产和截止日期，并向每张卡添加文件。如果你有一个团队，或者多个作者，你背后的这个工具是非常宝贵的。

## 2.内容营销目标设定

你的内容策略的目标会影响你正在创建的内容的类型。例如，你是否试图建立一个更大的社交追随者？驱使读者阅读你的邮件列表？得到媒体报道？提高自己的权威？

一个合理的战略将在两个方面发挥作用。实现你的商业目标，同时为你的读者提供大量的价值。事实上，唯一会积极建立你的业务的内容需要在这两个方面运作。

幸运的是，解决这两个因素的核心从你刚刚进行的研究开始。

由于内容营销有一个长期的重点，你将看到的一般目标将很难量化，如建立品牌意识，增加我们的权威，等等。但是，您可以将这些分解成易于量化的指标，让您知道您的策略是否有效。

一些与增长相关的指标可能是:

*   每条内容的社交份额提高
*   参与度或评论总数增加
*   增加目标关键词的搜索引擎排名
*   网站流量稳步增长
*   电子邮件列表增长

## 3.创造史诗内容

你的策略的成功将完全取决于你所创造的内容的种类。你不能简单地创造足够好的内容。

关联很简单。制作蹩脚的内容，人们会认为你的生意很糟糕。创造令人难以置信的、有用的、免费的内容来帮助人们，他们会更有可能购买你的付费产品。你不能在低质量内容的基础上建立业务。

一个典型的例子是拉米特·塞西。他在所有垂直领域制作的内容是关于给定主题的最佳信息。他的免费内容比大多数人收费的要好。这有助于建立大量的信任，人们知道在购买的时候他们会去找谁。

既然我们在上面提到了关键词，很多人仍然认为这给了他们写充满关键词的帖子的自由，这些帖子没有任何价值。记住，我们面对的是真人。你只有一次机会对你的内容留下第一印象。明智地使用这个。

写出真正为你的读者提供持久价值的内容，排名就会随之而来。

## 4.推广才是王道

一旦你发布了一篇文章，你的工作才刚刚开始。事实上，社交触发器公司的德里克·哈尔彭建议你只花 20%的时间创造内容，剩下的 80%用于推广。

也许有上百种方法可以推广你的内容(也许有链接)，但是下面的列表会给你一个很好的开始:

*   在你现有的社交媒体账户上分享你的帖子。确保提到你在文章中强调的任何其他公司或个人。
*   使用相关的内容联合网站来上传你的文章的一个版本，或者完整的文章。现有的一些平台有 [Medium](http://Medium.com) 和 [LInkedIn Pulse](https://www.linkedin.com/pulse/discover) 。
*   回到 Quora，找到任何与你的帖子相关的问题。回答问题，并添加一个链接到您的帖子以获取更多信息。
*   给你在文章中提到的任何有影响力的人发电子邮件，让他们知道你在最近的博客文章中提到了他们。
*   找到任何与你刚刚发布的帖子相关的帖子，并通过类似于 [Open Site Explorer](http://opensiteexplorer.com) 或 [SEOProfiler](http://seoprofiler.com) 的工具运行它们，看看有哪些网站链接到了那个帖子。然后，联系网站所有者，告诉他们你发表过类似的帖子，如果他们感兴趣的话。

有效的外联将有助于给你的职位插上翅膀。如果你想更进一步拓展业务，那么看看 Hubspot 的这张[精彩信息图](https://blog.hubspot.com/marketing/anatomy-email-outreach#sm.00000jjaxhh9modr7vsksat7i3hgk)

## 5.测试，然后再测试一些

衡量每篇文章的成功和衡量标准是你判断你所做的是否真的有效的唯一方法。一旦你在上面的步骤中创建了你的目标，你可以通过一个简单的工具来跟踪你的指标，比如 [Google Analytics](http://analytics.google.com) ，或者你的电子邮件营销软件，看看你的内容是否对你有用。

有了这些数据，你就能看到哪种内容对你的读者来说表现最好，哪种风格的帖子导致了最多的社交分享，以及你的内容是否真的增加了你的电子邮件列表或收入。

随着时间的推移，这些数据将有助于为您的策略指明正确的方向，因此您永远不会猜测下一步要创建什么样的内容。

内容营销失败的唯一方法就是停止生产内容。

因为内容营销是一项长期战略，你越早投资，你就能越早看到你的业务成果。尽管我们消费内容的方式会随着时间的推移而改变，但作为一种有效的集客营销策略，它仍然是必要的。
