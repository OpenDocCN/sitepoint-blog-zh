# WordPress 3.1 有什么新功能？

> 原文:[https://www.sitepoint.com/wordpress-31-whats-new/](https://www.sitepoint.com/wordpress-31-whats-new/)

*这篇文章已经过时了，[点击](https://www.sitepoint.com/whats-new-in-wordpress-4-6/)查看 WordPress 4.6 的新功能。*

WordPress 是网络上最成功的内容管理系统。它可能不是最优雅或写得最好的，但有一个压倒一切的原因，为什么超过 13%的前 100 万网站使用它:它很容易。WordPress 对最终用户来说很简单，重要的是，对开发者来说也很容易。这导致了像这样的主题、插件和文章的爆炸式增长。

WordPress 3.1 在几天前发布了，所以让我们来看看它的新特性和改进…

## 新管理栏

最明显的增加是新的管理栏。从“个人资料”屏幕，你可以设置工具栏出现在管理屏幕或网站的顶部(假设你的主题运行了 [wp_footer()](http://codex.wordpress.org/Function_Reference/wp_footer) 函数)。

就我个人而言，我不觉得管理栏特别有用，但第三方插件扩展其功能只是时间问题。

## 帖子格式

标准的 WordPress“页面”(静态内容而非文章)可以使用不同的模板来区分网站的不同部分。到目前为止，改变帖子风格的唯一方法是检查父类别。

WordPress 3.1 引入了帖子格式:可以在主题中使用的元信息来定制帖子的呈现。例如，最新的 Twenty Ten 主题定义了可以在帖子编辑面板上选择的标准、旁白和图库。

帖子格式可能是那些提供一系列可能性的低调特性之一。

## 内部页面链接

现在链接到你网站内的其他 WordPress 支持的页面更容易了:

我很惊讶这个设备没有早点出现。这是一个受欢迎的增加，尽管我希望 WordPress 开发者增加相对链接或者省略域名的选项。

## 杂项改进

180 个人向 WordPress 3.1 提交了 2000 多条代码。其他改进包括:

*   流线型的界面。默认情况下，很多不常用的选项都被隐藏了——WordPress 3.1 感觉更快更流畅。
*   管理面板的刷新蓝色方案(尽管灰色方案看起来更好)。
*   新的网络管理屏幕从多站点网络的常规面板中删除了超级管理菜单。
*   改进的数据导入和导出。
*   超过 [820 个 bug 修复](http://core.trac.wordpress.org/query?status=closed&resolution=fixed&milestone=3.1&group=status&desc=1&order=priority)。

大多数人只需点击几下就能升级 WordPress，但是为了安全起见，记得备份你的数据库和文件。
