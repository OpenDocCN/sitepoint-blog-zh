# 一切都是为了乳房，宝贝。

> 原文：<https://www.sitepoint.com/its-all-about-breasts-baby/>

好了，伙计们，我们该认真谈谈乳房了。乳房是(字面上)贴近我心的东西，没有什么比失去它们更让我心碎的了。不幸的是，这是如今太多女性的现实，所以在这个月的 [SitePoint 论坛](https://www.sitepoint.com/forums/) [上，我们都在谈论乳房](https://www.sitepoint.com/forums/showthread.php?t=704531)。我们正在尽我们所能支持乳腺癌研究，最简单的方法就是支持十月粉红运动。这些天来，和许多人一样，我的生活被疾病触动了。几年前我差点失去一个朋友，那个故事有一个美好的结局，但太多的人没有。所以，支持这场运动，通过把你的网站涂成粉红色来表示你的支持吧！如果你这样做，我们会让你打破我们的自我推广规则，并张贴一个链接到您的网站，这样我们就可以看到你做了什么。如果改变你的网站是不实际的，你可以通过改变你的论坛头像到我们的十月特别粉红头像之一来显示你的支持。

目前在我们社区发生的最令人兴奋的事情之一是我们的[社区书](https://www.sitepoint.com/forums/showthread.php?t=680804)的发布。这本书是论坛成员之间的合作，提交材料将于本周到期。到目前为止，我读过的几本书都很精彩，我对这种发展方式感到非常兴奋。我很荣幸能写介绍，从我目前所看到的，我将是一个让这个项目失望的人！如果这是你第一次听说社区书籍，而且听起来像是你想参与的事情，我们仍然有一些空间，所以如果你有时间立即开始写作，请联系论坛的工作人员以获得更多信息，或者[在这个帖子中发布你的问题](https://www.sitepoint.com/forums/showthread.php?t=680804)。
