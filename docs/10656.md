# 7 点电子邮件营销清单

> 原文:[https://www.sitepoint.com/email-marketing-checklist/](https://www.sitepoint.com/email-marketing-checklist/)

有许多不同的哲学可以应用于电子邮件营销，一个没有必要比另一个更好。事实上，你可能会发现，在你自己的电子邮件营销活动中，你在不同的时间出于不同的原因使用了许多不同的方法。

不管你如何精心制作你的信息，有几个项目结合在一起，使我认为是一个“必须拥有的清单”，为每一个电子邮件营销活动。

这里有 7 个项目应该列入你的电子邮件营销清单，并且应该在你每次活动点击发送之前核实。

## 1.你已经计划好了整个活动。

在你开始任何电子邮件活动之前，你应该决定你的电子邮件的频率，主题和它们如何一起流动，你将如何跟进，等等。这些都应该在你写第一封邮件之前想好。

## ![](../Images/93baab55daefc605ee75b032aed0ee9a.png "check") 2。您的电子邮件将会进入正确的列表。

如果您有多个列表，请确保电子邮件发送到正确的列表。你最不想做的事情就是把错误的信息传递给错误的受众。

## 3.您的邮件有一个有效的主题行。

你发送给你的列表的每封邮件都应该有一个描述性的和相关的主题，这将激发收件人打开它的好奇心。就像你的文章会经过多次修改一样，你的主题也应该如此。

## 4.你把它打印出来并审阅了。

通过将邮件打印出来并在纸上查看，你会从一个不同的角度来看它，这将帮助你找出你可能在屏幕上错过的错误。你永远不知道，你的一些名单可能会打印出他们的电子邮件阅读，所以这将让你知道你的电子邮件看起来像硬拷贝。

## 5.您提供了邮件的纯文本版本。

大多数电子邮件营销平台会自动将 [HTML 电子邮件](https://www.sitepoint.com/article/html-email-design)转换为纯文本版本，这样那些喜欢纯文本而不是 HTML 的收件人，或者那些在不显示 HTML 的设备上访问的收件人，仍然能够看到你的消息。然而，由于这些转换中的大多数在格式方面都有所欠缺，所以您应该总是进入并清理纯文本版本。

## 6.您的所有链接都已经过测试。

您应该始终测试电子邮件中的所有链接，以确保它们是正确的和有效的。没有什么比把收件人指向一个断开的链接更尴尬的了。一旦发生这种情况，你很快就会失去他们的注意力和再次点击的意愿。

## 7.您测试了您的电子邮件与电子邮件客户端的兼容性。

就像你测试任何网站的跨浏览器兼容性一样，你应该测试你的电子邮件与所有主要电子邮件客户端的兼容性。我使用加盐服务的石蕊进行测试。它节省了我大量的时间，并提示我我的观众可能会看到什么。
