# 设计师垃圾邮件

> 原文:[https://www.sitepoint.com/designer-spam/](https://www.sitepoint.com/designer-spam/)

一个人如何处理他们每天的电子邮件，这些邮件充满了廉价的伟哥、身体部位放大和来自可疑来源的药物的承诺？如果下面的这些设计师和艺术家是可靠的，我们应该和他们一起创造美丽的艺术和设计。

我第一次遇到一些好看的垃圾邮件是在设计师兼插画师[的](http://www.linziehunter.co.uk/) [Flickr](http://www.flickr.com/photos/linzie/sets/72157602417089145/detail/) 片场，林齐·亨特有一个别出心裁的想法，从她收到的许多垃圾邮件中选取一些，用她独特的字体来展示它们。她的作品如此成功，以至于产生了一系列商业印刷品。
![linzie hunter](../Images/fd101bf25293818c7d9657191ec33890.png "linzie hunter")

设计师 [Elliot Burford](http://www.elliottburford.com/) 也受到了[垃圾邮件](http://www.elliottburford.com/index.php?/illustration/spam-new/)主题的影响，设计了一组搞笑的图片，也有 t 恤出售。它们几乎让垃圾邮件看起来很迷人。

![elliot burford](../Images/18cec5ff4518226da6486e00c63f5a19.png "elliot burford")

更技术性一点的是亚历克斯·德勒古列斯库的作品，他是一位罗马尼亚视觉艺术家，创作了一系列“垃圾植物”。在他收到的垃圾邮件文本中发现的 ASCII 值决定了垃圾邮件植物的属性和质量。你可以在他的网站上找到更多这样的作品。

![alex dragulescu](../Images/626174a4560e9d9e275e70ad9947727c.png "alex dragulescu")

荷兰设计师 [Enkeling](http://www.enkeling.nl/category/typography/cover-versions/spam-series/) 从他自己的网站上获取评论，创作出漂亮的印刷插图。有些评论是垃圾邮件，有些是他从网站访问者那里收到的善意话语。

![enkeling](../Images/d2e0f00e697ebe5bc2ab97ed7e50670f.png "enkeling")

最后，如果你想对自己的垃圾邮件做些什么，而不是删除它，你可以将它发送到[垃圾邮件回收](http://www.spamrecycling.com/)网站。

这是我早些时候制作的一个，以我收到的鲍勃·加切尔的一封电子邮件为特色，承诺向我展示“成为互联网百万富翁的简单方法(这不是开玩笑)。”当你向 spamrecycling.com 发送垃圾邮件时，他们会给你发回一个链接，在那里你可以看到你的垃圾邮件艺术正在被创建，并可以影响颜色。

![spamrecycling](../Images/168a9534ae1ed110db1666eb7fb0441f.png "spamrecycling")

你如何看待设计师垃圾邮件？是创造性的浪费，还是创造性的浪费时间？
