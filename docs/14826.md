# 论坛软件综述

> 原文:[https://www.sitepoint.com/forum-software-roundup/](https://www.sitepoint.com/forum-software-roundup/)

“论坛”或“讨论板”在大多数成功的网站上都起着重要的作用，为常客提供了一个讨论网站主题的地方。这创造了一个社区，增加了你网站的“粘性”,激励人们一次又一次地回来。看一看成千上万个具有论坛的网站，您会发现各种系统被用来提供讨论板功能—发帖、投票、私人消息和主题类别，所有这些都是大型论坛系统的标准功能列表的一部分。

在本文中，我将下载、安装和设置 5 个最大和最流行的论坛应用程序。

我是根据在 Hotscripts.com 的清单以及我自己过去的经验和知识来选择系统的。我只选择了免费系统，因为这是大多数网站所有者使用的系统，也因为获取付费系统副本进行审查的难度。这种方法的结果是，我不得不省去非常流行的 [vBulletin](http://www.vbulletin.com) 。我还选择了 PHP、ASP 和 CGI 系统，让你在决定选择哪个系统时有尽可能多的选择。

本综述包括以下应用。每个都可以从相关网站免费下载:

*   phpBB–PHP
*   [视觉电源板](http://www.invisionboard.com)–PHP
*   [剖面图](http://forum.snitz.com)-ASP
*   [Web Wiz 论坛](http://www.webwizguide.info/web_wiz_forums/default.asp?mode=asp)–ASP
*   [ikon board](http://www.ikonboard.com)–CGI

我将使用相同的过程来检查每个系统:

1.  我将下载、提取文件并上传到我的网络空间。
2.  我将运行安装脚本。
3.  我将浏览配置设置和功能。

我将对一系列不同的申请标准进行评论，这将允许我得出一个最终结论，并在摘要中给出一个总体分数。

##### phpBB

[http://www.phpbb.com](http://www.phpbb.com)

phpBB，这个为我自己网站上的论坛提供动力的系统，已经存在很长时间了。自从 1.0.0 于 2000 年 12 月发布以来，2.0.x 版本已经走过了漫长的道路。随着 PHP 脚本语言的发展和各种特性的增加，phpBB 现在有了一个非常广泛的特性列表，其中包括一个基于 Smarty 系统的非常直观的模板(样式)系统。

它支持所有主流数据库，包括 MySQL、MSSQL，甚至 MS Access 数据库。除了这些基本功能，它还提供强大的安全性、轮询、用于格式化的 BBCode 标签、用户组和权限系统，以及头像、签名、配置文件选项、审核和管理功能。

由于自由软件的本质，一些论坛不提供免费支持。然而，phpbb.com 论坛上的帖子通常会在几个小时内得到回复——专门的支持团队成员总是在旁边，热情的成员总是愿意免费提供帮助。也就是说，你应该知道没有付费支持，所以如果没有人知道你的问题的答案，那么你就被困住了。

phpBB 速度非常快，并带有可选的缓存工具，使它更快。它是基于 PHP 的，是完全开源的，可以免费获得，没有任何限制。这使你可以轻松地将它集成到你自己的网站中，并开发风格和修改——有数百种，大部分可以免费下载。phpBB 默认可用的原始 subSilver 样式是所有论坛系统中最好的默认样式之一，他们甚至为您提供了 Photoshop。psd 文件，以便您可以修改该样式。

与其他应用程序中提供的管理控制面板相比，这是 phpBB 的弱点。您可以在这里配置 phpBB 中的每个选项，但是，该系统不提供其他系统所具有的动作日志、高级邮件列表和备份工具等标准功能。也没有办法在不修改的情况下添加新的配置文件选项，也不能创建自己的帖子图标或论坛图标。访问者统计报告仅限于用户和帖子的平均数量，与其他一些系统提供的统计报告相比，这是很糟糕的。但是，你可以自定义面板的外观，使其与你的主论坛风格相匹配。

![1210_forums1](../Images/480605543f8a9ace3cef27ab0427f367.png)

自 2.0.x 发布以来，phpBB 的开发团队并没有给它增加多少新特性。他们似乎只是为了修复 bug 或安全问题(这种问题很少)，似乎是为了即将到来的 phpbb2.2 保留所有新功能，这些功能现在是其他论坛应用程序的标准功能。该系统处于里程碑 2(共 4 个)，但许多主要功能已经开始工作。然而，由于这个版本尚未发布，我无法在此进行评论。你可以在开发网站[area51.phpbb.com](http://area51.phpbb.com)下载当前版本的简历。

phpBB 的安装非常简单。下载的. zip 文件只有 657kb。gzip 可用)。然而，解压缩文件会将文件大小增加到 2mb 以上，您必须将文件上传到您的网络空间。上传完所有文件后，浏览到上传文件的目录，等待安装脚本自动加载。该装置由两个屏幕组成。首先，您必须输入关于您的数据库和默认用户名和密码以及一些位置设置的基本细节。第二个安装屏幕告诉您已经创建了数据库表。

![1210_forums2](../Images/d7ce1936a5f8802990834a46eed7913b.png)

然后，您需要通过上传脚本为您创建的 config.php 文件来完成安装—如果您愿意，您可以简单地让安装程序为您完成安装。一旦你这样做了，你需要删除 2 个目录才被允许访问论坛。然后你会看到一个全新的空白论坛。

总的来说，phpBB 是一个非常受欢迎的非常好的系统。它拥有维护一个活跃论坛可能需要的所有功能，但是它没有提供标准的“额外功能”,这些功能虽然不是必需的，但是很有用。如果你陷入困境，这个巨大的社区是非常宝贵的，它的风格和修改的数量可以让你改变软件来适应你的网站和需求。

***汇总***

**语言:** PHP
**数据库支持:** MySQL、PostgreSQL、MSSQL、MS Access
**可用修改/样式:**数百

**总体:** 8/10
**优点:**非常容易使用，漂亮的默认风格，许多风格和修改
**缺点:**管理面板缺乏“标准”功能

##### 视觉电源板

[http://www.invisionboard.com](http://www.invisionboard.com)

另一个流行的 PHP 应用程序，IVB 提供了一些非常先进的标准功能。从网站上下载文件需要一些时间，因为你必须浏览不同的屏幕；由于下载被列为“免费试用”,定位下载也可能会令人困惑。还可以选择从 Invision Power Services 购买支持。

提取后。zip 下载，您会看到一个混乱的目录结构。你要上传的文件的位置并不明显，但是一旦你找到了，你只需要把它们上传到你的网站上。所有文件的初始上传都很大——超过 4mb——一旦你上传了它们，你不能只是导航到目录并让安装开始。相反，您必须导航到一个特定的文件。精心呈现的安装屏幕会引导您完成安装的每一步。您需要做的第一件事是 CHMOD 一个配置文件，以便脚本可以写入它。输入数据库信息后，会创建表和一些文件。然后，您可以使用您创建的用户名和密码登录到您的论坛系统。

![1210_forums3](../Images/5c51c9402654e77003b8efa2ba3108b1.png)

除了标准的论坛功能外，IVB 还有许多有用的功能；这些包括广泛的私人信息系统，一个完整的用户控制面板，简单的造型设施，宏设施，日历，生日，和 SQL 调试功能。IVB1.2 的两个新特性包括多重调节和警告系统。多重审核功能允许您一次修改多个主题，例如，同时为所有主题添加一个主题前缀。警告特性允许你为用户的行为提供可见的警告，这在一个繁忙的网站上非常有用。您甚至可以将主题下载成 MS Word 或 MS Excel 格式，存储在您的本地计算机上。您论坛上的成员还可以访问独特的“我的助手”，这使他们能够快速找到常见问题的答案，如“谁是版主？”，并允许他们查看他们最近的帖子。您也可以搜索论坛和帮助系统。

默认风格是蓝色的 chrome 设计——对于默认来说还不错。当然，你可以为你的论坛下载数百种风格和图片，并根据你的喜好进行定制。官方网站，[IB Planet.com](http://www.ibplanet.com/)提供所有适用于 IVB 的皮肤、风格和修饰。

IVB 的管理控制面板是最广泛可用的面板之一。索引页显示您的 IVB 是否是最新的，并显示系统信息，谁登录到控制面板，管理行动的日志，以及一些快速搜索选项。这个管理面板允许你绝对控制 IVB 的每一个方面——从删除版权声明到添加新语言。

![1210_forums4](../Images/03204c9990e6c57ae7e12dafbe09eff9.png)

我唯一的批评是 IVB 可能给普通用户提供了太多的选择！浏览每一个选项需要很长时间，而且网站所有者很难找到他们需要的选项。选项包含在主标题下，主标题可展开以显示备选方案列表；这使得界面更加整洁，尽管不管怎样，还是有点拥挤。

总的来说，IVB 是一个非常强大的解决方案，非常强调将尽可能多的功能打包到标准版本中。最初的下载和解压缩过程相对容易，安装也是如此。该论坛为您可能希望您的论坛做的所有事情提供了强大的支持。管理控制面板非常详细，让您保持对一切的严密控制。IVB 支持论坛似乎非常活跃，并提供非常重要的免费帮助。如果你真的陷入困境，总可以选择从公司购买支持。

***汇总***

**语言:** PHP
**数据库支持:** MySQL
**可用修改/样式:**上百种

**总体:**9/10
T3】优点:广泛的功能集，高级控制面板
**缺点:**杂乱的管理界面，混乱的目录结构

##### Snitz 论坛

[http://forum.snitz.com](http://forum.snitz.com)

自 2002 年 9 月首次发布以来，Snitz Forums 已有超过 151，000 次下载，是一个相当受欢迎的董事会解决方案。但是由于某些原因，基于 ASP 的论坛无法与 PHP/CGI 论坛的功能集相匹配。Snitz 论坛拥有你所期望的标准功能，比如完整的基于网络的管理、无限的论坛、私人论坛、粘性主题和文字审查，但除此之外别无其他。Snitz 论坛的完整功能列表可以在[这里](http://forum.snitz.com/specs.asp)查看。

虽然 Snitz 论坛确实支持修改，但很少有可用的——有限的列表包括签名预览和常见问题管理。这些可以从 Snitz 官方网站和一些小型 Snitz“粉丝”网站下载。不支持任何样式。但是，可以对图形进行基本的颜色更改和修改。也可以通过 Snitz 网站获得一组有限的图片。

论坛的 ASP 文件下载量为 525kb。zip 文件，可以提取 3mb 多一点的文件。没有目录结构，所有文件都位于同一个目录中。您必须按照 readme.html 文件中提供的说明来安装论坛系统，包括打开和编辑 config.asp 文件来设置数据库连接信息。完成后，您打开一个设置页面并配置论坛，然后您可以加载论坛并登录。

![1210_forums5](../Images/fd03c4645a4108c399c93601dd643179.png)

配置选项与软件的功能相匹配。您可以配置论坛中的每个选项，包括论坛类别和论坛、徽标——您甚至可以禁用注册。颜色/字体的改变仅限于名称，而不是 HTML 代码，用户可以使用的颜色列表和例子。没有权限选项(我可以看到)，只有基本的版主控制。当我试图找出如何从管理控制面板添加一个新的论坛/类别时，我意识到你不能。您实际上是在主索引页面上做的！

![1210_forums6](../Images/83f0d2d3a8a33d0e74caa23918e5924f.png)

总的来说，与 phpBB 和 IVB 相比，Snitz 论坛很差。然而，这可能是由于 ASP 的局限性，而不是缺乏添加这些功能的愿望。但与此相反的是，付费的 IdealBB 看起来似乎可以与 PHP 系统相匹配。

***汇总***

**语言:** ASP
**数据库支持:** MS Access、MSSQL、MySQL
**可用修改/样式:**无样式特性，修改很少

**整体:**4/10
T3】优点:ASP 的基本论坛功能
**缺点:**基本功能，没有造型系统，修改很少

##### Web Wiz 论坛

[http://www.webwizguide.info/web_wiz_forums/](http://www.webwizguide.info/web_wiz_forums/)

另一个 ASP 论坛解决方案 Web Wiz Forums 有一个相当大的功能列表，这使得它成为一个流行的 ASP 讨论板应用程序。完整的功能列表包括头像图库、图像和文件上传、样式/皮肤支持、强大的权限系统、私人消息、统计系统等功能。这意味着 Web Wiz Forums 几乎是 IVB 和 phpBB 的平等竞争者。

最新版本(7.01)于 2003 年 4 月 27 日发布，有两个版本可供选择:Access 2000 和 SQL Server 7.0/2000。Access 版本更受欢迎，但 SQL Server 版本是更高性能的系统，为高流量板做了一些调整。其他可用的下载包括 2 个头像画廊。Web Wiz 网站还提供了深入详细的文档来帮助用户安装、管理和排除软件故障。

此外，还有大量的修改和附件可供选择。这些也可以在 Web Wiz 网站上找到，范围从最新的新闻 Mod 到照片库 Mod。虽然没有 phpBB 那么多，但它们是可用的，并允许您根据自己的需求定制论坛。皮肤(Web Wiz 论坛称之为皮肤)也受到支持。虽然目前只有 10 个皮肤可用，其中一个将使论坛看起来像默认的 IVB 风格！Web Wiz 网站上所有可用的软件都是免费的。然而，你可以通过 WorldPay 系统付费移除链接。

下载完 988kb 后。zip 文件并提取其内容，所有需要上传的文件都可以在/forum 目录下找到；文档也是可用的。大约有 3mb 的文件要上传到您的网络空间，一旦您上传了这些文件，您不需要修改任何文件，甚至不需要运行任何安装脚本——Web Wiz“开箱即用”。

![1210_forums7](../Images/ce26170ee05d4ca0052b820d24e35b32.png)

默认风格并不惊艳。它以灰色外观和菜单选项图标为特色。个人资料页面看起来很像 phpBB 个人资料页面，管理区域也是如此。管理区比 Snitz 论坛区更全面。同样，它允许你修改论坛的每一个方面，建立新的类别，查看论坛统计数据，并向成员发送电子邮件。

![1210_forums8](../Images/1433562421db397d0df720c374fefbb5.png)

总的来说，Web Wiz 论坛系统可能是免费 ASP 论坛中功能最丰富和最广泛的，它支持皮肤和修改，以及一些在其他论坛应用程序中只能作为附加功能或修改功能使用的标准功能。Web Wiz 网站论坛是您可以找到该应用程序支持的地方。然而，他们似乎不像其他支持论坛那样繁忙。尽管有故障排除文档，但没有其他方法可以获得支持。

***汇总***

**语言:** ASP
**数据库支持:** MS Access，MSSQL
**可用修改/样式:**少量可用样式或修改

**总体:** 6/10
**优点:**良好的功能列表和管理区
**缺点:**样式/修改数量较少

##### Ikonboard

[http://www.ikonboard.com](http://www.ikonboard.com)

Ikonboard 在过去的几个月里有超过 50，000 的下载量，是目前最受欢迎的系统之一。Ikonboard 3.1 由 CGI 提供支持，提供了一整套功能，这些功能可以与其他应用程序中可用的功能相媲美，有时甚至超过它们。它的主要功能包括完整的语言支持，广泛的皮肤支持，SSI 启用的模板(用于包含您的网站的其他部分)，高级权限，控制面板中的图形和颜色修改选项，以及新闻管理。

除了 CGI 版本，您还可以获得一个托管版本、一个 PHP 版本和一个轻量级版本。其中大部分仍在开发中，但你可以通过[导游](http://forums.ikonboard.com/tour.php)了解更多。

还有一些专门的网站提供 IB 皮肤和修改。皮肤站点包括 50 多个 IB3 皮肤，以及 200 多个图片供下载。修改或“黑客”也可以通过 IB 网站的专门部分获得。

对应用程序的支持似乎是最好的。通过注册会员专区，您可以访问优先支持票证、支持论坛、常见问题解答和实时聊天，所有这些显然都是免费的。

下载 1.16Mb 系统不需要在会员区注册，但建议您注册以获得支持。其他下载包括旧版本和图形开发包。你必须上传超过 3.5 兆的文件到你的 CGI 网络空间。提供的说明很全面，但是对于新用户来说很容易混淆。在 Windows 上安装更容易，因为不需要更改代码；但是，您必须在 Unix/Linux 上对各种文件进行 chmod。

然后使用安装屏幕安装各种文件，以便 Ikonboard 可以运行。我选择使用 MySQL 数据库来存储数据，尽管其他可用的选项包括 PostgreSQL、Oracle 甚至 DMB 数据库。

![1210_forums9](../Images/d93a90ed5f292f551eb89386727dad5c.png)

论坛的默认风格相当不错——灰色/蓝色，带有 chrome 类型的按钮。在底部，它包括一个“邀请一个朋友到论坛”的选项，这很有趣，是一个很好的功能，可以帮助你推广你的论坛。控制面板允许您设置各种选项来发布和修改您的个人资料，它分为不同的部分，这使它保持整洁。我对这种默认风格的唯一问题是，三个搜索/成员/帮助按钮在右上角看起来不合适，因为它们是整个界面中唯一的图形按钮。

像 IVB 管理区一样，通过选择类别名称可以扩展旁边的选项，这也保持了界面的整洁。IB 管理控制面板的整体外观干净整洁。这里有许多选项允许您更改功能——从数据库管理到设置新闻功能。

![1210_forums10](../Images/1cb5ac7aa0096fb4d58772ef518f0560.png)

总的来说，IB 是一个非常好的应用。尽管它使用 CGI，但速度几乎没有明显下降，尽管管理区域确实感觉有点迟钝。该软件提供了一个非常好的特性列表，很容易匹配 PHP 系统。然而，安装有点复杂(我必须通过大约 6 个屏幕才能访问论坛)，管理控制面板皮肤可以改进。

***汇总***

**语言:** CGI

**总体:** 9/10
**优点:**你可能需要的所有功能
**缺点:**轻微的管理区减速，糟糕的管理区颜色

##### 结论

还有数百个其他论坛应用程序可用。我回顾了最受欢迎的，但这些可能不适合你的网站。我发现 ASP 论坛系统似乎提供的功能最少。考虑到 PHP 和 CGI 论坛应用目前超过了任何免费的 ASP 系统，这对于 ASP 网站所有者来说是令人失望的。

除了这些基于 HTML 的论坛，还有一些 Flash 论坛正在开发中——有些甚至现在就可以使用。时间会证明这些产品是否成功，但只要它们能与本文所讨论的应用程序所展示的特性相匹配，就没有理由不成功。
