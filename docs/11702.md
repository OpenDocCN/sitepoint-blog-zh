# 易贝对 Skype 的抛售

> 原文:[https://www.sitepoint.com/ebays-sells-skype/](https://www.sitepoint.com/ebays-sells-skype/)

易贝已经卖掉了他们在 2005 年购买的网络电话系统 Skype 的 65%。当时，易贝出人意料地收购了这项服务，出价高于谷歌和雅虎，支付了 31 亿美元。如果收购后 Skype 的业务没有停滞不前，这笔交易还会花费 10 亿美元。

Skype 拥有超过 4.8 亿注册用户，在全球国际通话分钟数中占有 8%的市场份额。易贝曾计划通过让买家和卖家互相交流来增加拍卖收入。他们希望 Skype 能帮助处理复杂的交易，比如汽车和高端商品的交易。然而，易贝最近得出结论，这次收购是不明智的，不适合他们的业务。

该公司计划在 2010 年让 Skype 上市，但由于与联合创始人尼可拉斯·曾斯特罗姆和贾纳斯·弗里斯的法庭斗争，公开募股变得黯然失色。易贝可能会失去一些点对点 VoIP 技术的许可，并正在开发自己的软件版本。

Skype 的潜在买家包括谷歌，但 65%的股份最终被银湖合伙公司(Silver Lake Partners)领导的一个财团收购。投资者包括加拿大养老金计划投资委员会和 Andreessen Horowitz，这是一家由网景创始人 Marc Andreesen 领导的新风险投资公司。有传言称，此次合作将为 Zennstrom 和 Friis 提供一个解决法律问题的投资机会。

虽然谈判仍在继续，但银湖可能会为他们的股份支付约 17 亿美元。这使 Skype 重新估值为 26 亿美元，相当于每个注册用户 5 美元多一点。
