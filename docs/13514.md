# 第一集:寻宝游戏！

> 原文:[https://www.sitepoint.com/scavenger-hunt/](https://www.sitepoint.com/scavenger-hunt/)

让我们从一个有点不寻常的事情开始:一个虚拟的寻宝游戏。

在某种程度上，几乎每个网络极客都有机会侵入一些公开数据，通常来自政府来源。这里的流行词是“mashup”，但知道如何找到和使用公开可用的数据在它的流行结束后仍将是一项有价值的技能。

不幸的是，政府，尤其是美国政府，在提供这些数据方面往往糟糕得令人难以置信。当然，它是可用的——但你必须先找到它。

所以这个问题就是要找到这些数据。因为我最熟悉美国，这个问题是美国特有的(但是我希望看到适用于其他国家的任何问题的答案)。

在每种情况下，答案都应该是一个 URL，您可以在其中下载相关数据，或者至少找到数据的直接链接。每一个都可能有多个来源，包括可以通过屏幕抓取的数据。然而，我并不是在寻找这些资源，只是寻找那些容易下载的数据，而且数据格式也容易被计算机解析(例如 CSV、XML、纯文本)。换句话说，就是“友好的”格式。

那么，我可以从哪里下载数据:

1.  分析食物的营养成分？
2.  查找我所在城市的人口(和其他基本人口统计数据)？
3.  分析上市公司提交的最新 SEC 文件？
4.  看看历史油价？
5.  寻找青少年逮捕率的趋势？

把你的答案贴在评论里。为了获得额外的加分，请告诉我们你是如何找到每一条数据的——是谷歌为你提供了很好的服务，还是你被迫转向其他地方？

如果你真的想开动脑筋，那就试着写一个工具，把每一大块数据导入你喜欢的关系数据库。几周后会有一个相关的问题，涉及到对这些数据中的一个进行建模，所以你们这些优等生现在可以开始考虑了…

祝你好运，这个周末回来看看答案。
