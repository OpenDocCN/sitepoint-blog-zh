# 生动的灯光混合模式

> 原文：<https://www.sitepoint.com/vivid-light-blending-mode/>

Photoshop 将生动的灯光混合模式描述为结合了颜色减淡和颜色加深模式的效果，如下所示:

> 鲜艳的光:通过增加或减少对比度来加深或减淡颜色，具体取决于混合颜色。如果混合色(光源)比 50%的灰色亮，则通过降低对比度使图像变亮。如果混合色比 50%灰色暗，则通过增加对比度使图像变暗。

然而，在现实中，生动的光使图像变亮，类似于颜色减淡模式，但不像颜色加深那样变暗。在下图中，我创建了两个图层，一个浅灰色的盒子，一个深灰色的盒子，并将前者设置为颜色减淡，后者设置为颜色加深。你可以看到颜色减淡层使图像明显变亮，颜色加深层使图像明显变暗。但是在第二个例子中，我把两个盒子放在同一个图层上，设置为生动的灯光。与颜色减淡相比，亮框确实会产生较亮的图像，但与颜色加深相比，暗框的对比度明显较低。需要记住的是，鲜艳的光模式将使用混合层，使亮的区域比暗的区域更亮。

对于我的实际应用，我将使用生动的光模式来覆盖其他图像上的文本。因为“生动的光”并不真正“显示”混合层，而只是使用混合层来影响基础层的颜色，所以文本看起来更像是图片的“一部分”，而不是图片顶部的一个稍微透明的层。

我先从一张照片开始，那是悲剧发生几天后，我在纽约的 911 纪念会上拍的。有人制作了数百只纸鹤，并将它们串在一起:

![](img/a6b3f4b635437b20b8ee231f561f8ada.png)

因为这张图片是如此的杂乱，我决定把它更多的用作背景，并试图找到另一张图片作为主要焦点。我在 Comstock 的网站上找到了这面旗帜，并在底部添加了轻微的蓝色渐变，以淡化旗杆。

![](img/d682b42d86a3810bb7887a60789fd8e4.png)

我在鹤层上设置了旗帜层，并设置为强光。(回想一下，强光会保留顶层的阴影/高光。)我喜欢天空最后把鹤染成蓝色的方式，并决定保持这种方式。

![](img/1bc500737cefa23366c73549ea924987.png)

我花了多一点的时间来弄清楚我想要添加到图像中的文本。我首先在图片底部添加了“我们永远不会忘记”。暗灰色一旦被设置为明亮的光线，就会“烧进”图像的底部。然后我添加了一个浅灰色的“9-11 ”,并旋转它以跟随旗帜的角度。设置为生动的光，这将减轻图像的一部分。最后，我认为底部的“我们永远不会忘记”文本过于微妙，不能作为主要文本，所以我在中间添加了一个更淡的“记住”。

![](img/4bd7ffdd462d28c5601938e5b129aa82.png)

将这三层置于明亮的光线下，我完成了最后的作品:
