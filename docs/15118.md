# 专业设计的 5 条黄金法则

> 原文:[https://www.sitepoint.com/rules-professional-design/](https://www.sitepoint.com/rules-professional-design/)

网页设计世界充满了兴奋和体验、创造力和技能。如今，自由设计行业的发展意味着即使是新手也有机会为他们的网站设计获得报酬。

这里有 5 个关键技术，每个专业设计师在开发客户网站时都会用到。让我们看看你，一个初露头角的自由职业者，如何为你自己的客户快速简单地实现它们。

##### 1.打击互联网垃圾邮件蜘蛛

通常，一只瘦骨嶙峋的黑蜘蛛会在一个网站上爬行，兴奋地扭动它的小脚，收集网站上所有的电子邮件地址，并将它们存储在它的钳子附近。然后，当它回到主人身边时，它会虔诚地将收集到的所有数据添加到一个巨大的电子邮件地址的巢中，它可以用这个巢作为诱饵来吸引它的客户——直销公司。

这些小小的黑色蜘蛛是软件工具，它们在你的页面表面下滑动，密切关注底层的 HTML 代码，特别是搜索标准格式的电子邮件地址(name@domain.com)。这可能是收集电子邮件地址的最有效的方法——这需要被阻止！我们网页设计师可以运用一些技巧来保护我们的客户免受这些令人毛骨悚然的骚扰。这里有三个可以很好地工作。

***1。使用 HTML 符号***

为了防止蜘蛛抓取电子邮件地址，做一些格式修改来欺骗软件。所有的电子邮件地址都使用 at 符号(@)，蜘蛛被教会去寻找它们。HTML 符号是直接写入 HTML 代码中的符号，由浏览器解释为它们的实际含义。@符号的 HTML 符号是`&#64;`那么，我们如何实现这样的东西呢？简单:

```
<a href="mailto:name&#64;domain.com"> 
```

我们所做的就是用代表它的 HTML 符号替换@符号。然而，这并不是一个万无一失的解决方案:尽管大多数蜘蛛还没有成熟到足以解释 HTML 符号，但有些却是。我们如何超越这个障碍？让我们试试 Javascript。

***2。使用 Javascript***

在这种方法中，我们只需将电子邮件地址分成四个不同的部分:

1.  名字

3.  @符号

5.  领域

7.  延长

这里有一个简单的 Javascript 实现来执行这个操作。

```
<script type="text/javascript">  

name ="you";  

domain ="domain"; 

extension =".net";  

document.write('<a href="mailto:' + name + '@' + domain  

+ extension +' " ' +' >e-mail link here ');  

</script> 
```

上面的代码相当简单明了。我们定义了三个变量:名称、域和扩展名，并简单地将这些变量连接到浏览器的一个`document.write`语句中。请注意，我们在 write 语句中使用了 HTML 的锚标记，以确保用户的浏览器能够正确解释电子邮件链接。

用这种方法还有一个问题…你能想出来吗？Javascript 是一种客户端技术，这意味着尽管蜘蛛可能无法解释你的电子邮件地址，但关闭了 Javascript 功能的浏览器也不能。在这种情况下，您的电子邮件地址将不会显示给用户。

我们如何解决这个问题？我们可以拒绝显示任何电子邮件地址吗？是的，我们可以，但我们仍然必须为用户提供一种联系网站所有者的方式。解决方案是一个相当简单的电子邮件形式。

***3。使用招式***

第三种可能是保护网站所有者免受垃圾邮件蜘蛛攻击的唯一简单的方法，那就是拒绝显示任何电子邮件地址。相反，如果您的服务器支持 Perl、ASP、Cold Fusion 或 PHP 等服务器端处理，只需设计一个反馈表单，并调用服务器端脚本在后台处理数据。蜘蛛挖不了那么深！

我是 PHP 的忠实拥护者，但是选择你的主机支持的任何语言。请注意，如果您的服务器不支持任何服务器端编程语言，那么在这方面您就不走运了——您必须在上述两种方法中做出选择。

##### 2.参加字体测试

一个网站没有内容就什么都不是，大多数内容都是由简单的文本组成的。许多网页设计师忽视了他们的排版，这一点显而易见。合适的版式根据文本在页面上的位置和用途而有所不同。虽然没有一成不变的规则，但是有一些通用的——在某些情况下是基本的——技巧应该遵循，但是经常被忽视。

***标题***

显然，标题应该比正文大 1 到 2 个字号。你可以考虑加粗，但要认识到字母的宽度。例如，Arial Black 可能会创作出对你或你的客户来说过于臃肿的字体。当你使用彩色字体时，要确保颜色与网站的背景颜色形成良好的对比。黑色和深灰色不太好对比，而黑色和白色(或浅灰色)可以相当不错。有时，甚至一个简单的颜色变化也能创造出有用的标题。

此外，一定要层叠标题。例如，主标题会比副标题大。这种效果在信息中产生了一种强调和流动的感觉。

不要用斜体显示标题。斜体字是为了强调特定的内容，但是因为文本是一个标题，而且无论如何都比较大，所以斜体字是多余的，并且经常使文本难以阅读。

***Fonts***

默认的是 Times New Roman，这很好，但是很多人觉得很无聊。我用 Arial、Georgia 和 Verdana 做了实验，发现 Verdana 是其中可读性最好的。这是个人偏好，但是字体应该限制在以上四种，以确保网站所有用户之间的兼容性。如果你的访问者的浏览器不支持你的字体选择，他们的浏览器将恢复到默认。

由于浏览器增加了对 CSS 的支持，只要有可能，就使用样式表来定义你的字体，而不是 HTML 的`<font>`标签。此外，确保你的字体足够大，以便所有年龄和视觉能力的访问者都能阅读。

***重音样式***

避免在文本中全部使用大写字母，因为这会使阅读变得困难，并且暗示你在对读者大喊大叫。

为了强调正文中的单词，决定一种格式并坚持使用。有的用粗体，有的选择斜体或者喜欢下划线。我个人最喜欢的是斜体，但这些都可以很好地工作。但是，使用下划线时要小心，因为这种格式可能会被误认为是链接。确保你不会过度使用这些压力类型。

你的整个正文不应该是粗体。粗体，就像斜体一样，应该用来强调单词，通常在常规风格的文本中，一整篇文章应该是不言自明的。我喜欢使用 CSS 将我的正文隔开一点，以增加可读性——试试看。

##### 3.元标签使你的网站搜索引擎友好

许多搜索引擎仍然使用 Metatag 内容为每个站点创建一个列表，当用户使用特定的关键字搜索引擎时，该列表将会显示出来。通常，专业设计师在他们的网站中至少包括三个元标签:标题、描述和关键词。

```
<META NAME="Title" CONTENT="Title here">   

<META NAME="Description" CONTENT="Description here">   

<META NAME="Keywords" CONTENT="Keywords here"> 
```

元标记的伟大之处在于它们非常容易实现:只需在适当的标记之间键入相关内容。虽然有无数种方法可以提高某个网站在搜索引擎列表中特定关键词的排名，但元标签在搜索引擎优化过程中绝对是必不可少的。

##### 4.提供联系方式

总是，总是提供一些沟通的方法，让用户可以与网站的所有者取得联系。许多网站开发者在他们的网站上包括电子邮件链接和反馈表格。这种方法有两个目的:

*   它告诉用户，运营网站的人关心他们的意见
*   它鼓励访问者分享他们的想法和提出问题，这反过来给网站所有者一个与他们的观众建立融洽关系的机会

那么，一定要鼓励你的客户经常查看他们的电子邮件。对于用户来说，没有什么比等待网站所有者回复他们的邮件一周或更长时间更令人沮丧的了。

如果服务器支持服务器端语言，如 Perl、ASP、Cold Fusion 或 PHP，您可以设计一个反馈表，或者从脚本存储库中下载一个，例如:

*   [Hotscripts.com](http://www.hotscripts.com)
*   [马特的剧本存档](http://www.worldwidemart.com/scripts/)
*   [资源索引(CGI)](http://cgi.resourceindex.com)
*   [资源索引(PHP)](http://php.resourceindex.com)
*   [总脚本](http://totalscripts.com/pages/)

##### 5.设计时要考虑到小家伙

在这个时代，我们想的都是花里胡哨的东西，尤其是当涉及到漂亮的新设计时。这些新技术不仅经常被滥用，而且会大大降低网站的速度，尤其是当你使用大型动画时。gif 和复杂的 Flash 电影。

虽然现在的宽带用户确实比以往任何时候都多，但不要犯专门为这一小部分用户设计的错误。目前仍有 28.8k 调制解调器用户，如果那些使用低端连接的用户(可能是网站访问者的大多数)访问一个页面的时间超过 10 秒，他们将很快转向竞争对手。

作为网页设计师，我们需要保持我们的网站简单。太多的人，尤其是新的设计师，在他们还没有掌握银牌的时候，就已经把他们的眼睛盯在金牌上了。只在需要的地方使用图形。千万不要用。gif 动画，除非他们绝对，不可避免地必要。不要在你的页面上使用大而复杂的背景:它们只会妨碍文本的可读性，增加加载时间。

这个故事的寓意是:设计时要考虑最终用户。假设您的用户使用 Netscape，通过 28.8 调制解调器与分辨率为 800×600 的 15 英寸显示器连接。相信我，你的访问者会因此而喜欢(并支持)这个网站的。

##### 包装它

我们已经看到了每个设计师在做自己热爱的工作时需要牢记的 5 个技巧。虽然沟通是必不可少的，但可以采取措施减少网站所有者收到的垃圾邮件数量。

作为设计师，我们还必须确保我们的字体易于阅读，设计合理。此外，搜索引擎友好的网站通常是受欢迎的网站，那些为小人物设计的网站大大提高了对所有潜在用户的兼容性——这只会让你的客户高兴！

使用这些技巧将会把你的网站提升到一个新的高度，并确保每个访问者都能得到他们想要的内容，并且你的每一个客户都对这份工作感到满意。
