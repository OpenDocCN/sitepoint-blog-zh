# 你周末工作吗？

> 原文：<https://www.sitepoint.com/do-you-work-at-weekends/>

周末到了，对于 It 界的大多数人来说，这是一个受欢迎的从繁重的工作中解脱出来的两天假期。然而，作为自由职业者的一部分乐趣在于你可以在任何时间工作:晚上、假期或周末。我现在就在做！

周末工作有几个好处:

*   很少有工作中断，比如客户的电话或电子邮件
*   加班是为了完成工作，而且
*   你可以完成很多平时拖延的小管理任务(会计、备份等。)

可以理解的是，也有缺点:

*   孩子和家人更有可能打扰你
*   你可能有承诺或错过社交活动，和
*   只工作不玩耍，聪明的孩子也变傻！

一些自由职业者拒绝周末工作。还有的会把周末当做正常工作日。如果他们绝对必须在周一早上之前完成一个项目，我怀疑许多人会工作。

就我个人而言，我不介意在周六或周日多工作几个小时。这感觉就像是“额外奖励”时间，否则它会被浪费在看电视或对孩子们大喊大叫上！我也发现这很有效率——我更容易集中注意力，因为分心的事情更少，我有动力更快地完成工作。
