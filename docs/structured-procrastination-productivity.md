# 结构化拖延:它能让你更有效率吗？

> 原文：<https://www.sitepoint.com/structured-procrastination-productivity/>

拖延是我们在生活和工作中都会遇到的一种瘟疫。一旦你进入了拖延的怪圈，你就很难把自己推出去。但是，信不信由你，如果你能让拖延对你有利的话，拖延也会带来一些好处。

首先，让我们看看拖延的一些原因。

## 拖延的原因

有很多事情会导致拖延，我们每个人倾向于推迟事情的原因可能因人而异。你可能拖延的一些常见原因包括:

*   你有你真的不能忍受做的任务或工作要做。
*   你完全被自己的工作压得喘不过气来。
*   你执着于追求完美并产生完美的结果。
*   你害怕失败(甚至害怕成功)，这让你止步不前。
*   你对你正在做的事情感到非常厌倦。
*   你在等待灵感开始。
*   你高估了完成这项工作所需要的时间。
*   你认为如果你等到在压力下工作，你会取得更好的结果。

拖延症可能来自任何地方，很多时候我们没有预料到它的到来，所以我们不能有效地阻止它。如果它在正确(或错误)的时间袭击我们，这种影响可能是有害的。

## 拖延的负面后果

如果你让你的拖延症打败了你，肯定会有一些破坏性的结果:

*   挫折
*   落后了
*   精神疲惫
*   自信心下降
*   对我们工作的不满情绪
*   焦虑
*   越来越强烈的压迫感
*   对关系的负面影响
*   错过最后期限和失去工作

但是拖延症真的都不好吗？不要！以下是你如何让它为你工作。

## 利用结构性拖延来完成更多工作

当你推迟一项特定的任务时，你会做什么？通常，我们会转向清单上其他我们认为不太重要的事情。所以我们仍然是有效率的，只是不是以我们认为需要的方式。

但是，如果你重新组织你的任务列表，这样你经常拖延的任务就不会在前面和中心嘲弄你，你可以欺骗自己去完成你真正需要完成的事情。

斯坦福大学哲学教授约翰·佩里称之为“结构性拖延症”正如佩里解释的那样，结构性拖延意味着塑造我们任务的结构，因此重要性的顺序欺骗了我们的大脑。基本上，利用拖延者的一个共同的性格缺陷(自我欺骗)来抵消拖延的影响。

例如，如果我们把注意力放在不太重要的任务上，而不是把最重要的优先事项列在清单的顶部，并给予它们最大的关注(以及围绕这些任务的完成的压力)，我们可能会向下移动清单，以找到(并更愿意完成)更重要的任务。

在阅读了关于结构性拖延的文章后，我意识到我已经倾向于无意识地这样做了。我把我所有的任务放在一起，并标记一些绝对不重要的高优先级任务。这让我觉得，如果有几个同等重要的任务，我可以选择是现在处理还是以后处理。因为我对高优先级任务的压力和必要性感觉更少，所以我更有可能马上去做。真的有用！

你有没有整合一些类似结构性拖延的东西来鼓励自己完成更多的事情？这对你有用吗？
