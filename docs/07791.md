下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 9.6%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

我对表格做了一些修改。Firefox 3.7 及以下版本市场占比不到 1.5%；它越来越无关紧要，所以它的数字已经与 Firefox 4+结合在一起。IE6 和 ie7 还会存在一段时间。它们也很低，但是看着这些数字下降给所有的网络开发者带来了小小的快乐。你还应该知道，Chrome 和 Safari 的数据包括桌面版和平板版，但不包括移动版([见下文](#mobile))。

9 月令人震惊的消息:Chrome 的市场份额首次下降。诚然，0.25%的损失可能是一个统计信号，但这是一个通常每月增长 1%的浏览器。Chrome 已经到达了它的自然高原了吗？我不认为是这样，但浏览器有更健康的竞争。

Firefox 也公布了近 1%的大幅下降。浏览器在过去的一年中下降了 4.65%，尽管最近的版本非常好，但人们没有忘记当他们切换到 Firefox 4.0 时带来的升级麻烦。Mozilla 正在努力重新赢得用户的信任，但这需要时间。

九月份最大的赢家是 ie 浏览器。在 Windows 8 和 IE10 发布之前，微软会对这个消息感到欢欣鼓舞。IE9 正在继续缓慢但稳定的增长。奇怪的是，令人担忧的是，IE8 也有小幅增长。这是怎么回事？我怀疑这是一个长期趋势；IE8 的使用在过去一年减半。

Safari 继续小有斩获，这个月过得很好。放弃 Windows 和旧的 Mac 版本不太可能有重大影响，尽管我仍然认为这是一个错误。

Opera 继续在 1.5%和 2.0%之间波动，其百分比在 12 个月内几乎没有变化。这是一个很棒的浏览器，值得拥有更多，但我不相信它能提供足够吸引人的好处来鼓励用户更换。

## 移动浏览器的使用

8 月份，手机使用量略微上升，占所有网络活动的 11.78%。我怀疑当暑假和奥运会结束时，它会开始回落一点。

主要的移动浏览应用程序有:

1.  安卓系统——24.40%(增长 1.26%)
2.  iPhone——20.81%(下降 0.64%)
3.  opera Mini/Mobile——19.34%(下降 0.08%)
4.  诺基亚浏览器——10.18%(增长 0.03%)
5.  UC 浏览器——7.79%(下降 0.40%)

Android 又有一个不错的月份，但总的来说，移动浏览器排行榜出人意料地没有什么变化。

IE、Chrome 和 Firefox 的移动版加起来目前只占不到 2%的市场份额。我怀疑，当微软的 Surface 设备出现，Chrome 成为默认的 Android 浏览器，Mozilla 发布其 Firefox 操作系统时，这些数字可能会开始迅速上升。
