# 草根火狐

> 原文:[https://www.sitepoint.com/grass-roots-firefox/](https://www.sitepoint.com/grass-roots-firefox/)

虽然向像你我这样的技术爱好者宣传火狐很容易，但“广大 IE 用户”毫无疑问并不知道它的存在。

我一直在尝试口口相传，向我认识的人推荐 Firefox，并对结果感到惊喜。我并没有“强行推销”Firefox，只是将人们指向 URL(对于无畏者来说，可能还有 [Texturizer](http://texturizer.net/firefox/) )，对安全性、性能和“不再弹出窗口”说了一些鼓励的话，然后就此打住。

有趣的是那些尝试过火狐浏览器的人(有些人没有烦恼)，所有的人都转变了态度，给我留下了非常积极的评价，说它有多棒，它已经成为他们的首选浏览器。有些人甚至指出了我没有发现的特征，这很好。我说的是“非书呆子”，他们可能不太关心标准遵从或技术问题。

就市场研究而言，这是非常不科学的(事实上，我认为我可以说迄今为止只有 6 个口碑转化者),但迄今为止，火狐的某些东西似乎打动了你的“普通”PC 用户——足以让他们放弃 IE。也许是因为尽管“有限的”技术专业知识，他们也能识别出一个高质量的软件——用户并不像一些供应商所认为的那样愚蠢。

想知道其他人对“宣扬火狐”的反应如何吗？

PS:对，我妈用的是火狐；)
