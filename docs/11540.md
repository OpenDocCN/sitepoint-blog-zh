# 冷打电话是死营销方式吗？

> 原文:[https://www.sitepoint.com/is-cold-calling-dead/](https://www.sitepoint.com/is-cold-calling-dead/)

问我这个问题，我下意识的反应是:“是的，肯定是。”繁忙的日程安排，收到语音邮件的可能性，[拒绝来电登记表](https://www.donotcall.gov)，以及我不是一个喜欢打电话的人的简单事实，我无法想象自己会使用陌生来电作为寻找潜在客户的方式。

但是，话虽如此，我还是要承认打电话推销可能是有价值的，如果:

*   你可以确定一个非常具体的目标受众，并通过电话联系他们(在核实他们没有列在拒收讯息登记册上之后)。
*   你有时间和兴趣打很多电话。
*   在打电话之前、之中和之后，你都有一个计划。
*   你自信、友好、健谈。
*   你已经准备好面对拒绝了。
*   这不是你唯一的营销手段，而是一个更大计划的一部分。

如果你是这个星球上少有的喜欢打电话给陌生人推销产品或服务的人之一，并且一般符合上面列出的标准，那么打电话推销可能对你是一个有益的营销工具。

## 陌生来电提示

以下是一些让你开始打电话的技巧:

*   **确定明确的目标**——在你打电话之前，你需要明确你的目标是什么，这样你就可以在对方失去兴趣之前迅速传达你的信息。
*   **准备好讲稿**——虽然你希望避免听起来像录音，并在电话中注入一些自发性，但你应该有一个你想说的内容的大纲，以确保在电话中触及所有要点。
*   **从现有关系开始**–是的，冷打电话通常是指给陌生人打电话，但你也可以用同样的技巧给你过去交往过的人打电话。
*   **做你的研究**——光有电话号码列表是不够的。花点时间研究一下你计划拜访的每家公司，并相应地个性化你的剧本。
*   微笑——当你微笑时，你的整个声音和语调都会改变，所以在拨号前花一分钟让自己处于正确的心态。
*   **制定跟进计划**——在通话中，确定下一次会面的时间，告诉潜在客户你将向他们发送更多信息，或者询问你是否可以在本周晚些时候再打电话过来。不要等他们采取下一步行动；积极主动。
*   练习，练习，再练习——就像公众演讲一样，你练习得越多，你就会越舒服，你就会越成功。做一些测试，记录你自己并评论你的陈述。
