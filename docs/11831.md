# 如何在 Illustrator 中创建锥形线条画笔

> 原文:[https://www.sitepoint.com/create-illustrator-brush/](https://www.sitepoint.com/create-illustrator-brush/)

我们将看看如何在 Illustrator 中制作一个简单的艺术画笔。锥形线条可以给插图带来可爱的效果，而且很容易制作。

1.创建新的 Illustrator 文档。
2。使用椭圆工具![ellipse](../Images/9caeccfdd9a814ddbb2e7400c6077c2a.png "ellipse")，画一个小圆，任何你喜欢的颜色。确保圆没有描边颜色。按住 Shift 键拖动鼠标，得到一个完美的圆。
T3】

3.使用直接选择工具![direct-selection](../Images/f9cfc11aefabc6937523c3adb0a6040e.png "direct-selection")，点击圆的底部锚点并向下拖动，创建一个更长的锥形，如下图所示。
![drag-out-anchor](../Images/a71090b13e8259bc2f1aac2202131b07.png "drag-out-anchor")

**可选步骤。如果你想要一个非常尖的锥形笔刷，那么选择转换工具![convert-tool](../Images/cd3295996dc3f1bbfa4c15a92f2855c0.png "convert-tool") (Shift + C)，它隐藏在钢笔工具下面。单击一次您拖出的底部锚点。这将点从曲线转换为直点，给你一个更尖锐的点。
![pointy-taper](../Images/e598cc5271858da14f11fb24358d6ce6.png "pointy-taper")**

4.打开画笔调板(按 F5 或选择“窗口”>“画笔”)。

5.点击调色板底部的新建笔刷图标。这将打开“新建画笔”对话框。

![newbrush](../Images/410ff3eaff2352f81fe08ca111c3abf0.png "newbrush")
选择新建艺术笔刷然后点击确定。

![clip_image018](../Images/06049359b19255f5b34d4812121f84a2.png "clip_image018")

6.你的新画笔选项打开。这里我们只需要考虑几个设置，然后就完成了。

a)将画笔方向设置为向上。b)将着色方法设置为色调偏移。这就意味着你可以在使用的时候让你的笔触变成任何你想要的颜色。c)给你的笔刷取个名字。

单击确定。

![art-brush-options](../Images/0ac0198e85b197c1847d8cf2618e856f.png "art-brush-options")

7.现在你应该在你的笔刷面板中看到你的新笔刷，可以使用了。

8.请记住，您可以将画笔描边应用于任何路径，而不仅仅是用画笔工具绘制的路径。所以你可以拖出一个星形，用铅笔乱涂，或者用弧线工具画一条曲线，然后在笔刷面板中点击笔刷笔画，瞧！
![brush-samples](../Images/865b5fed0de27736b9983eb7b547ebfc.png "brush-samples")
9。一旦你对你的画笔满意了，你需要保存它。点击笔刷面板底部的文件夹图标，选择保存笔刷库。(注:我这里用的是 CS3。在以前的版本中，只需点击调色板右上角的箭头图标，打开下拉菜单，然后选择保存画笔库)。

为其命名并选择保存。

下次你想使用它的时候，再次进入笔刷面板，点击底部的文件夹图标，选择打开笔刷库。

你可以用同样的方法制作很多有趣的锥形笔刷。简单地改变你开始画出的形状。尝试三角形或简单的叶子形状，而不是圆形。这是我用同样的步骤做的其他几个。

![samples](../Images/0beb17bffe9484cc4cbbacc8c6ee75b9.png "samples")
