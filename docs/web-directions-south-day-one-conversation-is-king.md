# 南方网络指南，第一天:对话为王

> 原文：<https://www.sitepoint.com/web-directions-south-day-one-conversation-is-king/>

SitePoint/99designs 团队今天去了 Web Directions South，与顾客们混在一起，与演讲者们分享咖啡和胡萝卜蛋糕。

Miles 在最近一期的 SitePoint 论坛报网站中提到，有很多理由让你应该参加会议，并给出了一些让你从会议经历中获得更多的技巧。然而，我们从一开始就明确表示，我们参加 Web Directions South 的目的有所不同。

因此，与其总结已经进行的不同会议(你可以[在 Web Directions South 网站](http://south08.webdirections.org/?page_id=7)上阅读关于它们的所有内容)，我想我应该首先反思我们来到这里的目标，并看看我们在第一天结束时是如何跟踪的。

**1。品牌认知度**

我们每年参加这个活动的一个非常重要的原因是品牌知名度。我们的 t 恤呈亮橙色是有原因的——它们很显眼。我们经常在走廊上遇到拥有我们的书籍或访问我们网站的客户，拥有一个存在(在观众中和在面板上)意味着 SitePoint 品牌在最有可能成为客户的人的脑海中保持新鲜。

我们带了两个手提箱的 t 恤衫，花了半天时间和人们聊天，并像游击队员一样分发他们。它们很快就被抢购一空，我们希望在明天的会议上看到一大堆人穿着它们。我们还赠送了一些书籍给那些承诺将它们放在办公室图书馆供整个团队共享的人，所以在这方面，我认为我们做得很好。

**2。招聘**

这可能是我们花钱让人们飞往悉尼的最大原因——参加会议是一个很好的方式，可以结识不仅在我们感兴趣的领域工作的人，而且可能对构成我们价值观一部分的主题也有真正兴趣的人——网络标准、开源、最佳实践……三年前我在这次会议上遇到这个团队时，我被 SitePoint 聘用，由于在 Web Directions 认识了他们，我们已经雇用了至少两名其他员工。从历史上来说，这比付钱给招聘机构要便宜。

到目前为止，我们已经遇到了一些很棒的人，有几个人表示对我们目前的工作感兴趣，这很有希望。当然，我们总是渴望与有才华的人交谈，所以我们仍然会尽最大努力与志同道合的人交流！

**3。支持当地产业**

我们可能会时不时地从 SitePoint 派人去  或  ，但我们从未错过网络方向会议。当然，将员工派往另一个城市比派往另一个国家花费更少，但我们也认为支持当地行业发生的事件很重要。我们相信这次会议与欧洲或北美的任何类似会议一样好，甚至更好，所以我们希望看到它继续发展。

**4。网站内容**

很自然，SitePoint 团队(包括我自己)已经采访了一些演讲者，并从他们那里收集了一些关于网络未来的想法、建议和预测。今年的阵容和以往一样好，所以与奥古斯特·德·洛斯·雷耶斯讨论哲学或与纪娜·波顿和丹尼尔·布卡讨论[灵感将会为 SitePoint 读者带来一些有趣的阅读内容，这些内容将在未来几天内出版。](https://www.sitepoint.com/article/jina-bolton-interview/)

当然，非正式的对话也同样有趣——与杰夫·克罗夫特讨论以像素为单位设置字体的好处，与杰夫·维恩谈论骑自行车，向唐娜·斯潘塞询问可用性技巧，这些都既有用又鼓舞人心。

**5。联网**

Miles 提到了演讲者和与会者“shmoozing”的重要性，对我来说，这是一次让今天变得非常值得的经历。与在工作中面临挑战的真实人物聊天总是有趣和令人振奋的，这也是我继续参与墨尔本[网络标准组织](http://webstandardsgroup.org/melbourne.cfm)活动的原因之一。我们也一直在寻找新的作者，这是一个认识他们的好机会！有了昨晚的[波特 80](http://forums.port80.asn.au/) 饮料和今晚的[网络堵塞](http://webjam.com.au/webjam8)活动，我想我们可能已经见过了所有与会者的很大一部分，我们还有一天要走！

讽刺的是，网民们从面对面的会议中获得了如此多的东西，这一点没有人不明白，但这仍然是事实。
