# 开源法律资源

> 原文：<https://www.sitepoint.com/open-source-legal-resource/>

随着新闻媒体继续报道昨天[软件自由法律中心](http://www.softwarefreedom.org/index.html)的开张，今天的讨论达到了高潮。

在开源开发实验室(Linux 创建者 Linus Torvalds 的家乡)的部分资助下，这家总部位于纽约的组织希望为开源技术的用户和倡导者提供法律和服务支持。

最有趣的目标可能是训练新律师理解开源许可世界的细微差别，包括 GNU 通用公共许可证(今年也进行了重写)。

该小组还将帮助开发人员寻找用自己开发的和现有的开源代码构建解决方案，以及协助更好的许可证选择。董事会成员包括业界重量级人物埃本·莫格伦(自由软件基金会总顾问)、劳伦斯·莱斯格(知识共享)和黛安·彼得斯(开源开发实验室)。

一些纯粹主义者对自由和开源软件在与商业软件相同的渠道中工作表示遗憾——经常使用相同的商业策略。然而，通过代码审查、法律援助和企业采用等服务实现的开源商业化为销售开源解决方案的企业打开了新的大门。

开源已经表明它可以与专有领域的大人物并驾齐驱，现在它将通过像这样的组织对掌握预算财权的高管进行关键的验证。
