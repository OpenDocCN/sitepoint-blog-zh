# 第九频道

> 原文:[https://www.sitepoint.com/channel-9/](https://www.sitepoint.com/channel-9/)

MSDN 的新[频道 9](http://channel9.msdn.com) 网站希望，正如他们所说，提供“一种收听微软驾驶舱的方式，一个学习我们如何飞行的机会，一个了解我们飞行员的机会。”换句话说，它让你看到开发人员是如何工作的，他们是如何思考的，并允许你与他们展开对话。与以往任何时候我们生活在微软的阴影下不同，作为开发人员，今天我们有机会接触这家公司及其背后的人。参与进来，谁知道呢，你想看到的那个特性或洞察力可能会实现。

这当然是一个非常可读和有用的网站，充满了来自微软内部的视频、评论和讨论。

就像博客给 MSFT 赋予了一张人性化的面孔一样，第九频道也有机会给这张面孔带上友好的微笑。

与任何社区驱动的网站一样，它的有用程度和成功程度取决于其成员的活跃程度。

而且，让我们面对现实吧，有时候戴着耳机坐在工作岗位上看视频也不错，而不是用眼睛阅读文本。
