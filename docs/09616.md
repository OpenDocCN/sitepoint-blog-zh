# 放下胡言乱语

> 原文:[https://www.sitepoint.com/drop-the-mumbo-jumbo/](https://www.sitepoint.com/drop-the-mumbo-jumbo/)

我最近看了几个网站开发者的网站，对他们中的大多数技术含量之高感到震惊。

当然，炫耀你理解行话是很好的，但是潜在客户想要阅读你的网站而不用查字典。

这是我看的一家小型网络公司主页上的开篇句子；

我们的开发团队将创建通过所有验证的 XHTML 和 CSS，同时满足您的项目需求。

听起来很棒，不是吗？确实如此；对于像你我这样的人来说，他们明白我们在说什么。这里的规则是使用主动语态，保持文案简明扼要。

我敢打赌，下面这句话会比他们现在的这句话更适合他们；

我们将超越您的网站需求，并确保所有‘幕后’事务都得到妥善处理。

马上对你自己的网站投以批判的眼光；发现任何首字母缩写词或长文？放下晦涩难懂的东西，把你的内容写给读者——而不仅仅是其他网站开发者。
