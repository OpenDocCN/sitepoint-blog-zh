# 自由职业者如何休假

> 原文：<https://www.sitepoint.com/how-freelancers-get-holidays/>

当你读到这封信的时候，我正在度假。

当然，这句话听起来比实际简单。作为自由职业者度假需要相当多的准备，再加上一些积蓄和决心——不一定是这个顺序。

正如你所料，我已经花了几周时间准备离开我的工作只有七天，其中一些是在我的国家的公共假期。当我度假的时候，我总是完全脱离，所以在我休息的时候，一切都必须自我维持。当我努力让一切井然有序时，我写了这篇文章——一种自由职业者的节前日记。

## 出发前一个月

每当我的脑海里突然出现我的假期时，我就有一种模糊的恐慌感。我把它安排在我的日历里。到目前为止，这是个好主意。突然，它变成了一种负担:在我走之前，我有很多事情要准备。我试图同时忽略这一点，并采取行动——这是一个无效的组合。

## 出发前三周

“好吧，”我告诉自己。"是时候认真对待，振作精神，抓住这头公牛的角了。"我决定在走之前把我需要做的事情列一个清单。我在这方面有一个不成熟的开始。

对事情失控的恐惧让我开始为我的缺席做准备。我开始写作——包括这篇文章——像松鼠囤积坚果过冬一样囤积内容。因为我提前做了这么多，我希望在我走之前事情不会变得太忙乱。

我也开始告诉我的客户我离开的日期。事实上，我周五就回到了镇上，但我把下周一作为我回到甲板上的第一天，因为我回家后需要一点时间振作起来。

## 出发前两周

在这一点上，事情似乎得到了控制，我让自己相信一切都会好的。我只是像没事人一样度过这一周。

然后，快到周末的时候，我的客户开始给我发作品。他们中的一些人已经忘记了我要离开(我确实告诉过他们，对吗？).我不得不拒绝他们，取而代之的是把工作安排在我休假回来的那一周。

大约在这个时候，我意识到，考虑到我去之前的一周有一些会议，而且事实上一周只有四天，我只有大约三天的时间来做最后一周的工作和假期准备。是的——是疯狂的时候了！

## 出发前一周

控制混乱。当我在最不可能的时间和地点工作时，会有清晨和深夜。

当我们面对堆积如山的工作时，这是常有的事，一旦我开始做，我发现它并不像我想象的那样难以克服。的确，妥协是存在的——我回来的第一周不会像我希望和计划的那样顺利——但我确实成功地为我的离开做好了一切准备，甚至更多。

事实证明，两周前我的恐慌真的很有帮助，至少给了我几天时间重返工作岗位。我不会放松——在过去一周左右的时间里预订了我的时间的客户已经看到了这一点——但至少我有一个生产基础，这样每天的任务至少应该顺利进行，而不需要我过多的额外投入。

就是这样——没有最后一刻的戏剧或通宵。我很确定那是因为我一个月前就开始为这个假期做准备了。

我希望你有一个不同的，比我冷静恐慌的假期准备哲学更明智的方法。你从自由职业中抽出时间的秘诀是什么？在评论中分享它们(我们都知道我需要一些帮助！).
