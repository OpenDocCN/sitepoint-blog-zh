# 想教自己的课程？我们是来帮忙的…

> 原文:[https://www . site point . com/want-to-teach-your-own-course-were-to-help/](https://www.sitepoint.com/want-to-teach-your-own-course-were-here-to-help/)

对于那些在推特上关注我们或者收听 T2 网站播客的人来说，你可能知道上周有几个团队参加了 T4 博客世界新媒体博览会。

除了让所有主持人在同一个房间里运行我们有史以来第一次直播播客，我们还向 4500 名与会者发布了一个特别公告。

会议现在已经结束，所以我想向你介绍…

[![Learnable: the way learning should be](../Images/c4b3c24276381fdd4c8cc432d90c302b.png)](http://learnable.com) 
**可学:学习的方式应该是**

[Learnable](https://learnable.com/) 将建立在 [SitePoint 课程](https://www.sitepoint.com/premium/library)的成功之上，在这里你可以在线学习或创建任何主题的课程。

### 针对学生

如果你是一名参加过 [SitePoint 课程](https://www.sitepoint.com/premium/library)的学生，你将会熟悉可学习课程的要素:

*   叙事内容的视频和文章
*   通过论坛和在线问答会议与您的老师、主题专家和同行进行互动
*   每日电子邮件提醒，帮助您保持正轨

我们的目标是提供最物有所值的在线教育体验。

### 对于教师来说

如果你是某个领域的专家，可学习性是:

*   一群想要学习的学生
*   一个易于使用的在线向导，可将您的内容组织到在线课程中
*   一种将你的激情货币化的方法

Learnable 处理技术方面的事情、支付和支持，让你专注于你最了解的事情:交流你热爱的事情。

Learnable 将于今年晚些时候公开发布。目前，我们正在寻找基金会合作伙伴，在发布之前参与我们的测试计划。

因此，如果你对课程有什么想法，我们希望听到你的意见。

开设课程非常有趣，而且我们的工具非常容易使用，你不需要成为技术专家就能让它运行起来。我们将给予你全力支持，当我们发布时，你将成为 [Learnable](http://learnable.com) 上的首批课程之一。

最初，我们只能容纳 50 名教师——如果你想开办一门课程，你需要迅速行动。

有三种方式可以表达你的兴趣:

*   请在下面留下您的评论，让我们知道您想参加的课程
*   前往 learnable.com 报名
*   给 beta@learnable.com 发邮件
