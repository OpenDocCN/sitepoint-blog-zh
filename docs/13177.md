# 即将推出:CSS 工作组的变化？

> 原文:[https://www . site point . com/coming-soon-changes-to-the-CSS-working-group/](https://www.sitepoint.com/coming-soon-changes-to-the-css-working-group/)

[**在之前的一篇文章**](https://www.sitepoint.com/clarke-calls-for-css-working-group-to-be-disbanded/) **中，我报道了 web 标准社区中的危机，尤其是围绕 W3C CSS 工作组的争议。安迪·克拉克和其他人提出的批评似乎没有被置若罔闻，因为工作组内部正在领导一场变革运动。**

12 月 14 日，CSS 工作组的主要成员之一，Mozilla 公司的 L. David Baron 发布了一个[公告](http://lists.w3.org/Archives/Public/www-style/2007Dec/0076.html):

> “我已经通知 CSS 工作组，我不再参加会员专属的邮件列表或会议。我认为该团体的成员保密性质损害了 CSS 的未来发展，因为它使该团体:
> 
> *   不接受许多愿意为社会保障基金捐款的人的捐款
> *   陷入辩论和拖延战术，公司不会舒服地在公共场合使用。
> 
> “我仍然打算参与在 www 风格、public-css-testsuite 和其他公共论坛上进行的任何讨论。
> 
> "我支持将 CSS 工作组重新命名为公共组."

虽然很少有公开跟进，但微软和谷歌的代表都表示了支持。CSS 工作组内部的变化似乎即将到来，但就像该小组所做的一切一样，它可能不会很快发生。
