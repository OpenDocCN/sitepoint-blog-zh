# 中央桌面推出企业版

> 原文：<https://www.sitepoint.com/central-desktop-launches-enterprise-version/>

总部位于加州帕萨迪纳的 SaaS 协作和项目管理套件提供商[中央桌面](http://www.centraldesktop.com/) ( [之前报道](https://www.sitepoint.com/central-desktop-database-api/))今天宣布发布他们的新企业版。这家成立三年的公司去年 3 月宣布了一轮 700 万美元的融资，为 20 万用户提供协作软件，其中最著名的是美国当选总统巴拉克·奥巴马的总统竞选。

首席执行官 Isaac Garcia 告诉我，尽管 Central Desktop 一直以中小型企业为目标，但他们已经为许多企业客户提供了服务。新的企业解决方案旨在满足这些客户的需求。

根据 Garcia 的说法，对于在当前经济低迷时期寻求省钱的中小企业来说，SaaS 应用程序是一个很好的选择。

他在一份新闻稿中说:“我们的企业客户需要深入的功能、高级别的安全性和可靠性，以及与现有系统和应用程序的集成，但今天的公司不想支付或维护昂贵的硬件或软件——每个人都在寻找减少浪费的方法。”。“我们的唯一目标是以最少的阻力为商业用户带来市场上最好的技术——SaaS 非常适合成长型公司的底线，也非常适合寻求节约成本解决方案的大型公司。”

Central Desktop 的企业包具有改进的安全性、Salesforce.com 集成、无限制的 API 访问以及保证 99.8%正常运行时间的 SLA。但是新软件包的核心是工作流管理器。工作流功能允许客户基于 Central Desktop 中文件和数据库级别的事件创建自动电子邮件通知和任务。

例如，可以设置软件在任务完成时自动通知特定的员工或部门，并根据结果创建新的任务。因此，项目经理可以设置 Central Desktop 来自动为 IT 人员创建一个任务，以便在编辑部批准后在网站上推送实时更改。

工作流管理器渗透到整个工具套件中，并且可以基于文件级的数据库。

该软件的新企业版起价为 499 美元/月。
