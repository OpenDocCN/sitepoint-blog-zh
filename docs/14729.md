# 回顾–Tinderbox 内容管理系统

> 原文:[https://www.sitepoint.com/review-tinderbox/](https://www.sitepoint.com/review-tinderbox/)

2003 年 10 月，我建议[网络内容应该为网络](https://www.sitepoint.com/article/caffeinate-hypertext)而写，而不是被纸和笔的假设所束缚。那时，我相信维基(T4)是实用超文本写作中最强大的工具。

然后我下载了个人内容管理系统[Tinderbox for MAC OS](http://www.eastgate.com/Tinderbox/)(9&X)的 [Tinderbox 2.1](http://www.eastgate.com/Tinderbox/) 。我从来没有用过这样的内容管理系统。

##### 什么是火绒箱？

Tinderbox 是一个真正的开放式写作工具。它可以用来建立网站，跟踪联系信息，制作流程图，并可视化数据库记录。而这些只是显而易见的用途。

Tinderbox 使用 [XML](https://www.sitepoint.com/really-good-introduction-xml/) 作为其本地文件格式，将信息存储在“notes”中。这些笔记可以用多种方式组织和链接:层次结构、二维地图或任何你能想到的其他结构。就像拖放一样简单。

![1302_researchmap](../Images/e2d9eb015f4f342a4e60cdc28739499c.png)
*在地图视图中浏览笔记链接*
*我用火绒箱整理了我对一个历史事件的研究。这个软件对于追踪我的消息来源非常有用。它很容易记录下我去过的图书馆的笔记，我自己的研究笔记，以及不同信息片段之间的联系。*

Tinderbox 使用简单直接的模板将笔记集合导出到您定义的任何类型的文档中。它们可以组合起来形成一个网站或一个 HTML 页面。灵活的导入和导出工具有助于 Tinderbox 与其他应用程序合作，如数据库、电子表格和联系人管理器。Tinderbox 模板甚至可以配置成用 PHP 和其他后端编程语言编写。

![1302_htmlview](../Images/a7c1bee9ca3059b957338a554b517dad.png)
*HTML 导出预览*
*在 Tinderbox 中，你可以看到导出模板的 HTML 输出，并在你首选的网页浏览器中预览。所见即所得编辑器工作良好，包括必要的拼写检查。*

使用 Tinderbox 代理，编写和发布到 Web 变得更加强大，这类似于数据库系统中的查询。这些代理可以像 Tinderbox 中的任何其他笔记一样被操纵，可以使用强大的搜索词以许多不同的方式自动收集和组织笔记。例如，一些 Tinderbox 网站使用代理来构建包含摘要和相关文档链接的边栏。其他人使用代理来总结即将到来的约会或跟踪他们的待办事项列表。

您是否处理可定义类型的信息，包括约会、博客帖子或书签？在 Tinderbox 中，您可以定义笔记原型。当你创建一条新的信息时，Tinderbox 可以自动将原型的属性和模板应用到新的笔记中。

Tinderbox 提供了多种方式来可视化它所包含的信息。地图视图在二维地图上显示注释，允许用户缩放嵌套信息。这对于鸟瞰你的数据和笔记之间的链接非常有用。如果您喜欢层次结构，资源管理器视图可能会满足您的需要。资源管理器视图在当前选择的笔记旁边显示一个树。这个视图对于整合网站特别有用。

![1302_exploreview](../Images/5d134d37cdf891fde76386a9aa561d29.png)
*火绒箱浏览器查看*

在 Tinderbox 中没有正确的方法来查看或编辑你的数据。事实上，您可以同时打开多个视图，以地图、层次结构、家谱、链接类别或其他方式查看您的信息。但你不必被这种复杂性吓倒。Tinderbox 文档可以是简单的，也可以是复杂的，这取决于您的选择。Eastgate 灵活、强大的界面设计让您除了自己的创造力之外没有任何限制。Tinderbox 有更多的功能，但它最强大的地方在于它有多简单，它拒绝强迫用户进入一种特定的思维模式。

##### 直接来自有关人士的

我给 Eastgate Systems 的创始人马克伯恩斯坦(Mark Bernstein)发了一封电子邮件，问他为什么称 Tinderbox 为个人内容管理工具。他是这样回答的:

内容管理系统通常与企业管理层强加的极其庞大、复杂和僵化的工具联系在一起，并且通常主要是为了控制创意。虽然 Tinderbox 是一个内容管理系统，但它的组织生态非常不同于，比如说， [Vignette](http://www.vignette.com/) 。此外，最不喜欢企业 CMS 系统的人——网页设计师和创作者——对 Tinderbox 的采用至关重要。

然而，Eastgate 不仅仅在网站上使用 Tinderbox:

东门的在线杂志《铁甲 T2》使用 Tinderbox 进行编辑，不仅是为了网站本身，更重要的是，为了协调每期的写作任务和编辑任务。我为管理任务和问题准备了一个单独的 Tinderbox 文件。每篇主要论文和讲座都是从一个火药桶文件开始的。我在会议和研讨会上做的大部分笔记都是在火绒箱里完成的。

##### 行动中的火药桶

在阅读了 Tinderbox 并构建了一些简单的文档后，我仍然不太明白所有的东西是如何组合在一起的，所以我决定在 Eastgate 网站上的一个示例文件的帮助下尝试构建一个博客。没有最后期限的压力，也不需要构建一些有用的东西，我能够快速彻底地学习 Tinderbox。我建议你也这样做。

首先，下载 [Tinderbox 演示](http://www.eastgate.com/Tinderbox/download.html)。然后下载我建的[样本火绒箱网站](https://www.sitepoint.com/examples/tinderbox/Sitepoint-tinderbox-sample.zip)。在里面，你会发现一个博客，有完整的模板，新闻源和其他有趣的功能。你还会发现一个详细的教程，记录了我如何把网站放在一起。你可以在这里看到完工的场地。

一旦在 Tinderbox 中建立了一个站点，添加和编辑内容只需点击并在 WYSIWYG 编辑器中键入一条注释。建好网站后，我意识到 Tinderbox 不仅仅是一个个人内容管理系统。在某种程度上，它是一个 CMS 构建器。Tinderbox 给了设计者或作者灵活决定内容如何组合的能力。如果设计师或作家不确定写什么，Tinderbox 也是一个很好的头脑风暴工具。

##### 学习曲线

火绒箱需要时间来理解。它不像你用过的任何软件，除非你有时间学习，否则 Tinderbox 可能不适合你。从好的方面来说，学习火绒箱的过程也是独特的。学习过程的大部分包括将大脑从我们使用文字处理器时得到的陈腐假设和简单限制中解放出来。一旦你克服了理解界面的小障碍，事情就开始有条不紊了。几乎每次我想，“我希望它能做到这一点”，我发现我可以使用火绒箱完成它。

在那些需要一点神秘知识的领域，手册就足够了。虽然它彻底地解释了软件(在 207 页中)，手册包括很少的例子和很少的教程信息。Eastgate 不喜欢告诉你如何组织你的文档，他们可能认为解释太多会扼杀用户的创造力。这是不幸的。这使得最初的学习曲线变得更加困难。幸运的是，一些善于表达的 Tinderbox 网络日志记录者分享了他们的发现。样本文件也是一个巨大的帮助；建议你看一下上面提到的样本 Tinderbox 网站。Eastgate 还在 Tinderbox 下载页面上发布了几个样本文件。

可悲的是，火绒箱最好是个人使用。它不具备基于 Web 的内容管理系统的所有有用的协作功能。导入/导出功能可以用来与多用户[cvshome.org](http://www.cvshome.org/)版本控制系统合作，但是 Tinderbox 并不是真正为协作而设计的。现在还没有 Windows 版本，但 Eastgate 预计在 2004 年上半年准备好。

##### 结论

总的来说，我给 Tinderbox 打了满分。这是一款功能极其丰富的独特软件。我不能要求更多的稳定性、便利性或能力。唯一明显的缺点是斯巴达文档。尽管如此，学习 Tinderbox 已经彻底改变了我思考和写作的方式。我可能会成为一个铁杆自由/开源软件狂热者，但这是我认为物有所值的少数软件包之一(售价 145 美元)。
