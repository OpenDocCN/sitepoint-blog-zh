# IE7 Beta 2 修复列表已发布

> 原文：<https://www.sitepoint.com/ie7-beta-2-fix-list-posted/>

Internet Explorer web 平台的首席项目经理克里斯·威尔逊昨天发布了一份为 IE7 Beta 2 做准备时已经修复的错误列表。

以下是列表(带有关于[位置、一切事物](http://www.positioniseverything.net/)和[古怪模式](http://www.quirksmode.org/)的完整描述的链接):

*   [躲猫猫](http://www.positioniseverything.net/explorer/peekaboo.html)
*   [断头台](http://www.positioniseverything.net/explorer/guillotine.html)
*   [重复字符](http://www.positioniseverything.net/explorer/dup-characters.html)
*   [边境混乱](http://www.positioniseverything.net/explorer/border-chaos.html)
*   [不滚动](http://www.positioniseverything.net/explorer/unscrollable.html)
*   [3 像素文本点动](http://www.positioniseverything.net/explorer/threepxtest.html)
*   [蠕动文字](http://positioniseverything.net/explorer/creep.html)
*   [悬停时的底部边距](http://www.quirksmode.org/bugreports/archives/2004/12/losing_bottom_margin_of_previous_element_when_hove.html)
*   失去了突出显示顶部边框下文本的能力(链接，有人吗？)
*   [IE/Win 行高](http://www.positioniseverything.net/explorer/lineheightbug.html)
*   [双倍浮动保证金](http://www.positioniseverything.net/explorer/doubled-margin.html)
*   [IE 中古怪的百分比](http://www.positioniseverything.net/explorer/percentages.html)
*   [重复缩进](http://www.positioniseverything.net/explorer/floatIndent.html)
*   将视区滚动条移动到 HTML 边框之外(链接？)
*   1 px 边框样式(链接？)
*   [消失列表-背景](http://www.positioniseverything.net/explorer/ie-listbug.html)
*   固定宽度:自动(链接？)

IE7 Beta 2 还将为以下内容提供新的支持:

*   [HTML 4.01 ABBR 标签](https://www.w3.org/TR/html401/struct/text.html#h-9.2.1)
*   增强版[<object>*   [CSS 2.1 选择器支持](https://www.w3.org/TR/CSS21/selector.html) ( [子](https://www.w3.org/TR/CSS21/selector.html#child-selectors)、[邻](https://www.w3.org/TR/CSS21/selector.html#adjacent-selectors)、[属性](https://www.w3.org/TR/CSS21/selector.html#attribute-selectors)、[首子](https://www.w3.org/TR/CSS21/selector.html#first-child)等。)*   [CSS 2.1 固定定位](https://www.w3.org/TR/CSS21/visuren.html#choose-position)*   [PNG 图像中的 Alpha 通道](http://blogs.msdn.com/ie/archive/2005/04/26/412263.aspx)*   [修复:悬停在所有元素上](https://www.w3.org/TR/CSS21/selector.html#dynamic-pseudo-classes)*   [背景-附件:将](https://www.w3.org/TR/CSS21/colors.html#propdef-background-attachment)固定在所有元素上，而不仅仅是车身
