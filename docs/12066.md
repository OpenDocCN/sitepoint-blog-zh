# WordPress vs Rails

> 原文:[https://www.sitepoint.com/fullcodepress-wordpress-vs-rails/](https://www.sitepoint.com/fullcodepress-wordpress-vs-rails/)

Matthew Magain 正在澳大利亚悉尼的 [FullCodePress](http://fullcodepress.com/) 网站一日赛现场报道，该赛事由 SitePoint 赞助。

Raena 和我到达 FullCodePress 活动时，事情刚刚开始——两个慈善机构已经宣布，团队正忙着询问各自的客户，并计划如何建立一个满足这两个组织需求的专业网站。

[![](../Images/6bb9d30020a895290175116671346d48.png) ](http://www.flickr.com/photos/fullcodepress/3523805401/) **澳大利亚队的慈善机构是新南威尔士州** 残疾歧视法律中心[，这是一个为因残疾而受到不公平对待的人提供法律咨询的组织。客户向澳大利亚人提供了他们喜欢和不喜欢的两个大型网站设计文件夹，这对澳大利亚设计师 Alexi Paschalidis 来说将是一个有趣的挑战。便利贴和白板草图正在慢慢成型。](http://www.ddlcnsw.org.au/)

新西兰团队 CodeBlacks 正在为总部位于新西兰、致力于支持同性恋青年的性倡导组织[彩虹青年](http://www.rainbowyouth.org.nz/) 建立一个网站。该团队花了几个小时与客户交流，目前正在建立一些模板和绘制信息架构。他们的客户也做了非常充分的准备(并且精通网站可以实现的功能)，CodeBlacks 项目经理海顿·汤姆森告诉我，这令人惊讶，但也意味着简报可能比预期的更雄心勃勃。

[![FullCodePress](../Images/a92995476e15aa4e896de18b14d04efa.png)](http://www.flickr.com/photos/fullcodepress/3524148064/) 这场竞争的一个有趣的方面是，澳大利亚人使用的是 [WordPress](http://wordpress.com/) 解决方案(基于客户希望能够通过插件进行扩展)，而新西兰团队使用的是 [BrowserCMS](http://browsercms.org/) ，一个基于 Rails 的开源 CMS。上次举办这个活动时，新西兰人[从零开始推出了自己的 CMS](https://www.sitepoint.com/article/clean-or-custom-cms/)。虽然他们最终取得了胜利，但这无疑是一个巨大的风险，可能会走向另一条道路。

气氛正在升温——可能是因为越来越多在 CeBIT 会议上闲逛的随机参与者对该活动以及每个团队如何处理各自的项目感到好奇。我认为在你工作的时候有观众一定会给竞争对手增加相当大的压力(就好像在 24 小时内为一个真正的客户建立一个完整的网站还不够)，但是到目前为止我还没有看到任何裂缝出现。每个人看起来都很平静，沉着，只是在继续生活。

我们将会看到，在他们没有睡觉之后，情况是否仍然如此…
