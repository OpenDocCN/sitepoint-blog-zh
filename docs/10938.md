# 谷歌发布 Chrome 5.0

> 原文:[https://www.sitepoint.com/google-release-chrome-5/](https://www.sitepoint.com/google-release-chrome-5/)

Chrome 5 可能只处于测试阶段几周(T1)，但本周已经发布。这也是第一个可以同时在 Windows、Mac OS 和 Linux 上使用的版本。谷歌渴望展示速度的提升。例如，从 1.0 版本开始，根据 [SunSpider 基准测试](https://www2.webkit.org/perf/sunspider-0.9/sunspider.html) : ![Chrome SunSpider benchmark](../Images/2c5096cce728c180d03649c90caf9aa2.png)的测试，JavaScript 速度提高了 305%，根据 [V8 基准测试](https://v8.googlecode.com/svn/data/benchmarks/v5/run.html) : ![Chrome V8 benchmark](../Images/eb211e4dc696df41625bf228f7d37bd0.png)的测试，速度提高了 213%。这些数字对大多数人来说没有任何意义，因此谷歌制作了一个视频，展示了浏览器与土豆枪相比的速度，即声音和闪电的速度。是的，说真的:

你不希望你的公司有这么多时间和金钱可以浪费吗？！除了速度，谷歌还解决了各种错误和 6 个漏洞(其中两个被评为高风险)。也可以在匿名模式下使用扩展。在[我的关于测试版](https://www.sitepoint.com/google-release-chrome-5-beta/)的帖子中提到的功能已经进入最终版本，除了集成的 Flash 播放器。谷歌产品经理 Brian Rakowski 表示，当 Adobe 发布 Flash Player 10.1 的最终版本时，它将被包括在内。Chrome 5 可以在 google.com/chrome[的](https://www.google.com/chrome)下载，适用于 Windows、Mac OS 和 Linux。现有用户可以通过点击**工具**图标，选择**关于谷歌 Chrome** ，点击**更新**按钮进行更新。自动更新很快就会发生，但是如果您现在还不想更新呢？SitePoint.com 即将推出… *如何禁用谷歌浏览器更新*。你安装了谷歌浏览器 5 吗？好点了吗？是不是更快？它已经成为你的默认浏览器了吗？
