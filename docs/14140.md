# 反向销售，大卫·桑德勒

> 原文:[https://www . site point . com/the-reverse-sell-per-David-Sandler/](https://www.sitepoint.com/the-reverse-sell-per-david-sandler/)

大卫·桑德勒的《你不可能在研讨会上教一个孩子骑自行车》是一本很好的销售入门书。桑德勒推出了一个特许经营系统，教人们销售。这是一个非常昂贵的系统，但许多专业人士对它表示赞赏。就我个人而言，当我接受它时，我发现它有点“虚假”，并不像我希望的那样适合建立职业关系。在我看来，它更适合产品销售或一次性销售。

然而，该系统包括一些了不起的想法，使销售更容易。其中一个，当你在他的书中读到时可能会觉得有点噱头，集中在他所谓的反向销售上。反向销售就是退后一步，不要急于出售你的服务。在最好的情况下，这甚至意味着鼓励潜在客户不要购买。例如，“嗯，听起来你对和我继续发展不感兴趣…”

我发现这种反向销售在自然发生时非常有效(不是作为一种噱头或技巧)。这让潜在客户相信他们处于主导地位，也让你与他们平起平坐(因为你不需要这份工作，尽管你很高兴拥有它)。

周五，一个长期客户打电话让我为他们做些工作。这项工作对我来说并不特别有吸引力，我也不认为让客户为我的服务付费是值得的。

我告诉他们，我不想做这项工作，他们不可能以我需要收取的费用获得我想提供的那种价值。

猜猜发生了什么？

他们试图说服我为他们工作。他们问我怎样才能让我给他们几天时间(还有旅行，我讨厌旅行)。他们提出了让工作对我来说更有趣，对他们来说更有价值的方法。

所以在通话结束时，我有一个 3 英镑的提案(三英镑！)乘以桌子上最初的价格，还有更有趣的工作。

我不建议你把这种方法作为你“技巧”的一部分。如果你想更多地了解如何做，你应该读读桑德勒的书。有时我会说一些类似“我不知道你是否有兴趣继续前进”或“你似乎没有认真花钱解决你的问题”的话，以测试前景到底有多严重。

但总的来说，当我真的有这种感觉，可以真实地说出来而不觉得我在演戏，并且已经有了一个真正稳固的关系时，这种类型的陈述效果最好。

无论如何，当时机合适且关系稳固时，反向销售是一种很有价值的策略。
