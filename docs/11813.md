# 合作适合你吗？

> 原文:[https://www.sitepoint.com/is-coworking-right-for-you/](https://www.sitepoint.com/is-coworking-right-for-you/)

共同工作是一种日益增长的全球趋势，它给在家工作的专业人士提供了一个在团队环境中工作的机会。根据维基百科，虽然每个人仍然独立工作，但共同工作小组的成员通常分享价值观，并对在同一空间与有才华的人合作可能产生的协同作用感兴趣。

共同工作空间通常是非正式的，它们经常被描述为咖啡馆里的办公室，尽管它们不应该与你可以租用的办公空间相混淆。主要的区别在于设置——办公室租赁有单独的办公室和隔离的工作空间，而联合办公地点侧重于社区和团聚。

一些更成熟的共同工作空间已经开始扩大，以满足员工的需求。以[立方体和](http://www.cubesandcrayons.com)蜡笔为例。他们成功地将共同工作与儿童保育设施结合起来，在加州有两个办公地点。

从本质上来说，共同工作是一种很好的方式，可以远离家里的干扰，同时在一个社交、协作和非正式的环境中，与其他当地专业人士一起完成工作。

共同工作的好处

共同工作的主要好处之一是能够离开你的家庭办公室，走进社区。如果你面临在家工作的孤立，就像我们许多人一样，共同工作是打破单调的好方法。

但是合作还有很多其他的好处。以下是一些例子:

*   它创造了责任感，这有助于防止拖延。
*   共同工作通过给你一个和其他志同道合的专业人士一起工作的机会来激发创造力。
*   它提供了在家工作时通常没有的社会福利。
*   它让你在工作的时候有机会进行专业交流。
*   合作可以帮助你向他人学习，合作和成长。
*   它营造了一种类似社区的氛围，在这里你可以找到支持和鼓励。
*   它满足了人们与他人并肩工作的基本需求，并能帮助你更好地享受工作。

**去哪里找同事**

世界各地都有协同工作的设施，在你所在的地区找到一个最快的方法是联系当地的其他专业人士或通过谷歌搜索。你也可以通过社交媒体、商业网络和当地组织了解当地的合作机构。

这里有一些到其他协作资源的链接，可以帮助您搜索要加入的协作组:

*   合作社区博客:一个关于合作的信息网站，列出了一些合作地点。
*   谷歌合作小组:一个为对合作感兴趣的人提供支持和讨论的网络。
*   协同工作办公室目录:一个新的目录，包含少量的国际协同工作小组。
*   协作 Wiki:协作资源和本地链接的集合。
*   工作空间(wurk spaces):协同工作设施目录。
