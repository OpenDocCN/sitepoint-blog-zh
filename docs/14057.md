# Atlas:微软的 Ajax 工具包

> 原文：<https://www.sitepoint.com/atlas-microsofts-ajax-toolkit/>

微软正在开发一个做 Ajax 的工具包，叫做 Atlas。据微软的 Scott Guthrie 称，这看起来像是 ASP.NET 2.0 的东西。令人放心的是，他说“Atlas 客户端脚本框架将在所有现代浏览器上工作，并与任何 web 服务器一起工作”，这意味着他们正在以跨浏览器的方式实现 Ajax 风格的远程脚本，这是一个非常好的举措；在过去，MS 在这类事情上倾向于相当特定于 IE，而从这种情况向来自它们的开发工具的适当的跨浏览器支持的转移可以向广大的 ASP.NET 开发者开放远程脚本，他们将在甚至不一定知道的情况下构建跨浏览器代码。看到这种情况会很有趣，但显然 MS 开发人员的心是在正确的地方，我们都希望看到更多这种情况。
