# Twitter 会开始收费吗？

> 原文:[https://www.sitepoint.com/will-twitter-start-charging/](https://www.sitepoint.com/will-twitter-start-charging/)

过去一年，Twitter 一直是互联网的成功故事。成立于 2006 年的微博服务每天发送 300 万条信息，已经被每一个科技名人所采用。即使是 Twitter 广为人知的技术问题、宕机、黑客攻击和名人窥探电视节目的新闻也没有让人们却步(尽管只有 40%的账户在注册一个月后仍然活跃)。

作为一家初创公司，Twitter 筹集了 5500 万美元的风险资本，现在的估值约为 5 亿美元。甚至你自己的 Twitter 账户也可能有价值:根据 TweetValue.com 的数据，T2 的 sitepointdotcom 价值 10991 美元，T4 的史蒂芬·弗莱为 42542 美元。甚至[我自己微不足道的努力](http://twitter.com/craigbuckler)也值 25 美元*(考虑所有出价！)*。

对于一个还没有赚到一分钱的公司来说，这已经不错了。那么，为什么 Twitter 价值如此之高，尤其是在经济低迷时期？

一个可能的原因是，Twitter 抓住了人们兴趣和愿望的本质。假设我正在考虑买一个 Foozle。我四处看了看，发了微博，得到了一些回复，找到了 100 美元的最佳价格。就在那时，Foozle 公司直接联系我，并以 80 美元的价格提供他们的顶级型号。他们在我最脆弱的时候抓住了我；我正在积极考虑购买，可能不会考虑其他选择。我也可能会在随后的推文中大肆吹嘘我的储蓄。

戴尔是一家已经从这种营销方式中获益的公司；他们最近宣布，Twitter 帖子的直接结果是 300 万美元的收入。

Twitter 的价值很大程度上归功于数据挖掘的可能性:

1.  这项服务是公正、独立和免费的。
2.  用户代表了广泛的社会阶层。
3.  Twitter 的使用正以指数速度增长。

然而，Twitter 不能仅靠潜力生存。他们需要收入，据报道，他们正在寻找网站的盈利方式。首席执行官伊万·威廉姆斯表示，他不反对横幅广告，但并不热心，因为这是“最没意思的事情”

正在考虑的一个选择是认证服务；公司将付费确保冒名顶替者不会以他们的名义发送信息。从技术上讲，这可能很难实现。

但是推特必须小心。如果他们开始对服务的任何方面收费，他们将会减少现有用户和新注册用户的数量。这也可能使会员资格向那些拥有最大经济利益的人倾斜:公司。

我看不到很多选择，除了:

*   **广告**。不幸的是，广告可能需要添加到推文中，因为许多人使用第三方服务，而不是 Twitter.com。
*   数据挖掘工具。公司可能会为类似分析的工具付费，尽管许多工具已经可用，因为 Twitter 的大部分数据都是通过他们的 API 公开的。
*   **更好的 API** 。或许他们可以对改进的数据访问收费？

在最终决定宣布之前，各种猜测仍将盛行。Twitter 能在不损失会员的情况下提高收入吗？
