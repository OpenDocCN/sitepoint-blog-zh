今天我们有一篇来自一个叫 Chico Charlesworth 或者简称为“Chico”的灵感 jQuery 开发者的客座博文。来自伦敦的家伙开发了一个很棒的全屏 jQuery 幻灯片插件。看看这个演示，太棒了！

[现场试玩](http://99layers.com/chico/slideshow) [立即下载](https://github.com/chico/jquery-fullscreen-slideshow/zipball/master)

jQuery 全屏幻灯片显示是一个简单的插件，允许你轻松地创建带有缩放/滑动动画的全屏图像幻灯片显示。

现场演示展示了与我最近的推文相关的图片幻灯片。下载内容包括插件和一个示例，该示例显示了国家地理杂志中引人注目的图片的幻灯片。

## 入门指南

*   在网页中包含 jQuery 库和 jquery.fullscreen.slideshow.js 插件。
*   打电话给$。slideshow()用一组图像开始幻灯片放映。
