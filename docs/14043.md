# 为什么许可营销走得不够远

> 原文：<https://www.sitepoint.com/why-permission-marketing-doesnt-go-far-enough/>

在最后一篇博客中，Steven List 发表了一篇精彩的回复，引用了 Seth Godin 和他对许可营销(人们选择加入或选择他们接收的信息)的观点。看一看。

然而，我认为即使是赛斯·戈丁的东西在这一点上也是过时的，不一定是在理论上，而是在大多数人实现它的方式上。大多数使用许可营销的人:

–通过网站上的一些好文章获取潜在客户信息

–通过电子邮件发送信息和报价

–等待潜在客户成为客户，同时建立价值和信任

这些都是好东西，也很重要。然而，它已经老了。人们开始厌倦给出他们的电子邮件地址。他们知道，他们往往没有真正从简讯中获得价值，而是太多的垃圾邮件。

所以我们专业人士又陷入了另一个困境。我们可以回到依靠老一套的基于推荐的营销，这从来没有失败过。但是如果你想主导一个全国性的市场呢？

我认为解决办法是将许可营销提升一个档次。例如，Sitepoint 就是许可营销的一个杰出例子。它形成了很好的社区，提供了很好的内容，而且你可以免费获得大部分内容。同时，他们也很好地提醒了你他们的各种工具包，甚至让你下载样本章节(当然，有很强的跟进)。

对于那些希望自己的服务得到广泛曝光的人来说，Sitepoint 模式可能是一个不错的选择。是的，他们销售广告和产品，但是一些精明的网页设计师和专业人士可以用他们的方法为他们的特定社区创建一个令人惊叹的网站，并获得比他们所能处理的更多的业务。

你怎么想呢?
