# 第 5000 名会员的免费书籍！

> 原文：<https://www.sitepoint.com/free-books-for-the-5000th-member/>

上周，网络标准组织达到了 5000 名成员的里程碑，多亏了 SitePoint，幸运的第 5000 名成员获得了一些令人敬畏的奖励。(澄清一下，WSG 是一个草根组织，专注于通过在线和定期演示分享知识来教育设计师和开发人员——不要与 Web 标准项目混淆，后者与浏览器供应商和工具创建者合作。)

从今天下午发给小组成员的公告来看:

> 我们的第一次聚会是 2003 年 3 月 6 日在悉尼的一个酒吧，有 8 个人。我们目前有来自 109 个国家的 5009 名成员。我们还在一系列城市举行会议，包括悉尼、墨尔本、堪培拉、布里斯班……惠灵顿、伦敦、新加坡和莫斯科，在接下来的几个月里还会有更多的会议。

为了庆祝这一里程碑，SitePoint 捐赠了一系列书籍赠送给第 5000 名会员:来自澳大利亚的 Michael Riethmuller。迈克尔将会收到[《美丽网页设计的原则》](https://www.sitepoint.com/books/design1/)、[《CSS 选集》第二版](https://www.sitepoint.com/books/cssant2/)、[简单 JavaScript](https://www.sitepoint.com/books/javascript1/) 和[交付一流网站:101 个基本清单](https://www.sitepoint.com/books/checklists1/)的副本，添加到他的个人参考图书馆。

祝贺你迈克尔！

成为 Web 标准组织的成员是免费的。任何人都可以加入，会员资格为您提供了访问一系列在线资源、向(非常活跃的)邮件列表和论坛提问的能力，以及免费参加会议的机会。

(免责声明:我是墨尔本 WSG 会议的组织者)。
