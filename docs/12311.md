# Firefox Exec:捆绑？别管闲事

> 原文：<https://www.sitepoint.com/firefox-exec-bundling-no-thanks/>

Internet Explorer 的网络浏览器优势是与 Windows 捆绑的直接结果。至少，这是 Opera 向欧盟提出的指控，欧盟已经决定同意这一指控——它最近裁定，微软通过将 Internet Explorer 与 Windows 捆绑销售，滥用了其在操作系统市场的垄断地位。

显然欧盟正在考虑的一个提议的解决方案是迫使微软开始捆绑其他浏览器——Opera、Safari、Firefox——并允许用户在安装操作系统时选择他们想要使用的浏览器。火狐开发者迈克·康纳说，这是个坏主意。

“我个人的观点是这不是正确的结果，”他告诉 PC Pro 杂志。“(安装 Windows 时的)选择会很奇怪。这方面没有好的用户界面。”

Mozilla Firefox 开发团队的负责人 Connor 也认为 Opera 关于捆绑销售等同于市场份额的断言是错误的。Connor 说，这种说法现在“被证明是错误的”，暗指 Firefox 的市场份额超过 20%，并且在没有捆绑销售的情况下稳步增长。当然，火狐确实享有谷歌的资金支持——自从谷歌发布了一款与之竞争的网络浏览器产品以来，这种关系最近变得更加复杂。

当然，康纳指出，谷歌 Chrome 浏览器的发布可能会受到 Mozilla 办公室的欢迎。在 PC Pro 的同一次采访中，他还提到 Firefox[担心成为垄断](http://www.pcpro.co.uk/news/246907/mozilla-worried-about-becoming-microsoft.html)。“我们有点担心垄断的事情，”他说。“我们不想杀死其他所有人。”

火狐要成为垄断者还有很长的路要走，但是他们现在看起来很强大，最近已经占领了 20%的浏览器市场。与此同时，IE 的增长一直在稳步下滑。在早期采用者和狂热用户中，Firefox 拥有更多的追随者。例如，在 SitePoint，超过 50%的访问者使用 Firefox。对于担心垄断浏览器市场的火狐团队来说，增加一个来自谷歌的可行竞争对手加入 Opera 和苹果的 Safari 可能是一件受欢迎的事情。

康纳说，他宁愿依靠有机增长，而不是捆绑销售来传播火狐的影响力。“当人们意识到还有其他选择时，你就不会陷入那种(垄断)局面。他说:“你必须让人感觉比 ie 浏览器更好。出于这个原因，Firefox 也永远不可能成为垄断者——对当今网络浏览器市场前所未有的消费者选择水平的认识应该会阻止任何一种浏览器占据 2/3 的市场份额。Connor 说:“最终，人们会根据自己的喜好使用不同的浏览器。”

**更新:** Mozilla 董事长 Mitchell Baker 在她的博客上的一篇[帖子中，有点反驳 Connor 的观点，即捆绑销售不等同于市场份额。“在微软开始开发 IE 之前，我就一直在从事网页浏览器的开发和发布工作，微软对竞争、创新和网页开发速度本身造成的损害是显而易见的，而且还在继续。是否有好的补救办法，以及补救办法可能是什么，这是不同的问题。但是关于适当补救的问题并没有改变基本事实。她写道:“微软的商业行为从根本上削弱了(事实上，几乎消除了)人们访问互联网的竞争、选择和创新。](http://blog.lizardwrangler.com/2009/02/06/the-european-commission-and-microsoft/)

她接着说，微软的行为“仍然以减少竞争、选择和创新的方式使微软受益。”贝克说，Mozilla 的小成功也不表明市场上有健康的竞争者。

贝克所说的不一定与康纳所说的相矛盾——康纳实际上并没有站出来说捆绑与 IE 成为今天的庞然大物无关，也没有说捆绑对市场来说不是一件坏事。相反，康纳说，捆绑替代浏览器不会自动提高这些浏览器的市场份额，捆绑和市场份额并不完全相关。

也就是说，似乎不是 Mozilla 的每个人都有相同的想法。
