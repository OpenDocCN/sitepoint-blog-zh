# 像三年级学生一样销售:2 + 2 如何等于更多的客户

> 原文：<https://www.sitepoint.com/sell-like-a-third-grader/>

当我在做网络生意的时候，有一个神奇的数字一直在我脑海中闪现。这个数字是 500。不，不是我有多少客户，或者我每小时收费多少，或者我一周工作多少小时。这是大多数潜在客户认为一个网站应该花多少钱。如果你的起始价是这个价格的三倍、四倍，甚至五倍，你会怎么做？

500 美元是很多钱吗？除了你得到的回报，你不能回答这个问题，对吗？作为一名自由职业者或试图出售你的服务的商人，你必须把你的价格放到合适的环境中。如果你没有，你的前景会。

让我换一种说法。当你向某人报价时，不管是 500 美元还是 5000 美元，他都需要另一个数字来与之比较。如果你不提供一个，他会想出自己的。不幸的是，他使用的数字将是他花血汗钱的其他东西——比如他的抵押贷款或汽车贷款。当你的潜在客户开始比较你的价格和他的其他账单时，你可能已经输了。在他看来，你只是又一个从他口袋里掏钱的费用。

问题是，你的潜在客户关注的是损失——他不得不花的钱。你可以通过提供一个不同的数字让他思考…他将会得到什么，来绕过这个思考过程。

## HTML x CSS ÷ PHP = ROI

投资回报(ROI)是一个通常不会在销售对话中提及的等式。然而，投资回报率只是一个简单的数学公式得出的数字。让我用一个实际的例子，因为我在数学上有困难，所以我会保持简单。

假设我正在考虑为我的网络业务开展直邮活动。如果我的库存图像总成本加上打印和邮寄 5000 张明信片的成本是 1500 美元，我的平均工作成本是 900 美元，那么我需要 1.66 份工作才能达到收支平衡:

$1,500 ÷ $900 = 1.66

直邮的回复率为 1-3%。让我们保守一点，用较低的数字。也就是说，在 5000 张明信片中，我应该得到 50 人的回复。如果我能每 10 个人中关闭一个，那就是 5 个工作和我投资的正回报。再次，简单的三年级数学:

50÷10 = 5x 900 美元= 4500 美元

如果这意味着我能挣 4500 美元，我会整天花 1500 美元。你也可以从你想带来多少收入开始，从另一个方向来计算。一旦你对你的潜在客户使用这个公式，剩下的就是他的风险商数:你愿意冒险 *x* 来得到 *y* 吗？

马克·吐温曾经说过，有“谎言，该死的谎言和统计数据。”当你开始用数字来说明你的情况时，你的许多潜在客户也会有同样的感觉。*“但这只是理论上的！”*我经常听到这样的话，“这比我的购房款还多！”但是要提醒你的潜在客户，他在自己创业时也做了同样的事情。他观察市场，以确定他需要多少顾客或多少收入，然后扣除他的管理费用，看看他是否能赚到足够的钱来维持生活。他当时冒了一次险。这只是他创业以来所冒的一系列风险中的又一个。他愿意接受吗？

## 最终方程式

将一笔业务支出与你的购房款相提并论是没有逻辑意义的，但请记住，人们是基于情感来购买的，然后用逻辑来证明这一点。所以顺理成章，人们也*基于情感拒绝购买*不是吗？所以最后的等式是:情感+逻辑=更多的销售。你需要向你的潜在客户提供购买 *的[情感理由和](https://www.sitepoint.com/it-works/ ""It Works!"")*逻辑理由来证明这一点。投资回报率是一个很好的方式，为他们提供了一个逻辑挂钩，让他们在你离开大楼后戴上帽子。
