# 给你的网站增加一些重点

> 原文:[https://www.sitepoint.com/add-some-emphasis-to-your-site/](https://www.sitepoint.com/add-some-emphasis-to-your-site/)

有时候，一项有趣的技术会让你坐起来想‘这真是个好主意！’今天就是其中之一。

与 TelSim Software 合作，测试他们的新“Noah”技术，该技术本质上是一种在 Flash 中驱动的网站增强设备，我惊讶地发现，在一个网站中创建一系列页面的更有用和更突出的体验是多么容易。

以[phireworx.com 主页](http://www.phireworx.com)为例，有许多链接和选项，在你去过几次后，很容易导航，但第一次访问者可能不知道他们在找什么，那么为什么不有一个友好的头像，指引你在屏幕上指出你可能有的主要部分/优惠。

我注意到，自从 phireworx 头像在过去几天出现在该网站的首页后，访问该网站主页并点击链接而不是点击另一个网站的用户数量增加了 37%，这是一个不小的成就！

这肯定会吸引你的注意力，TelSim 正在寻找购物车网站来测试之前和之后的结果，他们将与谁合作打折！
