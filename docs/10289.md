# Drupal 7 有什么新特性？

> 原文:[https://www.sitepoint.com/drupal-7-new-release/](https://www.sitepoint.com/drupal-7-new-release/)

经过三年的紧张开发，Drupal 7 今天发布了。数百名开发人员为使 Drupal 变得比以往更简单、更强大做出了贡献。

Drupal 是一个用 PHP 编写的开源内容管理系统。是的，另一个。然而，Drupal 是第一批优秀的 CMS 之一，并在 2001 年推出后成为一个流行的博客和社交社区平台。如今，许多热门网站都在使用它，包括[白宫](http://www.whitehouse.gov/)、[、【经济学家】T3、](http://www.economist.com/)[、【纽约观察家】T5、](http://www.observer.com/)[、HowToDoThings.com](http://www.howtodothings.com/)和 [Linux Journal](http://www.linuxjournal.com/) 。

## Drupal vs. WordPress

我确信你们中的许多人在想“我使用 WordPress，我很满意，谢谢。”许多 Drupal 粉丝会认为 WordPress 功能不足，扩展性差，缺乏安全性和灵活性。WordPress 的粉丝可能会认为 Drupal 过于复杂，更难维护，也不太用户友好。

单看安装，Drupal 无疑已经被 WordPress 盖过了。然而，Drupal 7 解决了许多批评，团队努力简化用户体验。当然，WordPress 也进行了更新，在版本 3 中引入了几个类似 Drupal 的特性。

最终，他们都是伟大的系统，他们的功能集是趋同的。为什么不两个都试试，然后为手头的任务挑选一个最好的呢？

## Drupal 7 的新特性

让我们来看看引擎盖下…

**安装更简单**
安装过程已经简化。Drupal 支持大多数 web 服务器(推荐 Apache)、PHP 5.2.4+和大多数数据库(推荐 MySQL 5.0.15+)。

改进的管理面板
管理界面已经过彻底检查，变得更加简洁。新的可配置快捷栏提供了对最重要任务的快速访问:

![Drupal 7 administration screen](../Images/0b72a9a6568b3eb8964777eb09a689e8.png)

**更灵活**
Drupal 7 允许你定义自己的内容结构，并为页面、用户、评论和其他类型的内容添加自定义字段。已知有 800 多个模块与新版本兼容。

**速度和可扩展性**
Drupal 速度很快，可以处理大量的流量，具有更好的缓存和改进的 JavaScript 和 CSS 优化。

可访问性
可访问性更好了，Drupal 现在通过 RDFa 提供语义 web 支持。

Drupal 仍然是开源的，任何人都可以免费下载、改编和使用。
