重大新闻:14 年来，Internet Explorer 第一次被挤出浏览器排行榜的首位。谷歌 Chrome 已经成为世界上使用最多的浏览器；当你考虑到它是在不到四年前发布的，这是一个惊人的努力。

虽然 Chrome 的增长在 4 月份有所下降，但在 5 月份恢复了 1%以上的正常增长。几个网站报道说 Chrome 在月中已经超过了 IE，但是直到今天才得到证实:

1.  浏览器很接近，榜首随着每天的波动而变化。
2.  许多国家在 5 月份有公共假期，这通常对 Chrome 有积极影响，但对 IE 的核心业务使用有负面影响。
3.  StatCounter 改变了它的记录方法，忽略了 Chrome 的页面预渲染技术的影响(Chrome 猜测用户可以点击什么，并开始后台页面下载，而不管页面是否被查看)。

Chrome 18 也是世界上使用最多的浏览器版本，市场份额为 16.19%。这不会持续太久，因为 Chrome 19 已经有 12.93%的份额，用户最终会迁移。

对谷歌的祝贺反映了对微软的同情。所有版本的 Internet Explorer 都失去了阵地，包括 IE9 的小幅下降。但即使是微软也会很高兴看到 IE6 和 IE7 分别降低到 1%和 2%以下。中国是唯一一个综合使用率达到相对较高的 17%的国家。

Firefox 仍然是四分之一用户最喜欢的浏览器，但三年来几乎没有变化。Safari，Opera 和其他人在争夺残羹剩饭。

下一个问题:Chrome 还能发展多久？我们现在有一个竞争激烈的浏览器市场，三分之一的用户使用 Chrome，另外三分之一使用 IE，剩下的用户使用 Firefox、Safari 和 Opera。这是一个很好的情况，没有占主导地位的供应商。让我们希望它保持这种方式。

## 移动浏览器的使用

五月份的手机使用量增加到了所有网络活动的 10.11%。

主要的移动浏览应用程序有:

1.  opera Mini/Mobile–21.90%(增长 0.38%)
2.  安卓——21.06%(下降 0.25%)
3.  iPhone——19.50%(下降 0.54%)
4.  诺基亚浏览器——11.53%(增长 0.11%)
5.  UC 浏览器——8.47%(增长 0.70%)

前五名没有变化。唯一显著的趋势是 UC 浏览器的持续增长。
