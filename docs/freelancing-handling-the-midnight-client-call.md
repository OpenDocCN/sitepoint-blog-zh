# 自由职业者:处理午夜客户电话

> 原文：<https://www.sitepoint.com/freelancing-handling-the-midnight-client-call/>

我在上一篇文章中提到的一点( [3 条在家工作的黄金法则](https://www.sitepoint.com/3-golden-rules-for-working-from-home))是；在家工作时，如何处理正常工作时间内外的电话。有三种基本情况:

## 1.工作时间打电话给客户

拿起电话。使用语音邮件在下班后仍有用武之地，但在工作时间:*拿起电话*。电话答录机对客户说——我现在不在，这(在客户看来)可以理解为，你不可靠，你不关心他们的业务。可以这样想:一个客户使用你——一个自由职业者——承担了额外的(感知的)风险，因为使用你不像在公司那样“安全”。因此，让客户“感受到爱”并能在工作时间与你交谈是至关重要的。这有助于向他们保证，他们的业务在你的手中是安全的，你不是一些“夜间飞行”的业余爱好者。

## 2.工作时间打私人电话

你的朋友或家人在工作时间给你打电话是很正常的，这是一个你需要尽快解决的问题。朋友和家人需要对你的工作进行一些“教育”。在正常工作时间，你在工作——句号。礼貌的<q>现在不能说话，忙着工作——我今晚晚些时候给你打电话好吗？</q>，通常会做到这一点。这不会伤害到一个微妙的提醒:你在工作时间有多忙，你可以在晚上多聊聊。

## 3.客户在工作时间之外打电话

在正常工作时间之外接到客户电话并不罕见(我在银行假日接到过深夜电话)。这通常是不可接受的，除非你提供 24/7 的支持。

唯一可接受的在正常工作时间(IMO)之外致电的时间是有重要的业务问题，即<q>求助。我的网站已经被一个推销药丸的色情网站</q>或<q>帮助所取代。我的网站宕机了</q>——这些都是关键问题。<q>我不记得如何创建链接</q>或<q>我忘记了我的密码</q>，这些都不是关键问题。

那么你是如何处理的呢？

1.  我的策略很简单:如果这不是一个关键问题，那么解释他们在你的正常工作时间之外打电话，你将在下一个工作日调查此事。要有礼貌，但要坚定。说明如果他们需要工作时间以外的支持，则需要额外收费。在 99%的情况下，客户会尊重你的隐私，不会再这样做。
2.  第二个工作日，解决任何问题，然后通过电子邮件让客户知道你做了什么，并重申你的支持时间。

你可能已经猜到了，这一切都与沟通有关。让人们知道什么时候打电话给你是可以接受的。TTFN。
