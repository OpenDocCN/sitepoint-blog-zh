# 机场 Wi-Fi 比…食物更重要？！

> 原文:[https://www.sitepoint.com/airport-wifi-food/](https://www.sitepoint.com/airport-wifi-food/)

最近一项针对航空旅客的调查发现，47%的受访者认为 Wi-Fi 是最重要的机场便利设施。奇怪的是，它比食物等基本旅行需求高出近 30%！

如果我们看得更深一点，我们会发现调查数据是由美国航空公司和惠普公司收集的——很难说是最公正的合作伙伴。这项研究还局限于 1500 名每年乘坐三家或更多航空公司至少 20 次航班的商务旅行者。我怀疑那些尖叫着要快餐的孩子的父母会有完全不同的结果。

然而，虽然这一结果应受到一定程度的怀疑，但它确实表明确实有普及互联网接入的需要。企业用户可以及时处理电子邮件和其他工作，而不是等待几个小时的航班。这当然比逛无止境的商店强，商店里卖的都是无用的东西，除了机场，在任何地方都卖不出去。

调查还发现，最大的抱怨不是航班延误、拥挤的条件或过高的食品价格:而是笔记本电脑电池没电了。超过 41%的人抱怨电池没电，26%的人抱怨机场和飞机上没有电源插座。

让我们希望机场和航空公司注意并实施适当的免费 wi-fi，而不是不存在或昂贵的接入点。
