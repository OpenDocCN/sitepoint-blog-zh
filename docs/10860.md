# jQuery 基础:免费在线书籍

> 原文:[https://www.sitepoint.com/jquery-fundamentals-free-book/](https://www.sitepoint.com/jquery-fundamentals-free-book/)

jQuery 并不缺少文档。主要的 jQuery 网站提供了大量有用的信息。像 [jQAPI](http://www.jqapi.com/) 这样的资源可以帮助定位特定的方法。如果这还不够，你可以浏览一下 [jQuery 源代码](https://www.sitepoint.com/jquery-source-viewer-tool/)。所有这些资源都很棒，但是它们是针对那些想要找到关于特定功能的文档的有一定能力的开发人员的。不熟悉 jQuery 或 JavaScript 语言的开发新手怎么办？

如果我忘记提到 SitePoint 的书，比如 Earle Castledine 和 Craig Sharkie 写的精彩的《jQuery:忍者新手》( T1 ),那我就失职了。然而，在你等待它被交付的时候，我可以推荐丽贝卡·墨菲的[**【jQuery Fundamentals】**](http://www.rebeccamurphey.com/jqfundamentals/book/release/html/)。这是一本全面的在线书籍，指导你从 JavaScript 基础到编写 jQuery 插件。即使是专业的 JavaScript 开发人员也会发现一些他们不知道的东西。

这本书很棒。它写得很好，很简洁，包含了大量的例子和 JavaScript 陷阱。更何况是免费的。

这些部分包括:

*   JavaScript 基础知识
*   jQuery 基础知识
*   实用方法
*   事件和影响
*   埃阿斯
*   编写 jQuery 插件

还有一个所有例子的链接列表。当您忘记匿名自执行函数或克隆 DOM 节点的确切语法时，这很有用。

如果你一直打算学习 JavaScript 和 jQuery，不要再拖拖拉拉了，去读读 [jQuery 基础知识](http://www.rebeccamurphey.com/jqfundamentals/book/release/html/)然后给[丽贝卡](http://twitter.com/rmurphey)写一封低声下气的感谢推特吧！
