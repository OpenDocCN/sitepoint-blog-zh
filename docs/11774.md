# 我能要薯条吗？客户真正想要的

> 原文:[https://www.sitepoint.com/what-do-clients-really-want/](https://www.sitepoint.com/what-do-clients-really-want/)

与任何客户建立成功和长期关系的一部分是理解并能够预见他或她的个人需求。这可能很难做到，尤其是当你和一个新客户一起工作的时候。

好消息是，不管你在做什么类型的工作，在与服务提供商的关系中，有几个核心的东西是大多数客户想要的。如果你能预测这些基本需求，你就走上了创造一个有很大潜力的互惠关系的道路。

以下是几乎所有客户在恋爱关系中想要的一些最基本的要素。

客户想知道他们正在为这项工作雇用合适的人。

在每一个成功的关系中，客户必须有一定程度的信任，认为供应商有必要的技能和经验来做好这项工作。

客户希望得到尊重。

相互尊重是每一个客户/供应商关系的基本要素。如果有一方在这方面有所欠缺，连接注定失败。

客户希望与专业人士合作。

专业并不仅仅意味着拥有所需的技能和经验。反应能力、责任心、声誉和许多其他因素也是成为专业人士的一部分。

客户希望感受到被重视、被倾听和被交流。

作为一个客户，可能没有什么比感觉自己对自己的项目一无所知更糟糕的了。客户需要一个平易近人的联系人，在过程中的每一步都被告知，并且他们的信息得到及时和完整的回复。

客户希望与对项目充满热情的人合作。

激情可以传染。你对客户的工作越积极，客户就越信任你，也就越容易合作。

客户想知道你会如何处理他们的信息。

客户数据是一项宝贵的资产，您的客户应该知道您是如何处理这些数据的，尤其是任何可能被视为敏感的数据(如密码)。你的工作是保护这些数据，并向你的客户保证这些数据在你手中是安全的。

客户希望感觉他们的钱花得值。

客户觉得他或她所做的投资越大，证明你所做的工作物有所值就越重要。要做到这一点，你可以兑现你的承诺，并在每一步都超越期望。

这些核心需求中的每一个都会导致其他需求，这些需求可以促进任何工作关系，并使这种关系得以持续。如果你一开始就做得很好，你会发现满足客户的需求是很容易的，你将能够发展出一种精心设计的给予和接受。如果你错过了客户需要和想要的东西，你可能会发现自己在为项目的生命而奋斗。

你会给这个基本客户需求列表添加什么？当与客户合作时，你如何预测这些需求？
