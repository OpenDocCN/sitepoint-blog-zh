# 你会在谷歌的 Wave 上冲浪吗？

> 原文：<https://www.sitepoint.com/will-you-surf-on-googles-wave/>

你还记得 Google Wave 吗？HTML5 项目是谷歌在 2009 年更令人兴奋的努力之一。谷歌认为电子邮件陈旧、笨重，需要为 21 世纪的 T2 重新设想。Wave 将是在线通信发展的下一步；它在一个中心位置结合了电子邮件、聊天、头脑风暴、日程安排、文档、推特和订阅。更重要的是，开发者拥有完整的 API 来开发他们自己的协作工具和小部件。

Wave 于 2009 年 9 月 30 日推出，是一项只接受邀请的预览服务。炒作和兴奋在同一天消失了。这个应用程序不直观，错误百出，很少有人理解它。部分问题在于 Wave 的界面:它看起来就像另一个电子邮件客户端。谷歌也发现很难解释 Wave 做了什么以及它提供的好处。

谷歌随后用 Buzz 打进了一个乌龙球。充其量，它盖过了 Wave。最糟糕的是，它造成了混乱，并引发了一些令人担忧的安全问题。

和很多人一样，我尝试过 Wave，但很快就放弃了。很少有用户采用这个系统，所以很少有机会尝试它的协作方法。尽管有缺点，人们理解电子邮件，每天使用它，很少遇到大的麻烦。

然而，谷歌仍然相信 Wave。他们重新启动了这个项目，并希望吸引新用户。你不需要邀请，所以在 wave.google.com T2 的每个人都可以参加。其实也不尽然*【大家】*:系统在 ie 浏览器加载失败，不支持 Opera。

谷歌的[博客文章](http://googlewave.blogspot.com/2010/05/google-wave-available-for-everyone.html)称:

> 如果你不久前试用了 Google Wave，发现它还没有准备好投入实际使用，现在是时候回来再试一次了。Wave 比我们开始预览时快得多，也稳定得多，我们也努力让 Wave 更容易使用。
