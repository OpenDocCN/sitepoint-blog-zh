# SitePoint 论坛报传奇下台

> 原文：<https://www.sitepoint.com/sitepoint-tribune-legend-steps-down/>

SitePoint 广受自由职业者和企业主欢迎的电子邮件通讯的长期关键人物布兰登·辛克莱(Brendon Sinclair)在任职五年半后离职。

在论坛报网站的最后一期中，他分享了一些关于继续前进的想法:

> 我一直有点坚持让时事通讯变得有信息性和娱乐性，但五年多后，我耗尽了精力，现在觉得是时候把接力棒传递下去了。

> 对我来说，这是一个写《论坛报》的绝好机会，我要感谢所有提出问题和反馈的读者。我希望我的努力对你的在线业务或营销技能有所帮助，哪怕只是一点点。

在掌舵的五年半时间里，布兰登撰写了 167 期杂志，并在扩大《论坛报》的订户基础方面发挥了重要作用。

Brendon 也是 SitePoint 最畅销的[网页设计商业工具包](https://www.sitepoint.com/kits/freelance2/)的作者，更新后的第二版继续畅销，帮助成千上万的企业主建立、营销和扩展他们的网络服务业务。该工具包是 Brendon 多年来在澳大利亚昆士兰州成功运营[网络服务业务的经验结晶。](http://tailored.com.au/)

《论坛报》的合著者迈尔斯·伯克(Miles Burke)将继续定期为时事通讯撰稿，几位令人兴奋的新作者也将在未来几周内发表客座文章。

布兰登，我们感谢你多年来的智慧之言！

你消息灵通吗？今天就订阅《论坛报》或 SitePoint 的其他重要时事通讯吧！
