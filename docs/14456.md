# Photoshop 的线性烧伤

> 原文:[https://www.sitepoint.com/photoshops-linear-burn/](https://www.sitepoint.com/photoshops-linear-burn/)

Photoshop 的线性加深混合模式会产生类似于乘法模式的效果，其中混合层中较亮的区域允许底层“显示出来”，但有一点不同，它实际上会使底层的颜色变暗。以下是官方描述:

> 查看每个通道中的颜色信息，并通过降低亮度使基色变暗以反映混合色。与白色混合不会产生任何变化。

听起来几乎与颜色加深描述相同，只是线性加深不是“增加对比度”，而是“降低亮度”并使基础层变暗。事实上，除非你在混合层上有一个完全白色的区域，否则基础层将一直变暗；使用线性加深总是会导致图像变暗。

例如，从这幅玫瑰图开始:

![](../Images/348522d1d9d1ae4db9871735ac3cc968.png)

复制图层并将顶层设置为线性加深会导致整体图片更暗，尽管较暗的区域“更暗”

![](../Images/9f27935facc2beca75bece61323da762.png)

现在进行实际应用。我先从这两张(裁剪过的)图片开始，这两张图片是我在[istockphoto.com](https://www.istockphoto.com)花几分钱下载的:

![](../Images/2ca95aecade775bbfdda3b30a54deaad.png)

把叶子层放在上面，我把它设置为线性加深，这是我得到的结果:

![](../Images/638000870eeb9ab7bd539961bca4c2b4.png)

与乘法模式形成对比:

![](../Images/87ebd1a8c32b6715baa09bfffa9aa341.png)

线性加深的例子稍微暗一点，线性加深比乘法模式能给你更多的“黑色”。这两种效果都是有用的…对我来说，这只是取决于我那天的“心情”!
