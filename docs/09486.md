# 2011 年 8 月浏览器趋势:火狐 5 增长 400%

> 原文：<https://www.sitepoint.com/browser-trends-august-2011/>

2011 年 7 月相对平静——也许在浏览器市场经历了几个月的疯狂活动后，供应商们正在享受暑假。

然而，这并没有阻止善变的用户转换应用程序。头条新闻是 Firefox 5 的用户数量在过去的 31 天里翻了两番多。但是您需要查看完整的 [StatCounter 统计数据](https://gs.statcounter.com/)来揭示真实情况…

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 8.1%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

虽然 Firefox 5.0 增长迅猛，但它是在 6 月底发布的，因此 7 月份提供了第一个完整月的使用统计数据。与此同时，Firefox 4.0 下降到了 3.38%——实际上，这两个版本在表中交换了位置。

火狐 4.0 和 5.0 加起来增长了 0.81%。然而，3.x 版本和之前的版本下降了 1.19%，Mozilla 浏览器的整体净损失为 0.38%。

IE9 又过了一个不错的月份，因为更多的用户迁移到了新的浏览器，可能还有 Windows Vista 或 7——唯一能运行它的操作系统。IE8 的使用率继续以每月 1%以上的速度下降。IE6 和 ie7 的使用率再次下降，虽然这是个好消息，但使用这两种浏览器的人仍然比 Safari、Opera 和“其他”浏览器的总和还要多。总体来看，IE 净亏损 1.13%——[或许是那些用户想增加智商？！](https://www.sitepoint.com/ie-users-low-iq/)

令人惊讶的是，Chrome 的使用量在 7 月份又增长了 1.5%——这是有史以来最大的月度增长之一。谷歌正在迅速赶上 Mozilla，Chrome 12 再次超过 Firefox 4/5，成为世界上第二大使用浏览器。也就是说，快速更新的时间表混淆了统计数据。当 Chrome 13 发布时，Firefox 5 将夺回第二的位置，这一循环将再次开始。

Safari 有适度的增长。歌剧略有下降。然而，Opera 已经很久没有获得市场收益了。

## 移动浏览器的使用

根据 StatCounter 的数据，[桌面浏览器占据了 web 活动](https://gs.statcounter.com/#mobile_vs_desktop-ww-monthly-201107-201107-bar)的 92.98%。上个月，手机使用量增长了近 0.5%，至 7.02%。这可能是一种季节性异常，有趣的是，移动浏览器市场几乎没有什么变化。也许那是因为我们离购买礼物的季节还有几个月…

1.  opera Mini/Mobile——22.07%(下降 0.74%)
2.  安卓系统——18.17%(增长 0.92%)
3.  诺基亚浏览器——17.10%(下降 0.56%)
4.  iPhone——15.10%(下降 0.12%)
5.  黑莓——12.30%(增长 0.32%)

Android 已经取代诺基亚的浏览器成为第二大浏览器。

iPhone 继续小幅下滑，尽管应该注意的是 iPod Touch 以 4.85%的份额排在第六位。苹果享有不成比例数量的技术新闻和开发教程报道。极客们可能喜欢这些设备，但要知道它们在移动浏览器市场的份额还不到 20%。
