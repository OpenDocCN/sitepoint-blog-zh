# 雅虎！想要加强你的网站搜索

> 原文：<https://www.sitepoint.com/yahoo-wants-to-power-your-site-search/>

当 CNN 在 2007 年 6 月推出他们的新网站时，他们抛弃了雅虎！支持谷歌在 CNN.com 搜索他们的网站。雅虎！然而，在网站的国际版上仍然使用。由于我在美国，我不经常查看 CNN 国际网站，但我今天决定去看看。我不确定现在是谁在网站上推动搜索，但是雅虎！logo 没了，所以看起来他们甩了雅虎！那里也是。有趣的是，这似乎是一个共同的趋势——我在网站搜索框旁边看到更多的“由谷歌驱动”按钮，而不是“由雅虎驱动”徽章。

谷歌的网站搜索非常容易插入，由此带来的用户体验非常好，以至于已经成为网络上成千上万个 T2 网站的中流砥柱。今天，雅虎！宣布了一个改进的网站搜索产品，旨在从谷歌的搜索市场份额中分一杯羹。

雅虎！ [Vertical Lens](http://www.ysearchblog.com/archives/000659.html) 是[打造自己的搜索服务](http://developer.yahoo.com/search/boss/) (BOSS)平台的一部分，允许开发者创建超越谷歌即插即用服务的网站搜索引擎。雅虎！这项倡议的发起伙伴是科技博客 [TechCrunch](http://www.techcrunch.com/) ，他们现在使用垂直镜头进行网站搜索。

![](img/5f49a544a67113a11162be38b8ec8358.png "yahoo-vertical-lens")

新产品提供的功能包括:

*   **专有内容的实时索引**–当新的博客帖子或评论添加到网站时，搜索索引几乎会立即更新以反映这些变化。(谷歌也很擅长掌握网站变化，所以这里没什么新鲜的。)
*   **定制排名**–BOSS 全是定制雅虎！搜索结果，所以垂直镜头允许网站找到调整算法，以适应他们的观众和用户体验。
*   结构化搜索(Structured search)–也许新的垂直镜头最酷的一面是它支持搜索的多面细化。对于 TechCrunch 来说，他们的新搜索引擎允许用户根据作者、评论数量、博客类别或日期来缩小查询结果。
*   **在单一搜索显示中融合网络和专有内容**–Vertical Lens 可以搜索网站和网络内容，并将结果融合在单一显示中。

雅虎！垂直镜头目前只提供给合作伙伴，但雅虎！s YaJie Ying 承诺，该公司“正致力于通过 BOSS API 更公开地分享技术。”
