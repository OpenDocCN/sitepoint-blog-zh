啊，..阿贾克斯。当它运行良好时，它可以如此漂亮，但当它让开发人员看起来(IMHO)很傻时，很少有技术可以与之匹敌。正如卡梅隆·亚当斯最近所说:

> Ajax 使 web 页面能够像桌面应用程序一样使用不可见的数据通信和独立页面元素的无缝刷新。这有可能毁掉网络，也有可能将它推进一个新时代。

这是一个很好的例子，我认为它包含了更多的 A 列内容和更少的 b 列内容。

Art.com 是一个大型的商业艺术网站，向公众出售超过 300，000 幅版画，包括装裱、装裱或其他形式。他们的展示页面是高效优雅的典范——在某些方面与 Flickr 风格相似——通常让作品自己说话。

但显然，没有足够的“惊喜因素”。“我们付钱给这些开发者——让他们想出一些酷的、热的或恶心的东西，或者现在孩子们想成为的任何东西”。

那么他们的解决方案是什么？

![Art.com pioneers the post-modern dropdown](../Images/0b64c28707922510a00308afacb55b6f.png)

Dropdowns 已经重生了！如果你点击一个下拉菜单，并期望它会' ***下降***“***下降*** '那么我恐怕你对自己是个傻瓜，对别人是个负担，我的朋友。[试试这里的下拉尺寸选择器](http://www.art.com/asp/sp-asp/_/pd--12103385/Aerial_View_towards_Chrysler_Building_and_Queensborough_Bridge_Manhattan_New_York.htm)看看未来。下拉菜单上的向下箭头实际上意味着“在屏幕中间具体化，然后向左上方滑动”。太棒了。

嘿，我想人们的想法是，如果你没有 JavaScript，你很可能要么是古怪的，要么是破产的，或者两者兼而有之。
