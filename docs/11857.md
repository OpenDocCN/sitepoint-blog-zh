# 伟大的领导者是天生的还是后天培养的？

> 原文:[https://www.sitepoint.com/are-great-leaders-born-or-made/](https://www.sitepoint.com/are-great-leaders-born-or-made/)

有些人认为，领导者天生就具备成功领导者的必备素质。而另一些人认为，领导能力，像许多其他类似的特征一样，可以在生活中学习和发展。对我来说，我认为很多争论取决于你如何定义领导力。

**定义领导力**

让我们假设领导力意味着拥有激励和影响他人的能力和愿望。基于这个定义，当我看到我的女儿用她与生俱来的能力去影响她的弟弟(有时是她的父母)去做她希望他们做的事情时，我当然可以看到领导者天生具有领导素质的论点。

如果领导力意味着勇敢并愿意为你周围的人的进步大声疾呼，我认为领导者不必天生如此。我见过那些认为自己是内向者和整体追随者的人在面对他们热衷的问题时成为成功的领导者。

如果我们将领导力定义为拥有非典型的智力、创造力和/或驱动力，那么我又回到了相信这是一种与生俱来的特质，尽管你可能会也可能不会在你的一生中继续发展这种特质。

那么这一切意味着什么呢？

任何人都可以成为领导者

在我们的生活中，我们都有天赋和成功的倾向。如果这也是你感兴趣的领域，你可能会散发出生活中其他领域所没有的特质。因此，虽然从严格意义上来说，你可能不是天生的领导者，但当你受到激励时，你肯定可以克服许多障碍，培养领导的愿望和能力。

让我们看一个例子。

假设你是一个动物爱好者，你在空闲时间为一个动物救援组织做志愿者。这个团体正在计划一个大规模的筹款活动，但是缺乏有效执行的必要指导。没有一个合适的领导者，这个活动注定会失败。你可能不太适应在别人面前说话、为一个团体做决定和管理他人，但是你对筹款活动的需求如此强烈，以至于你愿意克服这些挑战。在几名在各个领域(市场营销、活动策划、社区拓展等)经验丰富的志愿者的支持下。)，以及你对团体和筹款人背景的了解，你成功地管理了团体和活动。可能吗？绝对的。

**培养领导力**

除了将激情转化为行动，还有几种方法可以培养你自己的领导技能:

*   努力提高你的沟通能力，尤其是倾听他人的能力
*   愿意承担风险
*   建立关系网，寻找扩大人脉的方法
*   专注于正直地工作和生活，尊重他人
*   鼓励你周围的人找到自己的领导素质
*   对自己和自己的能力有信心
*   了解你的优势，并准备在你欠缺的地方寻求支持
*   创造一个愿景，并分享你的愿景来激励他人
*   积极思考并致力于实现你的目标

你怎么想呢?领导者是天生的还是后天培养的？
