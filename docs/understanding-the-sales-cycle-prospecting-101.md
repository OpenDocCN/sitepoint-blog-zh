# 了解销售周期:发现潜在客户 101

> 原文：<https://www.sitepoint.com/understanding-the-sales-cycle-prospecting-101/>

在我的系列文章*了解销售周期*中，我说过销售过程以两种方式开始，要么潜在客户联系你，要么你联系潜在客户。

联系潜在客户——也称为潜在客户——可能是一件可怕的事情。对许多人来说，寻找新业务并不是他们销售过程的一部分，有时是有意的，但更多时候是默认的。然而，为了赢得新客户，你必须与目标市场的人进行对话。有许多方法可以做到这一点:

1.  网络小组
2.  上门推销
3.  社会化媒体
4.  举办免费研讨会或讲习班
5.  在有不良网站的公司大厅里闲逛

好吧，我不会推荐最后一个，但你明白我的意思。但获得人们的关注只是第一步。我还提到了一份[营销白皮书](http://info.jigsaw.com/MP_LG_GTTP.html)，其中写道:

> 在销售和营销中，只有两件事很重要:
> 
> 1.  引起人们的注意，还有
> 2.  当你有的时候你告诉他们什么
> 
> 绝大多数投资和注意力都放在第一个方面——无论是潜在客户、需求客户、直邮广告、搜索引擎优化、谷歌广告词、电话推销还是其他。
> 
> 但问问任何一位成功的销售人员或营销人员:一旦你与他们通了电话，或者一旦他们拿起你的营销材料或访问你的网站——如果你没有非常有说服力的东西告诉他们——你就失去了他们。

你需要“一份关于客户从使用你的产品、服务或解决方案中获得的切实商业成果的清晰陈述”。这意味着你应该提前计划好你要说什么。有些人可能会称之为独特的价值或销售主张。在社交圈子里，这被称为电梯演讲。在电话推销中，那会是一个剧本。这里有一个不好的例子:

> 我是网页设计领域的专家。你可能看过我的一些作品:*******。com，* * * * * * * * * *。我可以以最低的价格为你的企业设计一个网站。如果你感兴趣，我可以给你一个小时的免费咨询。

这句话的问题在于“我”这个词……“我是……”我可以……“我的作品……”写这个剧本的人很可能是一个有着伟大作品的杰出设计师。问题是，这个人看着他或她能做什么，然后问，“我怎么卖这个？…而不是“我能解决什么问题或提供什么解决方案?”？“创建独特的价值主张有一个简单的公式:

1.  你是谁，你做什么
2.  你为谁而做(例如，你的目标客户)
3.  需要解决什么(用他们的语言描述需求或机会——需要解决的具体问题或他们正在寻找的解决方案)
4.  您如何帮助他们解决这个问题(即利益陈述)

让我们来看看虚构的 XYZ 创新公司，它开发了一种新的营销工具来帮助一个利基行业，包括当地拥有的房地产和保险代理。

1.  你是谁，你做什么:XYZ 创新提供营销工具
2.  你为谁做:本土保险机构
3.  需要解决的问题:找到更多合格的销售线索，进行更多的面对面会谈。
4.  你如何帮助他们解决这个问题:这是一种新的营销工具，已被全国各地的保险机构成功地用来识别他们最想要的客户，并有选择地向他们推销。

以下是你如何把它变成电梯演讲或电话推销的脚本:

> XYZ 创新公司帮助本土保险机构获得更多合格的潜在客户，获得更多面对面的会议，并撰写更多的保单。
> 
> 我们有一种新的营销工具，已经被全国各地的保险机构成功地用来识别他们最想要的客户，并有选择地向他们推销。

说到冷打电话，许多人拒绝使用脚本，因为他们不想听起来像是照本宣科。但是听起来像是在读剧本的问题在于你的表达，而不是使用剧本。几个月前，在进行了一些电话销售培训后，我要求每个人拿出一份脚本，然后练习到完美。在给每个人分发新剧本后不久，其中一个跑到我的办公桌前，说她刚刚安排了一个约会。同意见面后，他问她，“你想知道我为什么说‘好’吗？因为你听起来非常专业，我能看出你不是在照着稿子念。”

如果你害怕听起来像照本宣科，那就练习，直到你听起来很自然。这适用于电话推销或电梯演讲。来吧，你试试。
