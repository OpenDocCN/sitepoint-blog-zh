# 平面 UI 设计入门

> 原文:[https://www.sitepoint.com/getting-started-flat-ui-design/](https://www.sitepoint.com/getting-started-flat-ui-design/)

《福布斯》最近的一份报告将扁平化设计的日益增长的趋势坚定地归因于人们对数字界面越来越熟悉。作者说，在过去的几年里(1995 年至 2003 年)，设计师只为熟悉桌面软件的“某一类专业人员”设计，这意味着设计时要考虑可点击性。

快进到今天和 iOS7 的最新版本，现在我们仍然在设计中把用户放在我们头脑的最前沿，采用平面 UI 设计。在 iPhone 的早期，网络趋势也因苹果设备而变得流行，但着眼于气泡、阴影和眩光，仅举几个流行的 UI 技术。

《福布斯》的文章称，正是苹果的这一早期干预帮助现代用户真正适应了在更多设备上使用网络，而不仅仅是台式机或笔记本电脑。“我认为扁平化的设计趋势是网页和界面设计领域日益成熟的一个征兆，”文章写道。

“这种成熟度适用于我们的设计师——他们越来越擅长制作鼓励交互性和参与性的界面——也适用于我们的用户。”

当涉及到他们如何、在哪里以及为什么使用互联网时，用户当然变得越来越老练，正因为如此，让闪亮的按钮尖叫“点击我”的趋势现在已经谢天谢地地成为过去。

## 从哪里开始

趋势都很好，就我个人而言，我喜欢目前在设计界大量涌现的极简、扁平和简单的设计，但这并不是说你的所有客户都会喜欢。

那么首先要从平面 UI 设计是否适合你正在从事的项目开始。如果客户有一个好的、强有力的品牌形象，那么看看你能从颜色、布局和内容策略方面得到什么，以确保它符合你的想法。

最重要的是，把用户想成，如果客户的想法过时了，让你感到害怕，那么你就有一个很好的理由去克服它。用户可能比过去更懂网络/技术，但当涉及到可点击区域等时，他们仍然需要一定的指导。

## 平面设计的要素

1.  缺乏深度
2.  简单元素的使用
3.  排印
4.  颜色
5.  最低纲领

### 格子

和所有的 UI 设计一样，网格扮演着最重要的角色之一，平面设计也不例外。当你只使用几个元素时，它建立了视觉顺序，并给你一个工作指南。

使用网格来定义你的内容，把用户的注意力吸引到你希望他们去的地方。你可以使用更紧凑的网格来体验不同的视觉效果，这样你就可以衡量可以包含多少内容，而不会让整个界面看起来拥挤和混乱。

### 颜色

你可能已经注意到，平面设计往往色彩丰富，从同一个调色板中充分利用不同的色度和色调。重要的是不要被太多的颜色冲昏头脑，记住非常强烈的颜色可能会很霸道。

考虑到这一点，考虑用柔和的颜色，把最强烈的颜色留给你的可操作区域来吸引眼球和吸引点击。红色和橙色是很好的号召按钮，但是注意不要过度使用，因为这会赶走用户。

看看[平面 UI 颜色](http://flatuicolors.com/)寻找灵感和想法。

![colors](../Images/35e1ec37fe9857e32abfa88820acbb86.png)

### 排印

这是平面设计的另一个重要部分，重要的是排版要与整体设计相适应，使用简单易读的强大、醒目的字体。也要考虑可读性，使用简单、可靠的词语，并尽量减少，保持一致的语气和声音。

也要考虑留白的使用，你的设计应该理想地能够一目了然，有清晰的信息和视觉指示器来引导用户去你想去的地方。
