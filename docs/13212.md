# 谷歌将放弃 PageRank？

> 原文:[https://www.sitepoint.com/google-to-abandon-pagerank/](https://www.sitepoint.com/google-to-abandon-pagerank/)

谷歌化博客报道称[谷歌实验室](http://google.com/experimental/)正在试验一种新的搜索结果定制方式:digg 风格的投票按钮，允许用户定制他们的搜索体验。如果实验成功，可以想象，这种投票的结果可以折叠到一般的搜索结果中，补充甚至可能取代 PageRank 算法。

[![Google search results with digg-style voting](../Images/a99d5a26bd18861826f042d3bcc33386.png)T2】](http://www.google.com/experimental/a840e102.html)

从[实验的描述页面](http://www.google.com/experimental/a840e102.html):

> 这个实验让您通过添加、移动和删除搜索结果来影响您的搜索体验。当您再次搜索相同的关键词时，您会继续看到这些变化。
> 
> ![Google’s I like this voting button](../Images/12c4576b9760dcc219067e8bc3717269.png) **喜欢吗？**
> 该按钮会将结果移动到页面顶部，并在旁边添加橙色标记(图 1a ),以便您可以轻松识别。无论您将来何时搜索相同的关键字，您推广的结果都会出现在顶部。
> 
> ![Google’s Bury This button](../Images/79daaa7df599c627a59d535ebed9317e.png) **不喜欢？**
> 该按钮将删除搜索结果，以后您搜索相同的关键词时，该按钮将保持隐藏状态。

虽然最初的增强仅限于用户的结果，但你可以打赌，谷歌将非常密切地监控他们跟踪的数据，并将其与他们现有的算法进行比较。

PageRank 只能提供定制的搜索结果，这是改善谷歌用户个人体验的一种有趣方式。
