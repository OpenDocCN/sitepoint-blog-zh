# 虚拟主机带宽——简介

> 原文:[https://www.sitepoint.com/bandwidth-introduction/](https://www.sitepoint.com/bandwidth-introduction/)

在选择主机时，你购买的带宽对你网站的成功至关重要。一般来说，你的带宽越大，你的网站一次能处理的流量就越多。

 **##### 我需要多少带宽？** 

要确定您需要多少带宽，您必须首先:

*   估计一下你网站上的每个页面有多大
*   有多少人会去看。

为此，将页面上每个图像的大小和页面 HTML 文件的大小相加。然后将这个数字乘以你期望该页面每月的浏览量。

例如，如果您的页面上有三个 10k 的图像和一个 2k 的 HTML 文件，那么该页面上将有 32k 的数据。再乘以你的预期页面浏览量(假设每月 100，000 次)，你会得到该页面当月要传输的 3.2G 数据。

现在重新计算每个页面的这个数字，你就会知道整个网站大约需要多少带宽。

##### 如何节省带宽？

优化带宽使用有三种主要方法:

**1。让你的页面尽可能的小。**
这意味着紧密的 HTML 编程来减少文件大小，压缩你的图片和图形来减少图像大小( [NetMechanic](http://www.netmechanic.com) 有一个免费的，易于使用的文件压缩工具)。使用 JPEG 图像格式的照片和 GIF 格式的图形-他们的压缩能力是首屈一指的。

HTML 博士的网站有一些有用的工具，可以分析你网站的图片大小、传输量、表格结构等等。

**2。使用未存储在您网站上的图像。**
你可以通过将你的< IMG SRC >标签中的文件名替换为给出你想要使用的图像位置的 URL 来实现。但是，在此之前，请确保您拥有所链接网站的权限。否则，你就是在窃取带宽，这被认为是盗版，因此是非法的(更多信息，请参阅“我如何防止带宽盗窃？”下面)。

**3。使用级联样式表(CSS)。**
另一种节省带宽的有效方法是使用 CSS。旨在减小 HTML 文件的大小，在文档的开头为任何 HTML 元素/命令添加 CSS 属性值，而不是从头到尾重复它们。

关于 CSS 和其他与带宽节约相关的资源的更多信息，请访问 [Infohiway](http://www.infohiway.com/faster/) 或 [SitePoint CSS 教程。](http://www.webmasterbase.com/article.php/309)

##### 如何防止带宽盗窃？

未经许可通过你的标签链接到另一个网站的图片来使用它们是一种双重盗版:你在盗版网站的图片和带宽。尽管这很不道德，但这是许多网站管理员每天都要面对的现实。

**监管**
防止带宽被盗的最基本方法就是监管。分析搜索引擎、日志和其他网站，找出未经许可使用您的图像(和您的带宽)的人。

一旦你确定了违规者，你可以联系他们，命令他们停止链接到你的网站(如果他们试图告诉你互联网上的一切都是“公共领域”，不要买它)。不幸的是，自己监管网站是一项耗时的任务。

商业软件
保护带宽的一种成本更高但更省时的方法是利用现有的软件包。

像 [WebTrends](http://www.webtrends.com/redirect/14ASJP.htm) 这样的标准日志分析程序可以帮助跟踪带宽窃贼，因为像这样的大多数软件套件可以识别您网站最受欢迎的推荐者和访问者。有了这些信息，你就可以反向查找是否有网站搭你的便车。大多数虚拟主机都提供某种日志分析程序作为其主机包的标准功能。

如果你正在寻找一些更自动化的东西， [Artistscope](http://www.artistscope.net) 提供了许多实用程序，可以通过加密、安全虚拟主机和其他方法来保护你的图像。保护计划起价低至每 100 次页面浏览 10 美元。

或者，对于一个极其安全的高端解决方案， [Digimarc](http://www.digimarc.com) 开发了一项专利数字技术，在所有文件上嵌入“水印”，为您的电子财产提供版权保护和跟踪。
