# 避免电子商务托管恐惧

> 原文:[https://www.sitepoint.com/ecommerce-hosting-horror/](https://www.sitepoint.com/ecommerce-hosting-horror/)

似乎缺乏关于虚拟主机的信息。这让我很惊讶，因为虚拟主机实际上是你网站的支柱。如果你被一个劣质的主机卡住，你的网站就不会出现在网上。你的电子邮件发不进来。你的购物车不能用了。没有一个可靠的虚拟主机…你的网上业务将会破产！

那么，如何从带宽、兆字节、POP3s、FTPs 和 SSL 中得出正面或反面呢？由于缺乏可用的信息，大多数网站所有者做出两种选择之一:他们选择最便宜的软件包，或者他们选择提供最多功能的软件包“只是为了安全”。

请允许我定义几个你在寻找理想主人时可能会听到的常见术语。我还将提供一些指导方针，帮助您做出明智的决定，明确哪些特性是您真正需要的。

##### 基础知识

***存储空间***

这是您在主机服务器上租用的空间量，以兆字节为单位。那么你需要多少钱来建立你的网站呢？嗯，在大多数网站上，一个包含有限图形和一些文本的页面需要(平均)5kb。如果你有沉重的图形，照片等。您的页面可能需要 30 Kb。

将页面数量乘以千字节数，计算出您的网站目前需要多少空间的粗略估计。此外，不要忘记考虑你将存储在服务器上的其他东西所使用的空间。电子书占用了大量的空间，Flash 电影也是如此。它们也将存储在主机的服务器上，并且需要包含在您的评估中。试着预测一下你将来可能会给网站添加什么，并把它包含在你的总量中——你会想要留出一些增长的空间。

一个小提示:只要看看你电脑上的文件索引，就能知道每一页、电子书或 Flash 演示文稿的大小。

***带宽***

当文件从主机服务器传输到站点访问者时，它们会耗尽带宽。当页面被点击并显示在屏幕上时，当电子书被下载时，当购物车被投入使用时，带宽开始发挥作用。你的网站越“活跃”——网站访问者能做的事情越多——你需要的带宽就越多。

***邮箱账户***

网络主机提供两种主要类型的电子邮件。网络邮件是可以通过浏览器在线访问的电子邮件。你打开它，就像浏览其他网站一样。POP3 电子邮件可以与 Outlook Express、Outlook、Eudora 或 Pegasus 等电子邮件软件(称为电子邮件客户端)一起使用。

你需要小心的是决定你将如何管理你的电子邮件账户，以及每一种选择涉及的费用。一些主机提供有限数量的 POP3 地址，然后向您收取额外的费用。其他人可能不会提供“控制面板”来维护您的电子邮件地址，并且可能会向您收取费用来处理您需要进行的任何更改。

##### 临时演员

现在让我们考虑几个你可能听说过的项目，看看它们如何帮助你顺利经营网上业务。

 **另一项非常方便的服务是自动回复。这将允许你自动发送信息给给你发邮件的人。当您在度假时，或者当您想让客户放心，他们的信息已经传达出去，而不必每次收到电子邮件都亲自回复时，您会发现这项服务非常有用。

***FTP 和首页***

FTP 代表文件传输协议。这是用于将您的网站上传到主机服务器的协议。大多数网页设计软件都包括一个结合了 Dreamweaver 和 Go Live 的 FTP 应用程序。注意，微软的首页不包含 FTP。它也有一些其他的特殊考虑，使你的主机有必要特别支持 Front Page，如果那是你使用的软件——与你的潜在主机核实更多细节。

***【SSL(安全套接字层)***

这是允许安全、加密的数据传输的过程。有些人将此与接受信用卡相混淆。虽然大多数购物车和在线商家帐户使用 SSL，但这并不意味着您的主机在您的主机包中包含一个商家帐户。

除了支付选项之外，SSL 还可用于收集网站访问者的敏感数据，以确保您的电子邮件不会被在线拦截，并为您的客户提供安全感。

##### “还有更多吗？!"

是啊！选择网站主机是一个严肃的过程。在这个决策过程中需要考虑的其他几个问题围绕着你的网站的功能。

*   你会在网上购物吗？如果是这样，你需要一辆购物车。
*   你会提供电子杂志或时事通讯吗？如果是这样，找到一个提供列表服务器的主机将是有益的。如果没有，你可以找一个免费的(包括广告)或者你必须从另一个提供商那里付费。
*   您想跟踪您网站的访问者吗？那将是非常明智的！请务必查看主机提供的流量统计类型。不幸的是，有些不提供非常详细的信息。至少你会希望看到引用的网址(访问者来自哪里)，每月的点击数，以及你的网站上访问量最大的页面。
*   你认为你会不时地有问题吗？那么找到一个通过聊天或电话提供 24/7/365 技术支持的主机将是一个很好的主意。

最后，但同样重要的是，一定要检查主机的“正常运行时间”和安全性。正如我在本文开头提到的，如果主机不可靠，您的企业将会倒闭！当务之急是，您的主机必须采取措施，以防崩溃或停机。同样地，你会想要一个主机来为你存储在他/她的服务器上的信息提供安全保障。了解他们采取的防火墙和其他安全预防措施。

知识就是力量！知道该找什么可以帮你省下很多钱，时间和烦恼。它还能让你的企业继续运营。**
