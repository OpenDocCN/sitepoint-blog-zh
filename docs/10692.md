# 用照片作为灵感

> 原文:[https://www.sitepoint.com/using-photos-as-inspiration/](https://www.sitepoint.com/using-photos-as-inspiration/)

今天我为你准备了一些不同的东西。我有一个客座博主，他实际上有一些有趣的东西给你，而不是必须阅读我的另一个帖子。你可能还记得上个月我在[论坛](https://www.sitepoint.com/forums/)举办了一场摄影比赛。它被命名为“伟大的摄影挑战”,今天我们很幸运地邀请到了这位才华横溢的获胜者西蒙·拉姆齐的客座博文。

西蒙是这么说的:

上个月，我很幸运地遇到了一个比赛，它不仅让我参与到 Sitepoint 这个伟大的社区中，还让我有机会发现更多关于我长期以来的爱好之一——在电影中捕捉时间和空间。

Sitepoint 大摄影挑战赛于上个月举行，持续了一周。每天参赛者都有一个任务——一个特定的主题，然后我们有 48 小时的时间拍摄并提交。有五天的时间，任何参加全部五个任务的人都将参加抽奖，以赢得新的 Sitepoint 图书[网络摄影](https://www.sitepoint.com/books/photography1)，新的 Sitepoint T 恤和撰写客座博客的机会……这就是我来这里的原因。

说到拍照，我是个业余爱好者。我喜欢不断地建立一个图书馆，它在我如何看待这个世界上扮演着重要的角色。它极大地影响了我的 web 项目如何组合在一起。这部分是因为我有一系列的照片可以激发我的灵感，但也因为拍摄 1000 张照片的反复试验影响了我对构图、平衡和色彩的整体感知。有一次，我拍了一张启发了整个网站的照片。我用它从标题图像一直到调色板的灵感。这绝对是一个极端的例子，但它仍然是我所做的大部分工作的真实情况，所以我总是出去抓拍照片——自从比赛以来更是如此。设定小的个人使命或任务来实践的想法是我从挑战中得到的一件事。

带着这个重燃的拍照欲望(还有一本新书要加载到我的手机上！)我有了出去拍照的绝佳机会。我当时正在进行一次公路旅行，沿着日本海岸行驶 1000 公里，进入山区，参加世界上最大的摇滚音乐会之一。我真的不知道前方是什么，但我知道我会练习一种我读过的新技术:景深。这是我一直想重新学习的东西，只是没有时间去做。这本书解释得如此清楚，现在我完全理解了。

一旦你有了一项新技术可以练习，确保你就这么做。去拍 1000 张照片，然后研究下一个技术。只有当你控制了你的相机，并且有能力设置闪光灯时，你才会开始拍出一些真正令人惊叹的照片。那些不知从哪里冒出来的图像才是你最喜欢的。不要误解我的意思，有时候你会有世界上所有的时间来拍照，你可能不得不等待，以获得完美的姿势，但在这之间只是不停地拍照。

最后，我建议你买一台相当不错的相机。我有一台尼康 D80。拍摄原始照片，并总是尝试拍摄照片，这样就不需要太多的后期处理。我依靠相机内置的直方图来实现这一点。

这里有一些旅途中的照片。我希望你喜欢它们。
