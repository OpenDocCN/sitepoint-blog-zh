# OSDC 2005 总结

> 原文：<https://www.sitepoint.com/osdc-2005-wrap-up/>

本周在墨尔本举行的第二届[开源开发者大会](http://www.osdc.com.au/)，可以简单地描述为一次大规模的极客盛会。

将它与我今年早些时候参加的 [Web Essentials](http://we05.com/) 大会相比较可能是不公平的，因为用组织者的话说，OSDC 是“由开发者创造，为开发者服务”的，而 Web Essentials 的基层基础更少。

但是我还是要做一个比较，因为我认为它确实提供了一个有用的衡量标准。据我所知，OSDC 在五个方面与 Web Essentials 不同:

1.  演讲者没有报酬。这意味着演讲在质量和相关性方面有很大差异。有许多精彩的演讲吸引了所有的听众，但也有不少讲座是业余的，讲的是一些只有少数人感兴趣的完全晦涩的话题。
2.  演示要短得多。除了每天开始和结束时的 60 分钟主题演讲外，所有讲座都只有半小时。更有经验的演讲者很好地利用了这有限的时间，但对一些演讲者来说，这导致他们的 73 张幻灯片快速通过，任何人都无法消化。
3.  主题和行业代表的多样性令人难以置信。广泛的“开源”是连接所有演示文稿的唯一共同主线，这导致了从博士研究论文到创业建议到大型政府项目快照再到企业推销的各种主题。
4.  *三条小溪，而不是两条。*不幸的是，这意味着偶尔会有三个话题让我感兴趣，而我只能参加一个。
5.  少点创新，多点无所顾忌的极客。 WE05 令人兴奋，因为它展示了所有新技术和新的思维方式。微格式、AJAX、以用户为中心的设计、使网页可访问的创新方法——所有这些话题都是由发明它们的人或世界上最精通应用它们的人提出的。我们听到了前沿思想家们的最新和最伟大的观点，他们负责推进房间里每个人选择的媒介——网络。另一方面，OSDC 的主持人对他们的专业领域充满热情，但是除了“成为一个极客”和“自由软件”之外，没有一个共同的媒介让每个人感到兴奋。

也就是说，这是值得的三天，有很多与网络专业人士相关的话题。以下是我的一些亮点。

*   Jon Oxer 的开场演讲讲述了将任意硬件连接到你的 PC 上，并使用 C、PHP 或任何你喜欢的语言与它们对话是多么容易。我还不知道他演讲中最酷的部分是他演示邮递员投递邮件时，他的信箱是如何给他发邮件(以及给他的手机发短信)的，还是他的猫被注射了一个可以打开猫的翻盖的微芯片。
*   Paul Fenwick 关于如何进行精彩演示的有趣技巧，包括使用 Perl 模块将预先准备好的字符回显到屏幕上，而不管您按了什么键，这样您在进行现场演示时就不会出错。
*   Scott Penrose 谈到了他称之为 Zaltana 的雄心勃勃的项目，在这个项目中，他将 HTML 挂钩添加到常见 web 应用程序的模板中，如 [SquirrelMail](http://www.squirrelmail.org/) 、 [MediaWiki](http://www.mediawiki.org/wiki/MediaWiki) 和其他自定义对象和方法，以便为他的客户提供一个可重用的整个框架的通用外观和感觉。
*   Gus Gollings 对语义网的讨论。这是一个与我过去听到的不同的(不太技术性的)概念。Gus 提供了丰富的历史背景，并提出了一些发人深省的问题，关于有一天可能会实现什么，任何让我思考未来的事情总是能抓住我的兴趣。
*   SitePoint 自己的[凯文·杨克](https://www.sitepoint.com/blog/)关于 [Mozilla XForms](http://www.mozilla.org/projects/xforms/) 的谈话(请关注凯文的博客了解更多)。
*   Luce Chandon 描述了福特制造厂如何使用开源软件和“代理”的概念，以确保汽车零部件及时交付到装配线。
*   “闪电谈话”会议(与外面糟糕的天气无关)，感兴趣的与会者就他们喜欢的任何东西进行 5 分钟的陈述。这些精彩演讲的准备工作是令人难以置信的，这是一个很好的、令人鼓舞的环境，让那些刚接触公开演讲的人去尝试一下。

总之，OSDC 是一个专业运营的会议，如果你是一个 web 专业人员，它代表着物有所值，但如果你是一个热情的 Perl/PHP 程序员，到处都戴着一顶红帽子，给他的第一个儿子取名为 Linus，并且 Slashdot ID 为三位数或更少，它就更值得了。

网络方面也很好，我遇到了一些很棒的人，他们中的许多人参与了活跃在墨尔本和澳大利亚的各种用户群，有些人从日本、台湾和其他地方远道而来参加会议。

然而，这是一个需要你非常仔细地挑选你参加的会议，因为演讲者的演讲技巧和内容的相关性都各不相同(有时与演讲的标题有很大差异)。

组织者把事情办得如此顺利，这要归功于他们。我明年会回来的！
