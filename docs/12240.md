# 任何人都不应错过的设计天才

> 原文:[https://www.sitepoint.com/design-talent-no-one-should-miss/](https://www.sitepoint.com/design-talent-no-one-should-miss/)

在许多开发中，美学和图形设计似乎让位于可用性和编程优势。从工程的角度来看，当我们谈论可用性时，这是合乎逻辑的，但是用户也希望美成为他们体验的一部分。我今天在网上搜索了一些非常漂亮的设计，它们不仅在视觉上很“可爱”，而且在功能上也很先进。Flash 项目当然是最漂亮的，所以为了美观，我们今天不会担心加载时间。

Saizen Media Studios 是世界上最受欢迎的设计公司之一。他们的电影和高度细致的工作是首屈一指的许多。高度细致的艺术是一个非凡的艺术家，工程师，程序员，当然还有动画师团体的国际名片。Saizen 为东芝设计的最新项目(见下图)在意大利和俄罗斯都设有办公室，反映了他们在交互和情境设计方面的独创性。

sevenedge 互动媒体

是一家由 Frederiek Pascal 和 Dries Schaballie 创立的设计机构。他们的网站反映了在创造美感、交互性以及设计的精确性和细节方面的高超技巧。一个非常令人印象深刻的作品集页面(见下图)展示了从育碧软件的[彩虹六号](http://rainbowsixgame.us.ubi.com/vegas/index.php)到 VT4 的[诱惑岛](http://www.vt4.be/temptation/video.php)的项目(也见下图)。

这家媒体公司创造了从病毒游戏到微型网站、标志和完整网站的一切。在我看来，sevenedge 作品的每个方面都具有相同的艺术平衡。我能找到的这家设计公司工作的唯一不足之处是他们的[公司博客](http://blog.sevenedge.be/)——在这里有点沉闷。

[Pixel crimes](http://www.pixelcriminals.com/container/)是我今天遇到的最有趣的设计公司之一。据我所知，他们的主页几乎出现在所有关于网络设计的出版物上，老实说，他们的可能是最漂亮的。不幸的是，对于那些“想变得漂亮”的网站所有者来说，PC 似乎没有太多的客户端或者正在进行的工作。

我能找到的他们工作的最好例子是一个引人注目的俄罗斯人体艺术的极简主义网站。该网站围绕 CMS 设计，以 Flash、XML gallery 为特色，与主题的低调优雅完美结合。Pixel Criminals 是网页设计是艺术的证明，然而很少有客户关上他们的门。我怀疑他们是在为希望匿名的客户工作，因为这样的人才很少得不到回报。

2Advanced Studios 是列出的网站中最具视觉和技术震撼力的。2Advanced 由埃里克·乔丹、托尼·诺瓦克和约翰·卡罗尔于 1999 年创立。该工作室的客户名单读起来像是数字世界的“名人录”，包括 Eidos，福特汽车公司，卢卡斯艺术，任天堂，Adobe 和无数其他公司。2 先进的[量子安慰](http://www.007thevideogame.com/?lang=en#home/)游戏参与(如下)清楚地表明，该公司在设计艺术的持续领导地位。

##### 模范艺术

前面的网页设计工作的例子可能不是每个人的选择，但它们表明了世界上一些最美丽的网站的创造者。他们向我们展示了先进的矢量图形、flash 和其他编程和工程魔法确实与美学上引人注目的图像和动画携手并进。由此产生的作品从定义上来说就是艺术，在形式和功能上都很美。我知道还有这么多，邀请读者提交自己喜欢的例子。Web 开发已经超越了千篇一律的网页和原始的应用程序。世界上最伟大的开发者在任何意义上都是艺术家；最优秀的人将网络视为运动中的画布——活生生的作品。
