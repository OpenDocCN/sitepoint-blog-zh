# AJAX 的状态

> 原文：<https://www.sitepoint.com/state-of-ajax/>

在《AJAX 的现状:进展、挑战和对 SOA 的影响》一书中，迪翁·欣奇克利夫给出了他对围绕 Ajax 的所有炒作和轰动的看法。我还在《科技时报》第 120 期上发表了关于这个话题的社论。

尽管有些人声称 AJAX 将取代桌面应用程序开发，但我相信某些类型的应用程序——至少在可预见的未来——会不可避免地与桌面结合在一起。也就是说，多亏了 AJAX 和它的同类，网络在用户体验类型上确实有了显著的增长，它可以为*适合的应用类型提供用户体验。*

在他的文章中，欣奇克利夫对客户端 AJAX 风格开发的兴起将如何影响服务器端开发前沿的一些趋势做了一些观察。他特别指出，许多为 Web 服务开发的新标准并没有被设计成在 AJAX 驱动的世界中有用。虽然 AJAX 在与单个服务器的快速、轻量级、小型通信中发展得很好，但是当前 Web 服务的大部分工作是以一种安全和复杂的编排方式在多个服务提供商之间交换和处理大型文档。
