# 自下而上的网页设计工具

> 原文：<https://www.sitepoint.com/web-design-tools-down-under/>

有几种方法可以处理网站所需的图形。第一种，也是最简单的，就是简单地从其他网站窃取——呃，“借用”——它们。诚实的设计师和网络爱好者都或多或少地使用这种方法。在某些情况下，这不是问题。要么网站所有者不介意，要么他们使用了免费网站的图片。在其他情况下，这和从街角商店偷一条面包没什么区别。当有疑问时，你可能应该谨慎行事，避免使用任何不公开使用的图形(顺便说一下，我目前正在为 SitePoint 准备一个关于这个主题的专栏)。但是不管你怎么看，版权问题基本上意味着你的选择仅限于其他人已经使用过的图片。

第二种方法是让专业或业余图形设计师为你制作图形，或者从专门为此目的提供图像的图形网站下载预先制作的图像。第三是自己做。这是许多想拥有自己网站的人失败的地方。要么他们被复杂的(理解为:“难以使用”)商业程序所迷惑，要么他们被更便宜、效率更低的软件包的局限性所挫败。

市场上的两个共享软件程序，My Web Toolbox 3.3 和 Trellian Button Factory 2.02，满足了部分需求。它们提供了一种为网站创建按钮和类似图形的廉价方式。没有一个像 Photoshops、Paint Shop Pros 和 Flashes of the world 那样全面、复杂或昂贵，也没有一个试图做到这一点。相反，他们努力做一件特定的事情，并且做好。他们成功的程度，至少在我看来，是在这里考虑的。

*注意:这些评论以稍有不同的早期形式出现在 www.toejumper.net 的我的网站第 13 期时事通讯《Windows 95/98/ME 的故障排除和资源指南》中。新闻稿已转发给两位软件创作者，征求他们的意见，这些意见已被纳入本文。*

##### 我的网络工具箱 3.3 $30

[www.australianwineandbeer.com/mwt/toolbox.htm](http://www.australianwineandbeer.com/mwt/toolbox.htm)

我的网络工具箱是一个澳大利亚软件创作者 Ian Timmins 的作品。伊恩通过一个葡萄酒和啤酒网站营销 MWT，这可能是他没有得到尽可能多的曝光率的一个原因。这很不幸，因为 MWT 是这个项目的一颗小宝石。他目前正在考虑加大推广该项目的力度，并在寻找合适的商业伙伴(如果你愿意的话，这是给聪明人的忠告)。

引用创作者的话，“我的按钮选项背后的概念是提供有限范围的样式，但要确保这些样式可以快速简单地创建。”事实上，在 MWT 可用的样式范围是有限的，但按钮和横幅的创建再简单不过了。我审查了 3.2 版，但 3.3 版现在已经推出，并支持动画横幅的框架。正如网站上提到的，MWT 并没有创作真正的动画图形；你需要一个独立的程序来实现这个功能。只有 GIF 文件可以被动画化，而 MWT 不支持 GIF，原因详述如下。

***安装***

对于这么小的下载量来说，MWT 的功能相当不错，而且也很容易使用。该程序本身在一个小工具栏中打开，可以停靠在屏幕上的任何地方，直到需要时才打开。也可以缩小成系统托盘中的图标。MWT 安装简单，几乎不会影响你的系统:该程序不会侵入注册表，系统配置，或其他任何东西。要卸载它，只需从硬盘上删除该程序。我很欣赏这个功能。我对那些在你的系统中散布注册表项、侵入你的开始菜单、重写你的配置文件等等的程序感到恼火。没必要。正如 Ian 所说，“我对大多数共享软件最大的厌恶可能是它在注册表中存放垃圾，而这些垃圾通常不会被清除。我现在根本不会运行需要安装的共享软件。”我喜欢这个人的思考方式。

**有什么用**

 **MWT 为网站创建、编辑和修改图像(按钮和横幅)。不幸的是，该计划只创建 BMP 和 JPG 文件，而不是 gif。这是一个损失，但原因与 Unisys 声称对 GIF 格式的所有权有关，并可能影响创建 GIF 文件而不向 Unisys 支付赎金的程序的法律性。在网站上使用 BMP 文件的缺点是双重的:它们往往比 JPG 和 GIF 文件大得多，Netscape 不愿意支持它们。

MWT 工具栏由二十个按钮组成，将来可能会增加更多按钮。每个按钮都有自己的功能，大多数都是不言自明的。颜色选择非常广泛，可以从 24 位、HTML、特定于浏览器和全光谱调色板中选择。图标按钮允许您查看和创建图标图像。从 BMP 到 JPG 的转换是一键操作，反之亦然，因为:

*   字体选择和预览，
*   特殊字符显示，
*   捕捉和复制整个或部分屏幕，
*   将图像分割成更小的块，
*   值转换(二进制、八进制、十进制和十六进制)，以及
*   查看 ASCII 和/或十六进制格式的文件。

修改以前创建的图像并不像“从头”创建自己的按钮那么简单(实际上是从模板创建按钮)，但是做任何一件事都不难。创建新按钮的唯一缺点是可用的模板数量很少——基本上是十种基本形状，可以根据需要进行或多或少的更改——以及按钮大小的限制。新增的吸引力包括创建横幅、合成图像和彩色 HTML 字体(包括光谱和随机彩色)的能力。模板都是矩形或正方形的，尽管实际的图形设计通常是圆形、椭圆形等等。背景颜色必须仔细选择，以紧密匹配他们将继续的网页，因为没有透明选项可用(没有 gif，没有透明度)。您确实可以为按钮导入自己的背景图像，这可能是将按钮集成到您的站点的更好方式(Ian 说他认为 BMP 文件可以通过屏蔽左下角的像素而变得透明)。

***摘要***

总的来说，MWT 是一个小而优秀的程序，它专注于一个单一的任务——按钮创建——并且做得非常好。这并不完全正确:正如 Ian 所指出的，“更多的用户实际上使用这个程序来选择颜色，而不是创建按钮。事实上，按钮的创建是在程序开发的后期才出现的(最初只是一个颜色选择程序)。但在共享软件版本中，颜色选择选项不受限制，而按钮创建则受限制。”

比你预期的更多的选项是可用的，屏幕捕捉、HTML 着色和横幅创建的额外功能只会增加这个程序的价值。按钮模板的数量有限是一个问题，但不足以保证我指导你远离这个程序。我的建议是现在就抓住它并注册，在一些商业实体把伊恩的小发明拼凑起来，加上一些华而不实的东西，然后抬高价格之前。

##### 特雷里安纽扣厂 2.02 美元 45 元

[www.trellian.com/tbf/](http://www.trellian.com/tbf/)

Trellian 是一家著名的软件开发公司，总部也在澳大利亚。除了按钮工厂，他们还销售其他网络工具，包括 FTP 客户端、搜索引擎提交工具和电子商务平台。TBF 可作为一个免费的演示版本，以及作为一个功能齐全的注册版本。演示版在 30 天后超时，不允许您保存任何图像；从这个意义上说，它更像是一个试用软件，而不是一个完整的共享软件程序。顾名思义，这个程序除了为网站创建按钮什么也不做。它给了你大量的样式和选项供你选择，尽管和我的 Web 工具箱一样，你受限于所提供的模板(这个程序提供了大量的模板)。特雷里安也收到了这篇评论的时事通讯版本，他们的反应很简单，“听起来不错！”

***安装***

TBF 使用标准安装向导快速安装，并且可以通过添加/删除小程序卸载。打开 TBF 会给你一个全屏的每日提示(可以禁用)和两个工具栏，工具栏上有很多我们已经习惯了的标准按钮——新建、打开、剪切、复制、粘贴等。TBF 不像我的网络工具箱那样一目了然，可能会吓到网站设计新手。在我看来，它有一些大男孩的外观和感觉，特别是 Paint Shop Pro——那些熟悉 PSP、Photoshop 或其他类似程序的人会感觉更自在。几乎没有离线帮助:在线手册的链接会把你带到一个提供大量帮助的特雷利亚网站。我强烈建议新用户先浏览一下这些材料，因为这里有很多有用的信息和提示，尽管不是所有的选项都有令人满意的解释。顶部的工具栏有些可停靠，尽管它们不能被拖放到任何地方。此外，右键单击或通过顶部菜单可以获得属性菜单。

**有什么用**

 **你的大部分工作将在主要的空白区域完成，称为“工作台”程序是模板驱动的；你所有的工作将基于一个或另一个模板。你有很多选项来改变和修改模板——大小、形状、边框、背景颜色等等。—从工具栏和属性菜单(可通过右键单击访问)中。模板本身是多种多样的，非常漂亮，有一些非常复杂的图形设计可用(如果我没数错，这个版本有 125 种选择)。除了更好的选择之外，还提供了一些更简单、更基本的模板，包括那些适合标准 Windows、Mac、Linux 和 Unix 图形模式的模板。

创建按钮最简单的方法是打开图像库(通过窗口，显示库，在一个狭窄的右侧窗格中显示整个库的内容),并将选定的按钮样式拖到工作台上。一旦在工作台上，可以根据需要修改按钮模板并调整其大小。全色调色板使颜色选择非常广泛，并且易于使用。该程序缺少多次撤销，因此在创建过程中只能后退一步。但是，您可以使用“删除”按钮来删除不想要的创建，并使用新的模板重新开始。

所有的图像都是矩形或正方形，尽管许多设计本身是椭圆形、圆形、箭头等等。您必须指定一个与您的网页的背景颜色或图像非常匹配的背景颜色，以便按钮融入其中，或者导入您在网页中使用的相同背景图像。请注意，背景颜色的选择反映在整个工作台上，所以不要让这个问题困扰你。

像我的网络工具箱一样，TBF 也不支持 gif。但是，它支持 BMP、JPG、PNG 和 PPM(可移植像素地图文件)格式，以及专有的 TBF 格式(主要用于保存正在进行的工作)。由于该程序不支持 gif，背景通常不能是透明的(PNG 图像可以是可变透明的，但这不是简单的点击按钮的问题；如需了解更多信息，请访问位于[www.libpng.org/pub/png/](http://www.libpng.org/pub/png)的巴新主页。我想特里安不支持 GIF 格式的理由和 MWT 的创造者相似。在属性中有一个透明度选项，但是这并不能使颜色在公认的意义上透明；相反，它清空了整个模板，只留下文本和背景色。

我花了一段时间才弄明白如何修改模板中的文本。在线手册中提供的说明是错误的，错误的，错误的——根据说明和提供的图像，工具栏中有一个文本框，您可以在其中输入新文本。没有。然而，起作用的是突出显示图像，然后双击它。您的图像“消失”,出现一个空白的白色栏，您可以在其中键入新文本。在文本字段/图像外单击会将图像带回来，包括您的新文本。这是很容易改变和调整你的文字，你有大量的字体可供选择。据我所知，新的文本选项给了你一个按钮，除了基本的背景颜色之外，里面没有任何图形——对于知道如何使用这种选项的网页设计师来说，这是一个有用的选择(类似于上面详细介绍的透明度选项)。TBF 本身没有提供横幅创建，但是有了 Resize 选项，我看不出为什么你不能做一个足够大的按钮来作为横幅图像。

TBF 给你一个简单的方法来创建可点击的图像地图，正如手册中所概述的。然而，据我所知，你只能给一个特定的图片分配一个 URL——不要把一个图片分割成块，然后给每个块分配不同的 URL。

保存一个创作有点棘手；选择“另存为”仅允许您将按钮保存为. TBF 文件。如果突出显示该按钮并选择“文件”、“导出”,则可以从其他四种格式中进行选择。同样，在线手册的学习在这里是有帮助的。一些更好的选项，如选择网格大小，创建新的框架等。，没有很好的解释；你要么自己去理解它们，要么根据你对其他图形程序的了解来推断它们的意思。

***摘要***

总的来说，Trellian 的按钮工厂是一个优秀的程序，它能以最小的努力快速创建复杂的 Web 按钮。学习曲线比我的网络工具箱更陡峭，但是这个程序给了你一些 MWT 没有的功能和选项。我对该程序有时反直觉的操作方式并不十分满意，而且我对在线手册缺少关于一些特性的文档感到失望。然而，这个程序很好地完成了它设定的任务。大量的按钮模板和复杂的工具使得按钮工厂对于严肃的网页设计师来说是一个值得购买的东西。**
