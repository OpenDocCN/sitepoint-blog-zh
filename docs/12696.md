# Zend 与 Adobe 合作，将 PHP 和 Flex 结合起来

> 原文：<https://www.sitepoint.com/zend-teams-with-adobe-to-marry-php-and-flex/>

Adobe 和 Zend 今天宣布合作，将 Zend 的 PHP 框架与 Adobe 的 Flex 更紧密地集成在一起。目标是通过在服务器端利用 Zend 的 PHP 技术，在客户端利用 Adobe Flex，让开发人员更容易地创建企业就绪的富互联网应用程序。

两家公司之间的第一个集成点将是为 Zend 框架提供一个动作消息格式组件。AMF 是一种消息协议，允许基于 Flash 和 Flex 的客户端应用程序与服务器交换数据和富媒体。Adobe 去年 12 月发布了 AMF 二进制数据协议规范，并在当时发誓要让该协议在所有服务器平台上可用。

Adobe 最初在 7 月份提出了 Zend 框架的 AMF 组件，但现在正式宣布与 Zend 合作以实现这一目标。

此外，两家公司计划更紧密地集成各自的开发环境，Zend Studio 和 Flex Builder，这两个开发环境都基于开源的 Eclipse 平台。
