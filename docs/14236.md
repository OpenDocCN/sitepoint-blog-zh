# 如何雇佣网页设计师

> 原文:[https://www.sitepoint.com/how-to-hire-a-web-designer/](https://www.sitepoint.com/how-to-hire-a-web-designer/)

今天，任何想要提供信息、销售商品、分享信息或推广业务的人都知道，网站将帮助他们实现这些目标。

慈善组织可能希望向潜在成员和志愿者宣传自己，并通过时事通讯和与其工作相关的文章提供信息，以便任何感兴趣的人都可以了解该组织。攀岩中心可能希望展示一张地图，上面有如何到达该中心的方向、攀岩技术的提示、在哪里可以找到好的设备的提示、攀岩者动作照片的健身房照片库……与其他营销策略不同，网站覆盖全球，可以全天候在线访问。

如果你想要一个网站，但你不是设计师或开发人员，你该如何着手去做呢？你没有时间去学习如何成为一名网页设计大师，你也不相信你学计算机科学的堂兄有足够的经验为你建立一个专业网站。那么，你可以雇佣谁来建立你的网站呢？有成千上万的网页设计师和开发者，从个体自由职业者到大型网页设计机构，你如何确保你选择正确的帮助？

##### 你想要什么?

为了寻求帮助，你首先需要弄清楚你想要什么。问自己以下问题:

*   你在网站上想要什么样的信息？你认为你的网站会有多大？
*   你的用户是谁？你知道他们用的是哪个操作系统和浏览器吗？
*   你的网站需要定期更新吗？你想自己做些改变吗？
*   你要卖东西吗？
*   您是否需要一个数据库来存储和检索信息？
*   你想依靠搜索引擎给你的网站带来更多的流量吗？
*   你需要什么时候完成这项工作？
*   你的预算是多少？

##### 搜索开始了

那些能够发现一个好的网页设计师的人通常是一个好的网页设计师，或者至少他们自己已经做了相当多的网页设计。但对于那些不是设计师的人来说，选择专业人士似乎是一项艰巨的任务。

推荐是一个安全的赌注；虽然你知道你可能得不到最好的网页设计师，但你通常可以相信你也不会得到最差的。然而，如果你使用推荐，也进行你自己的搜索，你会有更好的机会找到一个适合你的设计师。一旦你从你选择的资源中收集了一份网页设计师和/或开发者的名单，你需要做一些认真的功课。

如果你名单上的一些设计师来自外地呢？如果你真的喜欢他们，就不要排除他们。如果你不介意通过电子邮件工作和打电话，你可能会对你的选择很满意。面对面交流总是有好处的，现场访问也很重要，尤其是在出现问题的时候。

当审查你的潜在网络专家名单时，有许多事情需要考虑。第一件显而易见的事情就是查看他们的网站。浏览网页，尽可能多地找到关于它们的信息。问问你自己:

*   找到信息并回到起点容易吗？
*   你喜欢导航系统吗？
*   页面是否可以访问(没有断开的链接)？
*   页面和整体设计是否一致？
*   是否有联系页面和网站地图，是否容易找到？
*   网站上是否有足够的相关信息(如公司的详细信息，包括位置、他们做什么、人员、政策等)。)?
*   事情是否排列正确？
*   这篇课文易读吗？
*   页面加载速度快吗？
*   页面是否很短，以至于不需要水平滚动，很少或没有垂直滚动？
*   链接在同一个页面上打开吗？
*   有没有可以查看的作品集？
*   网站讨论设计师的技术背景吗？
*   网站使用了正确的颜色吗？
*   页面标题是否恰当且信息丰富？

希望上述所有问题的答案都是肯定的。基本上，如果你不喜欢一个网页设计师的网站，你可能不会希望他们设计你的网站。检查他们的作品集，看看这种风格是否适合你。如果你看到你真正喜欢的网站，确保创建这些网站的员工仍然在职，并且可以在你的网站上工作。设计师使用了哪些技术？这项技术对你和你的观众有用吗？团队是否遵循 Web 标准，或者他们仍然像 90 年代一样编写网站代码？理想情况下，您希望您的站点独立于用户的操作系统和浏览器工作。

团队是否为你所在行业的其他企业创建过网站？如果是，他们能够正确反映业务吗？如果是的话，那么这个团队已经知道你的行业的需求，并且会比其他没有制作过这些特殊网站的网站设计通才更适合你。如果网站提供推荐，阅读它们，看看过去的客户对他们接受的工作有什么评价。除了技术技能之外，设计师还应该继续接受教育，以跟上最新的技术和标准。

小心那些自称是网页设计师和开发者，但主要从事平面设计和印刷媒体工作的公司和个人。能够使用像 Dreamweaver 这样的网页制作软件并不能成为一名网页设计师。你的网页设计师至少应该能够帮助你进行网页设计和开发、虚拟主机、图形制作、数据库制作、网页内容、维护以及网络营销和推广。

##### 自由职业者与大型网站设计公司

评估完选定的网站后，你可能需要在雇佣自由职业者和使用大的网站设计公司之间做出选择。一个大的网页设计公司可能看起来有很高的可信度，因为它有大量的投资组合，许多推荐，以及大量的设计和开发领域的专家。这些专家必须通力合作，为客户提供一致且成功的服务。这种组织的规模可以让客户感到安全和有信心加入他们的服务。

自由职业者是能够承担所有必要的设计和开发责任的个人。这些类型的提供者通常与其他人密切合作来完成工作，这种较少人(或者在某些情况下，只有一个人)之间的密切合作意味着一致性很容易实现。单独工作或在一个小团队中工作也能产生更多的动力和奉献精神来完成客户满意的项目。在这种类型的安排中，你所看到的就是你所得到的:你在网上遇到的专业自由职业者将是你项目的网络专家，并且可以承担个人责任。相比之下，在与大公司合作时，一旦交易成功，一个完全陌生的人可能会被指派为你的客户经理。

自由职业者也可能代表着更好的性价比。对于自由职业者来说，在项目开始之前，很少有任何隐藏的费用，也没有许多复杂的合同细节需要克服。如果需要的话，自由职业者也更容易到达现场。

根据网站的规模和复杂程度，大的代理机构可能是正确的选择。一个大公司可能比一个个体自由职业者更快地交付更大的项目。个体自由职业者可能经常需要转包合同或学习某些技能或技术来完成工作。这可能意味着额外的时间和/或成本，并且，取决于所涉及的自由职业者，也可能导致不那么专业的产品。出于这个原因，如果你的项目需要使用一种特殊的语言或技术，那么找一个已经精通该领域的设计师是一个好主意。

##### 定价和担保

为了进一步细化你的潜在设计师列表，你需要记录他们的服务价格。设计师对服务的定价可能会有很大差异。比较具有相似教育水平、经验和才能的设计师之间的比率。像大多数购买一样，网页设计会让你物有所值。如果你的项目相当小且简单，自由职业者可能比大机构收费低。“小”，我的意思是一个网站与一些表格和一个小数据库。

一旦你缩小了名单，与相关的公司或个人取得联系，解释你的项目，并要求一个准确的报价。确保你的设计师能为你详细概述所有的成本和工作。如果你有问题，不要害怕问，并且记住:如果你觉得报价不合理，协商一个比报价更低的价格并不是不合理的。

如果可能的话，也看看供应商的网络合同。确保客户在本合同下受到保护，并确保检查版权和支付政策。也记下供应商的响应时间。你想和一个随时可以联系，容易联系，并且会及时回复你的人一起工作。

寻找并询问工作保证。诸如“如果您不是 100%满意，我们将退还您的钱，”或“我们的价格是有竞争力的，但如果您找到更低的类似服务，我们将很高兴与之匹配，”这样的政策将让您清楚地了解设计师的信心，他们可以满足您的需求。保证很重要:没有什么比花大价钱买一个你不好意思给客户或顾客看的网站更糟糕的了。

担保向潜在客户表明，公司关心让他们开心，并尽最大努力确保你的项目成功。

##### 最后一步:联系并核实证明人

当你把选择范围缩小到几个设计师时，就该联系他们并核实他们的推荐了。首先，打电话给供应商并询问问题。他们在电话里有礼貌吗？他们是好的倾听者吗？他们有帮助吗？如果他们很难沟通，你不喜欢他们对待你的方式，那就很难和他们一起工作。

通过阅读网站上的任何推荐，甚至与过去的客户交谈，检查每个供应商的参考资料。访问他们的投资组合页面，找到几个客户的联系信息，然后给他们打电话。如果没有证明，当你打电话给供应商时，要求提供参考。你正在招聘，所以你有权查看他们的工作证明。

最后，亲自和设计师见面，讨论你的项目想法是个好主意。即使在这一点上，你也没有义务加入他们的服务，除非你完全相信他们是这项工作的合适人选。

##### 这项工作是值得的

遵循这些步骤，你会增加成功找到并雇佣一个满足你和你的项目需求的网页设计师或开发者的机会。这个过程可能看起来工作量很大，但是当你花了几千美元，花了很多年，在你的网上，做你的功课是值得的！
