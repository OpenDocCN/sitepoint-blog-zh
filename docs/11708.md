# 设计推特:如何将推特整合到你的网站中

> 原文:[https://www . site point . com/integrate-Twitter-into-web-design/](https://www.sitepoint.com/integrate-twitter-into-web-design/)

![Twitter](../Images/fc3d23fd6a8d85ac641c84352ffa8dcf.png)

为了抓住你的注意力，让你在 Twitter 上关注他们，一些网站所有者选择超越简单的“关注我”，还显示他们的最新推文。我不知道为了吸引粉丝，一种方法是否优于另一种方法，但我认为看看如何显示推文以及一些设计师如何让他们的推文脱颖而出，或者在某些情况下融入整体设计可能会有所帮助。

那么，你实际上如何让你的推文显示出来呢？有很多选择。

**推特小工具** 

如果你正在写博客或使用 MySpace 或脸书，你可以拿一个[小工具](http://twitter.com/widgets)在适当的社交媒体领域工作。它们有 Flash 和 HTML 两种形式；你所需要做的就是复制代码并粘贴到你的网站上。

![twitter-widgets](../Images/3fb4b6bc43f286427eafce7e5fa3be48.png "twitter-widgets") **T6】**

**推特美食**

Twitter 还为所有用户提供了一个[个人资料窗口小部件](http://twitter.com/goodies/widget_profile)，可以通过抓取提供的代码添加到大多数网站上。您可以在设置中配置徽章的大小、颜色和更新次数。

![twitter-goodies](../Images/2d9ace16d28228688a70b45dcae4a2de.png "twitter-goodies")

**jQuery 插件**

有很多这样的插件，它们都提供了 tweet 显示功能以及其他一些附加功能。

Juitter 是一个小工具，可以显示用户的推文，并可以选择显示头像。

[TwitStream](http://kjc-designs.com/twitStream/) 使用 [Twitter 搜索 API](http://apiwiki.twitter.com/Search+API+Documentation) 显示包含特定关键字的 twitter feed 列表，同时显示头像和网址。

如果你想把你的推文添加到你的 WordPress 博客，WordPress 的 Twitter 会显示你最近的推文和网址。
