表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 16.3%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

每次我认为 Chrome 已经达到饱和点，几百万用户就证明我错了。虽然它不是每个月都实现 1%的增长，但我不能对 0.95%的增长吹毛求疵。令人印象深刻。

然而，不同寻常的是，大部分市场保持稳定。IE9、Firefox、Safari 和 Opera 在过去三个月几乎没有变化(诚然，Opera 本月有所下降，但我怀疑这是被其较小的用户群夸大的统计信号)。

浏览器的使用模式很复杂，但 Chrome 目前的增长主要是用户从 Internet Explorer 6、7 和 8 的迁移。鉴于 XP 用户占操作系统市场的 26%,并且没有其他选择，这有什么奇怪的吗？Vista 只有 7%,这些用户都停留在 IE9 上。Windows 7 开始稳定在 53%，但 IE10 仍然是测试版浏览器。 [Windows 8](https://www.sitepoint.com/microsoft-windows-8/) 尚未对操作系统的数字产生影响。

微软放弃传统操作系统用户的决定对 IE 的用户群产生了不利影响。该政策主要是由业务驱动的，但很少有人会升级他们的操作系统，因为他们的浏览器已经过时。也许十年前这种方法会奏效，但是今天，IE 用户有很多选择。

知识也提高了，更多的人理解了什么是浏览器。具有讽刺意味的是，微软的 IE9 广告负有部分责任。此外，谷歌继续推广 Chrome——即用户只要进入搜索引擎，就会看到一个“安装”链接。

最后，当前的经济环境也有影响。假设您经营着一家拥有数百名 Windows XP/Vista 用户的企业，他们正在为 IE6/7/8 速度或兼容性而苦苦挣扎。你会花数千美元升级电脑和培训员工吗？或者你会允许员工采用他们选择的另一种浏览器吗？

IE10 是一个很棒的浏览器，但是没有人使用它。在 [Windows 8 发布](https://www.sitepoint.com/microsoft-windows-8/)一个月后，它只有 0.29%的微小市场份额，而且这些用户大多是从 IE9 迁移过来的。随着时间的推移，人们将升级 Windows，并可以选择切换回 IE。但是，有多少人会在意自己使用 Chrome、Firefox 或 Opera 已经有几年了？

除非 IE 变得明显优于其竞争对手，否则微软的政策将适得其反。

## 移动浏览器的使用

令人惊讶的是，在 11 月份，手机使用量增长了 1%,占所有网络活动的 13.08%。鉴于北半球寒冷，每个人都喜欢呆在室内，这是不寻常的。

主要的移动浏览应用程序:

1.  安卓系统——27.15%(增长 1.31%)
2.  iPhone——20.98%(增长 0.11%)
3.  opera Mini/Mobile–17.99%(下降 0.93%)
4.  诺基亚浏览器——9.34%(下降 0.86%)
5.  UC 浏览器——9.08%(增长 1.43%)

对于 Android 和 UC 浏览器来说，这是一个非常棒的月份。我原以为苹果 iPhone 5 发布后，Safari 的移动版本可能会增加，但我猜许多用户是从以前版本的设备升级而来的。诺基亚功能手机受到了相当大的冲击，尽管它们在非洲、亚洲和南美洲仍然很受欢迎。

Chrome 和 Firefox Mobile 分别仅占 0.59%和 0.57%。似乎大多数人对他们的默认浏览器很满意。

下个月即将到来:*2012 年浏览器回顾！*
