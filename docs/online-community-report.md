# 采访——在线社区报告的吉姆·卡舍尔

> 原文：<https://www.sitepoint.com/online-community-report/>

自从网络出现以来，社区就一直存在。许多人来了又走，有些人在几次经济衰退和不同程度的兴趣和不兴趣中幸存下来。

因为社区是关于人的，我们可以肯定，许多网站将决定将社区整合到他们的整体产品中，许多网站管理员将寻求那些有经验的人的方法和智慧来创建他们的新兴社区。

吉姆·卡舍尔是在线社区报道和 T2 论坛的主席。自从 Web 建立以来，Jim 就一直参与在线社区和交流，他对 Web 社区提供了非常有趣的见解…

SitePoint:你是自己的网络战略和开发公司的董事长，在此之前，你帮助建立了一个拨款组织。最初是什么激发了你创办一份关于社区的时事通讯？

很明显，纵观历史，点对点通信(电报、电话、传真……)和广播通信(书籍、广播、电视)都取得了巨大的进步。但是在群体交流领域还没有相应的进步。

今天，群体交流的主要形式仍然是面对面的商务会议——与 2000 年前没有太大不同。然而，在线社区可以改变这一切。我们认为这是一个强有力的趋势，也是我们想要追踪的趋势。

**SP:online community report 有什么历史？这一切是从哪里开始的？你花了多长时间才把报告做到今天这个地步？**

我们在 1997 年推出了时事通讯，当时社区开始在网络上蓬勃发展:门户网站增加了留言板，聊天蓬勃发展，群组通信技术以许多新的形式出现。

我们希望跟踪在线群体协作的新闻和趋势，也许在某种程度上有助于定义该部门的身份。在这一阶段，我们每月发布两次，面向大约 15，000 名“选择加入”的订户。

你认为有什么特别的技术或举措能真正推动“群体”交流，使我们能够彻底改变过去 2000 年的习惯交流模式吗？

我不认为有任何一种技术会彻底改变通信，但我确实认为，也许有 50 种相关技术会改进到可能出现新模式的程度。

如果你从一个有能力的平台开始，如雅虎集团，它提供了一套有用的服务，并想象这些将在未来几年通过更好的带宽，视频，语音识别，永远在线访问和更智能的配置文件得到增强，你可以开始想象非常强大的可能性。

你为什么认为社区已经成为在线体验中不可或缺的一部分？

网络有助于解决现实世界社区面临的许多问题:地理、便利、社会问题等。由于这个原因，社区正在将自己嵌入到网络的每个角落。

那么，你如何定义“健康社区”？你在网上见过真正健康的社区吗？

我认为一个健康的社区就是一个其成员从参与社区中受益的社区(而“受益”可以指很多东西——信息、社会关系、娱乐……)。使用这个定义，我可以说网络上有成千上万的健康社区。

你对现有在线社区的增长和质量满意吗？或者你认为互联网作为一个整体在网络社区方面有更多的工作要做吗？

我确信，在整个十年中，这个领域将会有重大的发展。最显著的趋势是，高调的“人人欢迎”社区将会缩小(主要是出于经济原因)，但那些迎合明确定义的受众和需求的社区将会激增。

建立和维护一个社区肯定是很昂贵的。个人和公司如何证明运营一个活跃社区的相关费用是合理的？有没有关于在线社区如何有利于公司底线的数据？

这是一个重要的问题，部分原因是运营社区真的很昂贵。我知道 Participate.com 和 PeopleLink 都有关于社区投资回报率的详细研究。不过，我会第一个承认，很难得出准确的数字，因为社区的一些好处(忠诚度、品牌等。)，都很难量化。

对于一般的网站管理员来说，你认为吸引新成员加入他们的社区的最好方法是什么？如果他们在自己的网站上安装了一个公告板，这是否意味着他们有一个“社区”或者有比这更多的东西？建立“社区”的其他主要因素是什么？

正如许多团体从艰难的经历中学到的那样，一个简单的公告板不能成为一个社区！我认为建立社区最重要的策略是理解公告牌通常放在最后，排在提供高质量信息、增加流量、有针对性的电子邮件策略、简单的参与技术(投票、留言簿等)之后。)，以及简短的线上活动。

你参加了哪些在线论坛(如果有的话)？哪些是你的最爱——是什么让它们如此美味？

我跟踪了很多论坛。我是 slashdot 的忠实粉丝，因为社区的参与和技术的出色使用。我认为沙龙做得很好。然而，对我来说，最有趣的社区是那些与我的兴趣密切相关的小而不知名的社区，无论是专业的还是个人的。

你对社区/公告板软件包有什么偏好吗？

我是 [ezboard](http://www.ezboard.com/) 的粉丝。他们有大约 5 亿的月浏览量，所以他们一定做对了什么。[开放话题](http://www.infopop.com/products/opentopic/index.html)乡亲们做了很多令人印象深刻的工作。我也很尊重[网络穿越](http://www.webcrossing.com/)的大部分工作。还有其他值得注意的倡议。

网站所有者或公司如何让现有的会员重返并活跃在他们的社区中？获取和保留之间的平衡应该是什么——哪个对在线社区的健康和生存更重要？

在一个全面的回答中会列出许多不同的步骤，但是我认为一个关键问题是要理解你的社区实际上是围绕着相当少的关键贡献者的。你需要宠爱和奖励那些关键的贡献者。我也认为电子邮件与社区的整合是至关重要的。

你对在线社区的未来有什么看法？什么会保持不变？又会从哪些方面进化？

我之前提到了社区特异性的重要趋势。其他重要的趋势包括越来越依赖电子邮件和通过各种通信技术“始终在线”访问社区。我认为在未来，我们都将通过这些新技术以某种方式与群体交流，而无需过多考虑它。

你有很多创业经验，无论是线上还是线下。你会给有抱负的网络企业家什么建议？

这有点自相矛盾，但我的主要建议是“警惕建议”。很多创业依赖于运用自己的判断。当然，很多人都有创办公司的经验，但他们的经验可能与你相关，也可能与你无关。做你认为有意义的事情——不要简单地接受流行的智慧。

你对未来有什么计划？你有什么新项目在进行中吗？

我花了大部分时间和我的同事一起建立一个智能网络战略和开发公司，Forum One。我们现在有 25 个人，一直都很有盈利能力，希望继续保持下去。在社区方面，最有趣的工作是与一些大公司一起努力思考社区在其内部和外部项目中的适当角色。我们还在建设一些创新的社区网站，将于今年晚些时候推出。
