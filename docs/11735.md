# 使用 Screenr 创建截屏

> 原文:[https://www.sitepoint.com/create-a-screencast-with-screenr/](https://www.sitepoint.com/create-a-screencast-with-screenr/)

Screenr 是一个新的免费网络应用程序，可以让你创建截屏。让我先说，Screenr 的制作者既没有要求也没有付钱给我做评论。我告诉你这些是因为这是一篇热情洋溢的评论。我绝对喜欢它。

首先，它非常容易使用。我在几分钟内就完成了我的第一个电影剧本。其次，画质真的很优秀。第三，截屏可以在 Twitter 上播放(如果你愿意，你可以完全绕过 Twitter)，它们可以立即上传到 YouTube，你可以下载为 mp4 文件，在移动视频设备和智能手机上播放。

我有没有提到它是免费的？

对于网页设计师和博客作者来说，这是一个制作视频的超级工具，你可以把它添加到你的网站或博客中。制作完截屏后，Screenr 会给你嵌入视频的代码。另一个很好的特性是，当你下载 MP4 格式的视频时，Screenr 标志不会出现在视频上。

如果你发现你在一遍又一遍地向你的客户解释同样的事情，或者也许你想向他们展示如何使用 CMS 或 WordPress，你可以制作一个截屏并把链接发给他们。你不需要担心托管，因为截屏是由 Screenr 托管的。

如果有缺点的话，那就是视频最长 5 分钟。但这也许是一件好事，因为它可能有助于让你在解释时保持专注和简洁？总的来说，虽然这是一个令人难以置信的令人印象深刻的应用程序，我将用于我自己的网站。
