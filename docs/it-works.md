# “管用！”

> 原文：<https://www.sitepoint.com/it-works/>

任何称职的销售人员都知道，没有人会根据产品的特性来购买任何东西，为了传达价值，你必须将特性转化为利益。

“功能与优势”是营销 101。然而，即使你是将特色转化为利益的大师，你仍然可能在购买过程的最后也是最关键的一步失败——引发促使你的潜在客户采取行动的需求或愿望。

你看，激励人们购买的不是利益本身，而是情感。情绪是促使我们花费时间、金钱或精力的原因。换句话说，人们购买是因为与利益相关的情绪，而不是利益本身。

我给你举个例子。最近，我大儿子的童子军举行了一年一度的爆米花销售筹款活动。我们在当地的一个加油站/小超市前(经允许)扎营，并安排了两个小时的轮班，每个轮班有两个侦察兵。轮到我们的时候，我教了孩子们一些基本的礼仪:“站直”，“允许人们进入迷你超市；不要挡住他们的去路”、“说话坚定但有礼貌”等等。

有一段时间忘记了我是谁，我是干什么的(嘿，那是周末)，我没有太注意男孩们的罐装投球(“你想买些童子军爆米花吗？”)对此，大多数人礼貌地回答，“不，谢谢，”或者完全无视他们。

直到有个顾客问:“这是干什么用的？”男孩们回答说，“这是给童子军的！”我的销售培训师雷达终于开始工作了。在告诉那个女人他们正在为去葛底斯堡旅行筹钱后，我把男孩们拉到一边，提出了一个新的方法。我指示男孩们说以下的话:

"你想买些爆米花来帮我们挣钱去葛底斯堡旅行吗？"

你看，他们最初的推销并没有解决潜在的爆米花购买者没有提出的问题:“我有什么好处？”额外的“帮助我们赚钱去葛底斯堡旅行”提供了一些超出价格过高、味道一般的爆米花的好处(值得怀疑)。引发了欲望，促使人们行动起来。换句话说，它提供了一个购买动机:帮助童子军的愿望。买爆米花满足了这种欲望，每个人都从帮助一个有价值的组织中获得了良好的感觉。

结果呢？在接下来的 45 分钟里，直到我们的轮班结束，每个路过的人要么买了爆米花，要么为这次旅行捐了款。用男孩们在第一次使用新方法销售后的话来说:“它起作用了！”
