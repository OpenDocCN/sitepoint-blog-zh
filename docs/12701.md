# 雅虎！挖掘它内心的吉普赛

> 原文:[https://www.sitepoint.com/yahoo-taps-its-inner-gypsy/](https://www.sitepoint.com/yahoo-taps-its-inner-gypsy/)

也许雅虎！毕竟微软会是一个很好的选择。他们的广告部看起来至少会配合得很好。本月，两家公司都发起了看似毫无意义的广告活动，这些活动似乎并不是扭转两家公司命运的必要良药(当然，要记住，微软仍然拥有巨额财富，并能赚得盆满钵满)。

微软古怪的新广告活动让比尔·盖茨和杰瑞·宋飞出现在尴尬的场景中，这与 Windows 或微软无关。这在博客圈遭到了严厉批评(根据我前几天晚餐时对朋友的非科学调查)。

今天， [TechCrunch 注意到](https://www.techcrunch.com/2008/09/15/yahoo-bleeds-purple-and-you-thought-microsofts-marketing-campaign-was-weird/)雅虎的一项新活动！围绕[的是紫色](https://startwearingpurple.com/)，这是该公司的官方颜色(我猜)。这是一个品牌认知活动，包括一个特殊的 Flickr 帐户上的紫色主题照片，紫色即兴喜剧视频，一队紫色自行车，以及其他紫色的东西。

该网站是由一段视频介绍的，视频中有所谓的吉普赛朋克乐队 *Gogol Bordello* 的歌曲“开始穿紫色”。

尽管我很喜欢果戈理妓院(我在 8 月份看到他们经过我的城市，他们上演了一场精彩的表演)，但很难看出这场广告宣传的意义。它可能没有花费太多的成本，而且它可能有助于做一些品牌建设，但这真的是雅虎想要的吗？需求？人们真的不用雅虎吗？因为他们从未听说过它？我不这么认为。正如迈克·阿灵顿所说，雅虎！花这笔营销预算想办法说服人们真正使用他们的服务可能是明智的。

网站最奇怪的部分？右下角的“分享”按钮链接到社交媒体网站 Delicious、脸书、Digg、Reddit、MySpace 和 StumbleUpon，但不是雅虎！的自有 Buzz 属性。也许这表明了雅虎的问题所在。:他们自己的营销努力未能推广他们的网站。

不管怎样，为了让你开心，这是果戈理妓院的“开始穿紫色衣服”的音乐视频
