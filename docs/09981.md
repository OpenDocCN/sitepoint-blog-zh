# 卡片分类介绍

> 原文:[https://www.sitepoint.com/an-introduction-to-card-sorting/](https://www.sitepoint.com/an-introduction-to-card-sorting/)

在所有可以用来改进你的网站的研究中，卡片分类是最令人愉快的一种。

## 什么是卡片分类？

卡片分类是一种观察人们将信息分组的独特方法。它使用实际的索引卡片，每张卡片上都印有内容。你观察参与者通过组织卡片创造意义。

最常见的卡片分类有几个基本步骤:

1.  在每张代表示例网站内容的卡片上写一个单词或短语。每张卡片在内容层次结构中应该处于大致相同的级别(这样就不会有一些卡片是博客文章的标题，而另一些是主要的导航项目)。
2.  概括地解释你为什么要做这个活动，并要求参与者以最有意义的方式将相似的卡片组合在一起。你可以提到这将有助于你的网络项目的结构，但是注意不要过分强调网络方面。这将有助于他们专注于内容本身，而不会陷入网站布局。
3.  请确保您获得了录制音频的许可，然后开始录制。如果你能以一种不会让人觉得打扰的方式录制视频，那就去做吧。无论哪种情况，手边都要有笔记本电脑或记事本来记笔记。
4.  当他们开始分类时，听和看。回答任何关于卡片内容的问题，因为你想确保每个参与者都清楚地理解卡片上的内容。在你完成一些排序后，你会意识到哪些卡片的措辞很差，需要修改。
5.  请参与者为每组创建一个标签和描述。准备一些便利贴和一支笔。
6.  感谢他们，然后在电子表格中记录你的结果。

就是这样！这种类型的卡片分类被称为“开放的”,因为你不需要为它们确定组名；他们可以选择他们认为合适的任何标签。在“封闭式”卡片分类中，你会给他们主要的类别，他们会把卡片放在适当的标题下。

## 不是银弹

对卡片分类价值的一个常见误解是，应该收集、编译数据，然后创建新的站点结构来模拟结果。你收集的数据应该是你整体研究方法的一部分，综合考虑所有的东西将有助于形成你的最终结构。

卡片分类的价值在于观察过程和收集数据一样多。该流程决定了您查看数据的方式。为了利用这一点，重要的是向参与者解释活动，然后要求他们描述他们进行活动时的思维过程。

然而，这种方法的固有问题是

a)有些人发现很难用语言表达他们的想法

b)一旦参与者意识到有人在记录和分析他们说的每一句话，结果可能会有偏差。

任何对行为的观察都有可能扭曲行为本身，但有一种方法可以减轻这种负担，那就是群体卡片分类。

## 分组排序有什么帮助

小组卡片分类是一种让两三个人一起完成任务的分类。

起初，我对尝试这种方法犹豫不决，因为它听起来类似于焦点小组，而焦点小组往往会受到群体思维和社会影响的影响。但最终，我发现这真的很不一样。

当两三个人被分配给分类信息并贴上标签的工作时，他们往往会非常投入。当他们互动、讨论、争论和开玩笑时，你和你的录音设备就变得不那么引人注目了。

我曾经进行过排序，结果是小组排序比个人排序更乱，但我记录的对话给了我洞察力，否则我不会有。对于难以放置的内容，个人可能会在内部决定做出并坚持最初的决定，但为了达成共识，两个参与者最终会用语言表达利弊。

这种技巧可以给你洞察力，挑战假设，在会议中提供有价值的讨论点，并告知你的整体战略。如果你不熟悉，我希望这能给你一个考虑尝试的理由！
