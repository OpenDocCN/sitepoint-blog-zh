# Ruby on Rails 1.0 已经过时了

> 原文:[https://www.sitepoint.com/ruby-on-rails-10-is-out/](https://www.sitepoint.com/ruby-on-rails-10-is-out/)

刚刚发布了 Ruby on Rails 1.0 ，从而消除了你的最后一个借口(“它甚至不是一个稳定版本！”)因为没有尝试今年最流行的 Web 开发框架。

对于那些一直生活在岩石下的人(或者，我必须承认，对于那些一直在等待 SitePoint 写关于它的文章的人)，Ruby on Rails 是一种编程语言(Ruby)和一个框架(Rails)的双重打击，它们共同形成了一个快速构建动态网站的强大平台。

Ruby on Rails 是围绕一种敏捷开发方法设计的，在这种方法中，您从匆忙地做一些简单的工作开始，然后在这个核心上做一些小的改变，逐渐产生最终的产品。例如，只需要几分钟就可以配置 Rails 来访问您的数据库，它可以自动生成一组页面来查看您的数据库记录，并通过管理界面添加/修改/删除记录。然后，您可以逐步调整自动生成 Web 应用程序，以获得您想要的外观和功能。

在新的一年里，请关注 SitePoint 对 Ruby on Rails 的大量报道。与此同时，你可以细读 [SitePoint 的 Ruby 论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=227)。

人们希望对这次发布的巨大反响，而不是 Ruby on Rails 平台本身，是导致其网站运行缓慢的原因…
