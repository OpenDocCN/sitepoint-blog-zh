## [6。用 jQuery](http://www.filamentgroup.com/lab/setting_equal_heights_with_jquery/) 设置相等的高度

我们的 equalHeights()函数确定容器中所有同级元素的高度，然后将每个元素的最小高度设置为最高元素的高度。当 JavaScript 被禁用时，框或列以不同的高度显示，但内容仍然清晰可见，页面仍然完全可用。
