# 单页网站的艺术

> 原文:[https://www . site point . com/the-art-of-the-single-page-web-site/](https://www.sitepoint.com/the-art-of-the-single-page-web-site/)

设计师们正在利用他们的技能创建一个网页网站来展示投资组合、应用程序、发布网站和登录页面。技巧在于在一个页面中为用户提供他们需要的信息。无论从设计还是写作的角度来看，制作一个简明扼要的网站都是一个挑战。在本帖中，我们将看看那些设法满足这些要求的创意网站。

虽然有些网站只有最少的信息，但其他网站已经设法在一个页面上塞进了与多页面网站相同的信息量。包含大量信息的站点倾向于使用导航栏，但不是跳转到一个新页面，而是在同一页面上向下跳转，通常使用 jQuery 来提供平滑的向下或向上滚动。

因此，废话不多说，为了你的灵感和浏览乐趣，这里收集了八个设计精美的网站，努力在一个网页中提供所有需要的信息。
