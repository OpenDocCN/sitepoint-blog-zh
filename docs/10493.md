# 铺开网络红地毯

> 原文：<https://www.sitepoint.com/rolling-out-the-cyber-red-carpet/>

我倾向于认为可能会有一个人注意到我上周没有发帖子。如果你是那个人，那么让我告诉你为什么。上周一对我们 SitePoint 来说是个大日子。我们将我们的论坛(和其他几个系统)从旧的遗留服务器迁移到亚马逊的 EC2 云服务。迁移进行得非常顺利，只有几个小时的停机时间，没有出现大的故障。这确实意味着我今天很忙，所以没有博客。幸运的话，如果你是论坛用户，你可能会注意到速度的提高。我当然是。现在迁移已经结束，我将注意力集中在即将到来的颁奖季。是的，没错，我们铺上了网络红地毯，穿上了电子连衣裙，因为现在是 2010 年 SitePoint 社区奖的时候了。有两周的时间[为 25 个奖项中的任何一个提名你最喜欢的论坛成员](https://www.sitepoint.com/forums/showthread.php?p=4728586#post4728586)。你可以提名任意多或少的类别，**提名将于 11 月 15 日**截止。这是你感谢某个可能在某个时候帮助过你的人的机会，或者认可某个让你在 SitePoint 过得更愉快的人。这些奖项都是关于你们的，所以[现在就获得提名](https://www.sitepoint.com/forums/showthread.php?p=4728586#post4728586)！![SitePoint forums](img/d9747a589561c9ab9868d1471efbf3a1.png "sitepoint-forums")对于我们的脸书粉丝来说，这是激动人心的一周，10 月份的“*了解我们*”活动将以一个小测验达到高潮。该活动有一个营销重点，你被介绍给我们的营销人员 Shayne Tilley，以及联合创始人 Mark Harbottle 和 SEO 大师 Kristen Holden。这个测验比上一对要难一点，因为奖金很大。如果你答对了所有的问题，你将参加抽奖，赢得一份我们最新的工具包，由克里斯汀合著。这个工具包的标题是*“SEO 商业指南”*，它充满了比你所能掌握的更有价值的技巧和信息。所以，今天晚些时候，请前往查看我们的[脸书人物标签](http://www.facebook.com/sitepoint?v=app_11007063052)进入。下周，我们将启动一项新的活动——这一次，我将重点关注社交媒体以及我们如何在 SitePoint 利用它。仍然是关于[脸书](http://www.facebook.com/sitepoint)的话题，你们中的一些人问我们是否有以前*了解我们*人物传记的档案，答案是没有。我已经决定将这些信息归档到选项卡中，可以通过选项卡下拉列表或通过单击当前[人物选项卡](http://www.facebook.com/sitepoint?v=app_11007063052)上的链接获得这些信息。我将在本周着手此事。**本周热门话题**在[通用应用设计与理论论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=233)上，正在进行一场极客式的讨论。如果你有任何用 PHP 开发 OOP 应用程序的经验，请过来加入。Dez 在 [HTML 和 XHTML 论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=52)提出了一个有趣的问题。如果你已经花了很多时间研究和编辑内容，什么是最好的方法使你的文本难以复制，同时让搜索引擎蜘蛛仍然可以使用它？保罗·欧博本周在 CSS 论坛进行另一个著名的 CSS 测验。如果您想测试您的技能，请查看[测试您的 CSS 技能编号 35——内嵌块](https://www.sitepoint.com/forums/showthread.php?t=709009)。本周最后一个是一个较轻的。跳转到[普通聊天](https://www.sitepoint.com/forums/forumdisplay.php?f=9)，告诉我们[你在万圣节](https://www.sitepoint.com/forums/showthread.php?t=710135)有什么计划(或者你做了什么)，如果这是你所在地区庆祝的事情。这就是我这周的全部内容。下周同一时间同一地点见。*特征图像来自克里斯·罗伯茨[数字精度](http://www.digitalprecisionnc.com/)*
