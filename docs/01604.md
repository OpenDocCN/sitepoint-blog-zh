你需要一个网站。你很清楚自己想要什么，并且已经仔细考虑过[你需要的托管类型](https://www.sitepoint.com/sort-hosting-choose-website/)。现在，有了信用卡在手，就该决定和哪家公司签约了。

选择主机提供商是你要做的最重要的决定之一。你的网站的未来取决于它。

如果你在为你的孩子寻找保姆服务，你不会只选择最便宜的。“把你的孩子交给我们，每天只需 50 英镑。随时都可以接。”你的孩子对你来说比那更重要。你会想确保他们是安全的，并且被知道他们在做什么的人照顾着。省钱不是你的首要任务；投资于他们的福祉才是。

投资一些时间在你的网站的未来健康发展上。你应该付钱给谁来托管你的网站？你在一家托管公司需要具备哪些素质？

以下是衡量选择时要考虑的六个关键标准。

## 1.速度和性能

你还记得上一次买新笔记本电脑是什么时候吗？你马上就注意到了性能的提升，旧的突然感觉出奇的慢。快是好事。你想要一个相当于新笔记本电脑的托管公司。

第一印象就是一切。你不希望新的访问者在你的主页加载之前离开你的网站。托管公司对绩效的承诺程度如何？它需要在硬件和软件方面的持续投资。

评估托管服务提供商时，仔细阅读此**绩效清单**:

*   多个大洲的服务器。联系越紧密越快。如果你的网站有来自世界各地的访问者，位于多个大洲的服务器将确保每个人都有最快的连接时间。

*   CDN(内容交付网络)旨在通过在世界各地建立代理服务器网络来减少延迟。许多公司提供付费的 cdn，有些公司免费提供。

*   装有固态硬盘而不是旋转硬盘的服务器会更快地加载你的网站。

*   HTTP/2 是互联网的未来，现在就在这里。它比原来的 HTTP 快十倍左右。确保你的主机支持它，并在 Craig Buckler 的文章中了解更多信息。

*   缓存选项可以显著提高高流量 WordPress 网站的速度——使其速度可能提高几百倍。留意 PHP 7 操作码缓存、Memcached/Redis 对象缓存、Nginx/Varnish 全页面缓存、CDN 和 ElasticSearch/Solr 全文搜索等选项。

很难判断一家公司多久升级一次硬件，或者他们在硬件上投资了多少。以下是你可能会发现的一些方法:

*   查看该公司的博客，了解硬件和软件升级公告。

*   使用负载影响和 WAPT 等服务来加载测试网站，看看它们的表现如何。

*   或者直截了当——打电话给他们的客服人员询问。

## 2.安全性、可靠性和正常运行时间

当你的网站关闭时，它会对你的业务产生不良影响，你可能会赔钱。如果黑客进入你的网站并安装恶意软件，不仅你会感染访问者的电脑，谷歌也会显示一条令人尴尬的警告信息。

公司对维护你的网站的稳定性和可靠性有多大的承诺？在当今黑客盛行的环境下，该公司采取了什么措施来保护您的网站安全？

黑客可以通过利用您正在使用的软件中的安全漏洞来获得访问权限。所以你需要一个主动更新软件的主机来修补这些漏洞。公司也应该采取额外的措施来确保你的网站的安全。

评估托管服务提供商时，请完成此**安全清单**:

*   服务器软件更新。公司运行的是最新的服务器软件(PHP，Apache，MySQL)吗？
*   漏洞处理。公司是否主动在服务器级别打补丁？
*   自动 CMS 更新。该公司是否为您的内容管理软件(如 WordPress)提供自动更新？它们允许您选择更新发生的时间吗？你不希望 WordPress 在一个重要的营销活动中更新。
*   公司是否采用暴力检测和预防系统？这些系统监控和分析所有到你的站点所在服务器的流量，并阻止欺诈性的流量——例如有人试图通过猜测你的密码来侵入你的 WordPress 管理仪表板。
*   公司是否有网络应用防火墙(WAF)？规则多久更新一次？针对新漏洞创建规则的频率如何？理想情况下，主机是主动的，每次有漏洞，他们都会添加一个 WAF 规则来保护你。

网站会发生不好的事情。像电脑上的其他东西一样，备份是必不可少的。它们使灾难不那么灾难性。

评估托管服务提供商时，请仔细阅读此**备份清单**:

*   该公司有备份吗？一些主机将责任留给最终用户。理想情况下，你需要一个能为你做定期备份的公司。
*   公司实际上是如何执行备份的，多久执行一次？理想情况下，您应该寻找定期的每日备份，副本保存至少 30 天。副本越多，为你保留的时间越长，越好。
*   必要时，您能以多快的速度恢复？理想情况下，主机提供简单的一键式恢复过程。
*   他们的恢复收费是多少？恢复备份可能并不便宜，一次恢复可能要花费 150 美元。

Web 服务器就像任何其他计算机一样。程序崩溃，事情出错，你需要重新启动。在网络服务器上，这会使你的网站瘫痪。你需要一个能把影响降到最低的托管公司。

当 web 服务器出现问题时，通常会发生以下情况:

*   通知服务器管理员，如果可能的话，登录服务器。
*   他们排除故障，解决问题，有时重启机器。
*   重启导致停机。

公司可以通过拥有出色的监控系统来最大限度地减少停机时间，这种系统可以尽可能频繁地主动检查服务器状态，并自动修复问题，而不是等待系统管理员来做。

你如何发现一个托管公司的监控系统的有效性？查看他们的网站和博客，或者询问他们的售前代表。拥有良好监控系统的主持人会毫不犹豫地分享他们是如何做到的。

你也可以使用像 [Uptimerobot](https://uptimerobot.com/) 、 [Pingdom](https://www.pingdom.com/) 和 [Uptrends](https://www.uptrends.com/) 这样的工具来监控你自己网站的宕机时间。

## 3.客户服务和支持

你正忙于建立你的网站。时间停止了，你全神贯注于创作过程。一切都在一起…直到你遇到了障碍。出了问题，或者你不确定下一步该怎么走。快到午夜了。你是做什么的？你向谁求助？

人们在正常的朝九晚五工作时间之外工作是很常见的。无论何时，当你需要帮助时，你都需要帮助。你需要有人倾听你的问题，知道该怎么做，并能以友好、可理解和人性化的方式与你交流。你需要很好的支持。

评估托管服务提供商时，请完成此**支持清单**:

*   何时提供支持？理想情况下应该是 24/7——你不会总是在上班时间工作。
*   有哪些支持渠道？电话？聊天？门票？哪些是全天候可用的？优先支持可以多付钱吗？选择越多越好，但是对于复杂的问题，票务总是更好。
*   支持团队有多友好？是内部的还是外包的？你是否在他们的聊天和票务系统中看到员工的个人资料，这样你就可以很容易地看到另一边有一个真实的人。
*   公司解决问题的响应时间是多长？你不想等三天才得到你的问题的答案。在聊天和打电话时，你要等多久才能接通接线员？
*   通过支持渠道可以解决什么类型的问题？许多主机只帮助解决与服务器相关的问题。尝试找出有哪些限制，哪些问题被认为是“超出了托管支持”。
*   寻找可用的文档和自助工具——教程、知识库。
*   支持是否以您的语言提供？

## 4.功能和限制

托管公司必须提供必要的软件和基础设施来运行你的网站和内容管理系统。它们还应该提供支持你喜欢的工作方式的功能，无论是极客式的还是脚踏实地的。他们对你的托管计划的任何限制都不能对你想在网站上实现的目标造成太大的限制。

评估托管服务提供商时，请完成此**功能清单**:

*   他们是否提供了运行和管理网站所需的所有系统软件和服务？CMS，操作系统，数据库，网络服务器，编程语言？请参考您的内容管理系统的系统要求，并自行制作所需软件的清单。
*   除了虚拟主机，公司还提供你需要的其他在线服务吗？这可能包括域名注册和转让、电子邮件托管和数据库管理。把所有东西都放在一个地方很方便。
*   如果你看重易用性，他们是否提供 cPanel 这样的控制面板，以及 WordPress 这样的流行内容管理系统的简易安装？
*   将你的网站转移到一个新的主机提供商可能会很棘手。他们提供网站迁移吗？他们是免费的还是付费的？条件是什么？最终，您希望找到一个主机来为您免费完成迁移。
*   如果你是极客，他们会提供你选择的工具吗？SSL？我们加密吧？分期？FTP？宋承宪？
*   他们把 WordPress 插件列入黑名单了吗？很多 WordPress 托管公司不允许你安装一些插件。
*   存储和带宽的实际限制是什么？使用“无限”这个词的公司仍然对他们认为太多的存储和带宽有限制。在他们的服务条款和合理使用文件中检查这些限制。每个计算机系统都有极限；重要的是公司对他们的透明度如何。

## 5.哲学、文化和声誉

你需要一个可以信任的公司。一个关心你的需求，而不仅仅是为了钱的公司。一家乐于助人、容易打交道的公司，而不是一家总是试图向你推销东西的公司。一家公开透明的公司，不会欺骗你。不会给你留下遗憾的公司。

评估托管服务提供商时，请仔细阅读此**信誉检查表**:

*   公司有多人性化？他们的“关于我们”页面上有什么？他们的价值观和哲学是什么？他们的故事是什么？
*   检查他们的社交媒体渠道。他们是真正致力于自己工作的人吗？他们如何应对社交媒体上的投诉和负面反馈？博客上的负面评论？
*   他们是如何对待员工的？他们的办公室是什么样子的？他们组织公司撤退吗？真正关心员工的公司也会关心他们的客户。
*   他们的网站是清晰易懂的，还是令人沮丧、迷惑和误导的？你有没有感觉他们是公开透明的，或者试图欺骗你？
*   谁写博客，他们写些什么？主要是客座作家创造链接诱饵，还是公司员工公开分享他们如何经营业务？
*   他们对环境负责吗？数据中心消耗大量电力。他们使用可再生能源吗？他们如何抵消他们的碳足迹？
*   公司有多稳定？他们是一个运行着托管公司网络的大公司，一个不可持续的希望被收购的小公司，还是一个有着坚实的商业计划，致力于长期存在的成长中的公司？
*   主机提供商从开源软件中获利。他们会回馈社区吗？他们会将现金或编程时间回馈给开源项目吗？他们组织或赞助像单词营这样的活动吗？

## 6.定价

选择虚拟主机公司时，每月省几块钱不应该是你的优先考虑。一定要货比三家，但要寻找最物有所值的东西，而不是最便宜的价格。做好你的功课，这样你就不会被不清楚或误导的广告所愚弄。

提供快速、可靠、安全的主机服务需要花钱。支付一个合理的价格是对你自己网站的一种投资，通过加强托管网站的公司。为优质产品付费甚至会感觉很好！

评估托管服务提供商时，仔细阅读此**定价清单**:

*   不要被铅价忽悠了，查查它提供了什么。许多公司会试图用他们最便宜的价格抓住你的注意力，而托管计划即使对于最基本的需求也是极其有限的。
*   一些公司宣传注册价格低，但在续订时收取较高的价格。检查持续的成本——你将会为此付出很多年。
*   检查订阅期。你需要预付多少钱才能得到一个好价钱？许多公司提供最长计费周期(比如 3 年)的主导价格，而包月和包年的费用更高。
*   公司提供试用期吗？他们的退款政策是什么。
