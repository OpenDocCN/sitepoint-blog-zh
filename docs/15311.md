# 垃圾邮件斗士的工具包

> 原文:[https://www.sitepoint.com/spam-fighters-toolkit/](https://www.sitepoint.com/spam-fighters-toolkit/)

如果你不知道这一行业的真正诀窍，与永无止境的垃圾邮件作斗争可能会变成一次非常令人沮丧的经历。毕竟，这不仅仅是简单地回复一个(通常是伪造的)发件人的投诉。

这里有一个相当广泛的资源概述，将帮助您有效地打击未经请求的电子邮件，并向您展示可能性(唉，限制！)的努力。

##### 一般资源

*   Spamhaus 项目的特色是一个数据库，它可以跟踪已知的垃圾邮件团伙、垃圾邮件支持服务以及通过在其网络上故意托管秘密垃圾邮件服务来保持有组织的垃圾邮件活动的提供商。大量的数据库允许跟踪已建立的垃圾邮件组织，包括统计数据等。
*   查看[已建立垃圾邮件列表](http://mosa.unity.ncsu.edu/brabec/antispam.html)。
*   阅读标题的资源列在负责任和道德的电子邮件论坛上。
*   这里列出了许多 spambot 骚扰程序。
*   [Spam.abuse.net](http://spam.abuse.net)呼吁抵制垃圾邮件，并提供大量关于垃圾邮件预防立法的信息及更多！点击这里，阅读他们题为“如何向垃圾邮件发送者的供应商投诉”[的实用指南。](http://spam.abuse.net/howtocomplain.html)

##### 邮件转发服务

*   通过注册 [Sneakemail](http://sneakemail.com/) ，使用一次性电子邮件地址保护您的邮箱。这项服务会将所有邮件转发到您的常规邮箱，而不会泄露您的真实地址。如果你发现你的 Sneakemail 地址被滥用，你可以直接删除它。此外，它还能帮助你追踪那些向第三方营销商兜售你的地址的商家。干净利落。

*   Spamex 采取了类似的方法，提供一次性的电子邮件地址作为对抗垃圾邮件的措施。然而，它并不理会复杂的垃圾邮件过滤器——当你的 Spamex 地址收到垃圾邮件时，只需将其销毁并获得一个新的即可。您还可以将他们的登录框链接放入 Web 浏览器的链接栏中，以便于访问。他们的口号也值得注意:“因为给你发邮件是一种特权，而不是权利！”
*   最著名的反垃圾邮件转发者之一是 [Spammotel](http://www.spammotel.com/) (多好的名字！)它还为你的电子邮件客户端提供了一个非常复杂的、获奖的插件，可以让你跟踪谁给了你哪个电子邮件地址。当然，这使得测试网站的隐私政策变得非常容易。此外，它还是一个很好的工具，可以帮助你组织你确实想收到的邮件。仅限 Windows。
*   [邮件滥用预防系统有限责任公司](https://stop.mail-abuse.org/)提供商业垃圾邮件防护转发服务。

##### 垃圾邮件过滤器

*   运行自己的邮件服务器的网站管理员可能会对地图垃圾邮件中继拦截器感兴趣，这是一个基于 DNS 的垃圾邮件中继服务器数据库。如果您想要拒绝来自这些类型的服务器的邮件，您可以配置您的服务器来利用它们的列表。
*   该网站还提供实时的[黑名单(RBL)](http://mail-abuse.org/rbl/) 。这是一种故意造成网络中断(“黑洞”)的系统，其目的是限制已知不需要的大量电子邮件的传输。RBL 是一个只订阅的系统，其工作方式使得没有人被拒绝连接到非 RBLSM 订户。

##### 举报垃圾邮件

*   这本[垃圾邮件投诉初级读本](http://combat.uxn.com/tracing.html)详细说明了这一切，并提供了一个投诉样本，涵盖了举报垃圾邮件的每个重要方面，以终止垃圾邮件发送者的帐户和网站。
*   向其报告垃圾邮件的最经典案例是非营利性的[邮件滥用预防系统有限责任公司](http://mail-abuse.org/)。
*   [网络滥用信息交换所](http://www.abuse.net/)是一项投诉服务，它会将你的垃圾邮件投诉转发给充斥你邮箱的罪犯。
*   MMF 耻辱馆用嘲笑的方式来打击垃圾邮件发送者。它的特点是大量令人讨厌的不请自来的商业电子邮件的模仿和欺骗。尽管如此，它有一个非常严肃的背景，并提供了关于欺诈起诉的详细信息，以及对垃圾邮件诈骗的初步法律分析。
*   这个网站提供每月垃圾邮件报告的案例研究，并列出了成功案例。你会发现虽然这是一场艰苦的战斗，但这是可以做到的。
