# 2012 年 8 月浏览器趋势

> 原文:[https://www.sitepoint.com/browser-trends-august-2012/](https://www.sitepoint.com/browser-trends-august-2012/)

自从我们[上次查看全球浏览器使用图表](https://www.sitepoint.com/browser-trends-july-2012/)以来，已经过去了一个月。年中的时候通常会更安静，所以让我们看看 [StatCounter 的统计数据](https://gs.statcounter.com/#browser_version-ww-monthly-201207-201207-bar)告诉我们什么…

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 8.8%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

虽然 Chrome 上个月努力实现 0.3%的增长，但其典型的 1%以上的增长在 7 月份又回来了。然而，它从 Firefox 而不是 ie 浏览器那里获得了最大的份额。去年这个时候，Firefox 占有 28%的市场份额，在过去的 12 个月里下降了 4%。这不是毁灭性的，但 Chrome 同期增长了 12%。

我怀疑由于内存消耗和稳定性问题，Firefox 已经失去了 Mac 和 Linux 用户。Windows 操作系统的性能更好，Firefox 与 Chrome 不相上下，但主流用户可能更喜欢谷歌浏览器提供的更简单的体验？Firefox 仍然是开发者的首选浏览器，但是，尽管它提供了更好的工具，许多人已经转向 Chrome。

IE9 有一个合理的月份，尽管它的上升速度比 IE6 慢，7 和 8 正在下降。微软的电视和电影广告可能会产生一些影响。我仍然傻笑在试图使 IE 看起来很酷，但它显然是工作。

Safari 继续稳步上升，MAC 和 iPads 的持续成功将是一个主要贡献者。我最近在推特上发了一个问题，问我应该买什么牌子的电脑，几乎每个
回答者都用*【苹果】*回答。

这个伟大的 [**StatCounter 视频展示了 2008 年至 2012 年**](https://www.youtube.com/watch?v=jtBlvutR9cU) 期间的浏览器市场动向。世界开始是蓝色的(IE)，闪烁着黄色(Firefox)和一点红色(Opera)，然后迅速变成绿色(Chrome)。

## 移动浏览器的使用

7 月份，手机使用量略微上升，占所有网络活动的 11.12%。暑假和奥运会将会产生影响。

主要的移动浏览应用程序有:

1.  安卓系统——23.14%(增长 1.17%)
2.  iPhone——21.45%(增长 0.46%)
3.  opera Mini/Mobile–19.42%(下降 2.00%)
4.  诺基亚浏览器——10.15%(下降 0.84%)
5.  UC 浏览器——8.19% 8.51%(下降 0.32%)

Android 这个月表现出色，尤其是当你考虑到 Opera 用户和 Dolphin(1.26%)也在使用这个平台的时候。个人认为两种选择都比股票安卓浏览器好。

Opera 下滑了一个名次，并出现了惊人的巨额亏损。然而，请记住图表记录了使用情况——Opera 可以在较低规格的手机上使用，但那些拥有 3G/4G 智能手机的人可能会更多地使用网络。我怀疑这是一个显著下降趋势的开始。

另一个大输家是 RIM。黑莓的市场份额从 2010 年 11 月的 19%下滑到今天的不到 5%。智能手机用户是一群善变的人！
