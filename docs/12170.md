# 石蕊碱:Mac 用户的 Windows 浏览器测试

> 原文:[https://www . site point . com/alkaline-by-litmus-windows-browser-testing-for-MAC-users/](https://www.sitepoint.com/alkaline-by-litmus-windows-browser-testing-for-mac-users/)

[Alkaline](http://litmusapp.com/alkaline/) 是 Litmus 公司刚刚发布的产品，该公司专注于开发浏览器和电子邮件简讯测试的应用程序。Alkaline 由 Shiny Development for Litmus 开发，面向 Mac 用户，作为一种在 Windows 浏览器和电子邮件客户端上进行测试的方式，无需使用 Parallels 或 VMware。这是一个桌面应用程序，与你的在线 Litmus 帐户配对，允许你做一些非常强大的测试。

[![alkaline](../Images/a7bb17be9687515e557b4b925fa24a94.png "alkaline")T2】](https://www.sitepoint.com/wp-content/uploads/2009/04/alkaline2.jpg)

**浏览器显示**

您可以选择多达 17 个浏览器进行测试。Litmus 中一个非常有用的特性是浏览器被分为流行和边缘类别，这对于区分结果的优先级和集中精力提高最流行选项的性能非常有用。

对于电子邮件简讯测试，您可以从许多电子邮件客户端中进行选择，这些客户端按企业和消费者进行划分。对于浏览器和电子邮件测试，您可以选择总体默认值进行测试，或者逐个测试地进行选择。

**易用性**

[![screenshots](../Images/cf4b6c43db30d7e61e963a7ce4b33d29.png "screenshots")T2】](https://www.sitepoint.com/wp-content/uploads/2009/04/screenshots.jpg)

要截屏，你只需输入网址或发送一封测试电子邮件简讯(提供了地址)就可以了。几分钟之内，测试就完成了，包括每个浏览器或电子邮件客户端的截图、HTML 和 CSS 验证结果，以及将截图下载到 ZIP 文件的选项。

**复习和复试**

[![litmus](../Images/2ac3f8ab13daedcc438a9672c84d0f26.png "litmus")T2】](https://www.sitepoint.com/wp-content/uploads/2009/04/retest.jpg)

测试完成后，您可以轻松浏览屏幕截图进行查看。截屏可以在各自的浏览器窗口中获得，也可以以整页视图的形式获得(你可以放大这两种视图)。从每个缩略图中，您可以将它标记为兼容或不兼容，这使得以后重新访问更加容易，而无需浏览所有的截图。

另一个很棒的特性是可以根据个人情况选择重新测试。如果您需要重新测试某个浏览器，这可以避免等待整个批次，从而节省大量时间。

**组织**

[![dashboard](../Images/79fdf35b4b18ef18320fe4a4c4bd3c80.png "dashboard")T2】](https://www.sitepoint.com/wp-content/uploads/2009/04/dashboard.jpg)

Litmus 为管理您现有的测试和运行新的测试提供了一个仪表板设置。您可以一目了然地看到您测试了什么以及结果是什么，这使得将来的参考更加容易和更易于管理。它会自动提取您在碱性环境中直接运行的任何测试，因此您可以将所有内容放在一个地方。

其他功能包括:

*   与其他 Mac 开发应用程序的集成——如 Panic 的 [Coda](http://www.panic.com/coda) 和 Macromate 的 [TextMate](http://macromates.com/)
*   附加功能包括通过向静态地址发送邮件来运行电子邮件测试的能力，以及允许您测试当前正在查看的任何网页的 bookmarklet。
*   发布兼容性报告以供团队成员和客户查看的选项。

碱性是[免费下载](http://litmusapp.com/alkaline/)和使用您的免费石蕊帐户。这种免费访问包括在 Internet Explorer 7 和 Firefox 2.0 上的测试。你需要一个付费的 Litmus 帐户来访问所有的浏览器和上面列出的其他功能。价格选项包括 24 小时通票(24 美元)，或每月订阅(49 美元)。
