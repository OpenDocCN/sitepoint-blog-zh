# 改进网站文案的 7 个技巧

> 原文：<https://www.sitepoint.com/how-to-improve-your-web-site-copy/>

如果你是一名设计师或开发人员，你可能会把自己的网站当作一个橱窗，展示你能为客户做些什么。通过你漂亮的设计，强大的功能和创造性的导航，你的网站可以成为你最好的投资组合。

但是有时当我们关注视觉的时候，我们忽略了一个强大网站的最重要的元素之一:文案。如果你的网站文案更多的是一种事后的想法，而不是一个主要的焦点，那么你的网站可能不会起到它应有的营销工具的作用。

那么，如果你对写作不感兴趣，并努力创作吸引潜在客户并有助于销售你的服务的文案，你该怎么办呢？这里有一些简单易行的技巧，你可以用来美化你的网站。

## 知道你的目标

所有好的营销文案，无论是网站还是其他，都集中在将读者转移到一个特定的行动上。在许多情况下，你的网站文案中的行动号召是让潜在客户联系你以获得更多信息，填写表格，或者做一些其他事情，让他们从潜在客户变成销售线索。

当你在写你的文案时，首先确定你的目标是什么，然后在你的整个文案中反复强调，让潜在客户更容易进入下一阶段。

## 专注于订书钉

你可以为你的网站创建许多不同的页面和部分，但是不要忘记四个最重要的部分:主页，关于，服务，联系。客户寻找这些页面，因为它们往往是商业网站的标准；这是他们比较专业人士的方式。所以，确保你在这些页面上的副本，尤其是你的主页，是一流的。

## 解决问题

您的潜在客户访问您的网站是因为他们需要您帮助他们解决他们自己无法解决的挑战。[设身处地为客户着想](https://www.sitepoint.com/understanding-how-your-clients-think)试着理解他们的想法，然后通过为他们的问题提供简单易懂的解决方案来展示你的专业技能。

为了解释你如何能有效地解决客户的问题，回顾一下你在创业前完成的任何市场和行业研究，以支持你的文案。

## 健谈

虽然你的网站文案是一种营销工具，但它不一定是推销。用对话的语气，从个人的角度直接与每个网站访问者交谈。向他们提问，解决你认为他们面临的挑战，并进行一对一的交流。这在你的主页和“关于”页面上尤其有效，在那里你可以提供一些关于你是谁以及为什么你非常适合客户需求的见解。

## 回答问题

对你的服务感兴趣的潜在客户在阅读你的网站后会有具体的问题。虽然你肯定希望他们直接联系你，但你应该试着预测一些最常见的问题，并提供答案，无论是在你的文稿中，还是在指定的常见问题部分。这将有助于让你的网站访问者相信你了解他们和他们的需求。

## 让它与您的设计相结合

为了避免您的副本和您的设计脱节，请花时间微调每个页面，以确保它流畅，并确保副本在您的设计外壳内仍然有价值。保持你的页面简短，使用图形来增强你的内容(而不是与之竞争),让你的导航支持你的文案。

## 获取校对帮助

当你准备好发布新改进的网站内容时，你能做的最好的事情之一就是让别人而不是你自己通读你的内容。当你自己写的时候，你校对错别字、语法错误和其他错误的能力会大大降低。因此，拥有一双全新的眼睛至关重要。
