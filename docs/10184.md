# 星巴克标志的演变

> 原文:[https://www.sitepoint.com/starbucks-logo-evolution/](https://www.sitepoint.com/starbucks-logo-evolution/)

我不喝茶也不喝咖啡，我也从来没有去过星巴克咖啡店，但是我非常清楚星巴克的标志，因为它在世界上很多城市都可以看到。这个标志性的标志已经经历了多年的变化，从七十年代初的一个棕色标签和一个美人鱼的木刻图像开始。现在庆祝它的 40 周年纪念日，星巴克的标志再次演变，去掉了星巴克咖啡的字样，放大了美人鱼图标——海妖——以填满绿色的圆圈。国际品牌公司[利平科特](http://www.lippincott.com/)进行了重新设计。

作为标志设计过程的一部分，设计团队分解了现有标志的四个主要部分——颜色、形状、字体和警笛。这一过程的结果是产生了一个简化的标志。文字被从标记中删除，警笛后面的黑色背景被替换成绿色，警笛从她的戒指中取出。

以前的标志是使用 AutoTrace 的早期版本创建的，因此需要对警笛进行重建和一些整容手术。构图通过使用更复杂的笔画宽度和间距得到了改善。她的头发变得光滑，五官变得精致，尾巴上的鳞片也被改变以突出她的脸。

[![Starbucks](../Images/c93e120a515d8c2e20ec92923d2f51de.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/02/Starbucks.jpg)

行动中的星巴克标识。

四十年来，警笛一直是星巴克标志的核心。公司成立时，设计师发现了一幅 16 世纪北欧木刻的双尾美人鱼。公司创始人认为她代表了一种混合了航海主题的诱人神秘，因此她成为了标志中最重要的元素。随着标志的每一次更新，她的形象变得更加精致(但也更加简单)。就我个人而言，我喜欢新的标志，但我也非常喜欢以前的绿色环，文字和星星标志，所以我仍然完全适应它。不知道客户会不会觉得和以前一样一眼就能认出来？

你觉得新的海妖专用标志怎么样？有用吗？
