*   [Mootools JavaScript 库发布](http://www.snook.ca/archives/javascript/mootools_javasc/)Mootools 是著名的小型 JavaScript 驱动效果 moo.fx 库的超集，增加了 Script.aculo.us 等更完整库中的功能。Mootools 仍然非常小，往往给开发人员更多的底层控制。(标签:[JavaScript](http://del.icio.us/sitepointlinks/javascript)[Ajax](http://del.icio.us/sitepointlinks/ajax))
