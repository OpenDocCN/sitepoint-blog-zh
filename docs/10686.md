# 自己动手做市场调查

> 原文:[https://www.sitepoint.com/do-it-yourself-market-research/](https://www.sitepoint.com/do-it-yourself-market-research/)

市场调查是数据的收集和分析，它对于每一个企业、产品或服务都是必不可少的。除非你做了必要的研究，否则你只是希望有一些运气，因为你不知道你提供的东西是否被需要、想要或有销售潜力。

虽然它通常用于评估新产品或服务，但您也可以出于许多其他原因进行市场调查，包括:

*   衡量新营销技术的有效性
*   评估你的竞争对手
*   识别需要您提供解决方案的潜在问题
*   衡量客户满意度

问题是，对小企业主或自由职业者来说，市场调查可能会令人生畏。你如何找出进行研究的最佳方法，尤其是当你只有很少的预算时，如果有的话？

这里有一些方法，你可以用很少的金融投资进行自己的市场调查，希望能获得一些有用的信息。

## 进行持续的调查

如果你有一个网站或博客，你有一个极好的机会进行市场调查。在大多数情况下，访问你的网站和阅读你的博客的人是你的目标受众的一部分，也正是你想听到的人。您可以创建一个作为网站标准组成部分的投票，或者设计一个包含在选择加入表单中的调查来收集意见。

## 征求客户反馈

将客户满意度调查作为做生意的标准部分是一个好主意；有很多东西可以学习并应用到未来的活动中。如果你有时间，与客户单独面谈可能是一个更好的方法，可以了解他们为什么雇用你，他们对工作流程的看法，以及他们如何设想再次使用你的服务。这些都是有用的数据，可以用在你的市场调查中。

## 参与在线交流

从你的同龄人身上可以学到很多东西，而且在线社交既便宜又容易。从在线网络中获取有用市场信息的最佳方式是参与有意义的互动，然后观察和倾听他人的言论。

## 开始测试

无论你是在一个新网站上收集反馈，衡量电子邮件营销活动的有效性，还是分析新产品的需求，你都可以从测试中学到很多东西。对广告进行分割测试，对网站进行可用性测试，甚至在投入全力之前进行测试，看看反应如何。

## 注意竞争

观察你的竞争对手在做什么是判断什么可行什么不可行的好方法。虽然你不想重复你的竞争对手所做的事情，但你肯定可以找到一些对他们有用的提示。如果你的竞争对手一遍又一遍地使用同样的广告、电子邮件营销和其他技巧，这很有可能对他们有用，他们已经找到了对他们所提供的东西的需求。

市场调查不一定是令人生畏或势不可挡的，不需要投入大量时间或金钱就可以收集到有用的信息。你是如何在你的企业中进行市场调查的？
