# Web 开发出现分歧了吗？

> 原文:[https://www.sitepoint.com/has-web-development-diverged/](https://www.sitepoint.com/has-web-development-diverged/)

在 web 的早期，web 开发人员(或网站管理员)被期望做所有的事情:一点图形设计、数据库开发、服务器端编码和前端工程。随着技术的发展，工作变得多种多样，变化也很快。

如今，就业市场中的差异要大得多:

*   一个设计师可能一整天都在使用 PhotoShop，却从来不接触网站代码
*   前端工程师可能会专注于 HTML、CSS 和 JavaScript，而不关心服务器端的进程
*   后端开发人员可能会创建数据库和 PHP 或 ASP.NET 代码，但从不担心用户界面。

大公司会发现分离工作职能更容易。他们可以创建一个工厂流程，其中每个组件都由一个专门研究该技术的团队来构建。显而易见的好处是，开发人员可以成为某个领域的专家，并在很短的时间内生产出一个可用的产品。

即使是自由职业者和小公司也可以专攻特定的技术。他们可以向更大的组织提供这些服务，或者在需要帮助的地方将部分项目外包出去。
