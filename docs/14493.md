# 最喜欢的 Illustrator 功能

> 原文:[https://www.sitepoint.com/favorite-illustrator-functions/](https://www.sitepoint.com/favorite-illustrator-functions/)

当我在写我的“矢量图形基础”文章时，我一直在思考 Illustrator 中的更多功能，这些功能让我的生活变得更容易。这里还有几个，还是那句话，排名不分先后…

–**概述文本**:曾经想要调整文本中的一些曲线和线条吗？复制一份文本(以防你决定以后要编辑它)，选择原始文本，然后按 Ctrl-Shift-O (PC)或 Command-Shift-O (Mac)。(你也可以键入>创建轮廓。)这会将文本转换为路径，以便您可以实际操作它们。

–**探路者**:我非常非常喜欢不同的探路者选项(窗口>显示探路者)。您可以将对象合并成一个，或者将两个对象重叠，然后“分割”它们，这样重叠的部分也是它自己的对象。

–**锁定或隐藏对象**:当您有一个包含许多对象的复杂文档时，这很有用。选择您想要暂时“锁定”的对象(您可以看到它，但不能选择它)，然后按 Ctrl-2 (PC)或 Command-2 (Mac)。要完全隐藏对象，请按 Ctrl-3 (PC)或 Command-3 (Mac)。另一方面，假设您的文档足够复杂，您最终会花费大量时间来选择所有对象并锁定/隐藏它们，就像您试图绕过它一样。为了更加集中精力，只需选择你*想要*处理的对象，然后按 Shift-Ctrl-Alt-2 或 3 (PC)或 Shift-Command-Option-2 或 3 (Mac)。这将锁定或隐藏所有未被选中的*和其他*对象。最后，要解锁或显示所有对象，请按 Ctrl-Alt-2 或 3 (PC)或 Command-Option-2 或 3 (Mac)。

咻！我希望这不会太令人困惑。
