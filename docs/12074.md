# 垃圾邮件制造者利用猪流感恐慌

> 原文:[https://www.sitepoint.com/spammers-swine-flu/](https://www.sitepoint.com/spammers-swine-flu/)

垃圾邮件发送者正在利用猪流感恐慌的电子邮件感染互联网。在疾病登上头条的几个小时内就开始发送信息，现在估计占全球所有发送电子邮件的 2%。

这些信息通常使用小报式的主题来吸引读者。其中包括“猪流感袭击好莱坞！”，“美国首例猪流感患者！”，还有“麦当娜得了猪流感！”。许多被垃圾邮件过滤器捕获，但是垃圾邮件发送者的策略总是变得更加复杂，并且会尝试成千上万的变种。

以“猪”为特色的域名注册也增加了 3000%。网络罪犯可以自动生成与流感相关的内容，试图利用搜索引擎查询。

大多数垃圾邮件和网络链接似乎指向销售猪流感“治疗方法”(目前没有已知的医学治疗方法)或可下载“信息”文档的虚假网络药店。那些收到假冒或无价值商品的人很可能是幸运者；许多人将什么也收不到，他们的信用卡信息被窃取，或者被鼓励下载恶意软件。

猪流感可能会也可能不会发展成世界范围的疫情，但是关于这种疾病、其传播和受害者的新闻将会持续几周。垃圾邮件发送者总是利用头条新闻和恐惧，但很少找到一个影响地球上每个人的全球感兴趣的主题。

像往常一样，避免打开明显的垃圾邮件或点击链接，即使它们看起来来自朋友或同事。
