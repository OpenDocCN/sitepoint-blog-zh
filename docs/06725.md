## 2012 年 5 月至 2013 年 5 月全球浏览器统计

表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 8.5%的 IE7 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

火狐已经跌破 20%。StatCounter 的记录只能追溯到 2008 年，但我猜 2005 年是 Mozilla 浏览器最后一次达到那个水平。Firefox 1.0 于 2004 年 11 月发布，在 IE6 主导的世界里立即受到了积极的评价。它仍然是一个伟大的浏览器，20%是一个健康的份额，但 Mozilla 永远不会有谷歌和微软的商业影响力。然而，Firefox 是唯一真正的开源浏览器——Mozilla 没有股东或利润目标。尽管 Mozilla 已经停止了相关项目的开发，比如 Camino(一款面向 OSX 的 Gecko 浏览器，现已被 Firefox 取代)和 Thunderbird(一款基于 Gecko 的电子邮件和新闻客户端)，但没有理由怀疑 Firefox 的长期前景。

我不知道 Chrome 是如何设法再增加 2.23%的，但对于一个现在占据超过 40%市场份额的浏览器来说，这是不可思议的。简单来说，Chrome 的用户数量是 Firefox 的两倍。我并不完全相信是这样，特别是因为 [version 27 引入了进一步的预渲染技术](/chrome-27-whats-new/)来在后台下载页面。这会影响统计数据；StatCounter 在过去已经修复了类似的倾斜问题，所以也许下个月我们会看到下降。

Chrome 的大部分增长是以 ie 浏览器下降了近 2%为代价的。IE10 继续取得令人印象深刻的进步，但主要是因为用户正在从以前的版本迁移。如果传言属实，IE11 将于本月晚些时候与 Windows 8.1 一同发布，给谷歌带来更多的竞争。

至于其他人，Safari 和 Opera 都没动。我仔细检查了 Opera——它仍然正好是 1%。基于 Blink 的移动版本正在逐步发布，所以桌面版本应该不会太久。

## 移动浏览器的使用

2013 年 5 月，手机使用量在所有网络活动中所占的比例略微上升至 14.62%——这通常发生在北半球接近夏季的时候。

主要的移动浏览应用程序:

1.  安卓——30.46%(下降 0.50%)
2.  iPhone——23.49%(下降 0.45%)
3.  opera Mini/Mobile——15.45%(增长 0.10%)
4.  UC 浏览器——9.09%(增长 0.35%)
5.  诺基亚浏览器——7.12%(增长 0.09%)

你们当中的数学家会注意到，Android 和 iPhone 浏览器的总损失为 1%，而 Opera、UC 和诺基亚的总收益为 0.5%。原因是:另一种浏览器正在迅速上升到图表的低端。移动版 Chrome 目前占 2.73%，本月应该会超过黑莓。如果它的增长速度和它的台式电脑兄弟一样快，它将在年底前把诺基亚挤出前五名。
