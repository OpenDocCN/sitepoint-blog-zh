# 电子商务网页设计艺术

> 原文：<https://www.sitepoint.com/art-ecommerce-web-design/>

在你设计一个成功的网站之前，你需要一个全面的设计蓝图，首先要定义你的在线存在的目的——以及各种目标的优先级。在过去的几年里，太多的电子商务在启动时没有明确的目标，因此，设计笨拙的网站很快就摇摇欲坠。

你的网站的目的是在网上宣传和定位你的公司吗？你想用花哨的设计和闪光灯打动客户吗？你希望互联网用户给你打电话还是参观你的实体店？或者你想在尽可能短的销售周期内创造高收入——纯电子商务？

可悲的是，在执行电子商务网站设计时，这些目标中的许多都被混淆、误解或排列不当，从而损害了商业网站的总体销售目标。换句话说，当开发一个电子商务网站时，获取客户(阅读:收入流)应该是你设计的首要任务。这意味着需要在网页设计艺术和设计营销驱动的电子商务网站艺术之间划清界限。

##### 智能电子商务的三个网页设计锚

为了有效地达到电子商务的目标，一个营销驱动的网站将是你的工具。这意味着围绕三个指导性的“设计锚”来规划你的网站——和定位你的网页设计

***设计主播 1 号:你的“价值主张”***

你独特的销售主张(或 USP)是什么？你能提供什么好处使你的企业与众不同？你提供的哪些功能或服务明确地创造了价值，同时给了你在竞争中的决定性优势？提炼出你独特的价值主张，并在你的网站上快速清晰地传达出来，这是有利可图的电子商务网站设计的第一要素。

***设计主播二号:你的目标受众***

了解你的市场——并定义你的目标人群的需求——是任何商业企业的重要组成部分。在网上也没什么不同:不是为你自己设计，而是为你的目标受众设计。阐述有意义的好处，并把你的风格和内容放在你的客户的直接需求和愿望的背景下。

平庸的设计是迎合特定市场的网上商店的一个好例子。在为这些受众设计时，网站可以用椭圆形的布局和一层层过分“酷”的东西轻易抹杀电子商务的目标。但这个网站不是“为艺术而艺术”,而是悄悄地为其独特的艺术界观众量身定制设计——同时保持清晰的电子商务议程和高度可用的导航和销售平台。与此同时，平庸的设计也提供了足够的设计元素，向这个利基市场中有辨识力的目标人群传达其可信度。

***设计锚 3 号:任务分析——实现既定目标***

一旦你对你的价值主张传达给你的目标客户感到满意，你就需要一个清晰、集中的销售流程。在这里，分析你的销售渠道的组成部分，提供必要的教育步骤和必要的产品信息，并设计一个网站，优化行动，同时消除分心。让顾客容易向你购买。

在数字游戏室的案例中，操作方式很简单:让在线客户拨打 1-800 电话，销售高度专业化的音乐和工作室设备。在这里，设计和内容是基于这样一种理解，即 99%的数字游戏室客户已经非常熟悉产品规格和竞争对手的定价。通过使用“标价”、“他们的价格”、“喊价”策略，Digital Playroom 向访问者提供足够的内容，然后动员客户行动起来，进行交流，以获得对已成为互联网商品的产品的个性化一对一体验。它甚至暗示了旧世界讨价还价的诱人前景。

##### 为“一秒钟原则”而设计

在互联网上，你只有一秒钟:

*   给人留下深刻印象的一秒钟。
*   一秒钟来建立你的专业精神，并开始建立信任。
*   一秒钟来引起目标观众的兴趣。
*   一秒钟的时间开始下载您的价值主张，并启动一个令人信服的销售提案。

上面的三个设计指南很好地服务于一秒钟原则，并为有效的网站营销和销售流程提供了蓝图。大多数网上购物者没有时间浪费在品牌建设 Flash、迟钝的多媒体插件、过分放纵的使命陈述或无止境的内容上。Gartner 和 Jupiter Media Metrix 最近的研究表明，对于网上购物者来说，便利性、可用性和营销清晰度甚至比价格更重要。

根据这些调查结果，成功的网页设计意味着对每个图形、每个图像和每个单词的位置都有一个战术性的基本原理。这意味着了解你的客户，预测他们的需求，甚至在他们提出问题之前就回答他们的问题。

虽然品牌等更抽象的问题在网上并非无关紧要，但“建立品牌”的过程应该只是一种后续印象，一个强大销售平台的一个系数。获取客户应该仍然是首要目标，无论是 Flash 还是企业品牌都不应该干扰你的营销和销售议程。在这里，燃烧品牌的最佳方式是简单地获得客户群:提供良好的电子商务体验，客户会记住你——即使是在商品驱动的市场。

这是否意味着设计和美学不重要？不，这只意味着电子商务网站的形式应该服务于功能，而不是相反。艺术很少为实用服务。事实上，真正的艺术通常会努力颠覆它。这就是为什么艺术本身不会在网上销售你的产品。
