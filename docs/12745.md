# 比免费好

> 原文:[https://www.sitepoint.com/better-than-free/](https://www.sitepoint.com/better-than-free/)

谷歌可能仍在为其员工提供[免费午餐](http://www.alleyinsider.com/2008/8/google-tells-cnbc-that-dinner-s-still-on)，但对许多公司来说，免费并不是赚钱的途径。博客作者 Allen Stern [今天早上写道](http://www.centernetworks.com/online-price-haggling)“大多数初创公司免费提供了太多东西，用户没有理由升级。”互联网上的许多人开始觉得有权享受免费的东西:免费音乐、免费电影、免费软件、免费信息等等。但是免费可能会带我们走上一条危险的道路。

在线图像编辑工具套件 Aviary 的创始人之一迈克尔·加尔佩特(Michael Galpert)今天在公司的博客上发表了一篇关于在一个免费主导的世界里让人们付费的精彩文章。Galpert 列出了他认为人们仍然愿意为之付费的 5 样东西。这些概念几乎适用于任何产品或服务:

1.  Galpert 说，人们很懒，这就是为什么他们仍然会在 iTunes 上付费购买音乐曲目，而不是从 P2P 网络获得。如果你能为他们提供比免费选择更方便的东西，他们会为此付费。
2.  **质量**–获得更高质量的内容、产品、支持或社区功能是人们愿意为之付费的。例如，如果订阅费使垃圾邮件发送者远离论坛，并使其对行业专业人士更具吸引力，那么人们就会愿意掏钱。
3.  **附加功能**–gal pert 引用了 37signals，他们通过免费赠送开胃菜，但对正餐收费，开创了网络上的“免费增值”模式。正如艾伦斯特恩(Allen Stern)所指出的那样，危险在于，要知道要付出多少，才能让你的付费产品对足够多的用户来说仍然是一个有吸引力的升级产品。
4.  **定制**——定制对于很多产品来说都是一个很棒的增值功能，人们愿意为此付费。Galpert 以 WordPress.com 为例。该网站为任何人提供免费博客，但要定制 CSS 或使用自己的域名，你必须支付一些现金。
5.  **隐私**–虽然隐私应该被纳入任何付费级别的任何服务中，但人们会为额外的隐私层付费。Galpert 谈到 GoDaddy，它提供了一项高级服务，允许人们在 WHOIS 搜索中隐藏自己的信息。

Galpert 的列表对任何提供网络产品或服务的人来说都是有帮助的，因为他们必须与免费产品竞争。与免费竞争是困难的，但正如上面的例子所示，这不是不可能的。
