# 5 款出色的 iPhone 生产力应用

> 原文：<https://www.sitepoint.com/5-awesome-productivity-apps-for-the-iphone/>

如果你和我一样，被 iPhone 的魅力所吸引，那么自 [iPhone 应用商店](http://www.apple.com/iphone/appstore/)推出以来，你无疑也失去了无数个小时的睡眠。在撰写本文时，有超过 2000 个应用程序可供下载，或者是免费的，或者是购买的。

因此，无论是为了弥补设备中已经存在的功能，还是为了将您的生产力提升到一个新的水平，或者只是为了说服自己，您的旧手机应该被替换，是时候[加入这个邪教](http://www.techcrunch.com/2008/06/10/i-am-a-member-of-the-cult-of-iphone/)了，我们认为我们应该将我们在 SitePoint 认为是“必备”的五个 iPhone 应用程序整合在一起。

虽然有些绝对值得付费，但有些是免费的。

1.  [**NetNewsWire (FREE)**](http://www.newsgator.com/NetNewsWire.aspx)

    iPhone 上有很多 RSS 阅读器，但据我所知，只有 NetNewsWire 允许你将阅读列表(以及你已阅读的文章列表)与在线服务(NewsGator)同步。因此，如果你在 Mac 电脑上使用 NetNewsWire(或者在 Windows 上使用 NewsGator ),你就可以在手机和电脑之间同步阅读列表。太棒了。

2.  [**Evernote (FREE)**](http://www.evernote.com/)

    如果你已经用 Evernote 收集和整理了你的文字、图像和声音，那么将免费的 iPhone 版本加入你的军火库是显而易见的。如果你还没有发现在 Evernote 帐户中捕捉生活的便利，那么这个应用程序的 iPhone 版本可能足以说服你尝试一下。免费帐户每月接收 40MB 上传，5 美元/月可获得 500 MB/月。

3.  [**OmniFocus ($23.99)**](http://www.omnigroup.com/applications/omnifocus/)

    桌面版的 OmniFocus 是所有受 GTD 启发的任务管理器之母(仅限 Mac)。iPhone 版本也不例外，具有基于 GPS 的上下文切换和通过苹果的 MobileMe 或 Mac 上的 WebDAV 服务器同步等功能。虽然它肯定有一个学习曲线，但是当从其他任务管理工具迁移时，它是值得坚持的。一个值得注意的遗漏是在桌面上非常有用的“透视图”;希望这将在未来的版本中添加。

4.  [**FileMagnet ($5.99)**](http://www.magnetismstudios.com/FileMagnet)

    iPhone 无法充当 USB 存储设备的问题已经困扰了许多用户一段时间。“这个功能应该内置到操作系统中”的呼声被苹果公司忽视了，但是开发者社区却没有。FileMagnet 是以不同方式解决这一限制的众多应用之一(其他应用包括 [Caravan](http://www.versiontracker.com/dyn/moreinfo/iphone/35945) 、 [Files](http://olivetoast.com/Files/) 和 [DataCase](http://iphone.iusethis.com/app/datacase) )。虽然并不完美(FileMagnet 在处理较大的文件时会遇到内存问题，不幸的是包括许多 SitePoint PDFs)，但其流畅直观的用户界面让它在竞争应用中脱颖而出。

    另一个潜在的障碍是——这是一个 Mac 专用的应用程序。添加 Windows 支持并提高其处理更大文件的能力会将此应用程序提升到列表的顶部；希望开发者能在未来的版本中解决这些问题。

5.  [**Note2Self ($3.99)**](http://iphone.iusethis.com/app/note2self)

    虽然 OmniFocus 允许你在任务中添加照片和音频附件，但对于简单的语音记录来说，这是多余的。Note2Self 是一个简单的应用程序，可以录制音频并通过电子邮件发送到您选择的电子邮件地址。这就是它所做的一切，但是考虑到在这个任务的基础上可以构建什么，这是一个非常强大的小任务。

    Note2Self 提供了一个直观的快速语音记录界面，确保您的想法可以在一瞬间被捕捉到。虽然 Jott iPhone 应用程序应该通过利用强大的音频转录将这一功能提升到一个新的水平，但不幸的是，Jott *在美国和加拿大需要通过电话号码注册*(谈论限制您的受众！)所以没能复习。

    在 Jott 将目光投向北美以外之前，Note2Self 是一个简单易用的给自己发短信的解决方案。甚至有可能使用电子邮件过滤器、automator 和语音识别应用程序来设置类似的东西，但我还没有探索这作为一种解决方案的可行性。对于简短的笔记，我觉得我不会从自动转录服务中受益太多，但对于更长期的目的，比如当我有心情口述我发出的下一份作者合同时，而不是打出来(咳咳！)我可以肯定地看到这个特性将提供的价值。
