# 如何找到客户

> 原文：<https://www.sitepoint.com/how-to-find-clients/>

上周，我告诉你如何在网络行业谋生。在一个假设的场景中，我说如果你需要一年赚 50，000 美元，你的平均项目是 2，000 美元，你必须每月至少获得两个客户。

对此有人评论道:

真的很有趣，但请帮助我们找到客户！

抛开假设场景不谈，这不正是我们都想知道的吗？我一个月去哪找那两个客户？有没有一个地方可以让潜在的网页设计客户都去那里，我可以出现？

这些是我刚开始时问的问题。我知道需要水管工的人会在黄页上找。需要网页设计师的人去哪里找？

我想可能不是黄页。也许是谷歌？(我肯定上第一页应该不是什么大问题。)几年后，在一次客户会议上，我问我的潜在客户，除了我之外，他是否还在考虑其他人。他回答说:“不尽然。我们不知道去哪里找像你这样的人。”

这个特殊的组织需要一个网站已经很久了，但是他们什么也没做。我很肯定我的潜在客户听说过互联网。然而，他似乎从未费心用它来寻找“像我一样的人”我坐在他办公室的唯一原因是因为我主动*联系他们*。

# 请帮助我们寻找客户！

在《几个好人》的经典场景中，杰克·尼克尔森扮演的内森·杰赛普上校正被汤姆·克鲁斯扮演的证人袖手旁观·丹尼尔·凯菲盘问:

你想要答案？

凯菲:我要真相！

杰赛普:你不能接受事实！

这是严酷的事实。尽管集客营销支持者和社交媒体专家声称，你必须寻找新的业务。

如果你刚刚起步，你必须寻找新的业务。如果你需要赚取比兼职更多的收入，你必须寻找新的业务。如果你想发展或者让你的公司“更上一层楼”，你必须寻找新的业务。寻找新业务是你寻找客户的方式。

# **猎人式勘探**

航天飞机的一对固体火箭助推器只在飞行的前两分钟使用；然而，它们提供了将其推进 26 英里进入太空所需的 83%的起飞推力。一旦进入轨道，引力或多或少会起作用。

同样，猎人式的潜在客户创造为你提供了所需的推力，推动你的业务进入营销引力“或多或少”可以接管的位置。然而,*将会花费超过两分钟的时间。*

为了每月获得这两个客户，你必须与足够多的人交谈，以“填满”你的销售漏斗。但是每个月你需要的不仅仅是两个潜在客户。冷打电话和冷游说是用足够多的潜在客户填充你的漏斗的两种方式，这样一些人就会从另一端掉出，进入你的“新客户桶”

# 但是电话推销不起作用！

我工作的公司销售平面广告、网络广告和移动广告。我们的客户群与您相似，都是中小型企业。我们有成千上万的客户，他们都是通过电话推销和游说获得的。在过去的五年里，我培训了数百名销售代表和数十名销售线索生成者来完成这两项工作。

当我听到人们说，“这没用，”通常他们的意思是，“这是太多的工作了。”我最好的销售线索生成器之一平均每月安排 23 次约会，其中三次变成了销售。她每周花 20 个小时，每个月打大约 600 个电话来达成这三笔交易。如果你认为这是“太多的工作”，并且你有一个更简单的方法一个月获得三个客户，那么无论如何，这样做吧。

但是，获得客户不仅仅是猎人式的潜在客户。狮子只能捕获它们猎获物的一小部分，而且经常会饿上好几天。严格依赖狩猎意味着你将不断寻找下一餐。当*客户找你*时，营销等式的另一面就被发现了。
