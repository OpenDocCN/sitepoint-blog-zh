# 天作之合:如何采访客户

> 原文:[https://www.sitepoint.com/how-to-interview-a-client/](https://www.sitepoint.com/how-to-interview-a-client/)

有几种情况下你会想要采访一个客户或潜在客户。如果你问了正确的问题，当你决定与某人合作并确定项目范围时，你会处于一个更有利的位置。

采访客户的很大一部分并不总是问问题，而是能够分析客户说了什么和没说什么。这里有两种你可能想要采访客户的常见情况，以及一些你应该努力收集的信息。

**确定客户是否是你的好客户**

花时间确定一个潜在客户是否符合你的“理想客户”标准，或者你通常最适合合作的客户类型是很重要的。当谈到和谁一起工作时，有点挑剔是一件好事；这让你和你的客户都处于一个长期成功的位置上…如果你在建立关系时对另一个人有所了解，希望你们在一起工作时都能开心。

您应该能够回答一些问题，以确定客户是否适合您，这些问题包括:

*   谁是客户，他/她做什么？
*   他们需要的服务与你提供的相吻合吗？
*   他们了解你提供的服务类型吗？
*   他们了解你的价格吗(而不是质疑他们)？
*   他们在提供商身上寻找什么？
*   他们了解你的工作流程吗？
*   他们的沟通偏好是什么？
*   是否存在难以克服的障碍(例如，被几个时区分隔开)？
*   他们看起来尊重你吗？
*   他们有什么样的声誉？
*   它们的一般可用性如何？
*   他们是有幽默感还是都是公事公办？

概述项目的范围

对一个具体的项目有一个现实的想法是至关重要的，尽管这很难做到。你可以使用客户填写的关于项目要素的项目申请表，或者你可以通过电话或电子邮件非正式地收集项目规格。无论您通常采用哪种工作方式，您都需要问一些一般性的问题，以获得以下问题的答案:

*   是什么类型的项目？
*   他们的预算是多少？
*   项目的时间表是什么？
*   观众是谁？
*   项目的目标是什么？
*   预期的交付成果是什么？
*   在响应和达到里程碑方面有什么期望？
*   这个项目的预期收益是什么？
*   该项目的背景是什么(它是全新的吗，以前开始过吗)？
*   你应该知道客户对项目有什么偏好吗？
*   客户需要定期的状态检查吗？
*   项目完成后是否有必要的培训、跟进或支持？
*   谁将是您的联系人？
*   谁是决策者？
*   你还会和谁一起工作？

这些问题中有许多在你问之前就已经被回答了，但是获得这种类型的信息是很重要的，这样你就不会感到惊讶，并且能够提供准确的估计和交付日期。
