# 2012 年 4 月浏览器趋势:Chrome 击败竞争对手

> 原文:[https://www.sitepoint.com/browser-trends-april-2012/](https://www.sitepoint.com/browser-trends-april-2012/)

2012 年已经过去了四分之一，是时候重新审视一下全球浏览器市场了。上个月对微软来说很糟糕，IE 损失了近两个百分点——主要是被谷歌 Chrome。Android 也超过 Opera 成为使用最多的移动浏览器。有没有公司在四月份的[全球统计数据](https://gs.statcounter.com/#browser_version-ww-monthly-201203-201203-bar)中显得很愚蠢？…

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 9.5%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

还是老一套。Chrome 又跃升了 1%，主要是以 Internet Explorer 为代价。如果我们看个别版本，Chrome 17 是世界上使用最多的浏览器，份额为 27.2%。IE8 以 16.0%远远落后于第二名。充其量，在 Chrome 超过 IE 的总数之前，微软还有两三个月的时间，我们将在 13 年内首次加冕一个新的冠军。

对微软来说，这是一种安慰；IE9 是 2012 年 3 月增长最快的浏览器。它又获得了 2.5%的市场份额，用户群增加了 20%，成为欧洲和美国使用最多的版本。许多用户会将 XP 升级到 Vista/7，这对公司的底线有好处。

然而，IE8 下降速度很快，而且比我预期的要快。在微软决定放弃 XP 用户后，我们许多末日论者预言 IE8 会变成另一个 IE6。浏览器在一段时间内不太可能变得无关紧要，但几个月内几乎肯定会达到个位数。

IE6 在中国的使用仍然是一个问题，它保留了近 26%的市场份额。在其他地方，IE6 和 ie7 的总份额已经下降到 4.25%，不到 20 个用户中的 1 个。你应该经常检查你网站的数据，但是我们已经达到了一个点，没有必要花费过多的时间来开发和测试浏览器。IE6 和 7 对我来说已经死了，但仍然像僵尸一样不时崛起。

火狐持有 25%。[版本 11](https://www.sitepoint.com/firefox-11-whats-new/) 几周内已经达到 6%，4+线正在稳步增长。也就是说，第 3 版及以下版本正以类似的速度下降。

Safari 和 Opera 都下降了，尽管统计波动被它们相对较低的数字放大了。他们可能会在下个月收复失地。

## 移动浏览器的使用

三月份的移动使用量增加到了所有网络活动的 8.99%。

主要的移动浏览应用程序有:

1.  opera Mini/Mobile——22.86%(增长 1.16%)
2.  安卓系统——21.16%(下降 1.51%)
3.  iPhone——20.10%(下降 0.96%)
4.  诺基亚浏览器——11.86%和 11.24%(增长 0.62%)
5.  UC 浏览器——6.35%(增长 0.45%)

对黑莓来说，还有更多坏消息。它下降了 0.35%至 6.18%，并被 UC 浏览器挤出了前五名，UC 浏览器是大多数平台的独立应用程序，用户必须选择并安装。

我希望 Android 享受它在排行榜榜首的单个月，因为 Opera 再次夺回了桂冠。Opera、Android 和 iPhone 之间的三方关系凸显了市场中的良性竞争。不要误以为每个人都在用和你一样的手机！
