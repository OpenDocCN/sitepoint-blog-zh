# 伙伴关系:对你的企业是好事还是坏事？

> 原文：<https://www.sitepoint.com/partnerships-boon-or-ruin-for-your-business/>

新企业以惊人的速度倒闭。2007 年至 2010 年间，美国小企业的失败率上升了 40%。然而，根据美国小企业管理局的说法，拥有多个所有者的公司比独资公司更有可能存活更长时间。此外，2008 年的一项研究显示，自 1980 年以来，合伙企业的平均收入增长了 157 %,而同一时期，个体经营者的平均收入下降了 51%。该研究还报告称，2008 年个体经营者的平均净收入约为 12，000 美元。

事实似乎证明了合伙企业的合理性，然而 72%的小企业是独资企业。伙伴关系有其优点和缺点，乍一看，优点似乎大于缺点。然而，一段糟糕的合作关系会让你后悔当初没有进入商界。

最大的风险是共同责任。每个人在合同上和法律上都受另一方达成的协议的约束。因此，如果你的伴侣做了什么见不得人的事，把你送上法庭，你同样要承担损害赔偿责任。如果一方逃离城镇，另一方要对他可能发生的所有债务负责。假设这些都没有发生，一切顺利，你仍然要分享利润。

好处是，你还将分担创业成本、费用、责任和风险。此外，有伴侣意味着在遇到困难时有一个支持系统和动力。

所有这些都表明，商业伙伴关系，就像婚姻一样，不是可以轻易缔结的。朋友们常常仅仅基于他们是朋友的事实，以及他们会永远是朋友的假设，就加入了合作关系。虽然选择一个技能与你相反的伴侣很重要，但你们必须有共同的价值观。你们可能是最好的朋友，但是你们有共同的职业道德吗？

## 一个坏掉的干衣机如何促成成功的商业合作

2000 年末的一个决定命运的晚上，我接到了教堂实习生詹姆斯的电话。他正在收拾行李，准备第二天早上乘飞机去看望他的家人，但是他的烘干机坏了，他陷入了困境。我妻子的母性本能发挥了作用；她告诉他把要洗的衣服拿过来，她会处理好的，这样他就可以完成打包了。

在那之前，我对詹姆斯的了解是半随意的——我们都建立了一些网站——所以当他来取他要洗的衣服时，我们花了一个小时左右的时间“谈论工作”原来，詹姆斯讨厌设计网站，他这么做只是为了做他真正热爱的事情——编程。我喜欢设计，但由于缺乏编程知识，我无法创建更多的交互式网站，这让我很沮丧。没多久我就意识到合作对我们双方都有利。

我们每个人都建立了一个适度的客户群，所以通过加入，我们的客户群立即翻了一番。结成伙伴关系让我们在客户中获得了更高的可信度。更重要的是，詹姆斯单身，是一名全职自由职业者，而我有一份全职工作，所以我们突然变成了一个“合法”的企业，在正常的工作时间有人可以提供服务。

但是直到我们引进了第三个合伙人，我们的业务才开始增长。杰里米是那些“超级网络”的人之一(你知道，有 700 多个脸书朋友的人)，让他负责业务发展给我们带来了超出我们每个人单独可以实现的范围的客户和项目。

约翰·洛克菲勒曾经说过:“建立在商业基础上的友谊比建立在友谊基础上的商业要好得多。”当伙伴关系出现问题或生意失败时，许多友谊就被毁掉了。我有幸与一群人合作，他们开始是熟人，但最终成为朋友，尽管我们最终在商业上分道扬镳。
