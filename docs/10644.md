# 利用自己的时间参加 SitePoint 课程

> 原文:[https://www.sitepoint.com/taking-a-sitepoint-course/](https://www.sitepoint.com/taking-a-sitepoint-course/)

我们从客户那里收到的关于我们课程的一个常见问题是:如果你错过了课程的预定开始时间，你什么时候能上那门课程？

我们所有课程的优点在于，您不需要等待，您可以立即注册并开始学习！

以我们的第一门课 JavaScript Live 为例。我们在 4 月份对课程进行了一次现场排练。在这个直播版本中，Kevin 发布了 12 个课程，包括文章、视频和下载，并举行了 6 次现场问答。论坛也因活动而欣欣向荣。

错过了？不用担心——您可以体验 JavaScript Live(重播！)现在就参加课程。虽然 Kevin 目前没有主持现场问答环节，但您可以访问 12 节课、之前所有问答环节的录音，并访问私人 JavaScript Live 论坛，在这里您仍然可以获得所有问题的答案。

到目前为止，我们已经发布了五门课程(还会有更多的课程推出)，所以如果你认为是时候复习一些网络技能了，那就去看看吧:
