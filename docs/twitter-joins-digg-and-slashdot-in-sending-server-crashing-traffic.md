# Twitter 加入了 Digg 和 Slashdot 的行列，发送服务器崩溃流量

> 原文：<https://www.sitepoint.com/twitter-joins-digg-and-slashdot-in-sending-server-crashing-traffic/>

每个人都听说过 Digg 效应和 Slashdot 效应，在这些网站上的提及可以一下子给一个网站带来如此大的流量，以至于网站的服务器瘫痪。但是推特效应呢？显然，是的。这可能是 Twitter 成功的最佳指标之一，该网站在崩溃网站的能力方面已经加入了社交媒体精英的行列。

昨天，博客 Mashable [的热门推特用户](http://twitter.com/mashable/status/1171075041)[皮特·凯什摩尔](http://twitter.com/mashable/)在推特上写了一篇他正在阅读的[博客文章](http://www.litmanlive.co.uk/2009/01/how-to-use-twitter-to-find-your-next-job/)。此后不久，这条带有链接的推文被大量传播和转发(被追随者复制),导致博客服务器[崩溃](http://twitter.com/mashable/status/1171106154)。Cashmore 想知道这是否应该被称为“Mash 效应”，但 Pingdom 有一个更好的名字:[Twitter 效应](http://royal.pingdom.com/2009/02/03/dawn-of-the-twitter-effect/)。

目前，像这样的事情可能很少发生。毕竟，Cashmore 有超过 50，000 名粉丝(也就是说，不是典型的 Twitter 用户)，并且必须在一天中的某个时间发布推文，以最大限度地暴露该群体，并谈论真正引起共鸣的话题。然而，随着 Twitter 越来越受欢迎，转发变得越来越普遍，Twitter 效应也可能成为一个经常发生的事情。

Pingdom 为 Twitter 效果建议了以下公式:

**推特效应公式=(原创推文*关注者)+(转发推文*转发者的关注者)+(转发推文的转发推文*那些的关注者)，以此类推。**

如你所见，它呈指数增长。Twitter 具有信息快速病毒式传播的潜力。我们在过去的突发新闻事件中看到过这种情况，所以对于其他不太紧急的信息来说，这种情况发生并不奇怪。Pingdom 在他们的博客上写道:“推文可以像树枝或根系一样扩散开来，到达大量的 Twitter 用户。”“传播基本上只受到 Twitter 用户群规模的限制。如果推文包含一个网站的链接，随着推文的传播，这个网站一定会获得大量的流量。”

在 SitePoint，我们非常重视 Twitter。我们还有一个[非常受欢迎的 Twitter 账户](http://www.twitter.com/sitepointdotcom)，拥有超过 20，000 名粉丝，通过 Twitter 上的链接，我们的帖子获得了大量的流量，特别是那些被我们的粉丝大量转发的帖子(顺便说一句，我们非常感谢！).然而，到目前为止，还没有任何一篇帖子的流量能与 Digg 首页链接、Slashdot 上的一次提及或一篇严重失误的帖子的流量相提并论。然而，我们确实每天从 Twitter 的所有帖子中获得更多*稳定*的流量。

这让我们想知道几周前 Twitter 能否取代 Digg。我们的结论是，从广义上来说，它可能不会很快取代 Digg 成为大规模流量爆发的来源，但它肯定会成为一个非常有价值的工具，为每个用户展示有用和有趣的链接。我们说过，构建工具在 Twitter 上展示链接的关键之一是转发。

有许多测量转发的工具，但是最好的可能是 T2 转发者。Retweetist 做了大量工作，每天在 Twitter 上产生最多被转发的条目，但它并没有做太多工作来确保它们与我这个读者相关。几周前，我们提出，关键是加大我朋友的推文的权重——这些人可能会谈论我最感兴趣的事情。

我们真诚地希望有人能造出这样的工具。与此同时，如果一个受欢迎的 Twitter 用户在 Twitter 上发布关于你的网站的消息，你可能要准备好应对 Twitter 效应。
