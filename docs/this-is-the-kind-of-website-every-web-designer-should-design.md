# 这是每个网页设计师都应该设计的网站

> 原文：<https://www.sitepoint.com/this-is-the-kind-of-website-every-web-designer-should-design/>

我刚刚与 www.carbonite.com 签约，这是一个基于网络的备份解决方案。这一过程几乎符合我所能想到的积极的网络电子商务服务的所有标准:

–首先，我从一位受人尊敬的同事那里听说了这项服务，所以我已经倾向于购买。

–主页简单而华丽。它用一句话告诉你这项服务是做什么的，费用是多少。它还提供免费试用——甚至不需要你的信用卡。它甚至以低廉的价格提供“无限”备份。我喜欢这样的主页。

–如果您想了解更多信息，可以点击任何链接了解更多信息。他们有音频剪辑，动画教程和文本。

–整个过程非常简单，并且专注于一个引人注目的价值主张—每月只需 5 美元即可保存您的文件。

–到目前为止，我的体验非常好。它很容易启动，系统正在做它所说的事情。没有下载问题。一切都很容易。

我希望你设计的网站能像这个网站一样简单、优雅、有力地传达你的客户(或你)所提供的东西！我希望你的客户能够兑现他们网站上的承诺。

如果你能做到，作为一名网页设计师，你将是无与伦比的。你将能够帮助公司用简单的术语传达他们的产品和服务，并将潜在客户转化为买家——也许还能确保他们的产品/服务以简单而强大的方式交付。你将不仅仅是一个网页设计师，而是在市场营销、产品开发和战略中扮演重要角色的人。或者，如果你专注于利用公司拥有的东西，通过一个不可抗拒的网站帮助他们将访问者转化为买家，你将远远超过大多数人！

(注意:我在这家公司没有任何股份。以上审查是基于无利益冲突或推广本服务的愿望)。

如果你有一个不属于你自己的分享网站，在那里没有利益冲突，你知道访客转化为买家，请分享你最喜欢的，为什么你认为它在创造销售方面表现出色。但是不要用这个来宣传你自己或者你的朋友/家人/同事。
