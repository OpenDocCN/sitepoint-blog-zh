# 设计中的视觉和谐第三部分:主题参考

> 原文:[https://www . site point . com/visual-harmony-in-design-part-3-thematic-reference/](https://www.sitepoint.com/visual-harmony-in-design-part-3-thematic-reference/)

上周我们看了设计师如何在他们的设计中使用[重复来创造视觉和谐](https://www.sitepoint.com/harmony-in-design-part-2-repetition/)。本周我们将注意力转向“主位参照”的概念。主题参考是一个有点浮夸的术语，用来描述如何在设计中建立主题之间的联系。例如，如果你正在为新娘设计一个网站，你很可能会避免暗褐色，头骨和交叉骨的图像以及粗糙扭曲的字体。这不是一个适合婚礼的主题。实现视觉和谐的一个方法是选择一个主题，并使用反映该主题的设计元素——图像、字体、纹理、颜色。

让我们用水果盒网站来举一个简单的例子。这个网站的功能是让潜在客户了解其水果配送服务。首先，这个标志以一盒颜色鲜艳的水果为特色。主页上的主要图片是一些颜色鲜艳、多汁水果的照片。导航栏中使用的颜色和页面顶部的联系信息是鲜艳的橙色、红色和绿色，让人想起多汁的水果。所以实际内容和设计是互相强化的。这看起来很简单，不是吗？但是让一个网站看起来简单和吸引眼球往往是一项艰巨的任务。

[![FruitBox](../Images/a0d1203243d32ebbc2f082942c087e15.png "FruitBox")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/FruitBox.jpg)

罗奇代尔苹果酒的包装是由[供应商](http://www.supply.net.nz/)设计的。简报要求品牌重塑应该符合历史和传统的主题。复古模板风格的印刷补充了标签和盒子上使用的副本，以及瓶子前面的旧风格苹果品牌标志。看起来自然的发黄的纸增加了怀旧的整体主题，同时看起来仍然很现代。

Opera Live 网站上有一个精彩的互动预告片。艺术品风格传达了一个古老的主题。美丽的水墨画和伴随的戏曲音乐相互呼应。字幕中使用的斜体文字增强了歌剧的优雅感。所有的元素以一种和谐的方式结合在一起。
