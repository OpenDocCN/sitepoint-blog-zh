# 一旦 Chrome 上出现扩展，Firefox 应该会担心

> 原文：<https://www.sitepoint.com/once-extensions-arrive-on-chrome-firefox-should-worry/>

![](img/83bb28eb2d7db8e14fdb8af601970e41.png "google-chrome")昨天， [CNET 挖出了](http://news.cnet.com/8301-17939_109-10110247-2.html)一份来自谷歌的文件，该文件概述了该公司在 Chrome 网络浏览器扩展方面的计划。虽然[文件](http://dev.chromium.org/developers/design-documents/extensions)没有提供任何关于扩展何时到来的具体时间表，但它确实概述了一些用例以及谷歌愿意支持的具体扩展。

> *   Bookmarks/navigation tools: Delicious Toolbar, Stumbleupon, web-based history, new tab clipboard accelerator
> *   Enhancements: Skype extension (clickable phone number), RealPlayer extension (saving video), Autolink (general microformat data-address, phone number, etc.). )
> *   Content filtering: Adblock, Flashblock, privacy control, parental control
> *   Download Assistant: Video Assistant, Download Accelerator, Downtown, FlashGot
> *   功能:ForecastFox、FoxyTunes、Web Of Trust、GooglePreview、BugMeNot

谷歌表示，它意识到支持 Chrome 的开源项目 Chromium“不能满足所有人的所有需求”，这就是扩展很重要的原因。谷歌的目标是让 Chrome 拥有一个扩展系统，与 Mozilla 和 Firefox 实现的系统相媲美。“我们应该从建立一个能够支持不同类型可扩展性的扩展系统的基础设施开始，”该文件说。随着时间的推移，系统应该能够支持开放式 API 列表，如工具栏、侧栏、内容脚本(用于类似 Greasemonkey 的功能)和内容过滤(用于家长过滤器、恶意软件过滤器或类似 adblock 的功能)

尽管 Chrome 浏览器的市场份额在过去的一个月里由于 Google.com 上偶尔出现的主页链接而上升到了大约 1%,但是据网络应用报道，许多早期用户仍然犹豫要不要从 Firefox 切换过来。我们最常听到的原因确实是 Chrome 缺乏扩展，正如 CNET[所指出的，扩展系统](http://code.google.com/p/chromium/issues/detail?id=18&sort=-stars&colspec=ID%20Stars%20Pri%20Area%20Type%20Status%20Summary%20Modified%20Owner)是 Chrome 项目的问题跟踪系统中最受欢迎的功能。

看看我们之前在 SitePoint 上关于 Chrome 的报道中的一些评论:

> “我喜欢 Chrome……它速度快，打开也快。然而…我在火狐上发表了这个评论。对我来说，这很简单…没有 Foxmarks，网站开发工具栏和其他几个插件。”— [格雷格](https://www.sitepoint.com/google-chrome-three-months-out/#comment-839818)

> “我今天花了一点时间在 Chrome 上，感觉很不错。非常快速和超级干净的用户界面。它显示了一些网站有多慢。我同意[…]关于 web dev 工具栏的说法。每次启动浏览器时，我都会使用它和 Firebug。”— [阿瓦森](https://www.sitepoint.com/google-chrome-three-months-out/#comment-840050)

> “由于缺乏对我每天都在使用的插件的支持，我仍然不打算转换。我的意思是，今天早上我打开 chrome，想给某人发送一个截图，但是我的截图在哪里…哦，对了，这是铬合金:-(。如果他们设法在 Chrome 中支持 FF 插件，那么是的，他们完全可以摆脱 FF，然后开始转化 IE 用户。”— [驼鹿](https://www.sitepoint.com/early-stats-google-chrome-hurts-firefox-not-ie/#comment-792658)

对于谷歌来说，将主流用户从 ie 浏览器上拉走将是一个艰难而缓慢的过程，就像 Mozilla 一样。最先瞄准的用户是早期用户——或者在这种情况下，是已经在使用 Firefox 和 Opera 等替代浏览器的人。这些人一般依赖浏览器扩展(如 Firebug 和 AdBlock)，Chrome 将需要与之竞争。

一旦谷歌将计划中的扩展框架添加到 Chromium 项目中，我认为我们可以期待看到它的市场份额大幅上升。它不会在一夜之间超越 Firefox，但它会成为一个相当重要的次要玩家——我的意思是:小心 Firefox、Opera 和 Safari。
