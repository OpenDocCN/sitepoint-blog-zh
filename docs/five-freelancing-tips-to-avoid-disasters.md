# 避免灾难的五个自由职业技巧

> 原文：<https://www.sitepoint.com/five-freelancing-tips-to-avoid-disasters/>

我最近听说了一个悲惨的故事，一个自由职业者似乎在一夜之间失去了她的生意，因为她已经病了一个多月了。被诊断患有相当严重的健康疾病已经够难的了，更别说在一个多月内你 95%的业务都消失了。

作为一名自由职业者，这里有五个建议可以帮助你度过灾难。

有储蓄把你的一部分利润放在一个单独的银行账户里，以防你突然无法工作。理想情况下，这足以支付你至少一个月的所有费用。

保持弹性
保持一个灵活的时间表，让你在有灵感的时候突然完成工作，并且有足够的时间休息几天来恢复，而不会让大多数客户注意到。

拥有收入选择
永远保持你的选择——密切关注市场动向，这样如果其他方法都失败了，你至少可以很快找到一份工作。永远不要与客户或竞争对手断绝关系——有一天，他们可能会成为你未来的雇主。

有保险
健康保险是显而易见的一种，然而许多国家也有失业或收入损失保险。如果你需要长时间停止工作，保险费可能是一个很小的代价。

拥有积极的态度
你的健康受态度的影响比你想象的要大。健康的生活并不意味着每周工作 60 个小时，并持续承受压力。抽出重要的时间享受生活，并努力保持积极的态度。
