# 制造轰动:普通人的草根病毒式营销

> 原文:[https://www.sitepoint.com/viral-marketing-for-everyone/](https://www.sitepoint.com/viral-marketing-for-everyone/)

如果你活跃在社交媒体上，尤其是推特、脸书和 YouTube，那么你已经看到了病毒式营销的影响。

根据维基百科，病毒式营销是:

*“一种营销技术，利用已有的社交网络，通过自我复制的病毒式过程来提高品牌知名度或实现其他营销目标(如产品销售)……它可以是口头传播，也可以通过互联网的网络效应来增强。”*

虽然它可能非常强大，但创建一个病毒式营销活动并不容易，除非你在 Twitter 上有数万名粉丝，在脸书有数百名朋友，在 YouTube 上有大量订户，以及一个庞大的回复邮件列表。哦，还有一个让其他人振奋起来并兴奋地传播它的信息。

那么，有兴趣为自己的产品和服务制造轰动效应的普通人是如何让某些东西像病毒一样传播的呢？

**是什么让某样东西成为病毒**

首先，让我们来看看是什么让某样东西成为病毒。如果你把它归结为基本要素，你可以用这四个要素来定义病毒营销的本质:

1.  能够激发人们对你的产品和服务的兴趣和热情
2.  获得主要影响者的支持
3.  专注于一致的、传播良好的营销
4.  在正确的时间出现在正确的地点

现在，你如何编排这些元素来创建一个有效的病毒式营销活动？

**产生兴趣和兴奋**

首先考虑的是你的产品或服务本身。它有能力让人们兴奋起来，并诱使人们采取行动吗？人们会对在某种程度上对他们有益的事情感到兴奋。如果你的信息不能给人们提供他们需要的东西，他们想要的东西，或者一个支持他们信仰的机会，你可能需要重新考虑一下病毒式广告。

**获得主要影响者的支持**

主要影响者基本上是那些其他人尊重和倾听的人。他们通常有正面的声誉，有很深的影响力，有能力毫不费力地制造轰动效应。如果你的人际关系网中没有重要的影响者，你可以通过一些方法获得他们的支持:

*   询问他们——就像向潜在客户推销你的服务一样，列出你的请求的细节，他们如何能提供帮助，以及他们为什么想要这样做。你甚至可以提供一些东西来回报他们的帮助。
*   遵循路径——遵循六度分离背后的意识形态，有一条路径可以通过你认识的人找到一个有影响力的人。虽然它并不总是有效的，但你可以定位你的活动，让它通过每个必要的点到达主要影响者。

**专注于一致而广泛的营销** 
仅仅坚持一个渠道并抱最好的希望是不够的。为了使你的活动有效，你需要尽可能地传播你的信息。让现有客户参与进来，向同事寻求支持，利用你的电子邮件联系方式，利用社交媒体，写博客，大声说出来。虽然你应该保持你的营销活动的一致性，但是要意识到你宣传你的产品/服务的次数和频率。如果有太多的人在太多的地方看到你的信息，你很容易扼杀你的广告。

**在正确的时间出现在正确的地点**

病毒式营销的真相是，很多时候它归结为在正确的时间出现在正确的地方。虽然你不能完全控制这一点，但如果你坚持做其他事情，并准备在机会出现时抓住它，可能性会很大。
