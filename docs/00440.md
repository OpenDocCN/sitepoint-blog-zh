### 2.给你的作品集网站一个独特和前沿的外观

你最不想做的事情就是把你的网站设计成一个无聊的公司网站。这当然会更容易建立，但它不会帮助你赢得任何新客户。

所以，想想是什么让你成为一个独特的网络开发者。你的网站作为一个整体应该反映这一点。这是你将潜在客户从 A 点(你的主页)带到 B 点(你的投资组合)的唯一途径。

例如，Adham Dannaway 是一名设计师/开发人员:

[https://gfycat.com/ifr/OrdinaryShadowyBarasinga](https://gfycat.com/ifr/OrdinaryShadowyBarasinga)

因此，Dannaway 的网站是这种分裂人格的真实体现。当潜在客户将图像向左或向右拉时，他们暴露了两个不同的方面:设计者和编码者。

这足以让潜在客户思考，“嗯……我还能在这里找到什么？”但是没有花里胡哨的。投资组合网站的其余部分设计得很简单，没有惊喜。这使得他们很容易接触到投资组合样本。

### 3.将您的 web 开发人员作品集放在一个专门的页面上

客户雇用你不仅仅是因为你的工作令人印象深刻。他们雇用你是因为他们想和像你这样的人一起工作。

所以，在建立你的作品集网站时，给你的作品一个专用的空间，让潜在的客户在网站的其他地方多了解你一点。

客户不仅想看到你在网站建设方面做得有多好。他们还想知道客户对工作、过程和结果是否满意。

### 5.让潜在客户很容易采取下一步行动

为自己创建完美的 web 开发人员组合所需的最后一个要素是一个显而易见的 CTA。

那是什么意思？

首先，你的导航需要简单。不要用过多的无关信息。你已经通过网站的设计和作品的质量推销了自己。保持简单，并将其作为潜在客户应该采取的自然步骤。

Jonny MacEachern 有一个很好的例子:

[![](../Images/681699efd420b3e63595370030677d85.png)T2】](https://jonny.me/)

有三个步骤:

1.  了解我的服务。
2.  看看我的作品集。
3.  获取定价。

应该就这么简单。

所有的 CTA 也需要清晰一致，这意味着决定您希望他们采取的行动。在 MacEachern 的案例中，潜在客户有两种选择。

如果他们感兴趣，可以请求报价:

[![](../Images/d888dc62cffa35e973377aecab265495.png)T2】](https://jonny.me/)

或者，他们可以通过联系表单或实时聊天小工具联系:

[![](../Images/4d2046e6fd9536693815b5e9e2a2fdf9.png)T2】](https://jonny.me/)

如果你已经花了很多时间来建立一个网站，清楚地展示你的技能，并赢得了潜在客户的信任，不要因为过于复杂的行动号召而放弃一切。
