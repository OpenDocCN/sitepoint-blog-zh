# 浏览器安全的尖端技术

> 原文：<https://www.sitepoint.com/the-cutting-edge-of-browser-security/>

[https://channel9.msdn.com/Events/WebPlatformSummit/edgesummit2016/ES1604/player](https://channel9.msdn.com/Events/WebPlatformSummit/edgesummit2016/ES1604/player)

本文是微软网站开发系列的一部分。感谢您对使 SitePoint 成为可能的合作伙伴的支持。

浏览器漏洞是一件大事。浏览器呈现出庞大的目标人群和广泛的攻击面，而且近年来攻击手段前所未有的成熟和产品化。在这次演讲中，我们将介绍我们正在做的事情，通过 MemGC、模块代码完整性和 AppContainer 隔离等改进，使 Edge 比我们发布的任何浏览器都更安全。

这个视频来自微软 Edge Web Summit，这是一个由构建微软 Edge 和 Chakra 的工程师组织和管理的免费会议。您可以找到一整天的技术讲座，涵盖 EdgeHTML 呈现引擎、开源的 Chakra JavaScript 引擎和开发人员工具。您可以直接从构建 Windows 10 的工程师那里听到支持 Windows 10 的 web 平台的下一步是什么，并且您可以深入了解强大的技术和新工具，让您作为开发人员的生活稍微轻松一点。

我们鼓励你跨浏览器和设备进行测试，包括微软 Edge——Windows 10 的默认浏览器——[用免费工具](https://developer.microsoft.com/en-us/microsoft-edge/?wt.mc_id=DX_838508)；包括 [F12 开发者工具](https://developer.microsoft.com/en-us/microsoft-edge/platform/documentation/f12-devtools-guide/?wt.mc_id=DX_838508)——七个不同的、完整记录的工具，帮助你调试、测试和加速你的网页。此外，[访问 Edge 博客](https://blogs.windows.com/msedgedev/?wt.mc_id=DX_838508),从微软开发人员和专家那里获得最新信息。
