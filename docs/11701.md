# 五个有用的博客编辑器

> 原文:[https://www.sitepoint.com/five-useful-blog-editors/](https://www.sitepoint.com/five-useful-blog-editors/)

说到更新博客，许多用户会直接进入他们博客的管理部分，然后在那里发帖。但是，从桌面或浏览器窗口发帖也有一些好处。在桌面上写作(理论上)可以最大限度地减少干扰，提高工作效率，而一些在线编辑器有一些有用的功能，你可能在 WordPress 或 Blogger 的默认编辑器中找不到。

下面列出的所有博客编辑器都需要(通常)简单的初始设置，这样它就知道要发布到哪里，但之后只需要输入你的帖子，按下发布，然后就可以了。

**桌面应用**

**[Windows Live Writer](http://download.live.com/writer)**是微软的一款免费博客编辑器，它看起来和感觉上都非常像 Windows 应用程序。不，等等，回来，很好！

![LiveWriter](../Images/a861bd034db0089b6d0ce6ac172280ba.png "LiveWriter")

它可以让你发布到几乎任何博客平台和多个博客。你可以在发帖前预览你的帖子，看看它到底是什么样子。它可以很好地处理照片，允许你添加阴影和边框等效果。您还可以轻松添加表格、视频和地图。我几乎所有的帖子都使用 Live Writer，我觉得它非常棒。

Ecto 可以被认为是 Mac 版的 Windows Live Writer。售价 19.95 美元。
T3】

Ecto 拥有许多与 Live Writer 相同的功能，你可以发布给所有常见的嫌疑人，如 [WordPress](http://wordpress.org/) 、 [Blogger](http://www.blogger.com) 、 [TypePad](http://www.typepad.com) 、 [Drupal](http://www.drupal.org/) 、 [MovableType](http://www.movabletype.org) 、 [SquareSpace](http://www.squarespace.com) 等等。它还支持 iPhoto 和 iTunes，如果你是亚马逊的会员，还支持嵌入亚马逊联盟计划链接，让你可以轻松链接到亚马逊的促销活动和产品。

Thingamablog 是一个开源的、跨平台的博客编辑器和 RSS 阅读器。

![Thingamablog](../Images/f9ae3856918b4910ab74f0dc695895b3.png "Thingamablog")

这是一个桌面应用程序，可以让你处理多个博客和离线写作(如果你想的话)。您可以在可视或 HTML 模式下编辑文章，并在文章发布前获得预览。图像可以从你的电脑或网址上传，一个很好的功能是能够使用订阅源创建帖子。

 ****基于浏览器的应用&插件**

[ScribeFire](http://www.scribefire.com/) 是一款免费的火狐插件和跨平台博客编辑器。

![ScribeFire](../Images/8338fbbb4208049228b83af2d06b2052.png "ScribeFire")

安装插件后，您可以使用编辑器撰写和发布到多个博客。当您在浏览器中打开 ScribeFire 时，编辑器会占用大约一半的屏幕空间，并且有一个在新标签页或单独窗口中打开编辑器的选项。很轻，很好用。您可以使用 FTP 帐户上传图像，使用富文本格式，HTML 编辑和实时预览。您也可以将未发布的草稿存储为笔记。

[Flock](http://www.flock.com/) 是一款免费的社交媒体浏览器和博客编辑器。

![flock](../Images/0100927894055380c6eddaef0492596f.png "flock")

如果你觉得你可以使用另一个浏览器，你可能会喜欢 Flock 内置的博客工具。它让你可以在主要的博客平台上写博客。您可以在可视或 HTML 模式下添加和编辑博客文章，指定标签，并在发布前预览整篇文章。您可以轻松地从 web 上剪切和复制内容，并使用 Web 剪贴板发布内容。就像 thingameblog 一样，内置了一个方便的 RSS 提要阅读器。

这就是你想要的，五个简单的旧博客编辑器的替代品，你可能想试试。
