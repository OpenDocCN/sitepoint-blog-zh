# 10 个 jQuery HTML 插件

> 原文:[https://www.sitepoint.com/10-jquery-another-html-plugins/](https://www.sitepoint.com/10-jquery-another-html-plugins/)

这里是另一个 10 个 jQuery HTML 插件的列表，可以让你的基于 HTML 的网站更加有趣；-).去看看吧，看看还有什么是你从未遇到过的。玩得开心！

## 1.jQCandy 浮动头脚插件

这个插件对于包含不适合浏览器视窗的大表格的 HTML 页面很有用。我们的目标是在滚动表格时将它固定在浏览器的顶部/底部，就像 Excel 做的那样。
![jQCandy-floating-head-foot-plugin.jpg](../Images/56a5f05810221d4945783d8f5476f95c.png)
[来源](http://code.google.com/p/jqcandy/wiki/FloatingHeadFoot)

## 2.jq-log

jQuery logger 可以记录任何递归到 html 元素或控制台的对象。
![jq-log.jpg](../Images/714708746dd547f75e0abb37799b3408.png)
[来源](https://github.com/ApmeM/jquery-logger)

## 3.“Sticklr”——粘性侧面板 CSS3 + jQuery 插件

jQuery 插件轻松创建粘性侧面板菜单。
功能:
>固定位置
>左侧/右侧
>自动居中
>无 JS 仍有效(CSS 回退)
>悬停/点击/Dblclick
>多列
>灵活样式
>跨浏览器兼容
>简单&轻量级
![Sticklr.jpg](../Images/a0cb368ef2e118aeea6c477a64facbd1.png)
[来源](http://codecanyon.net/item/sticklr-sticky-side-panel-css3-jquery-plugin/302039?ref=109&clickthrough_id=&redirect_back=true)

## 4.等离子 jQuery 插件

是一个简单的插件，可以帮助你在几分钟内创建漂亮的幻灯片。使用这个插件，你可以创建许多幻灯片，从简单的垂直/水平折叠幻灯片到可选的导航、标题等等，最后是幻灯片叠放在前面的幻灯片上。
![Plasma-jQuery-Plugin.jpg](../Images/70947bb1848c050505ddc479960f4314.png)
[来源](http://codecanyon.net/item/plasma-jquery-plugin/242160?ref=jtop)

## 5.kajabity wiki 文字外挂程式

将维基风格的文本转换成 HTML。
这个插件提供了以下特性:
> $。wikiText(plaintext)–一个 jQuery 实用函数，用于将 WIKI 格式的文本转换为 HTML。
>$ . fn . wikitext(plaintext)–一个 jQuery 包装函数，用于将 WIKI 格式的文本添加到 DOM 对象中。
![Kajabity-WIKI-Text-Plugin.jpg](../Images/f893eaca1328db9bae33a27b89007b17.png)
[来源](http://www.kajabity.com/jquery-wikitext/)

## 6.样式列表

用于灵活列表样式的脚本(ol，ul)。列表被转换成表格。
![StyleList.jpg](../Images/74309d00cc2641ff474e402e72e2e7b9.png)

## 7.jQuery 渲染插件

是通过模板和 JavaScript 对象呈现 html。在 MIT 或 GPL 版本 2 许可下的双重许可。
![jQuery-Render-Plugin.jpg](../Images/fd82e05d42bc409febd1317ca334f62b.png)

## 8.LightHTML

这个插件是一个基于 Steven Levithan 的 replaceHtml 函数的简单端口，旨在加速原生 innerHTML Javascript 赋值属性，并作为 jQuery 的 Html()函数的替代。
![LightHTML.jpg](../Images/ca9af821025818583215ae8741ecc282.png)
[来源](http://blog.stevenlevithan.com/archives/faster-than-innerhtml)

## 9.screw(scroll+view)-一个 jQuery 插件

是一个 jQuery 插件，当用户滚动网页时加载 HTML。Screw 将帮助您节省带宽，只加载滚动到视图中的内容。
![Screw.jpg](../Images/ba5fbf60e0981d86b00cdc6e1d185b70.png)

## 10.HDI ms–HTML 位置和尺寸

有助于网站的开发者和设计者有效地解决 html 组件的正确定位和尺寸问题。
![HDIMS-HTML-positions-dimensions.jpg](../Images/b7240dd4313cb31262849800131b4da2.png)
[来源](http://cyberbaza.com/hdims)
