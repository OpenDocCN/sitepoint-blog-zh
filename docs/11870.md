# 粗略的设计风格依然强劲:10 个例子

> 原文:[https://www . site point . com/sketchy-design-style-still-going-strong-10-examples/](https://www.sitepoint.com/sketchy-design-style-still-going-strong-10-examples/)

这可能是对 Web 2.0 光滑图标和倒影的反应，它开启了粗略的手绘风格网页设计的大趋势。做得好的话，这种风格在网站上创造了一种触觉的、艺术的和有点自制的感觉。

在许多情况下，设计元素被画在纸上并扫描输入，在这个过程中保持一定的粗糙度和纹理。铅笔、记号笔、钢笔和墨水提供了一种独特的艺术风格。

我认为这种风格在设计师全身心地投入时效果最好。我的意思是，设计师使用了许多手绘元素:线条、涂鸦、粗略的标题字体，而不是孤立地使用一个元素。这种风格当然会给人一种更加个人化的感觉，这种感觉在许多大型企业网站上是找不到的。

所以这里收集了 10 个很好的利用了手绘风格的网站。许多，但不是所有的都是投资组合网站。

[兄弟竞争](http://www.siblingrivalrywine.ca/index.html)在纹理纸背景上，使用带有轻微复古外观的图片、手写字体和网站周围的小草图。

![sibling-rivalry](../Images/3e6c6c5dd56f49cea3f9b5ba3b0c400e.png "sibling-rivalry")

道豪斯设计工作室是涂鸦、纹理纸和标题的混合体，使用粗略的字体。

![Dawghouse](../Images/45b12b6c55f1acae7f058afddd8c5e64.png "Dawghouse")

A 先生 t 恤是一个基于 flash 的网站，给人一种在抄写本上用钢笔画画的感觉。

![MrA](../Images/e595ec485d8a35549bb1a68ef5949040.png "MrA")

据这位老兄自己说，[杰夫·布里吉斯的网站](http://www.jeffbridges.com/main.html)是极简主义的，看起来几乎全部是手写的。

![jeffbridges](../Images/13e904ae608174a00778a1d2a82d4663.png "jeffbridges")

Grady McFerrin 的插图风格在他网站的排版上也有所体现。

![Grady](../Images/0e6ee2c7df93a6800b0344e9d3b1c6ed.png "Grady")

弗兰克公关公司看起来就像一张写满信息的大纸。每个部分都是可点击的。

![frankPR](../Images/f3c1686d3ee30019faa214a9d2730372.png "frankPR")

Web Designer Wall 被公认为是一个为设计师提供大量信息的网站，同时也是一个素描本风格的网站。

![web-designer-wall](../Images/8549c6a1923ff15f11803eec23442269.png "web-designer-wall")

纽约的网页设计师泰勒·高(Tyler Gaw)有一个彩色的网站，上面有大量手绘字体。

![TylerGaw](../Images/a79548592c8df0e0a17496c64637d5e4.png "TylerGaw")

[红鼻子日](http://www.rednoseday.com/)结合了手写文字、手绘线条、方框和图标。

![red-nose-day](../Images/1b978f1dd6e2398ae1bb8884c6d8bc7d.png "red-nose-day")

[Legwork Studio 的](http://www.legworkstudio.com/)网站是黑白的，到处都是涂鸦。

![legwork-studio](../Images/19d5c22b3dfa3c2b5c868525561adc8d.png "legwork-studio")

就这样结束了。这种风格显然不适合所有类型的网站，但它带来了有形的、个人的感觉。

你觉得这种风格怎么样？你认为是时候向前看了，还是你想看到它留在这里？
