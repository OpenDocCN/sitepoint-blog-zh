# 谁让你负责？

> 原文：<https://www.sitepoint.com/who-holds-you-accountable/>

每个人都需要对自己的行为负责。当涉及到公共责任和我们在日常生活中所做的事情时，我们有法律、道德标准、我们自己的道德和价值观以及社会的期望来作为责任检查。

此外，我们通常有家人，同龄人，有时甚至是陌生人，他们让我们站起来，为我们的行为负责。这在商业上没有太大的不同。商业法律、道德规范、公司政策以及客户和同事的期望。

当你在公众的视线中时，你只需要对你做了什么或没做什么负责，不管“公众”有多大或多小。

## 有个人责任吗？

当你为自己的工作和生活制定目标和计划时，责任感是关键。为了看到积极的变化，你必须对自己做出的承诺负责。

问题是，我们自己的个人责任经常排在我们的其他事情之后。我们对我们的客户、我们所属的团体、我们参与的志愿者活动、我们的家庭承诺，以及我们生活中其他人指望我们的任何方面都不负责任。这样一来，我们就没有太多时间来进行个人责任检查了。

## 呼叫增援

事实是，即使看起来不可能抽出足够的时间专注于自己的个人承诺，但如果你想继续成长和发展，这是必要的。我们中的一些人比其他人更善于寻找时间来承担个人责任，但几乎所有人都可以从寻求增援中受益。

增加你对自己负责并给予你自己的个人目标所需要的关注的机会的最好方法之一是让别人参与到这个过程中来。

通过合作，你自动公开你的个人责任。现在，你自己的目标和努力在优先级别上比其他人更接近，因为你有一个期待你行动的责任伙伴。当你有一个会叫你去做的人时，让你的责任溜走是不容易的。

## 责任伙伴选项

你最好的责任伙伴是愿意承诺支持你成长和发展的人。这个人应该了解你目前在哪里，你的目标是哪里，你有什么资源可以帮助你到达那里，以及你打算什么时候达到目标。

一个好的责任伙伴是诚实、专注、尊重和专注的人。你的伴侣可以是教练或导师、同事、合作伙伴，甚至是家庭成员。确保你尊重那些让你负责的人，这样你就可以接受并采纳他们的建设性批评。

为了互惠互利，成为他/她的责任伙伴。当你们都有很多收获的时候，对你们两人来说都更容易投入到这个过程中，并坚持到底。

## 你的搭档是谁？

你有一个负责任的伙伴来让你坚持下去吗？你们是怎么一起工作的？
