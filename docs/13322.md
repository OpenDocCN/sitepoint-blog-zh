# Photoshop CS3 Extended 中的真实 3D

> 原文:[https://www . site point . com/real-3d-in-Photoshop-CS3-extended/](https://www.sitepoint.com/real-3d-in-photoshop-cs3-extended/)

###### *重新发布自上周五的设计视图*

**虽然它还没有得到太多的关注，但 Photoshop CS3 Extended 中最有雄心和*潜在的*最有用的新功能可能是与 3D 文件交互的能力(*注意:不包括在标准的 Photoshop CS3 版本*)。**

我们不要低估他们在这里要解决的问题。教 2D 图像程序用 3D 方式思考是一项雄心勃勃的事业。Photoshop 已经是一个庞然大物，如果你曾经浏览过像 3D Studio 这样的应用程序的无休止的层叠菜单，你可能会在将它硬塞进 Photoshop 的前景中控制不住地流汗。我在想类似于把泰姬陵塞到 T2 希思罗机场的事情。

明智的是，至少对于这些进入 3D 世界的第一步，Photoshop 团队保持了相对适度的雄心。

首先，他们没有尝试在 Photoshop 中加入任何 3D 建模工具。所有模型都必须从第三方建模程序中构建和导入，包括 Google 的 SketchUp(。kmz)、Autodesk 的 3D Studio (.3ds)、Alias、Maya 和 Acrobat 3D。幸运的是，Maya 和 T2 SketchUp 目前都有免费版本，可以让你在不卖腿的情况下体验一下。

如果你不认为自己是一个建模者，或者没有时间，网上有大量免费的可下载模型。谷歌地球的仓库可能是一个很好的起点，但是有成千上万的 3D 图书馆以许多兼容格式提供高质量的兼容模型
。我从 SketchUp 仓库拿了一个 [iPhone
模型，如下图所示。](http://sketchup.google.com/3dwarehouse/details?mid=1163b38b4ed07c1a1face3cc16e3b67d)

它是如何工作的？

![iPhone model courtesy of  Google's Sketchup warehouse](../Images/0bf1e2bbfbba118627ff0935f5244767.png)为了合并 3D 对象，Photoshop CS3 Extended 添加了一种新的图层，不出所料地命名为“ *3D 图层*”。您可以通过主图层菜单(即 **`Layer/3D
Layers/New Layer from 3D file...`** )向任何文档添加 3D 图层。一个对话框将要求您选择您的模型，稍后它将出现在您的文档中。

导入时，没有明显的方式与对象交互，大多数菜单项都是灰色的。然而，在*图层*面板中右键点击你的 3D 对象会给你一个“ **`Transform 3D Model`** ”选项，这反过来给你提供了模型定位控件(如下所示)。

如果你对 3D 有一点点经验，你可能会认识这里显示的旋转、缩放和移动控件，它们允许你在虚拟空间中定位你的模型。相对于您的型号，还有一组类似的控件用于定位您的相机。

创造性控制的另一个主要领域是应用于模型表面的纹理和图像。与模型一起导入的所有纹理在主模型层下显示为链接的“子层”。

双击这些纹理层中的任何一个都会自动在 Photoshop 中将它们打开为单独的、可编辑的文档。您对这些 2D 纹理图像所做的任何更改都会立即反映在您的 3D 模型视图中。当然，当我们谈论“表面纹理”时，我们实际上是在谈论任何你可能想要应用到 3D 表面的*图像——从有机纹理到书籍封面(如下例所示)，到汽车细节，到赞助商标志等等。*

奇怪的是，据我所知，Photoshop 中没有办法从头开始将纹理应用到空白模型上——你需要导入带有某种纹理的模型，然后你可以在 Photoshop 中自由编辑或完全替换这些图像。当然，只要你理解这个事实，并在你的模型中应用默认纹理，这应该不会特别影响你所能完成的。

![3D Example](../Images/8a1b80d3d219475a6ffa0bf601ff216e.png)另一个有趣的功能是 3D 横截面视图工具。该工具允许您将一个不可见的 2D 平面穿过您的模型，显示其内部。在这个阶段，我认为这个功能更倾向于“哇”而不是有用的，但是摆弄它肯定很有趣。

这并不是说这种新的 3D 功能没有任何弱点。照明控制目前仅限于一系列大约十几个预设。当然，Photoshop 有一些很棒的 2D 照明和色阶工具，当你选择 **`Rasterize 3D`** 菜单选项时，它们就都可用了，有效地将你的 3D 模型压缩成一个普通的 2D 图层。

然而，如果你随后决定你的模型需要重新定向*向左*一点点，你可能就要倒霉了。如果不回溯历史并撤销您所做的任何其他更改，您就无法重新膨胀您的模型。

然而，即使考虑到怪癖、“第一代”粗糙的边缘和偶尔的错误，也不需要太多的想象力就能看出这个新特性有多有用。“3d 图层对象”为任何 Photoshop 用户提供了一种全新的方法来处理他们每天制作的商品照片、3D 徽标、产品模拟和 3D 插图。

如果你有 Photoshop CS3 Extended，找点时间，抓几个模特，玩一玩。
