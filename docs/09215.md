# Sprouter Resprouts！

> 原文：<https://www.sitepoint.com/sprouter-resprouts/>

早在 473 期《论坛报》上，我写了一篇关于 [Sprouter](http://www.sprouter.com/) 的文章，在当时，这是一项类似于 Twitter 的服务，只不过专注于初创公司和企业家。从那以后，它们演变成了一种类似于 Quora 的问答服务，为企业家提供活动和每周电子邮件简讯。所有这一切都紧紧围绕着商业和创业，所以这更像是一个转变，而不是一次彻底的改革。

然后，这一切都渐渐停止了。今年 7 月，我收到一封电子邮件，邮件中他们宣布要关闭；"不幸的是，由于资金限制，我们已经别无选择."这的确是一个悲伤的故事——然而，世界上到处都是跌跌撞撞并最终半途而废的初创公司。

然后，在他们的预期截止日期几周后，我收到了另一封电子邮件，其中写道:“律师已经允许我们告诉你正在发生一些事情，我们很快就能告诉你……显然，我们现在不能告诉你。”

听起来他们的帆上有了新的风？是啊！他们本周刚刚宣布，一家大型加拿大传统报纸公司收购了他们。不仅如此，他们现在正在招聘。

你可能想知道这一切与你有什么关系，特别是如果你从未使用过 Sprouter，然而这给了所有那些可能正在黑暗隧道中寻找的网络初创公司一个希望的灯塔。我认为至少可以从这里学到三条重要的经验:

**不要觉得害怕进化**。太多的商业人士以失败告终，因为他们坚信坚持自己的想法会有回报。如果你的生意没有任何进展，无论你多么努力，也许是时候转移你的注意力了。

让你的社区了解最新情况。这些年来，我分享了许多由于公司的沉默而导致负面宣传增加的故事。不要等到那个时候——让你的客户和社区了解情况。对一些人来说，分享可能令人畏惧，但是你的社区会因此尊重你。

如果其他方法都失败了，那就找一个买家或投资者。如果你完全致力于一个想法，并且你真的有 100%的信心它会成功，寻求经济上的帮助以避免关门。Sprouter 当然有一个很棒的想法——让我们希望这次资本注入能把他们从看起来肯定会失败的事情中拯救出来。

祝你好运，也祝莎拉和所有 Sprouter 团队干得好。
