# 网络之美？

> 原文:[https://www.sitepoint.com/beauty-of-the-web/](https://www.sitepoint.com/beauty-of-the-web/)

网络之美。这是微软用来描述最近发布的 Internet Explorer 9 测试版的术语。微软在说什么？它真的认为 IE9 会让网页变得更漂亮吗？它的专用网站 beautyoftheweb.com 促使我认为这是值得探索的。以下是我的一些发现。有一个新的用户界面可以探索扩展的功能，包括可以分离和重新定位的标签，一个名为 OneBox 的组合搜索和地址栏，以及超级谨慎的通知栏。微软自我设定的发布“最快浏览器”的目标可能已经成为现实。通过在这里调整 CPU 和 GPU 的工作角色，并在那里优化 JavaScript 计算，IE 团队已经非常接近他们的 Internet Explorer 9 目标。但是等等，还有呢！反应式和即时的插件和插件管理极大地改善了对页面下载时间的控制，而站点锁定看起来既实用又非常酷。你也可以参加我们编辑的微软赞助的测验，看看你能从这篇文章中回忆起什么。因此，现在就阅读[文章](https://www.sitepoint.com/article/whats-new-in-ie9)，或者参加[测验](https://www.sitepoint.com/quiz/microsoft/whats-new-in-ie9/)。
