# 谷歌更新更新

> 原文：<https://www.sitepoint.com/google-update-update/>

是的，谷歌在更新、改变、转移，你可以称之为任何你喜欢的东西，但不要称之为跳舞。发生了什么事？搜索结果像疯了一样四处移动，这就是正在发生的事情。这是什么意思？我不能告诉你，直到它停止移动，这可能需要几个星期。

我们从上周五就开始看这个了，我的电话一直响个不停。每个认为我正确地调用了 2003 年 11 月更新的人都在等着我从我的帽子里再拉出一只兔子来……我不能承诺一只兔子，但一旦这件事尘埃落定，我会有自己的看法，我们可以看看发生了什么变化。

眼下，还没有结束，除了看，或者更恰当地说，做点有成效的事情，我们什么也做不了。不幸的是，看这些东西是我的工作，所以我坚持…

谷歌搜索质量总监马特·卡茨(Matt Cutts)也是如此。

1.  假装他不是谷歌人
2.  在第一时间指挥这种事情发生
3.  处理所有积极和消极的反馈
4.  指导不可避免的调整
5.  努力向网站管理员社区解释这一切，但不要实际解释。

Matt 正在他的博客上发布“[天气预报](http://www.mattcutts.com/blog/more-weather-reporting/)，这是一个很好的开始。从最近更新开始，往回查。如果你每天有几分钟的空闲时间，读一读马特·卡茨的书是值得的。这种写作非常有趣，就像“我们掌握了所有的牌，而你看不到它们”一样。

关于“搜索界的米克·贾格尔”的更多信息，请查看亚伦·沃尔对卡茨先生的精彩采访。亚伦是个好人，以前是我在 [SEO 教练项目](http://www.seoresearchlabs.com/seo-course.php)的学生，他真的在让马特敞开心扉方面做得很好。
