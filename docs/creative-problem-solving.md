# 如何利用你的创造力更有效地解决问题

> 原文：<https://www.sitepoint.com/creative-problem-solving/>

面对问题时，创造力可能是一个有价值的工具，尤其是当你在一个困难的挑战上花了大量时间，却总是回到同样平庸的解决方案。创造性的解决问题的方法对每个人都有效，不管你的创造力水平如何。无论你是一个典型的创造性思考者，还是一个更有条理的思考者，你都可以通过创造性的解决问题来运用创造力解决问题和挑战。

## 创造性解决问题

创造性解决问题注重运用想象力和创新。目标是重新定义你的挑战，开发创造性的想法来解决你面临的问题。主要的方法论，创造性解决问题的过程，是由亚历克斯·奥斯本和西德尼·j·帕恩斯博士在 20 世纪 50 年代提出的。

在实践中，创造性解决问题的技巧被设计成将一个人的精神状态转变成一种培养创造力的状态，以便开发新的想法。这种方法基于这样一种信念，即大量的想法增加了其中一个有价值的机会，而且很多时候，从一个新的角度来看待一个问题可以产生一个不同但更有效的解决方案。

## ![](img/d4a6141e4cb875df855a05f5d01c9f62.png "puzzle-pieces")尝试的技巧

在处理问题时，有许多方法可以实现创造性的问题解决。总体目标是脱离传统思维，让你的创造力帮助你开发更好的解决方案。这里有一些创造性解决问题的不同技巧。

### 头脑风暴和思维导图

创造性的问题解决可以通过头脑风暴和思维导图等开放思维过程来启动。通过写下所有可能的选择和想法，甚至那些看起来不现实的想法，清空你的大脑，这是一种启动你创造力的有效方式。

### 随意的想法

关注随机想法是创造性思维的另一种方式。这项技术包括在你的直接环境中选择一个随机的物体，并将其与你的问题直接联系起来，即使它看起来完全不相关。强迫联想可以非常有效地鼓励创造性思维。

### 挑衅

这种技巧依赖于对形势做出明显不切实际的陈述(挑衅)。这个想法是，陈述然后分析这些古怪的想法会帮助你发现新的方法，可能会导致创造性的解决方案。

### 横向思维

横向思维是一种创造性解决问题的形式，它专注于面对问题，而不是以传统的直接逻辑方式，而是通过使用一种创造性的方法来探索替代方案和不太明显的解决方案。你可以从侧面而不是正面来思考问题。
