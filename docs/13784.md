# 如何产生销售线索

> 原文：<https://www.sitepoint.com/how-to-generate-leads/>

在最近的一篇博客中，worchyld 询问了如何挖掘潜在客户。这在我的博客、我在 Sitepoint 上的文章以及我的书中都有广泛的讨论，但是有一本入门书总是好的:

1.在你开始创造商机之前，确保你已经有了一个很好的战略基础。这意味着:

–选择一个目标市场进行重点营销。这样，你可以花更少的钱获得更多的客户。

–开发一条营销信息，传达您解决的问题、您提供的益处、您如何获得结果、您的与众不同之处以及您成功的证明。所有这些都必须是你的市场认为对他们(而不是你)有意义的。

2.变得可见:

–从一些低成本、高影响力的方式开始:加入你的目标市场所在的协会；说话；写；以领导的角色做社区服务；积极开发推荐来源；对你的目标市场的问题做一些研究；发布有新闻价值的新闻稿；提供一些关于您专业领域的信息产品(如音频、视频、白皮书)。

–瞄准 25 至 100 名潜在客户。给每个人发一系列信息信。在每封信之后跟进，介绍你自己，并更多地谈论你在信中传达的信息如何适用于他们。通过对潜在客户的网站做一些研究来定制你的信函，并就你如何能提供帮助提供个性化的想法和建议。4 个月后得到明确的“是”或“否”。用新的列表替换“是”和“否”的潜在客户。

–更善于向客户寻求推荐。有一个具体的方式要求引荐。不要只寄希望于正面口碑。

–通过提供潜在客户(而非您)关心的设计和开发信息和教育内容，吸引潜在客户访问您的网站。通过你的网站获取他们的联系信息。

–建立互补的专业人士和同事网络，他们希望互相帮助取得成功。比如，一起吃午饭，交换线索和想法。

3.虔诚地关注你联系人名单上的人。在他们允许的情况下，继续将人添加到您的列表中。用一系列信息丰富的文章和信息更新它们。

还有更多，但我相信读者会补充我错过的任何东西。
