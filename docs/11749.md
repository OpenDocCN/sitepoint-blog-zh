# 创建高效移动办公室的 7 个基本步骤

> 原文：<https://www.sitepoint.com/creating-an-efficient-mobile-office/>

为自己工作的好处之一是你可以随时随地工作。这意味着当你不得不离开家的时候，你可以继续工作(例如，如果你正在做你的房子)。当你处于等待模式时(例如，在 DMV 排队等待更新驾照)，你可以充分利用停机时间。你也可以休工作假(但是请务必安排好非工作时间！)。

移动性有不同的程度——从拥有智能手机和能够收发电子邮件，到拥有一个功能齐全的移动办公室。由于我们大多数人在离开工作时已经通过手机和其他设备以某种方式连接在一起，我将专注于后者——创建一个完全可操作的移动办公室。

我自己也经历过这一过程，以下是我采取的最重要的步骤，以确保我可以随时随地工作。
 **1。获得一台笔记本电脑**

这是显而易见但至关重要的一步。如果你被 22 英寸的宽屏液晶显示器束缚在你的大台式机上，你肯定不能成为一个移动工作者。理想情况下，您将拥有一台功能强大的笔记本电脑，它可以取代您的台式机，成为您的主要计算机，这样无论您的笔记本电脑在哪里，您都可以自动访问所有文件和必要的数据。你甚至可以在家时将其对接，这样你仍然可以利用你的大屏幕。

如果您计划保留您的台式机并使用第二台移动计算机，那么第二步是您必须完成的。

**2。使用在线数据备份网站**

如果你在多台计算机上工作，你必须有一个在系统之间共享文件的系统。如今，有了像 SugarSync 和 Box.net 这样的服务，这变得非常容易。只要按月付费，这些和其他类似的服务可以自动备份和同步你在多台电脑上的数据，甚至允许通过互联网访问你的文件。

即使您有一台笔记本电脑作为您的主要计算机，这也是一个很好的方法来确保在系统崩溃时数据的一致备份和可访问性。

**3。确保你能上网**

周围有很多 wi-fi 接入点，您可以连接到这些接入点以便在任何地方工作，但是为了获得更稳定和安全的连接(这是您的业务所需要的)，您需要一个高速移动接入卡。您可以从您的手机服务提供商处获得一个(或通过共享您的 BlackBerry 或 iPhone 获得访问权限)，或从提供预付费或现收现付无线访问的公司处获得。无论您如何连接，都要确保使用防火墙和数据加密。

**4。扔掉纸张** 
不可避免地，当你在旅途中工作时，你会需要访问办公室的一些纸质文件。如果您没有其他方法来获取这些信息，这可能会是一个问题。如果你开始养成将所有纸质文件电子化的习惯，你可以创建一个二级数字文件柜，最终完全取代你的硬拷贝文件。

对我来说，这意味着把我所有的会议记录都放在我的电脑上(或者以后再转移)，不用任何纸质文件管理我的财务和簿记，使用电子协议，扫描我收到的任何纸质文件。在我的电脑上可以访问(和搜索)所有的标准纸质文件，这大大节省了时间……而且对环境也有好处。

**5。获取手机/传真服务**

如果你离开了你的大本营，除非你能维持你的电话和传真服务，否则你不可能顺利地经营一家企业。虽然你当然可以在必要时使用手机拨打和接听电话，但使用虚拟电话线和数字传真服务可能更专业，这样你的客户就不会感受到你的可访问性有任何变化。

我被设置为每次收到电话或传真时都会收到电子邮件通知，并且可以直接从我的电脑上拨打电话和发送传真，这样无论我在哪里，它都不会离开我。

**6。更新您的操作系统和软件**

上路之前，确保您的操作系统和软件反映最新的更新和安全补丁。如果您必须在移动时进行，这不仅是一个主要的时间杀手，而且可能需要更长一点的时间，结果是一些恼人的停机时间。

**7。设置您的外围设备**

对我来说最大的外设就是轨迹球。我讨厌笔记本电脑触摸板，对它们毫无耐心，所以无论我去哪里，轨迹球都会跟着我。你可能要考虑的其他设备:电涌保护器、延长线、无线鼠标、便携式扫描仪、额外的电池/充电器和便携式打印机(如果打印是绝对必须的)。
