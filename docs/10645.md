# 你还有默认浏览器吗？

> 原文:[https://www.sitepoint.com/no-default-browser/](https://www.sitepoint.com/no-default-browser/)

网页设计者和开发者通常会在他们的电脑上安装一些最流行的浏览器。你不知道吗？真的吗？为什么不安装几个，给他们一个尝试。

操作系统允许你设置默认浏览器，我们都有自己的喜好。有些人喜欢 Firefox 的灵活性。一些人对 Chrome 的速度和干净的界面深信不疑。其他人更喜欢 Opera 的工具和特性。许多苹果用户喜欢 Safari 的操作系统集成。IE 用户喜欢浏览器是因为……嗯，他们有他们的理由*(还有[我们对 IE9](https://www.sitepoint.com/ie9-beta-release)*寄予厚望)。

我从网景 2 开始，迁移到 IE3，4，5，5.5 和 6，然后切换到凤凰，火鸟，最终火狐。尽管我安装了其他浏览器，但我很少用它们进行其他任何网页测试。

然而，在过去的一两年里，我注意到我的浏览器使用模式发生了变化。我现在使用最实用的应用程序——有时，它只是离我的光标最近的图标。有几个原因:

1.  5 款主流浏览器都是不错的应用。你可能更喜欢其中一个，但没有一个是完美的，即使是最差的也适合一般的网上冲浪。
2.  Chrome 和 Safari 可能会提供一些令人惊叹的 CSS3 效果，但这两种浏览器之间的差距比以往任何时候都要小。所有顶级浏览器都提供了不错的渲染能力。
3.  打开两个或更多不同的浏览器通常很实用，例如，如果您同时访问工作和私人 GMail 帐户。

不过，在大多数情况下，我会使用能为手头的任务提供最佳便利的浏览器。为了快速浏览，我可能会使用 Chrome。在上网本上，我经常使用 Opera 的速度，内置的电子邮件和涡轮模式的慢速连接。对于存储书签和网页开发，很难击败 Firefox。最后，我仍然使用 IE 进行测试和一些特定的公司应用程序。

虽然我怀疑许多普通互联网用户会在不同的应用程序之间转换，但超级用户越来越不可能只使用一个浏览器。话又说回来，也许只有我——我已经成了一个浏览器妓女。
