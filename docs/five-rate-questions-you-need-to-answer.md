# 您需要回答的五个评级问题

> 原文：<https://www.sitepoint.com/five-rate-questions-you-need-to-answer/>

您上次审核或更改费率是什么时候？我刚刚开始了一个新的财政年度，所以利率重估是我的议程。

对于自由职业者来说，时薪问题可能是一个经常引起恐慌的问题，但是有几个相关的问题你最好同时考虑一下。

## 1.初诊要收费吗？

免费提供初步咨询可能会降低风险，并鼓励潜在客户与你会面，但这确实会引发一个问题:你是否愿意与那些不准备——或负担不起——为你的时间支付报酬的客户会面。

有些提供免费的、有时间限制的初步咨询，这可以避免自由职业者作为学科专家可能面临的无休止的脑力劳动式的初步咨询。其他人只有在客户随后与他们接触时才收费——这是一个巧妙的解决方案，可以让你避免向那些实际上不是客户的人发送单小时的发票。

## 2.我应该为不同的任务收取不同的费用吗？

将服务打包成具有单一价格标签的“解决方案”可能是招徕新客户的好方法，或者使复杂的解决方案对某些受众来说更容易消化。您可能同意降低某项任务的费率，以便在某个行业中与他人竞争(例如，您可能无法对数据输入任务收取 web 设计费率)。但是除非这个任务和你的其他工作一样吸引你，或者比其他工作更吸引你，否则你会很快变得不抱幻想。

无论你是在设计还是在做数据输入，你的客户得到了你所有的脑力和精力——他们不只是得到了你知道如何复制、粘贴或敲击键盘的那部分。这就是为什么对特定的服务收取不同的费用从来没有真正对我起作用。我宁愿与需要并重视我全部投入的客户一起工作，而不是与那些认为他们只需要我技能的一小部分，并愿意只为那部分付费的客户一起工作。如果项目采用我的任何基本服务或技能，我希望能够收取我的标准费用。

## 3.我应该对不同的客户收取不同的费用吗？

你对这个问题的回答将取决于你所做的工作类型、行业和客户。一些行业仍然深受全球金融危机的影响，这使得你很难提高费率，甚至很难用你的标准时薪从这些行业吸引客户。

另一方面，某些项目和客户将保证更高的利率。最近，我作为一个主题专家给一个组织的员工开了一天的研讨会。我收取的费用超过了我的标准时薪，因为这位客户获得了与我通常的文案客户不同的价值——获得的技能和工具将改善他们的工作流程和产品质量。这个项目与我的日常服务产品相去甚远，它带来的好处与我的核心工作大相径庭，因此我觉得有理由对它收取不同的费用。

如果你担心你的客户说话的前景，并发现你向他们收取不同的费用，请记住，你没有敲任何人的竹杠，或要价过高(嗯，我们希望没有！).如果你认为你有充分的理由对不同的客户收取不同的费用，你应该不难向你的客户解释这一点，万一客户发现你向他们收取的费用比你向其他人收取的费用高。

## 4.我应该定期提高利率吗？一下子？

没有自由职业者会在真空中设定自己的费率，所以你是否会在某个特定时间上调费率无疑取决于市场、你所在的行业、你的财务状况等等。你可能会发现很难用日历日精度来提高你的利率。

当你决定提高你的利率时，你需要想出如何去做。如果你的费率在 1 月 1 日全面上涨，这是否意味着现有客户将收到按两种不同费率收取时间费用的发票？也许您会决定在每个客户的基础上实现新的比率，当每个客户的下一个新工作到来时。不过，你仍然需要想出如何对待现有客户(我的建议是:在你加息之前提前提醒他们，这样就不会有意外的惊喜了)。

## 5.我应该支付定金吗？

对于那些想确保每个月(或其他时间段)能分配给他们一定时间的客户来说，预聘费率包括批量折扣。它们在维护协议中特别常见，但也受到那些希望与你建立牢固、持续关系的客户的青睐。

预聘安排可能会很快变得复杂—如果您有每月预聘安排，那么给定一周中任何未使用的小时数会发生什么情况—它们可以在同一个月内滚动到另一周吗？你会根据客户希望获得的时间量来逐步提高折扣吗？你需要什么样的取消时间？因此，提前考虑一下可能性，以及你需要哪些基本协议才能让聘用合同生效，这是一个好主意。

设定费率并不像选择一个小时的数字那么简单。这些只是我在考虑和修改费率时所面临的一些挑战。你必须克服哪些利率障碍？
