# 投票:是什么让电脑成为电脑？

> 原文：<https://www.sitepoint.com/poll-what-makes-a-computer-a-computer/>

最近，我决定升级我的手机。我在黑莓和 iPhone 之间犹豫不决，前者可以让我继续使用威瑞森，后者在我所在地区的覆盖范围更广，而后者显然更适合满足我最迫切的需求(繁重的网页浏览、一些电子邮件和点滴的 Twitter 信息)。鉴于如今智能手机所做的事情类型，把它们贴上电脑的标签公平吗？

手机博客作者 James Kendrick [今天](http://jkontherun.com/2009/01/05/are-smartphones-computers/)指出，如果像苹果 iPhone、HTC/T-Mobile G1、黑莓 Storm 或诺基亚 N95 这样的手机被认为是电脑，那么诺基亚是 2008 年最大的电脑制造商。但是智能手机真的是电脑吗？是的，它们拥有越来越强大的处理器，并能完成许多计算任务，但正如 Kendrick 指出的，它们不能完成你的笔记本电脑或台式个人电脑能做的所有事情。

智能手机还不是个人电脑的替代品，而且可能永远也不会。很难想象在手机上写学期论文或者设计网站。尽管 iPhone 和 Blackberry 在许多常见计算任务上的体验很好，如电子邮件或网页浏览，但你不会想用智能手机编辑照片。

另一方面，在世界的许多地方，尤其是在发展中国家，移动电话已经被用作主要的计算设备。我们越来越多地将大部分计算时间花在智能手机上，完成日常任务，如收发电子邮件、即时消息、浏览网页、玩游戏、拍照、管理约会，甚至是监控服务器或管理我们的金融投资组合。如果我们能做到这一切，智能手机[不就是一台电脑吗](http://www.readwriteweb.com/archives/is_the_mobile_web_finally_set_to_take_off.php)？
