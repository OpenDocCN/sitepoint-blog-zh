任何拥有使用 php 的服务器的人都一定听说过 PHP . ini——它是用于控制和定制 PHP 运行时行为的配置文件。它提供了一种简单的方式来配置设置，如:上传目录日志错误最大脚本执行时间文件上传限制…等等。php.ini 是 php 启动时寻找的第一个文件，因为它设置的配置指令非常重要。
