# 民意调查:你会选择退出谷歌、雅虎吗？广告？

> 原文：<https://www.sitepoint.com/poll-will-you-opt-out-of-google-yahoo-ads/>

雅虎！[今天](http://ycorpblog.com/2008/08/08/choice-in-advertising/)宣布，它将允许用户选择退出定制广告，这些广告的目标是基于你以前的网上冲浪习惯。谷歌昨天发布了[类似的公告，为通过谷歌内容网络上的广告(双击广告)提供的 cookie 增加了一个全球](http://googleblog.blogspot.com/2008/08/new-enhancements-on-google-content.html)[选择退出按钮](http://www.google.com/privacy_ads.html)。雅虎！也有一个[选择退出链接](http://info.yahoo.com/privacy/us/yahoo/opt_out/targeting/details.html)，但这些改变要到月底才会真正生效。

新的退出政策是对上周美国众议院能源和商业小组委员会向 33 家互联网服务提供商和互联网公司发出的一封信的回应。，微软和美国在线，表达了对私人数据如何被记录和使用的关注。这封信要求这些公司回答一系列关于广告如何为用户“量身定制”以及消费者如何被告知这一做法的问题。

雅虎！今天给能源和商业委员会主席约翰·丁格尔(D-MI)写了一封信作为回应。你可以[在这里](http://yhoo.client.shareholder.com/press/releasedetail.cfm?ReleaseID=327212)阅读整封信。ZDNet 的 Larry Dignan 提出的一个担忧是，雅虎！T2 的退出程序不像谷歌的 T3 那么干净。除了雅虎！网络广告退出，蓝锂和右媒体似乎有单独的退出链接，这是雅虎的一部分！。(注:您可以在[本页](http://networkadvertising.org/managing/opt_out.asp)选择退出“网络广告计划”的任何成员。大量的公司，包括谷歌双击，雅虎！、AOL 的 Tacoda、24/7 Real Media 都是会员。)

所以我们的问题是，你会选择退出雅虎吗？和谷歌广告 cookies，现在你可以选择？
