# 值得一看的新 Webmin 版本

> 原文:[https://www.sitepoint.com/new-webmin-release-worth-a-look/](https://www.sitepoint.com/new-webmin-release-worth-a-look/)

对于那些熟悉 [Webmin](http://www.webmin.com) 的人来说——他们了解 Unix 工作站和服务器(包括 Mac OS X)上的 it 管理价值。尤其是当您需要管理多个系统时。

对于 unitiated——我不久前在 Webmin 上写了一篇文章，探索了它作为一个强大的根支持的系统管理工具的能力。

当我在自己的 webmin 使用中同时处理几个服务器时，这些新的更新对我来说非常有用——包括 LDAP。

更新亮点包括:

*   改进的 Postfix 邮件队列管理
*   能够在 MySQL 和 PostgreSQL 中一次编辑和/或删除多个表(和数据库)
*   群集工具，包括一次将打印机部署到多台服务器上(适用于生成打印报告和电子分发的服务器——在某些监管环境中很重要)
*   方便的 LDAP 增强功能，例如多个用户和组删除以及多个用户和组解锁
*   只读访问选项，允许可能需要查看 webmin 配置但不需要编辑权限的用户进入
*   加快了搜索和查看系统日志的性能——在处理非常繁忙的服务器时非常好。

如果你有强烈的系统管理需求，或者想探索比传统的 Unix 控制面板解决方案更强大的选项，Webmin 值得一试。最好的是——作为开源软件——它没有许可费。

像往常一样——如果你从中受益，考虑你可以帮助项目的方式——无论是通过代码、文档、测试、促销还是老式的现金捐赠。
