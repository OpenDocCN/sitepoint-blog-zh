# 17 种促进协作的在线会议工具

> 原文:[https://www.sitepoint.com/online-meeting-tools/](https://www.sitepoint.com/online-meeting-tools/)

在线会议工具的使用有很多原因——销售演示、网络研讨会和培训等等。此外，如果你像许多自由职业者和小企业主一样在家工作，你会面临一个独特的挑战，即需要与可能位于世界各地的客户进行实时会议。

有许多在线会议工具以不同的价格提供各种功能。这里有几个你可能想考虑的工具，它们的一些显著特征和使用它们的成本。

费用:30 天免费试用，按使用付费，每月和每年的计划选项(按使用付费是每用户每分钟 0.32 美元)
