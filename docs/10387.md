# Chrome 7 超越 IE7 成为世界第三大最常用浏览器

> 原文:[https://www . site point . com/chrome-7-overtakes-ie7-季军-browser/](https://www.sitepoint.com/chrome-7-overtakes-ie7-third-placed-browser/)

我关注浏览器市场已经有几个月了。IE 跌破 50% 是 10 月份的大新闻。本月的新闻没有那么引人注目，但网络开发者将欢呼雀跃:Chrome 7 已经超过 IE7，以 12.09%的市场份额成为世界第三大最受欢迎的浏览器。

Chrome 7 仍然落后于 Firefox 3.6 (25.32%)和 IE8 (29.49%)，但 IE7 的消亡是值得庆祝的！就个人而言，我觉得 IE7 比 IE6 更有问题。)

让我们更详细地看看 [StatCounter 统计数据](https://gs.statcounter.com/)…

| 浏览器 | 九月 | 十一月 | 变化 | 亲戚 |
| IE 9.0 测试版 | 0.09% | 0.32% | +0.23% | +255.60% |
| IE 8.0 | 29.38% | 29.49% | +0.11% | +0.40% |
| IE 7.0 | 12.98% | 11.90% | -1.08% | -8.30% |
| IE 6.0 | 7.42% | 6.45% | -0.97% | -13.10% |
| 火狐 4.0 测试版 | 0.26% | 0.41% | +0.15% | +57.70% |
| 火狐 3.5 以上版本 | 28.33% | 28.50% | +0.17% | +0.60% |
| 火狐 3.1- | 2.48% | 2.26% | -0.22% | -8.90% |
| 铬 | 11.52% | 13.32% | +1.80% | +15.60% |
| 旅行队 | 4.22% | 4.70% | +0.48% | +11.40% |
| 歌剧 | 2.03% | 2.02% | -0.01% | -0.50% |
| 其他人 | 1.38% | 0.95% | -0.43% | -31.20% |
| IE(全部) | 49.87% | 48.16% | -1.71% | -3.40% |
| Firefox(全部) | 31.07% | 31.17% | +0.10% | +0.30% |

“变化”栏显示市场份额的绝对增加或减少。“相对”栏表示相对变化，即 IE7 在过去两个月中失去了 8.3%的用户。

微软会很高兴看到 IE8 的使用几乎没有变化，而 IE9 用户增加了 2.5 倍！IE9 的统计数据有点不可靠——10 月份的数据非常低，因为测试版浏览器于 2010 年 9 月 15 日发布。然而，IE9 测试版正在赶上 Firefox 4 测试版，后者已经推出了更长时间。

Firefox 的总体份额基本保持不变。少数用户已经升级，但没有明显的损失或收益。

歌剧和游猎也是类似的情况。Safari 的适度增长在很大程度上归功于 iPad 的成功，iPad 占据了 0.26%的市场份额。

IE6 和 IE7 的总份额下降了 2%，Chrome 占据了这些用户的大部分。谷歌浏览器的增长仍然令人印象深刻，我怀疑这得益于 Firefox 4 的推迟发布和 IE9 日期的持续不确定性。

然而，2011 年一切都会改变吗？IE9 的速度提升会赢回转用 Chrome 的用户吗？火狐 4 会重新点燃人们的激情吗？RockMelt 会成为第六个主流浏览器吗？这将是有趣的一年…
