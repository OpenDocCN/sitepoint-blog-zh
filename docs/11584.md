# CSIRO 因 Wi-Fi 专利获得 1.85 亿美元

> 原文:[https://www.sitepoint.com/csiro-185-million-wifi-patent/](https://www.sitepoint.com/csiro-185-million-wifi-patent/)

您知道 802.11 a、g 和 n 无线网络协议是专利技术吗？我怀疑许多硬件供应商还没有意识到其中的含义。

2007 年，澳大利亚政府资助的研究机构 CSIRO(联邦科学和研究组织)赢得了对日本网络制造商 Buffalo Technologies 的法律诉讼。美国联邦法院裁定布法罗侵犯了美国专利 5487069，该专利于 1996 年授予 CSIRO，涵盖了*“在办公室和家庭等环境中高速无线交换大量信息”*。

这引发了进一步的法律骗局:

1.  该案件被用作对其他 wi-fi 硬件制造商进一步侵权诉讼的基础，包括 3Com，Accton，华硕，Belkin，D-Link，富士通，Marvell，任天堂，SMC 和东芝。
2.  包括惠普、苹果、英特尔、戴尔、微软和 Netgear 在内的科技公司联盟随后对 CSIRO 提起法律诉讼，试图使专利无效。
3.  随着案件在德克萨斯州的进展，CSIRO 与惠普、戴尔、英特尔、微软、华硕、富士通、任天堂、东芝、Netgear、D-Link、Belkin、SMC、Accton 和 3Com 达成了 1.85 亿美元的巨额庭外和解。

CSIRO 执行董事 Nigel Poole 表示:

> CSIRO 开始鼓励工业界获取专利技术的使用许可。当这证明不成功时，我们启动了法律程序，随后导致对 CSIRO 提起诉讼。
> 
> 今年早些时候的结果是，14 家公司根据保密条款与 CSIRO 达成了和解。迄今为止，这些和解带来的收入约为 2 亿澳元。
> 
> 我们相信有更多的公司正在使用 CSIRO 的技术，我们希望进一步许可这项技术。我们敦促目前正在销售 802.11 a、g 或 n 设备的公司联系 CSIRO 并寻求许可，因为我们相信他们正在使用我们的技术。

CSIRO 似乎有一个紧凑的法律案例，但是他们应该能够将无线网络作为一个通用概念申请专利吗？最终，在这些案件中获胜的仅仅是律师吗？
