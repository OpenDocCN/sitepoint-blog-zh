# 投票:Flash 还是 Silverlight？

> 原文:[https://www.sitepoint.com/poll-flash-or-silverlight/](https://www.sitepoint.com/poll-flash-or-silverlight/)

![](../Images/5b920e9d3da4edb54308762e5b11b5b2.png "flash-silverlight")SitePoint 播客最新一集的主题是在 Flash 和 Silverlight 等富媒体平台上开发和部署 web 应用程序的利弊。

许多人认为桌面应用程序正在向云迁移，未来的计算体验将是我们与实际运行并在其他地方存储数据的程序进行交互。Adobe 和微软分别拥有 Flash 和 Silverlight，是提供开发平台的主要候选公司之一，许多下一代富互联网应用程序都将建立在该平台上。

然而，即使 Silverlight 的采用很好，微软声称他们 20 个月前的技术的某个版本已经安装在 25%的联网计算机上，与 Flash 相比，这仍然是杯水车薪。Flash 几乎无处不在，Adobe 声称其安装基础超过 98%的网络连接计算机——大约是微软 Silverlight 安装基础的 4 倍。

尽管如此，微软实际上在 Silverlight 上的创新相当迅速，计划明年发布 RIA 技术的第 3 版。由于微软在企业中的优势，在过去的几年中，他们已经在 Silverlight 的高调部署中赢得了大量订单。到 2009 年底，Flash 和 Silverlight 的用户覆盖面可能会更加平等。

在上周的黑客新闻中，有人提出了一个有趣的假设。如果 Silverlight 与 Flash 有相同的安装基础，您会使用哪个？我们很想听到你的回答，所以请在我们的投票中投票，并在评论中让我们知道你为什么这样投票。
