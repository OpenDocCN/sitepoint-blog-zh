# 网络方向:创新的神话

> 原文：<https://www.sitepoint.com/web-directions-the-myths-of-innovation/>

魅力四射的斯科特·伯昆发表了一篇鼓舞人心的演讲，拉开了 Web Directions 大会第二天的序幕。

斯科特描述了关于创新的几个神话——包括想法对我们来说是外在的，以及顿悟在我们最意想不到的时候来到我们身边的误解。他举了历史上著名人物的例子:阿基米德，发现如何通过测量黄金的密度来鉴别黄金，以及牛顿对重力存在的著名揭示。人们会抓住他们想听的故事不放(比如阿基米德裸身从浴缸里跳出来，在街上奔跑，牛顿无所事事，直到一个苹果落在他头上)。现实是，发明家和创新者花了一生的时间试图解决一个问题，而这些著名人物所投入的努力很快就被遗忘了。

他建议，在一个公司里，你越少使用“创新”这个词，你就越有可能去做。如果你经常使用这个词，你就有大麻烦了。

创新者本质上是探索者:他们在地图上选择一个空白点，然后说“我想去那里，带回酷的东西”。然而,《星际迷航》是对探索的一种不准确的表现——所有艰难和无聊的东西都被曲速驱动器和传送机掩盖了。

历史上还能找到更有趣的探险家——以环球航行闻名的麦哲伦，居然中途就死了。英国探险家詹姆斯·库克因“发现”大部分大洋洲而被人们铭记，而不是因死于一次灾难性的夏威夷探险。

然而，人们谈论一项发明之前和之后的方式是非常不同的。事先，人们一致认为“这些人疯了，他们在浪费时间”。之后，人们寻找突破的时刻，神奇的时刻，因为那是他们想听到的。达芬奇、麦哲伦、库克等人都致力于其他人都不相信的事情，但他们还是坚持了下来。

任何时候你创新，都会有人对你的发明带来的变化产生情绪反应
