# 谷歌会收购 Twitter 吗？

> 原文:[https://www.sitepoint.com/will-google-buy-twitter/](https://www.sitepoint.com/will-google-buy-twitter/)

互联网上流传的最新谣言是，谷歌收购微博服务 Twitter 的谈判已进入最后阶段。

过去一年，Twitter 注册人数激增，部分原因是名人的支持和大众媒体的特写。对于谷歌来说，Twitter 无疑是其争取互联网所有权的合理选择。

Twitter 最近的估值为 2.5 亿美元，尽管该公司还没有赚到一分钱。就在几个月前，它拒绝了脸书 5 亿美元的收购要约(尽管其中一部分包括目前还没有明确价值的脸书股票)。谷歌的出价必须高。

但 Twitter 对谷歌的真正价值是什么？可能的可能性包括:

1.  新闻:突发事件通常会首先发布到 Twitter 上，例如孟买恐怖袭击、飞机坠入纽约哈德逊河，以及史蒂芬·弗莱被困在电梯里的惊人消息。
2.  热门话题:谷歌可以收集关于网上最新热门话题的有价值的营销知识。
3.  用户群: Twitter 拥有近 1000 万用户，谷歌可以瞄准这些用户。
4.  广告收入:聊天敏感广告可能会出现。
5.  搜索:如果谷歌不能将 Twitter 的搜索货币化，那就没人能了。
6.  市场支配地位:拥有 Twitter 让谷歌比以前更大更强。

谷歌还将能够向 Twitter 提供技术和专业知识，以有效和可靠地扩大其运营规模。

然而，谷歌并不总是做对每件事:例如，YouTube 尚未盈利。还有一种可能性是，Twitter 只是一种时尚，只会一直流行到下一个“大事件”出现。

你怎么想呢?谷歌和推特会成为好的合作伙伴吗？微软等竞争对手会作何反应？Twitter 能盈利，有长远的未来吗？
