# 微软今天发布了 Visual Studio 2010

> 原文:[https://www . site point . com/Microsoft-launch-visual-studio-2010-today/](https://www.sitepoint.com/microsoft-launch-visual-studio-2010-today/)

尽管 Windows 7 发布相当低调，但微软正在推出 Visual Studio 2010 和。大张旗鼓的 NET 4.0。我当然不记得以前的版本有这么大的事件。

为期一周的发布会在拉斯维加斯、伦敦、中国北京、印度孟加拉和马来西亚吉隆坡举行。[小型赛事](http://www.microsoft.com/business/2010events/DevEvents.aspx)正在以下美国城市举行:丹佛、奥兰多、橡树溪、波士顿、布鲁克林、阿灵顿、科斯塔梅萨、达拉斯、休斯顿、亚特兰大、旧金山、底特律、明尼阿波利斯、费城和西雅图。[全球会议](http://www.microsoft.com/visualstudio/en-us/visual-studio-events)也可以在阿根廷、澳大利亚、巴西、智利、法国、德国、匈牙利、意大利、爱尔兰、日本、墨西哥、波兰、俄罗斯、西班牙、瑞士和瑞典举行。

![Microsoft Visual Studio 2010](../Images/bcd1c87be3552bcfcd8f55020b749e84.png) [该套装将有三个版本](http://www.microsoft.com/visualstudio/en-gb/products/2010/default.mspx):

*   **VS 2010 专业版**:核心应用。
*   **VS 2010 Premium** :专业版，附加设施，微软 Office，Expression Studio，Visio，Project。
*   **VS 2010 旗舰版**:带有额外测试用例和管理设施的高级版本。

这三个版本都包括 Windows Server、Azure 和 SQL Server 的测试版。它们可以安装在 SP2 XP、SP1 Vista、Windows 7、SP2 Server 2003、SP2 Server 2008 或更好的系统上。

Visual Studio 可能不便宜，但是当您考虑您得到的东西时，它的价格和 MSDN 订阅与其他专业开发产品相比还是不错的。如果你想尝试这些技术，微软还提供了免费的快速版本。

如果你是一个 ASP.NET 开发者，下面是你在 VS 2010 中可以期待的:

*   。NET 4.0 —框架的最新版本
*   集成 Silverlight 开发
*   C#中的 MS Office 可编程性
*   F# (F-Sharp)，一种支持多种编程范式的新语言。从语法上看，这有点像 Python，但是我不是 Python 专家，所以请在下面随意嘲笑我！
*   并行编程工具
*   团队协作设施
*   代码分析帮助您避免常见错误
*   更清晰、更整洁的 IDE，具有更好的多显示器支持

即使您没有开发 ASP.NET 应用程序，VS 现在也支持 JavaScript 的智能感知。这很有意思——很少有 ide 提供像样的 JavaScript 支持，可能是因为这种语言中有大量的编程模式和原型继承。

我认为微软通过 VS 和 ASP.NET 正朝着正确的方向前进。他们似乎在遵循 web 标准，而且工具也很成熟。VS 2010 几乎肯定会成功，但是你会考虑从你目前选择的语言和平台转换吗？

(请注意，微软没有就这篇文章与我或 SitePoint 接洽或支付任何费用。)
