# 你的家人对你的创业有多支持？

> 原文：<https://www.sitepoint.com/how-supportive-is-your-family-of-your-entrepreneurship/>

家庭的支持对每个成功的企业家来说都是必不可少的，或者至少肯定会让这变得更容易。

我很幸运，我的妻子喜欢并支持我所做的事情，尽管她不时告诉我，我的一个想法很愚蠢。无论如何，我通常会继续这个想法，而且她通常认为这个想法很愚蠢是对的。我们最近就一件与商业无关的事情发生了争执，最后她大声说道:“顺便说一句，你关于一个致力于[新网站主题]的网站的想法是愚蠢的。行不通的。”无论如何，我会继续这个想法，但是你可以看到这种偶尔的冲突和有收入的配偶之间的冲突有什么不同。另外，鉴于她的过往记录，我对这个想法的信心动摇了，这有点令人沮丧。

而且，有时，当我投资新想法，同时等待其他想法实现，资金变得紧张时，可能会出现紧张。但这是恰当的，因为我的首要责任是养家糊口。

父母、兄弟姐妹、配偶——当我们开始创业时，他们都有话要说。有时他们会支持，有时他们会消极。有时他们真的会让你失望——尽管他们对你的市场或成功的机会一无所知。

有一个关于最坏情况的比喻:有一种螃蟹，渔民可以不用绳子把它们堆叠在一起，甚至就在水边。这是因为如果一只螃蟹试图离开螃蟹堆，其他螃蟹会把它拉回来。所以没有一只螃蟹能获得自由。因此，如果你有一个不习惯创业的家庭，有时他们可能会像这些螃蟹一样，试图把你拉回他们的世界。

知道什么时候前进，什么时候屈服需要勇气和信念。

你的家人支持吗？如果情况不是这样，你是如何处理的？
