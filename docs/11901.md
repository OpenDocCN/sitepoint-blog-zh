# 推荐:让客户愿意推荐你

> 原文：<https://www.sitepoint.com/getting-referrals/>

对于个体企业、小型企业和大型企业来说，推荐是大多数成功企业的重要组成部分。推荐是非常宝贵的，因为他们可以是一个被动的，间接的和廉价的方式来促进你的业务。

**为什么推荐有用？**

口碑营销相当强大。事实上，在 2007 年尼尔森媒介研究进行的一项全球调查中，78%的受访者更信任现有用户对产品和服务的推荐，而不是任何其他形式的广告。很有道理，不是吗？我们更容易信任我们的朋友、同事和亲身经历过某些事情的人，尤其是在我们进行某种投资或购买的时候。

虽然许多推荐是自动发生的(这也是它们如此有效的部分原因)，但你可以采取一些行动来帮助这个过程。这里有一些方法可以让你的客户谈论你，从而产生更多的业务。

让你的客户看起来不错。

从客户的角度来看，没有什么比堪称典范的服务和高质量的工作更能激发行动了。另外，如果客户经常被问到，“谁设计了你的网站？”或者对你所做的其他工作表示称赞，他们会不断地将人们引向你的方向。

花时间建立有意义的关系。

除了做好工作和让客户开心的重要性，人们通常更愿意推荐他们信任和尊重的人。与客户达成这一目标的最简单方法就是花时间建立真实而有意义的关系。诚实、尊重和真诚，不要害怕分享一点你个人的一面。

给推荐你的人一个奖励。

虽然大多数推荐都是自然发生的，不需要激励，但当你提醒客户你已经准备好并愿意被推荐给别人时，肯定会有所帮助。您的奖励可以是金钱奖励(每个合格推荐 XX 美元)、基于服务的奖励(免费服务时间)或分级折扣(折扣百分比)。
 **让人很容易为你歌功颂德。**

你不希望你的客户处于这样一种情况:他们试图向你推荐一位同事，但他们记不起你的确切网址或电子邮件地址。你可以通过定期向他们发送额外的名片、印刷材料和电子邮件来避免这种情况，这些信息可以传递给同事。保持可见，以获得更有效和更频繁的推荐。

底线是，快乐的客户不仅更愿意把你介绍给其他人，而且他们也有能力做出最好的介绍——介绍给像他们一样的人，你想与之共事的人。
