# 谷歌发布 Chrome 5 测试版

> 原文：<https://www.sitepoint.com/google-release-chrome-5-beta/>

Chrome 4 发布于——什么——3 分钟前？实际上是一月底，但是，说真的，谷歌，这些快速的版本更新是怎么回事？长达十年的 betas 版到底怎么了？

我不相信有人会关心浏览器的版本号，但这并没有阻止谷歌发布 Chrome 5 测试版。如果你不是 Windows 用户，你会很高兴听到它也可以在 [Mac](http://www.google.com/chrome?platform=mac) 和 [Linux](http://www.google.com/chrome?platform=linux) 上使用。

这款浏览器仍然简洁、简陋，但谷歌已经加入了一些新功能:

![Google Chrome](img/5eaba2d6beb92f247a899872d26cd973.png "Google Chrome") **扩展**
浏览器扩展将在所有平台上得到支持，API 将获得大量更新。

自动语言翻译、全屏模式和实时(非缓存)刷新也将在 Mac 和 Linux 上提供——Windows Chrome 4.1 用户已经拥有这些功能。

尽管谷歌是 HTML5 的拥护者，但 Adobe 的 Flash 插件将作为标准提供。Chrome 会安装 10.1 版本，自动更新插件，无需用户干预。

也有传言说将会有一个 PDF 插件。求你了不要。

同步
书签已经可以跨多个安装同步，但第 5 版将提供密码，表格自动填充条目和主题。

**Windows 7 集成**
Aero Peek 和跳转列表将得到支持，但幸运的是，如果您通常在打开 93 个标签页的情况下浏览，您将能够调整行为。

地理定位
这个新的 HTML5 功能可以告知网站你的位置(如果你允许的话)。使用 GPS 的移动用户将收到最准确的位置信息，但也有可能从您的 IP 地址和/或无线网络获得区域信息。

**Native Client 和 WebGL**
Chrome 5 将包含以下早期版本:

*   NaCl——允许本机 32 位 x86 代码在浏览器中运行的开源插件，以及
*   WebGL —硬件加速的 3D 图形

发布日期尚未公布，但如果 Chrome 5 最终版很快出现，我不会感到惊讶。按照这种速度，在 IE9 发布之前，Chrome 将达到两位数。

你会用 Chrome 5 吗？
