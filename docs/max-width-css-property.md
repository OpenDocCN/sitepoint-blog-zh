# 最大宽度(CSS 属性)

> 原文：<https://www.sitepoint.com/max-width-css-property/>

## 描述

此属性设置块或被替换元素的最大内容宽度。此最大宽度不包括填充、边框或边距。

应用了`max-width`的元素永远不会比指定的值宽，即使 width 属性设置得更宽。然而，这个规则有一个例外:如果指定的`min-width`的值大于`max-width`的值，容器的宽度将是最大值，这意味着在这种情况下`min-width`的值将是应用的值。

`max-width`通常与最小宽度一起使用，为相关元素产生一个宽度范围。

> 结合最大宽度和宽度
> 
> 注意`max-width`和 width 不应该使用相同的单位应用于相同的元素，因为一个会覆盖另一个。例如，如果宽度设置为 150px，而`max-width`设置为 60px，那么元素的实际宽度将是 60px，宽度声明将变得多余。
> 
> 下面的样式规则显示了如何解决使用相同单位(本例中为像素)为元素提供宽度和`max-width`的冲突:
> 
> ```
> .example {
>   max-width: 60px;
>   width: 150px;
> }
> ```
> 
> 在上面的例子中，元素的宽度固定为 60px。
> 
> 然而，当值是不同的单位时，设置`max-width`和 width 是可以接受的(虽然它可能不完全有用，但在一些情况下它可以产生良好的效果)。
> 
> 此样式规则将 160 像素的`max-width`分配给类别为“example”的图像，并分配 50%的宽度:
> 
> ```
> img.example {
>   width: 50%;
>   max-width: 160px;
>   height: auto;
> }
> ```
> 
> 上例中图像的最终宽度将是最小值。
> 
> 如果您希望在页面宽度较小时缩放图像，以便图像不会脱离列，您可以使用上面的示例来确保当可用空间小于 160 像素时图像的大小会减小。
> 
> 如果可用空间大于 160 像素，图像将扩展到 160 像素宽，但不会更宽。这确保了在空间允许的情况下，图像保持在合理的大小或正确的纵横比。
> 
> 最小宽度属性可用于此场景的相反情况。

如果一个块的内容需要的水平空间比已经设置的限制所允许的要多，那么这个行为由`overflow`属性定义。

## 例子

该样式规则为 ID 为`"example"`的元素中的段落分配最大 400 像素的宽度和最小 100 像素的宽度:

```
#example p {
  max-width: 400px;
  min-width: 100px;
}
```

## 价值

属性采用 CSS 长度(px、pt、em 等)、百分比或关键字 none。负长度值是非法的。

百分比值是指包含块的宽度。如果包含块的宽度为负，则使用的值为`none`。
