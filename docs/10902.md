# 如何成为一名优秀的在线社区成员

> 原文：<https://www.sitepoint.com/be-a-great-online-community-member/>

无论你是在寻找一个问题的快速答案，还是在寻找可以交流想法或获得灵感的人，如果你想充分利用任何在线社区，有一些事情你需要知道。这不仅仅是遵守规则和不被禁止；这是关于知道如何找到和使用对你最有利的资源。

要记住的第一点是，每个人都曾经是新的——对论坛和整个 web 开发行业都是如此。我们大多数人都有过“我希望他们不要认为我问这个问题很愚蠢”的时刻，无论是在 SitePoint 还是在其他论坛上。对于每一个带着这种想法问这个问题的人来说，就有 100 个人从这个答案中受益。所以放手一搏吧。

### 救命啊！

你应该知道的最重要的因素是你可以去哪里寻求帮助。一个论坛通常会有一套可供阅读的指南。SitePoint 论坛[指南](https://www.sitepoint.com/forums/faq.php)可以在页面左上角的灰色导航栏下找到。在 SitePoint 这里，我们喜欢称它们为 FAQ，因为这让它们听起来不太像规则——但是为了让它们更明显，我们在“R”字上加了一个。

如果你找不到你想要的建议，你的下一步就是联系工作人员。查找哪些员工在线的最简单方法是查找当前活动用户的列表；大多数论坛都有。在 SitePoint，转到论坛主页并滚动到底部。在那里，您将看到所有当前活动的用户(如图 1 中的[所示，“当前活动的用户显示在 site point”](#fig_active "Figure 1. The Currently Active Users display at SitePoint"))—它们的颜色编码如下:

*   红色:社区管理员(和论坛管理员)
*   格林:顾问，主持论坛。他们可以移动、编辑和删除帖子，以及处理引起任何问题的成员。
*   橙色:导师，充当版主的眼睛和耳朵，给你建议，给你指明正确的方向。
*   蓝色:团队领导，领导顾问和导师团队。

### 我该怎么做才能让你喜欢我？

嗯，有一些方法可以让你和其他论坛成员更亲近。最明显的就是要守规矩。当你在一个论坛中时，要记住的最重要的因素之一是每个人都是来学习的。没有人喜欢学校里的恶霸，我们特别不喜欢 SitePoint 的恶霸。他们肯定不会被大多数社区所容忍。当成千上万来自不同背景的人聚在一起时，很可能会有意见分歧。太酷了——这让这个地方变得有趣。这确实意味着有时你可能不得不保持沉默。有时候我觉得我已经把自己的咬掉了。但我也交了一些好朋友。

因为一个论坛吸引了对某个主题全新的人，这意味着你一定会遇到你认为愚蠢的问题。你可能也会在某个阶段问一些愚蠢的问题，所以确保你以你希望我们对待你的方式对待你遇到的每个人。我意识到这听起来好像是我在居高临下地对你说话，但令人惊讶的是有这么多人忘记了这一点。要记住的要点是耐心、支持和礼貌。

### 好吧，我会乖乖的。我还应该做什么？

实际上有几件事。我现在将详细讨论它们。

第一个似乎是常识，但是你会惊讶于有多少人被它迷惑了。我说的是 *垃圾* 。

#### 垃圾邮件？啊？我绝不会那么做。

我肯定这是无意的，但有时垃圾邮件是情人眼里出西施。你可能刚刚写了一个你认为是人类已知的最惊人的免费脚本。你注册了一个论坛，唯一的目的就是和尽可能多的人分享，因为你就是那种慷慨的人。现在从版主的角度考虑一下。他们看到一个新成员加入，而他们的第一篇文章恰好是一个链接，指向他们自己的网站。朋友们，这看起来很像垃圾邮件。在 SitePoint，我们有一些关于“自我推销”的非常仔细的指导方针，所以在你发表超级慷慨的第一篇帖子之前，请确保你已经通读了这些指导方针。

#### 不问问题就能找到信息

信不信由你，使用论坛通常有比简单提问更好的方式。大型在线社区的本质通常意味着一个成员可能已经经历了和你一样的问题。所以在你发帖之前，有几个策略你应该试试。

第一个是粘性线程。一个*粘帖*是一个被标记为重要:的粘帖，并且一直列在其论坛的顶部。顾名思义，这些线程被认为是重要的，因为它们通常包含与众所周知的问题或主题相关的信息。查看适当论坛顶部的帖子。如果帖子太多，你可能需要进行文本搜索。

如果便笺不包含您需要的信息，请尝试执行搜索。这里有一些搜索技巧，可以帮助你简化搜索过程:

*   首先，确定您的问题最有可能属于哪个(哪些)子论坛。在 SitePoint，[高级搜索](https://www.sitepoint.com/forums/search.php)选项允许您选择多个论坛进行搜索。
*   决定一个相关的关键字或简短的关键短语。这些关键字在返回时会在结果中突出显示。
*   如果您的问题可以缩小到特定的时间范围内(例如，它与上周发布的软件版本相关)，请输入一个日期参数来加快速度。

#### 我的搜索没有结果，所以我需要发布一个问题…唉！

太酷了，发布问题是我们的使命。与搜索一样，第一步是决定哪个论坛最合适。没有必要大惊小怪:如果你进入了错误的论坛，我们友好的版主会移动你的帖子——不会造成伤害。

一旦你找到了你的论坛，寻找新帖按钮。在 SitePoint 论坛上，你会发现它在左上方，橙色赞助商栏的正下方，如图[图 4，“SitePoint 新主题按钮”](#fig_newthread "Figure 4. The SitePoint new thread button")。点击它，你就可以开始了。有各种各样的格式选项，但是一旦你掌握了其中的窍门，你可以以后再去担心。

提交新帖子时要记住的两个最重要的准则是 *写一个伟大的标题* 和 *在发布* 之前预览你的帖子。

一个好的标题是至关重要的，因为当人们浏览新文章列表时，这是你文章中唯一能看到的部分。对于一个关于数据库索引的问题，一个标题为“在包含超过 3 百万行的表上添加索引”比一个标题为“新手需要帮助”更容易引起那些能帮助你的人的注意。！!"

在提交文章之前，最好先预览一下。这样，您可以进行任何必要的更改并检查格式。一旦你发了帖子，你只有 30 分钟的时间回去修改任何拼写或其他错误。这就引出了我的下一个建议。

### 互联网是永恒的

发帖前，请仔细考虑一下 *你发什么* 。如今，偶尔用谷歌搜索一下你的公司名称，看看人们对你的评价，是一种很好的商业做法。出于这个原因，如果你在寻求如何与难相处的客户打交道的基本建议，谨慎的做法是避免说出他们的名字。

另一个需要注意的情况是，如果你在客户的网站上工作。如果你需要帮助来使用下拉菜单，将示例页面上传到你自己的服务器上可能是有意义的，而不是直接链接到你客户的网站上——特别是如果你自称是 CSS 专家的话。

在 SitePoint，我们承认人们偶尔会犯一些愚蠢的错误，我们会尽我们所能来帮助你，但是我们根本没有时间来编辑数百个帖子的链接。所以请你先想想。

同样需要注意的是，我们不会删除账户或帖子。这有两个原因。第一是保持我们数据库的完整性，第二是因为人们将来可能会从你的问题中学习。

### 我有点不确定自己在做什么…可以练习发帖吗？

绝对的。一个论坛应该有一个地方让你在冒险之前理清思路。在 SitePoint，这是一般的讨论论坛。甚至有一个新成员介绍自己的线程。请留意[bonjour，hello，g'day 和 howdy ' all 的大怪物线程！](https://www.sitepoint.com/forums/showthread.php?834403-The-BIG-MONSTER-thread-of-bonjour-hello-g-day-and-howdy-y-all!)

一旦你做了自我介绍，浏览一下[一般讨论](https://www.sitepoint.com/forums/forumdisplay.php?f=9)论坛。如果你厌倦了极客的谈话，或者你只是想抱怨一下你的周末，或者找到其他人分享你对迷你小马的热爱，这是一个不错的去处。

### 为什么我的帖子下面没有像其他人一样的链接？

这些链接在论坛世界中通常被称为*签名*。论坛签名的目的有两个。首先，它让人们有机会了解你。如果你就他们感兴趣的主题发表了一些非常有见地的帖子，他们可能会点击你签名中的链接来寻求你的服务。第二，它给你一个宣传你的产品的机会。

不幸的是，这第二个目的意味着人们有时注册论坛只是为了尝试链接构建——为 SEO(搜索引擎优化)目的创建一个网站的反向链接——并没有做出任何有意义的贡献。为了解决这个问题，大多数论坛都变成了禁止关注的论坛。只需一点代码，一个论坛就可以把所有的签名链接变成非关注链接；这意味着搜索引擎索引蜘蛛不会跟踪它们，也不会用它们来增强搜索结果。

签名链接是 SitePoint 论坛上最大的误解之一。签名链接不会给你任何 SEO 好处。遗憾的是，这没能阻止人们的尝试。因此，我们不得不实施一些限制。新成员必须等待 90 天才能建立签名。如果你在其他论坛上看到这个，他们可能有类似的限制。

### 有什么我应该特别避免做的事情吗？

事实上，有一些负面因素会真正伤害一个社区。我们要求你避免粗鲁或不耐烦。最重要的是，永远不要贬低别人。友善点。没那么难。

使在线社区与其他类型的社区非常不同的一个方面是，你从来不面对面交谈。这意味着没有身体语言信号的好处。正因如此，发明表情符号(或笑脸)是一件幸事。如果你认为你的帖子在某些方面可能会给人负面的印象，在末尾加上一个:)，以消除刺痛。的力量:)是现代通信的奇迹之一。

### 如果我认为规则被打破了，我该怎么办？

这是另一个伟大的论坛发明发挥作用的地方——旗帜张贴按钮。大多数论坛都有一个按钮或图标，您可以单击它来吸引版主对帖子的注意。您可能会出于多种原因这样做。你可能会看到一个粗鲁或违反规则的帖子。这可能是在错误的论坛，或者它可能是一个松散的职位(更多关于这些很快)。

在 SitePoint，每个帖子的左下方都有一个小小的橙色旗帜图标；在[图 5 中，“SitePoint 的旗帜贴吧图标”](#fig_flag "Figure 5. SitePoint’s flag post icon")你可以在左下角看到。永远不会有足够的版主来看每一篇帖子，所以我们依靠你——成员——来帮忙。只要你认为有必要，就使用“标记发布”按钮，如果你有疑问，就这样做。如果主持人决定不需要采取任何行动，那么就不会采取任何行动，所以不会造成任何伤害。

### 那么到底什么是绒毛贴呢？

啊，可怕的无聊帖子是任何论坛管理员生活中的祸根。绒毛是一个昵称，指的是那些低于会员认为的我们帖子质量标准的帖子。人们访问论坛是为了阅读有趣的帖子，并为这个地方做出贡献。如果一个论坛充满了纯粹为了减少链接或增加帖子数量的帖子，人们会很快放弃它。

因此，在 SitePoint，我们有一个帖子质量指南。我们承认英语可能不是每个人的第一语言，所以没有要求你的语法或词汇完美。我们只要求每个帖子:

*   有意义:有实质内容的建议
*   相关:切题且有用，而不是泛泛而谈的建议
*   新的:信息还没有说在线程

这意味着“谢谢分享”或“不错的帖子”恐怕不能达标。

### 哇，太多了！

是的，所以让我总结一下要点:

*   您可以通过阅读论坛指南或联系工作人员来获得帮助。参见我们的[常见问题解答](https://www.sitepoint.com/forums/faq.php)。
*   联系会员最简单的方法是使用[私人信息系统](https://www.sitepoint.com/forums/faq.php?faq=vb3_user_profile)。
*   保持耐心、支持和礼貌。
*   发帖前搜索—您的问题可能已经得到了解答。
*   避免仅仅为了自我推销而发布链接。SitePoint 是一个无关注论坛，所以不会有我们的反向链接。
*   给你的文章起一个描述性的标题。
*   发帖前思考；你的话会在网上留存很久。
*   不要害怕使用旗杆按钮。
*   确保你的帖子是原创的，有意义的，相关的。
