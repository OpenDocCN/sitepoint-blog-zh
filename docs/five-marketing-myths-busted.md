# 五大营销神话终结！

> 原文：<https://www.sitepoint.com/five-marketing-myths-busted/>

![magnifying_glass](img/08e28acfab15f1f2fc55d1275536946c.png "magnifying_glass")无论你是在开始一项新业务，还是在寻找发展现有业务的方法，都有大量的小型企业营销建议可以“帮助”你。找出什么可能适合你和你的企业本身就是一项任务——这是在你开始考虑如何实际应用专家的智慧之前。

事实证明，在付诸实践之前，不仅仅是古怪的建议或更具创新性的建议需要深入考虑。一些最基本的建议，我们很多人认为是理所当然的，也值得质疑。

以下是我认为每个企业主都应该质疑的五个商业营销观念。

**神话 1:越大越好**

许多常见的商业营销建议都假设你希望你的业务变得更大——你想要更多的客户，更多的员工，更大的服务，更大的项目，或者更高的薪水。大企业面临一系列小企业没有的挑战，从人力资源和公司文化问题到财务、税收和法律障碍。

许多小企业主发现，成长让他们远离了他们热爱的任务，并承担了他们没有灵感或兴趣的各种工作。成功不是由大小决定的；这是由让你快乐的事情决定的。

找一个适合自己的小众，充分利用，可能会更好；让你的小企业适应你的生活方式可能更好。如果你真的想一周工作三天，花时间想办法实现这一点可能会更好，而不是夜以继日地把你的生意打造成你内心深处并不真正想要的东西。

**误区二:大众营销是最好的营销**

要让一个小企业运转起来，你需要“走出去”，对吗？你需要“制造轰动”，成为“最受关注的人”，不是吗？

不一定。大众营销——通过报纸、电视、广告牌或网络——会导致一些严重的“浪费”(没有任何收获的曝光)。近距离接触通常更有效，而个人接触——面对面的交流、对合格潜在客户的直接营销等等——可能是你的企业以最少的支出赢得业务的最佳方式。

当你评估你的营销机会时，不要自动告诉你自己，接触尽可能多的人是你的目标。相反，看看你的目标受众，非常清楚地定义这些人，找出他们使用什么媒体，并确定哪些媒体和信息适合你的业务。

神话 3:没有详细的商业计划，你不会成功

详细的书面商业计划有他们的位置…但是你有吗？如果你是单飞，清楚地知道你能提供什么，你能提供给谁，你可能不需要任何正式的商业计划就能做得很好。

我知道许多自由职业者在没有商业计划的情况下快乐地工作。他们知道他们能做什么，他们知道他们能为谁提供服务，他们依靠这个简单的等式-加上坚实的联系基础和强烈的方向感-来指导他们的活动。

整理一份详细的商业计划的挑战可能会让有好想法的人放弃追求。做你的研究，不要冒不必要的风险，但是同样的，不要被商业计划的需求所阻碍。

**误区 4:你的营销必须“巧妙”才能吸引注意力**

巧妙的营销很酷。它能让我们感受到娱乐、逗乐、印象深刻和品牌的尊重。是的，理想情况下，所有的营销都会像这样恭维消费者。但现实是，营销不需要很聪明就能有效。

一名会计发送电子邮件，向过去的客户提供财政年度末纳税申报表的折扣，这不需要很聪明:她只需要很清楚。一个试图向新的企业客户推销他的品牌开发服务的设计师需要有一个好的投资组合，有帮助的联系人，以及一个专业的方法来展示他能为这些前景做些什么。华而不实的直邮广告和可爱的广告是“最好拥有的”，排在真正的必需品之后。

不要被让你的营销变得酷和聪明的明显重要性所消耗。让你过去的工作和不断扩展的技能为自己说话，有很多话要说(几乎…在你的一点个人帮助下)。

**误区 5:你需要了解营销才能成功**

你的生意不是营销；是关于人的。所以你需要理解别人才能成功。你需要了解的人是那些可能需要你的产品或服务的人。一旦你理解了这些受众，他们的动机、生活方式、媒体使用模式、人口统计和梦想，你自然就能以一种与他们对话的方式与他们交流，与他们的需求产生共鸣，并为你赢得他们的尊重和习惯。

随着你越来越了解你的听众，你会发现更容易预测他们想要什么，并以一种让他们感到被理解的方式解决他们的问题。这种理解将使你从竞争对手中脱颖而出。这也是让你的客户保持忠诚的原因——甚至可能促使他们支持你的业务。了解你的受众，确定接触他们的适当方式，你会发现你可以忘记把“营销”列为你的待办事项。

我认为这些是最大的营销神话，但你可能遇到过其他的。你最近改变了哪些营销建议？
