# 令人惊讶的是，年轻用户更关心隐私

> 原文：<https://www.sitepoint.com/surprisingly-younger-users-care-more-about-privacy/>

我对去年的一个预测是，隐私将成为主流用户越来越关心的问题。今年我没有重复这个预测，但也许我应该重复。原因？显然，年轻的网络用户似乎更关心隐私控制。或者至少，他们用得更多。

根据脸书首席隐私官克里斯·凯利的说法，[比成年人更多的青少年](http://www.technewsworld.com/story/Will-2009-Be-the-Year-of-Multiple-Digital-Identities-65768.html)在社交网络上使用隐私控制，比例约为 60%到 25-30%。这令人惊讶，因为传统观点认为年轻的互联网用户往往不关心他们数据的隐私。

来自 Computer Associates 的一项最新研究证实，许多青少年至少在某种程度上关心网络隐私。该研究表明，79%的 13-17 岁青少年是 MySpace 或脸书等社交网站的成员，他们以某种方式保护自己的个人资料免受普通互联网的影响(即，只允许朋友或朋友的朋友查看他们的信息)。

当然，脸书上的个人资料会自动受到保护，不会被整个互联网看到，但要保护它们不被你的网络其他部分看到，还需要一些额外的步骤。青少年更有可能利用脸书的细粒度隐私控制，这表明了导致相同结论的两件事情之一:

1.  青少年比成年人更关心网络隐私，
2.  青少年比成年人更了解社交网络的隐私控制。

如果我必须为年轻的脸书用户比年长的用户更倾向于使用隐私控制的一个潜在原因下注，我想后者更有可能。脸书的隐私控制没有得到很好的宣传，隐藏在多个菜单中。更年轻的用户可能更懂网络，他们可能会发现定位隐私控制更容易。前面提到的 Computer Associates 调查还发现，年轻人很可能会在网上发布自己的照片，这可能是他们更有可能寻求和使用精细隐私控制的另一个原因——他们有更多需要隐藏的东西。

由此可以得出的结论是，不管什么原因，隐私和隐私控制在未来将是一个越来越大的问题。年轻的网络用户将开始期望网站包括越来越强大的隐私控制，允许他们更全面地管理他们在网上发布的信息如何与世界其他地方共享。

特别是对脸书来说，这一统计数据也可能会鼓励任何试图进军商业社交网络的计划。因为青少年和年轻成年人使用脸书与朋友分享信息和图片，也就是说，很多时候你不想与你的专业网络分享的东西，该网站需要坚如磐石，易于实施，非常精细的隐私控制，才能真正用于任何类型的商业网络。如此多的年轻用户(工人阶级的未来成员)已经在使用这种控制，这对脸书来说是个好消息。
