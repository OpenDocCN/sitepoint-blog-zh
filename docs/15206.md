# 快速下载的快速提示

> 原文:[https://www.sitepoint.com/fast-tips-fast-downloads/](https://www.sitepoint.com/fast-tips-fast-downloads/)

**你是否因为网站太慢而失去客户？遵循这四个提示来确保:**

*   你的页面加载又快又简洁，
*   你的客户停留在你的网站上
*   你的访客到买家的转化率提高。

##### 1.保持你的主页简洁

我和数以百万计的其他网站所有者一起，习惯于把所有东西——包括厨房的水槽——放在我的主页上。一位客户告诉我，他们等了四分钟才下载我的主页，我才明白过来，开始剔除不必要的内容。事实上，我一直在筛选，直到我只剩下网站链接、一个介绍性段落和六个我最重要的合作伙伴的网站链接。

一般来说，我会试着设计我的主页，这样访问者不用向下翻页就可以看到所有的信息。为什么不像我一样，通过创建一个“最新消息”和一个“奖项”页面，将内容转移到网站的其他页面上呢？主页以外的页面可能是其大小的两到三倍，尤其是如果内容是由文本而不是图形组成的话。请记住，尽管许多用户有有线互联网连接，但仍有许多人在使用 56 kbs 和 33 kbs 的调制解调器，这些用户下载页面的等待时间不太可能超过 15 秒。

一个简洁快速的好主页的例子，见[HimalayArt.com](http://www.himalayart.com/)。

##### 2.让您的图形保持“轻盈”

对于图形和数字图像，我总是发现自己在质量和重量(即图形的大小，以 KB 为单位)之间进行权衡。有许多软件程序可以缩小你的图片尺寸(比如 JPEG Cruncher Pro T1 或 T2 GIF Cruncher Desktop T3)，但是无论你使用哪种图形程序，我的建议都是一样的:当你处理图片或照片时，总是以最高质量保存。然后，在项目发布到您的站点之前，最后一次保存项目，以大约 80%的质量保存它。此方法考虑到，如果您以 80%的质量保存一个项目五次，则每次保存时，该项目的质量会降低 20%。

我发现保留一份所有图形项目的主拷贝也很有帮助，这样你就可以随时回到你原来的最高质量的拷贝。我还有一个一般原则，就是不要制作任何高度或宽度超过 500 像素的图形。这使得查看者可以看到整个图形，而不用上下翻页，并将图形的重量保持在一个可管理的大小——通常在 20k 到 40k 之间。

一个使用照片非常好的网站是 AsianArt.com。注意他们照片的清晰度、大小和加载时间。

##### 3.保持你的 HTML 简洁

我发现当我处理网页时——不断地修改和更新 HTML 代码变得杂乱无章，充斥着空的、冗余的和嵌套的字体标签(例如，将是空的、冗余的和嵌套的)。

大多数 HTML 编辑器都提供了一个选项，通过删除空的、多余的和嵌套的标签来清理 HTML(在 Dreamweaver 中，转到“命令”菜单，然后选择“清理 HTML”)。这不仅清理了代码的外观，而且使页面加载更快，因为需要处理的代码更少。

此外，我喜欢一个叫做 HTML Shrinker 的小程序，它可以删除页面上不必要的 HTML，有时可以将代码缩减 80%或 90%。该程序还提供了备份原始代码的选项，我强烈建议这样做(有几次我的代码在压缩后无法正常工作)。

##### 4.让你的网站“缓存友好”

当访问者第一次看到你的主页时，他们的浏览器会下载页面上的所有图片。当它们继续进入你网站的更深层次时，许多浏览器会存储(或“缓存”)主页上的图片，并在遇到它们的每个页面上再次使用它们。这样可以避免在网站的每一页都下载相同的图片。

为了利用缓存，您的站点需要在每个页面上使用通用图形。例如，建立一个“按钮”目录，并在该目录中保存每个按钮的一个副本。在站点范围内使用该目录中的按钮。例如，我在整个网站上使用一个“后退”按钮。第一次遇到后退按钮，就会下载。以后每次遇到它时，浏览器都会使用按钮的缓存副本，这样可以节省下载时间。

##### 快速下载的快速提示

遵循这四个建议将帮助你创建快速加载的内容——一个有效网站最重要的元素之一。还记得那句老话吗，营销中最重要的三件事是位置、位置和位置？网络营销中最重要的三件事是速度、速度和速度。速度会让客户经过你的主页，让他们在你的网站上停留更长时间，并鼓励他们一次又一次地回来。
