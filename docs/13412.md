# 利用社区赚钱——第一部分，问题

> 原文：<https://www.sitepoint.com/making-money-with-communities-part-1-problems/>

正如我在简历中提到的，我是用户生成内容的忠实粉丝。在互联网出现之前，我在某处读到过，世界上的内容是由世界上 10%的人口创造的。我认为，很明显，随着在线发布各种内容的便利，以及创建内容的工具价格的下降，创作者的比例大大增加了。

作为一个来自计算机时代的 BBS 的人，社区和论坛一直是我特别感兴趣的。我的第一个网站是一个论坛。事实上，我最早的网站之一是从一个论坛开始的，后来发展成了 Lit.Org。具有讽刺意味的是，我后来为 later 推出了一个名为 WritingForums.com 的网站。我现在不拥有它们，但它们教会了我管理一个社区的所有陷阱。

在这个关于通过社区赚钱的系列文章的第 1 部分中，我们来谈谈社区面临的一些问题。

社区和论坛面临的主要问题是:

1.  害怕用户生成的内容。
2.  每个独立访问者的高页面浏览量(频率上限)。
3.  广泛的话题，通常非常笼统，如 CPC (Adwords，YPN 等。)没有那么有效。
4.  横幅盲(每个人都有这个问题)。

让我们谈谈如何克服这些。

1.害怕用户生成的内容

论坛所有者面临着与任何用户生成内容网站相同的问题——广告商不愿意将客户广告放在无人控制的内容旁边。广告商需要考虑客户的品牌和形象，他们不希望这种形象与可能损害这种形象的内容联系在一起。为了确保广告商对在你的网站上投放广告感到满意，你需要制定一些编辑政策。向他们保证你不允许色情，仇恨或非法内容。让他们知道您有一个检查违规行为的审查流程，以及报告和处理任何可能发生的违规行为的方法。你仍然可以有一个开放和自由的系统，你只需要有正确的基本规则和程序来执行它们。

2.每个独立访问者的高页面访问量

所有的广告商都使用频率上限，这仅仅意味着他们在给定的时间段内只给唯一的用户展示给定的广告一定的次数。比方说，在 12 小时内给用户“x”看 5 次“蓝色小部件”广告。根据你的广告来源，这个数字可能会有所不同，但是论坛显示的页面浏览量往往是其他网站的十倍，所以你会比其他网站更多地达到这个频率上限。解决方案是签下多个广告合作伙伴，并链接你的广告。链接您的广告，将一个网络的默认广告设置为另一个网络的广告代码。例如:部落融合默认为双击，默认为 Azoogle，默认为卡萨勒，默认为你自己的房屋广告之一。因此，当用户“x”在部落融合上封顶时，他们就会看到双击广告，等等。我总是喜欢把一个房子广告放在链的末端，要么在你自己的网站上推广一些东西，要么是一个联盟项目，或者是一个“在这里做广告”类型的广告。赚钱已经够难了，不要留一点在桌上。

3.广泛的话题

我艰难地了解到，尽管 Lit.Org 和 WritingForums.com 是高度针对性的主题(写作)，但这些网站几乎从来都不是关于写作的。奇怪的是，作家们并不怎么谈论写作。他们可能会讨论情节点或人物或技巧，但他们谈论的是**具体的情节点**，而不是一般的“情节”。此外，他们还张贴故事、诗歌和人物简介，将内容遍布地图。大多数论坛不会有像这个例子那样糟糕的问题，但我想你明白这一点。恐怕这个问题的解决方案必须在你启动你的社区之前得到解决。虽然普通内容仍然可以很好地处理 CPC 广告，但它永远不会像高度针对性的内容那样好。底线是不要依赖点击费作为你的主要收入来源。我将在第 2 部分中更多地讨论替代方案。

4.横幅盲

每个网站都面临这个问题。用户已经习惯于看到标准尺寸和形状的横幅。人们非常善于发现广告，并完全“忽略”广告。当你有任何 CPA(每次行动的成本)或 CPC(每次点击的成本)活动时，横幅失明真的会很痛。解决办法很简单，跳出“框框”思考。使用非标准的广告尺寸和非标准的广告位置。让它们尽可能地贴近内容，尽可能地融入其中。目标不是欺骗人们认为广告是内容，而是帮助他们注意到广告。让广告引起注意的第一法则是改变。用户往往讨厌变化，但他们会注意到变化。每次我移动我的广告，改变大小或格式，或者对它们的外观做任何明显的改变，我的广告就会表现得更好。这需要一些时间，但是如果你偶尔改变一下，你会从横幅广告中得到更多。

明天我将更多地谈论从社区和论坛赚钱的具体方法，其中大部分不涉及传统的横幅和上下文广告。一如既往，欢迎提出建议、意见和问题。我很想知道你在自己的社区中遇到了哪些问题。
