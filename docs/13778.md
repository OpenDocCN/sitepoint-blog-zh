# 竞争激烈的行业中的垃圾邮件制造者。

> 原文:[https://www . site point . com/spammers-in-a-cut-throat-industry/](https://www.sitepoint.com/spammers-in-a-cut-throat-industry/)

做一个网站发布者没那么容易。

我的一个最新站点是 [4 激光脱毛](http://www.4laserhairremoval.com)。我花了大约 2000 美元启动这个网站。内容是 100%独特的，我想把这个网站变成互联网上激光脱毛信息的事实资源。我能从中得到什么？模仿者，事实上我确信通过写博客我会得到更多。

此外，我的网站是有一个问题，在谷歌上市。子页面，比如类别页面和文章，会被很好地列出。谷歌工具栏显示的页面排名为 5 或 6。主页虽然保持了 0 的页面排名，但没有被列出。我已经向谷歌发出了重新收录的请求，但至今没有回应。这个网站当然从来没有违反任何规则，所以我不明白问题是什么开始。

所以今天我做了一些检查，有一种方法可以看出你的网站是否会被处罚，那就是搜索网站的真实名称，看看它是否会出现。所以我试着用[搜索](http://www.google.com/search?sourceid=navclient&ie=UTF-8&rls=GGLD,GGLD:2004-44,GGLD:en&q=4+laser+hair+removal)。(如果你没有看到我看到的东西，这意味着谷歌已经对我的垃圾邮件投诉采取了行动)。我没有列出来，但第二个列表是 cgi.tripod.com/hair-removal-product/cgi-bin/index.pl 的，如果你点击那个链接，它会转发到 http://www.hair-removal-products.net/的。看到相似之处了吗？

你也可以在重定向 URL 上进行[谷歌链接搜索](http://www.google.com/search?sourceid=navclient&ie=UTF-8&rls=GGLD,GGLD:2004-44,GGLD:en&q=link%3Ahttp%3A%2F%2Fcgi%2Etripod%2Ecom%2Fhair%2Dremoval%2Dproduct%2Fcgi%2Dbin%2Findex%2Epl),看到这个垃圾邮件发送者运行着一个由伪装网站组成的完整网络，虽然其中大多数我肯定没有侵犯我的网站，但我打赌他们侵犯了某人的网站。

通常我不太关心模仿者，毕竟模仿者是一种生活方式，我的大多数网站都曾被缺乏创意的模仿出版商抄袭过。然而，在这种情况下，我真的认为这个垃圾邮件发送者严重伤害了我。事实是，我的网站未能在其自己的名字上排名，显然是受到了处罚，但这个垃圾邮件发送者是#2？我只能假设谷歌糊涂了，在为这个斗篷人的行为惩罚我。这个人也在使用 Adsense，但是让我们希望他被踢出去，因为我确信谷歌并不打算支持那些向他们的搜索引擎发送垃圾邮件的人。

有一些人对他们的网站非常保密，他们不分享他们，他们不公布他们，他们不从论坛签名链接到他们。我从来不是那种类型的人，因为在大多数情况下，抄袭者永远不会做得像原版一样好，所以他们并不真正是一个威胁。但是如果这样的事情继续下去，我可能会不幸地变成他们中的一员。

这个故事的寓意是，认真考虑你是否应该向其他网站管理员宣布你的新网站，并始终保持警惕，追查内容盗窃，这种事情比你想象的更经常发生。我甚至没有刻意去寻找这样的人，但在过去的 36 小时里，我已经发出了三封来自 DMCA 的投诉。
