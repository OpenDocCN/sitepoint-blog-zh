# 歌剧能流行起来吗？

> 原文：<https://www.sitepoint.com/can-opera-become-popular/>

浏览器市场比以往任何时候都更加强劲。主流浏览器不下五款，分支项目[Flock](http://www.flock.com/)[Maxthon](http://www.maxthon.com/)等几十个。Web 开发人员对他们选择的武器充满热情，花费大量时间将软件调整到完美，并向愿意听的人宣传产品。

歌剧比大多数歌剧存在的时间都长。最初开始于 1994 年，该浏览器于 1996 年公开发布。第三版于 1997 年问世，因其比 Internet Explorer 和 Netscape 先进得多而备受赞誉。

Opera 一直处于创新的前沿。它无疑是最早支持以下功能的浏览器之一:

*   多种操作系统，包括 Windows、Mac、Linux、FreeBSD、Solaris、QNX、OS/2 和 BeOS
*   手机和 PDA 等小屏幕设备
*   任天堂 DS 和 Wii 等游戏机浏览器
*   web 标准、CSS、XML 和 WML
*   选项卡式浏览和 MDI 界面
*   语音控制浏览和阅读
*   鼠标手势
*   来自[Opera.com](http://www.opera.com/)网站的开发者工具和帮助

Opera 是轻量级和快速的。早期的版本可以放在一张软盘上，即使在今天，下载量也只有 5MB 多一点。这远远低于其他浏览器，尤其是当你考虑到它包括一个电子邮件客户端，也许是最好的集成 RSS 阅读器。与竞争项目不同，Opera 使用自己的 HTML 渲染引擎，该引擎以速度最快著称。

Opera 在 2003 年之前也是一个成功的商业产品；鉴于 Mozilla 和 Netscape 6 的早期版本非常糟糕，它是 IE 的主要竞争对手。第 7 版引入了广告，但从 2005 年的第 8 版开始，该产品已经没有广告，可以免费使用。

然而，尽管有如此多的赞誉和狂热的追随者，Opera 的使用率仍然远远低于 IE、Firefox，甚至是更新的 Safari 和 Chrome 浏览器。统计数据各不相同，但 Opera 不太可能拥有超过 2%的市场份额。它可能没有微软、Mozilla、苹果和谷歌的营销影响力，但 Opera 是众所周知的。那么为什么用的人这么少呢？

我不能声称有一个明确的答案，但我怀疑 Opera 的利基使用的原因包括:

1.  Opera 已经被许多用户安装了——尤其是开发者——但它可能不是他们的默认浏览器。我用几个版本做网站测试，但一般浏览还是用火狐。
2.  该接口与操作系统无关。Opera 很吸引人，但它看起来并不总是像一个本地应用程序。
3.  选择太多了。用户偏好可以被广泛调整，但是选项的范围和一些术语可能会令人困惑。
4.  扩展和小部件最近才开始出现。
5.  Opera 很棒，但是它没有提供什么令人信服的理由让你放弃你最喜欢的浏览器。

即使这些原因也不能解释为什么 Opera 没有变得更加成功。你用歌剧吗？你认为为什么采用率一直低于竞争对手的浏览器？
