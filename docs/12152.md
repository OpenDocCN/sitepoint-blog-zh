# 太阳微系统公司的下一步是什么？

> 原文:[https://www.sitepoint.com/where-next-for-sun-microsystems/](https://www.sitepoint.com/where-next-for-sun-microsystems/)

在 IBM 的收购谈判破裂后，太阳微系统公司的股价本周暴跌。董事会拒绝了每股 9.40 美元的报价，并终止了他们与 IBM 的独家谈判协议。

几个月来，孙一直在寻找买家。该公司已经挣扎了几年，正在裁员 6000 人。大多数分析家认为，它从未充分利用其核心技术，如 Java。在 20 世纪 90 年代中期，Java 被吹捧为桌面和 web 应用程序所需的唯一开发平台，但是性能问题、许可混乱以及竞争技术的成功导致了它在市场上的稳步发展。

IBM 显然是一个理想的候选人，这将是他们迄今为止最大的一次收购。他们的出价似乎很有吸引力:在谈判开始前，Sun 的交易价格是发行价的一半。虽然预计很快会有更多的消息，但仍不清楚为什么会被拒绝。

目前，Sun 的长期未来似乎不确定，这可能是网络开发人员的一个担忧。该公司是开源软件的大力倡导者，目前支持的项目有[OpenOffice.org](http://www.openoffice.org/)、 [MySQL 数据库](http://www.mysql.com/)和 [VirtualBox](http://www.virtualbox.org/) 。虽然这些应用程序可以生存下来，但如果没有 Sun 的投资，它们会继续发展吗？
