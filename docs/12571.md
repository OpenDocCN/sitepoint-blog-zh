# 如何众包你的研发

> 原文：<https://www.sitepoint.com/how-to-crowdsource-your-research-development/>

从戴尔的 [IdeaStorm](http://www.ideastorm.com/) 和雅虎的成功中得到启示，在该活动中，消费者投票选出他们希望在戴尔产品中看到的功能。s [建议板](http://suggestions.yahoo.com/)，英特尔和华硕都推出了 [WePC](http://www.wepc.com/) 。WePC 是一个众包项目，让人们对功能进行投票——类似于戴尔的网站——并通过描述他们希望看到的关键功能来设计他们所谓的“梦想 PC”，并调整滑块，让华硕知道哪些功能是最重要的，而不是其他功能。该网站的梦想电脑部分也有一个简单的绘图应用程序，允许用户勾画出他们的梦想电脑。

华硕是世界上增长最快的个人电脑制造商之一，他们的上网本 [Eee PC](http://en.wikipedia.org/wiki/ASUS_Eee_PC) 系列点燃了一个新的细分市场。到目前为止，该公司已经在 T2 售出了 400 多万台迷你电脑。

华硕营销总监朱姿霖表示，WePC 是“在创新和技术与消费者愿望之间架起一座桥梁的努力。“这种态度肯定会让用户眼花缭乱，但这种收集客户反馈的方式明智吗？

我们认为它非常聪明。在这种环境下，众包你的研发不仅能让客户感觉到他们正在影响他们喜爱的产品的发展方向，还能让他们围绕你的产品相互交流。这反过来加强了你的品牌认知。只要华硕遵循他们得到的一些建议，WePC 应该会加强他们的品牌，并将用户转化为粉丝。

# 如何跟随华硕的脚步

我们认为这是任何一家公司收集对其产品或服务的反馈的好方法，有很多公司可以帮你做到这一点。我们下面列出的八家公司可以帮助你收集客户反馈，排名不分先后。

*   **[idea scale](http://www.ideascale.com/)**–idea scale 基本上是戴尔在 IdeaStorm 上实现的平台的克隆。用户可以提交想法，其他人可以通过使用类似 Reddit 的拇指向上/拇指向下机制对提交的内容进行投票来加入。IdeaScale 上的免费计划允许定制徽标，因此适合商业使用。高级审核功能、私人门户和自定义网址需要付费(每月 15 美元起)。*   –featurelist.org 几乎完全专注于软件，与 IdeaScale 的不同之处在于，虽然他们提供品牌服务，但建议板是完全公开的，任何人都可以提交任何产品的功能想法(或为您的产品创建项目页面)。换句话说，你可能无法控制自己的产品页面。从用户的角度来看，该网站与 IdeaScale 几乎相同。*   **[fed vote](http://www.fevote.com/)**——与 featurelist.org 类似，fed vote 采取“任何人都可以为任何产品创建建议板”的方式。尽管这些网站是以用户为中心的，但先去那里宣传你的品牌是个好主意，然后回头看看，确保你在用户最舒服的地方与他们互动，不管是哪个网站。*   –collab andrate 是另一个建议板产品，尽管他们的宣传是它不仅可以用来调查客户的反馈，还可以用来调查员工和合作伙伴。*   **[CrowdSpirit](http://www.crowdspirit.com/)**——曾经是一个雄心勃勃的项目，通过生产和营销来众包消费电子产品的生产和设计，crowd spirit 已经重新推出了一个想法市场，类似于 SitePoint 在姐妹网站 [99designs](http://www.99designs.com/) 上运行的设计竞赛市场。创业者可以向 CrowdSpirit 社区提问(比如，“用我们的新 API 可以开发出什么样的杀手级应用？”)，而最佳创意会获得现金或奖品奖励。*   **[suggestion box](http://www.suggestionbox.com/)**——suggestion box 是功能最全、设计最巧妙的客户反馈板之一。该网站对建议采用五分制评分，而不是更传统的 Digg 式的拇指向上/向下投票。它的费用是 49.50 美元/月，或者 4 美元/月，用于非营利和教育用途。*   user voice–user voice 是一款设计非常精美的建议板应用，beta 版免费。他们通过提供一个允许消费者直接从你的网站上提出建议和查看主要功能请求的小部件，使自己在竞争中脱颖而出。我发现自己在网络上越来越经常地看到 UserVoice 小部件。*   ——Get Satisfaction 实际上专注于客户支持，但用户经常通过网站提出如何改进产品的建议。像老鹰一样监控顾客的满意度，让顾客知道你在倾听，这是个好主意。
