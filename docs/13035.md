# 。3 月 25 日至 4 月 1 日:OOXML 和 IE8

> 原文:[https://www . site point . com/net-on-the-net-March-25-April-1-oo XML-and-ie8/](https://www.sitepoint.com/net-on-the-net-march-25-april-1-ooxml-and-ie8/)

抱歉，这次更新有点晚，为了弥补，我增加了一天。本周有一些来自微软的非常有趣的故事。最大的新闻是采用了作为 ISO 标准的 OOXML。虽然这与网络没有直接关系，但它一直是一个颇有争议的话题，可能会产生一些广泛的影响。随着 OOXML 被 ISO 和 ECMA 采用，它为 MS Office 文档提供了一个开放标准。不足为奇的是，IBM 和其他一些公司在格式上存在一些问题，但这可能是微软将致力于发布更多标准的迹象。我知道我会很高兴看到一些与 RIA 开发相关的开放标准。

IE8 团队在 MSDN 发布了一份文件，列出了 IE 各版本的 [CSS 功能。这包括 IE8 测试版和 IE8 最终版本中计划的实现。浏览文档，似乎所有的 CSS2 伪类都将得到支持，CSS2 规范也支持生成的内容，这两者都有点问题](http://msdn2.microsoft.com/en-us/library/cc351024(VS.85).aspx)

最后，微软宣布它将在下一个版本的 Windows Mobile 中包含一个“桌面级”的网络浏览器。该浏览器将支持 Flash 和 Silverlight，这是 web 设计人员和开发人员在构建 web 应用程序时必须开始考虑移动设备的又一个迹象。

和上周一样，这里有一些新的教程可能会引起你的兴趣。Betrand Le Roy 写了一个很好的教程来提高 UpdatePanel 的可访问性，Mike Snow 写了一系列关于 T2 为 SilverLight 开发游戏的教程。

这就是本周的全部内容。我要去玩我的 Xbox 360 复古版。
