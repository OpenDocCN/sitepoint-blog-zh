# Codebase 推出基于 Git 的项目管理服务

> 原文:[https://www . site point . com/code base-launchs-git-based-project-management-service/](https://www.sitepoint.com/codebase-launches-git-based-project-management-service/)

![](../Images/3559eebf5fe54a1f3ab463ca91ff1f59.png "codebase-logo") [Git](http://git.or.cz/) ，Linus Torvalds 创建的去中心化版本控制系统，由于 Git 仓库主机 [GitHub](http://www.github.com/) 的成功，最近聚集了大量的蒸汽。GitHub 和它的竞争对手 SourceForge 一样，在托管开源项目方面做得很好，包括一些备受瞩目的项目，比如 Ruby on Rails、Capistrano、Merb、Prototype 和 Scriptaculous。但是对于那些需要更多私有源代码库的小团队来说呢？

GitHub 确实提供私有库，但是英国小型开发公司 aTech Media 推出了一项名为 [Codebase](http://www.codebasehq.com/) 的新服务，围绕共享 Git 库的理念构建了一个项目管理套件。Codebase 围绕基于 Git 的核心版本控制和源代码库添加了问题跟踪和里程碑、部署跟踪以及活动提要。

Codebase 在公测期间对 5 个项目免费提供 100MB 的磁盘空间，并计划在一月份运行，它为用户提供了每个新项目的无限代码库。可以通过在线代码浏览器浏览代码、提交和变更集，用户可以对任何变更或任何文件发表评论。

![](../Images/b0ec0d293944f296bf501a300e42fab4.png "codebase-screen")

该应用程序还有一个强大的内置标签系统，用于保持对问题的关注，并定义里程碑和版本目标。票证系统已经被捆绑到 Git 中，允许开发人员直接从提交中更新票证，从而节省时间并保持事情有序。

活动流使团队成员与新的提交、问题、里程碑或部署保持同步。

一旦测试期结束，Codebase 将使用免费增值模式运营，付费计划可能从每月 5 英镑开始。该应用程序的免费计划将提供 20MB 的空间和 1-2 个项目(仍有待确定)。虽然免费计划的 20MB 上限听起来不算多，但由于 Git 的[压缩](http://en.wikipedia.org/wiki/Delta_encoding)，这实际上是一个相当大的空间。
