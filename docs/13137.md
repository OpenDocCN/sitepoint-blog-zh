# MySpace 开发者平台发布

> 原文：<https://www.sitepoint.com/myspace-developer-platform-launches/>

本周早些时候，MySpace 试运行了 [Myspace 开发者平台(MDP)](http://developer.myspace.com/) ，对应用程序的安装次数进行了限制:

> 在这个沙盒阶段，你创建的应用程序将被限制在几个安装(目前是三个)。这一限制允许您在我们的最终用户正式发布之前，在真实环境中测试您的应用程序。应用程序也不会触发活动更新，并且在发布之前无法与最终用户直接交流。我们将经常发布关于功能和日期的更新，敬请关注！

CNET 的 Rafe Needleman 在发布会上出现。与他交谈过的开发者对新的 MySpace 平台持积极态度，不仅因为它为他们打开了一个观众群，还因为 MySpace 在其核心使用了 [OpenSocial](http://code.google.com/apis/opensocial/) :

> 虽然 MySpace 将向该平台添加自己的专有扩展，以使开发者能够访问其特定数据，但生态系统至少在一定程度上符合标准，这一事实应该使开发者能够更容易地将 MySpace 应用移植到其他平台。

如果你在社交网络应用开发领域，你最好查阅一下 [MDP REST API](http://developer.myspace.com/community/RestfulAPIs/intro.aspx) 和 [OpenSocial JS API](http://developer.myspace.com/community/myspace/opensocialref.aspx) 文档。
