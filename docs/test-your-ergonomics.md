# 噢，我的背！测试你的工作空间人体工程学(并保护你的背部)

> 原文：<https://www.sitepoint.com/test-your-ergonomics/>

当我们感到舒适和健康时，我们往往更有效率。对于我们的舒适和健康来说，最重要也是最容易被忽视的因素之一是我们工作场所的人体工程学。

如果你在家工作，你可能没有公司的指导方针来确保你的设置符合人体工程学。此外，由于时间和预算的限制，你可能无法尽你所能让你的身体和大脑舒适地工作。

**风险**

你认为你的家庭办公室很好是因为你感觉很好吗？不要停止阅读！即使你现在没有任何复发的疼痛或不适，你的工作场所可能仍然是不健康的。非人体工程学工作环境的一些长期影响，尤其是对于我们这些一天中大部分时间坐在电脑前的人来说，包括:

*   肌肉和关节的压力
*   整体疲劳
*   手、手腕、手臂、肩膀、脖子或背部疼痛
*   腕管综合症
*   视力和眼睛疲劳
*   腱病和滑囊炎(由过度使用和重复运动引起)

**测试**

以下是你工作时应该遵循的最基本的人体工程学准则。

| 是/否 | 我的电脑显示器正对着我坐的地方，放在一臂之外，我的眼睛看到显示器顶部下方大约 2-3 英寸的地方。 |
| 是/否 | 我工作区的照明是间接荧光照明，不会闪烁。 |
| 是/否 | 我坐在一把可调节的椅子上，椅子有较低的腰部支撑，双脚平放在地面上，大腿与地板平行。 |
| 是/否 | 我遵循 20/20/20 法则。这意味着我每 20 分钟休息 20 秒，看着至少 20 英尺远的地方。 |
| 是/否 | 我打字时手腕保持中立，肩膀放松，手肘放在身体两侧。 |
| 是/否 | 灯光或窗户不会直接照射到我的电脑屏幕上。 |
| 是/否 | 我在白天会有规律地起床伸展我的背、手、手腕、腿、脖子和肩膀。 |
| 是/否 | 我的桌子已经摆好，所以两边都有空间，可以方便地拿到后面的电缆、插头和电线等。 |
| 是/否 | 我工作时不会太用力点击键盘或鼠标。 |
| 是/否 | 我用肘部而不是手腕来移动鼠标，必要时使用键盘快捷键(或者我有轨迹球或操纵杆鼠标，限制必要的移动)。 |
| 是/否 | 长时间通话时，我会使用耳机、听筒或扬声器。 |
| 是/否 | 我工作时用文件架支撑我正在阅读的纸张。 |

**你的成绩**

如果你对以上所有问题的回答都是“是”,那么你在人体工程学方面就及格了，日常工作中的长期负面健康风险也会降低。恭喜你！如果没有，对你的工作空间做一些改变，看看它们有什么不同。
