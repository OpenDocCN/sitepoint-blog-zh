# 雅虎！发布音乐 API

> 原文：<https://www.sitepoint.com/yahoo-releases-music-api/>

雅虎！今天[宣布发布](http://ymusicblog.com/blog/2008/08/04/introducing-the-yahoo-music-api/)他们最新的 API:[雅虎！音乐 API](http://developer.yahoo.com/music/) 。音乐 API 让开发者可以访问雅虎！据该公司称，它是“艺术家、专辑、歌曲、视频、收视率等的目录”。它允许开发者与 Yahoo！音乐数据库通过“图表，搜索，相似性，流派，艺术家，用户推荐和评级。”

基于 REST 的 API 返回 XML、JSON 和 RSS 格式的数据，并使用 BBAuth 来允许 Yahoo！音乐用户通过外部应用程序访问他们的个人数据。API 的使用目前仅限于每天 5k 次查询。

大约一年前，雅虎！发布了一款脸书应用软件,允许用户分享来自雅虎的音乐视频通过脸书与朋友分享音乐。今天，他们把这个应用程序作为一个例子，你可以用新的音乐 API 来构建它。事实上，该 API 包括一个可嵌入的基于 Flash 的视频播放器，可以在第三方应用中使用。

尽管雅虎！尽管他们遇到了麻烦，但他们仍然是访问量最大的音乐网站之一，每月有 2100 万访问者。这是大量的音乐数据，现在可以通过第三方服务解锁。
