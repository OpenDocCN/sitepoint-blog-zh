# 你的动机是什么？

> 原文:[https://www.sitepoint.com/whats-your-motivation/](https://www.sitepoint.com/whats-your-motivation/)

每个人都会在职业生涯的某个时候陷入低谷。如果你经常独自工作，摆脱它会比你想象的更加困难。当这种情况发生时，提醒自己你在为什么而工作，你在努力完成什么，以及是什么激励你继续前进是很有价值的。这是一个有用的练习，不仅仅是在你墨守成规的时候，也有助于你在独自工作的日常挑战中保持专注。

**确定你的动机**

使用下面的列表来开始你自己的动机列表，当经济衰退来临时，或者你只是需要一个提醒来提醒你为什么这么努力工作。

*   家庭——你是否希望让你的家人为你的所作所为感到骄傲？你的工作允许你在经济上支持他们吗？
*   金钱——金钱可以在很多方面成为动力。这是压力下最能推动你的事情之一吗？
*   骄傲——对你自己和你的工作感到骄傲对许多人来说是一种强大的动力。是给你的吗？
*   外界期望——你是否担心满足别人的需求，或者如果你失败了，别人会怎么看你？
*   可信度——你是否把你接手的每个项目都视为巩固你可信度的一种方式？你的正直会驱使你做更多的事情吗？
*   目标——当你开始创业时，你有没有设定目标？你有让你坚持下去的梦想吗？
*   帮助他人——当你在工作中帮助他人时，你感到满意吗？它激励你继续前进吗？
*   激情——对你所做的事情的热爱给了你战胜困难的勇气吗？
*   后果——你害怕如果你不履行承诺会发生什么吗？
*   挑战——你能在挑战中茁壮成长吗？你有没有故意让你的时间表排得满满的，因为那是你工作最好的方式？

我最大的动力是我的家庭，对我的事业和声誉的骄傲，以及对我所做的事情的热情。这也有助于我在为自己设定长远目标的挑战中茁壮成长。

**重振你的动力**

我发现，当我离开工作很长一段时间，我有一个特别具有挑战性的一周，或者如果我的待办事项清单上有大量的任务时，我的动力就会滞后。当你遇到困难时，有很多方法可以帮你摆脱困境，但这里有几个我最喜欢的重新聚焦的方法:

*   感激你所拥有的和你已经完成的
*   专注于一个目标，用狭隘的眼光工作，直到你完成它
*   消除所有的消极，专注于积极的一面
*   强迫自己做清单上最应该避免的事情(剩下的事情看起来会很容易)
*   振作起来(音乐起作用！)

是什么激励着你，当你处于低谷时，你如何重新投入其中？
