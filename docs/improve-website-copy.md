# 5 种快速更新网站内容的方法

> 原文：<https://www.sitepoint.com/improve-website-copy/>

许多企业正在将他们的网站转向 CMS 平台，这样他们就可以定期快速提供新的内容。这是让访问者和搜索引擎蜘蛛回来的好方法。但是它不是解决无效复制的方法，尤其是在你的站点的静态页面上。甚至 CMS 网站也有静态页面——关于，服务，联系等。–内容可能很快变得陈旧，或者在应该让网站访问者采取行动时完全停止工作。

也许你只是把这些核心页面放在一起，你的统计数据显示这些页面没有很好地转换，或者你只是早就应该进行修改了。好消息是，更新你的网页拷贝不需要花很长时间，而且会有明显的效果。下面是一些需要考虑的更新。

## 更新你的个人资料

如果你最近获得了一个奖项，出席了一个会议或者完成了一个培训项目，你应该把它写进你的简历。但是，您不需要对您的证书进行重大更改，就可以重新访问此处的副本。页面是否快速总结了你背景和经历的亮点，展示了你的一点个性，让访问者很容易“理解你？”如果没有，修改这一页是一个好主意。

## 加入社交元素

如果你有商务用的 LinkedIn 账户、脸书粉丝页面、Twitter 账户或博客，你的网站应该有易于识别的链接或图标。如果你发布的主要是面向商业的内容，你甚至可能想把你的 Twitter feed 整合到你的网站中。

## 更改格式

视觉改头换面对你的文案可读性和趣味性的影响是惊人的。如果你有一个完整的段落，试着添加项目符号和副标题来打破它，使它更容易阅读。甚至像改变你的链接样式(颜色，装饰等)这样简单的事情。)，能让访客更愿意点击周边。

## 把它切成两半

一段接一段的文字很少有效。这只是太多的信息，你的访客可能不关心或没有时间通读。如果改变格式本身没有帮助，好好看看你的副本，问问自己是否真的需要那么多。可能有一种更简短、更直接的方式来表达这一点。

## 重复你的行动号召

你希望你的网站访问者做一些事情；如果没有，你就不会有网站。行动号召需要放在相关的每一页上。你希望他们安排一个电话，给你发邮件询问你的价格，选择加入你的名单吗？告诉他们，不要害怕重复。

你最后一次仔细查看你的核心页面是什么时候？你认为你的未来会焕然一新吗？
