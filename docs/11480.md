# 微软确认 IE9 开发

> 原文：<https://www.sitepoint.com/microsoft-ie9-development/>

开发者又爱又恨的浏览器正在重建。几周前，微软开始开发 Internet Explorer 9.0，并在洛杉矶的专业开发者大会上展示了一些新功能。那么我们能期待什么呢？

## 改进的 CSS 和 JavaScript

IE9 将提供 CSS3 选择器和圆角(是时候了)。JavaScript 性能也将得到改善，尽管微软在赶上竞争对手的浏览器之前还有很长的路要走。

## Direct2D (D2D)

IE9 将使用微软新的 Direct2D (D2D)系统支持硬件加速的文本和图形。D2D 利用 Windows DirectX 来改进客户端渲染。它将被默认使用，因此开发人员不需要更改他们的 CSS 或 JS 代码库。结果将是使用更少的系统资源获得更清晰的文本、更好的图形和更流畅的动画(大部分渲染工作将由图形处理器处理)。

## 发布日期？

IE9 的发布日期还没有宣布，但我怀疑至少还有一年的时间。更重要的是，我希望微软不要放弃对 Windows XP 的支持。该操作系统可能正在老化，但它仍然是最广泛采用的 Windows 版本。如果 IE9 只支持 Vista/7，这将是用户坚持使用 IE6 的另一个原因。

就我个人而言，我很高兴微软继续升级 ie 浏览器。是吗？或者它会简单地在你越来越长的浏览器测试列表中添加另一个版本的 IE 吗？
