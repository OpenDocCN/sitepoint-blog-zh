# Evernote 可以使用的 5 款应用

> 原文：<https://www.sitepoint.com/5-apps-you-can-use-with-evernote/>

昨天，我写了关于 Evernote 的一个[新功能 Trunk 的发布。Evernote 的新平台让笔记工具更加有用，包括 Evernote 扩展应用的新展示。](https://www.sitepoint.com/evernote-introduces-the-trunk)

我一直在探索后备箱里展示的一些应用，发现了一些不错的；有些我已经知道，有些对我来说是新的。以下是我发现的亮点。

## [![](img/4087935bd7d95ea431724f5543c3d050.png "gruml")格鲁姆](http://www.grumlapp.com/)

Gruml 是一款允许你在 Mac 上查看和管理谷歌阅读器订阅的应用。当你找到一篇你想记住的文章时，你只需点击一个按钮，它就会直接保存到 Evernote 中。

## [![](img/a0787afe996b33e39ef9d5385e3772de.png "nitropdf") Nitro PDF 阅读器](http://www.nitroreader.com/)

硝基 PDF 阅读器让您创建 PDF，评论和协作，创建和完成表格，并提取文本和图像。你可以从文件菜单中选择发送到 Evernote 选项，将你的 PDF 文件保存到你的 Evernote 账户中。

## [![](img/17fe6d498d50efb6c05a296a867a2e23.png "reqall")请求](http://www.reqall.com/)

reQall 是一个语音记忆辅助工具，它将您的手机、电子邮件、短信和即时消息集成到一个强大的组织者和提醒系统中。当你将 reQall 连接到 Evernote 时，它会自动搜索你的 Evernote 帐户，将相关项目添加到你的待办事项列表、提醒列表和日历中。

## [![](img/4bcd73d7fff00d3d30fd5265baaeaad1.png "ritepen")rite pen 4.0 Windows 版](http://www.ritescript.com/Products/rp-en-promo.aspx)

如果你有一个基于 Windows 的支持书写的设备，ritePen 可以让你在屏幕上的任何地方书写，包括任何文档或应用程序。然后你可以直接把它发送到 Evernote，它就变成了一个可搜索的笔记。(我个人很喜欢这款，因为我最近买了一台平板电脑。)

## [Seesmic](http://seesmic.com/seesmic_mobile/)

Seesmic mobile 让您随时随地查看和更新您的社交流。现在，当你浏览你的网络时，你可以把你正在阅读的任何东西直接发送到 Evernote。
