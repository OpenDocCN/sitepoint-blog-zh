# 使用对比字体以获得最大效果

> 原文：<https://www.sitepoint.com/use-contrasting-fonts-for-maximum-impact/>

字体设计不仅仅是选择一种字体。它是选择一种易读和吸引人的字体排列的能力。有四个基本原则可以帮助你最大限度地提高网站的可读性和外观:

1.  对比
2.  等级制度
3.  空间
4.  大小

这里，我们来看看对比。

当我们谈论排版中的对比时，它可以指几件事:

*   大号字体和小号字体的对比。
*   不同字体之间的对比，例如用于标题的脚本字体和用于正文的无衬线字体。
*   字体颜色和背景之间的对比度。

简而言之，对比度是决定文本阅读难易程度的一个重要因素。你想避免让你的网站访问者眯眼，或者当他们访问你的网站时觉得需要戴上太阳镜，你可以通过选择合适的前景色和背景色来做到这一点。

让我们看一个非常简单的好对比的例子。白底黑字。它非常容易阅读。
