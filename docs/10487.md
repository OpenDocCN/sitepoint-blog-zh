# 免费照片和矢量的搜索引擎

> 原文:[https://www . site point . com/freepik-the-search-engine-for-free-photos-and-vectors/](https://www.sitepoint.com/freepik-the-search-engine-for-free-photos-and-vectors/)

Freepik 是一个新的搜索引擎，它提供了一种搜索免费资源的方法，即照片和矢量图。对于预算很少(或为零)的网页或平面设计师来说，这似乎是一个非常好的资源。

根据创作者的网站，“Freepik 的目标是收集互联网上所有免费的高质量图形，并使访问它们**快速而简单**，这样你就不必浪费时间在数百个网站中手动搜索。”

**它是如何工作的？T3】**

Freepik 机器人每天都出去访问新网站，浏览免费照片和矢量。图像被评估并按质量顺序排列。使用 Freepik 时，搜索结果会提供一个链接，指向图像的原始位置。照片的所有者将获得全额积分和他们网站的链接。

那么它的效果好吗？

绝对的。我搜索了“Terrier ”,结果返回了一大堆(69 张)非常高到中等质量的照片和 11 张矢量图像。最棒的是，它们都是免费下载的。

[![SearchResultsThumbnails](../Images/4a20198716d80190eb002fd7492df9cf.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/11/SearchResultsThumbnails.jpg)

当你点击一张图片的缩略图时，你会看到许可证信息、图片来源、下载次数、图片标签以及尺寸和分辨率等信息。

[![SearchResult](../Images/d058062b3d672c09b27fd1d719842288.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/11/SearchResult.jpg)

[![VectorSearchResult](../Images/77606734642c25af8c2681a16f96be2a.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/11/VectorSearchResult.jpg)

我真的很喜欢主色功能，也许在搜索引擎的未来更新中，可以点击查看具有相似配色方案的相似图像。如果你不太确定你在找什么，并且有时间，你可以利用“新的”链接，最受欢迎的矢量和最受欢迎的照片，看看其他人在看什么。

作为一名设计师，我经常搜索图片，所以对我来说，这绝对是一个值得收藏和使用的网站。
