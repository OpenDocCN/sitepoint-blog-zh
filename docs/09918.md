# 标志设计中轮廓的力量和简单

> 原文:[https://www . site point . com/the-power and-simplicity-of-the-silhouette-in-logo-design/](https://www.sitepoint.com/the-power-and-simplicity-of-the-silhouette-in-logo-design/)

我们之前看了一些标志设计的流行趋势和风格。这些包括使用[彩虹色](https://www.sitepoint.com/logo-design-trends-the-rainbow/)和[立体派](https://www.sitepoint.com/using-cubism-in-logo-design/)。今天我们来看看标志设计中另一种流行的技术，那就是轮廓的使用。剪影显示了主体的形状，没有任何细节。因此，轮廓在标志设计中特别有用，是很好的标志。

风格可以定义一个组织的视觉和情感情绪，它是通过使用图像、字体和颜色来实现的。为了找到一个标志所需的风格，设计师需要研究谁是目标受众。这将是选择你将在一个标志中使用的符号和/或字体的一个主要因素。

设计标志时要记住的要点是:

*   保持简单
*   使其美观
*   用符号或合适的字体来表达一个想法或概念

如果我们牢记这些原则，就很容易明白为什么许多标志设计师会在他们的标识工作中使用轮廓。下面你可以看到美国国家篮球协会的标志。拿球的篮球运动员的反过来的轮廓很容易被识别。

[![nba-logo](../Images/852dee79e9153ca57bbeca2de0a31290.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/04/nba-logo.png)

美国职业棒球大联盟的标志遵循非常相似的风格。

[![MajorLeagueBaseball](../Images/c72d3c18289205c134ba8c98fb9652fc.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/04/MajorLeagueBaseball.jpg)

使用 Illustrator 或 Photoshop 很容易创建轮廓。你可以在现有的照片周围画一圈，或者扫描一幅画，然后画一圈，然后用黑色填充。重要的是，在画轮廓时要用一只精致的手，并选择一个合适的、容易辨认的物体来描摹。如果你不能告诉对象是什么，当它充满了一个平面颜色，那么你需要简化。

[![elephant](../Images/47a93b29486f0bc3b15da5e9e9cd6d9c.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/04/elephant.jpg)
