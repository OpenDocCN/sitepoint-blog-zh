# 网络股票:澳大利亚的血汗股票市场

> 原文：<https://www.sitepoint.com/webequity-australias-sweat-equity-marketplace/>

阅读创办和资助网络创业公司的新模式总是很有趣。几年前, [Y Combinator](http://www.ycombinator.com/) 用他们当时新颖的启动资金方式对微小、敏捷、年轻的团队进行小额现金投资，彻底改变了现状。自那以后，至少有 7、8 家种子基金效仿了 YC 的做法。总部位于澳大利亚的 [WebEquity](http://www.webequity.com.au/) 的目标是对网络创业的方式产生同样的影响。

WebEquity 是一个创意的市场，开发者、设计师和其他人可以在公平劳动或收入共享的基础上提供服务。“与传统的现金支付不同，网络股权社区成员将根据他们创造的业务的收入和/或股权份额获得奖励，”该网站的“关于”页面表示。

从本质上来说，WebEquity 是一个分类广告网站，任何有想法的人(在网站上被称为项目的“发起人”)都可以发布项目范围文件，并发出求助电话。作为回报，创始人可以向帮助他们实现想法的人提供股权或收入分成(或两者都有)。该网站不仅希望汇集开发商和设计师，还希望汇集律师、营销和销售人员、公关专家以及愿意为 sweat equity 提供服务的商业导师。

该网站是由悉尼，澳大利亚本地人保罗米德尔顿，谁是，当然，使用网络公平得到网络公平建设。“这个网站最初的灵感来自于我实现自己想法的需要——那是我希望我个人收入来源的地方，”他告诉澳大利亚技术报。

科技作家金·赫拉斯指出，WebEquity 的一个潜在障碍是在制定团队协议时明显缺乏支持。目前，米德尔顿打算将股权分配交易和知识产权的细节留给每个项目的合作者自己解决。Heras 认为这不是一个好主意，CrowdSpring 联合创始人 Ross Kimbarovsky 也同意这一观点。“这个模式看起来很有趣，挑战也很大——我 100%同意[Heras]的观点，即从一开始就制定一个团队合作和处理知识产权问题的标准协议是很重要的，”Kimbarovsky 说。

至于米德尔顿，他在 TechNation post 的评论中发表了自己的意见。“如果潜在的社区成员因为我没有提供更多的支持(例如，标准协议)而被吓跑，那么为了我自己的利益，提供这些显然符合我的利益，”他写道。“但这也为社区内的一些人提供了一个机会，他们可以通过确定这种需求来提供服务，谁知道这会给他们带来什么？例如，如果我是一名知识产权律师，我会迫不及待地在网站上注册，并开始参与我看到的任何项目。”

WebEquity 是一种有趣的启动资金和团队创建方式，我们祝他们好运。我们将密切关注该网站，看看有什么，如果有的话，来自它。目前，WebEquity 主要集中在澳大利亚，但米德尔顿向 TechNation 暗示，扩张是可能的(现在没有什么可以阻止澳大利亚以外的任何人注册)。
