# 当心在新窗口中打开链接

> 原文:[https://www.sitepoint.com/beware-opening-links-new-window/](https://www.sitepoint.com/beware-opening-links-new-window/)

Jakob Nielsen 把在新窗口打开链接的行为比作一个吸尘器销售人员，他通过清空顾客地毯上的烟灰缸来开始拜访。

现在，我不会说它有那么糟糕，但点击一个链接，突然出现一个新窗口，这可能会很烦人。

不幸的是，在新窗口中打开链接在网上仍然是很常见的事情。许多网站现在都这样做，我相信许多网站会继续这样做。在你跟随他们之前，花点时间认真思考一下这样做是否正确。

##### 为什么不在新窗口中打开链接？

以下是你应该小心在新窗口中打开链接的五大理由:

1.  除非您警告他们，否则 Web 用户很可能希望新页面在当前窗口中加载。意想不到的惊喜可能很有趣，但不是在你浏览网页的时候。

3.  打开新浏览器窗口的行为会重置该窗口中的后退按钮。后退按钮是第二个最常用的导航功能(在超链接之后，来源:[useit.com](http://www.useit.com/alertbox/990530.html))，所以重置它是一个大忌

5.  打开一个新的浏览器窗口会使非常新手的网络用户和视力受损的人迷失方向。他们可能没有意识到一个新的窗口已经打开，可能会努力在窗口之间切换。

7.  打开一个新的浏览器窗口不尊重你的用户的愿望。如果他们想要新窗户，他们会要求的。除非有充分的理由，否则不要强迫用户使用新窗口。

9.  新的浏览器窗口会使已经混乱的任务栏更加难以使用。我们都花了很长时间在任务栏中搜寻我们想要的窗口。不要通过增加用户打开的窗口数量来增加这个过程的难度。

##### 但是使用新的窗口可以让用户留在我的网站上！

不是靠它自己，它没有！网络用户会停留在一个网站上，因为它有他们正在寻找的信息，或者因为它帮助他们实现他们的目标——而不是因为浏览器窗口仍然打开着。

如果用户想返回网站，他们可以使用后退按钮。如果使用一个新窗口，这个窗口中的后退按钮将被重置，这样用户就不能使用这种常用的方法返回到你的站点(提示沮丧的用户)。

##### 我该怎么办？

简短的回答是:在你打开一个新窗口的任何链接之前，要深思熟虑。如果你仍然认为使用新窗口是个好主意，那么你有两个选择:

1.  ***Warn the user that the link will open in a new window***
    If you warn the user that the link opens in a new window, it won’t come as a nasty surprise. There are a number of ways you can do warn users:
    *   使用 href title 属性让用户知道链接将在新窗口中打开:

    ![1443_SPnewwindow](../Images/6356ffcb6417f82c80a0541a32f1e8b1.png)

    *   Use text to indicate that the link opens in a new window:

        ![1443_link1](../Images/359e081f1f2402cf4b83c37dc49e9b45.png)

    *   Use a small icon and text to indicate that the link opens in a new window:

        ![1443_link2](../Images/6907708bc80c54f3250f004152c16610.png)

    就可用性和可访问性而言，使用带图标的文本通知是最好的解决方案。图标比文本本身更容易被发现，如果用户不确定，文本可以让他们知道图标的意思。文本也将被屏幕阅读器读取，因此视障用户将被警告链接将在新窗口中打开。

3.  ***Provide an icon that opens the link in a new browser window***
    This solution is used by websites such as [Yahoo!](http://www.yahoo.co.uk) and [AOL](http://www.aol.com). Yahoo! provides an icon to the right of a hyperlink, which opens the link in a new browser window:

    ![1443_YahooSPlink](../Images/2dec8c5d6eca635a18c180c8962541b3.png)

    虽然这是一个好主意，因为它让用户自己选择，但不幸的是，还不清楚链接文本(即“网站点:新文章...”)是否会在新窗口中打开。这就是为什么如果链接文本也在新窗口中打开链接页面，那么包含诸如“(在新窗口中打开)”这样的文本很重要。

##### 什么时候可以在新窗口中打开链接？

信不信由你，有些情况下使用新窗口是个好主意，例如:

*   该链接用于文档，如 PDF 或 Word 文件。打开新窗口将允许图像或文档在后台下载。它还可以防止用户在关闭文档时意外关闭浏览器窗口。
*   该链接是一个大图像。在这种情况下，新窗口允许网络用户在下载图像时保持浏览器窗口打开。
*   该链接是一篇文章或网页的可打印版本。这里，一个新窗口允许用户在后台打印文章或页面时保持当前窗口打开。

##### 结论

在你打开一个新窗口的链接之前，要深思熟虑。大多数时候，在当前窗口中打开链接是目前最好的解决方案。如果你需要在一个新窗口中打开链接，至少要提前警告用户——当然，除非你想让你的站点访问者得到一个不受欢迎的惊喜！
