# 如何获得优秀客户的推荐

> 原文:[https://www.sitepoint.com/how-to-get-great-testimonials/](https://www.sitepoint.com/how-to-get-great-testimonials/)

推荐书是证明你在销售文案中所宣称的卓越的一个很好的方式。虽然通常存在隐含的偏见——例如，没有多少人会在自己的网站上发布负面评论——但对于自由职业者、小企业主或其他在线营销专业人士来说，推荐有很多好处。

一份好的推荐可以帮助一个观望的潜在客户克服一个恼人的销售异议，从而把他们推向你的方向。客户评价还可以进一步描述你做了什么和你是如何做的，同时分享你通常合作的客户类型的信息。此外，分享真实姓名和真实经历的证明会增加你的可信度。

你的推荐目标应该是向潜在客户提供关于你和你的工作的切实的(即无瑕疵的)信息。你的证明应该与潜在客户相关，可信，能产生信任，并支持你的陈述，即为什么客户应该雇用你而不是下一个人。

这里有一些从你的客户那里获得优秀和有效证明的方法。

## 选择合适的客户

不是每个客户都是有效证明的合适人选。将你的证明请求限制在与你合作了一段合理时间的客户身上，对你做的事情和你的目标有所了解，并且愿意被公开确认为客户。理想情况下，你会有几个客户，他们对你、你的服务以及他们从与你的合作中获得的东西有很好的了解；这些客户应该是你的首选。

与你的客户分享一些背景信息也是一个好主意，比如你打算如何处理客户评价，以及当你提出要求时，你希望它们会产生什么样的影响，以此为基调。

## 使用调查表

有些人喜欢以自由形式分享。其他人需要一点鼓励才能开始。您可以使用调查表或问卷让客户开始，并在他们讲述自己的故事时给予指导。

一个标准的格式也给你一点帮助，当涉及到汇编多个证明。您甚至可以使用多项选择调查，并将实际结果作为报告卡发布，以支持客户评价。

## 添加利息

你的推荐书不必是直截了当的文字。如果你有愿意的客户，把你的证明作为采访来进行(或者更好的是，让其他人来进行采访)，并把它们录制成音频或视频片段。你也可以用客户的头像、作品的例子和其他支持材料来支持评论。

## 保持简短和甜蜜

你的推荐不需要很长就能有影响力。如果你有基于文本的证明，用简短的段落和带项目符号的文本使它们易于浏览。如果你使用音频或视频，目标是容易理解的片段——在大多数情况下，三分钟或更少是一个有效的长度。

## 运行 FTC 检查

在发布你的推荐之前，确保你遵守了联邦贸易委员会关于推荐和推荐的指导方针。一些方法可以做到这一点，包括确保你只使用当前客户的推荐，给你的客户一个机会来编辑或删除他们的推荐，只发布真实的付费客户的推荐。请参考 [FTC 指南(PDF)](http://www.ftc.gov/os/2009/10/091005revisedendorsementguides.pdf) 获取官方指南。

理想情况下，你的推荐书不会取代，但会支持你已经做的销售文案和投资组合，向潜在客户展示你能做什么，以及这对他们个人有什么帮助。一旦你有了这些证明，你就有很多方法可以在你的营销工作中使用这些证明。
