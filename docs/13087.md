# 和你的 Ask.com 吻别吧

> 原文：<https://www.sitepoint.com/kiss-your-askcom-goodbye/>

Ask.com 解雇了 8%的员工，并计划改变其搜索引擎背后的策略，转而专注于女性用户群，这一消息让巴里·施瓦茨的写了一篇很好的文章。

> Ask.com 正被重新定位于打造……一个为搜索健康和娱乐的女性量身定制的问题解答引擎。

正如[别人理直气壮地问](https://www.sitepoint.com/forums/showthread.php?t=535622)的，有什么大不了的？在 Ask Jeeves 的时代，能够输入冗长的问题并让可爱的插图管家回答这些问题是一个很好的新奇事物，但他们曾经是谷歌的严重竞争对手吗？

SitePoint 的 John Conde 总结了[为什么谷歌的竞争至关重要](https://www.sitepoint.com/forums/showpost.php?p=3744457&postcount=12):

> 搜索领域的竞争对手越少，就意味着现有玩家创新的理由越少。如果谷歌向 Ask.com 提供搜索结果(这是意料中的事)，而微软收购雅虎，那么搜索领域就只剩下两家公司了。鉴于雅虎和 MSN 迄今为止在与谷歌的竞争中表现不佳，垄断很有可能出现，创新也会放缓。

对 Ask 的核心用户来说是好消息，对搜索来说是坏消息。
