# 你能想象 20 年后的网络吗？

> 原文:[https://www.sitepoint.com/imagine-the-web-in-20-years/](https://www.sitepoint.com/imagine-the-web-in-20-years/)

八十年代末！迈克和机械师们在图表的顶端，老布什刚刚成为美国总统，一个名叫蒂姆·伯纳斯·李的欧洲核子研究中心的承包商正忙着写一篇题为*信息管理:一个提议的小论文。在这篇文章中，他描述了一种简化不同地点的人们分享信息的方法。他把它给了他的经理迈克·森道尔，他认为这是“模糊的，但令人兴奋的。”*

接下来的一年，蒂姆和他的同事罗伯特·卡里奥完善了这个想法，并更新了提案。它描述了一个叫做“万维网”的概念——一个浏览大量信息的简单界面，使用超文本链接文档。在短短几年内，它的发展已经远远超出了欧洲粒子物理研究所和学术领域，由于这种新的、更直观的界面，公众对互联网的使用激增。

为什么要上历史课？因为蒂姆·伯纳斯·李发表第一篇论文的日期，可以说是网络诞生的日期，是 1989 年 3 月 13 日。那是二十年前的上周五——对我来说，这是一个值得庆祝的理由！

想想从那以后网络发展了多少真是令人惊讶。我们已经看到:

*   一些最早的浏览器，如[万维网浏览器](https://www.w3.org/People/Berners-Lee/WorldWideWeb)、[大提琴](http://web.archive.org/web/20040217003215/http://www.law.cornell.edu/cello/)和[马赛克](http://home.mcom.com/)
*   早期关于 HTML 的[语义的争论(我们错在哪里？)](http://1997.webhistory.org/www.lists/www-html.1994q4/0015.html)
*   JavaScript 的首次出现及其[惊人的新特性](http://web.archive.org/web/19961027095935/www3.netscape.com/misc/nojavascript.html)
*   [进入电子商务的第一步](http://www.grokdotcom.com/wp-content/uploads/Bryan/Amazon_first_page.jpg)
*   浏览器战争(网景现在！)
*   网络泡沫及其随后的内爆
*   一些被称为谷歌的后起之秀
*   第[篇博客](http://web.archive.org/web/19990508053702/http://www.robotwisdom.com/index.html)
*   [YouTube](https://www.youtube.com/)
*   社交网络的开端，像 Orkut 和 T2 Friendster 这样的网站
*   手机、手持设备和小型笔记本电脑上的移动网络

现在很难想象没有网络生活会是什么样子。对我来说更难理解的是:如果没有网络，没有网络开发者、管理者、系统管理员、作家或设计师，我们会做什么？二十年前，我八岁，幻想着发明新的早餐麦片；我对电脑最激动的经历要感谢一个叫做“奶奶的花园”的游戏。包括我在内，我的朋友和家人都没有预料到我的职业生涯会是现在这个样子。

网络的下一步是什么？再过 20 年，我就快 50 岁了，我不知道那时会发生什么。不过，从短期来看，我认为我们会看到许多有趣的东西出现。在未来几年里，我很高兴看到以下三个增长趋势。

更多基于网络的应用程序:基于网络的应用程序已经存在了一段时间，所以这是显而易见的，但我认为一旦流行的应用程序成为主流，它就会大受欢迎。想想像 [280 幻灯片](http://280slides.com/)、[鸟舍](http://aviary.com/)图形应用、[谷歌应用](http://google.com/apps/)和 [Photoshop Express](https://photoshop.com/express/) 这样伟大的网络应用，我认为公平地说，我们正在走向这样一个世界:每个人——不仅仅是我们这些极客——都发现在线操作和存储文件很容易。随着这些应用程序变得越来越简单，越来越令人愉快，Joe Sixpack 越来越有可能采用它。最重要的是，像你我这样的 web 开发人员越来越有能力创建出色、有用和独特的应用程序。

**安全将成为一个更大的问题:**似乎每周都有另一个关于在线服务安全问题的故事——无论是[恶意的脸书应用程序、](http://blog.trendmicro.com/new-variant-of-koobface-worm-spreading-on-facebook/) [暴露在世人面前的私人文件、](http://googledocs.blogspot.com/2009/03/on-yesterdays-email.html)还是[受损的在线银行服务](http://news.zdnet.co.uk/security/0,1000000189,39539045,00.htm)。随着我们越来越多地在网上开展个人业务，随着越来越多的人在口袋里随身携带小型网络设备，这个问题将*出现在每个人的脑海中——不仅仅是安全专家，普通人也是如此。我认为改变的最有力证据将是更多的教育，以及人们对安全的态度的巨大转变。然而，从技术角度来看，现在是时候了(现在是*时刻！*)获得有用、友好的工具，使日常消费者和企业更容易保护他们的身份和数据。*

主流媒体拒绝消亡:最近，长期大声地(或许还带着一丝沾沾自喜)宣称主流媒体的世界末日就要到了，这似乎很酷。但这与事实相去甚远——尽管主流媒体作为一个整体可能会不时地被新方式绊倒，但该领域的领导者现在表明，他们已经准备好拥抱新媒体及其提供的一切。这里有一个最近的证据:英国报纸，*《卫报》，* [刚刚发布了一个 API](http://www.guardian.co.uk/media/2009/mar/10/guardian-open-platform) ，公开他们的新闻内容，免费混合和重复使用。开发者已经在使用这些数据来创建新闻的地理可视化。这是主流媒体接触新媒体的一个很好的例子，我认为这种趋势将继续增长。

就像我说的，我不可能预测二十年后会发生什么，但我确信这一点——2029 年，我希望能像现在一样对网络感到兴奋。你呢——你要一起去兜风吗？你认为未来二十年网络会是什么样子？
