# 不要忘记提供价值

> 原文：<https://www.sitepoint.com/dont-forget-about-providing-value/>

在这里，我们继续对各种赚钱网站的案例研究，这次是通过讨论价值。

我只想说明，在你提供的任何直销产品中(以及你做的任何事情)，关键是提供价值。你的一些评论谈到每分钟都有一个“傻瓜”诞生。如果你有这种态度，你就不会成功。

从你的角度来看，某种产品可能没什么价值。但是一个特定的目标受众可能会发现你所提供的东西很有价值。

例如，我曾经与一家利润惊人的公司合作，销售收藏品，如收藏盘、瓷娃娃、皮面书籍和装饰品。你不会相信这家公司有多赚钱。我仍然不相信这家公司过去(现在)有多赚钱。我认为老板每年拿回家大约 3500 万美元，尽管这只是一个估计。

当我第一次到达时，我认为这些产品是愚蠢的，是“每分钟都会诞生的东西”老实说，我现在也是。但是公司的领导非常重视他们的产品，顾客也是如此。他们非常努力地寻找和测试顾客会喜欢的产品。我每个月都在那家公司接听客户服务电话，客户们喜欢他们的洋娃娃和盘子。对他们来说，他们是在购买传家宝。该公司狂热地跟踪客户投诉，并让客户参与决定下一步开发什么产品。

同样，在我们用于本案例研究的网站中，您个人可能看不到所提供的产品有多大价值。但是也有一些客户群希望从专家那里获得好的、简明的、易于学习的信息。当他们试图过上更好的生活时，他们也需要一些支持和帮助。这就是这些计划所提供的内容，它们旨在帮助客户获得价值并取得成功。

因此，我们与这些客户保持密切联系，确保他们喜欢内容并感到满意。这很好，因为我们得到了改进现有产品的想法，也向客户展示了我们将付出额外的努力来帮助他们获得他们来网站想要的东西。

你只需要对你开发的任何网站或产品采取这种态度。
