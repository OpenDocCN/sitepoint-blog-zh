# CSS 设计教程

> 原文：<https://www.sitepoint.com/css-design-tutorials/>

Veerle Pieters 出版了一个题为“设计 CSS 模板”的系列教程，它与现有的许多教程相比有了令人耳目一新的变化，它首先关注设计，而在最后才讨论使用 CSS 的实现。有四部分可选:[第一部分](http://veerle.duoh.com/comments.php?id=208_0_2_10_C)、[第二部分](http://veerle.duoh.com/comments.php?id=213_0_2_5_C)、[第三部分](http://veerle.duoh.com/comments.php?id=218_0_2_0_C)、[第四部分](http://veerle.duoh.com/comments.php?id=221_0_2_0_C)。

说到设计，我今天遇到了一个很好的 CSS 设计的例子，Bastiaan Terhorst 的[生动化](http://vivified.net/)。至少可以说是惊人的。想想就在一年前，人们还在认为 CSS 导致的设计作品是四四方方的，没有创意的。时代确实变了。
