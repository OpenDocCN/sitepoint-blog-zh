# 邮件烂；Mozilla 想要修复它。

> 原文:[https://www . site point . com/email-sucks-Mozilla-wants-to-fix-it/](https://www.sitepoint.com/email-sucks-mozilla-wants-to-fix-it/)

![](../Images/235918a82fc603040478923a6c875223.png) Mozilla 已经推出了一家衍生公司， [Mozilla Messaging](http://mozillamessaging.com/) ，称其使命是改进“互联网的杀手级应用” **email** 。

> Mozilla Messaging 最初的重点是开发 Thunderbird 3，这将带来显著的改进，特别是集成的日历、更好的搜索和对整体用户体验的增强。Thunderbird 是一个免费的开源电子邮件应用程序，全世界有数百万人使用，它是使用与屡获殊荣的 Mozilla Firefox Web 浏览器相同的开源开发模型构建的。

大约 7 年来，Thunderbird 一直是我选择的邮件客户端(几年前我创建了一个功能的愿望列表，其中一些已经被合并)，所以我很高兴看到它得到这种支持。雷鸟有它的怪癖，就像每个邮件客户端一样。是的，有一些功能非常缺乏(日历集成就是其中之一)，但是当你考虑到可靠性、互操作性、垃圾邮件过滤和价格等因素时，你就是无法打败它。

有了一个全职的、付费的开发人员团队和一个由用户、测试人员和开发人员组成的了不起的社区，我很期待看到这个项目的成果。雷鸟最终可能会走出火狐的阴影。
