# 获胜的 DNA

> 原文:[https://www.sitepoint.com/winning-dna/](https://www.sitepoint.com/winning-dna/)

在 2011 年 Web Directions South 的前夕，我们很兴奋地宣布了 [BuildMobile WDS11 挑战赛](https://www.sitepoint.com/win-a-ticket-to-web-directions-south/)的冠军，其中那些足够勇敢的人被要求单独使用 HTML、CSS 和 Javascript 的 Web 标准三连胜来构建一些小而美的东西。

事不宜迟，请和我们一起为斯蒂芬·休金斯和他的作品《DNA》鼓掌。

我们将在 [#wds11](https://twitter.com/#!/search/%23wds11) 期间与 Stephen 保持联系，以确保他享受到免费参加 Web Directions 的奖励。当我们移交“[为智能设备建立移动网站和应用](https://www.sitepoint.com/books/mobile1/)”和“[用网络标准开发](http://www.amazon.com/Developing-Web-Standards-John-Allsopp/dp/0321646924)”时，我们还将举行一些即兴签售会。

我们认为你会同意，这是一件杰出的作品。如果我们非常幸运的话，斯蒂芬将会写一篇关于这一切是如何联系在一起的解构，现在请[查看来源](https://www.sitepoint.com/winning-dna/)。请在评论中留下你的问题。

如果你还没有注册参加 [WDS11](https://secure.webdirections.org/wds11/aff/build) ，那么*仍然*为时不晚。**更新**:现在*已经*晚了。
