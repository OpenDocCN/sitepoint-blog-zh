# 新闻专线:W3C 启动 HTML 工作组

> 原文:[https://www . site point . com/news-wire-W3C-launchs-html-working-group/](https://www.sitepoint.com/news-wire-w3c-launches-html-working-group/)

*   [W3C 重新启动 HTML 活动](https://www.w3.org/2007/03/html-pressrelease)由微软的克里斯·威尔逊和 W3C 的丹·康诺利共同主持的 W3C 新 HTML 工作组今天启动。该组织的目标是在 2010 年推出下一个 HTML 推荐标准(从现在起不到 4 年)，并与 WHATWG 的努力相融合。(标签:[标准](http://del.icio.us/sitepointlinks/standards) [html](http://del.icio.us/sitepointlinks/html) )

*   [W3C 重启 HTML 努力](http://blog.whatwg.org/w3c-restarts-html-effort)谷歌的伊恩·希克森代表 WHATWG 对 W3C HTML WG 的启动发表评论。"令人惊讶的是，在特许过程中，W3C 实际上从未联系过(我们). "他还就如何参与新工作组提供了指导。(标签: [html](http://del.icio.us/sitepointlinks/html) [标准](http://del.icio.us/sitepointlinks/standards) )
