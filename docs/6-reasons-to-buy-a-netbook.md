# 购买上网本的 6 个理由

> 原文：<https://www.sitepoint.com/6-reasons-to-buy-a-netbook/>

自从华硕在 2007 年末推出 Eee PC 以来，上网本已经成为硬件市场最大的领域之一。华硕的上网本系列取得了惊人的成功，早期型号很快就销售一空(我试图购买一台，但没有成功)。每个主要的个人电脑制造商现在都生产上网本，而且市场还在继续扩大。

上网本通常使用较慢的硬件和较旧的软件，这使得它们非常适合儿童或那些对计算要求不高的人。然而，他们也可以为网络开发人员和自由职业者提供一个理想的个人电脑设置。

**1。便携性**
和许多人一样，我使用全尺寸笔记本电脑来完成我的主要开发任务。然而，我有如此多的附件，它还不如一台台式电脑。如果我想移动机器，我需要拔掉第二个显示器、外部硬盘、键盘、鼠标和各种电缆。即使这样做了，机器仍然太重，无法长时间携带。

我现在在访问客户网站时会带上一台上网本。它有我所有的基本软件，可以像普通笔记本电脑一样插入投影仪或显示器。没有必要拆卸我的办公硬件，这个设备可以整天携带而不会引起疝气。

**2。沟通和快速任务**
上网本非常适合查看电子邮件、运行 Skype 或执行快速任务。大多数上网本都支持 wifi，并配有快速操作系统，如 Windows XP 或 Linux 的优化版本。这些设备可以比大多数 Vista 笔记本电脑启动更快地启动、使用和关闭。

顺便说一句，不要害怕购买 Linux 驱动的上网本。一旦你从最初的*“这不是 Windows”*的震惊中走出来，你会发现 Linux 是一个高性能、高可用性的操作系统，只要你安装 [Wine](http://www.winehq.org/) ，它就可以运行 Windows 软件。即使你不喜欢所提供的操作系统，你也可以下载并安装一个替代品，比如优秀的 [Ubuntu Netbook Remix](http://www.ubuntu.com/getubuntu/download-netbook) 。

**3。在移动中工作**
手机和黑莓设备很有用，但是小屏幕、小键盘和有限的软件使得很难做*真正的*工作。大多数上网本功能强大，足以运行 Apache、PHP、MySQL、OpenOffice 等软件，以及简单的 IDE 或编辑器，如 [Notepad++](http://notepad-plus.sourceforge.net/) 。我甚至认识运行 Photoshop 的人(尽管他们更多的是为了证明一个观点，而不是做设计工作！)

**4。摄影**
大多数上网本都内置了存储卡读卡器。这使得携带上网本进行拍摄、复制/备份文件以及在更大的屏幕上查看照片变得非常容易。大多数上网本都有网络摄像头，所以即使没有摄像头也可以拍几张照片。

**5。备份和同步**
如果你的主电脑出现故障，上网本可以为你提供另一台计算设备。几个月前我也遇到过这种情况，但我能够继续工作。

像 [Dropbox](http://www.getdropbox.com/) 这样的软件对于在你所有的电脑之间备份和同步文件也是非常宝贵的。

**6。价格**
第一台华硕 Eee PC 的价格是同等大小机器的五分之一。虽然硬件规格和价格有所提高，但 200 美元买一台上网本还是有可能的:远不如一部像样的手机。你甚至可以用它来代替 Kindle！

更换上网本不会导致严重的现金流问题，所以你不会担心丢失或损坏设备。
