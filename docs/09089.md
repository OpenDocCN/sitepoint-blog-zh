# 非营利不代表没钱

> 原文:[https://www.sitepoint.com/non-profit-doesnt-mean-no-money/](https://www.sitepoint.com/non-profit-doesnt-mean-no-money/)

我经常谈到[为你的客户提供价值](https://www.sitepoint.com/so-what-exactly-is-value-and-how-do-i-use-it-to-sell/ "So What Exactly is “Value” and How Do I Use It to Sell?")和[投资回报](https://www.sitepoint.com/sell-like-a-third-grader/ "Sell Like a Third-Grader: How 2 + 2 Equals More Clients")。这对于主要目标是创收和盈利的商业企业和小企业来说非常有效。但是慈善组织呢？你如何向他们销售？

不要搞错:非营利不代表没钱。当然，由于经济原因，预算被削减了，慈善捐款也减少了。但是，除非你有意提供一些公益工作，或者你对这项事业有强烈的感觉，否则不要被欺骗为这个没钱花的可怜的慈善机构提供免费服务。

你怎么知道这正在发生？你有没有参加过面试，面试官开始向你推销这个职位？(如果我错过了这个“机会”，有人甚至试图让我觉得自己是一个没有野心的失败者。正如一个比我更聪明的人曾经建议的那样，密切关注你的直觉——如果你脖子后面的头发竖起来了，那可能是有很好的理由的。这里有一个令人毛骨悚然的故事:我们曾经向一位竞选公职的女性推销一个网站。大多数竞选活动都获得捐款来支付广告和营销等费用，但显然她的情况并非如此……因为她整个会议都在谈论政治，衡量我们的政治观点，向我们推销作为竞选志愿者建立网站的想法。我们通过了。

# 还是价值的问题

不管是商业还是慈善，价值的概念仍然适用。如果你能提供一个帮助慈善组织完成使命的解决方案，你就是在提供价值。由于目标是基于使命而非收入，您需要关注[使命回报(ROM)](http://returnonmission.com/what-is-return-on-mission/ "What is Return on Mission?") ，而不是投资回报(ROI)。找出他们的主要任务是什么，并向他们展示你如何帮助他们完成任务。ROM 与金钱无关，但对于一些慈善机构来说，募集捐款是其运营的一项重要功能。你可以向他们指出的一件事是，如果他们成功地完成了他们的使命，这将提高他们的知名度，从而导致捐款增加，使他们能够继续或扩大他们的努力。

慈善机构依靠捐款来资助他们的努力，但是教会依靠其成员的“什一税”——也就是说，捐出他们收入的一部分。然而，大多数真正的牧师并不是为了钱。不是说钱对于完成他们的使命不是必要的；但是，建议一个网站将建立他们的会众，并带来更多的钱可能不是最好的方法。相反，找出他们觉得该做什么。这可能因教会而异。一个人可能会觉得有责任去帮助无家可归的人。另一个，帮助未婚妈妈。较大的教会可能有几个事工。关键在于提出试探性的问题，然后展示你的服务如何能够带来回报——不是投资回报，而是使命回报。

# 提供社交媒体营销服务

事实证明，社交媒体在提高对原因的认识方面非常有效——这是这类组织不会忽视的事实。我写了关于与 SEO 专家合作销售 SEO 服务的文章。同样，在瞄准非营利组织时，考虑与社交媒体专家合作。客户不希望不得不去找不同的供应商。如果你能提供一站式解决方案，你就有更大的机会被录用。

# 避免与非营利组织打交道的陷阱

这并不罕见，尤其是在教堂，一个非技术志愿者已经建立了他们的第一个网站。一旦教会发展到能够提供专业帮助的地步，那么“解雇”那些努力工作来建立网站的志愿者就是一个等待你踏入的雷区。我们曾经竞标一个项目，为美国的一个大型教堂重新设计其中一个场地。我们认为我们做了正确的事情；我的合伙人是个老会员，我们有内部渠道。但是，最终，我们失去这个项目的原因是教会不愿意从长期维护和更新网站的志愿者手中夺走这个网站。当许多人的利益为了一个人而被牺牲时，这是很不幸的，但它确实发生了。有时，只是问一个尖锐的问题，例如，“是否有任何隐藏在幕后的东西会阻止这个项目向前发展？”会让这些问题大白于天下。这样一个问题的美妙之处在于，它经常会引出这样的回答:“你是什么意思？”—这为你详细阐述打开了大门:

> “嗯，乔，我知道一些慈善机构决定保留他们的旧网站，只是因为他们不想伤害志愿者的感情，这些志愿者在他们有资金建立一个专业网站之前就已经建立并维护了这个网站。这里有类似的事情发生吗？”
