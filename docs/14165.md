# 回到大局

> 原文:[https://www.sitepoint.com/back-to-the-big-picture/](https://www.sitepoint.com/back-to-the-big-picture/)

我将在这个月底离开 Sitepoint，成为一名博客写手。因此，在我们一起离开的这一周里，让我们着眼于大局，回到基础。

过去关于实体模型的博客提出了一个基本观点:没有人愿意雇佣专业人士，除非他们感到痛苦，即使这样，许多人也会不惜一切代价避开专业人士。为什么？因为，就像去看医生一样，我们讨厌脆弱，讨厌暴露自己，讨厌和一个会让我们觉得自己很愚蠢的专家打交道。不管你喜不喜欢，it 行业的名声很差(正如周六夜现场那个讨厌的电脑家伙所展示的)。

这带来了营销挑战和机遇。挑战在于，在销售专业服务时，传统的营销方法不起作用。用推销语言接触到一个繁忙、多疑的市场实在是太难了。正如塞思·戈丁和其他营销人员所指出的，如果你想结婚，你不会在酒吧里向某人求婚；你必须先向他们献殷勤，赢得他们的信任。嗯，在推销你的服务时也是如此。

这意味着，要想在你的网页设计/开发业务中取得成功，你需要将以下几个部分整合在一起:

1.将你的业务视为与客户的专业实践，而不是一系列来来去去的项目。

2.将营销作为重中之重。一个伟大的营销者会打败一个几乎每天都不会营销的伟大的开发者。

3.让你做的每一件事都有助于你的业务发展:建立信任，展示价值，变得可信。

4.建立强大的战略基础。通过专注于有利可图的利基市场和精心制作不可抗拒的营销信息来做到这一点。后者来自于对你的潜在客户的世界的深刻理解:他们的问题，这些问题的代价，以及他们的期望。

5.使用低成本、高影响力的方式展示你的价值并建立你的信誉。采取一种教育/价值驱动的营销方法，并专注于推荐系统、写作、演讲、研究、社区服务等策略，并参与目标市场所在的组织。成为一个看得见的领导者，而不仅仅是一个找工作的佣兵。

6.开发自动跟进系统，并亲自跟进潜在客户。从收集潜在客户信息开始，然后以非垃圾邮件、有价值的方式跟进(免费报告、文章等)。这需要 4-5 次积极的互动，直到一个潜在客户觉得雇佣你很舒服，甚至愿意和你谈雇佣你的事。

7.学会如何评估适合的机会，这样你就不会浪费时间去追逐你永远不会赢得的机会。

8.学会赢得合适的机会，这样你就能赢得更多你追求的机会。

9.通过管理期望、关注业务结果(而不仅仅是你签约要做的任务)、管理关系以及始终保持一种“我能行”的态度来实现和取悦他人。

10.建立长期客户关系。

11.通过建立系统来平衡你的时间和金钱，并创造股票价值，从而发展你的公司。

就这样了，伙计们。张贴任何关于上述拼图的问题，我会尽我所能在下周解决它们。但是请先花点时间回顾一下过去 10-11 个月的博文，这样我们就不会在这里多此一举了。
