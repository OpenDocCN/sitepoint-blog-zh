# 网站管理员日记第一部分——我的网站设计清单

> 原文：<https://www.sitepoint.com/site-design-checklist/>

每周我都会收到大量的电子邮件，他们让我评论他们网站的设计和可用性缺陷。希望，如果你使用一个合理的过程来开发你的网站，你不需要批评。

以下是我每次设计网站时都会考虑的基本要素——以此为清单，我可以确保在我完成的每个设计中涵盖了所有的网页设计要点。当你创建下一个网站时，试试这些建议，并检查列表中的每一点！

##### #1:选择一种配色方案并坚持下去

一天，当你在网上冲浪时，你决定去看看一个新的链接。当它加载到您的浏览器中时，您会注意到主页是红色、黑色和灰色的。然后你点击一个链接，比如“关于我们”页面，你会看到一个黄色和绿色的页面，上面有蓝色的文字。

这种不一致的颜色足以阻止访问者再次访问一个网站——不仅看起来不舒服，而且也表明缺乏专业素养。

在你开始为你的网站编码之前，选择两到三种互补色并坚持使用。如果你为之建立网站的组织有一个使用特殊颜色的标志或品牌，你可能会考虑使用它们。如果你看一看任何一家大型或成功公司的网站(试试 www.coca-cola.com 的[或 www.ford.com 的](http://www.coca-cola.com))，你会发现，即使网站的不同子部分有不同的颜色编码，网站的每一页都有品牌颜色。

看一看你喜欢的其他网站:他们使用什么颜色，他们如何使用它们？当你加载他们的主页时，他们是逐渐引入颜色，还是全都出现在屏幕中间？我列出了网络上最常用的五种颜色组合:

1.  红色、黄色和白色
2.  蓝色和白色
3.  红色、灰色和白色
4.  蓝色、橙色和白色
5.  黄色、灰色和白色

##### #2:跨浏览器兼容性设计

这是设计师要掌握的最重要的方面之一。永远不要在一个站点中实现特定于 Internet Explorer 或 Netscape 的功能，除非它只被一个封闭的用户组使用(例如公司内部网)。

当然，实现像自动 iFrame 滚动这样的超级 dooper DHTML 效果是很诱人的，但是要注意:那些没有安装最新浏览器的人不会接受你对他们需求的忽视。如果你不顾一切地想在你的网站上实现飞猪或雪花飘落，但仍然希望跨浏览器兼容，那么看看来自 [Cyscape](http://www.cyscape.com) 的 BrowserHawk。BrowserHawk 是一套漂亮的 COMs，允许您检测关于客户端浏览器的各种事情，包括他们是否启用了 JavaScript，以及他们浏览器的版本和名称等等。

##### #3:提供直观、易于使用的导航系统

你是否曾经访问过一个网站，并且想知道它的其他页面的链接在哪里？菜单的可访问性是为网站访问者创造积极体验的关键因素之一。

大多数网站要么显示左对齐、垂直方向的菜单，要么显示上对齐、水平方向的菜单系统。调查表明，使用这些菜单风格中的任何一种(或者两者一起使用，如果你以互补的格式来布局的话)都可以保证为你的访问者提供积极的网站体验。使用这些熟悉的样式会让你的用户在页面间切换时感觉很舒服，也意味着他们不必每次想返回主页时都点击后退按钮。

为了理解我的意思，花几分钟时间在 Amazon.com 转转。现在，花同样的时间在 www.isonsw.com.au[的站点](http://www.isonsw.com.au)。你对哪个网站的菜单系统感到满意？我敢肯定你会回答亚马逊的，因为它是一致的、易于使用的，而且无论你在网站的哪个位置，你都可以清楚地看到你在哪个页面。当你开发一个新网站时，你应该设计至少三个菜单系统的原型，并询问你的朋友、家人和同事他们更喜欢使用哪一个，为什么。

##### #4:使用层叠样式表

级联样式表(CSS)允许您开发一组特定的样式类，您可以在整个站点中实现这些样式类。样式表还可以用来改变内置 HTML 标签的某些样式属性，比如将

# 标签的颜色设为红色，或者将Set the background color of the cell to yellow, and so on.

使用你在技巧 1 中确定的配色方案来创建各种样式，包括粗体标题、要点样式和默认文本样式。您可能还想更改锚标记的默认样式，以便您的链接与站点的配色方案相匹配。

如果您以前从未使用过级联样式表，那么这里有一些很好的文章可以帮助您入门:

*   [级联样式表介绍](https://www.sitepoint.com/cascading-style-sheets-css/)
*   [在 W3.org 增添一丝时尚气息](https://www.w3.org/MarkUp/Guide/Style.html)

##### #5:在新窗口中打开外部链接

一个容易实现但经常被忽视的技巧是:确保默认情况下，任何不直接将访问者带到网站页面的链接都应该在新窗口中打开。

仔细想想，这种方法给访问者带来的好处和给你自己带来的好处一样多:他们可以自由浏览外部链接，并可以选择通过关闭外部站点的浏览器窗口返回到你的站点。

要在新窗口中打开一个链接，只需为链接的锚标记的目标属性指定值“_blank”。例如，你可以用这个…

```
<a target="_blank" href="http://www.sitepoint.com">Click here</a>
```

…在新的浏览器窗口中打开 SitePoint.com。

##### #6:给你的超链接加下划线和颜色

大多数网络用户都很匆忙，以这样或那样的方式赶时间。我们的眼睛只注意某些东西，这些东西不包括看起来像是文档主体一部分的超链接。当你开发一个新的网站时，一定要确保你的超链接有下划线，最好是和周围的文字有不同的颜色。

这是几天前我注册会员计划时收到的一个页面的摘录:

<q>***注册成功！***</q>

感谢您成为我们网站的会员。你账户的详细信息已经通过电子邮件发送到 name@site.com。要返回我们的网站，请点击这里。如果你想把我们的网站推荐给你认识的人，请点击这里。

快速浏览上面例子中的文本，你发现了多少个链接？如果你回答没有，那么你就不是一个人了。

每个“点击这里”实际上是一个链接，但是网页设计者甚至懒得把这个链接从它周围的文本中突出出来。看看当链接都带有下划线和颜色时，发现它们是多么容易:

<q>***注册成功！***</q>

感谢您成为我们网站的会员。你账户的详细信息已经通过电子邮件发送到 name@site.com。要返回我们的网站，请[点击这里](javascript:;)。如果你想把我们的网站推荐给你认识的其他人，那么请[点击这里](javascript:;)。

##### #7:优化您的图像

这么多网站速度慢的一个主要原因是因为它们的图片没有完全优化。优化图像会减小文件大小，这意味着在显示之前需要从网站下载的数据会更少。以下是优化图像的三种方法:

***1。**缩小图像尺寸:*

确保你的图片尽可能的小。裁剪边缘周围的任何“空白”-这会增加图像的文件大小。

***2。减少颜色数量:***

包括图形交换格式(GIF)在内的许多图像格式允许您在不明显降低图像视觉质量的情况下降低图像的颜色深度。如果你有一个图像，比如一个标志，那么试着用一个程序，比如 PhotoShop，把它的颜色深度从 16 位降低到 8 位。保存图像并在浏览器中查看。如果它仍然看起来清晰干净，那么保存图像。注意到与 16 位版本相比文件大小的差异了吗？

***3。降低图像质量:***

如果您正在处理联合图像专家组(JPEG)图像，那么您可以通过降低其质量来减小其文件大小。使用 PhotoShop 等程序，您可以指定图像的“损失”量，从而减小图像的大小。

##### 告诉访问者你是谁，你做什么

这可能是实现起来最简单的技巧。当一个新的访问者第一次来到你的网站，他们想直接知道你是谁，你做什么，特别是如果你卖产品的话。

在你的主页上，你应该有一小段文字告诉他们你是谁，你是做什么的。这将增加他们对你公司的信心，如果你有他们想要的东西，他们就更有可能留下来。这是一个虚构的网站 Fred-Smith-Tools.com 的例子，它出售园艺工具:

<q>*“嗨，谢谢你顺道拜访 Fred-Smith-Tools.com。我们总部设在美国洛杉矶，销售各种园艺工具，包括铁锹、修剪工具、剪刀、喷枪和铲子。请向下滚动本页，看看我们的每日特色菜列表！”*</q>

##### #9:使用客户评价

客户评价只是网站访问者的评论，包括他们与贵公司交易的一些正面细节，例如:

“我上周在 Google.com 发现了你们的网站，我必须说，它的布局和你们丰富的产品种类给我留下了深刻的印象。我周四向你们订购了一张 CD，第二天它就直接送到了我家门口。要是网上有更多像你这样的网站就好了。干得好！”

通过在你的网站上展示客户评价，访问者可以看到其他人已经使用了你的网站或产品，并且发现它们是有用的和有价值的。客户证明也是增加访问者对你的企业信心的最好方法之一。

要在您的网站上显示客户评价，请在您的菜单系统中添加一个名为“客户评价”的链接。将此链接到一个页面，在该页面上，您可以显示从您的客户那里收到的所有评价。或者在侧边栏中显示评价，侧边栏链接到列出所有评价的页面。要真正收集评价，要么直接向你的一些客户索要，要么在你的网站上建立一个反馈表格。

##### #10:在每一页上提供联系方式

许多网络冲浪者感到沮丧的主要原因之一是许多网站缺乏联系方式。如果你经营一个销售产品的网站，那么你的许多潜在客户可能更喜欢打电话订购，而不是在网上订购。因此，请务必在每页的右上角显示销售电子邮件地址或销售热线电话号码。

##### 完整的清单

如果你是一名网站开发人员，那么当你设计每一个网站的原型时，试着检查一下这些提示——但是请记住，这些提示同样适用于现有的网站。

毫无疑问，随着你设计越来越多的网站，你会想出一系列你自己的设计技巧和指导方针。尽可能地使用它们，并与新的 Web 开发人员分享，这样他们就不会犯和你一样的错误了！
