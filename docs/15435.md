# 为了成功与他人建立关系网

> 原文：<https://www.sitepoint.com/networking-others-success/>

所以你想显著提高你的个人生产力，并在生活的各个领域取得成功？你可以单干，也可以享受别人的帮助。

网络是一个主要的时间管理工具，帮助我们向他人学习，更容易得到介绍，并在这个过程中节省宝贵的时间和金钱资源。我积累了五个可能有所帮助的“简洁网络”建议。

**1。列出清单。**

把事情写下来比试图记住你认识的每个人更有力量。我用一个软件联系程序来帮助。你包括谁？所有人。你认识的每个人，你见过的每个人。不要只是从你当地的电话号码簿上抄白页。“接触”就是你有过“接触”。将所有人添加到数据库中。不要漏掉任何人。包括姓名、地址和联系信息。普通人可以很容易地列出一个超过 1000 个联系人的初始列表。听起来不太可能？从你的家人开始，然后是你的邻居和同事。你上一份工作的同事怎么样？你的高中和大学毕业班和你的老师。你属于一个专业协会，一个俱乐部，一个教堂？银行出纳员，你的牙医，你的庄家，他们都在名单上。

**2。将列表分类。**

这就是联系人软件程序真正有用的时候。确定哪些人是朋友，哪些人是熟人，客户，供应商，政治家，专业人士，或者那些喜欢高尔夫或网球的人。你能把人们放入的类别越多，你就能越快地找到合适的联系人。通过我的数据库，对于我自己和我想帮助的人来说，没有什么问题我不能得到答案。

**3。喂名单。**

一旦你创建了一个列表，你就必须继续充实它。当你遇到越来越多的人时，更新、纠正和添加他们。我们可能每周都会遇到几十个新朋友。我的数据库现在已经增长到 4000 多个了。我每周五花大约一个小时来充实我的清单。这是个苦差事。不方便但是值得。

**4。要有朋友，先做朋友。**

以下是如何让所有这些数据为您服务。网络不是一种自私的技术。如果你想让这个工具发挥作用，你必须像一个好的国会议员。你为人们做事。你先帮助他们。我总是把我看到的文章剪下来发给我认识的人。我寄了很多生日卡。我每年至少给我数据库中的大多数人打一次电话，谈论他们，他们过得怎么样，他们需要什么，我可以为他们提供。然后到了“选举时间”，当我需要什么的时候，我会毫不犹豫地要求回报。

**5。使用它。**

每当我开始做任何事情，一个新的营销计划，职业变动，买房或买车，我首先会想到我的人际网络，并与我数据库中的人交谈，他们可能会给我一些答案。通过首先利用我的网络数据库，我节省了大量的时间和金钱，并在许多方面取得了成功。
