# 设计伟大的广告牌

> 原文:[https://www.sitepoint.com/design-great-billboards/](https://www.sitepoint.com/design-great-billboards/)

我们可以从离线世界中学到很多关于设计有用网站的知识。

例如，几周前我在阿德莱德探亲时，我开车绕着阿德莱德山转了一圈。我以每小时 120 公里的速度行驶在高速公路上，经过了右图中的标志。我首先想到的是“这到底是什么意思？”

我不知道你们那个地方是什么样子，但当我上次住在南澳大利亚的阿德莱德时，术语*raider bed*绝对不是通用语言。

(我有点期待看到一个穿着全套警服的吓人的特大号床垫从灌木丛中走出来，命令我靠边停车。)

我想到的第二件事是——这可能表明我在网上花了太多时间——这是:

想象一下，如果一个类似的晦涩的东西被用来描述一个网站上的标签。没有人会知道这条短信链接到了什么！”

Steve Krug 写了这个概念——我们应该把我们的用户想象成开车经过，并为他们提供易于理解的导航标签。然而，很明显，我们不应该使用我们遇到的每一个广告牌作为灵感。

这个故事的寓意是，当决定你的导航项目使用的单词时，使用你的观众立即知道和理解的语言真的很重要，而不用他们深入钻研他们的词汇并找出答案。具体是什么语言取决于你的网站和你的受众——如果你的网站是面向青少年的，那么把你的*联系我们*页面的链接叫做“给我们写信，哟”是完全可以接受的。然而，对于大多数网站来说，你最好坚持使用更安全的*联系人*或*联系我们*。

PS。我后来发现，路床实际上是一个紧急坡道，可供刹车失灵的车辆(通常是卡车)的司机使用。坡道提供了比栅栏门或树更安全的停车方式。它可能是一个美国术语；就我个人而言，我更希望看到它被称为“紧急安全坡道”。在最激烈的时刻，这将需要司机更少的脑力，从而潜在地挽救更多的生命。
