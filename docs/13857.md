# 从 0 美元到 100 万美元只用了一年时间

> 原文:[https://www . site point . com/from-0-to-100 万仅一年/](https://www.sitepoint.com/from-0-to-1-million-in-only-a-year/)

我的一位同事兼朋友今年从 0 美元到 100 万美元建立了一家 IT 解决方案提供商公司。对他是如何做到的感兴趣？以下是方法:

1.他准备离开他的公司(他在那里工作)几年，然后决定跳槽。在此期间，他在公司内部培养了一些非常棒的人脉——可以帮助他推荐业务的人，以及潜在的业务合作伙伴。他还积累了一些财政储备。

2.直到找到第一个大客户，他才离开。

3.他关注的是那些曾在他的公司工作，但后来在其他大公司成为 IT 高管的人。

4.他开发了一个大的解决方案集，专注于企业级应用。他用极其敏捷的开发方法使他的解决方案与众不同，这是许多公司仍然缺乏的。

5.他与另外 3 个人合作，他们都为他的公司带来了杰出的技术专长或人脉网络和销售能力。

6.他知道如何谈论“业务”而不是技术术语。因此，他的解决方案关注公司的实际结果。

7.他像疯了一样拼命工作，会见潜在客户，培养关系，并交付出色的成果。我不能低估他今年花了多少时间，包括旅行时间。

恭喜你！
