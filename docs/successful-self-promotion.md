# 吹响你自己的号角而不走调

> 原文：<https://www.sitepoint.com/successful-self-promotion/>

成功的一部分是积极探索新的机会，建立新的关系，并以积极的态度定位自己。你是自己最大的冠军，尤其是为自己打工的时候。如果你不为自己辩护，你就不能指望别人为你辩护。

然而，有效提升自己和过度提升自己之间只有一线之隔。做过头了——专注于自我中心的提升，忽视他人的利益，夸大自己的优点——会产生灾难性的影响。

这里有几个方法可以巧妙的自我推销。

**客户评价**

你做得很好，你的客户很激动。你可以通过你的网站或个人网站，为潜在客户收集和提供证明，让潜在客户更容易看到你能提供什么。

客户证明不一定是正式的推荐信。你可以收集并请求允许使用来自客户的随意赞美。您可以录制音频或视频推荐，甚至为客户准备一个标准的推荐表格，以便他们在项目结束时填写，从而更容易提供反馈。

**提供担保**

显示你对自己的能力和职业道德有信心的一个方法是为你的服务提供某种担保或保证。例如，如果您是一名开发人员，您可以考虑提供一个有限时间的支持包，作为客户处理任何错误或其他故障排除问题的费用的一部分。

为了更进一步，你还可以为你的服务提供退款保证。你要确保你澄清保证的条款，以避免以后任何潜在的冲突。

**维护一个完整的网站和投资组合**

你的网站和作品集是你可以为你的企业维护的最好的推广工具。让你过去成功的事实容易被潜在客户理解。包括过去工作的例子，项目案例研究，和证明。您还可以包括客户参考、关于您工作流程的详细信息以及常见问题的答案。

**不抱期望地帮助他人**

另一种确立你作为服务提供者的价值的方法是，愿意分享你的专业知识，帮助他人，而不期望得到回报。这可能意味着向面临挑战的同事提供帮助，通过社交媒体宣传他人(即在 Twitter 上转发，在脸书上分享链接，在 LinkedIn 上提供建议)，以及免费分享技巧、文章和其他有价值的内容。

**鼓励口碑**

我们都知道口碑广告的力量。你的客户会把你的工作告诉他们自己的同事，并定期推荐给他们吗？不要羞于让你的客户向别人推荐你。你甚至可以为推荐人提供奖励来鼓励推荐。永远要对介绍人说谢谢。

你如何按喇叭？你如何避免做得太多太频繁？
