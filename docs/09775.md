# 10 个 jQuery 元内容插件

> 原文:[https://www.sitepoint.com/10-jquery-meta-plugins/](https://www.sitepoint.com/10-jquery-meta-plugins/)

10 个 jQuery 元数据相关插件，将帮助您的网站元内容，元键，元刷新和元验证。用 jQuery 实现您自己的元管理！

元插件具有以下特征:

*   您可以用 jQuery 实现自己
*   易于跨浏览器实现
*   用户不会注意到它们。
