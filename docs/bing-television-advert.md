# 微软首播必应电视广告

> 原文：<https://www.sitepoint.com/bing-television-advert/>

作为该公司 100 美元营销活动的一部分，微软已经开始在美国电视上播放必应搜索引擎广告。随着 Bing 本地化工作的进展，该视频可能会出现在其他国家，但如果你还没有看到它…

在一分多钟的时间里，这个广告的形式多于实质。它浮华、快速、时尚，并警告人们不是在寻找，而是在逃避。显然，我们都*【迷失在链接中】*。我们不需要只会引出更多问题的查询和关键词，我们需要一个决策引擎。

当然，这个广告实际上并没有告诉我们什么。为什么公司认为我以前的求职经历有这么多缺陷？Bing 是做什么的？为什么要用它？然而，这种神秘的气氛会吸引许多人去尝试必应，而这正是微软所需要的。关于搜索引擎的反馈(对不起， *decision* engine)在很大程度上是积极的，但是只有当公司能够说服人们放弃谷歌和雅虎时，这个项目才会成功。

有趣的是，这里只是简短地提到了微软本身；他们的标志在广告的最后两秒钟出现在右下角。眨眼就错过了。

如果你还没有使用必应，这个广告会鼓励你这样做吗？该公司是否有意淡化微软品牌？电视广告对网站有用吗？
