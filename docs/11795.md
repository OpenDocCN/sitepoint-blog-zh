# 待办事项清单:12 个组织任务的在线工具

> 原文:[https://www.sitepoint.com/online-to-do-list-tools/](https://www.sitepoint.com/online-to-do-list-tools/)

昨天，我发布了关于[待办事项列表](https://www.sitepoint.com/to-do-lists-for-productivity)以及选择系统、设置和维护列表的提示。虽然明天我会回溯一点，给你一个更流行的任务和信息管理系统的概述，你可以把它结合到你自己的过程中，但今天我主要关注在线任务管理工具。

如果你不需要一个全面的项目管理工具或全面的信息管理系统，一个简化的待办事项工具可能非常适合你。这里列出了 12 个在线待办事项工具。

**[Bla-Bla List](http://blablalist.com/):**Bla-Bla List 是一个简单且可共享的列表工具，它可以让你用 RSS 发布你的列表，私下与任何人分享你的列表，并通过电子邮件给你自己发送一份列表副本。

[![](../Images/f908e816b2c7577299885d9bef20c0c1.png)T2】](https://blogs.sitepointstatic.com/images/business/blablalist.jpg)

**:**这个工具可以让你管理单独列表中的任务，添加笔记和提醒，并与他人分享你的列表。

[![](../Images/2ea61cefc96416b861e2a3315839e2f2.png)](https://blogs.sitepointstatic.com/images/business/e2dolist.jpg)

**[谷歌待办事项清单](http://www.google.com/ig) :** 追踪任务和提高生产力的简化方法，尤其是如果你已经是谷歌产品的用户。

[![](../Images/5b6305ba3615dc0c27a5d45cade3b5f7.png)T2】](https://blogs.sitepointstatic.com/images/business/google.jpg)

**[在线 CEO](http://theonlineceo.com) :** 在线 CEO 提供了一种创造性的任务处理方法，它让你给每一项任务打分，并有一个累计分数，这样你就可以衡量优先级和工作效率。你也可以用计时器来记录你在每项任务上花了多长时间。

[![](../Images/14807a5437682c970b6896068dbe300f.png)T2】](https://blogs.sitepointstatic.com/images/business/onlineceo.jpg)

**[orchestrath HQ](http://www.orchestratehq.com/):**一个快速简单的创建和管理多个待办事项列表的工具。你可以浏览你的列表，一次最大化一个。

[![](../Images/e1742d484f8ab21404857ec590c751e9.png)T2】](https://blogs.sitepointstatic.com/images/business/orchestrate.jpg)

**[忆奶](http://www.rememberthemilk.com) :** 忆奶是我见过的功能最丰富的任务管理工具之一。您可以使用丰富的键盘快捷键快速轻松地创建、管理和组织任务。您可以通过电子邮件、短信和即时消息接收提醒；标记您的任务；并在地图上定位基于位置的任务。还有一个 Pro 版本，提供了一个用于 iPhone 和 iPod touch 的应用程序，以及用于黑莓和 Windows Mobile 的 MilkSync。

[![](../Images/4b29153e0ab69f506d30c1a2cfff6f9b.png)T2】](https://blogs.sitepointstatic.com/images/business/rtm.jpg)

**[Ta-da List](http://tadalist.com):**Ta-da List 是一款 37signals 产品，可让您创建多个列表，并从每个列表中添加、编辑、重新排序和删除任务。您还可以为您的列表共享和创建 RSS 源。

[![](../Images/f19e719c3d5d82d53a1b44ddec6c9db3.png)T2】](https://blogs.sitepointstatic.com/images/business/tadalist.jpg)

**[TaskToy](http://www.tasktoy.com/docs/about.html) :** 使用 TaskToy，您可以创建具有多个列表的单个任务、周期性任务，并使用简化的项目管理操作。您还可以打印和共享您的列表，并通过移动设备访问您的数据。

[![](../Images/2da96928220fef3f1075388c5477a775.png)T2】](https://blogs.sitepointstatic.com/images/business/tasktoy.jpg)

**[Toodledo](http://www.toodledo.com/):**Toodledo 允许你在一个可排序的在线待办事项列表中组织和优先处理你的任务。您可以设置电子邮件和短信提醒，并通过即时消息在手机、电子邮件、日历、RSS 阅读器中访问您的列表，并直接集成到您的网络浏览器中。

[![](../Images/77c7d4630feabdefede28395d682c669.png)T2】](https://blogs.sitepointstatic.com/images/business/toodledo.jpg)

**[土豆列表](http://app.ess.ch/tudu/welcome.action) :** 土豆列表是一个非常简单的开源应用程序，它可以让你创建和共享列表，通过 RSS 订阅源访问你的列表，并根据你的个人需求进行修改。

[![](../Images/deb3bad180c1091e52d29d1cbe39df0d.png)T2】](https://blogs.sitepointstatic.com/images/business/tudulists.jpg)

这个工具可以让你按项目组织任务，通过电子邮件添加任务，跟踪每项任务花费的时间。您还可以使用该工具进行一些基本的任务共享。

[![](../Images/fea11378613511356174c040e9e3a9cc.png)T2】](https://blogs.sitepointstatic.com/images/business/voo2do.jpg)

**[WorkHack](http://workhack.com/) :** 一个基于白板的待办事项列表工具，简单易用，支持多列表和 RSS 订阅。您可以立即开始使用它，无需注册。

[![](../Images/113c9a4517e706d2164ec9f8f0fc16cf.png)T2】](https://blogs.sitepointstatic.com/images/business/workhack.jpg)
