# CSS 精灵

> 原文：<https://www.sitepoint.com/css-sprites/>

近几个月来，CSS 社区出现的最令人兴奋的技术之一是使用一个大背景图像作为几个小图像的容器，然后使用 CSS 背景位置属性在页面的不同位置(和不同的悬停状态)显示大图像的不同部分。

据我所知，这个技巧的第一个例子是 Petr Stanicek 的[快速翻转，不需要预加载](http://www.pixy.cz/blogg/clanky/cssnopreloadrollovers/)演示。它已经被社区的其他成员扩展了:Dave Shea 的一篇列表外的文章 [CSS 精灵:图像切片](http://www.alistapart.com/articles/sprites/)
