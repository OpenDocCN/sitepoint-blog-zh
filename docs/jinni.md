# 金妮——搜索的未来是个人的

> 原文：<https://www.sitepoint.com/jinni/>

不知何故，每次一个新的搜索引擎出现，我们都会听到一个预测未来的新口号。Hakia 希望打破规则，成为第一个基于本体语义和自然语言处理(NLP)的搜索引擎，但当其主要强劲竞争对手 Powerset 出售给微软时，它输掉了这场战争。

很少有人知道的是，自 50 年代以来，所有搜索引擎都在[中使用 NLP，所有处理语言的计算机程序都使用 NLP(包括你的拼写检查器)，所以吹嘘自己是基于 NLP 的搜索引擎不是我建议新搜索引擎做的事情。](http://www.marketingpilgrim.com/2009/03/understanding-natural-language-processing-for-seo.html)

hakia 和 Powerset 都曾自称为语义搜索引擎。这种情况已经改变，双方都提出了更广泛、更现实的主张。NLP 不是语义搜索的同义词——语义搜索只是技术的一部分——我们离拥有真正的语义搜索引擎还很远，尽管许多人正在朝着正确的方向前进。

这篇关于自然语言处理和语义搜索的介绍介绍了一个新的搜索引擎，它有一个明确的定义:[金妮](http://www.jinni.com/home.html)，这个新的视频搜索引擎自称为[“第一个味道引擎”，](http://www.jinni.com/pages/about.html)在其算法中陈述了语义搜索技术和个人推荐，两者都基于自然语言处理和味道分析。我们必须从一开始就明白，没有算法，任何记者都无法支持这样的说法。至于搜索引擎的主张，任何新的创新都需要利用一些东西，语义搜索的想法是模糊的，足以允许这样的公关调整。

然而，金妮更有趣的是他们对未来的展望，在未来，搜索引擎变得个性化，搜索结果基于个性化的推荐提供。引擎的核心使这些建议成为可能——电影基因组(由电影专业人员创建)包含数千个分配给每个标题的“基因”,用于描述情节、情绪、风格、背景、配乐等——这是常见流派语言的丰富替代。从搜索引擎优化的角度来看，电影基因组是一个进步:通过标记页面元素和视频内容，更准确地识别和搜索引擎可见，金妮实际上提出了对现有算法的改进。

金妮的另一个有趣的方面是“脉冲”——一个显示用户行动和意见的实时流的功能。对 Twitter、脸书和个人博客等社交网络的持续监控也是“脉动”的一部分，因为金妮希望成为对话的一部分，而不是实际上成为社交网络本身。

“品味剖析”的一个例子——可能是金妮最重要的特征。

最后但同样重要的是，个性化推荐确实是个性化的。不存在你在亚马逊、网飞和其他类似网络上发现的“喜欢这个的人也喜欢……”。金妮的模式是独特的。通过整合网飞、Hulu 和其他领先的内容提供商，金妮的目标是成为选择接下来要看什么的个性化起点，根据[、](http://news.cnet.com/8301-17939_109-10200031-2.html?subj=news&tag=2547-1_3-0-20&part=sphere )、*的说法，“金妮是网络上最好的电影推荐引擎。句号。”我倾向于同意。你有什么看法？*
