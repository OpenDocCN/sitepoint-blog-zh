# Firefox 仍然是最受欢迎的开发浏览器

> 原文：<https://www.sitepoint.com/firefox-most-popular-development-browser/>

上周 SitePoint 投票的结果已经出来了。我们问:[“你主要的 web 开发浏览器是什么？”](https://www.sitepoint.com/primary-development-browser/)

3126 人投票:

*   67.7%使用火狐浏览器
*   20.1%使用 Chrome 或 Safari
*   8.5%的人使用 Internet Explorer
*   3.7%使用 Opera

我们都期待 Firefox 表现出色。实际上，开发者真正想要的不是 Firefox，而是它的一系列优秀的扩展。Firebug 被认为是使用 Firefox 的主要原因之一，但是[还有很多其他扩展可以选择](https://addons.mozilla.org/firefox/browse/type:1/cat:4)。

然而，Chrome 开始侵占 Firefox 的领地。Webkit Inspector 是 Firebug 的一个很好的替代方案，而且很棒的扩展正在出现。Chrome 不能完全满足 Firefox 扩展的范围和能力，但它们正在变得更好，开发者尤其喜欢浏览器的速度。

几乎十分之一的人继续使用 IE 作为他们的主要开发浏览器。如果你正在为一个 IE 是默认/强制浏览器的公司开发内部内部网应用程序，这是有意义的。IE8 的工具比不上一些竞争对手，但它们足以解决大多数问题。

我原以为 Opera 的比例会更高，尽管它的市场份额是浏览器的两倍。Opera 提供了一些强大的开发工具，但也许它们不足以吸引人们去转换？

目前，Firefox 仍然是网络开发者的王者。但是 Mozilla 应该警惕——Chrome 正在快速追赶。
