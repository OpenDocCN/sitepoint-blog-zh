# Linux 不仅仅是臭鼬工厂

> 原文:[https://www.sitepoint.com/linux-not-just-skunkworks/](https://www.sitepoint.com/linux-not-just-skunkworks/)

Forrester Research 最近发布的一项研究“Linux 进入任务关键型应用程序”显示，Linux 现在被北美一些最大的公司广泛认为是主流。

多年来，Linux 在许多公司的 IT 部门的密室中被引入。这些往往是未经审查的项目或原型，以鼓励考虑该平台。既然各个层次的企业都已经看到了 Linux 和应用程序(如 Apache、MySQL 和其他应用程序)的可靠性和可伸缩性，Linux 就没有必要部署在暗处。

在那次调查中，Forrester 的分析师 Brad Day 和 Laura Koetzle 采访了 100 多家公司，其中一半报告的收入超过 10 亿美元，每个公司有 5000 多名员工。调查还发现，超过一半的企业在生产环境中部署了 Linux，运行用于网站托管、邮件以及关键后台功能(如 crm 和数据管理)的服务器。

与许多技术趋势一样，只有在世界上较大的公司在研究、开发和部署上投入了数十亿美元之后，广泛采用才开始。IBM 在 Linux 和开放源码上的大量投资通常被认为是推动企业 IT 认真评估 Linux 的原因。

该研究显示，超过三分之一的受访者选择需要 Linux 的新解决方案，而不是传统的想法，即大公司使用 Linux 将旧的遗留应用程序移植到新的技术平台。

这对各行各业的 web 开发人员来说都是很有见地的。由于“买入”在最高级别被发现，这为中小型企业级别的基于 web 的应用程序解决方案的 Linux 解决方案增加了额外的可信度。那些不愿意冒险使用 Linux 和/或开源解决方案的人可能会更容易接受这个想法，因为他们看到公开的统计数据显示关键应用程序得到了更广泛的采用。
