# 超过 10 个“必须拥有”WordPress 插件

> 原文:[https://www.sitepoint.com/10-must-have-wordpress-plugins/](https://www.sitepoint.com/10-must-have-wordpress-plugins/)

我爱我的 WordPress。事实上，在我用过的所有内容管理系统中，WordPress 是迄今为止最灵活、最容易处理、最有趣的。是什么让 WordPress 如此美丽？许多 WP 爱好者开发的插件。这些插件通过向您的站点添加吸引您的社区的交互元素，帮助您将博客转变为真正的富媒体体验。

##### 评论插件

我认为吸引社区响应的最佳方式是鼓励在你的网站上发表评论并奖励这些评论。奖励评论的方式有很多:个人回复、被关注的签名链接、最受欢迎评论者的排名，或者展示评论最多的帖子，向读者展示“热门”对话发生在哪里。没有插件可以取代个人对评论的回应，但是对于所有其他的“奖励”,博客世界已经喜欢上了一些特定的插件。

**[DoFollow]( http://www.semiologic.com/software/wp-tweaks/dofollow/)** 从评论中的签名链接中删除“nofollow”属性，以便搜索引擎可以跟踪它们。这样做的问题是，它吸引了垃圾邮件发送者和评论者，他们只是为了链接而评论，并没有给你的网站带来高质量的反馈。如果你使用 DoFollow，评论一定要适度。

展示 [**顶级评论员**](http://www.pfadvice.com/wordpress-plugins/show-top-commentators/) 会鼓励你的读者进行更多的讨论。通过显示他们的名字链接到他们的网站，你也给了他们“链接价值”和“流量价值”

为了显示最热门的讨论发生在哪里，你可以使用 [**WP 最多评论**](http://wordpress.org/extend/plugins/most-commented/ ) ，它检索评论最多的帖子列表。这基本上抓住了你的读者的注意力，并把它带到你最有争议的帖子上，让社区参与到对话中。博客条目就像一个论坛，读者可以在这里讨论、分享观点和知识。你的角色？一个积极的调解人。

##### “新鲜”插件

另一个让人们回到你网站的因素是“新鲜感”您可以通过添加旋转内容来保持网站的新鲜感。从一个 [**旋转标题**](http://wpimagerotator.com/) ，到 [**旋转侧边栏图片**](http://wordpress.org/extend/plugins/nextgen-gallery/) 和 [**随机帖子**](http://wordpress.org/extend/plugins/search.php?q=random+posts) ，一切皆有可能。

也许更有趣的是 [**特色内容插件**](http://www.featuredcontentgallery.com/)——它给你的网站添加了一个自动旋转的图库。它被许多杂志类网站使用，因为它是完全可定制的，并且容易集成到网站的任何地方，无论是文章级别还是页面级别。

##### “定位”插件

对于冗长的博客文章，忘记“阅读更多”的技巧——现在你可以将你的内容分成更多的页面，所有页面都有编号，并带有“上一页-下一页”导航。为此使用 [**多页工具包**](http://wordpress.org/extend/plugins/multi-page-toolkit/ ) 。

什么是没有面包屑导航的网站？就像去一个没有地图的外国城市。如果你不想让你的访客沮丧，使用一个 [**面包屑导航插件**](http://wordpress.org/extend/plugins/breadcrumb-navigation-xt/ ) 来帮助他们找到路。

##### “热门”插件

Twitter 现在势头很大。如果你不在 Twitter 上，你可能会错过很多社交媒体上的“热门和冷门”。你可以将 Twitter 作为“文章和摘要”添加到你的博客中，并在博客文章和 WordPress 中创建新的推文。Twitter 插件由亚历克斯·金创建，他的网站上有[一堆其他有用的插件](http://alexking.org/projects/wordpress )。

使用 [**Viper 的视频快速标签**](http://wordpress.org/extend/plugins/vipers-video-quicktags/ ) ，现在向你的 WP 网站添加视频比以往任何时候都容易。有了这个插件，你不再需要从 YouTube、Veoh、Vimeo 等视频网站复制和嵌入破解代码的 HTML。你只需要将视频的网址粘贴到提示框中，插件就会自动为你嵌入视频。

说到视频…你甚至可以允许你的用户在你的博客上发表视频评论，使用 [**Seesmic WordPress 插件**](http://wiki.seesmic.com/Wp-plugin) 。该插件允许回复和线程视频对话，甚至让你可以像文本评论一样调节视频评论。

##### SEO 价值插件

最后，但同样重要的是，我不能写一个插件列表而不把 [**全部包含在一个 SEO 包**](http://wordpress.org/extend/plugins/all-in-one-seo-pack/)——我最喜欢的 WP SEO 插件中。它很容易使用，对于页面标题优化来说非常强大。该插件允许你调整页面标题，元描述和元关键字，它还允许你通过添加“noindex”到类别，档案，标签等来避免重复的内容惩罚。

你有没有这里没有列出的最喜欢的 WP 插件，也可以为网站社区创造更好的体验？评论框欢迎您的建议！
