# 复习–在 24 小时内自学 CSS

> 原文：<https://www.sitepoint.com/teach-yourself-css-24-hours/>

级联样式表是网页开发的重要组成部分。使用它们的好处远远超过了 Web 开发人员学习如何在所有浏览器中应用和调整它们以获得最佳效果所花费的时间。在“24 小时自学 CSS”中，Kynn Bartlett 接受了挑战，解释了如何使用样式表来创建有吸引力的、可用的网页设计。

##### 封面和内容

当我第一次拿到这本书时，我首先被它的封面吸引住了。我仍然不太确定它意味着什么，但至少它是不同的。快速浏览了一下这本书，让我确信它很容易阅读，而且各部分都是按逻辑顺序排列的。有大量的标题来划分信息，代码以不同于正文的字体打印，这使得很容易与注释区分开来。

当我第一次打开一本书时，我总是很快地扫一眼目录，这次也不例外。它的结构似乎是这样的，如果我对 CSS 一无所知，我可以从头到尾读完这本书，或者如果我想重温这方面的知识，我可以直接进入我感兴趣的部分。第一部分是 CSS 的一般介绍。第二部分首先解释 CSS 盒子模型和 CSS 类的继承属性(“CSS 的核心原则”)。第三部分继续使用级联样式表来设计文本、背景、列表、表格、一般页面布局和 Web 设计。

##### 它能送货吗？

实际上，我从来没有看过一本声称作者可以在 24 小时内教会我一些东西的书，所以我很自然地有点怀疑这本书能否做到。然而，正如我所发现的，这本书可以让初学者在几个小时内上手并运行。前两个小时，任何精通级联样式表的方法和原因的读者都会看到，创建一个具有足够魅力的工作样式表，即使是最苛刻的学生也会感到满意。

成功实现 CSS 的关键在于理解各种浏览器如何使用和解释 CSS。这本书采取了独特的立场，教导读者如何根据浏览器支持和他们决策的预期效果，对如何以及何时应用 CSS 做出明智的判断。我真的很喜欢这种方法——毕竟，任何网站经得起未来考验的关键是确保它能在尽可能多的平台和浏览器上运行。

这本书里我最喜欢的一章是关于可访问性和国际化的。我真的很喜欢作者采用万维网联盟的 Web 内容可访问性指南，并在级联样式表中提供解决方案的方式。CSS 允许广泛的用户访问——并且该语言中有许多功能可用于帮助那些有视觉障碍的人。

本章还包含了 CSS 听觉属性的简要概述。我确实觉得这本书可以对听觉样式表有更详细的了解——即使它们目前不被许多浏览器支持……但是也许作者把这个话题留到了他的下一本书！

总之，对于任何想学习如何正确使用级联样式表，以及如何让它们在尽可能多的不同情况下工作的人来说，这是一个必须购买的东西。这是一本非常棒的书，适合初学者，或者任何已经知道一些 CSS 并想重温知识的人。
