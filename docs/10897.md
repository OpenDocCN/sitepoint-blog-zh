# 都是关于橘子的

> 原文:[https://www.sitepoint.com/its-all-about-the-orange/](https://www.sitepoint.com/its-all-about-the-orange/)

有几个星期，当我坐下来写这个博客时，我不知道该说些什么。这并不是说事情并不总是发生在 [SitePoint 论坛](https://www.sitepoint.com/forums/)上，但有时这些事情可能只有我感兴趣。或者我还没准备好向他们公开。或者我在周日下午喝的酒的数量让我很难记住我一周都做了什么。但不是今天。今天，我们的 SitePoint 社区发生了太多的事情，我不知道从哪里开始。

因为我最近被指责没有忠于我的头衔，所以我最好从橙色开始——这意味着论坛。上周发生了一些审美上的变化。其中最明显的是帖子的颜色(每个论坛顶部标有“重要”的帖子)。我们最近一直在努力清理这些帖子，以便它们是最新的和相关的。现在，我们希望更多的人会注意到它们，并按照他们的意图使用它们——提供 FAQ 类型的信息。因此，如果你有一个问题，你肯定以前被问过，在问之前快速搜索一下这些帖子。如果我们建议你去那里看看，请不要生气。

我还应该注意到，公众对这种“鲑鱼色”有一些抵触情绪，所以我们会在本周稍微低调一点。

更多关于橙色的主题…我们有了一个新的“旗杆”图标来代替旧的“报告杆”图标。您的反馈让我们清楚地知道，人们不确定在什么情况下举报帖子是合适的。一段时间以来，我们一直要求您报告您认为出于任何原因需要版主关注的帖子。我们需要你的帮助，以保持论坛的友好和免费的绒毛和垃圾邮件，所以我希望新图标将使事情不那么令人生畏。

![SitePoint forums](../Images/0aab08ac7da3eae1d2d833fb03220b40.png "sitepoint-forums")

足够的管理-现在是激动人心的事情。今天是[官方 SitePoint T 恤大赛](http://99designs.com/t-shirt-design/contests/sitepoint-needs-official-t-shirt-45558)的倒数第二天。参赛作品纷至沓来，有一些真的很棒。只选择五个将是一个艰难的决定，但这就是我明天要做的。在接下来的几天里关注[新闻和公告](https://www.sitepoint.com/forums/forumdisplay.php?f=38)论坛——你将有机会参与投票来决定获胜者。

最近我投入大量精力的一件事是我们的脸书粉丝页面。它已经得到回报，我们刚刚达到 5000 球迷的标志。我对那个成就很满意。事实证明，我们的 SitePoint 'I < 3 HTML 咖啡杯非常受脸书粉丝的欢迎。几周前我们赠送了一台，据说人们都渴望得到一台自己的。所以现在你可以了。在[我们的 Zazzle 商店](http://www.zazzle.com/sitepoint)查看它们。

最后，祝今天开始 [PHP Live](https://www.sitepoint.com/courses/php-live) 的所有人好运。我希望你会发现它既有挑战性又令人愉快。对于那些对事物的外观比对其工作方式更感兴趣的人来说，请查看将于 7 月开始的 [CSS Live](https://www.sitepoint.com/courses/css-live) 。
