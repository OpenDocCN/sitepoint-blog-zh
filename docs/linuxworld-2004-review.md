# LinuxWorld 2004 回顾

> 原文：<https://www.sitepoint.com/linuxworld-2004-review/>

我靠在狭窄的座位上，看着织物天花板。外面，夕阳从一座工厂被熏黑的屋顶上掠过，橙色的天空中升起滚滚浓烟的黑影。我扭过头去够后座上的相机。

“塑料”他说。
“什么？”
“天花板看起来像织物。它甚至摸起来像织物，但是如果你注意一下
，你会注意到它是塑料的。他们在所有合适的地方削减开支。它拥有所有的功能、所有的效率，但它的成本只是价格的一小部分。”

我把工厂框起来，按下快门。完美。我向后一靠，手指划过天花板。他是对的。

“感觉到塑料了吗？是啊。更容易清洗。”

一架飞机从头顶飞过，我拍下了它在夕阳下的侧影。污染对地球的对比度产生了奇妙的影响。

“那么，你觉得怎么样？你得到你需要的信息了吗？你觉得[麦芽](http://www.sussdorff.de/journal/)怎么样？”
我想了一下。“这和我想象的完全不同。”

前方，汽车是如此多的沙粒，滴在每天早上流入纽约市的沙漏中，然后翻转过来，每晚重复一次。回家的路会很长。

说实话，我不确定对 LinuxWorld 2004 有什么期待。GNU/Linux 和开源的世界就像我们的地球一样多样化。整个生态系统围绕新技术崛起、竞争和衰落。挤满了商务隔间的走廊或挤满了极客的咖啡店可能会突然获得成功，也可能会慢慢挨饿，直到消失。剩下的人被重新吸收，继续养活新的大公司，另一个有前途的项目，下一个明天。

当我和帕特里克走出科帕卡巴纳的停车场时，我把帽子拉了下来——他们把汽车架在液压升降机上以节省空间——距离贾维茨中心的玻璃幕墙只有两个街区。强烈的逆风让我希望我有像帕特里克一样的眼镜，即使他的角质架让他看起来像一个矮矮胖胖的奥斯汀·鲍尔斯。

“Linux 无处不在”我说。
“是吗？”帕特里克问道。
“IBM 广告牌”我解释道，并朝海滨点了点头。
“那我们一定来对地方了”他回答道，然后打开了门。

之前不确定的话，一介入就知道是 LinuxWorld。当然，巨大的横幅和行走的企鹅服给了我们提示，但“正常”参与者的外表也给了我们提示。生态系统？绝对的。帕特里克安排我们的通行证时，我四处看了看。这看起来更像一个极客加拉帕戈斯，而不是一个贸易展览，每个物种都有不同的羽毛。在一个角落里，一群青少年展示印有`#/bin/perl`的 t 恤，而不是通常的旧海军或阿伯克龙比和惠誉。不远处，一个男人咧嘴笑着，从他大理石般的灰色头发上摘下羊毛帽子。他把它塞进自己的风衣里，跟着兴奋的儿子走进了秀场。

“隔间墙比孩子更容易让人变老，”我想，然后信步走向洗手间。几个衣冠楚楚的西装笔挺的从我身边走过，他们把干净的米色夹克拉直，用耳挂式手机拨打紧急电话。只有一个小细节暴露了他们，比如齐肩的头发(当然，要仔细洗发和梳理)或者颜色保守的运动鞋。擦鞋人没有从报纸上抬起头来。他们今天不会有多少生意。LinuxWorld 是我唯一见过男厕所排长队的地方。

"给你，内特。"帕特里克把徽章递给我。上面写着“参展商”。除了在会议上闲逛，我还打算和他一起在一个参展商的展位上工作一段时间。
“会议人员都在用 Windows，你知道的，”他小声说。我笑了。

我们走上铺着地毯的博览会地板，经过甲骨文公司巨大的短程圆顶冰屋，经过看起来邪恶的机架服务器上无数闪烁的灯，进入博览会的 Dot Org 区域。在这里，一小群人聚集在令人兴奋的开源项目和非营利组织的展台周围。“就是它了。”帕特里克指着 OpenACS 展台。我把背包扔到桌子后面，拿出笔记本。
“我们暂时不需要你了。随便逛逛。你回来后，我会把你介绍给马尔特。”

世博会的地板比我想象的要小得多。大约半个足球场大小，大部分空间被大公司占据:IBM *(Linux 无处不在)*、Novell *(我们投资 Redhat)* 、SGI *(我们销售漂亮的电脑)*、甲骨文*(网格计算的牢不可破的 Linux)*，甚至微软*(Unix 的工具)*。几乎每家公司都建造了倾斜的人造墙，墙上排列着性感的机器，就像一个巢穴，一个通往中心的迷宫，销售主管们在那里准备向毫无戒备的旁观者发布他们的推销。在周界上，一个微妙的凸起地毯标志着掠夺区，试图看起来令人讨厌的公司员工远在边缘，但现在更远了。走进去……然后*砰！我避开他们的目光，好像他们每个人都是一个企业美杜莎的头。如果我看着它们，我会被冻僵，面无表情，无法逃脱，直到我听说高可用性复制或网格计算的细节，或其他一些技术、术语和鸡骨头的混合。*

此外，我对最快的机架式服务器、最小的笔记本电脑或最昂贵的 24/7 支持不感兴趣。两个穿着闪亮的白色塑料排毒服、背后插着吸尘器软管的男人透过他们薄薄的塑料护目镜盯着我。Linux 杀毒？我逃到了灌木丛中，从惠普显示器前的人群中出现。

网络公司在哪里？几乎每个人都在销售高性能服务器、重型数据库软件或桌面系统。

##### 谷歌

在一个角落里，我看到了谷歌标志的友好颜色。现在有一家公司将它整合在一起。我走上前去，听到了一段对话的结尾，

“那么，你来这里干什么？”
“哦。谷歌正在招聘。我们认为我们会在这里找到一些好人。”他们尴尬地看着我。我回头看，觉得同样尴尬。
“我想知道的是，谷歌是如何做到的？我是说，我什么都用谷歌！”有人从我身后涌出。
“当然是鸽子！”我不假思索地回答道。

几年前，谷歌曾就他们的专利技术[‘鸽子等级’](http://www.google.com/technology/pigeonrank.html)发布了一个幽默的解释。在我有机会思考之前，我的大脑直接连接到了我的嘴巴，我开始说话。

“粪便被转化为所有那些白色的小像素”现在停下来已经太晚了，所以我完成了谷歌的自我模仿。

谷歌代表没有笑，而是怒视着我。我不认为他们知道我在说什么。我向前看了。

##### OpenACS

在另一个角落，隐藏得很好的是自由软件基金会，GPL 许可证的创造者和开源思想的创始人。我捐了一些钱，然后走回 [OpenACS](http://www.openacs.org) 展台去看 Patrick 在做什么。他正在和一个高得多、同样矮胖的家伙说话，这个家伙穿着卡其裤和一件淡蓝色的纽扣衬衫。

"嗨，我是[马尔特·苏斯多夫](http://www.sussdorff-roy.com/)，你是内森？"
他高亢、和善的声音与他圆润的笑容和谦逊、梳理整齐的金发相得益彰。我向他询问了 OpenACS，一个用于网络社区的工具包。Matle 解释说，大多数使用 OpenACS 的网站不需要开发任何代码。ACS 系统自带许多模块，可以执行大多数网站任务。

OpenACS 看起来就像一个完美集成的内容管理系统。没什么惊天动地的。然后 Malte 给我看了 [DotLRN](http://dotlrn.org/) ，一个用 OpenACS 写的开源应用。从注册课程到与教授和其他学生合作，DotLRN 可以管理整个大学的内部网站需求，这看起来很简单。每个学生都可以很容易地组织一个个人门户，包括约会、上课时间和俱乐部会议。教师和学生可以使用 DotLRN 创建屏幕演示、共享书签列表、创建照片库、在线聊天、跟踪课堂讨论、向班级成员分发新闻和作业、收集上传的作业，甚至分发成绩。几乎所有内容都有修订历史记录。DotLRN 由许多顶级大学资助，包括麻省理工学院、剑桥大学、海德堡大学和悉尼大学，他们不想为一个他们无法控制的解决方案支付数十万美元。因此，他们创建了 DotLRN，并将其置于开源许可之下，这样每所大学都可以根据自己的需求对其进行定制。

##### 曼波舞

紧接着 OpenACS 的是 Mambo，一个传统的 PHP/MySQL 内容管理解决方案。曼波展台挤满了二十多岁、时髦的纽约网络设计师，他们炫耀着精心设计的时髦半染头发。这些孩子对星巴克来说都太性感了。甚至有一个女孩，但她留在其他设计师后面，没有回答很多问题。
“经典的布斯宝贝，”Patrick 后来评论道。

Mambo 最突出的特性是用于管理操作的 javascript 下拉菜单。“它能在 Konqueror 或苹果的 Safari 上运行吗？”我问代表
“不完全是，”他回答道，熟练地转向下一个问题。他穿着合身的毛衣，说话时摆出不同的姿势。我没有问他 CSS 菜单的问题。

虽然 Mambo 是一个漂亮的工具，但它似乎过于模块化。软件中的每个功能都很好地适应了一个布局框，但是它们之间并没有太多的交互。也许我太苛刻了。大多数内容管理解决方案都是这样，但我刚刚在 OpenACS 展台上看到了一些非常紧密集成的软件，我渴望得到更多。
“支持文档版本跟踪吗？”
“嗯，我们有自己的版本跟踪系统，但是如果你想进一步改进，你可以这样做。”

他开始和我后面的人说话，我继续往前走。在下一排展台，一大群人围着一个用户组展示一个自制的基于 Linux 的 [Dance Dance Revolution 控制台](http://icculus.org/pyddr/)。

在拐角处，我遇到了友好的领地，[纽约 PHP 用户组](http://nyphp.org)。在这里，一群不同的老年人、中年人和青少年程序员交换了战斗创伤和编码技巧。我和设计师、程序员、网站所有者谈过。

"嘿，你应该看看我们的邮件列表."
“是吗？”
“挺活跃的。我们的名单上有将近一千人。这是地址:【nyphp.org
“我去看看。”
“此外，我们有很多演讲者参加我们的会议。每次有会议，我们都把它以 MP3 格式上传到我们的网站上。我们欢迎任何人加入，即使你不是纽约人。”“谢谢。”

在 NYPHP 展台对面，一个扩音器里传来了一阵活跃的谈话声。这看起来像是一个广播节目在采访开源程序员。我没有停下来；在我回去帮助帕特里克之前，我还有最后一个地方要检查。

##### 关系型数据库

找到 MySQL 的展台很容易。我只是跟着光环走。在一个由海豚和公司名称围成的巨大帆布圆盘下面，MySQL 区域看起来像一个未来的游戏展示。一个角落的弧形墙自豪地展示着营销信息，宽大的装饰性讲台从毛绒地毯上延伸出来，仿佛通过看不见的辐条与角落相连。

主席台上展示着大型液晶显示屏；MySQL 的每个战略合作伙伴都有一个。我听一家公司的代表谈论数据库集群解决方案、高可用性和单点故障。之后，我跳过了其他人，直接进入了 MySQL 的主平台。

MySQL 的下一个版本将在几周内发布，将提供许多改进，包括子查询和存储过程。数据库中的引用完整性也将有许多改进。 [MySQL 版本 5](http://www.mysql.com/downloads/mysql-5.0.html) 将于今年夏天发布，它将允许内存数据库……”

“很好”我说。他几乎没有停顿。他势如破竹。
“…以及用于高可用性的同步数据库。它还将支持高级复制功能和数据库集群。”
“嗯，那肯定会让很多人高兴的。还有什么我可以期待你们尽快回复的吗？”

他把我领到对面角落的讲台上。"和那边的一个家伙谈谈我们正在开发的新图形用户界面工具."

当我走过去的时候，我听到他告诉另一位与会者，“不，我们并不是真的想与 Oracle 这样的高端数据库竞争。他们有他们的定位，我们有我们的。”我咯咯地笑了。复制和集群将于今年夏天推出，他们不会试图与大公司竞争。是的没错。

“他们让我问你一些新的 GUI 工具，”我问讲台上穿着整洁的销售员。他递给我他的卡，把他的笔记本电脑插入大液晶显示器，摸索了一会儿。它无法启动。“很抱歉。这就是为什么我们总是有一个备份，嗯？”
售货员摸了摸其中一个道具上的插销，掏出一个苹果 PowerBook。他关上了道具。很聪明，我想。
但是他似乎无法让液晶显示器正常工作。“你介意我帮你吗？”我问，并迅速帮助他设置显示。
“谢谢，我不经常用苹果电脑。”毕竟那是 LinuxWorld。

这位销售人员通过 VMWare 在 Mac 上推出了 Windows，启动了新的 [MySQL 管理软件](http://www.mysql.com/products/administrator/index.html)，并开始了他的推销。

“没有更多的配置文件，如果你不想使用它们。您可以在 MySQL 中配置的一切都可以在这里配置，从用户和权限、备份到诊断和统计。”我印象深刻。

“该工具的 alpha 版本将于 2 月初发布。它在 Linux 和 Windows 下都是原生编程，尽管 OS X 版本将紧随其后。事实上，这只是几个工具中的第一个。”

在我身后，十到十五个人挤在一起听消息。那天是星期五，他花了两天时间精炼他的演讲，他有一群专心致志的听众。他在那个区域。

“MySQL 根据合同将在今年夏天生产一个 SQL 查询生成器和一个类似 MS Access 的图形数据库布局设计器。他们甚至雇佣了四名新的程序员来开发 GUI 工具。”

最大的好处是什么？每一个 GUI 工具都将在 GPL 下发布，并且绝对免费。他结束了他的演讲，转过身去和我后面的人说话。我在那里站了一会儿，但他专注于人群。推销又开始了，所以我离开了。

我看了看卡片上的标题:GUI 程序员。他们从哪里找来这些人的？

##### 会见 Maddog

在我返回 OpenACS 展台的路上，我注意到一个穿着深蓝色运动衫的大个子，他卷曲的灰色头发向四面八方散开，头上戴着一顶巨大的软帽。那是约翰·麦道格·霍尔，神圣的内核开发者之一。

我停下来和他说话。与其说 Maddog 是怪胎，不如说是祖父，他是我这一整天遇到的最有风度的人。

“嘿，这是我的名片。我有家人在你附近。给我发一封电子邮件，我会去和你的 Linux 用户组谈谈。”
“真的吗？”
“当然。我期待着你的邮件。后来的西亚。”

我急忙去告诉帕特里克。正如我解释的那样，Malte 听到我说了一些关于 MySQL 的新功能，并借口要看看 MySQL 是否会支持 OpenACS 所需的功能。帕特里克说:“我希望我没有冒犯他。
“呃？”
“我在吃[阿特金斯](http://atkins.com/)减肥法，对吧？我有一罐打折买的维也纳香肠。这种饮食我应该主要吃蛋白质。”
“然后呢？”“嗯，我给了马尔特一些。我想他可能饿了。但他拒绝了。”
“也许他不饿？”“嗯，他不是维也纳人。他来自汉堡。”

现在轮到我在 OpenACS 展位上帮忙推销了。我向企业主咕哝了一些关于垂直企业解决方案的参考资料，并向罗格斯大学的 IT 人员讲述了不同软件生态系统中的 Web 应用工具包。当我需要了解一些情况时，我会推推帕特里克，他正在像发糖果一样分发名片，为他的虚拟主机公司招揽生意。

然后我注意到咧着嘴笑的技术人员把一袋袋战利品扛在肩上(甲骨文正在分发豆袋椅)，他们的身体在叉车的闪光灯下跳动。我看了看我的 PDA 时间。会议结束了。当我拉上背包的拉链时，贾维茨中心的员工拉起方块地毯，把它们堆在滚动的手推车上。

机架式服务器上闪烁的发光灯一个接一个地熄灭了。没有这场疯狂的 LED 迪斯科表演，会场看起来单调乏味。最后几个参展商拖着行李箱走向巨大的玻璃幕墙，我拍了最后几张照片。帕特里克和我冒着刺骨的寒风走回科帕卡巴纳餐厅。LinuxWorld 结束了。

我们看着服务员把帕特里克的车从液压支架上取下来。回家的路会很长。
