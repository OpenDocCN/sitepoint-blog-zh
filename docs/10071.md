# 标志设计排版:Helvetica

> 原文:[https://www.sitepoint.com/logo-design-typography-helvetica/](https://www.sitepoint.com/logo-design-typography-helvetica/)

设计师使用字体来传达一个公司的态度、历史和文化。选择一个合适的标志不是一件容易的事情。除了字母的整体易读性之外，还必须考虑字母组合的形状。对于许多标志设计师来说，这是 Helvetica 进入框架的地方。Helvetica 可能被认为有一个普通的外观，但也许这就是为什么它如此受欢迎。这是无害的，容易接近的，有吸引力的。事实上，它吸引了包括微软在内的一些世界大品牌，并被用于平面设计师 Massimo Vignelli 创作的纽约市地铁地图和标牌系统。你可能会惊讶于有多少大公司在他们的标识中使用这种字体。

Helvetica 已经五十多岁了。它最初被称为 Neue Hass Grotesk，不太像卷舌。经过 Mergenthaler Linotype 的更名和推广，这种字体大受欢迎。它因其现代、创新和干净的风格而广受欢迎，这也是许多公司想要传达的特征。

Helvetica 甚至有自己的电影。加利·哈斯特维特最近拍摄的一部纪录片充满了人们路过建筑物和使用 Helvetica 产品的日常场景。

最好的标志设计需要在不同的媒体和不同的尺寸下工作，从一个小的手机标志到一个巨大的广告牌。因此，Helvetica 简洁的线条大受欢迎。字母和数字的末尾没有令人分心的衬线或华丽的词藻。松下在产品上使用这种字体，通常以微小的尺寸出现在电子设备的侧面。汽车制造商吉普在他们的车辆前部使用这种字体，当它行驶时很容易阅读。其他动态 Helvetica 的例子包括汉莎航空公司和美国航空公司的标志。这种字体出现在他们飞机的侧面，在着陆和起飞时易于阅读，传达了一种功能感和现代感。

尽管 Arial 字体与微软关系密切(Arial 是 Helvetica 的派生物)，但该公司的标志使用了斜体 Helvetica。少量的定制出现在字母“o”上，它的侧面有一个切口。

Helvetica 被描述为几乎像一个模板，可以改变字距和字体风格，例如，轻或额外的粗体，以实现不同的，但相似的东西。为了让你看得开心，这里有一个小展示，展示了一些在商标中使用 Helvetica 的著名品牌。
