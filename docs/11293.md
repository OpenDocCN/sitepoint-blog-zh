# Mozilla 发布 Firefox Mobile 1.0

> 原文:[https://www.sitepoint.com/mozilla-firefox-mobile-1/](https://www.sitepoint.com/mozilla-firefox-mobile-1/)

Mozilla 的 Fennec 项目终于为基于 Maemo 的手机(如诺基亚 N900)发布了第一版 [Firefox Mobile](http://www.mozilla.com/mobile/) 。该浏览器提供了针对手持设备优化的现代浏览体验，包括:

*   HTML5、CSS2.1、CSS3、SVG、Canvas 和桌面浏览器中可用的大多数较新技术。
*   一个干净整洁的界面，最少的屏幕控制，一键式书签和棒极了的工具栏。
*   选项卡、缩放、搜索引擎集成、离线浏览、下载管理器、密码管理器、弹出窗口拦截器和拼写检查。
*   Mozilla Weave 可以将你的书签、打开的标签页和历史记录与你的桌面电脑同步。
*   位置感知浏览，获取与您当前位置相关的地图和信息。
*   全面定制、自动更新和附加支持。

如果你想在本地电脑上测试移动开发，这款浏览器也可以在 Windows、Mac OS X 和 Linux 上使用。

![Mozilla Firefox Mobile](../Images/70425847d54ea5bb09c00106692f3943.png)

毫无疑问，这是一个很棒的浏览器，如果你是一个使用火狐浏览器的 N900 用户，值得考虑。我希望它能做得很好，但是，实际上，有多少人能安装这个浏览器呢？随着 Windows Mobile 即将推出，1.0 版可用于顶级 Maemo 设备。然而，Android 开发才刚刚开始。更令人担忧的是，苹果 iPhone/iPad、黑莓或 Symbian 平台上的版本还没有计划。Mozilla 表示，他们愿意与苹果合作生产 iPhone 版本，但鉴于他们最近的历史，我怀疑苹果会急于提供 Safari 的替代品。

也许 Mozilla 正在展望未来，希望几年内我们都能拥有兼容的设备？但我不禁认为他们有点过于雄心勃勃了。他们是否应该开发一款更简单的浏览器，可以安装在各种移动设备上？随着市场的发展，可能会开发出更先进的版本。

幸运的是，Opera 仍在生产许多质量上乘的移动浏览器，它们可以在各种大众市场设备上运行。

你在手机上试过 Firefox Mobile 吗？是否优于你的其他手机浏览器？您是否考虑在移动平台上开发 web 应用程序？
