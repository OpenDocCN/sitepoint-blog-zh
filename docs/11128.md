# epub 和 mobi 阅读器:阅读 EPUB 或 MOBI 格式的电子书

> 原文：<https://www.sitepoint.com/how-do-i-read-these-epub-or-mobi-format-ebooks/>

为了庆祝我们大部分图书增加了 EPUB 和 MOBI 格式版本，SitePoint 发起了大规模的 5 对 1 销售活动。然后，我们认为您可能会寻求一些关于如何利用这些新格式的建议。所以我探查了我们的常驻电子书专家[凯文·杨克](https://www.sitepoint.com/blog/)的大脑腔，并提取了一些黏糊糊但美味可口的有用知识。

电子书软件的功能可以改善你阅读书籍的方式。像图书馆管理、书签、笔记和搜索这样的功能非常有用，尤其是像 SitePoint 这样的教育类图书。这里有几个电子书阅读器软件选项，你可能想看看。

[Adobe Digital Editions](https://www.adobe.com/products/digitaleditions/) 是适用于 Windows 和 Mac 的桌面电子书软件，支持 EPUB 和 PDF 格式。

Calibre 是一款非常强大的电子书软件，适用于 Windows、Mac 和 Linux。它支持多种格式，包括 PDF、EPUB 和 MOBI；它甚至可以将一种格式转换成另一种格式。

如果你是 Kindle 的用户，你需要使用 MOBI 格式。你可以通过 USB 线将 MOBI 格式的电子书下载到你的设备上，这真的很简单，正如我在这个快速截屏中演示的那样:

还有一个选择，当你设置亚马逊 Kindle 账户时，你会被分配一个特殊的电子邮件地址。你可以通过电子邮件将电子书文件发送到这个邮箱，它会通过 Whispernet 无线传输到你的 Kindle 上。这项服务将收取少量亚马逊费用。

Kindle 软件也适用于 Windows、Mac、iPhone 和黑莓，并支持 MOBI 格式的电子书。

如果你是 iPhone 用户，我们强烈推荐[节](https://www.lexcycle.com/)。iPhone 版本只支持 EPUB 格式。要把你的 EPUB 文件放到你的手机上，你需要安装 Calibre(如上所述)并从你的电脑上共享你的库。从你的 iPhone 上的 Stanza，你可以从你的 Calibre 库下载它。Calibre 网站上有一些[的说明。](https://calibre-ebook.com/user_manual/faq.html?highlight=stanza#how-do-i-use-app-with-my-iphone)

如果你不喜欢 Calibre 的外观，你可以为 Windows 或 Mac 抓取 [Stanza Desktop](https://www.lexcycle.com/download) 。它在阅读 EPUB 书籍方面做得不太好，但你在其中打开的任何 EPUB 书籍都会通过你的 wifi 网络共享到 Stanza iPhone。只要按照 Stanza 网站上的说明去做[。](https://www.lexcycle.com/faq#sharing_via_stanza_desktop)

顺便说一下，据预测，iTunes 9.1 将支持将 EPUB 格式的电子书添加到 iTunes 资料库，以便传输到 iPad 上的 iBooks。你能在你的电脑上阅读这些文件吗？我们都得等着瞧！
