# 提出一个想法，并付诸实践

> 原文：<https://www.sitepoint.com/take-an-idea-and-run-with-it/>

本周席卷我们办公室的是我们最新的促销活动——这个杯子。我不是一个营销型的人，我不会用这个博客来吹嘘这个想法的高明之处——如果你感兴趣，你可以自己去看看。我提出这个问题的原因是，它发生的方式是我们都可以借鉴的。

几周前，我坐在办公室里，我们的一名会计师走下来说:“嘿，我们应该围绕非洲的足球节做一些营销工作”，然后就走了。5 分钟后，市场部的人消失了，去参加一个紧急会议。15 分钟后，他们出现了，看起来很兴奋。一个星期后，我们准备带着这个奖杯出发。SitePoint 的这种“提出想法并付诸实践”的态度使它成为一个令人兴奋的工作场所。

每个人都参与到某件事情中，并在自己的业务领域开展工作，使其正常运行。有时候这些想法比其他的更有效。如果我在 Twitter 上看到的回复数量是值得肯定的，那么这一个是赢家。

除了愿意尝试新事物之外，企业还必须允许个人在自己的领域拥有自主决策的权力和自主权。强制执行严格的签署程序和创建官僚主义的繁文缛节是平息兴奋和购买的可靠方法，这是获得一些乐趣的必要条件。作为一家大公司的员工，我对这一点太了解了。

![SitePoint forums](img/a44ce429d6235abf9923f158fbac0082.png "sitepoint-forums")

无论如何，其他人的想法已经够多了，我自己也有一些想法。随着 t 恤比赛的结束和尘埃落定，我已经把获胜的设计变成了可以穿的东西，我们现在已经印刷了第一批样品。一旦我们对它们进行了质量检查，我就会给你一些东西。我等不及了！

另一件让我非常兴奋的事情是，目前正在进行的 [SitePoint 论坛](https://www.sitepoint.com/forums/)是我们的[伟大摄影比赛](https://www.sitepoint.com/forums/showthread.php?t=686674)。如果摄影是你的事，现在加入还不晚。[第一批主题已经公布](https://www.sitepoint.com/forums/showthread.php?p=4636861#post4636861)，今天下午开始提交。有一些大奖等着你去拿，所以不要错过！
