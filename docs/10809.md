# 在家管理办公室政治

> 原文:[https://www.sitepoint.com/office-politics-at-home/](https://www.sitepoint.com/office-politics-at-home/)

我不怀念企业界的一件事是永无止境的办公室政治。成为自己的老板无疑改变了这一点，但尽管我们不倾向于这样想，在家工作也不是没有政治。

“政治”部分来自发生在每段关系中的人际互动，它会影响你的工作环境。这是无法逃避的，因为不管你是谁，在哪里工作，做什么，关系都很重要。

虽然问题可能与办公室政治不同，但在家工作的政治同样棘手，难以忽视。

## 这些问题

当你在家工作时，政治可能会在任何时候突然出现，有人为了影响你的行为或观点而施加权力，即使这不是故意的。一些最大的罪魁祸首是:

*   不断挑战您现有政策和指导方针的客户
*   难以尊重您的工作空间的孩子
*   不支持你生意的家庭成员
*   其他人对在家工作的真实含义的假设
*   与同事甚至竞争对手的关系会影响你的行动
*   你自己尊重优先事项和个人时间的能力

## 远离拳击场

就像办公室政治一样，你必须意识到家庭办公室的问题，并有一个打击它们的计划，以避免被拖入这场戏剧。如果你不处理政治，你可能会失去从后果中恢复的宝贵时间。

避开政治的一些方法包括:

*   与你的沟通和期望保持一致
*   避免非建设性的抱怨
*   注重开放式沟通
*   立即解决问题，而不是等到事情升级
*   与一个能帮助你的家庭办公室政治专家结盟
*   采取措施保护你的时间和精力

如果你把关系和问题分开，在尊重关系的同时努力解决问题，大多数问题都是可以解决的。虽然，就像办公室政治一样，这需要做很多工作。

## 你的“政治”是什么？

在家工作的政治可以有许多不同的形式，通常会加剧我们自身的一些内部挑战。我的政治观点似乎完全建立在时间管理和外部在家工作观念的挑战上。我还没有完全摆脱这些问题，但我发现意识到这些问题并以清醒的头脑处理它们会使它们变得可控。

你在家工作，发现你还有政治要处理吗？你最大的问题是什么，你如何处理它们？
