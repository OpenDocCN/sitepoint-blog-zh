# Mozilla 取消了在 Windows Mobile 上开发 Firefox

> 原文：<https://www.sitepoint.com/mozilla-stops-windows-mobile-firefox/>

Mozilla 决定停止在微软 Windows Mobile 上开发 Firefox。尽管 Windows CE 6 上的浏览器[接近完成，但该组织已经确定 Windows Phone 7 不是未来版本 Firefox 的可行平台。](https://www.sitepoint.com/mozilla-firefox-mobile-1/)

自二月份宣布以来，微软一直在兴奋地宣传其 Windows Phone 7 系列。该公司认为，它已经创造了与苹果 iPhone、谷歌 Android、Palm Pre 和其他现代智能手机竞争的硬件和软件。

手机要到今年晚些时候才会发布，但看起来微软将遵循与苹果应用商店相似的营销模式。只能使用 Silverlight 或 XNA Game Studio 运行时环境开发应用程序。对 Mozilla 来说不幸的是，第三方开发者不能直接接触到手机的硬件。

Mozilla 移动工程总监 Stuart Parmenter 在他的博客上发布了以下评论:

> 虽然我们认为 Windows Phone 7 看起来很有趣，并且有潜力在市场上做得很好，但不幸的是，微软已经决定关闭对原生应用程序的开发。因此，我们目前无法为 Windows Phone 7 提供 Firefox。鉴于微软将他们在移动领域的未来押在了 Windows Mobile 7(而不是 6.5)上，并且因为我们不知道微软是否或何时会发布原生开发工具包，我们暂停了 Windows Mobile 的开发。

Mozilla 成为应用程序合作伙伴并获得该平台完全使用权的可能性很小，但鉴于微软希望推广 Internet Explorer 浏览器和品牌，这似乎不太可能。

那么这给 Mozilla 的 Fennec 项目留下了什么？这无疑是一个很棒的浏览器，但它只适用于诺基亚的高端 N900 和 N810 智能手机。Android 版本正处于早期开发阶段，但该组织将不会在 iPhone、黑莓、Symbian 或微软设备上出现。这是一个雄心勃勃的项目，所以很遗憾很少有人能真正使用这个应用程序。

如果 Mozilla 想进入移动浏览器市场，我建议他们效仿 Opera，开发一款可以安装在各种流行设备上的简单浏览器。当市场准备好了，就可以开发更高级的应用程序。

Mozilla 放弃 Windows Mobile 开发正确吗？这是 Fennec 项目结束的开始吗？
