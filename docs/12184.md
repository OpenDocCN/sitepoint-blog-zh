# 如何向客户追加销售附加服务

> 原文：<https://www.sitepoint.com/how-to-upsell-additional-services-to-clients/>

[![arrow](img/f1c35ce6a26dc7114d5bf4c7962ad508.png "arrow")](https://www.sitepoint.com/wp-content/uploads/2009/03/arrow.jpg) 虽然它可能有负面的内涵，但追加销售并不是一个肮脏的词。追加销售就是向客户销售附加服务的过程。当您专注于追加销售您的客户真正需要的服务时，您可以通过使用这种技术为他们提供巨大的价值。

为了进行追加销售，您不需要采用过度销售的方法。当追加销售恰当而真诚地进行时，它只是为您的客户提供优质服务的另一部分。这里有一些方法可以提升你的生意。
 **打包/升级**

包装服务是创造追加销售的好方法。例如，客户对标志设计感兴趣，但从与客户的交谈中，你知道他也需要设计一个网站。考虑将 logo 和网页设计服务捆绑成一个包，让客户更容易理解。这使得他们的两个主要需求中的一个成为可能吸引他们的交易。

您也可以创建升级来完成同样的事情。如果一个客户雇用你设计他们的网站，但她提到她还计划做一系列的 HTML 电子邮件营销信息，并为她的员工创建自定义名片，你可以创建一个升级包，其中包括一个价格的附加组件。

**提供限时折扣**

当客户与您签订特定服务的合同时，您可以通过以额外的价格向他们提供额外的服务来进行追加销售。与简单组合多种服务的套餐选项不同，您可以在有限的时间内提供附加服务折扣。这在诱使客户迅速行动方面也很有效。

**做一个自由分析**

一旦你与客户建立了关系，发现新机会的一个方法就是找出他们最迫切的问题。你可以为客户提供他们真正需要的改进的免费分析，然后解释你如何能做得更好。例如，分析他们过去三个月的分析帐户，并创建一份报告，概述他们网站上流量不大、退出率高的区域等。根据这些信息创建一个提案来解决问题。

**提供持续服务**

追加销售的最佳方式之一是为客户提供定期的持续服务。如果你创建了他们的网站，建议签订年度维护合同，每月进行链接检查，搜索引擎优化监控等。有很多提供持续支持的可能性，特别是对于网页设计客户。

**告诉他们你能做什么**

客户并不总是知道你的技能和经验的深度，因为他们在给定的时间专注于他们需要的特定支持。一旦你被录用，你的客户多久访问一次你的网站？大多数情况下不经常，所以他们没有太多机会看到你可能提供的新服务。定期与老客户联系，让他们知道你增加了哪些服务，给他们一个机会考虑你的新产品。

追加销售最重要的部分是与客户建立信任关系，然后只在合理的时候提供附加产品。关系是第一位的。这让你的客户确信你没有试图向他们出售他们不需要的东西，他们可能更有可能考虑额外的服务。
