# 在云端寻找大象

> 原文:[https://www . site point . com/finding-the-elephant-in-the-cloud/](https://www.sitepoint.com/finding-the-elephant-in-the-cloud/)

我相信我们都听说过这样一个比喻:一群盲人被带到一头大象面前，并被要求描述他们站在旁边的东西。他们被放置在大象周围的不同位置，并被要求触摸它。树干旁边的那个说像树枝。腿边的那个显示他碰到了一根柱子。在尾巴旁边，那个盲人说他在摸一根绳子。耳朵旁边的那个说他在摸一件皮斗篷。把他们聚在一起的那个人指出他们每个人都是正确的。因为他们每个人都触摸了大象的不同特征，所以他们的经历也各不相同。但是他们都在触摸同样的东西。

现在，我通常不会对这种禅宗感兴趣，但是考虑到任何关于云计算的讨论，这种类比似乎是合适的。我最初的博文指出了对云定义的不同观点。

最近一封回复最初帖子的电子邮件评论反对我对云的定义，并对它进行了更技术性的解释。因为这个评论来自一个开发者，所以他有不同的观点并不令我惊讶。这完全取决于你在哪里触摸大象…或者云。

美国国家标准与技术研究所(NIST)将云计算定义为三种服务模式:软件即服务(SaaS)、基础设施即服务(IaaS)和平台即服务(PaaS)。再次，从技术组织的技术角度来看。但是为我们的服务和软件付费的人——我们的顾客和客户——也已经开始使用云，并将其作为他们业务运营的核心部分。他们的观点和我们的有点不同。

## 云不仅仅是技术人员的专利

利用你提供的服务和软件的商业利益相关者认为云是模糊的。他们不像你一样是技术人员，看不到 API 背后的代码让他们能够通过云访问他们的数据。他们不知道自己的数据实际存储在哪里。无论走到哪里，他们只需通过浏览器查看数据。从他们的角度想一想——无论他们去哪里。那是手机吗？或者那是云？两者都有？

业务用户对云的定义是，他们从组织外部的某个地方访问您的应用程序和数据。他们不需要内部服务器，也不需要技术团队来支持他们的运营。他们让你照顾他们的需求。他们让您确保服务器已启动，所有软件补丁和更新已完成。他们让你确保他们的移动浏览器显示的内容与笔记本浏览器相同。您的应用程序为他们提供了完成工作所需的服务。本质上，他们将服务和数据管理外包给了你。当我们的企业客户想到云时，他们会想到外包。

当您构建任何云解决方案时，请记住您的客户对您的服务或软件的看法。他们看不到整个大象。他们只看到他们接触的部分。这取决于你把它们带到大象面前，并把它们放在大象周围，触摸它们需要触摸的部位。如果你让他们抓着尾巴，小心他们不要把尾巴举得太高。

## 抓住机遇

我的第一篇文章的另一位读者有不同的看法，并问他如何利用云计算的机会。他和我一样，是一个“经验丰富”的开发人员，仍然想写出优秀的软件并为客户提供服务。但他不知道从何说起。

我想我可以自信地说，对于一个人的企业或任何小型组织来说，今天是起步最容易的时候。云是让这一切变得简单的现象之一。

我不会过多地讨论创业步骤的细节，而是将重点放在一些促进创业并使创业阶段更容易的云服务上。从这个意义上说，不管你是开发人员、设计师还是有好主意的人，你都是在创业的商业用户。这就是你需要的视角。你订阅了别人提供的服务，而不考虑工作实际上是在哪里完成的。

需要注意的一点是，身在美国，我的观点偏向于美国的要求。我相信如果你在另一个国家，你会有自己的法律法规要遵守。不管怎样，在做我下面提到的任何事情之前，先寻求律师的建议。

## 组织启动任务

本着完全公开的精神，与一些针对这一活动的云服务提供商可能会说的相反，我认为一个特定的启动任务是正式建立您的业务组织。我的意思是选择你是否成立有限责任公司，或者你选择的任何结构。

有许多互联网服务和网站可以帮助你构建你的业务。但是我的经历和我与之交谈过的其他人的经历是相似的。互联网网站可以根据需要将你的文件归档，以合法地组织你的业务，但它们通常会遗漏特定于某些州的独特表格或费用。一些州有非常独特的要求，并且每年都有变化。找一个当地的会计师或律师来归档你的法律组织文件，让这些专业人士使用云来处理。他们还可以建议您哪种结构最适合您的具体情况。会计(出于税务目的)和法律服务是我更愿意保留在本地的两项服务，这样我就可以随时使用它们。

## 行政服务

有许多基于云的服务提供商愿意为您的组织提供会计、簿记和薪资服务。我当地的会计师推荐了几个我可以用来记账的账户，然后我只需要把结果发给他，让他在报税的时候工作。

这是一个至关重要的领域，我不知道为什么会有人想亲自去做。税收和商业立法的变化太多太频繁了。将 it 外包到云中不仅可以确保在税法和工资法发生变化时有人在照顾你，还可以让你腾出时间专注于自己的核心竞争力。

我用的是 CompuPay，一个工资单服务提供商，每次处理工资单的时候，他都会负责我的工资单和地税。我还使用 Intuit Quicken 跟踪我的簿记活动，并记录我的会计师每年需要的信息。这两项服务的成本都只是我内部找人提供的一小部分，并且只允许我为自己使用的服务付费。

## 构建您的云服务

另一位 CloudSpring 博客作者和 Amazon Web Services 布道者 Jeff Barr 已经在他最近的博客中列举了几个关于如何使用 Amazon 的云服务来支持软件开发的优秀例子。在谷歌上搜索“基于云的开发工具”会返回超过 3200 万个结果。开源软件和开发工具，无论是托管在云中还是您自己的服务器上，基本上都消除了您曾经被迫购买软件开发专用工具的启动成本。

云计算向您介绍了许多新技术和工具。没有一个人能跟上所有的最新工具，所以你要不断学习或雇佣外部开发人员来完成你的工作。几乎所有开发工具的培训都可以在 YouTube 上找到。有一些服务和网站可以帮助你找到优秀的人来提供你的解决方案。

你对云计算的正常定义包括使用 YouTube 作为培训资源或招聘平台吗？可能不会，但是记住你在这个例子中使用它的上下文。你在经营一家企业，不想将这些服务带到公司内部。使用外部资源完成任务是您的客户看待云的方式。

由于云服务，创业比以往任何时候都更容易。与创业活动相关的日常任务可以外包给专门从事这项工作的人，这样你就可以专注于打造自己的服务并获得第一个客户。

在这个更广泛的云定义下，您外包了哪些业务任务？无论你同意还是不同意我的观点，请让我知道你是如何专注于你的核心竞争力，并能够将这些其他任务推给团队以外的其他人。你可能已经找到了一项别人希望他们知道的服务。

感谢您的反馈！
