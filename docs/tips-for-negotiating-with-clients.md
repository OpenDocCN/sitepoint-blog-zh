# 谈判工具箱:与客户谈判的 5 个必备技巧

> 原文：<https://www.sitepoint.com/tips-for-negotiating-with-clients/>

在上周的一篇帖子中，我概述了为什么我反对与客户谈判你的利率(T2)。但在同意与客户合作之前，你可能还需要协商许多其他条款——付款日期、项目截止日期、具体的可交付成果和里程碑、工作所有权、文件和项目数据的移交等。这就是为什么在讨论关系的个别条款时，对谈判有一个坚实的理解并能够让它为你所用是至关重要的。

这里有 5 个建议，可以帮助你最大限度地与客户谈判，创造一个双方都满意的局面，准备好开始工作，并对关系感到兴奋。

**知道你的离开点**

在开始任何谈判之前，花点时间列一个清单，列出你最不想要的，但你愿意接受的可以忍受的条件。把单子放在你面前，如果客户想要的条件低于你的放弃点，准备好放弃这个机会。

**抛弃输赢思维**

如果你认为谈判是一个挑战，你要么得到你想要的，要么失去客户想要的，这可能不是一个成功的谈判，或成功的工作关系。不要专注于你想要什么，并试图说服客户给你，也要考虑客户想要什么，并努力达成妥协。

**别往心里去**

在谈判中，很难不让自己的情绪占上风。例如，假设客户要求两周的周转时间来完成一个通常需要四周才能完成的项目。你可能会想，“他真可笑！他不会对别人提出这种要求。只是因为我年轻/女性/兼职自由职业者/等等。”

相反，请记住，客户可能有一定的理由提出两周的请求。找出他为什么需要这么快完成工作，时间表是否可以协商，以及他是否愿意为加速周转支付更多费用。

**花时间进行教育**

很多时候，客户不知道某些项目通常需要一定的时间，不知道你有支付和交付的政策，以及你对工作的其他偏好。不要列出你的要求，告诉客户要么接受要么放弃，而是解释你为什么想要这些条款，你将如何做工作的过程，以及它将如何帮助工作关系，并从本质上为客户带来更好的结果。

**听**

在谈判中，倾听对方的意见有多重要，我怎么说都不为过。你可以尽可能做好准备，知道你的离开点，并准备好帮助客户理解你的观点，但如果你不听(并实际听到)客户在说什么，谈判就会失败。如果你不愿意倾听和修改你的条件，以达到一个双方都能接受的最终结果，你就不愿意谈判，也许应该就此打住。

如果你将这些建议融入到谈判中，你会呈现出一个更专业、更有经验的形象，而且很有可能你会发展出更令人满意的关系，并从工作中获得更多。请记住，并不是每个客户都是你的好客户，如果谈判到了某个阶段，你可以离开，为下一个客户做准备。
