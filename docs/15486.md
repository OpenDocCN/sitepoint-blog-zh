# 60 秒战斗:留住访客

> 原文:[https://www.sitepoint.com/battle-keeping-visitor/](https://www.sitepoint.com/battle-keeping-visitor/)

访问你网站的第一分钟是最重要的。它将决定访问者是会停留还是会转移到另一个网站。为了证明你的网站是有价值的，你的网站必须在访问者的心目中达到一定的标准。否则，点击一下，访问者(潜在客户，如果你经营一个电子商务网站)就消失了。

在输入网址或点击链接后的前 30 秒左右(希望不要超过这个时间)，访问者会等待你的网站下载。在大多数情况下，如果在最初的 20-30 秒内没有任何东西出现，访问者将会点击停止，转到另一个网站。拥有一个快速加载的首页是确保访问者留下来看你的内容的一种方法。

提示:如果你的网站是图形密集型的，在你的网站上有一个开启页面，警告访问者你的网站可能需要比平常多一点的时间来下载，并向他们提供一个替代方案，一个低带宽版本的
你的网站。

页面加载完成后，访问者会浏览一下，看看网站看起来是否专业，至少在某种程度上是如此。如果首页有 50 个屏幕长，有一个紫色的背景和绿色的文字，以及一个看起来像是用 Windows Paint 制作的标志，那么访问者就会离开。然而，你的网站不必看起来像 CNN.com 或 News.com，只要它提供好的内容。只要你拥有的信息是好的，即使设计不是很好，很大比例的互联网访问者会继续浏览你的网站。

访问者将会浏览首页来决定你的网站是否有用。如果他们来寻找一个特定的信息，他们会浏览你的网站，看看是否有一个部分/链接可以找到他们问题的答案。如果他们只是在网上冲浪，而不是寻找任何具体的东西，他们会看看你的网站是否有趣，是否有值得一读的内容。无论哪种方式，确保你的首页让你的网站内容非常明显。

如果访问者认为网站对他们有用或有趣，他们会试着掌握你的导航系统和网站组织。如果他们感到沮丧，很可能很快就会离开，除非你的网站提供了一些互联网上最引人注目的内容。这就是一个好的导航方案发挥作用的地方。请务必阅读[可用性 101](https://www.sitepoint.com/usability-engineering-101/) 以获得一些有用的提示和大量的好建议。

你网站的下一个目标应该是把到目前为止的访问者变成“常客”这可以通过在你的首页上贴一个通知来实现，上面写着“更新 __(每周、每天、每月等等。)”，拥有一份时事通讯，或者简单地让人们给你的网站加书签。阅读[营销](http://www.webmasterbase.com/article.php3?aid=15&pid=0)的另一面，了解更多关于如何充分利用现有流量的信息。

这最后一步只适用于电子商务网站。你必须将这些时事通讯的订阅者或访问者转化为付费用户。阅读:**将点击转化为顾客**和**人们为什么在网上购物？**了解更多关于将流量转化为销售额的信息。

祝你好运！
