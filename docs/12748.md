# 可伸缩性:来自狡猾野兽的牵引力

> 原文：<https://www.sitepoint.com/scalability-traction-from-a-slippery-beast/>

几年前，我有一家提供 IT 专业服务的公司。我们将专业服务定义为高级 IT 项目技能，如设计网络和调试 web 服务器群。

Edion 从一开始就快速发展，第一年销售额超过 100 万美元，第二年约为 300 万美元。然而，我们的销售停滞不前，我根本无法解决这个问题。

经过深思熟虑和与客户讨论后，我得出结论:我们的客户没有购买我们的服务；相反，他们正在购买我们的人。

我的员工有很高的声誉，这也是我的客户所追求的；谁宁愿等到有顾问可用(在一个案例中长达六个月)而不是使用其他人。

我不知道如何解决这个可扩展性问题，这似乎是业务中固有的问题。这就是我的线索。

在与我的一位心理学家朋友讨论这个问题时，他提出了一个让我至今难忘的建议。也许 Edion 不应该是可扩展的；也许我应该从中提取尽可能多的现金，然后把这些资金投资到其他方面。

我已经意识到，我并不是一个有这个问题的孤儿，事实上，我经常遇到销售产品或服务的人，并发现这是一项艰巨的工作。他们似乎有三个主要问题:

1.  一个单位的努力回报一个单位的回报，当他们停止销售时，他们就不再得到报酬。
2.  他们发现几乎不可能找到优秀的人才来扩大业务规模。
3.  他们发现，作为创始人，他们负责 90%的销售额。

如果这听起来像你，那么这个策略建议可能会派上用场。

1.  停止抗争，接受你拥有不可扩展业务的事实；然而，这并不是世界末日，而是一个激动人心的新阶段的开始。
2.  不要试图发展这项业务，因为这是浪费你的智慧和精力。有了你刚刚腾出的时间和精力，你就可以专注于从这个不可扩展的业务中获取最大利润。例如，停止对增长系统的投资，事实上将其削减到最低限度，并解雇那些低效的销售人员。
3.  利用现在产生的自由现金流投资于新企业的产品建设。这是对现有业务的补充，但也是可扩展的。

什么是可扩展的业务是一个不同的讨论，但是这并不难理解。环顾一下那些让白痴变得富有的企业，你会发现可扩展企业的大部分特征(比如手机、快餐、互联网服务提供商、汽油)。
