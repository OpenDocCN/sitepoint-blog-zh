# 那么第三方网页设计是一个死行业吗？

> 原文：<https://www.sitepoint.com/so-is-third-party-web-design-a-dead-business/>

你对上一篇博客的许多评论都很令人沮丧。你们大多数人都提出了两点中的一点:

1.为客户做网页设计报酬不高。在这个行业赚钱的唯一方法就是设计你自己的网站，并以这种方式赚钱。

2.大多数客户想要很多，但不会付钱。

如果这是真的，那么网页设计师就没有理由去做除了为自己设计之外的任何事情。

但是我觉得一点都不真实。我一直认为大多数人的想法都太小了。

我认为，一个人可以为自己的目的设计伟大的网站，也可以走出去，以高价向愿意为你能得到的结果支付大笔钱的公司和客户推销你的知识和智慧。如果你没有所有的知识，你可以和其他有知识的人合作，提供一个完整的解决方案。

如果我错了，而你正试图向那些想要很多却不愿意付钱的客户推销，那就滚出这个行业吧！有很多其他的方法可以让你过得很好，网页设计可以成为你的爱好。

否则，你们中的一些人需要一个重大的态度调整，需要开始变得更有创造力，看到可能性和机会。
