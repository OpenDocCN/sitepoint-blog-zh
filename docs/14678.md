# 如何推销可访问性

> 原文：<https://www.sitepoint.com/sell-web-accessibility/>

到目前为止，我们都听说过网络无障碍，以及它对我们互联网用户的重要性。你可能也这么想。但是你如何说服你的客户或者你的老板，可访问性是值得的呢？

总是有关于网站可访问性的法律要求的争论，但是，除非他们是雅虎！或者同等地位的企业，这不太可能会影响到你的客户。毕竟，全世界只有一个记录在案的由网页可访问性问题引起的法律诉讼案例。

但是，您关心可访问性。你希望你建立的所有网站都尽可能地易于访问，这意味着工作。对客户来说，这意味着更高的工程价格。那么，你如何证明可访问性的成本呢？

##### 商业案例

有两个很好的理由让你的客户开始考虑网页可访问性:

*   一个可访问的网站会让你的客户赚钱。
*   一个可访问的网站将为你的客户省钱。

如果这还不能引起他的注意，我不知道还有什么能引起他的注意。这里有八种方法可以帮助你的客户省钱或赚钱。

***1。他的网站将更容易管理。***

可访问的网站将每个页面的内容(HTML)和表示(CSS)分开。向桌子挥手告别！现在，要调整网站的布局，开发人员只需要对 CSS 文件进行更改，而不是修改网站的每个页面，从而节省了大量时间(因此也节省了资金)。

***2。他的网站将兼容新的浏览技术。***

在不久的将来，PDA、手机和车载浏览器都将被定期用于访问互联网。您认为您的客户知道仅在 2008 年就将售出 5800 万台 PDA 吗？(来源:eTForecasts)。

使用这些新技术的人通常是高收入的个人。为了吸引这些有利可图的观众，你的客户需要一个能在这些机器上有效运行的网站。(现在是向他展示他当前的网站在手机上看起来有多糟糕的好时机，使用 [Wapalizer](http://www.gelon.net/) 。)

***3。他的网站在搜索引擎排名中将会更靠前。***

通过让网站更容易被网络用户访问，你也让它更容易被搜索引擎访问。搜索引擎通常不能理解图像、JavaScript、Flash、音频或视频内容。为每一个程序提供替代内容，搜索引擎会更好地理解网站的目的。

在其他条件相同的情况下，搜索引擎对网站的内容越有信心，它在列表中对该网站的排名就越高。

你的客户是否知道 64%的互联网用户使用搜索引擎作为他们在互联网上寻找东西的主要工具？一定要告诉他！(来源:[凯尔西集团](http://www.kelseygroup.com/press/pr040211.htm))

***4。他的网站下载时间会有明显的提升。***

在英国，只有 17%的网络用户通过宽带连接到互联网。可以肯定的是，如果你的网站下载时间超过 10 秒，网站访问者就会成群结队地离开。

***5。他的网站的可用性将得到增强。***

网站可访问性和网站可用性之间有一定程度的重叠。

可用性专家 Jakob Nielson 已经证明了一个[可用性重新设计](http://www.useit.com/alertbox/20030107.html )可以将网站的销售/转换率提高 100%。当你告诉你的客户这个的时候，寻找他眼中的金钱符号！

***6。他会得到很好的宣传。***

“让每个人都可以访问你的网站，客户先生，你可以自豪地向全世界宣布！”

***7。他的网站不会导致他的业务招致法律费用(或负面新闻)。***

虽然客户认为这不重要，但现在可能是一个提及它的好时机，因为他会考虑通过确保他的网站可访问来获得良好的公关。

RNIB(皇家国家盲人研究所)和 DRC(残疾人权利委员会)一直在向商业和政府实体施加压力，以使其网站无障碍。DRC 目前正在调查 1000 个网站，以确定它们的可访问性。如果你客户的网站在这个列表上，他可以期待很快收到他们的回复。

***8。他会增加他的网站的影响力。***

这是一个很大的话题——让我们详细讨论一下。

##### 通过可访问性扩大覆盖范围

“残疾人不会访问我的网站，”你的客户可能会说。我为什么要在乎？"

好吧，客户先生，看看这些统计数据(英国):

*   英国有 860 万登记在册的残疾人，总计占人口的 14%(资料来源: [DRC](http://www.drc-gb.org/whatwedo/aboutus.asp) )
*   每 12 名男性中就有一人，每 200 名女性中就有一人患有某种形式的色盲，占英国总人口的 9%
*   两百万英国居民有视力问题——占总人口的 4%(来源: [RNIB](http://www.rnib.org.uk/xpedio/groups/public/documents/publicWebsite/P16_1214)
*   60 岁以上的人口有 1200 万，约占英国人口的 21%(来源:英国政府)

现在，这些群体之间肯定会有一些重叠，但如果你把数字加起来，你会得到英国人口的 48%。没错——48%的英国人，更不用说其他国家的人，在使用你客户的网站时可能会遇到问题。

##### 超越残疾

非残疾人也可能在网站上遇到困难。并不是每个人都使用最新版本的 Internet Explorer，其中包含所有插件和程序，可以让他们以最佳方式访问他们遇到的每个网站。

如果你客户的网站依赖于图片、Flash 或 JavaScript，并且不能提供替代方案，那么许多网络用户将无法访问他的网站。向客户解释以下内容:

*   慢速连接的用户通常会关闭图像以加快下载速度。有些浏览器，如纯文本的 Lynx 浏览器，根本不显示图像。
*   并非每个用户都下载了浏览网站所需的最新 Flash 插件。此外，Flash 网站上的下载时间通常很长，以至于低带宽用户失去了耐心，甚至不会等着看内容。请记住，英国只有 17%的网络用户通过宽带接入互联网。
*   大约 4%的网络用户不支持 JavaScript ,要么是因为他们关闭了它以防止弹出广告，要么是因为他们的浏览器不支持它。
*   网络电视、手机和 PDA 对大图像、Flash 和 JavaScript 的支持有限。下载[免费网络电视浏览器](http://developer.msntv.com/TOOLS/Webtvvwr.asp)，查看您客户的网站。

##### 他会答应吗？

在这种争论中，唯一会对无障碍网站说不的人是不道德的、厌恶金钱的人。如果你的客户是其中之一，我建议你逃跑。

否则，坚持下去，用这些论点说服他，也许再举几个已经响应了可访问性号召的大型知名网站的例子，你就能得到一个让你自豪的项目了！
