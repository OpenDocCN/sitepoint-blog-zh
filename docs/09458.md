# 机会和利润——网络商业的字母表

> 原文：<https://www.sitepoint.com/opportunity-and-profit-the-alphabet-of-web-business/>

我们又回到了网络商务字母表系列的另一部分；我相信你会发现这两个词是鼓舞人心和有帮助的。

O 代表机遇
我们很多人都在谈论创新或“不同的思维”，然而过去十年的许多伟大发明都是发现机遇的结果。为了取得成功，许多企业创造了一种产品或服务，但没有人提供，或者提供得很差。

每天寻找机会——经常听到客户抱怨，他们希望有一种简单的方法来发送营销电子邮件，或管理网络内容，或；你明白了。创造一个产品来帮助他们。寻找机会，然后展示产品是拓展业务的好方法。

**P 是为了利润**
把你的时间花在你的营业额上？除非你有固定的营业额利润百分比，否则你看的是错误的数字。开始关注你的利润，而不是你的营业额。营业额不允许你支付你的员工(或你自己！)更好，买新设备，多花营销。

让我们这样来看:一家 20 万美元盈利 20%的企业比一家盈利 3%的 100 万美元企业赚得更多——而且很有可能他们的客户更少，交易更少，管理费用更少，最终问题也更少！
