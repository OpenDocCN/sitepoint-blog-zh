# 什么是浏览器？谷歌解释说…

> 原文：<https://www.sitepoint.com/google-explains-browsers/>

![What's a web browser](img/8b633c7fbabc06abcc06d0ad3403d15a.png "What's a web browser")很简单的问题:*“什么是网页浏览器？”*不幸的是，[谷歌在纽约时代广场](https://www.sitepoint.com/what-is-a-web-browser/)的随机调查显示，只有 8%的人知道答案。其余 92%的用户要么不知道，要么把它和搜索引擎搞混了。谷歌有更令人担忧的问题——绝大多数受访者从未听说过他们的 Chrome 浏览器。这是一个直接影响 web 开发者的问题。我们希望人们使用最新的浏览器，这样我们就可以实现新的技术和技巧，而不用花时间在旧的应用程序上编码解决问题。IE6 的开发尤其成问题，但它在首次发布 8 年后仍保留了 20%的市场份额。当绝大多数人不知道我们在说什么时，我们如何鼓励人们升级？这些用户不可能理解新浏览器的好处，因为他们对现有的软件很满意。谷歌还有更深层次的问题。该公司通过搜索和在线应用赚钱。随着这些系统变得越来越复杂，它们变得依赖于 HTML5 等新技术。微软还没有对 HTML5 做出坚定的承诺，即使它实现了，他们也不太可能为他们的传统浏览器提供支持。然而，65%的人继续使用 Internet Explorer 他们无法运行 Wave 等谷歌应用。谷歌正从多个方面着手解决这个问题。它们是:

*   [在 YouTube 等网站上放弃对 IE6 的支持](https://www.sitepoint.com/youtube-drop-ie6/)
*   [创建 IE 插件，为旧浏览器添加新技术](https://www.sitepoint.com/google-chrome-frame-fixes-ie/)，以及
*   [推出免费操作系统，提供现代浏览器](https://www.sitepoint.com/google-chrome-operating-system/)。

然而，用户教育仍然是最大的障碍。谷歌发现，用户花在网上的时间比开车的时间还多，但他们更可能知道自己汽车的品牌和型号。很少用户意识到他们有选择，微软从冷漠和昏睡中获益。因此，谷歌的最新活动题为“什么浏览器”；他们发布了一个视频和一个迷你网站，向互联网大众解释浏览器是什么:
