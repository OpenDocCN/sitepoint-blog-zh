# 为什么页面权重会破坏你的搜索引擎优化工作

> 原文：<https://www.sitepoint.com/page-weight-google-seo/>

我最近写了一篇文章[“为什么页面权重仍然重要”](https://www.sitepoint.com/minimizing-page-weight-matters/)。我担心的是宽带革命让一些开发者忘记了效率和文件优化。越来越多的网站有接近几兆字节大小的网页，然而一些新兴市场只支持低带宽连接。

你解决了你网站的页面权重问题了吗？如果你没有注意到我的警告，也许你会注意到一个令人震惊的消息 [Jen 最近掉了](https://www.sitepoint.com/google-rankings-now-affected-by-site-speed/) : **页面权重现在影响你在谷歌搜索引擎中的位置！**

谷歌现在考虑你的网站加载时间。越胖的页面导致下载越慢——网站速度是影响你页面排名的一个因素。据[谷歌博客](http://googlewebmastercentral.blogspot.com/2010/04/using-site-speed-in-web-search-ranking.html)报道:

> 加快网站速度很重要——不仅对网站所有者，对所有互联网用户都是如此。更快的网站创造快乐的用户，我们已经在内部研究中看到，当一个网站响应缓慢时，访问者在那里花费的时间更少。但是更快的网站不仅仅改善用户体验；最近的数据显示，提高网站速度也降低了运营成本。和我们一样，我们的用户非常看重速度——这就是为什么我们决定在搜索排名中考虑网站速度。

深呼吸，不要慌张。谷歌的网站速度算法已经运行了几个星期。如果你没有注意到你的搜索位置的变化，你的网站可能是好的。暂时如此。

网站速度远没有其他因素(如相关性、话题性和声誉)重要。谷歌估计不到 1%的查询会因为新算法而改变。然而，一个简单的事实是:如果你的竞争对手的网页比你的更精简，他们可能会获得更高的位置。

这可能是一个重新考虑你老板的请求的好时机，他要求增加 15 个广告或 100KB 的 JavaScript 库来处理翻转效果。如果你的网站已经成为一个庞然大物，也许你应该考虑对你的业务的潜在危害。购买更快的服务器或增加带宽只是推迟这个问题——你的页面需要进行卡路里控制饮食。
