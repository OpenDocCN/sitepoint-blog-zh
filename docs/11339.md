# 停止使用互联网浏览器警告德国政府

> 原文：<https://www.sitepoint.com/german-government-stop-using-ie/>

德国政府的联邦信息安全办公室已经正式建议 ( [英文翻译](http://66.196.80.202/babelfish/translate_url_content?.intl=us&lp=de_en&trurl=https%3A%2F%2Fwww.bsi.bund.de%2Fcln_183%2FContentBSI%2Fpresse%2FPressemitteilungen%2FSicherheitsluecke_IE_150110.html))网络用户寻找微软 ie 浏览器的替代品。这份声明发布之前，有消息称 IE 是最近中国攻击谷歌 GMail 和其他系统的主要原因之一。

微软发布了关于问题的[信息，该问题影响所有版本 Windows 上的 IE6、IE7 和 IE8(只有 Windows 2000 上的 IE5.01 不受影响):](http://www.microsoft.com/technet/security/advisory/979352.mspx)

> 该漏洞作为 Internet Explorer 中的无效指针引用存在。在某些情况下，删除对象后，可能会访问无效指针。在巧尽心思构建的攻击中，在试图访问被释放的对象时，可能会导致 Internet Explorer 允许远程执行代码。

软件安全公司 Sophos 将[的威胁等级评定为“高”](http://www.sophos.com/support/knowledgebase/article/67851.html)，并警告说，利用 IE 安全的方法已经在互联网上公布。该缺陷允许黑客安装木马，控制受感染的电脑，并可能窃取用户的密码。

微软德国发言人 Thomas Baumgaertner 说，虽然他们知道政府的警告，但他们不同意:

> 这些不是针对普通用户或消费者的攻击。对普通用户没有威胁，因此我们不支持这一警告。

该公司建议用户安装最新的浏览器，并使用高安全性区域。然而，德国声明的结论是:

> 在保护模式下运行 Internet Explorer 并禁用脚本会使攻击变得更加困难，但无法完全阻止。因此，BSI 建议在微软发布补丁之前使用替代浏览器。

微软正在解决这个缺陷，可能会在 2 月 9 日的下一次更新前发布一个补丁。

如果你正在 SitePoint， [moviedl](http://moviedl.net/ "movie downloads") 上阅读这篇文章，你可能没有使用 IE 浏览器:超过 75%的访问者使用其他浏览器。虽然没有浏览器是 100%安全的，但这已经成为一个高调的故事，可能会影响 IE 的市场份额。谷歌的 Chrome 浏览器肯定会受益……也许这是谷歌如此急于公布 [GMail 攻击](https://www.sitepoint.com/google-threatens-to-quit-china/)的另一个原因？
