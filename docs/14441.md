# PHP 工作趋势(？)

> 原文:[https://www.sitepoint.com/php-job-trends/](https://www.sitepoint.com/php-job-trends/)

[警告/免责声明]可疑的统计“分析”在前面(不包括盐)…

不得不承认，我是 Jobserve.com 的忠实观众，这是一个主要集中在英国的求职列表，但也有越来越多来自欧洲、美国和澳大利亚的信息。

Jobserve 历史悠久(如果我没记错的话，可以追溯到 1994 年)，最初只专注于 IT 工作。从一开始，谷歌就有同样的原则——保持简单，谢天谢地，没有失去它。那些在上面做广告的往往是“猎头”/机构，这意味着他们通常会被调到正确的关键词(即使他们不知道他们是什么意思)，这有助于保持搜索的重点，同时反映当前的工资水平。上市往往会在一周内完成，因此这是 it 就业市场(至少在英国)在任何特定时间的一个很好的“快照”。

无论如何——顺便说一句，Jobserve 除了好消息什么也没说…

回到 2002 年 11 月，[记录了](http://www.phppatterns.com/index.php/intro)搜索“PHP”返回的结果数为 79。两年后，结果一直是两倍——现在 PHP 有 155 个匹配。在过去的几年里，web 开发人员的情况并不乐观…

另一个有趣的现象是“纯 PHP”工作越来越多。早在 2002 年，你主要会看到像“用 PHP 开发 Flash”这样的设计师工作，或者类似“Java + PHP 不错”或“Perl 和一点 PHP”这样的技术工作。2004 年的情况是，更多的雇主需要专门的 PHP 开发人员(当然还有 web 标准和一些数据库)，而且更有趣的是，他们似乎确切地了解 PHP 是什么(与两年前的“优秀的 Javascript + PHP”相比)。以[这个](http://www.jobserve.com/it/jobserve/JobDetail.asp?jobid=20819AB2F055EDCE)清单为例(注意:链接可能过几天就不行了)；

> 具有丰富的面向对象经验的 PHP 开发人员(PHP V5 或 PHP V4)。此外，你最好对 DHTML、XHTML、CSS 和 MYSQL 有很好的理解。这是一个理想的机会加入一个领先的互动电视公司提供良好的职业前景。

PHP 第 5 版还没有出现在 IT 目录周刊的封面上，但是这里有一家公司在找它。

在过去的几年里，我观察到的第三个“趋势”是 PHP 的薪水在上升。在英国，一份“体面的”或“平均的”IT 年薪大约是 3 万英镑(大约 5.5 万美元/4.5 万欧元)。几年前，PHP 的工资(通常是网页设计师类型的工作)徘徊在 20000 英镑左右——超过 25000 英镑是罕见的，而 15000 英镑在“将为食物而破解 PHP”的范围内并不罕见。

目前，PHP 的平均工资似乎在 30k 左右(与其他 IT 工作标准化)，25k 似乎是最低的，像[这个](http://www.jobserve.com/it/jobserve/JobDetail.asp?jobid=9155790A1D25C0CC)支付 PHP 开发人员 40k-47k pa(73-85k 美元)…这基本上是一个纯粹的 PHP 工作。

其他有趣的搜索包括“php oracle”，这表明至少有几个财大气粗的组织雇佣了 php 开发人员和“PHP soap”——他们很惊讶地发现了任何东西。

另外，Python 注意到:我上次(大约一年前)看到匹配的工作不到 5 个——今天我们达到了 34 个。

无论如何，也许这些都是乐观的“指标”。或许值得开发一些屏幕整形工具来实现自动化？或者也许只是一个指示，表明我现在的想法…)
