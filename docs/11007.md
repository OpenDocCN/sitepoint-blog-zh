# 将免费工作转化为付费项目的三种方法

> 原文：<https://www.sitepoint.com/three-ways-to-turn-free-work-into-paid-projects/>

我们已经讨论了一些你不应该免费工作的主要原因。但是，你能拒绝一个免费工作的请求，从而在现在或将来的某个时候产生有偿工作吗？

我也这么认为是的，总会有一些非营利组织和社区团体确实没有任何钱来支付他们需要的服务。但毕竟，许多不劳而获的客户都有资金。唯一的问题是，他们无法证明在这个项目上花费这些资金是合理的…现在还不行！我们的工作是扭转这种看法:让他们相信，这些资金会非常好地用在你的高质量的专家服务上。

对无偿工作说不不应该意味着过河拆桥或者让联系人不高兴。相反，应该尽可能把它看作是一个获得付费客户的机会。这里有三种方法可以让你拒绝没有报酬的项目……也许还能从他们那里得到一些有报酬的工作。

## 1.解释为什么你不能做没有报酬的工作。

上一次，我们认为你不能免费工作的一个重要原因是，你不可能把它放在比你的其他付费客户更重要的位置上，甚至和他们放在一起。如果你的客户希望得到和你其他快乐、满意的客户一样的待遇，他们需要付钱给你。

这将确保他们可以对你的时间提出必要的要求，获得全面的红地毯待遇，并满足他们的项目交付和质量目标。这将保证他们得到你最好的脑力，而不是你的业余时间。这将确保他们的项目得到应有的专家待遇。这些要点为向专家付费提供了令人信服的证据，而不是在业余时间四处找人来完成一个项目。

## 2.提供一个重要的组成部分。

好吧，所以你不愿意承诺免费完成几个小时的工作。但是你能不能还价做一小部分工程付款？在采取这种方法时，寻找机会以一种能让你的客户以他们能承受的价格获得最大价值的方式来运用你的技能。

举例来说，虽然你的客户可能无法支付整个网站的开发费用，但他们可能愿意让你承担这个过程中的一小部分，比如创建线框，或者与用户一起测试设计。横向地、创造性地思考关键的、短期的任务，这些任务将提升项目成果的整体质量*和*为你赢得一些现金。

## 3.正式提议做这项工作。

在网络工作的许多领域，客户不知道服务要花多少钱。要求免费赠品的客户可能会简单地认为他们买不起他们需要的东西，并且害怕去了解实际价格。或者，他们可能不知道他们需要什么，认为工作很短，很容易完成，而事实上它很复杂，很详细。

你需要根据你对客户的了解来衡量是否值得你花时间来整理一份建议书，甚至是一份概述他们计划的项目中可能涉及的服务的文件。如果你不认为这会导致客户期望他们的项目花费完全相同的价格，你可以提供大概的数字或引用你做过的其他类似项目的价值。

一旦你让客户谈论服务和金钱，你可能会发现他们在购买技能方面更加开放——你已经创造了一个向他们提出建议的机会。

这只是三种方法，你可以用它们来把无薪工作变成有适当报酬的项目。你遇到过哪些将客户心态从“免费”转变为“付费”的技巧(或麻烦)？
