嗯，今天带来了一些关于表达的有趣公告，但那是下周的正式新闻。这是过去一周发生的事情。影响我们大多数人的一条消息是 Windows XP 的 SP3 的延迟和 SP1 的自动更新的取消。由于几乎所有的 ASP.NET 开发者都在 XP 或 Vista 上工作，这可能是值得关注的事情。虽然已经发布了一些细节，但看起来服务包带来了一些兼容性问题，可能还有一些安全问题。

民主党选择使用 SilverLight 为民主党全国代表大会提供高清视频内容。除了提供网络内容，微软还将提供软件来帮助大会的其他方面，如投票和跟踪代表，顺利进行。谁知道呢，他们甚至可以让几个小时的讨论和演讲变得稍微有趣一点。

虽然这可能算不上什么大新闻，但[。NET 框架团队正在寻找一些优秀的开发人员](http://blogs.msdn.com/brada/archive/2008/04/30/job-openings-working-on-the-net-framework-product-team.aspx)。所以，如果你对发展充满热情，并且对加入感兴趣，看看布拉德·亚伯拉罕的帖子，里面有关于职位空缺的详细信息。

虽然我只有两篇关于教程和代码示例的文章，其中一篇是 Scott Gu 的链接文章，这应该算额外的。这一轮[链接](http://weblogs.asp.net/scottgu/archive/2008/04/28/april-28th-links-asp-net-asp-net-ajax-asp-net-mvc-silverlight.aspx)包括许多关于 ASP.NET AJAX 和 MVC 的优秀资源，以及一首关于 SilverLight 的特别令人讨厌的歌曲。我还找到了一个整洁的教程，教你如何为 TabContainer 做一些[的惰性加载技巧，允许添加一些不错的 ajax 特性。](http://www.developer.com/net/asp/article.php/3738616)

最后，对于 Xbox 360 机主来说，GTA IV 在本周发布。如果你想杀死一些随机的视频游戏角色，同时也杀死你的生产力，这个游戏是一个伟大的方式做到这一点。只要你在游戏网吧登记，你甚至可以在工作时玩游戏而不受惩罚。
