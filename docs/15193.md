# 个人信息管理综述

> 原文:[https://www.sitepoint.com/information-manager-roundup/](https://www.sitepoint.com/information-manager-roundup/)

面对来自新闻网站、时事通讯和留言板的源源不断的新信息，一个脑容量有限的人应该如何应对？

我们中的大多数人都有自己的方式来组织我们以后可能想要参考的信息，但是这些自制的系统很少是完美的。如果没有合适的工具，在需要的时候找到这些信息会更加困难。

今天，我将向您介绍几个管理信息的精彩程序，包括我正在使用的一个。

**1)** [笔记中心](http://ddelphi.hypermart.net/ )是一个共享软件信息管理器，我已经用了两年多了。您创建了一个简单的信息树，它可以包含多个子页面。树上的每个项目实际上就是一张巨大的纸，你可以在上面复制和粘贴相关信息。层次结构中的单个项目可以快速轻松地移动或重命名。

例如，您可以为每个 Web 设计客户创建一个顶级类别，然后为与该客户网站的设计、数据库后端和新闻稿发布系统相关的信息创建多个子页面。

NoteCenter 可以作为一个图标放在 Windows 系统托盘中，方便快捷地访问。

**2)** [eNotes](http://www.my-enotes.com/) 是 Internet Explorer 5+的一个工具栏。当你在网上看到有用的信息时，你可以选择页面的一部分(甚至是整个页面)，然后点击“做笔记”。这些信息随后会存储在您的硬盘上，包括指向源页面的链接以及您复制它的日期和时间。你可以通过点击“编辑注释”为任何保存的文本添加自己的注释，甚至以 Word 文档的形式传递保存的注释。

**3)** [Treepad](http://www.treepad.com/) 类似于 NoteCenter，只是它具有更高级的功能，例如保存图像的能力。Treepad 还包括一个高级搜索引擎，这是 NoteCenter 所缺乏的。TreePad 的免费版本也是可用的。

**4)** [MyInfo](http://www.milenix.com) 是 NoteCenter 概念的另一种变体。

MyInfo 是一个程序，它允许你以层次结构或大纲组织信息。例如，您可以在导航树的网站>营销部分复制并粘贴与营销相关的文本片段。

一个干净，易于使用的界面，一个初学者的教程，和一个全面的帮助系统，使使用 MyInfo 简单。功能包括模板、导出数据的能力、查找功能和多种排列页面的方式。

T2 是另一个了不起的信息管理者。与这里的一些其他程序不同，TexNotes 允许你用不同的大小和颜色来格式化你的文本，以突出重要的信息。您可以将您的 TexNotes 导出为 RTF、HTML(带或不带样式表)或纯文本。如果您的 PC 上安装了 Word，您也可以将 Microsoft Word 文档直接导入到 TexNotes 中。还包括带有提醒和闹钟的全功能日历。

快乐组织！
