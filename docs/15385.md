# 如何避免业余设计失误

> 原文:[https://www.sitepoint.com/amateur-design-mistakes/](https://www.sitepoint.com/amateur-design-mistakes/)

 ****##### 开球…** 

你会很高兴地听到，关于创建一个看起来专业的网站，其实没有那么多东西要学！

你不需要昂贵的 html 编辑器——只需要一个文本编辑器和一个还算不错的图片编辑器。
![perfect](../Images/79139105ef3080614491a4ca519a56ae.png)
在本教程中，我们将带你了解最糟糕的个人页面和我们渴望的专业外观之间的基本区别。我们将带你从左边的图像，到右边的一个！

![taste](../Images/a814095cc47cc914cf00a7a120b525de.png)

##### 空间

这确实是最重要的事情之一。音乐家经常说“重要的不是你演奏了什么，而是你遗漏了什么”。他们所说的是，你在演奏中留下的空间给了音乐呼吸的空间，让它能够律动起来。同样的原则也适用于网页设计(以及一般的设计)。

你的目标是把注意力吸引到重要的部分——信息、导航系统或任何页面的主旨。

人们经常犯把这些事情搞得很大的错误。他们不需要这样。眼睛需要被吸引到他们身上，而这最好(也是最专业的)通过空间的使用来实现。不要在主页的主要区域放置不必要的文字和图片，因为它们会分散重要部分的注意力。

不要错误地认为，如果你的页面上有一个很大的空白，你应该用一些东西来填充它。只要有必要的部分。如果其他东西真的那么重要的话，它们应该有自己的页面！

作为一个规则，也许只有 75%的页面应该覆盖图像和文本。

##### 一致性

你应该致力于保持一个网站的外观始终一致。

这意味着字体、颜色、图像风格、布局风格、导航风格等的标准使用。不要随便从别的网站上下载一个动画 gif，因为它可能会与你的发生严重冲突。

*   你的背景颜色应该始终保持不变，你应该注意一些小事情，比如保持 html 中定义的文本字体。不要混合字体太多，抵制诱惑！
*   各部分的标题图像应该具有相同的风格，这反过来应该与标志的主要外观(颜色，字体等)相关。
*   颜色。不要选择不协调的颜色——想想你的客厅用你选择的颜色会是什么样子，然后想想你是否愿意花很多时间在那里！在 body 标签中定义你的链接颜色，以匹配你的配色方案。

##### 背景

一个难看的网站的罪魁祸首之一就是这些彩色的背景图片。他们也倾向于有一个明确的模式，这使得文本很难阅读。

试着选择一个平坦的彩色背景，或者一个大的褪色图像(最小 800×600 像素),这样你可以很容易地阅读上面的文字。(不过要注意这些文件的大小！)大 fadey 文字看起来总是很酷，被淡出的黑白照片也是如此。另一个技巧是模糊这些背景，因为这使得顶部的清晰文本更容易阅读——而且看起来也很酷！

##### 滚动文本

如果你有一页需要滚动的文本，试着把它分成不同的页面，并为它添加一个导航系统。它只是稍微好一点，让用户不会在页面上迷路。不过要避开那些死角！

##### 形象

你经常看到网站上有参差不齐的图片。不需要这样，也不需要真正昂贵的软件来绕过它。

我们使用 Adobe Photoshop，这有点像行业标准，但现在有了优秀的 PC 版 Paint Shop Pro 5，这是 shareware。这个程序很棒，导入分层的 Photoshop 文件相当愉快。

gif 文件(。gif 主要的网络图像格式)是索引颜色，但是你应该总是以 RGB 模式创建图像(不要担心这些是什么意思)。这避免了我们都讨厌的“锯齿”现象，最终你会得到漂亮的、无锯齿的 gif 输出。如果你只有一个 GIF 开始，但需要调整大小和播放它，那么先将其更改为 RGB 模式。

JPEG 格式(。jpg)最常用于照片，因为这是它擅长的地方。这是一个 RGB 格式，所以不用担心。注意文件大小，但不要让它们在导出时看起来很糟糕，否则就没有意义了——找到一个折中的办法。你真的不应该有一个大于 30k 的图像文件——这是为了一个真正大的图片。图像宽度最多保持在 400 像素左右。

##### 布局

布局的诀窍是使用表格来提供焦点。它们还会根据浏览器的大小调整大小。考虑一下杂志的布局——它倾向于使用带有背景色的栏目和区域来突出重点。这同样适用于这里。你可以从正文中抽出引文，使它们成为文章的焦点。他们还可以很好地分解一大块文本，这总是一件好事！“从文本中提取引用”

##### 最后一句话

这些是避免网站设计陷阱的基础。还有很多，但是遵循这些指导方针，你将会成功。

环顾网络，你会发现很多好网站和坏网站的例子。当你看它们的时候，请记住你刚刚读到的内容，并试着看看为什么那些不好的书如此糟糕！
