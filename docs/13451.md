# 你最近为我做了什么？

> 原文:[https://www.sitepoint.com/what-have-you-done-for-me-lately/](https://www.sitepoint.com/what-have-you-done-for-me-lately/)

那么…你最近为我做了什么？我让你的生活更轻松。我允许你创建伟大的 web 应用程序，给你的同事留下深刻印象，让你的客户惊叹不已，而且我确信因为我，你每周至少有 3 到 4 次成为你老板的英雄。你最近为我做了什么？在最近的一次打架中，你有没有为我出头？你为我争取过进步吗？你真的尽全力提升我了吗？

如果你到现在还没想明白，我不是在说我，埃里克·琼斯，你的匿名博客。我说的是 ColdFusion，CFML 语和我们所知道的 CF 世界。很多次我听说一个新人对 ColdFusion 不感兴趣，因为他们认为 cold fusion 没有得到一个大社区的支持，或者他们没有听到关于它的足够多的“嗡嗡声”，或者他们认为“思想份额”太低。你猜怎么着当你看到他们运行的圈子时，他们可能是对的，但也许他们是对的，因为我们作为 CF 开发者允许它这样。

我一直很惊讶，为什么知道 ColdFusion 的人不宣传它，不推广它，也不在屋顶上大喊他们有多喜欢它。我多次被称为狂热分子，因为我对 ColdFusion 的热情支持，或者我与其他 web 程序员无休止的针锋相对。这肯定不是什么新东西，而且它是完全可逆的，有时几乎不需要任何努力。

那么你能做什么呢？首先要活跃起来。有一个庞大的活跃而活跃的 ColdFusion 程序员社区，还有更多人只是潜伏在暗处。当然你可以在 [SitePoint 论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=214)上闲逛，但是你是否也知道这里有给 [CF 开发者](http://houseoffusion.com/groups/cf-talk/)(甚至是那些[对语言不熟悉的人](http://houseoffusion.com/groups/cf-newbie/))和[找工作](http://houseoffusion.com/groups/cf-jobs/)的邮件列表？你知道吗？有一个专门介绍 CF jobs 的网站，Adobe 也有一个很棒的 CF 社区论坛。我们也不要忘记遍布世界各地的众多 ColdFusion / Macromedia(现为 Adobe)用户组，如果您身边没有，Adobe 将为您提供工具[来创建您自己的用户组](http://www.adobe.com/cfusion/usergroups/start_a_usergroup.cfm)！现在，我们甚至在 RiaForge 和 CFOpen 都有很好的开源 ColdFusion 项目资源，同时在远程合成 T21 也有很好的项目列表，你可以使用并参与其中。当然，我们不能忘记博客，哦，博客！

我个人认为我为 ColdFusion 做了很多。

*   我全职用语言工作，T2 在工作中写博客
*   我正在帮助启动一个开源 ColdFusion 项目
*   我是当地 ColdFusion 用户组的活跃成员，在那里我展示并颁发奖品(是的，这使我成为最受欢迎的人！)
*   我向各行各业教授 ColdFusion
*   你可能在 SitePoint 论坛上见过我，甚至可能在 CF-Talk 上见过我

但这只是我扮演的一个小角色，还有其他人比我做得更多。我今天想说的重点是，是时候让我们不再仅仅是开发者，向所有人展示我们是一个怎样的社区了。

所以告诉我你是做什么的，或者更好的是你打算做什么！
