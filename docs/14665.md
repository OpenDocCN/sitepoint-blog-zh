# MySQL 进化成大铁

> 原文:[https://www.sitepoint.com/mysql-evolves-into-big-iron/](https://www.sitepoint.com/mysql-evolves-into-big-iron/)

通常，web 开发人员(以及其他许多人)使用的高级数据库提供商因其巨大的容量和规模而被称为“大铁”。这些将是 IBM DB2、Oracle 和微软 SQL Server(可能还有 Sybase)。

MySQL 现在也加入了他们的行列，今天在佛罗里达州的用户大会上宣布了第一个完全开源的集群解决方案，被恰当地称为 MySQL Cluster。该公司表示，这种新的解决方案将为那些需要冗余的关键应用程序(如电子商务解决方案等)提供 99.9999%的可用性。

该集群解决方案的独特之处在于双重许可。各种规模的 Web 开发人员都可以在 GPL 开源许可下免费使用 MySQL Cluster，而那些需要商业支持的人可以选择购买许可。这打开了开发环境，现在可以测试集群，而不需要获得大型数据库许可(甚至一些提供商的开发许可也要 400 到 1000 美元)。

保持低成本的关键是硬件要求。MySQL 利用了在标准硬件上运行 Linux 的能力。这将进一步吸引各种规模的 web 开发人员。因此，与某些集群解决方案对大规模服务器和光纤通道磁盘阵列或存储区域网络的要求相比，使用具有适当 RPM 硬盘驱动器和健康 RAM 的典型 Linux 服务器就可以了。

MySQL 网站提供了预览版。
