# 你的钱还是你的命？

> 原文:[https://www.sitepoint.com/your-money-or-your-life/](https://www.sitepoint.com/your-money-or-your-life/)

上一篇博客谈到了如何过上好日子。这个博客进入你与工作和金钱的关系，推荐两本书。

第一本是乔·多明格斯和维基·罗宾的《你的钱或你的生活》。这是一部经典之作，非常适合那些似乎永远不会满足于自己所拥有的东西，但仍在为更多相同的东西而努力工作的人。当你达到足够的目标时，你要问自己——并制定一个让你真正独立的财务计划。

这本书很棒，因为与现有的关于如何通过增加收入致富的书不同，这本书聚焦于摆脱债务和减少支出。它也向您展示了如此努力工作的真正成本。

现在，如果你热爱你的工作，你可能会发现你的钱和你的生活是一个错误的二分法。尽管如此，这本书还是值得一读，尤其是前几章。

第二，考虑获得金钱和生命的意义。这本书把钱作为一种工具，让你发现自己的核心价值。金钱承载着意义，然而很少有人花时间去体会它对我们意味着什么。通过问一些 Needleman 的问题，可以解放你与金钱的关系。

是时候考虑更大的图景，以及我们如何变得更好、更满意了。想改造世界，先改造自己。思考金钱是一个很好的起点…..
