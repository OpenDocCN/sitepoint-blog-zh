# 孙:开源 Java 不在于是否，而在于如何

> 原文:[https://www . site point . com/sun-open-source-Java-not-when-but-how/](https://www.sitepoint.com/sun-open-source-java-not-whether-but-how/)

Sun 承认:Java 将是开源的。

在 [JavaOne 大会 2006](http://java.sun.com/javaone/sf/) 的[开幕主题演讲](http://java.sun.com/javaone/sf/sessions/general/)中，新上任的 Sun 首席执行官[乔纳森·施瓦兹](http://blogs.sun.com/jonathan)将软件执行副总裁 Rich Green 迎上台，并提出了一个尖锐的问题:“你会开放源代码 Java 吗？”

在一点假装的回避之后，格林解释了孙在这个问题上的最新想法，以底线结束:“这不是一个是否的问题，而是一个如何的问题。所以我们要去做这件事。”

格林确实花时间指出，从[社区参与](http://jcp.org/)和[对 Java 平台发展的贡献](https://www.dev.java.net/)来看，Java 已经*有效地*开源了:

> 这里有两股敌对势力。人们渴望完全开放——完全访问——我的意思是，许可证发生了如此多的变化，以至于它几乎都在那里。但是有一种想要完成这个项目的感觉。另一方面，兼容性真的很重要。我不认为任何人希望看到一个分化的 Java 平台。Java 的伟大价值之一是我们能够将它保持在一起——就其一致性、价值和发展而言，这是大家都可以依赖的。未来的挑战是如何解决这两个问题。
