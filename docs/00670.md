WordPress 插件提供了添加或扩展 WordPress 站点功能的简单方法。市场上有数百个这样的插件。但是你选择哪一个通常取决于你的网站的类型或定位。虽然其中一些对你设计的任何网站都是有帮助的，甚至是必要的。

## 1.[WP 数据表](https://wpdatatables.com/?utm_source=sitepoint.com&utm_medium=content&utm_campaign=wpplugins2019)

[![](../Images/065584f00620148e9dfb061581ed4c4c.png)T2】](https://wpdatatables.com/?utm_source=sitepoint.com&utm_medium=content&utm_campaign=wpplugins2019)

不是每个网站都需要图表和表格来传达信息，但是当你发现自己试图创建一个这样的网站时，你会想要一个工具来使这个过程尽可能简单；一个正确地做它并且在过程中节省你大量时间的方法。

如果 average 足够好，并且您没有时间紧张，那么有大量的图表/表格构建工具可供选择。

但是，如果您想要–

*   专业制作的图表或表格，引人注目
*   从大量(甚至是巨大的)数据中创建的准确和信息丰富的图表或表格
*   响应迅速、易于定制和编辑的图表或表格
*   具有条件格式的图表或表格，允许您突出显示单元格、行或列，
*   并且以最少的努力快速完成所有这些工作，对你来说最好的插件无疑是 wpDataTables。

wpDataTables 为您提供了处理大量数据的一体化解决方案，并且是唯一完全支持 MySQL、MS SQL 和 PostgreSQL 数据库的图表/表格构建解决方案。

[![](../Images/e0b6096a647697bf49180d3ea6c47041.png)T2】](https://wpamelia.com/?utm_source=sitepoint.com&utm_medium=content&utm_campaign=wpplugins2019)

简而言之，Amelia 完全自动化了企业的预订和预约流程。这样做，它节省了企业的所有者和管理者无数的时间，这些时间的特点是人工管理过程所涉及的成本和效率较低。

这个非常有用的插件非常灵活，易于设置，并提供了一个获奖的 UX。它可以轻松集成 PayPal、Stripe、Google Calendar 等关键业务系统，当然还有 WooCommerce。在推出后的头 6 个月里，有 1，000 多名用户注册了 Amelia。

在过去的 12 年里，NextGEN Gallery 一直是行业标准的 WordPress gallery 插件。这个 WordPress 插件目前每年接收超过 150 万次新的下载。这个免费版本很容易用来创建简单的图片库，但工具是为了满足专业摄影师，视觉艺术家和其他成像专业人士的需求。

NextGEN Gallery 使其用户能够高效地管理和显示数以千计的图像，在各种画廊显示类型中进行选择，组织相册，并通过 PayPal 或 Stripe 或个人支票销售照片和数字下载。

NextGEN Gallery 可以收费升级，以包含各种专业扩展，或者您可以选择注册高级 NextGEN Pro 版本。无论你选择哪种方式，你都可以成功地制作出令人惊叹的幻灯片或缩略图、相册等等。
