# 谁在你必须知道的名单上？

> 原文：<https://www.sitepoint.com/who-is-on-your-must-know-list/>

[https://www.sitepoint.com/blog-post-view.php?id = 197828 #评论](https://www.sitepoint.com/blog/)

在上面的博客中，gilbertglee 询问了一个网页设计师为了拥有一个强大的人际网络应该联系哪类人。

这是一个重要的问题。每个网页设计师/开发者都应该有一个不断更新的“必读”列表。这是 10-15 个你想认识的人，以便扩大你的关系网，接触那些认识人的人。

在《引爆点:小事情如何能产生大影响》(马尔科姆·格拉德威尔)一书中，作者介绍了“连接器”的概念联系人是那些认识很多人的人。

你的首要营销任务之一应该是确定你所在地区的联系者，并与他们建立关系。

那么，你应该寻找哪些连接器呢？

嗯，这取决于你的目标市场。如果你关注你所在的地理社区，那么就去寻找社区领袖:银行业高管、当地非营利组织董事会成员、宗教领袖、政治领袖、商会高管，以及你在报纸上反复看到的人。

如果你专注于一个特定的行业，那么就去找认识这个行业的人:协会的高管、专门从事这个行业的行业杂志的记者、向你的行业进行营销的成功的非竞争性销售人员，以及在你的目标市场招聘人员的人。

我们所有人都应该有一个 10-15 人的名单，我们想满足扩大我们的网络。然后制定一个计划来满足他们。寻求建议是一个简单的方法，然后随着时间的推移找到发展关系的方法。不是你名单上的每个人都会合作，但你需要的是一对夫妇。

要问的问题包括:
–鉴于我所做的工作，你会建议哪些人作为好人去了解？
–我如何帮助你实现目标？你寻求什么样的推荐/客户？
–你知道谁在寻找【你提供的好处，例如获得更多业务的方法，在竞争中领先或保持领先的方法】？
–为了在社区中引人注目，需要加入或成为哪些关键组织的一部分？

然后，当你浏览你的清单时，不断更新它。与你见过的人保持联系。不断认识更多的人。
