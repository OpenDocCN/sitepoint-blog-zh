# 消息传递反滥用工作组

> 原文：<https://www.sitepoint.com/messaging-anti-abuse-working-group/>

垃圾邮件和恶意邮件会给你和你的员工、客户和合作伙伴带来多少损失？这是一个大问题。对于许多独自经营企业的人来说，很难评估自己和客户的真实成本。

然而，当您考虑到硬数字时，如带宽、存储、管理垃圾邮件所花费的时间以及由此产生的任何支持问题，费用可能会相当大。这还不包括可能遭受网络钓鱼攻击的客户。

虽然很大一部分反垃圾邮件和其他恶意电子邮件方案可以通过客户端教育和桌面和/或服务器技术解决方案的组合来对抗，但没有广泛的反击标准。

消息反滥用工作组(MAAWG)是一个由深深扎根于网络空间业务的国际公司组成的独立组织，本周在佐治亚州的亚特兰大召开会议，以进一步履行其打击垃圾邮件和其他不受欢迎的消息的使命，并有望开始制定和贡献标准。

参与者包括 Adelphia、新成员 AOL、Bell Canada、Cingular Wireless、法国电信、微软、Sprint、沃达丰和全球众多其他公司。

这个组织的目标[可以在这里找到](http://www.openwave.com/docs/MAAWG_Summit.pdf)，它是为这个组织在今年初夏举行的一次峰会而创建的。

仅从垃圾邮件的角度来看，在 2003 年末，皮尤互联网和美国生活项目认为每天 3000 万封邮件中有 50%是垃圾邮件。这一数字在 2004 年有所上升，其中包括持续激增的[电子邮件病毒](http://www.messagelabs.com/emailthreats/default.asp)攻击和[网络钓鱼请求](http://en.wikipedia.org/wiki/Phishing)。

关于标准和监管要素是否应该来自政府、私营企业或两者，双方都有争论。我的拙见是，给这样的组织一次机会来提出在全球技术流的各个层面都可行的可靠标准，而不是试图让多个外国政府机构同意可能无法长期保持相关性的新法律。

附言:有趣的额外想法——MAAWG 正在考虑让传统电脑用户和移动设备都受益的措施。考虑到手持设备功能的日益增长，我认为这是一种很好的前瞻性思维。
