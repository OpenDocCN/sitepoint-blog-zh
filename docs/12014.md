# 7 个在线日程安排系统帮助你管理时间

> 原文:[https://www.sitepoint.com/online-scheduling-systems/](https://www.sitepoint.com/online-scheduling-systems/)

与客户打交道的一个重要因素是能够在需要时安排电话会议。如果你经常与客户见面，进行销售拜访，并且经常与分包商或团队成员见面，你会发现你每周都要安排几次会议。你可能会花很多时间来管理这些会议——协调会议时间、确认约会、发送拨号信息或其他后勤细节、处理取消和重新安排请求。工作量很大！

除了外包你的日程安排任务，还有一些方法可以让会议过程顺利进行，并减少你在会议过程中的时间。使用在线日程安排和日历系统是一种选择。你可以将这些工具作为内部系统使用，甚至可以将它们集成到你的网站中，这样客户和同事就可以很容易地安排与你的会议(无需你处理细节)。这里有几个你可能想探索的在线日程安排系统。

**1。[约会——加上![appointment-plus](../Images/502481793d9667da234dd226f482a851.png "appointment-plus")](http://www.appointment-plus.com/)** 

*   品牌选项
*   约会和客户报告
*   自动电子邮件提醒
*   PDA 入口
*   计划模板

Appointment-plus 提供额外收费的附加功能，包括销售点模块、时钟、文档上传和 Outlook 导出。

费用:每月 39 美元起。

**2。[约会任务![appointmentquest](../Images/a8f2b351dbd27a77b275f9744bf4a718.png "appointmentquest")](http://www.appointmentquest.com/)** 

*   发送电子邮件确认
*   电子邮件模板定制
*   与 Outlook、Palm Desktop 同步
*   导出到 Excel、CSV 和 PDF
*   可定制的计划和可用性设置

收取额外费用，AppointmentQuest 还通过几家支付网关供应商(包括 PayPal 和 Authorize.net)提供信用卡处理功能，允许您在客户预约时向他们收取服务费。

费用:每月 6.59 美元起。

**3。[Genbook![genbook](../Images/86f71f3b0cdd6fb3795e3d458e0b4312.png "genbook")T3](http://www.genbook.com/)**

*   与 iCalendar、Outlook 和 Google 日历同步
*   使用您的徽标进行定制
*   包括客户数据库
*   与您的网站和电子邮件集成
*   可定制的可用性设置

Genbook 可以集成到您自己的网站上，或者您可以通过实时处理日程安排请求的服务创建一个业务简介页面。

费用:有功能有限的免费版；付费版每月 19.95 美元起。

**4。[宽限排班预约排班系统![graceschedules](../Images/4c15d3d692f044095cd57df15cfc12de.png "graceschedules")](http://www.graceschedules.com/)** 

*   搜索功能
*   发送电子邮件确认
*   可打印的时间表
*   与 Outlook 集成
*   导出到 Excel
*   可自定义的方案和计划设置

这项服务的一个独特之处在于，你可以以固定的价格购买脚本，在你自己的服务器上进行修改和安装。

成本:托管解决方案每月 9.49 美元起。

**5。[调板![shiftboard](../Images/e0fd6250d0649ecf89bbed1a77c3ef01.png "shiftboard")](http://www.shiftboard.com/)** 

*   搜索功能
*   可打印日历
*   导出到 Excel
*   发送自动电子邮件通知
*   数据备份和恢复选项

对于额外的费用，Shiftboard 提供额外的功能，包括文件上传，工作板工具，时间表功能和 SSL 安全。

费用:每月 50 美元起。

**6。[时间中心![timecenter](../Images/31e0fa9567baffba78f877133e8185bb.png "timecenter")](http://www.timecenter.com/)** 

*   包括 128 位 SSL 加密
*   通过电子邮件和短信发送通知
*   群发电子邮件功能
*   包括客户数据库
*   可定制的约会可用性

TimeCenter 还包括一个日志组件，允许您为会议输入注释和评论，以便您可以对每个联系人进行更新。

费用:每月 29 美元起。

**7。[时间驱动![timedriver](../Images/53d794c68f5a9380648933e7b099971b.png "timedriver")](http://timedriver.timetrade.com/)** 

*   与 Outlook 和谷歌日历集成
*   128 位 SSL 加密
*   能够通过电子邮件发送日程安排邀请或纳入网站
*   发送自动电子邮件确认和提醒
*   可定制的可用性规则

TimeDriver 还允许您为特殊事件或电话会议创建自定义电子邮件邀请，并直接发送到您已添加到系统中的联系人。

费用:每年 29.95 美元。

有了所有这些选项，一旦您处理了初始设置，系统就会自动运行。这样你就不用花时间在电子邮件上来回奔波，试图找到共同的空闲时间，或者调整你的约会以适应重新安排的请求。
