# 头脑风暴:理解并采取行动

> 原文：<https://www.sitepoint.com/brainstorm-evaluation/>

本系列的前两篇文章讲述了如何开始头脑风暴，尤其是当你正在努力理清你有组织的思维时，并且包括了几个你可以用来产生新想法的头脑风暴过程。

一旦你完成了头脑风暴会议，你将会有许多想法需要分析、处理并整合到行动计划中。这个评估步骤可以帮助你摆脱那些不起作用的想法，并利用好的结果来制定你的下一步计划。

以下是如何理解你所创造的东西，并把你的想法和主意变成一个行动计划。

## 回到基本规则

在开始头脑风暴之前，你要做的第一步就是列出解决方案的基本规则。这些规则可以是要求你已经拥有的技能，避免不切实际的投资，或者坚持一套道德标准。

不管规则是什么，现在是时候仔细检查你头脑风暴中产生的一系列想法，消除一切与你确定的指导方针相违背的东西了。

## 消除噪音

现在，你仍然会有一个很长的想法、思想和概念的列表，你需要涉水而过。花几分钟时间看看每一个元素，并对其有效性做出决定，自动删除任何不相关或跑题的内容。

如果有些概念不能立即应用于挑战，但有价值，您可能需要创建一个可能考虑事项的次要列表。这个列表可以用来补充你的行动计划，甚至作为后续头脑风暴会议的起点。

## 挑出图案

记下你剩下的想法和概念，寻找一个共同的模式或主题。有没有什么想法产生了你以前没有考虑过的新观点？开始根据它们的共性、发展水平或完整性来组织你的想法。

## 开始计划

一旦你对将头脑风暴付诸实施所需的行动步骤有了清晰的想法，就该计划了。首先确定你的目标，然后将每个概念融入到具体的任务中。像所有的目标规划一样，一定要包括一个在你进行过程中衡量进展的方法。

如果你不能将粗略的想法分解成小组，然后分解成单独的任务，你可以考虑一次或多次头脑风暴会议，使每个想法更加详细。
