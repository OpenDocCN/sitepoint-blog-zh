# 与机构合作–第二部分

> 原文：<https://www.sitepoint.com/working-with-agencies-part-ii/>

在第一部分中，我们介绍了你为什么想为一家代理公司工作，代理公司在寻找什么样的承包商，以及如何超越他们的期望。但是仍然有一个主要问题需要回答——你如何找到寻找承包商的机构？

在我们进入如何寻找代理之前，我们在上一期简要介绍了代理对承包商的要求。重申一下，你要确保你的 LinkedIn 个人资料或简历是最新的，你的作品集展示了你绝对最好的作品，并且你至少有一个网站，上面有基本的个人资料、服务和联系信息(查看 About.me，这是为你的个人资料创建一个单页网站的好方法)。

现在，有两种类型的机构你会想要寻找——需要帮助的机构/现在/和正在寻找优秀人才的机构。当前项目需要承包商的机构可能看起来更有吸引力，但进入几家公司的合格资源名单可以带来稳定的持续工作流。

## 寻找承包商的机构

当像我们这样的机构有项目需要帮助时，他们通常会求助于外包网站，如 oDesk 和 T2 eLance。您可以在这些网站上免费注册并创建承包商档案。

许多机构也在工作板上发布项目，如[真实工作](http://www.authenticjobs.com/)(针对设计师)[堆栈溢出](http://careers.stackoverflow.com/)(程序员和 web 开发人员)和 [LinkedIn](http://www.linkedin.com/jobs) (所有类型的承包商)。这些工作中有许多被列为临时或兼职。我们过去使用过所有这些服务，我经常看到其他机构也向他们发布项目。

## 需要你服务的机构

你也可以锁定那些在你的领域里没有积极寻找承包商的机构。实际上，这是我的偏好，因为它可以让你在资源列表中名列前茅，所以当一个项目出现时，甚至没有必要将它发布到上面列出的网站之一。

你如何找到需要你服务的机构？他们无处不在。你可以在谷歌上简单搜索“互动机构”或“网页设计公司”，你会找到数以千计的公司。我不建议简单地向他们发送垃圾邮件，但是有一些方法可以阻止他们。

在推特上关注他们。比如他们的公司脸书·佩奇。将他们添加到你在 LinkedIn 上的社交网络中。介绍你自己并开始对话。不要发垃圾邮件，做好自己，诚实就好。简短对话后，你可以问是否可以发送你的文件夹。
