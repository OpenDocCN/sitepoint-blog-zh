# 与竞争对手合作:如何帮助您取得成功

> 原文:[https://www.sitepoint.com/collaborating-with-competition/](https://www.sitepoint.com/collaborating-with-competition/)

[![competition](../Images/0073b9c489bc7b8866c3b07c952fa4ee.png "competition")](https://www.sitepoint.com/wp-content/uploads/2009/07/competition.jpg) 在商业中，我们常常以批判的眼光看待竞争对手，很难看到和承认他们做得特别好的事情。这不仅夺走了一个宝贵的学习机会，也让我们失去了当前业务之外的新机会。

竞争当然是健康的，但如果我们愿意超越竞争的本质，与竞争对手合作会有很多收获。毕竟，他们和我们做着相同类型的工作，向相同的受众进行营销，并创建与我们非常相似的业务。这使得竞争对手几乎是任何类型业务的理想合作伙伴。

**协作基础知识**

理想的合作形式是一个具有可衡量的明确目标的特定项目，并且有一些合作的一般原则可以使新的关系有一个良好的开端:

*   确定一个共同的问题、挑战或议题
*   建立关系，花时间去了解对方
*   认真倾听，让对方自由表达自己，不加批评，即使他们的观点与你不同
*   尊重、体贴和友好
*   慢慢开始，让自然的动力推动合作
*   选择一个技能不同但互补的合作伙伴

请记住，与竞争对手合作并不意味着你在追求业务时放弃竞争力。然而，最成功的企业会找到战略上与竞争对手合作的方法，这对双方都有利。

**合作方式**

每个合作机会不一定要集中在一个特定的项目上。还有许多其他方法可以让你与竞争对手保持一致:

*   推荐:如果你收到一个潜在客户的咨询，而这个咨询不符合你的技能，或者你已经忙得不可开交了，那么把这个潜在客户推荐给一个同事，而不要错过这个机会。
*   指导/知识共享:你可以开始自己的指导计划，或者非正式地帮助行业中的新人。
*   合资企业:你最近推出的产品或服务似乎与另一家专业公司的产品或服务相吻合吗？强强联手，实现双重营销、双重覆盖和双重影响。
*   加盟机会:许多成功的企业都有加盟计划来奖励帮助促进他们努力的同事。如果你理解、欣赏并尊重这一事业，加入联盟计划，并创建自己的计划来帮助营销你的产品和服务。

成功参与竞争的关键是保持开放的心态，并愿意学习和分享自己的知识。如果你带着这种心态去合作，你很可能会成功。

你如何看待你的竞争对手？你会利用合作的机会吗？
