# 通过自由职业培养技能的五种方法

> 原文：<https://www.sitepoint.com/five-ways-to-build-your-skills-through-freelancing/>

是什么促使你去寻找新的工作和客户？对我们大多数人来说，主要的动机是支付账单。就目前而言，那很好。但作为一名没有轻松的公司教育或培训津贴的自由职业者，我喜欢把新客户和新项目视为培养技能的机会。这在实践中意味着什么？

**1。我一直在评估我做了什么，以及接下来会把我带到哪里。**

我最近的项目往往是下一个项目的跳板。我会对我刚刚做的事情有一个很好的想法，如果我有更多的时间或不同的简报，我会做些什么。这为我寻找新项目提供了动力和方向。

举个例子，我刚刚为一家当地公司举办了一个教育研讨会。研讨会与会者就其制作过程中直接影响内容质量的方面提出了一些问题。这些问题产生于我渴望发展和磨练我的技能，并获得宝贵的行业经验的领域。所以现在我正在研究策略，通过这些策略我可以把这些技能卖给那个组织。
 **2。在寻找客户的过程中，我寻找新的机会。**

我很容易感到无聊，所以当我找工作时，我会寻找能提供不同环境或框架的机会，在那里我可以发挥我的技能。这些机会让我能够以低风险的方式扩展我的技能、经验和能力，为客户带来良好的结果。

我刚才提到的讲习班就是一个例子。我不是一个有经验的公共演说家，我解释概念的方式可能是漫无边际的。这是我一直渴望解决的问题，所以我抓住了这个机会，向十名参与者组织了一次研讨会。作为一名作家，演示并不是我的主要关注点，但由于我对演示主题了如指掌，这是一个很好的机会来发展技能，并在一系列相关主题上进一步思考和积累经验，同时为我的专业知识支付报酬。

**3。当我投球时，我试着提供服务来培养我的技能。**

向尽可能多的客户推销相同的已建立的、高效的流程，并从您丰富的流程经验中获得回报，这是很有道理的。但是这样做太有规律了，生活会变得相当单调。如果我要学习什么或拓展我的能力，我需要将这些经验融入我所承担的项目中。

最近，我被要求帮助创建一个公司的多个网站重新开发的副本。当然，我可以写稿子。但当我与客户交谈时，我意识到我有机会在公司的各种网站项目中重复应用我为开发内容和沟通策略而创建的流程。这不仅让我能够集中测试和改进我的过程，并查看它在不同的应用程序中如何堆叠，而且我的客户手上有十几个网站，他们迫切需要策略。我向他们推荐了这项服务，以及我渴望磨练的线框和其他信息设计服务。

**4。当我和客户一起工作时，我会寻找接触新挑战的方法。**

我们都知道，为客户工作的经历很少像你想象的那样。他们说他们很忙，然后几周内不要签署任何东西。或者他们告诉你他们有五个迷你项目，然后这个数字一夜之间就扩大了(尽管最后期限没有！).但是那些工作上的变化会带来宝贵的新机会，你会疯了一样错过这些机会。

我工作过的一个客户——一个大型企业组织——需要写一些文稿。然后，他们没有时间去寻找信息来源或获得批准，所以我有机会采访信息来源并管理批准过程。接下来，董事会提前了项目的截止日期，我的客户需要所有人都参与进来，将内容输入到一个我以前从未使用过的专有且流行的 CMS 中。这个项目发展的每一步都给了我新的机会来磨练现有的技能和学习全新的技能。

**5。如果我没有经验，想获得一些，我会很乐意做一个免费的。**

我们每个人无偿工作的数量总是有限的。但是，如果我被邀请为一个项目贡献可管理的金额，这个项目将为我提供特定领域的基础——一个让我感兴趣或兴奋的项目，并且将提供我可以适应并出售给未来客户的技能——我通常会接受。为什么不呢？通常项目是有趣的和非商业的，这意味着有比我在传统的客户环境中尝试更多的空间。

通过免费赠品，我在内容采购、信息设计、项目管理和谈判方面获得了宝贵的经验；我为从游戏到日历到贺卡的所有东西写过文案；我从一些优秀的同事那里学到了很多。通过免费赠品，我发布了我的第一个广告、文章、评论和创意文案。通过免费赠品，我获得了一些伟大的专业对开作品和出色的曝光率，以及有趣的经历和宝贵的，可销售的技能。

你通过自由职业来发展自己的专业技能，使用了哪些技巧？
