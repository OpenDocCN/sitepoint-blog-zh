# 请他人评论

> 原文:[https://www.sitepoint.com/ask-others-for-reviews/](https://www.sitepoint.com/ask-others-for-reviews/)

上周，我在一个我从未去过的城市，决定来一次徒步旅行。这是很棒的几个小时，我学到了很多关于我在哪里的知识——完全值得我花的那点小钱。

基于 Trip Advisor 和 Google 上的好评，我选择了这次旅行。他们比其他旅游公司有更多的评论，基本上是两个人的操作。

在旅行结束时，我发现他们是如何在网上取得如此成功的。导游分发了名片，上面有 Trip Advisor、Google 和其他一些网站的地址。

导游要求所有参与者写一篇评论，如果他们喜欢这次旅行，如果他们不喜欢，请告诉他们他们的反馈。

多么聪明的营销举措——基于这一举措，他们最终让我写了一篇评论(这是我已经在考虑的事情),还有一些其他客人的承诺。

对公司来说，一些名片的费用很小，但对他们来说却很值得，因为他们所在城市的许多其他旅游公司不会提出同样的要求。

他们有一个非常好的网站，这对我来说很重要，但正是这个行业的评论可以成就或摧毁一个企业。你可能不会遇到同样的情况，但是让你的客户发表评论并不会花费你太多的钱。

它最终可能会吸引比你想象的更多的业务。

祝你好运！
