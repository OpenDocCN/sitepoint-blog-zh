# 了解自由职业者市场

> 原文：<https://www.sitepoint.com/understanding-the-freelance-market/>

一些网络自由职业者说话的方式，你会认为工作只是躺在地上等着被捡起来。有些技能现在、过去和将来都非常受欢迎。但其他技能的市场更小或更专业。

在这些较小的市场找工作可能是一个重大挑战。

我们大多数人在找工作时都采取一种自我第一的方式:

*   我们自己
*   我们提供什么
*   我们在哪里
*   我们身边有谁可以卖给我们

但还有另一种方法:利用你在市场中的经验，有意识地拼凑出你正在交易的空间的画面。这张照片可以告诉你找工作的方式，你提供什么，你如何收费。

## 老式市场调查的问题在于

自由职业者总是被鼓励在开始自由职业之前做市场调查。这是一个很好的想法，也是可敬的，但它有相当大的局限性。

我不知道你怎么想，但我总觉得这很像灵魂出窍。想象一个你不在的市场，并从品牌和合同、项目和资金以及联系等方面对其进行评估，这是对某种东西的抽象，最终是一种体验。

自由职业者需要知道的是“市场心态”，如果你愿意的话——而不是它的重要数据。我们需要进入市场的头脑，而不是描述它从外面看起来是什么样子。

相反，如果我们认为市场调研是一个成长过程，只有当我们在市场中工作时才会发生呢？

## 超越你

作为一名自由职业者工作一段时间后，你会不由自主地对那些想和你合作的企业有所了解。你正在建立的行业联系——无论是面对面的还是虚拟的——也会提供市场信息。

这些信息将不会是关于项目规模，收费率，或竞争对手。更有可能是关于:

*   不同类型或规模的组织的优先级
*   业内人士喜欢的工作方式
*   是什么激励和激励了这些人
*   谁受到尊重，为什么
*   什么影响了现有的工作类型

如果你在你的市场工作，你现在就有这方面的知识。放下你的笔，离开你的电脑，思考片刻。

忘掉个别专家对经济、技术等等的看法吧。

取而代之的是关注你所知道的*——你的经历向你展示了关于市场的什么。*

 *然后，试着将这些知识整合起来，形成一个连贯的市场图景，或者说“形状”。什么是高端，什么是低端。谁在做很酷的事情，谁有可能负责帮助你提高技能的项目。谁来付钱，谁来榨干你。重要的是，谁或什么提供了对不同种类的工作的访问。

一旦你有了这个概述，你就可以把自己带回到等式中来:哪里有机会让你的技能和激情适合这个图景？如果你想进入市场的某一部分，你必须做些什么:建立新的联系？学点新东西？重新调整你的对开本？改变你的价格？

经验市场研究将帮助你回答这些问题，并随着时间的推移更容易开拓你的利基市场。你的市场现在看起来怎么样？
