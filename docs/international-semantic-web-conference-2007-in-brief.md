# 2007 年国际语义网会议简介

> 原文：<https://www.sitepoint.com/international-semantic-web-conference-2007-in-brief/>

2007 年国际语义网大会(ISWC2007)在韩国釜山举行。那里的变化速度惊人，我遇到的人非常热情。我印象深刻。

[![Busan at night](img/64169980be49983fe9697a020dfb91a1.png)](https://www.flickr.com/photos/davidsethp/1974958808/ "Busan at night by davidseth, on Flickr")

在与我交谈的小组中，有越来越多的人希望打破“白塔”语义网络工作的模式，生产真实世界的东西。当前开发者可以使用的东西，并且符合绝大多数 web 开发者的需求。

这个工具包还不太存在，但是链接开放数据的团队正在创造下一代混搭技术，叫做 T2 混搭。这是以一致的方式互连和导航的数据。数据范围从整个维基百科数据集(通过 [DBpedia](https://www.programmableweb.com/api/dbpedia) )到美国人口普查数据和 Musicbrainz 以及将它们连接在一起的路径。

这是重要的第一步，想要了解这个新兴领域的开发者没有可用的数据集是做不到的。不仅有意义，而且是真实世界和真实价值的数据。

另一个看起来很有前景的领域是谷歌搜索与语义网技术的结合。在会议的第一天就演示了这样一个例子。该系统名为 [SearchPoint](https://searchpoint.ijs.si/) ，输入一个搜索后，它会返回你通常在谷歌上看到的结果。最突出的区别是出现了一个术语标签云——由一些神奇的 SemWeb 技术创建。这使您可以看到前 100 个搜索结果中最相关的含义，然后通过将焦点转移到标签云的新部分来微调您的搜索。这一切都是实时发生的，你的搜索结果会根据你对意义的重新定位而立即重新排序。这一切都是一个非常令人印象深刻的设置，有助于自然模糊的术语:捷豹(你是指汽车，动物或美国足球队)或有趣的是，这些的组合。

它也适用于不一定是模糊的术语，但是包含大量的匹配项。它可以让你很快看到搜索背后的主要概念，所有这些概念都分组在一起，最重要的是，你不必一页一页地跳过，并试图自己理解大量的点击。

好东西。
