# 谢谢你禁止我

> 原文:[https://www.sitepoint.com/thanks-for-banning-me/](https://www.sitepoint.com/thanks-for-banning-me/)

***“谢谢封杀我。我对你的论坛没有兴趣，我的目的只是通过签名来推广我的网站，否则会有成千上万像你这样的论坛。你的没有什么特别的。”**T3】*

像这样的电子邮件让我开心不已。不，真的，他们有。在漫长的一天工作和抚养孩子之后，他们给了我不必要的笑声。我收到的电子邮件通常分为三类。首先是新加入者，他们介绍自己，告诉我他们需要我做什么。我喜欢那些邮件。它们是一个机会，让你发现你目前正在做什么，你想要什么，以及我们如何满足你的需求。第二类是那些被困在某件事情上或者发现自己迷失在分论坛中的人发来的邮件。那些很酷。帮助别人看到光明是令人满足的。然后还有这些。那些被我们禁止的人认为他们会通过说一些刻薄的话来“让我回来”。通过咒骂或粗鲁。告诉我除掉他们是一个多么大的错误。真的吗？如果我们希望你在我们身边，我们不会禁止你！

![SitePoint community](../Images/ae2d0bf817bdf2f4cb034ec5893f7a6f.png)

我很轻松地管理着 SitePoint 论坛。有些人会说太轻，但我这样做是因为我记得它是新的感觉。在论坛的世界里有这么多不成文的规则，犯错误和感觉自己像个傻瓜是如此的容易，以至于我想尽可能地让它更容易适应并过得愉快——同时让人们能够找到他们需要的信息和帮助。犯错误的新人和故意打破规则的人之间的界限有时很难分辨。我宁愿稳妥一点，避免给人留下不受欢迎的印象。

但是有一个残酷的事实。我们不希望任何人都报名。

上面那个引用引言的家伙就是一个完美的例子。我们不希望他这种人出现在我们的地方。你的目的是提升自己？在市场上做。我们不是任何人的肥皂箱。我们并不声称自己与其他成千上万的论坛有什么特别或不同。我们所声称的是一个欢迎的地方，如果人们在以某种方式为网络开发时遇到困难，他们可以来这里。这些人不是让你去做市场营销的观众。他们不能忍受垃圾邮件。他们肯定不会为了暴露你的签名而去过滤你发的大量垃圾邮件！

我们在这里是为了让你可以和你的同事交流想法。帮助某人。卡住的时候问几个问题。参加一些有趣的对话。说不定还能交到一两个朋友呢！如果你能做这些事情，同时从你的签名中获得一点兴趣，那么你会更有力量。

所以帮我们大家一个忙。除非你注册我们的论坛，成为社区的一员，否则不要注册。就是不要。
