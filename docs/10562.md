## [实用 CSS:从头到尾构建一个真实世界的布局](https://www.sitepoint.com/premium/library)

这个拥挤的课程将带你了解如何从头开始建立一个专业的布局。从一个基本的模板开始，你将学习创建功能齐全的网页**，你可以从**中赚钱，帮助你将你的 CSS 基础知识转化为职业！

### 课程详情:

*   持续时间:**两周**
*   形式: **8 x 1 小时的课程**，每周五有现场直播，录音问&一节
*   开始:**2010 年 10 月 11 日** 
*   费用:**19.95 美元**

包括布局、表单、数据表和下拉菜单，Russ 将集中于从头到尾构建一个完整的 CSS 布局，使用一个可以应用于任何 CSS 布局的过程。你将学习可以转移到任何页面设置的技能，这样**你马上就能自己创造出令人敬畏的布局**。

### 这道菜是给我的吗？

如果你以前曾经摆弄过网页，那么这个课程就是为你准备的。也许你调整了别人的设计，或者使用了一个现有的模板作为你自己的基础。你可以在实用 CSS 课程大纲中阅读[课程每一课的详细内容](https://www.sitepoint.com/premium/library)。

如果你是一个绝对的 CSS 初学者，那么这个课程不适合你。Russ 的第一个课程是 [CSS Live](https://www.sitepoint.com/premium/library) ，对于那些从未输入过 CSS 代码但渴望学习的人来说，这是一个理想的课程。
