# 走近非营利组织——第二部分:找到工作

> 原文：<https://www.sitepoint.com/not-profits-2-landing-job/>

成功地与非营利组织做生意需要某种方法。上次，我们探讨了非营利组织的“灵魂”，他们是如何思考的，他们是如何设定目标的。现在，让我们来看看自由网站开发者在这些组织中获得成功的一些具体方法。

##### 致胜的一投

我接触过的大多数组织都不关心他们的技术有多先进。“最新的东西”不是他们关心的，他们也没有钱在被“前沿”上扔掉。当然，也有例外。但是一般来说，严格的预算限制不允许在技术领域建立大量的“基础设施”。

大多数公共资助的教育机构提供最先进的计算机实验室供学生使用，但这只是过去 5 年的一个新趋势。在那之前(甚至直到现在，在一些学校)，计算机实验室由破旧的旧拼凑系统组成，这些系统是由捐赠者提供的，几乎无法运行。

因此，重要的是要认识到，除非你能吸引组织的使命，并展示你和你的技术如何能使他们的工作更容易，否则你可能找错了对象。

考虑一下这个场景:一个非盈利的基督教牧师经营着一个培训中心，在那里学生接受“街头牧师”培训。一段时间以来，他们看到捐款减少，而培训中心的出勤率增加。他们的主要办公室使用的是一个基于 DB2 DOS 的旧数据库，灵活性有限，他们意识到他们正在努力让这个古董怪兽存活下来。

他们无力购买现代化的数据库系统来运行日常运营，但他们发现自己越来越依赖于陈旧、不灵活的系统。在一次董事会会议上，决定进一步从办公费用(所有与计算机相关的项目都集中在一起)中转移资金，以适应入学人数的增加。

要求你提一个建议。查看选项后，您推荐使用通过 Web 门户运行的 MySQL 数据库。您建议使用免费软件，这意味着该组织可以将更多资金转移到学校的培训方面，毕竟这是他们的主要关注点。当有了这种选择，董事会就会迫不及待地加入进来。如果您提供了一个商业替代方案，仅仅是因为它允许更多的同时用户，更有能力承载负载，或者具有其他技术优势，而这些优势虽然“很好”，但并没有给组织带来任何直接的好处，他们可能不会如此热衷。

这个故事的寓意是什么？确保你的推销符合客户的需求——那些在技术上显而易见的需求，以及那些虽然看起来不相关，但仍然对项目的最终结果有很大影响的需求。显然，这里的关键是研究——了解客户组织，以及他们工作的限制条件，这样你就有更大的机会成功推销。

##### 网络——您最大的盟友

鼓励你的客户为你工作是任何小企业的必要任务。在非营利行业，这是绝对重要的。非营利领域的组织彼此紧密联系，共享信息和供应商。如果你取悦了客户，他们将是你多年来最大的资产。让他们生气，你很可能会毁掉你在那个地理区域或那个社交圈的其他非营利组织中取得成功的机会。这就是病毒式营销的本质:你此时此地所做的一切，将会对你未来可能拥有的所有其他潜在交易产生连锁反应。

我采用的一个成功技巧是现金返还选项。我的每一个介绍新客户给我的客户(也就是实际签署合同的客户)，都会收到一张销售额 5%的支票。通过这种方式，客户可以赚回钱——这鼓励他们与他人交谈，并证明对他们和我来说都是有利的。

##### 非营利营销的错误

为了拓展业务，我做的第一件事就是列出一份我与之有过任何联系的非营利组织的名单——不管联系有多遥远。接下来，我整理了每个组织内部联系人的电子邮件地址列表。如果我读了乔治娜的[“热门标题如何”](http://www.promotionbase.com/article/809)，也许我的电子邮件活动会取得更大的成功，但可惜的是，我收到了几封回复我信息的仇恨邮件，而且从未获得任何业务。

我也建议不要打陌生电话。这种做法本质上是一种“镖盘法”。这通常需要打开一个本地电话号码簿，寻找随机的非营利组织，并打“礼节性电话”。我发现这是一种偶然的方法，很少成功。再次强调，了解你的目标组织的重要性是关键——这两种方法都不需要你了解客户的任何情况，这也是它们如此不成功的原因。

##### 进入市场

环顾四周:非营利组织无处不在。到处都有教堂、医院和教育设施，这只是在这个行业中可以运作的许多组织类型中的几个例子。互联网上也有很多网站和工具可以帮助你在非营利领域找到潜在客户。以下是一些例子:

*   [美国慈善机构](http://www.charities.org)
*   [志愿者比赛](http://www.volunteermatch.org)
*   [手网](http://www.handsnet.org)

##### 成功的关键

这些只是我个人接触非营利组织的少数方法。沟通至关重要:你必须让他们明白完成工作的目的。如果这个项目不能完成他们的使命，这将是一次非常艰难的推销。

自由职业者也必须在潜在组织的财力范围内工作。这些不是拥有数十亿美元预算的大公司，他们的首席执行官开着捷豹敞篷车。然而，他们在我们的社会中做着极其重要的工作，并为自由职业的 Web 开发人员提供了相当多的机会——对组织的情况感同身受，你可能会对他们的反应感到惊讶——以及他们在行业中的同行。
