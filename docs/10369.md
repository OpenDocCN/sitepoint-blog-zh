# 如何创建一个更好的非博客 WordPress 站点

> 原文：<https://www.sitepoint.com/non-blog-wordpress-site/>

WordPress 已经远远超过了它作为博客平台的预期用途，因为许多网站所有者已经将它用于非博客网站。即使他们不写博客，这么多人转向 WordPress 也不足为奇。对于想在非博客网站上使用 WordPress 的普通网站所有者来说，有很多吸引人的好处，包括:

*   避免定制设计的成本
*   无需任何编码经验即可添加或修改内容和页面
*   轻松合并照片和图形，同时能够修改大小和位置
*   不需要独立的 FTP 程序
*   轻松改变主题
*   使用插件来简化高级功能

不过，这并不总是像安装 WordPress 并把你的网站放到网上那么简单。我已经看到了很多这样的网站，它们看起来已经差不多了，但是缺少一些重大的改变，这些改变可以更有效地将 WordPress 转变成一个组织良好的静态网站的框架。

无论你是在创建一个非博客的 WordPress 网站供自己使用，还是如果一个客户要求帮助建立一个，这里有一些我的建议。

## 选择一个好的主题

有很多优秀的 WordPress 主题，但并不是所有的都适合非博客网站。例如，您可能不希望您的内容按日期组织，或者一次显示多个摘录。所以你必须做一些探索，为你的目的找到一个好的非博客主题。通常类似杂志的主题是一个不错的选择。你可以看看这篇文章中的几个， [5 个最好的非博客主题](http://crunchmeme.com/the-5-best-non-blog-wordpress-themes/)。

## 使用页面，而不是帖子

事实上，当谈到页面和帖子时，你可能会选择任何一种方式，尽管我更喜欢页面。在非博客的 WordPress 网站上，页面和文章基本上做同样的事情，但是如果你最终两者都使用，你可能会造成导航混乱。我的建议是选择一个或另一个(任何一个方向都有好处)，然后在你的所有内容中坚持这样做。

## 设置静态首页

如果你选择了使用页面，你也需要设置你的站点来显示一个静态的首页——你的主页。为此，请创建您想要指定为主页的页面，然后前往“设置”>“阅读”。选择“静态页面”单选按钮，并从下拉菜单中选择您选择的页面。

## 把你的手伸进密码里

即使你选择了完美的非博客主题，你也必须做一些代码调整来删除一些以博客为中心的内容和引用。例如，即使你关闭了评论功能，仍然使用帖子，大多数主题仍然会显示“无评论”行。你可能还想删除 RSS 订阅选项和边栏中一些与非博客网站无关的信息。

## 对插件吹毛求疵

有许多不同的插件可以使 WordPress 从博客到非博客的过渡变得容易。例如， [HeadSpace](http://urbangiraffe.com/plugins/headspace2/) 让你通过一个界面轻松管理元数据和各种 SEO 任务。而 [Flutter](http://wordpress.org/extend/plugins/fresh-page/) 是一个插件，可以创建完全定制的自定义写面板(单选按钮、文件上传、图片上传、复选框等)。
