# 面向设计师的 18 款 Adobe AIR 应用

> 原文:[https://www.sitepoint.com/adobe-air-apps-for-designers/](https://www.sitepoint.com/adobe-air-apps-for-designers/)

Adobe Integrated Runtime (AIR)是一个跨平台程序，允许您运行各种不同的小程序。它因一系列 Twitter 客户端而闻名，最受欢迎的是 TweetDeck，但这个环境不仅仅是一个流行的微博服务平台。

网页设计者可以找到大量方便的程序来减轻他们的工作。你能找到的一些工具包括选择配色方案、测量设计中的空间、编辑矢量图像等等。我们挑选了 18 个最好的来帮助你开始。
