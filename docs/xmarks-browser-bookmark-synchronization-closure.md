# Xmarks 浏览器书签同步即将结束

> 原文：<https://www.sitepoint.com/xmarks-browser-bookmark-synchronization-closure/>

对于在线服务关闭来说，这是一个糟糕的月份。我最喜欢的 RSS 聚合器 Bloglines 将于本周关闭，现在 [Xmarks 已经走到了路的尽头](http://blog.xmarks.com/?p=1886)。

Xmarks 是一个免费的同步服务，可以在 Firefox、Chrome、Internet Explorer 和 Safari 上备份和复制你的书签。该产品于 2006 年 10 月推出，是 Firefox 特有的扩展，名为 Foxmarks。2009 年增加了更多的浏览器，该公司将自己重新命名为 Xmarks。

除了书签，该插件还可以同步密码、浏览器历史和打开标签。早些时候，该公司意识到他们坐在一座拥有 1 亿用户验证页面链接的金矿上。他们试验了自己的搜索引擎，但是系统没有通过可用性测试。虽然它可以提供没有垃圾邮件的分类链接，但它不能回答人们在典型的搜索引擎查询中输入的具体问题。但是，该插件可以将信息附加到 Google 结果中，以指示某个链接的受欢迎程度。

对我来说，Xmarks 仍然是最好最可靠的书签同步服务。如果你在使用多台电脑，它的价值是无法估量的，而且它是我在每个浏览器上安装的唯一 Firefox 插件。它拥有 200 万用户，支持 500 万个桌面，每天接收 3000 个新注册。

尽管它取得了成功，但联合创始人兼首席技术官托德·阿古尔尼克解释说，他们一直在努力赚钱。该公司自 2010 年初以来一直在出售，但没有潜在买家前来。既然微软、Mozilla、谷歌和 Opera 都提供免费的浏览器同步，这种商业模式可能不会有可行的未来。不幸的是，这些选项中很少有像 Xmarks 一样好的，而且没有一个提供跨浏览器的功能。

该公司正在向用户发送电子邮件，Xmarks 服务将于 2010 年底关闭。
