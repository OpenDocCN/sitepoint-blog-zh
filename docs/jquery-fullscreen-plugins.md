# 12 个 jQuery 全屏插件

> 原文：<https://www.sitepoint.com/jquery-fullscreen-plugins/>

让我们面对它，有时全屏网站是超级酷的，如果一个网站看起来平淡乏味，你认为访问者会停留多久？所以我们认为我们可以提供一些帮助，并与你分享这 12 个 jQuery 全屏插件,让你的网站具有响应性的全屏外观。这些 jQuery fullsreen 插件的集合将会给你的网站带来巨大的影响。尽情享受吧！
