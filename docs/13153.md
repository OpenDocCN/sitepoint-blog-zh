# 漂亮的 HTML 电子邮件的原则

> 原文:[https://www.sitepoint.com/principles-beautiful-html-email/](https://www.sitepoint.com/principles-beautiful-html-email/)

作为网页设计师，我们习惯于针对网页浏览器的特定限制和功能进行设计，这里有很多很好的建议可以帮助我们。然而，HTML 电子邮件是一个不同的故事——它们经常是网页设计界的害群之马，要么被忽视，要么被积极排斥。

尽管如此，HTML 电子邮件的使用仍在继续增长，它已被证明是企业与客户沟通的有效工具。近年来的研究表明，平均而言，电子邮件营销每花费 1 美元就能带来高达 50 美元的收入。

让我们在这里明确:我们不是在谈论垃圾邮件！我们谈论的是人们明确要求的电子邮件，如 BBC 新闻标题或您的酒店预订确认。如果你没有给别人发邮件的权限，就不要给他们发邮件，不管你的设计有多好。我们也不是说每封邮件都应该是 HTML。

不管我们喜不喜欢，HTML 电子邮件已经存在，既然有人需要设计它，那么这个人应该是网络专业人士。然而，电子邮件不是网页。当你设计一个网站的时候，你知道访问者可能已经知道会发生什么。他们可能点击了一个搜索结果，或者另一个网站的链接，所以他们有一个上下文来理解你的页面。当它们到达时，你有整个浏览器窗口来显示你的信息。

邮件就不一样了。电子邮件收件箱是一个非常嘈杂的地方，有大量的邮件、文件夹、日历和其他干扰。你的订阅者可能只阅读主题行，或者可能通过一个小小的预览窗格看到你的电子邮件，所以你需要相应地设计它，并充分利用阅读器给你的屏幕空间和时间。

那么，什么是好的现代 HTML 邮件设计呢？

##### HTML 电子邮件的关键要素

有几个关键要素是每封商业邮件都应该使用的；它们将构成您设计的核心:

***权限提醒***

当你打商务电话时，你不会以“今天就购买我们神奇的新部件吧！”—至少，你不应该。相反，你通常从介绍自己开始，并提醒收件人你过去的会议或联系。

你的电子邮件应该以同样的方式开始，清晰、具体地提醒用户是如何进入你的名单的。这为他们提供了您的邮件的一些上下文，并降低了收件人将您的邮件标记为垃圾邮件的可能性。

![A permission reminder this tiny is useless](../Images/001048ba92ef0964fc46b5c6c42ec42c.png)

***格式良好的副本***

在电子邮件中使用 HTML 的一个最大的好处是能够格式化你的文本。您可以使用真正的标题、调整行高、突出显示重要内容，并以纯文本不容易做到的方式使副本可扫描。这确实是你应该花大部分时间设计电子邮件的地方——让邮件更容易阅读，更容易采取行动。

***突出退订链接***

每一封商业邮件都应该为人们提供一个快速而可靠的方法来摆脱这个名单。设计师和客户有时都试图隐藏这个链接，希望把订户留在他们的名单上。但是如果有人想离开，你必须让他们轻松地做到这一点，否则你只会以垃圾邮件投诉而告终。自动退订机制比需要手动处理或填写繁琐表格的机制好得多。

***【纯文本版(及网页版)***

你发送的每封商业邮件都应该有一个纯文本版本，让那些不能或者选择不看 HTML 版本的人也能看到。以多部分文本+html 格式发送电子邮件使这变得简单。此外，如果对您的受众和内容有意义，可以考虑链接到一个网页，让收件人获得完整的 HTML 体验。

***法律要求的详细信息***

一些国家的法律要求强制商业电子邮件中的某些元素。最著名的例子是 CAN-SPAM 法案，该法案要求在每封商业邮件中包含实际的街道地址。

***视觉连接到品牌***

作为一个设计环境，电子邮件不同于网络，但它应该有助于将网站的品牌延伸到电子邮件中，就像网站可以延伸印刷品牌一样。这有助于用户在脑海中将邮件和发送邮件的公司联系起来，同样，这也可以减少垃圾邮件投诉。

一旦你整理好了邮件的核心内容，就该创造性地设计你的内容了。

##### 电子邮件布局和设计

就像创建一个网页一样，设计一封邮件就是要了解你的主题和你的受众。不用回到页面布局和设计基础，电子邮件有几个定义性的特征可以指导你的设计。

***一封邮件不是一个网页***

一些设计者经常会这样说，好像这意味着 HTML 电子邮件不应该存在。好吧，那艘船已经起航了，但是电子邮件和网页是不同的环境，需要区别对待，这是绝对正确的。

***区分内容优先级***

一封电子邮件有很多东西可以竞争注意力。收件箱中有所有其他邮件，有垃圾邮件，经常有个人和工作相关邮件的混合，还有所有正在编写的草稿邮件。并且认为阅读电子邮件只是大多数人工作的一小部分！

你需要快速切入正题。从最有价值的信息开始，不要藏起来。你的读者应该能够看一眼邮件的顶部，立即知道它是否值得一读。考虑从目录或简短摘要开始，吸引他们阅读整封邮件。

![Consider starting with a table of contents](../Images/2ef74f3c8934a52dfa00b231de224a4c.png)

***全文 vs 摘录***

您可能需要做出的一个决定是，在提供完整信息的独立电子邮件和使用链接到网站的“预告”格式之间做出选择。你的选择将取决于你的邮件目标。如果你想增加网站的流量，巧妙的设计可以起到很好的作用。如果你的时事通讯更多的是关于建立你作为信息提供者的声誉，在邮件中包含完整的内容可能更有意义。

***行动号召***

考虑到让你的读者真正打开并阅读电子邮件的工作量，令人惊讶的是许多设计师在他们的活动中没有明确的行动号召。你应该清楚地知道你希望读者对你发送给他们的信息做些什么。

我们可能会联系您，或者访问您的网站以获取更多信息。这可能是为了购买一些东西或回答一个调查。不管是什么，确保你的设计使这个任务简单明了。

在正确的位置放一个大按钮或链接可以避免读者的“死胡同”体验。给他们一个地方去，你会从你的活动中看到更好的结果。

***也设计你的纯文本***

你发送的每一封 HTML 格式的电子邮件都应该附带一个纯文本的替代文本，以方便那些不能或者不愿意查看 HTML 格式的人。你应该在你的纯文本上花一些设计时间，因为你永远不知道哪个客户正在看它。
纯文本邮件的核心规则是:

1.保持你的线长度短。

2.电子邮件软件习惯在不方便的地方打断文本行。例如，你精心编写的句子可能会这样结束:

电子邮件软件有一个令人讨厌的习惯，它会在不方便的地方打断文本行。你精心写的句子可以
这样结束。

解决方法是保持行短，并在每一行的末尾添加硬回车。65 个字符的行长度提供了最大程度的保护，避免了参差不齐的格式问题。当你编辑和格式化一份简讯时，将你的编辑软件设置为 65 个字符的宽度，并自动硬回车。

3.尽可能使用最短的 URL。

除了保持你的行在 65 个字符以内，你还需要保持你的 URL 的简短。一行断掉的文字只是看起来很丑，而一个断掉的网址根本不起作用。一些 CMS 使用很长的参数，所以你可以考虑在你自己的服务器上使用重定向，或者像[http://www.tinyurl.com/](http://www.tinyurl.com/)这样的服务来获得你的 URL 的较短版本。

还要确保使用完整的 URL(包括`http://`部分),并且在 URL 的两边插入一个空格，以帮助电子邮件客户找到它们，并使它们可以点击，就像很多人做的那样。

4.创建标题和章节。

一大块文本很难阅读，尤其是当它是纯文本格式，没有颜色或不同的字体大小来帮助人们浏览时。你可以使用几行重复的字符——例如下划线(`_______`)或星号(`********`)来强调你的标题，并把你的邮件分成几个部分。

您不能完全控制纯文本的排版，但至少可以确保文本结构良好、清晰明了。查看一些[示例纯文本电子邮件](http://www.campaignmonitor.com/resources/plain-text-templates.aspx)以获得灵感。

##### 像 1999 年一样编码

一旦你计划好了你的设计和布局，就到了实际构建 HTML 的时候了，这是我们开始倒退的时候。在你的工具箱中有一些关键的技术约束和技巧，你需要用它们来实现你的奇妙设计。

***邮件客户端中的 CSS 支持***

典型的电子邮件客户端非常类似于 1999 年的 web 浏览器，对现代 HTML 和 CSS 的支持有限或不一致。最好从电子邮件中的 CSS 支持活动监控[指南开始，它为您提供了一个概述和有用的图表，显示了主要电子邮件客户端支持和不支持的内容。你可能会发现 SitePoint 关于](http://www.campaignmonitor.com/css)[编写 HTML 电子邮件](https://www.sitepoint.com/article/code-html-email-newsletters/)的文章也很有用。

这些资源将为你的电子邮件结构打下坚实的基础，你可以在此基础上进行构建。

***预览窗格:邮件预告片***

在您的订户真正看到您的电子邮件之前，他们可能只是在预览窗格中瞥了一眼。通常，这将是一个高度不超过几百像素的窗口，但你的订户可以使用该空间中可见的信息来决定他们是打开你的电子邮件还是忽略它。

优秀的电子邮件会利用这个空间来回答问题:“这是谁发来的？”以及“邮件内容是什么？”这可能意味着有一个一致的标题图像是可识别的，或者有一个标题抓住他们的注意力。特别有用的是在页面上方列出目录的电子邮件。这样，你就不会强迫你的读者在没有任何信息的情况下做出决定。

当然，预览窗格的大小和位置各不相同，所以你所能做的就是按优先顺序排列你的内容，从关键信息开始，就像我们之前讨论的那样。

***图像阻塞***

默认情况下，许多电子邮件客户端不会下载 HTML 电子邮件中引用的图像。相反，它们会显示一个占位符，有时会显示一条消息或信息栏，解释如何打开图像。对于这种情况的概述，请参见马克·怀纳的[电子邮件客户端中的图像屏蔽:当前状况和最佳实践](http://www.campaignmonitor.com/blog/archives/2007/02/current_conditions_and_best_pr_1.html)。

你的电子邮件需要立即可读和有用，即使没有任何图像显示。这意味着在页面上有真实的文本内容，标题和链接可以立即加载。你的文字必须给读者一个令人信服的理由去打开那些图片。全图片邮件会给你的许多读者带来非常糟糕的体验，就像下面的苹果邮件公告。

![Make sure your email is readable with images turned off](../Images/337397be0a2b6034cf1f83d70d496295.png)

你可能还想在你的邮件顶部加上一句话，要求人们将你的“发件人”地址添加到他们的白名单或地址簿中，因为对于许多电子邮件客户端来说，这一步将绕过图像屏蔽，并有助于将你的简讯排除在垃圾文件夹之外。

***不得的事情***

虽然一些电子邮件设计涉及修改元素或解决问题，但网页设计的一些部分最好完全远离电子邮件——如音频文件、Flash 文件和 JavaScript。

出于安全原因和保护电子邮件读者的体验，电子邮件客户端开发者没有花时间支持这些类型的内容，这是正确的。电子邮件收件箱不是放电影或播放歌曲的地方。你的读者不会期待它，可能也不会喜欢它。

比如电子邮件对 [Flash 的支持就很低。不要试图把你网站的每一部分都放进电子邮件里，而是把电子邮件作为你网站的门户。将静态截图链接到网站上的视频，或者使用歌曲图标链接到在线播放器。](http://www.campaignmonitor.com/blog/archives/2006/01/the_truth_about_flash_in_email.html)

克制是非常重要的，因为一封引发安全警告或意外动画的电子邮件会很快被删除。

***测试，*测试**

与 web 浏览器不同，在 web 浏览器中，您可以合理地覆盖自己桌面上的主要浏览器，电子邮件测试可能非常耗时。至少有 12 种电子邮件客户端拥有主要的市场份额，根据你的受众，你可能需要测试所有的客户端。有一些工具可以帮助你完成这个过程——例如，Campaign Monitor 的设计和垃圾邮件测试工具和 [Litmus](http://litmusapp.com/) 。他们可以给你看所有大客户端的邮件截图，为你节省很多时间。

当然，你也可以自己做，通过设置所有不同的客户端，并有一个电子邮件地址的测试列表。记住:在你向你的客户发送广告之前发现问题总是更好的。

##### 一直在变好

在你真正发送邮件之前，还有一个问题你需要回答:什么样的结果会是好的？你可以想花多久就花多久，但是你需要知道你的时间花得是否值得。在你发送之前，确保你已经为你的电子邮件活动定义了一些可衡量的目标。

要衡量和设定目标的一些典型方面包括:

*   一定比例的人会打开你的邮件。请注意，尽管[测量开口并不是一门精确的科学](http://www.campaignmonitor.com/blog/archives/2007/01/all_about_email_open_rates.html)。
*   人们点击你最重要的链接。你希望你的读者采取的最重要的行动是什么？这是要追踪的最重要的方面。
*   电子邮件回复。也许你只是想增加联系你的人数。这是一个简单的衡量标准。
*   销售或查询。如果你有合适的工具，你可以从你的电子邮件中跟踪人们访问你的网站，并通过结帐过程全程监控他们。

这些类型的测量最好通过一系列的活动来完成。首先，你不会有任何基准数字来判断。有很多宽泛的统计数据和指标，但每个受众都是不同的。更重要的是定义你自己的衡量标准，并随着时间的推移跟踪你的活动与它们相比如何。你离目标越来越近了吗？

***如何提高成绩？*T3】**

和网页设计一样，有无数种方法可以调整你的电子邮件设计和复制以获得更好的效果。如果你经常通过电子邮件发送一份清单，你有机会尝试一些小的、可控的改变。你可以用你的整个用户群来测试它们，或者选择一小部分，从你的主列表中给他们发送一个变体。

一次选择一个领域，这样你就可以控制变量的数量，并且更好地了解到底是什么在起作用。一些好的起点是:

**主题行**

在一个繁忙、活跃的电子邮件收件箱中，那简短的主题行可能是你说服某人打开你的邮件并考虑阅读它的唯一机会。试着让你的主题行容易辨认，有价值，确保它们准确地表达了邮件的内容。查看像主题艺术这样的资源来获得更多的想法，并尝试一些不同的主题来看看它们有什么影响。

**发送的时间和日期**

有时候，同样的邮件在周二下午会比周四早上更有效。在这个领域你会发现很多相互矛盾的证据，所以知道什么对你的听众和主题有效的唯一方法就是尝试几次。想想你什么时候花最多的时间阅读电子邮件——是早上的第一件事吗？或者午饭后，在你开始一个新项目之前？

**标题和链接文案**

就像主题行一样，标题是人们用来判断你的邮件是否值得一读的元素。你的文章可能非常有用，但是因为标题没有说服力而被忽略了。布莱恩·克拉克的[如何写出有效的标题](http://www.copyblogger.com/how-to-write-headlines-that-work/)是一个很好的起点。

**内容定位和优先排序**

不要害怕尝试移动你的简讯的不同部分，甚至完全删除表现不佳的内容。观察你的统计数据会让你知道你的受众最感兴趣的是什么，所以用这些知识来组织邮件——就像你组织网站一样。电子邮件设计可以是信息架构的一个缩影，你可以非常灵活地使用它。

##### HTML 邮件可以做对！

这需要一点努力，而且可能会令人沮丧，但是通过一些知识和实践，你可以创建人们会觉得更容易阅读、有吸引力和有用的 HTML 电子邮件。确保你也总是发送纯文本，但不要把 HTML 留给营销人员！
