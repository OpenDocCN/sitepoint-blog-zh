# 撕掉它:使用 CSS 的磨损类型

> 原文：<https://www.sitepoint.com/rip-it-up-worn-type-using-css/>

最近有这么多关于 DOM 脚本的实验，感觉我已经很久没有在普通的 CSS 中看到真正的新东西了。至少在今天之前是这样。

Khmerang.com 发表了一篇很棒的小文章，解释了他们是如何让标题看起来又脏又旧的。不，这不是 sIFR，甚至不是传统的图像替换。事实上，这是对 Tom Gilder 和 Levin Alexander 的“old skool”IR 技术的一种巧妙的变化，这种技术提倡在文本上浮动一个不透明的图像层，而不是更常见的将原始文本直接移出视窗的方法。

Khmerang 的新想法是用一种几乎透明的仿旧图案替换图像层，留下带有划痕和磨损的底层标题文本。

真正好的一点是模式重复，但很少会与同一个字母完全一致，所以每个字符都应该是一次性的。

正如你可能已经想到的，有几个注意事项需要考虑。

1.  文本仍然是可选择的，但是比平常更难。在中选择*航向几乎是不可能的。尽管如此，大多数其他的信息检索方法都没有允许文本选择的前景，所以这种技术已经比那些方法更上一层楼。*
2.  你需要选择一个大的，结实的，有大表面积的字体来获得有价值的效果。从一些快速测试来看，较轻的字体看起来有点脏。
3.  该方法需要一些稍微有点混乱的标记— `<h2><span></span>Worn Text</h2>` —所以如果你是一个坚持语义的人，你最好避免这种方法。

然而，它确实提供了一个非常独特的结果，虽然它不太可能是你在一个又一个项目中使用的那种技巧，但看起来像是军械库中的一个有用的补充。

干得好。
