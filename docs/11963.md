# $#?%*!当你陷入困境时该怎么办

> 原文：<https://www.sitepoint.com/what-to-do-when-you-mess-up/>

错误是不可避免的。无论你做什么，无论你做得多好，你最终都会搞砸。你会从中恢复过来，把它抛在脑后，但它会再次发生。可能不会很长时间，但在你的职业生涯中，你的记录上至少会有几个污点。

没有人喜欢犯错，但这是有意义的生活经历的一部分…尽管它们很糟糕，但如果我们选择学习，每个错误都给我们上了宝贵的一课。

好消息是，不是错误定义了你；这是你处理它们的方式。因此，虽然你犯的错误可能是没有预料到的，没有计划的和不想要的，但你可以通过积极的方式做出回应，防止一个孤立的错误演变成全面的灾难，从而重新获得一些控制。

以下是你下次犯错时可能想做的一些事情。

**负责**

当你犯错时，最糟糕的事情之一就是指责别人。这不仅让你看起来像是在逃避自己的责任，而且转移错误的责任会让其他人不必要的陷入困境。这也阻碍了问题的解决，因为问题的源头(你)现在已经不在循环中了。这实质上加剧了错误的影响。

一旦一个错误引起了你的注意，无论它是否可以避免，都要接受你在这种情况下的责任。然后开始调查，看看需要做些什么来纠正这种情况。

**避免找借口**

即使你对这个错误有一个坚实的理由，它也不能带走你一开始就犯了这个错误的事实，它可能对其他相关人员产生的任何影响都是不可抹去的。提出正当理由就像把责任推给别人，延长了时间，却没有解决任何问题。

这并不是说你不应该在必要的时候提供错误发生的背景，但是不要让推理成为你承认错误的主要方法。

**修复它**

一旦你承认了错误，并对这种情况有了主导权，就该去弥补了。把它当作头等大事，立即去做。如果其他人在导致失误方面发挥了作用，您将需要召集团队来制定解决方案。关键是要尽快纠正。

**从中吸取教训**

然而，这还不足以弥补错误。你应该花时间检查原因，它的影响，以及如何在将来避免同样的错误。记住，每一个错误都是自我提升的宝贵机会。接受学到的教训，让它让你做得更好，并在前进中应用它。

**继续前进**

错误已经被犯下，被承认，被修复，并从中吸取教训，但是如果你内心深处有任何完美主义的欲望(像我一样)，那么就很难走出错误，继续生活。认识到错误是不可避免的，在错误之下，它们真的是有益的。而你是人…就让它去吧。
