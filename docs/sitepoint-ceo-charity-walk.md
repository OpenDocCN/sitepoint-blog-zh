# SitePoint 首席执行官为慈善步行 100 公里

> 原文：<https://www.sitepoint.com/sitepoint-ceo-charity-walk/>

SitePoint 的首席执行官卢克·卡斯伯特森(Luke Cuthbertson)今天不去上班，而是去散步。

请注意，这不仅仅是在公园里漫步。卢克正在参加乐施会毅行者挑战赛，这是一个艰苦的课程，参赛队伍要徒步穿越 100 公里(62 英里)的澳大利亚丛林。今年，超过 700 个四人小组参加了这次挑战，为乐施会澳大利亚分会筹集资金。乐施会是一个久负盛名的非营利组织，为澳大利亚和世界各地需要帮助的社区提供[支持项目。](http://www2.oxfam.org.au/trailwalker/melbourne/trail/)

卢克和其他团队正在解决的问题不是轻而易举的(请原谅这个双关语)。很艰难。团队在夜间行走，需要训练、泡罩包装和支持团队。为了在不到 48 小时的时间内完成这一课程，步行者们忍受着水泡和疲劳，这一切都是为了一个良好的事业筹集资金。

如果你想知道卢克是如何追踪的，你可以在推特上[关注他，或者通过](http://twitter.com/lukie)[跑腿管理员](http://runkeeper.com/user/lukie/activities)获得实时 GPS 更新。如果你愿意，你也可以通过在线捐款来表达你的支持。SitePoint 和 99designs 都慷慨地向 Luke 的团队捐赠了 500 美元，办公室里的许多人也在赞助 Luke 的步行活动。到目前为止，他们已经设法筹集了 3，000 多美元，这是一个了不起的成就。

*   [在乐施会挑战赛中向卢克的团队捐款](http://www2.oxfam.org.au/trailwalker/Melbourne/team/310)

我代表办公室的所有人，祝你好运，卢克！

更新:卢克的手机似乎没电了，所以实时更新和推文已经停止。你仍然可以在乐施会网站上了解卢克团队的进展。人们也报告了在线捐赠的困难——如果你有兴趣支持这项有价值的事业，请继续努力。

更新:卢克的两名队员坚持到了最后。不幸的是，卢克不得不在 93 公里的检查站退出——离终点只有 7 公里了！
