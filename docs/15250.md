 ****##### 问题是** 

对于任何一个拥有大量域名的人来说，域名管理都是一个挑战。你如何跟踪你的域名注册了哪些注册商？哪些电子邮件地址被用作联系地址？哪些域名服务器被哪些域名使用？或许最重要的是，域名何时到期？

一种常见的域名管理方法是使用电子表格或数据库，并以这种方式跟踪域名。但是，无论何时发生变化，都要手动更新，这是一件非常耗时的事情。错误会发生，更改会被忽略，并且不可避免地，这种领域数据库的准确性总是会受到质疑。当然，如果你的记录不是 100%准确，会出现很多问题。特别是，未能在到期日之前更新域名可能会导致其永久删除并被其他方重新注册。

##### 解决方案

幸运的是，一个产品出现了，它真的有希望解决域管理的麻烦。 [Watch My Domains Pro](http://www.engineready.com/products/watchmydomains/index.html) (或 WMD)是一个 Windows 程序，它通过分析您提供的所有域名的 WHOIS 记录来工作。一旦分析完成，您就有了一个易于查看的域投资组合快照。在分析过程中收集并制成表格的信息包括:

*   登记员
*   国际电脑互联网地址
*   名称服务器
*   管理电子邮件
*   技术电子邮件
*   帐单电子邮件
*   编成日期
*   上次更新日期
*   有效期限

您可以将这些信息用于以下用途:

*   **Expiry Date** – This is probably the single most important information within WHOIS records, as your failure to renew your domain name before the expiry date could result in the immediate deletion of the domain name. WMD will default to highlighting in red all domains within 60 days of renewal, so you can easily see which domains need to be renewed, a really great feature.

    此外，你可以改变天数，从 60 天到任何你想要的。您也可以使用到期日期信息来快速查看本月、下个月等有多少域名到期续订。–非常适合财务规划。

*   **域名服务器**–WMD 能够跟踪每个域名的三个域名服务器。这允许你检查域名是否使用了你认为的域名服务器。您还可以毫不费力地查看是否有任何名称只使用一个名称服务器，如果您想保持网站持续运行，这是不推荐的。此外，如果您曾经移动网络主机，能够看到哪些名称服务器，你的名字是目前连接可以使整个过渡过程更容易，并确保没有名称被遗留在移动。
*   **IP 地址**–如果域名设置不正确，无法指向有效的 IP 地址，WMD 将显示错误。如果设置了，WMD 将指示域指向的 IP 地址。同样，如果您移动 Web 主机，或者只是想检查您的域是否确实被分配了您认为的 IP 地址，这是非常有用的信息。
*   **联系电子邮件**–WMD 将列出管理电子邮件、技术电子邮件和账单电子邮件地址。它还有一个叫做“标记邮件”的漂亮功能，允许你指定一个“已知”邮件地址列表。任何出现在域记录中且之前未被您标记的联系电子邮件将以红色突出显示。这是一个很棒的功能——跟踪联系电子邮件是良好域名管理的重要组成部分。许多域名已经丢失，因为管理电子邮件地址不再有效，或者不再由域名所有者定期检查。

一旦你有了所有这些信息，你就可以很容易地使用上面列出的任何字段对域名进行排序。此外，很容易将信息导出为逗号分隔的文本文件或 Access 兼容的数据库文件。对导出信息的唯一限制是，在此过程中不会导出电子邮件地址，除非它们是“带标签的电子邮件”。这确保了该产品不会被用于从 WHOIS 记录中收集电子邮件地址以发送垃圾邮件——因此，这是一项应该受到欢迎的限制。如果你真的需要无限制版本，你需要看看 [Watch My Domains ISP](http://www.engineready.com/products/watchmydomains/isp/index.html) 。
