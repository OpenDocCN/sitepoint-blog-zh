# 热情的红色

> 原文:[https://www.sitepoint.com/the-passionate-color-red/](https://www.sitepoint.com/the-passionate-color-red/)

[![virgin_logo](../Images/ba70fbfed4e093d7b623e66c9e35315d.png "virgin_logo")](https://www.sitepoint.com/wp-content/uploads/2010/01/virgin_logo.jpg) 今天是一个新系列的开始，看看设计中的个体色彩，简单看看色彩心理学和大量的视觉例子。色彩是设计中的一个重要元素。它被用来影响我们的情绪，影响购物选择，以及象征情感。所以让我们从可能是最情绪化的(也是我个人最喜欢的)颜色红色开始吧。

颜色感知可能是主观的，但普遍认为暖色位于色谱的红色区域(红色、橙色、黄色)，冷色位于色谱的蓝色区域(蓝色、绿色、紫色)。红色唤起温暖和舒适的情感，但也唤起敌意和愤怒。几乎在世界上所有的交通灯上，红色意味着停止。色彩疗法或光疗法研究表明，红色可以增加血压和呼吸频率。在远东，新娘经常穿红色，在南非红色是哀悼的颜色，在印度红色代表正直，在俄国布尔什维克推翻沙皇时使用红色旗帜后，红色开始代表共产主义。

我们在日常语言中用红色来描述许多不同的事物。我们可以看红色，过一个红色的节日，乘红眼航班，铺上红地毯，把城镇涂成红色。显然，红色对很多人来说意味着很多东西。

**红色设计**

说到设计，**红色抓住了注意力**。这不是一个壁花背景色，它从页面中喊出。它从照片中跳出来。你可以非常大胆地在整个设计中广泛使用它，尽管一点点就可以走很长的路。看下面两张电影海报。它们面向两种完全不同的受众，但都有效地利用红色来吸引我们的注意力。“阅后即焚”(这个设计是向索尔·巴斯致敬)有着明亮的红色背景和大而明亮的文字。看起来很乐观。
