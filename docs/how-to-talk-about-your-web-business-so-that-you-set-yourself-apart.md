# 如何谈论你的网络业务，让你脱颖而出

> 原文：<https://www.sitepoint.com/how-to-talk-about-your-web-business-so-that-you-set-yourself-apart/>

Sitepoint 访问者迈克尔写道:

我真的需要在竞争中脱颖而出。我遇到的大多数人听说我是一名网页设计师，我得到的回应是，“哦，又一个”。你能在即将到来的博客中给出一些关于如何定义这个问题的
提示/暗示/想法吗？

问得好，迈克尔！首先，请继续以我的方式发送问题，这样我们可以保持这个博客的互动性和趣味性。

现在来一个答案:

首先，停止用你做什么来描述你的生意。一旦你说，“我是一个网页设计师”，你就成了一个普通的供应商。

你需要用营销人员所说的“钩子”来介绍你自己。挂钩是让别人对你的工作感兴趣的一种令人信服的方式。实心挂钩回答了三个问题:

1.你的目标市场是谁？
2。你解决了什么问题(最好是情感方面的)？
3。有什么福利？

以下是一些例子:

“我帮助那些努力吸引更多顾客的家具零售商通过强大的网络业务拓展业务。”

“我帮助资源匮乏的非营利组织利用网络获得他们需要的所有捐款、志愿者和支持。”

“我帮助律师获得更多客户，让他们看起来比竞争对手更专业、更可信、更有见识。”

一旦你引起了他们的注意，你需要再回答几个问题:

1.与其他人相比，你有什么独特之处？
2。你是怎么做到的？
3。有什么证据？

以下是更多的例子:

“与其他网页设计师不同，我们只专注于律师。事实上，我们已经和这个地区的五大律师事务所合作过了。我们有一套专有的方法，涵盖了每位律师有效开展网络业务所需的六个要点。”

“与其他网页设计师不同，我们超越了设计的范畴，提供了交钥匙解决方案。我们可以将您的网络形象与您的会计系统、客户关系系统以及对外销售线索生成和跟进流程联系起来。我有一大堆客户的参考资料、tesimonials 和案例研究，讨论这种方法为他们带来的结果。”

“我们是唯一一家提供退款保证的网络公司。事实上，在你告诉我们你对我们的成品感到满意之前，你是不会付钱给我们的。”

迈克尔，我希望这有所帮助！每个人:除了发布更多的问题，请给出更多的例子，说明你如何谈论你的网络业务，让自己与众不同…..
