# 10+ jQuery 文本编辑器插件

> 原文:[https://www.sitepoint.com/10-jquery-text-editor-plugins/](https://www.sitepoint.com/10-jquery-text-editor-plugins/)

10 个 jQuery 文本编辑器插件，帮助你在你的网站上获得 **WYSIWYG** (所见即所得)。当我们需要在线编辑文章、帖子甚至文档时，JavaScript rich live text editor 简化了我们的生活。大多数编辑器允许用户直接编辑内容——所见即所得，就像用 Microsoft office 编辑文档一样。如今，我认为所有的内容管理系统、博客系统都在使用富文本编辑器。

所以，这里是我能在网上找到的 10 个 rte。从最基本的字体格式支持到图像上传、表格创建、Ajax 支持等高级功能。尽情享受吧！

## [1。马克图普。–jQuery](http://markitup.jaysalvat.com/home/)

马克图普。并不意味着是一个“全功能开箱即用”的编辑器。相反，它是一个非常轻量级，可定制和灵活的引擎，以满足开发人员在他们的 CMSes，博客，论坛或网站的需求。马克图普。不是所见即所得编辑器，也永远不会是。
[来源](http://markitup.jaysalvat.com/home/)

## 2.jWYSIWYG–jQuery

这个插件是一个内嵌的内容编辑器，允许动态编辑丰富的 HTML 内容。它是 WYMeditor 的替代产品，功能要少得多。这个小文件总共不到 17Kb 26Kb，只有 9Kb 18Kb 的代码和 7Kb 的压缩文件，主要概念是保持简单，不是所有用户都需要字体颜色或创建表格，只需要基本的。

## [5。WYMEditor–jQuery](http://www.wymeditor.org/demo/)

WYMeditor 是一个基于 web 的 WYSIWYM(所见即所得)XHTML 编辑器。WYMeditor 的主要概念是留下文档视觉布局的细节，专注于它的结构和意义，同时尽可能给用户带来舒适(至少作为 WYSIWYG 编辑器)。

## [6。TinyMCE–非 jQuery](http://tinymce.moxiecode.com/)

TinyMCE 是一个独立于平台的基于 web 的 JavaScript HTML WYSIWYG 编辑器控件，由 Moxiecode Systems AB 在 LGPL 下作为开源发布。它能够将 HTML 文本区域字段或其他 HTML 元素转换为编辑器实例。TinyMCE 非常容易集成到其他内容管理系统中。
![TinyMCE-non-jQuery.jpg](../Images/9b5e8126b62e9d0eb1b9bd312b38d1c2.png)
T3】来源

## [7。fckeitor–非 jQuery](http://ckeditor.com/)

这个 HTML 文本编辑器为 web 带来了桌面编辑器(如 MS Word)的强大功能。它是轻量级的，不需要在客户端计算机上进行任何安装。

## [8。雅虎 YUI 富文本编辑器-非 jQuery](http://developer.yahoo.com/yui/editor/)

富文本编辑器是一个 UI 控件，它取代了标准的 HTML 文本区域；它允许对文本内容进行丰富的格式化，包括常见的结构处理(如列表)、格式化处理(如粗体和斜体文本)以及图像的拖放和大小调整。
![Yahoo-YUI-Rich-Text-Editor-Non-jQuery-.jpg](../Images/0e7e58c3b1f83b1a5340d9a2b4198c80.png)

## [9。xinha–非 jQuery](http://xinha.webfactional.com/)

Xinha(发音像谢轩，战士公主)是一个强大的所见即所得的 HTML 编辑器组件，可以在所有当前的浏览器中工作。它的可配置性和可扩展性使得为多种目的构建合适的编辑器变得容易，从用于一个数据库领域的受限迷你编辑器到成熟的网站编辑器。其自由的 BSD 许可使其成为集成到任何类型项目中的理想选择。
