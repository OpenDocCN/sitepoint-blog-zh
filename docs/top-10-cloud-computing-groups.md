# 十大云计算集团

> 原文：<https://www.sitepoint.com/top-10-cloud-computing-groups/>

曾经想要更多地参与云计算吗？本周，我们将带您浏览 LinkedIn 上的 10 个顶级云计算小组，这将满足您的好奇心！

### [PHP 云开发](http://www.linkedin.com/groups?about=&gid=4262664&trk=anet_ug_grppro)

如果你是一个充满热情的 PHP 开发人员，想要在云计算开发中增长你的 PHP 知识和专业技能，那么这就是适合你的团队。不仅如此，如果你正在寻找一个伟大的角色，为你的组织或项目寻找帮助，那就在志同道合的人中寻找吧。

### [虚拟化&云计算集团](http://www.linkedin.com/groups?about=&gid=57400&trk=anet_ug_grppro)

一个由虚拟化和云计算专业人士组成的专门小组，他们可以就虚拟化、VMware、Microsoft Hyper-V、Citrix Xen、安全性、企业、移动、存储、VCP、VCDX、云能源、SaaS、PaaS、数据存储、安全性等行业相关话题进行交流和讨论。技术问答和新闻都是支持和鼓励的。

### [甲骨文云计算集团官方](http://www.linkedin.com/groups?about=&gid=3262238&trk=anet_ug_grppro)

官方，由甲骨文创建，甲骨文云计算集团，在这里您可以找到所有关于甲骨文的云战略，甲骨文的云计算产品的新闻，并与同行联系。

### [云计算——微软英国公司](http://www.linkedin.com/groups?about=&gid=3837607&trk=anet_ug_grppro)

该社区面向对云计算感兴趣的高级 IT 经理和高管以及企业领导人，供他们进行讨论、分享想法、寻求建议和意见。该组织的重点是促进关于云计算潜在影响的讨论和对话；要克服的障碍和挑战，增加业务和 IT 价值的机会，以及分享关于向云迁移的正确步骤的实用技巧。

### [Carbonite Business SMB 云计算集团](http://www.linkedin.com/groups?about=&gid=3984655&trk=anet_ug_grppro)

Carbonite Business 是关于小型企业的在线备份。

### [云安全联盟](http://www.linkedin.com/groups?home=&gid=1864210&trk=anet_ug_hm&goback=%2Egdr_1328754794893_2)

云安全联盟是一个非营利组织，其使命是促进使用最佳实践来提供云计算内的安全保证，并提供关于使用云计算来帮助保护所有其他形式的计算的教育。

### [云存储](http://www.linkedin.com/groups?home=&gid=132257&trk=anet_ug_hm&goback=%2Egdr_1328754794893_2)

云存储小组的成立是为了给云存储和计算技术的引入和发展提供一个共同的基础。

### [Cloud camp——非常规云计算](http://www.linkedin.com/groups?home=&gid=122638&trk=anet_ug_hm&goback=%2Egdr_1328754794893_4)

CloudCamp 的形成是为了给云计算的引入和发展提供一个共同的基础。

### [云计算](http://www.linkedin.com/groups?home=&gid=61513&trk=anet_ug_hm&goback=%2Egdr_1328754794893_1)

该组织组织了两次年度会议 Cloud Slam 和 up，这是世界上最重要的云计算活动，涵盖了云计算领域的技术、商业模式、行业经验、法律问题、研究、开发和创新。

### [云计算，SaaS &虚拟化](http://www.linkedin.com/groups?home=&gid=45151&trk=anet_ug_hm&goback=%2Egdr_1328754794893_1)

云计算和虚拟化专业人士小组，旨在扩大他们的联系网络，分享想法并讨论行业相关话题。

该集团涵盖 VMware、SaaS、PaaS、云安全、云计算和服务器虚拟化技术、企业 2.0 应用、技术和架构、crm、云服务、数据中心、软件即服务、按需应用。

### [Azure 开发者](http://www.linkedin.com/groups?gid=1731317&mostPopular=&trk=tyah)

这个群是为微软的。NET 开发人员讨论 Windows Azure 服务平台及其作为互联网和云应用程序和服务的实现。
