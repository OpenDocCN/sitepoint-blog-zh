# 关联数据的隐藏面

> 原文:[https://www.sitepoint.com/the-hidden-side-of-linked-data/](https://www.sitepoint.com/the-hidden-side-of-linked-data/)

我之前的两个[帖子](https://www.sitepoint.com/structured-data-set-to-take-on-google/)提出了一些我觉得非常有趣的问题，并引出了一些未回答的问题。一个[的评论](https://www.sitepoint.com/one-small-step-for-yahoo-one-giant-leap-for-embedded-metadata/#comment-641238)特别引起了我的思考。

当我们在网络上的信息不再分散，不再隐藏在数据库中——被成千上万不同的隐私政策锁起来，而我们没有人阅读(至少我没有)时，会发生什么？不管我们的社交网络有多广，我们都会在网络上留下痕迹。博客条目的每一个快速回复，亚马逊上的每一条评论，易贝上的每一个评级；所有这些都会让我们对自己有一个更清晰的“印象”。这张照片展示了更多关于我们是谁，我们做什么，甚至我们为什么做这些。

这可能有点可怕。其实这挺吓人的。我真的希望有人知道我在易贝上购买的所有东西、我在亚马逊上评论的所有书籍以及我在博客上的所有回复吗——所有这些都可以方便地在一个小小的搜索中找到？或者用嵌入微格式或 RDFa 的智能数据很好地连接在一起。

我不知道。

这些信息是公开的，如果有人真的很感兴趣，可以出去收集这些数据。事实上，已经有越来越多的人使用搜索引擎了，尽管人数不多。但是这些搜索引擎的工作方式就像 Google works 一样，大部分都是非结构化的原始数据。人民搜索引擎调整工作与原始文本和其他一些关键的标识符，以建立他们的个人资料。蛮力是关键。

微格式、RDFa、RDF、FOAF、开放社会(以及更多)都在构建一个更加结构化的网络，在这个网络中，暴力不再是寻找联系和寻找东西的唯一方式。关于这些之前隐藏的联系，有很多未知的东西变得非常突出，更重要的是，很容易被发现。如果受过良好教育的微格式受众[对所连接的内容感到惊讶](https://www.sitepoint.com/one-small-step-for-yahoo-one-giant-leap-for-embedded-metadata/#comment-641238)，那么当普通网络受众开始看到这些东西明显凭空“出现”时会发生什么？

我觉得他们不会开心，一点都不会。
