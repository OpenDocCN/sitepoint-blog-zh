# 设计中的视觉和谐第四部分:视觉呼应

> 原文:[https://www . site point . com/visual-harmony-in-design-part-4-visual-echo/](https://www.sitepoint.com/visual-harmony-in-design-part-4-visual-echo/)

在过去的几周里，我们简要地看了设计师如何在他们的作品中实现视觉和谐。我们已经看到元素的[重复](https://www.sitepoint.com/harmony-in-design-part-2-repetition/)如何带来和谐并吸引注意力，以及[“主题参考”](https://www.sitepoint.com/visual-harmony-in-design-part-3-thematic-reference/)如何在内容和设计元素的主题之间建立联系。在这个简短系列的最后一部分，我们将讨论“视觉回声”的概念。视觉回声在布局中的设计元素之间建立了一种吸引人的统一感，无论该布局是用于 web 还是用于打印。

**回声穿透颜色**

实现视觉呼应的一个简单方法是在布局的颜色、排版和样式中重复使用或呼应照片或插图中的颜色。使用 Photoshop 中的吸管工具或图形编辑器从图像中挑选一种颜色，或使用 [Kuler 从图像](https://www.sitepoint.com/how-to-create-a-color-palette-from-an-image-using-kuler/)中创建一种配色方案。

看看这张电影宣传海报。主照片在颜色方面非常柔和，但汤姆克鲁斯脸上的金色肤色在演员的名字和电影名称中都有所呼应。

[![collateral](../Images/3ce22d7ecad40ed4106ce41310c60279.png "collateral")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/collateral.jpg)

友好鸭网站看似简单。鸭子插图中的白色和橙色在字体、语音气泡和你可以在下面看到的页面上的圆形图表中得到呼应。这不是意外。如果每一个设计元素都是完全不同的颜色，那么这种设计就不会有效。

[![Friendly Duck](../Images/86eba6c9fd7f60b00f9afa5c481f4be1.png "Friendly Duck")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/FriendlyDuck.jpg)

**通过排版产生回声**

呼应也可以通过排版来实现。Kidd 81 网站上相同的字体出现在标识和标题、简介和侧边栏中。通常当一个设计看起来很弱的时候，那是因为设计师没有坚持使用排版。这位设计师并不害怕在网站的几个部分使用手绘风格。出于实用的目的，比如搜索引擎优化，设计者将被要求为网站的主要文本使用网页安全字体，但是在这个网站上，反响强烈且意味深长。

[![Kidd81](../Images/5b7c02ea83701c4c7f91f0592e216169.png "Kidd81")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/Kidd81.jpg)

Rikcat industry 是印刷术中一个类似但更微妙的回声例子。徽标、导航和侧边栏中的字体是相同的无衬线字体，而标题使用衬线字体。白底黑字和黑底白字的运用形成了有趣的对比。

[![Rikcat](../Images/8a9f0bedd92a3c3929879f2f1daeefd8.png "Rikcat")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/Rikcat.jpg)

**回显风格**

也可以通过风格创造呼应。这个[女性偶像的广告](http://www.behance.net/Gallery/LOU-DOILLON-FOR-I-GOLD-CAMPAIGN/147426)结合了手写的、肮脏的印刷字体和风格化的黑白照片，照片在粉色背景上稍微有点倾斜。它没有说“我是一个女孩，女孩”，它喊着“我是一个硬核，摇滚女人！"

[![RockIcons](../Images/8696b5ba87fc7c5ec7c54716591e29f7.png "RockIcons")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/RockIcons.jpg)

放眼望去，你会发现视觉和谐的例子。正如我在本系列的[第一篇文章中提到的，和谐并不意味着设计必须只针对一个“好”的主题，它只是意味着所有的元素彼此一致。只要你对颜色、风格、版式和质地做出选择，它们可以是粗糙的、当然的、令人震惊的，也可以像馅饼一样好。](https://www.sitepoint.com/new-series-harmony-in-design/)

这是关于在你的作品中实现和谐的简短系列。希望你觉得有用。
