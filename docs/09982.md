# HTML5 画布 JavaScript 动画示例

> 原文:[https://www.sitepoint.com/html5-canvas-animation/](https://www.sitepoint.com/html5-canvas-animation/)

非常酷的 HTML5 Canvas 示例，它使用 JavaScript 和 Canvas 元素来创建由 mouseover 事件控制的动画效果。

![google-html5](../Images/cce62280604ecc5522251d39a01239f0.png "google-html5")

**说明:**将鼠标放在谷歌标志上移动小球，然后观看它们重新组合。

```
< !DOCTYPE HTML>

<canvas id="myCanvas" width="578" height="200" onmousemove="handleMouseMove(event)" onmouseout="handleMouseOut()">

</canvas> 
```
