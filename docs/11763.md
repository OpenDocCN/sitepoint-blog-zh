# 哦，闪亮 Twitter 的 Flash 前端！

> 原文：<https://www.sitepoint.com/oooh-shiny-a-flash-front-end-for-twitter/>

你必须住在没有 wi-fi 的山洞里，才能听说这个叫做 [Twitter](http://twitter.com/sitepointdotcom/) 的东西，我们最新的教程向你展示如何用 Adobe Flex 编写一个闪亮的 Flash Twitter 前端。

在 [*使用 Flex 在 Twitter 上共享媒体，第 1 部分:基础知识*](https://www.sitepoint.com/article/share-media-flex-twitter)Andrew“Flex Daddy”Muller 向我们介绍了如何使用 *Adobe Flash Builder 4* (目前提供测试版下载)、 *Flash Catalyst* (更多全新的测试版软件)和少量 PHP 来构建一个用于在 Twitter 上共享媒体的 Flex 应用程序。这都是非常前沿的“社交媒体”…而且实际上相当有趣。

如果您还没有使用过 Flash Catalyst 或 Flash Builder 4，那么您将会喜欢这些实践教程-这是由三部分组成的系列中的第一部分，Andrew 将在后面的部分中扩展示例应用程序。

没有什么比接触并尝试新技术更令人兴奋的了，为了帮助你更好地理解新技术，有一个关于教程的测验来测试你的知识(由 Adobe 赞助)。
