# OSS 开发者的免费 IntelliJ 想法

> 原文：<https://www.sitepoint.com/free-intellij-idea-for-oss-developers/>

你有没有想过开始一个开源软件(OSS)项目？你已经捐过了吗？如果是这样，JetBrains 愿意放弃他们顶级 Java IDE 的免费许可。

这实际上正在成为 Java 开发领域中反复出现的主题；大型软件供应商似乎认为让他们的产品在 OSS 领域广泛流通是一种很好的广告形式。我不能说我不同意。

据报道，IntelliJ IDEA 是一项了不起的工程，尽管我已经使用 Borland JBuilder 很多年了，但我确实想尝试一下 IDEA 和 Eclipse 之类的替代产品。这个提议可能正是你想要的。

为了参与这项交易，你必须在网上填写一份申请表。如果你合格，你将获得一年的免费许可证。只要你继续积极参与一个合格的 OSS 项目，你可以每年更新许可证。

如果你想赶时髦，并且你还没有正在进行的 OSS 项目，你可以获得 IntelliJ IDEA 的 30 天免费评估版,开始你的项目，然后申请免费许可证。
