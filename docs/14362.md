# 洞察为什么你们中的一些人没有采取足够的行动来营销你的业务

> 原文:[https://www . site point . com/insights-into-why-some-of-you-parent-take-up-action-to-market your-business/](https://www.sitepoint.com/insights-into-why-some-of-you-arent-taking-enough-action-to-market-your-business/)

上周，一些客户意识到他们没有采取足够的行动来发展业务。他们都很聪明，都很可爱，都想建立一家 IT 服务公司。

然而，两人都承认，在营销他们的业务时，他们很懒。

当我们深入探究原因时，以下是缺乏行动的一些潜在原因:

–害怕给别人打电话，让他们说不。

–不喜欢权威。正如一位企业主所说，“我去为自己工作，以避免有一个老板。现在，似乎每个客户都是老板，所以我有 100 个老板。”

–很难对为某些企业工作感到兴奋。对这个人来说，太多的企业道德有问题，或者以不符合他们自己价值观的方式销售某些东西/市场。

在一些困难的问题确认他们真的适合自主创业后，以下是我们如何解决所有这三个问题的，如果你分享其中的任何一个:

1.他们正在创建一个他们希望用他们的解决方案服务的本地企业的列表——这些企业可以让他们感到兴奋，并且不会与他们的道德相冲突。这样，他们会对接近他们并谈论他们的 IT/网站感觉良好。

2.我们将就如何接触这些企业进行角色扮演，以便这些潜在客户更有可能与他们交谈。(更多信息请见下一篇博客，或下一篇)。

3.我们将做一个练习，这样“不”就是“不”，没有别的意思，所以没有人会把它当成个人问题。(下一篇博客将详细介绍这一练习)。

4.他们会专注于每周采取一些小行动，养成优先考虑营销的习惯。

很容易以对上述问题缺乏同情心的方式回复这篇博客。就我个人而言，我总是惊讶于有多少人为了摆脱“老板”而创业，却发现自主创业是一条艰难得多的道路；毕竟，当你独自一人时，当事情不顺利时，没有人可以责怪，而且你真的会有多个老板。

但是这些问题相当普遍。不要批评这些人——特别是当他们诚实地面对他们的问题时——看看你自己的情况，问问你可以采取什么具体措施来有效地营销你的业务。
