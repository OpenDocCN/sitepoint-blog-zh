# 谷歌应用程序放弃 IE6 支持

> 原文：<https://www.sitepoint.com/google-drops-ie6-support/>

谷歌宣布，从 2010 年 3 月 1 日起，他们将在文档和网站中停止对 Internet Explorer 6.0 的支持。此后，IE6 和其他过时的浏览器的关键功能可能会失效。

这家搜索巨头为这一行为辩护，因为“许多其他公司已经停止支持旧浏览器”。这当然是真的，YouTube 就是一个引人注目的例子。*(我敢肯定谷歌拥有 YouTube 是巧合！)*

谷歌有充分的商业理由鼓励 IE6 的消亡。他们未来的战略和网络应用依赖于新的网络技术，比如 HTML5，它不能在旧的浏览器中工作。放弃 IE6 也可能说服一些个人和公司转而使用谷歌 Chrome 浏览器。

该公司的公告称，用户有几个选择，如 IE7+，火狐 3.0+，Chrome 4.0+和 Safari 3.0+。Opera 没有被提及，但我怀疑这是一个新闻稿的疏忽——谷歌文件在 10.10 版本中工作，我怀疑谷歌是故意冷落浏览器。SynBay

作为一名开发者，我欢迎谷歌对 IE6 的立场。然而，他们的陈述有几个不同寻常的方面:

1.  IE6 的全球市场份额为 15%。它在美国和欧洲只有 5%左右，但使用它的人仍然比谷歌 Chrome 多。谷歌拥有巨大的权力，那么他们应该能够决定人们使用哪种浏览器吗——特别是当他们偏向于自己的软件时？
2.  根据我的经验，IE7 引发的开发难题和 IE6 一样多。用户没有什么理由不升级到 IE8，那么为什么不也放弃 IE7 呢？
3.  如果我在强制执行 IE6 的企业环境中使用 Docs，我会有点恼火——特别是如果我为 Google Apps 付费的话。我不知道这影响了多少人，但我打赌肯定不止一个人！
4.  谷歌文档目前在 IE6 中工作。像 YouTube 一样，我怀疑这项服务会在 3 月 1 日突然停止工作。新功能可能会引起问题，但 IE6 还需要几年时间才能变得不可用。

也许我分析过度了？任何促使人们升级的消息都不可能是坏事！查看[谷歌的完整公告](http://googleenterprise.blogspot.com/2010/01/modern-browsers-for-modern-applications.html)了解更多细节…
