# Netscape Navigator 静静地消失在夜色中

> 原文:[https://www . site point . com/Netscape-navigator-goes-quiet-into-the-night/](https://www.sitepoint.com/netscape-navigator-goes-quietly-into-the-night/)

新年前夕，美国在线传出消息，网景浏览器的漫长生命即将结束。

![Netscape logo](../Images/c14340ec6ba3f5e22a2ede1c943ece24.png)

[结束支持声明](http://blog.netscape.com/2007/12/28/end-of-support-for-netscape-web-browsers/)尊重了浏览器的悠久历史——比任何其他仍在普遍使用的浏览器都要长——因为它承认是时候叫停了:

> “尽管 AOL 内部团体投入了大量的时间和精力试图复兴 Netscape Navigator，但这些努力未能成功地从微软的 Internet Explorer 获得市场份额。最近，对网景浏览器的支持仅限于少数几个工程师，他们的任务是创建一个带有一些扩展的 Firefox 皮肤版本。

有趣的是，该公告将 ie 列为网景的主要竞争对手。虽然曾几何时这是真的，但在我看来，如果网景专注于向火狐用户营销，甚至让火狐用户向他们不太懂技术的朋友和家人推荐它，它可能会更成功——如果只是微不足道的话。

无论如何，Netscape Navigator 的积极开发现已停止，2 月 1 日以后将不会发布安全修补程序。

虽然这个消息对绝大多数 web 开发人员影响很小或没有影响，但我们这些网景是他们第一个浏览器的人会发出怀念的叹息。尽管如此，没有哪个浏览器能比 Mozilla 基金会和 Firefox 打造的浏览器更好了。

好吧 Netscape。
