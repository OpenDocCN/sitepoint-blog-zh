# 微软将在 2014 年前支持 IE6

> 原文:[https://www.sitepoint.com/microsoft-support-ie6-2014/](https://www.sitepoint.com/microsoft-support-ie6-2014/)

尽管许多在线活动呼吁 IE6 的死亡，微软已经确认他们对该浏览器的承诺直到 2014 年 4 月 8 日。IEBlog 上的一篇文章称，升级是用户的责任，公司将继续支持这款已有 8 年历史的浏览器:

> IE6 的工程观点始于操作系统供应商。放弃对 IE6 的支持不是一个选项，因为我们承诺在产品生命周期内支持 Windows 附带的 IE。我们信守承诺。许多人希望他们最初从操作系统中获得的东西能够继续工作，不管特定子系统的发布节奏如何。
> 
> 作为工程师，我们希望人们升级到最新版本。我们尽可能让他们轻松升级。最终，升级的选择权属于电脑的负责人。

不管喜欢与否，微软正在做正确的事情。Windows XP 扩展支持将持续到 2014 年，IE6 是该操作系统提供的浏览器。因此，IE6 支持和更新将至少再持续 5 年。他们不能也不会强迫用户升级。

微软不得不扩展 XP 支持有两个主要原因:

1.  Vista 晚了 4 年才发布，受到了媒体和用户的猛烈抨击。许多私人用户和公司继续使用 XP，并在新电脑上安装免费的操作系统降级选项。尽管 Vista 有所改进，但人们普遍认为它是一个糟糕的操作系统，只有 Windows 7 能提供可行的升级途径。
2.  低规格上网本的崛起。大多数设备无法有效运行 Vista，因此微软提供了 XP Home 作为替代。

然而，该公司致力于推广 IE8。微软自己的几个网络应用程序，比如新版的 online Office，不提供官方的 IE6 支持(尽管它们可能仍然可以工作，尤其是如果它们是基于 Silverlight 的话)。

微软对 IE6 的承诺重要吗？可能不会——鼓励浏览器升级是整个网络社区的责任。
