## 4.jQuery 的表排序器插件

主要的新特性包括:
*多列排序
*内嵌支持通过 TH 元素上的 class 属性设置选项。
*插件已经从头开始重新编写。
*支持 TH 元素上的 rowspan 和 colspan。
*新的 widget 支持，更多信息在 docs 中。
