# 5 jQuery 移动刷卡

> 原文:[https://www.sitepoint.com/10-jquery-mobile-swipe/](https://www.sitepoint.com/10-jquery-mobile-swipe/)

面对现实吧我们都喜欢时不时的刷刷手机！因此，今天我们收集了 5 个 jQuery 移动刷卡插件/教程供您查看。尽情享受吧！

## 1.jQuery iPhone 滑动手势

这个插件使用移动 Safari 的内置触摸事件来允许将滑动事件绑定到任何 DOM 元素。当检测到手势时，您可以覆盖 swipeLeft 和 swipeRight defaults 函数来创建您自己的自定义过程。
![jQuery-iPhone-Swipe-Gestures.jpg](../Images/ced477dac94f6414235bcc85c3e65426.png)
[来源](http://www.ryanscherf.net/)

## 2.wipe touch–在触摸设备上捕获擦除

是 TouchWipe 的增强版。它使捕捉划像手势(左、上、下、右和对角线)变得轻而易举，并且完全可定制。
![WipeTouch.jpg](../Images/8197392337c3db5953a49937aa396a59.png)
[来源](http://wipetouch.codeplex.com/)

## 3.jQuery 滑动图库

是一个轻量级的插件，它允许通过编写几行 HTML 代码，比如一个无序的图像列表，就可以创建一个图库。swipeGallery 针对移动网站进行了优化，并支持滑动手势。

## 4.jQswipe

向 jQuery 添加 swipe(或 rightSwipe)和 leftSwipe 事件。
![jQuery-Swipe-Gallery.jpg](../Images/9e2aa56bf29caafb8d76b48318fd158a.png)

## 5.多重擦拭

一个简单的插件，基于 Ryan Scherf 的 jSwipe，支持移动 Safari 中的多点触摸滑动事件。
![Multiswipe.jpg](../Images/1aa46b1af173a6e2c9f979eb2141f7d4.png)
[来源](http://plugins.jquery.com/project/multiswipe)
