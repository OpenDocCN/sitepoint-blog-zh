# 你真的有诚信吗？

> 原文：<https://www.sitepoint.com/do-you-really-have-integrity/>

今天我想到的是正直。

我的一个客户和他的投资人发生了争执。结果，他的投资者对我为他们工作而收取的美国运通卡费用提出异议，声称不认识我。客户没有回应，尽管他看起来对这项工作很满意(当我写这篇文章时，这项工作正好在他的网站上进行)。所以现在我不得不费力地与美国运通公司合作，来证明我的指控。

另一个客户已经收到了我的全额付款，但没有发布他所做项目的代码。实际上，是他的海外团队没有发布代码，但我认为他有责任。他正试图找到他们，但我怀疑他们。

完整意味着完整。在道德/伦理中，它意味着对什么是正确的有一个观点，做什么是正确的，并且在最高层次上采取立场纠正什么是不正确的。

迟到表明缺乏诚信。不守信用表明缺乏诚信。仅仅为了完成工作而提供有缺陷的工作表现出缺乏诚信。

我们都有食言的时候，通常是对自己而不是对别人。

在世界各地，我们在新闻中看到缺乏诚信的例子:缺乏对生命的尊重，人们因虚假谣言而煽动骚乱，为了推动自己的议程而不检查事实，腐败和贿赂，以及超越法律的界限。

但是缺乏诚信就是缺乏诚信，尽管其规模或后果有很大的差异。被问到的时候，大部分人都认为自己有诚信，不愿意面对自己可能没有的真相。指责某人缺乏诚信是一件严肃的事情。

在评判他人之前，第一步是向内看。我哪里缺乏诚信？

我的一个导师曾经说过，商业成功的方法就是给一个缺乏诚信的行业带来诚信。在网页设计和开发中似乎有机会这样做。

不幸的是，IT 世界充满了缺乏诚信的例子。
