# Orkut 被 XSS 蠕虫攻击

> 原文:[https://www.sitepoint.com/orkut-bom-sabado-xss-worm/](https://www.sitepoint.com/orkut-bom-sabado-xss-worm/)

先是推特被黑。然后脸书下去了。现在轮到 Orkut 了。谷歌的社交网站遭到了致命的“Bom 莎芭多”蠕虫的攻击。博姆莎芭多在葡萄牙语中是“耶稣受难日”的意思，葡萄牙语是巴西的母语，被认为是这种蠕虫的发源地。Orkut 是巴西、印度和其他几个国家最受欢迎的社交网站。

该蠕虫在不同账户间复制自己，并随机发送“轰炸莎芭多”信息到朋友的剪贴簿——Orkut 版的脸书之墙。谷歌支持部门最近宣布[蠕虫已经被遏制](http://www.google.com/support/forum/p/orkut/thread?tid=4ef1add575e866b9&hl=en)，他们正在清理受感染的账户。然而，该公司建议在访问账户时保持警惕——用户应该特别警惕点击可疑链接。

如果你已经被感染，你应该退出，清除浏览器的 cookies 和缓存，并在[google.com/accounts](https://www.google.com/accounts/)立即更改你的谷歌账户密码。

这些攻击提出了一个有趣的问题:黑客和垃圾邮件发送者正在将注意力转向社交网络吗？XSS 病毒比电脑软件可以检测到的病毒或恶意软件更容易制造和传播。此外，攻击传播迅速，跨越系统边界—使用哪种操作系统或浏览器并不重要。

事实证明，这些网络很容易受到攻击。所有公司都声明安全性是重中之重，但它永远不会像可用性或鼓励新注册那样重要。开放 API 和第三方应用程序也提供了另一种攻击手段。也许我们看到自我复制的蠕虫只是时间问题，它们可以在没有任何用户交互的情况下在整个网络中传播自己。

话说回来，这些攻击给社交网络带来了相当大的主流关注。无论如何，坏消息总比没有好。Orkut 在美国和欧洲相对不为人知，所以这个蠕虫可能是谷歌的一个大规模宣传噱头！但我绝不会建议这样的事。我会把这留给愤世嫉俗的阴谋论者…
