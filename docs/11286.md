## 截屏工具

[**all capture**T3】](http://www.allcapture.com)

一个 Windows 屏幕录制应用程序，支持全屏和多屏幕录制、特殊效果和注释、音频导入以及导出为多种格式。ALLCapture 还包括一项压缩技术，用于制作易于通过电子邮件发送的小演示。定价以每个许可证为基础。

[![ALLCapture](../Images/9ea681ccfdd3acc91313fc85c9ab8994.png)T2】](http://www.allcapture.com/)

**[CamStudio](http://camstudio.org/)**

免费、开源的 Windows 流媒体视频软件，允许您在计算机上录制屏幕和音频活动，创建 AVI 视频文件并导出到 SWF。CamStudio 有一个易于使用的界面，包括视频注释功能，自定义光标和选定的屏幕区域记录。

[![CamStudio](../Images/f7b7734c8859efe6906f354903dc67be.png)T2】](http://camstudio.org/)

[**卡姆塔西亚工作室**](http://www.techsmith.com/camtasia.asp)

TechSmith 的 Windows 截屏软件允许您捕捉屏幕上的任何内容，录制音频或稍后添加旁白，显示您的网络摄像头，并进行注释。Camtasia 还直接与 PowerPoint 集成，并利用 TechSmith 的压缩技术将文件导出为多种格式。它是在按许可付费的基础上提供的。 [Camtasia for Mac](http://www.techsmith.com/camtasiamac) 最近面向苹果用户推出。TechSmith 也是 Jing 的制造者(见下文)。

[![Camtasia](../Images/b3ae3630cb69f0f06a2a4ddec364360f.png)T2】](http://www.techsmith.com/camtasia.asp)

[**光斑**T3】](http://www.faculte.com/)

一个基于网络的屏幕录制工具，让您捕捉您的屏幕，网络摄像头和音频，以创建视频演示。您的视频可以在线存储和访问，并且能够进行支付和捕捉线索。费用是每月每观看/每个生产者的基础上，有一个免费的选项。

[![Faculte](../Images/0434ba39ae846f96bed850d47f05698a.png)T2】](http://www.faculte.com/)

[](http://freescreencast.com)

 **免费软件，让您记录您的屏幕，捕捉音频，控制光标，并出口到 FLV 格式。然后你可以上传到 FreeScreencast.com 免费托管(没有文件大小或分辨率限制)和共享。

[![Freescreencast](../Images/4cc9406d0d6b4919b3cf2d3428d18d6b.png)T2】](http://freescreencast.com)

Citrix Online(goto meeting 和 GoToMyPC 的制造商)的免费 Windows 软件，允许您录制屏幕视频、捕捉音频、编辑和托管视频，而没有带宽限制。截屏可以受密码保护，并下载/上传到其他网站。

[![GoView](../Images/0fe7eb214450e81e68486d920aee244c.png)T2】](http://goview.com/goldwyn/spring/play?method=indexPage)

[**HyperCam 3**](http://www.solveigmm.com/?Products&id=HyperCam)

HyperCam 3 是一个 Windows 屏幕记录实用程序，可以让您将数据捕获为 AVI 文件，在屏幕上做笔记和注释，并选择帧速率和压缩质量。它是在按许可付费的基础上提供的。

[![HyperCam](../Images/29203411db25f512808350ea4ff0fdeb.png)T2】](http://www.solveigmm.com/?Products&id=HyperCam)

[曲棍球 T3](http://shinywhitebox.com/)

一个用于 Mac 的屏幕录制实用程序，允许您录制屏幕视频和音频，包括鼠标捕捉、缩放、可调视频质量和多个导出选项等功能。可在按许可付费的基础上获得，也有高清版本可用。

[![iShowU](../Images/56f54feba68444c8b91fa0b5e13489bc.png)T2】](http://shinywhitebox.com/)

[**荆**T3】](http://www.jingproject.com/)

适用于 Mac 和 Windows 机器的免费简化屏幕录制软件，允许用户从桌面上的窗口或区域快速录制视频(包括音频)，包括鼠标、滚动移动和网站或应用程序上的点击。您最多可以录制五分钟，然后上传到 Screencast.com(有 2GB 的可用空间)或在本地保存为 SWF 文件。Jing 是 Camtasia Studio 和 Camtasia for Mac 的制造商 TechSmith 的产品。

[**像素点**](http://www.pixetell.com/)

Windows 软件，允许您创建和共享您的屏幕录像，嵌入您的网络摄像头，并添加文件附件到您的记录，然后将其导出到 FLV。Pixetell 还包括协作功能和安全的在线存储。免费试用后，定价是按月收费的。
