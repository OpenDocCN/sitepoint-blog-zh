# 不要害怕改变焦点

> 原文:[https://www.sitepoint.com/dont-be-afraid-to-change-focus/](https://www.sitepoint.com/dont-be-afraid-to-change-focus/)

一切都在变化，尤其是在商业领域。即使你的重点是一个利基行业或细分市场，你仍然必须准备改变路线，以确保你有一个可行的市场，并能保持竞争力。

#### Sprouter: Twitter 克隆到启动问答

Sprouter 最初是一项专门针对企业家和初创公司创始人的微博服务。然而，由于功能与 Twitter 基本相同，他们可能会发现很难与之竞争。去年，他们推出了一个新的问答功能，并亲自邀请专家参与，产生了更少但更可信和更有针对性的答案。该功能变得非常受欢迎，并揭示了一个最初没有考虑到的潜在市场。

Sprouter 最近宣布对他们的网站进行彻底重组，解散社交网络组件，并强调问答功能是这项服务的主要重点。

#### 可执行:从现场测试到分析

[Performable](http://www.performable.com/) 最初是一个网站测试工具，分析哪些设计元素、文案、颜色等对访问者最有吸引力。这个行业的竞争越来越激烈，许多初创公司都在争夺市场份额。Performable 决定改变他们的关注点，现在寻求提供对整个客户生命周期的分析。

我想分享一些来自 Performable 的建议，帮助你保持敏捷，为变化做好准备:

*敏捷*

您的业务越敏捷、越灵活，您在不断变化的行业环境中的机会就越大。

*关注比赛*

如果你不断看到新的竞争对手进入你的领域，这可能是一个信号，表明你需要进一步缩小你的关注点或完全改变方向。对于低成本的网页设计来说，这种情况越来越多。有这么多相互竞争的服务和服务提供商，市场已经过度饱和。

*倾听顾客的心声*

客户是否要求超出您核心关注范围的服务？你是否被要求投标你专业领域之外的项目？倾听您的客户，并尝试确定如何解决他们的问题。

*经常更换*

经常改变一点点，你就可以避免完全改变方向。通过不断测试新的市场、服务、产品和策略，你可以不断发展。
