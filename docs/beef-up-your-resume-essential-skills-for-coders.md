# 充实你的简历:程序员的基本技能

> 原文：<https://www.sitepoint.com/beef-up-your-resume-essential-skills-for-coders/>

如果你是在岩石下读到这篇文章的，你可能会感兴趣地发现，目前美国经济基本上是在枯竭。随着公司裁员，失业率不断上升，支出也明显减少。我有几个朋友和熟人最近被裁员，他们发现很难找到一份新工作。

最有吸引力的求职者是那些拥有多种技能的人，当工作稀缺时更是如此。即使你的工作看起来相当安全，也应该不断拓展你的职业技能。您可能已经掌握了最新的 web 开发技术，或者正在学习一门新的语言，但是更精明的极客们正在寻求增强和补充他们现有的技术优势。

所以今天，我们将看看一些职业技能，这些技能将会给一份令人讨厌的简历增加深度，为你自己在行业中树立一个名字，甚至可能会让你赚些外快。

## 跳上肥皂箱

极客喜欢酷的演示。一个有才华的演讲者可以把一个枯燥乏味的演讲变成一次难忘的、有趣的经历。虽然有些人能够从容应对一屋子的陌生人，甚至不需要尝试就能迷住他们，但我们其余的人必须努力做到这一点——而做到这一点的方法是通过练习和反馈。

你如何开始？好吧，你可以练习对你的狗说话或者在车里对自己说话，但是是*反馈*会帮助你提高这些技能。为此，你需要和*的人谈谈。*考虑在即将到来的会议上，与当地行业或用户组的组织者谈论一个热门话题。闪电式的谈话、教程和演示通常是受欢迎的，在非正式的环境中与你的同事交谈是练习你说话技巧的好方法。

如果你对此不感兴趣，或者你对在一群人面前讲话感到不舒服，你可以录下自己对一个你很熟悉的主题的讲话，并以视频或音频播客的形式发布到你的博客上。如果你受雇于一家公司，你可以要求和你的同事进行一次简短的交谈。

不管怎样，一定要向那些听你说话的人寻求反馈:问问他们喜欢什么，他们认为你在哪些方面可以做得更好。当你说话的时候，试着注意你的听众在做什么。他们笑了吗？人们是在点头还是在乱写笔记？随着你的练习，你很快就会知道你的听众喜欢什么。

如果你的时间表中有时间，你可以加入一些公共演讲团体，如[演讲会](http://www.toastmasters.org/)或[演讲俱乐部协会](http://www.the-asc.org.uk/)，那里的重点是提高你的演讲技巧。你甚至可以从侧面接近它，潜入一个表演团体，比如即兴剧院。自然，你可以从观察中学到很多东西。去听听你最喜欢的演讲者的演讲或下载他的视频，密切关注演讲者的表现。

你对演讲越有信心，你的产品演示或客户会议就越好。更重要的是，如果你选择在你的技术同行中发言，你将扩大你的专业网络，发展宝贵的行业信誉。也许我们会在下一次网络方向会议上看到你的名字！

## 参加培训研讨会

我们身处一个需要不断学习的行业。web 开发永远不会缺少新人，如果你的同事已经向你寻求建议或想法，很可能你已经很擅长解释困难的概念了。因此，也许是时候发展您的培训和研讨会交付技能了。

培养培训技能的一个简单方法是就你熟悉的话题开发并举办一个小型研讨会。无论是使用 Perl、利用社交媒体、操作数码相机，还是从头开始构建 PC，您的社区中都会有人有兴趣了解更多相关知识。调查当地的用户团体、社区大学或成人教育学校等组织——你可以接近他们，向初学者团体提供你的工作室。如果有一个新的爱好是你一直想学习的，你可能会想参加一些自己的课程；除了学习有趣的新技能，你还会发现什么对你有用，其他课程参与者喜欢什么，以及你的培训师如何解释棘手的概念。

网上有很多关于如何举办研讨会的信息，但是和演讲一样，你可以通过询问研讨会参与者的反馈来获得最好的学习效果。仔细关注他们提出的问题或他们遇到的问题，并考虑调整您的研讨会，以纳入这些问题。如果一个参与者问你一个棘手的问题，你甚至可以自己学习一两种新方法！

需要具备培训技能的人来完成一些任务，比如指导新员工、向客户展示如何操作你为他们打造的全新内容管理系统，或者在会议上举办研讨会。如果你发现你已经获得了培训的品味，从苹果到 Zend 的各种供应商都可以证明你是一名培训师。太棒了。

## 打磨那些写作技巧

清晰有效的写作是展示你藏在大脑里的东西的好方法。即使在 140 个字符或更少的时代，播客和 YouTube 总会有坚实的写作技能的一席之地。如果你是一名网站开发人员，那么最有用的写作技巧就是为网站创建内容，或者通过写作与你的客户和同事交流——通常是通过电子邮件。你会在网上找到关于这两个主题的大量信息，包括许多文章和博客文章[就在 SitePoint。如果你通过别人的教导而不是自己的努力学得最好，那么课程是一种便捷的学习方式；快速的谷歌搜索很可能会找到你附近的球场。你也可以从创意写作、专题写作或商务沟通课程中获得有用的技巧。](https://www.sitepoint.com/design-ux/content-strategy/)

很容易想象我们最喜欢的作家可以毫不费力地赶在午饭前写完一些文章，但事实是，伟大的写作只能来自于——你猜对了——练习。这一次，至少在开始的时候，把你的练习留给自己是没问题的。许多作家建议每天写一些粗糙的、零碎的文章，只是为了保持写作的习惯。保留一个笔记本，或者在你的硬盘上藏一些文本文件。每天花些时间写作，想到什么就写什么，即使和你为工作写的完全不同。把你的练习写作当作最粗糙的草稿；当没有对完美的期望时，你反而能够专注于你正在写的东西。

一旦你准备好向全世界展示你新的、提高了的写作能力，就很容易把它展示出来——用电子邮件发给朋友，放在博客上，甚至作为一篇客座博文放在你喜欢阅读的网站上。(无耻但有用的提示:[考虑为我们写一篇文章吧！寻求反馈并仔细阅读:你会从人们的话语中学到很多。我相信 SitePoint 的所有人都同意，我们的读者非常坦率地让我们知道他们的想法，好的和坏的！](https://www.sitepoint.com/write-for-us/)

有技巧的作家会发现，他们的专业知识几乎在所有可能的极客职业中都有需求。Web 应用需要清晰的文档，对开发者和用户都是如此；博客文章应该让读者兴奋，引发对话；新闻稿和营销材料必须向客户传达重要信息。如果写作困扰你，也许你可以考虑自由写作——现在是时候想办法增加你的收入了，即使只是一点额外的现金。

## 继续找，继续学！

如果这些想法都不吸引你，还有很多其他新技能可以探索——也许你可以学习项目管理，发展你的摄影技术，成为营销 2.0 的所有方面的专家，或者找到如何编辑视频。事实上，在纯粹的极客领域之外，有几十种方法可以让你展开翅膀。不管你选择做什么，它肯定会让你成为一个更有价值的雇员、承包商或自由职业者。
