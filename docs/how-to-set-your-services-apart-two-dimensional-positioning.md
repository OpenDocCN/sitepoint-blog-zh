# 如何让你的服务与众不同:二维定位

> 原文：<https://www.sitepoint.com/how-to-set-your-services-apart-two-dimensional-positioning/>

上一篇博客(还记得内陆海鲜餐厅的案例)谈到了一维定位:找到一个对客户重要的标准，并专注于此，让你的业务与众不同。成功的企业通过将劣势转化为优势，建立优势，或者专注于客户重视的、其他竞争对手尚未注意到的一件事情来做到这一点。所以，如果你没有海滨景观，那就关注你的海鲜菜单的质量和卓越吧。

那是一维定位。

现在让我们变得更加复杂。二维定位意味着选择对客户至关重要的两个标准，并确保您的业务在这两方面都是最好的。

为此，绘制一个带有 x 轴和 y 轴的标准图形。每个轴代表一个对客户很重要的标准。你的企业应该出现在右上角，而竞争对手应该在图中不太理想的位置。如果您选择了正确的轴(如客户标准)，您将使您的企业与众不同！

例如，我目前正在使用一个软件客户端。在他们的市场上，至少在他们的解决方案出现之前，你要么花大价钱购买定制软件，要么花小价钱购买不太健壮的现成解决方案。因此，他们决定创建一个既经济又可定制的解决方案。

现在他们的营销宣传资料上有一个二维图表。维度包括成本和定制。他们标出了他们的竞争对手，当然，他们在地图的右上角显示为两个世界中最好的。这是让他们与众不同的好方法。

应该考虑哪些维度？以下是一些想法:

–成本
–专注领域的专业知识
–特定解决方案的知识(例如导航方案、搜索引擎优化、端到端电子商务、基于订阅的网站、内容管理)
–定制
–跟进支持
–绝对保证
–灵活定价(例如按使用量)
–易于部署
–部署时间
–产品领先地位(例如专有技术)
–经验证的投资回报/销售额增长
–奖项
