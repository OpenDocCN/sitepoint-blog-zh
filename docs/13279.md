# 你能有多环保？

> 原文:[https://www.sitepoint.com/how-green-can-you-be/](https://www.sitepoint.com/how-green-can-you-be/)

在网络行业工作，电是我们做任何事情的基石。没有物理替代物，我们工作的任何一部分都离不开能源消耗。在我们自己的电脑上，在我们的办公室和家里，在我们的服务器上，我们把所有的时间都花在推动电子上。但遗憾的是，可再生能源的可用性有限，因此不可避免的是，我们所做的一些或所有事情都涉及不可再生能源的消耗和温室气体的进一步产生，从而导致我们的环境不断遭到破坏。

如果有一种简单的方法来减少我们的碳足迹，而不必改变我们的消费模式，这不是很方便吗..？

嗯。

在过去的几个月里，我越来越对碳抵消计划的声望和尊重感到不安。似乎每个人和他的狗都在购买碳信用额，相信这样做有助于保护环境，减少碳足迹。

并非如此。

当大公司兜售这种东西时，我并不感到惊讶；大公司就像政客——他们说人们想听的话，不管他们是否相信，也不管这是不是真的。但是当本土企业如 [Web Directions Conference 开始宣扬这条路线](http://www.webdirections.org/2007/09/20/is-your-website-greenhouse-friendly/)时，我再也不能保持沉默了。

我不相信碳抵消计划真的有任何帮助。事实上，我认为他们让事情变得更糟。通过购买碳信用额，你只是把钱扔在一个问题上，而不必真正解决它。不要去想我怎样才能减少我制造的碳量从而产生真正的影响，你可以简单地说<q>我正在购买碳信用额，这样我就可以想生产多少就生产多少</q>。净效应很可能是碳排放总量的*增加*。就像<abbr title="European Union's">欧盟的</abbr>排放交易计划，一个旨在减少碳排放但实际上增加了碳排放的想法，因为它允许各国购买和使用本来根本不会排放的碳信用额。

来自<abbr title="United Kingdom">英国</abbr>最近的一篇[卫报文章](http://www.guardian.co.uk/environment/2007/jun/16/climatechange.climatechange):

> …即使是最善意的计划也存在碳抵消理念的基本缺陷——这一理念并非来自试图设计扭转全球变暖的方法的环保主义者和气候科学家，而是来自试图在保持商业现状的同时满足行动要求的政治家和企业高管。它至少在三个要点上失败了。
> 
> 首先，它需要精确测量要抵消的排放量。这被证明充满了不确定性……其次，它需要精确测量其他地方节省的碳。大多数最早的抵消项目都涉及植树，树木会自然吸收碳，这是一个复杂而不可预测的过程，无法进行精确的测量……最后，抵消的想法依赖于所谓的额外性——证明碳减排不会发生在商业生活的自然秩序中。

即使这些计划是 100%准确和可信的，它们仍然是傻瓜的天堂。我们无法抵消我们产生的所有碳，甚至是其中的一小部分。我们只需要减少生产。这并不性感，也不容易，但这是我们必须要做的。

对于工业化的西方国家来说，碳抵消是一个神奇的解决方案，可以让一切照旧。这就是真正的破坏所在——因为只要我们拒绝接受环境破坏的现实，破坏就会变得更糟。碳补偿是一种想象中的商品，它允许我们继续随意污染，然后通过向那些向我们出售破坏原因的人购买赦免来减轻我们的罪恶感。

正如环保活动家乔治·蒙比尔特所说:

> 通过向我们兜售清白的良心，抵消公司正在破坏国内应对气候变化的必要政治斗争。他们在告诉我们，我们不需要成为公民；我们只需要成为更好的消费者。

如果你真的想减少你的网站、公司或家庭的环境足迹，那么可再生能源和更高的效率将会有所帮助。购买碳信用不会。
