# 开发您的设计流程

> 原文:[https://www.sitepoint.com/developing-your-design-process/](https://www.sitepoint.com/developing-your-design-process/)

网络变化很快，在你意识到这是一种趋势之前，一种趋势似乎已经超越了另一种趋势。

新的网页设计师可能犯的最大的错误之一，就是直接跳到 Dreamweaver/Photoshop/Flash/无论什么，在真正思考之前就开始“构建”。所以，如果你刚开始做网页设计，或者你已经有了几个网站，但是还没有开发出一个开发你的网站的过程，这里有一些小技巧可以帮助你走上正轨，信不信由你，这些大部分都可以用笔和纸来完成。

**1。为你的网站设定目标**
你网站的主要目标是什么？你想达到什么目的？如果你想开发一个出售婚礼请柬的网站，你应该包括一个专门的婴儿沐浴卡部分吗？他们有些关系——通常婴儿会跟随婚礼！但是你是不是偏离太多了？一个网站应该有一个明确的主要目标，也许还有一两个相关的目标。

**2。弄清楚你网站的内容**
为了避免混乱，你需要整理出你网站的结构以及需要包含哪些内容。我们都去过混乱的网站，似乎连网站的所有者都不太清楚这个网站是关于什么的。虽然有结构很重要，但是要保持它的灵活性，这样你就可以添加新的内容而不会有太多的麻烦。

**3。线框页面**
线框是你网站的一个骨架轮廓，它显示了页面之间的关系。它给你一个可用性、信息架构、布局和网站内容的概述，可以帮助你看到什么有用，什么没用。有几个专门为线框设计的应用程序，如 [iPlotz](http://iplotz.com/) 和 [OmniGraffle](http://www.omnigroup.com/applications/OmniGraffle/) ，或者您可以使用绘图或布局程序，如 Illustrator 和 Visio。我个人觉得在纸上画线框更容易更快，但许多人觉得线框应用程序非常有价值。在这个阶段彻底是很重要的。这也是你可以尽情发挥创造力的地方。

**4。建造**
在这个阶段，你已经准备好根据你在第 1-3 阶段所做的开始建造。这可能意味着在 Photoshop 等程序中整合更完整的设计，或者开始编写网站代码。你在给你用线框造出的骨头添加肉。

**5。测试**
这是至关重要的，你可以认为这是一个持续的过程。你需要测试可用性和可访问性，以及一些简单的事情，比如检查你所有的链接是否正常工作，或者你的图片是否出现在它们应该出现的地方。你可以先做你自己的测试，然后让一些朋友或家人浏览你的网站，或者问一下(如果你够勇敢的话！)在一个在线论坛上获得一些反馈。如果有问题，人们会很快告诉你。

简而言之，这就是网站典型的设计过程。显然，这是从 10，000 英尺的高度来看的，但是如果你能通过一个尝试和测试的过程来开发你的站点，你会发现你自己一直在进步和加速。
