# 你用的是哪个版本的 PHP？

> 原文：<https://www.sitepoint.com/which-version-of-php-do-you-use/>

[dvduval](https://www.sitepoint.com/forums/member.php?u=17021) 最近在 [SitePoint 论坛](/forums)提出了一个有趣的问题: [*我们是否应该停止为 PHP4 编码？在一个完美的世界里，答案是肯定的，这是大多数人的回答。很少有人会质疑 PHP5 提供了一个更健壮的对象模型，需要更少的代码，更安全，性能更好。它也被大多数网络主机安装，尽管许多主机同时提供 PHP4 和 PHP5。最后，PHP4 的开发和支持已经停止:如果您今天开始应用程序开发，您当然应该使用 PHP5。不幸的是，我们并不是生活在一个完美的世界里。许多应用程序是在 PHP4 是无可争议的平台选择时编写的，将遗留代码更新到 PHP5 可能很困难，或者不能提供商业利益。也可以创建在 PHP4 或 PHP5 上运行的应用程序。WordPress 是一个很好的例子，尽管有计划只切换到 PHP5，开发者意识到它将影响成千上万的安装和插件。这不是一时兴起就能完成的更新。我们想知道您的 web 应用程序使用的是哪个版本的 PHP。虽然我认为很多人会使用 PHP5，但了解是否有人仍在编写只支持 PHP4 的应用程序或创建跨平台代码还是很有用的。请*](https://www.sitepoint.com/forums/showthread.php?t=680452)*[在 SitePoint 投票](/)上投票，并在下面留下评论，让我们知道你是如何做出决定的。*
