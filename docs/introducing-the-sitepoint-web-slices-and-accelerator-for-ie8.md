# 介绍 IE8 的 SitePoint 网页快讯和加速器

> 原文：<https://www.sitepoint.com/introducing-the-sitepoint-web-slices-and-accelerator-for-ie8/>

对于那些使用 Internet Explorer 浏览网页的读者来说(统计数据不会说谎——外面有很多这样的人！)，对于你们当中那些[浏览器一夫多妻者](https://hicksdesign.co.uk/journal/browser-polygamy-movement)，我们很兴奋地宣布一项激励措施，一旦明年 IE8 发布，你们(或者你们公司的 IT 政策制定者)将升级:T2 的站点点网页切片和 T4 的站点点参考加速器。事实上，如果你可以运行 IE8 的测试版，你现在就可以使用它们！

## 什么是网页快讯？

我喜欢把网页切片想象成“类固醇上的书签”。网页快讯是 Internet Explorer 8 中的一项新功能(其 Beta 2 版本目前已经可以下载了)。

网页快讯允许您在方便的下拉窗口中查看页面的子集，该窗口可从收藏夹工具栏访问。与传统的下拉菜单不同，web slice 中的内容可以使用 CSS 进行样式化，并且可以包含图像，因此它实际上是一个 web 页面的“切片”。

## 什么是加速器？

另一方面，加速器是一种自定义的查找工具，用户可以将其添加到 Internet Explorer 的上下文菜单中。我们意识到这项技术非常适合包含在 [SitePoint Reference](https://reference.sitepoint.com/) 中的内容，并构建了一个加速器来搜索 SitePoint HTML 和 CSS 参考站点中用户高亮显示的任何文本。

假设您正在阅读博客上的一些代码，您注意到了一个您不熟悉的元素、属性或特性。您所要做的就是突出显示代码，右键单击以在 SitePoint 参考中查找它，您将立即获得描述、语法、浏览器兼容性等等！

## B … b …但是标准呢，还有所有那些…？

网页快讯和加速器是 IE8 的新功能，所以你需要运行浏览器才能使用它们。但是请注意，它们是使用现有的 web 标准(XML、HTML、CSS 和 JavaScript)构建的。此外，在你的网站上实现网页快讯或加速器不会阻止其他访问者访问你网站上的信息——它们只是一个附加的好处，IE8 访问者可以利用它来消费你的内容。

## 好，让我来对付他们！

如果你已经在运行 IE8(你需要最新的可用版本，Beta 2(T2))，那么我们希望你现在就安装我们的网页快讯和加速器，让我们知道你的想法！

#### 站点点网页快讯

我们已经为[sitepoint.com 主页](https://www.sitepoint.com/)创建了五个网页快讯，并且可能会创建更多(如果您有任何想法或请求，请在评论中告诉我们！).

安装网页快讯很容易。当鼠标指针经过包含网页快讯的网页区域时，会弹出一个绿色图标，指示该网页快讯所代表的网页部分。单击图标，系统会提示您将网页快讯添加到收藏夹工具栏。

将鼠标悬停在我们的[主页](https://www.sitepoint.com/)上您感兴趣的各个部分，即可获取网页快讯；您也可以使用本页底部的[链接直接安装每个网页快讯](https://www.sitepoint.com/advertise/)。

**现场点参考加速器**

安装和使用 SitePoint 参考加速器同样简单。只需访问 SitePoint 加速器的[主页，并点击*安装加速器*链接(同样，这仅适用于 IE8)。](https://reference.sitepoint.com/accelerator)

## IE 的黑仔功能？

网页快讯和加速器技术是对 Internet Explorer 的两个非常酷和受欢迎的补充。虽然 Opera 和 Firefox 等浏览器的铁杆粉丝不太可能被说服，但这些功能可能会成为许多“摇摆选民”在 ie8 发布后至少试驾一下的理由，并可能成为旧版本 IE 用户升级的巨大动力。例如，我们认为 SitePoint 加速器对于 web 开发者来说是一个非常有用的工具。而且是 IE8 才有的(反正现阶段)。

像这样的创新强烈表明微软非常认真地对待其网络浏览器的下一个版本。
