# 白宫发布开源 Drupal 代码

> 原文:[https://www . site point . com/white-house-releases-open-source-Drupal-code/](https://www.sitepoint.com/white-house-releases-open-source-drupal-code/)

今天，白宫宣布他们将向开源社区发布第一套 Drupal 模块。以下是从他们的公告中摘录的:

> 作为我们为 WhiteHouse.gov 开发开放平台的持续努力的一部分，我们将发布一些我们开发的定制代码。任何人都可以查看、使用或修改这些代码。我们很高兴看到世界各地的开发人员如何在他们自己的应用程序中很好地利用我们的工作。
> 
> 通过发布我们的一些代码，我们得到了更多人审查和改进它的好处。事实上，WhiteHouse.gov 的大部分代码已经是开源的，作为 Drupal 项目的一部分。

[![](../Images/ba59178948a967ee06ea70e658edd084.png)](https://www.sitepoint.com/white-house-releases-open-source-drupal-code/) 这是一个重大的转折。他们总共发布了 4 个模块:

*   [Context HTTP Headers](http://drupal.org/project/context_http_headers)–通过告诉缓存服务器保留数据多长时间以及何时提供新内容来实现站点扩展
*   [Akamai](http://drupal.org/project/akamai)–与 Akamai CDN 集成
*   [gov delivery](http://drupal.org/project/govdelivery)–根据用户喜好发送动态电子邮件
*   [Node Embed](http://drupal.org/project/node_embed)–确保我们网站上的所有图片都有适当的元数据，以便屏幕阅读软件能够阅读。

这是一项巨大的努力，我期待白宫团队以及其他政府做更多类似的事情。
