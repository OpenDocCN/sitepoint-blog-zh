# 分享成功的工作资源

> 原文：<https://www.sitepoint.com/share-job-sources-for-success/>

每个行业都有竞争对手。如果你像我一样，你的一些竞争对手会是你的好朋友。我想知道:你会和你友好的竞争对手分享工作来源吗？

我和我的朋友经常分享需要自由职业者的优秀招聘者、客户组织和服务企业的详细信息。当我们没有时间做工作，对项目不感兴趣(可能因为我们试图专注于新的行业或工作类型)，或者因为其他原因需要帮助时，我们经常互相推荐工作和客户。

对于自由职业者来说，分享工作来源和联系方式是一个非常有价值的策略。原因如下。

**1。它鼓励互惠。**

我曾经给过的每个人都给过我工作和客户。如果你主动推荐一个客户给一个友好的竞争对手，你可以为一个相互尊重的稳固关系打下基础，这可能会在未来带来更多的工作和更大的就业机会。

**2。它可以让你摆脱困境。**

我们都知道，在我们超负荷工作的时候，有一个可靠的、合格的企业指导工作是多么有帮助。有一个友好的竞争对手作为你的溢出出口可以是分享自由职业爱情的好方法，同时保留你的好建议和客户关怀的声誉。

**3。它可以帮助竞争对手。**

你可以通过承接竞争对手因任何原因无法处理的项目来帮助他们。无论他们是人手不足还是去度假，你愿意接受他们的客户，知道他们可能会希望在未来继续与该客户的工作关系，这将建立你在竞争对手那里的印象分，并可能鼓励他们在某个时候为你做同样的事情。

**4。它可以鼓励信息共享。**

为一个友好的竞争对手接受一份工作，或者给他们送一份工作，你很可能会发现自己在和你的竞争对手详细讨论客户过去的工作。这种互动可以让你深入了解客户、你的竞争对手使用的技术和工作流程，以及其他有价值的方面。你也许可以利用这些知识来改进你自己的经营，完善你的营销方法，甚至改变你的产品以适应不同的产品或细分市场。

**5。它可以扩大你的客户群、经验和能力。**

当然，分享工作来源联系可以帮助你建立你的业务。它可以大大扩展你所接触的项目和客户的类型。它可以让你体验到你可能不会参与的任务或行业。它可以发展你的技能和投资组合，带来其他推荐，并扩大你的客户群。这能给你带来满足感，因为你知道你已经帮助一个自由职业者获得了同样的优势。
