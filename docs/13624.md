如果你从一开始就考虑 SEO，你将会看到你工作的第一手成果。例如，在这本书出版的时候，我自己的网站是谷歌上关键词排名第一的网站，并且几年来一直保持在谷歌和其他搜索引擎首页的前三名(未过滤)列表中。是的，SEO 是重要的，但更重要的是，它是值得的！让我们开始吧。

##### 使用关键字和关键短语

在创建网站之前，研究关键词和关键短语。T3】

在创建网站时，记住关键词和关键短语，并在开发页面和内容时将它们放在适当的位置，这一点很重要。

***目标关键短语而非单个关键词。*T3】**

单个关键词更常被网络用户输入，这使得它们比多词关键短语更难有效定位。除非单个关键字非常独特，否则使用关键短语可以获得最佳结果。

***使用较长的单词和复数。*T3】**

如果你的目标是你的关键字或短语的复数版本，你会从搜索这些单词的单数和复数版本的人那里得到点击。

使用同义词词典！
*作为你集思广益寻找合适的关键词和关键短语的一部分，尝试使用同义词库来查找相似的单词。网上有一些有用的词库(你可以从 Thesaurus.com 开始)，但是也可以看看搜索词建议工具和 Wordtracker。*

***关注每个网页的几个特定关键词。*T3】**

在搜索引擎中排名靠前的页面倾向于关注特定的关键词，这些关键词通常出现在 HTML `<title>`元素、`<heading>`元素、面包屑导航链接、产品名称和描述以及交叉链接中。(Shari Thurow，五大 SEO 设计错误，ClickZ(2004 年 9 月 27 日)。)

创建关键词丰富的文本内容，尤其是你的主页。T3】

确定潜在访问者会用来找到你的网站的前两三个关键词，然后写 200-250 字的主页文本，利用这些关键词，而不是相反。在为其他网页创建内容时，也要遵循这种方法。如上所述，请记住，一些搜索引擎认为标题比其他内容更重要，所以在优化内容时要考虑到这一点。我们一会儿将讨论为搜索引擎优化标记的主题。

在你正在进行的网站 SEO 维护计划中包括关键词和关键词研究。T3】

##### 在标记中使用关键字和关键短语

搜索引擎使用页面的结构标记作为指南来排列其内容的相对重要性。在你的网页中包括关键词和关键短语是很重要的——尤其是你的主页——并且把它们放在特定的标记元素中。提供与你的目标受众相关的内容自然会有助于你的搜索引擎排名，尤其是如果你有意战略性地使用适当的关键词和关键短语。使用下面的清单来帮助。

在你网站的信息架构中包含关键词和关键短语。T3】

关键词和关键短语的研究结果对你在网站信息架构中使用的单词和短语起着重要的作用。例如，你可以在你的全球和本地导航、你的类别标签、页面`<title>`元素文本、`<heading>`元素文本和内部链接中使用关键词和关键短语。

将这些术语与第 5 章“网站可用性:关注用户”和第 7 章“信息架构”中的清单结合使用，不仅可以帮助你的网站访问者更容易地找到信息，还可以帮助你的网站获得好的搜索引擎排名！

***在每个网页的`<title>`元素中使用聚焦关键词。***

目前，为你的网页`<title>`元素创建关键短语丰富的文本是至关重要的，因为几乎所有的搜索引擎都会给`<title>`元素的文本很大的权重。创建每个页面的`<title>`元素文本，以反映该页面的具体内容，使用人们可能在搜索引擎中输入的关键词来找到你的网站。

***在每个网页的`<meta description>`元素中使用关键词。***

请注意，许多搜索引擎，包括谷歌，雅虎！、Inktomi 等，索引并使用`<meta description>`元素内的文本；例如，有些人用它作为搜索结果中显示的描述文本。然而，在这一点上，他们不使用文本进行搜索引擎排名。而且，他们很大程度上忽略了其他的`<meta>`元素，包括`<meta keyword>`元素。虽然不能保证这将有助于提高你的排名，但在你的网页中包含`<meta description>`元素仍然是值得的，如下所示。(Jill Whalen，元描述标签，高排名(2004 年 10 月更新)。)

```
<head> 

<meta name="Description" content="Add your descriptive sentence or two here." /> 

</head>
```

***在页面的`<heading>`元素中使用你的焦点关键词，从`<h1>`元素开始。***

*保持`<h1>`靠近`<body>`*
*越靠近开始的`<body>`元素标签，就越好。一些搜索引擎会给两者之间的接近度更高的评级——在搜索机器人看来，内容足够重要，足以为自己赢得高层次的标题，并且内容出现在文档的早期。此外，许多搜索引擎也认为越靠近页面顶部的内容越重要，所以请确保前 200-250 个单词包含丰富的关键短语内容。*

***在链接 URL 和相应的`title`属性中使用你关注的关键词。***

这里有一个例子:

```
<a href="/design/checklists/" title="Essential web site 

    Checklists: Best Practices Made Easy">Essential web site 

Checklists</a>
```

关键词和域名虽然一些搜索引擎营销人员认为在域名中放置关键词可以显著提升网站，但其他人认为这并没有多大作用。根据 SEO 专家 Jill Whalen 的说法，域名中的关键词可能会被搜索引擎赋予少量的权重，但只有当其他网站所有者使用其域名而不是网站标题链接到该网站时。结果类似于拥有富含关键字或关键短语的链接，这确实很有帮助。另一位 SEO 专家 Shari Thurow 也认为，其他因素比在域名中包含关键词有更大的影响，如关键词丰富的内容、链接和网站的受欢迎程度。(Jill Whalen，搜索引擎营销 Q & A，高排名顾问，第 091 期(2004 年 3 月 24 日)；Shari Thurow， *搜索引擎可见性*(印第安纳波利斯:新骑手出版社，2002)，20 页。)

***在你的`img`元素`alt`属性中使用相关的关键短语。***

这里有一个例子:

```
<img src="/images/checklists.jpg" alt="Essential web site 

    Checklists: Best Practices Made Easy, my kit with essential, 

    helpful checklists for web site designers and web site 

    owners" />
```

##### 对搜索引擎友好

如果搜索引擎不能访问你的网页来索引它们，伟大的关键词，关键短语，精彩的内容，以及有其他网站链接到你的网站对你的搜索排名没有任何帮助。使用此清单有助于确保您的网站基于搜索爬网程序友好的设计和标记。

***避免使用闪屏页面。*T3】**

启动页面是一个主入口页面，它显示一个大的图形图像或 Flash 动画，通常带有一个进入网站或跳过简介(跳过动画介绍页面)的链接。动画完成后，闪屏页面通常会重定向到新的网页。正如您所料，闪屏页面通常缺乏丰富的关键字或关键短语内容，因为它们除了输入或跳过介绍链接之外，几乎不包含任何可见的正文。给定很少或没有文本内容，搜索爬虫就没有什么可索引的。

通常，splash 页面使用重定向自动将用户带到网站的实际主页。目前，搜索引擎倾向于不索引使用重定向的网站，他们会禁止创建人为重定向的网站，以试图获得更高的排名。(Shari Thurow，五大 SEO 设计错误，ClickZ(2004 年 9 月 27 日)。)

因此，通过使用包含很少或没有文本内容的闪屏页面，并使用重定向，你可能会毁掉你的网站被搜索引擎索引的机会——更不用说排名了。

***平衡 HTML 文本和图形。*T3】**

正如我们在第 3 章“准备网站内容”中所讨论的，专业的、高质量的图形可以帮助提高你的可信度。为搜索引擎爬虫提供大量关键字和关键短语丰富的文本内容也非常重要。没有必要避免使用图形；事实上，避免图形可能会对你的网站搜索引擎的成功不利。相反，规划你的网站设计要使用有目的的图形，在每个`<img>`元素的`<alt>`属性中插入有意义的文本，并提供大量有助于你的搜索引擎排名的关键短语丰富的内容，并使阅读它们的网站访问者受益。

创建搜索引擎友好的 HTML 文本导航和交叉链接。T3】

搜索引擎爬虫跟随 HTML 文本链接。如果你在网站中提供了好的、搜索引擎友好的内部 HTML 文本链接，爬虫很可能会跟随这些链接。如果你的任何网页的网址改变，使用服务器端重定向发送搜索机器人使用旧的链接到新的网址。

而不是让搜索引擎爬虫(或网站访问者！)要找到死链接，创建服务器端重定向，也许可以使用。htacess 和 mod_rewrite 技术，这将自动把搜索引擎带到新的网址。在服务器端，对永久移动的文件使用 301 代码，对临时移动的文件使用 302 代码(假设你的主机允许你访问这样的设置；许多人没有)。

如果您无法使用服务器端重定向，请为每个更改的 URL 创建一个网页，以通知搜索引擎爬网程序和访问者该网页已被临时或永久移动，并提供指向新页面的链接。如果没有新的 URL，那么一定要提供一个链接到你的主页、站点地图和本地搜索，以帮助你的访问者和搜索引擎爬虫(这对于确保爬虫通过这些链接继续索引你的网站尤其重要)。不要在这些页面上使用 JavaScript 重定向！

*使用静态重定向页面的规则
如果你被迫使用静态页面来通知人们和搜索引擎某个页面被移动了，一定要准备一份清单——也许是一份电子表格——你可以在 6 到 12 个月后查阅。这对于搜索引擎爬虫来说应该有足够的时间来索引新的位置，此时你可以检查你的搜索引擎列表，并且如果搜索引擎正在列出新的页面，考虑移除这些旧的保存文件。不过，您可能会考虑将这些静态页面保留更长时间，尤其是供那些将您的移动页面添加为书签的访问者使用。(伊恩·劳埃德，《完美的 404，一份清单》(2004 年 1 月 16 日)。)*

***尽可能使用外部 CSS 和 JavaScript，最大化相关性。*T3】**

搜索引擎会从网页顶部开始读取你的标记，但是他们也会检查你的开始和结束元素之间嵌入的 CSS 和 JavaScript。一般来说，搜索引擎倾向于给出现在网页顶部的文本更多的权重，而不是页面底部的文本。(丹尼·沙利文，搜索引擎放置技巧，搜索引擎观察(2002 年 10 月 14 日)；丹尼·沙利文，隐藏 JavaScript，搜索引擎观察(2003 年 6 月 10 日)(订户专区)。)

***使用结构化标记，尽可能将内容与表现分离。*T3】**

搜索引擎爬虫使用结构化标记来理解你的网页内容是关于什么的，并且它们给予标题元素(`<h1>`、`<h2>`、`<h3>`等)中的文本更多的权重。).

内容填充者将被处罚！不要试图通过将内容填充到标题元素或其他地方来欺骗搜索爬虫，因为他们知道这种欺骗，并且你可能会因为试图人为提高你的排名而被禁止在搜索引擎列表中出现。诚实永远是最好的策略。创建访问者喜欢阅读的关键短语丰富的内容是最好的方法。其他网站也会链接到它，这只会有助于进一步提高你的搜索引擎排名。

提供一个网站地图，从你的主页以及其他所有网页链接到它。T3】

提供一个链接到你的主页的网站地图将有助于搜索引擎找到你的内容，也有助于网站访问者。如果你的站点地图包含超过 100 个链接，逻辑上把它分成不超过 100 个链接的页面。

***如果使用 JavaScript 链接，在其中包含`<a>`元素`<href>`属性链接。***

例如，在我的网站上，我使用 DHTML 来显示/隐藏切换菜单，它包括一个爬虫友好的 JavaScript 链接，带有一个类似于下面这样的

```
<a href="/inc/arch.shtml" onclick="return toggleMenu('arch');  

    return false;" onkeypress="return toggleMenu('arch');  

    return false;">Toggle Show/Hide</a>
```

这里，`<a href="/inc/arch.shtml"`是`<a>`元素`<href>`的属性链接。由于这是一个 HTML 文本链接，搜索引擎爬虫可以读取它并继续索引`/inc/arch.shtml`页面。当我决定采用这种特殊的显示/隐藏方法时，我考虑到了可访问性，但它也有助于搜索引擎爬虫。

JavaScript 和弹出窗口是使用 JavaScript 链接的另一种流行方式。Accessify.com fame 的 Ian Lloyd 为 SitePoint 写了一篇优秀的文章，题目是，完美的弹出窗口。查看一下这个有用的方法，它对可访问性和搜索爬虫都是友好的。

***避免包含？、&、$、=、+和%字符、cgi-bin、会话 id 或 cookies。*T3】**

这些 URL 通常是动态页面上查询字符串的结果。一些搜索引擎爬虫对这样的 URL 很警惕，因为它们可能是无限的链接，会使爬虫超载。因此，它们可能不会索引使用这些 URL 的页面，尤其是当地址中包含三个或更多这样的字符时。正如 MSN 搜索项目经理 Eytan Seidman 所说，“算法开始怀疑它是否会陷入一个循环，无休止地搜索查询参数的每一个排列。因此，具有许多(肯定超过 5 个)查询参数的 URL 被抓取的几率非常低。(埃坦·塞德曼，爬行互联网…，msnsearch 的博客(2004 年 11 月 18 日)。)以下是塞德曼先生所说的那种 URL 的一个例子:

```
http://mybyz.com/products/page.aspx?view=14&tab=6&pcid=24B8-100084  

9@@@@&section=586&origin=mybyzsearch&cookie=false
```

*URL 重写不会根除死循环
虽然你可以使用 URL 重写来消除问题字符并创建更人性化的 URL，但你不会消除更大的潜在问题——确保搜索引擎爬虫不会陷入会使它们过载的死循环是至关重要的。*

***用文字替代 Flash 内容。*T3】**

如果您的网站使用 Flash 来呈现您真正想要或需要搜索引擎爬虫来索引的内容，请提供它们可以索引的文本替换。如果你的整个网站都使用 Flash 而不提供文本替换，就不要指望搜索引擎爬虫会索引你的网站。

***为音频和其他富媒体文件提供元数据和文本替代。*T3】**

典型的元数据信息包括标题、作者、版权和内容描述。专业搜索工具，如 [Singingfish](http://singingfish.com/) ，使用这些元数据帮助用户找到丰富的媒体，包括 MP3 文件。一定要在网页中包含关于富媒体内容的有用文本信息。

##### 避免被搜索引擎禁止

搜索引擎会禁止那些试图人为提高排名的网站，所以不要无意中做任何暗示试图愚弄他们的事情，这一点很重要。使用下面的清单来帮助你避免网站和搜索引擎之间的误解。

***使用正版 SEO 方法。*T3】**

不要想在搜索引擎上耍花招来提高你的排名！你的网站很有可能被他们的搜索结果封杀。

***忌遮掩。*T3】**

伪装是一种人为的技术，恶意用户利用这种技术向搜索引擎发送那些没有发送给网站访问者的内容。(“什么是隐身？”在谷歌网站管理员信息，Google.com(无日期)。)一些网站试图通过提供只有搜索引擎才能看到的特定内容来欺骗搜索引擎。关于“隐形”这个术语涵盖或不涵盖什么，网络上有很多争论；例如，有人质疑使用 CSS 向网站访问者隐藏内容，或者使用 JavaScript 嗅探向特定搜索引擎提供与网站访问者不同的内容，是否是伪装。

***避免使用门道页面或用于门道的域名。*T3】**

门户页面，或用于门户的域名，是填充了关键短语并提交给搜索引擎的网页或域名。

***避免把关键词塞进评论标签或者`<img>`元素`<alt>`属性。***

注释标签中的内容没有任何好处，并且 image `<alt>`属性应该只作为图像的文本替换。

***避免使用 JavaScript 或`<meta-refresh>`重定向来欺骗搜索引擎。***

通常，搜索引擎不会索引使用 JavaScript 或`<meta-refresh>`重定向的页面，因为从历史上看，很多网站用它们来欺骗搜索引擎。所以，即使你想使用这些技巧而不考虑欺骗，也要避免这样做！否则，你的网站可能会被搜索引擎误禁。一些人认为这些技术属于伪装的定义，但是不管你是否同意这个观点，最好避免使用 JavaScript 或`<meta-refresh>`重定向。

##### 上市

让您的网站在搜索引擎、其他网站和目录上列出是一个重要的目标，但更重要的是确定这样做的正确时间，您应该向哪些引擎提交您的网站，以及您应该提交哪些页面。做一些仔细的研究和准备将有助于使提交过程更加顺利和高效，并有助于你在长期内取得更好的结果。使用此清单作为您的指南。

***提交您的网站搜索目录后，只有初步搜索引擎优化完成，内容已经定稿。*T3】**

只有在你的网站是活的并且已经过断链测试之后，才提交你的网站来搜索目录。T3】

单独研究每个目录或搜索引擎及其类别，阅读每个目录或引擎的常见问题，并在提交您的网站之前准确地遵循指示。T3】

在提交过程中很容易犯的错误在以后很难改变，并且可能会让你付出昂贵的代价。这些错误会极大地影响搜索引擎爬虫找到您的站点的可能性。

***在提交之前，研究每个搜索目录的最佳描述。*T3】**

在每个目录中搜索你最重要的关键词。研究结果和描述的类别，并根据那些在特定目录中已经被接受的网站来设计你的描述风格。此外，如果目录中没有适合你的业务类型的类别，可以考虑推荐一个与目录中已有的类别相似的类别。

***确保你独特的销售描述也包含在你的目录描述中。*T3】**

即使您的描述应该类似于该特定目录所接受的其他网站的描述风格，也要确保您的描述是独一无二的。例如，包括你提供的具体服务或产品，并提及你的目标受众。

***写几个长短不一的描述，复制/粘贴到提交表单中。*T3】**

每个目录对网站描述的字数都有不同的要求。在一个纯文本文件中编写几个描述，您可以根据需要将其复制/粘贴到提交表单中。包括 7、10、15、20、25、30 和 50 个单词的描述。

确保你最重要的关键词出现在每一个描述中，但避免关键词填充。T3】

确保你的网站最重要的关键词出现在每个描述中，但是不要塞进太多的关键词！目录编辑不喜欢那些基本上读起来像一列关键词的描述。

考虑付费或加急提交作为一种让你的网站更快上市的方式。T3】

你越早将你的网站列入搜索目录，你的网站就能越早从这些搜索引擎获得推荐。使用付费或加速提交是一种方法，有助于加快你的网站上市过程，有时可以使等待时间短至一两天。值得注意的是，付费加入并不能保证高排名。一定要做足功课，一定要阅读高质量的 SEO/SEM 网站和有用的书籍，比如前面提到的丹·提斯的《搜索引擎营销工具包》。(另见丹尼·沙利文，购买你的方式:搜索引擎广告图表，搜索引擎观察(2004 年 11 月 22 日)。)

***先提交自己的主页搜索目录，再提交其他网页。*T3】**

一旦你的主页被列出，提交其他页面，尤其是那些关于专业主题的页面。T3】

一旦你的网站主页被列在那个目录中，你将有更好的机会让你的其他网页，尤其是那些关于专业主题的网页，被不同的类别所接受。

***手动提交你的网站搜索目录，而不是使用自动提交软件。*T3】**

包括谷歌在内的搜索引擎爬虫建议不要使用自动提交软件。(谷歌信息网站管理员谷歌(2005 年)。)

*自动化软件违反了一些服务条款
请注意，一些自动化软件违反了许多搜索引擎和目录的服务条款，所以在您考虑使用任何自动化软件进行搜索提交之前，为了检查链接流行度或检查排名，请阅读每个搜索引擎和目录的服务条款，以确保您没有违反规则。*

***提交你的网站搜索目录如雅虎！、DMOZ、乔安特和金普西。*T3】**

带有主要目录的列表对于搜索引擎(如 Google)的爬虫成功找到你的网站至关重要。

***避免过度提交。*T3】**

检查你的网站是否已经在搜索引擎上列出，如果已经列出就不要再提交了。跟踪并保存好你的提交、接受、列表和排名的记录是很重要的。

***使用一个`robots.txt`文件和`<meta>`元素`<robots>`属性来表示您不想索引的内容。***

您可能有网页、图像、声音文件或其他不想编入索引的内容。如果是这样，您可以在保存到服务器根目录的`robots.txt`文件中指定忽略这些文件。这里有一个`robots.txt`文件的例子，它指示 Google 不要索引图像，并告诉所有搜索爬虫不要索引 MyByz.com 网站上的程序目录:

```
# robots.txt for http://mybyz.com/   

# exclude images from Google   

# see http://www.google.com/remove.html#images   

User-agent: Googlebot-Image   

Disallow: /images   

# exclude specific areas from all search engine crawlers   

# (in this case, a directory containing procedural files   

User-agent: *   

Disallow: /procedures
```

您还可以在特定页面上包含`<meta>`元素`<robots>`属性。例如，如果您有一个不希望被索引的页面，但是您确实希望蜘蛛跟踪该页面上的链接到其他页面，那么您可以将以下指令添加到页面的标记中:

```
<meta name="robots" content="noindex,follow" />
```

*忽略`<robots>`属性和`robots.txt`文件
注意，并不是所有的搜索爬虫都会尊重`<meta>`元素`<robots>`属性或`robots.txt`文件，但是很多会(包括最常用的引擎 Google)，所以如果需要的话，这些技术绝对值得使用。*

##### 创建一个持续的链接活动

所有主要的搜索引擎都非常重视网站整体链接的受欢迎程度。链接到你网站的高排名网站可以帮助提升你自己网站的排名。使用下面的清单来帮助您计划和管理您的网站的传入和传出链接。

***提供持续的、值得链接的内容。*T3】**

如果你在你的网站上提供高质量的内容，其他网站会链接到你的网站，通常你都不会问。

***提供大量你推荐的网站的出站链接。*T3】**

请其他网站所有者考虑链接到您的网站或交换链接。

*不要滥用论坛
避免仅仅为了创建网站链接而在论坛上发帖。论坛所有者会看穿你的策略，你可能会失去所有的信誉。这种策略也可能导致其他网站所有者避免合法链接到您的网站。如果你真的想为你的网站建立链接，那就创建一些别人想链接的好内容，不要犹豫，给其他网站所有者发邮件，问问他们是否愿意链接到你的网站。*

如果您的网站上的任何 URL 发生变化，请务必通知那些链接到它的人。T3】

让这些其他网站和网站所有者了解信息将有助于他们以及您的网站找到并链接到您当前的实时网页，而不是维护旧的、死的链接。记住，如上所述，使用服务器端重定向到新的 URL 是很重要的。

当谈到为搜索引擎优化你的网站时，没有一个单一的、神奇的解决方案。获得高排名需要时间和耐心，但你可以通过遵循 SEO 行业专家的建议，使用上面的清单，并坚持不懈地优化和推广你的网站来提高你的搜索引擎排名。不要使用人工技术。

##### 规划持续维护

每四到五周跟进搜索引擎和目录，以确保你的网站被列入。

如果你在四到五周内没有发现你的网站被列在某个特定的目录中，继续重新提交。T3】

*跟踪您的提交内容
跟踪您向每个目录提交内容的日期、您提交的类别的详细信息以及您使用的描述和标题内容非常重要。*

如果你在三次提交后没有发现你的网站列在目录中，请联系目录代表。T3】

定期检查您的服务器日志，了解访问者如何找到您的网站，并相应地调整您的网站。T3】

你可能需要做一些调整来提高你的排名。你也可以用新的关键字和关键词来写新的内容，以支持或改变你的搜索排名。

***添加关于主页新内容的通知和链接，以帮助网站访问者和搜索爬虫找到它。*T3】**

当你添加新的内容或对你的网站做其他改变时，给你的 SEO 策略添加新的关键字和关键词。T3】

如果你已经优化了你的网站，搜索爬虫会找到你的新内容，特别是如果你从你的主页链接到那个内容，如上所述。

考虑使用分析软件来检查关键词、搜索引擎排名和列表状态。T3】

Web 分析软件，如 Web Position Gold，可以帮助跟踪顶级入口页面、顶级推荐网站、顶级推荐 URL、顶级搜索引擎、顶级搜索关键字和关键短语。不过，你不一定要为昂贵的统计软件付费:免费服务，如 [Extreme Stats](http://www.extreme-dm.com/tracking/) ，提供了很多这种信息，尽管在使用这些服务时，你必须在你的网站上显示一个统计数据的图形链接，让所有的网站访问者都可以使用。

***利用关键词分析的结果来增加访问者所寻求的网站内容。*T3】**

在你优化网站并提交给搜索目录后，至少三个月内不要改变你网站的 SEO 策略。T3】

你的网站提交开始出现在搜索引擎和目录中需要时间——通常需要两到三个月甚至更长时间，尤其是如果你有一个新的网站。(Jill Whalen，Robots Meta Tag-Changing Domain Names，High Rankings Advisor 第 137 期(2005 年 4 月 20 日)。)尽量耐心点！持续的高排名需要时间。

避免为了提高搜索引擎排名而不断调整你的网站。T3】

正如我上面提到的，你调整过的页面出现在搜索引擎上需要时间，所以你需要等着看你的页面做得怎么样。在你给你的排名足够的时间过滤后，如果你决定你需要调整任何页面，在做更多的改变之前，等待看看这些调整会产生什么影响。时间和耐心是 SEO 成功的重要组成部分。

定期在搜索引擎上检查你的链接受欢迎程度。T3】

许多网站会在你不知道的情况下链接到你的网站，除了检查你的推荐日志，使用搜索引擎查找链接到你的网站也是一种简单的方法。这些信息可以帮助你确定和跟踪你的互惠链接运动的功效。

你可以在每个搜索引擎中输入特殊的查询来检查链接到你的网站。例如，在谷歌的搜索框中，输入:

```
link:www.domainname.com
```

在 AltaVista 的搜索框中，键入:

```
link:domainname.com
```

请记住，只有在特定搜索引擎中列出的网站才会出现在该搜索引擎的查询结果中。

***根据行业的不同，考虑创建一个关键词和关键短语的日历，以备将来参考。*T3】**

在一些行业中，给定关键词的流行度会在一年中上下波动。从事这类行业的网站所有者可能会发现，保持一个日历，列出一年中不同时间流行的关键字和关键短语是值得的。
