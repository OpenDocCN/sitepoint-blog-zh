# Firefox portable–没有笔记本电脑的移动办公室

> 原文：<https://www.sitepoint.com/firefoxportable-your-mobile-office-without-a-laptop/>

SitePoint 为越来越多经营基于互联网的小型企业(通常是单人经营)的人提供服务。鉴于不断的业务需求，个体经营者总是发现很难找到休假的时间。

互联网业务永不停机的特性只会加剧这一点——您的用户在许多不同的时区运营，当竞争激烈时，客户支持期望很高。

幸运的是，另一面是，你的互联网业务大体上可以在任何地方运营。这让你可以随时随地自由旅行——只要你能及时处理日常运营问题。

乍一想，这意味着你要带着笔记本电脑。我在去泰国之前仔细考虑过这个问题，我就是在那里写这篇文章的。但是把一台笔记本电脑和附件拖到一个炎热潮湿的地方，在那里它很容易被损坏/被风沙侵蚀/丢失，这个想法真的没有吸引力。

由于这是一次短暂的旅行，我检查了在我离开时可能需要做的事情的必要性。我当然计划不承担任何开发工作，而是负责客户支持、销售和任何不希望出现的紧急技术问题。

这些功能可以归结为网页浏览、电子邮件和 SSH 访问。所有这些都可以在网吧里进行，几乎在任何地方。

我对很多事情都很偏执——安全是其中之一——我一直不愿意在公共电脑上输入重要的密码，因为这些电脑可能会感染很多病毒和其他恶意软件。

此外，在公共电脑上，一个人的工作效率会大大降低，因为您无法访问存储的密码、表单和搜索历史记录，很可能还有您的非公共书签。此外，你很难找到一款能满足你超级生产力需求的浏览器。

解决方案非常简单:一个 u 盘和一份 FirefoxPortable 的拷贝。我的安装只消耗了 30MB，包含了我所有存储的密码、书签、工具栏设置——一切都与我的常规桌面环境完全一样。将这些设置移植到便携安装中就像复制 Firefox 的 profile 文件夹一样简单。

存储的密码受到 Mozilla 的主密码设备的保护，虽然当你开始你的会话时需要输入它，但对于任何可能在你之前使用 PC 的邪恶的键盘记录者来说，它没有什么价值。

但真正有价值的是通过使用*您的*浏览器所获得的生产力提升——完成一些工作并尽快回到海滩。

PortableApps 网站提供了一整套可移植安装的开源应用。可以想象，在他们的帮助下，我可以在大约 90MB 的空间内设置我的整个 LAMP 开发堆栈，并旋转 PHP 代码俯瞰[长江](http://en.wikipedia.org/wiki/Yangtze_River)的两岸。但我毕竟是在这里度假！

那些没有可靠的基于网络的电子邮件界面的人会热衷于他们的便携式雷鸟包。

最后，对于 SSH 访问， [PuTTY](http://www.chiark.greenend.org.uk/~sgtatham/putty/download.html) 一直是一个可移植的解决方案，只包含一个*。exe* 文件。带上你的私钥，这样你就可以在需要执行紧急系统管理操作时避免输入任何密码。但是要注意的是，在世界各地的许多网吧，等待时间可能是一种痛苦。

总之，如果你要去不想带笔记本电脑的地方旅行(谁真的想带着它去任何地方？)，给自己弄个 u 盘，在上面随意点。
