# PHP Conf Lon Live

> 原文:[https://www.sitepoint.com/php-conf-lon-live-1/](https://www.sitepoint.com/php-conf-lon-live-1/)

所以昨晚我回到了英国，会见了 PHP 伦敦小组，作为今天会议的热身。

**PHP UG London**

我能说的是，如果你在伦敦，却没有去参加 PHP 伦敦会议，那你就错过了。除了是一个有趣的社交网站——PHP 用户群往往比你的 Linux 用户群吸引“更高层次”的书呆子(就像普通人一样……差不多)——这里也有一些非常活跃的家伙，在一个昂贵的城市靠 PHP 和 web 开发谋生。出席人数各不相同，但通常情况下，10 名常客会在一个好的夜晚出现，最多可达 30 名(比如拉斯莫斯来访时)。不得不提一下 [PHP 厨房](http://www.phpkitchen.com)的德米安·特纳，他是我长久以来一直想见的人。

他们也以这样或那样的方式与大多数发生在伦敦的网络“场景”保持联系——所以即使你不是 PHP 开发人员，如果你从事一般的网络开发，这也是值得联系的。马库斯告诉我，他们已经开始(出乎意料地)吸引猎头形式的追随者，他们迫切需要 PHP 开发人员——这在伦敦显然是很大的需求

进一步尊重会议的组织。

**PHP 会议开始**

总之——“直播”博客。会议开始。超过 160 名访客，这对于第一次意义重大的英国 PHP 大会来说是惊人的。
