# 十大前端开发框架:第 2 部分

> 原文:[https://www . site point . com/top-10-front-end-development-frameworks-part-2/](https://www.sitepoint.com/top-10-front-end-development-frameworks-part-2/)

在本文的第一部分，我们已经看到了一些与 Bootstrap 相关的不错的框架。如果你厌倦了这些自举，深呼吸，准备做一些真正不同的事情。先说 Bootstrap 的主要竞争对手——Foundation。

## 5.基础

[Foundation](http://foundation.zurb.com/) 是一个功能强大、功能丰富、响应迅速的前端框架。使用 Foundation，您可以快速原型化和构建可在任何类型的设备上工作的网站或应用程序，包括大量的布局结构、元素和最佳实践。它以移动优先为理念，利用语义特性，并使用 Zepto 代替 jQuery，以便带来更好的用户体验和更快的性能。

Foundation 有一个 12 列的灵活、可嵌套的网格，足够强大，可以让您快速创建多设备布局。就功能而言，它提供了许多。版式、按钮、表单和各种导航控件都有样式。提供了许多有用的 CSS 组件，如面板、价格表、进度条、表格、缩略图和 flex 视频，可以在任何设备上正确缩放您的视频。当然，还有 JavaScript 插件，包括 dropdowns、joyride(一个简单而容易的网站游览)、magellan(一个粘性导航，指示你在页面上的位置)、orbit(一个具有触摸支持的响应图像滑块)、reveal(用于创建模态对话框或弹出窗口)、sections(一个传统手风琴和选项卡的强大替代品)和工具提示。

该框架还提供了许多有用的附加组件。

*   stencils——Foundation 中的所有 UI 元素都以 Omnigraffle stencils 和 vector PDFs 的形式提供，用于更快、更轻松地构建线框和模型。
*   HTML 模板——方便、现成的布局，让您可以快速开始和构建。要使用它们，你只需抓取代码并将其放在页面的标签之间。
*   图标字体–这些是存储在方便的网络字体中的象形图标的自定义图标集。
*   SVG 社交图标–一组独立于分辨率的社交图标。
*   响应式表格 Foundation 响应式表格的机制是获取第一列并将其“固定”在表格的左侧，从而允许您滚动其下的其他列。
*   画布外布局–这些布局允许您隐藏移动设备上的内容或导航，直到更大的屏幕尺寸允许其可见或用户采取行动将其暴露。当最后一种情况发生时，内容或导航滑动到视图中。

如你所见，Foundation 就像是 web 开发人员/设计人员的宝库。为了得到你所需要的，你可以使用定制器来构建你的定制下载。

## 6.地面工程

[ground workss](http://groundwork.sidereel.com/)是前端框架家族的新成员。这是一个完全响应的 HTML5、CSS 和 JavaScript 工具包，利用 Sass 和 Compass 的强大功能构建，使您能够快速原型化和构建几乎可以在任何设备上工作的网站和应用程序。

它提供了一个极其灵活、可嵌套、基于分数的流体网格系统，使得创建任何布局成为可能。GroundworkCSS 有一些真正有表现力的功能，如平板电脑和移动网格，当视窗宽度低于 768 或 480 像素时，它们会保持网格列结构，而不是将网格列折叠成单独的行。另一个很酷的特性是 jQuery ResponsiveText 插件，它允许您动态调整文本大小，以适应视窗的宽度:对于可伸缩的标题和构建响应表非常有用。

该框架包括一组丰富的 UI 组件，如选项卡、响应数据表、按钮、表单、响应导航控件、磁贴(单选按钮和其他无聊的标准表单元素的漂亮替代物)、工具提示、模态、Cycle2(一个强大的响应内容滑块)以及许多更有用的元素和助手。它还提供了一套很好的矢量社交图标和一整套包含在 FontAwesome 中的象形图标。

要查看运行中的框架，您可以使用浏览器窗口顶部中心的 resizer。这样，您可以在探索框架功能的同时，针对不同的大小和视口测试组件的响应性。

GroundworkCSS 有很多很好的例子，为了让你快速入门，这个框架还提供了几个响应模板。我认为唯一的缺点是缺少定制下载的方法。

## 软糖

Gumby 是一个简单、灵活、健壮的前端框架，使用 Sass 和 Compass 构建。

它的固定布局可以针对桌面和移动分辨率自我优化内容。它支持多种类型的网格，包括嵌套网格，具有不同的列变化。Gumby 有两个 PSD 模板，可以帮助您在 12 和 16 柱网格系统上开始设计。

该框架提供了功能丰富的 UI 工具包，包括按钮、表单、移动导航、标签、跳转链接、切换和开关、抽屉、响应图像、视网膜图像等等。遵循最新的设计趋势，UI 元素有 Metro 风格的平面设计，但你也可以使用漂亮的风格和渐变设计，或者根据你的意愿混合两种风格。一组反应灵敏、独立于分辨率的 Entypo 图标完全集成到 Gumby 框架中，供您在 web 项目中使用。

Gumby 也有一个非常好的定制颜色选择器，可以帮助您轻松地建立您的自定义下载。

## 8.HTML 快速启动

HTML KickStart 是一个 HTML5、CSS 和 jQuery 驱动的工具包，可以轻松创建任何类型的布局。它提供了干净的、符合标准的、跨浏览器的标记。

该框架具有网格、版式、表单、按钮、表格或列表的样式，以及跨浏览器的 web 元素，如 JavaScript 幻灯片、选项卡、面包屑导航、带子菜单的菜单、工具提示等等。

你可以使用 [99Lime UIKIT](http://www.99lime.com/uikit/) ，它拥有 HTML KickStart 的所有 UI 元素，可以在你的线框中使用。

## 9.象牙

IVORY 是一个轻量级、简单而强大的框架，可以处理从 320 像素到 1200 像素宽度的响应性布局。它包含了 12 列的流体宽度网格、字体样式和一些基本的 UI 组件，如按钮、切换开关、折叠、标签、工具提示等等。

当您需要一个简单灵活的多设备解决方案，并且您的设计不需要其他框架提供的额外功能时，IVORY 是一个完美的选择。

## 10\. Kube

最后，如果你的新项目需要一个坚实而简单的基础，没有不必要的复杂性和额外的东西， [Kube](http://imperavi.com/kube/) 可能是正确的选择。Kube 是一个最小的、响应性的和适应性的框架，没有强加的风格，给你创造的自由。它为网格、表单、排版、表格、按钮、导航和其他类似链接或图像的东西提供了基本的样式。

该框架包含一个用于轻松构建响应式布局的紧凑 CSS 文件和两个用于在设计中实现选项卡和按钮的 JS 文件。如果你正在寻找最大的灵活性和定制，你可以下载开发者版本，它包括更少的文件，变量，混合和模块。
