# 教育客户:质量与数量

> 原文：<https://www.sitepoint.com/educating-the-client-quality-versus-quantity/>

大多数为客户开展搜索引擎优化活动的人都熟悉这种情况:你的客户不顾一切地想让某个关键词或短语排名第一，并坚持不惜任何代价让更多的流量流向他们的网站。

公平地说，在过去十年中，几乎整个 SEO 行业都是围绕这两个不合格的指标建立的。不幸的是，他们鼓励黑幕运营商进入市场，并“保证”排名第一或大规模的流量增加，玷污了合法，明智的搜索引擎优化从业者的名字。

这给职业 SEO 带来了一个难题。我们是否倾听客户的意见，保持沉默，尽最大努力满足他们的要求？或者，我们是否有责任教育我们尊贵的客户 SEO 的真正价值在哪里？

让我们看一个例子。你的客户要求搜索术语“蓝色小部件”排名第一，他们的业务就是围绕这个词建立的。当然，任何关键词排名靠前都是好事，这就是为什么它们有时被称为“英雄关键词”但它能带来真正的价值吗？

解决这个问题将取决于你客户的业务和目标。就产生点击量而言，第一名的价值是第二名的两倍，几乎是第五名的五倍。但是数量并不总是意味着质量。你的客户需要合格的流量，带来真正对他们的内容感兴趣的访问者。当这种情况发生时，网站的商业目标是以最低的可能跳出率实现的；也就是说，人们到达一个网站后立即离开的速度。

多年来，搜索引擎已经花费了数百万美元来提高搜索结果算法的相关性。他们的目标是确保搜索者将登陆与他们的搜索查询最匹配的页面。当用户从一个登陆页面跳转时，搜索引擎认为这是一个失败。当搜索者必须精炼或重新措辞他们的查询时，这也是一个失败。

那么这意味着什么呢？这意味着你有两个选择。

你可以优化你客户的网站，尽可能广泛地撒网，吸引尽可能多的访问者，并确信巨大的访问量会转化为切实可行的销售。或者你可以采取有针对性的方法——忽略英雄条款，转而追逐长尾——优化更长的合格条款，吸引更有可能转化为销售的网站访问者。

这可以归结为一个选择，更多的访问者，但更少的转换，或更少的访问者，但更大的转换量。数量或质量流量。你必须为正确的路线选择正确的马。

以下是一些需要考虑的因素:

*   该网站是否有强大的现有品牌？如果是这样，瞄准品牌的英雄条款可能是合适的。点击率将受到品牌认知度的影响，因此您可以利用高流量，将访问者引向现场转化页面。
*   这是一个电子商务业务吗？这就是瞄准长尾的真正回报所在。产品名称的精确匹配总是有一个较低的跳出率，通常有较高的直接到页面流量的转化率。一个额外的好处是，如果需要用点击付费来补充，按产品名称的流量是非常便宜的。
*   这个网站是商业性质的吗？并不是所有的网站都是这样，所以他们可能会用站内广告来支付费用；在这种情况下，成交量比转化率更重要。接触大量受众最有可能有助于增加收入。
*   公司是否有实体存在？创建在线业务以推动客流量的实体企业，将需要与纯在线电子商务网站不同的战略。

清楚地了解你的客户的商业目标是至关重要的，这将决定你如何接近和解释他们网站的最佳 SEO 策略。任何现有的网站历史和流量表现将是有益的，尽管许多网站将缺乏适当的统计或跟踪到位。

找出你的客户为什么开始一个搜索引擎优化策略(或改变它)的一个好方法是问，“什么对你更重要:在谷歌上排名第一还是做一笔买卖？”这个看似简单的问题的答案会让你深入了解是什么驱动了他们，以及他们对搜索引擎优化的理解。

通过了解你的客户的期望，并确定需要什么来满足他们的网站目标，你将能够制定出一个 SEO 策略，提供所需的流量。
