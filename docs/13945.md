# 把好的和伟大的分开

> 原文：<https://www.sitepoint.com/separating-the-good-from-the-great/>

我正坐在家里的办公室里，等待承包商来完成给办公室添置橱柜的工作。我先付了 50%。承包商刚刚给我开了接下来 40%的账单。对他们来说，这是有意义的，因为 90%的材料都是安装好的。但对我来说，它们还没有完成 90%。需要添加门。微小的细节需要注意。对我来说，它们要么是 0%完成，要么是 100%完成。现在，他们处于 0%,因为他们没有完成。

而且，基于他们目前的工作，我担心他们能否快速有效地完成最后的 10%。所以我不想付钱给他们，因为我不想放弃我唯一的筹码——钱。他们一完成最后 10%的工作，我就付给他们钱。

这就说到点子上了:优秀的专业人士专注于工作的前 80-90%。伟大的人会非常小心地让最后的 10-20%变得杰出。从客户的角度来看，最后的 10-20%是结果。是最后的 10-20%决定了你是言出必行，还是马马虎虎。而正是最后的 10-20%有时需要你付出 80-90%的努力！

想在营销和交付方面脱颖而出吗？优雅而出色地完成项目的最后 10-20%。注意最后的细节。收拾残局。调试你的工作。让客户试驾你的作品，提出最终要求。

这可能是一种痛苦，但会把你从单纯的好中分离出来。
