# 雅各布·尼尔森访谈

> 原文：<https://www.sitepoint.com/interview-jakob-nielsen/>

**马特·密茨凯维奇采访了 Jakob Nielsen，他是新书[《优先考虑网站可用性](http://www.amazon.com/gp/product/0321350316/)》的作者，谈到了 AJAX、可用性与关键词广告的紧密联系，以及我们今天在网络上看到的一些广告形式。**

**如何看待当前 AJAX
(Google Maps，Writely，Google Suggest，Zimbra 等)的实现？)今天在网上？**

它们与绝大多数商业网站无关。(我所说的“商业网站”也指政府机构和非营利组织的网站。)

一个商业网站从编写更好的标题中获得的利润要比在页面上粘贴编程技巧多得多。拿地图来说:是的，它是商店查找器和定位器的可用性指南之一，用来显示你所在位置的地图。但是不要把它做成一个高度互动的地图——最好使用手绘地图，这种地图已经过优化，可以给你的地址指明方向。它显示地标、推荐的停车场和公交车站，并强调高速公路，但不包括人们在开车到您的大楼时可能不会使用的小街。

同样，当我们测试大量的[投资者关系网站](http://www.nngroup.com/reports/ir/)时，我们发现绘制股票趋势和财务数字的高级工具只会让大多数个人投资者感到困惑。一个更好的选择是在一个静态图中显示最重要的信息，这个静态图已经被一个优秀的设计师优化过了。(是的，在我们的测试中，机构投资者确实使用了先进的可视化工具，但他们是在他们的彭博终端上使用的。在一家公司自己的 IR 网站上，他们在寻找 CEO 对公司发展方向的看法。)

诚然，谷歌建议是一个整洁的黑客。我认为类似的想法在外联网上也是有用的，当用户填写的表单字段只有几千个可能的条目时，比如产品名称或货物的目的地地址。提前输入可以节省大量重复使用的网站按键。

可用性的[定义](http://www.useit.com/alertbox/20030825.html)指出可用性是五个质量属性的组合，包括易学性和使用效率。你必须根据你设计的用户界面的类型来平衡这些不同的品质。

重要的是要记住，大多数网站不会重复使用。通常，用户只会访问给定的页面一次。这意味着任何给定操作的效率都要让位于特征的可发现性和可学习性。因此，像拖放这样的交互技术几乎不应该在网站上使用。相反，重点展示一些重要的功能，并通过简单的链接或按钮点击即可访问它们。

一些重复使用的商业网站包括类似软件应用程序的功能。我想到了网上银行，我可以很容易地想象出这样一种设计，它使用户能够通过账户结单页面上的 AJAX 技术看到支票的正面或背面，而不是转到新的页面。

“AJAX”这个名字的乏味之处给了我希望，它将被用于比现在的 Flash 更有价值的事业。由它自己的名字注定，Flash 有类似的潜力，但由于“华而不实”的设计而被严重滥用，以至于它从未成功地为 Web 添加许多有用的功能。

由于 AJAX 是一个工程概念，它有一个相反的问题，即导致*到*的不成熟。请记住:仅仅因为您喜欢技术和高级功能，并不意味着您的客户也喜欢。他们只是想进出而不用担心你的网站。所以在功能上要放松。

在你的书《优先考虑网站可用性》中，你说未来关键词广告的增长将与可用性紧密相关。为什么

这是因为更好的可用性意味着网站更高的转化率。你转化的访客越多，你就越有能力支付吸引访客的费用。这再次意味着，当你提高网站的可用性时，你会抬高关键词拍卖的价格。假设某个商业领域至少有十家公司都在改进他们的站点，最终结果将是更高的关键词价格。

**所以提高可用性和提高转化率是紧密相关的？**

原则上，两者是不同的。可用性是转化率的关键影响因素之一，提高可用性通常会使过去可用性低的网站的转化率翻倍。但是还有许多其他因素影响转化率，包括品牌的声誉，产品本身，显然还有价格。降低价格，你几乎总能提高转化率——但当然你可能会赔钱，所以最好投资于其他因素。

我给你举一个我自己公司的例子:我们出售一份关于电子邮件简讯可用性指南的报告。上一版是 293 页。新版有 544 页，因为我们加入了最近一项眼球追踪研究中关于人们如何阅读电子邮件的发现。尽管价格从 298 美元涨到了 398 美元，但新版比旧版卖得好得多。

我不认为销售量的增加是因为每页的价格降低了。我认为这是交付更符合客户需求的内容的结果。发布电子邮件简讯的公司没有太多地方可以寻求可用性建议，因此他们对非常详细和彻底的研究更感兴趣。

在这个例子中，尽管价格更高，网站也没有变化，我们还是获得了更高的产品转换率。产品本身是销售额增加的原因。

当然，告诉公司制造客户想要的产品并不是特别革命性的建议，但这是值得的建议，因为这是增加销售的第一方法。重要的是要记住，在你花钱开发产品之前，用户研究是找出你的客户想要什么的最好方法之一。用户研究不应该仅仅用于网页设计；它也应该用来确定你的产品策略。

考虑到价格、产品、品牌和可用性都会影响转化率，这就变成了一个将资源投入到哪里的问题。我当然不会说可用性应该得到所有的钱，但是大多数公司在这个因素上投资不足。

想一想开展一场大型广告活动有多贵:你很容易就能花掉数百万美元。相比之下，一个简单的可用性测试花费 25000 美元——如果你自己做的话会更少。

也要考虑是否值得浪费金钱来增加一个糟糕的网站的流量，用户看一眼就离开了。除非你已经做了功课，并确保体面的可用性，你不应该投资任何促销活动。

许多搜索营销人员正在大力投资 A/B 分割测试，以最大限度地提高付费搜索引擎流量的转化率。当谈到增加转化率和最大化可用性时，你认为 A/B 分割测试的恰当角色是什么？它能代替现场测试吗？

A/B 测试永远无法取代现场用户测试，原因很简单，它不能让你对*为什么*用户做他们做的事情有关键的了解。你需要“为什么”,以便产生设计见解，为项目的下一步设定方向。

A/B 测试的目标非常狭窄，即决定两个设计中哪一个产生了最多的期望动作，只要这些动作是可以测量的。这意味着:

*   您需要设计并实现这两种方案
*   结果可以用计算机来计算

在很多情况下，一个网站最糟糕的问题并不是你在 A/B 测试中要考虑的问题，因为除非你进行了可用性测试，否则你不会意识到这些问题。当你把你的研究局限于你自己能产生的想法时，你已经对用户做的所有意想不到的事情关闭了你的头脑。在可用性测试中，你在计划过程中没有考虑到的问题通常会产生最大的发现。这就是为什么让用户完成任务，然后坐下来观察他们的行为是如此重要。

当一家公司最终准备投资促销时，你对如今在网络上如此流行的联合广告形式(如 Google AdSense)的可用性有什么看法？

我们的眼球追踪研究显示，用户确实会看一点这些纯文本广告。可能是因为联合广告看起来与人们习惯看到的搜索引擎广告非常相似。用户已经发现搜索引擎广告倾向于呈现相当有用的信息，所以用户经常在 SERP 上查看广告。

可悲的是，对用户来说，联合广告远不如搜索引擎广告有用，这就是为什么它们的点击率往往低得多。搜索和浏览内容网站在用户意图上有很大的不同。在搜索引擎上，用户被提示去别的地方，而广告可以显示几乎正是用户正在寻找的东西。相比之下，内容网站上的广告可能*看起来*与搜索广告一样，但它不提供相同的专业化水平，因此它没有相同的效果。

广告中一个更新的趋势是上下文文本链接。在这种格式中，某些关键字与特定的赞助商相关联并被热链接到特定的赞助商。普通用户对这些广告的看法如何，广告商是否应该购买这种新的形式？

“恶心”是这些广告唯一合适的词。

他们正在破坏网络用户体验中最核心的元素，即对蓝色下划线代表什么的理解。提示:它不代表广告。幸运的是，这些广告很少出现在网站上，所以它们还没有让用户感到困惑。但是如果广告变得突出，用户会变得更加不愿意点击链接，这又会大大降低整个网站的可用性。

当访问者最终到达一个网页时，你会发现浏览内容页面(42%)或 SERP (47%)的用户数量与浏览主页(23%)的用户数量相差甚远。你认为这是为什么？

在我们在书中描述的研究中，主页的滚动比其他页面少得多，原因有两个。

首先，许多网站在将最重要的功能或最大的新闻放在页面顶部方面做得很好。因此，用户通常不需要*来滚动。*

第二，用户不想使用主页。他们出现在网站上是有原因的，这个原因并不包括找到网站上所有的产品。人们离开主页的速度非常快，因为他们想进入解决他们当前关注的网站部分。内容页面和 SERPs 被滚动得更多，因为它们倾向于关注特定于用户的问题，而不是提供二十种不同事物的一般概述。

在你今天的网络可用性痛苦等级中，网站搜索现在是最坏的罪犯。对于一个工作过度、负担过重的 Web 开发人员来说，想要提高自己网站的搜索可用性，最简单的解决方案是什么？

最简单的解决方案是订阅外包搜索，将问题委托给其他人。我不一定认为这是最好的解决方案，但它肯定比安装高端搜索软件，然后调整它以适应您的环境更容易。如果你不定制搜索软件，你最终会得到一个悲惨的搜索，在这种情况下，最好使用专门从事网站搜索的公司的优化解决方案。
