# 撕裂史蒂夫·乔布斯

> 原文：<https://www.sitepoint.com/rip-steve-jobs/>

SitePoint network 的每个人都向史蒂夫·乔布斯的家人、朋友、同事和崇拜者致以最深切的哀悼，乔布斯在与癌症抗争多年后于 2011 年 10 月 5 日去世。

这位苹果公司的联合创始人将永远被人们铭记为一位真正的梦想家，他改变了人们思考和使用电脑和互联网的方式。

我们 SitePoint 分享他去世的悲伤，我们庆祝他为我们的文化做出的巨大贡献。
