# 客户是怎么找到你的？

> 原文:[https://www.sitepoint.com/how-do-clients-find-you/](https://www.sitepoint.com/how-do-clients-find-you/)

当你把市场剥离到最本质的时候，你会发现两件事情正在发生:

1.  一个卖家正在寻找买家
2.  一个买家正在寻找一个卖家

这两者的区别被描述为外向营销与内向营销，两者都有其固有的优势和劣势。我上周在[讲了第一个如何找到客户](https://www.sitepoint.com/how-to-find-clients/ "How to Find Clients")。但营销等式的另一半是:客户如何找到你？

# **种植一些作物的时间**

猎人式的创造潜在客户是你寻找客户的时候；然而，农业让客户很容易找到你。两者都是成功的基本要素。耕种需要更长的时间，但长期利益是巨大的，不应该被忽视。

种田不是“探矿”。这是建立关系。它基于这样一个理念，即在其他条件相同的情况下，人们更喜欢与他们喜欢和信任的人做生意。建立关系让双方都有机会喜欢和信任对方。

这意味着你必须付出比得到更多，这样当你的人际网络中的人在寻找像你一样的人，或者认识其他人时，你会是第一个想到的人。

这不是操纵战术。如果你不是真心喜欢和乐于帮助别人，那就别费心了。

# 在土壤肥沃的地方种植

当谈到被发现时，我们大多数人都会迅速转向博客和社交媒体。很舒服，很熟悉。但是很难穿过这些杂乱的东西并引起注意。

这也是本末倒置。一旦你开始与你的目标受众进行对话，博客和社交媒体的出现可以加强你的可信度和专业知识。毕竟，现在只有一小部分人愿意和你做生意；因此，一个博客和一条推特是一个很好的继续向其他人营销的 B 计划。

但是让大量的人“喜欢”或“关注”你需要[一个广告或促销渠道](http://www.smallbusinessmarketingsucks.com/confessions-twitter-newbie/ "Confessions of a Twitter Newbie")。“推广渠道”可以是披萨店门前的一个标志——或者，对你我来说，是我们建立关系的组织和团体。

# **种田意味着弄脏双手**

在网络世界的无菌环境中与人交流，不知何故感觉安全可靠。但如果你希望收获真正的前景，你需要把手弄脏。

我在 SitePoint 和 Twitter 上与很多优秀的人建立了联系，否则我永远也不会遇到他们。但是我想让你们所有人都在一个房间里，面对面地与你们见面，能够在非虚拟的世界里与你们交谈和分享。这是持久联系发生的地方。

有很多方法可以在网下与人联系，但最常见的是加入一个网络或线索共享小组。就像社交媒体一样，这个想法不是硬卖，而是建立关系。你可能六个月都找不到这个团队的客户。没关系。庄稼不会一夜之间从地里长出来。如果你花了足够多的时间从事猎人式的潜在客户创造，你应该会得到足够多的业务让你继续下去。随着时间的推移，你可以花更少的时间去打猎，更多的时间去种田。

就像航天飞机必须首先到达轨道，然后重力才能最终接管一样，也许有一天你可以宣称“我所有的生意都来自于口碑”
