# 联合国加入开源

> 原文：<https://www.sitepoint.com/united-nations-joins-open-source/>

联合国开始将其强大的能力投入到开源社区中，最初是通过其在亚太地区运作的国际开源网络(T3)进入 T2。

该部门自称为自由和开放源码软件的卓越中心，提供教育材料，与来自开放源码领域的演讲者(包括自由软件基金会的理查德·斯托尔曼)举行公开会议，并在该地区倡导开放源码技术。

这个使命被明确地提出来，作为一个缩小数字鸿沟的愿景，在预算紧张的情况下，Linux 和开源解决方案是一个经济的选择。

在过去的十年中，开源已经证明了大大小小的操作都可以部署复杂的平台，而不需要专有软件许可的巨大成本。这些努力在 Sun Microsystems 发布开源 Solaris 中得到了反映。在今天宣布开放 Solaris 的电话会议上，Sun 公司的代表(包括首席执行官斯科特·麦克尼利和刚从苹果公司回来的浪子汤姆·高根)表示，推出免费开源操作系统的部分动机是为了在软件许可成本会抑制开发和创新的发展中国家提供企业级功能。

看到联合国为这一活动提供部分资金，这当然令人着迷，也许我们将看到这一努力的种子开始在全球范围内的集团业务中萌芽。
