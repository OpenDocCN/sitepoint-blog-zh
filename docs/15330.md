# 论坛脚本综述

> 原文:[https://www.sitepoint.com/forum-script-round-up/](https://www.sitepoint.com/forum-script-round-up/)

所以你决定在你的网站上添加一个留言板，但是你不知道从哪里开始。如果你在 CGIResources 这样的网站上搜索这样的脚本，你会被返回的大量选择所淹没。

以下是目前最受关注的五个脚本的候选列表——用它们来为你的网站添加社区。

##### 1.vBulletin

[http://www.vbulletin.com](http://www.vbulletin.com)

这是一个崭露头角的、功能丰富的脚本。它使用 PHP 和 MySQL 来驱动快速后端引擎。该脚本目前有两种版本，一种是精简版，虽然免费，但几乎去除了所有功能，另一种是完整版，价格约为 160 美元。vBulletin 提供了一个非常可扩展和可定制的系统来启动您的社区。它有非常可定制的界面，确实有一个学习曲线，虽然你可以按你想要的方式设置它，但总体来说这是一个很棒的脚本。

韦恩的评分: **5 颗星**

##### 2.终极公告牌

[http://www.infopop.com](http://www.infopop.com)

这个软件创造了基于网络的 BBSes 的爆炸。现在到了版本 6，它仍然有能力做一个好的强大的社区网站。UBB 使用 PERL 作为其引擎，并将其数据存储为平面文件中的专有文件格式。这使您能够在几乎所有可用的服务器上安装该软件。唯一的缺点是 UBB 的价格，因为它可以相当昂贵的小网站。

韦恩的评分: **4 星**

##### 3.WWWThreads

[http://www.wwwthreads.com](http://www.wwwthreads.com)

这个软件有 PERL 和 PHP 两个版本，可以安装在任何地方。它使用 MySQL 存储数据，这导致了一个快速的系统。这个系统的一大优点是，主题可以以类似 UBB 或 vBulletin 的平面风格查看，或者以类似 WWWBoard(基于网络的 BBS 的鼻祖)的线程风格查看。WWWThreads 还提供了皮肤或主题，您可以将它们应用到您的板上，以便快速启动和运行。这与其他附加组件一起，为您提供了一个完整的社区构建工具包。对我来说，系统在较重的负载下似乎有点慢，但总的来说它表现不错。

韦恩的评分: **4 星**

##### 4.IkonBoard

[http://www.ikonboard.com](http://www.ikonboard.com)

Ikonboard 看起来是一个很有前途的系统。目前最新的版本(3.0)正在测试中，但是很多想法和计划已经进入了重写阶段。有了这个版本，你可以将你的数据存储到一个专有的数据库格式，或者使用 MySQL 作为你的后台。在 vBulletin 的带领下，Ikonboard 增加了许多简单消息之外的新功能。虽然这个软件对我的个人品味来说还有点粗糙，但它确实有很多追随者，而且会越来越好。3.0 正式发布后我还得再看看。

韦恩的分数: **3.5 星**
3.0 的发布会让它获得完整的 5 星吗？我们只能拭目以待。

##### 5.PHPBB

[http://www.phpbb.com](http://www.phpbb.com)

这应该是最受关注的基于 PHP 的免费公告板系统了。然而，我发现它非常粗糙，缺少一些在今天的环境中需要的关键功能(例如，安全的电子邮件地址)。在这里列出的所有系统中，它也是最没有吸引力的默认风格。PHPBB 的一大优势是它的开源特性，然而，在成为领导者之前，它还有很长的路要走。

韦恩的评分: **2 星**

##### 围捕

在我看来，vBulletin 是一个市场领导者，它提供了客户想要和需要的新功能。如果你今天就要购买软件来创建你的社区，我不得不推荐这款产品——vBulletin 的定制、特性和功能可以满足任何一个网站管理员。UBB 仍然是领导者，但如果它想保持目前的市场份额，就必须更多地倾听客户的声音。

如果你一定要有一个线程视图，在我看来今天只有一个选择，那就是 WWWThreads。而且，在我看过的两个免费系统中，我不得不推荐 Ikonboard。它有能力和潜力成为市场领导者。
