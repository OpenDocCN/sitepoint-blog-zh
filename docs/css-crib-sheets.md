# CSS 婴儿床床单

> 原文：<https://www.sitepoint.com/css-crib-sheets/>

CSS 并不难，它只是有一个与 HTML 成 90 度的学习曲线——至少如果你在嵌套表格单元格和间隔 gif 以及强大的字体标签统治一切的时代学习 HTML 的话。感谢上帝有 CSS 婴儿床床单——CSS 提示的集合，一旦你知道了它们就很简单，但否则会给你带来数小时的痛苦。

戴夫·谢伊的[婴儿床表](http://www.mezzoblue.com/css/cribsheet)提供了大量的 CSS 提示，甚至被翻译成了其他 3 种语言。安迪·巴德采取了一种不同的方法，一次发布一个有深度解释的提示。如果你曾经遇到过 IE5 中垂直导航元素之间的[间隙](http://www.andybudd.com/blog/archives/000128.html)、[清除浮动](http://www.andybudd.com/blog/archives/000129.html)(也被 Eric Meyer 覆盖的[)或](http://www.complexspiral.com/publications/containing-floats/)[居中 div](http://www.andybudd.com/blog/archives/000150.html) 的问题，Andy 就是你要找的人。

既然我们谈到了 CSS 技巧这个主题，这似乎是一个为[CSS-discuse Wiki](http://css-discuss.incutio.com/)插入插件的好地方。我[在 2002 年 10 月建立了 Wiki](http://simon.incutio.com/archive/2002/10/11/cssWikiLive),基本上让它做自己的事情。由于一个庞大而有才华的贡献者群体的努力，维基一直在稳步发展，现在是各种各样优秀信息的家园。我最喜欢的页面可能是这个对[三栏布局技术](http://css-discuss.incutio.com/?page=ThreeColumnLayouts)的无价总结，但是你可以很容易地花几个小时浏览其他内容。最重要的是，如果你认为有什么遗漏的地方，你可以自己直接添加进去。
