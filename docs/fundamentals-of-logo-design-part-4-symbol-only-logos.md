# 标志设计基础第 4 部分:纯符号标志

> 原文：<https://www.sitepoint.com/fundamentals-of-logo-design-part-4-symbol-only-logos/>

到目前为止，我们已经了解了[印刷标识](https://www.sitepoint.com/fundamentals-of-logo-design-part-2-typographic-logos/)和[类型和符号标识](https://www.sitepoint.com/fundamentals-of-logo-design-part-3-type-symbol-logos/)。今天，轮到那些大胆到不用伴随字体的标志了。第三类标志设计是“纯符号”标志。这些是标志设计界的麦当娜或布兰妮——如此著名以至于他们不需要姓氏，或者就标志而言，如此容易识别以至于他们不需要附上公司名称。

这种类型的标志通常是为真正的大男孩保留的，当公司如此之大，以至于它可以单独通过一个符号来识别时，通常会经过重新设计。当你第一次发布时，使用一个只有符号的标志作为你品牌的一部分可能不是一个好主意。如果公众知道公司的名字，那会有很大的帮助。符号 logo 的主要思想或优点是“一图胜千言。”一个好的设计加上重复的力量可以产生巨大的影响。符号和图标是强大的。它们能迅速传递大量信息。

象征性标志使用图像来传达一个企业的真实或抽象的表现。没有任何文字来解释这个标志，这个标志可以对公司的含义进行广泛的解释，这当然可能是一件好事也可能是一件坏事。耐克的“swoosh”标志可能是最著名的纯符号标志之一。它在衣服和运动器材上一眼就能认出来，并拥有一个坚实的追随者部落。我想知道有多少人会买完全一样的 产品，如果上面没有 swoosh 字样的话？

[![nike_logo.standard](img/a0729757684f3b0b1b9a77409e64bd26.png "nike_logo.standard")](https://www.sitepoint.com/wp-content/uploads/2010/04/nike_logo.standard.jpg)

许多符号不一定是主题的直接说明或“这就是我们要做的”，而是一个代表性的想法。苹果不种苹果。第一个苹果标志描绘了艾萨克·牛顿坐在树下，头上悬着一个苹果。外面边框上的短语是这样写的:“牛顿……一个永远独自在思想的陌生海洋中遨游的心灵。”苹果的标志现在看起来非常不同，但这是对最初想法的回归。

[![original_apple_logo](img/210881e52375cb37b49fb1ca4d61c08a.png "original_apple_logo")](https://www.sitepoint.com/wp-content/uploads/2010/04/original_apple_logo.gif)
