# 10 年过去了:SitePoint 的联合创始人泄露了秘密

> 原文:[https://www.sitepoint.com/sitepoint-cofounder-interview/](https://www.sitepoint.com/sitepoint-cofounder-interview/)

马克·哈博特尔是 sitepoint.com 的联合创始人，也是最近推出的[99designs.com](http://99designs.com)的创始人。在 Webmaster-Resources.com(现为 sitepoint.com)成立 10 周年之际，克里斯·怀特(Chris White)与马克一起回忆过去，谈论未来。

首先，感谢您花时间回答这些问题。再过几个月，sitepoint.com 就 10 岁了。感觉如何？

你这么说，让我觉得自己老了！不，说真的，当我想起过去的 10 年，我为我们所取得的成就感到骄傲。我们已经看到这个领域的许多网站来来去去，更大的竞争对手半途而废。但是，通过相信我们所做的事情，专注于质量，并强烈关注客户，我们不仅能够保持业务，而且能够上升到顶端。

自从你这么多年前创办 SitePoint 以来，网络发生了怎样的变化？

网络的基本要素并没有改变太多——我们仍然用 HTML 编写网页，我们仍然用浏览器浏览网页，我们仍然用文本驱动的搜索引擎查找资料。当然，所有这些事情都有所改善，但在我看来，真正的不同是网络给世界带来的变化。十年前，我的父母不理解我是为了一份工作而工作。现在他们在问哪里可以下载最新的电影。在相对较短的时间内，互联网已经成为我们日常生活的一个主要部分，当你想到这一点时，你会发现这是非常了不起的。

你认为在塑造网络未来的过程中，什么样的趋势是最重要的？

如果你谈论的是技术问题，我认为未来几年塑造 web 的趋势之一是 Web 应用程序基础设施作为按使用付费服务的兴起，这是我们在 SitePoint 和 99designs 中大规模采用的东西。亚马逊的虚拟化平台和谷歌新的 AppEngine 平台可以让开发者更快更好地编写 web 应用。编写一个快速、可伸缩和复杂的 web 应用程序从未如此简单或便宜。初创公司和其他创新者对这些平台的采用不仅是为了构建廉价的智能应用，也让我们看到了未来几年随着更多提供商的出现而可能出现的情况。

在创立 SitePoint 后的哪个阶段，你认为“好吧，现在这是一项成功的业务”？

我一直相信，如果你能从日常的业务运作中抽身出来，并且没有你它也能继续繁荣，你知道你有一个成功的企业。几年前我就遇到过这种情况，但我不得不说，我很幸运，我们在 SitePoint 有一个了不起的团队——这些人在工作上比我做得好得多。

随着最近 99designs.com 的推出和你在业务中的新角色，你下一个大的步骤是什么？

我的工作是努力在我们与 99designs.com 已经取得的初步成功的基础上再接再厉。我认为 99designs.com 首先是一个设计师社区——一个让他们推广作品、提高技能、与同行交流、赢得新客户的中心市场。虽然我们已经有超过 10，000 名注册设计师，但我们还有很长的路要走。目前，我们专注于建立更好地保护设计师的机制——我们正在做的最重要的事情之一是处理付款以及客户和设计师之间设计工作的安全转移。我们预计这将很快完成。

招募优秀员工的秘诀是什么？

归根结底，如果你不喜欢你所做的事情，你就不能指望做得很好，尤其是从长远来看。因此，在招聘员工时，我最想知道的是他们对什么有热情——他们在业余时间喜欢做什么。

如果我们正在寻找一个创造性的角色，我可能会发现候选人在业余时间喜欢摄影或绘画；如果是一个网络管理员的角色，可能是候选人家里有 UNIX 网络设置；如果我们雇佣编辑，他们可能会有个人博客，或者写诗来寻求刺激。

激情可以以不同的形式表现出来，但我们在 SitePoint 拥有一支高质量的团队且人员流动率很低的最大原因是，我们深入了解真正驱动我们员工的因素，并确保他们不仅适合这些角色，更重要的是，我们也适合他们。

说到激情，你在业余时间都做些什么来娱乐？

问得好，克里斯，但我不是在找工作…

事实上，这些天我没有很多空闲时间。我有两个小宝宝，第三个也快出生了。我把大部分业余时间花在换电池、建塔、玩捉迷藏和回答奇怪的问题上，比如我在周末回答了我 3 岁孩子的一个问题:“爸爸，我可以玩一些危险的东西吗？”

那么，你如何在你的商业承诺和年轻家庭之间周旋呢？

我的想法是，如果我不能在工作时间完成与业务相关的工作，那么我要么工作得不够快，要么我在看起来并不重要的事情上花了太多时间。我发现你在生活中承担的越多，你似乎就越能分清主次。有趣的是，如果你忽视了那些在当时看起来很重要的事情，它们往往会消失。

在您接受的上一次《社区报》采访中，您提到了一架公司飞机。你能告诉我们更多的信息吗？

克里斯，你真的做足了功课！不幸的是，我们的“派对飞机”的建造因为马特·密茨凯维奇和我关于飞机内部的小争执而陷入停顿。你看，马特已经要求一个长的加州特大号床，但我的卧室只能放一个大号。昨天我们的总经理告诉我，我可以在我的房间里有任何我想要的床；马特想要什么样的床都可以，所以我想我们快解决这个问题了。

对未来的网络开发者有什么临别赠言吗？

不要创业，太辛苦了！来为 SitePoint 或 99designs 工作吧——我们有足球桌、吉他英雄 III 和浓缩咖啡机。
