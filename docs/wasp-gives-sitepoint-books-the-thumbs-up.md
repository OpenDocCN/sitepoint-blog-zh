# WaSP 给 SitePoint Books 竖起了大拇指

> 原文：<https://www.sitepoint.com/wasp-gives-sitepoint-books-the-thumbs-up/>

网络标准项目的新[街道团队](http://streetteam.webstandards.org/goodbooks/)正在开展一场[游击营销活动](http://streetteam.webstandards.org/2008/03/06/street-team-make-your-mark/)来警告零售客户关于过时的书籍或者教授不被认为是最佳实践的技术。

这是该组织发起的第一个“街头团队”倡议，他们是通过热情而机智的战斗号召来实现的:

> 你的任务，如果你决定接受它，将是追踪和识别危险过时的网络资源，并揭露他们是误导的骗子。

书签将读者引导到 WaSP 网站上的一个页面，该页面包含推荐的最新书籍列表。显然，我们非常高兴地看到[超过 20%的书籍都是 SitePoint 标题](http://streetteam.webstandards.org/goodbooks/):

*   使用 HTML & CSS 以正确的方式建立自己的网站
*   HTML 乌托邦:使用 CSS 设计无表格，第二版
*   [《CSS 选集》第二版](https://www.sitepoint.com/books/cssant2)
*   [简单的 JavaScript](https://www.sitepoint.com/books/javascript1/)
*   [JavaScript 选集](https://www.sitepoint.com/books/jsant1/)
*   [JavaScript 的艺术&科学](https://www.sitepoint.com/books/jsdesign1)

WaSP street 团队有一个书签的 [PDF 可供下载(PDF，3.4MB)](http://streetteam.webstandards.org/files/goodbooks.pdf) ，所以你可以把它打印出来，然后小心翼翼地放在你在当地书店发现的任何过时的网页设计书籍中。
