## 搜索平台

雅虎！的平台方法现在可以分为两大类:搜索和社交网络。搜索平台的第一部分于 4 月在三藩市的 Web 2.0 博览会上宣布，几周后[在](http://www.ysearchblog.com/archives/000583.html)开放。叫做[搜索猴子](http://developer.yahoo.com/searchmonkey/)，雅虎这是雅虎“重新布线”的第一步。包括开放其最神圣的部分之一:搜索结果。

SearchMonkey 允许第三方开发者使用额外的上下文信息来增强搜索结果。SearchMonkey 对整个 web 都很重要，因为它鼓励使用微格式和其他自下而上的语义 Web 方法——因为结构化数据使创建和启动 SearchMonkey 应用程序变得更容易。然而，这不太可能改变雅虎的游戏规则。在搜索市场。就像微软的搜索现金返还计划一样，它感觉不足以成为谷歌的差异化优势，从而对搜索市场造成很大冲击。

今天，雅虎！公布了他们搜索平台战略的第二部分: [BOSS](http://www.ysearchblog.com/archives/000599.html) ，意思是“建立你自己的搜索服务”简单来说，BOSS 就是一个允许访问雅虎的 API！s 指数。换句话说，老板让我们垂直或利基搜索引擎开发商采取雅虎！并应用他们自己的排序算法。这对垂直搜索提供商来说是一个巨大的胜利，因为建立一个新的搜索引擎的最大成本和技术障碍之一是建立和维护一个索引——这个障碍刚刚被消除。

最初的发射伙伴包括[米迪亚](http://me.dium.com/)、[哈基亚](http://www.hakia.com/)、[克鲁兹](http://www.cluuz.com/)和[日生活](http://www.daylife.com/page/yahoosearchondaylife)。

## 行得通吗？

虽然我谈到了雅虎！就搜索和社交这两个独立的部分而言，这两者确实有着千丝万缕的联系。“雅虎！”的平台应该是一个巨大的内部重新布线工作，将整个地方彻底改变，使其更加开放，更加社会化。这是一个激进的举动，但雅虎。真的需要让它浮在水面上。

这肯定是悬而未决的一群利基雅虎！基于 BOSS 的 s 可以和 Google 竞争。话说回来，雅虎！没有透露货币化将如何在老板建造的财产上工作，只是说它将被允许。如果雅虎！允许开发者随心所欲地将搜索结果货币化，那么更多的垂直市场最终会成为谷歌的赢家。鉴于两家公司最近达成的搜索广告协议，谷歌可能会在未来将 BOSS served 搜索结果货币化。

然而，显然雅虎！凭借自己的搜索排名和显示算法，谷歌无法与谷歌竞争，因此开放其索引并让其他人试一试是一个明智之举。如果有人触及任何革命性的东西，以至于用户开始选择它而不是谷歌，雅虎！将处于收购的最佳位置，因为它已经使用了他们的索引和后端技术。

同样还有待观察的是 MyYahoo！可以从它目前的《晨报》感觉过渡到雅虎的社交中心。如此迫切地希望如此。我的雅虎！尽管[以每月超过 4000 万的用户数领先于所有起始页](http://www.techcrunch.com/2008/07/07/my-yahoo-rolls-out-new-interface-tries-to-fend-off-igoogle/)，但作为大多数用户的社交世界中心，它也远远落后于脸书(和 MySpace)。
