# 网络走向本地——通过网络建立区域性业务

> 原文:[https://www.sitepoint.com/regional-business-over-net/](https://www.sitepoint.com/regional-business-over-net/)

你的公司是本地的还是地区性的？你有没有考虑过建立一个网站，但认为这可能是浪费时间和金钱？或者你已经有了一个本地企业的网站，但是它并没有像你想象的那样吸引顾客。放松——你*可以*利用互联网发展你的本地或地区业务。

让我们面对现实吧，有许多企业根本无法服务于国际市场，例如:

*   豪华轿车服务
*   景观美化企业
*   本地送货服务
*   本地娱乐服务
*   餐馆和酒吧
*   美发美容沙龙
*   家政服务
*   夫妻店
*   独立加油站
*   汽车服务中心
*   当地承包商
*   电脑维修/升级服务

这些类型企业的所有者没有意识到的是，他们中的每一个人都可以利用互联网来增加利润——即使没有网站。以下是如何…

##### 忘记网络——进入:互联网！

假设你提供豪华轿车服务，你要么正在考虑建立一个网站，要么你已经有了一个。但是由于你只服务于当地社区，你的网站在销售潜力上受到地理位置的限制。所以暂时忘掉网站吧！这里有一个更好的通过互联网招揽生意的方法…

发展一个强大的客户“电子名单”,包括你曾经得到的每个客户的名字和电子邮件地址。当你的业务遭遇每一个低迷时期(预期的或意外的)时，花一个小时给这些老客户发一封私人邮件和一张特别的优惠券。

这项活动几乎是免费的，你的客户会喜欢你的特别关注。你是怎么做到的？简单！

***第一步:了解你的客户***

每次你得到一个新客户，确保在他们填写的文件上有他们的电子邮件地址。每天，让你办公室的人把这些信息输入你的数据库。

如果你还没有一个客户数据库，今天就在你的办公室电脑上创建一个。有许多方法可以做到这一点:

1.  Some software solutions for small businesses include database software and contact management software, such as…
    – [Access](http://microsoft.com/office/access/default.htm)
    – [Goldmine 5.0](http://www.frontrange.com/)
    – [Act!](http://www.act.com)
2.  If you’re completely non-technical, or don’t have the money for a software solution or a programmer, you can start a basic customer database yourself. Simply open Notepad on your computer, and type the following headings…

    <q>**先生/女士然后按标签
    **名字**然后按标签
    **姓氏**然后按标签
    **电子邮件地址**然后按标签**</q>

     **您将会看到这样的内容……
    先生/女士名字姓氏电子邮件地址

    将光标向下移动到下一行。

    现在输入你的第一个客户的称呼(先生或夫人)，然后按 tab 键。接下来键入他们的名字，然后按 tab 键。然后是他们的姓，然后是标签。最后是他们的电子邮件地址，然后按 tab 键。

    就这样:你有了一个基本数据库的开始！每次更新时保存此文件。在未来的几年里，它将成为你业务中的一个强有力的工具。** 

 *****第二步:准备联系你的客户***

如果你正在使用前面提到的软件解决方案，或者类似的程序，你已经有了内置的邮件合并功能。你可以跳到第三步。

然而，如果你用记事本处理基本的客户数据库，下一步是得到一个电子邮件合并程序。一些易于使用的程序包括:

*   [世界合并](http://www.bizweb2000.com/wm)
*   [Tuglet 基于网络的解决方案](http://www.tugnut.com/)

这些程序允许你打开你的制表符分隔的客户文件，它们将把任何字段直接合并到你的信中。

***第三步:撰写并发送您的客户信函***

现在剩下的就是写一封很棒的信并用电子邮件发送出去。

坚持上面的例子，这封信保证会提高你的豪华轿车服务的底线。请随意借用，并根据您的业务进行修改。一些小的改变可以使它适应任何当地的商业。

请注意，括号中的字段是数据库字段，您的电子邮件合并程序将使用您客户的详细信息填充这些字段。
