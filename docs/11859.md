# 考虑网站重新设计？让你集中注意力的 5 个技巧

> 原文:[https://www.sitepoint.com/site-redesign-5-tips/](https://www.sitepoint.com/site-redesign-5-tips/)

在某个阶段，大多数设计师或开发人员会想彻底改造他们现有的网站。可能你的网站看起来还可以，但是维护起来很痛苦。也许你有大量的内容，但是你的访问者需要一个导航博士来找到他们在网站周围的路。或者，你的网站看起来就像穿着一双袜套和一件蝙蝠翼羊毛衫，也就是说，完全过时了。

如果你正在考虑重新设计你现有的网站，问自己一些问题，包括:

*   网站做了我最初想让它做的事情了吗？如果是的话，我还想让它这么做吗？

*   网站的受众发生变化了吗？

*   该网站是否通过了 W3.org 网站的可访问性和可用性检查？

*   我还喜欢这个造型吗？

*   我需要一个新的标志吗？

*   我那 3 页起步的小网站已经变成庞然大物了吗？我需要帮助来维护它吗？

这些问题的答案可以帮助你确定你的网站是否需要一个小的更新或一个完整的检修。如果你几年前设计和开发了你的网站，你可能没有机会(或现金)使用内容管理系统(CMS)建立你的网站。几年前，CMS 被认为是一个专业的工具包，但今天，由于开源应用程序，任何人都可以拥有一个。

在规划重新设计时，请记住以下五点。

**1。简单=好。**
无论是功能还是设计，一个干净简单的网站很难出错。

**2。了解你的受众。**
在你现有的网站上使用谷歌分析等应用程序，看看你的访问者来自哪里，他们使用什么浏览器，他们去了你网站的什么地方。这可以帮助你弄清楚他们到底想要什么。

**3。你是谁？**
关于你自己，你会告诉你的访客什么？如果访问者不能很快明白你的网站是关于什么的，你很有可能会失去他们。

**4。获得并保持可用。**
尝试一些不同的东西是可以的，但是不要创建一个如此古怪而无法使用的网站来迷惑或惹恼用户。

**5。新潮不一定好。**
现在看起来很酷的设计，很快就会过时。好的设计是永恒的，不依赖于最新的字体来支撑。
