# 我比你更受欢迎！

> 原文:[https://www.sitepoint.com/im-more-popular-than-you/](https://www.sitepoint.com/im-more-popular-than-you/)

想象一下，你和一群人在一个晚宴上，有人插话让你知道他们比你更受欢迎。你的反应会是什么？你会有什么感觉？

最近，我看到像非常受欢迎的 [Klout](http://www.klout.com/) 这样的“社交评分”服务突然出现，它们是衡量企业社交媒体有效性的伟大工具，但当你将它们发布在你的社交媒体账户上时，这说明了你和你的追随者什么呢？

Twitter、脸书和其他服务似乎很适合社交和分享信息，但许多人无意中发现了它们，却没有考虑他们的互动可能会被人如何看待。

发布你的 Klout 分数，在推特上说你已经有 1000 名粉丝，以及其他类似的“吹牛”行为似乎变得越来越常见，发帖者显然没有意识到他们正在发送的潜意识信息。

社交媒体的问题在于发帖前缺乏思考时间。在传统营销中，一条信息经过精心制作、改写、讨论，然后被批准发布。在脸书公司的账户上发送推文或发帖只需要敲几下键盘。

我以前分享过我在社交媒体上看到的一些灾难，然而这些灾难对个人的影响较小，然而如果它们变得司空见惯，这对一个品牌意味着什么呢？

让我知道你从企业中看到过哪些社交媒体错误——给我发电子邮件到[tribune@sitepoint.com](mailto:tribune@sitepoint.com)——我会列出让我们所有人烦恼的事情，并发表在未来一期的[论坛报](https://www.sitepoint.com/newsletter/#subscription-info)上，供我们所有人借鉴。
