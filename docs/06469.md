表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 2.1%的 IE9 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

我不得不核对两次数字。Internet Explorer 的大多数主要版本都有所增长，所有版本的总增长率都不到 1%:

*   IE6 和 IE7 增长了一点。不要惊慌；两种浏览器的总和不到 0.7%，因此任何变化都显得有些夸张。
*   IE8 涨幅最大，为 0.64%。我肯定这是一个奇怪的现象，但它表明浏览器对许多用户来说仍然很重要。
*   IE9 下降了一点点，但速度比我们前几个月看到的要慢得多。现在是 5.2%，与全球 Windows Vista 用户数量相当(该操作系统不支持 IE10)。这只是一个巧合，但使用 Vista 是你想保留 IE9 的唯一原因。
*   最后，IE10 又度过了一个体面的月份。IE10 是 Windows 8 的默认浏览器(7%的操作系统市场份额)。Windows 7 用户(52%的市场份额)几乎没有理由不从 IE9 升级，尽管有些人可能出于传统应用程序的原因而保留 IE8。

Internet Explorer 的增长是以 Chrome 和 Firefox 为代价的。Chrome 下跌了近 0.3%，这是数月来的首次下跌。Firefox 下跌超过 0.8%，一段时间以来一直在 20%左右波动。

我个人认为这是个好消息。我们需要健康的竞争，而谷歌强大得令人担忧。微软的浏览器在本世纪早期占据主导地位，这导致了多年的停滞。虽然谷歌不太可能凭借 Chrome 获得类似的 95%的市场份额，但该公司也控制着我们所依赖的大部分移动空间和网络应用。

让我们不要忘记 IE10 是一个很好的浏览器…如果你喜欢没有 CSS3 动画的生活，IE9 很好。也许我不太走运，但是 Chrome 让我在开发方面更加头疼！

至于其他浏览器，Safari 几乎不动，Opera 有小幅增长。Opera 的 Blink 版本(版本 15 以上)占据了 0.2%的市场份额——或者说是浏览器用户群的 18%。Opera 用户通常升级很快，所以似乎不是每个人都相信新版本。

## 移动浏览器的使用

2013 年 8 月，手机使用在所有网络活动中所占的比例略有上升，达到了 18 %( T1)——创历史新高。移动取代桌面的预测可能有点为时过早，但大多数企业应该考虑他们的移动战略。创建新网站时，很难证明不采用响应式网站设计技术是合理的。

热门移动浏览应用:

| 移动的 | 七月 | 八月 | 变化 | 亲戚 |
| 机器人 | 28.64% | 28.62% | -0.02% | -0.10% |
| 苹果手机 | 22.43% | 21.32% | -1.11% | -4.90% |
| Opera Mini/Mobile | 15.73% | 16.10% | +0.37% | +2.40% |
| UC 浏览器 | 10.57% | 11.15% | +0.58% | +5.50% |
| 诺基亚浏览器 | 7.14% | 7.04% | -0.10% | -1.40% |
| 铬 | 3.77% | 4.25% | +0.48% | +12.70% |
| 黑莓 | 3.13% | 3.17% | +0.04% | +1.30% |
| 其他人 | 8.59% | 8.35% | -0.24% | -2.80% |

除了 iPhone 上的 Safari 略有不同寻常的下降之外，图表中没有什么变化。然而，如果你认为桌面浏览器的兼容性很难，9%的“其他”表明我们有更多的移动应用程序要担心！
