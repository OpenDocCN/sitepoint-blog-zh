# 去做吧！克服为自己工作恐惧的 5 种方法

> 原文：<https://www.sitepoint.com/fear-of-working-for-yourself/>

如果你正在考虑创业或开始自由职业，你可能会害怕得要死。让我们面对它；有很多要担心的。当你为自己工作时，有很多因素会发挥作用，以至于你在外部工作时可能不必面对这些因素。你可能会担心:

*   你会失败的。
*   你将赚不到足够的钱。
*   会对你的家庭造成负面影响。
*   你最终会讨厌它。
*   你不够自律。
*   你没有动力。
*   你会卖一些没人想要的东西。
*   你对市场营销、网络或企业管理了解不够。

看着这份清单可能会让人麻痹，你甚至可能会说服自己，你不适合创业。但事实真的是这样吗？如果你已经做了研究、工作和计划来[为自己创业成功](https://www.sitepoint.com/leaving-a-full-time-job/)定位，你不能让恐惧让你偏离轨道。

尝试这些行动来帮助你控制恐惧，追随你的梦想。

**1。让恐惧为你所用**

考虑你所有的担心和担忧，但不要让它们侵蚀你的决心，而是用它们来推动和挑战你前进的每一步。

**2。采取积极的态度**

拥有积极的态度不仅是成功的关键因素，而且你越乐观地思考，你就越相信它，它就越会自动成为你日常生活的一部分。

**3。设定现实的目标**

从小处着手，让你的动力随着时间积累。当你设定现实的、可实现的目标，并经常评估它们时，你会更受鼓舞，对自己能完成的事情更有信心。

**4。求助**

人生中很难完全靠自己做任何事情。从家人和朋友那里得到支持和鼓励，并且知道人们相信你，可以大大抑制恐惧。你甚至可以考虑找一位导师来帮助你。

**5。耐心点**

制定一个计划，努力工作，永远做到最好，但要意识到这不会一蹴而就。如果你接受这是一个充满成功和失败的过程，那么活在当下会更容易……欣赏你旅程中的每一步。

你为自己工作时最害怕的是什么？你是如何克服它们的？
