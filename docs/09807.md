# 脸书 JSON 文件示例

> 原文:[https://www.sitepoint.com/facebook-json-example/](https://www.sitepoint.com/facebook-json-example/)

这是一个脸书 JSON 文件的例子，当你从脸书 API 获取数据时，你可能会看到它。它还可以用来包含配置文件信息，使用简单的 JSON 格式可以很容易地在系统组件之间共享这些信息。

**也:** [查看更多 JSON 例子。](http://www.jquery4u.com/json/10-example-json-files/)

```
{
   "data": [
      {
         "id": "X999_Y999",
         "from": {
            "name": "Tom Brady", "id": "X12"
         },
         "message": "Looking forward to 2010!",
         "actions": [
            {
               "name": "Comment",
               "link": "http://www.facebook.com/X999/posts/Y999"
            },
            {
               "name": "Like",
               "link": "http://www.facebook.com/X999/posts/Y999"
            }
         ],
         "type": "status",
         "created_time": "2010-08-02T21:27:44+0000",
         "updated_time": "2010-08-02T21:27:44+0000"
      },
      {
         "id": "X998_Y998",
         "from": {
            "name": "Peyton Manning", "id": "X18"
         },
         "message": "Where's my contract?",
         "actions": [
            {
               "name": "Comment",
               "link": "http://www.facebook.com/X998/posts/Y998"
            },
            {
               "name": "Like",
               "link": "http://www.facebook.com/X998/posts/Y998"
            }
         ],
         "type": "status",
         "created_time": "2010-08-02T21:27:44+0000",
         "updated_time": "2010-08-02T21:27:44+0000"
      }
   ]
}
```
