# 谷歌捕获 reCAPTCHA

> 原文:[https://www.sitepoint.com/google-recaptcha-takeover/](https://www.sitepoint.com/google-recaptcha-takeover/)

谷歌收购了 reCAPTCHA 公司，该公司开发了流行的垃圾邮件和欺诈防范系统。

验证码(区分计算机和人类的完全自动化的公共图灵测试)是你在填写网页表格时必须阅读和输入的弯弯曲曲的字母或单词。它们是一个障碍，最终会被计算机的进步所毁灭，但是它们可以帮助从真实的人类用户中筛选出垃圾邮件。

reCAPTCHA 已经在 100，000 个网站上实施。这是较好的系统之一，许多人会说它是最好的:

*   这项服务是免费的。
*   这个小部件很容易嵌入到您的 web 表单中。
*   为视力受损的用户提供了验证码的听觉版本。

reCAPTCHA 也是独一无二的，因为它改善了书籍和其他纸质文件的数字化过程。这个小部件向用户显示两个扫描的单词，它们没有通过光学字符识别(OCR)。其中一个单词是已知的，但另一个需要进一步的人工验证。如果用户解决了已知单词，reCAPTCHA 假定第二个单词可能是正确的。如果几个用户输入相同的文本，系统可以高置信度地验证该单词。

reCAPTCHA 是众包的一个正面例子:它将一个费力的过程分配给数百万用户。书籍被快速而准确地数字化，开发者可以确信一个真实的用户已经访问了他们的网站，用户的不便被保持在最低限度。

这笔交易对两家公司来说是双赢的。reCAPTCHA 将受益于谷歌的财务和营销影响力。谷歌可以取代他们糟糕的内部验证码系统，并使用 reCAPTCHA 的数字化设施来协助他们雄心勃勃和有争议的图书扫描项目。

两家公司都没有透露收购的全部条款，但谷歌不太可能从 5 亿美元中得到太多改变。

你用 reCAPTCHA 吗？这笔交易会让我们更加依赖谷歌服务吗？
