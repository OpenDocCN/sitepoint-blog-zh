# 将网站截图放入你的网络应用的 9 种方法

> 原文:[https://www . site point . com/9-ways-to-put-site-screens-in-your-web-app/](https://www.sitepoint.com/9-ways-to-put-site-screenshots-in-your-web-app/)

如果您的网站或网络应用程序链接到外部网站，提供这些网站的截图可能是修饰链接的好方法。截屏为你的访问者提供了一个视觉提示，让他们在点击链接之前知道他们要去哪里。但是手动创建你的出站链接截图是一个耗时的过程，如果你想创建屏幕截图是完全不可行的。下面的九个服务将通过 API 自动为你的网站或应用创建截图。

*   Thmbnl 是一个非常酷的项目，由社交书签服务 [Ma.gnolia](http://ma.gnolia.com/) 的幕后人员发起。每月免费提供 1，000 个缩略图，900，000 个拇指的付费套餐高达每月 149.95 美元。该服务为 Ma.gnolia 上的网站截图提供支持，可以提供 5 种尺寸的截图。
*   Websnapr 是来自 Construktiv 的一项服务，该公司是另一个社会化书签网站 [Mister Wong](http://www.mister-wong.com/) 背后的德国网络商店。缩略图有 4 种尺寸，该服务每月免费提供多达 250，000 张截图。该网站已经提供了超过 10 亿张缩略图。
*   [webshots pro](http://www.webshotspro.com/)–免费计划每月最多 25，000 次查询，按访问付费计划每月最多 500，000 次查询，费用为 85 美元。拇指可以作为 jpg，gif 或 png 和自定义大小交付。
*   Thumbalizr–德国服务 thumb alizr 实际上可以免费使用，无需注册 API 密钥，尽管访问受到一定限制。对于包括自定义尺寸、无水印和按需截图刷新在内的一系列功能，你必须支付一些现金。
*   snap casa——snap casa 是一项免费服务(尽管他们接受 PayPal 捐赠),提供四种尺寸的缩略图。作为一项免费服务，当你排队等待新截图时可能会有延迟，但实现起来再简单不过了。
*   [ShrinkTheWeb](http://www.shrinktheweb.com/)–每月免费接受多达 250，000 次请求。除了购买更多的请求，付费用户还可以购买额外的服务，如漂亮的“全长网页缩略图”选项。
*   WebThumb–bluga . net 的 web thumb 服务为 PHP、Ruby 和 Python 提供了一个带有包装器的 REST API。缩略图每个花费一个信用点，API 密匙每个月有 100 个免费信用点。20 美元可为您多购买 6500 个信用点，您一次购买的信用点越多，价格越低。
*   Thumbshots.com——每月 9.99 美元可以让你在 Thumbshots.com 上获得 5 万次请求，这是使用最广泛的缩略图截图 API 之一，每月 999 美元可以获得 1000 万次点击。DMOZ、美国在线和网络解决方案等网络巨头都在使用 Thumbshots.com。他们还在 Thumbshots.org 的 T2 经营一个精简版的免费网站。
*   Alexa 网站缩略图服务(Alexa Site Thumbnail Service)是亚马逊网络服务家族中一个鲜为人知的部分，Alexa 网站缩略图服务采用“按使用付费”的模式，每个缩略图收费 0.0002 美元——或者说每 5000 个缩略图收费 1 美元。该网站可以返回任何网络“顶级网站”(可能是 Alexa 目录中列出的最多的网站)，以及任何 DMOZ 网站的缩略图。如果缩略图不存在于服务的数据库中，你不会被收费，网站将在 24 小时内排队等候收录。
