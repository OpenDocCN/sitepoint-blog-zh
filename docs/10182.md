# 介绍:设计节

> 原文:[https://www.sitepoint.com/introducing-design-festival/](https://www.sitepoint.com/introducing-design-festival/)

我们非常兴奋地宣布我们最新的家庭成员的到来:[设计节](http://www.DesignFestival.com)！

设计节是一个激动人心的新网站，由 SitePoint 热情的团队从头开始创建。我们想更详细地探索网页设计主题，并决定一个全新的网站将提供我们一直在寻找的焦点。

设计节将成为来自世界各地的网页设计专业人士的新家。我们正在探索不同的网页设计技术，重点是[标志设计](https://www.sitepoint.com/design-ux/design/)、 [UX](https://www.sitepoint.com/design-ux/design-html-css/) 和[排版](https://www.sitepoint.com/design-ux/copywriting/)。

如果你热爱网页设计，这个网站将为你的激情提供一个出口。请加入我们！

*   访问最佳实践 UX 设计，标志设计，排版和编码建议。
*   放纵你对设计的热情——并在过程中学习一两件事。
*   贡献你自己的专长来帮助别人提高技能。

请[访问新网站](http://www.DesignFestival.com) , [加入我们的脸书页面](http://www.facebook.com/designfestival),[在 Twitter 上关注我们](http://twitter.com/designfestival)了解所有激动人心的设计节新闻。我们希望听到您所有的反馈，所以请让我们知道您的想法。

如果你想[成为这个网站](https://www.sitepoint.com/write-for-us/)的贡献者，无论是写文章还是写教程，请联系:【editor@designfestival.com】T2。

一如既往地感谢您对 SitePoint 和我们姐妹网站的支持。我们对此表示感谢，并希望您和我们一样热爱设计节。
