欢迎来到 WordPress！开始你的第一个 WordPress 博客或网站是令人兴奋的，有时也是令人困惑的。你很可能担心犯错，也想尽快站起来运行。这篇文章的目的就是要做到这一点。我们旨在直接切入您需要的核心内容，以构建一个安全可靠的网站，该网站可根据您的需求定制，并且易于管理。

和任何网站一样，你应该花很少的时间更新、调整和排除故障，花更多的时间创建内容和与客户互动。考虑到这一点，让我们直接开始吧。

## 1.挑选和学习你的主题

挑选一个主题可能会令人生畏，尤其是考虑到有成千上万的主题可供选择。对任何 WordPress 初学者来说，好消息是有一些经过实践检验的真正的 WordPress 主题非常受欢迎并且可以定制。[迪维](https://www.elegantthemes.com/gallery/divi/)和[创世纪](https://my.studiopress.com/themes/genesis/)对初学者来说是很好的选择。它们是用户友好的，这意味着它们不需要深入的开发人员技能，它们是负担得起的，并且它们有一个大的社区来获得支持。

除了选择正确的主题之外，还要学习它是如何工作的，以及如何根据个人需要定制它。这通过两种方式实现:

**1。打开安装指南。**

![Setup guide](../Images/02f7299513d73f213409105c2a932b53.png)

它会给你深入的指导，告诉你如何下载你的主题，如何把它安装到你的 WordPress 站点上，以及如何根据你的需要来定制它。在其中，您可以找到常见问题的疑难解答资源，以及一个常见问题解答部分，当您感到有些困惑时，可以从中找到答案。当所有其他方法都失败时，请通过开具支持单联系客户支持。

**2。找到讨论板。**

![Discussion board](../Images/278277dbbb46203fbe45a420df9aef7b.png)

为你的 WordPress 站点选择主题时，一个活跃的社区是至关重要的。客户支持将帮助您修复您的网站并进行小的定制，但用户社区将为您提供反馈，向您展示主要变化，并引导您使用插件和工具，使您的网站看起来更好，性能更好。

## 2.通过备份你的 WordPress 站点来保护你的辛勤工作

刚接触 WordPress 意味着你可能从未见过死亡白屏，或者成为恶意黑客的受害者，他们将你锁在网站之外。定期备份你的 WordPress 站点将确保当你失去访问权，或者你的站点被某个有问题的插件破坏时，你可以恢复运行。

有三种方法可以备份你的 WordPress 站点:

1.  仔细选择提供备份服务的 WordPress 主机提供商。例如，我们的合作伙伴 [SiteGround](https://www.siteground.com/go/article-sp) 有一个自动备份系统，可以每天为你的网站制作一份副本。它们为您提供了只需几次点击即可恢复数据的灵活性。
2.  使用 FTP 服务器自己备份，并定期在服务器和计算机之间手动传输文件。
3.  [下载一个备份插件](https://www.sitepoint.com/5-wordpress-plugins-backups-migrations/)比如 [UpdraftPlus](https://wordpress.org/plugins/updraftplus/) 或者 [BackupBuddy](https://ithemes.com/purchase/backupbuddy/) 。

## 3.让你的网站搜索引擎友好

对于任何 WordPress 网站，你的目标都可能是获得更多的流量、更多的读者或更多的客户。最好的方法之一就是优化你的网站，让它在谷歌或其他搜索引擎上排名靠前。这种策略被称为搜索引擎优化(SEO)。虽然 WordPress 在搜索引擎优化方面做得很好，但是这两个小技巧会让你以最少的时间投入远远领先于你的竞争对手:

**下载一个 SEO 插件**

虽然有数百个 SEO 插件，但目前最流行的插件是 [Yoast](https://yoast.com/wordpress/plugins/seo/) 。许多其他基于网络的工具，如 [SEMrush](https://www.semrush.com/) 和 [Moz](https://moz.com/) 会随着你网站的发展而变得熟悉，但是对于任何 WordPress 初学者来说，Yoast 是一个很好的开始，因为它是一个一体化的解决方案，有一个清晰易用的界面。从管理界面下载 WordPress SEO 插件:

*插件>添加新的>搜索‘SEO’>点击立即安装*

![Adding a plugin](../Images/b4ae73dd8436713cea26ce5aa2040c38.png)

在下一个屏幕上，只需激活您的插件，并按照说明正确设置它。这些说明可以在 WordPress 下载页面或者插件本身的设置下找到。

**调整您的永久链接设置**

根据 Yoast 的说法，一个简单的永久链接结构对于 SEO 来说是非常理想的。为了更上一层楼，在帖子名称的某个地方添加你的主要关键词。在 WordPress 中调整你的永久链接结构很简单:

*进入设置>永久链接>选择“帖子名称”*

![Permalink settings](../Images/f3eb8bbbc71cd4a1d70238e338e8fb95.png)

想要更深入地了解 WordPress SEO，请查看 Yoast 的指南。

## 4.加强你的安全

为你的 WordPress 站点提供强大的安全性并不需要花费很多钱，也不需要花很长时间来设置。这三个步骤会让你立刻达到目的:

**生成强密码**

你的新 WordPress 站点可能给了你一个用户名和密码，应该尽快更新以阻止黑客使用暴力攻击来控制你的网站。生成一个安全密码需要使用一个与其他登录不同的*密码，并使用一长串数字、字母和符号。如果你想快速得到一个安全的密码，你可以使用一个[密码生成器](https://lastpass.com/generatepassword.php)来帮你完成这项工作。*

**定期更新你的网站**

定期更新 WordPress、你的插件和你的主题，关闭任何对黑客开放的后门和漏洞。看下图，你可以看到 WordPress 会在有更新时通知你，让更新变得简单，只要点击“更新”即可。

![Update WordPress](../Images/ee305a10796304f0f04ab0164caa0aff.png)

这个过程甚至可以为你的插件批量完成。一个小贴士是在更新之前备份你的网站。这样，如果你的新更新有任何问题，或者与你的 WordPress 站点不兼容，你可以简单地恢复以前的版本，让一切恢复正常。

一些主机提供商有自动升级的解决方案，可以节省大量时间，并消除手动升级的担忧。SiteGround 有一个定制的选择加入工具，可以自动更新客户的 WordPress 网站和所有插件。

**安装安全插件**

就像你的电脑可能内置了某种反病毒或反恶意软件一样，你的网站也需要一些类似的防御系统。到目前为止，最流行的是集安全和防火墙于一身的文字围栏和 T2。两者都是免费的和健壮的安全解决方案。他们也有付费版本以获得更多的额外保护。要了解更多关于 WordPress 安全的信息，请查看[WP 初学者循序渐进指南](http://www.wpbeginner.com/wordpress-security/)。

## 5.留意必备的插件

当你刚刚开始使用 WordPress 的时候，你可能会对最佳实践进行大量的研究，并解决你可能遇到的一些问题。这很好，你绝对应该这么做。一路上，你几乎不可避免地会碰到一些关于“必须要有 WordPress 插件”的文章。虽然这些文章可能有很好的意图，但是你应该知道更多的插件并不总是更好。事实上，通常情况相反。你应该保持你的插件最少来运行你的网站，并提供你想要的用户体验。过多的插件会大大降低你网站的速度。更多插件的另一个问题是它们会带来额外的漏洞。任何不是最新的插件都有可能被黑客攻击，甚至出现故障。只从更新频繁的可信来源下载高质量插件。

## 6.通过分析获得关于你的 WordPress 站点的见解

WordPress 在很多领域都很强大，但是内置分析和仪表盘不在其中。幸运的是，有许多插件可以让你轻松跟踪重要的事情，比如你的流量来自哪里，人们第一次访问或再次访问的频率，跳出率等等。

![Analytics](../Images/310e46f883df91fbed63fa9c24bda519.png)

跟踪你的网站指标的通用方法是用 [Google Analytics](https://www.google.com/analytics/#?modal_active=none) 建立一个账户。你可以在他们的仪表板上看到任何你想追踪的东西，但是一个更方便的选择是下载一个我们之前提到的插件，当你登录的时候就可以直接在你的 WordPress 仪表板上看到任何东西。

![Analytics](../Images/7a4177bfda73411e913d0906fb1227af.png)

## 7.鼓励交流

无论你在网上做什么，最终都会有读者、追随者和顾客因为各种各样的原因联系你。他们对你的产品有疑问，他们想因为一篇影响他们的文章而称赞你，他们想为你工作，等等。因此，你需要一些简单的方法让他们接触你。

**添加联系人表单插件**

![Contact form plugin](../Images/faba51caec7e7962386856fe06c6f6fb.png)

WP 表格和[联系表格 7](https://wordpress.org/plugins/contact-form-7/) 对你的 WordPress 网站来说是很好的选择，可以完全定制。如果你想快速设置一些东西，你也可以使用他们的预建模板。它们可以连接到你可能正在使用的任何列表构建或简讯解决方案，比如 MailChimp 或 AWeber。

**给他们多种选择**

虽然给你的客户和读者联系你的方式是很好的，但是清除垃圾邮件发送者也是很困难的。如果你担心注册昂贵的服务或一开始就泄露个人电话号码和电子邮件，你可以使用谷歌语音和 T2 谷歌套件。谷歌语音为你提供了一个免费的电话号码，这个号码可以转接到你的手机上，谷歌套件为你提供了一个定制的电子邮件地址，每个用户每月只需 5 美元。

最后，在线聊天的一个很好的选择是免费聊天。让你的用户从网站上与你和你的团队实时聊天的插件。这些信息甚至可以发送到你的手机上，更加方便。
