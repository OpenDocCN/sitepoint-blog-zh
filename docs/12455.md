# 哦，脸书即将成为互联网

> 原文:[https://www . sitepoint . com/uh-oh-Facebook 即将成为互联网/](https://www.sitepoint.com/uh-oh-facebook-is-about-to-become-the-internet/)

别看现在，但脸书刚刚做了一件很棒的事。比一年半前发布他们的平台更重要的事情。一些可能将脸书嵌入互联网结构的东西——就像这篇文章的标题一样，可能只是轻微的夸张。这也可能是危险的。

脸书所做的是正式向公众推出 [Connect](http://developers.facebook.com/connect.php) 。我们[在 7 月份将 Connect](https://www.facebook.com/sitepoint/) 描述为他们失败的 Beacon 广告平台的第二次迭代——第二次做对了(从用户的角度来看)。Connect 本质上是一个统一的登录平台，允许第三方网站从社交网络之外访问脸书用户资料和社交图数据。从本质上说，这意味着 Connect 将脸书平台扩展到了社交网络之外，但它附带了 Beacon 非常缺乏的用户隐私控制。

Connect 不会直接为脸书赚钱，有些人可能会认为这是该公司目前应该关注的主要问题，但这仍然是一个明智的举动。正如我们在 7 月份所写的，“脸书意识到，越多的人和网站开始依赖他们的 API，他们就越会在我们的网络习惯和体验中根深蒂固，这将使他们更容易赚钱。”(奥姆·马利克有一些关于他们如何做到这一点的好主意。)

## 为什么脸书会赢

Connect 之所以如此特别，是因为它很有可能击败 OpenID、Google Friend Connect 和 Windows Live ID 等竞争对手的单点登录和数据共享平台。几个月前，我的朋友莎拉·佩雷斯在读写网的一篇文章中阐述了为什么[。根据 Perez 的说法，脸书联通之所以成功，是因为每个人都已经在使用它，并且已经在投资它。大多数人在谷歌没有社交图谱数据，大多数人](http://www.readwriteweb.com/archives/facebook_connect_will_be_gamec.php)[仍然不明白 OpenID 是什么](https://www.sitepoint.com/yahoo-mainstream-has-no-idea-what-openid-is/)。但是“用脸书登录”非常有意义，如果它意味着在另一端有更丰富的体验，因为一个可信的网站可以访问你的社交数据，那就更好了。

![](../Images/08b5c4da72eacb94703a282b5162e5c3.png "facebook-connect")

Rafe Needleman 上周在 CNET 也表达了同样的观点。“因为脸书互联不仅仅是一个注册系统，也是一个营销渠道，拥有 1.3 亿月度活跃用户(据脸书称)，该计划将粉碎竞争注册系统，”[他写道](http://news.cnet.com/facebook-connect-officially-open/)。“网站将采用脸书连接有两个原因。首先，他们的用户已经在积极地使用它；数以百万计的用户拥有 OpenID 登录，甚至不知道它。第二，因为它不仅仅是一个注册系统，它还是一个营销渠道。(网站所有者的)私利战胜了哲学。脸书明白了。这就是它获胜的原因。”

据脸书高级平台经理戴夫·莫兰(Dave Morin)称，系统的初步测试发现，用户通过脸书连接以 2:1 的比例通过他们预先存在的登录凭据登录。早期真实世界的结果几乎证实了这一点。

Govit.com 的[新用户有 58%的时间选择用脸书登录。Govit 创始人 Taylor Norrish 告诉《脸书内幕》,他认为脸书互联实际上是在推动新用户的注册。这是一个强有力的声明，我们相信其他开发人员会关注它。正如 Needleman 所写:利己者胜。](http://www.govit.com/)

## 这并不全是好事

然而，围绕脸书联通有一些非常现实的担忧。正如佩雷斯在 7 月份阐明的那样，尽管这次表面上看起来比 Beacon 好得多，但仍有潜在的隐私问题需要注意。她写道:“随着越来越多的公司和商业网站采用朋友连接，侵犯隐私的机会就越大。”

脸书创始人马克·扎克伯格经常说，提供强大的用户隐私控制是脸书的一个中心主题，所以希望该公司在连接时对隐私问题保持高度敏感。

另一个担忧，当然也是更危险的担忧，是一家公司如此深入互联网是否真的是一件好事。OpenID 的支持者 Chris Saad 说:“脸书正试图用他们自己的登录来取代所有的登录，并使用他们的专有平台来控制社交图的创建、分发和应用。”。据他所说，这是一个可怕的提议。

如果脸书连接真的赢了，这可能意味着脸书基本上是独自掌舵控制我们整个网络社交体验。这是我们想要的吗？

去年五月，在为读写网写稿时，我得出了一个结论:脸书将会成为主流。让早期用户兴奋的新服务和创新最终会通过脸书进入主流。

我当时还猜测，“总的来说，主流用户对他们的数据放在一个地方没什么意见。”那过去(现在)可能是真的。但现在，脸书正试图教导主流用户为什么数据可移植性是一件如此伟大的事情——然而，他们是按照自己的方式做的，并一直控制着整个过程。

脸书已经想出了一种方法来利用开放网络的思想，并使它们为自己的利益服务。太棒了，还有点吓人。
