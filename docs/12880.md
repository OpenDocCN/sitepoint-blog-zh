# 谷歌增加关键词工具的搜索量

> 原文:[https://www . site point . com/Google-ads-search-volume-to-keyword-tool/](https://www.sitepoint.com/google-ads-search-volume-to-keyword-tool/)

![](../Images/cdd820c591143db7374290708291c7dd.png "google-adwords-logo")谷歌[昨天晚上](http://adwords.blogspot.com/2008/07/keyword-tool-updated-with-search-volume.html)宣布，它正在为其 [AdWords 关键词工具](https://adwords.google.com/select/KeywordToolExternal)增加搜索量。除了建议相关关键词外，该工具现在还包括每个关键词每月搜索查询的大致次数和最近一个日历月的搜索量。谷歌的特雷弗·克莱本(Trevor Claiborne)在 Inside AdWords 博客上表示:“这些近似数字旨在提供比之前该工具更好的关键词月度和平均搜索量。”

对谷歌关键词工具的改进仅仅发生在雅虎之后的几个星期。[关闭其关键词建议工具](http://www.webmasterworld.com/yahoo_search/3684134.htm)。雅虎！其搜索营销产品所基于的旧 Overture 系统遗留下来的工具，长期以来一直包含大致的搜索量数字。雅虎！大约 14 个月前，[停止了对该应用的官方支持，并表示](http://www.seroundtable.com/archives/013404.html)搜索量数据因此不准确。即便如此，由于查询数字，这个工具被视为搜索引擎营销商的武器库的一个有用的部分，直到雅虎！上个月被砍掉了。

![](../Images/0307a342c281a0f4c11a5cf94785879a.png "google-keyword-tool")

搜索查询数字有助于规划搜索营销预算，因为你可以判断每个关键词的大致流量，并据此规划你的广告总购买量。但是新的谷歌工具也有其他用途，比如市场调查。例如，看看我在下面制作的流行 Javascript 框架的搜索量表(从中随意抽取)。

| **框架名称** | **大约。搜索量** |
| 原型 | 368k |
| MooTools | 201k |
| jQuery | 16.5 万 |
| 脚本语言 | 49.5 千 |
| extjs | 49.5 千 |
| 道场工具包 <sup>*</sup> | 3.6k |
| Qooxdoo | 1.3k |
| Sproutcore | Two hundred and sixty |
| 注意:“Dojo”这个词有 460，000 个搜索，但是，好吧，这些真的可以是任何东西。 |

更透明的谷歌关键词工具是在 [AdPlanner](http://adwords.blogspot.com/2008/06/introducing-google-ad-planner.html) 工具发布后不久推出的，该工具显示竞争对手网站的大致流量。(在我们的测试中，总体上比 Compete、Quantcast、Alexa 或其他类似的公共交通测量工具更准确。)谷歌开放并分享更多的海量网络数据，这对所有站长来说都是一个胜利。
