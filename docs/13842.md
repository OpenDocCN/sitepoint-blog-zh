# 用贿赂建立一个论坛

> 原文:[https://www.sitepoint.com/building-a-forum-with-bribes/](https://www.sitepoint.com/building-a-forum-with-bribes/)

对于一个内容网站的发布者来说，让一个论坛起步比其他任何事情都重要，是最耗时的事情之一。在某些情况下，需要几年的时间来获得足够的帖子，论坛才能真正活跃起来。

问题是，如果人们没有主题可以回复，那么他们就不太可能注册，如果没有人注册，那么就没有主题可以回复。这在很大程度上是一种进退两难的局面。偶尔会有注册的人问你一个问题，经过一段时间后，你可能会有足够多的人开始讨论。然而，总的来说，论坛需要足够多的帖子，才能吸引每天的回头客。我估计至少有 1000 个帖子。

如果你不希望等待，尽管有技巧，你可以使用公共领域的内容自动制造商发布。你可以自言自语，字面意思，用多个用户名，可以求好友访问发帖，也可以花钱请人发帖。后者，即支付职位费用，是我今天想谈的。

**跑步比赛**

新企业总是将竞赛作为初始营销的一部分，互联网企业也不例外。所以你可以举办一个竞赛，奖励第一个达到一定级别的帖子，确保它们都是高质量的。你也可以举办一场竞赛，奖励推荐最多新会员的人。这样做的好处是，你将从第二名、第三名和第四名的努力中获益，而实际上不必付钱给他们。这种方法的缺点是，你可能不得不花大量的时间来调节比赛，以防止作弊，侵犯版权和无用的张贴。如果你有很多时间，但没有很多钱，这可能是给你的。我个人更喜欢其他方法。如果你确实想举办一场比赛，SitePoint 有一个专门的[论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=219)来做广告。

**收入分成**

贿赂你的海报的一个巧妙方法是分享网站的收入。从表面上看，这似乎是一个巨大的头痛。确保你有他们的税务信息，发出多张支票，建立信任，让他们知道你没有做空他们。然而，如果你使用谷歌广告，这真的不难。你真正需要的是让你的用户注册 Google Adsense，然后你修改你的论坛代码，允许他们将他们的 Adsense ID 输入到他们的论坛简介中，然后让你的论坛在用户发起的帖子上随机抽取一定比例的 ID，从而给他们分享他们自己帖子的收入。我认为这是一个鼓励用户活动的好方法。如果这听起来太技术化而无法解决，不要担心，人们已经为这个东西编写了代码，带有简单的安装说明。vBulletin 的是这里的。

我尝试了这种方法，但不幸的是，它并不像我希望的那样有效，我只获得了少数新会员。虽然我仍然对这种方法很有信心，但我只是认为会员必须从一开始就看到一些收益，否则他们会泄气。因此，对于一个流量很少的新网站来说，它不会真的那么有效。如果你已经建立了网站，但是想要快速成长或者在竞争论坛中获得优势，我认为这绝对是值得考虑的。

**付费发帖子**

也可以选择直接为帖子付费。几乎每天都有人在 [SitePoint 的论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=220)上发帖索要付费海报。你可以效仿这些人，向个人出租，但就像竞赛选项一样，你必须担心监管每张海报，并花更多时间向每个人支付费用。我个人更喜欢简单一点的。

由于近来付费发帖现象的增长，许多公司涌现出来提供论坛发帖服务。这些公司雇佣、支付并监管表现良好的海报。你只需购买一次，其余的由他们处理。价格也很合理，至少在我看来是这样。对我来说这听起来是个不错的选择。

现在，这里有一些需要担心的问题。我担心这些公司会给我发一堆英语水平很差的海报，会发跑题帖子的人，或者可能只是在我的论坛上充斥着“leetspeak”垃圾的不成熟的青少年。所以我联系了许多这样的公司来评估他们的服务，最后评估了其中的两家，是的，他们在提供服务的时候就知道他们正在被评估，所以他们总是有可能比他们的普通客户更好地对待我。尽管希望如果你们中的任何人尝试他们的服务，你会在这里的评论中发表你自己的评论。

我考察的第一家公司是 Inb0x.com，他们很快就给了我回复，比我接触的其他公司都要快。按照我的要求，这项工作在大约两周之后就完成了。他们的海报似乎大部分来自英国，因此具有出色的英语技能。我得到了两个中级套装，300 个帖子，65.99 美元。我让他们在我的两个全新论坛上发帖，一个我认为是[轻松话题](http://www.savvyshopping.net/forums/)，一个我认为是[艰难话题](http://www.4laserhairremoval.com/forums/)。

不出所料，先做了比较简单的题目，他们似乎在比较难的题目上更加努力，但老实说，他们在比较难的题目上出乎我的意料。作者们似乎在发帖之前实际上研究了这个话题，他们的帖子比我希望的要好得多。

我没有提到我不希望他们在普通聊天中发帖，但是 Inb0x.com 在普通聊天中的发帖确实比我调查的另一家公司多，总共有 36%的公司在这方面发帖。

Inb0x.com 还提供 75 个帖子 22.99 美元和 700 个帖子 145.99 美元，这将是我推荐的套餐，因为我真的认为一个论坛需要大约 1000 个帖子来启动。

我考察的第二家公司是 PostOnMyForum.com。这些人也很快回复了我，但他们告诉我，他们现在太忙了，无法承担任何额外的工作，所以过了几周他们才真正有空开始发帖。

像我对 Inb0x.com 做的那样，我请求了两个中级包，这里是 300 个帖子 95 美元，并让他们在我的每个论坛上做一个。这些家伙花了比 Inb0x.com 多一点的时间来完成他们的工作，但是我相信这是由于他们的一些工人偷懒。当我向他们提到这一点时，他们立即撤换了偷懒的人，把偷懒的人免费提供的职位给了我。

另一个区别是，PostOnMyForum.com 允许作者用多个用户名发帖。我最初接受了这个选项，因为我认为它会增加活动的外观。然而，我不会再这样做了。原因是，当允许这种自由时，作者似乎会对他们的其他帖子做出许多“是的，我同意”类型的回复。这增加了他们的帖子数量，但并没有给论坛增加多少高质量的内容。

PostOnMyForum.com 的大部分海报似乎来自美国，尽管它们看起来确实很年轻。和 Inb0x.com 一样，在更难的论坛上，他们似乎在发帖前做了研究，这很好，但是我觉得他们没有做那么多研究。就一般聊天中的帖子而言，只有 22%的帖子在这些领域。

PostOnMyForum.com 还提供 100 个 35 美元的帖子和 1000 个 285 美元的帖子，如果你真的想让事情进展顺利，我再次推荐最大的软件包。

总的来说，我对这两种论坛发帖服务的体验非常满意，如果你有一个新的论坛并且负担得起，我肯定会推荐你使用这种服务。使用这些服务可以让你在论坛的增长中快进几个月甚至几年，因此最终从增长中获得的广告收入应该超过你最初为这些帖子支付的费用。我知道下一次我启动一个论坛时我会这样做，我甚至可能为我在这里提到的两个新论坛再次这样做。
