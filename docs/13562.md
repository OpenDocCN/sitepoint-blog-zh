# WD06: Kelly Goto，“为生活方式而设计”

> 原文：<https://www.sitepoint.com/wd06-kelly-goto-designing-for-lifestyle/>

凯文·杨克在澳大利亚悉尼举行的[网络指南南方](http://webdirections.org/)会议上报道。

今天，Kelly Goto 在 Web Directions South conference 的开幕式上，他谈到了旨在融入用户生活的设备、产品和软件设计的当前目标、趋势和方法。

除了可用性之外，今天成功的设计要求界面能引起用户的情感反应。毕竟，我们是有情感的生物，我们对产品和服务的选择——不管是好是坏——往往是由情感驱动的。

Goto 观察到的相关设计方法包括通过自然语言模拟人与人之间交互的界面，通过预打包的解决方案和方便的默认首选项提供以前由早期采用者主导的服务(如移动互联网)的设备。

后藤指出，她所说的“2.0 设计”的主要内容之一是让一切看起来“可爱、可爱、可爱，以至于我们忍不住想与之互动。”

Goto 正在追踪的设计过程中的关键新元素是*人种学*的实践，即对一群人的独特行为和需求的研究，这使你能够构建自然适合目标用户生活的解决方案。

接下来的几周，这个会议将会在 [Web Directions South 博客](http://webdirections.org/)上进行播客。
