# 有所作为

> 原文:[https://www.sitepoint.com/making-a-difference/](https://www.sitepoint.com/making-a-difference/)

在最后一个艰难的话题之后，Photoshop 中的差异模式变得非常容易理解:

> 差异模式:查看每个通道中的颜色信息，并从基色中减去混合色，或者从混合色中减去基色，这取决于哪个具有更大的亮度值。与白色混合会反转基色值；与黑色混合不会产生任何变化。

当我考虑 RBG 价值观时，很容易想到这一点。当您用红绿蓝值表示颜色时，每种颜色可以有一个从 0 到 255 的值。如果你有红色=0，绿色=0，蓝色=0 (0-0-0)，你最后得到黑色。如果你有 255-255-255(红色、绿色和蓝色一直向上)，你最后得到的是白色。

所以让我们拿一个 RGB 值为 27-97-168 的蓝色正方形(你可以从数字中看出它大部分是蓝色，因为红色和绿色的值要低得多)放在白色背景的上面。让我们复制另一层的蓝色方块，并设置为不同的模式，并移动它。白色(255-255-255)将减去蓝色(27-97-168)，你将得到一个橙色(228，158，87)。很酷吧。

![](../Images/e2076657e525327810924b7f189ebfd7.png)

如果你像我一样通过了一年级的数学考试，你现在可能已经发现，如果你复制一个图层并保持两个图层在一起，然后将最上面的一个设置为差异模式，你会得到一个黑色的区域。由于层是相同的，颜色减去 0-0-0，导致黑色。

一个很酷的效果，你可以实现不同的模式是一个黑色霓虹灯发光轮廓类型的效果。

1.拍张照片，任何照片。(我将使用我得到的标准黄玫瑰图片。)

2.复制图层。

3.将顶层设置为差分模式。此时你只会看到黑色。

4.转到滤镜>模糊>高斯模糊(你可以稍后尝试其他的，好玩的)玩半径。我把我的设置为 8 像素，但这取决于你的照片。

![](../Images/07459bc3604fd7c1023eb922eb89510f.png)

5.画面会开始形成微弱的霓虹效果。您应该可以从预览中看到这一点。找一个适合自己的设置，应用模糊效果。

![](../Images/0a98c32ab843eee8c3fa705c184b5310.png)

另一个巧妙的应用是重叠的“剪切”效果，可用于快速的网页图形或徽标。

1.创建两个带有形状或文本的层。这最适用于实心形状。底层应该是黑色的，顶层应该是白色的，它们都应该在白色背景上。

![](../Images/932e331fdf85af9dcf2e25d12dfdcd1e.png)

2.将顶部的白色图层设置为差分模式，你将立即得到一个重叠的剪切效果！

![](../Images/8e7c5e05b0ece2ab943fb4afa50ea746.png)

最后，我之前没有考虑过的一个应用程序是使用差异模式来帮助您将两张图片“缝合”在一起——就像如果您正在制作全景图，或者如果您通过扫描较小的部分来扫描较大的文档。这个网站描述了它，但是我将简要地回顾一下。

1.假设你有一张大照片和一个小扫描仪。你先扫描大照片的一面，然后扫描另一面，中间有一小部分重叠。

2.在 Photoshop 中，你制作一个足够容纳整张照片的新文档，然后把它分成两半，放在各自的图层上。回想一下，有一些重叠；你尽力把它们排好。

3.将顶层设置为差异模式可以帮助您更容易地排列图片。请记住，如果重叠部分排列完美，那么颜色会相互相减，留下黑色条纹。所以用移动工具或箭头键轻推图层，直到你扫描到尽可能多的黑色条纹。将顶层调回到正常模式，看看效果如何！
