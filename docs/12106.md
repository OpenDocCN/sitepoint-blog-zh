# 你会犯这些电子邮件礼仪错误吗？

> 原文：<https://www.sitepoint.com/email-etiquette-mistakes/>

我喜欢电子邮件。我敢肯定，如果没有电子邮件来帮助我简化工作方式，我的工作效率会大打折扣。事实上，我认识的几乎每个人都使用电子邮件——客户、同事、家人、朋友——所以我很容易把它作为我的主要交流方式。

然而，仅仅因为电子邮件比电话或面对面的会议更快、更容易、更非正式，并不意味着你的电子邮件交流应该完全放松。如果你想维护自己作为专业人士的声誉，你应该在包括电子邮件在内的所有书面交流中遵循通用的[商务写作指南](http://www.amazon.com/10-Steps-Successful-Business-Writing/dp/1562864815)。

以下是我在电子邮件中遇到的一些最大的错误…你犯了几个？

你的电子邮件越来越多，越来越多。

如果你的信息超过了一个屏幕的长度，并且花了几分钟来阅读和消化，那么它可能太长了。试着坚持每封邮件只有一个主题，当你需要包含更多信息时，使用加粗的副标题，这样最重要的细节就一目了然了。

你的主题行不能描述邮件的内容。

当我打开一封来自客户的邮件，主题写着紧急，并标记为高优先级，却发现一个最小的非紧急问题时，我感到被欺骗了。写下你的主题，这样可以解释信息的内容。收件人应该能够只是阅读主题，并有一个想法是什么，它有多重要，以及他们何时需要采取行动。

您没有提供任何联系信息。

我无法告诉你我收到了多少署名为吉姆、苏或鲍勃而没有其他身份信息的电子邮件。最重要的是，很多时候使用的电子邮件地址都是免费的，所以没有办法追踪到底是谁发的邮件(详见下一点)。所有初始通信都应包括您的电子邮件签名，通常至少会提供您的全名、公司和 URL。一个电话号码、头衔和重复你的电子邮件地址也没有坏处。

你使用非专业的地址进行商务交流。

Gmail 很受欢迎。它是免费的，提供方便的移动访问，你可以获得大量的存储空间，每个人都可以使用它。我明白。但对我来说，从一个 sexybabe756 地址收到一封与业务相关的电子邮件是一大商业失礼。理想情况下，您应该使用公司地址(如 yourname@yourbusinessdomain.com)发送商务电子邮件。如果这不太可能，那就创建一个免费的地址，既不会损害你的职业素养，又能给人留下良好的第一印象。

有许多拼写错误。

拼写检查很容易。在您的电子邮件客户端中设置默认设置，在发送邮件之前检查所有邮件。是的，电子邮件更不正式，对拼写和语法错误也更宽容，但发送一条错误百出的信息看起来很糟糕。还有，请把时髦的缩写(thx，plz，u)留到推特或者短信上。

你的信息语气有问题。

有些对话不应该通过电子邮件进行，因为在电子邮件中传达某种语气不像在口头交流中那样容易。言简意赅、吹毛求疵，甚至是一个失败的笑话都会让对方感到受到攻击和不被尊重。如果你的信息语气有点粗糙(重读一遍以确定)，或者甚至只是传达给接收者，你可能会更好地拿起电话。而且，在你怒火中烧的时候发送一条写得很激烈的信息从来都不是一个好主意。点击发送前，花些时间冷静一下。

我承认，我倾向于在我的一些信息中有点过于罗嗦。我发现离开电子邮件一小会儿，然后回来，重新阅读和编辑是非常有帮助的。现在轮到你了…你犯了多少这样的错误？
