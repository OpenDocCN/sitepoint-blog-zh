# 设计颜色:白色

> 原文:[https://www.sitepoint.com/color-in-design-white/](https://www.sitepoint.com/color-in-design-white/)

在过去的几个月里，我一直在写关于设计中的颜色，从红色开始，现在以白色结束。白色几乎可以搭配任何颜色。它让人联想到清洁、天真和纯洁。西方新娘穿白色是传统，而在西方牛仔电影中，坏人总是穿黑色，而好人穿白色。

像黑色一样，白色是一种中性色，最常用于网页背景，当然也用于印刷。有一种误解，认为白色会令人厌烦，或者选择白色是可以预见的，但有许多设计师选择白色并与白色一起做出美丽的设计。

许多使用白色的最好的网站看起来“简洁明了”。这种类型的设计看起来似乎很容易，但它确实是一个挑战。最好的设计往往是那些看起来不像是被设计出来的设计。

**在设计中使用白色**

白色有多种变化。米白色包括古色古香的白色、奶油色、亚麻色、米色、纸色、雪色、珍珠色和象牙色，这些颜色看起来比纯白色柔和或不那么明显。大量的白色可以与浅蓝色搭配，创造出冬日的色调，或者与黄色和柔和色搭配，创造出夏日的感觉。

白色的点缀可以极大地提亮设计，并使暗红色、T2 绿色、T4 蓝色和紫色看起来更加突出。如果你是美国人、法国人、英国人、瑞典人(以及许多其他国家的人)，那么红色、白色和蓝色就构成了爱国色彩。黑色和白色是你能拥有的对比最强烈的组合，能传达严肃或戏剧的感觉。

使用白色并不意味着“简约”。极简主义不依赖于任何特定的颜色。同理，“留白”并不一定意味着任何设计元素周围的空间一定是白色的。你这个设计师必须做出颜色选择。

所以，让我们来看看一些网站，它们在白色方面做得很好。许多网站使用白色背景的照片或图像，增加了网站的整体时尚感，这并非偶然。
