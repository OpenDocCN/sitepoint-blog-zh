# Opera 能提高市场份额吗？

> 原文:[https://www.sitepoint.com/opera-market-share-rise/](https://www.sitepoint.com/opera-market-share-rise/)

写戏曲文章总是犹豫不决。浏览器有一个充满激情和声音的社区，在讨论它的竞争对手时，你永远不会遇到如此激烈的争论。

如果我斥责一个特性，我会被告知 Opera 的实现是最好的。如果我赞扬歌剧，我会被告知我没有给予它足够的赞扬。不管是哪种情况，很明显我太笨了，不会欣赏浏览器。

Opera 的市场份额是最有争议的问题，当我提到神奇的 2%时，一场激烈的战争就会爆发。最狂热的用户会声称各种阴谋，不准确，和谬误。所以，现在开始…

根据大多数 web 统计，Opera 的桌面版在五大顶级浏览器中市场份额最低。它徘徊在 2%左右，自 10 多年前推出以来几乎没有变化。

这并不是说 Opera 的用户群没有增长。随着网络使用的增加，安装总数也在增加，该公司报告说，全球有 1.4 亿用户(所有移动和桌面版本的用户总数)。然而，它没有像 Chrome 那样经历加速增长。谷歌的市场份额增长速度超过了互联网的总体增长速度。

此外，Opera 是目前安装最广泛的移动浏览器。IE、Safari 和 Firefox 的移动版显得无足轻重，因为它们都需要功能强大的智能手机。Opera Mini 和 Mobile 可以安装在更普通的设备上，是数百万部手机的默认浏览器。也就是说，我预计 iPhone 3G 用户浏览网页的频率会比典型的 Opera 用户更高。

## Opera 的杀手锏是什么？

但是让我们回到桌面版。Opera 的杀手锏是什么？

IE 是公司的老黄牛还是新手浏览器。Chrome 以速度和稳定性著称。Firefox 的声誉来自于面向高级用户的功能和可扩展性。对于那些想要苹果体验的人来说，Safari 是显而易见的选择。

Opera 是优秀的全能浏览器，但在某个特定领域并不擅长。很难确定目标受众并总结浏览器在市场中的地位。有各种各样的原因让它被平静地接受，但当替代产品提供更容易、更快或更可定制的体验时(即使它们在其他方面落后)，没有什么令人信服的理由推荐 Opera。

如果你想要速度，使用 Chrome ( [或者 IE9](https://www.sitepoint.com/ie9-beta-review/) )。如果你想要功能，使用火狐插件。Opera 可能很快很灵活，但它并没有打败竞争对手。一些用户想要一个万能的浏览器，但是我相信更多的人想要他们感兴趣的最好的选择。

歌剧迷可能会认为他们的浏览器是创新的。我同意——该公司推出了许多我们现在认为理所当然的功能。然而，我们已经达到了一个点，大多数新的选择有利基吸引力。Turbo mode 和 Unite 都是很棒的想法，但是有多少人理解或者想要它们呢？

## 歌剧的药片疗法

平板电脑已经出现了很多年，但随着苹果公司 iPad 的成功，这种设备变得越来越受欢迎。克隆产品将在未来 12 个月内充斥市场，提供各种功能、外形和操作系统。它们很可能会超过上网本，并威胁到笔记本电脑和智能手机市场。

Opera 有一个不可思议的机会来生产平板电脑的权威浏览器:

*   它可以在大多数平台上使用，包括 Windows、OS X、Linux、Symbian 和 iOS
*   它内置了电子邮件、聊天、笔记和小工具
*   它很快，而且 Opera 11 提供了扩展功能
*   该浏览器已经针对小屏幕设备进行了优化
*   它具有低功耗、低处理和低存储需求
*   鼠标手势可以适用于触摸控制

唯一的绊脚石是所有的平板电脑都将提供支持浏览器。然而，如果他们寻求许可协议或者能够证明他们的浏览器是一个更好的选择，Opera 有很大的成功机会。

或许平板电脑可以成为 Opera 的救星？如果用户在平板电脑上尝试 Opera，那么在个人电脑上安装它就不那么容易了。

我还不确定 Opera 能否突破其市场份额壁垒，但平板电脑可能是它成为明星的途径。
