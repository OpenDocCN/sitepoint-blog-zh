# 老板不在的时候…

> 原文：<https://www.sitepoint.com/when-the-boss-is-away/>

在过去的两周里，我病得很重——一直卧床不起——我都不记得上次是什么时候了。幸运的是，我现在正在康复，但这是一个有趣的练习，因为我在办公室没有被错过。当然，他们已经注意到我的缺席，我的工作任务不得不被推迟或委派出去；然而，在失望的背后，我不是不可或缺的，我很欣慰地知道，没有我在采煤工作面，世界没有内爆。那么，我该把它归因于什么呢？能够独立行动的同事让世界变得不同。**可靠的系统和流程**拥有聪明的人是一回事，但他们需要知道对他们的期望，以及在需要时如何完成另一个人的任务。记录良好的系统和流程可以一次又一次地拯救您的企业。**科技**如今，我们大多数人都认为基于网络的邮件和远程桌面访问是理所当然的；然而，尝试远程工作没有这些！直到我需要的时候，我才意识到我的电子邮件功能。谢天谢地，这一切都很顺利，虽然我在家里，但在与他人联系时，我并不觉得有什么不利。没有人计划生病。虽然在我离开的日子里，我计划了一大堆会议，但告诉我的客户我病得太重了，无法参加——更不用说接近他们了——这并没有妨碍我做生意。我的客户非常乐意推迟会议，并且完全理解其中的原因。**保险**幸运的是，我只休息了两周，没有长期影响，但如果是持续数月的重大事故或危及生命的疾病呢？我们都不喜欢想象这些情景，但最好为它们做好计划，以防万一。良好的计划和祈祷，你永远不需要请几周病假。但是如果不幸的是，你做到了，并且你已经考虑了以上所有的问题，你会像我一样惊喜地发现你可以集中精力康复，而不是去扑灭公司的火灾。
