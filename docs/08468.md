# 盒子:把所有东西都储存在云中

> 原文:[https://www . site point . com/box-store-everything-in-the-cloud/](https://www.sitepoint.com/box-store-everything-in-the-cloud/)

在同步基于云的文件存储之前，文件存储曾经是一件痛苦的事情。在不同的设备上有不同版本的文件会造成许多不必要的乏味和混乱。人们曾经试图通过基于电子邮件的备份或基于版本的文件名(如“project-proposal-version-5-3 . doc”)来解决这些困难，但这些技术很少被证明是有效的，导致硬盘驱动器和电子邮件帐户中堆满了各种完成状态的不同文档。

但是这些天来，[云计算](http://en.wikipedia.org/wiki/Cloud_computing)正在成为一种普遍的实践，并且成为这个普遍问题的一个很好的解决方案。这不仅仅是为了先进技术的大师们；像 Box 这样的应用程序对任何人来说都很简单和直观，不管他们的技术水平如何。

基于云的文件存储现在是拥有互联网连接的任何人保存重要数据的首选方法。这就是为什么像 Box 这样的应用程序是一个很好的工具，可以将所有东西放在一个地方，而不会占用宝贵的硬盘空间或在 USB 拇指驱动器上携带你的私人敏感数据，这些数据可能会丢失或被盗。

[Box](https://market.android.com/details?id=com.box.android&feature=search_result#?t=W251bGwsMSwxLDEsImNvbS5ib3guYW5kcm9pZCJd) 是一个文件存储应用程序，允许你使用他们的服务器存储你的照片、视频或任何其他类型的文件。它的 5GB 存储空间是免费的，Box 根据你的设备在[有限的时间内提供免费的 50 GB 存储空间](https://support.box.net/entries/20768867-box-50-gb-promotion-faqs)。

[![](../Images/3c36ab71961abdfa5f58a2eaffd52f2c.png)T2】](https://www.sitepoint.com/wp-content/uploads/2012/03/screenshot-1330451328972.png)

Box Android 应用在 Google Play 应用商店是免费的。它允许你从手机的 SD 卡上传照片、视频和任何其他类型的文件。您可以从桌面访问他们的[网站](http://box.com)，并使用您手机上使用的相同帐户登录。

### 用户界面

这款应用的设计很简单，有一种极简主义的感觉。文本和图形的大小刚刚好，不要太小，也不要太大。

用法并不完全直观，所以可能需要时间来了解所有的视觉图标代表什么。理解所有的特性也需要一些反复试验。

进出屏幕的导航可以通过一个中央 *Home* 或 *Back* 键来改善，这是该应用目前所缺乏的。为此，你必须一直使用手机的*后退*按钮来退出任何给定的屏幕。

### 表演

因为您正在访问存储在服务器中的文件，所以应用程序需要快速的数据或 Wi-Fi 连接才能以最佳方式工作。缓慢的数据连接可能不会带来良好的用户体验。

其他任务——如浏览一组图像或访问各种文件夹——可以非常顺利地完成。下载和上传文件花费的时间适中，既不令人印象深刻，也不令人失望。

### 特征

Box 是一种文件存储服务，所以它的主要功能是保存你的最新数据，并在你的各种电子设备之间共享。这些文件可以组织在文件夹中，并可以被视为类似于文件资源管理器的目录。用户可以添加、删除或重命名文件或文件夹。

[![](../Images/71b299ad0a600a85cb288481c2774255.png)T2】](https://www.sitepoint.com/wp-content/uploads/2012/03/screenshot-1330451375589.png)

这些文件可以下载到您选择的任何设备上。还有一个通过电子邮件共享这些文件的选项，以及一个通过电子邮件发送文件或文件夹链接的选项。

从你的手机上，你可以直接将你相机拍摄的图像、音频或视频上传到 Box 网络，这对于与你的朋友和同事分享信息非常实用。

[![](../Images/95b548e826bf0892a2c520bdef8ed74f.png)T2】](https://www.sitepoint.com/wp-content/uploads/2012/03/screenshot-1330451697694.png)

*提示:您可以在不打开应用程序的情况下从图库上传照片。只需点击一张照片，选择“分享”,然后从出现的应用列表中点击“盒子”。它会自动打开 Box 应用程序上的上传菜单。*

下载相当简单。当您打开存储在服务器中的文件时，它会自动将其保存在您打开它的设备上。例如，使用手机打开服务器中的照片会将照片保存到图库中。在某些情况下，它甚至会创建一个与您的 Box 帐户中的文件夹名称同名的相册。

### 缺点

在交付文件存储解决方案方面，Box 是一款非常高效的应用。但是，它可以设计得更直观，以增强导航并提供更好的用户体验。

这种服务现在很常见，类似 Box 的 app 也很多。一个著名的竞争对手， [Dropbox](http://db.tt/VnATHDV) ，有一个自动同步功能，这是 box 目前没有的。就连主要是社交网络应用的 Google +也有“即时上传”功能，可以自动将手机图片存储到用户的 Google+账户中。Box 还没有提供类似的快速上传功能。

### 结论

虽然这款应用缺乏更高级的功能，但它确实完成了工作，而且非常高效。Box 专注于基本文件存储，是一款值得保留的优秀文件存储应用。此外，没有其他文件存储应用程序提供 50 GB 的终身免费存储。

一些小的改进不会有坏处——特别是导航和功能方面——但对于一个提供它所说的功能而没有很多虚饰或不必要功能的应用程序来说，这些都是小问题。对于那些想要一个实用、基本和简单的方式在云中存储文件的用户来说，Box 绝对是一个值得拥有的应用程序。
