# 10 个非常棒的 jQuery 工具提示

> 原文:[https://www.sitepoint.com/10-awesome-jquery-tooltips/](https://www.sitepoint.com/10-awesome-jquery-tooltips/)

我们收集了另外 10 个很酷的 jQuery 工具提示插件，你可能见过也可能没见过。这组工具有一些简单而有效的工具提示选项，很容易在你的网站上实现。尽情享受吧！

相关帖子:

*   [**10 jQuery 工具提示插件和教程**](http://www.jquery4u.com/plugins/10-jquery-tooltip-plugins-tutorials/)

## [1。砰！](http://pop.seaofclouds.com/)

享受简单的下拉菜单与流行！一个不引人注目的 JavaScript 插件。
![Pop.jpg](../Images/0336d59dcf1d0a47cbc5c1fd41c79274.png)
T3】来源

## 2.BetterTip

JQuery 库的一个插件，允许你创建自定义的工具提示。
![BetterTip.jpg](../Images/716870db5b1a31682c5c8bcdacf01e5d.png)
T3】

## 3.EZPZ 工具提示

非常简单的工具提示，周围有边框，它还可以跟随你的鼠标。
![EZPZ-Tooltip.jpg](../Images/2daef07cd1604c247db8e3870065ba24.png)

## [4。BsTip](http://benchsketch.com/bquery/index.html)

各种简单的工具提示，带有淡入淡出过渡效果和边框。
![BsTip.jpg](../Images/7ffe4c1a04c289c846e5bf127c59668d.png)
T3】来源

## [5。线索提示](http://plugins.learningjquery.com/cluetip/)

当用户的鼠标悬停在脚本中指定的任何元素上时，您可以轻松地显示精美的工具提示。
![clueTip.jpg](../Images/5df1eee58a4f66154a95cb8e064634ae.png)
T3】来源

## [6。内嵌 HTML 工具提示](http://www.javascriptkit.com/script/script2/htmltooltip.shtml)

允许您定义丰富的 HTML 工具提示，这些工具提示直接嵌入到您的网页中，当鼠标滑过页面上的链接时会出现。
![Inline-HTML-Tooltip.jpg](../Images/75ae166a1dd6142c756be9ea9eab5d22.png)
T3】来源

## [7。输入浮动提示框](http://sites.google.com/site/nicolaenamolovan/jquery.inputHintBox.html?attredirects=0)

有趣的输入字段圆角工具提示。当您单击输入字段时，工具提示淡入，当您单击其他地方时，工具提示淡出。
![Input-Floating-Hint-Box.jpg](../Images/39d6ed2ec41f6ab28a25fb3075019552.png)
T3】来源

## 8.轨道工具

使用轨道工具提示，您可以选择将工具提示放置在周围 360 度的任何对象上。Orbital Tooltip 拥有多种间距、偏移量和样式选项，远远超过其他 jQuery 工具提示。
![Orbital-Tooltip.jpg](../Images/5becb667dbf9899507ac72645e3edeca.png)

## 9.jGrowl 工具提示

一个 jQuery 插件，在浏览器中产生不引人注目的消息，类似于 OS X 的 Growl 框架的工作方式。
![jGrowl-Tooltips.jpg](../Images/ad85db4747e74cd3c88890531c47164d.png)

## [10\. vTip](http://www.vertigo-project.com/projects/vtip)

使用 jQuery 框架，任何具有 vtip 类的元素都将把它的 title 属性变成一个流畅的、可定制的工具提示，而没有大型工具提示脚本的笨拙和加载时间。
![vTip.jpg](../Images/63da008fafa4cbaba292deeed78fd09a.png)
T3】来源
