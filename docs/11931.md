# 2013 年 ie 浏览器会灭绝吗？2009 年更新

> 原文:[https://www . site point . com/internet-explorer-extended-by-2013-2009-update/](https://www.sitepoint.com/internet-explorer-extinct-by-2013-2009-update/)

现在，我在上次[发布关于这个](https://www.sitepoint.com/internet-explorer-extinct-by-2013/)的帖子时受到了一些抨击，但是，嘿，我是一个喜欢惩罚的人。在我最初的帖子发表一年后，我想我应该再检查一下，看看我的预测(尽管没有使用精确的数学)是否仍然正确——然后喘气！

使用 Internet Explorer 的 sitepoint.com 游客数量的下降速度并没有减缓。使用同样的线性方法，最新的统计数据仍然表明，到 2013 年，工业工程将从 sitepoint.com 消失…

但是等等——让我补充一下这个预测。

随着目前 Chrome 用户的增长，到 2011 年，使用 Chrome 访问 sitepoint.com 的人数将超过 IE。

![IE Safari Chrome Usage](../Images/2cfef2c3093440f4906ae3c5de8870de.png "IE Safari Chrome Usage")

让我回答一些无疑会出现的问题。

线性分析:是的，有更好的数学方法。回归分析会更准确；然而，它更加复杂和耗时。我想说明的只是一个持续的趋势，所以一个简单快速的分析就足以说明我想要展示的东西。

对于那些感兴趣的人来说，线性方程是使用[最小二乘法](http://en.wikipedia.org/wiki/Least_squares)的一条直线。它的缺点是只接受一个变量，忽略了其他可能影响结果的因素。[另一方面，回归分析](http://en.wikipedia.org/wiki/Regression_analysis)可以处理众多变量，包括因变量和自变量。

sitepoint.com 交通:我们有一个早期采用，技术观众，这意味着我们的浏览器统计数据不是一般网络的真实反映。也就是说，人们很容易低估科技人群对浏览器使用的影响。

如果你是一个网站所有者，我很想知道你是否在自己的网站上经历了类似的 IE 用户减少。
