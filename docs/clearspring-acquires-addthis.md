# Clearspring 收购 AddThis 以支持 Reach

> 原文：<https://www.sitepoint.com/clearspring-acquires-addthis/>

Widget 分发和广告平台 Clearspring 将于今天上午宣布，它已经收购了超受欢迎的社交书签 widget [AddThis](http://www.addthis.com/) 。Clearspring 是网络上最大的微件管理、分发和货币化平台之一，每月提供大约 40 亿次微件浏览。AddThis 提供了一个小部件，使内容发布者更容易鼓励他们的访问者通过电子邮件和社交分享网站来添加书签和分享内容。

AddThis 让内容发布者只需一个按钮就能在几十个社交网站上分享他们的内容，更重要的是，它还能为他们提供关于内容分享情况的统计数据。对于 Clearspring 来说，这次收购实际上是关于数据和覆盖范围的。

根据 AddThis 主页的数据，AddThis 按钮每月被点击超过 300 亿次——尽管 Clearspring 首席执行官 Hooman Radfar 告诉我，AddThis 每月服务 200 亿个页面，这显然是来自 comScore 的更保守的数字。Radfar 告诉我，Clearspring 和 AddThis 每月的独立用户数将达到 2 亿，这将使它成为 reach 排名前十的互联网公司。

收购的第二部分是在数据端。AddThis 正被大量高流量的网站使用，包括时代、甲骨文、娱乐周刊、Topix、MySpace、福克斯、ABC、CBS、美国偶像和 E！在线。它大概收集的关于内容如何在网络上共享的数据对营销人员有很多潜在的用途。Radfar 告诉我，他希望最终通过一个 API 来公开共享数据，让人们真正了解共享趋势。例如，关于巴拉克·奥巴马的文章和关于约翰·麦凯恩的文章被分享的频率有多高？哪些分享网站最受欢迎？最受各国欢迎？性别？主题？

拉德法尔说，这是未来的工作。更直接的是，Clearspring 希望创建 Radfar 所说的“开放、通用的共享平台”该平台将允许开发者和内容发布者在任何设备上向任何社交网站共享任何类型的内容——网页、应用程序或小工具，并获得关于这些内容如何交互和共享的统计数据。最后，Radfar 告诉我，他计划创建一个连接 API 的平台，尽管他对自己的意思肯定有点含糊。

交易条款没有披露。
