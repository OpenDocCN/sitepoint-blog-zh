# PHP 5.1 有什么新特性？

> 原文：<https://www.sitepoint.com/whats-new-in-php-51/>

PHP.net 周末宣布，PHP 5.1 (5.1.0)的第一个公开测试版现在可以在 PHP.net 下载。

那么有什么变化呢？

*   [PCRE 库](http://au2.php.net/pcre)(提供 Perl 风格的正则表达式支持)已经更新到 5.0 版本。
*   几项性能改进，包括对垃圾收集、 [switch 语句](http://au3.php.net/switch)和几个[数组函数](http://au3.php.net/array)的更改。
*   在 [PostgreSQL](http://php.planetmirror.com/manual/en/ref.pgsql.php) 和[中的一些改进改进了 MySQL](http://php.planetmirror.com/manual/en/ref.mysqli.php) 的支持。
*   新增了 [array_diff_key](http://au.php.net/array_diff_key) 、 [array_diff_ukey](http://au.php.net/array_diff_ukey) 、 [array_intersect_key](http://au.php.net/array_intersect_key) 和 [array_intersect_ukey](http://au.php.net/array_intersect_ukey) 几个函数。

PHP 5.1 可以从 PHP 网站[下载](http://www.php.net/downloads.php#v5.1)。

到目前为止，在 PHP.net 的 RSS feed 上有一个关于这次发布的条目，但是在 http://www.php.net/的首页上却没有任何关于这次发布的消息。
