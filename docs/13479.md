# 成功出版

> 原文:[https://www.sitepoint.com/publishing-for-success/](https://www.sitepoint.com/publishing-for-success/)

本周，我很高兴地宣布我的第一本电子书《外包 Web 项目:通向更聪明的商业的 6 个步骤》已经在 SitePoint 出版。这本书为任何需要将 web 开发工作外包给国内或海外供应商的人提供了一个成功的策略，并提供了各种技巧来帮助读者找到合适的供应商，获得优惠价格，正确完成工作，并与可靠的供应商建立长期关系。我真诚地希望这本书对读者有益，我鼓励任何可能有助于使它尽可能清晰和有用的反馈或建议。我还要感谢 SitePoint 的员工，感谢他们的支持和友好相处。

自从这本书发布以来，我收到了来自 SitePoint 用户的各种询问，大部分是关于我写这本书的动机的问题。简而言之，最初的动机是曝光、宣传，并帮助我不断努力将自己定位为外包和管理 web 项目领域的专家。按照安德鲁·奈特里希的“成为你所在领域的专家”的方法，我希望电子书最终将成为一种营销工具，推动销售。

现在，我发现自己处于一个独特的位置来检验这一营销策略——与通过广告或传统销售方法相比，通过将自己定位为所在领域的专家，你真的能获得更多业务吗？写一本电子书会比开展另一场 Adwords 活动带来更多销售线索吗？找出答案会很有趣，我已经在最初几天做了一些观察。首先，仅在过去几天里，我就收到了数量惊人的销售线索和询问。非常鼓舞人心！第二，我注意到这些线索往往比过去质量更高，通常与更大的工作或更多的咨询活动有关，而不是小型外包活动。也很鼓励！

希望这种趋势将继续下去，将自己定位为专家以创造销售的概念将成为赢家。最棒的是，在这个过程中，我没有真正的成本。相反，唯一的花费是我写内容的时间。即使我把我写这本书的时间按小时计算(考虑到这种努力的机会成本),如果最终有两三个新客户签约，我仍然会很高兴，而且获得这些客户的成本很可能比我们通常通过广告获得的成本要低。

因此，电子书营销实验慢慢展开，到目前为止一切顺利。创造有益于他人的有用材料，在此过程中获得报酬，并因此获得新客户，这当然比做广告预算和抱最大希望更好！在接下来的几个月里，我会让你们了解最新的进展，同时我也期待着你们的反馈。
