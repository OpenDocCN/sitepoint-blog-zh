# 电子邮件与直邮；中断与许可

> 原文：<https://www.sitepoint.com/email-vs-direct-mail-interrupt-vs-permission/>

在最后一篇帖子中，worchyld 问未经允许向潜在客户发送邮件是否应该被视为垃圾邮件，更不用说访问他们是否也是一种入侵。

问得好。

有些奇怪的是，75 年来，直接营销世界成功地向消费者发送了未经请求的邮件(垃圾邮件)并收到了回复。虽然有关于垃圾邮件和电话营销电话(住宅)的法律，但没有法律禁止我们向某个名单发送邮件。

话虽如此，你的直邮广告必须有效。在销售专业服务的情况下，这意味着不要推销你是谁和你做什么。

相反，它意味着为你的观众提供免费的、有价值的信息和教育。例如，我的一个客户正在投资目标市场的直邮广告，他提供了一份关于行业面临的挑战以及如何应对这些挑战的专门的、信息丰富的白皮书(部分通过实施他的解决方案)。然后，他跟进以获得关于那篇论文的建议——改进它，帮助收件人了解他们的立场。

类似地，另一个客户正在向他的目标市场发送一系列 4-5 封教育信(同时也通过亲自拜访和电话跟进)。每封信都告知市场一个关键问题，以及如何解决这个问题。例如，一封信向收件人介绍了让他们更容易跟踪项目的新方法。另一个人用该行业的人使用的语言谈论技术如何帮助他们发展特定的业务。另一个描述了业内人士用他们的技术制造的 5 个致命缺陷。等等。

在这两种情况下，直接邮寄是不够的。跟进、建立融洽的关系，并在潜在客户心目中占据首要位置也很重要。

同样，你们中的许多人会讨厌这种方法，这没关系。这并不适合所有人。它确实使用了打断策略——尽管是通过教育潜在客户、逐步展示价值和建立信誉/信任。然后，随着时间的推移，潜在客户将更有可能给予许可，并愿意接受更多的信息和教育。
