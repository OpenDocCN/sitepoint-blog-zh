# ColdFusion 周:1 月 23 日至 29 日:让 CF 开发者崭露头角

> 原文：<https://www.sitepoint.com/the-week-in-coldfusion-23-29-jan-putting-cf-developers-on-the-map/>

想知道 ColdFusion 的开发者都在哪里？CFMaps 是一个雅虎！Maps mashup 允许 ColdFusion 开发人员将自己放在地图上，根据他们的经验水平进行颜色编码。现在，大多数注册在北美——所以如果你是一个 CF 男孩或女孩，为什么不把你自己添加到地图上呢？我们的人数肯定比目前注册的人数多得多，而且我知道在新西兰不止有一个 ColdFusion 开发人员！

将于五月在 DC 举行的网络狂人大会已经开始报名。演讲者包括冷聚变绝地大师雷蒙德·卡姆登、戴夫·沃茨、道格·休斯、约切姆·范·迪滕和乔·莱因哈特，还有许许多多其他人。CFUnited 大会——也在 DC 华盛顿举行，但在 6 月——继续他们的播客系列，在 Fusebox 框架上采访凯文·罗奇[。那么，一个不在美国也不在 DC 的会议怎么样？](http://cfunited.com/blog/index.cfm/2008/1/28/CFUnited-Podcast-Interview-with-Kevin-Roche--Fusebox-framework)[苏格兰威士忌加冰块将于 6 月在爱丁堡](http://www.scotch-on-the-rocks.co.uk/)举行，门票现已发售。

同样在播客上，[这位数字媒体人采访了 CF 大师本·福塔，作为“会见专家”播客系列](http://www.thedigitalmediadude.com/2008/01/29/tdmd-meet-the-experts-podcast-9-ben-forta-adobe/)的一部分。本讲述了他的背景，为什么 ColdFusion 优于 PSP、ASP.NET 或 JSP，以及即将推出的第 9 版(“Centaur”)。

雷蒙德·卡姆登(Raymond Camden)本周宣布，Adobe 系列产品的开源项目网站 RIAForge 已经达到了 400 个托管项目的里程碑。RIAForge 包括基于每种 Adobe 技术的项目，从 Flex 和 AIR 到 Illustrator 和 Photoshop，尽管迄今为止大多数项目都与 ColdFusion 相关(目前有 296 个)。看到这个数字后，我决定与其他开源项目托管网站做一个比较——在 Google Code 上找到了 [60 个 ColdFusion 项目，在 SourceForge](http://code.google.com/hosting/search?q=ColdFusion) 上找到了 [95 个项目。代码太多了！](http://sourceforge.net/search/?type_of_search=soft&type_of_search=soft&words=coldfusion) [Brian Rinaldi 通过他的开源更新来跟踪开源发布、更新和公告](http://www.remotesynthesis.com/blog/index.cfm/2008/1/28/ColdFusion-OpenSource-Update--January-28-2008)。

最后，本周，一点代码:史蒂夫纳尔逊一直忙于让自己沉迷于谷歌。为了使用 Google 电子表格和日历 API,他写了一系列关于[认证的文章，目前已经有四个部分。听起来，Google API 可能有点难，但是 Steve 已经为我们做了艰苦的工作。](http://www.webapper.net/index.cfm/2008/1/24/Google-API-The-cool-way-to-Login-Part-1)

我错过了你认为重要的事情吗？这周还听到其他与 CF 相关的好消息吗？留言评论！
