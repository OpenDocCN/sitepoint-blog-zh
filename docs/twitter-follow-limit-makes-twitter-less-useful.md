# Twitter 的关注限制让 Twitter 变得没那么有用

> 原文：<https://www.sitepoint.com/twitter-follow-limit-makes-twitter-less-useful/>

为了遏制垃圾邮件，上周末 Twitter 决定在其账户中添加[关注并更新限制](http://help.twitter.com/index.php?pg=kb.page&id=242)。这些限制如下:

*   在任何和所有设备上，每天总共更新 1，000 次
*   每天在任何和设备上总共发送 250 条直接消息
*   每小时 100 个 API 请求
*   一天中尝试关注的最大次数

一些人也遇到了不同的关注限制，这些限制限制了人们可以关注的朋友数量。最常见的数字似乎是 2000 个追随者，但这因人而异，部分取决于一个人拥有的追随者数量。

上周四， [Twitter 谈到了](http://blog.twitter.com/2008/08/making-progress-on-spam.html)引入 follow caps 的想法。“不管怎样，合理的人数是多少？公司联合创始人伊万·威廉姆斯在博客中写道:“大多数用户可能很难找到 500 个他们感兴趣的账户，而其他人会认为 1 万个的限制太低了。”“关键是，没有对错。没有完美的公式。我们尽最大努力采取多维度的方法。在应用限制之前，我们会考虑很多因素，包括有多少人在跟踪你。”

关于关注和更新限制的 Twitter 知识库页面提醒用户，Twitter 一直包括限制，从帖子的 140 个字符限制开始。“Twitter 一直是关于限制的，信奉约束激发创造力的理念，”它说。然而，尽管极限确实可以激发创造力，但是组建一个聊天小组并没有太多的创造力，所以我们不确定这个咒语在这里是否适用。

我以前的同事 Marshall Kirkpatrick 曾经说过一句名言:Twitter 正在为他支付租金。他这么说的意思是，他已经在 Twitter 上围绕自己建立了一个足够大的社区(他关注了 2，100 多人，被关注的人超过 4，200 人)，这样他就可以很容易地从他的联系人中获取故事，并依靠 Twitter 来帮助进行研究。如果 follow 限制阻止 Kirkpatrick 向他的社区添加有价值的贡献者，那么它将极大地削弱 Kirkpatrick 从服务中获得最大收益的能力。

从 Twitter 上许多[顶级关注者的关注者与被关注者的比例来看，他们中的很大一部分可能会受到影响，显然许多(大多数)不是垃圾邮件发送者。例如，美联社看起来像是](http://www.twitterholic.com/top100/friends/)[遇到了 2000 个追随者的限制](http://twitter.com/apnews)。

根据伊万·威廉姆斯的说法，限制是一项正在进行的工作，根据 Twitter 知识库，未来将会施加更多的限制。
