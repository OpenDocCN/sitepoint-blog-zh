# 在 Web Directions South 向大师学习 JavaScript

> 原文:[https://www . site point . com/learn-JavaScript-from-the-master-at-web-directions-south/](https://www.sitepoint.com/learn-javascript-from-the-master-at-web-directions-south/)

虽然我们可能生活在一个看似与世隔绝的小岛上，但由于网络的力量，澳大利亚的网页设计师和网页开发人员与他们的同行完全没有联系。

事实上，网络专业人士在世界上最大的会议之一实际上是每年在悉尼举行的会议，[网络方向南方](https://south08.webdirections.org/)。Team SitePoint 从第一天起就是这个会议系列的[固定](https://www.sitepoint.com/sitepoint-at-web-directions-south-webjam/) [出席者](https://www.sitepoint.com/web-directions-ux-making-your-users-feel-special/)[支持者](https://www.sitepoint.com/sitepoint-at-web-directions-south/)和[媒体合作伙伴](https://www.webdirections.org/uncategorized/announcing-the-sitepoint-closing-party-for-web-directions/)。今年的会议将于 9 月 25 日至 26 日举行(23 日和 24 日有两天的可选研讨会)。

## SitePoint 节目

今年的会议对 SitePoint 来说尤其重要，因为我们有大量的博客作者、图书作者和员工参加会议并发表演讲:
![](../Images/2a775eda241d474a7f8e93294f284bc2.png)

*   SitePoint 总部的系统管理员 Lucas Chan 将在 T2 主持一个两个小时的会议，比较和对比四个 web 开发框架: Ruby on Rails、Django、CakePHP 和 Struts/Spring。
*   《CSS 的[艺术&科学](https://www.sitepoint.com/books/cssdesign1/)的作者纪娜·波顿正在谈论如何创建性感的样式表。![](../Images/c78b88ff3028eefa0d42b357b28516b7.png)
*   我们的常驻记者 [ColdFusion 博客](https://www.sitepoint.com/the-week-in-coldfusion-2-8-july-object-oriented-cfml-for-fun-and-profit/)Kay Smoljak 正在谈论如何开始并运营一项成功的网络开发业务。
*   我们的 Ruby on Rails 博客作者 Myles Eftos 正在谈论 Oauth 和 OpenID。
*   曾经的 SitePoint Rails 博客作者 Tim Lucas 正在做一个关于 iPhone 开发的演讲。
*   [David Peterson](https://south08.webdirections.org/?cat=1#post-51) ，我们的另一位博客作者和[语义网专家](https://www.sitepoint.com/preparing-your-sites-for-the-data-web/)，正在谈论将语义网用于分布式社交网络。
