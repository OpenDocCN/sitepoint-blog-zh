# 嗨，我的名字是…

> 原文:[https://www.sitepoint.com/hi-my-name-is/](https://www.sitepoint.com/hi-my-name-is/)

让我自我介绍一下，我叫克里斯·伯克斯，是的，你可以叫我克里斯。自 2001 年 11 月以来，我一直是 SitePoint 的会员。我也是论坛的导师，写了一两篇文章。我有机会在 SXSW (2007)上遇到一些 SitePoint 帮的成员，他们不知何故说服我为他们写博客。

早在 1996 年，当我们把互联网上的每一页都称为“网站”时，我就开始涉足 web 开发作为一种爱好。我开始得太早了，以至于我不知道有谁能教我我需要知道的东西，所以我最终成了一个多面手。我做过各种事情，从建立互联网服务提供商和虚拟主机公司，到在各种当地公司做设计师或开发人员，最近我开了自己的公司，一个博客网络。

作为一名程序员和开发人员，我的技能是 PHP 和 MySQL，尽管我很擅长 Perl、Python、Shell 脚本、JavaScript 和其他一些必要的零碎东西。我不假装在这些事情上是最好的，但是我完成了工作。我对 Photoshop 也很在行，尽管我不是平面设计师。我对 Web 标准又爱又恨:我喜欢 CSS/XHTML，但它讨厌我。

虽然我会对上述主题做出贡献，但我认为我最近的经历是试图通过各种方式(CPM，CPC，直接广告销售，赞助等)经营网络业务并赚钱。)将会引起几乎所有想在互联网上谋生的人的特别兴趣。这是一段有趣的旅程，我才刚刚开始。

我希望通过 SitePoint 更多地了解你，我随时欢迎提问和故事建议。如果我不知道答案，我会尽我所能找到它。不要见外，我希望你经常回来看看。
