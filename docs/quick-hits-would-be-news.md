# 快速点击，会是新闻

> 原文：<https://www.sitepoint.com/quick-hits-would-be-news/>

今天是星期五……让我们总结一下本周在混乱中丢失的一些东西，有些是理所应当的。

谷歌宣布降低他们最近收购的顽童点播网络分析产品的价格。这可能会影响到这个博客的至少 2-3 个读者。他们可能已经知道了。

Hitwise 推出了一种叫做关键词智能的新服务。它仍处于测试阶段，还不可用，但如果你想去关键字智能网站，你可以尝试给他们一个电子邮件地址(我的失败是由于某种不明智的 DNS 查找地址验证方案)，以便在他们启动服务时接收信息。我没有这方面的进一步信息，但我正在努力。

“互联网营销大师”们目前正忙着开发 Googspy，当它真正可用时(他们的服务器可能已经被机器人敲死了)，这将是一个很好的免费竞争情报工具的开始。查看竞争对手，看看他们的搜索排名和广告。

既然 Stephen Pierce(“智能页面”门户页面模板的创建者)和其他人刚刚宣布了某种新的搜索引擎垃圾邮件工具，那么市场领域的 Googspy 狂潮应该很快就会平息。我有一个模糊的暗示，它可能使用某种自动开发的博客(也许抓取 RSS 提要，也许 trackback 垃圾邮件，我不知道)吸走搜索引擎流量，诱捕搜索者，让他们点击 Adsense 广告。再次感谢谷歌资助自动垃圾邮件。

SEO 浏览器不是一个网络浏览器，而是一个网站……提供了一个很酷的快速浏览网页的工具，有点像搜索引擎看到的。不要纠结简单模式，直接进阶。你是先进的，不是吗？我想是的。如果没有，继续关注这个空间，我们一起到达那里。
