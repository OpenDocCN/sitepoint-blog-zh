# 伟大的摄影比赛…还有呜呜祖拉

> 原文：<https://www.sitepoint.com/the-great-photography-competition-and-vuvuzelas/>

有几个星期，我被写一些东西的需要所征服。我一想到这些话，就脱口而出。其他几周，我坐在电脑前，苦思良久，不知该写些什么。我怀疑你可以通过阅读我的帖子来判断这是一个什么样的星期。我的感觉似乎与社区中发生的事情没有任何直接关系，但它会影响这些事情是主要主题(关于“想不出主题”周)还是次要主题(关于“有更多要说的，我可以合理地期望人们阅读“周”)。

在过去的一周里，我们经历了很多事情。自从我开始担任这个职位以来，我第一次意识到，有比我可能期望完成的更多的工作。出于这个原因，你会认为我会有很多东西要写——我确实有——但出于某种原因，我似乎找不到一根线将它们联系起来。这一周对我来说最突出的两件事是时区计算和呜呜祖拉的声音。

![SitePoint Forums](img/b246a466b5fbad22ce55d347d77e745e.png "sitepoint-forums")

计算的原因是[伟大的摄影比赛](https://www.sitepoint.com/forums/showthread.php?t=686399)。(我想我不必解释呜呜祖拉。)为了庆祝我们最新的书 *[网络摄影](https://www.sitepoint.com/books/photography1)* 的发布，我们决定做点什么来给我们的[摄影论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=302)注入一点活力。我完全被我们收到的回复震惊了，尽管我试图强制执行涉及复杂时区计算的提交时间，这让所有人都很困惑。有一些令人难以置信的照片正在提交。当我发布这篇文章的时候，提交图片已经太晚了，但是查看一些令人惊叹的作品绝对不会太晚。看看这些…

第一天，我让人们提交一张快餐店的图片或者一组办公用品的图片。

在第二天，受试者要么是公交车或出租车，要么是邮箱。

第三天是关于雕像和涂鸦的。

在第四天，人们必须拍摄一只昆虫或者三种水果的组合。

在最后一天——今天——选择是[一群在公共场所的人或者一个老人](https://www.sitepoint.com/forums/showthread.php?t=688662)。

到目前为止，我们已经有 4 名获奖者，今天晚些时候还剩下一名获奖者，一名大奖获得者的作品将在本博客中展示。我很期待。

其他一些正在进行的事情包括我们新的[移动设备开发](https://www.sitepoint.com/forums/forumdisplay.php?f=340)分论坛。必须与时俱进…

由于大众的需求，我们已经在论坛上设置了[报告私人信息](https://www.sitepoint.com/forums/showthread.php?t=687589)的功能。如果您不幸收到了垃圾邮件(或出于任何其他原因与您有关的邮件),请举报，以便版主可以为您处理。

我们在最近的脸书病毒活动中取得了一些进展，我非常兴奋能够向你们展示我们用每个参与者的脸制作的拼贴画。

**本周热门话题**

在 [HTML 和 XHTML](https://www.sitepoint.com/forums/forumdisplay.php?f=52) 论坛上有一些极客的言论。如果你对 [HTML 4.01 过渡 DTD 是否是答案](https://www.sitepoint.com/forums/showthread.php?t=688558)感兴趣，那就过来说出你的想法。

有趣的是，[普通聊天](https://www.sitepoint.com/forums/forumdisplay.php?f=9)中的话题都是关于西班牙和……[呜呜祖拉](https://www.sitepoint.com/forums/showthread.php?t=688724)的。

在[商业和法律问题](https://www.sitepoint.com/forums/forumdisplay.php?f=61)中有一个非常有趣的线索，关于如果客户认为你没有做足够的工作来证明你的发票是正确的，你是否应该[改变你的发票](https://www.sitepoint.com/forums/showthread.php?t=688713)。我不会…你会吗？

好吧，我想我毕竟有很多话要说…下周见。
