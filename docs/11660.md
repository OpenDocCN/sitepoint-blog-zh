# 两种非常小的字体

> 原文:[https://www.sitepoint.com/two-teeny-tiny-fonts/](https://www.sitepoint.com/two-teeny-tiny-fonts/)

**![](../Images/9534be788026959ebf74904d6fe51e6b.png)子像素着色**是一种用于提高 LCD 显示器表观分辨率的方法。LCD 屏幕上的每个像素由单独的红色、绿色和蓝色子像素组成。子像素着色允许改善字符形状和间距，并且几乎完全消除了位图图像中出现的“锯齿”。我为什么要提这个？

嗯，在 Typophile 上，有一个有趣的博客帖子，作者是 Miha T1，展示了他创造的一种新的微型字体。它被称为蜜蜂的膝盖，x 高度为 3 像素，设计时考虑到了网站图标。下面你可以看到实际大小的字体，它没有缩小。

![miha](../Images/597ce2b6d50219fc96365a89e1caea22.png "miha")

如果你仔细观察这些微小的字体，你可以看到颜色的变化。下面你可以看到字体放大了 1600%,并且可以看到每个像素的颜色。

![ipsumdolor](../Images/ac0231666b62b0db74126fdd52705699.png "ipsumdolor")

这种字体的神奇之处在于，它实际上在 3 个像素高时清晰可辨。

 ****另一种微小字体**

在 Typophile 博客帖子的讨论中(那里正在进行一些冗长而复杂的讨论)，出现了另一种亚像素字体。它是由肯·佩林设计的，你可以在下面看到它的实际尺寸。那是在那个小盒子里的美国独立宣言的前五百个单词。

![KenPerlinFont](../Images/c30fb014231f089fa99d89f65bcc71a7.png "KenPerlinFont")

他为什么做这个字体？

“我的目标是有一个清晰可读的东西，而且可以在 QVGA (320×240)屏幕上显示整个页面的文本(例如:苹果 IPod 或 T-Mobile SideKick)。相比之下，我注意到微软最小的屏幕字体是不可读的，而他们最小的可读屏幕字体比必要的要大得多。”

在撰写本文时，这两种字体都还没有可供下载，但我认为它们看起来很有趣，我想 Miha 最终会在完成实验后推出他的三像素高的小字体。
