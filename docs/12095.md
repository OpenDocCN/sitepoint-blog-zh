# RIP geo cities 1995–2009

> 原文：<https://www.sitepoint.com/rip-geocities/>

让我们默哀两分钟:GeoCities 已经死了，最终被它的所有者雅虎埋葬了。

GeoCities 差不多有 15 年历史了。用互联网的话来说，它诞生于 1995 年下半年大爆炸后不久。它的创建者大卫·博尼特和约翰·雷兹纳设想了一个虚拟社区，网页托管在网络城市和社区中。该系统获得了巨大的成功，并迅速吸引了大量的追随者。在巅峰时期，创建了数百万个页面，而在网络的早期，所有其他网站似乎都托管在 GeoCities 上。

这并不是说 GeoCities 网站很好。大多数照片糟糕得令人震惊，背景和广告也无济于事。尽管如此，受欢迎程度继续增长，在 1999 年互联网繁荣的高峰期，雅虎以 36 亿美元收购了该公司。没错:36 亿美元。当时，GeoCities 托管着 350 万个网站，每个网站的价值都超过了 1000 美元！(我们现在可能会笑，但不要忘了脸书和 Twitter 可笑的估值……)

不久之后，GeoCities 开始显示出疲态。侵入性广告、糟糕的开发工具以及廉价主机计划的引入使得 GeoCities 不再是一个有吸引力的选择。博客平台和社交网络(如 MySpace)的兴起使得创建网站和接收用户生成的内容变得更加容易。

即使在 GeoCities 不再流行之前，该公司从未盈利，雅虎也没有试图出售该系统就终止了业务。新账户注册已经停止，旧网站将在年底前消失。只剩下几个月的时间来狂欢在[网](http://www.geocities.com/feyphouka/)上的一些[的网站。](http://www.geocities.com/the4thpip/)

你会为地球流泪吗？这是一个互联网时代的终结吗？还是应该在几年前就灭绝了？
