# 谷歌在浏览器中提供 3D 渲染

> 原文：<https://www.sitepoint.com/google-3d-browser-plugin/>

谷歌发布了一个新的实验性浏览器插件，它可以直接在浏览器中显示详细的图形 3D 对象和环境并制作动画。

基于 3D 浏览器的技术似乎正在兴起。Mozilla 正致力于一个类似的 3D 插件项目，ID 软件最近发布了 [Quake Live](https://www.sitepoint.com/quake-live-gaming/) ，这是他们开创性游戏的网络版本。

谷歌的项目提供了一种额外的方法，该公司希望谷歌地球等应用程序最终将直接从网络上运行。他们的开源跨浏览器插件直接利用硬件加速渲染。开发人员可以使用 SketchUp、3ds Max 和 Maya 支持的开放标准 COLLADA 格式导入 3D 对象。然后可以使用 JavaScript API 操作这些对象。

虽然这提供了另一种 3D 技术，但谷歌承认，标准化的 3D web API 可能还需要几年时间，开发者应该只在早期阶段使用 O3D 进行实验。然而，他们怀疑 O3D 和 Mozilla Canvas 3D 扩展最终会融合成一个统一的平台。

## 你看过 3D 吗？

基于浏览器的 3D 并不新鲜。在 20 世纪 90 年代中期，VRML(虚拟现实建模语言)被认为是网络上的下一个大事件，但是这项技术从来没有像预期的那样普及。

开发 3D 应用程序并不容易，但也许这些项目将凸显 3D 网络环境的潜力？网络用户是否能看到好处，并准备好跳出 2D，则是另一回事。
