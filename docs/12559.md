# 用 Web 2.0 追踪美国大选

> 原文:[https://www . site point . com/tracking-the-us-election-with-we b-20/](https://www.sitepoint.com/tracking-the-us-election-with-web-20/)

在为 SitePoint 和 ReadWriteWeb 撰稿之前，我为一个相当大的政治博客撰稿。因为，我最喜欢写的东西之一是政治和技术的交集。在加入 SitePoint 后的几个月里，我已经把这个角度缩小了很多，因为我知道我这里的观众更加全球化，你们中的许多人真的不关心美国政治。但是，由于今天是美国的选举日，而且是在美国历史上最长的总统竞选之一之后，无论谁获胜(第一位非洲裔美国总统或第一位女性副总统)，这都将创造历史，并且因为这是第一次网络真正成为总统政治的一个主要因素——我想请你们原谅我一会儿。

虽然霍华德·迪恩在四年前竞选总统时可能是第一个展示互联网在政治中的重要性的候选人，但直到今年，竞选活动才真正开始弄清楚如何使用互联网。两个最好的例子可能是罗恩·保罗——他的基层支持者利用网络以他的名义组织了大规模的筹款活动，并集中他们的资源主导了社交媒体报道——和巴拉克·奥巴马，他利用网络建立了政治分析家称之为历史上最有效的竞选组织之一。

因此，我们可以通过网络进行多种选择。今年，各大媒体通过网络进行了大量的现场报道。ABC 新闻和[政治](https://www.ustream.tv/channel/politico-tv2)将通过 Ustream 直播。电视，而[《今日美国》](http://www.mogulus.com/usa_today_election_2008)和 C-SPAN ( [麦凯恩](http://www.mogulus.com/cspan_electioncam_mccain)和[奥巴马](http://www.mogulus.com/cspan_electioncam_obama))正在使用 Mogulus，例如。

美联社(Associated Press)将首次通过流媒体视频对选举进行现场报道，而且从美国有线电视新闻网(CNN)、T4 时报(New York Times)和谷歌(Google)都可以看到精彩的选举结果仪表盘。

但是 web 2.0 是关于人的力量的。因此，利用网络进行选举报道最有意思的地方是那些网站和项目，这些网站和项目汇总了现场的人们——真正的选民——要说的话。

**[GroundReport](http://www.groundreport.com/)** 是一个获得 CC 许可的公民新闻平台，人们可以在这里发布他们目睹的新闻的视频、照片或文本记录。我相信今天会有很多关于选举的报道。

**[社交|中位](http://election.socialmedian.com/)** 有专门的选举页面设置，人们可以提交选举新闻并对其进行讨论和辩论。

Techmeme 的政治对手 Memeorandum 密切关注着政治博客圈，可以理解，今晚的选举新闻将会主导政治博客圈。通常情况下，博客比主流电视网抢先一步，所以请关注 Memeorandum，了解最新的主要趋势和故事。

Digg 有一个选举类别已经有几个月了，今天期待有趣的选举相关新闻快速涌现。

**[Twitter 投票报告](http://blog.twittervotereport.com/)** 是一个很酷的项目，通过 Twitter 监控全国各地的投票站。如果有任何可疑的欺诈行为，请首先在这里听到。

如果你更喜欢视觉，YouTube 是个不错的去处。他们与 PBS 合作的“videoyourvote”频道鼓励人们拍摄投票站的视频，以减少不正当交易的可能性。当有如此多的眼睛在现场记录全国各地发生的事情时，要成功地进行投票欺诈变得更加困难。

Twitter 本身有一个专门的选举页面，可以发布关于候选人和选举的推文。一些人甚至[利用推特](http://mashable.com/2008/11/04/election-twitter/)试图预测选举结果。

Friednfeed 的美国政治室是政治迷们谈论、分享信息和辩论今晚选举的热门去处。对于那些厌倦了选举噪音的人，试试[禁止政治](http://friendfeed.com/rooms/no-politics-allowed)房间，那里有一些可爱的猫图片。

当前电视 今晚有一个特别的选举节目，由 Digg 和 Twitter 联合主办，视频来自 12seconds.tv。他们承诺用户指定的体验，将“最好的社交媒体与最重要的结果、事实和信息的实时广播”结合起来

**[Tweet Row to History](http://obamaftw.org/)**是我们名单上唯一的党派网站，但对任何一位候选人的支持者来说，观看它都应该很有趣。巴拉克·奥巴马随机挑选了 10 名捐赠者坐在他今晚胜利/让步演讲的前排。他只是碰巧选择了 Rails 性能监测公司 [FiveRuns](http://www.fiveruns.com/) 的 Bruce Williams，他和他的客人 Damon Clinkscales (Austin on Rails，Snaptweet)通过 Twitter 建立了一个快速网站来实时捕捉整个体验。
