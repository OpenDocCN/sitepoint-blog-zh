Rafael，首先让我欢迎你来到服务器端编程的世界！有许多优秀的语言可供你学习，ASP 绝对是一个值得选择的语言。如果 ASP 在任何一点上被证明是太多了，你可能也想看看 PHP 或 ColdFusion，因为你确实想选择一个适合你的需要和你舒适的。

也就是说，这与其说是 ASP 的问题，不如说是 web 服务器的问题。个人网络服务器(PWS)是一个很好的开发平台，而你似乎对 IIS 很迷茫。让我来减轻你的一些痛苦:它们几乎是一样的。PWS 只是 IIS 的简化版本，许多开发人员在将他们的网站转移到 IIS 之前，在 Web 开发的早期阶段使用它，就像您所做的一样。

我认为这种情况下的问题可能是 ASP 可能还没有在您的 IIS 副本上启用。如果它是在一个网站托管账户上托管的，你需要请求主机在你的网站上启用它，或者更换主机。如果服务器是公司服务器，请与 MIS 联系，确保他们正确安装了软件。

另一方面，如果您实际上正在运行服务器，但它仍然不工作，让我们运行一些快速测试。创建此 test.asp 页面:

```

%>
```

##### 人民的真相

医生:我很想知道一个简单的 pop under solution 的内幕。我的客户希望在他们的内部网中使用它，以此来提醒员工做显而易见的事情。标记

windows 下的 Pop 经常在“隐藏广告”中使用，是一个很有价值的工具。创建它们并不比创建自动弹出窗口更困难——我们只需在窗口下的弹出窗口中添加一行代码。

首先，让我们去 www.builder.com 制作一个 Popop 窗口脚本。我们将这段代码放在文档的 `<head>`和`</head>` 部分之间:

```
<script language="JavaScript">    

<!-- hide from JavaScript-challenged browsers    

function openWindow(url, name) {    

 popupWin = window.open(url, name, 'width=400,    

height=600,left=100,top=100')    

}    

// done hiding -->    

</script>
```

下一步是在页面加载后立即弹出该窗口，如下所示:

```
<body onLoad="openWindow('http://www.sitepoint.com','newwin');">
```

既然我们已经自动打开了窗口，我们需要确保我们的新窗口隐藏自己。只需将这行代码添加到弹出窗口中，就大功告成了！

```
<body onLoad="window.blur();">
```

##### 纠正:挑剔的框架

上周我介绍了如何使用 JavaScript 进行多帧切换。首先，JavaScript 必须包含在文档头中，就像这样:

```
<html>    

<head>    

<script language="javascript" type="text/js">    

<!--    

function doSwitch(frame1,frame2,frame1loc,frame2loc)    

{    

top.frame1.location.href=frame1loc;    

top.frame2.location.href=frame2loc;    

}    

// -->    

</script>    

</head>    

<body>    

<a href="javascript: doSwitch('main','top',    

'main.asp','top.asp');"> test</a>    

</body>    

</html>
```

我还放错了一部分已经调好的`<a href="href"` …。您的电话线路现在应该是这样的:

```
<a href="javascript: doSwitch('main','top',    

'main.asp','top.asp');"> test</a>
```

本质上，这是如何工作的，我们正在调用期望 4 个参数的`doSwitch()`函数。其中 2 个是我们将放置 2 页的框架的名称。另外两个是两页的文件名。因此，通过将这些参数更改为不同的值，我们可以在几个帧中多次重用我们的代码:

```
<a href="javascript: doSwitch('main','top',    

'main.asp','top.asp');"> test</a>    

<a href="javascript: doSwitch('banner','main',    

'banner2.asp','main2.asp');"> test</a>
```
