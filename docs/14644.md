已经有一些人来找我寻找博客脚本和预建的应用程序，所以我认为发布一些我为此找到的资源会很有用。请记住，虽然我在 SitePoint 上维护博客，但我并不热衷于博客，因此对博客及其术语没有什么经验。所以，当涉及到与可移动类型接口时，我不会有太多的帮助…

*   对于 CF5/IIS/Fusebox 3 有 [FuseBlog](http://www.riahsoftware.com/fuseblog/index.cfm?fuseaction=fuseblog.Download) 。它允许张贴，编辑和删除条目，RSS 0.92 饲料，类别，档案，多用户，搜索和更多。
*   [CFXML_Blog](http://www.veggiani.it/projects/cfxml_blog/) 是 SourceForge 中可用的开源项目。它完全基于 XML，所以不需要数据源。它提供了完整的 UNICODE 支持，类别，日历，视觉主题可通过 CSS 文件编辑，XHTML 和 CSS 兼容，最近增加了一个 XHTML 兼容的所见即所得编辑器。
*   CF_Blogger 也是免费提供的，具有存档、分类、书签等功能，并且能够添加/编辑/删除以上所有功能。

我也在想，我六月份的文章将会是关于如何创建你自己的博客系统的。我个人认为“如果你想把事情做好，你必须自己去做”。这并不是说你不应该使用上述系统之一。但我经常发现，如果我想要一个完全符合我需求的东西，唯一的办法就是自己编码，内置我想要和需要的所有特性。因此，我将在六月份写一篇文章，重点是构建你自己的定制博客系统。尽情享受吧！
