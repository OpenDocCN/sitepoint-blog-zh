# 如何使用 Twitter 进行客户服务

> 原文：<https://www.sitepoint.com/twitter-for-customer-service/>

企业和专业人士使用 Twitter 的最佳方式之一是作为一种客户服务工具。越来越多的公司开始监控他们在网上的声誉(T2 ),并回复推特上发布的客户询问，成功做到这一点的公司可以在很多方面受益。

虽然在 Twitter 上与客户互动时，大型知名公司是最引人注目的，但客户服务可能是每个专业人士的关注点，无论你是已经成立的小企业还是自由职业者。

## 社交媒体中客户服务的好处

Twitter 上以客户服务为导向的成功存在可以带来许多积极的好处。以下是一些需要考虑的问题:

**找出你的客户真正想要什么**

从 Twitter 这个客户服务工具中获取价值的一个关键方法是倾听。如果你不仅能够跟踪与你的特定业务相关的对话，而且能够跟踪潜在客户之间讨论他们需求的对话，你将获得宝贵的洞察力，了解客户真正想从你这里得到什么。

这可以在你的营销和销售工作中发挥巨大作用，因为你可以定制你的产品，以满足你在 Twitter 上看到的讨论的个人需求，并花时间关注当前客户在你自己的互动之外所说的话。

**让你的客户感受到价值**

对于不开心的顾客来说，没有什么比感觉没人关心更糟糕的了。当你投资一项产品或服务时，你对你将得到的回报有一定的期望。当这些期望没有得到满足时，你最终会尝到苦果。

如果你的客户直接转到 Twitter 来表达他们的问题，而不是先找你，这已经是一个严重的问题了。但是在日常生活中，你可以观察你的客户在说什么，并学会在他们告诉你之前预测他们的需求。这种预防性的客户服务将创造忠诚和长期的客户关系。

**快乐的客户=推荐人**

口碑营销的强大是众所周知的事实。如果你的客户对你提供的服务非常满意，你可能会看到推荐的增加。如果你的客户倾向于赞美你，Twitter 是一个很好的出口。如果他们已经习惯了在 Twitter 上与你交流，你就能从中受益。通过在 Twitter 上转发和公开支持你的客户，你将增加他们采取互惠行动的可能性。

## 如何通过 Twitter 提供卓越的客户服务

**使用监控工具**

为了在 Twitter 上提供客户服务，你需要知道人们在谈论你、他们的需求以及你所在的行业。这可以通过定期检查各种资源轻松实现，包括:

*   [Monitter](http://monitter.com/)
*   [TweetBeep](http://tweetbeep.com/)
*   [推特搜索](http://search.twitter.com/)
*   [TweetScan](http://tweetscan.com/)

有关各种社交媒体的在线声誉监测的更多信息，请参见:[在线声誉管理:16 种免费工具](https://www.sitepoint.com/tools-manage-online-reputation/)。

**反应迅速……而且要快**

因为 Twitter 是一个公共论坛，快速行动是必要的，尤其是在有人对你的服务表示不满的情况下。你应该承认这个问题，并尽快提出解决方案。

**观察不满意的地方**

当你监控 Twitter 时，特别是用关键词，注意那些关注你提供的服务类型的沮丧和不满的推文。虽然不开心的客户可能不是你的客户，但留意投诉可以帮助你识别出潜在客户。如果他们不满意的原因是你擅长的领域，他们可能会欢迎并感谢你向他们伸出援手。
