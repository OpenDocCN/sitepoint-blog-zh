## 您好 bjrgvin，祝贺您的在线业务不断增长！你的网站对你的成功有什么贡献？

我的主要商业网站是 [Audio Issues](http://www.audio-issues.com) ，在那里我帮助家庭录音室音乐人通过教他们如何录音、混音和制作来让他们的音乐产生更大的影响。这是我发布所有内容和收集电子邮件订户的地方。它基本上是我的商业中心——我带领大家去的地方。

我花了很多精力试图将流量转化为订户，所以我也通过电子邮件提供了很多很棒的独家内容。这是我的全职工作，我从中获得全职收入。

我还有一个投资组合网站，作为我作为一名企业家的个人网站/投资组合，[BBenediktsson.com](http://www.BBenediktsson.com)。我也通过我的时事通讯[10 周刊](http://www.bbenediktsson.com/newsletter)谈论商业、工作/生活平衡和企业家精神。

## 你的在线业务的目标是什么？

音频问题已经存在一段时间了，它扮演着各种角色:

*   我的店面，
*   我的杂志，
*   我的内容中心
*   和我的商业品牌。

我的目标是帮助尽可能多的家庭录音室音乐家获得更好的音乐，无论是通过我在网站上的免费提示还是通过我的高级音频产品。

我的使命一直是给你简单易用的实用技巧，你可以马上用它来提高你的作品。

## 你的 WordPress 主题和插件是如何帮助你实现的？

我的主题是一个古老的创世纪儿童主题，叫做 [Epik](http://appfinite.com/releases/epik-theme/) 。我可能会很快彻底改造网站，因为我厌倦了我的主题，那种外观和感觉越来越老了——我想差不多四年了。

我在网站上运行一堆插件。最重要的是:

*   [Thrive Content Builder](https://thrivethemes.com/contentbuilder/) ，我用于销售页面和登陆页面，
*   [SumoMe](https://sumo.com/) ，用于社交和电子邮件捕捉，
*   [重力形式](http://www.gravityforms.com/)，
*   然后是所有谷歌过“最重要的 WordPress 插件(此处插入年份)”的人常用的。

## 有人帮你布置了这一切吗？

几年前我设置主题的时候，我雇了一个开发人员来帮我迁移和调整。每当我需要非常具体的东西时，我要么寻找 Upwork，要么通过 Fiverr 找人快速完成工作。除此之外，我可以自己处理一切。

## 在这个过程中你学到了什么？

这么多！音频问题的第一个版本是硬编码的 HTML/CSS。谢天谢地，我不用再处理这些问题了，但它确实教会了我一些用这些语言编写基本 web 代码的方法。

其他经验包括:

1.  事情总是在变，所以确保你有好的主机和强大的支持。

2.  插件并不总是适合你的主题，所以有时你必须寻找正确的。

3.  你的网站对用户来说只是一个前端，所以要确保很容易找到东西。为每一页设定目标，以及你希望读者选择的路径(注册、分享、评论等)。).

4.  当涉及到外包帮助时，确保你仔细检查语言障碍。花更多的钱来获得内心的平静。欺骗自己会让你抓狂。

## 请告诉我们您的平台和主机选择。

我选择 WordPress 是因为它的用户友好性、通用性以及可供选择的主题数量。每个人都知道 WordPress，所以当你需要完成工作时，很容易找到人来帮助你。

我有超过 100，000 浏览量/月，所以我有一个 VPS，所以我可以支持流量。我的一些其他领域仍然停留在一个基本的共享计划，因为他们并不真正需要额外的带宽。

## 你如何维护你的网站？

老实说，我很不擅长。我只是确保定期更新我的插件，并在需要的时候更新 WordPress。我的主机提供商负责剩下的大部分工作，包括备份。

## 你未来的计划呢？

希望在夏天结束前能有所改进！我终于找到了我的品牌和声音的感觉，我想我需要修改我的设计，以创造全面的品牌一致性。
