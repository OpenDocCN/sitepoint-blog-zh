# WDS07 额外功能:安迪·克拉克和卡梅隆·亚当斯

> 原文：<https://www.sitepoint.com/wds07-bonus-feature-andy-clarke-cameron-adams/>

[![Andy Clarke presenting at Web Directions South 2007](img/53fdb75fe02d49d607e10277f93fb0d3.png)](http://flickr.com/photos/halans/1447409479/)

昨天在悉尼举行的 2007 年 Web Directions South 大会上，我采访了 SitePoint 的作者卡梅隆·亚当斯([themaninblue.com](http://themaninblue.com/))和著名设计师安迪·克拉克([stuffandnonsense.co.uk](http://www.stuffandnonsense.co.uk/))，他们两人都在会上发了言。

安迪谈到了漫画书的设计和布局(Matt Magain 的总结)，以及它可以教给我们的关于网页设计的一些经验。我问安迪关于漫画书排版的问题。

> 卡梅隆·亚当斯
> 
> “安迪，这也意味着你在提倡*更多地*使用漫画吗？”安迪·克拉克
> 
> “漫画桑斯是我绝对喜欢的字体之一，我期待它在几个项目中出现。”
> 
> 卡梅隆·亚当斯
> 
> “你先听到这里。”
> 
> 

[![Cameron Adams presenting at Web Directions South 2007](img/b80fce6d3f895642c9897c9de4c708fc.png)](http://flickr.com/photos/14230614@N05/1450565835/)

Cameron 谈到，由于 JavaScript 库的成熟，设计师现在可以获得广泛的用户界面可能性，并使用它们来提供真正可定制的用户体验。
