# 十大谷歌神话揭晓

> 原文：<https://www.sitepoint.com/top-10-google-myths-revealed/>

谷歌是网络上最受欢迎的搜索引擎，不仅为受欢迎的 Google.com 网站提供支持，也为雅虎提供支持还有美国在线。在 Google 中被收录是非常重要的，在 Google 中被高度收录可以给你的站点带来很大的好处。

然而，有许多关于谷歌如何工作的神话，尽管它们本身是无害的，但这些神话往往会让人们对谷歌如何工作得出不正确的结论。这篇文章的目的是纠正最流行的谷歌神话。

***误区一:你的 Google PageRank (PR)越高，你在搜索结果列表中的排名就越高***

这种说法很常见，也是许多抱怨的来源。人们经常注意到一个排名比他们低的网站被列在他们上面，并感到不安。虽然具有较高页面排名的页面确实倾向于排名更高，但是一个站点在结果列表中出现得更高是完全正常的，即使它比竞争页面的页面排名更低。

为了在不涉及太多技术细节的情况下解释这个概念，最好将 PageRank 视为由两个不同的值组成。其中一个值，我们称之为“通用页面排名”，就是页面上链接的权重。这也是[谷歌工具栏](http://toolbar.google.com)中显示的值。这个值是用来计算离开你的页面的链接的权重，而不是你的搜索位置。

另一个值我们称之为“特定 PageRank”你看，如果 PageRank 等同于搜索引擎结果排名，那么雅虎这个 PR 值最高的网站将会在所有搜索结果中排名第一。显然，这是没有用的，所以谷歌所做的是检查你的输入链接的上下文，只有那些与被搜索的特定关键词相关的链接会帮助你获得该关键词的更高排名。很有可能一个低排名的网站实际上比一个高排名的网站有更多的关于主题的链接，在这种情况下，低排名的网站在搜索结果中会排在竞争者的前面。

除了 PageRank，还有其他因素影响谷歌搜索结果——尽管 PageRank 仍然是主导因素。

***误区二:谷歌工具栏会列出你实际的 PageRank***

当谷歌创建他们的工具栏时，这对许多站长来说是一个福音，因为这是我们第一次看到与我们的 PageRank 相关的任何价值。然而，工具栏也造成了一些混乱。

工具栏不会显示你实际的页面排名，只是一个近似值。它给你一个从 1 到 10 的整数等级。我们不知道各种整数到底对应什么，但我们确信它们的曲线类似于指数曲线，每个新的“平台”都比上一个更难达到。我个人对这个 T1 做了一些研究，到目前为止，结果指向 4 的指数基数。因此，PR 为 6 比 PR 为 5 难 4 倍。

指数基数很重要，因为它说明了一个特定的 PR 值可以分配给多大范围的页面。6 的高 PR 和 6 的低 PR 之间的差异可能是数百或数千个链接。因此，如果你的 PR 在工具栏上显示上升或下降，重要的是要记住这可能是一个小的变化，也可能是一个大的变化。此外，有可能失去或获得链接，而看不到你的报告页面排名的变化。

工具栏的另一个问题是，有时它显示的 PageRank 只是一个猜测。人们经常会注意到 Geocities 或其他免费主机提供商的网页排名很高。这是因为当 Google 还没有爬行一个页面，但是已经爬行了根域的时候，工具栏会根据根域的值猜测一个 PageRank。因此，在地理城市上看到 PR 值为 6 或 7 的页面是很常见的。PageRank 并不等同于谷歌的高排名，事实上，在这种情况下，它表明了相反的情况:该页面甚至不在谷歌。一旦谷歌抓取了该页面，它将被分配一个更合适(通常更低)的 PageRank。

***误区三:PageRank 是一个基于你的网站*** 的链接数量的值

这个神话是对谷歌不正确假设的一个常见来源。人们会经常看到一个外来链接比自己站点少的站点，其 PageRank 却比较高，并假设 PageRank 不是基于外来链接的。

事实是 PageRank 是基于进来的链接，而不仅仅是链接的数量。相反，PageRank 是基于你的外来链接的值。要找到一个链接的价值，看看源页面的 PR 值，然后除以该页面上的链接数。如果你的链接足够“有分量”,很有可能从很少的链接中获得 6 或 7 的 PR 值。

还要记住，对于 PageRank 计算，每个页面都是一个孤岛。谷歌不在网站范围内计算 PageRank 所以你的页面之间的内部链接也算。这非常重要，因为[为你的内部链接](http://www.sitepointforums.com/showthread.php?s=&threadid=82447)建立一个合适的结构可以极大地提高你的排名。

***误区 4:在谷歌上用“link:”搜索链接，会显示出你所有的反向链接***

类似于神话 3，人们有时会在谷歌上寻找一个网站的反向链接，但是如果这个网站确实有一个 PR 列表，并且在谷歌的缓存中，他们知道工具栏不只是猜测。

这样做的原因是谷歌没有列出所有它知道的链接，只有那些贡献超过一定数量的 PageRank 的链接。这在一个全新的网站上尤为明显。默认情况下，谷歌的所有页面都有一个最低公关。因此，即使一个没有任何外来链接的页面也有 PR 值，尽管很小。如果你有一个有 20 或 30 个页面的全新网站，所有的页面都被 Google 抓取了，但是你没有来自其他网站的链接，那么你的页面仍然会有来自这些内部链接的 PageRank。由于你的主页很可能被链接到你网站上的每一页，它甚至可能从所有这些小小的提升中获得高达 1 或 2 的 PageRank。但是，在这种情况下，搜索传入链接可能会产生 0 个结果。

你也可以在已经存在一段时间的页面上看到这种情况。例如，[这个页面](http://www.promotionbase.com/article/801)在谷歌中列出了 [0 个链接，然而它的页面排名却是 3。我们可以看到，谷歌已经通过](http://www.google.com/search?sourceid=navclient&ie=UTF-8&oe=UTF-8&q=link:http%3A%2F%2Fwww%2Epromotionbase%2Ecom%2Farticle%2F801)[检查其缓存](http://216.239.51.100/search?sourceid=navclient&ie=UTF-8&oe=UTF-8&q=cache:http%3A%2F%2Fwww.promotionbase.com%2Farticle%2F801)搜索到了它，所以 PageRank 不是猜测。我们还知道，谷歌已经用[搜索了这个页面](http://dmoz.org/Computers/Internet/Web_Design_and_Development/Promotion/Ethics_and_Standards/)，再次通过[检查其缓存](http://216.239.51.100/search?sourceid=navclient&ie=UTF-8&oe=UTF-8&q=cache:http%3A%2F%2Fdmoz.org%2FComputers%2FInternet%2FWeb_Design_and_Development%2FPromotion%2FEthics_and_Standards%2F)。因此，我们可以肯定的是，谷歌知道至少有一个链接到这个有问题的页面，通过它列出的 PR，以及谷歌已经抓取了一个链接到它的页面的事实。

然而，如果你查看安装了[谷歌工具栏](http://toolbar.google.com)的 DMOZ.org 页面，你会注意到该页面的 PR 值为 0，非常低。此外，如果你数一下页面上的链接数量，你会注意到有超过 20 个。所以你在 20 多个链接中划分了一个非常低的 PR。因此，每个链接的权重都很小，所以当你搜索时，谷歌不会列出这些链接。然而，谷歌确实*统计了*链接，这就是为什么有问题的页面会列出 PR。

记住谷歌如何列出外来链接是非常重要的。通常，人们看到他们的链接数量下降，他们认为他们已经失去了这些链接。实际上，链接页面可能已经失去了一些权重，因此，链接可能已经低于列出链接所需的阈值。或者链接页面可能添加了更多的链接，导致每个链接的*份额*的权重更低，并再次导致链接低于阈值。在这两种情况下，链接仍然被计算在内，只是没有被列出来。

谷歌为什么要这么做？也许答案与技术限制有关。如果每页的平均链接数是 20，那么 Google 将不得不处理超过 600 亿个链接，这可能会创建一个太大而无法公开搜索的索引。

***误区五:被列入开放目录项目会给你一个特殊的 PageRank 加成***

谷歌使用开放目录项目(DMOZ.org)来驱动其目录。将这一事实与在 DMOZ 上市的网站经常获得令人费解的页面排名提升的观察相结合，许多人得出结论，谷歌给在 DMOZ 上市的网站提供了特殊奖励。这根本不是真的。

在 DMOZ 获得的唯一好处就是一个网站被其他网站链接到所获得的好处。然而，DMOZ 的数据被数百个网站使用。DMOZ 数据的最大用户是谷歌，但它也被成千上万的其他网站使用。这些网站的链接往往太弱，无法在链接搜索中列出，但谷歌确实会抓取它们，这些链接也很重要。因此，如果你在 DMOZ 上市，你实际上获得了数百个轻量级链接的好处，当你把所有这些加起来，总数可以达到一个不错的页面排名提升。

在 DMOZ 上市还有另外两个好处。首先，你的目录描述会出现在谷歌搜索结果列表中，这可能会增加有人点击你的链接的可能性。另一个好处是，正如谷歌搜索 DMOZ 一样，被列在那里将确保你也被列在谷歌上。然而，由于在谷歌上市如此容易，这种好处充其量是微不足道的。

***神话 6:在雅虎上市！给你一个特殊的 PageRank 奖励***

这个神话和神话 5 的演变过程是一样的。谷歌已经与雅虎合作！通过提供二级搜索结果，最近(2002 年秋季)，雅虎！开始使用谷歌提供主要搜索结果。

因为雅虎！使用谷歌，许多人认为谷歌也使用雅虎！，事实并非如此。唯一的 PageRank 你会从雅虎上市！和你从其他同等权重的网站获得的 PR 是一样的。然而，有些人从他们在雅虎的上市中获得了超乎寻常的提升。，再次导致这个不正确的结论。

事实是，在雅虎上市！雅虎的主目录通常会让你进入地区目录，所以，很像 DMOZ，一个雅虎！列表可以产生多个链接。这些链接本质上往往很弱，所以它们可能不会在链接搜索中显示出来，但它们确实存在——而且谷歌知道它们。

此外，一旦你被列在任何搜索引擎或目录中，你就有更多的机会被别人发现你的网站，喜欢它，并从他们自己的网站添加链接。因此，在雅虎上市。可能会导致你收到来自其他地方的链接——这些链接的权重太低而无法列出，但确实有助于你的 PageRank。

***误区 7:谷歌使用 Meta 标签对你的网站进行排名***

这个神话是大多数搜索引擎使用元标签时遗留下来的。然而，谷歌从未使用过它们。这个事实可能会受到一些人的质疑，所以我不会在没有证据的情况下发布它。

为了证明你自己，谷歌不使用元标签，把单词放入你的元标签中，这些单词不会出现在你页面的其他地方。然后，使用高级搜索，搜索这些词，同时将结果限制在您的域内。你可以在任何搜索引擎上尝试这样做——如果结果出现，你就会知道这个引擎使用了元标签。如果没有显示结果，那么您知道没有使用 meta 标签。不过，重要的是，这些词只出现在你的 meta 标签中，而不会出现在你页面的其他地方。

Google 有时可以使用 meta description 标签为您的站点创建一个摘要，因此如果您的主页主要由图形组成，它可能会对您有用。然而，不要指望它会增加你的排名。

***误区八:谷歌不会索引动态页面***

过去，一些搜索引擎在动态页面方面存在问题，也就是说，页面使用了一个[查询字符串]( http://www.webopedia.com/TERM/Q/query_string.html)。这不是因为任何技术限制，而是因为搜索引擎知道有可能创建一组无限量的动态页面，或者他们可以创建一个无限的循环。在这两种情况下，搜索引擎都不希望自己的爬虫抓取无限数量的动态生成的页面。

谷歌是一个较新的搜索引擎，从来没有查询字符串的问题。然而，一些动态页面仍然可以抛出谷歌的循环。

当 cookies 无法写入时，一些购物车或论坛会在 URL 中存储会话信息。这实际上扼杀了像谷歌这样的搜索引擎，因为搜索引擎用 URL 来索引，当你在 URL 中放入会话信息时，该 URL 会不断变化。这是事实，因为谷歌使用多个 IP 地址来抓取网页，所以每个爬虫会在你的网站上看到不同的 URL，这基本上导致那些页面不被列出。重要的是，如果您使用这样的软件，您修改它，以便如果 cookies 无法被写入，该软件只是不跟踪会话信息。

所以，你不需要使用[搜索引擎友好的网址](http://www.promotionbase.com/article/485)在谷歌中列出。然而，这些 URL 确实有其他好处，例如隐藏了您使用的服务器端技术(以便您可以在以后无缝地更改它)，并且它们更加友好。此外，虽然谷歌可以抓取动态页面，但它可能会限制从某个特定网站抓取的动态页面的数量。要获得好的排名，最好的办法是使用搜索引擎友好的 URL。

神话 9:如果你使用弹出窗口 ，谷歌不会列出你的网站，也不会处罚它

这是一个相对次要的神话，但它仍然时不时地出现(一语双关)。谷歌有一个名为 Adwords 的广告项目，他们的政策之一是不允许使用弹出窗口的网站参与这个项目。

这项政策只存在于 Google Adwords 程序中，但无论是通过道听途说，还是人们听说 Google 有一项针对弹出窗口的政策，并错误地认为这包括 Google 的主要索引，这个神话已经盛行。如果你使用弹出窗口，你将不会被列在谷歌上，这种说法是不正确的:许多使用弹出窗口的网站，包括 SitePoint，在谷歌上都有很好的排名。事实上，谷歌甚至理解所有可以创建弹出窗口的 Javascript 都是值得怀疑的。

神话# 10:如果你被链接农场 链接，谷歌会惩罚你

谷歌有政策反对使用人为手段来增加你的页面排名，这具体包括加入[链接农场](http://e-comm.webopedia.com/TERM/L/link_farming.html)之类的事情。有一些网站或服务建立了自动链接交换来提高你的网页排名。链接通常通过使用 CSS 对人们隐藏，或者使文本与背景颜色相同，或者将链接放在不可见的层中。由于搜索引擎不呈现 CSS，他们会看到隐藏的链接，从而在计算你的链接流行度时计算它们。

然而，尽管如此，谷歌不会因为你被链接农场链接而惩罚你。毕竟，你无法控制哪些网站链接到你，因此以此为由惩罚网站所有者是不公平的。此外，链接农场通常有较低的页面排名和大量的输出链接，所以每个链接只占总页面排名的很小一部分，因此这种滥用的方法不是很有效。

即便如此，如果你从你的网站链接到一个链接农场，或者在你的页面上放置隐藏链接，谷歌也会惩罚你。所以简单的事实是，你可以因为你对自己的网站所做的事情而受到惩罚，但不会因为被另一个网站链接而受到惩罚。
