# 微软 Kumo 搜索将挑战谷歌。又来了。

> 原文：<https://www.sitepoint.com/microsoft-kumo-search/>

微软宣布他们正在他们的公司网络上测试一个新的搜索引擎 Kumo。如果测试成功，Kumo 将进入 Live.com T2，或者可能作为一个新的搜索网站推出。

该公司一直在努力打造一个更好、更有竞争力的搜索引擎。索引系统将理解整个句子和单个单词之间的关系。理论上，这将导致更准确的结果。

有人有似曾相识的感觉吗？我们已经来过几次了，微软和其他公司都声称他们有 Google beater。大的声明和大的发布通常伴随着大的失败。

可以理解为什么微软想要挑战谷歌；他们是公司未来收入的最大威胁。一个更好的搜索引擎会击中谷歌的痛处。可惜，再好的 Kumo，也会纠结:

1.  引入一个新的搜索引擎需要时间。谷歌于 1997 年推出，但仍然花了两三年时间才成为主流。微软需要立竿见影的效果，但这不太可能发生。
2.  十年前，谷歌远胜于竞争对手。好多少？两倍好？三次？不可能给出具体的数字，但差别是显著的。Kumo 不可能有望与之匹敌:它可能比谷歌快 25%,也更准确，但这种差异会让人觉得微不足道。
3.  谷歌是一个与网络搜索同义的品牌。就像人们提到*“喝一杯可乐”*或*“打扫房间”*一样，用户将*“谷歌”*作为关键词。这样的品牌认知度在一代人中只会出现一两次；无论微软如何努力，它都不能被强迫或购买。

希望 Kumo 有微软宣称的那么好。竞争总是一件好事，谷歌不应该自满。然而，当谷歌对大多数人来说足够好的时候，有人能现实地创建一个更好的搜索引擎吗？
