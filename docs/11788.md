# 网络浏览器的统称是什么？

> 原文:[https://www.sitepoint.com/web-browser-collective-noun/](https://www.sitepoint.com/web-browser-collective-noun/)

有一群土豚。我们有多余的修女。然后是一些统计学家，极不可能的独角兽的祝福，和名词的集合。但是我们如何引用一组网络浏览器呢？据我所知，还没有人给一个术语下过定义，提及它们变得越来越尴尬。

也许是因为讨论一个以上的 web 浏览器是一个相对较新的概念？仅仅五年前，我们只有一个主流应用程序。然而，今天，我们有一个健康而竞争激烈的市场，有一个 *<插入名词这里>* 的网络浏览器来构建和测试。我们可以使用通用名词，如“set”或“group ”,但这有点无聊。

所以，因为现在是周末，我认为我们应该使用 SitePoint 来讨论并决定一组 web 浏览器的最终名称。以下是我的初步建议:

*   网络浏览器的浪潮
*   在网络浏览器上冲浪
*   网络浏览器的**渲染**

或者，对于我们这些仍在测试 IE6 的人来说…

*   网络浏览器的一个担忧
*   一个**畏缩**的网络浏览器
*   对网络浏览器的愤怒，或者
*   网络浏览器的谋杀*(尽管乌鸦们可能会反对这个)*

尽管如此，开发者和程序员并没有一个统一的名词。“团队”是乏味的，而且有一群“蹩脚的”黑客——为什么他们应该有所有的语法乐趣？我会投票支持开发人员的分歧或程序员的争吵。我相信下面的评论会证实我的建议…
