# SitePoint 服务:向供应商介绍客户

> 原文：<https://www.sitepoint.com/sitepoint-services-introducing-clients-to-vendors/>

部分基于今年早些时候我们的读者调查中的[重复请求，我们正在 SitePoint 上测试一项有趣的新服务，我们称之为](https://www.sitepoint.com/interpreting-the-results-of-the-2008-sitepoint-reader-survey-part-1/) [SitePoint 服务](https://www.sitepoint.com/an-introduction-to-services/)。这项服务是与领先的互联网营销公司 VendorSeek.com T4 合作提供的。它是这样工作的:

*   大量的供应商注册参与 B2B 市场。供应商指出他们最适合提供什么服务，并支付注册费。
*   希望购买诸如[标志设计](https://www.sitepoint.com/logo-design-101-what-is-a-logo/)、[网页设计](https://www.sitepoint.com/logo-design-101-what-is-a-logo/)、[虚拟主机](https://www.sitepoint.com/web/web-hosting-domains/)或 [SEO 服务](https://www.sitepoint.com/seo-bootcamp-for-beginners/)等服务的 SitePoint 访问者可以向参与的供应商索取报价。这项服务对希望获得工作报价的感兴趣的人是免费的。
*   SitePoint 因将客户和供应商撮合在一起而获得一小笔报酬(这是我们的免责声明)。

我们认为这是一个很好的模型，并且可能非常适合 SitePoint，这有几个原因:

1.  我们自己也使用过这项服务，效果很好。事实上，流程和为我们提供所需服务的供应商给我们留下了深刻的印象。如果我们不袖手旁观的服务，我们不会考虑将它集成到我们的网站到这种程度。
2.  我们已经有了一个由精明的企业家组成的欣欣向荣的社区，他们一直在寻找这类服务。
3.  我们还有一个更大的个人社区，他们每天向客户提供这样的服务，并且可能渴望通过这样的网络扩大他们的客户群。

此时，你们中的一些人可能会想，“这不就是 [SitePoint Marketplace](https://marketplace.sitepoint.com/) 的目的吗？”

是也不是。市场有利于拍卖和分类列表，但这项服务提供了以下额外的好处:

*   感兴趣的客户可以收到无义务报价。
*   客户可以接触到一个更大的供应商网络——一个超越访问 SitePoint Marketplace 的人的网络。

正如我上面提到的，我们认为我们应该自己尝试这项服务，看看它是否是我们想要的袖手旁观，我们确信这是一件好事。不过，我们很想听听您对这项服务的看法。
