# 搜索工具栏综述

> 原文:[https://www.sitepoint.com/search-toolbar-roundup/](https://www.sitepoint.com/search-toolbar-roundup/)

为用户提供免费搜索工具栏的最新搜索引擎是 Teoma。这款免费的导航工具让您可以从任何网页搜索 Teoma 的数据库，这是一个触手可及的有用信息资源。

好几个搜索引擎都提供类似的实用程序，既方便又节省时间。其中最受欢迎的是谷歌工具栏，但其他几个引擎也为忠实用户提供免费导航工具，包括 Alexa、Excite、Lycos、Teoma 和 Yahoo！如果你愿意，可以下载几个工具栏——一个会简单地出现在另一个的下面。例如，如果你是谷歌和 FAST 的粉丝，你可能想要谷歌工具栏和 Lycos SeeMore(由 FAST 支持)。或者试试新的 UltraBar，它允许你从一个可定制的工具栏中搜索你自己的搜索引擎菜单。

下面是各种可用工具栏及其功能的总结。Excite 是一个在桌面上运行的旧版本，可能是以“推送技术”为设计理念的。早在 1997 年，PointCast 就推出了“推”式营销方法，将信息直接发送到电脑桌面上，一次发送给一个潜在客户，这与传统的“拉”式技术相反，传统的“拉”式技术是向数千人广播新信息，然后等待他们采取所需的行动。

##### Alexa 工具栏

Alexa 工具栏可以让你快速找到你正在浏览的网站的有用信息。从“网站信息”中，你可以获得联系信息、网站统计和用户评论。“相关链接”是发现类似网站的好方法。该列表会不断更新您所在站点的其他用户访问的站点的相关信息。有一个谷歌工具栏和 Dogpile 的链接。

您可以使用“搜索”按钮从 Alexa、Yahoo！还有谷歌。如果你不能访问你最喜欢的网站，或者想看它以前的版本，使用“Wayback”按钮在 Way Back 机器上查看网站的存档副本。

##### excite 助手

download.com 仍在销售的 Excite 助手在你的桌面上插入了一个工具栏，你可以在上面搜索 Excite，并访问各种在线信息，如股票和新闻更新。无需打开浏览器，即可收听互联网广播并获取个性化新闻！适用于 Internet Explorer 或 Netscape Navigator。

##### 谷歌工具栏

用于 ie 浏览器的谷歌工具栏下载速度非常快，可能是使用最广泛的(迄今为止已经有几百万次下载)。这个工具栏在你的浏览器中实现了一个谷歌搜索框，允许你从任何网页访问谷歌的流行数据库:搜索网页、网站或谷歌目录。你也可以访问“PageRank”来显示网站的链接流行度。

“网页信息”给你一个网页的缓存快照，相似的网页，反向链接，甚至将外国网页翻译成英语。“突出显示”会在您的搜索词出现在页面上时以不同的颜色突出显示。“单词查找”可让您在页面上显示搜索词时进行查找。当启用 PageRank 时，Google 会通过工具栏收集信息，但如果您选择了“安装时不使用高级功能”，则不会收集信息。

##### lycos 看到更多

用于 Internet Explorer 4 或 5 的 Lycos SeeMore 略有不同。它让你只需右击一个单词、短语或图片就能快速搜索网页。你会看到一个菜单，从中选择“用 Lycos 查看更多”，然后实用程序会出现关于你的主题的更多信息链接，允许你点击获取更多信息。

例如，如果你在阅读一个描述最近机场安全漏洞的页面时，想了解更多关于“机场安全”的信息，它会向你推荐几个文本链接，从中选择相关信息。这很方便，因为你不会因为去搜索框而失去你的位置。

##### Teoma 搜索栏

用于 Internet Explorer 的 Teoma 搜索栏让您可以从网络上的任何地方即时访问 Teoma 的搜索技术。“结果”提供相关网页的链接，“细化”提供缩小搜索范围的建议，“资源”提供专门目录的链接。它可以快速下载和安装，每次打开新的 Internet Explorer 窗口时都会出现。它的功能包括一个字典查询，以及一个高亮按钮，当你的搜索词出现在页面上时，它会用六种不同的颜色高亮显示。

##### 雅虎！同伴

雅虎！Companion 在你的浏览器中放置了一个可定制的工具栏。它允许你用雅虎按钮个性化！服务，保存您的书签，以便从任何计算机访问，在 Yahoo！邮件，并允许您搜索雅虎！从任何网站。适用于 Internet Explorer 或 Netscape Navigator。

##### UltraBar IE 工具栏

UltraBar IE 工具栏可能是最方便的。它允许您从一个工具栏的菜单中选择最常用的搜索引擎，并且它包含了最新的搜索工具栏功能。

您可以自定义搜索引擎的顺序，在页面中突出显示搜索词，并直接转到您的搜索词或短语。它也不局限于主要的搜索引擎——你可以选择任何有搜索功能的网站。

有三个版本，两个是免费的(Ultrabar IE 工具栏和 Ultrabar 联名)。Ultrabar Pro 更适合 ISP 和大型网站，提供每年 500 美元起的无限制下载/安装的附加功能。

##### 为什么要使用搜索工具栏？

搜索工具栏很方便，因为它允许你从任何网站进行搜索，节省了你去搜索引擎进行查询的时间。较新的工具栏通过在整个页面突出显示您的搜索词来提供附加信息。有了 Lycos SeeMore，你还可以链接到你正在阅读的文本中的相关信息。

因此，如果你专门使用上述搜索引擎中的任何一个，那么下载适当的工具栏实用程序可能并不是一个坏主意。当你需要搜索多个搜索引擎或网站时，UltraBar 给你完全的灵活性。

虽然这些工具提供了许多积极的好处，但一个主要的缺点是，除了 Excite Assistant 和 Yahoo Companion 之外，所有工具都只能用于 Internet Explorer，而没有一个工具可用于 MAC。
