# 我计划如何将慈善事业融入我的业务

> 原文:[https://www.sitepoint.com/small-business-philanthropy/](https://www.sitepoint.com/small-business-philanthropy/)

一年中有几次，大多数人会想到仅仅为了他人的利益而做好事。例如，假日季节。我认为这么多的公司、个人和家庭在节日期间受到鼓舞，停下来回馈社会，这很棒。但是，如果我们能够一整年都保持这种帮助他人的愿望，这不是很神奇吗？

我冒昧地猜测，几乎我们每个人都想做慈善，但我也知道有很多借口阻止我们采取行动。其实我自己也找了很多借口:不在预算内；我没有多余的时间；我不知道如何给予；我不知道该给谁。

最近，我一直在思考如何将慈善事业融入到我的业务中，让它成为我工作的一部分。以下是我思考的过程、头脑风暴和想法。

## 慈善是什么意思？

我从“慈善事业”这个术语开始了这个过程慈善是通过金钱或实物捐赠以及其他慈善行为来改善他人生活的愿望。

这些天，当我们想到慈善事业时，我们往往会想到大的运动，如[捐赠承诺](http://givingpledge.org)，这已经得到了相当多的新闻报道，因为最富有的人捐赠了数十亿美元。这是值得称赞的，绝对的。但这并没有很好地转化为自由职业者和小企业主的日常生活。事实上，这几乎让以一种有意义的方式做慈善看起来是不可实现的。

## 慈善对我来说意味着什么？

我不得不努力思考我想通过变得更加慈善来实现什么。这是我想到的:

*   我想坚持不懈地捐助一两项与我相关的事业。
*   我想进行金融和其他慈善捐赠。
*   我想把我的行为和我生意的一部分联系起来。
*   我希望能够选择那些我觉得会真正有所作为但又不超出我能力范围的捐赠活动。
*   重要的是，我要确保我的捐赠是出于正确的原因。

## 我怎样才能做到博爱？

对我来说，这一过程中最困难的部分是确定将慈善事业融入其中的方法，这些方法对接受慈善的人来说是现实的，真正有价值的。以下是我一直在思考的一些想法，也有一些是我已经在做的。

*   通过撰写营销文案、设计营销材料等，自愿贡献我的时间来支持当地的慈善组织。
*   当一个有前途的企业家开始创业时，为他/她提供指导
*   送出一些我开发的智力资本(书籍、报告、培训材料等)。)给那些买不起的人
*   向我母校的奖学金项目捐款
*   主动提出在当地社区大学教授商业课程
*   每季度向特定的慈善机构捐款

现在，我已经开始了其中的一些活动，但我仍在努力发展慈善目标，将它与我的业务更紧密、更一致地联系起来。

你认为自己是慈善家吗？我很想听听你的想法，以及你是否已经成功地将慈善事业融入到你的业务中。
