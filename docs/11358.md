# 法国考虑对互联网广告征收“谷歌税”

> 原文:[https://www.sitepoint.com/french-google-tax/](https://www.sitepoint.com/french-google-tax/)

法国政府提议对使用点击付费广告的搜索引擎等网站征税。尽管该计划似乎针对谷歌、雅虎、必应和脸书等大公司，但它有可能影响任何通过广告增加收入的网站。

通过税收筹集的资金将被投入到创意产业中，这些产业已经被互联网削弱了。这可能包括为书籍、音乐和视频发行创建合法的在线渠道。(法国有一些最严厉的反盗版立法，新法律现在赋予当局切断联系和罚款惯犯的权力。)

该报告的作者之一 Guillaume Cerutti 说，该税将结束“没有任何限制或补偿的富裕”。作者希望这些提议能得到全欧洲的支持，但法国单干“在法律上和技术上”都是可能的。法国的互联网广告收入估计超过 11 亿美元，该计划预计将从各公司筹集 7000 万美元，无论它们是否在法国设有办事处。

当然，这只是世界各国政府众多无知且技术拙劣的策略中的一个。这个故事上了头条，对大多数选民来说听起来很棒，但几乎没有实用性，最终可能导致整体净亏损。

第一个问题:当法国用户点击赞助链接时，政府如何对其进行监控？互联网服务提供商可能会被迫实施链接监控软件，但这是一项巨大而昂贵的任务。此外，收入金额如何与每个链接相关联？点击费用从几分之一美分到数百美元不等。

税收将是下一个障碍。有多少外国公司会从一个他们没有办公室或服务器设施的国家缴税？如果这项立法得到公平实施，法国政府将需要向数十万家公司寄送税单，而收回管理成本的可能性微乎其微。

这项立法似乎有意打击美国的大型互联网公司。最便宜的选择是从法国办事处的利润中收取额外的钱。这几乎肯定会适得其反；跨国公司擅长将资金转移到税收最低的国家。

看来法国政府已经忘记了互联网是一个全球性的网络。谷歌、雅虎和微软都在法国设有办事处，但如果成本大于收益，它们可能会被关闭。失业和所得税收入的减少可能会对法国经济产生更大的长期影响。

谷歌法国公司总经理奥利维耶·斯珀呼吁政府“通过税收逻辑，支持互联网和文化世界之间的合作，而不是对立。”
