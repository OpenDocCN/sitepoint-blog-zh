# 域名评估——域名公平价值游戏

> 原文:[https://www.sitepoint.com/domain-name-fair-value-game/](https://www.sitepoint.com/domain-name-fair-value-game/)

既然域名已经被证明具有转售价值，对域名进行估价的“科学”正在成为一门大生意。现在已经有三十家甚至更多的评估公司，还有更多的公司出现，但是他们的方法真的科学吗？他们评估名字的方式有什么共同点吗？为了尝试回答这些问题，我向目前正在运作的评估公司提出了挑战，要求他们对一个名字 LiquidTrading.com 进行价值评估。

我选择 LiquidTrading.com 是因为它不是一个容易评价的名字，需要一些研究来充分了解它的潜力。如果您想了解更多有关“流动性交易”背后的概念，请参阅《连线》杂志 2000 年 3 月印刷版中的文章。

由此产生的评估是 2000 年末域名评估行业的一个迷人的“快照”…

[点击此处查看 LiquidTrading.com 估值图](http://www.eCommerceBase.com/static/domainappraisal.php3)

单击图表中的美元值查看实际评估。请注意，在某些情况下，评估指向外部网站或 Microsoft Word 文档。

如你所见，估价从 200 美元到 16 万美元不等。这样的估价范围提出了一些关于领域评估行业的基本问题。如果一个名字就能支撑如此大的估值范围，那么当前的行业还能比彩票更好吗？

##### 变化的原因

好吧，在把整个行业一笔勾销之前，让我们来看看这种巨大差异的可能原因:

1)正如本文开头提到的，LiquidTrading.com 这个名字是一个特别微妙的名字，可以从几个不同的方面来看。如果评估师了解报道该名称的《连线》文章，他们自然会从更有利的角度看待该名称，因为他们会看到该名称在下一代交易网站中的潜力。

2)一些评估师使用“相似”名称的销售价格数据库，以帮助他们得出美元价值。然而，这些数据库中有许多都非常小，大多包含来自域名拍卖网站 Afternic.com 的域名，该网站定期公布销售价格。为了在统计上有效，一个数据库至少需要包含几千个名字。

此外，Afternic.com 大多数品牌的售价都远低于其价值，因为买家往往是经销商，而不是最终用户。使用 Afternic 的销售价格进行比较不可避免地会导致对转售而非出售的估价。

最后，整个“相似性”概念的价值肯定是值得怀疑的。查看具有流动性或折价交易的名称的销售价值并不能真正表明流动性交易的价值。如果自然语言这么容易分析，自然语言解析器早在几十年前就已经完善了。正是“流动性”和“交易”这两个词的独特而精确的结合，使这个名字变得有趣。孤立的词语没有什么意义。

3)上面列出的一些评估显然是由一个或多个评估人“手工制作”的。另一些索赔似乎至少部分是通过运用一种通用的“估价公式”得出的，以便得出最后估价。虽然简单的公式化方法可能适用于简单的名称，但它不能指望抓住自然语言的细微差别，以及语言背后的微妙概念，并且总是无法取代一个或多个熟练人员仔细研究的评估。

##### 摘要

从估价的范围和涉及的方法来看，域名估价行业显然还不成熟。如果域名评估将被广泛接受，首先在域名行业，然后在更广泛的金融界，很明显需要引入一些标准，越快越好。
