# 工资和 x 光——网络商业的字母表

> 原文：<https://www.sitepoint.com/wages-and-x-ray-the-alphabet-of-web-business/>

我们正处于网络商务系列字母表的倒数第二部分！我希望你喜欢这些智慧的片段和思想的火花。这里还有两个鼓励你进一步…

### **W 代表工资**

你上一次进行工资审核是什么时候？如果你是一个人，那么很可能已经有一段时间了。在字母表的上一期中，我写了关于利率以及为什么定期检查你的收费是重要的。

这同样适用于你实际支付给自己的报酬。如果你仍然支付和几年前一样的工资，那么你现在已经落后于平均工资了。当然，牺牲有时是必要的，但是如果公司有能力支付你更多，那么我鼓励你给自己加薪。如果你为别人工作，而他们在过去的一年里没有和你讨论过工资评估——没有比现在更好的时机提出这个话题了！

### **X 代表 X 射线**

想象一下拥有 X 射线视觉——当服务器出现问题时，您可以直接看到银行金库，或者看到邻居在做什么！好吧，虽然个人 X 射线视觉可能仍然在科幻小说的领域中，但将这个概念应用到你的商业交易中就不是了。

下次有人要求你重新开发他们的网站时，用你的 x 光眼光。问正确的问题，深入了解他们要求重新开发的原因——使用这种策略意味着你更好地了解他们询问背后的真正驱动因素，并确保你将提供比竞争对手更好的结果。
