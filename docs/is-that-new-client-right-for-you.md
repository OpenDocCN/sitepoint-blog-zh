# 那个新客户适合你吗？

> 原文：<https://www.sitepoint.com/is-that-new-client-right-for-you/>

如果你有自己的生意，你可能会花很多时间担心如何获得和留住客户。客户意味着金钱，金钱意味着餐桌上的食物，对吗？

这个等式的危险之处在于，它让我们认为任何客户都是好客户。他们不是。对我来说好的客户对你来说可能是糟糕的客户。

让我们来谈谈你最近的潜在客户的一些方面，这些方面可能会让他们成为你公司的坏客户。

## 伦理学

我的一个联系人最近参加了与一个新客户的预备会议，了解到该客户希望使用我的联系人的服务来培训其员工。事实上，客户希望利用这项服务来培养公众使用其产品——在线赌博。我的联系人对这份工作的道德有问题，决定不再继续推销。

道德可以涉及工作的所有方面:如何与承包商和同事打交道，使用哪些第三方服务，以及项目如何营销。不要为了金钱而试图妥协你的道德准则——你最终只会恨你自己或客户。或者两者都有。

## 公司风格

还应该考虑客户组织的工作风格。你的客户联系人的运作方式，以及他们必须在其中运作的结构和系统，可能是有效的交易破坏者。

每个客户组织都有自己的风格，包括以下内容:

*   它的工作方式
*   其文化
*   审查和批准流程
*   员工和承包商的期望

虽然我知道我可以和风格令我不快的客户一起工作，但我尽量不这样做。一些客户报告要求非常糟糕。那就是。你可能不喜欢为创业公司工作。那也是事实。很少，如果有的话，值得与这些现实争论——花时间寻找你喜欢的客户比接受你不喜欢的客户，并试图用更高的费用或更长的时间来证明这个决定是正确的。

## 客户端风格

如果组织风格值得思考，那么你将在项目中与之共事的实际人员的方法也值得思考。在你投球之前，你可能会也可能不会对所有的团队成员有一个好的了解，但是你应该对关键联系人有一个感觉——以及你们是否能够很好地合作。

利用你过去的经验找出你喜欢与之共事的人是个不错的主意。我的基本标准大概是这样的:

*   他们有幽默感。
*   他们很擅长自己的工作。
*   他们公平合理。
*   他们让我感到鼓舞，尽我所能帮助他们。

就是这样。我已经按照对我来说的重要性列出了这些标准，但是在我看来，所有的标准都需要被满足，这个项目才值得去做。

## 名声

我不知道你怎么想，但我希望能够自豪地谈论我所做的每一份工作。这对[推广我的业务](https://www.sitepoint.com/the-truth-about-word-of-mouth-referrals/)很重要，因为我的关系网很小，所以声誉很重要。

我从未因为名声而拒绝过一个潜在的工作，但我接受了一些工作，因为我知道这些工作不值得和我的联系人谈论。这可不好。

如果你认为一旦项目完成，你就不能自豪地谈论它，考虑一下它是否适合你。你的沉默可能暗示了你和客户之间潜在的不匹配——也许是我上面提到的某一点。

这些是我在与潜在客户交谈时评估的事情——但你呢？你如何判断一个潜在客户是否适合你和你的企业？
