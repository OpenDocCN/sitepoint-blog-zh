# 回忆和放松

> 原文:[https://www.sitepoint.com/recollect-and-relax/](https://www.sitepoint.com/recollect-and-relax/)

几天前，我压力很大。你看，我要去海外旅行，这意味着我有两周的时间来计划和做工作，还有要写的文章，要联系的客户，要委派的工作，还有家务等等。

我开始感到不知所措——我有这么多事情要做，我应该从哪里开始呢？我一边走一边试图在脑子里划掉这些任务，感觉每次我完成一件事，就会有另外两件事出现。

我停下来写了一个清单。一个非常简单的，潦草地写在一张白纸上的待办事项清单。它花了我五分钟，在它结束的时候我几乎完全放松了。我以前写过要做的事情清单，但是我不相信我真的遇到过那种清晰的时刻，除了知道你需要做什么之外，为什么要做的事情清单是伟大的。

现在，我知道一个好的待办事项清单不仅能帮助你集中注意力，同时还能让你放松。你看，我最终列出了一个很长的清单——30 或 40 个项目，尽管很多项目在时间要求上很少。

我在每项任务旁边的括号里写下了我需要给自己多少时间，所以这个列表看起来有点像:

给公司的比尔打电话咨询提案事宜(10 分钟)
将电子邮件形式发回银行(15 分钟)
将电子邮件简讯副本发送给莎拉(10 分钟)
撰写新项目提案(1 小时)
获取旅行适配器(20 分钟)
购买旅行保险(30 分钟)

我很容易看到，回顾这个列表，大约有 30 个小时的分配时间。如果我有至少八天的时间，包括周末，我可以在离开前轻松度过。卸下我的负担的感觉很好，也很安心；我觉得我突然有了更多的精力来完成这些任务。

下次你感到压力巨大时，花五分钟写一份清单。在你意识到之前，你永远也不会完成的堆积如山的工作变成了一个可管理的任务列表，你的脸上又恢复了笑容。

试试看——这是一个奇妙的五分钟投资！
