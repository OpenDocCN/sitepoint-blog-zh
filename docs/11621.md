# 变焦变焦变焦:多任务处理的更好方式

> 原文:[https://www.sitepoint.com/zooming-better-than-multitasking/](https://www.sitepoint.com/zooming-better-than-multitasking/)

![Zooming](../Images/ff674a639271963a1313cda95a596bc8.png)我在 Inc.com 上看到了这篇文章，“[你变焦吗？](http://www.inc.com/resources/solo/articles/20070801/lonier.html)《泰莉·隆尼尔》。虽然这不是一篇新文章(写于 2007 年)，但它仍然激起了我的兴趣。我认为“变焦”是一个永恒的概念，许多小企业主和自由职业者都可以从中受益，特别是如果他们努力提高生产率，能够从一个任务转移到另一个任务，而不会承受更多的压力。

## 什么是缩放？

在 Lonier 的文章中，她将变焦定义为“在一个人的业务的宏观和微观视角之间快速反复转换的能力。”Lonier 的观点是，能够快速无缝地重新聚焦有助于那些身兼多职的企业家保持必要的速度，让他们的企业平稳高效地运营。

如果一个企业主能够成功地缩放，他们就能够更好地适应企业所有权的日常挑战，同时提高他们的生产力，完成更多的主要和次要任务。

## 如何最大化你的变焦

我认为变焦是一个很好的选择，替代了传统的多任务处理的想法，这种想法很糟糕，我们大多数人可能已经这样做了，但没有意识到这一点。虽然多任务通常指的是试图[同时专注于一个以上的任务](https://www.sitepoint.com/multitasking-kills-focus-productivity)(这可以被认为实际上让你没有效率，尽管在某些情况下我不同意)，缩放允许你单任务，或一次只专注于一个任务，但增加了你从一个任务快速转移到下一个任务的能力，而不会面临漫长的过渡。

如果你对提高你的变焦技术感兴趣，这里有一些方法可以让你专注于变焦，并开始消除多任务思维。
 **完善你的待办事项清单**

无论你的喜好是什么，几乎我们所有人都使用某种[清单来记录我们需要完成的事情](https://www.sitepoint.com/to-do-lists-for-productivity/)。让自己变得更有效率的一个方法是列出一个适合你个人心态的清单，支持你在一天中各种任务之间来回移动的愿望。这意味着你的清单应该被分解成小的和可管理的任务。

你可能还会发现，如果你使用一个分类和优先的待办事项列表，而不是一个庞大的列表作为你的任务转储场，你可以更有效地缩放。
 **考虑大局**

虽然你需要花时间专注于每一项任务，但不要忘记最终目标。你清单上的每个小任务都是一个大项目的组成部分，通过思考整个过程，你可以确保你在小任务上的工作确实支持大项目。你可以对你处理的每一个小任务都这样做，从大的方面来看，质疑任务在最终结果中的位置，并确保它保持了整个项目的完整性。

**专注于当下**

与多任务处理不同，变焦不会试图迫使你的大脑一次处理多个复杂的想法。不幸的是，我们想要通过在短时间内完成大量工作来达到高生产率的愿望可能会对我们不利。当你从一个任务转移到另一个任务时，专注于你正在做的一件事，把所有竞争的想法和分心的事情推到一边。当然，这说起来容易做起来难，但是如果你在做的时候能够完全专注于一项任务，你就已经走上了高效变焦的道路。

你变焦吗？你认为这可能比多任务处理更有效率吗？
