# 经济衰退的迹象正在逼近家…你能做些什么来战胜它

> 原文:[https://www.sitepoint.com/beat-the-recession/](https://www.sitepoint.com/beat-the-recession/)

在我们的个人生活和职业生涯中，我们都看到了经济衰退的影响。对于那些拥有自己的企业或自由职业者来说，挑战是显而易见的，有些挑战甚至是不可逾越的。你在职业生涯中可能会看到的一些迹象包括:

*   你的客户开始缩减他们外包给你的工作。
*   您的商业信用卡限额已降低。
*   客户开始向你施压，要求你降低费率。
*   你不再按时得到报酬。
*   你的业务支出在增加，而你的收入却没有。

人们很容易被这些变化的不可抗拒性所吸引。为自己工作可能很困难，即使是在经济和你的生意蒸蒸日上的时候。面对这些挑战，再加上低迷的经济，你将面临一场艰苦的战斗。但是它是可以打败的。现在你可以做一些简单的事情来对抗衰退，让自己在经济动荡中更好地生存。

**收紧你的政策**

现在可能是时候审视一下你目前的政策，看看你可以在哪些方面改进你的流程，以帮助管理经济衰退。你可能会决定停止提供任何形式的折扣，或者征收滞纳金可能是确保及时付款的一种方式。

你最不想做的事情就是赶走现有的有价值的客户，所以对你所做的改变要谨慎。在迎合客户的同时，采取新的政策来强化你的核心业务是可能的。做出合理的改变，发出通知，并愿意与客户讨论你修改的原因。

**提高您的价格**

虽然在经济衰退期间提高利率似乎有些矛盾，但这可能是一个明智之举。提高您的费率可以有效地将您推向更高的服务等级，从而吸引全新的客户群。假设你有足够的技能和经验来证明这种增长是合理的，你可能会发现你的企业变成了一家利润更高、效率更高、更受欢迎的公司。而且在这个过程中你也许能甩掉一些不理想的客户。

**多样化**

除了核心服务之外，你还可以向客户提供许多服务，以帮助增加收入和吸引新客户。参见“ [7 网页设计师可以为客户提供的增值服务](https://www.sitepoint.com/7-value-added-services-web-designers-can-offer-clients/)”了解一些想法。你可能还想增加自己在某些领域的知识和技能，让自己更有市场。

**看看定期支付**

当定期付款每月自动打到你的账户时，很容易被忘记。定期回顾你的陈述，看看什么是你真正需要的，什么是你可以消除的。有些服务允许您暂时暂停订阅，或者离开后再回来，因此您可能也想探索这些选项。

**考虑开源**

我们大多数人都使用最好的软件来完成我们的工作。虽然我不建议抛弃顶级的选项，但是有一些很棒的开源应用程序可以在第二台计算机上提供不错的选择。在 SourceForge.net 的[有许多](http://sourceforge.net)[开源图像编辑器](https://www.sitepoint.com/open-source-image-editorsfor-designers/)，以及数以千计的其他应用程序。
