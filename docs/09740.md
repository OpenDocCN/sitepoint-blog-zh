# Mozilla 发布了 Firefox 5 Beta 2 和 Aurora 更新

> 原文:[https://www.sitepoint.com/mozilla-firefox-5-beta-aurora/](https://www.sitepoint.com/mozilla-firefox-5-beta-aurora/)

你还在用火狐 3 吗？你刚刚升级到火狐 4 吗？你觉得版本之间的等待时间太长了吗？说到做到，Mozilla 已经开始了快速发布开发周期，就像谷歌对 Chrome 所做的那样。Firefox 5 将于 6 月 21 日发布，您可以从以下网址下载 beta 2:

[www.mozilla.com/firefox/channel/](https://www.mozilla.com/firefox/channel/)

通常的免责声明适用。这是测试版软件。它可能会覆盖您现有的安装。插件不太可能兼容。你的电脑可能会爆炸。你可能会染上一种严重的疾病。

新版本承诺提高性能、安全性和稳定性，并且:

*   静默更新
*   CSS 动画
*   改进了对 HTML5 和相关技术的支持
*   更好的 JavaScript 和画布性能
*   任务栏 web 应用程序支持
*   一些界面调整，更好地整合了 Mac OS 和 Linux
*   一个更容易访问的“不要跟踪”选项——显然，在 Firefox 4 中只有不到 2%的人使用它。

也就是说，您不太可能注意到版本 4 和版本 5 之间的显著差异。

也许尖端浏览器爱好者会对 Aurora 更感兴趣。Mozilla 的新频道提供了各种质量和水平的未来创新。Firefox 的 Aurora 版本也可以从上面的链接下载。

你能发现什么主要的新特征吗？我不能，但我不得不承认，标志是华丽的！…
