# 在欧洲购买 Windows 7…价格只有美国的一半！

> 原文:[https://www.sitepoint.com/windows-7-cheaper-in-europe/](https://www.sitepoint.com/windows-7-cheaper-in-europe/)

对于微软的欧洲客户来说，这是令人困惑的几个月:

1.  2009 年 2 月，欧盟委员会对微软的反竞争行为进行了调查，因为 ie 浏览器与 Windows 捆绑销售。
2.  3 月，微软透露可以从 Windows 卸载 [IE。](https://www.sitepoint.com/ie-windows-7-uninstall/)
3.  6 月，微软宣布将在欧洲发布 Windows 7 的特别版[“E”版](https://www.sitepoint.com/ie-windows-7-europe/)。IE 将不会提供，也不可能从 XP 或 Vista 升级。
4.  7 月，微软允许人们以 50%的折扣预购 Windows 7 E。这款操作系统受到了媒体的好评，这使得[所有的预订版本在第一天](https://www.sitepoint.com/windows-7-sold-out/)就销售一空。
5.  欧盟猛烈抨击 Windows 7e；他们认为不提供浏览器并不能替代提供选择。微软让步了，Windows 7 E 被放弃，IE 将被安装，但是 T2 的欧洲用户将被提供一个投票屏幕来选择他们的默认浏览器。

正在进行的欧盟口水战给微软带来了营销混乱。在欧洲无法订购 Windows 7 的升级版本，您必须购买完整版本。然而，它的价格很有吸引力，因为最初不提供浏览器:

*   在亚马逊美国网站上， [Windows 7 家庭升级售价为 120 美元](http://www.amazon.com/exec/obidos/ASIN/B002DHLUWK/mswindows7-21)。
*   Windows 7 家庭版的完整版售价为 200 美元。
*   亚马逊英国网站正在出售 Windows 7 家庭版的完整版，仅售 65 英镑(略高于 106 美元)！

是的，你没看错。**Windows 7 的英国价格明显低于美国价格！**欧盟居民也可以选择他们的默认浏览器，而美国用户必须手动安装！

我惊呆了。由于各种未公开的运输和货币兑换“成本”，英国和欧洲居民通常接受他们将被敲竹杠。看起来欧盟的调查帮了我们一个大忙。)
