# Microsoft Live Search 开始集成 Powerset

> 原文：<https://www.sitepoint.com/microsoft-live-search-begins-powerset-integration/>

读写网注意到微软已经开始整合一个多月前以 1 亿美元收购的 Powerset 搜索引擎的功能。Powerset 博客上的一篇[帖子详细介绍了 Powerset 的哪些功能已经添加到 Live 中。](http://www.powerset.com/blog/articles/2008/09/17/powersets-first-live-search-projects)

Powerset 详细介绍了他们的技术被整合到 Live search 中的三种方式，每种方式都面向消费者，所有搜索者都可以查看。首先，对于许多主题，Live search 现在将返回答案或 xRank 传记，其中包括主题摘要、相关链接和从 [Freebase](http://www.freebase.com/) 数据库中提取的数据。

![](img/4a2a5390b5c8a3a55930b8e5d818d75a.png "livesearch-xrank")

第二个集成是在 Powerset 的 Factz 提取技术支持的查询上添加了增强的相关搜索链接。最后，Powerset 和微软为维基百科提供的结果打开了改进的标题。Powerset 表示，他们将能够监控 Live Search 的摘要与 Powerset 生成的摘要的性能，并找出哪个更准确。

很高兴看到微软如此快速地整合 Powerset 的一些技术，但这可能不会帮助他们从谷歌那里获得市场份额。我们在 7 月份注意到，为了在技术上击败谷歌，你必须展示出*卓越的技术。虽然名人搜索和关于地标或城市的关键事实的更好排名很酷，但这不会让大众放弃谷歌。*
