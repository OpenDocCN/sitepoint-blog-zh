# 设计元素:线

> 原文:[https://www.sitepoint.com/elements-of-design-the-line/](https://www.sitepoint.com/elements-of-design-the-line/)

有许多设计元素可以作为我们设计的一部分，其中最简单和最有用的是线条。你使用的线条类型可以传达不同的感觉、情绪，并为你的想法增添力量。细线、点线、虚线和曲线可以增加设计的趣味。

当你开车时，你保持在白线的一边，当白线虚线时，你可以超车。为了与驾驶类比保持一致，这里有一些交通警告标志，其中绝大多数只由线条组成。我们每天都习惯于看到和服从线条(或它们所代表的东西)。

看看下面的几行，看看每一行都有自己的个性。通过改变粗细、曲线和笔画宽度，我们可以产生不同的感觉或想法。

**组织行**

通常，线条在设计中用于分隔页面或布局的各个部分，以及连接彼此相关的部分。在网站和杂志上，我们看到线条被用来给照片加框、分隔边栏和将文章连接在一起。

在下图中，你可以看到杂志的跨页利用了垂直线和水平线。在左边的页面上，一条垂直线分隔两列文本。在右边的页面上，通过使用两条水平线和斜体文本将引言与标题和正文分开。

阿拉巴马高山营地网站也使用线条将信息分成块。

**网格**

网格是由设计者创建的，用来帮助在页面上定位元素。网格本身在最终设计上是不可见的，但有时会画网格线来加强网格的效果。这有助于保持网站、杂志、小册子或宣传手册的一致性。Black Estate Wine 网站使用 12 列网格，在主标题下有可见的线条，并且在侧边栏中有单独的链接。

![BlackEstate](../Images/b1729a4038b1c44b6f27562167fd2a0c.png "BlackEstate")

《大河人》是一部独立纪录片。在这张海报中，线条坐在网格上时几乎被用作标点符号。你可能期望句号的地方，你看到的是一条橙色的水平线，而你可能期望逗号的地方，你看到的是一条垂直线。所以这是一个线条连接的例子，而不是分开内容。

**线条提供纹理**

你使用的线条类型可以给作品增加质感。粗糙的手绘线条给人的感觉与精确的发际线截然不同。作为一个简单的例子，看看下面的电影海报。

《远走高飞》的海报上有很多线条。线条松散地画在背景中，在两个主要人物的照片上。这种设计显然是针对特定观众的，并且有一种新潮的独立感。

**线路使用指南**

1.想想你会用什么样的线。粗、细、波浪形、点状、虚线、手绘。每一种线条都能传达不同的感觉。

2.这条线的目的是什么？是加入相关内容还是分离无关内容？是在一张图片周围加边框吗？

3.如果你使用网格，你的线条是在帮助加强网格，还是在打破网格？两者都可以是好的，只要你知道这些线条应该做什么，并且你正在有意识地做出决定。

4.想想用多条线来创造质感。重复的细线可以作为背景纹理，也可以是手工绘制的线条图案。

5.不要为了放线而放线。和任何设计元素一样，如果你不需要它，就把它拿走。
