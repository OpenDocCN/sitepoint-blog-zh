# OSCON 2007:人民黑客

> 原文：<https://www.sitepoint.com/oscon-2007-people-hacks/>

Adam Keys 是一名软件开发人员和作家。他的博客在 therealadam.com。

在这次演讲中，Adam 介绍了一些有效宣传和与其他开发人员相处的方法。换句话说，与人互动和与硬件或软件互动一样重要。这一领域的问题没有技术解决方案——它们需要社会解决方案。因此，人们黑客的想法。

People hacking 并不是植根于邪恶的黑帽黑客，而是简单地使用社交符咒工具(有时甚至是对你自己)来获得你试图倡导的想法的牵引力。更多地关注人的方面可以帮助你建立一个完整的、运作顺畅的团队，能够执行(借用一个篮球的比喻)“无表情传球”

一些示例想法包括:

*   简单的微笑——让你自己和你周围的人感觉更积极
*   避免消极——试试 ABBA 方法(详见幻灯片)
*   避免批评
*   批评之前先赞美

一些想法是关于如何在你的组织中处理混蛋的，包括简单但有效的“无混蛋规则”(来自同名书籍)，鼓励对你的团队中的混蛋行为零容忍——即使是所谓的超级明星程序员。

亚当还鼓励进行一点自我评估，以确定你自己是否是一个正在恢复的混蛋，通过专注于提高你与周围人良好合作的能力，你也可以改造自己。
