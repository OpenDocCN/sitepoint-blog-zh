# 如何克服限制性信念和不安全感，成为更有效的营销人员

> 原文：<https://www.sitepoint.com/how-to-overcome-limiting-beliefs-and-insecurity-to-be-a-more-effective-marketer/>

“不管你认为你能还是你认为你不能——你都是对的。”亨利·福特。

上两篇博客引出了一个大问题:如何克服不安全感，成为一名高效的商务人士？

一个没有安全感的人如何突然转型变得有安全感和自信？

在这样一个简短的博客中，我所能做的就是给你指出一些方法，你可以追求那些召唤你的方法。

首先，让我们弄清楚一件事:你的行为和言语需要一致。有很多人(我曾经是其中之一)会说这样的话，“我想创业，但是……”或者“总有一天我会成为一名成功的企业家，但现在我太老了，太年轻了，没受过教育，太穷了，还不为人知……]”

像这样的几条评论时不时的就好了，但是有些人(我曾经也是其中一员)没完没了。这就好像他们认为谈论创业实际上是在做一样。这些人是崇拜者。他们让家人、朋友和同事感到厌烦——而自己却不知道。

更糟糕的是，当你在做完全不同的事情时，说一件事是不健康的。

为了成功和满足，你必须匹配你的行动和你的行为。要么做你说过你会做的事情(例如，停止解释和找借口，实现你的梦想)，要么停止用你的白日梦来烦我们，开心地生活。

现在，对于那些真的想采取行动，但却停滞不前的人，这里有一些前进的方法。

首先，找到那些不顾自身境遇而取得成功的人。采访他们。仿效他们的行为。找出他们如何克服自己的不安全感。赢得让他们指导你的权利。主动为他们工作，或者向他们学习。

第二，研究你的限制性信念，直到它们变得如此乏味以至于你忽略它们。有时候，你必须对自己感到极度厌倦和厌恶才能前进。然后你的信念就变成了一部在你脑海中播放的烂片，对你没有任何影响。正如一位匿名的古鲁所说，“思想是头脑放屁的方式。”那么是什么让你的信念产生的呢？你多久注意到他们一次？它们对你有什么影响？等等。等等。令人作呕。(这是一种佛教/冥想的方法)。

第三，慢慢走出你的舒适区。每周做一些你以前没有做过的事情来发展你的业务。如果你害怕给“大鱼”客户打电话，打一个看看会发生什么。学习，再试一次。继续多走一小步。(这就是 ChangeOne 理念)。

第四，意识到不改变自己行为的代价。在做了一个练习后，我开始为自己做生意，在这个练习中，我看到自己又为我那智力有问题的老板工作了一两年。然后我就在脑子里放大痛苦，直到我再也受不了了。(这是一种神经语言编程技术，下一个也是)。

第五，相反，想象采取新的、强有力的行动来发展你的业务的快乐。你会有什么感觉？你会怎么说？你的面部表情会是什么样子？你将如何表现自己？你会穿什么？

第六，造假，直到做出来。桑德勒的著名著作《你不能在研讨会上教一个孩子骑自行车》把我们的身份和角色分开了。我们都是演员。做成功人士做的事，知道我们在一定程度上都是在假装。你不是你的工作。为了成功，你需要做什么就做什么。

目前有六种方法就够了。对于 16 岁的孩子:你可能需要耐心。有了经验和多一点时间，你会做得很好。你已经远远领先于大多数同龄人。

最后，引用尤达的一句话:“没有尝试，只有做。”

其他建议不胜感激…..
