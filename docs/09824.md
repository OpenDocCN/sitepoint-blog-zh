# 如何计算浏览器市场份额

> 原文:[https://www . site point . com/how-browser-market-share-is-calculated/](https://www.sitepoint.com/how-browser-market-share-is-calculated/)

我每一两个月发布一次浏览器市场份额报告。我希望你对它们感兴趣，或者至少对 IE6 使用率下降的消息感到高兴。然而，他们经常导致更多的问题，而不是答案。这是我试图解释浏览器市场份额是如何计算的。这是一个理论概述，而不是数学论文，计算方法将因系统而异。

## 谎言，该死的谎言和统计数据

…然后还有网络统计。众所周知，整理来自网络浏览器的信息非常困难——请看[为什么你的网站统计报告是错误的](https://www.sitepoint.com/website-statistics-reports-wrong-3/)。StatCounter 和 Google Analytics 生成的报告非常漂亮，数字精确到小数点后几位，但是你应该知道，分析是基于一系列假设的。

这并不是说网络统计是无用的。它们非常适合发现趋势，但试图将结果映射到个人用户的活动上通常是徒劳的。

## 什么市场份额不是…

在我们讨论什么是市场份额之前，也许最好先确定什么不是市场份额:

**1。浏览器下载量**
浏览器下载量不是比较市场份额的可行方法:

*   所有操作系统都提供了默认浏览器，许多用户永远不会考虑替代浏览器。
*   你可以多次下载一个浏览器，但永远不要安装它。
*   你可以下载一个浏览器，然后安装在数百台电脑上。
*   实现自动更新的浏览器将处于不利地位。

**2。浏览器安装**
浏览器安装也有类似缺陷:

*   您可能永远不会使用操作系统附带的浏览器。IE 安装在所有 Windows 电脑上，但这并不意味着 IE 和 Windows 一样拥有 90%的市场份额。
*   你可以安装另一个浏览器，但永远不使用它。
*   您可以多次安装和卸载同一个浏览器。

这并不是说这些数字没有用，但它们不一定代表市场份额。

## 那么什么是市场份额呢？

混淆的第一个原因是市场份额表显示了用户的百分比。实际上，市场份额是由浏览器的实际使用情况决定的。这些数字是概率。

假设 browserX 拥有 50%的市场份额。如果您检查随机网站上的随机文件命中，browserX 被使用的几率是 1/2。不管你看的是随机点击、访问者会话还是单个用户，概率都是一样的。

因为我们计算的是浏览器使用比例，所以底层数据不需要记录个人。然而，结果仍然与用户直接相关。我们可以得出结论:

*   每个人都有 50%的时间使用 browserX，或者
*   更现实的是，50%的用户一直在使用 browserX

结果介于这两个极端之间。最终，这无关紧要——我们正在分析一个群体的使用模式。

## 如何计算市场份额

当您访问网站时，每个文件请求(点击)都会被记录下来，并且您的浏览器会根据 HTTP 头中传递的用户代理字符串来识别。本质上，如果在 P 期间，50%的点击请求来自 browserX，那么它在该时间具有 50%的市场份额。

现实要复杂一点。文件命中可能是不明确的，因为不同的浏览器可以下载不同的资源，例如 IE 条件样式表或预先缓存的链接页面。因此，系统可能只分析实际页面视图或进行其他调整。

下一个重要的考虑因素是样本大小——分析了多少网站和点击量。没有所谓的“普通”网站:

*   SitePoint.com 拥有技术受众，因此 Firefox 和 Chrome 的使用率高于其他国家。
*   观众所在的国家也有影响，即歌剧在欧洲和俄罗斯比在美国更受欢迎
*   日期和时间会影响使用模式。例如，在工作日办公时间，IE 的使用率通常比晚上或周末高。

如果你分析来自许多不同国家的大量网站，统计异常就会减少。本质上，更多的数据导致更准确的浏览器使用数据。StatCounter 分析了全球 300 万个网站的流量——这似乎是一个健康的样本规模。

## 啊，但是如果……

下面是常见的浏览器市场份额问题列表。如果我没有回答你的问题，请在下面留下评论。

问:互联网的使用正在增长。
网民的数量每天都在增加。因此，当实际用户数量增加时，浏览器的市场份额可能会下降。

将网络用户的数量乘以浏览器的比例来估计人口的变化…假设你能找到合理的网络使用数字。

问:访问者或用户会话会比页面/文件点击更准确吗？
不会。这不会产生更好的数据，因为你减少了样本量，引入了意想不到的问题。如果你只有会话数据，那么在一个网站上 3 小时的娱乐性浏览相当于 30 秒浏览一个与工作相关的话题。由于许多人在工作中使用 IE 浏览器，它会比其他在家里使用的浏览器受到不公平的歧视。

请记住，我们正在分析浏览器的使用:没有必要去理解个人用户的行为。

问:我使用不止一个浏览器。我被多次计算了吗？
没关系。个人通常有复杂的浏览模式，例如，你可能 80%的时间使用 Firefox，20%的时间使用 Chrome。该使用被记录；如果你是唯一一个被抽样的人，Firefox 将拥有 80%的市场份额。

问:极客们一天使用网络 18 个小时怎么办？
市场份额是浏览器使用情况的记录。重度用户的浏览比每个月运行一次 IE6 的人更有分量。

然而，假设样本量足够大，个人或群体浏览习惯的影响可以忽略不计，不会扭曲结果。对于每个极客或技术恐惧症患者来说，有成千上万的人每天使用网络一两个小时。

问:我的浏览器使用了不正确的用户代理字符串。
它不会被正确识别，但是，再说一次，你是少数，它不太可能显著影响结果。

问:地区或单个国家的结果有多准确？
样本量越小，数据越不稳健。比起南极的数据，我更相信美国的数据。

问:我的站点的统计数据不同？
他们会。影响浏览器使用的因素很多，很少有网站能与全球平均水平相比。总是先检查你自己的数字。

问:这些数字我都不相信！
适度的怀疑对你有好处。在不了解基础数据或分析的情况下盲目使用报告是危险的。
