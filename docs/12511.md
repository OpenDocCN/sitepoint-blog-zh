# HelpingKind.org:一个向前支付的市场

> 原文：<https://www.sitepoint.com/helpingkindorg-a-pay-it-forward-marketplace/>

除了是一部很容易被人遗忘的凯文·史派西电影之外,“向前支付”这个概念如果真的付诸实施，可能会帮助很多人。这个想法是，我们都擅长某事，我们都有能力帮助别人，我们都需要帮助。如果你是一件好事的受益者，尽你所能帮助其他需要帮助的人。正如电影《T2》中的一个角色所说，这形成了一种人们互相帮助的“特里萨修女康加线”。

根据维基百科，这个想法是由本杰明·富兰克林在 1784 年首次提出的。“我不会假装给这样一笔钱；我只借给你，”他在给本杰明·韦伯的信中写道，他曾借钱给韦伯。“当你[……]遇到另一个处于类似困境的老实人时，你必须借出这笔钱给我；责成他清偿债务的类似操作，当他能够，并应满足另一个机会。我希望在它遇到一个无赖来阻止它前进之前，它能经过许多人的手。这是我用少量的钱做大量的好事的一个诡计。"

这种渐进式互惠的概念最近打动了 SitePoint Forums 的长期成员戴夫·道尔顿，并促使他创建了一个名为 HelpingKind.org 的市场，名为 T2。他本周试运行了新网站。

“这一切都是从一个朋友开始的，他有一些医疗问题，不能照顾家里的一些事情，”道尔顿说。这位朋友没有雇人来解决他家周围的一些问题，而是在一个周日的下午，其他几个朋友聚在一起伸出了援助之手。道尔顿说，这次经历让他的一群朋友走得更近了，帮助需要帮助的家庭减轻一点压力感觉很好。

“在(那天)之后，我只能想起‘过去的日子’，”他说，“当一个社区的成员需要帮助时，整个社区都会团结起来的日子。你的谷仓被烧毁了？几天之内，全镇的人都来帮你建造另一个。为什么？因为社区里的每个人都知道这很可能是他们的谷仓！”

道尔顿说，“人们帮助人们”是一个古老的想法，但他认为这个想法在今天仍然有很大的相关性。因此，在过去的一个月里，他花了大部分时间将 HelpingKind.org 建设成一个乐善好施者的市场。当用户需要帮助时，他们可以发布需求，并被鼓励承诺用未来的善行来回报这种善行(或者用实物捐赠的帮助与帮助者交换)，如果他们可以的话。其他人可以在市场上为需要帮助的人冲浪。

“如果我们都能联系到我们认识的每一个人，无论他们是否需要帮助，我们都可以将这个网站建成一个人们互相帮助的大规模网络，一个实现今天仍然有效的‘老派’原则的社区网络，”道尔顿告诉我。“从我们的日常习惯中抽出一两个小时来帮助别人，不仅帮助了那个人，也让整个社区团结起来。它以一种特殊的方式将人们聚集在一起，提高每个人的生活质量。它重新点燃了这样一种理念，即无论人们多么富有或贫穷，无论他们的生活多么复杂，他们都是彼此的邻居，共同构成了一个强大的社区。”

这个想法可能看起来过于理想化，也许确实如此。但我们为道尔顿的努力鼓掌，祝他好运。在内容上，他一开始就面临一个先有鸡还是先有蛋的问题。没有内容很难促进参与，没有用户参与也很难获得内容，所以现在，网站感觉有点空。
