# 实验闪光灯:开发环境

> 原文：<https://www.sitepoint.com/experimental-flashers-development-environments/>

我们都见过一些实验性的网站，它们拓展了我们认为 Flash 可以做的事情的范围；在我看来，最著名的是 levitated.net 和 T2 的比特 101，这两家公司都是由极具天赋的个人经营的。当然还有更多的实验地点，但这两个总是提醒我，作为有趣和迷人效果的开发平台，Flash 是多么多功能。

由于 Flash player 的限制，为了将插件的下载大小保持在可接受的水平，需要在插件的下载速度和封装的功能以及呈现速度之间进行权衡。不要误会我的意思，Flash player 对于我们看到的一些实验效果和作为数据交换的管道来说是很棒的，但是正如你可能已经看到的，当屏幕上的许多对象的效果变得复杂时，事情往往会变慢，并可能会突然停止。

有些人使用 Director 来实现某些效果，但其他人则使用基于 Java 的环境作为他们的数字游乐场，例如一个名为 Processing 的免费产品，因为它可以快速进行密集的数学计算。

Processing 是一种有趣的免费编程语言和开发环境，它实际上有一个非常强大的 API，并且已经存在了一段时间。levitated.net 的贾里德在处理环境方面做了很多工作，并创造了一些简单的令人惊叹的作品。

如果你对数学和图形实验感兴趣，这绝对是值得研究的事情；我已经涉猎了几个星期了，用几个简单的迭代函数就能做的事情令人惊讶！

[点击此处](http://processing.org/index.html)了解处理软件的概况。
