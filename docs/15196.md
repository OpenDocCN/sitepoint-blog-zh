# 从可悲到哇！–通往令人惊叹的网站的 6 个步骤

> 原文：<https://www.sitepoint.com/wow-6-steps-stunning-site/>

许多优秀的网页设计师完成了他们的网站——模板、内容等等——却发现它看起来一点也不专业。

这是一个常见的问题，本教程是为那些即将开始一个网站，正在开发过程中，或者只是需要一些指点的人而写的(当然，也包括那些不介意过程中有一点“新手”幽默的人)。

##### 1.不要吃我！

当你的潜在客户发现你的服务被评为介于地核和泰坦尼克号之间时，这可不是一个好日子。

给你的用户在你的网站上发表公开评论的能力是摧毁你自己的最有效的方法之一——让你的用户把你生吞活剥。

绝对不要让一个用户(或多个用户)有诋毁你的网站的权力。这种形式包括带有不可删除条目的留言簿，以及其他类似的不太明智的想法。我最喜欢的一个是“投票让我离开网络”投票，用户可以给一个网站打分，比如 1-5 分。

##### 2.点击计数器

这个网站如此绝望，以至于他们必须向用户证明他们有有价值的内容吗？

似乎很多站长喜欢在他们的网站上展示他们令人印象深刻(或不那么深刻)的点击率，然后当他们收到负面反馈时会感到惊讶。

对于一个相对来说比较新的网站管理员来说，这也是看起来很“酷”的事情之一，但是许多人认为点击计数器比它们的价值更麻烦。如果你真的想知道你的统计数据，添加一个看不见的计数器。

##### 3.“免费广告”礼物

他们真的如此伟大，值得免费广告吗？

我知道我们都喜欢表达对 ie 浏览器的支持。我们也对网景公司的存在感到遗憾。我们也是 WinZip、FrontPage 等网站的终身仆人。

至少当我浏览的网站中有一半显示这些程序的小按钮时，看起来是这样的。你知道，那些 88×31 的广告图片？但是这些程序真的如此伟大以至于值得在你的网站上做免费广告吗？

如果你用过这些产品，并且觉得它们很方便，那就太好了。但是向全世界展示你对这些产品的看法并不是一个好主意。为什么？因为你在浪费带宽，牺牲下载速度。更重要的是，你让“老大们”赢得了你的信任。

无论如何，记住你必须保持专业，真正的专业人士不会提供没有价格的好处(比如广告)。

##### 4.视觉优势

谁会想到万维网是为网站而设计的，而不是为幻灯片演示而设计的？

有几件事可以帮助你为网站的访问者创造一个更愉快的氛围。在向公众发布网站之前，将它们添加到您的清单中:

1.  确保你有 2-4 种颜色的明确配色方案。
2.  确保背景和文本之间有足够的对比度。黑色背景和红色文字是绝对不够的。
3.  移除任何“酷”的鼠标效果。这里，我指的是像跟随指针的单词这样的东西。
4.  不要添加页面过渡，如“框入”和“擦除”。
5.  去掉任何不微妙的页面背景。此外，尽量确保这些背景的颜色是一样的。例如，不建议将黄色、紫色和白色的背景混合在一起。
6.  避免全大写。又烦又乱。

##### 5.地球之旅

对某些事情来说，免费主机很棒。它非常适合小彼得的生日照片，或者杰米的毕业留言簿，甚至大鲍勃的烘焙义卖信息。然而，还记得之前的那条建议吗？

“无论如何，记住你必须保持专业，真正的专业人士不会不计代价地提供好处。”

免费主人收取的服务费用就是你的专业水平。你可以享受免费上传的好处，但这会让你名誉扫地。让彼得、杰米和大鲍勃在地球上安家。但是你很聪明。你很专业。你会做好的。

##### 6.移动花生酱

你见过像花生酱一样快速移动的网页吗？每当我的 DSL 调制解调器为一页制作出价值 23 万的动画 gif 时，我都会看到这种情况。移动花生酱是那些网站的所有者拒绝使用除动画以外的任何形式的艺术作品。世界上有谁愿意等待这些网站加载，更不用说观看所有的铃声和哨声了？如果你想让你的网站看起来不错，那就帮这个星球一个忙。不要过度使用动画！
