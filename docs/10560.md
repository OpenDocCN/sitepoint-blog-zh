# IE 市场份额跌破 50%

> 原文:[https://www.sitepoint.com/ie-market-share-falls-below-50/](https://www.sitepoint.com/ie-market-share-falls-below-50/)

2010 年 8 月，[谷歌 Chrome 超过 10%的市场份额](https://www.sitepoint.com/google-chrome-market-share-10-percent/)。另一个里程碑是在 9 月份实现的:ie 浏览器的总使用率十年来首次下降到 50%以下。

整个 web 设计和开发社区都欢呼雀跃，这个故事也传遍了技术和主流媒体。然而，详细检查一下 [StatCounter 统计数据](https://gs.statcounter.com/)是值得的…

| 浏览器 | 八月 | 九月 | 变化 | 亲戚 |
| IE 9.0 测试版 | 0.00% | 0.09% | +0.09% | 不适用的 |
| IE 8.0 | 29.40% | 29.38% | -0.02% | -0.10% |
| IE 7.0 | 13.91% | 12.98% | -0.93% | -6.70% |
| IE 6.0 | 8.02% | 7.42% | -0.60% | -7.50% |
| 火狐 4.0 测试版 | 0.00% | 0.26% | +0.26% | 不适用的 |
| 火狐 3.5 以上版本 | 28.03% | 28.33% | +0.30% | +1.10% |
| 火狐 3.0 以上版本 | 2.60% | 2.48% | -0.12% | -4.60% |
| 铬 | 10.76% | 11.52% | +0.76% | +7.10% |
| 旅行队 | 4.06% | 4.22% | +0.16% | +3.90% |

*“变化”栏显示市场份额的绝对增加或减少。“相对”栏表示相对移动，即 7.5%的 IE6 用户在过去一个月中更换了浏览器。*

我们不能从这些数据中做出太多的假设，但是有一个统计数据记者似乎忽略了:IE8 的使用几乎没有变化。IE9 测试版带来的 0.09%的增长远远超过了 0.02%的降幅。

IE 丢失的最重要的促成因素是从版本 6 和 7 的迁移。尽管这些用户中的一部分将会迁移到 IE8/9，但是更多的人已经转而使用 Firefox 或 Chrome。出现这种情况有几个原因:

*   IE8 是一个功能强大的浏览器，但是对于 IE6/7 用户来说还有很多更好的选择。
*   好的 web 应用程序是与浏览器无关的，或者可以在各种平台上工作。遗留的商业应用程序正在更新，对 IE 的依赖减少了。
*   微软和所有其他主要供应商都支持 HTML5。浏览器之间的渲染差异比以往任何时候都小，您使用哪个应用程序都无关紧要。
*   大多数企业使用 Windows XP，并且可能无意升级，尤其是在持续的经济不确定时期。然而 IE9 只能在 Windows Vista/7 上使用。为什么企业会继续使用供应商已经(实际上)放弃的应用程序？安装替代浏览器比升级操作系统要便宜和容易得多。

虽然我们应该感谢 IE6/7 使用率的下降，但仍有五分之一的访问者继续使用古老的浏览器。对 IE 灭亡的预测还为时过早，IE8 仍然是世界上使用最多的浏览器。

这对其他供应商来说是个好消息。Firefox、Safari 和 Opera 都有所增长，但 Chrome 仍然是最大的赢家。谷歌的浏览器每月增长近 1%，而且没有见顶的迹象。然而，它即将面临来自 Firefox 4 和 IE9 的更强挑战。
