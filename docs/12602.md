# 投票:你如何看待苹果新的攻击广告？

> 原文：<https://www.sitepoint.com/poll-what-do-you-think-of-apples-new-attack-ads/>

围绕美国总统大选的日益敌对的气氛似乎已经渗透到了苹果和微软的广告大战中。最新的攻击来自苹果公司，在他们成功的“买一台苹果电脑”活动中，两个新广告把每日秀的 T2 约翰·霍吉曼塑造成书呆子，脱离个人电脑，演员贾斯汀·朗扮演时髦年轻的苹果电脑。

新广告的问题在于，它们没有指出 Windows 的问题，也没有大谈 OS X 的优点，而是将目标对准了微软新的 3 亿美元广告宣传活动——不是根据价值，而是根据价格？问题是，对于普通大众来说，他们可能有点太深入棒球的内部了，他们很虚伪，而且他们，嗯，心胸狭窄，而不是提供信息。

第一个新广告嘲笑微软试图通过营销解决 Vista 的形象问题，指责微软花费 3 亿美元进行新的广告宣传，而不是将这笔钱投入 Vista。

不过，考虑到苹果广告在过去几年里是多么无孔不入，这有点虚伪。如今，打开电视很难不看到“买一台 Mac”、iPod 或 iPhone 的广告，这些广告也出现在知名网站和许多杂志上。事实上，苹果在 2007 年花了 4.67 亿美元做广告，而微软花了 13 亿美元——大约是它的三倍，但该公司的产品线更广，赚的钱也更多。微软今年的广告支出实际上略有下降。

微软也在 R&D 上花了更多的钱，比苹果去年的 7.82 亿美元多了 71 亿美元。微软在研发上的投入占总收入的 14%,苹果在研发上的投入约占净销售收入的 3%。不过，微软的产品线更广，苹果可能会说微软需要改进的地方更多。

撇开事实和数据不谈，关键是苹果的新广告感觉像内部笑话，并没有真正向我推销 Mac，只是看起来有点刻薄。他们肯定都很有趣，但在我看来，他们把苹果描绘成一个精英恶霸，与消费者关心的事情脱节，而不是可爱的失败者。这真的是他们现在想要的吗？或者这些广告只是为了迎合苹果用户群？(借用政治领域的一些术语。)
