# 如何获得你的第一个客户

> 原文:[https://www.sitepoint.com/how-to-get-your-first-client/](https://www.sitepoint.com/how-to-get-your-first-client/)

这里有一些想法给那些想要得到第一个客户的人，尽管这些想法适用于任何人的任何发展阶段:

1.写下你认识的每个人的名字，每个人。你应该能得到一份 100 人的名单。使用一些技巧来“唤醒”你的记忆。例如，浏览电话簿的业务页面，列出你认识的任何专业人士(会计师、建筑师等)。一直到 z)。包括你的邻居，你的家人，你以前的雇主/同事，和你一起运动/爱好的人，你做礼拜的地方的人，等等。列出红头发的人，有幽默感的人，叫汤姆的人，等等。

然后联系这些人，告诉他们你在做什么，请他们为你建立联系。问他们一些具体的问题，比如“你认识谁最近创业了？”然后跟踪那些人。要么他们需要一个网站或者重新设计，要么他们可以把你介绍给他们认识的人。

对于每一扇关闭的门，试着通过询问两个名字来打开另外两扇门。

2.瞄准一些看得见的当地企业。去那个网站看看。通过邮件向他们发送一份 3 页的报告，评估他们的网站并提供改进的想法。然后跟进会面，讨论他们如何利用你的想法获得更多业务。

3.我借用营销顾问大卫·弗雷的这个想法，我真的很喜欢它。从有潜在客户名单的黄页中选一页。把那一页放大到现在大小的 2-3 倍。圈出潜在客户的竞争对手的名字。写一封这样的信，“我将帮助这一页上的一家公司通过一个动态的、领先的网站大幅增加他们的业务和市场份额。我希望是你，而不是你的竞争对手！”

给你——3 个免费的好主意。创意便宜是有原因的。行动和跟进才是有价值的。所以开始行动吧。
