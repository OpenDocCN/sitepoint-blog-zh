# Firefinder 为 Firebug 增加了搜索功能

> 原文：<https://www.sitepoint.com/firefinder-adds-search-to-firebug/>

每个人都喜欢[Firebug](http://getfirebug.com/)，它最好的一点是它不仅仅是一个扩展——它是一个*扩展框架*。你们中的许多人可能已经很清楚[我们的 CodeBurner](https://getfirebug.com/)Firebug 和 [Yahoo！长期以来，s YSlow 性能监视器](http://developer.yahoo.com/yslow/)一直是许多开发人员的支柱。

罗伯特·尼曼又想出了一个非常好的补充——火虫的[火眼探测器。](http://robertnyman.com/firefinder/)

安装 Firefinder 后，你会在 Firebug 中发现一个闪亮的新标签，带有一个友好的搜索框。想知道特定的 ID 在页面中的什么地方被使用吗？输入 ID(我在下面的例子中使用了' #feature ')并点击回车键，您将看到:

*   用红色虚线突出显示的元素
*   下面列出了相关的代码片段

![Enter any ID, class or HTML element to see it highlighted in the page and listed in the firefinder panel.](img/33a12dd0e50219e92a19635c50552846.png)

同样，HTML 元素和类也可以很容易地被搜索和高亮显示。将鼠标悬停在匹配元素列表上会切换元素，以蓝色突出显示该元素，这对于隔离复杂页面中共享公共类的特定 div 非常方便。

如果你想变得更狡猾，你甚至可以通过元素的 XPath 来定位元素。

显然，Firefinder 不应该是 Firebug 默认界面的替代品，但它确实是一个很好的增强。[来看看](http://robertnyman.com/firefinder/)。
