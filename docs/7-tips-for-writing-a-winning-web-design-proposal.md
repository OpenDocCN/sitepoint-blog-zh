# 撰写成功网页设计提案的 7 个技巧

> 原文：<https://www.sitepoint.com/7-tips-for-writing-a-winning-web-design-proposal/>

如果你在用传统方式征求网页设计工作，赢得工作的一个重要步骤就是提案阶段。无论是在电子邮件中概述你必须提供的内容，还是提交一份完整的建议书，你都可以做一些事情来尽可能地展示你自己和你的能力。

这里有七个建议可以帮助你改进你的提案。

**1。做你的研究**

也许你能在你的提案中展示的最重要的事情之一就是你了解客户，你了解他们想要什么。这是迎合客户个人需求的第一步。研究该公司，并彻底审查他们的 RFP(提案请求)，然后具体说明他们需要解决的问题。详细重申他们的需求，为你解释如何比别人更好地满足他们的需求做好准备。

**2。具体说明你的能力** 
展示你能力的一个有效方法是回顾客户的需求，这是你之前在提案中确定的，并陈述你对每一项的体验。你对如何解决客户的问题越具体，客户需要做的假设就越少，他们在决定雇佣你时就越有安全感。

**3。提供例子和/或参考**

如果可能的话，给你的潜在客户提供过去相关工作的可视例子，每个例子都要提到他们陈述的需求之一。他们想知道你能做什么，但最重要的是，客户想知道你能为他们做什么。如果你在被邀请之前提供证明人，这也很有意义。

**4。提出一个他们无法拒绝的提议**

根据您所掌握的信息，提供尽可能完整的成本和交付信息。为这项工作提出一个详细的时间表。如果你通常按项目报价，列出估价中包含的所有东西，即使看起来微不足道。并为提案设定时限。这将鼓励客户采取行动，或者至少通过请求延期来表达兴趣，你可以决定是否同意。

**5。校对**

这看起来很容易，但是在提交之前要校对你的提议。没有什么比给客户一份有错别字或语法错误的建议书更糟糕的了。让别人帮你校对，确保没有错误，格式一致，看起来专业。你想让客户知道你花时间为他们准备提案。

**6。可访问**

请确保提供您的联系信息并邀请提问。你甚至可以提议在他们的初步评估后召开一次电话会议，讨论他们的问题或顾虑。

**7。概述后续步骤**

包括一个强有力的结论，让客户采取行动。重申提案条款的到期日；邀请他们随时与您联系；并告诉他们您需要什么才能开始(即，已签署的合同和押金)。

任何提议中最重要的一点是向潜在客户展示你了解他们的需求，然后让他们相信你是这份工作的最佳人选。记住这一点，你就走上了成功撰写建议书的道路。
