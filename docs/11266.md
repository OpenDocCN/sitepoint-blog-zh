# 做还是不做…企业主

> 原文:[https://www.sitepoint.com/should-i-start-a-business/](https://www.sitepoint.com/should-i-start-a-business/)

![question](../Images/70aa1e9eb79e0a7dd70e34d39e581316.png) *这是[创业](https://www.sitepoint.com/business-start-up-basics/)系列文章的第一篇。*

许多企业家在他们生命的早期就知道拥有一家企业是他们的使命，即使他们没有看到它立即开花结果。对于其他人来说，这更多的是一个逐渐的发现，是他们生活经历的结果。然而，这两个群体在决定成为企业主时，都有同样的问题需要考虑。

这个过程的第一步通常包括自我分析和诚实地审视你的处境、生活方式和目标。在你开始时，这里有一些具体的问题要问你自己。

*   我的经济安全吗？
*   当我的生意起步时，我是否有一个维持财务的计划？
*   我有创业所需的资金吗？
*   我对我正在考虑进入的行业有足够的了解吗？
*   在这个行业开展业务之前，我知道我需要做什么样的研究吗？
*   我将如何应对医疗保健？
*   我能身兼多职，做一点点事情吗，至少在开始的时候？
*   我对我将要做的工作有激情吗？
*   我是否愿意并且能够(大量)工作？
*   我独立且有决心吗？
*   我能做决定吗？
*   我有跟踪记录我开始做的事情吗？
*   我有家人的支持吗？
*   我是否灵活，能够随波逐流？
*   我对会计、市场营销和销售有基本的了解吗？
*   我有良好的沟通技巧吗？
*   我有韧性，脸皮厚吗？
*   我有创造力，思想开放吗？
*   我自信吗？
*   我愿意冒险吗？
*   我愿意做出牺牲吗？
*   当我需要帮助时，我能寻求帮助吗？
*   我可以授权吗？
*   我能看看大图吗？

这些问题可以提供一个很好的起点，但即使经过最深入的审查，你是否适合创业也不总是非黑即白的。在你冒险尝试之前，你可能不会有一个明确的答案。你可能在某一方面很出色，但在另一方面却很糟糕。你可能会在企业所有权的某个领域茁壮成长，而在其他领域苦苦挣扎。

你的目标应该是收集尽可能多的信息，以便做出明智的决定和承担适当的风险。你不会十全十美，但你应该能够在你擅长的和你不擅长的，你喜欢做的和你不能忍受的之间创造一个平衡。

你在考虑创业吗？你最担心的是什么？
