# Mac OS X 手册页的 GUI

> 原文:[https://www.sitepoint.com/gui-for-man-pages-on-mac-os-x/](https://www.sitepoint.com/gui-for-man-pages-on-mac-os-x/)

对于那些在 Mac 上的 SitePoint 社区的人来说(大多数情况下我也是)，Carl Lindberg 写的一个叫做 [ManOpen](http://www.clindberg.org/projects/ManOpen.html) 的好工具简化了在你的系统上访问和搜索 Man 文件。

它捕获了系统中所有手册页的列表，允许通过关键字进行搜索，还提供了一个可以阅读和滚动的查看器。虽然终端在大多数情况下可以用来查看手册页，但是在有些情况下，在手册页中进行一些简单的搜索可以节省时间。

Man entries 索引的好处是可以选择花时间浏览和学习在工作和研究期间尚未探索的新应用和程序…
