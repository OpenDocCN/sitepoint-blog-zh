# 平等相待

> 原文:[https://www.sitepoint.com/be-an-equal/](https://www.sitepoint.com/be-an-equal/)

当你和潜在客户交谈时，你把自己和他们放在平等的地位吗？如果你做到了，但没有令人讨厌或耍花招，你就有机会赢得他们的尊重，让他们更有可能雇用你。

以下是一些例子:

–我正在中国采购一种产品，许多供应商都在投标。一个供应商坚持要有一个原型，脱颖而出(其他人根据图片工作)。当我拒绝时，他们感谢我的时间。给我留下了深刻的印象，我认为他们是一个会从质量角度考虑我的需求的供应商。我寄了一份样本。

–一个 IT 供应商在投标过程中告诉我，项目的范围不合理，他可能帮不了我，而其他供应商根本没有回应。他解释了为什么他认为我的范围不合理，以及如何解决这个问题。他很有吸引力，所以我雇佣了他。

–一位潜在客户找到我，让我帮他们解决一个问题。我告诉他们，他们没有找到问题的根本原因，并告诉他们我对他们真正问题是什么以及如何解决的判断。他们雇佣了我。

–我认识一位专业人士，他每接受一个项目，就会拒绝 4 个项目。他寻找有非常具体标准的项目。因此，他为自己的成功做好了准备，获得了优秀的客户和出色的推荐。因此，他在自己的领域被称为“家伙”，可以收取高价。

这个原则并不总是导致工作，这并不是一件坏事。这是因为在平等的基础上，你更有可能发现什么时候前景是认真的，什么时候不是。这样，你可以在没有迫切需求(因为你会问)、没有预算(因为你会调查以确认他们有预算和权力)或没有前进时间表(因为你会问)的情况下，花更少的时间处理潜在客户。

通过保持平等地位，你可以获得你需要的信息，知道是否要继续一个项目，并让自己作为同事和值得信赖的顾问受到尊重，而不是乞求工作的供应商。
