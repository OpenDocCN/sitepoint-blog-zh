# 脸书下一步该做什么

> 原文:[https://www.sitepoint.com/what-facebook-should-do-next/](https://www.sitepoint.com/what-facebook-should-do-next/)

当我还是个学生的时候，我就一直在用脸书 T2，如果我墙上第一个帖子的日期准确的话，是 2004 年末。在这段时间里，我看到这个网站以及我对它的使用经历了许多变化。从将名字从 thefacebook.com 改为脸书，到向公众开放，并推出改变游戏规则的脸书平台，[全球最大的社交网络](http://www.techcrunch.com/2008/06/12/facebook-no-longer-the-second-largest-social-network/)在过去的 4 年里经历了一场变革。但是，即使脸书继续像野草一样生长，也有一些迹象表明，该网站的用户可能会变得疲惫不堪。

脸书正处于发展的关键时刻。在利润丰厚的美国市场，它仍然落后于 MySpace，它面临着来自谷歌 OpenSocial 平台的越来越大的压力，它正在努力证明其 150 亿美元的估值。那么，脸书下一步应该做什么？这里有两个激进的想法可以让它保持领先地位。

## 开放它的平台

不，我是说，*真的*开放了它的平台。一年多以前，通过 F8 平台向开发者开放脸书是一个明智的举动。有些人将脸书的平台增长与早年的微软视窗系统相提并论。虽然 Top Friends 和 Scrabulous 很难称得上是微软 Office 和 Outlook 在 Windows 上的杀手级应用，但这显然是脸书试图实现的故事。他们的计划是这样的:创建一个平台，吸引开发者，让用户迷上杀手级应用，锁定并留住他们，在你变得不可或缺后赚钱。

抛开对数据可移植性的担忧(我不相信早期采用者之外的人真的会在意)，这是一个非常好的计划。如果他们成功了，由于有机增长的应用生态系统，脸书将成为用户必去的目的地。

不过，有一种方法可以大大加快这个过程:进一步开放。我说的不仅仅是应用平台，还有所有社交网络的管道。最近，[宁](http://developers.facebook.com/fbopen/>open%20sourced%20the%20Facebook%20Platform%20in%20an%20attempt%20to%20spread%20Facebook%20apps%20around%20the%20web%20in%20the%20same%20way%20Google%20has%20done%20with%20Open%20Social.</p>%0A<p>The%20same%20concept%20could%20be%20applied%20to%20the%20entire%20site.%20%20They%20don%E2%80%99t%20need%20to%20make%20all%20of%20Facebook%20open%20source,%20but%20rather%20offer%20a%20<a%20href=)式托管的小众社交网络平台——让任何人都可以创建自己的小众脸书。用一个内置的社交图来称呼它宁。宁让像疯了一样发展，因为它的客户做了所有的推广工作。脸书可以做同样的事情，在数以千计的小众脸谱网上传播自己。

随着企业在 web 2.0 技术(包括社交网络)上的支出预计到 2013 年将达到 43 亿美元，他们可以给他们的产品贴上白色标签，并在此过程中赚点小钱。

## 成为一台虚拟计算机

实际上，这并没有那么激进。众所周知，脸书的目标是成为未来的网络操作系统(或者甚至是未来的网络，所有的东西都流经脸书)。然而，为了成为网络操作系统，脸书需要做两件事:他们需要一些应用程序开发人员为他们的平台开发一两个杀手级应用程序(这是任何人都离不开的)，他们需要开始成为你的大脑。

前者很清楚，后者需要一些解释。个人电脑的一大优点是它是一个数据转储。照片、视频、笔记、报税单——我们做的很多事情最终都存储在我们的硬盘上，所以我们不必记住它们。脸书已经成为我们生活中某些部分的数据转储站——照片、个人通讯、我们的地址簿和日历——这个网站就像大脑一样为我们记录这些东西。但是它可以做得更好。

脸书想要成为网络操作系统的原因是，它希望我们变得依赖脸书，就像我们依赖我们的个人计算设备(个人电脑、笔记本电脑、智能手机等)一样。).现在不是这样的。为此，脸书需要吸收更多我们的数据。它是怎么做到的？微软有答案: [Live Mesh](http://www.mesh.com/) 。

Live Mesh 是一个数据同步平台，可以跨多个设备和网络即时同步数据。与竞争对手的数据同步服务不同，微软将 Mesh 设计成一个任何人都可以利用的平台。当他们在 4 月份给我看它时，他们运行的一个演示应用程序是为脸书运行的——它自动将手机上拍摄的照片与脸书的一个画廊同步。如果脸书能够自动记住我们做的每一件事，如果用户能够理解奥威尔式的暗示，它将真正成为我们生活中不可或缺的一部分。

对于任何试图开发我们最终都会使用的网络操作系统的公司来说，在所有设备上跟踪所有数据的能力是一个重要的难题。因为脸书已经与微软建立了牢固的关系，将 Mesh 平台集成到脸书操作系统中似乎是显而易见的。
