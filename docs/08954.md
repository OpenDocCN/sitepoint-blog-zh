# 2011 年 12 月浏览器趋势

> 原文：<https://www.sitepoint.com/browser-trends-december-2011/>

谷歌做到了。尽管是 Mozilla 浏览器的一半年龄，Chrome 已经超过 Firefox 成为世界上第二受欢迎的(*或最常用的*)浏览器。Chrome 可能只有 0.5%的优势，但这已经足够了——而且[我在 9 月份的预测完全正确](https://www.sitepoint.com/browser-trends-september-2011/)！让我们来看看最新的[全球 StatCounter 统计](https://gs.statcounter.com/):

| 浏览器 | 十月 | 十一月 | 变化 | 亲戚 |
| IE 9.0 以上 | 9.59% | 10.14% | +0.55% | +5.70% |
| IE 8.0 | 23.83% | 24.00% | +0.17% | +0.70% |
| IE 7.0 | 4.29% | 4.26% | -0.03% | -0.70% |
| IE 6.0 | 2.47% | 2.23% | -0.24% | -9.70% |
| 火狐 4.0 以上版本 | 18.87% | 18.62% | -0.25% | -1.30% |
| 火狐 3.7- | 7.52% | 6.62% | -0.90% | -12.00% |
| 铬 | 25.05% | 25.74% | +0.69% | +2.80% |
| 旅行队 | 5.93% | 5.90% | -0.03% | -0.50% |
| 歌剧 | 1.80% | 1.84% | +0.04% | +2.20% |

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 9.7%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

我不想破坏谷歌的派对，但 Chrome 的市场份额在过去一个月里增长了不到 0.7%。虽然其他厂商会对这一结果欣喜若狂，但这低于通常的 1%的增长率。Chrome 的数据开始波动，所以我怀疑它很快会达到饱和点。

对微软来说也有好消息。IE9 的市场份额增加了 0.55%，奇怪的是，IE8 也有小幅增长。总体而言，IE 的使用率增长了 0.45%——我不记得上次出现这种情况是什么时候了。

Opera、Safari 和较少使用的浏览器在 11 月份几乎没有变化。那就只剩下一个大输家了。正如预期的那样，Firefox 3.x 及以下版本下降了 0.9%，但版本 4 至 8 的总数也下降了 0.25%。Firefox 的性能正在迅速提高，仍然是 1/4 网络用户的首选浏览器，但这显然还不够。

三个因素导致了 Chrome 领先于 Firefox:

1.  谷歌已经花费了数百万美元在线和离线宣传他们的浏览器。
2.  Chrome 很简单——这是大多数用户都喜欢的——升级也很容易(如果不是完全看不出来的话)。我为新手用户安装 Chrome，因为一旦安装完成，我就不用再担心它了。
3.  你永远不会遇到插件问题——部分原因是 Chrome 的扩展提供了最低限度的浏览器集成，很少有用户安装它们。火狐就不是这样了；附加组件对浏览器的成功至关重要，而 Mozilla 并没有完全意识到兼容性问题。情况有所改善，但我怀疑许多用户失去了耐心，并转向别处。

不管原因是什么，谷歌值得他们的成功。Chrome 的下一个目标:ie 浏览器。

## 移动浏览器的使用

2011 年 11 月期间，移动使用量占所有网络活动的 [6.95%。](https://gs.statcounter.com/#mobile_vs_desktop-ww-monthly-201111-201111-bar)

主要的移动浏览应用程序有:

1.  opera Mini/Mobile——22.49%(增长 0.97%)
2.  安卓系统——20.41%(下降 0.47%)
3.  iPhone——19.53%(增长 0.75%)
4.  诺基亚浏览器——11.94%(下降 0.46%)
5.  黑莓手机——8.20%(下降 1.06%)

市场不稳定，受地区和季节变化的影响。如果您正在构建原生应用程序，这很少会有影响，但移动应用程序现在正在成为可行的替代方案，因为:

*   设备越来越复杂，并提供对 HTML5 APIs 的支持
*   为所有主流平台创建原生应用通常是不切实际的:iOS、Android、黑莓、Windows Phone、Symbian 等。

移动市场正在快速增长。你应该考虑一下；你的竞争对手肯定会。
