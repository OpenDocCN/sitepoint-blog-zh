表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 10.3%的 IE6 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

Chrome 在 4 月份又上涨了 1%。这种增长不会持续，但也没有停止的迹象。只有一个应用程序成功抵抗了:IE10。继 Windows 7 自动更新之后，微软的浏览器在一个月内增长了 4%。虽然其他版本都下降了，但 IE10 远远弥补了这一差异。

说到这，IE8 有 1%的下降，已经跌破 10%！虽然浏览器在纠正 IE6 和 IE7 中的问题方面做了很多工作，但是当我们可以依赖广泛的 HTML5 支持而无需垫片时，开发将会容易得多。它仍然有一个健康的百分比，但当微软明年停止支持 Windows XP 时，许多企业将被迫考虑替代方案。

IE6 和 IE7 已经变得几乎无关紧要。我很想把它们从图表中删除，但是…

1.  一些开发者用这些数字作为放弃老掉牙的浏览器的理由
2.  看着这些数字暴跌，我有一种自鸣得意的满足感。

然而，我很欣慰微软可以挑战谷歌，因为其他公司都在苦苦挣扎。

Firefox 拥有略高于 20%的用户，但下个月可能会跌破这个门槛。

Safari 这个月表现不佳，下跌了 0.5%。这是一个合理的浏览器，但如果苹果不在 iOS 上强制使用，它会这么受欢迎吗？如果 Webkit 开发明显落后于 Blink ，用户数量可能会进一步下降。

最后，上个月有 14%的 Opera 用户转而使用 Opera，现在已经下降到 1%。也许这是可以理解的；Presto 已经被 Blink 放弃了那么坚持使用一个即将被取代的浏览器还有意义吗？

## 移动浏览器的使用

2013 年 4 月，移动使用量略微下降至所有网络活动的 13.9%。

主要的移动浏览应用程序:

1.  安卓系统——30.96%(增长 0.18%)
2.  iPhone——23.94%(下降 0.50%)
3.  opera Mini/Mobile——15.35%(下降 0.19%)
4.  UC 浏览器——8.74%(增长 0.47%)
5.  诺基亚浏览器——7.03%(增长 0.07%)

与台式机市场相比，没有什么值得报道的。我想这是一年中平静的时候，没有重大的发布或颠覆性的技术。这里没什么可看的。请往前走，下个月再来！
