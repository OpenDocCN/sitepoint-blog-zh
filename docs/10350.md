# 2010 年:在 SitePoint 的那一年

> 原文:[https://www . site point . com/2010-the-year-that-is-at-site point/](https://www.sitepoint.com/2010-the-year-that-was-at-sitepoint/)

随着年底的临近，我想我应该去挖掘一下博客档案，把 2010 年以来 SitePoint tech 博客上一些值得纪念的时刻整理成一个日历。

**1 月:** [大家最喜欢的 JavaScript 库](https://www.sitepoint.com/jquery-14-released/)1.4 版本发布了，我们都欢呼雀跃。YouTube 和 Vimeo 增加了对 HTML5 视频的支持。在我的新闻帖子中，我对 T4 选择的 H.264 编解码器持批评态度——当时我们不知道谷歌会在不久的将来用 HTML5 视频做好事。

**二月:**二月对于网络开发者来说是一个好消息的月份，至少在浏览器方面:[微软在欧洲推出了“浏览器投票”](https://www.sitepoint.com/microsoft-european-browser-choice/)，要求 Windows 用户选择他们喜欢使用的浏览器，[谷歌放弃了谷歌应用程序对 IE6 的支持](https://www.sitepoint.com/google-drops-ie6-support/)。

**3 月:**在艾略特·杰伊·斯托克的[推文中，](http://twitter.com/elliotjaystocks/statuses/9227592793)我们在[播客](https://www.sitepoint.com/podcast-51-real-web-designers-get-it/)和[博客](https://www.sitepoint.com/css-skills-better-web-design/)上讨论了网页设计师是否应该知道如何编码。此外，我们增加了大部分书籍的 ePub 和 Mobi 版本，作为 PDF 数字包的一部分。

**4 月:**微软[从 ie 浏览器中掉落三叉戟渲染引擎](https://www.sitepoint.com/microsoft-drop-trident-from-internet-explorer/)，[谷歌实现“预测性”搜索](https://www.sitepoint.com/google-implements-predictive-searching/)。哦，等等，是四月。我明白了。在现实世界中， [SitePoint 课程诞生了](https://www.sitepoint.com/introducing-kevin-yanks-javascript-live/)，我们自己的 Kevin Yank 的 [JavaScript Live](https://www.sitepoint.com/premium/library) 作为第一个课程。

从很多方面来看，2010 年是网络打字年，尤其是由于谷歌字体目录的推出。在谷歌的其他新闻中，[公司开源了 VP8 视频编解码器，并宣布了 WebM HTML5 视频格式](https://www.sitepoint.com/google-announces-new-webm-video-standard-open-sources-vp8/)。还有， [Opera 比土豆](https://www.sitepoint.com/opera-faster-than-a-potato/)快。

**6 月:** [WordPress 3.0 发布](https://www.sitepoint.com/whats-new-in-wordpress-3-0-for-developers/)，为开发者带来了大量新的好东西，PHP 庆祝了它的[十五岁生日](https://www.sitepoint.com/php-15-birthday/)。

**7 月:** Craig 列出了网站开发新手最常犯的 [10 个错误，你们中的很多人都附和了这个话题。说到声音上的分歧，Craig 还在博客上写了关于 Brendain Eich 建议从 JavaScript](https://www.sitepoint.com/top-10-web-development-mistakes/) 中删除`function`关键字的文章。

8 月:我们的新设计师 Mark Cipolla 在博客上发表了一篇关于 WordPress 自定义写作面板的[指南](https://www.sitepoint.com/guide-to-wordpress-custom-write-panels/)，以及一些前端优化的[建议](https://www.sitepoint.com/front-end-optimization-from-the-get-go-part-2/)。

**九月:**九月是一年中的第九个月，今年都是九个左右: [PostgreSQL 9.0 发布](https://www.sitepoint.com/postgresql-9-features/)，由于[是 IE9 的 beta 版](https://www.sitepoint.com/ie9-beta-download/)。我们还讨论了[你是否应该关闭你的 PHP 代码标签](https://www.sitepoint.com/should-you-close-your-php-code-tags/)，这与数字 9 没有任何关系。

**10 月:**jQuery 团队[发布了 jQuery mobile 的 alpha 版本](https://www.sitepoint.com/jquery-goes-mobile/)，这是一套用于移动 web 应用的跨平台 UI 组件。Craig 看了一下 [BBC 网站公布的 CSS 编码标准](https://www.sitepoint.com/bbc-css-coding-standards/)，Alex 写了一个很棒的技术，可以在 CSS 中制作跨浏览器、无图像的光滑按钮。

11 月:这是博客上争议不断的一个月，[克雷格关于 PHP 开发人员常见 MySQL 错误的帖子](https://www.sitepoint.com/mysql-mistakes-php-developers/)和[我关于 HTML5 更宽松的语法要求的帖子](https://www.sitepoint.com/is-html5-dirty/)都在评论中引发了激烈的争论。

**12 月:**12 月还没过半，但我们已经谈论了安卓 2.3 的[发布，Firebug 1.6](https://www.sitepoint.com/mobile/android/) 的[新特性，更不用说我们粉丝最爱的](https://www.sitepoint.com/whats-new-firebug-16/)[圣诞倒计时销售](https://sale.sitepoint.com)的回归了。

2010 年也见证了一些伟大的 SitePoint 书籍的发布(如果我自己这么说的话): *[jQuery:忍者新手](https://www.sitepoint.com/books/jquery1/)* ， *[创建令人惊叹的 HTML 电子邮件！](https://www.sitepoint.com/premium/library)* 、*、[网络摄影](https://www.sitepoint.com/books/photography1/)、*、[在云端托管你的网站:亚马逊网络服务使之变得简单](https://www.sitepoint.com/premium/library)、*、[、](https://www.sitepoint.com/premium/library)、*、*、[构建你自己的主题](https://www.sitepoint.com/premium/library)、*。**

总的来说，对 SitePoint 和整个网络来说，这是很棒的一年。希望 2011 年不仅仅如此。
