## 采访记录

凯文:2010 年 12 月 3 日。这是一场 WordPress 访谈盛宴！我是凯文·杨克，这是 SitePoint 播客#90:博客世界博览会采访，第三部分。

嘿，我是你的老朋友凯文，又是一周了，这意味着又是一批来自 2010 年博客世界博览会的采访。本周，我的搭档主持人布拉德·威廉姆斯将会带头，因为本周所有的采访都与 WordPress 有关。如果你认识 Brad，如果你听过这个播客一段时间，你就知道他是 WordPress 的人。他跑遍了整个展览大厅，找 WordPress 社区的知名成员聊天，所以本周我们有三个这样的采访。第一个是来自 [Headway Themes](http://headwaythemes.com/) 的 Grant Griffiths，Grant 经营着一家公司，为 WordPress 构建这个高级主题，但是它不仅仅是一个主题，它是一个用于设计你自己的 WordPress 站点的拖放工具，所以我们将会听到他关于这个主题的介绍。然后我们有 Syed Balkhi，他经营一个叫做[WP 初学者](http://www.wpbeginner.com/)的网站，这是一个 WordPress 的初学者教程网站。我一直喜欢在网上工作的一点是，任何人都可以在网上发表文章，Syed 在 WPBeginner 上的工作是为网络上最受欢迎的发布平台之一 WordPress 保持这种状态。然后我们有来自 [Page.ly](http://page.ly/) 的 Joshua Strebel，Page.ly 是一家有胆量的公司，因为他们通过为 WordPress 网站提供商业托管服务，在自己的地盘上与 WordPress 和 WordPress 背后的公司 Automattic 竞争。因此，如果你要与为 wordpress.com 提供软件托管服务的 WordPress 竞争，你最好有一些漂亮的锦囊妙计，我们将从约书亚·斯特雷贝尔那里听到到底是什么让他们与 wordpress.com 区分开来。所以现在我想把它交给我的搭档布拉德。布拉德，把它拿走。

**Brad:** 我是布拉德·威廉姆斯，这里是 SitePoint 播客在博客世界博览会上的现场直播，和我在一起的是 Headway Themes 和[盈利博客的 Grant Griffiths](http://www.blogforprofit.com/)，感谢 Grant 的来访。

**格兰特:**感谢布拉德。

Brad: 首先，让我们来谈谈进展主题。你为什么不告诉大家到底什么是进展？

嗯，Headway 是一个高级的 WordPress 框架主题，有些人称之为主题，当然我们以我们的名字称之为主题，它更像是一个人们可以用来在 WordPress 上建立他们的站点的框架。我们有一个独家的可视化编辑器，人们可以用它来拖放，移动东西，实际上编辑颜色和字体以及所有这些东西。至于拖放你可以移动导航到顶部或底部的标题，你可以很容易地上传你的标题到网站， 在 Headway 2.0 中，我们实际上有一个新的内置功能，称为快速启动向导，一旦你将横幅放入其中，它将根据横幅为你找到颜色，为你提供颜色选择，你可以将颜色放入网站的不同元素中，进行大量的字体更改等，并在大约五六分钟内完成网站的基本框架。

布莱德:听起来很棒。现在 2.0 已经发布了还是即将发布？

不，它已经出测试版了。2.0 可能会在 10 月 24 日左右发布。

布拉德:太好了，太好了。所以，是的，回到那种拖放式的可视化编辑器，我的意思是我会说这可能是你最出名的地方。我的意思是我不知道太多的其他主题，或者我能想到的任何其他主题，这是其他一些主题和框架的主要特征，事实上，它比基于代码的更可视化。

Grant: 我认为这确实是一个关键的特性，是的，我的意思是我们有一些普通的东西，比如内置的 SEO 特性等等，但是我认为可视化编辑器确实与其他主题或框架有所不同。我是一个非编码人员，我不懂任何编码；我知道足够危险的编码。我总是告诉人们我有克雷·格里菲斯，Headway 的开发者，我不需要学习如何编码。但是我自己可以用 Headway 建立一个站点，除了一些基本的 HTML 之外，完全不需要任何编码，并且使用可视化编辑器和拖放功能，不到一个小时就可以建立一个站点。可视化编辑器已经从去年 7 月我们第一次推出 1.0 版本发展到现在的样子，实际上是去年 7 月，发展到现在的样子。它已经发展成，当然我有偏见，但是我认为它已经发展成一个非常棒的工具，非编码人员也可以使用，但是我们也有开发人员使用 Headway。

Brad: 我想说的是，如果这是一个非常非常酷的主题特性，但是对于更多的核心开发者来说，它是否也提供了一种简单的方法来通过代码端来做类似的事情，或者它都是可视化的？

**Grant:** 不，我们在 Headway 中内置了简单的挂钩，我们实际上有一个内置在可视化编辑器中的实时 CSS 编辑器，所以你可以实际上放入你想要使用的 CSS，并且你会实际上看到它就在你面前发生。我们真的试图让它不仅能让非编码者、自己动手的人使用，而且我们也让开发人员使用它。

布拉德:正如他们所说，你是一个高级主题，那么现在 Headway 的购买价格是多少？

**Grant:** 个人执照 87 美元；一个开发者的许可证是 164 美元。

Brad: 开发者许可证是否允许你在许多网站上使用它，或者我可以在类似的客户网站上使用它？

**授予:**可以。现在，我们用个人许可证做什么你得到两个站点的支持，开发者许可证你得到你想要为你自己或你的客户建立的许多站点的支持。如果你的客户需要 Headway 的支持，他们需要购买个人许可证。

好的，很好，很好。你实际上最近刚去了分裂许可证，是吗？

格兰特:是的。我们目前正在拆分 GPL——目前。(笑)

**Brad:** (笑声)当你与高端主题的开发者和设计师谈论授权时，这总是一个有趣的话题，它可能会很快变得敏感，但我们不会太深入。

格兰特:不，我很感激。是的，但真正令人惊讶的是，当你看我们的用户群时，可能 95%的用户都不在乎。

布莱德:他们只是希望它能成功。

格兰特:是的，这就是我们真正想做的，那就是开发一个我们的用户可以使用的产品，并得到他们想要的结果。这真的是我们想要做的，许可证的事情太糟糕了，我希望我们已经解决了它，我们可以继续前进。

布莱德:对，绝对是。你也在——你的博客是[盈利博客](http://www.blogforprofit.com/),为什么不告诉我们一点呢。

格兰特:Blogforprofit.com 是我创办的一个博客。我不是说通过博客赚钱来快速致富；我不谈论附属链接，诸如此类的事情。盈利博客实际上是一个我用来教人们如何使用博客或社交媒体来推广或营销业务的博客。与其说是在线业务，还不如说是实体业务。这不像其他一些博客，他们谈论联盟链接和利用广告赚钱之类的东西，我真的试图专注于使用博客来营销你的业务。

布拉德:我在推特上关注你，我相信你每周都会有一次，我认为这是一个很酷的想法，你几乎可以通过推特聊天，通过推特谈论一周的话题，对吗？

**格兰特:**实际上，标签是#SBT10，代表今天开始写博客，而[今天开始写博客](http://startbloggingtoday.com/)实际上是我和一些人一起做的会员网站，我们有 26 个核心课程，我们有一堆额外的课程，然后我们还提供一些 WordPress 教程和那些东西，我们有一个会员专用的论坛。但是我开始了 Twitter 聊天，每周 Twitter 聊天，作为其中的一部分，任何人都可以来，你不必是 Start blog Today 的成员来参与，这很有趣，我们有 10 个人参加，我是说有一天晚上我们有将近 200 个人参加。所以我的意思是这很有趣。

**Brad:** 现在，你是在使用这方面的服务，还是只是用标签发推文，这就是你们的对话流程？

**格兰特:**我用了一个推特工具，叫做我想，它是什么，F-L-O-K-K-E-S，或者类似的东西，如果我破坏了它，我很抱歉，因为我面前没有它。我自己也用这个工具关注 Twitter 聊天，因为这就是它的全部功能，它会自动为你添加标签，这真的是一个很棒的工具。你不必使用它，我认为 Twitter 有——你可以使用 Twitter 聊天来跟踪对话，或者你可以只使用桌面应用程序，比如我使用 Market Me Suite 搜索一下标签，然后也这样做。

布莱德:格兰特，听起来你很忙。

格兰特:哦，是的。(笑)

布莱德:太好了。你为什么不告诉大家在网上哪里可以找到你。

**格兰特:**前进主题为 headwaythemes.com 的[，盈利博客在 blogforprofit.com](http://headwaythemes.com/)的[，然后是 startbloggingtoday.com](http://www.blogforprofit.com/)的[。](http://startbloggingtoday.com/)

布拉德:太棒了，太好了。非常感谢格兰特。

**格兰特:**谢谢。

布拉德:我是布拉德·威廉姆斯，我们正在直播 2010 年博客世界。我和 WP 初学者的帕特里克·奥基夫和赛义德·巴尔基在一起，你好吗，赛义德？

很好，你好吗？

布莱德:很好，真的很好。那么，你为什么不告诉我们一点什么是——对于那些不知道的人来说，wpbeginner.com 是什么？

**Syed:** 好的，[wpbeginner.com](http://www.wpbeginner.com/ "WPBeginner - Beginner's Guide for WordPress")是一个面向初学者的 WordPress 资源网站，但是我们有适合每个用户层次的内容，比如如何用 WordPress 做事情——基本的东西。

布拉德:这是一个相当新的网站，我的意思是它有多长；你已经活了一年多了，对吧？

是的，我们在 2009 年 7 月推出。

Brad:2009 年 7 月，在这么短的时间内，你已经在网站周围建立了一个相当大的社区。

**Syed:** 是的，我们主要使用 Twitter，现在 Twitter，你知道，脸书和所有这些，基本上是社交媒体，我们在网站广告上不花一分钱，它被罗伯特·斯考伯、wired.com 和 Mashable 采用，所以我们在这方面得到了一些帮助。

布拉德:太棒了。那么，除了 Twitter 和脸书，你有什么建议可以给那些想建立一个不错的社区并关注他们的话题和网站的人吗？

是的，我能给任何人的最好的建议就是客户服务。所以，不要试图获得新客户，专注于你现有的客户，并提供最好的客户服务。这些用户会像锚一样工作，他们会为你找到更多的用户，因为这正是我们成长的方式；你知道，人们不会真的在推特上转发东西，但是我们的东西被我们帮助的这些用户转发了；我们回复了每一封电子邮件，甚至回答了最基本的问题。

Patrick: 不用说，我想你相信初学者内容的力量。我的意思是，这是 Darren Rowse 与 ProBlogger 谈论的事情，他说他最好的东西是初学者的东西，基本的问题，回答这些问题，我想你创建了一个资源，一个市场，用 WordPress 来回答这些初学者的问题。

是的，实际上我认为有很多 WordPress 网站主要是针对开发者或设计师的。最好的书是《傻瓜博客》,实际上是 Lisa Sabin-Wilson 的《傻瓜 WordPress 》,但是大多数人在尝试使用 WordPress 时，他们不想买书和花钱。

布拉德:嗯，我的意思是取决于那本书是什么。(笑声)

帕特里克:有几本书你可能想买。

是的，如果你想进入 WordPress，专业 WordPress 是一本很好的书。但作为一个新用户，我们的目标首先是教人们为什么应该使用 WordPress，因为许多公司和许多人都有这样的思维模式，即它是开源的，你不想使用它，因为它不安全或诸如此类的东西，所以我们涵盖了这一点，然后开始进入初学者内容，展示大多数人没有注意到的基础知识。

Brad: 你有一个即将推出的新网站，你能给我们一些细节吗，一些内幕消息？

是的，我们计划推出 WPBulk，类似于 MacHeist 的概念；我们将为初学者捆绑最好的 WordPress 产品，包括主机托管、付费插件、商业主题、你可能会发现对博客有帮助的书籍，我们会以很小的价格提供给你，所以这将是价值 50 美元的东西。

Brad: 这实际上就像一个——比如说，一个插件包，你可以一次性安装所有的插件？

Syed: 我们要做的是，我们将提供插件下载，因为从本质上来说，这些产品应该能够在每个网站上运行，但如果有人不想使用某个主题，而只想使用插件，他们可以从我们的会员区下载插件，然后继续使用。它将是限时交易，并将定期推出，所以我们将有一个捆绑包，或者我们将为捆绑包起一个有创意的名字，它将持续一个月，而且数量有限，只是为了保持产品的独家性。

布拉德:太棒了，对，那就是 wpbulk.com，我们一定会密切关注。我们走之前你还有什么要告诉我们的吗？

帕特里克:是的，这是一个很难回答的问题，你知道，在你的生活历史中，最有趣的是什么？(笑声)

是的，我希望将 WPBeginner 扩展成几个部分，因为我不认为 WordPress Showcase 做得很好，我们的图片库将展示普通用户提交的看起来不错的东西；你不需要成为 Jay-Z 就可以把你的网站列在我们的展示区里，你可以把它列在我们的画廊里发布。我们也在 WP 初学者中推出了一些很酷的东西来让社区互动。

布拉德:太棒了，那么除了 wpbeginner.com，人们还能在哪里找到你？

你也可以在推特 [@syedbalkhi](http://twitter.com/syedbalkhi) 上找到我，你可以在我的博客【balkhis.com[上找到我，我偶尔会在那里写些东西。](http://www.balkhis.com/)

谢谢布拉德和帕特。

布拉德:我是布拉德·威廉姆斯，我是 SitePoint 播客，我们正在博客世界博览会现场直播，来自 Page.ly 的乔希·斯特雷贝尔为您报道。乔希，你好吗？

乔希:很好，谢谢你邀请我们，布拉德。

布莱德:对，绝对是。告诉外面的人，什么是 Page.ly，你们是做什么的？

**Josh:** 嗯， [Page.ly](http://page.ly/) 是一个托管的 WordPress 托管系统，我们在大约一年前开发并发布了这个系统，它可以让任何人在大约一分钟内使用一个新的 WordPress 站点，包括自动备份、核心升级、插件升级、与主题开发者的合作，几乎所有你需要的东西都可以让你在一个 quickie 上拥有自己的托管 WordPress 站点。

布拉德:这有点像主持 wordpress.org，这样解释公平吗？

乔希:没错。把它想象成 WordPress 即服务，wordpress.com 在为用户简化事情方面做得非常好，但是他们对插件和主题的使用设置了一些限制，wordpress.org 在安装和设置数据库以及自己做所有事情方面有点令人生畏，所以我们把自己定位在这两个选项的中间，我们负责很多技术方面的事情，但是仍然给你完全定制你的网站的自由。

Brad: 你还与一些著名的主题公司、高级主题和插件合作；所以你为什么不告诉我们这到底是怎么回事？

**Josh:** 是的，当我们第一次推出时，我们非常相信双赢的局面，你知道，在社区内工作，我们对此非常热情，所以当我们第一次推出时，我们联系了 StudioPress 的 Brian Gardner 和 WooThemes 的伙计们，现在又联系了 Press75 的 Jason。我们与他们建立了一些合作伙伴关系，我们可以让我们的客户访问这些开发人员的整个主题库，以及他们对每月固定费用的支持。

Brad: 所以我可以在一分钟之内注册，不管需要多长时间，30 秒，这是一个非常快速的过程，我已经完成了，我可以从这些高级主题中选择一个作为我的网站，在一两分钟内我就可以上线一个 StudioPress 主题或 WooTheme。

乔希:你当然是；不仅仅是一个，你可以从整个图书馆中选择，所以你可以挑选你想要的任何一个。

布莱德:一周的每一天都要开机。

乔希:是的，当然是日常主题。

**乔希:**完全的自由，做任何你需要做的事，信不信由你。一旦你通过了技术设置，然后是我们在幕后做的备份和升级，除此之外，这就像拥有你自己的 WordPress 网站一样。你可以在插件下添加新插件，从库中取出任何插件，从库中取出任何主题，安装它，做你需要做的事情，然后定制你的站点。

Brad: 你们非常重视安全性，我知道你们已经和 FireHost 合作了，是吗？

乔希:没错。

Brad: 你为什么不告诉我们关于安全和应该采取的步骤。

Josh: 如果你们看了今年夏天的新闻，就会发现一些大型主机因为恶意软件攻击和黑客攻击而遭到了严厉的指责，这很不幸。当我们开始创业时，我们知道自己擅长什么，我们擅长 WordPress 的应用程序方面，但我们不想成为服务器管理员，所以我们非常愿意花大价钱去寻找绝对最好最安全的服务器，而这恰好是 FireHost。他们负责管理整个基础架构，将其锁定，他们拥有价值数百万美元的硬件安全设备来保护我们的网络，我们花了大价钱购买这些设备，但如果我们的客户直接购买这样的安全主机，他们只需花很少的价格就能获得这些好处。

布莱德:是啊，太棒了。我也非常关注 WordPress 的安全性，所以我喜欢看到像你们这样的公司真正关注它，而不是忽视它，接下来你知道你的网站会被黑客攻击。
