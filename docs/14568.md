# Solaris 走向开源

> 原文：<https://www.sitepoint.com/solaris-going-open-source/>

金融时报的一篇文章宣布 Sun 微系统公司将在不久的将来发布他们的 Solaris 操作系统作为开源软件。

对于 web 开发人员来说，这是一个相当有趣的开源新玩家。Sun 长期以来一直是大型 web 应用程序托管数据中心的首选，当然也支持我们从 Linux 环境中了解到的许多最流行的 web 相关服务器软件，如 Apache 和 MySQL。

我最近读了一篇关于 Solaris 和 Linux 的评估，有一些令人惊讶的结果。虽然它不是在一个纯粹的测试实验室中完成的，但它似乎是一个公平的测试。你可以[在这里](http://www.osnews.com/story.php?news_id=4867&page=1)阅读评测。

结果中最有趣的是，Solaris 在应用程序编译、ssl 和 web 测试方面优于 Linux。Linux 文件系统确实比 Solaris 性能更好——但是——这在 web 托管方面不一定是一个重要的指标。

虽然 x86 (Intel)系统上的 Solaris 没有那么多可用的应用程序二进制文件，但转向开源肯定会鼓励开发人员重新审视系统，寻找安装端口。

这一举措也再次将传统的大型企业级操作系统转移到了没有经济实力投资于此的 web 开发人员手中(Solaris)。

我发现令人着迷的是，在众多的研究中，各种规模的公司都在以某种形式或方式寻找开源解决方案，而如此多的大型软件公司都在玩弄源代码发布。虽然我仍然相信我们将永远拥有多平台环境，但支持开源的 web 开发人员应该关注基准测试、研究和行业统计数据，并始终构建一个为什么开源可以为他们的客户工作的理由组合。
