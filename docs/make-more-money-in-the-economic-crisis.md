# 在经济危机中赚更多的钱

> 原文：<https://www.sitepoint.com/make-more-money-in-the-economic-crisis/>

以下内容转载自论坛报#419 网站。

太神奇了。我离开，离开这个地方几个星期，整个世界经济崩溃了！

对许多人来说，这意味着失业、减薪、支出减少和支出增加。

但是，这并不都是悲观的，尤其是对基于网络的企业或网络开发人员来说。所需要的只是重新调整你的关注点，你的生意会比以前做得更好。

如果你在网上销售产品，你通常会有很大的价格优势，使你可以卖得更便宜，这也是许多人想要的。所以也许是时候开始推动这个了。

如果你是一名网络开发人员，许多企业会想尽一切办法增加他们的客户数量，这将包括更加关注他们的在线工作。你为什么不开始告诉你的潜在市场呢？记住，广告费用会降下来，所以开发者可以少花钱多做广告。

*   增加你的营销:当经济不景气时，大多数企业都会削减成本。不要这样做——现在是获得关注的时候了，应该比以往更容易，成本更低，对关注的竞争更少。
*   **改变你的营销重点:**人们会寻找便宜货，所以提供买一送一、购物免费赠品，以及任何吸引精打细算者的东西。
*   灵活机动:准备好改变你的商业模式，以维持现金流，避免陷入困境。

通过正确出牌，不仅可以度过经济低迷期，还可以利用它！
