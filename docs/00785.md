## 3.如何使用幻灯片

一切从发电机开始。这是一款设计直观的工具，可以节省您的工作时间。虽然发电机很容易操作，但您可以找到其手册的链接。

您还可以看到一些有助于入门的示例解决方案和模板。无论如何，在你知道之前，你会像专家一样建立网页和网站。

[![](img/2e0e454b5a90bb8839a2fa62637447c8.png)](https://designmodo.com/slides)

**选择要使用的幻灯片。**这是第一步，选择合适的幻灯片来启动您的项目不成问题。

这是由于一个过滤菜单，涵盖了标题，功能和形式。它还包括价格、视频和其他类别。

[![](img/33d5126c47d7c6c77d420c3657f05bb5.png)](https://designmodo.com/slides)

当您选择想要的幻灯片时，单击右上角显示的三个按钮中的第一个。

这将允许你复制、删除和改变幻灯片的风格。或者，您可以拖放它们来更改顺序。

[![](img/800b933ac6b0728f1050336a3f6906a1.png)](https://designmodo.com/slides)

**幻灯片很棒，但不一定完整**。缺少的是*动画*。动画可以把一个优质的网站变成一个真正优秀的网站。虽然每张幻灯片都有一个默认的动画，你可以通过点击你喜欢的样式来改变它。

只需多点击几次，您就可以从一系列效果选项中进行选择。

[![](img/c0403ae092d17140147d38dc7ff3336f.png)](https://designmodo.com/slides)

**选择面板**。要制作一个获奖的网站，需要的不仅仅是一系列令人惊叹的页面。必须考虑 UX 因素，尤其是导航。

首先要做的是从众多可用的标题面板中选择一个。如果您愿意，只需复制它的代码并将其添加到 HTML 中。这同样适用于页脚面板。选择你想要的，选择它的设置，你就可以开始了。

[![](img/d7b48c3820fa252a1e821a8beb98009c.png)](https://designmodo.com/slides)

**选择导航**。有多种方法可以帮助用户浏览幻灯片。也许你的网站根本不需要任何特殊的导航功能。但如果确实如此，你可以选择圆点或箭头作为导航辅助。

[![](img/11c5c2d27a6176504aa8d69d77704768.png)](https://designmodo.com/slides)

**设置样式**。此功能允许您从预定义的排版样式中进行选择。这使得定制一个网站来匹配客户的品牌指导方针变得很容易。

您可以进行一些与代码相关的定制来完成您的项目，例如预加载内容或优化资源。为你的网站创建一个简化的移动版本也是可行的。

[![](img/7c9ba4f49b795b55bfa7e88a9d41753c.png)](https://designmodo.com/slides)

最后一项任务:下载 HTML 模板。在将您设计的模板上传到 live 网站之前，您需要下载它。添加您的客户想要的任何特定图像或颜色，就大功告成了！
