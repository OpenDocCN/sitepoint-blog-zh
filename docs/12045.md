# 我如何应对在家工作的挑战

> 原文:[https://www.sitepoint.com/working-from-home-challenges/](https://www.sitepoint.com/working-from-home-challenges/)

也许“精通”有点大胆，但我确实有多年在家工作的经验，并且摸索出了一条相对成功的道路。在家工作的挑战对每个人来说都是不同的，但是这里有我遇到的三个最大的挑战和对我有效的解决方案。

**挑战#1:孩子们**

作为父母，你爱你的孩子是不言而喻的。但是，尽管我们爱我们的孩子，也爱做父母，工作和孩子并不总是形影不离的。你的孩子可能是你决定在家工作的一个因素，但这并不一定会让兼顾工作和孩子变得更容易。

如果你像我一样有年幼的孩子，你就没有上学时间来专心工作的好处。这意味着你白天在孩子和工作之间分配时间，早起熬夜工作，周末工作，或者打电话叫增援(家人、保姆、保姆、托儿所)来帮助照顾孩子。

我已经做了上面列出的所有事情，并且在面临因睡眠不足而导致的严重疲劳和每个周末都工作以赶上进度后，我发现对我(和孩子们)来说最好的解决方案是每周有几天在户外照看孩子。这给了我专门的工作时间来经营我的生意，也给了孩子们休息的时间和一些和他们同龄的孩子的社交时间。当他们在家的时候，我有更少的工作要做，所以我和他们在一起的时间更加集中和专注。压力也小了很多。

**挑战 2:隔离**

让我们面对现实吧，在家工作会很孤独。白天，你没有同事坐在大厅里和你聊天，你可能也不会出去做很多面对面的交流。除此之外，作为一个像我一样的内向者，忘记它吧…你只能靠你自己了！

对我来说，解决这个问题的办法是定期与客户通话，安排一天中的一些时间去跑腿，并在网上与同事建立关系。虽然我使用许多社交媒体进行商务活动，但我也受益匪浅，与我在网上结识的一些优秀人士建立了个人关系，这对我非常重要。

**挑战 3:拔掉插头**

因为我热爱我的工作，我面临着一个挑战，我怀疑我们许多在家工作的人都面临着这个挑战——无法在一天结束时关闭、关掉和放下工作。这可能是一件好事，也可能是一件坏事。很好，因为你可以完成更多的工作，并且能够掌控一切。不好，因为我们都需要远离工作的时间。如果你经常在周末工作，你可能急需休息。

我对此的解决方案，我承认这是我正在改进的领域，是尝试规划空闲时间——当我的孩子在家时，周末的几个小时和晚上。我总是可以用我的黑莓手机回到圈子里，但我正在学习不要被束缚在我的笔记本电脑和收到的信息上。这给了我一些急需的休息时间，让我专注的工作时间更有效率。

最重要的是，我正在用自己的时间在家里做我一直想做的事情。我不会为任何事情改变我的处境，即使我经常不得不为出现的挑战解决问题。你的挑战可能与我和其他人面临的挑战不同，但关键是找到适合你的解决方案。

在家工作对你最大的挑战是什么？有哪些解决方案对你有帮助？
