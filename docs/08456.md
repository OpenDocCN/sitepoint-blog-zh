## 剧集摘要

Kevin、Christopher 和 Ari 介绍了会议，包括不同的会议方式、不同方式的优势以及如何充分利用会议。然后，Kevin 与 Evan 谈论了他在 Automatic 公司担任开发工程师的工作，特别是 CodePoet 上的新开发工作，以及它将如何为那些使用 WordPress 为人们建立网站的人工作。
