# 会员有它的特权

> 原文：<https://www.sitepoint.com/membership-has-privileges/>

“哦，孩子，”我听到你在读完上周的专栏时说。“成功的秘诀是让人们注册我的网站，然后拒绝大多数人并要求他们付费！”

抱歉，我让你等这周的专栏。如果你根据上面的段落建立了一份商业计划，撕掉它，继续读下去。

综艺节目可以要求注册，因为它已经赢得了尊重，几十年来花费数百万美元成为这个行业的“圣经”。事实上，它是如此受人尊敬，以至于它可以[出售订阅](http://store.yahoo.com/magazinecity/2781-52.html)，当它赠送一份时，它的读者知道他们已经“到了”。当你有这样的历史、信誉和财务实力时，你也可以对你的目标市场提出要求。在那之前，你比他们更需要他们。

那你是做什么的？首先，你不能“要求”你的目标人群“注册”。而是“要求”他们以“成员”的身份“加入”。向他们推销成为你专属团队一员的好处。

##### 推销你的福利

不要只是在你的主页上放置一个注册按钮(尽管你应该这样做，也许是作为一个弹出窗口)。你还需要附上一封聘书，详细说明潜在客户成为你网站的会员所能获得的好处——这些好处是他们仅仅作为访问者所享受不到的。这封信必须说服潜在客户不仅要输入电子邮件地址，还要填写你的 qualcard。

现在，我想让你写下所有你可能提供给会员的特殊优惠，这些优惠是你目前访客群中的一小部分，是你不能提供给普通访客的。这可能是一个特殊的电子邮件简讯，一个讨论列表，有节制的聊天，或其他好处。

这些好处可能是你目前提供给所有访问者的，但你也应该小心添加新的，“独家”功能。你能让你的成员与一位知名专家进行个人接触吗？这位专家可能会回答他们的具体问题。你能创作一本书——甚至是一个 PDF 文件——详细描述你的行业吗？

一旦你拿到了录取通知书和 qualcard(你还记得上周的事)，仅仅把它们放在任何可能出现的人面前是不够的。即使是最著名的印刷出版物也不会这样做。

不，如果这个成员是有价值的，如果他们愿意为你付出时间并帮助你达到目标，你必须做两件事:

*   找到他们，然后
*   恳求他们。

##### 抓住你的目标

你已经做了初步的恳求。现在你需要成为一个直接回应广告人。我建议你砍掉一些树，从一个名单经纪人那里买一份名单——一份地址的打印名单。

因为你追求在线订阅，你可以从电子邮件列表开始，但不要局限于此。还可以考虑制作一封邮件——一封印刷的纸质邮件——针对你的目标受众。如果这些人是有价值的观众，毕竟，他们值得你花钱去接触他们。

你所在市场的赢家将是从目标市场的最大部分获得最大忠诚度的出版商，也是能向广告商证明这一点的出版商。所以，一旦你有了一个可靠的会员数据库，考虑进行一次审计。

我知道这很贵。这是一个长期的商业计划。遵循这一计划的结果将是，由于在线出版的速度和较低的成本，让您的网站与那些在市场上竞争的印刷出版物正面交锋，然后摧毁它们。

你看，你不只是有一个内容网站。如果你的网站有内容，并且你对业务很认真，你现在就是一个出版商。而出版人的工作就是服务和倡导一个行业或者一种生活方式。是时候认真对待那件事了。
