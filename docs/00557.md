# 加速你设计生涯的 9 个关键方法

*本文是与 [StudioWorks](http://studioworks.me/) 合作创作的。感谢您对使 SitePoint 成为可能的合作伙伴的支持。*

当然，如果你想成为一名成功的设计师，你需要一定程度的设计技能。但是还有更多的事情要做。想一想你认识的人中有多少人能做出令人惊叹的食物，但在午餐高峰期间，他们在餐馆厨房里坚持不了五分钟。

如果我们能坐下来，设计漂亮的东西，然后回家，那就太好了。或者更好的是，呆在我们的家庭工作室里创作。不管不幸与否，设计就像其他事情一样是一门生意，这意味着你必须投入时间、精力，有时甚至是金钱来培养你设计生涯中的软技能和商业方面。

这意味着管理好你的时间，推销你自己，建立一个品牌，尝试，也许启动一个副业，通常只是把你的名字和工作放在那里让人们发现。

如今，光有投资组合是不够的。那只是赌桌赌注。你需要计划好你的整个职业生涯，要知道计划是会变的。

让我们看看你需要做些什么来发展你的职业生涯，直到你基本上成为下一个珍·西蒙斯或杰弗里·泽尔德曼。

## 让你的沟通技巧流畅起来

有些人天生就有沟通技巧，而有些人就不那么天生了。在这两种情况下，这些技能都会受到你最常与之交流的人的极大影响。我们大多数人都会想办法把我们的想法和意图清楚地传达给我们的朋友，以及我们行业和爱好社区的人。我们学习行话，我们知道哪些话题鼓励讨论，哪些是最好避免的。

为不属于你的直接社区的任何人写作，尤其是为不知道你所知道的东西的人写作，是很难的。与他们面对面交谈可能会更难，这取决于你作为一个人更喜欢如何交流。但是，尽管如此，你必须这样做。

即使你现在在一家代理公司和其他设计师一起工作，也不可避免地会有这样的时候，你必须向客户推销你的工作的好处，向一个新手解释你使用的流程，或者向反对你的开发人员或其他不知道你所知道的事情的人捍卫你的决定。

如果你从这篇文章中学到了什么，那就是关注你的沟通技巧。这将比清单上的任何事情都更影响你的职业生涯。如果你正在寻找一个开始学习这些技能的地方， [CopyBlogger](https://www.copyblogger.com/blog/) 总能帮你找到——至少在写作方面。

## 扩展到副业

有些兼职项目很棒，严格来说是因为它们让我们走出舒适区，尝试新事物，重新获得创造性发现的感觉。

其他活动可能会与你用来建立个人品牌的活动重叠，这一点我们很快就会谈到，额外的好处是，当你成为该领域的专家时，它们可以带来额外的金钱。

下面是一些比较流行的方法。

### 1.课程

当然，你可以在你的博客或 YouTube 上免费发布一些教程。你也许应该。但是如果你想把教别人你做什么作为副业，并在这个过程中推进你的事业，你将需要一个实际的产品。这就是课程发挥作用的地方。

如今相当受欢迎的高质量视频课程，设置起来可能既昂贵又耗时。不过，现在你可以使用一体化的课程开发和交付服务，如 [Kajabi](https://kajabi.com/) ，这变得更容易了。这个平台可以帮助您管理与您的优质教育内容相关的一切，并围绕它开展业务。

您可以在一个地方创建会员网站、举办现场活动、创建自动化渠道、追加销售、维护博客和管理联系人，因此不再像以前那样困难。但是，你还是得弄个还算过得去的摄像头，还算过得去的麦克风，理想的情况下还能学会一些基本的视频剪辑技巧。

坦白地说，如果你有时间，并且有一点额外的钱购买一些初级硬件，我只推荐这种旁门左道的把戏。不过，这可能会很有收获，所以不要轻易放弃这个想法。

### 2.直播

我在上一节提到了现场活动，所以我想我应该提到流媒体本身。流媒体不一定要有教育意义，尽管教育可能是推销你专业知识的最好方式。如果你愿意，你可以坐在那里在 Twitch 上分享设计师迷因。

问题主要是对视频和音频的要求没有改变，取决于你如何设置你的流时间表，它甚至可能比制作视频课程更苛刻。

话又说回来，如果你不介意赚不到很多钱，并且想以此为乐，这仍然是一个“认识”新朋友的好方法。

### 3.付费时事通讯

现在，当你已经通过其他方式，比如社交媒体和/或博客，建立了一些受众时，我会保留这个选项。但是 [Substack](https://substack.com/) 让人们比以往任何时候都更容易直接向作家付费。

如果你有智慧可以分享，如果你认为人们会愿意付费将智慧直接发送到他们的收件箱，那就去分享吧。

### 4.为其他设计师做东西

许多设计师和代理商通过为其他设计师制作资源来保持健康的“被动”收入流。

无论是一个模板，一个 WordPress 主题，一个 Sketch UI 工具包，一个图标字体，还是其他任何东西，如果它对你有价值，因为它解决了你的问题，那么你的同行很有可能愿意为它付费。只是别忘了偶尔也把东西送人。在设计界，感恩是很重要的。

## 主时间和项目管理

刚开始的时候，设计师们通常会接受所有他们遇到的工作，尽可能地把它们塞进他们的时间表中。

有时候你不得不这么做，因为食物。然而，从长远来看，这不是一个可持续的战略。这种方式会让你精疲力尽，你以后不会为自己的工作感到骄傲。

现在，时间管理是每个人都必须以不同的方式处理的事情，因为我们没有一个人是以相同的方式连接的。但是有一些基本原则要遵循:

*   给自己留有“回旋余地”的最后期限，因为有些事情会出错，你想有时间去处理它。
*   如果可以的话，不要直接从一个大项目跳到另一个项目。在演出间隙休息一天，或者至少几个小时。
*   为思考和计划预留相当多的时间。计划好你可以合理预见的过程中的每一步。一个可靠的计划会让剩下的工作进行得更快。

当然，合理管理时间不仅对你的心理健康有好处。这也让客户高兴，我真的不认为我需要解释这对你的职业生涯有什么好处。

## 发展你的个人品牌

当然，与客户沟通和合理安排你的时间只会让你到此为止，因为你首先需要有客户。这就是发展品牌的切入点:人们需要知道你是谁，他们需要一个理由来选择你而不是其他设计师。你需要一个“钩子”，这是你的专长和你对事物的独特视角真正发挥作用的地方。

再说一次，光有一个好的作品集是不够的——即使你选择了最好的字体，为你的个人网站设计了一个令人惊叹的标志。搜索引擎优化是很好，但你需要在人们会看到你的地方，你需要有所代表。

你可以从以下空间开始探索个人品牌发展。

### 1.社会化媒体

活跃在顶级社交网络上可能是这个过程的第一步。喜欢，评论，只是一般有助于对话。不要——我重复一遍，不要*不要*,在任何情况下——向人们发送指向你的网站或服务的链接。

炫耀你做的或写的东西，是的。为你的工作感到骄傲。炫耀(这很重要)你的个性和工作流程，这样人们就能看到你是如何做事的。

### 2.博客

说到写东西，博客仍然是展示你的名字和想法的最佳选择之一。这也是一种可以展示很多个性的媒介，所以尽情发挥吧。

像社交媒体一样，博客允许你炫耀一点，并为设计社区的持续对话做出贡献，但它也允许更多的深度和细微差别。

### 3.公开演讲

上台是另一种选择。现在，这可能会让人联想到在设计会议上发言的画面，这当然不是一件坏事。这是一次很好的建立简历的经历，也是对你品牌的一个很好的补充。但实际上我会从较小的地方商务会议开始。

走出去，宣扬设计、用户体验和可访问性。告诉其他企业为什么他们应该密切关注这些概念。这是建立关系网、获得新的潜在客户以及练习与非设计师交流的好方法。

## 你不必一次做完所有的事

现在，在这篇文章列出的所有事情中，你的沟通技巧和时间管理技巧是让你的职业起飞最重要的。你知道，在知道如何设计之后。你为实现自己的抱负所做的一切都必须植根于这些核心技能。

至于其他的——自我营销和兼职——慢慢来。没有人有时间做所有的事情，*和*设计漂亮的东西。从简单的事情开始，比如让你的社交媒体出现，然后扩展到更复杂的副业项目和业务。不算宇宙善变的本质，你的职业生涯会随着你的发展而发展，所以不要盲目地匆忙做出这些决定。

最重要的是，从中获得乐趣。
