# 用屏蔽加强安全性

> 原文:[https://www.sitepoint.com/tighten-security-with-dshield/](https://www.sitepoint.com/tighten-security-with-dshield/)

一个奇妙的资源传递给了我，它叫做 [DShield](http://www.dshield.org) ，自称是一个分布式入侵检测系统。

它实际上是一个强大的实时报告资源，提供关于最受攻击的端口、攻击类型以及攻击者是谁的信息。正如 DShield 的人所说,“DShield.org 试图从整个互联网上收集关于黑客活动的数据。这些数据将被分类汇总。它可以用于发现活动趋势，并制定更好的防火墙规则。”

我最近写了一篇关于使用 iptables 构建防火墙的文章，有了这样的资源，人们可以定制包过滤规则来阻止新的端口并收紧服务器周围的网络。

该网站的主页提供了一个显示攻击类型模式的全球地图，以及一个端口“股票”行情表，按这些端口细分攻击类型，以及哪些应用程序通常使用同一个端口。

DShield 还提供了“你被破解了吗”搜索功能，通过对集团数据库的 IP 搜索，可以查看你使用或管理的机器是否被破解了。

最后，防火墙管理员可以上传他们的日志，并为 DShield 提供的覆盖数据做出贡献。管理员可以随时联系网站，进一步讨论是否应该在提交之前编辑日志，或者如何使用数据。

DShield 的创建者表示，他们正在探索如何在未来扩展到包过滤之外，以覆盖更复杂的应用层防火墙。
