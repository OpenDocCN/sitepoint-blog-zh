# 感知就是现实

> 原文：<https://www.sitepoint.com/perception-is-reality/>

你向客户、你的员工或承包商展示自己的方式，在他们眼里就是现实。我在《企业家领导力》上读到了戴夫·拉姆齐写的一个故事，讲的是一个成功的企业主，他的企业一年获利超过一百万美元。他一直梦想成为一名职业高尔夫球手。他开始每周休息 3 到 4 天去打高尔夫，第二年他的利润下降了 70%！

损害他业务的不是他不在办公室，而是他的员工认为他不再在乎了。既然他不在乎，他们也不在乎。他们也“检查”过了，虽然他们身体上可能在办公室而不是去打高尔夫，但他们肯定没有尽最大努力工作。

我通常不会花太多时间担心我的客户或员工对我的看法，但这个故事引起了我的注意，让我意识到在某些情况下，感知真的可以产生影响。想象一下，开着一辆破旧的车去见客户，你认为客户会对你的能力有什么看法？现在想象一下，你开着一辆价值 10 万美元的豪华车，他们会认为你要价过高吗？

你不应该过分担心每个人的想法，而是要考虑你的行为向你的客户和员工表达了你的职业道德和你对企业的奉献精神。因为如果你想要敬业的员工，比如说，他们需要觉得老板也一样(如果不是更)敬业。
