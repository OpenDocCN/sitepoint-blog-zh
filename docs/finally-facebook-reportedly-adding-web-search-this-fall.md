# 最后，据报道，脸书今年秋天将增加网络搜索

> 原文：<https://www.sitepoint.com/finally-facebook-reportedly-adding-web-search-this-fall/>

根据美国消费者新闻与商业频道的报道，微软计划今年秋天通过在社交网站上增加网络搜索来扩大与脸书的广告关系。早在去年 10 月，Greg Sterling 就预测到了脸书的实时搜索，这是真的:这是显而易见的。这两家公司应该在微软去年秋天写给脸书的 2.4 亿美元支票墨迹未干时就这么做。

脸书拥有 9000 万活跃用户(根据他们的统计页面)，comScore 称他们每月为[提供 1 . 57 亿次搜索查询](http://www.comscore.com/press/release.asp?press=2337)(尽管奇怪的是，脸书报告称他们一年前就已经为[每月 6 亿次搜索提供服务](http://blog.new.facebook.com/blog.php?post=2535632130)，当时他们拥有大约三分之一的用户数量——这种差异可能是由于 comScore 只测量了美国的搜索)。这对于在 comScore 观察到搜索活动的网站列表中排名第九是件好事。

诚然，大多数搜索查询可能是针对人、团体、事件或应用程序的，但随着去年 11 月脸书页面的增加，该网站可能会出现更多与产品和娱乐相关的搜索。如果微软和脸书能够说服该网站的 9000 万用户开始在脸书境内进行网络搜索，这对微软来说可能是一个福音，实际上会对谷歌的搜索市场份额造成一点点削弱。

脸书的网络搜索是我在 5 月份呼吁的，但我指出，为了实现这一点，脸书将不得不做一些它似乎从未想过要做的事情:让它的用户离开网站。然而，随着昨天[推出脸书联通](https://www.facebook.com/sitepoint/)，脸书似乎不太关心让用户一直待在脸书。

如果脸书和微软能够将脸书用户转化为网络搜索者，他们将为微软在搜索市场提供一个潜在的立足点。这看起来不像是一个飞跃，尤其是如果他们按照 [Mashable 的建议](http://mashable.com/2008/07/24/facebook-microsoft-search-deal/)去尝试一些独特的东西，利用脸书的社交联系数据来增强搜索体验。我们希望在今年秋天看到一些颠覆性的东西。
