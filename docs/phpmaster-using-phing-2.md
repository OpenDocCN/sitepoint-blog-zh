# PHPMaster:使用 Phing

> 原文：<https://www.sitepoint.com/phpmaster-using-phing-2/>

Phing 是一个基于 Apache Ant 的 PHP 项目构建工具。构建系统帮助您使用单个命令执行一组操作。如果您想知道为什么 PHP 需要一个构建工具，请考虑一个工作流，您在本地机器上编写代码和单元测试，如果测试通过，您将代码上传到 staging/production server 并对生产数据库进行任何更改。
