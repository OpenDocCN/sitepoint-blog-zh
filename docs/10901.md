# 如何在生活和工作中感觉更平衡

> 原文：<https://www.sitepoint.com/balance-in-life-and-work/>

虽然创造完美的工作生活平衡可能有点不现实，但有一点是肯定的。我们都需要感觉到生活的各个方面都很稳定，运转良好，以便继续朝着我们的目标前进，尽可能健康地度过每一天。

感觉平衡是关于控制你的直接环境，尤其是你如何看待它。这种控制不仅可以降低你的压力水平，还可以让你在一个全新的水平上享受生活。一旦你达到那个水平，你减少的压力和因压力减少而增加的快乐甚至会让你周围的人更快乐。会传染的；你的平衡感对每个人都有好处。

那么，你准备好获得平衡了吗？这里有一些方法可以开始在你的生活和工作中创造一种平衡感。

## 设置边界

我以前说过，现在我再说一遍——你需要界限，尤其是如果你在家工作的话。你应该对你的工作环境、家庭、客户甚至你的总体期望有所限制。没有界限，你可能会发现自己处于一种自由竞争的状态，让你过度劳累，压力过大。

您的界限可以基于您的物理环境，例如设置家庭成员在您的家庭办公室可以做什么和不可以做什么的基本规则。界限也可以是精神上的指导方针，比如制定一个政策，概述你的周转时间和为客户工作的时间。

## 集中注意力

为了感觉平衡，你需要能够专注于你要去哪里，以及你打算如何到达那里。这是你的目标、优先事项和纪律发挥作用的地方。当你集中注意力时，你可以在任何给定的时间专注于你需要采取的最重要的行动，并消除(或至少忽略)那些可能模糊你注意力的常见干扰。

你可以专注于你的工作，你的目标，你的优先事项，甚至是休息时间。

## 灵活一点

不幸的是，对一些人(比如我)来说，在工作与生活的平衡上没有完美的空间。你需要放下你想立刻完美完成这一切的欲望。严酷的事实是，有些东西通常必须付出。

有时候，这种灵活性意味着给自己留点时间，委派任务，学习如何随波逐流。对一些人来说，这是一个艰难的心态，但是一旦你到达那里，你肯定会感觉更加平衡。
