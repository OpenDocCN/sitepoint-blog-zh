IE9 下载将在 beautyoftheweb.com 的[和 T2 的](http://www.beautyoftheweb.com/)微软浏览器页面进行。我确信我不需要提醒你，但是:

*   IE9 可以安装在 Windows Vista 或 Windows 7 上。尽管我不断抱怨，微软不太可能发布 Windows XP 版本。[也许是时候考虑升级了？](https://www.sitepoint.com/windows-7-review-3/)
*   IE9 将删除您现有的 Internet Explorer 版本(可能是 IE8，但一些 Vista 用户可能有 IE7)。你可以[还原回](https://www.sitepoint.com/how-to-uninstall-ie9/)，但是[用 XP 模式或者其他虚拟机软件](https://www.sitepoint.com/windows-7-browser-testing/)可能更好。

我们大多数人都体验过 IE9 的测试版或发布候选版本，所以不太可能有什么大的惊喜。然而，这是大多数家庭和商业用户将安装的第一个版本。不管你对 Internet Explorer 有什么看法，它仍然是世界上最常用的浏览器，IE9 将成为你开发生活的重要组成部分。

有几个重要的问题需要问自己:

1.  您是应该现在安装 IE9，还是等到大部分客户端都完成迁移后再安装？
2.  如果你一直没有使用 HTML5 和 CSS3，现在是时候开始研究这些技术了吗？
3.  哪些现代技术可以用于 IE9，更重要的是，还缺少什么？

你今天会安装 IE9 吗？请关注 SitePoint，了解即将发布的 IE9 评论、技巧和教程…
