# 让你的博客和 PHP 出现在地图上

> 原文:[https://www.sitepoint.com/get-your-blog-and-php-on-the-map/](https://www.sitepoint.com/get-your-blog-and-php-on-the-map/)

Artima 是一个专注于软件开发的网站。如果你想让你的软件开发平台/工具/方法/任何东西受到一些严肃的讨论和关注，这是一个非常好的地方。

另外，它有一些优秀的内容和对业务的洞察力。例如，如果你在寻找关于契约式设计的更多信息(马库斯·伯杰正在为[标准 PHP 库](https://www.sitepoint.com/php5-standard-library/)进行调查)，试试[对伯特兰·梅尔](http://www.artima.com/intv/contracts.html)的采访。

无论如何，PHP 现在已经在 Artima 的 [BUZZ](http://www.artima.com/buzz/index.jsp) 中崭露头角(我可以说是在 Perl 和 Python 之后很久)——一个软件相关博客的 feed aggegrator。比尔·凡纳斯在这里解释了 buzz 的概念。Buzz 帖子也会通过 RSS 再次转发(然后直接发送给 Redmond 的 RSS 读者……；)).

PHP 提要列表[在这里](http://www.artima.com/buzz/community.jsp?forum=271)——目前有点空，尽管[克里斯蒂安](http://blog.bitflux.ch)和[西蒙](http://simon.incutio.com)可以在这里[找到](http://www.artima.com/buzz/community.jsp?forum=126)。

因此，如果你是一个 PHP 博客作者(或者用 RSS 运行一个 T2 的维基网站)。你首先需要注册，然后通过“加入 Buzz”添加你的 feed。是时候把 PHP 放在地图上了！
