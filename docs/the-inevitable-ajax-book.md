# 不可避免的 AJAX 书籍

> 原文：<https://www.sitepoint.com/the-inevitable-ajax-book/>

以下内容转载自《科技时报》第 141 期。

或许不可避免的是，SitePoint 已经出版了一本 AJAX 书籍。 [<cite>构建你自己的 AJAX Web 应用</cite>](https://www.sitepoint.com/blog/) 今天在 sitepoint.com 开始销售，并附带一份由你的忠实用户提供的免费 AJAX 介绍视频教程。

[![Build Your Own AJAX Web Applications cover](img/4825914bc3a4770b4905ce6019af08bc.png)](https://www.sitepoint.com/blog/) 我记得去年在塞瓦斯托波尔的奥莱利销售会议上，我和计算机图书出版界的一些大人物在一起。负责向书店销售所有公司产品的一名销售代表把我和马特·密茨凯维奇逼到角落里:“我们将在未来三个月内出版六本 AJAX 书籍。你的呢？”

说实话，SitePoint 从来不是第一个出版特定主题书籍的网站。我们更喜欢稍微犹豫一下，等到我们有独特的东西要说的时候。

因此，你会发现 <cite>[构建自己的 AJAX Web 应用](https://www.sitepoint.com/blog/)</cite> 是一本脱颖而出的 AJAX 书籍。我们看到 subejct 上的其他书籍一次又一次失败的地方是在提供可访问性问题的答案:如何才能生成不破坏浏览器导航工具(即书签和后退按钮)的 AJAX 应用程序，或者让屏幕阅读器等辅助技术的用户处于黑暗中？

我很自豪能为一家出版商工作，在那里，这是一种能登上我们书籍封面的“促销文案”:

> 这本书将帮助您构建负责任的、专业的、健壮的 AJAX 应用程序，增强用户体验，同时确保最小的可访问性影响。

当然，在这本书里你会找到所有的代码来生成任何优秀的 AJAX 应用程序所需要的漂亮的淡入淡出、幻灯片、即时反馈和拖放功能，但是所有这些都是以你可以在现实世界中实际部署的方式实现的，在现实世界中向后兼容性和可访问性是非常重要的。
