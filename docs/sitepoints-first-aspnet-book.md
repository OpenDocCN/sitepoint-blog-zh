# SitePoint 的第一本 ASP.NET 书籍！

> 原文：<https://www.sitepoint.com/sitepoints-first-aspnet-book/>

如果我是一个政治家，我会让反对派尖叫“利益冲突！”我必须辞职才能有更多的时间和家人在一起。

幸运的是，我只是一个谦虚的书呆子，所以如果我谈论 SitePoint 的[用 c#&VB.NET](https://www.sitepoint.com/books/aspnet1/)建立你自己的 ASP.NET 网站，我只是在传播这本书有多好的诚实的话。

我通常不喜欢教科书。我经常发现在 Visual Studio 中不停地学习，感到很沮丧，但是边学边做是我喜欢的教学方法。我经常在书上看到例子，然后立刻失去了兴趣…哦，太好了，又是一个“你好，世界”或者“你叫什么名字”的剧本。

但是，当我有机会提供对 SitePoint 最新的的评论时，我被这本书的易读性所震惊。我对这些例子很感兴趣，这些类比令人耳目一新，尽管我认为自己非常精通。NET，我发现自己学到了一些新的技巧和方法。

我经常看书，问“是的，但是为什么要那样做？”或者“我如何在现实中使用这个？”。我对这本书也做了同样的事情，然而它回答了我几乎所有的问题。C# Unleashed 对我也有类似的影响，我认为它是最好的书之一。但在我看来，这本书甚至超过了那本书。不平凡的壮举！

很久以来，我一直听到人们对 SitePoint 书籍的优点赞不绝口..PHP 选集和[网页设计商业工具包](https://www.sitepoint.com/books/freelance1/)跃入脑海。而现在。NET 加入这一杰出的行列，也非常受欢迎:)

最重要的是，你可以在购买前在线阅读前几章。第一章[今天](https://www.sitepoint.com/article/asp-dot-net-introduction)在网站上提供，或者访问[书的主页](https://www.sitepoint.com/books/aspnet1/)通过电子邮件接收 4 章。
