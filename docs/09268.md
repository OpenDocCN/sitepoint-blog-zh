# 什么是销售周期，我为什么需要销售周期？

> 原文：<https://www.sitepoint.com/whats-a-sales-cycle-and-why-do-i-need-one/>

销售在转变中有赢有输。这是你需要一个清晰定义的销售流程的首要原因。它允许您过渡到下一个逻辑步骤，以完成销售。我的意思是:

你花了一个多小时与潜在客户交谈，倾听他的需求，提出建议，并给他报了一个大概的价格。他隔着桌子看着你，说:“我得考虑一下，然后给你答复。”

没有现成的销售流程，也没有清晰的“下一步”，你就失去了控制。如果没有回音，你会含糊地说“一周左右跟进”。然后你离开。

当然，一个星期过去了，你没有回音。你打电话留言。他不回你电话。

不想成为一个讨厌的人，你等了几天才发出一封电子邮件。你只能假设它一定是从 DNS 服务器上反弹回来，落到了一个黑洞里。(不，你听到的吸吮声不是塌缩星物质；你的客户群正在流失。)

这样再来几轮，你就会开始觉得你被拒绝了。最终，你放弃了，继续前进，但在你的内心深处，你想知道…也许如果你再打一次电话…也许他只是太忙了…也许他在医院，或者在度假，或者在异乡执行秘密任务。

我不知道你怎么想，但我需要一个了结。我就是那样敏感。

让我们倒回到他告诉你他会“回复你”的时候。这一次，你*确实*有一个销售过程，你知道这不是一条单行道，你做出所有的承诺并通过所有的关卡，而潜在客户什么都不做。相反，你知道这个过程涉及到相互的承诺。你承诺与他会面，讨论他的需求，并向他提出解决方案。他现在比你们认识之前更富有。他欠你一个礼貌，给你一个“是”或“不是”的回答，并承诺*他会在哪天什么时候*把它给你。所以这一次，你马上拿出你的记事本(或者你的装有你最喜欢的生产力应用程序的智能手机[，说:“没问题，鲍勃。哪天和什么时候我们可以再见面，这样我就可以得到你‘是’或‘不是’的回答了？”](https://www.sitepoint.com/my-desk-has-exploded/ "Help! My Desk Has Exploded")

在那简短的声明中，我完成了三件大事。首先，我要求一个承诺。第二，通过把它写在我的日历上，我要求他和我一样重视它。最后，通过问他“是或否”的答案，我就消除了潜在客户在事后回避我们的主要原因。要么他们觉得告诉你“不”很糟糕，要么他们害怕你会像推销员一样说服他们，想知道为什么，并试图说服他们继续前进。现在，你可以选择不轻易拒绝，但至少你让他知道拒绝你是可以的。

你看，“控制”销售过程并不意味着操纵潜在客户购买。它控制的是过程，而不是结果。你想让销售拜访圆满结束。我说的“成功”是指，在某些时候，你得到的是“是”或“不是”…而不是“可能”。因为“也许”是最糟糕的地方:

“亲爱的，我爱你。你是我的女孩。你愿意嫁给我吗？”

“嗯，我会给你回复的…”

控制只有在过度时才是不好的。在销售拜访中，保持一定的控制力是一件好事；否则——取决于你潜在客户的性格类型——你要么会觉得自己和一群三岁的孩子在操场上呆了一天，要么会觉得自己和新兵训练营的教官在一起。这就像控制你的膀胱一样…有“憋”的时候，也有顺其自然的时候。努夫说？
