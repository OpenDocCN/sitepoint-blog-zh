表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏表示比例变化，即上个月 10.5%的 IE7 用户更换了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

虽然 Chrome 无法赶上上个月的涨幅，但其他供应商将会因 0.39%的涨幅而欣喜若狂。IE8 下降了类似的数量可能是一个巧合，但 Windows XP 的[结束将是一个促成因素。迁移到 Windows 7/8 的用户和企业必须考虑升级所有软件；坚持使用 IE8 很难——如果不是不可能的话。IE9+比其前身更接近 Chrome 或 Firefox，因此没有理由不评估替代产品。](/is-today-beginning-of-end-for-ie8/)

到 8 月份，Internet Explorer 的市场份额可能会降至 20%以下。IE11 是一个很好的浏览器，但是除了在几个领域的原始速度之外，没有什么可以吸引 Chrome 用户更换。微软的纯 Windows 政策也开始影响使用。越来越多的开发者在使用 Mac OSX 或 Linux，这两个系统都不允许你安装 IE 的原生版本。

其他供应商的表现更好，火狐和 iPad Safari 的销量有所增长。也就是说，Mac 上的 Safari 和 IE 有相似的问题:它只能在苹果操作系统上使用，并不比竞争对手更好。

## 2014 年 4 月至 5 月全球移动浏览器统计

2014 年 5 月，手机使用量增长了 2%以上，占所有网络活动的 27.04%。人们预测手机将占据主导地位已经有一段时间了，但 StatCounter 的 [6 年手机/桌面对比图](https://gs.statcounter.com/#desktop+mobile-comparison-ww-monthly-200812-201406)强调，这在过去的 12 个月里开始成为现实。

热门移动浏览应用:

| 移动浏览器 | 四月 | 五月 | 变化 | 亲戚 |
| 机器人 | 25.52% | 25.26% | -0.26% | -1.00% |
| 苹果手机 | 21.14% | 21.34% | +0.20% | +0.90% |
| 铬 | 15.62% | 18.33% | +2.71% | +17.30% |
| Opera Mini/Mobile | 12.26% | 11.60% | -0.66% | -5.40% |
| UC 浏览器 | 11.09% | 10.39% | -0.70% | -6.30% |
| 诺基亚浏览器 | 4.82% | 4.35% | -0.47% | -9.80% |
| 不变的 | 2.38% | 2.38% | +0.00% | +0.00% |
| 其他人 | 7.17% | 6.35% | -0.82% | -11.40% |

Chrome 在一个月内又实现了 3%的惊人增长，但令人惊讶的是，Android 股票浏览器并没有出现类似的下降。谷歌的移动浏览器看起来将在 7 月或 8 月超越 iPhone 版的 Safari。

抱歉，黑莓用户——我已经从图表中删除了这些设备。使用率已经下降到 1.5%，很难看出该公司如何复苏。
