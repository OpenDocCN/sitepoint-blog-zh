# 如何把潜伏者变成海报

> 原文：<https://www.sitepoint.com/turn-lurkers-posters/>

建立一个在线社区可能是一个相当大的挑战——尤其是在你刚刚起步的早期。

世界上所有酷软件和 groovy 头像都不会把潜伏者变成用户——或者让用户发帖。没有帖子，你的会员就没有理由回来，你的社区就会失败。

最近，我们聚在一起，想出了我们的 5 大热门提示，以鼓励客人加入您的会员，并在您的论坛上吸引论坛成员的帖子。

##### 1.欢迎并帮助新成员

一些流行的论坛似乎对新手不太友好——在某些情况下，新成员会自动受到虚拟的戏弄。“嗨我是新来的！”在一些论坛中，帖子被认为是垃圾邮件，因此会受到冷处理，或者遭到猛烈抨击。

但是很自然的，用户会想要介绍他们自己，并且在社区中认识其他人。

这就是为什么“介绍”论坛是一个好主意。它不仅有助于打破僵局，让新人感到受欢迎，而且还会不断重复“嗨，我是新来的！”帖子不会出现在你论坛的随机位置。( *[)安德里亚夫](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=23900) )*

对于潜伏者和新成员来说，它提供了一种测试水域的方法，而不必在一个已建立的线程中发布。发帖的感知“风险”越低(还有，潜在的，让自己出丑！)你就越有可能发帖。甚至一个“潜伏者自我介绍”的帖子也可以吸引许多新成员，这是一个很好的方法来推动用户越过注册社区的障碍。

此外，一定要在 2-4 小时内回复任何新手的帖子——让这成为你整个网站的标准，并确保你的论坛工作人员执行这一政策。让我们面对现实吧，当我们参与谈话时，我们希望得到认可。忽视新海报的第一个帖子是死亡之吻:谁愿意冒第二次被忽视的风险？( *[)姆法默希](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=14021) )*

这还没完！尊重并关注新手的问题。尤其是在许多邮件列表中，有一句“你没读过手册的第 633 页吗？”一点都不好玩的态度。让你的新成员感到舒适和放松，以快速、礼貌和友好的方式回答他们的问题，他们将更有可能再次发帖。(*[)Samsm](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=17235))*

##### 2.与您的成员保持联系

除非你有一个重大的声明(比如赠品)，否则你可能不会给所有成员发电子邮件(除非这是一份他们有意签署的名单)。你不想让人们认为你在给他们发垃圾邮件:没有什么会比这更快地毁掉你的社区。

有一个 VBulletin hack 可以让你每周发送一次“社区公告”。它列出了最活跃的线程，生日，新成员，论坛活动等。那是另外要考虑的事情。( *[基尔切](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=14698)* [r](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=14698) )

但是无论你使用什么样的论坛软件，电子邮件都是一种有效的方式来激发人们对你的论坛的兴趣。

不难发出一个提醒，告诉成员们当前正在讨论什么话题，或者(如果你的留言板允许你发布一个投票)，让用户知道当前的投票。有些事情一定会激起你们社区成员的兴趣。在每个主题旁边包含一个特定主题的链接，这样会员就可以很容易地当场发表意见。( *[帕](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=21398)* )

还可以考虑在你的更新中创建一个小的“电子快讯”——一个快速、相关的社论可以帮助你吸引对重要新闻的注意，同时在你的社区周围建立一个更加个人化、友好的氛围。

##### 3.举行竞赛

鼓励发帖的竞赛可能是个好主意，但是要小心——这里面有一些障碍！

任何奖励人们发帖数量的竞赛都会鼓励人们毫无意义地发帖来获得奖励。记住，在一天结束的时候，你的论坛的质量会带来新的成员。每个新社区都想要帖子，但你希望他们是有质量的帖子！()

这里有一个恰当的例子:

<q>*“几年前，有人想推广他的论坛，找到了我。我犯了你现在犯的同样的错误:我组织了一场比赛，将有 3 个不同类型的获胜者获得 3 个奖项:我们为一个月内发布最多的成员提供了一个大奖，一个随机抽奖，以及一个最原创帖子的奖项(由论坛版主投票)。*</q>

<q>*结果呢？因为第一次比赛，有人在 3 周内发了 2500 条信息。我写了一篇详细的帖子，列出了构成“有效帖子”的规则和条件，但人们并不在乎。他们以为我们不会检查他们。”* ( *[)布鲁诺](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=9664)* )</q>

那么，办好一场比赛的秘诀是什么呢？提供与你的观众和论坛主题相关的好的奖品，根据成员帖子的质量而不是数量来授予奖品。

一个可能激发你想象力的想法是围绕一个在你的论坛上吸引大量注意力的特定热门话题来设计你的竞赛。向与讨论主题相关的产品和服务提供商提供赞助——这些赞助商提供奖品，作为回报，他们会在你的网站或你的时事通讯中获得广告空间。

一旦你的社区变得更加成熟，你也许可以用“客串”来扩充这种方法。询问赞助公司，他们的专家是否可以在你的论坛上客串，回答用户的问题，并在比赛期间加入讨论。也许专家也会愿意评判比赛结果并颁发奖品。同样，这可以增加你的论坛的个人友好感，激发有价值的会员兴趣和帖子。

##### 4.使用对话开场白

如果今天论坛进展缓慢，为什么不发起一些讨论呢？

当然，你不希望你的整个董事会被喋喋不休淹没——你需要与你的网站主题相关的好帖子。

但是你也希望你的用户谈论其他问题。当他们开始谈论他们最喜欢的电影、歌曲或书籍时，他们开始交网上朋友并建立关系。关系是一个强大社区的关键。( *[Pgowder](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=9010)* )

因此，创建几个假账户，在论坛上播下有趣的话题。通过吸引用户，你鼓励了他们的活动——他们会更频繁地发帖。

事实上，你可以买一些关于问题和对话的书籍。选择几个有趣的问题，张贴在你的论坛上，然后在你的时事通讯和网站首页提及它们。( *[Divamissx](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=13847)* )

当然，把问题针对你的观众…年轻的海报可能不容易被政治所吸引，老的海报不会那么关心视频游戏。( *[Samsm](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=23900)* )

争议牌有效。它不仅能让会员们附和自己的观点，而且能向你的会员证明这个版块用得很多，并激励他们开始自己的话题！()

##### 5.让会员资格成为必备条件

有很多方法可以利用社区的即时性——说服潜伏者咬紧牙关，花点时间注册并加入的方法。

例如，考虑将论坛的最后 5 篇文章添加到你站点的主页上。这样，每次有人浏览你的网站时，他或她都会看到论坛上的帖子。当他们遇到感兴趣的内容时，他们会点击进入……然后会发生什么呢？他们想有所反应，开始发帖！即时会员！( *[布鲁诺·德尔皮埃尔](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=9664)* )

或者，如果你使用 vBulletin，vbulletin.org 有一个黑客软件可以让你向客人而不是会员展示广告。广告旁边有一条信息，上面写着“立即注册删除此广告”。这是一种提供“会员福利”的简单方法，这种福利是真实的、有形的，对当今绝大多数网络用户都有吸引力。(*T3【基尔彻 *

##### 打开那个潜伏者！

潜伏者很像“学校里的新同学”——他们站在场外，观察他们的同学如何互动，并试图找到一个适合他们并让他们开心的地方。

当然，每当他们接近一群人时，新来的孩子都冒着被拒绝的危险，而且“看起来很笨”。在线上，潜伏者可以从更远的角度观看，但基本的动态是一样的。

你必须向潜伏者展示，你的论坛是个绝佳去处，在这里，他们的问题可以得到解答，受到欢迎和尊重。一旦你实现了这个目标，你可能需要给他们一些激励，让他们花时间去注册。当然，一定要制定一个好的策略来鼓励重复发帖:一旦你拥有了那些有价值的会员，你就想留住他们！
