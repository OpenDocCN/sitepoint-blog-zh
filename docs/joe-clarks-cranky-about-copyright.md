# 乔·克拉克对版权很敏感

> 原文：<https://www.sitepoint.com/joe-clarks-cranky-about-copyright/>

阅读一本正在进行的书是一个不寻常的概念。作为一名技术编辑，我可以在 SitePoint 做这件事，看着一本书成型是一件非常有趣的事情。如果你想成为这一过程的一部分，并且你对版权感兴趣，那么[乔·克拉克](http://joeclark.org/)(是的，就是无障碍迷们所知的那个乔·克拉克)刚刚宣布了他的最新项目:<cite>古怪的版权书</cite>。

如果你正在读这篇文章，很可能你也是一个创造者。无论你是写文字、画图形、拍照片，还是构建博客主题，版权都是一个影响我们所有人的重要概念。如果你想重用你在别处找到的内容，这一点尤其正确。很多人对现状不满意——当然，有[创作共用，](http://creativecommons.org)但是仍然有很多批评的空间。乔问道:

> 如果你所知道的关于版权的一切都是错的呢？
> 
> 嗯，不是的。我不是来告诉你这是真的。但我确实想让你摆脱自从拉里·莱西格成为你的世俗之神以来你一直被施的魔咒。
> 
> 真的有一种看待版权的方式，它不是为垂死的“内容”产业(如音乐产业和好莱坞工作室)辩护。这种新的方式也与“自由文化”、“创作共用”以及像莱西格和迈克尔艾斯特这样的大师的教导没有什么关系。

野蛮女友。无论你同意还是不同意这一立场，我相信我们都同意在版权辩论中应该有更多的声音，乔即将出版的书承诺做到这一点。我还感兴趣的是，他正在请求微缩技术来帮助他的书起步。捐助者(或者我们应该称他们为微受益者？)将能够在测试版章节出现时阅读它们，它们将在最终版本中得到认可，最终版本将以电子版和纸质版两种形式出版。

现在还为时尚早，但这看起来将成为出版界一个有趣的实验——这是一个重要的话题。你可以在其网站上找到更多关于古怪的版权书籍项目的信息。
