# 插图:HTTP 错误看起来像什么

> 原文:[https://www . site point . com/illustrations-what-http-errors-look-like/](https://www.sitepoint.com/illustrations-what-http-errors-look-like/)

这些实际上是几年前的，所以 Flickr 的狂热用户可能已经看过了。然而，我昨晚第一次看到它们，是通过黑客新闻上的[链接，发现它们非常有趣，所以我想我应该分享一下。他们做了一份不错的轻松的周五邮件。](http://news.ycombinator.com/item?id=370972)

以下是漫画家亚当·科福德(Adam Koford)的插图(他现在似乎把大部分时间都花在了写一对流浪汉笑猫的连环漫画上)。它们是 HTTP 错误的相当巧妙的例子。我在下面列出了一些最聪明的，但是在最初的 [Flickr 设置中还有更多](http://www.flickr.com/photos/apelad/sets/72157594388426362/detail/)(准确地说是 16 个)。

哪些是你最喜欢的？我更喜欢“400 错误请求”(右图)和“不允许 405 方法”(图中未显示)。亚当，如果你正在阅读:请把这些做成一套印刷品或明信片！
