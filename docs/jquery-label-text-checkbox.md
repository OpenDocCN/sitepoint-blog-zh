# jquery 获取与复选框关联的标签文本

> 原文：<https://www.sitepoint.com/jquery-label-text-checkbox/>

获取与复选框关联的标签文本的简单 jQuery 代码片段。

```
console.log($('label[for="carHireChk"]').html());
$("label[for='comedyclubs']")
$("label[for=' + this.attr("id") + ']").attr('class', 'error');
```

更多信息见 doco:[http://api.jquery.com/next-adjacent-Selector/](http://api.jquery.com/next-adjacent-Selector/)
