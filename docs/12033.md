# 当心“谷歌”骗子

> 原文：<https://www.sitepoint.com/google-swindlers/>

越来越多的我的客户和同事报告了自称为谷歌工作的人打来的陌生电话。打电话的人通常会把公司的网站放在搜索结果的顶部。这项服务的费用各不相同，但通常是每月 150 美元左右。

这些电话不一定是打给大型跨国公司的。事实上，大多数目标似乎是中型公司和个体交易者。奇怪的是，打电话的人坚持留下电话号码，而不是电子邮件或任何其他形式的互联网通信地址。

当然，这是一个骗局。打电话的人只是在推销 AdWords 广告；他们会花一些钱在网上活动，并为自己保留每月订阅的大部分。他们甚至可以使用高度具体的关键字，只针对客户的直接区域，以确保 AdWords 的成本保持在最低水平。

大多数合法的搜索引擎优化和营销公司都提供 AdWords 帮助，但伪装成谷歌的做法让这种服务看起来更可信、更合法、更划算。充其量，打电话的人是在虚假的借口下交易。最坏的情况是欺诈。

请记住:

1.  谷歌不会在他们的搜索引擎中出售好的位置:如果他们出售，搜索结果将毫无价值。他们确实通过他们的 AdWords 系统提供赞助链接，但是如果打电话的人没有提到，就要小心了。
2.  打电话的人应该能够回答关于他们提供的服务的基本问题，例如，什么关键字会成为目标？搜索结果会出现在本地还是全球？我应该向谁付款？
3.  如果谷歌“员工”联系你，他们应该能够提供一个内部电子邮件地址，而不是@gmail 或@googlemail 账户。

不幸的是，互联网上有太多的罪犯和骗子。帮助你的客户避开骗子是你应该提供的另一项服务。

你被谷歌的陌生来电者找过吗？你是闻到了骗子的味道，还是他们实际上是合法的？
