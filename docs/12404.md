# 如何在 Twitter 上找到突发新闻

> 原文：<https://www.sitepoint.com/how-to-find-breaking-news-on-twitter/>

![](img/c6e2ee51e7363fb602b37de307a9a28f.png "newspaper-icon") * **更新**:真是古怪的巧合。几乎是在我点击帖子的时候，我就从 Matt Magain (SitePoint 的执行主编)那里听到 Twitter 上关于洛杉矶地震的报道。你可以通过[搜索 Twitter](http://search.twitter.com/search?q=earthquake) 来跟踪直播。还检查了趋势[像疯了一样跳](http://twist.flaptor.com/?gram=earthquake&table=0&tz=-5)上扭。(注意:CNN 确实有一个关于地震的横幅，但根据许多推文，Twitter 又一次抢先了一步。不过，我无法证实。)*

关于 Twitter 的成功，一个经常被提及的大故事是关于它作为一个突发新闻平台的出现。Twitter 不与传统的书面新闻竞争(博客本质上是这样的)，但由于其速度和始终在线的性质，它已经发展成为记者获取突发新闻的一个伟大工具。

Twitter 的创始人意识到了它在小地震早期(T0)发布新闻的潜力，2007 年，Twitter 确实发布了墨西哥地震的新闻。去年，该短信服务击败了主流新闻机构(有一次甚至击败了美国地质勘探局)报道了两次地震——一次是 2 月份在英国，另一次是一个月后在中国发生的更大地震。

我们去年 10 月写道，Twitter 作为一个新闻发布平台[，有四个原因](https://www.sitepoint.com/twitter-as-a-tool-for-social-change/):

1.  很快。
2.  它开着。
3.  很简单。
4.  这是两条路。

但也很分散。在 Twitter 上跟上突发新闻并不容易，除非你碰巧关注了那些在他们发微博的时候正在发生的事情。这可能不会发生，除非你是罗伯特·斯考伯，跟随无数人，并且从不睡觉。

Twitter 也有点混乱——噪音和信号的比率很高，所以很难区分哪些有新闻价值，哪些没有。那么，如何通过 Twitter 了解正在发生的事情呢？最近推出的一个名为 [Twopular](http://twopular.com/) 的网站可能会有所帮助。

Twopular 是一个为 Twitter 打造的趋势跟踪器，可以识别 Twitter 上过去 2、8、24 小时、一周或一个月的热门趋势。不足为奇的是，今天 Twitter 主要被新的 Palm phone、CES、MacWorld、Windows 7 测试版、加沙-以色列冲突和美国 BCS 大学足球锦标赛所占据。Twopular 提供了导致该术语趋势上升的顶级 Twitter 用户的链接(允许您点击进入实际的推文)，一个关于该趋势的实时 Twitter 搜索的链接，以及相关趋势的链接。

当与 [Twist](http://twist.flaptor.com/) 结合使用时，Twopular 成为跟踪 Twitter 上突发新闻的强大工具，Twist 是一个独立的工具，可以在时间线上绘制 Twitter 趋势并允许它们进行比较，而与 [Google Trends](http://www.google.com/trends) 结合使用时，Twopular 是一个基于搜索、新闻和博客数据的类似服务。当然，经常谈论的话题可能会出现在 Twopular 这样的趋势网站上，即使没有突发新闻，但当事情发生时，Twopular 这样的网站通常会及时发现。

Twitter 的[搜索页面](http://search.twitter.com/)也列出了热门话题，尽管它不如 Twopular 强大。Twitter 的搜索实际上是监控特定主题突发新闻的一种很好的方式。因为它是实时的，并且所有的搜索都可以通过 RSS 订阅源获得，所以你可以使用 Twitter 搜索页面不断地监控某些话题，并及时了解新闻。所有这些来源可以一起用来描绘一幅更丰富、更准确的世界图景。

当然，另一个选择是关注 [@BreakingNewsOn](http://twitter.com/breakingnewson) ，这是一个基于 Twitter 的新闻专线，经常在主流媒体之前发布新闻。例如，去年 2 月，他们[在 BBC 报道之前很久就在推特上发布了英国](http://twitter.com/BreakingNewsOn/statuses/763176852)发生地震的消息，大概是通过监控推特上的热门话题。他们现在正在[为计划中的扩张筹集资金。](http://pledgie.com/campaigns/2481)
