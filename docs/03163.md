# 我是如何开始技术布道的——第 2 部分

> 原文:[https://www . site point . com/get-started-technical-evangelism-part-2/](https://www.sitepoint.com/got-started-technical-evangelism-part-2/)

![Technical evangelist's perception](../Images/475bdb525de7c688b8241e6e583bb5f8.png)

经常有人问我如何开始技术布道，所以我想我应该把它写下来，以便更容易分享。你可以在这里找到这个系列的第一部分。

## 建设一个社区

我现在与游戏记者和开发者有了一些联系，但是我如何将两者结合起来呢？嗯，开发商总是在寻求宣传和联系作者，那么我为什么不能以一种对双方都有利的方式把两者结合起来呢？考虑到这一点，我借鉴了独立游戏冬季起义的成功，这是 XBLIG 开发社区前一年发起的一场推广活动。这场运动进行得相当好，并为在平台上推出游戏的团队获得了大量宣传，但似乎没有人想拿起火炬进行第二次活动。

![Indie Games Uprising](../Images/3576374426766515e0c65fa3b9066620.png)

考虑到这一点，我和另一位 XBLIG 开发者克里斯·斯蒂尔(Kris Steele)开始概述独立游戏夏季起义，并建立一个更大社区的活动。我在 GDC 2012 上做了一个关于这个的演讲，我有一个[完整的事后分析，所以我不会说太多细节，除了说我相信我们做了很好的工作，突出了那个平台上的许多最好的开发者。](http://www.davevoyles.com/indie-game-summer-uprising-post-mortem/)

为此，我们创建了一个网站来展示他们的游戏，在两周内连续推出了总共 10 款游戏，并通过 Twitter 和脸书积极吸引社区的参与，并允许他们投票选择他们希望通过推广看到的游戏。更好的是，我们不知何故得到了微软的支持，不仅团队得到了大量的压力，而且我们还得到了 Xbox Live 仪表盘的正面，当玩家启动他们的主机时，它就会出现。这个位置通常是留给为广告支付大笔费用的广告商的，而我们，和我这个建筑工人，正在帮助实现这个目标。

![Windows Game apps](../Images/e60a83cbea4dde3ee0b01f0312a776c3.png)

第二年秋天，我们继续创建了一个类似的活动，主要由迈克尔·希克斯(Michael Hicks)领导，他当时是一名学生，后来在 Xbox 360 上发布了游戏，以及他最近在 PS4 上发布的 *[支柱](http://blog.us.playstation.com/2014/08/25/introducing-pillar-a-ps4-psychological-minigame-collection/)* 。克里斯、迈克尔和我刚刚启动了该平台的第四次也是最后一次推广活动，以此来庆祝让这一切成为可能的开发人员。

在那里，我创建了纽约虚幻引擎聚会小组，每隔 3 周左右在我们的工地办公室举办一次。起初，我们平均每次会议有 25 人左右——一点也不差！困难的部分是找到其他知道虚幻剧本的人，这是一种相对晦涩的语言。我还参加了 NYU 等学校的游戏堵塞，并把它作为学习新技能和人际网络的一种方式。就在那时，也就是 2011 年，我第一次开始使用 Unity，并运用我的 C#技能。

**外卖:**

你不可能独自完成这件事。与比你更有才华的人一起工作，结合你的技能，创造你喜欢的任何事物。

## 面对拒绝

![Facing rejection](../Images/7dbebebe051da153eb5e5a93f7849c92.png)

由于与我一起工作的出色的人，我的名字终于出现在谷歌搜索中，这引起了微软某人的注意，因为我被要求面试 Xbox Live 独立游戏团队的一个角色，作为一个社区/项目经理，因为该平台开始有点缩小。我仍然清楚地记得，当我放下正在扫地的扫帚，在午餐时间在建筑工地上找一个安静的地方接听第二个面试电话时，我是多么兴奋。

![Jesse Freeman](../Images/3f308ab2cccf52e138453f19d2083443.png)

一路上，杰西·弗里曼(Jesse Freeman)和他的老板鲍勃·福莫瑞(Bob Familiar)在纽约游戏社区偶然发现了我的名字，我相信是通过我的网站，并邀请我有一天去他们的办公室。我当然答应了，当我发现一个福音传道者在做什么时，我被迷住了。在那里，我会参加 Jesse 主持的所有游戏活动和聚会，正是在这里，我第一次学会了 JavaScript，因为他是一名经验丰富的 web 开发人员，经常会提供教程。

在这个团队活动了几个月之后，鲍勃邀请我去采访，所以我飞到波士顿进行采访，但是你在上面的段落中看到了采访的结果。如果我不积极参与社区活动，我就不会遇到这两位伟大的人，如果我没有遇到他们，我也不会有机会采访他们。

几天后，我收到了回信，得知他们希望我在被雇用之前对编程和技术有更扎实的理解。我崩溃了，但我把它作为一种催化剂来学习更多。我觉得我已经很接近了，如果我已经引起了他们的注意，那么我得到这份工作只是时间问题。

![Xfinity](../Images/981e9d9b71793a57bd42cef3a56038c2.png)

紧接着，我参加了 Comcast 的面试，并被录用开发他们的 Xbox 360 和 Xbox One 应用程序，这些程序在很多方面帮助了我。开始职业生涯最困难的部分是获得第一份工作的第一步，对我来说就是这样。

**外卖:**

学会处理拒绝。如果你拍了很多照片，你会错过很多。但是你也在这个过程中学到了很多。

## 做点什么，什么都行

他们说最难迈出的一步是第一步，对于编程来说尤其如此。我经常遇到学生在试图找出他们应该从哪里开始编码时感到慌张，所以我为他们创建了这个资源列表。我真诚地相信学习编码的最好方法就是做一些东西。不要担心你是否从正确的地方开始，是否使用了正确的语言，或者是否瞄准了完美的平台——只要做点什么，一切都会变得更容易。

![Piz-ong](../Images/b7bcb465f63b7b07e2dd716a8a651ba0.png)

*我的第一个游戏，Piz-ong*

我从一个 C#的 XNA 游戏开始，然后用 C# / XAML 为 Windows Phone 开发了一个[儿童应用](http://www.davevoyles.com/xna-windows-phone-8/)，接着在 UDK 开发了一些原型(虚幻引擎 3-虚幻脚本)，然后投身于 web 编程。每个项目都变得更容易了，因为我从之前的项目中学到了更多的东西。

对某事有很棒的想法？去吧。如果你已经开始编程，并向他们展示你可以在第一线领导，那么招募其他人加入你的项目或想法会容易得多。没有人愿意被人从后面推着走，所以树立榜样，开始编码吧。

**外卖:**

从今天开始——做点什么。语言和平台并不重要，因为一旦你建立了一个项目，你就会有一些动力，一切都会变得更容易。

## 游戏堵塞和黑客马拉松是你的朋友

![Global Game Jam](../Images/defbe86efe20783e1e18b5411e3dc503.png)

你是不是大学刚毕业，没有人会因为你没有专业经验而录用你？你当然知道！所有这些你一直在做的游戏堵塞、黑客马拉松和兼职项目都是你的职业经历。

当我住在纽约的时候，我会去任何我能找到的比赛现场。Molydeux 和[全球游戏堵塞](http://globalgamejam.org/)是我脑海中最突出的两个游戏，因为我从每个游戏中学到了技能，也从我一路上遇到的朋友那里学到了技能。你会惊讶于科技和游戏产业有多小。去年在旧金山的 GDC，我遇到了一个人，他是我在纽约的一个 game jam 项目中的同事，现在是芝加哥一家工作室的程序员。

我通过 game jams 认识的其他朋友去了 EA 和 Zynga 等公司工作，或者去了美国各地较小的独立工作室，现在我永远拥有了这些联系，他们通过我与微软建立了联系。

![Betraille](../Images/43e29d78011eb5ac97faa9d635ec01c4.png)

游戏卡壳之后，[我们会立即在 LinkedIn](https://www.linkedin.com/in/davevoyles) 上发布一个项目链接，并添加对此做出贡献的人。这不仅让项目可见，也让参与的人可见。如果你能在网上主持游戏，那就更好了！有像独立城这样的网站，你可以提交你的游戏，现在你有一个很好的店面来配合你的项目。把这个加到你的简历里。

![Hackathon](../Images/0c8c9ccf27687b83243eb16e3068dec6.png)

现在我在事情的另一边，我经常在黑客马拉松和游戏堵塞中帮助招募一些我们在这些活动中发现的摇滚明星，并把他们带到船上。当你在黑客马拉松上看到科技公司的桌子遍布赞助商区时，这正是他们要做的——找到并雇佣最好的人。想想你比那些盲目地在网上提交申请并希望被看到的人有什么优势。现在，你正与那些可以推荐你和你的作品的人互动，但他们也可以在这个过程中指导你。

![Penn Apps Hackathon](../Images/17db704c287871046541fcec8798df1b.png)

*宾夕法尼亚大学的 Penn Apps 黑客马拉松*

回想起来，我希望我做了更多这些。你甚至可以用这样的东西来做一个完整的教育。我看到学生每年花费 5 万美元去上大学，但他们对花费 200 美元去参加大型黑客马拉松的想法嗤之以鼻，在那里他们可以获得我上面列出的所有好处。这向人们表明，你不仅对你正在建造的东西很了解，而且对你所做的事情和整个领域充满热情。

如果你获奖了，也把它列在你的电子简历上，无论是在你的 LinkedIn 还是你自己的网站上。如果其他人看到另一家公司发现它是一次活动中的成功黑客，这可能会给你正在做的事情增加很多可信度。

**外卖:**

去你能找到的每一个游戏聚会或黑客马拉松。你学习新技能，在旧技能的基础上再接再厉，不仅与同事建立关系网，还与许多你想为之工作的公司建立关系网。

## 把所有的放在一起

这大概总结了我作为一名技术传道者开始在微软工作的故事。我在康卡斯特呆了大约 8 个月，这是一段我很享受的经历，在那里工作的感觉非常棒。微软的这个角色一直是我梦寐以求的工作，所以当机会来临时我马上抓住了。

![Interview](../Images/181e340612c188492a8cf4c0036dc545.png)

当我在上面谈到面对拒绝，而我没有得到纽约的第一个职位时，我很失望，但继续前进和学习。不到一年后，我在费城得到了这个职位，最终一切顺利。

**外卖:**

继续坚持下去。不断学习，做事情，尽可能多的人脉，分享身边人的成功故事。
