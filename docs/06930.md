表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 5.9%的 IE7 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

正如我认为谷歌不可能进一步增加他们的份额一样，Chrome 也占据了浏览器市场 1%的份额。上一次出现这种情况是在 2012 年 7 月。从上下文来看，这相当于每个 Opera 用户或所有 IE6 和 IE7 用户在一个月内放弃了他们的浏览器。

Chrome 是上个月唯一有正增长的主流浏览器，很明显其他供应商正在努力竞争。谷歌是否拥有不公平的商业优势？很少有用户一天不使用搜索引擎、GMail、Analytics、 [Reader](/goodbye-google-reader/) 或其他重要服务，这些服务会委婉地建议他们改用 Chrome。谷歌也有现金储备在黄金时段的电视、电影、报纸和杂志上推广他们的浏览器。我们不要忘记谷歌控制着世界上最常用的智能手机平台和 Chrome 操作系统。

也就是说，没有好的应用程序，广告就一文不值。开发者喜欢 Chrome。用户喜欢 Chrome。即使是最狂热的 IE、Firefox、Safari 和 Opera 粉丝，对这款浏览器也没什么不好的。它可能不再享有显著的技术优势，但其他供应商必须要么生产更具竞争力的软件，要么希望谷歌变得自满。

IE9 已经在 18%左右徘徊了六个月，但是在最近的 Windows 7 更新之后，IE10 已经开始蚕食它的份额。3 月末，IE9 为 15.81%，IE10 为 2.26%。IE8 比其他任何浏览器都下降得多，本月可能会降至 10%以下。IE6 和 IE7 几乎没有变化，但它们已经变得无足轻重。

Firefox 又下跌了 0.5%，开始看起来像是月度趋势。这款浏览器在 Windows 上运行良好，不仅仅是 Chrome 的对手，但我怀疑它在 Mac、Linux 和移动平台上失去了大部分阵地。除非 Mozilla 能够说服用户留下来，否则到 2013 年 5 月底，Firefox 的市场份额可能会降至 20%以下。

Safari 这个月的表现比大多数人都要好，但是将近 6%的 Opera 用户放弃了桌面浏览器。这可能是一个统计上的异常，或者用户可能对该公司[转向 Webkit](/opera-switches-to-webkit-rendering-engine/) 不感兴趣？让我们看看它们在移动设备上的表现如何…

## 移动浏览器的使用

2013 年 3 月，手机使用在所有网络活动中所占比例上升至 [14.44%。](https://gs.statcounter.com/#mobile_vs_desktop-ww-monthly-201303-201303-bar)

主要的移动浏览应用程序:

1.  安卓——30.78%(下降 0.76%)
2.  iPhone——24.44%(增长 0.08%)
3.  opera Mini/Mobile——15.54%(增长 0.14%)
4.  UC 浏览器——8.27%(下降 0.07%)
5.  诺基亚浏览器——6.96%(增长 0.05%)

很难发现趋势，因为移动市场是不稳定的，受当地因素的影响。例如，iPhone 在西方国家非常受欢迎，但在亚洲、非洲和南美洲就不那么受欢迎了，因为 Android 和 Opera 在这些地方争夺头把交椅。

Android 的下跌有点不寻常，但我不会对此做过多解读。Chrome 手机浏览器的市场份额为 2.02%，并开始像其桌面版浏览器一样快速增长。谷歌无处不在。
