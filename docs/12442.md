# Web 2.0 消亡的那一天

> 原文：<https://www.sitepoint.com/the-day-web-20-died/>

对许多人来说，“Web 2.0”这个术语在很久以前就不再有任何实际意义了。对一些人来说，它从一开始就没有真正的意义。作为一个以撰写所谓的第二版 web 为生的人，我认为我已经尽可能地抓住 Web 2.0 这个术语不放了。但是今天，“Web 2.0”已经正式为我[跳过了鲨鱼](http://en.wikipedia.org/wiki/Jumping_the_shark)。这并不意味着我会停止使用它——作为一个笼统的术语来描述我所写的行业可能会有所帮助——但我不得不承认，它现在已经变得有点滑稽了。

几年来，定义 Web 2.0 就像是一个有趣的室内游戏。长久以来,[的人们一直试图给出 Web 2.0 的统一定义。但是，就像物理学中难以捉摸的万物理论一样，对于 Web 2.0 的真正含义，很难有一个统一的定义。](http://www.readwriteweb.com/archives/web_20_definiti.php)

也许最广为接受的定义是 Tim O'Reilly 的简洁定义:“Web 2.0 是计算机行业的商业革命，由向互联网平台的转移引起，并试图理解在这个新平台上取得成功的规则。”

但是即使是奥赖利的定义也已经改变并进化到了那个地步。

那么是什么导致我最终承认 Web 2.0 已经跳过了鲨鱼？它今天醒来，在 PC World 上找到了这个故事的链接，这是一个非常主流的计算机出版物:[成功求职的 Web 2.0 策略](http://www.pcworld.com/businesscenter/article/155446/web_20_tactics_for_successful_jobhunting.html)。

《PC World》推荐的“Web 2.0 策略”包括:推荐信、掌握最新技能和人际关系网。这不就是人们一直以来寻找工作的方式吗？“Web 2.0”到底是什么？名单上唯一可以被温和地认为与我们通常喜欢认为的 Web 2.0 有某种联系的项目是“提升你的在线形象”，作者建议加入相关的在线社交社区，如 LinkedIn 和 Twitter，写博客，并确保你在其他社交网站上的个人资料没有大学派对照片。

换句话说:Web 2.0 现在是一个主流的营销术语。事实上，Web 2.0 一直是一个营销术语。例如，O'Reilly 的公司拥有这个词的商标，用它来推广他们非常成功的网络会议系列。但直到今天，我还没有真正看到它以如此公然针对主流观众的方式应用，以使一些相当沉闷的东西显得更加时尚(我肯定这以前发生过，这只是我第一次看到)。

尽管如此，对 Web 2.0 的困惑——不管它意味着什么，也不管它现在如何被使用——是有帮助的。

去年四月，我写道，实际上没有 Web 2.0 或 Web 3.0 这样的东西，只有 Web T1。“Web 2.0 和 Web 3.0——它们实际上并不存在。它们只是分配给没有版本的东西的任意数字，”我说。“但我认为，这些术语引发的讨论有助于弄清网络的发展方向以及我们如何实现这一目标；这才是重要的。”

我认为这仍然是正确的，只要我们继续讨论并试图定义这些模糊的想法，我们将继续从讨论中获得价值。我在 4 月份写道，我不会告诉人们我写的是 Web 2.0，我会告诉他们我“写的是 Web，你现在可以用它做什么，以及你将来可以用它做什么。”我没有很好地履行这个承诺，但我仍然喜欢这个想法。
