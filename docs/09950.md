# 使用 jQuery 的 8 个动画视差示例

> 原文:[https://www.sitepoint.com/animated-parallax-examples/](https://www.sitepoint.com/animated-parallax-examples/)

jQuery is really taking website animation effects to a whole new level. Parallax effects create the apparent position of an object when it is viewed along different lines of sight to give perception and depth to the animation. Here are 8 awesome examples of jQuery with parallax effects used on different websites.

*   [创建自己的 jQuery 视差](http://jquery4u.com/animation/jquery-parallax-tutorial/)

## 1.Parallaxbokeh

CSS & jQuery 动画视差散景效果。这是一个被用作网站背景的动画视差效果，这种效果似乎会使网站变慢一点，但这是一个很好的例子。

[![parallax-david-leggett2](../Images/60e33518c8f9912261df3941e21d4e33.png "parallax-david-leggett2")](http://theleggett.com/demos/parallaxbokeh/) 
[来源](http://theleggett.com/2011/03/21/css-jquery-animated-parallax-bokeh-effect/)

## 2.jQuery 图像视差

史蒂夫芬顿的 jQuery 图像视差。这是一个插件，通过重复它和动画运动，给一个透明的图像(PNG，GIF)一种深度的错觉。
