# 对微软 Office 的真正威胁:也许不是网络应用

> 原文：<https://www.sitepoint.com/the-real-threat-to-microsoft-office-maybe-not-web-apps/>

最近，当谷歌在它的 Docs 网络应用程序中添加脚注时，引起了很多争议。这并不是因为添加脚注的能力是一个如此杀手级的功能——可能从任何人有记忆以来，微软 Word 就已经具备了这一功能——也不是因为它代表了任何伟大的工程技术。相反，谷歌文档中新的脚注功能最令人印象深刻的方面是它添加的时机。

据为 ZDNet 撰写谷歌博客的 Garett Rogers 称，就在微软首席执行官 Steve Ballmer 将 Docs 描述为非威胁的两天后，这家总部位于山景城的公司在他们的网络应用程序中添加了脚注。“你甚至不能在文件里加一个脚注，”他说。

这不是真正的功能，谷歌在添加它时表现出的灵活性给人留下了深刻的印象。“谷歌背后的灵活性和马力是公司——甚至是微软——应该警惕的，绝对不应该掉以轻心，”罗杰斯说。

然而，尽管如此，谷歌除了蚕食微软占主导地位的办公应用市场份额之外，还需要做更多的事情。据微软称，全球有多达 5.5 亿的微软 Office 用户。与此同时，根据 Quantcast 的数据，谷歌文档每月仅吸引微不足道的 130 万访客。

显然，谷歌要想摆脱微软对办公软件市场的垄断，还有很多工作要做。那么微软的摇钱树真正最大的威胁在哪里呢？

它可能只是来自开源世界的桌面软件。具体来说，[OpenOffice.org](http://www.openoffice.org/)，它在本月进行了一次重大版本升级。自发行版本 3 以来的第一周，OpenOffice.org 被[下载了 300 万次](http://www.computerworld.com/action/article.do?command=viewArticleBasic&articleId=9117575)。与微软 Office 假定的用户数量相比，这仍然是杯水车薪，但对微软来说，最可能令人担忧的是，80%的下载者是 Windows 用户。这表明 Windows 用户对微软软件的替代品持开放态度(尤其是免费的)。

OpenOffice.org 营销主管 John McCreesh 认为这些数字实际上低估了 Linux 用户，他们通常通过供应商获得更新。他猜测，现在可能有 500 万份最新版 OpenOffice.org 的安装拷贝。

“[2004 年]我们的目标是到 2010 年占有 40%的市场份额。麦克里什说:“如今，这似乎不像四年前那样雄心勃勃了。好吧，如果微软的数字是准确的，那实际上看起来是相当雄心勃勃的。但迄今为止的数字表明，对微软办公软件统治地位的最大威胁可能来自开源桌面软件，而不是来自云计算和谷歌。

有一点是肯定的:微软开始感受到来自各方的压力。
