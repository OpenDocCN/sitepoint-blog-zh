# SitePoint 背后的人

> 原文：<https://www.sitepoint.com/the-people-behind-sitepoint/>

现在这里发生的事情太多了，我的头都晕了。我闭上眼睛，当我睁开眼睛时，已经是八月了。我不知道今年上半年发生了什么——尽管平心而论，我想我每年都这么说。八月是我最喜欢的月份，因为当我庆祝我的生日周时，它被我亲切地称为 Hawkstravaganza。更重要的是，新的一个月对我们 SitePoint 社区来说意味着一些新事物的开始。

其中最重要的是我在脸书页面上的“了解我们”活动。几周前，我征求了您对页面上您希望看到的内容的反馈，出现了两个关键主题。一个常见的要求是像游戏或测验这样的互动挑战，另一个是希望更多地了解 SitePoint 背后的人。我一直觉得这两个主题中的后一个很吸引人，但我不会详细讨论它，因为它是下一个社区倡导者的主题。我要说的是，我已经采纳了你们的反馈，并提出了上述活动。

![SitePoint Forums](img/ee81ae784c770ab3cd06ad322a9f7ec1.png "sitepoint-forums")

这就是这场运动的运作方式。每个月我都会专注于一个特定的主题。八月份将会是 WordPress。我在我们的脸书页面上创建了一个名为“人物”的新标签。每周五我都会写一篇 SitePoint 员工的传记，这位员工在我们的业务中以某种方式使用 WordPress。我会告诉你他们为我们做了什么，他们如何使用 WordPress 以及为什么选择它。月底我会根据这些信息做一个小测验，我们会分发我们的新书*建立你自己邪恶的 WordPress 主题*作为奖励。如果这听起来像是你可能感兴趣的东西，请确保本周你停下来[我们的粉丝页面](http://www.facebook.com/sitepoint)并查看新标签(随时会出现)。

新的一个月的开始也意味着在 [SitePoint 论坛](https://www.sitepoint.com/forums/)的新的竞争。这个月，我们决定为你们当中的程序员举办一场编程比赛。因为我们平等地爱所有的程序员，你可以用你选择的语言来写这篇文章。我们想要你做的是写一个 [SitePoint 私人消息实用程序](https://www.sitepoint.com/forums/showthread.php?t=693031)。你只有两周的时间来完成它，所以开始吧。

我们八月份推出的最后一个专题是我们的[摄影教程系列](https://www.sitepoint.com/forums/showthread.php?t=691567)。如果编程和人不是你的东西，那么成为开发社区的一部分可能不是你最好的地方，除非你想学习如何为网站拍摄好照片！如果是这样的话，那就去[摄影论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=302)看看入门教程吧。这个系列将延续我们最新的书*网络摄影*，由一位热心的论坛导师肖恩(TheDead)主持。

本周博客中的对话是关于[在……你猜对了……](https://www.sitepoint.com/forums/showthread.php?t=691970)[WordPress 论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=300)的免费 WordPress 博客中安装插件。

我今天已经说得够多了。以后再说。
