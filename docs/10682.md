# 使用 Polymaps 为您的站点创建免费的定制地图

> 原文:[https://www . site point . com/create-free-customized-maps-for-your-sites-with-poly maps/](https://www.sitepoint.com/create-free-customized-maps-for-your-sites-with-polymaps/)

Polymaps 是一项免费服务，为网页设计者和开发者提供了一个 JavaScript 库，用于在网页浏览器中制作动态的交互式地图。Polymaps 可以与来自 [Bing](http://maps.bing.com) 、 [OpenStreetMap](http://openstreetmap.org) 、 [CloudMade](http://cloudmade.com/products/web-maps-lite) 等等的制图结合使用。这个网站是由雄蕊设计公司和地理数据公司创建的。

Polymaps 使用可缩放矢量图形(SVG)在现有地图上显示信息。因为数据可以在不同的比例范围内加载，所以信息可以在许多级别上显示，从国家级别到各个街道。设计人员和开发人员可以使用级联样式表来设计数据的样式。

用户可以在地图上设置哪个方向是北，放大以查看街道表面的质量，并将数据分组。静态地图图像可以覆盖数据集和使用矢量数据的彩色演示。

为了在地图图像上发布信息，Polymaps 使用[球形墨卡托拼贴格式](http://wiki.openstreetmap.org/wiki/QuadTiles)。什么？引用开放街道地图 wiki 的一句话，“四分块是一种地理数据存储/索引策略。这个想法是存储一个地理数据库，这样就可以快速检索特定位置的数据，通过按位置划分数据，将世界划分为多个小块。”
