# full code press 2010:24 小时内三个惊人的网站

> 原文:[https://www . site point . com/full code press-2010-24 小时内三大惊人网站/](https://www.sitepoint.com/fullcodepress-2010-three-amazing-websites-in-24-hours/)

第三届 FullCodePress 正在新西兰惠灵顿举行。今年的比赛更上一层楼，不是两个而是三个团队竞争，每个团队在 24 小时内为不同的非营利组织建立一个网站。在第六个小时的标记中，团队在讨论、计划、框架中得到了很好的和真正的安顿，编码设计的第一个提示出现在现场网站上，你可以在整个比赛的任何阶段访问。

![The FullCodePress trophy, donated by SitePoint](../Images/6d494aa2e625b0a43e3588f95d3ee304.png)
**The coveted FullCodePress trophy,
donated by SitePoint**

与前几年一样，SitePoint 是赞助者之一。SitePoint 的社区经理 Sarah Taylor 和你的 truly 一起出现在惠灵顿市政厅，会场中的兴奋几乎是有形的——有这么多的人才在这些房间里一起工作，所以对这三个团队将能够制作出什么样的网站有很大的期望。这是每个团队的简要介绍。

## 澳大利亚队

*   **团队名称:** Codaroos
*   成员:凯莉·利金斯、詹姆斯·曼斯菲尔德、拉克伦·唐纳德、亚当·席林、比亚尼·沃克、罗宾·戴蒙德
*   **客户:**阿德莱德的狮听狗
*   **网站网址:**【http://www.fcpaus.netspace.co.nz/ T2】

### 每小时截图

![Codaroos screenshot 1](../Images/e8737c0b3cc3aa21e3626f88624a8e20.png) ![Codaroos screenshot 2](../Images/a485a48ad9fe591ea158fcbdb833b1f2.png) ![Codaroos screenshot 3](../Images/f84d04bffc9d0303b3940e49717617a8.png) ![Codaroos screenshot 4](../Images/cafbaed5365887cc59d7bd4b88450091.png)澳大利亚队决心从往年的错误中吸取教训。这个博客的老读者可能会认出一些团队成员(詹姆斯和拉克伦都是 99designs 的员工，亚当是前员工)。除了团队中的许多人一起工作的明显优势之外，他们还花了大量时间来相互了解，并提前准备好他们的基础设施和方法。参赛队伍很放松，也很自信，但是引入了一个团队昵称，Codaroos，这表明他们是多么认真地对待这场比赛。该团队的开发者拉克伦·唐纳德(Lachlan Donald)做出了一个大胆的举动，建立了一个[公共 GitHub 库](http://codaroos.com)，因此任何对监控网站代码库变更日志感兴趣的人都可以跟随。

### 每小时截图

## 新西兰队

*   **团队名称:** CodeBlacks
*   成员:阿曼达·伍德、露露·帕丘、马修·布坎南、迈克·哈丁、山姆·米妮、哈登·格林
*   **客户:**华特兰加塔希信托基金(帮助问题和弱势青年)
*   **网站网址:**【http://www.fcpnzl.netspace.co.nz/ T2】

### 每小时截图

CodeBlacks 在今年的比赛中输得最多。在连续赢得英超冠军后，这些新西兰球员非常清楚国家对他们的期望。在三个团队中，CodeBlacks 使用的组织材料最少，如便利贴、屠宰纸和其他工具。他们似乎也是三个团队中最安静的，在大多数情况下，当我把头伸进门里的时候，他们都在沉默地工作。也就是说，该团队确实有主场优势，而且众所周知，他们听取了去年获胜的 CodeBlacks 之一，才华横溢的 Darren Wood 的建议(阅读 [SitePoint 去年对 Darren 的采访](https://www.sitepoint.com/fullcodepress-darren-wood-interview/))。国家媒体的报道确保了即使是新西兰的非极客居民也很清楚该团队的目标是创造 FullCodePress 历史上的第一个三连冠。谁会是胜利者？随着网站的建立，你可以自己决定。
