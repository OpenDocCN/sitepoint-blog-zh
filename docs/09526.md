# 10 个随机 jQuery 插件

> 原文:[https://www.sitepoint.com/10-random-jquery-plugins/](https://www.sitepoint.com/10-random-jquery-plugins/)

jQuery – one of the most advanced JavaScript library these days and used by several programs all around the globe to create awesome [effects](http://www.jquery4u.com/plugins/10-amazing-image-effects-jquery/) and [animations](http://www.jquery4u.com/plugins/15-jquery-designing-extended/). We made a random list of plugins that you would like to use. Have fun!

## 1.手风琴画廊

Aga 是一个简单易用且完全可定制的手风琴插件。它可以水平(左、右)和垂直(上、下)使用。
![Aga-Accordion-Gallery.jpg](../Images/dc76ea2bd5baf2891b799c69f0ef4acf.png)
[来源](http://plugins.jquery.com/project/Aga)

## 2.鹦鹉螺

Nautilus 获取一个 DIV 标签(也可以是任何其他标签)中的所有图像，并根据类和设置，生成一个完全可定制的交互式动画，允许通过影线看到移动的背景。
![Nautilus.jpg](../Images/3c093682d7f344f105a0801a9b53411c.png)
[来源](http://plugins.jquery.com/project/Nautilus)

## 3.神奇的 jQuery 插件

一个易于使用、灵活的&网页布局管理优化插件&用户交互管理。它兼容 ie6 和 iPad、iPhone 等移动浏览器。
![Magic-jQuery-Plugin.jpg](../Images/c746272e7a9258d3b87af33b3023098d.png)
[来源](http://plugins.jquery.com/project/Magic)

## 4.jQuery 终端仿真器

一个为你的应用程序创建命令行解释器的插件。它可以在用户输入命令时自动调用 JSON-RPC 服务，或者您可以提供自己的函数来解析用户命令。
![jQuery-Terminal-Emulator.jpg](../Images/57d7f823f21d365f43092235496c993a.png)
[来源](http://plugins.jquery.com/project/Terminal)

## 5.TinySlider

一个简单的 jQuery 滑块，在选择器中从右到左移动图片。
![TinySlider.jpg](../Images/6b39bfd38d3b24dd9ff2b24eb091fc99.png)
[来源](http://plugins.jquery.com/project/tinyslider)

## 6.OIPlayer

它有一个后备机制，允许以通用的方式使用三个玩家。它筛选 media 标签中的源以找到合适的播放器，用自己替换 media 标签并启用一些通用控件。
![OIPlayer.jpg](../Images/f7799ffb59152d02ba3e905ae69e1456.png)
[来源](http://plugins.jquery.com/project/oiplayer)

## 7.3D 水车图像转盘

这将获取任意数量的图像列表，并以垂直或水平的方式旋转它们，级联成一个漂亮的模拟 3D 效果。
![3D-Waterwheel-Image-Carousel1.jpg](../Images/5162da93007633caf3dd3cca4237b1a0.png)
[来源](http://plugins.jquery.com/project/waterwheelCarousel)

## 8.zclip

使用流行的 Zero 剪贴板库构建的轻量级 jQuery“复制到剪贴板”插件。此插件使用一个不可见的 Adobe Flash 电影，完全兼容 Flash Player 10 及以下版本。
![zclip.jpg](../Images/2f7389bc05abc0b94ae669c212883754.png)
[来源](http://www.steamdev.com/zclip/)

## 9.可扩展文本

在你的页面上添加一个可扩展的文本块，让它不那么杂乱。JavaScript 插件会精确地逐字剪切文本。
![Expandable-Text.jpg](../Images/3376d78cf94cb7066d7c442582f3f2ea.png)
[来源](http://plugins.jquery.com/project/expandable_text)

## 10.jQueryFC

有了这个 Jquery Ajax FusionCharts 插件，在您的网站和 web 应用程序中使用图表变得前所未有的简单。
![jQueryFC.jpg](../Images/095378a4a4329e9dac0e229ae928f840.png)
[来源](http://plugins.jquery.com/project/JqueryFC)
