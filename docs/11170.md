# 微软揭秘 IE9

> 原文：<https://www.sitepoint.com/microsoft-reveal-ie9/>

微软通常会在他们的 [MIX 大会](http://live.visitmix.com/)上发布一个盛大的公告。去年，他们[推出了 IE8](https://www.sitepoint.com/ie8-released-today/)T5(它真的已经存在 12 个月了吗？)今年，他们公布了 IE9 的新功能，并发布了一个[可下载的演示](http://ie.microsoft.com/testdrive/)(仅限 Windows 7/Vista SP2)。

微软[在 2009 年 11 月](https://www.sitepoint.com/microsoft-ie9-development/)确认了 IE9 的开发，但是细节很少。我们现在知道更多…

**HTML5**
微软对 HTML5 采取了[观望的态度](https://www.sitepoint.com/html5-microsoft-internet-explorer/)。这可能有很好的商业意义，但如果没有微软的支持，HTML5 将不会成功。好消息是 HTML5 将在 IE9 中实现——至少在某种程度上。重要的是，`video`和`audio`标签得到了支持，微软已经在浏览器中展示了以 720p 运行的 H.264 视频。

有趣的是，IE9 还将支持用 mime 类型的 application/xhtml+xml 解析的 XHTML 文档。是时候了。

微软对边框半径的争论最为激烈，但 IE9 也将支持 CSS 不透明度、rgba 颜色模型和进一步的选择器定义。这很好，但这并不新鲜:竞争浏览器已经提供这些功能很多年了。

**SVG**
IE9 将最终支持 SVG — [可伸缩矢量图形](https://www.sitepoint.com/9-reasons-why-svgs-are-important-for-the-web/)。他们甚至制作了一个很棒的小行星版本来展示这种格式的好处。我还是不明白他们为什么等了这么久？

**更快的 JavaScript**
在所有主流浏览器中，Internet Explorer 的 JavaScript 处理速度最慢，但其名为*“Chakra”*的新引擎将使用独立的 CPU 内核在后台编译代码。最初的结果看起来令人印象深刻，尽管微软如果想赶上 Chrome、Safari 和 Opera 的话还有很多工作要做。

IE9 还将利用 Windows DirectX 来提高浏览器的渲染速度。

标准支持
IE 团队重申了他们对浏览器标准的承诺。IE9 目前在 [ACID3 测试](http://acid3.acidtests.org/)中的分数为 55/100，但它将会提高。这比 IE8 好得多，但让我们希望他们能达到竞争浏览器提供的完美分数。

开发者工具
没有公布具体的细节，但是开发者工具已经得到了改进，将会有一个新的 HTTP Inspector(网络标签)来捕获请求和响应。如果你今天想使用，Firebug、 [DragonFly](https://www.sitepoint.com/opera-dragonfly-open-source/) 和 [webkit inspector](https://www.sitepoint.com/safari-4-first-look/) 已经提供了这个工具。

**我们不知道的……**
微软在两个重要问题上保持沉默:

1.  发布日期。IE 已经确定了 2 年的时间表，所以我怀疑 IE9 不会在 12 个月内推出。这是一个相当长的等待时间，尤其是如果你渴望编写 HTML5 驱动的应用程序。
2.  操作系统支持。具体来说，IE9 会在 Windows XP 上运行吗？

IE9 平台预览版只能在 Windows 7 和 SP2 Vista 上运行。[常见问题解答](http://ie.microsoft.com/testdrive/info/FrequentlyAskedQuestions/Default.html)指出:

> Internet Explorer 9 的 GPU 驱动的图形利用了 Windows 7 中可用的新技术，并仅支持 Windows Vista。这些技术依赖于首先在 Windows Vista 中引入的显示驱动程序模型的改进。
> 
> 现在谈论 IE8 测试版的功能(兼容 IE8 和 XP)还为时过早。

发布日期越遥远，IE9 无法在旧版本 Windows 上运行的可能性就越大。尽管 Windows 7 销售强劲，但绝大多数人仍在使用 XP——如果他们不能升级，我们将在未来许多年里仍被旧版本的浏览器所束缚。

你会被诱惑转用 IE9 吗？新功能足以与竞争对手匹敌吗？是不是太少了，太晚了？浏览器是否越来越不重要？微软应该放弃浏览器开发吗？
