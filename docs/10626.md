# 今天是 IE9 测试版下载日

> 原文：<https://www.sitepoint.com/ie9-beta-download/>

我真的很兴奋。我不记得在 [Chrome 6](https://www.sitepoint.com/google-chrome-6/) 、 [Firefox 3.6](https://www.sitepoint.com/firefox-36-released/) 或 [Opera 10.60](https://www.sitepoint.com/opera-10-60-released/) 发布时有过这种感觉 IE9 只是一个测试版。我异常乐观有几个原因:

1.  我们通常在 IE 更新之间等待 2 年。
2.  IE 开发团队听取了我们的要求，IE9 是第一个支持 HTML5、CSS3、SVG、canvas 和其他几种技术的版本。
3.  与其他厂商不同，微软对 IE9 的新界面一直特别保密。一张[截图被泄露](https://www.sitepoint.com/ie9-screenshot-microsoft-leak/)，不过今天我们就知道是不是真的了。

在你匆忙下载浏览器之前，请注意:

*   IE9**不**可用于 Windows XP。我希望微软纠正这一点，但我怀疑他们是否会支持他们老化的操作系统…即使绝大多数人继续使用 XP。
*   在 Windows Vista 或 7 上安装 IE9 将会删除 IE7 或 IE8 的现有安装。由于这些浏览器将在未来许多年内保持流行，确保您在另一台 PC 或虚拟机上可以使用它们进行 web 开发测试。或者，将会有 IE9 平台预览版的另一个版本——它将与 IE 的现有版本一起运行，但它的局限性要大得多。
