# Windows 10 升级通知一瞥

> 原文:[https://www . site point . com/glimpse-windows-10-upgrade-notification/](https://www.sitepoint.com/glimpse-windows-10-upgrade-notification/)

本文是微软网站开发系列的一部分。感谢您对使 SitePoint 成为可能的合作伙伴的支持。

最近有很多关于 Windows 7、8 和 8.1 用户的 Windows 10 升级通知的报道。我想发表一篇关于这次经历的文章。无论你的 dev box 是 Mac、PC 还是 Linux，你都可以预期，你的许多用户将会尝试 Windows 10 和微软 Edge(默认浏览器)。所以这是一个很好的理由[做一个测试通过](http://dev.modern.ie/tools/?utm_source=SitePoint&utm_medium=article49&utm_campaign=SitePoint)。幸运的是，Edge 更像 Chrome 和 Firefox [这样的浏览器，而不是过去版本的 IE](http://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/?WT.mc_id=13425-DEV-sitepoint-article49) 。让我们来看看:

请注意，**只是为了预订 7 月 29 日提供的免费升级**；这基本上就是告诉微软你想要。**它不会马上安装！**如果你现在想用 Win 10，去 insider.windows.com[下载 ISO。](http://insider.windows.com/?WT.mc_id=13425-DEV-sitepoint-article49)

首先，在通知中心，你会看到一个小小的窗口图标:

![Windows 10 upgrade icon](../Images/9c6d2d7aee9aca96335fd7fe1d4048fd.png)

点击它会弹出一个对话框，让你知道你可以预订免费的 Windows 10 升级，并让你滚动查看好处(即:营销发言)。你不必滚动。相反，您可以点击“预订您的免费升级”按钮，直接预订您的 Win 10 升级:

![Upgrade mechanism](../Images/23cdffa18b58197dbeaed27b8dc5f0e4.png)

![Start Menu](../Images/46aa7cb0b1c2c94163da88c5a536f78d.png)

![Multitasking](../Images/4821410da47369badeeef852710a590a.png)

![Built-in apps](../Images/e0e43183136dcc60c6a4ad7d0cb3985f.png)

![Windows Store](../Images/932d27b95829a37883f15d3be926223f.png)

![Reserve free upgrade](../Images/291df9137064682abc9f05e1a7b5b419.png)

您还有一个小的滑出式菜单，基本上为您提供了所有相同的选项，您可以滚动浏览，还可以检查您的 PC 的兼容性:

![PC compatibility options](../Images/67cf871358f8cfc86fb71f7579ea49c0.png)

一旦你点击了“预订你的免费升级”按钮，你基本上就完成了，你的机器被标记为接收更新。请记住，当它准备好安装时，您会收到通知:

![Email input](../Images/d7623a7b130fd0a20ac083e30bf14130.png)

如果您想在升级准备就绪时收到电子邮件通知，只需输入并提交即可，这将带您进入以下屏幕:

![Upgrade reserved](../Images/4266b59f1b2d9c3e6f0ed3f2fe133faf.png)

就是这样。这是该对话框中常见问题解答的[链接。](http://www.microsoft.com/en-us/windows/windows-10-faq/?WT.mc_id=13425-DEV-sitepoint-article49)

没有唠叨和疯狂的 UX。简单。
