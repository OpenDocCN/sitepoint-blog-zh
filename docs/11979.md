# 同学们注意了！教学作为一份兼职如何能让你(和其他人)受益

> 原文:[https://www.sitepoint.com/teaching-as-a-side-gig/](https://www.sitepoint.com/teaching-as-a-side-gig/)

你有没有考虑过在业余时间教书，或者更准确地说，在你的主要工作之后安排时间教书？教授公共和私人课程、网络研讨会、辅导课和其他帮助他人学习你所知道的知识的课程形式有很多好处。

**为什么教书？**

除了可能在晚上和周末教书赚些外快这一明显的好处之外，教书还有很多让你和他人受益的方式。以下是一些例子:

*   经验——虽然教学给你一个帮助他人的机会，但是备课和完成详细的指导也能帮助你获得更多的经验。你可能会学到新的更好的做事方法，发现一个常见问题的新解决方案，并从你的学生那里学到意想不到的技巧。
*   网络——教学帮助你认识新的人，他们可能是潜在的客户、合作的同事、合格的分包商或正在发展的推荐网络。
*   回馈——花时间去帮助别人可以让你对自己和自己的能力有一个新的满足感。让我们面对现实吧，贡献你的时间，不管有没有报酬，感觉都很好。

**教什么？**

最初，你可以考虑教授你每天在工作中使用的技能、软件和技术。这往往是你最了解的主题，也是最适合教给别人的。

教你每天都在做的东西的一个危险是你会厌倦它，并有可能筋疲力尽。无论你多么热爱你的工作，任何事情做得太多都不是好事。但是适度的话，这是开始教学的一个很好的方式。

另一种参与教学的方式是专注于你热爱的业余爱好或其他业余消遣，甚至是你曾经做得很好但有一段时间没有时间做的事情。关键是选择你热爱的东西，并且有足够的知识教给别人。

**去哪里找机会？**

你可以通过同样教书的朋友和亲戚，当地的社区大学，甚至通过专业网络找到教学机会。思考超越正式的课堂环境；教学机会可以是网上研讨会、一次性面授课程，甚至是免费的电话会议。

找不到适合自己的机会？创造你自己的！您可以在培训课程中使用[在线会议工具](https://www.sitepoint.com/online-meeting-tools/)，并通过您的网站、客户、业务联系人和社交媒体进行推广。你也可以创建在线教程系列，甚至是教学博客。如果你喜欢面对面的教学，可以考虑联系当地的学校、教堂或餐馆，看看他们是否会为你提供举办面对面培训的空间。

可能性是无穷无尽的，好处也是无穷无尽的。

你教书吗？你用什么格式，你教什么？
