# 使用 Twitter 扼杀你的生意的 5 种方法

> 原文：<https://www.sitepoint.com/how-not-to-use-twitter/>

有很多人对 Twitter 应该和不应该如何使用持有非常强烈的观点，尤其是从商业角度。我认为 Twitter 是一个灵活的媒介，你可以塑造、扭曲和弯曲它来适应你的个人需求；它就在那里，你想怎么用就怎么用。

但是，话虽如此，我已经在 Twitter 上看到了一些产生可怕结果的活动，我真的无法想象这些活动在任何情况下都是合适的，除非你准备好接受在线抨击(这肯定会发生)。所以，这里是我列举的有可能毁掉你在线商业声誉的 Twitter 行为……并且在这个过程中让你看起来有点傻。

## 1.将您的姓名、位置、头像和/或简历留空

你可能会认为我们这些使用 Twitter 与他人联系的人(几乎每个人都是这样，不是吗？)，需要时间来补充一些关于我们是谁和我们做什么的基础知识。我见过很多甚至不提供名字的推特账号。更糟的是，有些人没有头像。理想情况下，你的头像应该是你的真实照片，但是至少，你需要一些东西在上面(一个标志，商业名称，等等)。)来替换默认的无头像图形。

## 2.从不发微博

当我看到有成千上万的关注者，却只有少量的推文发布的 Twitter 账户时，我总是想知道这是怎么回事。当然，这是许多名人推特账户的标准，但这有什么意义呢？所有这些告诉我的是，你正在玩一个数字游戏，并且很可能不愿意或没有兴趣去了解你的任何追随者。

使用 Twitter 的最大好处之一是我们可以用它来建立关系。如果你不发微博，不与任何人交谈，你就享受不到任何人际交往的好处。

## 3.疯狂使用附属链接

如果你一直发布嵌入了代销商跟踪代码的链接，主要目的是获得点击，你将开始建立一个负面的声誉。没有人想要被利用的感觉，如果你不把它作为一个创收活动来发布任何东西，你就会让人们失去兴趣。

## 4.自我宣传到极致

我们都在推特上放链接；这是分享信息和引发对话的好方法。但是看看你的链接历史。如果你的大部分链接都是自我推销的(例如，它们直接进入你的网站，将人们引向销售页面，或者只关注你自己的成就)，你可能没有抓住要点。

当然，人们想知道你在做什么，你有什么可以购买的东西，但这不能是你 Twitter 角色的唯一方面。不要忘记分享其他人可能会觉得有用的信息，转发，阅读和评论你的追随者发布的内容。当你得到评论和转发时，请承认它们并说谢谢！

## 5.攻击、辱骂或说人坏话

Twitter 是偶尔发泄的好地方。它不仅能让你快速、无痛苦地说出心中的话，还能让你分享一点你个人的一面。根据我的经验，这是网上关系可持续和真实的原因。我们都是人，都会犯错，都会有不如意的时候。分享你人性的一面会让你变得更加平易近人，更加可爱，也有助于他人与你建立联系。

但是，在使用 Twitter 攻击、辱骂或诋毁他人之前，请三思。公共论坛从来都不是对另一个人表达愤怒的地方，不管这是否合理。这种活动只会让你看起来很糟糕…而且对我们这些不得不目睹的人来说很尴尬。

## 还有一个提示…

不要忘记使用 Twitter 搜索或其他搜索工具来跟踪您的姓名、企业名称和其他目标关键词。这不仅能帮助你监控你的网上声誉，还能帮助你找到你想联系的人。
