# PHP–现在可以在你的墙上使用了

> 原文:[https://www.sitepoint.com/php-now-available-for-your-wall/](https://www.sitepoint.com/php-now-available-for-your-wall/)

早在二月份，当我在博客上写了关于 PHP 可用的[快速参考时，SitePoint 的 PHP 快速参考海报只是我们集体眼中的一个亮点。](https://www.sitepoint.com/blog/)

所以我很高兴现在可以说，那张海报已经成为现实，可以从我们的网上商店买到。

![](../Images/fa2b9ba308c7be3129a13742b2a50110.png)

当我深入研究 PHP 代码时，海报上有很多我需要经常查找的信息。不管我正在编写的代码有多高级，我发现自己一遍又一遍地查找的都是一些小事情。例如，当我最近在做一个日程安排应用程序时，我发现自己一直在引用我手边的[扩展 CHM](http://www.php.net/docs-echm.php) PHP 手册中的[日期](http://www.php.net/date)功能页面。如果我有一个多显示器设置，我可以在我的第二台显示器上打开该页面，但我希望海报已经完成！

果然，海报现在完成了，日期格式信息就在那里:

![](../Images/49b2eaead06a97d211988749d2a49e90.png)

在巨大的光滑 A2 (16.5 英寸 x 23 英寸)海报上阅读很好也很容易。

海报还提供了数组函数、字符串函数、mysqli、错误报告、文件函数、simpleXML 和正则表达式语法参考的语法、描述和几个例子。

海报本身售价 9.95 美元，但它也可以作为 SitePoint 的新书 [No 废话 XML Web Development With PHP](https://www.sitepoint.com/blog/) 的免费赠品，这本书是我在过去几天里一直在[努力推广的](https://www.sitepoint.com/forums/showthread.php?t=287423)。这本书肯定会让那些想了解 PHP 5.0 的 XML 处理特性的人感兴趣。关于那本书的完整信息可以在那边的[找到。](https://www.sitepoint.com/blog/)

现在我只需要找一面墙来贴海报…
