# MySQL 5 接近金牌

> 原文：<https://www.sitepoint.com/mysql-5-closing-in-on-gold/>

MySQL 领域每年都会在 2005 年的年度用户会议上获得最大的推动力，该会议于上周刚刚结束。

为了取悦数据库爱好者，MySQL AB 的联合创始人 David Axmark 和 Monty Widenius 宣布，备受期待的第 5 版是从开发角度完成的，目前完全处于调试模式。

这个版本通过存储过程、触发器、视图等等，将 MySQL 的特性集提升到了与它的私有和开源同类产品相同的竞争水平。然而，我们都把这条新闻翻了个遍。现在剩下的事情就是让使用 MySQL 的开发者和管理员忙于发现和报告 bug。

MySQL 欢迎每一点输入。比起那些有特性和/或代码建议的开发人员，他们更希望在获得金牌之前尽可能地消除任何物理缺陷。实际上，随着 MySQL 程序员清理 bug 报告，这经常导致令人垂涎的特性。

要参与其中，请前往 MySQL 的[版本 5 下载专区](http://dev.mysql.com/downloads/mysql/5.0.html)获取最新版本。预先警告，任何不在生产或稳定版本中的东西都绝对不要安装在生产服务器上，直到你彻底测试过。

下一步是花一些时间查看 MySQL 的 [Bug 专区](http://dev.mysql.com/downloads/mysql/5.0.html)，这里提供了过程、现有的报告和 Bug 提交功能。

这让你有机会熟悉 MySQL 5 的新特性和功能，调试你自己代码的潜在转换，并参与确保 MySQL 5 尽可能少的出现错误——这是任何好的开源项目的目标。
