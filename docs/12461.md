# 网络审查正在兴起吗？

> 原文：<https://www.sitepoint.com/is-online-censorship-on-the-rise/>

在中国、乌兹别克斯坦、古巴和伊朗等暴虐的政府试图控制信息流动的国家，互联网审查无疑仍然存在。保护记者委员会(CPJ)发现，截至本月初，全世界有 [125 名记者被关押在](http://www.cpj.org/imprisoned/cpjs-2008-census-online-journalists-now-jailed-mor.php)。这实际上比 2007 年下降了 2 个百分点，但是考虑到 45%被监禁的记者是博客写手或者只在网上工作，很明显，互联网作为信息来源的崛起导致了审查制度的平行上升。

CPJ 执行董事乔尔·西蒙说:“在线新闻已经改变了媒体的面貌和我们相互交流的方式。”。“但新一代网络记者的力量和影响力引起了世界各地专制政府的注意，他们加快了反击。”

这不足为奇。或许令人惊讶的是，西方国家开始出现互联网审查的案例。

我们最近报道了澳大利亚的审查计划，根据该计划，政府将强制互联网服务提供商过滤“不良内容”并阻止用户访问。该计划遭到了大量的公众批评，人们发起了多次请愿活动试图阻止该计划。

同样在本周，澳大利亚最高法院的一名法官裁定，对未成年人性行为的漫画描述与儿童色情制品相同。法官认定，反儿童色情法律的出台不仅是为了制止对真实儿童的虐待，也是为了遏制诸如*漫画*等材料的创作和传播，这些材料可能“刺激对涉及虐待儿童的材料的需求”

同样，在英国，非盈利的非政府组织互联网观察基金会(IWF)本周在他们的黑名单上增加了一个维基百科页面，因为其中有一张未成年少女裸体的图片。大多数英国互联网服务提供商使用 IWF 的黑名单来维护用户的体面标准。问题中的[页面](http://en.wikipedia.org/wiki/Virgin_Killer)(可能是 NSFW)是 1976 年蝎子乐队的专辑《处女黑仔》，包括专辑封面的一张照片，照片上是一个青春期前的裸体。IWF 认定该图片是“潜在的 18 岁以下儿童的非法不雅图片”，并将该页面添加到他们的黑名单中。

IWF 最终做出了让步，将该页面从黑名单中移除，尽管暗示如果该图片在英国被托管，可能仍会被屏蔽。

有些人可能会说，保护儿童是最重要的，如果这是以一些“变态”不能色迷迷地看小女孩的画或(主观性警告)艺术裸照为代价的，那就这样吧。毕竟，这些案件不是记者或博客因为公开反对政府而受到审查，而是对明显“令人作呕”的内容进行审查。

然而，有时站出来反对审查可能意味着站出来支持你不同意的言论，甚至是完全应该受到谴责的言论。滑坡论在这里绝对适用。

正如尼尔·盖曼上周在他的博客上提醒的那样，“如果你不为你不喜欢的东西挺身而出，当他们为你喜欢的东西而来时，你已经输了。”这篇文章是任何关心为什么言论自由是一项重要的斗争权利的人的必读文章

《个人电脑世界》的 JR Raphael 写道, [IWF 案尤其令人不安，因为它说明了私人审查，即使只持续了几天。为什么一个私人组织应该被允许维护“自己的权利，在法律之外，来决定你应该或者不应该被允许看什么法律上可接受的内容？”](http://www.pcworld.com/article/155156/wikipedia_censorship_sparks_free_speech_debate.html)

此外，这些案件的共同点是，它们对言论自由的限制提出了挑战。一幅画真的应该被视为非法吗？一张裸体孩子的照片能成为艺术吗？乔克·斯特奇斯或莎莱·曼恩等著名摄影师的有争议的作品是色情作品还是艺术作品？亚马逊是否应该因为收录了《蝎子》专辑或莎莱·曼恩书籍的封面图片而被列入黑名单？任何私人组织应该对什么是体面的和你被允许看什么有发言权吗？政府应该吗？

“这些都是棘手的问题，可能没有任何绝对正确的答案。我肯定没有，”拉斐尔写道。“但毫无疑问，这里正在酝酿一场远比这一案件更大的重要辩论——每个使用互联网的人都有理由对其结果进行投资。”
