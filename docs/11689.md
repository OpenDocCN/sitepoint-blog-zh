# 界限与障碍:3 种在不损失工作的情况下设定界限的方法

> 原文：<https://www.sitepoint.com/boundaries-vs-barriers-3-ways-to-set-limits-without-losing-work/>

不久前，我写了一篇关于[与客户](https://www.sitepoint.com/how-to-set-boundaries/)设定界限的文章，得到了一些很好的反馈。但是，当我进一步思考设定界限的过程时，我意识到，在以支持你的业务的方式设定界限和以让人们反感的方式设定界限之间，可能有一条非常细的线。

虽然设定界限来定义你对客户的期望和限制是绝对必要的，但我认为同样重要的是不要筑起一堵会伤害你们关系的墙。所以这里有一些方法来防止你的界限导致工作的丢失。

## 坚定但不固执

一个肯定能制造障碍而不是界限的方法是，以一种可能被认为是对抗性的、冷淡的或冒犯性的方式向你的客户告知你的政策。你完全有权利在你的业务中设置任何你想要的限制，但如果你像一个暴君一样——制定你的条款并走开——你将不会因此交到任何朋友或赢得任何客户。

很多时候，都是交付的问题。如果你以对话的方式提出你的政策，然后给客户一个机会要求澄清，你可以让你的条款更容易接受。

## 提供一些背景

我不喜欢解释我在业务中做出决策的原因，但有时分享一些背景信息有助于客户理解你的情况。这一切都与你希望别人如何对待你的客户有关。

比方说，你周三不工作，因为你每周都要去看望你妈妈并和她一起度过这一天。你可以直截了当地告诉客户，“不要指望在周三收到我的消息，因为我不在办公室，没有回复电子邮件。”但是如果你想一想，在一周中间休息一天并不常见，而且没有事先做好安排，这可能会对正在进行的项目造成一些小的干扰。

当然，作为一个自由职业者，你有权利在任何你想工作的时候工作，但是给客户一点背景信息，告诉他们你为什么这样做，他们会更容易接受。

## 不要把它私人化

当你设置界限的时候，一个主要的危险信号是你发现自己在响应个别客户的请求而改变它们。虽然你可能会发现你的限制随着时间的推移而演变，但你最不希望的事情就是在与客户设定限制时陷入权力斗争，试图占上风，而不是忠实于你的全面政策。这可能导致一段不可能维持的关系。

如果你承诺尊重和公平对待你的所有客户，不仅维护一套标准政策变得更容易，而且你可以避免让单个客户感到受到攻击和虐待。

这和在你生活的其他领域设限真的没什么区别。你可以修改你的方法，这样你就不只是说“不”，而是稍微修饰一下，以减轻拒绝的程度。
