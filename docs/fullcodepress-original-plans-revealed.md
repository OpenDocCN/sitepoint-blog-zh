# FullCodePress 组织者的卑鄙计划被揭露

> 原文：<https://www.sitepoint.com/fullcodepress-original-plans-revealed/>

FullCodePress 活动即将进入第 12 个小时，这意味着两个团队即将跨越一半的时间。

SitePoint 的 Matthew Magain 与 FullCodePress 的组织者之一 Russ Weakley 进行了几分钟的交谈，谈论了他举办这项活动的最初动机，他认为哪个团队在比赛中具有优势，以及他为使一天内建立一个网站的比赛更具挑战性而准备的其他一些(有问题的)想法。
