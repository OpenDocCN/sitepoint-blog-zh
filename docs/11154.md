# 在 Flex 中构建 Flickr Commons 浏览器

> 原文:[https://www . site point . com/building-a-Flickr-commons-browser-in-flex/](https://www.sitepoint.com/building-a-flickr-commons-browser-in-flex/)

![Adobe Flex](../Images/0898cf5a426f673466c8cbf3886ba562.png)

在今天的教程中， **[在 Flex 中构建 Flickr Commons 浏览器](https://www.sitepoint.com/article/flickr-browser-flex)** ，我们友好的 Flex 爱好者 Toby Tremayne 展示了 Flex 4 的许多新功能，展示了组件开发和皮肤是多么容易。

通过对 Flex 的一点熟悉和对 ActionScript 的大致掌握，您将能够让 Flex 与 Flickr 的 API 对话、下载和显示图像，并在阅读完本文时完成一些基本的换肤工作。您还可以下载代码，并将其作为 Flash Builder 项目归档文件导入。

在学习了如何检索和显示图像，并体验了 Flex 4 中改进的皮肤功能后，您将能够浏览来自 Flickr 的 creative commons 许可的照片。所以你可以在你自己的博客文章中使用它们。

我们还有另一个[小测验](https://www.sitepoint.com/quiz/flex4/flickr-browser-flex)供您试用，所以一旦您阅读了[文章](https://www.sitepoint.com/article/flickr-browser-flex)，一定要试一试！
