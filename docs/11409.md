# 如何让一家公司相信社交媒体是必由之路

> 原文：<https://www.sitepoint.com/how-to-convince-a-company-that-social-media-is-the-way-to-go/>

![stop](img/b14c216136f5ce2486d912ec8399e18c.png)我之前的帖子列出了[一些企业不愿涉足社交媒体的四个常见原因](https://www.sitepoint.com/why-companies-are-scared-of-social-media)。为了说服一家公司——无论是你为之工作的公司还是你与之合作的客户——发起社交媒体活动确实值得他们花时间，你需要帮助公司的决策者克服这些障碍。

## 障碍 1:缺乏理解

为了帮助企业主理解社交媒体的价值，他们必须了解网络的价值。对于任何希望最大化其营销潜力的公司来说，商业网站是可选的日子已经一去不复返了，不可否认的是，社交媒体正在沿着同样的道路前进。

因此，第一步是全面了解网络以及社交媒体在制定健康全面的营销策略中所扮演的角色。做到这一点的一个有效方法是概述积极的社交媒体可以带来的好处，包括:

*   扩大覆盖范围和新的可见性
*   与客户的接触和互动
*   提高品牌认知度
*   持续的市场研究
*   增强的客户服务

在充分解释了好处之后，缓解围绕社交媒体的一些不知所措和缺乏理解的一个很好的方法是揭开各种社交媒体渠道的神秘面纱。解释这些网站，它们是如何使用的，竞争对手是如何使用它们的，特别是它们能为企业提供什么。

## 障碍#2:对品牌管理的担忧

有人担心进入社交媒体意味着你的公司会受到负面宣传，这种担心有点不准确。事实是，如果一个不开心的客户决定在社交媒体上发泄他们的不满，无论公司有多活跃都没有关系；他们仍然会受到公众的影响。然而，如果企业已经在网上建立了业务，那么缓解这种情况并采用一些快速损害控制就变得容易多了。

很多时候，这种围绕品牌管理的担忧可以通过向企业主介绍一些工具(许多是免费的)来轻松缓解，他们可以使用这些工具来[监控他们的在线声誉](https://www.sitepoint.com/tools-manage-online-reputation/)，包括他们可以提供的一些有价值的数据。

## 障碍 3:缺乏时间和人员

这可能是一个棘手的障碍，除非你能够帮助企业主设计一个启动和维持社交媒体战略的计划，甚至可能主动提出自己承担责任。

根据个人情况，你也许可以通过提出一个计划来缓解这些担忧，该计划提供了一个在试验基础上启动社交媒体战略的机会。如果你能从试运行中为企业主提供一些快速、相关和令人印象深刻的结果，以及一个正在进行的活动的计划，你可能能够相当快地克服这个障碍。

如果对公司来说可行，你也可以建议雇佣一个社交媒体顾问或营销专家来帮助企业主为他们的企业创建一个有效的社交媒体营销计划。

## 障碍 4:相信这只是一时的流行

社交媒体营销不会很快消失。当竞争对手在新的在线环境中蓬勃发展时，选择忽视信息技术有效性的企业将不可避免地被甩在后面。

了解社交媒体可持续性的一个方法是回顾统计数据。社交媒体统计的一个很好的起点是，“统计显示，社交媒体比你想象的要大得多”。
