# 基于网络的图形应用程序:Picnik

> 原文:[https://www . site point . com/we B- based-graphics-application-picnik/](https://www.sitepoint.com/web-based-graphics-application-picnik/)

如果我养成了说“看看这个”的习惯。这是 Flash，非常 Web 2 .哦！”打我一下，因为我可能不舒服或者被外星人控制了。

尽管如此。[看看这个。这是 Flash，非常 Web 2。哦！](http://www.picnik.com/)。幸运的是，它也相当光滑。

Picnik 是一个基于网络的照片处理应用程序，我认为这是一个高质量 Flash 应用程序开发的很好的例子。该界面易于使用，反应灵敏，并且几乎拥有您想要的所有工具来校正和润色照片以供在线展示，特别是调整大小、旋转、裁剪、颜色调整、锐化和曝光校正。

在“创意工具”标签下还有一堆执行得很好，但可能不太日常有用的效果，包括边框、颜色色调、圆角和柔焦技巧，仅举几例。

现在，如果你正在阅读这篇文章，很可能你已经有一些工具可以很好地处理这些功能，你不会仅仅因为有在线应用程序就转向它。当然，如果你不在你的工作站上，很高兴知道你总是有一个现成的选择。

然而，我认为对我们最有用的地方是客户使用。建立一个 CMS，让它看起来漂亮，并训练他们使用它，通常只是成功的一半。从过去的经验来看，我发现新用户、新 CMS 和大分辨率数码照片通常是让你的布局支离破碎的好方法。试图解释 Photoshop 的复杂性并不容易。

Picnik 可以提供一个图形应用程序:

*   简单、用户友好的界面
*   几乎可以在任何系统上使用，不管是什么操作系统或浏览器
*   无需安装(绕过他们的技术专家)
*   几乎没有许可问题(绕过他们的行政纳粹)

以我的经验来看都是好事。

虽然它目前仍处于测试阶段(真正的 Web 2.0 风格)，但完全版显然只会将一些高级图像工具(液化、棕褐色效果等)限制在付费会员/订户中，而将所有最实用的工具留在免费版本中。

向它背后的团队致敬。干得好。
