# PHP xref–Perl 帮助 PHP

> 原文：<https://www.sitepoint.com/phpxref-perl-helping-php/>

经由[国际 PHP 杂志](http://www.php-mag.net/)(2004 年 2 月，印刷版):[PHP xref](http://phpxref.sourceforge.net/)——一个用于交叉引用 PHP 源代码的 Perl 实用程序，确实非常方便。

基本上，你把它指向某个包含 PHP 源代码的目录，它就会生成一组静态 HTML 页面，允许你浏览项目中的代码。最有用的特性是它构建的类、函数(+方法)和变量的列表。如果您需要知道代码中调用某个特定函数的所有位置，您可以立即找到。同样好的是 PHPXref 似乎知道[phpdoc](http://www.phpdoc.org)markup[edit]——至少知道“文档块”。[/编辑]

如果能有常量的交叉引用并能快速区分全局变量就好了，这两者目前都是不可能的(版本 0.4.1)，但在其他方面，这是一个非常有用的工具。

显然，我在这里抱怨的并非如此(这里有一些合理的后续和这里的，谢天谢地，它没有引发一场“语言战争”)。
