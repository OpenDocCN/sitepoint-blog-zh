# 谷歌诗中的冬天

> 原文：<https://www.sitepoint.com/winter-in-the-google-verse/>

季节性装饰和本地化一直是最近 SitePoint 办公室讨论的焦点。一方面，给网站增加一点季节性氛围会有什么错呢？也许这里有一根冬青枝？那里有颗星星吗？你会把 ***和*** 当成一个怪杰来反对，对吗？

问题是，每当你和一部分观众“交朋友”时，你就会情不自禁地背弃另一部分观众。如果你有一个非常明确的小众受众(即游戏玩家、金属头、徒步旅行者、黑客等)，创造一个“我们对他们”的氛围可能是有用的，但否则你可能需要小心。你为“我们”创造的积极氛围值得你为“他们”创造的小小的疏离感吗？

![](img/86b368b60f5507a044cde821007540eb.png)

经过最近对这个问题的深思熟虑，今天早上当朱利安注意到顽皮的北极熊厚脸皮地在谷歌上准备雪球时，我们很感兴趣。也许有点色彩，但这出现在网站的[Google.com.au](https://www.google.com.au)版本上，意味着他们可以很确定他们的观众穿着短裤、t 恤和防晒霜来找他们。

一眼看去，同样的情况也发生在[谷歌新西兰](https://www.google.co.nz/)、[谷歌南非](https://www.google.co.za/)和[谷歌阿根廷](https://www.google.com.ar/)身上，它们也都在经历着夏季的全面冲击。

我知道天气预报有时可能是一门可疑的科学，但据我所知，整个“夏季/冬季北半球/南半球”的事情已经得到了科学的“赞许”;)

但是，嘿，厚脸皮的北极熊很可爱，我猜。
