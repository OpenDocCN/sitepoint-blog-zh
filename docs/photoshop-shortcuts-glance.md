***全屏模式下进入菜单***

按 SHIFT + F，文件菜单将出现在屏幕顶部。

此外，请注意在全屏模式下，工具面板有一个小的附加物:在面板的右上角有一个小箭头。单击此箭头，还可以访问文件菜单。

***应用上次使用的滤镜***

**要应用最后使用的过滤器:**
按 CTRL + F

**用新设置应用最后使用的滤镜:**
CTRL + ALT + F

***不使用图层面板*** 改变混合模式

选择选框工具，您可以使用以下命令:

*   移动到下一个混合模式:ALT + SHIFT + "+"
*   移动到先前的混合模式:ALT + SHIFT + "-"
*   跳转到溶解:ALT + SHIFT + I
*   跳转到乘法:ALT + SHIFT + M
*   跳转到屏幕:ALT + SHIFT + S
*   跳转到覆盖:ALT + SHIFT + O
*   跳转到柔光:ALT + SHIFT + F
*   跳到强光:ALT + SHIFT + H
*   跳转到颜色减淡:ALT + SHIFT + D
*   跳到颜色加深:ALT + SHIFT + B
*   跳到变暗:ALT + SHIFT + R
*   跳转到变亮:ALT + SHIFT + G
*   跳转到差异:ALT + SHIFT + E
*   跳转到排除:ALT + SHIFT + X
*   跳转到色调:ALT + SHIFT + U
*   跳到饱和:ALT + SHIFT + T
*   跳转到颜色:ALT + SHIFT + C
*   跳到发光度:ALT + SHIFT + Y
*   当然还有最重要的一点…
*   跳转到正常:ALT + SHIFT + N

注意:这适用于任何提供混合模式的工具。这些工具是喷枪，画笔，铅笔，克隆图章，图案图章，历史画笔，渐变，模糊，锐化，涂抹，减淡，加深，海绵和矢量对象工具。

此外，Adobe 有一种讨厌的倾向，就是在每个新版本中改变这些不同的快捷方式。如果这些在 PhotoShop 的新版本中略有不同，请不要感到惊讶。

***在不使用画笔调色板的情况下更改画笔***

这是一个很好的方法，如果你在全屏模式下工作，所有的调色板都是隐藏的。

要将画笔切换到下一个/上一个选项，只需使用“[”或“]”键。

建立一套全面的画笔，并保存它们以供将来参考。

***改变图像外的背景***

1.  选择油漆桶工具(G)。

3.  选择要将背景更改为的颜色。

5.  将光标移出图像边界，然后按 SHIFT +左键单击。

***改变笔刷或图层的不透明度***

怎么做到的？只需使用数字键。例如，1 会给你 10%的不透明度，而 9 会给你 90%的不透明度(0 给你 100%！).

此外，如果您快速键入数字，您可以获得增量，例如 56%。这适用于画笔、橡皮图章，甚至图层。

***关闭工作窗口***

按 CTRL + W

***复制一层***

按住 ALT 并在画布上单击/拖动。

**如果你有一个选择**，那么 CTRL + ALT 并拖动(在该选择内)将复制并浮动内容。

**没有激活的选择**，CTRL + ALT 并拖动会将该层的内容复制到一个新层并拖动它。

***复制图层为新图像***

图层>复制图层>文档:新建

***在不知道如何使用路径的情况下创建裁剪路径***

1.  创建两个图层，一个是简单的形状(例如圆形)，另一个是图片。

3.  将带有图片的图层放在带有圆圈的图层上面。

5.  按住 ALT 键，单击分隔两层的线。当光标变成一个带两个圆圈的向左箭头时，你就知道你做对了。

这个技巧允许你创建一个基于底层对象透明度的裁剪路径。这意味着，如果您有一个边缘羽化的对象，图片也会被羽化。

***新建图层不使用图层面板***

您可以通过多种方式做到这一点:

*   按 CTRL + SHIFT + N 打开“新建图层”对话框。这将允许您在创建层的同时命名层。
*   将当前图层复制到新图层:CTRL + J
*   将当前图层复制到一个新图层，并将该图层命名为:CTRL + ALT + J
*   将当前选择复制到新层:CTRL + J
*   将当前选择复制到一个新层，并将该层命名为:CTRL + ALT

***不用油漆桶填充图层或选区***

有几种方法可以实现这一点:

**通过填充对话框**:按 SHIFT + BACKSPACE。这将打开“填充”对话框。然后，您可以选择几个选项之一，包括前景色、背景色、图案等。

**要用前景色**填充该区域，按 ALT + BACKSPACE。

**仅填充不透明像素:**

*   要仅用前景色填充区域的不透明像素，请按 ALT + SHIFT + BACKSPACE。
*   要仅用背景色填充区域的不透明像素，请按 CTRL + SHIFT + BACKSPACE。

这包括部分透明的区域，这意味着如果对象被羽化，该操作将填充(部分)部分透明的像素。这允许您保持阴影和羽化项目看起来与您最初设计它们的方式相同。

***释放暂存盘上的空间***

每个 PS 用户肯定都经历过暂存盘的问题，所以这里有一个解决方案。

PhotoShop 并不擅长处理磁盘空间，清空暂存盘的方法是在上面保存更多的信息！这听起来很傻，但如果你创建一个非常小的文件，比如 1×1 像素，并将其保存为 PSD，那么这将拉动链并刷新划痕，允许你保存你的大文件。

或者您可以选择编辑>清除>全部或

编辑>清除>剪贴板

按 TAB 键在“可见”和“不可见”之间切换调色板。

***隐藏遮罩层的效果***

按住 SHIFT 键并单击遮罩层图标。

***隐藏行军蚁***

按 CTRL+H(但是别忘了尽快放回去！)

***不使用图层面板*** 与图层交互

有几种方法可以做到这一点。

**不使用图层面板选择图层:**

1.  将鼠标放在要更改的图层上，然后按 CTRL +鼠标右键。

3.  这些层的菜单将直接显示在光标 ed 的下方。

5.  按 CTRL + ALT +鼠标右键，上面有对象的最上层将被选中。

请注意，在版本 6 中，透明度(或不透明度)低于 30%左右将阻止您使用第二个选项选择图层。

**不使用图层面板移动图层:**

*   将层向下移动一级，按 CTRL + [
*   将层向上移动一级，按 CTRL + ]
*   将一个层移动到底部，按 CTRL + SHIFT + [
*   将层移动到顶部，按 CTRL + SHIFT + ]

**不使用图层面板选择图层:**

*   选择向下一级的层，按 ALT + [
*   选择上一层，按 ALT + ]
*   选择最底层，按 ALT + SHIFT + [
*   选择顶层，按 ALT + SHIFT + ]

按 CTRL + SHIFT + I

***反转图像的颜色***

按 CTRL + I

***加载通道而不使用通道调色板***

要快速查看您可以收看的频道，请按 CTRL 和 1、2、3、4 等。这将显示颜色通道的灰度版本(CTRL + ~带你回到复合通道)。

要从频道中实际加载选择，请按住 CTRL + ALT + [number]。

***使两幅图像大小相同***

1.  选择一个图像。

3.  然后像往常一样从图像菜单中选择图像尺寸对话框。您会注意到窗口菜单和其他菜单(取决于您使用的是 Mac 还是 PC)仍然可用。

5.  选择窗口菜单，然后单击菜单底部第二个图像的名称。

7.  “图像尺寸”对话框中的尺寸现在将与第二幅图像相同。

***使自由变换出现***

按 Ctrl+T

***使图层出现***

*   您可以单击眼睛并下拉列表，或者
*   按住 Option 键点按(Mac OS)或 Alt 键点按(Windows)层的眼睛图标，以仅显示该层，或者
*   再次按住 Option 键单击(Mac OS)或 Alt 键单击(Windows)眼睛栏，以重新显示所有图层。

***不用魔杖进行选择***

选择并按住 CTRL 键单击包含该元素的层，以自动用行进的蚂蚁包围该层上的所有内容。

***打开图像文件的简单方法***

双击工作区(大多数用户的 PhotoShop 副本上的灰色区域)将打开对话框。

***将图像粘贴到图层蒙版中***

在要插入的图像上:

1.  按 CTRL + A

3.  然后按 CRTL + C

5.  转到要插入图像的图层，按住 ALT 键并点击它的蒙版

7.  然后按 CTRL + V

9.  再次按住 ALT 键并单击蒙版，停止查看蒙版

另外，点击图层和它的蒙版之间的小链接图标。现在，您可以在不移动图层的情况下移动蒙版(反之亦然)。

***保持透明***

选择图层或图层组，然后按“/”。

***使用一键操作***
下载[这些操作](http://www.webmasterbase.com/examples/psactions/creoles_actions.atn)，由 SitePoint Advisor [Creole](http://www.sitepointforums.com/member.php?s=&action=getinfo&userid=8109) 提供，并使用以下按键操作(Photoshop v6 以上)。

**F2–裁剪**
创建一个矩形选区，按 F2 键。对于裁剪掉“大数据”很有用:图像边界之外的信息增加了文件的大小。慎用。

**F3–描边**
创建一个任意形状的选区，按 F3。

将打开一个对话框，允许您更改厚度、颜色、位置和混合。

这将默认的前景色作为初始颜色。小心:此工具使用最后选择的绘画工具(和画笔)作为创建笔画的“画笔”。这意味着，如果铅笔工具是您使用的最后一个工具，笔画将被添加别名。

**F5–展开选区**
创建一个任意形状的选区，按 F5。

这将打开扩展对话框，并允许您输入 1 到 100 之间的值。

**F6–合同选择**
创建一个任意形状的选择，按 F6。

这将打开合同对话框，并允许您输入 1 到 100 之间的值。

**F7–高斯模糊**
选择任意形状，按下 F7。

这将打开 GB 对话框，并允许您输入一个值。默认值为 3 像素。

**F8–旋转画布**
打开旋转画布对话框，允许你输入一个值，以及选择顺时针或逆时针。

***查看图像和画布大小***

是(在 PC 上):右击一张图片的标题栏，你会得到一系列选项，包括复制、图片大小、画布大小等。
