# 谷歌的操作系统来了——但不是来自谷歌

> 原文:[https://www . site point . com/Google-operating-system-arrives-but-not-from-Google/](https://www.sitepoint.com/googles-operating-system-arrives-but-not-from-google/)

长期以来，一直有传言称谷歌正在开发一款操作系统来与微软的 Windows 竞争。当他们在九月推出他们的浏览器产品 Chrome 时，[我们注意到](https://www.sitepoint.com/google-working-on-browser-what-about-mozilla/)它本质上是谷歌的操作系统——网络正在成为的操作系统的用户界面。我们写道:“如果像许多人认为的那样，网络是未来的操作系统，那么一个专门为运行网络应用程序而设计的浏览器将非常重要，并有可能比其他玩家更具优势。”

现在，总部位于加利福尼亚州的 tiny Emeryville 的 Good OS 将浏览器即操作系统的想法推进了一步，发布了他们最新的操作系统，称为“[云](http://www.thinkgos.com/cloud.php)”Good OS 最著名的是 gOS，这是一个 Linux 发行版，去年在沃尔玛销售的 199 美元的 Everex gPC 上首次亮相。gOS 是 Linux 的一个精简版本，专门用于 web 应用程序和以 web 为中心的应用程序，如谷歌日历、文档、Gmail、Skype、YouTube 和 Firefox。

今天在法国巴黎举行的上网本世界峰会上宣布的新云 OS 产品是专门为上网本和上网本设计的。云“在几秒钟内”启动到一个浏览器，该浏览器专门设计用于通过内置的类似 Mac OS X 的 dock 快速轻松地访问基于云的应用程序，如谷歌的 web 应用程序套件。请注意，该浏览器看起来很像谷歌 Chrome，它是开源的。这似乎不太可能是巧合。

![](../Images/e4dcfff06f61e705516c23137cd42ea1.png "cloudos-screen")

由于云将浏览器与压缩的 Linux 内核集成在一起，用户还可以加载其他应用程序，如 Skype。该操作系统可以与 Windows 或另一个 Linux 发行版一起安装，用户可以在它们之间切换。Cloud 的内存只有 35MB，可以从 u 盘或 CD 启动。据该网站称，它“不需要额外的硬件，并与任何操作系统兼容。”

Good OS 创始人兼首席执行官刘中达表示，云是他“迄今为止最喜欢的 gOS 产品”

明年初，它将会和 Windows XP 一起出现在千兆字节的触摸屏上网本上。这些机器预计将在 1 月份的 CES 上首次亮相。
