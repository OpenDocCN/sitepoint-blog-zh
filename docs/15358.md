# WROX 出版社的三本 ASP 书籍

> 原文:[https://www.sitepoint.com/three-asp-books-wrox-press/](https://www.sitepoint.com/three-asp-books-wrox-press/)

如果你正在寻找一本好的 ASP 文本来帮助你开始学习这门语言，填补你知识中的一些空白，或者只是作为一个方便的关于 ASP 的案头参考，WROX Press 可能会有这本书给你。至少，前几天当我坐下来看他们的三款旗舰 ASP 时，我是这么想的。

以下是我对*开始 ASP 3.0* 、*专业 ASP 3.0* 、 *ASP 3.0 程序员参考*的迷你评测，均来自 WROX 出版社。希望这些能帮助你决定 WROX 出版社是否有你需要的。

##### 从 ASP 3.0 开始

![419beginningASP](../Images/90633a0f290107e41e3dda2ae40a6203.png)

无论你完全是编程新手，或者你有一些其他语言的经验，并且你正在寻找第一次学习 ASP，开始 ASP 3.0 将会给你一个很好的基础介绍。本书的介绍性章节以友好和支持的语气，带您了解 VBScript 语言的基础知识，VBScript 语言最常用于编写 ASP 脚本。不幸的是，虽然这些是书中最容易理解的章节，但我发现，从技术角度来说，它们也是最薄弱的部分。某些部分是不必要的混乱(特别是关于变体的早期部分，我不得不在在线文档中寻找更有说服力的描述)；同时，其他的包含一些相当基本的错误(例如，第 140 页的算术运算符列表错误地将整数除法运算符()标识为模数运算符(MOD)的替代形式)。

本书后面的章节涵盖了更高级的主题，如数据库访问，写得非常好，但是本书越深入，它就越偏离了提供一个简单、通用的 ASP 介绍的意图。例如，关于 XML 的最后一章似乎完全从本书的其余部分删除了，因为它与 ASP 几乎没有什么关系。乍看之下，相当大的附录似乎相当完整，但是要小心！您会很快发现在线文档比这里包含的信息更完整，这里的信息掩盖了重要的细节，以便将定义放入呈现它们的笨拙的表格中。

虽然*开始学习 ASP 3.0* 将为你提供一个很好的起点，如果你想学习 ASP 的基础知识，我不能推荐它作为这方面的权威著作。采取了它应该是什么-一个引导之旅，通过雷区的混乱概念，面对一个刚刚开始使用 ASP 这本书很好地服务于其目的。毫无疑问:如果你读了这本书，你将学会如何用 ASP 创建网站。请记住，如果有些东西令人困惑，或者看起来不太有意义，不要相信作者的话——仔细检查在线文档中的事实。

##### 专业 ASP 3.0

![419profASP](../Images/d4b431f764fadbdafa0d39cecde6085b.png)

如果你认为自己是一个有经验的程序员，特别是如果你有 1) Web 脚本，Visual Basic 或 VBScript 编程，或 3)初级或中级 ASP 的经验，那么你在 ASP 书籍中需要的与大多数初学者需要的是非常不同的。你需要的不是对这门语言的基础知识的温和的、示例性的介绍，而是对这门语言如何工作，以及你能从中得到什么和不能从中得到什么的严肃的、技术性的全面介绍。

*Professional ASP 3.0* 明确提供了这一点。从一开始，它就以惊人的细节描述了看似基本的特征。这种技术上的丰富不仅展示了作者的经验和知识，而且它确实给读者一种感觉，没有什么是因为“太难”或“不重要”而被掩盖的。如果你和我一样是个科技迷，你会讨厌厚厚的一本书留给你一些没有答案的问题，并停留在那些你只需花一半时间就能自己弄明白的“琐事”上。这本书没有让人失望。

涵盖广泛的主题，如数据库访问(使用 ADO 2.5、OLE DB、XML 和 RDS)、ASP 组件的开发、COM+组件服务、事务和消息服务、Win2000 Active Directory 交互、使用 CDO 和 CDONTS 动态生成电子邮件、ASP 性能和安全性，所有这些都具有相同的技术深度和准确性，这是一项艰巨的任务，这也解释了为什么本书的封面上有 15 个人！如果你是一个有经验的、有技术头脑的 Web 开发人员，不害怕接触语言的精髓，那么你会很高兴地看到 *Professional ASP 3.0* 不遗余力地追求细节。

##### ASP 3.0 程序员参考

![419progrefASP](../Images/8e808b7945f4a1af26c7f52231f580c9.png)

我不得不承认，在 ASP 3.0 开始的最后四分之一的时间里，那些毫无用处的参考资料给我留下了一个清晰的印象，那就是一个好的 ASP 参考资料的最佳去处是 ASP 本身附带的在线文档。 *Professional ASP 3.0* 几乎没有改变这种观点，因为它的参考部分几乎不存在。我一看 WROX 的 *ASP 3.0 程序员参考*，这个明显遗漏的原因就清楚了。

《ASP 3.0 程序员参考》大部分是由为 *Professional ASP 3.0 做出贡献的同一个团队的成员编写的，这让我感到惊讶，它比 ASP 的每个方面的在线文档都有更清晰的描述和更好的例子。从 Response 对象提供的缓冲方法等基本要素，到 XML 文档生成的复杂性，以及在 ASP 中与 Active Directory 服务器的交互，所有内容都以严格的精确度进行了记录，并且相关示例演示了每个功能。 *Professional ASP 3.0* 中涉及的每个高级主题在本书中都有相应的章节。*

无论您刚刚学习 ASP 并准备开始开发您的第一个真正的服务器端应用程序，或者您是一个经验丰富的 ASP 开发人员，厌倦了解释 ASP 在线参考的某些部分所需的猜测，您都会喜欢有 *ASP 3.0 程序员参考*在手边。
