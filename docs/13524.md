# 这是错误的…

> 原文：<https://www.sitepoint.com/this-is-just-wrong/>

用 [gscloud.pl](http://johnbokma.com/perl/google-search-cloud.html) 对照[网站周二](http://webtuesday.ch)的访问日志，注意到来自谷歌的搜索推荐数量惊人，如[11 月 14 日](http://www.google.com/search?q=14th+November)—*9 日*结果带你到[这里](http://webtuesday.ch/meetings/20061114)！(我没有从谷歌注销，所以我假设这些没有被我的搜索历史加权)
