# 查看 DRAMS 4.0 域注册软件

> 原文:[https://www.sitepoint.com/domain-registration-software/](https://www.sitepoint.com/domain-registration-software/)

你在寻找出售域名的机会吗？我也是…

当我决定创办一个域名注册网站时，我已经销售虚拟主机包有一段时间了。毕竟，这两项业务实际上是携手并进的，这是一个不错的额外赚钱的方法。然而，当我开始考虑建立一个网站时，我对可用的选项感到非常失望。这些解决方案要么不够强大，要么该网站必须看起来像所有其他使用该解决方案的网站(对我来说太千篇一律了)。最后，我从 dramsystem.com 来到了德拉姆。

DRAMS(域名注册和管理系统)是一个完全可定制的域名注册包，基于 PHP 和 MySQL 后端。这是一个面向 eNom 经销商的基于模板的解决方案，非常适合 Web 主机和其他任何有兴趣以较低成本建立域名注册站点的人。使用 DRAMS，您的用户可以在您的网站上注册、续费和转让域名！

在我安装、定制和使用 DRAMS 的过程中，我必须学习很多相关知识，并让它在真实环境中平稳运行。这里，简单地说，是我对 DRAMS 的看法。

##### 装置

一旦我获得了我的 eNom 经销商帐户，并在我选择的支付处理器(PayPal 和 2Checkout)上设置了帐户，我就准备好开始了。就支付处理器而言，DRAMS 是非常灵活的。它将与 Authorizenet、PayPal、InfoDial、2Checkout、WorldPay、PSIGate、Revecom/Paysystems 和 Echo 信用卡处理系统配合使用。

DRAMS 运行在 Linux，Windows 和 BSD 操作系统上，需要 PHP(关闭 safe_mode，打开 register_globals)，cURL + SSL 和一个 Web 证书(虽然不总是需要，但推荐)。

安装过程很简单，尤其是如果你曾经安装过 phpBB、vBulletin 或 Post Nuke 等脚本。上传文件，编辑几个文件，运行几个文件，就大功告成了！

##### 从用户的角度来看

DRAMS 允许用户在你的网站上建立他们自己的用户账户。这使得他们可以从一个简单的投资组合页面管理所有的域名，并利用 eNom 提供的所有功能，而无需离开您的网站。从修改 whois 记录到设置域名服务器，多亏了 DRAMS，他们可以做到这一切。事实上，这是 DRAMS 提供的主要好处之一。

##### 管理

DRAMS 管理部分简单易用。

您可以设置价格，设置经销商定价(人们可以通过您注册经销商帐户)，设置停车页面(注册后所有域名将重定向到该页面)，管理用户帐户，管理域名，查看交易，等等。

通过“管理控制台”，您可以在几秒钟内注册、更新和转让域名。如果客户为一个域名付费，但由于某种原因，注册中心关闭，域名注册没有通过，这就特别有用。你只需进入控制台，输入用户名，选择用户的账户，然后点击“注册”域名将被注册，钱(你的基价)将从你的 eNom 账户中扣除。

另一个值得一提的特性是填充工具。如果您之前已经通过您的 eNom 帐户注册了域名，这个工具将允许您将它们带到您的 DRAMS 站点上，甚至将它们分配给不同的用户帐户。

##### 模板定制

如果您希望定制 DRAMS 模板的外观和感觉，这并不十分困难(特别是如果您以前已经制作了 phpBB 或类似的模板)，尽管这可能会很耗时。

在模板文件中是你的网站的实际文本。如果您希望编辑某个页面上的文本——您很可能希望这样做——您必须编辑该文件。但是，所有的文件都坚持相同的写作风格，所以如果它与您想要的声音不同，您将需要打开大多数文件并编辑它们。就像我说的，不难，但是很费时间。

默认的 DRAMS 发行版包括 4 个模板；由第三方制作的有限数量的模板可以在 DRAMS 支持论坛上免费获得。

##### 模块和修改

DRAMS 团队提供了几个额外的模块来反映流行的 eNom 特性。这些服务的价格从 7.50 美元到 15.00 美元不等，包括我的手机名、我的地图名、动态域名系统、域名系统托管和文字搜索。

除此之外，在 DRAMS 支持论坛中还有一个修改论坛，在那里 DRAMS 用户可以发布免费的修改，你可以使用 DRAMS。我最喜欢的是通用查找框，它允许你创建一个可以放在任何网站上的查找框。用户在空白字段中输入一个域名，选择一个扩展名，然后直接到您的网站查看该域名的可用性并购买。

DRAMS 附带一个通常有用的手册。它可以稍微详细一点，但还不错。它回答了许多常见的 dram 问题，并会指导您完成许多 dram 的设置和定制。
