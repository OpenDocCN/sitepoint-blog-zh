# 设计原则:对比

> 原文:[https://www.sitepoint.com/principles-of-design-contrast/](https://www.sitepoint.com/principles-of-design-contrast/)

到目前为止，在这个系列中，我们已经看到了如何通过应用[平衡](https://www.sitepoint.com/principles-of-good-design-balance/)和[接近](https://www.sitepoint.com/principles-of-design-proximity/)的原则来改进你的设计工作。今天我们继续对比的概念。

当页面上的两个元素不同时，就会产生对比。例如，文本和背景颜色可以是不同的颜色。它可以是一个大而粗的标题和一个无衬线字体(阅读更多关于[排版中的对比和冲突](https://www.sitepoint.com/typography-concordant-contrasting-conflicting/))作为正文。这可能是大图形和小图形之间的差异，也可能是粗糙纹理与平滑纹理的结合。**我们的眼睛喜欢对比。**

对比重要的是元素要完全不同。不仅仅是一点点不同。

 **1。对比吸引眼球**

在你的设计中使用对比的一个主要原因，无论是印刷还是网页，是为了抓住注意力。

[Carsonified](http://carsonified.com/) 网站利用对比产生影响。该网站有大的粗体文本和图像，以及一个反过来，高对比度的配色方案。

![Carsonified](../Images/cc70b610180da254be4ede31fd0f9c03.png "Carsonified")

《近距离》杂志封面使用了一幅有趣的对比图像，一艘艘白色小船漂浮在深蓝绿色的海面上。

![michaelfreimuth23](../Images/1b70d846dd732376922c826c36c91700.png "michaelfreimuth23")

**2。对比辅助信息的组织**

当使用对比时，不仅页面更有吸引力，而且文档的目的和组织也更清晰。在下面的杂志跨页中， [Studio8](http://www.studio8design.co.uk) 使用对比、平衡和邻近法则制作了一个不同寻常的、引人注目的页面，上面有投稿人的简历。较重的黑色字体与较轻的正文形成了良好的对比。**T3】**

**![studio804](../Images/7e2a722f723935cf0647884ed65171cc.png "studio804")T2】**

该跨页也由 Studio8 分成两半，左侧为反向展开式。每页提供两种独立但相关的产品信息。一个巨大的&符号把这几页连接在一起。

![Caviar](../Images/22ce268965fb5c92cf6856c7d9c2f132.png "Caviar")

**3。对比产生焦点**

iPod 的著名广告熟练地运用对比将观众的注意力集中在音乐播放器上。广告以色彩明亮的背景衬托出一个人物的轮廓。iPod 和耳机以白色出现，在轮廓和彩色背景的映衬下清晰可见。

![ipod](../Images/d0240ffc8ccce24170ff821d41961f6f.png "ipod")

这款特别定制的酒瓶的设计依赖于白色文字和涂鸦与葡萄酒深红色的对比。它作为独特的圣诞礼物非常有效。

![WineBottles](../Images/f1e79617a8a836df578ec49898345f0c.png "WineBottles")
图像来源:[死亡线](http://www.thedieline.com/)

**在设计中增加对比时需要考虑的事情**

1.你是如何创造对比的？是通过纹理，字体，颜色，还是形状？

2.如果你想通过排版来达到对比，你在用哪些字体？它们是非常不同，还是仅仅有一点点不同？大胆选择字体，但要记得确保文字清晰易读。

3.对比是否强化了你的设计理念？
