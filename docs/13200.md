# 脸书灯塔:伟大的隐私公关噱头

> 原文:[https://www . site point . com/Facebook-beacon-the-great-privacy-pr-stunt/](https://www.sitepoint.com/facebook-beacon-the-great-privacy-pr-stunt/)

以下是我对[脸书灯塔](http://www.facebook.com/business/?beacon)的看法，这是最近流行的社交网站[脸书](http://facebook.com/)增加的功能，用于通知朋友关于外部网站的活动:**这场争议是精心策划的**。

上周，由于隐私问题，该功能被缩减，但我不禁认为这并不是脸书宏伟计划的一部分。

在你拿出火把之前，不，这不是基于事实的陈述。这是一种观点。这一假设基于这样一个事实，即 Beacon 是一项颠覆性创新，从设计上来说，它推动了一个人在个人订阅中可以接受多少广告的极限。

我确信，脸书团队——一个比大多数人更了解如何成功建立一个庞大社区的团队——预料到了这种反弹，但还是决定继续下去。

开发一个脸书应用程序来推广你的网站是一回事。但是从什么时候开始，打破 T2 的围墙花园需要打破网络用户所坚持的最后一丝隐私呢？

这场灾难让我想起了政界经常出现的一些陈词滥调。一个是“任何 PR 都是好 PR”这句话。这种关注和流量无疑会导致一群新的广告商注册在 Beacon 上做广告——可能比没有任何争议的情况下还要多。

第二个相似之处是政治世界中一直使用的一个经典的双管齐下的分散注意力的方法:**如果你同时提出两个争议，其中一个会吸引所有的注意力，而另一个会溜走**。在这种情况下，Beacon 实际上只是另一个被宣传为“功能”的广告程序，这一事实被超过 50，000 名脸书用户提出的隐私问题所掩盖，他们在[签署了一份尊重他们隐私的请愿书](http://civ.moveon.org/facebookprivacy/?rc=fb_front/)。

告诉我脸书不会想“如果我们在用户隐私方面突破可接受的极限，也许他们不会注意到他们的 feed 中突然有了更多的广告！然后我们可以表现得像是在倾听我们的用户，并从中得到好处！".已经签约的合伙人显然对此事保持沉默。

你怎么想呢?
