# 在野外发现的非拉丁顶级域名

> 原文:[https://www . site point . com/non-Latin-TLD-spotted-in-the-wild/](https://www.sitepoint.com/non-latin-tlds-spotted-in-the-wild/)

去年底， [ICANN](http://www.icann.org/) 决定开始接受非拉丁文字的国家代码顶级域名(TLD)申请。当时，他们承认这只是第一步，还有很多工作要做。然而，上周，第一个非拉丁顶级域名的实例被启用。这些域名是埃及的:مصر(埃及)，沙特阿拉伯的:السعودية (AlSaudiah)，阿联酋的:امارات (Emarat)。这些域名支持完全从右向左书写的 URL(这样域名会排在最前面)。

举个例子，一个已经在使用其中一个新域名的网站是 http://وزارة-الأتصالات.مصر/(很难看到，但你应该能看出前几个字符后面的句号，因为网址是从右向左写的)。这个网址应该会引导你到埃及通信和信息技术部的网站。

所有主流浏览器的最新版本都能够处理国际化域名(IDN)。您的浏览器应该加载正确的站点，尽管它可能不会在地址栏中显示阿拉伯语的 URL 这确实取决于您的浏览器的域白名单和语言设置。ICANN 已经发布了一个相当详细的解释，说明如何让新域名在 IE8、Firefox 和 Opera 中看起来更漂亮。

除了上述三个阿拉伯语域名，ICANN 还批准了俄罗斯联邦的рф (rf)域名，进一步的工作将在本周举行的俄罗斯互联网监管论坛上进行。

我想我们都同意这是网络的一大进步，有助于让更多的语言和文化可以访问和使用它。祝贺 ICANN 和这些国家做得很好！
