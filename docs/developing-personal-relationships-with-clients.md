# 发展与客户的个人关系

> 原文：<https://www.sitepoint.com/developing-personal-relationships-with-clients/>

[![talking on phone](img/5f0845d13d4cb8fb5fe9094307c21254.png "talking on phone")](https://www.sitepoint.com/wp-content/uploads/2009/03/phone.jpg) 我经历过各种各样的客户关系——从所有的业务到过于私人的——我花了一点时间才发现适合我的业务和个人的正确组合。当我刚刚起步的时候，我的直觉告诉我，任何时候都要全力以赴。我希望被认为是平等的，得到客户的尊重，而不是处于被利用的地位。一旦我有了一些经验和一些长期的客户关系，我就采取了一种更个人化的方法，可能是因为我对自己作为一个商人更有信心。

随着社交网络越来越受欢迎，我发现自己在脸书和 Twitter 上与客户的交集越来越多，许多业务/个人界限变得越来越模糊。对于一些人来说，这可能是太多的客户互动，但这为我创造了惊人的机会。当你像我一样进行虚拟工作时，在业务关系中创造个人元素可能是一个挑战。社交网站为个人互动创造了一个出口。

为什么重要？客户希望雇佣有经验、专业、能完成工作的人，但他们也希望雇佣他们觉得了解并能信任的人。首先，我们都是人，付出一点点自己会为一段繁荣的关系奠定基础。这增加了客户再次与你合作的可能性。

让你的商业关系变得有点私人化并不意味着你需要告诉你的客户关于你自己的私人事情或者问他们私人问题。但是如果你给他们多一点时间——提及你的周末计划，询问他们生病的弟媳近况如何，记住生日——你就可以创造一种多维关系，通过这种关系你可以完成很多事情:

*   在工作中创造忠诚和信任
*   给客户更多推荐你的理由，因为他们觉得他们真正了解你
*   更好地了解彼此，这使得合作更容易、更有效
*   为公司内部的额外工作创造机会
*   变得更加平易近人，能够避免误解
*   展示你是人类，就像他们一样

就像任何关系一样，这背后的驱动因素是沟通，无论是在线还是离线。此外，像其他关系一样，这种哲学并不适用于所有情况。有些客户你不想与之建立这种关系，要么是因为他们的个人偏好，要么是因为他们的个性问题。重要的是保持双方的舒适水平，并调整你们的关系以适应每个客户。

你和你的客户有私人关系吗？有没有事与愿违的时候？
