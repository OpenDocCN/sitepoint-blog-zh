# 网站风格指南:创建网站的基本设计原则

> 原文：<https://www.sitepoint.com/creating-web-sites/>

Patrick J. Lynch 和 Sarah Horton 写的 Web 风格指南宣传自己是网站设计者的基本指南。在很大程度上，这本书是成功的，因为它努力涵盖了网站制作中涉及的非技术问题。

虽然我个人读过几十本关于 HTML、层叠样式表、动态 HTML 和 JavaScript 的书，但这些书都没有以任何有意义的方式讨论过这些技术的应用。虽然能够在睡梦中编写复杂的表格是一种值得培养的能力，但能够以一种有意义的方式应用我们今天拥有的所有技术是完全不同的事情。这就是网页风格指南的用武之地。

这本书被清晰地组织成八个不同的部分，每一部分都涵盖了应用技术和过程来实现你(公司)目标的来龙去脉。这本书的第一部分是关于过程——基本上是在你开始你的网站开发之前你需要知道和决定的事情。在这里，作者直截了当地问了一些问题，迫使你真正弄清楚你想通过互联网实现什么。一步一步的引导你通过网站规划，信息架构，设计，跟踪和维护完成了本书的过程部分。

这本书的第二部分深入探讨了界面设计。你的网站的界面就是你的访问者将要与之互动的东西，你应该对此多加思考。作者简要讨论了网页设计与传统文档设计的区别，然后介绍了一些技巧，这些技巧集中在使每个网页独立、提供清晰的导航帮助和优雅的降级。

本书的网站设计部分讨论了组织信息、结构、网站设计主题和网站元素。这一章的建议，包括诸如文本分块的想法，非常实用，尽管有点过于简单。作者试图将所有网站分成教学和参考等主题，但这并没有让我满意。这本书的重点然后缩小到设计个人网页。

下一章的中心思想是页面设计，网站中的每个页面都应该传达清晰、有序和可信的信息。作者通过视觉层次、一致性、页面长度和设计网格来实现这一点的解决方案是一个很好的尝试，它打破了单个元素，当这些元素放在一起时，创建了一个清晰并传达可信度的页面。本章的后半部分讨论了表格在页面布局中的作用。除了对齐之外，还会讨论行长度、边距、列和间距。框架也有简要介绍。作者不是告诉你如何使用表格和框架来使你的网站更有吸引力，而是教你如何使用这些元素来使你的网站更有用。下一章集中在每页上最重要的元素:文本。

160 页的书中有 26 页是关于网络印刷的。这是一个我从未在任何网页设计书籍中看到过的主题，坦白地说，在阅读这本书之前，我对此相当无知。像对齐文本、字体、易读性、大小写、空白、如何强调文本和跨平台问题等主题，我在其他地方都没有看到过。涵盖了用于排版目的的样式表的应用，尽管这些信息已经过时了，因为这本书是在 1998 年末写的。

在排版一章之后，这本书聚焦于编辑政策。还是那句话，这是我没在其他书上看到过的话题。下一部分，网络图形，偏离了本书其余部分的基调。我没有收到关于在互联网上智能使用图形的信息，而是遇到了关于网络安全调色板、压缩、抖动、格式和图像映射的讨论；可以在几十本其他书中找到的信息。不幸的是，关于多媒体的下一章在多媒体处理、格式、编解码器、压缩和交付方面的说明并没有好到哪里去。

如标题所示，这本书涵盖了优秀网页设计的基本原则。书中的指导方针应该是建议，而不是永不打破的硬性规定。这本书的基调相当干燥，考虑到个人表达和参与主题的潜力，这令人失望。作者是耶鲁大学的教育家；因此，书中的大部分例子直接来自耶鲁网站，而不是来自不同类型的网站(新闻、娱乐等。)来自网络。尽管如此，这本书在组织、排版和设计流程方面的许多启发性想法和理论使这本书值得一读。

**评级:3/5**
