# MicroPlaza 解决 Twitter 链接共享

> 原文:[https://www . site point . com/micro paza-solves-Twitter-link-sharing/](https://www.sitepoint.com/microplaza-solves-twitter-link-sharing/)

由于它的速度和分布式本质，Twitter 在传播和传播突发新闻方面绝对是最棒的。正因为如此，有很多网站试图在 Twitter 上聚合链接，试图将它变成下一个 Digg 或 Techmeme。然而，Twitter 并不太适合告诉我们大众喜欢什么。依靠“转发”来对链接进行排名的网站是最接近的，因为转发可以被视为对该链接的隐性投票(很像 Techmeme 或 Google 使用的反向链接)。不过，基于转发的聚合器查看整个时间线还是有点笨拙。他们发现的链接都是漫无目的的，在持续向我展示我感兴趣的东西方面做得不好。

大约一个月前，在一篇名为“[Twitter 会取代 Digg](https://www.sitepoint.com/could-twitter-ever-replace-digg/) 吗”的文章中，我认为虽然 Twitter 不擅长展示人们喜欢什么，但它*可以*非常有效地指出你的朋友喜欢什么。在那篇文章中，我列出了一个我认为可以实现这一目标的系统的结构:“如果有一个 Twitter 工具可以查看我的朋友图表，然后找出我朋友在 Twitter 上发布的所有链接，并按照受欢迎程度(他们发布了多少次，对我朋友的权重)和转发次数(我朋友的朋友中有多少人认为这是一个有价值的链接)对它们进行排列，它将展示一种类似 Digg 的图片，显示我可能已经发现了有趣的人。”

帖子发布几个小时后，比利时企业家奥利维尔·韦贝克联系了我，他是知识广场的首席执行官。韦贝克让我知道了一个新的服务，它刚刚进入私有 alpha，名为[micro paza](http://microplaza.com/)，基本上做了我要求的事情。今天，MicroPlaza 已经超越了最初的 50 人测试集，并开始了邀请测试。

我有机会成为 MicroPlaza 的早期用户之一，并留下了深刻的印象。该网站绘制了你在 Twitter 上关注的人分享的所有链接，并按照受欢迎程度(被转发的次数)或日期显示在 Techeme 风格的链接+源视图中。MicroPlaza 非常棒，有几个原因。首先，它让我可以很容易地跟踪我感兴趣的人在 Twitter 上分享的所有链接，而不需要我全天候观看我的时间线。第二，它让我只关注最流行的链接，如果我愿意的话。第三，它鼓励我关注更多的人，并且只关注那些发微博谈论我关心的事情的人，因为我关注的人越多，这个网站对我的效用就越大。

![](../Images/ac99618fea837824e2cd0ebcf614b837.png "microplaza-screen")

的确，如果你关注那些经常转发的用户，比如 [@problogger](http://twitter.com/problogger) ， [@mashable](http://twitter.com/mashable) ，或者 [@sitepointdotcom](http://twitter.com/sitepointdotcom) ，你的“热门观点”时间表可能会被他们的链接所主导。事实上，如果你的追随者转发这些人的链接，你的时间线可能已经到了。如果能够屏蔽来自某些用户名的链接就好了——或者如果他们不是我关注的人，至少降低他们的权重(如果他们是，我大概也想看那些链接)。也许 MicroPlaza 可以对链接进行加权，这样如果我关注的不止一个人转发该链接(阈值由我关注的人的通常转发行为决定)，经常被转发的项目只会在我的时间线上被推得更高。

MicroPlaza 通过一个名为“部落”的特性来解决这个问题。部落本质上是你朋友的分组，起到过滤器的作用。例如，我关注了一些最常发政治链接的 Twitter 用户，一群主要发关于网络技术的 Twitter 用户，还有一群通常发关于企业家精神的 Twitter 用户。我可以将这些用户按部落排列，并查看他们正在谈论的链接的时间轴。这个过程很简单，但是如果你有很多追随者的话会很乏味，因为你必须一次一个地把每个追随者加入一个部落。

MicroPlaza 还有另外两个有用的功能:书签，允许你保存你最喜欢的链接，以及非常引人注目的“存在”功能。Being 让你成为任何其他 Twitter 用户，并根据他们朋友公开发布的链接查看时间线。这允许您做几件事情。首先，你可以看到你所在领域的思想领袖正在关注的事情类型，其次，如果你没有关注很多人，Being 功能可以让你以一种相当有组织的方式探索 Twitter 上共享的链接，以便找到那些在 Twitter 上发布你有兴趣阅读的东西的人。

很快，MicroPlaza 将增加一个搜索引擎，以及共享项目的图表和统计数据。该网站也是 Twitter 新 OAuth 实现的首批用户之一，这意味着很快你就可以通过 MicroPlaza 直接从网站上发布你找到的物品，而无需输入你的 Twitter 用户名和密码。

总之，MicroPlaza 是我见过的最好的 Twitter 链接分享网站。它很好地解决了只向我展示我关心的链接的问题，并且有许多引人注目的功能，使它真正从竞争对手中脱颖而出。该网站今天进入仅邀请测试，但公共时间表对任何人都可用，所以你应该能够立即探索这项服务是如何工作的。
