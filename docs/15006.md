# 回顾–实际上是一封群发邮件

**您是否有一份电子邮件简讯，并且需要一个可靠的解决方案来管理订阅者、退订者甚至那些讨厌的无法送达的邮件？您是否正在寻找专业、易用的一体化解决方案？Infacta 的 Windows 群发邮件软件可能是您正在寻找的答案。**

群邮件提供三个版本…免费，专业和 Plus。价格分别为 0 美元、79.95 美元和 179.95 美元，您将能够找到符合您需求和预算的版本。下载并试用免费版本来测试该软件。如果您的邮件列表非常小，免费版本可能就足够了，但是它缺少使群发邮件在竞争中脱颖而出的特性:插件，我们稍后将讨论它。

##### 哪个节目适合你？

那么，版本之间有什么区别呢？以下是每个程序的基本功能列表:

***群发邮件免费***

*   仅个人发送模式
*   有限的通讯簿导入支持
*   没有插件支持
*   一次最多发送给 100 个收件人

***群发邮件亲***

*   批量发送功能
*   直接发送(不使用您的 ISP 的邮件服务器)
*   从 Outlook Express、Eudora、Opera 6、AOL 7 和 Netscape 导入通讯簿和通讯组列表
*   免费插件

***群发邮件加***

*   Group Mail Pro 的所有功能加上…
*   发送手机短信
*   发送速度更快
*   导入 Outlook、Group Wise、Lotus Notes、Exchange 和 PINE 通讯簿和分发列表
*   直接从任何 ODBC 或 OLE DB/ADO 兼容数据库导入
*   免费插件

##### 插件—好东西

Group Mail 的 Pro 和 Plus 版本都支持附加插件，这些插件可从 Infacta 的网站上免费下载。这些插件允许您定制您的群邮件，以满足您的时事通讯的需求。只安装您需要的额外模块，或者全部安装。下面列出了其他可用的插件:

*   数据库导入程序
*   电子邮件检查器
*   GroupHTML(电子邮件创建者)
*   订户
*   支持
*   分组合并
*   群组清除

您可以在 Infacta 插件页面找到关于这些插件的更多信息。

##### 我的专业意见

对于 99%的用户来说，群发邮件可以满足他们所有的电子邮件通讯需求。这个软件已经为我们公司的电子邮件通讯维护节省了无数的时间。该软件订阅新的收件人(或者他们可以在几分钟内从任何当前的数据库中导入)，取消订阅删除，删除不良电子邮件地址，甚至允许你在一个免费插件的帮助下在程序中编写邮件。它还在发送消息时显示了一个很好的运行日志，因此您可以看到有多少成功的、失败的以及有多少被排除的成员。

不仅如此，群邮件还发送由两部分组成的 MIME 消息，因此您不必担心您的读者是否能处理 HTML 电子邮件。如果他们不能，他们会自动得到一个文本版本！对于任何认真对待邮件列表的人来说，这是一个非常有用的特性。这比管理两个单独的邮件列表要容易得多:一个是 HTML 邮件列表，一个是纯文本邮件列表。

您可能会想，要包含所有这些特性，程序一定过于复杂了。相反，群发邮件很容易安装，甚至更容易学习。它直观、易用的界面让我在几分钟内就能发送电子邮件！我不用看手册的一页。很简单—只需设置一个发件人、一个电子邮件列表，然后创建一封电子邮件！这是主界面的截图——我想你会同意，这样直观的布局很难出错。

##### 缺点是…

你可能会想，“这听起来很棒，但是有什么缺点呢？”在我们 2000 多名成员的邮件列表中使用这个软件几个月之后，我能想到的唯一的“缺点”就是群发邮件是一个驻留在你电脑上的本地程序。虽然这也可以被认为是“pro ”,但它确实使用了大量的系统资源来向成千上万的收件人发送个性化的电子邮件。我们有一台单独的电脑专门用于时事通讯管理，但大多数人将无法做到这一点。在这种情况下，我的建议是在你不使用电脑的晚上发送简讯。

Group Mail 获得了如此多的奖项，我无法在此一一列出…有几个是 ZDNet 编辑推荐、免费软件指南的月度计划、酷工具奖和 Tucows 5x Cows 奖。从他们网站上列出的奖项和褒奖，你会发现我不是这个软件的唯一支持者。

##### 最后的话

如果你需要管理一封电子邮件简讯，并且不想使用在线解决方案，这就是你的工具。我肯定会把这个软件推荐给任何人，从新手到专家。
