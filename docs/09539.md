# 超越讲故事和销售:在你的潜在客户的真人秀中的角色

> 原文:[https://www.sitepoint.com/beyond-story-telling-selling/](https://www.sitepoint.com/beyond-story-telling-selling/)

我一直在谈论利用故事的力量来说服，尤其是销售你的服务。但是在我的[上一篇文章](https://www.sitepoint.com/everything-i-know-about-selling-i-learned-from-mcdonalds/ "Everything I Know about Selling I Learned from McDonald’s")中，我说这不仅仅是讲几个好故事，你必须成为你潜在客户生活中的一个角色。但是具体怎么做呢？很高兴你问了。

要回答这个问题，我必须把你带回到你高中文学课的时代。你可能记得，一个故事包含某些元素，如人物、背景，当然还有情节。情节有以下几个部分:

一、目标:角色想要或需要什么
二、目标。障碍:一个角色和他的目标之间的东西——一个问题。尝试:角色做(或不做)什么来克服障碍
iv。结果:尝试的结果是什么
v .最终结果:角色对于目标的实现或未实现做了什么或说了什么

也许这对你来说很熟悉。当我第一次重新发现这一点时，它确实影响了我。这让我想起了我在销售过程的事实调查阶段向潜在客户提出的“需求分析”问题:

*   你为什么想要一个网站，你希望完成什么？
*   这会产生什么积极的结果？
*   实现这一目标的障碍是什么？
*   你为什么不自己做？
*   你对成功的愿景是什么？

你看，你的潜在客户有一个目标，一些他想完成的事情——否则，你不会坐在他的办公室里讨论你的服务。这是我们现实生活剧剧情大纲的第一条:

*目标。这个角色(你的潜在客户)想要或需要什么？*

但问题是:你的潜在客户遇到了障碍。在商业规划 101 中，你可能已经学会了缩写词 SWOT——优势、劣势、障碍和威胁。企业面临着无数的障碍，但你我可以影响的是帮助他们接触新客户、更多客户或更好的客户。销售就是寻找那些有你可以满足的需求的人。如果没有障碍需要克服，那就没有问题。如果没问题，就没必要了。我们现在到了第二项:

*障碍。介于你的潜在客户和他的目标之间的东西；一个问题。*

接下来，你的潜在客户做了什么来克服障碍？也许他们已经有了某种类型的营销，但正在努力找出如何将网络融入其中。可能他们自己尝试过什么，失败了。或者他们只是不知道从哪里开始，正在寻求专家的建议。底线是，他们遇到了一个他们无法克服的障碍。作为顾问，你的工作就是让你的潜在客户告诉你他的故事。这就是需求评估的目的——发现你的潜在客户想要什么，以及是什么阻碍他实现。所以，我们戏剧的第三项是:

尝试:你的潜在客户为克服这一障碍做了什么或没做什么？

这就是你进入故事的地方。如果你的潜在客户可以自己解决这个问题，他就不需要你了，不是吗？但是你扮演什么角色呢？你到底是什么角色？

这是我们大多数人犯错误的地方:我们认为自己是英雄。我们想成为《闪光的爱情》中的骑士，骑着马去拯救世界。但是，就在我们穿戴整齐，手握利剑，准备用我们的解决方案猛扑过去的时候，我们的潜在客户犹豫了。突然间，他的问题“没那么糟了”，他将“不得不回复你”

如果你一直碰壁，那是因为你没有认识到展开情节的一个基本元素。你的客户是英雄…不是你。(毕竟是他的故事不是吗？)而你却在这里……想成为明星，拿走所有的荣誉。难怪他会反抗。

如果你不是英雄，那谁是？故事中我们没有谈到的一个方面是*导师*的角色。想想文学中的一些伟大英雄。卢克抓了欧比万。佛罗多有甘道夫。吉利根有船长。在英雄的旅程故事原型中，英雄总是有一个导师，一个给英雄所需的资源、知识和信心来克服他所面临的障碍的人。

但是英雄最终必须独自面对他的斗争。甘道夫没有陪佛罗多进入魔多。卢克继续摧毁死星，在没有欧比万的情况下对抗达斯·维德。没有你，你的潜在客户也会继续实现他的希望、梦想和愿望。当他退休去某个遥远的南海天堂岛上的别墅时，你可以肯定你不会被邀请一起去。当他和其他富有的退休人员一起躺在沙滩上时，他想讲述的故事是他的故事，他是如何成功的。你在故事中的角色是，不管他承认与否，你在让他到达那里的过程中扮演了一个小角色。但是这样做的次数足够多，你就会成为你自己故事中的*英雄。*

> 通过帮助别人得到他们想要的，你可以得到生活中想要的任何东西。金克拉
