# 216 色网站管理员调色板

> 原文:[https://www.sitepoint.com/color-webmaster-palette/](https://www.sitepoint.com/color-webmaster-palette/)

**什么？你没听说过 216 色网页安全调色板吗？我来解释一下:**

1994 年，网景公司想出了一个解决操作系统问题的绝妙办法。你看，在 1994 年，许多计算机一次只能显示 256 种颜色。当一个以上的程序或网页同时出现在屏幕上，并且它们使用的颜色总数超过 256 种时，就会出现混乱——用户经常会看到扭曲得离谱的颜色。Netscape 的解决方案是定义一组 216 种颜色，它们在用户显示器上具有优先权。专门使用这些颜色的网页看起来会更好。

你为什么要在乎？

好问题。除非用户的计算机被限制为 256 色，否则在你的网站和图形中使用什么颜色没有区别。但是，如果您使用的是网页安全调色板之外的颜色，而某个运行 256 色的人来到您的站点，就会出现显示问题。很多时候，图形变得有斑点是因为它们被附近的颜色抖动了。有时，216 色调色板之外的颜色会被完全不同的颜色所取代。

你应该担心吗？

这取决于你的网站的主题是什么。如果你的网站迎合了网页开发人员、图形设计人员或游戏玩家的需求，你几乎可以肯定网站的绝大多数访问者都有能力一次显示 256 种以上的颜色。另一方面，如果你的网站面向的是不太懂技术的受众，那么坚持使用 216 色调色板是值得的。

如果你想知道你的访问者使用的是什么颜色设置，更先进的跟踪服务可以准确地告诉你有百分之多少的观众使用什么颜色设置。其中一个更好的免费跟踪服务，将为您提供这方面的信息是 HitBox.com。我建议在你的首页贴上跟踪器的代码一个星期，只是为了找到更多关于你的流量的信息——没有必要把它永久放在你的网站上，除非你不能访问日志文件。
