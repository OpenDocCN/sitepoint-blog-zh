您可以使用以下两种方法之一创建补间动画:

*   为动画创建开始和结束关键帧，并使用“属性”检查器中的“补间动画”选项来创建效果。
*   创建动画的第一个关键帧，然后在时间轴上插入所需的帧数。选择“插入”>“创建补间动画”,并将对象移动到舞台上的新位置。在这种情况下，Flash 会自动创建结束关键帧。

要创建补间动画:

**1。**点击一个层名，使其成为当前层。

**2。**在图层中选择一个空的关键帧，创建一个对象，将对象转换成符号。

**3。**在您希望动画结束的位置创建第二个关键帧。

**4。**在结束帧，做一些修改，比如移动一个对象的位置，改变它的颜色等。

**5。**如果属性检查器不可见，选择窗口>属性。

**6。**选择第一个关键帧。从“属性”检查器的“补间”弹出菜单中选择“运动”。

你完了！测试电影。
