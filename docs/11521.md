# 新系列:设计原理

> 原文:[https://www.sitepoint.com/new-series-principles-of-design/](https://www.sitepoint.com/new-series-principles-of-design/)

我正在开始一个名为《设计原理》的新系列，这听起来很崇高，似乎需要几年的学习。然而，对于我们要看的每一个原则，我敢打赌，即使你是设计新手，你也已经对它们有一点了解，但可能以前没有使用过术语或真正思考过太多。这些原则是基本的，它们出现在任何一件设计良好的作品中，无论是网页、宣传册、名片还是广告。没有一个原则是单独起作用的，它们通常一起起作用。

**元素与原理**

过去，我们一直关注设计元素，即[类型](https://www.sitepoint.com/elements-of-design-type/)、[线条](https://www.sitepoint.com/elements-of-design-the-line/)、[形状](https://www.sitepoint.com/design-elements-shape/)、[纹理](https://www.sitepoint.com/elements-of-design-texture/)。将这些视为您设计的构建模块。我们将在接下来的几周里探讨的原则是什么让这个结构变得坚固并把它结合在一起。有助于构建强大设计并使其脱颖而出的五个原则是:

*   平衡
    要创建一个干净、平衡的外观，页面上的每个元素都应该与另一个元素有视觉上的联系。不要在你的页面上随意填充东西(官方设计术语)。任何东西都不应该随意放在页面上。
*   当彼此相关的元素被紧密地放置在一起时，它们就变成了一个视觉单元，减少了混乱并给出了一个清晰的结构。将信息组织到适当的组中是改进设计的最快、最简单的方法之一。
*   对比
    如果页面上的所有东西看起来都一样，那么你的设计将会非常乏味。通过在设计中引入对比，你的页面会立刻变得更有吸引力。对比可以应用于形状、颜色、字体和线条。为了更好的对比，让元素有很大的不同。
*   **值** 值可以简单描述为物体的相对明度或暗度。像对比一样，价值可以增加设计的深度和维度。
*   **颜色** 颜色和值密切相关。颜色具有不可思议的力量来创造一种情绪，改变设计的意图。颜色的选择应该仔细，以确保任何设计的成功。

下周我们将从平衡开始，看看它如何改善你的设计，以及一些充分利用它的设计例子。
