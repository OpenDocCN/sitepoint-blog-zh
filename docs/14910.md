# Flash 在线游戏开发——简史

> 原文:[https://www.sitepoint.com/flash-brief-history/](https://www.sitepoint.com/flash-brief-history/)

这是两篇文章中的第一篇，这两篇文章将概述您可以使用 Flash MX 在在线游戏领域做些什么。

在这篇文章中，我将回顾这些指导方针，以及一些你可以用来通过 Flash Player 使你的游戏网页友好的技巧。第二篇文章将介绍如何使用 Flash MX 构建单板滑雪游戏。

在本文中，我们将从整体上探索游戏过程，并讨论为什么需要考虑使用 Flash MX 来开发游戏。

Flash 是一个非常灵活的工具，可以用来构建兼具娱乐性和教育性的游戏。

##### 开始的时候…

一开始有乒乓。感觉很好。它响了！除非，你输了，在这种情况下，你只能从控制台发出咕噜声。

那是 1975 年，我还是个孩子，住在密歇根州的弗林特。我的父亲，一个早期的电脑迷，刚刚在最近的清仓大甩卖中大获全胜，带回家一台游戏机。他把它带进了房子，很像几千年前尼安德特人可能对他们的第一头死牛所做的那样——当他发出胜利的吼声时，它被高举过头顶。我们整个周末都在电视上打乒乓球和网球。伙计，太酷了！

自从那些令人眩晕的日子以来，游戏系统经历了巨大的变化。在英国，我们在 1981 年看到了 Sinclair Spectrum(见下图)，紧接着是 Commodore 64。这两个系统都允许用户编写自己的游戏，并将游戏保存到录音带中。我仍然记得键入 Basic 并试图让简单的游戏 Break Out 运行起来。

上面的辛克莱·ZX 光谱拥有 8 种颜色和 48Kb 的内存！

任天堂和世嘉等公司随着任天堂家族系统和世嘉创世纪的推出，迅速改变了游戏机业务。一个全新水平的图形，声音和互动介绍给用户。

游戏用户的下一个飞跃来自个人电脑。《雷神之锤》、《沃尔芬斯坦》和《古墓丽影》改写了我们如何不仅在 PC 上玩游戏，还在局域网上互相对战。如今，微软的 X-Box、任天堂的 Game Cube，当然还有索尼的 Playstation II 等系统都在提供令人难以置信的图形，旨在模拟现实的细节。而索尼却在吹嘘，我们还什么都没看到。

##### 使用网络

那么……闪存在这一切中处于什么位置？

你必须去网上看看。虽然游戏控制台世界的一切都在快速发展，但仍存在一个重大缺陷。要玩为 Playstation 开发的游戏，你必须拥有一台 Playstation。另一方面，网络只要求你有一个互联网连接，一个网络浏览器和 Flash 播放器。

Flash 实际上是所有网络浏览器的插件。Flash 安装在每台新的 PC(它是默认 Windows XP 安装的一部分)和 Mac 上。事实上，Flash 从 Windows 98 开始就有了。你还会发现 Flash 安装在大多数 PDA 和许多交互式手机上，如在日本销售的 DoCoMo 的媒体电话。我基本上是说，闪存无处不在。事实上，已经有超过 5.2 亿的 Flash 安装记录，占所有互联网用户的 96%以上。

##### 为什么是 Flash？

现在，你我都知道闪存无处不在。但是，你怎么知道你想把它用于你的网络游戏呢？这引发了一个棘手的问题。传统上，网络浏览器对网络游戏并不友好。挑战在于找到一种通过拨号连接提供高质量图形的解决方案。即使大规模转向宽带，近 50%的互联网用户仍然使用拨号调制解调器上网。

图形、视频和音频叠加在一起可以生成大文件。多年来，解决这个问题的方法一直是出售光盘，它安装大型媒体文件，然后将你连接到互联网，这样系统就可以向你的计算机传递有关角色活动的信息，并用最新的补丁更新你的系统。同样，这里的问题是你需要一张 CD 才能开始使用游戏。

然而，Flash 能够帮助您解决这些问题，它将所有丰富的内容放到 Web 上，并通过您的 Web 浏览器交付给您。
Macromedia Flash 的优势

首先，Flash 的核心竞争力一直是通过拨号连接提供宽带体验。通过使用 Flash 的原生矢量图形动画格式，您可以在非常小的文件中提供清晰、令人惊叹的图形。

Flash 还有另一个锦囊妙计——它自己的编程语言，叫做 ActionScript。ActionScript 基于 JavaScript。如果你是一个 JavaScript 奇才，那么你将能够非常容易地迁移到 ActionScript。如果你是 ActionScript 和编程的新手，你会发现下图中的 ActionScript 既有逻辑性又容易掌握。

![1122_image4](../Images/0040268f1725dfeef71f0f8de81d0ef2.png)

最新版本的 Flash 提供了更多的功能，为您提供了在互联网上开发复杂游戏所需的工具。音频和视频得到了更多的关注。现在，您可以将 MP3 文件从 Flash 电影中分离出来，并从 Web 服务器上流式传输它们。Flash MX 使您能够将视频导入并压缩到您的 Flash 电影中。Flash Video 是为 Flash 转换的视频的名称，它基于 Sorenson 开发的压缩算法(如果这个名称听起来很熟悉，您会发现 Sorenson 也在 QuickTime 6 上做了很多工作)。

这两个功能也可以通过 Macromedia 的最新工具 Flash Communication Server MX 1.5 进行广播。Communication Server 使您能够与通过 Web 浏览器访问您的站点的人共享 Flash 电影。您还可以通过 Flash 播放器和通信服务器在互联网上广播视频/音频。结果是你有一个高影响力的媒体传送系统，可以通过一个插件来推送:Flash Player。

再加上新的 Flash MX 组件技术、Flash Remoting 与数据库的紧密集成以及对附加功能(如 3D)的广泛第三方支持，Flash 突然成为最新的游戏工具开发应用解决方案。

##### 当今的 Flash 游戏

现在开始在互联网上流行的 Flash 游戏类型，密切模仿了 80 年代的旧主机游戏。有很多俄罗斯方块的克隆版和吃豆人的翻版。但是，这些只是测试新技术所需的实验。同样的事情也发生在个人电脑上:最早的个人电脑游戏是经典游戏，如纸牌游戏(午餐时花几分钟在办公室里走一走，数一数在电脑上玩纸牌游戏的人数)。

Flash 将带领游戏开发者走向何方，现在已经开始成为现实。奇怪的是，我们看到 Flash 游戏发展的第一个地方是在广告领域。网络商业的新媒介是 Flash。你可以在 Weather.com、CNN.com 和 C|NET.com 等网站上看到它。所有这些网站都利用富媒体广告(又称 Flash)。在互动 Flash 广告方面非常成功的公司包括大众，他们的大众甲壳虫 Flash 广告。

你也可以去诸如 Shockwave.com 这样的网站看看。Shockwave.com 曾经是导演开发游戏的最后阵地。现在你会发现很多新游戏都是用 Flash 开发的。

今天，开发人员可以对 Flash 采取的下一步是使 Flash 游戏具有交互性。这可以通过当前的 Flash Player 来实现-您可以使用 Flash Remoting、Flash Communication Server 或 XML:Sockets 来实现。

##### 好处

这意味着你可以期待世界在线游戏因为 Flash 而显著增长。使用 Flash，您可以享受许多好处:

*   专为通过互联网分发而开发的小文件
*   支持富媒体格式，如图形、音频和视频
*   通过 XML 的多用户环境:套接字服务器和 Flash 通信服务器
*   其他工具的第三方支持，如 3D
*   通用格式—几乎每个人都有 Flash
