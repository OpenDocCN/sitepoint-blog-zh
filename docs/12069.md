# 验证码的末日即将来临

> 原文:[https://www.sitepoint.com/avoid-captchas/](https://www.sitepoint.com/avoid-captchas/)

验证码(区分计算机和人类的完全自动化的公共图灵测试)是当你填写网页表格时使用的弯弯曲曲的字母或单词。它们被系统用来确保你是一个人而不是一个机器人。虽然这些测试在阻止垃圾邮件方面是有效的，但它们的成功永远不会是一个长期的解决方案。

**1。可访问性**
大多数验证码会要求你从视觉上识别一系列符号。这对视力健全的人来说已经够难了，但对视力受损的人来说却是不可能的。一些系统确实提供了音频选择，但这只是给了黑客两种攻击方法。

**2。这不是图灵测试**
人类通常可以识别伪装成人的机器——这是图灵测试的标准定义。然而，验证码依赖于区分人类和另一台机器的机器。这是一个困难得多的命题，尤其是因为光学字符识别软件变得越来越好。

**3。所有验证码都能破解**
电脑会变得更快，软件会变得更复杂。所有的验证码最终都会被破解，这是必然的。

破解验证码已经成为许多黑客的一项有利可图的爱好。然而，人类的努力也同样有效:当任务可以外包给印度的数百名工人时，为什么要花费数千美元购买复杂的软件呢？

**4。验证码越来越难**
破解验证码的简单方法就是让测试变得更难。你有多少次验证码测试失败了？有些变得异常困难，许多替代方案更糟，例如

*   字母上完全无法辨认的猫或其他动物(是的，rapidshare.com，我指的是你！)
*   没有鼠标就无法工作的可拖动对象,仍然可以被欺骗
*   比验证码更容易破解的简单问题，例如“1 加 3 的总数是多少？”
*   或者谷歌新的[图像旋转验证码](https://www.sitepoint.com/googles-new-capcha-the-only-way-is-up/)需要客户端编码，黑客大概有十分之一的机会随机旋转到正确的角度。

**5。验证码衡量能力**
验证码的根本问题在于它衡量的是能力:你解读一组相当难懂的字母的效率。然而，计算机已经有效地综合了人类的一些能力，并且将会有所改进。

也许检测人类行为更好？当大多数人完成一个在线表单时，他们向下滚动页面，点击方框，添加文本，暂停，突出显示部分，删除和重新键入部分。随机页面交互可以更好地反映人类活动？

尽管存在这些问题，验证码仍然经常被用作最基本的网络表单的第一道防线。他们应该是最后一个。
