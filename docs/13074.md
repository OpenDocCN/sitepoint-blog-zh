# 任天堂 Wii 的获胜者是…

> 原文:[https://www . site point . com/and-the-winner-of-the-the-Nintendo-Wii-is/](https://www.sitepoint.com/and-the-winner-of-the-nintendo-wii-is/)

我很兴奋(也有点嫉妒)宣布我们的 [SitePoint 读者调查竞赛](https://www.sitepoint.com/take-the-sitepoint-reader-survey-win-a-nintendo-wii/)的获胜者是…

伊万·索利亚

Ivan 是一名来自墨西哥的设计师、学生和企业家，他的兴趣包括运动媒体和视频游戏。他幸运地赢得了一台任天堂 Wii 游戏机和两张《吉他英雄 3》的拷贝。

伊凡改进 sitepoint.com 的建议是从近 5000 份参赛作品中挑选出来的。他是这样写的:

> 我非常希望看到一个 SitePoint 桌面或 web 应用程序，在那里可以找到一些针对 web 设计、编码和业务的工具。例如，拥有一个可以快速访问/交互参考部分的 CSS 编码器会很酷。

但是伊凡不仅仅停留在一个想法上——他有很多想法。例如，他有一些关于内容类型的想法，这将使他更经常地访问 sitepoint.com。

> 我认为这将是有用的，有一个每周特色的数字/网络艺术家，并看到他或她的作品。我更希望看到 SitePoint 发展出更精细的美感，而不是有那么多教程。(我写这篇文章是因为我认为 SitePoint 更像是一个“思考”或“哲学”网站，而不是实践学习——我们都知道去哪里学习我们最喜欢的设计/编码包。)

嗯，Ivan，我们实际上有更多的人要求更多的教程，所以我不认为你会看到我们在这方面有所放松。但我们肯定会采纳你的建议，进行更多的采访和设计案例研究，因为许多其他人也要求这种类型的内容。

当然，伊凡并没有就此止步:

> 我希望所有与业务相关的东西都放在一个地方。某种日志、日历或项目管理工具可以集成到一个在线 SitePoint 应用程序中——一个机会板，人们可以在这里找到一个陷入困境的程序员或设计师，并为一些自由职业项目进行合作；如果有新的工作/合作或业务相关事宜，主页顶部的消息中心会很有用。这个应用程序或新的部分将被称为我的网站或我的网站点。我们可以从 SitePoint 的主页注册并登录，也可以把它作为一个社交网络工具，来建立一个强大的网页设计师和程序员社区。

Ivan 的想法无疑是一个雄心勃勃的想法，但它呼应了一个反复出现的主题——你希望 SitePoint 变得更加可定制，更加社会化，并为网站提供更多让你完成工作的工具。

> 我对你的网站的另一个想法，一个强有力的想法，是建立一个新的部分。“如何做”或“案例研究”部分，你可以选择一个成功的、设计良好的或代码惊人的网站，并尝试一步一步地解释整个产品(或其总结版本)，重点放在编程、设计和业务/营销方面。通过这种方式，我们可以从真实的商业实例中学习，并将您的一些书籍或文章知识*(用于我们自己的项目)*。

这是我们收到的回复中的另一个反复出现的主题——独立的文章很有趣，但完整的从头到尾的教程更有助于让你提高技能，无论是技术、创意还是商业相关的技能。我将更多地谈论我们即将推出的一些举措，这些举措将在一定程度上解决这一问题。

在接下来的一周，我会发布更多关于调查结果的信息，但现在已经足够了。请和我一起祝贺伊万！
