# 目前来看，全面饲料在财务上很有成效

> 原文：<https://www.sitepoint.com/full-feeds-financially-fruitful-for-now/>

像大多数在线出版商一样，SitePoint 的大部分内容都有提要(RSS 和其他格式)。另外，和大多数在线出版商一样，SitePoint 也很难选择在自己的订阅源中放入多少内容:只放标题、摘要、片段还是每个条目的全部内容。最终，我们选择了妥协，我们的博客供稿提供完整内容，而我们的[其他供稿](https://www.sitepoint.com/advertise/)(文章、论坛等)。)提供更有限的内容。

罗伯特·斯科博今天阐述了全内容提要的优势。他的论点是这样的:

目前，在可预见的未来，RSS 聚合器的用户是超级用户、记者和世界上其他信息上瘾者。这些人一般不会点击广告，除非他们非常非常有针对性。因此，如果你的收入来自广告，你实际上并没有因为提供完整的内容提要，让这些读者绕过你网站上的广告而错过任何可观的收入。出于同样的原因，在你的订阅源中放广告通常不会奏效。

从愤世嫉俗的角度来看，如果这些读者不点击广告，你为什么还要迎合他们呢？因为他们为你产生流量，这就是原因。使用 RSS 聚合器的超级用户也使用社交书签服务，如 [del.icio.us](http://del.icio.us/) ，在 [digg](http://digg.com/) 等网站上发布链接，甚至在自己的博客上发表评论。所有这些活动都将你网站的链接放在那些*对广告做出*回应的大众面前。

简而言之，让世界上沉迷于信息的超级用户通过他们的 feed 阅读器轻松访问你的内容，你就会为自己赢得大量有针对性的点击流量。

乌托邦式的观点是，feed 阅读器正在走向主流采用，当这一天到来时，这一论点将不再站得住脚。但斯考伯认为——我也同意——那一天还需要几年时间。
