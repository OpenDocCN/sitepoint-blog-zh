# Linux 的最佳降价编辑器

> 原文：<https://www.sitepoint.com/the-best-markdown-editor-for-linux/>

使用 Markdown 语法是我喜欢的写作方式。我用它来为网络创建内容，并记下简单的笔记。这是一种简单、极简的格式化文本写作方法，效果很好，而且不会把你束缚在一个专有系统中。这与 Linux 的理念非常契合。

它正在流行。自从约翰·格鲁伯在 2004 年创造了它，它已经越来越多地被用在网上和各种类型的应用程序中。值得学习。你会在 Gruber 的 [Daring Fireball](https://daringfireball.net/projects/markdown/syntax) 博客和我们的[免费可打印减价备忘单](https://www.sitepoint.com/grab-our-free-printable-markdown-cheat-sheet/)上找到语法。

有很多针对苹果电脑和 T2 视窗的降价编辑器。Linux 有什么可用的？我将在下面列出你的主要<abbr title="free and open-source software">自由/开源软件</abbr>和非自由/开源软件选项，并在最后提出一些建议。

让我们开始吧。

## 13 个支持降价的文本编辑器

您可以使用您最喜欢的 Linux 文本编辑器来编写 Markdown。您可能会发现它对 Markdown 有本地支持，或者有 Markdown 扩展或插件可用。以下是一些例子:

1.  [Vim](http://www.vim.org/) 拥有 [vim-markdown](https://github.com/plasticboy/vim-markdown) 和 [vim-instant-markdown](https://github.com/suan/vim-instant-markdown) 插件，这些插件具有语法高亮和折叠功能。
2.  GNU Emacs 有一个 Emacs 包的[降价模式，包括快捷键和语法高亮。](http://jblevins.org/projects/markdown-mode/)
3.  Spacemacs 有一个[降价层](https://www.spacemacs.org/layers/+lang/markdown/README.html)来增加降价支持。
4.  [Bluefish](http://bluefish.openoffice.nl/index.html) 包含降价文件的语法高亮显示。
5.  [Eclipse](https://eclipse.org/) 有 [Markdown 文本编辑器](https://marketplace.eclipse.org/content/markdown-text-editor)插件，它包括文档大纲、折叠部分、预览、导出到 HTML、任务标签、自动换行和段落格式。
6.  [Gedit](https://wiki.gnome.org/Apps/Gedit) 包含降价文件的语法高亮显示。
7.  Geany 提供了带有预览和语法高亮显示的 [Markdown](https://plugins.geany.org/markdown.html) 插件。
8.  KDE 文本编辑器 Kate 支持 Markdown 语法高亮显示。
9.  Atom 支持现成的 Markdown，具有语法高亮和预览等特性。这个功能可以通过几个社区生成的包来扩展，包括 [Markdown-Writer](https://atom.io/packages/markdown-writer) 、 [Markdown-Scroll-Sync](https://atom.io/packages/markdown-scroll-sync) 和 [Markdown-Format](https://atom.io/packages/markdown-format) 。
10.  [Notepad++](https://notepad-plus-plus.org/) 允许你使用 [MarkdownViewer++](https://github.com/nea/MarkdownViewerPlusPlus) 插件添加降价支持。
11.  [Sublime Text 3](https://www.sublimetext.com/) (不是 FOSS)可以变成一个全功能的 Markdown 编辑器。[我们在这里向你展示如何](https://www.sitepoint.com/sublime-text-perfect-blogging-6-ways/)。
12.  Visual Studio 代码(不是 FOSS)提供了许多 [Markdown 扩展](https://code.visualstudio.com/docs/languages/markdown)。这些允许您预览和编译 Markdown 文件，并提供一个折叠标题的大纲视图。
13.  [VSCodium](https://vscodium.com) 是 Visual Studio 代码的开源版本，没有遥测和跟踪功能。

## 10 个基于文件的降价编辑器

许多 Markdown 编辑器像文本编辑器和 ide 一样工作，允许您编写或编辑单个文件。这些提供了在大多数文本编辑器中找不到的额外有用功能，包括:

*   语法突出显示和预览窗格
*   快捷键
*   将 Markdown 导出/转换为 HTML、PDF、ODT 等格式
*   不受干扰的写作环境
*   字数统计、文档统计和拼写检查
*   高级格式，包括表格和数学表达式。

## 那么，对你来说最好的 Markdown 编辑器是什么？

品种还真不少！哪个 Linux Markdown 编辑器最适合你？这取决于你的优先事项，以及你将使用该应用程序的目的。

让我们为您缩小选择范围:

*   如果你已经有了一个最喜欢的文本编辑器，探索你能做些什么来使它更适合降价。
*   如果你喜欢 GPL 许可的软件，可以从撇号、ReText、GhostWriter、Marker、Abricotine、Zettlr、MindForger 和 Boost Note 中选择。
*   如果你重视一个不受干扰的写作环境，那么在撇号、标记文本、代笔、断句、Typora 或 WordMark 之间做出选择。
*   如果你只是在寻找一个支持基本语法并有预览面板的典型 Markdown 编辑器，看看撇号、标记文本、重新文本、标记和显著。
*   如果您需要高级格式选项，包括数学和表格，那么标记文本、标记、KeenWrite、Abricotine、卓越、Typora、MindForger 和 InkDrop 可以满足您的需要。
*   如果你需要协作功能，看看 HedgeDoc，Boost Note，或者 Write！App。
*   这些应用也在 Windows 上运行:Mark Text、Ghostwriter、KeenWrite、Abricotine、卓越、Typora、Joplin、Zettlr、Boost Note、Obsidian、WordMark、InkDrop、Write App！
*   而这些也在 macOS 上运行:Mark Text，Abricotine，Typora，Joplin，Zettlr，MindForger，Boost Note，Obsidian，WordMark，InkDrop，Write！App。

最适合你的应用程序将符合你的需求，并可用于你的发行版。探索愉快！
