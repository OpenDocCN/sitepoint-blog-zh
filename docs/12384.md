# TweetNews 将 Twitter 和 Yahoo！新闻

> 原文：<https://www.sitepoint.com/tweetnews-mashes-up-twitter-with-yahoo-news/>

几天前，我们发表了一篇文章，描述了如何在推特上[找到突发新闻。我们的建议是观察像](https://www.sitepoint.com/how-to-find-breaking-news-on-twitter/) [Twopular](http://twopular.com/) 和 Twitter 搜索页面这样的网站上的热门词汇，然后使用 [Google Trends](http://www.google.com/trends/) 将它们与搜索和新闻趋势进行比较，以找到关于热门话题的文章。

雅虎的一个新项目！开发人员 Vik Singh 称 [TweetNews](http://tweetnews.appspot.com/) 已经走了自动化进程的一半。辛格的项目将 Twitter 和雅虎混搭在一起！使用雅虎新闻！让我们建立自己的搜索服务。辛格的混搭配对雅虎上的新鲜新闻！最近，相关的推文，所以搜索关于下周二[就职典礼的新闻](http://tweetnews.appspot.com/fresh?q=Inauguration)会出现最新的新闻和推文。

到目前为止，TweetNews 还没有自动跟踪 Twitter 上的热门词汇，所以你仍然需要 Twopular 这样的工具。然而，一旦你确定了热门词汇，TweetNews 可能会更好地找到关于该主题的文章，这些文章实际上是最新鲜和最相关的——而不是像谷歌新闻那样让你费力地浏览大量类似和重复的故事。至少理论上是这样。

不过，还是有一些问题。推文实际上似乎从来没有与手头的文章直接相关。例如，今天早些时候，一篇关于加州州长阿诺德·施瓦辛格参加美国当选总统巴拉克·奥巴马就职典礼的文章附有链接到博客 Mashable 上一篇关于 Joost 就职典礼视频流的文章的推文。TweetNews 似乎知道推文和文章都是关于就职典礼的，但推文中没有任何内容表明这个特定的新闻故事比其他新闻故事更重要。

此外，今天有一条推文简单地说，“我要去参加就职典礼！”出现在几乎所有通过搜索出现的新闻报道的相关推文中。与多个新闻故事相关联的推文在该服务上是常见的事情。

然而，尽管有缺陷，辛格的想法也有可取之处。“如果构建得当，Twitter 作为搜索新鲜度的排名信号可能会非常有用，”Singh [在博客文章](http://zooie.wordpress.com/2009/01/15/twitter-boss-real-time-search/)中写道。通过努力，TweetNews 肯定可以很好地揭露关于突发新闻的最相关和最新的新闻故事。

该网站的代码是开源的，可以在 Singh 的博客上下载。我们真的希望一些有事业心的黑客会接受辛格的创造并解决这些问题。
