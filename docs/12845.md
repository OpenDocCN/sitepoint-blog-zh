# 推特与国会

> 原文:[https://www.sitepoint.com/twitter-vs-congress/](https://www.sitepoint.com/twitter-vs-congress/)

根据 paidContent 的特里西亚·杜尔耶的说法，推特最近经历了某种复兴。流量达到了历史最高水平，其估值也达到了顶点，服务甚至比过去几个月更加稳定。然而，即使 Twitter 正在“赢得微博军备竞赛”，正如[新闻周刊所言](http://www.newsweek.com/id/145216)，Twitter 可能不会出现在美国政府的大厅里。

在过去的几周里，这个微博网站在美国国会大厦引起了一场小骚动。因为众议院的规则规定，来自国会议员的任何官方通信都必须如此标记，所以通过 Twitter 与选民沟通——每条消息 140 个字符的限制——变得不可行。进一步提出的关于可能播放政治广告的网站的通信规则更是火上浇油。

对于拥有自己推特账户的德克萨斯州共和党众议员约翰·卡尔伯森来说，马萨诸塞州民主党众议员迈克尔·卡普亚诺提出的规则是不可接受的。这些规定旨在“防止成员使用公共资金在带有商业和政治广告的外部网站上进行交流”，但 Culberson 声称，这几乎会扼杀他通过 Twitter 与选民沟通的能力，因为寻求国会批准发布推文与 Twitter 的即时吸引力背道而驰。卡普阿诺表示，这些规定只是针对视频发布，而不是任何形式的书面交流。

不管怎样，Culberson 和俄亥俄州的民主党众议员蒂姆·瑞恩(他也在推特上发消息)一起，支持在 LetOurCongressTweet.org 的 T2 通过推特发起在线请愿。该网站呼吁国会“接受我们已经使用的通信技术”，而不是给推特增加障碍。任何使用#LOCT08 标签的人都会在请愿书上签名(“签名”由 Summize 编制，该公司现归 Twitter 所有)。到目前为止，只有 600 多人签名。

“公民正在克服他们对在线参与的恐惧，国会应该效仿，”无党派团体[阳光基金会](http://blog.sunlightfoundation.com/2008/07/08/member-web-use-reconsidered/)说，该组织发起了“让我们的国会发推特”请愿书。当涉及到接受消费者通信技术时，大型政府似乎很少快速行动，因此毫不奇怪，像使用 Twitter 与选民聊天这样看似良性的事情会陷入官僚主义。

一个有趣的转折是，众议员卡普亚诺在自己的选区可能面临与“[互联网候选人](https://www.sitepoint.com/is-the-world-ready-for-digg-style-government/)”的竞选连任活动——毫无疑问，这位候选人会发现接受 Twitter 和其他形式的在线交流是必要的。
