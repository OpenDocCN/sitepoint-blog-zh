# 设计元素:类型

> 原文:[https://www.sitepoint.com/elements-of-design-type/](https://www.sitepoint.com/elements-of-design-type/)

在设计元素系列的最后一部分，我们来看看字体。我把字体留到了这个系列的最后，因为在过去的几周里，我一直在写关于字体和它们所属的类别的文章。(更多[旧体](https://www.sitepoint.com/the-old-style-typeface/)、[现代](https://www.sitepoint.com/the-modern-typeface/)、[板型衬线](https://www.sitepoint.com/big-bold-and-sometimes-beautiful-serif-slab-fonts/)、[无衬线](https://www.sitepoint.com/the-sans-serif-typeface/)、[手写体](https://www.sitepoint.com/the-script-typeface/)、[装饰](https://www.sitepoint.com/the-decorative-typeface/))见这些帖子)。通过识别一些宽泛的字体类别，你可以做出明智的决定，为你的设计选择哪种字体，无论是在网页上还是印刷上。

字体是任何设计的基本组成部分。它可以作为文本阅读，作为一个形状和视觉元素，字体的选择传达了一种情绪或心情。如果使用得当，打字不需要照片或插图来支持。

**营造气氛**

你选择的字体可以传达很多情感。看下面四种不同字体的单词。

![Moods](../Images/0e1feae927a6ab11cec9c9f34d68cf76.png "Moods")

我们可以阅读相同的文字，但字体会引起不同的感受，有些字体比其他字体更合适。顶部的字体是一种软脚本字体，它(对我来说)传达的是爱而不是恨。第二种字体是黑色字母字体。黑色字母字体有着悠久的历史，从纳粹到黑帮说唱歌手都与它联系在一起。下面的第三种字体是一种叫做 Avenir 的无衬线字体，它给这两个词一种相当中性的感觉。最后，第四种字体是一种肮脏、扭曲的字体，它给仇恨这个词增加了更多的含义。

**选择哪些字体？**

这个问题没有简单的答案，但是使用和选择字体的一个好的经验法则是每个设计坚持两种或更少的字体。当谈到字体克制是一件好事。永远记住**少即是多**。选择两种**对比**字体即可达到趣味。例如，在你的网站上，你可能会在标题中使用脚本或者蹩脚的字体，而在正文中使用无衬线字体。你可以使用图片作为你的标题(我知道，不是很友好的搜索引擎)，或者使用诸如 T4 sIFR T5、T6、T7 和 T8 Cufon T9 这样的技术。

如果你想坚持网页安全字体，你可以通过使用衬线和无衬线字体来实现对比。单片眼镜网站就是这么做的。

![Monocle](../Images/b6f42040e586e904f3218db5dc80ccb2.png "Monocle")

英国政府认为广告对比衬线字体和彩色的童趣字体是一个非常有效的竞选海报。

![think-mobile](../Images/9af2de3f0f875a96880a3ef09efd2e1a.png "think-mobile")

**字体大小**

选择字体大小时，你需要考虑你的受众。形式服从功能。如果你正在为一个 60 岁以上的社区创建一个网站，避免使用 8 号和 10 号字体可能是一个好主意，因为这对许多读者来说阅读起来会很麻烦。另一方面，太大的字体会压倒设计，让它看起来不那么复杂。

在过去的几年里，大字体标题已经成为网络上的一种流行趋势，一些设计师选择大字体标题来与小字体正文形成对比。

[巨型创意](http://madebygiant.com/)

![GiantCreative](../Images/987686dd2ab4c91f58996243b86b450f.png "GiantCreative")

大赦国际的戏剧性海报使用非常大和非常小的字体叠加在一个鲜明的黑白图像上，以表达观点。你需要仔细看这张海报来阅读所有的文字，但第一眼看上去，它清楚地传达了令人震惊的信息。

![amnesty-international-darfur-rape](../Images/043375386487eb48587e36083d7bee0c.png "amnesty-international-darfur-rape")

**键入一个形状**

“正文”指的是较大篇幅的文本，而不是简短的标题和引用。当正文被添加到页面上时，文字会呈现一种形状，在网站上通常是一个简单的矩形。形状的强度取决于字体的大小、字体和文本的布局。在下面的杂志布局中，正文的两列完全对齐，并环绕标题中的圆形&符号。文本在美学上是令人愉快和有吸引力的，在主要标题上面有大量的空白，允许喘息的空间。

![winterhouse](../Images/3621f332f351a864523940a5dd610d98.png "winterhouse")

**时尚与永恒**

去寻找最新、最酷、最时尚的字体是很有诱惑力的。有这么多免费和便宜的字体可用，有数千种字体可以使用。其中一些非常“时髦”，但明年它们可能看起来非常过时。这并不是说你不应该使用它们。只是要意识到你为什么要使用它们。如果你是设计新手，你也会发现随着你了解的越来越多，你的品味也会改变。

![Dated](../Images/7de897c080dafbe8f8c22ada85fc033d.png "Dated")

上面是半个头，下面是伯恩哈德摩登。Hemi Head 有点像《星际迷航》,如果你正在设计一个复古的空间设计，它可能会很有用。伯恩哈德摩登看起来更加精致经典。一些最古老的字体是流传了数百年的真正经典。它们已经存在了那么长时间，而且还在继续使用，因为它们设计得很好，很容易辨认。随着时间的推移，你会发现实践和经验会告诉你选择哪种类型。

总而言之，在使用字体作为设计元素时，需要考虑的一些事情是:

1.  我使用的字体合适吗？这与我传达的信息一致吗？
2.  如果我使用了不止一种字体，它们是对比的还是冲突的？他们合作得有多好？
3.  尺寸是否适合媒体和信息？
4.  我用的是一种永恒的经典字体，还是这种字体会相当于下周 80 年代的垫肩？
