# 民意测验:易贝能扭转局面吗？

> 原文:[https://www.sitepoint.com/poll-can-ebay-right-the-ship/](https://www.sitepoint.com/poll-can-ebay-right-the-ship/)

易贝被夹在岩石和坚硬的地方之间，他们的股价反映了这一点，今天创下了五年来的新低。[根据](http://techland.blogs.fortune.cnn.com/2008/10/02/market-bidding-isnt-going-ebays-way/)摩根士丹利分析师大卫·约瑟夫的说法，他今天下调了易贝的评级，易贝表现不佳的原因是无法与其他在线卖家“在买家体验方面竞争”。约瑟夫说，买家现在寻求的是简单的购买体验，而拍卖形式不是这样。

约瑟夫说，人们还喜欢并期待免费送货和一键订购等功能，而易贝并没有实际控制商品的销售，因此没有多少余地。不足为奇的是，根据摩根士丹利的数据，易贝在电子商务市场的份额在过去几年下降了 2%，而亚马逊的份额增长了 1.5%。

尽管他们在转变买家态度方面遇到了麻烦，但易贝最大的麻烦可能是卖家——网站的命脉。正如我们在 8 月份报道的那样，易贝取悦买家的举措并不受卖家欢迎，这些举措包括更加强调固定价格拍卖和政策变化，如关闭对买家的负面反馈。许多卖家觉得易贝失去了灵魂，现在正在寻找其他地方销售他们的商品。

手工产品的供应商正转向像 Etsy 这样的网站，而易贝最初赖以生存的古董、二手物品或收藏品的卖家正被推向像 T2 Craigslist 这样的分类网站。与此同时，固定价格卖家有机会在其他网站销售，如亚马逊，它甚至也可以处理履行。

因此，给你一个问题:易贝能拯救自己吗？
