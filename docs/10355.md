# 2011:粉色之年

> 原文:[https://www.sitepoint.com/2011-the-year-of-pink/](https://www.sitepoint.com/2011-the-year-of-pink/)

每年，Pantone 都会预测下一年的流行色。他们预测蓝绿色会是 2010 年的颜色。今年，他们几乎直接跨过色轮，宣布 2011 年将是[金银花年(Pantone 18-2120)。](http://www.pantone.com/pages/pantone/Pantone.aspx?pg=20821&ca=4)

随着金融厄运和悲观情绪弥漫全球，潘通建议我们“让勇敢的金银花精神注入你，提升你，带你度过这一年。”一个健康的人有时被称为“身体健康”；内容和快乐的人被描述为“痒得要命”；因此，粉红色被视为积极的颜色也就不足为奇了。

[![honeysuckle-swatch](../Images/d22cd6b3534d10528135b953a37e3d08.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/12/honeysuckle-swatch.png)

关于选择金银花作为 2011 年的颜色，潘通色彩研究所的执行董事 Leatrice Eiseman 说，

> 在有压力的时候，我们需要一些东西来振奋精神。金银花是一种迷人、刺激的颜色，可以让肾上腺素上升——完美地抵御忧郁……金银花的积极品质来自于其母色红色的强大结合，红色是光谱中最自然、最生动的色调。

[![HonesuckleFashion](../Images/592a7ec9665dc89df8b2b82968a610f3.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/12/HonesuckleFashion.png)

如果您想在您的网站上使用这种颜色，十六进制的 RGB(近似)等效颜色是#E56284。而且，如果你喜欢粉色，你可能也会对我今年早些时候写的一篇关于[粉色在设计](https://www.sitepoint.com/color-in-design-pink/)中的文章感兴趣，这篇文章讨论并举例说明了设计师如何在标志设计、印刷设计和网页设计中使用粉色。与金银花混合搭配——黑色、木炭色、藏青色和各种灰色都可以很好地与粉色搭配，绿色也可以。

[![LilyBalouWebsite](../Images/12423a9545866880e16fcb1e50a2944f.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/12/LilyBalouWebsite.png)

就我个人而言，我曾经鄙视粉红色，但随着年龄的增长，我越来越喜欢它和它乐观的魅力，我喜欢今年的粉红色。

你喜欢这种颜色吗？你会在 2011 年的设计中加入金银花粉吗？
