# 网页导航:设计用户体验

> 原文:[https://www.sitepoint.com/designing-user-experience/](https://www.sitepoint.com/designing-user-experience/)

奥莱利以其大量面向技术人群的内容厚重的书籍而闻名，同时也是第一个意识到设计网络导航系统信息匮乏的主要出版商。为了填补这个空缺，他们发布了网络导航:设计用户体验，零售价为 34.95 美元。自 1998 年出版以来，经过多次再版，它仍然是关于这个主题的最好的包罗万象的书之一。

这本书的作者 Jennifer Fleming 在这本 251 页的综合书中做了出色的工作，从根本上解释了如何建立一个成功的网络导航系统，并引用了评论和截图，解构了当今一些领先的网站。

这本书展示了一个有效的网站不仅仅是干净、紧凑的代码和加载速度快的漂亮图形。如果没有一个可用的界面，网站的其他内容都是没有价值的。为 web 设计一个用户友好的界面需要一种全新的思维方式和一套新的技能，这与实际创建一个漂亮的页面所需的技能有很大的不同。詹妮弗·弗莱明在展示成功的用户界面所需的知识方面做得非常出色。

这本书的前半部分详细介绍了导航设计的过程和你应该为之奋斗的目标。

这本书的作者宣扬一种“以用户为中心”的方法，这种方法专注于满足特定受众的目标和需求。问自己一些问题，比如“我在哪里？”“我怎样才能回到我曾经在的地方”被推荐作为把你自己放在用户的位置上的一个步骤。Jennifer Fleming 还建议，创建将访问你的网站的人的类型、他们的目标和互联网体验的档案，作为建立以用户为中心的导航系统的一种方式。没有观众的参与，成功的导航系统是无法建立的。幸运的是，这本书的大部分注意力都放在了用户测试上。诸如您应该在项目的哪个阶段进行测试、谁应该参与测试、谁应该进行测试以及如何评估测试结果之类的问题会被详细讨论。

根据该书作者的说法，成功的导航取决于十种品质。这些品质包括“保持一致”、“提供反馈”和“适合网站的目的”。为了帮助读者学习如何应用成功导航的十个品质，每个要点都通过一个例子和一个真实网站的截屏来加强，这些例子和截屏严格遵循特定的原则。

贯穿整本书的是边栏，它提供了网站和书籍、经验法则和采访的便捷参考，以及十大列表，如“吓到普通用户的十大请求”。所有这些边栏提供快速阅读，很好地补充了书的内容。

无论是在网络上还是在现实生活中，对事物进行分组都是为了提供实现预期结果的逻辑路径。不幸的是，问题出现了，因为关系不是一成不变的，谁说牛奶不应该出现在饮料区？

这本书的作者提出了一个六步流程，从信息收集开始，到维护和增长规划结束，作为一种内容组织的方式，让您的用户可以毫不费力地找到他们正在寻找的内容。

经过六个章节的启发和实用信息，这本书进入了一系列章节，其中包括一系列关于不同类型网站导航设计的案例研究。唉，书中提到的许多网站都经过了重新设计，这使得人们不可能一边注意书中提到的导航设计元素，一边上网冲浪。

如果你参与创建网站，无论是作为一名程序员、设计师还是项目经理，我建议你拿起这本书。本书前几章提出的概念是永恒的，只要互联网存在，它们就会一直适用。

**评级:4/5**
