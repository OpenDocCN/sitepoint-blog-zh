# RubySource:我喜欢辛纳屈的 7 点

> 原文：<https://www.sitepoint.com/rubysource-7-things-i-love-about-sinatra/>

几年前我第一次听说 Sinatra，从那以后我就爱上了它。对于那些没有听说过的人来说，Sinatra 是一种用于创建 web 应用程序的 DSL。它是用 Ruby 编写的，最初是由 Blake Mizerany 设计的，但是现在有一个核心团队负责维护和开发代码。如果你想知道，它是以 Frank Sinatra 命名的，显然是因为他有如此多的课程，所以他值得一个以他的名字命名的网络框架。
