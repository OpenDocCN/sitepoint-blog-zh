# 这是一个美国节日…我很高兴今天能工作

> 原文：<https://www.sitepoint.com/working-holiday/>

今天是美国的阵亡将士纪念日，我决定花一天的大部分时间在电脑前。我对这个事实并不生气、怨恨或失望。原因如下。

## 我做出了选择。

几年前，当我决定为自己工作时，我有一个模糊的想法，这将意味着做出许多牺牲。这种意识在拥有企业几个月后变得更加清晰，就像许多企业家一样，当我面对一个企业启动过程时，包括身兼数职，不得不做每件事的一小部分。很多时候，小企业的成功转化为简单地投入时间和精力使其成功。

现在我的生意已经建立，我的选择已经转移到决定我什么时候想工作，是的，有时这意味着假期工作。另一方面，如果我想在所有同事都工作的时候休假一整周，我可以。当然，这只是需要一些认真的预先计划和移动接入。但最终，选择权在我。

## 另一种选择对我来说更不可取。

像预定一样令人愉快(付费！假期和假日伴随着相对稳定的雇员职位，但一想到我不得不去一份朝九晚五的工作，在那里我只是公司机器上的一个小齿轮，我就感到害怕。无意冒犯那些对自己的全职工作完全满意的人，因为我知道有很多，我的立场是，如果能让我不再为别人工作，我会在每个假期都工作。

## 我不是一个人。

嗯，我现在独自一人在办公室里(除非我的孩子突然来看我)，但当我知道我的许多美国企业家同事今天也在工作时，我有一种巨大的团结感。此外，还有美国以外的其他人，对他们来说，今天只是又一个星期一。在网络空间里，我绝对不是一个人。

## 我开始享受一个简短但富有成效的工作日。

我决定今天要工作 3-4 个小时，然后出去享受烈日、家庭时光和美食。这是一个简单的工作时间表，因为我的许多客户都休息了一天，我可以专注于清单上最大的项目。这 3-4 个高度集中的小时会让我剩下的一周轻松很多。我将在本周晚些时候获得一些额外的休息时间，因为我花了这些时间来取得进步。

所以回到工作中，我今天又多工作了几个小时…这很简单，因为我真的很喜欢我的工作。

你曾经决定在假期或周末工作吗？你觉得收益大于牺牲吗？
