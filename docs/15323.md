# 谁拥有你网站的版权？

> 原文:[https://www.sitepoint.com/who-owns-copyright-website/](https://www.sitepoint.com/who-owns-copyright-website/)

所以你是一个作家、出版商或艺术家，渴望表达自己。你不怕麻烦，花钱去寻找，然后付钱给一个专业的网站设计师，把你的电子世界拼凑起来。现在你可以告诉任何人你想告诉任何人的事。问题是:谁拥有所有人将要看到的东西？你心里的这一块其实是属于你的吗？

在没有有效的书面合同来解决这些问题的情况下，情况实际上并不像你想象的那么简单。但是在法律行业什么是简单呢？

##### 内容版权问题

以下是一些未解决的问题:

***1。什么是设计，什么是内容？**T3】*

我们必须首先区分设计的版权——网站的“外观”——和内容的版权。颜色、gif、jpegs、设置、超链接和其他有助于设计的元素最初都归创建者所有，在这里是指网站设计者。但即便如此，这也不是板上钉钉的，因为你无疑参与了潜在的创造性决策——因此，或许版权是共同拥有的。

这就是美国版权法定义的合作作品:

<q>*“联合作品”是指由两个或两个以上作者准备的作品，目的是将他们的贡献合并成一个整体中不可分割或相互依赖的部分。*T3】</q>

但是正如你所看到的，如果对最终产品——你的网站——的贡献是可区分的，那么这些独立的元素可能有独立的版权所有者。不仅如此，你的贡献必须超过最低限度，这样你才有资格成为一名合著者。合著的许多后果超出了本文的范围。

***2。内容来自公共领域***

此外，就 gif、jpegs 和其他元素是从网上下载的而言，即使假设它们本身没有版权限制(情况并非总是如此)，设计者和你都不能对已经存在于公共领域的内容主张版权。另一方面，在一定程度上，这些图形是由网站设计者创作的，这再次表明该设计者最初拥有其中的版权。但是，如果它们是你的创作、你的标志或你的照片，那么大概你拥有它们的权利(当然，除非照片是由其他人拍摄的，在这种情况下，摄影师拥有其中的版权，你必须获得许可才能在你的网站上使用它们)。我告诉过你——这并不简单。

***3。正文***

让我们继续这个网站的其他内容:文本。显然，如果文本是你创作的，你就拥有它的版权。但这不仅仅是出现在网上的文本的问题，也是文本以 HTML 或其他编码的形式设置的问题。如果设计者创建了该编码，那么与文本本身不同，该编码可能归设计者所有。编码中的版权概念包括设计者在网站中创建超链接的方式。因此，你可能拥有文本，但设计者可能拥有文本在网站上的设置方式。请记住，现在还不清楚编码是否可以获得版权。它可能不受版权法保护，但在某些情况下可能受专利法保护。

网站上可能还有许多其他受版权保护的材料。这些元素的版权归属问题取决于谁在什么情况下创造了它们。

##### 该决议

在概述了一些但肯定不是所有的问题之后，有两种潜在的协议可以用来解决这些问题:“职务作品”协议和“版权转让”协议。

***1。**一份为雇佣而制作的作品协议*

为了达成有效的雇佣作品协议，必须满足几个标准:

1.  如果设计师是你公司的合法雇员，那么他或她在受雇期间创造的一切都属于你。你拥有版权。
2.  另一方面，如果你雇佣了一个独立的承包商来设计网站，那么必须有一个有效的，由双方签署的书面协议。但请注意，它必须同时写入有效的*和*。要成为*有效的*雇佣协议作品，你必须“特别订购或委托”该作品(想必你已经这样做了)，并且书面协议必须声明这是雇佣协议作品。还有一个重要问题是，该协议必须在何时执行——是在工程实际开始之前还是之后。你不能通过雇佣协议的事后尝试来获得网站的权利。没有这回事。

网站设计作品还必须属于法定类别之一，才能被视为职务作品。为了这篇文章，我假设雇佣设计师是“对集体工作的贡献”，这是雇佣协议中允许的工作的法定类别之一。在这方面，我认为这可能是一项集体工作，因为你创造了某些元素——例如文本——而设计师贡献了设计元素。在某些情况下，它也可以是视听作品或汇编。在其他情况下，工作可能不属于这些类别中的任何一个，也不属于雇佣协议中允许工作的任何其他法定类别。在这种情况下，你不能依赖雇佣合同，你必须求助于版权转让来转移权利。

所以让我们假设当事人之间没有满足雇佣工作要求的书面文件。通过让设计者执行一份书面的版权转让协议，将所有这些权利从设计者转移给你，你或许可以获得设计者在网站中的所有权利。然而，请注意，在某种程度上，设计者没有创建材料(如以 gif 等形式)。来源于上述公共领域)，您不获得其中的任何权利。

你会想获得该网站的所有权利，而不仅仅是使用它作为你的网站的权利。因此，如果你没有书面的，有效的协议来传达给你所有这些权利，你可能最终充其量只是一个使用你自己的网站的许可。这不是特别令人满意，因为你可能会失去宝贵的权利。这毕竟是你，你对自己的表达，你应该自由自在地使用你内心的表达。你必须始终意识到，电子问题的这个领域在不断发展，今天看起来是一个网站的东西，明天可能是一个视频游戏或一部长篇电影。

但是，请注意，这对设计师来说并非不公平，因为设计师可以在他或她的费用中加入他或她认为适合全部“买断”的费用。这样双方都能赢。

除了基本的版权问题之外，还可能涉及其他问题。这些其他问题可能会涉及到设计师是否可以将他或她的标志放在网站上，或者设计师是否可以使用网站的副本作为他或她的作品集的一部分。这些问题更多的是合同问题，而不是版权问题，尽管它们可能来自版权问题的答案。

如你所见，在这个新领域没有简单的答案。最明智的建议是签订一份书面协议，解决这些以及其他可能出现的问题。
