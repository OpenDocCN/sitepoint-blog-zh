# 奥斯卡获奖网站？

> 原文:[https://www.sitepoint.com/oscar-winning-websites/](https://www.sitepoint.com/oscar-winning-websites/)

2011 年奥斯卡提名今天公布了，我想简单地浏览一下获得最佳影片提名的电影的网站，看看这些网站是否值得一去。所以，不分先后，敲鼓吧，以下是最佳影片提名者的网站。

黑天鹅网站没有太多值得大书特书的内容。这是福克斯探照灯网站的一个分支，以一个小的 Flash 演示为特色，给出了电影的概要，画廊，下载，视频和演员及工作人员信息。每一部分都以电影中的一个简短场景开始。在 Flash 演示的下面，有一个 HTML 部分，看起来信息太多了，显然是“容易”更新的部分。

[![BlackSwan](../Images/a08f7de33e968bd52b22543aa7d3181a.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/BlackSwan.jpg)

[战斗机](http://www.thefightermovie.com/)大部分功能与前场相同，但整体观感优于黑天鹅。大图片和低调的配色方案相结合，给网站带来柔和但复杂的外观。

[![TheFighter](../Images/72ffdfbf391145dcc929e21edea29d78.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/TheFighter.jpg)

《盗梦空间》是一个基于 Flash 的网站，有漂亮的图片和一个可以改变每个部分位置的菜单。我发现加载每个部分都很慢(但这可能是我的连接)，这让我想知道等待是为了什么。

[![Inception](../Images/bfb04b8489b3f196b3733272858f49e9.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/Inception.jpg)

[孩子们都很好](http://www.focusfeatures.com/the_kids_are_all_right)网站不会用花里胡哨把你淹没；它只是普通的 HTML 和 CSS，但它确实有明亮的蓝色和黄色配色方案。

[![KidsAreAlright](../Images/a07df8fc392e4b7cdcc764e141944347.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/KidsAreAlright.jpg)

国王的演讲是另一个简单的 HTML 网站。我认为他们甚至可能在菜单和正文中使用 Times New Roman 字体。顺便说一句，这部电影真的很棒，如果你还没看过，就去看吧。

[![KingsSpeech](../Images/da3521ce9f4af108540651f286d1f749.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/KingsSpeech.jpg)

[127 小时](http://www.foxsearchlight.com/127hours/)是另一个福克斯探照灯子网站，有 Flash 演示。部分之间的电影剪辑按钮是一个很好的接触。

[![127Hours](../Images/b1a330f6662a07de3c5d6bba7a3302ba.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/127Hours.jpg)

[社交网络](http://www.thesocialnetwork-movie.com/)使用 Flash 构建，并已更新以展示其获得的奖项和提名。图片和配乐被用来在网站上产生很好的效果。

[![TheSocialNetwork](../Images/0d3bbf566857f28c7b92beab3e51a6d6.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/TheSocialNetwork.jpg)

正如所料,[玩具总动员 3](http://disney.go.com/toystory/#/characters) 网站非常丰富多彩，有许多儿童活动、下载和出售的“东西”。

[![ToyStory3](../Images/71f1b1085672fd029927b117b6419739.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/ToyStory3.jpg)

网站上有很棒的约翰尼·卡什原声音乐，主页上有一些漂亮的西方风格的排版，除此之外没有什么突出的报道。

[![TrueGrit](../Images/09d672c6ee6418ba14b7f137951d84a8.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/TrueGrit.jpg)

《冬天的骨头》使用单一的摄影背景，只有文本在屏幕中央变化。这偶尔会导致一个可怕的滚动条出现——一个明确的设计

[![WintersBone](../Images/1839d0917a75bdb2b787b19eb3acbb06.png)T2】](https://www.sitepoint.com/wp-content/uploads/2011/01/WintersBone.jpg)

你觉得这些网站怎么样？鼓舞人心还是乏善可陈？如果有一个奥斯卡最佳网站奖，你认为这些网站有价值吗？
