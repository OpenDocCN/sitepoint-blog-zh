# 扩展您的服务的蓝图

> 原文：<https://www.sitepoint.com/a-blueprint-for-expanding-your-service-offering/>

有时，你的主要客户会推动你在可能成为你业务的新专业领域发展技能。但你需要为此做好准备。

当一大群客户开始来到我的公司门口，要求获得社交和移动应用程序时，我们已经在社交媒体领域拥有了技能和能力，这并不是一个令人高兴的意外。我们已经完成了该领域的一系列工作，并计划在这一领域扩大我们的产品。当那些客户开始敲门时，我们已经准备好了。

### 提前计划

你如何在这些[市场趋势](https://www.sitepoint.com/top-5-web-technologies-2010/)到来之前做好准备？我们测试新业务领域的方法是，首先，找到可以尝试的方法。然后，如果我们相信我们可以在那个市场空间交付，我们会想出如何创建一个团队成员的人才库来交付我们旨在赢得的项目。

大约五年前，我们可以看到社交网络将会成为一件大事。当时，我们尝试了一下，找到了一些有实力的主要客户，我们可以与他们合作来提高我们的能力。然后，当社交网络工作的海啸最终袭来，企业正在寻找在该领域有经验的提供商时，我们将是他们显而易见的选择。

当我们准备那一年的预算时，我们预计社交媒体领域将会升温。因此，当我们做出好年景、好年景和好年景预测时，我们预计了每种情况下的人员配备和技能要求。然后，当工作海啸袭来时，我们有了应急计划，可以去寻找优秀的人才，满足客户的需求，并按计划发展我们的业务。

### 试水

对于其他想法或市场空间，我们称之为我们的铁杆——我们相信它们有潜力，但我们不确定它们是否会在未来 12 个月内强势涌现——我们考虑现有团队，并确定谁应该在这些领域有技能。然后，我们走出去，寻找能够在这些领域与我们合作的主要客户，让我们大展拳脚。然后，如果其中一个环节真的开始升温，我们有人才来满足过渡时期的需求，同时我们也在寻找资源和发展这一业务领域的方法。

在确定可能拓展业务的领域时，从长计议非常重要。虽然我们 85%的重点将放在今天的业务上，但我们仍然花时间寻找我们业务的未来增长动力，并研究如何通过领先客户关系在该领域进行试验。

未来两三年，你认为对于你的企业经营的领域，游戏会有怎样的变化？
