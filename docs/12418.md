# IE 的衰落让“跨浏览器”变得更加重要

> 原文:[https://www . site point . com/ies-decline-makes-cross-browser-more-relevant/](https://www.sitepoint.com/ies-decline-makes-cross-browser-more-relevant/)

![](../Images/a831eff1587618163a4bbce7eb0a3e68.png "browsers-dock")2004 年，当流行的在线计费网络应用 [FreshBooks](http://www.freshbooks.com/) 推出(当时叫做 2ndSite)时，浏览器市场[极度一边倒](http://marketshare.hitslink.com/report.aspx?qprid=0&qpmr=15&qpdt=1&qpct=3&qptimeframe=Q&qpsp=23)。超过 91%的网络用户通过 Internet Explorer 连接。Firefox 的市场份额略高于 3.5%，Netscape 低于 3.5%，Safari 甚至不到 2%。FreshBooks 推出仅支持 Windows 和 IE 是可以的。

实际上，4 年前的 web 应用程序不兼容跨浏览器或跨平台是相当普遍的，这似乎很荒谬，因为在任何地方、任何计算机上工作的能力是 web 应用程序的一个主要卖点。但是，当 10 个潜在客户中有 9 个——在某些市场领域甚至更多——都在使用 Internet Explorer 时，为什么还要投入资源为其他浏览器设计呢？(请注意，2004 年，超过 96%的网络用户使用的是 Windows 操作系统，相比之下，现在这一比例约为 88%。)

快进到今天，风景看起来[完全不同](http://marketshare.hitslink.com/report.aspx?qprid=0&qpmr=15&qpdt=1&qpct=3&qpcal=1&qptimeframe=Q&qpsp=39)。IE 的市场份额不到 70%，Firefox 超过 20%，Safari 超过 7%，尽管新来者 Chrome 的份额不到 1%，[我们预测](https://www.sitepoint.com/whats-on-tap-predictions-for-2009/)谷歌的浏览器明年将大幅增长。这个搜索引擎已经在 Google.com 和 YouTube 上放置了 Chrome 链接，并为 IE6 用户在 Gmail 中放置了 Chrome 链接，另外这个浏览器可能会在明年[开始预装在个人电脑上](http://blogs.zdnet.com/BTL/?p=10974)。

尽管 IE 仍然在浏览器市场占据绝对领先地位，但 2009 年 Firefox、Chrome 和 Safari 应该会继续增长(尽管 Chrome 一旦推出扩展功能，可能会减缓其他替代浏览器的增长速度[)。与此同时，预计 ie 浏览器将继续下滑。随着浏览器对等性的接近(至少在用户范围方面)，跨浏览器和跨平台功能变得越来越重要。Firefox 和 Safari 已经占据了不容忽视的市场份额，Chrome 可能很快就会达到这一点。](https://www.sitepoint.com/once-extensions-arrive-on-chrome-firefox-should-worry/)

这对消费者来说是个好消息，他们有了更多的选择，对网络开发者来说也是喜忧参半。一方面，这意味着要测试和开发更多的浏览器。然而，另一方面，开源浏览器的日益流行意味着采用开放标准的趋势。竞争推动的开发意味着更好的渲染引擎、更快的 JavaScript 解释器等等。，这反过来意味着更好的产品。不仅仅是浏览器，更好的网络应用也将是浏览器竞争加剧的结果。

IE 的衰落可能意味着需要更多的初始开发，以确保最大数量的用户可以访问你的网站或应用程序，但从长远来看，这对每个人都是一件好事(可能除了微软)。
