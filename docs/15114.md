# 社区建设入门

> 原文:[https://www.sitepoint.com/community-building-primer/](https://www.sitepoint.com/community-building-primer/)

你听说过“耐心是一种美德”这句话吗？令我懊恼的是，我发现很多网站管理员在社区建设时并没有实践这一理念。在这篇文章中，我将告诉你在建立你的社区时应该避免什么，以及如何逐步建立一个成功的社区，让你从中获益。

假设你正在为模型火车爱好者建立一个爱好网站。从你发布网站的那一刻起，你就在想办法给它添加内容，并最终围绕它建立一个忠实的社区。显然，如果你是 [SitePoint 的社区论坛](hppt://www.sitepointforums.com)的成员，你会亲眼目睹一个社区有多成功。

那么，是时候加入社区潮流，为模型火车建造者创建自己的论坛了吗？首先，我们必须检查目前市场上的论坛产品。然后，我们将考虑您在构建社区时将面临的主要挑战。最后，你应该对一个社区需要什么有一个概念，以及你是否准备好建立一个社区！

##### 领先的论坛产品

有许多免费的产品可供使用，它们模仿了昂贵的商业替代品的特性。如果你刚刚起步，预算很低，你最好的解决方案可能是从一个免费的脚本开始。市场上最流行的脚本有 [phpBB](http://www.phpbb.com) 、 [IkonBoard](http://www.ikonboard.com) 和 [Invision Board](http://www.invisionpower.com) 。你可以用这些产品从头开始建立你的社区，然后如果你愿意的话，在以后的某个时候决定转向商业解决方案。

引领商业论坛产品的是 Jelsoft 的 vBulletin ，它目前的终身许可费用为 160 美元，一年的租赁费用为 85 美元。一旦您的社区显示出潜力，并且已经发展到相当大的规模，您应该考虑迁移到功能强大、功能丰富的 vBulletin。如果你真的想经营你的社区，这款产品正适合你。

##### 要问自己的问题

***1。我应该用论坛发布我的新网站吗？**T3】*

许多网站管理员和开发人员常犯的一个错误是，一打开网站就添加论坛或聊天室。他们希望他们的游客蜂拥而至，成为“去”这个词的常客。不幸的是，社区不是一夜之间建立起来的，这让许多网站管理员感到失望和沮丧。

无论你怎么看，社区成长是一个需要努力的持续过程。如果你经常浏览互联网，你很可能会偶然发现“Joe 的主页”,它提供了一个几乎不可见的链接到他的论坛——一旦你到达那里，你会发现，巧合的是，他的论坛的帖子比 SitePoint 在五分钟内收到的帖子还少。显然，这不是一件好事——这不是建立社区的方式。

除非你有一个已经建立的，至少是稍微受欢迎的网站，否则你可能不会在你的新论坛或聊天室中得到任何重要的参与。耐心是这个渐进过程的关键。

***2。所有的网站都应该有论坛，对吗？**T3】*

不对。你的网站真的需要一个论坛或聊天室吗？如果这个网站是以研究为目的的在线网站，或者是一个目录或搜索引擎，那么这个问题的答案可能是“不”。

但是，可以开发一个社区来补充大多数站点。这取决于你网站的主题和内容的性质——这个决定必须在网站的早期就做出。你有时间和精力去维护和发展一个社区吗？从很多方面来说，建立一个社区就像在你原有的基础上建立第二个站点。在你开始添加任何社区功能到你的网站之前，你真的需要问自己这些问题。

***3。我不应该提供尽可能多的论坛吗？这不会让我的板子看起来又大又受欢迎吗？**T3】*

越大并不总是越好。在你的新社区中增加 67 个论坛并不会像你想象的那样吸引那么多的访问者——事实上，更有可能把他们赶走。人们首先希望看到的是活跃的论坛，而不是一堆空洞的类别。

让我们回到你的模型火车网站的例子——你真的需要一个为所有品牌的模型火车制造商的论坛吗？当你开始你的社区时，你需要从三个或四个普通论坛开始，这些论坛可以作为讨论的跳板。因此，在我们的例子中，一个讨论模型铁路建造技巧和技术的论坛，一个展示展示和创作的论坛，以及一个允许您的社区建立联系的一般聊天论坛，可能就是您开始时所需要的全部。

***4。设计？我以为论坛软件自带外观和感觉？**T3】*

一旦你选择了你的论坛产品并创建了你的初始论坛，建立你的社区的下一步就是定制你的站点的外观和感觉。SitePoint 很好地做到了这一点，他们论坛上的标题与他们网站网络的其他部分相匹配。

让你的论坛尽可能吸引人，无论这意味着添加一个自定义的标志，还是改变基本的颜色。在你的主网站上练习同样的网页设计基础，使你的论坛快速加载，搜索引擎优化，易于导航。

##### 推广和启动您的论坛

现在你已经建立了你的新论坛，并准备好了，让我们来谈谈一个小小的推广。在你的网站上贴一个论坛链接比你想象的要复杂得多。

如果你想让你的论坛有最大的曝光率，你必须有创意。例如，SitePoint 不仅在所有页面顶部的显著位置有一个到他们论坛的常规链接，而且在整个网站上有各种链接，包括在他们的时事通讯中，在每篇文章之后，以及在主页上有一个最新热门话题的列表。虽然建立这种“网络”的链接可能会很乏味，但从长远来看，它会带来巨大的回报。

在推广方面要记住的重要一点是，成功不会在一夜之间到来——你必须保持耐心，成为你的论坛的积极参与者和推广者。毕竟:如果你不积极，为什么你的成员要参与进来？你必须树立一个榜样。

说到成员，你可能会问自己对版主该怎么办。在大多数情况下，版主会在社区启动后成为社区的一部分。找到成功的版主的最好方法是从你现有的用户群中挑选活跃的、成熟的、有知识的成员。参见 [Greg Berhardt 的超级版主指南](http://www.promotionbase.com/article/779)。

##### 入门！

因此，如果你一直在欣赏像 SitePoint Forums 这样成功的社区网站，那么这篇文章有望成为一个向导，引导你走向正确的方向，并回答你的许多基本问题。记住，要有耐心和创造力，你会成功的！
