# 自我推销:你需要新方法吗？

> 原文：<https://www.sitepoint.com/self-promotion-do-you-need-a-new-approach/>

最近，我的一个自由职业朋友发现自己失业了。他一直在和一个大客户签合同，但是合同被终止了。当我和他说话时，他不确定去哪里找工作。

他花了几个月的时间失业，尝试在线自由职业市场，并为项目发送邮件列表。他不想找一份全职工作，也不确定还能去哪里找。我告诉他应该更新他的网站，联系他过去的客户，但他不感兴趣。他说，他以前的客户已经离开了。他真的不擅长发邮件。

我说我会帮助他，然后我们开始在他的网站上工作。在他发表文章几天后，他接到一家跨国科技公司的电话，询问他感兴趣的项目类型。上周，我们终于给他以前的客户发了邮件。12 小时内，他就有了一系列的工作。

我这里没有夸张。这真的很简单。

如果他愿意尝试对他来说不那么吸引人的晋升途径，他能多快找到工作？

那你呢。你是否因为某些类型的推广对你没有吸引力而错过了一些伟大的项目？

## 超越你的舒适区

许多自由职业者独自外出，因为他们想要更多样的工作、更多的实验项目和更多的刺激。

但当涉及到晋升时，尴尬或不适的风险越低越好。至少对我们很多人来说是这样。

总会有你厌恶的促销方式。但是那些有点难的呢？找到一种方法来克服你的不适，就像我的朋友一样，你可能会进入一个全新的机会池。

首先，列出所有你不讨厌但不使用的促销方法。然后选择一两个，尽你所能让它们发生。

在你的日历上留出一些时间来计划如何将这个方法付诸实践。如果你不确定从哪里开始，就像我朋友做的那样，向朋友或业内人士寻求建议。一旦你有了一个计划，再次安排一些时间来实施它。

## 实验的力量

如果你愿意，可以称之为实验。你不必对这个想法或它的成功做出承诺。您可以简单地测试一下，看看它是如何工作的。

这听起来可能很傻，但我认识的许多自由职业者都避免某些晋升，因为他们不想成为“那种……那种擅长人际关系网的自由职业者”。做公开演讲的那种自由职业者。那种通过社交媒体自吹自擂的自由职业者。那种用垃圾邮件骚扰老客户的自由职业者。

好吧，别做那种自由职业者。只是运行一个小实验，看看会发生什么。如果不行，那就别想了。如果是的话，那太好了。这并不意味着如果你不想重复，你就必须重复。尽管我有这样的感觉，如果它真的有用，下次你需要工作的时候，你会更愿意再次使用它。
