# 火狐浏览器占据全球 20%的市场份额

> 原文:[https://www . site point . com/Firefox-hits-20-browser-market-share-world wide/](https://www.sitepoint.com/firefox-hits-20-browser-market-share-worldwide/)

![](../Images/4198910a4b06ea6e877c42cd2637a384.png "firefox-logo")通过读写网的朋友，Mozilla 报道称，根据网络应用，火狐浏览器有史以来第一次突破了全球 20%的市场份额。在 10 月份的四个星期中，有两个星期，网络应用程序认为 Firefox 的市场份额达到 20%或更多，这是第一次出现这种情况。在网络应用的下一个季度报告中，Firefox *应该会稳稳地超过 20%的门槛——上个季度为 19.48%，并且在过去的四个季度中每个季度都至少增长了一个百分点。*

在几乎所有的浏览器使用报告中，Firefox 在过去的一年里一直在向 20%的市场份额缓慢前进。例如，TheCounter.com 上个季度的 Firefox 市场份额为 18 %, ADTECH 2008 年的 Q1 报告也是如此。

在一些全球市场份额报告中，Firefox 实际上做得更好。W3Counter 的全球统计数据报告称，自 2007 年 5 月以来，Firefox 的市场份额超过了 20%,过去三个月超过了 30%。法国服务 [XiTi Monitor 报道](http://www.xitimonitor.com/fr-fr/barometre-des-navigateurs/barometre-des-navigateurs-septembre-2008/index-1-1-3-145.html) Firefox 的使用率在过去半年里稳定超过 30%。

不管你相信哪个统计数据，火狐的前景是乐观的。不同于 Safari 和 Opera 的使用情况——在所有的使用报告中，这两者的使用情况基本持平(尽管网络应用程序显示 Safari 在过去几年中有所增长), Firefox 自四年前首次亮相以来一直保持着快速增长。在早期采用者中，使用率甚至更高。面向超级用户的网站通常拥有更高的 Firefox 用户集中度。

例如，SitePoint 网站 50%以上的访问量来自使用火狐浏览器的用户(3.03 是最受欢迎的版本)。读写网报道说有 55%，我的 Rails 论坛网站看到超过 68%的 Firefox 使用率。

看看谷歌的新 Chrome 浏览器([我们的报道](https://www.sitepoint.com/early-stats-google-chrome-hurts-firefox-not-ie/))如何影响火狐的增长将会很有趣。现在下结论还为时过早，但到目前为止，它似乎没有对 Firefox 的使用份额增长产生太大影响，尽管早期的统计数据显示，它正在从 Firefox 夺走用户(可能只是一种新奇的效果)。在 SitePoint，Chrome 的使用率稳定在 3%多一点，全球范围内的使用率似乎在 0.25%到 1%之间，这取决于你信任谁。

所以恭喜 Firefox 达到了 20%的门槛。为持续增长干杯。
