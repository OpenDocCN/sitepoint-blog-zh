# 我在网站上犯的最大的错误

> 原文：<https://www.sitepoint.com/web-site-mistakes/>

最近，我在回溯机上浏览我的网站历史。开怀大笑之后，我开始思考一些我本可以做的事情，或者我现在的网站还可以做得更好的事情。我犯过的一个最大的错误，虽然已经改正了，但仍然困扰着我。

## 我最大的错误是…

有点尴尬的是，我最近才开始向网站访问者提供免费下载，作为一种分享知识、衡量兴趣以及为未来交流提供姓名和地址的方式。可以说，这与其说是一个明显的错误，不如说是一个失去的机会，但它仍然是我“应该知道得更好”清单上的一件事。尤其是当你考虑到我经常向我自己的客户建议提供下载和获取感兴趣的电子邮件地址的价值。哎呀。

**tip:** Back to the Future

如果您最近没有查看您的网站的时间表，我强烈建议您查看。这不仅是一个巨大的喜剧解脱，而且你可以回顾你的进步，看看发生了什么变化，从你的错误中吸取教训，集思广益，找到继续前进的新方法。

当然，在我的网站上，在我的生意上，在我的生活中，我也犯过错误(现在仍然在犯)。事实上，我犯过的其他一些与网站相关的错误包括在 SEO 上偷工减料，在重大检修后懒得验证网站，以及没有像我应该的那样经常检查链接。啊…活到老学到老，对吧？
