# CF 方面一切都很平静…

> 原文:[https://www.sitepoint.com/alls-quiet-on-the-cf-front/](https://www.sitepoint.com/alls-quiet-on-the-cf-front/)

这一周出奇的平静。框架方面没什么；开放的蓝龙和拉伊洛保持自己。嗯，看起来每个人都在埋头编码…让我们看看他们为我们准备了什么。

**代码 n’概念**

*   Adobe 发布了一份技术说明，简要演示了如何流式传输图像。如果你正在使用 CFIMAGE， [Scott Pinkston 提醒你要确保使用所有的补丁](http://www.scottpinkston.org/blog/index.cfm/2008/8/20/Patch-for-CFImage-and-javaioFileNotFoundException)，以免你发现自己不必要地拔掉了头发
*   Ben Nadel 的 OOPhoto 项目仍在进行中——最新的部分是[在服务/外观层](http://www.bennadel.com/index.cfm?dax=blog:1334.view)封装表单处理
*   雷蒙德·卡姆登回答了一位读者关于[用 CFC](http://www.coldfusionjedi.com/index.cfm/2008/8/26/Using-a-CFC-to-drive-content-to-cftooltip) 填充工具提示以及[在 CFFEED](http://www.coldfusionjedi.com/index.cfm/2008/8/21/Yet-another-post-on-cffeed-and-columnMap) 标签上跳动的问题
*   从他对业务逻辑的思考开始，道格·布德试图定义什么是应用逻辑
*   Doug Hughes 介绍了 CFANT 项目，这是一个用于编写远程 CF 部署脚本的工具包
*   [MXUnit 单元测试框架已经更新到 1.0.2](http://mxunit.org/blog/2008/08/mxunit-102-now-available.html)

你喜欢用嵌套集合来表示数据库中的树结构吗？[嵌套集合树](http://nstree.riaforge.org/)——一个用于管理各种相关操作的 ColdFusion 库——已经发布了[0.8 版本](http://stannard.net.au/blog/index.cfm/2008/8/25/Nested-Set-Trees-in-ColdFusion-v08)。

SQL 注入攻击仍然困扰着你吗？西蒙·沃特利在[上发布了如何修复 SQL 注入黑客](http://www.simonwhatley.co.uk/how-to-fix-a-sql-injection-attack)，以及如何在第一时间[防范恶意攻击](http://www.simonwhatley.co.uk/how-to-protect-your-website-from-a-malicious-attack)(向[史蒂夫·布莱恩特](http://www.bryantwebconsulting.com/blog/index.cfm/2008/8/25/CFBlogPicks-August-25-2008)致敬)。你知道人们常说预防胜于治疗… [Peter Boughton 发布了一个 Eclipse IDE 插件](http://blog.bpsite.net/item/63/QueryParam%20Scanner%20-%20Eclipse%20Plugin.html)，名为 [QueryParam Scanner](http://www.hybridchill.com/projects/qpscanner.html) ，它通过代码查找并报告带有未受保护输入参数的查询。现在没有借口-锁定代码！
