# 苹果对英特尔是官方的

> 原文：<https://www.sitepoint.com/apple-on-intel-is-official/>

苹果[在今天的全球开发者大会上宣布在 2006 年转向英特尔架构，稍微动摇了技术路线图](http://www.apple.com/pr/library/2005/jun/06intel.html)。这对用户和开发者都有影响。

一些资深苹果分析师对此反应消极，但很大程度上是谣言四起，大多数人认为有事情正在酝酿之中。传统观点认为，这对苹果来说是一个很好的举措，开启了更快、更耗电的性能，尤其是对 Macintosh 笔记本电脑而言。

更有趣的可能是这为 Linux 开发人员打开了一个新的天地，他们现在可以在熟悉的硬件平台上利用现在已经成熟的 OS X(和 BSD Unix 基础)。

库比蒂诺的工程师对此进行了一些思考——他们花了几年时间探索英特尔上 OS X 的达尔文内核，现在正计划发布 XCode，这是苹果针对 OS X 的编程环境，将帮助 PowerPC 开发人员移植到英特尔。

从 web 托管的角度来看，我已经是一名 Xserve 用户，我很高兴 OS X 服务器的功能和用户界面框架可以在英特尔平台的极速下使用。

对于外行来说，苹果已经注意到将配置和管理工具直接构建到 OS X 服务器中，这些工具利用熟悉的开源工具，如 LAMP、Postfix、Samba、LDAP 等，以实现快速部署和简化管理。

在某些情况下，我失去了几年的论点(由于不同的硬件架构)苹果的 OS X 可能是我们一直在寻找的 Linux 桌面和服务器现在可能成为现实。
