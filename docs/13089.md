# WaSP 释放酸 3 测试

> 原文：<https://www.sitepoint.com/wasp-releases-acid3-test/>

Web 标准项目昨天宣布针对 JavaScript 和 DOM 兼容性的 Acid3 测试已经发布。

> Acid3 测试旨在测试 Web 2.0 的规范，并揭示公共 ECMAScript 262 和 W3C 文档对象模型 2 标准实现中的潜在缺陷。这些技术统称为 DOM 脚本，它们支持高级页面交互，并为许多高级 web 应用程序(如基于 web 的电子邮件和在线办公应用程序)提供动力。
> 
> 作为一系列 100 个迷你测试，Acid3 已经被发现暴露了所有测试浏览器的缺陷，包括 Internet Explorer、Firefox、Opera 和 Safari。WaSP 希望 Acid3 能在浏览器制造商开发未来版本的产品时对他们有用。

测试背后的驱动力，伊恩·希克森，[接受了来自开发者社区的输入](https://www.sitepoint.com/help-create-the-acid3-test-for-javascript-compliance/)以使测试成为最全面的(大部分不相关的)边缘案例和不符合规范的集合。希望就像之前的 Acid2 一样，浏览器制造商将努力改进他们的产品，使其通过测试，从而(希望)使跨浏览器 JavaScript 开发既一致又可预测。
