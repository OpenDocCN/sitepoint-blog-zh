# 消除许多付款麻烦的简单方法

> 原文：<https://www.sitepoint.com/a-simple-way-to-eliminate-many-payment-hassles/>

在上一篇博客文章中，我附带说明了我向客户发送电子链接以进行信用卡支付。值得单独发帖讨论这个战术。

如果你和我一样，你讨厌任何和直接服务客户无关的事情。收款可能是最大的麻烦之一。

所以我建立了我的网上购物车(使用 www.1shoppingcart.com)来收集客户的付款。一旦客户同意，我会给他们发送一个付款链接。如果我们同意 3 期 1500 美元，我为他们设置了一个特殊的产品，这样他们就可以提前付款，并且每 30 天自动支付 2 个额外周期的费用。

好处显而易见:

–不用再开发票，也不用再强调何时能收到付款。

–立即付款。

–自动支付。

–只要客户同意，你就能拿到钱，所以你不必担心“买家反悔”或其他什么事情。

–我甚至向一些客户收取 2%或 2.5%的信用卡手续费，他们并不介意，因为这个过程也为他们节省了时间。

这并不适用于所有的客户。一些像大学这样的大组织不给他们的教员信用卡。对于大的任务，你可能会超出信用卡的限额。

但是这个策略确实让我 70%的工作都少了很多麻烦。
