# 如何找到设计灵感，第 1 部分:杂志

> 原文:[https://www.sitepoint.com/designinspiration-magazines/](https://www.sitepoint.com/designinspiration-magazines/)

杂志封面是线下设计灵感的绝佳来源。我们看一眼它们，通常会很快决定我们是否对纯粹基于封面的内容感兴趣。是的，健壮的男人或美丽的女人通常是光滑杂志的主要吸引人之处，但是使用的*颜色*是抓住你注意力的整体计划的一部分。

这篇文章着眼于目前报摊上的一些配色方案，以及我们如何从中学习并将其应用到我们的网页设计中。

你可以绝对 100%确信，今天杂志上的配色方案不是偶然出现的。艺术总监花了很多时间选择合适的图像，然后选择字体的颜色。带有可识别标志的杂志，如《滚石》、《时尚》和《新科学家》,经常改变标志的颜色，以匹配或补充封面上的图像，使其引人注目。

杂志封面上的标题相当于网站上的行动号召。他们大胆，突出，抓住你的注意力，让你做一些事情。给我买！你可能没有在自己的网站上销售，但我肯定你希望用户在阅读内容之前不要关闭页面。使用有效的配色方案可以帮助你做到这一点。

研究表明，当打开一个新网页时，网络用户在 1/20 秒内就形成了一个观点，所以正确的配色方案很重要。显然还有很多其他因素需要考虑，包括布局和字体，但是颜色是用户脑海中最先注册的东西。

为了说明使用线下配色方案的想法，我选择了几本杂志，并强调了他们使用的主要颜色。每本杂志下面都有一个使用类似方案的网站示例。
