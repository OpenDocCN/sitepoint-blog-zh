# 刷这个！使用滑动文件获取灵感

> 原文:[https://www.sitepoint.com/swipe-file-for-inspiration/](https://www.sitepoint.com/swipe-file-for-inspiration/)

传统上，滑动文件被用于广告和文案，作为文本、图像和布局的存储库。但是滑动文件可以应用于几乎任何类型的创造性工作、开发项目和写作任务。它们很容易创建，如果你持续做类似的工作，可以节省你大量的时间。

**什么是刷卡文件？**

卷帘文件只是一组模板或示例，您可以随身携带并在开始新项目时参考。如果你是一名设计师，你的滑动文件可能由你遇到过的伟大的标志设计、创造性的导航、独特的布局、字体和其他排版元素以及库存照片组成。

程序员的滑动文件可以包括代码片段和脚本，而作者可能会收集引人注目的标题和文章大纲。

为什么它们有用？

每个刷卡文件都是唯一的。你可能觉得鼓舞人心的项目对其他人来说可能并不鼓舞人心，所以一旦你创建了自己的项目，它对你来说将是无价的。您可以建立自己的 swipe 文件来支持您的工作流程，因此它是一站式灵感来源。

一个滑动文件也可以是一个很好的方式来记录你自己的创造性想法——你已经开发并取得成功的想法，以及你计划在以后工作的想法的片段。

**如何创建刷卡文件**

如果你有硬拷贝的例子(例如来自[报纸](https://www.sitepoint.com/offline-design-inspiration-part-2-newspapers/)或[杂志](https://www.sitepoint.com/designinspiration-magazines/)，就像 Jennifer Farley 最近写的那样)，你可能想要为不同类别的材料创建一个包含文件夹的物理文件。或者，您可以扫描它们并以电子方式保存。

对我们大多数人来说，大部分刷卡文件将是电子的。可以收集刺激例子的截图、pdf、URL、文本文件。您可以使用描述性文件名，通过文件夹结构来组织和归档您的数字卷帘文件。

你也可以使用笔记本应用程序，创建一个灵感文件。有许多笔记本工具提供了一种简单的方法来添加数据，然后搜索特定的信息。您可能想要查看的一些工具包括:

*   [Circus Ponies ' s NoteBook](http://www.circusponies.com/)–一款 Mac OS X 工具，用于编辑笔记、文件、网络研究、电子邮件和语音注释。
*   Evernote 是一款收集和同步数据的工具，包括笔记、网站、照片、电子邮件、扫描图像和音频，可以在你的桌面、网络账户和移动设备上使用(有限量免费版本)。
*   [OneNote](http://office.microsoft.com/en-us/onenote/default.aspx)–一款微软软件，它将笔记、图像、文档、来自其他 Microsoft Office 程序的文件以及其他媒体收集到一个可搜索的笔记本工具中。
*   spring note-一个基于维基的免费在线笔记本，用于编辑和分享想法、多媒体、项目和文档。
*   Stixy–一个免费的在线自由格式设计工具，用于组织笔记、图片、网页剪辑和任务。
*   Uber note–一款免费的在线工具，用于收集笔记、任务和网络剪报，可通过移动设备访问。
*   Zoho 笔记本(Zoho Notebook)——一个在线工具，可以编辑文本、图像、音频、视频，并允许你与他人分享，同时提供版本跟踪。

**注意事项**

卷帘文件的目的是为您的项目提供一个切入点，帮助您跳出框框思考，并简化您的工作流程。使用滑动文件的一个危险是变得过于依赖它，因此它阻止了你自己的创造力的发展。所以你要适度使用它，而不是用它来代替你自己的想法。

不言而喻，你应该确保你只使用你的滑动文件来获得灵感，而不是复制或复制你收集的任何作品。
