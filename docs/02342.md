## 什么是 Susy？

如果您使用 [Sass](http://sass-lang.com/) 进行样式设计，并且一直使用 CSS 框架来创建网格布局，那么您应该看看 Susy。Susy 可能是对按需网格系统的最好描述。Susy 没有强迫您在项目中包含成百上千行不必要的代码，也没有限制您的定制选项，而是采用了一种不同的方法，允许您轻松地创建定制的网格布局，以满足您的确切需求。

### 使用 Susy

使用 Susy 需要为你完成一些事情。工具不应该为了它们自己而存在——它们应该包含在你的项目中，因为它们有帮助。如果您需要的只是一个网格系统，那么使用一个成熟的设计框架可能不是一个好主意。然而，如果你需要这个网格，从头开始写你自己的也是一个糟糕的主意。有一个中间地带。

使用 Susy 可以让你快速原型化你的布局，并构建完全可定制的网格系统——而不是在你构建自己的系统时计算你的百分比，Susy 会为你做提升。

这就是使用 Susy 或使用 Bootstrap 或 Foundation 等成熟框架的区别——它们实际上是不同的工具集。如果您需要一个完整的框架，并利用它的各个部分、资产和特性，那么它会告诉您需要什么工具！但是，如果您发现自己只是感到沮丧，并试图解决您的网格布局问题，您可能想看看 Susy。
