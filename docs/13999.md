# 伸出手，接触谷歌

> 原文:[https://www.sitepoint.com/reach-out-and-touch-with-google/](https://www.sitepoint.com/reach-out-and-touch-with-google/)

谷歌本周在搜索之外拓展业务，发布了多面即时通讯客户端和带侧边栏的升级版桌面 2。

据《华尔街日报》报道，谷歌的即时通讯将被命名为 Google Talk，将与美国在线、雅虎和 Skype 等竞争。这将包括文本和音频聊天和会议。

虽然这些将给个人用户带来巨大的价值——但我也看到了 web 专业人员通过其 [API 和插件开发社区](http://desktop.google.com/developer.html)获得的市场价值(尽管这可能需要具备一些 Windows 编程技能或拥有相同能力的朋友)。

也许这两者中最有趣的(尽管我确信我们都会测试 Google Talk！)探索的是新的桌面 2，它包括侧边栏，一个交互式浏览器插件，可以收集电子邮件，RSS 提要，搜索等。一个充满活力的插件领域将在短时间内爆发，这些插件已经成熟，可供那些能够利用它们而不是花时间构建插件的网络专业人士挑选。然而，对于那些准备写一些代码的人来说，这种定制水平可能会扩大你与客户的关系(我一直在考虑向上销售)。人们可以想象向客户端提供插件:

*   将他们的网站扩展到桌面，可能跟踪网站上的新内容，或者提供跟踪网站上他们公司名称的新闻/实例
*   被配置为从您为他们管理的域中检查他们的电子邮件
*   查询您为他们开发的用于管理报告的 web 应用程序
*   在他们的网站上全天跟踪网络统计数据
*   这份清单还在继续…

谷歌进军新领域并对抗根深蒂固的网络大师们，肯定会推动全面创新——让我们所有人受益。同样可以肯定的是，它的技术将溢出到网络专业人士，提供可销售的新工具或至少是独特用途的灵感。
