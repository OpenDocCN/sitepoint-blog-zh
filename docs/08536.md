# 赢得客户和影响他人的六个步骤

> 原文：<https://www.sitepoint.com/six-steps-to-winning-clients-and-influencing-people/>

要真正影响客户，你需要在信任的基础上建立关系。如果客户信任你，他们会接受你的反馈，最终会得到更好的结果。

有多少次你建议了一个设计或功能，结果却被否决了，然后在没有你的参与的情况下，构建了一个你认为不合标准的东西？如果你有过这些情况(相信我，几乎我们所有人都有)，那么很有可能你的客户关系不是建立在信任之上的。

那么，你如何与客户建立这种难以捉摸的信任呢？这里有六个简单的步骤:遵循所有这些原则，你肯定会与你的客户建立更好的关系，这反过来意味着更大的影响力和更大的成果。

## 表示尊重

我不是说叫他们夫人或先生，我说的是真正的尊重。约会不要迟到，穿着得体，不要逃避指责，要有礼貌。分享他们的愿景，理解他们的决策理由。

## 要真诚

我指的是真正的正版，而不仅仅是营销上讲的正版。下一次你和你客户谈话时，要脚踏实地，摆脱公司的语言，像对待一个重要的朋友一样对待他们。

## 证明你的工作

不要只是发送一个设计概念或功能列表，而不解释你的思维过程。当然，你可以从技术上向他们解释，但是要给他们统计数据、案例研究、你选择那种颜色的原因或者你为什么反对那种 320 字段长的联系表格。做研究，用统计数据武装自己，支持每一个主张。

## 承认错误

我们都会犯错，不管怎样，我们当中的人类也是如此。我肯定你在客户时间犯了很多错误，就像我们所有人一样。你如何处理它们会让你变得更好——承认它们，让客户知道，或者分担或承担你的错误成本。

## 分享想法

有一个可能为你的客户创造奇迹的想法吗？与他们分享——不要害怕拒绝，让他们知道这是一个想法，并确保他们明白你昨晚或周末一直在想他们——这证明你充满激情，而不仅仅是一个朝九晚五的思考者。

## 不同意

听起来很疯狂，不是吗？虽然这是真的；没有人想要一个对他或她言听计从的供应商，即使供应商知道这是一个糟糕的举动。如果你的客户要求你做一些你知道行不通的事情，那么让他们知道——当然要礼貌，并详细解释你的理由。

如果你建立了一种信任的关系，你将会拥有一段更长久的关系，而不仅仅是现在的这个项目。你会对你的客户产生影响，并拥有一个愿意再次与你合作的忠诚客户。

我们都知道和以前的客户合作更便宜也更容易。他们了解你的流程和系统，他们不需要每一分钱都是合理的，他们会要求你的投入，这反过来会使项目更好、更有成就感。

为赢得客户和影响他人向你致敬！
