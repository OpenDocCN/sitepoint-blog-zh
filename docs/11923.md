# 电子邮件营销:值得你花时间吗？

> 原文：<https://www.sitepoint.com/email-marketing/>

我们很多人都为客户的电子邮件营销活动设计 HTML 模板，但你自己的业务呢？如果你愿意投入时间和精力来维护模板、分发列表和定期活动(即使你只限于纯文本电子邮件)，电子邮件营销可能是一种发展和扩大业务的可行方式。

事实上，我们中的许多人可能已经在我们的网站上有了注册表单，允许我们从感兴趣的人那里收集地址，但是我们实际上多久发送一次电子邮件呢？那你自己的通讯录呢？我知道我有数百个联系人，他们可能会对我会考虑发出的信息感兴趣。如果这听起来像你，可能是时候认真考虑一下电子邮件营销的好处以及发送什么类型的信息了。
 **好处**

电子邮件营销对任何类型的企业、自由职业者或企业家都有很多好处。你的邮件列表可以包括现在和过去的客户、同事、不合格的潜在客户、潜在客户、朋友、过去的联系人甚至你不认识的人。

重点应该是开发一种定期联系的方法，因为它可以:

*   向客户/同事传达他们不知道的重要消息
*   让你留在潜在客户的脑海中
*   让你有机会接受定期推荐
*   让您的销售过程更轻松、更高效
*   以轻松、随意的方式帮助人们了解你
*   与你的目标人群开始对话

**分享什么**

你可能认为你没有什么可分享的，但是电子邮件营销活动并不一定要复杂。甚至不一定是正式的时事通讯；简明扼要也很有效。

您可以发送到通讯组列表的信息可能包括:

*   面向其他客户的网站发布
*   你或你的客户或同事提供的新产品或服务
*   特殊折扣或限时订购优惠
*   [客户评价](https://www.sitepoint.com/how-to-get-use-testimonials/)
*   你推荐的可能对你的受众有用的产品或服务
*   新的伙伴关系或合作
*   调查或反馈请求
*   文章、博客帖子和其他信息资源

**未越线**

在发起电子邮件营销活动时，有一些重要的事情要记住。当你有真正有价值的东西要分享，而不仅仅是为了发出去的时候，你要确保你在适度地发送信息。请在你开始发送之前给人们一个选择加入的机会，并确保有一个明确的退订方式。越界做太多太频繁的事情，或者发送不想要的邮件，将会适得其反，损害业务。
