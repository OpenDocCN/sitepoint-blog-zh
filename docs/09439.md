# 和朋友一起工作:值得吗？还是浪费时间？

> 原文：<https://www.sitepoint.com/coworking-with-friends-worthwhile-or-a-waste-of-time/>

自由职业是孤独的。这是人们不自己出门的常见原因；这也经常被前自由职业者引用为他们放弃的原因。

对于这个问题，有无数的技术“解决方案”,但是在我的书中，没有什么能胜过传统的面对面的交流。我的几个自由职业者朋友同意这一点，我们已经多次谈到在共享空间工作。

但是直到这个星期才发生。

巧合的是，我在去一个我经常远程工作的地方的路上遇到了我的一个朋友，他刚刚被解雇，正在开始自己的生意:一个我喜欢的有免费 wifi 的咖啡馆。

我们一起度过了一个有趣、社交和富有成效的早晨——这个早晨打破了我对和一个自由职业朋友一起“工作”的许多期望。

## 1.这不会让人分心

由于我和我的自由职业者朋友在不同的领域工作，我一直认为合作安排需要更多的分心而不是行动。相反，反过来才是真的:我们不是在高度集中注意力后离开工作，而是聊上几分钟，然后继续工作。

在网上分心会模糊工作时间的地方，从电脑上抬起头和某人交谈的行为似乎让我的大脑意识到它已经休息了。也许它实际上也给了我更多的休息。不管怎样，我比平时完成了更多的工作。

## 2.很有用

虽然我们在不同的领域工作，但我和我的自由职业朋友们有很多共同之处:我们有客户，我们必须追逐发票，我们不断地想出想法，试图找到工作并谋生。因此，一起做我们自己的东西给了我们互相交流想法的机会——并在这个过程中互相提供想法。

在单干了几个月后，有一个现场共鸣板来立即产生想法是很好的。

## 3.我们不需要办公室

我一直认为共同工作是在一个专用的共同工作空间里的一件有计划的事情。好吧，如果你真的想把这些建议付诸行动，你可能需要打电话给一个自由职业的朋友，约个时间见面一起工作，这样计划就不会完全脱离议程。但是你不需要一个专用的共同工作空间——我们在咖啡馆工作。

如果你跟随我们的脚步，你将需要一个放松的咖啡馆，你不会在喝完咖啡后就被赶出来。当然，你需要一个像样的网络连接，根据你在那里的时间长短，你可能还需要使用幻灯片。我工作的咖啡馆都有这些好处——还有可爱的员工，这是一个巨大的奖励。

其他可以合作的地方包括:

*   当地图书馆的自习室
*   你的家庭办公室
*   你自由职业朋友的餐桌
*   公园(前提是屏幕不会太刺眼)

## 4.很值得

如果你没有和自由职业者共事过，那就试试吧。任何时候你开始感到有点孤立，这都是一个理想的机会去结识一个有同理心的朋友，重新接触现实世界。你会从中得到乐趣，完成工作，并激发一些新的想法。
