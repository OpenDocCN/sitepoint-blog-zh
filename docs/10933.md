# 成为更好的网络开发者

> 原文:[https://www.sitepoint.com/become-a-better-web-developer/](https://www.sitepoint.com/become-a-better-web-developer/)

在过去的一个月左右的时间里，围绕 SitePoint 网络空间出现了疯狂的抱怨声——都与我们的新口号有关。让 25 个人在任何事情上达成一致都很难，更别说像标语这样重要的事情了。但是我们做到了。我们已经决定，在 SitePoint，我们只关注一件事，那就是帮助你“*成为一名更好的 web 开发人员*。”所以，如果你还没有注意到它，看看我们的横幅。我们对它看起来如此时髦感到非常自豪。

在全新事物光谱的另一端，今天在 [SitePoint 论坛](https://www.sitepoint.com/forums/)上，我们向[金帖奖项目](https://www.sitepoint.com/forums/showthread.php?p=4605966#post4605966)说再见。比赛已经进行了一个月，我对结果非常满意。我们已经看到了一些非常精彩的帖子，我一直很喜欢免费赠送书籍！我希望这个比赛已经向你展示了一个好的帖子不一定要很长，也不一定要展示某个主题的专业知识。一篇真正伟大的文章可能只是一个让你思考的简单问题……也可能是一个用例子解释清楚的答案。但在我看来，最好的帖子是那些为他人着想的帖子。如果你寻求帮助并得到了，回来告诉我们进展如何。将来有人会因此而爱你，即使你没有得到一本免费的书。

既然五月的比赛已经结束，我为六月准备了一些新的东西。理论上，我应该等到明天才泄露秘密，但我是那种不能保守秘密或等待惊喜的人。事实上，当我还是个孩子的时候，我会在圣诞礼物一到树下就偷偷地打开，然后小心翼翼地重新包装，这样就没有人会知道了。有时，如果我姐姐的颜色更好，我甚至会和她交换。但是我有点跑题了。

![SitePoint forums](../Images/be78615632743abe85912c6c31f626c1.png "sitepoint-forums")

我已经决定，因为我这周的帖子写得太晚了，离六月已经足够近了，所以我现在可以告诉你，从明天开始，我将举办一场 SitePoint t 恤比赛。是的——你首先在这里听到的。距离我们发布上一件官方 t 恤已经有好几年了，如果我们的脸书球迷有什么值得关注的话，你一定很想买一件新的。因此，请查看明天的[新闻和公告](https://www.sitepoint.com/forums/forumdisplay.php?f=38)论坛了解详情。
