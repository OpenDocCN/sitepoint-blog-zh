# 满足您云托管需求的 10 个解决方案

> 原文:[https://www.sitepoint.com/cloud-hosting-solutions/](https://www.sitepoint.com/cloud-hosting-solutions/)

云，云，云。这似乎是这些天你听到的所有内容，以及我们最终将如何生活在“云”中。嗯，这是有原因的…这几乎是真的。

你已经有了像 Twitter 这样的服务，使用亚马逊 S3 来提供其服务上的所有头像图像，所以这只是表明它可以用于那些你的用户从未想到的小调整。如果您愿意，也可以选择在网格系统上运行整个 SaaS，只在最需要的时候扩展资源。大多数云托管解决方案最好的部分是，它们只对你使用的东西收费，那么你为什么要为你从未访问过的资源付费呢？

我们收集了 10 个云解决方案，帮助您开始您的天空之旅。

**[亚马逊 CloudFront](http://aws.amazon.com/cloudfront/)** :当你把 CloudFront 和 S3 结合起来，你就获得了你的原始文件的存储库，CloudFront 帮助你从多个边缘位置交付你的文件，这样你的用户就可以获得低延迟和高数据传输速度的交付。定价基于数据传输和获取请求。

![](../Images/143456223114a82c912fefd377a3f507.png)

**[亚马逊 EC2](http://aws.amazon.com/ec2/)** :亚马逊弹性计算云(Amazon Elastic Compute Cloud)，也称为亚马逊 EC2，允许您根据需要启动和移除 web 服务器，以及您需要的操作系统和配置。定价是基于计算机的时间和带宽，你消费没有最低月费。

![](../Images/b40ec27759819b942f0f24213db4c30b.png)T2】

**[亚马逊 S3](http://aws.amazon.com/s3/)** :亚马逊简单存储服务，通常被称为亚马逊 S3，几乎已经成为你能想到的所有主要网络应用的默认文件存储系统。与其他亚马逊服务一样，您只需为基于存储、传输和请求而消耗的资源付费。

![](../Images/3b61da552ae90fa8704dc68bd71e850f.png)

**[弹性服务器](http://elasticserver.com/)** :有了弹性服务器你可以选择预先存在的软件包添加到你的服务器，或者你可以上传你自己的并定制它来满足你的需求。

[![](../Images/3e6afc06b2831e0612334ae99ceb4c6b.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/cloudhosting/elasticserver.png)

**[flexi scale](http://www.flexiscale.com/)**:flexi scale 允许您根据需要混合虚拟和专用服务器，并且您可以从 web 控制面板或通过它们的 API 启动任意数量的虚拟服务器。映像包括各种风格的 Windows 和 Linux，提供高达 8 GB 的 RAM，新实例可以在不到一分钟的时间内启动。与大多数云托管选项一样，您只需为使用 FlexiScale 支付费用，并且没有合同最短期限。

[![](../Images/b2d0ce5a0082802c79d61f0ee533a345.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/cloudhosting/flexiscale.png)

**[GoGrid](https://www.gogrid.com)** : GoGrid 让你能够建立即时的、随需应变的服务器，具有“云中控制”的特性。基于 Windows 和 Linux 的服务器都可以通过 web 界面在几分钟内通过众多提供的服务器映像配置启动。

[![](../Images/3b61abfb13ae2d7d2745cac45cdb13e9.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/cloudhosting/gogrid.png)

**[谷歌应用引擎](http://code.google.com/appengine/)** :应用开发者可以免费在谷歌的服务器上托管自己的应用。如果你的应用增长超过免费配额，你可以为额外的资源付费，但你只需为你实际消耗的资源付费。Python 和 Java 是唯一官方支持的语言，但是用 PHP 和其他语言编写的应用程序也有变通方法。

[![](../Images/2cb37f2708cb0f3d709cbcab2ba88661.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/cloudhosting/googleappengine.png)

GridLayer : GridLayer 采用了更传统的主机价格结构，为其各种解决方案提供定价包。顾名思义，该公司专门研究按需扩展的网格解决方案，以满足您的服务或应用程序的需求。

[![](../Images/528cd45f4bf928077d961ff471666d32.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/cloudhosting/gridlayer.png)

**[Softlayer CloudLayer](http://www.softlayer.com/cloudlayer_computing.html)**:Softlayer 的 CloudLayer 可以让您在短短五分钟内，在最低 2.0 GHz 的处理器上部署各种操作系统的按需计算实例。您可以将每个实例与其他实例集成，也可以将它们作为独立实例运行。价格可以按小时计算，也可以一次支付一个月的费用来获得折扣。

[![](../Images/c9f79b026c077dab068f28fca481a046.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/cloudhosting/softlayer.png)

**[Rackspace Cloud](http://www.rackspacecloud.com/)**:Rackspace Cloud 的前身是 Mosso，它有一个三步流程来启动您需要的服务器:选择 256 MB 到 15.5 GB 的 RAM，选择您的操作系统，几分钟后您就可以拥有一台新服务器。起价低至每小时 0.015 美元。

[![](../Images/f4f8ac758ed35bbb43751e37f84ce70d.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/cloudhosting/rackspacecloud.png)
