# 震中设计

> 原文：<https://www.sitepoint.com/epicenter-design/>

今天早上，我偶然发现了 Jason Fried 去年在“震中设计”上发表的这篇[有用的小文章。](http://www.37signals.com/svn/archives/000737.php)

正如 Jason 所指出的，人们很容易养成以“给定元素”开始页面布局的习惯，这些元素包括徽标、搜索框、页脚等，并通过将页面目的“注入”到任何剩余元素中来结束页面布局。

“中心设计”通过识别每个页面的中心——赋予页面目的的元素——并从其向外设计，逆转了这一过程。

一个简单而相对微妙的变化，但我认为它让用户试图做的事情更加清晰。
