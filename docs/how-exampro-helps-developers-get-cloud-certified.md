## 云改变了我们构建应用程序的方式

在亚马逊网络服务这样的云服务提供商出现之前，你必须自己建立和维护你的数字基础设施。

以前，如果您想在 web 应用程序中添加一个全文搜索引擎，您必须:

*   安装开源软件，如 ElasticSearch
*   通过升级和安全补丁维护服务器
*   配置高度优化的开源软件
*   通过添加负载平衡和协调多个服务器进行扩展
*   并创建数据备份计划。

现在有了 AWS，你可以使用 AWS ElasticSearch 服务。它将启动一个可扩展的服务器，默认情况下是安全的，并且已经配置为高度优化。AWS 雇佣世界一流的工程师来维护这项服务。

到目前为止，云的广泛采用通过底层服务器的成本分摊降低了成本，当你只需支付每小时几分之一便士的服务费用时，很难证明支付内部工程师的费用是合理的。

## 使用云带来了科技公司的新需求

AWS 是创业公司和科技公司中最受欢迎的 CSP。CSP 是一家拥有一系列云服务的公司，您可以使用这些云服务来构建托管在云上的应用程序。

AWS 是一种特殊的 CSP，因为它有数百个云服务，可以用于任何可以想象的用例，并且它们都可以无缝地协同工作。

*   您需要托管数据库吗？然后使用 AWS RDS。
*   需要服务器吗？然后用 AWS EC2。
*   你需要聊天机器人吗？然后用 AWS Lex。
*   需要为万亿字节的数据生成报告吗？然后用 AWS 红移。
*   需要长期储存 7 年吗？然后用 AWS 冰川。
*   需要租卫星吗？然后用 AWS 地面站。

这样的例子不胜枚举。

AWS 允许公司在全球范围内竞争，并可以访问广泛的云服务来构建复杂的应用程序。

对于一家利用云的公司来说，他们需要知道如何使用 AWS 的人。由于有 200 多种服务，他们将需要知识渊博的人根据他们的使用案例和业务目标来组合不同的云服务。

## 什么是 AWS 认证？

AWS 已经正式创建了 AWS 认证，以帮助公司确定云角色的合格招聘候选人，在撰写本文时，他们有 12 种可能的 AWS 认证:

*   认证云从业者(CCP)
*   解决方案架构师助理(SAA)
*   开发人员助理
*   SysOps 管理员助理
*   DevOps 专业版
*   专业解决方案架构师
*   机器学习专业
*   数据分析专业
*   数据库专业
*   安全专业
*   高级网络专业
*   Alexa 技能生成器专业

两个最受欢迎的 AWS 认证是 CCP 和 SAA，因为它们涵盖了最广泛的 AWS 服务。

CCP 是自动气象站的介绍。在考试中心或网上参加考试的费用为 100 美元，通过一到两周的学习即可获得认证，有效期为三年。SAA 涵盖了 CCP 中包含的更深入的服务。在考试中心或在线参加考试的认证费用为 150 美元，通过三到五周的学习即可通过，有效期为三年。

当您通过 AWS 认证时，您可以获得下一半的折扣，因此人们通常以 100 美元的价格获得 CCP，然后以 75 美元的价格获得 SAA。

## 什么是 ExamPro？

ExamPro 是一个云认证的“智能”学习平台。是什么让平台变得“聪明”，让它有别于其他学习平台？我们嵌入了各种学习系统，然后根据以前学生学习习惯的收集不断调整。所以从根本上来说，我们的学习平台是由人工智能(AI)驱动的。

### 带问答的讲座理解

我们的课程可以包括许多小时的讲座视频内容。为了确保信息易于理解，我们将视频分成更短的视频，并且在每个讲座视频上都有小测验。这让学生参与并负责理解内容，避免只看不懂的不良学习习惯。

### 分步实验说明

从实践中学习对于从实践角度理解云服务至关重要。我们有后续视频，让您在自己的云帐户中创建云项目。我们还提供了您可以核对的每一步说明。它使我们能够确定人们通常在哪里遇到困难，并使我们能够帮助或调整经验教训，以更好地完成云项目。

### 抽认卡间隔重复学习

学习云认证需要记住许多需要存储在长期记忆中的技术事实。为了帮助学生记住信息，我们有一个使用间隔重复学习的抽认卡系统。

间隔重复学习是一种学习方法，在这种方法中，你根据一种算法来确定学习时间的间隔，这种算法可以确定你何时会忘记学习材料。每张抽认卡的每次学习间隔更远，增加记忆保持力。

## 练习考试掌握和问题变化

模拟考试是模拟真实付费考试的考试。没有模拟考试，学生很有可能失败。模拟考试通常只使用一次，这意味着一旦你参加了模拟考试，如果你重考相同的问题，你的大脑可能会无意识地记住上一个答案的位置或关键词，而不会试图理解问题和选择。这种学习行为会导致学生错误地认为他们做得很好，因为他们有更高的分数。

ExamPro 使用 mastery 系统来解决这个问题。我们像对待真正的考试一样对待第一次模拟考试，给它一个通过或失败的分数。我们鼓励您在继续进行模拟考试之前掌握不正确的问题。为了避免用户记住之前的问题，我们用一个变体代替之前错误的问题，这样你的大脑就无法根据关键词和位置作弊和回忆。

### 就绪表

大多数学生不知道他们什么时候准备好参加考试，他们可能会开始过度学习。过度学习会增加失败的机会，因为学生增加了他们的学习时间，并错误地认为他们在旅程开始时记住了信息，而他们已经忘记了。我们的准备度表提高了通过的机会，并帮助您快速取得新的认证，尽可能有效地推进您的职业发展和云知识。
