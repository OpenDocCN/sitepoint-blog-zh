# 用开源软件和其他自由软件省钱

> 原文：<https://www.sitepoint.com/open-source-software-for-business/>

昨天，我分享了一份你可以削减业务开支的方法清单。我清单上的一个项目是在日常业务管理中使用开源软件，而不是主流商业应用程序。

这里有一些流行的开源和其他免费应用程序，你可以考虑在你的业务中使用。

## 会计软件

*不要用 Quickbooks，试试:*

**[GnuCash](http://www.gnucash.org/) :** 一个个人和小企业财务经理，用一个类似支票簿的登记簿来输入和跟踪银行账户、股票、收入和支出。GnuCash 被设计得简单易用，但仍然基于正式的会计原则。

**TurboCASH Accounting:** 一个开源会计包，是世界上第一个面向小型企业的全功能开源会计包，拥有超过 80，000 名用户。

## 图像编辑应用程序

*不用 Photoshop 和 Illustrator，试试:*

[**GIMP**](http://gimp.org):GIMP 是 GNU 图像操纵程序。这是一个免费分发的软件，用于照片润饰、图像合成和图像创作等任务。它可以在多种语言的多种操作系统上运行。

[**Inkscape**](http://www.inkscape.org/) :多平台开源矢量图形应用，用于创建和编辑 SVG(可缩放矢量图形)文件。

**[Paint.NET](http://www.getpaint.net/):**一款面向 Windows 的免费图片和照片编辑软件。它具有直观和创新的用户界面，支持层，无限撤销，特殊效果，以及各种有用和强大的工具。

## 办公效率套件

*不要微软 Office，试试:*

**[谷歌文档](http://google.com/docs) :** 谷歌提供的一款免费的基于网络的文字处理器、电子表格、演示和表格应用程序。它允许用户在线创建和编辑文档，同时与其他用户实时协作。

**[KOffice](http://www.koffice.org/) :** KOffice 是基于 KDE 平台打造的跨平台办公套件。它的所有组件都是在自由软件/开源许可下发布的。

**[OpenOffice.org](http://www.openoffice.org/):**领先的开源办公软件套件，用于文字处理、电子表格、演示、图形、数据库等。它有多种语言版本，可以在所有普通计算机上运行。

## PDF 创建工具

*不要用 Adobe Acrobat，试试:*

**[PDFCreator](http://www.pdfforge.org/pdfcreator) :** 一个免费的工具，可以从几乎任何 Windows 应用程序创建 PDF 文件。

**[PrimoPDF](http://primopdf.com) :** 一款免费的高质量 PDF 创建工具，几乎可以从任何 Windows 应用程序打印成 PDF。

## 免费或开源适合你吗？

请记住，免费并不总是更好。在某些情况下，出于多种原因，您可能会选择付费应用程序而不是免费或开源工具，这些原因包括:

*   付费软件可能比开源更可靠。
*   开源应用程序更有可能在没有预先通知的情况下停止开发，让你陷入困境。
*   付费工具通常有一定程度的客户支持。开源支持，如果存在的话，通常是通过在线社区或维基来管理的，所以很难得到快速和正确的答案。
*   开源应用程序可能不提供任何类型的产品文档或培训。

最重要的一步是研究开源替代方案，并确保它将有效地满足您的需求，而不会给您的业务带来任何不必要的挑战。
