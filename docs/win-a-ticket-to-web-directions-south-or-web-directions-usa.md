# 赢取 Web Directions South 或 Web Directions USA 门票！

> 原文：<https://www.sitepoint.com/win-a-ticket-to-web-directions-south-or-web-directions-usa/>

**Web Directions South** 和 **Web Directions USA** 活动即将到来，像往常一样，它们将成为你不想错过的会议。两场活动的主题发言人将是斯科特·托马斯，奥巴马总统竞选活动的首席网页设计师。他将讨论如何设计能引起共鸣和激励的在线社区。

在设计方面，Dan Rubin 将负责 HTML5 和 CSS3，而 Juliette Melton 将专注于有效的远程研究。开发路线将涵盖一系列主题，包括移动应用和基于位置的服务，以及 JavaScript 性能。

这都是关于新技术，创新和网络。以下是每次会议的具体情况:

**网络方向南方**

*   2010 年 10 月 12 日至 16 日

*   2010 年 9 月 21 日至 25 日
*   洛斯亚特兰大酒店
