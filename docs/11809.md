# 滚出去！走出办公室的三个简单方法

> 原文:[https://www.sitepoint.com/foolproof-ways-to-take-vacation/](https://www.sitepoint.com/foolproof-ways-to-take-vacation/)

[![take-vacation](../Images/da0806d7d5f1f0582a15a9ddf3a11faf.png "take-vacation")](https://www.sitepoint.com/wp-content/uploads/2009/07/take-vacation.jpg) 那么，你上次休假是什么时候？坦白吧…你带工作了吗？如果你不好意思回应，不要难过。我完全理解。离开工作，即使是很短的一段时间，对独立职业者来说是一个非常普遍的挑战。让我们面对它；很难做到！

作为企业家，我们对企业投入了太多，以至于到了需要休息的时候，我们很难放手。然后，当我们真的离开并放下工作时，我们担心没有我们在那里，一切都会崩溃。

如果你问我，我认为这是一个企业家应该具备的优秀品质。正是这种动力和奉献帮助你成功，让你的客户欣赏你，并给你我们都渴望的成就感。但事实上，当你为自己工作时，休息不应该是可有可无的。假期是必须的，每个人都应该抽出时间去享受。

但是你是怎么做到的呢？有可能离开工作，真正享受你的下一个假期吗？是啊！如果你做了这三件事，你就能享受一个非工作假期。真的。

制定计划

一旦你开始计划休假，就开始清理你的日程表。通知你的客户你将离开办公室。给他们一个机会把他们最重要的作品寄给你，这样你就可以在离开前处理好，但是一定要让他们知道截止日期。

当你不在的时候，你也要开始寻找保险的过程。如果你还没有同事或分包商，给自己足够的时间去找一些有经验的专业人士来帮你。

记住你自己的商业管理任务…确保你在休假前计划好时间来处理你的簿记或其他正在进行的任务。

**设定你的优先级**

在你休假之前，花点时间弄清楚需要做什么(必须做的事情，而不是应该做的事情)，优先处理清单，并设定一个截止日期。不过，要让它变得可管理和现实，否则你最终只会破坏你的计划。

给自己足够的时间来处理最重要的事情，并在度假前几周一点一点地完成它们。请注意，这不是拖延的好时机。尽你所能让自己专注于工作。你可能还想寻求一个家庭成员的帮助，以保持你的责任感和时间表。

**下班回家**

如果你发现很难抵制不工作时偷看你的黑莓手机的诱惑，那就不要带着它！或者，在你享受阳光的时候，把它交给旅途中的其他人保管。说真的，如果你一天 24 小时都不在线，最坏的情况会是什么？如果你遵循你的计划，你应该没有什么可担心的。

如果把你的黑莓、iPhone 和/或笔记本电脑留在家里对你来说是非常可怕的(对我来说也是！)，设定每天入住的特定时间。至少在一小段时间内，感觉自己在圈子里可能会给你一种内心的平静，这样你就不会被诱惑去登记入住，并真正享受你的假期。
