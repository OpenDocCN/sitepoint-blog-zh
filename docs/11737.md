# 20 个关键的移动办公工具

> 原文：<https://www.sitepoint.com/20-tools-for-mobile-workers/>

上周，我写了创建移动办公室的必要步骤。这篇文章收到了一些关于移动办公人员使用的软件和工具的很好的评论，以便最大限度地利用他们的移动办公，所以我决定将这篇文章的重点放在一些工具上，这些工具适用于那些希望能够在任何地方运营业务的专业人员。

## 加密

**[文件夹锁](http://www.newsoftwares.net/folderlock/) :** 文件夹锁为所有版本的 Windows 上的任何类型的文件和文件夹以及 USB 驱动器上的数据提供 256 位 AES 动态加密。

**[SecuBox](http://www.aikosolutions.com/products/) :** SecuBox 是一款强大而便捷的动态磁盘加密解决方案，适用于 Windows Mobile PDAs 和智能手机，提供 256 位 AES 加密。

**[Truecrypt](http://www.truecrypt.org/) :** 一款免费的开源磁盘加密软件，适用于 Windows Vista/XP、Mac OS X 和 Linux，提供实时、动态、自动加密。

## 在线数据备份

**[Box.net](http://box.net):**Box.net 提供在线文件共享和协作、移动访问和版本历史的文件备份。它兼容 Windows、Mac OS 和 Linux。

**[Dropbox](http://www.getdropbox.com/):**Windows、Mac OS 和 Linux 的在线备份选项，提供自动备份、文件同步和跟踪更改的能力。

**[Mozy](http://mozy.com/) :** 这项兼容 Windows 和 Mac 的服务提供自动、安全的备份。

**[SugarSync](https://www.sugarsync.com/) :** 一种在线备份服务，为 Windows 或 Mac OS 机器、移动访问和文件共享提供安全和自动化的备份和同步。

## 生产力

生产力是一个非常广泛的话题，有数百种合适的高质量工具。对于这个列表，我主要关注在线任务管理和笔记选项。

**[Evernote](http://www.evernote.com/) :** 一款在线笔记工具，可与桌面应用程序自动同步，并提供移动访问。

**[记住牛奶](http://www.rememberthemilk.com) :** 记住牛奶是一个任务管理工具，允许你在线或在你的移动设备上创建、管理和组织任务。

**[Springnote](http://www.springnote.com) :** 一个基于维基的笔记本，允许你创建可以分享和协作的页面。
 **[Stixy](http://www.stixy.com/) :** 一个免费的笔记服务，可以让你分享笔记和文件。

**[Ta-da List](http://tadalist.com):**Ta-da List 是一款来自 37signals 的任务工具，可以让你创建多个列表，并对每个列表添加、编辑、重新排序和删除任务。

**[Toodledo](http://www.toodledo.com/):**Toodledo 允许你在一个可排序的在线待办事项列表中组织和优先处理你的任务。

**[Uber note](http://www.ubernote.com/):**Uber note 是一款笔记工具，可以让你分享笔记，通过电子邮件添加笔记，书签页面以及通过手机访问。

更多在线任务工具，请查看:“[待办事项列表:12 个组织任务的在线工具](https://www.sitepoint.com/online-to-do-list-tools/)”，并确保通读评论以获得更多伟大的建议。

## 远程存取

**[LogMeIn Pro](https://secure.logmein.com/products/pro/) :** 从任何一台连接互联网的电脑上无缝全面地访问您的家庭或办公室电脑。

**[GoToMyPC](http://gotomypc.com/):**GoToMyPC 可以让你从任何联网的电脑远程访问你的电脑。

## 虚拟电话/传真

**[eFax](http://www.efax.com/) :** 一种虚拟传真服务，提供唯一的互联网传真号码，允许您通过电子邮件收发传真，并在线存储传真。

**[eVoice](http://evoice.com/) :** 一种在线呼叫管理和消息服务，提供免费或本地电话号码、自动话务员和呼叫转移。

**[OneBox](http://www.onebox.com/) :** 一种虚拟 PBX 服务，可以将呼叫转移到任何位置，包括自动助理、语音邮件、传真和会议。
 **[振铃中心](http://www.ringcentral.com/) :** 一种虚拟 PBX 服务，提供即时电话、传真和消息服务，其功能包括免费号码、专用传真线路和多种分机。
