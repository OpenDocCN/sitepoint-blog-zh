# 欧洲浏览器投票今天开始！

> 原文:[https://www.sitepoint.com/european-browser-choice-today/](https://www.sitepoint.com/european-browser-choice-today/)

![browser choice](../Images/cb64177bb0893727eba1a565ff899a2e.png)微软今天发布他们的[“浏览器选择”](https://www.sitepoint.com/microsoft-european-browser-choice/)更新。所有将 Internet Explorer 设置为默认浏览器的欧洲 Windows XP、Vista 和 7 用户都将收到一个自动更新，提供以下选项:

*   微软 IE8(三叉戟渲染引擎)
*   Mozilla Firefox (Gecko)
*   谷歌浏览器(网络工具包)
*   苹果 Safari 浏览器(网络工具包)
*   歌剧(不久)
*   前(三角形)
*   群体(壁虎)
*   绿色浏览器(三叉戟)
*   K-meleon(壁虎)
*   傲游(三叉戟)
*   斯雷普尼尔(可在三叉戟和壁虎之间切换)
*   纤薄(三叉戟)

如果你担心这会从根本上增加你的浏览器测试制度，不要担心。一半的浏览器是 IE 的皮肤版本。Gecko 有四种浏览器，Webkit 有两种，Presto 有一种。如果你已经在测试 IE6+、Firefox、Opera、Chrome 或 Safari，你将会覆盖很大一部分市场。

请注意，用户还可以安装和试用多个浏览器。看看人们是否会尝试会很有趣，但我怀疑很多人会尝试一种替代方案，如果他们不喜欢它，就会切换回 IE。

大问题:*浏览器选择屏幕会对市场份额有什么影响吗？*下面是 2010 年 2 月 [StatCounter 的统计数据](https://gs.statcounter.com/)公元前*(选择之前！)*:

以下是我对下个月的预测:

*   IE 的使用在欧洲会下降几个百分点，但不会对全球数字产生巨大影响
*   从 IE7 到 IE8 会有一个明显的转变
*   IE6 的使用不会减少——许多用户将关闭更新，特别是在公司网络上
*   Firefox 将增加其份额，以赶上或略微超过 IE
*   Chrome 会增长一到两个百分点
*   Opera 和 Safari 不会有太大变化
*   “其他”浏览器会略有增加。

几周后我们会回来找出答案。
