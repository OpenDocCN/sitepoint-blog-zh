# SitePoint CSS 参考封闭测试版

> 原文:[https://www . site point . com/site point-CSS-reference-closed-beta/](https://www.sitepoint.com/sitepoint-css-reference-closed-beta/)

你可能已经听到了一些关于我们在 SitePoint 总部正在开发的一个新的超级酷的参考网站的窃窃私语——老实说，[一些非常响亮的呼喊](https://www.sitepoint.com/in-all-fairness-internet-explorer-still-stinks/)。首先是 CSS 参考，我们的技术编辑 Andrew 已经[声称](https://www.sitepoint.com/the-great-specificity-swindle/)将震撼你的世界！由 SitePoint Forums 资深人士 Tommy Olsson (AutisticCuckoo)和 Paul O'Brien (Paul O'B)撰写，该参考旨在成为 Web 上级联样式表的权威参考。

![The Three Layers of the Web - CSS Reference Beta](../Images/7af499ae2243bc2ede9c4f371484c630.png)

从今天开始，我们推出了 SitePoint CSS 参考的封闭测试版。在接下来的一个月左右的时间里，只有在 2007 年 12 月 1 日之前注册的 SitePoint 论坛成员才能访问。

如果你正在读这篇文章，很有可能你就属于这一类。如果你是，我们希望你能进去，四处看看，戳戳，把它撕开，然后告诉我们你的想法。我们正在寻找各种各样的反馈，从可用性、功能性、性能到你可能在内容中发现的任何打字错误。

在记录您的反馈一个月后，我们将在公开发布之前尽可能多地实施您的建议。

所以，有了你的论坛登录，准备体验前所未有的 CSS 吧…

鼓声…窗帘升起…[reference.sitepoint.com](https://reference.sitepoint.com/)

尽情享受吧！
