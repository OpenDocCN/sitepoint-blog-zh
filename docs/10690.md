# 利用 Tumblr 做更多事情的 5 个工具

> 原文:[https://www.sitepoint.com/tumblr-tools/](https://www.sitepoint.com/tumblr-tools/)

几周前，我写了一篇关于 Tumblr 及其在社交网络领域越来越受欢迎的文章。Tumblr 是一个简短的博客平台，允许用户向他们的“tumblelog”发布文本、图像、视频、链接、引用和音频

鉴于我从那篇文章中得到的反馈，我决定查看一些可以与 Tumblr 一起使用的工具。令我惊讶的是，它们似乎到处都在出现，其中一些与我之前强调的 [Twitter 应用](https://www.sitepoint.com/new-twitter-apps/)非常相似。

这里是我最近检查的 Tumblr 的五个工具、插件和扩展，可能也会帮助你使用 Tumblr。

## [翻滚吧](http://tumbleroo.com/)

一个支持文本、链接、聊天、问题、照片、音频和视频帖子的 iPad 应用程序。如果你有几个 Tumblr 博客需要管理，Tumbleroo 允许你选择发布到哪个博客，并安排发布时间。你还可以点赞、重新写博客、创建私人帖子，并选择是否将它们发送到 Twitter。

## [Tumblr YM](http://tumblrym.blogspot.com/)

![](../Images/b1ba9757d3e17d3d42e62db0bedeeae4.png "tumblr")允许雅虎信使用户直接向 Tumblr 发帖。不需要安装额外的软件。你所要做的就是将 tumblrym@yahoo.com 添加到你的即时消息列表中，并通过发送一条即时消息“注册”来注册

## [TumblrHelper](https://chrome.google.com/extensions/detail/okedoojhfbgmhdkfgojckbbhbffcoihm)

Google Chrome 扩展，允许用户创建新的帖子模板，保存常用设置作为新帖子的默认设置。TumblrHelper 还允许您添加用户注释和删除帖子。

## [翻滚](http://log.ijulien.com/post/193997383/tumblrize)

一个 WordPress 插件，允许你在发布新帖子或更新现有帖子时直接发布到 Tumblr。此外，如果你从你的 WordPress 博客中删除了一篇文章，Tumblrize 也会从 Tumblr 中删除它。你也可以使用这个插件向 Posterous 发帖，并从 WordPress 向 Tumlr 发送元数据，比如你的标签。

## [风滚草](http://tumblweed.org/)

允许用户查看和创建帖子的 Adobe Flash/Air 桌面客户端。它兼容 Windows 和 Mac OS X，并支持大多数 Tumblr 功能。
