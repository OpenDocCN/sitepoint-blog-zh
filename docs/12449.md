# 哦兄弟，；-)表情商标

> 原文：<https://www.sitepoint.com/oh-brother-emoticon-trademarked/>

俄罗斯商人 Oleg Teterin 拥有一家名为 Superfon 的移动广告公司，他已经将标志性的眨眼笑脸注册为商标。-)-在俄罗斯。泰特林说，表情符号的一年商业许可将花费“数万美元”Teterin 说，幸运的是，互联网的普通用户仍然可以免费使用 smiley。

“Rospatent 的决定意味着俄罗斯的任何商业组织都不能在其广告中使用该标志。泰特林说:“它没有涉及在互联网上使用表情符号的自然人。”。

因为其他的表情符号，比如:)，:-)，还有；)与 Teterin 的商标如此相似，它们也不能在俄罗斯合法使用。然而，法律专家认为俄罗斯专利局的商标授权最终不会成立。“这套电脑键盘字符是世界公认的。因此，将其注册为商标可能与公共利益相矛盾，”法律专家维克多·瑙莫夫告诉《真理报》。

2001 年，模仿灵感产品的制造商 Despair，Inc .，[在美国注册了皱眉笑脸](http://despair.com/frownonthis.html)–:-(–作为一个笑话，或者也许是为了指出美国专利和商标制度的缺陷。当时有几家媒体报道了这件事，并为《绝望》做了一些很好的报道。

不过，很难判断泰特林是不是在开玩笑。俄罗斯日报 Kommersant [报道称](http://dispatch.com/live/content/business/stories/2008/12/11/emoticon.html)这已经不是俄罗斯第一次有人试图将这个笑脸表情符号注册商标了。

我们担心谷歌申请专利。
