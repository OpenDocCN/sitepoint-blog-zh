# 你的营销是推、拉还是不适用？

> 原文：<https://www.sitepoint.com/is-your-marketing-push-pull-or-na/>

今天的博客讨论了营销中的一个关键区别:推与拉。在你的营销组合中，关于每种营销风格的适当组合有很多争论。

推式营销是传统的当面吹捧你的服务的营销方式。它包括直邮、广告、电话推销和其他推销服务的技巧。通常，推式方法包括一次推销，试图一步到位地推销你的服务。

拉动式营销把人带到你身边。这是一种更柔和、更低调的方式。它包括使用主动推荐系统、演讲、写作、在社区中引人注目，以及在潜在客户允许的情况下用教育材料跟进。这不是一种被动的方法，仍然需要大量的努力和创造力。

在这篇博客的标题中提到的“不适用营销”是很多 IT 专业人士都在做的事情。这根本不是营销，而是希望人们传播你的信息，然后等着电话铃响。

我的风格和专业经验——以及其他专业人士的经验——倾向于拉动式营销。因此，我投入大量时间写作(在这样的博客中)，演讲，在我的网站上分享文章，并努力将自己树立为业内的“思想领袖”。网页设计师有很大的空间去做同样的事情。

在今天的市场上，大多数潜在客户都过于怀疑，对推销技巧没有反应。正如最近直销协会时事通讯中的一篇文章指出的，推销技巧类似于在你第一次介绍自己的时候向某人求婚。

但也不全是黑白的。这是一个连续统一体。例如，电话推销可以奏效——如果你专注于一个非常特殊的领域，提前研究每一个业务/潜在客户，并个性化你的信息给他们(例如,“我看过你的网站，相信与你最接近的竞争对手相比，它排名第五。我可以帮你做以下 3 件事，让它在吸引更多业务方面处于领先地位……”).

我可以向你展示一些仍然通过传统推送技术获得大部分收入的企业。但是越来越多的，他们似乎是例外。

所以今天的问题是:

一:你的营销是“推”、“拉”还是不适用？

第二:你将如何改变你的营销，以达到你的风格和目标市场的理想组合？
