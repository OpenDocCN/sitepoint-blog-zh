# 停止过度调度！如何收回你的时间

> 原文：<https://www.sitepoint.com/how-to-take-control-of-your-time/>

忙碌的生活导致日程排得满满的，虽然这可能令人满意，但预订和超额预订之间只有一线之隔。可惜，超排的代价可不是那么小。

如果你发现自己从一个约会或任务跳到下一个，并且持续地不能完成你的所有责任，你可能处于危险地带。您可能经历的一些后果包括:

*   错过最后期限
*   受损的客户关系
*   丢失的工作
*   烧坏
*   不平衡
*   挫败感
*   受损的声誉
*   错位的优先级
*   压力增加
*   无法赶上进度

如果你曾经经历过因日程安排过多而导致的缺乏控制，你就会知道管理并让自己脱离这个循环有多困难。但是有可能收回你的时间表。以下是方法。

## 填充您的日历

如果你有很多约会，或者倾向于每天安排一些单独的任务，那么你的日程安排过满的一个原因可能很简单，就是没有为每个会议或任务留出足够的时间。开始给自己额外的时间来考虑那些超出预期的会议和任务。这样，你就有了一些灵活性，可以避免那天你计划的其他事情迟到。此外，你可能会发现，在会议间隙你会有一些额外的时间来休息和重组——这是我们白天都需要做的事情。

## 优先考虑一切

如果我没有有意识地思考，我倾向于优先处理我遇到的所有事情。所以，对我来说，现实地安排优先次序和计划我的时间需要一些努力。如果没有这个优先化步骤，您很可能会很快达到超计划状态。给工作项目分配一个优先级是至关重要的，但是你也应该对所有的个人任务和其他责任做同样的事情，以确保你以一种平衡的方式关注生活中最重要的事情。

## 开始外包

如果你的工作比你能处理的多，也许是时候外包了。虽然建立一个合格的、值得信赖的分包商团队并训练自己去授权可能需要时间，但是一旦你能够成功地做到这一点，你不仅会有更多的时间，而且你会发现你的业务增长会超出你的预期。

## 否定

如果所有这些都失败了，而你仍然超负荷、超压力，考虑拒绝不理想的工作或个人要求。有时候，有些东西必须放弃。很难说不，尤其是对即将到来的工作，但如果它不支持你的业务目标和整体理智，从长远来看，现在通过可能对你更好。
