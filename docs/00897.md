## 关于这本书

正如你将在这本小册子的第一章看到的，每个企业都可以使用网站。这本书不是 HTML、CSS 和 JavaScript 的指南(SitePoint 有大量的[书](https://www.sitepoint.com/premium/topics/all?q=&limit=400&offset=0&page=1&content_types%5B%5D=Book&slugs%5B%5D=all&states%5B%5D=available&order=)涵盖这些基础知识！)，而是贯穿了建立和运行我们自己的网站的基础。我们将讨论获得一个[域名](https://www.hover.com/sitepoint)、[托管](https://www.sitepoint.com/premium/l/sitepoint-premium-siteground)、用 cPanel 管理你的站点的基础知识，甚至讨论你的站点版权的滞后。

这是从 SitePoint 网站的[网络频道](https://www.sitepoint.com/web/)精选出来的简短教程，将帮助你立刻在网上建立自己的网站。它包含:

1.  *为什么每个企业都需要网站*杰夫·史密斯著
2.  *如何选择、注册并充分利用你的域名*
3.  我应该为我的网站选择什么样的主机？由阿德里安尝试
4.  Adrian Try 的《选择托管服务提供商的终极指南》
5.  克雷格·巴克勒的云托管:利弊
6.  杰夫·史密斯的《用 cPanel 管理你的网站的 6 个技巧》
7.  对网站进行版权保护意味着什么
8.  *如何创建一个投资组合网站，让你被 Jacco Blankenspoor 雇佣*

这本书适合初学者。
