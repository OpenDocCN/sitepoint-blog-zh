# PHP 协作项目官方问答

> 原文：<https://www.sitepoint.com/php-collaboration-project-official-qa/>

只是简单更新一下 PHP 合作项目<ins>(简称[PCP](http://en.wikipedia.org/wiki/Phencyclidine)……嗯……)</ins>，昨天我[推测的](https://www.sitepoint.com/zend-and-ibm-to-co-develop-new-php-ide-and-framework/)的传闻。似乎官方声明确实是在 Zend/PHP 大会上发布的，因为 Zend 已经在其网站上发布了关于项目的官方问答。

细节仍然很少，主要是因为没有细节可以分享。实际上，这只是 Zend 邀请开发人员参加它尚未计划的聚会。

谢天谢地，问答中表达的意图似乎与我希望看到的 PHP 发展方向一致。Zend 是否能在下一个冰河世纪到来之前将这个雄心勃勃的概念转化为可行的代码还有待观察。考虑到参与项目的各方数量——更不用说那些希望拥有自己发言权的固执己见的 PHP 开发人员了——这汤有很多厨师…
