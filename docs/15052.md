# 搜索引擎损害控制

> 原文：<https://www.sitepoint.com/search-engine-damage-control/>

谷歌和其他搜索引擎会将任何使用垃圾邮件策略来获得高排名的人踢出局。营销人员应该害怕这些威胁吗？你最好相信它。

但是，如果你怀疑自己受到了惩罚，那么作为损害控制，该怎么做呢？

如果你认为你无意中向谷歌或其他人发送了垃圾邮件，并希望你能改正你的行为，我有一些好消息要告诉你。

##### 重要的事情先来

首先，回顾一下将您的页面从数据库中排除的一些可能原因。例如，谷歌和大多数其他公司会发布一个页面，概述你的网站可能不被收录的原因。看一看每个引擎，以确定您是否无意中违反了他们的指导原则，或者只是没有按照正确的程序被列入数据库。

如果你怀疑你的页面因为垃圾邮件被手动从搜索引擎的索引中删除，搜索引擎可能不会评论删除的原因——他们甚至不会给你一个可能导致你被删除的行为的详尽列表。

##### 垃圾邮件策略

以下是一些被认定为搜索引擎垃圾邮件的常见策略:

*   ***掩盖:*** 当一个页面被提供给搜索引擎爬虫以达到一个好的排名，但是该页面的不同版本被提供给搜索引擎用户。这有时包括在搜索引擎定位完成后改变元标签。

*   ***欺骗/重定向/元刷新:*** 元刷新标签允许访问者自动被带到不同的页面。当这一功能被滥用时，用户会被带到与其搜索无关的内容。因此，搜索引擎对元刷新率快的页面和使用 JavaScript 执行重定向的页面持怀疑态度。如果需要合法的重定向，请使用服务器端重定向。
*   ***域名垃圾:*** 在不同域名下发现的相同站点，以增加搜索引擎流量，也称为镜像站点。
*   ***微小文字:*** 过度使用隐藏关键词填充。
*   ***不可见文本:*** 用于隐藏关键字填充，使填充的关键字与页面颜色相同(例如，白色页面背景上的白色文本)。
*   ***欺骗性标题和标签:*** 标题中无关的关键词和 meta 标签。
*   ***欺骗性/误导性链接:*** 以欺骗搜索引擎为唯一目的设置页面/链接。
*   ***过度提交:*** 使用 AddURL 表单提交数百个欺骗性页面。

##### 被抓

如果你故意用上述任何一种策略向引擎发送垃圾邮件并被发现，你将会被罚从引擎中删除你的链接。滥发不值得一时的利益。搜索引擎营销做得好是一个长期的营销投资，所以不要危及你的排名与任何建议滥发搜索引擎。

搜索引擎有许多方法来检测垃圾邮件，包括所谓的垃圾邮件过滤器。他们还积极鼓励用户举报垃圾邮件。因此，即使你几次躲过了垃圾邮件过滤器，其他人也可能会举报你——尤其是你的竞争对手。

##### 回到谷歌

如果你的网站被从谷歌的数据库中删除，首先要做的是清理页面，并向 help@google.com 发送重新收录请求。谷歌可能不会保证是否以及何时会重新收录你的网站。如果一切正常，你的网站应该会在一个月后下一次谷歌更新时重新出现。

在你重新提交你的网站之前，确保你的服务器没有技术问题:检查任何可能会吓跑搜索引擎蜘蛛的 robots.txt 文件。也要记住，如果你使用框架或 Flash，引擎很难索引你的网站。如果你的网站要被搜索引擎爬虫正确地索引，你还需要在你的页面和标签上有大量的相关文本。

无论如何，你应该以书面形式联系搜索引擎。也试着打电话联系他们。承认你的错误，并真诚地保证不会再犯。

##### 算法变化

你应该担心搜索引擎算法的变化吗？是的，但是处理这些变化有可接受和不可接受的方法。垃圾邮件是完全不可接受的。

目前，引擎强调相关性。算法似乎偏爱相关的内容、相关的标题和描述标签，以及相关的链接策略。换句话说，实话实说，描述要精确。知道哪些关键字被用来找到你的网站，并适当地使用这些关键字。并提供良好的导航，使引擎可以深入到您的网站。

算法已经受到搜索引擎优化实践和用户行为的影响。垃圾邮件和最佳实践就是这样浮出水面的。这也是搜索引擎不断调整算法的原因。

##### 不是镇上唯一的游戏

Google 只是一个搜索引擎，有内容、有诚信、有大量用户的还有很多。所有这些引擎应该是指搜索到您的网站。

每天都有数以百万计的新网页被提交，其中许多网页与你的网站竞争排名。不要通过滥发搜索引擎来拿你未来的网上生意冒险。了解规则并负责任地行事是你的责任。

##### 用引擎工作

 *与所有的引擎合作，使用他们的指导方针，当你犯了错误时承认错误，进行书面联系，打电话跟进，真诚地请求重新进入数据库。

请记住，你的编辑内容(搜索引擎链接到你的网站)在搜索引擎的广告文件夹下是你的企业获得目标受众的最佳途径。这些观众由精明的个人组成，他们进行搜索和研究，对你的工作感兴趣。你不能犯错误。如果你这样做了…马上开始损害控制。*
