# 2001 年的搜索是什么样的？谷歌知道

> 原文：<https://www.sitepoint.com/what-was-search-like-in-2001-google-knows/>

作为其[10 岁生日庆祝](https://www.sitepoint.com/google-celebrates-10th-birthday-gives-away-money/)的一部分，谷歌重新推出了大约 2001 年的老式搜索索引，并拼凑了一个旧搜索引擎的大规模复制品(搜索 13 亿个网址，而不是[1 万亿个](http://googleblog.blogspot.com/2008/07/we-knew-web-was-big.html))。搜索 2001 网站看起来像 7 年前的谷歌(这显然是他们最容易做到的)，包括旧的标志，缺乏垂直搜索链接。

不过，真正有趣的是搜索旧索引。谷歌的博客[提出了一些建议](http://googleblog.blogspot.com/2008/09/2001-search-odyssey.html):“[iPod](http://www.google.com/search2001/search?q=ipod)不是指音乐播放器， [youtube](http://www.google.com/search2001/search?q=youtube) 是胡说八道，如果你在找“[迈克尔菲尔普斯](http://www.google.com/search2001/search?q=Michael+Phelps)”，你很可能指的是科学家，而不是游泳运动员

搜索“ [SitePoint](http://www.google.com/search2001/search?hl=en&q=SitePoint) ”只能找到 2920 个结果，而今天却有超过 180 万个，包括旧论坛网址(sitepointforums.com)的链接，还提到了我们的一句老掉牙的口号:“帮助在线业务增长！”

谷歌 2001 年的搜索在其索引中包含了到互联网档案馆版本的链接，这意味着你可以像 7 年前一样搜索 T2 和浏览。你还记得 SitePoint 什么时候长得像[这个](http://web.archive.org/web/20010903105205/www.sitepoint.com/)吗？

其他一些有趣的搜索:

*   [ [ruby on rails](http://www.google.com/search2001/search?q=ruby+on+rails) ] =一种搁架单元
*   颇具预言性的是，这是一个哈佛学生的可搜索数据库
*   伯克利的科学教育门户
*   [ [firefox](http://www.google.com/search2001/search?q=firefox) ] =个人主页
*   “高级森林产品”的制造商威拉米特工业公司
*   [ [android](http://www.google.com/search2001/search?q=Android) ] =反恐精英的机器人
*   出于某种原因，Netscape.com
