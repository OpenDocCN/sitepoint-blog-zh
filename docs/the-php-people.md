# PHP 人

> 原文：<https://www.sitepoint.com/the-php-people/>

如果你曾经被一个问题困住，谷歌一下，你会发现一群用户很可能经历过同样的问题，并且已经分享了他们的解决方案。如果不存在，在公共论坛上提问，人们会帮你找到答案。如果你已经自己解决了，那就写下来！这样，你就能像别人愿意帮助你一样帮助别人。

这是 PHP 社区真正脱颖而出的一部分——人们分享他们的知识，并且非常愿意在他们的 PHP 之旅中帮助他人。如果你加入了一个 IRC 频道，里面会挤满渴望帮助你的人。如果你在 [Stack Overflow](http://stackoverflow.com/ "Stack Overflow") 或者任何论坛上发布什么东西，也会有人一窝蜂的帮你找解决方案。

大约在 2000 年，我加入了我的第一个 PHP 相关的 IRC 频道。我是那些问最琐碎问题的菜鸟之一，但频道里的人很有帮助，我从他们那里学到了很多。我甚至尊敬他们，他们知道一切是多么的酷！从那时起，我看到社区欢迎许多其他人，就像他们欢迎我一样，欣然张开双臂。

## PHP 社区“IRL”

在 PHP 社区活跃了多年之后，我决定更进一步，参加我的第一个 PHP 大会， [PFCongres 2011](http://www.pfcongres.com/ "PFCongres - The Web Development Congres") 。作为一个女人，我真的不知道会发生什么。这些家伙在 IRC 和论坛上很友好是一回事，但是他们在现实生活中是什么样的呢？

那是 9 月 17 日的一个清晨，我乘火车去乌得勒支，我的第一次会议就在那里举行。我正要进入主厅，突然碰到一群极客站在门外。结果他们中的一些人已经从 IRC 上知道我是谁了！他们在现实生活中就像在网上一样友好。

PFCongres 2011 分为两个部分，这有时让我很难决定我想参加哪一部分。一首是荷兰语，另一首是英语，因为我的荷兰语还不是很好，所以我大部分时间都在听英语课。不考虑语言，我看到许多主持人，我认为他们是天生的专业人士，他们应该永远在舞台上。

我参加的下一个发布会是 [PHPNW11](http://conference.phpnw.org.uk "Home | PHPNW11") 。这个会议绝对比国会大，而且持续了不止一天。在那里，我看到了更多精彩的演讲，但也体会到了一些其他的东西——强调所有演讲、走廊轨道和社交活动的社区欢迎感。

走廊轨道是一个很好的机会，可以和那些和我一样或者比我更讨厌的人交流，分享他们的知识。PHP 社区的优势在于，无论你是谁，他们都非常愿意分享你的经历，并且对你的经历真正感兴趣。如果你是一个女人，他们仍然把你当成一个程序员。如果你刚刚开始学习 PHP，他们愿意帮助你开始。

有些人去参加会议只是为了社交方面的目的，会见老朋友，结交新朋友。想象一下，你去一个酒吧，但是你很害羞，在那里你一个人都不认识。你有多大可能整晚坐在那里啜饮你的饮料？如果你是一名女性，可能会有一些人试图搭讪，但他们大多都很猥琐。在 PHP 社区中，情况完全不同。如果有人看到有人在回避，他们会试着让他们加入谈话。

我从 PFCongres 2011 回到家，受到了 PHPNW11 的启发。我只是忍不住告诉每个人我度过了多么美好的时光。有如此多的新面孔，如此多的精彩演讲，我受到了尝试新事物的启发…我的热情甚至感染了我的男朋友，他以前从未编程，但现在正忙于编写他的第一个 PHP 脚本！

## 这不仅仅是关于 PHP

会议上有一种友好的气氛，由于女性的影响，也有一种非常欢迎女性的气氛，但这只是表面。如果你深入观察 PHP 社区，你会发现朋友，那些互相关心的人。我想讲述一个关于伊丽莎白·纳拉莫尔的可怕故事，她是一个 PHP 爱好者，也是 PHPWomen 的总裁和联合创始人。

伊丽莎白即将在 10 月 25 日庆祝她的生日。PHP 社区中有人想:嘿，让我们给她一个有趣的礼物吧！突然，社区里的很多人毫不犹豫地加入进来，让它成为现实。当她的生日到来时，伊丽莎白不知道她将得到什么 PHP 社区的许多人送的一份美妙的礼物，一个有着有趣字符的粉红色键盘！她觉得很好笑，非常感动。

我经常看到人们与社区分享非常私人的事情，许多人会停下他们正在做的事情来帮助那个人，不管这有多尴尬或私人，以确保他们没事。我是那些在生活艰难时从 PHP 社区得到安慰的人之一。这是一种非常令人欣慰的感觉，不管生活中发生了什么，我永远都是受欢迎的，我永远都有你们——我在 PHP 社区的朋友。

## 摘要

PHP 中的社区非常庞大，并且每天都在增长，这一切都是为了分享。如果你看到有人需要帮助，你可以帮助他，给他提供指导。如果你看到一个开源项目，那很好；贡献并帮助它成长。即使是很小的事情也会有所不同，比如为一个项目写文档或者留下反馈。这是许多人正在做的事情，它以一种方式帮助了人和项目的成长，如果人们不愿意分享他们的一些时间，这是不可能的。如果你还没有参加会议，试着去参加吧。世界各地全年都有几个计划。这是一个很好的机会，所以面对面与他人分享，建立持久的友谊。
