# 自由职业者需要简历吗？

> 原文:[https://www.sitepoint.com/freelancer-resume/](https://www.sitepoint.com/freelancer-resume/)

我们知道，简历本质上是让你获得面试、会议或其他特定机会的工具。它概括了你的教育和经历，也可能是你对未来老板或同事的第一印象。我们大多数人在职业生涯的某个时候都用过。

但如果你是一名全职自由职业者或企业主，你需要为简历费心吗？答案是有也没有。

当你可能需要一份简历时

当你自己做生意时，提供证明经验的例子、相关的成功案例和客户推荐通常变得更加重要。简历并不是最好的格式；下面是一些修改简历格式的方法，以满足你作为企业家的需求。

那么在什么情况下你可能需要重新整理和修改你的简历呢？以下是一些可能性:

*   你要回去为别人工作了。
*   你正在申请加入一个专业组织。
*   您正在申请特定行业的认证。
*   你申请作为一个会议的演示者或参展商。
*   你正在竞标一家大公司或机构的工作。

如何在简历中加入自由职业者

如果你在“我需要简历”小组，你需要将你的自由职业或企业所有权努力与你可能拥有的任何其他经验结合起来。

作为一个企业主，你可以简单地把你公司的信息和你的其他经历放在一起。如果你管理一个团队，你也可以详细介绍你的工作类型，以及你合作过的客户类型。

作为一名自由职业者，这就有点棘手了，尤其是如果你已经为一些短期项目和很多公司合作过。你要避免给雇主留下这样的印象:你只是在失业期间做自由职业来填补时间上的空缺。

做到这一点的最好方法是确定你工作过的每一家公司，并把你的头衔列为自由职业者。如果相关的话，你也可以简单描述一下你所做的工作。如果你被要求提供一些自由职业客户作为参考，这也是一个好主意。

我还应该注意，在许多情况下，当你申请会员资格或认证时，你可能会被要求提供一份简历。简历基本上是一份加强版的简历；它比标准简历更长，包含的信息也更多。想了解更多关于写简历的信息，请看这篇关于 About.com 的简历指南。

传统简历的替代品

作为一名企业主，我确实偶尔会被要求提供简历，但这种情况并不多见，而且几乎总是另一种形式更有意义。很多时候，客户只是不知道该问些什么来验证你的体验，而你有责任为他们提供做出决定所需的信息。

除了传统的简历，这里有一些方法可以让你向潜在客户展示你的背景和经历:

*   网站——你的“关于”页面应该给客户足够的信息，让他们了解你是谁，你做什么。结合你的投资组合，这可以成为一个强大的销售工具。
*   宣传册——你可以创建一份印刷或电子文档，目的只是让你的客户了解你。它可以包括一份简历，一些案例研究，常见问题，一张图片，甚至一点个人信息。
*   音频/视频——考虑制作一个音频或视频传记，让客户更好地了解你。看到你的脸，听到你的声音，会让客户更有可能信任(并雇佣)你。

你有简历吗？这对你找到更多工作有帮助吗？
