# 最好的演出，以及如何得到它们

> 原文：<https://www.sitepoint.com/the-best-gigs-to-get-and-how-to-get-them/>

这个星期，一个客户雇佣我做一个大项目。回顾他们是如何成为客户的，并继续剥洋葱皮，这是一件有趣且有启发性的事情:

当他们看到我在一个类似领域的组织演讲时，他们听说了我，几个月后当他们有需要时，他们打电话给我。

由于前一年我在一家类似的公司做过一些工作，我得以在那个组织发言。

由于那家公司簿记员的介绍，我前年开始在那家公司工作。

当我在一家小公司做兼职记账员时，那家公司的记账员了解了我。尽管她与这个项目无关(除了支付我的发票)，但她喜欢我，我们一直保持联系。原来她认识很多我锁定的公司的人！所以她成了推荐人的来源。

在为一个协会做了一些战略工作后，我开始为那家小公司工作。那家公司的老板是会员。

最后，在培养了与该协会的关系并为其成员免费演讲后，我赢得了战略工作。

从开始到结束，这个过程用了大约 4 年时间。现在，由于良好的工作记录、可见性和坚实的成果，客户联系我。这无疑是获得理想项目和客户的最有回报的方式。

也是可以预见的。你无法预测谁会给你打电话，但如果你能取得好成绩，并且知道如何寻求推荐，你就会以这种方式获得项目。

你是如何进行这种营销的？
