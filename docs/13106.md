# 你的客户在推广你吗？

> 原文:[https://www.sitepoint.com/are-your-customers-promoting-you/](https://www.sitepoint.com/are-your-customers-promoting-you/)

在这篇文章中，我将要讨论的是，在客户服务方面多做一点，如何能给你的企业底线带来巨大的变化。

最近我的汽车排气管坏了。我需要修理它，但是在哪里？我把它带到最近或最便宜的地方了吗？

没有。我把它带到了 4 年前为我的旧车修理排气管的地方——115 英里(185 公里)外。*为什么？*

**客户服务。**

上次我去那里时，他们不仅在排气方面做了一流的工作，而且在客户服务方面也做得更好。

*   他们营业时间超过了正常关门时间，所以我们可以去取车(因为我们已经晚了)
*   他们同时免费修复了另一个小问题
*   他们很友好，以顾客为中心
*   他们把坏掉的排气管带来的烦人的刺激变成了积极的体验
*   他们甚至让我们的小男孩玩了一天遥控汽车

最终，作为顾客，我被赋予了与众不同的感觉，而不是一个“移动钱包”。他们做了他们说过要做的事，甚至更多。我很乐意放弃我辛苦赚来的钱。

当我回来时，我失望吗？

*不*——再一次，完美的服务更进一步。

这种卓越的服务对企业有什么帮助？

*   他们得到了回头客
*   他们被快乐的顾客(比如我自己)积极地免费推广
*   他们在营销和推广上省钱
*   他们开始为给人们的生活带来微小但积极的改变而感觉良好

所有这些都转化为更健康的底线。简而言之，优质的客户服务具有极佳的商业意义。

我是怎么发现他们的？
有人向我推荐了它们——当然。

**你如何超越顾客？**
