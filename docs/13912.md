# 与付款慢的人打交道

> 原文:[https://www.sitepoint.com/dealing-with-the-slow-payers/](https://www.sitepoint.com/dealing-with-the-slow-payers/)

一位客户刚刚创下了我职业生涯中付款时间最长的记录:大约六个月。

这是一所公立大学，因此有特定的官僚规则。另外，他们事先为我做了一点破例，这导致了后来的问题。

如果我没有正确跟进，我永远也不会得到报酬。

如果你有迟到或付款缓慢的客户，这里有一些提示:

1.礼貌但坚定。

2.让他们知道你已经完成了工作，并期待及时付款。

3.找出获得报酬的流程，并设定一个时间表。这样，你就可以跟进，把球踢回他们的球场。

4.如果客户错过了最后期限，或者消失了一段时间，再次联系他们。要有礼貌。解释他们错过了截止日期，并且你很关心能否及时拿到工资。我喜欢这句话:“我依靠及时的付款来养活我的家人和支付账单。”

5.继续跟进。问问你是否能做些什么来加快这个过程。我发送简短的电子邮件说:“只是检查付款状态。我还能做些什么来加速付款吗？”

6.不要做出任何威胁或下流的评论。永远不要威胁采取法律行动，除非你准备投入时间、精力和金钱。

7.接触各种决策者。就我而言，这所大学的应付款部门糟糕透了。但是一旦我复制了我的赞助商(一位副校长)和一位热爱我的工作的教授，事情就有了进展。

还有什么？
