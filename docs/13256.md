# Ruby on Rails:简单的艺术

> 原文:[https://www . site point . com/ruby-on-rails-the-art-of-simplicity/](https://www.sitepoint.com/ruby-on-rails-the-art-of-simplicity/)

大家好，我是 Myles Eftos，我是你们新的 Ruby on Rails 博客作者！我入侵 rails 已经快两年了，开发了许多在线应用，比如我的时间追踪系统: [88 英里](http://www.88miles.net "Simple time tracking")。我一直在努力达到 Rails-zen 状态，即简单漂亮的代码和强大功能之间的和谐平衡。

对于那些还没有被 Rails 迷上的人来说，这是你即将到达的 web 开发天堂(是的，我是一个 Rails 迷——我已经警告过你了)。我曾经在几乎所有的 web 平台上工作过，Rails 是我最喜欢的——用 PHP 需要几个小时的东西用 Rails 只需要几分钟。内置于数据库迁移和部署系统中的完全独立的开发环境意味着，在其他人完成设置 XML 配置文件之前，您就可以发布 Rails 应用程序。

即将发布的 Ruby on Rails 2.0 版本将再次加强该框架的简单性和强大功能，并进行大量改进，让普通 web 开发人员的生活变得更加轻松。为不同的设备提供不同版本的页面是小菜一碟，有了新的多视图功能，安全模型得到了改进，基于控制器的新奇异常处理程序使丑陋的嵌套 if 语句成为过去。

我将尝试在整个博客中呈现一个良好的主题分布，从初学者主题到更高级的前沿技术，但这最终将由你来驱动。请提出你可能有的任何问题，我将尽我所能回答他们。我也潜伏在 sitepoint 论坛周围，等待着回答任何关于 Ruby on Rails 的问题。
