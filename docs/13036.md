# SitePoint 正在寻找一名新的执行编辑

> 原文：<https://www.sitepoint.com/manage-the-sitepoint-publishing-team/>

我没有做过这方面的研究，但我敢打赌，登招聘广告时最常用的一句话是“一生一次”。

不使用这些确切的词语来描述 SitePoint 当前可用的角色是很困难的，但我会尽力而为。

当 Simon Mackie 选择从英国来到 SitePoint 担任我们印刷出版团队的执行主编时，他承诺要呆 18 个月。五年后，他决定最终搬回家，离家人更近，这意味着我们要扮演一个重要的角色。

西蒙的离开是别人的收获。

如果你(或你认识的人)有很强的项目管理技能，领导团队的经验，以及对网络的热情，那么我们希望你能与我们联系。

这确实是一个独特的角色。它令人兴奋，变化多端，富有挑战性。想出新书名的点子；委托作者；管理出版流程(以及参与生产我们产品的优秀团队)——这些都是这个职位的一部分。合适的人可能会有一些技术知识(你甚至可能拥有一两本我们的书),但无论如何不需要成为一个铁杆代码猴子——只要是一个对网络充满热情的优秀沟通者，并能在截止日期前管理一个项目。

这个职位不是一个可以远程执行的职位，所以你要么需要住在澳大利亚的墨尔本，要么准备搬到这里。事实上，Simon 逗留的时间比他预想的要长，这既是对这个角色的肯定，也是对在 SitePoint 工作的肯定。加入我们的行列无疑是一个激动人心的时刻。

**[阅读更多关于](https://www.sitepoint.com/jobs/)这个职位的信息，以及[在 SitePoint](https://www.sitepoint.com/jobs/) 工作的感受。如果你认为自己很适合，就在网上申请这份工作吧。**
