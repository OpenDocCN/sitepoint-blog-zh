# 充分利用您的离线时间:使用 Pocket 完成所有工作

> 原文:[https://www . site point . com/make-the-most-your-time-offline-save-articles-videos-and-images-for-late-view-with-pocket/](https://www.sitepoint.com/make-the-most-of-your-time-offline-save-articles-videos-and-images-for-later-viewing-with-pocket/)

![Pocket (formerly Read it Later)](../Images/27ed0a4f47a63bcec68a52fa3f561fa0.png)

“内容为王。”自从古腾堡发明了活字印刷术以来，这三个词听起来是正确的，但是当你在网络环境中看待它时，它甚至更重要。网络不再仅仅出现在台式电脑显示器上，而是出现在笔记本电脑、上网本、智能手机和平板电脑上。它甚至出现在手表、电子阅读器和电视上！随着网络无处不在，内容看起来更像是一个霸主，而不是国王。我们被文章、图片、视频等等淹没了，但是谁有时间把全部注意力放在这些无止境的信息上呢？

这里是[口袋](https://getpocket.com/ "Pocket")(原**后读**)可以帮忙的地方。

新设计的口袋可能是当今最好的阅读应用之一。许多人会同意；该服务目前有 450 万用户，用户保存了超过 2 亿件商品。它还被集成到 300 多个网络和移动应用程序中，如 Evernote 和 Google Reader。但是，是什么让 Pocket 成为如此出众的阅读应用程序，而其他应用程序已经有几十个了呢？我们来看看安卓版的就知道了。

的共享功能还包括更多选项，如复制链接、电子邮件文章视图和报告文章视图。让我们来探索如何通过 Pocket 查看内容，以了解这些不同的视图。</p> <hr/> <p><img class="alignnone size-full wp-image-4456" src="../Images/e6067c7fa848e71f7570212159062d03.png" alt="" <p>如前所述，你可以阅读保存到 Pocket 中的文章、查看图片或观看视频。视频和图像浏览器是非常标准的，所以我没有在这里包括这些截图。查看图像只需在 Pocket 中调出图像，就像在手机浏览器中查看一样。Pocket 支持观看来自 YouTube、Vimeo、TED 和吞食的视频，他们正在通过未来的更新添加更多的来源。现在，让我们集中讨论如何通过 Pocket 查看文章。</p> <p>口袋中的物品包含在两个工具栏之间。顶部工具栏包括图标，让您在文章视图和原始 Web 视图之间切换。刷新图标将在两个视图中刷新页面。原始 Web 视图将向您显示文章在原始 URL 上的外观，如果您需要查看 Pocket 在解析文章时可能去掉的任何特殊格式，这将很有帮助。</p> <hr/> <p><img class="alignnone size-full wp-image-4452" 的默认颜色是浅色，但是你也可以切换到深色和棕褐色的配色方案。(注意:深色配色方案也会改变整个应用程序的配色方案。)共享图标将允许您使用我之前详述的相同选项来共享文章。</p> <hr/> <p><img class="alignnone size-full wp-image-4447" src="../Images/8432d161e2bafed6fbd522c5255f3408.png" <p>长按内容中的任何单词都会在页面顶部显示另一个工具栏；这允许您将选定的单词(或短语，如果您扩展了选择范围)复制到剪贴板，通过前面详述的默认共享方法共享剪辑，在打开的文档中查找单词或短语，或使用该剪辑执行 web 搜索。单击最左侧的复选标记将关闭此工具栏。</p> <h3>设置</h3> <p>点击主屏幕上的菜单图标，将显示一个下拉列表，用于批量编辑(您可以在其中标记、删除、收藏或归档主屏幕上的多条内容)、设置和帮助部分。现在，让我们把重点放在设置上。</p> <p>Pocket 的设置相当简单。设置分为六个部分:<em>关于</em>、<em>阅读</em>、<em>离线下载</em>、<em>同步</em>、<em>列表</em>和<em>关于。</em></p> <p>在<em>账户</em>下，你可以编辑你的账户详细信息(用户名、密码、电子邮件地址)，注销，或者管理你的网站订阅(这在 Android 3.0+上显然是不可用的)。</p> <p><em>阅读</em>部分给你提供阅读文章的选项。默认情况下，Pocket 会在文章视图中打开文章，但是您可以让 Pocket 根据原始文章的格式选择您认为最好的内容。还有一个选项可以让你使用设备的音量按钮翻页，而不是滑动。</p> <hr/> <p><img class="alignnone size-full wp-image-4454" src="../Images/bc54df853c61cee70695067afb2cec1a.png" alt="" loading="lazy" 如何存储你保存到服务中的文章信息。这里有一些选项来管理当前保存的内容，选择一个移动用户代理，或者只在 Wi-Fi 上下载文章。您可能需要调整一下这里的设置，以确定最适合您的工作流程。</p> <p>默认情况下，Pocket 同步在后台进行，但您可以在<em>同步</em>部分对此进行更改。您也可以将应用程序设置为打开时同步。</p> <p><em>列表</em>部分为您的主屏幕提供了一些选项，例如以未标记的方式开始查看(以防您已经标记了一堆媒体),或者从最新到最早对您的列表进行排序，反之亦然。你也可以将应用程序的界面切换到深色主题，以便在夜间观看。</p> <p>最后，还有一个关于 T1 的 T0 部分，其中包括关注 Pocket online(脸书和 Twitter)的社交链接，以及该应用所用资源的积分链接。</p> <h3>缺点</h3> <p>如果我说 Pocket 是一个完美的应用程序，那我就失职了。虽然它为阅读体验做了很多事情，但我确实发现了一些小问题。</p> <p>方位支持用口袋感觉有点迟滞。如果你向左或向右使劲摇一摇你的手机，你会看到一个图标在屏幕中间一闪一闪的，旁边有一个箭头。Pocket 允许您以横向或纵向模式阅读内容，如果您想用手指在页面之间滑动，或者使用音量按钮翻页，您可以锁定模式。但是当你切换模式时，文章可能会跳到不同的地方，在某些情况下，文章的一部分会被屏蔽掉。退出文章，然后在主屏幕上再次点击它可以解决这个问题，但是如果选项中有这个功能就更好了。</p> <p>我发现的另一个小问题是缺乏阅读文章的字体。Pocket 允许你在衬线字体和无衬线字体之间切换(我相信是 Avenir)。虽然我发现这两种字体对于较小的文章来说很容易阅读，但对于较长的文章来说，sans serif 字体确实变得有点难以阅读。增加字体大小使无衬线字体更容易阅读，但话说回来，它使文章更长。这是一种权衡，但我希望 Pocket 的未来版本至少允许用户使用系统字体来阅读文章。</p> <h3>结论</h3> <p>Pocket
