# 非销售与顾问式销售

> 原文：<https://www.sitepoint.com/the-unsell-vs-consultative-selling/>

之前的一篇文章讨论了咨询式销售与我所谓的“非销售”(例如，根本不向客户推销解决方案)。这里有必要单独写一篇博客来比较和对比这两者。

首先，unsell 不是一种广泛的销售哲学，而咨询式销售才是。在我大力提倡的顾问式销售中，你和客户一起为客户开发最好的解决方案。它包括问很多问题、合作和协商。你是一名顾问，与客户坐在同一桌边，与他/她/他们一起工作，帮助他们取得成果。这一过程可能需要一段时间，但非常值得努力。在我的咨询行业，这是一个强制性的过程，也是达成交易的唯一途径。

博客中提到的“unsell”来自类似的哲学。在“取消销售”中，你基本上后退一步，让客户决定怎么做。你不推，至少不用力。你确实会问一些关于客户需求的很好的问题，并提供信息来帮助他们做出决定。然后你退后一步，让他们来决定。有趣的是——像这样后退更有可能吸引客户，尤其是与更具攻击性的技术相比。

两者都不是被动的方法。它们都需要很好的提问、倾听和解决问题的技巧。

“unsell”实际上只是一个在与计算机技术人员的一次特别好的经历后创造的术语。他问了很多问题，帮助我得到正确的系统，然后后退一步，所以我在做决定时没有压力。对他来说，不卖是他的咨询哲学的一部分，尽管他可能不会使用这些语言。
