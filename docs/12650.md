# 谷歌博客搜索——主流的迷因追踪器？

> 原文：<https://www.sitepoint.com/google-blog-search-the-memetracker-for-the-mainstream/>

![](img/08c10705cdbefed8318ed31602bf7107.png) [谷歌博客搜索](http://blogsearch.google.com/)推出了一个新的主页，以类似于[谷歌新闻](http://news.google.com/)或 [Techmeme](http://www.techmeme.com/) 的风格突出趋势并对相似的帖子进行分组。这将使博客搜索更像是一个目的地。

在过去，我主要使用博客搜索来搜索最近话题的新闻观点(因为常规的谷歌搜索可以很好地找到我已经知道存在的旧博客帖子)，并且实际上发现 [Technorati](http://www.technorati.com/) 通常在减少噪音和找到最近的高质量帖子方面提供更好的结果。通过添加一个 memetracker，博客搜索作为日常访问变得更加有用。

但是，如果它能击败我已经访问过的网站，也就是 Techmeme，它只会进入我的每日(或每小时)网站轮换。

下表比较了目前谷歌博客搜索技术、Techmeme 和谷歌新闻科技上的十大新闻。

|   | **博客搜索** | **科技迷** | **谷歌新闻** |
| 01 | iPhone NDA 掉了 | iPhone NDA 掉了 | iPhone NDA 掉了 |
| 02 | iTunes/音乐价格上涨 | 微软云操作系统 | 洛杉矶火车事故短信 |
| 03 | 网飞溪流 Starz | 谷歌博客搜索 | 工作室 vs. RealNetworks |
| 04 | 工作室 vs. RealNetworks | 网飞溪流 Starz | 网飞溪流 Starz |
| 05 | 微软搜索服务 | 脸书部件死亡 | iTunes/音乐价格上涨 |
| 06 | iPhone 的 Flash | Mac 笔记本电脑销售 | 微软云操作系统 |
| 07 | 微软云操作系统 | 谷歌手机销售预测 | 微软搜索服务 |
| 08 | 亚马逊 EC2 窗口 | 哥伦比亚广播公司公民新闻 | 火狐/IE 隐身模式 |
| 09 | 云计算蠢 | Skype 中国监控 | 美国宇航局 50 岁生日 |
| Ten | GIMP 2.6 | 暴雪机器人支付 | 偶然发现新功能 |

因此，似乎所有三个 memetrackers 都识别出或多或少相同的内容。测试将: **1。**他们找到它有多快，还有 **2。**他们展示得多好。在显示方面，我的钱在 Techmeme 上，它在一个页面上显示了*更多的*内容，而且比谷歌的新资产更优雅、更易读。

在接下来的几天里，我们将进行一些测试，以确定哪一个能更快地找到和更新内容，但如果我对谷歌新闻的经验有任何指示的话，Techmeme 也将是赢家(谷歌目前博客搜索的十大故事中有几个是昨天或前天在 Techmeme 上排名第一的——这是一个不好的早期迹象)。

尽管存在缺点，谷歌博客搜索仍有潜力获得比 Techmeme 更广泛、更主流的受众。Techmeme 的读者主要是早期采纳者和其他博主——那里的链接会从博客中创建链接，但通常不会产生太多流量。谷歌博客搜索理论上可以带来一些真正的流量(众所周知的谷歌新闻),这完全是基于其背后的品牌力量。
