# 如何在你的博客上获得更多评论

> 原文:[https://www.sitepoint.com/how-to-get-blog-comments/](https://www.sitepoint.com/how-to-get-blog-comments/)

啊啊啊……蟋蟀唧唧的叫声。很可爱，不是吗？可能没有什么比在你的博客上承诺一个定期发布时间表更令人沮丧的了，而且从来没有得到任何评论。当然，这并不像被巨魔跟踪那么糟糕，但是当你看起来只是在自言自语的时候，肯定不容易找到继续前进的动力。

在大多数情况下，我们知道，如果你定期发布高质量的信息，并引导人们对你的博客感兴趣，人们就会阅读。但是，不幸的是，大多数网络用户是潜伏者，而不是贡献者。当你的博客聚焦于商业话题、技术焦点或其他通常对大众没有吸引力的主题时，这种情况会变得更加复杂。

那么你能做什么呢？那么，最好的办法就是尽可能持续地发布好的内容。内容是你博客中最重要的部分，如果它有趣且信息丰富，你可以开始运用这些技巧来获得更多的博客评论。

*   **宣传你的帖子:**想要评论，就需要读者。如果你想要读者，你需要让人们知道你的博客和你写的东西。使用 Twitter 和脸书等社交媒体网站上的 feed 功能，让你的帖子得到一些宣传，并确保你在网站和其他营销材料上推广你的博客。
*   **征求意见:**在你的大多数帖子中，通常都有一个地方，你可以向你的读者提出一个特定的问题，比如他们会如何处理某种情况，或者他们对你的观点有什么看法。在更普遍的基础上，让人们知道你的博客总是欢迎评论，并告诉人们做出贡献是多么容易。
*   **回复评论:**当你收到评论时，花点时间回复它们会引发对话，让评论者感到被重视，并有助于吸引潜在者发表意见。
*   在以后的文章中使用评论:即使你的评论很少，当你得到一个非常好的评论时，也要好好利用它。你可能会受到启发，写一篇后续文章，解决评论者提出的问题。通过名字链接到原始评论和评论者是开始后续文章的好方法。
*   **创建投票:**投票有趣、互动且易于操作。他们给你的读者一个机会来权衡，而不必花时间写评论。然后你可以利用投票结果与你的读者展开对话。
*   **评论其他博客:**作为一个博客作者，你可能会阅读其他与你内容相似的博客。试着做一个贡献者，而不是一个潜伏者，定期在能引起你共鸣的帖子上发表评论。只要确保你的评论是适当的和真实的，而不仅仅是试图产生一些反向链接。
*   **链接到其他博客:**浏览一些你喜欢的博客，链接到与你的博客相关的帖子。这不仅会鼓励其他博主阅读你的帖子，还可以用额外的信息支持你自己的帖子。

你的博客上有很多评论吗？你做什么来鼓励参与？
