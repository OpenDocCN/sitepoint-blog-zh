# 三年内收入翻倍

> 原文:[https://www . site point . com/double-your-revenues-in-three-years/](https://www.sitepoint.com/double-your-revenues-in-three-years/)

我第一次碰到 Double 的时候没有把二加二放在一起(给我发的是复习文案)。在我开始阅读之后，我意识到正是这个卡梅伦·赫罗尔德做了我去年发布的 ted 演讲——[教你的孩子成为企业家](http://www.ted.com/talks/cameron_herold_let_s_raise_kids_to_be_entrepreneurs.html)。

Double Double 不仅仅是分析师的另一本商业书籍。卡梅隆在过去的 20 年里一直在经营公司，包括 1-800-get-JUNK，在首席运营官任职期间，他帮助该公司在短短六年内从 200 万美元增长到 1.05 亿美元。

与那些充斥着理论和预测的商业畅销书不同,《Double Double》充满了实用的、可操作的建议。这是一本打造世界级公司的食谱。根据 Herold 的说法，这本书的目标是教你如何在短短三年内将业务翻一番(相当于年增长率不到 30%)。

在书的开头，Herold 建议创建一个你希望你的公司在三年内达到的目标。这张“图画”应该非常详细，描述你公司的每一个方面。你将有多少员工，他们的角色是什么？你会做什么样的营销？你会受到什么样的压力？你是如何得到资助的？公司的财务状况揭示了什么？Herold 附上了他的公司“后袋首席运营官”的照片以供参考。

他完成了第一部分，给出了如何在你的公司创造世界级文化的建议。他的建议，比如每年给五周的带薪假期，对一些人来说可能听起来很疯狂。但这是有意义的——创造一个人们敲门去工作的地方，你会有一个了不起的最聪明的人为你工作。

这本书的第二部分概述了如何发展你的公司。它包括招聘、沟通、营销、会议、公共关系、生产力和技术等章节。赫罗尔德描绘了一幅如何经营公司的蓝图。虽然这些信息对一些人来说似乎微不足道，就像简单的常识一样，但以我的经验来看，很难找到一家公司真正遵循他的建议。这一节中我最喜欢的一章——如何在增长缓慢的时候实现增长。Herold 展示了如何在衰退中发展公司并“窃取”市场份额。

这本书的最后一部分集中讨论了领导力。我个人觉得，大多数创业者开始创建一个人的企业，可能是因为他们厌倦了为别人工作。我们很多人没有经验/领导/人。我读过很多领导力方面的书，但这一部分与你的典型建议非常不同。这是一些实用的建议，比如“设立一个顾问委员会”

在这本书的最后，他包括了一些额外的信息，比如可能的面试问题(这对我特别有帮助，因为我的两家公司现在都在招聘)。

总的来说， [Double Double](http://www.amazon.com/Double-Your-Revenue-Profit-Years/dp/1608320995/) 是发展你公司的绝佳路线图。即使你已经经营公司超过十年(像我一样)，你也会从这本书中学到一些东西。事实上，这本书给我留下了深刻的印象，我在完成我的评论后立即买了一本。
