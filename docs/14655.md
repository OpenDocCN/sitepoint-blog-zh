# 我们开始吧

> 原文:[https://www.sitepoint.com/lets-get-started/](https://www.sitepoint.com/lets-get-started/)

我很高兴被选中来写这篇博客！谢谢乔治娜和 Sitepoint 的员工对我的信任。

我的职责是每周发布大约 3 篇博客(每月一个专栏)。写了很多。虽然我很乐意写我目前的咨询客户在营销和销售 IT 服务时面临的问题，或者只是谈论常见的营销和销售问题，但我更愿意在这篇博客中直接回答您的问题。

那么，你有一个关于经营你的网络业务的紧迫问题吗？我的专长尤其是营销和销售方面。

如果是这样，请回复本部分的任何博客，或者直接给我发电子邮件(在主题标题中注明 Sitepoint 博客的问题)，地址是[andrewn@itprosuccess.com](mailto:andrewn@itprosuccess.com)。

例如，乔治已经写道，当他开始创业时，他并不知道最终会有多少“痛苦的行政 B/S”。

他再正确不过了。簿记/会计，开发票，提案开发，与客户和(可能)承包商和员工签约，福利管理，支付账单，等等。

还有营销和销售，这应该是你的首要任务，但许多 IT 专业人士认为这是“行政废话”

以下是关于乔治问题的一些高级建议:

1.今天就出去买一本迈克尔·葛伯的电子神话书。他的重点是为什么大多数小企业不工作。对他来说，大多数企业主花太多时间在业务上，而没有足够的时间在业务上。要创建一个成功的企业，你需要开发一个“特许经营”——一套可重复的业务流程，让你成长。那么，如果你在企业而不是在 it 部门工作，你会如何处理所有的行政事务呢？

2.在我自己的咨询实践中，我也受不了行政方面的东西。所以我每个月预算 2-3 天来做所有的事情。通过安排行政日，这让我更容易。

3.我还雇用兼职人员来处理我真的不能忍受做的事情——尤其是记账。这样，我可以花更多的时间营销和发展我的业务，并发展客户关系。

4.随着时间的推移，事情会变得越来越简单，特别是如果你开发了程序(例如，一个通用的建议书/合同模板)并使用技术(例如，带有自动银行下载和电子账单支付的 QuickBooks)来节省时间。

5.请永远不要认为营销和销售是行政废话。业务发展是你业务的命脉。一个不会营销的伟大的网站开发者将会输给一个还算过得去的网站开发者，他是一个伟大的营销者——毫无疑问，任何一天。

第一篇博客到此为止。请保持联系。
