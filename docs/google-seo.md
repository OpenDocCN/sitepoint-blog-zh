# 谷歌如何确定一个页面的相关性

> 原文：<https://www.sitepoint.com/google-seo/>

你有没有想过为什么 Google.com 为美国用户提供的结果与德国用户不同？这是我最近不得不为一个客户回答的一个问题，如果给他发一个链接，比花半个小时试图解释许多熟练的 SEO 已经知道的东西要容易得多。

然而对于网络搜索者来说，原因并不明显。当然，他们可以推断出谷歌是根据搜索者的地理位置提供结果的，而地理位置是由用户的 IP 地址决定的。这通常也适用于谷歌本地域名，如谷歌或 google.com.au，但是，IP 地址并不是影响搜索结果的唯一因素。此外，如果这是决定谷歌搜索结果的唯一因素，许多站长将会失去。了解以下一些内容，你也会知道为什么搜索结果会因地点而异，以及如何优化你的网页，使之与你的目标受众相关。

*   顶级域名:众所周知，顶级域名在谷歌的本地搜索结果中权重很大。其他顶级域名如。信息，。电视，。由于有大量带有这些扩展名的“垃圾”网站，biz 被授予的权限较少。权威高的 TLD 似乎是。edu。gov，。最受欢迎的 TLD 仍然存在。虽然它可以被任何国家的网站使用，但它仍然不能保证排名不会在谷歌搜索结果中波动。见下文。
*   **服务器位置**:如我所说，。任何人都可以使用这个域名，但是谷歌也会考虑域名所在的地理位置。托管在澳大利亚的网站被视为澳大利亚网站，而托管在德国的网站被视为德国网站。所以，如果你的目标是美国观众，比如说，你想在美国的搜索结果中被优先考虑，在美国获得主机服务是一个好的选择。
    
*   **ITL 的位置**(传入文字链接)。如果你的目标是国际性的，试着从尽可能多的不同国家的网站获取链接。同样的道理，如果你的目标是本地用户，那么从你的目标国家的网站上获取尽可能多的链接。例如，如果你的大部分链接来自澳大利亚的网站，谷歌会认为这个网站是澳大利亚的，或者是澳大利亚搜索者感兴趣的。
*   页面语言:你翻译成其他语言的页面也有可能被排名，即使你的网站不在你目标语言的国家。在这种情况下，除了语言、编码字符、元标题和描述之外，排名因素也应该被翻译。
*   最后但同样重要的是，注意你如何设置你的 [**谷歌网站管理员工具**](https://www.google.com/webmasters/tools/dashboard) 。当你注册这项服务时，谷歌会要求你设置网站的地理位置。您也可以在以后进行设置或编辑，但此选项不适用于特定国家/地区的 TLD，如。德，。fr 等。但是对于。和其他通用的 TLD 网站，这个谷歌网站管理员工具设置取代了服务器定位信号，对于为你想要的每个子域设置不同的地理位置特别有用。

我希望这个关于页面相关性的小入门有助于理解 Google 如何向用户展示结果，以及如何针对这个变量优化您的站点。在 SEO 中有如此多的因素需要考虑，使用像这些初级读本中的相关但重要的花絮是一种简单而轻松地逐步提高搜索排名的方法。显然，每个人都不能一下子改变他们的域名、服务器位置、语言和链接，但是那些可以优化的东西应该被优化，如果在搜索引擎中的排名对你的网站很重要的话。
