# 苹果咬 Windows 7

> 原文:[https://www.sitepoint.com/apple-bites-windows-7/](https://www.sitepoint.com/apple-bites-windows-7/)

就在你认为操作系统方面一切都很平静的时候，苹果打破沉默，抨击 Windows 7 复杂、昂贵，并且是用户头疼的原因。OS X 产品营销副总裁 Brian Croll 解释道:

> Windows 用户真的厌倦了这些年来他们遇到的所有令人头疼的问题，从 Windows Me 到 NT 到 Vista，再到现在的 Windows 7。因此，我认为人们正在寻找不同的东西，Mac 提供了真正的易用性、稳定性和安全性。
> 
> Windows 7 仍然只是 Windows。变化不大。它仍然很复杂，当你看到升级的成本时，它仍然非常昂贵，而且仍然存在安全问题。事实证明，当你得到 Windows 7 时，它甚至没有一些基本的应用程序，如邮件或聊天，或管理照片的程序。你实际上必须去找到并下载它们。
> 
> 对于消费者来说，Windows 7 内置了许多令人头痛和烦恼的东西。它建立在许多过时的技术之上，这些技术不会因版本不同而改变。我们认为大多数人可能不会从 XP/Vista 中跳出来，因此，他们会寻找新的电脑。

让我们仔细研究一下他的陈述:

**1。Windows 用户厌倦了令人头疼的事情**

真的吗？我们都遇到过 Windows 的问题，但是有哪个操作系统能提供无故障的体验呢？我不这么认为，克罗尔先生——而且苹果电脑也不例外。

**2。Windows 7 仍然只是 Windows**

这是一种解脱——如果不是这样，用户会非常困惑！操作系统已经进化，它当然更可靠，更少混乱。也许它的进化会更快，但是人们喜欢他们所知道的。

**3。没有基本应用**

微软已经从 Windows 7 中放弃了 Windows Mail、MSN Messenger、照片库、影音制作和其他几个应用程序。ie 8 也可以卸载( [*某种程度上*](https://www.sitepoint.com/ie-windows-7-uninstall/) )。

但这不正是这个行业想要的吗？每个人都抱怨 Vista 的臃肿，微软也因提供免费软件摧毁竞争对手而被罚款数次。看起来公司终于听进去了，但他们还是赢不了！

您还应该注意到，这些应用程序仍然可用。它们可以从[http://download.live.com/](http://download.live.com/)免费下载，它们也作为可选的“Live Essentials”条目出现在 Windows 7 更新面板中。但是，如果 Windows 用户愿意，他们可以选择其他应用程序。

**4。它建立在许多过时的技术之上，从一个版本到另一个版本都没有变化**

微软努力在不同版本之间保持程序兼容性。他们测试并支持成千上万的应用程序，所以保留旧代码并不奇怪。Windows 兼容性可能并不完美，但有多少其他公司努力支持 15 年前的软件？

也许克罗尔先生不知道 OS X 从 2002 年就已经上市了，而且是基于一个有 40 年历史的操作系统。我确信仍然可以找到隐藏在表面之下的一些原始 Unix 源代码！

有点奇怪的是，苹果的营销部门经常对微软进行抨击。他们最忠实的粉丝永远不会碰非苹果产品。甚至微软用户也承认苹果制造了一些辉煌的硬件和软件。那么，为什么苹果总是在进攻呢？微软很少报复。

苹果是否变得过于自大了？Windows 7 对他们来说是个大问题吗？
