*   SitePoint 的作者和全能好人 Earle Castledine(又名演讲者先生)已经发布了一个粗略但仍然非常有用的移动网络调试器，名为 [WadBug](https://github.com/mrspeaker/WadBug) 。它为您提供了一个不错的 JavaScript 控制台，您可以将它放入任何页面，开始检查对象并记录语句；就在您舒适的移动浏览器中。
*   我们在 Campaign Monitor 的好朋友[发布了他们的 API(版本 3.0)](http://www.campaignmonitor.com/blog/post/3343/huge-api-update/) 的重大更新。如果您一直在考虑将邮件传递或邮件列表管理集成到您的应用程序中，现在是时候开始了。
*   这一条实际上已经有几个月的历史了，但我只是在我的带星号的推文中偶然发现了它，并认为它值得分享: [Molly Holzschlag 用正好 140 个字符对 HTML5 的定义](http://twitter.com/#!/mollydotcom/statuses/24028193698)(我相信他们称之为“twoosh”)。
