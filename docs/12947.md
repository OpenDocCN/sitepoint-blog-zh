# Adobe Fireworks CS4 测试版发布

> 原文:[https://www . site point . com/adobe-fireworks-cs4-beta-released/](https://www.sitepoint.com/adobe-fireworks-cs4-beta-released/)

Adobe 已经提供了新的 [Fireworks CS4 测试版下载](http://labs.adobe.com/technologies/fireworkscs4/)，好消息是任何拥有 Fireworks CS3 的人——无论是独立的还是作为 web 套件的一部分——都可以尝试一下。

1.  您需要在 Adobe 拥有一个帐户并进行[登录。](https://www.adobe.com/cfusion/entitlement/index.cfm?e=labs%5Ffireworkscs4)
2.  然后，您将被定向到下载页面。Win 版本的下载量为 336 MB，Mac 版本为 633 MB。
3.  一旦安装并启动，你会被要求一个序列号。警告:这是**而不是**你目前的 CS3 系列(这是我最初尝试的)。[你必须进入这个页面](https://www.adobe.com/cfusion/entitlement/index.cfm?e=labs_fireworks_sn)，输入你的 CS3 序列号，然后你会被发给一个新的**Fireworks CS4 序列号才能进入应用程序。**

 **也许只是我，但是当你被告知你将需要你的旧序列号，并且在一个“*在这里输入你的序列号”*屏幕弹出后不久，在那里输入你的旧序列号似乎是完全合乎逻辑的——当然，这失败了，让我感觉很傻。

我回溯并弄明白了，但是我打赌其他人也在犯类似的错误——并且淹没了 Adobe 的支持。也许发布新的连载*然后*提供下载会是一个更好的过程。一旦下载开始，我想大多数人会离开 Adobe 网站——没有序列号。

启动时你首先注意到的是新的界面。Adobe 已经放弃了浅炭灰色的单色图标。它让所有东西看起来都很有凝聚力，但我发现自己花了更长的时间来定位一些工具——例如,“裁剪”和“导出区域”工具共用一个按钮，看起来非常相似，但没有 CS3 图标的红色和蓝色。

Adobe 也跟随了忽视 OS chrome 的增长趋势，支持他们自己的定制应用程序界面。除了我看不出这样做对用户有任何好处的事实之外，在这个测试版中，这是一件非常偶然的事情，操作系统风格的下拉菜单和面板与 Adobe 标签和对话框混合在一起。看起来更像是“对 Adobe 有利”而不是“对用户有利”的决定，但我很乐意听听背后的理由。

不管怎样，我只是在这方面探索了一小会儿，但是有两个非常好的新特性是显而易见的。

**改进型发动机**

![Firework's more advanced type engine](../Images/eef51d95154373ca23cc5ead491fb645.png)

首先，Adobe Type 引擎已集成到 Fireworks CS4 中，这使 Fireworks 用户可以访问适当设计的粗体、斜体和其他可用的字体变体。到目前为止，Fireworks 只提供了“以编程方式生成的”粗体和斜体。

正如上面的例子所示，这比什么都没有好，但每次都给出一个次等的结果。

**改进的路径编辑**

我真正喜欢的另一个新功能是新的路径编辑面板。我多年来一直使用 Fireworks 作为插图工具，因为它在矢量和像素方面有着很好的平衡，但这个新面板确实为矢量编辑增添了一些力量。除了标准的打孔/裁剪/合并以及缩放和旋转功能，新面板还向工具箱添加了路径挤压、路径混合(我的愿望列表功能—感谢 Alan)、鱼眼、变形为路径、反转形状和点选择工具等内容。

虽然这些工具中的一些已经通过扩展提供，但是新的面板将它们整合到一个更加一致的工作环境中。

当我知道更多的时候，我会写一份更全面的概述，但事情在这一点上看起来不错。**
