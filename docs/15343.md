# 访客登记的目的

> 原文：<https://www.sitepoint.com/purpose-visitor-registration/>

我所知道的每一个内容网站经理都在试图“强迫”注册。

这一过程有其吸引力。您可以向注册用户发送电子邮件。你可以在给潜在广告客户的演示中计算注册人数。你甚至可以命名和识别注册用户。你可以调查他们。有各种各样的方法从他们身上赚钱。

但所有这些都忽略了一个关键线索，一个每个商业出版商都心知肚明的线索:控制发行量的目的是向广告商证明你达到了他们的目标。

##### 每个广告商都想知道的是

就像所有与商业有关的事情一样，你看，这不是关于你的。而是他们，为你买单的人:你的顾客。或者在这种情况下，你的广告商。

人们购买广告有两个原因。一个是针对直复营销，网站一直都是用直复营销的指标。“点击率”是一种直接营销指标，它衡量的是一个动作，一个广告的“点击”，销售漏斗中至关重要的一步。大多数衡量点击率的活动也衡量“转化率”，即优惠在潜在客户中产生销售的成功。

但是如果有人为了品牌效应而购买广告，他们想要更多。直接营销指标是不够的:他们想知道他们的广告*达到了目标*。如果目标是网站管理者，这个活动必须多次打击他们所有人。建立一个包含各种媒体的活动的目的就是要做到这一点。

那么，广告商问，你的观众中有多少网站经理？他们代表多大规模的网站？他们花了多少钱？就人员和资金而言，这代表了多大的市场份额？你能证明吗(见[本报告](http://www.americanbusinessmedia.com/sourcebook/p77s5.htm)关于审计的信息)？

这些都是有效的问题。控制发行量的印刷出版物可以回答这些问题，但大多数网站不能。所以现在购买印刷品的广告商不会购买网络。除非网站能够像平面媒体一样准确地回答这些问题，否则它们就不是建立品牌的好媒体。

##### 你需要给他们什么

注册似乎可以让你回答这些问题。强迫人们注册才能访问你的网站似乎可以保证答案。

但是如果你这样想，你就想错了。再说一次，这不是关于你，而是关于他们。虽然“他们”的定义变了:现在我们说的是读者。

如果你要证明自己是一个品牌建设媒介，你必须说服你的目标市场的读者，让他们向你注册。杂志通过“质量卡”来做到这一点，质量卡是发给潜在读者的详细问卷。那些符合标准并达到目标的人可以得到一份免费的印刷杂志，而那些不符合标准的人则没有。

这些年来，我亲自填写了几十份质量卡，而且(因为我是一名记者，而不是广告或计算机网络的“玩家”)我也被拒绝过很多次。*PC Week*80 年代不要我，因为我没有买够 PC 设备和相关用品。*信息世界*不想要我，因为我不是网络设备的大买家。*广告时代*不要我，因为我不在那个行业。

简而言之，如果你没有击中目标，你的注册是没有价值的。另一方面，如果你真的达到了目标，你的注册就是金的。因此，设计一个能达到你行业目标的在线“质量卡”,只批准那些符合你标准的注册。

这个原则在今天的网络上最好的例子是 *[变种](http://www.variety.com/)* 。

##### *品种*价值目标

*综艺*瞄准娱乐圈。它的名单是电视、电影和其他的名人录。 *Variety* 想要这些人作为读者——它不想要我们其余的人。

因此，今年《综艺》杂志想出了一个计划。印刷版订户将继续免费访问 Variety.com。非订阅者可以申请订阅，但是如果他们被拒绝了，他们仍然可以以每年 59 美元的价格购买使用权。

怎么..？下周，我们将讨论网站如何让他们的读者对这样一个大胆的要求说“是”。
