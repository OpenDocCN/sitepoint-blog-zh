# 你的网站加载速度快吗？谷歌排名现在受网站速度影响

> 原文：<https://www.sitepoint.com/google-rankings-now-affected-by-site-speed/>

在谷歌官方博客上，最新的帖子讨论了谷歌如何将网站速度作为他们用来确定搜索排名的 200 多个信号之一。网站速度反映了网站对 web 请求的响应速度。去年 12 月，谷歌在[的一篇博客文章](http://googlewebmastercentral.blogspot.com/2009/12/your-sites-performance-in-webmaster.html)中首次提到，一场变革即将到来。

在博客上，他们说“虽然网站速度是一个新的信号，但它没有一个页面的[相关性那么重要”，目前，“不到 1%的搜索查询受到网站速度信号的影响”。1%的谷歌搜索仍然是一个巨大的网站数量。](http://www.youtube.com/watch?v=muSIzHurn4U)

除了为谷歌优化你的网站，你应该这样做的主要原因是改善你的访问者的体验。随着宽带速度越来越快，人们很容易忘记那些仍然在网速较慢的网络上浏览的人。没有人喜欢等待网站加载，所以在设计网站时，考虑一下网站加载的速度是个好主意。

[![300px-Speedometer](img/265830cc24d1c7506b1275de9c4f7134.png "300px-Speedometer")](https://www.sitepoint.com/wp-content/uploads/2010/04/300pxSpeedometer.jpg) 有很多免费的工具，设计师可以用来检查你的网站速度。

*   [页面速度](http://code.google.com/speed/page-speed/)是一个开源的 Firefox/Firebug 插件，在他们的页面上运行页面速度的站长会得到一组每个页面的分数，以及如何提高其性能的建议。
*   YSlow 分析网页，并提出改善网页性能的方法。这是一个 [Firefox 插件](https://addons.mozilla.org/en-US/firefox/addon/5369)，集成了 [Firebug](http://www.getfirebug.com/) web 开发工具。
*   [WebPagetest](http://www.webpagetest.org/) 提供了与优化清单的比较，以及页面加载性能的瀑布图
*   [网站管理员工具](http://www.google.com/webmasters/tools)网站性能显示全球用户体验到的网站速度。

**如何看待谷歌的这一改变？你试过这些或其他工具来检查和提高你的网站速度吗？**
