# 火狐 3.5 是世界上最流行的浏览器

> 原文:[https://www.sitepoint.com/firefox-35-most-popular-browser/](https://www.sitepoint.com/firefox-35-most-popular-browser/)

这是正式的。根据 StatCounter Global Statistics 最近发布的数据，Firefox 3.5 已经成为世界上最受欢迎的浏览器。它的全球市场份额略低于 22%，IE7 的份额略有下降，但 IE8 正在快速赶上。

和所有的统计数据一样，有一些注意事项:

*   图中测量了各个浏览器版本
*   Internet Explorer 仍然占有全球浏览器市场 55%以上的份额，但它被 IE6、IE7 和 IE8 瓜分了。
*   网络统计天生就有缺陷。不要相信他们。

还有一些地区差异:

*   在[美国](https://gs.statcounter.com/#browser_version-na-weekly-200827-200951)，IE8 依然以 25%的比例统治。火狐 3.5 和 IE7 拥有几乎相同的 22%份额。
*   在欧洲，火狐 3.5 在 2009 年 9 月超过了 IE7，现在占 28%多一点。IE8 几乎有 22%，但是 IE7 已经滑落到 17%以下。
*   在亚洲，火狐 3.5 以不到 17%的份额排名第三。IE7 占 23%，IE6 占 26%。
*   在[非洲](https://gs.statcounter.com/#browser_version-af-weekly-200827-200951)，火狐 3.5 是第二受欢迎的浏览器，占 18%。IE6 以不到 29%的份额占据主导地位。
*   在大洋洲，IE8 接近 27%,紧随其后的是 Firefox 3.5，仅超过 22%。
*   在南美洲，IE7 的市场份额迅速下降，目前不到 25%，而 Firefox 3.5 的市场份额上升到 22%以上。
*   在南极洲，是的，说真的，火狐 3.0 统治了超过 61%的用户(大概有 12 个用户)。

很高兴看到 IE8 的崛起是以 IE6 和 IE7 为代价的。然而，尽管 IE6 的使用在整个西方都是个位数，但它仍然是非洲和亚洲最受欢迎的浏览器——这两个市场正在快速增长。

但是我们应该祝贺 Mozilla。这花了五年多的时间，但是他们现在已经正式成为浏览器排行榜的第一名*(取决于你如何阅读统计数据)*。不幸的是，一旦 3.6 / 4.0 版本发布，Firefox 的地位几乎肯定会下降，但让他们尽情享受他们的荣耀时刻吧！
