# Web 3.0 网状风格——请拿着薯条

> 原文：<https://www.sitepoint.com/web-30-meshup-style-hold-the-fries-please/>

这是对我之前关于 Web 3.0 [meshups](https://www.sitepoint.com/web-30-meshup-style-would-you-like-fries-with-that/) 的帖子的快速跟进。我只是想澄清一些事情，因为我似乎没有非常有效地表达我的观点。

我不想重新引发一场关于语义网是否有用的争论。我想说的是，这是一套新的技术，可以用来提供新的用户体验和新的用户界面。这是一件好事。如果你完全理解，这一切都是题外话。我使用许多数据库，但对 b 树或复杂的索引算法一无所知。我可以举出更多的例子，但这是多余的。

语义网**被**一些非常**大的**公司使用。谷歌[在其 OpenSocial API 中使用了 FOAF](http://socialgraph-resources.googlecode.com/svn/trunk/samples/findcontacts.html) 。雅虎刚刚发布了索引微格式、RDFa 和 RDF 的微搜索，微格式都是关于语义标记的。语义网不一定是指 AI。路透社(世界上最大的通讯社)最近推出了 [Open Calais](http://opencalais.mashery.com/) ，为英文文本添加语义标签。

所以三大巨头正在使用语义网技术做有趣的事情。这些只是我想到的三个例子。

我没有在荒野中徘徊，看不到任何东西。*至少我不认为我是… :)*
