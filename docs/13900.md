# Mozilla XForms 项目受到削减的威胁

> 原文：<https://www.sitepoint.com/mozilla-xforms-project-threatened-by-cutbacks/>

几周前，我试乘了 Mozilla XForms 项目的第二个预览版，演示了如何将它与 Internet Explorer 的 FormsPlayer 插件结合使用，以创建 T2 跨浏览器 XForms。

不幸的是，尽管这个预览版充满了希望，但是 Mozilla 对 XForms 的支持突然变得非常糟糕。有消息称，Novell 在一轮裁员中解雇了该项目的整个开发团队，包括首席开发人员艾伦·博弗(Allan Beaufour)和他的博客(T3)。

尽管该项目是开源的，并且得到了开发社区的一些志愿者成员的支持，但是主要的开发工作是由 Novell 发起和资助的。如果没有这种支持，这个项目——非常接近 1.0 版本——在可预见的未来可能不会有继续发展的动力。

作为一个 XForms 的粉丝，我对这个消息深感失望。我甚至在为 12 月份的墨尔本[开源开发者大会](http://www.osdc.com.au/)准备一个关于 Mozilla XForms 的演讲。看起来我现在必须想出一些不同的话题来谈论了。

> 我现在还在附近徘徊，既有 IBM 的资源，也有志愿者在研究它，所以它根本没有死。不管怎样都会结束的，别担心。
