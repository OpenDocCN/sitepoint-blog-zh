# 你拥有自己的社交媒体身份吗？

> 原文：<https://www.sitepoint.com/do-you-own-your-social-media-identity/>

![](img/182b05c6c5d96588065c8009b0e9934f.png "twitter-ghost-bird")据*华尔街日报*报道[推特现在是主流](http://online.wsj.com/article/SB122461906719455335.html)。不管你是否相信这一点，这只能意味着一件事:域名抢注者即将集体涌入。那么问题来了，谁拥有你的 Twitter 名字？

去年三月的某个时候，我终于赶上了潮流，买了我的( [@catone](http://www.twitter.com/catone) )。SitePoint 就没那么幸运了。当我们去注册我们的官方 Twitter 账户时，不幸的是，我们已经来不及做出显而易见的选择了:别人已经有了。(可以在 Twitter 上找到我们，地址是 [@sitepointdotcom](http://www.twitter.com/sitepointdotcom) ，我们希望您关注我们！)

据在*网络世界*的 Richard Steinnon 称，“Twitter 蹲着”的做法越来越普遍。例如，看看[克里斯科](http://twitter.com/crisco)、[可口可乐](http://twitter.com/coke)、[百事可乐](http://twitter.com/pepsi)、[耐克](http://twitter.com/nike)和[雪佛兰](http://twitter.com/chevrolet)的名字。所有注册商标，都被商标所有人使用，而且很可能没有被商标所有人使用。

也许最厚颜无耻的推特抢注的例子是 [@glennbeck](http://twitter.com/glennbeck) 账户，其中有一条推特似乎在为该账户捞钱。显然，这个账户不是由电视和电台名人格伦·贝克控制的，而是由一个试图快速赚钱的投机者控制的。

> 你好。如果你喜欢这个 Twitter 账户，请联系我。glennbecktwitter@gmail.com

公平地说，拥有我们最初追踪的 SitePoint 名称的那个人，这不是一个纯粹的抢注案例——他实际上在瑞士拥有 SitePoint 名称的商标。出于这个原因，Twitter 帮不了我们——尽管 SitePoint Pty。有限公司拥有 SitePoint 名称的美国商标。

不过，这里的要点很清楚:

你应该立即在你能想到的所有社交媒体服务上注册你的公司名称。

Twitter、Digg、MySpace、脸书、YouTube——这些都是域名抢注的新领域。“如何保护自己的品牌？立即[…]确定您的姓名是否可用。斯坦农写道:“尽你所能去获得它。“当你这么做的时候，保留所有与你的品牌相关的名字。你可以决定你投资的任何域名都应该有自己的[社交媒体] ID。在这场新的抢地大战中，最先跳出来的终究是域名抢注者。”
