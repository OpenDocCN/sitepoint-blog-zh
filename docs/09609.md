# 浏览器趋势，2011 年 7 月:五分之一的人现在使用 Chrome

> 原文:[https://www . site point . com/1-in-5-use-chrome-browser-trends-July-2011/](https://www.sitepoint.com/1-in-5-use-chrome-browser-trends-july-2011/)

跟踪浏览器市场变得越来越困难。 [Chrome 12](https://www.sitepoint.com/whats-new-in-google-chrome-12/) 、[火狐 5](https://www.sitepoint.com/firefox-5-released/) 和 [Opera 11.5](https://www.sitepoint.com/opera-11-5-whats-new/) 于上月发布。有些浏览器会自动更新，有些不会。一些供应商推出了奢华的促销活动，其他供应商则只字不提。

7 月份的大新闻是 Chrome 的使用率首次超过 20%。让我们更详细地检查完整的 [StatCounter 统计数据](https://gs.statcounter.com/)…

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。‘相对’栏表示比例变化，即上个月又有 3.1%的 IE6 用户放弃了浏览器(耶！)有几个注意事项，所以我建议你阅读[浏览器市场份额是如何计算的](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

6 月份， [Chrome 11 取代 Firefox 3.6 成为世界第二大用户浏览器](https://www.sitepoint.com/browser-market-june-2011-chrome-11-worlds-second-favorite-browser/)。令人困惑的是，Chrome 12 的推出分裂了谷歌的用户群，因此 Firefox 4.0 现在已经位居第二。尽管推出才一周多一点，但随着火狐 3.x 和 4.0 用户的迁移，火狐 5.0 已经获得了 2.8%的市场份额。

然而，对 Mozilla 来说没有什么好消息。6 月份，Firefox 的总销量下降了近 1%:是 IE 的三倍，是浏览器经历过的最大跌幅之一。似乎没有什么特别的原因；Firefox 4 和 5 广受欢迎，但它们并没有阻止 Chrome 的发展。也许这些变化对某些人来说太激进了？还是用户调查了其他选项而没有升级？

尽管 IE8 仍然是最受欢迎的浏览器版本，IE9 还是取得了不错的成绩。IE6 和 7 继续下降，尽管速度正在放缓。

歌剧也经历了小幅下跌。然而，[版本 11.5](https://www.sitepoint.com/opera-11-5-whats-new/) 也许能够扭转这一趋势，并且在移动领域对该公司来说有更好的消息…

## 移动浏览器的使用

根据 StatCounter 的数据，[桌面浏览器占据了 web 活动](https://gs.statcounter.com/#mobile_vs_desktop-ww-monthly-201106-201106-bar)的 93.47%。上个月，手机浏览器的使用率增长了近 1%，达到 6.53%。这可能是一种季节性异常，因为西方世界的大部分地区都是夏天——网络用户可能在外面享受阳光*(或者对于我们这些在英国的人来说是毛毛雨)*。

移动浏览器市场的变化很不寻常，可能会受到季节性因素的影响。诺基亚可能正在经历商业问题，但他们会很高兴地发现，他们的(相当基础的)浏览器已经在 iPhone 上超越了 Android 和 Safari。Opera 在最新发布手机版后也有所收获:

1.  opera Mini/Mobile——22.81%(增长 1.00%)
2.  诺基亚浏览器——17.66%(增长 1.16%)
3.  安卓系统——17.25%(增长 0.24%)
4.  iPhone——15.22%(下降 1.49%)
5.  黑莓——11.98%(下降 0.78%)

如果你还没有这样做，也许是时候考虑一下移动平台的快速崛起会对你的业务产生怎样的影响了。
