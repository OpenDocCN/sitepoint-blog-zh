在过去，为 web 创建一个动画这样简单的东西需要多个应用程序，可能还需要几个插件。你需要一个程序来创建动画的单个帧，还需要几个插件来添加文本效果，比如投影。一旦创建了帧，你就可以将它们导出到动画程序中来创建动画 GIF 文件。最后，你将加载动画到一个优化工具来缩小它。如果您在过程中的某个地方犯了错误，或者如果您想要对其中一个帧进行更改，您将不得不再次经历整个过程。

现在，快进到现在。您可以在一个应用程序中创建、优化图形并制作动画。如果你需要编辑一帧动画，你不需要回溯。您不再需要在多种工具之间切换来创建网页图形，因为您只需安装一个程序，就可以完成创建令人惊叹的网页图形所需的一切。虚构？不现实。Macromedia Fireworks 2。

Macromedia Fireworks 的最初版本毫无疑问是最终的 web 图形设计解决方案。没有任何其他图形程序(包括 PhotoShop)能够做到 Fireworks 所能做到的一切，除非添加五到十个插件和几个第三方程序。Fireworks 第 2 版延续了这种一体化 web 图形设计包的悠久传统，增加了大量新功能和可用性改进。

**新功能包括:**

*   自动字距调整以改善小文本的外观。
*   支持 PhotoShop 5、Illustrator 8 等文件格式
*   Scriptlets 允许您创建拖放 JavaScripts，以将批量查找和替换与批量导出功能结合起来。
*   改进的批处理允许您批处理整个站点的图像。
*   热点翻转提供了触发翻转的不规则形状的区域。
*   与 Dreamweaver 2 更好地集成

新版 Fireworks 2 也比原版便宜 100 美元，售价为 199 美元。你只需花 99 美元就可以升级到新的电子版本。

##### 设计环境

那些使用 Paint Shop Pro 或 PhotoShop 的人在 Fireworks 2 界面(左)中将会有宾至如归的感觉。有一个标准工具栏和几个浮动面板，面板上有调色板和各种效果的设置。

在设计环境中，我最喜欢的功能是 Live Effects。我可以创建像斜面、发光和阴影这样的效果，然后回去编辑它们。当我在面板上进行更改时，我可以实时观察结果。Live effects 基本上允许你在任何时候对任何特效进行修改，而不必重新开始。

Fireworks 2 支持矢量图形和位图图形。这实际上模糊了两种类型之间的界限。Fireworks 对象的路径是可编辑的矢量路径。然而，它也可能有一个纹理填充，或一个效果，如斜面应用于它。如果您在 Fireworks 中放大任何对象，您将看到单个像素，但像素会根据矢量编辑进行重绘。编辑对象后，甚至可以重绘效果，这在以前只能在位图图像应用程序中实现。

该程序提供了对 PhotoShop 插件和格式的全面支持。但是，Fireworks 2 并不是一个用于制作非 web 图形的工具。

##### 创建 GIF 动画

Fireworks 2 提供了创建动画 GIF 文件所需的一切。Fireworks 2 的几个面板是专门为创建动画而设计的。VCR 面板允许您逐帧查看或预览动画。“实例”是一个很棒的特性，它允许你在多个帧上重现一些东西。如果在一个帧上编辑它，它的所有实例都会自动更新。

洋葱皮打开时，会在当前帧之前或之后的帧上显示对象，但会变暗，以便很容易将它们与对象的当前帧区分开来。

正是这些特性使得 Fireworks 2 成为设计网页图形的工具。Fireworks 将最好的独立图形、动画和优化程序都结合到一个框架中。

##### 限幅

在 Fireworks 中分割图像是一个简单的过程。使用切片工具，我能够在图像上绘制矩形切片对象。Fireworks 通过智能组合图像中不包含切片对象的部分，自动创建尽可能少的切片。结果是更少的 HTML 代码，以及稍微更快的加载页面。导出最终输出是一项简单的任务，需要我指定一个目标文件夹。Fireworks 2 会根据我在属性对话框中定义的内容自动命名切片图像。

切片的缺点是，如果你真的导出 HTML，它会以 CSS 的形式出现，而不是普通的 HTML。之前的版本其实做的是基本的 HTML 模板，而不是 CSS。

图像优化发生在“导出”对话框中。或者，您可以使用导出向导来帮助您优化图形。作为一个测试，我拿了你在这一页的左上角看到的 WR 标志，我在向导中运行了它。在回答了几个简短的问题后，包括我的目标尺寸，一个对话框弹出了向导的建议。读完之后，点击“确定”，弹出导出预览窗口。

图形被优化为我的目标尺寸。我稍微修改了一下设置，最终把文件从原来的 2.18KB 压缩到了 2.03KB，图像质量没有任何改变。变化非常小，但图像之前已经使用 Ulead SmartSaver 进行了优化。我没想到会有什么激烈的事情。

##### JavaScript 翻转

JavaScript 滚动允许当鼠标移动到图像上或被点击时改变图像的外观。这是通过一段 JavaScript 代码实现的，当访问者的鼠标悬停在图像上时，这段代码交换图像。在 Fireworks 2 中创建 JavaScript 翻转极其简单。一旦绘制了代表图像不同状态的不同图像，就可以选择热点，并导出最终结果。Fireworks 2 会自动生成翻转正常运行所需的 JavaScript。Fireworks 2 甚至允许创建不规则形状的翻转，这在过去是非常困难的。

但是，如果您没有彻底阅读 Fireworks 的所有文档，学习如何使用 JavaScript 翻转功能可能会有点困难。

##### 图像地图

图像地图允许创建带有热点的图像，当点击时，会将访问者带到特定的 URL。因此，单个图像可以链接到多个页面。我在 Fireworks 中创建一个简单的图像地图没有任何问题。使用热点工具之一，我选择了一个图像区域，并将其定义为热点。在 Image Map Options 面板中，我为我创建的每个热点指定了一个目标 URL。接下来，我从 File 菜单中选择 Export，选择一个目标目录，图像和包含图像映射代码的 HTML 文件就放在那里。

##### 成批处理

批处理旨在自动处理一组图形。例如，Fireworks 2 可以转换所选图形的格式或更改一组文件的设置(例如，将所有 GIF 文件更改为交错的 GIF)。如果是 Fireworks PNG、Illustrator、Freehand 或 CorelDRAW 格式，您甚至可以在一批图形中查找和替换文本、颜色、URL 和字体。

Fireworks 允许创建“脚本小程序”，这将在将来轻松地重新创建批处理过程。这些跨平台的 scriptlets 可以在计算机和公司中的其他个人之间共享。这使得标准化所有图形变得容易。

##### Dreamweaver 2 集成

Fireworks 2 作为独立产品很棒，但如果您还安装了 Dreamweaver 2，它会变得更好。我可以将 Fireworks 2 中创建的 JavaScript 翻转直接导出到 Dreamweaver 中，在那里我可以进行进一步的编辑。我还能够将 Fireworks 中制作的图形导出为对象，然后在 Dreamweaver 2 中使用它们。如果您安装了这两个程序，从 Dreamweaver 中启动 Fireworks 2 来快速更改图形是非常容易的。

##### 设计环境

那些使用 Paint Shop Pro 或 PhotoShop 的人在 Fireworks 2 界面(左)中将会有宾至如归的感觉。有一个标准工具栏和几个浮动面板，面板上有调色板和各种效果的设置。

Fireworks 2 支持矢量图形和位图图形。这实际上模糊了两种类型之间的界限。Fireworks 对象的路径是可编辑的矢量路径。然而，它也可能有一个纹理填充，或一个效果，如斜面应用于它。如果您在 Fireworks 中放大任何对象，您将看到单个像素，但像素会根据矢量编辑进行重绘。编辑对象后，甚至可以重绘效果，这在以前只能在位图图像应用程序中实现。

该程序提供了对 PhotoShop 插件和格式的全面支持。但是，Fireworks 2 并不是一个用于制作非 web 图形的工具。

##### 创建 GIF 动画

Fireworks 2 提供了创建动画 GIF 文件所需的一切。Fireworks 2 的几个面板是专门为创建动画而设计的。VCR 面板允许您逐帧查看或预览动画。“实例”是一个很棒的特性，它允许你在多个帧上重现一些东西。如果在一个帧上编辑它，它的所有实例都会自动更新。

洋葱皮打开时，会在当前帧之前或之后的帧上显示对象，但会变暗，以便很容易将它们与对象的当前帧区分开来。

正是这些特性使得 Fireworks 2 成为设计网页图形的工具。Fireworks 将最好的独立图形、动画和优化程序都结合到一个框架中。

##### 限幅

在 Fireworks 中分割图像是一个简单的过程。使用切片工具，我能够在图像上绘制矩形切片对象。Fireworks 通过智能组合图像中不包含切片对象的部分，自动创建尽可能少的切片。结果是更少的 HTML 代码，以及稍微更快的加载页面。导出最终输出是一项简单的任务，需要我指定一个目标文件夹。Fireworks 2 会根据我在属性对话框中定义的内容自动命名切片图像。

##### JavaScript 翻转

JavaScript 滚动允许当鼠标移动到图像上或被点击时改变图像的外观。这是通过一段 JavaScript 代码实现的，当访问者的鼠标悬停在图像上时，这段代码交换图像。在 Fireworks 2 中创建 JavaScript 翻转极其简单。一旦绘制了代表图像不同状态的不同图像，就可以选择热点，并导出最终结果。Fireworks 2 会自动生成翻转正常运行所需的 JavaScript。Fireworks 2 甚至允许创建不规则形状的翻转，这在过去是非常困难的。
