# 电脑美国上的火狐秘密访谈

> 原文:[https://www . site point . com/Firefox-secrets-interview-on-computer-America/](https://www.sitepoint.com/firefox-secrets-interview-on-computer-america/)

上周，就在我登上飞往加拿大的飞机之前，我接受了在全美播放的“计算机美国”电台节目的采访。在周四节目的第二个小时，我谈到了[火狐](http://www.mozilla.org/products/firefox/)浏览器，提供了从 SitePoint 的新书[火狐秘密](https://www.sitepoint.com/books/firefox1/)中摘录的一些技巧。

我提到的技巧包括:

*   HTTP 管道，Firefox 默认禁用的特性，利用 HTTP/1.1 功能来加速浏览。
*   用 Shift-Delete 从地址栏弹出历史中删除不想要的项目。
*   一次将一组标签加入书签，然后使用“书签”菜单上的“在标签中打开”项再次打开它们。
*   提供要在选项卡中打开的站点列表，作为您的浏览器主页。
*   设置一个自定义的搜索关键词，在[互联网档案](http://web.archive.org/)上快速查找一个网站的历史。

在接下来的一周半时间里，该节目的存档音频可以从[在线档案馆](http://www.businesstalkradio.net/weekday_host/Archives/cc.shtml)下载。这里是[直达链接](http://archives.warpradio.com/btr/ComputerAmerica/071423.MP3) (MP3，18MB)。

我在采访中获得了很多乐趣，我只希望它能更长一些——我真的只是触及了书中内容的皮毛。

如果面试让你想知道更多，样本 PDF 中有更多有价值的提示。
