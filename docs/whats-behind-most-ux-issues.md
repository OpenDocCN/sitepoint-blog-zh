**先说这个前提:**

> 绝大多数的 UX 缺陷并不是来自于缺乏技能或天赋或技术或过程。

他们不存在，因为工作人员中没有 UXers 或设计师。它们之所以不存在，是因为没有人“理解”好的 UX 或 UI 设计。

根据我的经验，他们不存在，因为*人们不知道他们在做什么*。

那肯定会发生。但特别是在我近三十年职业生涯的最后五年，这种情况极其罕见。

相反，我看到的是软件产品——网站、应用、系统——中存在的 UX 和设计问题直接来自于个人意图的不一致。这里有三种味道:个人的，组织的和政治的。

为了更加神秘，我会这样说:虽然我肯定你听说过你应该专注于并发现那个意图，但我说它远比那个更深。

事实上，在项目规划中需要发生的最关键的事情是几乎从来没有讨论过的事情。你不会在文章或博客中读到它。你不会在视频或在线课程或主题演讲中看到它。

事实上，几乎没有人谈论它。

当然，这就是为什么你和我，亲爱的读者，现在在这里。分享一个故事。

几年前，我和一个客户一起工作，我和他们的经历给了我一个永远不会忘记的教训。

像大多数事情一样，这个教训来之不易。

该客户是一个拥有 28 亿美元资产的组织，处于行业的顶端。基本上，它们为投资者从你能想到的任何一家投资机构收到的几乎每一份个性化的印刷抵押品提供内部履行服务。他们拥有 85%的市场份额。

但是，规模如此之大也意味着他们难以置信地慢——适应变化、适应技术的步伐、适应终端消费者不断增长的需求，让他们自己研究和管理自己的投资。

因此，该公司的客户 Merrill Lynches 和 TD Ameritrades 以及 Edward Jones of the world——一年多来一直强烈要求我的客户提供一个在线自助式门户解决方案……他们除了承诺“我们正在努力”之外，什么也提供不了

当我找到他们的时候，他们的几个大客户已经等得不耐烦了；他们取消了价值百万美元的合同，开始为客户开发自己的解决方案。如此巨额的资金正在稳步流失。

在挖掘为什么他们不能至少为客户推出一个可靠的 MVP 时，我了解到绊脚石是由几个遗留系统组成的不连续的工作流和不真实的人工干预。再加上跨流程和实践的一致性令人震惊的缺乏，以及为每个客户提供不同风格的定制功能的意愿，您就有了一个混乱的配方。

每一项工作本质上都是一个*定制项目*，几乎没有可重复的效率。

快进:几个月后，我们提出了用 Adobe Livecycle 替换这些拼凑的系统和流程的建议，当时，Adobe Livecycle 具有处理/自动化能力以及印刷和数字世界之间所需的紧密集成。所以我和另外四个人点点滴滴，提出了全面的建议和预算。

并且遇到了与客户履行最密切相关的利益相关者的**完全和彻底的敌意**。

我不是在谈论全面爆发的尖叫比赛、争论或推诿。更像是一种深沉而危险的沉默疗法。

完全拒绝讨论或分享。

只是双臂交叉，死一般的沉默和凝视，你可以感觉到你的双眼之间烧了一个洞。在一次又一次的会议中，我们得到的最多的是各种各样的“我们不会那样做”。

但我不会那么轻易放弃。

所以，正如我的方式，我继续挖掘；不断向组织中不同层次的不同人提问。我沿着这条链向上走，直到最后，我遇到了负责他们所谓的数字履行服务的人——在过去的四个月里，他一次会议都没有出席。

这个人带我们下楼，进入他们公司总部的腹部，向我们展示了显示器、CPU、服务器和打印设备的海洋，它们构成了他们当前的履行服务。他兴高采烈，解释每件作品的作用和工作原理。终于，我想，*我们有所进展了。这个好*。

除了…它不是。

参观结束后，这位高管突然转过身来，好像在过去的一个小时里他一直藏着一把刀。他捅了捅我的胸口，告诉了我一些我永远不会忘记的事情:

> 你最好听好了。我和我的团队，这里的这些人，我们从头开始建造了你在这里看到的每一寸。这些定制系统是我们的孩子，你一直在说的这些该死的“挑战”是我们都有工作的原因。
> 
> 我的工作是保护这些人和他们的家人，我告诉你:如果你继续建议自动化流程和系统来取代我们这里的客户工作，我会以你无法想象的速度扼杀整个项目。你最好相信我有能力也有权力这么做。"

至少可以说，我吃了一惊。

我不知道运行和维护这个由不同机器和代码组成的临时凑合的集合是**这 120 个人和失业之间的唯一障碍。**

吸取教训。

在那次实习中，我学到了已经成为我在 UX 咨询公司长期实习的基本原则:第一天，我希望公司每个部门至少有一名代表出席会议。

任何和我们的工作有一点点关系的人。因为如果房间里有一头公牛，在我们做任何事情之前，我想知道它是谁，以及那个人的意图和议程是什么。

如果做不到，或者客户不同意，我就不接这个活。

在上面的故事中，如果我知道悬在这些人头上的达摩克利斯之剑的存在，我会以一种完全不同的方式去完成这项工作。我会先从他们开始。他们的角色，他们的工作，他们的意图——以及他们的*恐惧*。

所以，当我们结束这个话题时，我请你们永远记住这一点:

你将面临的绝大多数顽固的反对来自于恐惧。这种恐惧会压倒一周中每一天的最佳实践。

在你做任何事情之前，你最初的计划努力应该直接集中在首先消除恐惧上。

因为一旦它摆在桌面上，一旦那些人意识到你是来缓解它的，你的对话和合作将会更有成效一千倍——和更积极的一百万倍。
