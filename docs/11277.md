[![image](../Images/e57f8e5d360a19ce99307def4cdffe2b.png "image")](https://www.sitepoint.com/wp-content/uploads/2010/02/image46.png) 如果你参加了情人节这个标志性的节日，你可能会想到为你爱的人买一张卡片。但是，当你可以在自己的卡片上花一点时间、爱和注意力时，为什么还要购买呢？嗯嗯？设计师你 ***正在*** 设计你的卡，是不是？好吧，如果你正在寻找一些字体渗出类或看起来像一些疯狂的笔迹，这里有一些来自脚本字体类别的建议。如果你想了解更多关于[脚本字体的信息，查看这篇文章](https://www.sitepoint.com/the-script-typeface/)，它也有一些不错的字体例子。

我将字体分为如下几种:

1.  经典书法风格
2.  书法遇到失真
3.  书法
4.  “近乎精神病”的笔迹
