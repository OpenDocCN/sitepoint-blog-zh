# 面向前端开发人员的 20 多份文档和指南(第 8 期)

> 原文:[https://www . site point . com/20-docs-guides-front-end-developers-8/](https://www.sitepoint.com/20-docs-guides-front-end-developers-8/)

最近几个月，我收集了许多不同的学习资源，包括指南、文档和其他有用的网站，以帮助学习不同的前端技术。我在这篇文章中收集了一些最好的。

因此，请享受我们的文档和指南系列第八部分,不要忘记让我知道我还没有包括的任何其他内容。

## [1。键盘事件查看器](http://w3c.github.io/uievents/tools/key-event-viewer.html)

一个可配置的交互式工具，允许您查看键盘事件的数据，显示遗留信息以及在 [UI 事件规范](https://www.w3.org/TR/uievents/)中概述的事件信息。

[![Keyboard Event Viewer](../Images/099cca4ca2bb7a93d00b6a8599d9c0f3.png)T2】](http://w3c.github.io/uievents/tools/key-event-viewer.html)

## [2。jQuery 快速 API 参考](https://oscarotero.com/jquery/)

jQuery 特性的一页备忘单，涵盖了 jQuery 1.10/2.0 之前的版本。只需点击一个特性，它就会在一个模态窗口中打开来自 jQuery 文档的页面。

[![jQuery Quick API Reference](../Images/eb04c4e1b1a6d736ab20ec80f1426571.png)T2】](https://oscarotero.com/jquery/)

## [3。BugRex](http://www.bugrex.com/)

好吧，与其说这是一个“文档”,不如说是一个交互式的帮助指南，在这里你可以和专家聊天。正如该网站解释的那样:“BugRex 聊天是由真正喜欢帮助他人的开发人员社区运营的。用它来理解新概念，提出问题或者修复恼人的 bug。”

[![BugRex](../Images/7ab1ecc33f1f3292f7301f7d6636766c.png)T2】](http://www.bugrex.com/)

## [4。全局 CSS 属性用法](https://developer.microsoft.com/en-us/microsoft-edge/platform/usage/)

如果你喜欢 [SitePoint 最近的 CSS 调查](https://www.sitepoint.com/results-ultimate-css-survey/)，那么你可能也会喜欢这个由微软 Edge 团队编辑的数据。他们每季度扫描超过 100 万个网页，收集 CSS 属性的使用统计数据。它也是可搜索的！

[![Global CSS Property Usage](../Images/f296165ed44b576cf3db22ee6d4216c7.png)T2】](https://developer.microsoft.com/en-us/microsoft-edge/platform/usage/)

## [5。今天的性能工具](http://www.perf-tooling.today/)

帮助您改进工作流程并提供更好更快的网站的资源集合。资源类别包括工具、文章、视频、幻灯片、书籍、课程和绩效审计。

[![Performance Tooling Today](../Images/e0e722f3c95ccb6961e50a9ee4a3b9ae.png)T2】](http://www.perf-tooling.today/)

## [6。歌剧迷你小贴士](http://operamini.tips/)

“Opera Mini 不支持的前端开发功能的集合，以及它们的一些众包解决方案。”

[![Opera Mini Tips](../Images/dd9b9cfd5d0095e84384d8f4a09fc1db.png)T2】](http://operamini.tips/)

## [7。触摸键盘式小抄](http://baymard.com/labs/touch-keyboard-types)

从事电子商务可用性研究的组织 Baymard Institute 编写了一份指南，鼓励开发者优化触摸键盘的表单域。比如如何设置合适的类型，什么时候使用类似`autocorrect`、`autocapitalize`等属性。页面的形式可以在触摸设备上测试。

[![Touch Keyboard Type Cheat Sheet](../Images/f08b4000037d84b4e6b61d450976d3f7.png)T2】](http://baymard.com/labs/touch-keyboard-types)

## [8。HTML5 备忘单](http://www.hostingreviewbox.com/html5-cheat-sheet/)

一个有吸引力的简单的 HTML5 标签及其含义的备忘单，以 PDF 或 PNG 格式提供。

[![HTML5 Cheat Sheet](../Images/0868cf4321cf6f1976b9366cbc20946e.png)T2】](http://www.hostingreviewbox.com/html5-cheat-sheet/)

## 9.HTMLs 的差异

" WHATWG HTML 标准和 W3C HTML 5.1 的非官方比较查看器。HTMLs 的差异提供了规范之间每个部分的并排差异视图。

![Diff of HTMLs](../Images/5685aa1b18154777d3f3da9d44cbc784.png)

## 10。电子邮件客户端 CSS 支持指南

对全球十大最流行的移动、网络和桌面电子邮件客户端的 CSS 支持的完整分类

[![CSS Support Guide for Email Clients](../Images/898fd83e5d39a619d773fa6af703f61d.png)T2】](https://www.campaignmonitor.com/css/)

## [11。巴别塔用户手册](https://github.com/thejameskyle/babel-handbook/blob/master/translations/en/user-handbook.md)

由 James Kyle 撰写的全面指南，涵盖了流行的 JavaScript 编译器 Babel，涵盖了使用 Babel 和相关工具所需了解的一切。

[![Babel User Handbook](../Images/5033d107f8738d23a2c831d61b62f4e9.png)T2】](https://github.com/thejameskyle/babel-handbook/blob/master/translations/en/user-handbook.md)

## [12。A11Y 项目](http://a11yproject.com/)

这不是一个新项目，但该网站最近已被重新设计。这是一个“社区驱动的努力，使网页可访问性更容易。”包括与辅助功能相关的操作方法、神话、测试、提示、辅助技术信息等。

[![The A11Y Project](../Images/0bf98cddb9f17f3b0dbc3e4ae8785c3c.png)T2】](http://a11yproject.com/)

## 13。Flexbox 游乐场

一个交互式 flexbox CodePen 演示，让您轻松测试不同的属性和值。

[![Flexbox playground](../Images/e8485e3a103a8bfe6d834f945f670db6.png)T2】](http://codepen.io/enxaneta/full/adLPwv/)

Joel Lovera 的项目提供了 JavaScript 技巧的档案。该项目原本应该每天提供一个新的提示，但看起来并没有保持下去。尽管如此，这仍然是一个很好的快速提示资源。
