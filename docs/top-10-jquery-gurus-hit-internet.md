# 互联网上最热门的 10 位 jQuery 大师

> 原文：<https://www.sitepoint.com/top-10-jquery-gurus-hit-internet/>

见到那些帮助 jQuery 成为今天这个样子的人不是很酷吗！这些人辛苦编写了大量 jQuery 教程，并在网上分享，帮助我们理解并成为更好的 jQuery 开发者。

嗯，你很幸运！我已经列出了我认为最热门的 10 个 jQuery 开发者。更加深入地了解 jQuery 大师，并从中获得灵感。

 [## 1\. John Resig](http://ejohn.org/) 

[](http://ejohn.org/)

John Resig 是创建 jQuery 的 JavaScript 大师。约翰目前是 Mozilla 公司的一名开发人员，同时也是一名作家，正在努力撰写他的第二本书，JavaScript 忍者的秘密，他计划在 2011 年出版。John 是 Google/Yahoo 等公司的特邀演讲者，并在许多与 web 技术相关的会议上发表过演讲。约翰的[博客](http://ejohn.org/blog/)从 2003 年开始运营，它是任何 JavaScript 框架开发人员的热门选择(22，672+ RSS 订阅者)，尤其是 jQuery。说真的，这绝对是一个传奇！
[网站链接](http://ejohn.org/)

## 2.雷米·夏普

Remy Sharp 是一名网络开发人员，他在自己的[博客](http://remysharp.com/)上写了许多有用的教程和插件。雷米也是非常有用的网站 [jQuery for Designers](http://jqueryfordesigners.com/) 的负责人，该网站有一个方便的简单、中等、困难的帖子搜索过滤器。雷米也是 HTML5 Doctor 和正面全裸的策展人之一，后者是在英国布莱顿举行的为期一天的 JavaScript 会议。雷米的网上业务始于 2006 年，受欢迎程度持续上升。这是一个我想见的人！

[网站链接](http://remysharp.com/about/)

## 3.卡尔·瑞典伯格

Karl Swedberg 是一位才华横溢的 Javascript 开发人员，他是几本 JavaScript 书籍的作者，如《学习 jQuery》和《jQuery 参考指南》。Karl 是流行的学习 jQuery 网站背后的主要人物，自 2006 年以来，他一直在那里为各种能力的开发者分享 jQuery 教程&技巧。Karl 还是 jQuery API 文档负责人，我相信他每天都在帮助成千上万的 jQuery 开发者。

## 4\. Ben Nadal

本·纳达尔是纽约震中咨询公司的首席软件工程师。我真的很喜欢本在他的开发博客上所付出的努力，他在那里分享了关于 ColdFusion 和 jQuery 的深度文章(从 2006 年开始)。我也非常喜欢 Ben 添加到他的帖子中的照片，这些照片让他的网站看起来很有个性。本可能还没有最大的追随者，但这肯定是一个时间问题，因为他把读者放在第一位，他知道他的东西！不同意？[问本！](http://www.bennadel.com/ask-ben/ask-ben-nadel.htm)

[网站链接](http://www.bennadel.com/blog/)

## 5.科迪·林德利

Cody Lindley 称自己是一名客户端工程师(也就是前端开发人员)和恢复型 Flash 开发人员。Cody 是 jQuery 专家，他创建了 jQuery ThickBox 和 jTip 插件，并且是 jQuery UI 开发人员关系团队的成员。科迪在 2009 年开始了他的[博客](http://blog.codylindley.com/archive)，在那里他分享了网络开发帖子的档案。科迪目前为 memolane.com 工作，担任首席客户端工程师。

[网站链接](http://codylindley.com/)

## 6.马克·格拉班斯基

Marc Grabanski 是一名网络开发人员，他从 2007 年开始在他的[网站](http://www.marcgrabanski.com)上分享网络开发技巧和 JavaScript/jQuery，在过去的 4 年里，他的网站已经有超过 300 万的访问量——太棒了！Marc 也是 jQuery UI Datepicker 的最初作者，这是一个很棒的工具，被全世界的 jQuery UI 爱好者使用。马克最近刚刚宣布，他将专注于他的工作“MJG 国际总裁”(阅读更多关于他的 [LinkedIn 简介](http://www.linkedin.com/in/1marc))。

[网站链接](http://marcgrabanski.com/tag/jquery)

## 7.艾迪·奥斯马尼

Addy Osmani 是一个热衷于网络开发的人，他是 jQuery ninja 时代的产物。你们中的许多人可能已经在他的博客(3150 名 RSS 订阅者)上看到了他的卡通忍者形象，自 2008 年以来，他一直在那里分享 jQuery 和 JavaScript 文章。Addy 还是官方 jQuery 核心 bug、docs &前端修复团队的成员，也是 github 的积极参与者，在那里他为当前的 JavaScript 项目做出贡献。好东西伙计！

[网站链接](http://addyosmani.com/blog/)

## 8.詹姆斯·帕多尔西

James Padolsey 是一名自由网络开发者，他在自己的网站上撰写关于 JavaScript 和 jQuery 的文章。此刻，詹姆斯正过着旅行的生活，并在泰国工作了 6 个星期教英语！(不知道他们在泰国做的 jQuery 多不多，呵呵)。詹姆斯也有一个[个人博客](http://blog.padolsey.net/)，在那里他分享了一些有趣的想法……嗯…
[网站链接](http://james.padolsey.com/)

## 9.Soh 田中

Soh Tanaka 是一位充满激情的设计师/前端开发人员，他对设计和灵感有着敏锐的眼光。在 Soh 的[网站](http://www.sohtanaka.com/web-design/category/jquery-tutorials/)上查看漂亮的 jQuery 教程。此外，Soh 最近推出了一个名为[设计炸弹](http://www.designbombs.com/)的 CSS 画廊。厉害的工作伙伴！

[网站链接](http://www.sohtanaka.com/)

## 10.山姆·迪林

萨姆·迪林是一名热爱 JavaScript，尤其是 jQuery 的 web 开发人员，他已经编写了数百个 jQuery 插件。2010 年，山姆的 jQuery 博客发生了变革，他在那里分享日常 jQuery 教程/技巧、插件&代码片段。该博客发展迅速，在最初的几个月里就有超过 300，000 的独特访问量，并且继续快速增长。Sam 目前是 AU 飞行中心的顾问网络专家。

[网站链接](http://www.jquery4u.com/)

### 结论

我希望你喜欢这篇文章，并且对那些试图在互联网上推广 jQuery 的人有所了解，帮助我们成为更好的 jQuery 开发者！
