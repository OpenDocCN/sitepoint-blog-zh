# RockMelt:新的一天，新的浏览器

> 原文:[https://www.sitepoint.com/rockmelt-new-browser/](https://www.sitepoint.com/rockmelt-new-browser/)

你知道，有时候，五种主流浏览器的多个版本是不够的。向前一步[rock melt](http://rockmelt.com/)——埃里克·维什里亚和蒂姆·霍维斯创办的创业公司。他们打算在日益拥挤的市场上推出一种新的改进的网络浏览器。

让 RockMelt 与众不同(并且有新闻价值)的是它得到了网景通信公司创始人马克·安德森的支持。20 世纪 90 年代中期，网景公司开发了互联网的第一个图形浏览器，并让数百万人接触到了早期的网络。尽管网景在第一次浏览器大战中被微软打得一败涂地，但安德森已经成为一名杰出的硅谷金融家。

安德森先生说，RockMelt 将提供不同的浏览体验。他认为，大多数其他浏览器没有跟上网络发展的步伐，仍然停留在静态网页领域，而不是复杂的网络应用程序:

> 如果你是从零开始构建一个浏览器，有各种各样的事情你会做得不同。

尽管他没有详细说明这些不同的东西是什么，但有人猜测新浏览器可能会提供与脸书等社交网络的更紧密集成。

目前，RockMelt 是纯粹的 vaporware——尽管它有一个可爱的标志。只有时间才能证明它是否能提供超越现有浏览器的任何东西。目前，我们可以得出两个结论:

1.  浏览器竞争是好事。我们不应该再回到 IE6 垄断的时代。
2.  如果您的日常工作涉及 web 应用程序测试，它将变得更加安全！

RockMelt 能提供新的东西吗？它不仅仅是一个漂亮的标志吗？多浏览器是帮助还是阻碍？
