表格显示了桌面浏览器的市场份额估计值。“变化”栏是市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 29.3%的 IE9 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](/how-browser-market-share-is-calculated)。

Internet Explorer 在一个月内大幅下降了 2.29%——8%的 IE 用户去了其他地方。虽然 IE10 继续表现良好，增长了 1.2%，但之前的版本下降了近 3.5%。应考虑五个因素:

1.  北半球现在是夏天，大部分人都在度假。当用户不在工作时，IE 的使用会减少——即使是周末也明显减少。
2.  20%的个人电脑用户正在使用 Windows XP。IE8 不可怕，但是老化严重。如果这些用户不能/不想升级他们的操作系统，IE 是一个死胡同，Chrome，Firefox 或 Opera 提供了一个更好的选择。
3.  IE9 很好，IE10 很出色，IE11 看起来令人震惊。然而，虽然 IE 可能是竞争对手的对手，但如果你已经采用了另一种浏览器，就没有什么令人信服的理由再换回来。
4.  Internet Explorer 有图像问题(感知而不是`<img>`！)微软正在努力让开发人员站在一边，但很明显，许多人还没有原谅该公司过去的违规行为。
5.  微软在对抗谷歌；拥有相当大一部分网络的公司。

微软正朝着正确的方向前进，我希望他们的努力取得成功——特别是因为 Chrome 在 6 月份又上涨了 1.3%，获得了 43%的市场份额。离微软 95%的统治地位还有很长的路要走，但是谷歌已经变得令人担忧的强大。他们正在做一些伟大的工作，但是当他们情绪激动时，他们不会反对一点点邪恶。)

其他浏览器这个月也表现不错。Firefox 重回 20%以上，Safari 超过 8%，Opera 小幅上升。这又一次让我相信，除了你和我，每个人都在阳光下玩得开心。

## 移动浏览器使用

2013 年 6 月，手机使用量跃升了几个百分点，占全部网络活动的 16.08%。虽然季节性变化可以解释这种增长，但随着手机和平板电脑成为越来越可行的云计算设备，个人电脑正在失去阵地。

主要的移动浏览应用程序:

1.  **安卓** — 29.06%(下降 1.40%)
2.  **iPhone** — 22.77%(下降 0.72%)
3.  **Opera Mini/Mobile**–16.06%(增长 0.61%)
4.  **UC 浏览器** — 9.89%(上涨 0.80%)
5.  **诺基亚浏览器** — 7.38%(上涨 0.26%)

不要费心去发现趋势或理解数字——移动用户是一群善变的人，他们不会效忠于特定的浏览器或品牌(除非你是一个铁杆苹果粉丝！然而，有趣的是，功能不那么强大的手机浏览器正在崛起。季节可能会解释这一点，尽管非洲和亚洲的快速采用可能是一个促进因素。

就在前五名之外，Chrome 已经超过黑莓，获得第六名的位置和 3.23%的移动市场份额。
