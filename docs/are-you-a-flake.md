# 你是怪人吗？

> 原文：<https://www.sitepoint.com/are-you-a-flake/>

It 世界中有多少 flakes 在工作，这一直让我感到惊讶。是的，薄片。

就在本周，我准备聘请一家 IT 公司开始一个项目。我们同意在星期三开始这个过程。他们出现了吗？他们交流吗？没有。他们最终会发一张便条，说他们将准备好周一开始工作。薄片！

你是怪人吗？如果以下任何一条适用于你，其他人可能会认为你是个怪人:

你开会迟到了

–您交付的产品迟到了

–你找借口，比如技术或网络故障，来解释迟到的原因

–您和您的客户认为您正在交付的东西之间经常存在不匹配

–你为了得到一份工作而夸大了你的简历(这意味着你可能不具备这份工作的资格，但不愿意公开和诚实地说明你的资格——这在科技领域非常普遍，尤其是在硅谷，我曾经在那里为 IT 人员查询过推荐信)

–你的客户抱怨你做事情花了太多时间，即使你的质量最终是好的

–在您的应用程序中发现了根本性的缺陷，比如安全漏洞，您的客户应该认为这些问题已经得到了解决(就像飞机乘客认为飞机会安全着陆一样)

–你倾向于指责超过 25%的客户“没有明白”(不明白的可能是你)。

–在销售过程中，您没有按照约定按时提供专业文档。如果你在这个过程中没有表现出专业水准，客户会对你的真实作品有什么看法？

–你很少能按时、按预算或按预期质量完成交付，所以当你告诉客户你将创建一个项目计划时，他们开始傻笑；换句话说，他们不信任你。

这是部分名单。有多少人申请你？我希望没有。
