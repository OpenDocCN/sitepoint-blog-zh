# 域名淘金第 2 部分-保护最佳域名

> 原文：<https://www.sitepoint.com/securing-best-names/>

**在[“上午 6.30 域名淘金热”](http://www.ecommercebase.com/article.php/352/)的文章中，我试图解释目前围绕着过期域名业务的讨论。这篇后续文章将尝试回答第一篇文章中提出的一些最常见的问题:**

**Q)** 如果有些名字如此有价值，为什么人们会放弃它们而不是更新它们？

**A)** 人们不续写有价值的名字有很多原因。有时所有者破产了，或者只是对网络世界失去了兴趣。其他时候，他们不知道这个名字的价值，只是为了节省续费而放弃它。许多域名丢失是因为注册人没有及时更新他的域名注册信息。特别是，由于所有者改变他/她的 ISP 或其他原因，电子邮件地址有变得“过时”的习惯。如果注册人不容易通过电子邮件联系，许多注册服务商根本不会尝试联系他们。最终结果:注册人不知道域名即将到期，丢了。然后，它会被放回“过期名称”池中。监禁和死亡是人们不更新域名的一些更永久的原因。这个清单实际上是无穷无尽的。

我可以创建自己的过期姓名列表吗？

**A)** 理论上，是的，有可能。您需要从下载区域文件开始。这些文件包含当前注册的所有. com、. net 和. org 的完整列表。一旦他们认可你，你可以从[威瑞信](http://www.verisign-grs.com/dns/dnsonli.html)免费获得它们:

不过，要注意。这些文件非常大，总共有 500 兆字节。这只是开始。一旦下载，你必须解压它们。这些文件有几千兆字节大小。您必须对它们进行分析，以找出哪些名称目前处于“保留”状态。一旦你有一个域名列表，然后你必须每天扫描它们，看看哪些即将到期。截至目前，我们正在讨论每周扫描成千上万的名字。

这就是理论。实际上，找一家专门从事这项工作的公司要容易得多。有几个这样的“过期域名”服务，但要小心。这些服务中有很多都“漏掉”了最好的名字，所以他们可以把这些名字卖给私人客户或者留给自己。[域名机器人](http://www.domainsbot.com)提供卓越的服务，最近刚刚被《T2 时报》报道

**Q)** 我想注册域名转售。我如何最大化我注册几个最好的名字的机会？

毫不奇怪，这是最受欢迎的问题！坏消息是，你正在与成千上万的其他人竞争，其中一些人是专业人士，已经这样做了多年。好消息是注册最佳过期域名没有秘密可言。如果您的请求是该名称可用后第一个到达注册表的请求，您将获得该名称。为了最大化你获得尽可能多的好名字的机会，首先你需要做很多事情。

所有这些技术的基础是，最终，您处理的是一个简单的概率问题，即在域名被删除时，您向注册管理机构发送的请求越多，您获得域名的机会就越大。

**1)使用几个不同的注册服务商**
在任何一天，一个注册服务商可能会工作缓慢，而另一个注册服务商可能会以最高效率工作。在另一天，一个注册商可能会完全关闭。在几个不同的注册商那里建立账户，同时尝试在所有的注册商那里注册名字！

**2)使用多个浏览器窗口**
对于许多注册商，可以使用一个帐户打开多个浏览器窗口。这样，您可以在短时间间隔内向他们发送多个请求。

使用几台电脑
对于真正认真的域名抢注者来说，这是必须的。甚至有传言说，韩国专业人士的房间里装满了电脑，只是被用来抓取下降的域名…

使用快速互联网连接
互联网连接越快，你就能越快地向注册中心发送请求，也能越快地得到回复。如果你对抢名字很认真，就把你的 56k 拨号线折价贴换；)

**5)使用具有“批量注册”功能的注册服务商**
如果您使用的注册服务商界面“缓慢”(这意味着向注册管理机构发送一个注册请求需要几分钟时间)，那么您可能无法获得任何合适的域名。您需要找到具有批量注册功能的注册商，最好是单击式批量注册功能。

**6)使用反馈机制**
每周都有一次大的下跌；列一个名单，上面有十个掉队的名字。第二天，去像 http://www.powerwhois.com 这样的 WHOIS 工具，记下哪个域名注册商注册了最好的域名。如果你在一段时间内做到这一点，你会对哪些注册服务商在注册被撤销的域名时真正做到了这一点产生好感。

警告:当然，发送尽可能多的注册请求的最终方法是使用定制的脚本。这样，就有可能在一分钟内发送数百个请求。这是一种非常反社会的做法，如果广泛采用，只会导致整个登记制度定期崩溃。大多数域名注册商也明确禁止，如果你被抓住，你的帐户将被关闭。不要这样做。

**Q)** 不想注册名字转卖；我只是对注册一个特定的网站名称感兴趣。我如何在投机者之前最大化我抓住它的机会？

好了，首先是坏消息。你可能知道这个名字是为了“正确使用”,但注册中心不知道。对他们来说，每个请求都和下一个请求一样有效。因此，你没有特别的优势，试图注册一个即将放弃的名称。好消息分为两部分:

**(i)** 大多数名称，90%或更多，对经销商没有价值。如果你的名字属于这一类，你注册这个名字就不会有任何问题。你可以使用任何域名注册网站这样做。
**ii)** 一家名为 [SnapNames](http://www.snapnames.com) 的公司推出了一项名为“Snap-Back”的新服务，让你在空投过程中有很大机会获得名字。每个名称的费用为 35 美元，但如果 SnapNames 没有为您获取名称，费用将会退还。那么 SnapNames 是如何在投机者之前抢到名字的呢？你刚才不是说‘每一个请求都和下一个一样有效’吗？绝对正确。但 SnapNames 比大多数投机者更有优势，特别是他们是一个组织良好、资金充足的团队，并且与几个域名注册商关系密切。他们使用许多投机者用来抢注名字的技术。他们只是更好地实现了它们。

名字到底是什么时候掉下来的？

在美国东部时间早上 6:30 之前，没有名字被删除。但大多数名字似乎在几分钟后就消失了。美国东部时间上午 6:45-6:50 之后，名字似乎永远不会消失。
