# 什么是社区？

> 原文:[https://www.sitepoint.com/what-is-a-community/](https://www.sitepoint.com/what-is-a-community/)

做一个人需要什么？是志趣相投的人，有共同的理想，信仰或者其他一些共同的价值观？做一个社群需要一定数量的人吗？

如果你周围的人鼓励和支持你，即使你充满了它，他们是一个好的社区吗？如果你渴望加入一个小团体或精英，而这个小团体决定接受你，那么你是这个团体的一部分吗？

我相信社区就是互相尊重。人们可以形成一个没有共同信仰的社区，没有共同的身份或根据彼此来定义自己。数字无关紧要；理想和欲望无关紧要；甚至相互支持和赞助的程度在很大程度上也无关紧要。群体是一种心态。这是一种归属感，来源于别人接受你，而不是因为你是谁。

当我来到澳大利亚，我这样做是出于个人原因；我不是为了事业而来，也不是为了人群而来，更不是为了归属感而来。我来这里是因为我的心告诉我要来，即使理由是错误的。我从来没有试图合理化。我没有试图说服自己来这里有其他原因，我的职业生涯，我的社交生活，或者我的生活方式会有所改善。哦，等等，不，那不是真的。我确实试图说服自己相信所有这些事情——在某种程度上，我成功了；因为在某种程度上，那些事情是真的。

但它们从来都不是理由，它们只是临时的理由。

我的理智化为尘土，我在这里，孤独，孤立，不快乐。我在这里只有一个真正的朋友(每天我都感谢他的友谊)，而我已经树立了一大堆敌人。但是你知道吗——我还是留下了。因为我要找的*是这里的*。我要找的东西到处都是。

世界是一个共同体；我们控制不了自己，事情就这样发生了。
