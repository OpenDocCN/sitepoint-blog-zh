# 你有恐怖症吗？

> 原文：<https://www.sitepoint.com/do-you-suffer-from-kakorrhaphiophobia/>

根据彼得·鲍勒的书《高人的第二本古怪和奇妙的话》，上述术语意味着对失败的病态恐惧。(他注意到这个词的讽刺意味，因为肯定有人必须告诉他或她的医生他患有这种疾病，会通过试图正确发音来引发这种疾病！).

对失败的恐惧是任何个体网页设计师/开发者必须克服的，而且要快。这很难做到，因为大多数专业人士都被训练成完美主义者。

为了最终取得成功，你几乎不得不在以下领域失败:

1.营销。有些营销策略会奏效，有些则不会。并且大多数需要不断测试和改进。如果你在第一次得不到好的回应后就放弃了，你将不会走得很远。

2.产品。有些产品会有需求，有些不会。而且推出一款“足够好”的产品比等待完美更好。

3.商业投资。大多数企业家很幸运能获得一次成功的创业。有时可能要尝试 10 次才能找到一个好的。我喜欢猎豹捕捉猎物的方式，正如 Sitepoint 上的其他文章所指出的:如果你看到一个机会，就快速追逐它。如果你看到它溜走了，停下来，喘口气，明天再去追逐不同的猎物。

4.任何形式的客户或人类互动。是的，我们都应该在任何时候都努力做到专业。但是人类很狡猾。我们每个人都不一样，对事物的解释也不一样。我们很难知道自己给别人留下了什么样的印象，因为这种印象既取决于自己，也取决于他人。所以我们难免会在给某些人留下好印象方面失败。希望随着时间的推移，我们能够更好地感知不同的风格并适应它们。

所以，走出去，做得足够好而不是完美，不时地失败。最好的(A+)学生经常最终为好的(B)学生工作。开始做一个好学生；别再做 A+的学生了。

跟我一起说:不要再有恐怖症了！
