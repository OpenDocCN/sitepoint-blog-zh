# 了解您的信息

> 原文:[https://www.sitepoint.com/know-your-message/](https://www.sitepoint.com/know-your-message/)

如果你在过去的几个月里看过或关注过新闻，你肯定听说过“占领华尔街”( OWS)抗议活动已经蔓延到美国的各个角落。不，我不想卷入政治，但从抗议者身上可以学到营销的一课。

上个月，我在纽约市遇到了一些 OWS 抗议者，在新闻、YouTube 以及通过电子邮件和社交媒体传播的媒体上，我看到了对全国各地占领运动抗议者的几次采访，有一件事非常清楚——他们传达的信息并非如此。

许多抗议者在几十个不同的问题上东拉西扯，常常语无伦次，没有明确的行动计划或解决问题的建议。“运动”是支离破碎的，缺乏一个明确和统一的信息是真正伤害他们的机会被听到。

对企业主来说，这是一个重要的教训……你必须知道你的信息。你是谁？是什么让你与众不同，比你的竞争对手更优秀？当有人在鸡尾酒会上问你“你是做什么的？”你能毫不犹豫地用一句话回答他们吗？

如果没有，你需要认真思考你的信息——你在做什么？我建议提出三条“信息”——简短的一行“这就是我们”，较长的一到两分钟的关于你的简介，稍微深入一点，最后是详细的信息，足够填充一个采访或“关于”页面。

一个清晰、简洁的信息不仅向潜在客户和供应商解释了你的公司是什么，它还传达了一种信心，即你知道你是谁，为什么你适合这个工作。
