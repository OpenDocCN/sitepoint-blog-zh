# 谷歌发布 Chrome 6

> 原文：<https://www.sitepoint.com/google-chrome-6/>

尽管只有 2 岁，Chrome 已经到了第 6 版。平均来说，谷歌每 4 个月发布一个新版本，第 5 版于 2010 年 5 月底出现。

你可以说我愤世嫉俗，但我确信快速发布时间表纯粹是营销炒作。当 Internet Explorer 9.0 最终到来时，Chrome 将会是第 8 版，并将很快超过微软的浏览器。虽然很少有人关心这些数字，但这并不能阻止像我这样的记者和博客作者报道新版本已经推出。关于火狐 3.6 或 Opera 10.6 出现了多少新闻故事？他们有了更多的新功能，但是版本号似乎并不重要，他们得到的媒体报道也少得多。

营销阴谋说够了 Chrome 用户能从更新中期待什么？

新的默认主题
Chrome 一直很干净，但新的主题比以前更微妙、更简约。在 Windows 7 上，窗口是透明的灰色，干扰更少。

![Chrome 6 screenshot](img/f475ec69319ebcf45cb7169feaaaeabd.png)

一些菜单已经合并，例如，剪切、复制和粘贴在一行中。还有一个新的“创建应用程序快捷方式”选项，允许你添加网页链接到你的操作系统桌面或菜单。

表单自动填充
自动填充在 Chrome 上花了一些时间，但谷歌有一个有趣的实现。Chrome 不会简单地将字符串与同名的输入字段关联起来，而是会记住一组地址和信用卡号码。它运行良好，只需点击一下就可以输入表单细节。

如果你在不止一台电脑上使用 Chrome，你可能会同步书签、密码和主题。新版本增加了自动填充和扩展的自动同步。当你升级操作系统或切换平台时，它使 Chrome 安装起来更快更容易——你没有理由再去购买另一款浏览器。

改进的地址栏
谷歌可能会称它为“万能栏”,但我绝不会这么说！最显著的变化是从网址的开头删除了“http://”。这是一个明智的举动，因为很少有用户知道或关心这意味着什么。安全地址仍然显示为完整的 https://以绿色突出显示。

书签星移到了右边,“前进”箭头被移除了(好消息！)您可能会注意到出现了一些其他图标，如 cookie 指示器。

谷歌自己的开源免版税视频格式 WebM 首次出现在 Chrome 浏览器中。它是否有机会对抗 [H.264](https://www.sitepoint.com/h264-video-royalty-free-html5-standard/) 还有待观察。

**标准和速度**
不用说，Chrome 6 对 web 标准的支持更好，速度也比它的前辈更快。它在这些方面已经很好了，所以大部分用户几乎察觉不到改进。

开发者工具也获得了更新。我怀疑许多人仍然更喜欢 Firebug，但它是标准浏览器附带的一套令人印象深刻的工具。

对 Chrome 没什么不喜欢的。你可能不欣赏谷歌的版本编号或野心，但他们已经产生了一个伟大的浏览器，在相对较短的时间内吸引了大量的追随者。

Chrome 可以从 www.google.com/chrome 下载。如果你已经是第 5 版，点击工具图标，关于谷歌浏览器，并按照更新说明。
