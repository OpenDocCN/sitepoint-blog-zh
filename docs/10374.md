# 7 个有用的 jQuery 拖放插件

> 原文:[https://www.sitepoint.com/7-jquery-drag-n-drop-plugins/](https://www.sitepoint.com/7-jquery-drag-n-drop-plugins/)

现在大多数网站设计师都不可阻挡地探索设计和用户界面的世界。这导致了聪明的想法和更好的计划。他们可以想出更多的特殊效果，将被现代网站使用。

现在，我这里有一个 jQuery“拖放”插件的列表。这与桌面应用程序和其他应用程序的拖放操作是一样的。我自己也尝试过一些 jQuery 拖放插件，我必须说它们真的很酷很有用。

所以在这里它们是:
 [](http://plugins.jquery.com/project/ppdrag) 

## [1\. PPDRAG](http://plugins.jquery.com/project/ppdrag)

一个简单的 jQuery 插件，提供典型的拖放功能，但主要关注性能。与其他插件不同的是，当用户有一个缓慢或旧的 CPU 时，它会从性能中脱颖而出。
![](../Images/146be92e99e1a54675cbae58e57374bd.png)
*来源:*
[http://plugins.jquery.com/project/ppdrag](http://plugins.jquery.com/project/ppdrag)

 [T2】](http://plugins.jquery.com/project/NestedSortable)

## [2。嵌套存储](http://plugins.jquery.com/project/NestedSortable)

这允许您垂直排序，并使用拖放功能使一个元素成为另一个元素的子元素。
![](../Images/25aaae3897fa700d24b4eb648758b28e.png)
*来源:*
[http://plugins.jquery.com/project/NestedSortable](http://plugins.jquery.com/project/NestedSortable)
