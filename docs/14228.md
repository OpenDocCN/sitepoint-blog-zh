# 流动的力量

> 原文：<https://www.sitepoint.com/the-power-of-flow/>

下周我会在华特·迪士尼巡游，没有能力写博客。所以让我们把这一周变成关于你的，而不是关于这个博客的。

我想让你把你的心流力量的经历贴出来。

流量是指每天有多少商机来找你。这是商学院职业管理课程中的一个重要概念。这个想法是有一个联系和工作机会的网络，给你带来更多的机会。成功人士似乎总是在寻找新的机会。他们就像河边的渔夫，站在那里，看着水和鱼从他们身边流过，让他们去采摘。

有很多方法可以增加流量”

–在你的社区中引人注目，尤其是在高层人士中

–说服一位成功人士来指导你

–在一家不断发展、充满活力的公司工作

–与心流中的其他人建立联系，通常是银行家、投资者、风险资本家、商业律师、某些会计师等。

–写和说——成为受欢迎的专家。

–建立一个网络，让至少 10 个人每月为你带来至少 1 条线索。

所以，在接下来的一周里，请在这个博客上发表你的积极经历和挑战。

–现在你最好的流量来源是什么？

–你将如何增加流量？

–有哪些你可以分享的成功故事？

–您面临哪些挑战？

祝你一周愉快。
