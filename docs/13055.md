# 向最优秀的人学习:SitePoint 大师

> 原文:[https://www . site point . com/introducing-the-site point-guru-lists/](https://www.sitepoint.com/introducing-the-sitepoint-guru-lists/)

[![The articles that comprise the CSS Guru List](../Images/608444694961d358e9b659d456a238af.png)](https://www.sitepoint.com/gurulists/) 面对庞大的在线文章档案库(最近一次统计超过 1400 篇)我们面临的一个挑战是查找信息可能很困难——尤其是如果你刚刚开始学习网页设计的话。

例如，我们有 35 篇关于 CSS 主题的文章——对于初学者来说，知道先读哪篇文章可能会很困惑。

正如我在之前的中[讨论过](https://www.sitepoint.com/interpreting-the-results-of-the-2008-sitepoint-reader-survey-part-1/) [一样，我们意识到文章导航方面可以做一些改进，并且有很多变化正在酝酿中。第一个是](https://www.sitepoint.com/interpreting-the-results-of-the-sitepoint-reader-survey-part-2-content/) [SitePoint Guru 列表](https://www.sitepoint.com/gurulists/)。

每年的 [SitePoint 社区奖](https://www.sitepoint.com/forums/showthread.php?t=515580)都会表彰那些在自己选择的专业领域帮助他人方面做出杰出贡献的 [SitePoint 论坛](https://www.sitepoint.com/forums/)成员。我找到了去年的获奖者，请他们为自己被认可的类别创建一个他们最喜欢的 SitePoint 文章的列表。结果是 [**一个由专家精心挑选的令人敬畏的文章集合**](https://www.sitepoint.com/gurulists/) 。

这些文章是按照阅读顺序排列的，所以如果你是第一次讨论一个话题，每个列表中的第一篇文章都会对你对这个话题的了解程度做最少的假设。你可能已经注意到这些列表也出现在一篇文章或博客帖子的左侧栏中——希望它们能够吸引那些可能从搜索引擎到达并正在寻找起点的访问者的目光。

非常感谢我们所有的大师们花时间整理这些清单。我希望您会发现它们有助于提高您对 SitePoint 上涵盖的许多主题的了解。

访问 [SitePoint 大师列表](https://www.sitepoint.com/gurulists/)。
