# 我不干了！冒险之前要考虑的事情

> 原文：<https://www.sitepoint.com/leaving-a-full-time-job/>

当人们开始从事自由职业或开始创业时，会有很多选择。当一些人决定为自己工作时，他们被解雇了。其他人在业余时间从事自由职业的同时也兼顾全职工作。其他人则完全放弃全职工作，专注于发展自己的事业。而有些人其实从职业生涯一开始就做自己的事，甚至从来不为别人打工。

我的旅程是被解雇、寻找另一份工作和在企业界工作期间兼职创业的结合，并建立了冒险的信心。无论你如何到达那里，考虑离开你的工作自己出去，这可能是一个非常可怕的(也是令人兴奋的)前景。根据我的经历，这里有一些你在递交辞呈前应该考虑的事情。

**财务**

辞职前要考虑的最重要的因素是钱。如果你一直在兼职做自由职业者，估计你的财务状况可能会更容易，至少在短期内是这样。如果你正在考虑戒烟，然后努力建立自己的企业，现实一点。独自工作可能需要几个月，但通常需要几年才能看到利润，所以确保你有足够的财务储备支付至少 6 个月的生活费用，如果可能的话，18 个月。

别忘了为自己工作的成本——设备、软件、法律费用、营销活动等等。

**医疗保健**

如果你没有配偶或有福利的伴侣可以声称你是受抚养人，你将需要留出资金购买健康保险。这可能非常昂贵，有时很难以可承受的价格找到足够的覆盖面。

**一个计划**

即使你没有一份正式的商业计划，你也应该写一份书面文件，概述你想自己完成什么，以及计划如何完成。创业需要多少成本？你需要赚多少？你将提供什么产品和服务？你的目标是谁？你如何联系他们？你的服务如何适应当前的市场？你什么时候在哪里工作？

**支持**

如果你没有你所爱的人的支持，靠你自己去努力会是一场持续的斗争。如果他们在船上，你的家庭成员可以成为你最大的啦啦队员。确保他们知道你计划的所有要素，并同意这是你实施计划的正确时机。

**性格**

你有支持自主创业的性格吗？你需要能够集中注意力，让自己坚持下去，即使你遇到了障碍。你还应该能够有效地独自工作，或者有一个寻找协作工作环境的计划。记住，独立工作需要大量的训练。如果你不能在截止日期前完成工作，你将面临一段困难时期。

虽然当你考虑为自己工作时，似乎会有很多事情与你作对，但如果你花时间为自己的成功做计划和定位，这绝对是可能的。一旦你冒险尝试，体验了在你想做的时候做你想做的工作的自由，你就再也不想回到公司了。

你走了什么自主创业的道路？你有什么要补充的吗？
