# 另一个你可以责怪潜在客户的博客

> 原文：<https://www.sitepoint.com/another-blog-where-you-can-blame-the-prospect/>

前两篇博客中的模型讨论仍然让我不满意。

我有 1 个非营利和 9 个电子商务网站，在看了数百名设计师的投资组合后，仍然只有 2-3 个为我完成了工作。

现在想象一下一个甚至不太关注网络的企业主的困境。

很难找到好的设计师，原因如下:

1.像 eLance 这样的网站，正如你们很多人注意到的，通常会提供一长串有着平庸作品和一般焦点的人(“我们什么都做！”)，并且不产生信任。

2.在谷歌上搜索网页设计师/开发者对潜在客户来说是一场灾难。成千上万的模仿公司出现了，无论是在第一页的顶部还是第十页的底部。完全没有办法知道该联系谁。

3.虽然本地公司可能在口碑和社区服务方面做得很好，但他们很少有符合我的特定需求的投资组合(“我的”是指企业主的特定行业和喜欢/不喜欢)。

4.设计师和开发人员使用他们的语言，而不是企业主的语言。

5.根据许多对以前博客文章的回复，许多(不是所有)设计师和开发人员都认为我是对的/你这个顾客是错的。这对潜在客户来说是一个直接的障碍。

6.有人提到“作品集，作品集，作品集。”这是第五种态度。对我来说，所有的投资组合看起来都有点相似，因为我没有你的平面设计或开发的区别。除非你做了非常接近我想要的东西，除非它符合我的规格，否则我很难得到它。我的错还是你的错？

7.我面试设计师的时候，过程往往令人失望。据称资历很好的设计师/开发人员开会迟到，几周内不能开始一个项目，对时间表不做任何承诺(甚至是大概范围)，没有试图理解我的业务，等等。因此，面试过程会剔除更多所谓的人才。创意/技术领域之外的商人想当然地认为专业人士就是专业人士。

因此，虽然你们中的许多人很难找到好的、合格的前景，但我们中的许多人也很难找到好的、合格的网页设计师/开发人员。可能更难。

接下来的几周，我们将回到基础，尝试解决这个难题。

好吧，去吧，责怪前景吧。找不到好的设计师是我们的错吧？
