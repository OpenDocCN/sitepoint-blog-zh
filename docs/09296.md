# 10 个简单干净的 jQuery 插件

> 原文:[https://www.sitepoint.com/simple-clean-jquery-plugins/](https://www.sitepoint.com/simple-clean-jquery-plugins/)

In this post is another collection of jQuery plugins for you daily jQuery head fix. They are very basic plugins but good nevertheless and do what they say on the tin. Could come in useful for your next project! Enjoy!

## 1.一批

jQuery 插件，它让你能够将任何 jQuery 方法或插件的结果批处理到一个数组中。
![Batch.jpg](../Images/8515a77a2bdd9e92a98407cb829a60f4.png)
[来源](http://www.github.com/brandonaaron/jquery.batch)

## 2.可数的

一个 jQuery 插件，提供了一个不显眼的输入和文本区字符计数器。
![Countable.jpg](../Images/fafb2a7d2638ae25037d2f72679d7e6e.png)
[来源](https://github.com/brandonaaron/jquery-countable)

## 3.Livequery

利用 jQuery 选择器的强大功能，自动绑定事件或触发匹配元素的回调，甚至在页面加载和 DOM 更新之后。
![Livequery.jpg](../Images/9bf3addcd4d4e39bb979bdfa2d40204f.png)
[来源](https://github.com/brandonaaron/livequery)

## 4.重复

检测一个或多个元素是否重叠的 jQuery 插件。
![Overlaps.jpg](../Images/12e18efe79104a3f5f53d3284da94487.png)
[来源](http://www.github.com/brandonaaron/jquery-overlaps)

## 5.每个切片

jQuery 插件，让你为每个元素片段迭代给定的函数。
![each_slice.jpg](../Images/9bdd80b847a93e1c543fed5ca0516354.png)
[来源](https://github.com/burin/jquery-each_slice)

## 6.拼写检查

一个 jQuery 插件，使用 Google 拼写检查 API 为输入添加拼写检查支持。
![spellcheck.jpg](../Images/b7be90554ea1adc710fa518564e4720b.png)
[来源](http://www.github.com/brandonaaron/jquery-spellcheck)

## 7.复制事件

一个 jQuery 插件，提供了将事件从一个元素复制到另一个元素的方法。
![copyevents.jpg](../Images/9a532a9a96aa19fd4cd260955b6a9887.png)
[来源](http://www.github.com/brandonaaron/copyevents)

## 8.可膨胀的

一个 jQuery 插件，可以自动扩展文本区域以适应用户输入的内容。
![expandable.jpg](../Images/547898e633bbcec4fb00f700c785b4d9.png)
[来源](https://github.com/brandonaaron/jquery-expandable)

## 9.重叠标签

一个 jQuery 插件，帮助实现迈克·布里泰因在 ALA 上的《让紧凑表单更易访问》中描述的覆盖标签技术。
![overlabel.jpg](../Images/d392ed9287f3965c32925dbe5b5627a3.png)
[来源](https://github.com/brandonaaron/jquery-overlabel)

## 10.实时计数器

jQuery plugin 是一个动态灵活的计数器，可以根据一些参数进行设置，并且可以直接在页面上更新计数，而无需刷新。
![Live-Counter.jpg](../Images/e0476bfec1e3f259d463da77fbc12b1a.png)
[来源](http://uniquemethod.com/projects/live-counter)
