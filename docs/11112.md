# iPad 热席卷美国

> 原文:[https://www.sitepoint.com/ipad-fever-hits-the-us/](https://www.sitepoint.com/ipad-fever-hits-the-us/)

今天是情人节。苹果公司的新 iPad 在美国发布后，你在网上任何地方都无法不读到它。看一看 [Twitter 的#iPad 流](http://twitter.com/#search?q=%23iPad)——它每分钟收到大约 300 条推文，这些推文来自在苹果商店外排队或拆箱他们心爱的小玩意的热切购买者。

尽管 iPad 有几个明显的遗漏——没有多任务功能、没有 USB 接口、没有摄像头、没有 Flash 插件——但评论普遍对其赞不绝口。

《纽约时报》的大卫·波格写道:

> iPad 如此之快、如此之轻，多点触控屏幕如此明亮、反应如此灵敏，软件操作如此简单，以至于它确实有资格成为一种新的电子产品

《华尔街日报》的沃尔特·莫斯伯格:

> 如果人们认为它是一种大部分时间取代更重、更笨重的电脑的方式——用于网上冲浪、电子邮件、社交网络、视频和照片浏览、游戏、音乐甚至一些轻松的内容创作——它可能会像苹果的 iPhone 一样改变游戏规则。

《今日美国》的艾德·贝格:

> iPad 不在于你能做什么——浏览网页、收发电子邮件、玩游戏、阅读电子书等等，而在于你如何去做。这就是苹果正在改写主流计算规则的地方。没有鼠标或物理键盘。一切都是基于触觉。

Xeni Jardin 为《泰晤士报》撰稿(英国):

> 没有人真的需要 iPad，但它把网络变成了我们想要的样子。

苹果的 iPad 宣传已经取得了成功，已经有 25 万人预订了这款电脑。不幸的是，我们这些美国以外的人需要再等一两个月才能亲自检验 iPad。

平板电脑已经出现很多年了，所以我还不相信它是一种改变人们对计算认知的过渡设备。然而，我能理解这种吸引力——这可能是第一款为触摸屏和网络浏览提供真正直观界面的平板电脑。对于娱乐和轻量级计算任务，它可能是上网本的更好替代品。

我真的很想听听谁有新 iPad。你的第一印象是什么？几个小时后你怎么想的？您将使用该设备做什么？是有能力的网页浏览机吗？评论正确吗？
