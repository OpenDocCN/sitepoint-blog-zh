# jQuery 代码仅在 Firebug 打开时运行

> 原文:[https://www.sitepoint.com/jquery-code-runs-firebug-open/](https://www.sitepoint.com/jquery-code-runs-firebug-open/)

![firebug1](../Images/a06b7e8b963c8d9c60d989e5cb73423e.png "firebug1") ![ie-sad](../Images/daa3a445daf37546fc7a82582df1b9c0.png "ie-sad")

所以您可能想知道为什么您的 jQuery 代码只在 firebug 在您的浏览器上打开时才运行。这可能是因为您在代码中使用了 console.log 命令，而 jQuery 代码因为控制台不存在而失败。

要修复这个简单的把你的 console.log 和 firebug 命令放在下面的代码里面:

```
if (window.console) {
	console.log(text);
}
```
