# 来自 Behance 网络的十种免费显示字体

> 原文:[https://www . site point . com/ten-free-display-fonts-from-the-behance-network/](https://www.sitepoint.com/ten-free-display-fonts-from-the-behance-network/)

[![VAL](../Images/84a59cd03723252623169df2ba84e442.png "VAL")](https://www.sitepoint.com/wp-content/uploads/2010/01/VAL.png) 除了为平面设计师、插画师和摄影师提供作品集，你还可以在 [Behance](http://www.behance.net) 网站上找到排版师和设计师提供的一些不寻常且有趣的字体。创造一种字体是一项漫长的工作，所以向免费赠送他们辛勤工作的印刷工人脱帽致敬。

这些字体中的许多是显示字体，使它们适合于海报和标题，而不是正文。

font fabric 的 VAL 是一种矮胖的泡泡风格字体。通过选择字母的大写或小写版本，可以翻转字母的方向。你可以从[字库](http://fontfabric.com/?p=585)下载。

[![VAL](../Images/69e537d13a9b5c06c133979cf7e35c93.png "VAL")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/VAL1.png)

Deibi 是一种无衬线字体，有点卡通风格，来自西班牙设计师 [Wete](http://www.behance.net/wedtmo) 。

[![deibi](../Images/8b49ae8210faa18ecf0461e3d64e7cbf.png "deibi")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/deibi.png)

[![deibi2](../Images/1aa162a09bc85e722f8dda39931e99b7.png "deibi2")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/deibi2.png)

Lot 是一种加粗的超粗显示字体。也可以从[字库](http://www.behance.net/Gallery/LOTa-Free-font/359440)下载。

[![LOT](../Images/b1aae47c60c9d92591cf7ee38f8615a1.png "LOT")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/LOT.png)

[![Lot-Pac](../Images/7205d2360950faee1eb64c7664798e40.png "Lot-Pac")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/LotPac.png)

[Days](http://www.behance.net/Gallery/Days-typeface/190108) 是一种显示字体，有一种重量，有点俄罗斯的感觉。

[![days](../Images/5fcffc451ca23506cab11faa4abbd114.png "days")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/days.png)

[Zag](http://www.behance.net/Gallery/Free-font-Zaga/251250) 是一种漂亮的无衬线字体，有两种风格(常规和粗体)可供免费使用，[购买了另外五种风格的](http://new.myfonts.com/fonts/font-fabric/zag/)。

[![zag](../Images/d35e975ec5321e0ff55c275d7dea07e1.png "zag")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/zag.png)

公共哥特式字体是由 Mehmet Gozetlik 发布的免费字体。"它没有多少工业气息，没有多少复古气息，没有多少浓缩，没有多少大胆."你可以从 a2591.com 的[下载。](http://www.a2591.com/)

[![Antrepo4](../Images/6aaf02d75109efb8791184ed365a3cb0.png "Antrepo4")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/Antrepo4.jpg)

从 noupe.com 的[可以免费下载](http://www.noupe.com/freebie/tribbon-layered-font-exclusive-free-download.html)[带状字母表](http://www.behance.net/Gallery/Ribbon-Alphabet/248759)字体及其变体 Tribbon。这不是一个可以安装在你系统上的字体，而是一个分层文件，但是你可以在 noupe 网站上看到一个解释它如何工作的视频短片。

[![ribbon](../Images/38cdcf28db6f1c3bbdb32a4d5ed26e46.png "ribbon")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/ribbon.png)

[![ribbon2](../Images/8dd53b5216c32f82902d16fbf332e937.png "ribbon2")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/ribbon2.png)

以詹姆斯·邦德的电影命名， [Thunderball](http://www.behance.net/Gallery/Thunderball-Typeface/246064) 是一种由约翰·斯凯尔顿/阿弗罗杰特创作的大字体，可以从 [Fontstruct](http://fontstruct.fontshop.com/fontstructions/show/thunderball) 下载。

[![thunderball](../Images/f78dd21373b0d67f34ffd598559ba297.png "thunderball")T2】](https://www.sitepoint.com/wp-content/uploads/2010/01/thunderball.png)
