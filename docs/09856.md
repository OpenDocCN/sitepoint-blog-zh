# 谷歌 Chrome 11 有什么新功能

> 原文:[https://www.sitepoint.com/google-chrome-11-whats-new/](https://www.sitepoint.com/google-chrome-11-whats-new/)

从上次 Chrome 更新到现在至少有 3 天了，所以现在我们发现自己已经是第 11 版了。与 IE9 和 Firefox 4 发布时的大张旗鼓不同，谷歌几乎没有发出任何声音。公司认为版本号并不重要。真正重要的是定期更新和轻松安装，这样用户就能始终拥有最新的浏览器。我同意。Mozilla 正在采取类似的政策，所以让我们希望微软也能效仿。

如果你用的是 Chrome，可能已经安装了 11 版。要进行检查，请单击工具图标，然后单击“关于谷歌浏览器”。浏览器将自动下载并安装更新。

最引人注目的变化是新的标志。旧的闪亮的 3D 标志已经演变成一个更简单的 2D 替代品:

![Chrome 11](../Images/a496356f197b35e87312e9e71745e672.png)

我想它在低分辨率下更容易辨认，但很难激起人们对新标志的热情。你喜欢吗？你在乎吗？虽然这是一个相当微妙的变化，但我觉得奇怪的是，谷歌花了这么多钱在 Chrome 广告上使用旧标志，却更新了他们的品牌。

GPU 加速应该会让你的浏览体验快一点。Chrome 还没有超过 IE9，但浏览器并不懒散，很少有人会抱怨它的速度。默认情况下，硬件加速的 3D CSS 效果是启用的，因此我们很可能会看到更多展示该技术的精彩的 Chrome 实验。

触摸屏用户还会收到一个为他们的设备设计的新标签页。但是，您可能需要在**关于:标志**设置中启用它。

最后，有一个新的语音输入 API，对我来说，这是最令人兴奋的发展。Chrome 可以解释你的语音，并将其转换为任何输入栏中的文本。这有明显的辅助好处，但我可以看到它被用在任何键盘或鼠标不切实际的环境中。我一直在玩这项技术，所以请关注 SitePoint 上的完整教程*(假设我能让它工作起来！)*
