# 生活中有比论坛更多的东西——至少有人是这么告诉我的

> 原文：<https://www.sitepoint.com/there-is-more-to-life-than-forums-or-so-im-told/>

我这周的帖子真的只给那些没有我的消息就无法度过这一周的人。我喜欢骗自己说你的存在(这对自我有好处)。不同于通常在 [SitePoint 论坛](https://www.sitepoint.com/forums/)上的更新，这篇文章是让你知道从现在开始你将在周一得到你的每周社区综述。从今天起，我将把星期五献给那些选择使用*其他*站点—[脸书](http://www.facebook.com/sitepoint)来关注我们的值得信赖的 SitePoint 社区成员。从现在开始，星期五在这个地方将被称为脸书星期五。

**哇！脸书星期五！多给我讲讲！**
每周五(新西兰时间…)我会为我们的[脸书粉丝](http://www.facebook.com/sitepoint)介绍一个新的交易。这些交易可以是任何东西，从印刷书籍或套装的折扣，到令人敬畏的数字捆绑。更重要的是，每个星期，每个利用那一周交易的人都将参加抽奖，赢得一份奖品。我们有各种各样的东西可以赠送，从签名书到信用卡。

所以，如果你喜欢我们，但你不[‘脸书喜欢’](http://www.facebook.com/sitepoint#)我们，现在是做出改变的时候了！桌上的第一张牌…

对于那些知道除了论坛之外，生活中真的没有什么比 T1 更重要的了的人来说，我会在周一的每周综述中再来看你。
