# 与 Firefox 1.5 相关的事情

> 原文：<https://www.sitepoint.com/firefox-15/>

火狐 1.5 已经发布，所有人都可以下载使用。

毫无疑问，你们中的许多人已经尝试了测试版和发布候选版本。这里列出了使用 Firefox 1.5 要做和要看的事情。

*   使用后退和前进按钮可以看到 Firefox 新的“快速后退”缓存带来的渲染速度的显著提高。
*   如果你安装了一个测试版或发布候选版本，你可以等到 Firefox 使用它的新的增量更新系统进行自我更新。
*   在选项卡式浏览时，用鼠标抓取选项卡，并将其拖动到您想要的顺序。
*   按 Ctrl+Shift+Delete 调出“清除私有数据”窗口，清除历史、缓存、cookies 等私有数据。
*   前往[websitedoesnt-exist.com](http://www.thiswebsitedoesnt-exist.com/)查看新的、不显眼的错误消息。
*   查看“帮助>报告损坏的网站”,看看报告在 Firefox 中不起作用的网站有多容易。
*   在您的 Mac 上安装 Firefox，看看从 Safari 迁移您的个人资料信息有多简单。

如果你仍然渴望使用 Firefox，你可以考虑以下几点:

*   准备好你的网络摄像头，前往 [SpreadFirefox](http://www.spreadfirefox.com/) 参加某种庆祝活动。
*   在 [MozillaZine](http://www.mozillazine.org/) 、 [Asa Dotzler 的博客](http://weblogs.mozillazine.org/asa/)和 MozillaZine 了解火狐相关新闻。
*   通过注册 [MozillaZine](http://forums.mozillazine.org/) 和/或 [Bugzilla](https://bugzilla.mozilla.org/) 账户并测试[后 1.5 版本](http://ftp.mozilla.org/pub/mozilla.org/firefox/nightly/)，参与 [Firefox 2.0](http://www.mozilla.org/projects/firefox/roadmap.html) 的开发。
*   从 SitePoint 的书 [Firefox Secrets](https://www.sitepoint.com/books/firefox1/) 下载免费章节。
*   开始让你的网站火狐友好！

针对您的语言和操作系统，Firefox 1.5 现已推出。
