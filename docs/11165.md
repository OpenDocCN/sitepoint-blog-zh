# 澳大利亚人:4 月/5 月和凯文·杨克一起学习 JavaScript

[![Simply JavaScript product shot](img/d012bbfb9769cbda3f52722fd5457550.png)](https://www.sitepoint.com/premium/library)

抬起头，澳大利亚人！在四月和五月，我将开始在澳大利亚各地教授 [JavaScript 讲习班](http://www.maxdesign.com.au/workshops/)，我希望在我去的时候能看到一些 SitePoint 的人。

如果你不认识我，我是[简单 JavaScript](https://www.sitepoint.com/premium/library) 的合著者和[网站播客](https://www.sitepoint.com/podcast/)的主持人。我还帮助撰写 [SitePoint 技术时报简讯](https://www.sitepoint.com/newsletter/)，并管理将 SitePoint 整合在一起的各色开发人员。

发布于 2007 年的 [Simply JavaScript](https://www.sitepoint.com/premium/library) 不仅因其通过解决现实世界问题来学习 JavaScript 的实用方法，还因其娱乐风格和传达关键概念的高度可视化方法而广受好评。从很多方面来说，这是我最引以为豪的作品，我真的很兴奋能够把这些材料带到路上，并以一种全新的方式呈现出来，为 2010 年做更新和改进！

在旅程的每一站，我都将进行半天的 JavaScript 介绍(针对完全的 JavaScript 新手)，然后是一整天的 JavaScript 深度研讨会(T2)。以下是日期:

*   墨尔本:4 月 12 日-13 日
*   悉尼:4 月 19 日至 20 日
*   堪培拉:5 月 3 日至 4 日
*   **布里斯班**:5 月 10 日-11 日

**[查看研讨会网站](http://www.maxdesign.com.au/workshops/)了解所有细节，**如果你要来，请务必在这里留言让我知道！
