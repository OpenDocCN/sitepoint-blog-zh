# 第二部分:股权与现金

> 原文:[https://www.sitepoint.com/part-ii-equity-vs-cash/](https://www.sitepoint.com/part-ii-equity-vs-cash/)

[https://www.sitepoint.com/blog-post-view.php?id=222721](https://www.sitepoint.com/blog/)

上述博客的三个附录:

1.一位读者问，如果初创公司能够支付，是否可以以贷款的形式免费为初创公司工作(现金加利息支付)。我建议你不要工作，除非你知道你会得到报酬——除非其他因素在起作用，比如你想有机会从一个很酷的客户那里得到推荐，你想做一个真正有趣的项目并提高你的技能，等等。这是确定潜在客户的基本步骤。就个人而言，我从不接受工作，除非我知道我会得到报酬。

2.选项听起来令人兴奋，但在你付诸实践之前毫无意义。除非你把股息或其他股东利益转换成真正的股票，否则你不会得到股息或其他股东利益。与此同时，股票是非常真实的，并伴随着股息(分配给股东的任何现金/收入的一部分)。所以不要对期权太兴奋，直到它们“物有所值”——也就是说，价值超过它们的行使价——并且直到你实际行使它们以获得流动性或成为真正的股东。

3.把股票/期权想象成一张彩票。一笔交易不太可能会有回报，但几笔股票交易的投资组合确实会有回报(如果你选择明智的话)。这就是为什么我想让你了解这种支付形式——这样你就可以评估公司和股权交易，并知道什么时候你面前有一个有吸引力的股权投资机会。
