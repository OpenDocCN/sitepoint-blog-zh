# 你是谁？员工还是个体户？

> 原文:[https://www.sitepoint.com/employee-vs-freelancer/](https://www.sitepoint.com/employee-vs-freelancer/)

当你从雇员变成自由职业者或企业主时，会有很多变化。如果你习惯了雇员/雇主的关系，你可能很难将自己的心态和行为转变为企业家的心态和行为。然而，为了充分利用你的独立地位和避免一些常见的陷阱，认识到这些变化是绝对重要的。

以下是成为一名员工和为自己工作的一些最大区别。
 **当你是个体户的时候……**

*   你决定做什么工作，什么时候做。
*   你向不止一个客户做广告并接受他们的工作。
*   你可以用特定的企业名称经营你的企业。
*   你不能从你的客户那里获得员工福利(如养老金计划、保险、假期工资等。).
*   你有责任支付自己的所得税。
*   你的客户通常不会就如何做好你的工作提供培训或广泛的指导。
*   你通常提供和维护工作所需的所有设备。
*   你可能有自己的员工或分包商。
*   您不受员工评估流程的约束。
*   你独自控制你工作的盈利或亏损机会。
*   你有一份概述工作关系条款的合同。
*   你的合同倾向于一个特定的项目或时间段。
*   你为完成的工作向你的客户开发票。

**区分雇员和独立承包商**

在美国， [IRS](http://www.irs.gov/businesses/small/ ) 倾向于考虑所有的工人雇员，所以从一开始就把自己定位为一个独立工作者是非常重要的。当国税局分析工作关系时，他们使用三类普通法规则来确定工人是雇员还是自雇者:

*   行为控制——客户是否控制你的工作，并指导你如何以及何时完成工作？
*   财务控制——客户是否决定你的报酬，并为你的工作提供必要的工具？
*   关系类型–您是否使用独立承包商协议声明您不会从客户那里获得任何员工福利？

**忽视差异的危险**

当根据上述标准衡量你们的关系时，如果不认识到这些差异并采取行动，会有一些潜在的危险。最严重的后果会影响客户，因为如果国税局认定你被错误归类为独立承包商，客户将面临严重的经济处罚。但是这些惩罚可能会对你产生不良的涓滴效应(例如，如果客户试图通过跟踪你来弥补他们的一些损失)。

如果你被发现是作为雇员而不是独立承包人工作，你也可能会失去一些个体户的税收减免。

除了可能被国税局处以罚款，被当作雇员而不是独立工作者会严重阻碍你的盈利能力，并伤害你的整体独立意识。如果你仅限于与一个客户合作，而不能招揽新的业务，你的业务增长将会受到影响。

一旦你处在一段可疑的关系中，这将是一个难以摆脱的循环。很多时候，建立一个清晰而独立的关系需要教育你的客户，让他们知道与雇员一起工作和与个体经营者一起工作的区别，并坚持确保这些区别得到尊重。

你有没有面对过客户对雇员/自由职业者的困惑？你是如何纠正这种情况的？
