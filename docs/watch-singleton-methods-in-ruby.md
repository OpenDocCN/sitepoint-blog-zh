# 观察:Ruby 中的单例方法

> 原文：<https://www.sitepoint.com/watch-singleton-methods-in-ruby/>

你知道在 Ruby 中你可以为对象定义*个人方法*吗？即只有一个对象会响应的方法。这种方法被称为**单例方法**，了解如何使用它们是值得的。在本视频中，我们将讨论`callbacks`以观察方法变化以及移除和取消定义方法之间的差异。红宝石的*下一个*谜底即将揭晓！
