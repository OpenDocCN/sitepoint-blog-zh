# PHP 案例研究

> 原文:[https://www.sitepoint.com/php-case-studies/](https://www.sitepoint.com/php-case-studies/)

[Marco Tabini](http://blogs.phparch.com/mt/) (来自 [php|arch](http://www.phparch.com) )正在[寻求 php 案例研究](http://blogs.phparch.com/mt/index.php?p=30):PHP 在企业中得到很好应用的故事(无论你认为那是什么意思)。
