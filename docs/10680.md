# 我生活中离不开的商业工具

> 原文:[https://www.sitepoint.com/must-have-business-apps/](https://www.sitepoint.com/must-have-business-apps/)

我在日常工作中使用许多不同的应用程序、工具和服务。如果我把我所有设备上每周使用的所有东西都加起来，我可能会达到 55-60 左右。我之前已经谈论过一些我最喜欢的移动应用和生产力工具，但是这篇文章关注的是我每天在我的主电脑上使用和需要的东西，以便运行我的业务。

以下是我认为必须具备的工具和服务以及应用程序，以便有效地运营我的业务。你可能知道并且自己也在使用其中的很多……或者你会受到启发去尝试一些新的东西。

## 合作

我使用这些工具来分享文档、电子表格、截图等形式的信息。

**[【谷歌文档](http://google.com/docs)**——谷歌提供的免费的基于网络的文字处理器、电子表格、演示、表格和数据存储服务。

**[Skitch](http://skitch.com/)**——Mac 用户免费的截屏、注释和分享工具。

[**YouSendIt**](http://yousendit.com)——一种免费的基于网络的服务，让用户按需发送、接收和跟踪大文件。

## ![](../Images/e158dd4520f87e029d0821ad56222f62.png "tools-2")沟通

这些服务构成了我沟通过程的支柱。

[](http://bluetie.com)**——商务电子邮件托管，具有联系人、日历和文件共享功能。**

 **[**HootSuite**](http://hootsuite.com)——管理多个社交网络的社交媒体仪表盘。

[**环中央**](http://www.ringcentral.com/)——用于商务沟通的虚拟电话和传真系统。

[**【Skype】**](http://skype.com)—一项免费的视频和语音通话、即时消息和文件共享服务。

## 图形/网络/其他媒体

从设计到编码再到创建音频文件，以下是我最喜欢的媒体工具。

[**Audacity**](http://audacity.sourceforge.net/)——一款免费的多平台音频编辑器和录音机。

[**Illustrator**](http://www.adobe.com/products/illustrator/)——Adobe 的矢量图形编辑器。

[**京**](http://www.techsmith.com/jing/)——TechSmith 的截屏、截屏、分享应用。

[](http://www.mozilla.com/firefox/)**(加上相关的[插件](https://www.sitepoint.com/firefox-add-ons-for-productivity/))——Mozilla 的一款免费开源网络浏览器。**

 **[**Photoshop**](http://www.adobe.com/products/photoshop/compare/)——Adobe 的一款图形编辑程序。

[**text wrangler**](http://www.barebones.com/products/textwrangler/)——一款来自 Bare Bones 软件的 Mac OS X 免费文本编辑器。

## 生产力

这些工具帮助我跟踪和管理任务，完成我的标准日常任务，管理数据并保持最新状态。

[**Evernote**](http://evernote.com)——一个免费的笔记应用程序，允许用户收集、排序、标记和注释笔记和其他杂项信息。

[](http://myintervals.com)**——全面的项目管理、任务管理和时间跟踪工具。**

 ****[iWork](http://www.apple.com/iwork/)/[MS Office](http://office.microsoft.com)**——办公生产力套件。我把这些放在一起，因为我每天都在使用 MS Word 和 iWork Numbers，并根据需要在各自的套件中使用一些其他应用程序。
