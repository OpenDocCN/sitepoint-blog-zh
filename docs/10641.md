# 在营销世界中避免类型化

> 原文：<https://www.sitepoint.com/avoiding-typecasting-in-the-marketing-world/>

除了是一位有成就的作家和导演，西尔维斯特·史泰龙还是一位有才华的演员。但是在《洛奇》、《T2》和《兰博》、《T4》电影之后，他被安排出演某个角色。阿诺德·施瓦辛格、金·凯瑞和好莱坞的许多其他演员也是如此。好莱坞并不是唯一一个选择角色的地方。在任何行业，感知往往都能变成现实。潜在客户对你的看法可能会对你接受的工作类型以及找你的客户类型产生持久的影响。被定型总是一件坏事吗？被定型并不一定是坏事。你必须确定你希望你的公司、产品或服务在市场上被如何看待。如果你的客户、潜在客户、同事或合作伙伴对你的看法与你自己的不同，这可能会导致你以一种消极的方式被定型。想象一下，如果你在当地与几家非营利公司合作，他们都联网在一起，参加相同的活动，并且联系良好。迟早，你或你的企业一定会出现在谈话中，可能会引发关于你的公司如何与非营利组织合作的讨论。一个话题引出另一个话题，然后是另一个话题，突然一个从未和你共事过的企业主和另一个从未听说过你的人谈论起你是如何只在非营利行业工作的。这只是一个例子；不是非营利组织，它可能是小企业，或者国家品牌，或者你只提供一个非常小众的服务——当你提供的实际上更多样化。因此，以这种方式被定型会导致你失去新的业务。如何辨别你是否被定型了呢？你经常听到“我不知道你提供这些服务”或“我以为你只做 XYZ”？这是客户或潜在客户对你的看法与你不同的第一个迹象。另一个警告信号是当你经常接到你不提供的服务的电话。你也可以在你提供的*级*服务中被定型。例如，客户可能会说，他们以为你提供了某种服务，如文案，而事实上这不在你的报价之内。另一个被定型的可靠方法是一次又一次地追求相同的工作，或者在你的投资组合中展示相同类型的工作。确保你向潜在客户展示的客户名单和投资组合充分展示了你提供的工作范围。**如何避免被定型**那么，你如何改变人们对你企业的看法呢？它从你的公司内部开始。你需要有明确的界限，比如你提供的服务(和不提供的服务)的轮廓。一旦你知道你的界限是什么，你就可以开始沟通了。下一步是教育你组织内的人。一定要向员工、承包商和合作伙伴解释你提供什么服务，谁是你的目标客户。最后，和你现在的客户谈谈你的服务。你可能会惊讶地发现，你的一些最好的客户对你提供的一些服务一无所知，或者你提供服务的水平。我们最近有一个客户将他们的电子邮件营销外包给另一家公司，仅仅因为他们不知道我们也提供这些服务。他们是伟大的客户。他们只是对我们提供的服务有误解。一定要在你的网站上和你留给潜在客户的任何销售材料上说明你的服务范围。
