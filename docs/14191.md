# 眼球的禅

> 原文:[https://www.sitepoint.com/the-zen-of-the-eyeball/](https://www.sitepoint.com/the-zen-of-the-eyeball/)

这是另一项来自 Eyetools.com 的 Greg Edwards 的有趣研究。

对于那些不熟悉 Eyetools 的人来说，该公司专注于网站可用性研究，这些研究基于用户与网站互动时眼球运动的直接实时跟踪。

正如他们所说的那样，客户通常使用我们的数据来指导他们网站(或购物车、或电子邮件活动、或登录页面)的重新设计，以确保他们希望人们在他们网站上阅读的内容确实被阅读了。

![](../Images/9c1fdfa7a9e71970aa350aa3fe13f88d.png)

这些研究的成果是“热图”。这些图表使用重叠的颜色来展示各种页面元素的相对吸引力——通常鲜红色用于吸引人的高“眼球流量”区域，黄色用于中等流量区域，绿色和蓝色用于吸引力逐渐降低的内容。确实是非常有益的东西。你可以推测用户在谚语所说的牛回家之前可能会做什么，也可能不会做什么，但是很难与一群自由转动的眼球争论。

我认为 Eyetools 研究给我印象最深的是用户之间结果的一致性。看起来，虽然更有经验的用户肯定更有可能以更快的速度和更大的信心浏览给定的页面，但他们通常还是会遵循同样的老路。

这个最近的测试案例特别酷的一点是，他们选择了一个我们(几乎)都非常熟悉的地点——CSSZenGarden.com。这里我们有一种不寻常的情况，可以看到用户以完全不同的方式与不同的布局交互，即使我们知道内容和标记是相同的。

除了更容易预测的结论——例如，用户倾向于首先浏览标题，并且“在文件夹上方”更好——这个例子似乎表明，当导航元素放在右边时，用户对它们的关注明显减少。当然，颜色、大小和其他元素在这个例子中发挥了作用，但我仍然认为它很有启发性。
