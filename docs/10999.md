# 谁拥有这个图像？通过 PicScout 图像交换找到答案

> 原文:[https://www . site point . com/who-owners-this-image-find-out-with-pics cout-image-exchange/](https://www.sitepoint.com/who-owns-this-image-find-out-with-picscout-image-exchange/)

许多设计师通过谷歌找到了漂亮的图片，他们很想把这些图片用在他们的设计中，但是你不能从网上拿走任何你想要的图片。那么，你怎样才能找到你偶然发现的那张漂亮照片的主人呢？答案可能在于 PicScout [图像交换](http://www.picscout.com/products-services/imageexchange.html)。

目前处于测试阶段，Image Exchange 帮助创意专业人员定位网络上图像的来源和版权信息。这是一个运行在 Firefox 中的免费应用程序，允许您:

*   无论何时何地，只要看到完美的图像，就能确定图像的所有者
*   无需搜索即可查看您想要使用的图像的所有许可选项
*   在看到图像后的几分钟内连接、许可和使用图像
*   发现关于图像的有用或有趣的信息——所有权和分配

为了测试它，在 Firefox 上安装插件后，我对“飞机”进行了图片搜索下面你可以看到缩略图结果。

[![ImageThumbnails](../Images/6cda2028099cbab43f8ef6df0991dc05.png "ImageThumbnails")T2】](https://www.sitepoint.com/wp-content/uploads/2010/05/ImageThumbnails.jpg)

其中三个缩略图显示有蓝色的小“I”图标，表示图像附有所有权信息。当你点击“我”的时候，一个新的窗口就会打开，里面有更多的信息和图片所在网站的链接。

[![Moreinfo](../Images/f607e813eaa6c2f4655ddc8ef42a386f.png "Moreinfo")T2】](https://www.sitepoint.com/wp-content/uploads/2010/05/Moreinfo.jpg)

图像交换扩展可以识别网络上任何地方的照片，而不仅仅是在谷歌图像搜索结果中。如果 PicScout 认为这张照片是 CC BY-NC 许可的，并且来自 Flickr，它会将你指向照片在 Flickr 上的原始页面。在上面的例子中，它识别出图片是免版税许可的，并在一个微型股票网站上包含了图片来源的链接。

图像交换还处于早期阶段，在图像搜索中返回的许多缩略图上缺乏版权/所有权信息。该扩展从 PicScout ImageIRC 中索引的数百万张在线图像中提取信息，pics cout image IRC 是世界上最大的图像指纹和已知版权所有者元数据索引。随着这个数据库的扩大，这将成为一个非常有用的工具。

如果你经常在网上搜索新的图像用于你的设计工作，并希望更多地参与这个项目，你可以注册 [PicScout 的图像交换测试程序](http://imagex.picscout.com/Download/Register)。
