# Photoshop 中的饱和度混合模式

> 原文：<https://www.sitepoint.com/saturation-blend-mode-in-photoshop/>

我们在 Photoshop 混合模式系列的最后阶段(天哪，这意味着我必须想出更多的话题来写！)，而今天的博客看的是饱和度混合模式。

回想一下色调混合模式是如何保持基础图层上的所有内容不变，但是应用了混合图层中的色调(颜色)。饱和度模式类似，除了应用混合层的饱和度之外，基本层上的所有内容都保持不变。

试试这个例子:从我的樱花图片开始—

![](img/9c78cc22d0d345ed4a8c77e34140f3c4.png)

添加这个混合层，它有一个高度饱和的蓝色和一个不是很饱和的蓝色:

![](img/a236a9c2bd6c9cabb03a1b6039622153.png)

将混合层设置为饱和模式会产生以下效果:

![](img/2a8862dd6b4f052ccdd0bfdd2426ed54.png)

正如预期的那样，上半部分(浅的、不太饱和的蓝色)产生了不太饱和的花图像，而下半部分则非常饱和。

现在是实际应用。我将从我的标准玫瑰图像开始，然后添加一个径向渐变来产生这个混合层，使用一个不太饱和的颜色:

![](img/d261ab1f6de8a11b67aa5e39730e120a.png)

(为此，我必须通过点击顶部工具栏中的渐变预览来打开渐变编辑器，从前景到透明渐变创建一个新的渐变，然后翻转定义渐变的滑块来创建一个透明到前景的渐变。)

![](img/740ac611cd1eb9742bb9d143602cc5da.png)

合并径向渐变混合层(设置为饱和模式)的结果是这样的图像:

![](img/99c46aec2dd6cc62faaad2ac8709a647.png)

这让我想起了一幅“给某物注入生命”的图像，也许代表了某种“内在生命”或“内在光芒”我相信你能为这种技术找到自己的用处…
