# 打破创意障碍的六个方法

> 原文:[https://www.sitepoint.com/breaking-creative-block/](https://www.sitepoint.com/breaking-creative-block/)

你有了一个新的客户，你很兴奋，渴望去。你拿起铅笔和画板，或者轻弹电脑，打开 Photoshop，然后……一片空白。空白的纸，空白的屏幕。几杯咖啡和几块巧克力之后，场景并没有太大的变化。

我们大多数人都想创造一个伟大的设计，一个让我们和客户都感到自豪的设计。作为有创造力的人，我们不想复制我们上一个“伟大的设计”压力开始积聚，你开始怀疑自己是否有能力胜任这份工作。你发现自己漫无目的地长时间浏览 CSS 画廊，但没有什么能激发你的灵感。你怎样才能翻过这座山？

我没有对每个人都有效的确定答案，但我确实有一些来自我自己的经验的建议，与其他设计师和书籍交谈(在互联网出现之前有一种东西叫做书籍)。假设你已经做了一些计划，并且已经弄清楚了你的客户想要什么和需要什么，以及竞争对手在做什么，那么是时候去做一些创造性的事情了。

1.  把你的想法写在纸上。
    人们面临的第一个障碍是试图把一些东西，任何东西，从他们的脑袋里取出来，放到那张纸上。我向我的学生推荐，无论他们多么热爱 Photoshop、Illustrator 或任何他们用其设计的程序，他们总是从纸开始。你可以在 5 分钟内把 5 个想法写在纸上。在那段时间里，你能在 Illustrator 中拼凑出多少创意？不会画也没关系，随便乱涂，素描，涂鸦，写想法，想往哪个方向走都行。提炼你的想法。思维导图在这方面非常有用。
2.  **从粗线条开始。**
    当你在设计一个网站的时候，没有必要把你的字体作为出发点。像这样的细节可以以后再补充。考虑大局。网站需要实现什么？网站上会有多少部分？开始凿掉这些石块，你会发现道路变得更加平坦。
3.  与其他人交谈。
    如果你独自工作，创意街区的世界可能会是一个孤独的地方。像 SitePoint 这样的论坛，挤满了乐于助人的人。问个问题。我也发现 Twitter 非常有价值。我通过 Twitter 建立了联系，否则我永远也不会建立这样的联系。我可以给一些人发信息，问一些问题，或者让他们看看我正在做的东西。
4.  **切换项目。**
    当你试图开始一件新的事情时，这可能会适得其反，但是如果你已经花了很长时间困在一个洞里，你需要摆脱它，获得一个新的视角。从事和思考一些完全不同的事情有时足以打破僵局。
5.  你可能只是觉得无聊。
    如果你发现自己一遍又一遍地从事同一类型的设计，当然你可能不愿意再做同样的事情。这就是个人项目重要的地方。在你自己的小工作上给你的创造力充电可以让你有动力为别人工作。我设计 t 恤，拍摄照片，毫无压力地满足我的创作需求。
6.  继续前进。
    说真的，有什么替代方案？你不会让一个空白的屏幕阻止你创造下一个伟大的设计，是吗？

你们是如何克服创意障碍的？我很想听听。
