# Firefox 4 Beta 11 中添加了“不要跟踪”首选项

> 原文:[https://www . site point . com/do-not-track-preference-added-to-Firefox-4-beta-11/](https://www.sitepoint.com/do-not-track-preference-added-to-firefox-4-beta-11/)

Mozilla 刚刚推出了即将到来的火狐 4.0 版本的 beta 11。这个版本主要包括错误修复和性能改进，但是一个特别有趣的新特性是引入了“不要跟踪”偏好。在 Firefox 偏好设置窗口的“高级”标签中，这是一个简单的复选框，用于通知您浏览的网站您不希望被“跟踪”这个选项看起来是这样的:

![](../Images/9d0f46628329ff01e425c1eb1c56c7e8.png)

在线行为跟踪是一种越来越流行的技术，网站、社交网络和广告商利用会话、cookies 或其他[甚至更隐蔽的方法](https://www.sitepoint.com/no-cookies-or-javascript-no-worries-you-can-be-tracked-anyway/)来跟踪互联网用户的点击、搜索和浏览模式。许多人认为跟踪是对隐私的侵犯，因为它通常在最终用户不知情的情况下发生。例如，电子前沿基金会有一个专门解释这种做法并概述潜在危险的[页面](https://www.eff.org/issues/online-behavioral-tracking)，还在 YouTube 上发布了一个[视频来传播对这一问题的认识。](http://www.youtube.com/watch?v=aA5hzeEX9cI)

也就是说，许多广告商从跟踪中获得了巨大的利益，因为这使他们能够根据网站访问者的浏览历史向他们提供超针对性的广告。

那么 Firefox 中的这个新特性实际上是做什么的呢？在幕后，当复选框被选中时，Firefox 简单地发送一个附加的“DNT”HTTP 头和所有请求。透过 Firebug 的网板镜头，看起来是这样的:

![](../Images/490c7d1c564644bf7f3389dc2de4588a.png) **火狐的新 DNT 头球**

当然，这个额外的标题没有任何作用，除非网站选择接受它。看到 Mozilla 这样做有点尴尬；如果他们在谷歌或脸书这样的大公司的支持下宣布这一功能，并同意尊重新的设置，这可能会是一个更大的炸弹。

然而，像这样一个功能的发布仍然有可能产生一些影响。一方面，它可以提高更多 Firefox 用户对跟踪问题的认识，当他们得知他们访问的网站忽略了他们不被跟踪的偏好时，他们可能会不高兴(尽管目前还不清楚他们会如何了解这一点)。但是更重要的是，美国联邦贸易委员会(美国联邦贸易委员会)正在考虑立法使这种设置成为法律要求。“在野外”对优惠设置进行现有的技术实施，可能会使立法者更有力地论证这种措施的可行性。

最后，我认为谷歌在这个标准的发展中可以扮演一个有趣的角色，作为一个大玩家，它既是在线广告商又是浏览器供应商。如果这个功能是 Firefox 4 的一个卖点，那么 Chrome 也将面临支持它的压力。但是，如果谷歌的网站和广告不支持标题的话，它不可能真的在浏览器中添加“请勿追踪”功能(至少不会显得很虚伪)。

无论发生什么，我认为 Mozilla 值得被称赞，因为它朝着正确的方向迈出了一步。在接下来的几个月(或许几年)里，观察这一过程肯定会很有趣。

你对“请勿追踪”标题有什么看法？它有腿吗？还是没有广告主支持的 DOA？
