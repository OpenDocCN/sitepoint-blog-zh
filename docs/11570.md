# 脚本字体

> 原文:[https://www.sitepoint.com/the-script-typeface/](https://www.sitepoint.com/the-script-typeface/)

继续我们的字体分类系列，今天我们来看看字体。属于脚本类别的字体是那些具有用毛笔、书法笔或铅笔手写字体外观的字体。它们可以进一步分为看起来像传统书法的非常优雅或正式的风格，以及更粗糙和随意的风格。

许多正式的文字都是以三峡的字体为基础的。乔治·比克姆、乔治·斯内尔和乔治·谢莉是十七和十八世纪的书法家，他们用羽毛笔或金属笔尖写字。

根据乔治·比克姆的版画创作的比克姆手稿。

![Bickham](../Images/19abbd9e4ecba33b7f6dd92cf4e2e487.png "Bickham")

Snell Roundhand ，是由[马修·卡特](http://www.aiga.org/content.cfm/medalist-matthewcarter)根据乔治·斯内尔的作品创作的当代字体。

![Snell](../Images/763a4ec901da9947a08b0d41595a0d95.png "Snell")

[英文字体](http://www.dafont.com/english.font?psize=l)，基于乔治·谢莉的作品。

![English](../Images/bb4f4fe663d20aa390efd3536051bb74.png "English")

有数千种正式的字体可供选择，包括免费的和商业的。它们被大量用于邀请函、卷轴和需要优雅排版的场合。它们不适合大量的正文，但是用在标题和徽标中时，大尺寸看起来非常漂亮。

使用正式字体时要记住的最重要的一条规则是，永远不要把字体全部设为大写。**曾经**。当所有的字母都是大写时，它们变得几乎无法阅读。另一件要记住的事情是少用它们。

 **散体字** 散体字看起来比较规整，不太正式。它们仍然可以有不同宽度的笔画，但看起来不像正式的笔迹那样复杂。一些例子是

同样的规则也适用于临时脚本字体。尽量少用，主要用于标题或很短的文本。
