# 海盗湾会沉吗？

> 原文:[https://www.sitepoint.com/will-the-pirate-bay-sink/](https://www.sitepoint.com/will-the-pirate-bay-sink/)

在自 Napster 以来最大的案件中，瑞典法院的一项里程碑式的裁决认定海盗湾犯有侵犯版权罪，因为它使 33 个受保护的文件可以被非法共享。

弗雷德里克·内伊、哥特弗里德·斯瓦托姆·沃格、卡尔·伦德斯特伦和彼得·桑德的辩护律师认为，他们应该被无罪释放，因为海盗湾没有任何文件，只是提供了一个 bit torrent 搜索引擎。法院判决所有四名被告入狱一年，罚款 400 万美元；这笔钱将由 21 世纪福克斯、米高梅、哥伦比亚电影公司、华纳兄弟、索尼音乐娱乐公司和百代唱片公司分享。

尽管这个奖项没有达到唱片业宣称的 1750 万美元，但国际唱片业联盟(IFPI)主席约翰·肯尼迪说:

> 有一种观点认为盗版是可以的，音乐产业应该接受盗版。这个判决将会改变这一切。

海盗湾创始人彼得·桑德表示:

> 我们被定罪是很奇怪的，更奇怪的是我们是作为一个团队被定罪的。法庭说我们有组织。我早上没法把 Gottfrid 从床上叫起来。如果你要判我们有罪，那就判我们无组织犯罪。
> 
> 我们付不起也不会付。即使我有钱，我也宁愿烧掉我所有的东西，甚至不会把骨灰给他们。

海盗湾的创始人将对该裁决提出上诉，该网站仍然活跃，主页上有一条不同寻常的信息*“别担心，我们来自互联网。一切都会好的。:-)"* 。瑞典当局没有法律权力关闭该网站，因为它托管在世界各地的数据中心。

媒体公司不太可能会收到钱，但他们一直坚持认为这是关于意识和教育的案件。那么它会阻止网络盗版吗？

海盗湾可能是最著名的种子追踪网站，这个案件给他们带来了更多的宣传。具有讽刺意味的是，它可能会带来关于 bit-torrent 技术的进一步“认识和教育”(大多数媒体都在出版有用的指南！)这一裁决是否也意味着其他搜索引擎，如谷歌，因链接到托管或推广非法内容的网站而被判侵犯版权？

电影和音乐行业的庆祝活动很可能是短暂的。在现实中，有许多种子搜索引擎。或许媒体公司应该采用现代数字发行方式，而不是徒劳地试图屏蔽个人频道？
