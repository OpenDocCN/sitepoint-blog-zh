# SxSW 关心

> 原文：<https://www.sitepoint.com/sxsw-cares/>

[![](img/e64d01553c74b8f070fd0155bd5d2160.png)](https://www.sxswcares.org) 在像西南偏南互动这样的会议上有很多伟大的想法，其中一些甚至在会议结束前就已经实现了。今年最好的想法之一无疑是 [SxSW 关心](https://www.sxswcares.org)。

当我们都在往西南偏南的时候，我们听到了日本地震和海啸的消息。SxSW Cares 在一天之内成立，向世界展示这里的人们关心帮助日本恢复。想法很简单——创建一个基本的捐赠页面，并在西南偏南地区传播我们需要支持的消息。

周五早上，CauseVox 的创始人罗布·吴(Rob Wu)在新闻上看到了日本海啸，他想做点什么来帮助日本。他建立了一个简单的网站，并与利·邓肯-德斯特、杰斯·林和黛比·吴分享。这四个人在这个项目上进行合作，与红十字会合作，使网站正常运行，捐款源源不断。

自上线以来，该活动已为日本救灾工作筹集了超过 35，000 美元(在撰写本文时)。CauseVox 支付所有的交易费用，所以所有的捐款将全部捐给红十字会。

[CauseVox](https://www.causevox.com) 是一个在线筹款解决方案，允许非营利组织轻松快速地发起定制的筹款活动。
