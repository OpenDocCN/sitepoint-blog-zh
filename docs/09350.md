# 如何获得报酬

> 原文：<https://www.sitepoint.com/how-to-get-paid/>

这听起来很容易，但有时获得报酬就像拔牙一样。这些是每次按时获得发票金额的基本要素。

## 1.在项目估算中概述你的付款条件

从一开始就向客户表明你对开具发票是认真的。在你的估价中列出你的付款条件，包括:

1.  你什么时候开发票
2.  不管你是提前开发票还是延期开发票
3.  他们需要支付多长时间
4.  您喜欢的付款方式

当您的联系人批准估价时，询问他们是否对发票条款感到满意，以及他们是否有任何问题。如果他们有自己怪异的流程或需求，这将会给他们一个黄金机会让你了解他们。

## 2.说开就开发票

如果你说你会在每个月初开发票，那就照做。把时间放在一边，把那些发票寄出去。将它们与其他信息分开发送:让你的发票邮件(或信件)只与发票有关。

对于每一个问题，邀请你的客户联系你，如果他们有任何问题。表达你对讨论发票的开放态度，如果你的客户确实有问题，他们会感觉更舒服。

## 3.检查您的发票收据

如果您在发送发票后的几天内没有收到回复，请联系您的联系人以确保他们收到了发票。不要忽视这一步:只管去做。

## 4.等待付款

无论你是通过银行支票还是电子转账获得报酬，都要留意付款方式。

如果你已经完成了第 1 步到第 3 步，你应该得到报酬，而且在任何情况下都应该按时得到。万岁！

但是，如果由于某种原因，你没有在到期日收到付款，请在到期日过后尽快联系你的客户。就像在下一个工作日。这表明你不会轻视报酬问题。如果客户希望在未来支付更多的发票，你的努力将鼓励他们从现在开始立即付款。

## 5.跟踪逾期付款

不要紧张:给客户发一封礼貌的邮件就行了。重新附上你的发票，说明它已经过期，最重要的是，询问你什么时候能收到付款。

不要忽略最后一点:询问你何时会收到付款。不要对此模棱两可。坦率地表明你认为这是你工作关系中的一个重要方面。

不要将此作为一个附加信息发送到关于另一件事的电子邮件中，这可能会被忽略。当然，在付款之前，不要为这个客户完成任何工作。

## 6.进一步跟进

 *如果你还是收不到钱，那就该把事情做得更好了。打电话给客户，亲自询问发票的事情。如果有必要，考虑一家托收公司。

## 发票不要

1.  不要继续为一个没有付钱给你的客户工作。
2.  在你的发票邮件中，不要表现得甜美、迷人、可爱或不那么担心。做你自己就好:一个自由职业者，已经完成任务，现在需要报酬。
3.  不要让后续邮件被解读。如果你在跟进一张过期的发票，不要只是说，“嘿，鲍勃，我只是想知道最后一张发票。如果你能回复我，那就太好了。”那到底是什么意思？不管是什么，对我来说都是微不足道的！
4.  不要忘记说明报酬对你有多重要，不管它有多小。
5.  不要认为我们的愤怒会让你得到回报。不会的。专业是你获得报酬的原因。
