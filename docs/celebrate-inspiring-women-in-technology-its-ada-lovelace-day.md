# 庆祝鼓舞人心的科技界女性——今天是阿达·洛芙莱斯日！

> 原文：<https://www.sitepoint.com/celebrate-inspiring-women-in-technology-its-ada-lovelace-day/>

每隔几个月，似乎都会有另一场关于为什么我们在技术领域很少见到女性的争论。我认为这很大程度上是因为我们不像听到男性榜样那样经常听到女性榜样。每一个你在会议上看到发言的女性，就有几个男性——事实上，在一些会议上，你几乎看不到任何女性发言人。作为一名 IT 界的女性，我亲眼目睹了这一点——我经常是房间里唯一的技术女孩，而我职业生涯中大多数更资深的人都是男性。很艰难！

好消息是，今天是庆祝女性科技榜样的日子。3 月 24 日是阿达·洛芙莱斯日，这是一个在极客圈凸显你最喜欢的女性的机会。[阿达·洛芙莱斯](http://en.wikipedia.org/wiki/Ada_Lovelace)被广泛认为是第一个计算机程序员——不仅仅是第一个女性计算机程序员，而且是第一个在*所有*的程序员——她极大地提醒了人们，很多很多年来，女性一直是这个行业的重要组成部分。

那么，如何参与呢？你可以写博客，制作视频或播客，发微博——事实上任何你可以放到网上的东西都是受欢迎的。当你完成后，[把它添加到列表中](http://ada.pint.org.uk/)——已经有越来越多的人发表了关于激励极客女性的帖子。

这里有三个鼓舞人心的极客女性，对我来说，她们是伟大的榜样。

丹娜·博伊德

早在大多数人听说“社交网络”这个词之前，danah boyd (是的，小写的使用是[故意的](http://www.danah.org/name.html))就一直在思考、写作和谈论我们如何在网上互动 danah 是一个著名的演讲者，一个多产的博客作者，目前是新英格兰—[微软研究小组的一员。](http://research.microsoft.com/en-us/news/features/boyd-032009.aspx)丹娜口若悬河，激情四射，神魂颠倒。如果你还没有读她的博客，并且对社交网络有一点兴趣，那你就错过了。

苏珊·卡雷

Susan Kare 是一名平面设计师，参与了第一台麦金塔电脑的设计。她在 Mac 的字体、图标和界面方面的工作帮助 Mac 在用户界面设计方面取得了突破性的成就。正是因为苏珊的工作，苹果电脑才变得如此有趣、友好和个性化。她后来参与了各种项目，比如 OS/2 和 Windows 的图标和界面，脸书的礼物图标，手持应用程序，甚至还有 [Chumby](http://chumby.com/) 。她是界面设计领域的先驱——如果你对网页设计或用户体验感兴趣，你可以从 Susan Kare 那里学到很多东西。

**莫莉木刻**

在标记大师和 CSS 朋友中， [Molly](http://molly.com/) 是不需要姓氏的人之一，就像 Sting、Bono 或 Madonna 一样。Molly 是一名顾问、培训师、演讲者、作家，在宣传 Web 标准方面，她是一名全面的摇滚明星。莫莉的写作、博客和演讲多年来一直与此相关，我相信它们还会保持一段时间。现在，她作为一名网络传播者与 Opera 的团队一起工作，传播关于网络标准的好消息。

你呢？你最喜欢的女性极客榜样是谁？
