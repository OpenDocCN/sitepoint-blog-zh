# 介绍雅虎出版商网络

> 原文：<https://www.sitepoint.com/introducing-yahoo-publisher-network/>

这已经不是什么新闻了。不久前，我决定我在这个博客上的第一篇文章将是关于雅虎的新出版商网络。不幸的是，我花了比预期更长的时间来建立和运行这个博客。我敢肯定，到现在为止，你们中的许多人已经对 YPN 很熟悉了，但我想我还是会继续下去，因为很难找到比雅虎进入广告网络游戏更大的网络出版新闻故事了。

雅虎出版商网络，发现[这里](http://publisher.yahoo.com/)  [谷歌 Adsense](https://www.sitepoint.com/article/introducing-google-adsense) ，那么你就很清楚 YPN 会带来什么。像谷歌一样，YPN 分析你的内容，然后根据你的内容提供适当的广告，或者这就是它的计划。事实上，我发现 YPN 的目标是糟糕的。当我运行我的测试时，提供的广告几乎与我的内容无关，有些是完全不相关的 MLM 广告(想想金字塔计划)。

所有这一切的好处是，雅虎的每次点击支付率远远优于谷歌。这可能有几个原因，例如，谷歌允许广告商向 Adsense 出版商支付比你在搜索结果页面上找到的正常 AdWords 位置更低的费用，雅虎还没有这样做。此外，谷歌的 Adwords 计划比雅虎的搜索营销计划(以前称为 Overture)的最低每次点击成本更低。

尽管最终你得到的是更少的点击，但每次点击会得到更多的钱，给你留下的是来自不太受欢迎的广告的相同收入。

我可以预见，如果你有一个点击率很高的广告单元，比如一个包含内容的矩形广告，你可能会从 YPN 获得比 Adsense 更多的收入，仅仅是因为位置的优势。此外，雅虎允许每页最多 4 个单元，而谷歌只允许 3 个，所以如果你真的有 4 个单元的空间，你也许可以通过转向雅虎来增加你的收入。我应该注意到，你不能做的是每页显示 3 个谷歌广告，然后再显示 1-4 个雅虎广告，因为你违反了网络条款。你必须选择一个或另一个，不能在同一个页面显示谷歌广告和雅虎广告。

雅虎还有一个谷歌没有的功能，叫做广告定位。这个特性允许你对你的网站进行分类，希望能提高广告显示的相关性。不幸的是，在我的测试中，它似乎不起作用，而且你可以选择的类别也不是那么全面，它们都很普通。例如，我记得没有“网络开发”的分类，只有一个“互联网”谷歌未公开的名为[谷歌提示](https://www.sitepoint.com/forums/showthread.php?t=249615)的私人功能效果要好得多。

除了已经提到的问题，这项服务或多或少是谷歌 Adsense 的克隆。你有很多单元尺寸的选择，可以很容易地定制颜色。付款条件也大致相同。

总的来说，至少花点时间测试一下雅虎，看看它是否比谷歌赚得更多，这是值得的。此外，如果雅虎能够提高他们的针对性，同时保持他们的支出，我认为许多人最终会从 Adsense 叛逃到 YPN。
