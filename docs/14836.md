# 通过推荐建立你的业务

> 原文:[https://www.sitepoint.com/build-business-referrals/](https://www.sitepoint.com/build-business-referrals/)

DJ Dart 和 Kelly Norris 在 1999 年创办了他们的网页设计公司 Verrius，当时他们用信用卡支付所有的电脑设备费用。与大多数新公司一样，他们开始时没有一个可以建立业务并产生经常性收入流的客户基础。

与其他公司不同，Verrius 选择不寄信、不打陌生电话或不参加社交活动来会见客户。相反，他们专注于与现有公司建立伙伴关系，然后这些公司向他们的客户提供 Verrius 的网络开发和营销服务。

“这些类型的企业通常在它们提供的整套服务中至少缺少一个重要元素。Dart 指出:“这些公司的声誉依赖于提供完整的一揽子服务，Verrius 为他们提供任何必要的服务，而他们的客户从来不知道我们已经签约介入。”

“例如，我们目前合作的一家软件开发公司缺少一名内部图形设计师。我们向他们提供我们的服务，软件公司从看起来他们有员工提供完整的服务菜单中获益。客户受益于从无到有只与一个联系人打交道，甚至不知道我们在这个过程中起了作用。这是一个对所有参与者都有利的安排。”

##### 寻找合作伙伴

当 Verrius 开始进入 Web 开发行业时，它通过给他们发电子邮件和打电话来寻找合作伙伴。“这有点像数字游戏，只吸引了我们联系的一小部分公司加入我们的推荐网络。但是坚持占了上风，随着合作伙伴/客户数量的增长，我们的工作量和业务也在增长，”Dart 说。

事实上，这一策略非常成功，以至于韦里厄斯没有做任何广告。相反，他们专注于培养合作伙伴关系，这反过来又促进了他们向合作伙伴客户转售服务——否则他们永远也不会接触到这些客户。

##### 为合作关系设定基本规则

根据合作公司的不同，Verrius 建立了两种类型的合作关系:

***1。转售服务。*T3】**

“通过这些交易，我们与合作伙伴密切合作，并确定项目成本。当项目完成时，我们向合作伙伴开账单，合作伙伴反过来向客户开账单，确保向项目添加百分比或金额。这使得合作伙伴可以从项目中获得收入，”Dart 说。

***2。推荐。*T3】**

Verrius 从其合作伙伴那里获得的每一个推荐都会得到一笔佣金。佣金的具体数额取决于项目的大小和他们与合作伙伴签订的合同。

Dart 说，Verrius 更喜欢做经销商交易，特别是与那些他们已经打交道很长时间的公司。倒卖不仅让 Verrius 的合伙人赚更多的钱，而且项目的周转通常也快得多，因为 Verrius 和合伙人都已经熟悉了彼此的工作风格。

##### 与客户合作

当 Verrius 赢得一个项目时会发生什么？

“通常，客户/合作伙伴仍然是主要的联系来源，我们在幕后工作。根据合同，合作伙伴可能会透露我们，或者他们可能希望隐藏我们，让我们看起来像是他们自己公司的员工。

DJ Dart 说:“这让客户的业务看起来更大，如果他们愿意，可以在我们的基础上增加费用。

##### 结果呢

3 个月后，这项业务开始盈利，并一直保持盈利状态。

目前，Verrius 雇佣了五名全职员工:营销人员、程序员、网页设计师、图形设计师和文案。事实上，生意非常兴隆，以至于他们不止一次需要从自由职业者那里获得额外的帮助。

##### 你能学到什么

没有必要直接拥有客户来赚取丰厚的收入。如果你有其他企业缺乏的专业技能或优秀团队，你可以与他们合作，让他们转售你的服务。

接近软件开发公司，网络营销公司，广告公司，甚至其他网页设计公司。你可以让他们为客户提供全套服务，而工作外包给你。他们赚取利润，他们控制客户，你不用出去“推销”就能获得业务。

和伴侣的关系就是一切。DJ Dart 说“我们与合作伙伴一起工作很开心，我们通常会与他们成为朋友”——这当然意味着 Verrius 需要更多的工作和推荐！
