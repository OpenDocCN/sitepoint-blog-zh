# 自由职业者的非虚拟世界:面对面交流

> 原文:[https://www.sitepoint.com/face-to-face-networking/](https://www.sitepoint.com/face-to-face-networking/)

在我的上一篇文章中，我写了关于[面对面客户会议](https://www.sitepoint.com/successful-in-person-meetings/)和一些你需要考虑的事情，以便有一个成功的本地会议。如果你正在探索非虚拟业务，并希望更多地在网下工作，面对面交流是建立联系和寻找新机会的好方法。

**在哪里寻找社交机会**

我们都知道网上有很多地方，你可以在那里建立关系网，拓展你的职业人脉。你也可以在网下的很多地方建立关系网，尽管这需要更多的研究和探索来找到最适合你的机会。

您可能想探索的一些流行的社交机会包括:

*   会议
*   专业组织会议
*   社区服务活动
*   推特
*   当地商业网络团体

你也可以计划和组织自己的活动。它可以简单到与你在网上认识的人进行一对一的会面(一定要遵循我之前帖子中的安全指南)。你也可以通过像 [MeetingWave](http://www.meetingwave.com/) 和 [Meetup](http://www.meetup.com) 这样的网站加入群组会议。

**面对面交流技巧**

就像网上交流一样，你应该带着明确的目标进入一个真实的交流环境。有兴趣认识志同道合的同事吗？你在寻找潜在的客户吗？你对结识新朋友和产生新想法感兴趣吗？你为什么在那里？

花时间为活动设定目标。通过关注与他人的联系，而不是自动销售，你将减轻这些活动可能带来的一些压力，并使自己在这个过程中更加平易近人。

准备好电梯演讲，准备好回答关于你的业务和工作的常见问题。你甚至可以提前练习，这样你在分享信息时会感到放松和舒适。别忘了带名片给你遇到的每个人。

这里有更多的技巧，让你的面对面交流尽可能成功。

*   穿着专业得体。
*   愿意接近别人并介绍自己。
*   和每个人握手和眼神交流。
*   问一些开放式的问题，一定要听听别人在说什么。
*   避免在活动期间呆在一个地方；为了最大的影响而交往。
*   真诚和诚实。
*   事后跟进，坚持到底。

你亲自交际吗？你能给别人提供什么建议？
