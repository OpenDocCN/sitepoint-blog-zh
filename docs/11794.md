# 雅虎的开发项目会变成什么样？

> 原文：<https://www.sitepoint.com/yahoo-development-microsoft-deal/>

尽管雅虎最近出现了盈利问题，但该公司仍继续投资几个创新和技术惊人的项目。其中包括:

[**SearchMonkey**](http://developer.yahoo.com/searchmonkey/) :一个允许网站所有者发布结构化数据的系统，这些数据在雅虎搜索引擎中提供更有用和视觉上更吸引人的结果。

[**老板**(建立自己的搜索服务)](http://developer.yahoo.com/search/boss/):一个允许产品利用雅虎搜索索引的平台。

[**【YUI】**(雅虎用户界面库)](http://developer.yahoo.com/yui/):一个雄心勃勃的 JavaScript 库，用于构建丰富的交互式 web 应用程序。

[**【YQL】**(雅虎查询语言)](http://developer.yahoo.com/yql/):一种类似 SQL 的语言，它将整个网络抽象化，使其看起来像一个巨大的数据库！

[**Yahoo Pipes**](http://pipes.yahoo.com/pipes/) :可视化内容混搭系统。

雅虎免费提供这项服务，许多公司已经在自己的网站和系统中采用了这项技术。然而，如果微软和雅虎的交易按计划进行，这些项目会怎么样呢？根据协议条款，微软将成为两家公司的唯一技术系统开发商。

这笔交易对 SearchMonkey 和 BOSS 有直接影响，但其他项目几乎肯定会受到影响。雅虎已经宣布，使用必应将为他们节省大约 2 亿美元的年度开支。我敢肯定，这个数字包括管理和营销成本，但它也将支付几千名开发人员的工资。微软有可能将雅虎的系统与必应整合，但这样做有合理的财务或竞争优势吗？

根据雅虎开发者网络的负责人 Chris Yeh 和 BOSS 团队的 Ashim Chhabra 的博客，不会有立即的变化，他们仍在考虑交易的影响。然而，有传言称雅虎的士气正在下降。

微软和雅虎的交易是否意味着雅虎开发项目的终结？前景并不乐观。
