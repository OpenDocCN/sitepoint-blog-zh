## Pivot.js

目前为止我见过的最好的 jQuery 表格插件！提供基于字段名的动态过滤、添加/删除字段、分页、快速实时搜索过滤和标准排序。优秀的插件。比所有这些 [30 个 jQuery 表插件](https://www.jquery4u.com/plugins/30-amazing-jquery-tables/)都要好。**警告:CHROME 上的 ABIT 小车。**
