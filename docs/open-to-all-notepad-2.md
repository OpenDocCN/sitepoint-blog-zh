# 向所有人开放:记事本 2

> 原文：<https://www.sitepoint.com/open-to-all-notepad-2/>

我打算在本周和下周为那些想使用。但是买不起或者没有机会使用 Visual Studio。

它始于 webref.info 上的上一篇文章，因为我在。NET 框架类。现在看起来你们很多人都在寻找帮助 ASP.NET 编码的工具。

今天的瑰宝是[记事本 2](http://www.flos-freeware.ch/notepad2.html) ，又一个记事本的替代品。但与一些试图改进记事本的臃肿而缓慢的实现不同，这个版本确实可以作为 Windows 附带的简陋文本编辑器的拖放替代物。

对于 starts，默认情况下，它看起来非常类似于经典的记事本。这是一件好事。记事本成功的原因之一是它开门见山，只显示文件。和记事本 2，当查看一个纯文本文件将做到这一点，没有标尺和工具栏。

然而，打开一个 C#文件，你会看到行号页边空白、工具栏、语法高亮和你正在使用的文件类型的额外功能。最重要的是，这些额外的东西不会使界面混乱。和大多数现代应用程序一样，它是 XP 主题感知的，所以可以很好地适应你的桌面。

它的启动速度也非常快:我无法区分加载记事本和记事本 2 的速度。对我来说，这是它和类似的利基应用程序之间的另一个巨大差异。

正如我将在下周重点介绍的任何工具一样，它可以免费使用和下载。
