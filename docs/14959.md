# 忘记广告销售——销售搜索！

> 原文:[https://www.sitepoint.com/forget-ad-sales-sell-search/](https://www.sitepoint.com/forget-ad-sales-sell-search/)

点击付费搜索引擎总是在寻找搜索流量。PPC 引擎有数以千计的满意的广告商要求更多的搜索流量，但只有这么多的搜索者。在这种情况下，任何能够产生搜索的网站管理员都会让点击付费搜索引擎敲开他们的门，要求建立分销关系。

对于你，站长，这意味着利润！如果你问“如何？”以及“为什么？”，请继续阅读。这篇文章将帮助你充分理解为什么与点击付费搜索引擎的分销关系往往是最有效的方法来赚钱你的网站流量。

##### 搜索心态

当一个互联网用户决定在网上买东西时，他们要么直接去一个他们以前体验过的网站，要么搜索一个出售他们想要的东西的网站。

直接去一个已知网站的用户很难赢得。向这些用户营销很难，因为他们已经是另一个网站的满意客户，很难让他们有一个以行动为导向的心态。

另一方面，当用户决定搜索时，这意味着他们是目标导向的，他们不熟悉能满足他们需求的网站。这些都是潜在的客户，他们准备购买，通常情况下，网站管理员谁出价最高的搜索结果排名将获得特定的客户。

##### 网站管理员的困境

作为一个网站管理员，你有能力将你的广告清单与你的用户目标相匹配。具有相同人口统计数据的用户，在同一时间浏览同一个网站，有不同的目标。由于这个原因，一个广告永远不可能完全优化，因为最终用户的目标只有在搜索或购买发生后才能知道。这使得站长很难将相关广告放在每个唯一用户面前。

然而，网站管理员可以显示一个搜索框，或有针对性的搜索结果，使用户能够得到他们想去的地方。这就要求网站管理员很少事后批评，并且减少了不属于某个特定广告目标受众的用户被你的现场广告惹恼的机会。

PPC 搜索引擎提供许多不同的广告客户，以满足每个用户的需求。所以，通过加入 PPC 搜索引擎，你可以接触到成千上万的广告客户。你的用户不仅可以选择他们看到的结果(通过完成他们自己的搜索)，还可以选择他们想要访问的广告客户。

高目标横幅广告的转化率无法与高目标搜索项的转化率相提并论。此外，你的网站访问者根据他们的最终目标输入不同的搜索词(例如，购买产品或研究他们的问题或需求的潜在解决方案)。与点击付费搜索引擎的分销关系允许你将用户发送给他们想要访问的广告商。横幅广告或任何其他形式的在线广告都不可能达到这种粒度水平。而且，随着在线广告商继续将他们的在线营销资金投入到搜索中，来自 PPC 搜索引擎联盟的收入潜力将继续增加。

##### 你看到光了吗？

最近，互联网上访问量最大的两个内容网站宣布，它们分别与领先的点击付费搜索引擎 Overture 达成了分销协议。ESPN.com 和 CNN.com 都宣布他们已经选择 Overture 作为他们的独家商业搜索提供商。

这些主要的在线品牌明白，在线广告的钱都花在了 PPC 搜索引擎上，并且意识到任何内容网站都可以通过向这些搜索巨头提供用户群来分一杯羹。

可以肯定地说，如果主要的在线品牌乐于将广告库存外包给 PPC 搜索引擎，那么这是所有网站都应该考虑的事情。

##### 网站管理员的其他优势

拥有成千上万的满意的搜索广告客户有很多好处。每个季度，CPC 价格持续上涨。当您注册成为分销合作伙伴时，您将获得基于广告商支出的收入分成。随着点击费的增加，你的收入也会增加。这是从这个不断发展的行业中获益的一种简单方式。

此外，当您注册成为分销合作伙伴时，您不必担心任何广告客户问题。PPC 支持广告商，给他们开账单，并给你寄一张你的作品的支票。你不必担心收到很多小额支票，也不必花时间维护与大量广告商的关系。搜索引擎会帮你处理所有这些事情。

最重要的是，你会一直按时拿到工资。当前的 PPC 模式要求搜索广告客户预先付费。大多数其他的广告单位都是先投放后付费，这可能会给网站管理员带来延迟和挫败感。事实上，搜索引擎已经收到了钱，这意味着你在支付方面遇到问题的可能性更小。此外，PPC 引擎是众所周知的支付他们的合作伙伴的时间每次。他们为什么这么勤奋？因为 PPC 引擎知道没有分发伙伴就无法生存。他们也认识到竞争的威胁:如果他们不支持他们的分销网络，他们的合作伙伴将会毫不犹豫地转向行业中的其他 PPC 引擎。

##### 开始吧！

今天就开始用分布式搜索赚钱吧。

如果你的网站吸引了大量的流量——不管你目前每月是否产生大量的搜索——你应该强烈考虑谷歌和 T2 的合作项目。

每月没有大量用户搜索他们网站的小型运营商应该关注小型搜索合作项目，比如由 [Portalgates](http://www.portalgates.com/corp/partner.shtml) 提供的项目。

##### 一个你已经注册，提示搜索！

有许多不同的方法可以用来提高你的网站搜索流量。最简单也是最常见的方法是在你的网站上放置一个搜索框。许多网站会有一个网站搜索框，和一个全网搜索框。这是搜索分发的实际应用！

然而，随着越来越多的公司试图推动用户进行搜索，将会开发出新的提示搜索的方法。例如，网站最近开始显示横幅广告和弹出窗口，其中包含实际的搜索结果。例如，如果你有一个体育网站，你可以显示一个弹出广告，显示“在线体育博彩”的前 3 名 PPC 广告商。这些类型的工具并不被所有的 PPC 搜索引擎所接受，但是随着业务的扩展，它们变得越来越普遍。

永远记住:在一天结束时，如果你有流量，就有钱可赚！
