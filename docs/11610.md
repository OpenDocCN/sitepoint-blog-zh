# 在 Web Directions South 赢得签名 SitePoint 图书

> 原文:[https://www . site point . com/win-signed-site point-books-at-web-directions-south/](https://www.sitepoint.com/win-signed-sitepoint-books-at-web-directions-south/)

如果你今天在悉尼参加 Web Directions South conference，我们会在今天下午 Kev 的 [CSS 框架](http://south09.webdirections.org/program/development#css-frameworks)演讲中分发几本 SitePoint 书籍，并且会有作者签名！

埃利奥特·杰伊·斯托克、卡梅隆·亚当斯和 SitePoint 自己的凯文·杨克都是今年 WDS 会议的发言人。如果你想要一本[性感网页设计](https://www.sitepoint.com/books/sexy1/)(埃利奥特签名)[&JavaScript 艺术科学](https://www.sitepoint.com/books/jsdesign1)(卡梅隆签名)或者[用 PHP & MySQL](https://www.sitepoint.com/books/phpmysql4) (凯文签名)建立自己的数据库驱动网站，那么你需要做的是:

1.  在 Twitter 上写一篇包含单词 **SitePoint** 的帖子
2.  注册 Web Directions South [Tweeps 应用程序](http://wds09.tweeps.webdirections.org/)(证明你在会议上)

我们将从这些推文中随机选出一名获奖者，但你必须在 Kev 的演讲室中领取这些非常酷的奖品。包含单词 **SitePoint** 的推文越多，你获胜的几率就越大！

凯文的演讲在午饭后开始。我已经预览过了，我可以确定这是一个精彩的演讲。
