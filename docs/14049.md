# 创业贷款

> 原文:[https://www.sitepoint.com/starting-up-business-loan/](https://www.sitepoint.com/starting-up-business-loan/)

Malikyte 还问了关于商业贷款的问题。

祝你好运！很少有银行会贷款给你专业创业。

在美国，你最好的选择是 SBA 贷款。然而，这些很难获得，银行通常需要看到房屋或其他资产形式的抵押品。因此，如果你有一套房子，并想冒险，你最好只是再融资套现你的股权。

但是不要那么做。

相反，储存足够的资金储备，这样你就可以支付你的业务费用和 1-2 年的生活费用。(或者确保你可以依靠配偶的收入生活，如果你已经结婚的话)。

当你有足够的财政储备时，会发生两件事:

1.你不会给你的客户留下绝望的印象。

2.你感到自信是因为你有足够的时间去做生意。

最后，如果可以的话，在有几个客户排好队之前，不要贸然行动。这听起来像一个奇怪的第 22 条军规，一边工作一边安排客户，但我的许多同事就是这么做的。只有当他们找到一些客户时，他们才开始创业(这是他们在全职工作的同时设法做到的——并且明智地利用假期)。那样的话，现金流根本不是问题。
