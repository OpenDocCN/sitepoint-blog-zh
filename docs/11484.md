# 苹果在创造广告支持的设备吗？

> 原文：<https://www.sitepoint.com/is-apple-creating-ad-supported-devices/>

苹果公司并不经常调查削减成本的措施。考虑 MP3 播放器。它们早在 iPod 出现之前就存在了，但苹果采用了这项技术，把它放在一个漂亮的盒子里，价格翻了一倍，并在世界范围内获得了成功。

iPhone、iMacs 和苹果生产的其他技术也是如此。苹果粉丝很少对价格敏感。当一个新的苹果产品出现时，他们会卖掉他们的孩子进行医学实验，只为了得到他们的设备。

然而，最近发现的一项苹果专利显示，该公司正在考虑其他收入选择。该专利提到了一个*“执行程序”*，它以预先定义的间隔播放广告，用户无法跳过。此外，设备可以询问问题或请求特定的按键组合，以确保用户一直在注意。不正确的响应可能会导致设备暂时锁定。恶心。

尽管许多人会觉得这种高压广告令人生厌，但它会让你以大幅打折的价格买到苹果产品——甚至可能是免费的。

不确定苹果会实施这项技术。像大多数主要的技术公司一样，他们经常提交专利，但这些专利从来不会产生实际的产品。苹果拒绝进一步评论，然而，与之前的许多申请不同，首席执行官史蒂夫·乔布斯在这项专利上被指名。

广告支持的收入是一个已经在 IT 行业获得势头好几年的想法。这是谷歌的主要收入来源，微软正在 MS Works 等产品中尝试。对苹果来说，这可能标志着重心从向忠实追随者销售高端硬件的极端转移。如果广告模式奏效，任何人都可以拥有苹果产品，排他性的感觉将永远消失。

苹果会认真对待广告专利吗？如果苹果用户喜爱的产品以很低的价格或免费提供给大众，他们会有什么感受？
