# 雷神之锤直播是游戏的未来吗？

> 原文：<https://www.sitepoint.com/quake-live-gaming/>

本周早些时候，《雷神之锤》LIVE 向热切的网络游戏玩家推出。这款新游戏基于 3D 游戏专家 [id 软件](http://www.idsoftware.com/)推出的 10 年前广受好评的《雷神之锤 3》。然而，这个版本是免费的，可以随时随地在浏览器中直接播放。

这款游戏在最初几天非常受欢迎。如果你已经设法玩它，那么你是幸运者之一。许多人在深度超过 50，000 的虚拟队列中等待。Id 软件公司已经为这些问题道歉，并且正在寻找解决方案。然而，除非你非常非常有耐心，否则你不太可能得到快速爆破修复。

虽然这款游戏是基于浏览器的，但也有一些限制:

*   它只适用于 Windows PC 用户——目前还没有关于 Mac 或 Linux 可用性的消息
*   它只适用于 IE7 或 Firefox 3(不是 3.1 版本)——所以 Opera、Chrome、Safari 和 Lynx 的粉丝可能会失望
*   需要安装浏览器插件才能玩。

有传言称这款游戏是用 Adobe Flash(T2)开发的，尽管这个专用插件可能会处理大部分图形游戏工作。浏览器主要用作游戏的交付机制，而不是其实现的基础。

然而，现在还为时尚早。无论你是否认为《雷神之锤》是真正基于浏览器的，这款游戏的质量、可玩性和速度都与一代或两代人之前基于主机的游戏不相上下。到目前为止，Flash 和浏览器游戏一直很简单，但也许这次高调发布可以让我们一窥游戏行业的未来方向。

你玩过雷神之锤吗？你需要等很久吗？这个游戏好吗？你会推荐它吗？这会是高端点播浏览器游戏的开始吗？
