# 需要推出博客？不运行 PHP？看看潜台词

> 原文：<https://www.sitepoint.com/need-to-launch-a-blog-dont-run-php-check-out-subtext/>

由于我不能真正进入的原因，我正在部署一个多用户博客包。这个项目中还包括一些通过 web 服务接口的自定义扩展。另外，由于其他原因，我现在不能说，它不可能是 PHP/MySql 包。

事实上，一年前我们也遇到了同样的问题，但是管理层最终拒绝写博客。那时唯一的选择是德高望重的人。文本引擎。本质上，这是一个非常可靠的应用程序，但也在许多方面显示出它的老化。设置充其量是过时的。甚至不要考虑在共享主机上运行它。在为那个项目研究选项的过程中，我偶然发现了[的潜台词](http://www.subtextproject.com/)。当时这个项目还处于起步阶段，还没有发布版本。但是现在它在摇摆和滚动。很难。

**什么是潜台词？**

潜台词是。文本引擎。当前发布版本(在撰写本文时为 1.5.2)的主要改进包括:

1.  设置简单得令人不安。并且可以在共享主机上轻松管理。一个对服务器管理知之甚少的人。
2.  股票模板要干净得多。而且漂亮多了。并且通常在非 IE 浏览器中工作。
3.  管理工具感觉好多了。Fck 编辑器的加入就是一个好的。
4.  配置和管理不需要命令行工具和服务器的基本知识。
5.  MetaBlogAPI 已经实现，允许它与所有 neato blogger 工具一起工作。以及简单、定制的扩展，而不需要使用核心包。
6.  它只是一个博客。不像其他一些软件包，如果你只是想告诉世界你穿着肚脐眼上的线头粘在裤子上的经历，你不会得到各种各样无关紧要的功能。NET 平台。

另外，[潜台词是一个非常活跃的项目](http://sourceforge.net/projects/subtext/)。这位仁慈的独裁者不亚于菲尔·哈克(他也有[一个基于潜台词的博客](http://www.haacked.com/))。他得到了[和一个优秀的开发团队](http://subtextproject.com/Home/About/MeetTheTeam/tabid/115/Default.aspx)的支持。在我撰写本文时，该团队正在向 1.90 版本迈进。此外，开发团队反应非常迅速。我在 SourceForge 网站上发布了一份错误报告，几分钟后我就收到了 Phil 的邮件。

**怎样才能得到潜台词？**

你需要得到的一切都在[潜台词项目 SourceForge 网站](http://sourceforge.net/projects/subtext/)上。更多关于该项目的信息，请查看[潜台词项目](http://www.subtextproject.com/)网站。
