# 这个麦克风开着吗？

> 原文：<https://www.sitepoint.com/is-this-microphone-on/>

大家好！

*   我的名字是丹·提斯，我是你的新的搜索引擎营销网站博主。我想简单介绍一下是合适的。

这个家伙是谁？

*   自从网景公司成立以来，我一直在建立和推广网站，自从有搜索引擎以来，我一直在处理搜索引擎优化问题。自从 2001 年我出版了我的初学者书籍《SEO 快速入门》以来，我已经更多地参与了搜索引擎营销。那本书我们卖了不到 5000 册，第五版将于下月出版。
*   你会发现我在几个不同的论坛上讨论搜索引擎营销。我是[高排名](http://www.highrankings.com/forum)和[最佳实践](http://www.ihelpyouservices.com/forums) [](http://www.sempo.org)论坛的主持人，是 [SEMPO](http://www.sempo.org) 的会员委员会志愿者，也是搜索引擎优化专家组织 [SeoPros](http://www.seopros.org) 的执行委员会成员。
*   2003 年，我创办了现在的公司 [SEO 研究实验室](http://www.seoresearchlabs.com)。我们为 SEO/SEM 顾问、网站设计师和其他网络专业人士提供支持服务和培训。从那以后，我没有直接为客户做过任何搜索引擎营销工作，相反，我担任顾问的顾问。
*   SEO 快速入门现在可以从我的网站上免费下载，如果任何 SitePointers 想剖析它的话。它并不打算作为一个全面的参考搜索引擎优化，而是作为一个简单的系统，供初学者在学习做自己的搜索引擎优化。
*   如果你想要一本关于搜索引擎营销的大型综合书籍，你不用等太久，因为 SitePoint 将在本月晚些时候出版我的新书。它花了六个多月的时间来编写，是一个庞然大物，所以请继续关注更多关于 **SitePoint 搜索引擎营销工具包的内容。**

**斧磨？你打赌！**

是的，我有一两条意见。

*   大多数“搜索引擎营销”更好地描述为“优化”优化网页，优化点击付费广告。营销不仅仅是技术。
*   搜索引擎营销人员的目标应该是改善商业结果，或者支持拥有网站的组织或个人的目标…搜索引擎排名不是商业结果，也不是“更多的流量”
*   关键词研究和关键词策略是搜索引擎营销的核心，关键词研究的学科需要更多的关注。所以我要讲很多。
*   黑帽和白帽无关紧要，但我不喜欢欺骗技巧、短期策略和快速解决方案。
*   更多观点将会出现…

那么有什么计划呢？

我的计划是，在大量的评论告诉我相反的情况下，每周尝试攻击一个不同的主题——周一介绍一些背景知识，周三进行更深入的讨论，周五查看相关的工具和资源。在这期间，也许在繁忙的一周里，我们会关注突发新闻和热点话题。
