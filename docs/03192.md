是的，Chrome 又涨了。0.5%可能没什么特别的，但月环比来看，Chrome 在过去一年的用户群增长了 12.7%。IE/Edge、Safari 和 Opera 几乎没有变化，这让我们只剩下一个失败者。

Firefox 遭遇了有史以来最大的一次掉线，20 个用户中就有 1 个迁移到了别处。我不相信有什么特别的原因？

Firefox 仍然是唯一不依赖商业实体的主流开源浏览器。没有董事或股东，所以应用程序可以选择对网络最有利的方向。Mozilla 开发了一个很棒的浏览器，尽管有人质疑 Mac OS 的性能，但 Firefox 提供了强有力的竞争，而且通常比 Chrome 使用的资源更少。内置的开发工具比其他地方的工具更好，数以千计的有用扩展提供了深度的浏览器集成。

*那么为什么用户会放弃 Firefox 呢？*

IE/Edge 是 Windows 上的默认浏览器。Safari 在 Mac OS 上是默认的，在 iOS 上是强制的。Chrome 在 Chrome OS 中使用，每次有人访问谷歌服务时都会做广告。相比之下，Firefox 在使用率相对较低的 Linux 发行版和 Firefox OS 中可用。

安装 Firefox 是开发人员和高级用户有意识的决定，他们想要它提供的可配置的体验。不幸的是，如果另一个浏览器提供更好或更稳定的环境，他们会毫不犹豫地切换。当开发工具变得有趣时，许多开发人员迁移到 Chrome 他们没有理由再回来。

Firefox 很少被非技术网络用户考虑。这在用户群积极推广和安装 Firefox 的早期并不明显，因为唯一的其他选择是老化的 IE6。今天，我们不在乎人们用什么。最新的浏览器都是不错的应用。

Mozilla 能阻止这种趋势吗？

火狐不会去任何地方。尽管是一个非盈利组织，但由于相关的广告和合作收入，Mozilla 有充足的现金。用户越少，赚钱越少，但浏览器仍然拥有庞大的粉丝群。

营销活动可能会有所帮助，但与谷歌对 Chrome 的投入相比，预算微不足道。Mozilla 可以简化 Firefox 来吸引更多非技术用户，但是已经有很多简单的浏览器可供选择。

总的来说，Mozilla 应该坚持使用 Firefox，并记住这是公司成功的主要原因。火狐操作系统、[火柴棍](http://www.matchstick.tv/)和 [Webmaker](http://webmaker.org/) 等。都是有趣的项目，但它们不是革命性的，并且有许多商业替代方案。

无论你如何看待火狐，没有它你就没有 Mozilla 开发者网络。W3Schools 将再次占据主导地位。

## 2015 年 10 月至 11 月全球移动浏览器统计

11 月的手机使用量下滑了 1.5%，目前占所有网络活动的 39.67%。寒冷天气的信号？我怀疑随着送礼季节的临近，价格会大幅上涨。

热门移动浏览应用:

| 移动浏览器 | 十月 | 十一月 | 变化 | 亲戚 |
| 铬 | 35.85% | 37.42% | +1.57% | +4.40% |
| 苹果手机 | 16.38% | 17.94% | +1.56% | +9.50% |
| UC 浏览器 | 17.42% | 16.94% | -0.48% | -2.80% |
| 机器人 | 13.06% | 11.73% | -1.33% | -10.20% |
| Opera Mini/Mobile | 12.54% | 11.37% | -1.17% | -9.30% |
| 不变的 | 2.07% | 2.04% | -0.03% | -1.40% |
| 其他人 | 2.68% | 2.56% | -0.12% | -4.50% |

同样，Chrome 经历了巨大的增长，但它被 Android 浏览器使用量的下降所抵消。因此，最大的上升者是 iPhone 上的 Safari，它在落后 UC 浏览器几个月后重新夺回了第二的位置。尽管有 Android 和 iOS(皮肤 Safari)版本，Firefox 在移动市场的份额只有 0.1%。

全球图表已经分裂。Chrome 和 Safari 在美国、欧洲和澳大利亚占据主导地位。UC 浏览器和 Opera 在亚洲和非洲非常成功。

下个月见，对 12 月下旬最受欢迎的设备进行初步讨论。
