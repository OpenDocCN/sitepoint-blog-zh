# 浏览器趋势 2012 年 3 月:IE8 跌破 20%

> 原文:[https://www.sitepoint.com/browser-trends-march-2012/](https://www.sitepoint.com/browser-trends-march-2012/)

又一个月过去了，是时候再次关注浏览器市场了。在二月，Chrome 16 成为世界上使用最多的浏览器版本，超过 IE8 5%的市场份额。Chrome 能继续它的迅速崛起吗？让我们看看最新的[全球统计数据](https://gs.statcounter.com/#browser_version-ww-monthly-201202-201202-bar)来找出答案…

| 浏览器 | 一月 | 二月 | 变化 | 亲戚 |
| IE 9.0 以上 | 11.45% | 12.09% | +0.64% | +5.60% |
| IE 8.0 | 20.82% | 18.86% | -1.96% | -9.40% |
| IE 7.0 | 3.63% | 3.32% | -0.31% | -8.50% |
| IE 6.0 | 1.56% | 1.48% | -0.08% | -5.10% |
| 火狐 4.0 以上版本 | 20.01% | 20.39% | +0.38% | +1.90% |
| 火狐 3.7- | 4.77% | 4.49% | -0.28% | -5.90% |
| 铬 | 28.45% | 29.88% | +1.43% | +5.00% |

下表显示了桌面浏览器的市场份额估计值。“变化”栏显示市场份额的绝对增加或减少。“相对”栏显示比例变化，即上个月又有 9.4%的 IE8 用户放弃了浏览器。有几个警告，所以我建议你阅读[如何计算浏览器市场份额](https://www.sitepoint.com/how-browser-market-share-is-calculated)。

对微软来说有一个好消息:IE8 再次成为最常用的浏览器版本。

坏消息是:

*   这只是因为谷歌发布了 Chrome 17。Chrome 上个月有一半的用户群升级，这导致 17 版增长了 13.7%，而 16 版保持了 13.4%。
*   自 2009 年 11 月以来，IE8 用户比例首次下滑至 1/5 以下。
*   Internet Explorer 在一个月内失去了近 5%的用户群。这是一个令人震惊的数字。

IE10 可能会令人惊叹，但它暂时不太可能出现，IE9 正在被抛在后面。微软放弃 XP 用户的决定也产生了负面影响:老化的操作系统仍占三分之一以上的个人电脑用户，他们要么坚持使用 IE8，要么转向替代浏览器。

从技术上来说，IE8 还不算太差，只要有一点 JavaScript 的魔力，它就能很好地支持 HTML5。但是它缺少基本的 CSS3 特性，比如圆角和阴影，这些我们都在用。结果是:网站是可用的，但与其他浏览器相比，在 IE8 中看起来很糟糕。

当然，微软希望用户升级到 Windows 7 或 8，这样他们就可以使用 IE9/10。不幸的是，这不会很快发生。即使经济和成本不是考虑因素，大型组织也会实施长期 IT 计划，并且升级成千上万的用户需要时间。PC 市场也已经饱和好几年了；许多人会坚持使用 XP，直到他们的机器坏了，他们被迫购买替代品。让我们不要忘记 XP 仍然是一个有能力的操作系统；一些用户更喜欢它，微软不会考虑撤销对 Office 等主要产品的支持。

微软有一个简单的选择:要么在 XP 上发布 IE9/10，要么继续失去大量市场份额。虽然会有技术障碍，但 IE9 已经有了软件渲染模式，如果 XP 版本更慢，很少有人会抱怨。相对较小的 [Opera](http://www.opera.com/) 可以创建一个在 Windows 2000 上运行的现代 HTML5 浏览器——微软没有什么借口。用户越来越懂浏览器，一旦他们转而使用其他浏览器，就很难再回到 IE 上来。

使用模式并不那么直接，但是大部分 IE 用户都转向了 Chrome。如果说有什么不同的话，那就是随着用户群的扩大，这款浏览器的市场份额似乎在加速增长。没有迹象表明会出现预期的增长平台。

Mozilla Firefox 的份额稳定在 25%。[第 10 版](https://www.sitepoint.com/firefox-10-whats-new/)广受好评，许多困扰用户的内存和附加组件故障已经得到解决。Mozilla 没有扭转下滑趋势，但 Firefox 的未来看起来更光明。

野生动物园和 T2 歌剧院都有小幅增长。值得注意的是，Opera 再次突破了 2%的关口——它已经低于这个水平超过 12 个月了。

## 移动浏览器的使用

2012 年 2 月，手机占全部网络活动的 8.53%。

主要的移动浏览应用程序有:

1.  安卓系统——22.67%(增长 1.28%)
2.  opera Mini/Mobile——21.70%(下降 1.64%)
3.  iPhone——21.06%(增长 1.55%)
4.  诺基亚浏览器——11.24%(下降 0.58%)
5.  黑莓——6.53%(下降 0.15%)

前三名的使用数字几乎相同，但 Opera 第一次被挤下了榜首。Android 浏览器一直在以类似于 Chrome 的速度稳步增长，Chrome 是桌面浏览器的近亲。

黑莓跌得更远了一点。就在 18 个月前，它还占有 19%的市场份额，这说明了移动市场的变化无常。
