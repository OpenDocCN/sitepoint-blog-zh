# 定期检查您的价格

> 原文：<https://www.sitepoint.com/review-your-rates-regularly/>

我最近和我的一个客户聊天，他提到了他们以前的 web 开发人员。她告诉我一些她不喜欢的事情，并重申为什么她更喜欢我的生意。

我受宠若惊。然而，我问了一个对我解释了这么多的问题，我只是必须与你分享它。

问题是。

我问她为什么离开他们，并决定寻找新的供应商。

答案？

他们开始要价过高。现在，当我挖得更深一点时，我发现了真正的原因:他们现在比以前收取更多的费用。

事实上，如果你看看我们的价格，和她的老供应商，我们现在实际上比他们现在每小时收费更高。

这不是因为他们收取每小时的费用，而是因为他们从一个低得多的费用开始，坚持了几年，然后一夜之间提高了 50%以上。让这位客户心烦意乱的是震惊和缺乏沟通，而不是实际的金钱。

那么这个故事给你的启示是什么？确保你现在充好电。如果你认为你在欺骗自己，开始计划定期小幅提高你的价格。

如果你定期回顾你的利率，你会发现每年小幅上涨一次比突然上涨 50%要好得多。这意味着你会避免让客户震惊，你付出的努力会得到更好的回报。
