# 互联网是永恒的

> 原文:[https://www.sitepoint.com/the-internet-is-forever/](https://www.sitepoint.com/the-internet-is-forever/)

今天，我每周的漫谈更像是一个明智的提醒:互联网是永远存在的。我收到了比平时更多的请求，人们要求删除他们过去发布的帖子。原因一般是三个之一。他们不想让老板看到他们正在努力解决与工作相关的问题。或者他们不想让客户知道他们并不像他们声称的那样是 CSS 大师。或者是因为他们不想出现在谷歌搜索中(尤其是如果他们一直在包装公司的话)。

互联网是永恒的，它会讲故事。

人们很容易认为网络是匿名的，但事实并非如此。如果你在一个鲁莽的时刻提名你的客户，不要使用他们的名字。他们可能会时不时地用谷歌搜索自己——见鬼，我们都是这样——而谷歌就是喜欢 SitePoint。不要写任何你不想放在广告牌上的东西。如果你在某个项目上需要帮助，没问题，这就是我们在这里的原因，但是想想你如何请求它。如果需要，将您的代码移动到测试站点，而不是链接到您的客户站点。

一般来说，我们会尽我们所能满足你，但我们没有时间从你上个月发表的所有帖子中删除链接，我们也不会删除帖子，因为其他人可能会在未来从他们那里学习。

讲座到此结束。
