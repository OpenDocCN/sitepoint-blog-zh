# 辩论——设计已死！

> 原文:[https://www.sitepoint.com/debate-design-dead/](https://www.sitepoint.com/debate-design-dead/)

网页设计已死！故事结束…但它真的活着吗？这是个问题。

我敢肯定，这一声明将很容易引发两个截然不同的阵营的激烈辩论，我们都知道在我们奇怪的小 IT 世界:网页设计师与网页开发者。会有什么解决吗？不太可能……但它总是能制造出好的焰火(双关语)。

我认为自己是一名网页开发人员，也就是说，我不仅设计网页，还设计网页、保护网页、托管网页等等。在我看来，网页设计师基本上是一个图形和布局人员，主要关心网站的“外观”。这就是我的论点和设计的定义的基础:就创造性的艺术创新而言，无论是在网络上，印刷上，还是其他任何媒体形式，设计都是死的。

现在，开发人员和设计人员都有一个共同的目标，那就是他们需要向给定的目标受众呈现一个有吸引力且可用的网站。吸引力和可用性这两个要素是任何希望成功的网站的必备要素。它们还需要根据目标受众的需求，在彼此之间保持重要的平衡。如果一个网站太漂亮，太时尚，或者太前卫等等。这可能会使许多用户，尤其是成千上万的新手到中级用户失去兴趣，他们构成了今天互联网浏览者的主体。然而，随着这些人变得越来越精通网络，他们会要求他们的网页具有一定程度的质量、效率或真实性，而这只有一个精心设计的专业网站才能提供——尤其是在电子商务功能方面。如果网站包含太多的“混乱”或干扰，或者对用户来说，仅仅知道基础知识就有太大的学习曲线，那么所有技能水平的访问者都会离开——而且会很快离开(我知道我肯定会)。

因此，一旦可用性和“抛光”之间的平衡达到，我相信就网站的设计和外观而言，会出现一种“收益递减法则”。例如，只有在非常排外的艺术界，或者像电影、电视或广告这样以图像为基础的行业，任何类型的复杂或精细的设计结构才会受到重视和欣赏。对于今天的大多数网络冲浪者来说，他们的注意力持续下降，网络连接也同样不起眼，网络内容必须吸引他们的注意力——是的——但也要足够熟悉，以便即时使用。如果不是，他们就走了。再说一次，如果页面没有提供比 flash 更重要的内容，即使是来自利基目标受众的熟练网络用户也很可能很快离开。

因此，我将从建立一个有效的、布局良好的信息呈现结构的角度，给出设计的“道具”;然而，网页设计作为一种建立独特的创意愿景或存在的形式，不仅已经死亡，而且从一开始就没有真正活着。

从一开始，互联网就是人们有效分享信息的一种方式。现在，我承认网络是一种不同于印刷或电视的媒介，并且有一些特定的设计原则来利用这种新的交流工具。然而，像所有好的基本面一样，这些原则不会改变。多年来，它们实际上一直是静态的——并将继续如此，尽管设计界声称情况正相反。再次，让我先说我的观点是从我的北美/西欧的角度，所以我坦率地承认，来自另一种文化的观点有可能使我的理论充满漏洞…但同样，这是从互联网起源的地区的角度，以及我们今天知道的许多其他强大的媒体形式。

成功的设计原则和来自这些其他媒体格式的用户行为模式被整合在一起，为网页设计和开发提供了一个基础。这些基本原则在两个最广泛使用和尊重的网页设计中显而易见:耶鲁大学的网页风格指南和 Jakob Neilson 的设计网页可用性:简单的实践。

Web 风格指南是一个很好的资源，它说明了对于一致的、熟悉的布局的需要，这种布局与我们人类处理信息的方式是一致的，尤其是在 Web 浏览器的范围内。Jakob Neilson 基于这一基本前提，阐述了使用可靠的设计方法的重要性，否则会失去你的观众。例如，在他 2000 年 7 月的“Alertbox”的一篇文章中，他特别指出“网站必须在所有方面淡化它们的个人外观和独特设计:视觉设计、术语和标签、交互设计和工作流程、信息架构”。尼尔森认为，我也同意，这是必要的，因为网站设计需要应对四个主要趋势:

1.  用户的体验，

3.  移动互联网，

5.  网络计算，以及

7.  联合内容和服务。

所有这些趋势的一个共同点是，它们要求标准惯例和简单的设计原则，不会混淆或压倒最终用户或新的硬件/软件创新。

此外，曾经被认为是“设计”的东西现在作为功能(例如后退按钮、书签、电子邮件/媒体工具等)被内置到浏览器中。构成网站的实际页面越来越多地只关注内容的显示，这是一项不需要艺术创造力的任务，并从标准化和高效的表示中受益匪浅。这种趋势在许多情况下都很明显，例如，旧的 HTML 标记被弃用，而代之以标准化的 XHTML、CSS、XML 和其他编码技术和实践。

许多人会认为，随着新的有线电视频道/渠道、杂志和网站的快速增加，我们的社会和媒体选择变得越来越复杂和复杂。支持者认为，这样一来，每种媒体都需要一个新的专业化水平，专门针对它所针对的利基市场。毕竟这是 21 世纪。我们需要更新更具创新性的设计来持续照亮和推动媒体爆炸，对吗？不对！

简单的事实是，这种增加选择的想法是一种错觉。是的，在频道、杂志和网页的数量上，肯定有更多的选择；然而，你的选择每天被更少的人或公司控制和传播。这些公司要想继续经营下去，就必须为了企业形象而限制和控制创新；这一切都是为了培养对他们的品牌或标志的熟悉度(阅读纳奥米·克莱恩的《没有标志》了解更多信息)。

你最后一次看到耐克的 Swoosh 和金色拱门是什么时候？为什么这些符号被称为专有名词，并在我们的大脑中立即触发识别？为什么所有的新闻频道和节目声音和外观都一样？为什么，就此而言，我们的文件？在所有形式的大众媒体中，个人控制和表达或多或少已经被消除。取而代之的是一个大型企业霸权，它认为，为了在今天的市场上生存，你需要品牌，品牌，品牌，在任何地方印上你的标志，并控制一切有关你形象的使用。不这样做就是公司自杀。

其结果是，无论人们是在沃尔玛购物、去多中心电影院看电影、吃快餐还是在网上冲浪，现在都需要在一个共同的、众所周知的环境中感到舒适。他们已经习惯了熟悉的事物，现在确实需要它。很明显，创新设计并不适合于满足现有消费者和吸引新消费者所需的亲切感的交流。

那么我们还剩下什么？我承认在网络上有一些非常酷的设计概念。然而，难道你没有注意到，即使是这些网站基本上看起来和感觉都是一样的吗？大多数都是在 Flash 中实现的，带有“时髦的导航”、“酷的淡入淡出”和“预加载器”等等……嘿，这听起来越来越熟悉了。即使是所谓的“前沿”网站仍然“遵守规则”。

但这是一件好事——我希望更多的设计师能够认识到这一点。也就是说:任何创作都需要有一个相同的核心元素，这样它才能对个体作者、艺术家或网站开发者之外的任何人“有用”。人们不仅在文体上受到以前出现的东西的影响，还受到以前“起作用”的东西的影响。几乎每天都有客户来找我，第一口气就说他们想要一个“新”的设计来满足他们的“个性化”需求。接下来，他们说他们想要“看起来像我的竞争对手的东西”，或者“看起来像 X 公司”等等。

那么，真正的创意设计是活的吗？不。但是，它从来就不是真的…嗯，也许是在格罗在他的洞穴墙壁上画第一只羚羊的那一瞬间。但是，从那时起，为了以一种有效的方式向一群人最好地传达一种共同理解的意思，就需要不断地重新解释以前发生的事情。

从一个“千篇一律”的网站，到一种“激进的冲击艺术”，设计师和艺术家通过一套规则和模式来引起目标受众的反应。事实上，由于媒体公司的聚集，我们现在面临的创新途径实际上更少，这进一步证明了这一点。设计确实死了。
