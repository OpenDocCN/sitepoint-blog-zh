# 13 种备份社交媒体内容的工具

> 原文：<https://www.sitepoint.com/backup-social-media-profiles/>

我们都(希望)定期备份我们的计算机数据，但在线数据呢？如果您丢失了脸书联系人、博客帖子、营销视频、推文历史或在线书签，您的业务和营销活动会受到什么影响？

在你的社交网络账户上备份信息可以节省你的时间、压力和潜在的业务损失。这里有一个工具列表，给你一个创建社交媒体备份计划的起点。

## 一体化社交媒体备份

**[Backupify](http://backupify.com) :** 自动、安全地备份你在 Twitter、脸书、WordPress、Delicious、Flickr、Google Docs 和其他使用亚马逊 S3 的网站上的数据。

**[社交软件同步](http://sync.socialware.com/) :** 脸书、Twitter 和 LinkedIn 的社交档案，包括搜索功能和活动统计。

## 博客备份

**[BlogBackupr](http://blogbackupr.com/):**WordPress 和 Blogger 的备份服务，包括帖子、评论、页面、类别和媒体。

**[WP-DB 管理器](http://wordpress.org/extend/plugins/wp-dbmanager/) :** 一个 WordPress 插件，允许你备份、优化和恢复你的数据库。

## 脸书备份

**[archive Facebook](https://addons.mozilla.org/en-US/firefox/addon/13993):**Firefox 的一个插件，允许你将脸书账户中的内容——包括照片、消息、活动流、好友列表、笔记、事件和群组——直接保存到硬盘上。

[**【social safe】**](http://www.socialsafe.net/):一个备份你的朋友、个人资料、照片和状态更新的工具，然后回到过去，看看你的个人资料发生了什么变化。

## Flickr 备份

**[Downloadr](https://downloadr.en.softonic.com/) :** 一款 Windows 应用，可以让你从 Flickr 下载照片到自己的电脑上。

**[FlickrEdit](http://sunkencity.org/flickredit) :** 一款基于 Java 的桌面应用，允许你从 Flickr 下载/备份或上传照片。

## Twitter 备份

**[BackupMyTweets](http://backupmytweets.com/) :** 一项提供每日推文备份的服务，允许你在一个网页上查看你所有的推文，或者以 HTML、XML 或 JSON 格式下载。

**[Tweetake](http://tweetake.com/) :** 一个支持你的关注者、你所关注的人和推文的工具。

**[TweetBackup](http://tweetbackup.com/) :** 导出并备份你的推文和好友列表的工具。

## 油管（国外视频网站）

**[YouTube 下载器](http://download.cnet.com/YouTube-Downloader/3000-2071_4-10647340.html) :** 软件，可以让你从 YouTube 上下载视频，包括高清和 HQ 视频，并转换成其他视频格式。
