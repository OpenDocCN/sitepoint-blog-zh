# 现成的:2009 年的预测

> 原文:[https://www . site point . com/whats-on-tap-predictions-for-2009/](https://www.sitepoint.com/whats-on-tap-predictions-for-2009/)

又到了一年的这个时候。正如我们在本周早些时候提到的，对于科技博客来说，没有什么比展望明年并试图预测未来更有趣的了。预测帖子是博客圈一年一度的传统，我们喜欢这样做。

因此，以下是对网络技术新的一年的八个预测。当然，如果你[看过我们的 2008 年顶级故事](https://www.sitepoint.com/the-top-15-web-tech-stories-of-2008/)列表，你就会知道现实往往太古怪而无法预测——而且一年中会发生很多事情。记得看看[我们去年的预测](https://www.sitepoint.com/2008-predictions-year-end-review/)做得怎么样。请在评论中让我们知道你认为 2009 年网络将会发生什么。

注意:我在这篇文章中使用了公司的“我们”,但这些预测实际上只是我自己的，而不是 SitePoint 任何其他人的。所以，不要怪他们可怕的缺乏远见！

**1。Twitter 获得了商业模式。** Twitter 在盈利方面有很多不同的选择。基于你的推文内容在 Twitter 流中投放定向广告，内置微支付，向大量用户收费，向使用 API 的开发者收费，等等。Twitter 可能会尝试这些选项中的部分或全部，但我们认为最有可能实现盈利的途径是企业账户。当 SitePoint 通过 Twitter 运行我们非常成功且广为人知的图书赠品时，它之所以真正成为可能，是因为我们能够登上 Twitter 的白名单，并在不被阻止的情况下发送大量直接消息。在登上那张名单之前，事情进行得并不顺利。这是一种只有公司账户可能需要的附加功能，Twitter 可以收费。

**2。生活流变大了，但不是通过 FriendFeed。过去一年，FriendFeed 在早期用户中产生了巨大的影响，但是生活流还没有成为主流。人们现在才开始经常同时使用足够多的社交服务——YouTube、Flickr、Twitter、Digg、博客——这种聚合将开始变得有意义。在 2009 年，在一个单独的信息流中发布你的网络生活将会很平常，但是这将会通过脸书来完成。**

**3。网络操作系统将真正开始成为现实。**网络操作系统竞赛开始了。2009 年，我们将开始看到 Adobe (Flash、Flex、AIR)、Google (Chrome、Gears、Native Client)和 Microsoft (Silverlight、Live Mesh)等主要公司的愿景真正开始融合。

**4。一些真正伟大的东西将来自雅虎！但这不足以拯救他们。雅虎！通过开放他们的搜索结果和最受欢迎的页面和应用程序，让他们更加社会化，给开发者更多的机会，已经做了一些很棒的事情。这种事情最终会让网络变得更好，但不幸的是，这不足以拯救雅虎！在华尔街。他们的股票将继续下滑，除非他们与微软签署一项大的搜索协议，或者彻底出售他们的搜索业务，专注于内容/平台方面。**

**5。到今年年底，Chrome 将占据浏览器市场至少 5%但不超过 10%的份额。**谷歌浏览器现已退出测试，并由谷歌积极推广，到今年年底将至少占据浏览器市场的 5%,最多可达 10%。随着 Chrome 和 Mozilla Firefox 的出现，IE 将继续下滑，但 Chrome 实际上将导致 Mozilla 的增长停滞，一旦它支持扩展，甚至可能从 Firefox 窃取一些市场份额。

**6。微软 Office 将使人们对网络应用程序感到舒适。当微软推出基于网络的 Office 版本时，主流用户将最终开始习惯网络应用。谷歌的“文档”和“电子表格”应用肯定已经将相当多的人推向了那个方向，但微软将对主流用户采用网络应用产生更大的影响。他们的“软件+服务”愿景将在 2009 年成为软件的清晰未来。**

**7。脸书接管了网络。**嗯，不是字面意思。但是脸书在世界范围内的规模将继续增长，并最终超过 MySpace 成为美国最大的社交网络。许多早期采用者喜欢的酷网络应用程序，如 FriendFeed，将作为脸书的特色成为主流，而脸书连接将通过把它与人们知道和信任的其他流行网站结合起来，帮助传播脸书品牌。该公司还将扩大与微软的搜索合作，使网络搜索整合更加突出和强大。对许多用户来说，脸书将在 2009 年成为他们默认的搜索引擎，这将为 2010 年的首次公开募股铺平道路。

**8。Palm 会在 CES 上给大家惊喜… Nova 会不错的。** Palm 新进入智能手机市场将会给人留下深刻印象。Nova 操作系统看起来可能会在移动市场挑战 Android、黑莓和 iPhone，但成功将取决于硬件。

你对 2009 年有什么预测？请在评论中告诉我们。
