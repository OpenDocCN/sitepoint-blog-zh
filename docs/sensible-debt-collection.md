# 明智的债务回收

> 原文：<https://www.sitepoint.com/sensible-debt-collection/>

这篇文章是从论坛报#427 网站转载的。[订阅《今日论坛报》](https://www.sitepoint.com/newsletters/),每周在你的收件箱中收到更多类似的商业提示！

前几天在 [SitePoint 论坛](https://www.sitepoint.com/forums/)上，我注意到 [artcoder 问了一个对企业主来说很常见的问题](https://www.sitepoint.com/forums/showthread.php?t=587534):追债值得吗？

和大多数人一样，这些年来我也有过坏账。我发现对付缓慢付款或不付款的最好方法是使用一个系统。你的系统是什么并不重要——只要你有一个。

这里有一个例子:

1.  项目完成:最终付款发票
2.  30 天后:提醒通知
3.  7 天后:电话提醒/请求
4.  又一个 7 天之后:最后索款通知，3 天支付
5.  4 天后:一封解释你将如何从这里开始的信
6.  2 天后:打电话确认你正在采取进一步行动

现在，你的下一步行动可能是把它寄给一个讨债人，请你的律师介入，或者通过法院系统提出小额索赔。不管是什么，就去做，然后忘掉它。你现在要么收到钱，要么收不到。

使用讨债人或法庭的好处是会在客户的信用记录上留下印记。无论他们将来何时申请贷款，这个标记都可能阻止他们；我听说过有人两年后才得到付款，因为客户想要一份干净的信用报告，以确保他们获得购房贷款。

坚持你的债务回收，你会少一些头疼，有更多的时间投入到重要的工作中——你会得到报酬的工作。
