### 问题 2

随着 2010 年末的发布，Windows Phone SDK 包含了许多控件，这些控件使用了与整个核心平台相同的 Metro 设计语言。这些控件包括标准控件，如用于呈现文本的 TextBlock、用于显示图形的 Image 和用于捕获文本输入的 TextBox。然而，有一些遗漏，如允许用户选择日期和/或时间的控件，以及相当于大多数桌面应用程序中使用的下拉选择框。Windows Phone 有一套核心平台使用的清晰的控件，但 SDK 没有提供其中的大部分。直到[Windows Phone Silverlight Toolkit](http://silverlight.codeplex.com)发布，其中一些控件才变得可用。最近，这个工具包有了一个升级，包含了许多新的控件。

除了 Windows Phone SDK 附带的控件之外，您还在应用程序中使用了哪些控件，它们是如何让您的应用程序震撼人心的？
