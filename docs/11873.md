# 微软扼杀了 Linux 上网本吗？

> 原文:[https://www.sitepoint.com/microsoft-killed-linux-netbook/](https://www.sitepoint.com/microsoft-killed-linux-netbook/)

华硕在 2007 年底推出了他们开创性的 Eee 电脑。硬件可能很简单，但上网本的概念是革命性的。这款设备是一款真正便携、功能齐全的笔记本电脑，价格不比高端手机贵，只是同等尺寸个人电脑的零头。[这些机器非常适合在移动中工作的开发人员](https://www.sitepoint.com/6-reasons-to-buy-a-netbook/)。

Eee PC 最不寻常的地方是 Linux 操作系统。该操作系统基于 Xandros 发行版，但经过大量定制以支持硬件。尽管 Linux 存在风险，但对华硕来说，这是一个显而易见的选择:

*   操作系统是免费的——在一台 300 美元的电脑上，100 多美元的操作系统许可毫无商业意义。
*   有用的开源软件，如火狐和 OpenOffice.org，可以免费加入。
*   该操作系统的系统要求比 Windows Vista 低，Windows Vista 肯定不能在早期的 2GB 机型上运行*(相比之下，我用了 8 年的 512Mb 台式机启动 Ubuntu 9.04 的速度明显快于我用了 3 个月的 3GB Vista 笔记本电脑！)*
*   Linux 的要求比 Windows XP 低，微软正在减少对该操作系统的销售和支持。
*   Eee PC 是作为一种用户友好的互联网设备而不是笔记本电脑进行营销的。

20 秒的启动时间，吸引人的图形界面，以及一系列可用的软件好评如潮。儿童和新手用户喜欢这个系统，而极客可以访问底层操作系统或安装其他 Linux 发行版。Eee PC 的库存立即售罄，几个月后，每个主要的 PC 制造商都宣布了他们自己的基于 Linux 的上网本系列。Linux 的未来看起来很好。

微软不得不采取行动。Windows 是企业中利润最高的部分，保护他们的业务至关重要:

*   Windows XP 的寿命被延长至 2010 年。
*   更便宜的 XP OEM 许可证以 25 到 40 美元的价格发行*(条件适用于确保制造商不在全尺寸笔记本电脑上安装 XP)*。
*   定于今年晚些时候发布的 Windows 7 被修改为支持上网本规格。

我肯定微软销售团队也提供了优惠，比如批量折扣和微软作品的拷贝。然而，对于许多原始设备制造商来说，Windows 客户支持热线可能是最有吸引力的节省成本的选择。

XP 的重新引入产生了戏剧性的效果:

*   现在 90%的上网本都预装了 Windows。
*   MSI 声称，基于 Linux 的上网本回报率是 Windows 的四倍。尽管这一点一直存在争议，但我怀疑许多人确实会退回上网本，因为它不运行 MS Office 或他们最喜欢的应用程序。
*   英国主要的个人电脑零售商现在都拒绝库存基于 Linux 的上网本。
*   上网本的销量占笔记本电脑市场的 20%,而且这个数字还在上升。纯 Linux 设备会出现这种情况吗？我们永远不会知道。

你不能把他们的行为归咎于微软。他们是一家商业企业，他们在上网本上销售 Windows 的收入仍然低于台式机或笔记本电脑。虽然 OEM 厂商可以选择不安装 Windows，但微软成功阻止了 Linux 在 OS 市场获得更强的立足点。这可能会限制用户的选择，但考虑到这种选择，大多数人还是更喜欢他们熟悉的操作系统。

也许最大的问题是市场混乱。Windows 不可避免地导致了上网本规格的提高。你买的是高端上网本还是低端笔记本电脑？分类可能并不重要，但许多上网本正在演变成资源饥渴、脆弱且昂贵的个人电脑。原始 Eee PC 的流行证明许多人仍然想要一个简单可靠的设备来处理日常电子邮件和浏览。

您最近购买过上网本吗？你选择了 Windows 还是 Linux？你有选择吗？规格或价格比你想要的高吗？
