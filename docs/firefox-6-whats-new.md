# Mozilla 发布 Firefox 6——有什么新消息？

> 原文：<https://www.sitepoint.com/firefox-6-whats-new/>

是的，说真的。尽管还不到两个月，火狐 5 已经过时了。Mozilla 于 2011 年 8 月 16 日发布了 Firefox 6。正如你所料，有许多错误修复，但新功能列表相当轻量级。

## 这样更快

Mozilla 声称新浏览器比 Firefox 5 快 20%(Firefox 5 并不是特别慢)。对于 Linux 用户来说，速度的提升也是显而易见的。

Panorama 是 Firefox 的标签分组系统。我最初有一些保留，但我现在喜欢这个功能，并广泛使用它。Firefox 6 按需加载非活动组——它改善了启动时间并减少了内存使用。

## URL 荧光笔

火狐 6 在地址栏*高亮显示了一个站点的域名(抱歉 Mozilla，我还是没办法让自己称之为牛逼栏！所有竞争浏览器都有这个功能。*

## 网站权限控制

浏览器让你控制诸如 cookies、JavaScript 限制、弹出窗口、密码存储等权限。在全局或逐页的基础上。然而，你不太可能和你的网上银行应用程序有着和脸书一样的关系。

新的数据管理窗口允许您设置基于站点的权限，以便将相同的规则应用于特定域中的每个页面。该功能是试验性的，但是您可以通过在地址栏中输入“关于:权限”来访问它。

## JavaScript 便签簿

新的 Scratchpad 控制台允许您执行 JavaScript 代码。它可以访问当前页面中的每个对象和变量，但是新的暂存变量不会漏入其中。

Scratchpad 类似于 Firebug 的控制台命令行，但是，它不需要附加组件，并允许您保存代码片段供以后使用。

此外，如果你喜欢 Web 控制台，你会很高兴听到它可以被移动和停靠在其他地方。

## 更多 HTML5、CSS3 和 DOM 可爱之处

Web 开发者将能够在 Firefox 6 中采用以下 HTML5 技术:

*   添加了图形进度条的`progress`标签。可以用`::-moz-progress-bar`伪元素来设计颜色。
*   一个新的`-moz-orient`属性允许你设置一些元素，比如进度条，为水平或垂直方向。
*   支持`track`元素，因此您可以将文本轨道应用到原生音频和视频。
*   CSS3 边界半径可应用于`iframe` s。
*   定制数据属性(data-*)现在可以通过 DOM 的`element.dataset`属性来访问。
*   诸如下划线、上划线和删除线之类的文本装饰现在可以在 CSS 中使用`-moz-text-decoration-line`(无、下划线、上划线、删除线)、`-moz-text-decoration-style`(继承、实线、双线、点线、虚线、波浪线)和`-moz-text-decoration-color`属性来设置样式。
*   新的`-moz-hyphens`属性允许您控制换行时如何处理单词的连字符(无、手动、自动)。
*   可以使用`window.matchMedia()`方法以编程方式评估媒体查询字符串的结果。
*   对[服务器发送事件](http://dev.w3.org/html5/eventsource/)的早期支持已经实现。这个 API 允许来自服务器的 DOM 事件形式的推送通知。
*   [触摸事件](https://www.w3.org/TR/2011/WD-touch-events-20110505/)支持触摸屏和触控板。

## 火狐 6 会破坏我的插件吗？

可能吧。我的几个插件在过去几周才开始在 Firefox 5 中工作。我怀疑火狐 6 会导致更少的问题，但你应该在升级前检查基本插件的兼容性。

火狐 6 现已上市。你可以在你的浏览器中升级(帮助>关于火狐)或者从 Mozilla 网站下载。
