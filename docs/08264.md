## jQuery Canimate 插件

一个 jQuery 插件，通过快速打印图像文件来实现除 gif 之外的图像文件的现场动画。你可以很容易地改变帧速率；此外，您仍然可以使用保存图像的元素做任何事情(例如添加边框)。

 [![jQuery Canimate](../Images/a3a583127850250e1eca5fcb03f20235.png)](http://isthatclear.com/jquery/canimate/) 
[源+演示](http://isthatclear.com/jquery/canimate/)

## jquery.pngFix.js

这个插件将修复 Windows Internet Explorer 5.5 & 6 中缺失的 PNG-Transparency。
功能:
>不引人注目的脚本，设置简单
>现在也可以使用 CSS-背景(但缩放背景)
>可以使用 PNG-链接内的图像
>标题、ALT、类和样式-属性被考虑在内

 [![jquery.pngFix.js](../Images/6c619174509334a737ab7673b8078137.png)](http://jquery.andreaseberhard.de/pngFix/index.html) 
[源+演示](http://jquery.andreaseberhard.de/pngFix/index.html)
