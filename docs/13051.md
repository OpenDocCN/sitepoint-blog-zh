# 。3 月 10-17 日:亚音速岩石和 MVC 是 Hawt

> 原文:[https://www . site point . com/net-on-the-net-March-10-17-subsonic-rocks-and-MVC-is-HAWT/](https://www.sitepoint.com/net-on-the-net-march-10-17-subsonic-rocks-and-mvc-is-hawt/)

这是比“本周初”晚一点发布的不过话说回来，我得到了迁移一些 PHP 应用程序的有趣工作(不幸的是，这是我自己的设计，所以我不能指责)。我从来没想过我会错过死亡黄屏，直到我撞上了死亡黑屏。没有 IIS7，所以没有 FREB love，只是有很多感觉，print_r()，SQL Server Profiling 和 on the fly code。我不怀念过去的日子。现在来看看 dotnet。。。

头条新闻是 Rob Conery 一直忙于后期混音。何[发布亚音速 2.1 Beta 2](http://blog.wekeroad.com/2008/03/11/subsonic-21-beta-2-available-now/) 并推出新的[亚音速项目现场](http://blog.wekeroad.com/2008/03/13/subsonic-new-project-site-online/)。此外，如果你错过了，SP1 视窗 Vista 已经发布。掷骰子，今天就升级！

ASP.NET MVC 是新的哈维。而且，当你在思考 MVC 的时候，你应该看看 Stephen Walther 关于 ASP.NET MVC 请求的[生命](http://weblogs.asp.net/stephenwalther/archive/2008/03/17/asp-net-mvc-in-depth-the-life-of-an-asp-net-mvc-request.aspx)的帖子，以及杰弗瑞·帕尔梅罗关于 ASP.NET MVC 控制器的单元测试[的帖子。最后，不要错过菲尔·哈克的](http://codebetter.com/blogs/jeffrey.palermo/archive/2008/03/09/this-is-how-asp-net-mvc-controller-actions-should-be-unit-tested.aspx) [MVC 路由调试器](http://haacked.com/archive/2008/03/13/url-routing-debugger.aspx)。

本周也有一些值得注意的发布。米格尔·德·伊卡萨宣布发布 [MonoDevelop 1.0](http://tirania.org/blog/archive/2008/Mar-14.html) 。Telligent 工作人员还发布了[涂鸦 CMS 服务包 1](http://graffiticms.com/blog/1-0-sp-1/) 。Thycotic 的 Jonathan Cogley &公司[发布了 SecretServer 4.1](http://weblogs.asp.net/jcogley/archive/2008/03/15/secret-server-4-1-goes-live.aspx) 。在“与其说是一个发布，不如说是我刚刚发现的一个真正整洁的产品”专栏[中，请查看 LINQPad](http://weblogs.asp.net/dwahlin/archive/2008/03/17/want-to-learn-how-to-use-linq-check-out-linqpad.aspx) 中的随机 LINQ 任务。这是一个非常灵活的工具，可以加快发现周期。

在“易于理解的代码”方面，我们有两个提交。首先，威尔·史密斯分享了一些方便的视图状态扩展方法。其次，对于那些正在经历痛苦的 PayPal 整合的人，请参见 [Anthony Grace 的解决方案](http://www.codersbarn.com/post/Solution-to-ASPNET-Form---PayPal-Problem.aspx)。

在“不那么技术化但仍然值得了解的类别”中，一定要看看 Kyle Baley 关于[人类生成的 id](http://codebetter.com/blogs/kyle.baley/archive/2008/03/11/auto-generated-vs-human-generated-ids.aspx)的帖子。虽然电脑更酷，但不幸的是，人类正在支付你的薪水，软件必须与湿件一起工作。

回到篮球锦标赛。去吧。
