##### ColorPic

专业的滴管工具当然不缺乏，但很少有人像 Nico Westerdale 的 [ColorPic](http://www.iconico.com/colorpic/index.aspx) 一样将所有正确的功能整合在一起。

如果你刚开始一个项目，除了一个标志或一张照片，ColorPic 绝对是一块宝石。使用方便的缩放窗口进行近距离的个性化操作，用快速的 CTRL-G 抓住主要颜色，然后从反锯齿边缘中挑出“桥接”颜色。

每组芯片都可以作为“调色板”保存，因此您可以在几秒钟内轻松调用归档的项目颜色集。所有颜色都有 RGB 和十六进制。

这款应用程序是在尼科位于 Iconico.com 的网站上免费提供的，但他也提供了其他同样出色、价格低廉的软件供你试用。毫无疑问，你会注意到出色的产品，屏幕卡钳，这也是值得一提的。完整版仅 15 美元，包括透明度选项、皮肤和多种测量单位。抓住他们，至少给他写一封漂亮的感谢信！

##### Mirekw.com 拍摄的 MWSnap

截屏？旧的打印屏幕按钮有什么问题？什么都没有……但是 [MWSnap](http://www.mirekw.com/winfreeware/index.html) 是一款制作精美的小应用程序，可以让你更快、更容易地继续你已经在做的工作。一切都是为了轻松和方便。

最重要的是，MWSnap 允许您更容易地瞄准屏幕的特定区域，无论是图形的一小部分、单独的菜单、完整的窗口、整个应用程序视图还是整个桌面。图像可以被直接捕获到你的剪贴板上，并在几秒钟之内转换到你最喜欢的图像编辑程序中。或者，您可以毫不费力地将几十个图像同时捕获到一个目录中，并为每个图像自动生成文件名(非常适合站点演示)。

这可能是您从 MWSnap 中所需要的全部，但是它也提供了一系列有用的小部件，包括标尺、可编程热键、虚拟光标(非常好！)、颜色选择器、裁剪和旋转工具以及信息面板。

MWSnap 是一个非常有用的小应用程序，通过非常有才华的 Mirek Wojtowicz 的良好风度，为您的设计乐趣而提供。

##### BradSoft 的 TopStyle Lite

回到原始互联网的泥潭中(嗯，实际上是 90 年代中期)，尼克·布拉德伯里厌倦了他正在使用的 HTML 编辑器，决定坐下来好好写自己的东西。HomeSite 继续成为 HTML 编辑的早期成功故事之一，直到它最终被 Macromedia Megaship 所包含。尼克溜出了 HomeSite，但继续做他最擅长的事情——为新兴技术编写漂亮的小编辑器。在这种情况下，它是一个漂亮的小 CSS 编辑器，名为 [TopStyle](http://www.bradsoft.com/topstyle/) 。

现在，尼克提供了两种风格的 TopStyle:一个“Pro”和一个“Lite”版本。

![1379_tslite](../Images/c31a5f8f913309a1eeed73a8db3fbe8d.png)

TopStyle Pro 是一个完整的包，能够对不仅仅是 CSS 进行高级编辑。大约 80 美元，对于每天花大量时间使用 CSS 的人来说，它代表了真正的价值。仅浏览器配置文件就值入门价。

但我们来这里是为了免费的东西，对吧？我们来谈谈 TopStyle Lite。

你需要一个单独的 CSS 编辑器做什么？Dreamweaver 等产品中的内联自动完成功能为您提供了所需的所有选项。也许是这样，但是他们会让你做以下事情吗？

*   即时检查各种浏览器和规格的属性支持
*   通过在处理复杂工作表时展开和折叠规则，轻松管理它们
*   使用强大的样式检查器逐行构建规则
*   预览即时更改。

我不得不承认，当我开始认真使用 CSS 布局时，这个小应用程序比我能找到的任何其他东西都更快地让我通过了学习曲线。书籍、在线资源和备忘单都很棒，但我花了一个月心不在焉地翻阅 TopStyle Lite 中的样式检查器面板，让我迅速尝试了我从未听说过的 CSS。用这种方式不需要很长时间就可以熟悉 CSS。

总而言之，即使你从未决定花额外的钱来获得专业版的好东西(包括最著名的“分屏，多浏览器预览”，“风格清扫器”，验证器和“HTML Tidy”集成)，TopStyle Lite 可能会在未来几年为你提供很好的服务。

点击下载 Topstyle Lite [。](http://www.bradsoft.com/topstyle/)

*   禁用样式
*   显示表单详细信息
*   启用自动完成

*   不带 Alt 标签的轮廓图像
*   大纲表格单元格
*   调整到 640×480
*   调整到 800×600
*   调整到 1024×768
*   验证 CSS
*   验证 HTML
*   验证第 508 节的可访问性
*   验证 WAI 可访问性
*   查看源代码

令人印象深刻。但是，仅仅过了 12 个月多一点，WDT 现在的功能增加了四倍多，而且都是杀手级的——没有填充物！

对于我来说，在这里有限的空间里，有太多太多的功能来公正地评价当前版本的 WDT。所以，这里是我个人最喜欢的几个:

*   编辑 CSS —当您观看时，在浏览器中实时更改任何站点上的 CSS
*   禁用 JavaScript/CSS/Java，而不必费力地浏览工具/选项菜单
*   一键生成定制的站点速度报告
*   显示所有图像的内嵌尺寸
*   显示 ID 和类细节(对于 CSS 调试非常方便)
*   没有 ALT 属性的轮廓图像—一眼就能发现您的可访问性错误
*   7 个不同的代码验证器，外加一个你可以添加自己的定制验证器的工具(可能运行在本地网络上)

Chris 甚至设计了一个令人惊叹的 1-click“拓扑视图”,它以图形方式显示了页面上每个元素的“嵌套”——嵌套越深，色调越浅。太神奇了。

我只想说，大约每隔几个月，SitePoint 团队的一名成员就会下载 Chris 的最新产品，当我们测试新功能时，我们会有一个集体的“Bill 和 Ted”时刻——“哇，哇，太棒了！”(众人会意的点头)。

##### Accessibility 的可访问性工具箱

对所有这些可访问性的东西感到有点内疚？我也是。多年来，我的表单和弹出窗口代码一直非常流行，但是突然之间，人们开始对我的代码翻白眼，并在走廊里对我窃笑。好吧，也许那是咖啡在说话，但是你知道我从哪里来。你可能想改变自己的方式，但你不确定目前的状态到底是什么。

输入:伊恩·劳埃德和他的可访问性工具箱。伊恩从他著名的[Accessify.com](http://www.accessify.com)网站上收集了一些积累的智慧，并将其提炼到这个有用的小代码向导中。在这一点上，它由一个“可访问的表单元素生成器”和一个“可访问的弹出窗口生成器”组成。有“可访问的表格”和“可访问的表格”的按钮，但没有任何消息表明这些是否会很快到期(当我们说话时，Ian 正在全球各地愉快地闲逛，所以没有屏息)。

![1379_accesstoolbox](../Images/bb0d5e1f64589cdb829f11e5d799cd32.png)

没关系。所提供的两个工具只需点击几下就能提供您需要的工作代码。

弹出式代码被设计成在 JavaScript 可用时利用它，而不会在它不可用时悲惨地失败。代码还足够聪明，可以在用户试图重新启动时，将一个“散焦”的、丢失的弹出窗口带回前端位置——当它没有被充分考虑时，这是一个特别令人沮丧的失败(向[Creatas.com](http://www.creatas.com/)致敬)。

表单元素构建器是摆脱传统表格布局表单的好方法。使用这个小向导制作的“标签和输入对”，洒上一些美味的 CSS (SitePoint Tech Times，第 58 期< )，你就可以在几分钟内拥有低脂、营养和开胃的表单。

请记住，自 2003 年初以来，伊恩一直孜孜不倦地开发 Accessify.com 的所有工具和向导，而且没有要一分钱。由于他已经辞去了最近的工作，花了一年时间去旅行，我几乎不需要指出，如果你发现他的工具对你的工作有帮助，他会感谢你的任何 Paypal 捐款。
