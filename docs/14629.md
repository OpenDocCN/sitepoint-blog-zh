# Windows 窗体 v2 中有什么

> 原文:[https://www.sitepoint.com/whats-coming-in-windows-forms-v2/](https://www.sitepoint.com/whats-coming-in-windows-forms-v2/)

Windows 窗体团队[发布了一些下一版本 Windows 窗体中一些新控件的截图](http://www.windowsforms.net/WhidbeyFeatures/default.aspx?tabindex=4)。

一目了然:

1.WebBrowser:我们终于得到了 IE ActiveX 控件的托管包装器！

2.活动文档宿主:允许托管 OLE 文档。这意味着你可以在任何地方使用编辑软件，比如 Word，Excel 或者 Photoshop。整洁！

3.音效播放器。我最初对此感到兴奋，认为这是 Windows 媒体控制的下降。但是，不，它只是播放一个 WAV 文件。Humpft:(

4.DataGridView。作为一个更加精细的数据网格，DataGridView 具有更好的设计时预览、更好的样式和更丰富的单元格类型。看起来不错。

5.SplitContainer 传统 Splitter 控件的一个简洁版本。把它想象成一个分成两个面板的面板，中间有一个分割线。或者只看漂亮的图片；)

6.MaskedTextBox。是时候了。NET 得到了一个本机屏蔽文本框。非常类似于 VB6 中的 MaskEd，或者 Delphi 中的 MaskedEdit。

7.ToolStrip 最后，体面的 UI 组件与 Office 2003 风格。状态栏，菜单，工具栏，上下文菜单，都可以自定义和拖放。非常好。
