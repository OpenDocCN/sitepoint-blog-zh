# 宣布一系列新的创业基础知识

> 原文：<https://www.sitepoint.com/business-start-up-basics/>

我写了很多关于营销、网络和各种推广你的业务的方法。一旦你建立，与分包商合作，提供优质的客户服务，扩大你的客户群。

我最近收到一些读者的来信，他们对创业感兴趣，但不知道如何应对一些最基本的行动。他们正处于为自己工作或想要建立自己的企业的初级阶段，但不知道从哪里开始。

这个新系列将包括每周一篇关于创业基础的文章。它将作为一份创业清单，带你从头开始经历创业和开始为自己工作所需的每一个步骤。

以下是我计划涵盖的主题概述:

*   [确定企业所有权是否适合你](https://www.sitepoint.com/should-i-start-a-business/)
*   [商业计划](https://www.sitepoint.com/business-start-up-planning/)
*   [为你的企业命名](https://www.sitepoint.com/how-to-name-your-business/)，加上 [10 条建议](https://www.sitepoint.com/tips-for-choosing-perfect-company-name/)
*   [企业所有权的法律效力](https://www.sitepoint.com/legalities-of-starting-a-business/)
*   [企业财务和预算](https://www.sitepoint.com/how-to-get-control-of-your-business-finances/)
*   [建立家庭办公室](https://www.sitepoint.com/how-to-set-up-a-home-office/)
*   [知道什么时候该投入全职工作](https://www.sitepoint.com/checklist-for-taking-the-plunge/)

这个系列的目标不是提供一个全面的创业和所有权指南，而是给出一个介绍，并列出你在上面列出的每个领域需要考虑的一些问题。

在每篇文章的结尾，我会包含其他文章、资源和工具的链接，你可以在那里获得更多信息，并在这个过程中采取进一步的步骤。和往常一样，欢迎对你自己的经历提出问题和评论。希望这次讨论能为所有读者提供更多的见解。

在这个系列中，你还希望看到其他的主题吗？现在是大声说出来的时候了！
