# 10 个非常省时的 CSS 清理工具

> 原文:[https://www.sitepoint.com/terrific-time-saving-css-tools/](https://www.sitepoint.com/terrific-time-saving-css-tools/)

拥有整洁有序的 CSS 有很多好处。一个拥有完美 CSS 的网站将会加载得更快，在搜索结果中出现得更突出，并为未来的客户提供一个你的专业性的有力证明。从日常访问者到最挑剔的设计师同事，每个人都将从你勤奋、干净的 CSS 标记中受益。

但是，从无数的设计迭代和变更中清理 CSS 可能是一个乏味、耗时的过程。你可能不想承担这种单调乏味的工作，你的客户也可能不想为这种提供如此微妙好处的耗时任务付费。从设计师的角度来看，手动清理 CSS 可能足以摧毁创造力，将你的注意力从工作所需的更高层次的思考上转移，并可能开始让你不那么喜欢你的职业。

幸运的是，有一些出色而有用的工具可以帮助你自动化 CSS 的一些更平凡的方面，并释放你的神经元来处理网页设计中更有创造性和更有趣的方面。以下工具将为您提供一个高效、优化、组织完美的 CSS 样式表，您的 web 服务器、您当前的客户和您未来的潜在客户都会非常欣赏它。

 [### 处理器](http://procssor.com/) 

[![aaprocssor](../Images/d707ec861c5527fd8719b8bed5d2141b.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aaprocssor.jpg)

ProCSSor 是一个精简的、实用的工具，可以对你的 CSS 进行即时改进。可通过 OS X、iOS 或任何浏览器使用，该工具获取 CSS 文件、粘贴的片段或 URL，并立即美化、格式化和验证您的 CSS。

 [### CSS 功能区](http://csslint.net/) 

[![aacsslint](../Images/2727cfa8fa20b8093d982131ed43a37e.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aacsslint.jpg)

与其他 CSS 清理工具相比，CSS Lint 提供了更有指导意义的体验。大多数人不会告诉你他们所做的 CSS 更改背后的基本原理，但是 CSS Lint 会给你一个简明的解释来解释每一个推荐的更改。它有相当多的功能集中在错误、兼容性、性能、复制和可访问性上，所有这些都可以随意启用或禁用。

 [### 肮脏的标记](http://www.dirtymarkup.com/) 

[![aadirtymarkup](../Images/0587bc7cb0b34363cfbb89e1615b152e.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aadirtymarkup.jpg)

脏标记有一种独特的代码优化方法；它作为一个集合体，结合了几种不同代码清理技术的力量( [HTML Tidy](https://github.com/w3c/tidy-html5) 、 [CSS Tidy](http://csstidy.sourceforge.net/) 、 [JS 美化](https://github.com/einars/js-beautify)、 [Ace 编辑器](http://ace.ajax.org/))，对任何提供的代码进行全面清理。正如您可能会怀疑的那样，它在 CSS 上的效果和在 JavaScript 或标准 HTML 上一样好。

 [### CleanCSS](http://www.cleancss.com/) 

[![aacleancss](../Images/3e06e7138282d48ddcb8958041866f44.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aacleancss.jpg)

CleanCSS 接受 URL 或粘贴的代码片段，并提供多种处理方式。您可以从五种不同的压缩程序设置中进行选择，这些设置在可读性和文件大小之间进行权衡。

 [### 代码美化者](http://www.codebeautifier.com/) 

[![aacodebeautifier](../Images/111f2ac390f21a14a4445038851763b6.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aacodebeautifier.jpg)

Code Beautifier 是一个简单的 CSS 清理工具，没有多余或琐碎的功能。它接受粘贴或链接的 CSS，并根据一系列实用的选项仔细清理它。如果你希望专注于手头的项目，而不是迷失在铺天盖地的功能海洋中，这可能是你理想的 CSS 清理工具。

 [### Spritemapper](http://yostudios.github.io/Spritemapper/) 

[![aaspritemapper](../Images/527f44ae015b13f0d962609e0b101498.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aaspritemapper.jpg)

其他提到的工具都没有深入到图像优化领域，这与代码本身的优化一样有益。Spritemapper 从一个 CSS 样式表中接受多个图像，将它们合并成一个图像，并为每个图像的相应“sprites”或图像片段生成适当的 CSS 定位。结果是更快的站点、更少的 HTTP 请求和更精简的样式表。

 [### 夹大衣](http://topcoat.io/) 

[![aatopcoat](../Images/6a3f7700dd58d16b1f27bc1f835536b5.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aatopcoat.jpg)

Topcoat 不是传统的 CSS 清理工具；它本质上是一个灵活的 CSS 开发框架。如果你从兼容的、写得好的 CSS 开始，你可能永远都不需要清理巨大的 CSS 混乱。因此，从面漆开始可能是一个更积极的，预防性的方法来处理干净的 CSS。

 [### CSSTidy](http://csstidy.sourceforge.net/) 

[![aacsstidy](../Images/0bc909ec4867a9d548d5a3e0388a2b8f.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aacsstidy.jpg)

CSSTidy 类似于其他几个 CSS 清理工具，但它确实提供了一些独特的属性。CSSTidy 可以通过命令行或 PHP 调用，它可以在 Windows、Mac 或 Linux 平台上使用。如果你有一个建立在这些技术之上的开发系统，或者同时管理大量的项目，CSSTidy 是一个清理工具，它可以很好地适应你现有的工作流程，并自动保持你的 CSS 整洁。

 [### 样式整洁](http://www.styleneat.com/) 

[![aastyleneat](../Images/1be55e8d795ca57a73793aee42b8da10.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aastyleneat.jpg)

如果你是仍然喜欢修补 CSS 的一丝不苟的类型，或者是不想完全信任工具的谨慎类型，Styleneat 可能是你的 CSS 清理工具。虽然它不提供缩小或压缩，但 Styleneat 的输出是最友好的，包括重新排序的属性、排序的选择器和缩进，从人的角度来看，这使得样式表非常干净。

 [### W3C 验证程序](http://jigsaw.w3.org/css-validator/) 

[![aaw3cvalidator](../Images/603722e70566ed33a544e08a88c74ac5.png)T2】](https://www.sitepoint.com/wp-content/uploads/2013/09/aaw3cvalidator.jpg)

没有提到 W3C Valdiator，CSS 工具的集合就不完整。虽然这并没有提供工具来简化、压缩和优化你的代码，但是在你的 CSS 有效性方面，它肯定是应该被考虑的。虽然前面提到的其他工具各有千秋，但这个工具是由 W3C 管理机构支持的。
