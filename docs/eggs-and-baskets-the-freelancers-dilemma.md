# 鸡蛋和篮子:自由职业者的困境

> 原文：<https://www.sitepoint.com/eggs-and-baskets-the-freelancers-dilemma/>

我的一个自由职业朋友最近非常关注一个非常好的客户，这个客户非常棒。但上周，另一个较小的客户带着一个简短的项目出现了。他是一个人，一天只有几个小时。那么，他应该拒绝小客户，专注于可靠的大客户吗？或者，他应该避免把所有鸡蛋放在一个篮子里:他应该也接纳小客户，但冒着让他们两个都失望的风险吗？

这是我们许多人都面临的困境，尤其是当选择变得越来越少的时候。我们应该为了不孤注一掷而破坏与大客户的关系吗？或者，我们应该专注于培养最好的、最赚钱的客户，并希望他们坚持与我们合作？

**一篮子神话**

我倾向于认为，为一个客户做自由职业者并不比受雇于一家公司风险更大。你可能会说，组织留住员工的时间最长，当需要裁员时，承包商是第一个离开的。但是，如果经济状况良好，在这两种情况下，你都可能很快失业。

或许一个同样重要的考虑因素是为一家公司工作背后的哲学:许多自由职业者自己创业，因为他们不想依赖一种收入来源；其他人这样做是为了享受自由职业通常带来的多样性。但是，如果这些问题没有困扰你，你继续关注你的一个好的大客户可能会很好。

**打碎多个鸡蛋**

当有人拿着一件小作品走近你时，你永远无法知道的一件事是，它会带来多远。毕竟，大事是从小处开始的。所以今天放弃一小部分工作，从长远来看，你可能会失去更多的工作。

当然，还有分散自由职业者风险的问题，这不仅仅是从多个来源获得工作，所以你总是有事情要做——这也是关于有多个可以寻求汇款的人，如果你的一个客户突然发现她/他无法支付，这可能非常有价值。如果这听起来像你的方法，你可能会尽你所能去适应这个小客户。也就是说，如果在工作开始的时候你就已经担心你要给它多少时间，那么你可能会因为接受它而走向灾难。

**选项**

我会假设一个月 24/7 工作不是一个选项！如果你面临这种困境，你能做什么？以下是我认为的选择。

***拒绝***
礼貌地拒绝工作，如果你知道的话，建议一个替代的承包商。是的，你可能会错过一份可能带来更大成就的小工作，但是你也减少了让你的客户失望的机会。

***Flex***
没错:老时间框架壁球。你或你的客户可以修改他们的时间表，这样你就可以和他们一起工作了吗？如果是这样的话，你可能愿意付出艰苦的努力来完成这两个项目。

***外包***
把小工作外包给另一个承包商。如果你以前没有做过，并且你不知道谁有你需要的技能，这种方法可能不太可能节省你的时间。但是，如果你知道有人能胜任这项工作，并且你知道你可以依赖他们，那么把一个项目外包给他们(或者两个人一起工作)可能会让你最大限度地留住两个客户。

我喜欢自由职业者提供的工作范围，所以大多数时候，我发现自己在变通:在时间框架和客户期望之间周旋，用坚定的外交手段完成一切(同时梦想着下一个平静期)。你呢？
