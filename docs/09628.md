# 告诉我们你想学什么，拍个傻逼照，(希望)赢！

> 原文:[https://www . site point . com/tell-us-what-you-want-to-learn-take-a-pisty-photo-and-hope-win/](https://www.sitepoint.com/tell-us-what-you-want-to-learn-take-a-stupid-photo-and-hopefully-win/)

在过去的几周里，我一直(有点过度地)对一个想法感到兴奋，这个想法是我为 Twitter 竞赛准备的，现在是时候向全世界发布了。

基本前提是让你告诉我们你想学什么，同时让自己出丑。为了做到这一点，我们想让你用任何你想要的东西——食物、物品、人、你自己，等等——来拍摄包含 SitePoint 徽标的可笑照片，以证明这有多简单——我自己就拍过一张照片——但我毫不怀疑你能做得更好。事实上，我的猫可能会做得更好(他甚至可能会把标志弄对，正如你所看到的，他穿着 SitePoint t 恤看起来很棒)。

![SitePointers doing the SitePoint sign](../Images/4634bb0fce1ee571785fba18d54bc5fd.png)

![Cat in a SitePoint shirt](../Images/e391ff3386d11d96cf98022bd18198a7.png)

然后，我们希望你上传那张照片，作为推文的一部分，告诉我们你想了解什么，以激起你朋友的好奇心。唯一的规则(除了附上照片之外)是**你必须在那条推文中的某个地方散列标签#sitepoint，才有资格**。

比赛将持续 5 天，每天最有趣/最聪明/最有想象力的照片将获胜。判断完全基于我个人的判断。(我不受贿，但我确实觉得愚蠢特别有趣。)

获奖者可以选择 SitePoint 课程和 SitePoint 书籍(印刷版或电子版)。为了找到你想学的东西和更多关于 SitePoint 课程的信息，请查看我们的姐妹网站 [Learnable](https://learnable.com/sitepoint) 。如果你赢了，你可以选择的书的细节，看看我们的[书店](https://products.sitepoint.com/#/?tag=books&filters%5Btag%5D%5B%5D=&filters%5Bdifficulty%5D=&simpleform_submit_marker=showme)。

**所以，总结一下** :

*   以一种有趣的方式拍摄一张包含 SitePoint 徽标的照片
*   在推特上发布那张照片，告诉我们你想在[学习什么](https://learnable.com/sitepoint)
*   tweet 中散列标签#sitepoint
