# 麦金塔工具包

> 原文:[https://www.sitepoint.com/macintosh-toolkit/](https://www.sitepoint.com/macintosh-toolkit/)

阿尔法–阿尔法是一个强大的文本编辑器，它拥有编辑大量文本文件所需的所有功能:多文件搜索和替换、文件集等。

[Astound web motion](http://www.astound.com/)–Astound web motion 允许任何人创建流媒体、Java 驱动的动画，无需插件即可观看。使用它来创建导航栏、闪烁的横幅、旋转的徽标和其他基于 Web 的图形，包括交互性和声音。

BBEdit 是世界上最好的 HTML 编辑之一。如果你不需要很多花里胡哨的东西，这就是你要的编辑器。一个伟大的基于文本的 HTML 编辑器，适合那些不怕原始 HTML 的人。这个 HTML 编辑器被许多专业的网页设计公司使用。

Eudora–一个极其强大的电子邮件客户端，拥有出色的过滤器、多个电子邮件帐户管理和大量极其有用的功能。我们 Sitepoint.com 公司使用这个程序的 Windows 版本来处理我们每天收到的数百封电子邮件。

[Fetch](http://www.dartmouth.edu/pages/softdev/fetch.html)–简单来说，这是 Mac 上最好也是最容易使用的 FTP 客户端之一。

Freeway–一款广受好评的所见即所得的 Mac HTML 编辑器，拥有类似桌面出版的熟悉界面。

[graphic converter](http://www.lemkesoft.de/GraphicsConverter)–graphic converter 是一个将图形转换成不同格式的程序。它还包含许多有用的图形处理功能。

Wallaby 是一个非常容易使用的麦金塔 HTML 编辑器。它是为有经验的网页设计师和初学者设计的。它结合了一个非常方便的用户界面和许多功能，包括标签着色，在多个窗口中拖放编辑等等。

page spinner–page spinner 是一款易于使用的 Mac HTML 编辑器，支持 HTML 3.2、HTML 4.0 以及附加的 Netscape Navigator 和 Internet Explorer 扩展。对于初学者和更高级的 Web 作者来说，这都是一个很棒的编辑器。
