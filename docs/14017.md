# Macromedia Studio 8 预览版

> 原文:[https://www.sitepoint.com/macromedia-studio-8-preview/](https://www.sitepoint.com/macromedia-studio-8-preview/)

开始节省你的便士吧，网页设计师们！Macromedia 今天[宣布](http://www.macromedia.com/software/studio)即将发布其旗舰产品的全新版本: [Dreamweaver 8](http://www.macromedia.com/software/dreamweaver/) 、 [Flash Professional 8](http://www.macromedia.com/software/flash/flashpro/) 和 [Fireworks 8](http://www.macromedia.com/software/fireworks/) 。一如既往，它还会将这些与之前可用的 [Contribute 3](http://www.macromedia.com/software/contribute/) 和 [FlashPaper 2](http://www.macromedia.com/software/flashpaper/?promoid=BINU) 一起捆绑在 [Macromedia Studio 8](http://www.macromedia.com/software/studio/) 中。由于慷慨的升级定价政策，首次使用 Macromedia 的用户可以从这个版本中获得真正的优惠。

在 Dreamweaver 8 中，新版本的重点是改进对基于 CSS 的设计的支持。Macromedia 改进了 CSS 样式面板，以更好地显示应用于文档中所选元素的规则和属性，并为现有规则添加新属性。精通 CSS 的设计者可以打开一种模式，按照层叠的顺序列出应用于所选元素的每个规则。

![](../Images/268b4f94fc0ca5b932697b3cbf3e2264.png)

WYSIWYG 视图在这个版本中也得到了一些额外的关注。现在，它清楚地显示了用交叉阴影线应用于所选元素的填充和边距，进一步帮助您可视化严重依赖这些功能的 CSS 布局。

![](../Images/69bc4ab2e90c072a6c116435ae7f0af4.png)

尽管 Macromedia 在此版本中大力宣传显著改进的 WYSIWYG 渲染，但某些 CSS 布局功能仍会给它带来一些麻烦。例如，从 Beta 2 开始，使用 CSS 固定定位，即使被黑客攻击到在 Internet Explorer 6 中工作，也会破坏 Dreamweaver 的显示，将 position: fixed 解释为 position: static，而不是像浏览器那样忽略属性定义(或者像 IE 那样将其视为 position: absolute)。这可能会在最终版本中得到解决，但总的来说，我认为 Dreamweaver 8 的 WYSIWYG 视图足够好，可以在今天的浏览器中做 CSS 可以做的任何事情，而无需黑客攻击。

当我问她关于所见即所得的渲染改进时，Dreamweaver 产品经理 Jen Taylor 说 Macromedia“在这个版本中非常注重渲染，整个开发人员只致力于渲染改进。我们的呈现表面的目标是呈现大多数用户将如何体验页面，直到最近，一直是 IE。对于这一版本，我们开始了向更符合标准的渲染表面过渡的过程。”但她承认，所见即所得的渲染仍然不完全兼容。“我们和浏览器都没有达到这一步。渲染是一件棘手的事情，因为不同的浏览器渲染的东西不同。我们无法达到像素完美，因为没有浏览器可以做到。但是，我们努力。”

Dreamweaver 8 还包括 XSLT 转换的可视化创作，以便将动态 XML 数据(如 RSS 提要)显示为网页或嵌入到另一个网页中。Dreamweaver 8 包括用于客户端和(更强大的)服务器端 XSL 转换的工具。这是对程序的一个令人印象深刻的补充，它可能会被不了解 XML 的设计者发现。

该软件的其他良好改进包括直接导入 Flash 视频(FLV)文件，支持 ColdFusion MX 7 和 PHP 5 等更新的服务器平台，能够在所见即所得显示中缩放和设置指南，新的编码工具栏，代码折叠支持，以及我个人最喜欢的后台文件传输。

尽管 Macromedia 将其吹捧为 Dreamweaver 产品历史上最重要的更新，但我个人认为这只是另一个自然的进化步骤。

Flash Professional 8 在此版本中也得到了类似的广泛更新，尤其是对 Flash Player 8 支持的令人眼花缭乱的新功能的创作支持(我在最近一期的 Tech Times 中对此进行了讨论)。除了这些新的创作功能和效果，Flash 8 还拥有一个新的、改进的视频编解码器，大大提高了在 Mac OS X 上的性能，并为其绘图工具提供了一个可选的“Illustrator/Freehand mode”。

虽然不像它的大兄弟那样广泛，但 Fireworks 8 中的更新同样受欢迎。对构建基于 CSS 的菜单的更好支持，以及大约 15 种新的混合模式名列前茅。

值得注意的是，Macromedia Studio 的这个版本中首次缺少了该公司的矢量绘图产品 Freehand。虽然 Macromedia 的公司路线是继续支持现有版本的 Freehand 并看到其成功，但面对 Macromedia 即将被竞争对手 Adobe 收购，该产品的日子显然屈指可数，Adobe 的 Illustrator 产品使用更广泛，并被视为行业标准。

Macromedia 将在 Studio 8 中捆绑其其他网页设计产品 Contribute 和 FlashPaper 的现有版本，而不是 Freehand。Contribute 为非技术用户提供了一个友好的界面来编辑网站的静态 HTML 部分，而不会破坏其设计。FlashPaper 允许您将任何文档转换为 Flash 电影，可以嵌入到网页中进行查看和打印，就像一个更轻量级和网络友好版本的 Adobe PDF。

应该注意的是，这两种产品的捆绑许可证都是在与 Studio 的其他产品相同的计算机上使用的，因此，例如，您不能在购买 Studio 8 并向您的老板提供 Contribute 来更新公司网站的同时，在 Dreamweaver 中完成更繁重的设计工作。你的老板将需要他自己的 Contribute 副本来做那件事。

除了你在 Studio 8 中找到的所有其他好东西，它的定价值得特别注意。Macromedia 首次为 Studio 8 提供 399 美元的单一升级价格，允许您从 Studio 的任何早期版本或其组件产品升级到完整的 Studio 8(现在包括 Flash Professional)。因此，如果你还没有这些产品中的一个，现在就值得去易贝找一个便宜的，合法的副本，比如说，Flash 3。你可能花不到 20 美元就能找到它，但拥有这个序列号意味着你只需要为 Studio 8 支付 399 美元的升级价格，而不是 999 美元的全价。

Dreamweaver 8、Flash Professional 8、Fireworks 8 和 Macromedia Studio 8 将于 9 月上市。Contribute 3、FlashPaper 2 和 Freehand MX 现已推出。
