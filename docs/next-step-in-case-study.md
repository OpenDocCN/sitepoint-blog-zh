# 案例研究的下一步

> 原文：<https://www.sitepoint.com/next-step-in-case-study/>

感谢你在之前的博文中对赚钱网站的所有预测。你的一些预测非常令人沮丧，让我怀疑自己是否还能继续作为一名企业家存在。其他人创造了可能性，我很欣赏。大多数人都很体贴。

我们来补充一些信息。我想让你了解我的思维过程，这样你就能明白这个策略的逻辑。

拳击网站是第一个推出的。我是通过与一位私人教练合作想到这个主意的，这位私人教练碰巧曾经是一名职业拳击手(绝不是伊万德·霍利菲尔德，而是一名有力的竞争者)，他创造了一个非常棒的拳击健身计划。他还告诉我，与你的一些帖子相反，健身界有很多在线认证项目。所以我们合作创建了这个网站。用视频创建它花了大约 1000 美元(最初是在额外开发之前)。它做得很好。我说的“好”是指获得一个客户的成本比客户愿意支付的要低。

因此，这导致了其他认证网站也可以工作的想法。这包括占星术(结果证明是有利可图的)、教练网站(即将测试)和医疗保健营销(即将测试)。在早期的草案阶段，简单地展示一个医疗实践的医疗保健网站已经导致了支付所有这四项费用的参与。

免费网络站点是我想出的一个不同的策略。这里的策略是通过广告和图书销售(如 sitepoint)来弥补成本，但最终会以目前高达 5-40 美元的“每活跃用户成本”卖给其他人。我不知道这将导致什么，但会让你保持联系。然而，这个网站有一个特定的目标市场。我将关注特定的社区，因此是地理上的而不是行业上的。一旦一个社区注册了，我就去下一个。我追求一些有创意的免费营销策略，稍后我会讲到。

那么这意味着什么呢？

1.如果你得到了一个成功的网站，推出不同类别的类似网站。

2.以低廉的价格推出产品。它花费我 700 美元的平均网站，这样 2 或 3 个用户支付开发费用。

3.测试并投资于有效的方法。拳击网站是如此之好，我将在下个月推出跆拳道和其他一些项目，有一个世界冠军跆拳道选手。

4.知道“够了”意味着什么。我的目标是有几个网站，让我在家过上好日子。我想如果我能得到 5-10 个拳击风格的网站，我可以实现这个目标。

5.你可以向任何人推销任何东西，如果你专注于他们想要的，并提供良好的价值。在所有这些网站的情况下，来自实际潜在客户和客户(而不是网站开发者)的初步反馈是积极的。
