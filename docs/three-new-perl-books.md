# 三本新的 Perl 书籍

> 原文：<https://www.sitepoint.com/three-new-perl-books/>

许多新的 Web 程序员会告诉你，Perl 对于今天的 Web 程序员来说过于庞大和复杂，它正在被淘汰，将被 ASP、PHP 和其他新兴的脚本语言所取代。然而，那些了解内情的人会告诉您，Perl 可以为几乎任何脚本问题提供一个强大而优雅的解决方案，并且其纯粹的多功能性和丰富的基层支持意味着它将成为未来几年的竞争者！

接下来是我对 WROX 出版社的*专业 Perl 编程*和*专业 Perl 开发*以及来自 O'Reilly 的*Perl CD Bookshelf 2.0*的迷你评论。如果您刚刚开始使用 Perl，请务必查看我的来自 WROX 的*开始 Perl 编程*的[完整评论](http://www.webmasterbase.com/article.php/263)。无论您是经验丰富的 Perl 专家，还是尚未编写第一篇“she bang ”, WROX 和 O'Reilly 都能为您提供这本书。

##### 专业 Perl 编程(WROX)

介绍 Perl 的书籍往往有一个主要的弱点:它们隐藏了语言的复杂性，以使主题看起来更容易理解。对于没有编程经验的读者来说，这可能有点太快了，*Professional Perl Programming*包含了技术细节和高级脚本特性。因此，这本书是有经验的程序员开始使用 Perl 的理想选择，或者是已经学习了基础知识并且正在寻找更多细节的人的理想选择。

由 Peter Wainwright,[Professional Apache](http://www.webmasterbase.com/article.php/299)(Apache 从未有过的手册)的作者，以及包括 Simon Cozens,[Beginning Perl Programming](http://www.webmasterbase.com/article.php/263)的作者在内的许多强有力的支持作者撰写，整本书的写作质量都非常好(我发现 WROX 出版社的其他合作作品有时并非如此)。这些页面代码繁多，并附有厚厚的参考信息表，但支持这些材料的文本总是令人信服、深思熟虑且切中要害。

除了本书第一部分对基本语言元素令人难以置信的全面介绍之外，*Professional Perl Programming*还涵盖了一些高级主题，如正则表达式(这是我在一本不专门讨论该主题的书中看到的最完整的讨论)、终端 I/O(用于创建交互式程序)、性能概要(用于定位脚本中的慢点)、Perl 中的面向对象编程、Perl 解释器的内部工作方式、网络应用程序、可移植编码实践等等！

如果您认为自己熟悉一般的编程概念，无论您是否有 Perl 经验，专业 Perl 编程都是理想的实用指南，可以让您尽可能快地开始使用这种语言，并尽量减少未回答的问题。

##### 专业 Perl 开发(WROX)

令许多人感到不愉快的是，一旦他们学会了 Perl，他们并不能立即应用这些知识来创建动态网页。这样做的原因是 Perl 可以用于许多不同的事情；创建用于网络的 CGI 脚本只是其中之一。在*Professional Perl Development*中，Randy Kobes、Peter Wainwright、Shishir Gundavaram 和一组投稿作者向读者展示了 Perl 最常见的应用程序(从最基础到更高级)。

这本书不适合 Perl 初学者，因为它从第一页开始就假设您已经有了坚实的语言基础。这并不是说这本书只适合有经验的程序员，只是说在你开始阅读这本书之前，你应该先浏览一下介绍性的文章，比如《开始 Perl 编程》。

第一章介绍了 Perl 最流行的应用:Web 服务器编程。具体来说，它提供了用 Perl 编写 CGI 脚本的完整内容，从 HTTP 协议的基础到 Web 服务器安全性的高级问题。第二章以同样的思路继续，涵盖了现代 CGI 的替代品 FastCGI 和 mod_perl。本书后面的章节着重于在 HTML 页面中嵌入 Perl 脚本，并使用 PerlScript 编写以 Perl 为脚本语言的 ASP 页面(这听起来很奇怪)。

虽然上述章节对于 Web 开发人员来说无疑是本书的核心章节，但本书还有更多内容。其他主题多种多样，包括支持各种网络协议的库(从 Traceroute 到 IMAP，从 NNTP 到 POP3)，文件和 RDBMS 的数据库访问，如 MySQL 和 Perl DBI 库，XML 支持，高级数学和计算应用程序，图形和图形用户界面。

总而言之，*专业的 Perl 开发*没有留下抱怨的余地。这是一本将 Perl 应用于计算专业人员所面临的实际问题的实用指南，它将让您大开眼界，了解 Perl 的多功能性。然而，我不禁想知道，如果有经验的开发人员可以很容易地掌握新的语言，那么对 Perl 语言的简短介绍可能会使这本书更加完整。

##### Perl CD 书架，版本 2.0 (O'Reilly)

如果说我在回顾 Web 开发人员和普通程序员可以阅读的各种 Perl 书籍时发现了什么，那就是很难或者不可能找到一本能够教会你所有关于 Perl 的知识的书。随着最初版本的*Perl CD 书架*的发布，奥莱利的头脑似乎已经意识到了这一点。他们没有试图创建一个涵盖 Perl 语言所有方面的单一的大卷，而是将他们最成功的书籍翻译成一种方便的、可搜索的 HTML 格式，并在一张 CD 上全部呈现出来。

在这本终极(最终也是可移植的)Perl 参考的 2.0 版本中，O'Reilly 从 CD 上包含的一系列书籍中删除了两本介绍性的书籍(*学习 Perl* 和*学习 Win32 系统上的 Perl*)，取而代之的是一本更符合本系列快速参考本质的新书*Perl for System Administration*。这个新版本还包括更新的第三版*编程 Perl* ，被许多人认为是 Perl 的权威著作(所谓的“骆驼书”)。

这一系列中的五本书通常零售价为 189.75 美元，这使得 79.95 美元(或者更便宜，如果你货比三家)的 CD 书架相当便宜！概括地说，*Perl CD 书架，版本 2.0* 包含以下 O'Reilly 书籍的完整版本:

*   简而言之 Perl
*   Perl 编程，第 3 版
*   Perl 食谱
*   高级 Perl 编程
*   用于系统管理的 Perl

最重要的是，CD 捆绑了一个完整的印刷版本的 *Perl，简而言之*，通常零售价为 29.95 美元！

高度超链接，完全可移植，可以在任何浏览器上查看，并且由于主索引和 Java 搜索引擎，可以立即搜索，我发现这比用于参考目的的相同书籍的印刷版本要方便得多。但是，如果您刚刚开始学习 Perl，那么在投资这个包之前，请确保您不介意在您的计算机显示器上阅读。
