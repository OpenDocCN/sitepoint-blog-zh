# JsLint for PHP 有人吗？

> 原文:[https://www.sitepoint.com/jslint-for-php-anyone/](https://www.sitepoint.com/jslint-for-php-anyone/)

有人有多余的时间和技能吗？看到将 [jslint](http://www.crockford.com/javascript/lint.html) 移植到 PHP(以及 Perl / Python / Ruby)会很棒。多字节字符串可能有一些问题需要解决，但是浏览[资源](http://www.crockford.com/javascript/fulljslint.js)看起来是可行的。

这意味着像简单测试这样的工具可以帮助验证网页上的 Javascript。简单测试自带基于正则表达式的解析工具，我在这里简单介绍了一下。考虑到字符编码问题，正则表达式是否是正确的方法是另一个问题。
