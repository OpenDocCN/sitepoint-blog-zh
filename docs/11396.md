# 其他受众:维护招聘人员关系

> 原文：<https://www.sitepoint.com/the-other-audience-maintaining-recruiter-relationships/>

我刚刚接到一个招聘人员的电话。几年前(是的，*年*)我在这家招聘公司的网站上建立了一个“个人资料”，输入了我的个人信息和工作经历、工作样本和一份简历。这是我第一次听到招聘人员的消息。他问我是否还在找工作(我没有，已经 18 个月没有找工作了)。他有一份合同工作，马上开始，“符合你在我们的数据库中输入的细节。”太好了。

尽管有这样的经历，我知道还是有好的招聘者的。好的招聘人员也值得保留，因为和工作一样，他们也是宝贵的建议来源。

因此，我倾向于像对待客户关系一样对待招聘人员关系。以下是我用来和我喜欢的招聘人员保持良好的、富有成效的关系的一些策略。

**1。欣赏他们的立场。**

是的，招聘人员通过填补职位获得报酬，但他们并不都是为了在客户组织中谋得一席之地。你首选的招聘者很可能是那些了解你、喜欢你的态度和工作，并希望在客户组织和专业人士之间进行愉快匹配的人。

招聘人员不只是工作机器:他们是真实的人，就像我们一样。当你和他们说话时，他们可能有压力。他们可能会在一个严格的截止日期前工作，或者渴望给你一份新合同的优先权。

当我接到招聘人员的电话时，我总是当场和他们说话(或者尽快给他们回电)。我尊重他们忙于其他客户和职位的事实，但我也意识到我需要关于这份工作的信息，所以我会在第一次电话中尽可能多地从他们那里获取信息。这意味着我可以更快地对项目做出决定，而不是更晚，所有招聘人员都喜欢这样！

另一方面，如果我向他们发送一些工作样本、询价或其他信息，我会记住，我的联系人可能不是他们今天的首要任务，只有在一周左右没有回复的情况下才会跟进。

**2。告诉他们你在做什么。**

你刚刚完成了一个很酷的项目吗？告诉你的招聘代理。也许你已经更新了你的简历或开本；给你的招聘人员发一份新版本的复印件。去度假吗？让你的招聘人员知道你什么时候会回来工作。

让你的招聘代理知道你在做什么——至少每隔几个月左右——是保持联系的好方法，确保当理想的项目出现时你是第一个想到的。这也是展示你的能力和表明你感兴趣的工作类型的好方法。

**3。征求他们的意见。**

在某些情况下，你可能值得与招聘人员会面，向他们请教。也许你正在考虑改变你的职业重心，改变你的物理位置，或者寻找更多或不同的工作？在所有这些情况下，你信任的招聘人员可能会给你提供合理的建议和方向。

当然，如果你被考虑担任他们推荐的职位，他们可能会很乐意提供各种建议——从公司文化和工作场所道德到你应该在你的文件夹中包括哪些项目。他们会告诉你应该为面试做什么样的研究和准备，以及你什么时候能得到成功的消息。

征求招聘人员的意见不仅让你有机会从他们的市场和客户知识中受益:它还有助于建立信任和尊重的关系，并促进双方对对方提供的东西感兴趣的理解。

**4。帮助他们帮助你。**

我们已经讨论过的三个建议都指向同一个方向:你需要帮助招聘人员来帮助你。基本上，我尽量给招聘人员提供最新的信息，这样我们都不会浪费对方的时间去考虑不符合要求的工作。

我努力让招聘人员给我提供职位变得非常容易:我努力变得容易联系，我让他们知道我对什么样的工作感兴趣，我把他们打电话给我的每个项目都视为一个机会——我不倾向于不考虑任何项目。如果我不能接一个项目，我会解释原因，这样他们就能了解情况，如果需要的话，他们就能对我感兴趣的工作有一个更清晰的了解。

**5。把他们当作你生意的另一个受众。**

你有客户和潜在客户——为什么不把招聘人员当作另一个你可以以专业、常规的方式与之交流的受众呢？如果你有几个招聘人员要管理，在你的日程表中设置提醒，定期联系他们，给他们发送你的项目和职业活动的更新，等等，可能是值得的。

将招聘人员纳入你的定期联系日程，并花时间维持这些关系，真的能产生效果。一个管理得当的招聘人员对你的企业来说是一笔巨大的财富——甚至更多。所以，尽你所能去寻找优秀的招聘人员，并与之建立持久的关系。
