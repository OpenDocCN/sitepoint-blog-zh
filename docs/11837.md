# 跟进的力量

> 原文:[https://www.sitepoint.com/the-power-of-follow-up/](https://www.sitepoint.com/the-power-of-follow-up/)

当你向一个新客户求爱时，回应询问和建立新的联系，花时间跟进并与对方再次联系往往是获得一个新项目或完全错过它的决定性因素。伸出援助之手会对你有利，尤其是当客户犹豫不决，或者他们正在考虑几个人来争取机会的时候。

如果你提议帮助客户解决问题，并且他们对你的提议表示出兴趣，跟进不会让你变得咄咄逼人或令人讨厌。事实上，这是销售的一个重要部分。

当你跟进时，你是在告诉客户:

*   你在想他们。
*   他们的工作和需求对你很重要。
*   你坚持不懈，兢兢业业。
*   你已经准备好和他们一起工作，接受他们的挑战。
*   他们的项目对你来说很重要。
*   你始终如一，值得信赖。

跟进是提醒客户你在那里并鼓励他们行动的好方法。这也提供了一个开始与客户建立真诚友好关系的机会，这也有助于做出对你有利的决定。

有很多方法可以跟进潜在客户，你不必总是使用相同的方法。

**向他们发送他们需要的东西**

跟踪潜在客户最明显的方法是重新联系他们，向他们提供信息。如果他们在你演讲时要求参考资料、作品集或正式提案，尽快发给他们。

**说谢谢**

感谢对方的时间永远不会错。通过在初次会面后发送一封感谢邮件，你还可以概述会面的重点，概述接下来的步骤，并安排下一次会面的时间。

**询问更多信息**

会议结束后，花点时间回顾电话，并为客户准备一份关于项目、时间表及其特定需求的问题列表。这可以是你的感谢信的一部分，也可以作为另一种拓展方式单独发送。

**打私人电话**

几天后，考虑直接给客户打电话，看看他们是否有问题，或者希望你提供更多信息。你还应该在你的跟进工具箱中保留私人电话选项，以防你和客户通过电子邮件有很多往来。

**给他们发些无关的东西**

如果你看到一篇与你的客户相关的文章或博客帖子，把它发送出去。即使它没有直接支持你或者他们为什么应该雇用你，它也会向他们表明你了解他们的业务。这对潜在客户来说是非常宝贵的。
