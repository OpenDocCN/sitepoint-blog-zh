# 免费的 Sybase 企业数据库

> 原文：<https://www.sitepoint.com/free-sybase-enterprise-database/>

在竞争激烈的营销活动中，Sybase 上周三宣布为 Linux 社区发布其旗舰企业数据库产品 Adaptive Server Enterprise 的 Express 版。

这对各行各业的 web 开发人员来说都是一个好消息，因为它是市场上唯一一个免费的健壮数据库，完全支持 SQL 的 ANSI 兼容性、全功能的存储过程以及 Java 的触发器和挂钩。Net 和 XML 以及其他 web 服务。

显然 Sybase 不是开源的。然而，许多 web 专业人员使用 MySQL 和 PostgreSQL 就足够了，我认为它们是优秀的生产工具。然而，在将功能提升到与专利竞争对手相同的水平方面，两家公司都有所落后。因为 Sybase 为生产提供免费的数据库服务器——我鼓励开发人员对它进行评估(毕竟它不会花费你任何东西！).对我们有利的是，我们可能会发现它提高了门槛，并鼓励 MySQL 和其他开源软件更快地开发这些预期的功能。

Sybase 的竞争对手 Oracle 和 IBM 提供开发人员许可证，但是，没有一家提供免费的生产许可证。Sybase 的 Express Edition 有 2 GB 的 RAM 和 5 GB 的磁盘数据库存储限制，尽管这对许多 web 应用程序来说应该有足够的空间。

Sybase 数据库和工具营销高级主管大卫·雅各布森(David Jacobson)说，目标是吸引人们尝试 ase。

“我们认为我们的市场分为三个部分，那些运行开源解决方案的，那些运行 IBM 和 Oracle 数据库的，以及那些运行微软 SQL Server 的，”雅各布森说。

雅各布森说:“这是一种帮助客户将单个 CPU 开发和生产服务器作为试点运行的方式，”这也应该有助于那些需求超过免费许可限制的人免费评估 ASE。

这对于那些运行 Macintosh 平台的人也有积极的意义，因为 Linux 和 PowerPC 体系结构的交叉是由 Terra Soft Solutions(黄狗 Linux (YDL)的制造商)提供的。Terra Soft 最近发布了 YDL 4.0，支持苹果最新的 G5 系统。

尽管 Sybase 指南建议 Adaptive Server Enterprise(ASE)Express Edition 安装已通过基于 Intel 的 Linux 发行版的认证，但苹果硬件上的 OS X 和黄狗 Linux 都能满足最低要求。Sybase 的商业版 ase 原生支持 OS X。

Apple 平台用户可能会发现 Sybase 解决方案可以与 G5 Xserve 一起使用，G5 Xserve 针对数据驱动的解决方案进行了优化，尤其是那些为 Web 架构构建的解决方案。

Xserve 带有现成的 LDAP、Apache Web 服务器和 JBoss/Tomcat Java 应用服务器组合，所有这些引擎都可以与 ASE 交互。

增长潜力:Gartner DataQuest 的统计数据显示，2003 年，Sybase 在 Windows 和 Unix 市场各获得了 9000 万美元的许可证收入。在 Linux 领域，该公司与 IBM 和 Oracle 相比相形见绌，这两家公司在 2003 年共占 Linux 数据库许可证收入的 95%以上。

根据 Sybase 高级集团营销经理 Amit Satoor 的说法，他的公司正在通过上述 ANSI 合规性和高可用性功能路线图使自己与其他免费产品区分开来。

必须指出的是，MySQL 最近发布了集群技术，尽管还不到一年，可能被认为是未经证实的。

雅各布森说，赛贝斯将以极具竞争力的价格与其核心竞争对手 IBM 和甲骨文展开正面交锋。
