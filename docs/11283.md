# 谷歌为 bug 提供现金

> 原文：<https://www.sitepoint.com/google-offers-cash-for-bugs/>

![Google Chromium](img/be7a8490a5f862c98cb2ebc60eaa8e8e.png)钱是不是有点紧张？信贷紧缩是你吗？如果你有一点黑客经验，谷歌会为你在 Chromium 中发现的每一个新的安全漏洞提供 500 美元 Chrome OS 和 Chrome 浏览器背后的开源项目。

谷歌已经认识到，大多数有趣和不寻常的安全漏洞都是由 Chromium 项目之外的程序员发现的:

> 由于这些人和其他人的共同努力，Chromium 的安全性更强了，我们的用户也更安全了。

臭虫现金的概念并不新鲜。Mozilla 为 Firefox、Thunderbird 和相关插件或扩展中遇到的安全问题提供了一个类似的 [bug 奖励计划](http://www.mozilla.org/security/bug-bounty.html)。

谷歌计划有几个条件:

1.  一个小组将评估哪些安全漏洞有资格获得 500 美元的奖励。特别严重的问题可能会得到额外的奖励。
2.  只有第一个在 [Chromium bug tracker](http://bugs.chromium.org/) 中报告问题的人才有资格获得奖励。
3.  Chromium、Chrome OS、Google Chrome 和浏览器附带的扩展中存在的错误是合格的。Webkit 等第三方组件中的 bug 也会被考虑。
4.  如果您愿意，您的身份可以保密，但奖励不能发放给未成年人或美国实施出口限制的国家的居民。

我认为这是个好主意，尽管报酬应该更高。雇佣有知识的测试人员并不便宜，而且程序可能会发现标准测试程序永远不会发现的问题。贡献者可能会花很多天去寻找一个不寻常的 bug，结果却挣得比美国最低工资还少！更高的奖励也将为系统程序员提供更大的激励来彻底测试他们自己的系统。

如果其他公司也效仿岂不是很棒？如果某个 IT 巨头向那些发现其操作系统或浏览器漏洞的人发放现金，我们都可以放弃日常工作，退休了！
