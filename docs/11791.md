# 谷歌揭示 Chrome 3 测试版

> 原文:[https://www.sitepoint.com/google-reveal-chrome-3-beta/](https://www.sitepoint.com/google-reveal-chrome-3-beta/)

尽管 Chrome 2 在五月才发布，谷歌已经发布了第三版作为可下载的测试版。那么，在过去的几个月里，谷歌在做些什么……**新标签页定制**“新标签”页显示你最近和最常访问的网站的缩略图。现在提供了许多自定义选项，因此可以重新排列、删除或使链接“粘滞”。

Omnibox 的改进这个奇怪的名字“Omnibox”只是智能地址栏的另一个名字(或者其他浏览器制造商如今对它的称呼)。下拉列表现在提供图标来帮助区分搜索或最近被建议、加入书签或查看的站点。

**自定义主题** Chrome 的 Chrome 现在可以改了！在[周围已经有 30 个主题可用](https://tools.google.com/chrome/intl/en/themes/index.html)，尽管大多数看起来非常相似，只有背景图形有所改变。我尝试了一些，但最终还是回到了干净不显眼的默认蓝色主题。

Chrome 仍然专注于浏览器速度，谷歌称 JavaScript 性能提高了 30%。Chrome 仍然感觉很快，但是 JavaScript 的改进不一定会让浏览器快 30%。在测试版中，性能提升是相当不可思议的。 **HTML5 支持**谷歌支持 HTML5，尽管它在某些其他浏览器中支持很少。Chrome 3 实现了更多功能，如视频标签和 web workers(类似线程的 JavaScript 代码，可以在后台加载和执行，而不会占用浏览器)。有趣的是，谷歌的新闻稿没有提到扩展。Chrome 3 支持它们，但你需要用命令行开关来启用它们。我怀疑完整的扩展系统可能要到 Chrome 4 才会出现。总的来说，Chrome 3 beta 是一款快速、稳定、有竞争力的浏览器。现有的 Chrome 用户几乎肯定会升级，但几乎没有什么引人注目的新功能会让我远离 Firefox。你会为 Chrome 放弃你的默认浏览器吗？Chrome 链接:
