# 创建更好的电子邮件简讯的五种方法

> 原文:[https://www.sitepoint.com/how-to-improve-email-newsletter/](https://www.sitepoint.com/how-to-improve-email-newsletter/)

上周，我分享了七个技巧，它们成为了我的电子邮件营销清单的第一位。但是在你开始编辑你的主题和测试你的链接之前，你需要有一些有用的和相关的东西发送出去。在撰写和分发电子邮件简讯时尤其如此。

电子邮件简讯不同于任何其他类型的电子邮件营销，因为它们通常内容丰富，通常在风格、内容和格式上是可预测的。而且竞争很激烈。想想每天你的收件箱里有多少邮件在乞求你的关注。

这里有五种方法可以改变标准的普通电子邮件简讯，让你的邮件成为邮件堆中的佼佼者。

## 给读者他们想要的

我经常阅读的电子邮件简讯给了我一些在其他地方得不到的东西。它可以是独一无二的内容，新的提示或者仅仅是我不知道的资源的链接。

在我自己的电子邮件简讯中，我每一期都提供免费下载，这些 pdf 的链接在我的每一期简讯中都获得了最多的点击。

## 与你的读者交谈

你是否以一封简短的信或你的简介开始每份时事通讯？这是一个很好的方式来介绍你的内容，指出感兴趣的项目，并在个人层面上打动你的读者。它不需要很长或者很深入，但是一个非正式的笔记可以在吸引你的听众方面发挥很大的作用。

## 设定主题

你的时事通讯的一般主题可能是非常广泛的，例如图形或网页设计。每期都应该有一个深入的焦点——例如，Photoshop 技巧或 CSS 最佳实践——让它更有趣，不那么分散。如果你很难为每一期找到内容，这也是一个打破写作瓶颈的好方法。

## 加入一些变化

很少有人有时间或精力去阅读一篇又一篇内容冗长的电子邮件简讯。混合不同类型的内容(文章、资源链接、简短的广告等)。)，拉报价，照片等。这对于 HTML 时事通讯来说很简单，但是你也可以通过使用这些选项来提高对纯文本邮件的兴趣。

## 明确你的行动号召

你希望你的读者在读完你的时事通讯后做什么？如果你想让他们下载一个赠品，注册一个特别优惠，提交一篇客座文章，或者点击一个特定的链接，不止一次地清楚地拼写出来，让你的读者行动起来。
