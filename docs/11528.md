# 火狐五岁生日快乐！

> 原文:[https://www.sitepoint.com/happy-5th-birthday-firefox/](https://www.sitepoint.com/happy-5th-birthday-firefox/)

今天，Mozilla Firefox 已经 5 岁了。

Firefox 1.0 于 2004 年 11 月 9 日发布。浏览器很快赢得了 web 开发人员的崇拜，他们的热情传遍了全世界。Firefox 已经被下载了超过 10 亿次，是仅次于 Internet Explorer 的世界第二大浏览器。(你可以说 Firefox 3.x 是最常用的浏览器，因为 IE 的市场份额被 IE6、IE7 和 IE8 瓜分了)。

Firefox 的历史可以追溯到几年前，它从 Mozilla 漏洞百出、臃肿不堪的浏览器套件的灰烬中崛起。这个项目被恰当地命名为“凤凰号”，由戴夫·海厄特和布雷克·罗斯于 2002 年启动。他们的目标是使用 Mozilla 的 Gecko 渲染引擎开发一个稳定、轻量、快速的独立网络浏览器。名字随后改为*【火鸟】*，最后改为*【火狐】*。

Mozilla 很快发布了 Firefox was the future，这款浏览器凭借其创新的营销活动获得了广泛的关注。随之而来的是赞美的评论，人们开始意识到 Firefox 不仅是 IE 的可行替代品，而且它以相当大的优势击败了老化的微软浏览器。

对于开发者和高级用户来说，Firefox 最大的吸引力是附加系统。浏览器通过扩展得到了增强，使得开发更加容易，并促进了 Web 2.0 和 Ajax 应用程序的兴起。今天，如果没有 Firebug、web Developer Toolbar 或许多其他重要的 Firefox 插件，开发一个 Web 应用程序几乎是不可想象的。

也许 Mozilla 最大的成就是 Firefox 改变了 IT 行业。五年前，很少有网络应用被认为是桌面的可行替代品，微软声称网络浏览器已经死了。Mozilla 的成功促使苹果和谷歌进入该市场，微软也收回了 IE6 将是他们最后一款浏览器的声明。

火狐生日快乐！你比我们大多数人都老！
