# 愿特遣队与你同在

> 原文:[https://www.sitepoint.com/may-the-task-force-be-with-you/](https://www.sitepoint.com/may-the-task-force-be-with-you/)

Web 标准项目现在有了一个 JavaScript 分支。它叫做 [DOM Scripting Task Force](http://domscripting.webstandards.org) ，我最好引用一下[宣言](http://domscripting.webstandards.org/?page_id=2) :
【引用】目前，JavaScript 受到过时、无知和不可访问的开发方法的困扰，这些方法阻碍了它，因此也阻碍了整个 web 开发发挥其全部潜力。

WaSP DOM Scripting Task Force 建议通过采用不引人注目的 DOM 脚本来解决这个问题，这是一种基于现代的、符合标准的、可访问的 web 开发最佳实践的思维方式。正如 WaSP 的 Dreamweaver 任务组帮助 Macromedia 使 Dreamweaver 更符合 web 标准以及 WaSP 自己对 web 标准所做的那样，脚本 TF(人们亲切地称呼它)的存在是为了推进优秀 DOM 脚本的事业。如果你正在读这篇文章，那么你可能已经同意并且可能在宣扬 TF 正在传播的信息，但是我说，越多越好。那里有很多好人，我也是。:-)
