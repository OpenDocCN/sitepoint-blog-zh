# 在线声誉管理:基础知识

> 原文：<https://www.sitepoint.com/online-reputation-management/>

所有专业人士，尤其是那些在网上开展业务的人，都可能受到负面宣传。你只需要在一个有着正确(或错误)流量组合的博客或网站上发表一条负面评论，就能让你的名声扫地。如果你没有听说过或者没有在做在线声誉管理(ORM ),这里有一些基础知识，以及为什么你需要现在就开始做。

这是 ORM 三部分系列的第一部分。

**这是什么**

ORM 是在网上追踪品牌、公司、个人或服务的过程，同时制定计划来处理任何负面反馈。您可以将它视为一个三步流程，尽管它们可能不总是按以下顺序发生:

1.  监控——维护一个持续的系统来研究和跟踪公众的看法。
2.  评估——考虑个人反馈，以及来源、渠道、范围和时机，以做出关于风险的决定。
3.  行动——根据你的评估，评论、反驳、起草一份正式回复或干脆忽略所说的话。

ORM 通常被认为是营销(包括搜索引擎优化)和公共关系的混合体。有许多公司提供 ORM 服务，尽管这是你可以自己免费做的事情。

**你为什么需要这么做**

假设你有多年的经验，坚实的客户基础，以及与客户和同事的良好关系。你最近参与了一家知名公司的一个大型设计项目的投标，并获得了成功。另一名候选人没有得到这份工作，他很不满。他不仅在这个过程中投入了大量的时间，而且公司内部的一位朋友基本上保证了他能得到这份工作。

当发现你被授予这项工作时，另一位候选人在他那有 5000 多名读者的非常受欢迎的设计博客中公开攻击你的性格、职业道德和价值观。你不知道的是，你的名字在谷歌搜索结果中名列前茅，现在是这篇尖刻的帖子，在这一点上，有 35 条来自甚至不认识你的人的评论，同意作者的观点。任何用谷歌搜索你的人现在都有可能在到达你自己的网站之前看到这一点。

你花了两个星期的时间，减少了正常情况下稳定的询问，并从一个长期客户的电话中询问详细情况，你才发现这个帖子。在这一点上，你没有办法知道这让你损失了多少生意，但你很快就开始寻找这篇文章的所有参考资料，并试图控制损失。

如果你一直在监控你的网络声誉，你不一定能够避免这种情况，但你可以立即做一些事情来弥补一些损失。在线声誉管理最重要的部分是意识到人们在谈论你、谈论谁以及为什么谈论你。它还要求你坚持不懈地去做，并在需要采取行动时制定一个计划。
