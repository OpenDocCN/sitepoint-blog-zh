# 3 个优秀的 JavaScript 和 Canvas 示例

> 原文:[https://www . site point . com/3-great-JavaScript-canvas-games-examples/](https://www.sitepoint.com/3-great-javascript-canvas-games-examples/)

JavaScript 驱动的演示变得越来越复杂。开发者现在正在创造高质量的基于浏览器的游戏，不需要 Flash 或其他插件。

这里有三个最好的添加到你的，呃，这个周末的“技术研究清单”。他们被选中是因为他们的技术才华和引人入胜的游戏性。它们可以在大多数浏览器中工作，除了没有本地支持的 IE 浏览器。在大多数情况下，Firefox 运行良好，但 Chrome 和 Safari 似乎为您的……*分析*提供了最佳体验。

## [![Thrust](../Images/64bb58e6b32a2f428da88bc1b44516d6.png)推力](http://joncom.be/experiments/thrust/play/)

推力是 1986 年的一款流行的 8 位游戏，由 Jon Combe 重新制作(这不是一款仿真游戏)。这是一款非常令人愉快的游戏，灵感来自于雅达利的 Gravitar。从技术上来说，很难挑剔乔恩取得的成就——甚至连音效都很准确。

## [![Thrust](../Images/1a99eac3283fcbce6d1c9686ac8264ab.png)小行星](http://www.kevs3d.co.uk/dev/asteroids/)

如果你认为所有的`canvas`例子都提供了简单的图形和效果，你会被小行星震惊——这是对原版[雅达利街机游戏](http://en.wikipedia.org/wiki/Asteroids_%28video_game%29)的光荣致敬。凯文·斯考特的版本以 3D 光线追踪图形和快速动画为特色。几年前你会为此付很多钱！

## [![Thrust](../Images/1a6394eacd47c72e5ce8388971f6ae01.png)环面](http://www.benjoffe.com/code/games/torus/)

这款 3D 俄罗斯方块游戏仍然是最令人印象深刻的 JavaScript 和`canvas`游戏之一。Ben Joffe 的游戏提供了流畅的 3D 动画和令人上瘾的“只剩最后一关”游戏。很棒的东西。
