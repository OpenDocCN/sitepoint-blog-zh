# 2010 年十大精美 Flash 网站

> 原文：<https://www.sitepoint.com/ten-beautiful-flash-web-sites-of-2010/>

继续回顾 2010 年出现在我们屏幕上的一些可爱的设计。昨天我们看了一下基于 CSS 的网站，今天我将重点介绍过去 12 个月中引起我注意的一些基于 Flash 的网站。当谈到 Flash 时，有些设计师可能会有点爱恨交加。它为创意设计和动画提供了极好的机会，如果你有动作脚本技能，只有你的想象力限制你能做什么。另一方面，因为搜索引擎优化的困难和缺乏精确定位网站特定部分的能力，以及因为某个智能手机制造商强烈反对在他们的(I)手机上使用 flash player，所以它是不受欢迎的。

但是在这篇文章中，让我们来思考和看看这项技术的一些伟大的用途。

我冲浪是因为是一个为冲浪者提供的网站，它汇集了关于这项运动的美丽图片、视频和鼓舞人心的陈述。

[![ISurfBecause](img/c0f6e6b937ccfe4f1e6e498c3a7e8389.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/ISurfBecause.jpg)

汉堡的声音融合了城市的声音和管弦乐。

[![SoundsOfHamburg](img/2c65798aa8e0e4e198596625371d8f42.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/SoundsOfHamburg.jpg)

我不太清楚这个网站实际上实现了什么，但我喜欢看它。Grey Shines 是一个音乐节目，以日本 Grey 公司员工的手绘角色为特色。

[![GreyShines](img/ecde1b705e953d112a5e96663e06ad1f.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/GreyShines.jpg)

水瓶座能量饮料是西班牙圣詹姆斯大道的赞助商。

[![Aquarius](img/0093261f37a0eb9e728fc2a29cbfeda1.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/Aquarius.jpg)

瑞典鼻烟。瑞典语那是什么？是的，这是无烟烟草，还有什么？

[![Swedish](img/aa8a89590a2ed86d455d46f03431dd7d.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/Swedish.jpg)

2010 年莫奈博览会的绝佳地点。

[![Monet](img/3e076af90a5b2e7f4c6daea7a7959488.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/Monet.jpg)

[BBC 地球——生命是](http://lifeis.bbcearth.com/)。来自 BBC 档案馆的精彩图片和视频集合。

[![BBC](img/26c9b2a8dc1c9b56101b660ab26f490f.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/BBC.jpg)

[我学到的二十件事](http://www.20thingsilearned.com/)是一个引人入胜的网站，基于谷歌撰写的描述现代浏览器如何工作的文章。

[![TwentyThings](img/5e00eb149e8d2ed5115bfc8d601e6e4a.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/TwentyThings.jpg)

[一天的生活](http://www.youtube.com/lifeinaday?x=explore)是一个全球性的项目，旨在制作一部用户生成的地球上一天的电影。

[![Life](img/a36eba9102b0167012f8e40d7f020e2a.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/Life.jpg)

作为雀巢快速饮料的网站，它结合了可爱的插图和一个叫做“快速环绕”的系统，即我们周围的空气。

[![FAST](img/bf44fc829be8d8b0757b37184e679055.png)](https://www.sitepoint.com/wp-content/uploads/2011/01/FAST.jpg)
