# Ruby 的人气在消退吗？

> 原文：<https://www.sitepoint.com/is-rubys-popularity-fading/>

主要是由于大肆宣传的 Ruby on Rails web 应用框架，Ruby 在过去四年左右的时间里被谈论了很多——也许与实际使用它的人数不成比例。然而，根据所有可用的指标，Ruby 的使用也像杂草一样增长。但这种情况最近有所改变吗？人们开始脱离红宝石潮流了吗？

分析师对 Ruby 的未来充满信心。Gartner research 副总裁 Mark Driver 在他 2008 年 4 月的报告《编程语言中的开源》中预测，到 2013 年将会有 400 万 Ruby 开发者。此外，他指出 Ruby 将会在企业中大受欢迎。“与 PHP 等典型的动态‘脚本’语言相比，Ruby 将在企业 IT 开发人员中享有更高的集中度，”他写道。

拥有代码搜索引擎 [Koders](http://www.koders.com/) 的 Black Duck Software 在 10 月份表示，去年[的 Ruby 搜索量增长了 20 倍](https://www.sitepoint.com/7-places-to-find-the-code-you-need/)，已经超过 Python、PHP 和 Perl 成为他们网站上第四大搜索语言。然而，与网络的其他部分相比，这个结果似乎有些反常。

TIOBE Index 查看了来自主流搜索引擎的搜索数据——谷歌、雅虎。MSN 和 YouTube，目前将 Ruby [排在第 11 位](http://www.tiobe.com/index.php/content/paperinfo/tpci/index.html)——一年比一年高。虽然 Ruby 实际上有第三大的一年增长，但这不足以突破前 10 名语言，并且两年前达到了最高排名。

在过去的五年里，Python 实际上比 Ruby 发展得更快——至少在搜索量上。

另一个衡量标准，Ohloh 的[语言比较工具](http://www.ohloh.net/languages/compare?measure=contributors&percent=&l0=perl&l1=php&l2=python&l3=ruby&l4=java&l5=-1&commit=Update)，衡量提交、代码行数和开源项目的贡献者，显示 Ruby 从 2005 年开始稳步增长，但最近有所下降。这一趋势在 Ohloh 衡量的所有指标中重复出现。另一方面，PHP 和 Python 在过去十年中或多或少显示出持续稳定的增长。

## 结论

那么 Ruby 的人气在下降吗？现在说这个可能还为时过早——TIOBE 实际上表明 Ruby 今年获得搜索份额的速度比除 C 和 C++之外的任何其他语言都快，而且根据 Indeed 的数据，每个月都有越来越多的 Ruby jobs 可用。然而，这篇文章中的数字可能显示的是 Ruby 的增长率略有下降。

确定什么是最流行的编程语言是一个困难的前景，但从我们所掌握的数据来看，Gartner 关于 Ruby 将在 2013 年超过 PHP 的猜测可能是不准确的。PHP 和 Python 几乎在每个方面都比 Ruby 活跃，而且没有下降的迹象——而 Ruby 的增长似乎有点平稳。

当然，好消息是，所有三种语言的就业趋势线都在上升。
