# Twitter 作为社会变革的工具

> 原文:[https://www . site point . com/Twitter-as-a-tool-for-social-change/](https://www.sitepoint.com/twitter-as-a-tool-for-social-change/)

Twitter 可能还不知道如何赚钱，但使用该应用的人们已经开始知道如何以多种有用的方式让它发挥作用。人们经常讨论 Twitter 如何允许毫秒级的新闻周期，并帮助从当地的公民记者那里获得新闻提示。但是，Twitter 也被记者和旨在影响社会变革的项目用于受控的新闻和信息收集。

最好的早期例子可能是保守派博主 Patrick Ruffini 今年早些时候在 Twittering 上发布的爱荷华项目。去年一月，Ruffini 在爱荷华州党团会议期间建立了一个特殊的 Twitter 账户，并要求参加实际党团会议网站的读者在实际党团会议期间用 Twitter 发布早期结果。结果是，在媒体证实之前几个小时，鲁菲尼就已经看到了巴拉克·奥巴马获胜的趋势。

最近，政治技术博客 [techPresident](http://www.techpresident.com/) 正计划使用 Twitter 和特殊的标签组成一个志愿者网络，这些志愿者可以观察投票站的欺诈投票活动迹象，并立即通过 Twitter 报告。TwitterVoteReport.comT2 展示他们的计划。他们写道:“想象一下，在一张全国范围的网络地图上，用大头针标出美国人等待投票超过 30 分钟的每个邮政编码，或者指出投票机不工作的选区。”“当队伍太长时，我们将共同通知对方，并确保媒体和监督团体知道哪里存在问题。”

今天，在佛罗里达州一些提前投票开始的投票点确实出现了问题——所以显然有必要进行这种监督。像 techPresident 这样的项目在跟踪这些类型的问题方面非常出色，原因与 Twitter 作为即时、早期新闻发布平台的出色表现相同:

1.  很快。
2.  它开着。
3.  很简单。
4.  这是两条路。

Twitter 作为新闻或数据收集的工具非常有用，因为它允许研究人员或组织者众包这些任务，并创建一个网络，可以比单个记者甚至一个记者团队更快地获取信息，覆盖更多地区。Twitter 的一个缺点是它相当混乱——信噪比很高。但是正如上面的例子所显示的，通过一些有针对性的 tweet 或者对标签的巧妙使用，从垃圾中分类出有价值的数据并不困难。

techPresident 还提出了一种使用 Twitter 来跟踪入侵机器人电话的方法。

Twitter 并不是唯一参与社会变革法案的网站。许多社交网站开始被用于社区组织和数据收集。例如，YouTube 正在敦促美国选民对他们的投票进行录像，以此来监督和最大限度地减少潜在的投票欺诈。去年二月，哥伦比亚[的一个反 FARC 组织利用脸书](http://www.facebook.com/event.php?eid=7284103926)在短短几天内帮助组织了数十万抗议者。

这种快速的数据收集和社会组织在几年前是不可能的。Patrick Ruffini 在他的爱荷华推特项目后不久，在 techPresident 上写道，Twitter“公开了开发想法和收集新闻提示的过程，给了我们一个了解新闻周期的完整窗口”，并预测 Twitter 将成为这次选举周期的突破工具。

虽然这并不完全清楚——从竞选的角度来看肯定不是真的——Twitter 和社交工具无疑正在成为社会变革代理人的无价技术。
