# 雷鸟反网络钓鱼工具

> 原文:[https://www.sitepoint.com/thunderbird-anti-phishing-tools/](https://www.sitepoint.com/thunderbird-anti-phishing-tools/)

更保险的说法是，几乎所有阅读本文的人都曾在某个时候收到过网络钓鱼邮件，就像你的大多数客户一样。

Mozilla 基金会的雷鸟电子邮件客户端现在将包括诈骗检测功能。(感谢 Thomas Rutter 的提示！)如果应用程序认为某封电子邮件可能是诈骗邮件，它会以可视队列通知用户。与垃圾邮件工具类似，它也将有一个“不是骗局”按钮来表示安全消息，例如来自与您合作的金融机构的在线报表通知。

在其功能中，Thunderbird 将协调 href 显示链接中显示的主机名和潜在的目的地 URL，这是“钓鱼”人们访问不像他们显示的那样的网站的主要方法之一。

如果用户继续点击一个链接，会有一个最后的警告对话框，并给出最后一次取消的机会。

更多信息可以在 Mozilla Bugzilla 上获得(包括截图)。

这不仅可以为精通电子邮件的人(如 web 专业人员)节省时间，而且也是与您的潜在客户和现有客户进行友好的非销售接触的另一个机会，并进一步介绍所有平台的开源替代方案，即 Mozilla 应用程序领域(Linux、Mac 和 Windows)。

托马斯还指出，他认为这是一个非常好的功能，“因为它将教育普通用户点击电子邮件中的链接的潜在问题。*即使它没有检测到所有的诈骗或有一些误报，它的存在也会告诉人们电子邮件中的链接并不总是安全的。”

雷鸟的夜间版本包括新的诈骗工具，预计将出现在雷鸟 1.1 中。
