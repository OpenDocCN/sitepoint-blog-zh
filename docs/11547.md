# 为什么您需要业务连续性计划

> 原文:[https://www.sitepoint.com/why-you-need-a-continuity-plan/](https://www.sitepoint.com/why-you-need-a-continuity-plan/)

持续性规划不只是大人物的专利。小企业主、自由职业者和其他个体职业者需要有一个 B 计划，这样他们就可以保持收入，并在一段时间内无法工作或生活受到某种干扰的情况下继续支持他们的客户。

这是我这些天的热门话题，因为我准备进医院生第三个孩子。因为他有特殊需要，出生后马上需要手术，我们预计他至少要离开家两周，但是未知的事情太多了。因此，我需要一个坚如磐石的计划，以确保我的工作完成，我的客户得到照顾，没有任何事情会被遗漏。

不幸的是，你永远不知道什么时候会发生让你无法工作的事情，即使是很短的一段时间。如果你没有做好准备，你可能会面临损害你未来几年生计的后果。我很幸运，因为我知道我们将面临什么，也知道我能做多少工作，但很多时候你无法预料这些挑战。有效管理危机的唯一方法是制定长期的业务连续性计划。

以下是我的计划的关键要素，其中大部分都是通用的。如果你还没有为自己制定连续性计划，希望它能给你一个好的起点。

## 支持团队

我有几个优秀的分包商，他们知道我的客户是谁，我做的工作以及如何联系他们，如果有必要的话。我的一个分包商已经承担了主要角色，在我离开之前，他将随时了解我所有未完成项目的最新情况。她可以访问我的密码、客户信息和其他重要数据，以便在我不在的时候让一切正常进行。

## 流程文件

当我接受新的工作时，我会创建针对每个客户的流程文档。有些是我最终与客户分享的正式文档，有些是我保存在项目管理软件中的非正式笔记。关键是把每个客户的“谁”、“什么”、“何时”、“何地”和“为什么”写在你的支持团队可以拿到的地方。这样，其他人就可以在必要的时候接手并继续做一个项目，并且确切地知道他们应该做什么。

## 沟通计划

因为几个月前我就知道我会离开办公室一段时间，所以我有足够的时间与我的客户和团队沟通情况。但是如果这是一个意想不到的事件，你将需要一个计划来与每个需要知道内情的人沟通。有一个现成的电子邮件列表会很有帮助，这样你就可以发送一条快速的消息，让每个人都一次跟上进度。制定一个计划让客户和你的团队了解最新情况也很重要，这样他们就知道发生了什么，什么时候他们可以期待你回来。

## 数据存取

您的团队可能需要访问大量文件来代替您填写。如果您已经将数据备份在一个安全且可访问的系统上，那么您就已经领先了。您只需要创建一个包含必要文件的目录，并为您的团队提供访问权限。我使用我的项目管理系统来收集工作文件、流程文档和客户信息，这样一切都在一个地方。我可以根据需要分配访问权限。

这些是我的业务连续性计划的主要组成部分，我发现一旦这些因素都考虑到了，剩下的就会水到渠成。你的连续性计划由哪些要素组成？如果你不能工作，你有信心一切会继续进行吗？
