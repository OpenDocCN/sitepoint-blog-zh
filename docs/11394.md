# 介绍购物狂社交网络 Blippy！

> 原文:[https://www.sitepoint.com/blippy-shopping-social-network/](https://www.sitepoint.com/blippy-shopping-social-network/)

![Blippy](../Images/bc031ba70e894a78881182ca680b009e.png)[【Blippy.com】](http://blippy.com/)是一个新的社交网络，它回答了一个简单的问题:*“你的朋友在买什么？”*该公司由 Philip Kaplan、Ashvin Kumar 和 Chris Estreich 创建，目前该服务只提供邀请测试版。

那么它是如何工作的呢？表面上看，它很像 Twitter。注册后，你可以关注其他人，查看他们在买什么，在哪里买的，以及他们花了多少钱。然后，您可以向任何交易添加评论或问题。

![Blippy screenshot](../Images/c6fbd1d864fa67c0ea01392cced7ae90.png)

这是一个有争议的想法。你的 Blippy 账户可以与银行、信用卡以及亚马逊和 iTunes 等大型在线商店“挂钩”。您购买的任何商品都会添加到您的 Blippy 流中，可以公开共享或展示给您认可的人。该网站建议你也保留一张“私人”信用卡，以免在购买某些东西或秘密礼物时感到尴尬。

在许多方面，Blippy 实现了 Twitter 所没有的东西:*被动分享*。你不需要发布更新，甚至不需要上网——你的位置和行为会被你购买的东西看到。创始人已经体验了使用该服务的机会和意想不到的后果:

*   这些信息远比典型的推文有趣。
*   它可以让你看到你的朋友在买什么。你知道哪里有，他们对什么感兴趣——这对礼物创意很有用。
*   你可以找到网上交易或发现你是否被敲竹杠。
*   追踪被盗信用卡将变得更加容易。
*   可以整理和分析有价值的商业数据。一个 API 正在计划中，所以公司最终将有可能回忆起你以前的访问和购买。这可能标志着商店忠诚卡的终结。

然而，有几个大的障碍可能会影响 Blippy 的成功。人们愿意让他们的购买习惯被追踪吗？我承认大型商店已经在这么做了，但是公开信息是另一回事。我怀疑年轻一代会更渴望分享他们的行踪和他们正在购买的品牌，但这项服务仅限于那些通过信用卡购买商品的人。很少有青少年能够使用这项服务。

被动分享是一个伟大的想法，许多社交网络可能会采用类似的技巧和技术。但是要不要宣传一下自己每天的位置和活动呢？我肯定窃贼会对这些信息特别感兴趣！

最后，安全性将是大多数用户最关心的问题。尽管他们对隐私和安全做出了强有力的声明，但你仍然需要用 Blippy 注册你的银行、信用卡、商店 id 和密码。我怀疑很多人会分享这些信息，尤其是当他们读到 Blippy 的服务条款时:

*“您理解并同意，您自行决定使用本网站和服务并承担风险，并且您将对因使用本网站和服务而导致的任何损失承担全部责任。”*

银行不断建议客户对他们的 id 和密码保密；很少有人会因为 Blippy 的服务直接或间接导致的安全问题而理解或补偿你。

但是嘿，我知道什么？我从来都不明白脸书和推特有多受欢迎！尽管存在争议，Blippy 是一项执行良好的有趣服务。也许明年圣诞节我们都会用到它？

你会用 Blippy 吗？是一个很棒的概念还是太有争议？
