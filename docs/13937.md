# 重塑自我

> 原文：<https://www.sitepoint.com/reinvent-yourself/>

我曾经得到一些关于约会的好建议:“想和一个 10 分的人约会吗？那就做个‘10’。”

做生意也是如此。如果你想要“10”个客户——知名的、高薪的、令人兴奋的任务——那么你必须是“10”个开发者/设计师。

有时候这意味着重塑你自己。幸运的是，这在商业中并不难做到。事实上，我现在正在做一些自我改造。我已经有点厌倦了不时打电话来的某些类型的客户(无聊的、跟风的公司)，我想专门与具有独特商业主张的有活力的公司合作。

所以:

–我得到了投资银行家和其他在这类公司工作的人的推荐/支持。

–我已经与相同的人建立了推荐关系(并将继续这样做)。

–即将推出一个新网站，其价值主张/营销信息更能打动我的目标受众。

–我已经调整了我的解决方案集，以满足这个群体的需求。

–我将很快给这个小组写信并发表讲话，以便让大家看到我。

我们通过自己的语言和外表(包括营销材料和我们的行为方式)来塑造对自己的认知。

只要我们的语言和外表能引起目标受众的共鸣，我们就能立刻决定与众不同，并实现这一目标。

注意:如果你认为这意味着“假装直到你成功”，其实不然。不仅仅如此。你必须有实质性的支持，比如经验、对市场需求的深刻理解、引起他们注意的营销信息以及支持你的网络。这需要时间，但你能做到。

想一想“10”客户对你来说是什么，以及你如何重塑自己成为“10”服务提供商。
