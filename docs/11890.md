# 如何撰写引人注目的专业简历

对于自由职业者或企业主来说，简历是否必要似乎存在意见分歧。但有一点是肯定的；所有专业人士都应该有一本伟大的传记。

## 传记基础

专业简历就是你通常会在简历中包含的信息的故事版本。这种形式不太正式，它给你一个机会来突出你自己的一些有趣的事实，同时注入一点你的个性。

简历的主要目的是让读者准确了解你是谁和你做什么，建立专业知识和可信度，并证明你的经历和背景。所有这些因素结合起来，可以培养对你和你的品牌的信任。

即使你有一份最新的简历，在很多情况下你还是需要简历，包括:

*   发布在您的网站和博客上
*   包括在您的营销材料中
*   向客户提供建议
*   提交用于演讲、演示或教学的申请
*   包括在你开发的任何书籍、电子书、报告或专业文档中

## 你的简历应该包括什么

简历的一大优点是灵活性。您可以根据需要包含尽可能多或尽可能少的信息。通常，大多数 bios 包括:

*   目前的工作、业务或职业经历
*   您完成的出版物或演示文稿
*   您目前拥有的专业会员资格
*   您获得的奖项、荣誉和证书
*   您的联系信息

你可以在简历中加入一些元素，比如你自己的照片、你的教育背景、客户的评价或推荐，以及你工作实例的链接，从而使你的简历更加个性化。

## 写好简历的小技巧

有许多格式可以用来写一篇好的简历，但是有一些通用的元素可以让它更有效。一般来说，你的简历应该用第三人称写，用“他/她”而不是“我”。把你的简历写得好像是别人为你写的一样，可以把你和作者区分开来(即使大家认为是同一个人)。第三人称也增强了专业性，使人们更愿意相信正在说的话。

让你的开场白引人注目，吸引读者，让他们想更多地了解你。使用对话式的语气会让你的读者更容易理解。

不要害怕在你的简历末尾加入一些关于你个人的或独特的信息，并且使用一种反映你和你是谁的语气。

你会希望你的简历尽可能简短，只包括需要包括的信息。尽可能把它分成短的段落，这样更容易理解，并以链接的形式包含支持信息。

一旦你有了自己满意的简历，记住它不是一成不变的。您需要定期修改和更新它，以反映变化并保持更新。创建几个不同的版本也是一个好主意——一个缩写版本，一个长版本，甚至是适合不同受众的副本。
