# 为什么你需要一个爱好，而不是另一个个人项目

> 原文:[https://www.sitepoint.com/why-you-need-a-hobby/](https://www.sitepoint.com/why-you-need-a-hobby/)

最近我听到了很多关于个人项目、管理它们的技巧和开始的想法。虽然我认为每个专业人士都可以从一两个正在进行的个人项目中受益，我自己也是这样认为的，但我也认为个人项目会导致与工作相关的压力、[过度安排并最终耗尽精力](https://www.sitepoint.com/working-when-sick)。

当然，你可以从直接有利于你和你的企业的项目中获益良多。但实际上，个人项目只是更多的工作。这可能是你喜欢的工作，最终可能有助于你的长期收入潜力，但这是工作。

## 我真的需要爱好吗？

个人项目通常适合付费客户的工作。如果你一开始就已经很忙了，你会发现当你真的应该休息的时候，你正在做你的个人项目。再说一次，这会让你离筋疲力尽更近一步。

我知道你在想什么:“我没有时间培养爱好；爱好是浪费时间；我的空闲时间最好花在更有成效的活动上。”但是你猜怎么着？这些想法可能是一个迹象，表明一个爱好正是你所需要的。

个人项目和绝对与工作无关的爱好的结合是平衡日常工作压力的最有效方法之一。相信我，我已经吃过苦头了。

## 一个爱好对我有什么实际意义？

不相信爱好会对你有任何积极的作用？以下是培养一种爱好的一些好处:

*   这是从工作压力和永远在线的心态中解脱出来。
*   爱好可以提升你的自尊、活力和创造力。
*   这是一种结构化的休假方式(像我这样的人需要的结构！).
*   除了作为一名专业人士，你还可以了解自己。
*   一个爱好能以你意想不到的方式激励你。
*   当你回到工作岗位时，是时候改变你对工作的看法了。

## 我如何选择一个爱好？

为了确保你的爱好是一种爱好，而不是进入个人项目领域，请确保它通过了以下每个测试:

*   这是我非常喜欢并期待做的事情。
*   我没有承诺在特定的时间，以特定的方式去做这个爱好。
*   我的目标不是赚钱或促进我的事业/职业。
*   这种爱好通常会让我感到精神焕发，焕发青春。
*   当我专注于我的爱好时，我常常惊讶于时间过得如此之快。

我学到的一个惨痛的教训是，一个好的爱好不一定是强烈的；它不一定是你做的能产生可测量的、有形的最终结果的事情。事实上，有时候最好的爱好是那些让你采取更被动的方式的爱好——看电视、阅读、与孩子共度艺术和手工艺时光、散步。

你有爱好吗？它如何帮助你在工作中找到平衡？
