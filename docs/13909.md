# Visual Studio 2005 速成版可以免费下载

> 原文：<https://www.sitepoint.com/visual-studio-2005-express-editions-are-free-downloads/>

微软已经发布了 Visual Studio 2005 的速成版:

*   Visual Web Developer 2005 速成版
*   Visual Basic 2005 速成版
*   Visual C# 2005 速成版
*   Visual C++ 2005 速成版
*   Visual J# 2005 速成版

其中每一个都是 Visual Studio 2005 的简化版，专注于用单一语言进行开发(Visual Web Developer 除外，它包括 Visual Basic 和 C#)。在[产品功能对比表](http://msdn.microsoft.com/vstudio/products/compare/)中可以找到这些速成版和 Visual Studio 2005 完整版之间功能差异的完整概要。

简而言之，建设 ASP.NET 2.0 网站的所有基本功能都在这里。速成版包括简化的用户界面和有限的功能(例如，您只能与本地计算机上的数据库交互)，但适用于构建简单网站或刚刚入门的开发人员。NET 用于 Windows 桌面或 Web 开发，这些产品已经足够了。特别是 Visual Web Developer 速成版，对于目前正在使用[ASP.NET Web Matrix](http://asp.net/webmatrix/)的任何人来说都是必下载的，尽管开发人员应该注意，这些速成版并不向后兼容。NET 框架——它们只能用于开发。NET 2.0。

尽管微软之前已经宣布这些速成版的价格都是 49 美元，但它现在推出了为期一年的免费下载。一年后，微软可能会选择开始对该软件收费，但任何在免费下载期间下载该软件的人都将永久拥有它——这不是过期软件。

同样可用的还有[SQL Server 2005 Express Edition](http://msdn.microsoft.com/vstudio/express/sql/)，它是老化的 MSDE 数据库的继任者。SQL Server 2005 Express Edition 非常适合用作 Web 开发的本地数据库，只要该产品可用，它就将保持免费。
