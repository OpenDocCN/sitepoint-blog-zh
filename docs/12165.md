# 如何向客户出售统计数据

> 原文：<https://www.sitepoint.com/how-to-sell-statistics-to-clients/>

有谎言，该死的谎言，还有统计数据。然后是网页统计。营销部门喜欢网络统计，它们被公然用来促进公司和网站的成功。有多少次你读到一篇文章说 X 站点已经获得了 Y 百万的点击量？这完全没有意义，但很少有人会质疑这些数字。

您可以提供[分析工具，例如 Rick Tansun](https://www.sitepoint.com/10-web-analytics-packages-for-tracking-your-visitors/) 列出的那些工具，但是很少有客户会理解这些应用程序或它们生成的报告。客户真正需要的是帮助他们理解这些数字。

## 统计陷阱

以我的经验来看，一个人请求网页统计数据的响度或频率与他们对这些数据的理解成反比。困惑经常出现，因为:

**1。统计术语被误解**
许多人将点击等同于唯一用户，尽管“点击”是单个文件请求。该 SitePoint 页面的一次浏览就产生了 50 多次点击。你可以理解为什么新闻稿指的是 50，000 次点击，而不是 1，000 次页面浏览或 100 个独立用户。

**2。统计数据被认为是准确的**
计算机是独立的数字处理机器，所以报告必须是正确的？如果这是真的，那么不同分析工具生成的数据应该是相同的——但它们很少是相同的:

*   采用了不同的数据收集方法。例如，AWStats 分析 web 服务器日志文件，而 Google Analytics 使用客户端 JavaScript 代码。
*   没有一种数据收集方法是完美的。日志文件不包含对用户浏览器或代理服务器上缓存的文件的请求。同样，禁用 JavaScript 的人对 Google Analytics 来说是看不见的。
*   每个工具做出不同的假设。例如，一个系统可以假设单个访问者会话将在网站 20 分钟不活动后超时。另一个人可能会选择 15 分钟，或者在数字上加 5%，以说明 JavaScript 被禁用的人。

**3。统计数据无法追溯收集**
统计数据的质量取决于收集到的数据。客户通常会询问听起来合理的信息，例如“我的客户中有多少是女性？”，但除非您首先获取了适当的信息，否则无法确定该数字。

## 统计服务

统计数据可以作为一项单独的服务提供，也可以作为一项增值功能包含在您的网站报价中。将您的水平提升到竞争对手之上的典型服务包括:

**1。备忘单**
提供了以下文档:

*   解释诸如点击率、独立访问者、访问量、页面浏览量、跳出率、访问深度、登陆页面、活动、目标等术语。
*   帮助客户使用和理解报告工具。

**2。培训和支持**
一两个小时的培训通常足以解释网络统计、技术问题以及客户如何充分利用报告系统。你也可以根据客户的要求提供持续的支持、进修课程或高级培训。

**3。总结报告和分析**
虽然统计数据可能很有趣，但客户真正需要的是简洁、可靠的营销信息，例如，产品 X 在周末卖得更好，关于话题 Y 的文章获得 50%以上的访客，营销活动 Z 导致 100 笔销售。收集这些信息需要专业技术知识，对你的客户来说价值连城。

这些服务可能会耗费时间，但它们也提供了更多的机会来满足你的客户，并帮助他们增加销售。你的商业关系会变得更加牢固，他们越成功，他们就会给你更多的工作。

你卖统计服务吗？你在考虑吗？你的客户充分利用信息了吗？
