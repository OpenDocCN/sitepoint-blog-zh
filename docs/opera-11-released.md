# 歌剧 11 发布

> 原文：<https://www.sitepoint.com/opera-11-released/>

与谷歌 Chrome 和另一个极端微软的 ie 浏览器相比，Opera 有一个合理的浏览器发布时间表。当添加了大量稳定的功能时，新版本就会出现( [Opera 10.0 于 15 个月前发布](https://www.sitepoint.com/opera-10-final-released/))。

Opera 11 经过几个月的测试后于今天发布。大部分特性不久前就已经发布了，但是让我们来仔细看看新的添加和变化。

## 连接

乍一看，10.x 的变化很小，但是如果它没有坏…

最引人注目的更新是地址栏。与 Chrome 一样，Opera 现在隐藏了 https://或 https://并显示了一个图形“安全”框和挂锁图标。当您点击地址时，完整的网址就会出现，这是一个很好的接触。

“个人栏”已被重命名为“书签栏”是时候了！它现在出现在选项卡下方，但仍然可以停靠在任何一侧(对于小型宽屏显示器很有用)。

视觉鼠标手势是新的。按住鼠标右键，地图会出现，显示当你以某种方式移动时会发生什么动作。我从来都不是手势的狂热爱好者，部分原因是我更喜欢键盘快捷键，而且永远记不住哪个鼠标移动做了什么。我现在可以再试一次。

## 标签堆叠

选项卡也可以固定在栏的左侧。与以前的版本不同，只显示网站的收藏夹图标节省了空间。

## 速度

Opera 声称他们的浏览器更快，但是，由于它已经是最快的浏览器之一，我不相信很多人会注意到一个显著的变化。然而，Flash 等插件现在按需加载，而不是在启动时加载。

## HTML5

Opera 处于 HTML5 开发的前沿，提供了最好的符合标准的浏览器之一。版本 11 的主要增加是 web sockets:一种允许正在进行的异步客户机-服务器通信的技术，它绕过了标准的 HTTP 请求和响应往返。

## 扩展… *终于！*

自 20 世纪 90 年代后期的标签以来，扩展可能是 Opera 中出现的最重要的功能。Opera 粉丝传统上认为扩展是不必要的，因为浏览器提供了比竞争对手更好的内置功能。然而，扩展是 Firefox 取得成功的主要原因之一，所以很难理解为什么 Opera 停滞了这么久。

在撰写本文时，可以从 Opera 扩展库中安装超过 200 个附加组件:[https://addons.opera.com/addons/extensions/](https://addons.opera.com/addons/extensions/)。典型的广告拦截器、GMail 检查器、Twitter 客户端和 HTML 验证工具已经可以使用了。还有一个 [XMarks 工具](https://addons.opera.com/addons/extensions/details/xmarks-mini/1.0/?display=en)，这样你终于可以从其他浏览器同步书签了。

扩展开发基于 W3C Widget 规范和 JavaScript，并带有少量 HTML5 和 CSS。这是明智的，web 开发人员将能够利用他们现有的技能。从技术上来说，Opera 扩展比 Chrome 和 Safari 中类似 bookmarklet 的插件更先进。它们没有火狐的 XUL 那么复杂，但也没有那么复杂。这是一个很好的妥协，它将吸引开发者到这个平台。更多信息请参考【Opera 扩展入门。

最大的问题:*扩展能帮助提升 Opera 的形象吗*？浏览器值得拥有更广泛的受众，让我们期待它吧。

现在扩展可用了，你会尝试 Opera 吗？您是否想从现有的浏览器中进行迁移？
