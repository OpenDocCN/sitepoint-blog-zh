# 如何应对你博客上的巨魔

> 原文：<https://www.sitepoint.com/how-to-deal-with-trolls/>

如果你有博客，你就会知道我指的那些巨魔。巨魔是典型的发布粗鲁评论的人，唯一的目的是冒犯或激怒他人。他们的评论通常离题，意在引发目标的防御性或同样恶毒的反应。

巨魔可能偶尔发帖一次，或者经常发帖，但他们的目标通常是通过煽动他人尽可能地制造混乱。如果你通过反应引起巨魔的注意，他们通常会赢，所以不去注意他们是对付巨魔的最好方法。

## 不要喂巨魔

DFTT 是博客世界中一个众所周知的首字母缩略词，它基本上重申了不承认巨魔及其评论的重要性。如果你回应，你就给了他们更多的弹药来继续他们的破坏性行为。当你完全忽视它们时，它们最终可能会消失。

但是如果忽视它们不起作用呢？如果它们成为一个习惯性的海报，并开始扰乱与你的读者的对话，并从本质上用他们的消极赶走人们，你可能需要尝试不同的策略。

## 找出它们的全部内容

你通常可以通过检查他们的账户信息，如电子邮件地址、网站和/或 IP 地址，来了解一些关于巨魔的信息。即使发帖人匿名评论，如果你保留了提交评论的记录，而且他们之前也评论过，你也许能识别他们。如果是过去正常评论的人，有可能他们只是过了糟糕的一天，他们的一条评论将会结束这一天。

你也可以在谷歌上搜索你所拥有的任何信息，看看他们如何在其他渠道上采取行动，以了解这是他们的习惯还是一次性的事情。

如果你能够了解是什么促使他们发布不恰当的帖子，你就可以做出明智的决定来处理这种情况。

## 用仁慈杀死他们

巨魔希望你生气，并对他们的嘲讽做出防御性的回应。你可以通过采取主动，感谢他们的评论，把他们拉进谈话中，甚至赞美他们说的话来扭转局面。你可能会让他们措手不及，尽管他们可能会回来，并继续试图煽动你。

## 挡住他们

如果你的博客上有连续的评论，你可以选择适度地发表评论，拒绝发表任何离题的内容。虽然这可能会给你的普通读者带来不便，但这可能是保持你博客完整性的唯一方法。

## 举报他们

在最极端的情况下，如果一个巨魔将情况升级为人身攻击，侵犯你的隐私和过度骚扰，你可能需要采取法律行动。记录下你与被试的所有互动，这样你就有了一份书面记录来支持你的观点。这是一种罕见但非常严重的情况，你需要为此做好准备。
