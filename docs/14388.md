# Photoshop 中的色调混合模式

> 原文:[https://www.sitepoint.com/hue-blending-mode-in-photoshop/](https://www.sitepoint.com/hue-blending-mode-in-photoshop/)

色调混合模式非常简单:

> 色调模式:使用基色的亮度和饱和度以及混合色的色调创建结果色。

因此，本质上，顶部混合层，设置为色调，将“颜色”下面的层与你放在混合层的任何颜色，并调整颜色，以采取下面层的亮度和饱和度。

例如，让我们从我的标准玫瑰图像开始:

![](../Images/5eac5cf729fbeeb17624c97ca2302740.png)

我将在它上面放一个纯蓝层，设置为“色调”模式。玫瑰层将被“着色”，有点像蓝色(但请注意，玫瑰周围的区域灰色比蓝色多一点，因为它也调整了颜色的饱和度)。

![](../Images/442010b26cbab5eb2b2d08ad836aa21a.png)

所以，还有另一种方法可以把彩色图像变成黑白的:添加一个黑色或白色图层，并将其设置为“色调”模式:

![](../Images/02bc3df17d730c519ed3e3414f15d1be.png)

设置一个新的空白层为“色调”意味着你可以“覆盖”图像的任何部分并改变颜色。作为一个极端的例子，我只选择了玫瑰，然后用彩虹渐变填充它。

![](../Images/b191cd44f39ff5dabbf6b04d0a39f133.png)

结果是这样的:

![](../Images/7872ab8e0468b647ec6c030a9140d7e4.png)
