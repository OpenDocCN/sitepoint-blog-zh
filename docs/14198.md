# 在 Windows 上运行 Apache

> 原文:[https://www.sitepoint.com/running-apache-on-windows/](https://www.sitepoint.com/running-apache-on-windows/)

在 Netcraft 报告说去年 Apache web 服务器的市场份额相对持平之后，Apache web 服务器在 2005 年获得了新的发展势头。二月份的网络服务器调查显示，今年到目前为止，他们增长了一个多百分点。虽然这看起来很低，但这是一项对超过 5900 万台服务器进行查询的调查。

虽然许多用户是在 Unix 风格的操作系统上使用 Apache 的，但首先在 Windows 上使用 Apache 的新用户数量正在上升。这方面没有明确的统计数据——但论坛问题会有所暗示。

我是在 1997 年构建我的第一个 Red Hat 系统时接触到 Apache 的。直到 2002 年，我才在 Linux/Unix 平台之外的任何平台上构建 Apache。作为二月份的总结——我收集了几个 Windows 上的 Apache 链接，这些链接已经发送给我了。我必须承认，目前我管理的众多服务器要么在 Linux 上运行 Apache，要么在 OS X 上运行——但是我确实把 Windows 2000 Advanced 服务器算作运行 Apache 2.x 的开发机器之一
