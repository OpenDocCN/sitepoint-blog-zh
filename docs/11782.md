# SitePoint PDFs 上不再有密码！

> 原文:[https://www . site point . com/no-more-passwords-on-site point-pdf/](https://www.sitepoint.com/no-more-passwords-on-sitepoint-pdfs/)

今年早些时候，索尼影视娱乐公司首席执行官迈克尔·林顿说，“我认为互联网不会带来任何好处。”表面上，他指的是盗版和文件共享的扩散，这继续减少娱乐业的某些收入来源。然而，这种说法的潜台词有更广泛的含义，除非严格审查，否则很可能证明对该行业本身的持续生存能力是灾难性的。

无法适应不断变化的客户趋势对任何行业来说都是死刑，尤其是以技术为中心的行业。这就是为什么在 SitePoint，我们非常高兴地发布以下公告:

从即日起，所有通过我们网站购买的 PDF 图书将不再受密码保护。。

对于那些对这个决定感兴趣的人，让我告诉你一些背景。在我为 SitePoint 工作的 18 个月里，几乎每周我都会收到至少两封来自客户的电子邮件，质疑我们密码保护政策背后的逻辑。基于 SitePoint 的理念，我的回答一直是，我们在盗版问题上采取的是一种道德立场(尽管很大程度上是象征性的)。但是，随着越来越多的取消密码保护的请求涌入，我们在将客户体验放在首位的同时，还能保持这种状态多久呢？

作为一个 web 开发资源和学习中心，我们知道我们必须接受不断变化的状态——不是作为一个崇高的理想，而是作为一个规范的必要条件。当你固执地坚持旧的、过时的流程时，你不能声称自己是最前沿的——尤其是当你自己亲爱的客户敦促你前进的时候。如果我们跟不上网页设计和开发不断发展的步伐，那么我们既不是资源也不是学习中心——我们是一个博物馆。

本着赋予客户权利的精神(娱乐业似乎对此犹豫不决，[尽管有明显的动机这么做](http://theworldneedsrappers.blogspot.com/2009/08/music-is-dead.html))，我们对 sitepoint.com 的最新公告感到非常兴奋。我们希望我们所有的 PDF 用户都能享受到访问电子书的自由，而不需要使用密码。

我们要感谢所有就密码保护问题与我们联系的有用的客户，并邀请那些以前从我们这里购买 PDF 下载材料的人重新下载我们的书籍的新的、改进的、无密码的副本[https://www.sitepoint.com/books/pdf/claimpdf.php](https://www.sitepoint.com/books/pdf/claimpdf.php)
