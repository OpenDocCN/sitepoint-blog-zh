# 过去三周的回顾

> 原文：<https://www.sitepoint.com/a-review-of-the-past-3-weeks/>

回顾一下你当前的项目从何而来总是一个好主意，这样你就可以在有效的基础上进行建设。例如，我的一个 IT 服务客户最近联系了我。他已经注意到，他的大部分业务都来自于推荐和人际关系网。他觉得这很有趣，因为他在直销活动中投入了大量资金。嗯嗯……

就我而言，以下是过去几周业务的来源，希望从中可以学到一些有用的经验:

–我前一段时间向您展示的网站有大量销售，这很好，因为这意味着我可以更少地依赖咨询式项目。事实证明，拳击健身学院是最大的成功，我继续在这个网站上建立新的项目和产品。教训:构建可重复的产品；利用你的网络专长创建创收网站。

–一个已经沉寂了 9 个月的长期推荐源重新联系上了 3 个项目。尽管他没有工作给我，我还是经常和他保持联系，因为我知道他的价值(以及我对他和他的客户的潜在价值)。经验:积极思考你最有价值的推荐来源，培养这些关系。

–一家长期客户重新聘用我从事重要工作。我经常与这位客户交谈，免费为他们提供建议。然后，当一个实际的项目出现时，他雇佣我去做。他与一所主要的州立/公立大学合作，该大学对供应商和竞标有严格的规定。然而，因为他喜欢我的速度和结果，因为我总是可以为一个重要的客户提供建议，他找到了绕过这个系统的方法。教训:培养与你最好的客户的关系；为他们付出你的方式，他们也会为你付出他们的方式。

–一位过去的客户，我已经一年没有和他合作过了，他突然联系我，让我给他安排另一个任务。我以为他们不再需要我了，然而他们碰巧和我的一个推荐人聊了聊，这个人告诉他们我可以带给他们一些新的专业知识。经验:你接触的积极评价你的人越多，运气和偶然事件就越有可能对你有利。

与此同时，我继续尝试新的营销途径，写作和演讲。另外，我正在努力联系各种协会，做各种联盟项目，这样我就可以联系到他们的成员。

这是最新消息。希望你觉得有用。

关于你最近的演出，你注意到了什么？它们来自哪里？
