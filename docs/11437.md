# 登陆页面设计的七宗罪

> 原文：<https://www.sitepoint.com/designing-for-conversion/>

上周，我刚从洛杉矶的[会员大会](https://www.affiliateconvention.com)回来，在那里我做了两个报告，还听了六个小组讨论和主题演讲。最受欢迎的演讲之一是蒂姆·阿什的主题演讲，他是[SiteTuners.com](https://www.sitetuners.com)的创始人和[登陆页面优化](https://www.landingpageoptimizationbook.com)的作者。在网页设计社区中有这么多关于布局、排版、颜色使用、可用性、可访问性、PHP this 和 CSS 的讨论，很少有自由职业者或网页设计机构真正考虑对客户来说什么是重要的:结果！因此，网络上充斥着成千上万个专业设计的网站，这些网站的开发成本高达数万美元——然而它们的性能却只有其潜力的一小部分。虽然如果每个客户都愿意花钱使用[谷歌网站优化器](https://www.google.com/analytics/siteopt/splash?hl=en)、 [SiteSpect](https://www.sitespect.com/) 或 SiteTuners 引擎进行广泛测试，那将是一件美妙的事情，但每个设计师在构建转换网站时都应该记住一些基本原则——几乎每个设计师都犯有至少一项登陆页面设计的七宗罪。一些突出的案例研究包括:

*   RealAge 如何将转化率提高了 40%,并增加了 328.5 万美元的利润
*   债务减免公司如何每年增加 4800 万美元的收入
*   Stratfor 如何通过一些简单的调整将转化率提高了 81%
