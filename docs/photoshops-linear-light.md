# Photoshop 的线性光

> 原文：<https://www.sitepoint.com/photoshops-linear-light/>

线性灯光混合模式是线性加深和线性减淡的组合，但类似于生动的灯光，这种组合产生的结果略有不同。

我们先来看看 Photoshop 的定义:

> 线性光:根据混合颜色，通过降低或增加亮度来加深或减淡颜色。如果混合色(光源)比 50%的灰色亮，则通过增加亮度使图像变亮。如果混合色比 50%灰色暗，则通过降低亮度使图像变暗。

我创建了一个类似于上一篇博客中的例子。底部的图像有两层，一层设置为线性减淡，另一层设置为线性加深。在上面的图片中，相同的矩形被组合在一个图层上，并设置为线性光。较浅的颜色确实使基础层变亮了很多——但没有线性减淡那么亮；深色不会像线性加深那样使基础层变暗。

![](img/aa728a224758685572d1eb52aa264209.png)

在我的鸭子照片上设置文字层为线性光，并添加一些广告文案，结果如下:

![](img/01b34e7c55819bcae4ac7374919a3623.png)

上图(设置为线性光)和下图有细微差别。在下图中，我使用了一个白色的文字层，然后将不透明度设置为 24%。

![](img/c462b08f7c05de12b237c0b91bbf9bbc.png)

虽然没有很大的区别，线性光版本有助于大“鸭”文本稍微微妙一点；因为它实际上是照亮了基础层的颜色，而不是简单地叠加它们，所以它更像是一个背景元素。对于我想突出一点的文本(例如，实际上是公司名称的一部分的文本)，我会使用白色文本低透明度方法，但是对于我想更多地作为背景的文本或对象，我会使用线性光。
