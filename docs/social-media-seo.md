# 在搜索引擎优化活动中利用社交互动的 3 种方式

> 原文：<https://www.sitepoint.com/social-media-seo/>

你们中的许多人已经注意到了像脸书这样的社交网络对搜索引擎优化的重要性。然而，他们也可能是巨大的黑洞，吸走有用的客户互动，这些互动可能会导致链接回到你的网站，进入他们自己不断增长的围墙花园。即便如此，仅仅制作一个脸书粉丝页面，并希望它能以某种方式改善你的搜索引擎优化，也只能到此为止。

社交媒体网站已经训练用户适应你可以利用的常见在线互动。用户“喜欢”网站、页面或故事，是因为脸书训练他们这样做。其他网站让用户可以轻松地“分享”和“投票”。你也可以让你网站的用户使用这些功能。

在尝试利用社交媒体进行任何 SEO 策略之前，你需要确保你开发的东西将会给你的用户带来真正的价值，同时也给你的网站带来好处。想想谷歌和雅虎会把什么标为垃圾邮件。如果你的社交媒体应用对用户真正有用，并帮助他们执行某项功能，或帮助他们解决真正的问题，它将毫无问题地通过任何审查。

这将消除一天早上醒来发现你的客户的网站已经被取消索引的恐惧。你也可以避免营销人员有时会有的那种痛苦的感觉。这叫做良心，如果你留意这一点，你的良心就会变得清白。

## 最简单的取胜之道:分享

到目前为止，吸引某种社交互动到你的 SEO 活动的最简单的方法是允许人们分享你在你的网站上发布的内容——和他们的朋友，以及他们的网站或社交源。像 [Digg](http://digg.com/) 这样的网站非常适合分享很酷的小信息图(下面会有更多)，而 [Delicious](http://www.delicious.com/) 和其他书签服务仍然会从它们的页面传递链接汁。在这些网站中的一个上被书签标记可以成为你发布的任何给定内容的非常有价值的反向链接。

如果你使用 WordPress 来发布内容，那么将内容共享功能集成到你的站点中的工作会变得更加容易。只需安装以下插件之一，就可以为用户提供一系列链接选项，这些链接会引导用户返回到您的页面:

*   [分享这个](http://sharethis.com/):一个小巧又不太招摇的插件。当与不喜欢向页面添加太多内容的开发人员打交道时，这很方便！

*   社交:这是由 [Yoast](http://yoast.com/) 的人开发的一个很棒的插件，设计非常巧妙。它包括所有主要的社交媒体网站。

*   AddToAny :这个插件帮助用户在任何服务上分享、保存和阅读你的页面。这是一个很好的可定制的应用程序，有很多时髦的小选项。

*   [告诉一个朋友](http://freetellafriend.com/):这个插件不像其他插件那样专注于搜索引擎优化，但是它可以让你的访问者直接而方便地联系他们的朋友来推荐你的内容。

所有四个选项都很容易集成到 WordPress 站点中。你的用户可能已经熟悉了它们的功能，它们可以给你带来一些显著的好处，尤其是如果你的用户对你的服务充满热情的话。

让这些为你的搜索引擎优化活动服务的关键是从原创和写得好的内容开始。从这个内容中，你可以链接到你的网站的其他相关和语义相关的部分。基于观点的文章通常会吸引读者，不管这个观点对某个特定话题是积极的还是消极的。

## 信息图

信息图是以一种吸引人且易于理解的形式对某一组数据的图形化表示。从几种普通咖啡饮料的成分到美国联邦税收支出的详细分类，范围很广。

最近有很多关于使用信息图来试图欺骗搜索引擎的讨论。大约一个月前，Reddit 进行了一次精彩的[曝光和讨论](http://www.reddit.com/r/IAmA/comments/d7e24/my_job_was_to_game_digg_using_infographics_voting/)，这导致了一个整洁的小[信息图的发布，它是关于整个信息图生态系统的问题](http://www.buzzfeed.com/awesomer/the-truth-about-infographics)。许多人担心信息图通过使用“诱饵和开关”技术获得排名的方式:发布关于某个主题的信息图，但切换锚文本和页面的其他部分，将用户引导到完全不同的主题。

成功且合乎道德的信息图表策略的关键在于你所展示的信息的质量。无趣或琐碎的信息图很可能不会带来任何巨大的 SEO 价值。开发信息图时，从关键词研究开始。关注那些足够受欢迎、值得关注的话题。使用[谷歌的关键词工具](http://adwords.google.com/select/KeywordToolExternal)进行简单的研究只需要 10-15 分钟，这是一个好的投资回报(ROI)和没有投资回报的区别。

一旦你做了关键词研究，在这个主题中找出真正有趣的东西。这可能是你独特的见解，也可能是有争议的观点。不管是什么，一定要保证是原创。如果你经营一个大型网站，分析你自己的网站内容:也许你可以提出一个不同寻常的观点或做生意的方式，让你与众不同。

无论你的信息图在用户情绪方面的反应是好是坏，这里真正重要的是它对你的搜索引擎优化活动的价值。你可能会选择惹恼你的行业，或者迎合他们的喜好。你选择的道路完全取决于你自己——取决于你能接受多少负面反馈。

从信息图中获得 SEO 价值的最重要的方面是允许其他人在他们的网站上轻松地重新发布它，并包含一个链接，返回到您的网站和信息图的来源。这是显而易见的，但是应该以一种道德的方式来做。使用与文章主题直接相关的关键词。请确保您的用户在将它发布到他们的网站之前，可以看到它的预览。只坚持白帽技术，你将避免谷歌的任何处罚。

在你的信息图页面上有一个 Digg 按钮也可以帮助你为你的帖子获得大量的关注。你可以也应该让你的用户尽可能容易地在 Digg 和所有主要网站上分享你的帖子。使用上面提到的技术和插件，你将获得最大的利益。

总结一下信息图表的首选方法:

1.  进行关键词研究，寻找与你的业务和目标市场直接相关的大量话题。

2.  开发一个信息图，回答一个有趣的问题，是滑稽的，甚至惹恼了很多人。

3.  将带有信息图的页面链接到网站上更多的商业页面。

4.  在同一页面上添加一个易于点击的社交共享插件。

5.  向 Digg 和其他各种信息图表网站提交您的信息图表。

这个过程中最难的部分是确定一个吸引你的市场的主题，并设计一个具有最大视觉吸引力的信息图。

## 小工具

小部件是一段代码，用户可以将它添加到自己的网站上，为访问者执行一些有用的功能。例如，[这个网站](http://www.mortgagecalculator.org/free-tools/)提供了一个抵押计算器小部件，任何人都可以把它放在他们的网站上。例如，你可以看到这对于一个个人金融博客来说是一个多么有吸引力的特性。因为小部件包含一个返回到父站点的链接，所以该站点将从安装该小部件的其他站点获得 SEO 好处。

也就是说，在很长一段时间里，微件被认为是搜索引擎优化的禁忌。有很多关于推出微件的公司最终受到谷歌巨额处罚的故事。然而，在我遇到的每一个案例中，开发者都有点可疑。

默认情况下，小部件并不是一件坏事。我将跳过搜索引擎认为有问题的做法的技术细节；这要由你根据具体情况来确定。通常，你会发现你的竞争对手使用的小部件有很多好的和坏的例子。经验法则是真正为你的用户提供真正的价值。如果你的小工具让他们的生活变得更容易，并且你对它在他们的网站上的呈现方式完全坦诚，那么锚文本链接回你的网站就不会被认为是垃圾邮件。

小部件的部署有很多不同的原因。您使用什么小部件以及如何实现它们将取决于您的业务和您试图解决的用户问题。但是，每个小部件都应该具有一些关键元素:

*   小部件在发布到用户页面时应该尽可能保持静态。显然也可以包含高度动态的元素，但是我推荐包含锚文本和任何 SEO 元素的部分作为静态 HTML。

*   小部件应该包含链接到特定主题最相关页面的关键字丰富的锚文本。将此作为短语的一部分通常会带来最大的价值，例如(锚文本强调):“服务休斯顿地区超过 80 年。参观我们的 *休斯顿管道服务* 。”这并不是世界上最优雅的句子——越短越好。然而，你得到的想法是:在上下文中保持链接。

*   总是向你的用户展示代码在他们的页面上实现时的预览，包括 *所有* 相关的锚文本。用户应该充分意识到他们正在插入一个以营销文本为特色的工具。

通过遵循这三个简单的规则，你可以相当肯定你的小部件将避免营销和业务团队因被谷歌除名而愤怒！一个很棒的小指南可以在 [SEO 友好窗口](http://www.john-shehata.com/seo-friendly-widgets/)找到。

确保小部件直接链接到网站最相关的页面。如果这恰好比你最商业化的页面更深入，那就这样吧。网站上最有深度的内容最有可能从外部链接和根域中获得显著提升。然后，你可以通过内部链接策略将这种爱推广到网站的其他部分。

## 获得 SEO 好处并不邪恶！

我知道许多开发者仍然认为 SEO 是邪恶的，我们黑暗的营销艺术正在把任何东西变成污染网络的垃圾邮件。平心而论，这种怀疑很大一部分是有根据的。有些人总是试图模糊“有用”和“垃圾邮件”之间的界限；对他们来说，这是为了得到一个结果，不管付出什么代价。然而，这篇文章的重点是要表明，你的用户的正常社交互动可以为你的 SEO 带来好处。

专注于开发高质量、独特的内容，这有助于将您的网站与符合您营销目标的搜索词进行语义链接。古老的格言“内容为王”仍然适用。可悲的是，“建造它，他们会来”并没有。在这个时代，最好的内容不会简单地上升到顶部。你需要把它推向正确的方向，让你的用户尽可能容易地积极影响你的排名。

下次你开营销会议时，一位同事说，“我们为什么不用‘推特’来帮我们排名呢？”，把上面的一些技巧拿出来。你将能够在晚上安然入睡，保持你的发展道德完好无损，同时带来一系列好的想法，有助于提高你的公司或客户的排名。
