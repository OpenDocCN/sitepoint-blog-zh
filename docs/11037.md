# 表情混合行为

> 原文:[https://www.sitepoint.com/expression-blend-behaviors/](https://www.sitepoint.com/expression-blend-behaviors/)

在他的微软表情混合教程三部曲的最后一部中，[表情混合行为](https://www.sitepoint.com/article/expression-blend-behaviors)，用户体验大师 Shane Morris 完成了原型，他向我们展示了如何在[第一部分](https://www.sitepoint.com/article/rapid-prototyping-sketchflow)和[第二部分](https://www.sitepoint.com/article/microsoft-expression-blend)中构建 Silverlight 3 应用程序，以及一些故事板和动画创作。

在简要介绍了最近在拉斯维加斯举行的 MIX 大会上发布的最新微软新闻后，Shane 马上向我们展示了如何向我们上次构建的应用程序添加数据。接下来是管理动画，可以通过 Blend 的故事板工具用 XAML 来处理。

本教程结束时，你将能够用行为触发动画，这意味着，例如，每当用户从主列表中选择一个新项目时，故事板就会播放。
