# 如何建立一个电子邮件营销名单

> 原文：<https://www.sitepoint.com/how-to-build-an-email-marketing-list/>

电子邮件营销是推销你的业务、留住潜在客户并与你的客户联系的好方法。在之前的帖子中，我概述了电子邮件营销的一些[好处，分享了一些你可以发出的信息的例子，并提供了一个](https://www.sitepoint.com/email-marketing/)[电子邮件营销服务](https://www.sitepoint.com/e-mail-marketing-services/)的列表，以帮助简化你的工作。

但是如何开始建立一个列表呢？这可能是电子邮件营销活动中最重要的元素，它可以直接决定你有多成功。如果你目前没有通过网站上的注册表单或其他方式收集地址，那么建立一个列表应该是你的第一步。开始时，请记住以下几点。

## 让它变得简单

将您的注册表单放在网站的前端和中央。人们越容易看到你有一个邮件列表，他们就越容易选择加入。如果你的网站上有联系表单或在线购物系统，你也应该在这些表单上包含一个复选框，这样访问者在完成你网站上的其他活动时就可以注册。

请记住，大多数电子邮件营销服务为您的注册表单提供了代码，您可以轻松地将这些代码集成到您的网站中，因此您可能希望在此过程的早期选择一个提供商。

## 保证隐私

我们都讨厌垃圾邮件，所以要弄清楚人们注册接收的是什么，并在每张表格上注明你永远不会分享他们的信息或将其用于除邮件以外的任何其他目的。这也是一个好主意，当用户注册时，让他们知道多久能收到你的信息。

## 保持基于许可

即使你的地址簿里有几百个联系人，你确定他们会收到你的邮件，给他们一个选择加入的机会。你可以给他们发一封介绍性的邮件，让他们知道你现在有一个邮件列表，并告诉他们你将要发送给他们的信息类型。给他们一个链接以确认他们的订阅，如果他们希望接收您的消息，他们将自动添加到您的列表中。

## 泄密

一些最成功的注册格式是那些在注册时给用户免费的东西。这可以是一个免费的报告，折扣，竞赛或抽奖，或其他一些想要的激励。一定要弄清楚，在申请免费赠品时，他们是在注册你的邮件列表(当然，他们随时可以退订)。

## 不要忘记“真实世界”

你的所有订户不必通过你的网站。如果你是任何当地组织的成员，在会议上发言，或经常参加现场网络活动，随身携带一份邮件列表注册表，这样当你面对面时，你遇到的人就可以选择加入。
