# 如何利用讲故事来增加销售

> 原文：<https://www.sitepoint.com/how-to-use-storytelling-to-sell-more/>

我最近读了一本书，埃里克·韦斯纳写的《成功变得简单:从内部看为什么阿米什人生意兴隆》，书中探讨了一个事实，即阿米什人拥有的企业有 95%的成功率。令人震惊，不是吗？书中解释了阿米什人成功的诸多因素，其中一个因素是讲故事是如何促进销售的。

## 为什么有效

我们都喜欢故事，一个个人的、有意义的轶事可以以传统销售文案通常不能的方式引发行动。这是让你在竞争中脱颖而出，并在你试图销售你的产品或服务时获得(并保持)关注。

好的故事令人难忘。一个好的故事提醒人们他们想要或需要的东西，并在故事结束后很长一段时间里一直伴随着他们。一个好的故事击中了离家近的观众，让他们行动起来。

如果你能够用一个故事对你的客户和潜在客户产生这样的影响，你将会看到销售的增加，公司忠诚度的提高和更多的口碑营销。

## 规则

在你生命中的某个时刻，你可能听过一个似乎没有意义、没有关联或没有兴趣的故事。没有什么能让它尽快结束。虽然作为一种营销技巧，讲故事本身并没有什么规则，但有一些事情要记住，这样你就不会成为一个永远讲不完、无趣、无关紧要的故事的企业。

以下是在营销活动中有效使用讲故事的一些基本规则:

*   确保你的故事有意义。
*   不要让它永远继续下去。
*   将它与目标受众的特定需求联系起来。
*   让你的故事独一无二。
*   讲一个可信的故事。
*   如果它支持你的故事，不要害怕拨动心弦。
*   要有说服力。
*   不要忘记永远重要的行动号召。
*   坚持一个故事。

## 没有故事？没问题。

说到讲故事，你不会被自己的经历所限制。你的故事应该以现实为基础，但只要它引人注目、可信并以真实的方式讲述，你就可以创造一个故事。

你的故事可以是关于一个虚构的人，和你的目标观众面临同样的挑战。或者，你可以根据已经存在的东西开发一个引人注目的故事——比如，讲述一个独特的企业名称或标志是如何产生的。

选择是无限的，几乎每个企业、自由职业者和个人都可以从讲故事中受益。
