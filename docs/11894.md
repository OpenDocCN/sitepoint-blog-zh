今天，我们已经出版了这个系列的开始:[介绍](https://www.sitepoint.com/article/php-mysql-tutorial/)并且直接来自这本书，[第一章:安装](https://www.sitepoint.com/article/php-amp-mysql-1-installation/)。周末你可以阅读第一章，确保你新安装的软件工作正常，为下周的章节做好准备。在这个周末，你会有**这本书的 4 个完整章节和一篇额外的文章**，包括另一本书的摘录。

以下是您将收到的内容:

**[第二章:介绍 MySQL](https://www.sitepoint.com/article/getting-started-mysql/)**

数据库的一般介绍，特别是 MySQL 关系数据库管理系统。如果您以前从未使用过关系数据库，您会发现这很有启发性，会激起您对未来的兴趣！在这个过程中，您将建立一个简单的数据库，以便在后面的章节中使用。

**[第三章:PHP 介绍](https://www.sitepoint.com/article/mysql-3-getting-started-php/)**

这才是真正有趣的地方。在这一章中，Kevin 将向你介绍 PHP 脚本语言，你可以用它来构建动态网页，向你的访问者展示最新的信息。

**[第四章:在 Web 上发布 MySQL 数据](https://www.sitepoint.com/article/publishing-mysql-data-web/)**

在这一章中，你将把 PHP 和 MySQL 结合起来创建你的第一批数据库驱动的网页。您将探索使用 PHP 从数据库中检索信息并将其实时显示在 Web 上的基本技术。Kevin 还将向您展示如何使用 PHP 创建基于 web 的表单，以便动态地向 MySQL 数据库添加新条目和修改其中的现有信息。

**[额外收获:如何用 PHP 处理文件上传](https://www.sitepoint.com/article/handle-file-uploads-php/)**
在这篇摘自第 12 章的额外收获中，你将学习如何安全地接受网站访问者上传的文件并存储它们。
