# 猎鹰用 Talon 抓取屏幕

> 原文:[https://www.sitepoint.com/falcon-grabs-screens-with-talon/](https://www.sitepoint.com/falcon-grabs-screens-with-talon/)

Aviary 通过最近发布的[Falcon](http://aviary.com/tools/falcon)——一种简单的视觉标记工具，扩大了他们提供的漂亮的气动、以鸟为主题的图形工具。

那么到底什么是可视化标记工具呢？Falcon 基本上只是一个用于选择、裁剪和标记屏幕内容的非常好的工具。博客作者、作家和普通论坛发烧友注意了！

像 Aviary 的[其他工具](http://aviary.com/tools) , [猎鹰](http://aviary.com/tools/falcon)是免费的，并通过 Adobe AIR 的魔力在你的浏览器中实时运行。然而，Falcon 特别有用的一点是它通过 [Talon Firefox 扩展](https://addons.mozilla.org/en-US/firefox/addon/11587)与 Firefox 的集成。

**所以，这是 30 秒的旅程:**

1)安装 Talon 后，你会在你的 Firefox 工具栏上发现一个新的按钮，在你的鼠标上有一个新的右键/CTRL-click 选项。激活这个控件，你可以选择捕捉区域、整个页面或整个应用程序窗口。

2) ![Falcon resizing text](../Images/4be3ebdfc107a2254d490bfb5bd7e257.png)一旦捕获，您的屏幕图像会自动加载到您浏览器内的 Falcon 应用程序中，并配有一套小而有用的编辑工具。

3)工具箱允许您通过简单的拖放操作添加吸引人的、可拉伸的箭头、线条、正方形和圆形。由于这些是向量，它们在任何时候都是完全可编辑的。

4)文本工具允许您为图像添加注释。我必须承认，一开始我发现缺少文本大小控制有点令人不安。然而，一旦你习惯了使用缩放工具来控制所有元素的大小，甚至是文本，这就很好了。

5)当你对你的图像满意时，你可以从两个选项中选择:保存到桌面或创建一个 Aviary 帐户。后者允许您将图像直接上传到他们的图像托管服务器。

虽然我承认我部分不愿意在我的清单上再增加一个图像托管服务，但很难击败猎鹰/塔龙系统的纯粹点击并忘记的乐趣。传输过程是完全不可见的，它还能为 Twitter、BBcode、脸书和 WordPress 自动生成现成的标记。光滑的东西。懒惰是有道理的。

我已经用了 Falcon 几个星期了，它很快成了我的最爱。没错，编辑工具相对来说不算多，但足以在 95%的情况下表达你的观点。在另外 5%的情况下，当需要时，跳出去使用更复杂的工具(包括 Aviary 自己的 [Phoenix](http://aviary.com/tools/phoenix) 编辑器)就足够简单了。

最重要的是，Falcon 速度超快(即使在普通系统上也是如此)，而且干净，可以在您需要的时候准确定位。

检查一下，让我知道你的想法。
