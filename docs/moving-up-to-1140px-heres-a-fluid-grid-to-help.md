# 向上移动到 1140px？这里有一个流体网格可以帮忙

> 原文：<https://www.sitepoint.com/moving-up-to-1140px-heres-a-fluid-grid-to-help/>

在过去的几年中，许多设计师一直在为 1024 像素的屏幕宽度进行设计。800 像素的分辨率早已被埋葬，但现在是时候开始设计更高分辨率的屏幕了吗？许多用户使用 1280 像素宽的显示器和笔记本电脑，你是如何设计这种尺寸的呢？

澳大利亚设计师[安迪·泰勒](http://www.andytlr.com/)创造的[1140 像素网格](http://cssgrid.net/)可能对你有用。1140 网格完全适合 1280 显示器。在较小的显示器上，它变得流畅，并适应浏览器的宽度。网格由十二列组成，可以平均分为二列、三列、四列或六列。在浏览器支持方面，安迪的网格可以在 Chrome、Safari、Firefox、IE7 & IE8 中工作。IE6(总有一个吧？)不支持 max-width，所以网格不会固定为 1140px。它横跨浏览器的整个宽度。

[![1140Grid](img/f9cb54efa9a43afb494392d28f2bf420.png)](https://www.sitepoint.com/wp-content/uploads/2010/12/1140Grid.png)

**装订线和图像**

因为网格是基于百分比的，所以檐槽的大小会有所不同。在全宽时，檐槽为 40px 宽。使用图像时，如果图像比它所在的列小，它将以原始大小显示。如果它比较大，那么它将被缩小到列的宽度。因此，不应该定义图像的宽度和高度。

**Grid For Mobile**
Grid 并不是为支持老款 WAP 手机而设计的，但已经过测试，支持 iPhone 3 和 4、iPad、部分 Android 手机和黑莓手机。它适用于在样式表媒体和/或媒体查询中识别“手持”的设备。对于媒体查询，您甚至可以为 iPhone 4 Retina 显示屏添加 x2 图像。

你可以从这里免费下载网格。下载内容包括一个 index.html 文件和六个样式表。它是根据[知识共享署名-共享 3.0 澳大利亚许可证](http://creativecommons.org/licenses/by-sa/3.0/au/)授权的。你也可以在这里下载一个 1140 像素的[网格的 Photoshop 模板](http://download.cssgrid.net/1140_CccGrid_Template.psd)，这是基于它在 Web Kit 中的显示方式，所以它实际上是 1133 像素，因为所有浏览器对子像素的舍入略有不同。
