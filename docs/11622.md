# 创建下一个大型混搭的 12 个 API

> 原文:[https://www . site point . com/APIs-for-creating-next-big-mashup/](https://www.sitepoint.com/apis-for-creating-next-big-mashup/)

应用程序编程接口，更广为人知的名称是 API，它允许你用网络上一些大公司提供的数据做大量的事情。由于这些方便的代码，当托运人在全国范围内运送包裹时，你可以跟踪包裹，或者你可以在实时地图上看到某人的最新推文来自哪里。

当您将这 12 个强大的 API 与我们在 [10 个 Web 应用中列出的工具结合起来，在不编写任何代码的情况下构建下一个大东西](https://www.sitepoint.com/10-web-apps-to-build-the-next-big-thing-without-writing-any-code/)时，您可能最终会构建出每个人都想使用的下一个大 mashup。那么，是什么阻止了你尝试成为网络巨星呢？

## 映射 API

Bing 地图:Bing 地图以前是微软的虚拟地球，对于那些不想使用谷歌地图创建另一个混搭的人来说，Bing 地图是一个不错的选择。微软提供了三种不同的 API 用于 Bing 地图，允许你做一系列的事情，例如选择数据的显示方式，决定你是否需要 3D，使用地理空间属性等等。

[![](../Images/c140a977d78abfd4521edb85f3c3fa23.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/bingmaps.png)

谷歌地图 :作为世界上最流行的 API 之一，谷歌地图工具可以让你做用户在标准的谷歌地图网站上可以做的任何事情。你可以添加 Google AdSense 来帮助你的 mashup 赚钱，迎合移动用户等等。基本上你能想到的，这个 API 都能搞定。

[![](../Images/7782aabdd7a1e96613a2712e39c508fe.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/googlemaps.png)

雅虎地图(Yahoo Maps):雅虎地图应用编程接口(Yahoo Maps API)提供了许多选项，让你以任何你能想象到的方式操作数据。实际的地图提供了 Flash，Ajax 和地图图像。然后，您可以使用他们的其他 API 从 Yahoo！本地列表、交通和天气预报、即将举行的活动和 Flickr 照片。

[![](../Images/1dcd9bff80302f2503210d7ae3f1fb65.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/yahoomaps.png)

## 购物 API

**[亚马逊电子商务](https://affiliate-program.amazon.com/gp/advertising/api/detail/main.html)** :如果亚马逊联盟计划对你来说不够强大，所有电子商务网站的国王还提供了一个 API，允许你推出自己的工具。Amazon API 利用 REST 和 SOAP，允许你调用产品数据，建立搜索，并确保你的会员 ID 包含在所有推荐中。

[![](../Images/8ff09a6220c45203ee5d4b80c7ea231a.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/amazon.png)

:易贝 API 为你提供了一个独一无二的机会，你可以从拍卖行赚取你在网站上做广告的物品的佣金。这意味着你甚至可以通过展示自己拍卖的物品来赚取一些钱。API 使用 SOAP 和 REST 协议，并以 XML 的形式交付结果。如果你的应用程序通过了他们的测试软件，你就可以被列入官方目录，并有资格每天接受 150 万次呼叫。

[![](../Images/49942797ef24c028c99060af236fe0c3.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/ebay.png)

**[【Shopping.com】](https://partners.shopping.com/APIProgram.html)**:Shopping.com 的 API 允许你以一种最适合你需求的方式将他们购物搜索引擎中的所有商品整合到你的网站中。

[![](../Images/dd0fc113988ef320de27e72e4436ae25.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/shopping.png)

## 社交媒体 API

Delicious.com 有了 Delicious API，你可以构建应用程序，让你添加或删除书签，检索特定日期的数据，抓取标签等等。

[![](../Images/134d87a3dc77709d63209e08e654e27a.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/delicious.png)

:Flickr API 允许你构建应用程序，从网站中检索公开的照片、视频、收藏夹、朋友、群组池、讨论等等。你也可以用它来构建上传照片和视频的工具。Flickr API 支持许多协议，包括 REST、SOAP 和 XML-RPC。

[![](../Images/4410052fd25caf6a6d10476be46c05e0.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/flickr.png)

Twitter :很难想象 Twitter API 还没有完成所有能想到的事情，但是亲自尝试一下又有什么坏处呢？数据可以以 XML、JSON、RSS 和 Atom 格式检索，以提供关于用户、推文、朋友、关注者、收藏夹等的信息。

[![](../Images/44ad7792669871cb4aa9e21263d47f25.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/twitter.png)

**[YouTube](http://code.google.com/apis/youtube/overview.html)** : YouTube 提供了多种 API，让你可以做很多不同的功能。data API 允许您将 YouTube 功能整合到您的网站中，例如上传、执行搜索、创建播放列表等等。有两种不同的播放器 API 允许你选择普通的播放器，或者给你一个完全定制的播放器。最后一个是基于小部件的，它允许你在你的网站上选择的任何地方添加小的 YouTube 工具。

[![](../Images/600bb1553ea52dcdaf99b7ee2c1b288c.png)T2】](https://blogs.sitepointstatic.com.s3.amazonaws.com/images/lists/apis/youtube.png)

**[谷歌 AJAX 搜索](http://code.google.com/apis/ajaxsearch/)** :谷歌 AJAX 搜索 API 允许你在你的网站上添加网页搜索、本地搜索、视频和博客搜索。如果你的网站是一个留言板，API 将允许你的用户轻松地将搜索结果复制到他们的消息中。
