# 在线声誉管理:如何处理负面宣传

> 原文：<https://www.sitepoint.com/handle-negative-publicity/>

理想情况下，你接手的每一个项目和建立的每一段关系，都将有助于建立职业声誉。您可以通过以下方式提高正面声誉的有效性:

*   做得很好
*   以客户服务为导向
*   让自己变得平易近人
*   与同行业的其他人合作
*   建立个人关系

但是，即使你做的每件事都是正确的，也可能有一天你会面临负面宣传。有害的反馈可能因多种原因而产生——误解、你的错误行为、不同的观点，或者任何你可能没有完全理解的其他原因。

你对负面反馈的反应取决于评论的类型，谁说的，在哪个论坛说的，以及它有可能损害你的声誉。但是总有一些方法来衡量负面宣传的风险，并确定如何最好地处理它。

**想清楚**

我们是人，所以我们对负面评论的第一反应通常是愤怒、好斗和/或防御。你能做的最糟糕的事情就是不考虑情况就迅速做出反应，因为你可能只会让情况变得更糟。设身处地为对方着想，对自己坦诚。深呼吸，问自己这些问题:

*   评论是否属实？
*   我能看出这个人是如何看待我的行为的吗？
*   我做了什么被误解或曲解的事情吗？
*   我错了吗？

很多时候，你可能没有看到负面反馈的到来，所以你会感到惊讶。通过花时间对自己坦诚面对这种情况，你可以避免造成进一步的伤害。你甚至可能想问问你信任的朋友或同事对这种情况的看法，以帮助你获得正确的观点。

**回应还是不回应？**

不是每个负面评论都值得回应。事实上，你可能会决定不回应，因为你觉得这种情况最好干脆忽略。如果影响微乎其微，不要在不必要的时候为自己辩护，以免火上浇油。

在某些情况下，你可能想找到问题的根源，试着离线解决。一次私人谈话可能会发现你原本不知道的信息。如果你错了，你可以纠正这种情况，并要求作者公开撤回他们的评论或提供进一步的信息，以抵消一些影响。你永远不知道，这种不幸的情况可能是一段很有潜力的新关系的催化剂。

你也可以通过在自己的网站或博客上发表公开评论或发表感谢信来应对这种情况，并提供你自己的观点。但是，一定不要过分防备或人身攻击对方；那只会让你看起来不专业。

利用它成为你的优势

“所有的宣传都是好的宣传”这句话可能不完全准确，但你肯定可以将一些消极的情况转化为积极的事件。负面宣传可以给你一个纠正错误的机会；它可以为您提供一个解决问题的平台；它能让你做得更好。

请记住，无论你选择什么方法来处理这种情况，你都不能改变他人的行为。以你认为最好的方式处理这种情况，但不要被其他人的消极情绪拉离正轨。
