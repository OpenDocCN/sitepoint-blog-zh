# 微软从 Windows 7 中移除 IE 不会有任何影响

> 原文：<https://www.sitepoint.com/ie-windows-7-europe/>

微软将在 10 月份推出不带 ie 浏览器的欧洲版 Windows 7。这一决定是在欧盟进行反垄断调查后做出的。在 Opera 发起的一个正在进行的法律案件中，欧盟的初步声明得出结论，微软将浏览器与 Windows 捆绑销售违反了欧洲竞争法。

根据该公司泄露的信息:

> 为了确保微软遵守欧洲法律，微软将在欧洲发布不包含 Windows Internet Explorer 的单独版本的 Windows 7。
> 
> 微软将单独免费提供 IE8，如果欧洲的个人电脑制造商选择在 Windows 7 电脑上预装 IE 8，将会变得更加容易和方便。PC 制造商可能会选择安装一个替代浏览器，而不是 IE 8，而且一直都是这样，如果他们愿意，他们可能会安装多个浏览器。

每个 Windows 7 有两种版本(家庭版、专业版、旗舰版等。)将在欧盟地区上市:

*   **Windows 7 E** :不带 ie 浏览器的版本，以及
*   **Windows 7 N** :没有 IE 和 Windows Media Player 的版本。

有几个原因可以解释为什么这是这个软件巨头的明智之举:

**1。微软遵守欧盟法律**
Windows 7 E/N 将没有默认浏览器，因此微软似乎致力于维护竞争法。我不是法律专家，但我相信欧盟在处理反垄断案时会经历一段艰难的时间。

**2。XP 和 Vista 中仍有 IE**
XP 和 Vista 用户已经安装了 IE——升级到 Windows 7 会移除浏览器吗？这似乎不太可能，也没有人希望这种情况发生。

**3。IE 仅隐藏**
Windows 依赖浏览器组件来提供几个核心服务，因此，尽管 [IE 可以在 Windows 7](https://www.sitepoint.com/ie-windows-7-uninstall/) 中卸载，但它只在表面上影响操作系统。

Windows 7 E/N 很可能以‘未安装’状态提供 IE。全新安装操作系统后，您会做什么？搜寻浏览器安装程序或快速重新启用 IE 并下载您需要的东西？

**4。OEM 厂商仍会选择 IE**
微软已经明确表示，个人电脑制造商可以提供替代或选择的浏览器。有多少 OEM 厂商会这么做？

*   浏览器选择屏幕需要自定义应用程序和最新的安装程序。开发和维护该软件会给制造商带来直接成本。
*   或者，假设 OEM 提供了不同的浏览器。新手会因为没有蓝色的“e”图标而感到困惑，并抱怨他们的新电脑不能上网吗？

任何使用 Firefox、Opera、Safari 或 Chrome 的人都已经下载并维护了他们选择的浏览器，因此默认的 IE 安装不太可能让他们感到困惑。对于 IE 用户来说，这就不一定了。提供另一种浏览器会增加 OEM 的支持成本，而且他们无法将用户直接转到 Microsoft 帮助热线。

**5。微软不需要推广其他浏览器**
欧盟和 Opera 的首选解决方案是在 Windows 安装过程中显示“选择浏览器”屏幕，可能在控制面板中显示。对于微软来说，这将是一场逻辑噩梦，我敢肯定，供应商们仍然会抱怨他们的浏览器没有其他浏览器突出。

该解决方案还要求微软宣传其他浏览器。IE 市场份额居高不下的原因之一是用户不知道有替代产品，或者无法安装。

微软将 ie 浏览器从欧洲版 Windows 7 中移除的决定，对浏览器的市场份额影响微乎其微。这可能不是歌剧或欧盟想要的结果，但我怀疑反垄断斗争至少晚了 10 年。
