## 采访记录

路易斯:大家好，欢迎来到 SitePoint 播客。在本周的节目中，我很高兴与我一起接受采访的是 Jan Lukacs，他是一家名为 Paymo 的公司的总经理，该公司是一种在线时间跟踪和发票服务。嗨，简。

一月:嗨路易斯。

路易:你好吗？

简:谢谢你邀请我。我很好，谢谢你。

路易斯:很好，很高兴你能来。Joel Falconer 向我们推荐了你，他一直在为各种 SitePoint 网站写博客，认为邀请你来谈谈你的产品和你作为自由职业者以及与 Paymo 的许多自由职业者一起工作的经历会很酷。

简:是的，我很乐意分享我从我们的经历中学到的任何东西。

路易斯:太好了。所以，在我们开始之前，你想谈谈你的网站，你的产品吗？它叫 Paymo，在 [paymo.biz](http://paymo.biz/) ，对吗？

简:对， [paymo.biz](http://paymo.biz/) ，对。

Louis: 你能告诉我们一些事情吗？告诉我们是什么让你有了创办这家公司的想法，以及它的进展如何。

Jan: 当然。因此，基本上 Paymo 是一个时间跟踪服务，一个在线应用程序，允许你管理你的团队和你自己的时间，它也适用于有业务的自由职业者。它的核心是一个时间跟踪服务，但我们围绕它建立了一系列工具，如项目管理和发票，所以它真的可以帮助你在线管理你的业务，特别是如果你正在做的项目需要知道一项任务需要多长时间之类的事情。

**路易:**对。你运营这个网站多久了？

我们从 2008 年开始，在一年的时间里，我们免费提供这项服务，所以可以说我们没有任何商业产品，2009 年，在我们从用户那里获得大量反馈后，我们重新设计了整个应用程序，基本上这就是我们开始销售的产品。

**路易:**对。那么现在这项服务还有免费选项吗？

简:是的，当然。从一开始，这就是我们的哲学。在我们的职业生涯中，我们使用的许多工具都是开源或免费的，它们帮了我们很多，所以当我们决定创建一个高级版本的产品时，我们知道我们必须免费提供应用程序的核心服务，所以我们将永远有一个免费版本的服务。

**路易:**对。所以你提到在做 Paymo 之前，你只是一个标准的雇佣型网络商店，对吗？

简:是的。我们是一家为其他公司和个人做项目的网络开发机构。

**Louis:** 那么，Paymo 是你为内部使用而开发的产品，然后意识到你有一个可以销售的产品，还是你知道最终它将是一个你会尝试并投入大众市场的产品？

**Jan:** 不，我们是为自己开发的，因为当时我们查看了可用的解决方案，但没有一个能满足我们的需求，所以我们开始在空闲时间创建它，然后我们发现其他人也可能会使用它，2008 年我们创建了一个网站，免费提供服务，这对我们来说确实是一件全新的事情。 我们在创造产品或服务方面没有任何经验，所以反馈很好，我们决定改变我们的模式，专注于产品。

路易斯:好的，那么目前你没有做任何客户工作吗？

**Jan:** 我们分离了我们的公司，所以我们为 Paymo 创建了一个新公司，旧团队现在只专注于 Paymo，旧团队的一部分专注于网络开发代理，所以他们现在完全分开了。

**路易:**对。您之前提到过，当您考虑开始或考虑开发一款时间跟踪产品时，您提到没有一款产品能满足您的需求。在这种情况下，我感兴趣的是，这不是一个全新的市场，就像很多次你听说的这些代理公司在产品上拓展业务，他们正在制作一些不存在的东西，但在这种情况下，互联网上肯定有一些其他的时间跟踪和发票以及项目管理解决方案。是什么让你觉得你需要一个不同的产品或新产品？

**Jan:** 主要是我们使用了一些服务，但团队中没有人真正使用这些服务，这背后的主要原因是他们在浏览器中运行，向系统中输入时间非常耗时，而且你必须查看所有这些表格，这非常复杂。所以我们认为，如果我们想从这项业务中赚钱，我们真的需要知道我们在项目上花了多少钱，所以这对我们来说一直是个问题，我们得到的报酬总是比我们实际工作的少，解决方案是创建一个桌面小工具，一个可以在你的桌面上运行并与在线系统交互的应用程序。所以这在我们开始开发 Paymo 的时候是不存在的。

**路易:**对。

这是我们创建 Paymo 的主要原因之一。我们想让在系统中添加时间变得更加容易。

路易:对。当你说一个桌面小部件时，你从一开始就建立了这个跨平台吗？

简:是的。我们的团队中有人使用 MAC 和 PC，所以这也是一个大问题，因为我们可以使用一些解决方案，但是没有一个可以跨平台工作。

**路易:**对。

Jan: 这又是我们需要的东西，我们制造了它。

**路易:**对。现在，你已经有了 Mac 和 Windows 的桌面插件？

简:对，没错。

**路易:**还有 Linux 还是不是？

Jan: 这不是很简单，但是你也可以在 Linux 上运行。

**路易:**对。你们内部有在桌面平台上工作的专业知识吗？还是你们都是网络人员，必须出去学习这些新技术？

**Jan:** 我们的一位同事确实是桌面专家，所以这对我们来说没什么大不了的，这确实帮了大忙，而且我们也有建设大型网站的经验，所以这很有用，因为像这样的系统确实需要各种技能，而这些技能在从事小型项目时通常是得不到的，所以这些东西肯定对我们有帮助。

**路易:**对。在这里，我想问一下，你对桌面小工具和网站本身(后端)使用了什么样的技术？

Jan: 对于后端，我们使用 PHP 和 MySQL，对于桌面，我们在 Windows 的情况下使用 C++或 Visual C++。在 Mac 上，但这里有一件很酷的事情，核心部件实际上只是一个网页，但它被一个桌面应用程序所包装，所以这使我们很容易维护它，并在 Mac 或 Windows 和 Linux 上运行它。

路易斯:对，那很酷。所以，我感兴趣的事情之一，我想很多听众也会感兴趣，是从一家主要专注于服务客户项目的公司到一家专注于向更广泛的互联网提供付费产品的公司的转变。我以前和一些人说过，当你考虑做这件事时，你应该小心，因为在同一个团队中很难同时做两件事。那么，你们是如何做到这一点的？在从与客户打交道到开发这一产品的转变过程中，你们遇到了哪些挑战？

Jan: 是的，过渡相当困难，因为它们是完全独立的业务，适用于一种业务类型的东西根本不适用于另一种业务类型，所以我们确实犯了一系列错误，甚至在我们开始开发产品时都没有意识到，如果你从这个角度来看，一切都花了比应该花的时间更长的时间。但这也让我们——拥有一个主要业务让我们可以创造这个产品，而不需要去找投资者，不需要贷款。

路易斯:对，没错。

简:所以我们用网络代理赚的钱来创造产品。然而，在最初的几年里，我们没有像应该的那样关注产品，这真的使业务发展得更慢，更有机，如果你从一开始就有一个业务顾问，并在营销和诸如此类的事情上花很多钱。但最终我们都过得很好。

**路易:**对。如果有人在听或参与或经营一家网络代理公司，并对他们认为可以开发和销售的产品有一个想法，但他们不一定认为他们有时间在客户工作之余做这件事，你会给处于这种位置的人什么样的建议？

**Jan:** 顶级建议，那很难(笑)。我认为他们应该真的，真的分析市场，看看机会是什么，并尝试尽可能详细地了解他们正在尝试启动的新业务，因为这真的，真的很难，每个利基市场都有大量的竞争，所以追求利基市场不再是好建议，因为每个小利基市场都有大量的竞争对手，你真的需要看看你是否有技能和资金来创建一个新的业务。

**路易:**对。

简:因为否则的话，至少一两年的工作就会付之东流，如果你做不到的话，那会很痛苦。

**Louis:** 然而，有趣的是，你的案例几乎证明了这一点，因为我几乎想不出一种针对网页设计师和网页开发者的基于在线服务的产品比这些时间跟踪产品管理应用更具竞争力，我认为原因是网页设计师和开发者希望开发他们会使用的应用。

简:对，绝对是。

Louis: 这是网页设计师和开发人员会使用的主要在线工具。尽管如此，你们设法进入市场相当晚，仍然建立了一个相当成功的业务。

实际上，当我们发布的时候还不算太晚，所以当我们发布第一个版本的免费应用程序时，在 2008 年只有几个其他的在线时间追踪工具相当不错。但在我说我们犯了很多错误的那一年，从 2008 年到 2009 年，如果我们真的真的专注于产品，我们可能会在那个时期获得很多市场份额，与此同时，我们的许多竞争对手也涌现出来，因为他们也看到了这个市场的机会。

路易:对。

**Jan:** 所以我现在的建议是，如果你想启动另一个时间追踪应用程序的话——

**路易:**是别！(笑)别这么做。

**Jan:** 不做。我不是说这是为了避免其他人进入，但现在真的很拥挤。

路易:对。

Jan: 所以我会寻找那些有两三四个竞争对手，但没有 20、40、50 个竞争对手的服务或利基市场，因为这样创业真的真的很难，除非你做了非常非常棒的事情(笑声)。

路易斯:我想总有一种区分的方法。那么，你们一开始是如何营销你们的应用程序的，是不是只是一种有机的方式，你和你的朋友交谈，他们又和他们的朋友交谈，你们并没有真正专注于营销，还是你们付出了一些努力，试图让应用程序出现在人们面前？

**Jan:** 在最初的几个月里，我不知道如何营销这样的东西，因为我来自服务行业，对我来说这是全新的。所以当我们开始使用这款应用的时候，当免费版出来的时候，我们真的没有任何营销资金，所以要让对这种事情感兴趣的人看到这款产品需要很大的努力，所以我们联系了很多博客，关注了一些关于时间跟踪之类的推文，我们真的试图在使用我们的系统后从人们那里获得意见，所以从我们的初始用户那里获得反馈是至关重要的，我们听取了他们的意见。

路易斯:抱歉，我要打断你一下。如果你不介意我问的话，在最初阶段，你的系统有多少用户？

简:第一年可能有两千个。是的，当你完全是市场上的新手时，让人们使用你的系统真的真的很难，我真的很感谢最初的用户对我们系统的信任，作为一个建议，如果你试图开始一个网络应用程序或服务，找到符合早期采用者特征的人是非常重要的，可以说，因为他们愿意接受沼泽，停机时间之类的东西，只是因为他们喜欢而尝试服务。因此，去尝试找到这些人并让他们使用你的服务是非常重要的，只要你能做到。

路易:对。这一定非常有趣。许多这类服务和创业公司都位于美国、澳大利亚或加拿大。你们的基地在哪里？

Jan: 我们的总部设在欧洲的罗马尼亚，这对我们来说有点困难，但我们会尽力而为。

路易:对。那么，当设计你的应用程序时，你正在为说英语的观众开发一个英语应用程序，你的策略是什么？让文案和所有界面以一种对说英语的观众来说清晰易懂的方式完成？

简:所有的工作都是我们自己做的。我认为，在美国和澳大利亚的公司工作多年后，我们对这个市场有了很好的了解，在这方面我们没有遇到任何大问题。

路易斯:这真的令人印象深刻，因为我是说，我正在看界面，文案写得非常好，对我来说非常清楚，所以我必须对做文案的人说一声祝贺，因为对于一个母语不是英语的人来说，我真的印象深刻。

Jan: 谢谢，谢谢。

**路易:**对。所以我想稍微回顾一下，你们已经作为一个自由网站开发和设计团队工作了一段时间，你说你们也和国际客户一起工作？

Jan: 是的。

在你决定创办 Paymo 之前，你从事自由职业或项目工作多长时间了？

简:我们已经工作了大约五年。

**路易:**对。显然你在那方面很成功；当你开始做 Paymo 的时候，有没有一个瞬间你认为我们可能会在一些事情上花费时间和金钱，而这些事情现在可能会在我们已经有了一个成功的业务的时候实现？

简:这是我们必须做出的决定。在我们看到我们推出的免费应用程序的吸引力后，这确实是我们必须决定下一步做什么的时候了，我们是专注于产品还是网络开发机构，这不是一个容易的决定。当然，当时这个应用程序没有赚到任何钱，所以这是一个风险，但我们真的想尝试一些新的东西，这种环境允许我们继续我们的旧业务，而不要过多地参与其中，我很高兴与几个能够接手业务并专注于此的人合作，我们成功地将公司分成两家公司，并且效果很好，至少到目前为止。

路易:(笑)不，那是一个伟大的成功故事。说到这里，到目前为止，你想抛出一些统计数据，就像你们做得很好，我在主页上看到一些统计数据，但事情对你来说进展顺利吗？

**Jan:** 是的，我们的业务一直在增长，我们的大多数客户都对这项服务感到满意，并继续订阅这项服务，这让我们非常高兴，它让我们能够建立新的东西并进行扩展，我们真的想创造一些人们实际上不害怕使用的东西，因为，让我们面对它，时间跟踪是人们不喜欢的东西(笑声)，但我们正在努力使它尽可能令人愉快，最终它真的支付更多，这就是 Paymo 的意义所在；它会让你得到更多的报酬，因为一旦你对自己或你的团队如何花费时间有了清晰的认识，你就能据此做出决定，你就会获得更多的利润。这不是什么——我是说这是我们每天从用户那里听到的。

路易:是的，我想是的。对于许多刚刚开始做客户工作的人来说，这很容易——小请求，对，15 分钟的小请求——哦，你能不能去更新一下这个文件

简:那会要了你的命。

路易斯:——你每天做其中一件事，突然之间，你实际上挣的钱比你在汉堡王挣的还少。

简:是的，我们犯过这样的错误。在我们创业的前三年，我们有很多这样的客户，我相信每个自由职业者和小公司都知道这一点。在某个时候，你必须做出决定，你必须解雇你的一些客户，除此之外别无选择，否则他们会让你永远被拖累，这对你和他们都没有好处。

**Louis:** 对于你的情况，我想提几件事，一件是你说你正在开发新的东西，你能给我们一些我们可以期待的暗示吗，如果有人在使用 Paymo，他们可以期待新功能的出现，或者你正在考虑新产品？

Jan: 是的，我们想在下一个版本中做一些彻底的改变。目前我真的不能谈论它们，但是我向你保证它们会是很酷的东西。

路易斯:每次我问这个问题，他们都这么说。

简:是啊(笑)。

**路易:**我要继续努力；有一天有人会说，是的，事实上我可以告诉你，下一个版本将包括这个功能和这个特性。

当我们更接近我们的愿景时，我相信我能够谈论它，但我在这里真的很诚实，我们做的很多东西被其他人很快复制，诚实地说，在这个行业中，每个人都在复制每个人，所以这是事实，但有几个关键特征使产品与众不同，这就是你有时必须保守秘密的地方。

路易:对。我觉得有趣的一件事是，我们提到你们有一个免费计划，你说你计划保留这个选项。

简:是的，永远，是的。

**Louis:** 看，这对我来说真的很有趣，因为我记得——所以事情是这样的，大约三年前我第一次研究这个问题，当时我正在做一些自由职业者的工作，我已经很久没有研究任何时间跟踪或项目管理 web 应用程序了，所以我把它搁置了两三年。最近，就在上周，我的女朋友打算开始做一些自由行政工作，她问我，“哦，你用什么开发票？”我说，“哦，就用这个吧，”她查了一下，说，“一个月 20 美元，”我说，“哦，不，不，有免费计划，”她说，“不，没有。”随着市场变得越来越成熟，许多这样的服务已经停止了他们的免费计划。

简:是的。

路易斯:但是你愿意承诺这种事永远不会发生在佩莫身上。

简:对，绝对是。每个人都是——如果你看看更大的玩家，不仅在这个利基市场，而且在在线服务领域，每个人都说把你的免费计划藏在某个地方，无论你需要什么来计算你的转换，免费计划都会让你失去平衡；我真的不相信这一点，我们希望向我们的用户免费提供核心服务，因为我们知道他们没有赚很多钱，他们中的一些人负担不起。所以，这里的概念很简单，我们给你免费的东西，我们希望它能帮助你成为一个客户，基本上就是这样。
