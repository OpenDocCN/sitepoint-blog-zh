# 歌剧的问题是

> 原文:[https://www.sitepoint.com/the-problem-with-opera/](https://www.sitepoint.com/the-problem-with-opera/)

我想从一开始就表明，我认为《T2》这部歌剧很棒。我安装了它并定期使用，但它从未成为我的默认浏览器。

Opera 目前占有全球桌面市场的 2%。尽管该公司的创新速度不变，但从未大幅超过这一数字。市场已经增长，因此 Opera 用户越来越多，但 Chrome 等新浏览器的增长速度要快得多。

营销是一个原因。Opera 永远比不上谷歌的广告预算，而且该公司位于挪威，因此它对庞大的美国市场的影响较小。名称和标志也不像其他网络品牌那样容易辨认。

然而，这从未阻碍过 Opera 在移动领域的发展。那是一个不同的市场；竞争对手较少，诺基亚通过在其手机上预装浏览器提供了帮助。但是鉴于 Opera 在移动设备上如此稳固，更多的人不使用桌面版本有点令人惊讶。

## 权力=痛苦？

在我看来，Opera 最大的优势就是它的没落。它是高度可配置的，并提供了许多选项——可能太多了。充分利用浏览器需要时间和精力。

Opera 的配置方式在其他应用程序中是看不到的。也许这是创新，但它会导致混乱。我已经使用这个应用程序很多年了，但是我经常很难找到我需要的选项。

![Opera screen](../Images/1d90e477986334404c99ab77a34083b8.png)比如 10.5 版本隐藏了菜单栏但是很容易显示出来(见截图)。这很好——但是试着找到“隐藏菜单栏”选项。它不在“视图”菜单、外观或首选项对话框中。连帮助都没有告诉你选项出现在“文件”菜单里？

这可能是一个小的深奥的怪癖，但它是浏览器如何惹恼新用户的一个例子。Opera 特有的术语也没什么用——有多少人知道“个人工具栏”是一个书签工具栏，或者“Opera 链接”是一个同步工具？

Opera 从第一天起就可以使用，但是您需要投入时间来了解可用的特性，它们是如何工作的，以及它们是如何配置的。很少有人愿意攀登陡峭的学习曲线，特别是当竞争浏览器提供更容易和更少迷惑的体验时。

Opera 确实吸引了一部分高级用户，尽管我还是更喜欢 Firefox 的终极定制。如果 Opera 想要吸引大量的主流观众，公司应该:

**解决令人困惑的术语**
，这可能意味着复制其他供应商的命名约定，但它将提供一个更容易的用户迁移路径。

**从基本浏览器中删除不常用的功能**
Unite、Notes、一些工具栏、蜻蜓、邮件和聊天等功能在默认情况下都是关闭的。它们仍然可以作为安装选项或插件提供。

**简化配置选项**高级选项可以被隐藏，但是一个单独的选项框就可以把它们全部带回来。

Opera 理应获得更高的市场份额，该公司报告称，在推出[欧盟浏览器选择屏幕](https://www.sitepoint.com/european-browser-choice-today/)后，下载量增长了 300%(尽管这可能部分是因为现有用户正在下载新版本)。然而，他们还没有透露实际数字，我怀疑与 Chrome 和 Firefox 相比，他们仍然相对较低。

歌剧应该改变吗？它应该面向更多的受众吗？你会如何改进浏览器？
