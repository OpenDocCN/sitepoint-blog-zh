# 了解 Debian

> 原文：<https://www.sitepoint.com/getting-know-debian/>

在这个快节奏计算和带宽迅速增长的时代，有一个操作系统脱颖而出——Linux。

在众多可用的 Linux 发行版中，最受欢迎的是 Debian。尽管其他发行版主要是商业事务，Debian 改变了配方。Debian 发行版中的每一个软件都是免费的。Debian 不包含任何商业软件。

虽然你可能认为确保 Debian 完全免费的决心有点极端，但从长远来看这是有意义的。很有可能一些 Linux 供应商会关闭商店，改变他们的业务方向或者以其他方式调整他们的实践。因此，您今天使用的发行版可能在 10 年后就不存在了。

Debian 的情况并非如此。Debian 项目完全由志愿者运营，并不寻求盈利。这基本上意味着，尽管有继续改进 Debian 的意愿，但不管经济问题如何，项目总是会进展的。

本文将介绍 Debian，它能做什么，以及安装它的指南。

##### Debian 的好处

除了我们刚刚讨论过的长期利益之外，使用 Debian 还有许多明显的技术优势，首先是它的广泛可用性。Debian 可以从网上免费下载，也有光盘。不需要填写注册表格，也不需要遵循任何复杂的流程来获得软件。

但是 Debian 最大的吸引力可能是它的软件升级工具 apt-get。任何使用过其他 Linux 发行版的人都会熟悉 RPM 工具。RPM 最初是由 Red Hat 开发的，用于打包预编译的软件，以便可以轻松安装，并且用户不必编译源代码。

虽然在理论上是好的，但在实践中，RPM 技术遇到了依赖关系的问题。当一个软件需要运行另一个软件才能运行时，就会产生依赖性。例如，假设我们希望使用 [GIMP](http://www.gimp.org) 。这个包要求我们也运行 wget、aalib、libc6、libgimp、libglib、libgtk、xlibs 等。显然，找到所有这些包(以及它们的依赖项)是一个令人沮丧和耗时的过程，毫不奇怪，术语“依赖地狱”被用来描述这种场景。

多亏了 apt-get，Debian 绕过了这个过程。在 apt-get 中，开发人员创建了一个强大的工具，当一个包被安装时，它会自动让 Debian 检索所有的依赖项，并按正确的顺序安装它们。因此，在我们前面的例子中，我可以用一个命令安装 GIMP——我甚至不需要找到 GIMP 下载文件！Debian 会直接从它的服务器上取回这个包。

然而，Debian 不仅仅是一个方便的软件升级系统。它提供了各种工具，使得编译内核、配置包和处理模块(仅举几个例子)的任务变得非常容易。然而，Debian 绝不是一个初学者的系统，它没有提供给用户 GUI 向导引导的演练，但是如果你有勇气，或者你已经有了使用 Linux 的经验，它是一个明智的选择。

让我们来看看 Debian 安装中涉及的一些关键概念。这个讨论只是作为一个指南，因为我们没有足够的空间来深入讨论整个过程。建议您将这些指南与 Debian 附带的文档结合使用，这些文档在 Debian 网站上提供。

##### 获得 Debian

因为它是一个社区驱动的项目，Debian 通常很容易掌握。有多种安装方法可用，包括网络安装，光盘，引导软盘等。最简单的方法是创建自己的 Debian 安装光盘(或者从 Linux 供应商那里购买)。

要创建安装 CD，我们首先需要获得一个 ISO 映像。该 ISO 文件描述了安装 CD 的确切内容，因此可以将其刻录到 CDROM 中。总共有 7 个 Debian 安装 iso，但是安装只需要第一个。我建议您简单地获取第一个 ISO，并使用 apt-get 通过互联网检索任何其他需要的软件。要找到你需要的 ISO 图片，请访问 http://www.debian.org/CD/http-ftp/。

Debian 项目将允许您从该页面上列出的镜像中下载 ISO 映像，或者您可以使用 Jigdo 应用程序更清楚地指定您的 ISO 是如何组装的。更多信息见[http://www.debian.org/CD/jigdo-cd/](http://www.debian.org/CD/jigdo-cd/)。尽管 Jigdo 是一个更好的解决方案，我还是建议 Debian 新用户获得一个普通的 ISO，以使安装过程更加顺利。

这里需要指出的重要一点是 Debian 有 3 个版本:稳定版、测试版和不稳定版。稳定版包含较旧的软件，但已经通过了严格的测试程序，没有突出的错误报告。Debian 的测试版本较新，但是这里包含的软件包已经 10 天没有错误报告了(所以，总的来说，它是一个稳定的系统)。不稳定的系统包含直接进入 Debian 的新包，可能有需要修复的问题。

如果你知道你在做什么，不稳定版本是唯一被推荐的。我建议您使用稳定版或测试版。

找到 ISO 映像后，您需要将它们刻录到 CD 上。这可以在大多数 CD 刻录机的 Windows 中通过选择菜单的从映像刻录 CD 选项来实现。在 Linux 中，您可以使用以下命令刻录 CD:

```
cdrecord fs=12m speed=4 isoname.iso
```

##### 引导系统

一旦你刻录了你的 ISO 或者购买了一个现有的 Debian 光盘，把光盘放入你的光驱并重启机器。您可能需要重置 BIOS 中的一些选项，以确保可以从光盘启动。如果一切顺利，您将看到一个包含大量文本的屏幕。然后，您可以按 Enter 键继续并引导安装程序。

启动安装程序后，您会看到一个蓝屏，您可以从这里开始进行安装。您将被问及一系列简单的问题，例如您在哪个国家，您的键盘布局等等。您可以使用 Tab 键在选项之间移动，并使用 Enter 键选择选项作为这些问题的答案。

##### 设置分区

在安装过程中的某个时候，会要求您对磁盘进行分区。这是重要的一步，所以在你到达之前，你应该清楚你想把 Debian 放在哪里。

Debian 系统使用 3 个基本分区；一个保存内核的'`boot'`分区，一个保存文件系统和相关文件的'`/`分区，以及一个在系统内存耗尽时使用的交换分区。还有其他可用的挂载点(比如用于将主目录保存在另一个分区上的“`/home`”)，但是，我刚刚描述的 3 个挂载点适合初学者设置。

你需要考虑的另一件事是，你是否用另一个操作系统(比如 Windows)双重引导你的机器。如果是的话，我建议你把每个操作系统都放在不同的磁盘上，但是如果你把它们都放在同一个磁盘上，Debian 会简单地利用空闲空间。

首先，会询问您要对哪个磁盘进行分区。Linux 将每个磁盘称为`/dev/hd(n)`，其中`n`表示一个字母(例如，第一个磁盘将是`hda`)。SCSI 磁盘的处理略有不同，但命名概念相似。当您选择了要分区的磁盘后，您将输入 cfdisk。该实用程序允许您创建一个分区并设置其类型。创建一个大约 50MB 的`/boot`分区，将其类型设置为 Linux。交换区的大小应该设置为内存大小的两倍，它的类型应该是 Linux 交换区，并且您的“`/`”分区应该占用剩余的可用空间。此外，确保您的 50MB 分区是可引导的。

虽然您已经创建了分区，但是还需要设置挂载点(比如/boot)。Debian 将询问您希望在哪个分区上设置挂载点，然后您可以将合适的点设置到相关的分区。

##### 设置模块

完成这些初始步骤后，您将被要求为您的设备设置模块。如果您知道设备需要哪些模块，您可以从显示的不同类别中选择它们。但是，如果您不确定需要哪些模块，您可以浏览类别，直到找到合适的模块。如果您的设备没有列出，您可能需要重新编译包含您的设备的内核。这可以在安装后完成。

##### 安装软件包

当您配置好磁盘和模块后，系统会要求您重新启动系统，安装将继续进行。在安装的后一阶段，会要求您选择要安装的软件包。只需选择您需要的不同软件包，安装程序将直接从光盘安装它们。

流程的这一部分通常将应用程序分成任务列表(有点像包的组)。如果你选择一个组，比如 C++编程，你会得到一个 C++编译器，库等等。不过，您可能还需要安装其他组，比如文本编辑器。我建议，如果可以的话，你安装所有的东西(假设你在一台台式机上工作),然后删除不需要的软件包。

##### 设置帐户

安装例程的另一个重要部分是用户和 root 帐户设置。Debian 和其他 Linux 发行版一样，有一个主要的超级用户，叫做‘root’，可以进行系统维护，比如升级和配置。普通用户应该建立自己的帐户，这比 root 用户更受限制。

在安装过程中，Debian 会询问根密码，并让你创建一个帐户。一旦你创建了一个帐户，你就可以在安装后登录你的系统了。在 Debian 中可以用“adduser”命令创建更多的帐户。

安装后设置

完成安装程序后，您将看到一个登录屏幕。此时，您可以登录系统并开始使用它。虽然系统已经安装，但是您还需要设置和配置许多其他东西。不幸的是，我们没有时间在这里涵盖所有这些项目，所以，相反，我将涵盖主要领域:上网和升级您的系统。

上网取决于您用来连接到互联网的硬件。最常见的选择是简易调制解调器和电缆调制解调器。安装调制解调器是一个相当简单的过程。你需要知道你的调制解调器在哪个端口上，并准备好你的登录信息。端口号很可能采用`/dev/ttyS(n)`的形式，其中 n 表示您的端口号。

您可以使用各种程序来设置您的调制解调器或直接编辑文件，但我推荐 pppconfig 程序。使用这个命令行程序，您可以指定您的连接细节，如电话号码，用户名和密码，该程序将试图检测您的调制解调器。该计划还提供了各种高级设置，您可以更好地配置您的连接。此设置假定您使用的是外置调制解调器。您需要首先为内核配置一个内置调制解调器。Debian 可能已经自动检测到了，但是如果你有困难，网上有很多帮助。

设置好调制解调器后，您可以通过键入以下命令进行连接:

```
pon <accountname>
```

您可以使用“`poff`”断开连接。

设置电缆调制解调器有点不同，因为它总是开着的，只需要使用 DHCP 获取一个 IP 地址。各种电缆调制解调器可能需要不同的驱动程序(快速的 Google 搜索应该会有帮助)，并且在设置和连接过程中可能会有变化，但是通常您只需要编辑/etc/network/interfaces 文件。添加以下几行:

```
auto eth0  

iface eth0 inet dhcp
```

然后，您可以通过重启 Debian 网络来获取 IP 地址:

```
/etc/init.d/networking restart
```

##### 使用 apt-get 升级您的系统

正如我们前面所讨论的，Debian 发行版最强大的元素之一是升级系统的简易性。但是在升级之前，首先需要配置高级包工具(apt)。如果在提示符下键入 apt-setup 作为 root，则可以设置从中检索包的 apt 服务器的位置。此过程完成后，您可以使用以下命令更新可用软件包列表:

```
apt-get update
```

然后，您可以使用以下命令升级整个发行版:

```
apt-get dist-upgrade
```

将向您显示将升级多少(哪些)软件包，以及需要下载多少数据。如果你需要的话，你可以停止下载，之后 apt 会恢复而不会丢失你下载的包。除了升级整个发行版之外，您还可以使用“`apt-cache search <term>`”来搜索软件包。举个例子，如果我想搜索基于数学的程序:

```
apt-cache search maths
```

然后你会看到一个包的列表。如果您看到想要安装的软件包，如 mathwar，请键入:

```
apt-get install mathwar
```

您也可以使用以下方法删除它:

```
apt-get remove mathwar
```

最后，要改变你正在使用的 Debian 版本(稳定、测试或不稳定)，只需编辑`/etc/apt/sources.list`，将单词`stable`改为`testing`或`unstable`。然后，您需要更新您的软件包列表和发行版升级，以完全升级您的系统。不过，有一个警告:降级通常不是一个好主意；有可能，但不推荐。

##### 结论

本文简要介绍了安装 Debian 发行版所涉及的各种问题和过程。正如我在开头所说，Debian 绝不是初学者的发行版。虽然最终受益于灵活性的提高，但要真正让包运行起来，还需要许多更复杂的步骤。虽然这篇短文已经涵盖了您需要了解的主要领域，但是它可能还不足以指导您完成 Debian 的安装过程。

Debian 是一个稳定、强大的 Linux 发行版，并将一直存在下去。该系统每天都在增长，并提供超过 8710 个安装包。你学习这个系统所花的时间将会得到回报，为了让你开始，这里有一些有价值的资源。
