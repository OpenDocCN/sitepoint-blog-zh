# FullCodePress:评审内幕

> 原文:[https://www.sitepoint.com/fullcodepress-inside-the-judging/](https://www.sitepoint.com/fullcodepress-inside-the-judging/)

在 FullCodePress，24 小时的工作已经结束——每个网站都已完工，客户和团队正在享受他们应得的休息。现在，是时候让评委们仔细审视每个网站了。让我们先睹为快。

我们藏在 CeBIT 的媒体室里，与客户和团队交谈过。就像每个团队一样，我们只有 24 小时来了解客户、他们的需求以及他们对网站的期望。现在网站已经完成了，我们的任务是根据一些关键标准来评估每个网站。

**标记**:由您的真实判断。我将进行检查，以确保网站上的标记语义适当、命名合理、结构清晰，当然也有效。作为一名内容管理背景的人，我还对每个网站的 CMS 是否有助于用户保持结果标记的整洁，并鼓励用户养成良好的习惯感兴趣。我见过很多好的网站在过去因为 CMS 在清理某人的代码错误方面做得不好而被毁掉，或者更糟的是，引入了他们自己的验证错误。

安全性:由来自 IBM 的 Lee Kinsman 评判。她将使用 IBM 的 Rational AppScan 来评估每个站点抵御可疑人物攻击的能力。“Rational AppScan 攻击网站并冒充黑客，”她说。“它针对网站的所有已知漏洞运行自动化测试。该套件包括一系列针对该平台上已知问题的平台特定测试。”Lee 对一个团队选择 PayPal 的事实印象尤为深刻。“这是一个安全的选择，”她说。“鉴于只有 24 小时，他们选择他们知道会安全的东西是件好事。”

当然，李会自己做一些测试。“在它运行的时候，我浏览了一下网站，并对表单字段运行了一些基本的脚本，只是想看看它们是否会带来什么有趣的东西。例如，一些表单可能会使网站容易受到 SQL 注入攻击。”

可访问性 : Gian Wild，一个著名的可访问性忍者，将评估每个网站的访问强度。由于澳大利亚团队正在与一个残疾人权益团体合作，这对于这个团队来说尤为重要——但鉴于彩虹青年对开放和包容性网站的要求，我们可以肯定她会以同样严格的标准对待“黑人守则”!

“我针对 WCAG1 进行了测试，还使用了各种残疾人可能会使用的众所周知的技术，例如关闭图像、增加文本大小和关闭样式表。”吉安说。“在某些方面，我印象非常深刻，而在另一些方面，我相当担心。例如，在某些情况下，增加尺寸或关闭图像会导致导航等重要元素消失，这显然不是很好。”

服务器端技术:Atlassian 的 Lachlan Hardy 负责评估每个团队做出的框架和后端选择。“显然，我关注的是服务器端技术和基础设施的可持续性。我对什么是 CMS 以及他们如何使用它非常感兴趣，所以我印象深刻地看到一个团队只用一个特定的产品只做了很少的修改就构建了它，但他们没有注释——这可能会给未来的更新带来一些麻烦。另一个站点根本没有修改他们的解决方案，尽管对框架本身做了一些修改，并且没有充分利用他们 CMS 的能力。”

设计:由 Adobe 的 Andrew Larkin 评估，他将寻找网站各自的设计如何与客户的简介相匹配。

“目前我两样都很喜欢，”安德鲁说。“我的三个关键标准是视觉吸引力、专业外观、艺术完整性和色彩和谐。因此，我在研究每个人翻译和传达客户需求的方式。”目前，每个都有利弊，但我总是需要一段时间来让设计的影响渗透进来。"

**创新和移动设备**:Opera 的 Lachlan Hunt 将评估每个站点对创新技术的使用情况及其对移动平台的适用性。如果你一直关注 HTML5 的进程，你可能会从 HTML5 工作组中认出拉克伦的名字。那么什么是创新呢？“我在寻找新技术的创造性应用，”拉克伦说，“比如 CSS3、微格式、任何新的 DOM APIs、canvas 元素的使用，也许是最近的浏览器支持的东西，以及旧浏览器的优雅退化。”没有一个团队敢尝试 HTML5，所以没有进行评估，尽管一些 CSS3 已经潜入每个网站的样式表中。

至于移动体验，拉克伦带来了各种 Opera 驱动的移动设备，用于测试每个网站，以及移动 Safari。“我借给每个团队设备，让他们进行测试。其中一个澳大利亚人甚至有一部装有 Opera Mobile 的手机，所以没有借口！”

**内容:** Natasha Hall 将密切关注每个团队如何成功地吸引那些有望成为网站主要用户的受众。每个客户端都有各种各样的用户，所以内容充分地服务于各种用户是很重要的。

Natasha 说:“我的感觉是，因为代表们深入参与了两个组织的方向和内部运作，所以任何人都很容易忘记内容对从未到过那里的人来说是什么样子和感觉，因为新访问者不具备内部人员所具备的背景和深刻知识。所以我将以一个新访客的视角来看待它。如果我对这些组织一无所知也能理解，那么希望其他人也能理解。”

**用户体验** : Janine Sanderson 将评估网站和内容管理系统的用户体验。“我在寻找引人注目的用户体验。我所说的令人信服，是指团队真正理解网站使用场景的证据。它必须满足目标受众的需求，帮助他们实现目标，并迫使他们采取行动。我正在寻找网站来帮助用户踢屁股，这是凯西塞拉的一句话。这两个网站都是为了给用户赋权，让他们对自己的处境感觉更好，所以网站需要与我对话，迫使我采取行动。”

谁的网站至高无上？评判工作刚刚结束，一旦结果揭晓，我们一定会及时向您通报。
