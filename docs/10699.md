# 如何在 Photoshop CS5 中使用木偶扭曲操作图像

> 原文:[https://www . site point . com/how-to-operating-images-with puppet-warp-in-Photoshop-cs5/](https://www.sitepoint.com/how-to-manipulate-images-with-puppet-warp-in-photoshop-cs5/)

木偶扭曲是 Photoshop CS5 中备受关注的新功能之一。木偶扭曲有点像液化过滤器。它的工作原理是在图像上放置一个网格。您将使用网格在您想要控制移动的位置放置图钉，或者确保没有移动–实际上是将图像的一部分固定到文档窗口。木偶扭曲可以应用于“正常”层，文字层，形状层，智能对象和矢量蒙版，好消息是，它的使用相当简单。此命令的结果可以是细微的变形或修饰，也可以是极度的扭曲。

1.打开要应用扭曲的图像。我正在使用这张图片，它是我从 Stock Xchng 免费下载的。你可以在这里得到它。这是一张很大的图片，所以我对它进行了裁剪，去掉了假人左侧的一大块空白。

[![dummyCropped](../Images/f97d416309721e64c1227b5b63ca65b5.png "dummyCropped")T2】](https://www.sitepoint.com/wp-content/uploads/2010/08/dummyCropped.jpg)

2.如前所述，木偶扭曲几乎适用于所有类型的层，但它不会**适用于背景层。如果你想要扭曲的物体被隔离在它自己的层上，你也会得到一个更好的结果，所以这就是我们现在要做的。使用魔棒工具(W)选择虚拟对象周围的白色区域，然后按 Shift + Ctrl + I (Windows)或 Shift + Cmd + I (Mac)来反转选区，以便选择虚拟对象。**

[![2-DummySelected](../Images/08d71461661a8b5bbf6742e8e6d34331.png "2-DummySelected")T2】](https://www.sitepoint.com/wp-content/uploads/2010/08/2DummySelected.jpg)

3.现在，要将选区放到它自己的图层上，请按 Ctrl + J (Windows)或 Cmd + J (Mac)。在图层面板中，你现在应该可以看到假人在它自己的图层上，周围是透明的。

[![3-LayersPalette](../Images/d4ac0a7093625c7d1dadf5f883ba3b2c.png "3-LayersPalette")T2】](https://www.sitepoint.com/wp-content/uploads/2010/08/3LayersPalette.jpg)

3.确保你的新虚拟层被选中(不是背景层)，然后选择**编辑>木偶扭曲**。虚拟对象上会出现一个网格。

[![4-dummywithpins](../Images/198866c7f56000102ba35de127d9752b.png "4-dummywithpins")T2】](https://www.sitepoint.com/wp-content/uploads/2010/08/4dummywithpins.jpg)

屏幕顶部的选项栏中有几个可用选项。

[![3-Options](../Images/feccc8b7f7f0b8a293f3adebd7bba43d.png "3-Options")T2】](https://www.sitepoint.com/wp-content/uploads/2010/08/3Options.jpg)

**模式:**决定网格的弹性。刚性，顾名思义，不是很有弹性，而扭曲会给你一个超有弹性的网格。正常介于两者之间。
**:密度:**设置网格的间距，移动销的微小变化会导致较大的扭曲。对于较大的图像也是一个很好的模式。刚性是相反的，当你的物体
更加僵硬时，推荐使用刚性，就像人类的附属物一样。
**扩展:**扩展或收缩网格的外边缘。
**显示网格:**取消选择仅显示引脚，不显示网格。

4.关闭背景图层的可见性。只需单击将大头针添加到您想要锚定的图像区域。我想弯曲假人的左肘，所以我在头部、上半身、肩膀和手腕周围放置了大头针，作为不会移动的锚。然后单击肘关节添加另一个点，并拖动以扭曲该点。当你完成拖动后，点击键盘上的回车键提交木偶扭曲。

如果你想取消一个木偶网格，只需按 Esc 键。如果你想删除一个大头针，点击它，然后按退格键。

[![5-DummyWithPinsDragged](../Images/4dd462e5b11dc749ae79bbabf2b5b255.png "5-DummyWithPinsDragged")T2】](https://www.sitepoint.com/wp-content/uploads/2010/08/5DummyWithPinsDragged.jpg)

5.除了拖动大头针点，您还可以围绕大头针旋转。再次选择**编辑>木偶扭曲**再次将网格放在假人上。单击虚拟对象的左膝以添加另一个大头针，然后按住 Alt(Mac 上的选项)键并将光标放在大头针附近，但不要放在上面。拖动以旋转。您的旋转角度会出现在选项栏上。

[![6-RotatePin](../Images/cea87897ee8d8a8f4c2f47b8c0ceaf7b.png "6-RotatePin")T2】](https://www.sitepoint.com/wp-content/uploads/2010/08/6RotatePin.jpg)

那是傀儡曲速。这可能不是你每天都会用到的东西，但它是一个非常有用的扭曲图像的工具，比液化过滤器更容易操作。
