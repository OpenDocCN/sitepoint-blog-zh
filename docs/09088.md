# 标志设计:接触目标受众的五个关键要素

> 原文：<https://www.sitepoint.com/logo-design-five-key-elements-for-reaching-your-target-audience/>

[![](img/59c1206d26460017528041c7325e088c.png "railroad")](https://www.sitepoint.com/wp-content/uploads/2011/11/railroad.jpg)

耐克、麦当劳、苹果和可口可乐有什么共同点？如果你的答案是它们都是非常成功的大公司，那么你就非常正确。但这不是唯一正确的答案。事实上，另一个答案可能有助于解释这些企业惊人的成功——它们都有独特而难忘的标志。

那么为什么标识如此重要呢？好吧，考虑一下。作为个人，你希望人们记住你的脸，对吗？好吧，把同样的思路运用到你的生意中。你的企业也需要一张面孔——人们会立即认出并永远记住的面孔。更进一步，你还会希望这张脸做更多的事情——激发信任和培养忠诚度。适合你的企业的标志可以帮助你实现所有这些目标。

选择正确的标志并不像看起来那么容易。你可能会想出一些可爱又时髦的东西，但对你试图锁定的观众来说毫无意义。或者你可以有一个在广告牌上看起来令人震惊的标志，但当你把它放在报纸广告上时，它就变得单调乏味了。有许多因素你需要考虑，有许多陷阱你需要避免。以下是一些帮助你设计最适合你的标志的建议:

### 标志应该是简单的。

我们都见过耐克的“嗖嗖声”还有比这更简单的吗？现在试着说出一个更有效的标志。想出一个不容易。事实是，在标志设计中，简洁是一件非常好的事情。它使一个标志易于识别，高度认同，并不断难忘。它在几乎任何地点或环境中都引人注目。它会一直留在你的脑海里。过度使用这种很酷的新字体，或者尝试将几种想法融合到一个图像中，这可能很有诱惑力，但是不要这样做。相反，使用 K.I.S.S .方法(保持简单，笨蛋！).

### 标志应该在任何地方都好看。

想想你的 logo 可能出现的每一个地方，然后想想它在每一个地方会是什么样子。你的 logo 是不是有很多细纹？如果是这样的话，当你缩小尺寸的时候，它可能不会保持很好。你正在使用位图图形软件设计你的标志吗？如果是这样，可以考虑改用矢量格式。放大位图图形通常会突出显示像素，并且会使图像失真而无法识别。颜色呢？爱上他们很容易，但如果你爱上了他们，你就掉进了一个陷阱。开始你的黑白设计。然后，只有在你确定当颜色被移除或只有一种颜色存在时，设计仍然保持良好之后，才添加颜色。请记住，您希望您的徽标在任何背景和所有设置下都好看。

### 标志应该是独一无二的。

你的标志就是你的品牌，所以要让它与众不同。没人认为模仿者有什么了不起。你也不希望你的顾客把你的商标和另一家公司的混淆。看看你的竞争对手在做什么从来都不是一个坏主意。但是当涉及到商标时，利用这些知识让你的设计与他们的不同，而不是模仿他们的。另一个要避免的陷阱是从网上下载股票图片，并把它们用在你的商标上。你可能会冒侵犯版权的风险，即使你躲过了这一劫，仍然很有可能有人在某个地方使用同样的图片。

### 这个标志应该能引起你的目标受众的共鸣。

你的标志应该适合你的业务，但这并不意味着它必须显示你的业务是什么。金色拱门上没有奶酪汉堡，苹果标志上也没有任何电脑的迹象。一个标志需要做的最重要的事情就是和你的目标受众交流。如果你经营一家儿童玩具店，在你的商标上有一个玩具的图像或者有“玩具”这个词也不是很重要。更重要的是使用一种充满童趣并吸引孩子的配色方案或字体。

### 标志应该是令人难忘的。

你想让人们记住你的商标，最好的方法是避免每隔几年就更换。所以你应该总是避免抓住最新趋势的诱惑。潮流不会持久，你的商标会很快变得过时。另一个要避免的陷阱是字体挑战。你希望你的字体和你的图标相匹配——但是如果它们匹配得太好，那么这两者可能会争夺注意力。你需要找到正确的平衡点。此外，请记住，太多的字体可能会干扰眼睛和排斥的记忆细胞！最多使用两个，这样你的标志将保持清晰和难忘。

你的标志就是你的名片。坏的会让人们忽略你，好的会让他们注意到你。一个非常好的会让他们一直想着你。像对待你的外表一样对待你的标志——让它看起来更好。如果你花时间让你的商标闪闪发光，你将会看到你的生意开始闪闪发光。
