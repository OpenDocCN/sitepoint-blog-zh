# 重新定位能提高你的底线吗？

很有可能你会错过很大比例的销售或销售线索。

电子商务的平均转化率为 2%，潜在客户网站的转化率也好不到哪里去。这意味着 98%的网站访问者不会马上转变为 T1。

出现这种情况有很多原因——一些消费者只是在做研究，其他人在比较不同零售商的价格，甚至更多的人只是被他们电脑上的东西或我们每天都经历的现实世界分心。但是，如果有一种方法可以找到那些没有改变信仰就离开的访问者，并给他们第二次机会呢？如果你可以用展示他们在你的网站上浏览的确切产品的广告来专门针对他们，会怎么样？这就是*行为重定向*的用武之地。

### 什么是重定目标？

行为重定位，或简称重定位，是基于访问者之前的浏览行为向其展示广告(尤其是展示或文本链接广告)的过程。在大多数情况下，重定向是用来更频繁地向以前访问过网站的人显示广告，但不会转化为客户或线索。

让我们以一个普通的利基电子商务商家为例…也许是一个定制服装零售商，如 Indochino。游客可能会在谷歌上搜索定制西装，然后参观 Indochino.com。他们浏览几个不同的页面，甚至可能会在购物车中添加一套西装或夹克。但是他们会被电子邮件通知、即时消息甚至婴儿的哭声分散注意力，没有完成购买就离开了网站。

但是当天晚些时候，甚至是那周晚些时候，他们又开始浏览网页了。它们可能在一个与男装完全无关的网页上，但它们显示的是来自 Indochino.com 的展示广告，因为它们在 T1 之前访问了该网站。那就是*行为重定向*。

看起来，重新定位是一种非常强大的营销策略，因为它只向以前合格的候选人展示广告。他们已经通过浏览你的网站、向他们的购物车添加商品或下载白皮书(触发因素由你决定)对你的产品或服务表现出了兴趣。他们比不合格的访问者更有可能改变信仰。

利用重新定位可以通过回收流失的访问者来显著提高利润率和降低每个客户的购买成本。AdRoll.com、Retargeter.com 和 Chango.com 是三家专门从事重新定位的公司，谷歌 Adwords 也提供重新定位服务。

### 谷歌削弱了再经销商吗？

谷歌最近做出了一项改变，这可能会对网站所有者和营销人员收到的关于其推荐人的数据产生深远影响。以前，当访问者进行谷歌搜索并点击进入一个网站时，特定搜索短语的信息也会被传递，然后可以由谷歌分析或其他统计包进行分析。

谷歌最近宣布，默认情况下，他们将不再为任何登录用户传递这些推荐数据。虽然这只占全球搜索量的 10%,但我很想知道美国有多大比例的搜索会受到影响。有一点是肯定的，拿走这些有价值的信息将伤害 Chango 和 AdRoll 等公司，这些公司依赖这些数据根据访问者的浏览历史向他们投放广告。
