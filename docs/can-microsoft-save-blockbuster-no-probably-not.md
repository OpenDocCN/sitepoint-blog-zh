# 微软能拯救百视达吗？不，可能不会

> 原文：<https://www.sitepoint.com/can-microsoft-save-blockbuster-no-probably-not/>

对百视达来说，事情看起来不太妙。在过去的五年里，该公司的股票一直处于自由落体式的下跌状态(T2)，在过去的几个月里跌幅尤其大。租金收入一个季度接一个季度地下降，根据 Compete 的数据，他们的总接入计划几乎无法与主要竞争对手网飞竞争。

但是如果你最近看了任何一个头条新闻，你可能会认为《一鸣惊人》正在蓬勃发展。这家曾经沉睡、笨拙的巨头，在过去的一周里已经改变了方向，做出了一些关键的举动来努力赶上竞争对手，包括推出机顶盒(正好赶上假期)，并宣布他们正在[与微软](http://www.dallasnews.com/sharedcontent/dws/bus/stories/DN-blockbuster_01bus.ART0.State.Edition1.4a219e8.html)合作，使用 [Live Mesh](http://www.mesh.com/) 框架([我们的报道](https://www.sitepoint.com/microsoft-opens-live-mesh-to-all/))将视频内容部署到多个屏幕上。这家总部位于德克萨斯州的公司的情况开始好转了吗？

这些无疑都是百视达寻求保持相关性的关键进展，但这可能太少，太晚了。理论上，Blockbuster 的 2wire MediaPoint 机顶盒与竞争对手的竞争是有利的:他们的总内容比网飞少，但新发布的内容更多，渐进式播放技术对网络连接较慢的人来说看起来不错，单点定价可能对临时租户有吸引力。但是网飞在更多的设备上提供流媒体服务。三星和 LG 的 Tivo、 [Xbox 360](https://www.sitepoint.com/xbox-360-netflix-another-nail-in-the-blockbuster-coffin/) 、蓝光和 DVD 播放器——网飞集成将这些变成多功能设备，对许多人来说将胜过 MediaPoint(大多数？)消费者。

与微软的 Live Mesh 集成可能最终会产生一些优秀的应用程序，例如可以在一台电视上暂停租借的电影，稍后在另一台电视或移动设备上，在电影的同一点继续观看。但百视达的问题是，如果 Mesh 成功了，我们看不出网飞没有理由不使用它。

毕竟，网飞也是微软的合作伙伴——正如我们所说，他们在 Xbox 360 上播放电影，他们的流媒体直播服务[使用 Silverlight](http://netflix.mediaroom.com/index.php?s=43&item=288) 。对于微软来说，以牺牲其他高调合作伙伴为代价与百视达建立独家合作关系是没有意义的。

从本质上讲，随着越来越多的人开始下载电影，而不是在光盘上租赁，百视达仍将继续追赶更灵活的竞争对手，如网飞、苹果和亚马逊。

然而，鉴于微软不会拯救百视达，百视达*将*对微软有利。微软已经能够在不到两年的时间里将其 Silverlight 技术推广到 25%的联网计算机上，这几乎完全归功于高调的企业部署，包括 Blockbuster 的部署。通过依靠他们的企业合作伙伴，微软可能会对 Mesh 做同样的事情，并将其推向消费者。不管 Blockbuster 使用 Mesh 对这家电影租赁公司来说是不是一个积极因素，它几乎肯定会为微软提供一个该技术能力的伟大演示。

尽管 Blockbuster 和微软的合作关系今天成为新闻，但它实际上是在大约一个月前的微软专业开发者大会上宣布的，当时 Blockbuster 和 BBC 正在开发的网格应用程序在台上演示。微软向我们提供了一段重磅炸弹应用程序演示的视频，可以在这里查看。
