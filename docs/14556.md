# 开源开发实验室

> 原文：<https://www.sitepoint.com/open-source-development-labs/>

开源开发实验室(Open Source Development Labs)简称 OSDL，其使命是评估、测试和全面致力于将 Linux 改进为一个专业平台。

随着全球向亚洲的扩张，OSDL 的重要性持续增长，并且作为 Linus Torvalds 的家乡，它可能会成为 Linux 世界的中心。(注意:Linus 和他的家人现在搬到了 OSDL 的总部俄勒冈州，在那里他将在该组织中担任更重要的角色。)

他们的实验室为那些寻求为电子商务、公司和公共数据中心等部署 Linux 的人提供了功能性数据源。虽然他们的注意力不在运行 Linux 的小企业上，但他们的努力确实通过托管设备的数据中心直接影响了 web 开发人员。

OSDL 出版了关于他们的测试和活动的文献，并允许公众要求在他们的实验室中进行项目。为了更好地了解他们的运营背景，请查看他们的[演示页面](http://www.osdl.org/newsroom/presentations/)。

目前，他们专注于几个领域，包括电信级 Linux(电信和大型企业公司)、数据中心 Linux(开发人员和主机以及企业领域感兴趣)、桌面 Linux 和 Linux 内核。

他们所有的实验活动和出版物都可以在 OSDL 网站上找到。
