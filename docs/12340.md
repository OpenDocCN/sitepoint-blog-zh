# Muxtape 带着独立天赋回归

> 原文:[https://www.sitepoint.com/muxtape-returns-with-indie-flair/](https://www.sitepoint.com/muxtape-returns-with-indie-flair/)

去年八月，流行的 mixtape 分享网站 [Muxtape](http://www.muxtape.com/) 因“RIAA 的问题”而关闭翻译过来，这意味着该网站有法律上的麻烦，因为它允许人们上传和分享包含版权音乐的虚拟混音带。近 6 个月后，该网站的创始人贾斯汀·韦莱特[发布了新 Muxtape 的预览版](http://muxtape.tumblr.com/post/73530628)。

新网站仍将允许用户听流媒体音乐，创建和分享混音带，但只能使用乐队明确上传到平台的曲目。Oullette 将新的 Muxtape 描述为“乐队推广音乐和听众创作混音的极简平台。”预览版精选了 12 个独立乐队的歌曲，并计划在未来几周内让其他乐队注册并免费上传歌曲。

根据网站上的一个常见问题，最初的 Muxtape 由于无法获得标签的适当许可而倒闭。因此，新网站的目标是绕过直接与唱片公司打交道的需要，成为一个易于使用的平台，让艺术家自己与粉丝分享他们的音乐，也让粉丝彼此分享。该网站称，“Muxtape 力求简单，并相信音乐需要一个干净、无争议的环境”，并承诺永远不会强迫艺术家在他们的页面上显示广告。

Muxtape 将面临一场与 MySpace、脸书和 YouTube 等知名音乐推广网站的艰苦斗争，而且可能永远不会成为主流音乐的发源地。不过，它可能会吸引独立艺术家，因为他们需要一个方便、廉价的地方来推广和分享他们的音乐，同时建立粉丝群。

为此，Muxtape 计划在未来扩大对艺术家的服务，包括他们出售下载、音乐会门票以及创作和销售商品的方式。Muxtape 还计划开发一个开发者 API 和一个分析包，让音乐人可以跟踪他们的音乐在平台上的收听和共享情况。Muxtape 可能会在这些扩张计划中找到自己的商业模式。
