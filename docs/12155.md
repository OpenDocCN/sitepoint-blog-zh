# 分包基础

> 原文:[https://www.sitepoint.com/subcontracting-basics/](https://www.sitepoint.com/subcontracting-basics/)

对于已经独自工作了一段时间的人来说，外包工作的决定可能是一个巨大的里程碑。这通常不是一个轻易做出的决定。这可能是你从自由职业者转型为企业主的时候，也可能是你根据需要偶尔做的事情。无论哪种方式，转包都会带来很多需要考虑的问题。

**外包的决定**

通常，在你迈出这一步之前，你会隐约感觉到外包可能在你的未来。以下是你决定外包的三个一般原因:

1.你有比你自己能做的更多的工作。

如果你夜以继日地工作，仍然发现很难赶上进度，这可能是你需要削减开支或外包的信号。事实是，在你面临[耗尽](https://www.sitepoint.com/13-ways-to-avoid-burnout/)之前，你只能保持这种工作量一段时间。外包可以让你从所代理的工作中获得收入，让现有客户留在你的名册上，并确保他们得到他们习惯的服务，同时腾出你的一些时间。

2.你不具备做特定工作的技能，但又不想放弃这项工作。

如果一个现有的客户或者一个非常理想的客户想让你接手一个你没有经验去处理的项目，这种情况尤其明显。分包让你置身于圈子里，同时确保项目由知道如何做好的人来完成。

3.你想拓展你的业务，做一些新的事情，但要保持现有的工作。

随着你业务的发展，你可能会考虑开始一项新业务，创建一个分支或专注于一个非常具体的利基。如果新企业与你目前提供的服务不一致，那么外包可以让你维持工作和客户，而不会阻碍你的新努力。

**为什么你不应该外包**

外包并不适合所有人。外包可能不适合您的一些原因包括:

*   你不想监督别人的工作。
*   你喜欢把一切都藏在家里。
*   你提供的服务很讲究，不容易外包。
*   让分包商跟上进度所花费的时间最终是不值得的。
*   你的客户和你所做的工作都有保密问题。
*   你的价格还没有高到可以转包出去还能赚钱。

**如何寻找分包商**

寻找分包商的最佳途径是搜索你自己的专业同事和熟人。在任何工作关系中，与你认识和信任的人合作会更容易、更快、更有成效。如果你个人不认识某人，请同事推荐。

另一个值得关注的地方是你参加的论坛和讨论组。假设他们与你需要外包的工作有关，你可以通过观察他们在论坛上的互动，了解他们的背景、专业知识和一些性格特征。

专业组织、网络团体和自由职业者招聘网站也是寻找分包商的好地方。然而，不管你在哪里找到分包商，你都要确保你验证了他们的技能，查看了过去的工作，询问了推荐人，并且花了必要的时间来确保你们可以一起工作。
