# 根——一个 HTML5 WordPress 主题框架

> 原文：<https://www.sitepoint.com/roots-a-html5-wordpress-theme-framework/>

几周前，我们发布了*构建你自己邪恶的 WordPress 主题*、的摘录，其中我们谈到了一些用于快速主题开发的主题框架。自从这本书最初出版以来，我们已经看到更多的主题框架出现，每一个都有自己的优点和特性。我最喜欢的一个新发现是**根源**，一个由丹佛开发者 [Ben Word](http://benword.com/) 创作的首发主题。无论你是头发斑白的老 WordPress 老手还是新面孔的主题新人，我打赌会有一两个功能会让你笑逐颜开。让我们看看里面有什么。

根的呃，*根*是什么？它基于 [HTML5 样板项目](http://html5boilerplate.com/)、埃利奥特·杰伊·斯托克的 [Starkers 主题](http://starkerstheme.com/)，以及你选择的[蓝图](http://blueprintcss.org)或 [960.gs](http://960.gs/) 风格框架的组合——此外还有少量开发人员自己的秘方。如果你不熟悉这些项目，这里有一个快速分类:

*   **HTML5 样板文件**高兴地宣称它是“HTML5 棒极了的坚如磐石的缺省值”——或者说，基本上是一个充满缺省值的模板，用于 HTML 5 开发的最佳实践。这里的每一个钟声和哨声都响起来了，但是仍然有支持旧浏览器的空间。
*   Starkers 提供了一个强大、简单的主题来作为你自己开发的基础——所有的功能都没有任何表象上的琐碎。一段时间以来，它一直是主题的最爱。
*   **960** 和**蓝图** CSS 框架提供了一个强大的基于网格的布局系统，即使在最顽固的老浏览器中也能工作，让你创建任何东西，从最普通的一堆列到漂亮的杂志布局。当你用根的时候，就挑一个你最喜欢的。[在以前的文章中，我们已经了解了这些以及其他框架的好处。](https://www.sitepoint.com/article/applying-lessons-from-css-frameworks)

以下是 Roots 对 WordPress 默认输出的一些改进，以及它对主题开发者的一些帮助。

*   激活主题会为您创建一些示例内容和页面，因此，最好在一个全新的网站上开始使用它。
*   Posts 的标记包括对 hNews 微格式的支持。
*   搜索结果的 URL 被重写，看起来不那么罗嗦，这可能对 SEO 有一些好处。
*   文件夹中项目的 URL 被重写，出现在网站的根目录，掩盖了任何特定于 WordPress 的 URL cruft。
*   主题的`robots.txt`文件已经根据 WordPress 团队的 SEO 建议设置好了，这意味着不允许访问`themes`、`plugins`和`includes`文件夹。
*   内置的图库功能将使用 HTML5 的`figure`和`figcaption`元素。
*   有一个小的管理面板，允许你对 Twitter 和 Google Analytics 进行一些基本的设置，以及进行一些基本的布局选择。当然，因为这个主题是为开发者设计的，所以它很难像一个亲博主的付费主题那样全面，但是对于你自己的目的来说它是一个很好的基础。

在我们发布的摘录中，我们谈到了如何决定什么样的框架最适合你。在我看来，如果你正在寻找一个框架，那么 Roots 就是一个很好的例子。它是当今最佳开发建议的最新版本，它基于许多开发人员已经熟悉的经过试验和测试的项目，并且它的设计考虑了速度和易用性——非常适合我们这些喜欢按时离开办公室，同时仍然做得很好的人。文档的收集越来越多，浏览一下项目的 GitHub 库就能发现代码和手册的频繁更新。对一些主题框架的批评有时可能是用它制作的网站有那种千篇一律的感觉，但 Roots 的图库展示了各种各样的布局和外观——只需发挥你的创造力，开始动手做吧！

如果你正在寻找一个超现代的 WordPress 主题框架，学习 WordPress 开发，或者如果你准备开始在你的 WordPress 项目中使用 HTML 5，考虑在你的列表中添加根。它是我下一个项目的有力竞争者！
