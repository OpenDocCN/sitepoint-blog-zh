## 本周流行语——火狐操作系统

![Firefox OS logo](img/a0acf66a0134e2f60d62aaca11450914.png)

Firefox OS(也叫 Boot to Gecko)是一款面向智能手机和平板电脑的开源操作系统。它由 Mozilla 创建，使用 HTML5、JavaScript 和其他开放网络技术直接与设备硬件通信。该平台针对入门级智能手机进行了优化，但也将用于其他设备，如可穿戴设备和智能电视。

相对而言，火狐操作系统是智能手机市场的新人。SitePoint 的作者 Elio Qoshi 回顾了 2014 年的一些里程碑事件，然后问[未来一年会发生什么](https://www.sitepoint.com/state-firefox-os-whats-store-2015/)。

最近，JavaScript Jabber 小组成员与 [Jason Weathersby](https://twitter.com/JasonWeathersby) (火狐操作系统的传播者)进行了交谈，涉及的话题从 [Gonk](http://en.wikipedia.org/wiki/Gonk) 、 [Gecko](https://developer.mozilla.org/en-US/docs/Mozilla/Gecko) 和[Gaia](https://developer.mozilla.org/en-US/Firefox_OS/Platform/Gaia)(《FxOS 的三个等级】)到[如何在你的树莓 Pi](https://www.youtube.com/watch?v=9cODs3ft8Xc) 上安装火狐。

火狐操作系统被一些人视为苹果和谷歌在智能手机市场主导地位的大胆挑战者。在这篇文章中，一位评论家分享了他对 s Firefox OS Fx0 手机的第一印象，并得出结论[它对 Android](http://www.androidauthority.com/lg-fx0-impressions-577569/) 没有威胁(至少暂时没有)。

火狐操作系统模拟器让你可以在浏览器中舒适地体验 Mozilla 的新操作系统。对开发者和好奇者来说都是好消息。

最后，这是一个有趣的视角，展示了火狐操作系统如何为盲人用户改变生活。
