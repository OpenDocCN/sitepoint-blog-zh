# 10 个用于 HTML 表格的 jQuery 插件

> 原文:[https://www.sitepoint.com/10-jquery-plugins-html-tables/](https://www.sitepoint.com/10-jquery-plugins-html-tables/)

你们中的一些人可能很久以前就碰到过这些桌子，但是对于那些还没有碰到过的人来说，这里有一个列表给你。大多数人可能已经放弃使用表格来实现网页布局，因为表格与基于 CSS 定位的无表格布局相比有几个优点。但是桌子也不是完全没用！通过添加一点 jQuery，您将能够创建非常有用的 HTML 表格解决方案，您的用户将会发现它使用起来简单而直观……尽管可能仍然只是将它们用于数据！:P

## 1.PicNet JQuery 表过滤器

这个插件在表格的一部分增加了一行，并增加了过滤器，允许表格数据的实时过滤。
![PicNet-JQuery-Table-Filter.jpg](../Images/e869142295c020b583a3762ac49ff764.png)
[来源](http://www.picnet.com.au/resources/tablefilter/demo.htm)

## 2.TinySort

一个插件，用于根据(子)内容或属性对子节点进行排序。它对于无序列表或表格的排序非常有用，但也适用于任何类型的节点。
![TinySort.jpg](../Images/b26e1f793764cc67737b98f730878137.png)

## 3.使用 jQuery 和 flot 绘制 HTML 表格中的数据

从 html 表格中绘制图表。graphTable()最多接受两个对象作为参数:第一个是带有 graphTable 参数的对象；第二个是一个带有参数的对象，传递给 flot。
![Graph-data-from-an-HTML-table.jpg](../Images/c2a096a6a63dcae2f61184e12dd6559b.png)
[来源](http://www.rebeccamurphey.com/jquery/graphTable/table.html)

## 4.jLINQ

如果你用过。NET 框架，您可能也使用过 LINQ。LINQ 的特性允许你使用内存中的集合来查询、排序和管理数据，就像使用 SQL 命令一样。LINQ 还允许您直接连接到数据库来管理 IDE 中的所有记录。
![jLINQ.jpg](../Images/ae8d0a418168f9f3246ec1895a8614ea.png)

## 5.浮动标题插件

一个 jQuery 插件，如果原始表头由于滚动而不可见，它会使表头浮动。插件会自动选择 head 标签作为表格的标题。
![floating-header.jpg](../Images/597c20566e05ddca468014deee196e38.png)
[来源](http://slackers.se/2009/jquery-floating-header-plugin/)

## 6.圆柱盘旋

将鼠标悬停在表格中的整列上时，可以高亮显示它们。
![columnHover.jpg](../Images/b60cfdf4d5bf9a0401cc96080a45718b.png)

## 7.可滚动的 HTML 表格

jQuery 的一个插件，能够将一个表格转换成可滚动的。
![Scrollable-HTML-Table.jpg](../Images/8fe223e298e641cb7623b2b167a3516c.png)
[来源](http://www.webtoolkit.info/demo/scrollable-html-table-plugin-for-jquery)

## 8\. HeatColor

我们以前发布过 heatcolor，但是它太棒了。一个插件，允许你根据从元素中得到的值给元素分配颜色。将导出值与自动确定或传入的一系列值进行比较，并根据导出值在该范围内的位置为元素指定“热”色。
![HeatColor.jpg](../Images/b61ca2ef154774931f0d7c9d170925c8.png)
[来源](http://www.jnathanson.com/blog/client/jquery/heatcolor/index.cfm)

## 9.固定标题表格

采用有效的表格 HTML，构建一个固定的页眉和带有可选页脚的滚动体。
![Fixed-Header-Tables.jpg](../Images/52ce2d84b9dcefb6a823060a6520fcbd.png)
[来源](http://fixedheadertable.com/)

## 10.NReco jSquared

一个 jQuery 插件，支持动态添加/删除 HTML 表格的列和行。它的功能非常类似于 Google Squared UI，对于构建高级搜索界面非常有用。
![NReco-jSquared.jpg](../Images/59b995a3bd9a8afa1b0d98018fefe4ee.png)
[来源](http://code.google.com/p/nreco/wiki/JSquared)
