# 庆祝自我推销月的 7 种不令人讨厌的方式

> 原文：<https://www.sitepoint.com/how-to-promote-yourself/>

十月是自我推销月，所以是时候炫耀一下了！来吧，开始大声宣传你的成功，积极推销你的服务，按响你的喇叭。不停地谈论你自己，在你表妹的婚礼上拿起麦克风来宣传你的网站，找一个邻居告诉她你最近获奖的消息。不要接受“不”的回答——这是你关注自己的一个月！

好吧，现实检查。为了庆祝自我提升月而做这些事情可能不是一个好主意，尤其是当隔壁的小老太太试图拿她的邮件时去追她。

但是你当然可以(也应该)庆祝自我推销月，定期推销自己……当然是以更好的方式。事实上，提升自己的最好方法之一就是把注意力从自己身上移开，专注于给予他人——我喜欢称之为“给予自我提升”。

这里有七个帮助你通过服务他人来提升自己的想法……而且在这个过程中，它们不会破坏任何你珍视的关系。

1.  专注于写博客文章，为你的读者提供好的、有用的信息。一个好的经验法则是，对于你写的每一篇推广你正在做的事情的博文，你应该至少写三篇以帮助他人为主要目标的博文。
2.  在行业活动中发言，分享你的知识，与同事交流，努力让自己成为专家。一定要在会议结束后留下来回答问题并提供个人意见。
3.  为一个你相信的地方或网络组织贡献你的时间，或者参加一个让你兴奋的慈善活动。
4.  指导一个刚刚涉足这个行业却没有任何回报的后起之秀。这可以是非正式的或有组织的安排，但重点应该是帮助学员推进她的业务。
5.  举行一次免费的电话会议，回答你所在行业的常见问题，或者提供一份可以从你的网站上下载的免费报告。
6.  通过写关于他的博客、转发他的声明、向别人推荐他以及以其他方式支持他来提升别人。
7.  为你所在行业的其他人创造一种联系和协作的方式。它可以是简单的电子邮件讨论组，或者更高级的在线论坛或免费会员网站。为行业专业人士组织一个本地推特也是一个很好的选择。

这些活动中的每一项都着眼于对他人的益处，而不是推销自己。而且，如果你是带着高尚的动机，而不仅仅是为了促销，这些活动都有可能获得额外的宣传，而不需要你在山顶大声呼喊你的名字。

当然，你应该毫不犹豫地自己推广这些活动—[撰写并发出新闻稿](https://www.sitepoint.com/press-releases-to-promote-your-business/)是帮助传播信息的好方法——但是如果做得好，这种“自我推广”可以不言自明，比你自己更有效地推广你。

这个月你打算做什么来提升自己？有没有办法把焦点放在对别人的好处上？
