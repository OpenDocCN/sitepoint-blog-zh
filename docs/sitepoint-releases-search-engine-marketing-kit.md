# SitePoint 发布搜索引擎营销套件

> 原文：<https://www.sitepoint.com/sitepoint-releases-search-engine-marketing-kit/>

去年春天，SitePoint 联系了我，询问我是否有可能写一本关于 SEO 的新书，类似于他们现有的《网页设计业务工具包》。我提交了一份计划书，非常高兴被 SitePoint 选为作者。

随后的讨论导致这本书的范围扩大到点击付费，并超越了“商业工具包”，创造了任何人都可以用来学习更多关于搜索引擎营销的东西。结果是代表 6 个多月工作的 [SitePoint 搜索引擎营销套件](https://www.sitepoint.com/books/sem1/)。

多年来，我一直是 SitePoint 的忠实粉丝。当我一直在做网页开发的时候，SitePoint.com 实际上是我的浏览器的主页。自然，我很高兴能与 SitePoint 团队一起工作。西蒙、乔治娜、马特、凯文和其他人帮了大忙。

无论如何，我确实相信 SEM 世界早就需要这样的东西，一个对艺术状态的全面回顾。它会成熟吗？时间会证明一切，但为了安全起见，我们已经开始讨论进行年度更新。

您可以查看 SitePoint 上今天的头条文章第一章中的一个示例，[搜索引擎营销工具包-第 1 章](https://www.sitepoint.com/blog/)。作为 SitePoint 的忠实粉丝，我非常高兴在那个地方看到我的文章……屏幕截图当然已经被拍摄下来并保存下来留给后代。
