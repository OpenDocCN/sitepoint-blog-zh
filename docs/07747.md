## 1.jQuery 的文件样式插件

浏览器不允许你设置文件输入的样式。文件风格插件修复了这个问题。它使您能够使用图像作为浏览按钮。您也可以使用 css 将 filename 字段的样式设置为普通的 textfield。
[![File Style Plugin](../Images/d3278c2019fe3b3b363a9d0c3e45983c.png)](http://www.appelsiini.net/projects/filestyle)
