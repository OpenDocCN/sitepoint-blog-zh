# 如果我有一美元买每一个“边框:1px 红色虚线”

> 原文：<https://www.sitepoint.com/if-i-had-a-dollar-for-every-border1px-dashed-red/>

虽然这不是严格意义上的网页设计，但我认为值得一提。

虽然我们 web 开发人员通常是挑剔的生物，每个人都有自己的工作方法、习惯和迷信，但有一些东西将我们所有人联系在一起。你编写第一个 CSS 布局的那天，可能就是你遇到 CSS 编码者的朋友' *border:1px dashed red* '的那天。

无论是内嵌在标记中还是附加在 CSS 中，没有比用花哨的红色边框准确地看到问题发生在哪里更快的方式来阐明一个古怪的 CSS 问题了。

同样，JavaScripter 的长期伙伴也经常被' *alert("变量是:"+variable)；*’。就此而言，你有多少次在一页标记中添加了一段“Lorem ipsum”文本？

现在，过一会儿，你可能会想到创建某种键盘快捷键或宏来更快地添加这段文本。

我的问题是，对于不同类型的作品，我倾向于在不同的编辑之间切换。我发现 [JEdit](http://www.jedit.org/) 很适合繁重的编码工作，Dreamweaver 对于大画面布局工作更快，而 [EmEditor](http://www.emeditor.com/) 和[PSP](http://www.pspad.com/)加载速度很快，可以快速编辑和修复错误。每一个都有自己的可编程快捷键，但在每个应用程序中不可能总是相同的。

我想要的是能在任何应用程序中同样工作的东西——只是我花了一段时间才发现我已经在运行它了。

几年前，我安装了迈克·林的剪贴软件。96kb，Clipomatic 是一个免费的，小巧但优雅的剪贴板缓存管理器，设计用于简单地跟踪你复制的最后 10 个文本项。如果你曾经把一些重要的东西剪切到剪贴板上，然后无意识地在上面复制了一些其他的东西，你就会明白为什么这很方便。它确实在很多场合下救了我。

直到最近我才明白 Clipomatic 还可以拼接预设的文本片段。只是不太清楚该怎么做。

如果您认为这可能对您有用，只需安装它，然后:

1.  右键单击系统托盘中的剪辑图标
2.  选择“剪辑集编辑器”
3.  默认情况下，编辑器显示‘default . clipset’——这是您安装后复制的内容(在这个阶段可能什么也没有)。浏览并选择“永久.剪辑集”。
4.  在文本区点击右键，选择“新建”(我一开始想不出来的地方)，然后将你的代码片段输入或粘贴到弹出窗口中。如有必要，按回车键并重复
5.  关闭编辑器

要访问任何可粘贴区域(编辑器、webforms、MSWord 等)中的代码片段，只需按 CTRL+ **ALT** +V，然后点击向上/向下箭头键选择文本字符串并输入以粘贴它。几天后，我记下了几乎瞬间模糊的击键。
