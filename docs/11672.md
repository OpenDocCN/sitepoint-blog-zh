# 你应该做一个你不相信的项目吗？

> 原文：<https://www.sitepoint.com/work-on-dumb-projects/>

自由职业的主要好处之一是自由和 T2。你可以选择做自己感兴趣的项目。同样，你可以拒绝接受乏味的工作或注定失败的项目。

受薪员工很少有这种奢侈。他们也许能够表达自己的恐惧，但除非他们能说服老板，否则工作还是会向他们走来。

假设你得到了一份不违法或不道德的工作，面对一个愚蠢的想法，你应该怎么做？是用户不会要的服务吗？企业是否正在进入一个已经拥挤的市场？是否存在严重的技术问题？你应该如何处理一个激动的客户，他的想法根本行不通？

在当前的经济环境下，接受任何工作机会都很诱人。然而，你在你所有的项目中都有既得利益。生意成功的客户会再次回来推荐你的服务。充其量，一个失败的企业不会提供额外的工作。最坏的情况是，客户可能会因为他们的不幸而责怪你。你也可能承担一项导致数月痛苦努力的任务，结果却错过了更好、更赚钱的机会。

我的第一个建议是沟通。客户为你的服务付费，他们通常会倾听你的担忧。问一些引导性的问题，避免听起来消极，确保客户从各个角度考虑了他们的项目。

记住要有开放的心态，不要过于轻视。成功并不总是用金钱来衡量，一些最不可能的想法也可能成功。谁能想到一个 140 个字符的网络消息广播系统会吸引数百万用户？如果一个主页显示一个 1000×1000 的图形，每个像素卖一美元呢？

商业繁荣永远无法保证。然而，如果项目让你感兴趣，并且你和客户有良好的工作关系，你会更有动力去完成任务。

你接受每个项目吗？你拒绝的比你接受的多吗？你如何判断这项工作是否值得？
