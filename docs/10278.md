# SitePoint 社区迎来新的一年

> 原文:[https://www . site point . com/a-new-year-for-the-site point-community/](https://www.sitepoint.com/a-new-year-for-the-sitepoint-community/)

当我今天早上签到的时候，注意到办公室的大部分人似乎都登录了我们的即时消息系统，我看了一下日历，意识到今天是星期一。博客日。我也意识到我已经完全错过了过去的两个星期一，但这并不是说社区没有向前迈进-只是我一直在偷懒。我喜欢假期——完全忘记时间。我出门的时候不戴手表——部分原因是时间不重要，但主要是因为我讨厌晒痕。所以这篇文章来自美丽的北国——新西兰北岛的顶端，我在那里的海滩度过了我生命中的每个夏天。

很快有一天我会因为这种炫耀而被解雇。

总之，2011 年和 SitePoint 社区。去年对我来说是找回自我的一年。这是我担任社区经理的第一年，我用它来摸索，并试图找出什么是有效的，什么是无效的。我致力于建立我们在脸书的形象，利用我们的 Twitter 粉丝作为社区工具和营销工具。今年我想推动社区向前发展。我们有一些令人兴奋的想法。我相信变化，没有巨大的风险，你就不会有伟大的成就。一切都是为了与时俱进。

![SitePoint Community](../Images/9bb2be66119f2936eef6bd9dee6364c7.png "SitePoint Community")

但是说够了。让我告诉你现在社区里发生了什么。

我们伟大的假日摄影比赛即将结束，有一些令人惊叹的作品。你还有时间拍一些照片，看看在你生活的地方，节日对你意味着什么。有一些很棒的奖品等着你去拿，赶快抢购吧。你可以在这里阅读参赛指南或查看当前的参赛作品。

随着我们社区奖项的结束和[结果的公布](https://www.sitepoint.com/forums/showthread.php?t=719967)，我们将进入新一年的新比赛。工作人员正在敲定细节，我们随时准备宣布。如果 HTML 和 CSS 是你的东西，这一个是给你的。

我们的[脸书社区](http://www.facebook.com/sitepoint)继续繁荣发展，今天的不祥之兆是关于古怪的圣诞礼物。Kindles 似乎是当下的礼物。如果你有什么有趣的东西，去那里炫耀一下吧。

我想这就是我今天要说的。下周我将回到船上，带着更多有趣的和工作相关的信息。
