# 案例研究——社区门户促进网页设计销售

> 原文:[https://www.sitepoint.com/boosts-web-design-sales/](https://www.sitepoint.com/boosts-web-design-sales/)

 **Doug Isom 住在阿肯色的尤里卡泉，一个只有 2000 居民的旅游小镇。

尽管道格的网页设计公司，DigitalCreative，自 1995 年成立以来已经建立了一个良好的客户名单，道格还是决定通过建立一个当地社区门户来扩大业务——这个想法是由论坛报网站的一篇文章引起的。

“我知道我想做一些不同于你所期望的典型网站的东西来促进旅游业。成为 SitePoint 的狂热粉丝后，我得到了我一直在寻找的灵感和想法:采访、餐饮评论、关于小镇的文章和活动列表，”道格说。

##### 规划和构建门户

这个网站花了大约 3 周的时间来建立。道格估计，“如果这是一份雇佣工作，成本会在 3500 到 4000 美元之间”。

从一开始，道格就意识到，在门户网站的生命周期中，DigitalCreative 可能无法在网站上投入太多时间。因此，该公司使用了几种方法来加快门户的开发和长期维护，包括为重复的站点元素(导航、页面标题等)提供 SSI。)，大量使用 CSS 进行布局和设计，以及数据库驱动的目录列表，以使网站更新尽可能快速且不耗费人力。

目前，DigitalCreative 每月花 10-20 个小时更新网站，尽管他们预计在淡季工作量减少时，每月会增加到 40 个小时。

##### 发布和推广

网站建好之后，推广就开始了。“我们对搜索引擎推广做了大量的研究，阅读了我们在 SitePoint.com 能找到的关于这个主题的尽可能多的文章。然后，我们根据我们所了解的情况对我们的副本进行了微调。我们还在当地报纸上刊登了广告，在其他网站上购买了横幅广告，并在网络上建立了互惠链接协议。”

该网站于 1 月份推出，到 7 月份，每月有近 10，000 名独立访客——与 DigitalCreative 也维护的当地商会网站一样多。道格说，这个旅游门户网站现在产生了“惊人的页面浏览量和粘性时间”。

这个门户网站也是 DigitalCreative 建立的其他本地商业网站的一个很好的营销工具。所有客户都免费获得位于相应目录类别顶部的文本和图形列表。“我们已经跃升为客户网站的前五名推荐人。这不仅提高了顾客的满意度，也培养了许多新顾客。"

##### 底线

在门户网站推出后的八个月里，它至少部分负责了八个新客户。“我们肯定是把这个网站作为我们获取新客户的销售策略的一部分，”道格说。“我们将它作为我们投资组合的一部分，向潜在客户展示我们的能力和工作质量。”

“太感谢了！SitePoint 为我们的业务增加了一个新的方面，并增加了我们的收入，这是一个重大的变化。”**
