# 查看徽标:像素徽标

> 原文:[https://www.sitepoint.com/looking-at-logos-pixel-logos/](https://www.sitepoint.com/looking-at-logos-pixel-logos/)

网络上的图标让我们领略了一些著名的图像和符号缩小成小方块后的样子。像素是数字图像的最小部分。像素本质上是位于图像特定部分的一小块颜色。作为数字图像的重要组成部分，设计师在他们的标志设计中使用看起来像巨型像素的东西可能并不奇怪。

就其本质而言，使用像素创建的符号往往看起来非常基本，而且往往非常抽象。这些符号是彩色的，但通常围绕一种单一的颜色有一系列的色调。它们传达了元素聚集在一起创造一个可识别的图标的想法。像素风格的设计需要技巧来有效地完成，否则它们会给人一种(在电脑上)画得很糟糕的印象。

现在，为了给你带来视觉享受和灵感，我们收集了一些以像素为主要设计元素的徽标。(其中一些企业的名称中有 Pixel 或 Pix，这可能不是巧合。)
