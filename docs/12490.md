# 我有一个(140 字)的故事给你！

> 原文：<https://www.sitepoint.com/have-i-got-a-140-character-story-for-you/>

*以下内容转载自论坛报网站第 423 期*

我刚刚开始使用微博服务 Twitter T1 来帮助建立我的在线品牌。到目前为止，我有 32 个追随者。其中之一是澳大利亚总理 T2。

嗯嗯。现在我有点紧张了。

我即将撰写我的企业在线时事通讯，总理在 Twitter 上关注我的事实将会赢得一次提及。我计划包括这个故事，因为它将做几件事:

*   抓住读者的兴趣，然后
*   对我的品牌产生积极影响(读者会认为这非常令人印象深刻)。

我在生活中经常使用故事。当我和一个客户一起为一个网站开发工作做宣传时，我会用故事来编织我们的专业知识、经验和能力来完成这个项目。

例如:“我们以前遇到过这个问题；我们是通过做 *A* 、 *B* 和 *C* 来解决这个问题的。结果，客户的网站销量激增。客户很高兴，他真的在那个周末出去给自己买了一辆新跑车！”

故事之所以有用，是因为人们能与它们产生共鸣。巴拉克·奥巴马在他的演讲中一直这样做。我妈妈隔着后栅栏和邻居说话的时候也会这么做。

聪明的商务人士用故事来联系潜在客户，同时展示他们的专业知识和对手头任务的适应性。
