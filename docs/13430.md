# 黄蜂街队

> 原文:[https://www.sitepoint.com/wasp-street-team/](https://www.sitepoint.com/wasp-street-team/)

SitePoint 的作者和 WaSP 的长期成员 Rachel Andrew 写道:

> 今天推出的[黄蜂街团队](http://streetteam.webstandards.org/)；在开始我们的第一个任务之前，我们试图引起一些兴趣。

我错过了在 SXSW 的 WaSP 年会上的公告，但是在注册页面上有更多关于这个计划的信息:

> 黄蜂街队是关于你的。不，不是所有其他人都在读这篇文章，而是你，穿着你真实的皮肤。我们的想法是一起创建一些任务——如果你愿意，可以称之为挑战——来帮助在你的当地社区推广网络标准。这些事情将有助于把信息传递给在你身边生活和工作的企业、教育机构、网上商店和个人。作为一个中心团队，我们很难接触到这些人，但作为一个分布式团队，这很容易。

这听起来像是得到了 Web 标准组织现有活动的支持(每个人都已经把这两个组织搞混了，为什么不联合起来打一场漂亮的仗呢？).如果你想参与，请前往[注册页面](http://streetteam.webstandards.org/)，你会得到及时通知。
