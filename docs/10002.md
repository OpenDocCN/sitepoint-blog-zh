# 你没有问的七个问题会让你失去销售机会

> 原文：<https://www.sitepoint.com/seven-questions-you-arent-asking-that-will-cost-you-the-sale/>

他们说知识就是力量。达成约定的关键要素之一是提问。但是你问对人了吗？这里有七个你可能没有想到的。

### 你为什么要见我？

无论是你对客户进行了展望，还是他找到了你，重要的是要发现*为什么*潜在客户有兴趣与你会面。在这里你将开始了解他的需求和需要，以及*他需要减轻哪些痛苦*(如果有的话)。

### 你还在考虑谁？

这是一个很有说服力的问题，因为如果你的潜在客户不愿意透露，你可能已经输了。他可能在购买商品，而你只是另一个潜在的供应商。或者你可能是一个被抛弃的出价者，仅仅因为他们需要得到不止一个。如果是这样，这是你想知道的事情，而且越早越好。俗话说:“如果你要输，那就早点输。”

当然，最好的答案是，“没有人。”潜在客户会告诉你吗？是的，他们会的——只要你开口。

### 你为什么考虑我？

这是一个很好的问题，因为你会发现你的潜在客户看重你什么。

我对潜在客户对此的反应感到惊讶。如果你问这个问题问得足够多，你就会开始意识到为什么客户会雇佣你。不止一次有人告诉我，这是因为我“脚踏实地”，容易交谈。客户不会害怕讨论他们不完全理解的技术，他们相信我会把它分解成他们能理解的东西。在一个任何拥有电脑和盗版软件的人都可以自称是网页设计师的市场，拥有一个让你与众不同的个人品牌是非常宝贵的。一个直接来自你客户口中的…无价之宝。

### 你为什么不在内部做？

这将让您对他们的内部能力和局限性有所了解。你会想知道他们是否考虑过让接待员(他有“设计眼光”并且“熟悉 Dreamweaver”)尝试一下。

我们曾经失去了一个大型教堂的项目，因为重新开发网站意味着“解雇”维护网站多年的志愿者…他们不想伤害她的感情。请记住，你也可能在和什么都不做的决定竞争。

### 像这样做决定的时候，一般喜欢和谁商量？

隐藏的决策者和影响者可能会扼杀一笔交易。通常，这些人只看底线成本，而不考虑任何其他因素。如果你在最初的咨询中没有让他们参与进来，他们将对你提供的价值一无所知，并且会让你的潜在客户相信雇佣你太贵了或者是个坏主意。发现隐藏决策者后的下一个问题是，“在准备提案之前，我需要与他们会面。你愿意那样做吗？”

### 如果这个项目成功了，对你个人来说意味着什么？

我们有时会忘记，尽管你的客户可能是一家公司，但你面对的是一个人。假设您的潜在客户告诉您，项目的成功结果将是收入增加 10%。如果你的潜在客户是企业主，额外的收入可能意味着扩大业务，送孩子上大学，提前退休，或者雇用更多的员工，以便有更多的时间陪伴家人。对总经理来说，这可能意味着销售佣金，在老板眼里看起来不错，甚至是升职。

金克拉说，通过帮助别人得到他们想要的东西，你可以得到生活中想要的任何东西。你的客户越是认为你的服务帮助他实现了他的希望、梦想和愿望，他就越有可能雇佣你。

### 如果我能以可接受的价格满足你的所有要求，你会怎么做？

毫无疑问，这是最难问的问题，但也是最有益的。如果你有勇气问，你会知道你的潜在客户的意图。他可能会说，“我们只是收到一些投标，”或者“董事会必须批准。”(这意味着你没有问前面的一些问题。)但他也可能说:“我给你开张支票。”

第一次听到这句话时，我非常兴奋。

现在你知道了…七个你可能没有问你的客户，但是应该问的问题。忽视询问，你会在无意购买的潜在客户身上浪费时间。否则你会失去这笔交易，并且不知道为什么会失败。

如果你已经在问这些问题，它是如何帮助你获得更多客户的？如果你不是，试一试，让我知道会发生什么。
