**为什么要学习 SQL？**

SQL 是一种简单的高级语言，功能强大。您可以使用几行 SQL 来执行任务，而这些任务在编程语言中需要一页又一页复杂的代码。它是今天所有主要数据库系统使用的语言，尽管它已经存在了大约 30 年，但由于数据库驱动的网站的巨大成功，它在 21 世纪迎来了复兴。

无论您的网站是用 PHP、ASP、Perl、ColdFusion 还是任何其他编程语言编写的，也无论您想使用哪种数据库系统——MySQL、PostgreSQL、SQL Server、DB2、Oracle 或任何其他系统——有一个事实几乎是肯定的:如果您想拥有数据库驱动的内容，您将需要使用 SQL。
