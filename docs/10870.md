# 用一个不要做的清单来管理你的分心

> 原文:[https://www.sitepoint.com/using-a-not-to-do-list/](https://www.sitepoint.com/using-a-not-to-do-list/)

我们经常谈论待办事项清单，以及它们如何让我们走上正轨，但是当我们努力提高待办事项的效率时(也就是我们当中的清单制定者！)，有这些丑陋的小杂念潜伏在后台。这些“反待办事项”的工作就是把我们从手头的任务中抽离出来，消耗我们的生产力。有时他们成功了。

## 分心的事

你的分心可以是任何打断你、阻止你做你知道你需要做的事情、强化你的坏习惯的事情。它们每天都在变化，这取决于你正在做的工作和你工作的环境。

例如，我最大的分心包括新的电子邮件通知、家人打来的电话和社交媒体。其他常见的干扰，有些你可能自己也经历过，包括阅读博客、看电视和即时通讯。

## 列出不要做的事情

处理这些不断分心的事情的一个方法是列一个“不要做”的清单来提醒你在一天中的任何时间点你应该注意哪里。不要做清单是你承认的一天中让你分心的事情的清单，你想阻止这些事情接管和扼杀你的生产力。对于我们这些已经习惯使用列表的人来说，这是一个完美的生产力提升。

你的待办事项清单上的项目可能是重复出现的问题，比如不停地查看电子邮件。或者它们可以是特定的情况，比如担心即将到来的演示。你的“不要做”清单是一个提醒，一个触发你的想法和行动回到优先事项的方法。

## 给你的不做清单的提示

这里有一些建议可以帮助你把你的待办事项列表变成一个有效的生产力工具:

*   把你的清单放在你容易看到的地方
*   当你准备钻研你的工作时，每天早上更新它
*   与你工作环境中的其他人分享你的清单，这样他们可以通过帮助你消除干扰来支持你
*   用你的待办事项清单上的项目作为完成任务的奖励
*   当你发现自己被拉离正轨时，把它加入你的清单

## 有名单吗？

我最近开始使用一个不要做的清单，并发现这是一个很好的方法来强化我的大脑需要去的地方，激励自己保持专注，并帮助我管理我的工作和个人生活中的优先事项。
