# 雇一个销售人员怎么样？

> 原文：<https://www.sitepoint.com/wht-about-hiring-a-salesperson/>

我的许多客户想雇用一名销售人员。他们认为自己不擅长销售，所以为什么不雇人呢？

在我看来，这可能是一个巨大的错误。原因如下:

1.销售人员要花很多精力去管理。

2.销售人员不是能立即为你销售的“灵丹妙药”。你必须为他们提供工具、期望、良好的薪酬结构、良好的解决方案以及营销宣传材料。

3.只有五分之一或十分之一的销售人员是优秀的。其他人即使不比你差，也不比你好。

4.很难找到十分之一的人是伟大的，让他们和你在一起开心就更难了。大多数优秀的销售人员都是有酬职业，收入丰厚。

5.你必须区分产生销售线索的销售人员和管理客户的销售人员。前者(获得线索)最有价值，因为这些人不介意打电话和拜访陌生人。但是你必须小心那些喜欢参加社交活动和“闲聊”的人它们通常毫无价值。

6.你需要知道如何问一些关于管道管理和活动的好问题，以保持销售人员的诚实和负责。很少有技术人员这样做。

在我看来，作为一名专业人士，你最好掌握市场营销和销售，开发一个系统，然后——当你的公司足够大时，开始招聘一名销售人员。换句话说，不要放弃销售和营销职能。

如果你和销售人员有过愉快的经历，请把它贴在这里，包括你做了什么让它为你工作。
