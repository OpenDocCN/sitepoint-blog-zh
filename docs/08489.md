# 电子邮件成功的另一个秘密

> 原文:[https://www.sitepoint.com/another-secret-to-email-success/](https://www.sitepoint.com/another-secret-to-email-success/)

你是否发现自己日复一日地写着同样的回复或者报道着同样的话题？你知道的。向客户索要材料，回复新的业务咨询，向拒不合作的客户催款，可能还要回复求职者的招聘咨询。

嗯，大约五年前，我尝试了一个自邮政服务诞生以来直销商一直在使用的屡试不爽的技巧；我使用模板。

没错——如果你曾经收到过我发来的一封电子邮件，是关于一个我反复提及的主题，那么这封邮件很有可能是根据一个模板生成的。我听到你问，为什么我会使用电子邮件模板？那不是没有人情味，没有效果吗？完全相反！这样做有很多很好的理由。

## 为自己节省宝贵的时间

你写邮件的时间越少，你花在做你真正应该做的事情上的时间就越多，当然，除非你的职位头衔是“发邮件的人”!你会更快地回复询问。因为你花更少的时间反复写同样的材料，你可以更快地回复那些紧急邮件。

## 节省收件人的时间

如果你确保你的邮件确实回答了接下来的问题，这将节省你的收件人通常不得不回复的大量时间。

## 这不是没有人情味的

我并不是建议写一整封邮件，只是简单地修改第一行；即使在只有 60%重复内容的电子邮件中，也有大量的生产力和时间节省。

如何入门？我是这么做的。

我从观察我经常写的电子邮件开始。我平均每天会收到 120 多封电子邮件，其中很多都在期待回复，也有很多都在期待非常相似的回复。所以我开始写一份我经常被问到的话题的清单。

下一步是写邮件的要点，并以此为开端。例如，项目更新电子邮件可能会包含以下内容:

1.  刚刚完成的任务
2.  接下来要完成的任务
3.  我们需要客户提供的材料
4.  任何截止日期/预算问题

然后，我开始写涵盖这些要点的电子邮件，并用我认为收件人需要的所有细节充实它们。随着时间的推移，我观察了这些电子邮件的回复，发现了一些重复出现的问题(例如，对我上面关于图像供应的电子邮件的回复“我应该以什么格式发送这些图像？”)我确保我的邮件也涵盖了这些内容，为我自己和收件人节省了阅读回复的时间。

我把这些模板保存在我邮件程序的一个文件夹里，有一个主题行，一般的主题是什么，所以我有主题为“跟进最近的报价”之类的电子邮件。

然后，我将这些邮件模板中的文本剪切并粘贴到我的回复中，或者在某些情况下粘贴到新邮件中，然后个性化并重写需要修改的部分。这样，这封邮件就真的是为收件人量身定做的，听起来不会“千篇一律”。

试试这个方法——我敢肯定，你会发现各种各样的电子邮件都可以预先写好，每天都可以给自己额外的时间去做更重要的工作。享受这段额外的时间吧！
