# 你好，我能帮你吗？商务座机的替代品

> 原文：<https://www.sitepoint.com/landline-alternatives/>

几年来，许多家庭一直在用手机或其他替代品取代传统的固定电话线路。美国国家健康统计中心(T2)去年的一项研究显示，2007 年下半年，近六分之一的美国家庭(15.8%)只有无线电话。

但是在商务中使用电话呢？如果你的家庭抛弃了你的私人座机，你可能没有单独的专用座机。如果你刚刚开始创业，增加一部座机可能不切实际，也不划算。或者，如果你是一个经验丰富的自由职业者，发现你花在电话上的时间很少，专用的商务座机可能没有多大意义。

那么你有什么选择？这里有几个固定电话的替代品，你可以考虑用在你的生意上，并简要看看它们各自的优缺点。

手机

如上所述，许多职场人士选择使用手机作为他们的工作线。这样做的好处包括减少开支，即使不在工作时也能随时联系，并且不必为业务维护单独的线路。

最大的缺点是接收和掉线和其他常见的手机问题缺乏专业精神。此外，一些好处也可以被认为是缺点——随时可以访问，以及将个人和商业电话混合使用。

**VoIP** 的实现

VoIP 是座机的一种非常流行的替代方式，有许多提供商可供选择。你可以通过电脑使用网络电话，比如 [Skype](http://www.skype.com) 。或者，您可以从您的宽带提供商或其他 VoIP 提供商(如 Vonage)获得 VoIP 服务，其中包括允许您使用普通电话的调制解调器。

VoIP 的好处通常包括每月服务费用降低、全套典型的电话线功能(如来电显示和语音邮件)、传真功能以及旅行时将电话线随身携带的能力。

网络电话最大的缺点之一就是可靠性。如果您失去了互联网连接或有不稳定的连接，您的电话服务将受到影响。您还可能遇到通话质量差的情况，这对业务线来说是一个重大的不利因素。

虚拟电话线路

虚拟电话线路是固定电话业务的另一种替代方式。通过像 [RingCentral](http://www.ringcentral.com) 这样的服务，你可以选择一个免费号码或者一个虚拟号码，你的电话要么被转接到一个私人语音信箱，要么被转接到另一条电话线(比如你的手机)。您可以从您的主电话线路拨打电话，并将其屏蔽，就像您从虚拟线路拨打电话一样。另一个好处是能够使用问候和扩展系统设置语音信箱，这可以增加你的专业性。

其中一个缺点是管理和规则设置，这是让您的线路按您希望的那样运行所必需的。例如，除非您将虚拟线路设置为转接到您的主电话线路，否则您的所有呼叫都将被转接到语音邮件。另一个缺点是，除非你使用它提供的所有功能，否则获取一个商务电话号码是一种昂贵的方式。

还有其他选择，如电话应答服务和专用的商务手机，但无论选择哪种，你都需要考虑自己的需求和每种选择的利弊，以确定最适合你的选择。需要考虑的一些方面包括:

*   通话质量
*   职业化
*   费用
*   安全性
*   应急服务的可及性
*   可靠性
*   便利
