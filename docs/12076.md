# 当你被最后期限打败时该怎么办

> 原文：<https://www.sitepoint.com/handling-deadlines/>

当你的最后期限快到了，你很快就会知道。你可能正在处理多个项目，或者刚刚开始感觉到工作太多的压力。你所有的[生产率技巧](https://www.sitepoint.com/4-ways-to-be-more-productive/)都失败了，你的[项目时间估计](https://www.sitepoint.com/how-to-estimate-time-for-a-project/)似乎也差得很远。你正面临着似乎不可能完成的最后期限，即使你能够连续通宵工作，但你仍然需要完成工作。你能做什么？

**重新评估优先级**

对我来说，偶尔与最后期限斗争带来的许多压力通常与工作无关。除了工作，我们都有自己的生活和责任，有时这些事情和我们的职业责任一样重要。但并不总是如此。检查你的清单，花时间诚实地重新评估什么是真正重要的，什么可以推迟几天。

作为企业主，我们想把所有的事情都做好(有时会马上把所有的事情都做好)，但是只有当我们专注于最重要的优先事项，让其余的事情顺其自然，我们才能有效和成功。用我六年级老师玛丽·克莱尔修女的话来说，“做你能做的，剩下的就做吧。”

**摆脱孩子(和其他干扰)**

当最后期限临近时，没有时间偏离轨道。事实上，你不能屈服于日常的干扰。所以，制定一个计划，让孩子们在这一天离开家，摘下电话，关闭你的电子邮件应用程序，关掉电视或音乐，只专注于工作。

如果你习惯于通过每天的分心来工作，你将真正从哪怕半天安静的工作时间中受益。为了让自己处于更好的位置，请遵循下一个提示。

**制定时间表**

一旦你计划好了一天的安静和不受干扰的工作(或者尽可能多的)，计划好它。创建一个你可以严格遵循的时间表，包括工作和休息时间。有一个时间表不仅能让你前进，还能让你制定一个适合自己的计划。如果你能从一个项目跳到下一个项目，然后再跳回来，那就以 60 分钟为增量来安排你的时间，这样你就可以来回地跳。如果你知道你只需要几个小时就可以完成一个项目，那就计划好这段不受干扰的时间。如果你像我一样是待办事项清单的奴隶，先安排时间做最快的任务。立即划掉事情的心理影响会激励你做更多的事情，减轻你感受到的一些压力。

**要求延期**

这是我的最后一招，我很少这么做，但是如果所有其他方法都失败了，你根本无法成功地满足最后期限，你可以随时要求延期。请求延期时要记住的一件事是尽可能提前通知。如果看起来你会在一两周前错过这个标记，那就提出来。(对你和客户或同事来说)这比等到最后一刻要好得多。
