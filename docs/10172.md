# 素描风格的字体

> 原文:[https://www.sitepoint.com/sketch-style-fonts/](https://www.sitepoint.com/sketch-style-fonts/)

在这里，我整理了一些资源，如果你有兴趣将流行的草图风格应用到你自己的设计中，这些资源可能会对你有用。
