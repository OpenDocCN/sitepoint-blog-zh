# 登陆页面现在影响 AdWords 质量分数

> 原文：<https://www.sitepoint.com/landing-pages-now-influence-adwords-quality-scores/>

众所周知，AdWords 广告客户的排名顺序是由两个因素的[组合决定的，这两个因素组合成一个总质量分数。](http://adwords.blogspot.com/2005/12/new-addition-to-quality-score.html)

然而，谷歌现在宣布，登陆页面质量[将是另一个因素](http://adwords.blogspot.com/2005/12/new-addition-to-quality-score.html):

> 今天，我们开始在质量评分中加入一个新的因素——登陆页面——它将查看从您的广告链接的页面的内容和布局。
> …
> 提供强大且相关内容的广告客户将不会看到什么变化。然而，对于那些提供不太正面的用户体验的人，质量分数可能会降低，从而增加运行关键字所需的最低出价。

看起来，至少有一些广告客户对这一变化不太高兴，因为内容优化和付费广告之间的界限变得模糊，而且需要额外的工作量。

现在的问题是，有了成千上万的广告客户和数百万的唯一登陆网址，谁来决定质量和如何决定？无论是收到投诉时的人工检查，还是查看登录页面的 PR 值，或者只是查看 AdSense 出版商经常过滤的 URL，如果谷歌公开他们对付费广告进行排名的方法，那就太好了。

目前，他们从谷歌得到的唯一公开建议是阅读他们的[网站设计技巧和指南](https://adwords.google.com/select/siteguidelines.html)，并遵守它们。
