# 纽约公共图书馆的新标志

> 原文:[https://www . site point . com/new-logo-for-new-York-public-library/](https://www.sitepoint.com/new-logo-for-new-york-public-library/)

在过去的几天里，关于纽约公共图书馆的新标志，网上有相当多的讨论。![Old Logo](../Images/70051bff669355eb14f0a5c50f8a789d.png "Old Logo") ![New York Public Library Logo](../Images/422b5684cc33a80827f50f7d9e0046ec.png "New York Public Library Logo") *旧的公共图书馆标志(左)和新的。*重新设计始于 2008 年，设计师们使用了图书馆台阶上狮子雕像的照片和草图。楼外的狮子以“刚毅”和“忍耐”著称。据报道，影响包括日本木刻，彩色玻璃和新艺术。矢量风格的作品被设计成足够多才多艺，可用于多种格式和颜色，以及在小尺寸下复制良好。新的标志来了，就像通常标志重新设计的情况一样，给图书馆带来了新的使命。他们想传达一种更加数字友好和现代的感觉。![LogoWith Text](../Images/17954337e44a7c8bf913c9cabfc3e5cc.png "LogoWith Text")作为狮子座的人，同时也是一家名为“狮子”的公司的老板，我不得不承认我对这个特别感兴趣。虽然我还在和自己辩论是否喜欢这个 logo。我喜欢这种简单和清晰的线条，但是对于如此重要的知识守护者来说，这是不是有点太卡通了？虽然我喜欢无衬线字体，但我对文本的位置不太感兴趣。字体叫做 [Kievit。有些人评论说它看起来像米高梅的标志。我认为他们指的是一个非常短暂的米高梅标志，而不是众所周知的经典。蓝色和金色的米高梅标志在 1968 年被用于少数电影和该公司的录音部门，但后来被我们所知道和喜爱的咆哮的狮子所取代。这里有一段来自纽约公共图书馆的有趣视频，概述了这个标志的由来。](https://typophile.com/node/16448)
