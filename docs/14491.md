# 一个关于行动计划的相关帖子和一个不相关但有趣的链接

> 原文：<https://www.sitepoint.com/one-relevant-post-on-action-planning-and-one-irrelevant-but-hilarious-link/>

嗯，我的渠道有点干涸，不难找出原因:在过去的几周里，我放慢了营销活动的速度。我感到羞耻。

因此，昨天和今天，我采取了具体措施，让我的业务再次加速发展:

1.我联系了三个潜在的推荐来源——真正的重量级人物——并继续或开始与他们讨论如何拓展业务。

2.我已经为接下来的几周制定了一个计划来加强我的营销活动。

你拿出你放进去的东西。在我的咨询客户中，大约 60%的人喜欢学习如何以正确的方式进行业务发展，但却从不付诸行动。就好像他们想要的是常识的答案或者是琐碎的追求。这是浪费他们的钱！

当我给他们打电话时，他们承认他们确实需要采取行动，我给他们布置作业，让他们制定行动计划，这是我最后一次听到他们的消息(即使有后续行动，也会导致借口)。是的，我可以在那里握住他们的手，但这很不寻常。许多人想要成功，但不想或不愿或害怕采取成功所需的行动。

那么:你需要提升你的营销活动水平吗？你得到你想要的结果了吗？你投入的足够多了吗？(另一个不同但也很关键的问题是:你是否使用了最有效的技术和策略来获得投资和时间的最大回报？)

总之，至少对于这篇博客来说，你采取行动了吗？如果没有，从今天开始。
