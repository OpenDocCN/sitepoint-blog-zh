# Adobe 宣布推出 Flash 媒体编码服务器

> 原文：<https://www.sitepoint.com/adobe-announces-flash-media-encoding-server/>

![](img/b5ab0f8024a136e226860b410648131c.png "flash-logo") Adobe 今天宣布发布其端到端平台的最新产品，用于在多个屏幕(桌面和移动)上创建和部署 web 上的富媒体内容，即 Flash Media Encoding Server。Adobe 的新产品定价为 6000 美元，是一种企业级编码和代码转换服务器，用于将各种格式的视频转换为 Flash。该软件支持从 Windows Media 格式、DivX、H.264、MPEG-4、Quicktime 和其他格式的视频到 FLV 或 F4V 视频的转换。

Adobe 的副总裁兼动态媒体总经理 Jim Guerard 告诉我，他们并没有试图取代其他编码产品或与之竞争，其中一些产品，如 Silversoft 的 [NCode](http://www.silversoft.com/ncode) 是免费和开源的，而是新产品针对的是那些宁愿从一个供应商那里购买所有 Flash 工作流的企业客户。

Geurard 告诉我，Adobe 的企业和教育客户一直在寻求一个交钥匙编码解决方案，这是对这些请求的回应。事实上，编码服务器产品是他们当前企业闪存产品系列中的一个明显的遗漏。Flash Media Encoding Server 填补了其端到端 Flash 产品生态系统中的一个漏洞。

然而，考虑到成本以及免费和低成本的替代方案，小型 Flash 发行商不太可能选择新的 Flash 媒体编码服务器。Adobe 似乎对此没什么意见。

除了将视频编码和转码为 Flash 之外，媒体编码服务器还可以对视频应用预处理过滤器，如色彩空间转换、标准化音频和锐化视频。

有关 Flash 媒体编码服务器的信息可在[http://www.adobe.com/products/flashmediaencoding/](http://www.adobe.com/products/flashmediaencoding/)获得。
