# 面向小型企业的云计算

> 原文:[https://www . site point . com/cloud-computing-for-small-business/](https://www.sitepoint.com/cloud-computing-for-small-business/)

就在过去几年里，云计算已经从新奇的想法发展成为行业标准，这是有充分理由的。无论您寻求的是安全性还是可扩展性，云正迅速成为托管的首选解决方案。最棒的是**任何人**都可以进行切换；你不必跻身财富 500 强，就能开始享受云的好处。

在选择主机时有无数的选项，但是为了本文的目的，我们将重点关注 Amazon 的 EC2 云，以及他们的一流服务中的相关工具集合。

# 为什么选择云计算

大型和小型企业都享受着在亚马逊的云中托管的相同优势，但对于没有 IT 团队和大量资本投资的较小公司来说，它允许他们进入一个新的托管世界，并开始利用所有优势。

## 可量测性

在过去的托管时代，升级你的服务器是一件痛苦的事情！需要一名专业人员亲自前往您的数据中心(假设您的 web 服务器不在您家的楼梯下)，安装新的硬件或软件，并祈祷一切顺利。这是一件冒险的事情。

幸运的是，事情已经改变了。在你最喜欢的咖啡店里，像亚马逊这样的云主机可以让你把 IT 呆子留在家里，并根据需要升级你的服务器。无论您的站点是保持持续的大规模增长，还是新产品已经将您的服务器推到了极限，向您的服务器分配额外的资源从来没有像现在这样简单。

任何在凌晨 3 点进入数据中心进行服务器维护的人都可以支持这一原则；你不会找到更便宜、更简单的解决方案。

## 可靠的数据

无论你存储的是你弟弟新生婴儿的照片，还是敏感的用户数据，亚马逊都有一种安全可靠的方式来存放你的文件。当你上传任何东西到云端，它会立即在世界各地的数据服务器上被复制。

## 减少停机时间

当云支持你时，可怕的服务器崩溃就不那么可怕了。想象一下在您的物理服务器崩溃后恢复它需要什么。如果它是自我管理的，你最好准备好去你的数据中心一趟，否则就准备好和你的主机打一些紧张的电话，他们会试图解决发生的事情。

亚马逊的云再次拯救了我们。借助随时随地可以访问的在线管理控制台，您可以从无数选项中进行选择，以恢复您的服务器并使其重新启动和运行。

### 低启动成本

没有什么比免费这个词更让我兴奋的了，有了亚马逊，你不用找太远就能找到它。如果你准备好进行转换，亚马逊将允许你建立一个为期一年的免费微型服务器。

这对小型企业意味着什么？首先，在创业的第一年(也可能是最艰难的一年),少了一项需要担心的成本。

如果你已经做好了准备，它可以让你不费吹灰之力地将你的平台转换到云上，并在你准备好的时候启动。

最后，如果你是一名网页设计师，你可以向潜在客户炫耀一年的免费托管服务，并把他们安置在自己的个人云服务器上。你会成为英雄，每个人都很开心。

# 小企业优势

任何想在网络上成功的人都必须面对这样一个事实:这是一个高度竞争的世界。很难知道在哪里投资你的时间和金钱，是在开发，设计，搜索引擎优化，服务器等。你可以确定的一件事是:**云计算让游戏场地变得平坦。**

X 公司是一家价值数百万美元的公司，而你才刚刚起步，这并不重要。在托管方面，竞争环境是平等的——双方都可以利用所有相同的工具和资源，投资比例相同。

事实上，你的网站在欧洲的加载速度比另一个没有利用亚马逊的云和内容交付网络(CDN)的网站快 3 秒**将**带来不同。你现在是老大了。

# 实现无缝切换

现在，我知道你们都在大肆宣传要迁移到云，那么，你们是如何做到的呢？这是最棒的部分！在大约 15 分钟内，您可以启动您的第一台 Windows 或 Linux 服务器，由于我们方便的免费微型服务器，它将为您免费！这里有一个如何开始的快速演练。

## 注册 AWS

进入亚马逊 EC2 云之旅的第一步是向亚马逊网络服务(AWS)注册一个账户。幸运的是，不费吹灰之力。只需[创建一个 AWS 账户](https://aws-portal.amazon.com/gp/aws/developer/registration/index.html)，通过一个简单的电话认证你是一个真实的人，你就可以上路了。

## 设置您的服务器

既然已经登录并准备好了，是时候设置您的服务器了。这里有大量的选择，看起来有点令人生畏。这里有一些指导方针可以帮助你做出选择。

主要选择在 Linux 和 Windows 操作系统之间。大多数人将需要根据他们的开发者选择一个或另一个，但如果你有选择权，值得一提的是，Linux 服务器的运营成本要低得多，亚马逊也有自己的 Linux 服务器选项。如果你在决定从哪个 Amazon 机器映像(AMI)中选择时有困难，你可以遵循大量的 windows 和 Linux 的教程来帮助你。

# 生活在九霄云外

现在，当您的服务器转动和搅动时，您可以高枕无忧，为您的 web 浏览器提供可扩展、可靠的内容和出色的体验。请记住，亚马逊有很多额外的工具来帮助你为你的用户量身定制最佳体验，或者进一步扩展你的网站。查看完整的 [AWS 产品列表](http://aws.amazon.com/products/)，在九霄云外享受生活的乐趣！
