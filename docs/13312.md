# Top 3(免费！)域名生成器

> 原文：<https://www.sitepoint.com/top-3-free-domain-name-generators/>

找到一个可用的域名一天比一天难，不仅仅是因为[域名投机](http://domainnamekiting.com/)，还有[域名投机的巨大生意](http://money.cnn.com/2005/11/30/technology/domains_biz20_1205/)。

值得庆幸的是，当你要为你的企业或事业注册一个新域名时，你并不是在孤军奋战。这里有三个(免费！)工具，帮助整理数以百万计的可用域名，并帮助您找到最理想的域名。

[**1。半身像一个名字**](http://www.bustaname.com/)

最强大的域名搜索工具。

输入几个关键词，点击 2 号框中的蓝色箭头使用包含的词库，然后观察软件将这些词组合成几十个甚至几百个可能的变体。您可以选择要组合的单词数量、想要的域名扩展名，以及要添加到关键字列表中的前缀和后缀。你甚至可以把你的名词复数化(pant -> pants)，掉元音(flicker -> flickr)。

最棒的是，一旦列表生成，你可以根据可读性、长度或音节数对其进行排序。

[**2。命名男孩**](http://www.nameboy.com)

只需输入一个主要关键字，第二个关键字(可选)，并指定您是否愿意接受带连字符的域名，然后就可以了！NameBoy 在域名的缓存数据库上工作，所以速度很快，但这也意味着偶尔你喜欢的域名会被注册，所以赶快行动吧！

**[3。123Finder](http://www.123finder.com/)**
输入关键字查找可用域名，或按长度浏览可用域名的冗长列表。寻找 4 个字母的域名？有四个。还有 4383。net 的。您可以通过仅显示包含或与某些关键字相关的域名，以及它们是否有数字或连字符来缩小列表。

更酷的是，你可以通过搜索十几本不同的字典来找到可用的域名。，包括意大利语、美国地名、男/女名词典。

[**奖励#4:挑剔域名**](http://www.pickydomains.com/)

还是卡住了？让别人替你做这件事。挑剔的域名搜索收费 50 美元。最棒的是，如果你对他们提出的东西不满意，你会得到全额退款。
