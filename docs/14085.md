# Ajax 模式

> 原文：<https://www.sitepoint.com/ajax-patterns/>

许多人都熟悉设计模式的概念；对可重用技术的简短解释，你会发现自己一次又一次地添加到应用程序中。Ajax 模式 Wiki 是将这种模式收集过程应用于 Ajax 应用程序的尝试的开始。虽然 wiki 列出了各种各样的模式，但是只有大约一半的模式被记录在案，并且文档通常只包含对模式的简单解释，而没有例子。这有可能成为一个有用的资源:当你开发一个 web 应用程序时，快速访问这里会给你一些提示，告诉你某人可能已经在哪里开发了你正在寻找的功能，或者甚至建议一个你没有想到的技术。然而，它需要人们权衡并填补空白。
哦，正如有人指出的，“Ajax”、“模式”和“Wiki”共同组成了这个名字，使它 100%符合时髦术语。:-)
