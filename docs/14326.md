# 回到健全的营销和销售策略的三个基本原则

> 原文:[https://www . site point . com/back-to-the-three-fundamental-of-a-sound-marketing-and-sales-strategy/](https://www.sitepoint.com/back-to-the-three-fundamental-principles-of-a-sound-marketing-and-sales-strategy/)

[https://www.sitepoint.com/blog-post-view.php?id=207189](https://www.sitepoint.com/blog/)

你在上一篇博客中的精彩评论(见上面的链接)表明，现在是回到营销和销售战略起点的好时机。

要在业务开发中取得成功，必须具备三个基本要素:内容、过程和环境。让我们看看每一个，你可以评估你做得如何。

内容:你需要在你所做的事情上表现出色。否则，你不会得到重复的业务。你知道这一点，但是在网页设计和开发中有很多黑客。如果你碰巧是他们中的一员，对你来说这将是一条漫长而艰难的路。

过程:你需要一个过程来获得可见性，保持可见性，取悦现有客户，并跟进以获得更多业务。太多的 IT 专业人士坐在办公桌后等着电话铃响。这是破产的秘诀。

语境:语境是营销中最重要也是最容易被忽视的方面。它指的是“你从哪里来”或“你的存在方式”在销售专业服务时，你需要重视价值、信任、关系、服务、信誉和专业精神。每一个都值得定义:

价值:你为你的客户解决商业和个人问题，给他们带来的利益远远超过你的费用。

信任:你努力让你的潜在客户和客户了解你，相信你的能力和诚信。

关系:每个联系人都是一个长期的关系，一个你能给你带来重大价值的人(如果你给他们带来价值的话)。

服务:你先做关于他们的，你再做。

可信度:随着时间的推移，你建立了自己的能力、价值和一致性，这样人们就知道你是一个可靠的专家。

专业精神:你总是有一种关于服务、卓越和持续改进的积极态度。

好了，这些是基本原则，我希望这个回归基本原则对你有用…..
