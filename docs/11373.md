# 如何启用 Windows 7 GodMode

> 原文:[https://www.sitepoint.com/windows-7-god-mode/](https://www.sitepoint.com/windows-7-god-mode/)

这篇文章写于 2010 年，至今仍是我们最受欢迎的帖子之一。如果你渴望了解更多关于 Windows 的知识，你可能会对这篇关于 Windows 8 生产力的文章很感兴趣。

![GodMode on Windows 7](../Images/22dcc9188d9d4d7d7e51fb4608fb3659.png)

这里有一个在互联网上流传的巧妙的 Windows 7 技巧。它启用了“GodMode”，这是微软开发团队设计的一个术语，它提供了一个访问所有 Windows 设置的单一位置，而无需浏览控制面板中的选项和文件夹。

要使用它:

*   创建一个新文件夹。
*   ![GodMode window](../Images/cb3ea1107f98da43fbf4757c6937c230.png)Rename the folder to

    **上帝模式。{ ed7ba 470-8e 54-465 e-825 c-99712043 e01c }**

    (注意可以更改“GodMode”文本，但后面的句号和代码号必不可少)。

*   文件夹图标会改变—双击它会显示 *GodMode* 窗口:

![GodMode window](../Images/013d489dc582afab2bbf3d4393d0d0fe.png)

该窗口显示了近 50 个带有配置选项快速链接的部分。严格来说，这不是一个上帝模式，因为所有的选项都可以在其他地方找到。它更类似于一个“所有任务”列表——但你可能会发现这比在 Windows 众多的屏幕和面板中磕磕绊绊要容易得多。

**红色大预警！**

这个技巧似乎在 32 位和 64 位版本的 Windows 7 上都有效。Vista 32 位和 Windows Server 2008 32 位应该也可以。但是，众所周知，64 位版本的 Vista 会崩溃，您可能需要在安全模式下启动或进入命令行来删除该文件夹。
