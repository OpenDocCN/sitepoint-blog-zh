# 营销人员可以从苹果的 Siri 中学到什么

> 原文：<https://www.sitepoint.com/what-marketers-can-learn-from-apples-siri/>

苹果新 iPhone 4S 最热门的新功能是 Siri，这是一款语音识别个人助理应用程序，可以执行多种任务，如打电话、发送短信或查看天气。Siri 吸引了许多 iPhone 4S 用户，包括我，他们花时间试图发现新的怪癖和幽默的回应。

Siri 对苹果来说是一个巨大的成功，营销人员应该注意到这一产品发布中的一些教训。下面我们来看几个。

### 直觉和自然

到目前为止，大多数语音识别程序需要特殊的命令和语法才能工作…大多数是用不自然的声音说出的不自然的命令。Siri 会倾听你的请求，确定你想问什么，并且在理解常规语言方面做得非常出色。

这里的教训是说你的客户的语言，而不是我们行业的技术术语。写关于 SEO、PPC 和 HTML 的提案很容易，但是大多数客户想知道你将如何给他们带来新客户，新网站将如何帮助他们的底线。

### 学问

Siri 随着时间的推移不断学习，对你要求她做的事情理解得更多。她在某种程度上也可以被“训练”——你可以告诉她你的配偶、父母或孩子是谁，她会记住以备将来参考。

这里的教训是要学会预测客户的长期需求，这样你就可以在他们提出要求之前为他们提供信息或服务。如果你的客户经常询问他们的搜索引擎排名，给他们发送详细的报告，概述他们在顶级搜索引擎中的排名。如果他们关心付费广告的投资回报率，给他们发送每周报告，准确显示他们的广告表现如何。

通过预测他们的需求，在他们提出要求之前或要求时满足他们，你将成为他们团队中有价值的一员。

### 幽默感

Siri 很有幽默感。问她“什么是最好的智能手机”或者告诉她“把我传送过来，斯科特！”听听他们的回答。有很多网站专门记录 Siri 的机智。

这里的教训是简单地享受乐趣和幽默感。肯定有认真完成工作的时候，但也有分头行动、享受美好时光的时候。在我们的办公室，我们喜欢玩飞镖游戏，我们经常互相恶作剧，当客户来拜访我们时，我们也会和他们开玩笑。

让客户看到你的个性可以扩大你和他们的关系。当然要知道自己的界限，但是玩得开心！
