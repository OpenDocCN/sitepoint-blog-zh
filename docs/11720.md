# 无聊？7 种将停机时间转化为正常运行时间的方法

> 原文:[https://www . site point . com/turn-down-into-uptime-in-business/](https://www.sitepoint.com/turn-downtime-into-uptime-in-business/)

当你是个体经营者时，经历停工期是很常见的。事实上，根据我的经验，往往会有一个多少有些可预测的周期，它会带来工作流增加的时期，然后是工作流减缓的时期。

对于每个企业来说，放缓何时发生以及持续多长时间都是不同的。但是，如果你的企业和我的一样，工作流程周期的周期可能会很紧张——从你处理不了的新工作到突然的安静期。不过，我已经开始欣赏这种休息时间了，我做了很多事情来把这种安静变成我工作的高效时间。以下是一些想法:

**1。重温你的目标**

花些时间分析你的目标，你已经取得的进展，以及你需要做些什么来坚持下去。创建一些小目标也是有帮助的，这将有助于这个过程，并给你一些具体的活动，这些活动有可衡量的结果，你可以立即去做。

**2。关注社交媒体**

查看和修改你的社交媒体账户，更新你的个人资料，加入新的在线商业网络。工作放缓也是在社交网络上变得更加活跃的好时机，并开始建立和加强你的人际关系。

**3。评估你的网站**

如果你接下来的客户工作比较少，这可能是仔细审视你自己网站的最佳时机。通读内容，检查您的联系信息，执行搜索引擎优化检查，并测试表单和功能。你可能还想更新你的作品集，让它反映你最近的项目。

**4。联系客户**

如果你在项目结束时没有完成工作，停工期是向客户发送反馈调查的好时机，以获得他们对你工作的评价。你也可以利用这段时间向客户索要推荐信，用于你的营销材料。

**5。跟进和重新推介**

你有一些提议还没有得到回复吗？跟进潜在客户，看看他们是否还有兴趣。你可能会产生一些新的工作，或者至少清除你的待处理的机会列表。
 **6。变得有条理**

如果你一直在摸索你的任务，努力管理项目，现在是一个很好的时间来处理你的系统。迁移到一个新的项目管理应用程序，或者尝试一个新的[待办事项列表系统](https://www.sitepoint.com/online-to-do-list-tools/)，变得更有效率，不那么分散。这也是一个很好的时间去[打扫你的办公室](https://www.sitepoint.com/feeling-overwhelmed-clean-your-desk/)，整理你的文件，关注那些容易被遗忘的行政任务。

**7。远离工作** 
工作流程的放缓可能是休息、远离工作、更新你的观点的最佳时机。根据我的经验，平静期过后通常会是工作的快速增长，所以花点时间享受暴风雨前的平静吧。

如果您可以预测这段停机时间，那么保留一个运行中的停机任务列表可以帮助您避免浪费时间。当工作进展缓慢时，保持专注和忙碌将有助于你避免因缺少工作而带来的压力和焦虑。

此外，如果你真的充分利用了停工期，你可能会发现停工期过得太快，让你无法完成所有的停工期目标，这是一个不错的位置。
