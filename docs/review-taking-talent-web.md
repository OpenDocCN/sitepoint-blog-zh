# 回顾——将你的才华展现在网络上

> 原文：<https://www.sitepoint.com/review-taking-talent-web/>

由网页设计传奇 Jeffrey Zeldman 撰写并由 New Riders 出版的《把你的才华带到网上》是一本涵盖广泛主题的书。从界面设计到不同 Web 开发团队成员的功能，都包括在内。泽尔德曼在这本书的开头描绘了一幅网页设计师的图画，他们的角色、职责和行业工具。从广义上讲，网页设计师是什么以及他们在网页开发和交流中的角色。

##### 事实真相

在了解了基本知识，并回答了诸如“什么是网页设计师？”这本书深入研究了网页设计师在日常工作中使用的不同技术。

HTML、Photoshop、样式表、JavaScript、SSI 和富媒体在 200 页的五章中都有涉及。Web 调色板、图像优化和跨浏览器兼容性等必备知识都被讨论到了，尽管没有一个足够深入地满足典型 Web 开发人员的日常需求。

我赞扬 Zeldman 对可用性和信息架构主题的深入研究。市场上有太多的网页设计书籍从艺术的角度来看待网页设计。但归根结底，网站是为用户服务的，用户必须能够使用和浏览网站来实现他们的目标。泽尔德曼认识到网站不仅仅是一幅数码画，这值得称赞。

这本书穿插了大量的代码片段，其中一些相当冗长。唉，没有提到可以下载代码的网站。我同情那些试图打出从 317 页开始的两页 JavaScript 的人…

##### 写作风格

这本书的写作风格非常好，每一章都充满轻松幽默。在开始的几章之后，一些笑话变得完全令人厌倦，仅仅是因为它们太多了。虽然我不责怪作者试图在一个平淡无奇的话题上给这本书增添趣味，但在一定程度上，一本书变得更幽默，而不是网页设计。

另一件困扰我写作的事是不断提到“我们”和“我们的”。在书的封面上没有提到任何合著者的情况下，我开始怀疑泽尔德曼是否患有分裂型人格障碍。似乎作者试图通过使用“我们”和“我们的”而不是“我”来转移对他的信仰和观点的任何批评。如果你在一本书里打印陈述，袖手旁观他们！

##### 谁应该读它？

话虽如此，尽管幽默乏味，我还是设法看完了整本书。将你的才华展现在网络上对于考虑向网络转型的平面设计师和艺术总监来说是再好不过了。泽尔德曼以清晰有趣的风格提供了一个非常坚实的概述。

希望成为专业人士的个人，以及希望在 Web 开发主题的整个范围上进修课程的专业人士，也将从阅读本书中受益。

《新骑手》以其高质量的书籍而闻名，它不会让 400 多页的书*把你的才华带到网上*而失望。40.00 美元或 59.95 加元，虽然不便宜。

评分:4/5
