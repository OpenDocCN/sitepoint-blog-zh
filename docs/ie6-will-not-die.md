# 就在你以为 IE6 会死的时候…

> 原文：<https://www.sitepoint.com/ie6-will-not-die/>

虽然像 YouTube 这样的主要网站正在放弃对 IE6 的支持，正如 Alex 指出的[，但是在大公司和公共机构中进展要平稳得多。这里有两个更令人担忧的例子…](https://www.sitepoint.com/hilary-ie6-and-the-state-of-the-state-department/)

## 柑橘

我们的第一个案例研究是 Orange——一家主要的电信和互联网服务提供商。据一名在英国呼叫中心工作的技术人员称，多达四分之一的员工转而使用 Firefox，主要是为了利用标签式浏览和更高的性能。然而，一封泄露的内部邮件显示，该公司已经禁止 Firefox 而支持 IE6。员工还受到威胁，如果他们在安装任何未经授权的软件后遇到电脑问题，将被罚款 250 英镑:

> 任何情况下都不要下载 Firefox。从网上下载任何应用都是违反 Orange 政策的。在操作环境中不支持 Firefox。Orange Web 应用程序都设计为在 IE6 上运行，因此在 Firefox 上功能可能会受到影响。

Orange 的主要原因是公司电脑的稳定性和安全性。IE6 作为公司补丁程序的一部分会自动更新，而 Firefox 和其他浏览器不会。这是一个重要的因素，但一个打了补丁的 8 年前的浏览器真的比一个更现代的版本更稳定、更安全吗？

## 英国国防部

大多数英国政府部门仍然强制员工使用 IE6，但大多数都计划在 2010 年前采用 IE7(我想，迟做总比不做好)。唯一的例外是国防部。尽管他们比其他部门更担心安全问题，但这并没有阻止他们向全球 30 万用户推出基于 IE6 的*安全信息基础设施计划。*

 *议员汤姆·沃森表达了他的失望:

> 许多公务员使用网络浏览器作为他们工作的工具。它们和笔和纸一样重要。因此，强迫他们使用世界上最破旧的浏览器是一种罕见的职场残酷行为，应该被制止。
> 
> 政府获得安全上网倡议建议公司从 IE6 升级，所以常任秘书应该身体力行。我不明白为什么公务员不能选择使用火狐、Chrome 或 Safari。英国网络工作者应该得到更好的待遇。

你可能会认为这些组织是勒德分子，落后，或无知。不幸的是，他们雇佣了成千上万的人，这些人可能是你的商品和服务的顾客。这些用户应该被忽视，因为他们继续使用 IE6，尽管他们自己没有错吗？
