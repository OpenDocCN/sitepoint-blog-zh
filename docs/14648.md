Via [西蒙](http://simon.incutio.com)，【MySQL 为何增长如此之快。在实际回答问题(这很难回答)时，提出了许多有趣的观点。

任何真正思考过 MySQL 相对于其他数据库引擎，特别是 [PostgreSQL](http://www.postgresql.org/) 的人都知道，它几乎是“不应该的东西”。

个人认为 PHP 也是一个[颠覆性的技术](http://radio.weblogs.com/0126029/Disruptive.htm)。

任何一个很了解 PHP 的人都可以告诉你一两件关于壁橱里的怪物的事情，这是在你向其他人公开辩论之前。

但是说到易用性，PHP 是最难超越的。它不仅对初学者来说是一种很好的语言，对于专注于解决一小组问题的技术来说也有很强的商业价值，并且像 PHP 脚本一样容易部署。PHP 吸引了“应该更了解”的开发者。

在 J2EE 和。NET，按理说 PHP 应该被谴责为默默无闻。相反，它还在继续增长，随着 PHP5 的出现，它似乎吸引了越来越多的“企业”关注。
