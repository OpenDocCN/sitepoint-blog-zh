# 当心变得依赖搜索

> 原文：<https://www.sitepoint.com/beware-of-becoming-search-dependent/>

Shayne 本周从悉尼的[搜索营销博览会](http://www.searchmarketingexpo.com.au/)发回报道。

在 SMX 悉尼的第一场会议中， [SEOmoz](http://www.seomoz.org/) 的首席执行官 Rand Fishkin 讲述了一个最近成立的公司的故事，一天早上醒来发现他们来自谷歌的有机搜索引擎流量莫名其妙地消失了。来自谷歌的流量代表了他们日常访问的最大份额，所以这对他们的底线产生了戏剧性的即时影响。我确信这是一个非常普遍的网站过度依赖单一流量来源的故事。

对于搜索引擎营销，尤其是 SEO，你永远不会忘记**这些引擎不欠你什么**——它们的行为符合客户体验的利益(通常是它们提供的结果的相关性)，这推动了需求，进而推动了它们的广告收入。诚然，所有的大玩家都与搜索引擎营销人群玩得很好，但归根结底，他们只有*真正*关心对他们有什么好处——而不是对你…

这里的关键建议是不要让你今天健康的搜索引擎性能欺骗你对明天的安全感。如果谷歌内部的一个开关被触动，或者一个影响搜索引擎算法的杠杆被调整，你的搜索引擎性能可能会发生巨大变化。你需要有能力应付它。

Rand 强调了与最佳实践 SEO 技术保持联系的必要性，这样你就可以确保自己处于变革的最前沿。从搜索引擎优化的角度来看，这当然是一个可靠的建议，但是这个建议不应该仅仅应用于搜索。依赖任何一个特定的流量渠道都是一个很大的风险。每一个来源(直接的、推荐的、广告、社交媒体等)都有改变的潜力，就像搜索引擎一样。

一个规划良好的网站将包括一个流量来源的均匀混合。网站经理将致力于建立所有渠道的流量，而不是只关注一个渠道。这样，如果井在一个地区干涸，大量的水(交通)仍然会从其他渠道流出。

看看你目前的[网络分析包](https://www.sitepoint.com/10-web-analytics-packages-for-tracking-your-visitors)中的流量来源，问问你自己:如果其中一个渠道明天停止提供流量，这对我的业务意味着什么？如果有一个渠道支配着所有其他渠道，那么可能是时候考虑平衡你的流量来源了。
