# 失落的谈话艺术——鼓励网上联系

> 原文:[https://www.sitepoint.com/encouraging-contact-online/](https://www.sitepoint.com/encouraging-contact-online/)

Web 行业充斥着诸如跨浏览器兼容性、可用性、可访问性等话题，但是许多 Web 设计师是否忽略了网站的一个关键功能:开始对话？

是的，我们都非常小心地确保每一张图片都有一个 ALT 标签，每一页都能在不同的浏览器上很好地呈现，我们的访问者可以很容易地获得我们的产品或服务信息，但是我们有没有把自己放在想要联系我们的客户的位置上呢？

不管你的网站是信息型的，还是电子商务型的，或者只是一个“小册子”，你可能犯的最大错误之一就是隐藏你的联系信息，或者不能鼓励网站访问者联系你。

网络用户是一群善变的人——他们只会在很短的时间内寻找你的联系方式，然后就放弃了。即使你在每一页都提供了普通联系方式的链接，这也只是战斗的一半。

当用户到达你的联系页面时会发生什么？很诱人吗？它要求反馈或评论吗？这些只是我们设计者容易忽视的网站可用性问题中的一部分。

##### 你的联系页面像名片一样吸引人吗？

你有没有在你的网站上写满有个性和强烈销售焦点的所有内容，只是通过张贴一个只包含简单细节而没有其他内容的联系页面来破坏这种感觉？快速——现在就访问你的联系页面，问问你自己“这个页面会鼓励我去联系这些人吗？”

你可能在想‘但我想做的就是卖小配件！’，也许是这样，但如果客户找不到一种简单的方法来询问你出售的小工具是否是他们需要的(即使你觉得你已经在你的网站上回答了他们的问题)，那么你就失去了一笔销售。

支持信息也是一样——你可能已经涵盖了你认为客户可能会问的关于你的服务或产品的每一个问题，但总会有你遗漏的东西。邀请人们给你发邮件或打电话，跟踪这些询问，为你的下一次网站改版指明方向。

这不仅仅是要求你网站的每个访问者给你发电子邮件(相信我，他们不会)，而是表明你在倾听，你是值得信赖的，你鼓励对话——积极的情绪只会帮助你赢得访问者。

许多人认为网站上标准的“联系我们”页面就像他们的名片，但区别在于交付方式。当你把名片递给某人时，你鼓励他们联系你，当你和他们说话时，你对他们微笑，你获得了他们全神贯注的好处。当他们访问你的网站时，你会给他们你的基本联系方式吗？

一个很好的电子商务网站的例子是美国的 Williams Sonoma。他们不仅在每一页的页脚都有一个“客户服务”的链接，还在每一页上都有一行文字“如有疑问或需要帮助，请 24 小时拨打电话！”，以及他们的电话号码。看看[他们的联系页面](http://ww2.williams-sonoma.com/cs/index.cfm?sect=feedback&src=csvs)。它不仅通过一个写得很好的介绍来鼓励反馈，而且它承诺在一个工作日内回复，并有一个带有联系主题选项的下拉框。

另一个例子，科尔斯·迈尔(Coles Myer)可以从他们的书中吸取经验。如果你有幸点击[【公司信息】](http://www.corporate.colesmyer.com/)，你会被带到一个在子菜单中有“给我们发邮件”链接的页面。[目标页面](http://corporate.colesmyer.com.au/ms/email CML/email CML.htm)只有三个文本链接，没有介绍性文本或表格。

联系信息的一个更糟糕的例子是澳大利亚全国快餐连锁店 [Hungry Jacks](http://www.hungryjacks.com.au/) ，他们在商店定位器中有商店位置和电话号码，但在其整个网站上没有一个全国客户的电子邮件地址或电话号码。

##### 隐藏你的真实身份？

我们都看到了研究数据显示，人们害怕在网上使用信用卡——如果他们找不到可以联系你的实际地址或电话号码，这是理所当然的。如果一个网站的联系信息只是一个邮政信箱地址，或者更糟的是，根本没有地址，你会乐意通过这个网站提交你的详细信息吗？

人们对隐藏真实身份或联系方式的商家变得警惕起来。事实上，澳大利亚联邦政府的[“电子商务最佳实践模式”](http://www.ecommerce.treasury.gov.au/)指出:

<q>*“企业应向客户提供准确、易于获取的信息，包括……电子邮件地址、电话和其他联系信息。”*T3】</q>

看看大型澳大利亚零售商[愿望清单](http://www.wishlist.com.au)。他们网站的每个页面不仅链接到他们的联系方式，还包括他们的电子邮件、免费电话和实际地址的详细信息。

像[一个邪恶的礼物](http://www.awickedgift.co.uk/)这样的购物网站需要查看它们的细节，以及它们有多难找到。我在这个网站上发现的唯一细节是一个电子邮件地址——没有电话号码或实际地址！当你从他们的网站上购物时，这并不能让你充满信心。

UnderwearFactory.com 稍微好一点。如果你深入他们的联系页面，你会发现一个电子邮件地址和电话号码，但他们的网站上没有列出他们的物理位置，免费电话号码，或其他细节。

不要认为你可以带着一个邮政信箱离开——他们也只是喊着“后院生意”。如果你绝对有必要，那就把它列出来，但是如果可能的话，也包括你的实际地址——这表明你愿意亲自支持你的网络形象。

##### 一半的网站不想要你的互动

看看你接下来访问的几个网站。我对一系列行业的 50 个信息和电子商务网站进行了快速研究，结果显示，50%的网站在每个页面上都有联系方式或详细信息的链接，但页面上没有解释性或欢迎文本。还有 25%的网站要么有很难找到的链接，要么根本没有链接。

这就好像这些网站所有者在说‘我们希望你把钱给我们，但请不要考虑联系我们。’

与其写着“通过 contact@business.email.address 或拨打 1234 5678 与我们联系”的空白页面，为什么不写为“我们期待您的回复，请通过……与我们联系”？这听起来更有吸引力，我保证会有更多的访问者联系你，如果你打得好，最终会成为销售线索。

设身处地为一个害羞的访问者着想，浏览你的网站——从第一页到你的联系方式，它鼓励对话吗？如果没有，想想有多少害羞的网站访问者。

看看 [Y3K 组的联系页面](http://www.y3kgroup.com/contact/)。它列出了他们的所有细节，包括实际地址等等，但是它鼓励你去联系他们吗？如果在顶部有一个段落邀请用户反馈和交流，这个页面会更有效。

关于邀请互动的一个很好的例子，请查看 skinnyCorp 的联系页面。它不仅提供了详细的联系方式和包括董事姓名在内的所有细节，而且它以一个伟大的开场白开始，鼓励你与他们联系。

##### 方便的联系人清单

下一次你开发或更新一个网站时，检查你是否包括了以下内容:这些要点将鼓励联系，并帮助你完善你的网站，促进销售。

*   总是有一个以上的接触媒介。不要只使用电子邮件或电话:人们希望两者都用。
*   在你网站的每一页都写上你的主要电话号码和客户服务邮箱地址。
*   确保每个页面至少有一个链接指向你的联系信息。如果有两个就更好了(例如，在导航菜单中作为图像，然后在页脚中作为文本)。
*   在你的联系人页面上写下吸引网站访问者的开场白。
*   不要以为你已经回答了网站页面上的每一个客户问题——我可以保证你没有。以一种鼓励客户在有问题或反馈时联系你的方式来写你网站的所有内容。
*   最后，一旦网站访问者联系了你，尽快回复。人们期望电子邮件比电话信息更快被执行。
