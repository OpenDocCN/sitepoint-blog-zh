# 这次封闭能为苹果效力吗？

> 原文:[https://www . site point . com/can-closed-work-for-apple-this-time/](https://www.sitepoint.com/can-closed-work-for-apple-this-time/)

《T2 纪事报》最近的一项民意调查询问谁更封闭，微软还是苹果？高达 55%的受访者认为这个可疑的荣誉属于苹果(21%的人说是微软，24%的人说他们在这件事上是平等的)。

尽管苹果目前的操作系统是建立在开源的 UNIX 基础之上的，尽管他们的网络浏览器是建立在开源的 WebKit 渲染引擎之上的，但苹果总是出了名的封闭。在 Register 的调查中，人们认为苹果比微软更封闭的最普遍原因是他们产品线的端到端专有性质。

“无论是 OS X 与 Mac 的联姻，iPod 对 iTunes 服务的依赖，还是通过苹果商店控制的 iPhone 软件分销，都有一种强烈的感觉，即开放并不总是苹果最优先考虑的事情，”戴尔·维尔解释道。

诚然，苹果喜欢保留对其产品的完全控制权。例如，通过只允许 OS X 在他们创造的硬件上运行，苹果不仅控制了他们产品的定价，还限制了降低用户体验的硬件不兼容问题的可能性。如果说所有苹果产品背后有一个驱动力，那就是维持高质量的用户体验。

这一次，控制我们计算生活的真正战斗可能发生在移动平台和网络上。这不是 Windows vs. Linux vs. OS X，而是 iPhone vs . Blackberry vs . Android vs . Windows Mobile。正如法尔哈德·曼朱所想的，“难道史蒂夫·乔布斯在过去的 30 年里没有学到任何东西吗？”

“看着谷歌和苹果在移动业务领域开拓空间，人们很难不想到历史正在重演。在 20 世纪 70 年代和 80 年代，苹果创造了第一台伟大的个人电脑。但是因为苹果关闭了它的平台，所以是 IBM、戴尔、惠普，尤其是微软从苹果的创新中获益，”曼朱最近在一篇关于 Slate.com 的文章中写道 Mac 的操作系统只能在 Mac 电脑上运行；Windows 可以在很多不同公司的硬件上运行。这使得非苹果电脑比苹果电脑更便宜——硬件制造商之间的竞争压低了价格——也更有用，因为第三方开发者蜂拥而至为 Windows 编写必备程序。苹果似乎对 iPhone 采取了类似的限制性策略。"

Manjoo 的理论是，也许乔布斯认为这一次人们更热衷于用户体验和头痛最小化，而不是他们关心的开放性。“开放平台在技术上比封闭平台复杂得多，”Manjoo 说，并指出开放平台有更多的技术问题，因为它们必须与更广泛的技术配合，并且它们吸引了令人讨厌的开发者(即病毒和间谍软件的制造者)，所有这些都损害了最终用户的体验。

然而，正如我们在 8 月份提到的[，封闭的平台经常让开发者与平台提供商发生冲突，这最终会驱使开发者转向其他选择。应用程序是任何平台的生命线。因此，如果封闭的平台最终以维护各种各样有用的应用程序为代价来保护用户体验，那么开放将最终获胜。](https://www.sitepoint.com/will-closed-platforms-ruin-the-web/)

如果《华尔街日报》的调查显示了普遍的情绪，那么乔布斯可能又犯了一个错误，并会为此付出长期代价。而且，正如 CNET 指出的那样，苹果开始对竞争对手采取一些非常类似微软的行动来保护其垄断地位。这对苹果来说是一条危险的道路，可能会改变公众对该公司的看法，这在目前看来是非常积极的。
