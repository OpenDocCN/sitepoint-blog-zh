| Chrome OS | 0.30% |
| Playstation | 0.09% |
| 微软游戏机 | 0.03% |
| 其他的 | 0.04% |

当一个人浏览 StatCounter 监控的 300 万个网站中的一个时，从收集的操作系统指纹中整理统计数据。他们强调这些平台上的网络活动，而不是安装。主要观察结果:

*   第二、第三和第五名由移动操作系统占据。
*   如果除去智能手机设备，Windows 占有 75%的市场份额。Windows 7 占安装量的 54.3%，其次是 Windows 10 (17.5%)、Windows 8.x (17.1%)和 Windows XP(尽管微软两年前就停止了支持，但仍占 9.0%)。每千名访问者中就有一人使用 Windows 98！
*   Linux 似乎有点低，尽管有大量的“未知”，其中可能包括一些发行版。世界上大多数的网络服务器运行 Linux，但是这些设备很少用于网络浏览(除了一小撮被虐待狂!)!).
*   鉴于谷歌的推广影响力，Chrome OS 小得令人惊讶。也就是说，Chrome OS 设备的价格通常与功能更强大的 Windows、Mac 和 Linux 电脑一样高。
*   销售数据显示，Playstation 4 的销售速度是 Xbox One 的两倍，但在 Playstation 上浏览网页的速度是 Xbox One 的三倍。

操作系统变得越来越多样化。原因:*移动设备* …

## 2016 年 1 月至 2 月全球移动浏览器统计

二月份的手机使用量几乎没有变化，现在占所有网络活动的 41.11%。

2016 年 2 月热门移动浏览应用包括:

| 移动浏览器 | 一月 | 二月 | 变化 | 亲戚 |
| 铬 | 37.69% | 35.92% | -1.77% | -4.70% |
| UC 浏览器 | 18.63% | 20.10% | +1.47% | +7.90% |
| 苹果手机 | 18.31% | 18.21% | -0.10% | -0.50% |
| Opera Mini/Mobile | 10.91% | 10.74% | -0.17% | -1.60% |
| 机器人 | 10.24% | 9.49% | -0.75% | -7.30% |
| 不变的 | 1.90% | 1.83% | -0.07% | -3.70% |
| 其他人 | 2.32% | 3.71% | +1.39% | +59.90% |

Chrome 出现了不寻常的下跌。这可能部分是因为三星 Android 互联网浏览器以 1.41%的市场份额排名第七。该应用程序适用于三星 Galaxy 手机、平板电脑和智能电视。它重量轻，具有广告拦截功能，并提供特定于设备的功能，如指纹扫描以进行 ID/密码认证。据我所知，当前版本最近从 WebKit 转移到了 Blink 渲染引擎，尽管[伺服](https://servo.org/)是未来的选项。

由于页面重量增加和网络拥挤，移动网络浏览可能会很痛苦且成本高昂。供应商已经用广告拦截选项做出了回应，这可以大大减少下载时间并提高响应速度:

*   [苹果在 iOS 9](http://www.bbc.co.uk/news/technology-34173732) 中引入了广告拦截功能。
*   新的[勇敢浏览器](https://www.brave.com/)提供广告拦截和替换选项。
*   [Flynx for Android](https://play.google.com/store/apps/details?id=com.flynx) 默认提供类似阅读列表的视图，并支持后台加载。
*   许多第三方浏览器，如火狐浏览器和海豚浏览器都提供广告拦截选项或扩展。

不出所料，谷歌尚未实施类似的屏蔽技术，因为它是网络最大的广告商之一。然而，道路是明确的。如果你忽略了处理沉重的或侵扰性的广告，浏览器会为你做这项工作。

我将在下个月更深入地讨论广告屏蔽的含义和选择。
