# YouTube 将放弃对 IE6 的支持

> 原文：<https://www.sitepoint.com/youtube-drop-ie6/>

现在我希望我不会让任何人难过，但是我们需要面对不可避免的事情。IE6——一个我们都学会喜欢的浏览器——不会永远存在。它存活的时间比我们任何人预期的都要长，它的市场份额仍然比 Chrome、Safari 和 Opera 加起来还要大。虽然一些较小的网站已经放弃了对 8 岁的浏览器的支持，但没有一家较大的公司敢削减五分之一的用户。直到现在。

谷歌旗下的视频分享网站 YouTube 正在逐步取消对 IE6 的支持。YouTube 是网络上最大和最受欢迎的网站之一，因此这一行动可能会对用户数量产生巨大影响。

YouTube 的 IE5.5 和 IE6 访问者现在会看到以下信息和有用的 IE8、Firefox 和 Chrome 下载链接:

[![We will be phasing out support for your browser soon](img/7ce3c9128b7ad9ae1b981e1da5e5de0a.png) *点击查看全尺寸*](https://blogs.sitepointstatic.cimg/tech/113-youtube-ie6-6msgfull.png)

当你使用 IE7、IE8、Firefox、Safari、Opera 或任何其他浏览器访问时，谷歌还在侧边栏添加了 Chrome 下载链接。这个想法最近在 SitePoint 上被提出来，以帮助增加 Chrome 的受欢迎程度。

![Download Google Chrome](img/db5777d8aedf2a467ef7a9f511f99b3a.png)

有证据表明，IE 的使用在过去一个月有所下降，尽管 Firefox 和 Safari 的新版本对这一转变负有部分责任。然而，YouTube 的声明肯定会说服许多个人 IE6 用户升级他们的浏览器。 [xBox360](http://synbay.com/category/XBOX360/ "xbox360")

不幸的是，YouTube 在企业部门的使用率并不高。当前的经济环境和传统的 IE6 应用程序阻碍了许多企业升级浏览器。我们只能希望更多的企业要么更新他们的应用程序，要么采用 Windows 7，这样他们就可以在必要的时候在 XP 模式下运行 IE6。

最后，有传言称 Digg 正处于放弃浏览器的边缘。强硬派已经受够了——这是 IE6 终结的开始吗？
