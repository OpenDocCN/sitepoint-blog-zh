# 再一次用表达式 3

> 原文:[https://www.sitepoint.com/once-more-with-expression-3/](https://www.sitepoint.com/once-more-with-expression-3/)

很久以前，一个聪明的小公司(Fractal Design)在森林里快乐地生活，创造了漂亮的图形应用程序(Painter 和 Expression)。他们有一种天生的天赋，能够创建软件，熟练地模仿真正的蜡笔、颜料和其他自然媒体的属性和质量。每个人都喜欢自己的工作，每个人都很开心。

然后有一天他们醒来，决定卖掉。Corel 最终选择了他们的旗舰产品——Painter。

但是他们有一个很酷的矢量程序——“表情”，后来怎么样了？我记得当时觉得它看起来很有希望，我真的必须去看看。

嗯，在逛了微软网站的密室后，[我又找到了她](http://www.microsoft.com/products/expression/)——独自一人，几乎被遗弃，等待着有人来爱她。

微软拒绝谈论她，可悲是，看起来她越来越有可能会留在那里。

无论如何，如果你还没有检查过她，你觉得你的设计可以使用一个新的角度，这可能是答案。

![](../Images/73c8f0b54881e8b183ccf112d5d72323.png)

在它的核心，‘Expression’是一个矢量程序，就像 Illustrator 和 Freehand 一样，它可以导入和导出 AI 格式的文档(在合理的范围内)。

不过，相似之处也就到此为止了。Expression 中的矢量线条用作“悬挂”自然媒体描边的骨架。虽然这在 Illustrator 技术上是可行的(通过“图像软管工具”)，但它代表了在 Expression 中使用的*核心方法。你不倾向于填充形状——你实际上是给它们上色。*

公平地说，这不是一个容易上手的应用程序。当然，不要急于表达你会在 30 分钟后交付一个闪亮的新设计(除非你和画家在家)。这是一个非常不标准的接口。这不是一个坏的，但你肯定需要投入一些时间来适应它。

然而，在长时间使用 Photoshop 和 Illustrator 之后，这是一种相当自由的使用方式，尤其是如果你有一台平板电脑的话。一支笔的粗细和密度的变化令人印象深刻。

![](../Images/e2fea1fec8c6a49ae4feb9cd55b79a58.png)

我必须承认钢笔工具使用了一种非常奇怪的方法来形成它的曲线。你需要在你的形状周围建造一个“直边围栏”，线条在你的围栏内弯曲*。有点难以解释，但比我最初想象的更直观。*

总而言之，虽然 Expressions 有点偏左，而且肯定开始显示出它的年龄，但它可能会给你的设计武器库增加另一拳——如果你通过了学习曲线的话。它现在在我的工具箱里找到了一个位置。

当然，最好的是，[表情免费送给一个好的家](http://www.microsoft.com/products/expression/)。所有你需要的是一个微软护照登录和一些合理的带宽(我认为这是大约。55MB)。

如果你以前没看过，当你有时间的时候去看看。
