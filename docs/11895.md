# 引荐:怎么说谢谢？

> 原文：<https://www.sitepoint.com/say-thank-you-for-referrals/>

昨天，我发表了一篇关于推荐人的[力量以及如何鼓励客户将你推荐给其他人的文章。虽然提供奖励——金钱支付、免费服务或折扣——可以同时吸引和感谢你的客户，但它们既不理想也不适合所有情况。那么，你该如何感谢你的客户花时间和精力把你介绍给别人呢？](https://www.sitepoint.com/getting-referrals)

**送上一份老套的感谢**

我非常支持手写的个人感谢信。作为一种最传统的表达感谢的方式，我发现现在已经没有多少人花时间这么做了，这有点悲哀。我知道我们中的许多人都受电子邮件的驱使，并欣赏它的即时性，所以我们倾向于立即发送一封简短的感谢邮件。虽然行动迅速很好，但是没有理由不在行动之后写一封真正的感谢信。手写的便条可以表达你对收件人的感激之情，这种方式他们很可能会欣赏，而其他任何形式都无法表达。

**赠送感谢的礼物**

根据你收到的推荐类型，你可能想送推荐人一份感谢的礼物。只要客户没有接受礼物的问题(如果你与大公司或政府机构合作，这可能是一个问题)，送一份感谢礼物是表达感谢的好方法。可以是一些简单的东西，比如一盒巧克力或一个礼品篮，或者你可以送一些表明你与客户关系的东西，以及你对他们的了解程度——一场体育赛事的门票，甚至是一瓶他们最喜欢的葡萄酒。

别忘了，你也可以向客户最喜欢的慈善机构捐款，而不是直接送礼物给他们。

**拿出来**

如果你和你的客户住在同一个地区，并且通常面对面地和他们打交道，带他们出去吃午饭或晚饭总是一个选择。事实上，如果他们向你推荐的新客户也在同一地区，你可以与双方一起计划一次饭局。这不仅是表达感谢的好方法，也是促进潜在商业发展机会的好方法。

**还人情**

也许对推荐人表示感谢的最好方式就是为你的客户做同样的事情。留心那些能从与你的客户的联系中获益的联系人，准备好并愿意提供推荐。
 **公开表示感谢**

虽然在某些情况下，你肯定希望在你和你的客户之间保留推荐，但有时公开是一个不错的选择。发推文介绍，将客户添加到您网站的合作伙伴页面，向您的电子邮件列表发送消息，或者在您的网站上提供对等链接。这不仅增加了你客户的知名度，也向其他客户展示了你有多重视推荐。

无论你决定如何感谢你的客户的推荐，一定要马上感谢。这绝对是一个你想优先考虑的行动，而不是推迟到以后去做。此外，记住你收到的推荐的类型，并在规模、成本和花费的时间上表达你的感谢。对于不同类型的推荐，有不同的感谢方式是有意义的。
