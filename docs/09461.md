# 不要做第一，要做第二

> 原文：<https://www.sitepoint.com/dont-be-first-be-second/>

我花了很多时间在网上看新产品和服务。我多次注意到类似业务的“我也是”——比如目前涌现出的大量每日交易网站。

你可能会翻白眼，认为“第一个通过邮件”这句老话适用于所有成功的在线投资，但事实并非如此。有几十个例子，成百上千的企业通过改进现有的想法取得了成功。

下个月我可能会去纽约。这时我发现了一个很棒的空房出租网站， [AirBnB](http://www.airbnb.com/) 。这个网站基本上允许介绍有空房间的人和正在旅行的人。我在浏览这个网站，它让我想起了我不久前看到的另一项服务，[沙发冲浪](http://www.couchsurfing.org/)。

事实证明，价值 10 亿美元的 AirBnB 的创始人受到了现在不太为人所知的沙发冲浪的启发，正如一位与该公司关系密切的人安德鲁·帕克在这篇博文中所讨论的那样。AirBnB 需要花钱，沙发冲浪不需要，但 AirBnB 有巨大的兴趣和极高的价值，而沙发冲浪不需要。

我们已经看到，亚马逊(Amazon)最近收购了一家起步较晚、发展迅速的竞争对手——T2 图书仓库(Book Depository)。没有人会相信 iTunes 是第一个音乐销售服务，或者谷歌是第一个搜索引擎。

这就是为什么位居市场第二往往更好的一些重要原因。

节省了研究成本。第一批上市的公司已经把钱花在了需求研究上。这意味着你不必这样做——如果第一个上市的公司获得了成功和知名度，那么你的产品或服务也可能会有需求。

节省营销预算。营销要花钱。如果你正在介绍一个新的想法，它可能会更贵。如果在你之前的公司已经在那里做了最初的消费者教育工作，那么你自己花在教育人们身上的时间就少了。

你从别人的错误中学习。当然，我们都会犯错；然而，从别人的错误中学习代价更低。购买他们的产品，或者订阅他们的服务，然后把它拆开——什么可以用，更重要的是，什么让它更难购买或使用？不要犯那些错误。

改进他们的想法。观察他们如何推销这个概念，并找到更好的方法或更好的定位；即使是细微的差异也会对你的成功产生很大的影响。

相信我，我并不是建议你应该抄袭想法，但是如果你发现自己正在开发一个产品或服务，并且在它上线之前，你听说有人已经抢先了一步，不要沮丧。紧随市场可能是一个伟大的战略。
