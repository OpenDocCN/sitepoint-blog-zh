# 免费图片和图片搜索

> 原文：<https://www.sitepoint.com/picfindr-free-stock-photo-and-image-search/>

*(转载自设计视图#31)*

我们大多数人都需要不时地使用照片库，我认为这是一个随着时间的推移而稳步改善的设计领域。

首先，如果你有非常具体的图像需求和/或时间不足，商业库存公司如 [Corbis](http://pro.corbis.com/) 、 [iStockPhoto](http://www.istockphoto.com/index.php) 和(有时是设计视图赞助商)【Fotolia.com】T4 都有，并且在过去的 3-4 年里变得更便宜、更灵活(许可证方面)和更有用。竞争万岁！

但是最意想不到的发展可能是免费图片资源网站的持续增长。Flickr 的惊人成功似乎有助于巩固与世界分享你的图像的想法，像 [SXC.hu](http://www.sxc.hu/) 、 [MorgueFile](http://www.morguefile.com/) 和 [Image:After](http://www.imageafter.com/) 这样的免费图片分享网站已经从他们的社区中获得了越来越多的优质图片。

![Picfindr.com](img/74e9d7b46973dc44f6ba70595e824068.png)现在，随着[Picfindr.com](http://www.picfindr.com/)的发布，找到免费的股票图片变得更加容易。

你可以将[Picfindr.com](http://www.picfindr.com/)描述为一个免费的股票网站元搜索应用，允许你同时查询几个网站，然后在一个窗口中查看结果。

该服务的开发者里克·恩格勒特(Rick Englert)将其描述为“目前的极简主义者”——介于 alpha 和 beta 之间——但它已经提供了一系列强大的功能，包括智能搜索和允许用户根据他们的许可要求(即作者信用，改变图像或许可先决条件)排除图像。

更高级的搜索选项也在计划中，包括“和/或”选项以及按颜色、大小或方向搜索的能力。

总结:Picfindr 仍然有点粗糙，但它已经是一个方便的免费服务，而且似乎只会变得更好。

很值得一看。
