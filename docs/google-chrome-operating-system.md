# 谷歌 Chrome 操作系统:事实与谬误

> 原文：<https://www.sitepoint.com/google-chrome-operating-system/>

谷歌最近发布的 Chrome OS 引起了媒体的海啸般的猜测。谷歌能在不到一年的时间里开发出可用的操作系统，并在他们自己的游戏中与微软较量吗？事实被观点所压倒，但事实却很少。以下是我们目前所知的总结…

*   谷歌 Chrome 操作系统将于 2010 年下半年发布。
*   由于所有的操作系统都先于网络，谷歌正在重新思考操作系统应该是什么。他们的目标是创建一个没有病毒和恶意软件问题的安全操作系统。
*   操作系统将专注于速度和简单性。它将快速启动，有一个最小的界面，对于那些花大部分时间在网上的人来说是理想的。
*   有传言称英特尔参与其中。最初，Chrome 操作系统将在基于 AMD 和 Atom 的上网本上提供，台式机和笔记本电脑安装将在稍后提供。
*   Chrome OS 硬件供应商包括宏碁、华硕、惠普、联想、高通、德州仪器和东芝。
*   该操作系统将是开源和免费的。

虽然谷歌已经拥有 Android 操作系统，但 Chrome OS 是一个单独的项目。从技术上来说，它将基于一个轻量级的 Linux 内核和一个新的窗口系统*(注意:Linux 使用 X Window 系统。任何图形用户界面都可以构建在上面，所以与 Windows 不同，Linux 用户可以从 GNOME、KDE、Xfce 或任何其他桌面环境中进行选择。谷歌的 Chrome 浏览器将是主要应用，我怀疑谷歌 Gears 将提供离线功能。*

基于云的操作系统并不是一个新想法。Netscape 在 20 世纪 90 年代末提出了这种可能性，尽管它是纯蒸汽软件，但它是微软赢得第一场浏览器战争的主要原因之一。即使在今天，谷歌仍在支持类似的项目，如 Moblin T1 和 T2 Jolicloud T3。

尽管 Chrome OS 的发布有营销炒作的成分，但有充分的理由相信谷歌是认真的。控制浏览器可以让谷歌将用户导向他们的在线应用。GMail、Docs 和 Wave 的注册量可能会增加，从而增加广告收入和用户锁定。

有可能在一年内开发出新的操作系统吗？操作系统开发是艰难的；微软和苹果在这方面已经努力了将近 30 年，但他们仍然会犯错。使用 Linux 可能会有所帮助，但即使是 Chrome 浏览器在该操作系统上也不完整。

那么，假设微软能生产出一种可行的 Windows 替代品，谷歌能与微软一较高下吗？不太可能:

*   虽然上网本市场正在增长，但它仍然很小，微软拥有 90%的份额。
*   基于 Linux 的快速启动上网本从一开始就已经存在，但大多数消费者仍然想要 Windows。
*   上网本在重要的企业部门仍不常见，大多数企业依赖桌面应用程序。
*   尽管用户乐于在网上冲浪、购物和社交，但很少有人在云中做真正的工作。
*   微软能竞争吗？绝对的。Windows 7 已经针对上网本进行了优化，IE8 已经足够好了。该公司可以很快创建一个操作系统的框架版本。

然而，谷歌可以研究几个机会:

*   如果这些设备不贵——可能不到 200 美元——安装 Windows 在商业上就不切实际了。
*   谷歌品牌的机器可能会吸引用户，特别是如果硬件是作为 PDA 或互联网设备而不是迷你笔记本电脑出售的话。触摸屏导航等创新功能也可能增加销量。
*   除了 Windows 之外，还可以提供 Chrome 操作系统。双引导系统将允许用户在启动时选择他们想要的系统。或者，Chrome 操作系统可以安装在可启动的存储卡或 USB 驱动器上。从长远来看，谷歌甚至可以研究基于 BIOS 的轻量级系统的可能性。

Chrome OS 将继续吸引大量媒体猜测，直到其发布*(是的，这包括 SitePoint！即使谷歌做对了，他们也面临着巨大的困难。他们在说服人们更换浏览器方面成效有限；期望他们更换操作系统是一个更加困难的挑战。*
