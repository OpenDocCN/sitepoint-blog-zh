# 设计节:网页设计中网格的介绍

> 原文：<https://www.sitepoint.com/designfestival-an-introduction-to-grids-in-web-design-2/>

由一系列交叉的垂直和水平轴组成的二维结构，用于结构化内容。网格就像一个电枢，设计者可以在上面以一种理性的、易于吸收的方式组织文本和图像。”—维基百科网格是一种无形的结构，它将网页中的所有元素收集在一起。网格从来都不是完全可见的，但是网格“规则”的痕迹可以通过元素在网页中的位置看到。网格还规定了设计元素的大小，如栏文本的宽度、元素的重复放置、图像周围的填充、单词间距、行高等
