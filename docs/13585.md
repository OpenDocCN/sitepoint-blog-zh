# 你的退休生活怎么样？

> 原文:[https://www.sitepoint.com/hows-your-retirement-looking/](https://www.sitepoint.com/hows-your-retirement-looking/)

我的理解是，有很多年轻人访问 Sitepoint，他们都是十几岁或二十多岁的年轻人。

我希望你能把赚到的每一美元都存下 20-50%。

房租和食物似乎很难，但以后只会变得更难。

如果你现在开始储蓄并明智地投资，你可能会比你的同事早退休。

我不会给你讲钱是如何复利的，以及你 20 多岁时存的 1 美元怎么会比你 40 或 50 岁时存的 1 美元更值钱。你知道的。

我们为什么工作？很多原因。但是工作是关于我们如何使用我们的生命能量，以及我们用这些能量做什么。如果你希望有一天能够选择工作与否，请现在就开始攒钱——攒一大笔钱。这需要纪律和承诺，但值得去做。

如果你想学习一种看待金钱的新方法，请阅读经典著作《你的金钱或你的生活》。也读读《自动百万富翁》,这样你就能看到储蓄的强大作用。

我很幸运，因为我很早就存了足够的钱，并在房地产和股票上做了一些明智的投资——还有我自己的生意。希望你也同样“幸运”

花点时间算一算你需要每月存多少钱才能在 55 岁时拥有 200 万美元。然后开始存钱。
