# SXSW 的啤酒书！

> 原文:[https://www.sitepoint.com/books-for-beers-at-sxsw/](https://www.sitepoint.com/books-for-beers-at-sxsw/)

如果你参加下个月在得克萨斯州奥斯丁举办的 [SXSW 互动节](http://2007.sxsw.com/interactive/)，就没有必要为了赢得免费书籍而去跑来跑去[集邮](http://themaninblue.com/writing/perspective/2007/02/22/)(当有派对要进行时，谁有时间去做那件事？)你所要做的就是给 SitePoint 的工作人员买一杯啤酒！

是的，SitePoint 团队的四名代表(包括两名联合创始人)将在 SXSW，坐在小组讨论中，参加破门派对，并在左右和中间分发免费赠品。

我们会有一卡车很酷的东西可以赠送:我们的一些最新的书会被拍卖，还有那些在南方网站很受欢迎的 t 恤。我们所要求的回报是，你请我们喝一杯啤酒，我们就送你一本书(也许还有一件 t 恤)！

这有多简单？

更有甚者，有传言称，SitePoint 即将发行的新书《CSS 的艺术与科学》的一些提前版本将会在出版的那一刻被快递到 SXSW 的*，所以你甚至可以在之前在任何一家商店买到一本*(包括[sitepoint.com](https://www.sitepoint.com/books/)！)**

 *当你在 SXSW 的时候，留意这些在各种[小组](http://2007.sxsw.com/interactive/programming/panels/)发言的 SitePoint 作者和贡献者:

*   **[卡梅隆·亚当斯](https://www.sitepoint.com/blog/)**:JavaScript 选集和即将出版的[CSS 的艺术&科学](https://www.sitepoint.com/books/cssdesign1/)的作者
*   **[伊恩·劳埃德](https://www.sitepoint.com/blog/)**:[的作者使用 HTML & CSS](https://www.sitepoint.com/books/html1/) 以及几篇文章以正确的方式构建自己的网站
*   **[莫莉·霍尔茨施拉格](http://molly.com/)**:[的专家评审员使用 Dreamweaver 8](https://www.sitepoint.com/premium/library) 构建自己的符合标准的网站
*   **[德里克·费瑟斯通](http://www.boxofchocolates.ca/)**:JavaScript 选集[的专家评论员](https://www.sitepoint.com/premium/library)
*   **[丹·鲁宾](http://www.superfluousbanter.org/)** :即将到来的[CSS 的艺术&科学](https://www.sitepoint.com/books/cssdesign1/)专家点评
*   **[乔纳森·史努克](http://snook.ca/)** :即将出版的《CSS 的艺术&科学的作者
*   卡梅隆·摩尔 :几篇关于 sitepoint.com 的文章的作者

此外，以下作者不在小组讨论之列，但也会出席会议，享受庆祝活动，甚至可能赠送他们的签名书:

*   **[【史蒂夫·史密斯】](http://orderedlist.com/)** 和 **[纪娜波顿](http://jinabolton.com/)** :两位作者都即将推出[艺术&科学的 CSS](https://www.sitepoint.com/books/cssdesign1/)
*   **[杰森·比厄德](http://jasongraphix)** :即将出版的[美丽网页设计原理](https://www.sitepoint.com/premium/library)的作者

SitePoint 的工作人员很容易被发现，因为我们会穿着我们的 [SitePoint](http://www.flickr.com/photos/mattymcg/141917858/) [t 恤](http://www.flickr.com/photos/gleddy/256029635/)。所以上来说再见吧！

虽然我不能就这个话题说太多，但我可以透露，可能会有另一场更大的竞争即将到来(针对所有读者，而不仅仅是参加 SXSW 的读者)，这一次将有最棒的奖项可供争夺。

但是你只能等那个了！*
