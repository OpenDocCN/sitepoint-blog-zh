# 你的头在云里吗？使用 Windows Azure 在云中托管 PHP

> 原文:[https://www . site point . com/is-your-head-in-the-cloud-host-PHP-in-the-cloud-with-windows-azure/](https://www.sitepoint.com/is-your-head-in-the-cloud-host-php-in-the-cloud-with-windows-azure/)

今天，我们鼓励你全心全意地把你的脑袋放在云端，通过 Maarten Balliuw 的云计算教程[用 Windows Azure](https://www.sitepoint.com/article/windows-azure-php) 在云中托管 PHP。

马腾首先让所有后来者跟上当前的云计算现象。在详细介绍了一些好处之后，比如云可以提供的大量资源，Maarten 向我们介绍了微软的 Windows Azure。这个 PaaS 云平台基于虚拟机，以可伸缩性和可靠性为目标，只需简单地更改一个配置值，就可以运行多个实例。
![](../Images/1e4790aa6f0433fdda4d7ab306985839.png "Windows Azure")

不需要太多的技术，Maarten 将带你在 Azure 平台上开发 PHP 应用程序。在本教程结束时，您将了解到在云环境中托管 PHP 应用程序是多么简单(并且是可伸缩的)。

我们有一个由微软赞助的[测验](https://www.sitepoint.com/quiz/microsoft/windows-azure-php/)供你试用，所以在你读完[文章](https://www.sitepoint.com/article/windows-azure-php)后，你可以随意发现你学到了多少。
