# 微软从 Internet Explorer 中删除 Trident

> 原文:[https://www . site point . com/Microsoft-drop-trident-from-internet-explorer/](https://www.sitepoint.com/microsoft-drop-trident-from-internet-explorer/)

网络开发者经常抱怨微软流行的浏览器和它的 Trident 渲染引擎进展缓慢。 [Internet Explorer 9.0](https://www.sitepoint.com/microsoft-reveal-ie9/) 可能会提供其他浏览器中可用的功能，但发布日期尚未确定，普遍预计是在 2011 年初。

许多人呼吁该公司将 IE 基于 Webkit 或 Gecko 渲染引擎。不幸的是，这两个系统都造成了实现上的困难，并且最近已经成为黑客和垃圾邮件制造者的目标。微软听取了您的担忧，并于今天发布了以下新闻声明:

> Trident 是全球使用最广泛的布局引擎。它已经为微软和我们的用户服务了十多年。然而，我们认识到我们两年一次的发布时间表对于现代互联网时代来说太慢了。所以 IE9 会是我们浏览器最后一个使用 Trident 的版本。Internet Explorer 10 将基于 Lynx 布局引擎。

行业评论员为微软的大胆决定喝彩。尽管缺乏媒体报道，但 Lynx 提供了几项技术优势:

*   渲染引擎是最快的可用引擎。供应商从未将 Lynx 纳入性能比较图表，因为它以相当大的优势击败了所有竞争对手。
*   Lynx 是迄今为止开发的最安全的浏览器。没有已知的恶意软件感染或跨站点脚本(XSS)攻击的实例。
*   Lynx 提供了最易访问的用户体验，无论有没有鼠标都可以操作。这可能使它成为手机和替代计算设备的理想选择。
*   这台发动机很紧凑。IE 的安装程序的大小将大大减少，这将大幅削减发行成本。
*   Lynx 是跨平台的，因此微软正在考虑为 Windows、Mac OS X、Linux、BeOS 和 OS/2 推出 IE10 版本。
*   它是开源的，比任何其他引擎的开发时间都长。

据微软的开发团队称，Lynx 为未来版本的 Internet Explorer 提供了一个坚实的基础，但前面还有几个技术障碍:

> 微软对商业客户负有义务，而 Lynx 引擎中缺少了许多 IE 广受欢迎的创新。因此，我们将尽早实现基本功能，如 blink 和 marquee 标签。
