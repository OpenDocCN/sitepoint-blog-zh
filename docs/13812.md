# coComment 将博客评论转化为对话

> 原文：<https://www.sitepoint.com/cocomment-turns-blog-comments-into-conversations/>

coComment 是一项即将推出的免费服务，可以追踪你在他人博客上发表的评论。有了它，您可以:

*   在一个地方记录您的所有评论
*   与他人分享您的评论的统一地图(参见[示例](http://www.cocomment.com/blogs)
*   在你自己的博客上重新发表你的评论
*   当人们回复您的评论时获得提醒

这是一个令人兴奋的走向[连通性](https://www.sitepoint.com/web-20-connectedness/)的举动，我之前在这里写过。如果成功，像这样的服务将有效地把网络变成一个巨大的论坛，但拥有个人网络出版的所有自由。

也就是说，我不认为 coComment 有所有的答案。例如，随着这个网络对话平台的发展，什么应该作为评论发布，什么应该发布在你自己的博客上？如果你使用 coComment 来做*和*，你如何将可能产生的两个独立的对话流结合起来？这些是一些有趣的问题，如果他们提出的不受限制的在线对话的灵丹妙药得以实现，这些问题将需要尽快得到回答。

目前，coComment 仍处于[仅限受邀者的测试阶段](http://www.cocomment.com/teamblog/?p=11)(这是我非常想参与的一个阶段)，但你可以注册以获得其即将公开发布的通知。
