# W3C 希望得到您反馈

> 原文：<https://www.sitepoint.com/w3c-redesign-feedback/>

实话实说:很少有人会花太多时间在 W3C 网站上兴奋地谈论最新的指南和规范。文件和设计本身可能最好被描述为*功能性*而不是吸引力。

然而，上个月 W3C 发布了他们网站的新的[测试版，融入了更新鲜、更清晰、更现代的设计。重新设计的目的是:](http://beta.w3.org/)

*   更容易找到您需要的信息
*   创建一致的页面导航
*   遵循其他网站使用的惯例
*   使以前隐藏的内容更容易找到
*   提供聚合的社区信息
*   改善移动设备上的渲染(火车旅行再也不会无聊了！)
*   确保页面符合 WCAG 2.0 准则
*   改善维护，以及
*   采用 RSS 等技术。

最好不要期望有令人眼花缭乱的图形和动画窗口，但是新的设计无疑是对他们日益过时的网站的一个改进。

你喜欢吗？它会说服你每天急切地访问网站吗？你讨厌吗？蓝灰色的设计会让你生病吗？W3C 正在寻求 web 开发社区对网站的设计、可用性和其他方面的反馈。
