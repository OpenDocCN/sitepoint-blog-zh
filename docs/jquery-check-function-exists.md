# jQuery 检查函数是否存在示例

> 原文：<https://www.sitepoint.com/jquery-check-function-exists/>

用于检查 JavaScript 代码中是否存在函数的 jQuery 代码片段。这可以通过使用 jQuery.isFunction()函数轻松实现。用于在调用 jQuery 函数之前检查它是否存在！

**注意:**从 jQuery 1.3 开始，浏览器提供的函数像 alert()和 DOM 元素方法像 getAttribute()并不能保证在 Internet Explorer 等浏览器中被检测为函数。

延伸阅读:[http://api.jquery.com/jQuery.isFunction/](http://api.jquery.com/jQuery.isFunction/)
