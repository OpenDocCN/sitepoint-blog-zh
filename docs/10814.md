# 网络工作者:你使用哪种操作系统？

> 原文:[https://www.sitepoint.com/what-os-do-you-use/](https://www.sitepoint.com/what-os-do-you-use/)

web 开发的吸引力之一是平台独立性。任何人都可以在任何操作系统上使用任何浏览器访问和查看您的网页。好吧——事情从来没有那么简单，但我们可以梦想！)

不像 IT 界的许多人，网页设计者和开发者有更多的选择自由，可以选择他们喜欢的操作系统。有一些注意事项——例如，Flash 和 Silverlight 开发人员需要适当的 Adobe 或 Microsoft 工具——但是标准 web 技术可以在使用任何操作系统的任何设备上开发。

让我们看看三个最受欢迎的选项。

## 微软视窗软件

Windows 是使用最广泛的操作系统，安装在 90%以上的个人电脑上。这显然是 ASP.NET 开发商的首选系统。(可以在另一个操作系统上编写代码，并部署到 Windows 服务器上，但这并不特别实用。)

Windows 提供了选择，它是我们拥有的最接近标准的东西。操作系统是稳定的，你不太可能遇到硬件支持问题，有大量的免费和商业软件，版本之间的兼容性非常好。

缺点是什么？…

*   与运行它的硬件相比，Windows 是昂贵的。你可以花 200 美元建造一台个人电脑，但需要花同样的钱在操作系统上。
*   这款操作系统以不安全著称。如今这已经不是什么问题了，但是 Windows 仍然是黑客和垃圾邮件制造者最明显的目标。
*   最后是最不刺激的 OS。大家对 Windows 都很熟悉；很少有人会对你最新的电脑印象深刻。

## 苹果 Mac 电脑

苹果电脑占据了大约 7%的操作系统市场。它以作为图形设计者操作系统而闻名，但是更多的开发者正在使用它。苹果提供辉煌的硬件和软件；您将获得稳定、安全和连贯的体验，而不会遇到驱动程序或配置问题。

并非一切都是完美的…

*   软件选择比其他操作系统更有限。苹果主导市场；你常常不得不按照他们的方式做事，或者干脆不做。
*   苹果很少争取 Mac OS 版本之间的兼容性。
*   苹果硬件和软件的结合通常会使个人电脑比竞争对手更贵。

## Linux 操作系统

Linux 有各种各样的风格，大约占操作系统市场的 1%。

也许最大的优势是成本——Linux 是开源的，你可以在任何地方安装它。它是高度可配置的，安全性非常好，并且提供了广泛的免费软件。Linux 和 Apache HTTP server 主导着 web，所以它是开发人员的合理选择。

问题是…

*   Linux 被认为是极客的操作系统。这有点不公平:许多发行版的安装过程比 Windows 更容易。然而，那些 IT 知识有限的人可能会发现诊断和修复问题很棘手。
*   硬件支持更受限制，制造商并不总是创建 Linux 驱动程序。
*   很少有 PC 厂商提供 Linux 作为操作系统选项。你通常需要依靠自己的技能和社区的支持。

## 你用什么？

Windows vs Mac vs Linux 的争论已经持续了多年。每一种都有其长处和短处——没有一种是完美的。

但这不是关于什么是最好的:我想知道你用的是什么操作系统。你可能厌恶这个系统，但是有实际的理由使用它。
