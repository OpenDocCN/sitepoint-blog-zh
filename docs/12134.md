# 利用特别促销来拓展业务

> 原文:[https://www.sitepoint.com/business-promotions/](https://www.sitepoint.com/business-promotions/)

回头客是最好的客户，因为你了解他们，他们也了解你；你们一起有过记录；你喜欢为他们提供服务，他们也喜欢为这些服务付钱给你。但有时，尤其是在经济衰退时期，可能很难让现有客户购买新产品和服务。

出于多种原因，我不支持一刀切的降价或降价——这对提供商不利；很难从中恢复过来；这会在工作关系中产生怨恨。但是有时候，对现有客户的折扣、赠品和特价可以帮助鼓励他们购买更多。

为你的客户提供这种额外激励的另一个大好处是，它对建立客户欣赏度大有帮助。这会让你的客户觉得自己更有价值，并能加强你与他们的关系。

以下是一些你可以利用特别促销从现有客户那里获得额外业务的方法。

**捆绑包**

向客户提供的一种很好的折扣形式是捆绑销售。捆绑包鼓励客户注册两个或更多的产品或服务，而他们最初可能只考虑其中一个。你得到了巨大的好处，因为你得到了额外的工作，客户购买捆绑产品和服务得到了折扣。

除了捆绑两个或更多自己的产品和服务，您还可以将自己的产品和服务与第三方产品或服务捆绑在一起。这使得你可以与另一家公司建立合作关系，后者可能会反过来为你捆绑销售你的一些服务。这也为你提供了另一种方式来满足更多客户的需求并增进你们的关系。

**赠品和赠品**

您可以每月进行一次抽奖，获得少量赠品，或者为现有客户提供一次免费的小服务(或一小时的服务)。赠品的目的是让客户体验你试图销售的产品或服务，以便让他们购买整个产品或服务。这是一个帮助客户在投资前确认购买资格的测试。

**限时特价**

在有限的时间内给予产品或服务一定的折扣有助于客户更快地参与进来并购买更多。关键是让它成为一个非常有吸引力的提议，给客户足够的时间来决定这是一个好的选择。

**买一送一**

很明显，这种方法并不适用于你提供的所有东西，但是它可以用于许多产品和一些你按小时计费的服务。你可以为购买的用户提供一个月的免费虚拟主机服务，或者为购买的用户提供一个小时的免费网站维护服务。

使用特别促销的关键是确保你要推销的产品或服务是客户所需要的，并且长期利益大于任何短期损失。
