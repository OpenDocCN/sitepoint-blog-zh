最近我也遇到了这种情况。当用户通过 Google 访问网站时，我需要一种方法来突出显示 HTML 块中的关键字。这样，你可以帮助你的用户快速找到他们在谷歌搜索的信息。

然而，一个简单的字符串。替换功能不能解决问题。很明显，它也将取代 HTML 标记中的任何关键词，因此会删除链接或图像。

例如，以关键字“sitepoint”为例，我希望用一些 HTML“site point”来替换它。如果我的图像名称中有“sitepoint”，我会以![](../Images/c54bbdb18cb818841397f1b8c5684e10.png) sitepoint.jpg" >结束。不是我想要的。
