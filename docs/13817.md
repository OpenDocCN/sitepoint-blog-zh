# Java Studio Creator 2 已经发布，并且仍然免费

> 原文：<https://www.sitepoint.com/java-studio-creator-2-released-and-still-free/>

当我第一次宣布 Sun 将提供 Java Studio Creator 产品供免费下载时，一些怀疑者认为这仅仅是 Creator 2 发布前的宣传噱头。

嗯， [Java Studio Creator 2 现在在这里](http://developers.sun.com/prodtech/javatools/jscreator/)，而且还是免费的。

Java Studio Creator 基于 [JavaServer Faces](http://java.sun.com/j2ee/javaserverfaces/) (JSF)技术，这是一种基于组件的 web 开发方法，风格类似于 ASP.NET。这将 Java Studio Creator 定位为微软 [Visual Studio 2005 速成版](https://www.sitepoint.com/visual-studio-2005-express-editions-are-free-downloads/)的直接竞争对手。两者都将拖放式快速应用程序开发带到了 web 上，并且都是免费下载的，但是 Creator 没有微软 Express Editions 上的许多限制(例如，它提供了一个完整的可视化数据库查询生成器)。

当然，Creator 是用 Java 编写的，可以在 Mac OS X 和 Linux 系统上运行。

一些新功能的简要概述:

*   改进了代码编辑器(Java、HTML、CSS)中的代码完成
*   一个新的图形 CSS 编辑器
*   EJB 支持(在创造者 1 中是测试版吗)
*   许多新的 JSF 组件
*   一个新的数据库支持层
*   HTTP 监控
*   AJAX 组件(包括一个谷歌地图组件)
*   在 Java 5 上运行，具有相关的速度和 UI 优势

如果您过去考虑过 Java web 开发，并且对代码繁重和晦涩的 XML 配置文件感到厌烦，那么您绝对应该查看 Creator 2 的[视频产品教程](http://developers.sun.com/prodtech/javatools/jscreator/overview/tours/),看看这有多简单。
