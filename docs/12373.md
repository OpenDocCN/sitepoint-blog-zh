# 新版本:使用 HTML 和 CSS 以正确的方式构建自己的网站

> 原文：<https://www.sitepoint.com/new-release-build-your-own-web-site-the-right-way-using-html-css/>

使用 HTML & CSS 的正确方法建立自己的网站，被广泛认为是建立网站的权威初学者指南。在第一版的数百条评论和建议之后，我们非常高兴地宣布伊恩·劳埃德畅销书的第二版。

伊恩利用他对网络开发前沿进展的理解来增强这本书，所以它兼容所有的网络浏览器(包括我们所知道的 IE8 ),并融入了谷歌分析等新工具。

**简而言之，终极初学者指南刚刚变得更好！**

那么，书里有什么？

如果你是网页设计的新手，但你对学习建立网站充满热情，那么[使用 HTML & CSS，第二版](https://www.sitepoint.com/books/html2/)以正确的方式建立你自己的网站是必须的。在这本简单易懂的指南中，你将学会如何以最好的方式建立一个网站——自己动手！

Ian 会帮助你，但是通过一步一步的建立一个示例网站，你会发现培训的车轮在你意识到之前就已经停止了。您将从零开始，学习入门所需的基本工具(文本编辑器、web 浏览器和 FTP 客户端)，到本书结束时，您将准备好为功能齐全的网站添加一些真正的特色。

通过这本书，您将了解如何实现对网站开发至关重要的功能，例如:

*   易于使用的导航
*   专业外观的标题
*   定期更新的新闻/事件页面
*   联系我们页面
*   一个简单的图库
*   一个涵盖你的网站以及相关网站的搜索引擎
*   还有更多…

关于作者…

Ian 是英国的一名高级网页设计师/开发者，他已经撰写或参与撰写了许多网页开发书籍，包括 SitePoint 的[终极 HTML 参考](https://www.sitepoint.com/books/htmlref1/)。他目前在 accessibility 运营(他于 2002 年创办的一个网络无障碍网站)，并且是诸如西南偏南(South By Southwest)等网络开发会议的常客。
