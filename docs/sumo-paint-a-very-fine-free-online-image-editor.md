# 相扑油漆:一个非常好的，免费的在线图像编辑器

> 原文：<https://www.sitepoint.com/sumo-paint-a-very-fine-free-online-image-editor/>

Sumo Paint 是一个免费的基于网络的图像编辑器，具有一些令人印象深刻的功能。我不会把它描述成 Photoshop 的竞争对手，甚至是一个严肃的替代品，因为它不是。然而，它是一个优秀的小图形编辑器，对于没有或不想使用 Photoshop 的网页设计师或开发人员非常有用。

相扑现在是 1.1 版本，使用 Flash 10 播放器，并且每次迭代都在改进。界面看起来很像 Photoshop。它的右侧有调色板，浮动工具箱和菜单，有一些和 PS 一样的功能和命令。Sumo 支持层和层效果，图像调整，如亮度，色彩平衡和水平。锐化，模糊以及其他几个过滤器也在这里，所以它可以做一些很好的照片编辑以及创建新的图形。

如果你以前用过 Photoshop，那么相扑工具箱里有很多熟悉的面孔。魔术棒，套索，克隆，铅笔，油漆桶，渐变和橡皮擦工具都在这里，以及许多形状工具。曲线工具的工作方式与钢笔工具有些类似，让您可以轻松地创建平滑的曲线和拐角。对称工具玩起来很有趣，对制作图案很有帮助。

[![sumo_screengrab](img/295872aff2df7bff3daf7f3fc38ecf18.png "sumo_screengrab")](https://www.sitepoint.com/wp-content/uploads/2009/06/sumo_screengrab.png)

缺点是，它只能打开 JPEG、gif 和 png 格式的文件，而且你只能将你的作品保存为 JPEG 或 png 格式。如果你想保存你创建的任何分层图像，你需要注册一个免费账户。话虽如此，如果你正在创建网页图形，或做一些简单的照片编辑，那么这不是什么大问题。你也可以和社区分享你的作品，相扑已经有很多粉丝了。

因此，如果你发现别人的电脑上只有颜料，或者如果你只是想免费试用一个好的图形编辑器，Sumo 非常值得一游。它轻而易举地击败了在线 Photoshop Express。它是免费的，快速的，不需要注册。
