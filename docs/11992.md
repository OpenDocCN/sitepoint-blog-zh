# 7 个免费预约调度系统

> 原文:[https://www.sitepoint.com/free-appointment-schedulers/](https://www.sitepoint.com/free-appointment-schedulers/)

这篇文章写于 2009 年，至今仍是我们最受欢迎的帖子之一。如果你渴望了解更多关于日程安排和计划的知识，你可能会对这篇关于实时跟踪你的时间的文章很感兴趣。

上周，我写了一篇关于[在线日程安排系统](https://www.sitepoint.com/online-scheduling-systems/)以及它们如何帮助你简化与客户和同事安排会议的流程的帖子。我提供了 7 个可用工具的概述，其中大部分都需要每月付费。

我收到了一些要求免费选项的评论和电子邮件，所以在这篇文章中，我将讨论几个免费选项，包括在线和可下载软件，以及您可以从每个选项中期待的一些功能。

**1。[![acuity](../Images/7f749233fffb6597abf077683e7bdae1.png "acuity")≥调度](http://www.acuityscheduling.com/)**

*   在线预订
*   网站整合
*   用户化
*   无限客户
*   无限约会

Acuity Scheduling 提供免费的单用户版本，但它不包括个性化预订网站或接受信用卡的能力。

**2。 [![appointmentplugin](../Images/4bee15cecdb3d7106cccdd6472f3bc44.png "appointmentplugin")约会插件](http://www.appointmentplugin.com/)**

*   在线预订
*   可用的定制选项
*   电子邮件确认
*   网站整合

该工具提供了一个有限的免费版本，每天有 20 个约会的限制，并在电子邮件确认中做广告。

**3。 [![createappointments](../Images/e1f1592f6e85422ed20434ebb57e7d94.png "createappointments")创建约会](http://www.createappointments.com/)**

*   在线预订
*   电子邮件确认
*   创建客户帐户

CreateAppointments 的免费基础版在在线日程安排的底部有一个广告，不包括定制。

**4。 [![genbook](../Images/e097e5b65e7a44069ac2768ad19131c9.png "genbook") Genbook](http://www.genbook.com/)** 

*   在线预订
*   立即预订网站按钮
*   个性化网站
*   无限员工
*   无限服务

Genbook Free 将未来日程安排限制为四周。他们的付费版本在我之前的帖子里提到过。

**5。 [![memotxt](../Images/2d929a5dbaefca4153de2950adb93239.png "memotxt") MEMOTXT 调度器](http://www.memo-txt.com/free_appointment_scheduler/)**

*   免费下载的软件
*   搜索功能
*   保留客户联系信息
*   发送短信约会提醒

MEMOTXT 目前适用于所有微软平台，但 Linux 和 Mac 版本正在开发中。

**6。 [![pimero](../Images/5a234e12d7fe7ee7e1d4df0ea0930860.png "pimero")皮梅罗免费版](http://www.freedownloadscenter.com/Business/Personal_Info_Managers/Pimero_Free_Edition.html)**

*   免费下载的软件
*   包括约会，任务列表规划和联系人管理器
*   同步多台电脑
*   保护并自动加密存储的数据
*   Outlook 导入

Pimero 免费版是一个功能丰富的可下载选项，用于标准化您的日程安排过程。

**7。 [![supersaas](../Images/e71eee3ab03766eb53c327c908d0c955.png "supersaas")苏萨](http://www.supersaas.com)**

*   在线预订
*   可用的定制选项
*   电子邮件集成
*   网站整合
*   客户管理

免费的 SuperSaaS 计划包括最多 50 个注册用户，并有广告。

当然有一些可靠的自由调度系统可用，尽管功能有些有限。正确的解决方案确实取决于您的个人需求。

上面列出的两个选项是免费软件，您可以下载并在本地使用。这些主要用于内部日程安排，因为它们不包括在线预订功能。
