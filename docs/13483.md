# 客户走了

> 原文：<https://www.sitepoint.com/pop-goes-the-client/>

在过去的一个月左右的时间里，我一直在与我们销售渠道中的两位特别优秀的潜在客户合作。这两个潜在客户都在我们的目标市场上，需要大量有利可图的服务，所以我们一直在努力争取与他们签约。

我们和两个客户进行了通常的电子邮件、电话和问题交流，但最终我们签下了其中一个客户，并决定放弃另一个。简而言之，随着谈判的展开，其中一个客户开始变得越来越没有吸引力，而另一个客户则被证明在几乎所有方面都很容易相处。

“客户 A”和“客户 B”都是以强劲的前景开始的——拥有良好的信用和大量工作的成熟的大公司。然而，当我们第一次寄出合同时，差异变得很明显。

客户 A 要求对合同进行一些小的常规修改，主要是关于仲裁规则和一些保密条款。这些变化非常合理，所以我们很乐意服从。与此同时，客户 B 接受了合同的所有条款，但突然要求降低时薪！这真是令人震惊，因为两周前客户对完全相同的利率做出了积极的反应。这不是一个好兆头，我们恭敬地拒绝了降价。

客户 A 然后问我们是否可以从我们的印度团队中给他们指派一名专门的 QA 经理。我们解释说这不符合我们的资源模型，但客户 A 很坚持，我们最终同意他们有一个专门的 QA 经理，前提是他们保持最低的每月 QA 支出。这对每个人来说都是一个双赢的结果，我们打电话讨论剩下的问题。客户 A 始终保持着尊重和积极的态度，我们很快就在两家公司之间签订了合同。

要是客户 B 有这么好的态度就好了。他们对我们不愿意降低费率感到沮丧，尽管事实上我们提出了从托管到更优惠的付款条件的一切。更糟糕的是，我们在客户 B 的主要联系人突然把我们交给了公司总裁，他解释说我们的费率“太高了”。很明显，他的意图是利用他的管理技能来谈判这笔交易，但它真正做的只是向我展示了他是多么粗鲁和无礼。客户 B 的价值是客户 A 的两倍，但我开始对他们感到恼火，我们甚至还没有开始一起工作——这不是一个好现象。

这时，我们与客户 B 的关系显然不会再好了:客户 B 的总裁告诉我，如果我们的设计作品像他以前的供应商一样“跃然纸上”，他“可能会考虑”我们的价格。当我听到客户说他们需要它“砰”的一声，我通常会跑出门去。为了好玩，我问他那个供应商怎么了，他说他们不再合适了。我敢打赌，他今年已经经历了许多供应商！

我恭敬地解释说，我们可能也不“合适”，并感谢他抽出时间。真是松了一口气！如果说这里有什么教训的话(除了避免客户说‘pop’)，那就是你付出的要得到回报，好的生意是建立在良好的、相互尊重的关系基础上的。
