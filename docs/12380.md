# 备份您的在线生活

> 原文:[https://www.sitepoint.com/backing-up-your-online-life/](https://www.sitepoint.com/backing-up-your-online-life/)

定期备份你电脑上的数据是一件聪明的事，尽管这可能会很乏味。每周刻录一次 CD 或 DVD，或者将文件移动到外部硬盘驱动器(或者您计划备份数据的频率)是一个非常耗时的过程。这就是为什么许多人寻求自动备份解决方案，无论是本地还是在线。

最近，我实际上一直在市场上寻找一种在线备份解决方案，这种解决方案将在我的计算机上自动备份新文件，并在旧文件发生变化时捕获新版本。不过，有一件事我没有考虑备份，那就是我网络生活中的点点滴滴。

随着我们在社交网络和媒体共享网站上花费越来越多的时间，构成我们身份的点点滴滴变得越来越分散。备份数据比手动管理硬盘上本地数据的备份更加困难。许多服务提供了获取数据的方法，但它们并不总是很容易，并且处理多个 API 和不同的数据流以访问数据，然后备份数据超出了许多用户的技术专长。

[Lifestream 博客最近将](http://lifestreamblog.com/launch-of-the-first-lifestreaming-backup-service/)指向一项名为 [LifeStreamBackup](http://lifestreambackup.com/) 的新服务，该服务有望在几周内推出后，让备份数字生活的过程变得更容易。最初，该网站将只为 Flickr 和你的博客(通过 RSS)提供备份服务，但计划在推出后不久增加对谷歌文档、Twitter、脸书和 YouTube 的支持。

该服务将充当你的数据在你的网络服务平台和亚马逊的 S3 之间传输的管道，备份将存储在那里。10GB 的价格将从 6.95 美元/月开始，或者 3.95 美元/月，以挂钩到您自己的 S3 帐户(如果您需要更少的空间，这可能会为您省钱)。

我们预计，备份您在线生活的服务将是明年的热门领域，而且我们预计，创建 web 服务数据备份的实用程序将内置于现有的桌面到 web 备份服务中。随着越来越多的人开始通过软件即服务应用程序(如 Google Docs 和 Twitter 等社交网站)将数据推送到云中，保留这些数据的备份对人们来说将变得更加重要。

无论对错，将数据放在网上*比将数据保存在本地硬盘上*更不持久。当你将数据放入在线服务时，你不仅要担心由于硬盘故障等物理问题导致的数据丢失，还要担心服务可能会停业或关闭。甚至[谷歌最近也在关闭服务](http://searchengineland.com/google-ends-google-video-uploads-shutters-notebook-catalog-search-dodgeball-jaiku-16166)，所以备份这些数据以防范未来服务的关闭或变化是很有意义的。

如果像 LifeStreamBackup 这样的 web 数据备份服务也能清理所有数据，然后提供备份给希望访问这些数据的用户下载，以便轻松地在本地备份这些数据，那就太棒了。否则，所有这些服务实际上是让你可以轻松地将数据从一个云转移到另一个云，但仍然不能让数据更容易被本地机器访问。
