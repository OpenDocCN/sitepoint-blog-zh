# 一份好的自我描述如何帮助你的生意

> 原文：<https://www.sitepoint.com/how-a-good-self-description-can-help-your-business/>

你是自由职业者吗？承包商？顾问？租来的枪？一个企业主？个体户？

你描述自己工作风格的方式可能对潜在客户、客户和同事如何看待你有很大影响。它会影响你得到的工作类型，以及你发现收取你喜欢的费用有多容易。也会影响你的自我认知和自信。

如果你还没有考虑这个问题，你应该考虑一下。虽然积极和不那么积极的含义与你可以应用于你的工作风格的每个描述符相关联，但仔细思考如何解释你所做的事情可以帮助你重新关注你想和谁一起工作，你将如何运作等等。

## 事实

试图让你的行动听起来不像是真的是没有意义的。如果你不能实现对自己工作风格的描述，这只会导致失望，并可能损害你的自尊和声誉。

如果你是一个个体经营者，把自己描述成代理或咨询公司可能会给潜在客户传达错误的信息。例如，如果你不能同时管理多个项目，他们可能会感到惊讶。

同样，如果你在繁忙的时候把工作外包给其他人，考虑用一种不暗示你是这个行业中唯一的人的方式来描述你的运作。如果客户认为他们在每个项目上都得到了你的个人意见，他们可能会震惊地发现真相。

首先，想想你的工作方式，把你的描述缩小到一个能准确描述你所做的事情的术语集合。

## 感知的问题

好吧，所以你知道你在做什么。但是你想和谁一起工作呢？你想要塑造的是你的客户和潜在客户的看法。一些客户可能会被“承包商”这个术语吓跑，认为你像一头受伤的公牛一样收费。其他人可能会觉得“自由职业者”听起来像是你在你的车库里兼职操作。

你的客户的动机可能会影响他们对不同描述的理解。想象一个大型组织，它用“伙伴关系”、“建议”和“关系”来描述它与第三方组织的合作。与“自由职业者”等词相比，“咨询”或“顾问”等词意味着一种强有力的、持续的安排，而“自由职业者”等词可能意味着一种根据需要而运作的时断时续的安排，而不是一种持续的提议。

相反，如果你的目标是小企业和个体经营者，他们可能会更愿意与自称是自由职业者或小企业主的人合作。将你的工作描述为一项业务可能有助于创造一种额外程度的融洽，一个共同点的领域，以及一种相互理解的感觉。

看看你刚才列出的单词表，找出你认为最吸引你的潜在客户的单词。也许你会考虑让他们通过一些相同职位、领域或行业的可信客户，以确保你是在赚钱。

## 原则问题

最后，想想你单子上的哪些词让你最舒服。例如，如果你认为“专家”听起来很老套，那就放弃它。选择一个描述是没有意义的——即使你认为它会吸引你的观众——如果下次你遇到一个问你做什么的新联系人时，你不能大声而自豪地说出来。

也想想你的小企业或自由职业者的发展方向。你对未来世界的梦想(或者至少是工业！支配可能会把你的选择引向某个方向。你可能正在努力从你目前的肇事逃逸的自由职业者工作中建立长期的、持续的客户关系。在这种情况下，现在称自己为“雇佣枪手”可能是准确的，但这并不能反映出你想去哪里，而且——我们希望——不会太久。

考虑你目前的自我认知和你未来的目标，选择一个能准确反映你现在和未来中期能为客户提供什么的描述。

我认为这些是自由职业者在设计如何描述他们的工作时需要考虑的因素。你叫自己什么？为什么呢？
