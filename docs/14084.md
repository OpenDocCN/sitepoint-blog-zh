# 一些 IDE 新闻

> 原文：<https://www.sitepoint.com/some-ide-news/>

看起来又到了集成开发环境(IDE)的季节了…

**[月蚀](http://www.eclipse.org/)** 3.1 即将推出，第一个发布候选版本(RC1)现在可供[下载](http://download.eclipse.org/eclipse/downloads/drops/S-3.1RC1-200505271300/index.php)。对 Java 5.0 的支持是我最喜欢的，尽管对 Ant 1.6 的支持也不错。也许是时候让我试一试了…

**[NetBeans](http://www.netbeans.org/)** 4.1 现已发布，并从孙那里得到一些必要的发布后的消息。这个免费的 IDE 构成了 Sun 的 Java Studio Creator 和 Java Studio Enterprise IDEs 的基础，现在它通过支持 J2EE 1.4 为 Web 开发提供了自己的支持，并为支持 MIDP 的设备提供了移动开发支持。NetBeans 是 Sun 在 5 月份主持的一次实时聊天的主题。《T4》的文字稿现在可供更随意的阅读。

JBuilder 继续宣传它使用 Eclipse 作为未来版本的基础。在刚刚宣布的[产品路线图](http://bdn.borland.com/article/0,1410,33079,00.html)中，Borland 宣布 JBuilder 2006 将在年底前发布，并具有扩展的团队开发特性，比如共享代码视图和调试会话。大肆宣传的 JBuilder“黯然失色”版本将于明年发布，代号为“Peloton”。

**[IntelliJ IDEA](http://www.jetbrains.com/idea/)** 也没有停滞不前。JetBrains 是[提供](http://www.jetbrains.com/allnews.html#idea5promo)免费升级到 5.0 版本，当它被发布给任何人谁将在 5 月 1 日之后购买 4.5 版本。和往常一样，IntelliJ IDEA 的早期版本在发布之前都是免费的。
