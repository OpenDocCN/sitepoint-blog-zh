# 使用 Dreamweaver 8 挥汗如雨

> 原文:[https://www . site point . com/working-up-a-sweat-with-Dreamweaver/](https://www.sitepoint.com/working-up-a-sweat-with-dreamweaver/)

与应用程序发展关系通常是一个缓慢的过程，而对于真正好的软件，它可能永远不会真正结束。

几个月前，当我第一次打开 Dreamweaver 8 的盒子时，我想到的都是“*,它有什么不同？*和*哎呀，我希望他们没有打破它*。我可能严重缺乏想象力，但在这个阶段，我总是觉得旧版本非常好，当然他们只能从这里破坏它？有时候我是对的，但是这个从一开始就很好。

虽然它在大多数功能列表中几乎被忽略了，但“后台 FTP”是给我印象最深的功能。在等待上传的过程中，我的键盘上有三个闪亮的指尖大小的凹痕。

不久之后，我校对了雷切尔的 Dreamweaver 8“绝活”(当时仍是 Word 文档)的最终编辑，并开始更加认真地欣赏版本 8 中的改进。值得庆幸的是，Macromedia 将大部分开发注意力放在了最需要它的领域——即以 CSS 为中心的设计工具、标准和可访问性，同时加入了一些不错的代码视图改进。

但是 Dreamweaver 实验室的所有工作都是埋头苦干和提高生产力吗？

当然不是！Macromedia 男孩知道如何找乐子。

如果你有 Dreamweaver，并且你在周一早上感觉有点疯狂，试试这个。

1.  打开任何 HTML 页面
2.  单击身体内的任意位置
3.  转到属性面板，找到颜色文本输入框
4.  输入“dreamweaver”并按回车键
5.  准备好用你的鼠标驾驶；)
