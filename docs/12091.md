# 哎哟！被拒绝的痛苦

> 原文：<https://www.sitepoint.com/ouch-the-pain-of-rejection/>

被拒绝很伤人。不幸的是，对于自由职业者和企业主来说，这是常有的事，尤其是对于那些在创意世界的人。我们被潜在客户拒绝；我们的设计想法被拒绝；我们被拒绝新的商业机会。你有没有想过你是一个新客户的忠实客户，或者创造了“完美”的设计却被拒绝？是的，我也是。

即使你预料到了被拒绝，这种感觉也不会很好。人性让我们想要成功，想要被接受，而被拒绝从来都不是我们所期待或者快乐接受的事情。拒绝之所以令人不快，是因为它打乱了我们精心制定的计划，剥夺了我们渴望已久的新机会，并对我们的自尊心造成了沉重打击，但它确实让我们在工作中做得更好，更聪明，更强大。方法如下:

你学到了一些东西。

其实你可能会学到很多东西。被拒绝可以教会你一些关于你自己，你的工作，你的能力，你有效沟通的能力，你需要改进的地方，以及你如何应对批评。如果是基于项目的拒绝，你也可以学到很多关于客户和他们想要什么的东西。这些信息是设计过程中至关重要的一部分，它有助于这样想，而不是作为一个个人的拒绝。

最好的拒绝方式包括建设性的批评和反馈，这将有助于你下次做得更好。每一个潜在的客户关系和未赢得的提议都是推销你自己和你能力的有价值的实践。你越是推销自己，无论你是否赢得合同，你就会做得越好，你就会越有信心，随着时间的推移，你就会取得越多的成功。

它能让我们正确看待事物。

大多数拒绝都不是针对个人的。最重要的是，拒绝并不能定义你是谁，或者你在你所做的事情上有多成功。拒绝仅仅意味着此时你不适合某个特定的机会。从大的方面来看，被拒绝并不意味着什么，它会让你有更多的机会。就像俗话说的，“当一扇门关闭时，另一扇门就会打开。”

这会让你更饿。

被拒绝给了你一个重新开始的机会，让你更有决心在下一次获得成功。被告知“不”会让我们更想听到“是”，这意味着我们可能会更加努力工作，更加关注细节，以便尽可能地将自己置于最佳位置，避免再次被拒绝。更不用说，在奋斗之后的成功会更加甜蜜，而拒绝会让胜利更受赞赏。

当你面临拒绝时，你的一线希望是什么？
