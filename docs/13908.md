# 爆还是不爆

> 原文：<https://www.sitepoint.com/to-pop-or-not-to-pop/>

在每个网站发布者的一生中，都会有这样一个时刻，他不得不问自己一个问题，“我该不该使用弹出窗口？”

从这个问题的正面来看，流行音乐有很多优势。它们是一种不占用任何屏幕空间就能让你的网站赚钱的方式。他们当然也比普通横幅支付得多。对于一般的受众网站来说，如今来自广告网络的排行榜的平均费用不到 1 美元每分钟，但是对于大众网站来说，通常在 4 美元每分钟左右。就我个人而言，我在一些网站上使用弹出窗口，如果我不这样做，我一个月会损失大约 5000 美元，这对我来说是一笔太大的损失。弹出式广告也致力于引起人们对某些东西的注意。例如，SitePoint 使用弹出窗口为自己的时事通讯和产品做广告，这是吸引用户注意力的好方法。几年前，我受到 SitePoint 的启发，使用弹出窗口为我的一份时事通讯做广告，订阅率增加了两倍多。

运行弹出窗口的缺点也很多。人们不喜欢他们，因为他们可能很烦人。一些广告公司或广告商肆无忌惮，可能会尝试在他们的广告中放置自动广告软件下载。如果你的弹出窗口惹恼了目录编辑或其他考虑链接你的人，你可能会失去那个链接。你也不允许用 Google AdWords 在任何有弹出窗口的页面上做广告。

此外，有些人认为，如果你运行弹出窗口，你实际上会失去游客，如果你不运行弹出窗口，你的流量会增加(因此你的横幅广告收入会增加)，这样你最终会比你仍然运行弹出窗口赚得更多。正如在[这个帖子](https://www.sitepoint.com/forums/showthread.php?t=304329)中讨论的那样，大多数人都不是这样。可能是因为那些非常讨厌弹出窗口的人可能已经安装了阻止软件，或者可能只是因为如果人们需要你的网站提供的信息，他们仍然愿意和弹出窗口打交道。

当然，所有这些都有一个中间地带。一般来说，每个用户会话只运行一个弹出窗口是可以接受的，大多数主流广告网络都允许你(有时强迫你)这样做。用户不会在每次页面浏览时都看到弹出窗口，而是每次会话只有一个。如果你正在使用弹出窗口来推广你自己的产品或服务(或一个附属程序)，你可以用一点简单的 javascript 来限制自己。

不过，你仍然需要担心目录编辑，如果你试图在 DMOZ 或雅虎的目录中使用免费提交弹出窗口，可能会损害你的机会。在某种程度上，在 DMOZ 发现的持续的编辑偏见有点不公平，因为他们不提供商业网站的付费提交程序，但在任何情况下，当提交时，你真的处于编辑的怜悯之下，你会想尽一切办法让他们尽可能愉快地访问你的网站。

那你是做什么的？偷偷摸摸。我制作了自己的小广告服务脚本，使用跟踪用户会话的 cookies 来决定何时从哪个网络提供弹出广告，以及何时从各个网络提供横幅广告。在这个脚本中，我有一小段代码来检查访问者的第一次页面浏览的推荐人。如果推荐人与我正在寻找的匹配，我会关闭该访问者的广告。所以理想的目录编辑应该有一个很好的免费广告访问我的网站。这是偷偷摸摸的吗？当然，但是我觉得对广告支持的网站有很多不公平的偏见，特别是在某些人群中，所以我认为这只是一报还一报。现在，我搜索的实际字符串是*surf.yahoo.com*、*dmoz.org/editors*和*google.com/evaluation/*。第三个并不是一个真正的目录，它是当谷歌向你的网站发送人工审查时你会看到的。除了与他们的 adwords 程序相结合，谷歌没有任何针对弹出广告的规则，但你永远不知道评论的个人态度，你也不想因为一个弹出广告而让他们破坏你的网站。关于谷歌人工评估的更多信息，请看[这个帖子](https://www.sitepoint.com/forums/showthread.php?t=308529)。

我想讨论的最后一个问题是“下载驱动”可能会打击你的访客。除了使用声誉更好的 popunder 公司，或者那些明确声明不允许这类广告的公司，或者明确标注这类广告的网络，你真的没有太多办法可以做，所以你可以选择不运行它们。即便如此，即使偶尔有广告商违反规则，你也只能希望电视网能迅速禁止他们的广告活动(或者有用户通知你，让你这么做)。当然，你也可以鼓励你的用户使用火狐浏览器，这应该会让他们的浏览体验更安全一点，而且有了谷歌新的 T2 推荐项目，你甚至可以为你推荐的每一个新的火狐用户赚取 1 美元。

你们中的一些人可能想知道为什么我只谈论弹出窗口而不谈论弹出窗口。原因是弹出窗口现在很少见了，因为它们通常被认为比弹出窗口更烦人。大多数网络现在只关注弹出窗口。尽管如此，如果你运行弹出窗口，我上面提到的概念仍然适用。

对于 popunder 提供商我推荐[部落融合](http://www.tribalfusion.com)，[爆裂！媒体](http://www.burstmedia.com)，和[卡萨勒媒体](http://www.casalemedia.com)。我以前也推荐过 [Fastclick](http://www.fastclick.com) ，但是最近他们的费率下降了，他们的[推荐计划变化](https://www.sitepoint.com/forums/showthread.php?t=274209)给我留下了不好的印象。在上面提到的那些中，卡萨勒对我来说似乎有最多的库存，部落融合有最好的整体价格，爆裂有最低的库存和最低的价格(除非他们给你一个有针对性的 popunder 活动，爆裂有时会这样做)。Fastclick 的库存也很高，但费率很低。
