# 虚拟商业的挑战

> 原文：<https://www.sitepoint.com/the-challenges-of-virtual-business/>

为自己工作的一大好处是，我们很多人都可以在家工作。在我的行业中，我很少有机会与客户面对面或做任何形式的现场工作。这对我和我所做的工作很有帮助。我完成工作更快；我可以做得更多；我受益于更稳定的高效工作时间。

如果你从事虚拟工作，你可能会面临一系列完全不同于其他人的挑战。虚拟工作并不像看起来那么容易，但是有办法克服虚拟安排带来的挑战。

**挑战:建立信任**

在与某人做生意之前，双方需要有一定程度的信任。作为人类，我们信任(并雇佣)其他人，他们的面孔、声音和个性都在我们的决策中发挥着作用。当你在一个完全虚拟的环境中工作时，大多数人很难获得同样程度的信任。销售你的服务会更加困难，你可能要加倍努力。这里有一些方法可以让你在生意中加入个人元素，从而促进信任的建立:

*   安排与潜在客户的电话会议
*   用个人的声音写博客
*   参与社交网络
*   将视频添加到您的网站或博客
*   提供参考
*   在你的网站上放一张你自己的照片

**挑战:沟通**

雇佣虚拟专业人员的客户通常担心他们可能会消失，客户无法找到他们。如果没有面对面的交流，客户很难相信他们雇佣的人会在项目期间有空。应对这一挑战的最佳方式是通过以下方式提供一致的沟通:

*   对所有电子邮件和电话做出回应
*   通过电话、Skype 或即时消息设置定期签到
*   确保明确项目的角色和职责
*   解决任何快速出现的问题

**挑战:信息共享**

在我的业务中，我不得不开发一个与客户和分包商来回共享信息的流程，以确保每个人都有完成任务所需的数据。对于客户，在某些情况下，这通常是通过电子邮件或普通邮件来完成的。对于分包商，我使用一个项目管理系统，让我们每个人都可以访问我们需要的信息。

对于任何类型的虚拟协作，您必须:

*   确定你需要的信息
*   拥有一个以可访问的方式归档信息的系统
*   在项目期间和之后记录和归档数据
*   为数据存储和共享提供安全可靠的系统

任何一种虚拟工作的底线是，你希望你的客户和潜在客户了解你和你的工作方式。仅仅因为你们不是面对面地工作，并不意味着你们之间的关系不那么结构化；这只是一种不同的动态。信任可能有点难以实现，但在创建可持续的虚拟关系时，它甚至更重要。
