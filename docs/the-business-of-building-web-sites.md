# 建立网站的生意

> 原文：<https://www.sitepoint.com/the-business-of-building-web-sites/>

大家好，我叫迈尔斯·伯克。五年前的这个月，我疯了，离开了我的前雇主，开了一家网站设计和开发公司，只有两周的工资，还要还房贷，还有一个 23 个月大的孩子和怀孕的妻子。

你可以想象，我的勇敢行为并没有给后者留下太深的印象，然而我很快扭转了局面，不到三个月后就雇佣了我的第一位员工。几年前我是自由职业者，但这一次我相信我从我以前的错误中吸取了教训。

我们现在在 web 团队中有 15 名员工，并且是一家更大公司的一部分，我是这家公司的股东和董事，员工总数刚刚超过 40 人。最近，我被授予“西澳大利亚 40 岁以下最杰出的商业领袖之一”，我很自豪能获得这个奖项。

在接下来的几个月里，我希望与你们讨论我的想法和观察，我所犯的错误，以及我在小企业所有权和管理之旅中学到的教训。我说讨论，因为我真的希望我能指望你对我的帖子发表评论，或者联系我，并为这次对话提供你自己的意见。

Simon Mackie 和 SitePoint 的梦幻团队邀请我加入他们的博客团队，并分享我的一些知识，我感到非常荣幸。我将同样荣幸地听到你们的问题，你们自己的教训和你们的意见。也许我们可以从鼓励下面的评论开始，并建议一些你想听的话题。
