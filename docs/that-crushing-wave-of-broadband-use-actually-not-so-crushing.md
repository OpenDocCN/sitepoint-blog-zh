# 宽带使用的毁灭性浪潮？实际上没有那么粉碎

> 原文：<https://www.sitepoint.com/that-crushing-wave-of-broadband-use-actually-not-so-crushing/>

关于网络中立性的争论的核心是互联网服务提供商的争论，即由于流媒体视频、在线游戏、丰富的互联网应用和像 BitTorrent 这样的 P2P 流量，网络使用正以如此惊人的速度增长，他们的网络不可能处理它。听互联网服务提供商说，即将到来的[大洪水](http://www.discovery.org/a/3869)将会在 2010 年把我们所知的互联网带向崩溃，除非我们采取措施阻止它。比如，带宽限制或[按使用付费网络接入](http://www.readwriteweb.com/archives/time_warner_per_usage_billing.php)。

这个理论只有一个问题:至少在宽带运营商可能最热衷于争取控制用户网络访问权的美国，这场大洪水看起来不会到来。Cogent Communications 最近报告称，今年第二季度[网络流量下降了](http://telephonyonline.com/ethernet/news/cogent-traffic-decline-0808/)1%。Cogent 表示，这在很大程度上是由于美国学校放假时通常的夏季放缓，但这仍是该公司首次看到整个季度的流量下降。

Cogent 的首席执行官指责竞争对手没有如实承认流量下降。

> “互联网将继续增长，但它正处于转型期，”[Cogent 首席执行官戴夫]谢弗说，并补充说，他认为 Cogent 的一些竞争对手在他们关于这个问题的轶事评论中没有“直截了当”地承认这种普遍的流量减速。—在线电话

明尼苏达大学的 Andrew Odlyzko 博士是研究美国互联网流量模式的专家，他表示，这种减速已经持续了 5 年。Odlyzko 在 4 月份告诉 Ars Technica，在过去的 5 年里,“交通增长一直在放缓”。在 90 年代末，网络流量增长每年翻一番，但在基本饱和的西方市场，这种增长大幅放缓，现在每年徘徊在 50-60%，这是 T2 目前的资本投资水平可以支持的速度。

当然，我们在[年](http://www.internetnews.com/reporters_notebook/article.php/3671686)后的[年](http://www.broadbandreports.com/shownews/55667)一直在听说网络即将崩溃，但它还没有发生。Qwest 首席技术官 Pieter Poll 在 2006 年承认:“我发现流量远低于一些业内人士所说的情况。我的意思是，你会听到 P2P 流量显著两位数渗透率的说法，但事实并非如此。"([链接](http://www.lightreading.com/document.asp?doc_id=91565&WT.svl=news1_2))

互联网没有在 2006 年崩溃。2007 年没有崩盘。2008 年不会崩盘。为什么这还是一个问题？
