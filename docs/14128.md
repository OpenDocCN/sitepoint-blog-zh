# Java 6“野马”新桌面特性发布

> 原文：<https://www.sitepoint.com/java-6-mustang-new-desktop-features-announced/>

这一次有点慢，但我想澳大利亚的长周末会做到这一点。Sun 发表了一篇文章，详细介绍了它正在为 Java 的下一个主要版本——代号为“Mustang”——开发的新的桌面特性。

与公开发布的[快照版本](https://mustang.dev.java.net/)一样，Sun 在这篇文章中采用了“开放和诚实”的策略，解释了它*希望*在 Java 6.0 中包含的内容——而不一定是它*能够保证*会包含的内容。如果一些特性不可行或者需要很长时间才能发布 6.0，那么它们可能会被删除(就像 Java 5.0 中的[共享虚拟机](https://www.sitepoint.com/blog/))。

对我来说，一些亮点是:

*   **真正的双缓冲**
    当切换到已经在后台运行了一段时间的 Java 应用程序/小程序时，不再有“灰色窗口”。
*   **原生 Windows 和 GTK 观感渲染**
    将使用原生操作系统的 API 来渲染用户界面组件，尽可能完美匹配原生应用的外观。
*   **字体抗锯齿(包括 LCD 子像素着色)**
    将尊重原生操作系统的设置，以产生与 Windows 上的 ClearType 相当的文本着色。
*   对火狐
    的全面支持，而不是像现在这样简单地支持 Mozilla。
