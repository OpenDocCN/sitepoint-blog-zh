# 调查显示:自由职业者牺牲高薪独自工作

> 原文:[https://www . site point . com/survey-says-freelancers-sacrifice-bigger-sales-to-work-alone/](https://www.sitepoint.com/survey-says-freelancers-sacrifice-bigger-salaries-to-work-alone/)

[![designer salary](../Images/824d503d576135667ee601e2987cf00a.png)](https://www.sitepoint.com/wp-content/uploads/2009/03/experience_salary_08_468.jpg) 作为我最近一篇文章[如何计算你的时薪](https://www.sitepoint.com/how-to-calculate-your-hourly-rate)的后续，我搜索了一些薪资调查，以下是我所发现的内容的摘要。

**[Coroflot 2008 年设计薪资调查](http://www.coroflot.com/creativeseeds/2009/03/the_2008_design_salary_survey.asp)**

Coroflot 是一个面向设计师和创意专业人士的求职网站。他们的调查包括 60 个国家的 3900 多名受访者，处于职业生涯的各个阶段。注意结果:

*   经验会得到更高的报酬，与专业知识无关。
*   太平洋西北部去年的工资上涨了近 17%。
*   那些有正式工作的人比自由职业者挣得多，自由职业者比学术界挣得多。
*   网页和平面设计师最有可能独立工作。

[![aiga](../Images/173636bce2f3187c61dda65585335b6b.png) ](https://www.sitepoint.com/wp-content/uploads/2009/03/aiga.jpg) Coroflot 从 2001 年开始对网络专业人士进行民意调查；2008 年和以往调查的全部结果都可以在他们的网站上找到。

**[AIGA/2008 年设计工资问卷调查](http://www.aiga.org/content.cfm/salary-survey)**

专业设计机构 AIGA 统计了 6400 多份美国人对 2008 年薪酬调查的反馈。它是这样说的:

*   网页设计师的工资中位数是 65000 美元；入门级设计师的是 45000 美元。
*   在网络开发人员中，位于大西洋中部地区的人工资中位数最高。
*   网络开发公司 87%的员工都有带薪休假。

AIGA 从 2001 年开始进行他们的工资调查，过去的结果可以在网上找到。

[![survey](../Images/0fdaf1e4ab11d8ec8ff95ff3e63f0067.png) ](https://www.sitepoint.com/wp-content/uploads/2009/03/how.jpg) ** [ HOW 杂志的 2008 年设计薪资报告](http://www.howdesign.com/article/08SalaryReport)**

《HOW》杂志在美国对 2200 多名设计师进行了调查。下面是调查结果:

*   从 2006 年到 2008 年，设计师的税后工资平均增长了 8.1%(从 2004 年到 2006 年增长了 6.2%)。
*   内部薪酬已经赶上了设计公司的薪酬，差距不到 100 美元。
*   自由设计师的工资大幅上涨，比 2006 年(52，236 美元)上涨了 21%。

杂志如何每两年进行一次调查。报告全文可在线下载。

即使自由职业者和独立设计师确实比内部设计师挣得少，但在我看来，自由的交换是值得的。你呢？
