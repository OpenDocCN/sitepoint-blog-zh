# 用社区赚钱——第二部分，赚钱的方法

> 原文：<https://www.sitepoint.com/making-money-with-communities-part-2-ways-to-make-money/>

昨天我谈到了社区在赚钱时面临的问题。今天，我们将讨论将社区自然吸引的忠实追随者转化为机会的具体方法。

正如我在第 1 部分提到的，CPM(横幅广告等。)和 CPC (Adsense，YPN)并不总是最适合社区。虽然我确实使用它们，但我总是把它们放在低优先级的地方，用作填充或剩余库存。我们不想把钱留在桌子上，对吗？也就是说，有更好的方法来使用宝贵的屏幕空间。

让我们来看几个例子:

1.  直接广告销售
2.  赞助
3.  文字广告和文字链接
4.  会员支持、捐赠和小额赞助
5.  联盟营销

1.直接广告销售

直接广告销售可能是所有广告类型中最难获得的。你真的需要坚实的流量数字和一个伟大的利基主题，但主要是你需要成为一个牛头犬。获得直接广告销售最困难的部分是找到一个为特定公司处理广告购买的人。假设你经营一个体育网站，你想让耐克买一个广告。你会有一份令人生畏的工作，一封又一封的发邮件，一个又一个的打电话，却只能被弹回一段时间。一旦你做到了，你仍然要向他们推销你的想法，随着许多公司在网上花费更多的转变，这变得更容易了。在你主动联系某人之前，做好你的功课。他们在其他网站上投放广告吗？他们是通过部落融合这样的网络来运作还是在内部运作？你是否整理了一份关于为什么有人想在你的网站上做广告的事实表，包括统计数据和人口统计数据？如果没有，那就别麻烦了。这是赚钱的地方，但是如果你不愿意投入一些工作，那么这个方法就不适合你。这不是灵丹妙药，不是被动收入，但这是真金白银所在。

2.赞助

除了直接广告销售，还有赞助。这些可以是赞助你网站上的一个页面，也可以是用另一个公司的信息来联合你的网站。你甚至可以更早开始，需要更少的流量，因为广告商不是真的为业绩付费，他们是为品牌付费。继续体育方面的例子，你可以让当地或区域性的体育用品商店(或连锁店)赞助你的“购买、销售和交易”部分。显然，网站范围内的赞助更容易管理，但你可能实际上这样做更少。你可以向多个赞助商收取他们各自版块的全价，但是如果你让一个赞助商买下你的网站，你可能会给他们一个折扣。这变成了时间和金钱的权衡，只有你能做出选择。时刻准备好各种选择。对你的赞助商的需求保持灵活，这样你就能从一开始就走在游戏的前面。

3.文字广告和文字链接

一旦你的网站建立起来，并且出现在搜索引擎中，你可能会被想购买你的网站或内容中的文字广告或文字链接的人找到。这是一个敏感的话题，我知道有些人不喜欢，但它确实能赚钱。不错的收入。比站长或 YPN 的钱还多。这取决于你的个人喜好，如果你决定探索这些，你会赚到一大笔钱。

4.会员支持、捐赠和小额赞助

成员可以是一个很好的支持来源，甚至是经济支持。任何社区都会自然地建立起忠实的追随者，毕竟所有人都在你的社区投入时间，与网站和其他成员建立关系。这是强大的东西，甚至大公司都试图复制不同程度的成功。成员们经常想要表达他们的支持，你可以给他们一些方法。这些方法不会让会员花费太多，但随着时间的推移会越来越多。你可以建立一些特殊的“成员团体”,加入这些团体需要付费。一个月 1 美元以上。价格取决于网站，您可以随时获得多种级别的支持。对我来说，最重要的是给成员提供“额外”支持，但不要从现有成员中移除功能。例如，支持者可以获得额外的邮箱空间、自定义图标/徽章、自定义标题等。小事情，但有趣的东西，它不会阻止非支持者正确使用网站。显然，这可能演变成付费的“会员专用”领域，但我还没准备好进入这一领域。

5.联盟营销

联盟营销可以做很多工作。你必须找到一个与你的社区相匹配的产品或服务，但它真的是值得的。社区是联盟营销人员花费大量精力的事情——建立他们的名单。该列表是要向其发送要约的人的邮件列表。你想确保你尊重你的成员，但联盟营销可以是一个很好的赚钱。联盟营销并不是为产品 x 竖起一面旗帜。忘掉它，现在就把它从你的脑海中抹去。我已经提到过，横幅广告不像以前那样好用了，但是对于联盟营销来说，横幅广告尤其没用。如果你找到了一个对你的社区有广告价值的产品，并且你找到了一种方法把它融入到网站、内容或时事通讯中，那么你会做得更好。对我的写作社区很有帮助的一个产品是作家文摘的订阅。找到适合你的社区的东西。不要试图在你的烹饪社区里卖 iPods，那样做效果不好。

这些只是过去对我有效的方法的例子。不是所有的方法都适用于每个网站，这才是在任何项目中赚钱的真正答案——试错法。不要害怕尝试一些事情，有些事情是行不通的，没关系。你从失败中学到的要比从成功中学到的多。我没有提到的另一件事是，当你的网站建立起来并有流量时，即使不是所有的机会，也是大部分的机会。如果你正试图在你的新社区中赚钱，我很高兴看到你有超前的想法，但是在你投入时间和工作建立网站之前，不要期望开始赚钱。

另一个确保你能最大限度利用的技巧是，这适用于任何类型的网站，不要公布你的广告价格。这听起来可能与你被告知的相反，你可能不喜欢谈判，但当你公布你的价格时，你就被它们卡住了。为了你自己和你的企业，你应该从每一次广告投放中获得最大收益。不是所有的广告客户都愿意花同样多的钱，通过电话或电子邮件吸引人们是成功的一半。如果你公布你的利率，你甚至在和他们说话之前就给了他们退出的机会。以更低的价格出售来填补你的库存总是比等待有人以全价购买要好，那样你只是在浪费库存，把钱留在桌子上。

从你的社区中赚钱是我经常提到的一个话题。同时，请随时给我留言或提问。祝你好运赚钱！
