# 建立一个内容丰富的网站

> 原文:[https://www.sitepoint.com/content-rich-website/](https://www.sitepoint.com/content-rich-website/)

是什么让一个网站对访问者有吸引力？当然是它提供的内容。

你可能会开展一场精彩的广告活动；开发病毒式营销工具和有吸引力的联盟计划。但是，除非你的网站内容丰富，否则你的努力所带来的流量高峰只会是短暂的。有用的、有价值的、信息丰富的、有教育意义的或者只是简单的娱乐性的内容比其他任何东西都更能吸引和留住观众。

任何人都可以通过遵循几个要点来创建内容丰富的网站:

遵守纪律
定期更新你的网站
知道如何创建内容，或者在哪里可以找到内容

**纪律和承诺**

要创建内容丰富的网站，你需要视野开阔，目光锐利。一句话；自律。从一个网站点击到另一个网站很容易浪费几个小时，甚至几天。不要让自己分心:将你的在线内容探索限制在与你网站主题直接相关的网站和资源上。

纪律也延伸到内容创作。成功的作家在开始写作之前不会等待灵感。相反，他们制定了一个有规律的写作计划，并坚持下去。不管他们喜不喜欢，他们都会坐在桌前写作。

你也必须制定一个时间表来添加内容到你的网站，并遵循它。耐克的想法是正确的——“只管去做”！

**定期更新您的网站**

没有什么比一个无人管理的网站更死气沉沉了。

定期更新或修改网站内容会让你在竞争中获得优势。如果人们注意到每次都有新的东西可以看、学习或享受，他们会不断回到你的网站。

**创建您的内容**

你最知道——或者应该知道——你的观众想要什么。然后，决定哪种类型的内容最能满足他们的需求，以及如何着手寻找或创建这些内容就变得很容易了。以下是一些建议:

社论
专题文章
新闻剪辑或故事
美术馆
汇集最佳内容
评论
公告
采访
互动功能——投票、反馈、讨论组、论坛、聊天

**社论**

社论是该领域公认的“专家”的意见(你或客座撰稿人)。它们是很好的内容，因为它们会引起读者的反应，不管是赞同还是反对作者。这可能会引发激烈的辩论。给你的读者一个让他们的观点为人所知的途径——在你的网站上放一个公告栏或留言簿。人们会回到你的网站，至少会阅读对他们评论的其他回应。你可以在将来的后续文章中使用你的读者的反馈。

**长篇特写**

专题文章是最常见也是最好的内容形式之一。

根据你网站的性质，文章可能长或短，正式或闲聊，技术或娱乐。以下是一些建议:

·文章不要太长。虽然没有任何规则，但将特写保持在 1200 字以下可能是安全的。如果它们较长，将其转换为多部分特征。

文章应该与你网站的主题相关。

他们应该教育、娱乐或告知。试着将每篇文章的信息限制在一到两个新的想法。

不要简单地重复你在别处读过的文章。通过发布其他地方没有的东西，你可以提高你网站的价值和可信度。

获取原创内容有几种方式。

–写出自己的特色。这假设你写得相当好。如果你不能，或者没有时间或兴趣，那么让别人为你写文章对你和作者都有好处。

**你可以提议:**

a.)向客座作者支付报酬或

b.)与他们的网站交换文章作为回报。你的网站访问者受益于为他们提供另一种观点。一个额外的好处是，你可以从客座作者的网站上吸引一些新的常客！

c.)排他性。如果别人为你写文章，确保同一篇文章不会被提交到其他几十个网站和时事通讯上。发布独家内容的好处包括在线和离线的其他出版物中的联合机会，以及读者忠诚度。

**转载**

转载他人所写的文章，但你必须始终获得许可。所有作品，从写作的那一刻起，就是版权，归作者所有，无论是否标有版权符号。内容不是免费的。然而，你可以通过在内容上添加你自己的“旋转”来使重印变得有趣和个性化。写一篇关于主题的介绍，或者评论作者的观点或结论。

注意避免未经作者许可就“编辑”原文。避免之前在其他网站和电子出版物上多次转载的文章。

**新闻服务台**

你的网站可以通过向你的读者提供感兴趣主题的及时新闻，成为感兴趣领域的最新发展和事件的来源。

在你的网站上创建一个单独的板块来处理行业新闻。或者你可以将整个网站用于新闻更新。

呈现新闻有很多种方式:

作为专题文章
作为简短的新闻剪辑，带有完整故事的链接
作为新闻故事，其中每一条都有详细的解释

无论你选择哪种方式来展示新闻，一定要打上你自己的个性。用你的写作风格，或者加入你的观点，让它变得有趣、个性化、健谈、有趣、独特，或者所有这些。

与专题文章一样，新闻剪辑应该

对你的读者
及时有用的相关
。记住，旧闻不是新闻！

有很多资源可以帮助你找到新闻故事。您可以:

订阅与你的主题相关的电子杂志
注册通过电子邮件定期发送更新的新闻服务
注册接收与你的主题相关的新闻稿
在网上冲浪时积极寻找新闻内容
阅读与你的主题相关的报纸、杂志和书籍
参加会议、研讨会或讨论会，写下你的感想

**内容聚合器**

通过使你的网站成为与你的主题相关的问题的最终内容资源，你将吸引并保留一个忠实的观众。你的网站将成为任何人寻找你的主题信息的目的地。

作为你的主题的专家，你可以评估网站和其他资源(电子杂志、目录、书籍、离线出版物)的优劣，然后把它们和你的评价和意见一起列出来。

**美术馆**

文本不是内容的唯一形式。声音、音乐、图片、照片、视频剪辑——很快可能还有气味和梦境(！)–可以成为网站上的吸引力。

如果你是一名专业摄影师，你可以创建一个快照展示。如果你是一个音乐团体，你可以提供一些你的作品供下载。业余电影制作人可以利用互联网作为他们低成本的全球分销渠道。

随着宽带连接变得更加普及，我们可能很快就会看到大多数网站转向这种互动和动态的内容模式。

**审核人**

利用你的网站向你的访问者推荐“最好”的产品和服务——书籍、网站、音乐、电影、艺术品、网页设计师、餐馆等等。你，专家，告诉他们什么是好的，什么不是。

分享一些他们感兴趣和有益的东西。如果你读了一本很棒的书，或者浏览了一个很棒的网站，那就告诉你的访问者。评论有助于建立网站的可信度和信任度。

**如何写评论？**

以下是一些建议:

为那些想了解更多信息的人提供联系/订购信息。

相关。将你的评论限制在你网站的主题上

要有选择性。为你评论的产品设定高标准。要善于分析。具体地告诉你的读者你所评论的东西的好与坏，以及为什么。

不要走极端。抑制你的冲动，提供极端的赞扬和批评。

提供建议。最后，说说你对这件事的看法。好吃吗？还是坏？你的读者应该购买产品、访问网站、订购服务吗？告诉他们。

你应该复习什么？

任何东西——只要它与你网站的主题相关，并且可能让你的读者感兴趣。以下是一些想法:

书籍:关于你的主题肯定会有很多书。复习好的，甚至不好的。如果一个作者/出版商给你发了一本书来评论，就写一写。

网站:如果你有一个最喜欢的网站，或者偶然发现一个有用的、有趣的或信息丰富的网站，浏览一下。你的读者会因此而喜欢你——他们自己可能永远也不会发现！

产品和服务也可以成为你的磨坊的谷物。只要记住遵循指导方针——对你的读者相关且有用。

**公告**

你的公司做了一些新的和创新的事情吗？你开发了产品的改进版本了吗？
你知道你所在行业的内幕信息吗？
你关注你所在领域的谣言和流言吗？
你会密切跟踪波动和发展吗？

如果有，在你的网站上宣布。人们喜欢了解最新的八卦和事件。

**面试**

对你所在领域的专家或名人的采访会产生令人兴奋的、有价值的内容。通过电子邮件或电话进行面试也很容易。

选择一位你认为你的读者会感兴趣的专家或嘉宾。更多地了解你的专家——特殊兴趣、经验、成就、领域地位等。这允许你提出相关的问题。

决定采访的主题。准备一系列你想问的问题。然后联系你的客人，提出采访要求。

大多数人会很乐意参与你的面试。对于那些犹豫不决的人来说，提及他们在你的网站上的曝光率就足以让他们同意了！

通过电子邮件、打电话，甚至安排一次面对面的会面，向客人发送一份你的问题。一旦你有了你的副本，做一些编辑上的修改，让它读起来更好。这可能意味着重新安排问题，以便有一个顺畅的思路。通过巧妙的添加额外的问题和评论，你可以让面试看起来像是亲自进行的，并且读起来更有趣！

询问其他问题，或在必要时要求澄清。最后，确保获得你的客人对最终采访版本的认可。然后在你的网站上发布它。

**互动内容**

为什么要和读者互动？

你可以与访客建立更密切、更私人的关系
你可以了解他们的偏好、喜好和厌恶
你可以倾听他们的抱怨和建议
你可以要求他们对你的表现进行反馈
你可以收集关于你的观众的有价值的人口统计数据

还有…你可以获得新鲜的内容在你的网站上使用！

所有这些都允许你对你的网站进行改进，增加内容，提高对访问者的价值和效用。通过使用交互式工具，您可以生成供以后使用的内容。有许多不同的方法可以做到这一点。这里有一些想法:

**民意测验和调查**

调查你的读者。问个问题。请求对某个问题或话题进行投票。
可能是关于

网站上的一篇文章——“你觉得本周的专题有帮助吗？”
时事——“你会在下次选举中投票给共和党吗？”
一个有争议的问题——“你相信学校里的宗教吗？”
用户偏好——“你想看到这个网站每天更新吗？”
人口统计数据——“你有小孩吗？”

**一些指南:**

·简明扼要。提问不要超过 7 到 8 个问题

提供选择。将问题框定为是/否选项

提供回复的动机。折扣、免费赠品、试用会员

向受访者提供调查结果。你的读者会乐于听到他们的同龄人对这些问题的想法和感受。

表明你正在根据收到的反馈采取行动

**反馈**

读者的反馈是非常有价值的——如果你以开放的心态倾听它，并采取行动的话。批评能激励你提高水平。问题和请求可以指出您必须在哪些方面提供有助于读者的内容。网站管理员和网站访问者之间良好的反馈是非常有价值的。

明确表示你欢迎反馈。

在每一页上提供一个链接，让访问者写下他们的评论、意见和改进建议。

·提供一个电子邮件地址，以便写信或使用反馈表

如何将反馈转化为内容？

发挥你的创造力，想出新点子。以下是一些建议:

写一篇专题文章，回答读者提出的问题

打印出读者提出的问题或评论，并回答它们。在引用他们的话之前，请务必征得他们的同意。大多数情况下，他们会同意的——尤其是当你在网站上提到他们的名字或网址，向他们承诺让他们成为焦点的时候！

根据读者的建议采取行动；然后写下这些变化

**讨论列表**

虽然反馈是一对一的交流，但讨论组允许多对多的交互，因为发送到该组的每封电子邮件都会分发给每个成员。

**你可以这样做:**

–使用免费或付费服务创建一个讨论组。

–在你的网站和电子杂志上公布，邀请所有人参与。

读者可以讨论有争议的文章和新闻故事，讨论与你的主题相关的其他问题，并与其他读者分享他们的观点和意见。

在如此激烈的辩论中产生的材料可以再次被用作种子，为你的网站增加内容。

**论坛**

电子公告板是一个网站，访问者可以阅读其他人发布的消息，并通过在公告板上写下另一条消息来回应。这个回复也显示给其他人阅读，产生一个不断增长的相关主题的讨论列表。

经过一段合理的时间后，特定主题下的消息可以被封装到一篇专题文章或您站点上的其他形式的内容中。

**聊天室**

实时在线聊天是一种奇妙的互动工具，但也有不可避免的局限性。聊天活动允许创建创新的内容，如

–采访参与者，尤其是专家和名人

–关于专题和问题的讨论

–为您的访客举办研讨会

聊天记录——根据谈话的连续性和流畅程度进行编辑——可以产生令人兴奋的内容。

你如何为你的网站获取内容？

有四个选项:

自己创作
邀请投稿
雇佣内容提供商
辛迪加或者从其他人那里获得内容许可

**创建您自己的内容**

如果你在你的专业领域有丰富的经验，并且你的写作技巧非常出色，那么内容创作将会是一件简单的事情，你只需要坐在电脑前敲几分钟键盘。

然而，如果你不是这样，为你的网站生成材料就像拔牙一样——没有麻醉！

但是不要绝望。网上有适合每个人的东西:-)

在您决定创建自己的内容之前，请考虑以下几点:

你有专业知识吗？

你真的是专家吗？你能定期写出有价值、可靠、有趣的内容吗？你的大多数读者对你的主题了解不多——事实上这就是他们首先访问你的网站的原因！但是偶尔会有专家。你不能用借口来愚弄你的读者。甚至不要尝试。

**你有技能吗？**

你能写好吗？你可能会理解广义相对论。但是你写它的方式可能会让爱因斯坦本人困惑！

你有时间吗？

写高质量的原创内容可能非常耗时。因为一个好的网站永远不会结束，它是一个持续的事情，日复一日，周复一周。你能做到吗？

**邀请投稿**

如果写自己的内容不适合你，周围有很多人会很乐意为你的网站写作。他们这样做是因为这让他们看起来像专家，让他们接触到你的读者。最常见的情况是，这些作者要求的回报只是一个副标题——在文章的结尾对作者和他/她的业务进行 3 到 5 行的简短描述。因此，如果你不想(或不能)创建自己的内容，请他人帮助。

在你的网站上声明欢迎其他作者的文章。然后仔细阅读提交的材料，选择最好的呈现给你的观众。

**雇佣内容提供商**

这才是有钱人的出路！付钱给别人为你创造内容。

内容提供商有不同的形式。

自由作者是合同作家，他们会按字数/文章付费提交文章。

全职作家和内容提供者是那些提供专业质量的材料并拿固定薪水的员工。虽然这可能会很贵，但仍然比试图自己获得技能要便宜。

**辛迪加或授权内容**

辛迪加是一种有趣的模式，可以以合理的成本从专家那里获得高质量的内容。个人或公司创建的内容会显示在其他合作伙伴网站或电子杂志上，以获得报酬(固定费用或收入百分比)，有时甚至只是在合作伙伴网站上展示。

另一种选择是在支付许可费的情况下许可内容一段特定的时间。

您可以考虑联合或授权来自以下网站的内容:

提供各种主题的简短更新和新闻快讯的新闻服务

内容聚合器，从网络上收集关于某个主题的信息

杂志和报纸

内容丰富的利基门户网站

与你的主题相关的其他网站

**网站内容的一些通用提示**

提供高度知识性、娱乐性和原创性的内容只是成功公式的一部分。

一个将自己的个性融入网站的网站管理员会让他们的出版物独一无二，与众不同。如果一个网站管理员能成功地向读者传达他们的价值观、理想、抱负和态度，你的读者会相信他们了解你，这个网站管理员。他们认同你，信任你，尊重你。

以下是一些想法:

坚持己见或陈述自己的观点是可以的

永远把你的读者放在首位

坚持你的价值观和道德观

情绪化，不要拘泥于逻辑

如果你喜欢你在网站上创造的东西，它会在你的作品中表现出来。你的读者也会感觉到。

**为网络写作**

在电脑屏幕上在线阅读与人们阅读印刷出版物的方式有很多不同。大多数人浏览一篇文章，而不是逐行阅读。

通过将内容分成小块，并在它们之间大量使用空白，你可以使你的网页看起来整洁、整齐、易读。

一些提示:

使用较短的句子
使用“吸引人”的单词和短语
避免使用长而难懂的多音节单词
马上切入正题
使用副标题来衬托段落

**拼写和语法**

你的读者会根据许多因素来评判你。拼写/语法很重要。

如果你想树立一个严肃的形象，合理的注意确保拼写和语法的正确性是至关重要的。

一些错误可能(并且将会)从缝隙中溜走！你的读者可能会接受这一点。但是如果经常发生，你就失去了可信度。

**风格**

一种正式的、非个人的、合乎语法的风格对某些主题来说是理想的。但是一种健谈、友好、边说边写的风格将非常适合其他人。没有任何规则——用对你有用的。

**大胆尝试**

管理网站可能是一次有趣的自我发现之旅。勇于冒险。尝试新想法。模仿你觉得有趣的模特。不断创新。

并监控对每个变化响应。你的读者喜欢吗？如果有，就留着吧。如果没有，再试验一次。

大多数网站都是通过这种试错法发展起来的。赢家是那些发现并发展成功模式的人，同时无情地淘汰任何失败的模式。

对想法和建议持开放态度，并愿意适应变化，这也许是你作为一名网站管理员所能培养的最好技能。
