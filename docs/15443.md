# MySQL(新骑手出版)

> 原文:[https://www.sitepoint.com/mysql-new-riders-publishing/](https://www.sitepoint.com/mysql-new-riders-publishing/)

MySQL 是一个关系数据库系统，作为开发数据驱动的网站和应用程序的后端解决方案，正得到越来越广泛的认可。MySQL 快速、强大，而且在大多数情况下是免费的。在他的书中，Paul DuBois 试图为这个高性能数据库引擎提供一个全面的参考和学习工具，并取得了相当大的成功。

第一次读这本书的时候，我已经有了一些关于 MySQL 的小经验。我对使用它的了解是通过阅读软件附带的在线参考手册获得的。根据记录，这是一种介绍数据库世界的糟糕方式。尽管 MySQL 是一款时尚、优雅的软件，但它在很大程度上是作为免费软件发布的。和大多数免费产品一样，这本手册看起来更像是一种事后的想法，是出于对所有东西应该如何工作的技术记录的需要而写的，而不是为了给初学者提供一个好的、简单的介绍。

这本书非常不同。在许多方面，它可以被认为是“MySQL 从未有过的用户指南。”作者不仅是 MySQL 参考手册的贡献者，这解释了他对程序的细微差别和复杂性非常熟悉，而且他以优雅的写作风格传达这些概念，设法保持事情的趣味性。

该书将主题分为三个部分:“MySQL 的一般使用”、“使用 MySQL 编程接口”和“MySQL 管理”。第一部分以一个与数据库进行命令行交互的教程开始，介绍了这个过程中的基本结构化查询语言(SQL)命令。本教程开发了两个简单的数据库，这两个数据库将在本书的剩余部分讨论各种示例时使用。完成这一快速概述后，本书深入介绍了 MySQL 数据/列类型、MySQL 支持的 SQL 语法以及索引等优化技术。

本书的第二部分涵盖了最流行的 API，这些 API 允许访问嵌入在其他应用程序中的 MySQL 数据库系统，无论是公司法律数据库、基于 Web 的照片库，还是介于两者之间的任何东西。涵盖的三个 API 分别是 C、Perl DBI 和 PHP。虽然考虑到这是一本关于 MySQL 的书，这些主题的覆盖范围令人惊讶地完整，但是假设对每种语言都有基本的了解。没有提供对 JDBC(Java 数据库接口)的讨论，考虑到 Java 作为开发平台越来越受欢迎，这似乎是一个奇怪的省略。

本书的最后一节“MySQL 管理”可能是读者参考最少的一节，但同时也提供了本书中一些最重要的信息。《MySQL 参考手册》中关于安全和访问控制等领域的部分尤其难以理解，这本书又一次完美地扮演了代理用户指南的角色。诸如安全性、帐户管理、数据库维护和修复等主题都以易于阅读的方式进行了深入探讨。

附录提供了从完整的安装指南到 SQL 语法、操作和功能的大量参考资料，最后是为构建数据驱动网站提供 MySQL 支持的 ISP 购物指南。

尽管这本书涵盖了所有内容，但令人惊讶的是，它并没有试图提供关系数据库概念的介绍。虽然以前没有关系数据库经验的读者应该能够相当快地掌握基本概念，但良好的数据库设计的主题(如数据规范化)很少涉及，如果有的话。这是否意味着你不能用这本书开始构建你的第一个数据库驱动的网站？不完全是，但是除非你对设计好的数据库有一种直觉(许多人都有)，否则你会发现自己犯错误，并且比本书包含一章纯数据库设计时更频繁地回去重新设计数据库。

这本书提供了你成为 MySQL 专家或开始使用它所需的一切。由于 MySQL 基本上是免费的，入场费可以是你花在整理枯燥的技术在线参考指南上的时间，也可以是这本写得很好的书的费用，它将通过实际的例子和无价的见解来指导你学习过程的每一步。

**评级:4.5/5**
