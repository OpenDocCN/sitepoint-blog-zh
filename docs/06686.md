# 十大前端开发框架

> 原文：<https://www.sitepoint.com/top-10-front-end-development-frameworks/>

*(编者注:纯粹出于篇幅和可读性的考虑，本文是两部分中的第一部分。这一篇着重于 Bootstrap 家族，更多的将在第二部分中介绍。)*

随着网络的成熟和我们用来访问它的移动设备的范围迅速扩大，我们作为网页设计者和开发者的工作变得相当复杂。

十年前事情要简单得多。当时，几乎可以肯定的是，我们的大多数用户都是坐在办公桌前，看着一个大显示器访问我们的网站的。960 像素或多或少被认为是一个好的网页宽度。我们主要关心的是处理十几个桌面浏览器，并跳过一些额外的浏览器黑客来支持古怪的旧版本的 Internet Explorer。但是现在，随着最近五到六年手持电子设备的繁荣，一切都变了。我们已经看到了各种尺寸的智能手机和平板电脑、电子阅读器、电视浏览器以及许多其他产品的推出。多样性只会与日俱增。

很快，更多的人将通过他们的移动和替代设备而不是台式电脑来访问网络。事实上，已经有相当多的人将手机作为上网的唯一途径。这意味着对我们设计师和开发人员来说，理解如何应对这个移动世界很重要。尽管截至本文撰写之时，我们还没有完全弄清楚如何让我们习惯于在办公桌上看到的所有内容在我们的手持设备上提供同样令人愉悦的体验，但实现这一目的的技术和工具正在变得越来越好。

当我们处理未知的视窗尺寸时，我们使用的一个主要策略是所谓的响应式网页设计。这是一种基于浏览器窗口大小为设备提供自定义布局的机制。默认情况下，智能手机和平板电脑等小型设备上的大多数浏览器会缩小网页以适应屏幕，并提供缩放和移动页面的方式。虽然它在技术上是可行的，但它并不是一种很好的体验。文本太小，无法阅读，链接太小，无法点击，所有的缩放和平移或多或少会让人分心。

响应式 web 设计的技术是通过根据屏幕大小应用不同的样式表，为所有设备提供单一的 HTML 文档，以便为该设备提供最佳的布局。例如，当在大型桌面浏览器上查看页面时，可以将内容放入具有普通导航元素的多个列中。但当在智能手机的小屏幕上查看同一页面时，它会出现在一个带有大链接的栏目中，以便于点击。你可以在[媒体询问](http://mediaqueri.es/)画廊网站上看到网页设计是如何反应的。只需在您的浏览器中打开一个设计，然后将窗口调整得非常窄和非常宽，并观察布局根据窗口大小的变化。

到目前为止，我们已经看到，至少就目前而言，响应式网页设计是应对日益增长的设备多样性的一个足够的解决方案，但是在我们的设计中实现它的实际工具和技术是什么呢？我们必须成为网络大师才能处理它吗？或者仅仅是我们已经拥有的基本技能就足够了吗？有什么工具可以帮助我们吗？

这就是前端开发框架发挥作用的地方。响应式网页设计实现起来并不难，但是要让它在所有目标设备上都能工作就有点棘手了。框架使这项工作变得更加容易。它们允许您以最小的努力创建响应迅速、符合标准的网站，同时保持一切简单和一致。框架给你带来了很多好处，比如速度和简单性，不同设备之间的一致性等等。最重要的优点之一是它们非常容易使用，即使是一个知识很少的人也可以毫无问题地使用它们。

简而言之，如果你对当今的 web 开发很认真，那么使用框架是必须的，而不是一个选项。如今，你的网站必须非常灵活，以满足不同的浏览器、平板电脑、智能手机和一大堆其他手持设备。

一个前端 web 开发框架只是一个生产就绪的 HTML/CSS/JavaScript 组件的集合，我们可以在我们的设计中使用。有许多框架，但其中一些非常突出。为了便于您的理解，您会发现下面列出了一些当今最强大、最流行的框架。请记住，这些不仅仅是 CSS 网格，而是全功能的前端开发框架。

## 1.引导程序

Bootstrap 无疑是当今最流行、最广泛使用的框架。这是一个漂亮，直观和强大的网页设计工具包，用于创建跨浏览器，一致和好看的界面。它提供了许多流行的 UI 组件，具有简单而优雅的风格，一个网格系统和用于常见场景的 JavaScript 插件。

它的建造成本更低，由四个主要部分组成:

*   脚手架-全球风格，响应 12 列网格和布局。请记住，默认情况下，Bootstrap 不包含响应特性。如果您的设计需要快速响应，您必须手动启用此功能。
*   基本 CSS——这包括基本的 HTML 元素，如表格、表单、按钮和图像，用可扩展的类进行了样式化和增强。
*   组件–可重用组件的集合，如下拉列表、按钮组、导航控件(选项卡、药丸、列表、面包屑、分页)、缩略图、进度条、媒体对象等。
*   JavaScript–jQuery 插件，将上述组件带入生活，加上过渡、模态、工具提示、弹出窗口、scrollspy(用于根据滚动位置自动更新导航目标)、carousel、typeahead(一个快速且全功能的自动完成库)、词缀导航等等。

Bootstrap 已经足够强大，可以支持任何 web 界面。但是为了更好地利用它并使开发过程更容易，您可以找到大量的工具和资源来补充它。其中一些列举如下:

*   jQuery UI Bootstrap——jQuery 和 Bootstrap 粉丝的绝佳资源，结合了两者的力量。它很好地为 jQuery UI 小部件带来了 Bootstrap 的巧妙之处。
*   [jQuery Mobile Bootstrap 主题](https://github.com/commadelimited/jQuery-Mobile-Bootstrap-Theme)–类似于上面的 jQuery UI 主题，这是一个为 jQuery Mobile 构建的主题。如果您有一个用 Bootstrap 构建的 web 前端，并希望为移动设备提供类似的外观，这是一个方便的资源。
*   [燃料 UX](http://exacttarget.github.io/fuelux/)–这用额外的轻量级 JavaScript 控件扩展了 Bootstrap。它易于安装、定制、更新和优化。
*   StyleBootstrap . info–Bootstrap 有自己的定制器，但 style Bootstrap 更详细，它有颜色选择器，能够为每个组件设计不同的样式
*   [BootSwatchr](http://bootswatchr.com/)–一个引导主题滚轮，显示您的更改的即时结果。对于每个生成的样式，应用程序都会生成一个唯一的 URL，以防您希望与其他人共享或在以后随时返回并编辑。
*   一套漂亮的免费自举主题。
*   Bootsnipp–一个很好的设计元素和 HTML 代码片段的集合。它还提供表单和按钮生成器。
*   [layout it](http://www.layoutit.com/)–根据 Bootstrap 的元素和组件拖放界面构建器。它通过拖放将不同的元素放置和排列到布局中，帮助您直观地组成您的设计，然后允许您编辑它们的属性。你得到基本代码，然后扩展它。简单易行。

## 2.Fbootstrapp

Fbootstrapp 基于 Bootstrap，为脸书 iframe 应用和设计提供相同的功能。它包括所有标准组件的基本 CSS 和 HTML，如排版、表单、按钮、表格、网格、导航等，具有典型的脸书风格。

## 3.BootMetro

BootMetro 是一个受 Metro UI CSS 启发的框架，构建在 Bootstrap 之上，用于创建 Metro/Windows 8 风格的网站。它包括所有 Bootstrap 的功能，加上一些额外的附加功能，如平铺页面、应用程序栏等。

## 4.支撑带

简单来说， [Kickstrap](http://getkickstrap.com/) 就是一种类固醇上的自举。它使用 Bootstrap 作为基础，并扩展了许多应用程序、主题和附加功能。这使得该框架成为一个完整的工具包，无需安装任何东西就可以构建网站。只要把它放在你的网站上，你就可以开始了。

应用程序只是 JavaScript 和 CSS 文件的捆绑包，它们在页面加载完成后作为一个包一起运行。默认包含的一些应用程序有 Knockout.js、Retina.js、Firebug Lite 和 Updater。你还可以添加更多。

主题让你能够区别于大多数引导网站的标准外观和感觉。

额外的是粉丝创造的扩展引导 UI 库的附加物。它们通常使用相同或相似的语法。
