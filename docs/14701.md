# 。学术界的网络

> 原文：<https://www.sitepoint.com/net-in-academia/>

我一直在参加我大学一年一度的研究生研究大会，听演讲，我自己也做了一个！

让我吃惊的是学术界发展得如此之快。网络和信息是如何传播的。开始于 Java 的项目现在正转向。网或者正在调查。NET 作为部署和研究的另一种选择。当然可以。NET 已经存在好几年了，但是学术界通常变化很慢。

对于. NET 的未来来说，这都是好的迹象。哪里有活跃的研究，哪里就有下一代解决方案，哪里就是开发人员应该集中精力的地方。

随后与代表们交谈，解释 Whidbey 即将发生的变化，并看到各种可能性，特别是 Indigo(我们的许多研究都是在中间件方面)，这也是很好的。

然而，一个问题似乎是不和谐的。跨平台支持。在我看来，这是最大的限制。NET 及其在商业和学术界的应用。我们有转子，我们有单声道…但它需要微软的思想转变，让球真正滚动起来。

这不会很快发生…让桌面和服务器提供商编写开发平台的一个大问题。长角牛想要打平。NET 更接近，这是一个内在的约束。NET 和一个令人失望的。
