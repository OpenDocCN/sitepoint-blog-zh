# Flash 脚本–列表框组件

> 原文:[https://www.sitepoint.com/script-listbox-component/](https://www.sitepoint.com/script-listbox-component/)

如何使用列表框组件？这里有一个快速游览！

**1。**打开组件面板—按 ctrl+f7。

**2。**将 Listbox 组件拖到舞台上，并选择第 1 帧中的 Listbox 组件。该组件的参数显示在属性检查器中。

**3。**给组件命名。在属性检查器实例名称文本框中键入“listbox”。

**4。**标签参数显示用户可以选择的值列表。点按“标签”栏，然后点按放大镜以打开“值”弹出式窗口。单击加号(+)按钮输入新值。

![1184_1](../Images/08b4f663e9d77ec7f0a37c7728d36447.png)

**5。**单击默认值字段，然后键入“一”作为第一个值。单击加号(+)按钮输入下一个值。在默认值字段中单击，并键入“二”作为下一个值。

重复此过程，将“三”、“四”、“五”和“六”的值相加。完成后，单击“确定”关闭标签框。

**6。**数据参数是可选的。它用于指定与列表框中的项目(或标签)相关联的值。在这种情况下，不需要将值与我们的标签相关联，所以我们不会为这个参数费心。

**7。**选择多个参数指定多个选择。将此项设置为 true，则允许用户一次从列表中选择多个选项。

**8。**命名点击处理程序。键入“f1”作为单击处理程序的名称。

![1184_2](../Images/d60839b712a1995c48ffedf8a0452d40.png)

**9。**现在，创建两个动态文本字段，如上所示。将第一个文本字段命名为“txt1”，将第二个文本字段命名为“txt2”。

10。从现有图层中选择第一个关键帧，按 f9 打开动作框。插入以下代码:

```
function f1() 

{ 

//to get the item label 

txt1.text = listbox.getSelectedItem().label; 

//to get the multiple item values 

txt2.text = listbox.getSelectedIndices(); 

}
```
