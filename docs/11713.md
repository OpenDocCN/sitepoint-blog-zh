# 商业计划:你真的需要吗？

> 原文：<https://www.sitepoint.com/do-you-need-a-business-plan/>

如果你正在创业，并打算寻求投资者或金融机构的财政支持，一份可靠而全面的商业计划是必不可少的。没有它你连门都进不去。

但是，如果你正在创办一个自己经营、投资和管理的小企业呢？作为一个创业者，你肯定需要一个计划，但是形式、长度和复杂程度会有所不同。事实上，在许多情况下，一份完整的严格的商业计划根本没有必要。

但是等等…在你欢呼商业计划已经死了之前，让我们回顾一下什么情况下你应该花时间去制定一份出色的商业计划。你必须有一个商业计划，如果你:

*   计划申请贷款或向投资者寻求资金支持，
*   将成立并向董事会报告，或
*   和别人一起创业。

如果这些动力不适用于你，那么你可能在没有正式商业计划的情况下勉强度日，仍然可以发展一个成功繁荣的企业。但是你需要一些东西。以下是我称之为“瘦身商业计划工具包”的重要元素。

**使命陈述**

我们都知道使命陈述概述了你的业务目的，但它并不那么简单。一份好的使命陈述不是一个小时就能完成的。要做到恰到好处，需要时间、他人的投入和大量的文字修饰。但是一旦你有了它(并根据你的业务变化进行必要的修改)，它会成为你业务的一个非常有价值的计划工具。

你的使命宣言应该代表你企业的基本价值观和原则，并为你所有的商业活动提供方向。它应该是一个简短的陈述，概括了你的业务是什么，你为什么做你所做的事情，以及你的公司的潜在价值。

有关撰写使命宣言的提示，请访问:

*   如何撰写 Entrepreneur.com 的使命宣言
*   [StephenCovey.com 的商业使命声明](https://www.stephencovey.com/mission-statements.php)

**独特的销售主张**

一个[独特的销售主张(USP)](https://www.sitepoint.com/unique-selling-proposition/) 是瘦身商业计划工具包中的第二个工具，它总结了是什么让你的企业与众不同并对其目标市场有价值。USP 是你业务的基础，不仅能塑造你的品牌，还能塑造你的市场定位、营销技巧以及你与客户的互动方式。

创建 USP 过程的一部分包括瞄准你的目标受众，分析你的竞争对手，直截了当地看看你能提供什么别人不能提供的。如果你花时间精心制作一份相关的 USP，你可以在你所有的营销材料、互动和商业品牌中使用它。配合你的使命宣言，你的 USP 可以让你的商业模式、你做什么以及你为什么这么做变得非常清晰。

**预算**

你工具箱中的第三个关键要素是你的[业务预算](https://www.sitepoint.com/develop-a-business-budget/)。虽然你的预算不一定要包括图表和深入分析，但至少应该包含以下内容:

*   一年的预计业务费用和运营成本
*   对你收入的预测
*   预期的营销预算

为了创建一个有价值的预算，您还需要包含一个列，在该列中您可以输入每个月的实际数字，以便与您的预测进行比较。这种工作预算可以成为持续管理你整体商业财务的有效工具。

有关创建预算的有用提示:

*   [如何在 Entrepreneur.com 制定一个简单的预算](http://www.entrepreneur.com/money/moneymanagement/moneymanagementcolumnistjosephbenoit/article201670.html)
*   为你的小企业制定预算:Intuit.com 的基础知识

当这三个项目结合起来创建一个精简版的商业计划时，会非常强大。但是诀窍是要确保你花了必要的时间在连续的基础上制定三个有效的文件，而不仅仅是起草和忘记它们。
