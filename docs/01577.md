Java 有幸拥有三大 Java 日志框架:Java Util 日志、Log4j 2 和 Logback。当你为你的项目挑选一个的时候，你有没有想过他们的表现？毕竟，仅仅因为选择了一个缓慢的日志框架或一个次优的配置就降低应用程序的速度是愚蠢的。
