# 成功对你意味着什么？

> 原文:[https://www.sitepoint.com/what-does-success-mean-to-you/](https://www.sitepoint.com/what-does-success-mean-to-you/)

前几天，我在电脑上翻一些旧文档，发现了一个关于成功的各种定义的列表。我不记得这到底是为了什么，但我怀疑这与我当时做的一次教练练习有关。

成功是一个非常主观的术语，它对两个不同的人来说并不意味着完全相同的事情。不知道你如何定义生活中的成功会让成功的过程变得更具挑战性。这就像当灯的开关就在你身边时，你在黑暗中摸索着找东西，只要你停止在黑暗中摸索，把注意力集中在灯的开关上。

所以，我想我应该分享一下我列出的成功对我意味着什么(至少当我去年创建它时，它对我意味着什么)，这样我们就可以看看成功的不同含义。我人生中的成功意味着(排名不分先后):
![success](../Images/c712193be42950c27dccd5c535ad412f.png)

*   快乐
*   每天做我喜欢的事情
*   帮助他人做他们喜欢的事情
*   忠于我的价值观
*   做一个好妈妈
*   与我周围的人保持尊重的关系
*   能够为我的家庭做出经济贡献
*   实现财务安全
*   被同行公认为知识渊博
*   有时间享受生活中的小事情
*   不要经常感到压力和不知所措
*   掌控自己的未来
*   追寻我的梦想
*   设定并达到有意义的目标
*   与他人建立真诚的联系

## 你的清单上有什么？

你最后一次思考成功是什么时候，成功对你的生活意味着什么？你对成功的定义与你每天集中时间和精力的地方相符吗？

这样的练习不仅让你有机会重新评估你的优先事项，帮助你明确实现目标所需的方向，还可以作为一种有效的方式来集思广益，解决生活和工作中你可能想尝试不同做法的事情。

花些时间想想你是如何定义成功的，列出你自己的清单，看看它与你的日常生活有什么联系。如果你愿意，请在评论中分享你对成功的定义，这样我们可以创建一个广泛的含义列表，并看看成功是如何因人而异的。
