# 平衡行为:工作与个人生活的平衡可以实现吗？

> 原文:[https://www.sitepoint.com/is-work-life-balance-achievable/](https://www.sitepoint.com/is-work-life-balance-achievable/)

在我们谈论平衡工作和个人生活之前，我应该澄清一下，在我的大脑中，我认为平衡是一种完美的对称。对我来说，平衡意味着所有元素之间完全平等。

从这个角度来看，我认为平衡问题的答案是否定的——没有完美的工作和个人生活平衡这样的事情。就像在生活的其他方面没有完美的平衡一样，因为生活是不可预测和不完美的。

所以，根据我的思考过程，这是否意味着我们这些工作并有个人承诺的人没有希望达到平衡？我当然希望不是！但是有什么解决办法呢？

## 摆脱“平衡”

第一步是使用正确的术语，尤其是对于我们这些倾向于拘泥于字面意义和精确规范的人(好吧，我)。如果我们不能以工作和个人生活的平衡为目标，我们应该为之奋斗吗？

我喜欢把它看作是对可接受的妥协水平的不断追求，因为我认为我们所能达到的最好目标不是平衡，而是给予和索取的水平，以尽可能最好的方式满足我们所有的需求。此外，通过抛弃“工作-个人生活平衡”并采用“工作-个人生活妥协”，我们摆脱了许多因实现无法实现的平衡而带来的压力。它考虑到了这种妥协所带来的非常个人化的方面。

## 一个可接受的工作-个人生活妥协的三个最重要的元素

说到底，你怎么称呼它真的不重要。你不能忽视这样一个事实，你有相反的力量在各个方向推你和拉你。无论是平衡还是妥协，我们都只想感觉自己存在于生活的方方面面，过着满意的生活，让周围的人开心。

为了做到这一点，我们需要做好三件基本的事情，以使我们争取可接受的妥协的努力能够实现。

**优先级**

第一个要素是对你的优先事项进行长期的、认真的和现实的审视。无论你是在外工作还是有自己的事业，有孩子或宠物或年迈的父母，或者有你不愿意放弃的爱好，你都需要能够对生活各个方面的重要性进行排序。

重要的是要认识到你的优先事项会改变，有时会很频繁，如果你不清楚你生活中的哪些部分需要你首先关注，达成一个可接受的妥协将是一场斗争。

**灵活性**

因为你的优先事项会改变，生活会带来意想不到的(有时是不想要的)惊喜，为了适应这些变化，你需要灵活，重新组合和调整你的优先事项，并在必要时改变方向。通过避免一成不变地看待你的优先事项，你将获得适应变化所需的灵活性。

**验收**

实现一个可接受的工作-个人生活妥协的最后一个必要因素是，愿意接受你所达到的妥协水平并不总是理想的。现实是，有些日子比其他日子更好，有些优先事项会比其他更容易满足。

关键是要记住，通过不断的给予和索取，以及在任何给定的时间尽你所能的目标，你可以相信最终一切都会扯平。

你是如何处理工作和个人生活平衡的内在需求的？你觉得可以实现吗？
