# 浏览器趋势，2011 年 4 月:火狐 4.0 击败 IE9，IE7 的市场份额降至 10%以下

> 原文:[https://www . site point . com/Firefox-4-beats-ie9-and-ie7-drops-under-10 %/](https://www.sitepoint.com/firefox-4-beats-ie9-and-ie7-drops-below-10-percent/)

我们上次关注[浏览器市场是在 2011 年 3 月](https://www.sitepoint.com/ie6-usage-below-5-percent-browser-trends/)。当 IE6 下滑到 5%以下时，每个人都发出了小小的欢呼，我们屏住呼吸等待 IE9 和 Firefox 4.0 的发布。

事不宜迟，下面是最新的 [StatCounter 统计数据](https://gs.statcounter.com/) …

该表显示的是市场份额，而不是绝对的使用数据。互联网的使用正在增长，所以理论上，浏览器可以赢得用户，同时失去市场份额。“变化”栏显示市场份额的绝对增加或减少。‘相对’栏表示比例变化，即上个月有 5.6%的 IE6 用户放弃浏览器 <sup>*(yahay！)*</sup> 。

许多个月来，谷歌 Chrome 首次不是最大的赢家。尽管是在 3 月 22 日发布的，火狐 4.0 还是取得了最大的进步，市场份额翻了三倍。现在下结论还为时过早，但 Mozilla 应该会感到高兴。

[微软 3 月 15 日发布 IE9](https://www.sitepoint.com/ie9-final-release-date-time)。它最初的增长并不令人印象深刻，但与 Firefox 4.0 不同，IE9 仅限于 Windows Vista 和 7 平台。在撰写本文时，两种浏览器都没有作为自动更新发布，因此早期采用者将是高级用户和开发人员。这些人拥有 MAC 和 Linux 电脑的比例比一般人高，所以 IE9 可能已经达到了任何人的预期。如果 Windows 更新很快出现，它可能会在 Firefox 上有所斩获。

然而，微软和 Mozilla 不应该自满。所有版本的 IE 和 Firefox 的总份额在 3 月份略有下降，许多用户转向了谷歌 Chrome。在被其他浏览器的营销宣传主导的一个月里，Chrome 的用户增长率是 IE9 的三倍。

最后，还有另一个值得庆祝的小里程碑:IE7 的使用率已经下降到 10%以下的个位数！IE7 的问题被 IE6 的抱怨所掩盖，但我认为它是一个更令人恼火的浏览器。IE6 有一些缺陷，但是这些缺陷都是有据可查的，很容易克服。IE7 解决了一些问题，但也带来了一些令人讨厌的问题，并增加了一个糟糕的界面。它的最终灭亡迫不及待！
