# SitePoint 博客的 3 大变化

> 原文：<https://www.sitepoint.com/3-big-changes-to-sitepoints-blogs/>

我本来打算把这篇文章的标题定为“欢迎来到新的 SitePoint 博客版块”，但是因为你们都已经来过了，而我是新来的，我觉得我只是在欢迎我自己。然而，SitePoint 博客上出现了一些新的东西。有三个变化特别影响到今天刚刚上线的博客。

## 1:新布局

很可能现在你已经注意到 SitePoint 的主页已经有了一点改变。新的设计旨在为您带来及时的内容和更多的内容。博客帖子现在与文章混合在一个干净整洁的空间里，好的内容受到关注，不管它来自哪里。有关新设计的更多信息以及我们认为你会喜欢它的原因，请查看 Matt Magain 的介绍文章。

## 2:新焦点

目前所有的 SitePoint 博客都还在，但是我们会更加关注网络技术新闻和趋势。这意味着每天都要报道网络技术领域的重大新闻，以及对最新新闻的分析和评论。目前的 SitePoint 博客将分为三个主要类别:商业、设计和技术。新闻和趋势类别将发布最新的网络技术行业新闻、分析和趋势，以及新网络应用的评论。

## 3:新作家

这个乔什·卡顿到底是谁？你们中的一些人可能在我的上一次演出中认识我，作为优秀的网络行业新闻和评论博客 [ReadWriteWeb](http://www.readwriteweb.com/) 的首席作家，我在过去的一年里每天都在那里写博客(如果你的 RSS 阅读器中没有 RWW 和 SitePoint 的博客，你真的应该有)。感兴趣的可以看看我的帖子目录。你们中的其他人可能还记得来自 [SitePoint 论坛](https://www.sitepoint.com/forums/)的我，在那里我被昵称为“Bleys ”,是一个长期的顾问和团队领导。你们中的许多人可能从未听说过我。

不管是什么情况，我真的很高兴来到 SitePoint。我的目标是给你带来网络技术行业最有趣的新闻和趋势的高质量博客报道。
