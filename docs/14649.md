# Novell 和 Web 开发？

> 原文:[https://www.sitepoint.com/novell-and-web-development/](https://www.sitepoint.com/novell-and-web-development/)

听起来有点奇怪……然而，Novell 正在成为 Linux 和开源市场上的一股强大力量，早在收购 Ximian (Linux 桌面开发者)和 SUSE (Linux OS)之前，Novell 就已经在各种规模的商业环境中站稳了脚跟。

虽然 Novell 平台从来没有像 Windows 和 Unix 平台那样吸引人的营销，但各地的管理员都在继续称赞它的稳定性和可伸缩性。

重要的是，对于在大中型商业市场培养客户的 web 开发人员来说，您仍然可以发现部署了早在 4.11 版本的 Novell 系统。更有趣的是，以下开源解决方案被移植到 Novell:

*   街头流氓
*   轻量级目录访问协议
*   关系型数据库
*   Perl 语言
*   服务器端编程语言（Professional Hypertext Preprocessor 的缩写）
*   一种数据库系统
*   RSync
*   雄猫

这提供了几乎完整的 web 开发平台，包括支持运行数据库驱动和 Java 驱动的解决方案。

这里有两条途径:1)接触那些对使用 Linux 犹豫不决但拥有 Netware 基础结构的客户，2)将 Linux 引入客户环境的机会，因为他们信任 Novell 名称(即 SUSE Linux Standard 或 Enterprise)。

对一些人来说，大约一年前 Novell 似乎是一个尘封在尘封的书架上的名字。形势已经发生了明显的变化，任何向中型企业市场营销 web 开发的人都应该研究一下新的 Novell 包。

Novell 当然可以在他们的公司网站上找到，他们的开源活动可以在 T2 的 Novell Forge 看到。

作为一个脚注，考虑到电子邮件服务通常在网络提供商的保护伞下，Ximian 的桌面包括 Evolution，它具有 Microsoft Outlook 风格的功能，并且还提供一个单独的 Exchange Connector，使 Linux 桌面上的 Evolution 用户能够参与 Microsoft Exchange Server 2000 和 2003 的协作环境。

当然，Novell 正在将其所有产品与 SUSE 的 Linux 平台和 Ximian 产品集成。
