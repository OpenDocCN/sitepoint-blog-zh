# 在线地图变得浮华

> 原文:[https://www.sitepoint.com/online-maps-get-flashy/](https://www.sitepoint.com/online-maps-get-flashy/)

Flash smarty pants[Paul Neave](http://www.neave.com/lab/)采用了[谷歌地图](http://maps.google.com/)和[微软虚拟地球](http://virtualearth.msn.com/)的公共 API，并构建了自己的 Flash 界面来浏览这些服务*提供的卫星图像。并且，希望 JavaScript 神原谅我，它比这两种服务各自的 Web 界面更流畅、更好用。*

目前，它缺少谷歌和微软 originals 的平面地图、本地搜索功能和其他有用的功能，但作为 Flash 作为前端技术的可能性的演示，它非常有效。特别是，旋转视图的能力在纯 DHTML 接口中还有很长的路要走。

我想知道还有多久，一些有事业心的人会试图用 Flash 制作一个谷歌地球的 2D 克隆版。如果谷歌没有买断这样一个项目，我会感到惊讶，如果它实现的话。
