# 向食物链的上游移动

> 原文：<https://www.sitepoint.com/moving-up-the-food-chain/>

希望本博客的所有读者现在都知道撰写教育和信息文章的重要性，从而让自己成为市场上可信的专家。也许你们中的一些人甚至想自己写一本书。我已经这样做了，这对我在客户面前的地位和我的收入都产生了巨大的影响。

现在我刚刚被一本书引用了。作者布莱恩·卡罗尔突然打电话给我(感谢我的写作)并采访了我。现在我出现在他的书《复杂销售的领先一代》的第 163 页，是“著名的商业建设顾问安德鲁·奈特里希”这本书由麦格劳·希尔出版社出版，将于本月出版。

我喜欢这样。现在看看这种事情对双方是如何起作用的:

–布莱恩的写作工作更容易，因为他在整本书中引用了很多像我一样的人。

–他也从像我这样的人那里得到宣传，因为我们告诉别人我们在书里。

–我因为被他的书引用而获得了更多的曝光率，因为并不是每天都有人在书中被引用。

我所要做的就是和布莱恩通几分钟电话。他为我做了所有的工作。

这是营销计划中的一件小事，但值得注意。
