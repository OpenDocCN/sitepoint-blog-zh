# 你能用一千字节的 JavaScript 做什么？

> 原文:[https://www . site point . com/what-you-can-do-a-kb-of-JavaScript/](https://www.sitepoint.com/what-can-you-do-with-a-kilobyte-of-javascript/)

上周宣布的 [JS1k](http://js1k.com/) 竞赛，要求参赛者用不到一千字节(1024 字节)的代码编写一个 JavaScript 演示。[规则](http://js1k.com/rules)很简单:你的演示必须在当前版本的 Firefox、Safari、Chrome 和 Opera 中运行而不会崩溃。款式不算，所以`eval`扔掉。每人只能提交一份参赛作品，截止日期前提交的最后一份参赛作品将被计算在内，因此可以进行修改和更新。当然，最小化是允许的。

您提交的脚本被放在一个只包含一个`canvas`元素的框架中的一个`script`标记内。到目前为止，大多数提交的脚本都是图形演示。在我看来，这里有一些最有趣的例子:

*   Marijn Haverbeke 提交了一个相当复杂的侧滚游戏，你可以控制三艘不同的宇宙飞船，并试图阻止敌人的飞船到达屏幕的右侧。[演示](http://js1k.com/demo/167)，[来源](http://js1k.com/source/167)。
*   汤姆·博肖文(以及其他一些参与者)实现了康威的生命游戏，包括播放/暂停和速度控制。[演示](http://js1k.com/demo/188)。
*   相当多的人已经实现了 JavaScript 语法荧光笔。
*   乔纳斯·瓦格纳(Jonas Wagner)有我最喜欢的一本书:自然选择进化的演示。他的剧本从随机颜色背景上的一群彩色方块开始。每个方块都有三个“基因”:颜色的 R、G 和 B 成分。每“代”，模拟杀死一些细胞-但不是随机的:颜色与背景最不同的细胞更有可能被杀死。然后，通过结合存活细胞的基因，“繁殖”新细胞来填充空间，每个新细胞都被赋予一个随机突变。这就是全部了。点击进入[演示](http://js1k.com/demo/129)观看方块进化它们的伪装，并阅读他的[博客帖子](http://29a.ch/2010/8/8/demonstrating-evolution-in-999b)看看它是如何完成的。似乎这还不够酷，他在最小化的脚本中使用空白来绘制查尔斯·达尔文的侧面像。

看看完整的[演示列表](http://js1k.com/demos)，看看其他人都做了些什么。

参赛作品将由评审团评判，并颁发奖品(包括一些书籍和 [HTML5 Mobile Pro](http://html5mobilepro.com/) 研讨会的座位)。截止日期是 9 月 10 日，我再问一遍:1KB 的 JavaScript 能做什么？
