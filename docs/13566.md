# Photoshop CS2 网站提示、技巧和窍门

> 原文:[https://www . site point . com/Photoshop-cs2-tips-trick-and-hacks-for-the-web/](https://www.sitepoint.com/photoshop-cs2-tips-trick-and-hacks-for-the-web/)

正如你可能已经在网站上(或者在你当地受人尊敬的书商那里)注意到的，我们的第一本全彩色书刚刚上架——科里·哈夫莱的[Photoshop 选集](https://www.sitepoint.com/premium/books/photoshop-cs6-unlocked-2nd-edition)。

![Photoshop Anthology - inside](../Images/3dd788fc869719350449c20258b3c39f.png)

上周晚些时候，我们将第一个脏兮兮的手指印放在硬拷贝上，它给人留下了非常深刻的印象——大~~(8 ' X10 ')~~(8 " X10 "),有光泽且色彩丰富( **6** 彩色印刷在封面上)！).如果你的大部分 Photoshop 汗水最终在网上，这很可能是你拥有的最有价值的 CS2 书籍。

我个人目前拥有 3 本 Photoshop 书籍(一本 WOW 书籍，一本 CS2 圣经和一本古代视觉快速入门指南),它们在当时都非常有用。然而，公平地说，它们的覆盖面都很广，这意味着我忽略了每本书的整个部分。简而言之，在大多数书籍中,“Photoshop for the Web”是一章的标题——而不是这本书。

有趣的是，在这本书发行之前，很容易找到一本完全专注于在 Photoshop 中使用 LAB color 的书，而不是一本专注于 web 开发的书。奇怪。

这本书还包括一个快速参考卡，给你你的十六进制颜色，一个可视化的字体指南和 Photoshop 快捷方式一目了然，这是第一本书，我们正在使购买完整的 PDF 格式。

我们希望在我写这篇文章的时候，这本书将会是第一本有着先进的 CSS 设计书籍的彩色书籍。看起来真的很酷。

不管怎样，如果你已经在 Photoshop 中虚张声势了，看看科里的 Photoshop 选集。这可能正是你要找的。
