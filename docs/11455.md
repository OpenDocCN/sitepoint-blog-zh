# 你的客户使用他们的 CMS 吗？

> 原文：<https://www.sitepoint.com/do-clients-use-cms/>

大多数客户都会问同样的问题。对许多人来说，第一个问题是*“它的成本是多少？”*(即使你们没有讨论过他们的要求)。下一个问题是*“你能让我在谷歌中找到第一个<术语 X >吗？”*(其中 X 通常是一个常用词，如“商店”)。最后，他们会问*“我可以添加页面和改变内容吗？”*

这最后一个问题可能是最容易回答的。内容管理系统令人眼花缭乱，包括 WordPress、Joomla、MODx、Drupal、TextPattern 和 Frog。CMS 对于任何超过几页的网站都是有用的，你一定可以找到合适的软件来满足客户的需求和 IT 素养。我怀疑许多开发人员提供了一个他们知道的系统，但是根据客户的意愿对其进行了定制。

这是通常的故事。您安装 CMS，集成插件，创建自定义主题，并添加客户端的初始内容。然后，你提供几个小时的培训，并给客户一些有用的建议，包括关键词以及他们从微软 Word 粘贴时不可避免会遇到的问题。在最后一次支付支票之后，你希望他们在网络之旅中一切顺利。

一周后，您注意到客户做了一两处更改，但破坏了格式或删除了重要信息。出于良心，你联系他们，解决问题，并提供一些免费的建议。

然后… *没事*。几个星期几个月过去了，但是这个网站再也没有更新过。

有几个促成因素:

1.  客户忙于经营自己的生意。如果他们不积极进行网上交易，他们就没有什么令人信服的理由来定期更新他们的网站。
2.  大多数人的 IT 技能一般。除非他们经常使用一个系统，否则他们会忘记他们的训练，并对打破一些东西保持警惕。
3.  许多 CMS 有很好的界面，但是它们可能令人畏惧，看起来很专业，使用术语，并且提供了太多的选项。

我敢肯定，许多人认为他们会定期添加内容，但发现它涉及的工作比他们最初怀疑的更多。

但是，如果你的客户避开他们的 CMS，这有关系吗？绝对:你卖给他们的系统不符合他们的要求。很少有客户了解他们的需求，但是一个好的网站开发机构会指导他们选择并推荐最划算的解决方案。

有网络代理提供 1001 个功能，不可避免地获得更多的初始利润。然而，如果客户不明白他们买的是什么，短期收益可能会变成长期损失。他们如何推荐该机构的服务？

下次你评估客户的需求时，问问他们是否真的愿意每周花几个小时来写和更新他们的内容。如果他们不是，也许你应该卖给他们内容审查和持续维护，而不是 CMS 和相关培训。

你的大多数客户使用他们的 CMS 还是你为他们工作？
