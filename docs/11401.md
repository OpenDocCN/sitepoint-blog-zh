# 为害怕承诺的人设定目标

> 原文:[https://www . site point . com/goal-setting-for-commitment-phobes/](https://www.sitepoint.com/goal-setting-for-commitment-phobes/)

随着年底的临近，你很可能会梦想着庆祝并与朋友和家人共度时光。现在也是一个思考未来一年的好时机，以及未来几个月你和你的企业会发生什么。

事实上，当你展望未来时，为什么不把目光放远一点呢？我们中似乎很少有人考虑下一个发票周期以外的事情，但这种方法可以看到我们只是为了过得去而工作，而不是为了特定的目标而工作——为了去某个地方而工作。

当然，我们很多人认为设定目标是一项艰巨的任务。通过说“我想做 X”，我们担心我们会让自己失望(如果我们不能实现 X 会发生什么？).我们想象我们将不得不盲目地遵循这个计划，即使我们后来决定我们根本不想要 X。至少，这是我过去对目标设定的看法。

好消息是:个人和职业目标的设定不必如此。

**自由职业者:8 球背后？**

受薪工人享有许多自由职业者没有的福利。这些可能包括带薪假期、清晰的职业轨迹，以及将部分工资存入退休或退休金基金。这些福利在某种程度上减轻了受薪员工的负担。但是他们也把这些问题抛到了脑后。

因此，许多自由职业者忽视休假(或者看不出他们如何负担得起)，未能为未来留出资金，也看不出他们如何才能设定一个大目标——比如买房或生孩子——并在财务上实现它。如此多的自由职业者最终都是为了生存而工作，而不是为了某个目标而工作。

如果这听起来像你，为什么不在接下来的一年里尝试设定并朝着几个目标努力呢？不，你不会在年底拿到记录你成功或失败的成绩单。如果你在这一年中改变了你的目标，你就不会失败。但是尝试设定目标可以改变你经营企业的方式，帮助你从生活中获得更多。是的，真的。

你的目标是什么？

要尝试设定目标，你需要确定自己的愿望。松散地思考你想要什么是一个好的开始。事实上，我们中很少有人是那种十年后我会在哪里的人。然而，我们大多数人都有模糊的想法，我们可能想花一年时间去旅行，或者有一天有孩子，或者回到每周工作四天，或者总是有足够的钱享受生活。

职业或工作相关的欲望至关重要，但你的个人欲望同样重要。最近，我意识到我想在我职业的一个特殊领域——写作——扩展我的技能，这样我就可以在家做更多的工作。这满足了我在世界不同地方远程工作的个人梦想。但是，我没有被我那些薪水丰厚的朋友们超越，我决定我想要他们从我的工作中得到的同样的福利:我也想要假期和退休基金。

你如何实现这些目标？

一旦你确定了一些你想要实现的事情，选择一两个，更清楚地思考——但仍然是假设，如果这让你更舒服的话——你会做些什么来实现这个梦想。

从专业的角度来看，我的第一步是参加一门适合我想进入的行业的写作课程。从那时起，我想我可能需要获得一些经验，并在这种特殊类型的写作中建立一份工作简历。因此，首先，我需要找到一个体面的，负担得起的学习课程。

从个人角度来看，我想在我去环游世界之前，我会有一个更好的财务状况。这很好，我想——我可以利用这段时间获得新的写作经验来省钱。一想到钱，我就想到了休假和退休基金的目标，就像我是一名受薪员工一样。

**让这一切加起来**

显然，我的目标需要以某种方式得到资助。你的一些人可能也是。当然，还有其他时间方面的考虑(我如何将学习纳入我的时间表？例如)。

在这一点上，你可能会发现看看你上个月左右的支出会有所帮助，从而了解你的钱都花到哪里去了，以及月底你还剩下多少。有些人会称之为预算，但如果这个词让你和我一样感到害怕，你可以把它想象成一个支出快照。

一旦你有了它，你就会知道你每个月能存多少钱——或者至少设定一个你想存起来的目标金额。不管这个储蓄目标有多灵活，如果你能承诺每月至少存一些闲钱，你就能实现你的目标。不错！

当你比较你存的钱和达到目标所需的钱时，你可能会决定减少其他开支，这样你就能存更多。既然你已经有了你的开支快照，你应该能够确定你可以削减开支的任何领域，这样你就可以更快地达到你的目标。

当我迈出这一步时，我发现我面临的最大财务障碍是储蓄:如果我想为未来储蓄，度假并有一个退休基金，我不仅需要减少一些支出，还需要将工作重点放在收入更高的行业。正如你所猜测的，这种认识现在正帮助我制定来年的商业战略。

**改变主意**

如果我上了写作课，发现自己对这个细分行业并不那么热衷，怎么办？如果我攒够了所有需要的钱，然后意识到我不想再去工作度假了，该怎么办？

像这样设定目标的最糟糕的情况是，我会发现自己有一大笔储蓄，但我不想再花在目标 x 上了。就我个人而言，我认为这并不是一个糟糕的处境。到那时，我肯定已经想出了另一个目标，如果我决定了，我可以把钱投到那个目标上。

我想另一个最坏的情况可能会出现，如果我发现我不能在我想从事的行业工作——我的写作风格不适合那个市场。这是一个令人畏惧的想法，但我认为尝试并失败总比根本不尝试要好。当我意识到这不会成功的时候，我一定会获得其他的经验，无论如何，这些经验会提供新的、有趣的机会。

这就是我如何克服对设定目标的恐惧，并开始思考我的人生目标的方法。你有没有想过你来年的目标？
