# Chrome 的市场份额超过 10%

> 原文：<https://www.sitepoint.com/google-chrome-market-share-10-percent/>

Chrome 6 于本周发布，但对网页设计者和开发者来说，还有一些更重要的消息。根据 [StatCounter 的统计](https://gs.statcounter.com/)，2010 年 8 月，谷歌浏览器的市场份额为 10.76%。这比前一个月的 9.88%有所增加。

0.9%的增长听起来可能不多，但它占了数百万用户，对于 2 岁的浏览器来说是一个重要的里程碑。在上个月的文章[当前浏览器市场份额和趋势](https://www.sitepoint.com/current-browser-market-shares-and-trends-august-2010/)中，我报道了双位数将很快实现，但上升速度似乎已经加快。

值得注意的是，浏览器的使用数据从来都不准确，自然会有波动，但 Chrome 的增长从未偏离其上升路径。

| 浏览器 | 2010 年 7 月 | 2010 年 8 月 | 变化 |
| 工业管理学(Industrial Engineering) | 52.69% | 51.34% | -1.35% |
| 火狐浏览器 | 30.69% | 31.09% | +0.40% |
| 铬 | 9.88% | 10.76% | +0.88% |
| 旅行队 | 4.09% | 4.23% | +0.14% |
| 歌剧 | 1.91% | 1.88% | -0.03% |
| 其他人 | 0.74% | 0.70% | -0.04% |

Chrome 的收益是以 IE 为代价的。这并不奇怪，但在前几个月，IE 的使用仍然相当稳定——如果不是略有增加的话。八月是 IE 市场份额一年来最大的月跌幅。

这些数字会让谷歌高兴，但在 Chrome 开始挑战 IE 和 Firefox 之前，谷歌还有很长的路要走。然而，该公司一直在大力宣传这款浏览器，很少有人会反对它。
