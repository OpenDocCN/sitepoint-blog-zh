# 创业中的法律问题——入门

> 原文:[https://www . site point . com/legal-issues-in-starting-up-a-primer/](https://www.sitepoint.com/legal-issues-in-starting-up-a-primer/)

在上一篇博客中，malikyte 问了几个关于创业的问题。一个是关于法律问题。

根据我对美国法律的理解，这里有一个快速入门:

在开始一项新业务时，第一步是选择你的法律结构。你基本上有几个选择。典型的法律结构有两个考虑因素:税收和法律责任。

1.独资经营。在这种最简单的情况下，所有产生的收入都流向你个人，并按普通收入税率征税，包括自雇税。和 C 类公司一样，所得税是一次性的，而不是两次，这是一个主要的优势。这种形式的所有权的另一个优势是，它的设立几乎是不经大脑的简单，如果你没有合并就开始创业，这是默认设置。缺点是你要承担 100%的法律责任(例如，你不能躲在公司的面纱后面)。我建议你从独资开始。这样，你可以节省建立公司的法律费用，并且在确认你有一个真正的生意时，可以花更多的钱在营销上。一旦你有了一个真正的企业，就转变成一个公司。

作为独资企业，你仍然需要在当地政府注册。在美国，这通常意味着花钱购买许可证，并在 1-2 份商业出版物上刊登广告，说明你已经开办了独资企业。你当地的城市职员会告诉你需要做什么。成本:如果你按照要求的步骤做，可能要花几百美元。

2.公司。它们是 C-Corps 和 S-Corps，以及现在越来越流行的 LLC(有限责任公司)。在所有的公司中，你都享有有限法律责任的优势，因为公司被认为是一个实体，所以别人很难因为你的个人持股而起诉你(不像独资企业)；但是，任何人都可以随时起诉你，所以这种保护可能没有你希望的那么强。

有了 C 类公司，你就有了各种各样有创造力的股东阶层和类别的优势。然而，你要交两次税:一次是公司税，另一次是股息税。

S 公司解决了双重征税问题。所有收入直接流向你，你仍然得到责任保障。你只能有一种股票，20 个股东拥有一个 S-Corp。

有限责任公司拥有股份有限公司的所有优势。然而，如果你和合伙人一起做生意，你会得到更多的灵活性。这是因为你可以指定哪一方获得哪一项扣除和哪一项收入——而不考虑相对所有权。你不能在 S-Corp 中做到这一点(例如，平等的合伙人被同等征税)。

如今大多数新企业都以有限责任公司的形式出现。

3.伙伴关系。我会避免合伙，因为合伙人仍然要为企业承担 100%的法律责任。因此，如果一方做了非常愚蠢的事情，你可能要负 100%的责任，包括你的个人资产。

所以，我的建议是:从独资开始，尽早保持简单。一旦你确定你有一个健全的企业，以后再合并。注册为有限责任公司或 S-Corp 以避免双重征税。如果你想上市，那就转投更复杂的 C-Corp 吧。

咨询律师和会计师，因为我既不是也不知道你的特殊情况。但你必须与双方都谈，以涵盖税收和法律责任问题。

最后，如果你和其他人一起做生意，解决一大堆关于所有权、责任、各自出资多少等问题。我已经在别的地方讨论过了。
