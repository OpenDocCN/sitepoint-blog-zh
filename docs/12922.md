# 我爱你 AMD 皓龙双核先生！

> 原文：<https://www.sitepoint.com/i-love-you-mr-amd-opteron-dual-core/>

你很早就在网上教育中了解到，性能对你的网站产生流量至关重要。但是随着你的网站成长，事情变得更加复杂，它的真正价值往往会变得模糊不清。你很容易被更多有趣的东西所诱惑，比如功能性、可用性、美学和界面设计。

但是这里有一个简单的原因，让你永远不会忘记为什么网站性能如此重要。

在过去的几周里，我们升级了我们的博客，并把我们的论坛移到了 vBulletin 的最新版本。很快，我们的系统管理员开始关注服务器上不断增长的压力。我不是专家，但即使是我，也能从我看到的一些利用率图表中看出巨大的变化…

转折点是周三，我们提高了一些长期计划的优先级，并对我们的服务器硬件(几个闪亮的新 AMD 皓龙双核 CPU)进行了快速升级。)

效果立竿见影，引人注目。我们的响应时间比升级前更快了…

结果:第二天，砰，我们论坛的流量增加了 33%!

所以你可以随心所欲地优化搜索引擎，进行 A/B 分割测试，开展广告活动，确保你的设计是可访问和可用的，但如果你的服务器不能处理流量，你就等于把访问者拒之门外。如今有了如此多的选择，他们不会在寒冷的室外等待，他们会去拜访你的竞争对手。
