# 保持清醒的五大秘诀

> 原文:[https://www.sitepoint.com/top-5-tips-for-staying-awake/](https://www.sitepoint.com/top-5-tips-for-staying-awake/)

根据我的观察，我们 web 开发人员是一群相当夜猫子的人。我们努力工作到深夜，构建下一个大型 web 应用程序，努力满足客户的最后期限，或者摆弄一些让我们都兴奋不已的新技术。

这让我想知道，我们这个行业的总产量有多少发生在半夜，是什么支撑着我们熬过那些凌晨时分？它是由强迫性行为的纯粹神经能量推动的，还是你可以做一些具体的事情来提高你的表现，而不仅仅是身体上的睡眠冲动？

对我来说，两者兼而有之——一部分是痴迷，一部分是实际的例行公事——只是在半夜更安静，没有电话，分心的事情也更少，这真的有助于我保持专注。

但是我也做了一些简单实用的事情来帮助我保持状态。我只能从个人经验来说，但是在这个前提下，我列出了我用来应对睡眠不足的小贴士。

1.  **喝很多咖啡**

    尽管你从咖啡中获得的短期快感很大程度上是心理作用，但几个小时后的效果是不可否认的。咖啡因是一种兴奋剂，咖啡中的咖啡因含量丰富，远远超过茶或可乐。适当的过滤咖啡是最强的，也是最好的，但速溶咖啡仍然有效；虽然我个人不能忍受速溶咖啡，如果没有真正的咖啡，我宁愿喝茶。

    也许你甚至可以考虑投资一台真正的咖啡机。我们在 SitePoint 总部有一个，它得到了很好的利用。

2.  ***喝很多咖啡！**T3】*

    就像<q>你不谈论搏击俱乐部</q>一样，这条建议非常重要，值得提及两次。当然，风险在于咖啡因上瘾是递归的——它自称为自我维持的循环:你喝咖啡是为了保持清醒，但这阻止了你正常睡眠；所以第二天你累了，所以你喝咖啡来保持清醒，但是…等等！

3.  合理饮食，不要只吃垃圾食品

    好的食物是让你的引擎运转的汽油，就像低级燃料会降低你的汽车的性能一样，劣质食物也会降低你的身体和大脑的性能。如果你想让尽可能多的气缸燃烧起来，你需要吃像样的正餐，喝大量的液体，比如水和果汁(以及所有的咖啡！).

    你不想做的是花一整天或一晚上的时间吃薯条、巧克力和甜甜圈(嗯…甜甜圈)，不管这有多诱人。不可否认，由糖引起的精力旺盛会带来即时的满足感，但这种满足感不会持续很久，也不会让你坚持下去。随着时间的推移，它还会让你逐渐感到更加恶心，所以要吃像样的食物。

4.  **避免打盹**

    许多人推荐强力小睡作为快速提神的方式，然而我发现它们经常会产生相反的效果——要么你一小时后醒来，感觉昏昏沉沉，和以前一样累，只是你浪费了一个小时；或者，你的身体接管了你的工作，结果你睡了三四个小时，这时你最好还是上床睡觉。

    如果你确实开始觉得自己睁不开眼睛，那就站起来，四处走走，甚至出去散散步；呼吸点新鲜空气，抽根烟什么的。再来一杯咖啡。那种打瞌睡的感觉很快就会过去。

5.  **呼吸大量新鲜空气**

    炎热、潮湿或闷热的环境肯定会让你昏昏欲睡。你需要保持警惕，流通的新鲜空气会帮助你做到这一点。打开窗户，如果天气太冷，多穿几层衣服，而不是把暖气打开——冷空气对你的呼吸更好，有助于你保持清醒。

好吧，实际上只有四个，但是第一个值得两个冠军！然而，我还有一个不太明显的建议:

6.  **允许自己有创造性的分心**

    你可能认为应该不惜一切代价避免分心，但我认为那不一定是真的；你想要避免的是*不想要的*杂念，但是杂念本身不一定是不想要的。这就是许多公司在追求更高生产力的过程中，没有看到更大图景的地方，因为他们对个人互联网使用有如此严格的内容政策。

    给自己 20 分钟时间在脸书四处逛逛，看看邻居，或者做其他任何有助于放松的事情，你会发现自己充满活力，为接下来几个小时的工作做好了准备。像这样短时间的放松会有一个有益的效果，让你把事情从脑海中抖出来，然后更客观地回到这些事情上来。

这些建议中的任何一个都有风险，分心可能太有吸引力，让你离开工作的时间远远超过你的预期；但是，嘿——当你推动你身体的自然节奏时，你就抓住了这个机会。一点点纪律总是必要的！
