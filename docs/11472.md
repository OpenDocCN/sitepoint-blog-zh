# 合同谈判 101

> 原文：<https://www.sitepoint.com/contract-negotiation-101/>

乔治娜是一名职业作家，住在澳大利亚墨尔本附近。在接下来的几周里，她将分享作为一名自由职业者的心得和经验。

你有多擅长合同谈判？当他们得到一份合同时，我认识的许多自由职业者都会签字，不管他们是否对合同条款满意。

他们通常花了很长时间才进入合同阶段，他们可能已经在讨论最后期限(并且不愿意浪费时间试图从客户那里获得更好的条款)，他们可能认为与客户在合同条款上的“不一致”可能会在工作一开始就给人留下负面印象。

但是你签的合同支撑着整个工作。订约过程值得小心对待和认真对待。你需要彻底。你需要对结果感到满意。以下是我用来评估工作合同的流程。

**将合同视为首次报价**

很容易将客户起草的合同视为他们的“报价”，但这种观点限制了你的谈判能力。为什么不把这份合同视为他们的“第一份报价？”这就给了你回旋的空间，或者对任何你不是 100%满意的地方提出问题。

一旦你接受了这种心态，你就可以开始客观地阅读了。当我看合同时，我通常会拿出纸和笔，这样我就可以记下任何我不明白或需要质疑的地方。

**从 go 读到 woah**

法律术语并不是最精彩的阅读，但是不管它看起来有多无聊，你必须从头到尾阅读合同。

如果你已经和你的客户讨论了这份工作，它的截止日期，可交付成果和付款时间表，你首先要做的是确保这些协议在合同中有详细的说明，并且你对这些内容感到满意。如果合同中没有你们讨论过的内容，或者一些细节没有反映出你们认为已经讨论过的内容，那就把它们记录下来。你会想和你的客户讨论这些。

当你阅读的时候，留意那些难以理解的语言，矛盾之处(这些经常会在标准或基本合同随着时间的推移被修改时出现)，以及任何你觉得奇怪的东西。

信不信由你，这个过程会很有收获。就在你翻到第 15 页，感觉自己快要昏昏欲睡的时候，你遇到了一些奇怪的异常情况，让你对项目中你无法控制的部分负有法律责任，你花在阅读合同上的时间开始变得非常值得。

当你阅读时，记下任何异常，以及任何你需要澄清的地方。不要羞于与客户确认合同的细节——安全总比后悔好。

**什么是公平？**

一旦你读了你的合同，你就会很清楚你是否满意，或者你是否有需要和你的客户一起修改的地方。但也可能有一些你不确定的地方。如果你发现你在质疑合同某些方面的公平性，相信你的直觉。不要忽视你的不安——找个人谈谈。

一位导师、值得信赖的同事或法律顾问可能会告诉你某个条款是否标准或似乎不合理。但是，如果你想得到可靠的建议，就去寻求专业知识——当涉及到你工作的合法性时，不要依赖朋友的推荐。

当然，并不是每一个条款都需要争论——你可能会对某些条款的公平性有一些小意见，你很乐意自己和客户解决。

**谈判**

好了，你已经有了问题清单。在你和客户讨论这些之前，考虑一下你对每个项目的理想结果是什么。然后确定你能接受的最小妥协。只要其他要求得到满足，你可能愿意在某些问题上完全让步；你不可以。但是在讨论争论点之前，你需要确定你的极限，这样你就不会在谈判中不满意。

虽然我通常对与客户讨论合同感到紧张，但我想不出客户不渴望找到一个适合我们双方的解决方案的时候。如果有人有兴趣与你合作一个项目，他们通常会渴望达成一个大家都满意的协议，尽管当然可能有一些点他们必须保持灵活，例如，由于公司政策。

举个例子，我的一个客户以 90 天为周期付款。该组织是一家大银行，我不得不接受现有的付款条件。

重要的是要记住，提供给你的合同不应该被视为一个现成的交易。合同草案是你的机会，以确保协议双方对协议条款感到满意，你和你的客户真的在同一页上。
