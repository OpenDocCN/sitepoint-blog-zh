# 你发挥你的长处了吗？

> 原文：<https://www.sitepoint.com/do-you-work-to-your-strengths/>

我们目前正在遭受气象学家们喜欢称之为“天气炸弹”的袭击。在世界上的大多数地方，你可能会称之为风暴，但这里相当温和，所以他们喜欢将任何稍微不寻常的事情戏剧化。这种特殊的天气炸弹使得我和我 18 个月大的双胞胎不可能出去，所以我们正在努力扩大我们的室内活动。昨天我决定去做橡皮泥。家庭主妇已经做了几十年了。会有多难呢？显然，对我来说太难了。在用家里所有的盐制造出一种沸腾的红色粘液后，我决定早上去趟玩具店。有些人被雇来做我不能做的事。

正是因为这个原因，当我们的洗衣机在周末爆炸时，我决定不再尝试自己修理它。结果我还是结婚了。

我想谈的就是这个发挥你的长处的话题。

我们的[首页调查](https://www.sitepoint.com/)询问你的网络工作主要涉及哪些因素。你参与平面设计，前端或后端编码，还是你被期待做很多？现在，结果表明，几乎一半(49%)的人被期望做所有的事情。随着最近可获得的工作数量下降，走上自由职业道路的人数急剧增加。对许多人来说这是一个好的选择，但是这是否意味着你必须在你制作的网站的标准上妥协呢？

我喜欢 SitePoint 论坛的一个原因是它打开了大门。以前从未使用过 WordPress 的人可以来论坛问一个初学者问题，然后得到答案。如果你需要一些代码的帮助，你可以得到它。如果你需要知道在哪里可以找到图形教程，你可以问。

因此，请确保您充分利用这一惊人的资源。如果你已经潜伏了一段时间，为什么不在今天注册呢？我知道你们都听过，但唯一愚蠢的问题是没人问的问题。我们不介意重复自己，这是一个论坛的本质。我们所要求的是你先做一个搜索——有人可能曾经在和你完全一样的地方绊倒过。查看每个论坛顶部的帖子——它们包含丰富的信息，很可能会回答任何初学者的问题。

![SitePoint forums](img/8a35299504d077ca51c6e30a20fb2a70.png "sitepoint-forums")

在此期间，请查看我们的 [Golden Post 奖励计划的第 3 周获奖者。](https://www.sitepoint.com/forums/showthread.php?goto=newpost&t=678753)只剩一周了，所以如果你想买一本你最喜欢的 SitePoint 书籍，你最好开始着手做吧。

当金榜结束的时候，我会发起一场新的比赛。如果你认为我这个月很兴奋，请注意你的帽子！六月，我将掀起一场真正的狂潮。但是现在我打算就此打住。

关于疯狂的话题，我们最近有点儿兴奋。Twitter 是一个神奇的工具，但也是一个非常容易被滥用的工具。所以我直截了当地告诉你。我们是不是过度发微博了？
