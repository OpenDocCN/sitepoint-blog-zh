# 如何虚拟处理客户支持请求

> 原文：<https://www.sitepoint.com/how-to-handle-client-support-requests-virtually/>

虽然自己当老板的好处之一是你可以在任何地方工作，但这也给你的客户提供日常支持带来了挑战。

即使您没有正式提供技术支持服务，也可能会有这样的情况:您需要引导客户使用定制的应用程序，在您刚刚设置的 CMS 上对他们进行培训，或者只是帮助他们在电子邮件客户端中配置新的电子邮件地址。如果您在地理上不靠近客户所在地，或者不想提供现场服务，您将需要一个虚拟培训和支持流程。

为了帮助培训您的客户使用您为他们创建的工具，您可以编写说明手册，并在电话会议期间指导客户完成各个步骤。但是很多时候，如果你有一种直观的方式向客户展示他们需要知道的东西，会更有效率，也更容易教。实现这一点的三种方法包括:

*   使用屏幕截图工具，通过直观的操作方法丰富您的说明手册
*   通过在线会议工具共享您的屏幕
*   通过远程桌面服务访问客户的电脑，帮助他们适当地配置系统

## 使用屏幕截图工具

如果你为你的客户提供指导，你能做的最好的事情之一就是包括每个步骤的截图，这样你就可以展示给他们看，并告诉他们如何做某事。图片可以让你的解释更清晰，更容易复制，减少你被问到的问题。

屏幕截图工具还可以帮助您处理一些更常见的支持请求。如果你的客户有并且知道如何自己使用截图工具，他们可以把他们在屏幕上看到的引起混乱的东西拍下来，然后发给你寻求帮助。这可以大大减少您在尝试查明和解决他们遇到的问题时来回奔波的时间。

这里有一个截图工具的[列表，可以帮助你完善用户指南，让日常支持变得轻而易举。](https://www.sitepoint.com/30-screen-capture-tools/)

## 通过在线会议工具进行协作

对于可以在单次会议中处理的有针对性的培训，在线会议工具可以是教客户如何做某事的好方法。其中许多工具允许您共享您的屏幕或访问客户的屏幕，以便您可以指导客户完成每个步骤。

在线会议工具的其他功能，如视频和音频录制，也可以提高培训会议的效率。请访问 [17 个促进协作的在线会议工具](https://www.sitepoint.com/online-meeting-tools/)获取值得探索的工具列表。

## 远程连接到客户端电脑

虽然对于需要基本的一次性培训或支持的情况来说，这可能是多余的，但有时您可能需要访问客户的计算机。您可能需要配置应用程序、处理安装或提供实时故障排除，访问客户的计算机可能会有所帮助。

您甚至可以提供需要定期访问客户计算机才能提供的服务，如日常计算机维护。在这种情况下，能够远程连接到客户的计算机可能是您日常活动的一部分。

这里有几个[远程访问工具](https://www.sitepoint.com/remote-desktop-access-client-support/)的概述，可以快速连接你到客户端的桌面。
