# 谷歌将 StarOffice 从 Pack 中删除；接下来是火狐？

> 原文:[https://www . site point . com/Google-drops-star office-from-pack/](https://www.sitepoint.com/google-drops-staroffice-from-pack/)

据 PC Pro 报道，![](../Images/b0cd195f1c095c02ddc0844b0b4aaba7.png "google-logo") [Google Pack](http://pack.google.com/) ，一个为 Windows 用户提供的免费互联网和生产力软件下载包，神秘地将 Sun 的 [StarOffice](http://www.sun.com/software/staroffice/index.jsp) 套件从其产品线中删除。该软件包包括许多谷歌和第三方软件程序，包括谷歌桌面、火狐、Skype 和诺顿安全扫描，此前包括微软 Office 替代软件 StarOffice。

StarOffice 基于开源的 OpenOffice.org 程序，也与谷歌自己的在线办公应用套件 Google Docs 竞争。谷歌是否试图将用户推向自己的在线办公工具套件？也许吧，尽管目前这个包似乎没有包括任何种类的 Google Docs 应用程序的桌面启动器(通过 Gears 插件支持离线访问)。

PC Pro 认为，谷歌也可能对微软首席执行官史蒂夫·鲍尔默最近的评论做出反应，他最近对谷歌网络应用的威胁嗤之以鼻，称“我们今天比谷歌文件和电子表格有更好的竞争。坦率地说，我们面临来自 OpenOffice 和 StarOffice 的更多竞争。”

尽管他的公司最终计划将 Office 带到网上，这清楚地表明微软感受到了来自其互联网对手的压力，但鲍尔默可能至少部分是正确的。几周前,我们在【】报道称，目前的数据显示 OpenOffice.org 比谷歌更有可能成为微软办公应用统治地位的竞争对手。然而，谷歌在升级和改进其 office 套件方面表现出的灵活性可能会让微软暂停一些，至少在内部是这样。

如果把人们推向他们自己的工具套件是谷歌正在玩的游戏，我们预计有可能我们会看到 Firefox 最终也从谷歌包中退出，并被 Chrome 取代(至少一旦该浏览器退出测试)。请记住，谷歌 Chrome ( [最近报道](https://www.sitepoint.com/early-stats-google-chrome-hurts-firefox-not-ie/))是专门为运行网络应用而打造的，因此谷歌推动人们在自己的浏览器中运行网络应用套件是有意义的。
