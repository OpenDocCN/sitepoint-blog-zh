# 地下 PHP 和 Oracle 手册

> 原文:[https://www . site point . com/the-underground-PHP-and-Oracle-manual/](https://www.sitepoint.com/the-underground-php-and-oracle-manual/)

[克里斯·琼斯](http://blogs.oracle.com/opal/2006/06/12)刚刚宣布了 [PHP 和 Oracle 手册](http://www.oracle.com/technology/tech/php/pdf/underground-php-oracle-manual.pdf) (PDF)的出版，从高速目测来看，它很好——基本上告诉了你需要知道的一切，以便能够用 PHP + Oracle 做有用的事情，但几乎没有假定的知识。

事实上，它似乎适合典型的 LAMP 开发人员——例如，有一个关于“在 Debian、Ubuntu 和 Kubuntu 上安装 Oracle XE”的部分，加上关于“限制行和创建分页数据集”和“自动增加列”的部分，表明读者可能会有 MySQL 体验。

也许有一个附录(我没有找到参考资料)会将读者引向 [SQL Developer](http://www.oracle.com/technology/products/database/sql_developer/index.html) ，这是 Oracle 作为一个基于桌面的开发工具提供的一个相当新的免费产品——地下手册侧重于 Oracle 的一个基于 web 的接口——逻辑上相当于 [phpMyAdmin](http://www.phpmyadmin.net/) ,我猜这是一个很好的起点，但是如果你必须做真正的工作，SQL Developer 可能是一个更好的选择。

无论如何，鉴于 [Oracle XE](https://www.sitepoint.com/oracle-10g-xe-and-php/) 是免费的(开发人员可以免费使用)和本指南，如果您想了解 Oracle，几乎所有您需要的。
