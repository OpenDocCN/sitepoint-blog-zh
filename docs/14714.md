# 推广和销售您的 CF 标签——入门

> 原文：<https://www.sitepoint.com/promote-and-sell-your-cf-tags-a-primer/>

*…发帖人* [大卫·梅德洛克](https://www.sitepoint.com/forums/member.php?u=24385) *:*

您是否考虑过开发一种革命性的定制标签，CFC 或 CFX 标签？也许它是一个超级灵活和易于安装的内容管理系统，或者是一个 CFX 标签，允许某人动态创建 zip 文件。无论你已经开发了什么，或者计划开发什么，你都应该研究各种各样的促销方式，以最大化你的知名度和销售额。

首先，你可能想从阅读由 Yau Wei Liang 在 SitePoint 上的[销售小本软件](https://www.sitepoint.com/article/sell-software-shoestring)开始。这是几年前的事了，但这些信息至今仍然有效。

一旦你开发了你的脚本，这里有一些建议来促进它:

首先，您需要为您的脚本创建销售页面。如果你经营一个销售(或将要销售)其他产品的网站，你会希望创建一个销售网页，专门介绍你正在销售的产品。在销售页面上，你需要访问者可能想要的关于你的脚本的所有信息。如果他们不能快速方便地获得他们需要的所有信息，他们会转向他们在谷歌上找到的下一个项目。您需要包括:

–它是自定义标签、CFC、UDF 还是整个应用程序？

–需要什么版本的 ColdFusion？是否支持 CF 5，CF MX，CF 3.0？这是至关重要的信息。

–它使用数据库吗？什么样的数据库？访问？SQL Server？甲骨文？用户需要知道你的脚本要求他们访问什么。

–它是否使用 CFFILE、CFDIRECTORY、CFHTTP 之类的标记，或者通常会被虚拟主机禁用的其他标记？你不必放弃你的代码，但是要确保用户知道他们在做什么。

–安装有多容易？您甚至可以将安装说明放在一个页面上，并从您的销售页面链接到该页面。这将让他们在购买后和安装时有地方返回。他们在你的网站上花的时间越多越好。他们可能更有可能把你推荐给其他人，或者以后购买其他产品。

–最重要的是:它支持哪些功能？列出应用程序支持的每个特性的详细列表。如果是自定义标签、CFC 或 UDf，提供使用示例，这样你的访问者就可以看到使用你的标签是多么容易。

–在容易找到的地方提供价格。你必须告诉用户价格。但是，不要只是告诉他们“这个 CMS 是$XXXX。”告诉他们这个 CMS 会给他们带来什么好处。通过关注你的应用程序的好处，你将注意力从价格上转移开，你将注意力集中在说服他们无论他们支付什么价格都是完全值得的。

你甚至可能想创建一个[产品包装盒图片](https://www.sitepoint.com/forums/showthread.php?t=109308)放在你的销售页面上。如果你不擅长图形，你可以找一个人以低廉的价格为你做这项工作。或者你可以用你现有的东西来支付到期费用，直到你完成几笔交易，然后雇人帮你做这件事。将你的产品作为一个专业开发的脚本来展示是非常重要的。人们不会愿意花大价钱去买那些看起来没有得到很好发展或者看起来对它的支持不够的东西。

你应该考虑的其他事情 _:你的产品是否有良好的文档记录？如果不是，就别想卖了。如果文档记录得很好，文档记录还能更好吗？例如，就用户如何在他们的站点上实现您的脚本编写案例研究。写一些关于如何用你的脚本做一些事情的文章，这些事情对别人来说可能没有对你来说那么明显。请记住，你写了剧本，所以你知道它的能力。别人没有，你的工作就是向他们展示你的脚本是多么强大和灵活。

另外，在网上搜索你能找到的每个脚本目录，并把你的脚本列在那里。像 [HotScripts](http://www.hotscripts.com) 或 [DevScripts](http://www.devscripts.com) 这样的东西真的可以增加你的流量和收入。就我个人而言，我绝不会付费加入其中一个，因为有这么多免费的列表。你可以选择获得赞助列表来增加你的知名度，但如果你预算紧张，你可能决定不这样做。

现在，是时候卖掉你的剧本了。最好的方法是通过互联网使用信用卡或支票。如果你不想马上完成建立一个商业帐户和一个安全服务器的过程，你应该考虑几种销售你的脚本的方法:

–接受 PayPal。许多人已经注册了 PayPal，现在 [PayPal 不再要求注册](https://www.sitepoint.com/forums/showthread.php?t=152866&highlight=paypal)，通过他们接受支付变得更加容易。而且费用也很合理。(虽然我记不太清了，但我确定我的 PayPal 账户只收了我不到 5%的费用……)

–使用第三方，如 CFTagStore.com。你注册，上传你的脚本，看着你的销售开始进来。他们负责商家账户和信用卡处理，并向你发送支票或贝宝支付网 30。唯一的缺点是他们收取高达 15%的费用(另外，如果你让他们通过 PayPal 给你汇款，你也要支付这些费用)。这可能是获得知名度和最初几笔销售的好方法，但如果你的产品变得非常成功，你可能不想支付这种佣金。

–您也可以寻求 PayQuake、2Checkout 或其他第三方支付处理商来处理这些支付。

(请注意，我与上述任何公司都没有任何关系。我只是提供这些公司的客观观点和销售你们产品的方法。)
