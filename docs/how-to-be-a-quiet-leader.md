# 如何成为一个安静的领导者并产生巨大的影响

> 原文：<https://www.sitepoint.com/how-to-be-a-quiet-leader/>

有许多不同的[领导风格](http://en.wikipedia.org/wiki/Leadership#Leadership_styles)与领导者的个性和性格直接相关，更多的是几种不同哲学的结合。然而，许多领导风格可以通过安静的方式成功实施。

## 安静领导的必备条件

安静的领导者是强大的。他们注重行动而不是言语，能够以独特的方式激发激情、鼓励主人翁精神并培养忠诚度。但是成为一个有效的安静的领导者并不容易。如果你想成功地悄无声息地领导，有很多因素会起作用，包括:

*   赢得团队的尊重
*   展示自信，但不过度自信
*   善解人意，富有同情心，心胸开阔
*   横向思考，不要分等级
*   具有可爱和可联系的个性
*   平易近人，容易交谈

## 如何成为一个安静的领导者

对于那些已经具备分析能力和内向行为的人来说，安静的领导可能更容易，但是任何人都可以通过采取一些有意识的行动来采用安静的领导风格。

### 听

一个安静的领导者最有价值的品质之一是他/她倾听和听到别人说什么的能力。一个安静的领导者不是暴君，他以“我的方式或公路”的态度领导。更重要的是让每个人都有机会做出贡献，分享过程，并对结果拥有所有权。

有效的倾听不仅意味着给团队成员一个谈论和分享建议、观点和想法的机会，还意味着每个建议都会得到考虑和尊重。

### 放下自我

安静的领导者缺乏优秀领导者固有的东西——夸大的自我。当你想到“领导者”，你可能会想到“大声”，但这两个词肯定不是同义词。事实上，很多时候，数量来自于过度自信、竞争的天性和满足自我的内在需求。

如果你能在不直接将成功或失败与自我价值联系起来的情况下进行领导，你就能降低音量，同样(或更)有效地进行领导。

### 跟随你自己的领导

对所有类型的领导者来说，尤其是安静的领导者，一个好的策略是永远不要让你的团队做你自己不会做的事情。其次是遵守你要求其他人遵守的团队基本规则和指导方针。

如果你以同样的标准和责任对待你的团队，你会更快赢得他们的信任和尊重。任何形式的有效领导的必备之物。

### (在困难的情况下)保持冷静

安静的领导者不会在事情出错时大发雷霆；他们能够在危机时刻保持冷静和镇定。

作为一个安静的领导者，保持冷静也意味着能够接受团队的建设性批评，思想开放，并能够承认自己的错误。

## 你有多大声？

我不一定认为安静的领导者总是比喧闹的领导者更有效，但我确实认为安静的领导在某些情况下会更成功。一旦你和你的团队建立起那种程度的尊重，你真的不需要大声说话就能被别人听到。这就是安静领导的力量。

但是什么对你更有效呢？你是安静地领导还是发现量的增加得到更好的结果？
