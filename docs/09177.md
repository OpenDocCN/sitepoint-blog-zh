## 1.多 Ajax 输入

这个插件将普通的文本输入框变成了“多输入”，可以有多个值，这些值可以由 AJAX 自动建议。它们也可以在运行中被分类和删除。
![Multi-Ajax-Input.jpg](../Images/13cfa8f46ff8c010d86b6f02a65e651a.png)
[来源](http://www.thimbleopensource.com/tutorials-snippets/multi-ajax-input)

## 2.安全呼叫

一个工具插件，允许你调用函数/获取嵌套在对象内部的成员变量，而不用检查对象的路径是否存在。
![Safe-call.jpg](../Images/f9953dad084f58bdd161e920af37e3eb.png)
[来源](http://www.tomrel.com/)

## 3.DL 到 tab

这是(目前)一个非常初级的插件。它可以将一个正确构建的 DL(由交替的 DT 和 DD 元素组成)转换成一个水平制表符界面。
![DL-to-tab.jpg](../Images/1eb16af6b7aa7392113a324ef6dd18ef.png)
[来源](http://plugins.jquery.com/project/dl-to-tabs)

## 4.jquery.stackShuffle

一个 jQuery 插件，可以堆叠一堆图像，并以幻灯片形式播放。
![jquerystackShuffle.jpg](../Images/eadd388d7f78caf4b895fbdf44456b54.png)
[来源](https://github.com/scronide/jquery.stackShuffle)

## 5.tabsLite

jQuery UI 标签页的简化版本。如果您只需要简单的选项卡功能，而不需要包含整个 jQuery UI 库。
![tabsLite.jpg](../Images/2524188a2954ef3d1ef443ccb33fea78.png)
[来源](https://sites.google.com/site/tpopsjqueryplugins/tabslite)

## 6.会话超时

经过一段设定的时间后，会向用户显示一个对话框，用户可以选择立即注销或保持连接。如果选择了立即注销，页面将被重定向到注销 URL。如果选择了保持连接，将通过 AJAX 请求一个保持活动的 URL。如果在另一段设定的时间后没有选择任何选项，页面将自动重定向到超时 URL。
![sessionTimeout.jpg](../Images/82fb445404523d85d51183b72cca695f.png)
[来源](https://sites.google.com/site/tpopsjqueryplugins/sessiontimeout)

## 7\. rowClick

如果表格行包含链接，则将可点击区域扩展到整个表格行。
![rowClick.jpg](../Images/a8a7eab8dd031e5c5e419e172bccc03d.png)
[来源](https://sites.google.com/site/tpopsjqueryplugins/rowclick)

## 8.首席专家

只有当用户在父容器中滚动时，才会将标题锁定在页面顶部。
![headerPersist.jpg](../Images/763e30126739dc156b80682937d7d185.png)
[来源](https://sites.google.com/site/tpopsjqueryplugins/headerpersist)

## 9.时尚精选

是一个小插件，可以把你的普通选项框变成一个美味的列 div。根据浏览器窗口的宽度，这些列也会重新堆叠。
![Stylish-Select.jpg](../Images/903ff01f6e2327dd8b07a65412ce85a3.png)

## 10.gl_shaderer

这个 jQuery 插件最初是为了在有纹理的平面上轻松使用 WebGL 片段着色器而设计的。gl_shaderer 创建一个 canvas 元素，并在其上初始化 WebGL。它将着色器，纹理和可渲染顶点作为参数，但除了片段着色器之外，其他都有默认值。
![gl_shaderer.jpg](../Images/c4da1efdae671ed0c96723a8ea5a29e2.png)
[来源](https://github.com/tehmou/jquery.gl_shaderer)
