# 当意外发生时，如何管理工作

> 原文:[https://www.sitepoint.com/managing-work-during-crisis/](https://www.sitepoint.com/managing-work-during-crisis/)

[![crisis](../Images/5aba8afbf93469966cf577d1294d4f57.png "crisis")](https://www.sitepoint.com/wp-content/uploads/2009/04/crisis.jpg) 当你为自己工作的时候，当生活中发生一些阻碍你实现承诺的事情时，你可能没有人替你顶班。不可避免的是，在某些时候，你将不得不放下一切，处理一个意想不到的情况。这个星期我就遇到了这种情况，这让我清楚地意识到，制定一个计划来应对这些突发事件是多么重要。

你如何应对挑战，如何振作起来，如何继续前进，这是一个非常个人化的过程，但你可以做一些事情，让它变得不那么有压力，更容易管理。

**获得支持**

这是双重的。首先，你要确保有人——分包商或同事——能在紧急情况下跳出来替你。不幸的是，当危机发生在你的生活中时，世界的其他部分还在继续前进。当事情变得困难时，你会非常感激有人能在高优先级的工作项目上收拾残局。即使只是定期监控你的语音信箱和查看电子邮件。

第二，不要低估家人和朋友的力量。拥有一个个人支持网络真的可以减轻一些压力。让一个家庭成员接管一些个人职责，这样你就有时间结束工作，或者只是在那里帮助你处理危机，这是非常宝贵的。

**创建一个通信流程**

虽然工作可能是你最不想做的事情，但是你的客户应该被告知。拥有一个与客户沟通的中心流程可以使更新变得快速而简单。它可以简单到创建一个地址列表，您可以复制并粘贴到电子邮件中，让每个人都同时知道发生了什么。

大多数人会感谢这些信息，理解这种情况，并愿意给你一点额外的时间来完成工作。但最重要的是确保他们随时了解情况，尤其是如果你要请假的话。

**确定优先级**

在意想不到的情况下，你将有更少的时间来完成更多的工作。现在，你有你所有的常规任务，加上所有的新的责任来管理这种情况。你需要看一看优先事项，首先关注最重要的事情，无论是否与工作相关。

**了解你的极限**

尽管你可能想继续下去，就像什么都没有改变一样，但你可能做不到。你可能长时间不在办公室，无法工作，或者无法通过电子邮件或电话联系到你。虽然你当然应该尽你所能去做，但要认识到，为了满足其他需求，你可能不得不对某些事情说不。你的健康和幸福，和家人在一起，和你的工作承诺一样重要。慢慢来。

当你的个人生活中发生意想不到的事情并影响到你履行工作承诺的能力时，你是如何处理的？
