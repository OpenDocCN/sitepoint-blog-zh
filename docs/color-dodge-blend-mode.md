# 颜色减淡混合模式

> 原文：<https://www.sitepoint.com/color-dodge-blend-mode/>

现在你可能已经明白了 Photoshop 有“相反”的混合模式:变暗与变亮、乘法与屏幕、颜色或线性加深与颜色或线性减淡。因此，Photoshop Help 中颜色减淡的定义听起来可能非常熟悉:

> 颜色减淡:查看每个通道中的颜色信息，并通过降低对比度使基色变亮以反映混合色。与黑色混合不会产生任何变化。

与屏幕类似，颜色减淡总是会产生更亮的图像——除了对比度会调高。它看起来是这样的:
