# 将脚注引入 WordPress 的 5 种方法

> 原文：<https://www.sitepoint.com/5-ways-to-bring-footnotes-into-wordpress/>

随着超链接的使用，对于博客作者来说，大量的脚注已经变得没有必要了。当几个链接就可以得到这份工作，而且你不必担心在哪里放置逗号或包含哪些出版物信息时，为什么要为一篇文章设置脚注呢？

然而，如果你想为你的工作提供更好的参考，特别是如果你涉及学术或研究领域，脚注可能是一种让你的工作对你的观众来说更体面和可靠的绝佳方式——想想脚注在维基百科等网站上是多么有效。通过检查作品的标题、出版日期，甚至通过点击超链接阅读文章，你可以立即检查原始材料的质量。网上引用提供了更大透明度的机会，这使得它们特别有吸引力。

如果你有必要的编码经验，你可以为你的博客编写你自己的脚注功能。但是如果你运行 WordPress，你只需要选择你更喜欢的插件(大约有五个被广泛使用)，安装它，定制它，并开始添加脚注。以下是 WordPress 的一些领先插件，它们使得引用资源作为脚注变得轻而易举。

## 简单的脚注

尽管这个插件目前的下载量很低，但是很多 WordPress 插件评论者确实很喜欢这个插件，这是有原因的。一旦你安装了[简单的脚注](http://wordpress.org/extend/plugins/simple-footnotes/ "WordPress - Simple Footnotes - WordPress Plugins")，它就非常容易使用。要做脚注，你只需简单地写:“句子中有文字。脚注文本。[/ref]"您的脚注将自动编号，这样您就不必担心弄乱它们的顺序。

如果你有带链接的分页文章，可以转到下一页或下一个主题，你甚至可以改变脚注的位置。只需在 WordPress 控制面板中的阅读部分为分页文章更改简单的脚注设置。

## FD 脚注插件

FD 脚注插件让你用你自己决定的数字来表示括号内的脚注。您可以设置脚注，使它们只显示在单个页面上，而不是您的主页上，这可能会降低您的网站速度。

此外，您可以将脚注设置为在被单击之前折叠，这样读者只有在选择了特定数字后才能查看。如果您正在编写提供大量补充信息并占用大量空间的脚注，或者可能只与少数读者相关的脚注，这可能会派上用场。

## JQuery 悬停脚注

JQuery Hover Footnotes 插件有一些高级功能，不仅仅是在你的博客文章底部添加一个脚注。你可以安装这个插件并启用一个“弹出/悬停”脚注功能，当读者鼠标经过它时，它会显示你的笔记内容。脚注字符也有很多种，包括数字、单词和 ANSI 字符。脚注的大小可以调整为普通文本、上标或下标，您也可以将它们放在页脚中或隐藏起来，这样读者就可以只依靠悬停功能了。

使用此插件的简短代码标记如下:“这是我的帖子，我想在文本中添加一个脚注{{1}}。默认情况下，它将显示为上标。这非常简单，根据您需要在博客上显示的内容，您可以在管理面板中提前进行设置。

## 学术出版社

如果你需要一个更强大的脚注插件来为你的网站提供一个整体的学术感觉，包含学术工作所需的所有格式和样式选项，  提供了你需要的一切。像其他插件一样，AcademicPress 的脚注很容易在你输入时被标记，并带有各种编号和字母选项。插件会在每篇文章的末尾自动生成一个脚注列表来显示脚注。

除了提供脚注，这个插件还支持多种引用风格，包括 APA，哈佛，芝加哥，MLA 和 Turabian。你的参考文献可以自动格式化成任何一种样式，这甚至可以为你的内联引用节省大量时间。当你输入短代码(这是插件工作的原因)时，学术出版社的插件会附带一个“VirtualBox ”,这样你就可以随时查看它们。

## 网络日志

NetBlog 是一个技术性更强的插件，为你的脚注提供静态和动态参考选项。静态选项意味着您必须手动定义每个引用，而动态选项意味着您的注释将被自动解析。

这个插件最大的不同在于，你可以为你博客上的引用创建书目属性，而不必手动输入。例如，通过在侧边栏中创建一个小部件，这个插件使您能够在参考小部件中显示 pingbacks 和 outgoing 链接，自动为您的帖子创建一个更整洁的外观。你甚至可以在每个博客文章页面上显示新的链接。

## 摘要

创建一个脚注列表会耗费大量的时间，尤其是当你需要手工编码和格式化每个脚注的时候。当你运行带有脚注插件的 WordPress 时，你可以利用参考资料的更高学术标准，并通过将脚注集成为弹出功能、帖子末尾的列表或侧边栏中的小部件来为你和你的网站建立可信度。一旦你学会了你需要的短代码，以及如何调整插件的设置来匹配你的博客，你就可以快速有效地引用你的来源，而不用担心或争论。
