# 谷歌威胁退出中国

> 原文：<https://www.sitepoint.com/google-threatens-to-quit-china/>

在公司系统遭到网络攻击后，谷歌宣布他们正在考虑关闭他们在中国的办公室，并关闭 Google.cn。

谷歌报告称，他们检测到来自中国的对其公司基础设施的高度复杂和有针对性的攻击。尽管最初被认为是重大安全事件，但谷歌发现这次攻击影响了至少 20 家大公司。他们有证据表明，黑客的主要目标是进入知名中国人权活动人士的 GMail 账户。

他们的调查发现，来自美国、欧洲和中国的数十名人权活动人士的账户经常遭到未经授权的第三方的侵犯。然而，在大多数情况下，这些账户可能是被恶意软件或网络钓鱼欺诈攻破的。

尽管谷歌小心翼翼地避免指责中国当局，但他们已经在官方博客上公布了攻击的细节。

谷歌于 2006 年 1 月推出了 Google.cn 搜索引擎，该公司因按照中国政府的意愿审查搜索结果而受到批评。谷歌首席法律官大卫·德拉蒙德表示:

> 这些攻击和他们发现的监控——再加上过去一年来进一步限制网络言论自由的尝试——让我们得出结论，我们应该审查我们在中国业务的可行性。
> 
> 我们已经决定不再愿意继续审查我们在 Google.cn 的搜索结果，因此在未来几周内，我们将与中国政府讨论我们可以在法律范围内运营未经过滤的搜索引擎的基础。我们认识到，这很可能意味着不得不关闭 Google.cn，并可能关闭我们在中国的办事处。

中国拥有世界上最多的互联网人口，超过 3 亿网络用户。大多数人使用中国自己的百度搜索引擎，但谷歌仍占有 29%的市场份额。

谷歌应该退出中国吗？谷歌的影响力已经变得如此之大，以至于它可以反击不同意它的政府了吗？我怀疑这个故事将在未来几个月内发展。
