# 在 Illustrator 和 Freehand 中创建矢量图形

> 原文:[https://www.sitepoint.com/graphics-illustrator-freehand/](https://www.sitepoint.com/graphics-illustrator-freehand/)

这篇文章写于 2009 年，至今仍是我们最受欢迎的帖子之一。如果你热衷于学习更多关于使用图像和图形的知识，你可能会对这篇关于精灵动画的文章很感兴趣。

我交谈过的许多设计师都喜欢 Photoshop。他们对它很有经验，知道它是如何工作的，所以他们几乎在任何事情上都使用它。但是当使用 Illustrator 或 Freehand 时，它们就不那么舒服了。“我知道我可以做那个 Photoshop，”他们想。“为什么要用 Illustrator 呢？”

我将在本文中回答这个问题，这是矢量图像系列的第一篇文章。在本系列的后续部分中，我们将利用我们在这里学到的基础知识，看看如何将它们应用到一系列更复杂的设计任务中。

如果你认为 Adobe Illustrator 和 Freehand 都是为了创建无聊的几何图形，那就再想想吧。到本文结束时，您将会看到矢量图形在设计实践中的一些非常酷的应用。

## 为什么是矢量？

从前，人们不得不用笔和纸来写信或写文章。大学论文是在一叠叠笔记本纸上起草的；许多行划掉了不需要的单词或句子，而小卡片插入了额外的单词或短语。看到满是皱巴巴的纸片的垃圾桶并不罕见(这也是在回收之前)，或者作家用箭头或脚注标记页面，指向他们想插入另一个或两个段落的其他地方。

相比之下，我是坐在屏幕前，手指放在键盘上写这篇文章的。如果我决定不喜欢某样东西，我会把手移到鼠标上，高亮显示不喜欢的文本，然后简单地点击一个按钮，删除它。当我读完我写的东西时，我决定澄清一下，所以我把光标上移，输入我的新想法。

学习如何创建和使用矢量图形就像学习如何使用文字处理程序一样强大。

## 什么是矢量图形？

你在网站上看到的大多数图像都是光栅图形，由颜色的小点(像素)组成。如果你拍摄一张 gif 或 jpeg 图片并放大，你会看到组成图片的单个像素:

![1382_vector1](../Images/5d41f33d15416fd1f727aff12e6dc633.png)

如果您使用成像程序来缩小图形，它会进行一些计算，将相邻像素“平均”在一起，以创建尺寸更小的图像。但是，如果你想放大图形，成像程序会利用现有的像素，进行一些计算，并添加许多额外的像素，使图像更大。这通常会导致图片模糊。在下图中，你可以看到我们原始的 100×102 像素的图像，放大了 300%。光栅图像放大不好！

![1382_vector2](../Images/75973ce6a903a14e17968fb5cde6a111.png)

光栅图形与矢量图形对比的时间*:矢量图形由定义直线和曲线的点组成。比如下面的矩形是由四个点用线连接而成的矢量图形，抽象的形状是由点用曲线连接而成。图形包含有关线条或曲线的方向、线条的粗细以及填充形状的颜色或图案的信息。*

 *![1382_vector3](../Images/334e82ca50ac281e540b4434ac34911e.png)

矢量图形的伟大之处在于，因为你只是在处理点和线，你可以很容易地放大它们而不会丢失任何细节。这使得矢量图形非常适合可能需要以不同大小输出的徽标和图表。例如，当设计一个标志时，我通常会使用矢量图形程序来制作初始文件，然后创建一个优化的光栅版本的网络文件。

矢量图形的另一个好处是它们的文件通常更小。光栅图形必须存储每个像素的信息。矢量图形只需要存储点的信息。

到目前为止，您应该对矢量图形非常感兴趣。让我们开始做吧！

## 基本形状

Adobe Illustrator 和 Macromedia Freehand 有许多基本形状工具，可以轻松创建矩形、圆形和其他形状。让我们快速浏览一下工具选项板(选项板可能会因您使用的程序版本而略有不同):

默认情况下，Illustrator 和 Freehand 都显示椭圆和矩形工具:

徒手:![1382_vector4f](../Images/d04cfd51c6635e1b2198a6d0fe90df17.png)

在 Illustrator 中，按住椭圆和矩形按钮以查看其他形状类型的更多工具，包括多边形、螺旋形和圆角矩形。

Illustrator 工具:

Freehand 在矩形工具选项下有一个附加的多边形工具，以及圆弧工具和螺旋工具，可从 Xtra 工具调板(“窗口”>“工具栏”>“Xtra 工具”)访问。要创建圆角矩形或星形，您可以修改工具或对象的一些选项(稍后将详细介绍)。

手绘工具:

使用这些形状工具中的一个来画一个形状是非常非常简单的。

1.  选择工具。

3.  在文档中单击，按住并拖动。

5.  当形状达到您想要的大小时，释放鼠标按钮。你完了！

多边形和螺旋工具总是从中心开始绘制；也就是说，您最初单击的位置将是对象的中心起点。默认情况下，矩形和椭圆工具将从左上角开始绘制。如果要从中心绘制矩形和椭圆，请在绘制时按住 Alt (PC)或 Option (Mac)键。这个技巧对 Illustrator 和 Freehand 都有效。

## 更改基本形状的选项或属性

***Illustrator*** :在 Illustrator 中使用形状工具时，您可以选择该工具，然后只需在页面(不要按住并拖动)上单击一次*即可弹出该形状的选项对话框。“选项”对话框允许您定义形状的特定尺寸(例如，精确的 2×3 英寸的矩形，或 200×200 像素的圆形)，并为某些形状提供附加选项。例如，圆角矩形工具选项框允许您指定圆角的半径:*

![1382_vector9](../Images/0d66995efc93bb822bbeef7dbf1018f4.png)

类似地，多边形(和星形)工具具有允许您定义形状的边数(或点数)的选项:

![1382_vector10](../Images/5a42e505e5774d632d261ca360da7540.png)

定义选项后，单击“确定”创建具有这些规格的形状。形状工具将记住您用该工具创建的其他形状的选项。要更改它们，只需单击一次，再次打开选项对话框。

***手绘*** :手绘的形状选项可以通过*双击工具按钮*本身来访问，在矩形工具的情况下，它允许你指定一个圆角半径来创建一个圆角矩形:

![1382_vector11](../Images/bd1af498fea73cfe1a78696316ae4606.png)

![1382_vector12](../Images/6dea2e5a54d0b6884c2d65ce2cf03aef.png)

双击多边形工具，您将能够指定边的数量——无论形状是多边形还是星形——甚至星形点的角度。

![1382_vector13a](../Images/3e47717a62d2239b8758c4a23c57d6f8.png)

![1382_vector13b](../Images/898623f9ba923c6c9b7bb454591e3c4f.png)

在 Freehand 中创建形状后，可以通过选择形状并更改“属性”>“对象”调板中的数字和选项来进一步调整其尺寸和属性。

![1382_vector14](../Images/561fdbe4812e89a64123f749791666ec.png)

如果精确的尺寸对您来说并不重要，但是您想要在 Illustrator 或 Freehand 中放大或缩小对象，请切换到黑色箭头工具(在 Illustrator 中称为“选择工具”，在 Freehand 中称为“指针工具”，在我们这种低技术类型中称为“黑色箭头”)，然后选择对象，最后拖动角手柄，将对象调整到您理想的大小。

![1382_vector15](../Images/abbb4334bc430e4e2189208551cd940f.png)

现在你已经掌握了基本的形状，是时候熟悉钢笔工具了。
