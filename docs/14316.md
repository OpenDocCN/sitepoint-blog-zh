# 如何让你的网络服务与众不同(并回答内陆海鲜餐厅的案例)

> 原文：<https://www.sitepoint.com/how-to-set-your-web-services-apart-and-answer-to-case-of-inland-seafood-restaurant/>

上一篇博客问你，在没有水景或海滨的佛罗里达旅游市场，你会如何定位一家海鲜餐厅。虽然您的许多想法都很好，但这家连锁餐厅选择了一个比您提议的更简单的解决方案。

答案与网页设计/开发服务有关，因为他们的方法是定位任何业务的一种常见的、成功的方法(这是我以前在这个博客中谈到过的一种方法):

化劣势为优势。

这家餐厅自豪地宣传他们没有海滨景观，而是注重菜单和海鲜的质量。他们的座右铭是:

"没有海滨景色，只有美味的海鲜。"

他们支持他们的主张:最广泛和最有创意的海鲜菜单，在当地美食比赛中的众多参赛作品以及奖项，以及每年在当地高档杂志上赢得“人民的选择”奖。

这被称为一维定位:选择一个对你的客户重要的标准，并通过成功/主导该标准来重新定义市场。在这种情况下，市场一直专注于壮观的海滨景观。与此同时，许多风景好的餐馆有时会有有问题的海鲜，这对游客和居民都很重要。因此，有一个餐厅开始专注于提供最好的海鲜。

其他示例:

美国邮政局通过关注价格从联邦快递手中夺回了地盘。他们的服务/可靠性/速度可能不如联邦快递，但他们的费用要低得多，而且你的包裹一天左右就能送到。这种价格优势对很多人都很重要。

与 Hertz 相比，Avis 在市场上排名第二，但通过关注他们如何“更加努力”来赢得您的业务，avis 将这一优势转化为优势。

在网页设计中，有很多服务维度可以让你脱颖而出:

–唯一一家了解导航和设计方案的公司，为通过网络销售服务和产品的公司获得响应

–专注于特定行业(例如，“我们为更多的健身俱乐部设计了网站，比其他任何人都多”

–专注于某个地区(例如，“被萨拉索塔商会评为最佳小型技术公司，被萨拉索塔经济发展委员会评为最佳新公司”)

还有二维定位，我很快会谈到。
