# 火狐 3 Beta 5 可供下载

> 原文：<https://www.sitepoint.com/firefox-3-beta-5-available-for-download/>

火狐 3 的 beta 5 版本于本周发布，包含以下改进:

*   用户界面的改进，包括对 Windows Vista、Windows XP、Mac OS X 和 Linux 的外观和感觉的改变
*   对新功能的更改和修复，如地址栏自动完成、书签备份和恢复、整页缩放等
*   对平台功能的修复和改进，以提高安全性、web 兼容性和稳定性。
*   性能改进，特别是 JavaScript 引擎和下载速度(很大程度上是由于最大连接数从 2 增加到 6。

这个新版本包括了 750 个来自之前测试版的变化。在 JavaScript/DOM 标准支持领域，Firefox 3 仍然达不到最近最新开发者发布的 [Opera 和 Safari 在 Acid3 测试](https://www.sitepoint.com/opera-and-safari-pass-acid3-test/)中达到的 **100/100** 分数，但 **71/100** 无疑是当前版本 Firefox 2 达到的 **53/100** 的改进。
