# 分支:不在 Twitter 上的 Twitter 广告网络

> 原文:[https://www . site point . com/the-branch-Twitter-ad-network-thas-not-on-Twitter/](https://www.sitepoint.com/the-branch-twitter-ad-network-thats-not-on-twitter/)

谈到 Twitter 上的广告(这里我们说的是广告，不是营销)，有两种流行的方法。第一个被推特网雇佣。他们在人们的推特账户背景上出售静态广告。Twitter 本身在日本也采用了这种方法。但是它不起作用有两个原因:第一，背景是静态的，无论如何都可以参与进来，第二，更重要的是，没有很多人使用 Twitter 的网站。大多数 Twitter 流量[来自 API](http://blog.programmableweb.com/2007/09/10/twitter-api-traffic-is-10x-twitters-site/) 。换句话说，大多数人通过第三方客户端看到推文，并没有多少用户会真正看到你的广告。

广告的第二种方法是直接的，流广告。这就是喜鹊做的事情，据报道，去年四月，推特[也在测试这件事。Twitter 从来没有添加流广告，可能有一个原因:人们讨厌它们，所以它们可能是无效的。Twitter 的信噪比已经很低，随着你关注的人越来越多，信噪比会越来越高——再加上广告，情况可能会变得更糟。这就是为什么我们](http://www.techcrunch.com/2008/04/14/twitter-testing-advertising-in-twitter-streams/)[在《喜鹊》发布时称其为一个糟糕的想法](https://www.sitepoint.com/terrible-idea-magpie-twitter-ad-network/)。

因此，向 Twitter 用户推送广告的两种主要方法并不奏效。一个名为分支的新广告网络认为他们已经找到了第三种方法。我们认为它比其他两种更有意义，但它不能真正归类为 Twitter 广告方法。

该分支本质上是一个传统的广告网络，专门处理 Twitter 用户流量大的网站和工具。这个想法是，虽然你不能有效地直接向 Twitter 用户做广告，但你可以在他们经常去的网站上做广告。现在，该网站在超受欢迎的 Twitter pic 和 T2 Twitter Gallery 上出售广告。

我们同意，该分支机构在接触 Twitter 用户方面可能比 Twitter 更有效，比 Magpie 更不具侵入性，但它也错过了 Twitter 中存在的瞄准机会。您不再能够锁定特定的用户、影响者或用户组。更确切地说，这个品牌本质上是在流行的网站上销售网络广告——那些网站只是碰巧受到也使用 Twitter 的人的欢迎。

这个分支机构是由 Zerk Media 创建的，它还拥有为 Ruby 和微软开发者社区服务的定向广告网络。所以他们肯定有一些广告网络信誉，肯定能成功。然而，该分支也仍然不是难以捉摸的 Twitter 广告解决方案。
