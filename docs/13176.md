# 利用目标设定建立你的企业

> 原文:[https://www . site point . com/building-your-business-using-goal-setting/](https://www.sitepoint.com/building-your-business-using-goal-setting/)

欢迎回到 2008 年的 [Web Pro Business](https://www.sitepoint.com/the-8-best-wordpress-themes-for-small-business-websites/) 博客，我希望你度过了一个轻松安全的节日假期。

我很高兴也很荣幸在 12 月被邀请为 [SitePoint](https://www.sitepoint.com) 写一篇文章，你[可以在这里](https://www.sitepoint.com/resolutions-boost-business/)阅读。在这篇文章中，我对 2008 年要实现的目标提出了十条建议，这将有助于你的业务增长。

现在，即使你还没有成为全职自由职业者或企业主，这十个决心(也可以打印成单页 A4 或 T2 字母大小的 PDF 格式)将帮助你达到这个阶段。

我希望在接下来的几篇文章中详细阐述我在这篇文章中提出的几点。首先，让我们从目标设定开始。

大多数设定目标的画面会让人联想到跑车、豪宅和在游泳池里游泳，然而这些对我的建议来说可能有点浮夸。让我们将其分解为中长期目标、短期目标和属于目标的行动。

比方说，你就像许多对我之前的帖子发表评论的人一样，还没有开始自主创业。你的长期目标是什么？成为个体经营者。有哪些短期目标可以为此提供垫脚石？

这里有几个可能的目标:

1.  在银行里存一些应急的钱
2.  写一份商业计划
3.  在周末和晚上做一些项目
4.  创建一个企业名称

现在，第一个很简单——假设我们需要在接下来的六个月里存下两个月的最低工资。属于这个目标的行动可以包括“将每周工资的 10%存入特别账户”或“在接下来的几个月里减少外卖晚餐”。

现在，写商业计划的行动可以包括“寻找商业计划或商业计划软件的例子”和“留出时间和自己的朋友谈论他们的计划(如果有的话！)'.

如果你努力把它们写下来，打印出来，贴在你经常阅读的地方，然后通过反复提醒，目标设定可能真的会成为一种习惯，一种通往你想去的地方的有效方式。

我建议你现在就写下一些目标，然后在三个月后重新审视它们，看看你进展如何。我鼓励你在下面的评论区让我知道你以前是如何使用目标设定的。

祝你好运！
