# 节日和客户需求

> 原文：<https://www.sitepoint.com/the-festive-season-and-client-demands/>

当我们都进入愚蠢的季节，有家庭活动，社交晚会和圣诞节的所有乐趣，这是一个适当的时间来反思你如何处理客户的需求。

当我第一次独自创业时，我尽可能地工作——更重要的是，客户要求我工作多少时间，我就工作多少时间。这通常会在晚上和周末结束，我开始相信所有的客户工作都是重要和紧急的，必须马上完成。

然后，一年后，我重新考虑把我的手机号码放在名片上，并在下一次印刷时把它扔掉，作为对客户和我自己的一个测试——不能把它放在手边的障碍会阻止他们在周日早上打电话吗？

我很欣慰地发现确实如此，更重要的是，我的客户几乎没有抱怨。那些抱怨的人已经习惯了让我一周七天随叫随到。当然，我可以责怪他们，但我也训练他们相信我是，所以我只是错了，因为他们，如果不是更多。

现在，五年过去了，我的员工中没有一个人会公布他们下班后的细节，他们也不应该这样做。事实上，我们下周五就关门了，一直到 2008 年的第一天。通过允许客户拆除营业时间和所有时间之间的墙，我们训练他们相信我们很乐意接听他们晚上或周末的电话。

仅仅因为你的客户没有自律来保持一致的时间表，并不意味着你也需要。与其要求他们不要打电话，不如干脆不接电话，让它转到语音信箱。如果听起来很紧急，请在一两个小时内回电，否则，请等待下一个工作日，然后再返还。

如果你在家工作，准备第二个电话号码或线路，这样你就可以区分私人电话和商务电话——如果你不这样做，你将一辈子都在工作，没有明确的“工作时间”和“私人时间”的划分。

圣诞节应该是放松的时刻，也应该是享受过去十一个半月辛勤工作所带来的成功的时刻。在你的客户群中鼓励这种哲学，并收回你的个人时间。

你已经这样做了吗？干得好！在下面的评论区告诉我们你是怎么做的。

我祝你有一个安全愉快的节日，我期待着在节日假期和 2008 年分享我的想法，倾听你的想法。
