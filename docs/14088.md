# Fedora 中的免费目录服务

> 原文:[https://www . site point . com/free-directory-services-in-fedora/](https://www.sitepoint.com/free-directory-services-in-fedora/)

正如许多人已经探索过的，Fedora Core 4 已经从五月开始测试发布。更好的是，经过几个月的许可和开发，Red Hat 基于其从 AOL 获得的 Netscape 安全解决方案，提供了一个企业级目录服务器。

免费 GPL 版本可用于 Fedora，使 Fedora 社区能够为大大小小的用户群试验目录服务。这无疑提高了那些寻求在其他机会中测试和实现单点登录解决方案的 web 专业人员的能力。

随着我们的网站演变成 web 应用程序，并进一步演变成不同规模的多个交互门户，我们开始面临身份验证的挑战。尤其是在将我们的解决方案放入现有的客户基础设施(包括内部网、外部网和第三方供应商系统)时。有了健壮的 LDAP，我们现在能够接入 Microsoft Active Directory 服务、Lotus Domino 目录和其他类似 LDAP 的环境，并增加更多的价值。

我继续观察和参与日益复杂的环境，这种环境本质上要求多个管理员、更大的用户组和具有挑战性的跨平台集成。随着客户端和最终用户处理大量的用户凭据，单点登录比以往任何时候都更加成为一项需求。

我很好奇谁已经深入研究了现有的开源 LDAP 解决方案(在任何 Unix 或 Windows 平台上),以及对这个新的 Fedora 功能的任何开发测试的反馈。
