# W3C 寻求对 CSS3 的反馈

> 原文:[https://www.sitepoint.com/w3c-seeks-feedback-on-css3/](https://www.sitepoint.com/w3c-seeks-feedback-on-css3/)

CSS3 已经工作了多年，有如此多的新特性和改进，以至于规范必须被分解成三十个模块。这些模块中的大部分预计将在未来一年半内完成。

根据[路线图](https://www.w3.org/Style/CSS/current-work#table)，最长的模块之一是[背景&边界模块](https://www.w3.org/TR/2005/WD-css3-background-20050216/)。这个模块负责一系列的增强，从多个背景图像到基于图像的边框，它负责克服当前应用于语义正确的(X)HTML 代码的 CSS 在视觉设计上的许多限制。该模块仍处于工作草案阶段，预计将于今年年底前完成。

为此，W3C 已经[向精通 CSS 的设计者发出号召](http://www.lawver.net/archive/2005/07/18/h16_help_the_css_working_group_with_backgrounds_and_borders.php),请他们就影响模块的几个问题提供反馈。一些已经提出来讨论的想法看起来确实非常有用，所以如果你对 CSS3 有兴趣(让我们面对现实吧，每个网页设计师都有)，那就进去发表你的意见吧！
