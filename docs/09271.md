# 手机如何工作

> 原文:[https://www.sitepoint.com/how-mobile-phones-work/](https://www.sitepoint.com/how-mobile-phones-work/)

手机已经成为现代生活中无处不在的一部分。据估计，目前使用的手机超过 50 亿部，覆盖了世界人口的很大一部分。然而，大多数使用手机的人并不了解手机的工作原理。本文展示了如何将复杂的无线电技术和计算机安全相结合，创造出许多人认为理所当然的移动电话系统。

￼The 首先要明白的是，在长途功能、精致的触摸界面和其他华而不实的功能之下，现代手机本质上只不过是双向无线电。所以，要了解手机的工作原理，首先要了解无线电的工作原理。

无线电天线是一种将无线电波转换成电流的装置，电流可以被无线电接收器处理成声音。此外，它可以做相反的事情，将现有的电子音频信号转换为其他天线可以接收的发射无线电波。当移动电话传输音频时，它向移动电话天线施加振荡电流。然后手机天线发射相应的电磁波，也就是所谓的无线电波。为了接听电话，手机天线会拦截特定频率的电磁波。然后，它的终端接收到极少量的电压，这些电压被其他组件放大并转换成声音。

通过上面解释的过程，移动电话天线向无线电塔发射信号，同时接收信号。然而，能够如此复杂地使用无线电信号的技术是几十年来逐渐发展起来的。

移动电话最接近的前身是安装在卡车司机车上的无线电，这使得他们可以在路上进行通信。因为这些收音机又大又重，对于消费者来说不实用。这些移动通信系统倾向于在它们服务的每个城市使用一个中央天线。每个塔上只有几十个频道。因此，如果太多的人试图同时使用这个系统，它就不会工作。此外，因为该系统只是半双工的，所以在任何给定的时间只有一方可以通话。这种简单和单向的需求导致了一些有趣的术语，但是对于普通人来说使用起来并不方便。

与旧的便携式无线电通信系统不同，现代移动电话使用蜂窝网络。蜂窝网络也是无线电网络，但它们远比驱动旧无线电通信系统的简单本地塔复杂。在蜂窝网络中，信号塔分布在称为蜂窝的陆地部分。这些细胞通常是六边形的，但它们也可以是方形或圆形的。每个陆地单元包含至少一个无线电塔。

每个小区还被分配了对应于无线电基站的多个频率。其他小区可以使用相同的频率，只要它们不相邻。换句话说，相同频率的使用必须相隔至少一个小区。这样做是为了避免信号使用相同频率的呼叫之间的干扰。这种重复使用频率的能力是蜂窝网络能力的关键；更多的人可以通过网络打电话，打更远的距离。虽然单发射机系统不能重复使用相同的频率，但蜂窝网络可以在多个区域使用相同的频率进行不同的传输，只要传输距离不会近到造成干扰。由于采用了蜂窝方式，对话可以跨细胞进行，使得与全国各地的人交谈变得容易。此外，这些系统是全双工的，这意味着手机可以同时发送和接收信号。这使得人们可以边听边说，从而以他们习惯的方式进行对话。

蜂窝网络的概念不限于移动电话，但是移动电话网络是一种特别复杂的蜂窝网络。蜂窝发射机和手机可以在自动计算机系统的指导下改变频率，自动计算机系统可以找到最佳范围以避免干扰。它们通常有 800 个不同的频率可用于每个相邻小区群。由于波长的性质，低频波传播的距离更长，不会被大气噪音打断。这个原则适用于手机信号。低频集群更适合长距离，而高频集群更适合密集区域。因此，农村地区的手机信号往往使用相对较低的频率，而在城市地区，他们使用较高的频率。

在城市中，每个蜂窝基站通常覆盖半英里。在农村地区，他们往往覆盖约五英里。然而，这并不是现代手机接收的真正极限。在没有干扰或阻塞的情况下，有时可以接收到 25 英里以外的基站发出的信号。移动电话也倾向于使用低功率发射机，以避免在紧凑布置的小区中通话之间的干扰。每个蜂窝基站都与电话交换机相连，这样就可以从手机向公共电话网络拨打电话。

对于移动电话网络应该如何相互通信和共存，已经建立了许多不同的标准。目前，最流行的手机通信标准是 [GSM](http://en.wikipedia.org/wiki/Gsm) 和 [CDMA](http://en.wikipedia.org/wiki/Cdma) 。一种被称为 [LTE](http://en.wikipedia.org/wiki/Long_Term_Evolution) 的新高速标准正在人口稠密地区引入。

为了抵制对其网络的未授权使用，移动电话提供商引入了安全措施，不允许电话在没有某些有效数据的情况下接收和发送呼叫。这些数据通常存储在被称为用户识别模块(T1)或简称为 T2 SIM 卡(T3)的微型芯片上。它们携带的数据被称为服务用户密钥。这种数据的技术名称是国际移动用户身份。这用于收集关于移动国家代码、移动网络代码和电话的移动站标识号的信息。它还带有一个认证密钥。SIM 卡最常用于 GSM 网络的手机。许多为 CDMA 网络制造的手机不使用可移动卡，而是将安全信息永久嵌入内存。然而，能够处理这两种通信协议的“世界”电话包括 SIM 卡。

通过一个巨大的互联网络发送和接收无线电信号，移动电话可以在发达国家的任何地方打电话，跨越数千英里的距离。手机是现代科技的奇迹。
