## 剧集摘要

感谢 SitePoint 论坛社区中为 Chris 提交问题的所有成员！

这一集我们向克里斯提出了以下问题:

*   IE 的平台架构师是做什么的？
*   为什么 IE8 的默认渲染模式变了？
*   IE8 中的兼容性视图按钮是怎么回事？
*   用户真的可以从 Windows 7 中删除 IE8 吗？
*   如果不是 IE，你最喜欢哪个浏览器？
*   让 IE6 用户升级的计划是什么？
*   IE 中的每种渲染模式会保留多长时间？
*   自 IE6 以来，我们使用互联网的最大变化是什么？
*   是否有计划允许多个版本的 IE 并行安装？
*   Internet Explorer 开发者工具栏与 Firebug 相比如何？
*   Internet Explorer 团队的规模和形式如何？
*   HTML5 中“铺垫牛巴”的真正价值是什么？
*   HTML5 的哪些重大变化最令人兴奋？
*   HTML5 会成为一个单一的规范吗？应该分吗？
*   在 IE 中加入新的 CSS 特性需要哪些步骤？
*   IE 有什么计划来对抗日益流行的替代品？
*   字体争论:为什么动态加载 TTF 还不够？
*   你在你的摇滚乐队演奏哪种乐器？
*   web 堆栈不适合构建应用程序。解决方法是什么？
