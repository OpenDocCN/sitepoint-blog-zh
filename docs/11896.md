# 五个 jQuery 图库和幻灯片

> 原文:[https://www . site point . com/five-jquery-galleries-and-slide shows/](https://www.sitepoint.com/five-jquery-galleries-and-slideshows/)

如今没有图片库的网站算什么？画廊是用来展示产品和作品集的，随着时间的推移，它们变得越来越时尚，越来越快，越来越轻。jQuery 插件可以用于你能想到的几乎所有功能，从小部件到动画效果。作为一名设计师，我想在网站上添加许多功能，但是我没有足够的技术力量来编程或把它们放在一起，但是我知道足够把一些 jQuery 放到页面上。令人欣慰的是，有许多开发者慷慨地分享他们的工作，确实有一些精彩的图库和幻灯片作为插件。

这里有五种时尚的产品，可以免费获得，并且很容易在你的网站上实现。

jQuery Lightbox 是在当前页面上覆盖图像的一种简单而优雅的方式。

![lightBox](../Images/f5f6da28359971912229df089b949781.png "lightBox")

[点击这里观看演示
点击这里](http://leandrovieira.com/projects/jquery/lightbox/)下载并阅读更多关于安装的信息

**空间对于一个名字来说，画廊景观**是一个非常干净和多功能的画廊。它包含缩略图、图像描述的半透明覆盖图和缩略图的标题。

![Space For Name](../Images/710a11927becab2cd7702200fab5b769.png "Space For Name")

[点击此处观看演示
点击此处下载并阅读更多关于安装的信息](http://plugins.jquery.com/project/galleryview)

Mb Gallery 是一个干净的全功能照片库，有导航工具栏、缩略图和自动调整大小的框架。图库会自动调整到当前选定的图像尺寸。

![MBGallery](../Images/224fcbdca3222acaf2c7008263191c30.png "MBGallery")

[点击此处观看演示
点击此处下载并阅读更多关于安装的信息](http://pupunzi.wordpress.com/2009/02/28/mbgallery/)

**S3 滑块**是一个幻灯片，提供图像的平滑过渡，半透明区域覆盖有图像描述。

![s3Slider](../Images/7674543acf9b128bcdc53e6cb088e8b6.png "s3Slider")

[点击此处观看演示
点击此处下载并阅读更多关于安装的信息](http://www.serie3.info/s3slider/demonstration.html)

**Galleriffic** 允许两种类型的画廊设置，“简单”和“高级”。简单选项使用文本链接来链接到画廊中的每个图像，而高级选项在左侧显示缩略图。其他功能包括书签友好的网址，图像标题和描述和幻灯片选项。

![gallerific](../Images/1d20812594c39acae11e6ea8d30bc514.png "gallerific")

[点击此处观看演示
点击此处下载并阅读更多关于安装的信息](http://www.twospy.com/galleriffic/#1)
