# 终结了。10 个常见的自由职业神话

> 原文:[https://www.sitepoint.com/10-common-freelancing-myths/](https://www.sitepoint.com/10-common-freelancing-myths/)

关于自由职业、雇佣自由职业者以及自由职业者的生活到底是什么样的，有很多误解。以下是一些常见的神话，看看每一个的真实情况。

神话:自由职业者不可能过上好日子。

这个神话伴随着那些说企业主永远不会收支平衡的人，当你为自己工作时，想要省钱是不现实的。当然，你可以通过自由职业和独立工作来谋生——人们每天都在这样做。就像任何其他努力一样，它需要努力工作、经验、技能、研究和规划，但这是一种非常可行和现实的创收方式。

神话:自由职业者只是因为找不到真正的工作才从事自由职业。

虽然一些自由职业者是在被解雇或离开全职工作后开始的，但他们是自由职业者的事实并不意味着他们不能被雇佣。我认识的大多数自由职业者都这么做，因为这是他们的选择。

神话:自由职业者非常需要生意，以至于他们愿意协商价格。

每一个企业家都可能不时地面临困难的金融时期，并考虑修改他们目前的利率和产品。但是说自由职业者更愿意协商他们的价格是不正确的。事实上，最成功的自由职业者是那些花时间去设定一个让他们满意的价格，然后找到愿意并有能力支付他们应得价格的客户的人。

神话:自由职业者可以在任何他们想做的时候做任何他们想做的事情。

任何在家工作过的人都知道，当生活和工作重叠时，平衡生活和工作并不总是一种轻松的体验。当然，为自己工作的一个好处是能够支配自己的时间。但是如果你有一个成功的自由职业者事业，白天休息通常意味着熬夜或下班后工作。是的，你可以在你选择的时候工作，但仍然有工作需要完成。

神话:作为一名自由职业者，我不需要与客户签订合同。

合同是任何一种商业的重要组成部分。它们不仅有助于设定项目的规格，而且还概述了付款条件。虽然一份合同肯定不能保证你会按时得到报酬，但和你的客户签订一份合同是个好习惯。
 **谬论:当一个客户可以雇佣一个大公司时，他们不会雇佣一个自由职业者。**

是的，有些客户会雇佣一家代理公司或提供全方位服务的公司，而不是自由职业者。但这并不意味着这些客户永远不会雇佣自由职业者。这实际上取决于项目所需的工作范围和专业知识。许多客户确实雇佣自由职业者，因为他们相信他们可以更快地完成工作，自由职业者反应更快，并且客户可能会意识到他们只需要处理一个联系人就可以完成工作。

神话:如果我是自由职业者，我不需要担心学习新技能。

作为一名自由职业者，如果你想拓展你的服务，提高你的市场竞争力，并在你的行业中保持领先，你总是需要继续学习。

神话:自由职业者比替别人工作压力小得多。

由于你责任的深度，自由职业者在某些情况下可能压力更大。你不仅要做好工作，让你的客户满意，还要管理你的企业财务，开发票和跟踪付款，提供客户支持，以及做其他一切让你的企业运转的事情。

神话:如果你不是一个善于交际的人，自由职业是一个不错的选择。

当你是自由职业者时，你必须愿意与你的客户、潜在客户和同事互动。关系是成为一名成功的自由职业者的重要组成部分，因为它们可以带来工作、重复工作、推荐和合作。良好的沟通和开放的人际关系是必须的。

神话:如果我不愿意辞掉全职工作，我就不会去做自由职业者。

自由职业的美妙之处在于，你可以兼职，全职，甚至偶尔做做。兼职做一些自由职业是获得更多经验、学习新事物和探索全职自由职业者是否适合你的好方法。
