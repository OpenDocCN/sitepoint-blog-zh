# 简单测试 1.0RC1 不可用

> 原文：<https://www.sitepoint.com/simpletest-10rc1-is-out/>

PHP 的单元测试框架已经到了 1.0RC1。马库斯在这里[发布了一个简短的公告](http://sourceforge.net/forum/forum.php?forum_id=402499)描述了一些变化。这也标志着通往稳定的 1.0 版本的道路(尽管在我看来，它已经围绕主流功能稳定了很长时间)。

我必须在适当的时候尝试的一件事是将 SimpleTest 的 [web 测试功能](http://simpletest.sourceforge.net/SimpleTest/tutorial_WebTester.pkg.html)与其他语言的类似工具进行比较，例如 [HTTPUnit](http://httpunit.sourceforge.net) 。开始有了这样的印象，SimpleTest 可能是一个优秀的 web 测试工具，即使你没有使用 PHP。现在，如果有人能神奇地基于[蜘蛛猴](http://www.mozilla.org/js/spidermonkey/)生成一个 PHP 扩展…

SimpleTest 也有可能与一个众所周知的 PHP IDE 集成。还没有具体的消息，所以我会暂时保持沉默，但如果真的发生了，那就太好了。

无论如何，感谢并祝贺马库斯，他最近做了父亲，所以这更令人印象深刻(时间管理？没时间了。)

更新:巧合的是， [PHP 杂志](http://www.phpmag.net)发表了 Marcus 不久前写的一篇文章，对所有人都是免费的:[编写一个登录框不会有什么坏处](http://www.php-mag.net/itr/online_artikel/psecom,id,613,nodeid,114.html)——看看 SimpleTest 的 web 测试特性。
