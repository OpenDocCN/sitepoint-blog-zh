# 网页设计与生活的意义

> 原文：<https://www.sitepoint.com/on-web-design-and-the-meaning-of-life/>

当我八岁的时候，我从一天的棒球训练中冲进房子，发现一张铅笔素描放在厨房的柜台上给我看。我认出这是我最喜欢的一本蝙蝠侠漫画书的封面。我知道我爸爸会画画，但这不像他的风格。当真正的艺术家现身时，我惊讶地叫道:“哇，妈妈！你也会画画？”(她真的很棒。)

当我还是个孩子的时候，从来没有人递给我一本素描本，告诉我开始画画。我刚刚做了。很多。我画了迪斯尼人物、超级英雄、查理·布朗和史努比。随着年龄的增长，我会从我读过的书中画出场景或人物。

因为我喜欢画画，我最终进入了平面设计领域。我从设计学校毕业的时候，只有极客才知道如何使用互联网。但这种情况很快开始改变。在我获得第一份图形工作后不久，我的主管离开了，去了一个网页设计师的职位。我印象深刻。我不知道制作网站需要什么样的戏法，但我确信这是由拥有超级计算机大小的大脑的人完成的。

最终，我学会了如何做，并开始自己建立网站。我热爱平面设计和网页设计的一切。我喜欢它涉及到一台电脑——尤其是一台麦金塔电脑。我喜欢 HTML。当我发现 CSS 并发现我可以通过修改一些代码片段来改变整个网站的风格时，我感到很惊讶。我有很多内在的原因来解释我为什么以此为生。

那么我是如何从网页设计师到电话销售经理和销售培训师的呢？这两条职业道路看似相距甚远。但事实并非如此——当你审视我所做的事情背后的外在原因时，你会发现并非如此。

你看，当我创建一个印刷品或网站，实际上帮助客户发展业务或增加收入时，我也发现了很大的满足感。当我看到一个客户获得顶级搜索排名，并听说他因此受到老板的表扬时，我觉得我为他们的成功贡献了一小部分。现在我培训销售人员和电话销售人员，当他们中的一个人学到了他们所学的东西并获得成功时，我会有同样的满足感。

## 为什么*你*做你做的事？

作家兼演说家西蒙·西内克提供了一个简单而深刻的解释，为什么像苹果电脑公司、莱特兄弟和马丁·路德·金这样的领袖能够激励他们周围的人。我简单地告诉你:

*你做什么人家不买；他们买你为什么这样做。*

换句话说，真正激励他人的公司和领导者“从为什么开始”。其余的都以“什么”开头

有道理。如果你不知道你为什么做你所做的事情，那么你怎么能让别人回应你，投票给你，或者购买你的产品呢？

我听说普通人一生中会换 3-5 次职业。在一个外包和离岸的世界里，知道自己独特的“为什么”能让你保持灵活性和适应性。

那你呢？你为什么做你的工作？你特别的“为什么”是什么？请听 Simon Sinek 的演讲[伟大的领导者如何激励行动](http://www.ted.com/talks/simon_sinek_how_great_leaders_inspire_action.html "Simon Sinek: How great leaders inspire action")，并告诉我你的想法。
