# 免费获取 Windows 8 专业版媒体中心！

> 原文：<https://www.sitepoint.com/free-windows-8-media-center/>

如果你最近升级到 Windows 8，你可能有专业版。它有几个好处，包括 Hyper-V 虚拟化、虚拟硬盘引导、文件系统加密等等。但它缺少 Windows Media Center。

Windows Media player 本身可能并不重要，但是您缺少许多电影播放所需的授权编解码器，例如 Dolby。我推测这是微软可以打折操作系统升级价格的原因之一。如果您需要这些编解码器，可以在 Windows 8 Pro 包中找到，价格为 69.99 美元。哎哟。

幸运的是，微软正在通过免费赠送 Windows 8 媒体中心包来传播一点节日前的快乐。在 2013 年 1 月 31 日之前，您最多可以申领五个许可证，因此即使您尚未升级，也值得获取一个密钥:

1.  前往[http://windows.microsoft.com/en-US/windows-8/feature-packs](http://windows.microsoft.com/en-US/windows-8/feature-packs)
2.  向下滚动页面到表单，输入您的电子邮件和验证码，然后点击“发送我的产品密钥”。
3.  等等。钥匙可能需要两三天才能到达。
4.  从魅力菜单(Win+C 或指向屏幕的右上角或右下角)中点击**搜索**，在搜索框中输入“添加功能”，然后点击其正下方的**设置**。
5.  点击**向 Windows 8 添加功能**，然后点击**我已经有一个产品密钥**。
6.  复制粘贴产品密钥，点击**下一步**，等待升级下载安装。请注意，在此过程中它会自动重新启动—首先保存您的工作！

免费的东西总是好的… *即使它是在你早期的 Windows 安装中提供的！*
