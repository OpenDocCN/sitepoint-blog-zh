# 你会使用 ShareThis 进行社交媒体分析吗？

> 原文:[https://www.sitepoint.com/sharethis-social-media-analytics/](https://www.sitepoint.com/sharethis-social-media-analytics/)

我本周收到了一封电子邮件，称社交媒体共享小工具 [ShareThis](http://sharethis.com/) 正在推出其标准共享功能的两个新方面，社交范围和受众指数，以增加用户对其共享内容的价值和范围的洞察力。

## ShareThis 有什么新功能？

社交范围是你可能已经知道的 ShareThis 按钮与计数器的组合。新按钮显示基于影响(分享，加上入站社交流量)衡量内容的数字，而不仅仅是出站分享。

受众指数旨在帮助用户了解并比较他们的社交受众与其他在网络上影响、回应和参与的受众。你将能够看到访问你的网站的人的类型，并将你的内容与其他类似的网站进行比较，然后使用这些数据来改善用户体验。

ShareThis 的高管们在他们的博客上说:

> 我们很高兴社交接触和受众指数为我们的出版商带来了可能性，无论是对于试图快速做出内容决策的编辑，还是对于希望验证和更好地定义其社交受众的产品和广告团队。

## 这对你来说意味着什么？

我们都知道，使用社交网络作为商业营销工具的很大一部分有效地依赖于观众的参与。无论你是否使用 ShareThis，这种分析的增加意味着社交媒体的日常用户更容易真正了解他们分享的内容的价值和用途。一旦我们知道了这一点，我们就可以做出明智的决定，决定我们应该分享什么，和谁分享。

就我个人而言，自从今年早些时候我把我的博客从 TypePad 转移到 WordPress 后，我就没有使用过我的 ShareThis 帐户，但我认为这是对一项存在已久的服务的一个非常有趣的补充。他们还推出了一个新的增强版 [ShareThis WordPress 插件](http://wordpress.org/extend/plugins/share-this/)，我打算去看看。

我完全赞同更多地了解我如何使用社交媒体，哪些有效，哪些无效，尤其是如果它能帮助我节省时间，并把我的内容提供给那些真正需要它的人。
