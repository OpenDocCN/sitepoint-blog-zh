# 受益的朋友:社会媒体营销手册

> 原文:[https://www . site point . com/friends-with-benefits-a-social-media-marketing-handbook/](https://www.sitepoint.com/friends-with-benefits-a-social-media-marketing-handbook/)

最近，我有机会读了 Darren Barefoot 和 Julie Szabo 的《 *[【朋友与福利:社交媒体营销手册】](http://nostarch.com/friends_benefits.htm)*》的预览版。这是一个有趣的阅读，为那些从社交媒体起步的人提供了很多价值。但是，尽管初学者的重点，这本书提供了一些伟大的提示，即使是最有经验的营销人员。这是我从这本书里看到的亮点。

**一个伟大轶事的开始**

这本书从一个叫德里克的家伙开始，他是一个“80 年代中期的书呆子”，以及他与公告板系统的关系。德里克的故事为互联网的历史以及 BBSs 的社交方面如何为我们当前的社交媒体网络铺平了道路提供了一个很好的引子。

Derek 的故事以一种深思熟虑且非常清晰的方式帮助解释了从“只读”Web 1.0 到社区驱动的交互式 Web 2.0 的转变，这种转变的核心是通过写作、分享和在群体动态中交流来参与。

**从哪里(以及如何)开始**

对于门外汉来说，应对社交媒体世界可能是极其困难的，有时甚至对于人群中最有经验的人来说也是如此。*利益之友*通过问一些非常尖锐的问题来帮助你找出最佳方法，然后列出你开始行动需要采取的步骤。

这本书提供了大量的指导，鼓励读者首先找出他们参与社交媒体的动机，这是导航社交网络的宝贵部分。

**关于博客的一切**

博客是社交媒体最强大的元素之一，这本书以广泛的覆盖面恰当地对待了它们。讨论的一些主题包括:

*   你应该(和不应该)写些什么
*   与你的读者互动
*   如何处理具有挑战性的评论
*   建立你的社区的方法
*   如何联系博客同行
*   创造一个推销来帮助你被发现

**社交媒体其他方面的概述**

就所涵盖的社交媒体元素而言，*有利益关系的朋友*非常全面。有几章专门介绍一些最受欢迎的社交媒体网站，包括脸书、YouTube、Twitter 甚至 MySpace。虽然这些章节中的每一章肯定都可以扩展，但提供的信息对于任何使用这些渠道作为其社交媒体活动一部分的人来说都是有用的和最新的。

**社交网络的注意事项**

这本书里我最喜欢的一章提供了与他人在线交流的最佳方式的宝贵建议。仅这一章就有许多宝贵的建议，可以帮助你与网上的人建立、维持和加强关系。

它涵盖了从倾听的重要性，到建立正面的声誉，再到发送垃圾邮件、撒谎和误导你试图影响的人的危险。

**不全是温暖而模糊的**

虽然 *Friends with Benefits* 广泛概述了参与社交媒体的所有好处，但这本书当然不仅仅呈现了幸福的结局。这本书的一章谈到了参与社交媒体营销的风险，例如不成功，太成功太快，以及得到你没有完全准备好的反馈。

然后，这本书进一步探讨了危机管理，以及如果在你的社交媒体营销活动中发生最坏的情况，如何处理。作者甚至提供了一些适用于任何社交媒体灾难的规则，来帮助你站稳脚跟……甚至把它变成你的优势。

**了解更多信息**

总的来说，对于准备进入在线社交媒体营销世界的个人，尤其是企业来说，这是一本很好的书，因为它为这个新的和不断增长的文化和社区提供了一个很好的入门。
