# 99designs 推出现成的标志商店

> 原文：<https://www.sitepoint.com/99designs-launches-ready-made-logo-store/>

[![readymade-logos](img/55ce384bc5c32922ac7ab0ed49cba328.png "readymade-logos")](http://99designs.com/readymade/logos) 我很兴奋地向大家透露我们来自的朋友们正在进行的一个有趣项目的细节。介绍 [99designs 现成 Logo 店](http://99designs.com/readymade)。这个想法是，你可以浏览一个商标设计目录(由 99designs 社区设计),然后花 99 美元购买“现成的”,根据你的企业名称定制。

我对研究和开发这项服务有第一手的了解，这项服务是 99designs 社区一段时间以来高度要求的功能。与他们广受欢迎的[设计竞赛](http://99designs.com/contests)非常不同，后者是高度定制的独特设计的理想选择，新的[标志设计商店](http://99designs.com/readymade)为小企业提供了更快的周转和更具成本效益的选择。

对于设计师来说，徽标设计商店提供了一种全新的方式来从你的设计中赚钱并展示你的才华——徽标可以独家购买，也可以非独家购买，所以你可以以每张 99 美元的价格多次销售同一徽标。

99designs 的工作人员目前正在“备货”,首批标识将于 3 月份上市销售。然而，已经提交的徽标目录令人印象深刻！我鼓励你去看看… [99designs 标志店](http://99designs.com/readymade)。
