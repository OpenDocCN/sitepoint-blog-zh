# Sublime2 vs Notepad++

> 原文:[https://www.sitepoint.com/sublime2-vs-notepad/](https://www.sitepoint.com/sublime2-vs-notepad/)

我听到每个人都说**“JavaScript 需要一个好的 IDE”**…Notepad ++作为一个编辑器是很棒的。但是随着最近发布的 Sublime 2，这种转变是值得的！

## (在我看来)Sublime2 可能比 Notepad++更好的原因:

*   **jQuery 函数模板**——这就是传说中的！如果你简单的输入$。然后按 tab 键，它会自动加载 ajax 函数的模板。这也适用于所有其他的 jQuery 函数，您可以定制它们来满足您的需求。
*   速度–两者都是速度极快的代码编辑器，但是 Sublime2 能够更快地找到文件、切换文件和在文件中查找。
*   **独立于平台的**–它可以在 PC/Windows 和 MAC 上运行！
*   **定制**–两者都可以定制，但 Sublime2 有一个基于文本/XML 的设置，非常容易更改
*   **自动完成方括号**–sublime 2 是标准配置，当您键入时，它会关闭方括号
*   **1000 英尺处的代码**–右边的滚动条更大，有代码的可视化视图
*   **智能标签更改**–当您创建新代码时，您可以通过标签浏览值，它会直接带您快速编辑。
*   **多重编辑**–您也可以用多个光标同时选择多个区域并进行编辑。

参见[Sublime 2 特性的完整列表](http://www.sublimetext.com/features)。
