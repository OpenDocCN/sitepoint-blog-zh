# 设计你的网上成功

> 原文:[https://www.sitepoint.com/design-online-success/](https://www.sitepoint.com/design-online-success/)

网站设计一直是我着迷的领域，不仅仅是在平面设计领域，还包括流程、导航、外观和内容。我喜欢上网冲浪，几乎完全是为了了解不同的感觉和口味。在这样做的过程中，我惊讶地看到一些网站看起来平滑而精致，而另一些网站看起来非常快——即使公司、产品或服务质量很高。

但是网站设计本身就是一个强大的营销过程。许多人倾向于忘记，在商业(和生活，就此而言)的所有领域，人们都在编造假设——无意识的平行假设。换句话说，当他们访问一个网站时，他们无意识地认为网站的设计和其背后的商业存在相似之处，更不用说它所推广的产品和/或服务了。如果设计很差、不专业或不清楚，人们会认为产品或公司都是一样的。

考虑到人们“以貌取人”的倾向，这在网络上是非常重要的，因为设计是唯一将你和你的顾客区分开来的东西，因此，它代表了整体。你的网站可以强调、支持或反驳你的营销信息——而且几乎不费吹灰之力，甚至不引人注意，有时甚至引人注目。

一家大型航空公司最近在乘客中进行了一项调查，以便进行一些营销研究。下面的问题被问到:“如果你的餐盘很脏，你会认为航空公司对引擎的维护也很差吗？”答案是，尽管听起来不合逻辑，但对绝大多数参与者来说是。

在《营销的 22 条不变法则》中，里斯和特劳特提出了我认为是商业世界中最强有力的观点，即营销不是产品之战，而是认知之战。我的导师曾经说过，“感知的真理比真理本身更有力量”，我同意。营销就是感知。

感知价值也是如此。如果你把你的网站和竞争对手放在一起，并且你们都以同样的方式、同样的价格提供同样的产品，那么将会赢得客户的公司将会通过它的设计向客户传达他们的选择中隐含着附加价值。

在我的研讨会上，我教授一些我称之为“番茄酱原则”的东西假设你刚刚遇到一个销售人员，他在介绍完自己后，给你做了一个销售演示。他穿得绝对无可挑剔，发表了完美的演讲，对满足你的需求非常感兴趣，并与你进行了一次非常完美的会面。但是在整个相遇过程中，你会情不自禁地注意到他的领带上有一个小污点——一个番茄酱小污点，如果你愿意的话。然而，两个星期后，如果我问你，“关于你的会议，你记得最清楚的是什么”，很可能首先想到的是番茄酱污渍！

正如老话所说，“你永远不会有第二次机会来留下一个好的第一印象！”这适用于你做的或展示的任何事情，甚至是最简单的事情，比如你的网站设计。因此，要注意你的网站的整体外观、吸引力、易导航性，以及最重要的内容。
