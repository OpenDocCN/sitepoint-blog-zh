# Photoshop 的光度模式

> 原文：<https://www.sitepoint.com/photoshops-luminosity-mode/>

Photoshop 是这样描述光度模式的:

> 使用基色的色调和饱和度以及混合色的亮度创建结果色。此模式创建了与颜色模式相反的效果。

只是为了证明这一点，我会采取一个樱花层，复制它，并降低其中一层的饱和度。

![](img/6f239d68ba493fe06547475e5bd953f6.png)

![](img/a5b5fbfced00f90ca546533998075db3.png)

我将两个图层都设置为“亮度模式”,一次一个，放在一个玫瑰图层上。两者得到完全相同的结果:

![](img/912f5fd5fc784a1fb4fe85e60df472e8.png)

混合层(樱花)的灰度应用于基础层(玫瑰)的颜色。

作为一种实际应用，我将从玫瑰层开始，通过点击图层面板底部的调整层图标，然后选择“曲线”来应用曲线调整层:

![](img/44f90d8d41107b91fb48d70a100c0799.png)

我在线上单击两次，然后拖动并调整曲线，如下所示:

![](img/e2a68705318141a978c2c9d0eafda3f2.png)

正如你所看到的，这使得图片有点“暗”或饱和。黄色真的很亮；图像中较暗的区域变得更暗了。

现在，我设置曲线层的模式为“光度”

![](img/a69668f6a099339b211de2a563643f7d.png)

这保留了原始图片的颜色，但保留了曲线调整层的对比度，并给图像一种近乎老照片的效果。
