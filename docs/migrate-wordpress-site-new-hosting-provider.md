## 首先，WordPress 迁移包括什么？

将你的 WordPress 站点迁移到一个新的主机提供商需要三个步骤:

1.  **传输 WordPress 文件**，包括 WordPress 本身，你的主题和插件，以及你的媒体文件。
2.  迁移包含你的文章和评论的 WordPress 数据库。
3.  **重新配置 WordPress 在新的服务器**上工作，并且更新你的 DNS 记录指向新的主机。

根据您可用的选项，有多种方法来完成每个步骤。检查您的主机提供商的文件，以获得他们的建议。

通常，你使用 FTP 或者你的主机提供商提供的在线文件管理器来传输 WordPress 文件(步骤 1)。您可以通过在当前服务器上压缩文件，然后在新服务器上解压缩来节省时间。使用您的主机数据库管理工具(可能是 phpMyAdmin)可以完成 WordPress 数据库的迁移(步骤 2)。

或者，您可以通过使用以下工具或服务来简化流程。

## 使用备份插件启动您的迁移

希望你定期备份你的站点。如果是，为什么不将当前备份作为迁移的第一步呢？你刚刚给自己省了一半的工作。

插件使备份变得容易。以下是一些好的例子:

*   Automattic 的个人计划每年花费 39 美元。工作人员显然知道所有关于 WordPress 的事情，并且可以帮助备份和恢复你的站点。
*   [备份 Buddy](http://ithemes.com/purchase/backupbuddy/) (备份、恢复、移动)一个站点每年的费用为 80 美元，还有其他可用的计划。这是一个备份插件，能够将 WordPress 安装克隆到一个新的位置，这一切都可以从你现有站点的仪表板上完成。
*   [UpDraftPlus](https://wordpress.org/plugins/updraftplus/) 很受欢迎，活跃安装数超过 100 万，而且是免费的。(提供高级版本。)免费版进行备份，而高级版(见下文)进行克隆和迁移。
*   目前有 200，000 多个活跃安装，并且是免费的。
*   [WordPress 备份到 Dropbox](https://en-au.wordpress.org/plugins/wordpress-backup-to-dropbox/) 有 90，000 多个活跃安装，并且是免费的。

一旦您备份了您的站点，您将需要在新服务器上执行恢复。查看插件的文档，了解推荐的方法。你可以按照上面的步骤手动恢复，或者一些插件提供帮助。

## 使用迁移插件移动您的网站

将插件的使用更进一步，一些插件能够备份和恢复您的站点，在一个工具中提供完整的迁移。

以下是一些能够克隆和迁移你的 WordPress 站点的插件:

*   [Duplicator](http://wordpress.org/plugins/duplicator/) (备份、复制、克隆)是一个流行的解决方案，有 900，000+的活跃安装，并且是免费的。 **[Duplicator Pro](https://snapcreek.com/)** (“几分钟内迁移一个网站”)有额外的功能，个人许可证费用为 39 美元。这个插件有很多有用的特性，但是对于初学者来说不是最容易理解的。
*   [一体式 WP 迁移](https://wordpress.org/plugins/all-in-one-wp-migration/)拥有超过 400，000 个活跃安装，并且是免费的(带有高级扩展)。它会将你的站点迁移到一个新的 WordPress 安装中，这个安装也必须安装并激活插件。这一个有一个容易使用的界面。
*   WP 学院的 WP 克隆版已经有 30 万以上的活跃安装，并且是免费的。它将你的站点迁移到一个全新的 WordPress 安装中，而不使用 FTP。虽然评价很高，但有些人报告了一些奇怪的情况，开发人员承认在 10-20%的情况下可能会失败，尤其是在迁移较大的站点时。
*   [UpDraftPlus Premium](https://updraftplus.com/) 是一款高评级的备份插件。收费 70-145 美元的高级版本可以克隆你的网站，并把它移植到一个新的 WordPress 安装上。
*   超级备份&克隆-迁移是一个全面的具有迁移功能的备份插件。售价 34 美元，可从 CodeCanyon 购买。
*   WP Migrate DB 目前有超过 200，000 个活跃安装。个人计划的 12 次安装费用为 89 美元。如果你愿意通过 FTP 将你的文件传输到新的主机，这个插件将为你完成更复杂的数据库迁移。

关于最好的 WordPress 备份和迁移插件的更多细节，请看杰夫·史密斯的文章， [5 个 WordPress 备份和迁移插件](https://www.sitepoint.com/5-wordpress-plugins-backups-migrations/)。

## 将工作移交给迁移服务提供商

与其自己执行迁移，您可能更愿意花钱请别人来帮您完成。也许你缺乏时间或专业知识。或者可能有很多利害关系——比如一个企业网站——把这项工作交给专家是明智的。

网络开发自由职业者和各种规模的公司都提供 WordPress 迁移服务。以下是一些关于如何找到适合您的服务提供商的建议:

*   在谷歌上搜索“WordPress 迁移服务”。许多 WordPress 开发者提供 WordPress 迁移服务。有的专门做。如果你已经有了一个你信任的网站开发者，就从那里开始吧。
*   在 [Envato Studio](https://studio.envato.com/search?utf8=%E2%9C%93&search%5Bquery%5D=wordpress+migration) 中搜索“WordPress 迁移”。价格从 50 美元左右开始。
*   在 Fiverr.com 搜索“WordPress 迁移”。价格从 5 美元左右开始。
*   Fanstastik 专门从事企业构建、迁移和支持。价格从 99 美元开始，通常在两三天内完成。
*   [Valet.io](https://www.valet.io/) (前身为 WP Valet)开始专门从事 WordPress 迁移。虽然他们现在提供更多的服务，但他们仍然是任何规模的迁移服务的最佳选择之一。联系他们了解价格。

## 向你的主机提供商寻求帮助

许多主机提供商也提供迁移服务。毕竟，他们想要你的业务，所以让他们尽可能容易地从你当前的主机转向他们是符合他们利益的。

查看该公司的网站或电话客服，了解是否有这样的服务，以及费用是多少。他们甚至可以免费提供有限数量的迁移。
