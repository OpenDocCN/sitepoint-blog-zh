# 10 必须有与客户沟通的工具

> 原文:[https://www . site point . com/10-与客户沟通必备工具/](https://www.sitepoint.com/10-must-have-tools-for-communicating-with-clients/)

沟通你的客户是合同网站开发、设计、编程、写作或任何其他自由职业艺术形式不可或缺的一部分。有些客户很棒——他们从一开始就非常清楚地表达了他们的需求，从规格到交付一切都很顺利。而另一些，则会让你沮丧地想要拔光自己的头发。下面我们总结的十个必须具备的基于网络的工具将帮助你与任何一种类型的客户沟通，通常会让你的事情变得更容易，并帮助你保持理智。一如既往，如果你对你使用的应用程序有任何其他建议，请在评论中告诉我们。

[![](../Images/75616d428fb24698d46abe23f8f7a90b.png "backpack-logo")T2】](http://www.backpackit.com/)

37signals 的一个很棒的应用程序，背包，可能是你应该开始的地方。对于任何自由职业者来说，这也是一个必须拥有的武器库，它可以让你把所有的东西都放在一起。你可以在一个背包页面上组织一个客户项目的所有细节，然后与你的客户分享这个页面，让他们了解正在发生的事情。您甚至可以授予他们编辑权限，这样他们就可以随着项目的进展进行修改或向您提供反馈。

[![](../Images/908cf36e167d975eec1a69461709bc9c.png "etherpad-logo")T2】](http://www.etherpad.com/)

今天，一个由前谷歌员工组成的团队大张旗鼓地成立了 Y Combinator 创业公司， [EtherPad](http://www.etherpad.com/) 实现了极其简单的协作写作。用户可以在同一个文档上一起写，实时地看到其他协作方在修改文档时修改了什么。这是一个很棒的应用程序，可以和 Skype 一起使用，与客户讨论项目规范。确保你从一开始就掌握了所有的细节，避免以后遇到麻烦。

[![](../Images/bac9de5b1408835aac66dfaccc9ba085.png "dimdim-logo")T2】](http://dimdim.com/)

有时，您需要与客户共享的不仅仅是简单的文本文档。也许你需要向客户介绍对他们的应用程序所做的更改，培训他们如何将内容输入到新的 CMS 中，或者向他们展示徽标设计推介。无论什么原因，当你需要演示一些东西时，你需要一个强大的开源屏幕共享和网络会议应用程序。

[![](../Images/2aa67dbd86fecf9dcbbba97c9e4491c4.png "senduit-logo")T2】](http://www.senduit.com/)

很多时候，在一个项目中，你需要和客户来回发送文件。电子邮件不适合大文件，私人 FTP 服务器可能会超出你客户的能力。我们喜欢用 [senduit](http://www.senduit.com/) 来来回回地传递文件。它超级简单，并且有一个慷慨的 100mb 限制。文件会在一段时间(30 分钟到一周)后被销毁，因此您不必担心私人客户信息泄露。

[![](../Images/8e4195349964f009ffafe10c87c475fa.png "rememberthemilk-logo")T2】](http://www.rememberthemilk.com/)

当你和多个客户一起工作时，很容易忘记需要做什么。RememberTheMilk 是最早的待办事项列表网络应用之一，也是最好的应用之一。它简单易用，可以在各种平台上运行，并且可以与客户共享列表，以便他们可以随时了解您的进度，或者在必要时添加或澄清项目。

[![](../Images/6cdd97a36d433dad053e89db3030964c.png "harvest-logo")T2】](http://www.getharvest.com/)

在与客户沟通时，有一件事你肯定会想尽力去做，那就是你完成了多少工作，花了多长时间。我们推荐 [Harvest](http://www.getharvest.com/) ，这是一个时间跟踪应用程序，也可以处理您客户的发票和账单。如果你把时间跟踪和其他客户沟通联系起来，他们永远不会想到他们欠你的比他们少。

[![](../Images/056e898075b82aa2898a31b3f44c268a.png "faxzero-logo")T2】](http://www.faxzero.com/)

传真可能是一个相当古老的学校方法，但有时它仍然是必要的(例如，传真签署的合同)。如果你只是偶尔发传真，投资一台传真机可能没有意义。而是用 [FaxZero](http://www.faxzero.com/) 免费发传真。如果你也需要接收它们，使用 [jConnect](http://www.j2.com/) 来免费接收。

[![](../Images/b4d5f8274a6e2b69bf38cd3aae6bf686.png "highrise-logo")T2】](http://www.highrisehq.com/)

现在你已经开始了所有的交流，你需要记录下你在和谁交流，说了什么，谁需要回电。37signals 的另一个应用程序 Highrise 非常适合跟踪每个人。它是一个客户关系管理器和地址簿工具，围绕着将任务与人联系起来的概念而设计(例如，“我必须回复 X 教授的一封电子邮件，回复:我与金刚狼的牛肉”)。

[![](../Images/fcd05c86a81e2e0a2500cfe2932969ef.png "conceptshare-logo")T2】](http://www.conceptshare.com/)

这个是给设计师的。与客户来回发送概念邮件，等待他们的回复和反馈是笨拙的。这会导致令人困惑的电子邮件，比如，“我喜欢概念 A 的这一部分，但概念 B 的这一部分。如果你能把概念 A 和概念 C 合并，并使用概念 B 的颜色，我想我们会更接近。”啊！进入 [ConceptShare](http://conceptshare.com/) ，这是一个专门创建的网络应用程序，用于获得对设计的反馈，使其保持有序，并使设计项目的协作更加容易。让你的客户参与反馈过程的每一步，消除头痛(并增加你赢得回头客的机会！).

[![](../Images/5d735d89238a871e37237c625dbf89bf.png "fluxiom-logo")T2】](http://www.fluxiom.com/)

对于那些从事大型项目的人来说，senduit 可能不适合来回传递文件。如果你被客户的文件淹没，你需要一个资产管理应用程序，比如 [Fluxiom](http://www.fluxiom.com/) 。Fluxiom 是一个超级光滑的基于网络的资产管理器，非常适合处理大型、庞大的项目，这些项目需要控制很多部分。
