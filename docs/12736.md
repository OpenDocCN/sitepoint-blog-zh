# Target 以 600 万美元解决无障碍诉讼

> 原文：<https://www.sitepoint.com/target-settles-accessibility-lawsuit-for-6-million/>

昨天，塔吉特百货公司与美国盲人联合会达成了集体诉讼协议(该诉讼最早于 2006 年初发起)。在和解中，塔吉特同意向 NFB 支付高达 600 万美元的赔偿金，赔偿金随后将分配给因 target.com 网站无法访问而受到影响的个人。

我不得不对公告的以下部分窃笑:

> 布鲁斯·塞克斯顿(Bruce Sexton，Jr .)从一开始就是该案的具名原告，他补充道:“这一和解标志着盲人访问网站的新篇章。我对 Target 致力于成为在线无障碍领域的领导者表示赞赏。”

这起诉讼存在的事实本身就足以让我们得出结论，塔吉特远非“在线无障碍领域的领导者”。Sexton 花费了相当多的精力游说 Target 在他们的网站上添加基本的可访问功能，包括在他们的图片上添加文字，但是他们拒绝考虑他们可能有义务这样做的想法。

Target 最终同意他们有义务接纳盲人游客的声明基本上是一个好消息，尽管有人担心患有其他残疾的游客可能不会被考虑在内，或者和解金额太低。事实是，这一解决方案将可访问性提上了公司的议事日程，否则这些公司可能会认为忽视残疾访问者访问他们的网站是可以接受的。

向可能因为视力不好或不会用鼠标而起诉塔吉特的游客支付 600 万美元，塔吉特不想再经历一次。现在已经开了先例；下一次 target.com 的可及性有问题，有人抱怨时，我打赌塔吉特会听的。
