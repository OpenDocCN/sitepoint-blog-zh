# 国会议员为网站创意采用众包

> 原文：<https://www.sitepoint.com/congressman-uses-crowdsourcing-for-ideas/>

政治家通常不以他们的技术实力而闻名，但是美国国会议员迈克·本田却逆势而上，坚信 Web2.0 技术可以改变公民和政府之间的关系。

他最近呼吁立法公开所有非个人数据；包括警察、卫生服务、社会服务和政府部门掌握的信息。最终，政府官员将被要求开放所有数据，除非他们能够证明为什么这些数据不应该免费提供给为其创造买单的纳税人。他指出:

> 在我们的 web 2.0 世界中，我们可以通过向公众提供原始数据来赋予他们权力，他们可以以新的和创新的方式重新混合和重用这些数据。有了这些工具，公众可以在项目上合作，帮助立法者制定更好的政策来应对我们国家面临的紧迫挑战。

本田认为，如果有人要求公开信息，他们应该得到他们想要的:这可能是一个迹象，表明他们理解数据的价值，而政府官员可能不理解。

本田认为政府 2.0 是一个可以实现的目标，并已开始全面重新设计他的网站。他的主要目标是开发与委托人合作的新方法，并且他正在使用社交网络工具从 web 开发社区征求想法。最具创新性的想法将在所有国会议员中分享。
