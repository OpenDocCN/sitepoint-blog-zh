# 关于 Internet Explorer 7.0 的更多信息

> 原文：<https://www.sitepoint.com/more-on-internet-explorer-70/>

关于即将到来的 IE7.0 的谣言开始满天飞。虽然微软只说新版本的浏览器将专注于安全问题，但 web 开发社区一直希望 MS 能借此机会改善 IE 落后于竞争对手的一些领域，如 CSS 支持和用户界面。虽然完全未经证实，但微软的合作伙伴已经开始透露一些建议，他们可能已经看到了这款浏览器的早期版本。微软观察写道，IE7 将有标签，国际域名支持，和一个内置 RSS 聚合器。此外，它将能够正确显示透明的 png。对于网络社区来说，更好地支持 CSS 仍然是一个悬而未决的大问题；看起来 IE 团队会改进 CSS 支持，但是在这个阶段不会承诺实现整个 CSS 标准。
虽然一个改进的网络浏览器应该是一件值得高兴的事情，但这并不意味着世界突然变得一片美好。不是每个 Internet Explorer 用户都会升级到 IE7，特别是因为新版本只对 Windows XP SP2 用户开放。【2005 年 2 月柜台的浏览器统计显示，IE6 占有大约 80%的市场份额，IE5.x 和 Mozilla/Firefox 各占大约 10%,其他浏览器不到 1%。众所周知，浏览器的统计数据对于做出正确的决策来说是不可靠的，但它们可能会给人一种粗略的印象；IE5.x 的公共使用率和 Firefox 差不多，我们不会忽视 Firefox。很可能类似的趋势将适用于 IE7 即使假设 Internet Explorer 保持其非常主导的市场地位，我们还需要 IE6(可能还有 IE5)支持几年。
