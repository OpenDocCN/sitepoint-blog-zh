# 标志设计 101:什么是标志？

> 原文:[https://www.sitepoint.com/logo-design-101-what-is-a-logo/](https://www.sitepoint.com/logo-design-101-what-is-a-logo/)

我们每天都被商标轰炸。它们在我们的衣服上，在我们的车上，在我们的手机上，在我们的街道上，几乎在我们看到的任何地方。那么到底什么是 logo 呢？

徽标是一个公司的符号、名称或商标。公司使用徽标是因为它们代表了公司的简明形象。一张图片，就像他们说的，能说明千言万语。人们通常发现简单的图片比单词更容易记忆。我们的眼睛会被视觉对象吸引，而设计良好的标志会给印刷文档和网页增加视觉吸引力。

标志设计肯定是平面设计中最具挑战性的领域之一，并且经常被严重低估。从个人的角度来看，当流程结束，顾客带着满意的 logo 离开时，我觉得要求很高，但非常满意。标志设计的挑战是把一个想法或概念压缩成一个符号或一个字体。

标志只是整体品牌或企业形象的一部分，但显然是非常重要的一部分。对一家公司的认知永远不会仅仅基于一个标志，它的产品、员工和服务都是品牌的一部分，但标志可以帮助在客户的脑海中形成一幅画面。这里有几个商标的例子，这些商标来自拥有一些忠实粉丝的品牌。

## 什么是好的标志？

最好的标志都具有以下品质

### 1.它们很简单。

标志需要在黑色和白色以及你可以选择的颜色。他们需要在不同的尺寸下工作。您可能需要在棒球帽上放置一个小徽标，并在公共汽车的侧面使用相同的徽标。该徽标可能需要在网站、报纸或电视上使用。要实现这一切，简单是必需的。

[![london-underground-logo](../Images/1171b516e5b172da928bb420f4eb375d.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/04/londonundergroundlogo.png)

### 2.它们在美学上令人愉悦。

一个标志可以是印刷处理或符号(或两者都是),但它需要吸引观众的眼球。经典的世界野生动物基金会标志很漂亮。

[![wwf_logo](../Images/7dd759eb85d4c9efb971a60bced66322.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/04/wwf_logo.jpg)

### 3.他们传达一个想法，一个原则或一个描述。

公司使用标志进行品牌识别。你注意到亚马逊标志上的橙色箭头是如何从 A 指向 Z 的吗？亚马逊的确出售所有种类的书籍，不是吗？

[![amazon_logo](../Images/d385af7c67d7fa4d813644e0d6184bce.png)T2】](https://www.sitepoint.com/wp-content/uploads/2010/04/amazon_logo.jpg)

所以所有这些品质都必须融入一个小小的标志中。正如已经提到的，这是一个相当大的挑战。你有过标志设计的经验吗？我很想在评论中听到你的想法。
