# Windows 7:第一个月回顾，第 1 部分

> 原文:[https://www.sitepoint.com/windows-7-review-1/](https://www.sitepoint.com/windows-7-review-1/)

经过数月的公开测试，微软 Windows 7 于 2009 年 10 月 22 日低调发布。这个操作系统卖得很好——或者至少比 Vista 好得多。我怀疑你已经阅读了许多评论，但是很少是基于在相当长的时期内使用操作系统进行开发的。

以上是我对 Windows 7 专业版 64 位最终版本的体验。这篇文章分为三部分。今天，我们来看看安装和界面。第 2 部分涵盖了[生产力、软件和安全性](https://www.sitepoint.com/windows-7-review-2/)。最后的总结涉及微软和 Windows 的性能和未来。

## 装置

与以前的版本相比，安装 Windows 7 轻而易举。你可以从 Vista 升级，但我不推荐。抓住机会清理杂物，进行干净的安装，去掉制造商*有益地*安装的垃圾。只要确保你已经备份了你的数据，并有你需要的所有软件的副本。

为 32 位和 64 位版本提供了两张 DVD。大多数不到几年的电脑都支持这两个版本，但是你应该检查一下。64 位操作系统是未来的趋势，许多原始设备制造商默认安装它。然而，您不太可能遇到令人困惑的 32 位版本兼容性问题…稍后将详细介绍。

Windows 支持大量的硬件设备，但是在开始之前下载最新的驱动程序是很重要的。Windows 7 在很大程度上与 Vista 驱动程序兼容，但要确保你获得了正确的版本，尤其是如果你要升级到 64 位版本的话。

一旦你冒险尝试，在一台现代电脑上安装一个完整的 Windows 7 大约需要 30 分钟。这没有某些 Linux 发行版快——尤其是考虑到包含的软件如此之少——但等待时间不会太长。Windows 将在一定程度上帮助你安装驱动程序，但我仍然需要为几个无法识别的设备手动安装软件。

## 界面

如果你喜欢 Vista 的 Aero 界面，你会爱上 Windows 7。当然，反过来也是如此。

就我个人而言，我关掉了大部分恼人的效果:看着一个窗口第九百次嗖嗖地进入视野变得令人厌倦。也就是说，当我回到 Windows XP 时，我怀念 Aero 的外观。

最显著的变化是任务栏。它现在是两倍高，默认情况下只显示一个应用程序的图标。

![Windows 7 taskbar](../Images/b2b440e2e36361c7ea38551ed18f8758.png)

图标可以重新排列或永久固定在任务栏上。这是一个我们从 Windows 95 开始就一直在等待的受欢迎的附加功能！

应用程序窗口被分组在一起。将鼠标悬停在图标上会显示所有打开窗口的缩略图，您可以单击这些窗口来激活它们。您也可以将鼠标悬停在缩略图上来“窥视”窗口(所有其他窗口暂时变为透明)。最后，您可以将鼠标悬停在任务栏右边缘的块上或单击该块，分别查看桌面或最小化所有窗口。

像许多开发人员一样，我经常使用打开许多窗口的应用程序，例如 Firefox、source、console 等。就我个人而言，我发现悬停在任务栏图标上，等待缩略图，然后切换到右侧窗口是一个缓慢的过程。要更改它，右键单击任务栏，选择属性，然后将**任务栏按钮**设置更改为*当任务栏已满时合并*。

窗口管理有几个小的改进。你会经常发现新功能，但这里有一些更好的:

*   将窗口拖到屏幕顶部可以最大化它。
*   将窗口拖到一边可以最大化高度，并将宽度设置为空间的一半(在比较浏览器渲染时很有用)。
*   摇动窗口会最小化所有其他应用程序。
*   Alt+Tab 和 Windows Key+Tab 仍然提供键盘窗口切换。

开始菜单几乎与 Vista 相同。它显示了最近使用的应用程序列表、搜索框、所有程序的链接、最近的文档、控制面板等。它运行良好，但我发现我不再像在 Vista 中那样经常使用它了。

Windows 7 中也有 Vista 侧边栏小工具，但它们现在可以放在桌面上的任何地方。Kinesys 的发射控制小工具让我的生活变得更加轻松，你会找到自己的必备工具。不幸的是，小工具可能很脆弱，如果一个小工具出现故障，小工具系统就会变得不稳定。当一个小工具不能 ping 通它的家庭地址时，经常会发生这种情况。

最后，微软确实需要提高界面的一致性。一些应用程序具有 Office 2007 样式的功能区。有些有 XP/Vista 风格的任务窗格。许多标签式对话框已经 15 年没变过了。这对任何关注 Windows 发展的人来说都不是问题，但新用户可能会迷失方向。
