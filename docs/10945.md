# 有效指导自己的 7 个步骤

> 原文：<https://www.sitepoint.com/7-steps-to-effectively-coaching-yourself/>

昨天，我写了自我指导的许多好处。但是，说实话。自我辅导很难。为了客观地看待一个围绕着你的情况，把你自己从这个情况中抽离出来是极其困难的。正如保罗·巴恩在他对我之前的帖子的评论中所说，“……作为人，我们很容易欺骗自己。”这种缺乏客观性正是自我指导如此具有挑战性的原因。

虽然很难，但并不意味着不可能。你可以通过采取系统的方法并完全投入到这个过程中来指导自己走向成功。以下是我为了成功指导自己度过过渡期而采取的步骤的概述。

## 第一步:确定你的目标

在你深入细节之前，先弄清楚你为什么要这么做。你为什么要自学？你的愿景是什么？你想实现的一个具体目标是什么？它可能会增加清晰度，让你可以做出一个困难的决定，一种新的方式来优先考虑一个超负荷的时间表，甚至是一个行动计划，为一个新的业务努力，你即将投入。

## 第二步:全部发泄出来

我认为这是自我训练最重要的步骤之一——头脑风暴。毫无疑问，你有很多想法、问题和难题在你的脑海里跳动；你需要把这些碎片打包、组织起来并区分优先次序，这样你就能以清醒的头脑应对挑战。把它们写在纸上，这样你就可以从混乱中走出来。

## 第三步:对自己负责

艰苦的工作从这里开始。这一步是任何辅导过程中的主要步骤之一。为什么会出现这种情况？你是不是失误了，现在需要恢复？你是否承担了太多工作，面临着工作过剩？你是否推迟了本该做的决定？如果你目前的处境有一个原因指向你，那么现在是时候承认了，这样你就可以改正了。

## 步骤 4:创建优先行动计划

回到你的大脑垃圾场，像对待任何面向项目的头脑风暴一样对待它。你的清单上不断出现的模式是什么？它们与你的目标有什么关系？根据这个清单制定具体的行动，让你从起点走向目标。

## 第五步:给你的行动附上日期

你的行动计划需要包括每项行动的最后期限，以便让你朝着目标前进。牢记你的责任，承诺你设定的每一个截止日期。

## 第六步:停下来审视现实

停下来，回到你的行动计划。把它看做是为别人制定的计划。你觉得可以实现吗？为了赶上最后期限，你必须放弃什么？你愿意做出这些牺牲吗？你生命中最亲近的人同意你做出这些牺牲吗？对自己诚实，必要时回去修改。

## 第七步:不要单干

是的，自我指导就是让你自己经历一个分析和改变的过程，这样你就能准备好处理大事情。但这并不意味着你必须独自完成。一旦你有了行动计划，就请你的导师、朋友、配偶或其他值得信任的人来执行细节，以确保你头脑清醒。对他们的反馈持开放态度，因为如果你没有看到危险信号，因为它离你很近，这可以节省你的时间和挫败感。

根据您自己的个人需求和组织偏好，您的计划可能会有所不同，但一般步骤是通用的。让它适应你自己的需要，并致力于完成它。
