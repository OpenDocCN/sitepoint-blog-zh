# 给我看看数据！

> 原文:[https://www.sitepoint.com/show-me-the-data/](https://www.sitepoint.com/show-me-the-data/)

没有人想通读成千上万行数据来寻找趋势。但是，将所有数据浓缩成一个简单的可视化表示的最佳方式是什么呢？阅读我们最新的教程来找出答案！

Toby Tremayne 将向我们展示如何使用 Adobe 的 Flex Builder 开发复杂的数据可视化。在第一部分中，他着重于从庞大的数据集中提取有用的数据子集，并以我们可以使用的格式将其导入 Flex。

这是最酷的部分:他使用的数据集是来自我们自己的 [SitePoint 论坛](https://www.sitepoint.com/forums/)的帖子集合！当第三部分开始时，我们将让这个工具运行起来，这样你就可以看看不同的技术是如何在论坛上被讨论的。如果你以前没有去过论坛，这是一个绝佳的机会来拜访[并介绍你自己](https://www.sitepoint.com/forums/forumdisplay.php?f=51)！

我们要感谢 Adobe，他们是我们的[选择题文章测验](https://www.sitepoint.com/quiz/flex3/flex-data-visualization-part-1/)的赞助商——如果你觉得这篇文章很有趣，一定要检查一下你有多认真听讲！
