## 什么是线框？

线框是设计的骨架模型——在添加字体、颜色和任何其他视觉美学之前就存在的无样式、准系统基础。线框化通常发生在构思之后，高保真原型制作之前。

线框有时被称为“中等保真度”，线框工具的定义是这样一个事实，即它们允许设计师将现成的设计元素和组件拖放到画布上，以便他们可以在非破坏性的环境中快速试验想法，收集利益相关者的反馈，甚至进行用户测试。

### 线框在 UX 是如何使用的？

线框给设计师提供了一个仔细思考他们设计的核心 UX 的机会，即可用性、可访问性、可扫描性、可点击性等等(通常特别关注导航、表单和结账 UX)。

虽然视觉设计也有助于 UX，但设计核心 UX 旁边的“表层”可能会很快变成一个复杂的混乱，这就是为什么我们在转向高保真之前设计这些被称为线框的骨骼模型。

## [香脂](https://balsamiq.com/)

*   平台:Web、macOS、Windows
*   [定价](https://balsamiq.com/buy/):网络和桌面的多种选择

Balsamiq 是线框空间的主要参与者。

让它脱颖而出的是它(可选的)素描般的视觉美感，这提醒我们线框应该是实验性的，低/中保真度的，甚至是一次性的——这种态度让线框蓬勃发展。

![An example of the Balsamiq interface](../Images/39f00eaf9725f75a2edaa53bffe38f7d.png)

Balsamiq 是精益、敏捷团队的理想之选，其功能可以帮助设计师在真实设备上测试线框，并与利益相关者就反馈进行合作。Balsamiq 集成了 bug 跟踪功能(例如，吉拉云)，适合设计师和经理类型。

至于线框组件，Balsamiq 是装备最齐全的组件之一，是当今市场上经验最丰富的工具之一。与所有线框工具一样，只需将组件拖到画布上，就可以快速地将想法可视化。

## [异想天开](https://whimsical.co/)

*   平台:仅网络
*   [定价](https://whimsical.co/pricing/):免费计划，8 美元或 12 美元(每月)

尽管对行业来说相对较新，因此具有最小的线框组件，但异想天开是最好看的，最容易使用的，最有趣的，最后是当今市场上最便宜的线框工具(除了 Justinmind，它对“仅线框”是免费的)。

![An example of the Whimsical workspace](../Images/e2a86610718a65696e6d454b0e3bdd9f.png)

一个主要的缺点是没有任何用户测试功能。然而，其他合作者至少可以通过评论线框来添加他们的反馈，甚至可以在类似 Trello 的看板风格的板上管理 bug、任务和改进。到目前为止，这个特性集在一个[设计冲刺](http://www.gv.com/sprint/)风格的工作流程中非常有用。

此外，思维导图功能也即将推出。

## [Axure RP](https://www.axure.com/)

*   平台:macOS 和 Windows
*   [定价](https://www.axure.com/buy):团队和个人的多种选择

Axure RP，像 Balsamiq 一样，是另一个巨大的线框工具，适度地提供动态的、数据驱动的内容特性、动画特性，以及自适应和有条件的布局。这也难怪“财富 100 强公司有 87%使用 Axure”。

![The Axure interface](../Images/37527a9ad40e5e38863b4cea8dbc0277.png)

Axure 已经存在了 16 年。它唯一的缺点是缺乏用户/可用性测试功能。

## [漫威](https://marvelapp.com/)

*   平台:Web
*   [定价](https://marvelapp.com/pricing):免费计划，12 美元、42 美元或 84 美元(每月)

漫威被认为是最容易掌握的工具之一，包括其线框特征。尽管大部分工作流程都是在草图阶段完成的，设计随后会交给开发人员，但漫威不太出名的线框工具既易于使用又美观——很像异想天开的工具。

![The Marvel workspace](../Images/1a6cba408234d8df117fa66663c01b1b.png)

由于漫威[将](https://marvelapp.com/features/integrations)与用户测试工具回看集成在一起，你可以说漫威是一个更有经验的异想天开，尽管由于增加了原型和移交功能而更加昂贵，但仍然是这个列表中非常值得添加的。

## [Moqups](https://moqups.com/)

*   平台:Web
*   [定价](https://moqups.com/pricing/):免费计划，13 美元或 19 美元(/月)

Moqups 是传统风格的线框工具的现代旋转，使其特别适合经理型的利益相关者。它提供了大量被称为“模板”的线框组件，还包括交互式原型功能。

对于 UX 的设计师来说，这可能有点笨拙，但对于那些希望在 UX 设计的早期阶段让设计更加民主的团队来说，Moqups 帮助非设计师成为设计师。

![The Moqups interface](../Images/2bad3b5a3ab046a666e3c8150cdbec0d.png)

## [OmniGraffle](https://www.omnigroup.com/omnigraffle/)

*   平台:仅限 macOS
*   [定价](https://store.omnigroup.com/main/omnigraffle):标准版 99.99 美元或专业版 199.99 美元(一次性)

如果您打算使用一种工具来实现高保真设计，但又将其基础建立在线框上，OmniGraffle 是一个不错的选择。虽然它看起来不像异想天开那么漂亮，但 OmniGraffle 是一款仅支持 macOS 的应用程序，它受益于使用原生 macOS 用户界面，这使它与 Moqups 等工具相比，感觉不那么杂乱。

OmniGraffle 还拥有其他线框工具所没有的强大功能，例如使用 JavaScript 或 AppleScript 动态创建扩展的能力，这可以帮助设计师自动化他们的工作流程。正如您所看到的，OmniGraffle 比大多数线框工具更具技术性，所以如果您决定继续使用这个工具，请记住这一点。

![The OmniGraffle interface](../Images/2ef9c5ada4046d2ad2da6ee17ca3ef34.png)

您将需要 Pro 计划从线框转移到可以导出的高保真设计。Pro 计划包括画板、共享层、矢量工具、混合模式、动作(即交互)，当然还有导出(包括 SVG)。

一个标准帐户将限制你的图表和线框，这是好的，如果这就是你所寻找的。

## [Justinmind](https://www.justinmind.com/)

*   平台:Web
*   [定价](https://www.justinmind.com/pricing):免费计划，0 美元、19 美元或 39 美元(/月)

Justinmind 是唯一真正免费的线框图工具，因为它允许无限的电路板、团队和用户——只要你还在进行线框图设计(还没有升级到高保真)。

![The Justinmind home page](../Images/a746880a8cddffafbcbe3e691a1a9f0d.png)

与其他迎合“全谱”(即低、中、*和*高保真设计)的工具相比，Justinmind 使用起来令人愉悦。它比 Axure 和 OmniGraffle 等老牌品牌更干净、更现代化。

Justinmind 包含了从线框到高保真到设计移交所需的所有功能。
