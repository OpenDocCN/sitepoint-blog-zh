# 搜索引擎优化的秘密好处:增加可用性

> 原文：<https://www.sitepoint.com/increased-usability/>

更高的搜索排名是许多网站所有者的梦想。他们没有意识到的是，如果做得正确，搜索引擎优化他们的网站也可以看到它为网站访问者优化。

最终，这意味着更多的人会找到该网站，这可以转化为增加销售和线索生成。但是，搜索引擎优化的任务和可用性兼容吗？在给搜索引擎他们想要的东西和给人们他们想要的东西之间不是有所取舍吗？继续读下去找出答案(虽然我相信你能猜到答案！).

##### 1.执行关键字研究

在你开始建立你的网站之前，你应该进行关键词研究，确定你的网站应该以哪些关键词为目标。使用公开可用的工具，如 [Wordtracker](http://www.wordtracker.com/) ，你可以识别最常被搜索的关键词，然后有针对性地搜索这些短语。

关键词研究对于你网站的可用性也是至关重要的。使用用户搜索的关键词来写你的网站内容，你就可以和你的访问者说同样的语言了。

例如，你可能会决定以短语“卖玩具”为目标，因为这就是你的网站所做的。关键词研究无疑会告诉你，网络用户实际上是在搜索“买玩具”(想想:当你想买东西的时候，你有没有搜索过“卖”这个词？).将短语“购买玩具”放在你网站的页面上，你将会和你的访问者使用相同的词语——他们将会更容易找到他们想要的东西。

##### 2.每页至少 200 字的内容

很简单，搜索引擎喜欢内容——一个页面上的内容越多，搜索引擎就越容易发现这个页面到底是关于什么的。搜索引擎可能会努力找出少于 200 个单词的网页的要点，并可能最终在搜索排名中惩罚该页面。

从可用性的角度来看，避免低内容的页面也是很好的。一个少于 200 字的页面不太可能包含大量可搜索的文本信息，因此网站访问者无疑需要点击其他地方才能找到更详细的内容。

不要害怕在页面上放置大量的信息。一般来说，[网络用户不再介意向下滚动](http://www.useit.com/alertbox/9712a.html)，而且，只要页面提供了帮助浏览的机制(比如使用副标题——参见下面的第 6 点),网站访问者定位他们想要的特定信息应该不难。

##### 3.100kb 最大 HMTL 大小

如果 200 个单词是页面内容的最小长度，那么 100kb 是最大长度——至少就 HTML 文件大小而言。包括比这更大的页面，搜索引擎可能会因为这些页面太大而放弃它们。

一个 100kb 的 HMTL 文件在一个 56k 的拨号调制解调器上下载需要 20 秒——截止到 2004 年 3 月，四分之三的英国网络用户使用这种调制解调器([来源:英国政府 PDF](http://www.statistics.gov.uk/pdfdir/intc0504.pdf) )。再加上页面其他部分(比如图像和 JavaScript 文件)的下载时间，你会发现这是一个对用户非常不友好的下载时间！

##### 4.使用 CSS 进行布局

正如我在本系列的第 2 部分提到的，网站从基于表格的布局转换到 CSS 布局后，流量会显著增加，这并不罕见。

搜索引擎可能更喜欢基于 CSS 的网站，可以在搜索排名中给它们更高的分数。干净代码的好处、重要内容放置的灵活性和更大的内容密度使搜索引擎更容易访问、评估和排列基于 CSS 的页面。

使用 CSS 进行布局对于可用性也是非常有利的。基于 CSS 的网站中固有的 HTML 文件大小的减少导致下载时间显著加快。

##### 5.使用有意义的页面标题

如果你对搜索引擎优化有所了解，你会知道很多搜索引擎更重视页面标题，而不是页面的其他属性。如果标题充分描述了该页面的内容，搜索引擎将能够更准确地确定该页面是关于什么的。

一个有意义的页面标题也有助于站点访问者了解他们在站点内和整个网络上的位置。页面标题是第一个加载的东西——通常比内容早几秒钟——所以一个描述性的、关键词丰富的页面标题对用户定位非常有帮助。

##### 6.使用标题和副标题

搜索引擎通常认为标题标签中包含的文本比文档中的其他文本更重要，因为标题(至少在理论上)概括了紧接在它们下面的内容。很多搜索引擎把最重要的分配给`<h1>`，然后是`<h2>`，以此类推。

标题对于人类站点的访问者来说也是非常有用的，因为它们对浏览有很大的帮助。一般来说，我们不在网上阅读:我们浏览，寻找我们想要的信息。如果我们，设计师和开发人员，用副标题来有效地描述页面下面的内容，我们会让用户浏览起来更容易。

务必不要滥用标题标签。页面标题标签中包含的文本越多，许多搜索引擎赋予它们的重要性就越低。

##### 7.开头段落描述页面内容

我们已经知道搜索引擎喜欢内容，但是许多搜索引擎尤其喜欢每页的前 25 个词。通过提供一个充分描述页面(或者网站，如果是主页)其余部分内容的开头段落，你应该能够在这个关键区域包含你的重要关键词。

当我们到达一个网页时，网络用户需要知道的第一件事是这个网页是否有他们想要的信息。一个伟大的(和逻辑！)找出答案的方法是浏览第一段，如果它充分描述了页面内容，应该会有所帮助。

##### 8.描述性链接文本

许多搜索引擎非常重视链接文本。他们通常假设链接文本将描述其目的地，因此检查指向任何页面的所有链接的链接文本。如果所有指向小工具页面的链接都显示“点击此处”，搜索引擎不访问该页面就无法获得任何关于该页面的信息。另一方面，如果所有的链接都是“关于小部件”，搜索引擎可以很容易地猜出这个页面是关于什么的。(我在本系列的第二部分中提供的例子就是一个很好的例子。)

描述性链接对于可用性也非常重要。如果网络用户浏览而不是阅读，一连串的“点击这里”链接对他们来说将毫无价值。描述性链接就像是浏览用户的路标:当用户浏览页面时，他们会立刻明白“关于小部件”链接指向哪里。

##### 9.避免框架

框架是一种非常老派的技术，虽然不像以前那样普遍，但仍然存在。使用框架通常被认为对你的搜索引擎排名不利，因为大多数搜索引擎不能跟踪框架之间的链接。

即使搜索引擎索引了你的页面，网络用户通过搜索引擎找到你，他们通常会被带到框架中的某个页面。这个页面可能是一个没有导航的内容页面(导航通常包含在一个单独的框架中)，因此，用户没有办法移动到网站上的任何其他页面！

框架对于可用性也是不利的，因为它们会导致后退按钮、打印、历史和书签的问题。简单来说，对框架说不！

##### 10.提供优质内容

这可能看起来像是一个搜索优化网站的奇怪特征，但它实际上是至关重要的。搜索引擎除了看页面内容，还看指向网页的链接数量。通常，一个网站的链接越多，在其他条件相同的情况下，它在搜索排名中的位置就越高。

通过在你的网站上提供创造性的，独特的和定期更新的内容，其他网站管理员会想链接到你的网站:这样做将为他们的网站访问者提供额外的价值。当然，你也会为你的站点访问者增加价值。

##### 结论

为搜索引擎和用户优化你的网站并不需要权衡。实现这两个目标所需的任务之间有很大的重叠，这种重叠可以为你所用。创建一个用户可以通过搜索引擎轻松找到并使用的网站应该不会太难。
