# 教科书“Mash 应用程序”

> 原文：<https://www.sitepoint.com/text-book-mash-app/>

目前，我正在尝试使用 Tumblr 和 T2 Jaiku，我有一个想法，通过将它们结合起来，我将为下学期在 T4 举行的用户焦点课程创建一个真正有用的“Mash 应用”。

**Tumblr**

Tumblr 是我最近一直在使用的免费博客工具，我不认为我会像使用这个应用程序一样如此迅速地爱上一个网络应用程序。事实上，Tumblr 将该工具比作废弃预订，这可能会削弱它对一些用户的吸引力，但看看吧；我看到了比这更难的多种用途！

我特别喜欢 Tumblr 的一点是评论被禁用；他们甚至不是一个选项。这是一次解放思想的经历，也更加亲密。文章以线性方式按时间顺序排列，用户可以发表五种不同类型的文章:

1.  文本
2.  照片
3.  引用
4.  链接，以及
5.  录像

**斋谷**

Jaiku 也是一个免费的网络应用程序，可以聚合各种类型的内容，由账户所有者决定。作为所有者，我可以将一系列数据拉入我的流中，Jaiku 将按时间顺序为我汇总这些数据。这包括博客、照片、书签、音乐、地点、事件、视频和 Atom 或 RSS 源。我也可以发表简短的，140 个字符的帖子。

Jaiku 中启用了注释功能。当然，任何想发表评论的人也需要有一个 Jaiku 账户，但是注册过程非常简单快捷。

那么……我有什么好主意？

我的想法是让每个学生都注册一个 Tumblr 账户，我将把这个账户作为课堂 Jaiku 账户的提要。

这种工作方式似乎有很多好处:

1.  每个学生都将对其 Tumblr 帐户中的内容拥有完全的所有权，他们可以在课程结束后很长时间内继续添加内容。
2.  学生们将从彼此的研究中受益，因为他们能够在一个地方，即课堂 Jaiku 帐户，轻松地查看这些研究。
3.  学生们将能够通过在 Jaiku 上互相评论 Tumblr 上的帖子来进行社交，并形成一个在线微社区。
4.  Jaiku 将汇总所有学生的作业，发布在一个中央存储库中，便于监控和评估。
5.  Jaiku 将作为一个中心点，从这里我可以访问每个学生的跌倒日志。
6.  作为 Jaiku 帐户的持有人，我将能够在班级帐户中添加更多信息以供讨论和评论，Jaiku 中的所有更新也将发布到我注册的电子邮件帐户中——很简单！

所以我真的很兴奋，但我喜欢的是通过结合这两个应用程序创建一个鼓舞人心的学习环境的可能性。因此，虽然我没有真正创建一个 mashup，但你可能会认为它是一个“Mash 应用程序”。

以这种方式使用应用程序将允许班级在 Jaiku 中协作创建一个奇妙的资源，同时在 Tumblr 中轻松保持信息的个性和所有权。该设置感觉非常容易、高效且易于使用；当你在教书时，这些事情会让你与众不同。

但是最好的部分是下周开始上课…我会让你知道我们怎么去！
