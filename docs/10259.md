# 2011 年:移动和本地

> 原文:[https://www.sitepoint.com/2011-mobile-and-local/](https://www.sitepoint.com/2011-mobile-and-local/)

上周，SitePoint 博客作者 Brandon Eley 预测未来将是移动的，我完全同意。另一个我认为在接下来的 12 个月里会蓬勃发展的领域是 Web 的本地化。

随着 foursquare、T2、谷歌、T4、脸书等服务的迅速普及，人们越来越重视网络地理定位。除此之外，还有大量使用内置 GPS 定位附近联系人、地标和其他地方的移动应用程序，你可以明白为什么我预测网站和网络应用程序的本地意识会增加。

主要关注本地的企业将从商业意识中获益最多。我们也应该看到越来越多的营销通过 foursquare 和类似的服务。

移动网络的出现，以及社交媒体的广泛采用，意味着人们已经习惯了在口袋里装一个完全可移动的互联网连接的想法；他们可以与路上、街上或世界各地的联系人联系。

虽然本地蓝牙和短信营销未能获得几年前所有人都预期的热潮，但我相信 2011 年消费者和企业都将采取更加本地化的方式。

我们拭目以待。与此同时，你现在应该开始审视自己的企业，看看你如何从这种转变中受益。
