# 3D JavaScript 动画–three . js

> 原文:[https://www . site point . com/3d-JavaScript-animations-three-js/](https://www.sitepoint.com/3d-javascript-animations-three-js/)

今天，我想和你分享一些由 three.js 提供的很酷的 3D JavaScript 动画。

> three.js 项目的目标是创建一个轻量级的 3D 引擎，具有非常低的复杂度；换句话说，对于假人来说。引擎可以使用< canvas >、< svg >和 WebGL 进行渲染。three.js 引擎允许开发人员使用 Canvas 和 WebGL 创建令人兴奋的 2D 和 3D 动画，从基本的立方体到高级动画，如此流畅以至于你会弄湿自己。
