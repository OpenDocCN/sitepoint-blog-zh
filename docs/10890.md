# 使用 Flash Builder 进行数据驱动的开发，第二部分

> 原文:[https://www . site point . com/data-driven-development-with-flash-builder-part-ii/](https://www.sitepoint.com/data-driven-development-with-flash-builder-part-ii/)

在今天的教程[使用 Flash Builder 进行数据驱动的开发第二部分](https://www.sitepoint.com/article/data-driven-development-with-flash-builder-part-ii)中，Flex 的 Funk 大师 Toby Tremayne 总结了他对 Flash Catalyst 和 Flash Builder 的 3 部分介绍。

在上一期的[中，Toby 向我们展示了如何在 Flash Builder 中从 Flash Catalyst 模型过渡到全功能的数据驱动开发项目。这一次，我们将了解 Flash Builder 的一些更高级的功能，并为我们一直在构建的应用程序添加一些数据编辑功能。](https://www.sitepoint.com/article/data-development-flash-builder-part-1)

![](../Images/16be4ea3ba44fbf82db8d80aa96b8eff.png "Flash Builder 4")

首先，Toby 解释说，您不需要成为 PHP 专家(因为 Flash Builder 会为您处理这些工作)，但是您需要安装 PHP 和 MySQL。然后，我们将导入我们在前两部分中处理的项目，并将一个联系人视图添加到我们已经创建的工作联系人列表中。

接下来，Toby 向我们介绍了 Flash Builder 的数据开发工具包中的一个有用的向导，它可以为您生成服务器端代码。一旦您完成了本教程，您将拥有一个通过 PHP 从数据库获取数据的工作应用程序——并且您无需编写任何 PHP 代码！
