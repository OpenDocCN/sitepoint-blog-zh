# 网络开发人员的 5 个精彩星期五链接

> 原文：<https://www.sitepoint.com/5-fabulous-links-web-developers/>

又是星期五了。过去的 5 天去了哪里？由于你们中的许多人开始了一个漫长的周末，你可能不会欣赏一个漫长而复杂的教程。因此，以下是我最近发现的一些有用的链接，它们可以在你下周回来时减轻你的开发负担…

## [![CopyPasteCharacter.com](img/13853e5208ff5d0031145c19a4de6102.png)【copypastecharacter.com】T2](http://copypastecharacter.com/)

如果你不知道什么是你的，copypastecharacter.com 是一个很好的工具，可以帮助你找到正确的 HTML 实体。单击任何符号，其字符或 HTML 代码将被复制到剪贴板。

如果您需要稍微复杂一点的东西， [XHTML 字符实体引用](http://www.digitalmediaminute.com/reference/entity/)提供了一个字符分类列表，包括它们的 HTML 实体代码、数字引用和 unicode 表示(对 JavaScript 有用)。

## [![ge.tt](img/b30054c34d379e9f0f02757f28523e88.png)
与 Ge.tt 轻松共享文件](http://ge.tt/)

你的客户是否经常试图从他们的 AOL 电子邮件帐户给你发送视频或其他大文件？救救他们的理智，建议他们用 [Ge.tt](http://ge.tt/) 。

虽然有很多文件共享系统，但 Ge.tt 是我见过的最简单的一个。它是免费的，不需要注册，也没有文件大小的限制。文件链接可以通过电子邮件、推特或脸书发送给他人。您的数据保留 90 天，您可以访问下载统计数据，开发者正在创建一个 API。

## [![supportdetails.com](img/0cffe282083d930db136efff60831f43.png)
支持详情](http://www.supportdetails.com/)

今天是星期五，下午 5:30，你的电话响了。是 ClientX 他们不高兴。“我的网站不行。”他们声称，“它坏了，我现在就想修好它”。你加载网址，一切似乎都很好。

当你试图找出问题时，接下来是一次长时间的谈话。不幸的是，客户不知道他们使用的是什么浏览器——他们不知道什么是浏览器。询问他们的操作系统、屏幕分辨率、颜色深度、Flash 版本或 JavaScript 状态是徒劳的。

幸运的是，supportdetails.com 可以为你解答许多问题。请他们访问该网站并输入您的电子邮件地址，以接收有关其系统的报告。他们可能正在他们姑母的 Windows 3.1 PC 上运行 IE3.0。

## [![8bitalpha.com](img/3e1ee5986e6735aa020d6d7de416f5b3.png)
用 8bitalpha.com 转换 PNGs】](http://www.8bitalpha.com/)

24 位透明的 png 很棒。然而，Fireworks 用户知道 [8 位 png 更小，可以在更多的浏览器中工作](https://www.sitepoint.com/png8-the-clear-winner/)。如果你不使用烟花，【8bitalpha.com】的可能会有帮助。只需将您的 24 位杰作拖到页面中，就会返回一个优化的 8 位版本。

## [![ninite.com](img/e1682fc68d101c83abd3554981e5ca7e.png)
用 ninite.com 安装和更新软件](http://ninite.com/)

你最近收到一台新电脑了吗？下载和安装您的重要软件、开发工具、浏览器、媒体播放器、运行时、病毒检查工具和实用程序需要多长时间？查找、下载和安装更新需要多长时间？

和 ninite.com 一起节省那些失去的时间。这很简单:选择你的应用程序，下载安装程序，让它为你做艰苦的工作。当您想要检查更新时，可以再次运行安装程序。

ninite.com 主要面向那些没有内置软件包管理器的 Windows 用户。也就是说，ninite.com 的 Linux 版本[也是可用的。](http://ninite.com/linux/)

## 奖励链接:[网址猎人](http://probablyinteractive.com/url-hunter)

你厌倦了花哨的 3D 游戏吗？URL Hunter 可能是有史以来最简单的基于浏览器的游戏。它在地址栏中播放，你的任务是拍摄' a '字符。这算不上使命召唤，但它会给你片刻的快乐，没有人会意识到你在享受。
