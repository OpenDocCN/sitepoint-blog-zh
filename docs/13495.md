#### 1.规范

在给出的四个规范(WSDL、APP、RDF 和 WS-Policy)中，只有 RDF 指定了非基于 XML 的数据表示。是的，RDF *可以用 XML 表示*，但它也有替代格式；其他三个没有。

#### 2.HTTP 方法

<cite>卡西米尔</cite> [明白了](https://www.sitepoint.com/episode-3-one-of-these-things/#comment-110705):“帖子不是幂等的”。

幂等性在 web 开发中实际上是一个非常重要的概念，但与其对此喋喋不休，不如让我给你指出维基百科关于主题的[单词，它们相当不错。](http://en.wikipedia.org/wiki/Idempotence_%28computer_science%29#WWW_behavior)

#### 3.MD5 哈希

<cite>明道加斯</cite> [明白了](https://www.sitepoint.com/episode-3-one-of-these-things/#comment-110777) : B 是空字符串的 MD5 哈希。

这是一个有点“抓到你了”的问题；我通常会尽量避免回答棘手的问题，但在这里我就是忍不住。
