# 知识共享和网页设计

> 原文:[https://www.sitepoint.com/creative-commons-and-web-design/](https://www.sitepoint.com/creative-commons-and-web-design/)

部分受到自由软件基金会工作的启发，[劳伦斯·莱斯格](http://www.lessig.org/blog/)共同创立了[知识共享](http://www.creativecommons.org)，并创造了一个前所未有的全新的合法内容分发和共享世界。

对各种艺术家(音乐家、摄影师、作家等)的直接呼吁是提供一种在国际上分发他们作品的方法，同时仍然保持对使用的某种控制。

许多 Creative Commons (CC)爱好者通过使用像 Movable Type 这样的博客软件发现了许可工具，这些博客软件在他们的博客配置工具中包含了 CC 框架。

许可过程是灵活的，并涵盖了广泛的使用范围，为更多的观众。在 CC 网站上[解释了许可证。](http://creativecommons.org/learn/licenses/)

对网页设计者和开发者的吸引力是双重的:1)一种工具，用于分发他们寻求利用互联网的病毒性质的设计/内容(可能是为了促销，或者可能是为了吸引对商业许可或作品销售的注意)，以及 2)一种内容搜索引擎，用于在 CC 许可下在网站上寻找可用的内容。

最后一种是最独特的，而且随着越来越多的 web 客户端将博客作为主网站或除主网站之外的网站，这种趋势肯定会增长。这为设计者和开发者为客户解决方案寻找图像、音乐和书面内容开辟了一条新的途径。

CC 搜索引擎[可以在这里](http://creativecommons.org/getcontent/)找到。另外，请务必查看商务 2.0 中的[专题文章。](http://www.business2.com/b2/web/articles/0,17863,608619,00.html)
