# 与物联网互动

> 原文:[https://www.sitepoint.com/interact-with-web-of-things/](https://www.sitepoint.com/interact-with-web-of-things/)

电脑界面疲劳症(Computer interface fatigue)——长时间使用电脑，你开始失去目标。你只是在运作，点击按钮，移动/删除内容。但是什么“真正”的事也没做。前几天晚上我经历了一次大事件。我无法正常思考，完全没有创造力(当然，这可能是凌晨 2 点的事实…不)。

然后我合上屏幕，放下笔记本电脑，去刷牙了。砰，纸和笔在哪里？突然间，我有了在过去 5 个小时里一直想不起来的话。从笔到纸，5 分钟，完成。

我停止使用电脑，至少有一段时间。拿起一支带血的笔。找到一些“真正的”纸，写道。哇，突然间我又能思考了，我又能写作了，我的思维障碍消失了。我实际上在做一些事情，不仅仅是运作，不仅仅是点击——做。

一笔勾销

想象一下，如果所有的界面垃圾都消失了，想象一下，你在生活中与事物互动，与对你有意义的事物互动。创造性的汁液将真正开始流动。

**简化**

好了，把电脑从电脑上拆下来，我们开始有所进展了。键盘很糟糕，鼠标很糟糕。真正的触摸是美好的，触摸是个人的，触摸是互动的，物体是触觉的，我们可以使用它们，并对它们进行创新。

我们如何超越这些限制？我们如何才能真正被允许从手腕酸痛和无休止的按键和点击的负担中解脱出来。一种新的互动方式是必不可少的。苹果用 iPhone 打开了闸门——谢谢你，史蒂夫。我们现在进入下一个阶段。

**下面是什么**

有[一些](http://www.vectorform.com/surface/?p=183)惊人的[事情](http://www.vectorform.com/surface/?p=3) [正在](http://www.youtube.com/watch?v=8pv_LtzRcbI)做表面文章。经验丰富的建筑师奥古斯特·德·洛斯·雷耶斯是这个群体中的核心思想家。几周前，他在 Web Directions South 做了一个精彩的主题演讲[主题是“预测过去:情感设计和微软 Surface 的愿景”。](http://south08.webdirections.org/?page_id=7#post-91)

[![August Keynote to Web Directions South](../Images/b9ec3222fa74305b97d3ccc2bf813c28.png)T2】](http://img.skitch.com/20081022-km41hpf4nctbewi8tgsxa6qjs5.jpg)

以下是他关于 NUI 八大原则的演讲的简要记录——*自然用户界面*设计:

1.  **做事情的乐趣**
    *   玩耍不一定等同于工作
    *   快乐来自于互动而不是实际的成就
2.  **WYDIWYG** –
    *   你做的就是你得到的
3.  **显示足够多的信息**以理解下一级的复杂性
    *   就像搭建脚手架一样——构建更高水平的平台
    *   电子游戏，第一关真的很容易
4.  **语境环境**
    *   在环境中寻找线索，这样我们就能找到路，即使我们以前从未见过它
5.  **超真实原理**
    *   利用现有的环境和物理世界；但是仍然有艺术表达的许可

**超级原则或支柱，是什么让表面与众不同**

7.  **社交的**
    *   体验是让人们彼此接触，因为当前的技术将我们分开
8.  **无缝**
    *   物质层在哪里开始和结束；数码什么时候接管？
9.  **空间**
    *   使用深度和感知线索来帮助导航

**过渡到物联网**

这是我看到奇迹发生的地方。目前，我们被困在一个文档网，内容汤。向物联网的转变正在发生。结构化数据是自我描述的，它“知道”它在谈论一辆汽车、一份工作清单或我儿子的照片。

自然的用户界面都是关于真实物体的。我们可以忘记当前浏览器的局限性和界面问题。我可以抓住我儿子的照片，把它放在相册里，用手指在汽车周围画一个圈，然后保存到我的笔记里。

突然间，我可以用更丰富的词汇和更有意义的联系来讲述我儿子生日聚会的故事了。

**不久的将来**

iPhone 彻底改变了我们与电脑的互动方式。它对更大范围的人来说是可行的，有意义的——尤其是那些不是电脑极客的人。我有一部诺基亚 N95，它很棒，但每当我看到有人在上面旋转、点击和滑动时，我都会嫉妒 iPhone……我还听说苹果已经为其电脑系列申请了一些多点触控专利。如果这个成功的话，那将是非常棒的，OS X 和多点触控的易用性结合在一起。

另一个有趣的发展是 Windows 7 将支持多点触控。我知道这可能是旧闻，但我只是在 [Twine](http://www.twine.com/) 上搜索“surface”时得知的。
