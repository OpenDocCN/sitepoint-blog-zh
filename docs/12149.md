# 你做特殊工作吗？

> 原文:[https://www.sitepoint.com/do-you-do-spec-work/](https://www.sitepoint.com/do-you-do-spec-work/)

设计行业的热点问题之一是规格工作。群众是分裂的，最坚决的似乎呼声最高。这是对规范工作以及使其成为一个问题的利弊的审视。

什么是规格工作？

投机性工作是在达成任何条款和付款协议之前为客户进行的工作。这是一种试驾，让客户有机会在付款或承诺付款之前一瞥最终产品。这在设计行业最常见，因为项目是可视化的，并且可能非常主观。

规范工作的一种流行形式是设计竞赛，公司为一个设计项目举办竞赛。设计师创作一个设计参加竞赛，只有最好的设计(或公司选择的设计)才能获得报酬。这种类型的规范工作也被称为众包。

**特殊工作的缺点**

根据[没有！spec](http://www.no-spec.com) ，一群坚决反对 SPEC 工作的设计师，“SPEC 工作贬低了设计的潜力，最终对客户不利。”该组织声称规范工作不利于设计行业的其他一些原因包括:

*   它创造了一种基于不信任的关系。
*   它降低了设计师经验的价值。
*   设计师没有足够的时间研究业务、市场和行业。
*   规格工作更倾向于风格(设计师的风格感)，而不是与公司的相关性。
*   这是不道德的商业行为。

AIGA 对 spec work 的长期[立场一直是反对的，但是在 2008 年，他们声明他们的立场可能需要考虑开源工作的频率以及它对行业的影响。他们有一个特别工作小组正在研究这个问题，他们将在本月向 AIGA 董事会汇报。](http://www.aiga.org/content.cfm/position-spec-work)

**Spec 工作的利弊**

那些支持 spec 工作阵营的人有一组同样令人信服的论点。规范工作的一些好处包括:

*   这给了新手和经验不足的设计师一个机会。
*   它能激发竞争，从而提高质量。
*   设计师有机会让他们的名字在那里，并找到更多的工作。
*   这是很好的实践，也是投资组合的建设者。
*   它给了设计师一个赚快钱的方法。
*   如果你的设计被选中，它代表了正在进行的工作的可能性。

《福布斯》最近的一篇文章《群体的创造力》称，众包是一个令人信服的概念，可以让竞争环境变得公平。这篇文章声称，这种类型的竞争模式本质上帮助自由职业者感受到经济下滑带来的痛苦。

Josh Catone 上个月在 SitePoint 上发表的一篇文章谈到了关于规格工作的辩论，并指出了一些额外的好处，包括设计竞赛是一些面临预算限制的公司唯一可行的选择，它们是设计未来的一部分。这篇文章的一些评论反对这些因素，强调这肯定是一个有争议的问题。

就我个人而言，我没有做过，也可能不会做具体的工作，但我并不反对这个想法。虽然它不支持我的商业模式，但我认为这对从中受益的设计师和公司来说有巨大的价值。它鼓励竞争，这使我们都在我们所做的事情上做得更好。

那么，你做特殊工作吗？如果是，为什么或为什么不做？
