# 获得前 10 个客户

> 原文:[https://www.sitepoint.com/getting-those-first-10-clients/](https://www.sitepoint.com/getting-those-first-10-clients/)

在我的第一篇文章中，我向读者征求他们对我报道的话题的意见。许多人回答了一些很棒的问题，其中一些人至少问了一些营销方面的问题，以及我是如何从坐在一张空桌子前，到开始几个月就找到工作的。

首先，我作弊了。我已经在 web 行业工作了很多年，所以我已经有了很多联系人，我做的第一件事就是联系其他自由职业者，特别是开发人员(记住，我是从设计师开始的)，并主动帮助他们解决他们无法处理的任何溢出工作。

我以这种方式结束了一些客户和一些直接的工作，然而我发现也很有趣。我期望在他们公司名称的面纱后面工作，然而我实际上发现这些同事足够信任我，可以直接与客户打交道，并且只给自由职业者一点回扣。太棒了。他们不仅给我工作，他们相信我会做得很好，因此我确保我做到了。

这里的关键点是建立联系，但要小心不要冒犯任何人——没有他们的同意，我肯定不会接近前雇主的客户，而不是朋友客户。

其次，我还接触了一些慈善团体，我知道他们需要一个网站，并提出以非常接近免费的价格(嗯，半费，当时感觉是免费的)为他们做网站，作为交换，他们会把我介绍给他们的同事，但前提是他们喜欢最终的结果。这里的重点不是削弱竞争对手，而是表明我支持慈善事业，但我需要一些报酬，包括现金和与其他组织和企业的联系。

这里的诀窍是确保接受者理解价值——我写了一份包含我通常期望的全部金额的建议书，然后用白纸黑字显示折扣，这样他们就可以清楚地看到我赠送了什么。我确保他们明白这是一个非常有限的提议，之后，我的正常利率将适用。在我从他们自己的组织以及他们网络中的其他企业那里获得足够多的工作之前，我只需要接触两三个组织。

说实话，我讨厌电话营销，冷不防打电话让我出冷汗。我避免了任何沉重的销售技巧，确保我要求介绍上述两种情况。我只想和已经通过资格预审的人交谈，要么是我的自由职业者联系人，要么是对我们为他们做的项目感到满意的组织。

整个过程中最重要的三点是:

1.  我做得很好，超出了客户的期望。
2.  我确保客户欣赏我的作品是伟大的。
3.  我直截了当地问他们，是否知道谁可以使用我的服务。

这是真的——如果你有出色的项目成果，你可以要求客户推荐。他们介绍给你的人可能已经有了你的第一个客户，他们对你的评价很高，他们也表示有兴趣使用你的服务。在这种情况下，不需要强硬的销售策略，你只需要向潜在客户介绍各种选择，最重要的是提供你愿意谈论的服务。

看起来挺简单，但是真的很管用。第一年，我花了大约 50 美元给我最喜欢的客户买了几瓶酒，这是我全部的营销预算。把大规模的广告活动留给你的竞争对手没有坏处。
