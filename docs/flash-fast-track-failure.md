# 全闪存是通向失败的快速通道

> 原文：<https://www.sitepoint.com/flash-fast-track-failure/>

这篇文章解释了 Flash 是如何扼杀一个商业网站的，但请放心，这不是一篇反 Flash 的咆哮！

Flash 在网页设计者的工具箱中有一个明确的、很可能是永久的位置。但是，如果使用时没有经过仔细的思考和考虑，Flash 可能意味着你的在线创业的死亡！我应该先提一下，我说的是全 Flash 网站——虽然完全用 Flash 构建的网站对于某些目的来说可能是可以的(例如在线作品集),但对于商业网站来说，这是一个真正的杀手！

##### 有什么问题？

那么，全闪存业务网站到底有什么问题呢？

*   可用性陷入困境
*   可达性消失
*   现代用户受苦了，而且
*   搜索引擎不索引 Flash 文件！

这些问题主要出现在没有经验或粗心的开发者试图创建令人讨厌的全 Flash 网站时。一个 10-20 页的大 Flash 文件？耶！那么就让我们来逐一看看这几点吧。

***可用性问题***

如果你以设计网站为生，那么你会知道可用性的概念经常在客户端消失。但它永远不会错过客户的访客！

一般的冲浪者都是简单的野兽:

*   他想要信息
*   他现在想加入，而且
*   他不想为了得到它而费尽周折

如果一个网站由于使用了 Flash 导航系统、冗长的介绍动画或者无用的效果而变得过于复杂，你将会失去大量的潜在买家。因此，如果你的网站是一个巨大的 Flash 文件，在你发布它之前，它就已经岌岌可危了。

通常，唯一对过多的 Flash 内容印象深刻的人是设计师和他为之工作的可怜的傻瓜。换个方式想:为什么你在亚马逊上看不到 Flash 菜单和效果？因为这种内容只会让用户远离网站的目的——说服人们购买。

***无障碍问题***

请记住，我们在这里讨论的是全闪存站点。任何需要调整文本大小、屏幕阅读等的人。，你的 Flash 内容就没用了。

好吧，如果你正在向精通网络、90% DSL 互联网接入的年轻观众销售一个高度可视化的产品，那就继续吧。但是如果你卖的是其他东西，谁能说你的产品不会引起其他技术不太先进的群体的兴趣呢？您真的想要拒绝这些用户(当然，他们是潜在客户)的访问吗？

全闪存网站没有任何可访问性！

***调制解调器问题***

没错，伙计:不是每个人都有超高速 DSL 连接。其实大部分人还是用 56k 的调制解调器。

当你在寻找信息的时候，你是怎么上网的？我甚至不给 Flash 网站一个机会。我会点击搜索结果列表，直到找到一个能立刻抓住我的页面，它会立刻呈现我想要的确切内容！

网络是关于信息的。用户根本不在乎一般商业网站的图形方面:他们想要的是你的内容！

如果你不马上给他们，别人会给的。

***搜索引擎索引问题***

你的全闪存网站需要流量吗？你做梦去吧。除非你想让你的网站和搜索引擎收到垃圾邮件，并冒着被竞争对手举报或被垃圾邮件过滤器过滤的风险，否则你注定要失败。

HTML 是一种内容媒介——搜索引擎很容易找到 HTML。Flash 是一种图形媒体，应该用于…呃…图形内容！

Flash 文件的索引充其量是不完整的，仍然是一个阴暗的、未开发的领域。这意味着所有的努力工作，或者如果你是客户，所有的现金，可能都是徒劳的。

如果你面临着优化一个全 Flash 网站的任务，那么目前最好的方法是开发一个替代的 HTML 网站。让主页作为 Flash 站点的加载页面运行，并带有指向备选 HTML 版本的突出链接。

仅供参考，最近与我交谈过的实施该解决方案的人报告说，平均有 70%到 80%的用户点击 HTML 链接。这说明了一切，真的！

##### 该怪谁呢？

那么，商业网站上围绕 Flash 使用的问题是谁的错呢？出现这些问题是因为闪存作为一种媒介不合适，还是因为设计者出于错误的目的盗用了这项技术？

今天，大多数 Web 专业人员的总体感觉是，总体上过度使用 Flash，更具体地说，在商业网站上使用该技术，绝对不是 Flash 本身的错。

问题在于，与许多其他 Web 技术一样，Flash 对于“尝试一下”的业余爱好者来说是一个相当简单的“拼凑”工具。

然而，专门的 Flash 设计师告诉我，和许多 Web 技术(如 PHP 和 ASP 之类的脚本语言)一样，Flash 确实需要多年才能掌握。

专注、经验和对 Flash 媒体的理解使设计师能够非常有效和高效地使用它来增强网站的整体外观和感觉，并传达特定的信息。

因此，对于一个主要目的是提供信息和吸引客户的商业网站来说，“我的朋友知道 Flash”的方法每次都是一个杀手。

##### 当闪光灯正常时…

正如我前面所说，Flash 不是一个坏工具，它只是需要以正确的方式使用，最好是由一个有才华和经验的 Flash 设计师来使用。

商业网站对 Flash 的良好使用包括:

***视觉产品说明***

用于描述产品的工作原理，或者在视觉帮助解释中，Flash 真的很有用。

***互动地图和示意图***

在需要高度视觉化内容的网站中，例如房地产网站，Flash 可能很有优势。在这里，走一遍蓝图或网站图可能对潜在客户最有帮助。

***非必要增强***

Flash 可以用于创建额外的东西，如徽标、广告和纯粹用于视觉增强的小效果，前提是要有所克制。

##### 不要让 Flash 扼杀了你的商业网站

嗯，作为网站所有者、开发者和设计者，我们应该注意的最重要的事情可能是，Flash 可能很容易学习，但它需要专门的专业人员来有效地实现它。

此外，请记住，全 Flash 商业网站通常是用户和搜索引擎的噩梦，如果你真的必须走这条路，你应该始终提供 HTML。

Flash 不是文本内容的好媒介:你不会把所有的文本都放在 gif 中，所以不要把所有的文字内容都放在 Flash 中！

最后……如果谨慎而周到地使用，Flash 真的可以给网站一个精致、专业的外观。但是，如果被业余爱好者或粗心的设计师使用，它真的会毁掉你的商业网站。

记住，你的商业网站不是为你，而是为你的用户！
