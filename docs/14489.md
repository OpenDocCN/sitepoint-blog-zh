# 面向 Windows 的开源应用

> 原文:[https://www.sitepoint.com/open-source-apps-for-windows/](https://www.sitepoint.com/open-source-apps-for-windows/)

许多经常阅读 Open Sourcery 的读者都知道，我的主工作站是一台运行 Mac OS X Panther 的 G4 1.25 GHz Powermac。然而，我这里的网络包括 Windows XP Pro、Red Hat 9、SUSE Linux 9 和一些旧的机器和操作系统(是的——Windows 98 SE 为了客户端测试而存在)。

虽然我已经在 Windows、OS X 和微软 Office 等两个平台的应用程序上投入了资金，但我总是在所有系统上研究和测试开源应用程序。

今天早上，我想记下一些在我的 Windows XP 系统上使用的我最喜欢的开源工具。我发现致力于 Windows 的开源项目的开发异常丰富，而且它们经常提供一种解决方案，在某些情况下替代昂贵的专有软件。

我郑重声明——我正在测试在 Windows 上用 Open Office 代替 MS Office，我只是还没有做好转换的准备。在我的 Mac 电脑上，我用 Keynote 制作演示文稿，用 Filemaker 制作数据库，用 AppleWorks 制作文字处理文档/电子表格——所以 Open Office 可能不会为我在 OS X 找到出路。

pdf creator-[https://sourceforge.net/projects/pdfcreator/](https://sourceforge.net/projects/pdfcreator/)

从众多 Windows 应用程序中创建 PDF 文件，最流行的是 MS Word。

feed reader-[http://www.feedreader.com/](http://www.feedreader.com/)

一个很棒的 RSS 新闻阅读器，可以在 Windows 95 及更高版本上工作，并且支持所有主要的 RSS 源类型。

缥缈——[https://sourceforge.net/projects/ethereal/](https://sourceforge.net/projects/ethereal/)

用于网络分析，下至数据包级别，支持实时网络检查或从磁盘上的文件中捕获数据。

getif-[http://www.wtcs.org/snmp4tpc/getif.htm](http://www.wtcs.org/snmp4tpc/getif.htm)(不是开源的，但是免费的！)

如果您正在管理网络或在系统中构建 SNMP 监控，这是一个很好的工具。它是一个 SNMP 浏览器，但也可以显示设备/服务器接口信息、路由表、端口扫描、IP 扫描等等。

密码安全-[https://sourceforge.net/projects/passwordsafe/](https://sourceforge.net/projects/passwordsafe/)

当您需要记住多个服务器和设备的管理员密码时，这是一个非常有用的工具。

ClamWin 免费杀毒软件-[https://sourceforge.net/projects/clamwin/](https://sourceforge.net/projects/clamwin/)

这是一个自由蛤蜊反病毒([http://www.clamav.com](http://www.clamav.com))的 Windows 前端，提供强大的病毒保护，集成到 MS Outlook 和一个调度程序。病毒定义文件可以在登录时或按时间表自动更新。

还有许多更多的 Windows 开源项目，然而，这些已经在我的日常使用中被证明是有价值的。让我知道你用过的一些应用程序，以及你对这些工具的看法。
