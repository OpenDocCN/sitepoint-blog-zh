# 巴西 2014 年世界杯标志

> 原文:[https://www.sitepoint.com/brazil-2014-world-cup-logo/](https://www.sitepoint.com/brazil-2014-world-cup-logo/)

就在你认为你已经受够了足球(或者在某些情况下可能还不够)的时候，国际足球组织公布了将于 2014 年在巴西举行的下一届世界杯的标志设计。

名为“灵感”的新设计是从 25 件参赛作品中选出的获奖作品。评委团包括著名的巴西人、模特吉赛尔·邦辰和作家保罗·柯艾略。这个标志是由圣保罗的机构非洲设计的。巴西总统路易斯·伊纳西奥·卢拉·达席尔瓦在揭幕仪式上发表讲话，他说“巴西人充满自信，热爱呈现一届美丽世界杯的挑战”。

[![logo-world-cup-brazil](../Images/299abf078b7fe53f71c1888d9bb76142.png "logo-world-cup-brazil")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/logoworldcupbrazil.jpg)

该标志是一个世界杯奖杯的代表，由三只相连的手组成，分别采用巴西的国色，黄色和绿色，并以鲜红色手绘字体显示比赛年份。虽然红色很显眼，但巴西国旗是黄色、绿色和蓝色的。手的样式画的很松散。它们看起来像是用 Photoshop 或 Illustrator 中的钢笔工具快速绘制的。

就像几乎每一次重大的 logo 发布一样，关于它的执行，网上有很多争论和一定数量的批评。一些评论员提到，足球比赛的标志不应该是双手抱住球。其他人把它称为廉价的剪贴画，手上的手指看起来像外星人的手指。
