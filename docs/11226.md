# 谷歌面临反垄断调查

> 原文：<https://www.sitepoint.com/google-faces-antitrust-inquiry/>

欧盟委员会喜欢好的调查。[甲骨文](https://www.sitepoint.com/eu-object-oracle-sun-mysql-takeover/)、[苹果](https://www.sitepoint.com/eu-investigate-apple-iphones/)和[微软](https://www.sitepoint.com/microsoft-european-browser-choice/)都受到了冲击，现在他们已经将法律的炮塔转向了谷歌。尽管谷歌此前曾在美国和欧洲面临反垄断指控，但这是针对该公司搜索和广告服务的首次官方调查。

## 搜索结果公平吗？

调查的第一个方面是由英国价格比较服务公司 [Foundem](http://www.foundem.co.uk/) 和法国法律搜索引擎 [ejustice.fr](http://www.ejustice.fr/) 的投诉引发的。双方都指责谷歌将他们的网站在搜索引擎结果中降到了不公平的低位。

谷歌否认故意惩罚对其服务构成竞争威胁的网站。独立的搜索引擎优化分析师也指出，Foundem 几乎没有原创内容，因为它从其他网站抓取产品信息。

谷歌当然可以修改单个网站的索引——例如，如果发现域名有欺骗内容或从事黑帽 SEO 行为，就可以将其屏蔽。然而，谷歌会认真考虑这两个网站是一个威胁吗？如果谷歌从事大规模排名操纵，这将涉及大量的人力和搜索结果可能会变得毫无意义。

这方面的调查应该被放弃，除非可以明确证明谷歌滥用其搜索优势，伤害竞争对手。如果 Foundem 和 ejustice.fr 由于一些法律技术问题而获胜，闸门将会打开——每个企业都将能够声称自己在搜索结果中的“合法”地位。

## 广告费公平吗？

调查的第二部分集中在 Google AdWords 上，由 Ciao 提出。

AdWords 可能是网络上最大的广告网络。费用由拍卖决定；如果你同意为每次广告点击支付 50 美分，你通常会出现在为同样的搜索条件支付 30 美分的人的上方。然而，谷歌设定了最低出价水平，热门关键词可能非常昂贵——10 美元或更多。

调查的这一方面可能会打击谷歌:他们拥有市场支配地位，可以人为提高最低出价。毕竟，一个公平的拍卖过程应该让你出价任意多——即使这意味着你的广告会在周日凌晨 3 点出现在搜索结果的第 57 页。

谷歌说 Ciao 被微软收购，是 Bing 的一部分，很难做到公正。我不是律师，但我认为谷歌需要一个更好的辩护理由。

你怎么想呢?谷歌是不是被不公平地盯上了？还是他们变得过于铁板一块，背弃了不作恶的承诺？
