# 网站破坏者——互联网的涂鸦艺术家？

> 原文:[https://www.sitepoint.com/graffiti-artists-internet/](https://www.sitepoint.com/graffiti-artists-internet/)

**网站什么时候变成艺术？我们中的许多人在某个时候问过自己这个问题，但无济于事:没有确定的答案。处理这个问题更有效的方法可能是从另一个角度来看。我们可以问“什么构成了艺术？”然后将我们的描述应用到互联网的特定方面。**

对我来说，“艺术”是任何形式的媒介，它引发并实现了一种情感反应或回应。《大英百科全书》中有一个很好的定义:

<q>*“在创造可与他人分享的审美对象、环境或体验时，技巧和想象力的运用。”*T3】</q>

现在，让我们在离线环境中思考这个问题。我们有充满“艺术”的画廊和博物馆。但在我看来，街头涂鸦往往比我能想到的大多数其他“艺术”形式更完全符合这个定义的要求。

涂鸦本身就是一个有争议的选择。但是请想一想:这不仅是一种极具煽动性的媒介——许多人认为涂鸦艺术家是罪犯，根本不是“艺术家”——而且它与在线艺术的一个领域非常相似，你们许多人可能以前没有考虑过艺术——网站污损。

事实上，涂鸦和网站丑化之间的联系代表了“现实世界”艺术和它的在线对应物之间最强的相似性之一。

##### 那么黑客是如何成为“艺术家”的呢？

想一想涂鸦艺术家创造了什么:他们使用他们选择的媒介来传达一个观点，这个观点通常是政治性的，通常是突出的，并且可以体现不同程度的复杂性。这位涂鸦艺术家劫持了铁路线旁的墙壁，油漆了交通桥梁，标记了政府建筑——这一切都是为了一个目的:让他们的观点被接受。

我想说的是，黑客的动机，甚至结果都是完全一样的。让我们再来看看艺术的定义，但不要去想当地的美术馆，让我们来想想网站破坏者:

*技能的使用*——相关人员必须入侵一台服务器，这显然需要技能

*和想象*——创造他们想要传达的信息

*创造审美对象、环境或体验*-检查

*可以与世界各地的其他人分享*-分享！

明白我的意思吗？要记住的重要一点是，艺术不一定要视觉上惊艳。它只会让你思考。

##### 检查相似之处

通过涂鸦艺术家的作品传达的信息通常具有高度的政治性和故意的攻击性。那么这和丑化网站的黑客有什么关系呢？让我提醒你注意一个名为 Break Ice(警告:强烈的语言)的黑客的例子。

如果对此负责的黑客只是建立了一个网站来显示这些文本，没有人会注意到。通过入侵数百个网络服务器，并在每个服务器上留下相同的信息(其中一些相当引人注目)，他或她以一种不容忽视的方式向不情愿的大众发布了他们的信息。这个黑客可能不认为他们做的是艺术，但如果你仔细想想，也许是。这当然是虚拟涂鸦——所以如果你认为一个是艺术，那么你肯定也会以同样的眼光看待另一个。

像黑客一样，涂鸦艺术家发现他们自己在持续的斗争中，让他们的作品被人看到，并逃避法律。当我住在赫尔(英国东约克郡)时，我的一个建筑师朋友开始了一个为合法涂鸦提供设施的项目。他想创造一个地方，让真正的涂鸦艺术家可以在不打扰任何人或惹上麻烦的情况下进行交易。它也将成为这些人的画廊，让更多的观众看到他们的作品。

然而，当我们开始与赫尔的艺术家交谈时，我们开始意识到这真的不是他们所做的重点。涂鸦是一种非常颠覆性的艺术。这些人不是为了喷漆而喷漆——他们想向观众传达一种信息，就像许多现代艺术作品一样。他们的部分特殊技能在于将政治和讽刺抛向人们的脸上，这确实对他们影响最大，而且他们不能忽视这一点。这样，艺术家的作品更有影响力。

这种方法不仅描述了街头涂鸦艺术家的特征，也描述了网站涂鸦者的目的。

##### 还有更多…

现在，在涂鸦的世界里，有涂鸦者、投弹者和拼凑者。区别？涂鸦者只是简单地把一个单词或一些首字母扔在墙上。投弹手是一个“精心制作的标记者”。炸弹客的标签更加复杂、多彩和详细，但它们很少携带信息。最后还有计件。实际上，拼凑者彻夜工作，创作出精心设计的壁画。

有趣的是，网站破坏者也可以分为三类:贴标签者、投弹者和拼凑者。这里有一个我称之为标记者工作的例子。下面是[一名投弹手](http://www.zone-h.com/defaced/2002/10/16/teamforce.com.tw/)的作品。我要说的是，我们看到的第一个例子可能代表了一个片段。

然而，我不得不承认，当我们考虑到互联网的时候，我对计件工资的定义有所改变。当谈到网站污损时，更多的是关于信息而不是视觉效果，但涂鸦者通常在他们的艺术中也有强烈的信息。

可悲的是，在网上和网下环境中，大多数留名者和投弹者倾向于给拼凑者一个坏名声。这并不是说留名者不一定是艺术家。例如，在 www.graffiti.org 上有一个由马修·史密斯拍摄的标签，上面简单地写着“布什万岁”。它只不过是一个标签，但它让你思考。

也就是说，贴标签被视为一件讨厌的事情。例如，如果我早上醒来走到外面，发现我租的房子的一面墙变成了一幅描绘城市斗争的壁画，我会暗自高兴。当然，我会打电话给我的房东，他可能会在这个问题上说几句话。然而，如果我走出前门，看到有人在花园门上喷了他或她的名字首字母，我会非常恼火。

同样，如果有人黑了我的网络服务器，只是在上面留下一些随机的字母和字符，我会认为“愚蠢的孩子！”但如果有人黑了我的网站，并在首页留下了一份政治声明，不管我是否同意，我都会暗暗好奇。我当然会把它拿掉，但我会想“干得好！”

##### 区别在哪里？

第一个区别直接来自最后一个平行。网络上的标签也许比现实世界中的标签更有价值。

你看，网络标签实际上是以一种有趣的方式提供服务。他们不断强调服务器安全性不足的情况，并在操作系统中发现新的漏洞。“现实生活”涂鸦中的涂鸦者没有这种说法。他们所证明的就是他们知道如何使用喷雾罐！我不会不同意网络标签是一种讨厌的东西，也不会不同意真正的黑客组织(如果这不是太矛盾的话)在做同样的工作时更有效。话虽如此，但对于懒惰的网站管理员或 IT 经理来说，它们无疑是一记警钟！

为了建立涂鸦艺术家和网站破坏者之间的第二个主要区别，让我向你介绍布里斯托尔涂鸦摄影师马修·史密斯的评论。他认为:

<q>*“我被它街头涂鸦的智慧和艺术性打动了。我也被它的内容打动了。起初，我试图拍摄我看到的东西，包括它的背景，但随着时间的推移，很明显这不是前进的方向。无论是从字面上还是象征意义上，涂鸦的内容都非常引人注目，而不是它的位置。它所反映的情感真相是如此的吸引人；就像在生活中一样，有必要忽略一些杂乱无章的东西，以便集中精力，突出和界定其本质。”从【http://www.graffiti.org/bristol/index.html/】的到*的</q>

我认为他说的有些是正确的，但我觉得如果你把涂鸦从它的背景中抽离出来，信息的力量就丧失了。例如，谁会听当地市场广场上的人通过扩音器大声疾呼民权？我不——我想“疯子”,然后继续走！然而，我清楚地记得看到一幅涂鸦，画着一架警用直升机在人们的房子上空盘旋，有人用火箭发射器将它击落。基本上和你那个拿着扩音器的男人是一个情绪。但是，把这幅涂鸦从一条沉闷的、刮着大风的街道上拿下来，那里的居民每晚都能听到直升机在他们的房子上空盘旋，把它放在利兹一家画廊的墙上，它没有任何意义。这只不过是漫画中的一帧画面。

另一个或许更有力的例子是一位名叫班克斯的著名英国艺术家。他做了一件作品，把伦敦东南部一座桥下的“河岸”改成了班克斯。再说一次，断章取义，他以当局为代价的煽动性笑话毫无意义。

相比之下，黑客的工作是没有上下文的。唯一的背景是世界:它是一幅全球性的画布。作为一种积极主动的媒体，它的目标是正确的人，并以这样一种方式执行，它不能不被注意到，黑客的工作变得甚至可能比涂鸦艺术家的工作更强大。那是一个有趣的想法不是吗？对我来说，这就是黑客和真正为我们提供有争议但有效的艺术形式之间的区别。

##### 放下你的喷雾罐，听着！

回到最初的想法，我不知道“艺术”的界限在哪里。事实上，我不太在乎。这不是我能说的——这纯粹是主观的。我只是想说艺术可以在最意想不到的地方找到。艺术当然不是由它的媒介来定义的。几个世纪以来，这个事实已经被反复证明了。五十年前，没有人会认为录像是一件艺术品，但是现在美术馆里的“录像装置”非常普遍。

下次你在网上看到吸引你注意的东西，无论是研究还是项目，无论是有趣还是令人沮丧，无论是什么——[在 SitePoint 论坛的这个帖子](http://www.sitepointforums.com/showthread.php?s=&threadid=91533)中发布它，展示你选择的作品，告诉我们你为什么选择它。

虽然我在这篇文章中关注的是涂鸦，但我相信互动艺术在现代艺术界占据一席之地只是时间问题。我的例子仅仅是为了给辩论火上浇油。我很想看看那里有什么，在你的帮助下，我们可能会有很多收藏。在我看来，网络艺术越早真正冲击主流越好。
