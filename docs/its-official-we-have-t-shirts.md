# 这是正式的——我们有 t 恤！

> 原文：<https://www.sitepoint.com/its-official-we-have-t-shirts/>

我已经谈论它很长时间了，如果你想知道我是否真的会实现它，这是可以原谅的，所以我放心地把我的钱放到我的嘴上，并宣布我们新的 SitePoint t 恤的发布。那些经常光顾 [SitePoint 论坛](https://www.sitepoint.com/forums/)的人可能还记得，我们在 [99Designs](http://99designs.com/) 举办了一场 t 恤设计比赛，然后在论坛上进行投票，选出最受欢迎的设计。[的投票结果如此接近](https://www.sitepoint.com/forums/showthread.php?t=684908)，以至于最后我们决定用两个选项进行竞选。我们已经决定使用按需打印服务，所以现在就来 Printfection 上查看我们的 [SitePoint 商店获取您的衬衫吧！](http://www.printfection.com/sitepoint)

![SitePoint t-shirt](img/c2cc0a0dab7b9bb5f6a6e58ec3ea13f8.png "InCode") ![SitePoint t-shirt](img/f4d551842477dffcf4be57066486b3c4.png "BetterDev")

我们社区目前正在进行的另一件令人兴奋的事情是在[脸书](http://www.facebook.com/sitepoint)开展的**了解我们**活动。九月我们将谈论云计算，首先我们的一位开发人员[马尔·柯蒂斯将讲述他在 SitePoint](http://www.facebook.com/sitepoint?v=app_11007063052) 的工作，以及他如何利用亚马逊的 ec2 云服务。周五，我们将听取首席技术官凯文·杨克的意见。如果你想参加我们月底的测验，别忘了特别注意男生们说的话。你可以给自己拿一本我们即将发行的书，书名是*在云中托管你的网站*。

今天我要提到的最后一件事是我们将在下周的[论坛](https://www.sitepoint.com/forums/)上开始的。我很高兴能和马克·布朗聊上一段时间，他是微软网络平台团队的一员。Mark 将回答关于在微软平台上配置和/或运行 PHP 应用程序的问题。如果这听起来像是你可以在一些帮助下做的事情，那么[将它们张贴在我们的专用论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=382)上，并在每个月的第三个星期五回来查看你的问题的答案。你可以[在这里](https://www.sitepoint.com/forums/showthread.php?t=699785)了解更多。
