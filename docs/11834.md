# Adobe 发布测试版文本布局框架

> 原文:[https://www . site point . com/adobe-releases-beta-text-layout-framework/](https://www.sitepoint.com/adobe-releases-beta-text-layout-framework/)

本周，Adobe 宣布了文本布局框架的测试版，这是一个基于 Adobe Flash Player 10 中新的文本引擎的可扩展库。这将允许开发人员和设计人员在 Web 应用程序中使用更复杂的排版布局。该框架旨在与 Adobe Flash CS4 Professional 或 Adobe Flex 一起使用，包含在 Flex 的下一版本中，代码为 Gumbo。

![Layout](../Images/5a10c1fc272e0f98ce52b0812de781d2.png "Layout")

文本布局框架为 web 提供多语言、印刷质量的排版，包括支持:

*   双向文本、垂直文本和 30 多种书写系统，包括阿拉伯语、希伯来语、中文、日语、朝鲜语、泰语、老挝语、印度的主要书写系统等
*   选择、编辑文本，使文本在多列和链接的容器中流动，并围绕内联图像流动
*   垂直文本、直排内横排(垂直文本中的水平)和东亚版式的对齐器
*   丰富的排版控制，包括字距调整、连字、排版大小写、数字大小写、数字宽度和任意连字符
*   剪切、复制、粘贴、撤销和标准的键盘和鼠标手势用于编辑
*   丰富的开发人员 API，用于操作文本内容、布局、标记和创建自定义文本组件

虽然这可能是开发人员特别感兴趣的(因为它看起来像是在幕后使用它的一点艰苦工作)，但对于设计师来说，这意味着在线排版设计正在接近印刷所能达到的水平。在 Adobe 网站上，你可以测试出八个如何操作和控制文本的例子。示例包括处理列、文本效果、链接容器、连字和图形。第一印象是，这种处理文本的能力有点像 InDesign“lite”，如果您喜欢打字，并且对当前 web 上的限制感到沮丧，这是非常令人兴奋的。你可以[在这里](http://labs.adobe.com/technologies/textlayout/)玩例子。

![LayoutEffects](../Images/38e509df25208f15853c908a28426be7.png "LayoutEffects")

你也可以尝试一下[文本编辑器布局演示](http://labs.adobe.com/technologies/textlayout/demos/) 。
为了看到文本布局框架的运行，一些使用该技术的网站有[纽约时报阅读器演示](http://timesreader.nytimes.com/timesreader/index.html)、波士顿环球阅读器[和](http://www.boston.com/bostonglobe/reader/) [Makebook](http://www.makebook.com/) 。
