# 新年决心:为 PHPMaster 写作

> 原文：<https://www.sitepoint.com/new-years-resolution-write-for-phpmaster/>

新年的开始通常是下决心的时候，比如健康饮食和减肥，学一种乐器，学一门外语等等。今年，为什么不做一个实际的决议:为 PHPMaster 而写！

PHPMaster 正在寻找人们来分享他们的知识、经验和对各种 PHP 和编程主题的看法。有什么要分享的吗？有令你兴奋的事情吗？联系我们友好的编辑人员。

文章长度在 500 到 1500 字之间，主题从教程到社论不等。当然，每篇文章都会给你报酬，但真正的回报是知道你帮助了阅读你作品的其他人。PHPMaster 已经成为各行各业、不同技能水平的 PHP 开发人员的优秀资源，从初学者到专家。

那么流程是怎样的呢？嗯，我们要求你先提出你的文章想法。推销不必太正式；它将涵盖什么，预期长度可能是多少，以及你的目标受众是足够的快速要点。这是为了确保它目前不是由另一个作者写的。(没有人喜欢浪费时间写一篇重复的文章！)

一旦被接受，写你的文章，并通过电子邮件发送给我们。你可以用任何一种对你来说最方便的格式提交，可以是 Microsoft Word、OpenOffice/LibreOffice、Google Doc、RTF 或 HTML。我们的编辑将审查文章的准确性，并在必要时对任何语言进行编辑，然后发回给你审阅。

当一切就绪后，文章就被加载到内容管理系统中，并发布给所有人看！我们将向您发送最终字数，您提交发票，这样您就可以获得工作报酬。

你的作品可以在一个月内被全世界超过 30 万初露头角的开发者和 PHP 专家所接受。这是一个赚取额外收入的好方法，同时也能在 PHP 社区建立你的声誉。你还在等什么？[今天给我发一封电子邮件](mailto:tim.boronczyk@phpmaster.com)介绍你自己并提出你的主题！
