或者不是。也许 IE6 即将消亡的报道被夸大了。

在最近的一次国务院市政厅会议上，希拉里·克林顿的一名工作人员想知道为什么 IE6 被授权给国务院使用。由于这名工作人员最近刚从使用 [的*火狐转移到国家地理空间情报局*](https://www1.nga.mil/About/Pages/default.aspx)(那是秘密间谍业务)，她推断这不可能是纯粹的安全问题。

副国务卿帕特·肯尼迪(Pat Kennedy)表示，尽管 Firefox 是免费的，但“没有什么是真正免费的”(在“免费”前后插入引号)，在一个大型、地理位置广泛的组织中，部署、培训、应用补丁和支持都会产生成本。

好吧，虽然毫无疑问这是真的，但这肯定是作为拥有电脑的一部分而被列入预算的吗？毕竟，这是他们主要的面向 web 的应用程序。如果他们说他们目前没有在修补和支持 IE6 上花费大量资金，我认为我们都应该担心——特别是考虑到 IE6 漫长而不光彩的安全漏洞历史。

然而，对于我们网民来说，这可能是一个发人深省的想法。如果像美国国务院这样的大型、资金充足且接受公众审查的组织发现很难从一个 8 年前的软件中继续前进，那么要克服一百万个城镇图书馆、社区大学、市议会和公司立方体农场的惰性需要什么呢？

我猜，在被 Vista 升级吓到之后，这些组织中的大多数仍然坚持使用 Windows XP。我祈祷 Windows 7 足够好，可以让 IE8 进入许多新系统——在这个过程中让 IE6 退出历史舞台。然而，这仍有待未来几个月的观察。

不过，考虑这个问题很有意思:这是否意味着希拉里使用 IE6 的员工无法观看 Youtube 视频解释他们为什么必须使用 IE6？
