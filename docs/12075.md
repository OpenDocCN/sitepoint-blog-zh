# Photoshop 中的选区，第 4 部分:提示和技巧

> 原文:[https://www . site point . com/selections-in-Photoshop-part-4-tips-tricks/](https://www.sitepoint.com/selections-in-photoshop-part-4-tips-tricks/)

在这个系列中，我们看了一下 Photoshop 的选择工具，以及每一个工具如何有其适当的用途。用于简单形状的[几何工具](https://www.sitepoint.com/photoshop-selections-geometric-tools/)，用于更复杂形状的[手绘工具](https://www.sitepoint.com/selections-in-photoshop-freehand-tools/)，以及用于彩色区域之间边缘清晰的区域的[基于颜色的选择工具](https://www.sitepoint.com/selections-in-photoshop-magic-wand-quick-selection/)。最后，我想向大家展示一些使用这些工具时的技巧和窍门。

**移动您的选择**

使用几何或手绘工具做出选择后，只需使用同一工具拖动即可移动选择(而不是内容)。

*   要移动所选内容，请切换到移动工具(![Move Tool](../Images/63bb6a4eb9292d8ff7e5f2fe2b744fc0.png))并拖到您想要的位置。
*   要将选区或选区内容一次移动一个像素，请在所需方向上按一次箭头键。
*   若要一次移动 10 个像素，请按住 Shift 键并按箭头键。

**创建选区时重新定位选区**
您可能会发现，当您绘制矩形或圆形选区时，选区并不总是在您想要的位置！但是您实际上可以在创建选择时移动它。让我们以椭圆选框工具为例。比如我想在盘子上的圆形肉周围画一个圆形选区。

1.选择隐藏在矩形选框工具下面的椭圆选框工具(![Elliptical Marquee](../Images/7b608f1a57bd2a56476177281e36f431.png))，拖出一个圆。记住，按住 Shift 键会将它限制为一个正圆。

![part4_clip_image004](../Images/5cc12f828bb83138b9370cc2ed7df786.png)

2.好的，我已经开始沿对角线拖出了，但是我可以看到选择的位置完全错了。**不要松开鼠标按钮。**如果您不小心松开了鼠标按钮，请再次绘制选区。

3.仍然按住鼠标按钮，按下空格键，你现在可以移动你的选择到正确的位置。释放空格键(但不是鼠标按钮)并继续拖动，直到你有你的圆在正确的地方，在正确的大小。您可能需要按住空格键几次，以获得正确的位置，但这没关系，您不必重新绘制您的选择，从而节省大量时间。当你对位置满意时，放开鼠标键。这也适用于矩形选框工具。

![Correctly positioned selection](../Images/e6ca167315cdaeca8ec9d3fb18e7f8e7.png)

**隐藏选区**
当您正在进行调整时，有时您可能会发现选定区域周围的边框令人分心，如果是这样，您可以通过按下 **Ctrl + H / Cmd + H** 来暂时隐藏选区。“跳舞的蚂蚁”会消失，但选择仍然有效。按下 **Ctrl + H / Cmd + H** 使选择边框再次可见。

**同时移动和复制**
如果你想复制你选择的东西，这里告诉你如何复制它并同时移动它。

1.使用您选择的任何工具进行选择。我在盘子的右上角选了一颗樱桃。

2.选择移动工具(![Move Tool](../Images/63bb6a4eb9292d8ff7e5f2fe2b744fc0.png))后，按住 **Alt / Option** 。指针变成一个双箭头，表示当您移动选择时将会产生一个副本。

![Duplicating](../Images/a1a56d2deaf2d4c64db6e254824c40a5.png)

3.继续按住 **Alt / Option** ，从您的原始选择中拖出一个副本。松开鼠标键和 **Alt / Option** 键。

您会注意到 cherry 的副本现在被选中了。如果您想制作另一份副本，请重复上述步骤，否则按 **Ctrl + D / Cmd + D** 取消选择。

**柔化选区边缘**
你可以通过消除锯齿和羽化来柔化选区的硬边缘。

***消除锯齿*** 通过柔化边缘像素和背景像素之间的颜色过渡来平滑选区的锯齿边缘。在使用选择工具之前，您必须在工具选项栏**上指定反走样选项。一旦做出选择，就不能添加抗锯齿。**

![Anti Alias](../Images/26b97b6402210fc8770668edb908563b.png)
***羽化*** 通过创建选区及其周围像素的过渡来模糊边缘。请注意，这可能会导致选区边缘的一些细节丢失。您可以通过在工具选项栏的羽化栏中键入一个数字来添加羽化。

**选择一层的内容**

如果你想快速选择一个特定图层的内容，进入图层面板，按住 **Ctrl 键并点击** / **Cmd 键并点击**你想选择的图层的缩略图。

![Selecting Contents Of A Layer](../Images/9480d698b0271e16761c725fbe0d6f2c.png)

好了，这就结束了这一系列的 Photoshop 选区制作。今天的建议将帮助你加快和改进你的选择。现在…练习！！

我建议你挑选出不同形状的物体，人，动物，任何东西的图片，并练习选择。任何人都可以使用几何工具，但你能在你的朋友周围做一个好的选择，并把他们放在一个新的背景上吗？别忘了，你可以通过按住 **Shift** 键来增加选择，也可以通过按住 **Alt / Option** 来减少选择。向前去选择吧！
