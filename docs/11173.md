# 火焰:一种不同的在线绘图工具

> 原文:[https://www . site point . com/flame-a-different-kind-of-online-drawing-tool/](https://www.sitepoint.com/flame-a-different-kind-of-online-drawing-tool/)

火焰是一个在线绘画程序，但它与其他在线工具有很大不同，如 T2 鸟舍。这是彼得·布拉斯科维创作的名为“我是艺术家”的实验项目的一部分？[的逃跑动作](http://www.escapemotions.com)。下面你可以看到整个用户界面的截图。

[![FlameApplication](../Images/2057b58fbf52e068f6926596098ddaa3.png "FlameApplication")](https://i2.sitepoint.com/blogs/wp-content/uploads/2010/03/FlameApplication2.jpg)T3】

该工具允许您选择黑色或白色背景，然后为前景画笔更改调色板。当你使用黑色背景时，调色板是加色的，当你使用白色背景时，调色板是减色的。对于网页设计师来说，我认为 Flame 是一个非常有用的工具，可以为网站创建不寻常的背景、页眉和页脚，或者在 Photoshop(或任何其他图像编辑器)中用作图层。除此之外，它使用起来真的很有趣，效果也非常令人印象深刻。

[![Site](../Images/b5137688dc82844bcc884eda253d3eb9.png "Site")](https://i2.sitepoint.com/blogs/wp-content/uploads/2010/03/Site.jpg)T3】

[![Point](../Images/0120f31ce140cce8d86901e4c6344244.png "Point")](https://i2.sitepoint.com/blogs/wp-content/uploads/2010/03/Point2.jpg)T3】

主要的工具是你用来绘画的刷子。您可以更改颜色、饱和度和不透明度。你也可以改变笔刷的焦点和参数，如混乱和噪音。有一个有用的橡皮擦工具，您也可以选择清除整个画板。使用这个工具是一种非常好的体验。

当您想将作品保存到电脑时，请按“保存”。您的照片没有保存到 Flame 网站上。你的作品将在一个新窗口中打开，然后由你右键单击并保存图片到你的计算机。您可以保存 1680 x 1050 像素的高分辨率图片。
