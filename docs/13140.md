# 根据 Linux 创始人的说法，苹果的 OS X 是“彻头彻尾的垃圾”

> 原文:[https://www . site point . com/apple-OS-x-utter-crap-依 linux-founder/](https://www.sitepoint.com/apples-os-x-utter-crap-according-to-linux-founder/)

上周在墨尔本举行的 linux.conf 会议上，linux 创始人 Linus Torvalds 对 OS X 说了一些严厉的话:

> Linux 开源项目的创始人 Linus Torvalds 说，苹果公司大肆吹捧的新操作系统 OS X Leopard 在某些方面不如 Windows Vista。
> 
> 托瓦尔兹上周在墨尔本参加了 linux.conf.au 会议，并应邀在一次广泛的采访中对 OS X 和 Windows Vista 进行了评判。
> 
> “我不认为它们同样有缺陷——我认为 Leopard 是一个更好的系统，”他说。“(但是)OS X 在某些方面实际上比 Windows 更难编程。他们的文件系统完全是垃圾，这很可怕。”

让我们澄清一下，在这里——Linus 谈论的是为 OS X 开发，而不是 T2 使用它。他确实对这个操作系统有很多积极的评价。有开发者愿意分享他们的经验吗？
