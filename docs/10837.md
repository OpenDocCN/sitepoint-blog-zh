# Robyn 在线视频中的 Twitter 与 3D 相遇

> 原文:[https://www . site point . com/Twitter-meets-3d-in-robyns-online-video/](https://www.sitepoint.com/twitter-meets-3d-in-robyns-online-video/)

如果你是 Twitter 用户，你很可能听说过或看过瑞典歌手罗宾的新音乐视频。如果不是，那么这就是大惊小怪的原因。新的在线视频是一个与 Twitter 整合的 3D 互动事件。这首歌叫做“别他妈的告诉我该做什么”，Robyn 唱的是所有杀死她的东西，几乎是她生活中的所有东西。该视频的特点是这首歌的歌词在彩色形状的雷区中旋转。我没有任何 3D 眼镜，所以它看起来有点模糊，对眼睛有点刺眼，但如果你是一副 3D 眼镜的幸运拥有者，你应该得到文字和形状跳出屏幕的特殊效果。

[![RobynKillingMe](../Images/f2721db4ade0abc0b6c636590f982257.png "RobynKillingMe")T2】](https://www.sitepoint.com/wp-content/uploads/2010/07/RobynKillingMe.jpg)

视频的后半部分是 Twitter 整合的地方。罗宾(和视频制作者)邀请你在推特上谈论是什么杀死了你。如果你想让你的推文成为视频的一部分，你所要做的就是使用#killingme 散列标签和你的抱怨。因此，例如，如果你在 twitter 上发布了类似“我的饮食中缺少巧克力是#killingme”的内容，你的完整推文以及你的 Twitter 名称会出现在视频的左下方。出现在前面的单词是#killingme 将出现在视频的中央。Twitter 消息是随机获取的，所以不能保证你的推文在发送后会成为视频明星，但它应该会在某个时候出现在屏幕上。随着 URL 的每次刷新，Twitter 消息会重新生成，在每个视图上提供一个独特的视频。

该视频由瑞典公司 [Stopp](http://www.stopp.se/index2.php) 制作，完全用代码编写。你可以在 Robyn.com/killingme 的[在线观看视频。](http://www.Robyn.com/killingme)
