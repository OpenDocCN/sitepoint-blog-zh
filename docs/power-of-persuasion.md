# 说服的力量

> 原文：<https://www.sitepoint.com/power-of-persuasion/>

上周，我写了一篇关于优柔寡断的客户 T2 的帖子，以及你可以采取的一些行动来推进项目。停滞不前的项目就是一个例子，在这种情况下，有说服力是值得的，特别是如果客户倾向于你的方向，只是需要一些保证，他们正在做出一个好的决定。

说服他人采取行动的能力是一种强大的技能，这种技能将在你与客户处于同一战壕的时候为你服务，敦促他们就项目做出决定，以保持项目进展。

从你的营销文案到你开展的每一项营销活动，说服在营销中扮演着重要的角色。这也是你成功向潜在客户推销服务的关键因素之一。更不用说它在你拥有的任何其他合作业务关系中都是无价的。

如果你的营销活动没有成功地让你的受众采取想要的行动，你发现很难让潜在客户做出决定，或者你无法让他们接受你的想法，你的说服力可能需要一些帮助。

## 微调你的说服技巧

说服不仅仅是能够说服某人去做某事，尽管这无疑是最简单形式的说服的一个关键要素。成功的说服还应着眼于帮助他人看到并理解如果他们做出某个决定，他们将获得的好处。你可以做几件事来提高你向听众传达这一基本价值观的能力。

 ****确定具体的最终目标**

想要对方选择对你有利的东西是不够的。为了有效地使用你的说服技巧，你需要明确地概述你希望对方做什么，他们应该如何做，以及你希望他们什么时候做。这些因素中的每一个都应该是你所有有说服力的沟通中的焦点，这样你的目标受众和你自己都很清楚。

 ****先说服自己**

如果你没有 100%参与到你想要推销、影响或说服对方的事情中，你的说服力就已经不足了。彻底了解你的受众通过做出某个决定将获得的利益，并相信这个决定将为他们带来最好的结果，这是至关重要的。对过程有信心以及你传递这种价值的能力同样重要。

 ****唱反调**

一旦你确信你的服务、产品、想法等。是最好的解决方案，是时候把它拆开了。考虑你的目标受众将会遇到的所有潜在问题，他们在做出决定时可能会面临哪些挑战，以及他们可能认为你的解决方案有哪些不足之处。然后花时间准备答案，解决问题，找出任何不足的地方，这样你就完全准备好处理和解决任何可能减缓决策过程的问题。

如果你能够完善你的说服能力，并在适当的时候使用它，你会发现这会使销售更容易，带来新的机会和更高的满意度。

当你需要的时候，你有说服力吗？什么使你在这方面最成功？
