# 需要建议吗？导师可能是答案

> 原文:[https://www.sitepoint.com/finding-a-mentor/](https://www.sitepoint.com/finding-a-mentor/)

你有没有遇到过这样的情况——面临与客户的冲突，努力突破新项目的挑战，或者发现很难推销自己——当你希望有人能给你正确的答案时？你可能需要一个导师。虽然导师不能完全解决你的所有问题，但他或她可以在你需要的时候提供建议，帮助你解决问题，并帮助你做得更好。

**导师的角色**

导师同时扮演着你的教练、共鸣板、顾问和啦啦队长的角色。一个导师可以指导和支持你，同时帮助你找到可以提高你的工作和你设定并达到目标的能力的资源。有时，导师甚至会唱反调，帮助你跳出框框思考，开发你自己可能想不到的新解决方案。

作为你的靠山，导师可以公开支持你，帮助你在你的行业站稳脚跟，并通过把你介绍给其他成功的专业人士来扩大你的人脉。

**在导师身上寻找什么**

使一个导师非常适合你的品质可能会非常具体地针对你的情况。但是你应该在你考虑作为导师的人身上寻找一些一般的品质。你的导师应该是:

*   愿意和你分享他们的时间
*   受到同事的尊敬
*   一个好的听众
*   正面的榜样
*   诚实直爽
*   愿意分享他们的知识和经验
*   尊重你
*   鼓舞人心的和热情的

**如何找到导师**

找导师不是一件容易的事。除非你已经有了一个可能很适合做你导师的私人联系人，否则你必须做一些详尽的研究。您可能想看的一些地方包括:

*   雇主或前雇主
*   [分数](http://www.score.org/counseling_training.html)(美国)
*   大学母校或校友会
*   专业组织
*   在线网络商店
*   家人和朋友

一旦你找到了一个潜在的候选人，你会想要被介绍(或自我介绍)，确保他们在专业上是合适的，感受一下这种融洽，然后提出成为你的导师的想法。是的，寻找导师需要做很多工作，有很多步骤，但这是值得的。
