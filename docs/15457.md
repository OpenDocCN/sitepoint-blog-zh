**问**我在首页重写一些 javascript 代码时遇到了麻烦，这些代码我已经添加到我的页面中，以帮助跟踪访问者信息。有什么方法可以阻止 Front Page 修改 SuperStats javascript？你知道为什么有些但不是所有访问网页的电脑会收到 javascript 错误信息吗？—劳拉·蔡斯

**A.** 所见即所得的 HTML 编辑器(比如 MS FrontPage)通常会修改你的 HTML 代码，把它搞砸。为了避免这种情况，使用记事本等文本编辑器打开 html 文件，将脚本粘贴到< head > < /head >标签之间。

问:HTML 中有没有一种方法可以在 HTML 表单上做数学运算？例如，我正在写一份网上订购单。如果客户可以点击一个总计按钮并计算购买的商品数量乘以每件商品的购买价格，那就太好了。此外，一个漂亮的总计按钮将是伟大的。我要学 JavaScript 才能做这个吗，还是可以用 HTML 来做？—杰夫·约翰斯顿

HTML 仅仅控制网页的显示方式。要进行任何形式的表单处理、数学等，都需要使用 JavaScript(客户端)或 PERL(一种服务器端编程语言)。服务器端只是指工作/处理由服务器完成，然后发送回浏览器。

**问。**我需要为我的网站下拉菜单，将允许访问所有 58 个加州县。我尝试过剪切和粘贴，但我不认为我了解所有我知道的 html。下拉菜单有标准格式吗？乔治·德拉克鲁兹

下拉菜单并不仅仅依靠 HTML 来工作。通常他们会调用一个 PERL 脚本来处理请求并重定向访问者。你可以在这里找到这些脚本的列表[。如果你不想或者不能在你的服务器上运行 CGI 脚本，你可以使用](http://cgi.resourceindex.com/Programs_and_Scripts/Perl/Redirection/)[马赫](http://www.mach.org/)——他们的服务器上有一个下拉菜单脚本，你可以使用(感谢杰克为我指出这项服务)。

或者，您可以使用 JavaScript，但是下拉菜单对禁用 JavaScript 的访问者不起作用。然而，它比 PERL 方法要快得多。您将在: [JavaScriptSource](http://javascript.internet.com/navigation/menu.html) 找到您需要的 JavaScript 代码。

问:我的网站有一个 4 框架布局，我想在其中一个框架上放置一些下拉框，并将其定位到另一个框架，有没有任何常规的 html/java 下拉脚本可以让你为它指定一个目标？我知道你可以用 cgi 重定向来做到这一点，但是对目标来说会容易得多..—丹尼·凯利

你可以用 JavaScripts 和 CGI 重定向脚本来实现。只需将 target=_NAME 标记附加到您在脚本中指定的 URL。例如:

<option value="https://www.sitepoint.com/faqs/">文章</option>

变成了:

<option value="http://https://www.sitepoint.com/faqs/" target="_FRAME-NAME">文章</option>
