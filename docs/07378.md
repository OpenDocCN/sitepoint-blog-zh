# 我的 5 大 AngularJS 演示应用

> 原文:[https://www . site point . com/top-5-angular js-demo-applications/](https://www.sitepoint.com/top-5-angularjs-demo-applications/)

以下是我看过的前 5 个演示/应用(目前为止！)其中使用了 [AngularJS 框架](http://angularjs.org/)。截至 2012 年 11 月 16 日的演示/排名。如果你以前没有听说过 **Angular JS，它是一个 JavaScript 框架，非常适合创建动态 HTML 模板，可以利用双向数据绑定和自定义 HTML 标签等酷功能。**尽管它现在绝对是一个可用的框架，但它是一个值得期待的未来。
