## 2.Ajax 上传

允许您轻松上传多个文件，无需刷新页面，并使用任何元素来显示文件选择窗口。它适用于所有主流浏览器，并且不需要任何库就能运行。AJAX 上传不污染全局命名空间，用 jQuery，Prototypejs 测试。
![Ajax-Upload.jpg](../Images/265afaf90fc7681279872268619f3b96.png)
T3】来源

## 3.Ajax 文件上传

由 yvind Saltvik 创建的 Ajaxupload 插件的黑客版本，对于正常使用来说已经足够好了。它的想法是创建一个 iframe 并将指定的表单提交给它进行进一步的处理。
![Ajax-File-Upload.jpg](../Images/469206936e92c9dc823c0cacf8ae549b.png)

## 4.jQuery 多文件上传插件

jQuery JavaScript 库的一个不显眼的插件，帮助用户轻松快捷地选择多个文件进行上传，同时还提供一些基本的验证功能，帮助开发人员识别简单的错误，而不必提交表单。
![jQuery-Multiple-File-Upload-Plugin.jpg](../Images/a1383a4f5d8276a199d1dbeb01b12cb6.png)

## 5.jQuery 表单插件–文件上传

这个插件是 jQuery 表单插件的一部分，它使你更容易使用 Ajax 提交表单，包括输入文件，并且不需要 flash。
![jQuery-Form-Plugin.jpg](../Images/f67de4329bc9faab952a68c825dc0394.png)
T3】来源

## 6.上传

一个 jQuery 插件，允许在你的网站上轻松集成多个(或单个)文件上传。它需要 Flash 和任何后端开发语言。一系列选项允许高级用户进行完全定制，但基本实现非常简单，即使是编程新手也能做到。
![Uploadify.jpg](../Images/e8a882a9d870d488ab27057bd7627563.png)
T3】来源

## 7.SWFUpload jQuery 插件

这个插件是基于 SWFUpload 和 Jquery 创建的，它支持单个或多个文件上传，并且需要 flash。
![SWFUpload-jQuery-Plugin.jpg](../Images/6062cd3c520c6bed100b91d954971fcd.png)
T3】来源

## 8.jqswfupload

这个插件是使用 swfupload 库进行多文件上传的更简单的方法。你可以用一行代码为基于 Flickr 的 multiupload 文件创建一个完整的界面，然后进行自定义设置来控制所有功能的工作方式。
![jqswfupload.jpg](../Images/0b539203aac642154f452af7279a9d2f.png)

## 9.jcUpload

这是一个免费的小插件，支持类似 ajax 的文件上传。它需要 Adobe Flash player 和 jQuery JavaScript 库。
![jcUpload.jpg](../Images/2a5b318fa8e154af7289c96088778b9f.png)
