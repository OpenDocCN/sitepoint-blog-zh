# 找回失去的客户——以“尝试、尝试、再尝试”为例

> 原文：<https://www.sitepoint.com/getting-back-a-lost-client-a-case-study-in-try-try-and-try-again/>

[https://www.sitepoint.com/blog-post-view.php?id=176295](https://www.sitepoint.com/blog/)

这是早在六月份的一篇博客的后续，名为“你总是在推销约定——即使你已经卖掉了。”在那篇博客中，我讲述了我们达成交易后失去一个客户的故事。

以下是最新消息:

我成功地转售了婚约。

以下是方法:

1.我打出了“义愤填膺”的牌，让首席执行官知道，终止我们已经同意的订婚是不合适的，也是不专业的。如果他们的需求发生了变化，我会主动提出转移关注点，但我仍然希望公司能同意他们的意见。

2.我去找了一个制造不和的中层员工。这位员工不喜欢我被雇用，因为对她来说，我对她的工作描述是一个威胁。所以我去找了我在公司的一个盟友，一个比她更受尊敬的人，说服他在首席执行官面前为我说话。这很容易，因为事实证明，这位中层员工以政治(而非结果)导向著称。

3.我巧妙地说明了公司在我参与的事情上仍然没有取得任何进展，我可以介入并迅速让事情回到正轨。这被证明是一个重要的策略，因为高层领导厌倦了看不到结果。

因此，正如你总是在出售你赢得的婚约一样，你也可以再次出售你失去的婚约，然后重新开始工作。

此处的课程:

–关注客户的权力和影响力。在问题出现之前，与有影响力的人发展关系。

不要放弃。几乎总有办法收复失地。
