# 倾向于协同定位

> 原文:[https://www.sitepoint.com/leaning-towards-co-location/](https://www.sitepoint.com/leaning-towards-co-location/)

随着在线行业的停滞，电子商务经济学家努力降低公司的运营成本。对于那些靠流量生存的网络公司来说，毫无疑问，他们一直在努力降低主机和带宽成本。许多人问自己“协同定位比专用托管服务器便宜吗？”

##### 什么是协同定位？

通过协同定位，网站管理员提供服务器，并将其存放在数据中心或连接到互联网的设施中。通常，除了直接接入服务器的互联网上行链路之外，客户还需要支付住房费(称为“机架费”)。另一方面，专用服务器只是这样:你有一台专用的、租给你的服务器，由数据中心或提供商管理，但你对服务器没有绝对的所有权。

这就是为什么许多人喜欢协同定位的原因。购买专用服务器一年后，您可能已经支付了服务器 3-4 倍的费用，但是您并不拥有硬件的所有权。然而，如果你不希望有很大的流量，你更喜欢提供商提供的额外支持，专用服务器可能是正确的选择。对您的企业来说，拥有服务器并负责更换硬件可能是不必要的成本。在竞争异常激烈的专用服务器市场，一台好的服务器每月的价格在 100 美元(低端)到 500 美元之间。然而，如果您已经经历了流量增长，并且需要更多的资源，甚至可能需要额外的服务器，我建议您考虑并置服务器。

与购买房屋和支付财产税或租赁公寓相比，共址和专用服务器更好。两者各有优势，但视情况而异。知道你正在进入什么的唯一方法是坐下来算一算。协同定位一开始比较贵，从长远来看比较便宜。从长远来看，专用服务器更贵，但前期成本更低。如果你觉得协同定位是你业务的最佳解决方案，这篇文章将解释你如何使用协同定位建立一个可靠的网站。
本指南将涵盖以下内容:

1.  操作系统的价格和优缺点。
2.  服务器硬件价格简介以及每种硬件的建议用途。
3.  选择 ISP 的推荐方法。

##### 选择操作系统

目前有几种不同的网络操作系统，因此找到一种既能满足您的需求又符合您的预算的操作系统是一项艰巨的任务。我选择了两个比较流行的来讨论:Windows NT 和 Linux。

***Windows NT***

这两种流行的操作系统之间的明显区别是，NT 试图将熟悉的 Windows 界面引入服务器环境。NT 允许网站管理员通过点击来操作他们的服务器，这是 NT 越来越受欢迎的主要原因。除了用户友好的界面之外，NT 对于计划运行第三方开发的软件的站点来说是理想的，并且 NT 使得使用数据库(Access、Oracle、SQL、DB2)变得特别容易。

价格:300 美元

***Linux*T3】**

Linux 是我个人最喜欢的，不可否认它更难学，但它可能是最安全的操作系统。你可以期待一台 Linux 机器运行几年而不崩溃。此外，在 Linux 机器上发现的病毒非常少，Linux 在几乎所有硬件上都可以非常高效地运行。Linux 站点通常运行免费的 Apache Web 服务器，虽然它缺少 nt 所包含的点击式工具，但大多数有经验的管理员更喜欢 Linux 通过允许您使用原始代码而提供的灵活性。

价格:免费

##### 选择服务器硬件

为您的服务器选择正确的硬件与选择操作系统一样重要。业界最受欢迎的三个平台是 Sun、Intel 和 Cobalt。Sun 在服务器业务中以其速度、效率和可靠性而闻名；然而，英特尔并没有落后太多，对于一般的网站管理员来说，它的价格要便宜得多。

由太阳微系统公司拥有的 Cobalt 最近已经成为服务器硬件的主要参与者。Cobalt 非常适合需要支持很多站点的站长，也是三者中最便宜的。Cobalt RaQ 服务器是一个很好的开箱即用的机器，但是不推荐用于高流量的站点。

我假设您正在构建自己的服务器，已经决定不购买开箱即用的机器。

***网卡***

下一步是选择您的网卡。有数不清的网卡可用，但我不建议你选择便宜的。您的网卡是您与互联网的连接，由于您是协同定位的(自己支付硬件费用)，如果您被迫进行更换，您将经历一些重大的停机时间和费用，包括人工和部件。

***记忆***

为您的服务器选择内存容量可以根据您的计划而有所不同，但我强烈建议您至少购买 256MB。服务器会将访问过的网页存储在内存中，所以内存越大，可以存储的网页就越多。

***硬盘***

硬盘驱动器在性能方面也起着重要作用。硬盘通常有 5400、7200、10000 和 15,000 RPM 的速度。这些数字代表硬盘内部金属盘的转速。不用说，转速越快，服务器访问数据的速度就越快。

有两种类型的硬盘–SCSI(小型计算机系统接口)和 IDE(集成驱动器电子设备)。Web 服务器的推荐硬盘是 7200+RPM SCSI。

##### 选择提供商

选择您的提供商无疑是这一过程中最重要的决定，不应该掉以轻心。

虽然看起来很明显，但第一步是确定你的需求是什么。我建议你写下你需要从主持人那里得到什么，不需要什么，在你“听”推销的时候，把这份清单放在面前。否则，你可能会被供应商如此积极营销的小东西所说服，而不能确保你得到你所需要的东西。

要开始你的研究，请访问[主机图表](http://www.HostChart.com)和[主机朋友](http://www.HostAmigos.com)。在 HostChart 找到一个提供商后，我建议访问 HostAmigos 来研究你感兴趣的提供商的总体声誉。

搜索提供商时要注意的事项包括:

***价格***

你应该特别注意的一个明显因素是价格。我不建议你去找最便宜的主机，因为“你为你得到的东西付费”这句话绝对适用于虚拟主机。你可能想走更便宜的路线，但是频繁的停机是非常不专业的，并且会导致你的用户立即失去信任。

***技术支持***

可用的技术支持是另一个应该认真对待的问题，应该在您的决策中发挥重要作用。大多数著名的主机通过电子邮件和电话提供 24/7 的支持。

不幸的是，如果您的服务器硬件有问题，支持将不包括这一点。然而，问题可能会在您的连接或设置服务器时出现，快速响应至关重要。

***冗余***

我强烈建议你关注的另一个重要方面是你的主人的关系。您的服务器可能拥有最好的硬件，但如果您的主机无法提供良好的互联网连接，您可能会遇到停机，导致业务损失。好的托管公司有多种互联网连接，并且应该在停电的情况下方便使用现场备用设备。

##### 要问主人的 5 个问题！

为了总结上述信息，我建议您提出以下问题:

***1。您提供全天候支持吗？通过电子邮件、电话等。？*T3】**

这是极其重要的。测试托管公司支持水平的一个好方法是打电话给他们的支持热线，看看你是否能联系到某人，或者简单地让他们发一封电子邮件询问一个托管包，并比较他们与其他公司的响应时间。

***2。您有什么类型的备份系统？*T3】**

除此之外，你应该问备份的费用是多少，多久进行一次。

***3。你提供 30 天退款保证吗？*T3】**

如果你支付了安装费，最后因为不满意而更换了主机，你可能会发现自己损失了几百美元。此外，大多数信誉良好的主机提供 30 天的退款保证激励，这也表明了他们对自己的服务的信心。

***4。您的工厂目前有多少个互联网连接？*T3】**

大多数声誉良好的主机都会有至少两个连接进入其数据中心。这一点相当重要:在生产线被切断的情况下，你不会经历停机时间。

***5。服务器安装的周转时间是多少？*T3】**

虽然这肯定会随着你的项目的紧急程度而变化，但我不建议主持人说需要一周或更长时间来设置你。如果他们让你上网那么慢，那就很难说他们能为你提供足够的支持。

我听过无数站长的恐怖故事，他们接到提供商打来的可怕电话，让他们掏钱买网卡或硬盘。如果你的财务状况同样不稳定，那就选择专用服务器。有一个专用的服务器并不影响别人如何看待你的业务，但有一个无法访问的主页意味着即时的信誉损失。如果您仍然认为主机托管是您的选择，请谨慎选择您的主机，并且在硬件出现故障的情况下，您可以随时向您的提供商提供备用网卡。
