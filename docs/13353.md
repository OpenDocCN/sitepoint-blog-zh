# 卖文字链接危险吗？

> 原文:[https://www.sitepoint.com/is-selling-text-links-dangerous/](https://www.sitepoint.com/is-selling-text-links-dangerous/)

毫无疑问，出售文字链接是一笔大生意，但这是一项有风险的业务吗？出售文字链接危险吗？这个问题越来越频繁地出现在站长论坛上。对我来说，我真的想到了两个问题:

**1。卖文字链接会伤害我的站点吗？**

我不认为出售(或购买)文字链接有什么不对。但我的看法无关紧要。事实是，买卖文本链接**可能会**潜在地伤害你的网站。来自谷歌网站管理员指南:

> “不要参与旨在提高网站排名或 PageRank 的链接计划”。

很明显这很符合。此外，谷歌的马特·卡茨说:

> 但是对于其他人，让我来谈谈为什么我们认为通过购买链接来获得 PageRank 不在我们的指导方针之内。谷歌(和几乎所有其他主要搜索引擎)使用超链接来帮助确定声誉。链接通常是由选择给出的编辑投票，基于链接的分析极大地提高了网络搜索的质量。出售链接会模糊基于链接的声誉的质量，并使许多搜索引擎(不仅仅是谷歌)更难返回相关结果。

但是搜索引擎检测不到付费链接吗？

是的，谷歌有各种检测这种链接的算法方法，而且效果很好。很明显，谷歌，可能还有其他搜索引擎，不赞成出售文本链接。

文字链接广告公司的卡梅伦·奥尔特伊斯说:

> “我看到的因购买链接而被批评的人的最大问题是他们懒惰。他们不是致力于创造引人注目的内容，自然会吸引链接，而是在购买链接上投入了大量资金，然后就收工了。如果你对此感到内疚，那你就是在自找麻烦。”

就连卖文字链接广告的公司(披露，我用的)也知道你会惹上麻烦，并确认卖链接是可以被检测出来的。

尽管许多公司推出了将链接直接嵌入内容的产品，但这将更难被发现。谷歌建议对你出售的链接使用“不追随”属性，我想我们都知道这是行不通的。对我来说，我的建议是只卖尽可能相关的链接，购买也一样。不要在婚纱网站上买/卖伟哥的链接。

**2。底层什么时候会退出卖文字链接？**

对我来说，这个问题远比购买/出售文本链接的危险和道德更有说服力。只要有市场，这种情况就在发生，并将继续发生。搜索引擎变得越来越聪明，随着时间的推移，搜索引擎会发现一些模式。买家和卖家通过使用更智能的布局和更相关的链接来适应，循环往复。只要搜索引擎找不到自动识别文本链接的方法，文本链接业务就会继续存在。你总是听到传言说谷歌和其他引擎正在雇人专门搜寻付费链接，但即使是这种方法也只能在人们适应后的短时间内奏效。

像任何行业一样，出售文本链接最终肯定会枯竭，但只要链接是搜索引擎提供结果的货币，那么文本链接的买卖就会持续下去，直到出现更好的东西来取代它。
