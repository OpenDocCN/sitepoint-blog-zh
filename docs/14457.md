# open NIC——民主命名系统

> 原文:[https://www.sitepoint.com/opennic-democratic-name-system/](https://www.sitepoint.com/opennic-democratic-name-system/)

今天早上，当我在搜索一些公共域名服务器进行测试时，我偶然发现了 OpenNIC(【http://www.opennic.unrated.net】T2)。我在 72 小时前做了一个 DNS 更改，一些宽带 ISP 仍然没有接受这个更改。

OpenNIC 不仅仅是公共 DNS——然而，为了我的测试目的，它很好地确认了它实际上是 ISP 的，而不是一些错误的绑定错误。

在解决了我的问题后，我进一步查看了网站，惊喜地发现了一个独特的公共 DNS 系统。人们不仅可以在服务器、系统或个人计算机级别使用这些域名服务器，还可以试验免费的替代顶级域名(TLD)供个人使用。印第。极客，。空，。oss 和. parody。当然，这些域名只对那些使用 OpenNIC DNS 的人可见，然而，这正是我感兴趣的地方。你可以把一个域放在一个公共服务器上，默认情况下，它只限于社区中的用户，非常适合开发和登台，或者只适合业余爱好者。

其次，由于 OpenNIC 支持所有主要的 TLD(。com，。网，。组织，。密耳，。gov，。edu 等，包括两个字符的国家代码)，人们还可以通过访问因特网其余部分的名称解析而受益。OpenNIC 还支持 AlterNIC 和 Pacific 根域。

绝对值得浏览一下，看看互联网上的一些社区是如何保持独立和以用户为中心的。
