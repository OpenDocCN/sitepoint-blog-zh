# RC9au–早上好，昆士兰

> 原文：<https://www.sitepoint.com/rc9au-good-morning-queensland/>

你好 RubySourcers。我曾在墨尔本以北很远的地方旅行到澳大利亚的铁路站。这一次是在一个叫做[安斯沃斯湖](http://maps.google.com/maps?f=q&source=s_q&hl=en&geocode=&q=Lake+Ainsworth,+Lennox+Head,+New+South+Wales,+Australia&aq=0&sll=37.0625,-95.677068&sspn=36.368578,70.3125&ie=UTF8&hq=&hnear=Lake+Ainsworth&z=15)的地方举行，澳大利亚铁路公司的精英将在那里全力以赴。

在接下来的 72 小时里，我将为大家提供大量来自 RailsCamp 的图片和新闻。我希望得到一顶有几张卡片和一根羽毛的帽子，这样更像记者，但是我找不到！我的一个朋友说我应该说我是一个搞笑的 HTML 程序员，所以我很有可能会这么做。

现在，我正坐在黄金海岸 Coolangatta 机场附近的一家旅馆的一个黑暗、寒冷的房间里，但我很快就会去机场赶一辆公共汽车，这条路的钢铁之马，南下到湖边。

不幸的是，我似乎把墨尔本的北极天气和我的寒冷带到了这里，所以希望咖啡像汹涌的洪流一样流动，让我继续前进。

要了解最新情况，您可以关注我们的 [Twitter](http://twitter.com/rubysource) 、[脸书](http://www.facebook.com/rubysource)，或者保持[真正简单的辛迪加](https://www.sitepoint.com/feed)。

暂时就这些，几个小时后我会带着更多回来。
