# ColdFusion 的一周:1 月 30 日-2 月 5 日:每样都有一点

> 原文：<https://www.sitepoint.com/the-week-in-coldfusion-30-jan-5-feb-a-bit-of-everything/>

随着一月的过去，这一年真的开始步入正轨，ColdFusion 博客比以往任何时候都更忙。

在编码部门，Steve Nelson 继续他的关于使用 Google Calendar API 和 [Google Calendar API 的系列文章——使用 ColdFusion](http://www.webapper.net/index.cfm/2008/1/31/Google-Calendar-API--Creating-a-new-Calendar-with-ColdFusion) 和 [Google Calendar API 创建新日历:使用 ColdFusion 获得日历列表](http://www.webapper.net/index.cfm/2008/2/1/Google-Calendar-API-Using-ColdFusion-to-get-a-list-of-Calendars)。随着谷歌日历集成工作的出现，我期待着史蒂夫在这方面的更多文章。Anuj Gakhar 展示了使用 [CFQuery 和底层 Java 对象](http://www.anujgakhar.com/2008/02/01/cfquery-and-the-underlying-java-objects/)检查 ColdFusion 的 Java 底层是多么容易，Rick Osborne 已经推出了一系列关于[的教程(目前有四个),使用 ColdFusion](http://rickosborne.org/blog/index.php/2008/01/28/generating-scalable-stretchy-and-smart-graphics-with-coldfusion-part-1/) 生成可伸缩、有弹性和智能的图形，而不使用 CFIMAGE，事实上根本不需要 ColdFusion 8。Ben Nadel 正在使用 ColdFusion 版本处理著名的 Matt 脚本存档[表单脚本(并且有一个有趣的故事，讲的是一个坚持要转用 PERL 的客户)。](http://www.bennadel.com/index.cfm?dax=blog:1136.view)

如果你追求开发最佳实践，Terrence Ryan 已经写了一篇关于如何说服自己进行单元测试的文章。需要鼓励？看看吧！

在过去的三周里，我发表了一些关于开源世界有这么多伟大的发布是多么“忙碌”的评论，所以我不会重复我自己，只是指出开源库 [RIAForge 已经更新了](http://www.coldfusionjedi.com/index.cfm/2008/2/1/RIAForge-Update)新的 SVN 特性，现在托管了 400 多个活动项目。Brian Rinaldi 在他的 [ColdFusion 开源更新](http://www.remotesynthesis.com/blog/index.cfm/2008/2/4/ColdFusion-OpenSource-Update--February-4-2008)中强调了一些最重要的新项目和更新项目。最让我感兴趣的是[特伦斯·瑞恩的《大本营》CFC](http://www.numtopia.com/terry/blog/archives/2008/01/basecamp_cfc.cfm) 。

本周在框架中， [Mach II 1.6 Spec: Caching](http://www.mach-ii.com/index.cfm/go/blog:showEntry/entryId/D807694C%2D9C20%2D26ED%2DD6087D6C36833208) 已经发布供社区反馈。[保险丝盒](http://www.fusebox.org/)团队正在寻求帮助[编写保险丝盒样本应用](http://www.teratech.com/blog/index.cfm/2008/1/29/Call-for-Fusebox-sample-apps)。FarCry 5 是一个应用程序框架和内容管理系统,[已经发布公开测试版](http://www.jeffcoughlin.com/blog/index.cfm/2008/1/31/FarCry-50-Beta-Released)。

最后，新一期的 Fusion Authority 季度更新已经发布—[cold Fusion 8 特别版](https://secure.houseoffusion.com/vol2issue3.cfm)—但由于我的副本尚未到达，所以除了官方网站上的内容，我不能告诉你更多信息。请继续关注即将到来的更多信息！
