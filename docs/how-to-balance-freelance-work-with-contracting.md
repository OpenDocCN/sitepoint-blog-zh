# 如何平衡自由职业和承包工作

> 原文：<https://www.sitepoint.com/how-to-balance-freelance-work-with-contracting/>

对于渴望项目的自由职业者来说，短期合同工作是一个值得尝试的途径。注册一个代理机构，如果你有销售技能和可靠的参考，你可能会发现自己在寻找以前没有的联系方式。

那是一个很好的位置。通常，这份工作也很好——我发现承包给我提供了获得没有广告的工作的机会，而且比大多数广告更适合我的技能和经验。暂时有一份有保障的收入也不错，不必对一个组织做出长期承诺。对于想接合同的自由职业者来说，最大的障碍是合同通常是全职的。

如果你签的是全职合同，你会如何对待你的其他客户？

与这个问题相关的心理障碍通常足以让自由职业者完全放弃签约。但是有一些策略可以让你在不影响你自由职业的情况下签订短期全职合同。这些是我用得最多的。

## 清理甲板

我接到的最后一份合同是在接到招聘人员电话的三天后开始的。这使得您没有太多时间来尽可能更新您的其他项目。另一方面，如果你正在考虑合同，你的周计划中可能会有一些空闲时间。

不要把时间花在踢你的脚后跟和享受你即将失去的自由上。把它花在清理自由职业者的甲板上:

*   提前做好你能做的任何工作
*   发送并跟踪任何未付款的发票
*   根据需要检查您的时间表和轮班承诺
*   重新组织你的工作空间:当你需要在第二天上班前将打印好的文件邮寄给客户时，没有什么比在合同签订三周后的周日晚上 11:30 意识到你没有纸更糟糕的了。

## 设定基本规则

在合同期间，你需要以最佳能力工作。要做到这一点，你需要自律。在你开始之前，设定一些基本规则。我的通常是这样的:

*   我不会在合同时间接听客户电话，我会让他们转到语音信箱，在我的午休时间回复他们。
*   我承诺在合同期内不做任何额外的工作，不管这些工作看起来有多好。
*   我在合同期内安排休息时间，并确保我休假。仅仅是一个周日或晚上的休假，就能极大地改变我的动力——更不用说我在合同期限内做出好作品的能力了。

我也和我生活中的人一起制定基本规则——比如，我告诉我的朋友和家人我在接下来的两个月里有一个疯狂的计划。这取代了“你为什么从来不回我电话？”打电话，同时请求他们的支持。

## 允许额外的时间

除了制定基本规则、安排休息时间等等，我通常还会在这段时间致力于工作的时候积累一些脂肪。

例如，在接触期间，我从不在周末工作的理念暂时停止。我不想一周工作七天，但我通常需要接受这样的事实，如果我的合同是一周工作五天，我会在周末做一些客户工作。

要接受这一点，我需要有意识地改变自己的心态，提醒自己牺牲是暂时的。通过给自己额外的时间来完成工作，我减轻了一些压力，否则我可能会在一周内完成工作，并给自己留出时间和空间来专注于做好工作。

## 尽你所能委派任务

如果你使用自由职业者或承包商来支持你的业务，看看你在处理合同时可以外包给他们什么。如果你不这样做，你仍然可以委派一些任务——例如，将你的时间跟踪服务升级为自动给客户开发票。

不要把授权局限于你的工作。与室友或你的伴侣达成协议，这样你就不会试图把去杂货店购物、打扫房间、去朋友家做客和八小时工作安排到下周六。当合同完成时，不要忘记履行你的承诺——不管是什么。

## 管理你的时间

制定好所有这些计划后，要注意时间管理，当机会出现时，要停止工作。

另外，要灵活。计划不同于行动。久而久之，如果你意识到有些事情不顺心，或者你发现你在生活的某些方面苦苦挣扎，停下来想想你该如何改善这种情况。不要盲目地坚持你的计划，因为你没有时间修改它们，或者因为“它们应该有效”

随着合同的深入，调整你的方法，你会找到机会简化你的操作方式。

## 别忘了告诉你的客户

你的客户需要知道你在特定的时间、特定的日期是不在的。我第一次接受合同时，我担心我的客户可能会感到被遗弃，或者认为我正在脱离自由职业，去寻找其他供应商。但是他们没有。

解释发生了什么，为什么，你的客户可能不会被你暂时降低的可用性所困扰。老实说，我觉得我的甚至都没注意到。我仍然在合理的时间范围内回复电话和电子邮件，满足我和他们的最后期限，完成工作。没什么大不了的。
