现代浏览器正迅速汇聚到一组标准化的 API 上，用于无插件、可互操作的实时通信。本专题讲座将涉及使用 Microsoft Edge 和开源库为 web 和移动设备开发实时通信应用，以实现跨平台和移动设备的互操作性。

这篇文章是微软技术倡导者和工程师关于实用 JavaScript 学习、开源项目和互操作性最佳实践的 web 开发系列的一部分，包括[微软 Edge](http://blogs.windows.com/msedgedev/2015/05/06/a-break-from-the-past-part-2-saying-goodbye-to-activex-vbscript-attachevent/?wt.mc_id=DX_838511) 浏览器和新的 [EdgeHTML 渲染引擎](http://blogs.windows.com/msedgedev/2015/02/26/a-break-from-the-past-the-birth-of-microsofts-new-web-rendering-engine/?wt.mc_id=DX_838511)。

我们鼓励您在包括微软 Edge(Windows 10 的默认浏览器)在内的各种浏览器和设备上进行测试，并在[dev.microsoftedge.com](https://dev.windows.com/en-us/?wt.mc_id=DX_838511)使用免费工具，包括[status.microsoftedge.com](https://developer.microsoft.com/en-us/microsoft-edge/platform/status/?wt.mc_id=DX_838511)，这是一个门户网站，提供微软 Edge 和其他浏览器(包括 Internet Explorer)中可互操作的网络平台功能的最新实施状态和未来路线图。此外，[访问 Edge 博客](https://blogs.windows.com/msedgedev/?wt.mc_id=DX_838511)，从微软开发者和专家那里获得最新信息。
