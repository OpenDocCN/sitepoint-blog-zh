# 来自 PDC 的最新 IE7 更新

> 原文：<https://www.sitepoint.com/lates-ie7-updates-from-pdc/>

随着上周在洛杉矶举行的微软专业开发人员会议，许多关于 Internet Explorer 前沿的新闻浮出水面。以下是对开发者的一个总结:

*   IE6 和 IE7 的 Internet Explorer 开发者工具栏已经准备好公开测试版。
*   当寻找 DOCTYPE 声明时，IE7 将正确地跳过 XHTML 文档开头的 prolog，而不会陷入 Quirks 模式。
*   IE7 发布后，IE 浏览器网络平台负责人克里斯·威尔逊[证实了](http://blogs.msdn.com/ie/archive/2005/09/15/467901.aspx)在 ie 浏览器中正确实现 XHTML 支持的愿望。
*   微软和亚马逊合作制定了 [OpenSearch 1.1 规范](http://opensearch.a9.com/)，该规范描述了浏览器如何提供特定于站点的搜索功能，以及网站如何以 HTML 或 RSS (0.9x、1.0、2.0 或 Atom)格式返回这些搜索结果，以便在浏览器中显示。IE7 中的搜索框将建立在这个规范之上。毫无疑问，其他浏览器如 Firefox 和 Safari 也会效仿。
