# 自由职业者的物物交换基础

> 原文：<https://www.sitepoint.com/bartering-basics-for-freelancers/>

物物交换是指在不换钱的情况下，用你的商品/服务交换别人的商品/服务的过程。虽然物物交换有许多不同的形式，但它可以成为自由职业者拓展业务和与他人合作的一种方式。作为自由职业者，这里有一些关于以物易物的基本知识。

**物物交换的好处**

以物易物让你在不产生金钱成本的情况下满足需求。如果你是一个项目的设计者，需要开发数据库的帮助，你可以考虑与需要设计帮助的开发人员交换。这可能会发展成为未来的工作和合作机会。与非行业的同事进行物物交换给了你一个与那些你可能不会与之共事的人建立关系网的机会。

以物易物还能增加你的工作经验，扩大你的投资组合，为你的清单增加另一个有价值的参考，假设进展顺利的话。

**如何开始易货交易**

建立物物交换关系最传统的方式是与某人单独接触。许多易货者是通过简单地遇见一个碰巧知道如何做你需要做的事情的人而形成的。更有意识地，你可以征求一个易货协议，就像你会征求新的业务。一旦您确定另一方是否有兴趣，一个好的起点就是提供一份建议书，一份服务价值比较细目表，以及一份对双方都有利的解释。

有许多易货团体或俱乐部为寻求易货服务的人们提供了一个聚会场所。有些比其他的更正式，你可以通过谷歌搜索“易货俱乐部”这个词找到很多这样的俱乐部基本上，物物交换俱乐部提供了一个找到适合与你建立物物交换关系的人的途径。易货俱乐部的主要特征是其他人提供与你相似的易货服务和提供你需要的服务。

还有一种叫做物物交换的东西。交易所是一个第三方，为其成员提供易货安排。与俱乐部不同的是，你不需要找到另一个会员并建立一对一的交易关系。通过物物交换，你提供了你的服务，而不是直接与他人交易，你得到了你所提供服务的价值。该信用随后适用于由交易所的另一成员为您需要的服务提供的服务。当你需要一个不需要你所提供的服务的人的服务时，这是很有用的，它允许你延迟兑现你的服务，直到你找到一个可接受的提供者。

**什么时候不要以物易物**

以物易物并不总是一个好主意。如果你要和某人做一个单独的安排，你要确保双方提供的服务的相对价值是一致的。这在与你所在行业的人进行交易时是相当容易的，但是如果你是在用草坪护理的网页设计进行交易，那就有点棘手了。在开始之前，你要确保你能达成一个令人满意的安排。

我会犹豫要不要进入一个没有文件准确描述每一方提供什么服务的易货关系。你应该起草一份合同，就像这是一份有报酬的工作，目的是确保双方意见一致，履行合同中的责任。对于你的书来说，文档也是非常好的。例如，美国国税局认为[易货服务应纳税](http://www.irs.gov/businesses/small/article/0,,id=187920,00.html)，它们应该包括在你的损益表中。

在达成物物交换协议之前，你也要确保对方有资格提供他们所交换的服务，并且他们有足够的经验。以物易物不应该意味着廉价或不专业，验证对方的经历是一个好主意，就像你正在雇佣他们一样。物物交换关系应该像你将进入的任何其他商业关系一样，唯一的区别是缺少货币交换。
