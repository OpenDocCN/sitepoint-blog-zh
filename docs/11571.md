# 微软视窗 7:发布日及其未来

> 原文：<https://www.sitepoint.com/microsoft-windows-7-launch-and-future/>

除非你过去一年都住在海王星上，否则你一定会注意到今天是微软新操作系统发布的日子。Windows 7 已经在世界各地发布，所以期待听到关于商店午夜开门和街区大排长龙的消息。亚马逊还宣布，Windows 7 的预购量超过了最新的《哈利·波特》。

与其他产品相比，Windows 7 一直相当低调。微软首席执行官史蒂夫·鲍尔默将出席在纽约举行的正式发布会，但没有什么浮华和魅力:

*   在当前的经济环境下，一个昂贵的大型派对可能会显得庸俗。
*   微软从 Vista 的*“wow starts now”*活动中吸取了教训；对许多用户来说，唯一令人惊叹的因素是他们如何设法让 XP 的继任者如此缓慢和不稳定。

## Windows 7 的新功能

首先，Windows 7 比 Vista 更精简、更快。该操作系统将在包括低功率上网本在内的一系列机器上运行。它会启动得更快，让你更快地上网。

Vista 的膨胀让 Linux 在上网本市场获得了立足之地。人们普遍认为 Linux 是为极客准备的——尽管许多用户无疑会因为一个比 Windows 更防篡改的操作系统而过得更好。微软面临的风险是人们可能会使用并喜欢上 Linux。尽管他们通过延长 XP 的寿命来抵消这种威胁，但微软需要 Windows 7 成为一个更有利可图的长期解决方案。

从美学的角度来看，Windows 7 和 Vista 没什么区别。我不喜欢不必要的桌面效果，最初我也不喜欢 Vista，但我渐渐习惯了。

如果你期待一长串的新特性，你可能会有点失望。支持触摸屏和手写识别，文件共享更容易，省电更好， [XP 模式](https://www.sitepoint.com/windows-7-browser-testing/)将很有用，并且有一些令人愉快的桌面增强功能。然而，操作系统是一种进化而不是一场革命；几乎所有东西都经过了打磨和改进，但没有添加任何激进的东西。也许这样最好。微软试图通过 Longhorn 实现雄心壮志，但技术上令人惊叹的操作系统从未交付。

总的来说，Windows 7 是一次值得的升级。你能否证明这一成本是另一回事。

## 这会是微软的最后一个操作系统吗？

有很多关于操作系统将如何应对云计算的猜测。谷歌计划在 2010 年发布 Chrome 操作系统——它将是免费的，可能包含一个轻量级的 Linux 基础，可以快速启动并启动浏览器。然而，企业用户更加保守，在云计算的好处完全实现之前，wi-fi 将需要普遍普及*(并且可能免费)*。

对微软最大的威胁是它自己的产品。用户通常乐于继续使用 Windows XP 和 Office 2003，因为它能做他们想做的一切。然而，很明显，微软正在将重点转向瘦客户端计算(或者至少是 *trimmer* 客户端计算)。因此，每隔几年收取 200 美元的操作系统升级费用将成为一种过时的收入模式。微软肯定会试用现收现付和广告支持的产品。如果他们成功了，同样的模式可以应用到 Windows 上。

尽管竞争加剧，我怀疑这是我们最后一次看到 Windows。Windows 7 应该会让微软回到正确的道路上，最初的迹象表明它将会成功。但愿如此:人们越早升级，[IE6 就越快被根除](https://www.sitepoint.com/microsoft-support-ie6-2014/)！

您购买了 Windows 7 吗？你安装了吗？你喜欢吗？微软做了足够多的工作来回应 Vista 的批评吗？

最大的问题是:Windows 是越来越不重要，还是用户会继续使用他们知道的东西？
