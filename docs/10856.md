# 在社交媒体上迅速传播

> 原文:[https://www.sitepoint.com/going-viral-with-social-media/](https://www.sitepoint.com/going-viral-with-social-media/)

当我开始担任社区经理时，公平地说，我的重点主要是在[论坛](https://www.sitepoint.com/forums/)上。有(现在仍然是……)这么多工作要做，以使这个地方秩序井然，这是一个明确的优先事项。直到我在那里摸爬滚打了几个月，我才决定着手处理一些其他的社交媒体渠道。我们已经有了一个[脸书粉丝页面](http://www.facebook.com/sitepoint)，它有一小群追随者，但是没有什么方向。我们的 Twitter 账户有 6 万多名粉丝，但除了自动转发新的博客帖子和文章之外，我们没有太多的策略。

我决定先解决 Twitter。这不是一个有意识的决定，而是一个被证明是非常幸运的错误。在对 TweetDeck 的早期尝试中，我犯了一个新手的错误，从错误的账户发推文。虽然它不像著名的沃达丰 tweet-tastrophy(T1)那么糟糕，但也足以让我的胃在嘴里呆上几分钟。对于那些不关注我们的人，我在推特上写道“孩子们在床上，带着我的笔记本电脑和一瓶啤酒到甲板上。是的，我称之为工作”。事实证明，这是我们有史以来最受欢迎的推文之一，它为我赢得了老板的大拇指。因此，我的推特运动由此诞生。人们喜欢知道键盘后面有一个人。当然，我们的自动推文作为资源是有用的，但它是引起回应的个人接触。

![SitePoint forums](../Images/1ca598dd6b7e55f9d0f60654f8b532af.png "sitepoint-forums")

我的脸书战略更有计划。我首先研究了其他受欢迎的粉丝页面，并询问我们当前的粉丝，他们想从我们这里得到什么。结果是非常标准的。人们想要便宜的东西。没什么好惊讶的。就这样，[脸书星期五](https://www.sitepoint.com/there-is-more-to-life-than-forums-or-so-im-told/)诞生了。我还专注于使用我们的粉丝页面来传播更多信息——有些人不知道我们的文章，有些人不知道博客——你们中没有人知道[我们的咖啡杯](http://www.zazzle.com/sitepoint)！我们的粉丝群开始快速攀升——几周前我们达到了 5000 人。

战略的下一阶段是对我们的粉丝页面的美学做一些事情。输入病毒。我们已经决定，因为我们的粉丝页面是关于我们的社区的，所以我们想做一些事情来帮助你在上面留下你的印记。我们希望根据您的照片，围绕拼贴画设计我们的新登录选项卡。如果你想参加，请到我们的[粉丝页面](http://www.facebook.com/sitepoint)上传你的照片。为了有资格参加拼贴，请确保您的照片是你的脸，不超过 50 x 50 像素。但是有一个问题，你需要在你的状态更新中标记我们。为此，键入(而不是复制粘贴)“我将永远在@SitePoint！请检查我…”键入@后，暂停并等待下拉列表出现。您可以从列表中选择 SitePoint，这将激活标签。你只能在这个周末之前参加，所以现在就去吧。

下周，我们将在论坛上发起一场新的竞赛。随着我们新书[网络摄影](https://www.sitepoint.com/books/photography1)的发布，七月在 SitePoint 是关于摄影的。如果您想拍些奇怪的照片，请确保下周这个时候光临我们的[摄影论坛](https://www.sitepoint.com/forums/forumdisplay.php?f=302)，了解您是如何胜出的。我会给你一个线索——如果你喜欢寻宝，这是给你的奖励。
