# SitePoint 祝您节日快乐！

> 原文：<https://www.sitepoint.com/happy-holidays-from-sitepoint/>

在我写这篇文章的时候，Team SitePoint 的大部分人都在三楼开派对。对我们来说，这是重要的一年的结束，我们有很多值得庆祝的事情……当然，在未来的一年里也有很多值得期待的事情！

在 2008 年，您可以从 SitePoint 获得以下内容:

*   我们今年推出的 SitePoint 设计竞赛的新版本如此成功，以至于我们觉得是时候让它自由地在世界上闯出自己的路了。这个活跃的设计社区将被拆分到自己的网站上，并更名为“99 设计”。当然，这次更名将伴随着一个全新的网站视觉设计。请关注我们在 SitePoint 设计竞赛社区举办的设计 99designs 徽标的竞赛！
*   **SitePoint Reference**我们的 [CSS Reference 站点](https://reference.sitepoint.com/css)已经处于关闭测试阶段，我们从 site point 论坛社区获得了大量的反馈。2008 年初，这种“终结所有 CSS 引用的参考”将向世界敞开大门，随之而来的将是 HTML 和 JavaScript 的引用……每一种都是由每种技术的权威人士撰写的。
*   **更多伟大的书籍**我们 2008 年的第一本书将会是<cite>JavaScript 的艺术&科学</cite>，由一个众星云集的 JavaScript 专家团队带给你。
*   …还有更多！

SitePoint 总部正在关门，因为我们都在假期休假。当我们不在的时候，我们有一些很棒的文章排队等待上线，毫无疑问，我们的一些定期博客会附和任何突发新闻，但否则下周这里可能会有点安静。请放心，我们只是在为即将到来的激动人心的一年积蓄能量！

2008 年再见…
