## 什么是域名？

像电话系统一样，互联网也是基于数字系统。在表面之下，每个网站都与一个 IP 地址相关联——一个告诉你的浏览器在互联网上哪里可以找到该网站的数字地址。

大多数人发现名字比数字更容易让人联想到。您更喜欢通过拨打电话号码来拨打电话，还是在“通讯录”应用程序中点击姓名来拨打电话？同样，你可以把域名想象成 IP 地址的人类可读版本。

一些例子是*sitepoint.com*、*wordpress.org*和*example.com.au*。域名由以下部分组成:

1.  一系列字母、数字和连字符，
2.  一个扩展(称为顶级域名)，如*。com* ，*。组织*、*。biz* ，*。edu* 。
3.  可选的[国家代码](https://www.sitepoint.com/web-foundations/complete-list-country-code-top-level-domains/)，比如。au，。新西兰，英国，美国网站没有国家代码。
4.  这些由一个“点”(句号，或句号)分隔。

域名必须注册后才能使用。

## 选择域名

仔细考虑你的域名——这是人们对你网站的第一印象。它可以强化你的品牌，影响你的搜索结果评分，影响访问者记住如何访问你的网站的难易程度。这个选择值得仔细考虑。

以下是为您的企业或项目选择有效域名的一些提示:

*   保持简单。
*   尽可能保持简短，但不要以牺牲清晰为代价。
*   让它值得纪念。当心棘手的拼写。
*   避免复杂化。添加连字符只是你的访问者要记住的又一件事。如果可能的话，也避免使用数字。
*   瞄准一个*。com* ，是人们所期待的。*。org* 对非营利组织有利，*。gov* 用于政府网站，以及*。edu* 为教育机构。*。net* 也常用。
*   许多新顶级域名已经上市，比如 T2。信息，*。biz* ，*。商业*，*。教会*、*。投资*、*。摄影*、*。软件*和*。电视*。如果它们很适合你的业务，就考虑它们，但是不使用*时要三思。com* 如果有的话。
*   如果你的企业是本地的，你可能希望用相关的国家代码来结束你的域名，比如 *.com.au* 或者 *.org.nz* 。
*   如果你的公司名称已经被占用，可以考虑在开头或结尾加上一些东西，比如*johnsmithplumbing.com*或*hirejohnsmith.com*甚至*iamjohnsmith.com*。
*   避免使用他人的商标，即使该域名是可用的。
*   在[morewords.com](http://morewords.com)(拼字游戏资源)寻找灵感——它可以帮你想出有趣的域名创意。
