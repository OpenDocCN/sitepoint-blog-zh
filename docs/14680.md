# 多语言文档工具

> 原文：<https://www.sitepoint.com/multi-language-documentation-tool/>

随着 David Hecker 关于文档的优秀文章在 SitePoint 上运行，似乎是时候引入一个跨越 14 种编程语言的有趣的开源文档工具了。

在 Sourceforge 找到的 Natural Docs 包括对 C++、C#、Java、JavaScript、Perl、PHP、Python、PL/SQL、Visual Basic、Pascal、Tcl、Ada、Ruby 和汇编的支持。

这个项目的目标是引入自动化工具，使 HTML 文档的生成变得容易，并为整个项目的文档包提供一个易于阅读的起点。

这个项目的路线图是健壮的，包括对所提到的语言的增强支持，从其他文档平台(如 JavaDoc)迁移的能力，输出到 PDF 和 HTML 的能力，最后，允许用户创建自定义设置文件来控制自然文档的输出。

Natural Docs 的有趣之处在于，它可以使那些在 Unix 上使用 Java(使用 JavaDoc)和在 Windows 上使用 Visual Basic 或 C#的开发人员(使用专有的或其他文档工具)在一个文档平台上实现标准化。

我很想知道 SitePoint 读者在使用什么文档工具。
