# 来自小本经营市场的真实数据

> 原文：<https://www.sitepoint.com/real-data-from-marketing-on-a-shoestring/>

我刚刚启动了一个在线非营利援助组织([http://www.actioncorps.org](http://www.actioncorps.org)如果你感兴趣的话)，我想和你分享在小本经营中营销这个项目的经验。

首先，最初的发布是通过以下营销努力进行的:

1.给我所有的朋友、家人和同事发送一封精心制作的电子邮件，让他们发送到他们的网络，邀请人们注册。

2.PRWeb 上的新闻稿，我花了大约 80 美元获得了 2 星发布。

3.随后在 Businesswire 上发布了一篇新闻稿，我花了大约 160 美元在我的地理区域发布了一篇新闻稿，并利用了潜在志愿者可能会阅读的数百份重点出版物(如老年人、慈善、教育)。从历史上看，我更喜欢 Businesswire 而不是 PRWeb，我想测试一下这两者，看看更便宜的 PRWeb 是否有效果。

4.在我居住的地方发行 5 种出版物，从当地报纸到我的房主协会时事通讯。

以下是结果:

到目前为止，PRWeb 和 BusinessWire 都没有产生太多的反应(这是意料之中的，因为公关需要持续的努力来获得动力)。一个注册来自 BusinessWire，没有一个来自 PRWeb。还没有媒体调查。

给我朋友和家人的电子邮件导致了一些注册，尽管坦率地说结果令人失望。我想我自己的关系网认为我无论如何都会给他们发电子邮件，告诉他们这个项目，所以不需要注册(或者他们告诉我他们不感兴趣，但根据电话交谈，我对此表示怀疑)。

最好的回应来自于当地的在线商业新闻服务。这导致了许多注册，并邀请在当地扶轮社发言。在那里演讲会导致更多的注册。

像往常一样，有针对性的无成本努力会得到最好的结果，并且很快会带来更多的结果。

随着更多结果的出现，我会随时通知你，并且我会继续以小本经营的方式推广这个产品。随后的营销将需要花费几乎 0 美元，所以它将全部基于电子邮件，电话等。与可能的人和组织建立关系和伙伴关系。这就是小本经营的方式。
