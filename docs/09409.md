# 伟大的销售=伟大的生意？

> 原文:[https://www.sitepoint.com/great-sales-great-business/](https://www.sitepoint.com/great-sales-great-business/)

9 年前，当我开始创业时，我为自己的销售转化率感到自豪——它非常接近 100%。太棒了，对吧？直到我把这件事告诉一位商业顾问，他才把我拉回现实。

看，他的推理是，如果我几乎赢得了我提出的每一个提议，那我就太廉价了。事实证明这是真的——我询问了类似规模企业的其他人，以及办公桌上还有第二份或第三份提案的客户，结果发现我比所有竞争对手都便宜得多。

我提高了价格，开始赢得更少的工作，然而开始获得更好的利润，总体上有更多的预算来做更好的工作。

当然，有些人的血液中有天生的销售能力，90%的成功率是可能的，但是考虑到这一点；如果你是最便宜的人之一，你如何赚足够的钱来发展你的业务？你可能不是。

扩大业务需要资金，新员工和新设备会在短期内对你的现金流产生负面影响，你需要有能力弥补这些影响。

现在，当我听到人们说他们正在完成几乎每一笔交易时，我会分享同样的宝贵建议，并观察他们意识到自己在欺骗自己时的反应。

继续——收取更多费用。这意味着你将有更多的钱来做更好的工作，你永远不知道，有一天你可以负担得起自己做更少的工作，并享受在你的业务上工作而不是在 It 上工作。

祝你好运！
