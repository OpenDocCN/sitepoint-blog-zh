# 网站管理员一言以蔽之，第二版

> 原文:[https://www.sitepoint.com/nutshell-2nd-edition/](https://www.sitepoint.com/nutshell-2nd-edition/)

**![Webmaster in a Nutshell](../Images/539758429f611b3ddc04c14dd7bcd0e2.png)T2】**

随着互联网的发展，作为一名网站管理员，你必须对更多的编程语言和技术越来越了解和熟练。首先是 HTML，然后是 JavaScript。CSS 是下一个热点。每一次你都必须通过阅读来提升你的技能。

到目前为止，我们大多数人都有大量关于 HTML、JavaScript、HTML 4、XML 等的书籍。坐在我们的电脑前做参考。如果你能把那堆书挤成一本会怎么样？你说不可能。嗯，不再是了。奥赖利刚刚发布了果壳第二版网站管理员。这本书设法将几种语言和技术压缩成一本紧凑的参考书

作者 Stephen Spainbour 和 Robert Eckstein 设法在一本厚达 523 页的书中涵盖了 HTML 4.0、CSS、XML、CGI、SSI、JavaScript 1.2、PHP、HTTP 1.1 和 Apache 服务器的管理。

本书的第一部分介绍了 HTML 4.0，随后是该语言的完整参考，包括关于框架、表格和表单的章节。

本书的第二部分还包含了对级联样式表的精彩介绍，包括对 53 种可用属性的完整参考。第三部分介绍了 XML，这是一种新兴技术，可能最终会主宰互联网。

第四部分是 JavaScript/JScript 1.2 的完整参考。这本书的这一部分包括一个很好的图表来说明客户端对象的层次结构。

第五部分涵盖了 CGI 和 PERL，包括关于服务器端的详细信息，Perl CGI.pm 模块和使用 mod _ PERL(Apache 的一个模块)的 web 服务器编程。

第六部分讨论 PHP，这是一种嵌入 HTML 的脚本语言，允许您创建动态站点、处理表单和从任何 ODBC 兼容的数据库中提取数据。

本书的第七部分介绍了 HTTP 1.1(超文本传输协议)。本书的这一部分包括对 Cookies、媒体类型、客户端请求、服务器响应和状态代码的概述。

第八部分讨论服务器配置，特别是 Apache，它是当今最流行的 web 服务器软件，占据了大约 70%的市场份额。这一节提供了关于 Apache 的几十个模块的信息，此外还有关于提高服务器性能的大量信息，我发现这些信息特别有趣和有用。

不要在你的办公桌上放着涵盖 JavaScript、HTML、PHP 和 Apache 的独立书籍，而是简单地买一本《网站管理员》,把它们都放在一个地方。

售价 24.95 美元，这本书在节省时间和提高生产率方面将会获得一千倍的回报。
